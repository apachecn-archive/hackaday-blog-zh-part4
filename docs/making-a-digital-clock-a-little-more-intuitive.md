# 让数字钟更直观一点

> 原文：<https://hackaday.com/2019/06/18/making-a-digital-clock-a-little-more-intuitive/>

数字钟非常有用，通常被认为非常容易阅读。然而，它们有时会有相当神秘的界面来设置时间和闹钟。对于[迈克尔·韦塞尔]，他指出，在 20 世纪 80 年代，正是因为这个原因，他不得不例行公事地帮他的祖父母调时钟。这激发了他最近的项目——一个直观易用的数字时钟。

许多数字钟都以同样的方式工作，先设定一个数字的时间，然后按下另一个按钮循环到下一个数字。这可能会令人困惑，所以[迈克尔]走了一条不同的路。相反，每个数字都可以使用自己的按钮进行循环，这使事情变得更容易。人们如何选择设置时间、日期或闹钟并不容易理解，但如何创建这样的界面是一个有趣的尝试。

时钟依靠 Arduino Mega 来运行，RTC 用于计时，温度传感器用于启动。还有一个声音传感器，可以通过拍手或大喊“停止”来关闭警报。总来说，这是一个整洁的建筑，具有黑客最喜欢的七段美学。当然，你也可以将这个概念发挥到极致。休息后的视频。
