# Hackaday 播客 029:你的脸在银色的沙子里，未来的轮胎，ESP32 所有的 CNC 东西，还有水壶里的 Sub

> 原文:[https://hackaday . com/2019/08/02/hackaday-podcast-029-your-face-in-silver-sand-tires-of-the-future-esp32-all-the-CNC-things-and-sub-in-a-jug/](https://hackaday.com/2019/08/02/hackaday-podcast-029-your-face-in-silver-sand-tires-of-the-future-esp32-all-the-cnc-things-and-sub-in-a-jug/)

Hackaday 的编辑埃利奥特·威廉姆斯和迈克·斯奇斯对最新的黑客攻击进行了研究。本周我们看到了几个巧妙的 CNC 构建，它们利用了 GRBL 的一个很棒的 ESP32 端口。基于柠檬水水罐的潜艇项目是你认为无法在水下机器人中工作的一切。亚马逊最新的 Dot 已经取消了保修，以炫耀这些天 22 英镑能给你带来什么。还有一个关于调试电路的很棒的教程，是从一周的失败中成长起来的。此外，我们对雄心勃勃的无气汽车轮胎发布时间表感到失望，并对 USB-C 的混乱世界表示同情。
