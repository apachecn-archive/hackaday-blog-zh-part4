# 无人驾驶船航行西雅图

> 原文:[https://hackaday.com/2021/03/11/drone-boat-sails-seattle/](https://hackaday.com/2021/03/11/drone-boat-sails-seattle/)

由于廉价、强大的自动驾驶模块的存在，制造小型自动驾驶汽车现在是普通制造商力所能及的事情。[rctestflight]长期以来一直是该领域的爱好者，并且已经尝试在华盛顿湖上执行远程自主任务有一段时间了。他最近的尝试证明是非常成功的。(视频，下面嵌入。)

这一建造是早期尝试的继续，尝试用一艘汽艇执行 13 公里的任务，选择 T1 本身是为了避免早期测试中海藻缠绕螺旋桨的问题。在这次尝试中，[丹尼尔]选择用玻璃纤维建造一个定制的船体，并结合水下螺旋桨和风扇。目的是提供足够的推力，同时也是为了冗余。另外，风扇可以随着船舵旋转，有助于提供更大的转向权限。

在对 ArduPilot 控制系统进行了大量的调整后，被恰当地命名为 SS 香蕉蛞蝓的飞船已经准备好执行它的远程任务了。尽管由于寒冷，早期有人担心电池电压低，但这艘船完成了横跨湖面 13 公里的长途运输，总任务时间超过 3 小时。后来的效率计算表明，船上的电池可能在耗尽之前处理超过 100 公里的任务。

这表明，即使有现成的自动驾驶仪和地图解决方案，任何成功的远程任务仍然需要大量的工程，无论是[陆地](https://hackaday.com/2019/09/12/a-4g-rover-and-the-benefits-of-a-shakedown-mission/)、[海洋](https://hackaday.com/2017/07/30/autonomous-boat-sails-the-high-seas/)还是[空中](https://hackaday.com/2019/10/09/soaring-with-the-sun-4-years-of-solar-rc-planes/)。
