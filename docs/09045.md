# 苹果手表获得定制透明外壳

> 原文：<https://hackaday.com/2021/01/27/apple-watch-gets-custom-transparent-case/>

苹果手表是这家科技公司将腕式电脑带入主流的尝试。它自然有各种各样的款式和表面，但是如果你想要一些真正的原创，你必须定制。[[无用的 Mod]为流行的智能手表做了一个清晰的案例。](https://www.youtube.com/watch?v=c9rJFID30JQ)

mod 从耐心、小心地拆卸腕表开始——考虑到腕表内部的精密部件，这是必要的。最终只需钻出 1 个螺钉，不幸的是，冠轮轴断裂。然而【无用 Mod】压上，硅胶铸出苹果原装外壳。该视频讲述了所有的细节，从脱气到使用丙烯酸塑料条作为滑道。一旦完成，硅胶模具被用来生产透明环氧树脂的复制品，手表被重新组装。

最终的结果令人印象深刻，表壳光学清晰，展示了手表的内部。通过移除一些原始绝缘带来更好地显示内部的 PCB，外观得到了改善。不幸的是，手表的设计很大程度上被屏幕和心跳传感器覆盖，这意味着它不是 clear case mod 的最佳选择，但它仍然有效。我们以前也看到过类似的作品，比如 Mavic Mini 的透明无人机外壳。休息后的视频。
