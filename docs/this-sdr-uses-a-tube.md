# 这个 SDR 使用一个电子管

> 原文：<https://hackaday.com/2019/03/07/this-sdr-uses-a-tube/>

当您想到软件定义无线电(SDR)设置时，可能会想到一两个 IC，可能是为计算机供电。你可能不会想到真空管。[米尔科·帕夫莱斯基]使用[布尔夏德·凯恩卡]的一些德语指令建造了一个单管短波 SDR，但是如果你想复制他的壮举，[谷歌翻译](https://translate.google.com/translate?sl=auto&tl=en&u=http%3A%2F%2Fwww.b-kainka.de%2Fbastel100.htm)已经足够好了。你可以在下面看到[米尔科]创作的视频。

这个建筑是一个实验，看看一个电子管接收器是否能够足够稳定地接收数字短波广播。为了避免交流线路嗡嗡声，收音机是电池供电的，而原来使用的是 EL95 电子管，[米尔科]使用的是 EF80。

为了获得必要的稳定性，重要的是一切都是安全的。最初的建造确保了管子在操作过程中不会移动，尽管[Mirko 的]管子安装看起来更传统，但仍然非常安全。天线的松散耦合也有助于稳定性，并且调谐调整应该具有更长的轴，以最小化调谐旋钮附近的手电容。另一位建造者[Karl Schwab]指出，只有大约 1/3 的调谐范围是可用的，因此电容器上的减速齿轮也将是受欢迎的。

电子管既是振荡器又是混频器，所以接收机是一种直接变频接收机。电子管的灯丝消耗约 200 毫安，因此电池供电是可行的。

根据[布尔夏德]的说法，他的体型每分钟漂移不到 1 赫兹，这还不错。正如你在视频中看到的，它的效果足够好。顺便说一下，EF80 本质上是一个不同底座的 EF50，电子管帮助赢得了第二次世界大战。如果你喜欢建造一切，也许你可以用一根[自制的管子](https://hackaday.com/2016/05/04/home-brew-vacuum-tubes-are-easier-than-you-think/)尝试同样的壮举。
