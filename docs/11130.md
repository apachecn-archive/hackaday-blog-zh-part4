# 看似简单的过程将瓶子变成细丝

> 原文:[https://hack aday . com/2021/08/25/看似简单的过程——将瓶子变成细丝/](https://hackaday.com/2021/08/25/deceptively-simple-process-turns-bottles-into-filament/)

如果你知道大多数汽水瓶是由 PET 塑料制成的，你可能会想到如何用它们制造细丝，并拥有源源不断的廉价印刷材料。[Mr3DPrint]说[他有一个方法](https://www.youtube.com/watch?v=Eecbdb0bQWQ&t=311s)并分享了几个让它看起来简单的视频。我们想知道灯丝的质量是否达到商业产品的标准，但假设视频是准确的，那么最终的灯丝似乎完成了任务。

细节有点粗略，但看起来足够简单。第一步是去除瓶子上的任何凹痕。他对此进行了多次演示，有些使用了瓶中的加压空气，有些则没有。不过，在每一种情况下，钻头都会穿过瓶盖，在火焰上旋转瓶子，直到表面光滑。

一旦基本弄平，他切掉瓶底，用安装好的刀片将瓶子切成细条。下一个夹具是标准热端，带有水平安装的 1.75 毫米喷嘴。我们假设他钻了一个 1.75 毫米孔的标准喷嘴，或者考虑到模具膨胀，可能是一个更小的孔。

据推测，剃刀产生了足够窄的条带以适合喷嘴的顶部。他把塑料推进去，用打印机的热障把它绑在一根绳子上，一个小机器把剩下的部分拉过来，同时把它缠绕在一个卷轴上。

我们猜测灯丝的直径到处都是，我们想知道垂直或水平放置灯丝是否会得到最好的结果。然而，如果你想尝试的话，这似乎是一件容易的事情。在评论中，[Mr3DPrint]提到了一些让你开始的温度范围，但我们没有找到任何关于机器设计的参考。另一方面，它们似乎很容易制作，尤其是如果你有一台 3D 打印机的话。

我们见过的[挤压机](https://hackaday.com/2020/07/17/make-your-own-filament/)要复杂得多。然而，大多数的[和那些](https://hackaday.com/2013/11/01/low-cost-filament-extruder/)使用颗粒饲料。
