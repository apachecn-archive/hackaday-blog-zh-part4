# 珀尔帖装置实验

> 原文:[https://hackaday.com/2019/11/13/peltier-device-experiments/](https://hackaday.com/2019/11/13/peltier-device-experiments/)

固态热泵或帕尔贴设备曾经是一种舶来品，现在已经相当主流。这个想法很简单:将电流通过珀尔帖装置，一边变热，另一边变冷。[DroneBot]最近发布了一个视频[，展示了这些非常酷的设备是如何工作的](https://dronebotworkshop.com/peltier-effect/)。你可以看下面的视频。

物理学中的许多事情都是可逆的，珀尔帖也不例外。该器件实际上是一种热电偶，在热电偶中，温度差会导致电压差。这就是所谓的塞贝克效应，与帕尔贴效应相反，在帕尔贴效应中，电流在电压差之间流动会产生温差。人们知道它已经很多年了，但是直到现代半导体材料的出现，它才变得非常实用。

问题是所有的热量都要去某个地方，所以需要一些方法将热量从热端带走。视频中的实验使用的是 TEC1-12706。这个数字可能看起来很神秘，但它实际上标识了视频中解释的模块参数。

这些冷却器的效率不是很高，但是对于那些你需要用很少的额外体积和质量来冷却一个区域的地方来说，这些正是你想要的。我们喜欢看到这些在奇怪的上下文中使用。例如，在云室中使用帕尔贴代替干冰。虽然效率不高，但你可以用它来做一种[空调](https://hackaday.com/2011/07/02/poor-mans-peltier-air-conditioner/)。
