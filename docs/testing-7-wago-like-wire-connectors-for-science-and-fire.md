# 测试 7 个用于科学和消防的 Wago 型接线器

> 原文：<https://hackaday.com/2022/04/25/testing-7-wago-like-wire-connectors-for-science-and-fire/>

在节省几块钱和因房屋火灾引起的昂贵的家庭保险索赔的交叉点上，我们发现了经过认证和测试的电气连接器的克隆产品，即使许多人会认为所谓的接线螺母是火灾隐患，不管它们上有多少认证标签。当谈到没有大惊小怪的电线连接器，Wago 夹具连接器是一个有吸引力的目标，以节省一些钱，因为他们认为高成本。但是它们到底有多贵呢？

这是最近由[GreatScott！](休息后也嵌入)当他跳上大家最喜欢的电商网站，搜索‘钳杆终端’的时候。最终选择的七个连接器有各种各样的形状、颜色和配置，尽管所有连接器都被认为是额定电源(250 VAC)电压，并且足够安全，可以永久安装。

在对连接器进行高电流、防火和机械强度测试时，得出的结论是，所有连接器都足够好，适合业余爱好者使用，并且在测试时进行一些简单的连接，但只有那些具有独立认证标志的连接器(如 VDE)让他有足够的信心考虑在室内布线中使用。其中之一是德国品牌 ViD 的连接器，这似乎是 Wago 连接器的一种稍微便宜的替代产品，具有类似的安全保证。

归根结底，重要的是认证，因为长期可靠性是房屋布线的首要考虑因素，而不是是否节省了几欧元的材料成本。
