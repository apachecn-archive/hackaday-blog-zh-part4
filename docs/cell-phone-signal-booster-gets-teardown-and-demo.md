# 手机信号助推器得到拆卸和演示

> 原文：<https://hackaday.com/2020/09/10/cell-phone-signal-booster-gets-teardown-and-demo/>

有没有想过手机助推器里面有什么，或者设置或使用一个助推器是什么感觉？如果是这样的话，【Kerry Wong】已经让你知道了[他对 Nextivity](http://www.kerrywong.com/2020/07/05/teardown-of-a-cel-fi-go-x-cell-signal-booster/) 的 Cel-Fi Go X Cell 信号增强器的拆卸。[Kerry]不仅仅是为了搞笑而拆一个便宜的单元；他的房子信号很差，这个单元是经过仔细研究的，是对更好的 4G 连接的真正投资。

整个系统由三个不同的部分组成:上图中的放大器单元和两根天线。一种是用于室内的全向圆顶天线，另一种是用于室外的定向对数周期偶极子阵列( [LPDA](https://en.wikipedia.org/wiki/Log-periodic_antenna) )天线。手机连接室内天线，室外天线连接远处的手机发射塔。放大器单元使用蓝牙连接和移动电话上的应用程序来管理设置并主动监控设备，这很好，但奇怪的是，似乎没有采用任何类型的密码保护或访问控制。

总的来说,[Kerry]很开心，他说他的手机在整个房子里都能享受到稳定的连接，这在以前是不可能的。观看拆卸操作以及下面视频中的简短演示。
