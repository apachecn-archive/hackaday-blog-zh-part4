# 口袋大小的毁灭战士实际上是可玩的

> 原文：<https://hackaday.com/2022/01/26/pocket-sized-doom-is-actually-playable/>

过去，如果你想玩《毁灭战士》,你需要一台装备精良、价格昂贵的新米色电脑。现在，你可以在一个比信用卡更小的外形中做到这一点，Adafruit 的这个漂亮的小构建就展示了这一点。

构建依赖于用于 ESP32 设备的 [Retro-Go 固件](https://github.com/ducalex/retro-go)，它可以模拟一系列机器，从任天堂 nes 和 Game Boy 到 NEC PC 引擎、雅达利 Lynx，是的，还有 Doom 本身。它甚至可以通过游戏使用的 WAD 架构来运行末日模式。

将 Retro-Go 移植到微小的 QT Py ESP32 Pico 板上运行是一件简单的事情，一切都井井有条。有了六个触觉按钮，它不仅可以运行《毁灭战士》，还可以全速运行，包括经典的音乐。1.3 英寸 240×240 的屏幕看起来出奇的清晰，在显示游戏的同时保持一切可读。

这是我们见过的最小的厄运便携式电脑之一；我们认为如果你足够努力的话，你可以把它塞在牛仔裤的零钱袋里。我们永远也不会忘记看到世界上最受欢迎的 FPS [在商用厨房硬件上运行，尽管](https://hackaday.com/2021/04/17/kitchen-bump-bar-plays-doom-between-orders/)。休息后的视频。
