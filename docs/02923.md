# 超级马里奥土地 DX ROM 黑客展示了什么样的游戏男孩可能看起来像

> 原文：<https://hackaday.com/2019/05/06/super-mario-land-dx-rom-hack-shows-what-game-boy-could-have-looked-like/>

是时候了(马里奥时间)，原来的游戏男孩超级马里奥土地被重新访问。这款游戏是数百万人进入便携式游戏世界的入口，也是玩家可以从少量 AA 电池中期待的冒险类型的早期例子。最初的 Game Boy 系统本身可能只能显示四种灰色，然而，这从未阻止《超级马里奥大陆》的玩家想象游戏在惊人的颜色下会是什么样子。现在多亏了[toruzz],我们不再需要去想象，因为他们的超级马里奥之地 DX ROM Hack 就是这么做的……还有更多。

超级马里奥陆地 DX ROM hack 坚持 Game Boy Color 的 16 位调色板，所以它实际上运行在真实的硬件上。游戏玩法没有任何改变，它也以 Game Boy 的 10:9 纵横比运行。根据补丁自述文件，建议使用合法来源的《超级玛丽之地》1.0 版本的转储文件，[利用 Lunar IPS](http://www.romhacking.net/utilities/240) 应用补丁。此外，还提供了 CRC 校验和，以确保每个人都从相同的起点开始工作。

《超级马里奥之地》是 1989 年 Game Boy 推出的游戏，但那一年也发布了另一款手持游戏系统(Atari Lynx)。Lynx 配备了全彩色背光液晶显示器，因此那个时代的手持游戏系统并不局限于单色。诚然，Lynx 的价格几乎是 Game Boy 的两倍，但一个革命性的 ROM 黑客，如超级马里奥土地 DX one 几乎可以作为一个替代历史。我们现在都可以体验到的另一段历史，可以是在桌面上，也可以是真正的便携形式。
