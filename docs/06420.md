# 将拱廊橱柜放入 NES 控制器中

> 原文:[https://hack aday . com/2020/05/01/puting-an-arcade-cabinet-inside-an-nes-controller/](https://hackaday.com/2020/05/01/putting-an-arcade-cabinet-inside-of-an-nes-controller/)

[从[泰勒·伯利]的任天堂娱乐系统控制器](https://hackaday.io/project/171204-tiny-arcade-stuffed-in-an-nes-controller)中硬塞进去的街机游戏，通过从那些现在风靡一时的微型橱柜中的一个开始，当然使*稍微容易了*，但是因为他仍然在一个控制器中实现了整个街机游戏的壮举，我们仍然袖手旁观在我们的标题中的评估。

[![](../Images/77f1c43270ecbfbb1d725f635f5b54dd.png)](https://hackaday.com/wp-content/uploads/2020/04/nesarcade_detail.jpg) 事实上，他把不是一个而是*四个*街机游戏放进了控制器。[泰勒]从微型游戏系统中解放出来的电路板实际上可以通过短路 PCB 上的不同焊盘在板载游戏之间切换。通常情况下，这将在制造过程中使用零欧姆电阻来完成，但在这种情况下，他将焊盘连接到一条从 LED 遥控器中解放出来的薄膜键盘。通过在系统通电时按住不同的按钮，用户可以选择他们想要启动的游戏。

原来的按钮和方向键被保留了下来，在休息后的视频中，[Taylor]展示了他如何将它们连接到街机 PCB 上。“开始”和“选择”按钮不得不取消，因为那是现在微型彩色液晶显示器的位置，但无论如何它们都不会在这些游戏中使用。通过添加一个小型电池组和充电控制器，这种构建是一种在旅途中随身携带几款经典街机游戏的聪明方式。

随着这些小型拱廊橱柜越来越受欢迎，我们已经看到许多黑客侵入它们。如果你想用这些怀旧的小玩意来构建一个项目，那就不要错过[wrongbaud]在修改它们以运行其他 rom 方面所做的工作。
