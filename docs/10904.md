# 火箭联盟启发家酿逆转到任天堂游戏魔方

> 原文:[https://hackaday . com/2021/08/03/rocket-league-inspired-home brew-reverses-on-Nintendo-game cube/](https://hackaday.com/2021/08/03/rocket-league-inspired-homebrew-reverses-onto-nintendo-gamecube/)

把一个关于足球和遥控汽车的游戏叫做足球汽车会不会太明显了？嗯[马丁]这样认为，并选择将他的任天堂 GameCube 家酿游戏命名为，复古联盟 GX。这款游戏显然受到了 Psyonix 开发的火箭联盟的启发，因为它在一个球场上让多队汽车进行比赛，并配有完整的助推系统。这里展示了一些令人印象深刻的物理学，据 GBATemp 的 Krista 说[所有东西都可以在原始硬件上玩。虽然那些没有 GameCube 的人肯定可以通过 Dolphin 模拟器找到匹配。](https://gbatemp.net/threads/homebrew-demo-retro-league-gx-demakes-rocket-league-for-the-nintendo-gamecube.591981/)

有很多方法可以在任天堂 GameCube 上启动自制软件，然而，最重要的软件是瑞士的。Swiss 是一个自制工具,它可以通过各种方式将代码加载到 GameCube 上。加载自制软件的常见方式包括将文件保存到 SD 卡上，然后使用插入存储卡端口的 SDGecko 设备，或者插入位于控制台底部的 GameCube 扩展端口之一的 SD2SP2 设备。那些喜欢完全抛弃光盘驱动器的人可以通过光盘模拟器加载自制软件，比如 GC Loader。

仍然在路线图复古联盟 GX 是 3DS，PSP，Wii 和 Linux 的端口。局域网和在线多人游戏也在开发中。因此，至少 GameCube 宽带适配器拥有者可以超越幻想之星在线一次。最棒的是，[马丁]声明 GX 复古联盟的代码将在明年的某个时候开源。
