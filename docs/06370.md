# 仅使用电弧焊机从水中提取燃料

> 原文:[https://hack aday . com/2020/04/26/燃料来自水仅使用电弧焊机/](https://hackaday.com/2020/04/26/fuel-from-water-using-only-an-arc-welder/)

水，高电流，令人眼花缭乱的等离子球，以及一种高度易燃的气体，如果你呼吸足够多，这种气体的毒性足以在三分钟内杀死你。这个等离子驱动的水煤气发生器有什么不让人喜欢的？

严肃地说，[NightHawkInLight]正在玩一些危险的东西，他非常坚定地认为这是“不要在家里尝试”的一类。但这也是令人着迷的东西，因为它只用一箱水和一个电弧就能非常快地产生大量有用的燃料。很容易得出这样的结论，他说的是将水电解分解成氢氧混合物 HHO，但这完全是另外一回事。

使用一个连接到他的弧焊机的碳电极焊炬，一个类似于他用来制造[人造红宝石](https://hackaday.com/2020/03/06/a-quick-and-easy-recipe-for-synthetic-rubies/)的装置，[NightHawkInLight]能够在一艘看起来像双管水枪的船内点燃水下电弧。等离子体产生了一氧化碳和氢气的混合物，这种混合物在他建造的[煤气表](https://hackaday.com/2018/07/11/building-a-gasometer-to-store-wood-gas-and-other-bio-fuels/)中迅速积累，以收集木材气化炉产生的易燃产品。

水煤气燃烧起来非常干净，但实际用途可能有限。除非你住在电费几乎为零的地方，否则很难做到收支平衡。尽管如此，这仍然是一个有趣的现象，看看当等离子体和水混合时会发生什么。
