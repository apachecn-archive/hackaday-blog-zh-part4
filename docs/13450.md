# 90 年代苹果电脑终于运行未签名代码

> 原文:[https://hack aday . com/2022/04/28/90s-apple-computer-finally-runs-unsigned-code/](https://hackaday.com/2022/04/28/90s-apple-computer-finally-runs-unsigned-code/)

早在 90 年代，游戏机大战如火如荼。任天堂 vs 世嘉起初是一场史诗般的对决，但当任天堂似乎肯定会赢得胜利时，索尼突然推出了 PlayStation。虽然这些是当时最受欢迎的游戏机，但周围还有其他一些游戏机，尽管它们在某些方面是革命性的，但在很大程度上被历史遗忘了。苹果公司生产的游戏机 Pippin 就是一个例子，到目前为止，它还不能运行任何未经苹果公司签名的软件。

Pippin 是苹果进军游戏主机的唯一尝试，但它的作用远不止于此，它还包括一个原始的社交网络系统，以及运行苹果 Macintosh 操作系统的能力。这个想法是成为一个完整的媒体中心，它运行的软件将在每次启动时从光盘中加载。[Blitter]终于破解了这台电脑，允许它运行定制软件，方法是创建一个认证文件，放在 CD 上，告诉 Pippin 它已被苹果“批准”。

构建日志对这些机器的运行方式进行了令人难以置信的详细描述，如果你还有一个 Pippin，那么是时候把它从盒子里拿出来，按照你一直想要的方式进行定制了。对于那些对其他不知名的苹果产品感兴趣的人来说，看看这个版本[吧，它将现代 WiFi 引入了苹果牛顿](https://hackaday.com/2016/02/09/giving-wifi-to-an-apple-newton/)，这是他们早期的 PDA。
