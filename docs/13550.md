# ElectriPop 将切割聚酯薄膜转变成定制的 3D 结构

> 原文:[https://hack aday . com/2022/05/06/electri pop-turns-cut-mylar-into-custom-3d-structures/](https://hackaday.com/2022/05/06/electripop-turns-cut-mylar-into-custom-3d-structures/)

聚酯薄膜有很多有用的特性，因此，不仅在 DIY 项目中，而且在我们的日常生活中，它经常出现。但直到今天，我们从未见过一块聚酯薄膜跳起来试图引起我们的注意。但这正是 ElectriPop 提供的承诺，它是卡内基梅隆大学未来接口小组的一个令人着迷的项目。

这里工作的核心原则相当简单。当带静电时，一条聚酯薄膜可以垂直升到空中。从中间切开，两边会互相排斥，形成一个“Y”形。通过对这一概念进行足够仔细的切割，有可能创造出令人惊讶的复杂三维形状，一旦施加电荷就会弹出。甚至可以通过调节输入功率来引入一定程度的运动。休息后的视频提供了几个这一原则的例子:比如一朵 3D 花，它要么挺立，要么相对于外部数据源枯萎，或者一个化身疯狂地挥舞手臂以引起用户的注意。

[![](../Images/e455a48ef989289c416527c683ad0818.png)](https://hackaday.com/wp-content/uploads/2022/05/electripop_detail.jpg)

A mini Van de Graaff produces a variable static charge.

由于聚酯薄膜片上的嵌套切口、狭缝和孔洞与最终 3D 形状之间的关系不是特别直观，团队[开发了一个可视化工具](https://github.com/FIGLAB/MylarFilmSimulator)，可以与现有的矢量艺术程序结合使用，以创建 2D 切割文件。一旦你满意的设计将膨胀到预期的形状，它可以很快实现乙烯切割机或激光。进入门槛低使得这个项目特别适合 DIY 复制，我们渴望看到创客社区如何将这个概念付诸实践。

有很多方法可以为你的新变形聚酯薄膜显示器充电，但我们特别喜欢团队的微型 Arduino 控制的 Van de Graaff 发电机。它可能代表了我们见过的这个经典高压发生器的最小和最简单的实现，本身看起来就像一个有趣的小项目。

ElectriPop 由[Cathy Fang]、[Jianzhe Gu]和[Lining Yao]与未来接口集团总监[Chris Harrison]合作开发。读者可能还记得，在 2018 年的[中，我们报道了振动视线](https://hackaday.com/2018/10/22/vibrosight-hears-when-you-are-sleeping-it-knows-when-youre-awake/)，这是【Chris】和他的学生的另一个项目，该项目最终被扩大到[，去年刚刚在城市规模上演示了激光非接触传感](https://hackaday.com/2021/05/21/with-a-big-enough-laser-the-world-is-your-sensor/)。请在不久的将来继续关注这个创新实验室的更多信息。
