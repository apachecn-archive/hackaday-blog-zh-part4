# 戴夫的声音，现在是玩具形式

> 原文:[https://hack aday . com/2019/01/26/the-voice-of-Dave-now-in-toy-form/](https://hackaday.com/2019/01/26/the-voice-of-dave-now-in-toy-form/)

达夫·琼斯是黑客和创客社区的名人，他将自己丰富的知识和经验应用于各种电子和工程主题。他独特的声音和坦率让他赢得了许多人的喜爱，他是目前现场最受欢迎的 YouTubers 用户之一。考虑到这一点，[Andrei Aldea]决定以他的名义建造一座演讲台。

这个建筑的灵感来自于[本·赫克]的早期作品。【Andrei】从拆解 Speak 'n' Say 开始，确定有一个简单的按钮矩阵，可以让玩具的位置很容易被读取。这些按钮被连接到一个 Arduino Nano 上，然后它读取 SD 卡并使用 DFR 机器人的 MP3 模块播放声音样本。

[Andrei]确保玩具充满了定制图形，使外观更加完整。此外，公告板上的每一句戴夫的话都有一系列由 Arduino 随机选择的原声片段，比原来的玩具多了一点变化。

这是一个有趣的构建，我们希望看到一个带有 line-out 或一些更大扬声器的版本，以真正令人印象深刻的音量播放大卫福音。我们也见过其他会说话的 Arduinos 比如这个令人印象深刻的 HAL build 。休息后的视频。
