# 机器人洗碗池帮助洗碗

> 原文：<https://hackaday.com/2020/11/02/robot-sink-helps-with-the-dishes/>

在过去的一个世纪里，简陋的厨房水槽相对来说保持不变。虽然现在有更花哨的混合水龙头和更时尚的配件，但对我们大多数人来说，有一个喷嘴可以喷出不同温度的水，还有一个浴缸可以浸泡盘子。当然，这留下了很大的改进空间，正如[杰克]在他的机器人水槽建造中发现的那样。

水槽由一个机器人喷嘴组成，他称之为“连续体机械手”。本质上，这是一个可以用操纵杆控制的喷嘴，可以让水流通过水槽。为了移动喷嘴，马达拉动连接在软管周围的 3D 打印套环上的钢缆。结合水流的开/关开关，该水槽可以成为残疾人的一项有用的辅助技术，正如该项目的演示视频所示。

我们可以很容易地想象这种硬件与简单的计算机视觉系统相结合，进一步自动化洗碗。该项目实际上是对[杰克]正在做的工作[的概念验证，以探索具有类似硬件的 3D 打印混凝土](https://www.jakeammons.com/projects/concrete_printing/)，尽管规模扩大到更工业化的水平。我们以前也展示过类似的技术——[以 DIY 触手构建的形式，非常适合万圣节](https://hackaday.com/2016/09/13/the-bootup-guide-to-homebrew-two-stage-tentacle-mechanisms/)。休息后的视频。
