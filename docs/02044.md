# 反向断层扫描 3D 打印

> 原文:[https://hack aday . com/2019/02/05/3d-反向断层成像打印/](https://hackaday.com/2019/02/05/3d-printing-with-tomography-in-reverse/)

我们最熟悉的 3D 打印机使用熔融沉积工艺，在这种工艺中，热塑料从喷嘴中喷出，在一层一层的基础上构建零件。我们还看到了立体平版印刷机，如 Form 2，它使用投影仪和特殊树脂来生产零件，也是以一层一层的方法。然而，北卡罗来纳大学的一个团队受到 CT 扫描仪的启发，想出了一种生产 3D 打印零件的新方法。

![](../Images/bfd2a0ff82e952e95a64e2cc0291bd7a.png)

The process, as outlined in the team’s paper.

这项技术被称为计算机轴向光刻。该团队将该系统描述为像 CT 扫描一样反向工作。创建 3D 模型几何形状，然后通过围绕垂直轴旋转零件来创建一系列 2D 图像。这些 2D 图像然后被投射到一个光敏树脂的圆柱形容器中，容器在这个过程中旋转。当圆柱体旋转时，零件不是由 Z 轴上的一系列层构建而成，而是由一系列轴向切片构建而成。

生产的零件具有光滑的表面光洁度，并且非常透明。该团队打印了各种测试对象，包括著名思想家雕塑的复制品，以及人类下颚的复制品。特别令人感兴趣的是能够制作包含现有物体的印刷品，用包含现有钢柄的螺丝刀手柄来演示。

假设正确的树脂不是很难获得，这种技术可能被足智多谋的制造商复制。树脂市场正在升温，Prusa 在最近的一次展销会上宣布了新产品。我们很高兴看到接下来会发生什么，特别是当树脂的高成本因规模经济而降低时。休息后的视频。
