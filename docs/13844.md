# 从零开始的太阳能框架

> 原文：<https://hackaday.com/2022/06/06/a-solar-frame-from-scratch/>

“从零开始”是一个有点模糊的表达。有多痒？你只是自己烤面包还是自己种小麦？生产自己的太阳能装置可能不包括制造自己的光伏电池。但是说到制作框架来支撑你的面板，为什么不自己加工支架，并从附近的树木中获取木材呢？。这就是克里斯·哈伯(Kris Harbour)在他过度设计的 8.4 千瓦太阳能支架上所做的。

他已经有了一个风力涡轮机和一些太阳能电池板，但是克里斯想要更多的电力。在视频开始时，[克里斯]即兴评论说，他希望这至少持续 30-40 年。从对混凝土基脚的关注到超大的支架和横梁，一切都反映了这一点。支架是在他的等离子 CNC 上切割的，木头是用他的锯木厂切割成大致尺寸的。他承认边材在大约 20 年后会有点腐烂，但由于柱子的核心是心材，从结构上来说，它们会持续很长时间。刨到合适的尺寸后，切孔和凹槽是一个复杂的过程。这个结构一旦建立起来就相当高，我们喜欢看它组合在一起。这里最令人印象深刻的是这个结构是由一个人组装的。所有的椽子都被切割并用手工凿成合适的尺寸，然后吊装到位。电池板最终没有按时交付，他不得不改用新电池板，而不是他期待的旧电池板。

之前，[克里斯]已经展示了他令人印象深刻的水力发电装置。我们期待着看到新的太阳能电池阵列为[克里斯]未来计划的所有项目供电。休息后的视频。
