# 冷管吸收热量

> 原文:[https://hackaday.com/2020/08/27/cold-tube-draws-the-heat/](https://hackaday.com/2020/08/27/cold-tube-draws-the-heat/)

如果你住在热带附近，空调不是奢侈品，而是必需品。然而，问题是潮湿的气候会导致传统空调在对空气进行除湿时消耗更多的能量，而不仅仅是冷却空气，这就增加了运行空调所需的能量。早在 1963 年，就有人提议创建一个不会产生冷凝的冷却系统，并将其与不同的除湿方法结合起来。新加坡的研究人员现在已经创造了这样一个系统。它使用一种可透过红外辐射的薄膜，但防止冷却装置周围出现冷凝。

在下面的视频中，你可以在新加坡的一个展馆里看到这个装置的视频。冷水流经通过热辐射的薄膜后面的管子。由于管子不暴露在周围大气中，冷凝是最小的。但是热量从管子较热的区域辐射到较冷的区域。

干燥剂被动地干燥空气。有用吗？报纸提到在一月份进行测试，这对来自寒冷地区的人来说听起来很奇怪，但新加坡离赤道如此之近，以至于即使在一月份也不是真正的冬天。快速搜索显示，一月份的平均温度为 87 华氏度(约 30.5 摄氏度)。人们认为该地区主观上凉爽舒适。

当然，我们想知道，在接触表面的气流或振动可能成为问题的应用中，例如在显微镜下，是否可以用这样的方案来冷却电子设备。如果你喜欢的话，你可以用零头组装一台经典的空调。[沼泽冷却器](https://hackaday.com/2014/08/08/a-do-it-yourself-air-conditioner-with-evaporative-cooling-5-gallon-bucket/)运转良好，但不适应潮湿环境。
