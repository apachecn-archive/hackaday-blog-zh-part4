# 一个简单的相机滑块构建

> 原文:[https://hack aday . com/2019/05/08/an-easy-camera-slider-build/](https://hackaday.com/2019/05/08/an-easy-camera-slider-build/)

随着每一代智能手机摄像头的改进，制作高质量的视频内容变得越来越容易。这意味着要脱颖而出还需要一点时间，所以在你的电影摄影中发挥创造力是值得的。滑块是拍摄不同照片的好方法，[你也可以非常便宜地制作一个](https://www.youtube.com/watch?v=fSsezE0jGOE) (Youtube 链接，嵌入在下面)。

为了平稳运动，[Nikodem Bartnik]使用铝挤压材料制作轨道，并设计了一些适合的滚柱轴承轮。车轮被内置在一个 3D 打印的车厢中，车厢还装有一个球形夹紧相机支架。这些都是用一些内六角螺钉和 3D 打印支架包裹起来的。

尺寸精度是滑块顺利运行的关键，所以如果你要尝试这一个，你会想要有你的打印机设置好。[Nikodem]展示了滑块能够承受中等大小的单反相机的重量，但如果你想拍摄长焦，你可能会想要更大的东西。你也可以考虑用机动钻机来代替。休息后的视频。
