# 工作中的乐高太空电脑是一个模子刻出来的

> 原文：<https://hackaday.com/2021/08/04/working-lego-space-computers-are-a-chip-off-the-old-block/>

我们都有自己最喜欢的经典乐高积木，如果一台或多台不同的太空电脑出现在几乎每个人的清单上，也不会感到惊讶。[dyoramic]非常喜欢它们，以至于[他们构建了两个不同的工作版本，做不同的事情](https://www.reddit.com/r/3Dprinting/comments/oufcov/i_wanted_a_working_lego_space_computer_but/)。

第一个大约是原砖的六倍大。在 3D 打印的外壳内是一个 ESP32 和一个 1.5 英寸的有机发光二极管显示器。[dyoramic]将最上面的六个按钮连接起来作为输入，其余的只是为了外观。屏幕默认为绿色背景上的经典白色十字，看起来很合法。但是开始按按钮，你会发现其他模式——一个是十字成为雷达屏幕，另一个是计算机吐出空间事实，还有一个落砖游戏，最后是时间和日期屏幕。

第二个乐高太空电脑模型更大——两者都是根据屏幕尺寸设计的。它有一个 Raspi 4，显示一个仪表盘，上面有天气、时间、日期、最新的 xkcd 和一些加密货币的价格。[dyoramic]正在开发一个更大的版本，它将使用 720 x 720 的屏幕和一些棕色按键开关作为输入。我们等不及要看那部电影了！现在，休息之后看看前两个的构建和演示。

你不能用乐高做什么？感觉就像我们已经看到了一切，从[相机](https://hackaday.com/2021/04/19/large-format-lego-camera-is-a-bit-near-sighted/)到[显微镜](https://hackaday.com/2021/06/28/lego-microscope-aims-to-discover-future-scientists/)到[无级变速器](https://hackaday.com/2021/02/05/building-a-continuously-variable-transmission-with-lego/)。你不想开着它绕着街区转吗？
