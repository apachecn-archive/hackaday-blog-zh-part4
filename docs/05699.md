# USB 加密狗维修艺术

> 原文：<https://hackaday.com/2020/02/20/the-art-of-usb-dongle-repair/>

面对损坏的无线设备 USB 加密狗，我们大多数人可能会扔掉零件并订购一个替代品，毕竟这些东西的尺寸很小，可能意味着很难甚至不可能修复，对吗？嗯，[Equalizor]把它作为一个挑战，并利用这个机会[练习他的微观焊接技能](https://youtu.be/ZLgPyO7hS3I)只是为了好玩。

有问题的无线适配器来自他的一个客户，他在将它插入笔记本电脑时不小心将其弯曲，在任何情况下都拒绝被计算机识别。在滑出 USB 插头的金属外壳并折断塑料外壳后，[Equalizor]发现轻微弯曲的外部隐藏了一个深深开裂的 PCB。然后，通过检查切断的迹线和抬起的元件，只需将焊料回流几次，就可以让电路板恢复完整。一旦电子狗被确认工作正常，一个新的 3D 外壳就被打印出来，取代了原来必须被折断的外壳。

对一些人来说，这似乎并不奇怪，但这个视频是一个很好的例子，表明在如此小的范围内维修精致的电子产品是可行的，并可以减少我们不断倾倒的电子垃圾的数量。仅仅因为一些电子产品看起来令人生畏地精致或[超越了抢救](https://hackaday.com/2018/07/15/incredible-atari-800xl-case-restoration/)，这并不总是意味着隧道尽头没有[之光](https://hackaday.com/2019/07/09/liquid-damaged-macbook-saved-with-a-keen-eye/)。休息之后，您可以看到在这个小电子狗上执行的工作。
