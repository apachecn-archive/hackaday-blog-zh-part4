# 3D 打印扬声器，吸取了许多经验教训

> 原文:[https://hack aday . com/2020/04/18/3d-printed-speakers-with-many-lessons-learned/](https://hackaday.com/2020/04/18/3d-printed-speakers-with-many-lessons-learned/)

尽管我们都希望我们的项目能够完美无缺，但从一个令人沮丧的项目中吸取的教训有时会比项目本身更有价值。[Thomas Sanladerer]在尝试为 5.1 环绕声系统构建五个[卫星扬声器](https://www.youtube.com/watch?v=stFMvPXvraY)时发现了这种情况，并幸运地与我们分享了整个过程的所有混乱荣耀。

[Thomas]想要比简单的矩形盒子更有吸引力的东西，所以他选定了一个非常好的弯曲设计，没有几个平面，也没有尖角，用 PLA 3D 打印。每个内部都有一个负担得起的宽带扬声器驱动器和高音扬声器，以及一个分频电路来改善音质和保护驱动器。驱动器的制造商 Visatron 提供了非常好的扬声器模拟软件来选择合适的驱动器和设计分频电路。每个扬声器的正面由一个 3D 打印的框架组成，上面覆盖着裁剪好的 t 恤材料。这些封面用磁铁贴在主体上，看起来很像。

印刷后，[Thomas]将所有部件浸泡在水中以清洗 PVA 支撑结构，但发现外表面不防水且大量水渗入部件时为时已晚。为了让它们变干，他把它们放在太阳下晒了一会儿，结果有些部分翘曲了，所以他不得不重新印刷。主体被打印成两部分，然后粘在一起。这需要大量的打磨来消除胶缝，并且需要多次油漆和打磨来消除层线。当组装不同的部件时，他发现许多部件不能装配在一起，他怀疑这是由他使用的 delta-bot 打印机的不正确校准引起的。

最终，构建花了将近两年时间，因为[托马斯]需要在所有的挫折中休息，最终只使用了一个扬声器。我们很高兴他分享了项目中混乱的部分，这将有望为项目中的其他人省去一点麻烦。

聆听高质量的音频设置总是一种享受，我们已经涵盖了来自音响发烧友的几个项目，包括价格实惠的 [DML 扬声器](https://hackaday.com/2019/10/26/building-the-worlds-best-dml-speakers-for-under-115/)和 [3D 打印扬声器驱动器](https://hackaday.com/2019/09/07/putting-3d-printed-speaker-drivers-to-the-test/)。
