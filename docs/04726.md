# DIY 盒式磁带吉他延迟

> 原文：<https://hackaday.com/2019/11/03/diy-cassette-tape-guitar-delay/>

现在的数字延迟踏板非常好，甚至便宜的踏板在模仿旧的模拟延迟效果的声音方面也做得很好。这很好，因为最初的延迟效果可以让你花一大笔钱。不过，如果你喜欢 DIY 电子产品，模拟延迟效果仍然可以在不花太多钱的情况下制作出来，作为一个例子，[Matsound]已经用一个旧的卡带和普通的盒式磁带制作了一个磁带延迟[。](https://www.youtube.com/watch?v=ABU-psBnmxk)

构建的核心是一个便携式 3 头盒式录音机，在这种情况下，一个马兰茨 PMD430。该电路已经存在了一段时间，最初是在 90 年代的一期 Stompboxology 中发现的。基本思想是，对于三头录音机(擦除、录音、播放)，录音头和播放头之间的距离会产生延迟，通过减慢录音机的电机来增加这种延迟。你把录音机的输出和你输入的干信号结合起来，中提琴，磁带延迟。

[Matsound]增加了一个很酷的功能，你可以用控制电压来控制电机的速度，所以如果你把它连接到键盘上，从不同的按键产生不同的电压，你会得到奇怪的，太空的效果。该视频概述了这些功能，视频描述了构建过程的一些细节。

一个很好的建立到一个很好的情况下，一个很大的影响！也许你不会把它拿出来和你一起演奏，但它听起来确实很不错！其他延迟踏板之前在网站上已经提到过，像[这个数字延迟踏板](https://hackaday.com/2019/04/10/this-is-the-delay-pedal-you-can-build-yourself/)和这里的[另一个关于盒式磁带延迟的](https://hackaday.com/2015/10/07/tape-loop-echo-made-with-an-actual-tape-loop/)。
