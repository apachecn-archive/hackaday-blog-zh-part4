# 我们期待已久的 Vectrex 投影仪

> 原文:[https://hack aday . com/2019/01/23/the-vectrex-projector-we ve-be-waiting-for/](https://hackaday.com/2019/01/23/the-vectrex-projector-weve-been-waiting-for/)

与大多数老式控制台不同，Vectrex 的独特之处在于它有一个基于矢量的显示器。这使它与大多数同时代的电视有着非常不同的外观，并需要内置显示器，因为普通电视不是为接收矢量信号而设计的。不局限于普通屏幕大小的人， [[Arcade Jason]决定 Vectrex 需要一个投影升级，并准确地建造了这个](https://www.youtube.com/watch?v=68aHmyKWrpw+)。

这个建筑依靠的是[杰森]从一台旧背投电视中回收的镜头。这些装置使用带有大透镜的阴极射线管将图像投射到屏幕上。这正是这里正在发生的事情，矢量显示器取代了原来电视中使用的 CRT 显示器。这里使用的矢量显示器本身是一台小型黑白电视机的显像管,[Jason]将其修改为使用 Vectrex yoke，使其能够进行矢量操作。

通过一些修改和仔细组装，[Jason]的 Vectrex 控制台获得了一个墙壁大小的显示器。这是用一些漂亮的发光向量演示来演示的，伴随着适当的哔哔声音乐，这是当时的风格。Cantina 乐队是一大亮点。
