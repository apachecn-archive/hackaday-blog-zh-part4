# 独特的乐器难以描述

> 原文：<https://hackaday.com/2020/04/11/unique-musical-instrument-defies-description/>

自从我们的第一个祖先发现用棍子敲打中空的木头会发出欢快的声音，我们人类就一直在寻找新的不寻常的方法来制作音乐。我们还没有接近挖掘出新乐器的潜力，但话说回来，我们并不是每天都会遇到一种独特的乐器和一种新的声音，就像这个拨弦机器人竖琴一样。

以建造者[弗兰克·皮西克]的朋友[格雷戈尔]的名字命名为“格雷格的竖琴”，这种三弦乐器几乎无法归类。它有点像竖琴，但又不同，有点像电吉他，但又不完全是。每根钢弦都有三种不同的演奏方式:弗兰克称之为“踢击”，即螺线管敲击琴弦；“eBow”线圈刺激器；还有一个带塑料拨片的小马达来拨动琴弦。每个都以琴弦的基频产生独特的声音，而每个琴弦周围的伺服控制环充当机器人指板来改变音符。声音由压电传感器接收，一切由一对 Nanos 和一个 Teensy 控制，后者负责 MIDI 任务。

看看下面的视频，看看你是否觉得这声音既熟悉又全新。我们一直在展示独特的乐器构造，从[不太小提琴](https://hackaday.com/2017/02/13/its-a-synthesizer-its-a-violin-its-a-modulin/)到[自动演奏的卡林巴斯](https://hackaday.com/2020/02/25/self-playing-kalimba-v2-thanks-to-readers-like-you/)到[the remincello](https://hackaday.com/2013/04/03/building-a-theremincello/)，但我们仍然觉得这一个很迷人。
