# 优化屏幕显示心跳的时间

> 原文：<https://hackaday.com/2018/08/31/optimizing-screen-time-to-heart-beats/>

现在孩子们在屏幕前花的时间太多了。他们也不会离开我的草坪，而今天的音乐只是一堆静电。他们也不尊重长辈。虽然现在的孩子很糟糕，但我们可以解决第一个问题——整天坐在屏幕前。为了他的 Hackaday 奖参赛作品，[Donovan]创造了一种设备，它可以优化屏幕时间以减少感官超载。这是 Optimote，一个遥控器和生物反馈的结合。

Optimote 背后的想法实际上是为了在观看屏幕上的东西时减少刺激。对许多人来说，包括自闭症患者，看电视或 YouTube 视频经常会导致感觉超负荷。这种状态下你无法放松，无法学习，当然也无法从眼前发光的长方形中获得任何娱乐价值。

Optimote 使用了一个脉冲传感器、一个 Arduino、一条似乎在任何其他类似的可穿戴设备中都没有的令人难以置信的可分离电缆，以及一个与 VLC 交互的软件堆栈。在高脉冲频率期间，视频会跳到低强度镜头。有一个“平静”模式，让媒体音量和节奏与心率同步。“惊悚”模式播放一个诡异的场景，以*大白鲨*为主题。

到目前为止，原型是成功的，[Donovan]期待着大规模的用户体验测试，以确定这项技术可以变得多么有效和令人愉快。
