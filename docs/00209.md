# 令人难以置信的 3D 打印看守气枪手枪

> 原文:[https://hack aday . com/2018/07/30/incredible-3d-printed-over watch-airsoft-pistol/](https://hackaday.com/2018/07/30/incredible-3d-printed-overwatch-airsoft-pistol/)

如果你需要证据来证明游戏玩家是所有粉丝中最专注的个人，那么看看这个非常受欢迎的“ *Overwatch”中“脉冲手枪”的奇妙 3D 打印再现就知道了。*由【危险博士】的人建造，这个复制品不仅仅看起来像零件，它还是一把功能齐全的气枪。在休息后的详细建造视频中，[长达一年的枪的设计和建造被分解，供您观看](https://www.youtube.com/watch?v=HAKMNZ6TJX0)。

因为最终目标是做出看起来像是来自游戏本身的东西，所以投入了大量时间来确保外部内容忠实于数字版本，同时仍然能够包含他们需要塞进去的所有硬件。这是一把全自动枪，所以它需要一个电池和马达，以及一种不需要伸出过时杂志的发射机制。

他们结合了现成的发射机制和大容量弹匣，但需要大量定制设计的零件来配合一切。该杂志有一个发条装置来推进 BBs，这需要用户手动启动，但这被一个电动马达取代，使事情变得更具未来感。除了枪体上的所有 led 外，还有一个内部的紫外线 SMD LEDs 阵列，当它们穿过弹匣时，可以为黑暗中发光的“tracer”BBs 充电。在光线较暗的情况下，这让枪里的子弹有一种激光效果。

在之前，我们已经在游戏中见过 [3D 打印枪，但很少关注细节和工程。老实说，这种](https://hackaday.com/2018/02/24/fully-3d-printed-nerf-thirst-zapper/)[甚至让一些*真正的* 3D 打印枪也物有所值。](https://hackaday.com/2015/11/21/building-a-better-3d-printed-gun/)
