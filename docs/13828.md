# 危险的令人愉快的泰国自制火箭

> 原文：<https://hackaday.com/2022/06/04/the-dangerously-delightful-homemade-rockets-of-thailand/>

每隔一段时间，我们在 Hackaday 会偶然发现一些东西，这些东西并不完全符合我们所展示的所有其他令人惊叹的黑客，但似乎仍然是我们亲爱的读者需要尽快看到的东西。这个泰国自制火箭的视频就是其中之一。

它来自我们的朋友 Leo Fernekes，他记录了一种业余火箭技术，这种技术让我们年轻时代的 Estes 火箭看起来很平淡。看起来比描述起来容易得多，但是简单总结一下，火箭是直径上有一根钢管的竹环。管子里装满了自制的火药，并配有能产生推力和旋转的喷嘴。当火把接触到非常粗略的引爆器点燃时，火箭开始旋转，最终离开发射台，在一股扭曲的灰色烟柱上旋入天空。

直径有三四米，这些都不是小型车辆。火箭工程师们设计了一个聪明的回收系统，当火箭发动机最终融化一些塑料带时，该系统会部署一个降落伞，而不是让一根钢管从数百米的高度垂直降落到地面。用香蕉树的树皮做隔热板来保护降落伞简单而有效；你可以用这种方式来描述整个企业。[狮子座]有另一种方式来描述它:“危险的疏忽的疯狂”，当然，带着所有应有的尊重和喜爱。这看起来也是一件大事——空气中显然充满了竞争精神，更不用说火药的臭鸡蛋味了。

你应该在家试试这个吗？可能不会——我们可以想出很多理由来解释为什么这是个坏主意。尽管如此，观看起来还是很令人惊奇，看到这些拼凑起来的火箭设法获得如此高的高度真是令人惊奇。向[利奥]致敬，感谢他为我们找到了这个。
