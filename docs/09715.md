# 触觉黑客与纳米孔技术

> 原文:[https://hack aday . com/2021/04/05/haptics-hack-chat-with-nanopt-technology/](https://hackaday.com/2021/04/05/haptics-hack-chat-with-nanoport-technology/)

加入我们太平洋时间 4 月 7 日星期三中午的 [Haptics Hack Chat](https://hackaday.io/event/178149-haptics-hack-chat) 与 [Tim Szeto](https://hackaday.io/t1m.co) 和 [Kyle Skippon](https://hackaday.io/NanoKyle) ！

在我们所有的感觉中，触觉可能是最不受重视的。我们理解并接受伴随视觉或听觉丧失的悲剧，嗅觉和味觉对生活质量的影响可能是严重的。但出于某种原因，我们不会再考虑我们的触觉，这确实很奇怪，因为我们实际上布满了触摸传感器。这有点令人遗憾，因为触觉可以揭示我们周围世界的很多东西，我们的情感福祉与触觉紧密相连，以至于那些在婴儿期被剥夺了触觉的人可能会留下终身的疤痕。

触觉是触觉反馈的技术，它寻求利用人类对触觉体验的需求来丰富与技术世界打交道的体验。触觉反馈设备现在无处不在，并且已经远远超出了自寻呼机成为地位象征的日子以来所使用的简单的不平衡电机。为了帮助我们理清触觉世界的新情况，来自[纳米孔技术](https://nanoport.io/haptics/)的 Tim 和 Kyle 将在 Hack Chat 上停下来。Nanoport 是一家站在触觉学前沿的公司，所以他们会有大量关于什么是触觉学的细节，这个领域的发展方向，以及你如何开始考虑让触觉成为你项目的一部分。
