# 一种完全不同的图形化编程

> 原文:[https://hack aday . com/2018/09/13/a-whole-other-kind-of-graphic-programming/](https://hackaday.com/2018/09/13/a-whole-other-kind-of-graphical-programming/)

Java 并不是每个人都喜欢的。尽管有这么多样板文件和开销，但最好还是用一个合适的 IDE 来处理所有的事情。然而，如果你学习一门新的语言，你真的不想设置一个笨重复杂的 IDE。如果你能使用一个你很可能已经熟悉的简单、标准的 Windows 程序就好了。这个愿望促使[RubbaBoy]创建了 [MSPaintIDE，这是一个 Java 开发环境](https://ms-paint-i.de/),让你可以在其中编写代码——是的——MS Paint。

如果你现在认为你将最终用 MS Paint 的文本工具编写你的程序，并用它创建一个常规的图像文件，那么你是对的。设置完成后，MSPaintIDE 会将所有 PNG 源文件编译成一个普通的 Java JAR 文件。是的，它有语法突出和一个黑暗的主题。[RubbaBoy]使用定制的 OCR 将图像内容转换为文本文件，并将其打包到一个只需点击几个按钮的环境中，包括 git 集成。你可以在休息后的视频中看到它的演示，并在 GitHub 上找到[源代码。](https://github.com/RubbaBoy/MSPaintIDE)

考虑到这个项目半开玩笑的性质，人们不得不真正钦佩[RubbaBoy]走了多远。玩笑归玩笑，如果你对 OCR 感兴趣，这可能是足够简单的开始。或者你可以用一些文本到语音的功能扩展它[。
