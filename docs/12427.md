# 拥有它的年份

> 原文:[https://hackaday.com/2022/01/08/the-year-of-owning-it/](https://hackaday.com/2022/01/08/the-year-of-owning-it/)

在播客的[回顾中，汤姆·纳尔迪和我讨论了我们认为的](https://hackaday.com/?p=514793)[2021 年的单一总体趋势](https://hackaday.com/2022/01/03/2021-as-the-hardware-world-turns/)，我们提出了许多不同的话题:维修权的胜利，越来越反乌托邦的服务合同，cyberdecks 的繁荣，甚至更普遍的逆向工程风格的黑客攻击。然后我们意识到:它们都是同一只野兽的不同面孔——人们只是想*拥有*他们拥有的设备。

就像哲基尔博士和海德先生一样，我们现代的网络连接——任何事物都有两面性。一方面，我们从网上获得了如此多的附加功能。但另一方面，如果你的车总是联网的，它给了[丰田一种手段，让你支付每月费用来使用汽车离岸价格](https://hackaday.com/2021/12/30/new-cars-will-nickle-and-dime-you-its-automotive-as-a-service/)，如果你必须使用 Cricut 的免费在线服务来上传设计给切割机，[他们可以突然决定开始向你收费](https://hackaday.com/2021/03/15/cricut-decides-to-charge-rent-for-people-to-use-the-cutting-machines-they-already-own/)。它让三星不仅可以监视你目前在智能电视上看的任何东西，而且如果他们愿意的话[还可以屏蔽它。越来越多的情况是，我们实际上并不拥有(从控制的意义上来说)我们所拥有的(从购买的意义上来说)设备。](https://hackaday.com/2021/11/29/samsung-bricks-smart-tvs/)

我们不必忍受它。一方面，消费者的抗议让 Cricut 收回了他们的计划，可能对丰田也是如此。通过谈论我们的不满，并让相关公司知道我们的感受，我们可以共同取得很多成就——当然也包括我们的钱包。但是作为黑客和全能技术人员，我们可以做得更多。当某个东西因为糟糕的服务而损坏时，我们通常可以用固件或通过建立我们自己的服务版本来修复它。我们可以干掉他们。

但是，在过去几年里，网络平台和极端的 DIY 运动不仅仅是防止锁定或解放硬件。也有真正拥有某物的骄傲，因为它是你做的。不只是因为你买了它而拥有它，或者因为你控制它而拥有它，而是因为你理解它，因为你创造了它。

无论你以何种方式拥有自己的房子，我认为这都是 2021 年的单一总体趋势——无论是积极主动的一面，还是消极被动的一面。谈论它，逆向工程它，或自己建造它，2021 年是拥有它的一年。
