# 即使有空调，防化服也不是很酷

> 原文：<https://hackaday.com/2021/06/21/even-with-ac-hazmat-suit-isnt-really-cool/>

我们承认，由防化服制成的空调人造宇航服的凉爽因素很大程度上取决于你穿在哪里。在你的下一次象棋俱乐部会议上，也许会成功。相亲的话，大概不会。[Saveitforparts]显然不介意，非常温暖的天气加上捐赠的过期危险品套装，[激发了他的想象力，正如你在下面的视频中看到的](https://www.youtube.com/watch?v=QU9clRAxDug)。

一个电池组、一个鼓风机和一个装满冷冻水瓶的盒子就完成了这一整套。无线温度传感器可以显示外部温度，也可以显示宇航服不同部位的内部温度。有用吗？我们猜测肯定是，但是风扇的轰鸣声震耳欲聋，我们对冷冻水冷却系统产生了怀疑。另一方面，如果你正在拍摄一部低成本的科幻惊悚片，这可能正是你想要的。

甚至[Saveitforparts]也承认这并不实际，正如我们所怀疑的，当冷凝水开始顺着他的腿流下来时，他决定离开。原来宇航员和坦克驾驶员使用由小管流水制成的内衣来保持凉爽。

这个项目让我们想起了不久前看到的正压服。更不要说那辆[竟然全身](https://hackaday.com/2020/05/09/halloween-costume-turned-positive-pressure-suit/)了。
