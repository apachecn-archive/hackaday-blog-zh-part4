# 小垃圾桶排斥垃圾熊猫，中型猫

> 原文：<https://hackaday.com/2020/10/09/tiny-trash-can-repels-trash-pandas-medium-sized-cats/>

你厌倦了猫和其他野生动物在你的户外花盆里解手吗？好像偶然的疏忽不会很快杀死他们。[TecnoProfesor]有一个解决方案，它甚至不涉及微控制器。它能探测到正在靠近的动物，然后用恼人的声音和几个亮绿色的发光二极管来驱赶它们。

多亏了几个模块，电路真的很简单。有一个 PIR 来检测动物，一个蜂鸣器，和一个基于 555 的脉冲发生器通过蜂鸣器播放音调。这个电路可以在一对给电池充电的 6V 太阳能电池板上全天候运行。我们特别喜欢桌面垃圾桶外壳，尽管我们想知道这个系统的防水性如何。休息之后，请观看简短的演示。

对于所有对这里的 555 实现满意的人来说，[你的奖励是这个巨大的功能 555](https://hackaday.com/2020/07/15/a-shrine-for-all-the-555-lovers/) 。如果你是 555 的反对者，你会如何做得更好？
