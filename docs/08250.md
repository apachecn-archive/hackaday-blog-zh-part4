# NASA 黄金时代任务控制“大板”背后的一瞥

> 原文：<https://hackaday.com/2020/10/29/a-look-behind-the-big-boards-at-mission-control-in-the-golden-age-of-nasa/>

公认的太空迷和全能的复古技术大师[Fran Blanche]刚刚超越自己，全面了解了美国宇航局如何运行任务控制“大板”，为阿波罗和未来 20 年的载人航天提供飞行数据。

我们不得不承认，[弗兰]这次让我们大吃一惊。我们一直以为，在一排排薄荷绿色的控制台前显示的图形和图表，以及他们穿着紧身领带的工程师，是使用 eidophor 投影仪的视频投影。可以肯定的是，不久前[Jenny]对的技术进行了描述的 eidophor 被用在其中一个屏幕上，以向任务控制中心提供视频，无论是从月球直播还是从发射和回收操作的报道中。但是，即使粗略地看一眼“The Pit”前面的其他屏幕，也会显示出远超过 20 世纪 60 年代视频所能达到的清晰明了的投影。

取而代之的是，使用一个完全的机电系统将图表投影到巨大的屏幕后面。玻璃和金属模板被用来投射图标、地图和网格，一层一层地构建图像。每一层的颜色通过使用二向色滤光器获得，并且图标被物理地移动以实现动画。图表和绘图是以蚀刻草图的方式创建的，用伺服控制的触针穿过由一薄层金属制成的不透明载玻片。整个事情非常复杂，完全是乱糟糟的，是围绕技术极限的工程的一个很好的例子。

向[弗兰]致敬，因为他挖掘了这个被遗忘的太空竞赛技术。相比之下，看到这样的东西会让今天的任务控制中心看起来无聊透顶。
