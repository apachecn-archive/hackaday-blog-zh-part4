# 定制车刀切割复杂的油槽

> 原文:[https://hack aday . com/2022/09/08/machining-zig-zag-figure-8-oil-grooves/](https://hackaday.com/2022/09/08/machining-zig-zag-figure-8-oil-grooves/)

油槽用于润滑轴承内部，可以有多种形式——从滴几滴油的单个孔到将油分布在整个内表面的螺旋模式。理想的排列是一个类似示波器李沙育图形的环形 8 字形图案，但切割这些图案是一场噩梦。也就是说，除非你有合适的工具。

我们认为[机械技工]一定需要切割很多，因为他们[花了相当多的时间来完善这个定制的车床附件](https://www.youtube.com/watch?v=9EPfRd79FPM)来自动化这个过程。通过各种巧妙的连接和原位焊接在一起的连杆曲柄，该装置将车床的旋转运动转化为往复运动，将刀具移入和移出轴承。顺便说一句，这个小工具的商业端最初是一个螺栓，后来被拧下来，并在末端钎焊了一块工具钢。

[![](../Images/574819f8775d7d7f55bb225296b1abc4.png)](https://hackaday.com/wp-content/uploads/2022/09/oilgroove_detail.jpg) 稍加调整，看来这种装置也可以用来在工件外面雕刻花纹。但即使这是它唯一能成功的把戏，我们仍然印象深刻。对于非常专业的机械车间操作来说，这是一个聪明的技巧，大多数人会认为你需要一个四轴 CNC 来完成。

车床初看起来像是相当单一用途的机器，我们总是很高兴地发现用它们做的奇怪而奇妙的事情，像这个[看似不可能车削的工件](https://hackaday.com/2022/07/16/turning-irregular-shapes/)，还有这个[电火花线切割机床和车床的组合](https://hackaday.com/2022/07/20/cut-just-about-anything-with-this-combination-lathe-and-wire-edm/)。
