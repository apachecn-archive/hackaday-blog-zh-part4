# 一个简单的炉子，为豆子而建

> 原文:[https://hack aday . com/2022/10/14/a-simple-stove-build-for-beans/](https://hackaday.com/2022/10/14/a-simple-stove-built-for-beans/)

在大多数情况下，围坐在篝火或壁炉旁是一种审美愉悦的体验，甚至可以提供一些温暖。但是，除非你有一个现代的木材燃烧设备，否则生物质中的大部分能量很可能会以未燃烧的蒸汽形式逸出。令人惊讶的是，解决这个问题几乎就像在商店买一罐豆子一样容易，结果是一个非常高效的炉子，可以在必要时用于加热。

[Robert]正在演示这个气化炉，不是用豆子，而是用一罐豌豆和一大罐土豆。在每个罐头上按特定的模式钻不同的孔，然后将较小的豌豆罐头放入较大的马铃薯罐头中。一旦着火，这些孔允许空气流动，以某种方式捕获燃料中逸出的未燃烧的蒸汽，并在它们流过这个装置时燃烧它们。不需要移动部件；这都是由火的热量产生的自然气流驱动的。

像这样建造的结果不仅是炉子可以提取更高百分比的可用燃料，而且对于给定的热量需要更少的燃料，产生更清洁、更少烟的火。[罗伯特]还增加了一个屏幕罩，允许它更多地用作热源，但类似的建筑也可以有效地用于烹饪。
