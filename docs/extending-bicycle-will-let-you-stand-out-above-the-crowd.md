# 加长自行车会让你鹤立鸡群

> 原文：<https://hackaday.com/2021/09/02/extending-bicycle-will-let-you-stand-out-above-the-crowd/>

一些自行车主要是为了实用性而建造的，而另一些则更注重新颖的外观。[Make It Extreme]的最新项目[加长自行车](https://www.youtube.com/watch?v=9TG_HA7mSLo)，正好属于后一类。

这款电动自行车围绕四个定制的气动活塞建造，只需轻轻一按开关，就可以将骑车人举到空中约一米。前一对形成自行车的叉，而后一对安装在车架和摆臂之间。一个小型车载压缩机用于填充一对经过改造的灭火器，灭火器通过安装在车把上的气动阀向活塞供气。轮子和刹车是从一辆旧摩托车上拆下来的。由于曲柄组和后轮之间的长度从不改变，因此无需随着行驶高度的变化而与链条张紧器进行斗争。

虽然我们不喜欢从那个高度面对植物，但它看起来肯定是一个有趣的乘坐和谈话的开始。许多[让它变得极端]的项目都是如此，比如[可驾驶坦克履带](https://hackaday.com/2019/09/11/rideable-tank-tread-its-a-monotrack-motorcycle-that-begs-you-to-stop-very-slowly/)和[独轮摩托车](https://hackaday.com/2019/01/16/this-monowheel-is-bright-orange-and-we-want-one/)。
