# 勇敢的卡林巴扮演自己

> 原文：<https://hackaday.com/2020/01/11/plucky-kalimba-plays-itself/>

[Gurpreet]爱上了《阿凡达》系列中和平、漂浮的主题，并买了一把卡林巴琴，这样他就可以通过指尖听到它的共鸣。他很快意识到，尽管弹奏卡林巴琴很好，但如果它自己弹奏的话会更酷。然后，他可以放松下来，享受音乐，而不会磨损他的拇指。

在对伺服喇叭的拔齿延伸部分进行了一些印刷实验后，[Gurpreet]决定通过将伺服安装在印刷底座上来开始设计过程。伺服系统通过它们的安装片固定到位，并用热胶固定。我们认为这是一个不错的选择——它很实用，看起来很酷，就像一个散热器。

[Gurpreet]的未来计划包括更多的伺服系统来拨动其余的齿，并弄清楚如何给它提供 MIDI 并实时播放它。对于休息后的演示，[Gurpreet]说他从后面翻领麦克风，大胆地切断伺服噪音，但最终想知道如何直接让它们安静下来。他将尝试陆兵齿轮和泡沫隔音罩，但如果你有任何其他想法，让他知道下面。

我们在这里没有看到太多的 kalimba 项目，但这里有一个与一个十几岁的 looper 有关。
