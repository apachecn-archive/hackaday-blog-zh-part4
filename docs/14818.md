# Retrotechtacular:最初的机器人手臂

> 原文：<https://hackaday.com/2022/09/19/retrotechtacular-the-original-robot-arm/>

你知道乔治·德沃尔这个名字吗？大概不会。1961 年，他获得了“程序化物品转移”的专利。“我们称他的发明为第一个机器人手臂，它的名字叫 Unimate。不像一些发明家，这不是一些未实现的梦想。[德沃尔的]手臂去了新泽西的一家通用汽车工厂工作。这个 4000 磅重的手臂花费了 25000 美元，并且堆叠了炽热的金属部件。有了管道和液压系统，我们可以想象要让它继续工作需要做很多工作。另一方面，大约 450 个兵种最终去了某个地方工作。

尤尼马特成为了名人，至少在一部新闻纪录片和约翰尼·卡森秀中出现过。不出所料，新闻片中的机器人正在倒饮料。

这个机器人实际上可以追溯到 1954 年，尽管专利直到 1961 年才授予。到 1969 年，通用汽车甚至使用 Unimate 进行点焊，每小时可以生产 110 辆汽车。当时，这据称是任何其他汽车厂所能达到的速度的两倍。

新闻短片提到 Unimate 有内存，从我们所知，它确实使用了某种鼓形内存系统。我们还瞥见了 Unimate 制造水槽，甚至(脸红)繁殖。

 [https://www.youtube.com/embed/VdolSBpyCaU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/VdolSBpyCaU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



如果你看一点那段视频，你会了解到我们应该在 2000 年的时候在家里有一个这样的东西。哼。我们只希望它问的是，“你想吃点吐司吗？”不过，到最后，还是会涉及到酒精。

西屋电气早在 1984 年就收购了 Unimate。为什么不呢？他们总是喜欢机器人。就此而言，我们仍然没有来自威斯汀豪斯或任何其他公司的机器人烤面包机。但是他们并不是第一个对未来进行预测的 T2 人。
