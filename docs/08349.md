# DIY 高压 Kirlian 摄影钻机

> 原文：<https://hackaday.com/2020/11/15/diy-high-voltage-kirlian-photography-rig/>

玩高压电是一件有趣的事情，尽管很危险。[Mirko]是这一领域的爱好者，[他建造了一个高压 Kirlian 摄影设备，能够拍摄出一些令人惊叹的图像](https://hackaday.io/project/175773-diy-a-high-voltage-kirlian-photography-device)。

设备本身的结构是基本的。高频高压电连接到放置在玻璃板一侧的金属物体上。在另一侧，汇集盐水，并连接到第二电极。这就形成了一个透明的窗口，通过这个窗口可以用肉眼或数码相机观察放电。历史上，[这是用照相胶片代替透明窗](https://en.wikipedia.org/wiki/Kirlian_photography)完成的，但原理是一样的。

克里安摄影在超自然和另类医学领域广受欢迎，但实际的科学应用却微乎其微。相反，这是一种探索高电压的有趣方式，可以产生一些漂亮的结果！如果你发现自己成为了高压的粉丝，[你可能也会考虑自己的特斯拉线圈。休息后的视频。](https://hackaday.com/2020/10/20/this-mostly-transparent-tesla-coil-shows-it-all/)
