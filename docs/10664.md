# 拨号锡德是一个光荣的芯片调点唱机

> 原文:[https://hack aday . com/2021/07/08/dial-a-sid-is-a-glorious-chip tune-jukebox/](https://hackaday.com/2021/07/08/dial-a-sid-is-a-glorious-chiptune-jukebox/)

在今天这个时代，老式的旋转式电话对于他们最初的预期目的来说并不是特别有用，但是它们对于侵入新的项目来说是非常有趣的。Linus kesson 通过他的 [Dial-a-SID jukebox build](https://www.youtube.com/watch?v=W5sFOk52hTI) 做到了这一点。(视频，嵌在下面。)

该版本在电话主体内安装了一个 Raspberry Pi 3，运行 SID 芯片仿真器，并加载了[高压 SID 采集](https://hvsc.c64.org/)。内部的 Pi 向外部立体声系统输出声音，用于在聚会上播放 chiptunes。

然而，真正的聚会部分是可以拿起听筒并拨打电话来收听和选择播放列表的曲目。曲目可以通过单个代码、作曲家甚至年份来选择。在播放列表变空的情况下，默认行为可以被设置为同时保持播放随机曲目。

里面有超过 2000 个小时的 SID 音乐，Dial-a-SID 不太可能很快就没有曲子了。我们也希望看到一个类似的界面，用于连接现代流媒体服务的点唱机。如果你建造了一个，请告诉我们！或者，[考虑将你的旋转式手机连接到你的智能手机上。](https://hackaday.com/2019/09/24/finally-a-rotary-cell-phone-with-speed-dial/)休息后的视频。
