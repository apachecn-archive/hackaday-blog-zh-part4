# 几乎现代的电传打字机是无声的

> 原文：<https://hackaday.com/2021/12/08/almost-modern-teletype-is-silent/>

如果你曾经使用过真正的电传打字机，或者看过一部有新闻编辑室的电影，你就会知道一台电传打字机会发出很大的噪音，有几台会发出更大的噪音。[CuriousMarc]获得了无声的替代品，这是当时的一个真正奇迹，TI Silent 703。如果你认为拖着一个 25 磅重的手提箱到处跑是便携的，那么这个 2600 美元的机器就是便携的。在 1971 年，这绝对是一个进步。

这台机器使用热敏打印机，可能有一个内置的声音耦合器用于电话交谈。你还可以得到一个双磁带机，它的作用就像一个无声的纸带阅读机和穿孔机。

当然，热敏打印机需要热敏纸，这有其自身的问题。[Marc]不只是打开机器，而是通过 RS232 分析仪和示波器连接它，让它作为一个真正的 I/O 设备工作。他还对此大加赞赏，这在以前是不可能做到的，因为你可能会租赁它们，而不是支付现在将近 18，000 美元的总价。

令人惊讶的是，里面几乎没有什么东西，当然，与真正的电传打字机相比，几乎没有什么机械部件。如果你去掉打印机和电源，就有一个简单的 CPU 板和调制解调器板，它们看起来都不复杂。然而，最精彩的是观看它与 ASR33 的交易流量。

如果你真的想进入电传打字机，你可以[使用一个作为 Linux 控制台](https://hackaday.com/2020/04/15/logging-into-linux-with-a-1930s-teletype/)如果你有一个波特到 ASCII 解码器环。或者用一个给[发短信](https://hackaday.com/2019/04/23/texting-with-a-teletype/)。
