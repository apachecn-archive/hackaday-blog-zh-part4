# 3D 打印的圆形原型表演道具吸引了马戏团观众

> 原文：<https://hackaday.com/2022/02/02/3d-printed-circular-prototype-performance-prop-captivates-circus-spectators/>

当受到数学启发的制造商[Henry Segerman]与马戏团演员和杂技演员[Marcus Paoletti]合谋，用圆形金属物体(如 cyr 车轮和德国车轮)推进杂技工艺时，他们想到了一个迷人的概念，这个概念已经形成了[Henry]所说的[道线](https://www.youtube.com/watch?v=g8K26R_4HJc)。

类似的表演装置可以沿直线进行，也可以侧转，但陶线要灵活得多。这是因为道线并不是一个连续的圆柱体，而是由无数的圆形组成，这使得道线可以在旋转的不同点上转动和倒转。

虽然一个马戏团道具可能不是你日常的普通食物，但它值得注意，因为陶线最初是一个 3D 打印的原型，然后变成了你在休息下方的[视频中看到的金属制品。这是一个很好的例子，说明了如何将复杂的形状建模为物理产品——而不仅仅是屏幕上的 3D 模型——对全尺寸作品的整体设计和建造有所帮助。](https://www.youtube.com/watch?v=g8K26R_4HJc)

如果你想在大顶下建造一些东西，但又不太在顶上，你可能会喜欢这个混合媒体数字时钟。感谢[Keith]的精彩提示。请务必通过[提示热线](https://hackaday.com/about/)提交您的酷发现！
