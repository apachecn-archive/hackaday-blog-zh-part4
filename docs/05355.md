# a 通过 APA102 的简单地球仪 POV

> 原文:[https://hack aday . com/2020/01/10/a-simple-POV-globe-via-APA 102/](https://hackaday.com/2020/01/10/a-simple-pov-globe-via-apa102/)

POV 构建有各种形状和大小，通常依赖于 led 的高光输出和快速响应时间。考虑到这一点，[Great Scott]从货架上拿了一些 LED 条，开始制作一个 POV LED 地球仪。

作为一个旋转视点构建，有必要考虑如何为旋转元素提供能量。[Great Scott]决定采用一种简单的解决方案，即在旋转组件上放置一个 LiPo 电池，它运行 led 和 Arduino Nano，是操作的核心。所述的发光二极管是 APA102 型的，这使得它们易于寻址并且能够具有宽色域。这一切都是由一个简单的刷 DC 电机旋转，从平台底部的一个单独的供应运行。

这是一个非常黑客的建筑，用胶带和拉链连接在一起。尽管如此，它在运行时看起来很整洁，因为所有重要的硬件都隐藏在地球的中心。有一点振动问题，但[伟大的斯科特]认为这可以通过一些框架修改来解决。

我们希望看到构建运行一些更高级的操作，如地球的表示，或某种太阳时钟。如果您有兴趣了解有关 POV 显示的更多信息，[我们已经为您提供了所需的入门知识](https://hackaday.com/2019/10/29/the-basics-of-persistence-of-vision-projects/)。休息后的视频。
