# 照亮听力损失

> 原文：<https://hackaday.com/2018/07/26/shining-a-light-on-hearing-loss/>

当听觉细胞被改造成接收光线时，你看到声音，还是听到光？对于德国哥廷根大学医学中心在[托拜厄斯·莫泽]照料下的一些受过训练的沙鼠来说，这个问题没有实际意义。当管理员播放声音时，沙鼠被指示移动到笼子的不同部分，当它们修改后的细胞上的耳蜗灯被激活时，沙鼠服从它们的条件反射，去它们应该去的地方。

在链接的文章中，有一个软件可以让你模拟通过人工耳蜗听到的声音，或者你可以查看与文章无关的视频。无论哪种方式，技术的改进都是受欢迎的，根据[Tobias]:“光学刺激可能是提高频率分辨率的突破，并继续改善耳蜗植入”。第一个人工耳蜗安装于 1964 年，因此它有着悠久的历史和坚实的未来。

这不是改进人工耳蜗的唯一方法，有些不需要任何修饰细胞，但[Tobias]解释了他的推理。“我基本上选择了光遗传学这条更难的路线，因为它有一个我理解的机制，”如果这听起来不像那么多黑客伸手去拿他们熟悉的工具，我们不知道是什么。陶醉于你的 Arduinos，555 定时器，晶体管，或光基因改造细胞，并知道你选择的工具和持用者一样强大。

光遗传学可能成为生物制造商空间的热门门票。我们之前已经讨论过实验室啮齿动物的[光遗传学，但是它也在斑马鱼和蛔虫](https://hackaday.com/2016/08/13/the-quest-for-mice-with-frickin-laser-beams-pointed-at-their-brains-building-a-laser-controller/)中发现[购买。](https://hackaday.com/2017/08/03/optogenetics-for-100-euros/)
