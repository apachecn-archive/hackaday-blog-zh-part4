# PlayStation 4 控制器获得 USB-C 升级

> 原文:[https://hack aday . com/2022/03/01/PlayStation-4-controller-gets-a-USB-c-upgrade/](https://hackaday.com/2022/03/01/playstation-4-controller-gets-a-usb-c-upgrade/)

微型 USB 曾经是 USB-A 太大的应用程序的首选连接器，但现在 USB-C 已经占据主导地位。它正在成为许多外设的标准，而[Ian]最近决定将他的 PS4 控制器升级到新的标准。黑客攻击接踵而至。

黑客包括一个小的分线板，使 USB-C 连接器能够安装到 PS4 控制器中，取代原来的微型 USB 端口。[Ian]解释了完成 mod 需要做的事情，首先是小心地拆卸控制器以避免损坏。然后可以移除原始的 microUSB 分线板，并安装 Github 上可用的替换板[中的一个，以适应 PS4 控制器的各种版本。然后需要稍微锉一下，以使新连接器适合控制器外壳，并且[Ian]指出，使用 0.8 毫米厚的 PCB 是使新分线板适合外壳内部的关键。](https://github.com/12ian3/DS4-Type-C)

这是一个巧妙的黑客技术，使得在现代环境中为 PS4 控制器充电更加容易，而不必保留传统的微型 USB 电缆。实际上，我们已经看到类似的黑客攻击也发生在 iPhones 和其他硬件上。休息后的视频。
