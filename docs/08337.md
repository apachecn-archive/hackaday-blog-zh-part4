# 快速和肮脏的投石机投掷土豆泥

> 原文:[https://hack aday . com/2020/11/12/quick-and-dirty-trebuchet-flings-massed-potato/](https://hackaday.com/2020/11/12/quick-and-dirty-trebuchet-flings-mashed-potato/)

感恩节就要到了，[mrak_ripple]担心在社交距离条件下提供食物。他没有采用标准的最佳实践，而是选择了一条更刺激的路线——用微型攻城武器投掷配菜。(视频，下面嵌入。)

土豆泥投石机是一个建立在现代风格，依靠 8020 铝挤压，使快速和容易组装。它还利用了似乎是重型激光切割机的优势，这种激光切割机可以制造出坚固的钢支架来将所有东西固定在一起。盛放麦芽浆的发射杯是一个 3D 打印部件，由树脂制成，用胶带固定在手臂的末端，因为合适的螺栓不会落到手上。

最后，重复性是一场斗争，我们怀疑投石机实际上不会在节日本身提供食物服务。然而，这肯定会成为一个有趣的饭后游戏，看看谁能在一个愿意的目标上得到最多的麦芽浆。我们也很想看看 mash 加农炮，所以如果你已经造好了，给我们写信吧。当然，如果你喜欢更古怪的高性能设计，[飞轮投石机可能更适合你](https://hackaday.com/2020/10/02/flywheel-trebuchet-spins-right-round/)。休息后的视频。
