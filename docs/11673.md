# 面向所有人的机械联动 CAD

> 原文:[https://hack aday . com/2021/10/21/mechanical-linkage-CAD-for-every one/](https://hackaday.com/2021/10/21/mechanical-linkage-cad-for-everyone/)

尽管我们中的一些人不喜欢它，但真实地建造东西需要一些机械部件。也许它只是一个简单的外壳，甚至是 PCB 的脚，但除非你只写软件或玩模拟器，否则你最终还是要做点什么。在为前面板钻孔和试图建造会移动的东西之间，这是一条危险的斜坡。有时这就像一个铰链和一个弹簧一样简单，或者可能是一个成熟的机器人关节臂。这就是为什么[RectorSquid]建立了[Linkage](https://blog.rectorsquid.com/linkage-mechanism-designer-and-simulator/)，一个“让你设计和编辑二维机制，然后模拟该机制的运动的程序”(引用自[文档](https://blog.rectorsquid.com/linkage-mechanism-designer-and-simulator/))。

该程序已经有了几个版本，目前已经超过了 3.15。为了了解该程序的功能，下面的第一个视频展示了一个模拟球提升的旧版本。第二个视频展示了根据设计构建的实际机制。相关的 YouTube 频道也有更多最近的视频，展示了各种模拟。

该软件适用于 Windows，如果您仍然运行 Windows XP，还有一个不同的版本可供使用。文档显示，如果您喜欢在 Linux 下运行它，它也可以在 Wine 下运行。

这不会取代高端 CAD 程序的分析功能。但是价格合适，看起来简单易学。如果模拟让你在一个项目中避免了一次错误的开始，它的价值就超过了免费的价格。

如果你需要一些模拟的灵感，[这里有几个机器人机制](https://hackaday.com/2020/12/25/ten-robot-mechanisms-for-your-design-toolbox/)可以试试。如果你想要更多——更多——看看[这些视频](https://hackaday.com/2016/02/29/2100-mechanical-mechanisms/)。
