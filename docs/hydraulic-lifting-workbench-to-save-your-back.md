# 液压升降工作台，保护您的背部

> 原文：<https://hackaday.com/2020/08/16/hydraulic-lifting-workbench-to-save-your-back/>

在很高的地方操作重型机械设备是一件非常累人的事情。【从零开始的车间】深知这一点，所以他给自己造了一个非常干净的[液压升降工作台](https://www.youtube.com/watch?v=BrpKofJV-Q0)在车间周围使用。

正如我们对这个名副其实的频道的期望，该设备上的一切都是从头开始构建的。尽管他使用了现成的手动液压活塞。提升机构由一个平行连杆组成，该连杆允许工作台在其整个运动范围内保持平行。为了舒适，液压活塞的手柄被转换为脚踏板，底座有一些坚固的滚轮，可以在车间内移动。不可否认，由于需要手动抽水，提升桌子相当缓慢，但最终还是完成了工作。

制作自己的工具和设备会给你带来很大的满足感，尤其是如果你最终会经常用到它的话。【车间从无到有】制造一些优秀的工具，像这个[磁力钻床](https://hackaday.com/2020/07/07/salvaged-drill-gets-a-magnetic-upgrade/)、[磁力虎钳](https://hackaday.com/2020/05/01/scratch-built-magnetic-vise-stays-where-you-need-it/)和一个[车间起重机](https://hackaday.com/2019/09/26/a-crane-fit-for-any-workshop/)。我们希望看到更多。
