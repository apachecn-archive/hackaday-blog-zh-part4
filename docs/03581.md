# 傅立叶解释:【3Blue1Brown】风格！

> 原文：<https://hackaday.com/2019/07/13/fourier-explained-3blue1brown-style/>

如果你让大多数人解释傅立叶级数，他们会告诉你如何将任何特定的波分解成正弦波的和。我们以前也用过这种解释，它并没有错。事实上，傅立叶就是这样第一次计算出他著名的级数的。然而，这只是故事的一部分，视频制作大师[3Blue1Brown] [以他一贯的娱乐性和知识性的方式解释了这个故事](https://www.youtube.com/watch?v=r6sGWTCMz2k)。你可以看下面的视频。

自相矛盾的是，[3Blue1Brown]断言，通过思考具有复数输出的函数在二维空间中产生旋转向量，更容易理解这个级数。如果你看了视频，你会发现这是一个更简单的解决方法，而且还能让你画出非常酷的图片。

当然，我们的老朋友正弦波只是一个特例，其中复数的虚部为零。这导致了无聊的绘图，但是你可以想象，如果你理解了复数的情况，处理实数是很容易的。

视频中的另一个项目是这些方程的历史。原来傅立叶并没有试图解决任何与波有关的问题。他在研究热量方程。这个视频是热量方程系列的一部分，但它本身就很好。

这个话题[经常出现](https://hackaday.com/2019/04/10/interactive-demo-shows-the-power-of-fourier-transforms/)，部分原因是它有很多应用，从过滤到医学成像。如果你想要另一个关于[小圆圈如何兴风作浪](https://hackaday.com/2015/09/17/visualizing-the-fourier-transform/)的视觉化图像，而不仅仅是图片，我们以前也看过。
