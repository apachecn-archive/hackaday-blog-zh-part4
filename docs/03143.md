# 钙钛矿:不再仅仅用于太阳能电池

> 原文:[https://hack aday . com/2019/06/13/perovskites-not-just-for-solar-cells-more/](https://hackaday.com/2019/06/13/perovskites-not-just-for-solar-cells-anymore/)

如果你在这呆的时间够长，你会知道材料科学的进步有很长的历史，但被技术媒体和大众媒体夸大了。最近的大多数研究似乎都集中在碳的化学性质上，尤其是石墨烯和纳米管。回到过去，超导体风靡一时，在那之前是高级陶瓷、半导体和人造钻石。总会有一些新的奇迹材料被媒体屏息而无休止地报道，有这样或那样的充满希望的故事将我们从今天的灾难中拯救出来【这里是>。

虽然不可否认，这些材料中的每一种都导致了科学、工业和数十亿人生活质量的巨大进步，但从实验室到商业化的开发周期通常比媒体认为的要慢一点。因此，当一种新材料开始成为头条新闻时，就像最近的钙钛矿，我们觉得这是一个近距离观察的好机会，可以尝试平滑炒作曲线的起伏，管理预期。

## 小心带隙

在一定程度上，你听说过钙钛矿——你可能听说过，在这些页面上有[的报道](https://hackaday.com/2016/07/15/perovskite-solar-coming-soon/)和[一个完整的安培小时播客](https://theamphour.com/433-an-interview-with-sam-stranks/)专门介绍它们——你可能会有这样的印象，它们是一些新发现的材料，可以用来建造比目前的硅光伏电池更好的太阳能电池。然而，除了光伏，钙钛矿还有很多其他的东西，它们背后的化学成分也很有趣。

首先，钙钛矿——注意单数——是一种矿物，钛酸钙(CaTiO₃),于 19 世纪晚期在乌拉尔山脉首次发现，并以著名的俄罗斯矿物学家列夫·佩洛夫斯基的名字命名。世界各地都有天然存在的钙钛矿矿床，这种矿物被商业化开采。

另一方面，钙钛矿是天然和合成的化合物，与天然钙钛矿的晶体结构相匹配。钙钛矿的一般形式是 ABC₃，其中 c 通常是氧，a 和 b 是带正电荷的离子或阳离子。a 通常是一种碱金属或稀土金属，而 B 通常是元素周期表中过渡系列的金属之一。一些合成的钙钛矿甚至用有机阳离子代替无机金属，产生了具有从超导到光电子学的有趣和有用性质的杂化钙钛矿。

钙钛矿在光伏方面如此有前途的原因是它们在吸收光方面比硅好得多，这要归功于钙钛矿具有直接带隙，而不是硅的间接带隙。材料的带隙指的是在其中移动电子所需的能量，间接带隙材料必须在两步过程中来回穿梭电子。这一点的实际效果意味着间接带隙材料需要更厚，以吸收足够的光来工作。需要 200 微米厚的硅层来吸收与半微米层混合钙钛矿如三卤化甲铵铅(CH₃NH₃PbX₃，其中 x 通常是碘或溴)相同的能量。此外，混合钙钛矿的带隙可以通过调整卤化物的比例来覆盖更多的光谱，如上图所示。

杂化钙钛矿的另一个卖点是可以制成水溶液，然后使用非常简单的技术，如气相沉积或旋涂，将其应用于玻璃等基材上。甚至有人正在使用改良的喷墨打印机打印钙钛矿。将这些过程与硅太阳能电池生产的费力方法进行对比，硅太阳能电池生产基本上需要半导体工厂的大部分设备，你可以看到钙钛矿太阳能电池(PSCs)的兴奋是有道理的。

当然，钙钛矿并不是太阳能电池技术的全部。存在重大问题，包括效率低于硅电池。不过，这可能是一个暂时的问题；尽管硅光伏电池用了近 70 年的时间才从 6%的效率达到目前 27%的商业电池效率，但太阳能电池已经达到 23%，它们从 2009 年才开始开发。随着更多的工作，钙钛矿效率可以超越硅 PV，接近理论上的最大值 31%。

PSC 的真正缺点是工程上要经得起风吹雨打；混合钙钛矿的水溶性使其易于加工，也使其难以防潮。染料的紫外线降解也是一个问题——有机化合物在紫外线辐射下非常不稳定，需要在钙钛矿层上施加能够可靠地过滤紫外线而不阻挡其他波长的层。

## X 射线传感器

效率的潜在提高和成本的降低使得围绕 PSC 的工程问题将得到解决，全面商业化将开始。但是将太阳光转化为能量远非钙钛矿所能完成的唯一技巧，甚至可能不是它们最有利可图的应用。医学成像，尤其是 X 射线成像，将从钙钛矿型传感器中受益匪浅。

作为另一种波长的光，X 射线同样与钙钛矿相互作用，将它们转化为可以直接感知的电荷。这不同于当前用于数字放射成像的大多数 X 射线传感器，它们是间接传感器——一个覆盖有被 X 射线击中时会发光的化合物(称为闪烁体)的平板，被夹在标准电荷耦合器件(CCD)中间，该器件拾取光并创建 X 射线对组织的差异穿透的数字表示。这样做的好处是非常灵敏，并减少了产生图像所需的 X 射线剂量，但代价是降低了分辨率，因为来自闪烁体颗粒的光在到达 CCD 检测器之前就散开了。

另一方面，直接传感器将入射的 X 射线直接转换成电荷，大大提高了分辨率。但是传统的半导体直接 X 射线传感器通常使用硒，当用作太阳能电池时，硅也会遇到同样的问题——它不能很好地吸收电磁辐射。这使得硒探测器获取图像的速度很慢，需要更高的 X 射线剂量。然而，钙钛矿直接传感器吸收更好，导致分辨率更好的传感器，需要更低的剂量。

钙钛矿在太阳能电池中的商业化面临的许多同样的障碍仍然存在于 X 射线传感器中。但是价值数十亿美元的医学成像行业可能会在首先将钙钛矿推向市场方面占据上风。保护钙钛矿 X 射线传感器免受与患者护理相关的潮湿和粗暴处理要比设计 PSC 来承受任何实用太阳能电池阵列都必须应对的多年雨夹雪紫外线和风吹雨容易得多。

也就是说，医学成像应用对钙钛矿的了解无疑将推动 PSC 的发展，我们猜测钙钛矿将在未来五年左右成为一个家喻户晓的词。
