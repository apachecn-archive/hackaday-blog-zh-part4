# 模块化数控构建让您同时拥有一台铣床和一台激光切割机

> 原文：<https://hackaday.com/2019/05/12/modular-cnc-build-gets-you-both-a-mill-and-a-laser-cutter/>

CNC 构建有各种形状和大小。有三角洲机械手，实验极地钻机，和各种各样的笛卡尔建筑，大的和小的。在完成他们的第一个 CNC 构建后，[【jtaggard】将他们所学到的东西应用到新机器](https://www.instructables.com/id/Modular-DIY-CNC-V2/)的开发中。

这是一个书桌大小的笛卡尔设计，框架由圆锯切割的 V 型槽挤压成型。这是为定制构建获得高质量挤出的一个很好的方法，并且容易获得和使用。机架骑在轮子上，X 和 Y 轴由皮带驱动，z 轴由螺旋驱动。几个 NEMA 17 和一个 NEMA 23 提供动力，一个带步进驱动器的 Arduino Uno 是操作的大脑。1/4”厚的 PLA 板用于组装一切，虽然[jtaggard]打算在轨道上用铝代替这些板，但到目前为止，塑料已经证明足够坚硬，足以用于加工和雕刻木材的早期测试。

这是一个很好的入门级 CNC 构建，已被证明可用于 500W 主轴和 2.5W 激光雕刻。由于本质上是模块化的，因此很容易添加其他工具，如笔式绘图仪或乙烯基切割刀片，以实现进一步的多功能性。

DIY CNC 构建总是很受欢迎，因为你最终会得到一个有用的工具，作为对你辛勤工作的奖励。休息后的视频。
