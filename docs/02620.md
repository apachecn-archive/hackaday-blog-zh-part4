# 廉价制造振动滚筒

> 原文:[https://hack aday . com/2019/04/05/building-a-vibration-tumbler-on-the-cheap/](https://hackaday.com/2019/04/05/building-a-vibration-tumbler-on-the-cheap/)

很少有像打磨和抛光这样费力的工作。任何需要大量时间和体力的工作都是机械帮助的首选，这份工作也不例外。在朋友的建议下，【VegOilGuy】决定是时候投资一个震动不倒翁了。自然，建造它而不是购买它是当时的主流。

如果你曾经使用过电动打磨机，你会知道它们是一个极好的振动源。最初，我们的目的是制造一种滚磨机，其磨砂机是可拆卸的，但仍可用于其原始用途。一个夹紧基座是用扎带、木头和邦多建造的，但遗憾的是无济于事。Velcro 与塑料不倒翁碗的连接不够牢固，无法承受重复使用。

取而代之的是，打磨机被永久地固定在转鼓上，产生了更好的效果。然后在碗中加入一个漏斗，以改善培养基的循环并减少所需的量。最初的测试是积极的，使用碎核桃壳的滚筒成功地抛光了一些铸造黄铜零件。然而,[VegOilGuy]仍在寻找一种研磨性更强的介质用于初始铜锈去除。

有时候最好的工具是你自己打造的。在这种情况下，这是一个廉价和简单的方式来获得振动不倒翁和结果是伟大的。如果你不喜欢桑德斯，为什么不试试这种基于粉丝的风格呢？休息后的视频。
