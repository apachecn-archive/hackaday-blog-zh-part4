# 爬虫空间爬虫

> 原文：<https://hackaday.com/2023/01/03/the-crawlspace-crawler/>

这个 FPV 机器人是一个相当简单的建筑。[Jeff G]买了一个四四方方的底盘套件，包括框架、电机、车轮、安装的灯和摄像头，我们可以看到它的运行(视频，嵌入在下面)。

一如既往，细节是它的所在，他的概述涵盖了大部分的高点。[Jeff]使用相对较慢的 60 转/分的马达，这样他就有足够的咕噜声。FPV 的设置非常简单——他买了一个便宜的 Flysky i6 发射机和接收机，以及一个 Eachine TX05 一体化摄像机和发射机。一个有趣的选择是一个 USB UVC 视频接收器，这样他就可以在电脑、平板电脑或手机上观看视频，这意味着他不必支付昂贵的 FPV 护目镜。我们也喜欢用棍子和拉链作为触角，让他知道什么时候他会被卡住，但这也可以作为摄像机的视觉框架。

FPV 竞赛刚刚结束，我们将很快宣布获胜者！如果你在那里为你自己的项目找到了灵感，[Jeff]这里的简单基础应该能让你开始步入正轨。
