# 科学研究工厂

> 原文：<https://hackaday.com/2022/10/19/researching-factorio-for-science/>

科学已经肯定地回答了许多问题，回头看，这些问题一开始就被认为是奇怪的。像“这种发霉的奶酪能治病吗”或者“如果我们给它通电，这块石头能进行数学运算吗？”这个列表中最近的一个问题是电子游戏 *Factorio* 是否可以作为其他学术工作的基础，在这个游戏中，玩家建造了一个复杂的工厂。[正如【肯尼斯·里德】在这次演讲中所讨论的，它肯定能够](https://www.youtube.com/watch?v=9ie01GLGB5s)。

如果你没有玩过这个游戏，这是一种即时战略(RTS)游戏，玩家收集材料建造一个工厂，同时抵御敌人。从表面上看，它似乎类似于*帝国时代*或*星际争霸*，但是它的复杂性达到了其他 RTS 游戏所没有的极限。复杂性隐藏了细微差别，[Kenneth]指出，它是一个优秀的模拟器，可以研究现实世界的问题，如车辆路线问题、决策、人工智能、装箱问题和生产计划，以及其他一系列有趣的潜在研究领域。

[Kenneth]和他在这个项目上的合作伙伴还开发了一些软件工具，可以与一款 *Factorio* 游戏进行交互，而不必直接玩游戏。该游戏包括一个 API，该团队使用它来开发工具，以便其他研究人员可以将其作为模拟和研究的基础。[还发表了一篇研究论文](https://www.researchgate.net/publication/349158902_The_Factory_Must_Grow_Automation_in_Factorio)，以便更深入地阅读该主题。我们也不应该对一款游戏能以如此令人难以置信的高效方式使用感到太惊讶。这是另一个例子，[玩具被用来训练工业自动化领域的工程师](https://hackaday.com/2017/04/16/desktop-factory-teaches-plc-programming/)。
