# Kamehameha！！PCB 徽章

> 原文:[https://hackaday.com/2021/12/19/kamehameha-pcb-badge/](https://hackaday.com/2021/12/19/kamehameha-pcb-badge/)

在过去的几年里，PCB 艺术确实吸引了我们，我们曾经对社区在他们的作品中加入的复杂细节很感兴趣。我们对[Arnov]的工作并不陌生，他的 [Kamehameha PCB 徽章](https://www.hackster.io/413148/goku-pcb-badge-f26f11)再一次给我们留下了深刻的印象。

不幸的是，在这个项目的制作过程中没有使用 555 定时器，但是不要因此而拒绝你。相反，我们有一个 ATtiny84 微控制器来实现控制 led 的逻辑，一个基于 MOSFET 的驱动器来驱动电流通过 led，当然，led 可以让“海龟破坏波”发出毁灭性的光芒。请特别注意[Arnov]在丝网印刷中的细节，因为你可以看到这是这个版本中非常重要的一部分。

除了惊叹于[Arnov]的工作之外，OrCAD PCB 设计软件的爱好者还将在[Arnov]的教程中学习如何将图像文件导入到他们的项目中。如果你想尝试一下 SMD 焊接，他甚至有一些非常好的回流焊接技巧。

另一个很酷的建筑[Arnov]。保持下去！
