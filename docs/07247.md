# 机器人，须藤帮我叠衣服

> 原文:[https://hackaday.com/2020/07/22/robot-sudo-fold-my-laundry/](https://hackaday.com/2020/07/22/robot-sudo-fold-my-laundry/)

[Ty Palowski]不喜欢折叠他的许多衬衫。他在电视上看到一种据说可以简化折叠的板子，但它确实需要你手动移动板子。这是不行的，所以[Ty]将它机动化，创造了一个衬衫折叠机器人。

电路板的想法并不新鲜，可能很多人不会介意所需的简单操作，但除了为衬衫折叠机制作电机支架，你还能拿 3D 打印机做什么呢？当然，折叠板对于 3D 打印来说太大了，所以他首先用纸板做了这个部分，然后是看起来像泡沫板的东西。

侧面的“翅膀”很容易操作，但是顶部的折叠需要更多的努力。机器在最后仍然需要手动折叠，当然，你必须把衬衫放在正确的位置，这样事情才能解决。

老实说，我们不确定这是一个非常实际的项目，但我们仍然喜欢这个想法，我们不能否认它似乎是可行的。我们不认为有太多的扭矩需要，所以我们想知道，如果一些牛肉钢筋混凝土伺服会一样有效，可能更容易工作。尽管如此，几乎任何可以移动的东西都可以工作。你甚至可以用一个弹簧和一个螺线管来达到同样的效果。没有太多的构建细节，但我们认为你可以使用手头上的任何电机来解决它。

如果你觉得洗衣服太费时间，总有十一种选择。如果你想要[一个更好的折叠机器人](https://hackaday.com/2016/02/24/the-challenges-of-a-laundry-folding-robot/)，你将不得不投入一些认真的工作。
