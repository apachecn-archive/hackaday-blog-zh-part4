# N64 电源适配器适用于世界各地

> 原文:[https://hack aday . com/2020/10/29/n64-power-adapter-works-around-the-world/](https://hackaday.com/2020/10/29/n64-power-adapter-works-around-the-world/)

无论你身处世界的哪个角落，手机和笔记本充电器等现代电子产品的用途都非常广泛。能够运行任何从 100-250 伏，所有你需要的是一个插座适配器，你很好去。然而，20 世纪 90 年代的视频游戏机没有这么灵活。[MattKC]厌倦了用降压变压器来运行他的美国市场 N64，并决定纠正这一点，[建立一个通用适配器来运行控制台。](https://www.youtube.com/watch?v=vxNnUvdotA8&feature=emb_logo)

这是一个真正的粗制滥造，由一堆旧零件组装而成。一个破损的 N64 电源适配器因其外壳和独特的 DC 插头而被收获，该插头将 12V 和 3.3V 传输到控制台。很少有紧凑型电源能够提供这一对电压，因此[MattKC]发挥了创造性。一台旧路由器采用 12V 2A 供电，并与一个 3.3V 降压转换器相结合，为两条供电轨供电。通过一些创造性的包装和大量的安装胶带，这些用品被塞在原来的箱子里，并连接到原来的插孔和 8 字形电缆，允许在不同的国家轻松更换插座，而不用使用丑陋的适配器。

虽然我们中很少有人经常带着 25 年前的任天堂游戏机旅行，但对于那些这样做的人来说，单一通用供应的便利性不能被夸大。毕竟，在随身行李中安装一个降压变压器是不切实际的。我们早在 2006 年就有类似的黑客攻击[，或者最近的](https://hackaday.com/2006/01/05/xbox-360-atx-power-supply/)[一个寻求为古老的阿米加 500](https://hackaday.com/2018/03/22/converting-power-supplies-for-antique-computers/) 重建新 PSU 的项目。休息后的视频。
