# 伟大的电脑黑客让黑客攻击电脑

> 原文：<https://hackaday.com/2022/03/18/great-computer-hacks-make-hackers-hacker-computers/>

在 1995 年，计算机，嗯…很无聊。未来还没有到来，电脑是单调、笨重的米色盒子。当然，有一些很酷的电脑在销售，但是这个领域仍然相对贫瘠。但是正如你将在休息时间下面的视频中看到的，事情并不一定是这样的，[黑客馆长]通过重现 1995 年电影*黑客*中李炯楷·米勒的电脑向我们展示了这一方法。

《黑客》问世时并不流行，但多年来它已经赢得了相当多的追随者。它以全新的方式描绘了电脑和热爱电脑的人们，代表了一种从未存在过的文化。即便如此，它激发了如此多的年轻黑客类型。受到启发的人中有[【黑客策展人】](https://hackerscurator.com/)的工作人员，他们已经承担起自己的责任，嗯……策划……围绕这部电影的道具、服装和故事。

重现戴德标志性的迷彩“可携带”电脑有相当大的难度。原来，最初的电影道具是使用定制外壳和 Mac Powerbook 180c 内部部件的定制电脑。Dade 的(又名 Zer0 Cool 和 Crash Override)是 Compaq Portable 486c 和前面提到的 Mac 的混搭。[黑客策展人]用两台坏掉的电脑可爱地再造了这个道具，但选择用树莓派来运行内部。

创建这个糟糕的网络平台所使用的技术可以用于构建许多其他项目，即使你不是一个黑客。定制塑料和把轨迹球放在最尴尬的地方是很专业的，我们将来做类似的项目时会参考它。

电影复制黑客是你的事吗？你很幸运！事实证明，这不是[黑客馆长]的第一次建设。2019 年他们攻克了尼康勋爵的笔记本电脑，当然，我们[也报道了那台！](https://hackaday.com/2019/10/15/recreating-lord-nikons-laptop-from-hackers/)
