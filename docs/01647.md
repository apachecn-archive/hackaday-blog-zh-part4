# 任天堂做索尼，比索尼好

> 原文:[https://hackaday . com/2018/12/23/Nintendo-does-Sony-better-Sony/](https://hackaday.com/2018/12/23/nintendo-does-sony-better-than-sony/)

随着过去明星的一系列官方重启进入市场，来自电视游戏设备黄金时代的游戏机粉丝们在过去几年中享受了一次款待。这些所谓的“经典”游戏机严格遵循原版的风格，但在引擎盖下，它们包装了运行模拟器的现代硬件，可以玩 rom 中的一系列游戏。更好的是，通过一点黑客技术，他们可以运行不仅仅是提供的模拟器，人们已经成功地使用它们来模拟完全不同的控制台。即便如此，人们还是意外地发现，超级任天堂经典游戏机上的 PlayStation 模拟器运行 PlayStation 游戏的效果比索尼自己的 PlayStation Classic 游戏机上内置的相同模拟器要好。

尽管两台机器基于 Allwinner R16 片上系统具有几乎相同的硬件规格，但还是实现了[8 位闪回]这一壮举。任天堂提供了更流畅的动作和更灵敏的控制，带来了更优越的游戏体验。这是如何实现的？最显著的区别是 SNES 经典版安装了 [RetroArch](https://www.retroarch.com/) 前端，这可能会带来一些优化和调整，使系统更有效。

关注不寻常游戏机的读者可能还记得另一个任天堂/索尼的混合体，命运多舛的 20 世纪 90 年代早期的原型 SNES，带有一个 CD-r om，这是第一个以“Play Station”命名的机器(有时被称为“PlayStation”，导致索尼营销人员在几年后热衷于作家使用单词之间的空格)。
