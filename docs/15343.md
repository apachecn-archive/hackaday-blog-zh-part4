# 与这个喷火的亿万富翁一起保护你的财产

> 原文：<https://hackaday.com/2022/11/12/protect-your-property-with-this-fire-breathing-billionaire/>

让我们面对现实吧:如果你不能相信一个喷火的亿万富翁实业家会保护你的东西，你还能相信谁呢？(视频，嵌在下面。)

这是直接来自真正糟糕的想法文件，来自[Marc Radinovic]。他的故事是，他想保护他新房子里的东西，并认为带有火焰喷射器的面部识别系统是解决这一问题的最佳方式。为了让它更好，这个系统将会成为每个人最喜欢的财阀的可笑肖像。该系统的内部结构与您预期的差不多——一个摄像头和一个运行 OpenCV 的 Raspberry Pi 和一个面部识别库，一个丁烷储罐和一个电磁阀，一个作为点火源的电弧点火器，一个 Arduino 和一些完全不粗略的控制所有部件的布线。当然，还有给[埃隆]眼睛用的液晶显示器。

该系统经过训练，能够识别[马克]的脸，并在看到他时愉快地向他打招呼。[非马克]人，然而，对待有点不太适应，直到和包括融化脸的火球。其他亿万富翁的模拟像得到了这种待遇；奇怪的是，[马克]的人脸识别算法甚至没有将另一个[马克]识别为人脸，当你想到这一点时，你会觉得非常有趣。

因此，肯定不是一个实用的安全系统，也肯定不是你应该建立的东西，但无论如何它是非常有趣的。这让我们想起了几年前看到的喷火鸭。
