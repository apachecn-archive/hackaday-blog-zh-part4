# 手掌大小的加特林机枪有 32 个迷你橡皮筋，上面有你的名字

> 原文:[https://hack aday . com/2019/02/13/手掌大小的加特林枪上有 32 个刻有你名字的迷你橡皮筋/](https://hackaday.com/2019/02/13/palm-sized-gatling-gun-has-32-mini-elastics-with-your-name-on-them/)

3D 打印机擅长的一件事是能够轻松地创建其他方法难以实现的对象，这也允许快速设计迭代。这一点在[Canino]的手掌大小、加特林式机动 32 弹发射器中显而易见。

这种大炮有一个由小型马达驱动的旋转炮管，一个巧妙的扣机设计像蜗轮一样利用炮管的旋转。旋转的桶具有螺旋结构的钩，这些钩锚定拉伸的弹性带。一个小斜坡骑在螺旋间隙上，随着组件的转动，一次抬起一个拉伸带的末端。这种移动(以及因此的发射控制)是通过一个小的连续旋转伺服机构来完成的。虽然理论上任何电机都可以，但使用伺服系统的优势在于它是一种标准化的形状，因此易于集成到设计中。下面嵌入了一个视频，你可以看到它的工作，以及一些行动的特写镜头。
