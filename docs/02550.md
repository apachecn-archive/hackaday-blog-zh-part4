# 制作手工卷尺

> 原文:[https://hack aday . com/2019/03/27/building-an-artificial-tape-measure/](https://hackaday.com/2019/03/27/building-an-artisanal-tape-measure/)

一些工具如此普遍，如此基本，以至于我们认为它们是理所当然的。一个完美的例子是低卷尺。我们可能都有一些这样的东西在实验室里到处乱放，当你在使用它们的时候，你不会仔细考虑它们。但是，虽然我们大多数人可能不会再考虑我们的卷尺，但[【Ariel Yahni】决定为他的](https://openbuilds.com/projects/tape-measure-modern-enclosure.221/)创造一个绝对华丽的新围栏。因为如果你要衡量什么，为什么不做得好看一点呢？

[![](../Images/0999276efa6c771732199f8ffcaeb9ed.png)](https://hackaday.com/wp-content/uploads/2019/03/woodtape_detail.jpg) 一台数控刻字机被用来在一块坚固的木头上刻下新卷尺的主体，并在透明的丙烯酸树脂上切割出一个顶板。[Ariel]然后用角磨机切下一小段钢棒，用环氧树脂将其固定在底座的雕刻凹槽中。最后，商业卷尺的内部被插入这个新的外壳，丙烯酸顶部被拧紧到位。

[Ariel]已经公开了这个项目的 DXF 文件，供任何想雕刻出自己的传家宝卷尺的人使用，尽管看起来设计可能需要一些调整，这取决于捐赠卷尺的品牌和型号。虽然这可能不是在 Hackaday 上运行的技术上最令人印象深刻的项目，但它仍然是现代制造方法使定制设计成为可能的一个极好的例子。

这个设计让我们想起了一个类似的项目，通过添加一些数控硬木和铝，将一个基本的本田钥匙链变成一个真正的话题。
