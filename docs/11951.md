# 自制钻机可精确测量木材的强度

> 原文：<https://hackaday.com/2021/11/22/shop-built-rig-measure-strength-of-wood-accurately/>

木材是一种用途极其广泛的材料，但像其他任何东西一样，它也有其局限性。用不结实的木头做一把椅子，最坏的情况可能也没那么糟。但是如果你用木头制造机床，用错木头的风险可能会高一点。

这就是[木材强度测试装置](https://www.youtube.com/watch?v=hxR-dexHiI4)【马蒂亚斯·万德尔】提出的想法。此前，他有一个有点应急的测试装置，用液压千斤顶对试件施加力，并用浴室秤进行测量。这种设置是次优的，所以第二个版本使用了一个螺旋千斤顶来施加力，但浴室秤仍然留下了测量结果的解释。第三个版本，也就是下面视频的主题，配备了应变仪和连接到 Raspberry Pi 的模数转换器，以自动收集数据。螺旋千斤顶也与步进电机和[【马提亚斯】](http://woodgears.ca/gear_cutting/template.html)著名的木制齿轮整合到测试设置中。

虽然试验台的设计非常简单，但校准有很多微妙之处，以确保它测量的是测试材料本身，而不仅仅是机构内的合规性。这只是[马蒂亚斯]似乎乐此不疲的一长串数据收集练习中的又一个，就像他最近的 woodshop electrical explorations。
