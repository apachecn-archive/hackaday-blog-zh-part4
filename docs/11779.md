# 骷髅密切关注你，因为这是万圣节，好吗

> 原文：<https://hackaday.com/2021/10/30/skeleton-watches-you-intensely-because-its-halloween-okay/>

如果你曾经看过一幅画，画中的眼睛跟着你在房间里走来走去，你可能会觉得有点不安。[CuriousInventor]进一步发展了这个概念，它的骨架可以注视视野内的任何人。

该系统的核心是一个树莓 Pi Zero，配有 Pi 摄像头。运行 OpenCV，代码被设置为跟踪人类，并将骨骼的头部转向任何被检测到的人。这是通过骨骼颈部的伺服系统实现的。伺服帽用于驱动伺服系统，而不会不必要地拉紧树莓皮。

骨架本身看起来没有任何改变，尽管大多数电子设备都安装在一个非常明显的塑料盒内。我们希望看到第二版，所有的硬件都整齐地放在头骨里。

这是一个有趣的黑客，使一个愉快的万圣节装饰。然而，OpenCV 也可以做其他有用的事情，比如识别杂草。休息后的视频。
