# 飞刀机的旋转恰到好处

> 原文:[https://hack aday . com/2022/08/23/飞刀机-获得-旋转-恰到好处/](https://hackaday.com/2022/08/23/knife-throwing-machine-gets-the-spin-just-right/)

尽管在糟糕的动作电影中可能会出现，但扔出一把刀并让它扎进目标并不是一件容易的事。从前面提到的电影中获得灵感，[金特]和他的儿子建造了一个[杂志馈送的飞刀机](https://www.youtube.com/watch?v=-BKEZbYOMpI)，能够在其范围内的任何距离插刀。

用机器投掷一块锋利的金属并不难，但选择先击中目标点的时机却是一个真正的挑战。为了实现这个目标，[金特]使用了一对高性能的伺服电机来驱动一对平行的同步带。在皮带之间安装一个带有旋转刀架的托架，通过稍微加快一条皮带的运行速度来实现旋转投掷。托架在一对铜轨道上滑动，铜轨道也通过几个重新利用的碳刷为刀架提供动力。

起初，刀架是一个电磁铁，但它不能可靠地保持或释放不锈钢飞刀。这被改变为一个螺线管驱动的机制，锁定在切刀上加工的槽中。只要按住扳机，刀就会从后面的弹簧弹匣中自动进给，从技术上讲，这是全自动的。为了使旋转速度与投掷距离相匹配，激光雷达传感器用于测量距离，它还可以调整瞄准激光的角度，以补偿刀的轨迹。

开发过程充满了挫折、失败和危险。不可靠的刀架、爆炸的车厢和看似自动点火的故障电子设备都是需要克服的挑战。然而，结果是一台既能扔刀又能培养孩子对构建和编程的热情的机器。
