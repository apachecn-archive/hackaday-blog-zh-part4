# 挑选和放置黑客聊天揭示组装秘密

> 原文:[https://hack aday . com/2022/02/11/pick-and-place-hack-chat-reveals-assembly-secrets/](https://hackaday.com/2022/02/11/pick-and-place-hack-chat-reveals-assembly-secrets/)

这些天来，我们有强大的免费工具来进行 CAD 和电路设计，廉价的桌面 3D 打印机可以敲出定制的外壳，便捷的服务可以旋转一堆 PCB，并以远低于任何人想象的价格将它们送往你的前门。简而言之，如果你想打造自己的专业外观的小工具，唯一的限制就是你的时间和雄心。好吧，假设你只想建造其中的一些。

一旦你开始在你要生产的产品数量上加一些零，手工组装 PCB 很快就行不通了。进入取放机。这种现代技术的奇迹可以在人类花费的时间的一小部分内将所有那些微小的元件放到你的板上，并且永远不需要去洗手间。本周，Chris Denney 来到 Hack Chat 谈论这些不可思议的机器以及将你的电路板设计变成成品的所有细节。

[![](../Images/7f1ac9f46b163f6471188d67ad3b83c7.png)](https://hackaday.com/wp-content/uploads/2022/02/PickAndPlaceHackChatPoster1-01.png) 克里斯是[沃星顿组装](https://www.worthingtonassembly.com/)的首席技术官(CTO)，这是一家位于马萨诸塞州南迪尔菲尔德的快速周转电子制造商，自 1974 年以来一直在制造和运输定制电路板。他对 PCB 生产略知一二，并希望帮助初级和中级工程师创造更容易制造的设计，当 COVID hit 和亲自参观工厂不再可能时，他开始了“ [Pick，Place，Podcast](https://www.pickplacepodcast.com/) ”。现在他说，他可以通过他的建议有多少被纳入设计来判断一个板子何时来自一个固定的听众。

那么，你应该做些什么来确保你的主板装配尽可能顺利地进行呢？克里斯说，很多都是常识性的东西，比如包括清晰的极性指示器，清晰的丝网印刷，以及基准标记的使用。但有些建议可能会让人感到意外，比如他建议坚持使用经典的绿色阻焊膜。虽然现代板房可能会让你从五颜六色中选择，但事实是，大多数设备在历史上都是绿色的。

黑色印刷电路板看起来可能很光滑，但可能会混淆老式的拾取和放置机器或输送机，这些机器或输送机的设计考虑了经典绿色印刷电路板的反射率。这也使得自动光学检测(AOI)更加困难，尤其是对于较小的元件封装。也就是说，白色和红色等其他颜色问题不大，通常只需要对设备进行一些微调。

他还稍稍揭开了合同制造(CM)世界是如何运作的。虽然许多人可能有这样的印象，PCB 游戏已经转移到海外，克里斯说，少于 10，000 个单位的订单仍然主要由国内 CMs 掌握，以尽量减少周转时间。他还指出，许多组装厂几乎完全由几个关键客户支撑，因此，尽管它们可能要应付 50 个客户，但通常只有两三个“大鱼”提供了它们 80%的业务。对于这样一个紧密团结的团队，他警告说 CMs 可能有点选择性；因此，如果一个客户很难相处，他们很容易发现自己吃亏了。

虽然黑客聊天官方只安排了一个小时，但 Chris 花了近三个小时，与社区成员聊天，聊一切与电子设计和生产有关的事情。他对这个主题的知识和热情是显而易见的，我们很高兴他能在日程安排中抽出时间来加入我们。
