# 旁门左道攻击黑客与萨米卡姆卡聊天

加入我们太平洋时间 3 月 25 日星期三中午与 Samy Kamkar 的[旁道攻击黑客聊天](https://hackaday.io/event/170293-side-channel-attacks-hack-chat)！

在计算机安全领域，好消息是许多供应商现在终于认真对待安全问题，结果是直接攻击更难得逞。坏消息是，在很多情况下，他们仍然敞开着大门。旁路攻击有各种各样的形式，但它们都有一个共同点:它们通过一个意想不到的媒介泄露关于系统状态的信息。从监测你打字时键盘发出的声音，到观察薯片袋响应附近对话的微小振动，旁道攻击利用这些漏洞来泄露信息。

旁门左道利用可能是黑帽黑客的谋生之道，但了解它们对我们这些更关心保护系统的人来说可能是有用的，或者可能为我们的逆向工程工作提供信息。Samy Kamkar 对旁道攻击知之甚多，以至于他在 2019 年 Hackaday Superconference 上就这个主题做了一场精彩的演讲。他会顺便访问黑客聊天，以“扩展和增强”该谈话，并回答您关于侧渠道利用的问题，并讨论它们提供的逆向工程潜力。加入我们，了解这个迷人的世界，系统的复杂性会导致意想不到的后果，这些后果可能会反过来伤害你，甚至可能帮助你。
