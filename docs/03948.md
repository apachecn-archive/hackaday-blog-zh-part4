# 自动复卷机让风筝回收变得轻而易举

> 原文:[https://hack aday . com/2019/08/21/automatic-rewinder-makes-kite-retrieve-a-breeze/](https://hackaday.com/2019/08/21/automatic-rewinder-makes-kite-retrieval-a-breeze/)

所以你做了一个很好的风筝，把它带到沙滩上，让它放在一个线轴上乘风高飞。最终它必须下来，而收线的过程很可能是一个比你想面对的更大的苦差事。怎么办？

如果你像马特·比尔斯基一样，答案很简单:造一个电动风筝线轮，让它毫无痛苦地回来。当然，简单的概念往往难以执行，正如下面的第二个视频所示，[Matt]在开始制造零件之前，经历了广泛的设计和原型阶段。必须回答一些基本的问题，比如卷风筝需要多大的扭矩，标准风筝线卷轴的尺寸是多少。有了这些信息和一个纸板原型在手，一个无绳电钻的内脏结合了一堆 3D 打印的零件，形成了行走装置。我们真的很喜欢对自动换向螺杆的研究，这种螺杆用于将细绳均匀地缠绕在卷轴上；谁知道有人能做关于绕纱的博士论文？在下面第一个更短的视频中，看看“卷纸器-内部”的作用。

随着一些额外的电力从原来的钻头电池剩下，[马特]功能-爬行了一点与 USB 充电器端口和电压表，但谁能责怪他呢？就我个人而言，我们会包含一个计数器来记录输出了多少行；类似于打印机灯丝计数器的东西可能会起作用，只要你能把沙子挡在外面。
