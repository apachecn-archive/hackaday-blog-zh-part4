# 麻省理工学院的火腿天线

> 原文：<https://hackaday.com/2020/02/04/ham-antennas-from-mit/>

对付天线是我们觉得自己了解不够的话题之一。麻省理工学院有一个由校园业余无线电俱乐部赞助的关于天线的现场直播讲座。你可以看到下面的录音。

这篇演讲的主要论点是，除非是一个环，否则一切都是偶极子。虽然教授可能在极高的理论水平上处理天线，但她做了很好的工作，使其针对业余无线电爱好者。

这个演讲大约有一个小时长，所以它不是为 YouTube 一代优化的。有一些介绍性的材料，看起来好像是在我们以前的物理课上。然而，谈话越到最后越实际。

这里必须提到 Yagis 和 loops。甚至还有一张史密斯图表。如果你不知道 Chu 极限是什么，你绝对应该看这个视频。演讲的最后将介绍一些使用有源器件甚至移动部件的非常小的天线。

如果你想更多地讨论天线背后的原因，我们真的很喜欢这个视频。这部[两部曲也值得你花时间](https://hackaday.com/2016/11/27/start-your-path-to-becoming-an-antenna-guru/)。
