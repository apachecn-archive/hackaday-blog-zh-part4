# 用工业级硬件制造的机器人酒保

> 原文:[https://hack aday . com/2021/05/15/robotic-barber-build-with-industrial-grade-hardware/](https://hackaday.com/2021/05/15/robotic-bartender-built-with-industrial-grade-hardware/)

机器人调酒师是这些地区的一个热门项目。毕竟，如果有一件事是黑客们喜欢的，那就是任务自动化——挑战和实际节省的时间一样多。这个由一群[Teknic Servo]工程师构建的模型是一个令人印象深刻的例子，展示了一些工业级硬件可以做些什么。

酒保是作为 ClearCore 控制器的演示项目建造的，[tek NIC]工业级设备能够与一整套伺服电机和传感器接口，以完成工作。控制器被连接到一组 ClearPath 伺服电机上，这些电机负责旋转瓶子传送带，搅拌或搅拌饮料，或通过机器运输饮用玻璃杯。还有几个互锁装置，以避免顾客在工作时接触到调酒师的移动部件，还有一个标准的条形混合分配器，由螺线管驱动，以保持简单。饮料的选择和控制是通过一个触摸屏，滑块选择偏好，如酒精含量和甜度。

酒保当然有能力生产一种纯净的饮料(双关语)，并作为一个很好的例子，说明一个项目可以多么容易地与工业级硬件结合在一起。如果您有预算，您可能会发现使用工业即插即用组件比在分线板上组装开发板、电机控制器屏蔽和其他附件更快。毕竟，完成工作的方法总是不止一种。
