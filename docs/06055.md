# 线控转向是机械师的第六感

> 原文:[https://hack aday . com/2020/03/24/turn-by-wire-is-a-machinists-第六感/](https://hackaday.com/2020/03/24/turn-by-wire-is-a-machinists-sixth-sense/)

当你第一次走进机械商店时，很难不被吱吱声和呼呼声吓到。在这里，熟练的机械师驾驶着巨大的铸铁块，将金属变成成堆的碎片，生产出漂亮的零件。但是如果机床本身不需要看起来那么可怕呢？如果使用它们能感觉更直观一点，甚至，我们敢说，从一开始就是自然的，会怎么样？

进入[线控转向](https://rutian.github.io/projects/TBW/)，Berkelely 的研究人员【东润·田】、【韦丹特·萨兰】、【马莱克·克里兹勒】、【弗洛里安·迈克尔斯】和【埃里克·保罗斯】为您带来一套应用于车床的独特力反馈和机器控制概念。

线控转向以两种方式极大地重新想象了用户的控制输入和机器输出之间的关系:(1)通过改变手动曲柄和机器运动之间的映射，以及(2)通过改变机械师感觉到的触觉反馈。由于这两种交互都可以通过编程来定义，研究人员创建了三种与车床交互的独特方式。首先，通过在图形用户界面(GUI)中定义刀具路径，机械师可以使用一个手动曲柄沿该刀具路径及时前进和后退。第二，通过在 GUI 中应用虚拟引导线，当机器和手动曲柄足够接近时，它们将物理地捕捉到引导线。最后，手动曲柄可以用来教机械师一种技术，当机械师通过像啄钻一样的过程时，通过根据运动给手动曲柄增加阻力。

这是[汤姆·奈特]的一个很好的例子，“只要用电脑把它包起来！”正如[Bunnie Huang]在介绍 [IQ 电机模块](https://hackaday.com/2018/03/26/iq-makes-smarter-motors/)时提到的。这是一个强有力的例子，说明在控制器和机器之间放置一台计算机可以纠正现实世界中的缺陷，无论是操作者的机器结构。出于好奇，看看[东润的][T2 的]论文发表在尤伊斯特和[韦丹特的][T4 的]硕士论文。
