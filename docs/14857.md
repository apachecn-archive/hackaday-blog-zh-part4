# 用二极管激光切割金属？

> 原文:[https://hack aday . com/2022/09/21/用二极管激光切割金属/](https://hackaday.com/2022/09/21/cutting-metals-with-a-diode-laser/)

对于能够切割的材料类型和厚度，业余爱好者级激光切割机可能会有一些限制。我们通常在这里谈论二氧化碳 [2] 和基于二极管的机器，如果你想切割非塑料板材，你通常会寻找天然材料，如皮革、织物和薄木材。

但是金属呢？这是一个常见的初学者问题，通常带着一种听天由命的表情问，他们已经知道答案将是一个艰难的“不”。然而，YouTuber [Chad]决定回应一些关于使用高功率二极管激光切割金属板的可能性的评论，用 [一个简单的实验来实际确定什么是极限](https://www.youtube.com/watch?v=lSN5Yz1Xq1Q) 。

使用 XTool D1 Pro 20W 作为试验台，【Chad】尝试了各种材料，包括不同厚度的低碳钢、不锈钢、铝和黄铜薄板。厚度从千分之一英寸到千分之八英寸的钢垫片在适当的空气辅助和速度设置下似乎完全可以切割，较厚的垫片需要多次切割。您肯定可以看到工件过热的影响，导致一些变色和明显的翘曲，但这些问题可以得到缓解。铜和铝根本没有被光束碰到，可能是因为额外的反射率，但我们确实想知道适当的表面处理是否可以改善这种情况。

显然，我们已经看到二极管激光可以对金属产生影响，只需在工件 [上涂抹一点芥末](https://hackaday.com/2021/01/02/laser-etching-stainless-steel-with-mustard/) 似乎就能让打标变得轻而易举。当我们谈到二极管激光器时，你可以从仅仅将这样一个激光模块捆绑到一个台式 CNC 上得到很多好处。
