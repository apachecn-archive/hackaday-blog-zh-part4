# 一个真正的工作乐高 NES

> 原文:[https://hackaday.com/2020/09/09/a-real-working-lego-nes/](https://hackaday.com/2020/09/09/a-real-working-lego-nes/)

对于许多年轻人来说，乐高是进入工程世界的入口，对于更多的人来说，这是一种令人愉快的追求。如今，从比萨斜塔到任天堂备受尊崇的 NES 游戏机，高质量的套件可以用来制作任何东西。[TronicsFix]拿起后一套，[决定它需要功能齐全](https://www.youtube.com/watch?v=9JYseZLR0LI)。

由 2646 块组成的官方乐高 NES 是原版的忠实再现，尽管只有原来的 80%。在按规格制作了套件后，[TronicsFix]注意到没有办法将墨盒放入插槽中。

鉴于这一失败，彻底的重建是有序的。从最初 NES 的内部开始，[TronicsFix]着手建造一个适当大小的底座，并从那里开始工作。支架是为了安装各种组件而建造的，控制器端口做得特别好，视频输出和电源开关稍微有点复杂。官方工具包中的许多化妆品在这里派上了用场，为最终产品提供了符合要求的美感。

最终的结果是一个真实的，功能的乐高案件 NES。[TronicsFix]就是这样演示的，展示的是游戏机在玩《超级马里奥 3》。任天堂游戏机仍然是改装者的最爱；有些人甚至建造了喷火的作品。休息后的视频。
