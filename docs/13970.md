# 这款 Arduino 笔式绘图仪专为速度打造

> 原文：<https://hackaday.com/2022/06/19/this-arduino-pen-plotter-is-built-for-speed/>

我们在这里看到很多简单的笔式绘图仪项目，虽然我们都很欣赏它们，但大多数都有点慢。没关系——只要能完成工作，有时需要的就是缓慢的步伐。但是可以说，把踏板踩在金属上并没有错。这正是[这款基于 Arduino 的超高速绘图仪](https://www.youtube.com/watch?v=wX90X4rVUr8)的意义所在。

正如故事所述，[IV Projects]在构建了一个早期的笔式绘图仪项目后感到需要速度，该项目工作正常，但未能令人兴奋。为了保持绘图仪易于用廉价部件制造，该设计以 Y 轴的“砂砾辊驱动器”为中心——实际上是来回移动纸张的驱动器。和移动它，使用 Dremel 工具砂鼓轻质轴上，以最大限度地提高加速度。事实上，所有的移动部件都尽可能地保持轻便，结果真的显示了——当这台绘图仪在运行时，三个步进器真的在唱歌。

[IV 项目]的设计中有一些非常巧妙的细节。我们特别喜欢提笔机构的工作方式，一个晾衣夹弹簧作为皮带张紧器的意外出现真是一种享受。从一堆被拒绝的原型零件来看，要把这个设计做好需要相当多的工作。如果你想自己制作，打印零件的 [STLs 可用](https://github.com/IVProjects/Engineering_Projects/tree/main/ProjectFiles/Pen%20PlotterV2)。

如果你对速度标尺的另一端感兴趣，看看这个最小的笔式绘图仪。
