# 通过 3D 打印玻璃过滤咖啡

> 原文:[https://hack aday . com/2021/08/13/filtering-coffee-through-3d-printed-glass/](https://hackaday.com/2021/08/13/filtering-coffee-through-3d-printed-glass/)

通常，当我们想到 3D 打印时，我们会想到粘稠的熔融塑料或紫外线固化树脂。然而，有大量的研究正在进行中，围绕着打印内部含有替代材料的特殊浸渍细丝。[Ahron Wayne]一直在研究这些材料，并决定用原型打印为自己制作一款啤酒。

![](../Images/9193f40211ca08804db9e5a2b0e01222.png)

好吃，但是【Ahron】提示你不应该在实验室喝酒。

【Ahron】的实验对象是虚拟铸造厂开发的玻璃浸渍灯丝。这种细丝充满了微小的玻璃颗粒，其想法是可以像任何常规塑料细丝一样打印出来。在那里，它被加热，这就是所谓的脱脂过程，去除印刷中的塑料。然后，在烧结过程中再次加热，将剩余的玻璃颗粒粘合在一起。

这是一个复杂的过程，会导致最终零件的尺寸收缩和多孔性。然而，在一些人可能看到失败的地方，[Ahron]看到了机会。多孔的印刷部分被用来过滤咖啡，借助于一个听起来像水文丘里管的小真空。

[Ahron]指出这不是一种特别有效的煮咖啡的方法，但它确实有效。我们也看到了令人兴奋的钢丝浸渍工作[。休息后的视频。](https://hackaday.com/2021/08/11/3d-printing-steel-parts-at-home-via-special-filaments/)
