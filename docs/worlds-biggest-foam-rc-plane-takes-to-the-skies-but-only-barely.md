# 世界上最大的泡沫遥控飞机飞上天空，但只是勉强！

> 原文：<https://hackaday.com/2022/07/18/worlds-biggest-foam-rc-plane-takes-to-the-skies-but-only-barely/>

当你有一个仓库大小的设备和一个工业大小的数控泡沫切割机时，你会怎么做？显然，唯一的选择是建造巨大的遥控飞机，而这正是[FliteTest]的人与杰出的[Peter Sripol]合作完成的。成功了吗？是的。效果好吗？休息时间下面的[视频让你来评判。](https://www.youtube.com/watch?v=WEKgTXl7xh4)

以用泡沫制造载人超轻型电动飞机而闻名的彼得·斯里波尔被邀请为飞机的设计师。一种非常轻的 EPS 泡沫用于切割飞行表面，而一种密度更大的绿色泡沫板来自当地的家居建筑商店，用于建造机身。

这个建筑一点也不普通，孩子们也参与了建造，尽管视频中没有详细说明。你可以在机翼和机身上的涂鸦中看到他们兴奋的证据——在这方面肯定是一个巨大的成功！至于坐飞机？四个大型发动机提供运动，这几乎不足以让野兽飞行。重心的一次事故要求在最后一刻改变设计，这使得方向舵几乎无用。但是，它确实飞了起来，这是迈向下一个迭代的一大步。就像每一个好的黑客！

如果你想看载人泡沫电动飞机，可以看看[【彼得·斯里波尔】的 DIY 电动超轻型 MK4。](https://hackaday.com/2020/11/02/peter-sripols-diy-electric-ultralight-mk4/)
