# 水壶是流体力学的巧妙应用

> 原文：<https://hackaday.com/2021/12/17/gluggle-jug-is-neat-application-of-hydrodynamics/>

Gluggle 水壶是一个名副其实的东西——它是一个在倾倒时发出响亮、令人满意的咕嘟声的水壶。但是它是如何工作的呢？[【史蒂夫·莫尔德】着手调查。](https://www.youtube.com/watch?v=2IKZIWVHXvo)

[史蒂夫]的第一个计划是切开一个现有的 Gluggle 水壶，看看它是如何工作的，但切割陶瓷可能是困难和耗时的，不对称的设计只会让事情变得更难。相反，他只是砸碎了一个罐子，看看里面是什么样子，然后用透明的激光切割版本复制了基本设计。

设计很简单——咕嘟咕嘟的声音是由于水倒出时气泡进入壶内的封闭空腔而产生的。停止倒酒，空腔中的空气会通过更多的气泡从壶口逸出，发出更大的咕嘟咕嘟声。声音的频率由水壶的高度决定，水壶实质上是一个封闭的管道谐振器。

了解了其中的机制后，制作你自己的 Gluggle 水壶就像在你选择的 CAD 软件中快速设计并以一种食品安全的方式打印出来一样简单。休息后的视频。
