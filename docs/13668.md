# 用 3D 打印笔制作的锤子柔性手柄

> 原文：<https://hackaday.com/2022/05/19/flexible-grip-for-hammer-made-with-3d-printing-pen/>

当谈到在工具上放置柔性把手时，你可能会使用自熔硅胶带或其他类似的产品。然而，[强力打印材料]发现，你可以使用 3D 打印笔和一些柔性细丝轻松创建一个灵活的手柄。

在这种情况下，锤子首先会在木柄上缠绕一层蓝色颜料带。这作为一个基层，以促进良好的粘附力。然后打印一个简单的纸模板，作为在柔性手柄上创建图形的指南。柔性细丝通过 3D 笔输送，首先打印图形的红色和黑色细节。然后，白色柔性细丝被用来制造柔性握把的其余部分。然后，在将第二层添加到顶部之前，使用木材燃烧工具来弄平第一层弯曲细丝。

结果是锤子上的一个灵活的白色把手被牢牢卡住，这可能是由于印刷后塑料冷却时的收缩。我们之前也见过其他一些用 3D 打印制作的创意把手[。休息后的视频。](https://hackaday.com/2022/02/26/3d-printed-climbing-holds-now-with-texture/)
