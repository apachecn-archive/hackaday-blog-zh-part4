# 复古:能斯特灯

> 原文:[https://hack aday . com/2020/06/16/retro tech tacular-the-nernst-lamp/](https://hackaday.com/2020/06/16/retrotechtacular-the-nernst-lamp/)

在主导照明市场一个多世纪后，很容易认为标准白炽灯的发光灯丝是人们发现的唯一将电转化为光的方法。但是许多富有创造力的人提出了不同的设计，其中之一是迷人的能斯特灯，这是我们以前从未听说过的。

如果这个名字听起来很熟悉，很可能是因为接触了[Walther Nernst]的电化学方程，或他的“新热定理”，该定理最终成为热力学第三定律。作为[爱因斯坦]的朋友和最终的诺贝尔奖获得者，[能斯特]也是一个小修补匠，他在 1897 年提出了一种白炽灯的设计，其效率是碳丝灯的两倍。下面的视频来自爱迪生技术中心，详细介绍了这一设计，它使用了一个陶瓷“光杆”，当电流通过它时，它会发出白炽光。不过，在相当热之前，怒视器是不导电的，所以包括了单独的加热线圈，让怒视器在这个过程中启动；一旦强光灯开始导通，这些灯就会被装在灯座上的继电器关掉。

这是一个复杂的设计，但它的效率，加上更好的光谱，以及它不需要真空灯泡的事实，因为 glower 不会像碳或钨丝一样氧化，这使它具有某些优势，让它在早期的电照明市场占有相当大的份额。它甚至是第一台传真机的光源。我们发现这是对当时稀有材料的巧妙利用，并想知道这是否会导致没有真空的真空管。
