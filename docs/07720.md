# 游戏男孩永远玩下去

> 原文:[https://hackaday.com/2020/09/11/game-boy-plays-forever/](https://hackaday.com/2020/09/11/game-boy-plays-forever/)

对于我们这些年龄足够大的人来说，第一代 Game Boy 非常不可思议，但确实有一个主要缺点:电池消耗。在 90 年代，充电电池并不常见，这导致我们大多数人在电源插座旁玩掌上电脑。一些现代人对经典游戏男孩的看法用现代硬件解决了这些问题，但来自代尔夫特理工大学和西北大学的这个小组创造了一个完全不需要任何电池的游戏男孩克隆体，尽管它可以无限期地玩游戏。

这种构建是一种称为“间歇计算”的概念验证，这种计算允许计算机保持处理状态，直到它获得足够的能量来执行下一次计算。Game Boy 克隆产品与最初的 Game Boy 硬件完全兼容，配备了许多微型太阳能电池板，可以收集能量，如果检测到没有足够的能量继续运行，可以停止运行，并将状态存储在非易失性存储器中。这意味着超级马里奥大陆并不完全可玩，但其他不那么动作的游戏可以享受，对游戏性的影响很小。

研究人员指出，他们的能源感知平台在设备中变得普遍并取代电池还需要很长时间，但他们确实认为，不需要持续运行或通电的联网设备将是一个良好的开端。已经有一些低功耗选项可用,它们可以在其他一切都关闭时保持显示器活动，因此希望在不久的将来我们会看到更多节能选项。
