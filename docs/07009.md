# 没有一卡车设备的 DJing 秘密配料是覆盆子酱

> 原文:[https://hack aday . com/2020/07/03/djing-without-the-trumpy-pi/](https://hackaday.com/2020/07/03/djing-without-the-truckload-of-equipment-secret-ingredient-is-raspberry-pi/)

过去的 DJ 时代有一个浪漫的概念——开着一对 Technics 1200s 在书库的两边相形见绌，通过激光色调的黑暗向饥饿的人群泵送电子低音。即使周六傍晚的婚礼上为 50 多岁的人播放无害的轻唱并不符合现实。

旧式 DJ 的问题是，需要大量的物流来转移所有的设备，更不用说唱片集了，所以对一切都数字化的现代 DJ 来说，他们的经营规模确实很幸运。对于一些人来说，这甚至太多了，而[Dennisdebel]已经将 DJ 设备最小化到了一个新的水平，[通过在连接到他的 DJ 控制器](https://github.com/dennisdebel/pi_dj)的树莓 Pi 上运行流行的 mixx DJ 软件。你可以在休息时间下面的视频中看到结果。

这更像是如何在 Pi 上安装一套软件包来实现一个目标，而不是一个特殊的硬件黑客，但正如他指出的，人们的兴趣在于重新获得对过程的控制。DJ 领域由商业产品主导，越来越多的 DRM 和专有云产品，因此这代表了一种夺回过程控制权的方式。如果还不够 hacky，你可以随时加[一个自制的 DJ 搅拌站](https://hackaday.com/2015/11/09/two-turntables-and-no-microphone/)。
