# 用砂纸进行逆向工程

> 原文：<https://hackaday.com/2018/11/29/reverse-engineering-with-sandpaper/>

每隔一段时间，尤其是现在，你会发现一个真正整洁的芯片，没有任何文档。在[大卫]的情况下，这是一个非常酷的 USB 3.0 eMMC/ SD MMC 控制器。使用这个芯片，在一端连接一个 USB 端口，在另一端连接一些内存，你就有了一个完整的网桥。也有司机。有些产品会附带这种芯片。问题是，没有数据表。想要使用这个芯片，[David] [求助于砂纸](https://research.801labs.org/reverse-engineering-4-layer-pcb/)来找出这个芯片的引脚。

带有这种芯片的产品的最好例子是来自 hardkernel 商店的一个简单的板，很高兴带有相当高分辨率的产品照片。在等待这些板被运送的时候，[大卫]追踪最上面的一层铜。这足以让我们对正在发生的事情有所了解，但是真正的工作是在电路板到达之后才开始的。这些照片被放在一个平板扫描仪中，并仔细拍照。

下一步是拆下所有部件，小心测量和分类每个部件。然后，用 200 和 600 粒度的湿砂纸打磨。慢慢地，阻焊层被去除，顶部铜层出现。之后，只需要打磨和扫描，用你选择的图像处理软件将所有的图层叠加在一起。

手工打磨 PCB 以对铜层进行逆向工程需要注意一些事项。首先，它会造成混乱。不过，这是湿/干砂纸，你可以也应该用水来打磨。其次，还要施加压力。我们不确定[大卫]是否拿着砂纸，但是最好的技巧是实际上拿着板子本身。

尽管有一些问题，[大卫]确实得到了每个铜层的照片。在组合这些图像后，他可以为 eMMC 的读者在他的《任天堂 Switch》中画一只鹰。
