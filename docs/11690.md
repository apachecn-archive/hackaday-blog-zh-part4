# 建造魔鬼牙膏火箭发动机

> 原文:[https://hack aday . com/2021/10/21/building-a-devils-牙膏-火箭-发动机/](https://hackaday.com/2021/10/21/building-a-devils-toothpaste-rocket-motor/)

当谈到怪异和古怪的自制火箭实验时，[Integza]让自己相当忙碌。他现在尝试了一种将魔鬼牙膏[重新用于推进的设计。](https://www.youtube.com/watch?v=vVven2CPaaw)

魔鬼牙膏真的和著名的[大象牙膏实验一样，](https://www.scientificamerican.com/article/make-elephant-toothpaste/)只是用更高浓度的双氧水执行。在这种情况下，[Integza]使用 50%的过氧化氢结合高锰酸钾作为催化剂。当这两者结合时，过氧化氢分解成氧气和水,[Integza]用它们来推动滑板。

高锰酸钾催化剂被浸渍到 3D 打印的多孔陶瓷部件中。然后通过压缩空气机制将过氧化氢注射到这种基质中，在那里它分解，产生水和氧气的射流，然后从 3D 打印的火箭喷嘴中喷出，产生推力。

它的效果出奇的好，即使这是建造火箭的一种混乱和非传统的方式。它也比大多数[【integ za】以前的项目](https://hackaday.com/2021/09/17/putting-an-afterburner-on-an-electric-ducted-fan/)少了很多火药味。休息后的视频。
