# 制作自己的量子点

> 原文:[https://hackaday.com/2019/03/21/make-your-own-quantum-dots/](https://hackaday.com/2019/03/21/make-your-own-quantum-dots/)

量子点听起来似乎应该是很酷的东西，但也携带着危险的包袱，有时由镉制成，可能是危险的。它们是什么？本质上，它们是纳米尺度的粒子，小到当高能光击中它们时，光子会被吸收，并以较低的能量状态重新发射。你可以在你的厨房里轻松制作无毒的量子点。除了凉爽的因素，他们可以用作荧光染料，墨水，并可能油漆。[奇异有趣]在下面的视频中解释了这一点。

你不需要太多设备。微波炉、耐热玻璃(硼硅酸盐)容器和手套。你还需要塑料吸管和一个黑光灯——可能是清单上最奇特的两样东西。配料也很普通:醋、小苏打、水和糖。

醋可以让糖分解(或转化)得更快，一旦醋分解完，小苏打就会中和醋。你只剩下果糖和葡萄糖。一旦糖溶液达到合适的 pH 值，剩下的问题就是再次加热以形成量子点。这些点在黑光灯下会发绿光。

据[奇趣]称，整个过程大约需要 10 分钟。除了发光，你还能用它们做什么？我们不太确定。在商业上，量子点在激光器、太阳能电池、光发射装置、甚至生物标记中具有应用(或潜在应用)。然而，我们不知道如何从荧光糖溶液到任何这些设备。如果你想通了，一定要给我们一个[提示](https://hackaday.com/submit-a-tip/)。

我们已经看过在之前制作[不同大小的点。我们不建议处理镉点，但是我们](https://hackaday.com/2017/10/13/carbon-quantum-dots-in-your-favorite-color/)[已经在一个合适的实验室里看到了。](https://hackaday.com/2011/10/12/brewing-up-some-quantum-dots/)
