# 通过电子设备进行更好的辩论

> 原文：<https://hackaday.com/2019/05/12/better-debating-through-electronics/>

这些天看任何新闻专题节目，你会发现事情会很快变得难以控制。嘉宾们通过互相叫嚷和试图打断对手的话题来争夺播出时间。[cutajar.sacha]在工作场所遇到了这个问题，并着手制定解决方案。

结果是[有争议的审议者](https://www.instructables.com/id/The-Debatable-Deliberator/)，它结合了基本的“谈话棒”练习和通过羞辱进行的行为训练。两名参与者每人佩戴一个装有电子设备的头带。只要计时器倒计时，魔球持有者就可以发言。如果他们的对手在此期间说话，他们的头带会用轻轻的耳光斥责他们。如果持有者发言超过指定时间，他们同样会被机械性地扇耳光。

这是一种有趣的方式来帮助警方在两方之间进行讨论，这一切都有可能与三个韦莫斯 D1 ESP8266 板。头带充当客户，而球充当服务器，记录每个发言者违反规则的次数。

随着 ESP8266 等芯片的广泛使用，像这样的 WiFi 项目在过去几年变得更加容易。当然，如果你需要更多的咕噜声，你可以随时升级到 ESP32。
