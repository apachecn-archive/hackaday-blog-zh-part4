# 史上点击量最大的键盘

> 原文:[https://hack aday . com/2020/05/05/the-clickiest-keyboard-ever/](https://hackaday.com/2020/05/05/the-clickiest-keyboard-ever/)

无论你的键盘多么咔嗒作响，没有什么比得上使用打字机的感官体验。打字机发出的声音，从敲击空格键的低沉沉闷的声音，到敲击纸张的断奏声，再到回车时的棘轮声，比起即使是最嘈杂的电脑键盘发出的毫无生气、没有灵魂的咔嗒声，都是一种享受。哦，还有钟声——谁不喜欢钟声呢？

不愿错过真正打字的感觉，[Jatin Patel]发明了这个螺线管驱动的打字机模拟器。第一个版本具有核心功能，一排六个螺线管安装在一根木条上。线圈通过继电器板连接到 Arduino 在他的电脑上运行的 Python 程序读取每一次按键，并告诉 Arduino 启动哪个螺线管。每个听起来都不一样，可能是因为它在电路板上的位置，也可能是因为安装方法的不同。不管原因是什么，这种效果是声音的真实变化，就像一台真正的打字机一样。

第二个版本，在下面的视频中显示，用一个电机将螺线管架移动一步，以模拟打字机的移动托架。最后一个螺线管在该返回车厢时发出铃声，这是用组合扳手作为手柄来完成的。奇怪的妖术，不过还好。
