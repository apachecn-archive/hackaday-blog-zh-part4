# 从游戏传奇中找到秘密信息

> 原文:[https://hack aday . com/2020/07/04/finding-a-secret-message-from-a-gaming-legend/](https://hackaday.com/2020/07/04/finding-a-secret-message-from-a-gaming-legend/)

岩田聪可能因领导任天堂开发 DS 和 Wii 而被人们铭记，这两款系统非常成功，不可否认地将游戏带给了更广泛、更主流的观众。但在 2002 年成为该公司总裁的几十年前，他作为一名开发人员在该行业起步，开发了许多早期的游戏机和电脑游戏。[【罗宾·哈布朗】最近决定挖掘岩田最早的项目之一](https://www.youtube.com/watch?v=QJCwsi6O5xU)，*VIC-20 的星际大战*。

[![](../Images/fecff2470835a2cf6486e8934eaecd7a.png)](https://hackaday.com/wp-content/uploads/2020/06/iwatac64_detail.png)

Finding the message was easy, if you knew were to look.

众所周知，当时年仅 22 岁的岩田在游戏源代码中隐藏了自己的名字和信息。但是[罗宾]想知道这个故事是否还有更多。查看内存中的文本，他注意到这些行实际上是以空终止的。意识到这个信息可能会在游戏开发过程中的某个时候出现在屏幕上，他开始寻找一种方法来触发这个有近 40 年历史的复活节彩蛋。

事实证明，它隐藏在代码中的单个标志后面。只要把它从 0 改成 1，游戏就会显示岩田隐藏已久的信用画面。这证明了该消息最初是打算让玩家看到的，但它仍然没有解释他们应该如何在正常游戏中触发它。

这就是事情变得有趣的地方。当[罗宾]带我们参观了《星球大战》的内部运作时，他解释说，岩田最初打算让玩家点击一个特殊的组合键来勾选复活节彩蛋的启用标志。所有的代码仍然存在于游戏的商业发行版中，但是已经被禁用了。由于岩田的生命在 2015 年因癌症并发症而悲惨地缩短，我们可能永远不会知道他在游戏发布前注释掉有问题的代码的原因。但至少我们现在终于可以看到这个隐藏的信息，来自一个游戏的真正的杰出人物。

上次我们听到[罗宾]的消息，[他发现了一个隐藏在黑胶唱片](https://hackaday.com/2019/11/24/secret-c64-program-found-on-a-christian-rock-bands-vinyl-record/)上的秘密 C64 程序。鉴于他迄今为止的记录，我们迫不及待地想看看他下一步会挖掘什么。
