# 疫情给了路人一扇机器人控制的窗户

> 原文：<https://hackaday.com/2021/09/07/pandemic-gives-passersby-a-window-on-cyborg-control/>

这是什么？[Niklas Roy]和[Kati hyypp]的另一个神话般的创作，将艺术和电子与我们的时代精神和许多回收的零碎东西结合起来？你猜对了。他们在东柏林的工作室曾经是一家零售店，因此有一个很大的展示橱窗。这对一对穿着疫情的艺术家来说似乎是完美的，因为他们可以通过橱窗里展示的东西与社区交流。最近，我们选择称这个互动的电子人宝宝为电子宝宝。

你可能会认为 Cybaby 是一个非常容易被黑客攻击的机器人，但它有一个娃娃头。(它还有一只红眼，真的很好看。)在橱窗里，Cybaby 活了过来，蹒跚学步，背景是今年春夏的几个星期里不断成长和演变的。路人可以加入网络，用智能手机从外面控制 Cybaby，让它四处走动，按下各种按钮，改变它的环境，并触发一些传感器。Robosapien 已经存在了大约 20 年，所以已经有 Arduino 代码可以模拟它的遥控信号。[Niklas]和[Kati]使用 NodeMCU (ESP12-E)向机器人的红外输入发送脉冲。

回到滑稽的时代精神前沿，有一家发廊、一家便利店和一家跳舞的夜总会，跳舞需要首先成功地通过测试中心(自然)。哦，还有夜店隔壁有个实验室，Cybaby 无论怎么试怎么哭都进不去。休息之后来看看。

出于某种原因，Robosapien 的帖子很少，所以这里是[Niklas]和[Kati]在 Cybaby 世界之前在他们的窗口中看到的东西——一个非常酷的笔式绘图仪，可以打印出走过的人发送的信息。
