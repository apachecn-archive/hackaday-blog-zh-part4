# 如何在平面上实现滚花

> 原文:[https://hack aday . com/2022/10/15/how-to-achieve-on-a-plant-surface/](https://hackaday.com/2022/10/15/how-to-achieve-knurling-on-a-flat-surface/)

滚花是一种加工工具手柄和其他硬件的流行方式，通过在金属上轧制交叉线的图案来提供更好的握持和更有吸引力的光洁度。这通常是在车床上完成的，但如果你有合适的工具，也可以在平面上完成。当然，[你可以自己做一个](https://www.youtube.com/watch?v=padBm0aBYXM)。

建造很简单，是基于用一块坚固的钢制造一个特殊的载体。这是一个长杆，中间有一个空间，可以放两个轮子。一对现成的滚花轮，然后安装在酒吧，与凹头螺栓作为轴。

随着工具的完成，这是一个简单的问题，然后将载体棒安装在车床上，并在一个平坦的工件上来回运行。工件被来回滚动以允许轮子完成它们的工作，同时也被水平移动以允许整个平面被加工。

一个漂亮的滚花真的可以提升任何工具或其他金属工艺的形式和功能。[我们之前也探索过如何打造自己的滚花旋钮](https://hackaday.com/2013/08/06/replacing-knurled-thumb-screws/)。
