# 可爱的 NFL 积分榜追踪器使用小迷你头盔

> 原文：<https://hackaday.com/2022/08/18/cute-nfl-standings-tracker-uses-little-mini-helmets/>

如果你是一个铁杆体育迷，没有什么比从上到下紧跟联赛发展更让你喜欢的了。[Kiu]有一个家庭成员很喜欢 NFL，所以开始为他们制作一个非常酷的梯子追踪器。

由于来自 MySportsFeeds.com 的付费直播数据，追踪器显示 NFL 联赛表有 10 分钟的延迟。当更新到来时，代表比赛中每个队的微型头盔被移动到正确的顺序。头盔放在小塑料标签上，使移动它们变得容易、可靠和可重复。这款追踪器使用 3D 打印机领域熟悉的部件制造，在 Arduino Nano 的指挥下，依靠步进器和 V 型轨道进行线性移动。

这是一个在任何游戏室都会看起来很棒的建筑，我们打赌一个放大的版本会在高档体育酒吧里看起来很有商业价值。老实说，联盟的顶级四分卫都会争取尽快拥有这样一个机会。这并不是说，在经历了一场痛苦的失败后，回到家，戴着你的团队的头盔在冲浪板上快速滑行就不会感到刺痛。

这些年来，我们也看到了一些其他有趣的体育追踪项目。休息后的视频！
