# 从零到 LED 立方体，不到 7 个月

> 原文:[https://hack aday . com/2020/05/22/从零到 led-cube 不到七个月/](https://hackaday.com/2020/05/22/from-zero-to-led-cube-in-less-than-seven-months/)

我们知道 LED 视频立方体已经过时了，但这并不意味着我们不喜欢看到它们。毕竟，任何包含超过 24，000 个 led 的项目都一定会令人印象深刻。但是关于[【迈克·坎】的独立 LED 立方体](https://www.youtube.com/watch?v=l_GSGFhnWwo)更有趣的一点与他选择的完成产品的过程有更多的关系。

有两种方法可以着手一个新项目，尤其是当你像[Mike]一样是硬件黑客新手的时候。一种是双脚跳进去，看看会发生什么，不管是好是坏。另一种方法是从一个起步项目开始慢慢适应，找出你的局限所在，然后逐渐克服它们。[Mike can]在他的 LED 立方体项目中明智地选择了后一种方法，从一个 LED 沙玩具开始。单一的 64 x 64 LED 面板更容易使用，并让他能够快速处理这些硬件，以及驱动它所需的代码。下面的视频讲述了将该项目放大六倍以制作立方体的故事，这一过程有其速度障碍。不过，一切最终都很好地结合在一起，让[迈克]开始软件方面的工作。这就是这个项目真正的亮点——运行立方体的智能手机应用程序真的很棒，动画也很棒。

很明显，在[Mike]的立方体上有新功能的空间，所以希望他能抽出一些时间来做一个更好的构建。为了获得灵感，他可能想看看[这个侧面滚动的恶魔城立方体](https://hackaday.com/2019/11/04/ambitious-led-cube-provides-endless-video-game-scrolling-plays-castlevania/)，或者读一读[用于 LED 立方体的 OpenGL](https://hackaday.com/2019/04/30/opengl-shaders-and-an-led-cube/)的细节。
