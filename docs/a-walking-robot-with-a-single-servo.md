# 单伺服步行机器人

> 原文：<https://hackaday.com/2020/09/16/a-walking-robot-with-a-single-servo/>

我们都有过这种经历——你看到有人在 YouTube 上做了很酷的事情，你就必须试一试。对[孤独者冲浪者]来说，放下一切然后建造碰巧是一个小的四足步行机器人，它运行在一个单一的伺服系统上。虽然这可能很简单，但真的没有什么比看到你创造的机器人迈出第一步更好的了。

[lonesoulsurfer]的步行器主要由废铝和其他废弃部件制成，如衣架、回形针和香蕉插孔中的金属碎片。对于第一次建造的人来说，破旧弯曲的车身可能是最难模仿的，但任何允许用螺丝和螺栓固定的坚固底盘都应该可以工作。还有，不要指望马上就能见效。这将需要一点调整，以获得正确的步态，但这都是乐趣的一部分。为连续旋转修改 180°伺服系统也是如此。

我们真的很喜欢这个机器人走路的方式——它像一只长长的牛头犬一样四处闲逛，看起来它几乎可以应对任何地形。休息后观看它的行走，并留下来观看构建视频。

没有微控制器的简单机器人。如果你从未听说过光束机器人，[试试这个圆腿机器人](https://hackaday.com/2012/05/18/a-rocking-and-walking-beam-robot/)。
