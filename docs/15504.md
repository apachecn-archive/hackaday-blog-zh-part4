# 神经辐射场

> 原文:[https://hack aday . com/2022/11/30/nerf-neural-radiance-fields/](https://hackaday.com/2022/11/30/nerf-neural-radiance-fields/)

制作叙事电影变得越来越容易。曾经需要一个工作室的东西，现在对于专注的业余爱好者来说触手可及。而[神经辐射场](https://www.youtube.com/watch?v=YX5AoaWrowY)让这一戏剧性的步骤变得更加容易。[走廊工作人员]的家伙给了一个早期的窥视。

拍摄和剪辑已经到了 easy 的手机和笔记本电脑阶段。但是布景、服装、演员、灯光等等并没有变得便宜很多，制作自己的短片仍然是一个大项目。

输入 3D 图形。有了一台好的游戏笔记本电脑，任何人都可以在 Blender 中制作逼真的场景，并在其中放置真人演员。但是这需要大量的技巧和工作。通常，你正在制作的场景是真实的，但是你不能得到许可去拍摄或者把演员、道具、工作人员等带到片场。

一项名为“神经辐射场”的新技术 NERF 已经大大减轻了这种头痛。该软件从场景的视频开始，机器学习“辐射场”——场景如何反射光线的模型，而不是制作场景的 3D 模型并使用它来预测什么到达相机。

如果使用辐射场来预测落在 3D 模型上的光线，软件可以渲染 3D 模型，就像它在场景中被照亮一样。因此，如果你的演员站在一堵红墙附近，红色的倒影会显示在他们的脸上。

结果是戏剧性的——演员的现场视频通过摄影测量转换为 3D 模型，并放入辐射场中，看起来像在现场拍摄的现场动作视频。放入场景中的 3D 模型看起来异常真实。跟踪演员视频的摄像机运动可以应用到辐射场中的摄像机，因此摄像机可以在拍摄过程中移动。

元素是 CG 对象，因此可以移动或缩放。在视频中，他们用这个来插入一个演员，作为一个矗立在仓库上的巨人。您可以在事后调整相机运动，这样就不会再有抖动的相机移动或后悔。

对于低成本电影制作人来说，这是一个强有力的新工具。如果你喜欢 VFx，这是一个真正的进步。NERF 是新的，但是我们已经报道过很多次摄影测量学，包括这个整洁的转盘“甜甜圈”版本。
