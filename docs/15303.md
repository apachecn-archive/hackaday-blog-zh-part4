# 准将 Datasette 做自己的校准

> 原文:[https://hack aday . com/2022/11/07/commodore-dataset-does-it-own-calibration/](https://hackaday.com/2022/11/07/commodore-datasette-does-its-own-calibration/)

啊，敬爱的 64 号准将。“有史以来最畅销的计算机系统”。爱好者们保持着这个梦想，今天仍然在为它制作软件。这就给我们带来了使用这种旧设备的问题。当你把 Petscii 机器人拷贝到磁带上，并试图快速加载它时，你的机器可能总是无法加载程序。好了，是时候拿出球杆头和外用酒精了，好好清洗一下读数头。但是如果这样做没有用呢？你可能会遇到另一个问题，比如[磁带速度漂移](https://janderogee.com/projects/1530_tapespeed/1530_tapespeed.htm)。

有几种不同的方法来测量当前的磁带速度，以正确拨号。最好的可能是已知音调的参考磁带。只需连接您的频率计或数字示波器，并在调整锅拨号，直到您的数据集产生预期的音调。哦，你没有频率计数器？好消息是，[简·德罗吉]为你提供了一个解决方案。看，你已经把你的数据集连接到一个非常好用的频率计数器上了——你的 Commodore 计算机。他推出了一个免费程序，可以计算一秒钟内来自数据集的脉冲数。因此，播放一个参考磁带，运行程序，并在你的数据集甲板拨号。简单！休息后留下来，认真地演示一下问题和解决方案。
