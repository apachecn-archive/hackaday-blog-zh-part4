# 建造你自己的消声室

> 原文：<https://hackaday.com/2018/12/14/build-your-own-anechoic-chamber/>

对于专业级的录音，你需要专业级的设备。麦克风和混音设备是显而易见的必需品，还有一台装有正确软件的好电脑。但是一旦你有了这些东西，你还需要一个地方来记录。如果没有一个好的声学空间，你的录音中会有各种各样的反射和伪像，如果你不能租一个录音室，你可以随时建立自己的消声室。

虽然可以在房间的墙壁上铺地毯或随机将鸡蛋箱泡沫粘在墙上，但[Tech Ingredients]根据商业可用的解决方案测试了一些不同形状、大小和材料的自制面板。为了做到这一点，他使用了一个特殊的封闭扬声器指向材料，和一个麦克风来测量声音反射。测试显示，自制吸声板的效果很有希望，其成本只是现成吸声板的一小部分。

从那里，我们看到如何制造和组装这些面板，以获得最佳性能。在处理声学问题时，即使是用于将所有东西粘合在一起的胶水也会改变材料的属性。我们还看到了一些其他的节省成本的方法，这些方法在建造面板时也有帮助。而且，虽然这个建筑侧重于声学消声室，但不要忘记还有使用相同原理的电磁辐射消声室。
