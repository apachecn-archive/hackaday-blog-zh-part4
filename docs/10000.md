# 一台 50 美元的 CNC

> 原文：<https://hackaday.com/2021/04/29/a-50-cnc/>

理论上，制造一台数控机床没什么大不了的。把一个钻头挂在电机上，用一些丝杠和步进电机移动电机。简单。但是，当然，细节决定成败。[DAZ]做了一个好看又便宜的钻机，它可能不是世界上最精确的 CNC，但看起来它做得足够好了，他声称他花了大约 50 美元买了它。下面的视频展示了它做的一些工作，看起来还不错。

这不是一个雨天下午的项目。你需要砍一些木头，3D 打印许多零件。驱动器使用 M8 螺纹杆。电子学只是一个运行标准软件的 Arduino。

踏步机看起来很轻，我们想知道是否值得将它们换成更结实的，而不是修改用于双极操作的踏步机。尽管如此，50 美元的结果看起来还是不错的。775 主轴是另一个地方，你可能会花多一点，得到更好的东西。非印刷线性轨道，和一个更好的螺丝？关键是你已经有了一个建立的基础。

我们以前见过用废料制成的廉价 CNC。如果你觉得 Arduino 对你的 CNC 来说功率太低，试试 32 位的。
