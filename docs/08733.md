# V12 克尔维特获得电子仪表模块

> 原文:[https://hack aday . com/2020/12/22/V12-corvette-gets-electronic-gauge-mod/](https://hackaday.com/2020/12/22/v12-corvette-gets-electronic-gauge-mod/)

[韦斯利·卡根]正在制造一辆换了 V12 引擎的轻巡洋舰。大部分传动系统将被完全取代，这意味着驱动机械速度表和转速表的组件将不再出现在最终的汽车中。相反，[【卫斯理】想出了自己的电子规转换来做这项工作。](https://www.youtube.com/watch?v=eDLsrjLIHxM)

这是一个尊重赛车原始美学的建造，重新使用原始的仪表，但驾驶它们是不同的。取代原来的机械传动装置分别从传输和分配器，车速表和转速表，而是获得伺服安装在后面的 3D 打印齿轮系。里程表也有自己的连续旋转伺服系统。Arduino Nano 用于驱动伺服系统，使用来自 GPS 模块和汽车点火系统的数据。

任何希望 3D 打印零件的人都可以使用这些文件来修改他们自己的量规。我们迫不及待地想看到压力表最终安装后的样子。我们可以想象一些转换率或更新速度的初期问题，但我们确信没有什么是[韦斯利]通过一些修改不能解决的。定制仪表是我们在这些部件上见过几次的东西；[这种数字设置对发动机数据](https://hackaday.com/2019/05/29/turbo-subaru-gets-diy-gauges/)特别有用。休息后的视频。
