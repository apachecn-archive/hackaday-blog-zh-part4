# 笔式绘图仪非常简单

> 原文:[https://hackaday . com/2021/08/06/pen-plotter-is-as-simple-as-it-can-get/](https://hackaday.com/2021/08/06/pen-plotter-is-about-as-simple-as-it-can-get/)

当然，我们看到了相当多的绘图仪和其他运动机器，但是【DAZ 项目】的[看起来非常简单。当然，Arduino 和 CNC shield 已经过时了。但是一些 3D 打印的滑轮和一个看起来非常简单的核心 XY 排列看起来这可能是一个非常快速的构建。](https://create.arduino.cc/projecthub/ddavidezeta05/drawing-robot-cheap-and-easy-8956b2?ref=platform&ref_id=424_recent___&offset=5)

你可能会问；如果你有一台 3D 打印机，为什么不在上面装上一支笔，然后就此收工呢？当然，你可以这么做，但这有什么意思呢？此外，那也会占用你的打印机。你可以在下面看到这个项目的视频。

这个建筑看起来很便宜，只用了几个轴承和马达。大部分看起来像你可能挂在周围的东西，如绳子和橡皮筋，还有一些废木料。我们喜欢运动机制中的游戏如何允许 RC 伺服轻松地将整个笔从纸上提起。当然，笔或伺服喇叭在纸上滑动，所以我们不认为你可以轻松地挂一个激光模块来制作雕刻机，尽管我们不怀疑你可以解决一些问题。

我们可以看出[DAZ]喜欢制作基于动作的项目。我们还认为这让我们想起了最近看到的[白板绘图仪](https://hackaday.com/2021/01/03/blackboard-becomes-tidy-pen-plotter/)。
