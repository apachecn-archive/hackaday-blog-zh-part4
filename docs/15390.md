# 桌面周围的微型遥控卡车和拖车马达

> 原文：<https://hackaday.com/2022/11/18/tiny-rc-truck-and-trailer-motors-around-tabletop/>

大多数遥控车复制现实世界的赛车或户外探险的幻想越野车。[diorama111]是建造微型桌面漫游模型的专家，[建造了这个精致的微型半挂卡车和拖车。](https://www.youtube.com/watch?v=w7TURoGE7To)

基于 1/150 比例的卡车和拖车模型，构建从拖拉机单元开始。它被拆开了，它的塑料轮子是在一台微型车床上加工的，这样它们就可以安装由 O 型圈材料制成的防滑橡胶轮胎。前轮有轮毂，安装在电机驱动的螺旋式转向组件上。光电探测器用于帮助自定心。后轮轴装有齿轮传动系统，由一个小型 DC 发动机驱动。多个齿轮级用于为牵引拖车提供足够的扭矩。该模型的远程控制是通过蓝牙实现的，其中一个 ATtiny3217 与电机驱动器一起运行表演。

微控制器还运行一整套驱动、尾灯和指示灯。拖车装有红外线接收器和自带电池。它从牵引车上的红外 LED 接收信号，告诉拖车何时打开尾灯和指示灯。

在 1:150 的比例下，没有太多的建模者在 RC 空间中工作。不过，这里已经有了模型，我们以前也展示过丰田皇冠的原型。
