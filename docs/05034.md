# 用空调压缩机制造发动机

> 原文:[https://hack aday . com/2019/12/04/building-an-engine-with-an-a-c-compressor/](https://hackaday.com/2019/12/04/building-an-engine-with-an-a-c-compressor/)

空调压缩机并不完全是普通黑客的垃圾箱的主要部分。通常情况下，它们要么被安装在汽车上做它们原来的工作，要么被放在工作台上翻新。然而，有了正确的改装，就有可能把它变成一台正常运转的内燃机。

构建从拆卸压缩机开始，压缩机包含三个双面活塞。外壳钻有端口，以允许气体流入和流出气缸，以及从活塞的一侧转移到另一侧。丙烯酸端板安装在组件上。一端作为进气歧管，向气缸输送空气和燃油。另一侧作为气缸盖，安装火花塞。所有的东西都用丙烯酸管连接起来，一小块方形的丙烯酸管就变成了汽化器，提供空气燃料混合物。点火由飞轮运动触发的线圈控制。

由于丙烯酸歧管破裂，在最初的失败后，一个更强的零件被制造出来，发动机爆发出生命。丙烯酸端盖可以很好地观察燃烧过程。我们很想看到一个 dyno 图，显示该装置输出了多少功率和扭矩，或者看到它挂在自行车或手推车上。
