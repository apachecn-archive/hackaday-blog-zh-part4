# 用一点点热量修复冰柜的设计缺陷

> 原文：<https://hackaday.com/2021/12/23/fixing-a-freezer-design-flaw-with-a-little-bit-of-heat/>

作为意外后果定律的一个光辉例子，[Lou]展示了某些类型的冰柜/冰箱组合如何在寒冷的环境中无法工作，例如冬天的车库。正如[Lou]在他的视频中指出的[(也在休息后链接)-使用他自己车库中的冷冻单元-问题在于将温度传感器放在冰箱部分的设备，但从冷冻部分开始循环冷空气。](https://www.youtube.com/watch?v=E4hoLL6-x4M)

这在室内温度对人来说很舒适的家庭环境中非常有效，因为冰箱会由于外界的热量而不断轻微升温，从而触发冷却循环，并确保冷冻室保持凉爽。当放置在，比如说，一个车库里，当它接近结冰时，冰箱部分将不会变热，因此不再有冷却循环被触发。

显然，冷冻室应该保持比大约 0℃低得多的温度，所以冷冻室可以加热到足以使冷冻物品解冻。正如[Lou]所展示的，简单的解决办法是在冰箱里增加一个持续的热源，比如让冰箱的灯一直亮着。

除了对缺乏能效的明显抱怨和这种组合设备乐观的传感器设计之外，这是一种相当简单有效的修复方法。即便如此，也许把这样一个组合单元换成一个在冷冻区有专用温度传感器的单元，可以防止类似这样的头痛。
