# 胶合板自行车使框架建筑更容易

> 原文：<https://hackaday.com/2022/11/04/plywood-bicycle-makes-frame-building-more-accessible/>

自行车框架表面上很简单，但如果你想自己制作一个，它会很快变得复杂。铜焊和焊接往往比知道如何将东西栓在一起更不常见，所以[arqui MAA]为我们带来了 [OpenBike](https://openbike.cc/download/) 让更多人可以使用这一过程。

OpenBike 是一套为 CNCs 和 3D 打印机设计的开源文件，它使用容易获得的材料，如薄板胶合板，来制作一辆坚固的自行车，尽管它看起来非常规。像许多其他消费品一样，大多数自行车车架目前都是在亚洲制造的。这可以实现规模经济，但将当地人排除在设计过程之外。通过使用更简单的工具，OpenBike 允许对特定地区可能需要的功能进行更多的本地指导。

将一小部分出行转变为更积极的交通方式是降低碳排放的重要组成部分，因此让自行车成为更具吸引力的交通工具总是受欢迎的。在一个地区可能重要的东西在另一个地区可能是多余的和昂贵的(例如，在丘陵地区的多个齿轮)。OpenBike 可能特别有用，可以在为更大批量的自行车投入更传统的框架构建技术之前，为特定区域快速原型化不同的功能集。
