# 3D 打印弯曲显示精确的行动

> 原文：<https://hackaday.com/2022/06/03/3d-printed-flexture-shows-precision-in-action/>

这里有一个更老但很棒的视频，虽然很短，却很有启发性。[Topias Korpi]展示了一个 [3D 打印弯曲，一端有一个千分表，另一端有一个 M3 螺钉](https://www.thingiverse.com/thing:1424169)。随着螺钉的转动，千分表稳定移动，螺钉和千分表之间的移动比例大约为 15:1。0.01 毫米的稳定偏转很容易拨入，它的整洁看到它的工作，而弯曲本身没有显示出明显的运动。分页符下方嵌入了一个演示，时长不到一分钟，所以可以观看一下，或许可以获得一些想法。

Flexures 是奇妙的设计[，能够实现各种各样的物理功能](https://hackaday.com/2021/12/14/fabulous-flexure-mechanism-makes-for-resetting-cat-calendar/)，正如【Topias】的演示所示，它们可以成为[3D 打印的自然补充](https://hackaday.com/2021/06/22/print-your-own-flexures/)。事实上，弯曲是 [JWST 的镜面致动器](https://hackaday.com/2022/02/08/working-model-reveals-amazing-engineering-of-webbs-mirror-actuators/)的设计和功能的重要组成部分，这些致动器负责对太空望远镜的 18 个镜面部分进行令人惊讶的微小调整。
