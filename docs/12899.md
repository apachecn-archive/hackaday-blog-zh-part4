# 杂货店火箭燃料:不要在家里尝试这个！

> 原文：<https://hackaday.com/2022/02/25/grocery-store-rocket-fuel-dont-try-this-at-home/>

似乎每当提到火箭科学的话题，话题就会迅速转移到火箭燃料上。正如在休息时间下面的精彩【Scott Manley】[视频中所讨论的，有许多火箭燃料可以以某种方式、状态或形式在你当地的杂货店或酒类商店找到。该视频本身是对犹他州一些大学生的反应，他们在煮火箭燃料时发生爆炸，导致疏散。](https://www.youtube.com/watch?v=1Na3reZIYsw)

[斯科特]自己推断他们正在烹饪的燃料是[火箭糖](https://en.wikipedia.org/wiki/Rocket_candy)，一种糖和硝酸钾的挥发性混合物，已知有时会爆炸。事实证明，这种混合物在你所在的地区甚至可能是不合法的，因为它可以用作火箭燃料，也可以用于其他爆炸的东西。

那么，你当地的超市里还有什么可以用来进入轨道的呢？[斯科特]谈到不同种类的酒精，气体，清洁剂-所有可以用作火箭燃料的东西。他还谈到了你不想在家做这件事的所有充分理由。

如果这种类型的事情让你的分子兴奋，你可能会喜欢我们最近发布的关于使用另一种[杂货店主食来拯救火星殖民者免受重力](https://hackaday.com/2022/01/24/saving-martian-colonists-using-table-salt-and-rocket-science/)的影响。
