# 奇妙的可折叠可印刷十二面体

> 原文:[https://hack aday . com/2022/06/20/wonderful-foldable-printable-十二面体/](https://hackaday.com/2022/06/20/wonderful-foldable-printable-dodecahedron/)

以[GeekMomProjects]出名的 Debra Ansell 想出了一个简洁的 3D 设计，它可以平面打印，然后折叠成每个人都喜欢的柏拉图式立体:D12。

你为什么想这么做？好吧，折叠你的 3D 打印给你一个“免费”的第三维空间，而不需要使用所有的支撑材料。在这里，十二面体的所有外表面都是平贴着构建板印刷的，这可能是印刷中最好的一面。在最终的形状中嵌入 led 可能很容易，因为它们都在一个平面上。说到 led，我们有点期待在这里看到它们，因为黛布拉的座右铭是:“[led 改善一切](https://hackaday.com/2022/01/20/remoticon-2021-debra-ansell-connects-pcb-in-ways-you-didnt-expect/)”——这部分取决于你。

黛布拉指出，她喜欢 PETG，而不是 PLA，因为它在薄印刷铰链上有额外的强度，我们敢打赌，你的打印机的公差需要精确到将整个东西固定在一起的夹子上。(我们很想涂一点超级胶水。)

正如黛布拉的项目一样，这里展示了一些创造性的解决方案，可以帮助你决定是需要 D12 还是 D20，所以来看看吧！

谢谢[彼得]的提示。
