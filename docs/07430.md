# 爆米花口袋电脑开源

> 原文：<https://hackaday.com/2020/08/11/popcorn-pocket-p-c-open-sourced/>

如果你错过了你可以得到一个运行 Linux 的管理器的日子，你可能会对爆米花电脑的[Pocket PC .](https://blog.popcorncomputer.com/2020/08/05/pocket-p-c-open-sourced/)感兴趣，这是最近在 GitHub 上开源的。但是，在您开始构建之前，有几件事情您应该知道。

首先，文件未经测试，因为第一个单元还没有发货。此外，虽然原理图看起来很完整，但没有实际的材料清单，PDF 文件中的 PCB 层可能不太容易复制，因为它们只是一系列图像，每层一个。您可以在下面看到该设备的概述视频。

尽管我们还没有看到软件，但信息还是在那里。该设备本身很有趣，内置键盘。规格相对简单。运行频率为 1.2 GHz 的四核 ARM，2GB 内存和 32GB eMMC。IPS LCD 不到 5 英寸，分辨率为 1920×1080。有一套常见的连接器和接口，你可以得到一个版本，包括 LoRa。

我们希望爆米花将继续在设备上发布信息，并将为设备提供足够的软件信息，以真正实现开源。当然，我们中的大多数人无论如何都会买一个，但是如果你想要的话，知道来源就在那里是很好的。

正如我们之前提到的，[设计 Linux 板](https://hackaday.com/2018/08/12/build-your-own-linux-single-board-computer/)不像以前那么难了。尽管如此，T2 仍然面临一些挑战。
