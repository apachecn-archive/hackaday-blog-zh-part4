# 2022 年科幻竞赛:发光的 LED 立方体制造迷人的艺术品

> 原文：<https://hackaday.com/2022/04/20/2022-sci-fi-contest-glowing-led-cubes-make-captivating-artifacts/>

LED 立方体曾经是 IO 掌握中的一项练习，需要多路复用技巧来驱动许多 LED 阵列。采用 RGB 只会增加挑战。这个来自[DIY GUY Chris]的构建展示了这些天来有多么容易，当每个 LED 都有一个智能可寻址控制器时，[并作为一个伟大的科幻道具启动。](https://hackaday.io/project/184544-glowing-rgb-led-magic-cube-ws2812)

是的，该建筑依靠古老的 WS2812B 可寻址 led，焊接在立方体六个面的每个面上的 5×5 网格中。主持节目的是 Raspberry Pi RP2040 微控制器，在这里它是作为一个独立器件而不是以开发板的形式采购的。片上有一个 SPI 存储器芯片用于编码，还有一个 USB-C 连接器用于编程。信号通过沿着定制 PCB 边缘的焊接连接在立方体周围传递，定制 PCB 构成了固体的表面。

坐在它的 3D 打印支架上，这个立方体发出明亮的光，同时汲取整整 2 安培的能量。[Chris]编写了各种各样的动画，从简单的彩色呼吸程序到疯狂的炫目动画，肯定会吸引任何来偷你的魔法发光神器的赛博朋克小偷。

不过，如果你不喜欢直角棱镜，你可以考虑给自己造一个发光的 D20。休息后的视频。
