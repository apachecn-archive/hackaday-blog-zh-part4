# 100MHz 和 200 美元的差分探头时钟

> 原文：<https://hackaday.com/2021/12/12/differential-probe-clocks-at-100mhz-and-200/>

[Voltlog]经常查看有趣的测试设备，在下面的视频中，他回顾了一些在爱好实验室中不太常见的东西:[差分示波器探头](https://www.youtube.com/watch?v=Xc6HPzJ5k4c)。这些通常都很贵，但视频中的 Micsig 探头价格不到 200 美元。当然，问题是你需要差分探头做什么？

典型的示波器探头有一个直接连接到实际接地点的接地线。如果您试图测量某个部件上的电压高于您希望的对地短路电压，这可能会导致问题。它可能会损坏您的被测设备、您的示波器或两者。

正如他在视频中解释的那样，这种调查不是传统意义上的孤立，而是防止了问题的发生。探针由 USB 供电，这可能看起来很奇怪，但[Voltlog]指出，你可能不会经常使用这种电池，所以在它搁置的几个月内会变坏的电池并不是一个好主意。有道理。

最后，您可以看到探头在工作，测量开关电源中的信号，这是传统探头难以测量的。如果你更愿意建立自己的探测器，[从这个进入 Hackaday 奖](https://hackaday.com/2021/08/25/this-modular-differential-probe-shows-great-attention-to-detail/)开始。或者，[试试这个比较老的设计](https://hackaday.com/2016/03/31/diy-active-sub-ghz-differential-scope-probe/)。
