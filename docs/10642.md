# 把我们的忍者刀放下！

> 原文:[https://hackaday.com/2021/07/07/leggo-my-nintoaster/](https://hackaday.com/2021/07/07/leggo-my-nintoaster/)

如果你是今天从未见过烤面包机模型的 10，000 名幸运儿中的一员，[伙计，我们很高兴你能先看到这个](https://www.instructables.com/Nintoaster-Build/)。[Dizzle813]发现了一个闪亮的旧 Sunbeam 烤面包机，看起来就像我们长大的那个。虽然原创者做了一个 build 视频，但是解释上还有改进的空间，而且有人更喜欢阅读，反正。这个方便的指南参考并建立在[呕吐锯]的原始 Nintoaster 视频上。

[Dizzle813]真的让难的部分看起来很容易，像这样的构建似乎大部分都是难的部分。除非你找到确切的 Sunbeam 年份，否则你必须根据需要编排内部结构以适合你的烤面包机。最困难的部分可能是将 72 针连接器连接到 NES 主板上，但[Dizzle813]成功地使用 22 AWG 实芯线完成了它，并且仍然让所有东西都可以弯曲和装配在一起。尽管如此，他们还是折断了一个引脚，试图将它轻松地插入到 perfboard 中，但在连接器上切一个孔并插入一个 bodge 线替代物就可以了。

[![](../Images/24dfd2a0e7e204b600d990443d136cc4.png)](https://hackaday.com/wp-content/uploads/2021/07/nintoaster-inner.jpg) 我们非常喜欢它的外观和操作方式，尤其是杠杆式电源按钮和内部的六个橙色 led，通过烤箱旋钮控制亮度。请务必在休息后观看演示。

当事物被构建到其他事物中时，这不是很棒吗？例证:[这台打印机](https://hackaday.com/2019/05/15/comprinter-hides-a-laptop-inside-a-printer/)里藏着一台笔记本电脑。
