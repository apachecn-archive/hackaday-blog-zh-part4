# 空罐升级为便携式扬声器

> 原文：<https://hackaday.com/2018/08/28/empty-can-upcycled-into-portable-speaker/>

我们并不是建议你去翻垃圾找空罐子，但是如果你在自己家里已经有了一些空罐子，你当然可以做得比把它们变成你的电子项目的独特外壳更糟。当然，总比坐在垃圾堆里好。

[![](img/e92dac5768bdcc64a9cb6706ebebcc27.png) ](https://hackaday.com/wp-content/uploads/2018/08/canspeaker_detail.jpg) [这个来自【罗宾·哈特利】的黑客将一个空的吉百利热巧克力罐变成了一个便携式扬声器](http://hartleyhacks.com/making/the-smooth-and-creamy-hot-chocolate-speaker/)这肯定会引起一些关注。但不要被愚弄:这个项目投入了惊人数量的工程技术，在易拉罐内部形成了 3D 打印结构。即使你不太想把你的下一个项目变成一堆垃圾，也有一些东西可以说一下在这个版本中所有的东西是如何专业地结合在一起的。

这种构建的关键是 3D 打印的“骨架”，它将扬声器和电路板固定在适当的位置。一个特别好的地方是[Robin]为扬声器设计的支架:由于它没有法兰可以连接，他制作了一个两件式夹具，用螺丝固定在扬声器的后部并固定住。

你可能会奇怪，为什么像[Robin]这样精通 CAD 和 3D 打印的人可能会想用一个空罐子作为外壳；他当然可以设计并打印一个案例？毫无疑问。但是这里的目标是再利用那些本来是垃圾的东西，这偶尔意味着走“风景优美”的路线。

为了让这个概念更上一层楼，[看看我们最近讨论过的升级音箱](http://hackaday.com/2018/08/20/an-upcycled-speaker-box-with-hidden-features/)。我们已经看到一些华丽的家庭音频构建，开始是在路边发现的，但取决于你有多幸运，[这几乎就像作弊](https://hackaday.com/2016/02/21/mid-century-modern-speaker-from-90s-road-trash/)。
