# 零件短缺迫使这种递归时钟的创造力

> 原文：<https://hackaday.com/2021/07/25/parts-shortage-forces-creativity-for-this-recursive-clock-of-clocks/>

我们最近已经看到了很多元时钟——一种数字时钟，它的显示是由一系列单独的模拟时钟的指针构成的。它们看起来很棒，我们肯定见过一些很棒的例子。

但是在这个供应紧张的时期，并不总是能够收集到一个人全面建造所需要的零件。令人高兴的是，这并没有阻止[Erich Styger]只用他的 13 个定制双轴步进器模拟运动来执行[这个圆形多元锁](https://mcuoneclipse.com/2021/07/11/round-metaclockclock/)。通常，他的钟表使用的[是机芯数量](https://hackaday.com/2021/03/19/clock-of-clocks-adds-light-pipe-hands-for-beauty-and-function/)的两倍，他将机芯排列成矩阵，这样指针就可以定位成虚拟的七段显示。通过将动作排列成圆形，光导管指针可以模仿模拟钟面，或表演任何[Erich]标志性的“间奏曲”动画，每一个都优雅而迷人。看看这个迷人的递归时钟在下面的视频中提供了什么。

[Erich]可能很容易陷入最初的设计— [毕竟，他在这个元锁游戏上已经有一段时间了](https://hackaday.com/2020/01/01/building-a-giant-meta-clock-made-of-smaller-clocks/)。事实上，零件数量的减少迫使他在显示器上变得更有创意，这是这个版本最好的部分，至少对我们来说是这样。
