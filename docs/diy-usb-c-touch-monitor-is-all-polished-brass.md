# DIY USB-C 触摸显示器是所有抛光黄铜

> 原文：<https://hackaday.com/2021/02/25/diy-usb-c-touch-monitor-is-all-polished-brass/>

我们早就知道，你可以购买接口板，将旧的笔记本电脑屏幕变成独立的显示器，但带有 4K 面板和控制板的成套设备现在也在易贝和全球速卖通等网站上广泛出售，价格也在下降。这些套件还配有微型 HDMI 和 USB-C 等薄型连接器，可以实现一些非常紧凑的构建。

来自[DIY Perks]的[Matt]用其中的一套做了一个带有黄铜外壳的超薄 USB-C 显示器。休息后的视频。外壳由铜片和 U 形槽件组成，铜片和 U 形槽件焊接并拧紧在一起。焊接后有相当多的残留物和变色，但这是通过一点打磨和抛光去除的。增加了一对可调节的腿，使它能够独立站立，背面的附加腔室容纳了控制板、旧智能手机电池和电池保护电路。[Matt]还增加了一对可拆卸的扬声器，它们是密封的扬声器单元，覆盖着黄铜网和金属板。

这些年来，我们已经介绍了几款 [DIY 显示器](https://hackaday.com/2019/03/14/an-hdmi-input-for-laptop-screen-minus-laptop/)，它们非常适合作为笔记本电脑的[附加显示器，或者与带有集成键盘的 Raspberry Pi 400 搭配使用。我们真的[马特]的建设，其中包括一个](https://hackaday.com/2019/09/28/dual-screen-laptop-is-a-slick-diy-build/)[智能手机为基础的 4K 投影机](https://hackaday.com/2020/06/22/a-true-4k-projector-from-scrap-ebay-components/)，和一个非常有效的[冷却系统为昂贵的 DSLR 相机](https://hackaday.com/2020/12/01/watercooling-a-canon-dslr-leads-to-serious-engineering-upgrades/)。
