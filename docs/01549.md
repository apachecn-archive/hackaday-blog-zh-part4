# 一种用于蚂蚁的抓挠工具

> 原文:[https://hack aday . com/2018/12/20/a-scratch-instrument-for-ants/](https://hackaday.com/2018/12/20/a-scratch-instrument-for-ants/)

如果你认为这个刮擦工具看起来至少应该…大三倍才能有用，那你就错了。尽管体积很小，但这种强大的袖珍乐器确实能让俱乐部兴奋不已。尽管如此，建造的质量以及几乎每个部分都使用现成的组件意味着，如果你需要一个小型的便携式转盘，你终于可以自己建造一个了。

[rasteri]制造了 SC1000 数字刮擦乐器，作为 portabilist 场景中的一员，专注于缩小适当的 DJ 设置所需的设备。该仪器使用 ARM 微处理器 Olimex A13-SOM-256 系统级模块，并可以使用 USB 棒将 beats 加载到系统中。刮擦轮本身使用磁性旋转编码器来感应位置，滑块也被小型化。

如果你想学会抓痒，也想学会做其他事情，下面有一个演示，展示了该工具的演示，以及项目页面上的操作视频。所有的构建文件和软件都是开源的，所以只要你有印刷电路板的经验，为自己弄一个也不会太难。如果你需要 DJ 棚的其他设备，当然[你也可以建造](https://hackaday.com/2017/07/24/meet-the-video-dj-machine/)。
