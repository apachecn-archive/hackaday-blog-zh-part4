# 独轮车配有手工制作的变速箱

> 原文：<https://hackaday.com/2018/10/28/unicycle-given-a-hand-crafted-gear-box/>

能够骑自行车滑行是一种通常被认为是理所当然的功能。由于显而易见的原因，飞轮的使用是自行车历史早期的一项改进。这也释放了制造多档自行车的能力，使更高的速度很容易达到。然而，在独轮车上，没有链条，踏板永久固定在轮轴上，这意味着(通常)没有飞轮和齿轮。[johnybondo]想让他的独轮车速度更快，并意识到他可以用自己自制的内部齿轮轮毂来实现这一点。

内部轮毂齿轮是手工加工和焊接而成的一次性原型。有商业产品，但在 1700 美元，这几乎是最好的投资自己的机器店。它使用一个比标准齿轮更紧凑的行星齿轮组，允许它安装在轮轴上。一旦所有的加工都完成了，是时候把所有的齿轮组装到轮毂上，用辐条把它系到轮子上，然后开始踩踏板。由于它如此成功，他计划建造另一个并把它系在一个更大的轮子上，这样他就能达到更高的速度。如果这对你个人来说还不够快，还有其他的[选择来达到可笑的速度](https://hackaday.com/2018/04/23/3000w-unicycles-only-limitation-is-personal-courage/)。

现在，这个档位仍然是“固定的”,这是他的独轮车的永久传动比，不允许他换档或滑行。没有飞轮机制，所以独轮车仍然可以像传统的独轮车一样向前和向后蹬。这种设置的优点是踏板每转一圈，车轮旋转 1.5 圈，使他更容易达到更高的速度。
