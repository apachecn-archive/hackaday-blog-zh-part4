# 绝密拆除揭露苏联导弹秘密

> 原文：<https://hackaday.com/2018/12/27/top-secret-teardown-reveals-soviet-missile-secrets/>

技术发展速度如此之快，以至于几十年前最机密的军事技术现在在易贝已经过剩。一个典型的例子:[msylvain59]拿起一个[苏联时代的 K-13 红外导引头](https://www.youtube.com/watch?v=VbxcS1AwquA)用来引导空对空导弹攻击目标。内部是一个转速超过 4,000 RPM 的机械陀螺仪，一个由锗制成的阻止可见光的过滤器和一个光敏电阻。想到如今你可以在几个小封装中获得所有这些，即使不集成到一个 IC 中，也是令人清醒的。

安装在导弹顶部，这个设备无论如何都不会很大，但它一点也不像现代设备的样子。一套复杂的电子设备处理信号，并移动控制鳍和其他控制装置的转向致动器，以引导导弹的飞行。你可以在下面看到这个设备泄露秘密的视频。

与现代设计不同的不仅仅是红外传感器。致动器、控制逻辑——所有的东西，真的——我们今天会用更简单的方式来做。当然，你仍然必须确保设备满足发射的冲击要求，尽管可能不是冲击。但如果有什么不同的话，那就是使用更少更小的组件会更容易。

老实说，我们只是看着他移开外壳就喘不过气来。然后是大量的灌封材料——可能是为了帮助抗震和满足环境要求。

我们同意你可能永远不会为弹头设计红外跟踪器。如果你做了，它就不会看起来像这样。但看看设计师如何在没有我们今天拥有的大量微控制器和微型设备的情况下使事物工作是很有说明性的。

如果你想了解更多关于所有陀螺仪的工作原理，[从这里开始](https://hackaday.com/2018/11/15/hows-a-gyro-work/)。我们不知道 K-13 的新成本是多少，但我们知道这个复杂得多的陀螺仪[的成本按今天的美元计算约为 15000 美元](https://hackaday.com/2013/03/26/rate-gyroscope-circuitry-explained/)。
