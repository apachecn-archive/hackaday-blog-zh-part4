# 通过返光进行塑料清理

> 原文：<https://hackaday.com/2020/03/13/plastic-cleanup-via-retrobrighting/>

如果你在旧收音机上工作，电子学只是你需要的技能之一。另一个是木材或金属工作，以恢复内阁和底盘。然而，最近的电子产品有塑料，旧塑料往往会变黄。【奇怪的实验】[展示如何使用紫外线光源、铝箔和过氧化氢来增白塑料](https://www.youtube.com/watch?v=jdWRsjnVD3s)。一般来说，ABS 是有问题的塑料，尤其是那些混有溴作为阻燃剂的塑料。你可以在下面的视频中看到结果。

请注意，使用的过氧化氢浓度为 12%——比你的药箱里可能有的浓度要高得多。这通常只是 3%的解决方案，虽然你可以得到不同的强度，包括一些超过 30%，如果你购物。然而，如果你搜索一下，你会发现人们已经成功地使用了 12%，6%，甚至 3%，尽管我们会想象使用 3%会花费更多的时间。

你也需要一些耐心。最好的结果出现在 24 到 48 小时内。我们已经看到其他人使用少量的 TAED(四乙酰乙二胺；洗衣添加剂)来加速该过程。TAED 与过氧化氢混合形成过氧乙酸。

我们也看到人们用过氧化氢基的头发漂白膏来做这个，他们可以像浆糊一样涂。我们不能保证这些不会削弱塑料，但在大多数情况下，它看起来确实有效。

在之前，我们已经[研究过这个过程，那篇文章解释了一些正在发生的化学反应。如果你想恢复旧旋钮，也许](https://hackaday.com/2017/11/03/yellowing-the-plastic-equivalent-of-a-sunburn/)[尝试 Neosporin](https://hackaday.com/2012/04/17/neosporin-the-retrobright-for-bench-equipment/) 。
