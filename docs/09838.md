# 像 1975 年一样计算:第六版 Unix 重生

> 原文：<https://hackaday.com/2021/04/21/compute-like-it-is-1975-6th-edition-unix-reborn/>

如果你渴望体验或重温“那时”的计算，你有很多选择。当然，一个选择是加载一个仿真器，假装你有你感兴趣的硬件和软件。另一个昂贵的选择是在二手市场上购买硬件。然而【mit-pdos】却另辟蹊径:[将第 6 版 Unix 移植到 RISC-V](https://github.com/mit-pdos/xv6-riscv) 上，用一个现代的 CPU 运行一个老的喜欢的操作系统。

当然，这不是一个精确的拷贝，但 Xv6 是在 2006 年作为麻省理工学院的教学操作系统开发的。您可以在[项目的主页](https://pdos.csail.mit.edu/6.828/2020/xv6.html)上找到资源，包括到原始 Unix 源代码的链接、源代码的注释以及关于原始 PDP 11/40 主机的信息。

另外，你可能会找到感兴趣的关于 Xv6 的[书。我们知道您可能不想使用 20 世纪 70 年代的 Unix 作为您的日常操作系统，但是通过阅读原始代码或更新的 Xv6 源代码，您可以学到很多东西。这也是理解 RISC-V 内部结构的一个不错的方法。](https://pdos.csail.mit.edu/6.828/2020/xv6/book-riscv-rev1.pdf)

一些计算机科学操作系统类以这样或那样的形式使用 Xv6。你可以在下面看到哈维·马德学院的几个讲座中的一个。我们已经[报道 RISC-V](https://hackaday.com/2019/02/13/western-digital-releases-their-risc-v-cores-to-the-world/) 有一段时间了，因为我们喜欢它的开放性。更不用说，CPU [不一定要复杂](https://hackaday.com/2018/02/09/a-risc-v-that-the-rest-of-us-can-understand/)。
