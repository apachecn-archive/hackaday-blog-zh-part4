# 触觉反馈“步枪”让你瞄准虚拟现实

> 原文：<https://hackaday.com/2022/01/05/haptic-feedback-rifle-lets-you-take-aim-in-vr/>

曾经有一段时间，虚拟现实似乎至少在可预见的未来仍将停留在科幻小说的领域。后来，我们有幸拥有了像“力量手套”和“虚拟男孩”这样的产品，这似乎使它更接近现实，如果不是一个笨重和有限的产品的话。然而现在，随着虚拟现实技术的进步和越来越多游戏的发布，虚拟现实正在占据越来越重要的位置。我们能看到的最好的证明就是，一些游戏玩家正在构建他们自己的定制控制器，以更有意义的方式与虚拟世界互动，[就像这个专为第一人称射击游戏](https://github.com/zhyma/vr15)构建的游戏控制器。

该控制器基于气枪，但完全没有发射弹丸的能力，而是使用枪作为构建控制器的基础。事实上，枪的操作被有效地逆转，以便通过使用加压空气提供的触觉反馈使玩家沉浸在游戏中。当检测到扳机拉动时，空气被泵入枪管前部，并通过接收器排出，以产生后坐力效应。该控制器还包括许多其他功能，包括重新装载弹药、改变射击模式和跟踪运动的能力，这要归功于它的一对集成 Oculus 控制器。

该控制器的所有部件都是 3D 打印的或现成的，这使其成为定制和改进的理想平台。Unity 还提供了一个演示游戏，可以很容易地进行测试。虽然控制器看起来是享受 FPS 虚拟现实体验的一种极好的方式，但如果你正在寻找一种更通用的控制器，我们也开始看到[在这方面也有很多发展](https://hackaday.com/2021/12/16/a-hackers-journey-in-developing-a-new-vr-controller/)。
