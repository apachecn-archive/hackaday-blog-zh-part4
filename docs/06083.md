# 在家工作时自动播放口哨

> 原文:[https://hack aday . com/2020/03/27/在家工作时自弹自吹/](https://hackaday.com/2020/03/27/self-playing-whistle-while-you-work-from-home/)

在可笑的时候，它可以帮助你演奏可笑的乐器，比如滑动哨子来控制你的毛发。但是，由于唾沫星子现在不仅仅是一点点危险，[想出替代的方法来打发禁闭期间的日子](https://www.youtube.com/watch?v=Zt_jPwtBkOY)是值得的。

由于一些聪明的 Arduino 驱动的自动化，[Gurpreet]可以在与他的滑动哨子交互时保持安全距离。滑动哨子需要两样东西——从顶部进入的空气，以及商业端的驱动。吹的力量现在来自一个集中的风扇，就像那些一旦热端挤压它就冷却你的印刷塑料的风扇。步进电机使用印刷齿条和齿轮上下移动滑块。

这里有一个平滑的触感——[Gurpreet]增加了一个微型伺服系统，用纸板瓣来阻挡和打开音孔，使音符更加清晰。休息之后，看看制作视频，其中包括“我心永恒”的音乐视频，也就是《T2》泰坦尼克号的主题曲。这几乎就像是这艘船自己在用来自远方的汽笛演奏。
