# 桌面风洞将航空航天工程带给家庭游戏玩家

> 原文：<https://hackaday.com/2020/11/20/desktop-wind-tunnel-brings-aerospace-engineering-to-the-home-gamer/>

计算机模拟在验证设计中是必不可少的，并用于工程的各个方面，从有限元分析到交通模拟到流体动力学。模拟做了一件了不起的工作，而且花费的时间和费用只是建造和测试比例模型的一小部分。但那些发自内心的啊哈时刻，以及一些现实世界的小妖精，可能更容易被真实的东西揭开。现在，你不需要大学研究或 megacorp 实验室来运行空气动力学研究 IRL，你只需要花很少的钱就可以建造一个功能性的桌面风洞。

[Mark Waller]展示了这个整洁的小设计，它只占用大约两英尺的桌面空间，包括使风洞有用的核心功能。使用安装在隧道排气侧的风扇将空气吸入隧道。入口是喇叭状的勺子，他在那里堆积了一堆吸管，以帮助确保空气进入隧道时的层流。(吸管绝招[常用于层流饮水机](https://hackaday.com/2013/10/05/a-laminar-flow-nozzle/))。它还通过一排直径约为手指的管子，以防止风扇的旋转将涡流引入气流。

为了进行分析，五根管子从一个 vape 笔中喷出烟雾，由一个水族馆的泵驱动进入室内。沿着隧道的顶部有一条发光二极管，有一个挡板可以防止光线照射到房间的黑色后壁上，以获得最佳的对比度。休息后的慢动作视频显示了设置的有效性。

无论你是一名业余编辑[用泡沫和热丝切割他们自己的滑翔机机翼剖面](https://hackaday.com/2020/09/24/dynamic-soaring-545-mph-rc-planes-have-no-motor/)，还是只是想了解一下[不同剖面的表现](https://hackaday.com/2020/07/11/aerodynamics-for-dummies/)，这都能帮你实现。它的尺寸[只有我们在之前的](https://hackaday.com/2016/08/09/hackaday-prize-entry-non-computational-fluid-dynamics/)[风洞建造](https://hackaday.com/2014/05/07/wind-tunnel-testing-now-available-to-the-common-man/)中看到的的一小部分。
