# 2022 年网络甲板竞赛:Gibson Rev 001 在鹈鹕案之外思考

> 原文：<https://hackaday.com/2022/10/25/cyberdeck-contest-2022-gibson-rev-001-thinks-outside-the-pelican-case/>

正如我们之前在这些页面中滔滔不绝地介绍的那样，我们看到了有史以来第一次网络甲板比赛的惊人参与人数——如此多的“酷”甲板蜂拥而至，以至于对它们进行评判都是一项壮举，如果我们不展示出于任何原因未能晋级的最受欢迎的甲板，那将是我们的失职。其中之一是来自[Gadjet]的名副其实的 [Gibson Rev 001](https://hackaday.io/project/187572-gibson-rev-001) 。

这个网络平台可能只有口袋大小，但不要让它欺骗了你，因为它装载了大量的 I/O 和传感器。Pimoroni Breakout Garden 提供了粒子/烟雾和脉搏血氧定量法、温度/压力/海拔、空气质量传感器和 UVA/UVB 光传感器——大量用于即时判断条件的触角。正如你所料，操作的大脑是一个 Raspi 4，它运行的是 Twister OS。

我们喜欢 7 英寸触摸屏和彩色电子墨水显示屏上的双显示屏，这真的给人一种拼凑而成的未来感。愿后启示录时代的其他小玩意也有这样干净的线条和欢快的颜色(如果你喜欢的话)。
