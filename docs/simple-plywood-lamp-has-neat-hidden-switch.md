# 简单的胶合板灯有整洁的隐藏开关

> 原文：<https://hackaday.com/2022/11/17/simple-plywood-lamp-has-neat-hidden-switch/>

电灯发明后不久，大概就出现了电灯开关。当然，显而易见的开关是老式的，这个由 Giovanni Aggiustatutto 设计的简洁的灯与一个更加害羞的设计相匹配。

这种灯本身是一种极简主义的现代设计，由胶合板制成的立方体形灯体。它很容易通过简单地堆叠几层胶合板来创建模板。在外壳内，安装了一个灯泡固定器，连接到一个壳状智能继电器，使灯能够用作智能设备。继电器还有一个用于直接控制的开关输入。这是连接到一个微型开关是塞进基地。将灯向一侧倾斜会触发微动开关，根据需要打开和关闭灯。

总的来说，这是一个优雅而实用的简单构建。它避免了灯绳上的开关和其他繁琐的细节，同时具有智能控制和直接开关。[我们之前也展示过其他一些很棒的灯具](https://hackaday.com/2022/08/31/building-a-glowing-demon-core-lamp/)。休息后的视频。
