# 终于来了！你可能想用的网络平台

> 原文：<https://hackaday.com/2022/05/31/at-last-a-cyberdeck-you-might-want-to-use/>

Cyberdecks 有助于有趣的项目，有些有点粗糙，有些则很漂亮，但这可能是即使是最狂热的爱好者也会同意的事情——这些自制的便携式计算机并不总是最方便使用的。因此，我们非常高兴地看到[这台来自【TRL】，](https://cyberdeck.cafe/mix/trl-22121)的机器，因为它采用了赛博甲板的美学，并以一种看起来很实用的形式呈现出来。

它采用了 Raspberry Pi 和 Waveshare 1280×400 电容式触摸屏，并将这种组合与键盘安装在一个设计独特的 3D 打印机箱中。它的屏幕是平的，类似于 20 世纪 80 年代早期古老的 TRS-80 100 型“平板”电脑，但把它翻过来，一个令人惊讶的可用笔记本电脑就出现了。电源来自带导线的外部电池组，但这更多是由于 PSU 电路板的热管理问题，而不是必要的。点睛之笔是一个时尚的定制笔记本电脑包，让我们可以随时带上它在火车上撰写日常文章。

环顾四周，我们认为它可能会给[发条装置发展术语](https://hackaday.com/2020/12/08/devterm-beats-cyberdeck-builders-to-the-punch/)一个机会。或者，你可以看一看[这种升级的 TRS-80 型 100](https://hackaday.com/2021/01/17/trs-80-model-100-gets-64-bit-cpu-and-a-very-wide-lcd/) 。
