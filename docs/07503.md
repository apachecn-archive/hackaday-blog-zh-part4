# 面对冠状病毒

> 原文:[https://hackaday.com/2020/08/22/facing-the-coronavirus/](https://hackaday.com/2020/08/22/facing-the-coronavirus/)

我们中的一些人不知道我们触摸自己的脸有多频繁。目前的发现是，我们每三到四分钟就会伸手去摸我们的眼睛、鼻子或嘴巴。当我们应该让我们的手远离可能传播和接收疾病的腺体时，每小时 20 次是非常多的戳、挑、痒和戳。为了遏制这种习惯并参加 2020 年黑客日奖，[Lloyd lobo]建造了一个概念验证设备[，当你伸手抓脸时它会发出警报](https://hackaday.io/project/174326-the-cap-that-doesnt-let-you-touch-your-face)。

我们看到一个 Arduino Uno 连接到经典的 HC-SR04 超声波距离传感器、一个 LED，我们必须假设一个 USB 电池组。[Lloyd]推荐较小的 Nano，我们可能会接触邮票模型，并将超声波模块换成小得多的激光飞行时间传感器。本质上，这是一个入侵警报。不要让你的兄弟姐妹呆在你的房间里，而是让你的手远离安装传感器的帽舌下方的区域。如果你经常把咖啡杯举到唇边，它可能会惩罚你，如果你嚼葵花籽，你可能会建立一个节奏。*嘎吱* *噼* * *叭** *嘎吱* *噼* * *叭* *

我们已经回顾了改善我们习惯的技术，比如记录分数的手镯，也许还有《T2》这本书可以帮助我们避免一些不太好的行为。
