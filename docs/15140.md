# HP-41C，第四版

> 原文：<https://hackaday.com/2022/10/21/hp-41c-the-forth-edition/>

如果你有一台 HP-41——可以说是有史以来最好的计算器，你可能没有注意到它有一个 Forth 版本。这段代码是用汇编语言编写的，可以在任何能够正确模拟器件的设备上运行，比如 SwissMicros DM41X。[计算器小团体] [在最近的一个视频中向你展示它是如何工作的](https://www.youtube.com/watch?v=6uljanU6ehI)，你可以在下面观看。

原始代码可以追溯到 1984 年，但[天使边缘]最近的一些[侦探工作](https://www.hpmuseum.org/forum/thread-1065-post-86dddd89.html?highlight=forth+module#pid8689)让代码再次运行。如果您了解 41C 上的合成编程及其内部架构的奇特之处，您一定会对其印象深刻。

当然，Forth 应该易于移植，但是如果您阅读了一些架构挑战，您会开始意识到这可能是您见过的最困难的实现之一。不要忘记，以今天的标准来看，你所拥有的资源是极其有限的。

话虽如此，称 HP41C 为计算器几乎是一种犯罪。它实际上是一台藏在计算器盒子里的微型计算机。话说回来，最好的计算器总是。

我们想知道代码是否能在一个[仿真 41C 上运行？](https://hackaday.com/2021/12/03/building-the-diy-hp41c-a-field-report/)您是 TI 计算器帮的成员吗？[没问题](https://hackaday.com/2019/01/25/pocket-forth-invades-your-ti-calculator/)。
