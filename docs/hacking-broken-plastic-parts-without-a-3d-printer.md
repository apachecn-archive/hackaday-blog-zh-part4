# 在没有 3D 打印机的情况下破解破损的塑料零件

> 原文：<https://hackaday.com/2019/10/13/hacking-broken-plastic-parts-without-a-3d-printer/>

我们都遇到过奇怪的塑料零件坏了，拿不到。当然，3D 打印机可以打印大型替换零件，但有时你只需要重建一个非常特殊的零件。[AkBkukU]展示了一种技术，使用一种你几乎可以称之为手动 3D 打印的过程。我们以前见过小苏打用来固化氰基丙烯酸酯胶，但这项技术用它来建立显然非常坚固的胶层。

下面的视频中有相当多的细微差别，但基本的想法是把一堆苏打水放在一片锡纸和一团胶水的一边。你把零件浸在胶水里，然后放进苏打水里。每次你都会得到一层更厚的胶水。

之后，你必须锉掉或者塑造新的部分，但是它能被锉掉的事实应该告诉你一些事情。我们想起了一些人如何使用环氧树脂来形成修复部件，然后将它们加工成所需的精确形状。在视频的最后，他在一个他不能蘸的部分上建立了层。成功了吗？看着看吧。

除了手工 3D 打印技术，他还演示了如何用小苏打来修复旧收音机闹钟的滚花旋钮。这有点太传统了，但如果你以前没见过，这几乎是奇迹。

胶水太神奇了。我们见过热熔胶做[注塑](https://hackaday.com/2019/06/11/injection-molding-with-a-hot-glue-gun/)。还有更多种类的。
