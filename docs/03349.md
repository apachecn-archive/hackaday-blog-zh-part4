# 开源电脑控制织布机为你编织皮卡丘

> 原文：<https://hackaday.com/2019/06/18/open-source-computer-controlled-loom-knits-pikachu-for-you/>

软件起源的故事把我们带回到过去的穿孔卡片计算机和巴贝奇的差分机到一个叫约瑟夫·玛丽·贾卡的法国织布工。雅卡尔创造了一种自动化机械织布机的方法，使织工能够通过简单地切换穿孔卡片来改变织布机的图案。这一发明不仅使以极其简化的方式生产精细织物成为可能，而且是计算机程序设计发展中一个极其重要的概念性步骤，影响了巴贝奇对分析引擎的开发以及其他许多事情。

因此，当[Kurt]看到他的儿子对简单织布机的热情时，他开始思考如何通过设计和建造一台开源计算机控制的织布机来向软件的根源致敬。他知道这将是困难的:织布机是由数百个小零件组成的复杂机器。[Kurt]与不稳定的车厢运动，凸轮堵塞，钩子尺寸灾难和大量来自电机控制板的魔法烟雾搏斗。经过一年半的织布机黑客研究，他成功地制造了一台 60 线的电脑控制织布机，由一个使用蓝牙的 iPhone 应用程序驱动。

除了在上面链接的博客上写下这个构建的故事之外，[Kurt]还在 GitLab 上提供了他所有的设计文件、PCB 布局、固件和代码[。](https://gitlab.com/electronicsnufkin/WeaveMe)

这些年来，我们展示了一些编织技巧，包括这款[便宜、简单的 3D 打印织机](https://hackaday.com/2017/07/25/make-a-bit-of-cloth-with-this-3d-printable-loom/)和一款[提花灵感位图显示器](https://hackaday.com/2015/06/04/jacquard-loom-becomes-a-display/)。
