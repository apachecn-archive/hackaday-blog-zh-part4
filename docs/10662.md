# 一台可以在家里使用的塑料注射机

> 原文:[https://hack aday . com/2021/07/07/a-plastic-injection-machine-you-can-use-in-home/](https://hackaday.com/2021/07/07/a-plastic-injection-machine-you-can-use-at-home/)

如果你想要一样东西，3D 打印当然很好，但如果你想要很多基本相同的塑料零件，你应该考虑注射成型。你可以花钱请人帮你做这件事，或者，按照真正的黑客方式，[你可以在自己的车库里建立一个完整的注射成型系统，就像[Action BOX]所做的那样。](https://www.youtube.com/watch?v=l4gGWufoIYI)

该建筑依靠一对强劲的 3 马力电机来驱动基于螺杆的注射系统。它们负责从料斗中输送塑料颗粒，然后将它们熔化并填充注射容器，然后将热塑料压入模具。进一步步进电机处理夹紧模具，然后释放它和弹出完成的部分。Raspberry Pi 处理机器的操作，并配置有能够正确循环操作的自定义 Python 程序。在高峰期，这台机器每分钟可以生产 4 个零件。

这是一个令人印象深刻的工业型硬件。如果你想在自己的工厂里生产大量的塑料制品，像这样的机器是非常合适的。这也不是我们见到的第一台同类型的机器。休息后的视频。
