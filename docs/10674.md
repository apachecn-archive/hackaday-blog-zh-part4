# 用手工工具制作小夹子

> 原文：<https://hackaday.com/2021/07/11/making-teeny-clamps-with-hand-tools/>

有时候你需要一个小夹子，有时候用自己做的工具也不错。[Pask Makes]的[Neil]在这两个方面都有所贡献，他在自己的家庭商店里制作了非常可爱的小夹子。你甚至可以用手工工具制作它们！

第一步是剪下一段扁钢条，然后在中间钻一个孔。然后可以使用钢锯切割形成键夹紧表面的平面。接下来，就是用锉刀清理由此产生的空白，去掉锋利的边缘，整理平整。

下一步工作是在夹子底部钻一个主孔，让孔变直变准是制作一个好的可用夹子的关键。该视频分享了如何使用一个简单的无绳电钻来完成这项工作的技巧，通过使用老虎钳和木质导轨来保持事情正常进行。然后，旋转螺母由一根圆棒制成，并安装在螺栓的末端，以形成良好的夹紧面。一个可爱的小铜把手用来拧紧它。

这是一个有用的工具，视频继续展示了如何使用更高端的设备更快地制作夹子。虽然小夹子很便宜，但视频指出制作工具很有趣，我们认为这是自己制作工具的最好理由。
