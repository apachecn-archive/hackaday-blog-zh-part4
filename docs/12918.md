# 旧的便携式电视成为独特的 CRT 电脑平台

> 原文：<https://hackaday.com/2022/02/28/old-portable-tv-becomes-unique-crt-cyberdeck/>

还记得电脑的“手提箱”外形吗？在 latops 出现之前，这些笨重的机器正是忙碌的经理们需要的东西。好吧，也许不是真的——但将 PC、显示器和外围设备放在一个包装中的能力具有真正的吸引力，这种基本原理是 cyberdeck 现象背后的原因。所以当我们看到这个复古的便携式电视变成了一个电脑平台时，它真的吸引了我们的眼球。

具有讽刺意味的是，[卢卡斯·杜尔]选择的便携式黑白电视作为他的网络平台的基础，与便携式个人电脑在阳光下度过短暂时光的时间大致相同。从易贝得分，Magnavox 电视/广播组合经历了更好的日子，需要一点手术来修复可能已经下降的损害。随着 CRT 的恢复和视频和音频路径的定位，电视有了一个树莓 Pi、一个小触摸板和几个隐藏的 USB 连接器。Pi 的复合输出驱动 CRT，得到您所期望的结果。这个键盘看起来大小正好可以作为外壳，但是[卢卡斯]说这是一个未来的项目。

尽管如此，由于电视的原始手柄充当支架，这个 cyberdeck 散发出真正的康柏或 IBM 便携式 PC 的氛围。我们以前见过一些看起来像便携式电脑的平台，但是没有一个敢用 CRT 显示器。这可能与 HDMI 相去甚远，但我们真的很感谢[卢卡斯]选择了这种方式，而不是扔进一个 LCD。
