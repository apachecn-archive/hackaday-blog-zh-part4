# 将战斗机器人带入现代课堂

> 原文:[https://hack aday . com/2019/05/21/将战斗机器人带入现代课堂/](https://hackaday.com/2019/05/21/bringing-battle-bots-into-the-modern-classroom/)

年轻学生可以获得各种各样的数字娱乐，教育者很难抓住他们的想象力。在过去的几十年里，用小苏打和醋制成的“火山”足以让一群五年级的学生坐立不安，但这些项目对那些可能以皇家战役比赛开始他们的学校生活的学生来说没有那么大的冲击力。今天的教育者的任务是激励那些已经对世界了如指掌的孩子们。

[![](../Images/1366e391a8669100d4aaefcc080671e7.png)](https://hackaday.com/wp-content/uploads/2019/05/kidsbots_detail.jpg) 希望通过参加 2019 年 Hackaday 奖来迎接挑战，[【Misty Lackie】正在组装一套工具，允许中小学生建造他们自己的战斗机器人](https://hackaday.io/project/165500-educational-battle-bots)。由于模块化组件的使用，年轻的学生不必陷入焊接或所有硬件实际工作的复杂性。另一方面，年龄较大的孩子将能够扩展基本平台，而不必从头开始。

机器人的电子设备主要包括一个带传感器屏蔽的 Arduino Uno，一个双 H 桥电机控制器和一个 PS2 控制器的无线接收器。这使得学生可以用他们已经非常熟悉的输入方案来控制机器人的双驱动马达。通过将控制器的面部按钮映射到 Arduino 上的数字引脚，额外的功能，如休息后在机器人中看到的微调器，很容易被激活。

[Misty]已经用这个工具包的早期版本做了一些测试，到目前为止它取得了巨大的成功。学生们可以自由设计他们自己的身体和远程控制平台的附件，看到最终结果是如此独特，真是令人着迷。我们过去已经看到，当学生被要求定制他们自己的机器人时，他们会多么兴奋，所以在我们的书中，进入这个领域是一个积极的发展。
