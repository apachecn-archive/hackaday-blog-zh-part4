# 跺脚按钮，接受糖果

> 原文：<https://hackaday.com/2020/10/19/stomp-button-receive-candy/>

如果有什么节日值得为奇怪的时代而调整，那就是万圣节。你倾向于在门廊上放一碗糖果来避开门铃吗？如果是这样的话，这是一个完美的一年，最终找出某种计量装置，这样贪婪的青春期前的孩子就不太可能在一次扫描中偷走你的存货。我们仍然有时间制造一些有趣的东西，比如[【布兰克利】的自动糖果分配器](https://www.youtube.com/watch?v=NNeSOPg2tVk)，我们认为它应该会在未来许多年内继续存在。视频在休息后发布。

在骷髅南瓜灯头下是这座建筑的心脏——一个与之完美匹配的橙色 5 加仑桶。只需踩下巨大的发光街机按钮，同样巨大的 NEMA-23 步进电机在 Arduino Nano 的帮助下移动桶内的 3D 打印转盘。这将糖果移向 3D 打印的斜坡，并从杰克灯的口中出来，在那里它降落在一个碗中，当它触及底部时，由于一个继电器和第二个纳米而变亮。

[布兰克利]巧妙地利用了红外线光束开关，它位于坡道上两个方孔的下方。一旦糖果经过其中一个，转盘就会停止并向后旋转，将糖果移动到够不到的地方。

坦率地说，我们喜欢[布兰克利]重复使用南瓜灯自带的音效模块。这个建筑是完全开放的，如果你想自己制作，Brankly 甚至会赠送 40 块印刷电路板。现在，您可以检查代码并开始打印 STLs。

如果时间紧迫，[建造一个幽灵般的滑梯，在你和不给糖就捣蛋的人之间留出 6 英尺的距离](https://hackaday.com/2020/09/25/candy-slide-keeps-halloween-spooky-and-socially-distant/)。
