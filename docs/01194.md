# 数码相框翻转 Vectrex 贴面

> 原文：<https://hackaday.com/2018/11/08/digital-picture-frame-turned-vectrex-overlay/>

对于那些在 1983 年崩盘之前可能不太熟悉家庭视频游戏世界的黑客读者来说，Vectrex 是一次将矢量图形带入玩家客厅的有趣尝试。以今天的美元计算，这台机器的价格约为 500 美元，它的独特之处在于它包括自己的黑白 CRT 显示器，而不是要求所有者将其插入他们的电视。为了增加一点趣味，游戏将包括一层薄薄的塑料覆盖层，你可以把它放在屏幕上，给游戏添加人工色彩。我们能说什么呢？那是 20 世纪 80 年代。

像许多老式游戏系统一样，Vectrex 仍然拥有忠实的粉丝，其中一些人在系统发布近 40 年后仍在寻找破解和修改系统的方法。一个这样的粉丝是[Arcade Jason]，他最近一直在摆弄一个想法，用一个被黑掉的 LCD 显示器创造一个现代的叠加概念。虽然它的边缘还有点粗糙，但它确实有希望。他希望有人能够将这个想法付诸实践，并将其转化为 Vectrex 社区的适销产品。

[Jason]从得到一个旧的数码相框开始，把它拆了，直到他解放了 LCD 面板。通过仔细拆卸，他能够移除背光，并留下一个透明的显示屏。然后，他将面板安装在 Vectrex 的显示屏上，让相框的 PCB 和控件悬在一边。延长显示器的带状电缆应该足够容易，以便进行更稳固的安装。

然后，他在相框里装上了他在网上找到的随机迷幻图片，以及一些他在艺术程序中使用彩色块快速制作的自定义叠加图片。在休息后的视频中，[Jason]使用 PCB 上的按钮在画面上切换图像，同时加载不同的演示，以展示可能的视觉效果。

虽然这是一个简洁的概念，但在真正付诸实践之前，还有几个问题需要解决。首先，LCD 面板的尺寸或长宽比与 Vectrex 显示器不匹配，因此它不能覆盖整个 CRT。选择在 LCD 面板上显示的图像也相当困难；一个改进的版本可能使用类似 Raspberry Pi 的东西在面板上加载图像，同时在某种类型的辅助屏幕上显示控制界面。

这不是[【杰森】第一次试验威格斯](https://hackaday.com/2017/11/02/finally-a-big-screen-vectrex/)，甚至也不是[第一次尝试给经典系统](https://hackaday.com/2018/01/03/vectrex-finally-in-color/)添加色彩。我们很想知道他下一步会想出什么。
