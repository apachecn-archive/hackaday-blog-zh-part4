# 用卡迪 B 杀蚊子

> 原文：<https://hackaday.com/2020/12/05/killing-mosquitoes-with-cardi-b/> 杀死蚊子

在你的院子里放一个鸟浴或池塘是增加氛围和风格的好方法，但是这两样东西都可能成为蚊子的天堂。清除它们的流行方法通常是使用刺激性杀虫剂，但[Shane]给我们带来了一种更环保的方式来处理这些携带疾病的昆虫，方法是在水下循环播放 Cardi B 播放列表，杀死蚊子幼虫。

虽然这个版本确实包含了一些其他的最爱，比如“小鲨鱼”，并且可能适用于任何歌曲(或者足够大音量的音频),但是这个版本仍然非常有趣。它是基于一个 555 定时器电路，为一个超声波枪提供动力，但被重新用于这个版本。超声波模块被用硅酮防水的压电模块取代。产生的声音以与蚊子幼虫共振的频率振动，对它们是致命的。[Shane]将构建放入一艘小船中，该小船可以漂浮在任何池塘、鸟浴、马槽或水景中。

对这种建筑的主要警告是，它可能会损害其他有益的动物，如鱼或青蛙，所以他建议将其用于无人居住的死水。不管怎样，这都是一种非常独特的解决蚊子问题的方式，不像另一个建筑以稍微不同的方式处理水中的昆虫。
