# 为您的 3D 打印机添加显微镜

> 原文：<https://hackaday.com/2019/08/03/add-a-microscope-to-your-3d-printer/>

有很多方法可以在 3D 打印机打印出一层层的打印效果时保持警惕。我们大多数人都会时不时地瞥一眼，以确保我们不是在做塑料粉丝，我们中的一些人会更进一步，用树莓派相机或类似的相机。[Uri Shaked]更进了一步，他在 Creality Ender 3 的热端旁边的定制支架上添加了一个 USB 显微镜。

这个支架本身只是一个普通的 3D 打印部件，但人们的兴趣在于它能做什么。Ender 3 在 X/Y 轴上的分辨率为 12.5μm，在 Z 轴上的分辨率为 2.5μm，这意味着它可以定位在任何微小物体的毫厘之内。当然，这实现了检查 3D 打印完整性的主要目标，但它也允许用显微镜跟踪或扫描任何物体。

例如，在检查一片罗勒叶时，[Uri]注意到它的表面有一只小昆虫，并能够通过一些匆忙输入的 g 代码跟踪它。更好的是，他拍下了追逐的视频，你可以在画面下方看到。从自动化 PCB 质量控制到艺术创作，我们完全被低成本机器人显微镜平台的可能性所吸引。
