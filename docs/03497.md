# 这个 MDF 音棒听起来很棒

> 原文:[https://hack aday . com/2019/07/03/this-MDF-sound-bar-sounds-great/](https://hackaday.com/2019/07/03/this-mdf-sound-bar-sounds-great/)

每个人都应该在一生中至少建造一个音箱，只要他们能够意识到建造一个简单的盒子需要花费多少心思。ibuildit.ca 的[John]想要为他的家庭影院系统安装一个条形音箱，[，这意味着要建造一个条形音箱](https://ibuildit.ca/projects/how-to-make-a-sound-bar/)。结果很漂亮，证明了你只用一个路由器和一个台锯就能做多少事。

[John]几乎完全用中密度纤维板制作了这个条形音箱，这不是最好的材料，但对于安装在墙上的音箱来说，它已经足够好了。首先建造侧面，用槽口保持正面和背面。低音扬声器和高音扬声器都嵌入到前面，一个标准的管道作为低音端口。将一个圆形钻头放入路由器中，并进行轻微打磨，涂上一层黑漆后，一切看起来都很棒。

与任何扬声器音箱一样，该设计实际上是参数化的，完全围绕所使用的驱动器进行设计。在这种情况下，[John]使用的是名为“Unibox”的电子表格，它为您提供了设计扬声器音箱的所有公式和图表。

盒子做好了，扬声器安装好了，剩下的唯一问题就是一些美学上的选择。[John]采用标准的黑色表面，非常漂亮的木质格栅用磁铁固定在前面。这是一个流行的设计，但对扬声器的真正考验是它听起来如何。这有点难以在互联网上传达，但[John]在构建视频的结尾提供了一些声音样本，如下所示。
