# 木质外壳为这款 DIY 带状话筒增添了温暖

> 原文:[https://hack aday . com/2021/05/14/wood-enclosure-lends-warm-to-this-DIY-ribbon-microphone/](https://hackaday.com/2021/05/14/wood-enclosure-lends-warmth-to-this-diy-ribbon-microphone/)

我们喜欢有人把他们在 Hackaday 上看到的想法付诸实践，并把它带到一个新的不同的方向。毕竟，这正是我们在这里的目的，看到像[这种木制丝带麦克风](https://hackaday.io/project/179720-custom-designed-ribbon-microphone)这样的项目成为现实，我们感到非常欣慰。

现在，我们还不能完全确定[玛雅·罗曼]的灵感来自于我们对[弗兰克·奥尔森]的报道[向 RCA Model 44 录音室话筒致敬的报道](https://hackaday.com/2021/03/18/wooden-you-love-to-build-a-ribbon-microphone/)，但无论如何我们都要在这里拍拍自己的背。关于[Maya]的建造，有趣的是她为她的项目选择了完全不同的材料和设计风格。在[Frank]尽可能多地用木材制作麦克风的地方，[Maya]很好地采用了混合媒体的方法——CNC 铣削的胶合板用于外壳和支架，激光切割的丙烯酸树脂用于带状电机框架，以及根据需要到处打印的 3D 部件。用作挡风玻璃的编织黄铜布是一个不错的细节；虽然整个东西看起来——听起来——都很棒，但我们认为，如果再涂上一层深色颜料与黄铜形成对比，再涂上一层漂亮的聚氨酯光泽涂层，效果会更好。

下面的视频展示了整个设计和构建过程，这是[Maya]在大学本学期音频制作课的一个期末项目。希望它能得到我们能给的好成绩。
