# 绞盘驱动拉着这个动态四足动物的弦

> 原文:[https://hack aday . com/2021/08/07/capstan-drive-is-pulling-the-strings-on-this-dynamic-quadroped/](https://hackaday.com/2021/08/07/capstan-drive-is-pulling-the-strings-on-this-dynamic-quadruped/)

当谈到腿式机器人时，很容易想到复杂性和加工成本会使这些生物远离成为任何人的车库爱好。但是，通过一系列聪明的设计选择，[Damian Lickindorf]找到了一种方法来克服困难，赋予斯坦利生命，一种低成本，充满活力的四足动物，有一些严重的踢腿！

仿佛建造一个工作腿机器人已经不是一个棘手的任务，[Damian]已经做出了一些经典的设计选择，以保持低价格，降低制造复杂性，而不牺牲性能。为了跟上准直接驱动腿式机器人的最新趋势，从[麻省理工学院的迷你猎豹](https://hackaday.com/2019/09/22/watch-legged-robot-run-circles-around-its-bigger-brethren/)，【达米安】制造了一种齿轮减速比低于 1:9 的小型传动装置。这种选择略微减少了在高扭矩下低速运行电机所产生的热量，而不会牺牲太多的控制带宽(想想:“腿部响应”)。

然而，与使用行星变速箱的猎豹不同的是，[Damian]选择了绞盘驱动，这是一种电缆驱动的传动装置，既无间隙又可反向驱动:这是力敏感动态腿机器人的两个必备条件。对于腿，他选择 2d 加工的 FR4(想想:电路板材料)。对于电机，他选择了一套大间隙半径的无刷电机，由 [Moteus Drivers](https://hackaday.io/project/179234-moteus-brushless-controller) 驱动。其结果是高保真、动态构建，其成本只是我们看到的从学术研究实验室中出现的一些生物的一小部分。

如果你想在一些动作镜头上大饱眼福，看看[达米安]的[YouTube](https://www.youtube.com/user/damlic11/videos)和[的 insta gram](https://www.instagram.com/artysta_automatyk/?hl=en)就知道了。如果你想跟踪这个项目，[可以看看 Hackaday.io 项目](https://hackaday.io/project/176726-stanley-the-capstan-based-quadruped)。虽然我们渴望看到该项目继续展开，但我们对它的进展感到兴奋。与此同时，一定要看看该项目的灵感之一:Mjbots Quad A0 。

最后，由于我们在 Hackaday 上没有看到多少 capstan drives，如果你对这些机制感兴趣，并且能够越过付费墙，[这](https://ieeexplore.ieee.org/document/8593569)两篇[研究](https://ieeexplore.ieee.org/document/7807309)论文可能是深入挖掘的好地方。
