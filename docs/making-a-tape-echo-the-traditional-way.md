# 让磁带以传统方式回响

> 原文：<https://hackaday.com/2022/09/14/making-a-tape-echo-the-traditional-way/>

[Juan Nicola]从古代的音乐家黑客那里获得了灵感，并将一个卷到卷的磁带录音机改造成了他的吉他的[磁带回声，内置电子管放大器(西班牙语视频)。](https://www.youtube.com/watch?v=K6GsQq_qGPo)

其原理是将吉他的声音录制到一段移动的磁带上，然后过一会儿再读一遍。该信号与现场输入信号混合，并重新录制到更远的磁带上。这种效果听起来像回声，在数字效果出现之前，这种方法非常流行。

[Juan]在他的 Grundig TK40 上安装了一个新的阅读头，并设法找到了一个合适的机械装置来保持它的位置。此后，他更新了这个项目，转而使用像带循环，通过一遍又一遍地重复使用同一段像带，允许无限的播放时间。

将录音机变成回声效果并不是一个新的想法，在过去的[年](https://hackaday.com/2010/12/20/tape-delay-made-from-recycled-cassette-decks/)中，我们已经[向](https://hackaday.com/2021/08/21/modern-tape-echo-made-easy/)展示了[中的](https://hackaday.com/2019/11/03/diy-cassette-tape-guitar-delay/)和[中的](https://hackaday.com/2015/10/07/tape-loop-echo-made-with-an-actual-tape-loop/)，但是每一个都略有不同！

两个版本都在休息后播出。YouTube 的隐藏字幕自动翻译功能可能会对不会说西班牙语的人有用。
