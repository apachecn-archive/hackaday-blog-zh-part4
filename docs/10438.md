# 1 级方程式电视广播，比例为 1:87

> 原文:[https://hack aday . com/2021/06/18/formula-1-TV-broadcasting-in-187-scale/](https://hackaday.com/2021/06/18/formula-1-tv-broadcasting-in-187-scale/)

[Gerrit Braun]是汉堡[微缩奇境]模型铁路和微缩机场景点的联合创始人，他对自己的模型建筑非常认真。五年多来，他和他的团队一直在精心规划、测试和建造 1:87 比例的一级方程式摩纳哥大奖赛。在[神童乐园](https://en.wikipedia.org/wiki/Miniatur_Wunderland)的模型是按照第 n 个细节制作的，所有合理的努力，以及一些不合理的努力，都是为了达到逼真的效果。在[下面的视频](https://www.youtube.com/watch?v=azM9TIt7d7Q)，Gerrit 的项目日记的一部分，他讨论了模拟现实电视广播的问题和解决方案(视频是德语的，但有英语字幕)。

目标是模拟设置在观众席上的广告牌大小的大屏幕。在现实生活中，这些显示器接收来自遍布赛道的摄像机的图像，其中大部分由摄影师操作。近年来，相机的小型化取得了很大进展，例如 ESP32-CAM 模块或 Raspberry Pi 相机。但是，缩小摄影师的平移和倾斜动作，虽然可能，但从长期来看是不可靠的(这些在神童世界的展览是永久性的，几乎每天都在进行)。相反，该团队能够使用软件从高分辨率视频中提取一个裁剪窗口，移动这个裁剪窗口的位置可以模拟摄像机的指向。更多细节在视频里。

[Gerrit]和他的团队的技术和创造力令人难以置信。这个项目的其他视频涵盖的主题包括音响系统、道路使用的 PCB 技术，以及令人瞠目结舌的使用电动立式办公桌来提升整个城市街区，以便工人可以进入该区域。公平的警告——这些会让人上瘾，下面的视频是一个未完成系列的#76。早在 2016 年，当[Gerrit]和他的双胞胎兄弟[Frank]与[谷歌地图合作制作他们的复制品城市](https://hackaday.com/2016/01/17/the-smallest-google-street-view-in-miniatur-wunderland/)的街景时，我们就写过关于神童世界的报道。感谢【Conductiveinsulation】给我们发来的提示，他说本周[播客#122](https://hackaday.com/2021/06/11/hackaday-podcast-122-faster-than-wind-travel-sisyphish-alu-desktop-calculator-and-mice-in-space/) 中关于[互连三角形 PCB 板](https://hackaday.com/2021/06/07/triangle-tiles-form-blinky-networks-using-clever-interconnects/)的讨论出于某种原因提醒了他这一点。我们的读者中有谁以前参观过微缩奇观吗？请在下面的评论中告诉我们。
