# 麻省理工学院 IAP 解决无线电问题

> 原文：<https://hackaday.com/2019/02/18/mit-iap-tackles-radio/>

麻省理工学院以严格的课程而闻名，但他们在每年年初也有一个特殊的为期四周的学期，称为 IAP——独立活动期。今年，麻省理工学院无线电协会就无线电的历史和应用做了几个有趣的报告。你不在那里？没问题，因为九个讲座都被录制好了，你可以在闲暇时观看。你可以在下面看到九个中的一个。

这些也不是一些五分钟的快速视频。它们基本上是现场拍摄，时长从一小时到近两小时不等。主题非常丰富，包括无线电历史、软件定义无线电、传播、射电天文学、雷达，甚至 5G。

你可能不得不挑挑拣拣。有些讲座适合任何人。一些人认为在电子或数学方面有更多的无线电专业知识。尽管如此，它们都至少值得粗略浏览一下，看看你是否真的想坐下来仔细观察。唯一吹毛求疵的是，一些演示者使用了一个激光笔，而这个激光笔在视频的插图中并没有显示出来。这是有道理的，因为插入的幻灯片并不真的在房间里，但这可能会使人有点难以理解演讲者在拥挤的幻灯片上指的是什么。

当然，如果你想深入研究，你需要更多的背景知识，麻省理工学院和许多其他机构会让你免费使用他们的学习材料。我们特别喜欢《T2》的《电路课》,但是还有很多其他的，包括来自 [OCW](https://ocw.mit.edu/index.htm) 的原材料。
