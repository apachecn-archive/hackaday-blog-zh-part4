# 60 美元 PC 示波器评论

> 原文:[https://hackaday.com/2022/02/26/60-pc-oscilloscope-review/](https://hackaday.com/2022/02/26/60-pc-oscilloscope-review/)

拥有一台示波器是一个真正的游戏规则改变者，如今，示波器比以往任何时候都更强大、更便宜。然而，花费数百美元的示波器和许多非常便宜的——不到 100 美元——比玩具好不了多少的仪器之间有很大的区别。[阿德里安]看了一个汉德的基于个人电脑的示波器，价格约为 60 美元。它是玩具吗？还是有用的工具？他回答了下面视频中的问题。

Hantek 6022BE 拥有两个 20 MHz 带宽的通道和每秒 4800 万个样本。该设备还包括探针。当然，你还需要一台个人电脑，尽管如果你不想带着笔记本电脑到处跑，显然有适用于安卓系统的第三方软件。

凭借两个通道和相对较低的带宽和采样率，这种 Hantek 不会取代一个好的台式示波器，但你不会在这个价位上得到其中之一。你要问自己的问题是，你真正需要的是什么？在[Adrian]的案例中，他想从事 Commodore 64 计算机之类的工作，因此 20MHz 对于这类工作应该是合适的。

该软件看起来有点过时，但它确实有很多你会在传统示波器上发现的功能。然而，触发功能不是很强，但由于该设备使用 USB2，您可以猜测触发是在设备中发生的，该设备可能无法处理任何过于复杂的事情。结果与[阿德里安]手头上一台更强大的仪器相当吻合。

判决结果？[阿德里安]认为这可能是值得的钱，但确实希望有些事情会更好。然而，许多评论者敦促他尝试该设备的非官方软件，该软件被认为要好得多。我们对作为示波器的设备并不感兴趣，但鉴于该协议显然被理解，我们想知道它是否可能在未来的某个项目中作为廉价的数据采集模块找到一个家。
