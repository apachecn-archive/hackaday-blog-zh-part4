# 建造一个半玩具半模型坦克机器人

> 原文:[https://hack aday . com/2021/02/12/building-a half-toy-half-model-tank-robot/](https://hackaday.com/2021/02/12/building-a-half-toy-half-model-tank-robot/)

对于一些人来说，花几个小时来绘画和设计具有微小细节和特征的复杂模型听起来像是度过了一个愉快的下午。我们中的一些人更希望它已经涂上了马达，这样它就能移动了。[科里·柯林斯]试图通过建造一辆被称为坦克机器人 2.3 的高度精细的机动坦克来结合这两种爱好。(视频，嵌在下面。)

它基于一个简单的在线订购的六足套件，包括一个内置的 Arduino 兼容板(它基于 Arduino 2560 Mega)。腿部经过重新设计，符合[Cory]追求的审美。重新设计允许在腿部机构中有一个额外的枢轴。炮塔部分是在底座的顶部设计和建造的，支持伺服系统来转动它(尽管固件还没有完全到位)。在所有的零件都被 3d 打印出来后，开始了艰苦的绘画过程。用一些精致的喷枪和一些快速模板切割贴花，这是完整的。

我们惊讶于你能在网上找到的套件和零件的类型，以及它们通常不贵的事实。自 2013 年[我们报道了一个更简单的基于 Arduino 的坦克](https://hackaday.com/2013/01/03/building-a-rather-rudimentary-arduino-tank-bot/)以来，我们已经走过了漫长的道路。
