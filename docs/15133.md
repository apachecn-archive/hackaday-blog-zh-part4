# 2022 年 Hackaday 奖:一款便宜有效的嘴部操纵杆鼠标

> 原文:[https://hackaday . com/2022/10/19/hackaday-prize-2022-a-cheap-and-effect-mouse-mouse/](https://hackaday.com/2022/10/19/hackaday-prize-2022-a-cheap-and-effective-mouth-joystick-mouse/)

有些人使用普通的计算机鼠标有问题，需要适合他们需要的替代输入设备。[奥曼·奥罗兹科]设计了一款价格实惠的嘴部操纵杆，旨在作为鼠标使用，以适应这种特殊的使用情况。

![](../Images/f1716c40a13cdafd5a5f7ddbf7c718be.png)

A simple frame allows the controller to be held near the mouth for use.

该版本基于一个简单的 USB 游戏手柄。鼠标模拟器程序用于将拇指操纵杆的运动转换为鼠标的运动，将按钮的按下转换为鼠标的点击。

诀窍是拇指操纵杆变成了一个嘴部激活的操纵杆，使该设备非常适合那些手臂运动受限或不能运动的人。这是通过用于延伸操纵杆的笔体来实现的，因此它可以容易地用嘴来致动。一个定制的喷烟开关也被集成到嘴部操纵杆中，由一个气球、瓶盖和一个微型开关组成。这使得用户能够点击东西，而不需要另一个按钮。

总的来说，这是一个伟大的黑客技术，它将廉价的日常组件变成了一个有用的辅助硬件。另外，它可以只用简单的工具来构建。[Olman]注意到不需要 3D 打印机或其他先进的部件或工具来制造设备。

当涉及到易访问性黑客攻击时，游戏控制器通常是重要的设备，[正如我们在](https://hackaday.com/2013/06/20/building-custom-game-controllers-for-people-with-physical-disabilities/)之前所看到的。
