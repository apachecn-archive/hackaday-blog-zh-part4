# 商店自制的固定装置揭示了可切换永久磁铁的魔力

> 原文：<https://hackaday.com/2019/04/07/shop-built-fixtures-reveal-the-magic-of-switchable-permanent-magnets/>

你有没有想过可切换磁铁是如何工作的？不是电磁铁，而是那些永久磁铁固定装置，比如那些把千分表固定在机床上的装置，或者是用于平面磨床的大而有力的卡盘，只要轻轻一拨杠杆就可以神秘地消磁。这看起来像魔术。

多亏了[Andrew Klein]和[这个关于商店自制磁性开关](https://www.youtube.com/watch?v=MptNafdF_q8)的视频，魔法消失了。事实证明，抵消一堆稀土永磁体的强大磁场的能力，就像引入另一组磁体来抵消第一组磁体的磁场一样简单。

[Andrew]的磁性圆盘由两个厚胶合板圆盘组成，边缘镶有磁铁。这些磁铁的极性在圆盘周围交替变化，它们与嵌入圆盘表面的低碳钢极片相匹配。这两个圆盘在一个公共轴上旋转；当旋转顶部圆盘，使顶部和底部磁铁的极性对齐时，磁铁打开。旋转顶部 60°使相反的磁场彼此成一直线，抵消了所有磁铁的强大合力，释放了夹具。

[Andrew]出售一套磁控开关的[计划，这是他使用标准木工工具制作的。我们认为这个设计非常适合 CNC 路由器，虽然繁琐的钻孔和扩孔操作可能会更容易一些。也许 3D 打印的版本也是可能的。当然，这不是我们看到的第一个](https://www.in-kleind.com/store/Magnetic-Switch-Template-p131654057)[可切换磁铁](https://hackaday.com/2015/03/17/switchable-magnet-with-a-simple-circuit/)，但是我们喜欢这个，因为它全是机械的。
