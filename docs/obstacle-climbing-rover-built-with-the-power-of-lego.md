# 用乐高的力量建造的越障漫游车

> 原文：<https://hackaday.com/2021/01/23/obstacle-climbing-rover-built-with-the-power-of-lego/>

当我们想要制作一个漫游者的原型时，我们倾向于立即使用 3D 打印机和 Arduino 或 Raspberry Pi。很容易忘记我们许多人成长过程中使用的原型制作工具:乐高。[砖块实验频道]没有忘记，在休息后的视频中[展示了他如何使用乐高技术组件原型制作一个令人印象深刻的小障碍攀爬机器人](https://www.youtube.com/watch?v=MwHHErfX9hI)。

这个小小的乐高漫游车开始是一个简单的四轮漫游车，试图爬上一本书的顶部。换上四轮驱动变速箱和防滑轮胎，它就清除了第一个障碍。在书架上增加几本书会导致翻转角度成为一个问题，所以漫游者得到了一个倒 V 型底盘。随着障碍物高度的增加，电池被移动以更好地分配重量，但真正的改进是增加了一个致动中间关节，将其变成了一个轮式[尺蠖](https://hackaday.com/2020/10/18/worm-bot-inches-along-as-youd-expect/)。清理突出的悬梁和缝隙只是找到正确技术的问题。

由于乐高的模块化，所有这些都可以在一两个小时内完成，而 3D 打印机和 CAD 可能需要几天时间。这个机器人确实有不能转弯的限制。传统汽车转向或麦克纳姆车轮是两种选择，但你会怎么做？
