# 这些剥线钳是由多氯联苯制成的

> 原文：<https://hackaday.com/2019/04/13/these-wire-strippers-are-made-from-pcbs/>

廉价 PCB 制造的兴起使得绕线等老式原型制作方法有些过时，但它仍有一席之地。如果你要绕线，你会想要一个快速简单的方法来剥去那根纤细的 Kynar 绝缘电线。那么为什么不用 PCB 材料来制作[这种方便的绕线剥线钳](https://www.danielrp.net/projects/proj-wrapping-wire-stripper)？

[danielrp]制造的工具非常简单——只是将一对刀片放在一起，形成一个狭窄的槽来切割绝缘层，同时不接触导体。该工具的主体由两个 PCB 形成，刀片夹在这两个 PCB 之间。[danielrp]在 DraftSight 中设计 PCB 的轮廓，然后将 DXF 导出到 EAGLE 中以制作 Gerbers。制作好的电路板需要一点后期处理，包括在一侧开孔，以接受将工具固定在一起的螺钉。我们感到惊讶的是，FR4 采取了线程，但它似乎适用于这种低扭矩的应用。将一次性卡扣型刀片夹在 PCB 之间，并调节它们之间的间隙以进行无划痕剥离。下面的视频展示了设计和构建过程。

我们总是欣赏自制的工具，事实上你几乎可以免费得到一堆印刷电路板，这让我们想知道我们还能用它们做什么。我们最近看到它们被用在[一个独特的字钟](https://hackaday.com/2019/03/27/leds-shine-through-pcb-on-this-tiny-word-clock/)，甚至变成了[一个折叠电路雕塑](https://hackaday.com/2019/01/13/cnc-turns-a-single-pcb-into-origami-hemisphere/)。
