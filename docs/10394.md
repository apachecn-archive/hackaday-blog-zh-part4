# 含酒精的平滑 3D 打印

> 原文：<https://hackaday.com/2021/06/14/smooth-3d-prints-with-alcohol/>

曾经有一段时间，大多数 3D 打印机使用 ABS，这是一种很好的韧性塑料，但很难打印，因为它容易弯曲。更糟糕的是，它很臭，而且烟雾可能对你有害。如今，大多数人已经转而使用 PLA 印刷，但你可能会错过这种更宽容的塑料的一件事是丙酮蒸汽平滑；平滑的打印不会显示层线，看起来更像没有通过喷嘴的塑料部件。

[主要硬件]喜欢蒸汽平滑部件的外观，但不喜欢与 ABS 和丙酮烟雾一起工作，[所以他开始使用 Polysmooth](https://www.youtube.com/watch?v=Swxp6LFpPhg) 。正如你在下面的视频中看到的，结果看起来不错，但要警告的是，灯丝相对昂贵。此外，你需要使用一台 300 美元的机器将你的酒精雾化成雾状。我们确信你可以花更少的钱做同样的事情，因为它看起来就像一个加湿器，但我们也建议小心将易燃物质放入消费级加湿器，当然不要使用蒸发器。

长丝听起来像是与 PLA 一样易于印刷。该材料的玻璃化温度高于 PLA，但低于 ABS。拉伸强度和杨氏模量(刚度的一种量度)数值与 ABS 相当。虽然所有的平滑都有一些不完美的地方，你可能需要试验时间和其他参数。平滑确实融合了一些可移动的关节，所以任何移动或贴合在一起的东西都可能是这个过程的坏候选。我们还听说薄壁零件在水中会因酒精残留而变软，但你可以干燥或浸泡零件以避免这种情况。

如果你想自己动手制作薄雾，这个[可能会让你开始](https://hackaday.com/2015/05/16/ultrasonic-misting-vapor-polisher-for-3d-printed-parts/)。毕竟，如果它能处理丙酮，我们认为酒精也不会更糟。虽然它不像酒精那样容易处理，但我们听说 THF 或乙酸乙酯等溶剂可以平滑常规 PLA。[热风枪和明火](https://hackaday.com/2018/05/03/smooth-pla-through-the-fire-and-flames/)也很受欢迎。
