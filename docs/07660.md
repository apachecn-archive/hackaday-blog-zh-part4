# 不要在更新状态上懈怠

> 原文:[https://hack aday . com/2020/09/02/dont-slack-off-on-updating-your-status/](https://hackaday.com/2020/09/02/dont-slack-off-on-updating-your-status/)

2020 年，在 Slack(或任何其他雇主提供的你可能被迫运行的协作程序)中显示一个准确的状态是至关重要的。如果你需要做一个三明治或带狗出去，但你没有更新你的状态以显示你不在，你可能会回到一串越来越担心或沮丧的信息，带有许多烦人的问号和偶尔的询问。

[【Becky Stern】认为一个物理界面将是一种更有趣的方式来记录她的状态，也是一个很好的视觉提醒来真正做到这一点](https://www.instructables.com/id/Slack-Status-Updater-With-ESP8266/)。我们完全同意。盒子里面是一个 NodeMCU，它使用[Brian Lough]的用于 Arduino 的 Slack API 库。这使得[Becky]可以很容易地创建状态的开关/案例选择器，在每个状态中，她可以将在线令牌设置为自动或离开，并显示带有适当表情符号的自定义消息。这些当然与半圆形选择器的表情符号相匹配，这是一个带有非常好的旋钮的旋转开关。

当我们谈到懈怠通知者的话题时，当有人提到你时，一只陪伴你的猫挥挥手怎么样？
