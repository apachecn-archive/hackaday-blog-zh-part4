# 建立一个两栖机器人使用游泳池面条为车轮

> 原文：<https://hackaday.com/2021/08/18/build-an-amphibious-robot-using-pool-noodles-for-wheels/>

如果你只认为轮子是圆的，你就限制了自己去体验更广阔的怪异设计世界。[wadevag]一直在试验一些这样的概念，并成功地用泳池面条制成的星形轮子[建造了一个两栖机器人平台。](https://www.instructables.com/Amphibious-Robots-Using-Pool-Noodle-Wheels/)

这个概念类似于 whegs 的概念。轮腿的组合，它们实际上是一种旋转运动的腿。本质上，轮子上的星星点点就像腿一样，一个接一个地推动机器人前进，而不是像典型的圆形轮子那样与地面持续接触。

泡沫提供的浮力使机器人能够轻松地坐在水面上，星形形状也使它们能够充当可行的桨。这也许是他们的主要优势。一个圆形的轮子不能提供如此大的向前推进力。

[wadevag]展示了这一概念在水、混凝土和雪地上的能力，它巧妙地处理了所有这些问题。令人印象深刻的是，它可以依靠自身的力量进出水面。虽然对于一个非常重的机器人来说，这可能不是一个可行的解决方案，但对于一个轻量级的设计来说，它可以创造奇迹。[我们也不是第一次看到一些古怪的车轮设计](https://hackaday.com/2021/05/20/3d-printing-omni-balls-for-robot-locomotion/)。休息后的视频。
