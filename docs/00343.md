# 用棉花糖和计算机视觉强化快乐的面孔

> 原文:[https://hack aday . com/2018/08/14/reinforce-happy-faces with-marshmallow-and-computer-vision/](https://hackaday.com/2018/08/14/reinforce-happy-faces-with-marshmallows-and-computer-vision/)

宾·克罗斯比有一首著名的歌曲“让微笑成为你的雨伞”然而，乔治·卡林说，“让微笑成为你的雨伞，你最终会满脸雨水。”[BebBrabyn]可能更同意前者，并使用带有 Open CV 的树莓 Pi 来检测微笑，这是一些数码相机很久以来就有的功能。然而，这个项目并没有采取快照。它使用电动弹射器发射棉花糖。我们想知道他最初是不是尝试了柠檬糖，直到太多的人没能正确地抓住它们。

对于年轻人来说，这是一个不错的项目——正如下面的视频所示——尽管你可能需要做一些工作来复制它。这个弹射器是由一个坏掉的儿童玩具升级而来的。你可能不得不跑去玩具店或者自己动手装配一些东西。也许你可以 3D 打印它，或者用投石机或压缩空气代替它。

在我们心情不好的时候，我们可以想到其他我们在发现微笑时可能想做的事情，但那会破坏这个轻松项目的精神。我们有点失望，虽然，没有自动加载的弹射器。也许这将是第二阶段。

我们更有可能使用 launcher 来处理狗食，但这是一个很好的例子，说明将 Open CV 烘焙到 Pi 项目中是多么容易。肯定没有[运动追踪空气软枪](https://hackaday.com/2017/08/04/opencv-turret-tracks-motion-busts-airsoft-pellets/)那么暴力。如果你想试着用气动加农炮取代弹射器，你[可以从这里](https://hackaday.com/2014/01/13/the-face-tracking-confectionery-cannon/)开始。
