# 解除双三极管的封装

> 原文:[https://hack aday . com/2020/06/21/解封-a-dual-triode/](https://hackaday.com/2020/06/21/decapsulating-a-dual-triode/)

我们看到相当多的工作，人们解封 IC 或其他固态设备，以暴露其内部工作。但是空心州呢？[Tomtektest]有一个失去真空完整性的双三极管——他称之为空气——并决定将其开放以更好地暴露其内部工作方式。(视频，嵌在下面。)

当然，你总是可以透过玻璃看到里面的东西，但是把信封放在一边很有趣。显然，如果你不想在移除玻璃时损坏工作部件，如何移除玻璃有点棘手。

有趣的是，电子管的元素明显分开，正如你在双三极管中所期望的那样。然而，其中一个子管道比另一个小得多。[Tom]数据手册显示，两个三极管的规格非常不同。较小的一个通常用作振荡器，较大的一个通常用作放大器。

尽管我们不再像过去那样经常使用电子管，但它有着悠久的历史。然而，在一些地区已经有了复苏，一些人正在[制造他们自己的](https://hackaday.com/2016/05/04/home-brew-vacuum-tubes-are-easier-than-you-think/)。
