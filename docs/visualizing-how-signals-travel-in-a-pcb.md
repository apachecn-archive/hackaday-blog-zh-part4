# 可视化信号在 PCB 中的传输方式

> 原文：<https://hackaday.com/2019/05/29/visualizing-how-signals-travel-in-a-pcb/>

如果你玩高速设计玩得太久，最终你会遇到时钟偏移和其他奇怪的效果。[Robert Feranec]最近遇到了这个问题，并找到了一个有趣的解决方案来可视化 PCB 中的电场。

在我们深入研究这个之前，有一个警告，对于我们在 Hackaday 上看到的大多数项目来说，像这样的东西完全是多余的。这里没有多少人处理高速接口，也没有多少人处理 100 千兆位每秒的数据链路。也就是说，这不是闻所未闻的，至少看起来很有趣。

本视频的基本内容是在(虚拟)印刷电路板上模拟差分对中的信号。这个软件叫做[Simbeor](http://www.simberian.com/),【Robert】在看了一段关于[模拟微分轨迹电场](https://www.youtube.com/watch?v=C8EYieACvrI)的视频后，与这个软件背后的公司创始人进行了交谈。这个软件说到做到，很好地说明了为什么差分对必须具有相同的长度。

虽然这可能不适合所有人，但它是高速设计中信号的奇妙可视化，超出了 Spice 仿真的预期。即使你不做高速设计，有一天你可能会对电子如何工作有一个直观的了解，这永远不会太早。
