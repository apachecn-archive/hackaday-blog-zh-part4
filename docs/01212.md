# 研究人员试图创造数字嗅觉界面

> 原文：<https://hackaday.com/2018/11/15/researchers-seek-to-create-the-digital-smell-interface/>

我们听到数字音频，看到数字视频，感受到数字触觉反馈。然而，我们没有嗅觉的模拟物。[Kasun]和他来自马来西亚幻想工程研究所的研究小组正在改变这一现实。他们的项目[旨在通过电子刺激传递香味](https://spectrum.ieee.org/the-human-os/biomedical/devices/these-researchers-want-to-send-smells-over-the-internet)。尽管这实际上是朝着创造多感官互联网迈出的一步。

该团队的“电子嗅觉机器”由一个可变电源组成，连接到缠绕在内窥镜摄像头周围的银电极。当电流脉冲通过电极时，相机是必要的，以确保与用户的嗅球接触。电流值根据被复制的气味而变化，并且在 0.2 毫安附近。该机器的早期试验显示，大约四分之一的测试对象能够识别被复制的气味。他们报告说气味有水果味、甜味和木质味，尽管所有的气味都带有类似化学物质的气味。

“嗅觉视觉”的概念并不新鲜，因为它比有声电影存在的时间还要长。以前试图给电影和电视配上香味是化学反应的结果。这些类型实验的装置通常包括当化学物质耗尽时需要更换的药筒。[Kasun]的团队方法是避免化学方法，而是直接刺激嗅觉受体。那些对具体细节感兴趣的人可以阅读 T2 关于数字化气味的研究论文。
