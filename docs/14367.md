# 安全地展示危险

> 原文:[https://hackaday.com/2022/07/30/demonstrate-danger-safely/](https://hackaday.com/2022/07/30/demonstrate-danger-safely/)

在播客中，丹·马洛尼和我谈论了打断一个孩子手指的机器人手臂，事实证明我们都非常尊重机器人手臂，特别是机器人手臂。Dan 有过一个编码器轮损坏导致失控行为的故事，这几乎击中了他，我甚至不会靠近这些东西，除非我在看到一台完美运行的机器中的编程错误对 2×4 的影响后知道它们已经关闭。

这让我想到了我做过的所有危险的事情，但是除此之外，我还想到了所有可以让它们变得不危险的非常简单的预防措施，我认为这是非常重要的。在我的清单上名列前茅的是前面提到的工业机器人手臂和高功率激光器。

使用工业机器人手臂保持安全就像当它通电时保持伸手可及一样容易。我们的程序是在地板上画一条线，描绘出手臂的最大半径，当灯亮着的时候，你总是呆在这条线的外面。这并不是万无一失的，因为你可以给机器人一个高尔夫球杆或其他东西，但这是一个很好的最低预防措施。当你需要进入这条线的时候，你会关掉它。许多工业机器人生活在门上有联锁装置的笼子里，这是有充分理由的。

激光安全也差不多。你需要知道光束要去哪里，确保它被适当地端接，千万不要射到眼睛里。这可以像把设备放在盒子里一样简单:激光留在盒子里，没有人会失明。如果你需要看到里面，网络摄像头是不可思议的。但有时你需要聚焦或校准激光，然后你戴上激光安全眼镜，认真思考光束的方向。做完后你再关上盒子。

这些安全措施没有一个实施起来特别有挑战性，或者在概念上很难:在地板上画一条线，把它放进一个盒子里。最近有一系列关于安全制作 Lichtenberg 数字的视频，作为高压项目的一般规则，一个很好的预防措施是两个按钮的紧急开关盒。这至少保证了当高压接通时，你的双手不在高压附近，代价是两个开关。

如果所有的安全预防措施都很简单，一旦你听说过，我自己也不会想到。我都是从其他黑客那里学来的。我车间里的台锯也是如此，甚至开车也是如此。但是由于更多的黑客行为不太常见，古怪领域的“常识性”安全预防措施就不太为人所知了。作为知道安全秘密并与他人分享的人，这是我们的工作。当你做危险的事情时，展示你的安全技巧！
