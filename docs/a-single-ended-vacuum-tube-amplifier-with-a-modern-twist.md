# 具有现代风格的单端真空管放大器

> 原文：<https://hackaday.com/2023/01/03/a-single-ended-vacuum-tube-amplifier-with-a-modern-twist/>

尽管最古老的固态音频电路现在有资格获得养老金和公共汽车通行证，但热离子管仍在音频世界中占有一席之地——因其据称能传递温暖的声音而受到发烧友的喜爱。对于电子发烧友来说，电子管音频放大器是一个有趣而不寻常的项目，因此它是许多人关注的对象。[Keri Szafir]也不例外，她生产了一款带有一些现代特征的立体声电子管放大器。

电气上，它是一个相对传统的单端设计，每个通道使用一个双三极管和一个功率五极管。它遵循一种所谓的超线性电路，在输出变压器上有一个抽头，为五极管的一个栅极供电。现代功能通过开关蓝牙输入和电动音量控制来实现，这在这样的放大器上是绝无仅有的，因为它们是最先进的。

我们不得不承认这种类型的放大器的弱点，我们特别喜欢这一个非常时期的建筑风格，使用电缆系带来保持布线在控制之下。我们更经常看到这些放大器[使用更便宜的集成三极管五极管](https://hackaday.com/2022/12/26/tube-audio-amplifiers-neednt-be-complex/)，这使得它们特别容易制造，所以单独的前置放大器有点不同。不过，我们不确定我们是否会额外花费在这些花哨的 E88cc 电子管上。
