# 全轮自行车原型机器人

> 原文:[https://hack aday . com/2022/02/24/prototype-robot-for-omni wheel-bicycle/](https://hackaday.com/2022/02/24/prototype-robot-for-omniwheel-bicycle/)

尽管科学有能力以各种可感知的方式推进现代社会，但它仍然需要解释一些看似基本的概念。在我们的理解中仍然有一些漏洞的一件事是自行车工作的方法。当然，我们已经知道足够制造功能自行车，但就像重力被纳入标准模型一样，我们还没有找出一组方程来管理宇宙中所有的自行车。然而，为了进一步加深我们对自行车的理解，一些人正在进行实验[，比如这个自平衡全轮自行车机器人](https://www.youtube.com/watch?v=NCW0HhTaww8)。

功能转向对于让自行车朝着正确的方向行驶很重要，但对于保持自行车直立也很关键。这就是[詹姆斯·布鲁顿]测试全向轮的地方。通过将它放置在自行车的前面，垂直于行驶方向，他既可以驾驶自行车机器人，又可以保持平衡。这确实需要 Arduino Mega 与惯性测量单元配对的计算工作，但最终[James]有了一个功能性自行车机器人，他可以用它来实验不同转向方法对自行车的影响。

虽然他还没有一辆人类用的全轮自行车，但我们至少希望这是[詹姆斯]或任何人建造一辆前部有全轮的真正自行车的重要一步。希望这很快成为现实，但与此同时，我们将不得不满足于拥有普通车轮的自行车，这些自行车可以平衡并自动驾驶。
