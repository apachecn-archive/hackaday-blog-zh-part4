# 用覆盆子酱冲击世界

> 原文：<https://hackaday.com/2019/07/29/punch-the-world-with-a-raspberry-pi/>

机器人无疑让世界变得更加美好。事实上，从汽车装配到食品生产的每一件事情都在过程中的某一点上使用机器人，更不用说那些可以打扫你的房子或为你煮早餐咖啡的机器人了。但并不是每个机器人都需要这样一个生产目的。这个可以让你猛击世界，虽然不能像装配线上的焊接机器人那样产生大量的物理价值，但至少似乎有一些治疗效果。

物联网星球穿孔机来自[8BitsAndAByte]，他们建造了许多功能同样可疑的不同东西。这个可以让我们通过猛击这个世界(或者说，它的一个小模型)来释放我们对这个世界的挫败感。一个小的着色球体位于安装在线性致动器上的 3D 打印拳击手套前面。线性致动器由树莓 Pi 驱动。然而，Pi 的工作并没有就此结束，因为该项目还使用 Pi 相机拍摄全球视频，并将其发布到网页上，任何人都可以通过网页控制拳击手套。

虽然没有立即派上用场，但我们确实很开心地敲了几下，有一次一只神秘的手也进入了镜头，对系统进行了调整。像这样的项目很有趣，有时你只需要建造一些东西，即使是愚蠢的东西，因为这种冲动会打击你。
