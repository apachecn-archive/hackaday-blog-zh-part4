# 谢妮时钟故障分析，[Dali bor farn]风格

> 原文：<https://hackaday.com/2019/10/13/nixie-clock-failure-analysis-dalibor-farny-style/>

可悲的是，我们已经习惯了保修期一过，消费类设备似乎就要断气了。甚至当我们“幸运”地发现设备在被覆盖的情况下发生故障时，也有可能没有人尝试去修复它；该设备将被替换为新的，而出现故障的设备将被扔进电子垃圾箱。

然而，并不是每个制造商都采取这种方法。当优质是你品牌的基石时，你需要认真对待实地失败。[Dali bor farn]是高端谢妮电子管及其插入的圆滑、复杂的时钟的制造商，他意识到了这一点，一个新的视频[深入介绍了他用来诊断问题并在未来预防问题的过程。](https://www.youtube.com/watch?v=Dd72TYVFbo8)

一个数字被卡住的时钟被追溯到由桶疲劳引起的通孔故障，或者通孔内的电路板材料开裂并破坏了镀通铜。这促使电路板重新设计，以增加所有过孔的直径，消除这种故障模式。另一个时钟有一个数字卡在上面，结果是引脚错位导致对地短路；当管子插入时，引脚滑动，将一些焊料从插座上刮到电路板的接地层上。这导致了另一次重新设计，不仅通过取消电路板上侧的接地层解决了问题，还极大地改善了电路板的美观。

与所有的东西[Dalibor]一样，该视频是一场视觉盛宴，抛光玻璃和铬管中温暖的橙色光芒与珠光处理的铝底盘形成鲜明对比。如果你还没有看过视频的制作，你也应该去看看。
