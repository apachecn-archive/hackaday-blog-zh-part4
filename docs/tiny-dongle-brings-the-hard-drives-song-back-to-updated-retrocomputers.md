# 微型加密狗将硬盘上的歌曲带回更新的复古电脑

> 原文：<https://hackaday.com/2022/09/26/tiny-dongle-brings-the-hard-drives-song-back-to-updated-retrocomputers/>

回到计算的“米色盒子”时代，只要听一听就很容易知道你的机器在做什么，因为硬盘不断地来回晃动。这有时很烦人，但从来没有像听到盖革计数器一样的咔哒声停止时那么烦人，因为你知道它还没有加载完一个程序。

这种“快乐的声音”越来越难获得，即使是在复古的机器上，它们越来越多地用紧凑型闪存和其他固态驱动器取代了原来的 thrash-o-matic 驱动器。这款硬盘声音模拟器旨在填补任何对你来说不够响亮的机器上的诊断和怀旧空白。遗憾的是，[马蒂亚斯·沃纳]没有为他的创作提供任何构建细节，但在下面的长演示视频(由一位满意的客户制作)和第一版[的细节之间，很容易就能看出这里发生了什么。一个 ATtiny 和几个支撑元件以及一个压电扬声器安装在一个小 PCB 上。加密狗连接到硬盘活动指示灯，它会触发扬声器发出一连串的咔哒声，听起来非常像硬盘正在寻道。下面视频中 7:09 开始一个 demo 非常勇敢的人——或者非常怀旧的人——可能想看看从 13:11 开始的完整碎片整理。](https://www.retrokits.de/index.php/hdd-clicker-hdd-sound-simulator-v0-1/)

当然，这个可能有点夸张，但是在逆向计算的世界里，以怀旧的名义付出任何代价都不为过。而且它还远远不是我们见过的最荒谬的硬盘活动指标。
