# 打印出来:手摇摄影转盘

> 原文:[https://hack aday . com/2019/07/24/printed-it-hand-cranked-photography-turntable/](https://hackaday.com/2019/07/24/printed-it-hand-cranked-photography-turntable/)

即使是相对低端的桌面 3D 打印机，在为你的最新项目运行定制外壳或部件时也不会有问题，对许多人来说，这比入场费更值得。但是，如果你愿意投入时间和精力来精通必要的 CAD 工具，即使是一台基本的 3D 打印机也能够生产复杂的小工具和机械装置，这将非常耗时或难以用传统的制造技术生产。

[![](../Images/3f6d8c1328a003fe475e758313acba78.png)](https://hackaday.com/wp-content/uploads/2019/07/3dptable_internal.jpg)

Printable bearing cross-section

一旦你发现自己处于 3D 打印职业生涯的这个阶段，就有了一个岔路口。最常见的途径是设计零件，印刷后用胶水或标准紧固件组装。这无疑是最简单的方法，让您以非常熟悉的方式使用打印的零件。如果你想把自己的打印部件和现有的硬件结合起来，这也是很有优势的。

另一个选择是完全接受 3D 打印的独特功能。忘记螺母和螺栓，而是设计卡扣在一起的组件。开始使用更多的有机形状和曲线。理解对象不再局限于简单的实体，可以有自己复杂的内部几何形状。铰链真的需要由两个独立的部分用一个大头针连接起来吗，或者你可以通过将一个印刷部分放在另一个里面来实现想要的动作吗？

如果你愿意走这条少有人走的路，有一天你可能会发现自己在创造设计，比如 Brian Brocken 的[这个全 3D 打印的转盘。该设计旨在拍摄或 3D 扫描小物体，而不会破坏银行，不使用滚珠轴承，螺钉，甚至胶水。每一个部件都是打印的，并通过摩擦或集成锁定功能装配在一起。这是一个可以随时随地打印和使用的功能设备。你可以在国际空间站打印其中的一个，而不必等待麦克马斯特-卡尔公司的命令来完成它。](https://www.thingiverse.com/thing:3723618)

有了这样一个聪明的设计，我忍不住仔细看看它是如何工作的，它是如何打印的，甚至可能是一些可以适应或改进的方法。

## 避免失望

随着曲柄，齿轮，和一个巨大的联锁轴承组件，你当然不能责怪认为这将是一个具有挑战性的打印。但是 Brian 设法降低了他的转盘的复杂性，至少部分是通过相对较低的零件数量。事实上，转盘轴承本身打印在一个零件有助于实现这一目标。

[![](../Images/7f8a497e7446dc69ede63cfbd6275e5a.png)](https://hackaday.com/wp-content/uploads/2019/07/3dptable_printing.jpg)

Printing the bearing. Note locking tab on inner ring.

当然，你需要有一台相对较好的打印机，但是当你想打印比简单的支架更复杂的东西时，这是不言而喻的。任何就地印刷设计的诀窍(这是 FDM 对包含移动元素的单一印刷的说法)是确保各部分不会融合在一起。所以[要注意的一个主要问题是过度挤压](https://hackaday.com/2016/01/27/3d-printer-tool-set-your-extruder-steps-with-ease/)，这往往会使物体比其应有的厚度更厚。

也就是说，Brian 已经使公差足够宽，即使是入门级打印机也不会有问题。事实上，这些间隙可能有点太宽容了。虽然这不是一个关键问题，但我发现在旋转中有一点点多余的间隙，因为内圈要小得多。除此之外，所有其他组件的配合都非常到位，所有东西都只靠摩擦力结合在一起。

如果这个设计有任何主要的缺陷，那就是在打印轴承时，你需要基本上完美的第一层粘合，因为木筏或边缘几乎肯定会导致零件在早期融合。好消息是，圆形的外部形状不太容易卷曲或以其他方式脱离床。不幸的是，如果你的打印机不能干净地打印出最初的细丝珠，内环上的所有齿都可能成为绊脚石。

## 印刷和组装

如前所述，这个项目的主要障碍是打印轴承。这不仅是最难制作的部分，也是最耗时的部分。尽管如此，这是我建议开始的地方；因为如果你不能把轴承打印出来并自由旋转，那么就没有必要再为其他部件费心了。在我的 Prusa i3 MK3 上，打印 0.2 毫米层高的轴承花了六个多小时。

[![](../Images/171280904730a1c357fea907af20c138.png)](https://hackaday.com/wp-content/uploads/2019/07/3dptable_pipes.png)

You’ll need to enable support for this piece.

一旦轴承出了问题，其余的打印真的很顺利。就像腿一样简单，我一次完成了三个，总共打印了大约两个小时。手柄和齿轮每个都需要大约一个小时来打印，并且没有任何问题。一个小麻烦是，加载时许多部分相对于床身的方向不正确，所以在切片前需要手动旋转它们。这是一件小事，但希望 Brian 能在 STLs 的后续更新中解决它，因为它确实让经验不足的用户更难获得好的结果。

虽然没有轴承那么难，但当你到达“管道”时，你可能会想让事情慢一点，这就是 Brian 所说的保持齿轮的部分。这部分不仅有点细长，而且具有集成的锁定机制，需要一些特殊的照顾。我将图层高度降低到 0.1 毫米，并启用自动支持生成。我可能会使用 PrusaSlicer 中新的自定义支持选项对[进行一些改进，但默认设置做得相当不错，清理也不算太差。](https://hackaday.com/2019/05/24/3d-printering-the-past-and-future-of-prusas-slicer/)

一旦所有的部分都被打印出来，这项任务将花费你大约 14 个小时，是时候将你的塑料小块组装成一个工作转盘了。这个过程是不言自明的，但是如果你有任何不确定的地方，Brian 提供了一个简短的视频，很好地直观地解释了所有东西应该如何组合在一起。

## 个人分歧

[![](../Images/2cd3766c3e21152340f7b20e56af0a73.png)](https://hackaday.com/wp-content/uploads/2019/07/3dptable_platev2.jpg)

A faster printing version of the top plate.

总的来说，我认为 Brian 在这个设计上做得非常好，这是一个如何发挥桌面 3D 打印优势的完美例子。不过，有一个设计元素，我认为可能会受益于不同的方法。最初，转盘的顶部是一个可移动的盘子，上面贴着一张纸。考虑到所涉及的巨大表面积，印刷这种固体板需要三个多小时。这似乎没有必要，因为它无论如何都会被纸覆盖。

对于我自己的使用，我对原来的顶板 STL 进行了修改，以最小化与打印机底座接触的表面积。通过使外径变小，并用一个简单的十字代替实心中心，我能够将这篇文章的打印时间减少到一个小时以下。布赖恩说，他也在寻找替代的表面几何形状，可以用来减少打印时间和材料的这件特殊的作品。

## 科学纺纱

自从打印了这个转盘，我的大部分空闲时间都用来拍摄旋转物体的视频和图片。知道它的每个部件都是你自己生产的，使用这个小玩意会让你非常满意。许多被称为“完全可打印”的设计通常仍然需要几个螺丝、轴承或其他机械部件来完成，但这一个不是。

[![](../Images/a7e8fb1c9bcb189a4de9ff218c5b0a35.png)T2】](https://hackaday.com/wp-content/uploads/2019/07/3dptable_anim.gif)

虽然在目前的形式下，这是一个有趣的玩具，但我发现自己在想未来可能对设计进行改进，允许你移除曲柄组件，并用一个小型步进电机取代它。这将允许视频中更平滑的旋转，并且[让你很好地走向自动化摄影测量装备](https://hackaday.com/2019/04/07/get-great-3d-scans-with-open-photogrammetry/)。我已经向布莱恩提到了这个想法，所以谁知道呢，也许将来这个小转盘会学到一些新的技巧。
