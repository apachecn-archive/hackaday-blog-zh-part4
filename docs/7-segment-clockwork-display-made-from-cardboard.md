# 由纸板制成的 7 段发条显示器

> 原文：<https://hackaday.com/2018/10/31/7-segment-clockwork-display-made-from-cardboard/>

我们过去已经见过各种古怪的 7 段显示器，但这款使用了一种新材料:既用于显示器，也用于驱动它的机械装置；[纸板](https://www.youtube.com/watch?v=la30_i-NDFM)。是的，整个东西是由纸板、木头和一些橡皮筋做成的。[The Q]展示了他是如何在这个漂亮的视频中组装起来的，从展示片段如何制作的基本原则开始:简单的纸板片，一面涂上荧光颜料。一块木头将元素推出来使其空白，每个元素都连接到一个凸轮，凸轮将木头推入或推出。

真正聪明的一点是,[ Q]将数字 0-9 映射到一个矩阵上，7 个段中的哪一个是“开”或“关”。然后，他利用这些信息在一个中心轴上创建了一组 7 个凸轮。当你转动轮轴时，凸轮转动，移动锯臂。然后，当这些手臂通过数字向上计数时，它们会使元素翻转。本质上，他设计了一个物理的十进制到七进制解码器，很像 [SN74LS47](http://www.ti.com/product/SN74LS47) 中的电子解码器。整个组件由一个旋钮覆盖，该旋钮指示当前显示的数字。如果你喜欢这样的机械展示，看看这个由乐高部件制成的[展示](https://hackaday.com/2014/08/07/lego-technic-mechanical-seven-segment-display/)，或者[这个令人敬畏的 3D 打印作品](https://hackaday.com/2017/12/19/a-gloriously-impractical-electromechanical-display/)。
