# 稳健的手将 GBA 推车从坟墓中带回来

> 原文:[https://hack aday . com/2021/06/22/steady-hand-brings-GBA-cart-back-from the grave/](https://hackaday.com/2021/06/22/steady-hand-brings-gba-cart-back-from-the-grave/)

Game Boy Advance (GBA)卡盒中使用的闪存芯片旨在比早期系统中用于保存玩家进度的电池支持的 SRAM 更可靠，体积更小。但随着一些 GBA 游戏现在迎来了它们的 20 周年纪念日，旧的购物车在加载保存或创建新的保存时遇到问题也不是没有听说过。也许这就是为什么前主人试图在他们的*金太阳*、[副本上回流闪存芯片，但正如[泰勒·伯利]在打开箱子](https://hackaday.io/project/180347-badly-damaged-gameboy-advance-game-lives-again)后发现的那样，他们最终只是让情况变得更糟。

[![](../Images/b4c097ff1a6a028b34e1e8765a283d37.png)](https://hackaday.com/wp-content/uploads/2021/06/gbafix_detail.jpg)

A previous repair attempt left the PCB badly damaged.

当 PCB 上出现这么多损坏的迹线时，最合理的做法是获得一个供体墨盒并交换 save 芯片。但是在易贝快速查看一下就会发现《金太阳》的拷贝并不便宜。所以[泰勒]决定弯曲他的焊接肌肉，用一根小心弯曲的 30 号线修复每一个痕迹。如果你需要你的每日剂量的禅，只需在下面的视频中观看他有条不紊的过程。

虽然这肯定不会减损[Taylor]令人印象深刻的焊接工作，但应该说盒式 PCB 的设计确实有所帮助，因为许多受损的迹线附近都有过孔，这给他提供了方便的位置来连接他的新电线。PCB 的设计似乎也是为了接受不同物理尺寸的闪存芯片，这为维修提供了一些额外的喘息空间。

看到他的杰作，你可能不会惊讶地发现这并不是[泰勒]第一次做一些挽救生命的显微手术。就在去年，他修复了一个 XBox 控制器的印刷电路板，这个电路板被折断成了两半。
