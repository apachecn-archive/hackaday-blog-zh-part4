# 偏执狂的 Linux 为你工作

> 原文:[https://hacka day . com/2021/12/22/为偏执者工作的 Linux/](https://hackaday.com/2021/12/22/linux-for-the-paranoid-does-the-work-for-you/)

我们都知道，追踪我们在互联网上的活动并不难。只是有些人比其他人更讨厌。如果你真的是铁杆粉丝，你会学习人际关系网的所有细节来帮助掩盖你的踪迹，但是如果你不想投入那样的时间呢？或许，正如[TechRepublic]所建议的，试试 [Kodachi Linux](https://www.techrepublic.com/article/kodachi-is-the-operating-system-for-those-who-value-privacy-but-dont-want-to-learn-linux/) 。

当然，你可以从你自己的现场图像开始。然后，当您启动时，您可以采取以下步骤:

*   随机化您的 MAC 地址
*   通过 VPN 建立 TOR 连接
*   通过 TOR 路由所有互联网流量，并使用 DNS 加密
*   设置一个计划任务，定期对您的 MAC 地址进行加扰

但这就是哥达奇所做的事情，你并不需要付出任何努力。

发行版基于 Ubuntu，所以所有熟悉的工具都在那里。还包括一些安全和隐私工具，如 KeePass、Tox、OnionShare、i2p 等等。桌面显示安全网络信息的摘要

你需要柯达吗？可能不会，如果你是一个 Linux 专家的话。另外，大多数人不会做任何有趣的事情。但是如果你想保护你的隐私或者你有什么打算，试试柯达。话又说回来，如果你真的那么偏执，也许这正是他们想要你做的。自己做决定。你也可以看看下面【eBuzz Central】的视频评论。
