# 疯狂还是天才？FDM 树脂印花

> 原文：<https://hackaday.com/2022/11/17/madness-or-genius-fdm-printing-with-resin/>

我们不知道是什么让他想到了这个主意，但是[适当印刷]决定让一台 [FDM 打印机打印树脂而不是灯丝](https://www.youtube.com/watch?v=aRH8MGxnZdY)。为什么？我们仍然不确定，但我们仍然钦佩这种努力。原则上，挤压树脂应该与你打印的其他液体东西(如糖衣或混凝土)没有太大区别。然后你需要快速地用紫外线固化这种粘稠的液体。事实上，他们最终用几种化学物质和一种填料制成了一种糊状树脂。

用浆糊武装起来，似乎大的障碍就要过去了。取代部分冷却风扇，打印机现在有两个激光头聚焦在打印区域。以花瓶模式打印避免了一些问题，但最初的几次尝试并不太成功。

只要有一点毅力，这个设置确实奏效了——在一段时间内。更多的微调得到了可接受的结果。然而，他最终改变了填充材料，得到了一个过得去的 Benchy 来打印。没什么值得骄傲的，但是有辨识度。老实说，我们感到惊讶的是，激光没有治愈仍然在喷嘴内的材料，并导致可怕的堵塞。

为什么要花这么大力气做这件事？我们不知道。你应该试试吗？大概不会。当然，能够打印出浆糊也是有其价值的。例如，可能传送胶水或焊膏。但是你通常不需要用那种材料制作高的印刷品。话又说回来，我们从来不反对“仅仅因为”而做某事。

毕竟，为什么要用一个游戏男孩做一个[乐器呢？为什么要用锡罐做一个](https://hackaday.com/2018/10/24/this-ukulele-does-chiptunes-and-not-just-because-its-made-out-of-a-game-boy/)[调制解调器](https://hackaday.com/2020/07/18/a-tin-can-modem-just-for-fun/)？你还不如挤出树脂。
