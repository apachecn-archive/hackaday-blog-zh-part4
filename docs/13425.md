# 乘式割草机 CVT 升级真的让事情动起来

> 原文:[https://hack aday . com/2022/04/23/riding-mower-CVT-upgrade-really-gets-things-moving/](https://hackaday.com/2022/04/23/riding-mower-cvt-upgrade-really-gets-things-moving/)

正如我们从过去的经验中了解到的，来自[HowToLou]的视频往往会有点争议。他独特的权宜工程风格并不是每个人都喜欢的，尤其是当它与不顾一切的安全态度相结合时。另一方面，毫无疑问，他的方法取得了成效。他关于将 18 马力的乘式割草机改造成类似卡丁车的视频就是一个很好的例子。

该项目的第一阶段包括移除所有与割草相关的硬件，因为显然你不会在 48 公里/小时(30 英里/小时)的速度下割草。这既减轻了重量，又消除了许多机械上的复杂因素，这些因素会妨碍进一步的改进。也就是说，这也使割草机无法移动，因为发动机和变速驱动桥之间不再有任何连接。

[![](../Images/3eb4a51bb65456c50cd5b65beeda2197.png)](https://hackaday.com/wp-content/uploads/2022/04/racingmower_detail.jpg)

The new drivetrain features some beefy bracing.

在它的位置，[HowToLou]安装了现成的变矩器套件，使用无级变速器(CVT)离合器。正如他快速演示的那样，CVT 技术允许齿轮比自动适应发动机转速，这要归功于滑轮，滑轮可以根据旋转速度改变大小。这比他最初拉出的系统有了很大的改进，尽管正如你所料，将它安装到割草机上需要一些定制工作。最后一步是将旧滑轮从变速驱动桥上取下，换上一个不到原尺寸一半的滑轮。

穿着保护性的人字拖，[HowToLou]跳上加大马力的割草机，一踩油门，差点被甩到后面。显然，修改是成功的，视频以一些开放的道路测试结束——大概是他骑着马去商店买头盔。

实际上，当这个视频第一次传播时，我们错过了它，但它已经获得了动力，并吸引了一些令人印象深刻的数字。[HowToLou]告诉我们，他认为这是因为许多人正在升级到更现代的零转向割草机，这意味着市场上这些二手迷你拖拉机过剩。不管是什么原因，我们很高兴看到这个后院工程师获得一些主流的成功；他的方法[可能不总是照章办事](https://hackaday.com/2021/03/03/generator-runs-on-natural-gas-with-field-expedient-tip/)，但是[他们总是很有趣](https://hackaday.com/2022/02/13/this-minibike-will-land-you-in-hot-water/)。
