# UV 树脂完善了 3D 打印，但不是你想的那样

> 原文:[https://hack aday . com/2022/01/04/uv-resin-perfects-3d-print-but-not-how-you-think/](https://hackaday.com/2022/01/04/uv-resin-perfects-3d-print-but-not-how-you-think/)

在这一点上，每个人都知道，即使是入门级的 UV 树脂打印机，打印质量也远远超过基于细丝的熔融沉积成型(FDM)机器。但是有一个交易:为了钱，你用 FDM 可以得到更多的建造量。因此，在大幅面树脂打印机的物流得到解决之前，希望制造类似道具头盔的人别无选择，只能花大量时间对他们的打印进行后期处理，以消除明显的层线。

但是[多亏了[PropsNstuff]](https://www.youtube.com/watch?v=4rIHF2rf-Z4) 展示的这个有点讽刺的技巧，你实际上可以使用 UV 树脂来提高你的 FDM 印刷品的最终质量。这个想法是在 3D 打印的层线和其他缺陷上涂一层树脂，用手持紫外线手电筒固化，然后打磨光滑。本质上，这就像用树脂代替 Bondo 之类的身体填充物，其优点是树脂可以在几秒钟内固化。

[![](../Images/7bfd8c39fe4f76f3be285ae58e86f5a3.png)](https://hackaday.com/wp-content/uploads/2021/12/resinsmooth_detail.jpg)

The thick resin fills in tough spots quickly.

现在要明确的是，这不是一个新的想法。我们自己的[【唐纳德·帕普】在 2018 年](https://hackaday.com/2018/03/08/3d-printering-print-smoothing-tests-with-uv-resin/)调查了这一过程，[【托马斯·桑拉德勒】在第二年](https://hackaday.com/2019/12/25/finishing-fdm-prints-with-sls-resin/)自己的视频中报道了这一想法。但这里的不同之处在于，[PropsNstuff]不只是用树脂覆盖整个印刷品，他采取了一种更有条理的方法。他在小区域工作，目标是真正需要这种技术提供的高层建筑特性的区域。

棘手的问题解决后，他开始用树脂覆盖更大的面积。但这一次，他将 SLA 打印机剩余的树脂与滑石粉混合，制成一种可以刷在上面而不用到处跑的混合物。这需要几层薄薄的涂层，但通过这种混合，他能够在不丢失任何表面细节的情况下构建出大片的印花。

还麻烦吗？绝对的。但是最终的结果看起来确实很壮观，所以在我们弄清楚如何建造《星际迷航》中的复制人之前，看起来我们必须通过一点点努力来弥补我们的技术缺陷。
