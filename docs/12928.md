# 分裂皮瓣显示告诉我们这个词

> 原文:[https://hack aday . com/2022/03/02/split-flap-display-tells-us-the-word/](https://hackaday.com/2022/03/02/split-flap-display-tells-us-the-word/)

LED 和 LCD 显示器是一个技术奇迹。他们已经把电视和显示器的价格降到了自 CRT 时代以来前所未闻的水平，但这种优势可以说是伴随着审美成本。当所有的东西都被平淡无奇的电脑屏幕覆盖时，这个世界往往会显得更加单调。几十年前并非如此，当时有许多截然不同的显示信息的方式。这种不同时间的一个例子通过[这个[Erich]一直在重现](https://mcuoneclipse.com/2022/01/09/diy-split-flap-display/)的分瓣展示来呈现给我们。

分体式翻盖显示器通过在一系列翻盖上打印字母或数字来工作，这些翻盖通过步进电机连接到主轴上。马达每走一步，显示屏就转动一个字符。与现代显示器相比，它们可能会有噪音，并且需要大量的维护，但也有一些优点。[Erich]的版本由新的丙烯酸树脂和 MDF 制成，并使用 Arduino 作为控制板。3D 打印机和 CNC 机器保持足够紧密的公差，以使显示器顺利工作，并且还使他能够根据需要扩展显示器，因为每个字符显示器都是相当模块化的。

现在，[Erich]的显示器在两个不同的行上有 20 个字符，这无疑将我们带回了过去的时代，这种风格的显示器在机场和火车站非常突出。这款显示器[使用了我们几年前推出的另一款分体翻盖显示器](https://hackaday.com/2018/02/20/easy-modular-alphanumeric-displays-are-full-of-flappy-goodness/)的许多基本元素，但做了一些改进。此外，如果你更喜欢[旧陈列](https://hackaday.com/2016/04/20/split-flap-train-display-uses-punch-cards-serviced-with-station-ingenuity/)的修复，而不是现代的化身，我们也为你准备了。
