# 这个经典但混乱的齿轮时钟让你猜个不停

> 原文：<https://hackaday.com/2021/02/08/this-classy-but-chaotic-gear-clock-keeps-you-guessing/>

有很多种方法来判断时间，但几乎所有的方法都涉及某种顺序刻度——指针扫过模拟时钟的表面浮现在脑海中，数字时钟的增量变化也是如此。时钟本质上是可以预测的，因此有点无聊。

[这款无时序齿轮钟](https://funwithelectrons.blogspot.com/2021/02/none-sequential-gear-clock.html)旨在打破这种可预测性，打造一款与众不同的时计。这是[Tony Goacher]的作品，他显然做了大量的工作，并且在做的时候拿出了几乎所有的工具。他从一个激光切割胶合板原型开始，以解决基本问题——一对带有内部齿轮齿的嵌套环，每个环都挂在一个步进驱动的小齿轮上。内环代表小时，外环代表分钟，每个环上的数字或多或少是随机分布的，因为没有两个连续数字的旋转间隔超过五秒。

时钟的成品由黄铜、丙烯酸、硬木和少量铝制成，表壳让人想起昔日的大教堂收音机。有一些非常好的触摸，比如定制的黄铜螺丝，带有传统时钟艺术的 CNC 雕刻黄铜面板，以及小时环驱动齿轮上的拉丁铭文，大致翻译为“时间主宰一切”。当我们查找时，我们发现“tempus rerum imperator”是令人崇拜的钟表匠公司的座右铭，我们发现它的存在非常令人愉快。

在下面的简短视频中，时钟运行其初始化程序。我们不确定我们是否想把它放在床头柜上，但这肯定是一种独特而令人愉快的方式来展示时间的流逝。这多少让我们想起了[这个三环万年历](https://hackaday.com/2020/07/26/concentric-rings-keep-this-calendar-perpetually-up-to-date/)，但只是多了一点随机性。
