# 并非所有的 7 段显示器都是电子的

> 原文：<https://hackaday.com/2019/10/13/not-all-7-segment-displays-are-electronic/>

电子电路显示数字的方法有很多种，最普遍的可能还是七段显示器。以我们熟悉的矩形 8 字形中的 7 个发光二极管、灯、液晶显示器、VFD 段或机械翻转点样式单元为例，您的微控制器或类似设备可以显示数字。有各种不同的接口，但最多只需要一个电平转换器和一个驱动器。

虽然有时我们会遇到一种全新的 7 段显示器，例如【Fhuable】的[全机械式单数字显示器](https://cults3d.com/en/3d-model/gadget/7-segments)。它在表面上与 flipdot 显示器相似，但它不是磁性致动器，而是使用一个复杂的齿轮和凸轮系统，通过转动一个小曲柄来依次翻转各个部分。这似乎是他在他的订阅计数器项目中使用的相同机制，我们将他的视频放在了休息时间的下方，这确实是一件美丽的事情。我们不完全确定它在当前形式下作为通用显示器会有多大用处，但是，我们可以看到它相对容易地被改编。例如，时钟可能是一个引人注目的项目。

这里的大多数显示器都有一些电子元件，所以很少看到完全机械的。但情况并不总是如此。
