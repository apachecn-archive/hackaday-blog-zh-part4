# 60 年代的天然气管道电脑退役打游戏

> 原文：<https://hackaday.com/2020/05/17/60s-natural-gas-pipeline-computer-retires-to-play-games/>

自 20 世纪 60 年代以来，电脑游戏已经走过了漫长的道路。虽然那个时代的计算机可能不会运行 Doom 或任何接近它的东西，但我们许多人第一次接触到计算机是在一台老式的电传打字机上玩 Hunt the Wumpus、Adventure 或 Star Trek。如果你错过了那些日子，或者你只是错过了它们，你可能会喜欢来自[某个计算机家伙]的视频，他点燃了一台旧的退役天然气管道计算机，并在其中装入足够的纸带来播放月球着陆器。(视频嵌入下方。)

我们不怀念切换启动加载程序的日子，这样你就可以在进入你想要运行的实际程序之前加载第二个启动加载程序的纸带。

与 DEC 或 Data General 计算机相比， [SEL 810A](http://s3data.computerhistory.org/brochures/systemsengrglab.sel810a.1967.102646093.pdf) 并不是一台非常知名的计算机。公司被古尔德收购，后来又被安可收购。这台计算机实际上在当时是非常先进的，提供了一台全集成电路为基础的 16 位计算机。公司的一些部分仍然作为[康博](http://www.compro.net/about-us.html)的一部分存在。

至于月球登陆器游戏，这是一个典型的非常简单的模型，描述的是物体在特定的重力下坠落。你用有限的燃料来抵消重力。成功的着陆是以相对较慢的速度着地。后来的版本将包括左右移动击中目标，甚至一些粗糙的图形。然而，你在视频中看到的版本已经在许多计算机上出现，包括 HP-41C。

我们还喜欢看旧的 H19 航站楼。建造这些连接器时压装连接器是一件痛苦的事，但它们看起来确实不错。如果你想要一个更土气的戈德堡版本的月球着陆器，试试这个。如果你稍微年轻一点，并且渴望矢量图形街机版的月球着陆器，[拿起你的示波器](https://hackaday.com/2015/12/29/32c3-vector-video-games/)。
