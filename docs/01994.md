# 微波零件变得又快又讨厌雅各布的天梯

> 原文：<https://hackaday.com/2019/02/02/microwave-parts-become-quick-and-nasty-jacobs-ladder/>

雅各布之梯是一种电气设备，以圣经中的“通往天堂的梯子”命名，由一对垂直定向的导体组成，它们垂直分开。这些导体被高压充电，产生了我们从过去的科幻电影中了解和喜爱的反复攀爬的电弧。

[看，妈妈，没有电脑]在一次寻找电子垃圾的寻宝游戏中，他在一个垃圾车里发现了一个微波炉，正等着被黑客攻击。在考虑了几个想法后，[决定让微波炉捐出它的高压变压器来创造一个雅各布之梯](https://www.youtube.com/watch?v=8MRddr8gUwY)。变压器首先用螺栓固定在一块木头上，短路时会在工作台上产生一些火花。然后，输出连接到一对铜管上，创造出经典的效果。

不幸的是，该设备不能自启动，需要电极暂时短路来产生初始电弧。我们怀疑，增加电压可能会有所帮助，无论是与另一个变压器串联或与电压倍增器。

不言而喻，高压项目确实会给生命和肢体带来不可忽视的风险。如果你仍然渴望危险，看看这台自制的 x 光机。休息后的视频。
