# 新的 Pi 零增益天线再次未获批准

> 原文:[https://hack aday . com/2021/12/13/new-pi-zero-gains-unapproved-antennas-yet-again/](https://hackaday.com/2021/12/13/new-pi-zero-gains-unapproved-antennas-yet-again/)

我们才刚刚开始挖掘全新 Pi Zero 2 的潜力。终于收到了他的板，[Brian Dorey] [向我们展示了如何增强你的 Pi 的 WiFi](https://www.briandorey.com/post/raspberry-pi-zero-2-w-external-antenna-mod) ，黑客的方式。在板载 WiFi 天线上可以找到一个 u.FL 的足迹，你只知道有人不得不添加一个外部天线。这就是[Brian]的用武之地，下面嵌入了丰富的照片和[视频教程](https://www.youtube.com/watch?v=IZqaKZLIqFg)，可以让你立刻修改自己的 Zero。他的测量显示，在他以前只能看到四个网络的地方，他看到了十四个网络，并且报告的 RSSI 水平提高了 5 dB -10 dB，这对于获得更远或更稳定的连接来说是很大的。

随着旧笔记本电脑成为一个不错的 WiFi 天线来源，你只需要获得一个 u.FL 连接器，并在使用它之前练习焊接一下！这样一个项目最难的部分往往是不小心把任何焊料放在 u.FL 连接器的金属罐上——而[Brian] *多半*成功做到了这一点！他展示了如何断开外部天线以避免信号反射之类的情况，当然，在没有连接天线的情况下，您永远不要开启 Pi Zero，以免您的发射机因硬件定义的阻抗期望不匹配而陷入致命的混乱。Pi Zero 不是您可以添加连接器的唯一地方，可以说，这是您作为一名黑客的职责——以添加功能的方式修改您使用的东西。别忘了分享一下你是怎么做到的！

如果你想把你的新 Pi Zero 放在全金属外壳里，这个技巧会很有帮助。对 Raspberry Pi 天线的内部工作原理感到好奇？我们之前已经讨论过了！如果你想看一些以前的覆盆子 Pi 修改版，这里有一个[给 Pi 3](https://hackaday.com/2016/03/18/hacking-the-raspberry-pi-wifi-antenna-for-more-db/) ，这里有一个[给最初的 Zero W](https://hackaday.com/2017/03/07/adding-an-external-antenna-to-the-raspberry-pi-zero-w/)——也是来自【Brian】！
