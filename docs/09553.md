# 用 PCB 进行 PCB 回流

> 原文:[https://hackaday.com/2021/03/21/pcb-reflow-with-a-pcb/](https://hackaday.com/2021/03/21/pcb-reflow-with-a-pcb/)

我们想知道当[Carl Bugeja]想到用 PCB 制造 PCB 回流加热器时，他是否在看 3D 打印机的加热床。他尝试了一个快速测试来加热一个标准 PCB，并使其自回流。这很有效，尽管它显然不适用于所有的主板。但这证明了他能做到。

为了提高加热性能，他铺设了一个金属芯 PCB，以及一些定制的控制电子设备。电路板的电阻不太符合计算，但幸运的是，它太高了，因此分流线能够降低总电阻。需要考虑的一件重要事情是，加热器板需要使用更高温度的焊料，这样它就不会使自己的元件脱焊

我们很高兴看到[Carl]使用金属芯板，因为标准 PCB 材料在超过 130 摄氏度的高温下会变得不稳定。即便如此，最好还是检查一下你打算用这种方式使用的电路板，确保它们符合所要求的温度。

我们见过很多回流热源。[卤素灯](https://hackaday.com/2017/07/13/a-bright-idea-for-reflow-soldering/)浮现脑海。或者，突袭玩具储藏室，找到一个[简易烤箱](https://hackaday.com/2020/12/13/can-you-use-an-easy-bake-oven-for-reflow-soldering/)。
