# DIY 高流量 3D 打印喷头

> 原文：<https://hackaday.com/2021/11/23/diy-high-flow-3d-printing-nozzle/>

有时候，当有人意识到常识性的方法不是最佳方法时，进步就会发生。拿无线电来说。无线电的成功需要更大的天线和更大的功率，对吗？但手机的存在是因为有人意识到，如果你使用更少的功率和更小的天线来限制每个基站的范围，就可以在一个频率上容纳更多的人。随着 FDM 3D 打印的出现，更小的喷嘴风靡一时，因为它们提供了更精细的细节。然而，如今如果你想要精细的细节，你应该使用树脂打印机，更大的喷嘴可以提供更快的打印时间和更强的零件。火山旅馆开创了这一趋势，但现在有其他选择。CNC 厨房的[Stefan]决定制作自己的高流量喷嘴,他声称这比其他选择更好。

不要被 DIY 的部分冲昏了头脑。正如你在下面的视频中看到的，他从一个标准的喷嘴开始，所以这真的是一个喷嘴转换或黑客。高流量的问题不在于喷嘴上的孔。它融化塑料的速度足够快。塑料通过喷嘴的速度越快，熔化的时间就越少。

Bondtech 有一个高流量喷嘴，可以将灯丝分成碎片，以便更好地加热灯丝。显然，他们从一个专利持有者那里获得了这项技术的许可。[Stefan]受到启发，简单地在标准喷嘴上钻了一个孔，然后插入了一根铜线。我们不认为金属丝实际上是在分割灯丝流，但它确实向内部传递了更多的热量。或者热丝经过时确实会把灯丝分开，但至少有些丝没有完全穿过中心。不管怎样，他不会卖设备的，所以叫你的律师走吧。

魔鬼当然在细节中。[Stephan]展示了他如何使用高温焊料焊接电线，将它们切平，并使用 M6 模具给喷嘴穿螺纹。看起来不难，但确实需要一些细心。幸运的是，标准喷嘴非常便宜，所以如果尝试几次，它不会打破你的存钱罐。

像往常一样，[斯特凡]科学地测试一切。你可以看到普通喷嘴和较长的火山喷嘴以及 Bondtech 喷嘴之间的区别。只有一根电线的自制喷嘴能够接近火山喷嘴的性能。更多的钻孔和另一根电线提供了更好的性能。

这项技术受专利保护，所以我们不希望看到大量合法的克隆出现。但是，如果你能熟练使用微型钻头和焊料，你或许可以把它们做成自己用的，至少在世界上的许多地方。然而，因为表演和火山差不多，也许现在就坚持下去。

如果你对高流量感兴趣，你可能会想要建立强大的部分。设计也与此有很大关系。这对于——嗯——[搭桥](https://hackaday.com/2021/07/24/walk-the-first-3d-printed-bridge-and-be-counted/)尤为重要。
