> 原文:[https://hack aday . com/2020/07/06/Linux-in-the-machine-shop-hack-chat/](https://hackaday.com/2020/07/06/linux-in-the-machine-shop-hack-chat/)

加入我们太平洋时间 7 月 8 日星期三中午的 [Linux 在机械商店与](https://hackaday.io/event/173261-linux-in-the-machine-shop-hack-chat)[安迪·皮尤](https://hackaday.io/hacker/443547-andy-pugh)的黑客聊天！

从上个世纪中期数字控制开始进入机械工厂开始，直到最近，用除了熟练的人手之外的东西来控制机床的能力是显而易见的。不幸的是，这样做的设备是昂贵的，因此数控技术仍然牢牢地留在大商店，在那里可以实现可观的投资回报。

几十年过去了，一切使计算机化的数控成为可能的东西都很便宜，很容易买到。伺服系统、步进器、驱动器和运动控制组件可以一起插入 CNC 机器，这些机器可以以令人难以置信的精度和可重复性将工具移动到空间中的固定点。但是没有 CNC 软件，这些都没有意义。

进入 [Linux CNC](https://linuxcnc.org/) ，免费开源的 CNC 包。Linux CNC 支持实时操作、一步安装和由志愿开发人员团队提供的大量功能，并得到活跃社区的支持，使 CNC 机器的世界民主化。

安迪·皮尤经常为 Linux CNC 代码库做出贡献，也是论坛的版主。他对 Linux CNC 特别了解，对机械车间的 Linux 也略知一二。他会在 Hack Chat 上与我们分享他在 Linux CNC 项目上的经验，告诉我们 Linux 是如何革新机械车间的，也许还会分享一些 CAD、CAM 和使用 Linux 制作芯片的故事。
