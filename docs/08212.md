# DIY 正则表达式

> 原文：<https://hackaday.com/2020/10/29/diy-regular-expressions/>

在《星球大战》宇宙中，不是每个人都使用光剑，使用光剑的人必须自己动手打造。关于这一策略，我们有话要说。制造一辆汽车或一台收音机是了解这些东西如何工作的好方法。这就是[低级 JavaScript]对正则表达式指出的。当然，很多人认为他们很可怕。那么为什么不编写自己的正则表达式解析器和引擎呢？明白了这一点，你可能再也不会害怕另一个正则表达式了。

当然，我们大多数人很可能不会自己动手，但是你还是可以在下面的视频中观看这个过程。代码出奇的短，但是不要期望你在 Python 甚至 Perl 中找到的所有的铃铛和哨子。

在技术熟练的人手中，正则表达式非常强大，提供了一种分割文本数据的快速方法。像许多强大的想法一样，有限状态机的基本概念非常简单。应用于实际问题变得困难了。

如果你想要一本不需要你自己编写工具的正则表达式入门书，我们有一些帖子可以帮到你。如果你只是想练习一下，试试[一个纵横字谜](https://hackaday.com/2016/01/31/crosswords-help-you-learn-regular-expressions/)。
