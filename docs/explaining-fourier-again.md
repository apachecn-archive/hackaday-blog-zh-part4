# 再次解释傅立叶

> 原文：<https://hackaday.com/2018/12/21/explaining-fourier-again/>

生活在互联网时代的一个好处是，创建令人惊叹的模拟和动画在今天相对简单。[SmarterEveryDay]最近做了一个视频来展示这一点，讨论了一篇[博客文章](http://bilimneguzellan.net/fuyye-serisi/)(用土耳其语)来展示正弦波如何相加在一起产生任意波形。你可以在下面看到英文视频。

我们以前见过类似的事情，但是如果你没有见过，你真的可以看到运动圆上的一个点是如何描述正弦波的。通过添加这些波，任何事情都可以完成。

原始帖子的作者[Doga]是佐治亚理工学院的学生，[SmarterEveryDay]实际上拜访了他以获得现场演示。使用的软件是 Mathematica。

然而，也许最有趣的事情是[Doga]接受了一项挑战，用傅立叶级数绘制了一个复杂的标志。他只是将 X 和 Y 坐标数字化，然后分解结果函数。这让我们想起了如何在示波器上画出圣诞树。

这不是我们看到的第一个关于主题的[视频。我们已经看到用](https://hackaday.com/2018/02/10/all-the-stuff-you-wished-you-knew-about-fourier-transforms-but-were-afraid-to-ask/) [MATLAB](https://hackaday.com/2015/09/17/visualizing-the-fourier-transform/) 做了同样的事情。但是傅立叶变换对于电子学和许多其他学科来说是如此的基础，以至于它值得我们去学习和回顾，另外我们也喜欢那些很酷的动画。如果你想亲自尝试一下，这里有一篇今年早些时候由[Alex Miller]撰写的英文[博客文章](https://alex.miller.im/posts/fourier-series-spinning-circles-visualization/)。在文章的底部是一个在线工作簿的链接，你可以自己尝试一下。
