# 钢筋混凝土船去无刷的速度和可靠性

> 原文：<https://hackaday.com/2018/08/24/rc-boat-goes-brushless-for-speed-reliability/>

遥控船非常有趣，并且有各种各样的形式。有无动力的帆船，有在内燃机动力下尖叫的快艇，当然，也有靠电力驱动的。特别是过去 20 年的无刷电机革命已经证明能够创造一些令人兴奋的遥控船艇，并且 [[Matt K]决定他想上船。](http://mhkaufman.blogspot.com/2018/08/kyosho-jetstream-1000-nitro-to.html)

[Matt]拥有一架阔嘴猴睡衣 Jetstream 1000 已经好几年了，但是发现硝基发动机是不稳定的，并且对于在湖边的 high-jinx 来说不是最有趣的。带有机械速度控制的老派有刷电机设置同样无法激励。然而，在体验了遥控飞机无刷的力量后，[马特]知道他必须做什么。

使用在线计算器，[马特]确定他早期的硝基动力装置大约输出 900 瓦。当谈到去无刷，他决定规格一 Turnigy 动力装置有两倍多的权力，随着必要的速度控制器。有一些工作要做，以整合新的电机与原来的螺旋桨驱动轴和水冷系统，但最终[马特]结束了一个更快的船，这是一个很少麻烦的设置和运行。

也许，虽然，你的遥控船需要大脑，而不是肌肉？[也许是时候考虑自治了……](https://hackaday.com/2015/06/01/simple-autonomy-with-an-rc-boat/)
