# 把 DS 塞进游戏机里

> 原文：<https://hackaday.com/2021/11/16/cramming-a-ds-inside-a-gameboy/>

许多假日食谱和控制台黑客有一个共同的主题:把一个东西塞进另一个东西里。无论是 turducken 还是 Gameboy 里面的[任天堂 DS，](https://www.youtube.com/watch?v=TFpNyPohHMY)结果总是出人意料。这个 mod 的底盘是一个不起眼的 Gameboy 颜色，顶部有一个 Gameboy SP 屏幕作为辅助显示器。不幸的是，这个 mod 失去了触摸屏功能，限制了你可以玩的一些游戏。

【TheRetroFuture】从【GameboyCustom】处收到定制掌机，在运输过程中有些损坏。原来的螺丝安装不得不被删除，案件粘在一起，以适应 DS 主板。所以对[TheRetroFuture]来说，进去开始排除故障需要刀片和耐心。测试各种点和交换组件更接近根本问题。修复的结果是一些电线在运输过程中松了。最后，在重新拔插显示器连接并小心焊接后，它启动并开始玩游戏。

总的来说，在这个小小的掌上电脑上同时运行马里奥赛车给人留下了深刻的印象。但你可能会问，为什么？为什么要把一个掌上电脑塞进另一个掌上电脑？有时是为了获得新的功能[，比如 PSP 主体](https://hackaday.com/2018/10/21/stock-looking-psp-hides-a-raspberry-pi-zero/)中的树莓 Pi。有时候，只是因为我们可以。休息后的视频。
