# 观看沙雕弹弓制作，从开始到结束

> 原文:[https://hack aday . com/2019/12/18/watch-a-sand-cast-slings hot-made-from-start-to-finish/](https://hackaday.com/2019/12/18/watch-a-sand-cast-slingshot-made-from-start-to-finish/)

[![](../Images/be3392457f9d8b5d092ffada4762216d.png)](https://hackaday.com/wp-content/uploads/2019/12/sand-casted-slingshot.jpg) 砂型铸造金属零件是一项由来已久的技术，但从头到尾观察这一过程还是很有教育意义的。这正是[Frederico]用[他自己设计的](https://www.manmademayhem.com/?p=3665)砂铸弹弓向我们展示的，他说这是第一次尝试，这并不坏！

首先，[Frederico]制作一个由两部分组成的[湿砂](https://en.wikipedia.org/wiki/Sand_casting#Green_sand)弹弓主体模具。湿砂是一种沙子和粘土的混合物，从湿的或“未加工”的意义上来说，它只是*绿色的，没有进一步加工。模具做好之后，就该在丙烷动力炉中熔化铝了，然后将熔化的铝倒入模具中。*

冷却后，[弗雷德里科]打破了沙子，揭示了粗略的铸造对象。有后期处理要做的形式，浇口切割和一些接缝周围的闪光消除，但总的来说，它看起来很好。你可以在视频中观看整个过程，嵌入在下面。
