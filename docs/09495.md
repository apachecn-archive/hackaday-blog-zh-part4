# scratch-build 滚动工具柜是一个金属加工技能建设者

> 原文：<https://hackaday.com/2021/03/12/scratch-built-rolling-tool-cabinet-is-a-metalworking-skill-builder/>

是的，市面上可以买到各种形状、尺寸、抽屉排列、甚至颜色的工具柜，你可能不应该自己动手做。但是正如“制造东西”的鲍勃指出的，这有什么意思呢？

尽管如此，我们可以想出很多理由来从头开始制作自己的滚动工具柜。除了练习金属加工技能和充分利用工具的明显好处之外，自己动手轧制意味着你可以得到你想要的东西。几乎我们购买的每一个工具柜都在某种程度上有点次优——对可用空间来说太宽了，或者抽屉太浅了，放不下那个古怪的工具。能够创建自己的橱柜意味着你可以准确地达到规格，正如[Bob]所示，如果你有合适的工具，这甚至不是很难。

构建从[Bob]的数控等离子切割机开始，[这是一台车间制造的机器](https://hackaday.com/2018/06/20/a-cnc-plasma-cutter-table-from-the-shop-floor-up/)，多年来已经升级过几次。等离子切割机可以快速切割钢板制成的抽屉毛坯，并配有凹槽，便于将钢板制成抽屉。机柜的框架是钢管，焊接起来，里面填充更多的钢板。侧面增加了全伸展滚珠轴承滑轨，用于支撑抽屉；我们不得不承认，焊接看起来像镀锌钢的东西会让我们紧张，但我们认为[鲍勃]对潜在的有毒气体采取了预防措施。

下面的视频展示了整个构建过程，并展示了非常清晰的最终产品。这也让我们很想建造自己的等离子切割机。
