# 计算机科学家的量子计算

> 原文：<https://hackaday.com/2018/07/13/quantum-computing-for-computer-scientists/>

量子计算即将到来，所以很多人都在试图阐明我们为什么想要它，以及它是如何工作的。大多数解释要么是关于自旋和纠缠的硬核物理学，要么是非常轻松和笨拙的，这有助于获得一点理解，但对应用技术没有用处。微软研究院有一个视频试图击中中间的那个点，这是给目前使用传统电脑的人的实用信息。你可以看下面的视频。

该视频从大多数视频中关于向量表示和运算的基础知识开始。你必须经历大约 17 分钟这样的事情，直到你进入量子位。如果你对数学不感兴趣，听听[Andrew]在数学之后给出的“索引数组”解释，你会更开心。

将德国甲骨文公司作为量子计算优越的例子让我们紧张。前提是你可以在一次操作中识别一个黑盒，而不是在经典计算机中识别两个黑盒。问题是，要做到这一点，您需要修改黑盒以获取额外的位。好吧，如果我可以修改黑盒，用一台经典计算机多取一位(以不同的方式)，我也可以在一次操作中识别出这个函数。然而，它很好地解释了一个基本概念——它只是经不起推敲，它证明了量子的优势。

在我们看来，展示它如何反映经典计算中的并行性可能更好。例如，如果我可以修改我的黑盒来并行地对两位进行相同的操作，我会得到相同的结果。量子修改——当然——更简单，但它仍然需要您添加额外的位并修改黑盒。

视频以一些使用微软工具的现场演示结束。如果你看了这个视频，并想在你的浏览器上动手操作——甚至是真机——你可能会喜欢[我们的系列教程](https://hackaday.com/2018/01/24/quantum-weirdness-in-your-browser/)。如果你只是试图在更高的层面上解释或理解量子计算，[IBM 的视频](https://hackaday.com/2018/06/26/wired-and-ibm-explain-quantum-computing-to-students-from-grade-school-to-grad-school/)要轻松得多。
