# 模拟人生风格的铅锤广播你的心情

> 原文：<https://hackaday.com/2020/11/28/sims-style-plumb-bob-broadcasts-your-mood/>

虽然《模拟人生》中有很多我们希望是真实的物体，但我们可能更希望每个人都有一个情绪指示器一直在他们头上盘旋。例如，这将使在家工作变得更加顺利。[【8 bitsandabyte】制作了这个蓝牙控制的铅锤作为他们模拟人生万圣节服装的一部分](http://www.instructables.com/A-Bluetooth-Controlled-Sims-Plumbob/)，但是我们认为随着这个疫情戴上它，它有真正的日常价值，要么作为一个心情戒指，要么作为一个便携式忙/闲指示器。

硬件非常简单——一个 Adafruit Feather nRF52 Bluefruit 控制一对 NeoPixel 环，每个环对应半透明 3D 打印铅锤的一半。电力来自一个 500 毫安的电池，所有的电子设备都装在一顶漂亮的帽子里。休息之后，请观看构建视频。

使用颜色传达信息的方式不止一种。[这款七段式温度显示器采用热变色薄膜](https://hackaday.com/2020/03/20/thermochromic-display-tells-you-the-temperature-despite-your-current-mood/)来显示温度。
