# 最安全的过山车模型

> 原文：<https://hackaday.com/2021/12/02/the-safest-model-roller-coaster/>

[Jared Holladay]是辛辛那提大学计算机工程专业的学生，也是一名终生的过山车迷。很多人将过山车视为一个令人兴奋的物理学例子，就像势能与动能或惯性相比，这是正确的。[[Jared]看着它们，对控制器感到好奇](https://www.youtube.com/watch?v=iartGfOZwMk)。视频也在下面，有一个更详细的[特征长度解释](https://www.youtube.com/watch?v=TbvSVWfpUbc)。一些黑客读者和作家可以识别组件，所以我们认为他的过山车模型属于这里。

像这个领域的许多人一样，他建造了 K'nex 模型来处理建筑。他巡回了 STEM 秀的曲目，无疑赢得了孩子们、成年人和物理老师的喝彩，但由于他可以对着程序说话，他是一个三重威胁。现在，他已经从玩具结构塑料中成长起来，进入了具有精密公差的 3D 打印部件。

他的最新底座是挤压铝，就像你想要的刚性 CNC 或打印机。除了工业级的 surface，罗克韦尔自动化还给他送去了安全可编程逻辑控制器、PLC 和触摸屏 HMI。业内人士告诉我们，这些价格远远超出了普通爱好者的承受范围。但不要害怕；你的 Arduino 克隆体在你得到第一笔资助之前足够了。

除了真实性之外，所有加固硬件的要点是以与业内相同的方式实现安全功能。冗余 PLC 连接到感应接近传感器，以检查列车速度和位置。其他移动部件，如摩擦制动器，有传感器报告是否有堵塞。毕竟，如果你不能阻止一辆载满人的火车，那是没有用的。有成百上千的事情可能会出错。问问[Jared]就知道了，因为他为所有这些设备编写了屏幕指示器，并对它们进行了分类，让操作员知道他们是否可以保持游乐设备运行，或者他们是否需要呼叫维护人员。

并非所有的自制杯垫都是比例模型，一些传统杯垫的 T2 比看上去的要多。
