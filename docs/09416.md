# Retrotechtacular:双倍免费

> 原文:[https://hackaday . com/2021/05/21/retro tech tacular-double-for-white/](https://hackaday.com/2021/05/21/retrotechtacular-double-for-nothing/)

如果你在 2021 年进入虚拟主机市场，并与云计算提供商签约，你很快就会看到不同的资源是如何定价的。存储和带宽便宜，CPU 时间贵。这反映了现代计算机的状态，其中典型的磁盘驱动器现在可以容纳 1tb 或更多，并且逐年增加，而新的处理器正在成为一个瓶颈，其性能没有像制造商希望的那样比几年前的型号增加得多。

## 一点软件换来两倍的硬件？

但在 20 世纪 90 年代初，情况就不同了。与预期运行的 DOS 或 Windows 3.1 软件相比，486 或早期的奔腾处理器相当强大，而限制性能的是它所附带的内存和磁盘空间，而且成本高昂。大约在 1995 年的一段时间里，某个地方的一家芯片工厂发生了一场假定的火灾，导致 RAM 价格上涨到每兆数百美元，短暂地导致了 RAM 抢劫的流行，犯罪分子会闯入办公室，只从计算机中取走模拟人生。

这个问题的解决方案可能出人意料地来自软件业。Disk Doubler 是一个 DOS 驱动程序，承诺提供更多的磁盘空间，通过压缩磁盘以容纳更多的数据来实现这一看似不可能的壮举。在当时，用处理器能力交换磁盘空间是一种合理的交易，因此它变得非常流行，最终微软将他们自己的磁盘压缩技术整合到了 DOS 中。在某些情况下，它甚至可以提高装有慢速磁盘驱动器的计算机的速度，正如我在学生时代用 286 装 MFM 驱动器时发现的那样。

## 不劳而获，也许好得不像真的。

如果压缩可以增加磁盘空间，那么它不能为 RAM 做同样的事情吗？业界用一系列 RAM 倍增器产品再次拯救了我们，首先将磁盘倍增器技术应用于磁盘虚拟内存，然后对内存本身的内容进行同样的处理。第一种方法以降低系统速度为代价，而第二种方法效果不太好。事实上，这只不过是一个骗局，软件产品承诺很多，但在幕后却什么也没有提供。

休息下面的视频看一下 [Syncronys SoftRAM](https://en.wikipedia.org/wiki/SoftRAM) ，带我们看一下它的操作。有针对 WIndows 3.1 的内存优化器，它通过将尽可能多的项目从 DOS PC 中的 640k 传统内存移动到 386 和更高处理器支持的扩展内存来工作。这些在允许更多进程运行方面工作得非常好，但是尽管它们增加了可用的传统内存，但它们并没有声称提供了任何额外的内存。

相比之下，SoftRAM 的工作方式似乎与磁盘倍增器完全相同，并且有一个令人印象深刻的仪表盘，上面有一个仪表，显示使用它获得了多少额外的内存。不幸的是，这是该软件唯一能做的事情，调查人员很快就识破了这个骗局。诉讼和联邦贸易委员会的调查接踵而至，该公司最终在 1998 年倒闭，许多罚款和用户回扣都没有支付。

当时，大多数非公司个人电脑用户正坐在他们的第一台电脑前，通常是从更习惯于销售电器的大型电子产品零售商那里购买的。它们被当作 CD-ROM 多媒体光盘等时代奇迹出售，它们的主人还没有学会不信任来历不明的软件产品。为 SoftRAM 创造的术语是“安慰剂软件”，这种软件只是让用户放心，它正在做一些事情，让他们想象他们的机器更快。鉴于该公司销售了 70 万份软件，而这些用户中的许多人甚至从未怀疑过它有什么问题，显然他们有所发现。
