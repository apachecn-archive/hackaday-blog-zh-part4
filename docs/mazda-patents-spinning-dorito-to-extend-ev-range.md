# 马自达专利纺纱多利托扩大电动车范围

> 原文：<https://hackaday.com/2022/12/16/mazda-patents-spinning-dorito-to-extend-ev-range/>

好吧，所以汪克尔发动机并不真的使用多利托作为它的气缸，但它看起来确实像一个。该公司宣布将为 MX-30 电动“跨界”车提供增程转子发动机，但[CarBuzz] [挖掘专利文件](https://carbuzz.com/news/mazdas-new-rotary-engine-variable-valve-timing-is-weirder-than-the-wankel-itself)发现它有一些有趣的变化。

MX-30 是一款配备相对较小的 35.5 千瓦时电池的电动汽车。像混合动力汽车一样，这种汽车包括一个可以给电池充电的小型内燃机。然而，它在任何时候都不会直接驱动车轮。汪克尔有几项改进，包括第二个端口，当发动机必须产生高功率时，允许更多的空气进入燃烧室。但是有一个问题…

二级端口在你用力的时候是很棒的，但是在低速的时候，会产生低效。为了解决这个问题，马自达安装了一个阀门，当没有必要打开第二个端口时，它会关闭第二个端口。但这还不是奇怪的地方。奇怪的是，发动机也有自己的电动辅助电机，以主电池为动力。没错。您正在充电的蓄电池提供一些能量来操作电动辅助电机，以帮助给蓄电池充电的发动机。如果这让你的头像汪克尔的转子一样旋转，你并不孤单。

辅助马达可以在进气冲程中辅助或延迟输出轴。这可以优化燃烧室的进气。当然，这将导致发动机输出的奇怪运动，但既然它不驱动汽车，谁在乎呢？如果输出不平稳，电池不会介意。
