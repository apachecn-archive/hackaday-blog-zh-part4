# 氧化还原键盘

> 原文：<https://hackaday.com/2018/09/27/the-redox-keyboard/>

像 Colemak 和 Dvorak 这样的替代键盘布局并不新鲜；它们允许更容易地使用更经常使用的键，以减少打字过程中对手的压力。在符合人体工程学的 Ergodox 键盘受欢迎的基础上，[Mattia Dal Ben]开发了氧化还原键盘，即简化的 Ergodox，以制造更小、更符合人体工程学的键盘。

与 Ergodox 一样，Redox 采用柱状布局，按键排列成列，每列基于相应的手指偏移。氧化还原脱离 Ergodox 设计的地方是拇指键。[Mattia]的小指开始疼痛，所以他希望拇指布局能够减少小指的一些额外工作。thumb 群集比它的祖先小，并且包括一个附加的旋转 thumb 键。

氧化还原比麦角洐生有一些很大的改进，以帮助治疗与打字最相关的劳损类型，有望导致与最有用的外周细胞更好的相互作用。

机械键盘社区不断推出伟大的新设计和不同的 DIY 键盘，我们在网站上展示了其中的许多。在你检查完[Mattia]创建的图片和示意图后，看看这个 [3D 打印机械键盘](https://hackaday.com/2015/03/29/3d-printed-mechanical-keyboard/)，键盘设计和制造的细节[在 2017 年的 Hackaday 超级大会上展示](https://hackaday.com/2017/11/22/a-passion-for-the-best-is-in-mechanical-keyboards/)。
