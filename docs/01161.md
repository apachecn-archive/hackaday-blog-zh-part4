# 谁要赌你的 50 元？

> 原文：<https://hackaday.com/2018/11/03/whos-going-on-your-fifty/>

通过观察一个国家的钞票，你可以了解这个国家的很多情况，包括它的历史和政治。谁在其中扮演角色，或者在精心策划的国际妥协(如欧元)中，谁*不*在其中扮演角色。多年来，英国居民的现金上出现了一连串的历史名人，当宣布新版钞票时，你可以肯定，选择名人来装饰钞票将成为头版新闻。今天，我们有一种新的钞票在路上，这一次的选择完全在 Hackaday 的兴趣范围内，因为[公众被敦促提名一位科学家获得这一荣誉](https://www.bankofengland.co.uk/banknotes/50-pound-note-nominations)。有问题的纸币是 50，没有人使用，很多商店甚至不会接受，但它仍然是一个重要的选择，将取代目前版本的现任者，蒸汽机先驱[马修·博尔顿和詹姆斯·瓦特](https://en.wikipedia.org/wiki/Boulton_and_Watt)。

所以，给你一张 50 元的空白支票，你会填谁？候选人必须是英国人，不是虚构的，也不再活着。名单中的名字包括[艾达、女伯爵洛夫莱斯](https://en.wikipedia.org/wiki/Ada_Lovelace)、[斯蒂芬·霍金](https://en.wikipedia.org/wiki/Stephen_Hawking)和[艾伦·图灵](https://en.wikipedia.org/wiki/Alan_Turing)，尽管有如此广泛的领域可供选择，肯定会有更多的领先者。例如，你可能希望考虑罗莎琳德·富兰克林，但是你可以忘记艾萨克·牛顿、查尔斯·达尔文或者迈克尔·法拉第，因为他们都已经出现在英国钞票上了。

Hackaday 在这种努力中不偏袒任何一方，但这仍然是一个支持你最鼓舞人心的人物的机会。作为你的抄写员，这是洛夫莱斯和图灵之间的艰难抉择，尽管图灵可能以微弱优势胜出。你想在下一张 50 英镑钞票上看到谁？银行制作了一个简短的宣传视频，我们把它放在休息时间下面。
