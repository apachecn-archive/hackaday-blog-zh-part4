# 如果让学习编码成为儿戏

> 原文:[https://hack aday . com/2019/11/11/the-ifs-make-learning-to-code-childs-play/](https://hackaday.com/2019/11/11/the-ifs-make-learning-to-code-childs-play/)

任何做过一点编程的人都知道“你好，世界！”程序。这是一个原型程序，你进入它是为了感受一种新的语言或一种新的架构；如果你能让一台机器印出“你好，世界！”还给你，剩下的只是细节。但是教孩子编程呢？一个人如何让蹒跚学步的孩子以逻辑的、程序化的方式思考？更具体地说，什么是“你好，世界！”识字前的节目是什么样的？

这些问题导致了[Makeroni Labs]的[if](https://hackaday.io/project/167281-the-ifs-coding-its-childs-play)。Ifs 是教育玩具，可以教三岁的孩子基本的编码知识。每个 If 都是一个彩色的塑料立方体，有一个卡通脸和一个反映积木功能的“个性”——一些积木有致动器，一些有传感器。通过在顶部放置代表条件和动作的磁性标签来对块进行编程。一个孩子可能会选择编程一个积木来检测它何时被摇动，或者灯何时亮起，然后通过播放声音或振动来做出响应。这些程序块也可以相互通信，因此当一个程序块的条件满足时，另一个程序块就会发生某种情况。

“如果”看起来很有趣，对于编码人员和非编码人员来说，它们都是逻辑思维技能的良好开端。我们不是唯一一个认为这是一个非常热切的项目的人——今年 Hackaday 奖的评委选择了 Ifs 作为 20 个决赛入围者之一。它会赢吗？我们将在下周的[2019 年黑客日超级大会](https://hackaday.io/superconference/)上找到答案。如果你不能和我们一起在帕萨迪纳，一定要收听[的直播](https://www.youtube.com/hackaday?sub_confirmation=1)来观看公告。
