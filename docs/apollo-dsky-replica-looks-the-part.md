# 阿波罗 DSKY 的复制品看起来很像

> 原文：<https://hackaday.com/2020/09/23/apollo-dsky-replica-looks-the-part/>

很难说阿波罗 DSKY 号到底吸引了多少黑客的想象力。不管它是什么，阿波罗制导计算机的“显示和键盘”单元多年来激发了数十次拆卸、模拟和复制，取得了不同程度的成功。但是这个机械上忠实于 DSKY 的复制品在对细节的关注上确实让它与众不同。

作为[M. daSilva]的产品，这个 DSKY 复制品走了一条与我们见过的其他复制品有些不同的道路。通过研究尽可能多的原始文件，他能够非常精确地复制 DSKY 的物理尺寸和形状——当研究原始纸质印刷品的副本时，这是一个不小的成就。尽管如此，捕捉到的细节，如加强原始压铸零件的角撑板和加强件，确实使这辆 DSKY 看起来很像。它也很实用，这要归功于运行 VirtualAGC 的 Raspberry Pi，以及取代原始电致发光显示器的 Nextion 4.3 英寸 LCD 显示器。我们惊讶地得知，DSKY 有一个端口氮气吹扫的情况；看看下面的视频之旅吧。

当然，仅仅因为[M. daSilva]选择专注于尺寸精度，并不排除未来更忠实的电子产品。也许他可以和[【本·克拉斯诺】](https://hackaday.com/2019/05/19/ben-krasnow-makes-a-dsky/)或者[【弗兰布兰奇】](https://hackaday.com/2017/05/29/re-creating-the-apollo-dskys-display/)合作，真正让这成为一个展示品。
