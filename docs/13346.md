# 氮气烙铁综述

> 原文:[https://hack aday . com/2022/04/14/a-氮-焊-铁-评论/](https://hackaday.com/2022/04/14/a-nitrogen-soldering-iron-review/)

如果你曾经焊接过，你会知道一些焊工出于不同的原因在工件上吹保护气体。例如，您经常使用气体来置换该区域的氧气，以避免氧化。你也可以用氮气保护来焊接。这允许更高的温度和减少焊料中所需的焊剂。波峰焊经常使用氮气，JBC 提供了一种可以使用氮气保护气体的烙铁。[SDG Electronics]在下面的视频中测试了那块铁。

正如你所料，这不是一个 50 美元的烙铁。这种铁的价格不到 1000 美元，这还不包括电源和氮源。将压缩空气转化为氮气的氮气发生器特别昂贵，因此[SDG]只使用了一瓶气体。

抛开成本不谈，值得吗？视频显示了在有和没有氮气的情况下不同焊剂水平的焊接工作。我们看不出太大的区别，尽管[史蒂夫]提到在氮气下焊接似乎更容易一些。更高的焊剂也表现得更好。在显微镜下检查显示一些差异，但没有什么会迫使你花在气体铁。没有助焊剂，焊料不想变湿。添加少量的助焊剂导致了与氮气的良好连接，虽然普通的烙铁没有那么好，但这并不坏，也不清楚为什么这在实际应用中会是一个很大的优势。

在一些特殊的焊接工作中使用氮气有一些优势，但它似乎并不令人信服。使用熔剂有所帮助，但你必须购买大量的熔剂才能在这种铁的投资上达到收支平衡。
