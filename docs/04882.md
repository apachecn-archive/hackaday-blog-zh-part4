# 设计一张 PCB 名片并不像看上去那么简单

> 原文：<https://hackaday.com/2019/11/19/theres-more-to-designing-a-pcb-business-card-than-meets-the-eye/>

一个从计算机出现之前就存在的奇怪习俗是名片。如果你在一个贸易活动中走过大厅，你会遇到一堆这样的东西，每一个都有你遇到的某个人的联系方式，每一个都在社交媒体和在线联系的世界中注定会在你办公桌的某个积满灰尘的角落里枯萎。在 21 世纪，当一部手机收获的电子接触有了粘性，一张植根于过去时代的卡片怎么可能有竞争力？

这是一个[Anthony Kouttron]在设计他的完全现代化的名片时解决的问题，一路上，他向我们讲述了一个有趣的故事，如何使名片不仅具有联系信息的功能，还能传递电子联系。最终的卡有一系列的尺子和脚印作为电子设计师的辅助工具，还有一个 NFC 天线和芯片，当扫描时可以点亮 LED 并提供他的网站地址。有一些小的妥协，如 NFC 天线下的 PCB 垫，但正如他在下面的视频中解释的那样，它们不足以阻止它的工作。[如果你想做类似的事情，他已经把他的作品放在了 GitHub 库](https://github.com/anthonykouttron/pcb-business-card-qr-nfc)。

这些页面上有丰富的名片项目，但到目前为止，令人惊讶的是，几乎没有 NFC 设备。尽管如此，这并没有阻止某些人制作具有用户交互功能的 NFC 卡。
