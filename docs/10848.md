# 如果社会面临崩溃的危险，我们应该尽自己的一份力量

> 原文：<https://hackaday.com/2021/07/26/if-society-is-in-danger-of-collapse-heres-how-we-should-do-our-bit/>

如果你一直在关注新闻，你一定不会错过似乎发生在世界各个角落的一系列洪水、干旱和野火。紧接着一个有着自己极端天气事件的北半球冬季，现在不认真对待气候变化可能是鲁莽的。你可能也已经看到了关于 20 世纪 70 年代的一篇论文回归的新闻，在这篇论文中，麻省理工学院的水晶球观测者预测我们的文明将在 21 世纪中叶崩溃，还有[一篇基于自那以后收集的经验数据的评论](https://advisory.kpmg.us/articles/2021/limits-to-growth.html)得出结论，我们可能会正确地预测大约在 2040 年发生。

这是发人深省的东西，很容易成为许多好莱坞灾难电影的基础。但是坐在 2021 年的这里，面对极端天气事件和全球疫情，这肯定是值得思考的事情。虽然我们还没有骑着沼气动力的武器车穿越后世界末日的沙漠，但我们仍然有机会做一些事情来避免灾难，毫无疑问，在未来十年，大量的变化将减少我们的二氧化碳影响，并使我们的基础设施更有弹性，以抵御任何即将到来的危机。

我们的思绪回到了疫情之前的宁静时光，回到了 2019 年 8 月的丹麦 BornHack 黑客营。活动中的一个演讲来自[Igor Nicolic]，他是代尔夫特理工大学的一名学者，从事生态学和可持续性的研究。[他在书中审视了全球可持续发展的现状，并确定了硬件黑客社区在不确定的未来可能扮演的角色](https://www.youtube.com/watch?v=-cJ0uZObzJI)。这是一位该领域专家的精彩演讲，非常值得一看并记下他的观点，所以我们把它放在休息时间下面。
