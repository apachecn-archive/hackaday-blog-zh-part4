# 复古记忆黑客聊天

> 原文：<https://hackaday.com/2021/11/01/retro-memory-hack-chat/>

加入我们太平洋时间 11 月 3 日星期三中午与 Andy Geppert 的[复古记忆黑客聊天](https://hackaday.io/event/181742-retro-memory-hack-chat)！

随着现代存储芯片变得如此廉价和易于集成，人们很容易忘记这样一个事实:不久前，内存还是大多数计算机设计的限制因素。在硅存储器出现之前，工程师们不得不用各种怪异而奇妙的技术来提供几个宝贵的内存字节。诸如布满铁氧体磁心的错综复杂的金属网、奇怪的声学延迟线，甚至磁泡都曾被尝试过。他们工作得很好，至少足以让我们登上月球，但在硅存储器的进步面前，没有一个是可行的。

这并不意味着复古记忆技术不再有一席之地。一些爱好者，如安迪·格伯特，保持着复古记忆的火焰。他的 [Core 64 project](https://hackaday.io/project/166155-core-64-interactive-core-memory-badge) 将一个核心内存模块放在你的手掌中，甚至可以让你用磁铁直接“吸引”到内存中。安迪在取得这一成就的过程中学到了一些技巧，并希望我们所有人都能欣赏复古记忆技术的时代魅力。停下来聊聊你对记忆的回忆，或者只是了解一下存储一点点数据需要什么。
