# OPARP 远程呈现机器人

> 原文：<https://hackaday.com/2020/08/31/oparp-telepresence-robot/>

[埃里克·克努特森]被困在一堆机器人零件里，我们知道那条路上有什么。他的[开放式个人助理机器人平台](https://hackaday.io/project/171791-open-personal-assistant-robotic-platform)旨在帮助做家务，比如填充宠物食物碗，但目前，他正在一步一步地解决问题，然后再添加新功能。明智。

该版本开始时有一个狭窄的底座，一个动力不足的 RasPi 和一个安静的扬声器，但这些都被依次升级。现在，它是一个带轮子的个人助理。Alexa 是第一个竞争者，但 Mycroft 是焦点，因为它有更多的多功能性。起初，mobility 是一个带有 D-pad 的简陋网络服务器，但现在它利用了距离传感器和视觉，甚至可以通过语音命令跟随你。

顶部的屏幕让它看起来很有个性，但它将成为你想在机器人助手上看到的一切的显示器，比如天气、食谱或可以陪你四处走动的视频聊天。[Erik]想做一些东西来帮助那些可能需要帮助做家务的老人，并帮助像他一样被困在里面的人联系起来。

表情机器人早就吸引了我们的注意力，我们非常喜欢以隐私为中心的个人助理(T2)。
