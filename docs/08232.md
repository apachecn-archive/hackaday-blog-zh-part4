# 快速简易毫欧计

> 原文:[https://hack aday . com/2020/11/04/快速简单的毫欧计/](https://hackaday.com/2020/11/04/quick-and-simple-milliohmmeter/)

用户[mircemk]展示了他的[“毫欧计”](https://hackaday.io/project/175599-diy-arduino-miliohmmeter-for-measure-low-value-res)项目，你可以用一个 Arduino、一些来自你实验室的常用部件和一个雪茄盒来构建它。伙计们，没有比这更简单的了。虽然这是你不会用 NIST 可追溯性校准的东西，但它看起来像一个有趣而快速的项目，非常适合业余测量爱好者。它不仅易于构建，而且 Arduino 草图不到三十行代码。这是一个很好的学习项目，而且完成后你会得到一些对你的实验室有用的东西。

我们喜欢在项目的盒子上创造性地使用彩色胶带而不是油漆。如果这种风格适合你，[mircemk]在他的 Hackaday.io 页面上发布了其他几个类似的实验室仪器项目，包括一个[频率计](https://hackaday.io/project/174836-diy-simple-arduino-frequency-meter-up-to-65mhz)、一个[音频频谱分析仪](https://hackaday.io/project/172449-diy-fft-audio-spectrum-analyzer)和一个[自动测距电容计](https://hackaday.io/project/174205-diy-simple-arduino-autorange-capacitance-meter)等等。你可能会从我们介绍的其他项目中认出他，比如疯狂的[运动臂，它每分钟都设置一个时钟的指针](https://hackaday.com/2019/05/09/robot-arms-nudge-the-hands-of-time-in-the-strangest-clock/)。
