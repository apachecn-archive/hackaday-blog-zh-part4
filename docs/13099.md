# OpenCV 带来捏放大现实世界

> 原文：<https://hackaday.com/2022/03/24/opencv-brings-pinch-to-zoom-into-the-real-world/>

十多年前，随着触摸板和触摸屏变得更加流行，手势控制进入了公众的意识。手势控制的主要限制，至少在[诺伯特]看来，是它们只能控制虚拟空间中的物体。他希望用手势来控制现实世界中的物体，[并创造了这个用手势来控制实际图片的设备](https://hackaday.io/project/183358-control-objects-with-your-hand-gesture)。

在这个独特的增强现实设备中，由于运行 OpenCV 的计算机视觉系统可以观察他的手，不仅物体在现实世界中受到控制，而且手势也受到监控。位置数据被输入一种算法，该算法控制安装在细长机械臂上的物理图像。现在，当[诺伯特]“捏变焦”时，附在照片上的伺服系统会将照片拉近或拉远他的视野。他也可以使用其他手势来移动图片。

虽然这个手势控制的机器肯定是一个概念验证，但现实世界中的物体的手势控制还有很多其他用途。任何机器人平台都可以从这样的界面中受益，甚至是像 office PowerPoint 演示这样稍微平凡一点的东西。机会比比皆是，但是如果你需要 OpenCV [的初级读本，看看这个追踪手部细节的版本](https://hackaday.com/2021/12/23/opencv-knows-where-your-hand-is/)。
