# 时钟收音机接收升级

> 原文：<https://hackaday.com/2019/05/05/clock-radio-receives-upgrade/>

在每个人都有手机闹钟在早上叫醒他们之前，我们大多数人都使用闹钟收音机，它会忠实地坐在我们的床边很多年。你可以用刺耳的闹铃来叫醒你，也可以被当地的广播电台轻轻唤醒。这些设备现在已经不常用了，所以如果你的零件抽屉里有一个，你可以做一些小小的改变，用它来接收比你想象的更远的广播电台。

这款来自[Ryan Flowers]的松下时钟收音机与旧的时钟收音机硬件相比有几处升级。首先，它现在可以接收 7 和 14 MHz 频段(40 和 20 米)的信号。它通过为每个频率范围使用单独的带通滤波器来实现这一点，由 QRP 实验室 VFO 套件控制，一旦编程，可以在两个滤波器之间自动切换。整个装置由 8 节 AA 电池供电，就像 90 年代的收音机闹钟一样。

[Ryan]指出，他的第一次迭代有点安静，但他现在能够用这个接收器接收来自日本的广播电台。即使没有许可证，你也可以做这些改变，收听世界各地的电台，只要你不开始传输。如果你想对这个时钟收音机做一个小小的升级，[进入](https://hackaday.com/2019/03/08/the-50-ham-getting-your-ticket-punched/)并不难。
