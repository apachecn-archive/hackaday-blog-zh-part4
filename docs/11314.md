# 找回丢失软件的爱心工作

> 原文：<https://hackaday.com/2021/09/13/the-labor-of-love-that-is-recovering-lost-software/>

在家用电脑革命之初，不起眼的小型盒式磁带无疑是微机数据存储最受欢迎的选择，尤其是在欧洲大陆。作为计算机博物馆的一名志愿者，[Keith]在恢复和归档 Roger Dymond 的早期作品方面发挥了重要作用，Roger dy mond 是英国早期计算机软件的先驱开发者。

在他的视频中，[Keith]详细描述了 Roger Dymond 对早期家庭计算领域的影响。从他的理事会学徒生涯中被释放后，罗杰将注意力转向为 ZX81 开发游戏，后来又转向了 ZX 系列。在家人的帮助下，他继续经营了几年还算成功的邮购游戏出版公司。不断增加的广告成本和拥挤的开发场景让罗杰的生意在 1983 年变得不可行，但在此之前，他开发了几款赌博风格的游戏和一款出色的《太空入侵者》克隆版。

时间快进到 2021 年，虽然罗杰的一些光谱软件已经存档，但许多软件已经被在线档案员标记为“丢失”。经过进一步的研究，[Keith]意识到另一盘可能很重要的磁带被遗忘了。ZX81 的“游戏纲要”已经完全湮没在时间中，唯一的证据是它曾经存在于 1983 年《辛克莱用户》杂志上的一则广告。这份为早期型号 ZX81 编写的概要无疑会引起软件档案管理员和游戏历史学家的兴趣。

罗杰在 1999 年意外去世，他的父母后来将他的一些计算机软件捐赠给了斯温顿的计算机博物馆。第一次参观博物馆 16 年后，[Keith]能够与 Roger 的父母取得联系，询问丢失的软件。罗杰的父母很乐意帮忙，找到了丢失的母带，并友好地将它和罗杰的其他珍贵的计算机财产一起捐赠，包括他的原始频谱。

母带很可能是现存的唯一拷贝，于是[基思]承担起自己的责任，小心翼翼地修复了这个有 40 年历史的盒式机构。虽然降解泡沫垫片是一项容易更换的工作，但异常粘稠的磁带表面有可能在第一次恢复尝试中折断磁带。令人欣慰的是，磁带能够得到充分的清洁和修复，数据也成功恢复。在罗杰父母的允许下，录像带的内容被[上传到网上](https://drive.google.com/file/d/1nSR1tRB9mfG6xNgN4G_9y93npQkL0iMy/view)供所有人欣赏。

再过一年，游戏概要磁带的内容可能已经丢失。关于在正确的环境中储存这些脆弱的介质已经说了很多，但熵最终会赶上一切。我们过去报道过一些软件存档的特殊故事，但是不可能确切知道有多少软件正处于永远腐烂的边缘。
