# 又一个 DIY 手持 Pi 游戏机

> 原文:[https://hack aday . com/2019/04/02/yet-another-DIY-hand-held-pi-gaming-console/](https://hackaday.com/2019/04/02/yet-another-diy-handheld-pi-gaming-console/)

Raspberry Pi 是运行复古视频游戏的一个很好的平台，加上一些按钮、TFT 屏幕和一些扬声器，它相对便宜，并且很容易让一个工作控制台启动并运行。如果你有一台便宜的 3D 打印机，一台好看的 DIY 游戏机不花很多钱就能买到。YouTube 用户[DIY Engineering]拥有一堆消费级制造工具，并设计和制造了[一款高端但仍可 DIY 的 RetroPi 游戏机](https://www.youtube.com/watch?v=6V3yGEygNio)，RKDR II。

[DIY 工程]拥有的工具包括一台 FDM 和 DLP 3D 打印机、一个回流炉、几台不同的数控机床和一台激光切割机。都是消费级的，但不一定便宜——尤其是组合！【DIY 工程】使用 Fusion3D 对表壳、表圈和电路板进行建模，后者是在 Eagle 中设计的 4 层板，并被发送出去进行 fabbed。按键，D-pad，屏幕，电池都是买现成的，其他都是自己动手做的。查看视频了解详情——使用的工具和设计文件在 YouTube 视频下的信息部分有链接。
