# 世界上最贵的 3D 打印机

> 原文:[https://hack aday . com/2022/06/07/世界上最贵的 3d 打印机/](https://hackaday.com/2022/06/07/the-worlds-most-expensive-3d-printers/)

你会花多少钱买一台 3D 打印机？诚然，当我们开始时，一台像样的打印机可能需要 1000 多美元，但成本已经大幅下降。当然，除非你成为职业选手。我们很失望这篇[All3DP] [帖子没有包括价格](https://all3dp.com/1/biggest-large-3d-printer-world-most-expensive/)，但我们注意到一个趋势:如果你的 3D 打印机有楼梯，那可能是一笔大买卖。根据海报上的标签，这些打印机的价格都在 50 万美元以上。

昂贵的打印机通常有独特的技术，更高的自动化程度，大容量或一些组合，以及其他一些因素。提到的打印机中至少有两台有楼梯可以到达机器的顶部。黑布罗——一台水泥打印机——使用了一个看起来像音乐会灯光表演的一部分的门架。它是可扩展的，但显然可以达到三层楼高！

规模并不总是最大的因素，但它有所帮助。ExOne S-Max Pro 的构建容量为 1，260 升(参见下面的视频)。另一方面，Optomec 镜片 860 看起来很大，但不是那么大。然而，它确实能制造大型钛部件，所以就有了这个。对于哇因素，SLM 解决方案 NXG XII 600 共有 12 个激光器。每束激光的功率为 1000 瓦。

下面我们放几个视频。看看黑布罗上的高分辨率打印头。开个玩笑，我们的打印机几乎和那玩意儿的管口一样大。

在之前，我们已经见过一些[相当大的打印机。当然，有些打印机不一定要很大才能做出](https://hackaday.com/2022/02/28/3d-printering-giants/)[真正大的印刷品](https://hackaday.com/2022/02/03/conveyor-belt-printer-mod-is-nearly-all-printed/)。
