# 没有电子设备的 Flipdots

> 原文：<https://hackaday.com/2020/06/13/flipdots-without-the-electronics/>

我们习惯于 flipdots，一面是明亮颜色，另一面是黑色的单个机械像素，通过磁场翻转。驱动小电磁铁使其工作是我们社区的一个常规挑战。[Johan]然而对 flipdot 有了新的看法，这是我们以前从未见过的。他没有制造磁场来翻转他的点[，而是完全不用电子设备](https://hackaday.io/project/172388-passive-pixel-pole-project)，只使用一块磁铁。

这个项目是一个水箱的水位指示器，它包含一个漂浮在塑料瓶中的磁铁。这在以前已经被用于触发控制再填充泵的簧片开关。在这些簧片开关上，他添加了一排 flipdots，但这些不是你曾经在当地公交车上看到的装饰性商业圆点。相反，它们是由垫圈制成的定制圆点，通过点焊悬挂在枢轴上，并安装在透明管内的框架中，以防止灰尘进入。正如你在断裂处下面的视频中看到的，当磁铁漂过水箱内部时，它会将它们翻转到一个方向，在返回途中，它会将它们翻转回另一个方向。结果是一个完全可用的 flipdot 显示器，完全没有正常的电子设备，我们很喜欢它。

(这可能是我们向你展示的第一个无电子翻盖手机，但它不是第一个自制的。)
