# “反转短裤”揭开手机安全的神秘面纱

> 原文:[https://hack aday . com/2022/10/24/reversing-shorts-demystify-phone-security/](https://hackaday.com/2022/10/24/reversing-shorts-demystify-phone-security/)

有没有想过是什么让手机操作系统变得安全，或者你刚刚安装的应用程序在你背后说了你什么？在[全新的视频系列](https://www.youtube.com/playlist?list=PLkOopkYm0fCV45i_n8z5LSUL3QBXNAP2G)中，【Jiska】让我们一窥智能手机软件逆向工程的不同主题。

例如，[她的最新视频](https://www.youtube.com/watch?v=lXMawXNtXrQ)，嵌入下面，带我们通过一些步骤来戳苹果的 RTKit OS，这是一个实时操作系统，运行在他们的大多数外围设备中，包括 AirPods，但也在他们更大的设备上运行。我们不太了解 RTKit OS，但[Jiska]在这个视频中的技巧是通过查看两个不同的 RTKit OS 版本并注意哪些符号是通用的来获得立足点——这些符号可能是 OS 函数名。现在你有东西要找了。

每一个视频都很短，很中肯，并且包含了很好的提示，也许是给那些想进入手机领域的中高级反向器。见鬼，即使你不是，她对 Frida 动态追踪工具的演示也值得你花时间。

如果你想更深入地了解手机的内部结构，我们强烈推荐她的演讲，“所有的无线堆栈都坏了”。
