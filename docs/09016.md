# 什么是好的天线？

> 原文：<https://hackaday.com/2021/01/24/what-makes-a-good-antenna/>

有时，天线和射频设计似乎被描绘成某种黑色艺术，是一群射频神秘主义者的专属领地，普通人无法企及。事实上，尽管他们有困难的时刻，但还是有可能理解这个话题，而[如何开始就是来自【Andreas Spiess】](https://www.youtube.com/watch?v=6cVYsHCLKq8)的一段视频的主题。题为“如何构建一个好的天线”，它使用一个简单的四分之一波长接地平面天线的设计和设置作为手柄，向观众介绍关键主题。

这个视频之所以好，是因为它关注的是实践，而不是理论。我们得到关于连接器和天线材料的建议，我们通过在线计算器而不是大量的公式了解数学。当然，有兴趣的人可以学习完整的计算，但对于许多建造者来说，它们可能有些令人生畏。我们看到 NanoVNA 是天线制造商的一个有用工具，与以前的反复试验相比，它提供了一个革命性的性能窗口。甚至连接地层也受到了重视，它对阻抗和增益的影响得到了探究，其角度也成为性能的一个关键因素。我们认为这种方法有效地打破了天线周围的神秘感，我们希望这将鼓励观众进一步实验。
