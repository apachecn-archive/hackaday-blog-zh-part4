# 多功能 ESP8266 智能手表

> 原文:[https://hack aday . com/2018/10/26/a-多功能-esp 8266-智能手表/](https://hackaday.com/2018/10/26/a-multifunction-esp8266-smartwatch/)

我们在 Hackaday 上展示的大多数 DIY 智能手表项目并不完全是大多数人认为实用的日常使用设备。笨重的设计，短暂的电池寿命，有限的功能:它们更像是极客信誉的可穿戴展示，而不是功能性的钟表。奇怪的是，市场上的许多“真正的”智能手表也是如此，所以 DIY 版本可能比我们想象的更接近最先进的水平。

[![](../Images/ee9a92e46816cf17fe04964c419c3135.png)](https://hackaday.com/wp-content/uploads/2018/10/espwatch_detail.jpg) 但是这款由【希亚姆·拉维】创造的 [ESP8266 智能手表正在*危险地*接近你可以带着它离开家的东西。它仍然缺少一个外壳，防止你戴着它接受 PCB 针灸，但除此之外，它有一个更值得尊敬的功能。它甚至看起来是一个相当合理的尺寸(有可能更小)。所有这些的总建造成本不到 20 美元，我们认为这可能是一个值得关注的项目。](https://github.com/theTechnowright/DIY-Smartwatch-using-ESP8266)

不满足于仅仅显示时间的手表，[Shyam]增加了一个天气功能，当按下中央按钮时，从雅虎天气 API 中提取他所在地球角落的当前情况，并将其显示在手表的多色有机发光二极管显示屏上的时间和日期上方。坦率地说，鉴于 DIY 手表的现状，这已经足够令人印象深刻了；但他并没有就此止步。

左右按钮控制联网的继电器，[希亚姆]用它来打开和关闭他的灯和空调。当他按下相应的按钮时，手表甚至会显示设备的状态，无论他的旅行可能将他带到哪里。

虽然我们已经有一段时间没有看到 ESP8266 手表了，但还是有一些 DIY 手表从我们的视线中掠过。最近[我们看到了一款 Arduino 手表](https://hackaday.com/2018/08/24/an-arduino-watch-without-a-clock/)，以及一些[彻头彻尾的华丽模拟作品](https://hackaday.com/2017/10/19/scratch-built-watch-case-is-a-work-of-art/)。
