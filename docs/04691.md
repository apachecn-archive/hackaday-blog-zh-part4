# 构建无限的虚拟桌面

> 原文：<https://hackaday.com/2019/12/21/building-a-limitless-vr-desktop/>

[加布·霍瓦特]认为即使是两个显示器也太小了，无法真正合理地布置他的窗户。这就是为什么[他直接从我们最深处的赛博朋克幻想中建造了一个虚拟现实桌面](https://hackaday.io/project/164922-vr-prelimutens-3d-desktop)。

该软件目前在 Windows 和 Android 上运行。用户可以将多个窗口放在它们周围的球体中。随着他们的[头部移动，](https://hackaday.com/2007/12/21/wiimote-head-tracking-desktop-vr-display/)正前方的窗口变得清晰。想象一下，您可以同时打开多少个堆栈溢出窗口！

另一个令人兴奋的可能性是，数字工作空间可以在多个用户之间共享。结对编程并没有那么糟糕，现在远程有效编程的可能性似乎更大了。甚至配对 CAD 也是可能的，这取决于它是如何完成的。想象一下，在另一个用户的屏幕上共享您的个人 CAD 会话，并在您的屏幕旁边看到他们的会话，允许同步设计。

总的来说，这是一个非常酷的技术演示，可以变成更多的东西。这让我们不禁想知道，科技工作者去吃午饭的路上，还有多久会被他们脸上的红色圆圈标记出来。
