# GBA 远程播放升级让你在公交车上玩 PlayStation

> 原文：<https://hackaday.com/2021/11/22/gba-remote-play-upgrade-lets-you-play-playstation-on-the-bus/>

任天堂 Game Boy Advance 基本上是那个时代的掌上游戏机，因为它在市场上没有真正的竞争对手。由于硬件的限制，该系统在很大程度上是为 2D 游戏所知的。

然而，[罗德里戈·阿方索]最近升级了他的 [GBA 远程游戏系统](https://github.com/rodri042/gba-remote-play)，让他可以在他的经典游戏《男孩前进》上玩 PlayStation 游戏和其他游戏。我们在 7 月份首次介绍了这个项目[，](https://hackaday.com/2021/07/13/playstation-games-on-the-gba-with-a-few-extra-steps/)，它使用一个 Raspberry Pi 3 来模拟游戏，并将视频数据传输到手持设备进行显示，接收按钮按压作为回报。

从那以后，[罗德里戈]对这个项目进行了一些升级，以 3D 打印的外壳的形式，将电池供电的 Pi 直接安装在控制台的背面，以便便携式播放。此外，超频 GBA 允许通过手持设备的链接端口实现更快的传输速率，这意味着可以输入更多像素的视频数据。当以 240×160(GBA 屏幕的最大分辨率)运行时，这允许更多的可播放帧速率。

结果是一个 Game Boy Advance，你可以用它在公交车上玩 *Crash Bandicoot* 来迷惑正常人。当然，人们可以简单地从零开始构建[一个树莓派掌上电脑](https://hackaday.com/2019/05/16/the-raspberry-pi-portable-console-you-wish-you-had/)来玩模拟游戏。然而，这种方法利用了 GBA 的外形，并且非常有趣。休息后的视频。
