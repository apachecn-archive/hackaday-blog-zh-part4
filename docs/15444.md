# 内部带有框架的机械键盘

> 原文:[https://hack aday . com/2022/11/30/mechanical-keyboard-with-a-framework-inside/](https://hackaday.com/2022/11/30/mechanical-keyboard-with-a-framework-inside/)

像 Commodore 64 和过去的其他键盘计算机一样，[[高架系统]的 CJ64](https://www.youtube.com/watch?v=ewVp96JN0nQ) 将其所有的处理和 I/O 装入一个键盘形状的包中。

这个项目的迭代将它带到了一个新的水平，用铝制成的外壳代替了以前版本的 3D 打印外壳。对于框架主板，端口可通过框架扩展卡系统进行配置，为您提供更多选项来定制这种构建。为了使它更完整，这款键盘 PC 在键盘部分没有节省，也没有机械开关和 MT3 轮廓键帽。

如果你想为自己建造一个这样的系统，[高架系统]已经上传了 [3D 打印附件文件](https://github.com/CJ-Elevated/Project_CJ64)到他的 GitHub 仓库。用于加工的文件也是可用的，但只对顾客开放。

对于一些更多的基于框架的 mod，检查这个[框架板](https://hackaday.com/2022/08/07/build-a-tablet-out-of-your-framework-motherboard/)、[框架板](https://hackaday.com/2022/05/17/transparent-framedeck-is-clearly-capable/)，或者这个其他[复古风格的框架构建](https://hackaday.com/2022/05/10/framework-board-gets-this-round-display-pc-rolling/)。如果你想要一个全键盘平板电脑，那么也许可以看看[平板电脑是计算的未来吗](https://hackaday.com/2022/11/25/are-slabtops-the-future-of-computing)？
