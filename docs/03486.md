# 完整的台式 PCB 蚀刻站

> 原文：<https://hackaday.com/2019/07/07/a-complete-desktop-pcb-etching-station/>

现在，你只需花几美元，就可以在一周左右的时间内将定制电路板送货上门。几乎没有理由再在家里制作自己的电路板，但当你现在需要一块电路板时，你会希望有这种能力。[Tuval Ben Dosa] [设计了一个完整的 PCB 蚀刻站](http://fenneclabs.net/index.php/2019/06/29/the-complete-pcb-etching-station/)这是在家制作印刷电路板的完美工具。它拥有完美蚀刻所需的一切，使用这种设置，您可以在几个小时内制作电路板，而不是等待几天。

任何蚀刻装置的化学性质都很重要，近年来，整个社区都从氯化铁转向氯化铜，这是有充分理由的:你可以通过向氯化铜中鼓入氧气(或空气)来补充氯化铜蚀刻剂，而氯化铁是一次性蚀刻剂。

这个建筑的机械部分由一个密封的玻璃食物容器组成，它位于 PCB 加热元件的顶部，与 3D 打印机的加热床没有什么不同。除此之外，还有一个装在硅胶管里的 I2C 温度传感器、一个搅拌棒、隔膜泵和几个将空气吹入蚀刻剂并抽出产生的氯气的泵。这是由一个小的微控制器控制的，其用户界面仅由一个编码器和有机发光二极管显示器组成。

如果你正在寻找能在家蚀刻铜和黄铜的建筑，这是以前做过的事情。Etchinator 是一个神奇的建筑,能够制作从版画到印刷电路板的任何东西。这是一个需要大量工作的构建，这个小巧紧凑的蚀刻站可以满足您的一切需求，而不会占用车间太多空间。看看下面的视频。
