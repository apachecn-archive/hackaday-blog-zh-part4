# 用树脂打印机制作键盘

> 原文：<https://hackaday.com/2019/05/07/building-keyboards-with-resin-printers/>

除了在电路板上安装大量轻触开关之外，还没有人想出如何为可穿戴项目制造非常小的键盘。【Madaeon】可能有答案，它正在使用基于树脂的 3D 打印机[创造一个没有硅树脂的柔性键盘](https://hackaday.io/project/165186-diy-flexible-keyboard-without-using-silicone)。

小键盘的世界充满了实际上是遥控器的粘糊糊的部分。这使用了一块硅胶和每个按钮底部的微小碳“点”。按下按钮，这些碳点将 PCB 上的两条轨迹连接起来，从而闭合开关。虽然 ESP32 WiPhone [背后的人一直在试验铝模具](https://hackaday.io/project/159811-esp32-wiphone/log/159683-testing-the-keypad-mold)，但是还没有人掌握自制硅树脂。

[Madaeon]决定使用 3D 打印，特别是树脂 3D 打印，使用一种非常灵活的树脂，而不是走铸造和固化硅树脂的道路。构建过程如你所料——只是一些按钮形状的对象，但当涉及到桥接键盘矩阵上的连接时，这变得很聪明。这是用导电涂料完成的，小心地涂在每个按钮的下面。

目前，这是一种轻松获得小键盘的可行方法。颜色是一种花哨的粉红色，每个按钮上的标签都不像任何人希望的那样清晰可见，但后者可以通过丝网印刷来固定，就像遥控器上的硅胶按钮一样。
