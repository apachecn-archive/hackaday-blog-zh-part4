# 机械手使用旧的光盘部件

> 原文:[https://hack aday . com/2022/05/26/robotic-hand-uses-old-CD-rom-parts/](https://hackaday.com/2022/05/26/robotic-hand-uses-old-cd-rom-parts/)

机械臂和致动器是引人注目的东西，在制造商中很受欢迎，因为它们对现代工业至关重要。[kthod2000]建立了他们自己的设计，[依赖于从旧光盘驱动器中回收的部件。](https://www.youtube.com/watch?v=-X5VyPOTz5s)

手臂本身由许多看起来是 3D 打印的组件构成，沿着它的长度可以看到三个主电机。这些看起来是从几个光驱中收获的弹出马达，这些马达在输出轴上有用地带有螺纹，这使它们非常适合线性驱动应用。弹出电机由 TMC2208 驱动器通过微控制器运行，控制机械臂上下移动时的几个阶段的运动。

其意图似乎是这些三层组件中的一个可以充当单个手指。结合多次，这可能允许创造一个类似于完整的五指机器人手的东西。[kthod2000]在控制手臂的软件方面也做了大量的工作。运动学可以在屏幕上模拟，与手臂的真实运动一致。

我们以前也见过类似的构建，[像这个用废弃 DVD 驱动器构建的绘图仪](https://hackaday.com/2019/04/29/build-a-plotter-using-scrap-dvd-drives/)。他们是小型项目的优质机电元件的重要来源，所以看到他们在这里工作并不奇怪。休息后的视频。
