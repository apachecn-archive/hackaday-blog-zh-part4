# 一个机器人造型师为你锁定加长锁

> 原文：<https://hackaday.com/2020/07/14/a-robotic-stylist-for-your-lockdown-lengthened-locks/>

也许很容易认为，尽管技术发展迅速，但有些工作永远不会自动消失。我们注意到，一般来说，被称为防机器人的工作是由发布公告的人担任的。但是可以肯定的是，为人们剪发和做发型的工作永远不可能由机器人来完成，对吗？

我们不会把农场押在这上面，尽管从谢恩·威顿的隔离理发机器人来看，世界上的造型师们还需要一段时间才能领到失业救济金。据说是因为需要修剪他孩子气的长发，这个奇妙的装置是一门实物课，再现了造型师带给他们工作的每个人的微妙手工技能——毕竟，需要 1500 小时或更长时间的培训才能获得许可。[Shane]很早就发现了这一点，并意识到精确复制人手的灵巧性是不可能的。他的切割头使用真空将头发直立起来，3D 打印的手指夹住一小束头发，伺服驱动的剪刀将头发剪成一定长度。剪刀的攻击角度可以通过多个轴来调整，整个东西在一个地狱-不-我不会把我的头放在那个东西里的机制上旋转。

值得称赞的是，在戴着假头和假发进行了几次非决定性的生命安全测试后，[谢恩]勇敢地成为了机器的零号顾客。我们不会破坏结局，但足以说，事情实际上没有流血，只有最低限度的损害[谢恩]的风格。然而，这位长期忍受痛苦的(威顿夫人)并没有被说服去试驾。

严肃地说，感谢谢恩解决了如此复杂的问题。我们喜欢他为自己的体型所做的一切，比如他的篮球捕手和机器人高尔夫俱乐部，我们期待着更多。
