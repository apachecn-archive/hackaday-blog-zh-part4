# 也许你真的可以感应磁场

> 原文：<https://hackaday.com/2019/03/23/maybe-you-really-can-sense-magnetic-fields/>

我们多年前就知道许多动物能以某种方式感知磁场。鸟类显然是利用地球磁场来导航的。狗能更好地找到装有磁铁的盒子，而不是类似的装有食物的盒子。但是，显然，没有帮助，人类无法想象磁场。加利福尼亚、新泽西和日本的几位科学家已经做了实验，似乎表明当磁场旋转时，人的大脑确实会发生变化。如果这篇题为“人类大脑中阿尔法波段活动所证明的地磁场转换”的论文对你来说有点难，可能会喜欢下面来自 Veritasium 的视频，它比这篇论文更容易解析。

为了观察它的工作，受试者坐在一个黑暗隔离的房间里，带着一个电极帽来采集受试者的脑电图。这项研究表明，不同的人对磁场有不同的敏感度。此外，在一个隔离的房间里获得磁场不同于在人行道上获得磁场并用它来导航。

然而，有一些证据表明，人类文化可能在过去就受到了这种影响。许多人类语言缺乏特定的词语来描述前方和后方，而是使用基本方向，如南北。

如果你真的想感应磁场，你最好的办法可能是[在你的手指](https://hackaday.com/2017/09/08/magnet-implants-your-cyborg-primer/)里植入一块磁铁。话说回来，实际上[并不一定要用你的手指](https://hackaday.com/2013/05/18/dermal-implants-means-strapless-watch/)。
