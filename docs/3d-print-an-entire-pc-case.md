# 3D 打印整个电脑外壳

> 原文：<https://hackaday.com/2021/06/12/3d-print-an-entire-pc-case/>

随着笔记本电脑已经成为一种商品，单板电脑已经征服了我们社区的低端，为自己建造一台个人电脑不再像过去那样是一种仪式；除非你是个游戏玩家。但是在选择和组装电脑硬件的过程中仍然有很多乐趣，尤其是像[makerunit]所做的那样，[你自己设计和 3D 打印你自己的外壳](https://www.youtube.com/watch?v=OsDuiwyvBx8)。

这不是旧披萨盒中的主板，而是一个高度紧凑和设计良好的插座，用于配备 ITX 主板的合理性能游戏机。容纳所有部件的底盘位于滑入式纹理套管内，并特别注意空气流动和冷却。GPU 卡受机箱大小的限制，根本没有空间容纳传统硬盘，所以 PCIe SSD 板承担了这个角色。

我们大胆地认为，如果这种情况是由苹果这样的公司制造出来的，它会被誉为某种设计杰作，这就是它的质量。这当然表明，除了普通的矩形塔式机箱之外，还有更多的东西可以用来构建您自己的 PC。

在过去的几十年里，我们给你带来了如此多的电脑机箱，最近一个值得一看的是[这个用于英特尔 NUC 主板](https://hackaday.com/2019/10/26/lego-and-minecraft-team-up-for-custom-gaming-pc-case/)的乐高《我的世界》机箱。
