# 你可以 3D 打印你自己的遥控摩托车

> 原文：<https://hackaday.com/2021/09/11/you-can-3d-print-your-own-rc-motorbikes/>

几十年来，无线电控制汽车一直是一种受欢迎的玩具，但由于平衡问题和妥协的灵活性，它们的摩托车对手总是落后。与此同时，一个小玩具摩托车可以在行动中令人着迷。他自己设计了一些看起来非常好玩东西。

仿照杜卡迪·德拉克斯特和铃木·GSX-RR，3D 打印车身在模仿全尺寸车身方面做得很好。只要稍微涂上一点颜料，再做一些精细的修饰，这些东西绝对是精致的，但是它们在裸露的塑料中看起来还是很棒的。这种自行车使用皮带传动将动力从车身上的马达传递到后轮。为了保持直立，一个加重的前轮被用作被动陀螺仪以保持稳定。

但是最精彩的部分可以说是坐在每辆摩托车顶部的骑手。铰接，手放在车把上，骑手随着自行车的转向而移动，创造出一种令人毛骨悚然的真实感，这种真实感我们永远也看不够。头部甚至还有一个微型伺服系统，可以让骑手像你预期的那样旋转和观察运动方向。

那些希望在家里重新创造这些设计的人可以得到文件。或者，[深入研究陀螺稳定设计](https://hackaday.com/2019/09/06/who-needs-four-wheels-when-youve-got-a-gyro/)，了解更多有关其工作原理的信息。休息后的视频。
