# 镜子，墙上的镜子，我的眼睛到底欺骗了我吗

> 原文：<https://hackaday.com/2022/02/19/mirror-mirror-on-the-wall-do-my-eyes-deceive-me-after-all/>

不管你怎么说错觉，[Create Inc]有一些 3D 打印品，当在镜子里观看时，它们看起来会改变形状。例如，圆形转化为星形，反之亦然。2016 年【koki chi sugi hara】表演了一个[类似的魔术，他在镜子里展示了看起来像正方形的圆圈。为了让这个魔术成功，相机的位置(或者你的眼睛)很重要，因为从不同的角度看形状是不同的。当你的大脑忽略任何额外的信息，并得出一个复杂得多的形状就是一个简单的形状的结论时，这种错觉就出现了。[Create Inc]带你了解幻觉是如何工作的，以及它是如何在 Blender 中创建的。](https://hackaday.com/2016/07/03/hackaday-links-july-3-2016/)

当他在 Reddit 上发布视频时，大多数人似乎认为这不是一面镜子，有一些相机的诡计。从本质上来说，这是一个逆向工程魔术，我们认为这是一个令人印象深刻的魔术。如果你想打印自己的，STL 文件[在 Thingiverse](https://www.thingiverse.com/thing:5204153) 或 [Etsy 上。我们讲述了第二个错觉，即[Kokichi]使用了类似的技巧。](https://www.etsy.com/in-en/CreateIncDesigns/listing/1161746179)
