# 天气预报告诉你你需要知道的，仅此而已

> 原文:[https://hackaday . com/2020/12/20/weather-note-tell-you-what-you-need-to-know-to-more/](https://hackaday.com/2020/12/20/weather-note-tells-you-what-you-need-to-know-and-no-more/)

智能手机是信息洪流的入口。是的，它们是了解时间、公交车时刻表和天气的好方法，但它们也充满了蜂鸣器和铃声，每三分钟响一次，提醒你你叔叔已经重新发布了一张他十年前抓到的鱼的照片。有时候，最好只显示要点，这就是天气笔记的作用。

它是围绕 Adafruit Feather Huzzah 构建的，这是一个围绕古老的 ESP8266 构建的开发板。对于像这样的物联网项目来说，这是一个很好的基础，内置 WiFi，随时可用。在 If This Then That 或 IFTTT 的帮助下，天气笔记与各种在线平台对话，收集天气数据和有用的提醒。遛狗或买牛奶的提醒显示在一个小有机发光二极管屏幕上，同时还有一堆字母数字显示其他信息。WS2812 LEDs 用在阴影盒后面，显示天气状况，有可爱的云、雨和太阳图标。它被包裹在整洁的框架中，非常适合放在壁炉架或早餐桌上。

学习物联网编程是一个很好的构建，有了这些明亮的 LED 显示屏，它可能也是一个可行的夜灯。毕竟，这是一个既能告诉你天气又能防止你在厨房里踢到脚趾的罕见项目。那些想要更隐蔽身材的人应该考虑走智能镜子路线。休息后的视频。
