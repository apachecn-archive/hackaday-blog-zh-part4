# 不可更换电池？如果这项拟议中的欧盟法律通过，就不会了！

> 原文：<https://hackaday.com/2022/12/24/non-replaceable-battery-not-if-this-proposed-eu-law-passes/>

消费电子产品中一个令人不安的趋势是，可更换电池在我们的设备中逐渐消失。找到一款可更换电池的手机是一件困难的事情，许多其他设备也跟随这一趋势，密封在锂电池中。结果是电子产品的寿命越来越短，设备回收或更糟的是，填埋带来了更大的问题。然而，希望就在眼前，这要归功于一项提议中的欧盟法律，该法律如果获得通过，将使“T2”电器中的电池被设计成消费者可以很容易地自行拆卸和更换。

如果世界其他地方的读者想知道这与他们有什么关系，欧盟代表着一个如此巨大的市场，制造商既不能忽视它，也不能在大多数情况下负担得起分别生产欧盟和世界其他地方版本的产品。因此，如果欧盟要求在它的领土上销售某种东西，在大多数情况下，这就成了设计在世界范围内销售的任何东西的事实上的规范。[我们已经在欧盟的维修权立法](https://hackaday.com/2019/10/09/europeans-now-have-the-right-to-repair-and-that-means-the-rest-of-us-probably-will-too/)中看到了这一点，虽然我们毫不怀疑制造商将尽最大努力阻止这项新法律，但我们不认为他们最终会获胜。
