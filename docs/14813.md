# 在本周的黑客聊天中搭建桥梁

> 原文:[https://hack aday . com/2022/09/16/building-bridges-in-this-weeks-hack-chat/](https://hackaday.com/2022/09/16/building-bridges-in-this-weeks-hack-chat/)

在电子世界里，即使对于业余爱好者来说，这些年来东西也变得越来越小。我们从通孔元件发展到表面贴装，现在有经验的 DIY 者开始接受在设计中使用球栅阵列(BGA)元件的想法。我们想知道再过几十年，事情会变成什么样子，但坦率地说，这让我们紧张不安。

因此，虽然我们现在非常精通小事情的方法和原因，但我们很少看到大规模的工程项目。这就是为什么我们很高兴安迪·奥利弗本周能来参加重型工程黑客聊天。他的日常工作是设计和检查可移动桥梁的控制系统，或者许多人俗称的吊桥。

现在你可能认为对这种特殊技能的需求不是很多，但我们愿意打赌，这种桥梁比你想象的要多得多。安迪首先透露，仅在佛罗里达州和路易斯安那州之间，就有大约 200 座不同大小的可移动桥梁。他指出，在更大的范围内，BridgeHunter.com 在其数据库中列出了令人难以置信的 3166 座可移动桥梁，尽管不可否认其中许多是有历史意义的，而且已经不复存在。(真的什么都有网站！)

Andy Oliver

还有一个巨大的动机是尽可能长时间地保持现有桥梁的功能——如今建造一座新桥可能要花费数亿美元。相反，修理和升级是游戏的名称。安迪说，如果维护得当，一座好桥可以使用大约一个世纪。

人们可能不会惊讶地发现，保持事情尽可能简单是确保一座可移动桥梁能够经受住时间考验的关键。虽然我们可能会想象各种各样的高科技自动化系统都在工作，而且如果我们中的任何一个人负责，它们可能会工作，但安迪说，大多数时候这都是旧学校的接力逻辑。

甚至控制马达的速度也常常是使用结实的继电器将一些额外的电阻切换到电路中。但是，当可靠性和易于维修是重中之重时，谁会反对经典呢？安迪回忆说，有一次，一位政府客户明确表示，维护一座特定桥梁的控制系统，唯一需要的工具就是一把锤子。

当然，在搬运 100 万磅的钢铁时，不仅仅是电气方面的考虑在起作用。你还必须考虑像桥上的风力这样的事情，特别是你的齿轮和马达可以承受额外的负载而不会撕裂自己。这座桥还需要一个紧急停止系统，可以在接到通知的瞬间停止移动，但在此过程中不会损坏任何东西。

聊天中涉及了许多关于这些机动庞然大物的迷人细节，所以我们邀请任何曾经看过一座桥慢慢重新配置自己的人仔细阅读完整的文字记录。特别感谢 Andy Oliver 的来访，并与社区分享了他独特职业生涯的一些细节，请记住，如果您有自己的工程故事要讲，我们很乐意倾听。
