# 用古老的机械制作茶壶

> 原文:[https://hack aday . com/2021/09/23/用古董机械制作茶壶/](https://hackaday.com/2021/09/23/making-tea-pots-with-antique-machinery/)

我们西方人认为很多事情是理所当然的。我们已经习惯了在家里和工作场所的特定护理标准，以至于我们很少花时间去注意它。工作场所事故是一件大事，不报告可能会导致你被扫地出门。因此，看到在基本健康和安全的某些领域没有那么严格的方法的国家里，东西是如何制造出来的，这有点发人深省。

由于迫切希望将价格压低到尽可能低的水平，服装和家庭用品等低技术产品往往不会在高度复杂的自动化工厂生产，而更有可能是在雇佣更多劳动力的较小工厂生产，这有利于劳动力更廉价、更容易获得的国家。我们在这里强调的[视频](https://www.youtube.com/watch?v=L7KLNWCw-Gs)显示了一个小工厂，很可能是在巴基斯坦(但也可能是其他几个地方，我们只是猜测)，这似乎是制作搪瓷茶壶所需的复杂程度的典型代表。

视频显示了生产过程，从用剪刀手工切割钢板开始，在冲压成浅盘之前进行校正。前两台机器由裸露的皮带驱动，考虑到一些工人穿着自由流动的织物衣服，这是特别危险的。在背景中，你可以看到电线吊在周围，挂在钉子上。整个建筑都是一样的，没有保护功能的临时机器，由熟练的手工工人管理，专注于他们分配的任务，所有人都在完美地协调工作。看起来很可爱，但同时也很悲伤，因为你知道那些家伙正处于一千个潜在的危险之中，只有他们的技能和灵巧才阻止了一些不好的事情发生。这些机器本身到处都磨损严重，但显然被某个知道如何让它们继续运转的人黑了。只需检查视频中[4:20]刀架的深度磨损即可！
