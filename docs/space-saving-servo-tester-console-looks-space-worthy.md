# 节省空间的伺服测试控制台看起来空间值得

> 原文：<https://hackaday.com/2020/01/05/space-saving-servo-tester-console-looks-space-worthy/>

你知道这是怎么回事——有时你不得不在项目中途停下来，为自己构建一个工具，一旦项目完成，它就会极大地改善你的工作流程。YouTube 上的[Ikkalebob]又名[Will Cogley]正在进行一个超级秘密的项目，需要一整套伺服系统。由于所有这些都需要测试和设置，[他建造了这个可爱的伺服测试器](https://www.instructables.com/id/16-Channel-Servo-Tester-With-Arduino-and-3D-Printi/)作为节省时间的礼物送给自己。

该测试仪围绕着一个 Adafruit 16 通道伺服驱动器和一个 Arduino Uno。伺服系统以四个一组出现在屏幕上，可以用罐子一次测试四个。按钮可让[Ikkalebob]在群组之间上下移动。SainSmart LCD 被证明比其他的更难设置，但是[Ikkalebob]帮了你一个大忙，调整了库。它与他的代码和 STLs 一起提供。

说到 STLs，我们真的很喜欢迷你的 NASA 控制台外观和折叠式外壳。利用打印过程来制造铰链和其他东西是很棒的，使用更少的紧固件也是如此。你可以在休息后的视频中看到一些[Ikkalebob]是如何设计的。

取决于你在伺服系统上做什么，你可能想要一个不同类型的测试套件。[这里有一个面向遥控飞机飞行员的](https://hackaday.com/2016/11/23/a-dual-purpose-arduino-servo-tester/)。
