# 戴着面部投影面具的人或事

> 原文：<https://hackaday.com/2019/10/31/be-anyone-or-anything-with-facial-projection-mask/>

在市场上寻找一个低聚改变你的样子？讨厌参加化妆舞会却发现有人戴着和你一样的面具？那么[这个变脸正投影面具](https://www.instructables.com/id/Face-Changing-Projection-Mask-Be-Anything/)可能正是你想要的。

老实说，我们不确定(肖恩·哈金斯)的最新项目与角色扮演有多大关系。他似乎在对监视状态下的生活进行微妙的评论，尽管这可能不是阻挠面部识别摄像头的策略。【编者注:也可能只是万圣节？]

这个建筑由一个树莓皮和一个我们之前见过的微型投影仪组成。这些通过一个定制的印刷电路板连接在一起，并生活在一个附在一个长杆末端的小外壳内。吊杆连接到 3D 打印面罩的下巴，后者又连接到焊接头盔的悬挂系统。该投影仪由电池组供电，由智能手机应用程序控制，可以将任何你想要的东西投射到面具上——视频、效果，甚至其他人的图像。即使 Photoshop 做了一些调整来解决低角度投影的梯形失真，也有足够多的失真，其效果比化妆舞会更有艺术感。但是说实话，你的脸突然着火是很酷的。我们只是想知道佩戴者在明亮的 LED 灯照射下的能见度如何。

作为奖励，[Sean]把这个建筑变成了一个虚拟的寻宝游戏。去 13thkey.com 看看，看看你能从那里最少的线索中得到什么。
