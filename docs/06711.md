# 双向镜像改善了视频会议

> 原文:[https://hack aday . com/2020/05/29/two-way-mirror-improves-video-conferencing/](https://hackaday.com/2020/05/29/two-way-mirror-improves-video-conferencing/)

像最近的每个人一样，[Matt]最近花了更多的时间进行视频会议。问题是你很自然地想看着屏幕，但这意味着你没有看着摄像机，因此，你没有进行眼神交流。如果你使用笔记本电脑，有一个相对容易的解决办法，尽管它不是特别时尚。[马特]用泡沫板做了一个黑色的裹尸布，放在双向镜里。这有什么帮助？好了，有了这个装置，你可以把一个非常薄的黑色网络摄像头对着镜子。因为裹尸布是暗的，你可以透过镜子看到屏幕，但镜头看到的是你。

你从哪里得到一个薄薄的黑色网络摄像头？你用旧的笔记本相机做一个。它们很小，很容易重新利用，这是 Matt 以前分享过的技巧。作为奖励，这篇文章展示了一种简单的方法，用一个 LED 条制作一个漫射灯来照亮你的网络摄像头通话。

看起来它的缺点是它会使键盘难以使用，所以你可能需要将它与蓝牙键盘配对。它也有点笨重，但如果你在路上，它看起来像你可以移除镜子，折叠裹尸布，整个东西会平放运输。

即使你不想改善你的视频聊天，这里也有一些好东西。这款灯非常简单，使用一点额外的 LED 照明是个不错的选择。我们可以想出很多理由想要一个轻薄的网络摄像头。但是给我们印象最深的是它可以用作提词器。如果你让屏幕上的文字滚动(或者用鼠标滚动)，你可以在直视摄像机的同时阅读文字。这正是你下次竞选公职时需要的。

我们看到的大多数双向镜是用于无限远镜的。如果你尝试这个项目，也许可以增加一些马达和[眼球追踪](https://hackaday.com/2018/05/05/low-cost-eye-tracking-with-webcams-and-open-source-software/)，这样它会迫使你进行眼神交流。
