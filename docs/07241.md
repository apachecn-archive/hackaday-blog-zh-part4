# 这种四轴模板印刷机是 SMD 对准工具中的极品

> 原文：<https://hackaday.com/2020/07/26/this-four-axis-stencil-printer-is-the-ultimate-in-smd-alignment-tools/>

在 Hackaday，我们喜欢各种各样的构建，任何时候任何人将零件组装成其他东西，我们都会庆祝。虽然我们喜欢快速和肮脏的构建，但[这款四轴 SMD 模板打印机](https://dengler-mechatronik.de/?p=560)的适合度和完成度确实让我们很不爽。

这个构建来自[Phillip]，他和许多表面贴装用户一样，厌倦了各种胶带和 PCB 方法，这些方法通常用于将焊料模板与 PCB 走线对齐。他的解决方案是这种完全可调的模板支架，由 3D 打印零件连接的铝挤压件制成。该装置的上翻框架有一对夹子，用于牢固地固定不锈钢模板。夹钳导杆上的弹簧提供了一些预载荷，以保持模板张紧，并防止过度张紧。

模板可以在 X 轴、Y 轴和 Z 轴上移动，以便与顶部框架下方的底座上用 3D 打印的支架固定的 PCB 对齐。底座本身会轻微旋转，以克服 PCB 对齐时的任何偏斜。[Phillip]被该轴的现成回转环轴承的价格吓呆了，但幸运的是，他能够打印出一些零件，只需使用简单的滚柱轴承就可以做同样的事情，而成本只是它的一小部分。下图显示了正在使用的框架；当垫子在模板中完美地排成一行的时候，会有一种奇怪的满足感。

这让我们想起了最近我们报道过的一款类似的模版印刷机。这种方法简单得多，但这两种方法都比权宜的对齐方法简单。
