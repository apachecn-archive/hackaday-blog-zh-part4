# 像 1989 年一样安装 Linux

> 原文:[https://hack aday . com/2021/06/30/installing-Linux-like-its-1989/](https://hackaday.com/2021/06/30/installing-linux-like-its-1989/)

今天，几乎任何人都可以获得巨大计算能力的一个常见例子是将智能手机与阿波罗导航计算机进行比较。这款经典电脑是第一台使用集成电路的电脑，因此很明显，大多数现代技术都要强大几个数量级，但我们不需要回到 20 世纪 60 年代就能看到这种差异。简单地回到 1989 年，让那个时代的康柏笔记本电脑再次运行，同时使用 Raspberry Pi Zero 来帮助它运行，就足以说明这一点。

[beffinitiv]能够在原始电脑机箱内安装一个 Raspberry Pi，而不是简单地连接原始键盘和显示器，然后称之为完整的构建。最初的 286 处理器通过串行链路连接到 Pi，因此两个设备可以相互通信。将计算机启动到 DOS 并运行一小块软件允许计算机进入托管在 Raspberry Pi 上的 Linux 终端仿真器。可以退出终端，计算机将返回到原来的 DOS 设置。这也有助于绕过软盘驱动器将文件传输到 286，因为文件可以在 Pi 上无线检索，然后发送到 286。

这是新旧技术的一个非常有趣的混搭，Pi 比 286 强大两个数量级，并被楔入原始机箱内的空白空间，[beffinitiv]指出，这种计算机的融合是“边缘有用的”。这当然是对康柏的升级，对于其他试图在互联网上获得古老硬件的人来说，不要忘记你总是可以使用这样的硬件来访问 Hackaday 的复古网站。
