# 对一台 Pi 笔记本电脑来说是很大的努力

> 原文:[https://hack aday . com/2021/04/13/a-lot-of-effort-for-a-pi-laptop/](https://hackaday.com/2021/04/13/a-lot-of-effort-for-a-pi-laptop/)

构建一台 Raspberry Pi 笔记本电脑并不罕见。事实上，只要从任何一家主要的电子产品供应商那里点几下鼠标，就能让这样一个项目所需的零件迅速送到你家。但是[joekutz]持有一个无可争议的信念，即这些零件的价值有某种程度的递减回报，所以他开始建造自己的 Pi 笔记本电脑，配有€4 DVD 播放器屏幕和一大堆电路魔法来使他的零件箱笔记本电脑工作。

他需要克服的主要障碍是如何用手头的两个小型电池组为显示器和 Pi 供电。为 Pi 获得 5V 很容易，但显示器需要 8V，所以他增加了一个串联的锂离子电池(带保险丝)以达到所需的电压。这确实使充电有点困难，但他手头还有一个独特的四极[先开后合](https://en.wikipedia.org/wiki/Switch#Contact_terminology)开关，这并没有完全简化事情，但它确实使项目运行，而没有短路他使用的任何电池的风险。

该项目还利用了一个有趣的定制电路，为一个单独的锂电池提供低电压保护。总的来说，这是一门使用一些高质量的电路构建技能和电气理论来凑合手头零件(和一些 3D 打印)的硕士课程，而不是简单地花钱解决问题。最终结果是，从看电影到玩一些复古游戏，任何事情都很棒。
