# 从零开始建造的松木德比赛道上的慢速比赛

> 原文：<https://hackaday.com/2022/05/20/slow-races-on-a-pinewood-derby-track-built-from-scratch/>

松木德比赛马是童子军团体和许多其他团体的一项受欢迎的消遣。[科斯特先生]在一些 3D 打印部件的帮助下制作了自己的赛道，并用它来举办一场比赛，在通常的主题上进行有趣的转变。

赛道从一对 MDF 面板开始，面板上放置了一些条带，作为参赛者的指南。还有一个由铰链和一对销钉组成的释放机制，确保两位参赛者在完全相同的时间开始比赛。为了给赛道一个从下坡到水平的良好过渡，Fusion 360 设计了一系列弯曲的过渡件，3D 打印，并添加到球场上。

至于竞争，[科斯特先生]决定避开通常对绝对速度的关注。取而代之的是，学生们被要求制造出最慢的汽车，但仍能完成全程。不过，为了好玩，孩子们有一天时间改装他们最慢的车，参加更典型的最快获胜的比赛[。](https://twitter.com/CosterMath/status/1525660514885455875)它给学生上了一堂针对不同性能参数进行优化的重要课。

不过，你可能是守旧派，想把它看得太重而破坏乐趣。这些竞争对手可能希望考虑一些我们之前介绍过的[先进设备](https://hackaday.com/2021/07/29/pinewood-derby-scale-measures-cg/)。或者，你可以举办一场无拘无束的[作弊版比赛](https://hackaday.com/2016/11/29/fail-of-the-week-pinewood-derby-cheat-fails-two-ways/)。休息后的视频。
