# 多米诺层让你专注于推翻

> 原文:[https://hack aday . com/2021/01/01/domino-layer-let-you-focus-on-topping/](https://hackaday.com/2021/01/01/domino-layer-lets-you-focus-on-toppling/)

在一个雨天的下午，推倒多米诺骨牌是一种有趣的消遣，但把它们都摆好却是一件麻烦事。令人欣慰的是，[DIY Machines]的[[Lewis]已经建造了一个有用的机器来为你做这项工作](https://www.youtube.com/watch?v=QsAplPLtriw)，让你专注于有趣的部分！

该机器由 Arduino Uno 运行，可以通过布局进行预编程或通过蓝牙进行实时控制。它使用减速的 DC 马达在光滑的表面上行驶，用伺服系统控制方向。第二个伺服系统用于转动一个装有多达 130 张多米诺骨牌的转盘，使机器能够在不需要补充的情况下进行长时间运行。它的设计易于更换，因此可以打印多个旋转木马，以快速运行延长长度的课程。

由于 3D 打印和智能设计，该建筑是机器能够轻松完成棘手任务的一个很好的例子。我们对集成到驱动系统中的简单多米诺传送机制印象深刻，它不需要额外的电机或伺服系统。[这也不是我们看到的第一个多米诺骨牌层](https://hackaday.com/2019/09/09/one-motor-domino-laying-machine-works-for-tips/)。休息后的视频。
