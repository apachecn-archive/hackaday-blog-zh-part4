# 逆向技术:金属成形的戏剧

> 原文：<https://hackaday.com/2021/05/29/retrotechtacular-the-drama-of-metal-forming/>

这可能看起来有些过度紧张，但是金属成型的戏剧 实际上是非常戏剧性的。

这部电影是另一部典型的在学校播放的中世纪企业传播的经典，赞助商——在这种情况下是壳牌石油公司——试图表明进步的必然趋势，并成功地向儿童和年轻人展示了当他们进入一个最需要强大后盾的工作世界时，等待他们的是什么。

尽管叙述者的口音，显示的工厂似乎是在英国，其中执行的工作是一个残酷而美丽的芭蕾舞精心协调的动作。在电影开始时，碾锭机的巨大力量令人震惊，尤其是当我们被告知碾锭机毫不费力地运出的锭块重达 14 吨时。看到来自同一个金属锭的金属在被卷成卷之前高速穿过辊式轧机的最后一段，也令人暂停；如果这种锋利炽热的金属以某种方式逃出工厂，将会导致的灾难不堪设想。类似地，后面显示的拉丝过程听起来甚至很危险，随着模具直径的下降和金属丝速度的增加，声音变得越来越刺耳。

还有一些常见的迷人的时代错误，比如完全没有安全装置，以及对上百件可能会立即杀死你的事情的肆意漠视。让我们印象深刻的一点是听力保护的缺失，这无疑导致了大范围的听力损伤。然而，那是更简单的时代，进步的步伐不能因为安全装备而停止。
