# 迷你激光雕刻机可以在你的桌子上雕刻出一块地方

> 原文：<https://hackaday.com/2021/10/02/mini-laser-engraver-could-carve-out-a-place-on-your-desk/>

有几个旧的 DVD-RW 驱动器躺在那里，只是收集灰尘？你当然知道。如果没有，你可能知道在哪里可以找到一副，这样你就可以为你的办公桌制作[这个非常可爱又非常危险的激光雕刻机](https://www.instructables.com/DiY-Mini-Laser-Engraver/)。休息之后，请查看完整的构建视频。

[Smart Tronix]不仅仅告诉你如何从驱动器中回收步进电机，他们还向你展示如何完成，甚至花时间书面解释什么是步进电机以及为什么你会想在这个项目中使用它们，这是 Thingiverse 上的[[Maggie _ shah]的设计](https://www.thingiverse.com/thing:3521286)的混搭。正如你所料，这两个步进器通过一个带有一对 A4988 电机驱动器的 CNC 防护罩连接到 Arduino Uno。这些构成了运动的两个轴 250mW 的激光器连接在 x 轴上，平台在 y 轴上来回移动。我们很想有一个这样的东西来玩。那个平台上的任何东西都不安全！只是不要忘记适当的激光阻挡安全眼镜！

需要大得多又不占地方的东西？[卷起袖子，做一个 SCARA 手臂来握住你的激光](https://hackaday.com/2021/08/31/small-footprint-scara-laser-engraver-has-massive-build-area/)。
