# 纤巧德尔塔是机器人能做到的最小尺寸

> 原文：<https://hackaday.com/2018/12/28/dainty-delta-is-about-as-small-as-a-robot-can-be/>

德尔塔机器人有种令人着迷的东西。无论它们是以庄严的步伐用于 3D 打印机，还是快速移动到你几乎看不到它们在拾取和放置机器中移动，三个旋转致动器可以一起工作以产生三轴运动的方式总是值得一看。尤其是有了像这个这么小的三角洲机器人。

[KarelK16]说这是一个没有实际应用的“仅仅因为我能”的项目。他似乎已经为此努力了一段时间。下面的视频是八年前的。不管怎样，这个帖子是新的，而且很有趣。手臂上使用的微小球关节是由珠宝零件制成的；小型铜曲柄臂将三个上臂连接到微型伺服系统。附加的机械手[KarelK16]也非常聪明——第四个伺服系统通过鲍登电缆打开和关闭柔性塑料抓钳，而不是在手臂末端加载重物。它出奇的敏捷，能牢牢抓住小物体。

当然还有更大的增量，更大的增量，更有用的增量，但是我们真的很喜欢这个版本。谁知道呢——也许有一天机器人模型会作为一种爱好加入铁路模型。如果是这样的话，[KarelK16]的小三角洲可能是未来的形状。
