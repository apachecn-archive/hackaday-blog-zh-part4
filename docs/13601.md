# 夏天来了——让莫韦里诺给你割草吧

> 原文:[https://hack aday . com/2022/05/11/summers-coming-let-mower ino-cut-your-grass/](https://hackaday.com/2022/05/11/summers-coming-let-mowerino-cut-your-grass/)

在北半球，夏天即将来临。虽然我们喜欢这个季节，但我们不喜欢草坪维护。显然，围绕 Arduino Mega 2560 板开发 Mowerino 的[【salmec】也是如此。](https://www.hackster.io/salmec/mowerino-81b09e)

正如你所料，机器人使用锋利的刀片，所以你可能要小心。有传感器可以让机器自我导航，或者你可以通过蓝牙控制它。这是那些看起来很容易的事情之一，直到你尝试实际去做。尼龙修剪器线可能更安全，但它会断裂，很难保持切割。刀片更坚固，但对岩石、手指和宠物等物体也更危险。

在院子里走动也是一个问题。Mowerino 的前部有一些看起来很普通的脚轮。这可能是一个需要改进的地方，因为大多数场地对这种轮子都不友好。我们担心的另一件事是剪下的草会怎么样。在这里，一周的雨水意味着你的割草机会被剪下的草卡住。另一方面，Mowerino 的刀片较小，因此可能有助于减轻修剪堵塞。

不过，总的来说，如果你梦想机器人园丁在你的庄园巡逻，这看起来可能是一个好的开始。我们看到的大多数像这样的割草机都有大轮子。但是，当然，不是所有的[都是](https://hackaday.com/2019/04/23/mowerbot-keeping-the-lawn-in-check-since-1998/)。
