# ERRF 22:建立灯丝颜色库

> 原文：<https://hackaday.com/2022/10/20/errf-22-building-a-library-of-filament-colors/>

如果你曾经在五金店翻阅颜色样本，试图匹配一种特定的颜色，你就会知道这有多难。不仅有几乎无限的颜色变化，而且每个制造商都有自己的公式和色彩。通常情况下，获得您需要的确切颜色的唯一方法是进行定制混合。

不幸的是，当谈到 3D 打印机的灯丝时，这真的不是一个选项。Hatchbox 里的橘子卷真的会和 Overture 里的橘子卷一样吗？[这就是灯丝图书管理员](https://filamentcolors.xyz/library/)的用武之地。该项目由[Joe Kaufeld]创建，旨在对尽可能多的 3D 打印机细丝进行编目和拍照，这样你就可以确切地看到你得到了什么。

[![](img/d9cecb23a16692cf0523f8b33d2ebc9b.png)](https://hackaday.com/wp-content/uploads/2022/10/filamentlib_detail.jpg) 现在，当然，如果像在你的电脑上看灯丝样本一样简单，你就不需要这项服务了。那么诀窍是什么呢？由 Raspberry Pi 驱动的定制自动化相机装备用于定位、照明和拍摄库中的每个灯丝样本。因此，虽然[Joe]不能保证您的显示器完美呈现每根灯丝的颜色，但您至少可以确定它们彼此之间的关系看起来是正确的。举例来说，该网站可以帮助你找出当地的微中心是否有任何接近匹配 [Prusament 的银河银计划](https://filamentcolors.xyz/swatch/145/)。

[Joe]带着他收集的样品和他精巧的相机装置参加了 2022 年东海岸 RepRap 节,这样与会者就可以直接看到他如何向数据库添加新的灯丝。凭借易于使用的触摸屏界面，只需几秒钟即可让相机为下一次拍摄做好准备。

现在他已经有了硬件和程序，[Joe]正在请求社区的帮助，通过向他提供细丝样品来处理。这并不需要太多:他所要求的只是你从他身上剪下几米长的细丝，在一张预先做好的表格上写下它是什么以及是谁做的，然后把它投到邮箱里。如果你在美国，你可以直接把它寄到他在印第安纳的地址，对于地球另一边的人，你可以使用他在荷兰的投递点。

我们喜欢 Hackaday 的一个很好的激情项目，所以这里希望灯丝图书馆员从全球各地收到大量新的灯丝样本，以馈入他的花哨相机设置。
