# Rhysonic Wheel 自动播放现场音乐

> 原文:[https://hack aday . com/2019/05/18/the-rhysonic-wheel-automates-live-music/](https://hackaday.com/2019/05/18/the-rhysonic-wheel-automates-live-music/)

在音乐界掀起波澜越来越难了。几乎任何能上网的人也能接触到几把吉他，可能认识一个鼓手，或者能给鼓机编程。在所有这些竞争中，想要脱颖而出是很难的。然而，你可以[从零开始制作你自己独特的乐器](https://www.youtube.com/watch?v=A2GkGcQloUg)，用它来制作你独一无二的音乐和现场表演，而不是使用典型的乐队设置或自己制作平庸的说唱曲目。

[皮特·奥康奈尔]的乐器被称为 Rhysonic Wheel，这是他在车库里花了一年时间创造的。该装置由几个轮子组成，所有轮子以相同的速度驱动，并有一根公共轴。在每个轮子的不同位置，都有金属片或橡胶片系在绳子上。金属和橡胶钻头四处乱飞，并能以特定的间隔撞击各种其他乐器。[皮特·奥康纳]用它们来敲击一系列打击乐器、一套钟，甚至在演出的后期弹奏吉他。

虽然这看起来有些危险，但我们认为这给已经很有才华的音乐表演增加了一点刺激。毕竟，在熟练的人手里，[任何数量的东西都可以被用来创造一个迷人的和无与伦比的音乐表演](https://hackaday.com/2016/03/03/incredible-marble-music-machine/)用各种我们大多数人从未听过的声音。
