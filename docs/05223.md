# 城市时钟是对巴黎建筑的美丽致敬

> 原文:[https://hack aday . com/2020/01/04/city-clock-is-beautiful-致敬巴黎建筑/](https://hackaday.com/2020/01/04/city-clock-is-beautiful-tribute-to-parisian-architecture/)

二进制时钟通常是作为编程练习创建的，或者是为了用最少的实用组件显示项目中的时间。然而，用二进制显示时间并不总是出于实用的考虑。[城市时钟表明它也可以很巧妙的应用](https://www.instructables.com/id/The-Binary-Clock-the-City-Clock/)。

电子方面的事情很简单 Arduino Nano 运行 13 个 led，每个都有一个数字 IO 引脚。包括一个实时时钟模块是可选的，虽然我们认为如果你希望时钟保持准确的时间是非常必要的。发光二极管安装在一个网格里，网格安装在建筑物有窗户的正面后面。这有助于阻挡相邻片段之间的漏光，使最终设计看起来更加完美。整个组件由激光切割的木材制成，如果你手边有这样的工具，这将是一个快速简单的制作过程。

这是一个简单的概念，但在行动中却特别引人注目。即使对于那些不知道它的钟表功能的人来说，它也呈现出一个活生生的建筑的外观，里面的居民整天开关着灯。它可以做一个极好的书架或咖啡桌，我们非常想尝试自己动手做一个。休息后的视频。
