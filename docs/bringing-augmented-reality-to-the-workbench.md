# 将增强现实带到工作台上

> 原文：<https://hackaday.com/2018/07/26/bringing-augmented-reality-to-the-workbench/>

[Ted Yapo]对使用增强现实作为工具来增强电子工作台有很大的想法。他的概念是使用一个摄像头和投影仪系统一起工作来检测工作台上的物体，并将信息投射到物体上及其周围。[Ted]设想将数字多用表、示波器、逻辑分析仪和其他仪器的虚拟显示投影到实际工作区域的一个方便位置，消除在工具和仪器显示之间来回扫视的需要。然而，这仅仅是开始。一个好的相机和投影仪系统可以读取元件袋上的条形码来跟踪库存，通过预测元件的去向来指导手动 PCB 组装，显示参考数据等等。

一个开源的、可访问的机器视觉系统与一个投影仪协同工作将打开许多大门。幸运的是[Ted]在这方面有经验，他曾经为房间尺度的动态投影环境编写过计算机视觉代码。这是他可以应用于设计工作台规模系统的可靠经验，作为他参加 Hackaday 奖的参赛作品。
