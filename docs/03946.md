# 用 3D 卡坦岛为游戏之夜做好准备

> 原文：<https://hackaday.com/2019/08/20/get-ready-for-games-night-with-3d-catan/>

卡坦的定居者是棋盘游戏爱好者的主要选择。有些粉丝喜欢通过建立自己的定制集来表达自己，[Maclsk]也不例外。[进入 3D 卡坦！](https://www.instructables.com/id/3D-Catan-Designed-3D-Printed-and-Painted/)

各个部分的模型是在一个伟大的开源 3D 建模程序 [Blender](https://www.blender.org/) 中设计的。然后在 Anycubic i3 Mega 上打印，耗时约 80 小时，使用 700 克 PLA 细丝。有 116 个游戏棋子，有大量的归档和打磨工作要做。

完成后，是时候油漆了。[Maclsk]展示了对模特绘画基础知识的深刻理解，从干刷到使用 PVA 胶水赋予水元素光泽。如果你是技术新手，和你当地的战锤玩家坐下来——他们会给你指出正确的方向。

总的来说，这是一个非常棒的构建，非常适合游戏。我们已经看到其他卡坦的铁杆粉丝也推出了他们自己的版本。休息后的视频。
