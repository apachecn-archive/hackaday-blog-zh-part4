# 巨型 LED 显示屏是 1200 个球对着墙

> 原文:[https://hack aday . com/2019/08/29/giant-led-display-is-1200-balls-to-the-wall/](https://hackaday.com/2019/08/29/giant-led-display-is-1200-balls-to-the-wall/)

当你要构建一个大的东西时，从小的开始并首先解决 bug 通常是一个好主意。这就是[bitluni]最近在汉诺威 Maker Faire 上展示的他的巨大的 1200 像素 LED 视频墙所做的。

我们不久前报道过他的原型，一个只有 300 个乒乓球隐藏在一个大面板上的发光二极管。你可能还记得他在建造过程中的艰辛，包括为面板选择有问题的钢板，以及用手钻钻 300 个孔所需的费力工作。为了避免重蹈覆辙，[比特鲁尼]使用了他自己制造的[定制打孔机](https://hackaday.com/2019/08/07/punch-those-hole-drilling-blues-away-with-a-homebrew-punching-tool/)而不是钻孔机，并为所需的四块面板配备了铝板。这仍然是一个很大的工作量，他必须装配一些帮助，使工具使用起来更舒适，但最终打孔看起来比钻孔更整洁。

[bitluni]掌握了足够多的 TIG 焊接技术，可以为电池板制作精美的铝框架，重量轻，便于运输。1200 个乒乓球、一个发硬的烙铁和一包热胶棒之后，这面墙就可以安装电子设备了。它需要一个 70 安培的电源和一个 ESP32 来运行一切，但这足以制作一些令人印象深刻的图形，甚至是直播视频——不连贯和低分辨率，但仍然可用。

我们喜欢这面墙的外观，也感谢为它付出的努力。看到[比特鲁尼]在他的建造中获得如此多的乐趣总是很好的——这很有感染力。
