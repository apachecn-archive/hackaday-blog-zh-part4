# 远程学习天地

> 原文：<https://hackaday.com/2020/09/16/distance-learning-land/>

[family over mammy]甚至在疫情之前就已经在家教育她的孩子了，所以她作为一名学习教练和老师相当精通。她为孩子们设计的活动之一是让他们用 Tinkercad 制作 3D 模型。本着开放和培养自由思维的精神，她没有给他们太多的限制。而是给他们自由去创造性地设计他们想象的任何场景。

在 Instructable 中，她分享了儿子们的设计，以及重建模型的说明。你将看到的设计非常广泛，所以她直接把 Tinkercad 的设计嵌入其中。你甚至可以看到一些视频展示。

这是一个非常酷的展示一些相当出色的工艺的橱窗。此外，可能对我们的一些读者有一点启发，他们正在创造自己在家工作的活动。
