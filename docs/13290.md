# 享受这个电子眼球的平滑移动

> 原文：<https://hackaday.com/2022/04/08/enjoy-this-animatronic-eyeballs-smooth-moves/>

[Enza3D]展示了一个惊人紧凑的关节式电子眼球，可以用 Wii 双节棍控制器直观地控制。该设计使用 3D 打印部件和一些微型伺服系统，所有必要的电子设备都可以很容易地在网上购买。眼睛的机械设计令人印象深刻，[Enza3D]经历了几个不同版本的设计，最终结果是一个整洁的小组件，可以很好地适合面具，服装或其他项目。

[![](img/395e543ce465e26a53b15c3a8223e886.png)](https://hackaday.com/wp-content/uploads/2022/04/Animatronic-Eye-Animation.gif)Wii 双节棍非常适合手动控制这样的设备，这要归功于它的[人体工程学设计和轻松的界面](https://hackaday.com/2021/01/26/arduino-learns-the-martial-arts-with-nunchucks-input-device/)(双节棍通过 I ^( 2 ) C 进行通信，即使是最普通的微控制器也能轻松实现。)当然，由于如今驱动伺服系统也几乎是微不足道的，因此将它纳入自动化项目看起来不会构成太大的挑战。

眼球看起来很棒，但如果你想自己尝试，[访问设计文件和代码将花费你 10 美元](https://www.myminifactory.com/object/3d-print-213696)，如果像这样的眼睛是项目中缺少的一环，这可能看起来很有吸引力。

另一方面，欣赏视频(嵌入在下面)并从[Enza3D]的设计笔记中获得灵感只需几分钟。
