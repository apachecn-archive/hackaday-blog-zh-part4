# 在这种受风琴启发的乐器中，音轮发出颤音

> 原文：<https://hackaday.com/2021/02/10/tonewheels-warble-in-this-organ-inspired-musical-instrument/>

年轻的读者可能不记得每个购物中心都有音乐商店的日子——不是那种卖磁带和唱片的商店，而是那种你可以买到乐器的商店。这些地方不可避免地会有员工用电子琴为所有人演奏购物音乐。更多的时候，管风琴手在演奏哈蒙德风琴，这些乐器的独特声音是由类似于这个音轮风琴机器人的东西产生的。

转速轮是带齿的铁磁性轮，在拾音线圈附近旋转。这感应出一个可以被放大的电流；改变齿廓或改变转速，你就能控制发出的声音。虽然哈蒙德风琴使用这种技术来产生各种各样的声音，但[混合信号]的努力要小得多，但仍然很有趣。一个步进电机和一个 1:8 比率的 3D 打印变速箱为一对轴提供动力，每根轴都携带三个不同的转速脉冲轮。转速脉冲轮本身由软钢激光切割而成，从看起来像直齿轮的东西到只有几个大叶片的轮子都有。这比之前的乐器版本有了很大的进步，之前的版本使用的是由磁丝 3D 打印的音轮。

每一个音轮都有自己的拾音器，用之前制作的【混合信号】绕线轮缠绕。每个线圈都有一个软铁芯，允许添加一个或多个钕偏磁，这极大地改变了音调。下面的视频展示了构建和演示；如果你只是想听乐器演奏，就跳到 16:10 左右。这很有趣。但这显然是一项正在进行的工作，我们渴望看到它的进展。
