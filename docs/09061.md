# 梦想一个透明的(PCB)圣诞节

> 原文：<https://hackaday.com/2021/01/26/dreaming-of-a-transparent-pcb-christmas/>

[Carl]想把他的力传感器放在一个透明的 PCB 上，不得不向他的电路板供应商要一个特殊的样品。柔性 PCB 可在 PET 制成的透明基板上使用，但不如聚酰亚胺板常见。正如[Carl]发现的那样，这些电路板有点厚，柔韧性差，不能像标准电路板那样承受很高的温度。他没有被吓倒，他用透明的木板设计了一棵 3D 圣诞树。你可以在下面的视频中看到的结果看起来非常好，用传统方法很难复制。

当你建造木板时，它是一个平面螺旋，但是从中间提起它可以让它扩展成一个圆锥形的树形。电路本身只是一个 LED 闪光灯，但柔性板是有趣的部分。

不要让高温影响了你，因为电路板在 215 摄氏度左右会变形，需要小心焊接，你的项目可能无法在这种温度下运行。

曾几何时，任何一种柔性 PCB 都是一个大项目。现在很容易做到，也不是特别贵。[Carl]提到他为了速度和美观违反了一些最佳实践，但结果很好，看起来很酷。

我们注意到[Carl] [非常喜欢柔性印刷电路板](https://hackaday.com/2020/06/26/hinge-brings-new-meaning-to-flexible-pcb/)。我们也注意到他们出现在许多[艺术项目中](https://hackaday.com/2020/05/24/flexible-pcb-earrings-put-the-art-in-art-deco/)。
