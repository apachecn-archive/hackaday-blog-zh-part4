# 数控路由器框架改造为彩色字符串艺术机器人

> 原文:[https://hack aday . com/2020/12/11/CNC-router-frame-purposed-for-colorful-string-art-bot/](https://hackaday.com/2020/12/11/cnc-router-frame-repurposed-for-colorful-string-art-bot/)

疫情的封锁是残酷的，但它们也有副作用，刺激了创造力，并承担了足够复杂的项目，以避免变得疯狂。这个 CNC 弦乐艺术机器人是一个很好的例子，展示了一点想象力和大量时间的可能性。(视频，嵌在下面。)

根据[knezuld11]的说法，机器人通过数学算法创建艺术，通过 Python 程序将它们翻译成钉子位置和字符串路径。改进的数控刳刨机框架由激光切割胶合板制成，有两个可互换的工具头。第一种方法是将钉子放在一个小漏斗中。在由伺服控制的磁性臂拾起并垂直握住后，齿轮驱动的撞锤将每个钉子按正确的坐标推入木板。在换上不同的工具后，机器人能够从九种不同的线分配器中选择一种。激光传感器确认线嘴的位置，然后线开始绕着钉子的漫长旅程。看起来有点令人着迷，艺术看起来很棒，有一种把我们带回 70 年代的感觉。太棒了，伙计。

这让我们想起了最近的一个项目，用重叠的琴弦制作艺术品。那个很酷，因为它只用一种颜色就完成了任务，而这个真的给派对带来了色彩。选择你的选择，放置你的指甲，然后开始穿弦。
