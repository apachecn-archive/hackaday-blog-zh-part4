# 日本将发射木制卫星

> 原文：<https://hackaday.com/2020/12/30/japan-to-launch-wooden-satellite/>

根据本月[京都大学](https://en.wikipedia.org/wiki/Kyoto_University)和[住友林业](https://en.wikipedia.org/wiki/Sumitomo_Forestry)发布的[公告，我们可能会在几年内拥有木制卫星，这两个组织的历史可以追溯到 550 年前。](https://www.bbc.com/news/business-55463366)

伍德在高科技领域的地位由来已久。在第二次世界大战期间，木船被用于扫雷舰，云杉鹅号被设计用于规避战时材料限制，英国的胶合板制造的哈维兰蚊子号雷达截面非常低。在本世纪，波斯尼亚的一名男子甚至用橡木制造了一辆大众甲壳虫。

由退休宇航员和工程师土井隆雄教授领导的新宣布的航空航天项目计划发射木制卫星，以减少重返大气层时产生的空间碎片和危险物质。从危险物质的角度来看，我们有些怀疑(在这个问题上[我们并不孤单)，但作为一种帮助确保重返大气层时完全燃烧的方式，木材是一种有趣的材料。它还实现了很大的强度重量比，作为一种可再生资源，它很容易获得。](https://arstechnica.com/science/2020/12/wooden-satellites-an-intriguing-idea-but-wont-solve-space-junk-problems/)

土井教授几年来一直在研究木材在太空中的用途。早在 2017 年，他就开始了关于木材在太空中的[可用性(第 16 页)](https://research.kyoto-u.ac.jp/files/7015/6937/8766/spirits_report_2017_2018_KUR.pdf)的基础研究，他的团队在真空环境中对针叶树(雪松和柏树)和硬木(缎木、木兰和榉树)进行了实验。基于成功，他们预测木制卫星将在 21 世纪 20 年代中期发射(他们本月宣布的时间是 2023 年)。住友商事的工程师还没有公布卫星最终将使用哪种木材。

你可能还记得在国际空间站上进行实验的宇航员土井，他在那里成功地演示了在太空中飞行回旋镖(下面的视频)，他还在业余时间发现了两颗超新星。我们祝他好运。
