# 3D 打印火车套装以速度为目标

> 原文：<https://hackaday.com/2020/05/12/3d-printed-train-set-aims-for-speed/>

对于大多数参与这项爱好的人来说，模型火车包括从现成的供应商那里购买轨道，以及许多精细的油漆和修整。相反，【伊凡】只想做一些快速有趣的东西，[适时推出 3D 打印机。](https://www.youtube.com/watch?v=ikFAfHOeNl8)

虽然“世界上最快的玩具火车”的头衔有些可疑，但作为一种有趣的做事方式，这种建造有其价值。这列火车是 3D 打印的，带有压入式滚珠轴承和转向架的金属轴。与通常的做法不同，这种火车以脂肪电池的形式在车上携带电源。它连接到一个无刷电机上，由一个标准的遥控汽车装置控制。

轨道是一个令人印象深刻的结构，由 3D 打印的轨道和支架组成。这些被组装起来，然后用螺丝固定在胶合板基板上，这些基板被热粘在[Ivan]工作室平坦的混凝土地板上。绳子被用来尽可能直直地排列所有的东西。这条赛道以陡峭的坡度为特色，有助于转弯。然而，直道仍然是倾斜的，以避免复杂的过渡模型。这导致在平坦路段上以较高速度脱轨。

虽然它还不完美，但[Ivan]已经做了很好的工作，展示了一种快速简单的方法，用几乎完全 3D 打印的组件建造一个铁路模型。我们迫不及待地想看到铁轨和轨道车的改进，并希望在未来的测试中看到速度大幅提高。3D 打印往往会给模型火车世界带来一些有趣的结果，[比如这个垂直悬挂装置](https://hackaday.com/2019/11/28/vertical-train-hauls-up-the-wall/)。休息后的视频。
