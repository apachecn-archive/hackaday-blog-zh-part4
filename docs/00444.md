# 用这个 DIY 计算器轻松地数到 F

> 原文:[https://hack aday . com/2018/09/06/count-to-f-easy-with this-DIY-calculator/](https://hackaday.com/2018/09/06/count-to-f-easily-with-this-diy-calculator/)

一些有史以来最伟大的电子计算器，包括古老的 HP-16C，都包含了在不同基数之间转换数字的功能。3735928559 在 10 进制中可能没什么意义，但是把它转换成十六进制，你会冒犯素食主义者。如果昔日伟大的计算器有办法在数基之间转换，这意味着必须有人做一个独立的设备来做同样的事情，对不对？这就是[【leumasyerrp】为进入 Hackaday 奖所做的事情，无论如何](https://hackaday.io/project/135653-base-convert)。

Base Convert 项目是一个简单的桌面计算器，用于在十六进制、十进制和二进制之间进行转换。为此，数字 0 到 f 有一个 8×8 的按键矩阵。有 16 个单 led，4 个七段 led 用于十六进制显示，6 个七段 led 用于十进制显示。

虽然这实际上只是[leumasyerrp]想出的一个学习 MSP430 微控制器的项目，但鉴于它大量使用磨制 PCB 作为前面板，仔细选择支架高度，并且所有东西都藏在一个封装中，看起来就像你所能期望的完全由 PCB 制成的设备一样专业，这*看起来*像一个了不起的项目。当然，基数转换计算器按预期工作，可以很容易地在数字基数之间转换。

有时，这并不是关于构建最困难或最复杂的项目，而是简单地将最大量的润色放入一个简单的项目中。这是一个做得很好的项目，我们很高兴这个项目有望获得 Hackaday 奖。
