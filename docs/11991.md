# N-Gage 控制器使用所有按钮

> 原文:[https://hack aday . com/2021/11/25/n-gage-controller-uses-all-the-buttons/](https://hackaday.com/2021/11/25/n-gage-controller-uses-all-the-buttons/)

如果你能保证一个视频游戏系统有什么，那就是在一个人遭受商业失败的 20 年后，会有一小群狂热的爱好者痴迷于这个系统。虚拟男孩、雅达利捷豹，当然还有诺基亚 N-Gage 都是如此。对于那些不熟悉的人来说，这是 Game Boy Advance 的一个古怪的竞争对手，也是一部手机。由于这个原因，它比四人游戏室有更多的按钮，[这导致了像这个定制控制器](https://twitter.com/adeeelnv/status/1462034340364177410?t=6R02x-uS_4sHcD5HkboMyA&s=19)这样的东西。

如今大多数 N-Gage 游戏都在[模拟器](https://12z1.com/)上进行，这个版本是专门为模拟器体验打造的。最初的系统有如此多的按钮，以至于很难找到一个标准的 102 键键盘与之相匹配，所以定制几乎是必要的。这个项目的创造者[Lvaneede]从他现有的拱廊橱柜中取出一些零件，3D 打印出这个盒子，以便制作这个定制控制器。他选的按键对他的喜好来说有点生硬，但比用键盘好多了。

在下面的视频中，[Lvaneede]用 N-Gage 的几个游戏演示了它。它似乎保持得很好。在索尼和世嘉的支持下，很遗憾这些游戏平台没有获得更大的成功，但周围有很多拥有原始硬件的人仍然在修补和修复它们，所以他们仍然可以玩一些独特的游戏。
