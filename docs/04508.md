# 雄蜂长出翅膀

> 原文:[https://hackaday.com/2019/10/21/a-drone-sprouts-wings/](https://hackaday.com/2019/10/21/a-drone-sprouts-wings/)

虽然业余爱好界有一些固定翼无人机，但当提到这个词时，我们大多数人都会想到四轴飞行器。已经有一些固定翼飞机和许多多旋翼飞机，但没有太多两者的混合。[pawez]想看看如果他将这两者混合在一起会发生什么，[并创造一个带可伸缩机翼的四轴飞行器](https://www.youtube.com/watch?v=xyE4Zi1S_MI)，本质上只是想看看会发生什么。

这也不是像 V22 鱼鹰或鹞那样可以从固定翼飞行转换成直升机式悬停的东西。升力和推力完全由旋翼产生，而“翅膀”本质上是可展开的空气制动器，允许无人机快速减速，而不需要单独在螺旋桨动力下消耗那么多能量。空气制动翼被设计成可以根据油门位置自动展开，所以理论上，未来可以在这个想法的基础上做很多事情。

[pawez]指出，这种设计有些争议，尽管我们中很少有人在无人机竞赛社区中，但我们可以想象这样的功能变化可能会在这样的舞台上产生怎样的影响。他也只看到了边际性能的提高，并不打算进一步研究这个想法。如果你对有“真正”机翼的无人机感兴趣，看看这个从榴弹发射器发射的无人机。
