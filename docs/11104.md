# 使用此 HP15C 仿真器进行计算，就像现在是 1989 年一样

> 原文：<https://hackaday.com/2021/08/31/calculate-like-its-1989-with-this-hp15c-emulator/>

过去，你对计算器的选择很能说明你的能力，没有什么能比传说中的惠普 Voyager 系列可编程计算器更能说明问题。从横向布局到神秘的键帽，再到反波兰符号，这些计算器的每一个细节都体现了严肃的目的。

遗憾的是，这些计算器现在无论如何都很难买到。因此，如果你觊觎它们独特的外观和感觉，你最好的选择可能是像[alxgarza]和[一样构建自己的航海家系列模拟器](https://hackaday.io/project/181296-hp15c-calculator-emulator)。这个特殊的构建模拟 HP15C 并在 ATMega328 上运行。纯粹主义者可能会反对 192×64 LCD 矩阵显示器，而不是原版的十位数七段显示器，但我们一点也不介意更新。构建仿真器的 PCB 大小刚好合适，键盘由分立开关组成，其咔嗒声与原件一样令人满意。我们也欣赏除了通孔组件之外什么都不用——这看起来很复古。下面的视频显示，该计算器在没有外壳的情况下完全可用；不过，3D 打印的外壳是可用的，因为它是复制原始键盘的覆盖层。

我们已经看到了过去其他经典计算器的仿真器，包括[辛克莱](https://hackaday.com/2018/06/22/your-own-sinclair-scientific-calculator/)、[德克萨斯仪器](https://hackaday.com/2013/01/03/web-based-ti-graphing-calculator-emulator/)，甚至[其他惠普产品线](https://hackaday.com/2015/08/18/hackaday-prize-entry-the-70s-called-they-want-this-calculator/)。但是这个有一个非常好的设计，让我们继续前进。
