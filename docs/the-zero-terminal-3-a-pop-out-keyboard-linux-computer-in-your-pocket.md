# 零终端 3:口袋里的弹出式键盘 Linux 电脑

> 原文：<https://hackaday.com/2020/08/10/the-zero-terminal-3-a-pop-out-keyboard-linux-computer-in-your-pocket/>

手机革命给我们带来了包装精美、方便易用的电脑，但却没有我们在社区中喜欢的那种简单易用的黑客能力。与此同时，单板计算机的出现给了我们负担得起的超级强大的硬件，可以运行非常强大的 GNU/Linux 操作系统，满足我们所有的黑客计算需求。但是把两者结合起来呢？很多人都尝试过，但很少有人成功地用后者的开放力量制作出像前者一样流畅的东西。如果你想让你的便携设备有一种 cyberdeck 的氛围，这很好，但也许不是你会带到董事会会议室的东西。不过不要害怕，因为[N-O-D-E]有解决方案，在第三版零终端中。这是基于 Raspberry Pi 的手持计算的终极，它类似于一部略显笨重的手机。

它的核心是一个 Waveshare 有机发光二极管 5.5 英寸触摸屏，背面安装了一个 PCB，上面有一个 USB 集线器和电源电路。一个 Pi Zero 直接安装在上面，一个巧妙设计的 HDMI 适配器板将其连接到显示器。电源板是一种通用板，为 PCB 设计的电源板很难手工焊接。有一个设计非常巧妙的外壳，让它有移动电话的感觉，背面是一组插座，带有所有相关的 Pi 连接。这开启了一些令人兴奋的附加功能的可能性，其中第一个是类似于早期 Android 手机上的滑动键盘。这种板是基于 Bobricius 的设计，尽管遗憾的是还没有完全投入使用。

正如你在下面的视频中看到的，这是一个尽可能光滑的移动 Pi。我们想要一个。拿走我们的钱！
