# 米奇的大计时器让滑翔机比赛变得更好

> 原文：<https://hackaday.com/2021/06/13/mickeys-big-timer-makes-glider-competitions-better/>

世界上有很多不为人知的运动。他们中的许多人可以受益于定制设备，如得分显示器，但太模糊，无法支持这方面的商业努力。无线电控制滑翔机比赛就属于这一类。这使得一个名叫[米奇]的人发明了他称之为米奇大计时器的东西，来帮助举办这样的活动。

滑翔机比赛在户外阳光充足的情况下进行，因此该系统使用大而明亮的 LED 矩阵显示器来显示其计时信息。该系统围绕 STM32 发现平台构建，使用几个微控制器板来驱动几个显示器以及处理时序的主控制器。它还包括一个音频系统，用于向参赛者发出指令。它还可以显示飞行员的名字以及指示，比如选手在预赛结束时应该何时着陆。

Github 上有一些代码可供那些对其工作原理感兴趣的人使用。RC 论坛上流传着这样的话，[米奇]建造了几个系统，其中一些最终远在新西兰，他们在那里帮助举办了多年成功的滑翔机比赛。

这些年来，我们已经看到了大量的记分牌项目；一个小巧便携的手机可以为你的邻里游戏增添一些趣味。休息后的视频。
