# ALEKYA 用砖块和砂浆建造

> 原文:[https://hack aday . com/2019/09/08/alekya-builds-with-bricks-and-mortar/](https://hackaday.com/2019/09/08/alekya-builds-with-bricks-and-mortar/)

建筑通常是一项劳动密集型任务，尤其是在电力和机械供应有限的发展中国家。一如既往，机器人承诺帮助减轻负担。[[Nishant Agarwal]正在从事这样一个项目，名为 alek ya](https://hackaday.io/project/167297-alekya-open-source-construction-robot)。

ALKEYA 的目标是在机器人技术的帮助下，使建筑更加容易和自动化。在之前，我们已经见过大型混凝土打印机，但 ALEKYA 采取了不同的策略。它注重利用当地材料，将两个门架结合在一个框架上。其中一个放下一滴砂浆，然后另一个猛扑进去把砖块放好。接着是另一层砂浆，继续建造。

通过使用这种构造方式，与必须一层一层地建造高度的更传统的 3D 打印技术相比，进展可以快得多。目前在小规模运行，该项目的下一步是建造一个 20×20 英尺的版本，用于创建全尺寸的建筑。
