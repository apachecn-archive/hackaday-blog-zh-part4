# 低调定制键盘的清晰指南

> 原文:[https://hackaday . com/2022/03/16/a-clear-guide-for-a-low-profile-bespoke-keyboard/](https://hackaday.com/2022/03/16/a-clear-guide-for-a-low-profile-bespoke-keyboard/)

冒着踩上我们神奇的 Keebin with Kristina 系列的风险，由[Ben Vallack]编写的一个美丽的教程详细介绍了如何[非常详细地创建一个定制的薄型键盘](https://www.youtube.com/watch?v=UKfeJrRIcxw)。

我们已经介绍了[构建你自己的](https://hackaday.com/2018/07/03/the-a-to-z-of-building-your-own-keyboard/)和[项目的完整指南，制作定制的橡胶圆顶键盘](https://hackaday.com/2017/02/10/a-guide-for-building-rubber-dome-keyboards/)。此外，还有几个围绕定制键盘构建的子网站，以及几十个专门销售零件的网站。那么为什么不多加一个指南呢，尤其是在像[本的]做得这么好的网站上？

[Ben]专注于布局键盘的高级工具和方法，以及它们是如何组合在一起的。这一切都始于使用 [Ergopad 确定您的具体手型和布局。](https://pashutk.ru/ergopad/)处理好形状后，您可以进入 [Ergogen](https://ergogen.xyz/) ，它允许您选择您想要的布局，并生成一个只需布线的 KiCAD 电路板布局。可翻转板、开关的各种尺寸以及连接不同的微控制器都受到支持。

一旦你拿到了你的 PCB，[Ben]会带你在 ZMK 用 Github Actions 完成背面插座的焊接和电路板固件的安装。这是一个巧妙的指南，结尾是一个漂亮的产品。休息后的视频。
