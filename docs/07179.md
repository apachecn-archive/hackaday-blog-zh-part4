# FreeCAD 与 SolveSpace

> 原文:[https://hackaday.com/2020/07/16/freecad-vs-solvespace/](https://hackaday.com/2020/07/16/freecad-vs-solvespace/)

当你准备设计真实的东西时，你会发现简单的 CAD 程序会有很大的局限性。严肃的现代设计倾向于使用参数化建模，在这种情况下，您不必设置所有东西的尺寸和位置，而是通过描述不同元素之间的关系来约束设计。例如，您可以创建一条垂直线，并将其他线约束为平行、垂直或与该线形成给定角度。有许多工具可以做到这一点，包括 FreeCAD 和 SolveSpace，这两个程序[Joko Engineeringhelp]用来创建一个复杂的压缩机叶片，它真正显示了这两个工具之间的差异和相似之处。

你可能不需要这种特殊的设计，但是在别人做一个复杂的设计时监视他们是非常有价值的。能够看到这两种工具之间的差异可能会说服你去学习其中一种，甚至可能会改变。

我们的选择？我们经常使用 FreeCAD，尽管我们喜欢 SolveSpace 处理某些事情的方式。不过，我们并没有经常使用它，FreeCAD 与我们经常使用的 OpenSCAD 很好地集成在一起。所以，总的来说，我们使用 FreeCAD，但主要是因为习惯。这两个工具都有很多很棒的特性。
