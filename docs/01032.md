# 布拉德利·高斯罗普喜欢布线，你也应该喜欢

> 原文:[https://hack aday . com/2018/10/19/bradley-gawthrop-loves-wiring-and-so-should-you/](https://hackaday.com/2018/10/19/bradley-gawthrop-loves-wiring-and-so-should-you/)

布线是我们在一个项目中不得不做的事情之一，但可能没有给予太多的考虑。这通常是构建过程中发生的最后一件事，几乎可以肯定没有任何预见。你看着你需要连接的元件，翻遍零件箱，直到你找到看起来应该合适的东西，用一滴焊料把它钉进去，如果你觉得真的很棒，可能还需要一些热熔胶。我们时不时都会为此感到内疚，但是布拉德利·高斯罗普告诉你还有更好的方法。

如果你希望[他在 2017 年 Hackaday 超级会议](https://hackaday.com/2018/01/12/bradley-gawthrop-what-you-need-to-know-about-wiring/)上的演讲包含“一个疯狂的技巧”,可以将你普通的老鼠窝变成一个配得上航天飞机的线束，抱歉让你失望了。布拉德利承认，这需要一些额外的规划和一些专门的工具，但最终结果不言自明。虽然他的演讲是任何希望掌握电子围栏神秘艺术的人的必看之作，但他在休息后与埃利奥特·威廉姆斯的谈话是一个很好的入门读物，介绍了如何以及为什么每个人都最不喜欢建造自己的硬件。

布拉德利知道我们大多数人都不喜欢他的线束，这没关系。他没有指责任何仍处于电子行业试验阶段的人，毕竟我们都是从那里开始的。但他确实认为布线受到了不良影响。有一种感觉，这太难了，或者不值得努力，这是他试图在他的演示中解决的关键问题。

你有多少次[购买预制电线和电缆](https://hackaday.com/2018/01/20/the-unnecessary-art-of-connector-crimping/)并试图将它们硬塞进你的项目？或者把电缆的一端切掉，这样你就可以把它剥开，把各个导体焊接到 PCB 上？布拉德利已经看到了这一切，他说，这往往会导致难以诊断的小鬼。你最终花费时间和精力试图追踪你的设备中的一个错误，却发现它是你试图匆忙拼接的电缆中的一个冷接点。

他应该知道:他花了十年时间为电子器官布线，这些器官由令人眼花缭乱的螺线管阵列控制。当你职业生涯中相当大的一部分时间都用来组装成千上万的定制电线和线束时，你会情不自禁地学到一些技巧；更不用说培养识别潜在失败点的敏锐眼光了。

布拉德利学到了什么？他说，像焊接和拧紧端子这样的标准在短期内足够好，但总会导致心痛。焊料很脆，曾经流行的螺丝端子早餐就吃线端。但另一方面，他说你不必为了得到好的结果而去买最贵的卷发器。他说他已经用那种你可以在 Adafruit 上买到的消费级装备做了数千次卷缩；没有必要使用 NASA 规格的工具来完成你的下一个 DIY 项目。
