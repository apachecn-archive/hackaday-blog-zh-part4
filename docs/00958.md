# 激光切割机复活揭示了灰烬下的宏伟机器

> 原文:[https://hack aday . com/2018/10/19/laser-cutter-resurrence-uncovers-a-graphic-machine-under-the-ash/](https://hackaday.com/2018/10/19/laser-cutter-resurrection-uncovers-a-magnificent-machine-beneath-the-ash/)

垃圾是相对的。当我的同事不小心在公司的激光切割机里点燃了一块烤肉时，他制造了垃圾。残骸正被运往垃圾场，但我设法保存了下来，并把它交给了我的朋友(艾米)。四个月后，她把它从垃圾中凤凰化为生命，变成一个比原来更强大的新系统。这是她的故事，在一个三部分的系列(第一部分[第一部分](https://amymakesstuff.com/2018/04/06/restoring-a-chinese-laser-cutter-after-fire-part-1/)，第二部分[第三部分](https://amymakesstuff.com/2018/07/13/restoring-a-chinese-laser-cutter-part-2/)[第三部分](https://amymakesstuff.com/2018/09/28/finishing-restoring-a-chinese-laser-cutter-part-3/))中详细讲述，带我们踏上从垃圾到成功的旅程。她甚至录下了整个过程的[视频](https://www.youtube.com/watch?v=8t0cNaydtAk)(下面也有嵌入)

拿出你的笔记，因为虽然[Amy]不惜一切代价保持这个项目的廉价，但她在与这些野兽一起工作时，不惜一切代价为任何人的成功之路列出细节。

至于起源故事，我们的故事从我上一个雇主的办公室开始。我在机械车间问我们的一个技工一个问题，这时实习生用手指着房间的一角问道:“嘿，那应该是着火了吧？”我转过身来，看到滚滚火焰从我们的预算中国激光切割机。“没有！”我说。“我们需要灭火器！”但是我们的机甲已经在上面了。不一会儿，他拿着灭火器回来了。一次喷水就把火扑灭了，但是机器上粘满了令人讨厌的粉末状碎片。原来另一个同事犯了激光切割的大罪:他打开了它，然后走开了。更好的是，它用断开的空气喷嘴切割 ABS。

这台切割机本来是去垃圾场的，但几个恶作剧之后，我设法把这堆东西转移给了(艾米)。油漆工作在外面是一场绝对的灾难，黏糊糊的 ABS 和粉末混合物在里面结块了。[艾米]挖，剥离油漆薄片，重新涂层。除了皮带之外，她还回收了机器内部的所有其他部件。她的秘密:“IPA 和钢丝绒。”在那里，她建造了自己的抽油烟机，并将整个系统安装在她自己焊接的框架上，这样她就可以将抽油烟机和切割机作为一个整体在她的木材店周围推动。这些天来，它看到了一些为她的木工项目切割夹具的里程。

也许这个项目真正特别的地方在于她用摄像机还原了它。好像建造项目还不够困难，在你工作的时候获得正确的照明和拍摄角度更是难上加难！她的车库里没有下拉式高架摄像机设置，所以每个记录的步骤都经过精心设置，以便捕捉屏幕上发生的事情。虽然 IPA 和钢丝棉可能是一个俏皮的把戏，但在这些视频结束时，你会发现真的没有任何秘密:只有一个工程师看到了项目做得好的尊严，并有耐心执行它。

在她的博客上了解一下【Amy】[，你会发现她将拾荒和工程结合在一起的真正技巧。从倒下的树枝上的尤克里里琴到 Craigslist 冒险系列中的车库木材店，一个坏掉的激光切割机在她手中找到新的生命并不奇怪。](https://amymakesstuff.com/)
