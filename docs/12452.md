# 将 EDM 的力量带入家庭商店

> 原文：<https://hackaday.com/2022/01/20/bringing-the-power-of-edm-to-the-home-shop/>

当你看到由金属制成的看起来不可能制造的东西时，很有可能它是由各种放电加工制成的。EDM 是难加工金属、高纵横比钻孔和精密表面加工的首选方法，可使配合零件几乎无间隙地滑动在一起。问题是，EDM 有点麻烦，因此还没有进入家庭商店。

【行动箱】旨在用[一台 DIY 电火花线切割机](https://www.youtube.com/watch?v=2ewSbI52ICc)改变这种状况。在线电火花加工中，细黄铜丝被用作电极，在电介质槽中慢慢腐蚀金属。金属丝是可消耗的，并且必须不断地从供应线轴穿过工件移动到收紧线轴上。下面视频中显示的大部分构建都与电线处理机制有关，这是由 3D 打印零件和一大堆滚子和轴承制成的原型。保持电线的适当张力至关重要，因此传动系统安装了伺服控制的制动器，传动系统本身由闭环步进器驱动。张力由一对应变仪和 Arduinos 测量，它们控制轴制动伺服的位置和收线盘上电机的速度。

不幸的是，在测试中，这种设置证明辜负了 EDM 的挑剔声誉。一旦切割开始，黄铜丝就不断断裂，而[动作盒]从未进行任何实际切割。当然有希望，但是，我们期待着发展。关于电火花加工理论的更多信息，请查看[【Ben Krasnow】关于电火花钻孔的观点](https://hackaday.com/2019/03/14/ben-krasnow-drills-really-small-holes-with-electricity/)。
