# 以自制电子管为特色的自制电子管放大器

> 原文：<https://hackaday.com/2022/10/27/a-homemade-tube-amplifier-featuring-homemade-tubes/>

如今，市场上有大量廉价且高度集成的音频放大器模块，需要特别的奉献精神来制作自己的器件。尤其是当那些部件包括真空管时，尤其是当你从头开始制造真空管时。

现在，我们明白了——一些读者会发现很难花一个小时观看[jdflyback]制作一对三极管来构建他的放大器。但说真的，你得看看这个。用所有合适的设备——吹玻璃工的车床、各种氧燃料火炬、所有右手工具——制造真空管已经够难了。但当你的车床是一个无绳电钻，而你使用的点焊机看起来像是由垃圾拼凑而成的时候，你的制管游戏就变得困难多了。鉴于所有这些，你可能会认为电子管看起来比实际情况粗糙，但即使有普通钨丝加热器和粗铜线制成的栅极，它们实际上工作得很好。当然，加热器像灯泡一样明亮，但这也是魅力的一部分。

说到魅力，我们只是喜欢这些电子管放入的放大器。建于 20 世纪 20 年代的试验板风格，具有一些漂亮的老式云母电容和线绕电阻，以及一个我们从未见过的可变电阻。对现代的一个认可是门铃变压器的巧妙使用，一个用于扼流，一个用于扬声器变压器。它们听起来并不伟大，但毫无疑问，他们的工作。

我们之前可能已经看到过其他自制的真空管，我们甚至最近展示了 T2 的一个 DIY X 射线管，但是有一些关于 jdflyback 的电子管真正让我们前进。
