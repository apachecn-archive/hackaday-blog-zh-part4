# 黄铜和镍在这个磁致伸缩耳机中一起工作

> 原文：<https://hackaday.com/2020/05/21/brass-and-nickel-work-together-in-this-magnetostrictive-earphone/>

当你使用像[Simplifier]这样的名字时，你已经对你的项目做了一个任务声明:你将把复杂的过程浓缩到它们的本质。因此，着手重建这款他自己也承认“已经简化和优化”的设备，似乎有点跑题。但事实证明，[磁致伸缩原理可以让低微的扬声器变得更加简单](https://simplifier.neocities.org/earphone.html)。

我们大多数人都熟悉扬声器的操作。一个强大的磁铁位于一个线圈的中心，线圈连接到一个薄隔膜上。流经线圈的电流产生磁场，移动振膜，产生声波。另一方面，磁致伸缩是铁磁材料在磁场中改变形状的现象。为了利用这一点，[Simplifier]将一卷细铜线缠绕在一个纸模板上，镍 ~~TIG 电极~~焊接填充棒穿过该线圈。镍棒的一端被固定，另一端被固定在一个薄的黄铜圆盘上。电流通过线圈会使杆改变长度，振动圆盘发出声音。在下面的视频中听一听；这听起来很不错，我们喜欢翻橡木手机和黄铜饰品的旧时代外观。

你可能还记得[Simplifier]最近尝试的碳棒麦克风；虽然这已经足够好了，但它无法直接驱动这个耳机。如果你需要了解更多关于磁致伸缩的知识，[【本·克拉斯诺】几年前解释了它在防盗标签](https://hackaday.com/2015/12/03/how-store-anti-theft-alarms-work-magnetostriction/)中的应用。
