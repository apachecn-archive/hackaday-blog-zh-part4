# 军用滑翔机正在卷土重来，这次是以无人驾驶的形式

> 原文：<https://hackaday.com/2019/10/14/military-gliders-are-making-a-comeback-this-time-in-unmanned-form/>

孙子曰:“无序与有序之分，在于物流。”这在 2500 年前的现代世界也是如此，几个世纪以来，物流已经帮助赢得和输掉了许多战争和战斗。为此，后勤滑翔机公司正在为美国军方开发[一次性使用的无人驾驶运输滑翔机](https://logisticgliders.com/)。

让人想起二战中使用的军用滑翔机，这种滑翔机被设计成可以从各种飞机上扔下，滑翔 70 英里，并向战场上的部队运送物资。特别是为了便宜到可以在使用后丢弃，滑翔机由胶合板、一些用于加固的铝部件和注射成型的机翼面板制成。滑翔机有两个版本，都有巨大的有效载荷。LG-1K，有效载荷能力为 700 磅/320 千克，以及更大的 LG-2K，有效载荷能力为 1，600 磅/725 千克。机翼在运输过程中平行于机身折叠，然后在气体弹簧的帮助下释放后打开。滑翔机既可以在开阔区域腹部着陆，也可以在低空从尾部展开降落伞，在可压扁的机头着陆。

像这样的滑翔机可以用来在自然灾害后运送物资，或者运送到公路运输困难或不可能的偏远地区，同时减少传统飞机所需的飞行时间。动力无人机甚至可以用于携带/牵引滑翔机到所需的释放点，然后以更轻更小的重量返回，减少所需的燃料或电池。

在卢旺达和加纳，无人驾驶飞机已经被用来运送医疗物资，并且有可能建造自己的 T2 无人驾驶滑翔机。休息之后，请观看视频，了解大男孩们的行动。
