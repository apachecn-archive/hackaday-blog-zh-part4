# 我的示波器使用火

> 原文:[https://hackaday.com/2018/12/12/my-oscilloscope-uses-fire/](https://hackaday.com/2018/12/12/my-oscilloscope-uses-fire/)

如果你想形象化声波，你就要用示波器，对吗？这在 1905 年是不可能的，所以物理学家[海因里希·鲁本斯]想出了另一种涉及火焰的方法。[Luke Guigliano]和[Will Peterson]建造了一个这样的电子管——被称为鲁本斯电子管——并且将向你展示你如何也能这样做。你可以在下面看到他们结果的视频。以防火焰示波器不足以吸引你的兴趣，他们正在用特雷门琴驱动这个东西以获得额外的书呆子分数。

这些人展示了一个短火焰和一个高火焰。结果令人惊讶，尤其是短火焰。当然，时基是管的长度，所以这限制了你的测量。该管沿长度方向有许多气体喷口，并带有声源，火焰的高度对应于管内声音产生的空气压力。

根据他们的计划，这根管子是 2 英寸长，6 英尺长。他们使用 42 号钻头在一英寸的距离上制造气体喷射孔，尽管他们提到如果再做一次，他们会做得更小，间隔更近。工作气体是丙烷，如果你想完全复制它们的构造，你需要焊接。不过，他们提到，你可能不用焊接就能造出来。总成本？大约 350 美元。

你可以将鲁本斯管的概念扩展到正方形——我们不喜欢称它为鲁本斯立方体。或者你可以把它缩小到一个[单点](https://hackaday.com/2011/11/09/single-column-rubens-tube/)。无论哪种方式，它是火，所以你要小心，但也有一定的吸引力。我们总是惊讶于人们在必要的时候是多么足智多谋。鲁本斯电子管的发明就是一个例子，尽管人们[有许多其他方式弥补没有示波器的不足。](https://hackaday.com/2018/09/11/the-pre-crt-oscilloscope/)
