# 刺客的茶壶是一个恶作剧的设计

> 原文:[https://hack aday . com/2021/12/20/the-assassigners-茶壶-is-a-mischevious-design/](https://hackaday.com/2021/12/20/the-assassins-teapot-is-a-mischevious-design/)

许多电影在谈到中毒时都使用了类似的比喻。这个有抱负的杀人犯喝下了一种饮料，这种饮料是从给他们的目标受害者的同一个容器中倒出来的，以表明这种液体是可以安全饮用的。刺客的茶壶是一种实现这种诡计的方式，允许从一个看似普通的茶壶中倒出两种不同的液体，如[史蒂夫·莫尔德]所示。(休息后的视频。)

诀窍很简单。茶壶内有两个独立的空腔，在同一个壶嘴中通过各自的路径排出。每个空腔的顶部也有一个气孔。如果给定空腔的孔被倾倒者的拇指堵住，液体将不会流动。

每个空腔都可以填充自己的液体。比如一个可以装茶叶，一个可以装毒茶。凶手在给自己倒饮料时堵住了毒穴，把茶送到了自己的杯子里。然后，在为敌人倒酒时，为茶腔开的洞被堵住，让毒药流入自己目标的杯子里。

工作很简单；如果空气不能流入茶壶的空腔来代替流出的液体，气压将完全阻止液体流动。[史蒂夫]的 2D 娱乐巧妙地展示了这个概念，让我们很容易地想象茶壶的工作方式。

它还显示了设计中的一个小缺陷，应该考虑到这一点——如果喷口设计不仔细，有时流体会从一个空腔流入另一个空腔。除此之外，还有可能会弄不清该盖哪个洞来倒毒药，所以手头留一些解毒剂是值得的。或者，也可以给你的客人倒茶，他们会喜欢的！
