# 自吸尘乐高盒子让生活更美好

> 原文：<https://hackaday.com/2018/07/27/self-vacuuming-lego-box-makes-life-better/>

《银河系漫游指南》三部曲的第四本书的最后一章提到了两个让生活变得足够愉快以避免战争的窍门:一个可以从半开的窗户中飞出的超级苍蝇，和一个儿童开关。这是其中一种类型的黑客。此外，这只是一个很棒的想法，看起来很有趣，可能适用于研讨会。

当孩子们上床睡觉后，乐高积木散落在地板上、家具上、楼梯上(哎唷)和其他地方，如果你能简单地直接用真空吸尘器把它们吸到乐高盒子里不是很好吗？这个来自[国王或随机] YouTube 频道的巧妙黑客展示了如何做到这一点。他们在盒盖的相对角上打了两个孔，一个紧密配合柔性进水软管，另一个用于 Shopvac 软管，或者普通家用真空吸尘器软管，如果你喜欢的话。一个从 flyscreen 上切下来的圆盘盖住了 Shopvac 的孔，以防吸力强到足以将砖块从盒子中拉回来并进入 Shopvac。他们还通过混合一些硅酮密封胶和玉米淀粉来制作盖子的垫圈，玉米淀粉是为了防止固化混合物保持粘性。我们当然非常喜欢他们制作的版本，它在盒子的侧面有一个窗口，可以看到砖块飞进来。看看他们的构造和下面视频中的动作。

我们想知道这还能有什么其他用途。一个用来从工作台或车库地板上吸走一堆松散的五金器具以便以后分类的容器怎么样？

真空吸尘器还能在哪里派上用场？这里有一个[真空台，用于在使用激光切割机](https://hackaday.com/2016/03/19/diy-vacuum-table-makes-lasering-even-easier/)时压住柔性材料，另一个[用于在数控机床](https://hackaday.com/2016/04/12/professional-cnc-vacuum-table-holds-workpieces-with-ease/)上固定零件。
