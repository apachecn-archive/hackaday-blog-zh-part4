# 第一人称黑客的激光瞄准模块

> 原文:[https://hack aday . com/2019/05/25/a-激光瞄准模块-第一人称-黑客/](https://hackaday.com/2019/05/25/a-laser-aiming-module-for-first-person-hacking/)

你可能已经注意到[杰里米·库克]在 YouTube 上相当多产，定期发布他最新和最伟大的作品的视频。他想在他的视频制作包中添加一个头戴式 GoPro，但发现让相机指向他实际正在看的地方比预期的要稍微复杂一些。解决办法？[go pro 的 3D 打印激光“瞄准器”](https://github.com/JeremySCook/GoProLaserAim)让他在制作视频时对准目标。

这里的想法非常简单:将一个小型激光模块放在与 GoPro 相同的底座上，这样你就会有一个方便的红点，或多或少地显示相机正在看的地方。红点相对于相机视野中心点的位置会随着距离的变化而略有不同，但对于像 GoPro 这样拍摄非常广阔区域的东西来说，这在实践中并不是一个问题。

听起来是个好主意，但是那不会在所有的视频中留下一个奇怪的红点吗？[Jeremy]已经走在你前面了，他在模块前面增加了一个小按钮开关，这样他就可以快速轻松地打开和关闭激光。这个想法是，他打开激光器，大致在他希望相机指向的地方找到点，然后再关闭。

[Jeremy]已经把单件 3D 打印模块的 STL 文件放在他的 GitHub 上，供任何可能会发现它们有用的人使用。除了打印的部分，你只需要提供一个适当大小的 3.7 伏脂电池和激光二极管本身。如果你需要找到便宜的激光器，你可能想去大盒子商店看看清仓货架。
