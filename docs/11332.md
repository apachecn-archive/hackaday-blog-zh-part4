# 钢筋混凝土:适用于任何尺寸？

> 原文：<https://hackaday.com/2021/09/26/reinforced-concrete-versatile-at-any-size/>

在我们的社区里，我们对制作东西并不陌生，我们当中有很多人致力于模型制作。然而，用和它复制的东西完全一样的技术来制作某样东西的比例模型是不常见的。相反，模型可以由卡片、泡沫、玻璃纤维或树脂制成。[微小世界]采取了相反的策略，[建造比例模型土木工程项目，就像它们本来是真实的一样](https://www.youtube.com/watch?v=S6RK9hGlsfw)。(视频，嵌在下面。)

这里，胡佛大坝旁桥的比例模型是由钢筋混凝土制成的。代替钢筋的是铁丝网，代替木制模板的是看起来像泡沫板的东西，混凝土是更光滑的砂浆，但除此之外它是真实的东西。我们看到各种桥梁部件被就地铸造，结果就像你所期望的那样坚固。

我们可以看到，这是一项模拟混凝土建筑和结构的伟大技术，但它也是一种我们认为在这个尺度上可能有其他应用的材料。例如，与 20-20 挤出、3D 打印塑料或木材相比，小规模增强水泥的刚度、强度和质量如何？不管怎样，看起来很有趣，你可以从休息下面的视频中看到。
