# 你能制造的最好的 Linux 平板电脑

> 原文：<https://hackaday.com/2019/06/06/the-finest-linux-tablet-you-can-build/>

在过去的几年里，我们都可以在一个模块上使用小型、廉价的系统。这些神奇的芯片是完整的 Linux 系统，带有 WiFi、一个还算不错的 GPU 和足够运行一个真实系统的内存。这是一个构建平板电脑的完美平台，唯一的问题是必须有人实际去做。[DLT 的那台](https://hackaday.io/project/164845-dlt-one-a-damn-linux-tablet)是【法特斯巴克教授】的“该死的 Linux 平板”。这是一个问题的答案:什么时候有人会围绕这些廉价的片上系统中的一个来建造一台平板电脑。

有许多模块可供选择，第一个任务实际上是选择这些 Linux 模块中的一个。[Fartsparkle]最终推出了 Nvidia Jetson Nano，这是一款令人印象深刻的小主板，它有一个独特的优势:它可以与 Raspberry Pi 计算模块 Raspberry Pi-on-an-SODIMM 兼容。给定一个机箱，[法特斯巴克教授]可以通过获得新版 Jetson Nano(或计算模块)来简单地升级他的平板电脑。

板的当前状态基本上是板上的 SODIMM 插座，带有用于以太网、电源插孔、USB 和 HDMI 的分线点。这就是它的全部，平板电脑机箱的设计正在进行中。

这款平板电脑的未来更新将包括 USB-C PD、允许 M.2 存储的 mPCI-E 连接器、某种形式的 LTE 以及允许你将它变成类似 Surface 的笔记本电脑的坞站。这是一个令人惊叹的建筑，也是我们在今年的 Hackaday 奖中看到的最好的作品之一。
