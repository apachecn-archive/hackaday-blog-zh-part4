# 奇怪的输入和特殊的外围设备:像他们过去制造的操纵杆

> 原文:[https://hackaday . com/2022/06/15/odd-inputs-and-special-peripherals-a-joystick-like-they-used-to-make/](https://hackaday.com/2022/06/15/odd-inputs-and-peculiar-peripherals-a-joystick-like-they-used-to-make/)

随着几代游戏控制台游戏手柄的兴起，操纵杆已经成为一种几乎被遗忘的外设，被排挤到飞行模拟器的世界中，其设计倾向于复制飞机操纵杆的设计。8 位和 16 位时代的经典操纵杆是更普通的设备，更适合太空入侵者，而不是 T2 的微软飞行模拟器，这是 T4 用 3D 打印形式重新创造的。

他提出的设计与他已经拥有的经典手杖 Zipstik 非常相似。这是一个相当简单的设备，所有触点都使用微动开关，因此非常坚固。他制作了一个 3D 打印的轴，但不相信它的强度，所以通过使用一个带有一对弹簧圈的金属轴来复制原始轴。我们记得我们的 Zipstik 有一个钢制轴；他用铝代替了它。一个方便的夹具和一把钢锯让他可以为弹簧圈制造凹槽，从而制造出一个坚固的 ZipStik 克隆体，应该可以满足任何复古游戏玩家的需求。

该棒为 Amiga 连线，包括一个基于 555 的快速发射电路，但这不是电子产品的终点，因为他还为 Amiga 操纵杆创建了一个 USB 接口。不是每个人都有经典机，所以现在每个人都可以享受复古外设体验！这两个版本都可以在休息时的视频中看到。

这不是我们带给你的第一个 Amiga 操纵杆，但[它比以前的一些设计](https://hackaday.com/2016/04/01/a-keypad-joypad-for-your-retro-gaming/)更复杂。
