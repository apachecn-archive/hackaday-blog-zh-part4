# 太阳能电池织物让任何东西都是太阳能的

> 原文:[https://hack aday . com/2022/12/18/solar-cell-fabric-makes-any-solar/](https://hackaday.com/2022/12/18/solar-cell-fabric-makes-anything-solar/)

麻省理工学院一直在研究由几微米厚的薄膜制成的超薄太阳能电池。问题？细胞很薄，很难处理。你可以在玻璃片上制作一个小型太阳能电池，但这并不有趣，因为你可以使用传统技术制作出像玻璃一样易碎的完美太阳能电池。但是在一篇新论文中，麻省理工学院的研究人员描述了创造 50 微米薄的织物，可以利用太阳能发电。

这个过程仍然包括使用化学气相沉积在玻璃上生产太阳能电池。然而，细胞被从玻璃上取下，准备好电极，然后转移到一块充当新基底的织物上。

本文中使用的织物是一种复合织物，称为 Dyneema 复合织物。它使用超高分子量聚乙烯纤维和聚酯薄膜片。这种材料重量轻，但强度很高。紫外线固化粘合剂将织物和太阳能电池粘合在一起。

老实说，我们怀疑任何人会很快在他们的车库里制造这些东西。但我们很想看看你能用这卷布料做些什么。可穿戴设备、自充电笔记本电脑包或空中无人机中的太阳能仪器都可以利用这种材料的灵活性和低重量。
