# 双摆使用定制滑环

> 原文:[https://hack aday . com/2019/11/15/双摆-用途-定制-滑环/](https://hackaday.com/2019/11/15/double-pendulum-uses-custom-slip-rings/)

当需要通过旋转机构传递动力时，旋转机构可能会令人头疼。[Igor Brkic]在创建他的双钟摆建筑时就面临着这样的挑战，并通过一点 DIY 解决了这个问题。

该项目被称为 KLAATNO，灵感来自克罗地亚语钟摆的单词 klatno。这是一个机械安装件，由一个助力摆组成，另一个摆安装在摆臂的末端。24 伏齿轮马达用于驱动该组件。它由 Arduino Pro Mini 控制，Arduino Pro Mini 测量电机端子的反电动势，以确定电机运动的速度和方向。

为了使装置更具视觉冲击力，EL 线被安装在双摆的摆臂上。这需要将动力传递给旋转组件，这是通过使用定制的滑环来实现的。铜片与来自吉他弦的柔性金属线结合使用。这并不像[伊戈尔]希望的那样低摩擦，但它完成了工作。

这是一个有趣的装置，在任何大学工程建筑的公共区域都非常适合。当然，我们最喜欢的钟摆是攻城武器。休息后的视频。
