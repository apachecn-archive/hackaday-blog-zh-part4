# Alexa 控制着这个野蛮的南瓜

> 原文:[https://hack aday . com/2020/01/18/Alexa-controls-this-savage-pumpkin/](https://hackaday.com/2020/01/18/alexa-controls-this-savage-pumpkin/)

传统上，在万圣节期间雕刻南瓜来代表邪恶和可怕的野兽。与此相对的是[迷你汉娜]对亚当·沙维奇的雕刻，她称之为英雄南瓜。这是对传统的一个有趣的改变，当然，它还打包了 WiFi。

建造始于典型的橙色冬季南瓜品种的雕刻，使用了从萨维奇先生的传记封面抄袭的艺术品。在里面，有一束 led，全部由一个 adafruit feather M0 控制，它通过 WiFi 与更广泛的互联网对话。南瓜可以由 Alexa 控制，这要归功于 Adafruit 的组合。IO 和 IFTTT。

这是一个有趣的物联网建筑，为现代智能家居做好了准备，在现代智能家居中，你已经习惯了大喊关灯。我们希望看到一个类似的比利·考根建设，如果只是双关语。如果你试一试，一定要给我们写信。休息后的视频。
