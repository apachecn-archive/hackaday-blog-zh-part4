# Hackaday 播客 093:快速热树莓 Pi、苏斯博士无人机、M&M 质量计和 FPGA 磁带备份

> 原文:[https://hackaday . com/2020/11/13/hackaday-podcast-093-hot-and-fast-raspberry-pi-dr-Seuss-drone-mm-mass-meter-and-FPGA-tape-backup/](https://hackaday.com/2020/11/13/hackaday-podcast-093-hot-and-fast-raspberry-pi-dr-seuss-drone-mm-mass-meter-and-fpga-tape-backup/)

Hackaday 的编辑迈克·什奇斯(Mike Szczys)和埃利奥特·威廉姆斯就本周出现在我们屏幕上的史诗般的黑客攻击展开了争论。埃利奥特本周深入研究了树莓 Pi 4 的所有三种口味的超频，发现散热片占据了主导地位。Mike 展示了他对糖衣巧克力的热爱，同时对一台可以检测花生 M&M 豆何时丢失的机器垂涎三尺。当 led 开始播放时，核心内存变得有趣得多，一个小轮子是一架非常奇怪的多旋翼无人机的省电秘密，音频磁带数据传输的价值比你想象的更大——让 FPGA 向你展示这是如何做到的。
