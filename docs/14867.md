# 3D 打印一个带有烟雾效果的小型蒸汽火车

> 原文：<https://hackaday.com/2022/09/21/3d-print-yourself-a-tiny-steam-train-complete-with-smoke-effects/>

模型火车很有趣，但有时电动模型中呼呼作响的小马达与真实铁路中笨重的金属巨兽相去甚远。[DIY Machines]的[Lewis]用这个小蒸汽火车模型添加了一些味道，包括烟雾效果！

火车本身的车身是用 PLA 3D 打印的。它是专为 O-gauge 规模，并配有完整的三维打印轨道模型。这些部分被涂上一层油漆，以更好地接近真实事物的表面；毕竟，有时裸露的塑料是不够的。

推进是由于一个板载电池和一个简单的齿轮电机，由 HG7881 电机驱动器驱动。一个 ESP32-CAM 负责运行该节目，允许火车被无线指挥。作为奖励，摄像机安装在列车的最前端，让人们可以观看列车在轨道上前进的实时画面。与此同时，烟雾效果要归功于安装在火车烟囱里的一个小型水雾化器，这使得火车看起来更真实一些。

自供电火车和 3D 打印轨道的结合是一个引人注目的组合。[Lewis]已经能够把他的 PETG 3D 打印轨道放在外面两年多了，它仍然工作正常。当使用金属轨来传输功率时，这并不容易实现。

总的来说，这是一种制作自己的火车模型的有趣方式，而且比简单地从商店里购买预制模型要实际得多。从那里开始，天空真的是你创造力的极限！休息后的视频。
