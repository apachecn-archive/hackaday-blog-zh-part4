# 在旧诺基亚手机中寻找数字慰藉

> 原文：<https://hackaday.com/2022/09/20/finding-digital-solace-in-an-old-nokia-phone/>

我们不必告诉你，对于重视隐私、安全和开源的人来说，当前的手机市场有点暗淡。虽然有一些值得注意的尝试来改变这一切，从流行的 Linux 发行版的手机优化版本到模块化手机的承诺，但我们仍然发现自己被少数公司发布的基本相同的平板电脑所困扰，这些公司似乎经常将客户视为一种产品。

与其坐等可能永远不会到来的技术救援，[【vrhelmutt】已经决定通过回顾过去把事情掌握在自己手中](https://hackaday.io/project/186887-nokia-asha-210-hackfone)。具体来说，通过拥抱相对不常见的诺基亚阿莎 210。这款于 2013 年发布的所谓“功能手机”提供了完整的 QWERTY 键盘、诺基亚的 Series 40 操作系统、WiFi、蓝牙和可拆卸的 BL-4U 电池。不幸的是，随着 2G 蜂窝网络迅速关闭，它不太可能长时间收到信号(如果有的话，取决于你住在哪里)。

[![](img/f5537a00526986392cc80cbad85e56d5.png)](https://hackaday.com/wp-content/uploads/2022/09/asha210_detail.jpg) 那么为什么要在 2022 年使用一些怪异的老式诺基亚手机呢？[vrhelmutt]认为，从游戏到 SSH 客户端，整个 S40 软件世界仍然可以使用。用 Java 开发自己的 S40 应用程序也相对容易，原始的软件开发工具包仍然可以在网上免费获得。结合坚实的(如果相当过时的)硬件，这使得诺基亚 Asha 210 成为一个令人惊讶的口袋黑客平台的选择。

无论你是在寻找一款能让你坐在沙发上通过 IRC 聊天的廉价设备，还是想编写自己的定制软件来控制你的家庭自动化或机器人项目，你都可能想去二手市场看看诺基亚 Asha 210。或者，如果你渴望立即体验，那么[vrhelmutt]实际上正在销售这些预装了大量游戏和程序的手机。不要认为这是官方认可；坦率地说，我们对重新分发所有这些软件的合法性并不太有信心，但至少这是那些寻求摆脱现代智能手机惊险刺激的人的一种选择。

如果你正在寻找与今天的移动超级计算机更遥远的东西，也许我们可以让你对[旋转式非智能手机](https://hackaday.com/2022/09/10/the-open-source-rotary-cell-phone-two-years-later/)感兴趣。
