# 寻找，你将命令

> 原文：<https://hackaday.com/2020/10/02/seek-and-ye-shall-command/>

如果算上生活中所有的屏幕，那就需要热一分钟。其中一些是触摸屏，一些需要鼠标或键盘，但我们习惯了所有的输入设备。不是每个人都可以使用各种方法，就像依赖眼球追踪硬件的脑瘫患者一样。传统上，这只能在连接的计算机上工作，所以从椅子上安装的屏幕切换到桌子上的平板电脑不是一个选项。为了让人们能够毫不费力地控制不同的电脑，[扎克·弗里德曼]正在开发一种不依赖于一台电脑的头戴式眼球跟踪器。在某种程度上，这就像一个 KVM 开关，但更具未来感。[托尼·斯塔克]会感到骄傲的。

耳机上的红外探测器可以识别视线范围内的兼容屏幕，并与相关的 HID 加密狗同步。安装在耳机上的彩色摄像头跟踪头部相对于屏幕的位置，而红外摄像头扫描眼睛，以计算用户聚焦的位置。这里的所有技术都是经过验证的，但这个新配方可能会改变任何一个对传统键盘、鼠标和触摸屏有困难的人的游戏规则。也许 [QR 码](https://hackaday.com/2018/06/27/improving-indoor-navigation-of-robots-with-ir/)可以帮助屏幕识别和定位，就像 [Wii 遥控器和感应条如何一起工作](https://hackaday.com/2010/09/16/wii-sensor-bar-projector/)。
