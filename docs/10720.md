# 累了为什么还要煮咖啡？让机器人替你做吧

> 原文:[https://hack aday . com/2021/07/25/why-make-coffee-when-you-stay-let-a-robot-do-it-for-you/](https://hackaday.com/2021/07/25/why-make-coffee-when-youre-tired-let-a-robot-do-it-for-you/)

像我们一样，[阿尔贝托]不会妥协，当谈到一杯好咖啡。我们认为，如果他在以前去办公室，他是那种自带咖啡设备的同事，以免被办公室的咖啡冲泡。或者他自愿订购办公用品，因此可以为其他人做决定。是的，这绝对是一种方法。

[![](../Images/22e5f96d3929c31a5c2c700c6b3331da.png)](https://hackaday.com/wp-content/uploads/2021/07/cafeino-inner.jpeg) 但是像我们许多人一样，他现在在一个家庭办公室里工作。即便如此，比起每天早上站在那里倒一杯完美的咖啡，他还有更重要的事情要做。看，这就是我们的不同之处，[阿尔贝托]。但是我们确实喜欢 [Cafeino，你的自动倒酒机](https://hackaday.io/project/174630-cafeino-a-pour-over-coffee-robot)。它是如此光滑可爱，我们确信它比我们手工做的要好得多——尽管我们喜欢自己倒酒。

Cafeino 的设计是为了模仿训练有素的咖啡师的手的动作，因为显然你应该慢慢地、有意识地将水倒入漩涡中，以均匀地覆盖咖啡渣。(我们的水壶有一个厚实的壶嘴，所以我们只是稍微用了一下。)Cafeino 是通过从电热水壶中抽水，并在两个伺服系统的帮助下将细流注入圆圈中来实现这一点的。

这三个按钮分别代表一种不同的配方设置，它指定了水量、手动倒水模式以及开花和实际倒水之间的休息时间。这些食谱是通过 ESP32 使用附带的网络应用程序设置的，尽管主要的大脑咖啡师是 Arduino Nano。休息之后，拿起杯子，看看演示。

有一个古老但现代的咖啡机器人躺在周围？你可以把它变成一个自动浇水的花盆。
