# 行星逃生(盒子里的房间)

> 原文：<https://hackaday.com/2021/03/19/planetary-escape-room-in-a-box/>

一个有趣的密室的诀窍是分层。对于[doktorinjh]的*太空箱*，你从一个[神秘的铝制公文包](https://imgur.com/a/JrivmEm)和一个美国宇航局的拉绳背包开始。一位游戏大师朗读介绍性的演讲来营造气氛，你就可以开始逃离这个星球了。第一层是装有谜题的背包，你需要解开谜题才能进入公文包。在那里，你会发现一个隐藏的隔间和足够多的科幻参考资料，让我们脸上露出傻乎乎的笑容。我们喜欢看到工具被重复使用，就像在早期的拼图中一样，你使用紫外线 LED 来揭示隐藏的信息，但这种光也照亮了后来的拼图线索。

所有的技术都让它成为混合媒体的奇迹。物理层具有激光雕刻的木头，具有 1975 年 NASA 标志的字体、按钮、旋钮、发光二极管、拨动开关和伺服系统。在可见的面板下面是一个 RGB 传感器，音频播放器，扬声器，在中心是一个 Arduino MEGA。我们很想得到 Spacecase 的游戏，我们受到启发，全力以赴，用我们的个人风格打造游戏。也许是带捕鼠器的东西。

这不是我们看到的第一个[逃生室硬件](https://hackaday.com/2018/05/13/held-captive-by-arduino-and-multiple-rfid-readers/)，【doktorinjh】同样制作了一个[炸弹~~扩散~~拆除游戏](https://hackaday.com/2019/11/27/in-case-you-cannot-make-it-to-an-escape-room/)。
