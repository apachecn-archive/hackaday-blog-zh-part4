# 在所有陌生的地方寻找低保真

> 原文：<https://hackaday.com/2022/02/23/finding-lo-fi-in-all-the-strange-places/>

如果你还没有听过任何 lo-fi 音乐，它代表低保真度音乐。今天的 Lofi 音乐的特点是音频不完美，如电缆噪音或磁带嘶嘶声。为了获得令人愉快的温暖的不完美的声音，许多艺术家转向老式设备。[HAINBACH]发现了一种极好的工具，即被称为磁卡录音机的过时课堂工具。

该设备的基本机制是，它通过它读取和写入四分之一英寸磁带上的两个轨道。一条是给老师的，一条是给学生的。最初是为了帮助语言学习者而设计的，我们可以理解为什么它会是一个好的低保真样本的理想来源。麦克风和扬声器需要高质量，才能听到例句的细微差别。[HAINBACH]也赞赏这种设备的总体音调和音色，而不仅仅是使用盒式录音机。

有问题的胶带粘在小塑料卡片上。通过一些修改，你可以将卡片倒放，形成一个循环，或者将部分缝合在一起。对于多台机器，您可以将卡从一台机器直接运行到另一台机器。它们是由几家公司制作的，在网上拍卖行可以找到相对便宜的。虽然我们不能相信是[HAINBACH]想出了电影《小司机》中的这个主意，但它仍然是一个令人敬畏的例子。

磁带一直是乐器的魅力所在。这个 Crudman 是对 1963 年的更古老的 Mellon 的现代诠释，是一个很好的例子。
