# [Leo]修理 MIDI 音序器

> 原文：<https://hackaday.com/2019/01/23/leo-repairs-a-midi-sequencer/>

我们都有这样的朋友，他给我们带来了他们可悲的坏掉的电子产品。在 Leo 的例子中，他从一个音乐家朋友那里得到了一个 MIDI 音序器。它有一个死显示，制造商建议，驱动 IC 可能是坏的，甚至发送一个替换表面贴装部件。

[利奥]不相信，虽然。他知道人们总是按安装在电路板上的开关，他推测这可能只是一个不良的焊点。正如你在下面的视频中看到的，这并没有被证明。

下一步是点燃热风枪。他不想移除芯片，而是想让焊料回流。他有点担心融化 7 段 led，所以他建立了一个小箔盾来保护它。这也没有让事情运转起来。

使用相同的热风枪，他删除了集成电路，然后使用接触铁清理垫。我们不知道他是否把灯芯浸在助焊剂里。这似乎总是有助于清洁灯芯吸收焊料更好。对于焊接工作，他使用较小的接触铁和非常薄的焊料。当然，用热风，他也可以用浆糊。

更换的芯片看起来不错。成功了吗？看视频你就知道了。如果你喜欢这种修复色情，我们已经涵盖了很多有趣的修复。例如，检查[大规模电源维修](https://hackaday.com/2018/11/30/1000-watt-power-supply-tear-down-and-repair/)。或者是在[一台电脑在电池堆里洗澡之后的清理工作](https://hackaday.com/2018/07/30/amiga-repairs-put-one-tough-little-machine-back-in-service/)。
