# 一个隐藏的模型铁路应运而生

> 原文：<https://hackaday.com/2022/12/13/a-concealed-model-railway-rises-to-the-occasion/>

偶尔摆放的家具除了填充空间和展示一两件珍贵的装饰品之外，几乎没有其他用途。谁没有一个亲戚拥有一两张小桌子或展示架，把可用空间塞得满满当当呢！

但是，如果你去参观[彼得·瓦尔德拉夫]的房子，那些家具可能并不像它们看起来那样。[例如，他的展示基座隐藏了一个 N 轨距铁路布局](https://www.youtube.com/watch?v=UCEZvondDRM),它通过滑轮系统从深处升起，通过移动展示在其顶部的花瓶来触发动作。花瓶里藏着一块磁铁，磁铁控制一个簧片开关，进而控制绞盘马达。

布局是一个松散的*蝙蝠侠*主题火车追逐，同心螺旋轨道形成一个连续的环路，两列火车在上面运行。有一个巧妙的安排，一个簧片开关和一块死轨道，以确保追赶的火车总是保持在它们之间的间隙。景观是一组悬崖，顶部是韦恩庄园的模型，甚至还有一辆 LED 照明的蝙蝠战车。其中一辆机车显然是基于《T2》书中的一个角色。

总的来说，我们喜欢这个布局的独创性，但是如果你也喜欢它，那么我们为你准备了一份礼物。眼尖的读者会记得[这不是 Peter](https://hackaday.com/2022/04/11/this-end-table-conceals-a-close-encounter/) 的第一个这样的项目。
