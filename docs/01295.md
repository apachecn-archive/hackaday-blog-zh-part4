# 失传的蒸汽加热艺术

> 原文：<https://hackaday.com/2018/11/18/the-lost-art-of-steam-heating/>

[packrat]让我们看了[Dan holo Han]2015 年关于蒸汽加热系统的历史和艺术的报告。在[整栋建筑的中央供暖系统](https://www.energy.gov/energysaver/home-heating-systems/heat-distribution-systems)出现时，蒸汽被用来代替水或空气作为输送介质。这些系统被安装在包括帝国大厦在内的标志性建筑中，直到今天仍在使用。

基于蒸汽的加热系统的一个主要优点是不需要泵:蒸汽会通过管道自然上升，冷凝并返回源头。这可以实施为单管，其中冷凝物通过与蒸汽相同的管返回，或者双管系统，其中冷凝物通过其自己的管返回。

在演示中，Dan 向我们介绍了他在美国主要建筑中使用这些蒸汽供暖系统的经历、系统的类型、去世很久的工程师实施的修复措施以及维护这些系统的特点。
