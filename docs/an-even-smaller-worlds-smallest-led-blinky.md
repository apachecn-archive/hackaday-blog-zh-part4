# 更小的世界上最小的 LED Blinky

> 原文：<https://hackaday.com/2019/04/11/an-even-smaller-worlds-smallest-led-blinky/>

用 555 什么都能做。众所周知，这是一条普遍规律。你可能会想，闪光灯在这之前已经出现过很多次了。但是还没有人创造出像[ TWires ]创造的那种小的 555 闪光灯，他用 TI LMC555 芯片级封装的 555 和[使用 01005 分立元件在它的表面安装了一个工作闪光灯](https://hackaday.io/project/164885-pico555)。有一个视频展示了它的运行，我们把它放在了断点下面，它是*微型*。我们可能都认为自己很擅长焊接，但这完全是另一个级别的工作。

该项目的灵感来自于[迈克·哈里森]之前的最小 blinky 奖获得者，他在一个微型表面贴装超级电容器上使用了[PIC 微控制器。它使用相同的电容器供电，但我们会说它将 blinky 带到了新的微小水平。这是否意味着一场新的军备竞赛正在我们这个弹丸之地展开？我们希望如此，虽然很难想象它们会变得更小，但我们迫不及待地想看看人们会想出什么。如果有一件关于我们社区的事情，那就是说某事不能做是不明智的:如果有可能，你们中的一个会找到方法。即使是](https://hackaday.com/2018/09/23/worlds-smallest-led-blinky/)[微芯片的 MIC1555](http://ww1.microchip.com/downloads/en/DeviceDoc/20005730A.pdf) 也可能有点大，所以需要一些创新。

想要了解 555 艺术的迷人现状，请阅读[这篇来自我们自己的【Ted Yapo】的文章](https://hackaday.com/2019/03/25/the-worlds-fastest-555-timer-and-the-state-of-the-555/)。
