# 物联网保险箱锁定儿童手机

> 原文:[https://hack aday . com/2019/09/21/IOT-safe-keys-latch key-kids-phones-on-lockdown/](https://hackaday.com/2019/09/21/iot-safe-keeps-latchkey-kids-phones-on-lockdown/)

手机很棒。在大部分情况下，当用作电话时，它们可以将我们从糟糕的情况中解救出来，并让我们在自由漫游的同时进行通信。作为电脑使用，它们经常成为吞噬时间的黑洞，扭曲我们对自我和现实的感觉。假设你打电话时他们会接，手机对孩子来说是个好东西，尤其是现在你几乎找不到付费电话。但是，你如何教孩子们负责任地使用它们，这样他们仍然可以成为功能正常的成年人，并在某一天搬出去？[Jaychouu]认为答案在一个专门的锁箱里。

这个外表光滑的盒子内部有一个电磁锁，可以通过键盘解锁，或通过 OBLOQ 物联网模块远程解锁。[Jaychouu]增加了一些功能，把它赶出 Arduino 锁箱领域。为了防止挂钥匙的孩子欺骗系统，将石头(或什么都不放)放入盒子中，有一个数字重量传感器和一个超声波传感器来验证内容的凭证，并将它们与已知值进行比较。

想要一个基本的锁箱，让你的手机在你工作的时候够不着？这里有一个带倒计时器的。
