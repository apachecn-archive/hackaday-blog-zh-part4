# 用吹制电弧等离子体给你的刀片增加一些刀刃

> 原文：<https://hackaday.com/2018/10/01/add-some-edge-to-your-blades-with-blown-arc-plasma/>

如果你让科幻迷调查他们最渴望电影中描绘的哪项技术，很有可能《星球大战》系列的光剑会在列表中名列前茅。有那么多的权力在你的手中，但仍然需要近距离和个人的斗争。此外，能够融化舱壁上的洞是相当敏锐的，因为是很酷的声音。

可悲的是，我们能够在刀片状的场中塑造和控制等离子体的那一天可能还很遥远，但这并没有阻止[艾伦·潘]尝试下一个最好的东西:[一个手持等离子体投射刀片](https://www.youtube.com/watch?v=8d60sR6mugg)。他从基本的雅各布之梯开始。在之前，我们已经见过[很多这样的东西，但是基本的想法是电离两个平行的垂直导体之间的空气；热等离子体加热空气，使其上升，直到到达顶部并熄灭，在底部重新开始这个过程。他的想法是用叶片鼓风机的空气迫使等离子体进入电极之间的薄片，形成吹弧等离子体。这本身看起来很酷，但他也沿着锋利的刨刀刀片拉伸电极，以避免额外的危险。我们不得不承认，这东西看起来相当吓人，即使等离子体并不真正包含舱壁熔化热能。看看下面视频中的结果。](https://hackaday.com/2017/06/09/portable-jacobs-ladder-for-when-you-know-you-need-a-portable-jacobs-ladder/)

我们很乐意看到[艾伦]兑现他的承诺，用电动管道风扇或迷你喷气发动机使整个系统自给自足。尽管如此，它仍然很整洁。这并不是他的第一次光剑竞技，但至少这次不需要丁烷。
