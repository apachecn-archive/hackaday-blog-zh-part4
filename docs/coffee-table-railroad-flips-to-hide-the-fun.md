# 咖啡桌轨道翻转隐藏的乐趣

> 原文：<https://hackaday.com/2021/04/10/coffee-table-railroad-flips-to-hide-the-fun/>

客厅的咖啡桌一直是模拟铁路的最爱。但是当你真的想喝咖啡的时候该怎么办呢？[Peter Waldraff]想出了一个最有说服力的答案，他设计了一个咖啡桌模型铁路，能够把世界颠倒过来。

这不是[彼得]的第一次牛仔竞技表演。在下面的演示视频中，他展示了 20 年前用玻璃下的矩形布局制作的咖啡桌火车。这一次，圆形的设计意味着一个球形体积可以围绕两个滑板轴承枢轴点旋转，露出一边的山区场景和另一边无聊的旧木桌。但是重力反向时，N 尺度列车本身会发生什么？有一个绝妙的解决方案！

咖啡桌的框架包括用于火车存放的外环。在颠倒模型之前，列车本身被送到这个侧线进行安全保管。在[一个早期的制作视频](https://www.youtube.com/watch?v=24vvFtsCFRM)中，我们可以看到使用螺线管并由桌子中央的磁铁驱动的闩锁机制。当弹簧加载的臂接触到间隙另一侧的一些螺钉头时，肘节螺栓(有时称为蝶形锚，用于将东西固定在干墙上)的巧妙使用使它们将动力转移到存储轨道的外环。电力的来源是一个可充电的牧田电动工具电池，在山内的一个隐藏的房间里。

当然，我们也见过其他隐秘的茶几火车，比如[这个可爱的手工雕刻版本](https://hackaday.com/2017/12/22/coffee-table-model-railroad-with-all-the-bells-and-whistles-and-lights-and-sirens/)。但你不得不佩服[彼得]是如何设法将所有东西都整合到一个独立的单元中，而不需要存放一个可拆卸的盖子。如果你是一个想总是炫耀你的手工作品的人，[这就是有机玻璃盒子咖啡桌设计发挥作用的地方](https://hackaday.com/2019/06/22/coffee-tables-computers-and-railways/)。
