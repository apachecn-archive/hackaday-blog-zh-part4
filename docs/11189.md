# 用这个磁场可视化工具看不见的东西

> 原文:[https://hack aday . com/2021/09/01/用这个磁场可视化器看不见的东西/](https://hackaday.com/2021/09/01/see-the-unseen-with-this-magnetic-field-visualizer/)

普通的黑客读者可能知道，至少在学术意义上，磁场是什么样子的。但是由于我们头骨中的胶状球体只能感知一小部分电磁波谱，我们不得不相信教科书上的图表。也就是说，除非你有一个由[斯通博士] 开发的漂亮的磁场可视化工具。

使用 XMC1100 微控制器开发板和 TLV49 3D 磁传感器，该设备能够实时跟踪磁铁的磁极，并在其电子纸显示器上产生近似的磁力线。相对磁场强度由可视化的大小指示，这允许用户容易地比较多个磁体。顺便提一句，[斯通博士]指出，当前版本的硬件和软件一次只能处理一个磁体；可视化复杂的磁场和两个以上的磁极需要一系列传感器，可能还需要一个更强大的处理器。

你*需要*来可视化磁铁周围的磁力线吗？也许不是。但是能够快速了解磁体的强度并确定其磁极的位置肯定会派上用场。我们希望看到[斯通博士]将项目推进到下一阶段，将它变成一个方便工作台使用的手持设备。这将比[我们之前看到的一些可视化磁场](https://hackaday.com/2019/02/25/visual-magnetic-fields/)的方法要简单得多，尽管如果你只是担心磁场强度，[有更直接的方法来显示它](https://hackaday.com/2021/04/04/a-magnetic-field-strength-meter-using-an-arduino/)。
