# 树莓派创造旋律

> 原文：<https://hackaday.com/2022/03/28/raspberry-pi-creates-melody/>

对于那些不喜欢 70 年代摇滚音乐或 40 年代广播节目的人来说，特雷门琴可能是一种陌生的乐器。作为一个纯粹的电子设备，它远远超出了传统乐器的范畴。两个无线电天线检测音乐家的手的位置，以发出传统上与怪诞或科幻小说相关的独特声音。

通常使用一组滤波器和放大器来构建这种乐器[，但是这种构建用树莓 Pi Zero 2](https://www.youtube.com/watch?v=o5xgLNlJiYk) 代替了几乎所有的东西，并且使用了一组两个 HC-SR04 距离传感器来代替无线电天线来检测音乐家的手的位置。借助 Pi 提供的处理能力，现代化的乐器也能够输出 MIDI，这使得该乐器能够轻松地与 GarageBand 等程序或任何其他支持 MIDI 的软件进行交互。

项目构建分为两个视频，第二个视频链接如下。项目代码也可以在项目的 GitHub 页面上获得，因此任何人只要有 Pi 和其他可用的设备，就可以很容易地开始尝试这种深奥且经常被忽视的乐器。[它已经有 100 多年的历史了](https://hackaday.com/2020/10/30/the-theremin-is-100-years-old-celebrating-the-spookiest-of-instruments/)，它的分支(包括这个版本)就像它们能发出的声音一样多种多样。
