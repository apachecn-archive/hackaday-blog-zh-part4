# WiFi 网络成了这个口袋游戏的目标

> 原文:[https://hack aday . com/2020/06/20/wifi-networks-turned-targets-in-this-pocket-game/](https://hackaday.com/2020/06/20/wifi-networks-turned-targets-in-this-pocket-game/)

为了让他的徒步旅行变得更具互动性，[Roni Bandini]想出了一个有趣的方法来将新 WiFi 网络的发现游戏化。使用 Heltec WiFi Kit 8，它集成了有机发光二极管屏幕和 ESP8266，这种口袋大小的设备可以拾取无线网络，并使用它们的信号强度和加密类型作为游戏的元素。

[![](../Images/a09f56aaf7e826a2c8cd0e2059b8b330.png)](https://hackaday.com/wp-content/uploads/2020/06/routerattack_detail.jpg) 选择要对战的网络后，屏幕上会出现一个目标。目标和玩家之间的距离是由信号强度决定的，目标能够受到的伤害与它的加密强度相关。正如你在休息后的视频中看到的，游戏有点让人想起*焦土*，玩家需要调整他们的火炮角度来击中远处的目标。

Heltec 板附着在一个 3D 打印的前面板上，它可以整齐地放入 Altoids 罐中。控制器由一个按钮和一个电位计组成，加上一个从旧手机中回收的电池组，这个小设备随时可以在你漫游的任何地方进行战斗。

虽然目前这只是一个有趣的消遣，但[Roni]表示，实际上，将网络记录到一个文件并生成一些关于其强度和加密类型的统计数据并不需要太多时间。如果便携式 WiFi 扫描伴侣的想法似乎很有趣，[你绝对应该看看 Pwnagotchi 项目](https://hackaday.com/2019/10/16/a-tamagotchi-for-wifi-cracking/)。
