# 这张茶几隐藏了一次亲密接触

> 原文：<https://hackaday.com/2022/04/11/this-end-table-conceals-a-close-encounter/>

如果你到了一定的年龄，也许你小时候就有一列火车。一个椭圆形的轨道，一个机车，和一些车辆；不管是坦克引擎托马斯还是大型特快列车，它们在当时都是非常酷的玩具。向前推进几十年，模型铁路要么已经成为超昂贵的房间填充布局，要么已经成为新奇的圣诞装饰品，以至于轨道的基本回路急需拯救。也许【彼得·瓦尔德拉夫】可以帮忙，用[一个隐藏在床头柜](https://www.youtube.com/watch?v=wOiZjiyVNWU)里的构造精美的 N 轨距圆形布局。更妙的是，当你仔细观察时，你会发现这不是一个普通的火车场景，这是《T4》中第三类 亲密接触*的一个场景。*

这是一个由两件同样制作精良的家具和火车组成的项目。前者完全是临时搭建的，圆柱形的外部由精心切割的胶合板环制成，中间有一个滑动升降机构，配有混凝土配重。向上滑动圆柱体，布局就展现出来了——在轨道环的中心有一座临时建造的小山，上面有 UFO 被照亮的底面。当火车在轨道上行驶时，它甚至触发了一组交叉灯和声音，以获得额外的真实感。完整的故事可以在下面的视频中看到，非常值得一看。

我们过去已经报道过不止一个隐藏的铁路模型布局，当浏览时发现[【Peter】的作品在](https://hackaday.com/2022/01/06/hidden-model-railroading-taken-to-the-nth-scale/)之前就已经在这里出现了，这并不奇怪。
