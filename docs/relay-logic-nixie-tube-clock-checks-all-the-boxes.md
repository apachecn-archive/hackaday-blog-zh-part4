# 继电器逻辑数码管时钟检查所有的方框

> 原文：<https://hackaday.com/2021/08/09/relay-logic-nixie-tube-clock-checks-all-the-boxes/>

电气工程词典中有几个词会让任何硬件黑客竖起耳朵。第一个当然是“谢妮地铁”,温暖的冷战时代氛围和怀旧氛围。紧随其后的是“中继逻辑”。鉴于它们在计算机和电信史上的地位，以及咔嗒咔嗒的交响乐能让一个极客的心怦怦直跳，我们大多数人只是喜欢一个好的中继黑客。然后是经典的黑客项目:一个独特的时钟来装饰我们的巢穴，并提醒我们什么时候我们已经在我们的项目上工作了太长时间，如果这样的事情甚至存在的话。

考虑到这些事情，如果当[Jon Stanley]的[中继逻辑数码管时钟](http://www.electronixandmore.com/projects/relaylogic/index.html)通过[提示线](https://hackaday.com/submit-a-tip/)来到我们面前时，我们稍微有点神魂颠倒，你可以原谅我们。 装饰着过多的点击继电器，由四个谢妮管点亮，继电器逻辑数码管时钟检查*所有的*我们的盒子。

[Jon]从模仿 CD4000 系列 CMOS 逻辑芯片的继电器模块开始构建。当原型阶段完成时，电路在一个新的板上被重新创建，该板将所有 55 个欧姆龙继电器安装在同一个 PCB 上。结果呢？一个辉煌的数码管时钟将罢工嫉妒甚至最纯粹的黑客的心。休息后一定要看视频！

【乔恩】亲切地记录了整个构建，甚至[还提供各种继电器逻辑板供购买](http://www.catahoulatech.com/index.php?product=RELAY)，如果你想开始自己的继电器逻辑开发。他的网站[也充斥着独特的时钟项目](http://www.electronixandmore.com/projects/index.php)，所以你可以肯定我们会检查这些。

如果你觉得有灵感来构建自己的继电器逻辑项目，请确保你采购了[正版欧姆龙继电器](https://hackaday.com/2018/05/11/fake-omron-relays-are-worth-what-you-pay-for-them/) ，尤其是如果你的[继电器计算机杰作需要六年时间来构建](https://hackaday.com/2020/10/23/relay-computer-consumes-six-years-and-4-5-suitcases/)。
