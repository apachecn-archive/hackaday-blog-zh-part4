# 黑客日播客 136:打小行星，解码航海家，磨便宜，和 PS5 触发

Hackaday 编辑埃利奥特·威廉姆斯和迈克·什奇斯回顾了硬件黑客的伟大一周。当你可以使用开源工具解码来自早已离开我们太阳系的探测器的信号时，这是一个多么美好的时代啊！我们欣赏两个非常便宜的构建，一个用于测量主电源的电流，另一个用于以很高的精度加工小零件，只需几美元。由数百个 7 段模块组成的显示器回避了一个问题:谁说像素需要相同的尺寸？我们惊叹于自主电动货船的概念，惊叹于用太空探测器撞击小行星的挑战。所有这一切，我们甚至没有提到使用 GLaDOS 作为个人助理机器人，但这也在议事日程上！
