# 过度设计的抽油烟机，版本 2

> 原文：<https://hackaday.com/2022/11/07/overengineered-fume-extractor-version-2/>

我们都知道在你的最新项目中增加一个新功能的诱惑。[Arnov Sharma]没有忍住冲动。在他的 3D 打印排烟装置的基础上，他[开发了一个由 PCB 材料制成的新版本](https://www.instructables.com/Overengineered-Fume-Extractor-Project/)。

该设备有一个 18650 电池，并纠正了我们前面提到的原始设计中的几个缺陷。特别是新版本使用了静音风扇，耗电更少。还有一种 3D 打印的过滤器外壳，使用棉花作为过滤介质。

像[以前的版本](https://hackaday.com/2022/09/04/diy-fume-extractor-with-attiny13-speed-control/)一样，ATTiny 控制电机速度。该板使用表面贴装元件，但我们认为，如果您愿意，也可以用通孔元件复制该器件。

这将是一个容易建立的焊接站。我们也想知道，它是否会对一台有臭味的树脂打印机有所帮助。我们可能会在过滤器中添加活性炭，虽然电池很方便，但我们想象我们需要它的大多数地方，我们也会很高兴把它插上电源。

我们总是喜欢使用 PCB 作为结构元件。毕竟，这是一种获得精密加工玻璃纤维的廉价方式。我们甚至看到了用于扳手的技巧。
