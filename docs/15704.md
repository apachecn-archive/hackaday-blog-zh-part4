# NTC 热敏电阻到 ThingSpeak 仪表是一个很好的物联网启动项目

> 原文:[https://hack aday . com/2022/12/20/NTC-热敏电阻-to-thing speak-meter-make-a-great-IOT-starter-project/](https://hackaday.com/2022/12/20/ntc-thermistor-to-thingspeak-meter-makes-a-great-iot-starter-project/)

现在有很多物联网解决方案和框架，[Davide]展示了如何用他的 [ESP8266-to-ThingSpeak 项目](http://davidegironi.blogspot.com/2021/07/esp-temperature-to-thingspeak-smart.html)制作一个简单的数据记录和跟踪应用程序，它可以读取多达四个 NTC(负温度系数)热敏电阻，并通过 WiFi 将数据发送到 ThinkSpeak。

物联网可能是一个非常深的兔子洞，所以如果你正在寻找一个简单的项目来演示工作部件并提供一个起点，[该项目的 GitHub 资源库](http://davidegironi.blogspot.com/2021/07/esp-temperature-to-thingspeak-smart.html)可能会帮助你开始。我们还看到 ThingSpeak 用于[跟踪卫生纸用量](https://hackaday.com/2022/02/03/keep-track-of-toilet-paper-usage-with-this-iot-roll-holder/)，这是一个如何与移动部件的物理对象进行交互的很好的演示。

另一方面，如果你发现阅读 NTC 热敏电阻是更有趣的部分，你很幸运，因为[Davide] [有更多关于那个](http://davidegironi.blogspot.com/2018/01/esp8266arduino-ntc-library.html)的信息，以及修改后的 ESP8266 Arduino 库。观看下面嵌入的视频中他的温度记录硬件的运行之旅。
