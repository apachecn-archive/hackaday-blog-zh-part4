# 从倒下的树中走出来

> 原文：<https://hackaday.com/2022/12/12/travel-uke-from-a-fallen-tree/>

当面临用她院子里的胡桃树的边角料做什么时，[Amy Qian]决定做一个[无头旅行尤克里里](https://amymakesstuff.com/2015/12/11/travel-ukulele/)。[ [via MAKE:](https://makezine.com/article/workshop/woodworking/unique-ukulele-scratch-built/)

无头乐器将调谐器重新定位到乐器的主体，而[钱]必须在主体上进行相当多的修整和削减，以使调谐器刚好合适，并且仍然可以通过在侧面切割的四个铲子来操作。心轴在琴弦上产生的摩擦量最初出现了一些问题，之后她用一组[滚珠轴承和一个她用铝制成的固定器](https://amymakesstuff.com/2017/05/28/travel-ukulele-upgrade/)代替了心轴。

我们喜欢[钱]的广泛的建造日志是如何通过从修剪死核桃树枝到建造一个可演奏的乐器的整个过程来制作这个小乐器的。像脖子上的枫木条和黄檀木口音这样的小细节真的远远超出了雪茄盒乐器，让许多人走上了制琴师的道路。
