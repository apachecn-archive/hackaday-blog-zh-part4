# 控制台控制器 Mod 让截肢者回到游戏中

> 原文：<https://hackaday.com/2020/12/15/console-controller-mod-gets-amputee-back-in-the-game/>

不管是怎么发生的，失去一根或多根手指会在几千个方面改变一个人的生活。我们是一个善于操纵的物种，非常习惯于通过我们手臂末端令人惊叹的附属物与世界互动。当然，找到解决截肢带来的问题的方法是一件严肃的事情，即使仅仅是为了使用假手而修改游戏控制台控制器。

至少从他的视频和 Instagram 帖子中，我们已经非常了解(伊恩·戴维斯)这些地方。[Ian]很难被忽略——他属于他所说的“缺失部件俱乐部”,由那些失去全部或部分肢体的人组成，他通过他的完全机械的半手假肢解决了这个问题。尽管那个假肢的机械连接令人惊叹，但他还没有恢复全部功能，至少还没有达到完全使用现代游戏控制台控制器所需的程度，所以他安装了几个伺服系统和一个小饰品来帮助工作。

一组三个按钮位于[Ian]的 OEM 拇指容易触及的位置。按钮按下那里被翻译成伺服运动，按下原来的保险杠按钮，这是特别不友好的他的售后市场解剖。所有东西都装在一个 SLA 印刷的盒子里，盒子粘在 Playstation 控制器上。[Ian]经历了几次设计迭代，甚至一度想过支持速射，最终决定采用如下视频所示的设计。

这可能不会让他再次变得有竞争力，但这个体系确实让他重返赛场。他对自己的目标相当开放，那就是让人们看到他的设计，让其他截肢者也能广泛使用。希望这能有所帮助。
