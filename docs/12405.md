# 乐高朋克奥瑞知道正确的技术

> 原文:[https://hack aday . com/2022/01/06/Lego punk-or ery-know-just-the-right-technics/](https://hackaday.com/2022/01/06/legopunk-orrery-knows-just-the-right-technics/)

乐高积木被挖穿时那种清晰的声音是否已经深深印在你的脑海里？越来越复杂的技术产品目录是否让你的想象力飞上了更高的层次？从休息时间下方视频中的[巧妙装置来判断，我们相当肯定【Marian】能与这些事情联系起来。](https://www.youtube.com/watch?v=oM5eikh1usI)

毫无疑问，灵感来自于经典的发条驱动系列，[Marian]的乐高太阳-地球-月亮系列是由手摇曲柄或电动马达驱动的。这项研究旨在达到天文上的精确。为此，手摇曲柄旋转一整圈产生一整天的运动。

可以演示日食和月食，以及许多其他原理，如地球倾斜、月相、潮汐锁定等等，[可以在项目页面](https://rebrickable.com/mocs/MOC-88534/marian/sun-earth-moon-orrery/#details)找到。

虽然古典奥瑞系列早于维多利亚时代，但奥瑞系列和蒸汽朋克美学之间似乎有一种几乎无法解释的联系。但是[玛丽安]的故事让我想起了“乐高朋克”这个词。会不会是？鉴于有 2305 件，264 页，436 步的指令，我们认为是！

我们过去只报道了几个系列，从这个[有点简单的激光切割或手术](https://hackaday.com/2017/07/12/laser-cutting-orreries/)到这个[极其复杂而美丽的东西](https://hackaday.com/2022/01/02/an-astronomical-mechanical-clock-in-more-ways-than-one/)。
