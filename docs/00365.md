# JB 焊缝–强度足以修复连杆吗？

> 原文:[https://hack aday . com/2018/08/18/JB-weld-strong-found-to-repair-a-connecting-rod/](https://hackaday.com/2018/08/18/jb-weld-strong-enough-to-repair-a-connecting-rod/)

JB Weld 是一个特别受欢迎的环氧树脂品牌，在许多传说中都有出现。我表哥的邻居的训狗师的爷爷曾经在法国用那个东西修理过一个谢尔曼坦克炮管！他们会说。令人欣慰的是，随着新媒体的出现，有大量的内容让人们对这些疯狂而有趣的主张进行测试。正如德高望重的格蕾丝·赫柏曾经说过的那样，“*一个精确的测量值抵得上一千个专家的意见*”，所以很高兴看到这些实验的发生。

[Project Farm]就是其中之一，[这次尝试用粘性物质](https://www.youtube.com/watch?v=kDT02YamWdE)修复一个小型发动机的连杆。测试中的连杆来自典型的 Briggs 和 Stratton 发动机，磨损非常严重，已经断裂成大约 5 块。首先，用溶剂清洗零件，并让其适当干燥，然后用两部分环氧树脂将零件一件一件重新组装起来。使用适当的技术，给环氧树脂足够的时间固化。

结果呢？可悲的是，可怜的——尽管 JB Weld 尽了最大努力，杆在几秒钟内就解体了，完全无法保持在一起。是的，这是一种奇妙的材料——但它不能做所有的事情。或许它可以用来铸造一个汽缸盖？
