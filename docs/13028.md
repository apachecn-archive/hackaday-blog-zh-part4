# 触觉智能旋钮做几个工作

> 原文：<https://hackaday.com/2022/03/13/haptic-smart-knob-does-several-jobs/>

旋钮就是旋钮，开关就是开关，就是这样，对吧？那些在制造时就固定下来的有棘爪的旋钮呢？哦，还有那些旋钮，让你从左到右慢跑，然后又回到中间——那是不能修改的……对吗？嗯，你可能知道这是怎么回事，在休息下面的视频中， [[scottbez1]展示了一个新的开源触觉输入旋钮](https://github.com/scottbez1/smartknob)，它可以*所有这些东西*，只需一些配置更改！

可能性的列表很长:虚拟咬合点、虚拟弹簧负载、虚拟棘爪、虚拟端点。这是一个虚拟的配置选项集，让这款触觉智能旋钮成为满足您所有旋钮需求的一站式商店。这一切都是可能的，因为旋钮包含一个高分辨率的磁编码器芯片，具有一度的分辨率。该传感器通过软件与无刷 DC 电机相连。圆形 LCD 也能提供视觉反馈。

正如【我自己】在 [Hackaday Discord](https://discord.gg/NkbHrAW7NG) 频道上指出的，为制动器、弹簧和制动器提供可配置的间距*和*强度，简直不可思议。能够随意重新配置旋钮意味着它可以变得上下文敏感。它非常独特*而且*是开源的，所以你可以用 GitHub 上的信息做你自己的。

据它的创造者说，触觉智能旋钮*唯一不能做的事情是帮你报税或调鸡尾酒。嗯，它是开源的，所以也许我们一些更有事业心的读者可以提交合适的拉取请求。*

这不是 Hackaday 的第一个[电动音量旋钮](https://hackaday.com/2020/12/07/an-epic-quest-for-a-motorized-volume-knob/)功能，但它可能是我们迄今为止见过的最简洁的功能之一。感谢 [Hackaday Discord 服务器](https://discord.gg/bWn42ju5j9)上的【mattvenn】提供的精彩提示！
