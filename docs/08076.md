# 精密金属探测器在干草堆中发现针

> 原文：<https://hackaday.com/2020/10/18/precision-metal-detector-finds-needles-in-haystacks/>

全尺寸金属探测器非常适合缩小区域范围开始挖掘。但是，如果你有一个更小的金属探测器可以精确定位呢？这样你就可以花更少的时间去挖掘，花更多的时间去寻找金属。

金属探测器的工作原理是金属在电磁场中的行为方式。[mircemk]重新使用旧中波收音机的铁氧体磁芯来制造天线线圈。当金属物体足够近时，感应的电磁会改变频率，Arduino 会闪烁 LED 并以新的频率及时发出蜂鸣声。

[mircemk]的手持式金属探测器相当灵敏，尤其是对较小的物体。正如你在休息后的演示视频中看到的，它可以感应到大约 4 厘米远的硬币，大约 7 厘米远的盖子等较大的物体，以及几毫米远的针头等微小的东西。还有一个 LED，用于在弱光下寻宝。

不想精确定位一堆没用的垃圾？[内置一些相位检测来帮助你辨别](https://hackaday.com/2019/11/26/progressive-or-thrash-how-metal-detectors-discriminate/)。
