# 一切旧的又是新的:另一个 6502 板诞生了

> 原文:[https://hackaday . com/2021/02/12/everything-old-is-new-again-6502-board-is-born/](https://hackaday.com/2021/02/12/everything-old-is-new-again-another-6502-board-is-born/)

[Jeff]说设计自己的 6502 计算机是一种仪式，他想要这种体验。他的[板](https://www.youtube.com/watch?v=bTi02hrR3P0)可以接受真正的 6502 或仍然可用的较新的 CMOS 变体。有一些现代的便利，如 USB 电源和使用 USB 串行端口的规定。

我们今天被微控制器宠坏了，它把所有的东西都集成在一个封装中，但是对于这种类型的 CPU，你需要自己的内存、I/O 设备和其他支持芯片。[Jeff]采用了传统方法，但选择了仍然容易获得的组件。一些设计现在将所有的支持功能都推到更现代的处理器上，如 Arduino，这很容易做到，但不知何故感觉不太真实。

对于软件，BASIC 有几个版本，一个是基于俄亥俄科学的变体。还有一个监视器图像。凭借 32K 的内存，这在当时应该是一台相当不错的机器。BASIC 解释器始于 1977 年。那时有很多古老的基础游戏，[Jeff]展示了一个著名的扑克游戏版本。6502 的汇编语言也没那么差。

我们喜欢这些古老的复古建筑。如果你不喜欢所有的支持芯片，正如我们提到的，你可以使用另一个处理器。或者，试试一个 [FPGA](https://hackaday.com/2020/11/03/brain-in-a-vat-6502/) 。
