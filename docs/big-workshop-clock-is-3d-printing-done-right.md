# 大车间时钟 3D 打印做得对吗

> 原文：<https://hackaday.com/2020/11/01/big-workshop-clock-is-3d-printing-done-right/>

时间对人类来说是独一无二的重要东西，他们仍然是这个星球上唯一能够建造设备来定期跟踪时间进程的生物。【伊万·米兰达】就是这样一个人，[他为自己的工作室建造了一个巨大的七段钟，将整个房间联系在一起。](https://www.youtube.com/watch?v=4z82I-032zs)

这座钟证明了伊万在 3D 打印领域的设计技巧。利用他的大幅面打印机，每个部分包括一个前面板，大型单件扩散，LED 载体和背板。从每个数字之间的互锁脊到内部指导 LED 灯条安装的整体印刷箭头，有许多漂亮的触摸。Fit and finish 接近商业产品的水平，这是对[Ivan]在该领域多年实践的奖励。

电子方面，ESP8266 运行节目，通过内置的 WiFi 连接同步时间。每个部分包含 9 个 WS2812B LEDs，连接成一个长条形，由微控制器寻址。这意味着这些片段可以被点亮成任何颜色的彩虹，尽管[Ivan]是最欣赏经典红色的人。

[Ivan]一直是大型 3D 打印建筑的支持者——他的坦克履带电动滑板是一个特别好的例子。休息后的视频。
