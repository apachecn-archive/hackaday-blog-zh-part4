# 这个 Arduino 圆形切割机没有切角

> 原文：<https://hackaday.com/2020/05/08/no-corners-were-cut-on-this-arduino-circle-cutter/>

黑客的眼睛总是盯着一笔好交易，所以当[Gadget Reboot]在方形贴纸上看到一个好价格时，他不得不扣动扳机。只有一个问题:他的标志是一个圆形。他本可以在一个慵懒的下午用一把剪刀解决这个问题，但我们认为[建造一台精致的切圆机是对他时间的更好利用](https://github.com/GadgetReboot/GR_StepperIO)。

但是这个项目不仅仅是为了乔托的崇拜者。即使你没有发现自己需要无限供应的完美纸圈，休息后的视频提供了一个很好的案例研究，让步进电机做你的投标；不管那是什么？[Gadget Reboot]引导观众完成双步进电机控制器的设计和构造，该控制器可用于多种任务。

通过板载 Arduino Pro Micro、有机发光二极管显示器和旋转编码器，该控制器只需一些定制软件即可实现您的 CNC 梦想。[Gadget Reboot]甚至在这个版本中使用了低成本的光学终点挡板，这是一种非常好的非接触式方法，可以确保你的机器不会越界。当机器碰巧挥舞着刀片时，这一点尤其重要。

如果你是一个彻头彻尾的现代硬件黑客，喜欢使用网络浏览器而不是摆弄旋钮，那么也许你会对我们最近讨论的支持 WiFi 的步进电机控制器[感兴趣。](https://hackaday.com/2020/02/10/new-part-day-the-wi-fi-stepper-gets-ideas-working-faster/)
