# 数控铣床维修 IPhone 7

> 原文:[https://hackaday.com/2019/01/20/cnc-mill-repairs-iphone-7/](https://hackaday.com/2019/01/20/cnc-mill-repairs-iphone-7/)

现代智能手机是高度集成的设备，将巨大的计算能力放在一个人的手掌中。这种便携式计算能力和连接性已经在无数方面改变了社会，也倾向于导致所述强大的计算机最终掉落在地上或掉进厕所。维修通常仅限于更换屏幕或更换损坏的模块，但也有可能更进一步。

这部手机是 iPhone 7，服务中心报告说 CPU 有问题，唯一的解决办法是更换整个主板。然而，[卡迪实验室]并没有大惊小怪，开始工作。主板安装在数控夹具中，A10 的 CPU 被一层一层地小心翼翼地磨掉。然后使用手术刀和热风枪进一步清理焊盘。然后对各种衬垫进行一些传导性测试，原因还不完全清楚。

在这一点上，一个备用的 A10 CPU 是来源，一个模板是用来申请锡膏或球-这不是立即明显的。新的芯片被重新焊接到主板上，手机被重新组装起来。然后，设备通电，显示功能正常。

这是一次令人印象深刻的修理，表明现代电子产品并非不可能修理——只要你手头有合适的工具。聪明的是，通过使用带有预烘焙程序的 CNC 机器，它大大减少了拆卸阶段所需的劳动力，使维修更具成本效益。该团队特别乐于助人，链接到视频描述中用于完成修复的工具。我们也见过类似的黑客攻击，比如升级 iPhone 的内存。休息后的视频。
