# 借助真人大小的传送门炮塔定位目标

> 原文:[https://hack aday . com/2019/08/12/locating-targets-with-charm-courty-of-a-life-size-portal-turtle/](https://hackaday.com/2019/08/12/locating-targets-with-charm-courtesy-of-a-life-size-portal-turret/)

对于像 SHA2017 这样的大型黑客营地来说，还有什么比开始最后一分钟的疯狂建设更好的方式呢？这是[Yvo]的想法，当时他决定从 Valve 的门户视频游戏中制作一个真人大小的可爱致命炮塔。由于该建筑当时没有添加所有功能就到达了终点线，他为 CCC camp 2019 事件完成了它，包括关闭、打开、瞄准和发射 Nerf 飞镖的能力。

最初基于[微型 2014 炮塔](http://ytec3d.com/turret/)(在 Hackaday 上也覆盖了[),【Yvo】在第一](https://hackaday.com/2014/06/12/3d-printed-portal-turret-searches-and-destroys/)和[第二工作日志](https://ytec3d.com/the-making-of-big-turret-2/)中详细描述了这个新项目[，并详细解释了](https://ytec3d.com/the-making-of-big-turret-1/)[如何将](https://ytec3d.com/how-big-turret/)和工作结合在一起。虽然 2017 年的版本仅用了 50 天就完成了，但整个项目花了大约 300 个小时的 3D 打印。它还配备了四个使用飞轮发射飞镖的 Nerf 枪。车轮由转速为 25，000 转/分的四轴飞行器输出马达驱动。发射飞镖的理论速度超过 100km/h，每枪 18 镖，射速每秒 2 镖。

系统的基本运动控制由 Arduino Mega 处理，而说话和视觉方面由 Raspberry Pi 3+处理，它最终还决定如何移动系统。正如人们在链接后的视频中看到的那样，该系统似乎工作得很好，公司员工的死亡人数可以忽略不计。

尽管这显然不是一个为没有经验的修补者设计的项目，但[Yvo]已经将所有的设计文件和软件一起提供给了用户。然而，我们仍然怀疑完成这些炮塔之一的承诺蛋糕的说法。
