# 轮子上带有 RepRap 的无限构建体积

> 原文：<https://hackaday.com/2019/02/24/infinite-build-volume-with-reprap-on-wheels/>

普通的 3D 打印机是非常有用的工具，如果有足够的时间，非常适合生产小型塑料零件。大多数构建较大 3D 打印对象的项目使用各种技术将它们分成较小的部分，这些部分可以放入大多数基于笛卡尔的打印机的有限构建体积中。然而，没有理由打印机需要坐在盒子里，也没有理由打印机不能四处走动。因此，我们得到了车轮上的 RepRap HELIOS。

[Nicholas Seward]创造了 HELIOS，并将其纳入 2017 年的 Hackaday 奖，使用 SCARA 手臂建造了一台具有大体积且没有移动步进器的打印机。[Nicholas]的一名学生随后做了一项测试，将 HELIOS 安装在一辆有角度的机动车上，让打印机在一个轴上具有无限的构建容量。

[Nicholas]希望当前的基本设置能够打印 200 毫米宽、100 毫米高，并且理论上无限长。通过允许打印机使用独立控制的电机来控制自己，该设备还有可能制造大型弯曲零件。

还有更多的工作要做，特别是允许打印机相对于其工作空间定位自己，以避免大型打印的尺寸问题，但初步结果非常令人印象深刻。我们也见过其他无限容量打印机——[像这样使用传送带设计的打印机。](https://hackaday.com/2018/03/29/a-high-speed-infinite-volume-3d-printer/)休息后的视频。
