# 赌徒的开瓶器

> 原文:[https://hackaday.com/2020/04/09/a-gamblers-bottle-opener/](https://hackaday.com/2020/04/09/a-gamblers-bottle-opener/)

当涉及到完成简单的任务时，比如去掉瓶盖，人们肯定会变得有创造力。木工人[马特·汤普森]发明了一种下一级开瓶器,它不仅能完成工作，还能作为一种机会游戏。(视频，嵌在下面。)

这个过程通常始于转动他的家务转盘，它会出人意料地经常建议你喝一杯啤酒。当瓶盖被一个标准的壁挂式开瓶器打开时，有趣的事情就开始了，当瓶盖穿过一个木制迷宫般的各种机械装置，让人想起 Rube Goldberg 机器。最后，帽子穿过一排钉子，被称为[高尔顿板](https://en.wikipedia.org/wiki/Bean_machine)，这也可以在一些弹球和历史悠久的游戏机中找到，然后落在标有“赢家”和“再试一次”的两个容器中的一个容器中。前者将触发自建花生分配器的旋转轮，为口渴的人提供一些美味的小吃。虽然我们很想看到这个项目更多技术细节的制作视频，但我们仍然欣赏精致的木工和精细的工艺。

顺便说一句，如果你曾经需要一个 Arduino 板，也可以作为开瓶器，那么看看 [HaDuino](https://hackaday.com/2013/10/11/haduino-open-your-beer-using-arduino/) 。
