# 家庭自动化控制器使用粉笔

> 原文：<https://hackaday.com/2021/06/04/home-automation-controller-uses-chalk/>

作为对 [2021 年黑客奖竞赛](https://hackaday.io/prize)的*反思展示*挑战的回应，自由设计工程师【Rick Pannen】为[带来了他的 DIY 家庭自动化控制器](https://hackaday.io/project/179832-retro-home-automation-controller)的复古外观。你可能会被原谅，甚至没有意识到它是一个控制器一眼。[Rick]使用磁性粉笔板建造了这个，并在背面安装了所有的控制电子设备。主处理器是运行 Raspian 的 Raspberry Pi 400，带有 IOBroker 和 Node-Red。你猜对了，面板上的文字和图形是用粉笔徒手完成的。

这个面板上的控制装置是一个由仪表、开关和传感器组成的大杂烩，这些都是[瑞克]在易贝身上找到的或者从朋友那里收集来的。我们对外观简单的旋转拨号盘感到好奇，它根据拨号盘上设置的数字发送脉冲串——这似乎具有旧手机旋转拨号盘的所有功能，但没有任何乐趣。

但是[Rick]的设计允许简单的改变——我们敢说，这鼓励了他们——所以也许我们会在未来的版本中看到一个回收的转盘。还要注意室内照明开/关开关，操作起来一定很有乐趣。我们想知道，有没有什么方法可以使控制装置被磁化并在电路板上自由移动，而不需要永久固定它们？也许是第 4 版或第 5 版的想法。

这种设计有很多可能性，我们期待这种独特的家庭自动化控制器的任何升级或衍生版本。如果你有任何扩展这个想法的建议，请在下面的评论中告诉我们。
