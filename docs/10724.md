# 焊料桥有助于脱焊

> 原文:[https://hack aday . com/2021/07/24/solder-bridges-aid-de solder/](https://hackaday.com/2021/07/24/solder-bridges-aid-desoldering/)

正如我们的[自己的埃利奥特·威廉姆斯所展示的](https://hackaday.com/?p=488379)，许多人认为焊接是电子产品的一项关键技能，但我们并不经常想到脱焊。即使你的技术很完美，也总有可能装上一个坏零件，或者某个零件在以后出现故障，需要更换。[Robert]有一个简短的视频展示了他移除通孔组件的[方法](https://www.youtube.com/watch?v=9jpotpIO1-U)，你可以在下面看到。

当然，这不是我们第一次看到它。事实上，它非常像使用热空气，尽管它不需要热空气，只需要额外的焊料和一个普通的熨斗。当然，如果我们知道连接器是坏的，我们会试图把每一个针脚都拆开，一次取出一个。加热一个接头，然后把你的手放在长凳上可以创造奇迹。

我们总是认为拆焊泵是个好主意，但是电动的往往会有问题。带弹簧的通常更好，但仍有局限性。最后，我们将坚持使用使用热风的，但是如果你只有一个熨斗，这个方法值得一试。你可能也会对[针法](https://hackaday.com/2017/04/03/have-you-ever-tried-desoldering-needles/)感兴趣。
