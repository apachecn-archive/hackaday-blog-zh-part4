# 便携式火腿天线使用 SMD 电容

> 原文:[https://hack aday . com/2020/11/17/portable-ham-antenna-uses-SMD-capacitors/](https://hackaday.com/2020/11/17/portable-ham-antenna-uses-smd-capacitors/)

【K6ARK】喜欢便携操作，所以他把非常轻便的天线组装在一起。他的最新之一使用微小的环形线圈和 SMD 电容器来形成陷阱元件。你可以在下面的视频中看到它的构造。

你通常会认为环形绕组是构建发射机或接收机时会用到的东西，尤其是像这样的小型设备。我们假设天线最适合 QRP(低功率)操作，因为微小的核心在较高功率下会很快饱和。你应该通过 FT50-43 内核的确切功率取决于具体的应用，但我们已经看到大约 5 瓦左右的数字。

SMD 电容安装在一个被破坏的 BNC 连接器内。这使得一个非常紧凑的建设为代价的一点点罚款工作与烙铁，而建设。你可以在下面的第二个视频中看到更多类似陷阱的构造细节。

天线分析仪设备的出现使得这样的项目变得更加容易。端馈半波的概念有反对者也有支持者。[AA5TB]有一个[页面，解释了一些关于天线](http://www.aa5tb.com/efha.html)的理论。不管怎样，[K6ARK]说他已经用这种类型的天线在世界上工作了，许多人都相信它们。

还不是火腿？[丹·马龙]希望你投资 50 美元来改变这种情况。不过，我们怀疑，你会对[更多的空中连线](https://hackaday.com/2019/02/01/portable-ham-antenna-gets-a-workout/)更满意。
