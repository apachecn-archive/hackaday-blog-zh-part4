# 一堆弹簧垫圈为这台数控铣床的牵引杆提供动力

> 原文：<https://hackaday.com/2020/01/11/stacks-of-spring-washers-power-the-drawbar-on-this-cnc-mill-conversion/>

随着 Tormach 和 Haas 占领了大量数控机床的入门级专业市场，我们不再看到太多的手动铣床的数控转换。因此[精密马修斯轧机的动力牵引杆转换](https://www.youtube.com/watch?v=5HjaMnmtYbg)真正吸引了我们的目光。

你说那是什么？[物理匿名]不是已经为磨机造了一个动力拉杆了吗？[他们做了](https://hackaday.com/2018/09/03/air-wrench-becomes-a-milling-machine-power-drawbar/)，还挺成功的。但这是基于气动冲击扳手，虽然它在手动铣床上工作良好，但同样的方法在数控铣床上就有点慢和麻烦了。对于这种构建，他们选择了一种完全不同的方法来提供必要的向上力，以将夹头拉入夹头夹并夹紧工具:弹簧。具体来说，[贝氏弹簧垫圈](https://en.wikipedia.org/wiki/Belleville_washer)，形状像浅杯，可以在很短的距离内施加巨大的轴向力。

[PA]计算出他们需要在几英寸的长度上施加 2700 磅(12000 牛顿)的力，这似乎超出了贝尔维尔垫圈的规格。幸运的是，弹簧可以堆叠，要么“串联”嵌套在一起以增加负载力，要么“并联”交替以在更大的距离上施加额定力。为了压缩它们的堆叠，他们使用了一个漂亮的多级气压缸来挤压弹簧并释放夹头。他们还必须想出一种机制，只有在需要更换刀具时才能与机床主轴接合。下面的视频详细介绍了设计并展示了构建；跳到 11:32，观看牵引杆的动作。

我们期待着[匿名物理]的其他转变。毕竟，他们对修改现成的机器来完成他们的投标并不陌生——见证了他们对 SLA 打印机的改进。
