# RGB《我的世界》标志不仅仅是为了好看

> 原文:[https://hack aday . com/2020/07/24/RGB-minecraft-sign-not-just-for-looks/](https://hackaday.com/2020/07/24/rgb-minecraft-sign-isnt-just-for-looks/)

这个由[极客制造]创造的激光切割和 LED 照明版本的《我的世界》标志看起来足够好，足以在任何游戏玩家的书架上占据一席之地。但这不仅仅是装饰:它还可以通过可寻址的 led 灯通知你《我的世界》的服务器状态，并告诉你玩家何时在线。

在休息后的前半段视频中，[极客制造]展示了如何通过在他的激光切割机上切割出白色和黑色丙烯酸片来构建标志本身。当堆叠在一起时，它会产生令人印象深刻的 3D 效果，但也会隔离每个字母。堆叠后的 RGB LEDs 仔细对齐，每个字母都可以用自己的颜色照亮(或根本不亮)，而不会有光渗入任何一侧。

[![](../Images/217f1e8b240f311051944ccd19e17c9c.png)T2】](https://hackaday.com/wp-content/uploads/2020/07/rgbminesign_detail.jpg)

一旦他有了单独照亮每个字母的方法，就只需要为树莓派写一些代码，就可以对它们做一些有用的事情了。当服务器关闭时通知他很简单，只要让他们都闪烁红色就行了。但是[极客制造]想出的代码也将每个字母与他一起玩的一个朋友联系起来，并在他们上网时点亮他们。所以他不仅一眼就能看出游戏中已经有多少朋友，还能知道他们是谁。这自然意味着显示屏只能显示九个朋友的状态……但是，嘿，这比我们现有的要多。

多年来，我们一直看到人们以奇怪而奇妙的方式将现实世界与《我的世界》联系起来，而且似乎没有任何放缓的迹象。虽然我们认识到这款游戏并不适合所有人，但你必须尊重[它激发了年轻和年长玩家不可思议的创造力。](https://hackaday.com/2020/07/20/finding-the-random-seed-of-minecrafts-title-screen/)
