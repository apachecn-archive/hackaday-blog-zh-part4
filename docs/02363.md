# 微型 3D 打印叉车是非常托盘化的

> 原文:[https://hack aday . com/2019/03/10/miniature-3d-printed-铲车-is-quit-pallet-able/](https://hackaday.com/2019/03/10/miniature-3d-printed-forklift-is-quite-pallet-able/)

如果您有小的物流问题，我们可以为您提供解决方案。[【Leon】结合 3D 打印技术、一些 CNC 工作和精细的焊接技能，制造了一辆小型铲车](https://www.youtube.com/watch?v=_YNgo8A7p8A)，它有 LED 照明、工作叉和遥控器。

这个建筑的电子设备基于几个伺服系统和一对齿轮传动的 DC 电机，并通过 Arduino Mega 驱动。连通性和远程可控性是你对一个 Arduinified 项目的期望。板上有一个 HC-05 蓝牙模块，远程控制由一个定制的 Android 应用程序处理。

在这个项目中值得注意的是实际工作的叉子，几乎像一个真正的叉车。这使得迷你 Arduino 叉车可以拾起迷你托盘，将其放在某个地方，并让迷你 DIY 爱好者上来为迷你 Etsy 建造迷你家具，这些家具将在迷你两层无电梯公寓的迷你门厅中占据显著位置。不，这不是小中产阶级化；这台迷你铲车正在帮助迷你地方经济。

你可以看看下面的整个构建视频，它是以通常的 maker 演示方法拍摄的，加快了整个构建过程，但不知何故保持了无对话的音频。我们有很多东西要感谢[吉米·迪雷斯塔]，不仅仅是摄影。这个叉车的所有文件[都在 Github](https://github.com/leonvandenbeukel/ForkLift) 上，如果你想自己构建的话。
