# 宏观模型使原子力显微镜更容易理解

> 原文：<https://hackaday.com/2021/06/12/macro-model-makes-atomic-force-microscopy-easier-to-understand/>

对于任何摆弄过放大镜的人来说，很容易理解光学显微镜的工作原理。正如显微镜只是一个简单的手持透镜的精致，电子显微镜也是光学显微镜的精致，电子和磁铁代替了光和透镜。但是原子力显微镜呢？现在，这些需要花一点功夫来让你的大脑环绕起来。

幸运的是，YouTube 频道 Breaking Taps 的 Zachary Tong 最近得到了一个非常小巧的原子力显微镜，这导致了这个关于 AFM 如何工作的视频。在进入商业单元之前——但不是在分享一些它能做什么的赏心悦目的照片之前——Zach 很有帮助地介绍了 AFM 基础知识，相当于该仪器的宏观版本。

他的 macro-AFM 使用一台旧的 3D 打印机作为 X-Y-Z 台架，并在打印机的挤压机上增加了一个探头。探头是一个简单的尖尖的铁笔，安装在一个有弹性的电枢末端，由一个音圈和一个磁铁激发上下振动。探针扫过一个样本——他看着[他的 3D 打印点阵](https://hackaday.com/2021/03/01/electroplating-3d-printed-parts-for-great-strength/)——同时在表面特征上上下跳动。电枢在音圈中感应的电流产生一个信号，该信号与探针到达表面的距离成比例，使他能够绘制样本的特征。

实际的原子力显微镜基本上做同样的事情，尽管在一个更精细的尺度上。该探针是一个附着在一块 PCB 上的 MEMS 装置，与之相比，它显得很小。[Zach]使用该设备对一系列样品进行成像，所有这些样品都揭示了纳米领域的迷人细节。诚然，扫描很美，但我们真的很欣赏 AFM 清晰易懂的解释。
