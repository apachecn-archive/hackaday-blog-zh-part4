# 冻结激光束——算是吧

> 原文：<https://hackaday.com/2020/04/23/freeze-laser-beams-sort-of/>

他们说一张图片胜过千言万语，按照这种逻辑，一段视频一定价值百万。然而，从大约 1840 年摄影的黎明开始，摄影师们就制作了假照片。在现代，Photoshop 和 Deepfake 让你不信任图片和视频。[动作实验室]有一个很棒的摄像技巧，看起来他可以控制光速。你可以看下面的视频。

你大概能猜到他不可能真的做到。但在他的视频中，一束真实的激光束似乎在屏幕上缓慢移动，就像电影中的激光枪一样。你可能认为你只需要降低视频速度，但光真的很快，所以你可能实际上不能拉这个特技。

诀窍是制造一个能阻断光束的斩波器。光圈的间距和速度将控制光线的外观。当然，如果你只是看着这个场景，这并没有什么不寻常的。激光束像往常一样出现，然后消失，看起来像是在瞬间移动。

诀窍在于数码相机如何拍照。扫描速度很快，但是光更快。因此，即使激光一次穿过整个画面，相机也可能只捕捉到其中的一部分。通过控制扫描速度和斩波器的旋转，你可以很容易地做出不同的效果。这个视频的灵感来自于之前的一个视频，你可能也会喜欢。

当然，你可以以后再用数字方式添加激光镜头。但事实是，激光螺栓看起来是真的，因为——嗯——它们是真的。在这两个视频中，你可以看到人们与光束互动，这真的很清楚，这不是一个特殊的效果，或者，如果是的话，这是一个非常好的效果。

如果这能激发你制作自己的科幻作品，我们可以给你一些建议。也许甚至在[中加入一点闪电](https://hackaday.com/2011/02/26/diy-lightning-special-effects/)。
