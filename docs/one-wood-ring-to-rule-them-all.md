# 一个木戒指来统治他们

> 原文：<https://hackaday.com/2021/08/31/one-wood-ring-to-rule-them-all/>

[奥利维耶·戈米斯]无法进入末日火山的火焰来锻造一个大的至尊魔戒复制品，所以他只能满足于耐心、枫木和一台木头车床。它确实有额外的便利，不需要开火来暴露它的真实本质，只需要从墙上插座愤怒的小精灵。

[Olivier]用 72 块枫木将戒指分为内外两部分。这些砖块被粘在一起形成 12 面环，然后层层叠叠以达到想要的宽度。表面被切割光滑，并在木材车床上变薄，并为 LED 灯条创建了一个内部通道。使用手动刳刨机穿过内外表面的壁切割黑色语音。利用圆环本身作为模型，他为路由器制作了一个木制底座，使其能够在表面上滑动而不会晃动。

内壁被切割成多个部分，并粘合到外部的凹槽中。铭文覆盖着枫木贴面，当内部发光二极管打开时，仍然可以看到它。电线从支架的底部穿过一个 S 形杆，该杆由夹在模型中的多层单板制成。这件艺术品总共花费了 53 个小时的心血，但最终的成品会让任何一个铁杆《指环王》粉丝羡慕不已。
