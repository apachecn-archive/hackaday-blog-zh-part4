# 建造一个气动压力清洗机

> 原文：<https://hackaday.com/2021/04/23/building-a-gas-powered-pressure-washer/>

虽然你总是可以买到你需要的工具，但是当你拿起一个你自己制作的工具时，你会获得满足感。[Workshop From Scratch]通过制作他自己的装备已经有了一批追随者，[最近的一次是组装一个气动压力清洗机](https://www.youtube.com/watch?v=59olT268vKc)。

构建的关键是保持事物完全独立。所有消耗品——水、肥皂和蜡——都放在洗衣机上，以避免使用软管等。一台小型燃气发动机是该建筑的核心，与一台高压水泵相连。它甚至配有一个启动马达，使它成为一个认证的豪华园艺工具。它还连接到两个装有清洗液的水箱，用于洗车，清洗液通过辅助端口输入泵中进行混合。它全部装配在一个定制的钢架上，由矩形空心型材焊接而成。

这一构建展示了如何使用您的技能来构建适合您的工作流的工具，而不是仅仅忍受现成的工具。我们以前也见过他的工作——[制造其他工具，比如这个电动等离子切割机。](https://hackaday.com/2020/12/29/magnetic-motorized-plasma-cutter-track/)休息后的视频。
