# 全球地位委员会密切关注新冠肺炎局势

> 原文:[https://hack aday . com/2020/04/06/global-status-board-keep-eye-on-新冠肺炎-situation/](https://hackaday.com/2020/04/06/global-status-board-keeps-eye-on-covid-19-situation/)

谈到与新冠肺炎疫情保持同步，基本上有两个学派。有些人宁愿不听到确诊病例或死亡人数，只想关注那些康复的人。这很公平。但其他人希望掌握所有可用的数据，这样他们就可以对这种病毒在全球范围内发生的情况形成自己的结论。看着这个令人难以置信的新冠肺炎状态板，我们会给你一个猜测[鲁本]属于哪一类。

[![](../Images/1720d8be44f71f0e1b172d17ac505a80.png)](https://hackaday.com/wp-content/uploads/2020/04/covidboard_detail.jpg)

Note the laser engraved component labels

这款壁挂式显示器采用 3D 打印和激光切割部件，由 2020 挤压材料制成，经久耐用。很明显[鲁本]相信我们会长期坚持下去，看看这个设备一次可以显示的大量数据点，就不难明白为什么了。

ESP32 每小时从 JSON API 下载一次统计数据，并存储在 SD 卡上。显示屏后面的几个 TFT 显示屏上显示了连续的确诊病例、死亡和康复总数。在右边，32 个不同国家感染的相对严重程度用不同亮度的 led 显示。

也许显示屏上最引人注目的元素是左侧的大型信号器面板，它会亮起以显示世界各地的各种情况。我们很欣赏[鲁本]的超前思考，并增加了一盏一旦为新冠肺炎部署疫苗就可以使用的灯，但纳入“戒严令”指标肯定无助于我们摆脱我们都发现自己处于适当的反乌托邦的感觉。

对于那些更愿意从源头获取信息而不是通过媒体过滤的人来说，我们最近报道了一些 API，它们将允许你获得自己的最新新冠肺炎统计数据。无论你是想建造像这个显示屏一样精致的东西，还是只是想将它反射到终端，确保你看到的是[准确的数据是识别转折点](https://hackaday.com/2020/03/28/reading-the-tea-leaves/)的关键。
