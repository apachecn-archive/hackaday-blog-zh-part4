# 盖革计数器的来龙去脉，出于个人原因

> 原文:[https://hack aday . com/2018/08/12/the-in-out-of-the-geiger-counters-for-personal-reasons/](https://hackaday.com/2018/08/12/the-ins-and-outs-of-geiger-counters-for-personal-reasons/)

在人的一生中，有时环境会驱使我们对一个特定的主题产生强烈的兴趣，我们会把精力投入到尽可能多的关于这个主题的信息中。[目前的来源]，又名[德里克]，似乎在这样的情况下，这些天来，他的兴趣领域是放射性及其测量。因此，在空闲时间里，他制作了这个关于放射性和盖革计数器如何工作的视频回顾。

为什么对放射性感兴趣？直截了当地说，因为他有放射性，至少在接下来的一周。你看，[德里克]最近被诊断出患有甲状腺癌，甲状腺切除术后清除任何杂散甲状腺细胞的治疗选择之一是饮用碘-131 鸡尾酒，这是一种在甲状腺细胞中积累并杀死它们的放射性同位素。问题是，这会使病人受到危险的辐射，需要隔离一周或更长时间。为了打发远离家人和朋友的时间，[Derek]拆卸了一台商用盖革计数器，这是一台经典的 Ludlum Model 2，带有一个扁平探头。电表的内部结构出奇的简单，电路的每一级都很容易识别。接下来，他自己动手制作了一个盖革计数器套件，也非常简单——只是一个由 555 定时器和微控制器组成的高压部分。他用自己作为一个来源来测试这两种仪器；我们不得不说，听到他仍然如此炙手可热，真是令人震惊。看看下面的视频吧。

在这种情况下，我们很惊讶[德里克]不仅保持冷静，而且表现出良好的幽默感。我们祝愿他康复顺利，如果像这样的拆卸或像[这个冷冻报警器](https://hackaday.com/2017/10/29/junkbox-freezer-alarm-keeps-steaks-safe/)或[一个无集成电路双极电源](https://hackaday.com/2016/10/29/build-the-simplest-bipolar-power-supply/)这样的项目能帮助他应对，那么我们都赢了。
