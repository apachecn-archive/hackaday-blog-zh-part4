# 修理一个老式的锋利的记忆刻录机

> 原文：<https://hackaday.com/2019/03/26/repairing-a-vintage-sharp-memowriter/>

如你所知，我们是 Hackaday 的建筑爱好者。但是我们也很喜欢修理那些可能最终会被填埋的东西。尤其是当这些东西碰巧是有趣的老式硬件时。所以[【ekriirke】为让这台 20 世纪 80 年代早期的 Sharp MemoWriter EL-7000 恢复运行而投入的工作](https://imgur.com/gallery/ClD5lOK)绝对是我们的拿手好戏。

MemoWriter 有许多问题需要解决，但没有一个比 NiCd 电池突然出现在外壳中更严重的了。电池漏电是一种我们大多数人可能都见过不止几次的故障模式，但看到绿色腐蚀像病毒一样在内部蔓延，这种痛苦从未减少。当[ekriirke]打开这个小玩意时，迎接他的是一个特别糟糕的盒子，有一大块 PCB 痕迹被腐蚀掉了。

草酸去除了腐蚀，大大降低了有害因素，但并没有使计算器恢复正常工作。为此，[ekriirke]用一根电线重新连接了每一条损坏的痕迹；他甚至尽可能地遵循原始痕迹，这样最终的结果看起来更整洁一些。一旦所有东西都恢复了导电性，他就用一层指甲油覆盖整个修复部分，以粘附电线并添加一层保护涂层。指甲油可能不是我们的首选封闭剂，即使它是，也可能不是那种特定的颜色，但有时你必须使用手头上的东西。

在多年不用之后，色带盒不出所料地变干了，所以[ekriirke]用涂在内部海绵上的永久性记号笔中的液体来更新它。他还对电池盒进行了一些修改，这样他就可以插入可充电的镍氢 AA 电池，而不是建立一个专用的电池组。外壳中没有电池门，因此要取出电池需要打开计算器，但至少他有能力在将设备存放起来之前取出电池。应该有助于避免第一次发生的事情重演。

如果你是一个好的修复爱好者，我们有足够的东西让你开心。从[起死回生被摧毁的雅达利](https://hackaday.com/2018/07/15/incredible-atari-800xl-case-restoration/)到[让一些珍爱的儿童玩具重获新生](https://hackaday.com/2019/03/10/the-fine-art-of-restoring-matchbox-cars/)，修理旧东西就像从头开始建造一样令人着迷。
