# 所有人登上垃圾快车

> 原文:[https://hack aday . com/2023/01/02/all-on-the-board-the-garbage-express/](https://hackaday.com/2023/01/02/all-aboard-the-garbage-express/)

齿轨铁路是一种有点不寻常的火车移动方式，通常只在火车需要攀爬陡峭地形时安装。任何超过 10%的坡度都需要额外的牵引力，因为车轮和铁轨之间的摩擦力不足以推动火车前进或防止它向后倒。即使没有陡峭的山坡要爬，有时有齿轨铁路也是必要的，正如[Max Maker]在为他的垃圾桶建造火车时发现的那样。

这个建筑开始是为了避免每月把他的七个垃圾桶推到路边。最初，他建造了一条更标准的铁路，用一个简单的电机来驱动火车，但他很快意识到，即使使用塑料车轮，也没有足够的抓地力，即使这条轨道遵循相当平坦的地形。由于铁轨是钢制的，他很快就在其中一条铁轨上焊接了一个齿轮齿条系统。在他最终确定解决问题的设计之前，这个建筑经历了多次迭代，它还包括其他几个功能，如远程控制和在后院车站为火车安装的弹簧自动充电器。

虽然我们总是欣赏那些将每月只发生一次的相对简单的任务自动化的人的古怪行为，但[Max Maker]希望将这种想法变成一种针对老年人或残疾人的商业产品，他们将真正受益于一种可靠的半自动系统，为他们将垃圾桶带到路边。而且，如果你的系统只涉及一个垃圾桶，[还有其他方法来自动完成把垃圾拿到路边的任务](https://hackaday.com/2020/09/10/garbage-can-takes-itself-out/)。
