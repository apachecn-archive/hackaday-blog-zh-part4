# 少臭树脂两种方式

> 原文:[https://hackaday.com/2021/05/13/less-stinky-resin-two-ways/](https://hackaday.com/2021/05/13/less-stinky-resin-two-ways/)

在看了[鞠波叔叔]关于 Elegoo 的大豆基 3D 打印树脂和他们的微型空气净化器的视频后，我们无法确定这种树脂是否没有其他树脂那么难闻，或者这种空气净化器是否有奇效。也许两者都有一点。

我们使用了 [Eryone 超低气味树脂](https://www.youtube.com/watch?v=Ev2IeJ1vdQ0)，它的气味比油漆还少。这听起来像是挽歌。然而，我们总是怀疑任何树脂真的是由天然成分制成的说法。显然拥有化学博士学位的 Brent 指出， [AnyCubic Eco resin](https://www.youtube.com/watch?v=ht4tbCiFxeM) 做出了类似的声明，但很可能只是部分由大豆制成。当然，不到一半是大豆，但还有另一半。尽管如此，我们认为这总比没有好。如果你想知道为什么树脂会在紫外光下固化，或者单体是什么，那么这个视频(也在下面)值得一看。

我们已经盯上了小型空气净化器，想知道它们如何处理激光切割机的烟雾。它们很小，很安静，充满了杂乱的木炭过滤器，所以也许它们会工作。我们会通知你的。

至于树脂打印机，它们似乎没有问题，尽管放置它们可能是一个挑战。如果打印机的盖子是铰接的，你可能需要一个支架，鞠波发现你不希望它们离 Z 轴太近。

如果要用臭树脂，我们对[如何处理](https://hackaday.com/2020/04/07/3d-printering-when-resin-printing-gets-smelly/)有一些建议。如果你还没有开始用树脂打印，[让我们来帮你](https://hackaday.com/2020/04/30/3d-printering-will-a-resin-printer-retire-your-filament-based-one/)。
