# 我有两个转盘和一个激光雕刻机

> 原文：<https://hackaday.com/2022/11/27/ive-got-two-turntables-and-a-laser-engraver/>

数字媒体给我们带来了很多好处。对于像录制和播放音乐这样的事情，数字拷贝不会降级，它们可以具有任意高的质量，并且可以通过多种不同的方式播放，包括通过数字流媒体服务。也就是说，许多人觉得数字体验并没有尽可能忠实于原声，而是选择了模拟方法。然而，创造音乐的模拟拷贝是一件更加困难的事情，正如[Marco]通过使用激光雕刻机生产黑胶唱片所展示的那样。

[Marco]通过组装和校准激光雕刻机开始了这个为期一个月的项目。它有足够好的分辨率将模拟数据编码到一张乙烯胶片上，但他必须开发软件。第一步是生成音频样本，然后通过滤波器对其进行处理，去除一些不需要的频率。从那里，波形变成了一个螺旋，说明了唱针在移动到中心时速度的变化。然后数据最终准备好发送给激光雕刻机。

[Marco]在转移到乙烯基之前，使用木材进行了几次非常成功的练习，经过激光雕刻师的一些校准，他已经准备好了几乎完美的 45 rpm 记录，可以开始播放了。这是一个很好的手表，如果没有任何比看到一个工作木材记录。[我们实际上在](https://hackaday.com/2015/08/07/laser-cut-your-own-vinyl-records/)之前已经看到了一个类似的项目(没有木头原型)，还有一个[从图像中播放唱片](https://hackaday.com/2014/03/03/atomic-powered-robots-and-records-played-with-optics/)，但是已经有相当长一段时间了。
