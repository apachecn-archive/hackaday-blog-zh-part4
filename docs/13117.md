# 旧打印机变成直接激光光刻机

> 原文：<https://hackaday.com/2022/03/22/old-printer-becomes-direct-laser-lithography-machine/>

在家自制集成电路需要什么条件？相对来说，很少有勇敢的黑客试图回答这个问题，答案通常是“大量二手设备”但是它不一定都是从半导体工厂抛弃的，正如[Zachary Tong]用他的自制直接激光光刻装置向我们展示的那样。

我们大多数人都熟悉掩模光刻法，这要归功于使用光致抗蚀剂制造 PCB 的古老工艺——用[光聚合物](https://hackaday.com/2021/12/21/plastics-photopolymers-for-3d-printing-and-beyond/)处理覆铜板，应用包含待蚀刻迹线的掩模，将电路板暴露在紫外光下，在蚀刻前选择性地硬化抗蚀剂层。[Zach]探索了这一主题的一个变体——无掩模光刻法——以及用这个装置将其按比例缩小。光具座聚焦并引导紫外激光进入电流计，电流计是从旧激光打印机中抢救出来的。检流计非常精确地控制准直激光束的位置，然后将它聚焦在显微镜上，极大地缩小了它的视野。激光在覆盖有光刻胶的硅晶片表面跳动，蚀刻掉光刻胶，使硅为蚀刻和进一步加工做好准备。

由于它是由回收的组件、铝挤压和 3D 打印零件制成的，[扎克]的设置远非最佳。但是他能够得到一些非常令人印象深刻的结果，其特征小至 7 微米。当然，还有很多优化的空间，包括更好的[检流计](https://hackaday.com/2019/06/29/lighting-tech-dives-into-the-guts-of-laser-galvanometers/)和不那么特别的光学设置，但我们很想看看这将走向何方。[扎克]说他的目标之一是自制[微机电系统](https://hackaday.com/2017/06/26/mems-the-biggest-word-in-small/)，所以我们很期待。
