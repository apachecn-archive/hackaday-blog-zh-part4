# 绘图仪使用双盘

> 原文:[https://hackaday.com/2021/02/19/plotter-uses-dual-disks/](https://hackaday.com/2021/02/19/plotter-uses-dual-disks/)

如果你想在 X 和 Y 平面上移动一支笔(或者一个 CNC 工具，或者一个 3D 打印热端)，你的选择通常是非常简单的。许多机器使用简单的笛卡尔 XY 运动，使用两个电机和某种线性驱动。还有 core-XY 排列，其中两个电机移动皮带，使磁头向两个方向移动。Delta 打印机使用另一种安排，但我们见过的更奇怪的方法之一是[双盘极性打印机](https://hackaday.io/project/177633-dual-disc-polar-printer)，顾名思义，它使用两个旋转的磁盘。

这种独特的机制使用一个电机来旋转磁盘，另一个电机来旋转整个组件。打印头——在这种情况下是一支铅笔——保持静止。正如你在下面的视频中看到的。

我们喜欢[海因茨的]评论，他提到了自己的影响。有一台[θ打印机](https://hackaday.io/project/812-theta-printer)。早在 2006 年，在 [RepRap 论坛上也有一个提议。但这个主谋是他的。](https://hackaday.io/project/812-theta-printer)

我们经常想到使用旋转 Z 轴进行 3D 打印，尽管这需要一些近似的数学计算，因为每一层实际上都是一个螺旋，这取决于你在旋转中的位置。然而，如果你想做一个绘图仪或其他像激光雕刻机一样的平面机器，那就帮不了你了。这种设置适用于这些情况，但我们不确定与现有方案相比是否有任何真正的好处。除了它在运动中看起来很酷。

如果你想了解更多关于 core-XY 的信息，我们已经为你准备了一些阅读材料。Theta 打印机是我们竞赛的一部分，所以[也可以](https://hackaday.com/2014/08/28/thp-semifinalist-theta-printer/)了解更多。
