# 五英尺键盘让一切尽在掌握

> 原文：<https://hackaday.com/2022/10/07/five-foot-keyboard-lays-it-all-on-the-line/>

我们敢打赌，在我们最倾向于科技的读者群中，有大量的打字员。因为当然，快速打字和从家排打字与智力无关，一切都与练习和死记硬背肌肉有关。但是如果键盘都是 home row 呢？

没错——[Google Japan](https://landing.google.co.jp/________________/)([译](https://landing-google-co-jp.translate.goog/________________/?_x_tr_sl=auto&_x_tr_tl=en&_x_tr_hl=en&_x_tr_pto=wapp))又回来了，推出了另一款 100%真实且开源的玩笑外设。无论你是想在咖啡馆里埋头苦干的时候与他人保持距离，还是真的，*真的*想进入结对编程，这就是你要的键盘。虽然原型高达 7 英尺长(或宽，无论如何)，但最终版本更短，更友好，并且可以在户外休息时充当拐杖，外加一只保护鞋。

就像他们的[马克杯键盘](https://hackaday.com/2021/10/11/cant-spill-coffee-on-your-keyboard-if-its-already-inside/)一样，我们感谢让这个键盘变得真实的工作，就像这个笑话本身一样。我们最喜欢的事实是，这是由 17 个不同的电路板组成，包括控制板。休息之后，一定要看看相当搞笑的宣传片。
