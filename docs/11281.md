# 无用的机器是时钟

> 原文：<https://hackaday.com/2021/09/14/useless-machine-is-a-clock/>

无用的机器是一类有趣的设备，一旦打开就会自动关闭，因此得名。尽管没有真正的意义，但建造和操作它们还是很有趣。[伯克]在精神上遵循了这个想法，他把一个他不得不使用的旧时钟放在一个毫无用处的机器上。但是，这个无用的机器不是简单地在开机时自动关机，而是在任何人看到它的时候从墙上的支架上脱落并掉落在地上。

很难判断这个时钟最初是在他开始这个项目时坏掉的，还是多次检查时间导致时钟损坏了自己，但无论如何，这个项目都是一个即时的经典。这台单板计算机由一个驱动树莓派的小电池供电，运行 OpenCV，并被编程为识别任何指向其大致方向的人脸。当它这样做时，它会激活一个小伺服系统，将它从墙上撞下来，使它毫无疑问地变得毫无用处。

[伯克]不知道他为什么会有这个想法，但它是愚蠢和有趣的。将所有东西粘在一起的胶带也是最终的点睛之笔，我们真的无法证明为一个在房间里到处乱扔的项目花费太多的成本是合理的。另一方面，如果你在寻找一个更精致的无用机器，我们[已经看到了一些令人印象深刻的复杂程度](https://hackaday.com/2012/12/22/an-even-more-useless-machine/)。
