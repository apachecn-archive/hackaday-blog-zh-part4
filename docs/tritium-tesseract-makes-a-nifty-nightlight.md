# 氚宇宙魔方是一个漂亮的小夜灯

> 原文：<https://hackaday.com/2019/02/11/tritium-tesseract-makes-a-nifty-nightlight/>

正方体是三维的，宇宙魔方是四维的。这个宇宙中的凡人发现很难思考四维几何，但是在我们自己有限的世界中，有方法对这种异端形状进行投影。[Sean Hodgins]对几何学很感兴趣，决定建造一个宇宙魔方，里面有每个人都喜欢的氢同位素氚。

构建从 3D 打印的内部和外部框架开始，[在这种情况下来源于尼龙的 Shapeways](https://www.shapeways.com/product/W8ZL2G3E7/tritium-tesseract-prototype-hypercube?li=shareProduct)。两个框架都有孔，这些孔被设计成与现成的氚小瓶摩擦配合。这些小瓶利用氚的放射性衰变和磷涂层来产生持续大约十年的昏暗辉光。内部框架固定在外部框架内，小瓶充当结构支撑，然后内外表面安装半透明的镜面丙烯酸树脂，创造出一种美妙的无限效果。

这是一个有趣的小饰品，在任何情节薄弱的科幻电影中作为麦高芬都是完美的。[肖恩]指出，虽然氚辉光令人失望地暗淡，但该设备确实是一个很好的夜灯。如果你已经建造了一个超立方体，并且对它感到厌倦，你可以随时将你的氚小瓶改造成核电池。休息后的视频。
