# 不要猜测，倾听你的植物对水的请求

> 原文:[https://hack aday . com/2020/10/23/don-guess-listen-to-your-plants-please-for-water/](https://hackaday.com/2020/10/23/dont-guess-listen-to-your-plants-pleas-for-water/)

有植物在身边很好，但它们都有不同的浇水需求。如果他们渴的时候能叫出来就好了，对吗？现在他们可以了。你所需要听到的就是你自己的克劳斯纳机器。[RoniBandini]克劳斯纳机器基于罗尔德·达尔的一个短篇故事，故事讲述了一个发明家制造了一种机器，可以在植物被切割时发出尖叫声。

[在【RoniBandini】的克劳斯纳机器版本中，重点是根据植物的土壤含水量](https://www.instructables.com/The-Klausner-Machine-Hear-Your-Plants/)来判断植物的感受。Arduino Nano 从土壤湿度传感器读取数据，如果土壤干燥，植物就会尖叫。如果土壤潮湿，植物会发出 DF 播放器 Mini 和 SD 卡发出的快乐声音。我们认为模拟米是一个伟大的触摸，跳针真正拟人化的植物。

向前迈进，更好地欣赏你的植物的感受，因为这个项目是开放的。也许这会帮助你更经常地给它们浇水。有些植物需要修剪，所以我们认为如果你能在修剪时让它尖叫会很酷。休息后请欣赏演示。

这并不是我们第一次看到模拟仪表与土壤湿度结合使用。到底什么是 VU 米？[我们自己的【丹·马洛尼】不久前确实在这个问题上取得了进展。](https://hackaday.com/2018/08/09/the-vu-meter-and-how-it-got-that-way/)
