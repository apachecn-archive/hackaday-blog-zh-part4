# Rŏ̽ta:计数，有风格

> 原文：<https://hackaday.com/2022/08/19/ro%cc%86%cc%bdta-counting-with-style/>

[Rǒta 数数事情](https://hackaday.io/project/186093-rota)。就是这样，真的——一个廉价的小机械计数器用拇指按压做什么，或者你用手指和脚趾数能做什么，这差不多就是 Rǒta 所做的一切。但是它用的是*风格*。

好吧，这对凯文·桑托·卡普西奥有点不公平——rǒta 还有一些比简单的计数更好的方法。但实际上，这些功能只是这个小工具外观的锦上添花。Rǒta 是围绕着旋转式电话拨号机制和三个谢妮电子管的无与伦比的组合而建造的。表盘看起来像是来自一部老式付费电话，全是闪亮的铬合金，看起来超级结实。Nixies 位于定制 PCB 上的表盘顶部，所有东西，包括电子管的高压电源，都封装在一个 3D 打印的盒子里，有点*辐射*的感觉。

但是这个东西是做什么的？其实还挺多的。它会上下计数，用你拨进去的任何号码。您可以从零开始递增，也可以输入任何三位数作为起始计数。如果你喜欢的话，它可以记录你高尔夫比赛的分数，而且它还有秒表功能。你甚至可以拨号显示当前的电池电压。只使用拨号盘来实现所有这些功能需要一些独创性，但这就像过去拨打运营商的电话一样简单——拨 0 会将其置于菜单模式，允许您访问拨号盘中心卡上打印的任何功能。这很聪明——看看下面的视频。

是不是特别有用？也许不是。但是这什么时候成为衡量一个项目价值的标准了呢？像这个旋转式手机这样的东西可能更有用，但有时看起来很棒就足够了。
