# 像 Ebike 电池一样的 Ryobi 电源组

> 原文：<https://hackaday.com/2021/06/07/ryobi-power-packs-as-ebike-batteries/>

到目前为止，制造定制锂离子电池组的过程已经广为人知，大多数制造商都可以做到。但这不是一条没有风险的道路，电池保护和管理的失误可能会付出高昂的代价。对于那些担心电池问题的人来说，令人高兴的是，匹兹堡大学的一群工程学生提供了一个解决方案。[他们的项目是将脚踏自行车转换为电动助力](https://hackaday.io/project/180098-diy-ebike-battery-system)，在此过程中，他们没有自己制作工具包，而是使用现成的 40V Ryobi 动力工具包。

自行车转换相对传统，曲柄由曲柄和电机组件取代，一对 Ryobi 包位于车架上的 3D 打印支架中。它的价值在于提醒人们，这些包已经发展到可以替代昂贵得多的自行车专用包的地步，并且它们包含所有的平衡和保护电路，这也使它们成为比自己制作包更安全的选择。这种做法的好处是巨大的，因为它们让更多的自行车车主能够实现高质量的改装，所有零件都只需简单的在线订购。请看休息时间下面的视频了解更多细节。

这些 Ryobi 细胞显然已经在我们的社区中为自己赢得了一席之地。
