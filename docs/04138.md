# BST-863 热气返工站拆卸

> 原文：<https://hackaday.com/2019/09/09/bst-863-hot-air-rework-station-teardown/>

【Voltlog】早就有 952 热风返工站了。当你看到它时，你会认出它——它们是来自中国的无处不在的烙铁和热风枪的组合，以众多品牌出售。他认为旧的工作站不如现有的一些新设备好，并对 BST-863 工作站进行了拆卸和检查，该工作站的价格远低于 200 美元。你可以看下面的视频。

他对工件夹具的制造质量印象深刻。它可以让你储存热风枪，并保持在待机模式。他也喜欢触摸屏，尽管哔哔声似乎有点烦人。然而，总的来说，运行噪音小于它所替换的旧设备。

它只需要 8 个螺栓打开情况下，主要是内部建设看起来相当不错。箱子里有很多浪费的空间，接地有点可疑，但我们肯定见过更糟的情况——比这更糟的情况。奇怪的是，通孔和表面贴装元件混合在一起。

注意到的另一个问题是交流线路连接器的刀片连接器有点松动。很容易把它们压平，使之更合身。总的来说，这看起来很划算。有了这个漂亮的屏幕，我们希望有人能对固件进行逆向工程，这样这个东西就能像其他廉价电台一样被黑客攻击。
