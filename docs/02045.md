# 磁性魔方

> 原文:[https://hackaday.com/2019/02/16/the-magnetic-rubiks-cube/](https://hackaday.com/2019/02/16/the-magnetic-rubiks-cube/)

谈到以 ernő·鲁比克命名的魔方的遗产，他有很多要解释的。它既迷惑又折磨了几代人，让一些人在操场上看热闹，而另一些人在看似棘手的难题面前被迫承认失败。碰巧[汤姆·帕克]一直在研究一个具有新颖磁性设计的魔方。

是的，没错——[Tom]的魔方避开了原魔方传统的旋转滑动机制，而是全部用磁铁代替。立方体的每个部分，以及隐藏的中心部分，都是 3D 打印的。通过使用熔融沉积打印机，并在某些层暂停打印，可以在打印过程中将磁铁嵌入零件内部。

[Tom]提供了几种不同版本的部件，以适应不同功能的打印机。最终的魔方既允许常规的魔方运动，也允许玩家作弊并重新组装它，而不必像最初的玩具一样先用力扔向墙壁。

这是一个有趣的构建，对于掌握在 3D 打印中嵌入零件所涉及的技术是一个很好的构建。它可能无法自行解决，[但我们已经看到另一个建筑可以完成这一令人印象深刻的壮举](https://hackaday.com/2018/09/24/self-solving-rubiks-cube/)。休息后的视频。
