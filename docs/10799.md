# PnPAssist:用于手动 PCB 组装的“智能”构建平台

> 原文:[https://hack aday . com/2021/07/23/PNP assist-a-smart-build-platform-for-manual-PCB-assembly/](https://hackaday.com/2021/07/23/pnpassist-a-smart-build-platform-for-manual-pcb-assembly/)

在过去的几年里，开源的拾放机已经取得了很大的进步，但是如果你一次只制造几块 PCB，就不值得花费设置时间和机器成本。[Nuri Erginer]发现自己经常处于这种情况，所以他创建了 [PnPAssist](https://hackaday.io/project/179878-pnpassist) ，一个“智能”构建平台，以加快手动 PCB 组装。休息后的视频。

PnP assist 由一个小型圆形平台和一个激光十字准线组成，该平台可以自动平移和旋转，将当前足迹放置在平台的中间，即显微镜视野的正中央。整个设备还可以在其底座上自由旋转，以避免扭曲您的手臂来匹配足迹方向。只需从您喜欢的 PCB 设计软件中导出 PnP 文件，将其加载到 micro SD 卡上，插入 PnP ist，然后开始组装。相关的零部件信息显示在机器右侧的小型有机发光二极管显示屏上。[Nuri]还创建了一个组件组织托盘，它将通过 RGB LED 指示正确的隔间。

在构建平台下方，一个 3D 打印齿轮与一对由步进电机驱动的平行丝杠相接触。丝杠的相对运动允许平台旋转、平移或两者兼有。这种安排也意味着机器比传统的 XY 工作台更紧凑，不用时可以打包。底座用一套吸盘或螺丝牢牢地固定在工作台上。电力通过使用滑环的固定底座提供，因此当你旋转机器时没有电缆缠绕。

我们可以肯定地看到，这台机器对任何小型电子装配工作都是一个巨大的帮助，尤其是考虑到快速设置时间和相对简单。它还可以与我们过去推出的 [3D 打印元件分配器](https://hackaday.com/2020/10/05/spare-smd-storage-with-stacking-smt-tape-reels/)或[元件转盘](https://hackaday.com/2021/02/03/manual-pick-and-place-turntable-makes-board-assembly-easier/)配合使用。
