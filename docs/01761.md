# 关于业余卫星的一切

> 原文:[https://hackaday.com/2019/01/07/all-about-ham-satellites/](https://hackaday.com/2019/01/07/all-about-ham-satellites/)

建一个地面站通过卫星与人交流有多难？可能没你想的那么难。[现代火腿]有一个新视频展示了这有多简单。事实证明，在收音机方面，你只需要一台便宜的中国收音机。然而，你确实受益于有一点天线。

对技术感兴趣的人对太空感兴趣并不罕见。因此，许多业余无线电爱好者将太空与这项爱好联系起来也就不足为奇了。一些人做射电天文学，其他人从月球或流星反射信号。还有一些人发射了卫星，尽管这可能不完全准确，因为据我们所知，所有的业余无线电卫星都是搭乘商业火箭，而不是由业余无线电爱好者自己发射的。尽管如此，在太空中设计和运营一个业余无线电台并不是一件容易的事，但随着每一代卫星变得越来越复杂，这项工作已经进行了很多次。

虽然使用定向天线可以获得更好的效果，但是使用一个相当普通的天线也可以进行一些接触。回到 20 世纪 70 年代和 80 年代，当卫星在头顶上时进行跟踪是一项主要任务，但现代火腿只需要一个手机应用程序。

如果你有火腿坐在他们的收音机前进行冗长讨论的图像，你没有看到典型的卫星传递。你的时间不多，所以联系要快，要中肯。事实上，这个视频消除了很多对火腿的成见。一个年轻人展示了如何用业余无线电做一些令人兴奋的事情，而且投资很少，如果你有契约限制也没关系，因为当你不使用它时，所有的设备都可以放在你的车库里。

缺点是[现代汉姆]的演示没有显示他进行任何可靠的接触，尽管他清楚地听到卫星和人们听到他。他承认这不是他最好的传球。下面的第二个视频显示了一个更典型的具有相同设置的过程。如果你想看看用一个更普通的天线能得到什么结果，[看看这个视频](https://www.youtube.com/watch?v=mv4K41Ztax8)。

除了 hams 建造的卫星之外，一些卫星已经开始执行不同的任务，并在后来被 hams 接管。如果你没有业余无线电爱好者执照(顺便说一句，他们比以前更容易获得)，你仍然可以在收听一些非常有趣的太空通讯。
