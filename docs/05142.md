# 194 LED 球是另一个层面上的自由形式焊接

> 原文：<https://hackaday.com/2019/12/23/194-led-ball-is-free-form-soldering-on-another-level/>

我们都在这些页面中看到了大量令人印象深刻的自由形式焊接，也许我们中的一些人甚至自己也尝试过。只用裸露的导体、电子元件和焊料，就可以制作出复杂而美丽的电子雕塑。[但是来自【jiří·普劳斯】](https://www.youtube.com/watch?v=Q5d8gTppuYo)的最新自由形式项目将这种媒介提升到了一个新的水平，因为他采用了不少于 194 个裸露的表面贴装可寻址 led，并将它们形成了一个完美的球体，仅由焊接的铜线支撑。

这一壮举是在 3D 打印的半球形夹具的帮助下实现的，每个 LED 都有空间，每一半都是在马拉松焊接过程中组装的。由于一些干接缝，需要进行一些重大的故障检测，在两半接合时，将 ESP32 板和 Li-Po 电池安装在两半之间。结果是一个胜利，一个与众不同的圣诞小玩意，我们不确定即使在好天气里也能焊接得这么好。

如果你认出了[Jiří ]的作品，那可能是因为我们以前展示过他的几件作品。很难选出一个最喜欢的，但最吸引我们眼球的是他的全功能自由式 Arduino Uno。
