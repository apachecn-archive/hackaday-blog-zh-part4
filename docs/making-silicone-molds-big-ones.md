# 制作硅胶模具——大模具！

> 原文：<https://hackaday.com/2021/01/21/making-silicone-molds-big-ones/>

如果你已经有了一样东西，还想要更多，用硅胶模复制它是一个很好的方法。这适用于 3D 打印你需要很多副本的东西，以及各种各样的其他用例。[Eric Strebel]为自己在这一领域的能力感到自豪，并发布了一份指南，以简单可靠的方式生产超大型硅胶模具。

总体过程很简单，但如果遵循得当，它会产生很好的结果。[Eric]首先用木头做一个模具盒，涂上虫胶以确保它不会粘在硅胶上。然后，主零件被粘在底座上，由激光切割的纸板带包围，作为封条和钥匙。一旦适当脱气的硅树脂被注入并固化，第二部分就可以制作了。将模具在模具箱中翻转，移除密封键，并将脱模剂施加到硅树脂表面。随着另一次浇注和固化，模具准备好铸造新零件。

虽然简单，但如果没有使用正确的设备或跳过一些步骤，你最终会得到一个充满气泡或表面不规则的无用模具。这有助于了解如何获得如此大规模的模具(13 英寸 x 19 英寸！)完成的毫无瑕疵。我们之前已经介绍过[Eric]的作品，[比如他对 Apple Pencil 的细节改进](https://hackaday.com/2020/06/18/one-percenter-mods-for-premium-apple-pencil-usability/)。休息后的视频。
