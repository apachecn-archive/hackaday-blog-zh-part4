# 老式晶体管收音机获得互联网移植

> 原文：<https://hackaday.com/2019/10/21/vintage-transistor-radio-gets-internet-transplant/>

晶体管的发明使无线电发生了革命性的变化，使得接收机比以往任何时候都更加紧凑和便携。在 20 世纪中叶，这种设备大受欢迎，袖珍晶体管收音机风靡市场。[MisterM]对这样的时光有着美好的回忆，当他在一次汽车行李箱拍卖中发现一台 20 世纪 70 年代的调情收音机时，它导致了一个可爱的小建筑。

收音机被剥去了原有的硬件,[MisterM]更喜欢互联网广播而不是地面广播。在它的位置上，安装了一个树莓派零。它配备了一系列现成的模块，使其适合播放音乐。扬声器 PHAT 用于音频，而 Adafruit Micro Lipo 板处理电池充电，Pimoroni Lipo Shim 用作电源。所有这些都被捆在原来的箱子里。

这台收音机的控制装置非常精巧。原来的音量和调谐盘被拆下，切成薄片，粘在两个杠杆微动开关上。这使得它们可以充当按钮。一个新的电源开关被安装在原来的后面，扬声器 PHAT 的发光二极管被放在调谐盘的后面，作为一个基本的显示器。

这是一个整洁的建筑，将一个有吸引力的古董人工制品重新变成了一个互联网时代的设备，同时在外观上保持不变。[MisterM]在这个领域有自己的形式，他的[老式无线电控制台](https://hackaday.com/2019/08/27/vintage-car-radio-now-plays-games-and-chiptunes/)和[视频门铃是特别的亮点。](https://hackaday.com/2019/05/13/video-doorbell-focuses-on-quality-aesthetic/)休息后的视频。
