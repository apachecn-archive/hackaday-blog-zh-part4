# 把残羹剩饭留在身边

> 原文：<https://hackaday.com/2021/08/21/keep-scraps-around/>

当我在建造东西的时候，我喜欢手头有一大堆废料。因为当我要测试一些东西时——这种胶水会粘到这种织物上吗？如果我敲击它并拧入一个螺钉，它会不会断裂？如果用碳纤维和环氧树脂加固，它会不会断裂——手头有一些有问题的材料只是为了实验，这很好。所以我从废料堆里拿出一大块！

但是废料堆不可能永远膨胀，我们都知道“好事过犹不及”是一件事，对吧？废料堆需要不断修剪。你实际上不需要超过几个铝挤压切断，所以当你开始建立过剩的库存，是时候废弃它了。我是说，扔掉它。

我最近才开始意识到的一个必然结果是，如果我限制我正在使用的材料的数量，就可以更好地控制废料堆。这是简单的数学。如果我用二十种不同的材料，那就是二十堆不同的废料。但是如果我可以用一个重量的玻璃纤维来做所有的事情，那一堆废料可以做两倍或三倍的工作。还有一个额外的好处是，我已经知道材料是如何工作的，甚至可能手头有旧的测试样本。

事实上，我是一个废料狂热者，以至于开始用一种新材料工作而还没有一堆废料堆积起来几乎是痛苦的。我总是不愿意为了测试某样东西而切割成一个漂亮的正方形。但这也是生命大循环的一部分。由于没有事先测试，我几乎肯定会搞砸，并从我原本希望成为成品的东西中产生废料。看到了吗？没问题！下一版本。

你怎么想呢?废料、下脚料和它们的近亲——试件和样品——值得留在你的店里吗？你有严格的方法吗，或者你只是把他们扔在角落里？每个项目都进行清理，还是只有当堆积如山的 XPS 泡沫达到您的头顶时才进行清理？
