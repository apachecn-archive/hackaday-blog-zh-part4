# 在覆盆子派的帮助下，PDA 原型恢复了记忆

> 原文:[https://hacka day . com/2021/08/26/proto-PDA-借助树莓-pi-pico/](https://hackaday.com/2021/08/26/proto-pda-regains-its-memory-with-the-help-of-a-raspberry-pi-pico/) 恢复记忆

还记得心灵感应组织者吗？如果你这样做了，你很可能是一个早期采用者，因为 8 位掌上电脑在 20 世纪 80 年代中期达到了全盛时期。当然，自那以后事情已经有了很大的进展，但是树莓派 Pico 可以代替心灵感应的原始内存包这个事实很好地说明了进展有多快。

像许多早期将电脑放入口袋的尝试一样，Psion II 有可移动模块，被称为“数据包”。最早版本的 Datapak 只不过是一个小 PCB 上的 EPROM 芯片，当时的技术限制加上寻址存储器的古怪方式使得[Amen]有可能使用现代微控制器模仿 data pak。

第一个版本是一个分线板，大大超出了 Datapak 插槽，带有一个微型有机发光二极管显示器、SD 卡插槽和一堆按钮。这个原型证明了皮科确实足够快以至于可以欺骗心灵术士认为一个合法的数据插件被插入了。[阿门]后来改进了设计，制作了一个板，可以把所有东西都塞进 Datapak 插槽，除了有机发光二极管仍然悬在可以看到的地方。他在下面的视频中测试了人造记忆。

很高兴看到像心灵感应这样的突破性技术被妥善处理并重新使用。我们以前见过其他人尝试；这里有一个黑客使用 Pi 通过 RS-232 串口[将一个心灵术士组织者连接到互联网](https://hackaday.com/2019/08/03/raspberry-pi-helps-vintage-psion-find-its-voice/)。
