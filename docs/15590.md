# 寻求自制液氮的 DIY 脉管制冷机

> 原文:[https://hacka day . com/2023/01/01/a-DIY-pulse-tube-sloccooler-in-the-quest-the-home-made-liquid-nitrogen/](https://hackaday.com/2023/01/01/a-diy-pulse-tube-cryocooler-in-the-quest-for-home-made-liquid-nitrogen/)

如果你需要液氮，但你不想简单地从当地供应商那里订购，该怎么办？在这种情况下，你可以建立自己的脉管制冷机，因为[超空间海盗]正在 YouTube 上做。休息后，您可以使用直线电机抓住[部分 1](https://www.youtube.com/watch?v=GjRoThMyNGA) ，使用基于往复活塞的版本抓住[部分 2](https://www.youtube.com/watch?v=WOmjJFk8rl0) 。尽管还在进行中，第二个版本的低温冷却器设法将温度降低到零下 75 度。

[脉管制冷机](https://en.wikipedia.org/wiki/Pulse_tube_refrigerator)是用于创造冷却效果的多种系统之一。商用冰箱和冰柜倾向于使用[朗肯](https://en.wikipedia.org/wiki/Rankine_cycle)循环冷却器，因为它们成本低，并且在(相对)较高的温度下有效。对于低温环境，[斯特林发动机](https://en.wikipedia.org/wiki/Applications_of_the_Stirling_engine#Heating_and_cooling)被普遍使用，尽管它们在制冷方面也有一些用途。这三者有共同的元素，但它们在较大的温度范围内效率不同。

在这个视频系列中，观众将了解这些冷却器背后的物理原理以及阻止它们简单冷却到零开尔文的瓶颈。尽管脉管制冷机看似简单——只有一个活塞、一个回热器网和一些管道——但让它们工作良好是一种耐心的锻炼。随着它的发展，我们非常期待这个系列未来的视频。
