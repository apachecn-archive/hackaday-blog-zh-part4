# randomaut 设备会告诉您何时何地

> 原文:[https://hackaday . com/2021/01/28/randonaut-device-tell-you-where-to-be-when/](https://hackaday.com/2021/01/28/randonaut-device-tells-you-where-to-be-and-when/)

Randonauting 是在追求冒险的过程中，使用随机数生成一个要访问的目的地的消遣方式。当然，任何可以在网站上用脚本完成的事情，用定制的硬件会更酷，所以[Decker]为此搭建了一个平台。

该装置使用 USB 硬件随机数发生器，通过量子效应产生*真*随机数；至少，根据我们最好的宇宙理论。这些数字然后被用来选择一组随机的 GPS 坐标和到达那里的时间，这是对[Decker]自己创造的传统随机方法的一个有趣的扭曲。

它的核心是一个随机数生成器，通过一些 Python 脚本进行抽取。这个建筑提升自己的地方不在于机械，而在于表现。该装置在一个钟形玻璃罩内的树莓皮上运行，带有真空荧光显示器、仙女灯和管道组件。它符合赛博朋克美学，当时间和地点由一台神秘莫测的机器用发光的数字给出时，就很难忽视一个人的使命。

这看起来很有趣，尽管要小心随机的危险——一些参与者发现了比他们期望的更多的东西。这和[过去的地理哈希热没什么不同。](https://xkcd.com/426/)休息后的视频。
