# 3D 打印的 V8 发动机使用螺线管

> 原文：<https://hackaday.com/2018/11/25/3d-printed-v8-engine-uses-solenoids/>

通常，当你想到 V8 发动机时，你会想到由爆炸燃料驱动的活塞，推动曲轴。[米勒星球的] [版本](https://www.youtube.com/watch?v=DUSDV2c1Ne0)不使用活塞，而是使用螺线管和 3D 打印曲轴。成品将成为一个伟大的科学项目或课堂演示，展示曲轴如何将往复直线运动转化为旋转运动。

有很多 3D 打印的零件，链接在帖子里。很多视频(见下文)是以无言的工作室风格拍摄的，只有一些文字覆盖来解释正在发生的事情。但是在中间，你会听到关于螺线管如何产生力的解释。真正的回报是在最后，当你看到这个奇妙的装置在运动的时候。

有一个由两部分组成的换向器，看起来像是给螺线管排序。这里涉及到一些弹簧，但是一个图表可能会使这部分更清楚。如果你有可以处理线圈电流的输出电路，用 Arduino 实现点火时序控制并不困难。大多数真正的电子点火发动机也有一个曲轴传感器，以确定轴何时完成完整的旋转。

我们在过去的中已经看到过一些类似的[版本，我们甚至看到它们是用](https://hackaday.com/2016/02/19/home-made-solenoid-motor/)[的旧硬盘](https://hackaday.com/2015/02/21/spin-up-an-old-hard-drive-with-a-solenoid-motor/)完成的。如果你想知道为什么你看不到这些被实际使用，那是因为它们的效率太低了。相对于你能获得的输出功率，它需要大量的输入功率。如果你喜欢你的活塞是基于 LED 的，你总是可以[建造一个时钟](https://hackaday.com/2018/02/11/4-stroke-clock-fires-on-all-cylinders/)。
