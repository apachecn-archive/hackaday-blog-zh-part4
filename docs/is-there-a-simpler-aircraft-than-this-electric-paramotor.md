# 还有比这种电动动力伞更简单的飞行器吗？

> 原文：<https://hackaday.com/2021/09/22/is-there-a-simpler-aircraft-than-this-electric-paramotor/>

乘坐飞机的梦想可能已经吸引了我们中的许多人，但对大多数人来说，它仍然遥不可及，因为围绕动力飞行的安全，监管和培训框架使它不是胆小的人的努力。[Justine Haupt]可能已经交付了最简单的动力飞机，她的[软式飞艇驱动](https://www.justine-haupt.com/blimpdrive/index.html)，这是一个双螺旋桨电动附加装置，用于她的滑翔伞装置，允许她自我发射，并在翱翔时维持飞行。

它采用碳纤维管的形式，两端用 U 型螺栓固定着大型无人机发动机和道具，在 3D 打印的碳纤维中间有一组支架，电池和滑翔伞背带连接在支架上。整个装置重量轻，噪音小，由于有两个对转螺旋桨，它也不存在影响单螺旋桨飞机的扭矩问题。

在 Hackaday，我们不是飞行员或滑翔伞爱好者，所以我们对使用中的飞行器的印象不是来自飞行员的视角。但它的简单和易于进入空中看起来是其他任何东西都无法比拟的，我们不得不承认有一丝嫉妒，因为在休息时间下面的视频中，她飞过了海滩，那是她的测试场地。

如果你从过去的 Hackaday 文章中认出了贾丝汀，那你就对了。可能最令人难忘的是她的旋转式手机。
