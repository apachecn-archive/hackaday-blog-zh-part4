# 逆向工程被遗忘的 Konami 街机硬件

> 原文：<https://hackaday.com/2022/06/28/reverse-engineering-forgotten-konami-arcade-hardware/>

当全 3D 视频游戏在上世纪 90 年代初开始出现时，一些公司比其他公司对这种变化准备得更充分。事实上，在 3D 虚拟空间感觉自然之前，需要将近十年的实验。即便如此，Konami 似乎在这个时代开始时首次涉足 3D 街机游戏，搬起石头砸了自己的脚。[Mog]向我们展示了这些平台的来龙去脉[，同时试图通过 MAME](http://blog.polygonal-moogle.com/mame/poly-go-not/) 让它们起死回生。

这些街机是第一批拥有全 3D 环境的产品，但与其他公司的产品相比，动力不足，即使是在当时。它们只包括一个数字信号处理器，其任务是计算所有的场景几何图形，而竞争机器将使用多个 DSP 芯片来完成相同的工作。因此，分辨率和帧速率非常低。尽管如此，【Mog】还是着手让它在 [MAME](https://en.wikipedia.org/wiki/MAME) 运作起来。

为了完成这项任务，[Mog]在 90 年代早期转向为 Konami 开发人员提供的一套开发工具，这些工具将在当时的 PC 上模拟该系统。令人惊讶的是，它仍然可以在 Windows 10 上工作，只需稍加调整，并且通过几十年来在 MAME 工作的其他人提供的一些其他工具，这些旧的 Konami 机器在这个模拟器的支持下获得了一些新的生命。

并非一切都完美，但[Mog]报告称，大多数错误和其他问题最近已经解决，或者正在由该领域的其他专家积极解决。如果你还记得 80 年代和 90 年代初街机时代的这些游戏，也许是时候拿起一台旧的 CRT 显示器再次启动了。
