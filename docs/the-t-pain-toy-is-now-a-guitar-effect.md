# T-Pain 玩具现在是吉他效果

> 原文：<https://hackaday.com/2019/01/09/the-t-pain-toy-is-now-a-guitar-effect/>

T-Pain 是来自佛罗里达州的说唱歌手，他通过创造性地使用自动调优效果而出名。直到今天，还没有人像 T-Pain 这样做，但随着“我是 T-Pain”麦克风的发布，世界各地的孩子都有了机会，这种麦克风可以影响用户的声音，使他们听起来尽可能地飞，不包括电池。本着音乐探索的精神，[Simon]认为将效果变成吉他踏板会很有趣。

最初的计划是将麦克风连接到输入插孔，将扬声器连接到输出插孔，但事情并没有那么简单。这个玩具已经带有一个线路输入和一个耳机插孔，但布线方案很奇怪，在某些情况下，其中一个输入也可以作为输出。[西蒙]采取了厨房水槽的方法，在电路上扔了一堆插孔，把它们都放在一个踏板箱里，上面有一些旋钮可以旋转一些参数。

当吉他信号输入时，最终的结果是一个颤音，高保真颤音。这与原始玩具的声音非常不同，但当与活套一起使用时，会让我们想起[反鹦鹉螺踏板](https://hackaday.com/2012/08/23/playing-with-dsp-and-building-a-guitar-pedal/)。休息后的视频。
