# 掌上旋转打字机

> 原文：<https://hackaday.com/2022/10/18/playdate-handheld-turned-typewriter/>

Playdate 是一个有趣的游戏系统。这是一款手持设备，黑白屏幕，从表面上看，它让我们想起了最初的 Game Boy，甚至是它的按钮布局。但是它有第二个控制器从侧面弹出，这个控制器是一个曲柄，整个系统是由制作*无题鹅游戏*的同一批人制作的，这一事实让我们很感兴趣。显然它也对其他人产生了影响，[因为这个项目把游戏系统变成了打字机](https://github.com/t0mg/playwrite)。

Playdate 没有对 USB 附件的原生支持，除非它被插入这个定制的 3D 打印坞站。dock 内部是一个 Teensy 4.1，它处理键盘和控制台之间的一些转换。一旦 dock 处理好了，文本编辑器也需要加载到设备上。文字处理器能够移动光标，插入和删除文本，该项目的创建者[t0mg]计划在未来的版本中添加更多功能，如支持多文件，改变字体，以及其他一些功能。

对于任何有兴趣重新创建这个项目的人来说，所有可打印的文件、文本编辑器和原理图都可以在 GitHub repo 中找到。对于一个不太知名的主机来说，这是一个令人印象深刻的项目，我们还没有看到许多其他的黑客，除非你把这个一次性的 Arduboy 项目[计算在内，它从 Playdate 的曲柄控制器](https://hackaday.com/2019/07/02/the-arduboy-gets-a-crank-mod/)中获得了一些主要的灵感。
