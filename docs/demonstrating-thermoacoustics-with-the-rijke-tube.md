# 用里克管演示热声学

> 原文：<https://hackaday.com/2021/10/05/demonstrating-thermoacoustics-with-the-rijke-tube/>

Rijke 管是一个非常简单的装置，非常清楚地展示了 T2 热声学的原理。构造非常简单，只需将一个金属网放在管子的底端，大约四分之一的位置，给金属网加热，就能发出声音。热量产生对流，形成纵向驻波。这是因为空气经过热纱布时，突然膨胀并引起压力变化，从而冲出管子。接下来，气流冷却并减速，空气开始回到管道中，如此循环往复。通过在管上滑动一个套管来调整管的长度，可以调整音符的音高，这仅仅是因为空气传播的距离不同。如果有火焰从下面对准纱布，声音将停止，因为当它碰到热纱布时空气已经是热的，没有压力变化发生，也没有振荡。

正如[Keith]，发送提示的读者，建议将伺服系统连接到电子管的套管上，建立多个单元，并将整个系统挂在 MIDI 控制器上，这将会很有趣。这可能会带来一些有趣的时光，我们不得不同意。保持纱布热的问题可以用很多方法解决，直接电阻加热可能有效，但也许感应加热会更干净？

现在，我们找不到一种以这种方式工作的乐器，这听起来像是为某个有这种倾向的人设计的一个黑客。

这些年来已经有一些以火为导向的音乐设备，比如这个 [Rijke 管风琴](https://www.youtube.com/watch?v=qdII8sMYinY)，各种各样的焦斯通，包括[这个用特斯拉线圈表演的简洁的音乐设备](https://www.youtube.com/watch?v=mktgrGmVphY&t=40s)，当我们谈论音乐火的时候，一个[二维鲁本斯管风琴变体](https://www.youtube.com/watch?v=2awbKQ2DLRE)怎么样？
