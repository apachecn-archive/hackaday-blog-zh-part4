# 飞机无线电无屏蔽

> 原文:[https://hackaday.com/2020/11/11/aircraft-radio-bares-all/](https://hackaday.com/2020/11/11/aircraft-radio-bares-all/)

老式电子设备有某种魅力。重金属底盘和明显的手动线束可以是一件艺术品，即使它们对于大多数现代装备来说在经济上不切实际。看着[msylvain 59]拆掉一个[柯林斯 51R VOR 接收机](https://www.youtube.com/watch?v=_h2n649oStY)就是一个很好的例子。建筑看起来很坚固。

如果你不熟悉 VOR，它代表 VHF 全向范围，允许飞机调谐到固定的地面信标，并确定其相对于信标的方向。在某些情况下，它还可以计算距离。

几年前，VOR 天线实际上是快速旋转的，但今天它更有可能是电切换的天线阵列。无线电波束有几种编码方案，包括允许飞机接收器(如 51R)确定信标方向的 FM 参考信号。如果你知道两个灯塔的方向，你就能知道你在哪里。即使只有一个灯塔，如果你保持稳定的航向，你也可以保持航向接近或远离它。

像柯林斯号这样的接收器可以驱动飞机驾驶舱上的航向偏差指示器(CDI)、水平位置指示器(HSI)和无线电磁指示器(RMI)，还可以向自动驾驶仪提供输入。与现代齿轮相比，这些部件看起来很大。

了解一架飞机在哪里一直是一件大事，有很多方法可以应对。[应答器](https://hackaday.com/2020/06/22/just-how-do-aircraft-transponders-work-anyway/)是等式的一部分。我们也看到许多与 [ADS-B](https://hackaday.com/2019/08/14/remote-ads-b-install-listens-in-on-all-the-aircraft-transmissions-with-rtl-sdr-trio-phones-home-on-cellular/) 相关的黑客攻击。
