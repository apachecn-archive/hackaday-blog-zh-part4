# 让我旋转吧，宝贝:为凡人做的发电机制造实验

> 原文：<https://hackaday.com/2019/08/13/spin-me-right-round-baby-generator-building-experiments-for-mere-mortals/>

你们中有多少人计划在明年建造风力发电机？好了，你们两个都可以把手放下了。即使你不想手动缠绕线圈，学习发电机的[原理可能会激发你的兴趣。设计一个商业模型需要大量的数学计算，但是如果我们通过一次查看一个组件来处理一个简单的版本，就更容易理解了。](https://www.kjmagnetics.com/blog.asp?p=wind-turbine)

在这次冒险中，[K&J Magnetics]从解剖一台商用发电机开始。他们选择了一个简单的版本，可以很好地服务于营地，所以没有传输或叶片角度设备来使事情复杂化。这是你所期望的部件，一个转子和一个定子，一个带有永磁体，另一个带有线圈。

这个项目的乐趣在于复制商用硬件中的元件，改变绕组和线圈数，看看它如何影响性能。如果你曾经将电磁线缠绕在钉子上制作电磁铁，你就会知道这是一项繁琐的工作，所以看看他们的 3D 打印线圈架，内置磁铁来触发绕组计数，还有一个插座来安装在缝纫机绕线器上。如果你要做一堆线圈，这将节省头痛和手腕肌腱。

他们使用迭代过程来演示发电机上多个线圈的效果。第一次试运行仅使用了三个线圈，但根本没有产生多少电力，即使用电钻旋转也是如此。六个绕组更好，但是十几个绕组最终也能达到目的，即使是用手转动发电机。我们不知道他们使用廉价的硅二极管，虽然，这似乎是无意的蹒跚，但我们跑题了。

制作[涡轮叶片](https://hackaday.com/2011/10/08/sustainability-hacks-wind-turbine-generator/)也不一定是一件痛苦的苦差事，聚氯乙烯可能是那里的门票，你也可以考虑[垂直轴风力涡轮机](https://hackaday.com/2008/09/28/diy-vertical-axis-wind-turbine/)，它在露台上更安全。现在，你们这些建造发电机的人，记得[给我们提示](https://hackaday.com/submit-a-tip/)！
