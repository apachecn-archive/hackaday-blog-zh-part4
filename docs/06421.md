# 查看全国天气一瞥

> 原文:[https://hack aday . com/2020/05/03/viewing-country-wide-weather-at-a-glance/](https://hackaday.com/2020/05/03/viewing-countrywide-weather-at-a-glance/)

在他的最新项目中，气象显示爱好者[Richard]组装了一个漂亮的小装置，显示遍布英伦三岛的九个不同机场记录的温度。当然，这个概念可以适用于你称之为家的任何地方，假设该地区有足够的联网气象站来填写地图。

电子元件非常少，由 NodeMCU ESP8266 开发板、几个七段 LED 显示模块和一个简单的电源组成，组装在一块废弃的 perfboard 上。如您所料，代码也相当简单。它只需要从一个在线 API 上拉下温度，然后点亮显示屏。这个项目的特别之处在于演示。

[![](../Images/feefd813d56b96fd6ab6523d557133cc.png)](https://hackaday.com/wp-content/uploads/2020/04/uktemp_detail.jpg) 正如[Richard]在广告之后的视频中所展示的，关键是一片经过打磨的丙烯酸树脂，它可以漫射 42 个 led 的光，这些 led 被精心安装在玻璃边缘钻的孔中。结合印刷的覆盖表，这在弱光条件下照亮地图及其图例。这是一个简单的技术，不仅看起来很棒，而且让显示屏无论白天还是晚上都易于阅读。这绝对是一个值得记住的技巧，因为它在这个特定的构建之外有很多可能的应用。

通过他的项目，[Richard]已经证明了自己是独特且数据丰富的天气显示的大师，而[则是标志性的七段 LED 显示](https://hackaday.com/2019/10/25/24-hours-of-temperature-data-at-a-glance/)的狂热爱好者。虽然他的特定品牌的气候数据超载可能不适合每个人，但你不得不佩服他可视化数据的诀窍。
