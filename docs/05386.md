# TRS-80 克隆人使用现代部件

> 原文：<https://hackaday.com/2020/01/19/trs-80-clone-uses-modern-parts/>

在 RadioShack 认为电子产品商店的最佳商业模式是骚扰客户购买高价电池和手机之前，它是大多数分立元件、旋钮、电阻、无线电设备甚至流行电脑的一站式商店。那台计算机，TRS-80，在逆向计算领域很受欢迎，如果你找不到原始部件来还原一台，你总是可以制造你自己的克隆体。

这个造型来自于[Glen]又名[glenk]，他以逆向计算造型而闻名，比如一年多前我们展示的经典宠物，这个 TRS-80 是他的最新项目。他对硬件也很感兴趣。这不是一个 FPGA 或树莓派运行 TRS-80 的外观相似的硬件。[Glen]使用现代 CMOS 元件完全重新设计了计算机，以制作 RadioShack 经典的现代、功能完善的复制品。

由于[Glen]的详细程度，这是任何对计算硬件(相对于软件，你可以通过更简单的仿真器了解)和逆向计算感兴趣的人的必读之作，并且也将我们大多数人带回了一个更怀旧、更简单的时代，在那个时代， [RadioShack](https://hackaday.com/2018/07/23/the-return-of-radioshack/) 之旅是有趣和有趣的。
