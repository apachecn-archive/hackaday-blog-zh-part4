# 六角灯是胶合板的时尚应用

> 原文：<https://hackaday.com/2019/02/21/hexagonal-lamp-is-a-stylish-application-of-plywood/>

灯具非常有用，可以为房间增添风格和照明选择。顶灯必须为各种任务提供足够的照明，当需要休息和放松时，一盏精致的台灯可以为房间增添一抹忧郁的光芒。通常，一个时尚的灯可能会因为在电源线上挂了一个普通的塑料开关而让人失望，但也不总是这样。[【Emiel】设计了这款带有隐藏开关的六边形灯，效果非常好。](https://www.youtube.com/watch?time_continue=1&v=9CkBbQPKDc8)

[Emiel]首先在胶合板和有机玻璃板上铺设六边形纸模板。胶合板是在带锯上切割的，而有机玻璃的内部切割是在卷轴锯上进行的，以避免难看的切割入口线。灯的外半部分在一对钢棒上上下滑动。弹簧支撑着灯的外半部分，按下它就可以启动里面的开关来开关灯。

该建筑具有干净和吸引人的美感，led 隐藏在内部，通过内置在机身中的有机玻璃片发光。这看起来就像你在[宁静基地酒店&赌场](https://variety.com/2018/music/reviews/album-review-arctic-monkeys-tranquility-base-hotel-casino-1202805259/)的房间里找到的东西。然而，[如果普通灯具对你来说还不够，你可以考虑打造一些互动的东西。](https://hackaday.com/2016/03/01/building-an-interactive-led-lamp-to-annoy-yourself/)休息后的视频。
