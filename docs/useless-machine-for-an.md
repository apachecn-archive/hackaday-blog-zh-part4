# 生存困境的无用机器

> 原文：<https://hackaday.com/2020/05/27/useless-machine-for-an/>

没有什么项目比一台无用的机器更能陷入生存困境，因为它们一旦通电就会自动关闭，从而真正说明了存在的无用性。通常这是通过一个简单的开关来完成的，但对于一些可以真正熄灭灯光，然后重新照亮它们的东西来说，[詹姆斯]的最新项目是[一个用蜡烛](https://github.com/squirtifier/The-Conflicted-Candle)来完成这项工作的无用机器。

该项目由安装在一组齿轮上的两个臂组成。一个臂上有一个打火机，另一个臂上有一个安装在伺服电机上的阻火器。随着齿轮的转动，打火机越来越靠近蜡烛芯并点燃它，然后整个组件旋转回来，这样阻火器就可以熄灭火焰。一切都是围绕 Arduino Nano 构建的，Arduino Nano 是一个为 Pitman 齿轮电机供电的电机驱动器，以及一组向微控制器提供位置数据的霍尔效应传感器。

如果你有心情在自己家里为存在而焦虑，那么[James]已经在他的 GitHub 页面上提供了项目文件。我们总是欣赏这里无用的机器，尤其是像这样独特的设计，它很容易让人意识到点蜡烛是徒劳的。
