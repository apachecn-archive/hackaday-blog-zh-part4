# 立体平版印刷大获成功

> 原文:[https://hackaday.com/2019/10/31/stereolithography-goes-big/](https://hackaday.com/2019/10/31/stereolithography-goes-big/)

说到业余水平的 3D 打印，我们大多数人都使用热端沉积的塑料细丝。几乎所有其他的都在使用立体平版印刷术——将光线投射到光敏树脂上。细丝打印机的典型制造量从 1，000 到 10，000 立方厘米不等，甚至更大也不罕见。相比之下，SLA 打印机通常要小得多。1,200 CC 的 SLA 打印机是典型的，便宜的打印机有时只有 800 CCs。也许这就是为什么[3D 打印书呆子](又名[Joel])很高兴能得到一个建筑面积超过 17，000 立方厘米的[peoply Phenom](https://www.youtube.com/watch?v=D1h968apFaw)。你可以看看下面的视频评论。

他声称它甚至比 Formilab 3L 还要大，尽管根据我们的数学计算，它的建造量约为 20，000 立方厘米。另一方面，人造大理石最长的尺寸是 40 厘米，比 3L 长 6.5 厘米，所以也许这就是他的意思。不管怎样，打印机很大。这几乎是 16 英寸，即使对于灯丝打印机来说也很大。不管哪一个更大，Peopoly 肯定要便宜得多，大约在 1800 美元左右，而 3L 的价格接近 10000 美元。

你可以在视频上看到这台机器很大。它也没有包装得很好，并采取了一些外观损坏。由于内部有一个巨大的风扇，它的声音也很大。不过，照片看起来不错。

尤其是，看着巨大的精灵灯从树脂浴中生长出来，令人印象深刻。[Joel]发现他在小型打印机上使用的一些技术在处理这么大的打印时必须放大。经过一些整理和粉刷，[Joel]最终完成了一盏不缺少印刷细节的超棒的灯。

如果你想知道灯丝印刷和树脂印刷的区别，我们可以帮助你。如果 1800 美元对你来说还是太贵了，你可以用几百美元买一辆[，但是要有 750 CC 的容积](https://hackaday.com/2019/08/05/resin-printers-are-now-cheaper-still-kind-of-a-hassle/)。
