The demogorgon just wants to be friends. See? He’s waving hello.

本周，主编埃利奥特·威廉姆斯和作业编辑克里斯蒂娜·帕诺斯(Kristina Panos)用汗水换来了沉默，在八月的炎热(太吵了)中，在他们各自的阁楼办公室里录制。我们决定本周没有缺乏政治倾向的真正新闻，除了 Winamp 可能带着四年来制作的新版本回来。(Winamp 有分裂性吗？)现在还用鞭子抽美洲驼的屁股吗？你是法官。

在 Elliot 给 Kristina 上了一堂关于 3D 打印机喷嘴尺寸增加面积的简短数学课后，我们谈了一点 3D 笔，对一个真正可定制的 macropad 垂涎三尺，该 macro pad 为每个按键开关使用了一个微控制器，并讨论了树木测量仪和树木健康。然后，它又回到了键盘，这是一个令人难以置信的模块化构建，带有电子墨水显示屏和触觉反馈旋钮，很快就会开源。

最后，我们谈论微型阴极射线管，一个 USB 驱动器，必须通过默默无闻的终极安全，讨论逆时钟的优点，并大声怀疑跳 PCB 的效用。先别急着反驳我们——直到你听说我们的第一个电子产品获奖，并了解到克里斯蒂娜在炎热的天气里不会做的一件事。
