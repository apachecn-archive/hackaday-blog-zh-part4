# 打造一盏灯，帮助缓解季节性抑郁

> 原文:[https://hack aday . com/2022/12/13/build-a lamp-to-help with-seasonal-depression/](https://hackaday.com/2022/12/13/building-a-lamp-to-help-with-seasonal-depression/)

太阳在我们的生活中扮演着重要的生物角色，调节着我们的生理节奏，还做了很多其他的事情。一些人发现冬季日照减少会对情绪产生负面影响，并依赖人工照明来抵消这种影响。为了这个目的，他们自己设计了一盏灯。

这盏灯是围绕一个用风干粘土制成的抽象雕塑造型而建造的。光线由一串 Neopixel RGBW LEDs 提供。由 Adafruit Feather Huzzah 运行，它们被编程为随着日出而触发，在灰色的日子里提供明亮的光线，当外面的世界不完全相同时。Adafruit 通过查询在线天气 API 来获取每天正确的日出时间，无需用户干预。该灯也可以被编程为在以后的时间里提供温暖的光。

我们之前展示过一些昼夜节律照明项目，包括一个用智能灯泡实现全屋效果的项目。对一些人来说，它们是帮助提升情绪和保持稳定睡眠周期的宝贵方式。休息后的视频。
