# 宇宙射线翻转位，辅助马里奥 64 Speedrunner

> 原文:[https://hack aday . com/2021/02/17/cosmic-ray-flips-bit-assists-Mario-64-speed runner/](https://hackaday.com/2021/02/17/cosmic-ray-flips-bit-assists-mario-64-speedrunner/)

我们都听说过宇宙射线在这里和那里翻转比特，但总的来说，这种情况很少发生，所以我们在日常生活中并不太担心。然而，在一场比赛中，似乎正是这样一束光线击中了帮助一名短跑运动员的关键部位。

翻转发生在[DOTA_Teabag]身上，他突然发现马里奥向上飞到了一个关卡的更高部分，完全出乎意料。[由【pannenkoek 12】](https://www.youtube.com/watch?v=X5cwuYFUUAY)进行的测试似乎表明，这可能是由于 Mario 的身高值从 C5837800 到 C4837800 的单个位变化，导致勇敢的意大利水管工在关卡中向上弯曲。主流理论认为这种比特翻转是由宇宙射线事件引起的，尽管这种事件发生的可能性非常小。

可能还有另一个原因导致了这种翻转，尽管在社区中做了大量的工作来模拟这种情况，但没有发现任何原因。其他建议涉及导致翻转的电气噪声或其他故障，尽管人们很少会认为这种情况只会改变 RAM 的一个位。目前，还没有定论，但谁知道呢——也许将来我们会发现这是一个隐藏的、未被发现的漏洞。当然，如果任天堂不能让你继续下去，试试运行 Windows 95 的速度。休息后的视频。
