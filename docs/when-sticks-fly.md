# 当棍子飞起来

> 原文：<https://hackaday.com/2022/05/11/when-sticks-fly/>

说到业余爱好旋翼机，似乎旋翼越多越好。我们见过四轴飞行器、六轴飞行器和八轴飞行器，可能还有一架十二轴飞行器在某处嗡嗡作响。但是走另一条路呢？旋翼最少的旋翼机怎么样？

于是我们就有了[这种独特的“飞棒”双翼飞机](https://youtu.be/LehUH8Fx_Uk)。[帕韦·斯彼哈尔斯基]的创作让我们想起了一个微型版本的[“飞行床架”](https://www.nasa.gov/multimedia/imagegallery/image_feature_316.html)，美国宇航局曾用它来训练阿波罗登月飞船的飞行员，著名的是[【尼尔·阿姆斯特朗】在让飞船进入这个小机器发现自己所处的一些姿态后，将它从](https://youtu.be/tUJDbj9Vp5w?t=42)中弹出。这架双直升机是独一无二的，因为它的机身是碳纤维管，大约一米长，两端各有一个转子。为了控制扭矩，转子彼此反向旋转，每个转子都安装在一个伺服控制的万向架上，以实现推力矢量。控制电子设备和电池策略性地安装在管子上，使重心在转子之间的距离相等。

但是它能飞吗？是的，但只是勉强。下面的视频显示，它当然离开了地面，但在试图找到稳定的姿态时做了很多弹跳。[pawe]似乎认为万向伺服不够快，无法进行推力矢量调整，以保持驾驶杆飞行，我们不得不同意。

这不是[pawez]第一次涉足双翼飞机；他在 2018 年因他的同轴双直升机赢得了[“本周失败”荣誉。总的来说，飞棒似乎做得更好，他甚至成功地让它离开了地面，这是值得称赞的。](https://hackaday.com/2018/06/12/fail-of-the-week-two-rotors-are-not-better-than-four/)
