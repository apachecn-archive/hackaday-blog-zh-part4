# 魔眼管进入固态与这个插件的替代品

> 原文：<https://hackaday.com/2022/08/29/magic-eye-tubes-go-solid-state-with-this-plug-in-replacement/>

也许在真空管的全盛时期，没有什么比“电眼”指示管更能增加真空管电路的魅力了。凭借其圆形表面鬼魅般的绿色光芒，魔眼取代了更昂贵的动圈式仪表，如调音指示器和 VU 仪表。虽然今天它们可能越来越难得到，但不要担心——这种替代魔眼管的固态产品已经准备好为您的修复项目服务。

为了实现这一巧妙的构建，[高德·拉贝约翰]从最初的 6E5 和 6U5 电眼管开始，这些电眼管可能不再工作或变得太弱而看不见。阴极射线管的玻璃外壳被小心地从插座上切下，管的内部被丢弃，为位于两块印刷电路板上的替换电路腾出空间。一个矩形控制板容纳一个 LM3915 条形图 LED 驱动器芯片，而一个圆形显示 PCB 容纳 120 个表面贴装绿色 LED。圆形显示板安装在控制板的顶部并与之垂直，led 上方安装有一个漫射器。所有东西都被塞回原来的玻璃外壳和插座中，使其成为电子管的插件替代品。

效果相当有说服力，如下图视频所示。没错，即使有扩散器，你也可以看到一些单个 led 的证据，但老实说，这只是让显示器看起来更像眼睛的虹膜。我们真的很喜欢它的外观，我们很欣赏[Gord]投入的工作，尤其是文档。关于电子管如何工作的更多信息，请查看 [[Al Williams]的文章。](https://hackaday.com/2021/01/22/meet-the-magic-eye-vacuum-tube/)
