# 通过在 3D 打印中浇铸热胶来减轻应力

> 原文：<https://hackaday.com/2020/02/11/cheap-strain-relief-by-casting-hot-glue-in-a-3d-print/>

[Daniel Roibert]找到了一种为 JST-XH 连接器添加廉价应力消除的方法，这种连接器在爱好飞机的人们中更为人所知，是锂聚合物电池组上的充电和平衡连接器。他的解决方案是[在 3D 打印模具的帮助下，用热胶水浇铸它们](https://www.prusaprinters.org/prints/15081-hot-glue-sleeve-shell-mold-for-jst-xh-connector)。他的项目提供了适用于 2 到 8 个导体连接器的模具，所以只要选择合适的一个就可以打印出来了。[Daniel]说要在 PETG 印刷模具部件，这样它们就能承受融化胶水的温度。

3D 模型看起来不是特别直观，但一个教学视频让一切都变得清晰。首先在模具的内表面涂上一层脱模剂(像硅油这样的东西就可以了)，然后在底部涂上少量的热熔胶。接下来，将连接器放在胶水上面，涂上更多的胶水，将模具的顶部压入。顶部的小孔不是用来填充胶水的，是为了让多余的胶水在模具关闭时溢出。完全冷却后，只需打开模具(螺丝刀尖端的小切口使这变得容易)并修剪任何多余的部分。这就是全部了。

最后一件事:在下载中，你可能会注意到一个额外的模型。这种模具是分开提供的，因此人们可以根据需要拉伸中间部分，然后将它们合并在一起，从而制作出任意宽度的模具。毕竟，[有时 STL 文件并不完全正确](https://hackaday.com/2018/05/16/3d-printering-when-an-stl-file-is-not-quite-right/)如果出于某种原因共享 CAD 文件不是一个选项，那么提供可以更容易调整的 STL 是一种受欢迎的礼貌。你可以在下面观看一个短片，展示整个过程。
