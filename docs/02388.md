# 发动机以简单的方式转动铝

> 原文：<https://hackaday.com/2019/03/11/engine-turning-aluminum-the-easy-way/>

发动机车削，或 jeweling，或 guilloché，或一大堆其他名称，是一种在一块金属上打磨圆形图案的艺术。你可以在精美的手表上看到它，也可以在装饰艺术风格的金属外壳上看到它。【Ariel】决定探索这一技术，最终[用铅笔橡皮擦和牙膏](https://www.youtube.com/watch?v=uXPZ2COTYNc)取得了不错的效果。

这个过程从一块铝开始，在这个例子中是一个铝开源硬件标志。唯一需要的其他部件是一支 2 号铅笔、一些牙膏和任何种类的旋转工具，在这种情况下是一个钻床。将牙膏涂在要转动的工件上，并将铅笔放入卡盘中。之后就是在铝上画圈了。

顺便说一句，这正是真实世界中发动机转动和切割的方式。是的，工具是贵了一点，但是你仍然会看到一个有点软的工具将精细的磨料刮到一块金属上。发动机车削的诀窍在于在工件上获得一致的图案，这在数控机床上很容易做到。如果有人想把铅笔放进 CNC 路由器的夹头里，我们很乐意看到结果。
