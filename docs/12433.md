# 雅达利朋克游戏机，现在有了更多的真空管

> 原文:[https://hackaday . com/2022/01/11/the-atari-punk-console-now-with-more-vacuum-tubes/](https://hackaday.com/2022/01/11/the-atari-punk-console-now-with-more-vacuum-tubes/)

我们大多数人都见过雅达利朋克游戏机的声音荣耀，这种高保真合成器的经典化身是一对 555 定时器，以有趣的方式发出颤音和哔哔声。然而，我们中很少有人见过由真空管制成的 555 装甲运兵车。

对于普通读者来说，这一点并不奇怪，这是通过天兔电气公司的[大卫]告诉我们的，他还没有遇到一种电路不能通过在真空管中实现而得到改进。他的“空心状态”雅达利朋克游戏机从[开始，这是他不久前为了好玩而制造的 555](https://hackaday.com/2021/03/28/should-have-used-a-vacuum-tube-555/) 的 18 管版本，这被证明是足够受欢迎的，以至于他正在开发一个套件版本，其原型作为 synth 的第二计时器。32 根管子在跳楼者的鼠窝中闪闪发光，控制台设法发出必要的声音，但缺乏某种优雅。[大卫]然后大大简化了设计，将 BOM 减少到只有四个双三极管管。位于定制木箱中的 CNC 铣削 PCB 上，synth 做了一项令人尊敬的工作，看起来做得很好。下面的视频展示了这两个版本的运行情况，并详细介绍了它们的构造。

尽管真空管合成器可能很酷，但我们意识到并不是每个人都喜欢热玻璃方法。别担心——这里有大量的硅雅达利朋克游戏机可供选择。有一个内置在操纵杆中的[，一个配有迷你 CRT](https://hackaday.com/2012/11/07/atari-punk-stick-puts-a-synth-in-a-joystick/) 的[电路雕塑版本，甚至还有](https://hackaday.com/2018/12/29/freeforming-the-atari-punk-console/)[八个 APC 配合 MIDI 控制](https://hackaday.com/2015/02/07/modular-555-synth-is-controlled-by-midi/)。
