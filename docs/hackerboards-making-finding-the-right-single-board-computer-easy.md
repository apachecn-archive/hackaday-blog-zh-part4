# Hackerboards:让寻找合适的单板计算机变得容易

> 原文：<https://hackaday.com/2022/12/02/hackerboards-making-finding-the-right-single-board-computer-easy/>

单板计算机(SBC)的广泛可用性的伟大之处在于，它在 CPU 性能、GPU 功能、RAM 大小、I/O 选项等方面提供了如此广泛的选择。然而，这也是最大的问题，尤其是每年都有大量新功能的主板涌现。试图理解所有这些产品的是最近彻底改革的 Board-DB，也被称为 [Hackerboards](https://hackerboards.com/) 。

正如[【Martijn Braam】在关于变化的博客文章](https://blog.brixit.nl/finding-an-sbc/)中解释的那样，对旧黑客板的主要升级(我们在 2016 年报道过[)是一组更广泛的可以过滤的参数。这使得对细节特征的精细选择变得非常容易，这也反映在技术规格](https://hackaday.com/2016/07/14/hacker-friendly-sbcs-which-ones/)[比较特征](https://hackerboards.com/compare)中。有超过 450 个活动条目，有许多电路板已经可以过滤，但制造商被邀请采取联系，以添加更多的条目，这应该使列表保持最新。

顺便说一句，如果你想知道[Martijn]是如何获得这些漂亮的 PCB 照片的，[他写了一份完整的独立文章，详细介绍了他的相机设置](https://hackaday.com/2022/12/01/taking-good-pictures-of-pcbs/)。
