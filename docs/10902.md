# Arduino 发现宝藏

> 原文：<https://hackaday.com/2021/08/12/arduino-finds-treasure/>

海滩永远是一个放松的暑假目的地，是一个喝饮料、看书或在海里游泳的好地方。然而，对于那些需要更活跃的电子产品海滩活动的人来说，金属探测也总是一个受欢迎的选择。当然，有了 Arduino 和一些专业知识，就有可能[制造出一种金属检测机，它拥有你甚至可以从商业产品中得到的所有功能](https://hackaday.io/project/181020-diy-sensitive-arduino-ib-metal-detector)。

这个建筑来自[mircemk]，他在 Arduino Nano 周围建造了这个金属探测器，并使用一种称为感应平衡探测的方法来寻找金属。类似于雷达的工作原理，一个线圈发出信号，另一个线圈监听地下金属物体的反射。构建线圈并确定其共振频率是这一构建过程中最重要的部分，一旦解决了这一问题，系统的其余部分就可以得到完善，隐藏的宝藏也可以很容易地被挖掘出来。

这种构造的一个更有趣的特征是它能够区分黑色金属和有色金属，并且它可以检测 50 厘米以上距离的大型金属物体。还会有改进，因为[mircemk]计划增加传输线圈的功率，这将提高设备的范围。对于[mircemk]的其他一些金属探测器，请务必检查一下[这款探测器，它使用智能手机](https://hackaday.com/2020/05/02/a-smart-diy-metal-detector/)来帮助金属探测过程。
