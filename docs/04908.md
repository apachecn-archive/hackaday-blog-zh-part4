# 多材料 3D 打印制造软机器人

> 原文：<https://hackaday.com/2019/11/30/multi-material-3d-printing-makes-soft-robot/>

当你放大一个分形，你会发现它是由更多的分形组成的。也许这有助于启发哈佛 3D 打印机，它有各种混合喷嘴阵列。在下面的[视频中，你可以看到一些你可以用混合喷嘴阵列做的有趣的事情。我们认为最酷的是一个小型多足机器人，它利用真空在长凳上行走。然而，](https://www.youtube.com/watch?v=2WL4b03Tfjg)[报纸](https://www.nature.com/articles/s41586-019-1736-8)却在付费墙后面。

这里真的有两个想法。混合喷嘴并不新鲜。通常，您使用它们来模拟具有两个热端的打印机。也就是说，一次打印一种材料，当切换到新的灯丝时，清除旧的灯丝。这通常比使用两个头更简单，因为对于两个头的布置，两个头必须处于相同的高度，您必须知道两个头之间的精确偏移，并且您通常会损失一些打印空间，因为右头不能穿过左头，反之亦然。添加更多的头，你会增加这些问题。我们也看到混合喷嘴提供不同的颜色。

视频中的打印机可以放下不同类型的细丝，并实现非常快速的转换。据作者称:

> …利用多种粘弹性材料汇聚在一个结点时产生的类似二极管的行为，实现多达八种不同材料之间的无缝高频切换…

这应该在现有的业余爱好水平混合热端的范围内，尽管它可能需要不同于我们目前使用的材料。

除了机器人之外，还有一种类似折纸的折叠结构，它使用了刚性和柔性塑料。唯一让我们震惊的是，虽然喷嘴阵列可以快速打印出重复的结构，但似乎不可能关闭它们。因此，你可以用 8 头喷嘴打印的东西会受到喷嘴间距的限制，因为所有喷嘴都是一起打印的。

尽管如此，对于任何有混合爱好的人来说，这可能是一个很好的研究项目。这种功能甚至可以在[微型打印机](https://hackaday.com/2018/06/28/tiny-printers-get-color-mixing/)上使用。如果你想混合四种细丝，总会有[钻石](https://hackaday.com/2015/11/18/diamond-hotend-opens-the-color-gamut-for-3d-printing/)。如果你尝试这种方法得到了积极或消极的结果，写下来，用[提示](https://hackaday.com/submit-a-tip/)打我们，我们会把它传递下去。
