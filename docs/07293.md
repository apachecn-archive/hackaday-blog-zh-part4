# 台式工具的瑞士军刀

> 原文：<https://hackaday.com/2020/07/28/the-swiss-army-knife-of-bench-tools/>

[splat238]有一大堆备用传感器，要么是为一个单独的项目购买的，要么是一时冲动购买的，所以他知道他必须用它们做点什么。他决定[打造自己的数字多功能工具](https://www.instructables.com/id/Digital-Multi-Tool/)，专注于在车间环境中特别有用的传感器。巧合的是，他的灵感来自于我们不久前报道的[的一次黑客攻击](https://hackaday.com/2019/06/18/digital-multimeasure-helps-you-get-the-job-done/)。

他为他的设备配备了气泡水平仪、转速计、红外温度计、量角器、激光指示器和许多许多其他功能，这些功能将为任何黑客的工作空间带来巨大的增加。每个传感器都有一个很好的总结，这使得他的说明书在某种程度上成为硬件设计师常用检测模式的快速指南。转速表、温度计、激光笔和一些其他功能是我们之前强调的项目的显著升级。我们也很欣赏更大的显示屏，允许更详细的用户反馈，特别是在使用指南针和靶心数字水准仪等功能时。

[splat 238]构建中的组件数量太多，无法在本文中一一详述，所以请参阅上面链接的说明以了解所有细节。[splat238]为安装每个传感器制作了自己的 PCB，并在模块化设计方面做得很好，因此如果不需要，就不必添加某些组件。大多数元件采用通孔焊接，只有少数 0805 电阻需要。外壳被设计成使得用户可以用一只手操作工具，并且可以通过按下按钮在每个功能之间切换。

最后，该设备使用可充电锂聚合物电池供电，使其非常可重复使用。如果还没有足够的功能，电池可以通过 USB 或安装在外壳单元中的两块太阳能电池板充电。好吧，太阳能充电可能是一个不成熟的例子，但无论如何仍然是一个很酷的建筑。
