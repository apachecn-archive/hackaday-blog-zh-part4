# 使用这个合成器时，不要忘记你的薄荷糖

> 原文:[https://hack aday . com/2018/08/15/don ' t-forget-your-mints-when-use-this-synthesizer/](https://hackaday.com/2018/08/15/dont-forget-your-mints-when-using-this-synthesizer/)

虽然合成器在音乐世界中非常普遍，但它们并不像你想象的那样都是基于键盘的乐器。特别是如果你试图从合成器中获得一种特殊的感觉或声音，以模仿真实的乐器，可能有一种更好的风格合成器可供你使用。其中一种类型是呼吸控制器，这是一种专门构建的合成器，使用自然人的实际呼吸来模拟管乐器的声音。然而，现有的呼吸控制器可能会很贵，所以[Andrey]建造了他自己的。

为了制作合成器，[Andrey]使用了一根 [melodica 软管](http://www.hohnershop.com/melodica-tube-mouthpiece/)和连接到压力传感器的吹口。然后，他在一个定制的 Arduino 屏蔽上建立了一个电容电路，并将其全部插入 Arduino Mega(尽管他指出这有点过头了)。从那里开始，Arduino 需要被编程来充当 MIDI 设备并与压力传感器交互，他正在向管乐器合成器迈进。

合成器的美妙之处不仅仅在于它们能够[匹配现有乐器的外观和声音](https://hackaday.com/2014/09/18/a-diy-midi-wind-controller/)，而且还能做传统乐器无法做到的事情[，有时还能大幅降低价格。](https://hackaday.com/2017/07/26/maywa-denkis-nonsense-machines/)
