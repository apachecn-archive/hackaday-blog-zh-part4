# 廉价的电动等离子切割机

> 原文:[https://hack aday . com/2019/12/23/廉价的电动等离子切割机/](https://hackaday.com/2019/12/23/motorizing-a-plasma-cutter-on-the-cheap/)

如果你在加工金属板，手持等离子切割机是一个很好的工具，但它不太适合进行长时间或重复的切割。这就是为什么[workshop from scratch]发挥了他惯常的废料堆魔法，并[建造了他自己的机动化轨道来进行完美的直线切割](https://www.youtube.com/watch?v=HywS3uKJ8G8)。

[![](../Images/2d3b6f00d563053aeffe3fcc75e95c9b.png)](https://hackaday.com/wp-content/uploads/2019/12/plasma_detail.jpg) 大部分的车架，甚至是骑在上面的小卡车，都是用各种尺寸的方料做成的。几个轴承足以确保运动平稳，不会有太多的倾斜。运动由一根长螺杆和两个螺母提供，它们焊接在卡车的侧面。

如果你有耐心(和前臂力量),你可以在杆上放一个曲柄，然后就完成了，但在这种情况下,[从头开始工作]使用旧电钻的电机、变速箱和卡盘抓住螺纹杆，为他旋转。他在机架的一侧安装了一个外壳，里面装有电机、DC 电源和电机控制器，以及几个开关和一个控制速度的旋钮。

一项修改允许他用面板上的一个开关来启动等离子切割机，这给了设置一个更完整的感觉，而不仅仅是在扳机上放一个拉链。采用这种设计，等离子切割机本身仍然可以从支架上拆下来正常使用。如果你只是想在床上做手工切割，你甚至可以用几个螺栓拆下电动组件。

在休息后的视频中，眼尖的观众可能会注意到背景中一些熟悉的齿轮，如我们今年早些时候报道的[液压台钳](https://hackaday.com/2019/07/22/hydraulic-bench-vise-a-masterpiece-of-scrap-metal-and-angle-grinding/)。正如该频道的名称所暗示的，[workshop from scratch]是关于构建许多人认为理所当然的 workshop 工具，它们都是非常迷人的项目。虽然我们钦佩尝试用废弃花岗岩板制造车床的勇气，但 DIY 工具最终看起来几乎和商业产品一样好。
