# 柔软的液压肌肉作为一个团队举重

> 原文：<https://hackaday.com/2018/08/26/soft-hydraulic-muscles-lift-weights-as-a-team/>

使用液压系统通常意味着要有一个油箱和阀门。[conscious flush]取消了他的 DIY 液压人造肌肉的两个功能。取而代之的是，他使用了一对肌肉，两者都预装了液体。为了收缩其中一个，他将液体注入另一个，使那个膨胀，反之亦然。双向齿轮泵移动流体，同时也作为阀门。柔性材料取代了笨重的金属圆筒。

正如我们所说，这是一个 DIY 项目。他通过用芳纶纤维套管包裹硅树脂管来制造肌肉，增加了力量。两端的积木也是定制的。齿轮泵是他购买的，并进行了实质性的修改，包括移除油箱，在一端安装无刷 DC 电机。最后的定制件是电机的控制板。Gerber 文件、原理图和技术图纸以及进一步的细节都在他的 Hackaday.io 页面上。与此同时，看看下面视频中的负荷测试，肌肉分别举起和放下 5 公斤(11 磅)。

对 Hackaday 的搜索显示，液压人工肌肉可能很少见，所以这可能是许多人工肌肉中的第一个。例如，复制人类手臂肌肉如何一起工作，一个收缩而另一个扩张，怎么样？我们已经看到使用气动技术和[【詹姆斯·霍布森的】外骨骼手臂](https://hackaday.com/2014/08/30/homemade-superhero-james-diy-exoskeleton/)完成了这项工作。也许有人应该用这对柔韧的液压肌肉来做这件事？
