# 开着一辆遥控风扇车倒过来

> 原文：<https://hackaday.com/2021/02/03/driving-upside-down-with-an-rc-fan-car/>

我们都见过那些很小的遥控汽车，由于风扇产生的吸力，它们可以爬墙。他们的原理与气垫船的原理基本相反。[下班后的工程]想造一辆自己的遥控汽车，也能做同样的事情，[倒着开，产生巨大的抓地力。](https://www.youtube.com/watch?v=6BUtn0xp6AE)

这个版本是基于一辆 Traxxas 遥控汽车，但是为这个任务做了很大的改动。制作了一个底架，管道为一对双 50 毫米电风扇供电。裙边环绕底盘边缘，有助于形成密封，最大限度地提高下压力。这种裙部是需要大量工程努力的领域，因为它必须与地面形成良好的密封，特别是在小扰动的情况下，而不会产生过度的摩擦。悬挂组件相应地需要加强，以防止赛车在风扇系统产生的巨大下压力下触底。

经过多次试验后，问题解决了，汽车能够成功地倒着行驶。正如预期的那样，它产生的下压力[比【下班后工程】](https://hackaday.com/2021/01/03/active-aero-for-a-radio-control-car/)的早期机翼实验大得多——以更高的重量和复杂性为代价。随着计划创造一个钢筋混凝土汽车能够巨大的横向加速度，我们迫不及待地想看看接下来会发生什么。休息后的视频。
