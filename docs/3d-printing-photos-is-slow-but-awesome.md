# 3D 打印照片很慢，但很棒

> 原文：<https://hackaday.com/2019/03/05/3d-printing-photos-is-slow-but-awesome/>

从历史上讲，石版画是用蚀刻或模制工艺在瓷器上制作的图像，由于材料的厚度不同，在背光时图像是可见的。瓷器不是最容易处理的东西，但谢天谢地，对于我们现在的人来说，3D 打印机可以让一切变得更好。[【RCLifeOn】一直在试验印刷石版画，效果很好。](https://www.youtube.com/watch?v=XmbdcmvZbP8)

印刷好的石版画的诀窍就在于准备。选择灰度好看的图像很重要，因为这不是以任何方式复制颜色的过程。[RCLifeOn]然后讨论了打印机设置的细节，以获得好看的打印效果。层的高度应尽可能小，以避免可见的垂直条纹，并且石版画应在垂直方向印刷，以避免印刷因缺乏支撑而下垂。填充最好设置为 100%。最重要的是，打印机应该避免跨越打印的轮廓，以避免任何拉丝塑料文物破坏最终产品。

这是一个很好的指南，即使是一个 3D 打印新手，也可以帮助他以最少的麻烦创建一个伟大的打印。由于需要精细的细节，lithophane 可以成为一个很好的礼物，也是对印刷能力的一个很好的测试。我们以前也见过它们在奇妙的灯箱结构中被生产出来。休息后的视频。
