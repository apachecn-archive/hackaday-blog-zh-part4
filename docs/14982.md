# 通过 PS2 模拟器颠覆 PS4 和 PS5

> 原文:[https://hack aday . com/2022/10/03/通过 ps2 仿真器颠覆-ps4 和-ps5/](https://hackaday.com/2022/10/03/subverting-ps4-and-ps5-through-the-ps2-emulator/)

游戏控制台黑客仍然是一个迷人的领域，当有人为我们带来探索的战利品时，我们很高兴。这一次，我们正在看[cturt]的[mast 1c 0 re]hack story——努力寻找索尼 PlayStation 4 和 5 游戏机上的 PS2 仿真工具包中的漏洞，最终证明卓有成效。更重要的是，这个漏洞似乎是无法修补的——从技术上来说不是，但根据索尼的安全实践，这个模拟器属于他们在发现时拒绝修补的类别。

在这个故事中，我们踏上了 PS2 模拟器内部的旅程，浏览了已知的可利用的 PS2 游戏，并了解了一个潜在的切入点。绕过它，收集原始数据和小工具，绕过 ASLR，模拟器最终逃脱，一路上分享着宝贵的见解。作为演示，[cturt]成功地从 PS2 模拟器之外加载了一个不同的 PS2 游戏，并通过 WiFi 将其传输到 PS4 上！

我们现在正在不耐烦地等待 [Part 2，](https://cturt.github.io/mast1c0re-2.html)404——从这一点探索任意本机代码编写的可能性，以及描述索尼对此的反应——这已经在一年多前向他们披露了。这样的黑客往往会给我们带来大量的自制和仿真功能，能够在 PS5 上工作是件好事。如果你既不喜欢自制也不喜欢仿真，你可以考虑让你可能还未被充分利用的 PS4 [运行 Linux，或者](https://hackaday.com/2015/12/31/32c3-running-linux-on-the-ps4/)，甚至[把它变成 Linux 服务器！](https://hackaday.com/2022/02/10/turning-the-ps4-into-a-useful-linux-machine/)

我们感谢[瞌睡]与我们分享这个！
