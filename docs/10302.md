# 便携式数字记分板随处可用

> 原文：<https://hackaday.com/2021/06/07/portable-digital-scoreboard-goes-anywhere/>

这是一年中两个半球的时间——在天气变得难以忍受的热(或冷)之前，是时候出去玩了。无论你玩什么游戏，都不要在脑子里或者用石头来记分——像[LordGuilly]一样，给自己做一个便携式、可折叠的记分牌，走在显示技术的前沿。它真的更容易卷起来，而不是折叠起来，这很棒，因为你可以像老板一样展开它。

你只需要一个地方把它挂起来，你就可以走了。这个东西依靠一个强大的 10，000 mAH USB 电源运行，而且[LordGuilly]说即使在非常晴朗的日子里也很容易阅读。正如你可能已经猜到的，这些是 WS2812 条，它们被设置成矩形 PVC 条。这些杆在一个框架中彼此等距设置，该框架由电缆轨道的改进版本制成，该电缆轨道是用于电缆管理的塑料链节。

除了好看，我们特别喜欢这里有两个控制器选项。如果你想指定一个专门的记分员，有一个使用 STM32 蓝色药丸的手柄版本，并连接到显示器。但如果人手不够，就用 ESP8266 版本，用配套的 app 更新比分。休息后请查看演示，以便您可以看到它的实际效果。

这些年来，我们已经看到了一些记分牌，包括[这款专为室内游戏设计的游戏](https://hackaday.com/2020/08/19/building-the-scoreboard-of-your-imagination/)。
