# 将游戏机相机带入 21 世纪

> 原文:[https://hackaday . com/2021/07/08/将游戏机相机带入 21 世纪/](https://hackaday.com/2021/07/08/bringing-the-game-boy-camera-into-the-21st-century/)

Game Boy 相机可能是大规模销售的规格最有限的数码相机之一，但它在许多人的心中占据着特殊的位置，因为尽管它是一个只有微不足道的 128×128 单色传感器的玩具，但它是许多人拥有的第一台相机。马特·格雷(Matt Grey)就是其中之一，他总是对这款设备无法将图片导出到 Game Boy 打印机之外感到沮丧。因此，在十年前拼凑了一个界面但并不满意之后，他回到了这个项目，并为相机制作了一个无线载体，可以通过 WiFi 轻松传输到他的手机上。

在平板状的 3D 打印外壳内有一个 GBxCart RW Game Boy 盒式读卡器，它的 USB 端口连接到一个 Raspberry Pi Zero，上面有一组读取相机的脚本，并通过网络浏览器下载照片。最后，相机是一个独立的单元，允许他轻松地捕捉和检索尽可能多的微小的黑白图像。有一个视频展示了这个设备的运行，我们把它放在休息时间下面供你欣赏。

这几年来，这款相机已经出现在很多项目中，但是我们猜测这款相机的工作将会是它的工作原理被逆向工程的时候。
