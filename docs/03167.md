# 蚀刻铝硬币，只是为了好玩

> 原文：<https://hackaday.com/2019/05/29/etching-aluminium-coins-just-for-fun/>

[MakeFailRepeat]当时正前往伯明翰的 MakerCentral，许多制造商带着硬币来参加这个交易会。想要参与其中，他决定蚀刻一些属于他自己的硬币。

蚀刻铝是一个简单的过程，普通的制造者很容易掌握。[MakeFailRepeat]从一根铝棒开始，并在表面涂上粘性乙烯基。然后用硬币艺术品进行激光切割，将碎片移除，留下用于蚀刻的负空间设计。在抗蚀剂层就位的情况下，将铝置于盐水浴中，并附着到 DC 电源或电池的正极上。将负电极连接到螺栓上，留下铝进行蚀刻，小心避免过度蚀刻。作为最后的最后一步，硬币被放入一个用废弃的 3D 打印机细丝作为介质的鹅卵石组合的岩石滚筒中。

硬币的边缘有点粗糙，但我们认为对于第一次尝试来说它们很棒。有很多不同的蚀刻方法。[墨粉转印是一种特别受欢迎的方法](https://hackaday.com/2016/09/12/take-your-pcbs-from-good-to-great-toner-transfer/)。休息后的视频。
