# 美国宇航局希望你…设计他们的机器人

> 原文:[https://hack aday . com/2018/08/19/NASA-wants-you-design-their-robot/](https://hackaday.com/2018/08/19/nasa-wants-you-to-design-their-robot/)

没有人比黑客日更喜欢精彩的比赛了。我们跑得足够让任何人忙起来。但是如果你在设计完你的一英寸 PCB 后有一点空闲时间，你可以看看为美国宇航局的 Astrobee 机器人开发机械臂的比赛。

一些挑战已经结束了，但是有相当多的挑战仍然开放了几个月(尽管公布了截止日期，这些看起来像是黑客的伟大项目。特别是，软件架构和指挥、数据和电力系统尚未启动。

但是不要被这 25000 美元骗了。这分散在整个系列的许多奖项上。每项任务都有 250 美元到 5000 美元不等的奖励。然而，你也必须赢得那个奖项，当然。然而，如果你注册了，你会得到一个贴在空间站上的标签。

如果你没看过 Astrobee，它是一个飞行机器人，用来帮助国际空间站的宇航员。该机器人实际上是一个浮动的传感器平台，可以完成一些自主任务，但也可以作为飞行控制器的远程呈现机器人。如果你以前没有看过 Astrobee，你可能会喜欢下面的第二个视频。

我们之前报道过太空蜂。如果你想亲自去参观空间站，那还不太现实，但是[谷歌可以帮你解决](https://hackaday.com/2017/07/23/ok-google-navigate-to-the-international-space-station/)。
