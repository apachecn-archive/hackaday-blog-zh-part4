# CNC 节省了水冷设置

> 原文：<https://hackaday.com/2021/07/17/cnc-saves-water-cooling-setup/>

一个经典问题。你有一个新的 CPU 和一个 15 年前的水冷系统。当然，支架不合适。是时候买一个新的冷却器了吗？如果你是[德 8 奥尔]就不会。你设计一个新的支架，然后用铝制成。

老实说，这看起来有点夸张，但是很有道理。毕竟，不管 CPU 有多新，原则上用水冷却还是一样的。

作为铣削零件，支架并不复杂。出于某种原因，这些洞并没有完全贯通，所以需要一些手工加工。但我们对他交付 CPU 的技能印象深刻(第二个视频)。

如果你想看冷却支架的结果，在下面的第三个视频中。我们想知道支架的工作温度是多少。如果它足够低，你也许可以用正确的材料 3D 打印它。

我们不得不怀疑 CPU 最终是否会将液冷配件集成到封装中。也许有一天。

你可以用水冷却任何变热的东西。甚至是 [3D 打印机](https://hackaday.com/2018/02/06/water-cooling-a-3d-printer/)或者[树莓派](https://hackaday.com/2017/05/15/liquid-cooling-overclocked-raspberry-pi-with-style/)。
