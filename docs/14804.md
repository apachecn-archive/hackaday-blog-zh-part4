# Eurorack 格式的机械接力打击乐器

> 原文:[https://hack aday . com/2022/09/15/mechanical-relay-叩诊-in-a-eurorack-format/](https://hackaday.com/2022/09/15/mechanical-relay-percussion-in-a-eurorack-format/)

Eurorack 格式提供大量模拟和数字合成模块。但是，有没有一个真正的物理打击，同时捕捉输出呢？[VU Perc 继电器模块就是这样做的。](https://gieskes.nl/eurorack/?file=vu-perc-relay)

概念很简单。Eurorack 控制电压馈入 VU 电表，当指针碰到铜条时，电表会摆动并发出噪音。这条带子连接到一个压电元件上，压电元件捕捉声音。还有一个继电器在这种情况下被触发，声音也被压电元件捕获。因此，输入控制电压会通过 VU 计和继电器产生真实的撞击噪声，然后将其捕获并输出到机架的其余部分。

在一个紧凑的 Eurorack 模块中拥有实际的物理声音设备是很棒的。事实上，它的透明更酷，因为它可以让你看到打击乐的动作。值得注意的是，该模块的物理特性意味着，当您在舞台上使用 PA 进行表演时，您需要在机架下放置一些气泡包装或其他隔离材料。否则，你有可能通过压电传感器得到反馈。
