# 殴打，损坏的 NES 控制器得到恢复生命

> 原文:[https://hack aday . com/2021/08/11/thrashed-damaged-nes-controller-gets-bring-back-to-life/](https://hackaday.com/2021/08/11/thrashed-damaged-nes-controller-gets-brought-back-to-life/)

对某些人来说，电子产品也可能是一个神奇的黑匣子。在一些人看来，坏掉的 NES 控制器注定要被扔进垃圾箱的地方，[泰勒]看到了修复的机会。因此，损坏的硬件重新投入使用。

控制器是作为工作批次的一部分购买的，当它进入[泰勒]的所有权时严重损坏。任天堂在那些日子里建造了坚固的硬件，但控制器仍然被打碎了，外壳裂开了，印刷电路板也裂成了两半。

不过，对于具备基本电子技能的人来说，修理很简单。破损的印刷电路板用环氧树脂粘了起来。断裂的走线刮掉了阻焊膜，这样跳线就可以桥接受损区域，使电路恢复功能。

从那时起，3D 打印一个新的外壳是一件简单的事情，控制器又重新投入使用。有问题的表壳是由[亚历山大·迈曼] 设计的[，有一个整洁的小马里奥设计，通过填充嵌体的油漆可以看得见。](https://www.cgtrader.com/alexr-myrman)

虽然这很容易解决，但对于电子艺术的门外汉来说，这简直就是魔术。记住总是有新人加入电子爱好是值得的，像这样的项目是学习的好方法。同样重要的是要注意，带回旧的复古硬件往往有很大的价值，因为在许多情况下，他们不再制造了！我们在这些地方也看到了一些很棒的修复。休息后的视频。
