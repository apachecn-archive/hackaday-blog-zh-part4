# 不屈不挠登陆火星的惊人镜头

> 原文：<https://hackaday.com/2021/02/22/stunning-footage-of-perseverance-landing-on-mars/>

毅力号火星车上的入口下降和着陆(EDL)相机套件的备受期待的视频已经下载到地球，它没有让人失望。观看下面的视频，你会大吃一惊。

这段视频是在今天的 NASA 新闻发布会上播放的，当我们写这篇文章时，发布会仍在进行。下面的简短视频有所有的亮点，但从工程角度来看，好东西在[完整的新闻发布会](https://www.youtube.com/watch?v=gYQwuYZbA6o)中。这些相机捕捉的细节水平，以及这些壮观图像揭示的丰富的工程信息，与它们主要作为事后想法被包括在任务中的事实形成了鲜明的对比。美国国家航空航天局(NASA)并不经常在任务中增加“有就好”的功能，因为向火星运送包裹的每公斤成本令人难以置信。但谢天谢地，他们做到了，主要使用现成的相机。

这组摄影镜头几乎覆盖了在任务的[【7 分钟恐怖】](https://hackaday.com/2021/02/08/getting-ready-for-mars-the-seven-minutes-of-terror/) EDL 阶段发生的一切。一个仰视摄像头目睹了超音速降落伞的突然猛烈展开——我们被告知，降落伞的红白相间的血块中有一个复活节彩蛋——而漫游者上的一个俯视摄像头目睹了隔热板分离并脱落。火星车和下降台上的其他摄像机捕捉到了天鹤机动的惊人细节，既有从火星车向上看的，也有从下降台向下看的。我们对下降引擎扬起的灰尘量感到惊讶，这完全掩盖了“探戈三角洲”在火星车着陆时的图像。我们唯一的抱怨是没有看到下降阶段的“控制拆卸”距离着陆 700 米，但一个人不能拥有一切。

老实说，这些照片我们可以仔细研究好几天。细节的水平是惊人的，他们使火星成为一个真实的地方而不是一个抽象的概念的程度不能被夸大。向 EDL 成像团队致敬，他们让这一切成为可能。
