# 被黑的一套乐器拯救了音乐家的演出

> 原文：<https://hackaday.com/2021/10/08/hacked-set-of-instruments-saves-musicians-gigs/>

你听到的大多数关于航空旅行的恐怖故事似乎都与行李有关。航空公司在让人们安全到达目的地方面做得很好，但托运行李有点冒险——它可能在你着陆时就在那里，可能最终走了风景优美的路线，也可能就这么消失了。当它包含你的衣服时，那已经够糟了，但当它包含你的生计时，那就更糟了？说说压力！

这是音乐家尼古拉斯·布拉斯(Nicolas Bras)在最近一次旅行后发现自己的处境。[Nicolas]正要去演出，但是由于布鲁塞尔航空公司，他的乐器收藏去了别的地方。他无法挽回那天晚上的演出，但他需要考虑以后的约会。令人欣慰的是，[Nicolas]擅长 DIY 乐器，大部分是用 PVC 管和从商业乐器中回收的零件制成的，所以他的问题的解决方案完全掌握在他手中。

给乐器爱好者一个合理的警告——从破损的尤克里里琴上取下琴颈是非常可怕的事情。连接到一块托盘木并配备压电拾音器，琴颈成为一种奇异而迷人的混合弦乐器的一部分。接下来是精选的即兴管乐器，由 PVC 管制成，听起来同样令人惊叹；我们尤其喜欢低音 chromojara，一种带有迪吉里杜管声音的笛子。自行车打气筒 beatbox 也是天才之作，它真正展示了音乐与其说是关于你的装备的花哨，不如说是关于用手边的任何东西制作它的愿望和天赋。

这里希望[Nicolas]最终能和他的装备团聚，但同时也要向他脱帽致敬，因为他找到了替代品。如果他看起来很熟悉，那是因为我们以前看过他的一些作品，比如[他的同情钉小提琴](https://hackaday.com/2021/03/03/a-sympathetic-nail-violin/)和[“爆米花”在 PVC 管上演奏](https://hackaday.com/2021/02/22/pvc-pipes-play-popcorn-perfectly/)。
