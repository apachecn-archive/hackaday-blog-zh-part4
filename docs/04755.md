# 如果你需要一个测量工具，只需建立一个测量工具

> 原文:[https://hack aday . com/2019/11/23/if-you-need-a-measurement-tool-just-build-a-measurement-tool/](https://hackaday.com/2019/11/23/if-you-need-a-measurement-tool-just-build-a-measurement-tool/)

[Darlan Johnson]正在从事一个可穿戴项目，需要一种方法来测量电压和电流随时间的变化。

大多数测量工具被设计为在很短的时间内拍摄系统状态的快照，但是很少有通用工具被设计为观察和记录更长的时间。这是一个有趣的点，例如，许多电源相关的故障，如复位，偶尔发生。更长的时间尺度的测量设备可以检测到这些。

[达伦]有一吨的羽毛和盾牌躺在周围，并把它们组合成所需的工具。INA219 电流传感器记录测量结果。然后，它们会显示在 TFT 上，并记录到 SD 卡中。一切都被捆绑到一个整洁的 3D 打印外壳中，还有一个用于无线操作的电池。一组筒式连接器为电流测量提供了分线点。

这是一个干净利落的黑客，我们可以把它看作是任何黑客的测量抽屉的一个很好的补充。
