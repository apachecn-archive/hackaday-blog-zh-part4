# 现在几点了？无限时间

> 原文：<https://hackaday.com/2020/02/28/what-time-is-it-infinity-time/>

自从无限热潮开始以来，我们已经看到了各种各样的项目——镜子、智能镜子、咖啡桌、时钟，你能想到的都有。不幸的是，所有这些很酷的项目都呆在家里，不被公众欣赏。嗯，不再是了。【nolandoktor】带着[这款漂亮又实用的漩涡腕表](https://www.instructables.com/id/Vortex-Watch-an-Infinity-Mirror-Wristwatch/)将 infinity 带上街头。

尽管这个项目非常先进，但它完全是开源的，对于任何有时间和工具的人来说都是触手可及的。这款手表基于 ATmega32u4，使用 DS3231 实时时钟来记录代表数字的 WS2812 LEDs 上的准确时间。时间使用 R、G 和 B 显示，并分配给小时、分钟和秒。实际上，阅读时间是有点棘手的，直到你明白颜色是如何一起工作的，但这个可爱的东西值得保持轻微的神秘气氛。

手表的外壳部件都是印刷的——表圈是金属的，白色内环是 SLA 的，让一点光从侧面泄漏出来，以便照亮 USB 端口和充当触摸触点的两个不锈钢螺钉。在未来，[nolandoktor]希望添加手电筒模式，将所有的 led 变成白色，一些垫圈可以防水，并通过 IMU 实现手势唤醒功能。休息之后，请花点时间来看看演示。

如果你喜欢一款更传统、更有趣的时计，[这款时计走得更自然，但看起来仍然很酷](https://hackaday.com/2019/12/14/its-about-time-we-saw-another-infinity-mirror-clock/)。
