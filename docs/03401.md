# 等离子球揭示了你的下一个线索

> 原文:[https://hack aday . com/2019/06/24/plasma-globe-reveals-your-next-clue/](https://hackaday.com/2019/06/24/plasma-globe-reveals-your-next-clue/)

如果你喜欢在现实世界中解谜，你可能以前去过密室，或者至少熟悉它的概念，即(自愿地)被锁在一个地方，并寻找最终能让你再次出去的钥匙或门锁组合的线索。虽然有很多模拟选项可以实现这一点，但现在你可能会遇到越来越多的电子游戏，特别是如果它符合逃生室的主题本身。[Alastair Aitchison]喜欢创造这样的谜题，最近他发现如何在他的一个装置中利用 USB 供电的等离子球作为瞬时开关。

这个概念非常简单，[Alastair]注意到当等离子球被触摸时，与闲置状态相比，它会吸取更多的电流，他使用连接到 Arduino 的 INA219 分流器来测量这一点。作为他视频中的演示设置，他使用了两个球体，当同时被触摸时会触发线性致动器，这使它成为一个理想的多人安装。不幸的是，手指的数量、它们在地球上的位置或运动是否足以在电流消耗方面产生可靠的差异，以实现更多维度的切换，这还不清楚，但肯定是值得尝试的事情。

如果你打算建造自己的逃生室，并以疯狂科学家实验室为主题，你显然至少需要一个在角落里发光的等离子球，所以这肯定会派上用场——甚至可能伴随着[稍微大一点的东西](https://hackaday.com/2016/10/28/giant-plasma-tube-makes-a-great-attention-getter/)？对于所有其他主题，你总是可以求助于[一个基于 RFID 的解决方案](https://hackaday.com/2018/05/13/held-captive-by-arduino-and-multiple-rfid-readers/)。
