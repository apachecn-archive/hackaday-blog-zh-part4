# 新设备拯救旧打印机

> 原文：<https://hackaday.com/2022/07/05/new-gear-saves-old-printer/>

随着数码摄影革命的开始，每个人都买了一个闪亮的新的无胶片相机，照片打印机出现了短暂的时尚。这个想法是，你可以得到和胶片上一样的照片，但是这些打印机的介质总是很贵，所以消费者很快就离开了。

现在在二手商店和类似的地方出现，[这就是【阿门】如何获得佳能 Selphy 740](https://trochilidae.blogspot.com/2019/12/canon-selphy-740-fix-i-recently-bought.html) 的原因。它不起作用，经调查发现，一个特别小的塑料齿轮出了故障。大多数人会把打印机扔进垃圾桶，但他们选择了数控加工一个新的齿轮。你不是每天都能处理这么小的工作，所以这是一个有趣的故事。

虽然第一反应可能是伸手去拿 CAD 软件包，但[Amen]却写了一个脚本来创建原始的 GCode。用一个 0.2 毫米的钻头磨削出所需的轮廓。结果是:一个让打印机再次工作的齿轮。这是一种染料升华打印机，它在墨盒中留下负片图像，允许通过一点墨盒倒带来制作负片。对于那些可能已经拥有自己的 Selphy 的人来说，还有一篇关于使用更便宜的售后墨盒的文章。
