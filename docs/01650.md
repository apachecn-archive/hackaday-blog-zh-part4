# 雄心勃勃的自制 x 光机揭示了内在的东西

> 原文：<https://hackaday.com/2018/12/24/ambitious-homebrew-x-ray-machine-reveals-what-lies-within/>

我们不太确定如何评价这台 DIY X 光机。一方面，这是一个令人印象深刻的建筑，有着令人难以置信的规划和对细节的关注。另一方面，这是一个能够释放危险剂量的电离辐射的装置。

最后，我们会让其他人来评判[弗兰·皮尔纳斯]创作的利弊。但是我们只能说，没有提供该项目的详细构建日志可能是一件好事。尽管如此，下面的构建视频还是给了我们一些要点，这些要点肯定花了很长时间和相当多的现金才完成。商用端是固定阳极种类的牙科 X 射线管。如果你需要初级读本，我们之前已经讨论过这些电子管的解剖和生理学，但基本上，它们使用高电压将电子加速到钨靶中以产生 X 射线。高压电源的驱动器是[另一个项目](https://hackaday.io/project/162935-piernass-fast-zvs-mazzilli-driver)的主题，它连接到一个定制的绕线变压器，以达到 150V，然后连接到一个电压倍增器，最终升压到 65 kV。电子管和电压倍增器被密封在一个单独的充油外壳中用于冷却，里面巧妙地衬有铅。

整个机器通过一个 USB 端口控制。增感屏将 X 射线转化为光，各种物体的图像都非常清晰。我们对笔记本电脑硬盘寻道时的荧光图像印象特别深刻，但对一只手的图像印象不太好，可能是[Fran]的；类似的画面会让[威廉·伦琴]自己后悔。

除了安全考虑，这是一个令人难以置信的雄心勃勃的建设，没有人应该尝试。[不是说在](https://hackaday.com/2012/08/27/making-images-and-videos-using-a-diy-fluoroscopic-x-ray/)之前没做过，但是安全地做这件事还是需要很多小心。
