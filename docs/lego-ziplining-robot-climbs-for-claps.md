# 乐高滑索机器人爬上来鼓掌

> 原文：<https://hackaday.com/2020/09/28/lego-ziplining-robot-climbs-for-claps/>

互联网给了我们很多很酷的机器人项目，但是我们不认为我们在 T1 之前见过滑索。至少现在还没有。

这个很酷的小滑索机器人是由[Tart Robotics]的人提供的。正如他们所描述的那样，机器人使用四连杆机构移动，电机的扭矩“通过一对锥齿轮和一个蜗杆传动装置传递给手臂机构。”更酷的是，机器人是通过拍手激活的。你拍手越快，机器人移动越快。这肯定会在你下次虚拟黑客聚会上让你的朋友们大吃一惊。

他们不得不做一些定制的 3D 打印工作，以获得一些乐高组件来连接他们的非乐高成品，所以这需要一点时间。具体来说，他们使用了一些便宜的非品牌 DC 发动机，这些发动机与用于创建机器人身体其余部分的乐高技术组件不自然地匹配。没有什么定制 3D 打印工作不能解决的。

你能用几块乐高积木组装出多么酷的玩意儿，这总是让我们感到惊讶。你最喜欢的乐高项目是什么？
