# 制作雕刻精美的锤子

> 原文：<https://hackaday.com/2019/03/05/making-a-hammer-with-beautiful-engravings/>

工具是任何制造者追求工作的关键部分。对于我们大多数人来说，我们的工具来自当地的五金店或在网上购买。然而，有些人更喜欢自己动手制作，尤里·塔奇曼就是这样一个人。

【Uri】从绝对开头开始，有一块未完成的矩形股票。工件被标记出来，锤头的形状被手工小心翼翼地锉出。类似地，木柄也是由原料制成的，包括刨子、凿子和锉刀。

这可能不是最快的工作方式，但仔细的手工工艺总是令人印象深刻。这座建筑还展示了[尤里]的雕刻天赋，复杂的设计一次从表面刻出一毫米。[Uri]推荐用搪瓷漆来突出这类作品，因为它色彩丰富，容易去除多余的油漆。

当[Uri]建造了一件漂亮的艺术品，并兼职做锤子的时候，[有可能完全朝着相反的方向建造](https://hackaday.com/2018/08/25/unmaker-2-0-is-wile-e-coyotes-dead-blow-hammer/)。休息后的视频。
