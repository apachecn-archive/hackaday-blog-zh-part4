# 廉价打印机器人手臂的简便方法

> 原文:[https://hacka day . com/2019/08/29/a-handy-way-to-廉价打印机器人手臂/](https://hackaday.com/2019/08/29/a-handy-way-to-cheaply-print-a-robotic-arm/)

人形机器人手有一些迷人的地方，仅仅是因为它们非常接近我们自己的手。人们几乎可以想象他们身上覆盖着肌腱和皮肤。可悲的是，制造自己的东西是非常困难的，因为除了是复杂的机器之外，制造一个这样的工程奇迹通常是相当昂贵的。

[【Gray Eldritch】的人形机器人手臂](https://hackaday.io/project/165067-a-modular-and-ambidextrous-humanoid-robotic-hand)项目试图通过提供一个准备打印的项目来解决这两个问题。它所需要的是大约一公斤的 PLA 细丝，一些 TPU 细丝，五个 MG996r 伺服系统(或等效物)，一个 SG90 伺服系统或类似系统，一个 Arduino Uno 板和一些其他的零碎东西。这应该会产生一个有手的机器人手臂，如马克 3 版本的视频中所述，在休息后嵌入。
