# 唱歌鱼钉海棚屋试听

> 原文：<https://hackaday.com/2022/07/02/singing-fish-nails-sea-shanty-audition/>

大嘴巴比利巴斯和其他会唱歌的鱼是在 90 年代末首次带给我们的祸害。[[Kevin heck art]一直在教他们唱新歌](https://www.youtube.com/watch?v=BuP9E1T4Xz8),没有那种微弱的音质和做作的民间曲调。为此，他必须受到称赞。

Teensy 4.1 或 Teensy 3.2 用于为[Kevin]的各种歌唱鱼构建提供动力。一条会唱歌的鱼体内通常有两个马达——一个用来转动鱼的身体，另一个用来开合嘴。把这些连接到一个马达驱动器上，用一点点指令，你就可以启动并运行了。为了使鱼与音乐同步，MIDI 数据通过 USB 被发送到 Teensy。Teensy 接收音符数据，并以此来控制马达，使鱼看起来像是在跟着唱歌。

上面链接的教程是一个学习黑客如何实现的好方法。但是，真金白银在业绩。[凯文]的鱼合唱团表演著名的韦勒曼海上小屋的视频在 YouTube 上有超过 5000 万的点击量，他在 Tiktok 上收集了超过 2600 万个赞。

有时候，简单的小技巧会带来最大的快乐。休息后的视频。
