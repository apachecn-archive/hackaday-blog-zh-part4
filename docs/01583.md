# PIC 驱动的游戏机是块状的好东西

> 原文：<https://hackaday.com/2018/12/19/pic-powered-game-console-is-blocky-goodness/>

学习电子领域的新技能通常最符合经典的口头禅——“边做边学”。[Juan]和[Leo]正是这么做的，他们决定为一个大学项目制作一款手持游戏机，并交付了 PGC-32。

作为康乃尔大学数字系统设计课程的最后一个项目，PGC-32 承担了一大块令人生畏的功能，并及时完成了评分。该团队为硬件编写了一个基本的基于块的游戏，控制可以在模拟棒和内置加速度计之间切换。游戏显示在 320 x 280 的彩色 TFT 显示屏上。学习编写一个基本的游戏是有用的，因为它教学生工程师考虑重要的概念，如时间、竞争条件、中断和显示例程。

作为一个大学项目，它被很好地记录了下来，团队通过解释事情是如何和为什么完成的来逐步完成他们代码中的每个细节。内部也非常整洁，整洁的 PCB 布局和 3D 打印的外壳将所有东西放在一起。

我们以前也看过很多康奈尔大学课程的作品，比如这个睡眠质量监测器。休息后的视频。
