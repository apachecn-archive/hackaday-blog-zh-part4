# 磁铁将柔性印刷电路板变成电动蚱蜢

> 原文:[https://hack aday . com/2020/03/31/magnetes-turn-flexible-PCB-into-electric-grasshopper/](https://hackaday.com/2020/03/31/magnets-turn-flexible-pcb-into-electric-grasshopper/)

仅仅因为某样东西似乎没有明显的目的，这并不意味着我们不应该尝试去做它。随着柔性印刷电路板变得越来越便宜，越来越容易从小规模工厂订购，我们看到业余爱好者正在尝试它们的新用途，如[【Carl bug eja】的跳跃电路](https://www.youtube.com/watch?v=yhKtwCmlFfg)。

该电路基于印刷在柔性 PCB 上的线圈，其本身充当电磁体，但与使用相同技巧的其他设计不同，在这一设计中，线圈被制作成致动器的静态侧。用折叠臂连接到电路上的是一堆两个永磁体，它们作为移动部件工作。由于磁铁构成了电路的大部分质量，当它们被推下和弹回时，它会导致整个物体像一只小电动蚱蜢一样从桌子上跳下不到一厘米。

这远远不是[卡尔]第一次在 Hackaday 上露面，他显然一直忙于探索柔性印刷电路板的新用途，利用它们的电磁特性，从用它们制作[POV 显示器](https://hackaday.com/2020/02/28/latest-flexled-milestone-refines-the-pov-display/)到通过振动四处移动的[小型机器人](https://hackaday.com/2019/08/19/tiny-two-legged-pcb-robot/)。我们很想看看他还能想出什么，休息之后你可以看到他的表演。
