# 不怕变化的 Bartop 街机

> 原文：<https://hackaday.com/2021/09/03/bartop-arcade-machine-that-isnt-afraid-of-change/>

街机具有独特的外观和感觉，大型的机柜和较小的 bartop 机器试图保持传统立式街机机柜的外观和感觉，同时占用较少的空间。这个【DIY 工程】制作的 bartop 拱廊的[工程奇迹被赋予了全新的美感。膨胀的棱角分明的盒子不见了，取而代之的是光滑纤细的曲线。让这款产品发挥作用的关键是曲面显示器。](https://www.youtube.com/watch?v=AKtaqkGCBck)

他从 Fusion360 中的一个详细设计开始，这个设计真正关注的是[DIY 工程]知道会起作用的工具和技术。该设备的主干由木制销钉构成，3d 打印部件在销钉周围滑动。在销钉的两侧，拧上两片丙烯酸树脂作为 LED 扩散片。在这种丙烯酸树脂上，两块 CNC 红橡木与两个弹球式驱动的拱廊按钮相连。在顶部，浇铸的丙烯酸树脂被加热，然后在两件式模压机的帮助下弯曲成想要的形状。屏风正好插在合适的位置。顶部显示屏的一部分被预留给了一个字幕，深色亚克力材质看起来非常特别。十个街机按钮和一个八向操纵杆提供了一系列输入选项。

内部，一个温控风扇和一个树莓皮正在运行显示。控制连接为 GPIO，由 Pi 读取。所以很自然地，SD 卡上的游戏往往在长的垂直屏幕上看起来最好:垂直射击之类的。

可以说，这个项目最棒的不仅仅是执行(这太棒了)，而是幕后的过程。如此多的潜在问题在建模阶段就被解决了，因此制作过程相当顺利(或者说我们认为 youtube 隐藏了大量的罪恶)。结果不言自明，我们认为这是一个令人羡慕的拱廊机。【DIY 工程】已经提到将来提供文件让你自己建。如果这看起来有点吓人，为什么不给[一个更小的 3D 打印桌面](https://hackaday.com/2020/04/22/3d-print-your-way-to-a-bartop-arcade-cabinet/)试试呢？
