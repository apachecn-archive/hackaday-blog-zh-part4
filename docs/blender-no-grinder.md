# 搅拌机？不，磨床

> 原文：<https://hackaday.com/2021/12/25/blender-no-grinder/>

[莱安德罗·费利佩]对这种肮脏的手段并不陌生，[这段他将搅拌机改造成手持旋转研磨工具的视频](https://www.youtube.com/watch?v=3buxQdFP7Yw)也不例外。(嵌在下面。)但最终结果是非常有用的——一个更轻、更易操作的旋转研磨机，有更多的咕噜声。

(视频是葡萄牙语的，但字幕效果很好，一旦你明白机器人很多时候把“研磨工具”翻译成“整流器”的事实。不管怎样，你是来找黑客的。)

亮点是手工制作的联轴器，将搅拌机电机与单独购买的柔性轴和卡盘连接在一起。将展平的 PVC 管用作安装支架。他用马达本身对着一个锉刀“车削”传动轴。还有…

当他用金属老虎钳夹住搅拌机电机进行测试时，这一天就到了。金属和旋转磁铁——最坏会发生什么？火花、烟雾，以及去旧货店买另一台二手搅拌机。

如果你只想看完成的作品，你可以[跳到结尾](https://youtu.be/3buxQdFP7Yw?t=509)。但是基本上，给你自己一个速度可调的搅拌机，把它和现成的研磨机的轴连接起来，你就搞定了。

这是一个概念上如此简单的想法，你可能会想 Hackaday 是否在之前[展示过 blender dr3mel。我们有。你还能用搅拌机马达驱动什么？](https://hackaday.com/2011/02/07/rotary-wall-plug-from-scrap/)
