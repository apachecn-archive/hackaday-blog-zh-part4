# 无脑的跟随者

> 原文：<https://hackaday.com/2022/04/19/a-line-follower-with-no-brains/>

线跟随器是任何希望开始机器人学的人的常见项目，一种小型轮式设备，通常带有某种光学传感器，允许它跟随在其运行的表面上绘制的线。在大多数情况下，它们结合了一个小型微控制器或者可能是一个模拟计算机，提供动力和转向控制，但是[正如【Greg Zumwalt】的蜡笔车展示的](https://hackaday.io/project/184891-crayon-car)一样，完全不用任何大脑就可以制造一个线路跟随器。

这个看似不可能的壮举是由于线和路面实现的，它运行在一张纸上，线是用蜡笔画的。该机器人的一端有一个直线驱动轮，另一端有一对相互成 90 度的从动辊，神奇的事情发生在纸张和蜡笔之间的摩擦力上。机器人毫无问题地遵循圆形轨道，尽管我们可以看到它并非没有缺陷，但我们怀疑它是否有可能制造出更简单的跟随器。

眼尖的读者会注意到[这不是我们向你展示的第一个声称没有大脑的线路跟随器](https://hackaday.com/2017/03/18/line-follower-has-lots-of-recycled-parts-but-zero-brains/)，但我们会声称，由于之前的机器有模拟电路，这一个更值得争夺桂冠。
