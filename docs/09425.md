# 摩托车需要定制的转向灯闭锁开关

> 原文:[https://hack aday . com/2021/03/13/motorcycle-needs-custom-latching-switches-for-turn-signals/](https://hackaday.com/2021/03/13/motorcycle-needs-custom-latching-switches-for-turn-signals/)

虽然现代汽车已经拥有了各种各样的花哨功能，如触摸屏、蓝牙、防撞缓冲区和方向盘控制，但许多摩托车仍然牢牢地停留在过去。有些可能有额外的选择，如燃料表，甚至 ABS，如果你愿意花额外的钱，但很大一部分人都有法律要求的最起码的设备。该设备已经过时，也适合进行一些改进，比如这个[符合人体工程学的定制转向灯开关，带有定制的闭锁开关](https://www.youtube.com/watch?v=mEpW8sqDA14)。

由于摩托车转向灯不像汽车信号那样会自动取消，所以骑车人必须自己取消，通常是通过按下一个不方便的小按钮。该总成由四个独立的开关组成，其中两个控制左右转向信号。由于两者不能同时打开，它们包括可以检测它们位置的电路和一个小电机，如果另一个被按下，它可以物理地解锁它们。整个组件是 3D 打印的，包括闭锁机构，它们与一个用于控制的小型微控制器绑在一起。

这种结构真正令人印象深刻的部分是小型化，因为所有四个按钮都必须用拇指触及，而不需要将手从车把上移开。微小的电路和机械凸轮锁令人印象深刻，值得观看视频。此外，如果您需要对您的摩托车进行更多的人体工程学改进[，还有一些巡航控制选项](https://hackaday.com/2020/12/08/home-crafting-a-motorcycle-throttle-hold/)，这是摩托车通常缺乏的另一个功能。
