# 等离子切割机+ Sharpie 出奇的有用

> 原文：<https://hackaday.com/2020/04/10/plasma-cutter-sharpie-is-surprisingly-useful/>

我们想要的是星际迷航式的复制器。我们有一堆不同的机器，可以喷出各种 2D 和三维形状。在可预见的未来，您仍然需要以某种方式对您构建的大部分内容进行后期处理。[这里制造的东西]遇到了挑战。他经常使用他的等离子切割机来制作复杂的金属板材。但是刀具是二维的，所以除非你在正确的地方弯曲它，否则它看起来不太对。如果你在做一个简单的盒子，这很容易弄清楚，但是在一个复杂的弯道上找到正确的点可能是一个挑战。他的回答？在机架上安装一个标记，这样机器就可以直接在金属板上画线。

听起来很容易，如果你愿意做笔通过单独，然后删除笔，做等离子切割，这将是相对容易的。然而，这似乎有点粗糙。永久安装它需要一种在切割时将它抬起的方法——而且它需要在火炬附近的嘈杂环境中存活。如果你不盖上笔帽，钢笔也会变干。答案是使用一个带有点击回缩器的永久标记，并让机械装置根据命令伸缩笔尖。

火炬使用空气系统来提升和降低火炬。笔的机械装置使用两个阀门和两个弹簧来允许笔适当地上下移动。似乎没有任何确切的计划，但无论如何，你的切割机将是不同的，任何使用等离子切割机的人都应该能够从视频上的解释中弄清楚。

结果看起来很好，而且在平板上画标记非常有用，特别是对于弯曲线。如果你缺少切割者，我们在过去已经看过很多版本了。你也可能从[笔式绘图仪制造](https://hackaday.com/2019/08/26/3d-printed-pen-plotter-is-as-big-as-you-need-it-to-be/)中得到一些想法，只要记住火炬周围不友好的电气环境。
