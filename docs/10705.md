# 明亮的自行车灯可能会让他们后退

> 原文:[https://hack aday . com/2021/07/29/bright-bike-light-may-make-them-back-off/](https://hackaday.com/2021/07/29/bright-bike-light-might-make-them-back-off/)

[Tegwyn☠Twmffat]最近找到了一份兼职自行车信使的工作，并开始意识到将腿驱动的交通工具与各种尺寸的发动机驱动的机械混合在一起是多么危险。有些人会满足于一盏灯，但是当你可以有一个巨大的、有照明的、明确号召行动的标志时，为什么要用一两个小灯泡呢？因为说到安全防范措施真的有天花板吗？

我们认为 180 个发光二极管排成一个熟悉的队形应该可以做到。超声波传感器在苹果羽毛的帮助下探测自行车后面的汽车。所有这些 led 都由一对 L293 电机驱动器芯片和一个滑动电位计控制，以进行一些调光操作。毕竟，它们需要足够的电量，以便在大白天也能看得见，但也要能调光，以免在晚上使人失明。

[Tegwyn☠Twmffat]称这是一个适合初学者的简单项目。我们认为这很好，因为每个人都应该可以获得定制的安全措施。所以去拿那些嘉宝给你自己做一个吧！休息过后，你可以骑着三轮车去看看。
