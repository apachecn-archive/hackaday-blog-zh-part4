# 审查价值 200 美元的树脂打印机

> 原文:[https://hackaday.com/2020/02/14/200-resin-printer-reviewed/](https://hackaday.com/2020/02/14/200-resin-printer-reviewed/)

[3DPrintFarm]得到了早期版本的 [Phrozen Sonic Mini](https://www.youtube.com/watch?v=hsxo3iXdhxY) 树脂打印机。如果你看下面的视频，他对它的构建质量和结果印象深刻。据报道，价格为 200 美元，尽管我们在一些网站上看到的价格略高。构建质量看起来确实不错，尽管你不得不承认，树脂打印机的运动机制非常简单，因为你只需要上下移动。

打印机使用单色液晶显示器，这使得它可以非常快速地固化各层(显然，单色面板可以通过更多的紫外线)。该面板的额定寿命也比彩色液晶显示器长

打印的步骤非常简单。液晶屏测试可让您确保液晶屏正常工作。调平或 Z 轴校准，一如既往，是一些工作量，虽然它是引导和不是很难相比，灯丝打印机。

一旦完成，你就可以开始了，视频显示了一个实时构建，每一层只用了两秒钟。看着物品从一桶液体中生长出来，对我们来说似乎是一种魔法，而且速度相当快。建筑尺寸不是超级大——4.7 英寸乘 2.6 英寸，高约 5 英寸。这对于很多用途来说还是足够大的。

除了在公司的网站上，我们很难找到 200 美元的打印机。亚马逊列出了该产品，但显示它不可用，也没有列出价格。尽管将网站设置为美元模式，购物车还是以台币(TWD)结算，运费大约是 60 美元(新台币 1699 元)。购物车告诉我们，美元总额约为 258 美元。他们确实提供了 PayPal 作为支付选项。

实际上，我们可以以更低的价格买到一台 ELEGOO Mars。当然零售价更高，但有免费送货的选择。我们已经看到了一些价格低至 180 美元的其他树脂打印机。我们很想看看这些打印机如何与这台相媲美。根据视频，这台打印机可能比许多其他打印机都快，如果不说别的的话。

如果你想进入树脂打印机，确保你知道[你正在进入](https://hackaday.com/2019/10/02/when-does-moving-to-resin-3d-printing-make-sense/)什么。
