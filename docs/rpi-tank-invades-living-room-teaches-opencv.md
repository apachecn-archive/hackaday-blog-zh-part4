# RPi 坦克入侵客厅，教 OpenCV

> 原文：<https://hackaday.com/2019/09/23/rpi-tank-invades-living-room-teaches-opencv/>

如果你正在寻找一个简单的项目来开始探索 OpenCV 和机器人技术的交集，那么[由[Vishal Varghese]](https://github.com/vishalvarghese46/RPiTank) 创建的 RPi Tank 可能是一个不错的起点。一个树莓派和几个辅助硬件被绑在一个玩具 M1 艾布拉姆斯坦克的顶部，成为一个低成本的平台，用于测试网络远程控制和视觉路线跟随等概念。当然，你不需要*在艾布拉姆斯坦克周围建造*，但是如果你要做的话，你也可以做得有风格一些。

由于这更多的是一个技术演示，硬件细节非常少。[Vishal]说你只需要一个相对较新版本的 Raspberry Pi，一个 MotoZero 电机控制器和一个相机模块。为了给电子设备提供电力，你不需要比 USB 电源组更奇特的东西，在他的情况下，USB 电源组被方便地连接到炮塔的顶部。他没有提供 MotoZero 如何连接到 Abram 电机的确切细节，但我们认为这足够简单，普通黑客读者可能不需要为他们详细说明。

最终，软件是这个项目的核心，这也是[Vishal]真正交付的地方。他提供了按复杂程度排序的示例 Python 脚本，从在 Raspberry Pi 上建立网络连接到跟随地面上的一行磁带。无论是一起使用还是单独检查，这些脚本都为您的第一个项目提供了一个很好的框架。字面上。

许多形式的循线机器人多年来一直是黑客们最喜欢的项目。无论是用模拟电路回家还是用隐藏的电线取代 T2 的线路，它们都是半自动机器人入门的好方法。
