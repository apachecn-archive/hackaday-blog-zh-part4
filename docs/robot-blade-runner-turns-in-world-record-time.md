# 机器人银翼杀手在世界纪录时间内转弯

> 原文：<https://hackaday.com/2022/10/06/robot-blade-runner-turns-in-world-record-time/>

虽然我们希望学院和大学在学术上更有竞争力，但我们不能否认更多的人对他们的体育项目感兴趣。然而，俄勒冈州立大学在这两方面都做了一点，因为根据吉尼斯世界纪录，他们的双足机器人 Cassie 成为了世界上最快的双足机器人。你可以在下面看到 100 米跑的视频，但是不要眨眼。机器人在大约 25 秒的时间内转向。

令人印象深刻，但仍无法与乌桑·博尔特 10 秒以下的相同距离相提并论。如果你想知道那会是什么样子，试着跑过一个足球场，看看你在 25 秒内能跑多远。关于这个机器人没有太多的技术细节，但是你可以通过观察它的运动来直觉地了解一些事情。你还可以在该大学的网站上找到更多关于这个机器人及其兄弟姐妹的信息。

如果你认为机器人永远不会跑得像人类一样好，我们过去对下棋也有同样的想法。这看起来不像我们通常想象的双足机器人。话又说回来，没有任何理由机器人必须像我们一样[看，或者移动。](https://hackaday.com/2021/10/07/leonardo-the-hopping-flying-bipedal-robot/)
