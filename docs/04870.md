# 用激光制作模型

> 原文:[https://hackaday.com/2019/11/16/making-models-with-lasers/](https://hackaday.com/2019/11/16/making-models-with-lasers/)

好的设计始于一个好的想法，并且能够用一个模型来充实这个想法。在电子世界里，我们会先在试验板上建立一个模型，然后再把所有东西焊接在一起。就像工业设计师埃里克·斯特雷贝尔(Eric Strebel)在创作最终作品之前先制作模型一样。在他最新的视频中，[他展示了使用二氧化碳激光器制作模型的过程](https://www.youtube.com/watch?v=17Pg3QJk00o)。

虽然这个视频可以被认为是使用激光切割机的入门，但观看[Eric]使用的一些精细的细节工作是有趣的，就像观看任何大师级工匠一样。他用各种材料制作了几个立方体，演示了激光切割机的操作，并展示了如何最好地组装“模型”。[Eric]从丙烯酸树脂开始，然后转向木材、纸板，最后是[他的首选材料:泡沫芯](https://hackaday.com/2018/02/10/zen-and-the-art-of-foam-core/)。最终的模型有斜面边缘和内部圆柱体，展示了模型制作的许多“技巧”。

当然，您可能希望构建比立方体更复杂的对象模型。如果你从未有机会使用激光切割机，你会很快意识到使用像这样的高质量工具，设计过程是多么简单。有[Eric]的经验和对工业设计的掌握来帮助也无妨。
