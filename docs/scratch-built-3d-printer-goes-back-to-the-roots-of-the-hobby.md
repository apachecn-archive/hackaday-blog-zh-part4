# 白手起家的 3D 打印机可以追溯到这个爱好的根源

> 原文：<https://hackaday.com/2018/09/02/scratch-built-3d-printer-goes-back-to-the-roots-of-the-hobby/>

订购像数控路由器和 3D 打印机这样的东西是如此容易和便宜，以至于我们忘记了拥有这样的机器曾经涉及到什么，这是情有可原的。过去，你别无选择，只能从头开始建造你的机器。虽然这在今天不太真实，但如果你想推动商业可用的极限，它仍然是这样的，这个巨大的刮擦式 3D 打印机就是一个很好的例子。

这并不是一个全新的版本——[ Thomas Workshop]去年发布了这个版本——但它当时没有引起我们的注意，我们认为下面详细介绍该版本的三部分视频系列值得一看。当我们说临时建造的时候，我们是认真的。这台机器是从一捆铝和钢开始的。没有 80/20 挤压件，没有现成的线性轨道-只有金属和一个平面。一台小型 CNC 刳刨机极大地帮助了建造者，该刳刨机也具有 DIY 外观，但是大部分零件都是用简单的手工工具切割和完成的。由此产生的龙门允许每个维度 40 厘米的巨大工作空间，加热床使用四个加热垫。让我们印象深刻的是，[托马斯]得到的构建刚刚足以打印用于完成构建的部件——这就是黑客精神。

它可能不是我们见过的最大的 3D 打印机——这一区别可能要归功于[这台巨大的 8 立方英尺的机器](https://hackaday.com/2016/11/26/a-big-3d-printer-built-using-the-power-of-procrastination/)——它肯定不能[打印一栋房子](https://hackaday.com/2015/09/27/enormous-delta-bot-3d-designed-to-print-an-entire-house/)。但这是一个令人印象深刻的构建，其成本可能比类似容量的商用机器低得多，而且它拥有白手起家的信誉。
