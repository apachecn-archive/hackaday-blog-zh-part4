# 可移植 PC 的灵感来自过去的 MAC 电脑

> 原文：<https://hackaday.com/2022/10/14/toteable-pc-is-inspired-by-macs-of-days-gone-by/>

早在 20 世纪 80 年代，个人电脑是一个时髦的新事物，这项新技术将采取的形式仍在争论中。当时，各种奇怪的蛤壳、面包箱和一体化设计冲击了市场，苹果 Macintosh 被证明是后者的一个成功例子。受他们第一台电脑 Macintosh 128K 的启发， [[Arnov Sharma]决定打造一台他们自己的现代一体机。](https://hackaday.io/project/[-latteintosh-diy-mini-pc)

它被昵称为 LATTEintosh，因为它是围绕 Latte Panda 3 Delta 构建的。这是一台单板计算机，配有英特尔赛扬 N5105 CPU、8GB 内存和 64GB eMMC 存储。它能够运行完整的 x86 操作系统，而在这里，它运行的是 Windows 10。

外壳是[Arnov]自己创作的定制 3D 打印设计。它配备了 7 英寸高清显示器、散热风扇和集成在外壳中的扬声器。当然，它的顶部有一个手柄，便于携带，就像多年前的麦金塔一体机一样。

对于一台可以拿起来带走的电脑，有一些东西要说，我们喜欢四四方方的外形。有时笔记本电脑根本不行，我们可以想象许多工程师和技术人员会欣赏这样的构建。[我们以前也见过一些很棒的一体机](https://hackaday.com/2021/06/28/a-modern-take-on-the-luggable-computer/)。休息后的视频。
