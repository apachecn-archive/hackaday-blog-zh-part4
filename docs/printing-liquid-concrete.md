# 印刷液态混凝土

> 原文：<https://hackaday.com/2020/02/24/printing-liquid-concrete/>

在添加制造的世界里，总是需要将材料添加到潜在的细丝列表中，以用于打印物体。Stoneflower 3D 的[Anatoly Berezkin]设计的一种[混凝土快速液体打印方法](https://www.stoneflower3d.com/2019/10/08/rapid-liquid-3d-printing-of-concrete/)可以从混凝土中打印出各种形状，同时避免快速脱水的负面影响。该技术基于麻省理工学院在 2017 年开发的一种[打印聚氨酯](https://selfassemblylab.mit.edu/rapid-liquid-printing)的方法。这种技术需要使用化学固化过程在凝胶悬浮液中物理绘制 3D 对象。凝胶使重力不会影响印刷过程，同时也有助于卷曲。Berezkin 是一名工程师和业余爱好者，在他的车库里工作，他还发表了其他作品，包括打印头、陶瓷打印和微型打印设备。

人们可能会怀疑材料的重量是否会在打印过程中导致潜在的坍塌，或者考虑到混凝土干燥所需的时间，打印物体是否是不现实的。他们的演示展示了在家庭用品中完成的过程——碗和特百惠——结合了粘土、混凝土和沙子等可负担得起的物品作为基质和砂浆。这种粘性粘土足够坚固，可以作为一个很好的支架，在印刷时将混凝土结构保持在原位。正如他们的视频所展示的，至少对于简单的物体来说，这个过程似乎相对较快。

RLPC 不需要有毒化学品或凝胶和悬浮液等专有成分。它在潮湿环境中最终印刷物体的浸入也优于用于硬化混凝土的液体沉积的标准过程。此外，该方法只需要粘土或缓凝砂浆作为基体，以及砂浆膏用于转变成混凝土。它被宣传为生态友好，但仅仅是基质和砂浆所需材料的简单性就使它成为一种有前途的技术。
