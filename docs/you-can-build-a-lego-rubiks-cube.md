# 你可以建造一个乐高魔方

> 原文：<https://hackaday.com/2021/10/23/you-can-build-a-lego-rubiks-cube/>

魔方是一个很受欢迎的谜题——它是令人兴奋还是令人愤怒取决于你的个人爱好。[PuzzLEGO]设计了一个乐高魔方，最新版本在早期脆弱的设计基础上进行了改进。

第一步是设计一个核心，允许立方体自由旋转而不会太松。这涉及到大量的试验和错误，直到[PuzzLEGO]找到合适的零件组合来完成这项工作。从那里，这是一个问题，介绍边缘件和角落件，而不堵塞一切。

为了让一切顺利进行，需要做一些实验，但最终结果令人印象深刻。这当然不是一个你可以用来加速立方体的建筑；脆弱意味着仅解决一张脸就要花 20 分钟。[PuzzLEGO]希望进一步改进，增加可玩性。

如果你想复制这一壮举，你将需要大量的小积木，但它绝对是一个可复制的构建。或者，[考虑用乐高搭建一个魔方求解器，而不是](https://hackaday.com/2010/04/29/mindstorm-rubiks-solver/)。休息后的视频。
