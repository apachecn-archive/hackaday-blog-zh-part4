# 步进电机模块改进数控平卷绕线机

> 原文：<https://hackaday.com/2018/11/04/stepper-motor-mods-improve-cnc-flat-coil-winder/>

找到合适的现成零件来完成项目是一种令人满意的体验——购买、安装、继续工作。然而，事情并不总是那么容易解决，这往往需要更令人满意的经验，即修改现有零件来完成工作。通过在轴上钻孔来改造步进电机可能有资格获得年度最佳改装奖。

这就是[Russ]所做的，以对他的 CNC 扁平线圈绕线机进行必要的改进，该绕线机使用改进的 delta-style 3D 打印机将精细的电磁线卷到粘性纸上，以形成各种尺寸和形状的美丽线圈。自从[我们推出](http://hackaday.com/2018/06/16/delta-printer-morphs-into-cnc-flat-coil-winder/)以来【Russ】一直在调整他的设计，并推出越来越好的线圈。在试验过程中，业务端的被动角色被证明是一种负担。问题是接触点落后于三角洲的中心轴，导致 g 代码的问题。[Russ]认为接触点在死点的新工具会有帮助。缺点是必须主动旋转工具，使其与 X 轴和 Y 轴运动保持一致。下面的视频显示了他的修改，其中包括拆卸 NEMA-17 步进机和钻轴通过线圈导线。[Russ]还花了一些时间在框架中反转转子，并提供了一个小的预载弹簧，以保持卷辊与纸张接触。

实时线圈缠绕会议在 21:18 分开始，我们不得不承认看着它令人感到奇怪的安慰。我们不确定[罗斯]到底想用这些线圈做什么，他自己也不知道。但是看起来仍然很酷，步进电机模型是一个需要记住的巧妙技巧。
