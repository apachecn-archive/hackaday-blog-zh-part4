# 黑客日链接:2022 年 8 月 14 日

> 原文：<https://hackaday.com/2022/08/14/hackaday-links-august-14-2022/>

这是什么？关于机器狗的新闻出来了，却没有机器人在舞池里跳舞的视频？不——随着“地面无人机”被部署到卡纳维拉尔角巡逻，看起来四足机器人终于要真正工作了。美国太空部队没有使用熟悉友好的波士顿动力公司的“大狗”机器人，而是使用了 Ghost Robotics Vision 60 Q-ugv，即“四足无人地面车辆”。这些机器人与大狗有着相同的基本布局，但有着明显更健壮的外表，而且不知何故更加邪恶。这些狗的防护等级为 IP67，适合全天候使用，将被部署用于“损害评估和巡逻”，无论这意味着什么。虽然这是[同一只被枪顶着的狗](https://www.theverge.com/2021/10/14/22726111/robot-dogs-with-guns-sword-international-ghost-robotics)，但我们会小心不要偏离肯尼迪航天中心的参观太远。

如果你住在乡下，你的高速互联网选择有时仅限于 Starlink 或支付数万美元将公用设施扩展到你的房子。或者，如果你是来自密歇根州 Scio 镇的 Jared Mauch，有另一种方法——将黑客信条发挥到极致，建立自己的 ISP。康卡斯特(Comcast)向他家开出了 5 万美元的延长线，但大约五年前，他决定自己开一家 ISP，铺设两英里长的地下光纤，连接到最近的网络提供商。他安装了为自己的房子和周围邻居服务所需的所有设备。他已经建立了长达 14 英里的光纤网络，为大约 70 个客户提供服务。他现在与密歇根州签订了一份合同，将服务扩展到 Scio 和周围城镇的 417 个地址。100 Mbps 每月 55 美元，1 Gbps 每月 79 美元，听起来他可以轻松支付 Comcast 想要的 5 万美元，但实际上，成本的增加真的在侵蚀他的迷你 ISP 的底线。尽管如此，还是要向 Jared 脱帽致敬。

希望现在你已经听说了[我们新的网上竞赛](https://hackaday.com/2022/08/08/load-your-icebreakers-the-2022-cyberdeck-contest-is-here/)！我们真的对此感到兴奋——cyber decks 是一种庆祝赛博朋克美学的有趣方式，同时有可能建立一些真正有用的东西。我们特别喜欢的网络平台是那些布满传感器和无线电的平台，但我们很想看看你能想到的任何东西。我们知道在任何项目中引入 Pi 都是一件困难的事情，但也许这是一件好事——它会迫使你跳出鹈鹕案例的思维，也许会关注非典型的架构。您可以在[竞赛页面](https://hackaday.io/contest/186672-2022-cyberdeck-contest)查看所有详情，在这里查看当前竞赛[。](https://hackaday.io/submissions/2022-cyberdeck-contest/list)

我们不太去酒吧——好吧，从来不去——但是从我们收集的情况来看，很大一部分经历可能是与酒保的社交互动。有一个古老的比喻，关于在船上喝一点液体勇气，并向酒保倾吐心声，这对制作饮料的人来说听起来真的很糟糕。所以，调酒师似乎有一些工作保障，对吗？也许不会，如果你不介意在一个有个性的双臂机器人面前哭着喝啤酒的话。这个机器人有点可怕——链接只是一个摘要，但有一个嵌入的视频显示它在 8:30 左右的动作。我们会让你决定你是否愿意让这个东西调酒，和你聊天来获取提示。

最后，天文爱好者这些天过着他们最好的生活，因为新的詹姆斯·韦伯太空望远镜传来了令人惊叹的图像，更不用说不断扩大的火星漫游者舰队传来的似乎无穷无尽的照片了。很难跟上美国宇航局公开的内容，所以太空迷可能想看看 Z3R0C1PH3R 对自动下载有趣的天文图片的过程做了什么。该脚本通过其 [API](https://data.nasa.gov/Space-Science/Astronomy-Picture-of-the-Day-API/ez2w-t8ua) 自动下载来自 [NASA 的每日天文图片](https://apod.nasa.gov/apod/astropix.html)的图像——这是我们很久以来听说的最好的纳税人资金使用方式——并使用亚马逊的 Polly 文本到语音 API 创建图像描述的画外音。这是一件很好的作品，订阅 YouTube 频道是了解 APOD 最新动态的好方法。
