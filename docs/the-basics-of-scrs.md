# SCRs 的基础知识

> 原文：<https://hackaday.com/2019/07/14/the-basics-of-scrs/>

尽管硅控整流器或 SCR 自 1957 年就已问世，但它远没有普通晶体管那样受欢迎。这是一个遗憾，因为当涉及到控制交流和 DC 电压时，它们非常方便，比如灯调光器、电机速度控制器，甚至烙铁温度控制器。[Lewis Loflin]有一个简短的视频介绍，可以帮助你开始使用这些设备。

这种设备的一个有趣的特性是，一旦你打开它，它就会一直开着，直到你做一些特定的事情来关闭它——有点像，[刘易斯]在视频中解释了这一点。

该视频还提供了一些使用廉价设备制作原型的实用建议。他还谈到了 SCR 光耦合器，也有一些是光敏的。你经常会听到这些器件与三端双向可控硅开关一起被提及，三端双向可控硅开关非常相似，但与可控硅不同的是，一旦它打开，电流可以双向流动。令人惊讶的是，SCR 是这种设备的特定商标名称，从技术上讲，它是一种晶闸管。我们总是认为这两个术语是同义的，但有些人认为所有的晶闸管都是晶闸管，但并非所有的晶闸管都是晶闸管。

我们已经在[线圈枪](https://hackaday.com/2012/09/13/simple-concepts-behind-complex-coilguns/)的核心部分看到了 SCR。他们还可以帮助进行[高速闪光摄影](https://hackaday.com/2008/12/14/high-speed-photography/)。
