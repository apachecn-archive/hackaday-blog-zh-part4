# 紧急情况下，现场临时电烙铁也行

> 原文:[https://hacka day . com/2019/08/08/field-advetent-soldering-iron-will-do-in-a-pinch/](https://hackaday.com/2019/08/08/field-expedient-soldering-iron-will-do-in-a-pinch/)

如果你认为[可疑工程]的绰号只是一个名字，那么看看下面视频中由打火机砍出的相当可怕的[烙铁。没有人认为这是一个好主意，但在紧急情况下，它可能会派上用场。我们喜欢筷子的使用和铜丝线圈热交换器的形成。你不得不把筷子钻出来，这有点令人失望，但我们认为你只要稍微想一想，就可以找到不同的方法。](https://www.youtube.com/watch?v=5NmIk-3Ds8o)

当然，管道胶带的使用让它立刻获得了黑客的信任。我们认为这可能不仅在机器人起义后有用，而且如果你必须在远离电源的地方进行一些快速焊接，并且你没有电池供电的熨斗。

援助之手似乎对此没什么帮助。我们有一个坏习惯，就是拿着焊线轴把电线靠在工作台上。我们也使用了这个 [3D 印刷夹具](https://www.thingiverse.com/thing:1203831)，但是你必须小心不要将热焊料滴在塑料上，或者在缝隙中放一些箔片来接住水滴。

如果你真的想在紧急情况下使用这个，蜡烛可能比打火机更有用。当然，[电池铁杆](https://hackaday.com/2017/07/24/review-ts100-soldering-iron/)已经不像以前那么少见了，所以也许就买一个吧。还有一批铁杆，你可以从 [USB](https://hackaday.com/2018/06/27/a-sneak-peek-at-the-ts100-soldering-irons-younger-sibling/) 获得不同程度的成功。
