# 沼气会带你回家

> 原文:[https://hack aday . com/2021/07/28/沼泽气体将带你回家/](https://hackaday.com/2021/07/28/swamp-gas-will-get-you-home/)

为摩托车提供动力的能源必须来自某个地方，可以是发电站、太阳能电池板、加油站或氢工厂。多年来，有许多方法可以降低提取能源的成本，但我们认为[Gijs Schalkx]可能找到了一种我们见过的最便宜、最简单的方法。可能不是游离气，但是是[游离沼泽气](https://hackaday.io/project/180896-uitsloot)！他的 [Uitsloot](https://uitsloot.nl/) (我们认为那是荷兰语的“来自沟渠”)摩托车从荷兰许多水道底部的沉积物中产生的甲烷中获取能量。

其核心是一个古老的本田幼狮助力车，我们猜测的 50 毫升版本。在它的后座上是一个大的透明容器，里面是一个充满气体的气球。他没有在中断处的视频中详细说明，但我们猜测他正在将气体注入本田的空气箱，发动机可以从空气箱中吸入气体/空气混合物。我们喜欢他的气体收集器，一个巨大的内管，中心有一个收集漏斗，漂浮在水面上。他穿上防水靴，用一根长棍戳沉积物，释放甲烷气泡。然后，他用一根长软管和一个自行车打气筒将收集到的气体充入气球。我们看到他在这种非传统的力量下在阿纳姆的街道上飞驰，尽管遗憾的是我们看不到一个装满的气球能把他带多远。

关于这个项目的环境资质还有待讨论，但我们认为，鉴于自然产生的甲烷最终会进入大气层[对气候的影响比发动机产生的二氧化碳](https://www.epa.gov/ghgemissions/overview-greenhouse-gases)更大，他可能会成为赢家。然而，这不是一个可以扩展到几个司机用棍子戳沼泽的系统。
