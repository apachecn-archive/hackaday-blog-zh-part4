# 利用 TempestSDR 暴露计算机监视器旁道漏洞

> 原文：<https://hackaday.com/2020/07/15/exposing-computer-monitor-side-channel-vulnerabilities-with-tempestsdr/>

在无休止地享受了旁道攻击和远程利用的故事后，我们大多数人现在都意识到几乎每件设备都像筛子一样泄露数据。从路由器到电视，再到电脑的电源和散热风扇，所有的东西都可以被泄露秘密。这是可怕的东西，但听起来也很有趣，有了 SDR 和一个小软件，[你也可以参与侧频道活动](https://www.youtube.com/watch?v=QjqpKtGNbQo)。

下面的视频通过软件定义的无线电爱好者[Tech Minds]向我们展示了如何几乎不费力气或费用地窥探显示器上正在显示的内容。使之成为可能的软件是专门为这项工作设计的 TempestSDR。除了一个 AirSpy Mini 和一个橡胶鸭子天线，[Tech Minds]能够在几英寸的范围内重建一个可读的黑白图像；一个更好的天线和一些摆弄可能会提高到数米的范围。他还分享了为所有流行的 SDR 设置 TempestSDR 的技巧，包括 SPRplay、HackRF 和 RTL-SDR。

了解侧信道攻击的可能性是避免它们的关键，所以向[技术头脑]致敬，他们制作了这个简单、易于复制的演示。要了解更多信息，请听听[【Samy Kamkar】对主题](https://hackaday.com/2020/03/09/fear-of-potato-chips-samy-kamkars-side-channel-attack-roundup/)的看法，或者检查一下[电源](https://hackaday.com/2020/05/11/side-channel-attack-turns-power-supply-into-speakers/)、[加密货币钱包](https://hackaday.com/2019/09/13/side-channel-attack-shows-vulnerabilities-of-cryptocurrency-wallets/)和[混合信号微控制器](https://hackaday.com/2018/07/30/side-channel-attacks-against-mixed-signal-microcontrollers/)都容易受到攻击的地方。
