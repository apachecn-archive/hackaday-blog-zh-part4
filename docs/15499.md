# 作为通用接口的廉价 3D 打印机控制面板

> 原文：<https://hackaday.com/2022/12/01/a-cheap-3d-printer-control-panel-as-a-general-purpose-interface/>

浏览中国电子产品的常用网站，几乎每一个可以想象的任务都有大量的电子模块。有些是为业余爱好者或实验者市场设计的，但其中许多是最初为特定产品设计的模块，这些产品可以在其他地方提供有用的功能。一个这样的模块，3D 打印机的通用控制面板，引起了[Bjonnh]的注意。它包含一个有机发光二极管显示器、一个旋转编码器和一些其他好东西，他开始把它作为一个通用的人机界面板来使用。

要进行逆向工程的是一对 5 针连接器，其上连接着旋转编码器和显示器、一个按钮、一组用于背光照明的可寻址 led、一个蜂鸣器和一个 SD 卡插槽。每个函数都经过了仔细的分析，并提供了示例 Arduino 代码。有用的是，该板自带 5 V 电平转换。

虽然我们都喜欢从零开始构建一切，但如果有这样一个普遍可用的程序集，使用它是有意义的，尤其是如果它很便宜的话。我们猜测这一个将进入相当多的项目，并且那只能是一件好事。
