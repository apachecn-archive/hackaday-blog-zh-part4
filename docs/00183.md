# 阿米加维修使一台坚固的小机器重新投入使用

> 原文:[https://hack aday . com/2018/07/30/amiga-repairs-put-one-tough-little-machine-back-in-service/](https://hackaday.com/2018/07/30/amiga-repairs-put-one-tough-little-machine-back-in-service/)

对修复者来说，将一件复古的硬件恢复到出厂状态通常是一件充满爱的工作。另一方面，维修更多的是让设备重新投入使用。但是修复和复原之间的界限有时很微妙，一方的目标渗透到另一方，就像在[拯救备用电池泄漏的全新 Amiga 2000】的努力。](https://www.youtube.com/watch?v=oyMgr7haHFA)

[为了阻止电解液从旧电池中流出并防止进一步的损坏，【Retromat】已经进行了紧急维修](https://hackaday.com/2018/07/08/amiga-2000-emergency-repair/),进入了项目的修复阶段。蔓延的软泥在 PCB 上蔓延时，损坏了几个盖子和 CPU 插座，但主要损坏的是阻焊膜本身。在下面的视频中，你可以清楚地看到面具上片状、气泡状的区域，这是笨蛋造成伤害的地方。

使用玻璃纤维橡皮擦，一些异丙醇，以及比我们更大的耐心，[Retromat]能够移除损坏的保护层，以揭示下面损坏的真实程度。值得庆幸的是，大部分痕迹仍然完好无损；当他卸下 CPU 插座时，只有下面的一对线脱落了。在用细金属丝替换它们，替换腐蚀的盖子和插座，并添加硬币电池固定器来替换旧电池后，暴露的痕迹被涂上清漆来保护它们，机器几乎像新的一样。

Amigas 在他们那个时代是伟大的机器，并且推出了不止一项业务。他们也证明了自己的持久力，[有些人甚至在关键任务中扮演角色](https://hackaday.com/2015/07/23/this-little-amiga-still-runs-school-districts-hvac/)。
