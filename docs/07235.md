# 简单等离子切割机碰撞检测系统

> 原文：<https://hackaday.com/2020/07/25/simple-plasma-cutter-collision-detection-system/>

机床通常有强大的驱动电机，使它们能够快速准确地工作，快速完成工作。然而，如果工具头与意外的物体碰撞，这可能会导致严重的损坏。为了防止这种情况发生，[【Xnaron】开发了一个简单的系统，在发生碰撞时关闭他的等离子切割机。](https://www.youtube.com/watch?v=JfkQGADILok)

该系统由一个安装在等离子切割炬周围的 3D 打印环组成。轴环有两个配合部分，它们用三个磁铁和三个滚珠轴承固定在一起，作为一个键，保持正确的方向。然后安装三个限位开关，由两个配对的半部分保持闭合。当焊炬与物体碰撞时，会导致磁耦合分离，触发一个或多个限位开关，并安全关闭机器。

意外碰撞的视频显示该设备工作正常。这是一个简洁的解决方案，可能适用于其他类型的机床，不经历高横向力。当然，如果你还没有等离子切割机，[你可以自己做](https://hackaday.com/2019/10/12/make-your-own-plasma-cutter/)。休息后的视频。
