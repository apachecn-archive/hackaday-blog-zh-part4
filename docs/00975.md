# 万圣节前夕:又一个会说话的头骨

> 原文:[https://hack aday . com/2018/10/16/just-in-time for-Halloween-another-talking-skull/](https://hackaday.com/2018/10/16/just-in-time-for-halloween-another-talking-skull/)

这并不是一个独特的想法，但我们喜欢(埃里克·维默斯的)万圣节经典动画头骨。除了向您展示代码和布线之外，视频还花了一些时间讨论音频看起来是什么样子，以及如何将音频转换成适合 Arduino 的格式。你可以在下面看到这个恐怖的视频。

当然，这也是一个 3D 打印项目，虽然头骨是现成的。我们想知道他是否觉得自己像一个脑外科医生在给可怜的头骨做手术。为了固定设备的两个部分，他使用了热固定的黄铜螺纹插件，这种东西我们以前见过，但总是惊讶于我们不常见到。

当然，该项目使用伺服。我们可能已经错过了它，但除了冻结视频，我们没有在网上看到 Arduino 源代码。虽然代码不多，但是可以选择从视频中输入代码。当你意识到上半部分是原理图，下半部分是“竣工”布局图时，原理图就更容易阅读了。

我们很高兴这个头骨没有像我们看到的一些人那样嘲笑我们剩余的时间。顺便说一句，我们已经看到用[更少的部件](https://hackaday.com/2011/10/27/halloween-hacks-simple-robotic-skull-is-a-perfect-last-minute-decoration/)完成了这一过程，你可以比较视频，看看电路的响应有何不同。
