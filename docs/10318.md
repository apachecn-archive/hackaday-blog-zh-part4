# 手表比布谷鸟钟更好吗？

> 原文:[https://hackaday . com/2021/06/05/do-wrist watches-get-any-better-than-a-cuckoo-clock/](https://hackaday.com/2021/06/05/do-wristwatches-get-any-better-than-a-cuckoo-clock/)

作为一名硬件黑客，没有什么比创造自己的特殊时钟更好的方式来彰显你的独立精神了。如果是手表，尤其是在智能手表时代，更是如此。然而，很少有国产手表能像[赤坂清孝出品的布谷鸟时钟手表](https://hackaday.io/project/180059-cuckoo-watch)那样接近手表的天堂，我们可以大胆地称它为获奖手表。在自制钟表领域没有人会超过这个！

除了最高级，这是手腕上的电子布谷鸟钟，带 LED 环形表盘和电动布谷鸟，都装在一个真正的乡村小型木制布谷鸟钟壳里。它通过 BLE 与智能手机进行交流，甚至还有一个布谷鸟声音的声道。令人沮丧的是，几乎没有关于电子产品本身的细节，但我们猜测，几乎支持蓝牙的微控制器可能会投入使用。看看休息下面的视频。

所以我们已经确定这是一个布谷鸟时钟手表，我们非常喜欢它。然而，这并不是我们带给你的唯一新颖的布谷鸟钟。
