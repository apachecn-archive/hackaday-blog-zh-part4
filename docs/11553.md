# 我们希望 DIY 太阳能自行车环游世界

> 原文：<https://hackaday.com/2021/10/07/diy-solar-ebike-goes-around-the-world-we-hope/>

[马克·哈夫兰]正在执行一项任务，要骑着自行车独自完成环球旅行。对于这样一个漫长而艰苦的旅程，除了他和他的自行车所能携带的东西之外，没有任何东西支持，他设计了一辆独特的车辆，它拥有完成旅程所需的一切。这辆自行车有很多我们以前见过的东西，如太阳能电池板和电动机，但也有很多完全新颖的东西。

对于这样的长途旅行，大多数人的首选自行车是卧式。这使得骑车人在骑行时可以采取更放松的姿势，并且比直立自行车更有效。[Mark]的自行车还在前轮中使用了轮毂电机，由一组锂离子电池组供电。该自行车还利用四块太阳能电池板和三个充电控制器(以减少电池板遮挡的影响)，其中三块电池板放置在拖车上，一块电池板位于自行车上方，以便在骑行时给他提供一些遮挡。[Mark]还在两个阵列中建立了太阳能跟踪能力，允许太阳能电池板围绕拖车和自行车自动旋转，以比静态安装的电池板更有效地捕捉阳光[。在大风的情况下，它们也可以手动控制。](https://hackaday.com/2018/07/16/row-your-bike-to-china/)

从下面链接的视频中，我们可以看到自行车的许多其他附加功能，这些功能将使它能够进行这样的长途旅行。首先，他得到了一台新的发动机，这台发动机比他的旧发动机有了许多改进，他的旧发动机已经跑了 30，000 多公里。第二，有一些值得一提的安全功能，如他从应急响应车辆借来的照明设置，甚至包括用于任何灾难性电气故障的灭火器。当然，如果你没有对你的卧式电动自行车进行长距离优化[，你也可以对它进行一些其他的修改，以提高它的越野能力](https://hackaday.com/2019/07/28/russian-ebike-goes-everywhere-possibly-legal/)。祝你好运，马克！
