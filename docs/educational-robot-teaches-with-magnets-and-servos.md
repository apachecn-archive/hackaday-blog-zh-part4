# 教育机器人用磁铁和伺服系统教学

> 原文：<https://hackaday.com/2020/10/08/educational-robot-teaches-with-magnets-and-servos/>

向孩子们教授机器人技术会给他们的未来带来有价值的技能，而且对所有参与者来说都是相当有趣的。然而，教育孩子往往需要采取一种不同的方式来教育大学生，而且往往需要更多的手把手的方法。这个机器人项目就是试图做到这一点，使用一些经典的历史悠久的技术和一种独特的推进方法。

磁力运动机器人，简称 MMR，是一个非常 DIY 的项目。它由手工切割的胶合板制成，由单个模块拼接而成，是进入教育机器人世界的低成本产品。它不是轮子或马达，而是使用安装在伺服臂上的电磁铁来移动。打开和关闭磁铁，并及时移动伺服系统，使机器人能够沿着铁磁性表面移动。

机器人配备了蜂鸣器和发光二极管，使用这些功能给学生带来了进一步的编程挑战。自然，也有一个循线程序，这是开始教育孩子自主机器人操作的一个很好的方式。这一切都是在 Arduino Nano 上运行的，用 Makeblock 的特殊积木式编程软件编程。

虽然它的 DIY 性质使组装比一般的现成套件更复杂一些，但它确实提供了自己的学习机会，如焊接和硬件集成。教育机器人将在未来很长一段时间内继续流行和有趣；我们自己就是一个相扑迷。休息后的视频。
