# 基于 ESP32 的恒电位仪

> 原文:[https://hack aday . com/2022/12/27/an-esp32-based-oppositor/](https://hackaday.com/2022/12/27/an-esp32-based-potentiostat/)

有没有想过自己制作无线化学传感器？来自加州大学欧文分校(UC Irvine)的研究人员已经用他们的[基于 ESP32 的恒电位仪](https://doi.org/10.1016/j.electacta.2022.140481)覆盖了你。

[在](https://hackaday.com/2020/08/02/electrochemistry-at-home/)之前，我们已经在 Hackaday 上讨论过恒电位仪。恒电位仪是分析电活性化学电池电特性的仪器。想想你化学课上的氧化和还原反应(氧化还原),如果你还记得的话。恒电位仪可以在几种不同的模式/配置下使用，但总的想法是让这些仪器在给定的电活性化学电池内引发氧化还原反应，然后测量反应产生的电流。通过测量电流，研究人员可以确定样品中已知物质的浓度，甚至可以确定未知物质的身份，这只是一些潜在的应用。

这些仪器已经成为世界各地研究实验室的支柱，在消费领域有着不可思议的效用。[血糖仪](https://hackaday.com/2018/12/03/why-is-continuous-glucose-monitoring-so-hard/)，用于测量血糖水平的设备，是由于过去几十年电化学和恒电位仪研究的进步而进入日常生活的技术的一个例子。考虑到它们在科学研究和医疗技术中令人难以置信的效用，大量的努力已经投入到了使恒电位仪大众化的工作中，使得它们更容易被普通大众用于教育和业余爱好的目的。当然，任何医疗应用都必须经过严格的测试，并得到每个国家相应管理机构的批准。所以我们在这里谈论更多的非医疗目的。

第一个流行的开源 DIY 恒电位仪是 CheapStat，[我们在](https://hackaday.com/2011/09/14/cheapstat-an-open-source-potentiostat/)之前已经在 Hackaday 上讨论过了。从那时起，[开发更新更先进的开源恒电位仪](https://hackaday.com/2020/07/22/diy-stress-meter/)已经成为科学界的一个热门话题。来自加州大学欧文分校的研究人员希望在开源恒电位仪热潮中加入自己的特殊元素，他们将 ESP32 作为他们的主处理器。这显然为他们打开了一个无线功能的主机(看我们在那里做了什么),其他人在他们之前没有探索过。

有了 ESP32，他们开发了一个基于网络的图形用户界面，使得从恒电位仪控制和收集数据非常无缝和用户友好。你可以想象这里的巨大可能性。教师主导的课堂演示，教师可以通过云轻松访问每个学生的设备，以帮助排除故障或解释结果。开发可以部署在农场各处的土壤监测传感器，以远程收集关于饲料、土壤成分和植物健康的数据。这里的可能性肯定是有希望的。
