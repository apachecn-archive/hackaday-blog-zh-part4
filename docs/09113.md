# Hackaday 播客 103:人人都有天线，一个由链条制成的时钟，电眼管和一点谷歌抨击

> 原文:[https://hacka day . com/2021/01/29/hacka day-podcast-EP-103-每个人的天线-链条制成的时钟-魔眼管-和一点点-谷歌抨击/](https://hackaday.com/2021/01/29/hackaday-podcast-ep-103-antennas-for-everyone-a-clock-made-of-chains-magic-eye-tubes-and-a-little-google-bashing/)

Hackaday 的编辑迈克·斯奇斯和埃利奥特·威廉姆斯讨论了本周最伟大的黑客事件。天线不是火箭科学，所以本周我们真的很喜欢一个视频，它揭开了天线设计的神秘面纱，并以最简单的方式在廉价的无线模块上调整天线。谷歌本周的新闻报道了 project Loon 的结束，以及与花费数年时间确保 Chromium 浏览器在 Linux repos 中的志愿包维护者之间的争吵。埃利奥特是电眼管和疯狂乐器的狂热爱好者，而迈克则对基于链条的时钟显示欣喜若狂。我们结束这一集，谈论协和式飞机，以及电缆机制背后的数学。
