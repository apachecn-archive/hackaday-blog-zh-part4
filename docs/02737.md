# 混凝土桌子甚至包括一个 USB 集线器

> 原文:[https://hack aday . com/2019/04/21/concrete-table-even-includes-a-USB-hub/](https://hackaday.com/2019/04/21/concrete-table-even-includes-a-usb-hub/)

设计家具时，材料的选择对成品的特征和风格有很大的影响。木材是一个经典的选择，而更现代的设计可能会使用金属，塑料甚至纸板。不太受欢迎，但同样有价值的是混凝土。它很重，很便宜，你可以很容易地把它铸造成各种各样的形状。[KagedCreations]认为这将是理想的，[并迅速制作了这件带有集成 USB 集线器的漂亮家具。](https://www.instructables.com/id/Concrete-USB-Hub-Table/)

拆除了一对三聚氰胺架子来建造模板，在其中浇铸混凝土桌子。三聚氰胺是一个受欢迎的选择，因为它便宜，容易获得，并且容易从成品混凝土中释放出来。与 USB 集线器一起，一块木板被浇铸到混凝土桌面的底部。这是预制发夹式支腿的简易连接点，可以用木螺钉安装。

最终的结果是一个整洁的边桌，有足够的重量来保持稳定和安全。[这不是我们见过的第一个混凝土 USB 集线器](https://hackaday.com/2019/04/16/concrete-usb-hub-isnt-going-anywhere/)，但它可能是迄今为止最重的。我们也希望看到一个集成充电板的版本——如果你造了一个，[一定要让我们知道。](http://hackaday.com/submit-a-tip)休息后的视频。
