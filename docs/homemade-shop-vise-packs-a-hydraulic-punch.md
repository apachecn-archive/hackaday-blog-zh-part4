# 自制商店老虎钳包装液压冲床

> 原文：<https://hackaday.com/2018/10/23/homemade-shop-vise-packs-a-hydraulic-punch/>

这是一个悲伤的日子，商店里最简单也是最可靠的工具之一——台钳——寿终正寝了。只有一对结实的铸件和一个沉重的 Acme 螺钉，老虎钳几乎不会出错，但当它发生时，为什么不把它作为一个机会来制作自己的呢？而且，为什么不[避开螺杆，转而使用液压呢？](https://www.youtube.com/watch?v=7BANZRdqFvU)

这就是[Darek]绘制的路线，当时他那有点被滥用的老虎钳因明显的灾难性铸造失败而走到了生命的尽头。他的替代品完全由钢条和槽钢制成，大部分是在他漂亮的等离子切割机轨道上切割的。老虎钳有一个固定的底座和后钳夹，带有一个活动的前钳夹。里面藏着一个 5 吨重的单作用液压缸。单作用气缸不会自己打开虎钳，所以[Darek]想出了一个聪明的返回机制:一对来自汽车行李箱的气体弹簧。

有了一对硬化钢钳夹，对动力缸进行了一些修改以允许脚操作，并进行漂亮的油漆工作，老虎钳就可以使用了。查看下面视频中的构建；我们对老虎钳的力量印象深刻，免提操作是一个意想不到的好处。

是的，大多数人都买老虎钳，但是从小的 T1 到大的 T3，很高兴看到他们从零开始制作。
