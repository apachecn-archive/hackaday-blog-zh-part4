# 3D 打印应变波齿轮箱

> 原文：<https://hackaday.com/2022/10/31/3d-printed-strain-wave-gearbox/>

3D 打印的变速箱总是一个有趣的设计挑战，尤其是如果你想让它们变得紧凑。[ZeroBacklash]创造了一个小的[应变波变速箱](https://www.youtube.com/watch?v=-wFO_AbnbZQ)(谐波传动)，当你想在 NEMA 17 步进电机上用速度换取扭矩时。

应变波齿轮的工作原理是将固定的柔性花键变形为椭圆，使轮齿与输出花键的内齿啮合。在输出端增加几个额外的齿，你会得到一个高减速比齿轮箱，与同等的正齿轮传动相比，零件更少，体积更小。保持柔性花键静止是通过使其一半与具有相同齿数的固定花键啮合来实现的。

在这种情况下，输入侧有 60 个齿，输出侧有 62 个齿，因此传动比为 30:1。使用一组轴承滚珠和电机轴上的椭圆形插头使柔性花键变形。它有助于紧凑的设计，匹配步进电机的正面尺寸，只有大约 27 毫米长。[ZeroBacklash]没有发布任何设计文件，但这个想法应该很容易复制。

我们已经展示了几个不同[尺寸](https://hackaday.com/2018/06/19/printing-strain-wave-gears/)的 3D 打印[谐波驱动器](https://hackaday.com/2019/05/20/a-compact-strain-wave-gear-assembly/)，但它们通常使用一对[滚珠轴承](https://hackaday.com/2017/01/17/3d-printed-strain-wave-gear-needs-your-help/)作为波发生器，这不太适合较小的设计。
