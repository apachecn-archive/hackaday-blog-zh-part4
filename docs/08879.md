# [食本者]的面包板技巧

> 原文：<https://hackaday.com/2021/01/05/ben-eaters-breadboarding-tips/>

无焊试验板是想法第一次变成现实的地方。通常，这是一件有些混乱的事情，随机跳线飞来飞去，连接可以快速切换到正确值的零的元件，或者快速改变电路拓扑。试验板不是制作电路艺术品的地方。

然而，情况并不总是如此，我们已经在接近电路雕刻工艺水平的面包板上看到了不少[Ben Eater]的例子。像任何优秀的工匠一样，[本]在一个新的视频中[分享了他的一些面包板技巧和诀窍](https://www.youtube.com/watch?v=PE-_rJqvDhQ)。从一个简单的 555 blinkenlight 项目开始，这个项目以传统的无所不能的方式连接起来，[Ben]带我们了解他制作一个更值得展示的版本的过程。

他的工具质量高但简单，剥线钳是取得好结果的最关键因素。令人惊讶的是，[Ben]最大程度上依赖于简单的“剪刀式”脱模器的多功能性，而不是复杂的半自动工具。我们发现这是视频中最大的收获，也是实践的结果。[Ben]以前做过很多这种类型的试验板，这意味着当他用“眼球”剥离 0.3 英寸的绝缘层时，他可以做到万分之一的精度。

诚然，这里没有太多新的东西，但观看这个视频有点像观看[鲍勃·罗斯]绘画——既放松又奇怪地引人注目。你可以从我们报道的他的几乎任何视频中获得更多相同的东西，比如[这个 6502 试验板计算机构建](https://hackaday.com/2019/09/19/riding-the-nostalgia-train-with-a-6502-from-the-ground-up/)。我们也看到了受[食者]启发的令人印象深刻的构建，比如[这台全 8 位试验板计算机](https://hackaday.com/2020/11/23/full-8-bit-computer-on-breadboards/)。
