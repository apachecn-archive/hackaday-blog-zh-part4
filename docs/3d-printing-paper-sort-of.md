# 3D 打印纸——算是吧

> 原文：<https://hackaday.com/2019/12/26/3d-printing-paper-sort-of/>

只有几种真正古老的工程材料，其中最古老的是纸。传统上，纸是平的，可以弯曲成各种形状。然而，纸可以被模制成例如包装材料或蛋盒。【XYZAidan】有一种工艺可以[将纸张回收成 3D 纸板状物体](https://www.instructables.com/id/Recycle-Cardboard-Into-Anything-With-3D-Printing/)。你需要一台 3D 打印机，但它实际上并不能打印纸张。取而代之的是，你可以用打印机制造一个模具，这个模具可以用再生纸和搅拌机制造纸浆。

[Aidan]提供了七种不同的模具，从桌子托盘和盘子到简单的立方体和杯垫。模具由三部分组成，有助于取出成品。

3D 打印模具必须非常坚固，因为你要用老虎钳将果肉中的水分挤出来。你几乎可以使用任何一种纸，除非它是光滑的或者有塑料涂层。你可以用手或者用碎纸机把碎片撕碎。搅拌机会产生纸浆，但你也必须加入水和粘合剂。[Aidan]显示您可以使用 PVA 胶水、米糊或玉米淀粉。你可以很容易地用普通大米做成米糊，只要把它煮成糊状就可以了。

也有一些不好的方面。你必须让纸干燥几个小时，甚至一两天。此外，纸浆干燥时会收缩，因此获得准确的尺寸可能是一个问题。这种材料很像木头，但潮湿时就不好用了。

我们在这里看到的大多数纸工艺品更类似于折纸。我们已经看到了在造纸工艺中使用的布，我们想知道在纸浆中加入布纤维是否会产生一些奇怪的效果。
