# 激光雕刻，近距离

> 原文:[https://hackaday.com/2022/10/15/laser-engraving-up-close/](https://hackaday.com/2022/10/15/laser-engraving-up-close/)

你知道你不应该看着你的激光切割或雕刻。但是[亚历克斯]在他的工作室主持连线节目，向他们展示激光如何用光纤激光器雕刻金属。你可以看下面的视频。

如果你没有使用过光纤激光器，你可能会感到惊讶，虽然 60 W 的模型可以燃烧金属，但它对[亚历克斯的手]完全没有影响。顺便说一句，我们不会用如今大多数黑客实验室里常见的二极管激光器来尝试。这个视频并不是很专业，但是看到不同的金属屈服于强大的激光是很有趣的。这里有一些关于以不同方式标记不同金属以及如何处理热膨胀和其他影响的提示。

光纤激光器不像私人商店里的二极管雕刻机那样普遍，但我们认为它们变得更便宜只是时间问题。更不用说它们在商业上的广泛应用，这意味着过剩的单元也可能变得可用。

如果你对激光感兴趣，【Alex 的】 [YouTube 频道](https://www.youtube.com/LaserEverything)有不少有趣的视频可以看看。如果你需要更多的动力，[200 千瓦怎么样？](https://hackaday.com/2019/12/02/fun-with-a-200-kw-fiber-laser/)话说回来，[就算 20 W](https://hackaday.com/2021/01/11/laser-blasts-out-high-quality-pcbs/) 也会让你得到有用的东西。
