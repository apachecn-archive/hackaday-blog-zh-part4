# 这不是一个辩论机器人

> 原文:[https://hack aday . com/2020/03/17/this-is-not-a-argument-bot/](https://hackaday.com/2020/03/17/this-is-not-an-argument-bot/)

虽然如今面对面的辩论越来越难，但我们将永远拥有互联网(希望如此)。那么，在一个小轻浮被称赞的时代，你能做些什么来保持你的游戏呢？建立一个辩论机器人，与最好的人斗智斗勇——一个表情严肃的约翰·克立斯！

[这个来自【8 Bits And A Byte】](https://www.instructables.com/id/Monty-Pythons-Argument-Bot/)的最新产品指的是一个以辩论服务为特色的 Monty Python 小品——一个有接待员的办公室，他会收下你的钱，并把你送到大厅进行健康而热烈的讨论。如果你从未去过巨蟒剧团，那么，这可能是一个很好的时机。

在电子方面，争论机器人是一个非常简单的构建。一个配有谷歌 AIY 帽子的树莓派 B+会倾听你的意见，并决定挑选哪块骨头。你明显被误导的陈述会与对话流的意图相匹配，异议会通过说话者被发送回来。与此同时，克里斯的下巴在一个印刷和伺服驱动的线性致动器上上下移动，同时保持着僵硬的上唇。在您开始研究 Python 之前，请在休息之后观看构建视频。

你见过两个机器人争论会发生什么吗？这只是一点擦伤。
