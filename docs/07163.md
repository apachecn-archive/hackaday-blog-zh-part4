# 用口袋大小的轮子带着陶器兜一圈

> 原文：<https://hackaday.com/2020/07/14/take-pottery-for-a-spin-with-a-pocket-sized-wheel/>

如果 2020 年可以以任何积极的方式被记住，那就是这是爱好选拔的一年。为什么不是陶器？当然，扔锅不会像自制面包一样填饱肚子。但我们敢打赌，你可以看到拥有一堆容易(而且相当令人满意地)被砸成碎片的消耗品的价值。最棒的是，在工作台、垃圾箱和回收站之间，[你或许可以用你已经有的东西](http://www.instructables.com/id/Pocket-Sized-Pottery-Wheel/)为蚂蚁建造【Jadem52】的陶轮。奖金！

陶轮没那么复杂。老实说，就其本身而言，它们有点贵——一个电机和一条皮带驱动一个旋转的盘子。它就像一个电唱机，但不那么挑剔。他们真正让你花钱的地方是把那些罐子热处理成坚固容器的窑。但是你总是可以使用风干的粘土，特别是如果你做这些东西只是为了在你需要发泄的时候粉碎它们。

所以不管怎么说，你只需要一个马达，一个可以装轮子的罐子盖，一个让轮子平稳旋转的轴承。商店买的陶瓷轮子有一个脚进给来控制电机速度，但这个口袋版本要么在 9 伏电压下旋转，要么不旋转。像这样一个项目的伟大之处在于，一旦你掌握了一般原则并使用了它，你就可以随心所欲地迭代和升级。休息之后带它去转一圈。

如果你想组装一个更传统尺寸的轮子，[一个旧的吊扇马达应该绰绰有余](https://hackaday.com/2013/09/14/repurposing-a-ceiling-fan-into-a-pottery-wheel/)。
