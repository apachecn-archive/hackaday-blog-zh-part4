# 用虫眼和古老的智慧建造 GPS

> 原文：<https://hackaday.com/2019/09/06/building-a-gps-with-bug-eyes-and-ancient-wisdom/>

全球定位系统(GPS)在我们的现代生活中根深蒂固，很容易忘记该系统是为美国军方创建的，并且仍然由美国军方运营。虽然存在竞争技术，如 GLONASS 和伽利略，但它们仍由各自国家的政府运营。那么，如果你想知道自己在地球上的位置，而不依赖于任何政府运营的基础设施，你该怎么做？

根据[Aweigh]背后的团队所说，你所要做的就是[从古代水手和昆虫那里得到启示，然后查阅](https://hackaday.io/project/167202-aweigh)。使用两个光偏振传感器、一个指南针和一点数学，他们的设备可以通过查看白天的天空来计算你的纬度和经度。凭借他们定制的 Raspberry Pi shield 和开源 Python 3 软件，该团队设想了一个完全独立的全球定位可以被附加到各种项目上的未来。

这个概念依赖于瑞利模型，它本质上是天空的偏振图。当来自太阳的光在地球大气层中散射时，它产生了偏振带，可以从地面上识别。从本质上来说，这与人眼看到天空呈现蓝色的原理是一样的，但是如果你有两个光传感器观察适当的波长，你可以利用这种效应来计算出太阳在哪里；研究小组称这正是一些昆虫的导航方式。一旦知道了太阳的位置，[Aweigh]就像一个现代化的自动六分仪一样工作。

当然，这并不是所有可能情况下的理想解决方案。在城市环境中，并不总是可能看到清晰的天空，当然，一旦太阳落山，该系统将根本无法工作。理论上，你可以在晚上切换到通过星星导航，但是在城市地区你会遇到同样的问题。尽管如此，这仍然是一个迷人的项目，我们渴望看到它进一步发展。

顺便说一句，如果你在寻找一种类似的解决方案，仍然保留那种*霍雷肖霍恩布洛尔*的氛围，我们在之前已经见过自动化的六分仪。
