# 液晶面板灯罩使醒目的照明

> 原文：<https://hackaday.com/2019/11/11/lcd-panel-lamp-shade-makes-for-eye-catching-lighting/>

乍一看，[[凯尔]的埃尔罗伊灯](http://www.kylescholz.com/wp/the-elroy-lamp/)简直是一件现代风格的室内陈设品；它干净的线条、木纹和现代图案是这个房间的财富。但是当他拿出手机时，事情就变了。因为这盏灯隐藏了一个秘密:它的核心可能是一个标准的 LED 灯泡，但灯罩隐藏了四个由 Nvidia Jetson 驱动的 LCD 屏幕。这些可以通过一个 web 应用程序来控制，以显示各种纹理，完成效果。

然而，这不仅仅是一套固定在灯罩上的笔记本电脑屏幕。毫无疑问，这种屏幕最初出现在笔记本电脑中，但后来被移除了反光背衬，形成了透明的 LCD 面板。然后，必须找到一个合适的扩散器，经过大量的实验后，它变成了一个包含不止一张纹理纸的复合材料。最后，整体被封闭在一个吸引人的木制灯架里，成为家具的一部分。我们喜欢它，不仅因为它是一盏美学上令人愉悦的灯，也因为它是对常规的真正背离。

这个[不是我们带给你的第一个引人注目的灯罩](https://hackaday.com/2013/03/13/cardboard-lampshade-makes-ordinary-recycling-a-centerpiece-of-your-room/)，但它无疑提高了标准。你可以在休息时间下面的视频中看到它的运行。
