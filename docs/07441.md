# 声学摄像机使用许多许多麦克风

> 原文：<https://hackaday.com/2020/08/10/acoustic-camera-uses-many-many-microphones/>

如果你是一个有两只耳朵的人或其他动物，你可能会发现你辨别周围世界声音方向的能力非常有用。当然，这真的只是这类能力的一个最小起点。当[约翰·达菲]着手建造他的声学摄像机时，[他选择使用 96 个麦克风来完成工作](https://hackaday.io/project/172053-acoustic-camera)。

声学摄像机的工作原理是将一组麦克风排列在规定的网格中。通过测量每个麦克风上出现的信号的时序和相位差，可以确定阵列前方声源的位置。麦克风越多，数据越好。

[John]在项目博客上详细介绍了项目是如何完成的[。他概述了组装问题等难题，并分享了如何有效地调试阵列以及如何同时有效地处理这么多麦克风的信息。特别令人印象深刻的是](http://course.ece.cmu.edu/~ece500/projects/s20-teame0/)[【约翰】用设备追踪声音的来源的视频。](https://www.youtube.com/watch?v=i6a2K4w8dn0&feature=youtu.be)这项技术在工业中具有潜在的应用价值，例如用于确定压缩空气泄漏的位置。

总的来说，这是一个做得很好的大学研究项目，对最终结果有很好的记录。[John]的项目对于任何试图构建类似项目的人来说都是一个很好的起点。正如麻省理工学院的这个项目所展示的那样，相控阵技术也适用于射频。休息后的视频。
