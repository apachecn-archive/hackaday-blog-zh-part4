# 改进一个已经惊人的星际迷航道具

> 原文:[https://hack aday . com/2022/01/14/improving-an-independent-an-behaviour-star trek-prop/](https://hackaday.com/2022/01/14/improving-an-already-phenomenal-star-trek-prop/)

当《星际迷航:航海家号》处于开发阶段时，概念艺术是为一种新型的三录仪而创造的，供这艘有名无实的星际飞船的船员使用。但就像弟弟妹妹经常遇到的情况一样，这部剧最终不得不在很大程度上凑合使用《T2:下一代星际迷航》(Star Trek: The Next Generation)中的旧道具，该剧最近刚刚结束了电视播放。

*Trek*aficionado[Mangy _ Dog]在 2019 年完成了这个未使用的三录仪设计的令人瞠目结舌的再创作，但无法就此罢休，[他最近完成了第二个版本，真正提高了粉丝复制品的标准](https://www.youtube.com/watch?v=4kC63O7zNdg)。毫不夸张地说，他创造的道具比他们在节目实际拍摄期间拥有的任何东西都要高得多的质量和保真度。

现在你可能会认为构建第二个版本的三录仪比第一个版本更容易，事实上，[Mangy_Dog]从早期的构建中学到了一些重要的经验。但这并不是说这个新复制品的建造，实际上是委托完成的，一帆风顺。事实上，他几乎立刻就遇到了一个严重的问题。当他试图从 Nextion 订购新显示器时，他发现质量比他以前使用的显示器明显下降。[视角和色彩还原糟糕透了](https://unofficialnextion.com/t/i-was-scammed-by-nextion-and-how-nextion-dont-want-you-to-know-about-it/384)，所以他被迫回到绘图板，不仅找到了新的显示器，还找到了全新的图形芯片与之对话。

[![](../Images/4beb924638ee827210b3ca1557f4d0bd.png)](https://hackaday.com/wp-content/uploads/2022/01/voyagertricorder_detail.jpg)

Simplifying the hinge meant making the electronics more complex.

被迫重新设计主印刷电路板当然令人遗憾，但这几乎不是[Mangy_Dog]在建造他的新三录仪时遇到的唯一挑战。发现第一次构建中使用的 SLA 3D 打印铰链对于长期使用来说不够坚固，他不得不找出如何用黄铜制作新的铰链，同时仍然保留电线在其中隐形的能力。

在这个过程中，他意识到，如果他在不同的印刷电路板之间改用双线串行通信，他就可以减少第一次使用的电线数量。像这样重组硬件和代码的大部分会给整个项目增加越来越多的时间。多少时间？好吧，给你一个想法，[Mangy_Dog]最初计划在 2019 年圣诞节前后交付这个委托道具。

不过，我们认为你会同意这是值得的。说到这里，我们知道抛开花近两个小时观看[Mangy_Dog]制作的关于这个版本的三个视频是一个很大的要求，但请这样想:如果你在下次重新观看*航海家*时跳过*纹身*和*阈值*，它基本上会自己平衡，不会丢失任何有价值的东西。

对[Mangy_Dog]的更多陆地项目感兴趣？早在 2020 年，我们就特别喜欢他的[触摸屏回流焊炉，他的](https://hackaday.com/2020/07/16/touch-screen-reflow-oven-pulls-out-all-the-stops/)[刮刮乐复古游戏掌机](https://hackaday.com/2019/11/26/an-epic-quest-to-build-the-perfect-retro-handheld/)仍然是我们见过的最好的之一。
