# 小四足动物使用许多伺服系统

> 原文：<https://hackaday.com/2021/07/26/little-quadruped-uses-many-servos/>

步行机器人曾经是大公司在研究项目上花费巨资的领域。现在，你可以在家里进行实验。[[tech novation]一直在用他的微型四足机器人做这件事。](https://www.instructables.com/Arduino-Powered-Micro-Quadruped/)

该建筑运行 12 个伺服系统——每条腿 3 个——以实现每个肢体的大范围运动。伺服系统都是由安装了 Arduino 传感器屏蔽的 Arduino Uno 控制的。一切都与一个 3D 打印的底盘和肢节装配在一起，这些肢节直接栓在伺服输出轴上。这是用伺服系统构建快速、简单、轻量级程序集的一种常见方式，在这里效果很好。反向运动学用于计算每个关节所需的运动，机器人可以以各种步态迈出 1 至 4 厘米长的步伐。

我们很乐意看到增加一些传感器和电池组，让机器人以不受束缚的方式进一步探索。[Technovation]留下了一些条款来安装额外的硬件，所以我们期待看到接下来会发生什么。
