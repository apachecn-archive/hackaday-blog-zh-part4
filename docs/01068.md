# 通向花园的排水沟:雨水桶

> 原文：<https://hackaday.com/2018/10/26/gutters-to-gardens-the-iot-rain-barrel/>

没有什么比在你的后院拥有一个花园更好的了。你会得到有味道的西红柿。新鲜的草药很容易。如果你想疯狂，你可以种一个巨大的南瓜。但是园艺有一个问题:工作。你得浇水，你得除草。你还得对付你买来逗乐的上千只瓢虫。

对于他的 Hackaday 奖参赛作品，[Kent]至少解决了其中一个问题。[这是一个物联网雨桶](https://hackaday.io/project/94611-gutters-to-gardens)。它被设计得尽可能简单，这样任何人都可以在几个小时内安装好它，还有一个选项是让这个雨水桶由太阳能供电，使它非常可持续。

这个雨水桶的设计始于你所期待的，一个 55 加仑的雨水桶从[肯特]的排水沟收集水。在这个桶的底部有一个桶孔，从那里，一个 12 伏的水泵把水吸上来，分配到花园的花坛里。一切都是通过粒子光子控制的，这是建立物联网项目最简单的方法之一，是的，你可以用 Alexa 控制整个设置。未来就是现在。

下面，你可以看看[Kent]为他的项目制作的几个演示视频。其中一个是螺线管点击关闭深紫色的*水面上的烟雾*，因为如果你要用 clicky 机电阀建立一个物联网，你还不如让它播放音乐。
