# 在你的口袋里有一个奇怪的强大的 Z80

> 原文：<https://hackaday.com/2019/09/17/a-curiously-strong-z80-in-your-pocket/>

像许多黑客一样，[Tom Szolyga]对古老的 Z80 情有独钟。指令和寄存器的数量使得在 ASM 中编程相对容易，并且他仍然对以前在其上运行的令人耳目一新的简单 CP/M 操作系统有着美好的回忆。事实上，他非常喜欢 Z80 电脑，以至于决定制造一台可以放在口袋里随身携带的电脑。

其结果是薄荷 Z80，之所以这么叫是因为它生活在一个罐子里，这个罐子以前是每个黑客最喜欢的奇怪的强薄荷。但是这个版本的目标不仅仅是让它变小，还要让它便于使用。[Tom]正在使用 ATmega32A 来帮助 Z84C0008 微处理器与现代世界对接，这允许诸如支持微型 SD 卡之类的细节。没有板载 USB 转串行功能，但有一个外部适配器连接到 Minty 的头部，很容易使用在计算机或移动设备上运行的终端仿真器登录到这个经典计算的微观世界。

[Tom]已经在该项目的 Hackaday.io 页面上提供了 Minty Z80 的原理图和材料清单，但截至最新更新，他一直推迟发布电路板文件，直到他确定所有的错误都已解决。目前还不知道他是否在电路板设计的第一次迭代中发现了任何阻碍，但他发布了一张完全组装好的微型逆向计算机的照片，这似乎是一个好迹象。

Minty Z80 的设计[类似于 Z80-MBC2 的[Just4Fun]](https://hackaday.com/2018/07/28/the-4-z80-single-board-computer-evolved/) ，但规模更小。令人鼓舞的是，看到几个项目利用现代设计和组件来防止经典计算成为遥远的记忆。
