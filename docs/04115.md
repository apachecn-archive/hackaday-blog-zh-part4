# 带π边的便携式 PS2

> 原文：<https://hackaday.com/2019/09/06/portable-ps2-with-a-side-of-pi/>

在过去的 30 年里，家用游戏机在市场上占据了特殊的位置。它们是各自公司的皇冠上的明珠，激发了大批铁杆粉丝和对另类硬件追随者的强烈敌意。对于一些人来说，仅仅是手持设备是一种淡化的体验，根本不行。不，游戏机本身必须变得便携！

正是这种思路[可能激发了【黑暗之翼 Mod】生产这款优雅的便携式 PlayStation 2](https://bitbuilt.net/forums/index.php?threads/pis2-ps2-portable-with-build-in-smb-server.2925/#post-34856) 。它开始于 2013 年底，是六年断断续续工作的产物，这种情况对许多黑客来说都很熟悉。它在一个光滑的黑色和蓝色的盒子里包装了一个原始的 PS2 主板，用塑料和油灰精心制作了一个光滑的表面。树莓 Pi 2 也住在里面，提供桑巴共享游戏。之所以选择这种方法，是因为它的加载时间短，而且对于便携式外形来说很耐用，而不是试图将一个完整的 DVD 驱动器塞进去。它与免费的 MCBoot 结合使用来加载游戏。

[工作日志是广泛的](https://bitbuilt.net/forums/index.php?threads/pis2-a-long-overdue-ps2-portable.1681/)，详细描述了完成的漫长道路。很明显，这是一项充满爱心的工作，我们希望现在它已经启动并运行，可以使用很长时间。[这不是我们看到的第一台便携式 PS2](https://hackaday.com/2010/06/22/playstation-2-portable/)，也不会是最后一台。休息后的视频。
