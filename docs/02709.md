# Hackaday 播客 014:保持 Raspberry 的 SD 卡活着，我们爱 MRRF，倒装芯片有多热？

> 原文:[https://hackaday . com/2019/04/12/hackaday-podcast-014-keeping-raspberrys-SD-card-alive-we-love-mrrf-and-how-hot-are-flip-chips/](https://hackaday.com/2019/04/12/hackaday-podcast-014-keeping-raspberrys-sd-card-alive-we-love-mrrf-and-how-hot-are-flip-chips/)

埃利奥特·威廉姆斯和迈克·斯奇斯(Mike Szczys)着眼于摄影测量学的进步(从普通相机的许多照片中构建 3D 模型)，一个在美学和听觉上都令人愉悦的延迟踏板，以及人工智能识别花园蛞蝓的能力。Mike 在深圳电子市场的街道和商店里采访 Scotty Allen。我们用 Raspberry Pi 深入研究 SD 卡问题，在你的桌子上放置工业控制，为 WiFi 建立盖革计数器，以及金属 3D 打印的可悲事实。
