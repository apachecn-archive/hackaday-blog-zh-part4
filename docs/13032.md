# 模块化 Pockit 计算机不仅仅是看上去那么简单

> 原文:[https://hack aday . com/2022/03/12/modular-pockit-computer-is than-meet-the-eye/](https://hackaday.com/2022/03/12/modular-pockit-computer-is-more-than-meets-the-eye/)

“模块化”和“计算机”在历史上一直是一个相当尴尬的领域的对立面。有人可能会说，业余爱好者级别的个人电脑是模块化的，但只是在一定程度上。动态地重新配置它并不容易。模块化笔记本电脑正在慢慢出现，但手持设备呢，我们的需求可能会定期发生变化？

进入 Pockit:一个完全模块化的物联网/边缘计算设备(T1)，可以在运行中重新配置(T2)，而不必重新编程(T3)。没有看休息下面的演示视频，不要浏览本页。它可能是当前十年的“大众之母”。

像许多其他项目一样，模块化基础以 Raspberry Pi 的形式提供基本的计算能力。底座有 12 个磁性连接器，每个连接器有 20 个 I/O 和电源引脚。当添加一个模块时，操作系统检测新的模块并实时加载一个合适的程序*。当加载更多模块时，它会自动配置自己，以便所有模块都有用途。这使得 Pockit 成为一个集成的物联网设备，一个边缘计算中心，一台台式计算机，一台黑莓式手持设备，或一台触摸屏平板电脑，等等。*

 *例如，如果添加了摄像头，它会在屏幕上显示图像——如果有屏幕的话。如果添加了按钮，当按钮被按下时，它会自动拍照。如果你想让相机被运动激活，只需添加一个运动传感器。完成了。外部设备可以通过继电器控制，家庭自动化几乎可以无缝集成。

为了抓住要点，我们忽略了大量的功能:去看视频，当你看完后，也许你会和我们一样惊讶。在过去的中，我们已经表达了对 Pockit 等模块化硬件的[热爱，在观看了这个演示之后，我们只能希望这就是计算和电子产品的未来！](https://hackaday.com/2021/04/27/ask-hackaday-why-make-modular-hardware/)
