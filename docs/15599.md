# 这把长方形的吉他一点也不方正

> 原文：<https://hackaday.com/2022/12/09/theres-nothing-square-about-this-rectangular-guitar/>

我们似乎已经知道这一点，但似乎[尤里·塔奇曼]真的可以建造任何东西。今年秋天，他被邀请参加伟大的吉他制作比赛，这涉及到一个完全可定制的吉他套件发送给每个参赛者作为起点。为了最大限度地发挥创造力，像车身和车头这样的木制部件都是方形的。很有创意的是，[Uri]保持了这种方式。也就是方形的。

虽然是的，从方形中升起的身体实际上是莱斯·保罗的侧面，但有大量的细节使它成为[尤里·塔奇曼]。对于初学者来说，一切都是方形的，从音量和音调电位计的定制黄铜旋钮开始。我们不确定它是不是和亨巴克一起出现的，但如果是这样的话，那肯定是一个令人高兴的意外。如果颈部坯料是方形的话，[尤里]就可以做一个膝盖钢了。一旦完成，[Uri]就把它带到一个制琴师那里进行安装、微调和质量评估。当然是带着飞扬 ~~Vs~~ 的色彩过去了。

还有很多其他的[Uri]标志，像颈板上的鸟，还有一个藏在手绘和手工雕刻的镐把里，所以休息后一定要去看看。
