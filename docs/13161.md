# 鼓舞人心的黑客，未完成的黑客

> 原文：<https://hackaday.com/2022/03/26/inspiring-hacks-unfinished-hacks/>

我们本周得到了一个提示，提示者的评论大致是这样的:“这看起来还不像是一个完成的作品，但我认为它无论如何都很酷”。这是完全正确的。问题中的工作基本上是[将一个简单的网络摄像头连接到 CNC 路由器上，然后用 OpenCV](https://hackaday.com/2022/03/24/you-draw-it-cnc-cuts-it/) 来处理它，而【vector76】的应用程序是从木头上切割出自由手绘曲线。逗他女儿开心。

[![](img/ce7ade829ad497daffe3ffc3a31dd11f.png)](https://hackaday.com/wp-content/uploads/2022/03/eb355f2bc7f9f4d5992c51836ab19e5237d4be94_2_281x500.jpeg) 但是展示一个正在进行的作品是没有必要道歉的。未完成的黑客是可怕的！它们留下了进一步改进和解释的空间。它们就像一个未完成的故事，邀请黑客去想象他们自己的结局。至少这一招在我身上奏效了。

我的思维飞速发展——在 CNC 路由器上添加智能和可扩展的计算机视觉，不仅可以实现线跟踪，还可以实现更智能的边缘查找、破损刀具检测，谁知道还有什么。如今软件端如此灵活，附加硬件需求如此之少，这是一个邀请。就像巴甫洛夫敲钟，而我是狗黑客。或者什么的。

所以，当你完成了一个项目的一半，开始一个可行的第一阶段演示，但你还没有找到每一种可能性时，请记住这一点。留给其他黑客想象的空间也同样强大。你的概念证明不一定要成为所有演示的母体——有时一个可用的鼠标就足够了。
