# 软件效果的硬件控制器

> 原文：<https://hackaday.com/2018/10/27/hardware-controllers-for-software-effects/>

有一个有趣的多重效果提供给所有的音乐家。这是糖字节里的特纳多。这是一个实时效果单元，它利用计算机的马力为您输入的任何音频添加混响和铃声调制。有法兰盖和相位枪。如果你把一个鼓声循环输入你的电脑，就会有一个口吃功能，这意味着你也可以成为一个 Soundcloud 说唱歌手。

可惜这种多效运行在电脑上。这意味着你必须处理台式机或笔记本电脑的用户界面——图形用户界面、鼠标和键盘。如果幸运的话，可能是触摸屏。

我们刚刚结束了 Hackaday 奖中的乐器挑战，如果有一件事是音乐家们喜欢的，那就是他们所有怪异设备的物理接口。这就是为什么从[Liam Lacey] 开始的 [Turnado 硬件 MIDI 控制器。它是基于计算机的软件工具的硬件接口。](https://hackaday.io/project/160792-turnado-hardware-midi-controller)

船上有九个独立的操纵杆，更多的编码器，几个按钮和一个显示器，让一切都有意义。所有这些都由 Teensy 控制，它主要是一个控制 Turnado 的即插即用解决方案。这是一个伟大的项目，它让一个伟大的软件工具变得更加有用，我们很高兴看到它最终入围今年 Hackaday 奖的乐器挑战赛。
