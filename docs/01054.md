# 通过 MIDI 升级，绞弦琴变得更加现代化

> 原文：<https://hackaday.com/2018/10/25/hurdy-gurdy-gets-modernized-with-midi-upgrades/>

如果你以前从未听过绞弦琴，你一定会喜欢。没有多少人听过，因为这些乐器在一些东欧国家之外并不常见。想象一把小提琴，它用手摇轮取代了弓，并添加了一些额外的弦，其功能类似于风笛上的无人机。这种乐器已经存在了数百年，但现在通过 MIDI 的魔力得到了升级。

所有这些新功能都来自[【barna by Walters】](https://gurdy.is/)他手工制作手风琴，但最近一直专注于他的 MIDI 界面。该接口可以进行变调复调，允许乐器制作自己的和弦和和声。它还有一个混合聚合成器，可以播放完全不同的声音，并可以将它们叠加在一起。它还可以将键盘分成两种乐器，上半部分播放一种声音，下半部分播放另一种声音。这是对一种有趣乐器的有趣拍摄，视频绝对值得一看。

当然，与鼓或小提琴相比，绞弦琴不是一种常用的黑客工具。事实上，我们不得不追溯到十多年前，才能找到任何其他以绞弦琴和弗比绞弦琴为特色的文章。这是一种名副其实的乐器。
