# 使用智能手机摄像头来确保司机在看路

> 原文：<https://hackaday.com/2020/05/03/using-smartphone-cameras-to-make-sure-drivers-are-looking-at-the-road/>

我们大多数人可能都很清楚分心的司机驾驶汽车会造成的损害。在理想世界中，驾驶汽车的人不会让手机之类的东西分散他们的注意力，使他们无法完成他们的主要任务，即成为他们控制的 1+公吨车辆的主要导航系统。

多年来，许多智能手机应用程序和车载信息娱乐系统都增加了一些功能，试图阻止司机使用它们，但它们遇到了一个问题，即很难区分乘客和司机。事实证明，询问人类司机他们是否是司机并不总是得到预期的结果。在这里，卡耐基梅隆大学(CMU)的 Rushil Khurana 和他的团队想出了一个更简单的方法。

在[他们的论文(PDF)](http://www.cs.cmu.edu/~rkhurana//assets/eyesroad.pdf) 中，他们介绍了使用智能手机自带的前置(自拍)和后置摄像头来从汽车内部确定用户坐在汽车的哪一侧，并由此推断用户是否坐在驾驶座上的算法和软件实现。由此可以相当安全地假设，如果用户坐在驾驶座上，并且汽车在移动，那么这个用户不应该看手机屏幕。

在一项涉及 16 辆不同汽车和 33 名用户的测试中，他们在手持手机时达到了 94%的整体准确率，在对接时达到了 92.2%。这比本文中介绍的其他方法更可靠，并且不需要任何额外的硬件。谁知道呢，未来的智能手机可能会包含这样的功能，这样应用程序就可以轻松地确定司机应该使用什么功能集(如果有的话)。
