# 定制 LCD 模块出乎意料地便宜和容易

> 原文：<https://hackaday.com/2018/07/24/custom-lcd-module-is-unexpectedly-cheap-and-easy/>

希望您的项目在功能和外观方面更上一层楼？定制的液晶显示器可能是让你达到目的的东西，至少与任何人和他们的叔叔花几分钱就可以从通常的来源购买的点阵或七段显示器相比。但是，如何创造这样一个东西，涉及的成本是什么？就像现在经常发生的情况一样，它比你想象的更简单、更便宜，而且[达夫·琼斯]有一本关于设计和指定定制液晶显示器的很好的入门书。

下面的视频是正在进行的系列的一部分；[之前的一个视频](https://www.youtube.com/watch?v=ZYvxgl-9tNM)涵盖了设计过程，把设计变成规范，选择厂商；[另一个](https://www.youtube.com/watch?v=10D5jpO45Lo)讨论了制造商的设计文件批准和开发模块的测试计划。这一张显示了在极其便宜的模块上的测试计划——[ Dave]只花了 138 美元(包括 33 美元的运费)就可以小批量生产五个模块。该显示屏用于定制电源，有 200 多个部分，包括四个数字部分、一个时钟显示、一个条形图以及用于伏特、安培、毫焦耳和瓦特小时的定制图标。这是一块很大的玻璃，就价格而言，质量非常好。它并不完美——[ Dave]注意到相同公共线路上的一组线段比其他线段稍暗，但可以通过稍微调整电源电压来解决这个问题。

我们惊讶于定制电子产品的门槛变得如此之低，即使你不需要定制液晶显示器，在这些价格下，订购一台也是很诱人的，因为你可以。当然，你也可以[完全从零开始](https://hackaday.com/2016/06/17/how-to-make-a-custom-lcd-from-scratch/)打造自己的 LCD 显示屏。
