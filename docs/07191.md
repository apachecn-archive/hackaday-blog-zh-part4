# 遥控汽车成为电缆凸轮

> 原文:[https://hackaday.com/2020/07/19/rc-car-becomes-cable-cam/](https://hackaday.com/2020/07/19/rc-car-becomes-cable-cam/)

无人机的流行使得空中摄影更加广泛，特别是在业余爱好摄影师和摄像师中。然而，无人机照片并不是没有问题。你必须让无人机跟随拍摄，这可能很难，除非你有一个非常昂贵的无人机。更糟糕的是，你无法真正驾驶无人机穿越茂密的树林或其他障碍地形。

[make some]的朋友面临这些问题，想买一个[电缆摄像头](https://www.youtube.com/watch?v=hag8qnlPXbc)——一个可以在两棵树或其他结构之间的电缆上来回移动的摄像头支架。他们没有从头开始设计，而是决定拆下一辆便宜的遥控汽车和一台惠普打印机，效果非常好——正如你在下面的视频中看到的那样。

重新利用玩具是一个光荣的传统，毕竟，除了一个前进和倒退的马达，你还需要什么？我们不禁注意到，尽管现在玩具黑客变得更容易了，你可以 3D 打印定制的部件来连接所有的东西。

该视频并不是真正的教程，因为你可能会从不同的汽车开始，并有不同的需求。但是那里有足够的信息给你一些想法。看起来汽车的贡献主要是射频遥控部分。还有一个 Arduino 和一些驱动电子设备来维护一台旧打印机捐赠的电机。

如果你喜欢把你的[相机放在导轨](https://hackaday.com/2019/05/08/an-easy-camera-slider-build/)上滑动，我们已经看到了很多这样的设置。或者试试带轮子的相机。
