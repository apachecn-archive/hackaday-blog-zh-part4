# 粉碎的笔记本电脑成为时尚的一体机

> 原文:[https://hack aday . com/2020/03/08/smashed-laptop-being-student-all-in-one/](https://hackaday.com/2020/03/08/smashed-laptop-becomes-stylish-all-in-one/)

我们中的许多人都会在某个时候摔坏笔记本电脑，毁坏屏幕或打碎外壳。这可能会令人沮丧，因为里面有一台完全可用的计算机，被困在一个破碎的外壳中。[Matthew]认为这不是一个问题，而是一个机会，并制造了一台漂亮的一体化台式电脑。(视频，下面嵌入。)

手头有一台严重损坏的 Thinkpad 笔记本电脑，华硕显示器采用薄机身和平板背部，非常适合安装硬件。创建了一个 MDF 基座，在其上安装了笔记本电脑主板。然后添加了 USB 集线器和音频放大器，以及 USB 电源隔离器和声卡，以避免板载耳机输出的接地环路问题。扬声器是从一台旧电视中抢救出来的哈曼卡顿单元，为建造提供了高质量的声音。

视频中有很多很棒的想法，从使用环氧树脂进行坚固的永久组装，到巧妙的破解使电源按钮工作。它让我们考虑在垃圾堆里为我们自己的破笔记本电脑造一台。我们也看到过其他有创意的一体化产品，比如打印机内部的这个。
