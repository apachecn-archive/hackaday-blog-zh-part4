# 泡泡，讨人喜欢的疫情熊猫

> 原文:[https://hack aday . com/2020/07/13/bubbles-the-people-pleasing-疫情-熊猫/](https://hackaday.com/2020/07/13/bubbles-the-people-pleasing-pandemic-panda/)

今年，[托马斯]所在的街区已经从一个安静的小镇变成了一条熙熙攘攘的小巷，挤满了外出散步锻炼和换换风景的家庭和孩子。早期，出现了一种游戏，通过将这些散步变成猎熊来分散孩子们对疫情的注意力——也就是说，寻找坐在房子窗户上的填充熊并数它们的数量。

T2 家里没有毛绒熊，他决定加入到这个有趣的活动中，在窗户上贴上一只 2D 熊猫，它可爱到足以安抚任何人的神经。这在一段时间内很有趣，[但后来他制作了一个互动的 3D 版本](https://www.rc-cam.com/forum/index.php?/topic/4949-bubble-blowing-bear-the-story-of-my-pandemic-panda/)，命名为泡泡熊，它会吹泡泡，用友好的声音说话。

Bubbles 坐在二楼的窗户上，等待路人按下安装在下面电线杆上的一个按钮。两个按钮都连接到一个 433MHz 的遥控器上，遥控器向泡泡栖息地的 ESP32 发送信号，告诉它该表演了。

我们特别喜欢[托马斯]设计的泡泡制造机，它将一个带空气浓缩器的鼓风机对准 3D 打印泡泡棒的传送带。风扇和旋转木马都可以通过一个定制的网络应用程序来控制，每次 Bubbles 有访客时，他都会收到一封电子邮件，告诉他还剩多少泡沫液体。休息之后看看有趣的大小演示。

泡沫很有趣，尤其是当你能大量制造泡沫的时候。气泡也可以工作——[下次你需要随机数发生器的时候记得这个](https://hackaday.com/2019/12/09/generating-random-numbers-with-a-fish-tank/)。
