# 一天之内设计一个电子卡坦板

> 原文：<https://hackaday.com/2021/05/23/design-an-electronic-catan-board-in-a-day/>

制造商有时会跳过的一件事是他们正在创建的项目的设计。我们中的一些人根本不做任何设计，我们只是裤子。制作某样东西的设计部分**可能**要花相当长的时间——有草图要做，还有 3d 建模和 PCB 制作。[Sam March]想尝试创造一些有趣的东西，他只用一天就完成了设计。结果是，或将是一个 [3D 打印，电子，卡坦游戏板](https://www.youtube.com/watch?v=BDRLvolLdK4)的定居者。

 [https://www.youtube.com/embed/BDRLvolLdK4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/BDRLvolLdK4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



当[Sam]在工作中整理他的想法时，我们很快就对设计过程有了一个全面的了解。很高兴看到这可以做到，但这只是设计阶段——[ Sam]将在收到 PCB 和印刷电路板后发布该设计的构建视频。看看网站上卡坦岛项目的其他定居者，比如这个 [3D 卡坦岛板](https://hackaday.com/2019/08/20/get-ready-for-games-night-with-3d-catan/)或者用这个[卡坦岛计时器](https://hackaday.com/2016/08/08/electroshock-timer-will-speed-up-every-game-of-settlers-of-catan/#more-216442)震撼你的朋友。
