# 打印您自己的没有线性轴承的钻孔指南

> 原文：<https://hackaday.com/2022/07/18/print-your-own-drill-guide-without-a-linear-bearing/>

通常我们不会在 Hackaday 上讨论付费产品，但我们不禁对这个来自[USSA] 的 [3D 打印训练指南印象深刻。虽然访问 STL 文件和计划需要付费，但有一个精彩的视频展示了幕后的一点魔力，你可以免费观看。有几个有趣的见解和一些伟大的技术放入这个设计，任何人都可以采取和应用到自己的项目。](https://ussadesign.com/products/p/small-drill-guide)

首先，什么是演练指南？我们很多人都没有全尺寸钻床，所以我们只能用手钻凑合着用。有各种各样的夹具和技巧来获得更直的孔，但它可以令人沮丧地标记出非常精确的螺纹螺丝插件，却发现所有的插件都在一个角度，电路板不适合。钻孔导向器确保钻孔上下垂直，深度可靠。

[USSA]首先展示构成设计的基于节点的 CAD(一个叫做 [Grasshopper](https://hackaday.com/2018/10/17/use-nodes-to-code-loads-of-g-code-for-3d-cnc-carving/) 的程序)。当他组装它时，简单的螺母和螺钉把它固定在一起。但是，螺丝不是将两个独立的部分夹在一起，而是压缩单个塑料，在侧面有一个巧妙的槽，允许塑料弯曲。几个 3D 打印的夹具用于装配轴承轴。最终结果看起来相当令人印象深刻，这是我们自己的印刷项目的灵感。
