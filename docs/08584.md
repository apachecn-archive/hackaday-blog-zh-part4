# 食品安全印刷技术

> 原文：<https://hackaday.com/2020/12/07/food-safe-printing-techniques/>

围绕 Hackaday bunker 的一件事总是引发激烈的辩论，那就是使用 3D 打印塑料接触食物有多危险。我们大多同意这不是一件好事，但我们也知道有些人经常这样做，他们也不会立即死去。[Jakub]决定做一些测试并提出一些建议。甚至还有一个解释结果的视频。

与我们过去读到的许多关于这个话题的内容不同，[Jakub]的帖子经过了充分的研究，并进行了实际测试，包括从装牛奶的杯子中培养细菌。他首先确定了欧盟和美国关于食品级的规定。人们还认识到，虽然基础塑料与食品接触可能是安全的，但没有办法确切知道塑料中有哪些添加剂和其他物质会改变其属性和颜色。

塑料的化学成分只是食品安全的一个方面。另一个重要的方面是有害病原体能够进入印刷物体的小空隙和层中。还有打印机用的材料。安全塑料在肮脏的热端或接触打印机中的有毒物质时可能会变得不安全。

gap 问题让[Jakub]得出了一个不可避免的结论:不管你用什么打印，你都需要给你的打印品涂上涂层，使它们实际上是食品安全的。然而，这种涂层不适合像案板这样的东西，也不适合暴露在热环境中的东西，包括微波食品或热洗碗机。如果你认为 SLA 树脂可能更好，他也讨论了这个问题。

我们已经看到了大量与消耗品接触的印刷品，即使它们可能并不完全安全。我们甚至在[讨论过](https://hackaday.com/2017/10/17/3d-prints-and-food/)安全性，但我们没有培养任何文化。
