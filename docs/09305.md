# 只需点击几下鼠标，即可定制 3D 打印维生素

> 原文:[https://hack aday . com/2021/02/18/custom-3d-printed-vitamins-is-just-a-less-clicks-away/](https://hackaday.com/2021/02/18/custom-3d-printed-vitamins-are-just-a-few-clicks-away/)

最近我们注意到，一家名为“滋养”的公司为自己开辟了一个利基市场，提供用他们自己定制设计的 3D 打印机生产的定制粘性维生素。顾客既可以从一系列预先配置的“堆栈”中选择，也可以拨入自己的七层凝胶状营养物质，获得完全定制的补充。

现在，我们不能保证服用像这样的定制补充剂是否比服用传统的复合维生素更好，但我们承认，滋养开发的硬件非常有趣。正如在休息后的视频中简要看到的那样，大型注射器装满了七种不同的维生素悬浮液，然后装入一个似乎是加热的挤压室中。这与我们见过的其他食品级 3D 打印机没有什么不同，[比如可可压榨机](https://hackaday.com/2018/09/29/maker-faire-ny-cocoa-press-chocolate-printer/)。

看起来所有的注射器都被一个盘子和一对结实的丝杠同时压下，所以每个喷嘴的分层顺序似乎都不一样。[去年初公司网站上的一篇博客文章](https://get-nourished.com/blogs/nourished/3d-printing)显示，一种完全不同的机器正被用于生产维生素，所以要么他们的核心技术正在迅速变化，要么可能正在使用的打印机取决于他们是否正在运行定制的堆栈而不是标准配方。

有趣的是，这与美国陆军战斗给养局(CFD)早在 2014 年提出的概念非常相似。他们推断，3D 打印机可以用来生产为每个士兵的个人营养需求定制的餐吧。由于在战场上很不实用，这个项目没有走多远。但是由于消费者愿意为这项服务支付额外费用，这个想法似乎已经变成了一个有利可图的商业模式。
