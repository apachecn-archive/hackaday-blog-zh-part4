# 万圣节时，怪异的棺材铃会吓到路人

> 原文：<https://hackaday.com/2021/09/21/spooky-coffin-bell-spooks-passers-by-on-halloween/>

我们想象，被活埋并不好玩。对它的恐惧导致了 18 世纪和 19 世纪各种安全棺材想法的发展，今天[格伦·阿金斯]精彩的万圣节道具[重复了这一传统。](https://www.youtube.com/watch?v=E7tYGaNCkb4)

对于那些担心这种事情会发生在自己身上的人来说，安全棺材是一个简单的解决方案。一个概念是在新挖的坟墓上安装一个铃铛，用一根绳子将它伸入棺材。一个发现自己被意外活埋的人可以拉动绳子，按铃求救。

[格伦的]装置消除了棺材和尸体，只是在柱子上安装了一个铃铛。里面有一个超声波测距仪，可以探测路人。当有人走到离道具足够近的地方时，一个微控制器就会触发一个伺服系统，发出令人难以忘怀的紧急铃声。

这是一个简单的构建，但适当地安装了 led 照明，它真的很流行。这将是一个给万圣节派对或鬼屋增添气氛和情绪的好方法。这些年来，我们已经看到了一些很棒的万圣节黑客，其中一些最好的是南瓜。休息后的视频。
