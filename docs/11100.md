# 巧妙的谢妮机器人雕塑可以看到、思考和说话

> 原文:[https://hack aday . com/2021/08/22/articular-谢妮-bot-sculpture-sees-think-and-talks/](https://hackaday.com/2021/08/22/artful-nixie-bot-sculpture-sees-thinks-and-talks/)

当[Tavis]和他的父亲受到启发，将他们的才能用于建造一个机器人雕塑时，他们分担了责任。[塔维斯]“父亲制造了一个机器人头部，[塔维斯]利用新旧设计为他们的创作注入了活力。

多年来，许多硬件黑客都受到了机器人艺术的启发。无论是阿西莫夫和克拉克等人的生动描述，还是 1927 年电影《大都会》的宏伟视觉效果，还是《迷失太空》中疯狂挥舞手臂的机器人，科幻小说中的机器人已经促使许多人将自己的创作带入生活。

对于[Travis]的创作，额头上的两个罕见的俄罗斯谢妮管传达了机器人的想法，而 Adafruit 的双 8×8 LED 矩阵则为想象力提供了一扇通往二元灵魂的窗户。Adafruit 公司的一个声音板给机器人发出声音，用一种只有他自己知道的语言说出渴望的话语。

一个直流到 DC 转换器将 LiPo 提供的 3.7v 电压提升到 Nixies 所需的 170v 电压，一旦电池的两小时寿命到期，一个隐藏的 USB-C 端口就会为电池充电。两个定制的谢妮驱动板都是 Arduino Pro Micro 的主机，【Tavis】已经[为那些希望建立自己的谢妮项目的人](https://oshpark.com/shared_projects/3guozuG9)提供了 PCB 设计。

正如你在广告下方的视频中看到的，结果简直令人着迷！
