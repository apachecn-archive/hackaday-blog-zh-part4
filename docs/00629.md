# 把你自己变成一个有神经网络的机器人

> 原文：<https://hackaday.com/2018/09/16/turn-yourself-into-a-cyborg-with-neural-nets/>

如果智能手表和微型蓝牙耳机是某种迹象的话，那么未来属于可穿戴电子产品。这就带来了一个问题:开发可穿戴电子产品并不像制造一个可以放在架子上的设备那么简单。不，可穿戴电子产品会移动，会伸展，人们会跳，会踢，会打，会流汗。如果您正在开发可穿戴电子产品的原型，那么构建智能物联网可穿戴开发板可能是个好主意。这正是[【戴夫】为他的 Hackaday 奖参赛作品](https://hackaday.io/project/110479-bodihub)所做的，这真的，真的很棒。

[Dave]的 BodiHub 是他参加去年 Hackaday 奖的作品[的衍生作品。虽然这个项目可能看起来不怎么样，*这就是问题的关键；*【Dave】之前的项目包括将价值数千美元的设备缩小到一块可以读取肌肉信号的小板子上。该项目通过创建一个可穿戴的板，支持电池充电，并使可穿戴电子产品的原型制作变得容易，从而进一步发展了这一想法。](https://hackaday.io/project/21523-a-wearable-iotann-dev-board-for-body-sensing)

你可能会问你能用这样的板做什么。为此，[David]建议了一些项目，比如拳击手套，它们可以相互交流，或者告诉你用多大的力来击打某物。或者，你可以读取身体动作，并将 LED 灯光秀与舞蹈表演同步。然而，它可以走得更远，因为[大卫]建立了一个使用增强现实界面的网状网络物流跟踪系统。这实际上是在纽约的 TechCrunch Disrupt 上演示的，观众们惊叹不已。[你可以在这里查看那个演示的视频](https://techcrunch.com/video/crate-zero/)。
