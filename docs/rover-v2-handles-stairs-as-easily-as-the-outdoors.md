# 罗孚 V2 处理楼梯和室外一样容易

> 原文：<https://hackaday.com/2018/08/15/rover-v2-handles-stairs-as-easily-as-the-outdoors/>

[V2 漫游者](https://hackaday.io/project/158458)是一个开源的、3D 可打印的机器人漫游者平台，它经历了它的创造者【tlalexander】的许多进化和发展。罗孚 V2 的设计有许多有趣的地方，比如轮毂本身包含电机和定制行星齿轮箱的方式。该系统结构紧凑，重量低至地面，有助于保持漫游车的稳定性。该平台是全轮驱动的，像悬架这样的移动部件保持在高处，尽可能远离地面。软件是运行在 Raspberry Pi 上的自定义 Python 堆栈，提供基本控制。

路虎 V2 是前一版本的完全机械重新设计，它在轮毂内的[复杂的行星齿轮装置引起了我们的注意。[tlalexander]的目标是创建一个强大、可靠的探测车开发平台，由于其设计，该平台几乎可以 3D 打印，只需要最少的专用硬件。](https://hackaday.com/2017/11/19/gorgeous-engineering-inside-wheels-of-robotic-trail-buddy/)
