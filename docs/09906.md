# 魔杖一挥，灯就亮了

> 原文:[https://hackaday . com/2021/04/22/挥动魔杖点亮你的灯/](https://hackaday.com/2021/04/22/turn-on-your-lights-with-a-wave-of-a-magic-wand/)

智能手机和语音助手是我们大多数人在家中与智能设备互动的典型方式，但它不一定是唯一的方式。[山姆·马奇]想让事情感觉更神奇一点——[所以做了一根魔杖来代替。](https://youtu.be/HailNA14bNk)

该棒依赖于 DA14531 蓝牙低能耗(BLE)片上系统，并与运行在同一硬件上的智能插头配对。通过棒上的加速度计，它能够检测到挥动的动作，然后通过蓝牙向智能插头发送信号，以打开或关闭插座。就魔法这一方面而言，[山姆]继承了[亚瑟·C·克拉克]的衣钵，他有一句名言“任何足够先进的技术都和魔法没什么区别。”因此，人们努力将电子设备小型化到一个微小的 PCB 上，让它隐藏在一个包裹在皮革中的木制旋转棒中。

最终结果是一个有趣的项目，当[Sam]想不用起床就关灯时，这个项目可能也很有用。我们可以想象，如果配置得当，可以一个房间一个房间地工作，这对不知道电灯开关在哪里的客人来说可能是有用的。

如果这个名字听起来很熟悉，那是因为我们以前听说过[Sam]的伟大的 DIY 智能手表。休息后的视频。
