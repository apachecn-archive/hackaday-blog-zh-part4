# 不用离开你的床就能感受到外面的温度

> 原文:[https://hackaday . com/2021/05/04/不离床感受一下外面的温度是什么样的/](https://hackaday.com/2021/05/04/feel-what-the-temperature-is-like-outside-without-leaving-your-bed/)

你的智能手机也许能告诉你外面的天气，但你必须亲自出门才能真正感受到。为了在你自己的床上舒适地做到这一点，[Sagarrabanana]建造了一个[时钟，让你真正感受温度。](https://www.instructables.com/Tactile-Thermometer-Clock/)视频下方有英文字幕。

它基本上是一个盒子，里面有一个螺线管来敲时间，顶部有一个珀尔帖板。用压电元件敲两下盒子，它会以“钟楼”的形式告诉你当前时间，精确到 15 分钟。敲三下，ESP8266 将从云服务获取外部环境温度，并使用 H 桥电机驱动模块将珀尔帖元件冷却或加热到该温度。如果你想自己开发，可以从 GitHub 上获得代码和设计文件。

所有组件都装在一个有吸引力的 3D 打印盒子里，盒子顶部是加工过的木头。虽然我们认为这是一个非常有趣的想法，但我们不禁怀疑，在那些寒冷的冬天早上，让你起床可能会适得其反。

虽然闹钟已经失宠，但它仍然是黑客们的热门产品。我们已经报道了一个看起来像是来自辐射掩蔽所的，还有一个看起来非常酷的 [VFD 显示器](https://hackaday.com/2019/06/14/stylish-alarm-clock-rocks-a-vfd/)。
