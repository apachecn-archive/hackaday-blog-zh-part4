# 用废木料自制的 SMD 虎钳

> 原文:[https://hackaday . com/2023/01/04/a-home brew-SMD-vise-build-from-scrap-wood/](https://hackaday.com/2023/01/04/a-homebrew-smd-vise-built-from-scrap-wood/)

我们在这些部件周围没有看到太多的木制项目，但是当[【oli Kraus】将几块废木料变成一个功能性的 SMD 虎钳](https://www.instructables.com/SMD-and-PCB-Holder/)时，我们怎么能不注意到呢？这个想法很简单。两块带槽的木头固定住印刷电路板。另外两个部件形成了一个带有可调针的手臂，可以在焊接时压住微小的部件。磁铁将每一件物品固定在金属工作面上。简单而优雅。

我们可能已经 3D 打印了一些零件，但话说回来，如果你走这条路，你必须小心你的烙铁去哪里。使用木头的另一个好处是，你可以很容易地抓起几块废料，在几分钟内就可以拥有一个不同尺寸的老虎钳。

我们可能会提出一些改进建议。例如，用一个指旋螺丝来固定针头是受欢迎的。看起来你可以把固定针的部分做得更小，这样可以帮助你把烙铁放在同样的地方。但是它看起来是可行的，没有任何改变。

使用废木料并不迷人，但它确实有助于快速简单地构建功能。如果你不介意它凝固的时间，这里的一些孔和螺栓甚至可以用胶水代替。

当然，你可以将它与其他设计混合搭配。我们喜欢[“一元店 PCB 支架”](https://hackaday.com/2018/07/06/dollar-store-pcb-holder-system/)，但它会与这个项目的手臂很好地工作。看到这个项目，我们不禁想起了 [SMD 鸟嘴](https://hackaday.com/2015/02/05/diy-smd-twofer-manual-pick-and-place-and-the-beak/)。
