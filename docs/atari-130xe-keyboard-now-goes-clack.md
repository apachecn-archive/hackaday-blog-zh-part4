# 雅达利 130XE 键盘现在发出噼啪声

> 原文：<https://hackaday.com/2021/12/12/atari-130xe-keyboard-now-goes-clack/>

在一件儿时心爱的科技产品上表演 resto-mod 可能会相当刺激。在[Bertrand]的案例中，是他们 Atari 130XE 的键盘。虽然它有那些很酷的双摄键帽，但它们下面隐藏着一层蹩脚的薄膜，这可能真正受益于机械升级。放松——薄膜部分被打破了。

[Bertrand]为 Kailh box pinks 设计并印刷了一些新的花茎，可以接受两种已知的变体，而不是标准的 Cherry MX 花托。他还制作了一个新的 PCB (natch)和一个键盘适配器来取代薄膜接口，并定制了一个钢制按键开关板。所谓的 Atari 130MX mod 可以与 Atari 130XE 计算机一起使用，或者作为 PC 的常规键盘，如果你在 Pico 中焊接的话。

[Bertrand]说这种爱的劳动意味着被复制，并告诉我们[对于 Atari 社区的一些人来说，它已经像大金刚](https://atariage.com/forums/topic/324588-atari-130xe-keyboard-rebuild-vintage-keycaps-on-modern-switches/)一样开始了。如果你打算尝试这种模式，要知道灯丝打印机根本不能很好地打印这些微小而精确的零件。[Bertrand]在分辨率为 1/20 毫米(50 微米)的 Elegoo 上打印茎干。好的一面是，新旧股票雅达利键帽并不难找。休息之后来看看。

我们喜欢看到老式键盘得到现代升级。[看到核导弹发射井键盘/轨迹球二合一了吗？](https://hackaday.com/2021/12/01/nuclear-missile-silo-keyboard-re-launched-in-usb/)当我们得知它来自 ebay 时，我们的钱包进入了一级戒备状态。
