# 电动工具黑客采取一个新的角度对钢筋混凝土发电厂

> 原文:[https://hack aday . com/2022/10/05/power-tool-hack-take-a-new-angle-on-RC-power-plants/](https://hackaday.com/2022/10/05/power-tool-hack-takes-a-new-angle-on-rc-power-plants/)

亿万年来，有黑客头脑的人看着他们的一堆东西中的各种项目，想出了一个古怪的想法，并想“我想知道这是否可行？”我们中的一些人止步于此，说服自己这是一个永远行不通的坏主意。众所周知，我们中的其他人，如彼得·斯里波尔，不仅有这些想法，而且有勇气探索它们不切实际的极限，正如休息时间下方的[视频所示。](https://www.youtube.com/watch?v=JUQRfxpGql4)

![](../Images/a0c31a656782d3a9ae4cd5d5535fa9ca.png)

Peter 首先将一个模型飞机的螺旋桨安装到他的 9500 转/分的电池驱动研磨机上，然后检查不同螺旋桨的推力，看哪一个看起来最有效。然后[彼得]做了任何一个精神正常的航空工程师都会做的事情:他让他的兄弟设计了最终的飞机，这是受一个模糊的德国二战不对称飞机设计的启发。

它飞了吗？是的，你可以在视频中看到 It 工具的几次迭代。但接下来发生的事情同样有趣:首先，一架研磨机驱动的单叶片直升机及其随后令人捧腹的失败，以及它稍微更成功的继任者。

我们当然已经讨论过很多角磨机的技巧，例如[这种完美切割的夹具](https://hackaday.com/2019/10/03/solve-your-precision-woes-with-a-sliding-angle-grinder/)(众所周知，用手持研磨机很难做到的事情)，但这是我们第一次看到角磨机不仅仅是因为沮丧而飞起来。你有自己的角向磨光机吗？一定要让[线报](https://hackaday.com/submit-a-tip/)知道！
