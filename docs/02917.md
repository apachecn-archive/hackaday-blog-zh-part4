# 轻松获得霓虹灯外观

> 原文:[https://hack aday . com/2019/05/06/轻松获得霓虹灯外观/](https://hackaday.com/2019/05/06/getting-that-neon-sign-look-without-all-the-hassle/)

我们都喜欢霓虹灯招牌的外观，但在玻璃吹制设备、气瓶、高压和相关技能之间，它并不适用于每个人。幸运的是，如今有了一个很好的替代品:“氖”柔性 LED 灯条。这是【本尼】最近在[制作大型标识展示](https://tinker-fun.chayns.net/NeonLEDSignLogo)时采用的方法，结果不言自明。

[本尼]从全球速卖通采购的。它们宽 8 毫米，可以切割成 4.2 厘米的倍数。在内部，有一条条的 RGB 发光二极管，使得显示器比真正的霓虹灯更加通用。覆盖在发光二极管上的是一个硅树脂扩散片，它完成了霓虹灯的幻觉。这些条带的柔韧性使它们很容易弯曲成不同的形状，但也意味着需要某种固体基底来保持它们的形状。在[本尼]的案例中，他使用了金属框架，并用氰基丙烯酸酯粘合剂将金属条粘在框架上。当胶水固化时，他用拉链将带子夹住，事实上他夹住了拉链的尾部，这证明了他注重细节的天性；我们可能会让它们开着。所有的关注都得到了回报，因为最终产品看起来棒极了。点睛之笔是一些 3D 打印的挡板，带有丙烯酸扩散板和传统的 LED 灯条，外加一个 DMX LED 控制器。

我们以前看过[本尼]的作品，比如这个[光滑的 USB 旋转编码器外设](https://hackaday.com/2019/01/23/crisp-clean-shortcuts/)，就像那次，有一个视频真正展示了这个项目。看一看，跳完之后。
