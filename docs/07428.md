# DIY HID，OMG！

> 原文:[https://hackaday.com/2020/08/08/diy-hid-omg/](https://hackaday.com/2020/08/08/diy-hid-omg/)

威廉·英格利西，60 年代鼠标的创造者之一，于上周去世。这让我想到，如果能在*和*这个发明了现代计算界面的地方，那该有多棒。多么特别的时刻啊！当然，他们可能不知道。

从这里看，似乎鼠标改变了一切，但你必须意识到他们是在一个用光笔工作的世界里，在那里你可以在屏幕上画画。相比之下，鼠标似乎完全没有未来感。他们一定知道他们已经找到了改善现状的方法，但是他们知道他们已经创造了一场革命吗？

那么 DIY 人机界面设备的革命精神去哪里了？我敢说它还活着。事实上，我们自己的 Kristina Panos 有一个名为“[感兴趣的输入](https://hackaday.com/?s=inputs+of+interest)”的系列，我们最近已经看到了大量的 DIY 键盘。然后是多品种的[拨输入](https://hackaday.com/2015/01/17/dial-is-a-simple-and-effective-wireless-media-controller/)。我曾经有一个由硬盘驱动器拼成的专用滚轮，当我在屏幕上阅读大量 pdf 文件时，我不得不说它赢得了桌面空间。见鬼，我们甚至看到人们[制作自己的鼠标](https://hackaday.com/2020/03/12/build-your-own-mouse-for-high-performance/)。

但是我喜欢鼠标发展的故事，是因为他们问了一个问题“在屏幕上定位一个点的最好方法是什么”，并试图回答这个问题。他们成功的一半可能是简单地问了正确的问题，另一半是原型化了的东西。我的直觉告诉我，我们在手机上还没有 100%的输入。这听起来像是黑客日的工作。下一个大的人机界面设计需求是什么？你有什么疯狂的想法来解决它吗？
