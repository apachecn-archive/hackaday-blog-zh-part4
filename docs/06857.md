# Arduino 音乐盒将填充动物变成电视人物

> 原文：<https://hackaday.com/2020/06/11/arduino-music-box-turns-stuffed-animal-into-tv-personality/>

孩子般的想象力是一件美妙的事情。赋予无生命的物体以生命，并假装它们如何过着自己的生活的能力是珍贵的，这在许多电影中是一个成功的故事线。随着严酷的事实或成年和现实最终向我们所有人走来，很高兴看到有些人随着年龄的增长从未完全失去这一点。更好的是，当两个人在生活中找到彼此，就像[er13k]和他的女朋友，他们喜欢开玩笑说他们的巨型狗形毛绒玩具[Tobias]可能会在他们不在的时候偷偷搞恶作剧。另一方面，成长的好处是可以自由支配的先进技术机会，这些机会赋予想象中的人格一张真实的面孔，并让它活在一个旧的 CRT 屏幕里。

最初的想法是做一个小小的音乐盒作为礼物，它用一个 Arduino 扬声器发出[er13k]女朋友最喜欢的歌曲，这个扬声器是他从一台旧收音机中抢救出来的。但是随着事情的发展，他决定让礼物更加个性化。结果仍然是那个音乐盒，内置在 3D 打印的盒子里，有一个小钢琴，可以照亮它演奏的音符，但此外，Arduino 现在还可以通过旧电视上的复合视频显示卡通版的[Tobias]。你可以在休息后的视频中亲眼看到他是如何度过一天的，赠送鲜花和绘画，思考工作和替代职业计划——成年人的问题显然是普遍的。

当然，音乐盒的声音有点一维，但它仍然是一个非常有思想的礼物想法，具有最大的个性化因素。如果[er13k]想要改变声音，也许在几周前我们见过的这个鼓机或[这个袖珍采样器](https://hackaday.com/2019/05/07/the-sampler-that-fits-in-your-pocket/)中会有一些灵感。
