# 通过喷涂上光腻子进行 3D 打印整理

> 原文:[https://hack aday . com/2022/05/10/3d-print-finishing-by-spray-glazing-putty/](https://hackaday.com/2022/05/10/3d-print-finishing-by-spraying-glazing-putty/)

完成 3D 打印是一个劳动密集型的过程，获得适合绘画的好看、光滑的表面需要大量的时间和大量的练习。较深的印刷层线条或较小的表面缺陷可以用各种材料(从腻子到树脂)进行平滑处理，但缺陷越深，填料越厚，这会对表面细节产生影响-使这些变得平滑，使细节变得不明显。[Darkwing dad]有另一个看起来很容易实现的解决方案，通过混合[丙酮和上光腻子](https://www.youtube.com/watch?v=MIbUvQpnJAs)它可以一次性喷到打印表面上。在对油灰与丙酮的比例、大开喷嘴和低压进行了一些试验后，发现可以实现良好的均匀喷涂。重要的是，它可以在几分钟内干燥，从而在短时间内完成多道涂层。

一旦应用了足够的厚度，涂层可以很容易地打磨，以获得平滑的效果，填充最差的间隙，并很好地隐藏层线。填充过程的最后一部分更为典型，直接从拨浪鼓罐上涂上几层填充底漆，然后涂上一层细沙，就可以开始上漆了。

我们讨论平滑 3D 打印的时间几乎和我们讨论 3D 打印本身的时间一样长，而且有多种方法可以做到这一点，这取决于灯丝材料、您的预算和您对有毒气体的容忍度。这里有一个使用紫外线固化树脂平滑[的指南，使用一种带有 IPA](https://hackaday.com/2018/03/08/3d-printering-print-smoothing-tests-with-uv-resin/) 的[特殊平滑灯丝，最后，如果这只是太花哨、太臭或太贵，就拿出旧的](https://hackaday.com/2021/06/14/smooth-3d-prints-with-alcohol/)[丁烷喷灯，用老式火平滑这些印刷品](https://hackaday.com/2018/05/03/smooth-pla-through-the-fire-and-flames/)。
