# 用无人机群进行 3D 打印？

> 原文:[https://hack aday . com/2022/10/18/3d-用无人机群打印/](https://hackaday.com/2022/10/18/3d-printing-with-a-drone-swarm/)

即使在工程等技术学科中，我们仍然可以从大自然中学到很多东西。毕竟，生活中无止境的实验和考验会产生一些最优雅的解决问题的方法。考虑到这一点，一大群研究人员从这种不起眼的(尽管相当烦人)黄蜂身上获得了灵感，特别是它的筑巢技能。这个想法是通过将挤压机安装到无人机上，探索没有框架机器限制的 3D 打印结构。

正如你所料，这种尝试最明显的问题之一是无人机有轻微漂移的趋势。该团队提出的解决方案是将效应器安装到悬挂在无人机底部的 delta bot 载体上，使其能够补偿其测量的运动，并抵消大部分位置误差。

[![](../Images/6bffcc1ac1e267dbf96ba309cb1142f5.png)](https://hackaday.com/wp-content/uploads/2022/10/droneprinting_detail.jpg) 这种印刷方法依赖于两种雄蜂的使用。第一个 done 作为扫描仪工作，测量印刷表面和任何已经完成的印刷。然后，第二架无人机靠近并放下一层，然后它们交换位置并重复，直到结构完成。

通过编队飞行，多架无人机可以同时打印。使用定制的类似水泥的材料，以及似乎正在膨胀的泡沫，展示了印刷品，至少可以说这是一个令人印象深刻的壮举。

目标是使用一群无人机在任何表面上打印大型、复杂形状的结构，每个无人机沉积所需的任何材料。这有点像一群黄蜂筑巢，不管它们遇到什么样的小角落，都会钻进去，但它们是在翅膀上。

一段时间以来，我们已经得到了 3D 打印建筑的承诺，虽然我们不确定这项研究是否会让我们更接近于住在挤压成型的房子里，[我们对 Hackaday](https://hackaday.com/2019/12/26/ask-hackaday-drone-swarms-replace-fireworks-where-are-the-hackers/) 的无人机群很感兴趣。
