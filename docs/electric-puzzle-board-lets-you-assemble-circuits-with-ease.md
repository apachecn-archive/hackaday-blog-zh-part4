# 电动拼图板让你轻松组装电路

> 原文：<https://hackaday.com/2021/01/06/electric-puzzle-board-lets-you-assemble-circuits-with-ease/>

多年来，许多黑客通过 Radio Shack 及其同类公司提供的家用实验套件了解了电子学。为了更容易的基于螺丝或弹簧的连接，他们避免焊接，让没有经验的人以最小的忙乱构建电路，一路上教他们电子的神秘方式。[victorqedu]用他的电子益智游戏给这种形式注入了现代元素。

该构建由一系列 3D 打印块组成，每个打印块包含一个特定的电子组件或模块。这些模块可以连接在一起形成电路，模块中的磁铁与主板中的螺钉配合，将所有部件固定在一起，并在部件之间建立电接触。这是一个需要大量 3D 打印和前期组装的项目，但它使组装电路变得不在话下。

可以构建的电路种类之多令人印象深刻。[victorqedu]展示了从简单的 LED 和开关排列到触摸传感器，甚至是低功率“特斯拉线圈”的一切。我们认为玩积木和把电路安装到位会很有趣。我们以前也见过其他非传统的方法——[,比如建造用于教育目的的粘糊糊的电路](https://hackaday.com/2011/04/15/squishy-circuits-for-tiny-tinkerers/)。休息后的视频。
