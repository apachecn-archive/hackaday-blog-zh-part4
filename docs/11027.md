# 使用 3D 打印嵌件铸造硅树脂部件，以提高硬度

> 原文：<https://hackaday.com/2021/08/17/casting-silicone-parts-with-3d-printed-inserts-for-stiffness/>

多产的制造商[Jan Mrázek]分享了他铸造软硅胶部件的过程，这些部件仍然具有一些额外的刚度，他是通过在铸造过程中将多孔的 3D 打印“肋”嵌入部件中来完成的。3D 打印的插入物充当了一种骨架，因此，零件具有柔软的硅树脂表面，但获得了结构和刚度，如果零件完全由硅树脂铸造，则无法获得这种结构和刚度。好的一面是不需要新的材料或工具；[Jan] 3D 打印了零件模具和结构嵌件。使用相同的工具和材料来完成不同的功能总是好的。

[Jan]正在制造的零件也很有趣。他观察到，由于多种原因，在他的打印机的构建罐中交换树脂的过程是一种不愉快的经历，其中主要是树脂又粘又脏，并且构建罐的形状使从其中倾倒树脂不是一项干净的工作。

他的解决方案是设计一个可以压在建造槽上的倾倒口，以及一些特别设计的橡胶扫帚，以便轻松地将槽刮干净。硅树脂是制作这些部件的理想材料，因为事实证明，粘性树脂会很好地附着在硅树脂的表面。在其他任何地方，树脂往往会散开，形成粘糊糊的一团，但在硅树脂上，它会形成整齐的液滴，更容易清理。

这是一个值得记住的技巧，因为没有人知道它什么时候会派上用场。例如，制造软体机器人往往需要硅树脂铸造和巧妙的技术。请参见下面嵌入的视频中[Jan]的零件。
