# 自主漫游车用激光雷达导航房子

> 原文:[https://hack aday . com/2020/09/22/autonomous-rover-navigates-the house-with-lidar/](https://hackaday.com/2020/09/22/autonomous-rover-navigates-the-house-with-lidar/)

对于那些希望探索机器人自主性的人来说，没有比边做边学更好的方法了。[Greg]就在那个营地里，他决定建造一辆自主漫游车在他的房子里漫游，【to】并且一路上学到了很多东西。

[Greg]该项目的目标是制造一个能够在没有外部帮助的情况下在家中导航的机器人。为了完成这项工作，一台 Raspberry Pi 3 负责，并配备了一台激光雷达进行测绘。Pololu Roboclaw 电机控制器随后被用于允许 Raspberry Pi 驱动机器人的各个车轮电机，从而赋予四轮机器人滑动转向能力。

[Greg]详细介绍了该项目的记录，探索了其自主能力背后的代码和概念。创造一个可以使用激光雷达导航的机器人不是一件容易的事情，但是[Greg]很好地解释了它是如何工作的，以及为什么。

这不是我们在这里看到的第一辆自主漫游车，我们确信这不会是最后一辆。如果你已经在实验室组装好了自己的产品，[一定要让我们知道](http://hackaday.com/submit-a-tip)。休息后的视频。
