# 这个老鼠标:为一个老式的德普拉兹鼠标建立一个 USB 适配器

> 原文:[https://hack aday . com/2021/08/09/this-old-mouse-building-a-USB-adapter-for-a-vintage-depraz-mouse/](https://hackaday.com/2021/08/09/this-old-mouse-building-a-usb-adapter-for-a-vintage-depraz-mouse/)

当[约翰·弗洛伦]得到一只老式的德普拉兹老鼠时，他开始满足于拥有这样一段伟大的历史。但是如果你像他一样，你知道仅仅拥有某些东西是不够的。*用*会是什么样子？

为了找到答案， [[John]开始着手为他并不算新的外设制造一个 USB 适配器。](https://jfloren.net/b/2021/8/6/1)
最初用于非常早期的 Unix GUI 终端，Depraz 鼠标使用了一种不寻常的公 DE9 连接器，而不是 RS232 串行鼠标中更常见的母 DB9 连接器。进一步偏离常规，他发现正交编码器直接连接到 DE9 连接器。

带着一台 Arduino Pro ~~Mini~~ Micro 和一些有问题的样本代码，他开始工作。前述的错误代码被废弃，Arduino Pro ~~Mini~~ Micro 的[新草图给了 Depraz 鼠标它所缺少的 USB 接口。[John]还发现他并不是第一个改装鼠标供他们使用的硬件黑客。一定要把文章读完，去发现潜藏在鼠标外壳本身的复古惊喜！在休息时间下方的视频中可以看到鼠标的演示。](https://github.com/floren/depraz-arduino)

寻找一个有趣的鼠标黑客？也许你想在复古电脑上使用更现代的 USB 鼠标，或者尝试在现代电脑上使用 T2 的早期苹果鼠标。
