# 西北大学嵌入式电子课堂的 ESP32 项目

> 原文:[https://hack aday . com/2022/12/12/esp32-projects-from-northwestern-universitys-embedded-electronics-class/](https://hackaday.com/2022/12/12/esp32-projects-from-northwestern-universitys-embedded-electronics-class/)

西北大学的嵌入式电子课程今年交付了大量 ESP-32 项目。学生团队在 hackaday.io 上记录了他们的进展，包括项目描述、日志、BOM、图表、照片和视频，以便与其他开发者分享。虽然所有团队都利用了 ESP32 提供的网络连接，但有些团队选择使用 ESP32 摄像头来整合照片、视频和计算机视觉。我们喜欢团队创建的各种项目，一些定制版本的消费产品和其他新型智能设备。

Polterchime 智能门铃项目重建了一个基本的视频门铃。在单独的网络服务器的帮助下，ESP-32Cam 提供流视频、捕获图像的存档和播放预先录制的声音文件的扬声器。查看他们的视频演示，了解他们在设计过程中使用的用户研究的更多详细信息。

[明亮的灯泡](https://hackaday.io/project/185735-bright-bulb/)看起来像是变色 LED 灯泡的简单复制，但具有一个用于 ESP32 和 neopixel LED 的漂亮定制板和一个 ESP-NOW 联网远程环境光&运动传感器。运行，你猜它是另一个 ESP-32，但电池供电依靠深度睡眠来实现合理的运行时间。

[模仿机器人手臂](https://hackaday.io/project/185833-mimic-robotic-arm/)名副其实，它看起来不像真正的人臂，而是通过使用 EMG 传感器来采集肌肉活动，模仿操作员手臂的运动。

最后， [Dum-E Smart Sentry](https://hackaday.io/project/185773-dum-e-smart-sentry) 使用一大堆 ESP32 摄像头进行全景监控，并使用云台机制将另一个摄像头直接对准任何靠近的人。

想变得更健康、更长寿，以便更好地阅读？其中一个团队创建了一个 [SmartScale](https://hackaday.io/project/185702-smartscale) 来称重并拍摄放在上面的食物的照片，使用机器视觉从食物数据库中自动传递营养信息(大多数情况下)。休息后，请观看该项目的视频，并查看其他项目页面了解更多信息。

我们感谢这些团队为他们的项目提供的文档，足以为每个项目专门写一篇文章，并且很高兴看到新诞生的黑客们一头扎进编程、使用电子设备、3D 打印，并在 hackaday.io 上与我们所有人分享他们的所学和所做。感谢关于您的学生的提示，[Ilya]！
