# 关于低音——弹珠机 X 一直在增长

> 原文:[https://hack aday . com/2020/05/16/all-about the bass-marble-machine-x-keep-growing/](https://hackaday.com/2020/05/16/all-about-that-bass-marble-machine-x-keeps-growing/)

如果你在这个问题上喊出*不是黑客*，我们没意见，因为“黑客”真的没有公平对待来自【Wintergatan】的【Martin】的创作。你可能对几年前像病毒一样传播的大理石机器很熟悉，虽然它当时给人留下了深刻的印象，而且大多数人都会宣布项目在那时结束，但它已经变成了一个似乎永无止境的在建项目，从那以后肯定走过了漫长的道路。它的最新成员:[*网络警察*作为低音](https://www.youtube.com/watch?v=BFUW_7n3O0k)的升级，你可以在它的构建视频中找到所有关于它的信息——也嵌入在下面。

如果你演奏一种弦乐器，并且曾经使用过弦枕——一种夹紧的小辅助装置来提升音高——你可能会发现自己希望可以在每根弦上的任何品上使用它。当然，有部分变调弦轴和蜘蛛变调弦轴来选择单独的弦，但是你仍然被限制在沿着单个品移调。嗯，[马丁]的电子弹夹，一个沿着脖子滑动的四个手臂的机械构造，正是为了这个目的，这让他可以腾出手来做其他事情，同时弹珠保持弹跳。

但你不必是一个贝斯手，或任何真正的音乐家，欣赏[马丁]的建设视频。当我们第一次提到大理石机器时，我们就已经称赞了他的总体态度和黑客精神，看着他对自己的工作感到兴奋，感谢支持和协助项目的人们，同时接受他的错误，这是一种真正的快乐。

不用说，[马丁]喜欢[在乐器](https://hackaday.com/2020/03/28/ondophone-on-point/)中的一些独特性，低音有单独的音量控制和每根琴弦的输出，这本身就符合条件。如果你想知道更多，休息后会有另一个视频。对于真正没有耐心的人，你可以在第一个视频中看到 12:35 左右的投诉警察课。
