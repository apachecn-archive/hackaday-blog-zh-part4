# 超级切片器评论:另一个 3DP 切片器？

> 原文:[https://hack aday . com/2021/10/26/super slicer-reviewed-another-3dp-slicer/](https://hackaday.com/2021/10/26/superslicer-reviewed-another-3dp-slicer/)

当你想到 FDM 3D 打印的切片器——尤其是免费切片器——你可能会想到 Cura、Slic3r 或 PrusaSlicer。MatterControl 有粉丝，很多人为 Simplify3D 付费。然而，有相当多的其他切片器，包括[教学技术]已经转向的:[超级切片器](https://github.com/supermerill/SuperSlicer)。你可以看看他的视频评论，在下面。

当然，就像 PrusaSlicer 是 Slic3r 的一个分支一样，SuperSlicer 是 Prusa 软件的一个分支。根据该项目的主页，切片机做了 Prusa 做的一切，但增加了定制校准测试，熨烫，更好的薄壁支撑，以及其他几个与填充和顶部表面相关的功能。该软件可以在 Windows、Linux 或 Mac 上运行。

[教学技术]多年来一直使用购买的 Simplify3D。然而，他有一个驱动器故障，不得不重建他的配置文件，所以他尝试了一些其他的选择。他不喜欢 Cura 的界面。他以前用过 Slic3r，因为那是 PrusaSlicer 的母亲和 SuperSlicer 的祖母，所以对他来说似乎更熟悉。

SuperSlicer 可以上传到 WiFi 打印机，并且有一些 Simplify3D 中没有的填充和支持选项。该视频展示了一些智能功能，如果你没有在顶层和挑檐下提供合适的支撑，可以改善顶层和挑檐。

我们喜欢有人支持克利珀。内置校准打印有助于为您的打印机和材料找到正确的回缩和其他参数。如果你没有用过 PrusaSlicer，想知道它是什么样子的，你可以[阅读几年前我们对它的看法](https://hackaday.com/2019/05/24/3d-printering-the-past-and-future-of-prusas-slicer/)。
