# 用力反馈感受虚拟道路

> 原文：<https://hackaday.com/2019/11/22/feel-the-virtual-road-with-force-feedback/>

当你驾驶着你的虚拟超级跑车在意大利乡村兜风时，你最不想要的就是不真实的方向盘感觉，这就是 [Open FFBoard](https://hackaday.io/project/163904-open-ffboard-former-ohsc) 的用武之地。赛车游戏爱好者将去不可能的，有时[难以置信的昂贵](https://hackaday.com/2017/11/24/racing-simulator-built-from-scrapheap-finds/)长度，以建立奢侈的模拟器。[Yannick]认为这些产品中的许多只是有点太贵，没有太多的需求。

现在他的董事会仍在迭代过程中，尽管它已经可以与 Assetto Corsa 整合。你可以在休息后的演示视频中看到，它对视频游戏状态的反应非常逼真，然而在寻找解决方案的过程中问题不断出现。电机驱动器烧坏，增加了功率电阻:这些能量必须转移到某个地方。接下来将切换到越来越受欢迎的 Trinamic 驱动器。无论哪种方式，我们都迫不及待地想看到下一个版本，并得到另一个惊人的模拟器建设发送给我们，也许围绕开放的 FFBoard。
