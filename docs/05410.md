# CircuitPython 现在在 Teensy 4.0 上工作

> 原文:[https://hack aday . com/2020/01/14/circuit python-now-working-on-teensy-4-0/](https://hackaday.com/2020/01/14/circuitpython-now-working-on-teensy-4-0/)

Python 经常被吹捧为初学者学习的好语言。然而，直到最近，它在嵌入式领域还不是一个可行的选择。随着 CircuitPython、[等项目的出现，这种情况已经开始改变，现在它已经可以在 Teensy 4.0 上使用了！](https://blog.adafruit.com/2020/01/10/testing-circuitpython-on-teensy-4-0-iot-made-easy-ioteensy-adafruit-circuitpython-arturo182-tannewt-nxp-paulstoffregen/)

这一里程碑部分归功于[arturo182]，他为 CircuitPython 在 iMX RT 系列微控制器上运行做了基础工作。这是由 CircuitPython 项目的负责人[tannewt]建立的。

有一些 bug 需要解决；目前，该项目正处于非常早期的发展阶段。Teensy development 的负责人 Paul Stoffregen 已经指出[在 Teensy 4.0 的存储中需要有 4096 字节的恢复分区，例如](https://forum.pjrc.com/threads/59028-Teensy-4-Circuit-Phython?p=226156&viewfull=1#post226156)。开发仍在快速进行，那些对项目何去何从有想法的人[可以在网上发表意见。](https://blog.adafruit.com/2020/01/01/what-do-you-want-from-circuitpython-in-2020-circuitpython2020-circuitpython/)

这是一个令人兴奋的发展，它为市场上更强大的嵌入式开发平台之一带来了简单的 Python 开发。我们期待看到更多的项目利用 Teensy 4.0 的力量向前发展。如果你渴望了解 CircuitPython 能做些什么，一定要看看我们之前介绍过的项目。休息后的视频。
