# 大众面具制作大师班

> 原文:[https://hack aday . com/2020/05/02/mass-mask-making-master class/](https://hackaday.com/2020/05/02/mass-mask-making-masterclass/)

正如世界各地的 3D 打印机已经在大量生产面罩一样，家庭缝纫师的螺纹注射器已经在缝合织物面罩。在过去的几个星期里，[Becky Stern]已经为朋友、家人、邻居以及她所在社区中任何偶然发现她留在附近公交车站的免费口罩盒的人制作了口罩。除此之外，她还获得了一些分数，并捐赠给了医护人员，以便他们能够延长 N95 口罩的使用寿命。

[![](../Images/46eff5f09479fa05c9a23c1d8de999c7.png)](https://hackaday.com/wp-content/uploads/2020/04/sausage-masks.jpg) 如果你要生产多种产品，那么同时生产多种产品并调整批量生产的流程是有意义的。[【贝基·斯特恩】有一些很好的想法来进一步加速组装](http://www.instructables.com/id/Make-Many-Fabric-Masks-at-Home/)，包括同时切割多个主要面具，并以循环方式熨烫几个面具的褶皱，这样你就不会在它们冷却时浪费时间。

即使你不涉足织物艺术，她制作面具的方法也是对小规模生产的有趣观察。

我们最喜欢的想法是关于侧面捆绑和带子，这是制作的最后一部分，也是最耗时的部分。[Becky]用 3D 打印的斜胶带折叠器提前制作几英里长的带子，然后将它们缝成一条连续的带子。她可以一次把短边捆绑物加到一堆面具上，一个接一个地放进去，这样它们就像香肠一样串在一起了。然后她就可以把它们剪开继续走，既节省了时间又节省了线头。休息之后看[Becky]制作一个单独的面具，看看它有多简单。

如果缝纫对你来说是一个禁忌，有很多方法可以通过启动打印机来帮助个人防护设备。
