# 利用阳光将环氧树脂变成家具

> 原文：<https://hackaday.com/2022/10/29/using-the-sun-to-turn-epoxy-into-furniture/>

环氧树脂已经被用来制作一些非常酷的家具，但由于它仍然是一种相对较新的材料，制造商仍在探索使用树脂的新技术。来自 Blacktail 工作室的[Cam]可能是第一个只用一种形式和太阳的力量弯曲完全固化的环氧树脂的人。

受到一个朋友将环氧树脂桌子放置在阳光下太久的事故的启发，[Cam]想看看他是否可以故意将环氧树脂板弯曲成有趣的形状。桌面在被刨平并进行初步表面处理之前，被分成几部分浇注，以给人一种粗略的印象。然后将环氧树脂板夹在由切角胶合板制成的模板上，并在 104°F(40 ℃)的日光直射下放置。一旦薄板在阳光下开始变形，棘轮带和更多的夹子就会被加入，以使薄板符合弯曲的形状。

让桌面放松几天后，[Cam]用大量的砂纸打磨和自动抛光程序完成了表面。然后将环氧树脂连接到一条斑马木腿上，制成了一张看起来非常现代的瀑布式桌子。

你可能想看看更多有趣的树脂项目，比如这个[海洋主题的环氧树脂小夜灯](https://hackaday.com/2022/02/12/epoxy-resin-night-light-is-an-amazing-ocean-themed-build/)、[如何在廉价的树脂上脱气](https://hackaday.com/2019/12/19/degassing-epoxy-resin-on-the-very-cheap/)，或者[用树脂冷铸的一些技术](https://hackaday.com/2016/07/14/learn-resin-casting-techniques-cold-casting/)。
