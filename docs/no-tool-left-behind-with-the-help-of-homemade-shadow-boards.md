# 借助自制的阴影板，不会留下任何工具

> 原文：<https://hackaday.com/2022/06/21/no-tool-left-behind-with-the-help-of-homemade-shadow-boards/>

隐藏式工具存储——工具轮廓一目了然地显示出存储中缺少的东西——是保持店铺整洁的一个非常聪明的方法。它们对于遗留工具可能导致悲剧的情况也非常重要。想想看，那个在喷气发动机上工作的 10 毫米插座在哪里？重要。(总是 10mm 的插座。)

但是仅仅因为阴影板很聪明，并不意味着它们很容易制造。这就是为什么[Scott Prince]想出了[这种制作工具箱阴影板](https://youtu.be/Wcxh1lLltAE)的半自动方法。在某种合适的材料上描绘每个工具并切割出形状的工作似乎很简单，但诀窍在于给定可用空间和工具的特定集合来组织轮廓。

[Scott]的方法从捕捉每个工具的图像开始。奇怪的是，他使用了一个 PiCam 和一个灯箱，放在一个储物长凳上；我们很想听到背后的完整故事，但几乎任何数码相机都可以胜任这项工作。在用 OpenCV 补偿失真、裁剪图像并将图像转换成工具的矢量轮廓后，[Scott]剩下的任务是将工具放入逻辑组并合理地布局。在调整了工具轮廓并添加了便于拾取的手指切口后，[Scott]开始使用他的 CNC 路由器。他选择使用他的雇主生产的高密度聚乙烯产品，这看起来很棒，但 MDF 也可以。

我们必须承认相当程度的工具箱嫉妒，现在我们已经看到了影子董事会能做什么。不过，我们也有点纠结——[【扎克·弗里德曼】的网格有限存储系统](https://hackaday.com/2022/04/18/gridfinity-3d-printed-super-quick-tool-storage-and-retrieval/)也有很多优点。
