# 黑掉云！

> 原文:[https://hackaday.com/2021/04/24/hack-the-cloud/](https://hackaday.com/2021/04/24/hack-the-cloud/)

对“云中”软件或服务的明显不满是，你不拥有它，你的数据不在你自己的硬盘上，或者，当互联网瘫痪时，你根本无法完成工作。但真正让我头疼的是，至少对于许多云服务来说，你不能随便使用它们。这有什么关系？好吧，作为一个黑客类型的人，当然喜欢鬼混，但是更深层次的，我觉得这种邀请鬼混才是要成长下一代黑客的东西。开放不仅对现在很重要，对未来也很重要。

当然，把这一切都寄托在云上是不公平的。有很多服务都有很好的开放 API，可以让你随心所欲地使用它们的系统——见证你可以用 Twitter 或 Twitch 做的大量有趣的事情。尽管如此，似乎每天都有另一个以前开放的 API 被一些富有的公司收购并关闭。我用一米长的 WS2812 条和 ESP8266 制作了一个很好的“今天会下雨吗”显示器，但[黑暗天空 API 被苹果公司收购了，很快就会变暗](https://blog.darksky.net/)(嘻嘻！)让我思考如何在接下来的几个月里获得简单的天气数据。

[![Whisper your tip in our ear](../Images/29edf53f9e196e638a33efaf383f5ba6.png)](https://hackaday.com/wp-content/uploads/2015/03/had-tips-scaled-300-by-300.png) 或者拿走我的邮箱信号器。我写了一个小脚本，当我收到与工作相关或来自我妻子的新邮件时(阅读:重要)，它会在我放在显示器上的 VFD 上显示主题行。这适用于我在工作中必须使用的 Gmail，因为它们支持 IMAP，所以至少当邮件到达我的服务器时，我可以做一些很酷的事情。但是，我能对我们用于[黑客在线](https://hackaday.com/submit-a-tip/)的谷歌群组做些什么吗？不可能！

所以有好的“云”也有坏的“云”。好云是开放的云。好云邀请你去玩，去创新，去想出适合自己的解决方案。好的云让你可以访问你的数据。好云是可黑客的云。让我们多看一些。
