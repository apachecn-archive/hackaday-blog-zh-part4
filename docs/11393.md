# Arduino 管弦乐队演奏行星组曲

> 原文：<https://hackaday.com/2021/09/26/arduino-orchestra-plays-the-planets-suite/>

这些年来，我们已经看到了很多 Arduino 合成器项目。我们喜欢看到一个 Arduino 发出一些单音音符。从那里，许多黑客抓住了漏洞，天空才是真正的极限。[Kevin]就是这样一个黑客，他现在拥有一支 Arduino 管弦乐队，能够演奏古斯塔夫·霍尔斯特的《行星组曲》的全部七个乐章。

表演者不是带着昂贵仪器的人类，而是简单的微控制器，运行着[凯文]亲手编写的代码。完整的管弦乐队由 11 个 Arduino Nanos、6 个 Arduino Unos、1 个 Arduino Pro Mini、1 个 Adafruit Feather 32u4 和最后一个树莓 Pi 组成。

不同的合成器处理演奏的不同部分。竖琴和贝斯上有通用的 MIDI 合成器，一个 FM 合成器处理风和喇叭部分，还有一堆继电器和伺服系统作为打击部分。整个管弦乐队聚在一起，对整部管弦乐作品进行了出色而又高保真的演奏。

虽然它不太可能赢得任何古典音乐奖，但它是对一首古典作品的迷人再现，而且来自如此多不同部分的合作更有趣。这是一种完全不同的体验，而不仅仅是在一副耳机上听 MIDI 音轨。

我们希望看到某种黑客大会来举办一场最佳硬件乐队的比赛。它本身可能会成为一种民主竞赛。与此同时，在进行这个项目的过程中，观察一下[凯文]的早期项目之一——[12 个 Arduinos 一起演唱《星球大战》曲目](https://hackaday.com/2021/06/11/12-arduino-orchestra-plays-star-wars-fanfare/)。休息后的视频。
