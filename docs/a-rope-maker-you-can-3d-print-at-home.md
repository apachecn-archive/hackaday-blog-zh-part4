# 一个你可以在家里 3D 打印的制绳器

> 原文：<https://hackaday.com/2020/02/24/a-rope-maker-you-can-3d-print-at-home/>

绳子是这样一种东西，虽然可以手工制作，但在制造过程中有一点机械帮助会大有帮助。[b33ma247]想要这样一个钻机，[所以着手从头开始建造一个。](https://www.instructables.com/id/3D-Printed-Rope-Maker/)

这是一个简单的设备，但它使任务变得容易得多。一系列齿轮被打印出来，组装到一个框架上，形成编织绳子的缠绕机构。还有一个滑道、一个绳索分离器和一个重量托架，以确保在编织过程中绳子保持适当的张力。该机构由电钻驱动，但如果需要完全手动操作，也可以很容易地用手摇曲柄代替。

这个项目表明，如果你有一台 3D 打印机，你也可以为你的工作室制作许多其他有用的工具。我们看到在卷绕线圈时也采用了类似的方法。休息后的视频。
