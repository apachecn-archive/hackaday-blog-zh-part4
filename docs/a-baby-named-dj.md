# 一个叫 DJ 的婴儿

> 原文：<https://hackaday.com/2019/06/19/a-baby-named-dj/>

我们中的一些人会因为从旧货市场、跳蚤市场和好心的亲戚那里买到有问题的硬件而感到内疚。拒绝一个好的投资和囤积之间有一个平衡，如果我们知道如何区分，你会是第一个知道的。[Clem Mayer]可能从不明智的收购开始，但他通过将一个坏掉的胎儿探测器转换成模拟合成器，将它推向了一个令人敬畏的领域。

他从一个基本的拆卸开始，我们可以看到只用两个标准螺丝就可以维修旧的硬件。这也是一件好事，因为镍镉电池比你们中的一些人更老，它们需要更换。新的镍氢电池使它启动并运行，但[Clem]没有婴儿肿块，所以它的功能转向平克·弗洛伊德时代的合成器电路弯曲。电路弯音包括修改电路，以产生原本不需要的声音。

确切的电路从未透露，因为它是在工作台上多次制作和重新制作的，但要点是从股票信号开始，并将其作为鼓机效果，其中 555 个计时器提供时钟信号和声音合成，从那里计数器和开关建立一个模式，然后进行一些滤波，回到计时器和计数器进行斩波，最后到放大器和扬声器。你注意到没有微控制器了吗？我们打赌你有。

尽管这种类型的电路来自几十年前，但有一个关于使用现成硬件的信息。我们了解到，即使我们找不到旧的胎儿探测器，现代组件和技术也会在下一次“老式”硬件进入实验室时为我们服务。

一旦电路开始运行，我们就会听到由我们的黑客创作的几首歌曲，分别是 11:40 的*二元梦*和 12:11 的*精神面条*。他几乎不是附近唯一的作曲家，看看写了一本关于电子音乐的书的[达芙妮·奥兰姆](https://hackaday.com/2018/12/04/daphne-oram-and-the-birth-of-electronic-music/)和一个[读这本书有点不同的家伙](https://hackaday.com/2018/11/14/rock-out-to-the-written-word-with-booksound/)。
