# 自制金属带锯

> 原文:[https://hackaday.com/2021/05/07/homemade-metal-band-saw/](https://hackaday.com/2021/05/07/homemade-metal-band-saw/)

顾名思义，[Workshop From Scratch]正在从头开始构建一个不断增长的工具和机器列表。他的最新版本是重型金属带锯。

和他所有的工具一样，框架由厚的焊接钢构件组成。叶片在一对经过改造的皮带轮上运行，由一台带蜗轮齿轮箱的电机驱动。刀片张力是可调的，这对刀片导向器也是可调的。为了在切割时缓慢降低刀片，[Workshop From Scratch]增加了一个带有可调阀的液压活塞，以限制降低速度。当它到达底部时，限位开关关闭电机。锯安装在一个沉重的钢台上，可以在底座上旋转，以不同的角度切割。一个重型虎钳，也是从零开始制造的，将工件安全地固定在适当的位置。

从他为自己的项目切割的钢材量来看，我们认为这把锯子会是商店里受欢迎的新增产品。令人印象深刻的是，他只用一个钻头、角向磨光机和焊机就能建造出这样的东西。视频中使用的许多其他工具，像[磁力钻床](https://hackaday.com/2020/07/07/salvaged-drill-gets-a-magnetic-upgrade/)和[液压虎钳](https://hackaday.com/2021/02/16/a-hydraulic-bench-vise-made-on-the-bench/)也是他的杰作。
