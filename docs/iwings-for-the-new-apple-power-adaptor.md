# 新款苹果电源适配器

> 原文：<https://hackaday.com/2020/07/03/iwings-for-the-new-apple-power-adaptor/>

你可能还记得旧的苹果 MagSafe 适配器，它带有可爱的折叠“翅膀”,不仅是一种实用的电缆管理工具，而且根据我们的经验，它还加速了充电器不可避免的磨损。苹果似乎已经通过在最新设计中选择可移动的 USB-C 电缆来补救后者，但完全省略弹出式电缆缠绕装置是有问题的。

[Eric]是一名工业设计师，[承担起为新一代充电器设计 3D 打印附件的责任。他的视频肯定是那些令人满意的描述之一，从概念草图到工作黑客的整个过程都整齐地包含在一个视频中。该设计很大程度上基于原始版本，在 PLA 中实现，钢琴丝作为铰链销。我们认为这是一个很好的例子，说明 3D 打印机可以用来个性化和调整商业产品，以适应特定的需求。](https://www.youtube.com/watch?v=pQAJRCcBfiE&feature=youtu.be)

如果您正在寻找一种更通用的 3D 打印电缆管理工具，请查看我们之前介绍的这款[齿轮式电缆缠绕器。](https://hackaday.com/2019/02/01/geared-cable-winder-keeps-vive-sync-cable-neatly-wound/)
