# 使用多个部分车轮的旅行自行车

> 原文:[https://hack aday . com/2022/07/31/trippy-bicycle-uses-multiple-partial-wheels/](https://hackaday.com/2022/07/31/trippy-bicycle-uses-multiple-partial-wheels/)

自行车至少需要两个轮子才能骑，但是[Q]意识到你不一定需要轮子是一个整体。只要你至少有两点与地面滚动接触，你就可以将负载分散到多个部分车轮上。他展示了这一点，他首先将自行车的后轮分成两半，然后分成三份，分别是 T1 和 T2，从而创造出一个绝对的头部翻转器。

由于传统的带有张紧辐条的自行车车轮如果被切开会塌陷，[Q]使用了单件铝车轮代替。轮胎被切割成碎片，内胎被替换成厚壁高密度聚乙烯管，在人的重量下不会塌陷。轮胎和高密度聚乙烯“内胎”被铆接在车轮上。

为了在车架上安装额外的车轮，[Q]在后面焊接了一组延伸件，带有部分车轮的安装点。为了保持同步，计时是通过焊接在盘式制动器上的链轮上的链条来完成的。在第二个视频中，他试图分开前轮，但发现前叉无法承受扭矩，当接触点太靠前时会危险地弯曲。相反，他满足于后面有三个轮子。

与他的无轮毂自行车非常相似，它的设计并不比标准自行车更好，但却非常吸引眼球。虽然至少在某些情况下，[他去年造的全轮驱动自行车](https://hackaday.com/2021/05/26/all-wheel-drive-bicycle-using-hand-drill-parts/)可能会派上用场。
