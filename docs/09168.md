# 用暖手宝煮鸡蛋

> 原文：<https://hackaday.com/2021/02/04/cooking-eggs-with-hand-warmers/>

在寒冷的天气里，戴手套可以很好地保持手指的功能，但是它们还能做更多吗？[Greg]想知道他们是否可以用在煮鸡蛋上，然后开始工作。

有问题的手工武器是 HotHands 品牌，基于铁的化学成分。取暖器通过铁和氧之间的放热反应产生热量，从而产生铁锈。因此，这些手持设备需要大量的气流来加热。为了做到这一点，[Greg] 3D 打印了一个有很多通风口的小板条箱，他在里面放了六个暖手袋。中间放着一个鸡蛋——这是一个完美的选择，因为它可以在自己的包装中烹饪。该堆达到了 160 华氏度的温度，并能够保持这一温度一个小时，这很重要，因为鸡蛋蛋白质往往会在 150 华氏度左右开始凝结和变性。一旦取出，[Greg]希望找到一个固体鸡蛋，但结果却是有点粘稠的半固体。不管怎样，鸡蛋已经达到高温，并显示出一些烹饪的迹象，因为[格雷格]似乎仍在发布视频，我们将假设他还没有死于沙门氏菌。

不过，我们希望看到这变成一场工程竞赛，奖励用最少的手工制作的最好的熟鸡蛋。[我们以前也展示过其他煮蛋设备](https://hackaday.com/2019/01/16/breakfast-bot-does-eggs-to-perfection/)。休息后的视频。
