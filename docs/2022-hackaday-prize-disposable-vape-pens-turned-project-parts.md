# 2022 年 Hackaday 奖:一次性 Vape 笔车削项目零件

> 原文：<https://hackaday.com/2022/05/05/2022-hackaday-prize-disposable-vape-pens-turned-project-parts/>

一次性 vape 笔是电子烟的一个分支，已经流行了几年了——它们是小型的独立设备，内置可充电电池和一些 vape 液体。随着电池放电和液体耗尽，整个 vape 笔通常被扔掉。然而[Dimitar]想要改变这种情况，并教我们如何尽可能多地重复使用 vape 笔——这是我们可以在项目中使用的另一个未被充分重视的零件来源。

在一份详尽的工作日志中，他分解并记录了[一支 vape 笔的内部工作，](https://hackaday.io/project/184624-reusing-single-use-vape-pens/log/204466-the-anatomy-of-a-single-use-vape-pen)加上我们放在分解处下方的视频，展示了[分解它们的方法。](https://www.youtube.com/watch?v=_vUW4QMNIxk)在这些视频中，他展示了如果我们中的任何人对恰好符合 e-cig 外形的项目感兴趣，我们可以如何重复使用外壳和塑料部件。注意触发蒸发的传感器——它可能看起来像麦克风，但实际上是一个带有高端开关的专用压力传感器！在另一个视频中，他对着其中的一个[流泪，展示如何将它重新用作电容式触摸控制器。他还打算在 GitHub 上收集一个相关资源的小型数据库](https://www.youtube.com/watch?v=Bc5MO6WgRZc)[，](https://github.com/MitkoDyakov/VapePenReuse)目前，托管着他开发的保护电路的文件，该保护电路是他安全再利用 vape pen 锂离子电池的建议的一部分。

[Dimitar]的旅程正在进行中，我们迫不及待地想看到这些组件的一些有趣用途，他在旅途中肯定会偶然发现这些用途！例如，这里有一个黑客[使用电子香烟电池](https://hackaday.com/2019/08/12/that-e-cig-battery-probably-fits-into-sunglasses/)为一副 RGB LED 装饰的太阳镜供电，取代了它们最初配备的 AAAA 电池。我们已经看到黑客们为重复使用[微波炉](https://hackaday.com/2013/09/16/one-mans-microwave-oven-is-another-mans-hobby-electronics-store/)、[打印机](https://hackaday.com/2014/09/11/re-using-the-lcd-button-assembly-from-a-broken-inkjet-printer/)和[笔记本电脑](https://hackaday.com/2022/03/17/remoticon-2021-arsenijs-tears-apart-your-laptop/)的每一个部件制作了指南，我们自己也谈到了重复使用 [ATX 电源](https://hackaday.com/2016/10/28/not-quite-101-uses-for-an-atx-power-supply/)和电脑鼠标。
