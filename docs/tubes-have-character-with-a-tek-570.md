# 管子具有 Tek 570 的特性

> 原文：<https://hackaday.com/2020/04/20/tubes-have-character-with-a-tek-570/>

当显像管为王的时候，你可以带着电视里的一箱子显像管去药店。会有一个测试器告诉你哪些管子坏了，当然，你也可以买新的。那种试管测试器非常简单。如果你真的想知道如何设计电子管或测试其参数，你最好使用类似于[Tektronix 570](https://www.youtube.com/watch?v=AphQLWbN8nI)(tomtektest)在下面的两个视频中展示的曲线跟踪器。

多亏了一位观察力敏锐的送货司机，那件装备才落到了[汤姆]的手里。1955 年的仪器非常类似于半导体曲线描迹仪，但当然，有能力为电子管提供更高的电压。基本思想是 X 轴扫描范围从几伏到数百伏。垂直刻度将显示板、屏或栅极电流。从这些曲线中，你可以了解到很多关于电子管的特性。

适应各种管引脚需要一个配线架，这将是在家里的老式电脑。你可能想知道为什么你需要商业电视上的所有这些数据。像晶体管一样，电子管也公布了数据，但是没有两个电子管是完全一样的。在某些回路中，手动分拣相同类型的管子以找到紧密匹配的管子是很常见的。

[Tom]很好地向我们介绍了设备的框图，展示了一些原理图，当然，还演示了操作。他甚至分享了一些修复测试仪电路振荡的技巧。虽然我们不怀念拖着那些笨重的动物到处跑，但不知何故，使用屏幕重量超过 25 磅的仪器会让你觉得你完成了一些事情。

如果你想[为固态而不是中空状态构建你自己的示踪剂](https://hackaday.com/2016/07/22/super-in-depth-15-curve-tracer-project/)，它并不贵。当然，如果你正在[建造你自己的地铁](https://hackaday.com/2018/12/31/the-art-of-vacuum-tube-fabrication/)，这些旧的 570 中的一个将会是你想要的票。
