# 瓦特表里有什么？

> 原文：<https://hackaday.com/2022/05/13/whats-in-a-wattmeter/>

瓦茨背后的想法看似简单。根据定义，一瓦特是一安培的电流在一伏特的电势之间流动时所做的功。想想看，一瓦特基本上就是 1V 电源通过 1ω电阻所做的功。这说起来容易，但在现实世界中如何衡量呢？[DiodeGoneWild] [在最近的一个视频中有了答案](https://www.youtube.com/watch?v=SVP3on0oHHs)，他撕开了几瓦特表。

有很多实际问题。例如，对于交流电，元件的相位很重要。视频的前 11 分钟有点像理论回顾，但后来猫介入了，我们看到了一些实际的硬件。

在第一个瓦特表中，他发现了他在视频开始时绘制的基本相同的电路，并添加了一些实用的内容，如范围选择。使用的原理与他正在画的略有不同，但核心原理是相同的:测量电压和电流，以找到功率。

我们倾向于对《主要力量》更谨慎一点，但不像其他一些著名的 YouTubers，他设法不震惊自己或放火，至少在镜头前。

我们惊叹于这些电表的机械设计，我们也喜欢家酿功率表的电源板。所以，如果你喜欢这个理论，拆卸，或者一个自制项目，这里有适合你的东西。

测量射频功率[完全是另一门科学](https://hackaday.com/2021/09/27/homebrew-wattmeter-pays-homage-to-sturdy-original/)。自然，测量瓦特数有[多种方法，并且不是每种仪器都使用相同的方法。](https://hackaday.com/2018/12/18/old-wattmeter-uses-magnetics-to-do-the-math/)
