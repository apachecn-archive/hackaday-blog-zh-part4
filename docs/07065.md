# 遥控汽车成为有用的小割草机

> 原文:[https://hack aday . com/2020/07/07/RC-car-变得有用-little-mower/](https://hackaday.com/2020/07/07/rc-car-becomes-useful-little-mower/)

当我们想到割草机时，我们首先想到的是笨重、生锈的旧机器，它们需要一个下午的大部分时间才能启动。当然，总有另一种方法，正如[马克]用他自己的身材巧妙地证明的那样。

从一个非常规的起点开始，[Mark]选择了一辆可以在两个方向翻转和驾驶的遥控汽车。失去了控制器，他开始撕掉原来的电子设备。在它的位置上，一个 ESP32 接收来自 FlySky RC 接收器的信号，并使用一个 [Sparkfun Monster 电机护罩运行驱动电机。](https://www.sparkfun.com/products/retired/10182)接收器上的另一个频道直接连接到驱动无刷电机的无人机速度控制器，配备有锯条来割草。

这是一个小平台，通常你可能会怀疑它能否胜任这项工作。然而，就[Mark]的目的而言，该钻机工作正常，并且在过去两年中一直表现良好！我们以前也见过割草机被黑，就像这个在野外的自主钻机。休息后的视频。
