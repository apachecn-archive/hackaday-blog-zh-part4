# Clean Slate 是一款受放大器启发的老式电脑

> 原文：<https://hackaday.com/2022/12/09/clean-slate-is-a-vintage-amplifier-inspired-pc/>

为现代电子产品带来复古风格的黑客永远不会过时，而[杰弗里·斯蒂芬森]带来了他的受复古电子管放大器启发的 [Project Clean Slate](http://slipperyskip.com/page43.html) 。

跳出传统的单盒 PC 的思维模式，[Jeffrey]将他的计算机构建成一系列特定组件的盒子，所有这些盒子都连接到容纳微型 ATX 主板的平台上。底座由带有鸟瞰枫木贴面的胶合板制成，每个组件盒都有两种不同尺寸的金属丝网，以操纵观众对尺寸的感知。甚至 I/O 和图形卡板也是由铝制成的。

如果你真的想深入了解这台电脑是如何诞生的，这里有一份非常详细的[构建日志](http://slipperyskip.com/forum/index.php?topic=17.0)，包括从裸板到成品的每一个步骤。我们喜欢深入了解一个产品中每个设计决策背后的思考过程。

我们之前介绍过[Jeffrey]和他的[雪茄盒集群](https://hackaday.com/2006/09/22/humidor-cluster/)，你可能也会喜欢这个装在老式收音机里的[电脑。](https://hackaday.com/2016/10/29/vintage-radio-transformed-into-custom-pc-case/)
