# 将钻床改造成铣床

> 原文:[https://hack aday . com/2020/01/05/converting-a-drill-press-into-a-milling-machine/](https://hackaday.com/2020/01/05/converting-a-drill-press-into-a-milling-machine/)

磨机是一项巨大的投资，对于没有空间安装自己的个人磨机的爱好者来说，有时不得不找到一个带磨机的设施来完成你的项目是一件痛苦的事情。

如果你能[把你的钻床](https://www.youtube.com/watch?v=3menFwDDFOI)转换成铣床会怎么样？YouTuber [ [小型金属加工机器](https://www.youtube.com/channel/UCMGnd8LN-TefAoT1fJdwGKQ) ]在他的视频中探索了这一点，他拿了一个小型中央机械钻床，并添加了一些模块。他采取了一些措施来改进套筒轴、主轴和轴承——钻下套筒轴，更换轴承，最后在 1/2-20 处转动主轴并重新攻丝。

通过调整，他能够添加一个便宜的钻夹头，它非常适合，两边只有千分之五的轻微抖动。为了引入一些控制，他增加了一个蜗轮来啮合主轴上的齿轮。一个支点脱离蜗轮，而轴承提供它从蜗轮控制的运动。

他还从易贝买了一个便宜的铣桌，装在钻床的底座上，总共 120 美元。虽然它并不完美，但仍然比买一台磨机便宜得多！
