# 人工智能带来全息手机

> 原文:[https://hack aday . com/2023/01/02/holographic-cellphones-coming-thanks-to-ai/](https://hackaday.com/2023/01/02/holographic-cellphones-coming-thanks-to-ai/)

伊萨克·阿西莫夫预见了 3D 虚拟会议，但给了他们一个尴尬的名字“三维拟人化”虽然你现在几乎可以用 VR 头戴设备和 3D 相机做到这一点，但这充其量只会很尴尬。很容易想象会议室里充满了计算机设备和扫描仪，但是麻省理工学院的一名学生有一种方法可以通过使用机器学习来简化全息图的生成。

不过，和往常一样，大众媒体可能会有点飘飘然。这里的关键突破是，你可以使用 TensorFlow 以每秒几帧的速度生成实时全息图，使用高端手机中的消费级处理能力，从带有深度信息的图像中生成全息图，这在一些手机上也可用。还有另一面显示全息图的问题，你的手机做不到。因此，任何暗示你将下载一个应用程序，使全息图电话是夸张的，这是在 photoshop 领域的图像。

尽管如此，你可以从相机实时生成 3D 图像的想法还是很令人兴奋的。即使你的手机距离全息显示屏还有几年时间，我们也可以想象它会很快进入虚拟现实耳机。毕竟，VR 头戴设备必须选择是连接到一台大型计算机，还是与一台类似手机的机载计算机连接的能力有限。

这项技术本质上训练了一个神经网络，以使用监督全息图作为参考来预测场景看起来像全息图。然而，最近的论文对此进行了扩展，使用了两阶段训练机制，一个训练集有监督，另一个训练周期没有监督

如果你想尝试这些，你需要的模型和其他东西都在 GitHub 上。这可能还不适合周末随便做黑客，但如果你是全息图像大师，也许你会找到一种方法让我们更接近你在关于这个主题的流行视频中看到的那种东西，如下图所示。
