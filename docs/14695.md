# 用科学修理一家热店

> 原文:[https://hack aday . com/2022/09/02/fixing-a-hot-shop-with-science/](https://hackaday.com/2022/09/02/fixing-a-hot-shop-with-science/)

我们知道，在北半球，几乎每个人都经历了一个地狱般的夏天，当有人大喊“哦，是吗？你觉得那很性感吗？好吧，让我告诉你……”但是你不得不同情一个住在北德克萨斯州，有一个钢制匡西特小屋做商店的人。那一定非常热。

但是愚蠢的热可以用一点小聪明来解决——或者至少减轻——这就是[Wesley Treat]用[这个巧妙设计的商店门隔热屏](https://www.youtube.com/watch?v=CmOxuwojt_c)带来的结果。当气温超过 42 摄氏度时——抱歉，这听起来远没有 108 华氏度那么可怕——他店里朝南的卷帘门变成了一个巨大的煎锅，向店里散发着空调难以控制的热量。他的想法是用折叠屏障来阻挡辐射热，但为了确保这种努力是值得的，他模拟了几个潜在的设计，并测量了每个设计的性能。他的实验向他展示了一层覆盖有反射性聚酯薄膜的挤塑聚苯乙烯(XPS)泡沫绝缘材料比只有泡沫或聚酯薄膜更好。

成品隔热板是一个巨大的三折胶合板怪兽，当商店里的东西变得温暖时，它会紧贴在门上。金属门和防护罩内表面之间的温差很大，这有望让商店更加舒适。我们想象门和屏蔽之间的空气仍然会变热，对流仍然可以将所有的热空气分配到车间中。但至少他给了进化人一个战斗的机会。

除了像这样的购物技巧和他定制的储物箱之外，[卫斯理]还是一个天才的招牌制作人。他也很有趣——或者这只是热言热语。
