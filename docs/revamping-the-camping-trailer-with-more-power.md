# 用更大的动力改造野营拖车

> 原文：<https://hackaday.com/2022/08/16/revamping-the-camping-trailer-with-more-power/>

在你的自行车后面拉一辆拖车有一种自由和探索的感觉。然而，事实是，在你的自行车后面拉一个又大又重的箱子是非常消耗体力的。所以[德鲁]回到他的[自行车露营车的绘图板，添加了一个马达，并在途中做了一些调整](https://www.youtube.com/watch?v=6SL7l2tN3fw&feature=youtu.be)。

我们报道了他对自行车拖车的第一次尝试。这个更新包含了他提到的所有未来的改进。首先，他加强了车轴，拖车支架直接固定在链撑臂上以增加强度。然后，他用 18650 节电池制造了一个定制电池组，功率不到 3kw。接下来，他在自行车的后轮上安装了轮毂电机套件。最后，一个灵活的 100 瓦光伏太阳能电池板被添加到屋顶上，并连接到内部的一个小型电池组，为野营时的笔记本电脑和手机提供 USB 和一些交流电源插座。[Drew]确实注意到他可以用较小的电池组给较大的电动自行车电池充电，但是由于电动自行车电池比较小的电池大得多，所以需要几个周期。

[Drew]参加了一次音乐节，他很高兴地报告说，即使他不踩踏板，电池也有更好的稳定性和出色的续航里程。我们喜欢看到一个好的项目被重新审视，我们希望[德鲁]能从他的野营车中得到一些好的利用。休息后的视频。
