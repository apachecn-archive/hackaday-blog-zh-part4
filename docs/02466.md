# 树莓派追踪人类，用热射线轰击他们

> 原文:[https://hack aday . com/2019/03/22/raspberry-pi-tracks-humans-blast-them with-heat-ray/](https://hackaday.com/2019/03/22/raspberry-pi-tracks-humans-blasts-them-with-heat-rays/)

考虑到人类暖身的时间，你可能会认为我们现在已经解决了所有的问题。但是即使有了中央供暖系统，事实上有时因为它，我们经常去的一些地方并不那么舒适。在这种情况下，加热人而不是房间通常是有好处的，但退一步说，这可能会很尴尬。

黑客专家马蒂亚斯·万德尔(Matthias Wandel)用这个目标跟踪红外加热器(T1)为他的冷店设计了一个解决方案。加热器是抛物面碟状辐射器中的一种，任何走过好事多展示区的人都知道，它们以非常窄的光束散发大量热量。[马提亚斯]利用[之前的一个项目](http://woodgears.ca/imgcomp/index.html)作为项目的核心，他为离线监控而突然创造了这个项目。这个定制软件运行在一个带摄像头的 Raspberry Pi 上，分析图像并定位整个帧宽度上的运动。驱动步进器旋转加热器的平台。下面的视频显示了构建和成功的测试；然而，马蒂亚斯的粉丝们应该做好震惊的准备，因为他差点买了一个懒人苏珊作为加热器的底座，而不是自己做一个。

我们从未对[马提亚斯]的视频感到失望，我们总是对他作为一名黑客的广度印象深刻。从 [DIY 电动工具](https://hackaday.com/2015/12/01/matthias-builds-a-belt-sander/)到[木制逻辑电路](https://hackaday.com/2009/10/13/binary-adder-will-give-you-slivers/)到[他最近的乐高巧克力雕刻机](https://hackaday.com/2019/03/07/watch-this-lego-pantograph-carve-chocolate-messages/)，他总能找到让事情变得有趣的方法。
