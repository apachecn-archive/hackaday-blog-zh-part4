# 混凝土 USB 集线器哪儿也去不了

> 原文:[https://hack aday . com/2019/04/16/concrete-USB-hub-not-going-anywhere/](https://hackaday.com/2019/04/16/concrete-usb-hub-isnt-going-anywhere/)

当开始一个新项目时，材料的选择会对成品的特性产生很大的影响。木头很时尚，有一定的优雅感，而抛光或拉丝铝则更具未来感。不过，有时候你只是想要又大又便宜又重的——在这种情况下，混凝土是你的朋友！

[BALES]缺少 USB 端口，需要一个具有大量连接能力的集线器。混凝土的优点是坚固和沉重，而且不渗透饮料。因此，三聚氰胺的形式产生了，选择它是因为它的表面没有给混凝土任何东西可以抓住。一个泡沫头骨被切割出来，并添加了一个镶嵌物作为装饰，7 端口章鱼式集线器被放在里面。

小心注意混合物的稠度，将混凝土倒入模具中并让其凝固。小心避免气泡，并确保混合物完全流入模具，而不会在插入的组件后面留下气穴。让它凝固几天后，将零件脱模，小心翼翼地尽量减少边缘破碎。泡沫头骨被移除，并用黑色环氧树脂填充，再用一点点来覆盖轮毂的顶部和侧面。作为点睛之笔，底部安装了一个泡沫垫，使它可以放在桌子上，而不会刮伤任何东西。

最终，[BALES]拥有了一个巨大的集线器，在插拔设备时不会晃动。在寒冷的冬夜，它还可以作为一个坚固的饮料杯垫。如果你自己正在尝试一个类似的项目，[请注意，有时混凝土会有惊人的导电性](https://hackaday.com/2017/11/12/conductive-concrete-confounds-circuitry/)。休息后的视频。
