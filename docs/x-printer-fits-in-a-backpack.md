# x 打印机适合放在背包里

> 原文：<https://hackaday.com/2019/10/02/x-printer-fits-in-a-backpack/>

3D 打印机非常适合快速原型制作，但它们通常不像你所说的那样轻便。对于【马尔特·施拉德】来说，这根本不行——[于是，X 打印机诞生了！](https://hackaday.io/project/165889-x-printer-portable)

X-printer 是一款基于 CoreXY 设计的熔融沉积打印机。它的聚会作品是其折叠手风琴式的 Z 轴，这使得打印机的体积为 160x220x150mm 毫米，而折叠时的尺寸仅为 300x330x105mm 毫米。小到可以装进背包里！

让折叠机制工作需要一些额外的努力，非线性 Z 轴需要在固件中特别注意。打印机运行 Marlin 1，选择 Marlin 1 是因为它的编译时间比 Marlin 2 快。其他设计选择着眼于耐用性。铝制框架没有想象中那么轻，但增加了急需的刚性和强度。我们希望看到一个定制的盒子，您可以将打印机滑入其中，以便在存放时得到保护。

这一构建表明，即使面对当今市场上前所未有的选择，自制自己的打印机仍有很多好处。我们以前也看到过便携式打印机概念的其他独特之处。休息后的视频。
