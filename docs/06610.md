# 它是玩具吗？原型？是黑客！

> 原文：<https://hackaday.com/2020/05/16/is-it-a-toy-a-prototype-its-a-hack/>

一些最酷的黑客用很少的资源做了很多事情。我刚刚重新看了一个来自[Homo Faciens]的视频，他在用垃圾箱里的零件和大量的独创性建造了一台令人惊讶的[功能强大的数控机器之后，意外地受到了 Hackaday 自己的[Dan Maloney]的挑战，要更进一步。[Dan]问“有人能用硬纸板和回形针制造数控机床吗？”时，他只是在开玩笑，但随后【Homo Faciens】回复:](https://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/)[纸板和回形针数控绘图仪](https://hackaday.com/2016/04/27/cardboard-and-paperclip-cnc-plotter-destined-for-self-replication/)。嘭！

纸板项目中我最喜欢的部分不仅仅是聪明的“编码器轮”,它是由一个浸在环氧树脂中的螺栓制成的，刮掉的部分足够让它每旋转一周接触一次回形针。也不是他为了模仿真实交易的功能而建造的相当复杂的可调节滑梯和方式。没有。

这个项目中我最喜欢的部分是[Norbert]解释说，由于框架弯曲，机器在这里有反冲，在那里有间隙。这是这台机器的一个优点。全金属机器会有的相同缺陷在这里都存在，但由于劣质的结构材料，你可以用肉眼看到它们，而不需要千分表。因为它能摆动可见的十分之一英寸，而专业的研磨机只能摆动不可见的千分之一英寸，这有助于你建立对系统的直觉。

这个设备不是“原型”,因为[诺伯特]不可能把它用于严肃的用途。但它肯定也不仅仅是一个“玩具”。“教学模型”听起来像是一种教学辅助工具，由一位无所不知的大师创造，旨在供学生消费。如果有的话，在这个项目中有一种真正的探索、即兴创作和直接黑客的感觉。我确信[诺伯特]从挑战中学到的和我们看他应对挑战学到的一样多。它也抓住了黑客的本质:用技术做一些意想不到的事情。
