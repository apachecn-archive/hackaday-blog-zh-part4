# BSD 为过时的设备注入新的活力

> 原文:[https://hack aday . com/2019/01/04/BSD-breaths-new-life-into-obsolete-equipment/](https://hackaday.com/2019/01/04/bsd-breathes-new-life-into-obsolete-equipment/)

曾经风光过的旧笔记本电脑或台式电脑可能仍然有一点点用在专门的任务上。抓住一个轻量级版本的 Linux，运行一个 web 服务器、防火墙或超级任天堂模拟器，可能会多花几年时间。你还可以很有创意地重新利用过时的单一用途机器，就像[Kristjan]利用一些旧的思科服务器设备所做的那样。

有问题的电脑也不是常见的东西。这是一个入侵检测系统，旨在安装在服务器机架上，保护服务器本身免受恶意活动的攻击。虽然[Kristjan]提到思科设备似乎是计划淘汰的定义，但我们认为这款配备 IDE 硬盘的英特尔赛扬机器可能在很久以前就已经过时了。不管怎样，它足够现代，可以让你以其他身份重返工作岗位。

为此，安装了一个通用操作系统，他没有使用 Linux，而是使用 BSD 来启动和运行系统。然而，除了一些冷却问题之外，还有一个问题。因为这台机器是用于服务器的，所以没有 ACPI，也就是说没有软件关闭功能。尽管有这些奇怪的地方，如果你想让它完整的话，你仍然可以用它来重新实现一个网络安全系统。
