# 用这个数控喷漆机器人给你的世界上色

> 原文：<https://hackaday.com/2019/08/06/color-your-world-with-this-cnc-painting-robot/>

比方说，你看了几集《绘画的快乐》，你想要释放你内心的鲍勃·罗斯。你得到了油画的必备用品——别忘了茜素红！–开始在画布上涂颜料，却发现你快乐的小树并不快乐，整个绘画过程比看起来要难。

[圣鲍勃]当然会鼓励你坚持下去，但如果你没有耐心，数控喷漆机器人可能是一件值得建造的东西。[John Opsahl]的“If Then Paint”背后的想法与其说是创造性的，不如说是在绘画中复制数字图像。目前处于概念验证阶段，If Then Paint 似乎有两个主要组件:油漆管理系统，通过注射泵挤出不同的油漆来实现正确的颜色，以及涂抹器本身，这是一个强大的六轴设备，通过使用不同的刷子来支持工具更换，这些刷子被夹在单独的手钻卡盘上。头部的额外轴将允许控制画笔如何呈现在画布上，也允许在颜色之间清洁画笔。下面的视频展示了[约翰]正在探索的清洁刷子的许多方法中的两种，但遗憾的是，没有一种像[正确的[鲍勃·罗斯]方法](https://www.youtube.com/watch?v=ZNAzYEM1pxo)那样令人兴奋。

看起来 If Then Paint 还有很长的路要走，但是它已经完成的一些作品给我们留下了深刻的印象。这正是我们希望在 2019 年 Hackaday 奖中看到的那种项目——经过深思熟虑，有很好的文档，有很多乐趣。
