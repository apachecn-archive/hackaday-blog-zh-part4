# 应急手电筒不用电池

> 原文:[https://hack aday . com/2020/02/14/emergency-torch-runs-without-battery/](https://hackaday.com/2020/02/14/emergency-torch-runs-without-batteries/)

手边有个手电筒以备不时之需总是好的。不幸的是，有时这些电筒会被遗忘，当你最需要它们的时候，它们却没电了。对于这些情况，[来自[techrallyofficial]的这个构建正是合适的。](https://www.instructables.com/id/Emergency-LED-Torch-Without-Battery/#C5XK4N8K5F4HT30)

火炬依靠一个 1.5 法拉的超级电容器储存能量，而不是电池。火炬的主体由 PVC 管和配件构成，内部装有强钕磁铁。一个缠绕的线圈形成在一个旧的焊接线轴上，当摇过磁铁时，就会产生电流。这由一系列二极管整流，给超级电容器充电，为灯供电。

这是一个商业上可用的经典设计，但它也很容易在家庭商店中复制。这将是一个伟大的教育项目，特别是当学生们在课程结束时可以带回家一个有用的设备。我们也看到其他人通过升级来复活商业版本。休息后的视频。
