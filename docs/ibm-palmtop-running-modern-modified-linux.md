# 运行现代(修改后的)Linux 的 IBM 掌上电脑

> 原文：<https://hackaday.com/2022/02/06/ibm-palmtop-running-modern-modified-linux/>

今天，手持计算市场似乎由智能手机主导，但在智能手机被大规模采用之前，对于那些需要移动计算能力的人来说，还有其他产品可供选择。如果 90 年代的笔记本电脑太笨重，总会有 IBM 掌上电脑，它的尺寸重量比和它诞生的时代都很强大。[Mingcong Bai]仍然保留着这些古董中的一个[，并决定通过在它上面加载一个定制的 Linux 发行版来看看它是否仍然可用](https://imgur.com/gallery/saf4PhN)。

这款掌上电脑的硬件即使在当时也很普通，它配备了运行频率为 33 MHz、内存为 20 MiB 的 Intel 486SL。这款电脑还使用了 1 GB 的 CompactFlash 卡作为存储空间，尽管[Mingcong Bai]指出，可以在上面运行 Windows 95，但这并不是一种特别好的用户体验。一个为古董硬件定制的 Linux 发行版，[AOSC/复古](https://wiki.aosc.io/aosc-os/retro/intro/)，帮助解决了一些可用性问题。有了它，只要包含以太网适配器，就可以启动命令行，甚至进行一些有限的基于文本的网络浏览。

虽然计算机正在以最大能力运行，只是为了启动和执行基本的系统功能，但令人钦佩的是，像这样的古董计算机仍然可以工作，特别是考虑到它的体积小和硬件功能有限。如果你对更多的掌上电脑感兴趣，看看第一台生产出来的电脑:HP-200LX 。
