# 压电晶体解释

> 原文：<https://hackaday.com/2019/05/24/piezoelectric-crystals-explained/>

北半球的夏天意味着户外烹饪。火柴是老式的，你更可能使用压电打火机来启动你的烤架。[Steve mold]有一个，但他不明白为什么它会工作背后的物理原理，所以他决定[做这项研究，并在视频](https://www.youtube.com/watch?v=wcJXA8IqYl8)中分享。

前两分钟是对你已经知道的事情的回顾。但在那之后,(史蒂夫)进入石英的晶格结构。利用一些电脑动画和花生酱盖子，他向你展示了压缩晶体发电的确切原因。

如果你认为你不关心烧烤打火机，[史蒂夫]提醒你，同样的效果是我们在[晶体振荡器](https://www.youtube.com/watch?v=_2By2ane2I4)中使用的石英晶体工作的原因。特别是，他研究了一个以 32，768 赫兹振动的小晶体是如何制作手表的。

我们非常喜欢第二个视频中的钢尺演示。这对科学课来说是一件很棒的事情。事实上，水晶取代了手表中的音叉，这表明了过去 50 年来事情发生了多大的变化。

如果你认为你了解人字拖，顺便说一下，你可能会发现第二个结尾有点令人惊讶，而且相当不寻常的使用人字拖。

我们以前看过旧手表的内部。我们也经常谈论[石英晶体](https://hackaday.com/2017/01/17/understanding-the-quartz-crystal-resonator/)的实际应用。但这是一组很好的演示，看起来很有趣。
