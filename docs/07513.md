# 开放式 Exosuit 项目帮助残障人士将一只脚放在另一只前面

> 原文：<https://hackaday.com/2020/08/18/open-exosuit-project-helps-physically-challenged-put-one-foot-in-front-of-another/>

人类让走路看起来很简单，但当然这是一个很容易被小伤打破的幻想。失去行走能力对你一天中的每一部分都有巨大的影响，所以康复进步无异于改变生活。不同能力项目的 Open Exosuit 正在狂热地进行他们的 Hackaday 奖参赛作品，以提供几个不同层次的帮助，让人们重新站起来。

我们在过去已经看到了许多 exosuit 项目，它们都在一些共同的地方挣扎。很难将直观的用户控制整合到这些构建中，并且非常重要的是它们不妨碍用户自己的平衡。这种机器人通过使用助步车来解决这些问题，这种助步车既提供了一种保持自身稳定的方式，又便于向宇航服发送命令。使用助行器上的有机发光二极管屏幕和按钮，用户可以选择和控制行走、坐姿和站姿模式。

外骨骼旨在为虚弱或缺乏控制力的人提供帮助。他们仍然可以自己行走和保持平衡，但希望这些设备可以在没有人类护理人员的时候提供帮助，否则就会出现不稳定的活动能力或完全丧失活动能力。使用辅助设备有助于在康复过程中不断加强力量。

该团队正在努力进行设计，距离 2020 年黑客日奖的参赛截止日期[还有不到两周的时间，我们很高兴看到最后的冲刺会给这个项目带来什么！](https://prize.supplyframe.com/)
