# 自制齿轮拯救这台碎纸机

> 原文：<https://hackaday.com/2019/06/23/home-made-gears-save-this-shredder/>

很有可能大多数读者都有过设备齿轮故障，导致设备无法修复的经历。这是一个问题，尤其是塑料齿轮，它脱落牙齿的速度比一个发现牙齿仙女的财务回报的孩子还快。

[BcastLar]有一个带齿轮的碎纸机，已经把*撕碎了*。[他发布了一个由三部分组成的视频系列](https://www.youtube.com/watch?v=_lsaYeQUaEQ)，虽然表面上是关于修理他的碎纸机，实际上是一个关于如何使用 FreeCAD 创建定制齿轮的三部分教程。虽然齿轮的原理对大多数观察者来说是显而易见的，但它们的错综复杂隐藏着重要的复杂性，他做了大量的解释工作。如何测量给定齿轮的参数，解释螺旋角或β等奥秘，他以易于理解的步骤分解一切。

他选择的工具是 FreeCAD，虽然他解释说 FreeCAD 能够从头开始制作齿轮，但视频中使用的工具是 [Gear Workbench 插件](https://www.freecadweb.org/wiki/Macro_FCGear)。他展示了该软件如何消除创建齿轮的复杂性，并在屏幕上显示了他创建自定义碎纸机部件的过程。

最后，阐述了三维打印齿轮的过程。你可能会问为什么不加工它，对此他回答说非标准齿轮轮廓的加工非常昂贵。我们把三个视频都放在了休息时间下面，我们认为你可能想给自己泡杯茶什么的，然后完成它们。

谢谢[安迪·皮尤]的提示。
