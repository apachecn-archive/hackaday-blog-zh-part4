# 扁平包装发电机

> 原文:[https://hackaday.com/2018/07/30/flat-pack-generators/](https://hackaday.com/2018/07/30/flat-pack-generators/)

我们刚刚完成了 Hackaday 奖的电能收集挑战，随之而来的是一些在非常偏远的地方获取电能的解决方案。[Vijay]的项目是最好的项目之一，因为他的项目正在南极洲获得电力。这是一个艰难的环境:你一年中的大部分时间都没有阳光，天气很冷，你需要实际上把你的设备带到欧洲大陆。[Vijay]的解决方案是将南极洲最大的资源之一——风— [用在一个精巧的平板风力涡轮机](https://hackaday.io/project/159568-portal-point-generator)上。

在贫瘠的环境中获取风能有几个问题。第一个想法是采用一个标准的现成电机，并附加一些叶片，但[Vijay]发现制动力矩太大，无论如何电机都会太大。

这个问题的解决方案是给自己的马达上发条，这种马达没有现成的无刷马达的问题。[Vijay]确定的设计是一个双轴磁通发电机，或者是一个带有固定定子和两个转子的电机，定子上有磁铁，转子上有铜绕组。你可以把它想象成你无人机上的一个扁平倒置的马达。

这种设计的一个有趣的方面是，它比传统电机占用的空间小得多，同时仍然能够在风吹的情况下输出大约 100 瓦的功率。加上一些传动装置，使转子的速度正确，你就有了一个简单的风力发电机，它可以在几分钟内安装好，并可以携带到任何地方。这是一个伟大的项目，我们很高兴看到它进入了 Hackaday 奖的决赛。
