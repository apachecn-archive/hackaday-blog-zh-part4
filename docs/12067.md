# 纸板环绕音响系统

> 原文:[https://hack aday . com/2021/12/03/paper-plate-surround-sound-system/](https://hackaday.com/2021/12/03/paper-plate-surround-sound-system/)

随着假期的到来，你可能会在家人或朋友聚会后使用纸盘子来减少洗碗的次数。但是你怎么处理多余的呢？如果你是[TKOR],你可以做一些扬声器。这个过程相当简单，如果你知道一个扬声器是如何工作的，你不会发现任何惊喜，但有一些巧妙的技巧你可能会学到。可以看下面的[视频。](https://youtu.be/3_1LHrASWdQ)

钻头和钢棒有助于线圈缠绕。你也许可以采用这种技术来制造其他类型的线圈，我们也可以安装一个编码器来计算转数。

除了作为换能器的纸盘之外，纸碗形成了扬声器的背面。他们最终得到了 16 个扬声器，这可能会很贵，但它可能不会很有吸引力，这取决于你的时尚感。

最后，他们用绝缘泡沫板制作了一个巨大的扬声器，尽管他们使用了现成的驱动器，你可以想象，比他们使用的自制驱动器声音更好。

听起来怎么样？很难说，因为你是通过视频听到音频的，但听起来还不错。我们喜欢他们将泡沫板悬挂在天花板上，形成一个完整的房间音响系统。一点油漆和一些 LED 效果会比纸盘子更适合装饰。他们甚至将一些木板切割成装饰形状，并制作了一个播放音乐的标志。

这些可能不适合音响发烧友，因为对一个扬声器来说，有比看起来更多的东西。还可以用 3D 打印做一些[音箱魔术](https://hackaday.com/2021/03/25/speaker-is-fully-3d-printed/)。
