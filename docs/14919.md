# 今晚观看美国国家航空航天局撞击小行星的探测器

> 原文：<https://hackaday.com/2022/09/26/watch-nasa-crash-a-probe-into-an-asteroid-tonight/>

今晚有什么计划吗？没有吗？那么你很幸运，因为作为双小行星重定向测试(DART)的一部分，美国国家航空航天局(NASA)在几个小时后将故意撞击小行星 Dimorphos(T1)，这标志着人类第一次故意试图将太空岩石撞离轨道。如果成功的话，我们就离拥有一个可行的行星防御系统更近了一步，以防我们探测到一颗小行星与地球相撞。如果不行。。。嗯，我们还有时间想出另一个计划。

需要澄清的是，170 米(560 英尺)宽的双形字**不会对我们构成任何威胁，在美国国家航空航天局用离子推进的航天器撞击它之后也不会。这只是一个测试，看看一个小航天器正面撞击一颗小行星是否能让它减速到足以明显改变它的轨道。我们在一周左右的时间内不会知道撞击是否成功，但现场观看撞击发生仍然是令人着迷的。**

我们在下面嵌入了两个 NASA 流。第一个节目将在撞击前半小时开始，将在 DART 飞船瞄准目标时显示 Dimorphos 的实时导航图像，第二个节目将报道主要事件。请记住，这不是我们在谈论的好莱坞电影——不要指望当时钟到达零点时会发生任何戏剧性的爆炸。当遥测数据停止传回时，这意味着这是一个靶心。
