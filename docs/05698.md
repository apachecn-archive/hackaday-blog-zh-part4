# 不安全的监控摄像头提供了反乌托邦的西洋镜

> 原文:[https://hack aday . com/2020/02/18/不安全-监控-摄像头-提供-反乌托邦-西洋镜/](https://hackaday.com/2020/02/18/insecure-surveillance-cameras-provide-dystopian-peep-show/)

当你听说全世界有成千上万的联网摄像头被设置为默认凭证时，你可能不会感到惊讶。实际上，可能还有更多，但我们可以向你保证，至少有 70，000 个左右只需点击一下鼠标。通过这个项目，[Caroline butte]证明了在我们摇摇欲坠、荒谬可笑的监视状态下进行艺术创作是完全可能的——而[它始于窥视孔视角](https://www.hackster.io/carolinebuttet/virtual-peephole-355c1c)。

你自家前门的窥视孔赋予你不可剥夺的权利，监视你的门廊、门廊或公寓楼走廊的地毯，而几乎不会被发现。在虚拟窥视孔中，窥视孔变成了窥视世界各个角落的虚拟视图。

拉开盖子，连接到 Arduino Micro 的 LDR 会检测到光线水平的变化。这种变化使微处理器向 Raspberry Pi 发送一个按键，Raspberry Pi 随机获取一个新的相机，并在盒子内的屏幕上显示它。休息之后，您可以观看一个简短的演示，随后是几个简短的构建/演练视频。

如果你是一个偷窥者，在那里放一台相机，从任何地方观看。
