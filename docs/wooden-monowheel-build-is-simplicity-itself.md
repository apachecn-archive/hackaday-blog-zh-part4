# 木制独轮建筑本身就很简单

> 原文：<https://hackaday.com/2021/08/23/wooden-monowheel-build-is-simplicity-itself/>

Monowheels 不是什么新东西，在 19 世纪中叶首次获得专利，但由于很多明显的问题，从未真正成为主流。我们遇到了前方视野、稳定性、转向方面的问题，特别是在速度方面，还有一个有趣的名字“沙鼠”,骑手可以在车轮内旋转，就像沙鼠在车轮中一样。欢乐时光！但很明显，这并没有阻止[Q]用木头做一个来增加 monowheel 的主体。

有时候，人们接受这些项目只是为了好玩，就像我们不久前报道的这个亮橙色项目。有时它们由电机驱动，无论是电动的还是内燃的。有些是手摇的，有些是脚踏的，都试过了。

[The Q]对有趣的木制建筑并不陌生，这个一年前的视频显示他建造了一个非常简单的直接踏板驱动的 monowheel。绝大部分的结构是木头，用老式的方法胶合和拧紧，在必要的地方用一点金属加工。我们特别喜欢简单的配重解决方案，它可以兼作停车制动器。这可能看起来有点笨拙，但我们想不出更简单的、更有意义的解决方案。

休息后的构建视频是六分半钟的精心制作的视频，供您观赏。
