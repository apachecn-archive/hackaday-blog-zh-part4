# 这有一个特斯拉线圈，你可以戴上

> 原文：<https://hackaday.com/2019/04/18/heres-a-tesla-coil-you-can-wear/>

这是 badgelife 的季节，如果你需要一个可穿戴电子产品杀手的想法，[看看这个 PCB 特斯拉线圈](https://hackaday.io/project/164819-open-pcb-tesla-coil)就知道了。是的，这是致命的，如果你戴着心脏起搏器的话更是如此。

这个项目的灵感来自于[一个早期的 PCB 项目](http://www.megavolts.nl/en/projecten/tesla-spoelen/201-pcb-spiral-teslacoilBut)上的特斯拉线圈，它使用电路板上的 160 匝 6 密耳走线作为次级线圈。所有的电子设备都在那里，它由 USB 供电。插上这个东西，你就有了一个装满大约 30 千伏的闪电的口袋。如果你碰它，它可能不会杀了你，但我们不要测试太多。[Bobricious]采纳了这一想法并付诸实施，将电路精简到最低限度。现在它只是一个晶体管，所有其他部分都印刷在电路板上。

在 PCB 上制作特斯拉线圈有一个问题，那就是线圈的匝数。你会发现的任何特斯拉线圈实际上只是一根细线缠绕数百或数千次的巧妙应用。这个特斯拉线圈没有什么不同，在这种情况下，它是缠绕在 150 平方毫米 PCB 上的 240 匝单迹线。[Bobricius]是在 PCB 上放置微小线圈的王者之一，[他的玻璃纤维无刷电机](https://hackaday.com/2018/03/28/a-brushless-motor-on-a-pcb-made-from-pcb/)就是一个证明。我们还刚刚介绍了[的圆形直线电机滚道](https://hackaday.com/2019/04/16/circular-linear-motor-becomes-a-micro-motor-raceway/)，它也使用 PCB 线圈。

电路很简单，只有一个接受大约 20 伏电压的电源插孔、一个 BD243 晶体管、一个 LED 和一个 82k 电阻。有了这个，你就可以在 PCB 上放一个小霓虹灯管，看着它亮起来。用另一个 PCB 和另一个霓虹灯管，这个电路板可以传输无线电源。这是一个有趣的玩具，而且都是 PCB 技术。
