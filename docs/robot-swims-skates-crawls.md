# 这个机器人会游泳、滑冰和爬行

> 原文：<https://hackaday.com/2018/12/18/robot-swims-skates-crawls/>

你经常听说艺术模仿生活，但有时技术也是如此。受范能源系统公司的 Velox 机器人更像一种水下生物，而不是机器人，因为它使用波状鳍来推进自己，正如你在下面的视频中看到的。

视频显示这头野兽会滑冰，也会游泳和走路。它看起来更像一个生命体，而不是一个装置。据该公司称，该机器人具有出色的静态推力/瓦特，并能防止被植物和其他碎片缠住。

这份报告的缺点是缺少技术细节。然而，因为提到了[专利](https://patents.google.com/patent/US8610304B2/en?inventor=Benjamin+Pietro+Filardo&oq=Benjamin+Pietro+Filardo)，所以通过这种方式找到更多信息相当容易。通过尝试解码专利法律术语，看起来关键是使用换能器(我们会说致动器)来弯曲材料的柔性鳍的变形。换句话说，塑料鳍“弹出”(或进入)，致动器拉或推它变形，然后让它弹回到原来的位置。我们认为。

虽然他们确实显示它在固体表面上滑行，但我们打赌它在任何不太平坦或摩擦系数高的地方都会有问题。不过，在水里看起来确实不错。

如果你搜索该公司拥有的其他专利，你会发现像很多东西一样，你可以反向操作该设备，并用它从移动的液体中提取能量。这也给了我们一些想法。

传动系统让我们想起了我们见过的许多蛇形机器人。如果你喜欢像我们一样游泳，有 [Swumanoid](https://hackaday.com/2012/06/11/swumanoid-a-swimming-humanoid-robot/) (网站已经死了，但视频还在)。
