# 逃生室的经典电视比喻

> 原文：<https://hackaday.com/2022/08/03/a-classic-tv-trope-for-an-escape-room/>

如果没有一条穿过书架的秘密通道，任何一座幽灵大厦都是不完整的——至少好莱坞是这么教导我们的。电影中的陈词滥调在密室里也同样适用，只要有密室和科技相结合，[Alastair Aitchison]就不远了。他的最新创作:你猜对了，[是一扇隐秘的书柜门](https://www.youtube.com/watch?v=GY9U429J6Mg)。

在本教程中，他将一个普通的书架安装在一扇木门上，门本身作为书架的背板，使用门铰链作为主要的移动机构。他知道一旦装满了书，它会变得有多重，所以他在底部增加了一些脚轮作为支撑。至于(开锁)机制，[Alastair]确实考虑过在门的背面安装一个机械锁，由一根绑在书上的电线拉动。但由于安全是他主要关心的问题之一，他希望将没有紧急出口的人被锁在里面的风险降到最低。一个自动防故障的磁锁连接到一个 Arduino 上，一个切断开关作为替代方案。

由于他的主要目标是一个逃生室，使用 Arduino 还可以将更多种类的暗门集成到谜题中，以及实际解锁的方法。用右手触摸等离子球来解决魔方或[怎么样？](https://hackaday.com/2019/06/24/plasma-globe-reveals-your-next-clue/)
