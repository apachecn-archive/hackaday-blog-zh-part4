# 从管和机翼到单翼:客机的未来

> 原文：<https://hackaday.com/2021/08/08/from-tube-and-wing-to-just-wing-the-future-of-airliners/>

客机已经成为现代生活中不起眼的一部分，但除非你碰巧是一个飞机爱好者，否则你会被原谅认为最新的空客型号看起来或多或少与开创喷气式飞机时代的波音 707 相同。但这种情况可能很快就会改变，混合翼客机看起来像是航空旅行效率的下一步。在广告之后的视频中，[真实工程]带我们领略了喷气式客机的过去和未来。

当代的客机仍然沿用同样的老式“管翼”设计，但是效率大大提高了。与 50 年前相比，最新的喷气式飞机每乘客公里耗油量减少了近 50%。这要归功于更好的发动机、改进的空气动力学、减轻的重量以及一系列其他的、通常看不见的变化。然而，看起来需要更剧烈的变化来保持进展，美国国家航空航天局、波音公司和空中客车公司都在赌混合机翼设计能做到这一点。

混合翼飞机基本上是飞翼，飞机的载货部分更短，更宽，并产生升力。这种布局可以通过取消尾翼来增加飞机的内部容积，并改善空气动力损失。研究表明，混合机翼设计可以减少多达 27%的燃料消耗。由于载荷和产生的升力更均匀地分布在飞机的整个宽度上，这也减少了机翼所需的结构加固量，特别是在根部。大的内部体积也允许使用其他能源，如氢燃料电池。

混合翼飞机并非没有挑战。它们天生不稳定，需要复杂的控制系统才能飞行。这些控制系统依赖传感器、致动器和软件来正常工作，并且需要多级冗余。这些冗余的遗漏最终导致了 2008 年 B-2 轰炸机的坠毁，以及最近波音 737 MAX 客机的致命坠毁。此外，与管状机身不同，混合机翼设计不是理想的压力容器。然而，这不是一个主要问题，因为碳复合材料可用于制造坚固、轻质的结构。

随着飞机技术的飞速发展，我们期待看到未来会发生什么。无论是个人旋翼机还是[商业太空飞行](https://hackaday.com/2021/06/01/virgin-galactics-long-road-to-commercial-spaceflight/)，肯定不会无聊。
