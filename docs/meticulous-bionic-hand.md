# 精细仿生手

> 原文：<https://hackaday.com/2020/11/12/meticulous-bionic-hand/>

[威尔·科格利]正在缓慢但坚定地制作一只漂亮的仿生手。(视频，嵌在下面。)设计中投入的大量工程和思想令人难以置信。那些认为他们的手理所当然的人通常不会考虑他们的手指可以移动的不同方式。有侧向运动、旋转、弯曲和伸展。总的来说，[Will]试图用可以 3D 打印或容易获得的零件来设计机械装置。这将手限制在像伺服系统、电缆驱动或直接驱动这样的东西上。

然而，拇指有一个特别棘手的运动范围。因此，拇指[将]设计成使用蜗轮传动方法来产生拇指的弯曲和伸展运动。这些齿轮需要加工，以便经得起负荷。连接到主蜗轮的小侧 3d 打印齿轮连接到电位计以形成反馈回路。因为它不承受任何负荷，所以可以 3d 打印。虽然还有数百个小问题有待解决，但剩下的大问题是电线管理，最终确定 IP(指间)关节，以及将整个组件连接到前臂。

所有的 step 文件、大量的研究和定义[都在【Will 的】GitHub](https://github.com/ikkalebob/NM-bionic-hand/) 上。如果你想制造任何类型的手假肢，那么[威尔]在这方面的研究和关注将会融入到你的项目中。我们已经在之前[见过仿生手，也见过](https://hackaday.com/2016/08/12/hedberg-is-a-bionic-hand-made-from-a-single-keurig/)[铝制手指替代物，](https://hackaday.com/2020/10/22/what-if-you-could-design-your-own-aluminum-hand/)但这是一只拥有神奇范围和逼真度的完整手。
