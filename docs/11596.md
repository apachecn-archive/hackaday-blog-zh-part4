# 一个重新设计的 ZX 频谱台式电脑，工作得非常好

> 原文：<https://hackaday.com/2021/10/17/a-redesigned-zx-spectrum-desktop-computer-that-works-surprisingly-well/>

复古计算机爱好者经常会思考他们爱好中的重大问题。如果 Commodore 有一个像样的营销部门会怎么样是一个流行的说法，但辛克莱研究公司出了名的吝啬方式也是一个丰富的来源。如果辛克莱赢得了英国学校的计算机竞赛，不仅是第一次 Acorn 的 BBC Micro 获得奖项，而且如果他们在 1983 年再次加入竞争，又会怎样？[10p6]研究了这种可能性，[推出了一款 Spectrum 台式电脑](https://www.youtube.com/watch?v=QVhYvCXFslE)，你可以在广告下方的视频中看到。

视频的前三分之二是渲染，虽然看起来很漂亮，但没有任何实质内容。在后面的部分，我们看到了一个构建，将 Spectrum 48k 板、接口 1 和两个微驱动器与电源一起放在一个细长的外壳中。与此同时，一个 ZX 橡胶键盘独立安装在电缆的一端。我们知道，这是一台过去许多孩子渴望拥有的计算机，我们同意视频中的说法，即它可以集成到一个主板上，而不需要单独的接口 1。我们觉得这是不可避免的，尽管 Sinclair 的成本削减会导致一些东西误入歧途，而且肯定只会有一个 Microdrive，尽管我们非常喜欢那个独立的键盘。

他们声称 STLs 将从脸书的一个小组获得，但是除非你碰巧有一组微驱动器和一个接口 1 来配合你准备为这个项目大开杀戒的频谱，我们猜测主要的兴趣在于观看它的展开，并且一些想法可能会转化到其他平台。同时，如果你对 Microdrive 感兴趣的话，[我们去年拆除了它们](https://hackaday.com/2020/09/18/the-zx-microdrive-budget-data-storage-1980s-style/)。
