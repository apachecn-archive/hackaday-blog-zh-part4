# 抓住斯坦福计算机科学教育

> 原文：<https://hackaday.com/2020/10/14/grab-a-stanford-computer-science-education/>

上学有两个原因:学习一些东西和得到一张你梦寐以求的能帮你找到工作的纸，或者至少是工作面试。有这么多学校把材料放在网上，只要你不指望学校帮你或给你那张纸，你就可以不花多少钱做第一部分。斯坦福大学有一个庞大的计算机科学系，[马瑞]在网上以某种形式提供超过 150 门计算机科学课程。隔离期间消磨时间的好东西。

显然，[芮]拿着 2020 年的课程目录找到了校园课程，并找到了每个课程的配套网站，为我们的利益组织它们。这个列表不包括实际的在线课程，你可以直接从斯坦福大学找到，尽管还有另外一个列表。

在尝试一些高级课程之前，了解 CS103 计算的数学基础等基础课程是很重要的。系统类别从实用 Unix 上的 CS1U 到涵盖并行计算的 CS149 和 CS315B。

如果你想了解更多关于你的 LED 闪烁算法扩展性差背后的理论，有 CS254，计算复杂性。现在机器学习很流行，深度学习研究有很多选择，包括 CS20，Tensorflow。

还有更多的，但我们会让你读列表。电话编程、机器人、视频游戏和生物计算都在菜单上。甚至还有一些课程，如工程师个人理财、伦理和电脑音乐。

公平地说，并不是所有这些类都很容易自己消化。有些是辅助材料，你真的需要教授的讲座。然而，对于像 [CS106L(标准 C++编程)](https://www.classcentral.com/university/stanford)这样的一些课程，你会发现完整的课堂笔记，一些课程甚至有视频讲座，就像下面 CS224N 的一个，深度学习的自然语言处理。

我们很高兴(芮)整理了这份名单。当然，如果你是为了自学而抓取大学内容，你[不必从一个地方](https://hackaday.com/2018/01/04/hack-your-own-computer-science-degree/)获取所有内容。如果你想要一个非常实用的教育，[海军的材料](https://hackaday.com/2017/06/09/electronics-education-courtesy-of-the-us-navy/)是另一个来源。
