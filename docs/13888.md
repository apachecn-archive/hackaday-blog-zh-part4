# 以下是往钢琴里装满水的方法

> 原文:[https://hackaday . com/2022/06/10/heres-what-it-take-to-fill a-piano-with-water/](https://hackaday.com/2022/06/10/heres-what-it-takes-to-fill-a-piano-with-water/)

对许多人来说，往钢琴里倒水听起来可能既无聊又愚蠢。然而，这也引发了人们对它会是什么样子的好奇。[谢天谢地，[Mattias]花了几个小时的时间来寻找答案，这样我们就不用这么做了！](https://www.youtube.com/watch?v=38VG3oaY4OQ)

![](../Images/510d17d956878da77559c71285a58560.png)

It doesn’t make a great pool, though.

立式钢琴的第一次尝试很快失败了。在水中浸泡仅仅四分钟后，木锤就会因受潮膨胀而卡住。

为第二次尝试找到了一架大钢琴。弦首先被放松，以便更容易操作，内部框架从周围的钢琴体中撬出。为了防止水溢出琴键和琴弦，我们实施了一个简单的解决方案:将钢琴向上倾斜，让水留在下方的琴体中。然后明智地使用各种密封剂来密封框架。令人惊讶的是，密封钢琴的最佳信息来自用胶合板箱建造水族馆的爱好者。去想想。

正如你所料，水对钢琴的声音有静音效果。当通过放置在充满水的空腔中的水下麦克风听到声音时，这种声音特别引人注目。它听起来几乎像是一种弹拨乐器，让一切都有一种奇怪的海洋感。声波也可以在水面上看到。

当钢琴被灌得太满，水溅到琴键和琴槌上时，这个实验悲惨地结束了。这导致每一个键卡住，永远杀死了钢琴。

如果你能忍受看一架钢琴被这样处理，这是一个有趣的建筑，也是一个非常愚蠢的建筑。[马蒂亚斯]在古怪乐器领域也有自己的风格，正如我们之前提到的他的氦吉他。休息后的视频。
