# 使用 CT 扫描仪深入了解逆向工程

> 原文:[https://hack aday . com/2022/08/10/a-使用 ct 扫描仪进行更深入的逆向工程/](https://hackaday.com/2022/08/10/a-deeper-dive-into-reverse-engineering-with-a-ct-scanner/)

我们最近了解了[Ken Shirriff]如何使用工业 CT 扫描仪作为逆向工程工具。结果是惊人的，照片清楚地显示了自 60 年代该模块被封装以来尚未公开的部件的内部排列。现在，[肯]的团队[好奇的马克]已经发布了一个视频，提供了这台奇妙机器的更多细节，并深入探讨了更多阿波罗时代的硬件

如果你喜欢看[Ken]用来对模糊的触发器模块进行逆向工程的剧照，你会喜欢看[Marc]使用 Lumafield 扫描仪的 3D 软件对几件阿波罗文物进行无损检测。首先进入 CT 扫描仪样品室的是一个名为中央计时设备的密封模块，它是阿波罗指令模块的主时钟。盒子的镁外壳被证明对 CT 扫描仪的光束没有任何障碍，而且根据一系列 2D 图像建立的 3D 模型细节惊人。虚拟模型最好的部分是能够在任何平面上切割它们— [Marc]利用这个功能找到了时钟的石英晶体。

[马克]的阿波罗陀螺仪是下一个，密封的外壳里面的外观很有启发性。机械结构的细节令人惊叹，一直到支撑陀螺转子的轴承。一个曾经风光无限的电源模块也得到了这种待遇；它的扫描显示爆炸的电容器造成了它粗糙的外观。所有的扫描数据都可以在 Lumafield 的网站上公开获得，不过如果你想玩这些模型，你需要创建一个账户。

至于扫描仪本身:它可以在家里制造吗？也许吧。我们见过很多自制的 x 光机，甚至还有 T2 的 CT 扫描仪和 T4 的两台 T5。如果你解决了这样的问题，请告诉我们——我们很想看看里面的情况。
