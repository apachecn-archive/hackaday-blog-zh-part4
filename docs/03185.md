# 当你忘记哈蒙德器官的存在时，你就会得到运动合成器

> 原文：<https://hackaday.com/2019/05/30/the-motor-synth-is-what-you-get-when-you-forget-hammond-organs-exist/>

从来没有新的东西。都做完了。但这并不意味着你不能发明一些有趣的东西。一个恰当的例子是 Motor Synth，[一个来自 Gamechanger Audio](https://www.indiegogo.com/projects/motor-synth#/) 的众筹项目。这就是你将先进的四轴飞行器技术与模块化和半模块化合成器市场相结合的结果。

电机合成器的核心功能是一个隐藏在有机玻璃窗户后面的八位无刷电机。这些(通过电磁拾音器或稍微聪明一点的东西)产生一个音调，给马达合成器四音符复调，每个键两个声音。在这些马达的顶部是用红外探测器感应的反射光盘。这些被混合为基频的谐波。结果呢？这个月早些时候，他们在 Superbooth 得到了[让·米歇尔·雅尔]的支持(见下面的视频)。真是令人印象深刻。

虽然使用旋转的轮子和马达似乎是一种产生声音的新方式，但这实际上是第一个“合成器”产生声音的方式。[音轮风琴](https://en.wikipedia.org/wiki/Tonewheel)实际上是一个轮圈上有凸起的金属轮子(想象一下类似齿轮的东西)，它在磁性拾音器旁边旋转。当车轮旋转时，这些突起在拾音器中感应出电流，该电流被发送到放大器并输出到扬声器，产生一个单音。这是上世纪初发明的，在哈蒙德风琴中得到了显著的应用。转速脉冲轮绝对有局限性；每个轮子只产生一个频率，并且除了将整个装置调谐到标准音调之外，不能改变。电机合成器通过使用标准的无刷电机*和*绕过这一限制，在每个电机上安装一个反射盘，用于红外传感器，从而产生每个“车轮”的谐波。这些谐波可以与基本的“马达”音混合。

虽然这绝对是下一代“产生音频的转盘”技术，但马达合成器最引人注目的是它的新颖性。为什么直到现在都没有人把吉他拾音器放在无刷电机旁边？任何人都可以将一个四轴飞行器马达和一卷电线塞进 Eurorack 模块，并获得“The Verge”或“T2”主板的赞誉。我们认为，仅仅因为没有什么新发明，并不意味着你不能创造一些有趣的东西。
