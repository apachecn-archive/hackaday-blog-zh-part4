# 有什么值得保存的？

> 原文：<https://hackaday.com/2020/11/14/what-is-worth-saving/>

下雨时，倾盆大雨。支撑波多黎各阿雷西博天文台碟形天线的一根主要支撑电缆[刚刚断裂](https://hackaday.com/2020/11/11/tensions-high-after-second-failed-cable-at-arecibo/)，离开了它已经不确定的命运。它在 2017 年被飓风玛丽亚严重损坏，经过几年的筹款，修复工作正要开始修复这一损坏，电缆断了。因为剩余的电缆现在承受着增加的重量，所以在灾难性故障的风险被排除之前，人类不允许在穹顶上工作——他们正在通过无人机进行检查。

阿雷西博天文台[已经运行了很久](https://hackaday.com/2020/08/24/damage-to-arecibo-leaves-gaping-hole-in-astronomy/)。它最初是冷战时期洲际弹道导弹跟踪雷达的一部分，这解释了为什么它既能接收信号又能发射信号(T2)。这是世界上最大的发射天线。它被用于 SETI，提供了引力波的第一个线索，并发现了第一个重复的快速射电爆发。它的雷达能力意味着它可以用于小行星探测。让它继续运行的原因有很多，不仅仅是它的历史意义。

因此，当我们刊登这个故事时，许多评论者担心会发生最坏的情况，写下了他们的哀悼。但是有些人写信来，对它可能无法修复表示愤慨。常见的疑点出现了:未能在科学或基础设施上投入足够的资金。然而，从旁观者的角度来看，或许在进一步的结构研究完成之前，我们不知道修复阿雷西博要花多少钱。之后，我们必须决定是否值得。

根据 2018 年的拨款，国家科学基金会正在与由中佛罗里达大学领导的财团分摊 2000 万美元的维修和维护费用，该财团将管理该网站。随着进一步的破坏，这可能是一个低估，但我们还不知道有多少。

你什么时候决定停止这种事情？虽然阿雷西博是最大的，但它并不是唯一的发射器。然而，下一个最大的发射器是深空网络的一部分，正忙于与我们太阳系周围的航天器保持联系。对于纯接收，[中国的 FAST 更大更好](https://en.wikipedia.org/wiki/Five-hundred-meter_Aperture_Spherical_Telescope)。当然，自从阿雷西博被设计出来后，我们已经了解了很多关于射电望远镜的知识。

我不是说如果阿雷西博没有得到修复，我们不会流泪，但这并不是说国家科学基金会的预算受到了巨大的打击，或者他们没有意识到各种大型天文项目的相对价值。如果没有站在他们的立场上，阅读了成千上万的竞争拨款提案，很难说花在支撑一个 70 岁的望远镜上的钱不会更好地花在其他事情上。
