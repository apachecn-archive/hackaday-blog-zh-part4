# 一个射频识别戒指的身体模神经质

> 原文：<https://hackaday.com/2019/09/08/an-rfid-ring-for-the-body-mod-squeamish/>

一些人纹身，而另一些人纹指甲花或那些你可能在一盒饼干盒里找到的水转印纹身。[Becky]想要手指上有 RFID 标签的好处——用手指轻轻一划就能开门或登录你的电脑——但还没准备好植入。她的解决方案是:制作一个隐藏了微型玻璃胶囊 RFID 标签的艺术戒指。

除了不必在你的表皮下塞进一些技术，还有一些其他的优势:你可以像换戒指一样容易地更换标签。你也可以很容易地把你的戒指借给别人，就像你可以把你的房门钥匙给他们一样。

这个环类似于由细长的柱子连接的两个环。三个支柱中的两个是金属的，但第三个是 RFID 标签。我们想知道标签和支柱是否可以采用某种珐琅来隐藏其外观，但不干扰操作。

[Becky]是个相当不错的工匠，甚至在 Instructables 上有一个免费的在线珠宝课程。我们也对她紧凑整洁的工作空间印象深刻，你可以在下面的视频中看到。

这不是我们看到的第一个 [RFID 戒指](https://hackaday.com/2014/03/16/stuffing-an-rfid-card-into-a-finger-ring/)，但它看起来很漂亮。另一方面，你也可以牺牲[的一张信用卡，](https://hackaday.com/2019/04/25/rfid-payment-ring-made-from-dissolved-credit-card/)但那有它自己的问题。
