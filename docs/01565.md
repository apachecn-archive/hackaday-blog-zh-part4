# 令人毛骨悚然的叮当声和随机噪音的曼哈顿之谜解决了

> 原文：<https://hackaday.com/2018/12/15/manhattan-mystery-of-creepy-jingles-and-random-noises-solved/>

给你一个难题:如果你说的东西不完全是电话，那你还是电话吗？

这个问题可能从来没有出现在纽约人的脑海里，这个夏天，他们在曼哈顿通常平静的人行道上被街边 WiFi 亭发出的令人毛骨悚然的声音所攻击。听觉攻击在当地引起了轩然大波，导致了俄罗斯黑客是幕后黑手的疯狂理论。幸运的是，[谜团已经被解开](http://gothamist.com/2018/12/13/mister_softee_linknyc_video.php)，结果证明这部分是恶作剧，部分是抗议，部分是行为艺术作品。

为了理解这种利用，要知道纽约市最近已经从城市人行道上移除了数千部传统的付费电话，并用 LinkNYC 亭取而代之，这些亭基本上是内置巨型高清电视显示器的 WiFi 热点。任何人都可以使用内置的 VOIP 应用程序打免费电话，但代价是在散步时被广告轰炸。这是让[马克·托马斯]发起攻击的关键，他是这些平台取代的付费电话的老派发言人和铁杆粉丝。这不完全是火箭手术。[Mark]拨打他用预先录制的音频片段设置的几十个电话会议号码中的一个。一分钟的延迟让他把扬声器的音量调到 11，然后潜逃。录音各不相同，但每个人似乎都被熟悉的冰淇淋车特许经营权的顺口溜吓坏了，放慢速度，扭曲声音，让它听起来像是发烧梦里的东西。

是的，这是一个最小的黑客，通常我们不会宽恕滥用公共设施，即使是像 LinkNYC 这样令人讨厌的。但它确实对公共广场的商业化做了一个声明，老实说，我们很高兴看到一些至少接近 phreaking 的东西。这比[从目标 PA 系统](https://hackaday.com/2016/03/04/social-engineering-your-way-to-the-target-pa-system/)播放色情音频少了一点孩子气，也比[远程激活公共安全警报器](https://hackaday.com/2017/04/12/every-tornado-siren-in-dallas-hacked/)危险得多。
