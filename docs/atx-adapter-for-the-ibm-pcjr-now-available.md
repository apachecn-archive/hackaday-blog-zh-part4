# IBM PCJr 的 ATX 适配器现已上市

> 原文：<https://hackaday.com/2018/09/10/atx-adapter-for-the-ibm-pcjr-now-available/>

我们之前提到过维护老式计算机带来的挑战，在某些情况下，这些计算机已经使用了 40 年。组件，即使是高质量的组件，最终也会出现故障，需要更换。现在，如果它是一台相当受欢迎的老式机器，替换零件通常不会太难得到。但是，如果你面对的不仅仅是一台老式的机器，而且是一台商业上的败笔，零件稀缺，那该怎么办呢？

这就是任何一个拥有 50 万台 IBM PCJrs 的人的生活，在这款产品上市的一年左右时间里，蓝色巨人成功地卖掉了这款产品。正如[AkBKukU]发现的那样，为奇数球计算机更换一个交流适配器将比他支付的费用更高，所以[他开始致力于创造一个适配器，这样他就可以在机器上使用现代的 ATX PSU](https://github.com/AkBKukU/ATX2PCjr)。经过几个月的努力，这个设计终于可以上市了。

最后，PCB 设计本身相当简单。这实际上只是将一些引脚从标准的 ATX 插头切换到 PCJr 上的边缘连接器。还有一个为软盘驱动器供电的连接器，以及风扇和电源开关的接头。

[AkBKukU]提出了两种使用适配器的方法。你可以选择标准的 ATX PSU，在这种情况下，由于它的尺寸，它需要放在机器外面，或者使用 PicoPSU，它允许你把整个东西放在内部。如果你不介意花钱，PicoPSU 方法是一种更清洁的安装方式，仍然可以提供充足的电力。根据您选择的路线，有不同的 3D 打印板来调整计算机的后面板，以适应新的硬件。

构建您自己版本的所有文件都在 GitHub 库中，而且[AkBKukU]正在对套件和组装适配器进行少量运行。如果这个项目看起来很熟悉，那是因为[我们曾报道过它，当时它还是一个手动*划伤的* PCB，并不总是像预期的那样工作](https://hackaday.com/2018/08/10/ibm-pcjr-revived-by-an-atx-power-supply-and-many-false-starts/)。
