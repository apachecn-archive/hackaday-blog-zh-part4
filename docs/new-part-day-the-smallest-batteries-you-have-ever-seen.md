# 新零件日:你见过的最小的电池

> 原文：<https://hackaday.com/2022/11/15/new-part-day-the-smallest-batteries-you-have-ever-seen/>

由于锂聚合物袋电池的制造，我们已经习惯了小型化电子产品中的一些非常小的电池。但它们仍然很大，而且它们几乎不是最稳定的储能解决方案。不过，法国 ITEN 公司可能会给微能源设备的设计者一个答案，那就是一系列微型表面贴装固态可充电锂电池。这些器件的容量范围从 0.1 mAh 到 0.5 mAh，在 3.2 乘 2.5 mm 的封装中，看起来非常像任何其他稍大的 SMD 芯片元件。

这些设备最有可能出现在远程无线传感器等应用中，它们可以存储来自小型太阳能电池或类似设备的能量，以产生传输一个数据包所需的突发功率以及保持正常运行所需的微小电流。固态化学应该提供长的寿命并且没有泄漏。目前他们提供一些评估套件，除非我们遗漏了什么，否则没有完整的数据手册。我们特别有兴趣了解焊接时它们的温度敏感性，因为我们已经牢记焊接到更传统的锂电池上的警告。

通过 [CNX 软件](https://www.cnx-software.com/2022/11/14/100-uah-micro-battery/)。
