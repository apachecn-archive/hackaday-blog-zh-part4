# 建造一把火药驱动的锤子

> 原文：<https://hackaday.com/2021/10/20/building-a-hammer-powered-by-gunpowder/>

锤子是非常简单的工具。如果你需要更大的冲击力，就用更大的锤子。或者，你可以通过化学手段来提高成绩，我们不是指使用类固醇。不，相反，你可以尝试借助火药进行锤击，[，【我做了一件事】正是这样做的。](https://www.youtube.com/watch?v=Lti_zl3MnT4)

建造依赖于使用为 Ramset 品牌的爆炸钉子驱动器设计的 6.8 毫米空白弹药筒。然而，我没有购买现成的工具，而是以传统的 hammer 格式构建了一个工具。该设备看起来像一个锤子，在两件式头部上有一个铰链，可以将一个空白的弹药筒放在里面。当锤子在坚硬的表面上挥动时，冲击力会触发空白，以令人难以置信的力量推动钉子前进。

[我做了一件事]能够用这个装置穿透钢铁，还能把一颗钉子穿过冲浪板。这是一件非常危险的事情，所以如果你在这个空间做实验，一定要小心。休息后的视频。
