# 蜂鸣器进行输入和输出

> 原文：<https://hackaday.com/2021/01/25/buzzer-does-input-and-output/>

压电元件具有双向的有用特性；也就是说，当你给它们通电时，它们可以移动，但当你移动它们时，它们也可以发电。[Carl]利用这一事实制造出能够提供触觉反馈的[按钮](https://www.youtube.com/watch?v=egLA0GgF3xw)。你可以在休息时间下面看到他努力的视频。

他做了两个版本的按钮。一种使用 3D 打印外壳，另一种使用夹层结构的 3D 打印垫片。正如您将看到的，我们尝试了几次才把它做好。这些元件接受并产生相对较高的电压，因此大部分工作是来回调整电压。事实上，他甚至设法用更高的电压烧坏了他的 CPU 芯片。

我们可能会寻找一种更简单的方法来感测按钮的按下，因为这似乎是一个很好的电路来做到这一点。但是整个电路提供了一个输入按钮，触觉反馈，以及将蜂鸣器用作蜂鸣器的选项，所以如果你需要所有这些功能，至少它是相对经济的。
