# 暴风云灯将天气带入室内

> 原文:[https://hack aday . com/2019/10/19/storm-cloud-lamp-bring-the-weather-inside/](https://hackaday.com/2019/10/19/storm-cloud-lamp-brings-the-weather-inside/)

对于寻求创造性表达自己的黑客来说，不起眼的灯是一种常见的构造。通常，大自然可以作为灵感的来源，就像迈克尔·皮克的暴风云一样。(视频，嵌在下面。)

在电子方面，构建非常简单，由一个 Arduino Uno、一个 MP3 盾和一串 WS2801 LEDs 组成。这些驱动与更常见的 WS2812B 类型略有不同，但 Adafruit 库甚至让初学者也能轻松使用。还有一个射频密钥卡，用于设备的远程控制，还有一个语音合成器，用作用户界面。

视频还包括灯体的构造。创建纸板模型，然后用胶带覆盖，以创建一个粗略的 3D 椭圆形状。这个模具然后用玻璃纤维制成两个外壳，然后用螺栓连接在一起。这使得 led 和电子设备可以整齐地安装在内部。然后用喷胶将看似棉絮或多填充物的东西粘在外面，创造出云的效果。

最终的结果是相当美观的。正如你所料，这里有常见的柔和的彩虹，但真正的亮点是风暴模式，这使得闪烁的灯光分散在云的表面。伴随的声音效果从 MP3 屏蔽有助于增加戏剧。
