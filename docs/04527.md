# 这台干冰驱动的造雾机非常适合万圣节

> 原文:[https://hack aday . com/2019/10/14/this-干冰驱动的雾机是万圣节的完美选择/](https://hackaday.com/2019/10/14/this-dry-ice-powered-fog-machine-is-perfect-for-halloween/)

树叶变黄了，更恐怖的季节即将来临。如果你目前正在为一个顶级的万圣节派对制定计划，[你最好考虑用【DIY Machines】](https://www.instructables.com/id/Ultimate-Dry-Ice-Machine-Bluetooth-Controlled-Batt/)制作一个类似这个单元的烟雾机器！

这种造雾机是基于干冰的，所以你需要从外部供应商那里采购。该机器由一个装满热水的密闭容器组成，容器内部是一个装满干冰的可移动桶。将干冰放入水中，雾就产生了。

Arduino 用于控制水桶，允许通过智能手机应用程序控制产生的雾量。还有内置的可控发光二极管，给雾一个合适的怪异的辉光。该建筑依靠一系列 3D 打印零件来实现这一机制，并采用几种不同的喷嘴设计来实现不同的效果，如上升的间歇泉或厚厚的低洼雾。

基本概念很简单，任何人都可以用一台 3D 打印机和一个亚马逊账户在一个周末内完成。这是增加万圣节气氛的一个很好的方式，但是当然，[这并不是雾能做的全部](https://hackaday.com/2014/04/17/interactive-3d-projection-is-foggy-at-best/)。休息后的视频。
