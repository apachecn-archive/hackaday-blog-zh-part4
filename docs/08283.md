# 与危险握手

> 原文:[https://hack aday . com/2020/11/10/retro tech tacular-shake-hands-with-danger/](https://hackaday.com/2020/11/10/retrotechtacular-shake-hands-with-danger/)

好了，你得带上你的安全斜眼，坐下来欣赏这个:美国重型设备制造商卡特彼勒 1980 年的经典安全宣传，名为“[与危险握手”](https://www.youtube.com/watch?v=v26fTGBEi9E)

事实上，你可能也需要为这部 23 分钟的电影使用你的劣质滤镜，因为歌词和主题曲都很难接受。这部电影是那些“直吓”的尝试之一，旨在表明当工作由比人的血肉和骨头更坚固的材料制成时，无论是在野外还是在商店里，事情都会变得多么糟糕。在这方面，这部电影非常有效——我们发现自己在几个地方有点反胃，那个手被吸进台式研磨机的可怜家伙既可怕又令人同情。[三指乔]确实。

现在，你可能会对表演提出异议，但当你观看所有这些短片时，请记住，这些都是老派的特技——他们撞毁的实际上是一辆巨大的 D9 推土机，刹车室爆炸真的炸毁了那辆卡车的窗户。他们做得很好，把一时疏忽的潜在后果描绘得令人作呕。尤其是挽臂那一幕。呃。

无论你用什么方式练习黑客艺术，都要注意安全。不要“与危险握手”
