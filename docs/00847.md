# 这个火箭炉灶是热的东西！

> 原文:[https://hacka day . com/2018/10/05/this-rocket-cookwoot-is-hot-stuff/](https://hackaday.com/2018/10/05/this-rocket-cookstove-is-hot-stuff/)

如果你在网上搜索，你会了解到人类很久以前就开始用火来烹饪食物。事实上，你可能会认为在基于火焰的烹饪中没有什么新东西。幸运的是[Bongodrummer]没有得到那个特别的备忘录，因为他创造了一个相当不寻常的火箭炉烤架，能够烹饪大量的食物。

火箭炉被设计成通过实现高表面积燃料的最完全燃烧来尽可能高效地利用能量。它的特点是燃烧面积小，烟囱带有补充空气，以确保废气也能燃烧。这个把所有的热气直接送到烤盘，然后通过一对烟道把它们带走。作为一个额外的奖励，有一个比萨烤箱的圆顶附件，是在以前的项目中有一些剩余的建筑材料时制作的。看看休息时间下面的综合构建视频。

也许令人担忧的是，燃烧室和烟囱是由气瓶制成的，但使用中央供暖散热器是一个非常好的主意。底部的涡流进气口和烟囱上方的二次空气喷射器完善了这个装置，是传统烧烤的绝佳替代品。

值得一提的是，这不是我们看到的第一个火箭炉，有[这个简单的设计](https://hackaday.com/2013/03/05/quick-and-easy-rocket-stove/)以及[这个设计非常好的空间加热器](https://hackaday.com/2016/11/27/diy-rocket-mass-heater-build-log-for-commercial-greenhouse/)。
