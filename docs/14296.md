# 机器学习婴儿监视器防止饥饿游戏

> 原文：<https://hackaday.com/2022/07/23/machine-learning-baby-monitor-prevents-the-hunger-games/>

新生婴儿可能很难弄清楚，尤其是对第一次做父母的人来说。尽管以前生过孩子的人(以及许多没有生过孩子的人)提供了大量未经请求的建议，但大多数新父母很快就适应了婴儿经常发出的模糊信号。但是[Caleb]将他对新生儿的观察更进了一步，建立了一个机器学习的饥饿婴儿早期预警系统,这非常巧妙。

通常情况下，婴儿对饥饿并不感到不安，他们会大声向世界宣告自己的需求。但事实证明，哭泣是饥饿的滞后指标，在哭泣之前有大量的面部、头部和手部暗示。[Caleb]他的系统基于谷歌的 MediaPipe 库，使用他的婴儿监视器的摄像头来跟踪诸如咂嘴、拒绝奶嘴、握拳和生根等行为，所有这些都是某人的肚子需要填充的迹象。通过建立一个系统来识别这些线索并给它们分配一个权重，[Caleb]现在可以在婴儿进入尖叫阶段之前收到一条短信，这不仅有利于小 nipper，也有利于他那些睡眠不足的仆人。下面的视频有一些无价的比特在里面；不要错过 5:11 的[婴儿迦勒]或者结尾搞笑的自动喂食器插科打诨。

我们最近看到了一些来自[Caleb]的有趣视频，大部分是关于[他的狗的浴室习惯](https://hackaday.com/2022/01/17/ai-camera-knows-its-st/)和[在事后得到帮助清理](https://hackaday.com/2022/05/24/point-out-pups-packages-with-this-poop-shooting-laser/)。我们只能猜测，当这个孩子长大一点，开始如厕训练时，这些项目将如何发挥杠杆作用。
