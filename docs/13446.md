# 物理控制面板提升飞行模拟体验

> 原文：<https://hackaday.com/2022/04/26/physical-control-panel-elevates-flight-sim-experience/>

像我们中的许多人一样，[pgsanchez]已经被飞行模拟器的错误所困扰。这是一种只能治疗，但永远无法治愈的疾病——就像许多爱好一样，它有滋生更多爱好的令人讨厌的趋势。作为一名软件开发人员，[pgsanchez]也擅长 Arduino 和电子产品，他关于 [PGS-2 飞行模拟器控制面板](https://pgsanchez.net/pgs-2-diy-sim-control-panel)的博客帖子很好地展示了他的优秀能力，休息时间下面的视频也是如此。

作为数字战斗模拟器的玩家，他厌倦了不得不记住笨拙的组合键来控制模拟器。驾驶喷气式飞机，即使是在模拟器中，也需要敏捷的思维和快速的反应，所以有一个按钮按下，一个开关翻转，或一个旋钮转动，甚至比最简单的基于键盘的命令都要优越得多。

Arduino 将按钮连接到计算机，白色丙烯酸外壳用于保持所有部件保持队形飞行。是的，一个白色的盒子——非常小心地让盒子有背光。效果非常好，看起来面板就像在苏霍伊 Su-25T 的家里一样，它被设计成在游戏中控制。

我们赞赏面板中对细节的关注，因为即使是起落架状态灯和襟翼指示器也与模拟器中的相匹配，这是一个不错的触摸！[pgsanchez]还能做什么？我们倒想看看！如果你喜欢飞行模拟人生之类的东西，你可能会对这个[全 3D 打印的飞行模拟人生控制器](https://hackaday.com/2020/09/03/3d-printed-flight-controls-use-magnets-for-enhanced-flight-simulator-2020-experience/)感兴趣。
