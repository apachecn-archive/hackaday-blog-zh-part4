# 疯狂的数字电台

> 原文：<https://hackaday.com/2022/11/17/number-stations-gone-wild/>

(曼彻斯特环路)对数字电台感兴趣。这些神秘的电台可能发送编码数字或其他编码信息。然而，很少有人声称这些电台的功劳。通常，他们像军队一样精确地运作，遵守严格的操作时间表，无误地发送信息。当这些间谍站的事情[没有按计划进行时【Ringway】看了五次。](https://www.youtube.com/watch?v=vIW-7IsSaHg)

然而，也许这并不奇怪，因为机器可能已经取代了人类操作员。这使得他们在计算机出错时容易出错。许多错误是频率错误，两个数字站同时发送。我们认为间谍都使用相同的频率。然而，有些人的计算机也出了故障，开始遍历字母表，当然，这可能是某种秘密信息协议的一部分，但似乎更可能是一个简单的错误。

然而，我们被逗乐了，听到一个捷克间谍站的故事，它不仅有一个许可的呼号，而且会给报告接收的人发送 QSL 卡片。也许他们没有得到保密备忘录！

在我们这个时代，我们已经听过几个数字电台。如果你没有合适的天线，你可以试试[在线猎杀它们](https://hackaday.com/2015/09/09/steal-this-ham-radio-technology/)。但是不要指望发现他们犯任何错误。
