# 下载一点辛克莱的历史

> 原文:[https://hack aday . com/2020/04/06/download-a-bit-of-Sinclair-history/](https://hackaday.com/2020/04/06/download-a-bit-of-sinclair-history/)

如果你是 Sinclair 系列 8 位家用电脑的爱好者，那么来自剑桥计算历史中心的一条新闻可能会让你感兴趣，[他们发布了他们的 ZX 光谱原型的 rom 副本](http://www.computinghistory.org.uk/det/51620/Sinclair-ZX-Spectrum-Prototype/)。这部机器去年作为最初签约编写光谱 ROM 的公司的捐赠形式的一部分浮出水面，从那以后一直被放在他们展览的首要位置。他们一直在做一些非常仔细的工作，虽然[寄存器报告](https://www.theregister.co.uk/2020/04/02/zx_spectrum_prototype_rom/)他们还不能使董事会启动，他们已经提取了代码进行研究。在中断下方的视频中，我们看到它在一台旧的 Windows PC 上的 Speccy 模拟器上运行。

该 ROM 邀请 ZX 频谱社区对照股票版本进行分析，希望揭示僵化的代码片段，如从未进入股票频谱的 Microdrive 存储外设。但是如果你只是想用最早版本的 ROM 尝试你最喜欢的游戏，你也可以这么做。

我们去年报道了机器的出现，同时，如果你还没有去过计算历史中心，我们建议你[看一下我们几年前的评论。](https://hackaday.com/2017/03/30/review-centre-for-computing-history/)
