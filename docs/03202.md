# 采用旧技术的新电路

> 原文：<https://hackaday.com/2019/06/02/new-circuits-with-old-technology/>

在晶体管发明之前，真空管统治着世界。当时，获得放大或开关(或任何电流的电子控制)的唯一方法是使用电子管。但即使在那时，一些电子管设计的局限性也是显而易见的。首先，它们在正常运行时会产生大量的热量，从而导致可靠性问题。电子管很难小型化。令人欣慰的是，晶体管解决了所有这些使真空管过时的问题，但如果你想稍微调查一下过去的情况，市场上仍然有一些电子管。

[kodera2t]能够得到其中的一些，它们似乎相对较新。这并不奇怪；有一些特殊的应用仍然使用电子管。这些设备也比它们的祖先有了一些改进，与一些旧设备的数百伏电压相比，它们的工作电压只有 30 伏。[kodera2t]带我们通过这些电子管构建的几个电路，从简单的超小型真空管收音机到更复杂的[反射式收音机](https://en.wikipedia.org/wiki/Reflex_receiver)。

回顾这段历史是一项有趣的活动，值得看看晶体管电路与电子管电路的不同之处。但是，如果你有足够的兴趣超越简单的无线电电路，你也可以开始[用真空管](https://hackaday.com/2014/06/04/keep-those-filaments-lit-design-your-own-vacuum-tube-audio-equipment/)建造自己的音频设备。
