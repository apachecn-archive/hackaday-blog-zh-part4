# 让速食拉面更方便一点

> 原文：<https://hackaday.com/2019/07/21/making-instant-ramen-a-bit-more-instant/>

速食拉面，一文不名的未来科技天才最喜欢的食物！它很便宜，营养价值可疑，而且只需要一分钟就能做好。但是如果你在创意区到了五分钟烧水太多的程度呢？为此，你需要一个自动拉面炊具，这就是[Mayermakes]从一个升级的电动过滤咖啡机中创造的[。](https://www.youtube.com/watch?v=M8f3jSmpM2w)

过滤咖啡机是一个令人惊讶的有效的方便面炊具，因为它已经包含了一个电热板和一个煮水器，可以将热水滴在一些面条上。但是它缺乏任何添加调料或必要的辣酱的方法，所以他创造了一个由步进电机驱动的 3D 打印旋转漏斗和一个伺服驱动的注射器，而咖啡机本身被赋予了一个固态继电器来打开它。

控制表演的是一个 Arduino MKR 板，它提供了一个网络界面，可以选择拉面或辣酱拉面。结果是一罐 0.49 美元的自动面条，不会让任何美食家心动。话说回来，高级餐厅并不是速食拉面存在的原因。

这似乎是我们的第一个煮拉面的咖啡壶，但是我们已经看到了用面条做的吉他！
