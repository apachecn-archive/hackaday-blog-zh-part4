# 钢骨增强 3D 打印

> 原文：<https://hackaday.com/2019/04/22/steel-reinforced-3d-prints/>

继续关于如何让 3D 打印更强大的永无止境的冒险，【Brauns CNC】带着[一种涉及钢筋增强 3D 打印部件的新技术](https://www.youtube.com/watch?v=XpDG8VxZsw4)向我们走来。

我们已经看到了许多创建更强的 3D 打印的方法，从使用碳纤维丝到以打印层与力的方向正交的方式简单地打印零件。我们甚至已经看到将碳纤维棒铸造成 3D 打印品，但当然这只适用于直的部分。[Brauns]技术使用钢丝，嵌入到打印本身，从一些测试来看，该部分的强度增加了约 50%。

将钢缆嵌入 3D 打印零件的过程只是简单地拆开模型，并为钢缆放置一个通道。在特定的层高度，打印机停止，在烙铁的帮助下嵌入钢缆，打印机继续做它的事情。

有少量的 Gcode 黑客攻击来实现这一点，将钢缆嵌入印刷品的过程有点挑剔。不过，如果你想要更强的 3D 打印，有更差的方法，当然也有效率更低的方法。你可以看看下面这个技巧的视频。
