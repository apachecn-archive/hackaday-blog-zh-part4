# 无线耳塞自己充电

> 原文：<https://hackaday.com/2021/08/11/wireless-earbuds-charge-themselves/>

随着越来越多的端口从我们的智能设备中移除，我们似乎有两个选择来使用外设:要么购买一个加密狗来继续使用有线设备，要么切换到蓝牙并处理永久维护的电池。然而，如果这两个选项都不适合你，还有第三个选项可用，正如[beffinitiv]在这个构建中向我们展示的那样，他将一个微型太阳能电池板集成到他的耳塞外壳中，以允许它们自动充电。

首先，他从拆开耳塞外壳开始。对于那些还没有尝试过这些设备的人来说，他们通常只在放在手提箱里时充电，在他的箱子里还包含一个小电池。将电线直接焊接到电池上可以让电池充电，而不会像他连接到电路板上的 USB 引脚那样有太多的电力损耗。即使这样，电池也只能产生一伏电压，所以他需要一个 5V 的升压转换器来给电池充电。不过，这也有它自己的问题，因为它不适合放在箱子里。为了解决这个问题，他拆下了所有组件，并将它们固定在一起，以便将转换器安装到一个更小的空间中，而不必以任何其他方式修改外壳。

完成所有这些工作并在外壳上安装小型太阳能电池后，[befinitiv]有了一个智能解决方案，可以让他的无线耳塞充满电，而不必每天带着电缆或软件狗到处跑。我们已经看到了许多有趣的解决方案来解决各种电子产品制造商移除无处不在的 3.5 毫米耳机插孔的问题，但并不是所有的解决方案都解决了这个问题[而没有出现其他一些奇怪的结果](https://hackaday.com/2018/05/21/wireless-headphone-hack-dangles-batteries-like-earrings/)。
