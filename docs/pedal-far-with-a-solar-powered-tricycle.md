# 骑着太阳能三轮车远行

> 原文：<https://hackaday.com/2018/11/28/pedal-far-with-a-solar-powered-tricycle/>

最近，越来越多的电动自行车出现在街头，因为人们意识到与汽车相比，电动自行车是多么便宜、容易骑行和使用。它们也可以像普通自行车一样蹬动，所以仍然可以用它们进行一些锻炼。大多数电池的续航里程大约为 10-30 英里，这取决于电池的大小、重量和空气动力学，但通过一些升级，如太阳能电池板，充电后可以走得更远。

[漫无边际的牧羊人]有一辆三轮车(在美国，从法律角度来看，通常仍被视为自行车),他已经将它转换为带有轮毂电机和电池的电动三轮车，并在用它补充手动脚踏时获得了令人难以置信的里程。然而，他想做得更好，并决定在他的建筑中增加一些太阳能电池板。他的第一次尝试并不顺利，因为面板的 3D 打印支架失败了，但通过快速修改，他的第二次尝试在 50 英里的旅程中幸存了下来。更令人印象深刻的是，在旅程开始时，他的电池只充了一半电，但由于电池板增加的能量，他仍然能够成功。

如果你认为这看起来很熟悉，我们最近推出了一款[双人三轮车，它用类似的设计从欧洲到中国](https://hackaday.com/2018/07/16/row-your-bike-to-china/)进行太阳能旅行。与普通自行车相比，它的优点是允许骑车人在阴凉处以相对舒适的骑行姿势骑车。未来计划的升级包括 MPPT 充电控制器，以提高电池板的效率。
