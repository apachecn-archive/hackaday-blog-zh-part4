# 使用这款鼻子形状的洗手液分配器，对着你的手打喷嚏，而不是你的胳膊肘

> 原文:[https://hack aday . com/2022/07/09/用这个鼻形消毒剂分配器对着你的手而不是你的肘打喷嚏/](https://hackaday.com/2022/07/09/sneeze-into-your-hand-not-your-elbow-with-this-nose-shaped-sanitizer-dispenser/)

你必须喜欢像德语这样的语言，它不仅可以用一两个词来代表一个复杂的概念，而且你甚至不需要说这门语言就能猜出它的意思。当然，当你的项目是一个向你的手里模拟打喷嚏的消毒剂的大鼻子时，你称它为 T2、T4 并不重要。毕竟，基于身体机能的幽默是一种通用语言。

不管有没有德语工作知识，弄清楚(尼娜)到底在这里做什么并不太难。从下面的视频和构建日志来看，这个想法是通过一个简单的红外反射传感器连接到某种微控制器来检测分配器下面的手的存在，在这种情况下是 ESP32。打喷嚏的音频剪辑存储在 SD 卡上，并通过一个小扬声器播放，而一个业余爱好伺服按下雾化器上的按钮。似乎选择合适的分配器是这个项目中最困难的事情；[尼娜]最后选定了一个电池驱动的先生，大小正好适合鼻子。哦，我们没提到那个巨大的，粉红色的，3D 打印的鼻子吗？抱歉，这很微妙，很容易被忽略。

无论如何，整个项目很有趣，当我们看到它时，带来了真正的笑声。这是一种巧妙的方式，对那些想出其他不那么古怪的分发洗手液的方法的细菌恐惧症患者开一个温和的玩笑。但是让我们面对现实吧，他们最终被证明在很多事情上都是正确的。
