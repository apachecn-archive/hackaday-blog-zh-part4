# 是一艘船？它是一只鸭子？是 DIY 保鲜膜皮划艇！

> 原文：<https://hackaday.com/2020/09/14/its-a-boat-its-a-duck-its-a-diy-plastic-wrap-kayak/>

很少有电影时刻像《狮子王》中[木法沙]的死亡和《T2》中[威尔森]的死亡一样令人心碎。想想看，如果[汤姆·汉克斯]扮演的角色在被冲走的货物中找到一卷~~角色~~的弹性包装，他就可以建造一艘有足够空间安全存放他忠实伴侣的船。听起来不太可能？嗯，[【sg19 point 3】不敢苟同，并且有一艘皮划艇可以证明](https://www.instructables.com/id/10-Shrink-Wrap-Kayak/)。

它的建筑材料简单明了:树枝、包装带，当然还有弹力包装。[sg19point3]使用了两种不同类型的树枝，一种弯曲的程度刚好能使皮划艇的长度成型，另一种更灵活，可以形成将它固定在一起的环。去掉树皮后，他用地上的钉子把树枝做成需要的形状，然后让它干几天。一旦准备好了，他就把它们放在一起，用包装带固定结构，直到它准备好用几层保鲜膜包裹整个东西的压轴戏。为了证明他相信自己的作品，他带着它去了最近的水边，并活着讲述了这个故事——并制作了一个视频，在休息后嵌入。

诚然，在一个偏远的岛屿上独自组装可能有点费力，所以好事[sg19point3]有一些朋友来帮助包装。你是否想把它带到你当地的浅池塘之外，这可能是另一个故事——你肯定想避开锋利的岩石。想要更结实的东西，可以看看几年前的 3D 打印皮划艇。但是如果你喜欢木头，[这里有一艘漂亮的独木舟](https://hackaday.com/2017/03/19/you-will-want-to-build-this-canoe/)。
