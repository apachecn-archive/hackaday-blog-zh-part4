# 你不应该尝试这些泰瑟枪 NERF 飞镖

> 原文：<https://hackaday.com/2018/10/12/you-should-not-try-these-taser-nerf-darts/>

对于我们大多数人来说，童年的很大一部分都是在别人的后院(或房子里)跑来跑去，试图用一把玩具 NERF 枪击中目标。有趣的程度很高，人身伤害的风险很低。既然我们大部分都是成年人，可能是时候[把我们的 NERF 游戏带到下一个级别](https://www.youtube.com/watch?v=6RJ9I1HnniI)了，有一些严重的人身伤害的风险。

为了帮助他的哥哥报复他年轻时的恶霸行为，[艾伦·潘]送给他一把升级版的 NERF 枪。具体来说，就是那种有飞镖的。每个“精英”飞镖内部都装有一个 300 V 的电容器。新的尖端是 3D 打印的，带有特殊的金属尖端，允许电容器在受到冲击时放电。

除了危险之外，还涉及到一些科学问题。零件是从一个新的(价格惊人地昂贵)一次性相机中收集的，在飞镖枪的枪管周围构建了一个定制的电路，允许飞镖在装载时充电。这是一个令人印象深刻的构建，对你来说重建起来相对简单，但它可能不是我们见过的用高电压和几个小电容做的最糟糕的事情。
