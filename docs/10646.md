# 机翼可以展开，在短距离起飞和降落时飞得非常慢

> 原文:[https://hackaday . com/2021/07/10/wing-can-expand-to-fly-really-slow-for-short-take-and-landing/](https://hackaday.com/2021/07/10/wing-can-expand-to-fly-really-slow-for-short-take-off-and-landing/)

[迈克·帕蒂]因制造高性能实验飞机而声名鹊起。在他的最新项目中，他增加了一个[变形机翼，可以将其弦长延长多达 16 英寸](https://www.youtube.com/watch?v=JcckHHiYkL0)，以实现低速和大迎角性能。

问题中的飞机是一架名为 [Scrappy](https://hackaday.com/2021/01/30/scrappy-drag-racing-bush-plane/) 的丛林飞机，早在【迈克】开始制造机翼之前就已经引起了人们的注意。专为极短的起飞和着陆(STOL)性能而设计，只有机身框架的一些部分保留了原始的 Carbon Cub 套件。机翼是定制设计的，前缘有双缝翼，后缘有大襟翼和下垂副翼。对于正常飞行来说，缝翼形成了几乎无缝的机翼部分，但可以使用集成到每个精密机器翼肋中的一系列连杆来扩展。通过广泛使用 CFD 模拟，缝翼被设计成保持升力中心接近机翼中心，即使襟翼为 50 度。如果没有缝翼，飞行员将需要使用几乎所有的升降舵权力来抵消襟翼并保持飞机机头向上。

前缘缝翼在二战前就已经出现，但你不会看到它们像这样成对使用。像 Scrappy 这样的飞机永远不会有商业可行性，但像[Mike]这样的人的创新推动了航空业的发展。[迈克]以前的项目飞机， [Draco](https://hackaday.com/2019/09/15/monster-bush-plane-is-a-one-off-engineering-masterpiece/) ，是一架围绕 PZL-104 Wilga 建造的大型涡轮螺旋桨飞机。遗憾的是，它在 2019 年一次考虑不周的起飞中被毁，但[迈克]已经在计划它的继任者， [Draco-X](https://www.youtube.com/watch?v=jX6ZwDcLACA) 。
