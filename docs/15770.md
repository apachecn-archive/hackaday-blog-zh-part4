# 老鼠享受它的自由

> 原文：<https://hackaday.com/2022/12/28/mouse-enjoys-its-freedom/>

虽然花了一点时间来标准化两个带滚轮的按钮的设置，但自 80 年代以来，大多数计算机都使用鼠标或类似鼠标的设备来指向屏幕上的对象。但是除了鼠标的标准“点击”功能，除了舒适的生活之外，很少有突破性的创新。至少，直到【shin saku hi ura】的[“太空蘑菇”鼠标撞上我们的提示线。](https://www.thingiverse.com/thing:5739462)

这种鼠标抛弃了典型鼠标可能具有的大部分功能，支持类似操纵杆的界面，使其具有六个自由度，而不是通常的两个自由度，同时仍然只有鼠标大小，可以握在手中。它甚至没有办法将动作直接映射到屏幕上的移动。相反，它使用这三个物理连接在一起的操纵杆传感器，将每个自由度映射到鼠标本身的类似运动，并通过一些底层编程将每个运动转化为屏幕上的预期运动。

虽然这可能不会很快取代每个用例的标准鼠标，但它似乎在 3D 建模软件、CAD 或任何以定向虚拟对象为主要目标的领域都有巨大的好处。此外，由于可以连接到计算机的鼠标数量没有限制(除了 USB 限制)，这种鼠标可以很容易地与普通鼠标结合使用[，就像宏键盘与传统键盘一起使用](https://hackaday.com/2022/04/07/custom-macro-keyboard-with-sweet-backlighting/)。
