# 遥控飞机非常不稳定

> 原文：<https://hackaday.com/2019/02/06/r-c-whirlygig-is-terrifyingly-unstable/>

在第二次世界大战期间和紧接其后的日子里，航空航天研究是国家安全的首要考虑。当民族国家试图在生存斗争中占上风时，各种各样的疯狂设计都被探索出来。Hiller Hornet 就是在此期间建造的一种这样的飞行器——一种通过安装在顶端的冲压式喷气发动机驱动旋翼的直升机。不出所料，这种配置有很多缺点，妨碍了它达到满负荷生产。然而,[FliteTest]的团队对这种飞行器情有独钟，并以此激发了他们最新的无线电控制实验。

最初的实验包括一个水上飞机模型的改良泡沫机翼，两个左翼面向相反的方向，并在中间连接。机翼上安装了两个马达和推进器来提供旋转运动。在一些最初的振动问题得到解决后，这个临时制作的飞行器产生的升力几乎不足以离开地面。由于涉及高转速，向心力将螺旋桨从机翼上扯下，这也面临着其他问题。

第二次尝试从零开始，使用四翼设置，弯度更高，目的是用更具侵略性的翼型产生更大的升力，允许转速降低。飞行器能够离开地面，但是被比作[摆式火箭谬误](https://en.wikipedia.org/wiki/Pendulum_rocket_fallacy)的不稳定性阻止了高度的任何重大提升。

我们希望看到一个重新设计来解决一些问题，并允许飞行器航行到更高的空中。如果你认为你知道 whirly bird 的动态问题的解决方案，一定要在评论中让我们知道。这应该是可能的，因为我们以前见过受枫树种子启发的成功设计。休息后的视频。
