# DC UPS 保持互联网畅通

> 原文:[https://hack aday . com/2021/11/01/DC-ups-keep-the-internet-up/](https://hackaday.com/2021/11/01/dc-ups-keeps-the-internet-up/)

如今，我们偶尔会对如此多的设备占用了无处不在的“壁瘤”感到恼火。但是有一个优点是这些设备在 DC 电压下工作。[TechRally]利用这一点创造了一种具有双输出的[自动 DC UPS](https://www.instructables.com/DIY-Fully-Automatic-WiFi-Router-UPS-With-Dual-Outp/) ，以便在断电时为路由器和调制解调器供电。你可以在下面看到两个关于这个项目的视频。

有些人可能会说使用传统的 UPS 会更好，但是请仔细想想。那个 UPS 里面有一个可以转换成交流电的电池，所以壁式电源可以把它转换回 DC。当然，每次转换都会损失一些能量，在廉价壁疣的情况下，你甚至会损失相当多的能量。

该项目包含八个 18650 电池，一个现成的充电控制器和电源转换器。你能做一个更有效的定制设计吗？也许是这样，但是使用这些廉价且普遍可用的模块使得将这样的东西组合在一起变得快速而容易。

没有人会把这个 UPS 误认为是一个商业单元，但它确实有一定的黑客美学。不过，我们不会带着它穿过机场。有了这些数字显示器和所有的线路，它看起来就像一个糟糕的电视节目的炸弹道具。

如果你不关心自动切换，我们听说如今 5V 将为许多设备供电，这使得电池操作简单得就像剥掉 USB 电缆一样。这可能会驱动其他一些齿轮，如连接的树莓 Pi。或者，你可以用[一些超级跑车](https://hackaday.com/2020/11/05/a-super-ups-for-the-pi/)来完成这项工作。
