# 一个发光的土豆削皮器可以做一盏能斯特灯

> 原文:[https://hack aday . com/2021/10/02/a-发光土豆削皮器-制作能斯特灯/](https://hackaday.com/2021/10/02/a-glowing-potato-peeler-makes-a-nernst-lamp/)

在过去的几十年里，电气照明发生了巨大的变化，首先是向更紧凑、更高效的荧光灯发展，然后是向 LED 灯泡发展。旧的白炽灯泡虽然发出令人愉快的光，但一点也不节能。挖掘历史书上的白炽灯泡，因为我们知道它不是镇上唯一的游戏；虽然将灯丝悬挂在真空中可以防止其被氧化，但还有另一种在大气压下使用陶瓷元件的灯。能斯特灯要求灯丝加热后才能导电，而[Drop Table Adventures] [已经用陶瓷土豆削皮器](https://www.youtube.com/watch?v=rxo-Rw6ppc0)的刀片做了一个。

选择合适的陶瓷并不是问题，因为陶瓷厨具很容易找到，但有两个问题让实用灯具变得困难。铜连接本身变得过热并氧化，并且在保持均匀加热的同时用喷灯预热陶瓷是困难的。最后，他们确实制造了一盏自给自足的灯，尽管不是最亮的那盏。

如果你认为能斯特灯听起来很熟悉，可能是因为我们在我们的复古系列中报道过它[。](https://hackaday.com/2020/06/16/retrotechtacular-the-nernst-lamp/)
