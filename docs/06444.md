# 给织物染色以制造传感器

> 原文：<https://hackaday.com/2020/05/22/dyeing-fabric-to-create-sensors/>

具有电气功能的织物已经出现好几年了，但很少用于主流服装。布料非常昂贵，而且供应也不可靠。由于对此感到沮丧，[反化学家]开发了 [PolySense](https://hackaday.io/project/168380-polysense) ，这是一种简单的开源技术，可以将任何纤维材料制成导电材料，用于感应压力、拉伸、电容触摸、湿度或温度。

PolySense 使用一种叫做*原位聚合*的工艺，有效地将织物染色，使其成为压电材料。首先将织物浸泡在水和有机化合物吡咯的混合物中，然后加入氯化铁引发反应。发生的聚合过程将织物的单根纤维包裹在导电聚合物链中。

除了均匀地涂覆织物之外，可以使用各种遮蔽技术将图案染色到织物上，用于各种使用情况。休息后的视频展示了一系列这些应用，包括使用聚合手套和绑腿进行动作捕捉，像线性电位计一样的拉链，以及触敏织物。该项目页面列出了欧洲和美国所需化学品的来源，我们期待着看到社区可以提出的其他应用。

这个项目被很好地记录了下来，大量的科学论文覆盖了所有的细节。[柜台化学家]还将在 [2020 虚拟创客大会](https://makerfaire.com/maker/entry/72292/)上展示 PolySense。

这种技术也可以用来制作一架[布艺钢琴](https://hackaday.com/2017/06/17/fabrickeyboard-is-piano-theremin-and-more/)省力很多。在更机械的方面，你也可以在预拉伸的织物上进行 3D 打印，使其弹出 3D 形状。
