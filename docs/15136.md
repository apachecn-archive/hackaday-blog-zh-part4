# 非常规的长板由单片树木制成

> 原文：<https://hackaday.com/2022/10/20/unconventional-longboard-built-from-single-slice-of-tree/>

通常情况下，滑板和长板是由许多层压木材制成的。这使它们具有令人愉悦的柔韧性，从而带来舒适的乘坐体验。然而，这不是唯一的做事方式。[DesignCo]采用了一种非传统的设计，用一大块树木代替。

用一段树干做木板的好处是，它看起来很有吸引力，所有的年轮都清晰可见。要把它做成板材，首先要粗略地切割成形，然后再刨成均匀的厚度。然后使用角磨机上的瓣轮进一步成型。木材在涂上几层桐油后，用哑光漆进行最后的密封。然后准备了一个坚固的钢尾与之匹配，形成一个漂亮的曲线，并拧上了两个螺栓。这些螺栓然后用环氧树脂固定在木板上，连接两者，卡车安装在下面。

最终的构建看起来令人惊叹，也是可骑的。它可能比带防滑带的主板稍微滑一点，而且它可能不会像传统设计那样处理颠簸。然而，长板很少与性能有关，这种板看起来很有趣。
