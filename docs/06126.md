# 无限飞行滑翔机

> 原文：<https://hackaday.com/2020/04/02/infinite-flying-glider/>

如果你在过去几周试图呆在家里的时候已经完成了你的电子项目清单，那么现在可能是时候停下来做些不寻常的事情了。[PeterSripol]向我们展示了一种选择，即[制造几架无人驾驶滑翔机](https://www.youtube.com/watch?v=ojwvAi_VDiE)并试图让它们永远飞行。

Walkalong 滑翔机通过跟随小型滑翔机工作，类似于纸飞机，但由泡沫制成，用一大块纸板制成。纸板会产生上升气流，只要有空间，滑翔机就能一直飞行。[PeterSripol]和他的朋友们尝试了许多其他技术来让这些重约半克的小滑翔机尽可能长时间地停留在空中，包括点燃几十根茶蜡烛来产生上升气流，使用箱式风扇和其他方法。

如果你的项目真的需要一些电力，泡沫滑翔机的构造展示了一个简单的热线切割工具，使用一些附着在一块木头上的镍铬合金线，以及如何组装滑翔机，使它们尽可能轻便。这是一个有趣的项目，肯定会让你分心至少几个小时，甚至更多，如果你有一个稍微大一点的泡沫滑翔机和一些备用的遥控部件的话。
