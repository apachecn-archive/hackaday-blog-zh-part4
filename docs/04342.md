# 馅饼脸游戏被设计成向除你之外的任何人扔馅饼

> 原文:[https://hack aday . com/2019/09/29/pie-face-game-rigged-to-through-any one-but-you/](https://hackaday.com/2019/09/29/pie-face-game-rigged-to-throw-pie-at-anyone-but-you/)

馅饼脸是一款游戏，做的事情和你从标题中所期待的差不多。玩家坐在装有弹簧的手臂前，轮流转动曲柄。最终，一个不幸的玩家会松开手臂，被溅上鲜奶油，让整个团队开心不已。[哈里森]不是一个听天由命的人，然而，[决定操纵游戏](https://www.hackster.io/HarrisonMcIntyre/control-pie-face-wirelessly-with-your-phone-270715)。

除了让装有弹簧的手臂被内部旋转的滚筒释放之外，[哈里森]还有一个更好的主意。将鼓打磨光滑，以去除释放机构使用的齿。然后，释放机制被一个伺服系统取代，由一个装有蓝牙模块的 Arduino Nano 控制。只需在他的智能手机上轻轻一点，[哈里森]就可以触发游戏，保证他的朋友每次都能得到奶油。

这是一个整洁的建筑，并且小心翼翼地避免被微妙的设计发现。如果不是他在 Youtube 上发布了一段视频，哈里森的朋友们可能不会知道。如果你对欺骗的渴望仍未满足，[考虑为卡坦的一轮*定居者掷骰子。*](https://hackaday.com/2017/12/18/how-provably-loaded-dice-lead-to-unprovable-cheating/) 视频中断后。
