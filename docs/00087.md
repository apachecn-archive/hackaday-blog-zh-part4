# 用手语和 Alexa 交谈

> 原文:[https://hack aday . com/2018/07/25/用手语与 alexa 对话/](https://hackaday.com/2018/07/25/talking-to-alexa-with-sign-language/)

正如威廉·吉布森曾经指出的，未来已经在这里，只是没有被平均分配。对于我们这些残疾人来说尤其如此。Abishek Singh 想要做些什么，所以他为听障人士创造了一种使用亚马逊 Alexa 语音服务的方式。他使用 TensorFlow 深度学习网络将美国手语(ASL)转换为语音，并使用语音到文本转换器来解释响应。这一切都在笔记本电脑上运行，所以稍加调整，它应该可以与任何语音界面一起工作。特别是，[Abishek]似乎创建了一个自定义的 ASL 位来触发 Alexa。也许下一步将是使用机械臂[直接在 ASL](https://hackaday.com/2017/08/24/3d-printed-robotic-arms-for-sign-language/) 中创建输出，并完全去掉回声设备？[阿比舍克]还没有发布这个项目的代码，但他已经发布了其他项目的代码，例如 [Peeqo](http://peeqo.com/) ，这个机器人可以用 gif 来响应。
