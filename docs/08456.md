# 从头开始制造车间式起重机

> 原文：<https://hackaday.com/2020/11/26/building-a-workshop-crane-from-scratch/>

买工具固然很好，但是不符合 Youtube 频道【从零开始的工作坊】的风气。建造你需要的东西要容易得多，这个自制的电动车间起重机巧妙地证明了这一点。

使用基本的钢铁加工技术，以简单明了的方式将起重机组装起来。板材和棒材是用钻床、带锯和磨床加工的，尽管我们可以想象如果你愿意，你可以使用手工工具。一个 ATV 绞车被压入服务做沉重的起重，由一组 12V 铅酸电池放置在基地供电。这种设计选择具有双重功能，既是起重机的移动电源，又是底座的配重。

最终的结果看起来清晰的橙色漆面，并且很好地完成了车间内重型设备的移动。腿是可重新配置的，因此即使非常重的负载也可以在背部放置适当的配重来提升。这是对我们去年推出的早期版本[的重大升级，该版本采用液压操作](https://hackaday.com/2019/09/26/a-crane-fit-for-any-workshop/)。休息后的视频。
