# 风滚草涡轮机赢得戴森基金会奖

> 原文：<https://hackaday.com/2018/11/22/tumbleweed-turbine-wins-dyson-foundation-award/>

当风的流动是可预测的时候，风力涡轮机是伟大的。在城市环境中，尤其是在摩天大楼林立的城市中，风的模式可能会非常混乱。那么，你需要的是一个不管风从哪个方向吹都能工作的风力涡轮机。而正是这样一台涡轮机获得了[全球一等奖詹姆斯·戴森奖](https://www.jamesdysonaward.org/)。看看他们休息时的视频。

涡轮机的设计非常简洁。它本质上是一个带有通风口的球体，因此无论风从哪里吹来，它总是朝一个方向旋转(比如说，顺时针方向)。发明者说，他们受到了美国宇航局风滚草项目的启发，该项目开始时是一场头脑风暴会议，然后是在南极洲的 T2 滚动。我们[阅读了这份 PDF](https://ntrs.nasa.gov/archive/nasa/casi.ntrs.nasa.gov/20030068113.pdf) 和[这份总结报告](https://ntrs.nasa.gov/search.jsp?R=20110014767)，但是如果你们中的任何人知道一些关于风滚草的信息，我们会很乐意了解更多信息。

不过，说回涡轮机。效率如何？规模扩大的可能性有多大？3D 打印版本将如何驱动我阳台上的垃圾箱无刷电机？陪审团还没有出来。但是如果很大一部分风来自其他不可用的方向，这件事可能是一个胜利。谁会是第一个 3D 打印的人？
