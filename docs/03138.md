# 纸链蜂足够强壮，可以行走

> 原文：<https://hackaday.com/2019/06/01/paper-strandbeest-is-strong-enough-to-walk/>

大多数读者都熟悉荷兰艺术家 Theo Jansen 的作品，他的 Strandbeest 风力驱动的机械行走雕塑徘徊在荷兰的海滩上。詹森联动装置提供了一种方法，通过旋转输入来制造具有奇怪但有效的行走步态的机器，并被热情地复制到从桌面玩具到自行车的所有东西上。

有人可能会认为詹森链接会超越某些材料，你可能会惊讶地看到一个文件。然后[路易斯·克拉夫特]走上前，与[一个走纸的比目鱼](https://www.youtube.com/watch?v=C8dFgdqU37E)。它在 Blender 中设计，在桌面数控切纸机上切割，由一对连接到 Arduino 的小型机器人驱动，由蓝牙链接控制，它有四组腿，可以推动桌面上的物品。我们不会认为这是可能的，但它确实存在。

他声称这是一个折纸 Strandbeest，但我们不太确定。在 Hackaday，我们不是纸艺专家，但当我们戴上学究气的帽子时，我们坚持认为折纸必须是日本风格的折叠纸，而不是这里使用的剪切和粘合。正如你在下面的视频中看到的，这并没有降低作品的质量。

我们认为这是我们看到的第一只纸树熊，但是这些年来我们已经给你带来了无数其他的。下面是[【詹森】对这个想法](https://hackaday.com/2018/08/16/theo-jansen-invents-a-faster-simpler-wind-powered-strandbeest/)的最新、波浪式的看法。
