# 用树脂铸造制造相似的瓣膜

> 原文:[https://hack aday . com/2019/11/21/creating-looklike-valves-with-resin-casting/](https://hackaday.com/2019/11/21/creating-lookalike-valves-with-resin-casting/)

电子管在电子领域有一批铁杆粉丝，因其温暖的声音、理想的失真特性和吸引人的美感而受到称赞。然而，有时你只是想要一个道具或玩具的阀门的外观，而实际上并不需要它的功能。对于这些情况，[来自[Ajaxjones]的这个项目可能就是入场券。](https://www.instructables.com/id/Resin-Cast-LED-Vacuum-Valve/)

该构建包括采用现有的瓣膜，将其与 3D 打印底座结合，并使用它来创建硅胶模具。然后，3D 打印零件和裁缝的大头针被用来制作阀门的内部零件，并被插入模具中。然后将透明树脂脱气，倒入模具中制成零件。一旦固化，零件被移除，基底被涂漆以完成外观。然后在底座的空隙中安装一个 LED，让它发出你所期望的温暖的光。

这是一个生产高质量透明塑料部件的简单教程，应该对许多道具制作者和角色扮演者都有用。如果你想让你的树脂游戏更上一层楼，考虑尝试一些包覆成型的部件。休息后的视频。
