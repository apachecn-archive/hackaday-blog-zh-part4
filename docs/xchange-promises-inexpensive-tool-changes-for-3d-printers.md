# XChange 承诺为 3D 打印机提供廉价的工具更换

> 原文：<https://hackaday.com/2020/12/24/xchange-promises-inexpensive-tool-changes-for-3d-printers/>

[教学技术]有兴趣为他的 3D 打印机添加一个工具更换器。E3D 提供了一个系统，允许你切换打印头，甚至为激光或(可能)轻型数控头更换热端。然而，整个设备的价格约为 2500 美元，这让他望而却步。但现在他对 PrinterMods 的一款名为 XChange 的产品感到兴奋。这是一个工具包，将允许在许多现有的打印机上快速更换工具，成本相当少。KickStarter 上的预购价格约为 150 美元，但这可能不是最终价格。

并非所有打印机都兼容。看起来该装置连接到具有线性轨道的打印机，并且有一个适配器用于具有 V 形辊的打印机。据推测，有一种适配器正在为使用杆和轴承的打印机工作。

看起来安装不会很困难，一旦你把主要部分连接上，工具就连接到一个板上，简单地扣进扣出。有一个坚固的机械附件，也有一个电气附件。

这可能会导致打印头位置发生变化，从而影响您的打印区域、中心位置，并且通常需要更改您的生产 g 代码或固件。尽管如此，您可能认为像这样的任何东西都需要一点小小的改动才能工作。显然，在这些设备真正上市时，将会有一些为流行的机器预建的固件。

切换热点当然很有吸引力。但真正的价值似乎是增加了其他东西。激光和笔式绘图仪似乎是个好主意。不过，我们不确定一台普通的打印机是否足够坚硬，能够用机械数控系统做任何严肃的事情。该视频还提到了未来拥有取放系统的可能性。

这似乎是一个好主意，我们已经看到[快换车厢设计有一段时间了](https://hackaday.com/2020/07/18/tool-changing-3d-printers-shouldnt-break-the-bank/)。真正的问题是，每次更换工具时，系统是否需要重新校准。如果是这样，任何像这样的系统的价值都是有限的。我们还想知道工具板上是否应该有一个串行 EEPROM 选项，以便固件可以询问工具头的身份，甚至将配置数据存储在板本身上。也许那会是第二版。我们自己的【约书亚·瓦斯奎兹】在去年年底发布了他的工具更换平台 [Jubilee](https://hackaday.com/2019/11/14/jubilee-a-toolchanging-homage-to-3d-printer-hackers-everywhere/) ，这也值得一试。
