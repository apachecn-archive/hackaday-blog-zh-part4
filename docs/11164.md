# 1949 年陀螺仪再次旋转

> 原文:[https://hackaday.com/2021/08/29/__trashed-23/](https://hackaday.com/2021/08/29/__trashed-23/)

[好奇的马克]有一个阿波罗时代的陀螺仪，但没有一些实践，还没有准备好让它通过这一步。因此，他借了一个 1949 年的老式 [Sperry C5 陀螺仪](https://www.youtube.com/watch?v=pz-LA71tMWw)并用它做了一些实验，他计划在另一个陀螺仪上使用三相电源。

有一点故障排除和这些机电奇迹的许多华丽的特写镜头。不过，它们确实很吵。

[Marc]想要一个能够控制待测陀螺仪方向的陀螺仪测试台。他走拍卖路线，以相对较低的价格获得了一件相当昂贵的设备，但没有昂贵的软件。幸运的是，他设法从对他的项目感兴趣的供应商那里获得了所需的软件。在我们看来，像这样的一张桌子从零开始也不会太难。

我们对[马克]下一步会用他的陀螺仪做什么感兴趣。很难想象陀螺仪已经从这种设备发展成为可以安装在手机上的微型集成电路惯性测量单元。想象一下，在你的下一个步行机器人或自平衡独轮车上安装斯佩里装置。

需要复习陀螺仪的工作原理吗？我们也得到了那个。它甚至包括现代的那种。
