# 科学解释的抗重力化妆品！

> 原文：<https://hackaday.com/2020/02/15/gravity-defying-cosmetics-explained-by-science/>

在每个女性技术记者的职业生涯中，可能都会有这样一个时刻，她会报道自己的第一个化妆故事，并大声质疑自己的生活是否会变成这样。但这个化妆故事涉及到一些物理学，并且跟随了抖音社区的一系列病毒视频，在这些视频中，专业化妆品博主惊讶地看到唇彩明显从涂抹器的末端漂浮起来——无视重力。这引起了【史蒂夫·莫尔德】[的注意，他继续他的直觉，静电可能是原因](https://www.youtube.com/watch?v=_4VgQo-bAnY)。接下来的视频是对这一现象的各种重现和描述。

在英国一月潮湿的空气中，在地毯上摩擦脚这种屡试不爽的方法没有引起任何运动，他开始尝试范德格拉夫发电机，即使它产生的大量静电电荷也没有产生超过一个微小的信号，但至少给出了一个可能是静电效应的建议。

最后，他用 FunFlyStick 静电玩具复制了美妆博客的结果，令人满意的唇彩喷向空中。FunFlyStick 本身就是一个有趣的设备，是一个玩具形式的范德格拉夫发电机，能够产生大量电荷。飞行唇彩是一个有趣的现象，但它进一步说明了在干燥的气候下，普通物体上会积聚多少静电荷。那些生活在潮湿地区的人在旅行前最好做好记录。
