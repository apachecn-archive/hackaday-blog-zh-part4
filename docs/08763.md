# 为您的假期放松:点击春天日晷建立大切

> 原文：<https://hackaday.com/2020/12/23/for-your-holiday-relaxation-the-clickspring-sundial-build-megacut/>

我们当中的幸运儿很可能会有一段时间不工作，虽然大部分时间可能会被家庭义务和庆祝活动所占据，但可能会有一些停工时间。如果你碰巧有半个小时的空闲时间，你可能会想去看看 [the Clickspring 拜占庭日历-日晷 mega edit](https://www.youtube.com/watch?v=Uc-hUvbqqh8) 。我们很乐意提前接受你的感谢。

机械加工视频的粉丝无疑已经熟悉了业余钟表师 Clickspring，又名[Chris]，他通过结合惊人的工艺和顶级的生产价值，成功地使钟表制造成为一项观赏性运动。我们首先通过[他的开放框架时钟制作](https://hackaday.com/2016/12/06/for-your-binge-watching-pleasure-the-clickspring-clock-is-finally-complete/)抓住了 Clickspring bug，它最终成为了一件合法的艺术品。【Chris】然后立刻进行了两次制作:[一个著名的 Antikythera 机械装置](https://hackaday.com/2015/11/23/the-antikythera-mechanism/)的复制品，以及下面视频中看到的日历日晷。

这部影片浓缩了 1000 个小时的机械加工、车削、铸造、热处理，甚至手工雕刻黄铜和钢，制作成一部令人难以置信的放松视频。没有叙述，没有解释——除了金属被分成几十个部分，最终完美地组合成一件配得上拜占庭王子的乐器的声音，什么都没有。这个视频确实激起了我们对更多安蒂基希拉建造细节的兴趣，但我们知道[【克里斯】最近很忙](https://hackaday.com/2020/12/12/hackers-discovery-changes-understanding-of-the-antikythera-mechanism/)，所以我们会耐心等待。
