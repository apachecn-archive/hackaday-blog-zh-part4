# 塞尔达 II Redux ROM Hack 播放你如何记得原来的

> 原文:[https://hackaday . com/2020/12/01/zelda-ii-redux-rom-hack-plays-how-you-remember-the-original/](https://hackaday.com/2020/12/01/zelda-ii-redux-rom-hack-plays-how-you-remember-the-original/)

回归经典游戏可能是一段艰难的经历。时间的流逝让技术停滞不前，而那些旧游戏的记忆可以以神秘的方式变形。这就存在一个问题，即你如何记住一个游戏的玩法，以及它实际上是如何玩的。开发者[Jorge]看到了围绕《塞尔达 II:林克的冒险》出现的情况，这激发了他创作《塞尔达 II》的灵感。

经过多年的制作，塞尔达 II Redux 采取了相对宽松的方式来修改最初的 NES 游戏。图形增强包括:一个重新制作的 HUD，完整的系列传统的心，动画敌人图标在世界各地，一个新的标题屏幕，并给予链接盾从[Famicom 磁盘系统发布的盒艺术](https://3.bp.blogspot.com/-cwU5J9CFh6s/T5RjJna0pMI/AAAAAAAAB34/vTON2WFEiBU/s1600/Zelda_2_FDS_A.jpg)。文本速度有所提高，并纳入了日文脚本的修订翻译。在引擎盖下，各种各样的 boss 战斗被重新平衡，而施法不需要多次返回暂停菜单。虽然塞尔达 II Redux 最重要的功能可能是通过暂停菜单上的“Up + A”手动保存。为了让林克的第二次冒险与《塞尔达传说》系列的其他部分更加一致，塞尔达 II Redux 还加入了一系列其他的变化，这些变化[可以在项目的变更日志](https://www.romhacking.net/hacks/5440/)中找到。

玩《塞尔达 II Redux》需要一个 IPS 补丁程序，比如 lunar IPS(T1)，以及一个干净的《塞尔达 II:林克历险记》的转储图像。转储 NES 墨盒比以往任何时候都容易，由于许多墨盒转储设备是即插即用的 USB 接口。成功修补的 ROM 文件可以通过 flash cart 在模拟器或实际的 NES 硬件上播放。下面包含了一个工具辅助速度跑的视频，所以可能会有一些新的策略可以采用。
