# 瞬间就生锈了

> 原文：<https://hackaday.com/2022/07/10/grok-rust-in-a-flash/>

在 Hackaday，我们大力提倡使用最好的工具来完成工作(或者在需要时自己制作工具)。但是当你只知道如何使用 Java 时，一切看起来都是面向对象的。除了糟糕的笑话，重要的是要有许多工具供你使用，让你做出明智的选择。为什么不花几分钟时间与[【无样板】一起了解 Rust](https://www.youtube.com/watch?v=br3GIIQeefY) 的基础知识呢？

视频的重点是尽可能多的去过 Rust，教你怎么读。这个想法不是从基本概念开始，而是[没有样板文件]会检查你在基于 Rust 的程序中看到的绝大多数内容。无论你是来自面向对象的、函数式的，还是普通的基于 C 的背景；你会感觉很舒服，因为他会努力去比较你已经知道的东西。还介绍了 Rust 的一些更独特的特性，比如可变性、范围、匹配和字符串。然而，为了保持视频简短，生命期、闭包和特征都被省略了。这些话题在[一篇由【比石灰更快】撰写的优秀博文](https://fasterthanli.me/articles/a-half-hour-to-learn-rust)中有所涉及，该视频正是基于这篇博文。

没有讨论的是[在 PIC32](https://hackaday.com/2022/01/09/blink-an-led-on-a-pic32-with-rust-easily/) 这样的无 std 环境下运行 Rust。Rust 在过去的几年里经历了令人兴奋的发展，T2 的 Linux 内核变得越来越生锈，编译器也变得越来越好。休息后的视频。
