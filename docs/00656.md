# 相机使用算法而不是镜头

> 原文：<https://hackaday.com/2018/09/15/camera-uses-algorithms-instead-of-lenses/>

普通相机使用镜头来弯曲光线，使其照射到传感器上。针孔相机没有镜头，但这个小孔也有同样的功能。现在犹他大学的两名研究人员。已经使用软件[从散射的未聚焦光](https://www.osapublishing.org/oe/abstract.cfm?uri=oe-26-18-22826)中重建图像。质量不是很好，但没有镜头，甚至没有针孔。你可以看一个视频，在下面。

这种相机在一块透明窗户的边缘有一个传感器。这些图像可以在 150 毫米的距离上分辨率为 0.1 线对/毫米，景深约为 10 毫米。这似乎是一个需要解决的问题，但想想在没有传统摄像机的情况下，摄像机可以透过挡风玻璃或一副眼镜进行观察的应用。

窗户唯一特别的地方是，除了带传感器的那条边，所有的边都贴上了反光带，其余的边都是粗糙的。所用的窗户是一块有机玻璃，三个边缘经过热退火，第四个边缘用砂纸打磨。第四个边缘上的传感器拾取一组看似随机的点。

这不是唯一的无镜头相机。加州理工学院、莱斯大学、日立、微软和其他公司都有自己的想法，尽管它们通常更引人注目。本文讨论了其中的一些，虽然它们不需要镜头也能工作，但它们通常需要的不仅仅是你和场景之间的一片透明材料。

这些照片并不精彩，但同样，第一台数码相机拍摄的照片也不精彩。对于许多计算机视觉应用来说，它们仍然是足够的。如果你宁愿[玩针孔](https://hackaday.com/2012/10/25/a-beautiful-pinhole-camera-takes-wonderful-photos/)，它们的效果可能比你想象的要好。
