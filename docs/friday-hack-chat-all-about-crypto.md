# 周五黑客聊天:所有关于加密

> 原文：<https://hackaday.com/2018/11/14/friday-hack-chat-all-about-crypto/>

什么是加密？ [Crypto 的意思是“隐藏的”](https://hackaday.com/2018/04/23/what-does-crypto-actually-mean/)，它的意思是“隐藏的”，早在希腊字母被书写之前就有了，但不要让这阻止你争论。[在本周的黑客聊天](https://hackaday.io/event/162249-cryptography-hack-chat)中，我们将谈论所有关于密码学的话题，一种交换秘密的媒介。如果你需要机密性、完整性或真实性，你需要密码学。

我们本周黑客聊天的嘉宾不是别人，正是尼克·塞耶。尼克是黑客日聚会的常客，多年来他一直在制作小玩意和小发明，并在 Tindie 上出售。他做了[关于制造设计的演讲](https://hackaday.com/2015/12/24/nick-sayer-making-isnt-the-same-as-building-one/)。今年，他设计并开发了 orth RUS T8，这是一种从两张 microSD 卡创建加密保护的 USB 卷的设备。基本上，它就像行星队长团队，只是你需要所有的 SD 卡，而不是行星队长，你召唤你的数据。

在本周的 Hack Chat 中，我们将坐下来讨论所有关于密码学的事情，包括了解你需要什么，不需要什么，以及选择正确的工具。感兴趣的项目包括:

*   当需要加密时
*   加密工具
*   加密的最佳实践

当然，我们鼓励你在讨论中提出自己的问题。你可以在[密码术黑客聊天](https://hackaday.io/event/162249-cryptography-hack-chat)上留下评论，我们会把它放在黑客聊天讨论的队列中。
