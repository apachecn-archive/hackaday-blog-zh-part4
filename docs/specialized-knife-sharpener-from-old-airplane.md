# 旧飞机专用磨刀器

> 原文：<https://hackaday.com/2018/08/01/specialized-knife-sharpener-from-old-airplane/>

“磨刀肯定不会那么难”有人可能会想，因为他们在不确定颗粒的磨刀石上疯狂地挥舞着小刀，破坏了它的刀刃。事实上，磨刀和其他领域一样是一项细致入微的实践，获得高质量的抛光比看起来要困难得多。不同的刀片也会变得越来越复杂，正如【涡轮征服巨鹰】展示的定制磨刀夹具。

任何刃磨中最难的部分是获得合适的斜角。较大的角度适用于重型工具，如斧头，但对于精细的工作，如剃须，则需要更尖的角度。通常，需要一个安装在桌子上的夹具，但是由于生产的限制，使用了一个手持夹具。它由飞机发动机的推杆和凸轮从动件制成(零件很多，因为这种特殊的发动机总是会坏)，可以在叶片上施加非常特定的斜角。例如，弯刀在手柄附近有一个较大的角度，但在尖端有一个较细的点，这个工具有助于快速磨刀。

如果你想尝试另一个不像看起来那么简单的项目，你可能想在尝试使用磨刀工具之前[从头开始制作一把刀](https://hackaday.com/2018/06/07/building-a-knife-by-hand-is-just-as-hard-as-you-think/)。这也是一个微妙的过程，但是只要稍加练习，只需几个工具就可以完成。
