# 我去了月球，得到的只有这件恶心的 t 恤

> 原文：<https://hackaday.com/2019/06/03/i-went-to-the-moon-and-all-i-got-was-this-lousy-t-shirt/>

已经过了很长时间，但[弗兰]终于有了一个 DSKY 显示器，一个在阿波罗导航计算机中发现的用户界面显示器的复制品。最精彩的部分？这是一件 t 恤。

这个构建是一个很长很长的时间，从 2015 年弗兰开始研究阿波罗制导计算机的 DSKY 开始。当时，有复制品，但老实说，它们都很糟糕。复制品使用现成的七段发光二极管或光管。真正的 DSKY 是一件艺术品，在当时可能是有史以来最复杂的电致发光显示器。这让[弗兰]有了一次非常特别的旅行，她被允许参观航空航天博物馆的附属建筑，在那里她参观了一个真正的 DSKY 展览。她得到了所有的尺寸，通过一些非破坏性的调查，她能够拼凑出这个非常特殊的展示是如何组装起来的。

有了这些信息，[弗兰]就能弄清楚这个展示是一个相当复杂的丝网系列。如果是丝网印刷，你可以把它印在 t 恤上，所以[弗兰]就是这么做的。这使用了 DIY 丝网夹具和磷光油墨。这不是场致发光显示器，但是它*在黑暗中确实*发光。

虽然这件 DSKY t 恤在黑暗中会发光，但这意味着它不像最初的 DSKY 那样是电致发光显示器。也就是说，在 t 恤上丝网印刷电致发光显示器也不是闻所未闻。几年前，一家丝网印刷公司[对可穿戴设备](https://hackaday.com/2016/01/21/screen-printing-electroluminescent-t-shirts-from-tron/)上的 EL 显示屏做了一些实验。当然，如果你想要一个真正的电致发光 DSKY 显示器，【本·克拉斯诺】有一个[非常现代的丝网印刷显示器](https://hackaday.com/2019/05/19/ben-krasnow-makes-a-dsky/)。[本]项目中的电子设备与飞往月球的东西毫无相似之处；最初的 DSKY 有*继电器*。也就是说，我们从未如此接近阿波罗制导计算机显示器的现代再现，我们有[弗兰]和[本]的工作为我们指明前进的方向。
