# 你闻到神奇厨师在做什么了吗？

> 原文:[https://hackaday . com/2020/03/19/do-you-smell-what-the-magic-chef-is-cookin/](https://hackaday.com/2020/03/19/do-you-smell-what-the-magic-chef-is-cookin/)

自动机已经很酷了，但是能够愚弄我们的是非同寻常的东西。传奇人物[格雷格·朱姆沃尔特]最近将他的玩具制造注意力转向了虚幻的自动机，我们认为他以他令人钦佩的开胃菜[神奇厨师](https://hackaday.io/project/170352-the-magic-chef-a-3d-printed-automata)开了一个好头。

厨师的目的是取悦顾客，只要他有权力这样做，他就会继续提供他的六项菜单中的菜肴，包括汉堡、热狗、披萨片、BLT、单面煎的鸡蛋和我们不太确定的带状凝胶块。令人惊讶的是，这个单人餐厅用一个 6VDC 齿轮电机、一些磁铁和 58 个打印零件做所有事情，包括齿轮、凸轮和杠杆。食物转盘在一种磁性滑环系统上移动的方式是蛋糕上的糖衣。

如果你想做一个你自己的神奇厨师，[所有的 STL 文件都可以从 Instructables 页面](http://www.instructables.com/id/The-Magic-Chef-a-3D-Printed-Automata/)获取。想知道更多细节吗？休息后去洗漱一下，因为我们会提供一个演示视频，里面有一些内部工作的特写镜头。哦，还有[这里有一些自动冲泡的咖啡](https://hackaday.com/2016/09/24/cute-but-serious-faced-automata-produce-a-pour-over/)作为甜点。
