# 周五黑客聊天:制造模块化硬件

> 原文:[https://hack aday . com/2018/07/25/Friday-hack-chat-making-modular-hardware/](https://hackaday.com/2018/07/25/friday-hack-chat-making-modular-hardware/)

无线的未来是分散的。网状网络正在慢慢成为 WiFi 标准，很快我们就能处理分散的电话。那是无线的，但是电子的呢？对于大多数嵌入式工作，我们都在处理主设备和从设备，但是如果我们不需要处理这些呢？这是模块化电子产品的挑战，本周的黑客聊天将会谈论这一切。

[![](../Images/bd75a1614bdba57c86a40cbae73170f5.png)](https://hackaday.com/wp-content/uploads/2018/07/makingmodularhardware_2-01.png) 我们本周黑客聊天的嘉宾是来自西雅图的电子工程师【Asaad Kaadan】。[Asaad]拥有俄克拉荷马大学电气工程硕士和博士学位。他的日常工作是为 Freefly 系统制造高端摄像机控制器。晚上，他设计 Hexabitz 电子原型模块。什么是 Hexabitz？这就是事情变得有趣的地方。

如你所料，Hexabitz 是一个小小的六边形，里面装满了电子元件。每个六边形板上都有一个微控制器，这些六边形通过电路板边缘的焊盘连接器连接在一起。在你问之前，是的，有五边形六边形，所以是的，你可以做 [*那个*](https://en.wikipedia.org/wiki/Ball_(association_football)) 。

在这次黑客聊天中，我们将谈论所有关于模块化电子设备和[阿萨德]的六位。我们将涉及以下问题:

*   如何设计测试板的连接器
*   网状电子设备的协议是什么样的
*   如何在系统中一起使用模块化电子设备
