# 电影魔术聊天

> 原文:[https://hackaday.com/2021/01/18/movie-magic-hack-chat/](https://hackaday.com/2021/01/18/movie-magic-hack-chat/)

1 月 20 日星期三中午，太平洋时间，加入我们与[艾伦·麦克法兰](https://hackaday.io/hacker/143)的[电影魔术聊天](https://hackaday.io/event/175899-movie-magic-hack-chat)！

如果它们被神奇地传送到过去，过去的电影观众可能不知道如何看待我们现代的 CGI 驱动的史诗，在从未存在的风景中表演物理上不可能的壮举。但是，尽管电影的计算已经变得如此复杂，但很少有电影仍然不需要至少一些老式的电影魔法，如手道具、物理模型和其他实际效果。

为了让他们的设想成真，尤其是在科幻电影中，电影制作人转向专门从事实际效果的艺术家。我们都看过他们的工作，在许多情况下，他们的工作包括将普通的家用物品转化为尚待发明的技术，或者创建飞船和外星景观的比例模型。但是要真正销售这些效果，增加一点电子设备真的会有所不同。

艾伦·麦克法兰是电影业的电子设计师和工程师。凭借电影摄影、电子和嵌入式系统方面的背景，他已经能够在我们都看过的电影中制作效果。他为 *Tron: Legacy* 设计了电致发光可穿戴设备，为*第五元素*中的迷你失乐园建造了照明系统，并为*星际迷航:第一次接触*设计了博格人的服装。他在让想象看起来真实方面有着丰富的经验，他将加入我们的 Hack Chat，讨论他在制作电影魔术的实用效果工具包中保留的技巧。
