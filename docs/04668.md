# 种植 2000 万棵树，使用无人机、大炮和更多非常规方式

> 原文：<https://hackaday.com/2019/10/29/planting-20-million-trees-using-drones-cannons-and-more-unconventional-ways/>

当 YouTuber MrBeast 达到 2000 万用户时，它开始承诺到 2020 年种植 2000 万棵树。虽然对一个人来说尝试这样的壮举似乎有点疯狂，但该频道已经开始在“[团队树](https://teamtrees.org/)”的旗帜下进行有组织的努力。随着许多知名和不太知名的 YouTubers 和其他在线人物以及许多公司和组织的加入，这个想法似乎不像最初看起来那么遥不可及。

我们在休息后嵌入了 MrBeast 的视频，在那里你还可以找到 Mark Rober 的视频，他组建了一家名为 [DroneSeed](https://www.droneseed.com/) 的公司，他们使用大型无人机在大面积范围内分发营养舱中的种子。他们的重点是在大的野火和其他事件后的重新造林，这些事件使得土地上没有树木。当然，这是种子，需要一段时间才能看到结果。

Linus Media Group 的不耐烦的加拿大人[认为，他们宁愿以极快的速度种植树苗，拼凑一个氮炮，向土壤中发射营养颗粒，为幼苗创造洞，或者交替地从缺口装载炮向土壤中一次发射颗粒和幼苗。显然，后一种方法的结果更有问题，从他们计划添加到当地自然环境中的大约 300 棵幼苗中拿走了一大块。](https://www.youtube.com/watch?v=6l8K1eH624I&t=0s)

无论选择哪种方法，任何在世界范围内的大规模植树造林都可能是减少全球大气二氧化碳增加和由此带来的气候挑战的关键部分。^(。有消息称，今天世界上的树木总数约为 3 万亿棵，2000 多万棵似乎不算多，但是我们今天学习的加速重新造林的技术可能会在不久的将来发挥重要作用。)
