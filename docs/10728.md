# 延长线上的廉价偶极天线

> 原文：<https://hackaday.com/2021/07/16/a-cheap-dipole-antenna-from-an-extension-cord/>

偶极天线是典型的建筑天线，毕竟它们通常只不过是两根电线和一根馈线。但是正如[Rob]在下面的视频中向我们展示的，[有一些事情需要考虑](https://www.youtube.com/watch?v=2ZryRltHBjI)。

第一件事是从哪里得到电线。一根损坏的延长线捐赠了电线。这实际上是一个有趣的想法，因为延长线中有多根长度相同的电线。当然，把导线从延长线上拔出来很容易，但是怎么给它供电呢？一个小型巴伦将不平衡馈线转换为天线的平衡连接。尽管标题上写着“自由偶极”,但这种巴伦是商用的，除非你碰巧已经有了，否则可能要花些钱。然而，如果你碰巧有一个铁氧体磁环，构建一个巴伦并不那么复杂。

如果你想传输，你可能需要一点不同的安排，但接收这个肯定会完成工作。调谐器会让生活变得更容易。

尽管从技术上讲这是一个偶极子，但如果没有调谐，它更像是一根随机的线。然而，它的工作原理和天线分析仪现在常见的齿轮，这将很容易缩短偶极下降到任何波段你想要的。

当然，在机械世界里，他们说[如果你不能让它精确，那就让它可调](https://hackaday.com/2021/02/23/capstan-winch-central-to-this-all-band-adjustable-dipole-antenna/)。如果你在寻找便宜的车站装备，总有[、50 美元的火腿](https://hackaday.com/2021/01/19/the-50-ham-a-cheap-antenna-for-the-hf-bands/)，或者被称为【丹·马洛尼】。
