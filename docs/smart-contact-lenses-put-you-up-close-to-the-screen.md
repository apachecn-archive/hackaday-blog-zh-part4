# 智能隐形眼镜让你靠近屏幕

> 原文：<https://hackaday.com/2022/05/22/smart-contact-lenses-put-you-up-close-to-the-screen/>

谷歌眼镜并没有像预期的那样起飞，但是——老实说——你真的想头上戴着那个硬件到处走吗？英国广播公司最近报道了 Mojo T1，这是一家开发 T2 智能隐形眼镜 T3 的公司，这种眼镜不仅能矫正视力，还能显示画面。你可以在下面的技术上看到来自 CNET 的视频。

这种镜片有微型 LED 显示屏、智能传感器和固态电池，类似于心脏起搏器中的电池。根据 BBC 的文章，该公司声称有一个“功能完整的原型”，并准备开始测试。我们想象你不可能在隐形眼镜里塞进太多的电池，但据推测，这是开发这种技术如此困难的原因之一。

这篇文章还提到了其他正在开发的智能隐形眼镜，包括萨里大学的一种镜头，它可以使用集成在镜头中的各种传感器来监控眼睛的健康。你会想这在现实生活中会是怎样的。假设显示屏关闭，你什么也看不到，但你的手机不停地发出嘟嘟声，而你的视野中却始终收不到信息，这已经够烦人的了。

当然，这似乎是一项即将到来的技术。如果不是这次，那么在将来的某个时候。虽然我们通常认为黑客社区应该起带头作用，但我们并不确定我们想黑一些触动人们眼球的东西。不是每个人都可以这么说。对我们来说，我们将坚持使用[耳机](https://hackaday.com/2021/09/02/a-microcontroller-friendly-ar-headset-on-the-cheap/)。
