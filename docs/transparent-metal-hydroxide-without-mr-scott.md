# 没有斯科特先生的透明金属(氢氧化物)

> 原文：<https://hackaday.com/2022/12/07/transparent-metal-hydroxide-without-mr-scott/>

在《星际迷航》电影中有一个著名的场景，回到过去的斯科特教一家金属公司制造透明铝，他需要这种铝来将一些鲸鱼带回未来。但是[行动实验室]显示我们已经有了透明金属，只是没有铝。你可以看一个关于为什么金属通常不透明的视频。

正在讨论的金属是钠。正常情况下，它是不透明的，但是熔化的氢氧化钠在它之后会变透明——嗯，有点像爆炸。当然，氢氧化钠不是真正的金属，但被吹捧为真正透明铝的氧化铝也不是。氧化铝也能制造像红宝石一样的透明宝石。然而，在视频的结尾有某种透明的铝。

然而，塑料衬底上的薄铝膜可能装不下一头鲸鱼。它也不是完全透明的。这种材料看起来像一面镜子，但一面的激光会在另一面产生部分外观。

然而，这里有趣的是对为什么金属通常不透明的解释。嗯，还有爆炸的钠。
