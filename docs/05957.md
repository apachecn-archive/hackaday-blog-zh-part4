# 重新设计的紫外线固化器使你的指纹像指甲一样坚硬

> 原文：<https://hackaday.com/2020/03/15/repurposed-uv-curer-makes-your-prints-hard-as-nails/>

树脂打印机的价格在过去几年里大幅下降，现在你可以花不到 500 美元买到一台相当不错的 DLP 打印机。虽然这是一个好消息，但要成功打印，除了树脂之外，你还需要一些东西，比如一种进行后处理紫外线固化的方法。

不久前，我买了一辆价格高得吓人的万豪 D7。他没有在 UV 固化箱上花费另一个打印机的纸张价值，而是拯救并重新利用了一个小型商业固化设备，该设备用于凝胶指甲油。你把指尖伸进去，打开开关，它运行 60 秒，然后关闭。

这是个好主意，但不幸的是指纹没有手指甲固化得快。因此，第一项任务是通过将 UV LEDs 直接连接到电源来绕过基于 dual 555 的计时系统。制造商从来没有打算让灯连续运行，所以为了防止电路板熔化，[抑制]增加了一个 12 V 的小型计算机风扇来冷却。甚至还有一个带角鳍的小印刷格栅，以防止紫外线泄漏并灼伤附近的视网膜。

[Inhibit]还设计并打印了一个供印刷品放置的托盘，以及一个前围板，以尽可能多地将光线集中在零件上。两个部分的文件都漂浮在这个世界上，休息之后，我们已经得到了所有~~治愈的~~制作视频。

这些小的商用盒子并不花费太多，但是你总是可以建造你自己的。
