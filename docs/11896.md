# 单伺服机械七段显示器

> 原文:[https://hack aday . com/2021/11/13/a-one-servo-mechanical-seven-segment-display/](https://hackaday.com/2021/11/13/a-one-servo-mechanical-seven-segment-display/)

经过这么多年，七段展示可能有点平淡无奇，但这并不意味着没有办法增加它的趣味。提出典型的光子显示器的机械版本是一个受欢迎的项目，多年来我们已经看到了很多这样的例子。但是[这种七段显示](https://www.thingiverse.com/thing:5066794)是一种相当机械的享受，也是翻阅数字的一种独特方式。

对于大多数机械显示器，我们习惯于看到每个片段的状态随着某种致动器而改变，如螺线管或伺服机构。[Shinsaku Hiura]决定采用更时尚的设计，使用 3D 打印的枪管，每个部分都带有一个凸轮。每个铰接部分都连接到一个作为从动件的臂上，骑在凸轮上，以设定的模式打开或关闭。显示哪个数字取决于枪管的位置，枪管由单个伺服机构和一对齿轮控制。它用机械的复杂性换取了电气的简单性和整体的优雅，正如你从下面的视频中看到的，它非常漂亮。

我们认为这个构建最好的部分是计算出凸轮的形状。我们想知道它们与[【格雷格·朱姆沃尔特】的机械展示](https://hackaday.com/2021/06/27/mechanical-7-segment-display-uses-a-single-motor/)中的凸轮轮廓相比如何；它使用两个带凹槽的独立圆盘，但原理基本相同。
