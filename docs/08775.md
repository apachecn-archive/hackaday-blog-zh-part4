# 这是世界上最小的任天堂 64 吗？

> 原文:[https://hackaday . com/2020/12/25/is-this-the-world-small-Nintendo-64/](https://hackaday.com/2020/12/25/is-this-the-worlds-smallest-nintendo-64/)

游戏机爱好者的一个利基活动是将全尺寸游戏机缩小到更小的格式，将曾经只能在家庭电视上播放的内容变成适合口袋的便携式格式。一个特别令人印象深刻的例子是，[GmanModz]制造了他声称是世界上最小的便携式任天堂 64。特别值得注意的是，他没有使用仿真器或定制的 PCB，而是使用了一个经过大量修整的真正的任天堂 64 主板。

下面的视频详细介绍了这些模块的最新技术，并显示了他如何避开最新的技术，而是限制自己只使用商用分线 PCB 和现成的模块。N64 电路板大幅削减，需要对各种 PCB 进行大量细磁线焊接，以替换被移除的器件。卡匣槽以直角伸出到板的背面，从 3D 打印外壳的背面伸出，在用于 18650 单元的空间上方，并允许玩原始游戏卡匣。有一个微控制器，以促进对任天堂控制垫较少使用的一些妥协，但结果是一个完全可玩的迷你手持控制台。他承认“*电池寿命很糟糕，拿着【】* *很不舒服，但是，嘿——它很适合放在我的口袋里。你的 N64 呢？”在那一点上，我们不能责备他。*

这不是我们见过的第一台便携式 N64，但它会长期保持最小的称号吗？只有时间能证明一切。
