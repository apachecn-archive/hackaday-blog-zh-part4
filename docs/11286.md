# THICC GBA SP Mod 在发布前易于安装

> 原文:[https://hack aday . com/2021/09/11/thicc-GBA-sp-mod-gets-easy-install-ahead-of-release/](https://hackaday.com/2021/09/11/thicc-gba-sp-mod-gets-easy-install-ahead-of-release/)

早在 8 月份，我们报道了任天堂 Game Boy Advance SP 的一项独特修改，该修改用一个扩展版本取代了手持设备的后面板，该版本具有足够的内部容量来升级电池、蓝牙音频发射器，并支持 Qi 无线和 USB-C 充电。不利的一面是，安装 10 毫米 3D 打印的“背包”并不完全是最用户友好的操作。

但今天我们很高兴地报告，所谓的 *THICC BOI SP* [背后的梦之队不仅对 mod](https://www.youtube.com/watch?v=Yt0juD1gYhc) 进行了一些巨大的改进，而且他们打算在未来几个月内将其作为商业套件发布。让这一重大升级变得更宽容一点的诀窍是使用定制的扁平柔性电缆，该电缆允许用户轻松焊接所有必要的测试点和连接，以及将所有所需硬件整合在一起的定制 PCB。

在下面的视频中， *Macho Nacho Productions* 的【蒂托】讲述了他与【凯尔】和【埃尔德】一起工作的最新版本的 mod，并提供了一个完整的分步安装教程，让观众了解一旦套件上市，他们将面临什么。虽然这仍然是一个相当复杂的修改，但新的设计令人惊讶地平易近人。正如我们在之前的控制台改装中看到的，[扁平柔性技术的使用意味着安装不会对任何有焊接经验的人构成太大的挑战](https://hackaday.com/2021/03/04/arduboy-fx-mod-chip-now-youre-playing-with-power/)。

[![](../Images/c92b93d89ca0fe4eb6a0eb4b50666f37.png)](https://hackaday.com/wp-content/uploads/2021/09/thiccboi_detail.jpg)

The flat flex cable allows for an exceptionally clean install.

有些人可能会因为这次更换的后面板甚至更厚而推迟，但希望隐藏在改造单元内的 4,500 mAh LiPo 电池组所实现的前所未有的运行时间将有助于缓解你在携带笨重外壳时可能产生的任何不适(身体或其他)。即使配备了像售后 IPS 显示器和闪光推车这样的耗电设备，新电池也可以让您的 SP 运行近 20 个小时。如果到那时你还没有打败 *Metroid: Zero Mission* ，无论如何是时候休息一下，反思一下你的生活了。

根据[Tito]的说法，获得 ABS 注塑成型的新后面板所涉及的物流挑战和相当大的前期成本是目前阻止套件发布的主要障碍。目前的原型似乎是用树脂 3D 打印的，只是与 GBA SP 的原始外壳的外观和感觉不匹配，不足以成为一个可行的选择。众筹活动应该会让他们度过最初的难关，随着事情向生产发展，我们会关注更多的更新。

作为一个一次性的项目，这个 mod 的前一个版本已经足够令人印象深刻了，但是我们很高兴看到这个团队采取下一步措施，使 GBA 的这一引人注目的演变更加广泛。这是一个极好的例子，说明当你利用现代硬件黑客武器库中的所有工具时，小团队甚至个人都可以做些什么。
