# 来自迪拜的 led:你买不到的皇家灯

> 原文：<https://hackaday.com/2021/01/17/leds-from-dubai-the-royal-lights-you-cant-buy/>

[Clive]有一段关于飞利浦 LED 灯的有趣视频。除非你住在迪拜，否则你买不到它们。显然是受到了迪拜酋长穆罕默德·本·拉希德·阿卢·马克图姆的启发，他想要更高效、更持久的灯泡。秘密？一个普通的 LED 灯泡使用一个 1 瓦的 LED“灯丝”。迪拜的灯泡运行时间大约是它的四分之一，这意味着他们需要更多的 led 来获得同样的光量，但它们应该持续更长时间，运行效率更高。

在探索了不同灯具的亮度和颜色后，[Clive]撕开了一盏，发现里面有一些惊喜。每个发光二极管的电压超过 200 伏，驱动电路有许多对元件，可能是为了在高电压下保持较小的尺寸，尽管也可能是为了提高可靠性，[Clive]不确定。

通过降低功率，[Clive]能够计算出每个 LED 条包含 21 个 LED。他还指出了建筑中的一些奇怪之处，看起来是为了制造的可靠性和简易性。我们不确定这与传统灯泡的构造相比如何。该电路包括一个桥式整流器和一个使用 MOSFET 的线性电流调节器。

灯泡的价格会高一点，但如果你考虑到可能的长寿命，它们的总成本应该是合理的。总的来说，有趣的是，一个好的设计来自于政府监管。当然，这是有代价的:作为开发灯泡的交换，飞利浦在未来几年拥有制造和销售灯泡的专有权。他们预计到 2021 年底销售 1000 万盏灯，尽管目前只在迪拜有售。
