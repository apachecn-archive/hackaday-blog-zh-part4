# 冰上电子设备

> 原文:[https://hackaday.com/2019/04/12/electronics-on-ice/](https://hackaday.com/2019/04/12/electronics-on-ice/)

我们看到各种各样的电子外壳穿过这些神圣的页面。激光切割的木制建筑，带有别致的弯曲切口，专业制作的丙烯酸盒子，甚至偶尔还有用混凝土浇铸的设备。[Mike Kohn]认为所有这些都太持久了，于是选择了不同的材料——冰。

[Mike]分享了他在电子和冰水实验中的起伏。用密封在 DAP Ultra Clear 中的电路进行的初步测试基本上是成功的。安装了一个倾斜传感器，以使电池在冷冻过程中保持不排水，一旦在几个小时的解冻后释放，电路就可以工作了。后来的建造需要更多的工作——特别是钢筋混凝土车需要一些尝试来避免模具泄漏。不过,“冰手”是一个特别的亮点——由橡胶手套制成，在万圣节到来时会是一个非常怪异的装饰品。

这是一项关于使用这种短暂材料所需技术的研究，从中可以学到一些东西。密封电子设备是好的，最好的结果是简单的电路和几个发光二极管，让一切都很好地发光。
