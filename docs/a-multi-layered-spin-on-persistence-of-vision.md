# 视觉暂留的多层旋转

> 原文：<https://hackaday.com/2019/07/20/a-multi-layered-spin-on-persistence-of-vision/>

通过利用人类视觉的持久性，我们可以使用少量的硬件来创造一个大得多的显示器的幻觉。我们在这里介绍了许多 POV 项目，但它们几乎都是二维探索。[Jamal-Ra-Davis]用他的[体积视点显示器](https://hackaday.io/project/159306)将它扩展到第三维。

已经建造了一个 6x6x6 的 LED 立方体，[Jamal]想把它做得更大，但不喜欢增加三维阵列的工作量。为了避开所需努力的指数增长，他转而使用视觉暂留，通过旋转光源，从而增加其效果。

当前版本有六个垂直堆叠的臂，每个臂有八个可单独寻址的 APA102 LEDs。当旋转时，这 48 个 led 产生有效分辨率为 60x8x6 的 3D 显示。

一年多前，我们在 2018 年湾区创客大会上看到了这个项目的早期版本。(下面是当晚的演示视频。)它被搁置了一段时间，但现在已恢复积极开发，成为 2019 年[黑客日奖](https://prize.supplyframe.com/)的参赛作品。[Jamal-Ra-Davis]希望将他的原型发展成可以作为套件出售的东西，所有信息都已公开，因此其他人可以在这项工作的基础上进行开发。

我们已经在一个玩具陀螺中看到了二维旋转 POV LED 显示屏[，我们也看到了一些 POV 项目采取](https://hackaday.com/2018/12/29/pov-tops-hobbyist-abilities/)[步骤进入第三维](https://hackaday.com/2019/05/04/star-wars-electrostaff-effect-done-with-spinning-leds/)。我们喜欢这个趋势的走向。
