# 西西弗球赛机器人优雅地，磁性地辛勤工作

> 原文:[https://hack aday . com/2022/05/20/sisyphean-ball-race-robot-toils-gracefully-magnetic/](https://hackaday.com/2022/05/20/sisyphean-ball-race-robot-toils-gracefully-magnetically/)

球类比赛和大理石赛跑不是很有趣吗？如果你不用每次都爬上梯子，然后重新设置，这不是更有趣吗？[Johannes]写信告诉我们关于[一个小机器人的西西弗式任务，在一个简单但有趣的赛道上设置一个滚珠轴承，从终点收集它，然后空运回赛道的起点](https://www.printables.com/model/208954-small-robot-arm-with-ball-race)。

[Johannes]制造了这个机器人来测试小规模树脂印刷强度以及 Ali 的一些小型线性致动器的寿命，这些致动器可能会或可能不会随时提供。重点是看看这些小家伙在直接连接到 Arduino 或其他微控制器时的表现，而不是走某种电机驱动器的安全路线。

有些东西工作得很好，比如将轴保持在一起的 c 形夹，以及使用快速脉冲从夹持器中释放磁性连接的球。其他方面就没那么好了。首先，微小的树脂部件对力的反应不好。然后是执行器本身。连接很脆弱，马达很弱，但它们在不同的部分质量相差很大，所以 YMMV。有些人失去了步伐，有些人偶尔抓住。但是你不会从下面的视频中捕捉到的优雅动作中知道这些。虽然它看起来是自动化的，但由于发动机的问题，这个机器人处于远程控制之下。

不喜欢球跑？还有其他永无止境的任务，比如[以冥想的名义移动沙子](https://hackaday.com/2021/10/31/building-a-kinetic-sand-art-table/)。
