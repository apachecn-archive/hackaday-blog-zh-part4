# IOT 弹球让啤酒节充满乐趣

> 原文:[https://hack aday . com/2020/08/03/IOT-pinball-puts-Oktoberfest-fun-on-tap/](https://hackaday.com/2020/08/03/iot-pinball-puts-oktoberfest-fun-on-tap/)

我们真的不太怀念去酒吧的日子，除非你说的是市中心那家有弹球机的酒吧。不要误解我们——弹球模拟器已经变得非常好，你可以找到大多数机器的精确数字副本，在你的手机或其他设备上玩。但就是比不上玩真柜子的快感。

[![](../Images/d1d22424ec5b6108fe3a497b06ed7ba6.png)](https://hackaday.com/wp-content/uploads/2020/07/Oktoberfest-pinball-guts.png) 不要绝望，因为在接下来的几周，你可以[排队在芬兰埃斯波](https://www.surrogate.tv/game/oktoberfest)的一台真正的啤酒节弹球机上玩。控制器通过一个定制的帽子连接到一个树莓 Pi 4，还有一个摄像头指向操场，另一个聚焦在后玻璃屏幕上。游戏开发/视频流媒体公司 Surrogate 正在互联网上举办一场锦标赛，并将向前十名大赌客颁奖。

我们通常要等到假日季节才能遇到这些远程现实游戏机会。现在已经玩了几次了，我们建议在你对延迟有所感觉之前先滥发脚蹼。此外，当球在操场的上半部分时，只要举起脚蹼，就会抓住很多球，否则你可能会因为脚蹼滞后而失去这些球，有时它们会回到发射器前面再次射门。休息之后，看看一个简短但有趣的视频，视频中设置了摄像机和 Pi，其中包括啤酒节音乐的味道。

锦标赛将持续到 8 月底，这应该足够有人设置 CV 和键盘来自动播放了。需要灵感？这里有一个开源的弹球机，可以自己玩。
