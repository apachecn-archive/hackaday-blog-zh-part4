# 一个游戏男孩进步-降级！

> 原文：<https://hackaday.com/2022/11/21/a-game-boy-advance-downgraded/>

我们在这里展示了大量的游戏主机 mod，因为增强使用经典机器的体验通常需要一些非常聪明的工作。但这里有一个有点不同，不是升级他的 Game Boy Advance，[张文婷][已经将其从彩色屏幕降级为单色液晶显示器](https://hackaday.io/project/188324-game-boy-advance-gba-with-monochrome-screen)。看看休息下面的视频。

有人可能会问，鉴于 GBA 已经有大量背光彩色 LCD 升级，为什么这是必要的，但也许玩过原版的人可能会理解，这是为了改善质量相当差的彩色 LCD 原版的可视性。

有趣的还有显示控制器的选择。在那里可能会发现一个 FPGA，而不是有一个 PR2040。他详细介绍了它的程序设计，我们希望它能启发其他人关注屏幕移植。同时，如果[张文婷]这个名字对你来说意味着什么，那应该是因为他的另一项单声道液晶显示器的工作
