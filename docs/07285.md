# 伪装的坏掉的智能手机笔记本电脑

> 原文:[https://hack aday . com/2020/07/28/broken-smart phones-laptops-in-covering/](https://hackaday.com/2020/07/28/broken-smartphones-laptops-in-disguise/)

现代智能手机是关于有计划淘汰的令人眼花缭乱的论文。无论是没有四个小时和一系列微小的专门工具就无法拆卸的电池，轻微震动就会破碎的屏幕，还是(最糟糕的是)故意破解的软件，让手机看起来比实际更旧更慢，每个角落都有一些购买新设备的借口。真正悲惨的事情是，这些老旧的、有时有些破损的设备往往还有很长的寿命。

这段视频向我们展示了如何将一部旧智能手机变成一台完全可用的笔记本电脑。该产品从一个带有 USB-C 输入的屏幕和控制板开始，大多数手机都可以用它来输出视频。它内置于一个定制的铝制外壳中，带有一些铰链，然后连接到笔记本电脑底部的电池组和键盘。从那里开始，安装一个键盘，然后将旧手机固定在屏幕后面，这样铝制机身就不会干扰 WiFi 信号。

如果你需要的只是互联网浏览、信息和基本的文字处理，那么一旦摆脱了有限的移动用户界面，大多数手机实际上都有足够的能力完成所有这些工作。这种构造的天才之处在于，由于手机没有嵌入笔记本电脑的机身，这种构造也可以很容易地用于扩展现代工作手机的功能。如果你想把手机换成树莓派之类的东西，这也不是从废品抽屉里拿出一台带零件的笔记本电脑的唯一方法。
