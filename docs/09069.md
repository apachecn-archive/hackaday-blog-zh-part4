# 为您的 2070 额外添加 8GB VRAM

> 原文：<https://hackaday.com/2021/01/29/add-an-extra-8gb-of-vram-to-your-2070/>

我们大多数人都用显卡自带的 VRAM 来凑合。我们可以等到下一个出来，再多一点内存。毕竟，试图在 RAM 芯片这样对时间敏感的东西上精细地焊接新组件是疯狂的，对吗？

[VIK-昂]承担起自己的责任去做这件事。灵感来自于一张泄露的图表，显示 RTX 2000 系列可以通过使用 2GB 芯片支持 16 GB 内存。NVIDIA 从未发布过 2070 的 16GB 版本，所以这张卡是真正的独一无二的。在互联网上仔细搜索了一番后，GDDR6 芯片被采购并小心翼翼地用热风枪焊接上。必须移动一些电阻来容纳新的 RAM 芯片。在开机期间，[VIK 开机]看到所有 16 GB 枚举，并能够运行一些压力测试。不幸的是，卡不稳定，并开始有黑屏问题和不稳定的时钟。这是一个糟糕的焊点还是固件问题，很难说，但他很确信这是一个 BIOS 错误。将电阻切换回 8GB 配置产生了稳定的系统。

虽然时间更近一点，但这并不是我们在过去几个月中讨论的唯一一次 RAM 升级。休息后的视频(不是英语，但有字幕)。
