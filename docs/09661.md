# 卫星地面站回收垃圾

> 原文:[https://hack aday . com/2021/04/03/satellite-ground-station-upcycles-trash/](https://hackaday.com/2021/04/03/satellite-ground-station-upcycles-trash/)

虽然“升级周期”这个术语相对较新，但我们觉得[saveitforparts]已经做了很长时间。他以前制造的设备是为了接收低地球轨道卫星，但现在他想接收地球同步鸟类，这需要更好的天线。虽然他的[设置不会赢得选美比赛](https://www.youtube.com/watch?v=EK8mFrxxSbY)，但它看起来确实有效，还从垃圾填埋场救了一些垃圾。(视频，嵌在下面。)

剩余市场上的小菜很便宜。使用经典的 cantenna 设计，一个罐子可以制作一个很好的喇叭天线，尽管这需要铝带，因为垃圾桶里唯一的罐子是一个纸板燕麦片纸盒。当盘子被证明小了 25%时，胶带也派上了用场。

这道菜不仅难看，可能经不起日常使用。误码率有点高，但他设法从 GOES-16 下载了图像。听起来他计划对它进行风化处理并永久安装碟形天线。Raspberry Pi 脚本无法在他的笔记本电脑上正常运行，所以他最终改用了 Pi。图像看起来很棒

用垃圾建造一个地面站似乎是一种成年礼。你可能有更好的运气与其他可用的软件[。](https://hackaday.com/2020/03/14/get-your-weather-images-straight-from-the-satellite/)
