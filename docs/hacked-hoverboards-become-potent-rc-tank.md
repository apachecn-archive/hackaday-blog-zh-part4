# 黑客悬浮滑板成为强大的遥控坦克

> 原文：<https://hackaday.com/2019/08/24/hacked-hoverboards-become-potent-rc-tank/>

悬浮滑板是过去几年的宠儿，也可能是祸害，被警惕航空公司禁止，它们是打断脚踝或让你的房子着火的好方法。然而，它们也是黑客攻击的宝贵零件宝库，[[Aaron]用他的 RC 坦克模型](https://www.youtube.com/watch?v=MQ_EklhKByk&feature=youtu.be)巧妙地展示了这一点。

[Aaron]的构建不仅利用了悬浮滑板的扭矩轮毂电机，还利用了原始的控制硬件。这很容易重新利用，这要归功于由[露西·福斯]开发的原始控制器的[定制固件，我们在](https://github.com/NiklasFauth/hoverboard-firmware-hack/blob/master/README.md)之前已经介绍过他的作品[。](https://hackaday.com/2017/12/03/boxes-form-an-orderly-queue-behind-the-armchair/)

被砍下的零件被塞进一个铝挤压制成的底盘，最终的结果是一个灵活而坚固的坦克，每个轮子都有一个马达。这使得一些令人兴奋的驾驶动态。此外，利用所有可用的扭矩，[亚伦]甚至能够像骑电动滑板一样骑坦克。

这是一个有趣的构建，展示了悬浮板硬件的原始功能。我们完全有望在未来几年看到这些部件在黑客领域继续流行。休息后的视频。
