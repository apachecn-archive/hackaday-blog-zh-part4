# 来了解新一代吧

> 原文：<https://hackaday.com/2022/11/27/come-learn-about-new-attiny-generations/>

随着芯片短缺的打击，许多熟悉的 ATtiny 芯片已经变得不可获得且价格过高，并且从那时起基本保持不变。然而，如果你曾经在你最喜欢的电子元件零售商网站上搜索过“ATtiny ”,你会注意到大部分时间都有相当多的 ATtiny 芯片库存——只不过它们来自比我们通常看到的更新的一代，具有不兼容的引脚排列，略有不同的架构和更长的型号，如 412 和 3227。来自[technoblogy]的[戴维·约翰逊-戴维斯]在这里澄清一些事情，并对新一代 ATtiny 所能提供的东西进行总结。

2019 年，他发布了[关于 0 和 1 系列 ATtiny 芯片，](http://www.technoblogy.com/show?2OCH)将它们与我们所知道的 ATtiny 系列进行对比，为我们解编了零件编号方案，并提供了对比表。现在，他回来告诉我们关于 2 系列 ATtiny 芯片的情况，将比较表合并在一起，以便您可以根据其 ROM/RAM 大小和所用的 SMD 封装快速评估可用的器件。他还描述了哪些外设适用于哪些系列，以及三代产品之间外设操作的细微差别。最后，他提醒我们对所有这些新器件进行编程的一种简单方法——目前，您只需要一个 USB-UART 适配器和一个 4.7K 电阻。

在过去的几十年里，我们已经看到了许多鼓舞人心的 ATtiny 项目——从 5 GPIOs 中挤出我们所能挤出的一切，或者为更大封装的 ATtiny 芯片挤出更多。[David] [为我们树立了一个榜样](https://hackaday.com/blog/?s=David+Johnson-Davies)，带来了像这样的项目[信号发生器](https://hackaday.com/2018/03/02/tiny-function-generator-on-the-attiny85-complete-with-oled/)，这样的[连续性测试仪](https://hackaday.com/2017/11/25/continuity-tester-uses-the-attiny85s-comparator/)，或者[一个带有有机发光二极管屏幕的红外接收器](https://hackaday.com/2018/04/27/ir-detective-eases-development-with-compact-decoding/)用于诊断——所有这些都与 ATtiny85 有关。这不仅仅是引脚数量的限制，还有 RAM 和 flash 的数量——尽管如此，人们以前已经将[机器学习](https://hackaday.com/2020/01/07/tiny-machine-learning-on-the-attiny85/)和[整个图形堆栈](https://hackaday.com/2022/04/12/mindblowing-graphics-from-an-attiny85/)装入这些芯片。如果你被困在家里什么也做不了，就像我们中的许多人在封锁期间一样，[你可以随时试着做一件事情，看看你能做多少。](https://hackaday.com/2020/04/02/the-attiny-series-is-a-great-companion-in-isolation/)
