# 国产缝纫机可能是唯一的一种

> 原文:[https://hackaday . com/2022/07/17/a-自制缝纫机可能是唯一的一台/](https://hackaday.com/2022/07/17/a-home-made-sewing-machine-may-be-the-only-one/)

缝纫机是一种工具，我们许多人在工作间的某个地方都会有。隐藏在其中的是一个复杂而迷人的机制。我们中的一些人可能已经窥探过内心，但是我们中很少有人会去努力建立自己的内心。如果你曾经想知道这是否可能，[Fraens]已经做到了，他声称这可能是互联网上唯一一台完全自制的缝纫机。

如果你曾经研究过缝纫机的历史，你会注意到它与一些最早的商业机器有惊人的相似之处，具有相对较短的长度和完全开放的结构。主要的底盘似乎是激光切割的丙烯酸树脂，而所有的配件都是 3D 打印的，其他金属部件是机械加工的黄铜衬套和铝杆。该设计利用了手摇曲柄，但也与 DC 电机的图片。这是缝纫机如何工作的一个有趣的例子。遗憾的是，我们看不到任何设计文件链接(更新:他联系我们，告诉我们[他们现在在 Thingiverse 上](https://www.thingiverse.com/thing:5427892)。)，所以如果你想建立自己的方式，你可能必须有创造性。在休息下面的视频里看一下。
