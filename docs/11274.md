# 翻盖式铸造有助于管理金属铸造的危险

> 原文：<https://hackaday.com/2021/09/11/flip-top-foundry-helps-manage-the-danger-of-metal-casting/>

熔化铝实际上很容易做到，这就是为什么它是如此受金属铸造初学者欢迎的金属。建造一个可以安全熔化铝的铸造厂完全是另一回事，这得益于[安迪]在他的新丙烷动力炉中加入的一些体贴的东西。(视频，嵌在下面。)

对安全的担心一点也不过分，因为虽然铝在家庭商店的合理温度下熔化，但它仍然是一种液态金属，如果你给它一半的机会，它就会找到伤害你的方法。[Andy]的设计主要通过盖子的手动设计将这种风险降至最低。虽然大多数炉子都有一个盖子，需要用户将他或她的手靠近里面肆虐的地狱，或者在打开时危险地改变整个东西的重心，但这个炉子有一个奇妙的踏板操作的盖子，既可以抬起又可以扭转。让双手自由操作钳子也是这种设计的一个好处。

熔炉遵循了很多我们之前见过的设计线索，从一个空的派对气球氦气罐开始。衬里是陶瓷毯材料和耐火水泥的混合物；另一个很好的安全特征是在熔炉底部铸有排水通道，以防坩埚破裂。炉子也很大，至少与[Andy]以前的 DIY 单元相比是这样，并且有一个坚固的底座，有助于稳定性——这是安全栏中的另一个优点。

每当我们看到一个新的熔炉设计，我们就渴望开始进入金属铸造领域。而且进入门槛低到[一个肯德基桶](https://hackaday.com/2014/12/02/kentucky-fried-induction-furnace/)或者[一个旧灭火器](https://hackaday.com/2017/08/15/pouring-1200-tea-foundry-in-a-fire-extinguisher/)，为什么不试一试呢？尽管在开始之前[知道什么会出错](https://hackaday.com/2018/07/16/fail-of-the-week-when-good-foundries-go-bad/)肯定是有好处的。
