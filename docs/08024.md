# 复古电脑教练得到一个树莓派改装

> 原文：<https://hackaday.com/2020/10/12/retro-computer-trainer-gets-a-raspberry-pi-refit/>

我们知道你在想什么:这又是一个“为它的酷旧外壳挖出复古设备，然后用物联网垃圾装满它”的项目。好吧，请放心，[扩展和增强这款 20 世纪 70 年代的电脑训练器](https://www.smbaker.com/designing-a-memory-io-eta-3400-addon-for-the-et-3400-trainer)在很大程度上是对原始设计的尊重，虽然里面有一个 Pi，但它不会破坏复古的美好。

像许多与斯科特·m·贝克相似的年份一样，希斯基特目录可能只比年度无线电室目录翻得少一点点。一个特别令人满意的希斯基特项目是 ET-3400 微型计算机学习系统，它基本上是一个基于 6800 的计算机，周围有一个实验区。[Scott]得到了其中的一个，但没有可选的扩展附件，该附件允许它做一些有趣的事情，如运行 BASIC 或甚至支持串行端口。所以[斯科特]决定推出自己的扩展板。

[Scott]设计的扩展卡严格来说不是忠实的复制品，至少在原始 BOM 方面是这样。他转向更现代、更容易获得的组件，但仍然设法提供了原始单元的串行端口、盒式接口和 RAM/ROM 扩展。Raspberry Pi 是一个可选的附加组件，只允许他在需要时无线连接。该卡安装在 ET-3400 下方的 3D 打印盒中，保持了原始教练的外观和感觉。下面的 longish 视频显示了建立，并给出了一个参观的 ET-3400，之前和之后的 mods。

看起来，像这些和[这样的](https://hackaday.com/2019/06/17/a-faithful-replica-of-an-early-computer-trainer/)[训练器，以及来自个人电脑革命早期的其他文物](https://hackaday.com/2020/06/28/reproduction-1960s-computer-trainer-really-pushes-our-buttons/)越来越具有收藏价值。让我们希望我们没有扔掉一些东西。
