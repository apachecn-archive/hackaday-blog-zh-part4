# 电动独角兽是我们这种可以骑的

> 原文:[https://hack aday . com/2020/01/20/electric-unicorn-is-our-kind-of-ride able/](https://hackaday.com/2020/01/20/electric-unicorn-is-our-kind-of-rideable/)

当[Charlyn]带着一匹独角兽摇马去工作时，它立刻受到了欢迎。自然，人们想要更多，并建议它需要电力推进。自然地，她接受了挑战，[独角兽洛奇得到了机动升级](https://charlyn.codes/electric-unicorn/)。

该建筑由 PVC 管制成的框架组成，用 Formufit 配件连接。这些都是用 PVC 管建造有用结构的好方法，并且使建造变得轻而易举。框架有一个供骑手使用的脚踏板，摇马也有一个舒适的座位。为了推进，一个悬浮滑板安装在底座上，框架带有一对脚轮以避免翻倒。双 PVC 手柄用于与悬浮滑板的踏板连接，允许用户驾驶和转向，以及就地转向。一束假花圆了美感，并隐藏了一些拉链。

这表明 PVC 管是快速、有趣项目的绝佳材料——你只需要合适的配件就能实现这一切！看到悬浮滑板以这样一种不会导致严重受伤的方式使用也很有趣。独角兽也总能给项目带来很好的天赋。休息后的视频。
