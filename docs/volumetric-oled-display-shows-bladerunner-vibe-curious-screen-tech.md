# 体积有机发光二极管显示显示 Bladerunner 盛传，好奇屏幕技术

> 原文：<https://hackaday.com/2021/03/04/volumetric-oled-display-shows-bladerunner-vibe-curious-screen-tech/>

[肖恩·哈金斯]带着他的最新视频出来了，它本身就是一件艺术品。除了传统的项目展示和讲述，他还编织了一个赛博朋克氛围，首次展示了他用有机发光二极管的叠层搭建的立体展示。更新:他还[记录了建造](https://www.instructables.com/TENEX-Solid-State-Volumetric-OLED-Display/)。

立体显示的诀窍是能够添加第三维空间来定位像素。在这里[Sean]提供了这种能力，用十个屏幕叠加来增加深度元素。这不是一个简单的把戏。这些小型有机发光二极管显示器随处可见，但它们有一个共同的元素:像素出现在黑色背景上。[Sean]得到了一些透明的有机发光二极管面板和一些 Duck-Duck-Go-Fu，我们认为它可能是一个 Crystalfontz 128×56 显示器。为什么我们看不到更多这样的东西？任何人都知道，如果它是可能的，从其他有机发光二极管显示器移除支持到这里。(在评论里告诉我们。)

其余的构建相当简单，一个羽毛 M4 板通过 SPI 驱动十个屏幕，一个 MPU-6050 IMU 用于运动输入。外形因素赋予了一种增强现实设备的美感，视频的制作方法将这种设备置于 *Bladerunner* 或 *Johnny 助记符*的世界中。用一个隐含的背景故事来扩展这个令人敬畏的建筑，值得称赞！

如果你找不到自己的透明显示器，旋转的东西是这一领域的流行趋势。我们上周刚刚看到一个[旋转 LED 矩阵形成圆柱形显示器](https://hackaday.com/2021/02/21/volumetric-3d-television-is-here/)。我们的另一个最爱是[，一个旋转螺旋投影屏幕的立体显示器](https://hackaday.com/2012/06/14/volumetric-display-looks-like-a-davinci-sketch/)。
