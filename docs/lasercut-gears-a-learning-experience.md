# 激光切割齿轮——一次学习经历

> 原文：<https://hackaday.com/2018/08/02/lasercut-gears-a-learning-experience/>

激光切割机是非常棒的工具:它们非常适用于制作扁平的东西，甚至是你后来会弯曲的扁平的东西！这使得它们特别适合用扁平的原料制造齿轮。[鲨鱼鱼]需要亲自动手为他的机器人[生产一些齿轮，并决定分享他在这个过程中所学到的东西](https://www.youtube.com/watch?v=3ZH_rJUGLMg)。

有问题的齿轮是从中密度纤维板上切割下来的，这种板除了你在普通制造空间里能找到的最弱的激光切割机之外，几乎可以用在任何地方。面临的第一个问题是生产齿数少的齿轮时——根据所用的具体几何形状，齿数少的齿轮容易卡住。对于[鲨鱼]的齿轮来说，6 个齿似乎太小了，不能很好地工作。其他问题出现在影响齿轮啮合的切口周围，以及使用销来改善齿轮与轴的耦合，[sharvfish]在视频中对此进行了详细阐述。还有一个厚脸皮的头足类动物客串。

看到项目实施过程中面临的独特挑战总是很有趣；本周我们可能会看到六个激光切割项目，我们也可能会看到建筑商面临的六个独特问题。这是对构建过程的一个很好的洞察，当制造商分享他们的旅程以及成品时，这很棒。休息后的视频。

想知道激光切割齿轮能为你做什么？看看这个可以旋转整台电视的建筑。
