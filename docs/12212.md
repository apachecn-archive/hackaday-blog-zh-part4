# 用水瓶建造的遥控潜水器

> 原文:[https://hack aday . com/2021/12/15/RC-sub-built-with a water-bottle/](https://hackaday.com/2021/12/15/rc-sub-built-with-a-water-bottle/)

潜艇是无线电控制形式中较难建造的运输方式之一。这样做涉及到密封电子设备防止进水，并找到一种方法来控制水下的东西。这是一个挑战，但[Project Air]在他的最新作品中津津乐道。

潜艇的主体由一个饮料瓶制成，作为一个坚固的容器，上面可以安装所有必要的硬件。将瓶子装满水可以将浮力调整到中性水平。双无刷电机用于驱动，而伺服防水使用橡胶垫圈，橄榄油和密封喷雾的组合。

控制是通过一个浮动的 2.4 GHz 接收器进行的，因为高频无线电信号不会穿透水很远。浮箱还携带一个 FPV 发射机，允许潜艇通过视频信号进行导航。潜艇不使用压舱系统，而是通过将自己推入水面下进行动态潜水。

不幸的是，水在半满的饮料瓶里晃荡意味着控制潜艇的俯仰几乎是不可能的。为了解决这个问题，[Project Air]将瓶子完全装满，然后在身体外部使用一些堵塞的注射器来调节浮力。又长又重的系绳也换成了短得多的系绳，潜艇在水下行驶变得有趣多了。

这个建筑实际上是为了与[DIY Perks]进行一场友谊赛而建造的，他是 Youtuber 的同事[，我们最近报道了他的努力。](https://hackaday.com/2021/12/10/build-your-own-submarine/)还需要注意的是，使用[低频无线电装置](https://hackaday.com/2021/09/19/lego-submarine-gets-balloon-ballast-system/)可以获得更好的效果。休息后的视频。
