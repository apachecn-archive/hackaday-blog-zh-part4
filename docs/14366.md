# 从自然中再造:复兴 DT80 终端

> 原文:[https://hack aday . com/2022/07/30/re-recreated-from-nature-resuring-a-dt80-terminal/](https://hackaday.com/2022/07/30/re-reclaimed-from-nature-resurrecting-a-dt80-terminal/)

早在 1979 年，当 Datamedia 宣布他们的新 DT80 终端是 VT100 杀手时，他们对其可靠性非常有信心，他们提供了一年的保修。现在，几十年过去了，在一个旧的非法垃圾场附近的小溪中打捞出一个这样的码头后，这种信心再次受到了打击。不知道会从这个满是污垢的人造物品中期待什么，他开始了缓慢地将生命呼吸回设备的旅程。

[![](../Images/28a83d33048b1483a358fdef835a4bba.png)](https://hackaday.com/wp-content/uploads/2022/07/dt80_detail1.jpg)

Brings new meaning to the term “rooted”

考虑到主板上已经长满了一层层的泥土和树根，人们只能猜测终端实际上已经在那里呆了多久。但是清理它仅仅是个开始:一些组件不见了，其他的被证明是坏了，包括一些 rom，这[RingingResonance] [推测](https://forum.vcfed.org/index.php?threads/datamedia-dt80-and-telex-078-restoration.1237170/#post-1255299)可能是由闪电引起的，闪电首先决定了 DT80 的命运。

这才是冒险真正开始的时候，深入挖掘终端的内部生活，最终为它编写调试器和自己的固件。在 GitHub 资源库中可以找到该代码，以及所有其他研究、笔记和更多图片的链接，如果您对过去的技术感兴趣，绝对值得查看。

尽管 DT80 声称具有优势，但 VT100 占了上风，是历史记忆中的终端——并模仿它，[无论是小巧的可穿戴设备](https://hackaday.com/2017/10/24/tiny-wearable-8-bit-vt100-terminal/)还是完全相似的。但这种被遗忘也是[ring resonance]继续修复 DT80 的动力之一。总得有人去。因此，如果你碰巧有什么可以为他的努力做出贡献或与他分享的，我们相信他会感谢你向他伸出援手。
