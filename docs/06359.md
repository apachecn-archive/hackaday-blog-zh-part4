# 黑客日链接:2020 年 4 月 26 日

> 原文:[https://hackaday . com/2020/04/26/hackaday-links-April-26-2020/](https://hackaday.com/2020/04/26/hackaday-links-april-26-2020/)

天哪，真可惜:事实证明，也许有 20 亿部手机无法使用谷歌和苹果联合开发的 API 进行新冠肺炎联系人追踪。问题是这两个科技巨头炮制的方案，最近由[埃利奥特·威廉姆斯专家剖析](https://hackaday.com/2020/04/16/google-and-apple-reveal-their-corona-tracing-plans-we-kick-the-tires/)，是基于蓝牙 LE 的。如果一部手机没有 BLE 芯片组，那么它就不能与基于联系追踪 API 的应用程序一起工作，这种 API 使用有限范围的 BLE 信号作为任何两个人之间物理距离的代理。如果用户被报告为新冠肺炎阳性，那么在规定的时间段内被感染的用户的电话接收到其 BLE 信标的所有人可以被匿名通知他们的联系方式。正如埃利奥特指出的那样，围绕这一计划出现了许多问题，尤其是隐私问题，但就目前而言，成熟智能手机市场中约有三分之一的手机无法参与，发展中市场中可能有三分之二的手机不兼容。对于那些不喜欢这个方案的隐私威胁方面的人来说，拿出一部旧手机并掸去灰尘可能不是一个坏主意。

我们偶尔会报道工业环境中的工程师使用 Arduino 实现快速而简单的自动化解决方案的故事。在断言 Arduinos 不适合工业使用的评论中，这一点一致遭到了许多咬牙切齿和令人毛骨悚然的指责。不管是真是假，这样的评论忽略了一点，Arduino 解决方案通常是权宜之计或概念验证交易。但是现在纯粹主义者和学究们可以放松了，因为 [Automation Direct 正在提供兼容 Arduino 的工业级可编程控制器](https://www.automationdirect.com/adc/shopping/catalog/programmable_controllers/open_source_controllers_(arduino-compatible)/productivityopen_(arduino-compatible)/controllers_-a-_shields/p1am-100)。他们的 ProductivityOpen 系列与 Arduino IDE 兼容，同时具有工业认证并针对恶劣条件进行强化，具有丰富的屏蔽线，可用于拼凑完整的自动化控制器。对于家庭游戏玩家来说，一个能承受恶劣环境、价格仅为 49 美元的 Arduino 可能会填补一个空白。

说到 Arduinos 和 Arduino 配件，最好小心你是否有任何模块，你会受到专制政权的审查，因为你可能会被指控为炸弹制造者。据称，香港警方逮捕了一名 20 岁的学生，并张贴了一张他用来制造“远程引爆炸弹”的零件照片。奇怪的是，炸弹的炸弹没有任何无线功能，也没有真正的炸药，而是看起来很像我们在任何工作台或垃圾箱中找到的东西。相当可怕的东西。

如果你已经浏览了网飞的每一个值得狂欢的系列，并且正在寻找一点空间呆子娱乐，我们有一个给你。斯科特·曼利有一个新视频，详细介绍了用于阿波罗任务的四台不同的计算机。我们知道指导指令舱和登月舱的阿波罗制导计算机，以及将整个堆栈送入轨道并前往月球的运载火箭数字计算机，但我们从未听说过中止制导系统，这是登月舱 AGC 的备份，旨在在紧急情况下让宇航员返回月球轨道。我们也从未听说过这些机器没有一个通用的架构，以至于每台机器都有自己的字长。麻省理工学院和 IBM 之间的内讧也很有趣。

最后，如果你发现自己还有时间，为什么不试试在轨道上测试一颗军用卫星呢？这是摆在桌面上给美国空军黑客的提议，他们拥有一些轨道上最顶级的秘密硬件。 [Hack-A-Sat 太空安全挑战](https://www.hackasat.com/)旨在揭露太空硬件在几十年的封闭开发和保密过程中不小心被植入的弱点，这些弱点可能会给价值数十亿美元的不可替代的资产带来风险。资格赛要求各队在 8 月份的 DEFCON 期间攻击一颗接地的测试卫星，然后继续攻击一个轨道平台，获胜队将获得奖励。拿着黑政府资产的钱却不被逮捕？也许 2020 年并没有那么糟糕。
