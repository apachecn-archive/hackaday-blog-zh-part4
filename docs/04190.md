# 怪物布什飞机是一次性工程杰作

> 原文：<https://hackaday.com/2019/09/15/monster-bush-plane-is-a-one-off-engineering-masterpiece/>

我们所有人都梦想在生活中达到这样一个阶段，在这个阶段，我们拥有知识、技能、能量和资源来实现我们最大胆的梦想。[迈克·帕蒂]正在实现他的梦想，他对工程和航空的热情极具感染力，他建造了世界上最厉害的丛林飞机 Draco。

Draco 最初是一架 PZL-104MA Wilga 2000 ，它已经拥有令人印象深刻的 4 座短距起降(STOL)能力。它最初的 300 马力莱康明活塞发动机在 2017 年灾难性地失败了，几乎将[迈克]倾倒在犹他湖。他决定这是一个开始建造他的梦想飞机的好借口，并用普拉特&惠特尼 PT6 涡轮螺旋桨发动机取代了马达，产生了巨大的 680 马力。

几乎整架飞机都进行了升级，投入其中的工程令人敬畏，尤其是考虑到[迈克]自己完成了大部分工作。这包括重新设计的燃油系统，扩大的机翼和控制表面，新的航空电子设备，氧气系统，升级的起落架和一系列灯。翼尖着陆灯实际上来自一架波音 737。[Mike]估计升级费用在一百万美元左右。建造的所有亮点都记录在[【Mike】的 YouTube 频道](https://www.youtube.com/playlist?list=PL9OFkVHYEhoEfWmN0lQRZTemiuJCm2EZo)上的一系列视频中。我们愿意为这样的个人工作室付出什么…

[迈克·帕蒂]对疯狂飞机颇有经验。他之前建造的一架名为“湍流”的[比赛飞机，保持着美国洲际速度记录和单引擎涡轮螺旋桨飞机的平均速度记录(438 英里/小时/ 704.9 公里/小时)。](https://www.youtube.com/watch?v=2emGc7t4e4Y)

在此之前，我们已经介绍了一些[自制的飞机](https://hackaday.com/2018/06/16/building-an-ultralight-in-a-basement/)，对于普通航空爱好者来说，这些飞机更容易从[到达](https://hackaday.com/2018/11/29/adopting-an-orphaned-ultralight/)。但是永远记住，仅仅因为你的梦想是疯狂的，并不意味着它是不可能的。
