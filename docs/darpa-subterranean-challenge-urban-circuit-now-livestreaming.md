# DARPA 地下挑战赛城市赛道现已直播

> 原文：<https://hackaday.com/2020/02/20/darpa-subterranean-challenge-urban-circuit-now-livestreaming/>

目前正在进行的是 DARPA 地下挑战赛(SubT)城市赛道系统竞赛[在 YouTube](https://www.youtube.com/user/DARPAtv) 上直播到 2 月 26 日星期三。

2004 年的 DARPA 大挑战推动了自动驾驶汽车的研发。今天的自动驾驶汽车上的许多组件都可以追溯到为这场比赛开发的系统。为了刺激进一步的发展，DARPA 已经举办了几次挑战，重点是推动自主机器人技术的发展。

为了在这一挑战中取得成功，机器人必须应对会让今天的自动驾驶汽车感到困惑的地形。杂乱的环境，不同材料的不平整表面，甚至偶尔被淹没的部分都是公平的游戏。这些机器人也失去了一些以前可用的工具，如 GPS。“系统轨道”表示团队构建物理机器人系统，而不是模拟机器人的独立“虚拟轨道”。“城市赛道”是本次比赛四个阶段中的第二个阶段，这一阶段的环境集中在人造地下结构上。(想想地铁站。)有关本次比赛的更多详情以及各阶段的描述，请参见[我们的介绍帖](https://hackaday.com/2018/08/16/darpa-goes-underground-for-next-challenge/)或[比赛现场](https://www.subtchallenge.com/)。

那些不愿看机器人试探性探索未知领域(偶尔失败)的人可能会选择等待比赛结束后发表的总结。2019 年 8-10 月第一阶段(隧道电路)这里用 IEEE 谱总结[。或者你可以直接去 DARPA 了解关于](https://spectrum.ieee.org/automaton/robotics/robotics-hardware/all-of-the-winners-in-the-darpa-subterranean-challenge-tunnel-circuit)[系统赛道](https://www.darpa.mil/news-events/2019-08-22)和[虚拟赛道](https://www.darpa.mil/news-events/2019-10-30)的详细信息，总成绩[发布在比赛网站](https://www.subtchallenge.com/results.html)上。
