# 窥视这些超真实的 3D 打印眼球

> 原文:[https://hack aday . com/2019/10/29/peep-these-ultra-real-3d-printed-obstructures/](https://hackaday.com/2019/10/29/peep-these-ultra-real-3d-printed-eyeballs/)

对于人类来说，生命在眼中。机器人也是如此。眼睛越逼真，机器人就越逼真(也可能令人不安)。[lkkalebob]知道这个。[lkkalebob]在他的超真实眼球[中如此专注于视觉真实感，以至于他完善了一种从棉线的耳语中制作微小静脉的方法。](https://www.instructables.com/id/How-to-Make-Ultra-Realistic-Eyes-Using-3D-Printing/)

首先他用 ABS 打印出一个眼球空白。你会问，为什么是腹肌？它具有半透明性，使它看起来更真实。还有就是比 PLA 容易沙。用力打磨后，是时候画虹膜和纹理了。[lkkalebob]从红色棉线上刮下棉绒，用镊子将它涂在强力胶上。

[![](../Images/a53a221e32de9e22af0092e6d1ef9dd0.png)](https://hackaday.com/wp-content/uploads/2019/10/eyeball-jig.png) 我们最喜欢的部分来了。为了使整个过程更容易，[lkkalebob]设计了一个夹具系统，将眼球从制造阶段一直带到自动装置的插座中。中空的眼杯压力安装在尖头上，使其保持在适当的位置。这也给了眼球一个轴，可以夹在钻子里，方便喷枪。在休息后的构建视频中，他使用眼夹具铸造硅胶模具，然后用树脂密封眼睛。

没有打印机或者任何制造人类自动机的欲望？制作催眠装置并不需要太多时间。
