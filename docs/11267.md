# 声光滤波器利用声音来弯曲光线

> 原文：<https://hackaday.com/2021/09/10/acousto-optic-filter-uses-sound-to-bend-light/>

我们都知道光和声音都是波动现象，但却是完全不同的种类。光本质上是机电的，而声音是机械的。光可以在真空中传播，而声音需要某种介质来传播。所以看起来似乎很难用声音来改变光线，但是有了合适的设备，这其实很容易。

也许很容易，如果你习惯于到处扔激光，像“声光可调谐滤波器”这样的术语从你的舌头上脱口而出，就像[Les Wright]的情况一样。AOTF 是一种接收射频输入并将其应用于压电换能器的设备，压电换能器与氧化碲晶体相连。射频信号激励换能器，换能器振动 TeO [2] 晶体并在其中建立驻波。晶体内压缩和膨胀材料的交替带的作用类似于衍射光栅。改变激励频率，滤波器的频率也会改变。

为了探索声音弯曲光线的方式，[Les]从剩余市场上拿起了一个商业 AOTF。遗憾的是，它没有配备射频驱动器，但没关系——易贝很快买了几个，就把所需的射频发生器和功率放大器放在了他的工作台上。这些模块被放入一个外壳中，使驱动器更像一个工具，而不是一次性的，带有一个漂亮的多转壶和游标旋钮，用于精确的滤波器调整。看着输出光束在旋转旋钮时改变颜色真的很酷，更酷的是了解它是如何工作的。
