# 临时搭建的 XY 工作台可以完成工作

> 原文：<https://hackaday.com/2020/10/06/scratch-built-xy-table-gets-the-job-done/>

除非你有一大笔钱可以花光，否则要花好几年的时间才能花光一小堆的钱。无论是为了节省一点钱，还是仅仅为了迎接挑战，[从头开始工作室]正在做他的频道名称所暗示的事情，他的最新项目是一个 XY 工作台。(视频，嵌在下面。)

XY 工作台或十字工作台允许工件在二维空间平移，通常在钻床或铣床上进行。在钻床上，它们使重复的工作，如钻一系列孔变得更简单、更快捷。【车间从零开始】用钢扁条建造了大部分框架，移动部件在带有线性轴承的地面钢条上运行。带手轮的丝杠用来平移桌子。

像这样的机器要求每个桌子的相对板完全对齐，这是通过将匹配的板点焊在一起并在一次操作中钻孔来实现的。他还增加了 T 型槽顶面，通过在较窄的扁钢上焊接宽扁钢来形成。

由于缺少刻度盘，它看起来不像是用于精密工作，但我们仍然有兴趣知道丝杠定位的可重复性如何。不管怎样，它仍然是商店的一个有用的附加物。

[车间从零开始]正在建立一个相当令人印象深刻的 DIY 工具集，包括一个[磁性虎钳](https://hackaday.com/2020/05/01/scratch-built-magnetic-vise-stays-where-you-need-it/)(他已经与 XY 工作台一起使用)[磁性钻床](https://hackaday.com/2020/07/07/salvaged-drill-gets-a-magnetic-upgrade/)和一个[液压升降台](https://hackaday.com/2020/08/16/hydraulic-lifting-workbench-to-save-your-back/)。我们期待看到清单上的下一个项目。
