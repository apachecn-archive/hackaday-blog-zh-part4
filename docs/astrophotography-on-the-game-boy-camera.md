# Game Boy 相机上的天体摄影

> 原文：<https://hackaday.com/2022/04/08/astrophotography-on-the-game-boy-camera/>

Game Boy 相机是我们许多人曾经接触过的第一台数码相机。在当时，不用胶片拍照是相当开创性的，尽管按照现代标准，分辨率非常低，而且只能拍摄两位色。自从它发布以来，已经有足够长的时间，它开始成为一个受欢迎的经典，有各种各样的修改和修改，像[这个增加了现代单反相机镜头](https://hackmakemod.com/blogs/projects/taking-amazing-pictures-with-a-gameboy-camera)，让它可以拍摄月球照片。

摄像机的局限性使得建造相当具有挑战性。像曝光这样的设置在 Game Boy 相机上是自动的，不能改变，系统只允许用户改变对比度和亮度。但是小尺寸的传感器意味着天文摄影可以用一个比现代 DSLR 小得多的镜头来完成。一旦一个支架被 3D 打印出来，可以更换镜头，三脚架也做好了，就该给月球拍照了。

由于这种结构的镜头可以互换，相机也可以捕捉微距图像。该版本详细介绍了如何设置所有这些，甚至给出了如何更好地 3D 打印互锁线程的技巧，因此非常值得一看。对于其他 Game Boy 相机构建，看看[这款允许平台通过 WiFi 发送图片的相机](https://hackaday.com/2021/07/08/bringing-the-game-boy-camera-into-the-21st-century/)。
