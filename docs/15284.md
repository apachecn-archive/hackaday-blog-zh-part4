# 用水进行纳米组装

> 原文:[https://hackaday.com/2022/11/05/nanoassembly-with-water/](https://hackaday.com/2022/11/05/nanoassembly-with-water/)

水有时被称为万能溶剂。但是哈佛大学的研究人员想用水把东西放在一起，而不是把它们拆开。非常小的事情。在下面的视频中，你可以看到一个简单的 3D 打印机器，它可以编织微观纤维。

关键似乎是表面张力和毛细作用。毛细管机器使用排斥漂浮物的通道。通过移动通道，材料移动以避开通道，并且通过成形通道，可以进行各种操作，包括编织。这是那种看到就比较容易理解的东西，所以如果没有意义，就看下面的视频吧。这个例子使用微小的凯夫拉尔纤维。

像所有的微加工技术一样，有许多可能的应用。文中提到的一种是编织天线，以获得更好的微波频率天线。

这项技术很有趣，因为[不需要激光](https://hackaday.com/2022/04/19/micromachining-with-a-laser/)。根据这篇论文，这些通道是普通 3D 打印机无法处理的。激光就是我们通常看到的。但是使用这种技术，只需要一台 3D 打印机、一箱水、一些微小的纤维和一些稍大的浮子。
