# 自动行星探测器太棒了

> 原文:[https://hack aday . com/2020/06/05/automatic-planet-finder-is-out-of-this-world/](https://hackaday.com/2020/06/05/automatic-planet-finder-is-out-of-this-world/)

当世界在你的肩膀上时，想起我们只是在一块大石头和许多其他石头旁边的空间中飞奔的无毛猴子，会令人放松。如果你发现自己不知道其他主要岩石的确切位置，而不是担心，我们认为这是一个好迹象。

无论[雪球]住在哪里，在某个地方的公共广场上都有一个大型行星探测器，它很久以前就停止定位岩石了。渴望看到这样的事情发生，[雪球]制作了[一个非常好看的桌面版本](https://www.instructables.com/id/Raspberry-Pi-Planet-Finder/)，它使用水平坐标系来定位行星。里面是一个 Raspberry Pi 3，它向 NASA 查询方位角和高度数据，并将这些数据与预先确定的北方读数相结合，通过旋转顶部的打印望远镜来指出任何行星。望远镜本身不工作，几秒钟后返回北方等待输入。

如果你想制作自己的作品，这个项目是完全开放的。就像现在一样可爱，围绕滑环设计将消除所有那些长电线，使其更加光滑。休息后看一眼。

不要停止你的桌面太空玩具收藏——建造一个 ISS 跟踪灯来搭配它。
