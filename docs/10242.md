# 带有电磁换刀装置的 3D 打印机

> 原文：<https://hackaday.com/2021/06/12/magnets-how-do-they-work-on-3d-printers/>

3D 打印机的多功能性简直令人惊叹。它能够生产各种各样的原型、各种零件、艺术品，甚至其他 3D 打印机，是任何商店或制造商空间的绝佳补充。更小、更便宜的打印机可能只用一种工具就能很好地打印一种类型的东西，但如果你真的想把 3D 打印机的多功能性提升到一个新的水平，你可能想尝试一种带有自动工具更换系统[的打印机，就像这个使用磁铁](https://www.youtube.com/watch?v=A37mhOC04vM)的打印机。

来自[Will Hardy]的 3D 打印机使用电磁铁将工具连接到打印机上。机械臂能够移动到工具存储区域，并在运行时快速存放和连接各种工具。故障保护机制可以防止工具在断电的情况下从打印机的头部脱落，还包括其他几个设计功能，允许其他人根据自己的特定需求调整这种设计，例如封装打印机，并根据需要增加或减少 Core-XY 打印机的工作区域。

虽然该项目看起来运行得非常好，但[Will]指出，它仍处于原型阶段，需要对软件进行改进，以完善其操作，使其适用于更多的通用用途。虽然这是一个优秀的设计，并显示出前景。这也让我们想起了[几个月前我们展示的另一个换刀系统](https://hackaday.com/2020/12/24/xchange-promises-inexpensive-tool-changes-for-3d-printers/)，尽管电磁扭曲较少。
