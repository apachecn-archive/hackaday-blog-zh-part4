# 使用这款创新的 SMD 胶带切片机自动装配零件

> 原文:[https://hack aday . com/2022/06/07/automate-parts-kitting-with-this-innovative-SMD-tape-slicer/](https://hackaday.com/2022/06/07/automate-parts-kitting-with-this-innovative-smd-tape-slicer/)

没有人喜欢单调乏味、容易造成重复性压力伤害的体力工作，这类任务理所当然地激发了自动化解决方案。这款自动 SMD 胶带切割机是将此类杂务自动化的一个很好的例子，同时为进一步开发留下了充足的空间。

我们已经习惯了从[创新先生]那里看到这样的战术自动化项目，每个项目都围绕着一个奇怪的特定任务。在这种情况下，任务包括从卷轴上切下包含特定数量的 SMD 电阻的条带，可能是为了组装零件套件。机制很简单:一个带橡胶摩擦轮的步进电机来驱动磁带，一个看起来很恶心的闸刀来切断磁带。该切割机特别有趣，因为它使用了一个短长度的线性轴承来承载垂直于 SMD 胶带安装的剃刀刀片的支架。固定器通过曲柄安装在一个小电机上，当适当数量的零件被送出时，电机旋转一周，驱动倾斜的刀片快速下降，然后再上升。这导致剪切切割，而不是由[创新先生]在[这台自动钢丝剪](https://hackaday.com/2020/12/09/this-automated-wire-prep-machine-cuts-and-strips-the-wire/)中看到的剪切动作。

奇怪的是，似乎没有反馈机制来实际测量分配了多少电阻。我们假设[创新先生]只是计算步数，但集成一个光电传感器来计算磁带中驱动链轮孔的数量似乎很容易。似乎几个简单的改变就能使这种机器适应不同尺寸的 SMD 带，使它对 SMD 配套普遍有用。不过，作为一个战术项目，它仍然很酷，并且在激励未来的改进方面做得很好。
