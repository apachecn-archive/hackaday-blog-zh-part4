# 3D 打印刚性链机构

> 原文：<https://hackaday.com/2020/12/08/3d-printed-rigid-chain-mechanism/>

3D 打印的主要优势之一是能够快速测试，然后迭代机械设计。[gzumwalt]做了很多这方面的工作，最近一直在研究各种版本的刚性链机制。(视频，嵌在下面。)

刚性链机构是将长横梁装入小盒子的一种方法。它的工作原理类似于拉链，用特殊设计的齿将两个独立的“链”啮合起来，形成一个刚性梁。由于齿间的间隙，光束往往有点松软。[gzumwalt]制造了各种尺寸的机械装置，并在后来的版本中减少了间隙以减少翻转。他还通过添加可逆变速箱和二元速动开关，将其集成到一个很酷的[“篮子里的蛇”自动机](https://www.youtube.com/watch?v=76c8taVgwKs)(下面的第二个视频)。

这种机制的一种可能用途是在太空中自主组装长结构，这是 2017 年 Hackaday Prize 决赛项目之一， [ZBeam](https://hackaday.com/2017/08/10/hackaday-prize-best-product-finalist-shape-shifting-structures-for-space/) 提出的。

[gzumwalt]还没有提供这些文件的下载，但是你可以继续关注他的 [Instructables](https://www.instructables.com/member/gzumwalt/) 页面的更新。他得到了一些已经上市的令人着迷的 3D 打印设备，比如[多米诺骨牌铺设机](https://hackaday.com/2019/09/09/one-motor-domino-laying-machine-works-for-tips/)或者 [WiFi 控制的漫游车](https://hackaday.com/2018/01/02/remember-when-scratch-built-robots-were-hard/)。
