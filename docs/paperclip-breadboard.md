# 回形针试验板

> 原文：<https://hackaday.com/2019/05/21/paperclip-breadboard/>

TV 的 MacGyver 会喜欢我们最近看到的试验板布置:它使用[回形针和压接来制作电路](https://www.instructables.com/id/Make-a-Breadboard-for-Electronic-Circuits-Papercli/)，无需焊接就可以或多或少地保持永久性。基本想法很简单。纸板底座上贴有一张纸。金属回形针是弯直的，用 PVA 胶水粘在纸上(你知道，像普通的 Elmer 的；热熔胶也可能起作用)。你也许可以从旧房子的电线中回收电线，这也是可行的。

该方案使用两种尺寸的回形针。大的是直的，形成轨道，小的是连接。轨道被弯曲成有一个小“耳朵”,可以推入纸板底座，使它们保持不动。一点胶水就能稳定它们。耳朵从后面伸出来，所以作者建议用胶带、热熔胶或另一张纸板盖住耳朵。使用鞋盒的顶部也可以解决这个问题。

使用钳子，较小的回形针可以夹住元件。显然，你需要通孔元件，任何类型的 IC 都可能需要一个小小的适配器板。这种技术更适合没有 IC 的简单电路。对于快速连接，您可以卷曲一个单钩或使用一个双钩进行更牢固的连接，这种连接更难安装。

还有其他临时物品，如磁性电池座。如果你真的喜欢焊接，我们已经做了一些非常类似的事情，使用铜箔带作为轨道。你可以在出售玻璃染色用品的工艺品商店找到这种带子。实际上，你可以像对待 PCB 走线一样对待胶带，只要你记得在它们交叉的地方焊接，因为粘合剂可能会使两片胶带绝缘。

这将是一个有趣的有孩子的雨天项目。我们怀疑它会取代无处不在的试验板，尽管有一些想法正试图做到这一点。我们也看到其他人用回形针做各种[不寻常的用途](https://hackaday.com/2018/08/17/no-sd-card-slot-no-problem/)——包括曾经流行的[回形针电脑](https://hackaday.com/2015/10/19/diy-computer-1968-style/)。
