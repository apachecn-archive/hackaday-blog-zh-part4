# 一个巨大的，吝啬的，膨胀的机器

> 原文:[https://hack aday . com/2019/03/01/a-big-mean-inflated-machine/](https://hackaday.com/2019/03/01/a-big-mean-inflated-machine/)

吉普车越野很有趣，摩托车也许更有趣。顽固的叛逆者甚至走得更远，开着单人摩托和雪地摩托到处跑。然而，随着他的最新项目——令人惊讶的充气汽车，他可能已经超越了所有这些项目。

尽管名字如此，它却是一种挑战清晰定义的交通工具。由轻质铝框架和暴露的座位组成，结构几乎 100%被黑客攻击。PVC 织物与高级胶带一起用于制造直径为 2 米的充气车轮。真空吸尘器用于给巨大的轮胎充气，定制的 3D 打印阀门确保均匀充气。驱动是由四个手持混凝土搅拌机提供的，重新利用了它们的扭矩电机和坚固的齿轮系。甚至用户界面也是现成零件的胜利——包括以前的无绳电钻，用于 PWM 硬件，并覆盖有额外的开关。

这台机器看起来像一个奇怪的 20 世纪 50 年代未来版本的月球着陆器，就其大小而言，令人印象深刻地灵活。第一集从一个连接到充气装置的轮子和一个驱动马达开始。短短几个月后，第七集的原型机器从后花园爬出，来到了街上。这台机器的速度和能力已经令人印象深刻，我们迫不及待地想看看接下来会发生什么。

这是一个在规模上确实令人印象深刻的建筑，尽管我们已经开始对[令人惊叹的迪伊项目]抱有同样的期望。休息后的视频。
