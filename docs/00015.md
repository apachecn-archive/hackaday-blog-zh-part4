# 用于拾取和放置的眩目按钮

> 原文：<https://hackaday.com/2018/07/14/blinging-buttons-for-pick-and-place/>

随着 3D 打印、廉价的 CNC 机器和大量可用硬件的出现，看起来非常光滑的控制面板变得越来越普遍。我们特别喜欢那些带有镀铬边框的漂亮指示器，以及带有 LED 背光的配套按钮；这些真的可以在一个小组上发表意见。

然而，对[Proto G]来说可悲的是，他选择的指示器中的 led 只是无聊的旧单色单元，所以他把它们换了出来，制成了这些可寻址的 RGB 指示器。库存灯是不便宜的单位，但他们确实有一定的外观，他们足够大，允许一点点修改的空间。最初的内脏用 Dremel 移除，为 Neopixel 板让路。[Proto G]希望将电路板的焊盘扩展到螺丝端子，因此他必须将手头的 3.0 毫米节距块适配到 Neopixel 板上的 2.54 毫米节距块，但这实际上比你想象的要整洁。用一点热胶水把它们粘在一起，他现在有了完全可寻址的指示器，可以菊花链在一起，只占用一个 GPIO 引脚。

这些指示器和它们上面漂亮的面板是德尔塔拾放机器人建造的一部分[Proto G]已经工作了一段时间。他也有一些有趣的兼职项目，比如[世界上点击量最大的数字钟](https://hackaday.com/2018/01/29/the-noisiest-seven-segment-display-ever/)和[为最终用户简化 ESP32 设置](https://hackaday.com/2018/01/23/easier-end-user-setup-for-esp32-projects/)。虽然我们喜欢他所有的东西，但我们迫不及待地写下了完成的三角洲。
