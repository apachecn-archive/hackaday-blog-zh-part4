# 为飞行模拟器制作 DIY 飞行轭架

> 原文:[https://hack aday . com/2022/03/29/building-a-DIY-flight-yoke-for-flight-simulator/](https://hackaday.com/2022/03/29/building-a-diy-flight-yoke-for-flight-simulator/)

飞行轭是在玩飞行模拟器时获得真实体验的关键，但[米歇尔·雷希丁]不想花大价钱购买商业解决方案。最后，他用自己手头的很多零件做了一个设计，既省钱又很棒。

构建基于 Arduino Micro，它从轭架和踏板读取一系列电位计来控制俯仰、滚动和偏航，然后添加一系列按钮来控制飞机和模拟器软件的辅助功能。

大部分构建使用旧的 3D 打印机组件，包括用于平稳操作的俯仰轴的线性轴承和杆。甚至还有一个油门设置和一些更多的按钮和开关，以获得更完整的飞行体验。

任何人都可以在 Thingiverse 上找到想要复制[Michael]的版本的文件。我们喜欢看到一个从零开始构建的枷锁，尽管我们也看到过创造性的构建将 PlayStation 控制器[重新用于相同的目的](https://hackaday.com/2020/11/04/dualshock-flight-simulator-yoke/)。休息后的视频。
