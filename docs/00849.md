# 用空汽水瓶打造你的下一个跳舞机器人

> 原文：<https://hackaday.com/2018/10/05/build-your-next-dancing-robot-from-empty-soda-bottles/>

当你考虑下一个大型跳舞机器人的材料时，汽水瓶可能不是你首先想到的。但是根据哈索·普拉特纳学院一群学生的项目，它们是可行的。他们的系统使用空可乐瓶和 3D 打印连接器来建造大型结构，在检查其负载平衡和安全性的软件中建模。该团队已经模拟并建造了高达 5 米的设计。现在，该项目更进了一步，增加了[线性致动器和铰链，这样你就可以创造移动](https://hpi.de/baudisch/projects/trussformer-3d-printing-large-kinetic-structures.html)的东西，包括一个 4 米高的电子机器人。

 [https://www.youtube.com/embed/SUwl50CpymI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/SUwl50CpymI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



新的 TrussFormer 软件模拟了当你在模型中添加线性致动器时物体将如何移动，计算出当它们移动时它是否稳定。然后，软件创建一组参数，用于在保持模型稳定和安全的同时如何激活致动器。

这是一个看起来很吸引人的项目，到目前为止他们制作的版本看起来非常棒。他们在 [Thingiverse](https://www.thingiverse.com/HassoPlattnerInstitute_HCI/collections/trussfab) 上发布了用于瓶子的内部连接器。他们还没有发布完整的软件，因为它仍处于测试阶段。然而，这确实看起来像是一个有趣的项目，添加运动给它带来了另一个角度。
