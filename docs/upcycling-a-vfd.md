# 升级 VFD

> 原文：<https://hackaday.com/2021/09/23/upcycling-a-vfd/>

许多电子产品最终被扔进了垃圾填埋场，当[好玩的电子产品]看到一台旧的收银机被扔进垃圾堆时，他决定给它的 [VFD 显示器一个新的生命](https://github.com/playfultechnology/arduino-VFD-RS232)作为 Arduino 外设。虽然你可能找不到完全相同的部分，但看着他完成这个过程仍然很有趣，下次你看到一些旧技术走向牧场时，你可能会找到一些做自己的 upcycle 项目的技巧。

该项目相对简单，因为显示器的数据是可用的。这意味着通过 RS232 与销售点打印机连接，因此使用它非常简单。

虽然 VFD 不是最新的技术，但它们确实工作得很好，特别是在明亮的环境中，如车内。看起来像这样的 RS232 显示器是有价值的。然而，我们确实在网上找到了一些出售的单位，它们的价格相当高。不过，我们敢打赌，在某个地方肯定有一些剩余物资。

我们不禁注意到，并不是所有的 VFD——或任何种类的嵌入式显示器——都这么容易被黑掉。另一方面，如果你想真正黑掉一个 VFD，你[不必把它当显示器](https://hackaday.com/2013/11/09/vfd-display-becomes-an-amplifier/)。
