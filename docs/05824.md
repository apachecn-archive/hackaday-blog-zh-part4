# IMac 维修的新深度

> 原文:[https://hackaday.com/2020/03/03/new-depths-for-imac-repair/](https://hackaday.com/2020/03/03/new-depths-for-imac-repair/)

修理一台使用了十年的台式电脑没有太多的经济意义，尤其是当它是那种屏幕集成在电脑机身上的花式电脑时，屏幕就是那个坏掉的东西。对[JnsBn]又名[BEAN]来说幸运的是，有问题的计算机在有第二个显示器的情况下仍然可以工作，所以他决定实施一个廉价的修复，通过使屏幕透明来使屏幕再次工作。

屏幕上唯一破碎的部分是背光，它与显示器本身是分开的。为了在没有昂贵的替换部件的情况下至少在屏幕上看到一些东西，他决定完全移除背光，但保留安装的显示单元。边缘有一条发光二极管，除了计算机的内部深度之外，屏幕再次可见。在电脑内部涂上一层白色塑料后，它产生了一种有趣的效果，使电脑的显示屏再次变得有用。

虽然我们中没有人，包括创造者，建议在 iMac 内部涂上 Plasti Dip，因为有火灾和/或其他灾难性故障的风险，否则不会有太大损失。只是不要把这个推到墙上。
