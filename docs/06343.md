# 用卫生纸卷筒摇滚

> 原文：<https://hackaday.com/2020/04/21/rock-out-with-toilet-paper-rolls/>

洗澡时唱歌是如此普遍的现象，很少有人对此视而不见。另一方面，在厕所里唱歌可能会引起一两个眉毛的注意，德国人委婉地称之为“stilles ortchen”，即小而安静的地方，这不是没有道理的。但是我们有什么资格去评判你在家里的隐私呢？因此，如果你觉得缺少乐器伴奏，或者忘记带吉他，[Max bjrverud]为你准备了最完美的装置。(视频，嵌在下面。)

受自行车电脑确定速度的方式的启发，[Max]采用了一套卫生纸支架，用 3D 打印的附件扩展了每个滚动支撑部分，其中包含一个磁铁，并安装了一个霍尔效应传感器来确定滚动活动。然后用 Arduino Mega 收集劳斯莱斯的传感器数据，并传递给运行纯数据的 Raspberry Pi Zero，从而产生实际的声音。传感器设置在[另一个视频](https://www.youtube.com/watch?v=N2zNemrBvtA)中简要显示。

在你拿起你的干草叉之前，[Max]早在卫生纸成为极度渴望的对象之前就已经开始这个项目了。作为互动媒体领域的艺术家，这也不是他的第一个此类项目，你可以在他的网站上找到更多他的作品[。那么，为什么我们偏偏选择了这个呢？嗯，我们能说什么呢，我们](http://bjorverud.com/)[肯定](https://hackaday.com/2017/11/08/key-tar-lets-you-jam-at-the-hackerspace/)有[的弱点](https://hackaday.com/2018/11/01/video-quick-bit-the-best-diy-musical-instruments/)对于[奇怪的](https://hackaday.com/2020/04/02/wheelbarrow-bass-drives-a-sound-garden/)和[不寻常的](https://hackaday.com/2019/08/23/when-toothbrushes-typewriters-and-credit-card-machines-form-a-band/)乐器。也许[这里有一些合作的潜力](https://hackaday.com/2020/03/21/the-fart-box-a-synthesizer-not-quite-like-others/)？
