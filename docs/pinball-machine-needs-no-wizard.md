# 弹球机不需要向导

> 原文：<https://hackaday.com/2020/05/18/pinball-machine-needs-no-wizard/>

自从他还是个小男孩时，[泰勒]就开始玩银球。和我们一样，他一生都对围绕在他们周围的错综复杂的机电兽着迷。在他最近完成的大学四年级课程中，[泰勒]组建了一个由[凯文、科迪和奥马尔]组成的机电一体化梦之队，帮助将这些愿景转化为自我游戏弹球现实。

你确实可以手动玩这个机器，Arduino Mega 会像一个普通的柜子一样记录你的分数。如果你需要挠痒痒，忽略一个电话，或者只是想看一个弹球机自己玩，它可以在飞行中来回切换。安装在操场上方的 USB 摄像头可以跟踪球的速度。每当球进入脚蹼向量时，相应的脚蹼就会自动将球击出。

我们最喜欢这个版本的部分(除了它可以自己玩的事实)是柏青哥多球功能，它设法挤进了第二个游戏和第二个级别。这个项目是完全开放的，即使你对复制它不感兴趣，[Tyler]也在构建日志中提供了大量有用的信息和链接。休息后参观一下，我们会安排免费游戏。

[泰勒]的机器使用实际的弹球机零件，这可能会迅速增加成本。如果你[滚动你自己的目标，并创造性地利用螺线管资源](https://hackaday.com/2017/11/11/a-sandbox-for-diy-pinball-design/)，建造一台弹球机不一定会耗尽你的钱包。
