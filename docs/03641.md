# 一顶记录过去的棒球帽

> 原文:[https://hack aday . com/2019/07/21/a-baseball-cap-that-films-the-past/](https://hackaday.com/2019/07/21/a-baseball-cap-that-films-the-past/)

按下按钮，绝大多数相机就会开始录像。如果你想拍摄还没有发生的事情，这是完全可以接受的行为。如果你想拍已经结束的东西，那你就不走运了。然而，[Johan Link]制造了一台专门为此设计的相机，并把它戴在帽子上。

该项目包括一个树莓派 3B，结合 1080p USB 摄像头和 5000 毫安时的电源。它们被固定在棒球帽上，以便从佩戴者的角度拍摄镜头。相机持续记录，当按钮被按下时，保存最后 7 秒钟的记录视频——非常适合在事情发生后立即捕捉。

这是一个滚动记录功能，类似于许多仪表板摄像头和动作摄像机所包含的功能。感兴趣的人可以在 Github 上找到软件。虽然[Johan]选择了纽约洋基队的帽子作为基础，我们相信它应该与你的西雅图海鹰队的帽子一样好。攻略迷应该联系服装制造商。
