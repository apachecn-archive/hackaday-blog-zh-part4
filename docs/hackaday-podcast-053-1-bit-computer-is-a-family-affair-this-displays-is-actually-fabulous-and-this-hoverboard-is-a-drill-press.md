# 黑客日播客 053: 1 位计算机是一个家庭事务，这个显示器实际上是神话般的，这个悬浮板是一个钻床

> 原文：<https://hackaday.com/2020/02/07/hackaday-podcast-053-1-bit-computer-is-a-family-affair-this-displays-is-actually-fabulous-and-this-hoverboard-is-a-drill-press/>

Hackaday 的编辑迈克·什奇斯和埃利奥特·威廉姆斯在黑客圈拥挤的街道上寻找过去一周最有趣的硬件项目。忘记翻转点显示，你需要为自己制作一个亮片显示，使用机器人手指和亮片覆盖的织物来发送信息。你可以用一台名为 WDR-1 的 1 位计算机做很多事情(也可以学到很多东西)。通过使用这些 FTDI 和 Bluepill 技巧，将 USB 端口变成嵌入式系统开发套件从未如此简单。你不会想错过联盟号的硬件拆卸。
