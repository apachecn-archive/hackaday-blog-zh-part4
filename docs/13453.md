# 自动转台让摄影测量变得轻而易举

> 原文：<https://hackaday.com/2022/04/28/automatic-turntable-makes-photogrammetry-a-cinch/>

摄影测量是制作真实物体的精确 3D 模型的一种好方法。转盘通常是这项工作中常用的工具，因为它有助于从各个角度对物体进行成像。[林建岳]想要一种在最少人工干预的情况下运行摄影测量过程的方法，[并着手建立一个自动化的转台设置。](https://www.youtube.com/watch?v=wZMI4w9Kjwo)

该建筑依靠智能手机来拍摄实物的图像。手机由 ESP8266 触发拍照，通过手机的音频插座触发快门。微控制器然后在每次拍摄后打开转盘一小段时间，将它旋转一个设定的角度。

该构建仍然需要不时地在转盘上以各种方向重新定位物体，以便捕捉否则会被遮挡的顶部和底部区域。然而，拍摄照片和旋转物体的繁重工作现在完全自动化了。

对于那些发现自己经常对物体成像以创建 3D 模型的人来说，这是一个有用的构建。[彼得]的钻机的结果看起来很棒，作为一个奖金来与生活一样的纹理准备好了。我们以前也见过其他一些伟大的摄影测量建筑。休息后的视频。
