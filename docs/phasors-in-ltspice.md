# LTSpice 中的相量

> 原文：<https://hackaday.com/2021/03/17/phasors-in-ltspice/>

[Ted]最近使用[一张纸、Octave 和 LTSpice](https://www.youtube.com/watch?v=258o7ArryTo) 演示了 RL 电路的分析。如果您愿意，Octave 代码也可以在 MATLAB 中很好地工作。如果你想认真研究电子理论，这是一个相当简单的例子，也是一个用一些工具进行锻炼的好机会。

我们喜欢这种方法，因为太多时候，仅仅使用计算机是很容易的，而不能获得手工解决问题时所获得的理解。你确实需要理解复数，但是，总的来说，数学并不复杂。

在现实生活中，你不太可能使用[Ted]展示的所有工具。但是，各有各的位置。手工解决问题有助于理解和解决问题。如果你正试图编码一个模拟，Octave 是很棒的。但是对于实际操作来说，LTSpice 是一个非常好的工具。价格也合适。

交流电路有多种工作方式，但相量法的优点是使数学相对简单。你花在计算上的时间越少，你花在理解电路行为上的时间就越多。当然，有了模拟工具，你可以不花时间在数学上，如果这是最适合你的。

当然，也有我们喜欢的其他模拟器。有一个是我们喜欢在浏览器中运行的[。或者，下载](https://hackaday.com/2015/07/20/a-breadboard-in-a-browser/)[微帽 12](https://hackaday.com/2020/01/08/commercial-circuit-simulator-goes-free/) 也是免费使用的。
