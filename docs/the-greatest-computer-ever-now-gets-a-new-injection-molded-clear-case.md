# 有史以来最伟大的计算机现在得到了一个新的，注射成型的透明外壳

> 原文：<https://hackaday.com/2019/01/21/the-greatest-computer-ever-now-gets-a-new-injection-molded-clear-case/>

麦金塔 SE/30 是有史以来最伟大的电脑。当它在几乎整整 30 年前的今天启动时，它是一个发电站。你可以把 128 兆的内存塞进去，对于 1989 年来说，这绝对是一个荒唐的内存量。你可以把以太网放进去。您可以将 1 位黑色*或*白色内部显示转换为 8 位灰度显示。我想它有一个 Lisp 卡。这些只是对 SE/30 的同期攻击。现在，人们正在积极地为这台机器开发，并把 Spotify 放在上面。这个时代的 MAC 电脑有一个工具栏扩展，可以让你连接到 WiFi 网络。你很难找到一台电脑在发布 30 年后仍有这么大的粉丝群。

现在，有一个项目是为 Mac SE/30 (和 plain 'ol SE)制造[新的注塑外壳。这些案例将会被*清除*，就像那个时代的苹果原型一样。这也是 retrocomputer 爱好者接手的最困难的注射成型项目之一。](https://68kmla.org/forums/index.php?/topic/55862-custom-injection-molded-mac-se-se30-case-project-opinions-wanted/&)

多年来，我们已经看到了一些有趣的项目，为旧电脑创造新的塑料外壳。最著名的可能是 Commodore 64C 外壳的[再制造。这不是一个纯粹由社区推动的项目，而是一个历史的意外。故事是这样的，一个人，[达拉斯·摩尔]去了一个注塑厂的拍卖会。店主提到了一台旧电脑，有人的脑袋开始转动。后来在 Kickstarter 上，每个想要新 C64 外壳的人都得到了一个。你可以买一个半透明塑料的来搭配复古美学。](https://hackaday.com/2015/03/11/hackaday-retro-edition-a-new-commodore-64-case/)

[由于一名粉丝的 Solidworks 技能和 Kickstarter 活动，Amiga A1200 的新外壳也已制作完成](https://www.kickstarter.com/projects/a1200housing/new-amiga-1200-cases-made-from-new-molds)。显然，翻新计算机的再制造外壳有一个市场，而且这个市场仅仅大到足以支持制造新的注射成型工具。

所以，关于那个 SE/30。68k 麦金塔解放军论坛上的人们正在讨论为苹果有史以来最伟大的电脑制作新外壳的可能性。[这个故事的主人公是【maceffects】](https://68kmla.org/forums/index.php?/topic/55862-custom-injection-molded-mac-se-se30-case-project-opinions-wanted/&page=6&tab=comments#comment-597132)他已经做好了 SE/30 的后“桶”模型，并在细丝打印机上打印出了一个(查看下面的视频)。然后这个被打印成了清晰的 SLA，下一步就是众筹了。

虽然这不是一个完整的案例——需要一个前挡板来完成这个案例——但它是逆向计算社区所能做的一个惊人的例子。实现这个项目的总成本约为 15，000 美元，这完全在众筹活动可以接受的范围之内。第二次运行可能包括半透明的邦迪蓝聚碳酸酯外壳，但这纯粹是某人的猜测，他知道什么是有史以来最酷的项目。
