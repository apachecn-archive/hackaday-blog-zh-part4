# 这是一个电视范围吉他放大器！

> 原文:[https://hack aday . com/2019/10/26/its-a-TV-scope-guitar-amplifier/](https://hackaday.com/2019/10/26/its-a-tv-scope-guitar-amplifier/)

吉他放大器是一个常见的项目，尽管在纸面上它只不过是一个简单的音频放大器，但在寻找特定声音的过程中，它隐藏了数量惊人的变量。我们见过很多，但从来没有一个像[内特·克罗森]的[CRT 电视吉他放大器](https://www.youtube.com/watch?v=tyBmo3i8Eac)一样。LM386 不仅驱动扬声器，他还用它将电视变成一个粗糙的示波器，以形成声音的可视化。

展示这一壮举的视频低于休息时间，由于缺乏技术信息，它使我们陷入了困境。他用电视的 50 赫兹锯齿场时基驱动水平线圈，用 LM386 的音频驱动垂直线圈。我们不确定他是旋转了轭还是连接被交换了，但结果肯定令人印象深刻。

既然技术细节没有我们想要的那么多，为什么这个项目会引起我们的兴趣呢？因为它提醒人们，CRT 电视不仅仅是一个无用的时代错误，它是一个复杂的模拟设备，具有重大而独特的黑客潜力。特别是旧的那些提供了无限的修改和电路弯曲的可能性，并以非常优惠的价格成为一个迷人的模拟游乐场。然而，值得指出的是，其中涉及的一些电压可能会使粗心的黑客面临危险。如果你感兴趣的话，[看一下我们对一个老型号的探索](https://hackaday.com/2016/04/04/my-most-obsolete-skill-delta-gun-convergence/)。
