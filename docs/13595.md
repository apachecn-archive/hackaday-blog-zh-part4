# 填缝枪成为燃油管路配件的有用冲压工具

> 原文:[https://hacka day . com/2022/05/11/caulking-gun-变得有用-冲压-工具-燃料-管路-配件/](https://hackaday.com/2022/05/11/caulking-gun-becomes-useful-press-tool-for-fuel-line-fittings/)

简单的填缝枪真的很有用，当你在修理浴室或四处喷洒建筑粘合剂时。然而，通过几个简单的修改，它也可以成为技工车间的一个很大的帮助。

![](../Images/c658162424d888957fb77360383a01e0.png)

It’s a great tool for cleanly pushing fittings into nylon fuel line.

这种建筑由一系列 3D 打印部件组成，可以很容易地适应普通的填缝枪。首先是一对燃油管路夹，它们用螺母和螺栓紧固在一起，尼龙燃油管路插在它们之间，拧紧螺栓，将管路牢牢固定在填缝枪末端。然后，将要安装到管路中的配件放在填缝枪的活塞上。然后，只需拉动填缝枪上的扳机，慢慢将配件压入尼龙线中。

这是一个伟大的黑客创造了一个有用的线性新闻只有几分钟的 PETG 细丝。如果你发现自己在现代汽车上做一次性的燃油管路工作，这可能正是你需要的工具。那些渴望打印自己的零件的人可以在 [Thingiverse](https://www.thingiverse.com/thing:5364234?fbclid=IwAR1NMR8j2YqVGQUo36iuariPSarhrojKKrlKI8FMfejbyLS4mVYdN2CUUj4) 上买到。该设计是为 3/8 英寸管线设计的，但是可以容易地修改或重新设计以适应其他直径。

3D 打印工具可以在各种方面发挥作用，甚至在重型应用中[如冲压工具](https://hackaday.com/2021/10/29/can-3d-printed-press-tools-produce-repeatable-parts/)。它通常没有传统金属工具的寿命长，但是对于小型的一次性工作，价格的节省往往比工具本身的坚固性更重要。[如果你自己已经有了一些很棒的 3D 打印工具，不要犹豫，给我们写信吧！](http://hackaday.com/submit-a-tip)
