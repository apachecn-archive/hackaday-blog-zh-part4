# 手推车低音驱动一个健全的花园

> 原文：<https://hackaday.com/2020/04/02/wheelbarrow-bass-drives-a-sound-garden/>

制作音乐最大的好处之一就是它很容易做到。如果你愿意尝试一下，有无数种方法可以让几乎任何东西发出有趣的声音——只要问问曾经用雪茄盒和扫帚柄制作过吉他的人就知道了。

[邪恶的乌贼]在园艺工具世界的肥沃土壤中挖掘，培育出一种三弦竖式贝斯，声音浑厚、深情，来自一辆熟悉的老黄牛——一辆铝制独轮车。大部分建筑由再生木材制成，如旧门框的实心桃花心木颈部和扫帚柄。

低音是拱形顶部风格，这意味着共鸣板——前面有 f 孔的木头——是一个平板，固定在横跨“手推车”宽度的弯曲肋上。前后安装的扫帚柄音柱将振动从音板推至铝制机身。为了使农业美学更加完整，[邪恶的鱿鱼]用除草机低音弦来串起它，毫无疑问，这是受实际修剪线的启发。

声音已经很大了，但是[邪恶的乌贼]增加了一个压电拾音器来把它推进录音棚。拍击你的方式过去休息听一首小曲。

你的仪器制造技能是在树苗阶段吗？从简单的东西开始，比如一个滑动的橡皮圈。
