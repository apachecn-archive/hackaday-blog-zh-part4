# 忘记为你的浴缸酒打印标签，为什么不雕刻瓶子？

> 原文:[https://hackaday . com/2019/11/02/forget-printing-labels-for-your-tubbush-hooch-why-not-ember-the-bottle/](https://hackaday.com/2019/11/02/forget-printing-labels-for-your-bathtub-hooch-why-not-engrave-the-bottle/)

[BlueFlower]送来了这个酷酒瓶雕刻师。这是一个简单的机器，让我们想起了[臭名昭著的蛋机器人。](https://hackaday.com/2019/04/17/what-can-you-learn-from-an-eggbot/)一个轴可以在 x 和 z 方向移动，而另一个轴旋转工件。蛋机器人在球坐标系中工作，而这个机器人生活在圆柱世界中。

该设备的基础似乎是[BlueFlower]的一个 [XY 绘图仪/切割机](https://www.thingiverse.com/make:374703)的旧项目。绘图仪本身是一个非常标准的双电机龙门式设计。事实上，当机器转换为瓶子雕刻时，先前移动 Y 轴的驱动器被重新用于移动两个滚筒。滚筒本身与 2D 打印机内部发现的滚筒十分相似。我们都把它们放在我们的垃圾抽屉里，但很少看到它们被真正使用。spindled 只是一个 DC 发动机与球磨机耦合到结束。

至于最后的结果，我们不得不承认，雕刻的瓶子相当引人入胜。休息后捕捉雕刻过程的视频。
