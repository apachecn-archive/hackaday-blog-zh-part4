# 给被驱动的手指增加制动器

> 原文：<https://hackaday.com/2021/11/08/adding-brakes-to-actuated-fingers/>

为人类制造外骨骼是机器人学的一个快速发展的分支。无论是通过增加力量来提高人类的自然能力，还是帮助残疾人，该领域都有足够的空间来进行新发明，以增强人类的能力。芝加哥大学的一个团队最近向我们展示了一种给机器人手套添加刹车的方法，这种方法给人留下了深刻的数字控制。

机器人手套被称为 DextrEMS，但实际上它本身并不移动手指。这是通过前臂上的一系列电极来处理的，这些电极使用肌肉电刺激(EMS)来刺激手指肌肉，因此得名。电磁操纵手指的问题是精度不是很高，容易引起振荡。这就是手套的用武之地:每个手指都包括一系列棘轮机制，充当制动器，可以足够精确地定位手指，以便用手语做出可理解的手势，甚至弹吉他或钢琴。

对于任何对机器人或外骨骼感兴趣的人来说，这份白皮书值得一读。将这种精确度添加到操纵小到手指的外骨骼上，开辟了一个勇敢的机器人新世界，但如果你正在寻找一种可以在整个人体范围内操作的东西，那么看看这个全尺寸力量倍增外骨骼，它可以帮助你举起超人的重量。
