# 万用表走向大屏幕

> 原文：<https://hackaday.com/2021/12/07/multimeters-go-big-screen/>

我们最近注意到，一些便宜的仪表已经转向彩色大屏幕。屏幕不是点阵，但仍然有许多有用的图形，也可能分散人们的注意力，这取决于。真正便宜的似乎更像是一个噱头，但[OM0ET] [看起来像一个公平的中音乐器](https://youtu.be/by3jdy3SeBA)有一些有用的显示功能，GVDA GD128。

显示屏上的很多内容显示了血糖仪的当前功能。不需要昂贵的多位置开关或成排的互锁按钮。许多这种新型电表还配有非接触式电压传感器，非常方便。否则，它看起来就像一个非常传统的廉价仪表。

现在，许多这类仪表都有自动模式，它们会尝试猜测您需要哪种模式来进行特定的测量。本质上，它寻找电压、电阻或短路。当然，您也可以选择特定的模式。如果你感兴趣的话，它还可以测量和显示温度。

我们喜欢的是显示器底部有一个条形图，这样您就可以获得模拟仪表的一些优势。尽管数字电子产品盛行，但人类仍然是模拟的。光是这个功能就值 40 或 50 美元，这取决于你如何使用计价器。

当然，便宜的电表有着便宜的历史。此外，像这样的电表没有那么差的可靠性开关。如果你想要终极米，钩住一个 [HP3458A](https://hackaday.com/2021/10/18/the-hp3458a-king-of-multimeters-for-three-decades/) 。但是它更大也更贵。
