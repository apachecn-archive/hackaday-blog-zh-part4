# 在实验室培育蛋白石

> 原文：<https://hackaday.com/2021/01/11/growing-opals-in-the-lab/>

蛋白石在宝石中是独一无二的，由微小的二氧化硅纳米球组成，以精确的结构排列，当从不同角度观看时，赋予它们特有的变色。[思想商场]喜欢挑战，所以开始培养自己。

这不是有史以来最难的宝石合成，但也不是小菜一碟。这一过程需要原硅酸四乙酯，或 TEOS，这可能很难找到，但所需的其余化学物质是普遍存在的。初始阶段包括将 TEOS 与反应物混合以形成 200-350 纳米宽的纳米级二氧化硅球。球体溶解后，混合物必须小心干燥，以形成合适的结构，产生蛋白石著名的色彩效果。在这个阶段，工业生产者添加更多的二氧化硅来稳定基质，尽管[思想商场]无法找到解释如何做到这一点的文献。相反，他依靠树脂，虽然不完美，但确实让标本稳定下来，并在视频中展示出来。

该视频指出，这一过程中的许多步骤在几十年前就已经完善，但仍被视为商业秘密，使得复制成为一项实验。尽管如此，我们还是成功地制作出了明显的乳白色标本，我们迫不及待地想看到 DIY 过程的进一步完善。
