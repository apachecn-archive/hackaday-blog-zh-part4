# 用这个文字喷涂机器人上街

> 原文：<https://hackaday.com/2019/10/25/beat-the-streets-with-this-text-spraying-robot/>

在争取气候变化意识的过程中，你可能需要一些额外的帮助。这就是[Anred Zynch]在建造[Strettexter](https://www.instructables.com/id/Streettexter-Writing-Robot/)时的想法，这是一个喷到街道上的文本喷射写作机器人。

该机器装有 8 个喷雾罐，放置在一个木箱中(一条带有木制壁架的停止线，以防止罐掉出)，并固定在滑板的顶部。它使用焊接在 Arduino Uno 上的 PWN/伺服屏蔽，连接到 8 个伺服电机(TowerPro SG90s)来控制每个喷瓶。表格将每个字符转换成 5×8 位字体，以适合喷射模块的大小。该装置还包括一个安全开关，以及一个用于测量水平移动距离的编码器。

Strettexter 在设置好并连接上电源后，通过拉动滑板来激活(为了便于携带，它使用了 8000 毫安时的电源)。在它当前的配置中，单词很长，但是根据画布的限制，一些额外的测试可能会产生更好的结果。单词越短，白色文本越难辨认，因为打印的比特之间有很大的间隔。

我们不宽恕公共破坏，所以使用这个黑客在你自己的判断。
