# 自定义咬合传感器取代键盘表情踏板

> 原文:[https://hack aday . com/2021/01/26/custom-bite-sensor-replaces-keyboard-expression-pedal/](https://hackaday.com/2021/01/26/custom-bite-sensor-replaces-keyboard-expression-pedal/)

有时，标准硬件在不同能力的情况下无法发挥作用。[Ben Krasnow]发现自己最近在为一位不能使用双腿的音乐家工作，因此需要创建一个新的界面来取代标准的表情踏板。

[Ben]是“构建优先”工作方法的忠实粉丝，并展示了为什么这种方法效果很好。在这种情况下，第一次尝试涉及气动设计，其中用户将咬下一个气囊，该气囊通过管道和波纹管驱动远程电位计。然而，虽然这种设计有效，但触觉反馈很差。这导致了机械设计的实验，最初的尝试涉及 3D 打印机制和旋转锅。这是更好的，但仍然有牙齿损伤和手感差的问题。最终的设计基本上是一个模拟按钮，用织物浸渍的硅胶制成，以提高耐磨性，并使用线性罐来实现平滑的反馈。

最终的设计令人印象深刻地整洁，而且[Ben]指出，虽然它看起来很简单，但它是通过首先尝试大量更差的解决方案才实现的。我们也看到了游戏世界中的其他工作；最近，[模块化控制器开始流行，以满足各种各样的需求](https://hackaday.com/2020/04/10/accessible-controller-plays-around-with-modularity/)。休息后的视频。
