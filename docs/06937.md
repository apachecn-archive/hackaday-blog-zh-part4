# Eico 信号发生器得到修复

> 原文：<https://hackaday.com/2020/06/23/eico-signal-generator-gets-a-repair/>

Eico 型是一种非常常见的音频信号发生器。[无线电机械师]拿起一个 1956 年的形状相当好的，并分享了一个[拆卸和修理](https://www.youtube.com/watch?v=MKuzfZW0bmk&t=0s)的单位，你可以在下面的视频中看到。这个装置可以用几根管子产生正弦波和方波。

该单位是一个有点不同的内部比预期，因为有几个版本，共享相同的型号。箱子底部有一些粘性物质，这可不是什么好现象。不出所料，罪魁祸首是一个旧电容器。

振荡器使用文氏电桥电路，该电路采用白炽灯泡作为动态电阻元件。这是一种常见的低失真音频振荡器电路。灯泡起着自动增益控制的作用，因为冷时它的电阻较低。

我们要警告你:这个视频的结尾并不成功——它有点扣人心弦。但是第二部分就在第一段视频的正下方。如果你想更好地理解灯泡，[Jeri Ellsworth] [之前带我们走过那条电路](https://hackaday.com/2012/11/18/jeri-uses-light-bulbs-in-an-oscillator/)。我们甚至[虚拟模拟了他们](https://hackaday.com/2018/04/27/circuit-vr-oscillating-bridges/)。
