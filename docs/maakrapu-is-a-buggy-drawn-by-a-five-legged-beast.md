# Maakrapu 是一辆由五条腿的野兽拉的马车

> 原文：<https://hackaday.com/2021/10/03/maakrapu-is-a-buggy-drawn-by-a-five-legged-beast/>

蒸汽朋克美学可以采取多种形式，虽然管道、阀门和锅炉看起来很棒，但拥有带有许多移动部件的复杂机械确实会让你的项目大放异彩。坦佩雷 Hacklab 的一组蒸汽朋克爱好者建造了一辆名为 Maakrapu 的汽车。这是一辆双轮马车，看起来像是被某种五条腿的生物拉着前进。它的腿极其平滑的运动让人想起龙虾或螃蟹的形象(在芬兰语中“Maakrapu”是“陆地蟹”的意思)，也让人想起[西奥·詹森的 Strandbeesten](https://hackaday.com/2010/03/24/theo-jansen-like-the-professor-from-gilligan/) 。

木制腿与金属曲轴连接在一起，金属曲轴是由等离子切割零件焊接而成的。虽然车辆的实际转向是通过差动制动来完成的，但方向盘是用来将腿定位在行驶方向上的。早期版本没有推进力，仅用于下坡骑行，但最新型号配有一个电动马达和一个电池，这使它实际上有点用作城市跑步车。

下面嵌入的视频显示了 Maakrapu 的设计以及从坦佩雷市中心返回 Hacklab 的漫长车程。如果你喜欢有许多像这样的小移动腿的车辆，看看 [Strandbeest 自行车](https://hackaday.com/2016/06/30/a-strandbeest-bicycle/)。想要更真实的“蒸汽”朋克体验，试试[这辆蒸汽动力自行车](https://hackaday.com/2019/09/20/steam-bike-rocks-it-old-school/)。
