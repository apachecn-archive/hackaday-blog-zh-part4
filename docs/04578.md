# 手工制作的滚筒梳理机为纺纱准备羊毛

> 原文:[https://hack aday . com/2019/10/26/mand-made-drum-carder-gets-wool-ready-for-spinning/](https://hackaday.com/2019/10/26/hand-made-drum-carder-gets-wool-ready-for-spinning/)

将像羊毛这样的天然纤维制成像毛衣这样有用的东西需要很多步骤。我们可能熟悉剪羊毛、将羊毛纺成纱线，或者编织和纺织，但是在剪羊毛和纺纱之间，还有一个你必须经历的不熟悉的过程。被称为梳理，它有助于对齐纤维，使它们能够被纺，当然它需要一个昂贵的工具[或一个你自己制作的工具](https://www.reddit.com/r/homestead/comments/dioumc/my_girlfriend_has_started_spinning_wool_so_i_made/)。

这个架子鼓就像它听起来的那样。它使用两个覆盖着金属网的滚筒，以不同的速度旋转，将纤维拉成有序的形状。像这样的小鼓梳理机[可以运行大约 600 美元](https://woolery.com/fiber-tools/drum-carders.html)，但是用一些优质木材和车床，你可以很容易地用零头制作一个。用车床制作这个系列的鼓相当简单，然后你需要确保它们用优质的皮带或链条连接，然后用合适的金属网覆盖。

[kris]在 reddit 评论区写道，他希望第二个版本能转得更快一点，更耐用一点，但这仍然是一个很棒的工作卡。从那里你会想继续把羊毛纺成纱线，你可以用轮子或者电动机来完成。
