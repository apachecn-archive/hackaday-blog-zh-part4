# 农村黑客拆除和重建水力发电机

> 原文:[https://hack aday . com/2022/03/31/rural-hacker-de-crufts-and-rebuilds-hydrogen-generator/](https://hackaday.com/2022/03/31/rural-hacker-de-crufts-and-rebuilds-hydroelectric-generator/)

YouTuber(lingoer)有一个诀窍，而且是我们在 Hackaday 的页面上不常看到的诀窍:重绕和重建破旧的电机和发电机。[在](https://www.youtube.com/watch?v=t6Ivb7_8zzU)下面的视频中，你会看到[林]穿着她标志性的蓝黄相间的牛仔连身衣，拿着一个看起来像是一直放在湖底的水力涡轮和发电机，把它变成一个工作单元。

正如大多数制造商使用 MIG 或 TIG 焊机一样，[lingoer]使用简单的(可能是 A/C)焊条焊机。发电机绕组是手工计算和缠绕的，曾经是发电机的外壳是露天喷砂处理的。丢失的零件是用角向磨光机从零开始组装的。“简单”是当今的主流。

【林果儿】经常称自己为“村姑”。不管她用什么专业工具，都是初级的。不管她用什么方法，都是手动的。你很快就会明白，[林果儿]不仅仅是一个有技能的人，而且是一个无论在什么情况下都有激情完成工作的人。[林果儿]是一个黑客，如果曾经有过的话！

如果水电黑客旋转你的水斗轮，给这个令人印象深刻的离网水力发电厂一个旋转。
