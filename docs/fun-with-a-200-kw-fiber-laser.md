# 200 千瓦光纤激光器带来的乐趣

> 原文：<https://hackaday.com/2019/12/02/fun-with-a-200-kw-fiber-laser/>

我们都听说过“不要用剩下的眼睛盯着激光”的笑话。这很有趣，因为这是真的，因为除非采取适当的预防措施，否则爱好者很容易得到的任何激光都会对眼睛造成永久性伤害。但是峰值功率为 200 千瓦的光纤激光器完全属于另一个危险等级。

诚然，像这样的超大功率额定值有点误导，因为它们是基于飞秒长脉冲的。可以肯定的是，在下面的视频中，[Marco Reps]拆除的光纤激光器在他得到它时就像小猫一样无害，这要归功于它的输出光学器件被它的前主人毫不客气地从放大器上剪掉了。重新连接输出和拼接光纤对于让激光再次发射是必要的，但是[Marco]心里有其他优先考虑的事情。他想了解光纤激光器的工作原理，但机箱内两个独立层面上的光纤缠结有点令人费解。缠绕在机箱内铝鼓上的纤维线圈被证明是放大器；由半导体种子激光器馈送的光脉冲穿过两级放大器的掺镱光纤，该光纤是受激发射并因此发生放大的有源增益介质。

通过一点逆向工程和在线手册的帮助，他能够理解激光器的操作。一家激光公司帮助他将光学元件拼接在一起——看到拼接装置的运行，光是门票的价格就值得了——该装置目前似乎或多或少处于工作状态。通常我们在这里看到的最强大的激光器是[的 CO [2] 激光器，那些廉价的中国激光切割机](https://hackaday.com/2018/09/27/laser-noob-getting-started-with-the-k40-laser/)，所以我们期待着了解更多关于光纤激光器的知识。
