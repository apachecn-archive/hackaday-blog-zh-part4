# 3D 打印的汽车轮胎轮辋终于上路了

> 原文：<https://hackaday.com/2022/03/07/the-3d-printed-car-tire-rim-finally-hits-the-road-sorta/>

当你想到“汽车轮圈”时，你可能会想到冲压钢或加工合金，带有坚固的鼓，以承受道路上的艰苦生活，而不是由 ABS 3D 打印出来的东西。那太疯狂了，对吧？不要对来自 *Proper Printing* 的【Jón Schone】说，他最近发布了一个关于他长期寻求为[的老式轿车配备挤压轮圈](https://www.youtube.com/watch?v=7lnvv-khE6Y)的更新。

有一些最初的尝试并不像希望的那样顺利。主要的问题是层分离，因为空气压力会拉伸碎片，形成气泡。他尽可能把厚度增加到绝对最大值。对刹车卡钳的快速 3D 扫描给他一个精确的模型，以确保他不会走得太远。他也无法使轮辋更大，以适应更大的车轮来清理卡钳，因为他已经从他改装的 Creality 打印机中最大限度地利用了他令人印象深刻的 420 毫米制造量。

一位有帮助的评论者建议使用一根螺纹杆贯穿印刷品，作为一种钢筋。最初不考虑这个想法，因为轮辋的厚度变得很薄，以适应卡钳，[Jón]意识到他可以弯曲杆，并以这种方式连接两半。拿着一张图纸，他切割并弯曲棒，将它们插入到新的印刷品中。这是一个令人印象深刻的大量的细丝，2.7 公斤的 ABS 仅用于轮辋的一半。

当他们给轮胎充气时，它没有爆炸，当他们在他们选择进行测试的小巷子里尽最大努力滥用它时，它也没有爆炸。从技术上讲，这款车不再合法上路，所以我们很欣赏他们在其他地方测试时的谨慎。在一个胜利但反气候的结局中，篮圈顶住了他们对它的所有辱骂。

[我们已经跟踪这个项目几个月了](https://hackaday.com/2021/01/03/abs-mercedes-rims-push-the-limits-of-3d-printing/)，很高兴看到[Jón]最终完成了这个项目。听起来似乎还有一些测试要做，但总的来说，我们会称这个实验是一个巨大的成功。
