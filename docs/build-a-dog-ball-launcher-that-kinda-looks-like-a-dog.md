# 制作一个有点像狗的狗球发射器

> 原文：<https://hackaday.com/2021/10/18/build-a-dog-ball-launcher-that-kinda-looks-like-a-dog/>

普通的花园狗玩捡球游戏的时间会远远超过普通人的耐力。如果你想让你的狗在不伤害你的投球手臂的情况下得到锻炼，[布兰克利的]自动狗球发射器[可能正是你所需要的。](https://www.youtube.com/watch?v=WglqSjnlZMQ)

这个设计很简单。3D 打印的外壳具有一个大漏斗，球可以落入其中。然后，伺服系统控制住球，同时一对滚轴由有刷 DC 电机带动旋转。两秒钟后，伺服系统将球释放到滚轮上，滚轮将球推出机器。一个树莓皮皮科运行显示，控制球发射的时间和改变电机速度，以改变每次发射球的距离。

那些渴望建立自己的文件的人可以在 Thingiverse 上找到这些文件。如果你很优秀，你甚至可以训练你的狗自己投球。[我们以前也见过类似的建造！](https://hackaday.com/2021/04/21/auto-ball-launcher-will-be-your-dogs-new-best-friend/)休息后的视频。
