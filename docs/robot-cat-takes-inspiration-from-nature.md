# 机器猫从大自然中汲取灵感

> 原文：<https://hackaday.com/2020/09/30/robot-cat-takes-inspiration-from-nature/>

通常，一个项目从一张白纸开始，我们开始疯狂地朝着我们脑海中的目标画草图。然而，先做你的研究是值得的，正如[陈亮]用这只伟大的机器猫展示的那样。

[梁]在对他们在网上看到的现有机器动物不满意后开始了这个项目。梁决定先研究一只真正的猫，以确保最终的机器人与它的生物灵感非常相似，而不是简单地试图根据记忆来制造一只猫。(梁)的重点是骨骼，因为复制真实骨骼的工作方式将创造出运动更真实的机器人。

使用 3D 打印部件和许多许多伺服系统，我们认为[梁]在创建一个基本的机器猫平台方面做了令人钦佩的工作。随着 ESP32 的运行，猫可以通过网络界面控制其各种关节的伺服位置。我们期待着未来的升级，使流体运动和其他能力，特别是涉及机载相机。

这不是我们看到的第一只机器猫，也不会是最后一只。如果你有一只生活在自己的实验室里，[给我们留言](http://hackaday.com/submit-a-tip)。休息后的视频。
