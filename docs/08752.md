# 变形无人机可以是正方形也可以是龙

> 原文：<https://hackaday.com/2021/01/01/transforming-drone-can-be-a-square-or-a-dragon/>

在建筑物内和周围飞行无人机时，无人机的大小通常受到你想要穿过的开口的限制。东京大学的研究人员通过为无人机框架使用一种[铰接结构来解决这个问题，](https://spectrum.ieee.org/automaton/robotics/drones/flying-dragon-robot-transforms-itself-to-squeeze-through-gaps)允许无人机从一个大正方形转变为一个窄而长的形状，以适应更小的间隙。

这架无人机被称为 DRAGON，不知何故是舌头扭曲描述“具有多自由度空中变换能力的双旋翼嵌入式多连杆机器人的缩写。无人机由四个部分组成，每个部分之间有一个 2 自由度的驱动关节。一对涵道风扇发动机连接到每个扇形段的中间，带有一个 2-DOF 万向架，允许它在相对于扇形段的任何方向上引导推力。对于正常飞行来说，分段将被布置成正方形，分段之间的运动最小。当遇到一个小间隙时，正如休息后的视频所示，这些片段重新排列成类似龙的形状，可以通过任何平面的间隙。

每个部分都有自己的电源和控制器，使所有部分协同工作所需的控制软件相当复杂。不幸的是,[全文研究论文](https://ieeexplore.ieee.org/document/8258850)在付费墙后面。小直径的螺旋桨和所有增加的部件在提升能力和飞行时间方面将是一个严重的限制因素，但这个概念肯定是有趣的。

变形机器人的想法已经存在了一段时间，当不同的部分可以分离并重新连接起来成为模块化机器人时，它会变得更加有趣。2016 年 Hackaday 大奖得主 [DTTO](https://hackaday.com/2016/07/03/hackaday-prize-entry-dtto-modular-robot/) 就是一个完美的例子，尽管它确实缺乏飞行能力。
