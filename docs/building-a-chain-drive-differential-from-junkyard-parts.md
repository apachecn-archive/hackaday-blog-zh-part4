# 用废旧零件制造链传动差速器

> 原文：<https://hackaday.com/2022/12/15/building-a-chain-drive-differential-from-junkyard-parts/>

差速器对车辆来说是非常有用的东西。它允许两个从动轮以不同的速度旋转，例如在转弯时。[车间从零开始]需要一个链条驱动的差速器，[所以着手用一个回收的汽车部件制造一个。](https://www.youtube.com/watch?v=3jvhlRzDtNk)

差速器本身取自宝马 E46 3 系列，具体来说是一款 2.0 升柴油车型。这项工作是从把差速器的中心齿轮从它巨大的铁外壳上拆下来开始的。然后进行拆卸，将星形齿轮从托架上拆下，并丢弃其他部件。差动齿轮本身被安装在一个新的紧凑的外壳中，通过大量的焊接和车床加工来制造。外壳装有一个大链轮来传递驱动力，取代了原来的差速器齿圈和小齿轮。

该视频的描述表明，这将是一个理想的卡丁车，童车，或其他此类小型车辆的差异。鉴于差速齿轮最初是为处理全尺寸汽车而制造的，它们应该能够更好地处理这种应用。

如果你对微分的工作原理不太熟悉，[看看这本 20 世纪早期的入门书](https://hackaday.com/2013/01/15/retrotechtacular-the-differential/)。这被广泛认为是这方面最好的教育。休息后的视频。
