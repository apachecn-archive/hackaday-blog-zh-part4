# 微:比特负责照看植物

> 原文:[https://hack aday . com/2020/12/15/microbit-put-on-plant-minding-duty/](https://hackaday.com/2020/12/15/microbit-put-on-plant-minding-duty/)

BBC 在支持学校技术教育方面有着悠久的历史。BBC Micro 向整整一代学生介绍了计算机，最近，Micro:bit 正在向今天的孩子们讲授嵌入式系统。[Michael Klements]碰巧是一个成年人，但他已经用这个小板子创建了一个自动植物浇水系统。

迈克尔的发明不是一个简单的基于计时器的系统，而是使用一个电容传感器来测量土壤湿度。这样做的好处是不需要像电阻传感器那样与土壤直接接触，因此传感器可以以最小化腐蚀的方式构建。Micro:bit 使用模拟输入读取该传感器，并使用其内置的 LED 矩阵以图表形式显示湿度水平。一旦水位下降到设定的阈值以下，水泵就会启动，向植物输送水，直到土壤再次达到合适的湿度。

这是一个简单的项目，但这将是一个很好的方式来教学生如何与泵和传感器接口，以及控制系统的基础知识。[迈克尔]还指出，进一步的工作可能涉及使用板载无线硬件连接多个微型比特。到目前为止，我们已经看到 Micro:bit 应用于从手持游戏到口香糖递送的方方面面。休息后的视频。
