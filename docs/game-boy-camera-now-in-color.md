# Game Boy 相机–现在是彩色的

> 原文：<https://hackaday.com/2018/12/22/game-boy-camera-now-in-color/>

Game Boy Camera 是 90 年代的一款传奇游戏硬件，尽管它根本不是一款游戏。它由一个低分辨率灰度相机组成，安装在 Game Boy 墨盒上，你可以用它来拍摄你的朋友，破坏他们的照片，然后在热敏打印机上打印出来。硬件之所以有趣，是因为它有局限性，而不是尽管有局限性。然而，[Matt]想知道是否有办法使用早期的摄影技术来获得彩色照片。

这种技术很简单——获得红色、绿色和蓝色滤镜，并拍摄三张照片——每张使用一种滤镜。然后，以数字方式组合照片，创建彩色图像。当然，将照片从 Game Boy 传输到 PC 需要一个非常复杂的过程。

有一些限制——由于 Game Boy 相机的速度，它最适合静态场景，因为拍摄需要几秒钟。此外，由于分辨率较低，最好选择色彩较宽的主题。尽管如此，[Matt]还是设法拍摄了一些具有丰富多彩而又复古的数码魅力的伟大图像。当然，还有其他方法可以实现这一点——[比如利用神经网络的力量来处理你的低分辨率游戏机图像。](https://hackaday.com/2017/02/20/neural-nets-and-game-boy-cameras/)休息后的视频。
