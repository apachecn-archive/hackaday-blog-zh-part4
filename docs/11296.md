# 成功众筹硬件:幕后一切

> 原文:[https://hack aday . com/2021/09/12/成功众筹-硬件-幕后一切/](https://hackaday.com/2021/09/12/successfully-crowdfunded-hardware-everything-behind-the-scenes/)

众筹硬件有其独特的挑战，[Uri Shaked]写了一篇精彩的报告，详细描述了他将众筹硬件项目变成现实的经历。

![A skull-shaped PCB with two red eyes](../Images/7fc812fdc6143924bda7a28aa4a0c147.png)【Uri】的项目是[头骨 CTF](https://www.crowdsupply.com/wokwi/the-skull-ctf/) ，一个 PCB 头骨形状的电子硬件拼图，他的幕后细节几乎涵盖了每个角度，从最初的概念到最后的总结，以及他在每个阶段的想法和反馈。他的项目达到了资金目标，完成了生产和运输，最终获得了成功。

[Uri]从一个工作项目开始，但除此之外，在众筹方面几乎完全是个新手。他最终决定利用大众供应来实现他的想法，他的文章详细解释了这一过程的每个阶段，包括金额。令人高兴的是，[Uri]不仅解释了所涉及的步骤和决定，还解释了每一步的研究，以及他对每一步的最终结果的感受。

整件事都值得一读，但[Uri]这样总结了硬件项目的众筹经验:这是测试创意需求和实现产品的绝佳方式，但要知道，除非项目取得巨大成功，否则在那个阶段它可能不会产生太多收入。这是一次很好的学习经历，但也涉及到他的大量时间和精力。

[Uri]真的很了解他的东西，考虑到他在[追捕讨厌的 bug](https://hackaday.com/2021/08/26/digging-into-an-attiny-simulator-bug-with-gdb/)的技能，这[不是他的第一个硬件难题](https://hackaday.com/2019/11/07/tiny-sao-tough-ctf-challenge/)可能就不足为奇了。
