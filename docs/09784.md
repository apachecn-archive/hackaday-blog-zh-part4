# 临时搭建的机器人手臂看起来像是工厂车间里的东西

> 原文:[https://hack aday . com/2021/04/10/scratch-build-robot-arm-looking-something-off-the-factory-floor/](https://hackaday.com/2021/04/10/scratch-built-robot-arm-looks-like-something-off-the-factory-floor/)

[杰里米·菲尔丁]对工业机器人手臂的力量和精度印象深刻。你在装配线上看到的焊接汽车的大臂本身就是工程壮举，这就是为什么他的飞跃——在家庭商店里建造一个——承诺是一次相当大的冒险，也是我们渴望跟随的一次冒险。

从下面的视频来看，[杰里米]的手臂已经基本完成，所以看起来他将发布视频，详细说明他是如何到达这个令人印象深刻的大而有力的手臂接管了他的商店的这么多地方的。他没有在这里瞎闹——这是一个由铝制成的七轴铰接臂，由交流伺服系统提供动力。[Jeremy]承认，一些结构部件仍然是 3D 打印的原型，他正在使用这些原型来完成设计，然后开始切割金属，这是一个明智的举动，因为他指出，他需要完成建造的大多数金属加工技能对他来说仍然相当新。它看起来仍然很神奇，我们期待着这个系列的其余部分，看看他是如何走到这一步的。

我们总是欣赏[杰里米]的热情和演讲风格，我们通常从他的视频中学到很多东西。无论是[数控台锯](https://hackaday.com/2020/09/08/measure-1024-times-cut-once/)、[自制测力计](https://hackaday.com/2020/03/02/this-diy-dynamometer-shows-just-what-a-motor-can-do/)，还是[超声波融化棒球](https://hackaday.com/2020/09/11/making-baseballs-go-supersonic/)，他的视频总是很好看。
