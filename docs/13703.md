# 把笔记本电脑扔到桌面上

> 原文：<https://hackaday.com/2022/05/23/ditch-the-laptop-for-the-tabletop/>

网络平台的想法很简单。一个相对便携的盒子，主要是一个键盘和一些屏幕。Cyberdecks 经常试图达到一种特定的审美或氛围，而不是专注于可用性或实用性。[卡特·赫德]后退一步，问自己什么样的电脑平台系统才是他可以每天使用的。

[Carter]的构建是一个原型，允许他尝试外形并将其用作日常驱动程序，因此做出了许多决定来加快构建并获得一些功能。例如，他没有花时间调整和打印他自己的键盘，而是使用他喜欢的现成键盘。虽然一个[框架主板](https://hackaday.com/2022/05/17/transparent-framedeck-is-clearly-capable/)对于这样的事情来说是完美的，但不幸的是，当【卡特】开始构建时，它们还不可用。所以[卡特]用一台二手游戏笔记本电脑来完成这项任务。他曾希望直接从主板驱动显示器，因为许多笔记本电脑内部使用嵌入式 DisplayPort。不幸的是，这不起作用，因为主板不支持他试图驱动的分辨率，所以他只是使用外部端口来驱动屏幕。一个 3d 打印的底座安装在键盘下方，以固定笔记本电脑主板，并为不太好用的部分(如 wifi 卡)提供一些延伸。机箱也有一个插槽，允许辅助显示器插入其中。

最终，它有点像现代的打字机，有点像网络平台。不管怎样，我们都喜欢。休息后的视频。
