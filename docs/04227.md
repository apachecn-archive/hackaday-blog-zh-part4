# 触觉时钟让你晚上闭上眼睛

> 原文:[https://hack aday . com/2019/09/27/haptic-clock-let-you-keep-you-eyes-shut-in-night/](https://hackaday.com/2019/09/27/haptic-clock-lets-you-keep-your-eyes-shut-at-night/)

想象一下:半夜你躺在床上，想知道几点了。床头闹钟已经成为过去，现在你依靠你的智能手机来告诉时间。只是，如果你打开屏幕，你会发现在黑暗中看它就等于在没有保护眼睛的情况下盯着太阳看。[迈克尔]描绘了同样的事情，他对此场景的解决方案是[一个聪明的触觉反馈时钟](https://hackaday.io/project/163588-no-eye-haptic-clock)。

背后的想法很简单，一个不用眼睛就能知道时间的钟。这个给了你两个选择，第一个是一系列触觉脉冲，让你只需触摸设备就能知道时间。第二个是通过存储在闪存芯片中的声音样本来可听地告诉时间，这是在[迈克尔]继续完善他的设计时添加的。除了帮助我们评估黑暗中的时间，值得注意的是这对那些有视觉障碍的人也很有用。

在我们看到最终产品之前，你可以帮助他查看设计，并在项目的 [GitHub 页面](https://github.com/Aliasmk/NoEye-Haptic-Clock)上发送 pull 请求，或者只是在 Hackaday.io 页面上观察他的进展。我们以前见过一些有趣的方式来判断时间，从俄罗斯方块游戏[到装在老式相机闪光灯外壳内的时钟](https://hackaday.com/2019/06/13/a-tetris-clock/)，但我们以前从未见过使用触觉反馈的方式。为了我们的眼睛，我们希望它能流行起来！
