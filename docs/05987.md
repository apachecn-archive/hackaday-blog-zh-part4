# 听到植物咯咯的笑声就像你想象的那样令人毛骨悚然

> 原文：<https://hackaday.com/2020/03/18/hearing-plants-giggle-is-just-as-creepy-as-you-think/>

虽然罗尔德·达尔因《查理和巧克力工厂》而闻名，但他在那个时代也写了不少类似的古怪故事。其中一个叫做*的声音机器*，是关于一种允许用户听到树木被砍伐时痛苦的尖叫声的设备。对我们来说听起来有点奇怪，但是[【罗尼·班迪尼】非常喜欢这个想法，他决定建立自己的版本](https://medium.com/@Bandini/la-m%C3%A1quina-del-sonido-de-roald-dahl-85ece09b1cd6)。

现在公平地说，这个装置并没有*只有*在痛苦中尖叫。事实上，大多数时候它应该发出笑声和快乐的声音。该设备使用一个安装在花盆土壤中的湿度传感器，通过这些声音提示来告诉你多叶朋友的相对健康状况。所以假设你有一点园艺技能，事情应该没问题。

但是一旦土壤变得过于干燥，设备确定植物处于“痛苦”中，事情就会变得更糟。我们认为从技术上来说，它不会像僵尸一样发出咕噜声，但这仍然不是我们在晚上穿过房子时想听到的声音。幸运的是，似乎你需要点击 3D 打印外壳前面的按钮，才能让它播放来自其 DFPlayer 模块的适当音轨。

就我个人而言，我们更愿意建造一些东西来确保植物得到自动照顾，而不是一个痛苦地呼喊来提醒我们不知道自己在做什么的小玩意。但是，每个人都有自己的灵感。
