# 透明有机发光二极管用小米的米电视 LUX 透明版冲击市场

> 原文：<https://hackaday.com/2020/08/21/transparent-oled-hitting-the-market-with-xiaomis-mi-tv-lux-transparent-edition/>

与 LCD 面板相比，有机发光二极管的一个主要优势是前者可以使用更少的层，因为像素本身发光，而不是控制背光的光。这让一些人提出了这样一个问题:是否有可能制造出透明或至少半透明的 OLED 面板。正如[小米新推出的小米电视 LUX 有机发光二极管透明版](https://www.gsmarena.com/xiaomi_announces_mi_tv_lux_oled_transparent_edition_the_worlds_first_massproduced_transparent_tv-news-44704.php)所示，答案是[响亮的‘是’](https://blog.mi.com/en/2020/08/11/turning-moment-for-tv-industry-xiaomi-unveils-mi-tv-lux-oled-transparent-edition/)。更好的是，以大约 7200 美元的低低价格，你可以拥有这些 55 英寸的奇迹之一。

当然，透明有机发光二极管技术并不新鲜。早在 2018 年，LG 展示了一款使用早期透明有机发光二极管面板的原型电视。在插播的视频中，来自 Linus Tech Tips 的[Linus]在韩国 LG 亲自动手操作 LG 原型机，同时还包括了对该面板背后的一名工程师的采访中的一些重要细节。

事实证明，仅仅去掉 OLED 面板上不透明的背衬并不足以让它变得透明。为了使 OLED 面板变得透明，像素层和 TFT 层中的电路需要尽可能地对齐，以允许在显示器上打出许多许多小孔。

看看[Linus]在 LG 原型方面的经验，这种技术确实非常适合用于标牌目的，同时也允许在房间里放置隐形电视或显示器，可以放在绘画或其他装饰的前面。一旦显示图像，屏幕足够亮，你可以舒服地辨认出图像。只是不要在电视后面放任何强光灯。
