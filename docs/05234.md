# 卡特彼勒餐厅现在由新管理层管理

> 原文：<https://hackaday.com/2019/12/30/cat-diner-now-under-new-management/>

这些故事大多以一只猫在早晨某个猥亵的时刻站在某人的胸口上乞讨食物开始。但不是这个。猫查兹有糖尿病，他需要在早餐时补充胰岛素。问题是 Chaz 喜欢通宵吃饭，这破坏了他的早餐胃口和正常代谢胰岛素的机会。[Becky]试着在睡觉前把碗放好，但是让我们面对它——解决一次问题比每天晚上解决同样的问题更有趣。

[Becky]的解决方案是设计并打印一个带盖子的碗架，并在猫餐厅关闭时使用一个小伺服系统和 NodeMCU 盖住碗。它看起来不错，而且用很少的组件就能完成任务。查兹得到了他的胰岛素，[贝基]得到了内心的平静，每个人都很高兴。这并不适用于所有的猫，因为安全措施相当松懈。但是 Chaz 是一只年长的小猫，因此对扒盖子看看会发生什么不感兴趣。抓紧休息时间，观看[Becky]的构建/演示视频，其中包含大量的卡特彼勒税收报道。

我们在这一带已经看到了很多喂猫的装置，但是很少能像这样解决一个特定的问题。[如果你想要 overengineering 和 cat metrics，就来拿吧](https://hackaday.com/2019/12/05/automated-cat-feeder-leaves-little-to-chance/)。
