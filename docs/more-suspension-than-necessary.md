# 不必要的暂停

> 原文：<https://hackaday.com/2018/08/06/more-suspension-than-necessary/>

传统山地车的三角形车架需要是最刚性的结构，三角形可以是非常坚固的形状。所以[【柯林·福尔泽】抛出了一个扳手](https://www.youtube.com/watch?v=N39uwTykTQk)，或者，在这种情况下，一束弹簧。视频在休息时间下面，但是请试着想象你在一个聚会上，盯着一些美味的沙拉，然而有人递给你的不是玉米片，而是一大块明胶甜点。自行车有点像那个。

任何购买过弹簧的人都知道有很多选项和术语，比如牛顿米力、拉伸、压缩和[屈曲](https://hackaday.com/2016/12/03/do-you-miss-the-sound-of-your-model-m/)。弹簧有一个学习曲线，所以一个简单的陈述，例如“我想做一辆弹簧自行车”，没有任何简单的答案。这很像说，“我想用晶体管制造一个[微处理器”。这个项目从直径大约相当于旧自行车内胎的弹簧开始，这是一个巨大的失败。试着用煮熟的意大利面条做一座桥。](https://hackaday.com/2016/07/06/42300-transistor-megaprocessor-is-complete/)

第一组定制的[弹簧](https://hackaday.com/2018/04/13/mechanisms-the-spring/)不能胜任这项任务，但是第三轮产生了一些可以骑的东西。结果似乎是一个荒谬的方式来锻炼你的腹肌，大约是一个训练独轮车与船锚交配。

这算什么黑客？这个视频和[科林]制作的任何东西一样有趣，但这并不意味着它本身就是一个黑客。问题是，有人问了一个荒谬的问题，可能是在酒精的作用下，答案是通过建造这个愚蠢的东西得到的。从周日开始，一辆弹簧自行车可以在一部旧的 Android 手机上模拟六种方式，但这种冒险值得在现实生活中这样做。
