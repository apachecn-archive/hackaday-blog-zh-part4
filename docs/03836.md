# 瓶状器官分解

> 原文:[https://hackaday.com/2019/12/14/bottle-organ-breakdown/](https://hackaday.com/2019/12/14/bottle-organ-breakdown/)

一位目光敏锐的评论者向我们展示了一种像钢琴一样演奏的自制管风琴。脚踏风箱和定制钥匙增加了复杂性，但[迈克]和[西蒙·海塞尔]的车库自制乐器的魔力丝毫未减。我们在休息时间下面还有一个视频，是由[Coyote Merlot] 演奏的[管风琴。](https://www.youtube.com/watch?v=s1HOiEoOTww)

视频中很好地解释了工作原理，从风箱开始。在视频的前几秒钟，我们看到一名管风琴手在演奏时摇摆，可以准确地说是音乐打动了他。摇晃是踩几个杠杆，挤压一对气囊，在轮子下滑动，看起来像是在五金店买的。弹簧复位机制是一个重新利用的橡皮筋，你知道我们挖掘这种足智多谋。每一个波纹管阀门都是由传统的皮片制成，这种皮片在橡皮筋和照相手机出现之前就有了。这些气泵给后面的一个大储气罐充气，为歧管提供持续的压力，其中 36 个键中的每个键控制一个负责一个瓶子的阀门。这两个人建造了我们提到的每一个木制部件，目的都是为了制造这个器官。

一个黄铜的 fipple， [那是一个真正的单词](https://hackaday.com/2019/08/06/forming-fipples-and-accompanying-accoutrements/) ，吹过每一个玻璃瓶子，虽然它们展示了一个木制版本的喷嘴。这些管子安装在一个倾斜的架子上，将空气精确地导向瓶口。融化的蜡给瓶子调音，但我们的第一个想法与他们的第一个想法一致，即[将容器装满水](https://hackaday.com/2018/07/31/the-precise-science-of-whacking-a-wine-glass/)，但这些东西会蒸发，所以我们将把调音宝藏放在我们的工具箱中。
