# 远程视频:有限元分析的力学

> 原文:[https://hack aday . com/2020/12/31/remoticon-video-the-mechanics-of-finite-element-analysis/](https://hackaday.com/2020/12/31/remoticon-video-the-mechanics-of-finite-element-analysis/)

硬件黑客可能涉及多个学科。如果你只知道比特和字节，而不知道焊料和电子，那么你能制造的东西就有限了。机械设计也是如此，压力和张力突然施加到你的项目上，一堆代码和 PCB 轰然倒地。

[![](../Images/5f5038ced68b32aadb46ad9446bd822f.png)](https://hackaday.com/wp-content/uploads/2020/12/fea2_featured.png) 在工作坊的前半部分，纳曼·普什普[带你了解机械工程](https://youtu.be/Y8gWBflyxwU)中一些重要的基本概念——如何*思考*世界上作用于物理物体的力。他还带来了一系列自制的 [Jugaad](https://en.wikipedia.org/wiki/Jugaad) 道具，包括一个抗重力的张拉整体弦雕塑和一些花哨的折纸，有助于深入了解主题。

在研讨会的后半部分，Naman 将这些概念引入计算机模拟，并让我们深入了解[有限元分析](https://en.wikipedia.org/wiki/Finite_element_method)模拟包如何在项目的微小几何块上模拟这些相同的力，以查看它是否能承受真实世界的负载。他使用的软件从任何意义上来说都不是免费的——除非你有学生许可证，否则它甚至不便宜——但尽管如此，观看他粗略设计一个物体，对其施加模拟压力和张力，并解释结果的流程还是很有启发性的。如果你以前从未使用过有限元分析工具，或者正在寻找第一学期机械工程的压缩介绍，这个演讲可能正合你的胃口。
