# 遥控割草机让你这边的草坪更绿

> 原文：<https://hackaday.com/2020/06/01/rc-lawn-mower-keeps-the-grass-greener-on-your-side-of-the-fence/>

对于一些人来说，修剪草坪是一件可怕的家务，这导致他们想到在院子里浇一块水泥板，然后把它漆成绿色。其他人认为这是在户外度过一个阳光明媚的下午的最佳时机。还有一些人一开始就没有对这个主题的偏好。[elliotmade]例如，他有一个坐在轮椅上的朋友，通常需要依靠他人来维护他的草坪，并对这项任务的愉快性形成看法。所以为了保持他朋友的独立性，他决定给他造一台遥控割草机。

在对已经成功使用了几年的概念进行初步验证后，[elliotmaded]发现了一些改进的空间，并认为是时候进行升级了。他解放了电动轮椅的驱动部分，在它周围焊接了一个框架来容纳电池和割草机本身，并添加了一个交流发电机，直接从割草机的发动机为电池充电。一个连接到电机驱动器的 RC 接收器由 Arduino 控制，还有一对继电器来切换点火和一个无需拉绳的电动启动器。除此之外，还有一个摄像头，花园里的琐事现在可以从远处轻松处理，没有任何深度感知的问题。

远程控制一台锋利的机器肯定需要一些额外的安全考虑，似乎[elliotmaded]考虑得很好，所以任何相关部件的故障都不会有致命的后果。然而，从休息后嵌入的演示视频来看，有问题的花园可能不是未来将它变成 GPS 辅助的自主割草机的最佳环境。不过话说回来，遥控车辆本身就很有趣，[不管它们的形状或大小](https://hackaday.com/2018/03/16/go-big-or-go-home-this-arduino-rc-car-can-take-you-there/)。
