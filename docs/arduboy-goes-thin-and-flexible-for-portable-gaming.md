# Arduboy 轻薄灵活，适合便携式游戏

> 原文：<https://hackaday.com/2019/04/18/arduboy-goes-thin-and-flexible-for-portable-gaming/>

我们的口袋或钱包里都有一个游戏系统，我们中的一些人可能正在用它阅读。那个口袋空间是很有价值的，所以我们必须对我们放在那里的东西进行预算，如果它占用太多空间，增加另一个游戏系统是不可能的。[凯文·贝茨]为他的全尺寸阿都博伊克隆体 Arduflexboy 做了最小的口袋不动产预算。它很薄，符合他的口袋，因为定制的 PCB 使用了柔性基板，他已经取消了传统的触觉按钮。

弹性系统不会很难玩吗？是的。[Kevin]自己说的，虽然我们不反对，但柔性电路上的功能性 Arduboy 通过整洁的制造演示弥补了实用性。这属于“因为我能”的范畴，但其中的思想也是显而易见的。所有组件都安装在屏幕的对面，因此从前面看起来很干净，组件不会受到太多弯曲，输入与传统的 Arduboy 相同。

> 成本=低，实用性=极低，客户服务问题=高
> 
> ~[凯文·贝茨]

这些柔性电路板使用聚酰亚胺基板，与 Kapton 胶带的材料相同，订购电路板越来越便宜，因此我们可以期待看到更多的电路板出现。我们提到过我们目前有一个柔性电路的[竞赛吗？我们有奖品可以让你](https://hackaday.io/contest/163267-flexible-pcb-contest)[唱歌](https://hackaday.com/2018/09/21/the-diaphragm-is-the-coil-in-these-flexible-pcb-speakers/)，仅仅因为你发表了你的柔性印刷电路板概念。
