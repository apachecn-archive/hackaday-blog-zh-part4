# 给你的工作空间上色，让你的头脑平静

> 原文:[https://hack aday . com/2020/07/07/color-your-workspace-calm-your-mind/](https://hackaday.com/2020/07/07/color-your-workspace-calm-your-mind/)

每天，放松和放松似乎变得越来越难。一个人在开始占用工作时间之前只能洗这么多的热水澡。听音乐是一个很好的选择，但是很难找到既能让人放松又不会造成干扰的东西。那么我们还能尝试什么呢？哦，是的，闪光灯。磨砂的、发光的闪光灯给房间增添了色彩。Ahhhh 。

[![](../Images/63c4a0fd7e7b8d6ee08a33325f50a9ed.png)](https://hackaday.com/wp-content/uploads/2020/07/LED-wall-boxen-guts.jpg) 这些围栏很好地完成了这些。[ChrisParkerTech]使用喷有透明涂层的桤木，呈现出令人愉悦的干净的中世纪外观。我们还挖掘了天花板和未完成的顶部边缘的缺失，因为它给泄漏的光一点无限泳池的神秘感。

当然，如果你每次想调整它们的时候都不得不从沙发、椅子或豆包上站起来，这些就不是什么放松的工具了。每个长条都连接到一个 Wemos D1 迷你，所以[克里斯]可以通过 WLED 用他的手机控制它们，或者让 Alexa 来做。休息之后，请观看构建视频。

如果你真的喜欢 led，就不要离开家。[用一顶时尚的 LED 帽子向世界展示你的感受](https://hackaday.com/2020/06/10/light-up-the-night-with-a-tetrahedral-led-hat/)。
