# 机械七段显示器真正脱颖而出

> 原文：<https://hackaday.com/2020/07/27/mechanical-seven-segment-display-really-sticks-out-from-the-pack/>

我们使用分段显示器显示数字已经有近 120 年的历史了，这项发明比通常为无处不在的设备供电的 led 早了六七十年左右。但是 led 远不是运行七段显示器的唯一方式——看看这个机械七段显示器就能证明这一点。

最近我们已经看到了很多机械七段显示器，当我们第一次发现[indoorgeek]的构建时，我们认为它会是常见的“flip-dot”机制的变体。但是这个不一样。为了形成每个数字，必要的段从显示器的表面稍微突出。所有东西都是由白色细丝 3D 打印而成，收缩时看起来很干净，但伸展时会投射出锐利的阴影。每个片段的背面都有一个小磁铁，它紧贴着定制缠绕电磁铁的钢芯，通电后会排斥磁铁并延伸片段。我们认为它肯定会很吵，但下面的视频显示它真的很安静。

虽然我们喜欢显示器的微妙对比度，但对于一些用户来说这可能还不够，尤其是在侧光不切实际的情况下。在这种情况下，他们可能会想看看[这个更早的类似展示](https://hackaday.com/2012/04/15/a-glorious-mechanical-seven-segment-display/)，并尝试在每个片段的侧面使用对比色。
