# 一个部件很少的简单太阳追踪器

> 原文：<https://hackaday.com/2021/07/25/a-simple-sun-tracker-with-very-few-parts/>

如果你有理由的话，有很多方法可以追踪太阳。你可以使用基于时间的算法，或者从互联网上输入坐标，或者你可以用最少的部件，完全不需要电子处理。[后者就是这个来自【3D 打印机学院】的项目是如何运作的。](https://www.youtube.com/watch?v=Oj1E7o7J3qc)

这个项目的一个关键是，你不应该被太阳能电池板愚弄。他们不是来发电供外部使用的。相反，它们以相反的极性连接到 DC 齿轮马达上。电机转动面板组件。当一个电池板被阳光照射时，只需向电机施加 DC 电压，它就会转动组件，将另一个电池板也带到阳光下。另一个面板以相反的方式连接，所以如果它在阳光下，它也会使另一个面板对齐。

这是一个非常简单的平面太阳能跟踪器。如果你想用最少的部件跟踪太阳，这是一个非常简单的方法。你只需要把你想瞄准的任何东西放在组件的顶部。如果碰巧是一个更大的太阳能电池板，它可能会很笨重，另一个更复杂的设计可能更合适。

这是一种追踪太阳的巧妙而简单的方法，尽管目前还不清楚这种设备在目前的形式下会有什么用处。如果你有如何使用这种机制的想法，请在评论中告诉我们。
