# 一个低成本的虚拟现实耳机

> 原文:[https://hackaday.com/2019/01/03/a-low-cost-vr-headset/](https://hackaday.com/2019/01/03/a-low-cost-vr-headset/)

几十年来，虚拟现实系统一直处于发展的前沿。虽然现在有商业产品，但回到系统更加受限的时代还是很有意思的。[科林·Ord][最近完成了他自己的虚拟现实系统](https://sites.google.com/site/colinord/Home/3d-virtual-reality-hmd-and-controller-project)，以 20-30 年前的可用系统为模型，让我们看一看那些系统会是什么样子，以及使用今天的技术以非常低的成本建造。

这个项目的核心是一个头部跟踪器，它使用了两个 BBC 微型机器人，因为它们都有实现项目目标所需的加速度计和指南针。它还能够跟踪虚拟空间中的项目及其位置。为了这个项目，[科林]自己做了一切，包括电子设备和编程。它还利用谷歌纸板将屏幕、镜头和传感器都放在耳机中。所有这些都降低了成本，不像几年前首次推出的类似系统。

这个项目采取的脚踏实地的方法确实值得称赞。希望我们可以看到代码发布，其他人可以在这个优秀的工作基础上。你甚至可以用它来进行一次虚拟现实的英国自行车之旅。
