# 用分立元件构建视频乒乓

> 原文:[https://hack aday . com/2019/09/17/building-video-pong-with-discrete-components/](https://hackaday.com/2019/09/17/building-video-pong-with-discrete-components/)

Pong 是视频游戏时代初期的经典之作。对于那些编码新手来说，重新创建它仍然是一个流行的练习。然而，它简单的逻辑使这个游戏特别适合全硬件构建；[格伦沉着应对的事情。](http://www.glensstuff.com/videopong/videopong.htm)

不满足于简单的解决方法，[格伦]选择了一种特别硬核的建造方法。这款游戏在构造上完全不使用集成电路。相反，它依赖于 431 个双极晶体管、6 个 JFETs 和 826 个二极管的服务。所有的东西都被绑在原板上，用整齐有序的彩色电线连接起来。[示意图可供好奇的](http://www.glensstuff.com/videopong/video_pong.pdf)参考。

这也是一个功能齐全的构建。视频输出是彩色的，分数显示在屏幕的顶部，甚至还有音响效果的立体声平移。它只是展示了一些不起眼的组件在以正确的方式投入工作时可以做些什么。我们以前也看过[Glen]的一些作品，例如这个运放弹跳球装置。休息后的视频。
