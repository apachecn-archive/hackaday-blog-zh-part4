# Karakuri Kaizen:工厂车间的黑客

> 原文：<https://hackaday.com/2019/01/04/karakuri-kaizen-hacks-for-the-factory-floor/>

任何对制造业感兴趣和/或有职业生涯的人都会听说过 *Kaizen* ，这通常是一个在任何地方持续改进流程的概念。在那把巨大的保护伞下是 *Karakuri Kaizen* ，鼓励工厂车间的工人采取黑客心态，改善他们自己的工作站。不管是不是制造商，这都是我们的拿手好戏，[让《汽车新闻》](https://www.autonews.com/manufacturing/assembly-tricks-give-japan-inc-edge)*的这个概述成为一个有趣的读物。*

 *Karakuri 可以翻译成“机械”,但在英语单词 gadgets、gizmos 或我们敢说它:hacks 的脉络中隐含着某种新奇的东西。*卡拉库里*的历史可以追溯到几个世纪前的发条自动装置，一直到现代的鲁布·戈德堡装置。当应用于现代制造业时(作为[工厂培训](https://www.autonews.com/manufacturing/toyota-workers-get-crash-course-karakuri)的一部分)，它鼓励每个人设计简单的改进。每种方法可能只节省了几秒钟的组装时间，但节省的时间会随着时间的推移而增加。

现代全球制造业竞争非常激烈，生存需要比竞争对手更高效地生产。虽然关注的焦点可能集中在技术、自动化和“外星无畏舰”的建造上，但这种关注可能会忽略在更小和更简单的规模上发现的收益。 *Kaizen* 意味着不断寻求改进，而答案并不总是更多的技术。

通过比较价格标签，这些文章中的几个观点断言纯机械 *karakuri* 比自动化解决方案便宜得多，这些价格标签显然是针对工业自动化设备的。我们很想知道我们最喜欢的低成本工具——AVR、PIC、ESP32 和 friends——是否会在这一领域取得进展。我们当然看到过小规模生产[的黑客](https://hackaday.com/2017/04/21/automated-parts-counter-helps-build-a-small-business/)。
