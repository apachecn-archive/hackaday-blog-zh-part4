# 廉价范围故障排除准将

> 原文：<https://hackaday.com/2022/11/08/cheap-scope-troubleshoots-commodore/>

[Adrian]有一台 Commodore 计算机需要修理，他决定[看看他最新的微型便携式望远镜如何工作](https://www.youtube.com/watch?v=Uqrel5fQpK4)。他花了 57 美元买了这个小小的测试仪器，尽管现在的价格似乎更高。它声称拥有 120MHz 带宽和每秒 500 兆样本。有几个不同规格的版本，但我们确实发现了一个不到 60 美元的类似设备。你可以在下面的视频中看到拆箱和它是如何工作的。

当然，这类仪器经常夸大其规格，而且[阿德里安]也很怀疑。该设备的一个奇怪的功能是它可以将其输出回声到 NTSC 视频输出，这样你就可以将屏幕发送到外部显示器。

如果你想跳过范围拆箱，前进约 19 分钟，看看里面的准将 64。这个范围足以扫描电脑中的芯片，并显示出一个可疑的地址线。该线路连接到一个 PLA 和一个 mux 芯片，这两个芯片都不在插座中。他将 PLA 从电路中取出，地址线开始看起来正常。所以结论是解放军已经死了。

之后就直接把芯片拆下来换了。好吧，从技术上来说，换成插座，方便以后维修。57 美元的示波器会取代你的大型台式仪器吗？也许不是。但它是一个有用的故障排除工具。

即使你不想要便宜的示波器，如果你面临任何数字维修项目，你也可以从[Adrian]周到的故障排除和分析中学到很多东西。我们确实喜欢这里便宜的望远镜。与仅仅几年前的相比，现在 100 美元能买多少 scope 是令人惊讶的。
