# 将你的目光传递到这款光纤 LED 灯上

> 原文：<https://hackaday.com/2021/02/03/transmit-your-gaze-to-this-fiber-optic-led-lamp/>

我们很容易被逗乐，但我们认为用一个微控制器、一些 RGB LEDs 和一点点塑料可以做的事情非常惊人。一个很好的例子就是【andrei.erdei】的[漂亮又非常平易近人的光纤 LED 灯](https://www.instructables.com/Fiber-Optic-LED-Lamp/)。这是一个桌面友好版本的类似作品，大约是这个的九倍大，挂在墙上。构建可能很简单，但产品非常可爱。

我们真的很喜欢这盏灯的视觉密度——它的灯管数量恰到好处，在过于稀疏和过于混乱之间取得了平衡。正如你所料，它包含一个 Arduino 和一些 RGB LED 灯条。但这种构造的关键在于 16 根侧面发光的塑料光纤管。侧面发光的设计是为了让光线沿着灯管的长度逃逸，而端面发光则是为了像数据管道一样，最大限度地减少从一端到另一端的光线损失。这允许各种有趣的效果，休息后，您可以在演示视频中缓慢而舒缓地观看[andrei.erdei]通过不同的颜色和模式。确保你看了足够长的时间，看到管子像旧的 Windows 3D 管道屏幕保护程序一样移动
