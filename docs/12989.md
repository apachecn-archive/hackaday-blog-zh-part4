# 升级的 3D 打印坦克获得更好的传动系统和控制

> 原文:[https://hack aday . com/2022/03/09/upgraded-3d-printed-tank-gets-better-driving-and-controls/](https://hackaday.com/2022/03/09/upgraded-3d-printed-tank-gets-better-drivetrain-and-controls/)

就 3D 打印构建而言，由于大多数打印机的构建体积较小，它们的尺寸通常有限。然而，[伊万·米兰达]总是以他的体型变大，他最新的坦克设计表明了这一点。

![](../Images/c4a0de9bb918993528da4e649737524a.png)

Looks comfy!

[Ivan]多年来一直致力于打印人体大小的坦克，他的最新版本旨在解决过去阻碍其性能的许多问题。皮带传动是第一次重大升级，旨在提高传动系统的可靠性，这是过去的一个痛点。马达支架这次也由铝制成，以帮助保持冷却，因为熔化以前是一个潜在的问题。

坦克的控制也进行了升级，这一次使用一个简单的踏板系统来控制无刷电机，以方便驾驶。甚至还有一个更符合人体工程学的印花座椅。结果是一个大得足以容纳一个成年人的巨型坦克，另外，它现在很容易驾驶，不再需要[伊万]躺在里面适应。

[【Ivan 的】大型打印机是他在大型建筑上取得成功的关键](https://hackaday.com/2019/12/18/giant-3d-printer-for-giant-projects/)。坦克的一个新零件重达 5.8 公斤，仅用 2.5 天就打印出来了！我们迫不及待地想看看他下一步会造出什么巨大的东西。据说坦克也会有炮塔。休息后的视频。
