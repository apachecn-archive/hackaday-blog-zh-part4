# 是突袭。用软驱。

> 原文：<https://hackaday.com/2022/06/30/its-raid-with-floppy-drives/>

有些事情应该是可能的，所以只有*有*可以尝试。【动作复古】有一个很棒的视频展示了这样一个恶作剧，[用一堆 USB 软驱](https://www.youtube.com/watch?v=1hc52_PWeU8)创建一个大型 RAID 0 阵列。是的，采用最小和最不可靠的数据存储介质之一，将它们组合在一起，这样，如果其中一个出现故障，所有数据都会丢失。

令人惊讶的是，创作过程比我们预期的更快更简单，一个稍显老态的 Mac OS X 版本完成了这个过程。从 30 个 USB 软盘和一对大型 USB 集线器开始，他将这一堆减少到 13 个驱动器，这些驱动器可以很好地一起运行和 RAID。当宝贵的兆字节充满数据时，如此多的驱动器一起点亮的景象在软盘复制机领域之外可能不会出现，这使我们这些在过去几年中从事消费软件业务的人想起了不好的回忆。

你会这么做吗？可能吧，但是你应该这么做吗？当然不是，但话说回来，他已经这么做了，所以我们其他人就不必这么做了。在 2022 年，也许[一对软驱会有更好的用途](https://hackaday.com/2022/06/14/behold-the-mighty-floppotron-3-0/)。
