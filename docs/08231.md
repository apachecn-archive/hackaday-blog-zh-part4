# 大众资助的跳跃立方体

> 原文:[https://hackaday.com/2020/10/30/crowd-funded-jumping-cubes/](https://hackaday.com/2020/10/30/crowd-funded-jumping-cubes/)

日本宇宙航空研究开发机构( [JAXA](https://en.wikipedia.org/wiki/JAXA) )最近将他们的 [Int-Ball](https://en.wikipedia.org/wiki/Int-Ball) 技术贡献给了由日本电子产品制造商/经销商 [Bit Trade One](https://bit-trade-one.co.jp/ourcompany/) (日本网站)运营的 [Kickstarter 活动](https://www.kickstarter.com/projects/bittradeone/3axis-attitude-control-module-3/description)。这项技术基于位于苏黎世的瑞士联邦理工学院(ETH Zurich)的 [Cubli 项目](https://ethz.ch/content/dam/ethz/special-interest/mavt/dynamic-systems-n-control/idsc-dam/Research_DAndrea/Cubli/Cubli_IROS2012.pdf)，我们在 2013 年曾报道过该项目[。从那以后，基于 Cubli 的技术一直出现在各种项目中，包括 2016 年的](https://hackaday.com/2013/12/19/a-balancing-walking-cube-named-cubli/)[非线性机电立方体](https://hackaday.com/2016/03/25/resistance-is-futile-balancing-cubes-are-taking-over/)。唉，目前位于 JAXA 的“三轴姿态控制模块”项目还没有一个吸引人的名字。

这些跳跃立方体的一个有趣的应用，大概是 JAXA 如何参与这些设备的，是一个[浮动摄像机](https://www.youtube.com/watch?v=ojFNo19HYSo)，它于 2017 年在国际空间站(ISS)上使用。Kickstarter campaign 提供的版本不包括摄像头，您需要提供自己的无重力环境来复制该应用程序。相反，他们似乎是为了教育用途来推销这个。如果你想要一个，你最好从你的钱包里掏钱——一个完全组装的单元需要超过 5000 美元的承诺(有一个“需要一些组装”的套件可以为你节省大约 1000 美元)。我们中的大多数人不会仅仅因为这个原因而支持这个项目，但很高兴看到这样一项酷技术的进步:从开始到空间应用，再到变得对公众可用。感谢[Lincoln Uehara]发送此提示。
