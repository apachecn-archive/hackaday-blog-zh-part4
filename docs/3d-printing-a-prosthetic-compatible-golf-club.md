# 3D 打印假肢兼容的高尔夫球杆

> 原文：<https://hackaday.com/2022/09/02/3d-printing-a-prosthetic-compatible-golf-club/>

对于截肢者来说，重新学习一项旧的运动，或者开始一项新的运动，都是一项挑战。除了明显的物理方面，定制的假肢或适配器可能需要制作和安装，这可能非常昂贵。凭借 3D 打印和一些机械加工的力量，[Ian Davis]能够建造一个定制的假肢高尔夫球杆，让一名四腿截肢者重返果岭。

这个假肢的接受者失去了手腕以上的双手，所以[Ian]必须想出一个可以握住球杆并在整个挥杆过程中模仿手腕运动的机制。他能够用每个手臂一个简单的四板铰链来实现这个动作。为了实现最佳的人体工程学设计，[Ian]还增加了双轴调节功能，每轴只需拧松一个螺栓。可以使用标准的高尔夫球杆，并将其夹在印刷的支架中。

机械加工的假肢接受腔被用于允许快速连接到用户现有的假肢前臂。理论上，这也应该允许他在没有过多麻烦的情况下更换俱乐部。伊恩自己也是一名截肢者，他利用自己的工程技能制造了一系列的[假手](https://hackaday.com/2020/10/22/what-if-you-could-design-your-own-aluminum-hand/)，甚至还有一个[定制控制器模块](https://hackaday.com/2020/12/15/console-controller-mod-gets-amputee-back-in-the-game/)，以便用更少的手指重新玩游戏。
