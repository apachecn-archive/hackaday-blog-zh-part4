# 软件定义无线电学院走向虚拟化

> 原文:[https://hack aday . com/2020/07/08/软件定义-无线电-学院-goes-虚拟/](https://hackaday.com/2020/07/08/software-defined-radio-academy-goes-virtual/)

他们说乌云背后总有一线光明。很难在当前全球疫情的所有坏消息中找到积极的一面，但它推动了更多的会议和活动，允许在线参与，无论是现场还是事后。一个典型的例子是:[软件定义无线电学院的年度活动](https://www.youtube.com/channel/UC1GAlgAQrkjeeLmIkCB8pgQ/videos)都在 YouTube 频道上，所以你可以虚拟地参加。

并非所有视频都在这里，但主题演讲以及一些非常技术性的讨论，涉及从 FPGAs 到频谱监控和频谱相关密度的各种技术，您可以在下面看到该视频。我们假设你最终能够观看[节目](https://2020.sdra.io/pages/programme.html)中列出的所有演示。

频道上也有一些老视频，包括一些 GNU 电台的资料。我们听说一些即将到来的视频也会有一些新的 GNU Radio 内容，包括一些 Android 的 GNU Radio 实现。

软件定义无线电是如何从一项新奇的技术转变为普通技术的，这一点非常引人注目。如果你喜欢这些科技含量很高的演示，你可能也会喜欢这本[免费书籍](https://hackaday.com/2018/06/29/free-e-book-software-defined-radio-for-engineers/)。不管你想要什么便宜的特别提款权，T2 已经有评论了。更不用说对[高端设备](https://hackaday.com/2020/04/08/the-esa-reviews-software-defined-radios/)的评论了。
