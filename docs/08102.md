# 备件快递

> 原文：<https://hackaday.com/2020/10/17/spare-parts-express/>

我有备用零件，我不能说谎。

这个星期，我给朋友和同事发了两个爱心包裹，因为我手头有太多可破解的东西，没有足够的时间全部破解。一个是时髦的键盘，另一个是 FPGA 开发板，但这不是重点。关键是，这个世界太有趣了，我们中的许多人都有更多的项目堆积在待办事项框中，以及相关的设备，以至于我们永远没有时间去完成。

回到以前，我们会见面，谈论我们正在进行的黑客活动，并且总是有人会说“哦，你需要一个 X，我有半盒 X”，然后送你一个。或者你可能是手头有额外部件的人。我知道我在这两个位置都很开心。

无论哪种方式，对于给予者来说都是一个胜利，他可以从一堆小部件中取出一个小部件，对于接受者来说，他不必去小部件商店，对于环境来说，他必须生产更少的小部件。(我向小部件制造商和中间商道歉。)

这让我想起了 Lenore Edman 和 Windell Oskay 在最近的一次活动中提出的电子垃圾互联网迁移箱。浏览[维基](https://tgimboej.org/Box_Tracking)就像沿着记忆的小路旅行。[这个盒子](https://tgimboej.org/Box_Code:_Yokohama)拜访了我的老黑客空间，然后和 Bunnie Huang 结束了。好时代，好人，好黑客垃圾！然后是我们自己的布莱恩·本考夫的[旅行黑客盒子](https://hackaday.io/project/7373-travelling-hacker-box)和[衍生](https://hackaday.io/project/19939-goodie-box-royale)。

这些都是伟大而有趣的项目，但它们最终都在一个方面失败了:要想有意义，货物的价值必须超过邮费，如果你只对任何给定盒子里的几样东西感兴趣，那就增加了大量的运输成本。

所以我试图想出一个更好的解决方案。某种集中式插接板，在那里“有太多 h 桥驱动器”的人可以与“需要 h 桥”的人联系？或者说，我们已经拥有的这个特别的社交网络已经足够好了吗？

你怎么想呢?我们怎样才能把货物送到那些想加工它们的人手中呢？
