# 使用多平台街机游戏设计器创建 8 位游戏

> 原文：<https://hackaday.com/2019/03/29/creating-8-bit-games-with-the-multi-platform-arcade-game-designer/>

从头开始创作一款游戏可能是一项艰巨的工作。要实现这一切，需要设计概念，编写代码，创造艺术。然而，这并不总是那么困难。有各种各样的开发工具可以让初露头角的游戏设计师通过点击的方式开始游戏。[[Jonathan Cauldwell]已经提出了这样一个工具，它可以让你在各种 8 位平台上做到这一点。](https://jonathan-cauldwell.itch.io/multi-platform-arcade-game-designer)

[Jonathan]的项目被称为多平台街机游戏设计器，因其能够为过去的几个 8 位系统创建游戏而得名。目前，Spectrum、Amstrad CPC 和 Acorn Atom 都得到支持，并计划在未来增加更多支持。

创建一个游戏是一件简单的事情，[[Jonathan]在视频教程系列](https://www.youtube.com/watch?v=9evsT7vL0xM&feature=youtu.be)中对此进行了探索。软件内置了精灵和背景编辑器。可以自动生成脚本来创建各种各样的基本游戏类型，从滚动射击游戏到经典的平台游戏。还有一些功能允许高级用户通过提供一些他们自己的代码来添加更多的功能。

如果点击不适合你，你可以用经典的方式打造你的 ZX 频谱游戏，[汇编和基本。](https://hackaday.com/2012/10/04/try-out-your-assembly-and-basic-skills-with-a-zx-spectrum-project/)休息后的视频。
