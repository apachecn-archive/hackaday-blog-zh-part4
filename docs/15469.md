# 特斯拉线圈制造钠等离子体

> 原文:[https://hack aday . com/2022/11/26/特斯拉线圈制造钠等离子体/](https://hackaday.com/2022/11/26/tesla-coil-makes-sodium-plasma/)

寻找一个巧妙的方法来处理你的特斯拉线圈？【行动实验室】用他的线圈[制造金属等离子体](https://www.youtube.com/watch?v=-k-o7oYTG9Q)——特别是钠。你可以在下面的视频中看到结果。

为了产生金属等离子体，你需要一种金属蒸汽，钠可以在相对较低的温度下产生蒸汽，尤其是在真空中。由此产生的辉光看起来很漂亮，但你需要一点实验室设备来实现它。

等离子体——有时被称为物质的第四种状态——是一种非常热的物质，电子被拉走，形成电离气体。与气体、固体和液体相比，这听起来很奇怪，但事实证明，超过 99%的可见宇宙是等离子体。像闪电、太阳和霓虹灯都是等离子体的形式。然而，从金属蒸汽中制造等离子体有点不寻常，钠的实验非常有趣，也是你壁橱里特斯拉线圈的有趣用途。
