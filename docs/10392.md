# Alexa，给我拿瓶啤酒来！

> 原文：<https://hackaday.com/2021/06/14/alexa-bring-me-a-beer/>

语音控制的家庭助理是我们这个时代的奇迹，无论如何，一旦你与大公司拥有的数据中心分享你生活中的隐私问题达成和平。它们让我们领略了过去几十年乐观预测中的未来:Alexa 和 Siri 可以讲笑话，控制你的灯，回答问题，告诉你新闻，等等。

但是，尽管他们的电子对话完美无缺，你的电子朋友却不能满足你最基本的需求，也不能给你一杯啤酒。这是[luisen engineering]已经解决的问题，他提供了问题的适当答案。”。如果你想要的话，我们也把视频放在休息时间的下面，视频是德语的，带有 YouTube 的自动隐藏字幕，但是我们认为你不需要学习说一点*德语*就能理解他的笑话。

随着他开发他的啤酒配送系统，我们开始意识到看起来相对简单的任务其实并不简单。他拿了一个现成的机器人，给它一个啤酒瓶抓取器和冰块漏斗，但从冰箱到沙发的路径仍然需要一点点工作。最终的解决方案包括大量的试验和错误，以及地板上的一条黑线供“机器人”遵循。终于，他的电子朋友可以给他带啤酒了！

我们喜欢[路易斯]有趣的演讲风格，以及使用道具作为麦克风支架。我们会密切注意他下一步的行动，你也应该如此。同时，你可能不会感到惊讶，这不是我们带给你的第一个啤酒递送机器人。
