# 哦，多孔的光

> 原文:[https://hackaday.com/2020/09/30/oh-holey-light/](https://hackaday.com/2020/09/30/oh-holey-light/)

谈到贴在五金店零件上的技术文献，我们认为自己很精通。首字母缩略词不会吓到我们，我们的 Google-fu 足够强大，可以克服大多数谜团。我们不明白的一点黑魔法是关于 LED 灯的天书。瓦数是容易的，色温是有意义的，因为它与暖色和冷色相对应，但显色指数(CRI)听起来像是深魔法。当然，有些人对这些术语理解得如此透彻，以至于他们可以教我们其余的人，比如[Jon]和[Kevin]，他们正在建造一个[灯光控制器，通过在一个单元中安装几个灯来纠正廉价灯](https://hackaday.io/project/174098-lighting-color-control-with-commodity-lamps)的不足之处。

通过阅读他们的日志，我们学到了很多东西，就像照明工程师教科书中的悬崖笔记一样，但我们将把它作为学生通读的练习。他们的项目使用精确的光传感器来测量廉价灯发出的光的“味道”,这样你就可以混合出令人满意的比例。在某些方面，他们正在复制白炽灯泡的效果，白炽灯泡发出的光在可见光光谱中相对均匀，一直到红外光谱。不幸的是，廉价的 led 在其光谱覆盖范围内有漏洞，一个[暖白色](https://hackaday.com/2018/10/29/history-of-white-leds/)单元与[日光](https://hackaday.com/2019/08/17/great-artificial-daylight-via-broken-tvs/)相比有不同的差距，但将它们恰到好处地结合起来会产生丰富的输出，而不会破产。
