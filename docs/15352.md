# 用昏暗的灯泡测试器给老式电子设备加电，不那么不安全

> 原文:[https://hack aday . com/2022/11/13/power-up-vintage-electronics-less-unsafety-with-a-dim-bulb-tester/](https://hackaday.com/2022/11/13/power-up-vintage-electronics-less-unsafely-with-a-dim-bulb-tester/)

插上古董收音机之类的东西看看它是否工作是一个很好的方法，因为一些旧组件不会很好地老化。对于老式电子产品，检查和维修是第一步和第二步。当需要谨慎使用电源时，最好使用所谓的调光器测试器，大多数黑客都可以用废品组装一个。

[![](../Images/e92fc304850f4d32312c1a5dbaa12f92.png)](https://hackaday.com/wp-content/uploads/2022/11/dim-bulb-tester-DIY-2.jpg)

Being able to use one (or both) bulbs adds some flexibility, and the embedded power monitor is an inexpensive and handy addition.

这些测试仪可以更容易、更安全地判断设备的电源是否有大问题。最简单的形式是，昏暗灯泡测试仪将一盏白炽灯串联在一个设备(如旧收音机)和墙上插座的交流电源之间。由于这一点，如果设备短路，灯泡将简单地点亮，而不是造成任何损害。

理想情况下，使用额定功率大致等于被测设备功耗的灯泡。如果一切正常，灯泡将发出微弱的光，设备将正常工作。明亮发光的灯泡表示电流消耗过大。为了允许一定的灵活性，[Doz]的测试器设计允许串联使用一个或两个 60 W 的白炽灯泡，甚至包含一个廉价的功率监视器。

[昏暗灯泡测试仪](https://antiqueradio.org/dimbulb.htm)不是一种深度诊断工具，但它有效、简单，即使出现短路等严重问题，也能安全启动。它有助于保护有价值的硬件不被烧毁。事实上，在发生故障的情况下，限制功率以保护硬件的基本概念也已经应用在逆向计算的世界中，如果出现问题，它[有助于保护否则不可替代的硬件](https://hackaday.com/2018/05/24/a-smarter-psu-converter-leaves-the-magic-smoke-inside/)。
