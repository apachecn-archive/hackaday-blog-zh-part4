# 猫机器人瘦腿的秘诀？驱逐马达！

> 原文:[https://hack aday . com/2018/09/15/cat-robots-secret-to-slim-legs-放逐-马达/](https://hackaday.com/2018/09/15/cat-robots-secret-to-slim-legs-banish-the-motors/)

关于【Bijuo】的[猫大小的四足机器人设计](https://blog.naver.com/PostList.nhn?blogId=louis0814&from=postList&categoryNo=11)(链接是朝鲜语，谷歌[的翻译在这里](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=https%3A%2F%2Fblog.naver.com%2FPostList.nhn%3FblogId%3Dlouis0814%26from%3DpostList%26categoryNo%3D11&edit-text=&act=url))首先要注意的是它们的腿是多么苗条和光滑。这是因为与大多数腿式机器人不同，四肢本身不包含任何马达。相反，马达在主体中，一个驱动半圆滑轮，另一个整体移动肢体。动力是通过一根充当钢筋束的电缆传递的，并被接头中的弹簧张力抵消。结果是轻盈纤细的双腿以非凡的步态抬起并移动。

[Bijuo]认为 [Cheetah_Cub](https://biorob.epfl.ch/cheetah) 项目是他们最初的灵感来源，并将他们自己的变体命名为 *Mini Serval* ，考虑到耳朵并与猫科动物的命名保持一致。下面嵌入了两个视频，第一个显示了腿部和步态细节，第二个演示了运动中的机器人。
