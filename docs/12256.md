# 制氢变得容易

> 原文：<https://hackaday.com/2021/12/20/hydrogen-generation-made-easy/>

即使你从来不想[产生氢气](https://www.youtube.com/watch?v=RY08xO-_11w)，【Maciej Nowak 的】视频(嵌在下面)看起来也很有趣，因为电极是由不锈钢垫圈巧妙形成的。你需要热缩管，但无论如何你应该把它放在身边。使用视频中的技术构建电极会产生大量表面积，这对电化学反应非常重要。

一个标准的可充电电池为装在改良塑料瓶中的发电机供电。整体建筑看起来很好，即使它都是重新利用的材料。

里面的化学物质是普通的水和下水道清洁剂——氢氧化钾。我们不需要告诉你要小心，也要注意你对爆炸性气体的处理。我们说“爆炸性”而不是“易燃性”，因为这种设计没有将氢气和氧气分开，由此产生的混合物很容易爆炸。视频显示了一些自制的使用燃料的火箭，虽然它们不会去月球，但它们确实携带了相当多的能量。

我们对这个小瓶子产生的气体量印象深刻。我们只是想不出一个好的理由来为假期准备爆炸性气体。也许你想喝点咖啡？或者你可以用[氢糊](https://hackaday.com/2021/02/06/the-future-of-hydrogen-power-is-paste/)燃料做实验。
