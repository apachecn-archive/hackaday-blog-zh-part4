# 衡量你在 YouTube 上的重要性

> 原文：<https://hackaday.com/2019/07/10/measure-your-youtube-importance/>

你是如何黑掉自己的动力的？你每天锻炼的时候会在纸质日历上划红色的马克笔 x 吗？你用鸡蛋计时器冲刺几十封邮件吗？你会锁上所有的门，关闭你的数据来写一些防弹代码吗？如果你是[浩克]，你可以创建一个[YouTube 桌面通知程序](https://hackaday.io/project/163459-diy-youtube-desktop-notifier/details)来显示他的 YouTube 订阅者和浏览量。这是他离开沙发制作一个关于这样一个设备的视频的门票。建立一种机制来监控自身的成功，形成一种反馈循环，这很有诗意。Hackaday.io 页面跟随视频，因此任何想要建立自己的视频的人都不必在暂停视频时乱写笔记，视频也发布在休息时间下方。

硬件列表是合乎逻辑的，从通过 Arduino IDE 编程的 NodeMCU 模块开始。可寻址的 7 段显示器以红色显示统计数据，但您可以使用背光 led 显示您喜欢的颜色。如果你足够重要，需要更多的数字，应该可以共享显示器上的 CLK 别针。[浩克]已经列出了一系列改进，包括切换到可寻址背光和增加每日和每月跟踪。

在没有电脑显示器的情况下监控在线价值在某种程度上令人满意，因为它显示了我们的动机，无论是比特币还是 T2 天气。
