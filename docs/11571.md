# 树脂印刷黑客聊天

> 原文:[https://hackaday.com/2021/10/11/resin-printing-hack-chat/](https://hackaday.com/2021/10/11/resin-printing-hack-chat/)

加入我们太平洋时间 10 月 13 日星期三中午与 Andrew Sink 的[树脂印刷黑客聊天](https://hackaday.io/event/181910-resin-printing-hack-chat)！

从本质上来说，3D 打印是一个如此简单的想法，难怪没有人更早想到它。诚然，熔融沉积成型确实可以追溯到 80 年代，相对而言，最近廉价的大众市场 FDM 打印机的爆炸更多地与廉价组件有关。但实际上，到了最后，商用 3D 打印机实际上只不过是美化了的热胶枪，尽管它们仍然是创客运动的基础技术，但它们已经变得有点乏味了。

因此，很自然，我们这个群体会寻找其他方法来推动 3D 打印信封，立体平版印刷术已经成为新的热点。这是有充分理由的——尽管它可能很乱，但通过选择性光聚合逐渐从一罐粘糊糊的东西中拉出一个模型的能力看起来很神奇，树脂打印机能够做到的精细细节水平也同样迷人。树脂打印机的价格也是如此，它正迅速与 FDM 商品打印机竞争。

如果你的未来有一台树脂打印机，那么当安德鲁·辛克拜访我们的时候，你会想顺便聊聊的。安德鲁最近在他的 YouTube 频道上做了很多 3D 打印的东西，特别是树脂打印。我们特别介绍了他的几个[技巧](https://hackaday.com/2021/08/27/resin-3d-prints-get-a-new-look-with-paint-injection/)和[技巧](https://hackaday.com/2021/10/06/making-the-most-of-your-resin-printer-investment/)，以便从树脂打印机中获得最大收益，他将分享一些他最近学到的东西。加入我们，讨论树脂印刷的来龙去脉，采取行动需要什么，以及 SLA 与 FDM 的利弊。
