# 用 RGB 点亮胶棒自行车轮胎

> 原文：<https://hackaday.com/2022/10/31/lighting-up-glue-stick-bicycle-tyres-with-rgb/>

对于骑自行车的人来说，被驾驶者看到是一个持续的问题，但我们怀疑[Q]用胶棒制成的[RGB LED 照明轮胎](https://www.youtube.com/watch?v=bJzk1XN3zDs)会有这个问题。

该项目从一套 3D 打印的轮胎模具开始，这些模具栓接到标准车轮上。将融化的胶棒倒入模具中，冷却，冷却后借助热风枪取出模具部分。我们怀疑重量和硬度使轮胎特别实用，但你不能让正常的轮胎从内部发光。

照亮轮胎的想法可能是在成型后产生的，因为它们必须被切掉以适合发光二极管。[The Q]用两个螺钉之间的一根镍铬合金线制作了一个简单的热线夹具，并用它从轮胎内部切割几毫米，并在车轮中安装一个带套筒的 RGB LED 灯条。电力来自一组三个 18650 电池，与无线控制器一起安装在 3D 打印的集线器安装外壳中。

就像【The Q】的[无轮毂](https://hackaday.com/2021/06/24/fat-tire-bike-turned-hubless/)和[偏轮自行车](https://hackaday.com/2022/07/31/trippy-bicycle-uses-multiple-partial-wheels/)一样，绝对令人眼花缭乱，功能跟形式。
