# 经典玩具直升机再次飞行作为 DIY 版本

> 原文：<https://hackaday.com/2020/02/11/classic-toy-helicopter-flies-again-as-diy-version/>

对于我们中许多生长在 20 世纪 70 年代的人来说，“VertiBird”，这种自己动手的室内直升机，是一种经常被索要的玩具，最终它出现在了圣诞树下。到了圣诞节下午，许多迷人但精致的玩具都报废了，它们是旺盛的游戏和价格点工程的致命组合的受害者。但是现在，这个经典玩具的 DIY 版本再次起飞了，这次是一个更加坚固的设计。

公平地说，玩具公司美泰(Mattel)的设计者推销 VertiBird 时，这个玩具相当令人惊讶。这架塑料直升机由位于中央底座的马达驱动，该马达转动穿过刚性系绳的驱动杆。底座和直升机上的小弹簧起到万向节的作用，将动力传递给旋翼。这些弹簧是设计中的薄弱点，尤其是底座中的弹簧，经常断裂成两半。

[卢克·j·巴克]的重新设计将一个微型齿轮马达放在飞机里，而不是放在底座里，这在最初的设计中是不可行的。为了解决从基地到飞机的电力问题，[卢克]避开了昂贵的滑环，而是使用了标准的 3.5 毫米音频插孔和插头。插头充当底座中主齿轮的轴，主齿轮为直升机的旋转提供动力；可悲的是，这个版本没有像原版一样机械地倾斜飞机来控制向后和向前飞行。一对带有 3D 打印杠杆的锅通过 Arduino 控制油门和飞行方向；请观看下面的视频。

这些页面充斥着旋翼机的制造，既有[直升机](https://hackaday.com/2017/07/24/controlling-a-micro-helicopter-with-a-ps2-controller/)又有[多旋翼](https://hackaday.com/2019/03/11/quadcopter-uses-bare-metal-stm32/)。我们欣赏各种各样的飞行器，但是这个真的把我们带回到过去。
