# 这台电脑绝对不是玩具

> 原文：<https://hackaday.com/2022/09/30/this-computer-is-definitely-not-a-toy/>

如果你曾经打量过一台儿童笔记本电脑，并想知道它是否可以升级为单板机，你并不孤单。【Labz】从十多年前的[拿了几台巴西 *Max Steel* 玩具电脑，把它们改造成非常规的便携电脑](https://www.youtube.com/watch?v=VEwS330gj2A)(巴西葡萄牙语，但 YouTube 的字幕翻译是你的朋友)。

这些电脑类似于你可能熟悉的 VTech 之类的电脑，一个 QWERTY 键盘和相当传统的外形，但有一个微型单色液晶显示器和几个内置游戏。在广告下方的视频中，我们看到笔记本电脑和台式机都被旋转工具切割，以获得新的更大的屏幕，笔记本电脑得到了一个树莓 Pi，台式机得到了一个小尺寸 PC。笔记本电脑需要一个 3D 打印的延长线来腾出额外的空间，而台式机则需要一根 PCI Express 延长线来连接显卡。最后，一个 Arduino 搞定了键盘。

这部视频的蛋糕上的樱桃出现在结尾，当他们发现原来广告中的孩子已经长大成人。同时，[儿童电脑已经在这里出现过几次](https://hackaday.com/2020/04/27/teardown-vtech-whiz-kid-luggable-computer/)。
