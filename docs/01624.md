# 以 Arduino 方式实现禅

> 原文：<https://hackaday.com/2018/12/25/achieving-zen-the-arduino-way/>

禅宗花园的目的是做事情，这些风格化的景观是通过煞费苦心地放置岩石和耙砾石形成完美的图案而创造的。确保每一条线都完美地形成不是一件容易的事，掌握它所需要的专注是整件事的关键。但是谁有时间做这个？为什么不制造一个机器人来创造一个完美的微型禅宗花园呢？

这就是 Tim Callinan 和他的同学在一个学期的项目中所做的,[Zen xy 沙盘绘图仪](https://youtu.be/D8uCUroAHD4)就是成果。设备本身没有构建日志，尽管下面的视频清楚地说明了他们是如何做的。沙盘本身是一个胶合板箱，箱底铺有细白砂，内含单个钢球。桌子下面是一个 X-Y 扫描架，上面有一块强力磁铁。骑在 Uno 上的 [gShield](https://github.com/synthetos/grblShield) 将 g 代码转换成球在沙地上缓慢而庄严的运动。这些模式非常复杂，虽然这可能不等同于精确地控制身体来耙砾石，但看着球推动沙子本身就很有禅意

这不是我们第一次看到有人试图将传统的日本做法自动化。这个茶道机器人浮现在脑海里，[这个制作精美的沙盘](https://hackaday.com/2017/05/16/zen-and-the-art-of-arduino/)与 ZenXY 非常相似。
