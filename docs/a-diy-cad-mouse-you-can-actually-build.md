# 一个你可以自己动手制作的 CAD 鼠标

> 原文：<https://hackaday.com/2022/01/18/a-diy-cad-mouse-you-can-actually-build/>

当你花大量时间在电脑上做某些更专业的任务时(不，我们不是在谈论在 twitter 上浏览猫迷因)，你开始认为你的基本触控板或鼠标，比方说，缺少某种东西。我们认为有些东西可以被称为“可用性”或者易用性？无论如何，许多 CAD 用户都喜欢他们最喜欢的鼠标替身，一个特别有趣的输入设备类别是太空鼠标，它基本上获得了全部专利，只能从 3DConnexion 获得。但是，您可以自己构建的开源替代方案呢？进入舞台左侧，[由【FaqT0tum】创建的圆形](https://github.com/FaqT0tum/Orbion_Space_Mouse)。]这个简单的小构建将模拟操纵杆与旋钮相结合，后面有一个按钮，前面有有机发光二极管显示屏，从而完善了用户界面。

这个想法很简单；你用你想使用的应用程序设置固件，它会向连接的 PC 发出 HID 事件，取代鼠标或键盘输入。因为你的机器将从多个来源获取输入，它不会取代你的鼠标，而是增强了它的功能。对于详细的 PCB 布局工作，它可能不是非常准确，但对于在 3D 视图中四处移动，或在视频编辑中拨号，这可能是对您的工作站非常有用的补充，所以为什么不尝试一下呢？布线简单，零件容易找到而且便宜，而且只是几个印刷零件！这位抄写员现在已经在打印塑料了，如果你仔细听的话，你也许能分辨出背景中 Lulzbot 的声音。

这个想法还有许多其他的想法，复杂程度各不相同，像[这个来自【阿姆斯维尔】](https://www.youtube.com/watch?v=yGbshYLfDnc)的令人难以置信的构建，遗憾的是它没有公开提供 PCB，这里有一个我们之前报道过的[将昂贵的 3DConnexion spacemouse](https://hackaday.com/2017/07/27/unholy-mashup-of-spacemouse-and-sculpt-keyboard-is-rather-well-done/) 混搭成 keeb。
