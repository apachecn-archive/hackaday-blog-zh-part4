# 参观 FACOM 128B 1958 中继计算机

> 原文：<https://hackaday.com/2019/12/06/visiting-the-facom-128b-1958-relay-computer/>

如果你研究计算机的历史，你可能听说过 FACOM 128B，一台 1958 年的日本中继计算机。它是仍以其原始形式工作的最古老计算机的竞争者，因为它位于日本沼津的富士通大楼内。[CuriousMarc]参观了这台旧电脑，并制作了一个关于它的视频，还画了一幅其他当代机器的图片。你可以看下面的视频。

[Marc]解释了一台接力机在 1958 年就已经落伍了，还展示了 5000 接力机是如何布局的。展出的这台机器来自东京的一所大学，它所做的计算可以让你用今天的电脑进行工程设计。

看着——听着——野兽运行一个程序，把 3 和 3 相加，很能说明问题。键盘让我们想起了老式的加法机。机器使用浮点或二进制五进制，将每个数字存储为 7 位。这导致了一个很宽的字宽度，但是当一个继电器被卡住时很容易看到，因为每个数字有两个部分，并且对于任何数字，每个部分只有一个继电器是开的。

该视频不仅仅是参观的家庭电影，还展示了操作员如何输入数据和指令，以及如何阅读输出显示。这台机器有 180 个字的内存，需要几个柜子。这相当于 13K 的内存。这仍然足以建立一个 5×5 矩阵求逆的解决方案。看到程序循环实际上是穿孔带的循环，我们觉得很有趣。

我们喜欢驾驶喷气式飞机，但也怀念飞艇。出于同样的原因，我们有点希望我们的台式电脑使用这样的继电器。如果这些都是真的，我们可能会很不高兴。话说回来，最近我们还见过一个[单板中继电脑](https://hackaday.com/2017/09/18/single-board-relay-computer/)。
