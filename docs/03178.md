# 动作跟踪脸真的会在房间里跟着你

> 原文:[https://hack aday . com/2019/06/04/motion-tracking-face-really-follow-you-around-the-room/](https://hackaday.com/2019/06/04/motion-tracking-face-really-does-follow-you-around-the-room/)

我们中的许多人都有过在画廊里观看艺术品的经历，在这种情况下，眼睛似乎在房间里跟着一个人走。在我们的高技术工作中，这不再需要用艺术技巧来实现。你可以建造一些真正会动的东西。

黄绿色是[阿灵顿]的创造，有它自己的个性。一张脸是用激光切割的木头做成的，然后一层一层地组装起来。然后给它装上发光的 LED 眼睛，并安装在一个旋转的盘子上。结合 Arduino 和超声波传感器，它能够跟踪在其视野内移动的目标，并旋转以跟随它们。黄绿色的表情也随着情况的变化而变化，从开心到绝望。

这是一个很好的例子，展示了通过激光切割材料分层可以实现的艺术效果，以及如何通过简单的制造商钉书钉(如伺服系统和微控制器)将艺术带入生活。运动追踪也有很多有用的应用——[比如将热量直接对准冰冷的人](https://hackaday.com/2019/03/22/raspberry-pi-tracks-humans-blasts-them-with-heat-rays/)。休息后的视频。
