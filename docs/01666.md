# 让自己拥有一个递归的小圣诞:印有装饰物的装饰品

> 原文：<https://hackaday.com/2018/12/24/have-yourself-a-recursive-little-christmas-ornament-that-prints-ornaments/>

当然，偶尔会有实用的圣诞树装饰品；我们有一个插在灯上的耳机，听起来应该像鸟儿轻轻地鸣叫，但事实上它太刺耳了，我们被禁止使用。但总的来说，装饰品只是用来看的，对吗？没那么快— [这个 3D 打印的饰品内部有一个 3D 打印机，可以打印其他饰品](https://www.youtube.com/watch?v=VCcJjI2u4AU)。有一天，它可能会成为实用圣诞装饰中的必备物品。

考虑到(肖恩·哈金斯)只有几天时间来研究这种树栖 3D 打印机，有问题的打印质量和微小的打印量可以忽略不计。但事实上，他让这个工作是相当了不起的。起初，我们很惊讶他选择建造立体平版印刷(SLA)打印机，而不是更常见的熔融沉积成型(FDM)打印机，但这是有道理的。SLA 只需要在 Z 轴上移动，在这种情况下由旧 DVD 驱动器的内部提供。构建平台在一个微小的树脂罐中进进出出，树脂罐的底部有一个小型 LCD 屏幕，其背光已被一束 UV LEDs 取代。羽毛 M0 控制建造阶段的高度，并在 LCD 上显示预先切片的位图，每次固化罐中的树脂一片。

结果喜忧参半，细小的雪花是最好的。然而，对于一个紧急的工作，一个与[竞争合作一个包裹盗窃威慑闪光炸弹](https://hackaday.com/2018/12/17/hacker-makes-a-flawless-booby-trap-strikes-back-against-package-thieves/)的工作，这是相当令人印象深刻的。希望这能像[Sean]承诺的那样变成一个完整的 SLA 版本。
