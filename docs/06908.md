# 本周失败:如何不让电脑水冷

> 原文：<https://hackaday.com/2020/06/17/fail-of-the-week-how-not-to-watercool-a-pc/>

对于那些选择超频个人电脑的人来说，这通常是一笔“不惜代价”的交易。花哨的散热器、复杂的液体冷却装置和炫酷的透明外壳都是魅力的一部分。但是并不是每个人的口袋都有足够的钱来购买现成的零件，所以尝试更便宜的替代品，比如使用汽车燃油泵来移动冷却液似乎是个好主意。实际上——没有那么多。

当我们看到[BoltzBrain]视频的标题时，我们想到的第一件事是一名机械师很久以前发出的警告，永远不要在燃油喷射汽车中耗尽汽油。事实证明，汽油是电动泵的冷却剂和润滑剂，在泵仍然通电的情况下，让油箱保持干燥状态会很快耗尽汽油。因此，虽然[BoltzBrain]预计会看到由于他使用水作为工作流体而造成的电刷腐蚀，但我们预计会看到轴承卡住是故障的根本原因。看来我们错了:在大约 6:30 的标记处，你可以看到连接电刷的铜线上有明显的腐蚀痕迹。这几乎看起来像是 Dremel 工具切断了电线，但绿色的氧化铜是赠品。我们怀疑轴承也不是很好，但这可能是电线腐蚀的次要原因。

无论根本原因是什么，这是一次在普通零件中的有趣旅行，建造一台在高度易燃液体中运行的有刷电机所需的工程水平令人印象深刻。我们特别喜欢电刷和换向器的轴向排列。我们想知道燃料泵是否仍然可以作为 PC 冷却器——也许换成电介质流体就可以了。
