# 借助于绘图纸构建的 LED 矩阵眼镜

> 原文：<https://hackaday.com/2021/06/18/led-matrix-glasses-built-with-the-help-of-graph-paper/>

如今，有各种各样的可寻址发光二极管，可以很容易地用来制作闪闪发光，华而不实的项目。然而，没有什么能阻止制造商用传统的方式做事，用原始的 led 拼凑出一个矩阵。迪帕克·卡特里用他自己的定制版本做到了这一点。

[Deepak]选择自由设计，而不是依靠 PCB 或其他基板将矩阵固定在一起。在绘图纸的帮助下，在纸板模板上切出一排孔。然后将发光二极管按所需的图案插入孔中，并将它们自己的引线焊接在一起，形成眼镜的框架。然后安装所需的额外电线，兼作桥梁，让眼镜舒适地放在鼻子上。然后在背面使用黑色环氧树脂来阻挡光线使佩戴者失明。矩阵由一对移位寄存器控制，移位寄存器由一个微控制器寻址，显示器的动画显示非常流畅。

这是一个有趣的建筑，我们怀疑它在晚上看起来特别令人印象深刻。他们也会让你的朋友很容易在黑暗的俱乐部里认出你。[这些年来，我们也看到了一些令人印象深刻的时尚 LED 眼镜，](https://hackaday.com/2020/05/08/these-led-shades-will-blind-you-with-science/)可以追溯到 2012 年的【macetech】的[眼镜。休息后的视频。](https://hackaday.com/2012/05/22/macetechs-led-glasses-prototype/)
