# 双持机器人用弯曲的金属丝从泡沫中雕刻出 3D 形状

> 原文：<https://hackaday.com/2020/06/03/dual-wielding-robot-carves-3d-shapes-from-foam-with-warped-wire/>

“每一块发泡聚苯乙烯泡沫塑料内部都有一座雕像，双臂热导线机器人的任务就是发现它。”——[米开朗基罗]，可能吧。

准备好被这款双用途热丝 3D 泡沫切割机震撼吧。我们以前都见过简单的热线切割机，无论是[手动送料切割机](https://hackaday.com/2016/01/28/super-slick-super-portable-styrofoam-slicer/)还是[像 3D 打印机一样的数控切割机](https://hackaday.com/2020/02/01/cnc-hot-wire-cutter-gives-form-to-foams/)。这个想法是让电流通过一根电线，当它被引导通过一块聚苯乙烯泡沫时，加热到足以融化一条路径。与用刀或锯切割相比，热线切割像丝绸一样光滑，幸运的是产生的聚苯乙烯泡沫碎屑很少，而且会弄得你的工作空间到处都是。

但是热线切割机除了直切之外不能做太多，因为电线必须保持绷紧。苏黎世联邦理工学院(ETH Zurich)的西蒙·杜恩泽(Simon Duenser)和他的同事称他们的发明为“机器人足球赛”，但却没有这些限制。使用 ABB 米语，一个双臂协作机器人，他们设计了一种方法，通过使用 1 毫米厚的可变形杆而不是柔软松软的金属丝作为切割工具，在泡沫中进行可控的弯曲切割。机器人的每只手臂都有七个自由度，在永久损坏之前，杆只能变形这么多，因此涉及的运动学问题非常复杂。每一次通过泡沫都是为了去除尽可能多的材料，需要多次通过才能达到最终设计。

下面的视频显示了释放被困在泡沫中的斯坦福兔子所需的迷人扫描，以及其他常见的 3D 测试模型。我们不确定这是否能被家庭游戏玩家轻松重现，但看起来肯定很有趣。
