# 其他人用 8 比特做的事，德国人只用了 1 比特

> 原文：<https://hackaday.com/2020/02/01/what-everyone-else-did-with-eight-bits-the-germans-did-with-only-one/>

在 20 世纪 80 年代，年轻人第一次有了掌握计算机知识的动力。推出了各种不同的教育节目，通常涉及计算机制造商和广播公司之间的合作，并以在当时的一台 8 位家用计算机上进行基本编程为特色。一个这样的教育计划有点不同，德国广播公司 WDR 制作了一个教育系列，使用模块化计算机，具有一个不寻常的 1 位处理器，用十六进制机器代码编程。[Jens Christian Restemeier] [制作了这台机器](https://www.youtube.com/watch?v=PrpStnNA2GM)的复制品，这是他所能制作的最接近原型机的复制品。(视频，德语，嵌入下方。)

这台计算机被称为 [WDR-1](https://www.old-computers.com/museum/computer.asp?st=1&c=834) ，在被广播公司使用之前，它的起源是一台配套的机器。不寻常的 1 位处理器是一个[摩托罗拉 MC14500](https://en.wikipedia.org/wiki/Motorola_MC14500B) ，它从 1977 年开始生产，用于工业控制应用。他在视频中带领观众穿过机器的零件，解释每个子卡和主板的用途。由于没有原件可供复制，他转而从照片中复制原件的芯片布局，用引脚接头取代了 20 世纪 80 年代机器上不常见的插座。看看他的视频，在休息下面。

关于 WDR-1 [的更多信息可以在网上找到德语](http://wdr-1-bit-computer.talentraspel.de/) ( [谷歌翻译链接](https://translate.google.com/translate?sl=auto&tl=en&u=http%3A%2F%2Fwdr-1-bit-computer.talentraspel.de%2F))。与此同时，我们之前介绍过小型嵌入式计算机中的[MC 14500。](https://hackaday.com/tag/mc14500/)
