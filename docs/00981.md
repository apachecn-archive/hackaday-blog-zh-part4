# 火焰喷射器得到更新，保留一些粗略

> 原文：<https://hackaday.com/2018/10/17/flamethrower-gets-update-retains-some-sketchiness/>

喷火器有趣的部分原因是其固有的危险性。然而，这正是许多事情变得有趣的原因，从滑雪到跳伞到骑摩托车。然而，和所有这些明智的爱好一样，重要的是尽可能避免不必要的风险，以确保你尽可能长时间地享受你选择的活动。考虑到这一点，[斯蒂芬]决定对他经典的腕戴喷火器进行一些改进。

首先，他放弃了为这个装置供电的沉重的铅酸电池，转而使用更小的 5 V 电池。事实上，整个构建更加紧凑和高效。他还能够使用相同的电池来运行一个微型泰瑟枪，作为火焰喷射器燃料的点火源。燃料本身是丁烷，由于燃料输送系统的改进，改造后的火焰喷射器能够发射比原来更远的火焰。这些改进还包括“找到一种方法来防止丁烷液滴点燃并落在[他的]手上”，这似乎也是一个必要的功能。

现在整个版本都非常精致和专业，这也是对第一个版本的一个重大改进。同样值得在休息后观看视频，其中包括与纽约市消防队长的小冲突。而且，它仍然保留了一些原始版本的危险和乐趣，这是我们一直想看到的。
