# 通过定制的 D-Sub 连接器供给灯丝和电子

> 原文：<https://hackaday.com/2019/12/13/feeding-both-filament-and-electrons-through-a-custom-d-sub-connector/>

我们有时会忘记，3D 打印机只是一个附加了云台的数控平台，还有一系列可供选择的工具头。[Jón Schone]一直在这样做，并且需要一种方法来快速地将他的 hotend 从他的打印机上完全断开，所以他 [3D 打印了他自己定制的 D-sub 连接器](https://www.youtube.com/watch?v=I-izkShIDXU)用于灯丝和电线。(视频，嵌在下面。)

[Jon]为他的 Creality CR10 3D 打印机增加了许多升级，包括一个[快速更换工具架](https://www.youtube.com/watch?v=7eTGU4crL_s)，使他也可以使用激光雕刻机，甚至一个小主轴。当取下盖子时，没有办法快速断开电线，所以打印头通常保持连接并放在打印机的一侧。为了快速分离电线和鲍登管，他首先[改装了现成的 D 型接头](https://www.youtube.com/watch?v=7eTGU4crL_s)。连接器相对昂贵，而且管子容易弹出，导致一些打印失败。

[Jon]希望在连接器内部使用合适的波顿管配件，因此他设计并印刷了自己的 D-sub 连接器，并购买了松散触点。事实证明，在不破坏触点的情况下将触点推入外壳非常困难，因此他正在努力简化这一过程。这只是 3D 打印 3D 打印机升级的众多例子之一，而 3D 打印机升级从一开始就是 RepRap 项目的核心功能。休息之后请看视频

我们不缺少 3D 打印机黑客，未来还会有更多。最近一些很酷的产品包括从一开始就考虑自动换刀的 Jubilee CNC 和可以装进背包的打印机。
