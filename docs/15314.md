# 三个北方排成一线，它甚至不在北方

> 原文：<https://hackaday.com/2022/11/08/three-norths-align-and-its-not-even-up-north/>

有时在 Hackaday，我们会给你带来一些与我们的科技世界稍有不同的故事，因为它们有一个有趣的角度。也许它们与科学或天文学有关，或者在英国国家测绘局解释英国的三北如何对齐的情况下，与地理有关。

你们有些人可能知道英国君主有两个生日，但是三北，这到底是怎么回事？你会猜测其中两个是真北，指向北极，磁北，指向地球北磁场，但是第三个呢？它是格网北方-该国地图格网系统的北方，其中曲面被投影到一个平面上。

它与格林威治以西 2 度的真北对齐，新闻是有史以来第一次由于磁北极的移动，三个不同的北方将在英格兰南部的一点对齐。在过去的几十年里，磁北一直在以一定的速度移动，从加拿大北极群岛的某个位置向北移动，碰巧的是，对于英国人来说，它的方向与我们对北极的看法短暂地一致。英国国家测绘局的故事很有意思，但要获得丰富的信息，还是值得咨询一下美国国家航空航天局。看看休息下面的视频。
