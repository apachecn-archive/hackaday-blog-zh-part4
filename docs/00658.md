# 惠普推出金属 3D 打印机

> 原文:[https://hack aday . com/2018/09/14/HP-rolls-out-metal-3d-printers/](https://hackaday.com/2018/09/14/hp-rolls-out-metal-3d-printers/)

你通常认为惠普生产喷墨和激光打印机。但他们一直在悄悄地制造面向商业客户的 3D 打印机。现在，他们带着名为“惠普金属喷射”的金属打印机离开了。视频(见下文)有点炫目，但基本思想是打印棒将粉末沉积在 21 微米的网格上。粘合剂在粉末上打印，大概是以与传统喷墨打印机相似的方式。然后热源从粘合剂中蒸发液体。

对每一层重复这个过程，直到你取出零件，然后用第三方类似烤箱的设备烧结。据惠普称，他们的技术比用激光在床上熔化粉末具有更均匀的材料特性。他们还声称比金属注射成型快得多。

该网站声称分辨率为 1200 DPI，实际上只有 21 微米多一点。嗯，严格来说，我们想还不到一根头发。构建体积为 430 x 320 x 200 毫米

随着圣诞节的到来，你可能想让你的另一半知道你想要一个。只要确保他们意识到价格标签将是“低于 399，000 美元。”我们希望你今年表现良好，尤其是因为你必须单独购买烧结机。当然，它们要到 2020 年才能上市——这是一个很长的准备时间，但 4999 美元的存款将会让你排队。

如果这个价格对你来说太贵了，惠普将在 2019 年的某个时候安排一个合作伙伴为你打印零件——至少如果你在美国或西欧。我们不知道价格，但从阅读他们的信息来看，似乎他们的目标用户是高容量用户，所以我们预计一些零件的价格将令人望而却步。

老实说。如果我们需要一台金属制品打印机，我们会持怀疑态度。当然，押金是可以退还的，但是你真的是在打赌这些会很好，不会有更好的东西，他们会按时交付或根本不交付。当然，惠普有良好的记录，我们认为如果押金可以退还，只要你不介意借给惠普 5000 美元几年，这不是一个很大的风险。但是两年内会发生很多事情。

我们不知道它是否真正涵盖了一切，但我们发现了一篇文章，声称是今年年初 3D 金属打印机的综合列表。如果你想要家里的金属零件，你最好的选择是[仍然可能铸造](https://hackaday.com/2018/02/25/casting-metal-parts-and-silicone-molds-from-3d-prints/)。你可以 3D 打印模具原件，也可以[打印模具本身](https://hackaday.com/2017/08/27/casting-metal-directly-into-3d-printed-molds/)。
