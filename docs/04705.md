# 向下通风装置消除灰尘

> 原文：<https://hackaday.com/2019/11/04/downdraft-device-dismisses-dust/>

木工是一项复杂的工作，尤其是打磨部分。但是，即使你不在乎你的车间会发生什么，你也不希望你的肺里有死树颗粒物质。戴上口罩甚至呼吸器是一个好的开始，但集尘系统更好。有一天，[XYZ 创造]可能会在全店范围内安装一台木屑粉碎机。与此同时，[他用废弃的胶合板和一个塑料储物盒](https://www.youtube.com/watch?v=fPwX_vMyhmU)做了一张下拉式桌子。

他手头唯一没有的东西是一个与他商店的吸尘器相匹配的端口。我们喜欢他的解决方法，以避免在塑料上钻一个肯定会破裂的大洞——使用软管夹来获得端口的外径，在热板上加热夹子，让它在盒子上融化一个洞。希望他至少打开了一扇窗。[XYZ 创造]将四块废胶合板粘在一起作为顶部，并用手钻了所有 117 个孔。谁需要钉板？

不够花哨或不够大，无法满足您的需求？这里有一个内置过滤系统的。
