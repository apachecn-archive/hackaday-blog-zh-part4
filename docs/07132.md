# 制造多氯联苯的简单方法

> 原文:[https://hackaday.com/2020/07/10/making-pcbs-the-easy-way/](https://hackaday.com/2020/07/10/making-pcbs-the-easy-way/)

在家制作 PCB 可能会很麻烦。如果你正在蚀刻，有化学物质和墨粉转移的细微差别。如果你在铣削，获得恰到好处的表面高度，并且不破坏那些尖尖的小 v 形刀具总是一个挑战。[Robin]对这两种情况都有提示，并通过使用铣床解决了许多常见的麻烦。

无论是刮掉蚀刻抗蚀剂还是整个铜隔离线，[Robin]都使用非旋转刮擦工具，而不是 v 形钻头:它们更耐用，也能切割每一点。他掌握了使用 FlatCam 和 KiCAD 绘制划痕的技巧。他的注册系统允许他以最少的麻烦获得双面印版。此外，他还做了一些将 SMT 零件嵌入电路板的实验。请务必查看他的整个指南，或者只观看下面的视频。

我们很确定你会学到一两个技巧，也许你会被说服咬紧牙关投资一家不错的工厂。如果你喜欢更传统的 PCB 铣削，试试我们自己的[【Adil Malik】指南](https://hackaday.com/2018/01/04/guide-why-etch-when-you-can-mill/)。
