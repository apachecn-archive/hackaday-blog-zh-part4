# 开源磨床让塑料挤出机的压缩螺杆变得简单

> 原文：<https://hackaday.com/2020/07/09/open-source-grinder-makes-compression-screws-for-plastic-extruders-easy/>

在一个塑料垃圾泛滥的世界里，不得不购买新的塑料细丝卷来喂我们的 3D 打印机，只是让它们产生更多的塑料垃圾，这似乎是一个遗憾。打破这种恶性循环需要将塑料回收与添加制造相结合，这需要一些带有不容易获得的零件的巧妙工具，如为塑料挤出机提供动力的压缩螺杆。

[这款开源压缩螺旋研磨机](https://www.mdpi.com/2411-5134/5/3/26)旨在让小型塑料回收机更容易建造。来自密歇根科技大学的 Joshua Pearce 的实验室和 Jacob Franz 的合作，这个设备是一个小型车床和研磨机的组合。一块圆钢被卡盘夹住，其自由端由尾架上的轴承支撑。机器的底座上有一个由 3D 打印零件和电线导管组成的 X-Y 托架。由于滑轮和螺纹杆的作用，当原料旋转时，托架沿着床身的长度向下移动，携带一个带有厚砂轮的无绳角磨机。附在前裙板上的模板控制研磨机沿着杆行进时切割的深度；不同的模板允许容易地定制螺杆轮廓。下面的视频展示了运行中的机器以及它能够生产的复杂螺杆轮廓。

我们以前见过很多自制塑料挤出机，其中大多数都使用改造过的螺旋型钻头作为压缩螺杆。那些缺乏适当的压缩螺钉的可变几何形状，所以[约书亚]和[雅各布]使[这种机器的所有设计文件可用](https://www.appropedia.org/Open-Source_Grinding_Machine_for_Compression_Screw_Manufacturing)应该是回收实验者的福音。
