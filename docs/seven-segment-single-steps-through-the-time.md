# 七段单步穿越时间

> 原文：<https://hackaday.com/2020/04/10/seven-segment-single-steps-through-the-time/>

你有没有看过时间，然后不得不再次看，因为它只是没有登记？这种现象似乎在手机计时器中更为普遍，但众所周知，标准挂钟也存在这种情况。由[mircemk]设计的一系列不同寻常的时钟中的最新产品解决了这个问题[,它迫使观众在时间以一系列由连字符](https://www.instructables.com/id/DIY-3D-Printed-Single-Digit-Arduino-Clock/)分隔的一位数闪过时注意时间。

在四四方方的蓝色底座内是一个 Arduino Nano、一个 DS3231 实时时钟模块和一个装满晶体管的 perfboard，用于切换段内的 LED 灯条。前面有一个 LED 闪烁秒，老实说，我们对这部分有点犹豫。如果它能淡入淡出，或者不那么令人分心，那就太好了，但当我们观看演示时，它确实让我们越来越喜欢。

我们喜欢这个时钟庆祝七段显示的方式，只希望它能大得多。如果你想做一个，STLs 和代码是可用的，尽管它们只涵盖了 7 段部分——基础是由泡沫板制成的。休息后，查看演示并制作视频。

你愿意听到时间在温柔的钟声中流逝吗？[这是一个使用旧硬盘驱动器的报时时钟](https://hackaday.com/2020/01/21/hdd-driven-chime-clock-is-quite-striking/)。
