# 超级坚韧的树脂简直和钉子一样坚韧

> 原文：<https://hackaday.com/2022/05/06/super-tough-resin-is-literally-as-tough-as-nails/>

树脂打印似乎仍然在黑客类型中形成两极分化的观点，一些人认为这种机器是完成正确任务的好工具，而一些人干脆直接拒绝它们。有许多支持和反对的论点，但像熔融沉积成型(FDM)机器一样，树脂打印机正在突飞猛进地发展——液体树脂本身也是如此。如今，低气味树脂很常见，颜色和表面处理也各不相同，现在由于一些专门的开发工作，这种印刷品的脆性问题正在得到解决。[Mayer Makes]设计了一种超级坚韧的“工程树脂”，他展示了这种树脂是如此坚韧，[你可以打印一枚钉子，然后把它锤成一块木头](https://www.youtube.com/watch?v=vbz6tPyOUkk)！(视频，嵌在下面，不信的话。)

这种特殊的树脂注定要混合，因为它的自然固化色调是一种绿灰色，但它确实有一个巧妙的技巧，在没有完全固化时呈现出明确的黄色色调，这非常有帮助。这在移除支撑结构时特别有用，因为您可以利用固化过程中的颜色变化来判断合适的时机来折断较厚的部分，从而将损坏印刷品的风险降至最低。最终的印刷部件也足够坚韧，能够承受随后的传统后处理，例如铣削，从而提供更大的最终精加工公差。试着用 FDM 印花做这件事。

树脂化学的一个巧妙之处在于，你可以简单地将它们以液体形式混合，自己调整树脂的特性，它们也可以用特殊配方的染料着色，而不会对其他特性产生太大影响，因此这种新的超级坚韧树脂给原型制造人员提供了树脂武库中的另一个工具。
