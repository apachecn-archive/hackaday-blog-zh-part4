# Nerf 枪械升级版

> 原文：<https://hackaday.com/2021/02/18/nerf-gun-upgrade/>

我们很多人都怀念童年的玩具，只要它们不像草坪飞镖，这种怀念往往会导致有趣的升级，因为我们中的一些人已经成年，拥有工业尺寸的空气压缩机。像超级酒鬼和 Nerf 枪这样的经典是特别受欢迎的改进目标，来自[Emiel]的这款 Nerf 机枪也不例外。

该建筑采用了一种 Nerf 发射球的玩具武器，并且基本上把它扔出了窗外，取而代之的是一种定制的 Nerf 发射球步枪。他从下面的接收器开始，制造了一个气动机械装置，将球装入弹膛，然后发射出去。这使得步枪既可用于单发模式，也可用于全自动模式。从那里，一个桶连同原料和其他收尾工作一起成型。

[Emiel]也使用高速摄像机来确定他的新 Nerf 枪的速度，但不幸的是，它的速度不够快，遭受了与有史以来最快的人造物体之一相同的命运，他的速度下限只有 400 公里/小时。如果你不想让你的 Nerf 建造速度太快，也许你应该[建造一些巨大的东西来代替](https://hackaday.com/2020/11/20/big-nerf-bazooka-packs-a-wallop/)。
