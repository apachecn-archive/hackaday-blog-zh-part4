# 修改墓碑焊机包含一个黑客主机

> 原文：<https://hackaday.com/2019/09/12/modified-tombstone-welder-contains-a-host-of-hacks/>

最先进的焊接机并不便宜，这是有充分理由的:以可控的方式推动这么大的电流，并在一整个工作日内完成这项工作，需要一些重型零件。不过，还是有便宜货的，尤其是最基本的机器:交流电焊机。人们熟悉的、名字恰如其分的“墓碑”焊机可以胜任这项工作，而且它们是学习如何铺设焊珠的一个很好的工具。

然而，墓碑也不是没有缺点，虽然其他人可能会在遇到这些限制时购买不同的焊机，但[Greg Hildstrom]决定[将他的交流焊条焊机改装成带 TIG 的交流/DC 焊机](http://www.hildstrom.com/projects/ac-225/)。他详细介绍了他对焊机所做的全套改装，从由三根延长线制成的新 50 A 电线组件，其中每根电线中的三根 12 号电线连接在一起，以形成更大的有效导体，到添加由微波炉变压器框架制成的整流器和扼流圈，以产生 225 A 额定功率的 DC 输出。在项目结束时，墓碑上堆满了黑客，包括一个自制的电压控制脚踏板，所有东西的新行业标准连接器，以及一个老式林肯“高频”控制器的帮助下，支持 TIG 或钨极惰性气体焊接。他的博客帖子展示了他用机器放下的许多测试珠中的一些，下面链接的视频播放列表显示了构建的亮点。

这不是[格雷格]第一次涉足热金属领域。几年前，我们报道了他的电弧炉建造，由另一个更有能力的焊工提供动力。
