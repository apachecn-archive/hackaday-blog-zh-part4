# 机器人篮球架 V2

> 原文：<https://hackaday.com/2020/05/19/robotic-basketball-hoop-v2/>

几周前，[肖恩·威顿]发明了一种篮球篮板，这种篮板使得即使离篮筐很近也不可能投篮不中。然而，作为一个被动的设备，篮板也有它的局限性。速度极快的投篮进不去，而且(像大多数篮板一样)它缺少面部识别软件。所以他开始着手开发解决这些问题的第二个版本，并在游戏中扮演更积极的角色。

这个版本是平面的，在游戏开始之前看起来很不起眼。平板篮板是机械化的，包括一个摄像头，所以当篮板移动到一个位置将球送入网内时，可以实时分析传入的投篮。或者，由于它确实包括面部识别，篮板总能将球送离篮筐，确保[肖恩]无论对手投篮多少次，都能赢得篮球比赛。

如果你没有机会看到原图，[我们不久前曾特别报道过这个图](https://hackaday.com/2020/04/20/a-basketball-hoop-that-never-lets-you-brick/)，当你了解到在制作这个图形的过程中进行了多少分析时，这真的是一个奇迹。新版本更令人印象深刻，实时进行所有的数学运算，我们迫不及待地想知道[谢恩]下一步会有什么。
