# 动漫灵感全息虚拟助手

> 原文:[https://hack aday . com/2022/11/22/anime-inspired-holographic-virtual-assistant/](https://hackaday.com/2022/11/22/anime-inspired-holographic-virtual-assistant/)

[Jessp]创造了一个非常可爱的 DIY 虚拟助手，名叫 Maria 。该建筑结合了 3D 打印的外壳，使用现代的“佩珀的鬼魂”幻觉来呈现虚拟的三维动漫灵感助手，可以接受命令来获取天气信息，播放音乐或设置定时器。

该中枢装有一个 Raspberry Pi 4B 和一个 3.2 英寸的 LCD HDMI 屏幕，平板安装在它的背面，使用从 [Pepper's Cone](https://github.com/roxanneluo/Pepper-s-Cone-Unity) 项目借来的技术来呈现经过透视校正的“Maria”角色。聚碳酸酯板形成一个圆锥，允许渲染虚拟助理模型的 3D 效果。消费级迷你 USB 麦克风用于接收语音命令，消费级 USB 扬声器用于音频反馈。虚拟助手将文本语音服务卸载到谷歌云，并使用天气 API 和 Spotify 开发者账户来提供音乐选项。

所有源代码都可以在[Jessp]的 GitHub [页面](https://github.com/jessp/holo-assistant)上获得，包括构建说明和外壳的 STL 文件。我们过去曾推出过开源语音助手，包括[迈克罗夫特](https://hackaday.com/2015/09/23/echo-meet-mycroft/)，甚至还有一款 [HAL-9000](https://hackaday.com/2022/05/24/hal-9000-becomes-a-helpful-voice-assistant/) 虚拟助手(运行卡利俄珀)，但很高兴看到这一领域的进一步实验。
