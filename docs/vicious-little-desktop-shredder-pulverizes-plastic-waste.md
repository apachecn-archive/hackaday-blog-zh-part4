# 邪恶的小桌面粉碎机粉碎塑料垃圾

> 原文：<https://hackaday.com/2022/05/15/vicious-little-desktop-shredder-pulverizes-plastic-waste/>

我们可能都看过巨大的工业粉碎机的视频，它们把发动机缸体当午餐，并吐出一股细小的金属碎片。这些咀嚼金属的怪物的原始力量确实令人生畏，它们似乎是小型塑料粉碎机 SHREDII 的灵感来源，这种小型塑料粉碎机用于在家中回收塑料废物。

SHREDII 并没有那么大，这一事实并没有降低它的危险性，至少对比发动机组更小更软的东西来说是这样，比如手指。粉碎机的核心是一个六边形轴，带有多个激光切割的钢板刀片。旋转叶片沿着轴间隔开，因此它们嵌套在固定叶片的底座之间；旋转公共轴产生切碎塑料所需的剪切和切割动作。

在粉碎机的第一个版本中，每个刀片都有两个钩形齿，整个切割头由相对较厚的钢制成。当由 NEMA 34 步进机驱动时——公认的奇怪选择，但这是他们可以快速获得的东西——通过 50:1 的行星齿轮箱，碎纸机肯定完成了任务。碎片有点太粗了，所以第二版的刀片用了更薄的钢，并给了旋转刀片更多的齿。差别是巨大的——更精细的碎片适合于他们自制的直接进料注射成型机 in jekto。

在桌面制造过程中使用塑料的闭环有很多可以说的，SHREDII 和 INJEKTO 的团队致力于帮助家庭游戏玩家有效地再利用塑料废物。虽然这都是好的，但让我们面对它——看着像这样的碎纸机咀嚼塑料的奇怪的满足感，就像它甚至没有足够的理由建造这样的东西。
