# 带着露齿的键盘和时尚的设计，这个合成器展示了一切

> 原文:[https://hack aday . com/2018/10/28/with-Lingling-keyboard-and-sleek-design-this-synth-shows-it-all/](https://hackaday.com/2018/10/28/with-grinning-keyboard-and-sleek-design-this-synth-shows-it-all/)

*有型！*是一款可穿戴式音乐合成器，结合了精巧的设计和基于触控笔的操作，产生了一个巨大的卡车司机风格的皮带扣，可以播放电子音乐。有了 PCB 键盘和 LED 包围的嵌入式扬声器，就像一个咧着嘴笑的眼球，[时尚！肯定有独特的外观。](https://hackaday.io/project/160859)其他合成器设计可能功能更多，但风格肯定不会更多。

![](../Images/25abc058850ae5ad1263cce496e98535.png)该设备的触控笔和 PCB 按键接口类似于 [Stylophone](https://hackaday.com/2010/09/23/analog-stylophone/) ，但【Tim Trzepacz】增加了许多声音合成功能，以及流畅的设计和 LED 反馈，所有这些都与电池电源和集成的扬声器和耳机输出捆绑在一起。它可能最初被认为是皮带扣，但*时尚！*当然可以给会议徽章设计一个机会。

显示的照片是一个渲染，但原型正在使用铣削 PCB 和 3D 打印的情况下进行。[蒂姆]的[谷歌图片库](https://photos.app.goo.gl/Ampxw5hUv5ACdvGE6)有一些很好的正在进行的图片，展示了原型制作过程和一些测试，而[的 GitHub 库](https://github.com/softegg/Stylish-Trucker-Belt-Synthesizer)保存了所有的设计文件，如果有人想更近距离地了解一下的话。*有型！*是入选 2018 Hackaday 奖乐器挑战赛部分的[二十名决赛选手之一，因此也是角逐大奖的众多项目之一！](https://hackaday.com/2018/10/16/these-twenty-projects-won-the-musical-instrument-challenge-in-the-hackaday-prize/)
