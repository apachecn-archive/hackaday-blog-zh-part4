# 卫生纸加热器是中国特有的奢侈品

> 原文：<https://hackaday.com/2022/01/14/toilet-paper-warmer-is-a-unique-chinese-luxury/>

[Handy Geng]生活在中国保定，那里的冬季平均气温可低至 7.7 摄氏度(18.1 华氏度)。他认为他可以做得更好，而不是简单地在使用浴室时冻死。于是就有了他那相当独特的[卫生纸加热系统](https://www.youtube.com/watch?v=Cj9W4UEklIw)。

该建筑使用煤气灶加热炒锅。卫生纸通过电动滚筒被送入锅体，这些滚筒是从一个看起来像是旧的伪钞检测器中抢救出来的。然后用第二个马达摇动炒锅，以便更均匀地加热里面的卫生纸。然后，可以关掉燃烧器，打开锅盖，以便取出烤卫生纸。

该系统将卫生纸加热到 75 摄氏度(167 华氏度)；有点太热，摸起来不舒服，但幸好卫生纸没有太多的热量，所以冷却相对较快。令人欣慰的是，它远低于纸的自燃温度 451 华氏度。

这是一台嘈杂、铿锵作响的机器，但却提供了你可能更熟悉的来自办公室复印机的同样的温暖和舒适。[这也不是【Handy 耿】的第一次浴室黑客](https://hackaday.com/2021/11/20/the-pros-and-cons-of-a-power-flush-toilet/)。休息后的视频。
