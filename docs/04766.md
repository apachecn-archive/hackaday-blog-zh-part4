# 3D 打印杂志驯服了贴片胶带怪兽

> 原文:[https://hack aday . com/2019/11/07/3d-printed-magazines-tame-the-SMD-tape-beast/](https://hackaday.com/2019/11/07/3d-printed-magazines-tame-the-smd-tape-beast/)

很有可能你的商店里有一个盒子或箱子，里面装着成卷的 SMD 元件带。如果你幸运的话，线圈在某种程度上包含在他们的导电聚酯薄膜袋中；如果你更像我们，磁带四处松散，试图将它们绑在一起。无论是哪种情况，[这些 3D 打印的 SMD 杂志](https://hackaday.io/project/168275-smd-component-magazines)都会给混乱带来一点秩序，让电路板组装变得稍微容易一点。

当我们看到[Robin Reiter]的构建时，我们认为这些将是某种拾放机的磁带。但是，虽然它们看起来确实可以适应自动化的 PnP 设置，但[Robin]的主要目标是为松散的磁带提供有组织的存储。每个料盒都有一个圆形储槽来容纳成卷的胶带，前端有一个出口槽和一个楔块，用于将盖带引向相反的方向。这将移除盖带以暴露元件，将其从拾取区域清除，并且作为奖励，只需向后拉盖就可以使元件带前进。每个弹匣都有一个弹簧闩，夹在一个看起来有点像 DIN 导轨的底座上；加重底座可容纳多个料盒，便于设置手动取放环节。下面的视频展示了所有的细节。

对于某些性格类型的人来说，这真的很难接受。我们喜欢模块化的设计，这种设计给我们的商店带来的组织将会真正有助于清理一些东西。如果[Robin]将这一设计提升到一个新的水平，添加像这样的会很有用。
