# 科技纹身追踪两个维度

> 原文：<https://hackaday.com/2019/01/05/tech-tattoos-trace-two-dimensions/>

柔性电路板会弯曲，就像你从扑克牌中看到的一样，而皮肤会像针织物一样伸展。因此，制作电路板和临时纹身的规则需要有所不同。不仅仅是暂时的纹身，皮肤上也有[回路](https://sci-hub.tw/10.1002/adma.201801852)所以请不要有不受控制的热痕迹。除了弯曲和拉伸，这些纹身可以应用于不平坦的表面，并保持完整。电路可以添加到项目的外部，或者使用该结构作为电路板，以减少重量和尺寸。卡内基梅隆大学的软机器实验室和科英布拉大学的系统和机器人研究所的研究表明，这两者都是可能的。

这些电路是对现有方法的改进，现有方法依赖于用放大镜、镊子和稳定的手切掉金属箔。取而代之的是，在用共晶镓铟(在室温下是液态金属)涂覆迹线之前，用喷墨打印机打印银颗粒。如果我们过于简化，我们可能会将其描述为类似于我们也在[DIY OLED](https://hackaday.com/2014/03/02/making-oleds-in-the-kitchen-sink/)中看到的无毒汞等价物。从医学到角色扮演，这一发展可能会在一系列领域引起人们的兴趣。
