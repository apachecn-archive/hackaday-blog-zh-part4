# 有腿的机器人装上轮子滑走了

> 原文:[https://hack aday . com/2020/12/26/legged-robots-put-on-wheels-and-skate-away/](https://hackaday.com/2020/12/26/legged-robots-put-on-wheels-and-skate-away/)

我们不知道从发明轮子到有人把轮子放在脚上之间过了多长时间，但我们预计这是一个伟大的发现时刻:结合了快速滚动的能力和我们的腿快速适应不断变化的地形的能力。现在我们有了各式各样的休闲轮式鞋，接下来呢？教机器人滑冰怎么样？IEEE Spectrum 对苏黎世联邦理工学院的 Marko Bjelonic 的采访描述了许多研究小组中的一个在这个问题上取得的进展。

对于我们许多人来说，我们第一次看到在主动关节腿末端的动力轮上滚动的机器人是在几年前波士顿动力公司“手柄”项目[的镜头浮出水面的时候](https://hackaday.com/2017/02/27/i-am-science-fiction-incarnate-i-am-handle/)。在各种各样的地形上上下滚动，偶尔跳跃，它的运动能力在机器人界引起了不小的轰动。但是当 Handle 作为一种商业产品推出时，它的工作是……[在仓库里堆放箱子](https://www.bostondynamics.com/handle)？太令人失望了。仓库的地面相当平坦，使得手柄的灵活性没有得到充分利用。

波士顿动力公司通常对他们的机器人开发细节守口如瓶，所以我们可能永远不会知道 Handle 背后的完整故事。但是他们确实完成了让更多的人思考相关的控制问题。即使对人类来说，我们也面临着一个非同寻常的学习曲线，上面布满了伤痕累累、偶尔断裂的身体部位，这甚至发生在我们开始给车轮施加动力之前。所以有很多问题需要解决，产生了一系列描述机器人如何掌握这种运动模式的研究论文。

更令人兴奋的是，这正成为一个现实正在赶上虚构的领域，因为有轮腿的机器人已经被想象成类似于《T4》中的*幽灵的 Tachikoma 的形式。虽然这些虚构的机器人激发了从[乐高创造](https://hackaday.com/2012/04/03/working-tachikoma-brings-the-manga-to-life)到 [28 伺服野兽](https://hackaday.com/2016/06/03/hexapod-tank-from-ghost-in-the-shell-brought-to-life/)的各种项目，但它们的轮子和腿的运动还没有像这一代研究机器人那样自主协调。*

随着控制算法在世界各地机器人研究实验室的成熟，我们相信我们会看到轮腿机器人在其他领域找到应用。这个概念太酷了，不能放在仓库里堆箱子。
