# Pi 的机器人控制器拥有令人印象深刻的功能列表

> 原文:[https://hack aday . com/2019/11/05/robotics-controller-for-pi-saws-impressive-feature-list/](https://hackaday.com/2019/11/05/robotics-controller-for-the-pi-boasts-impressive-feature-list/)

[迈克尔·霍恩]最近分享了他对 **RedBoard+** 的想法，这是一种针对机器人应用的 Raspberry Pi 电机控制器板。他给忙人的简短版本是:[如果你对机器人感兴趣，买一个吧，因为它太棒了](https://www.recantha.co.uk/blog/?p=19827)。

[![](../Images/840172e0ad963c23c062327e4bf74a6e.png)](https://hackaday.com/wp-content/uploads/2019/10/RedBoard-board-closeup.jpg)red board+本质上是一个电机控制器，但它挤满了 I/O 和功能，使其高于通常的价格。它可以驱动两个 DC 电机和多达 12 个伺服系统，但更有用的是 7-24 V 的宽输入范围及其供电和控制底层 Raspberry Pi 的能力。用户可编程按钮默认为重启或安全关机，具体取决于按住按钮的时间。另一个巧妙的功能是能够使用板载 RGB LED 闪烁显示 Pi 的 IP 地址，这在必要时总是很方便。

RedBoard+有一个 [GitHub 库](https://github.com/RedRobotics/RedBoard)，它提供了各种测试脚本和一个易于使用的库，以及各种连接指南和快速入门指南。甚至有一个预先配置的标清图像，供那些喜欢直接进入的人使用。
