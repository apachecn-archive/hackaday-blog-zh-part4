# 这个机器人可能是拯救回收的方法

> 原文：<https://hackaday.com/2019/04/13/this-bot-might-be-the-way-to-save-recycling/>

回收至少是纸上谈兵，这是一件好事。将废物转化成新的可用材料通常比挖掘更多的原材料更有效。然而不幸的是，分类这些废料是一个劳动密集型的过程。随着中国实施废物进口禁令，世界突然发现很难找到地方接受其废物进行再加工。为了帮助解决这个问题，麻省理工学院的 CSAIL 小组开发了一种回收机器人。

该机器人旨在减少对人类分拣员的依赖，从而提高回收作业的可行性。这是通过一种新颖的方法来实现的，这种方法使用根据材料硬度和导电性进行分类的特殊致动器。这种致动器被称为手动剪切增力器——一种在拉伸时宽度会扩大的致动器。通过使其中两个彼此相对，它们可以夹持各种物体，而不必像传统的刚性夹持器那样担心方向或夹持力。使用压力传感器来确定材料的挤压程度，使用电容传感器来确定电导率，可以将材料分类到纸张、塑料和金属箱中。

[研究论文详细概述了抓手的发展](http://lillych.in/files/Chin-2019-robosoft.pdf)。人们小心翼翼地建造一些足够健壮的东西来处理回收环境，并且能够处理分类任务。将这一概念验证推向商业可行阶段还有很长的路要走，但这是解决资源难题的一个有希望的开端。

麻省理工学院的 CSAIL 是有趣项目的温床，开发了从视觉麦克风到图像识别系统的 T2 伪装等各种东西。休息后的视频。
