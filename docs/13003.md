# 非常别致的指甲实际上是一个秘密的珠宝藏匿处

> 原文:[https://hackaday . com/2022/03/10/very-fancy-nail-is-a-secret-jewlery-stash/](https://hackaday.com/2022/03/10/very-fancy-nail-is-actually-a-secret-jewlery-stash/)

通常，钉子是专门建造的东西，用来将木头固定在一起，它们的整个设计都集中在这个目的上。然而，[W&M·莱弗沙]却走了另一条路，精心制作了一枚非常奇特昂贵的钉子，我们只能将其解释为他们技艺的精湛展示。

建造从一根铜管开始，在自动车床上刻上精致的图案。清理后，螺旋线条在抛光黄铜上很有吸引力。然后在管子的末端做一个塞子，把塞子锉成一个类似钉子的点，隐藏塞子和管子之间的接缝。

然后将管子穿上螺纹，以接受拧入顶部的钉子头，让“钉子”充当一个别致的小藏物，[W&M·列夫莎]通过在里面放置一个手镯来炫耀。这个项目是通过制作一个漂亮的木箱来装这个别致的钉子来完成的。
