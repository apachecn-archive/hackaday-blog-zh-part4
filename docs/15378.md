# 逆向工程揭示电动汽车充电器有安全感

> 原文:[https://hack aday . com/2022/11/16/reverse-engineering-reveals-ev-charger-has-a-sense-of-security/](https://hackaday.com/2022/11/16/reverse-engineering-reveals-ev-charger-has-a-sense-of-security/)

随着越来越多的电动汽车进入市场，建在停车场、公寓甚至私人住宅中的充电站的数量也将相应增加。这种情况发生得越多，我们就会看到越多的充电器，在它们的设计中，安全性充其量只是一个事后的想法。

但是正如[这个电动汽车充电器拆卸和逆向工程](https://www.mnemonic.io/resources/blog/reverse-engineering-an-ev-charger/)所显示的，事情不一定非要这样。充电器是一个 [Zaptec Pro](https://zaptec.com/en/charging-station-zaptec-pro/) 站，可以达到 22 千瓦，分析是由【哈里森·桑德】和【安德烈亚斯·克莱松】完成的。这些只是未来十年可能会广泛安装的充电器类型，令人惊讶的是，它们的数量很少。[Harrison]和[Andreas]发现了一对 PCB，一个用于电力电子设备，一个用于控制电路。后者支持许多连接选项，如 4G、WiFi 和蓝牙，以及一些 RFID 和电力线通信。有两个微控制器，一个 PIC 和一个 ARM Cortex-A7。

尽管有 ARM 芯片，但电路板似乎缺少一个明显的 JTAG 端口，虽然一些未填充的焊盘最终有一条 UART 线，但不可能有 shell 访问。板载微型 SD 卡插槽似乎是一个明显的攻击目标，他们尝试的一些 Linux 映像至少产生了部分启动，但在不知道板上具体硬件配置的情况下，这只是在瞎猜。这时 NAND 闪存芯片从主板上弹出来转储固件，这允许他们提取 devicetree 并构建一个自定义引导加载程序，最终拥有 root。

这篇文章有很多关于漏洞的有趣细节，以及他们在进入后发现的情况，比如即使你有出厂设置的蓝牙 PIN，你也不能免费充电。所以总的来说，这是一个非常好的安全设置，即使他们能够通过转储固件进入。这让我们想起了我们的朋友[Hash]在方法和结果方面所做的智能电表逆向工程。
