# 速度运行 Windows 95

> 原文:[https://hackaday.com/2019/03/09/speedrunning-windows-95/](https://hackaday.com/2019/03/09/speedrunning-windows-95/)

速跑是一种尝试在尽可能短的时间内击败电子游戏的练习。有各种各样的方法和风格。有 100%完成的加速跑，工具辅助的加速跑，以及不择手段尽可能快地到达游戏屏幕的加速跑。现在，安装 Windows 95B 只需 1 分 10.9 秒，创下了世界纪录。

当前的最佳尝试是在 Google Sheets 文档中收集的[。到目前为止，竞争对手很少，但我们预计未来会有更多的活动。目前的世界纪录竞赛规则要求使用原始软盘和光盘图像，但对硬件没有限制，所以随着时间的推移，纪录应该会下降。所有的顶级时间都是在虚拟机中完成的，但我们希望看到在原始硬件上进行的尝试。](https://docs.google.com/spreadsheets/d/1gJkoQ-_3MS2FP7aUlO5v4pQrhrwy0XjQvuojKlnre48/edit#gid=0)

这一切都是从[oscare zek]厌倦了试图在传统游戏竞速赛中竞争开始的，所以发明了一个新的类别来代替。比赛已经从最初的 4 分钟跑了很长一段路，竞争对手现在正在考虑使用 RAM 磁盘等先进技术来加快他们的速度。目前所有的击键都是手动的，但是我们可以看到一个工具辅助的竞赛在未来开始。

我们以前见过速度跑技术被推到令人印象深刻的极限——[像在《超级马里奥世界》里为了好玩而跑乒乓。](https://hackaday.com/2014/01/10/teaching-mario-to-play-pong-and-snake-through-innumerable-exploits/)休息后的视频。
