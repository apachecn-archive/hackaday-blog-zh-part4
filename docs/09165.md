# 劳拉信使做它最好的黑莓印象

> 原文:[https://hack aday . com/2021/02/05/lora-messenger-does-best-blackberry-impression/](https://hackaday.com/2021/02/05/lora-messenger-does-its-best-blackberry-impression/)

虽然事实上的智能手机设计最终走向了不同的方向，但不可否认的是，经典的黑莓布局提供了一些引人注目的优势。这是一个主要设计用来发送和接收电子邮件和短信的小工具，它显示。那么[【MSG】会按照它的形象](https://www.msglab.co/project/lo-ra-msg)建造他的口袋大小的劳拉信使真的有什么好奇怪的吗？

[![](../Images/68e28726e55d74402766aff3f9db7ecf.png)](https://hackaday.com/wp-content/uploads/2021/02/msglora_detail.jpg) 当然，他确实有一些帮助。沟通者使用[arturo182]，[的键盘羽翼，它将多余的黑莓 Q10 键盘放在定制的 PCB](https://hackaday.com/2019/06/20/the-feather-fauxberry-is-now-a-real-thing/)上，旨在接受 Adafruit 羽毛系列的电路板。[MSG]最终将他的与羽毛 M4 配对，因为他想与 CircuitPython 合作，并附带 900 MHz LoRa 羽毛翼。他指出，将他的代码切换到 Arduino 风格的 C 语言将允许他使用集成了 LoRa 的羽毛 M0；这一改变可以让他把这个小玩意做得更薄一点。

在 3D 打印的外壳内，他为 3.7 V 1800 mAh 的袋状电池腾出了空间，应该可以提供足够的运行时间。还有一个带 uFL 尾纤的外置天线，用于连接收音机。[表壳用热固嵌件](https://hackaday.com/2019/02/28/threading-3d-printed-parts-how-to-use-heat-set-inserts/)固定在一起，这应该会让它更加坚固，足以应付一些冒险。

[MSG]说硬件版本的轻微变化意味着他的 STLs 可能需要一点调整来适应你的组件，并警告说他的代码基本上只是他在网上找到的例子的混搭，但他仍然为任何想[伸出手接触某人而没有所有讨厌的基础设施的人分享这些商品。](https://hackaday.com/2020/04/25/a-lora-im-me-for-the-end-of-the-world/)
