# 没有单个转速脉冲轮的 STM32 转速脉冲轮风琴

> 原文：<https://hackaday.com/2019/12/05/a-stm32-tonewheel-organ-without-a-single-tonewheel/>

在[Laurent]漂亮的[音轮风琴中，你可能会惊讶地发现根本没有任何音轮。](http://www.pomad.fr/node/47)

音轮是产生电子琴声音的早期方法:通过[旋转不同频率的齿轮](https://hackaday.com/2019/11/07/rock-n-roll-with-3d-printed-tonewheels/)并以某种方式超越信号，就有可能合成一系列声音。我们喜欢想象它们仍然在[Laruent]的器官中，尽管非常小，但事实是它们完全是在 STM32 微控制器上合成的。

建筑本身很漂亮，看起来非常专业。我们不知道可以购买定制合成器的键盘，但是 FATAR 的一个模型是展览的中心。里面有一个 MIDI 编码器板和一个 Nucleo 开发板，用一个定制的 PCB 连接在一起。UI 是一个瞬时编码器轮和一个来自 [Mikroelektronika](https://hackaday.com/2016/04/26/belgrade-experience-mikroelektronika-museums-and-fpga-computing/) 的显示器。
