# 动感合成烤肉串雕塑依次播放朋克

> 原文：<https://hackaday.com/2021/10/04/kinetic-synth-kebab-sculpture-plays-punk-sequentially/>

有什么比雅达利朋克主机合成器更好的？四个雅达利朋克控制台合成器怎么样？还有什么更好的方式来展示他们，但作为铜丝艺术雕塑。如果[iSax]停留在四个铜丝合成器上，我们会原谅他，但他用他的运动雕塑将事情带到了另一个层面，它具有机械序列器的双重功能。被称为[Cyclotone——机械朋克控制台音序器](https://hackaday.io/project/181880-cyclotone-the-mechanical-punk-console-sequencer)，它以木头、黄铜、刷子和 555 个计时器为特色。你可以在休息时间下面的视频中看到演示。

如果你不熟悉雅达利朋克游戏机，它是一种电路，在 1980 年首次出版的 Forest Mims 的“工程师笔记本:集成电路应用”中首次被描述为“声音合成器”。它在单个芯片 556 中使用了两个 555 定时器。后来被称为“雅达利朋克控制台”，电路经受住了时间的考验，仍然很受各种黑客的欢迎。

[iSax]的构建增加了一个序列元素，允许合成器自动播放。合成器在一个方形销钉上相互成 90 度角，由雕塑底部的旋钮控制的步进电机以可变的转速转动。

每个合成器的两侧都有一个换向器，与回收的旋转工具刷接触，通过六边形黄铜支架提供动力。每个合成器都有自己的扬声器和控制，并有自己的分段数字，当每个合成器播放时，离散的 LED 就会亮起。

我们为[iSax]出色的执行和富有想象力的建筑喝彩，它成功地将电路雕塑、纯动态雕塑、经典电子设备甚至闪光灯结合在一起。如果你喜欢这个构建，你也应该去看看一个自由形式的雅达利朋克控制台构建和[另一个内置于操纵杆的](https://hackaday.com/2012/11/07/atari-punk-stick-puts-a-synth-in-a-joystick/)。如果你遇到任何吸引你的项目，一定要通过[热线](https://hackaday.com/submit-a-tip/)让我们知道！
