# 黑客日链接:2022 年 8 月 21 日

> 原文：<https://hackaday.com/2022/08/21/hackaday-links-august-21-2022/>

就旁道攻击而言，这是我们听说过的最奇怪的攻击之一。但是本周的科技新闻充斥着关于珍妮·杰克逊的“节奏之国”实际上是一种网络攻击的故事。这听起来有点可疑，但显然这是一个老漏洞，因为它是在笔记本电脑普遍配备 5400 RPM 硬盘的时候首次发现的。当在笔记本电脑上播放该特定歌曲的视频时，该漏洞浮出水面，很快就会崩溃。附近的同类笔记本电脑也会受到影响，这表明无论是什么使机器崩溃都与软件无关。事实证明，歌曲中的一些频率在驱动器中引起了共振。不清楚当时是否有人问了重要的问题，比如这首歌的哪一部分应该负责，或者驱动器的故障模式是什么。我们只是猜测，说这是驱动器头弹出和锁定。

说到安全，在美国民事准备通知系统紧急警报系统(T1)中的一个漏洞的新闻在 DEFCON 之前传出。下面视频的第一部分涵盖了这个漏洞，它涉及[一个品牌的 EAS 编码器/解码器](https://www.digitalalertsystems.com/EAS_pages/prod_r189se.html#)，这是一个所有广播公司都必须在他们的演播室里拥有的盒子，它可能会让攻击者忽略合法的警报或发出错误的警报。[来自国土安全部的最初警告](http://content.govdelivery.com/accounts/USDHSFEMA/bulletins/3263326)敦促广播公司在 DEFCON 之前修补其固件，那时漏洞细节将会公开。我们很好奇这个花哨的盒子到底是什么，但从[一张后面板照片](https://www.digitalalertsystems.com/EAS_pages/product_large_view/r189SE_back.html)来看，这似乎只是一个装在机架上的小型商用电脑，有几个 GPIO 连接器和一些天线插孔。用户手册确实说明了这台电脑运行的是 Linux，所以至少它不是 Windows XP，但是我们可以很容易地看到一个像这样的盒子被藏在某个地方的控制台里，然后被遗忘了。挺吓人的。

我们经常觉得，如果你想知道未来会是什么样子，就要注意科幻小说中的小细节。科幻作家特别擅长推断当前趋势，并利用它们来为自己的作品增添趣味。当我们读到用蘑菇制作假培根的新尝试时，这种想法浮现在脑海中，这将会引起任何读过《T2》系列的人的共鸣，因为几乎所有的食物都是用蘑菇制作的。有道理；蘑菇在没有太多阳光的情况下生长良好，可以在堆肥废物中存活，这使它们成为资源受限环境中太空物种的理想食物来源。陆地上的假熏肉工厂将在大型垂直农场中种植巨大的真菌菌丝体；油、调味料和颜色会使切片的菌丝体看起来和吃起来有点像培根。老实说，这听起来很可怕，但并不比《T4》中提到的一些事情更糟糕。有人要蘑菇威士忌吗？

最后，我们可能都看过阿波罗计划中那些标志性的阶段分离镜头，并接受它们的存在，它们是惊人工程壮举的非常酷的人工制品。我们只是有点想当然地认为，太空飞行是高度仪器化的事件，布满相机和传感器的火箭捕捉每一个可交易的时刻——看看你，SpaceX。但是在 60 年代呢？没有那么多——我们几乎没有尼尔·阿姆斯特朗第一次登上月球的视频。那么，我们是如何获得这些令人敬畏的工程视图的呢？Fran Blanche 有答案，它的简短版本是电影摄影机，它们在降落伞降落时将有效载荷降落到地球上，被飞机钩住。视频中有很多简洁的技术细节，包括用于拍摄液氧罐内部的光纤。结尾的原始舞台分离镜头也非常令人放松。
