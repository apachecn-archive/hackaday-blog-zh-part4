# Hackaday 播客 016:钢的 3D 打印，膨胀泡沫的成型，成套零件和老化焊料

> 原文:[https://hackaday . com/2019/04/26/hackaday-podcast-016-3d-printing-with-steel-molding-with-expanded-foam-quip-package-parts-and-aged-solder/](https://hackaday.com/2019/04/26/hackaday-podcast-016-3d-printing-with-steel-molding-with-expanded-foam-quip-package-parts-and-aged-solder/)

加入编辑埃利奥特·威廉姆斯和迈克·斯奇斯的行列，回顾本周的硬件黑客活动。这一集着眼于使用 Shrinky Dinks 的微流体，膨胀泡沫建造飞机机翼，时间对组件焊点的潜在影响，以及 3D 打印中使用的气枪 BBs。在这一集的结尾，我们采访了在深圳电子市场成功创业的两兄弟。
