# 电子刺绣生日卡是技能的庆典

> 原文：<https://hackaday.com/2021/02/01/electronic-embroidery-birthday-card-is-a-celebration-of-skills/>

谈到送礼，黑客和制造商有时会觉得自己吃亏了。你在外面为人们制作深思熟虑的、复杂的电路雕塑、有用的软件或口罩，你得到了什么回报？好吧，是的，通常是礼品卡或工具或其他能激发你创造力的东西。但是该死的，偶尔收到一份手工制作的礼物也不错，不是吗？

因此，你要做的是:与足够多的其他创客交朋友，让你觉得他们是你的生日双胞胎，或者足够亲密，让你们都能感受到你们共享的个人假期的温暖。[然后你让他们同意和你交换手工制作的生日礼物](https://beckystern.com/2021/01/27/glowing-birthday-cake-embroidery-for-estefannie/)。这或多或少发生在[贝基·斯特恩]和[埃斯特范尼]之间，他们似乎通过在 YouTube 上分享项目的魔力找到了彼此。

[Becky]给[Estefannie]的礼物是一个繁忙的创客元素的交叉，包括平面设计、刺绣、电子和 3D 打印。[Becky]从刺绣开始，这多亏了一个叫做 PEmbroider 的新开源处理库。完成后，她 3D 打印了框架，并添加了电子设备——生日蛋糕的蜡烛闪烁发光二极管，以及通过拧入框架侧面的触摸触点可以访问的一些歌曲。[Becky]添加了一个实时时钟模块，所以它会在[Estefannie]的实际生日那天播放一些额外的歌曲。

这里最深思熟虑的因素是个性化，当你把 100%的自己投入到 100%关于别人的事情中时，会发生惊人的事情。艺术的每一点都是[Estefannie]个人的，建筑的每一个原子都是纯粹的[Becky]。看看演示和制作视频，看看休息之后[Estefannie]为[Becky]做了什么。

[Becky]多样的创造力曾多次为这些页面增光添彩。[看她如何告别 2020](https://hackaday.com/2021/01/27/goodbye-2020-contest-winners-announced/) ，[打造日常肯定镜](https://hackaday.com/2020/05/28/begin-your-day-on-an-uplifting-note-with-a-daily-affirmation-mirror/)，疫情前期给[做面膜大师课](https://hackaday.com/2020/05/02/mass-mask-making-masterclass/)。
