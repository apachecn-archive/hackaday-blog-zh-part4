# 老 Kindle 显示 HDMI 视频，最终

> 原文：<https://hackaday.com/2021/01/08/old-kindle-shows-hdmi-video-eventually/>

即使最近独立面板的价格有所下降，拿起二手 Kindle 仍然可以说是获得大型电子纸显示器的最具成本效益的方式。尤其是当你考虑到 Kindle 包括电池、外壳和驱动显示器的电子设备时。当然，让 Kindle 软件随心所欲会带来独特的挑战，但只要稍加调整，旧的电子阅读器就能如你所愿地重生。

一个典型的例子是【Brendan Sleight】的 [OkMonitor 项目。通过使用有点令人眼花缭乱的软件和硬件组合，他找到了一种方法，可以将一台旧的 Kindle Paperwhite 变成即插即用的 HDMI 显示器。是不是很棒的显示器？远非如此。顾名思义，你能从这个解决方案中得到的最好的东西就是一个 OK 显示器。但至少这是件好事。](https://barwap.com/projects/okmonitor/)

OkMonitor 的幕后发生了很多事情，[Brendan]通过项目页面上的幻灯片描述了这些事情。但高层次的想法是，一个带有简单 USB HDMI 捕获设备[的树莓 Pi 4](https://hackaday.com/2020/12/21/heavy-raspberry-pi-user-keep-an-hdmi-to-usb-capture-device-around/)获取视频输入，并将其转换为适合 Kindle 的缩小格式。转换后的视频通过 WiFi 传输到越狱的 Kindle 上，由本地视频播放器播放。在休息后的视频中，你可以看到最终的结果看起来非常令人印象深刻，即使有相当大的延迟。

尽管[Brendan]为 OkMonitor 做了演示，但我们不能说我们会在这个场景下看很多电影。但事实上，你可以将任何 HDMI 设备插入“基站”，并将视频发送到一个或多个 Kindles，这无疑令人印象深刻。这绝对值得仔细看看，即使你只是拿了这个项目的一些概念，让你自己的 Kindle 重新利用的想法离开地面。

我们最近看到在重新利用大型电子纸价格标签方面取得了一些有希望的进展，但很难想象这种利基设备会比二手 Kindles 便宜。[随着软件的持续开发](https://hackaday.com/2018/07/25/hope-xii-a-foss-operating-system-for-e-readers/)，这些旧的电子阅读器很可能在黑客中仍然很受欢迎。
