# 阿基诺钟就像它听起来那样

> 原文：<https://hackaday.com/2020/02/21/arkanoid-clock-is-exactly-what-it-sounds-like/>

时钟可以是一个让你知道当前时间的简单装置，但它也可以有更多的含义。它可以表达过去的审美理想，也可以是流行文化试金石的回归。甚至可以两者兼而有之，[来自【维克多·塞拉诺】的辉煌的阿卡洛克。](https://hackaday.io/project/169655-arkaclock-retro-arcade-arkanoid-with-clock)

当[Victor]希望创造一款老式街机风格游戏时，这个项目就开始了。为了在有限的硬件上工作，就像先驱者一样，他决定使用 PIC18F86K22，RAM 不到 4KB，程序空间只有 64KB。他连接到一个 256 x 64 的绿色有机发光二极管屏幕上，开始用汇编语言重现 Arkanoid。

完成后，[Victor]注意到复古外观的展示相当令人愉快。在这一点上，该设备被重新设计成了一个时钟，程序生成了一个当前时间形状的阿卡努等级。然后人工智能会玩游戏，在关卡改变前的每一分钟摧毁砖块。

这是一款出色的手表，非常适合挂在任何独立游戏工作室的墙上。其他复古游戏也可以制作伟大的时钟。休息后的视频。
