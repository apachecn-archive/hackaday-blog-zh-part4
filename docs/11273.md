# 最小磨坊:米纳米尔

> 原文：<https://hackaday.com/2021/09/10/minimal-mill-the-minamil/>

拥有几台可以随意使用的机床是一种奢侈品，不是很多人都买得起，而且往往是很贵的。这是我们大多数人梦寐以求的事情，然而，利用一些常见的工具和便宜的电子设备，一些人已经组装了一些非常便宜的数控机床。最新的是 Minamil，它使用旋转工具，并将其绑在一个经济的框架上，以获得一个功能性的数控铣床设置。

这个项目拥有令人印象深刻的低成本，每个轴大约 15 美元。每个轴都使用容易获得的零件，如轴承和螺杆，这些零件很容易安装在磨机中，对于切割头来说，构建是基于 Dremel-like 旋转工具，其价格标签也很低。让我们不要忽视所使用的基本上免费的配重。

对于控制，一个带 CNC 屏蔽的 Arduino 驱动三轴设备，这可能是这个项目的大部分成本。[Paul McClay 还指出，他建造这座建筑所需的很多材料都可以从旧打印机之类的东西中回收，所以 45 美元的价格标签是天花板，而不是地板。

Minamil 已被证明能以极高的精度铣削多种材料。丙烯酸和铝都可以用这台机器加工，但[Paul]也展示了它加工 PCB 的能力。它确实有一些局限性，但就价格而言，它似乎是无与伦比的，即使与他以前改造旧 CD 驱动器的 CNC 相比也是如此。
