# 在 OpenSCAD 中设计的微型四轴飞行器

> 原文：<https://hackaday.com/2021/04/02/micro-quadcopter-designed-in-openscad/>

四轴飞行器是梦幻般的东西，现在有各种各样的风格，从轻盈的轻量级赛车到价值数万美元的工业级拍摄设备。来自[masterdezign]的 Beatle-1 以较小的规模出现，[它的主体完全是用代码创建的。](https://www.instructables.com/Micro-Quadcopter-in-OpenSCAD/)

为了创造 Beatle-1，[masterdezign]使用了 OpenSCAD，这是一个 3D 建模程序，它使用代码而不是可视化工具来生成几何图形。因此，通过一系列布尔运算、挤压和旋转，一个基本的轻型四轴飞行器框架就在几行文本中创建出来了。然后，这只是一个简单的 3D 打印零件的工作，连接四个 Olimex F1607 马达，连接一个飞行控制器，这个小无人机就可以起飞了。

Beatle-1 不仅是一个有趣的飞行玩具，也是将 OpenSCAD 建模技术应用于现实世界的一个很好的例子。那些想自己动手的人可以在 Thingiverse 上买到零件。 3D 打印的无人机框架很受欢迎，[我们以前在这些零件周围见过一些。](https://hackaday.com/2018/01/15/3d-printing-a-better-quadcopter-frame/)休息后的视频。
