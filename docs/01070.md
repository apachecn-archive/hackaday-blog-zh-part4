# TouchYou:可穿戴触摸传感器和刺激器

> 原文：<https://hackaday.com/2018/11/04/touchyou-wearable-touch-sensor-and-stimulator/>

我们中的一些人可能永远不知道另一个人的触摸，但 Hackaday Prize 中的这个项目可能就是解决方案。这是[莱昂纳多]关于可穿戴设备的想法，允许任何人通过互联网发送触觉和多感官刺激。它是通过互联网与人接触，是的，这项技术今天就在这里。

在 TouchYou 内部是一个连接到蓝牙模块的 Arduino Pro Mini。该 Arduino 与力传感器和触摸传感器通信，并且还具有用于小型振动马达的输出。有了蓝牙模块，TouchYou 就变成了一个物联网，能够与世界各地的其他 touch you 进行通信。这是一种相互关联的、世界范围的感人体验，也是我们见过的最好的人机交互的例子之一。

像这样的项目需要大型触摸传感器，如果你不知道，这些有点贵。没关系，因为[莱昂纳多]想出了一种廉价制造大型柔性触摸传感器的方法。这个过程很像你在家里制作 PCB 用激光打印机打印出设计的两面，然后用铜箔和 Kapton 层压板包裹起来。接下来，只需在氯化铁中进行一点蚀刻，然后小心地将柔性 PCB 连接焊接到细导线上。

从一个伟大的想法到一些令人印象深刻的 DIY flex PCBs 制作工作，这是 Hackaday Prize 中较好的项目之一，并被提名为人机界面挑战赛的决赛选手。
