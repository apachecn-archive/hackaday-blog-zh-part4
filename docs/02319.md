# 每辆拉达有三个引擎

> 原文：<https://hackaday.com/2019/03/09/three-engines-for-every-lada/>

如果你不住在前东欧国家，很可能你从未见过拉达在你的社区里行驶。这种车在俄罗斯及其邻国随处可见，不过，这是有充分理由的:价格。拉达给了许多人负担得起的交通工具，否则他们就会步行，但这种低廉的价格意味着[它也是一些优秀汽车黑客的一个伟大平台](https://www.youtube.com/watch?v=zSVJFfVXip0)。

俄罗斯汽车商店[Garage 54]的伙计们给其中一辆经典折扣车配备了两台额外的发动机。这超出了你通常看到的从日常驾驶中获得适度马力的普通螺栓改装。工作人员不得不焊接一个伸出汽车前部的框架，以承受所有额外的重量，并制造所有需要的零件，以使每个发动机上的曲轴相互连接。在此之后，这是“简单”的工作，调整引擎，使所有的行为与对方。

这个视频真的很值得一看，因为赛车也升级了背面的双重设置，并配备了防滑轮胎，以增加在冰面赛道上的抓地力。很有可能这辆车不合法上路，所以这可能是他们唯一能开它的地方。其他东西也可以用拉达车制造，比如[割草机](https://hackaday.com/2016/07/15/this-motorless-pull-behind-mower-is-made-from-junk/)。
