# ARM 编程示例

> 原文:[https://hackaday.com/2022/10/11/arm-programming-by-example/](https://hackaday.com/2022/10/11/arm-programming-by-example/)

ARM 处理器到处都是。从 Raspberry Pis 到手机，再到类似蓝色药丸 Arduino 的主板，如今你不必走很远就能找到 ARM 处理器。如果你用 C 语言编程，你可能不会太在意，甚至不会去想它。但是你知道 ARM 汇编语言吗？嗯，如果你从一个角度来看，这不会太难。CPU 只有大约 30 种不同的操作——这就是它被称为 RISC 的原因。当然，有时更少的指令实际上会使事情变得更困难。但是你可以在 [ARM Assembly by Example](https://armasm.com) 网站上获得包含 21 个程序的入门教程。

你需要一个 32 位的 ARMv6 或者更好的——所以 Raspberry Pi 可以在这里工作。当然，编译器是 gcc 和所有相关的工具。如果您有合适的硬件，也有关于使用浮点单元和 NEON 协处理器的章节。

前几节可能是你所期望的:程序基础和内存映射教程。但是在“你好世界”的强制性停留之后，你会发现像“寻找水獭”和“魔法与爱”(见下面的视频)这样的节目来挑战你的新兴技能。

如果你对汇编语言感兴趣，尝试一下 ARM 是一个简单的方法。如果你对 64 位 Intel/AMD CPU 更感兴趣，[我们可以帮助](https://hackaday.com/2016/06/14/linux-assembly-required/)。如果你想去裸机，[我们也去过](https://hackaday.com/2022/05/08/free-your-pi-with-this-bare-metal-programming-environment/)。
