# 3D 打印滑板车拉链左右

> 原文：<https://hackaday.com/2021/09/15/3d-printed-scooter-zips-around/>

乘坐个人电动汽车在市中心逛逛很有意思，但如果你真的喜欢詹姆斯·迪茨(James Dietz)并骑着自己的 3D 打印电动滑板车(T1)就更好了。作为 Hackaday 奖的参赛作品之一，RepRaTS(可复制快速原型运输系统)的目标是为踏板车做最初 RepRap 项目为 3D 打印所做的事情:提供一个用户友好的设计基础，您可以扩展、修改和维护。它甚至不需要电动工具来建造，当然，除了你的 3D 打印机。

这种设计使用螺纹杆和特殊的塑料垫片来承受大负载。原型是故意过大的大型轮毂电机，理解大多数建设可能会更小。正如你在下面的视频中看到的，滑板车似乎跑得很快，操控也很好。

[詹姆斯]警告说，滑板车还不安全使用。有许多电池安装和安全问题需要修复。我们期待看到最终版本。

3D 打印的零件都在 PETG，0.8 毫米的喷嘴，0.4 毫米的层高。填充物是相当适度的 30%网格，不需要支撑。拉紧螺杆至关重要，并且有关于所需的扭转和胶合步骤的具体说明。
