# 一名程序员正在将 Portal 移植到任天堂 64 上

> 原文：<https://hackaday.com/2022/05/18/one-coder-is-porting-portal-to-the-nintendo-64/>

当*门户*在 2007 年问世时，开发者 Valve 选择不在一款早已停产的过时任天堂游戏机上发布这款开创性的游戏。当然，当时没人在意，但(詹姆斯·兰伯特)是来纠正这个错误的。是的，[他正在把*门户*移植到 N64](https://github.com/lambertjamesd/portal64) 。

这个港口，或者[詹姆斯]称之为“德马克”，已经建设了一段时间。该项目带来了一些挑战: *Portal* 是为比 1996 年的任天堂 64 功能强大得多的个人电脑开发的。因此，最初的担心是控制台将无法处理[游戏的物理](https://www.youtube.com/watch?v=v2A_s0paqfc)或渲染递归门户图形。

然而，努力得到了回报。[詹姆斯]一点一点地改进他的引擎。最新的作品很好地渲染了门户，配套的立方体也如你所料。还有一个可见的门户枪，当通过镜像门户查看时，该引擎甚至可以渲染 15 个递归层。十六岁太多了。

当然，还有很多事情要做。还没有玩家模型，基本的动画和声音也欠缺。然而，核心概念是存在的，看着[詹姆斯]掠过不太圆的入口绝对是一种享受。更好的是，它甚至可以在任天堂原装硬件上流畅运行。这是一个值得称赞的壮举。

当我们展示他在 N64 硬件上创造实时阴影的作品时，我们不知道[詹姆斯]有什么打算。现在我们知道了！休息后的视频。
