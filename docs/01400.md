# 亨利·胡佛获得武器升级

> 原文：<https://hackaday.com/2018/12/01/henry-the-hoover-gets-a-weapons-upgrade/>

在这个军费开支空前的时代，我们已经习惯于看到各种作战硬件的武器升级——坦克、直升机、攻击机，应有尽有。我们不太习惯在家用电器上看到同样的东西。[不管怎样，我们现在有亨利·胡佛包装一些严重的热量。](https://www.lookmumnocomputer.com/projects/#/henry-just-got-an-upgrade/)

最初只是一个真空吸尘器，亨利通过两个电机和变速箱获得运动，这两个电机和变速箱来自一个儿童乘坐的车辆。一个由铜管制成的容器用来储存可燃气体(看起来像是丁烷，就像在打火机中使用的一样)，排放由一个电磁阀控制。然后由一对继电器点火的电子点火器进行点火。这一切都由一个标准的业余无线电控制器控制，[所以你可以站在一个安全的距离，同时燃烧你的地毯](https://www.youtube.com/watch?v=FlsWttQcrnc)。

这是一个危险的项目，但考虑到亨利茫然和开心的表情，这个项目就特别有趣了。不过话说回来，你可能喜欢把喷火器戴在手腕上。休息后的视频。
