# 反作用轮几乎控制了这架不寻常的无人机

> 原文：<https://hackaday.com/2019/01/24/reaction-wheels-almost-control-this-unusual-drone/>

当你考虑到保持无人机稳定需要平衡的所有力时，这些奇妙的装置留在空中是一个奇迹。当产生这些力的唯一选择是吹更多或更少的空气时，很自然地开始寻找其他可能更好的方法来实现飞行控制。

受航天工业的启发，[汤姆·斯坦顿]决定探索控制无人机的反作用轮。这个想法很简单——在无人机上放置一对相对笨重的相互成直角的机动轮，利用它们加速时产生的力来控制无人机的俯仰和翻滚。[Tom]下面的视频给出了一个漫长而清晰的解释，解释了建造之前所涉及的物理过程，这导致了一个笨拙的飞行器，有点让人想起月球着陆器。无人机实际上管理了一些短暂的，有点稳定的飞行，但总的来说，反作用轮似乎不能胜任这项任务。[Tom]将此归因于他使用每个反作用轮电机的电流作为其扭矩的衡量标准，这并不完全适用于所有情况。他认为带有编码器的马达可能会做得更好，但在视频结束时，这种小型无人机并不完全适合继续进行实验。

当然，躲闪的反应轮不仅仅会给无人机带来问题。当太阳变得烦躁不安时，它们也可能成为航天器的一个问题。
