# 用廉价的气动压力机制造的 DIY 注射成型机

> 原文：<https://hackaday.com/2022/03/06/diy-injection-molder-built-from-a-cheap-pneumatic-press/>

[Kurt Schaefer]正在观看 YouTube 视频，视频中人们使用他认为是玩具的 3018 数控机器制作注塑模具，看着结果，他决定要分一杯羹。然而，一旦你有了模具，下一个显而易见的问题是缺少注塑机。但是这些东西很贵。幸运的是，你可以花不到 350 美元买到一台漂亮的气动压力机，再多花一点钱，[Kurt]发现他可以把它改装成一台功能齐全的注塑机(视频，嵌入在下面)，并从中获得一些不太好的结果。

在订购了易贝号上的印刷机后，最终到达的是一堆乱七八糟的东西，显然没有按照它的重量包装好，而且在运输过程中受到了一些损坏。尽管如此，看起来功能部分还不错，所以[Kurt]决定继续建造。第一个明显的变化是需要一个加热室来处理原料。使用由 [buster beagle 3D](https://busterbeagle3d.square.site/product/injection-molding-machine-mk2-chambers/8?cs=true&cst=custom) 生产的现成注射成型室，只有几个支座和一个支架需要加工以完成机械结构。常见供应商提供的普通 PID 控制器，用一些加热带包裹在腔室周围，处理注射温度要求，一些 3D 打印外壳整齐地包裹所有这些。

在一些最初的摇摆和一些对设计的修改之后，[Kurt]从这个简单的设置中得到了一些非常好的结果，并且它看起来非常可调和可重复，这将有助于保持这些结果的质量。简而言之，一个简单易得的部件，也许是你附近黑客空间的一个受欢迎的补充？

在 [Thingiverse 页面](https://www.thingiverse.com/thing:5264980)上有 3D 打印的零件，还有一个 [Fusion360 CAD 模型](https://myhub.autodesk360.com/ue2ad6f7d/g/shares/SH9285eQTcf875d3c539f33a31b69dd731c7)。零件的购物清单可以在视频描述中找到，如果你想试着复制它。

这些年来，我们已经看到了一些 DIY 注射成型的尝试，比如这个[光滑的塑料成型装置](https://hackaday.com/2021/07/07/a-plastic-injection-machine-you-can-use-at-home/)。这里有一个用 [3D 打印的模具](https://hackaday.com/2021/07/20/injection-molds-from-your-3d-printer/)，如果你只是需要一些合适的形状，你可以只[用热熔胶注射模具](https://hackaday.com/2019/06/11/injection-molding-with-a-hot-glue-gun/)。
