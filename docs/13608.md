# Hackaday 播客 168:数学使球体变平，FPGAs 模拟拱廊，我们不能动摇宝丽来图片

> 原文：<https://hackaday.com/2022/05/13/hackaday-podcast-168-math-flattens-spheres-fpgas-emulate-arcades-and-we-cant-shake-polaroid-pictures/>

加入 Hackaday 主编埃利奥特·威廉姆斯和特约撰稿人丹·马洛尼，他们将回顾本周的顶级黑客。这一次是一场真正的复古盛宴，C64(大部分)由新零件制成，Altoids Altair，并通过经典视频游戏学习 FPGAs。我们还研究了从旧设备中捕捉数据的 LCD 嗅探，重新想象了电阻颜色代码，重温了宝丽来即时相机的魔力，并沿着电视的记忆小路进行了一次旅行。但它不全是旧东西——有用数学平板包装球体，在 3D 打印零件上喷涂精细涂层，无 DRM 的标签打印机，以及看看你口袋里的智能手机里有什么——包括一些非常奇怪的光学元件。
