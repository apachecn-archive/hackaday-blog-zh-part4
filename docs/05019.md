# [本·克拉斯诺]造了一个质谱仪

> 原文：<https://hackaday.com/2019/12/04/ben-krasnow-builds-a-mass-spectrometer/>

使《科学美国人》杂志伟大的特色之一是一个名为“业余科学家”的专栏每个月，读者都可以在家里做实验，或者用廉价制造一些科学仪器。幸运的是，[本·克拉斯诺]的粉丝们还记得这个系列，并敦促他从这个系列着手制作:[一个 DIY 质谱仪](https://www.youtube.com/watch?v=nIKhUizkXxA)。(视频，嵌入休息下方。)

[Ben]刚刚发布了下面的视频，展示了历时五个月制作的铜管装置的早期实验；事实证明，分析粒子物理学并不像听起来那么简单。质谱背后的想法是电离样品，在离子通过磁场时加速离子，并测量粒子的偏转，作为其质荷比的函数。但是正如[本]发现的那样，把一个简单的原理变成一个可行的工具的细节是极其重要的。

他的实验装置使用从小心压碎的白炽灯中提取的灯丝来电离碘化钾氯化钾样品；涂在细丝上并干燥，当细丝被加热时，盐溶液被电离。离子流被高压电场加速，流过两个刀片形成的狭缝。检测器与发射器垂直放置，穿过强大的磁场，并连接有高增益跨阻抗放大器。使用老式的模拟仪表和大型自耦变压器，整个事情有一种伟大的疯狂科学家的氛围，这让我们想起了[他的单分量干涉仪装置](https://hackaday.com/2018/12/13/ben-krasnow-builds-a-one-component-interferometer/)。

[Ben]从钾样品中获得的数据与预期结果一致，该仪器几乎足够灵敏，可以辨别钾的两种不同同位素之间的差异。他承诺未来将升级质谱仪，也许包括样品的激光电离。我们期待着那一天。
