# 1200 个可寻址的发光二极管构成了完美的第一支舞

> 原文：<https://hackaday.com/2022/06/16/1200-addressable-leds-make-for-the-perfect-first-dance/>

新婚夫妇在婚宴上的第一支舞是一件既伤感又难忘的事情，所以为什么不用一些特别的东西让它更有意义呢？Hackaday 的校友[Brett Haddoak]和他的妻子[Rachelle]确实做到了这一点，他们用 1200 个可寻址的 led[将她的婚纱变成了迪士尼*睡美人*](https://bretthaddoak.com/2022/06/15/sleeping-beauty-wedding-dress-the-details/) 中奥罗拉公主变色礼服的真实再现。

传统规定新郎在大喜的日子之前不能看到礼服，因此发光二极管被固定在衬裙和紧身胸衣下面。该设计需要如此多的 led，以至于超出了 Arduino 所能处理的极限，因此需要两个 Arduino 来控制整体。电子设备和电池穿在一条 polo 短裤里，经过一些涉及片状连接的紧张时刻，整个事情变得生动起来。结果可以在休息时间下面的视频中看到，当然还有一个显著的惊喜因素！

我们希望这对幸福的夫妇未来一切顺利，我们希望这不会是他们最后一次这样的电子合作。如果你还想要更多，[这不是我们带给你的第一套亮灯婚纱](https://hackaday.com/2013/08/16/electronic-wedding-attire-for-a-geeky-wedding/)。
