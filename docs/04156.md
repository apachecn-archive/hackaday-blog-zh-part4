# 会议徽章为旋转式电话注入了活力

> 原文：<https://hackaday.com/2019/09/14/a-conference-badge-breathes-life-into-a-rotary-phone/>

这些年来，我们已经在很大程度上报道了会议徽章的惊人多样性，我们总是对其创作中的创造力和独创性感到高兴和惊讶。但最令人难过的是，这么多徽章在活动结束后都被放进了抽屉，再也没人碰过，这是多么错失的机会啊！

这是一个趋势，但[丹]已经扭转了，他的[旋转拨号电话带来了生活与电动势蒂尔达 MkIV](https://trandi.wordpress.com/2019/09/09/rotary-dial-revival/) 。这是去年 [EMF Camp 2018](https://hackaday.com/2018/10/05/electromagnetic-field-2018-event-review/) 的徽章，它的定义特征是[内置 GSM 手机](https://hackaday.com/2018/08/26/2018-electromagnetic-field-badge-its-an-entire-phone/)。我们当时没有对它进行全面的审查，因为它在活动中的 GSM 部分有问题，从负面角度展示这个令人惊叹的徽章是不公平的，但一旦我们把它带回家，就足够简单地把商业 SIM 卡放入插槽，并使用公共网络。

[Dan]的手机是通过他祖父找到的东欧型号。在内部，这是一个相对传统的设计，他通过一个去抖电路将几条蒂尔达的 I/O 线从表盘中接入。但是仅仅选择几条线路是不够的，因为它的扩展端口上的大多数线路都来自端口扩展器。他需要可以产生中断的输入，所以他从机载操纵杆上劫持了一对夫妇。他加入了 Python 代码，你可以在下面的视频中看到。值得注意的是，他还没有将音频连接到徽章上，所以这是一项正在进行的工作，但这仍然是一个有趣的项目。

旋转式手机在硬件黑客中占有特殊的位置，我们已经展示了包括他们在内的许多项目。这个[不是我们带给你的第一个 GSM 旋转电话](https://hackaday.com/2015/11/18/old-school-rotary-phone-gets-gsm-upgrade/)，别忘了[他们也可以通过蓝牙](https://hackaday.com/2019/05/04/rotary-phone-gets-bluetooth-upgrade/)通话。
