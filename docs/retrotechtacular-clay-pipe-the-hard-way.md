# 反技术:硬的粘土管道

> 原文：<https://hackaday.com/2022/05/31/retrotechtacular-clay-pipe-the-hard-way/>

浏览 YouTube 足够长的时间，你很有可能会遇到各种“它是如何制作的”类型的视频。随着煎锅、跑步机和牙线制造商的出现，毫无疑问，人们将会深入探究烟斗是如何制成的。方法因材料而异，但铜、聚氯乙烯、铸铁，甚至混凝土，制管厂的共同点是都采用高度自动化。对于像管道这样的商品，很难在功能上与其他制造商区分开来，所以价格是唯一的竞争方式。这意味着将成本削减到最低限度，也意味着尽可能多地解雇员工。

当然，情况并不总是如此，就像 20 世纪 80 年代,展示的那样，看看我是如何制作陶土管道、排水瓦和烟囱烟道的。制作一个简单的陶制烟斗所涉及的手工量是惊人的，从事各种工作的人手数量也是惊人的。该工厂位于爱尔兰莫纳汉郡的 Carrickmacross，靠近构成其产品原料的页岩露头。开采页岩并将其磨成粘土是这一过程中为数不多的机械化步骤之一；虽然管子的挤压本身也是机械化的，但是机器需要几组工人来装卸。

从挤压机出来后，进入管道的手工数量惊人，尤其是下水道管道。“阿姆斯特朗接点”是一种复杂的配件，用作下水道管道的清洗和检查口，它的创造非常有趣，尤其是因为几乎没有使用夹具，也没有进行测量。这是严格意义上的吸引眼球的东西，还需要技巧和几十年的经验。

我们喜欢这些记录片，它们记录了一些失传已久的制作方法。“手”系列是由爱尔兰公共广播公司 rté在 20 世纪 80 年代制作的，人们可以感觉到，即使在那时，远在当前的离岸外包和全球化浪潮开始之前，他们就知道他们正在捕捉垂死行业的最后时光。我们很高兴他们这么做了。
