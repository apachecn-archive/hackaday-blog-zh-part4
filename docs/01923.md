# 用 3D 打印模具浇注混凝土

> 原文:[https://hack aday . com/2019/01/27/用 3d 打印模具浇铸混凝土/](https://hackaday.com/2019/01/27/casting-concrete-with-3d-printed-molds/)

[Thomas Sanladerer]想要使用 [3D 打印技术为混凝土和石膏](https://www.youtube.com/watch?v=OykSwSCgrrQ)制作一些模具。他使用了一台带有柔性细丝的 delta 打印机，并在下面的视频中记录了他的过程。

如果你以前印刷过柔性细丝，你知道你需要一个有封闭路径的挤压机。[Tom]借了一台打印机，但是没有那种设置。第一步是用另一台打印机替换挤出机。

一些碎木头做了一个阳模的盒子。[Tom]使用了混凝土砂浆，因为他没有准备好真正的混凝土来做这个试件。然而，他的最终目标是使用混凝土，他在创作最后一件作品时使用了混凝土，这是一个直播标志，他在其他几个视频中记录了这一点。

第一次尝试出现了一些裂缝，看起来他可能太早脱模了。石膏的第二次尝试给出了更好的结果。[Tom]然后尝试在石膏上涂色，以获得不同的外观。在后续视频中，他提到了之前视频中的一些评论。

我们经常忘记混凝土很容易使用，而且能创造出好的物质。当然，你可以不铸造它，而是用合适的齿轮直接[打印](https://hackaday.com/2016/04/15/3d-printing-houses-from-concrete/)。
