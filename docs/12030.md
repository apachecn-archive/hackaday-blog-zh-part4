# 唤醒机器人轻轻地做它

> 原文：<https://hackaday.com/2021/12/03/wake-up-robot-does-it-gently/>

几百年来，人们都是在深夜躺在床上看书的时候睡着的。现在更糟糕的是，当我们开始打瞌睡的时候，我们会把手机放在脸上。至少它们不像书一样有尖角。虽然你可能不想和机器人分享你的卧室，但这个由[诺伯特·扎雷亚]设计的唤醒机器人可能就是让你保持清醒的东西。

它是这样工作的:伺服系统上的树莓 Pi 相机在眼睛高度徘徊，它所连接的 Pi 使用 OpenCV 来确定这些眼睛是睁开还是开始变得沉重。这个机器人还会说话——它使用 eSpeak 来介绍自己是一个旨在不让你睡觉的机器人。然后，当它发现你在打瞌睡时，它会用一种无聊的英国口音反复吟诵“醒醒”。

我们确信这个东西是设计来扇(诺伯特)耳光的，就像(西蒙·吉尔兹)的机器人闹钟一样，但是没有，那只长手指的手只是慢慢地摆动下来，轻轻地拍(诺伯特)的手臂(或者任何在拍打的手的路径上的东西)。休息后，查看简短的演示和构建视频。

你想被更温柔地唤醒吗？试试日光灯。我们有几十种存货，但是这一种逐渐变得像太阳一样明亮。
