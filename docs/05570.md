# 踮着脚走过郁金香狂潮

> 原文:[https://hack aday . com/2020/02/03/踮着脚穿过郁金香狂热/](https://hackaday.com/2020/02/03/tiptoe-through-the-tulip-mania/)

春天又回到了北半球，我们很快就会看到灿烂的郁金香从泥土中醒来，用鲜艳的色彩点缀着解冻的大地。这些春天的预兆与荷兰密切相关，但它们实际上原产于土耳其和中亚，直到 16 世纪才被带到欧洲。郁金香变得如此受欢迎，以至于市场达到了被认为是第一次投机性金融狂热，并在 1637 年严重崩溃。

[这款机电客厅游戏](https://imgur.com/gallery/KlhUbID)用荷兰黄金时代的另一件艺术品——手绘代尔夫特瓷砖来装饰郁金香，用来装饰壁炉。[BuiltByBlatt]用画笔在电脑上完成了他所有的 114 幅作品。玩这个游戏时，你将一个小球向一排不同分值的球洞滚动。每个洞都有一个光束探测器，所以树莓皮知道你得分。

还有一个旋转奖金洞，它会根据剩余的球数而变化。随着你的分数上升，郁金香泰特斯向右移动。看起来似乎很容易击中中间的 5 点洞，但瓷砖给了它一种水平弹球的感觉，使它的移动不那么可预测。穿上你的木屐，休息后再来看看。

我们喜欢机电装置发出的沉重的咔嗒声，尤其是老式的弹球机。你甚至可以把它们拆下来，做成其他游戏。
