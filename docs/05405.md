# RemoteXY 简化了 Arduino 控制

> 原文:[https://hack aday . com/2020/01/30/remote xy-simplifies-arduino-control/](https://hackaday.com/2020/01/30/remotexy-simplifies-arduino-control/)

[Labpacks]想造一辆用手机控制的机器人汽车。作为一个业余读者，你当然可以想象制造汽车。大多数人甚至可以编写一个手机应用程序来进行控制。但是你想吗？在大多数情况下，你最好专注于你需要做的事情，并使用现成的东西来做你能做的部分。在[Labpacks']案例中，他使用 Visuino 避免编写普通代码，使用 [RemoteXY 处理智能手机界面](https://labpacks.blogspot.com/2020/01/remotexy-controlled-arduino-elegoo.html)。

RemoteXY 是一个网站，允许您轻松地构建一个电话接口，该接口将通过蓝牙 LE、USB 或以太网(包括 WiFi)与您的硬件对话。有趣的一点是:尽管界面构建器是基于 Web 的，但服务声称界面结构保留在控制器上。操作用户界面时，无需与远程服务器交互，因此无需外部互联网连接。

系统支持 Arduino 和 ESP 控制器。手机端，可以用安卓或者 iOS。RemoteXY 网站有很多这样的例子。

我们知道还有其他方法可以做到这一点，包括自己动手。然而，拥有不同的选项是很好的，RemoteXY 拥有所有常用的控件，包括操纵杆、颜色选择器、关卡、图表等等。

我们用 [Blynk](https://hackaday.com/2016/12/02/blynk-with-joy/) 做了这个项目自己的版本。我们以前也见过 [Visuino](https://hackaday.com/2017/11/30/skelly-the-skeleton-is-a-scary-good-musician/) 。
