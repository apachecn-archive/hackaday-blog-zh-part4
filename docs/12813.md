# 独特的时钟是所有的手，没有拨号，并没有蠕虫

> 原文：<https://hackaday.com/2022/02/17/unique-clock-is-all-hands-no-dial/>

在过去，我们没有花哨的数字钟。不，我们有很好的模拟时钟，有一个大指针和一个小指针，如果你想知道时间，你必须看着时钟，找出每个指针指向的数字，或者说是指向的数字。这并不容易，我们喜欢这样。

所以现在，出现了一种模拟时钟，除了指针什么都没有，没有表盘，没有数字，只有指针。这种事怎么可能？线索就在钟的名字中:AKUROBATTO，以及下面的视频中，视频显示了钟在做它的事情时指针的杂技动作。系列不可思议时钟制造商[ekaggrat singh kalsi]显然对这种机制花了很多心思，它由指针和单独的底座组成。指针在一端连接在一起，由小型步进电机驱动。基座有两个对接区域，伺服驱动的爪子可以抓住手组件，要么在中心枢轴，要么在任何一只手的尖端。在转换点稍微移动一下，指针以一种令人惊讶的可读方式扫过小时和分钟。

因为尽管 AKUROBATTO 的设计很酷，但其内部却是另外一回事。有定制的滑环向电机和控制电机的 Arduinos 发送电力，传感器确定每只手的位置，以及为步进器定制的齿轮箱。底座上的锁定机制也值得研究——要做到这一点并不容易。

总而言之，令人印象深刻的建设。无论是在磷光屏幕上显示时间，还是在一片亮片中显示时间，看起来似乎[ekaggrat]对独特的时钟情有独钟。
