# 吉他拾音器增加了 MIDI 功能

> 原文：<https://hackaday.com/2021/12/06/guitar-pickguard-adds-midi-capabilities/>

作为一个从 20 世纪 80 年代就开始使用的标准，即使在今天，MIDI 仍然是音乐舞台上最主要的力量之一。它快速、灵活，为许多不同类型的电子仪器提供了业界公认的标准。通过 MIDI 的魔力，即使不是乐器的东西也可以变成音乐设备，如臭名昭著的香蕉键盘，它还允许增加标准乐器的其他功能[，如这把吉他，它在拨片保护罩中内置了 MIDI 接口](https://hackaday.io/project/182793-guitar-pickguard-wireless-midi-controller)。

[Ezra]是这种独特乐器的创造者，为他的吉他增添了不少功能。设置相当简单:12 根电线连接到拨片保护装置，这些装置被设置为电容传感器，并与半音阶上的音符相对应。他不用触摸板，而是用线来弯曲他不需要的音符。电线连接到吉他琴颈后面的 Adafruit Feather 32u4 微控制器，该微控制器也有几个选择器，用于改变设备创建音调的方式。他可以设置界面发出单个音符或连续播放音符，改变风格，可以改变它们的八度音阶，以及许多其他功能。

这个项目的目标之一是增加吉他演奏者在现场演奏时的多功能性，我们不得不同意这给了音乐家更广泛的能力，而不需要舞台上大量复杂或昂贵的设备。我们最近也看到了一些其他基于 MIDI 的构建，专注于现场表演，就像这个[允许乐队彼此保持同步](https://hackaday.com/2021/11/23/live-jam-kit-helps-electronic-musicians-stay-in-sync/)。
