# 用 DIY 荧光显微镜在新的光线下观察细胞

> 原文：<https://hackaday.com/2018/11/21/see-cells-in-a-new-light-with-a-diy-fluorescence-microscope/>

自从 17 世纪一位荷兰商人通过他的黄铜和玻璃装置窥视微观世界以来，显微镜已经有了漫长而丰富的 DIY 创新历史。这种 DIY 荧光显微镜是 DIY 道路上的又一步，可能会为业余科学家和生物黑客开辟一种强大的成像技术。

在荧光显微镜中，细胞用各种荧光染料处理，这些荧光染料可以被一种波长的光激发，并发射另一种波长的光。但是正如乔纳森·布姆斯泰德指出的，荧光显微镜的价格通常超出了生物黑客的承受范围。他的自制范围有点公平竞争。诀窍是使用 3D 打印零件来装备常见的数码相机——USB 显微镜、DSLR，甚至智能手机相机。激发是由一圈 Nexopixel LEDs 提供的，而一个可移动的支架上有一个过滤器，可以阻挡激发波长，但允许发射波长通过进入相机。他演示了这项技术，用荧光笔的荧光墨水染一些线，然后把它们放在一张薄纸上；在传统的明场模式下，这些线几乎消失在背景中，但在荧光下会跳出来。

诚然，光学系统并不完全符合实验室质量，并且显微镜目前仅设置为进行反射成像，而不是更典型的透射模式，在透射模式中，光穿过样品。不过，这很容易解决，反射模式仍然有用。我们之前已经看到过[荧光显微镜变得相当复杂](https://hackaday.com/2017/06/03/fluorescence-microscopy-meets-diy-fluid-management/)，但是这个简单的范围可能足以让一个生物黑客开始。
