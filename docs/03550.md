# 光致变色屏幕是一个有趣的时钟

> 原文：<https://hackaday.com/2019/07/12/photochromic-screen-makes-for-an-interesting-clock/>

时钟项目将永远是黑客的主食，给建设者一个伟大的方式来建立一些有用的东西，并表达他们的个人天赋。[Mosivers]正在进行他们自己的构建，并决定尝试一下，[创造一个带有光致变色显示屏的计时器](https://www.instructables.com/id/Photochromic-Glow-in-the-Dark-Clock/)。

时钟使用 Arduino Nano 来运行显示，连接到在 protoboard 上定制的 4 位 7 段显示器。通过使用紫外发光二极管并将它们放置在反应屏幕后面，就有可能创造出独特的显示器。该时钟可以与两种不同的屏幕一起使用:一种是由紫外线反应 PLA 细丝制成的光致变色显示器，当受到紫外线激发时会变成紫色，另一种是夜间使用的夜光屏幕。

这是一个简单时钟设计的有趣转变，白底紫字的数字肯定会引起好奇旁观者的惊讶。光致变色材料玩起来很有趣，可以让[鸡蛋](https://hackaday.com/2017/04/18/photochromic-eggs-not-for-breakfast/)和[玻璃](https://hackaday.com/2017/11/28/the-chemistry-and-engineering-of-diy-photochromic-glass/)看起来更有趣。休息后的视频。
