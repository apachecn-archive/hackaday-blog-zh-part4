# 为 ESP32 重现 MS Paint

> 原文:[https://hack aday . com/2022/01/27/recreating-ms-paint-for-the-esp32/](https://hackaday.com/2022/01/27/recreating-ms-paint-for-the-esp32/)

在 20 世纪 90 年代，当许多孩子第一次接触计算机时，Microsoft Paint 是他们最早的创意渠道之一。现在，[Volos Projects]将这个简单应用程序的乐趣带到了 ESP32 上更紧凑的格式中！

GUI 甚至比 MS Paint 的 Windows 3.1 版本还要简单，看起来更像早期 GUI 时代的东西。无论如何，人们可以像过去一样用块状颜色绘制简单的形状，用一对电位计来移动光标，用两个触觉按钮来选择工具和对画布进行更改。

该构建表明，即使是 1.3 英寸 240×240 的 TFT 显示器也可以显示一些迷人的彩色图形，实际上它与 80 年代末大多数计算机的分辨率相差不远。我们也希望看到软件获得更多的工具，比如喷罐和刷子，它们是 MS Paint 体验的关键部分。对于那些渴望自己玩 ES Paint 32 的[来说，代码是可用的](https://www.youtube.com/watch?v=At0v7X9Xutk)。

值得注意的是，尽管有些说法与此相反，但 Paint 女士并没有死。顺便说一下，如果你是一个受虐狂，[你甚至可以在每个人都喜欢的 Windows 捆绑艺术程序中编程。](https://hackaday.com/2018/09/13/a-whole-other-kind-of-graphical-programming/)休息后的视频。
