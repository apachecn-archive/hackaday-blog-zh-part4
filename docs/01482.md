# 降临节日历——现在是季节

> 原文：<https://hackaday.com/2018/12/08/advent-calendar-tis-the-cnseason/>

正确使用的话，数控机床是强有力的工具，但是在进入正式的生产工作之前，经常有必要测试一台新的机床。这个降临节日历是一个伟大的节日项目，旨在测试 CNC 的速度。

日历主要是由木头制成的。这是测试加工项目的最佳选择，因为与钢或铝相比，它更软，不太可能造成工具或机器损坏。日历底座首先是用立铣刀铣出来的，而 30 度 V 形钻头用来雕刻星期几。黄铜铜焊棒被用来制作日历标签的挂钩。

由于黑板漆和可移除标签的巧妙使用，日历可以重新配置为任何给定的年份和月份组合——以防你希望全年都有降临节日历。总的来说，这是一个很好的低强度加工项目，对孩子们来说也是一个有趣的工艺项目。

因为现在是一年中的这个时候，你可能也会喜欢这个闪亮的降临节日历。休息后的视频。
