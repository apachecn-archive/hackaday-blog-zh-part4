# 迷你火星漫游车运行在皮皮科 W

> 原文:[https://hack aday . com/2022/10/28/mini-mars-rover-runs-on-pi-pico-w/](https://hackaday.com/2022/10/28/mini-mars-rover-runs-on-pi-pico-w/)

美国宇航局的火星漫游者是启发了世界各地许多崭露头角的工程师的机器人。[Nikodem Bartnik]本人对它们有着特殊的喜爱，并着手打造自己的微缩版。

Raspberry Pi Pico W 是操作的大脑，充当微控制器和远程无线控制链接。该机器人使用四个麦克纳姆轮进行移动，每个轮子都有自己的电机。这使得机器人可以通过旋转不同配置的轮子向各个方向移动。在顶部，漫游者配备了一个由伺服系统控制的关节机械臂，这使得它可以拿起和放下东西。此外，顶部有一个 FPV 摄像头，可以传输视频，因此机器人可以远程驱动。这是通过 WiFi 实现的，这要归功于用 Python 编写的一些自定义控制代码。

这是一个在光滑表面上有惊人能力的机器人，因为麦克纳姆车轮允许扫射和其他普通车轮无法做到的运动。多亏了它的机动附件，拥有一个可以与环境互动的机器人也很有趣。
