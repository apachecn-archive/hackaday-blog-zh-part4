# 钻压件紧固件 101

> 原文：<https://hackaday.com/2022/04/17/drill-press-piece-fastening-101/>

将工件固定在钻床工作台上有哪些选择？雷克斯·克鲁格向我们展示了大量的 T1，你应该了解它们。他指出了普通 C 形夹钳的缺点，并展示了像常规钻床虎钳和重型版本这样的选项，甚至提供了工件倾斜机制，并指出夹钳上的 V 形凹槽这样的小细节有助于圆形坯料的工作。对于较大的作品，他推荐了一个不被看好的选择——木工木制手动螺丝钳，它与钻床的配合出奇的好。然后，他谈到了压紧式钻床夹具，这是他的最爱，尤其是当涉及到金属板或塑料等平板材料时。

作为对我们这些处理圆股的人的奖励，他展示了一个他为钻入其侧面而制作的 V 形块，以及圆股夹，通过仔细钻一对木制手动螺旋夹而制成，当你需要从顶部钻入销钉时。这个十分钟的视频对于那些没有掌握钻床工件紧固知识的人来说是必看的，它可以帮助你在没有皮肤的情况下提高你的钻孔技巧。

我们之前已经报道过一些巧妙的和非常规的钻件紧固选项，从这种被重新利用的自行车快速释放部件压住的 wise[，到](https://hackaday.com/2016/06/14/vise-quick-release-locks-down-your-parts-for-good/)[一种电磁 wise](https://hackaday.com/2017/09/08/your-work-wont-move-with-a-magnetic-drill-press-vise/) ，让我们的读者褒贬不一。
