# 浮士德号在 77 年后起死回生

> 原文:[https://hack aday . com/2022/03/09/the-fliegerfaust-roars-77 年后起死回生/](https://hackaday.com/2022/03/09/the-fliegerfaust-roars-back-to-life-after-77-years/)

随着他们在第二次世界大战中的胜利前景变得越来越暗淡，德国人开发了一系列稀奇古怪的“神奇武器”，他们希望这些武器能帮助扭转战争的趋势。虽然这些神童显然不足以确保对盟军的胜利，但他们中的许多人代表了武器发展的绝对尖端水平，在一些情况下最终成为重要的技术里程碑。其他的逐渐消失，变得默默无闻，有时没有更多的轶事证据来证明他们曾经存在过。

这些被遗忘的发明之一是 Fliegerfaust，一种便携式多管火箭发射器，用于对付低空飞行的攻击机。尽管在 1945 年有数千辆被命令去保卫柏林，但是只有不到 100 辆被生产出来，并且有一些关于有多少在战争中幸存的争论。但是这并没有阻止*野生武器研究&开发*的[【乔纳森·怀尔德】根据当代文件和蓝图建造该武器的功能复制品](https://www.youtube.com/watch?v=5fskM6iUSbA)。

建造发射器相对简单，因为它只不过是用一个手柄和一个简单的电点火器捆绑在一起的九根管子。诀窍是在 20 毫米(0.78 英寸)火箭本身，这是由尾部四个斜孔排出的废气稳定旋转。由于没有尾翼或主动制导，每枚火箭的路径都有些不可预测，但这也是最初火箭的真实情况。

[![](../Images/762c4054819249aa6aaafa4a9b368731.png)](https://hackaday.com/wp-content/uploads/2022/03/fliegerfaust_detail.jpg)

Note angled exhaust ports on each rocket.

历史记录不同意火箭实际上是如何发射的，一些人声称每个桶是按顺序点燃的，而另一些人声称火箭是齐射的。[Jonathan]决定设计他的点火器电路，这样所有的火箭将被同时触发，但由于推进剂的变化，它们以交错的速度离开发射器，如下面的演示视频所示。

从纯历史的角度来看，这个项目是一个绝对的胜利。考虑到这种武器最后一次被使用是在柏林战役期间，很有可能在这之前没有任何活着的人看到或听到它被发射。当然，我们永远不会知道它与真实的东西相比如何，但至少我们比几张 B&W 的照片能提供更多的背景。[乔纳森]说他目前正在写一本书，详细介绍了如何研究和建造他的复制品，我们很想看一看。

对于那些想知道这种努力的合法性的人来说，[乔纳森]一直与酒精，烟草，火器和爆炸物管理局(BATFE)保持密切联系，并确保在他的复制品的开发和测试过程中遵守所有地方，州和联邦法律。尽管火箭中没有爆炸弹头，但根据《国家火器法》,这种发射器已被登记为“破坏性装置”。联邦政府允许平民合法拥有这些武器，但具体的法律可能因州而异，所以不建议涉足自制神奇武器，除非你在快速拨号上有一个好律师。
