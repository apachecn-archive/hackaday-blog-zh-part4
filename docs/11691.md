# 建造一个通过降落伞着陆的水火箭

> 原文：<https://hackaday.com/2021/10/25/building-a-water-rocket-that-lands-via-parachute/>

水火箭很有趣，但是如果你对工程着迷，它们会更有趣。[The Q]就是这样一个实验者，他建造了一个双推力水火箭，甚至有一个着陆用的降落伞！

![](img/8655e209d8d30b8d8401fc893a14a2cd.png)

The testing took place in an area strangely reminiscent of a certain operating system.

双推力概念是一个有趣的概念，YouTube 频道[空军司令部火箭]对此做了很好的解释。基本想法是在水火箭上使用几个燃烧室，其中一个提供高加速度的初始短“助推”阶段，随后是第二个燃烧室的更长“维持”水平的加速度。

这是一个让水火箭飞得更高的好方法，但是这个问题并没有就此打住。该建筑还安装了一个来自小型步行玩具的发条模块，在水火箭场景中被通俗地称为“Tomy timer”。一根橡皮筋缠绕在计时器的输出轴上，保持装有降落伞的门关闭。发射时，卷紧机构被释放，其输出轴转动，最终释放降落伞。诀窍是设置计时器，在火箭完成推进阶段后立即释放降落伞。

这是一个整洁的建筑，对于那些渴望开始自己的先进水火箭兔子洞之旅的人来说，这将是一个很好的指南。我们以前也见过类似的工作。休息后的视频。
