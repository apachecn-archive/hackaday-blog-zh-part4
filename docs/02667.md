# 随机单词配对在这个不寻常的时钟上标记时间

> 原文:[https://hack aday . com/2019/04/15/random-word-pairing-mark-the-time-on-this-unregular-clock/](https://hackaday.com/2019/04/15/random-word-pairings-mark-the-time-on-this-unusual-clock/)

天哪，当数字计算器变得足够便宜，以至于小学生都可以带着它去上课时，我们有多开心。当颠倒看时，数字可以被解释为各种脏话的字母，这一发现是无尽欢笑的源泉。那是更简单的时代。

这个由四个字母组成的单词“时钟”旨在稍微重现那个异想天开的时间，除了可以完全控制七段显示屏，而且不需要倒过来看。这是从[以前制造的单词钟【威士忌 TangoHotel】发展而来的，并依赖于超过 1000 个四个字母的单词库，这些单词可以使用七段显示器合理显示，其中大多数是 SFW，但有些不是。PICAXE 用于选择四个字母的单词中的两个，大约每秒显示一次，从最宽松的定义来看，这是一个时钟。单词选择是伪随机的，由浮动 ADC 引脚的噪声产生，但下面视频中的一些单词配对似乎掩盖了非随机的幽默感。事实上，有超过一百万种可能的配对；添加两个和三个字母的单词可能会很有趣，看看会有什么样的欢乐。](https://hackaday.com/2012/08/30/word-clock-of-a-different-nature/)

虽然我们喜欢这里的*回到未来*氛围，但我们最近也看到了其他一些非常不错的单词钟。有[使用 PCB 作为字符](http://hackaday.com/2019/03/27/leds-shine-through-pcb-on-this-tiny-word-clock/)的遮罩，然后[一个背投字钟](http://hackaday.com/2019/04/02/a-word-clock-the-hard-way/)看起来真的很棒。
