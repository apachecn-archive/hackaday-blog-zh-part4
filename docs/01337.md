# 已经令人印象深刻的数控路由器得到一个额外的轴

> 原文：<https://hackaday.com/2018/11/25/already-impressive-cnc-router-gets-an-extra-axis/>

大多数 DIY 者经济能力范围内的数控机床通常是三轴的，有一个适中的工作包和一个似乎永远不够强大的主轴。当然，这并不是说我们不想在自己的商店里拥有这样一台机器，但是将小型机器与“大男孩”五轴工具进行比较可能会让家庭游戏玩家感到有些不足。

幸运的是，有一个不一定会倾家荡产的解决方案:[给你的 CNC 路由器](https://www.youtube.com/watch?v=mAZei3wpBe0)增加第四个轴。[这位老托尼]开始使用他的数控刳刨机——[，这是我们之前介绍过的](http://hackaday.com/2016/02/07/a-home-cnc-built-by-someone-who-knows-their-stuff/)的一种构造，非常令人钦佩——增加了一个机器主轴，让他可以像操作数控车床一样操作机器。下面的第一个视频涵盖了构建的机械部分，包括焊接和加工一个坚固的组件，以固定将四爪卡盘连接到 Lexium MDrive 的主轴，Lexium m drive 是一个集成了驱动器和反馈的步进电机，使其更像一个伺服系统。[老托尼]在之前的视频中介绍了如何在[将硬盘集成到 Mach4 中。](https://www.youtube.com/watch?v=NARodU-vVWs)

组装好的机器主轴看起来很结实，可以平稳地达到 3000 转/分，并有足够的保持扭矩，可以作为车床的索引头。下面的第二个视频显示了一些测试转向铝和钢；在主轴停转之前，我们惊讶地发现切割的力度如此之大。

不，它不是一台 Tormach 或 Haas 甚至不是一台 [Pocket NC](https://hackaday.com/2013/09/23/pocketnc-p5-takes-desktop-cnc-to-the-5th-dimension/) ，但它是对一台已经有能力的机器的一个很好的补充，我们期待着[老托尼]用它制造出什么。
