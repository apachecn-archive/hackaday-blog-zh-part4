# 用紫外线摄影像蜜蜂一样看

> 原文：<https://hackaday.com/2018/12/19/seeing-like-bees-with-ultraviolet-photography/>

当谈到看到奇怪的光谱时，大卫·普鲁奇是你想与之交谈的人。他在[长、中、短紫外光](https://hackaday.com/2017/07/24/diy-illuminator-for-uv-fluorescence-photography/)下拍摄了岩石的照片，他给消费类相机添加了[热成像](https://hackaday.com/2018/03/08/adding-optics-to-a-consumer-thermal-camera/)，他还让[相机看到了偏振](https://hackaday.com/2018/04/20/real-time-polarimetric-imager-from-1980s-tech/)。除了视网膜上的视杆细胞和视锥细胞所能看到的，这个世界还有更多，大卫是揭示这一点的最佳者之一。在今年的 Hackaday 超级大会上，David 谈论的是 DIY 紫外线摄影。蜜蜂是这样看的，这是蜜蜂的膝盖。

光的可见光部分只是光谱的一小部分；现在，有数公里长的无线电波在你周围飞行，希望不会有太多的伽马射线穿过你的头骨。在电磁波谱的可见光部分，我们有红外线和紫外线。红外可视化已经得到了很好的研究，自从 Raspberry Pi 基金会开始运送不带红外滤镜的相机以来，研究就更深入了。而紫外线就有点陌生了。这并不意味着它在自然界中不被使用；蜜蜂可以看到紫外线的深处，而花朵已经进化到蜜蜂可以看到。

但这是一个关于紫外线摄影的演讲，这意味着修改一个相机。这意味着拆开相机，取下红外和紫外滤镜。是的，所有相机对红外和紫外都非常敏感，大多数相机都有一个只通过可见光谱的滤光片。当然，一旦你去掉了红外和紫外滤镜，你就需要屏蔽掉红外和可见光。这可以通过简单地堆叠 UV 带通滤波器和 IR 短通滤波器来实现。专用的紫外线镜头非常昂贵，但照片放大镜头对紫外线几乎是透明的，一些旧的苏联和东德的优质镜头没有防紫外线涂层。

这是一个令人惊叹的演讲，深入到了 UV 摄影的深度，最终简单地通过堆叠滤镜产生 UV 图像，没有 Photoshop 的参与。当然，这只是一个引子；大卫[用 DSLR](http://www.prutchi.com/2016/07/02/new-book-on-ultraviolet-photography/) 写了一本关于紫外成像的书，快速浏览这本书会发现它远不止三十分钟的演讲能涵盖的内容。
