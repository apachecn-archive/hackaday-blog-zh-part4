# 用这种组合车床和电火花线切割几乎可以切割任何东西

> 原文：<https://hackaday.com/2022/07/20/cut-just-about-anything-with-this-combination-lathe-and-wire-edm/>

他们说，如果你有一台车床，你就有所有其他的机床。在某种程度上，这是真的——你可以在一台车床上制造几乎任何东西，包括另一台车床，甚至在其他机床上最好制造的零件通常也可以在必要时在一台车床上制造。但是在看到[这个 DIY 放电加工工具](https://youtu.be/czCgWDuHHOY)的车床附件后，我们可能倾向于将 EDM 视为统治所有工具的一个工具机。

现在，我们承认[BAXEDM]构建这个工具的工作可能有点做作。他想为他的瑞士军刀制作一些定制的六角刀片，看起来它们很容易在传统车床上用六角棒料制作。再者，淬硬钢是制造电火花线切割机的材料，而且似乎有许多附件的使用案例可以使工件相对于电火花切割线旋转。

这真的是这个建筑的诀窍——在水下旋转一部分。为了实现这一点，[BAXEDM]建造了一个平台来承载支撑标准 ER-25 夹头的轴承座，并带有一个支架来保持步进器远离 EDM 切割槽中的水。也有很多 3D 打印的绝缘体，以保持大多数附件与 EDM 电流电绝缘，加上像陶瓷轴承这样的奇特部件，它们在水下不会腐蚀。还有一大堆其他的考虑；[BAXEDM]在下面的视频中经历了漫长的迭代设计过程，并在大约 27:00 标记处开始使用他的新工具进行文字旋转。

如果您对 EDM 能实现的功能感兴趣，谁不想呢？——但是你需要更多关于这个过程的背景知识，[我们已经为你准备好了](https://hackaday.com/2019/03/14/ben-krasnow-drills-really-small-holes-with-electricity/)。
