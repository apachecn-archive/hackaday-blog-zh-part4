# 在一个基督教摇滚乐队的黑胶唱片上发现的秘密 C64 程序

> 原文：<https://hackaday.com/2019/11/24/secret-c64-program-found-on-a-christian-rock-bands-vinyl-record/>

你多久能在旧黑胶唱片中找到一次复活节彩蛋？

当罗宾·哈布朗(Robin Harbron)得知一个 Commodore 64 程序藏在基督教摇滚乐队“浪子”1985 年专辑中的一张唱片的一面时，他真是大吃一惊。YouTube 频道 [8 位展示和讲述](https://www.youtube.com/channel/UC3gRBswFkuteshdwMZAQafQ)的主持人展示了黑胶唱片一面上的“C-64”蚀刻，这是他在网上发现该唱片包含隐藏程序后发现的。

唱片上的引出槽通常是一个无止境的槽，防止唱机脱离唱片(除非有自动返回功能，它只是重放唱片)。在黑胶唱片的第一面，跳动的凹槽看起来正常，但在第二面，它有点厚，包含一些隐藏的音频。将音频录制到磁带上并加载到数据集上揭示了一个简短的 C64 程序。

这个过程要稍微麻烦一点，但是经过几次尝试后[Harbron]揭示了一个秘密信息，这是由阿尔伯特·爱因斯坦和耶稣基督提供的。这不是有史以来最令人印象深刻的程序，但 35 年前的程序员能够将它放入几秒钟的音频中，这是非常酷的。

不幸的是，我们将不会听到专辑中太多真正的音乐——为了避免 YouTube 上的版权问题,[Harbron]选择不播放这些歌曲。
