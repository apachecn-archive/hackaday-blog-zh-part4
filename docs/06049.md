# 带有坦克履带的自主 3D 漫游车统治着这片土地。几乎

> 原文:[https://hack aday . com/2020/04/02/autonomous 3d-rover-with-tank-tracks-rules-the-fields-almost/](https://hackaday.com/2020/04/02/autonomous3d-rover-with-tank-tracks-rules-the-fields-almost/)

在现实世界中，范围蔓延是一种真正的痛苦，但对于激情项目来说，它可能会产生一些有趣的后果。[rctestflight]在摆弄 3D 打印的漫游者变速箱，它变形为一个 [3D 打印的坦克建造](https://www.youtube.com/watch?v=UdL4M4JpdsU) 。

【rctestflight】的 [之前的自主漫游车](https://hackaday.com/2020/03/15/rover-runs-slow-and-steady-on-solar-power/) 项目在廉价的齿轮电机方面存在问题，他开始尝试自己的变速箱设计，以使用更低的 RPM / Kv 无刷无人机电机。坦克的出现是因为他想要一种简单的交通工具来测试他的设计。“简单”很快就过时了，最终产品是完全 3D 打印的，除了紧固件、轮轴、轴承和电子设备。

履带和齿轮很吵，但它运转得很好。在户外测试中,[rctestflight]确实发现轨道容易钩在藤蔓和树枝上，这在一个案例中导致它在铝轴弯曲后抛出一个轨道。添加了一个 Ardurover 导航系统，使用 32 Ah 电池能够自动运行一整天，并且令人惊讶的是，3D 打印的变速箱和轨道几乎没有磨损。所有的 STL 文件都在 [Thingiverse](https://www.youtube.com/redirect?redir_token=i5wLnXx-DD8IrCvhEnJ_eNKn9l58MTU4NTc1NjM4MUAxNTg1NjY5OTgx&v=UdL4M4JpdsU&q=https%3A%2F%2Fwww.thingiverse.com%2Fthing%3A4234076&event=video_description) 上，但【rctestflight】建议等待即将到来的更新，因为他在休息后拍摄视频时发现了设计中的缺陷。

对于稍微复杂和昂贵的火星车，请查看我们对美国宇航局 2020 年火星探测器[](https://hackaday.com/2020/02/13/mars-2020-rover-curiositys-hi-tech-twin-is-strapped-for-science-includes-a-flying-drone/)的报道。
