# 飞机罗盘拆卸

> 原文:[https://hackaday.com/2021/05/30/aircraft-compass-teardown/](https://hackaday.com/2021/05/30/aircraft-compass-teardown/)

在我们看到下面的[大卫的]视频之前，我们不知道什么是 C-2400 LP，但事实证明它非常有趣。这个设备是一个飞机罗盘，在更换它之后，他决定为我们把它拆开。事实证明，就像航海罗盘一样，这些设备需要根据周围的金属进行调整。虽然船上的指南针有巨大的钢球来实现这一目的，但小巧轻便的航空指南针必须更加节俭。

这个代表罗盘补偿器的小装置——通常被称为开尔文球——几乎就像一个机械表。微小的齿轮和棘轮，都是黄铜的。很明显，这个设备相当可靠，因为这个上面的日期是 1966 年。

黄铜补偿装置被剥离，但你仍然可以购买单位约 80 美元，所以有可能修复设备。然而[戴夫]在他的驾驶舱里放了一个更现代的指南针。当然，如今你实际上不需要一块浮动磁铁来制作指南针。

事实证明，地球的[磁场不是一个静止的目标](https://hackaday.com/2019/03/06/ask-hackaday-earths-magnetic-field-shifting-rapidly-but-who-will-notice/)，并且——最终——磁极将再次翻转，或者至少科学家们是这么说的。如今，你可以在[的一个小包装](https://hackaday.com/2015/06/18/neopixel-ring-compass-takes-things-in-a-new-direction/)里买到指南针和其他仪器。
