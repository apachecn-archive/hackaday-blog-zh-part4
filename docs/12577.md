# 用食盐和火箭科学拯救火星殖民者

> 原文：<https://hackaday.com/2022/01/24/saving-martian-colonists-using-table-salt-and-rocket-science/>

想象一下，你是早期火星殖民地的一员。你被困住了，唯一能把信息带回家的方法就是发射一个远高于地面的无线电。更糟的是，你没有火箭！正是这个思想实验促使[Thoisoi2]尝试只用普通火星殖民者可用的成分和方法制造火箭发动机。在休息时间下方的视频中可以看到他选择的方法。

如果你跳过了火箭学 101，快速复习一下可能会有帮助:火箭的工作原理是在一个封闭的室内燃烧燃料，然后以高速向一个方向排出。为了让燃料燃烧得更快(从而为愤怒的一端增添更多活力)，燃料中添加了一种叫做氧化剂的补充物。它为燃料的燃烧创造了一个富氧环境。这和氧丙烷火炬比丙烷本身燃烧的温度高是一个道理。

The Sugar Powered Rocket Motor says “Boom!”

首先，被困的火星人需要火箭燃料。如果你还记得 1999 年的电影《十月的天空》,四个高中生用蔗糖作为燃料。你可能还记得，这些往往会得到所有的解释。这种易变性导致[thoi so 12]放弃糖作为燃料，转而使用任何火星殖民者都可以获得的燃料，但不太可能导致快速的计划外拆卸。

氧化剂呢？十月的天空，男孩们用氯酸钾做实验。这通常用于火箭，但对于普通的火星殖民者来说可能更难获得。但是，事实证明，可以从食盐中提取的氯酸钾和氯酸钠效果相同。然而，这比那要复杂得多。

简单地添加盐和燃料并不能制造出火箭发动机。细微差别、科学和化学都在[Thoisoi2]整理的精彩视频中展现出来，我们相信你会像我们一样喜欢它。

你还将发现我们被困的火星人是否能回家，或者他的土豆种植是否是徒劳的。

我们也想呼应视频中的警告:这是一个相当危险的实验，所以不要在家里尝试！绝对先去人家家试试。或者在火星表面。
