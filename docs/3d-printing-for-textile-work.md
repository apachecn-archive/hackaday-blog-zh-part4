# 纺织品的 3D 打印

> 原文：<https://hackaday.com/2022/09/11/3d-printing-for-textile-work/>

虽然不是每个人都适合，但我们知道的一些最好的创作者都是处理纺织品的专家。虽然这门艺术很古老，但并不容易，需要巧妙的工具。[Lauren]收集了一些 3D 打印，可以帮助你[编织，钩针，甚至针织机](https://all3dp.com/2/3d-printed-knitting-machines-crochet-hooks-looms/)。

有些设计非常简单，比如纱线碗，或者很容易理解，比如简单的羊毛重绕机。坦率地说，我们很惊讶你可以 3D 打印一个钩针，尽管帖子提到破坏它们是一个真正的问题。

不过，我们对[袜子编织机](https://www.youtube.com/watch?v=dDcyL0LyxS8)印象深刻。实际上有几个这样的，你可以在下面的视频中看到一个类似的。当然，像 RepRap 打印机一样，它需要金属杆、紧固件等形式的“维生素”。还有一台便携式织布机，看起来很有趣。

我们对织物艺术还不够熟练，不知道这些工具是否是商业产品的有力竞争者，但我们不得不承认袜子编织机看起来可能是。我们最近看到了一台[复杂的织布机](https://hackaday.com/2022/06/07/3d-printed-power-loom-shows-how-complex-weaving-really-is/)，尽管这可能比大多数人需要的要多一点。我们也研究了开源的[针织机](https://hackaday.com/2014/02/20/openknit-the-open-source-knitting-machine/)。当然，如果你不想用纺织品创作，你可以用[总是在上面 3D 打印](https://hackaday.com/2020/12/18/remoticon-video-how-to-3d-print-onto-fabric-with-billie-ruben/)来代替。
