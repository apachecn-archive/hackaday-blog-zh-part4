# 圣安地列斯电台获得六星通缉级别

> 原文:[https://hack aday . com/2018/07/25/GTA-San-Andreas-radio-earns-six-star-wanted-level/](https://hackaday.com/2018/07/25/gta-san-andreas-radio-earns-six-star-wanted-level/)

[raphal Yancey]希望能够随时收听 FM 和 Radio:X，而不必在圣安地列斯偷车或清道夫。作为喜欢放《模拟人生》中悲伤的钢琴音乐和写黑客帖子的人，我们完全可以理解。

但这不仅仅是又一个把果酱馅饼放进老式收音机，然后称之为三明治的项目(并不是说这些有什么错)。这东西就像一台真正的收音机。无论你是否收听，所有的电台都会连续播放，当你上下拨动旋钮时，它们会相互融合。

经过多次试验和错误之后，[raphal]找到了一个可以工作的 Python 混合器，但是不再维护它了。他用叉子叉起它，压扁了一两个 bug，[为 KY040 旋转编码器](https://github.com/raphaelyancey/pyKY040)写了一个模块，让它们能很好地与圆周率配合。迷人的蛇不止于此:这个项目的摇滚明星是【拉斐尔】的[虚拟无线电软件](https://github.com/raphaelyancey/Virtual_FM_Band)，当他在电台之间调谐时，它处理音频混合。一步一步的教程即将推出，所以请关注[raphal]网站的更新。调过去休息一下，听一听。

拉斯皮电台的冒险不一定是单向的。[以下是如何使用 DVB-T 加密狗和 SDR](https://hackaday.com/2015/11/04/rpitx-turns-rasberry-pi-into-versatile-radio-transmitter/) 将一台设备变成 AM/FM+发射机。
