# 时尚、复杂的彩虹糖分拣机

> 原文：<https://hackaday.com/2019/12/16/sleek-sophisticated-skittle-sorter/>

根据颜色对糖果进行分类是一个经典问题，这个问题源于摇滚明星的合同骑手，他们只是试图确保音乐会场地不会忽略更重要的合同义务。多年来，糖果分类已经成为爱好者用各种方法解决的经典问题。在几年前的一次失败后，[小法国 kev]被迫重新启动这些计划，并尽可能制造出最紧凑的分拣机。

这种极简主义的美使用 Arduino Nano 和 RGB 传感器来评估颜色。在顶部，一个小型伺服系统旋转料斗内的一个臂，这个臂既可以摇动彩虹糖，又可以在传感器前将它们排成一列。另一个小伺服旋转管架来捕捉彩虹。底座中有一个 RGB LED，从下方照射与彩虹糖相匹配的光线。在[little french kev]的个人项目网站上查看显示每个部分如何工作的 gif 系列,然后在休息后观看构建视频。

你知道吗[你可以用一个 RGB LED 和一个光电池制作自己的颜色传感器](https://hackaday.com/2017/08/29/color-sensor-from-an-rgb-led-and-a-photocell/)？如果你认为糖果不那么好，你可以一直[给你的乐高](https://hackaday.com/2017/05/07/sorting-two-tonnes-of-lego/)按颜色分类。
