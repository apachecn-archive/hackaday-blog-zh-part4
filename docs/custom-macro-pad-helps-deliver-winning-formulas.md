# 定制宏垫有助于传递制胜公式

> 原文：<https://hackaday.com/2022/10/05/custom-macro-pad-helps-deliver-winning-formulas/>

对于我们这些具有科学和工程背景的人来说，打开字符映射表或记住各种符号的 Unicode 快捷键是编写报告或演示文稿的一个乏味但熟悉的部分。马格纳·劳里岑认为一定有更好的方法，于是开发了数学板(T3)。

Mathboard 拥有 80 多种“最常用的数学运算符”和完整的希腊字母，对许多学科都非常有用。硬件方面，Mathboard 是一个 4×4 宏板，但特殊的酱料是在关键设置实现固件中。虽然最直接的方法是为棋盘挑选 16 或 32 个符号，但[Magne]认为这样做并没有公平对待大范围的 Unicode 符号。通过实现一个列和层的系统，他能够在每个键上获得 6 个以上的符号，这比仅仅 16 个键和一个移位层提供了更大的符号宽度。旁边带有圆点的符号通过双击或三击该键来解锁该符号的变体。例如，希腊字母的小写或大写。

Mathboard 目前在 Microsoft Office 的方程式编辑器中工作，并作为一个纯文本 Unicode 板。[Magne]目前正在致力于 LaTeX 支持，并希望在未来添加 Open Office 支持。这款设备在我们的奇怪输入和特殊外设竞赛中获得了[荣誉奖](https://hackaday.com/2022/07/13/overwhelmed-by-odd-inputs-the-contest-winners-and-more/)。如果你想看另一个有趣的数学主题板，可以看看 1974 年 [MCM/70](https://hackaday.com/2022/02/23/inputs-of-interest-canadian-mcm-70-was-kinda-like-the-first-cyberdeck/) 上的那块。
