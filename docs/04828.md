# 纸板长板采用四分之一等网格结构

> 原文：<https://hackaday.com/2019/11/12/cardboard-longboard-uses-quarter-isogrid-structure/>

滑板通常是手工制作的，由木头精心分层雕刻而成。然而，许多制造商已经寻求探索替代技术。[Technovation]决定结合替代材料和数字制造技术[来生产这种有吸引力的纸板长板。](https://www.instructables.com/id/Quarter-Iso-grid-Cardboard-Longboard/)

电路板的结构在 Fusion 360 中设计，[采用四分之一等网格设计](https://sheldrake.net/quarter_isogrid/)。该结构由肋骨连接的纵梁组成，全部由纸板制成，带有互锁的插槽，可以将所有东西固定在一起。然后使用 1/4”胶合板来加固卡车支架，并安装 4 毫米厚的丙烯酸顶板和底板来保护纸板免受损坏。

该板的零件都是激光切割，使生产和组装变得轻而易举。也没有使用胶水——这种结构能够通过其开槽结构很好地将自身结合在一起。该团队指出，在板上有一个骑手确实会产生一些显著的柔性，但在实践中并没有导致失败。

滑板是一个很受欢迎的创客项目，这些年我们见过各种各样的。现代制造技术经常被用于，或者[设计被用来解决棘手的旅行问题](https://hackaday.com/2017/09/08/this-electric-longboard-collapses-for-air-travel/)。休息后的视频。
