# 3D 打印 Pi 笔记本电脑向标志性网格指南针致敬

> 原文:[https://hack aday . com/2021/01/11/3d-printed-pi-laptop-honors-the-iconic-grid-compass/](https://hackaday.com/2021/01/11/3d-printed-pi-laptop-honors-the-iconic-grid-compass/)

如果你熟悉老式便携式电脑，你应该知道网格罗盘。即使你对过去的电脑不感兴趣，你也很有可能看到过一两个指南针而没有意识到。从《T2》中的异形大战到《T3》中的太空之旅，引领潮流的翻盖式电脑在 20 世纪 80 年代似乎无处不在。虽然对于当时的普通消费者来说过于昂贵，但其不折不扣的设计和时尚的外观为今天无处不在的笔记本电脑奠定了基础。

与 1982 年相比，在 2021 年获得一个可用的网格罗盘并不容易，所以[【迈克】决定退而求其次，建造自己的](https://imgur.com/a/BkcIFII)。他的 GRIZ 六分仪肯定不是复制品，但家族的相似性足以说明这一点。如你所料，Raspberry Pi 驱动的机器在后部有一个大大减少的“主干”部分，但整体布局非常相似。受 Commodore 64 启发的配色方案可能是对原始材料的最大偏离，但很难对结果提出异议。

[![](../Images/18f50f87bb1b7f72e3426147d4e6e838.png)](https://hackaday.com/wp-content/uploads/2021/01/gridz_detail.jpg) 一目了然的是，六分仪的外部美学投入了大量的思想，但其内部细节同样令人印象深刻。[Mike]告诉我们他有产品设计的背景，这一点显而易见。他没有把这个项目作为一次性的创作来对待，而是尽可能确保设计的可重复性。

3D 打印框架和外壳的所有单个组件都经过精心设计，因此它们将适合普通台式机的体积。电子元件是用螺丝而不是胶水固定在内部框架上的；使得将来的维修和维护更加容易。当与充足的可用内部空间相结合时，这种模块化方法也应该使添加定制硬件成为相对容易的过程。

那么你什么时候能造出你自己的 GRIZ 六分仪呢？希望，很快。[Mike]说他仍然需要解决电源的一些问题，并最终确定如何将扬声器安装到机箱中。一旦这些最后的调整被锁定，他计划发布所有的 STL 文件和一个完整的材料清单。对于那些想在开始预热挤压机之前先睹为快的人来说，他也开始在他的 YouTube 频道上记录六分仪的组装过程。

在过去的几年里，我们已经看到了社区成员创造的数量惊人的定制便携式电脑。从稀奇古怪的电脑平台到像六分仪这样充满爱的劳动，T2 向过去几十年的技术致敬，越来越强大的桌面 3D 打印机和 Linux SBCs 最终让黑客们创造出他们梦想中的真正的个人电脑。
