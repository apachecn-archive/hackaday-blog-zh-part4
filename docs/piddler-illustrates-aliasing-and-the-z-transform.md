# 皮德勒举例说明了锯齿和 z 变换

> 原文：<https://hackaday.com/2020/05/18/piddler-illustrates-aliasing-and-the-z-transform/>

学习太多控制系统理论的一个问题是，你开始意识到几乎所有东西都是某种控制系统。(费尔南多·日古诺夫)就是这种情况。在他的厨房水槽中观察到瑞利平台的不稳定性后，他决定制作一个小展示件，展示水显然在对抗重力，他称之为[Piddler](https://zigunov.com/2020/04/25/the-anti-gravity-spout-a-demonstration-of-aliasing/)。

当然，我们以前见过这样的事情。一个咖啡泵，一个止回阀，一个带控制器的闪光灯就是全部。这个项目的有趣之处在于长达一个多小时的视频讲座，讲述了其工作原理以及它与混叠和 z 变换的关系。你可以看看下面的视频。

此外，还有一个使用高速摄像机的物理分析，这很有趣。有时我们会看到这样的视频掩盖了数学，但不是这个。带上你的微积分书和计算尺。甚至还有一两个微分方程藏在里面。

尽管 Piddler 不像我们已经看到的这些中的一些那样花哨，但它可能是最好的记录，至少从理论的角度来看是这样。如果你不想切开你的咖啡机，也许你有一个[自行车打气筒](https://hackaday.com/2017/05/22/gravity-defying-drips-of-a-bike-pump-controlled-fountain/)？
