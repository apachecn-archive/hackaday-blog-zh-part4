# 装有定制方坯水泵的微型汽油发动机

> 原文：<https://hackaday.com/2021/05/31/tiny-gasoline-engine-fitted-with-a-custom-billet-waterpump/>

我们通常不经常使用小于 50 毫升左右的汽油发动机。在这个尺寸以下，电动马达通常不那么杂乱，更有能力完成工作。然而，这并不意味着他们不可爱。[Johnny 90]是一个微型内燃机的爱好者，[决定为他的一个](https://www.youtube.com/watch?v=B_Xl4PcB73A)制造一个小水泵，这样它除了制造噪音还能做一些有用的事情。

该泵由铝锭制成，展示了[Johnny 90]的加工技能。组成泵的主体和盖板的两个零件足够令人印象深刻，但真正的派对零件是微小精致的叶轮，它实际上完成了大部分工作。泵叶片的精致曲线被精心雕刻出来，完成后看起来很精致。

泵的性能是足够的，微型汽油发动机的噪音很大，但这是一个伟大的加工技能的展示。如果需要的话，如果连接到一个安静的电机上，这种泵也可以在小型液体输送系统中发挥很大的作用。铝制设计的好处是在正确组装时相对无泄漏，这是许多 3D 打印设计难以实现的。

我们以前也见过[JohnnyQ90]的微型发动机实验——像这个小型发电机制造的。休息后的视频。
