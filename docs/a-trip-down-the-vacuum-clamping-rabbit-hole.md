# 下钻真空夹紧兔子洞

> 原文：<https://hackaday.com/2021/05/07/a-trip-down-the-vacuum-clamping-rabbit-hole/>

我们都知道掉进兔子洞是多么容易的事情，一些看似简单的工作变成了实验和发现的意外之旅。也许没有人比[马蒂亚斯·万德尔]更容易钻牛角尖了，至少从他最近在木工车间量化不同的真空夹紧技术来看是这样的。(视频，嵌在下面。)

为了理解这一切的来源，你必须拨回[【马蒂亚斯】的第一个视频](https://www.youtube.com/watch?v=7X66yFudKgI)，在那里他只是试图做一个简单的公告板。为了在整个木板表面上获得均匀的压力，他想出了一个商店方便的真空夹具，由一片厚塑料、一些木屑和夹具以及几个真空吸尘器制成。将工件夹在光滑平整的桌子和塑料片之间，他能够吸出空气，并对软木板施加巨大的力。

对第一个视频的评论引出了下面的链接，其中[马提亚斯]旨在探讨对他的方法的一些批评。使用四个 BMP280 压力传感器分线板和一个树莓 Pi，他能够很好地绘制他的夹紧夹具内的压力。他发现，传感器不仅使发现和修复泄漏变得容易，还证明了在工件和真空袋之间增加一层多孔层不太可能改善夹紧。他还展示了他收集的真空吸尘器中效果最好的——不出所料，Miele 的吸力最大，尽管他发现它不适合连续夹紧。

我们可以看到像这样的夹具的许多用途，并且当沿着兔子洞的旅程产生如此有趣的结果时，我们真的喜欢它。尤其是定量结果；还记得[【马提亚斯】对地下室湿度的探索](https://hackaday.com/2019/07/01/exploring-basement-humidity-with-a-raspberry-pi/)吗？
