# 在完全流体系统中成型复杂光学元件

> 原文:[https://hack aday . com/2021/12/09/molding-complex-optics-in-a-complete-fluid-system/](https://hackaday.com/2021/12/09/molding-complex-optics-in-a-completely-fluid-system/)

传统的透镜制作是一项苦差事——毫不夸张地说。一种方法是从一块玻璃开始，在一个研磨面上摩擦，磨掉多余的碎片，最终得到适合工作的形状和大小。不管是用机器还是用手，这都是一个耗时的过程，而且看起来肯定有更好的方法。

多亏了 Technion:以色列理工学院的莫兰·贝尔科维奇，才有了。他领导的团队利用液体快速廉价地创造出复杂的光学系统，这个过程看起来非常简单。这有点类似于大规模生产的光学设备中常见的注射成型镜头，但有一点不同——它本身没有模具。相反，一种紫外线固化树脂被注入到一个 3D 打印的约束环中，该环位于一罐液体中。树脂的形状由约束环的几何形状和作用在树脂上的重力、流体静力和表面张力决定。一旦树脂形成正确的形状，一束紫外线就会将其固化。很快，即时镜头！

树脂和抑制液体之间的界面使得透镜非常光滑；他们引用了一纳米范围内的表面粗糙度。使用流化床来约束透镜也意味着这种方法可以扩大到直径为 200mm 或更大的透镜。[论文](https://www.osapublishing.org/optica/fulltext.cfm?uri=optica-8-11-1501&id=464960)并不完全清楚使用的是什么液体，但当我们就此询问我们的朋友【Zachary Tong】时，他说他听说树脂是一种光学级紫外线粘合剂，而抑制液体是甘油和水的混合物。
