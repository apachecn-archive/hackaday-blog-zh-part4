# 黑客日播客 159:僵尸黑仔或地毯制造商，3D 打印轮圈，20 世纪 50 年代的鼓机，轮子上的电池

> 原文：<https://hackaday.com/2022/03/11/hackaday-podcast-159-zombie-killer-or-rug-maker-3d-printed-rims-1950s-drum-machines-and-batteries-on-wheels/>

加入 Hackaday 主编埃利奥特·威廉姆斯和执行主编汤姆·纳尔迪的行列，回顾上周的最佳黑客和故事。新闻中有很多值得谈论的内容，尽管在故障高度计和正在进行的乌克兰冲突之间，这并不是积极的变化。但随着讨论转移到 3D 打印汽车车轮、1958 年的一台极其复杂的鼓机、七段翻转显示器的独特表现以及精心设计(并记录在案)的咖啡机升级，事情很快变得明朗起来。在那里的某个地方，一个家伙还重现了一个二战时期罕见的德国防空火箭发射器，但这都是以历史的名义。我们还将解决两种非常不同的电力推进形式，从出现在世界各地车库和车道上的巨大轮式电池到用于深空任务的高效推进器。
