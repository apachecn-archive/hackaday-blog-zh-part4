# 谐波与摆线显示下降

> 原文：<https://hackaday.com/2022/12/23/harmonic-vs-cycloidal-show-down/>

什么更好？谐波传动还是摆线传动？我们不确定，但我们知道该问谁。【How To Mechatronics】[3D 打印了两种变速箱，并通过了几次测试](https://howtomechatronics.com/how-it-works/harmonic-vs-cycloidal-drive-designing-3d-printing-testing/)。你可以在下面看到测试的视频。

这两个齿轮箱大小相同，减速比均为 25:1。设计用的是 SolidWorks 相对便宜的 maker 版本。观察软件过程也很有趣。但视频的真正内容是对这两种设计的测试。

如果您想尝试自己的测试，也可以自己下载模型。博客帖子“正在建设中”，但你可以在视频中看到一些测试。例如，驱动器在间隙和扭矩测试中确实显示出差异。他还观察了故障期间关键部件的磨损情况。

这两个驱动器看起来都很棒，根据您的需要，您可能更喜欢其中一个。但是我们总是喜欢这些面对面的测试。这使得为您的下一个设计进行权衡分析变得更加容易。

在之前，我们已经看过【如何机电一体化】[解释驱动机制。如果你认为一个](https://hackaday.com/2021/12/25/do-you-need-a-cycloidal-drive/) [3D 打印的变速箱不能产生很大的扭矩](https://hackaday.com/2021/05/24/a-high-torque-3d-printed-harmonic-drive/)，那你就错了。
