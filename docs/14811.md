# 老式试管测试器拆卸

> 原文：<https://hackaday.com/2022/09/16/vintage-tube-tester-teardown/>

[卡尔森先生]有一台 1940 年代的老式电子管测试仪，就是杂货店和药店里常见的那种。它的状况非常好，他非常好心地为我们把它拆了。测试器是来自伊利诺斯州埃文斯顿的放射技术实验室的 X 型，和[卡尔森先生]一样，我们感到有趣的是，设备上的一个指示器是一个类似占卜板的“可疑”读数。当它亮起来的时候，看起来很神奇。

这比我们小时候记忆中的老式“电视显像管测试器”要古老得多，但想法是一样的:你有一台坏的收音机或电视机，里面有显像管，很可能是显像管出了问题。即使你不太了解电子产品，你也可以小心地取出电子管，开车去药店，测试你的电子管，买一个坏的替换。独特的是，这个测试仪甚至有一个扬声器，你可以在测试时用它来听电子管的输出。

我们怀念老式的晶片式旋转开关。里面没有多少东西，但我们惊讶地看到其中一个测试插座上完全没有电线连接。里面有一个真空管整流器，这导致了一个问题，如果它本身有一个坏管，管测试仪会发生什么。

我们想知道今天你是否能把这么高的电压放在消费者旁边。大概不会。但是能看到这种曾经很常见的设备的一个非常古老的版本是很棒的。

我们看到一台 1958 年的测试机[用的是打孔卡](https://hackaday.com/2021/06/15/your-1958-punch-card-machine-tested-tubes/)。如果你有足够的装备，你甚至不需要专用的试管测试器。
