# 这款超级逼真的 LED 蜡烛非常抢手

> 原文：<https://hackaday.com/2019/03/15/this-super-realistic-led-candle-is-smoking-hot/>

在过去的几年里，LED 蜡烛变得越来越普遍；而且理由很充分。从远处看，一个像样的 LED 蜡烛是一个非常令人信服的真实物品的复制品，提供了一个低闪烁的光芒，没有烧毁你的房子的恼人风险。但是对于真正的蜡烛的*体验*还是有话要说的；比如当你吹灭一只烟时，你会得到一股芳香的烟。

[![](img/ded92a9012a207a3ce125724ba6f6fd6.png)](https://hackaday.com/wp-content/uploads/2019/03/ledcandle_detail.jpg) 这就是为什么[【Keith】开始了为期三年的史诗般的探索，试图打造最真实的 LED 蜡烛](https://www.instructables.com/id/About-This-Candle-How-to-Read-This-Instructable/)，特别关注商业产品所缺乏的功能。因此，它不仅使用真正的蜡作为 led 的扩散器，而且你可以用实际的火柴“点燃”它。当麦克风检测到你对着它吹气时，它甚至会喷出一点逼真的烟雾。具有讽刺意味的是，它产生烟雾的能力意味着它并不能完全排除无人看管时让你的房子着火的可能性，但我们认为这是你为真实性付出的代价。

正如你现在可能已经收集到的，[Keith]对这个东西非常认真，并且不遗余力地记录他的蜡烛的漫长开发过程。如果你想建立一个类似的蜡烛，他的书面文件以及休息后的视频肯定会让你走上正轨。他甚至将设计分解成越来越复杂的“里程碑”,例如，如果你不在乎蜡烛冒烟，你可以跳过这一部分。

那么[Keith]在他的终极 LED 蜡烛里放了什么呢？在最基本的形式中，电子设备由一个 Arduino Pro Mini 和一大块包含六个 led 的 RGB WS2812B 条组成。如果您希望蜡烛能够检测到火柴的存在，请添加一个红外传感器，如果您希望能够向蜡烛吹气以关闭蜡烛，请添加一个麦克风。如果你想全力以赴，事情只会变得棘手，老实说，你想全力以赴。

为了安全地产生一股芳香的烟雾，[Keith]正在使用一卷 28 号线缠绕在“Tiki Torch”的灯芯上，以及一个足够强大的电源和 MOSFET 来使它变得温暖。灯芯注入了他自己的植物甘油和芳香油的混合物，当线圈点燃时，它产生令人印象深刻的浅灰色烟雾，携带着你添加的任何油的气味。即使你目前没有在寻找最终的电子蜡烛，它也是一个整洁的小工具，可以在万圣节使用。

你可能会惊讶地发现 [LED 蜡烛在黑客社区中是一个相当受欢迎的项目](https://hackaday.com/2017/01/05/blowing-out-a-candle-flicker-led-the-death-of-flame/)。[从极其简单的](https://hackaday.com/2018/12/23/lighting-up-a-very-wiry-candle/)到[极其复杂的](https://hackaday.com/2018/09/05/led-candle-gets-the-555-treatment/)，这些年来我们已经看到了令人印象深刻的一系列电子蜡烛。非常适合在听最新一集 Hackaday 播客的[流畅声音时营造气氛。](http://hackaday.com/2019/03/08/hackaday-podcast-ep9-on-the-edge-of-ai-comment-your-code-big-big-wheels-and-makers-of-munich/)
