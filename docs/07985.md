# 石墨烯产生一点能量

> 原文:[https://hack aday . com/2020/10/05/graphene-generates-a-little-power/](https://hackaday.com/2020/10/05/graphene-generates-a-little-power/)

我们从来不知道大学的新闻稿到底是怎么回事，因为我们看到许多新闻稿气喘吁吁地宣称有新电池或超材料，但除此之外，我们看不到其他东西。有时，新闻稿中的说法在报纸上站不住脚，而其他时候，这些说法似乎在现实生活中不切实际。我们不太确定如何理解阿肯色大学的一份新闻稿，该新闻稿声称他们可以完全通过温度波动从一片独立的石墨烯中获取电流。

新闻稿似乎声称，这是一个突破，导致“清洁，无限的权力。”但是如果你看看实际的纸张，正常的室温会像布朗运动一样在石墨烯片上引起微小的位移。即使系统达到热平衡，带有两个二极管的扫描隧道显微镜也可以检测电流。不过，请记住，这是在存在偏置电压的情况下，我们讨论的是纳米级位移和 20 pA 电流。你可以从大学里看到一个简单的视频，展示了设置的框图。

这里的新闻似乎是，电流在热稳定系统中流动，而不违反热力学第二定律。显然，创新之处在于使用两个二极管来收集正负波动。不过，没多少能量。新闻稿称，数百万个这种微型电路可以作为低功耗电池的替代品。

新闻稿中仍有一些相当神秘的评论，关于二极管电阻的变化增加了“额外的功率因素”，以及“二极管类似开关的行为实际上放大了传输的功率……”我们将不得不费力地通过论文中的数学来看看我们是否能弄清楚这意味着什么。

我们一直听说石墨烯将会做令人兴奋的新事情。我们只是在野外看不到它，至少现在还没有。所有这些关于石墨烯和二极管的讨论让我们记住，你可以在石墨烯中制造出 [PN 结](https://hackaday.com/2016/10/14/graphene-refractions/)。
