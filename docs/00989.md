# 模块化小提琴带弓

> 原文：<https://hackaday.com/2018/10/16/modular-violin-takes-a-bow/>

他们说小提琴和小提琴的唯一区别是你演奏的方式。如果是这样的话，这种模块化小提琴将需要一个新的名字，因为它可以在几分钟内被分解和改变，使它听起来完全不同。

这把小提琴是[大卫·佩里]的作品，有 3D 打印的琴身、琴颈、琴身和琴马。虽然表面上看起来这是一种让世界上几乎任何人都可以使用的不太昂贵的乐器的方法，但真正有趣的品质是在[大卫]开始演奏他创作的所有不同版本时显示出来的。根据印刷风格、使用的塑料类型和许多其他品质，声音会发生明显的变化。

当然，你需要一把弓、弦、钉子和指板，但如果你身边有一台 3D 打印机，其余的都是可用的。如果你已经是一个熟练的小提琴手，这可能是一个非常实惠的尝试新声音的方法。这不是我们第一次看到 3D 打印的小提琴，但这是我们第一次看到它们被专门设计来改变它们的声音方式，而不是它们的物理特征。如果你想自己做，所有的。stl 文件可以在项目网站上找到。
