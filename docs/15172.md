# 飞行模拟器聚焦于驾驶舱门的另一侧

> 原文:[https://hack aday . com/2022/10/25/flight-simulator-focuss-the-other-side-of-the-cockpit-door/](https://hackaday.com/2022/10/25/flight-simulator-focuses-on-the-other-side-of-the-cockpit-door/)

当一个人想到进入飞行模拟器时，他会认为这是从飞行员的角度出发的。但是这个另类的飞行模拟器采取了不同的策略，让你从乘客的角度实现你的空中旅行幻想。

那些想要全动作模拟客舱体验的人会失望，因为[亚历克斯·莎士比亚]——我们假设没有关系——已经为这个模拟器建造了一个最小的客机客舱。不过，这是有道理的。理想情况下，航空公司飞行员的目标是为乘客提供尽可能乏味的旅程。飞行最令人兴奋的地方，也是[Alex]在这里很好地捕捉到的地方，是到达目的地的最后途径，当机场及其周围的环境在长时间盯着云之后终于进入视野。这是通过在一个合理的飞机座舱的窗户外面安装一个 LCD 监视器来实现的，该座舱配有一排座位。最初位于客机驾驶舱中的控制面板用于选择各种异国目的地机场的进场视频，如拉斯维加斯。视频由 Pi Zero 播放，而 ESP32 负责控制灯光、风扇和非常逼真的头顶面板上的服务员呼叫按钮。播放瑞安航空抵达顺口溜的按钮加分。

[Alex]的模拟器令人印象深刻地完整，尽管在概念上有些令人费解。不过，我们不做评判，对游客来说，这看起来可能很有趣，尤其是当饮料车经过的时候。
