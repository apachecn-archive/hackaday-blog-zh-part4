# 3D 印花面料按需硬化

> 原文:[https://hack aday . com/2021/08/15/3d-printed-fabric-stiffens-on-demand/](https://hackaday.com/2021/08/15/3d-printed-fabric-stiffens-on-demand/)

新加坡和加州理工学院的研究人员开发了一种 3D 打印织物，它有一个有趣的特性:它通常是柔性的，但可以根据需要变硬。你可以在下面看到一段关于这种新面料的视频。

这种材料由尼龙八面体互锁而成。这块布被封在一个塑料封套里，并进行真空包装。一旦进入真空，薄片会变得更加坚硬，可以承受数倍于自身重量的重量。

据推测，这种想法是让材料在塑料外壳中弯曲，直到需要增加刚度，然后去除空气。当然，这有很多实际问题。例如，如果封套不再密封，操作将会失败。也很难快速去除袋子中的空气，以制作像蝙蝠侠的斗篷一样的东西，这是研究人员进行的比较。

这种织物使用一种叫做“堵塞过渡”的东西来工作，这种东西会导致粒子从类似流体的状态转变为刚性状态。这是一个众所周知的原理，但对固体粒子并不实用。
3D 打印的中空结构要轻得多。除了尼龙之外，该团队还对铝织物进行了实验，结果显示了类似的结果，正如你所料，金属的强度提高了。

看起来这种设计很容易复制，也许有聪明的方法来克服这种材料实际应用的障碍。
