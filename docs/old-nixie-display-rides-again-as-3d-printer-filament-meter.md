# 老谢妮显示器再次成为 3D 打印机灯丝测量仪

> 原文：<https://hackaday.com/2019/05/09/old-nixie-display-rides-again-as-3d-printer-filament-meter/>

我们不确定[斯科特·m·贝克]建造的这个数码管灯丝测量仪的名字。他称之为“细丝表”，也许是“细丝”和“里程表”的组合，在这种情况下，它是有意义的。它可能不会脱口而出，我们也想不出更好的名字，但不管你用什么名字，它实际上是一个非常酷的建筑。

filadometer 最初是一种完全不同的东西，是数码管项目的典型代表——一种温度和湿度测量仪。[Scott]决定回收八管显示器来跟踪他的 Prusa，这样做显示了他在设计过程中相当程度的深谋远虑。最初的谢妮显示器拥有所有常见的配件——驱动芯片、移位寄存器和高压电源。最突出的是他设计的模块化:电子管插座和驱动器位于背板 PCB 上，有一个树莓 Pi 和一个单独的高压供应板插入其中。最初的显示器有一个型号 B Pi，所以有足够的空间容纳新的 Zero W。一个新的印刷外壳和一点编程来捕捉 Octoprint 的灯丝使用情况，这一切都是为了让这个漂亮的小建筑重新投入使用。下面的视频展示了细节。

我们总是很高兴看到来自(斯科特)的新视频，因为我们从他的虚拟肩膀上学到了很多东西。如果你还没看过他的东西，看看[他对 8 英寸软盘](https://hackaday.com/2017/01/22/an-eight-inch-floppy-for-your-retrocomputer/)的敬意或者[他对复古游戏](https://hackaday.com/2018/07/26/dual-port-memory-and-raspberry-pi-team-up-for-retro-console-multicart/)的双端口内存破解。
