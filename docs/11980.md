# 遥控开始了！

> 原文：<https://hackaday.com/2021/11/20/remoticon-is-on/>

如果你在阅读这篇文章的时候还记忆犹新，那么离 2021 年黑客日的第二天还有大约四个小时。你能感觉到空气中的电流，对吗？今年我们有一个绝对[的明星阵容](https://remoticon.io/#Schedule):每一次谈话都将是一次精彩的谈话。

星期五非常有趣，如果你抓紧时间，你仍然可以倒带星期五的现场直播。否则，在太平洋标准时间上午 11:00，我们将开始第二天的[活动，由 Keith Thorne](https://www.youtube.com/watch?v=x281ImH68Ik) 做[主题演讲，他将带领我们经历所有的工作和黑客攻击，这使得 LIGO 实验成为人类有史以来最灵敏的仪器。在宣布 Hackaday 奖和](https://remoticon.io/#Keynotes)[派对后 DJ Jackalope 现场设置](https://www.twitch.tv/missjackalope)之前，杰里米·菲尔丁将讲述关于他的机械臂的一切和[你需要知道的关于任何会动的东西的信息](https://remoticon.io/#Keynotes)。

观看一些精彩的演讲，然后在我们的 [Discord 服务器](https://discord.com/invite/NkbHrAW7NG)上与演讲者一起闲逛。周六你还想要什么[？](https://youtu.be/x281ImH68Ik)

我们真的不想让你错过任何东西，但是万一你来晚了，你可以倒带直播来赶上进度。全部完成后，我们将把会谈内容分成小块，这样你以后就可以找到了。但是如果你错过了讨论，错过了向演讲者提问的机会，错过了无处不在的欢乐气氛，你就错过了。如果可能的话，请参加我们的现场直播！我们很想见到你。
