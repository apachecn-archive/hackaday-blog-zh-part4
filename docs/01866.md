# 用泡泡说话

> 原文：<https://hackaday.com/2019/01/18/talking-with-bubbles/>

尽管标题如此，这并不是一个与迈克尔·杰克逊的黑猩猩交谈的故事。更确切地说，这是关于【KyungYun】的机器，它将语音转换成异想天开的气泡。虽然语音控制很新颖，但我们更感兴趣的是这种机制如何使用一个字符串系统来吹泡泡，以及使设备便携的工艺。

开火的速度不是很快，所以你说话的时间越长，气泡看起来就越大。从本质上来说，这种设备增加了虹膜(吹泡泡的部分)的大小，直到你暂停说话。然后它会吐出一个泡泡。

虹膜机制借鉴了[一个更大的气泡机器](https://www.instructables.com/id/CNC-bubble-machine/)的想法，尽管实际构建要小得多，并且使用激光切割和 3D 打印件。青少年提供大脑，还有一个泵将气泡溶液输送到虹膜。

据我们所知，肥皂液顺着接触的琴弦滴下。当绳子分开时，它们之间会形成一层肥皂膜。空气的爆发会产生气泡。有可能制作出[彩色气泡溶液](https://www.mamaplusone.com/diy-color-bubbles/)，我们正试图想办法为不同种类的声音制作不同的颜色，尽管有三个虹膜机制会使设备不太便携。也许更实际的做法是拥有多个溶液箱，并根据合理的分析对它们进行不同的混合。在任何情况下，这将是一个有趣的项目，以一些创造性的补充。

我们已经看到了不止一种吹泡泡的方法。如果你想要很多泡泡，你可以 [3D 打印这个装置](https://hackaday.com/2015/08/12/blow-14000-bubbles-per-minute-with-this-3d-printed-contraption/)。
