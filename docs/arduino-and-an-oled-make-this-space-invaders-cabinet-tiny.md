# Arduino 和一个有机发光二极管让这个太空入侵者的柜子变得很小

> 原文：<https://hackaday.com/2022/01/12/arduino-and-an-oled-make-this-space-invaders-cabinet-tiny/>

现在看来很简单，*太空入侵者*是一台来自视频游戏黄金时代的机器，它似乎总是排着长队等待失去几个硬币的机会。为了庆祝这款开创性游戏的影响力，[尼克·克兰奇]制作了可能是世界上最小的[*太空入侵者*复制品](https://hackaday.io/project/183483-space-invaders-on-the-arduino)。

看起来，这主要是作为一种利用手头的资源实现可能的一种练习，其中包括几个相当小的有机发光二极管展示。对于建设照片，它看起来像有一个 Arduino 纳米运行显示；[Nick]提到所选的硬件被证明是具有挑战性的，他必须破解驱动程序库才能使其工作。一旦他得到了一个可行的游戏，[尼克]就不会满足于他的荣誉。更确切地说，他做了一个微型橱柜来存放所有的东西。

下面的视频质量可能很差，但足以看出他在细节上花了多少功夫。游戏机柜的原始美国版本的图形被精确地呈现，精确到前面玻璃上的艺术。橱柜本身是由他手工切割的 1.5 毫米厚的胶合板制成的。它甚至看起来像是他在单色屏幕上重新创造了玻璃纸覆盖的原始方案，为游戏添加了一点色彩。不错的手感！

我们真的很欣赏这里对细节的关注，我们唯一的挑剔是没有图表或代码被张贴。希望我们以后能看到这些，但现在，这看起来像一个有趣的项目和一次美好的回忆之旅。但是如果你认为它太小，不要担心——我们也有一个更大版本的游戏。
