# 小 ThinkPad 玩小游戏

> 原文：<https://hackaday.com/2019/07/31/tiny-thinkpad-plays-tiny-games/>

[保罗·克林格]似乎永远也不会满足于建造小而神奇的游戏装备，我们因此而热爱他。他们结合了我们最喜欢的两个:微型和便携式游戏。他的最新作品展现了强大的 ThinkPad 的外形。

当然，它有红色的奶嘴和盖子 LED——没有它们就不是 ThinkPad。ThinkTiny 的乳头是一个 5 向操纵杆，可以在有机发光二极管屏幕上玩蛇、俄罗斯方块、月球着陆器等游戏。像它的前辈[一样，微型 PC](https://hackaday.com/2019/02/23/ultra-tiny-pc-plays-snake/) ，【Paul】使用的是 ATtiny1614，它(仅供参考)有一个新的单线 UDPI 接口。他可以通过内置于表壳中的弹簧针孔轻松地对其进行重新编程。

这里也有一些不错的风格细节。lid LED 通过一根 2mm 的光纤电缆进行传输和扩散。[Paul]在封面上印上了一种颜色变化的透明灯丝，让 Think 标志和充电 led 发光。操纵你的方式过去休息，看看它在行动中。

如果你还没有达到 AVR 编程的水平，向你自己介绍一下 [Arduboy](https://hackaday.com/?s=arduboy) 。
