# 梅卡诺麦克斯被黑了

> 原文:[https://hackaday.com/2019/09/20/meccano-max-gets-hacked/](https://hackaday.com/2019/09/20/meccano-max-gets-hacked/)

市场上有很多“智能”玩具，有些比其他的功能更多。然而，大多数制造商希望完全控制一个平台，这在任何商业产品中都是缺乏的。正是这种渴望促使[MrDreamBot]开始破解 Meccano Max。

Meccano Max 是一个小型陪伴机器人，大约 30 厘米高。不满足于缺少一个 API，[MrDreamBot]决定首先试验[创建一个 Arduino 库来运行 Max 的硬件](https://github.com/MrDreamBot/MeccaChannel)。完成后，开始将 Hicat Livera devboard 集成到硬件中。这是一个嵌入式 Linux 系统，兼容 Arduino，并能够通过 WiFi 传输视频和连接。到目前为止，可以通过浏览器控制 Max，同时查看来自“机器人”的实时视频。还可以自定义 Max 脸上显示的表情。

通常，更换库存硬件比尝试使用原始设置的限制更划算，本项目也不例外。尽管如此，[我们仍然希望有人能找到让 Jibo 重新上线的方法](https://hackaday.com/2019/03/08/uncertain-future-of-orphaned-jibo-robots-presents-opportunities/)。照顾好你的机器人朋友！休息后的视频。
