# 一座永远毁灭的无用坟墓

> 原文:[https://hack aday . com/2019/10/01/a-useful-tomb-of-eternal-doom/](https://hackaday.com/2019/10/01/a-useless-tomb-of-eternal-doom/)

现在是正式的十月，这意味着我们可以开始释放万圣节黑客了。就拿这个不安分的骷髅来说吧，主自动机制造商【格雷格·朱姆沃尔特】[已经注定要在一个无用的盒子](https://www.instructables.com/id/RIP-Skeleton/)里度过永恒。如果这还不足以唤醒死者的话，那么每当某个爱开玩笑的人按下按钮，这些闪亮的灯就会亮起。嘿，至少没有歌剧音乐。

具有讽刺意味的是，无用的机器有很多制造方法。这个以万圣节为主题的无所事事的盒子没有使用微控制器，甚至没有使用 555——它完全是机电的。当按钮被按下时，两个 AAA 为一个小齿轮马达提供动力，该马达同时抬起盖子，举起死者，并把他扭转四分之一圈，这样他就可以关上盖子，让自己回到永恒的休息中。

复杂印刷的骨架并没有真正按下按钮——他已经死了，身体太虚弱了。齿轮马达还转动一个双凸轮，启动一对滚轮开关，控制蜡烛，把骨头先生放回他的墓穴。像血一样清澈？迅猎兽越过裂口，近距离观察机械装置。

不管是不是万圣节，我们都喜欢这些地方的无用机器。这里有一个结合了真正的蜡烛和[的，谁会忘记这个八月切换的野兽](https://hackaday.com/2012/12/22/an-even-more-useless-machine/)？
