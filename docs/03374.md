# 香蕉炸弹可能会给你带来麻烦

> 原文：<https://hackaday.com/2019/06/22/banana-bomb-is-likely-to-get-you-in-trouble/>

如果说好莱坞喜欢什么的话，那就是创造戏剧的时间。核武器，恐怖分子的炸弹，以及其他各种各样的装置都可以在一系列 7 段显示屏上看到不祥的倒计时数字。在这种情况下，[【deshi pu】对经典比喻](https://hackaday.io/project/7564-banana-bomb)发展了一种相当荒谬的观点。

该项目是围绕一个滴答作响的四位数显示器构建的。蓝色 led 给它带来了现代感，它被安装在 Arduino Pro Mini 3.3V 的顶部。这使得整个模块可以由一个硬币电池供电，这是一个非常紧凑和整洁的计时器，比显示器本身大不了多少。还有一个蜂鸣器，每秒钟鸣叫一次，在某种程度上提高了附近的压力水平。

随着计时器的工作，[德希普]然后通过将它挂在三个香蕉上来获得喜剧点。人们普遍认为这比把它连接到雷管电路和真正的炸药上更有礼貌，也是远离政府监视名单的关键。

这是一个在万圣节派对或类似场合会很有趣的作品，任何初学 Arduino 的人都很容易完成。不言而喻，虽然这很有趣，但这是一个非常不应该在公共场合流传或用于恶作剧的构建。在这个时代，[即使是兜售一个定制的时钟也会引起不受欢迎的注意](https://hackaday.com/2015/09/15/9th-grader-arrested-searched-for-building-a-clock/)，所以小心是很重要的。休息后的视频。
