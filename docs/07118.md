# 这种柔顺机构千分表不需要装配

> 原文：<https://hackaday.com/2020/07/15/no-assembly-required-for-this-compliant-mechanism-dial-indicator/>

如果你曾经有幸——或者在一次商店事故后，不幸地——看到过一个刻度盘指示器的内部，你就会知道这些商店必需品的工作原理与一只精美的瑞士手表一点也不像。内部的小齿轮、齿轮和弹簧将仪器柱塞的最轻微运动传递到一系列刻度盘，使得即使是最微小的差异也很容易发现。

不是每一个有用的刻度盘指示器都需要有那些机械的内脏，甚至连刻度盘也不需要。[这种柔性机械装置 3D 打印的免拨号指示器](https://www.thingiverse.com/thing:4524389)非常适合许多简单的任务，包括[阳光]设计的床调平家务。[SunShine]没有打印一堆齿轮并组装它们，而是选择打印柱塞、一套精细的柔性连杆臂和一根充当针的长杠杆臂。针头连接到一个柔性支点上，该支点是容纳柱塞的针筒的一部分。柱塞在针筒内的轻微运动推动或拉动针头，将它们放大成易于读取的偏转。当连接到 3D 打印机的头部并在床上扫描时，很容易看到即使是最轻微的高度变化，并做出相应的调整。看看下面的视频吧。

我们是柔顺机构的忠实粉丝，从[机械臂](https://hackaday.com/2020/02/09/simple-3d-printed-robotic-arm-uses-compliant-mechanism/)和[腿](https://hackaday.com/2020/03/22/compliant-quadruped-legs-using-servos/)到用于遥控飞机的[推力矢量](https://hackaday.com/2019/06/16/thrust-vectoring-with-compliant-mechanisms-is-hard/)的一切事物中都可以看到它们。这可能看起来像麦片盒子里的东西，它肯定没有斯塔雷特或三丰的持久力量，但话说回来，它基本上没有成本，我们也喜欢这一点。
