# EL 电线是一个很棒的人造霓虹灯

> 原文：<https://hackaday.com/2019/08/12/el-wire-makes-for-a-great-faux-neon-sign/>

霓虹灯很吸引人，但是需要专门的工具和技能来制作。如果你没有时间学习玻璃吹制和正确的气体来源，你就太不走运了。然而，EL 线可以给人一种类似的美感，有了现成的电源，很容易连接起来并开始工作。 [[sjm4306]将这与 3D 打印相结合，实现了快速简便的构建。](https://hackaday.io/project/166958)

该项目首先选择一个任天堂 64 霓虹灯标志作为设计的基础。在 Inkscape 中描绘了标志的图像，并将轮廓导入到 CAD 软件中。从那里开始，设计了一个框架，上面有供 EL 线缠绕的柱子，还有供 EL 线穿过的孔，可以通到标志的背面。然后 3D 打印出框架，并用必要的颜色系上 EL 线。

最终的结果令人印象深刻，EL 线充当了一个巨大的小型氖管模拟物。这是一种构建方法，也应该尽可能扩大 3D 打印组件的规模。如果你需要掌握它是如何工作的，[有一个使用 EL wire](https://hackaday.com/2010/12/07/how-to-work-with-el-wire/) 的教程。休息后的视频。
