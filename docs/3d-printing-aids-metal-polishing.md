# 3D 打印有助于金属抛光

> 原文：<https://hackaday.com/2022/09/20/3d-printing-aids-metal-polishing/>

虽然机械师可以用他们的车床或铣床在一块金属上做出漂亮的光洁度，但要达到最终的光洁度，还需要一套抛光程序。越来越细的磨料被用于一种称为研磨的过程，这种过程尽可能地去除任何瑕疵，最终留下镜面般的光滑度。但这并不是没有问题，特别是在工件的边缘，当磨料摩擦它们时，它会导致圆角。[机械师亚当]有一个解决办法，[，他用 3D 打印机找到了它](https://www.youtube.com/watch?v=7B5u5cMBNrI)。

为了避免边缘变圆，解决方法是将一块金属或木头与要研磨的表面齐平，这样压力就不会作用在边角上。这可能很不方便，解决方案是通过 3D 打印定制件来避免这种情况，该定制件适合整个加工对象，在边缘周围提供平坦的表面。我们看到它正与一个演示件一起使用，该演示件在同一平面上有三个独立的表面可以重叠，如果没有 3D 打印的帮助，这将是一个挑战。

我们在这里不常看到研磨的过程。但是[它突然成为一种极端的超频技术](https://hackaday.com/2020/07/21/die-lapping-for-better-cpu-performance/)。
