# 天空中…是一只鸟…是一架无人机…哦，是的，是一架无人机

> 原文:[https://hackaday . com/2022/06/21/up-in-the-sky-its-a-bird-its-a-drone-oh-yeah-its-a-drone/](https://hackaday.com/2022/06/21/up-in-the-sky-its-a-bird-its-a-drone-oh-yeah-its-a-drone/)

科幻小说中的一个主要内容是扑翼机，这是一种有移动翅膀的飞机。虽然这些还没有被证明在一般意义上非常实用，但最近的一篇论文谈到[模仿自然翅膀改变形状](https://arxiv.org/abs/2205.09431)以提高无人机和其他飞机的机动性。特别是，论文谈到了许多鸟类和蝙蝠的飞行性能如何远远超过常规飞机。

不出所料，比传统飞机机动性更强的技术术语叫做超机动性。执行类似于[普加乔夫眼镜蛇](https://en.wikipedia.org/wiki/Cobra_maneuver)机动的飞机(观看下面的视频，或最新的壮志凌云电影)需要这种类型的操作，对于现代飞机来说，这意味着使用推力矢量技术以及不稳定的机身和复杂的计算机控制。然而，这不是鸟类或蝙蝠的工作方式，该论文使用现代飞行模拟技术来表明仿生和推力矢量技术并不一定相互排斥。

那么如何将鸟类、蝙蝠、甚至飞鼠的经验应用到无人机上呢？答案似乎是让翅膀动态改变形状，或者如论文所称的变形。

这篇论文中涉及的一个关键机动动作对军用无人机来说尤其有趣:RaNPAS 或快速机头瞄准射击。据推测，你不会担心你的下一个无人机项目，但能够像鸟一样机动会很棒。

并不是说我们在这里没有见过真正的扑翼飞机。其中一些是几乎史前的。
