# 国际空间站业余无线电中继器

> 原文：<https://hackaday.com/2020/09/10/iss-ham-radio-repeater/>

航天器携带业余无线电爱好者无线电设备有很长的历史，因为航天飞机，和平号和国际空间站都有业余无线电爱好者携带能够与地球通话的设备。然而，这个月，国际空间站开始运行一个与地面中继器没有太大不同的调频中继器。你可以在下面的中继器上看到[TechMinds]视频。

中继器有 2 米的上行链路和 70 厘米的下行链路。虽然您可以使用普通的双频业余无线电收发机来使用中继器，但您可能需要特殊的天线和特殊的操作技术。

你会发现的一个问题是，国际空间站移动得足够快，你会观察到频率的多普勒频移。该视频再现了一个频率表，您可能需要移动该表来接收偏移信号。

在没有特殊设备的情况下，你可能会听到国际空间站顺利通过，但[TechMinds]无法在视频中近距离接触。但是[K0LWC]使用非常标准的无线电设置非常接近，正如你在第二个视频中看到的。

国际空间站已经广播了一段时间的数字中继器和传统的调频收音机。你需要的天线不一定是一个巨大的圆盘。我们已经看到用[手持波束天线和手持无线电](https://hackaday.com/2019/01/07/all-about-ham-satellites/)做到了这一点。
