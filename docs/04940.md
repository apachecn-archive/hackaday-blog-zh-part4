# 建造你自己的张拉整体结构

> 原文：<https://hackaday.com/2019/11/29/building-your-own-tensegrity-structure/>

张拉整体结构似乎正在网上流行，这可能是由于其建造看似不可能的本质。这些弦似乎没有任何合理的理由就悬浮起来，但如果你以正确的方式弯曲它们，它们就会屈服于重力。

线索就在名字里。[张拉整体是“张力”和“整体性”的统称](https://www.anatomytrains.com/fascia/tensegrity/)。如果你手里有一个模型，这是最容易理解的——剪断弦，结构就分崩离析了。我们习惯于从压缩的角度考虑完整性。从帝国大厦到公寓大楼的地基，大多数人造结构都依赖于这种工程概念。

张拉整体允许应变分布在整个结构上。虽然由持续压缩建造的建筑物可能不会表现出这种特性，但像我们身体这样更有弹性的结构会表现出这种特性。这些结构可以建立在连续分布应变的较小单元之上。此外，这些结构可以以“压缩力”无法展现的方式收缩和缩回。

把结构弄塌怎么样？这发生在最薄弱的地方。哪里的负荷对结构产生最大的张力，哪里就有可能断裂，这一特性在桥梁、穹顶甚至我们的身体中都有所体现。

着迷？幸运的是，创建自己的结构并不太困难。
