# 不受欢迎的自动化解决方案

> 原文:[https://hack aday . com/2019/05/31/the-automated-solution-to-your-popularity/](https://hackaday.com/2019/05/31/the-automated-solution-to-your-unpopularity/)

你觉得自己不受欢迎，没有人喜欢你。坏消息是，如果这是现实世界中的情况，没有简单的方法来解决它。好消息是有一个很好的替代品——你在 Instagram 上的人气。有了这台自动售货机，你可以用 Instagram 上的粉丝来代替你的个性。很遗憾我们要等一年才能到科切拉。

这个项目是一个来自[Dries Depoorter]的互动装置，使得在几秒钟内购买关注者和喜欢成为可能。这不仅限于 Instagram，你也可以在 FaceBook、YouTube 和 Twitter 上获得粉丝。硬件包括一个树莓派 3B+，一个 Arduino，硬币接收器，几个字符液晶显示器运行在 I2C，有点令人惊讶的一次性“艺术装置”，许多 DIN 导轨安装在一个真正的工业外壳上。这里有人知道他们在做什么；这个盒子里有类似于电缆管理的东西，这个自动售货机是为了耐用而制造的。

使用这种自动售货机很简单，只需在投币口投几个欧元硬币，然后选择你喜欢的关注者或喜欢者的数量。几分钟后，你的手机上会弹出数百条通知。除了用 Python 编写之外，没有提到这台自动售货机中的软件，这让我们想知道这些 Instagram 机器人是基于哪里的。看看下面的视频。
