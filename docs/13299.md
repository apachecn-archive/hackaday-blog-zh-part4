# 涡轮增压器机器人拖把拯救你的约会

> 原文：<https://hackaday.com/2022/04/10/a-turbocharged-robot-mop-to-save-your-date/>

清洁机器人虽然很棒，但在速度方面并不出众。如果你的房间看起来像个猪圈，而你的约会对象将在 20 分钟后到达，你需要的不仅仅是一个 Roomba 来给人留下好印象。[Luis Marx]遇到了这个问题，并决定通过建造世界上最快的清洁机器人来解决这个问题。

[Luis]从地面开始建造他的机器人，灵感来自于普通机器人真空的设计:圆形，有两个驱动轮和一些传感器以避免障碍。一个坚固的铝板形成了底盘，其上放置了两个强大的电机来驱动一对大直径的车轮。机器人的身体由 3D 打印组件制成，顶部有一个巨大的 LED 显示屏，可以充当各种速度表。

建造一个真空系统被证明是相当困难的，因为[Luis]已经有了一个机器人真空吸尘器，他决定用机器人拖把代替。一个小水箱储存水和肥皂，这些水和肥皂被泵到一个用磁条固定的微纤维布上。避障是通过三个超声波距离传感器实现的:当机器人将要撞到什么东西时，它会刹车并转向它感觉到最空旷的地方。

所有这些听起来都很棒，但是速度呢？根据[路易斯]的计算，它应该可以达到 60 公里/小时，尽管他的客厅太小，无法付诸实践。在这样的速度下，它是否能提供更多的清洁方式也是有争议的，但谁在乎呢:拥有自己的超高速机器人拖把肯定会比任何清洁量都更能打动你的约会对象。

我们之前展示过一个自制的机器人拖把，但是和这个相比，它看起来非常慢。如果你计划建造灵活的室内机器人，你可能会想看看快速导航系统，如[跟踪天花板灯](https://hackaday.com/2021/10/25/fast-indoor-robot-watches-ceiling-lights-instead-of-the-road/)。
