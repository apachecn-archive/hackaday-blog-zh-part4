# 复制品 Marshmello 头盔是一个整洁的万圣节建设

> 原文:[https://hack aday . com/2019/11/22/replica-marshmello-helmet-is-a-tidy-Halloween-build/](https://hackaday.com/2019/11/22/replica-marshmello-helmet-is-a-tidy-halloween-build/)

俗话说——你不需要一个风格化的、令人眼花缭乱的头盔来在 EDM 领域取得成功，但这很有帮助。Marshmello 是最新一个展示定制头饰的音乐家，[和【MikeTheSuperDad】为万圣节制作了一个复制品。](https://www.instructables.com/id/Marshmello-LED-Helmet/)

建造从一块混凝土模板开始，作为头盔的基础。这与 3D 打印组件相结合，创建一个网格来放置 WS2812B LED 灯串。这些由 Arduino Pro Mini 控制，它负责处理动画。进一步的 3D 打印部件被用作模板来切出有特色的眼睛和嘴巴，以及覆盖顶部。然后在所有东西的顶部使用塑料薄膜来漫射 led 并提供最终的外观，眼睛和嘴巴后面的黑色网格使它们适当地突出。

Marshmello 应该受到称赞，因为他创造了一个具有独特视觉风格的头盔，同时又易于复制，[不像过去几年流行的蠢朋克制造。在开始自己独特的构建之前，构建一个副本可能是一个很好的实践。休息后的视频。](https://hackaday.com/2016/09/06/a-helmet-to-make-daft-punk-jealous/)
