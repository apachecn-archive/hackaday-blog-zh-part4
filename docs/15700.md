# 大型水族馆倒塌背后的物理学原理

> 原文：<https://hackaday.com/2022/12/20/the-physics-behind-the-collapse-of-a-huge-aquarium/>

上周末，世界上最大的圆柱形水族馆 Aquadom 意外破碎，引发了紧急情况，因为它淹没了柏林酒店和周围的街道。从工程的角度来看，这是一个迷人的故事，因为它的结构是这样的，这不应该发生。我们分析了[路易斯·巴塔尔哈] ( [尼特](https://nitter.net/luismbat/status/1604627113742581760))可能出了什么问题，从中我们可以了解到一些所用塑料的特性。

这个水族馆是由一种丙烯酸聚合物制成的，这种聚合物具有一种有趣的特性——在一定温度下，它会在类似玻璃的状态和类似橡胶的状态之间转变。即使在室温下，丙烯酸树脂也远低于转变温度，但是随着温度下降，丙烯酸树脂变得更加易碎。当室外温度下降到零下时，门厅的温度也下降了，高水压足以粉碎丙烯酸树脂。

可悲的是，水族馆里的鱼很少存活，但幸运的是，没有人在事故中丧生。新闻报道显示了水的力量如何摧毁了门，并将残骸带到了街上，我们猜测，在任何其他酒店考虑将这样的项目作为一个景点之前，还需要一段时间。与此同时，我们获得了一点关于丙烯酸性质的知识，也许有一天会派上用场。
