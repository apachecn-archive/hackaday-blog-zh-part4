# 电子土星五号发射塔将乐高模型送上月球

> 原文：<https://hackaday.com/2020/12/17/animatronic-saturn-v-launch-tower-sends-lego-model-to-the-moon/>

当谈到他们更面向成人的模型时，乐高真的用他们的土星五号火箭模型击败了它。在乐高零件宇宙的限制下，这个一米高的模型非常详细，当它出来时，成千上万的太空迷急切地抢购。

但是没有发射台的火箭有点可悲，这就是为什么[马克·豪]为他的火箭模型设计了这个电子动画土星五号发射台和门架。发射台的细节水平完美地补充了土星 V 模型的功能，并突出了一旦它被推出到发射台，它需要为船员和火箭服务。正如你所想象的，广泛使用 3D 打印的零件是获得合适外观的关键，也是制造真正会动的零件的关键。

当发射时间到了，摇摆控制臂和锤头起重机在伺服控制下摆动，嵌入底座的 Arduino 播放真实的倒计时音频。机组人员走上狭窄的过道，发动机亮了，服务臂也摆了回来。然后在*的抵抗*中，土星五号开始在五根火焰柱上从发射台缓缓升起。[Mark]使用三个步进器驱动线性致动器来提升模型；五根透明塑料管内的 WS2812s 串串巧妙地提供了火焰效果。我们不得不说，把价值 1969 英镑的珍贵模型放在这样的电梯上需要一些勇气，但这种效果非常值得冒险。

这个项目有一个伟大的外观，显然是一个爱的劳动，是对阿波罗计划的许多成功的巨大敬意。我们的页面上还有一大堆其他阿波罗时代的黑客，包括[一个复制的 DSKY](https://hackaday.com/2020/09/23/apollo-dsky-replica-looks-the-part/) 、[一个更新的自动增益控制](https://hackaday.com/2019/08/04/apollo-guidance-computer-saved-from-the-scrap-yard/)和[一个任务控制大板背后的样子](https://hackaday.com/2020/10/29/a-look-behind-the-big-boards-at-mission-control-in-the-golden-age-of-nasa/)。
