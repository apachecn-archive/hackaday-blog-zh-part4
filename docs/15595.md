# 学生反抗热感应胯部监视器监视装置

> 原文:[https://hack aday . com/2022/12/12/students-rebel-against-heat-sensing-dick-monitor-surveillance-devices/](https://hackaday.com/2022/12/12/students-rebel-against-heat-sensing-crotch-monitor-surveillance-devices/)

监控已经成为现代生活中无处不在的一部分。公共场所里里外外都布满了闭路电视摄像头。近年来，这项技术已经扩展到了郊区，门廊摄像头进一步扩大了大型技术和执法部门的视野。

除了单纯的摄像头之外，各公司也在争先恐后地开发各种新设备来监视个人。一种旨在跟踪学生的设备很快引起了东北大学学者的愤怒，这群人进行了反击。

## 你在吗？

有问题的设备是 Spaceti 公司开发的“占用监控系统”的一部分，Spaceti 公司专门致力于最大限度地提高商业建筑的效率。该大学在课桌下安装了一系列大致对准胯部高度的热传感器，旨在检测人类(或其他合适的温暖物体)何时坐在课桌旁。

值得注意的是，传感器是在没有通知受影响的学生，也没有征得他们同意的情况下连夜安装的。这发生在东北大学跨学科的科学与工程综合体(ISEC)，网络安全和隐私研究所的所在地。正如你所料，在这个领域工作的研究生敏锐地察觉到异常硬件在一夜之间神秘地出现。得知自己突然遭到监视，他们也相当不安。

此事很快被大学行政部门处理，该部门表示，这些传感器将用于研究该设施中桌子的使用情况。接下来的解释和与学生团体举行的会议并没有减轻对新技术的担忧。该大学没有为这项研究寻求伦理批准的披露引起了愤怒，传感器本身的突然到来也引起了愤怒。管理人员试图证明这项研究不需要伦理批准，因为传感器跟踪的是“热量”而不是人类。这个借口在这所大学有安全意识的研究生中站不住脚。

> 现在，这些传感器找到了一个新家，作为公共艺术品放在 ISEC 大厅的地板上。本着 [#ISEC](https://twitter.com/hashtag/ISEC?src=hash&ref_src=twsrc%5Etfw) 的精神，这是一个跨学科的工作，来自无数部门的研究生的贡献。
