# 磁性黏液机器人的攻击

> 原文:[https://hack aday . com/2022/05/14/攻击磁性粘液机器人/](https://hackaday.com/2022/05/14/attack-of-the-magnetic-slime-robots/)

[张丽]和他在香港中文大学(CUHK)的同事开发了一种粘性物质，它可以在复杂的环境中导航，长出“手臂”，抓住电线并移动它，封装一个小物体并携带它。正如研究论文中所解释的，秘密在于制造机器人的非牛顿材料。

你可以在家里用玉米淀粉和水制作一种类似的混合物，通常称为“粘液”。慢慢变形，会像流体一样运动。迅速变形，它的行为就像一个弹性固体。CUHK 的版本是聚乙烯醇、玻璃涂层钕铁硼微粒(钕磁铁)和硼砂。

这种双重行为让机器人可以做出惊人的事情。放在一个表面上，他们通过用磁铁在下面拖动来使 blob 延伸伪足，然后使用圆形场来使它抓住并传输导线。他们在另一个轴上使用了类似的技术来吞下一个物体。CUHK 小组正在推广这种方法，将其作为一种取回体内异物的方法(比如意外吞下的纽扣电池)。

[![](../Images/6460040ee80108cbb186bb979211a57a.png)](https://hackaday.com/wp-content/uploads/2022/05/slimebot_detail.png)

Researchers will need to develop a non-toxic coating before it can be used in the body.

钕磁铁是通过在强磁场中烧结 Nd2O3 或 NdFeB 制成的。从 SigmaAldrich 可以买到 Nd2O3，价格非常便宜。聚乙烯醇和硼砂很容易得到。这似乎是一个业余爱好者可行的项目(Nd 有毒，使用注意事项)。

我们讨论微型机器人已经有一段时间了。早在 2014 年，我们就报道过[群体微型机器人](https://hackaday.com/2014/04/23/micro-robots-are-scary-awesome/)。[这个项目使用外部磁场来移动小型钕磁铁](https://hackaday.com/2014/06/07/micro-robots-made-at-home-who-needs-darpa-funding/)，早在 2014 年[我们就报道过这个领域的早期工作](https://hackaday.com/2008/06/13/magnetically-actuated-microbots/)。
