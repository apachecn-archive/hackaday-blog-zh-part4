# 没有危险的开封

> 原文：<https://hackaday.com/2019/05/19/decap-ics-without-the-peril/>

无论是向我们展示大规模半导体制造早期的经典器件，还是对最新最棒的产品进行逆向工程，我们中很少有人没有对 IC 去盖器的工作着迷过。但是他们的工作经常需要一些核心的科学设备或者特别危险的化学物质。我们从来没有想过我们可能会加入到这种乐趣中来。[普通人]打算改变这一切，通过使用普遍可得的化学药品和易于应用的技术去盖芯片。特别是，我们通过他们的工作发现松香——这种松香的气味你会从助焊剂中熟悉——可以用来溶解 IC 封装。

当然，面对焊接时容易溶解的集成电路不会取得商业上的成功，所以使用助焊剂的实验几乎没有成功。然而，纯松香似乎是一种有效的解封剂。[普通人类]给我们展示了一个在里面煮过的主板电压调节器。当用丙酮去除松香时，碎片中就有硅芯片，提醒我们这些东西有多小。我们确信你们都渴望亲自尝试一下，现在，请花点时间看看下面的视频，它展示了他们在 CCC 大会上的演讲。

芯片解封的大师当然是[Ken Shirriff]，他的作品我们已经介绍过很多次了。我们的编辑[Mike Szczys]去年采访了他，非常值得一看。
