# 实心橡木拱廊橱柜:刨花板不行的时候

> 原文：<https://hackaday.com/2021/01/08/solid-oak-arcade-cabinet-when-particle-board-wont-do/>

在 80 年代和 90 年代初的电子游戏街机全盛时期，拥有一个自己的街机柜是那些长大的人的共同梦想。这是一个相当普通的构建，不需要太多的专业知识。另一方面，这个橱柜[为橱柜本身](https://www.reddit.com/r/DIY/comments/kr0cji/i_built_a_solid_oak_arcade_cabinet_with_pixelated/)做了所有的阻挡，展示了令人印象深刻的木工专业水平。

橱柜外壳由红橡木板制成，创作者[obst repruss]将其锯平，然后粘合在一起，以创建各种面板(更多细节可在他的 Imgur 相册中获得)。侧面和正面的马里奥艺术品也不仅仅是乙烯贴纸。他用各种硬木切割成小方块，在橡木表面创造像素艺术镶嵌。在漂亮的木制品完成后，建造完成了一些 USB 街机控制器，一个平板屏幕和一个运行游戏的树莓 Pi。

虽然内部是相当标准的，但我们不得不称赞木工难以置信的质量。这是对经典街机令人印象深刻的致敬，我们不介意在自己的家里也有一台类似的。然而，如果你缺少木工设备，你可以用 3D 打印机为自己买一个精致的(更小的)拱廊橱柜。
