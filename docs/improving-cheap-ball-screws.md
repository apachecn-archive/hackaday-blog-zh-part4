# 改进廉价滚珠丝杠

> 原文：<https://hackaday.com/2021/02/22/improving-cheap-ball-screws/>

大多数 3D 打印机至少在一个轴上使用丝杠。这些都是简单的装置，基本上是一个钢螺纹和一个黄铜螺母。然而，为了达到最大的精度，你应该使用滚珠丝杠。这些通常非常昂贵，但比丝杠有许多优点。[MirageC]发现了更便宜的滚珠丝杠，但是因为它们不贵，所以它们有一定的局限性。他设计了一个简单的装置来提高这些廉价滚珠丝杠的性能。

从表面上看，滚珠丝杠看起来像带有奇怪螺纹的丝杠。然而，坚果是非常不同的。螺母内部是滚珠轴承，安装在凹槽中，使螺母以更小的摩擦旋转。一个特殊的路径收集滚珠轴承，并将它们再循环到螺母的另一侧。一般来说，滚珠丝杠非常耐用，可以处理更高的负荷和更高的速度，并且需要更少的维护。与丝杠不同，它们更贵，而且通常非常坚硬。不过，它们也有点吵。

滚珠丝杠被评定为 C0 至 C10 精度，其中 C10 精度最低，价格随着精度的提高而上涨。[MirageC]展示了如何用更便宜的滚珠丝杠来代替精密磨削。这些螺钉更便宜、更硬，但比精密螺钉跳动更大。

这种跳动会在 3D 打印过程中造成抖动，这在打印件上非常明显。使用机械师的千分表，[MirageC]发现螺钉根本不直，即使是相对较差的 C7 滚珠丝杠也会更精确。

解决办法？3D 打印零件的巧妙排列。滚珠轴承和磁铁。该装置允许螺母横向移动，而不会将其传递到印刷床上。这是一个聪明的设计，似乎工作得很好。
