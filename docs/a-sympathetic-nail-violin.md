# 同情的钉子小提琴

> 原文：<https://hackaday.com/2021/03/03/a-sympathetic-nail-violin/>

作为一个黑客社区，我们对美丽独特的乐器并不陌生。由尼古拉斯·布拉斯(Nicolas Bras)制作的一把富有同情心的钉子小提琴是这个兼收并蓄的家庭中受欢迎的新成员。Nicolas 从一个简单的想法开始，在一块木头上钉一个钉子，通过将钉子敲进木头来调整音高，他扩展了这个想法。通过仔细的计划和调整，指甲可以具有交感神经的特性。这些特性意味着当一个指甲通过琴弓弹奏时，它会引起其他指甲发出声音，产生和声和延音。

经过一点仔细的木工加工和少量的金属加工，一件乐器制作完成了。它提供了巨大的灵活性，因为它可以通过拉弓、用手指拨弦或弹奏来演奏。有几层指甲，每层都有一对交感神经指甲。这允许多样化和多功能的工具。

在 Hackaday，我们似乎对微型小提琴情有独钟，无论是物理的 T1 还是 T2 的 T3。虽然钉子小提琴可能看起来不像你的传统小提琴，但我们肯定能欣赏它创造的美妙音乐。
