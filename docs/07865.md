# 更年期投资

> 原文：<https://hackaday.com/2020/09/23/investing-in-menopause/>

如果偶尔疼痛，我们大多数人都会去买非处方药，但更年期不会这样。潮热不会像头痛那样形成，眩晕不会等到好时机，恐慌症不会自己发作。[预测和解决荷尔蒙的突然变化](https://hackaday.io/project/173355-menesto)是 Menesto 背后的意图，这是一件带有传感器、冷却装置和配套应用程序的背心。

温度计和湿度检测器监测皮肤的温度和湿度峰值，以识别穿着者何时出现潮热。当一个事件被记录时，一个风扇吹过帕尔贴面板的冷侧，希望能在背心内提供足够的冷空气。珀尔帖面板是一种热电热机，它将能量从一个陶瓷板转移到另一个陶瓷板，因此一半变冷，而另一半变热。电源来自可充电的 18650 电池，所有硬件都与运行 Arduino 的 NodeMCU 上的 ESP8266 通信。

下一个最酷的东西是内置的恐慌按钮(恐慌发作和焦虑不太常见，但有些人在更年期会经历)，它可以与配套的应用程序对话以发送电子邮件。收件人名单上的任何人都会收到佩戴者遇险的消息，以及他们手机的位置。所有的硬件都连接在一起并在说话，因此一种可穿戴的测试设备即将问世，以解决循环和散热等问题。还有其他漂亮的功能，像内置的[补水提醒](https://hackaday.com/2018/07/28/h2go-keeps-us-from-drying-out/)和[手动冷却模式](https://hackaday.com/2014/03/02/this-desktop-air-conditioner-is-really-cool/)。
