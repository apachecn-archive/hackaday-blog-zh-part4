# 助力车转向灯，现在装了更多的牛铃

> 原文：<https://hackaday.com/2019/06/07/moped-turn-signals-now-with-more-cowbell/>

提示[克里斯托弗·沃肯]迷因，是时候用更多的牛铃给[助力车打转向灯了。因为牛铃少的助力车转向灯显然是助力车转向灯中的劣等品。](https://www.youtube.com/watch?v=DJUqErCCAJw)

似乎[Joel Creates]和我们一样患有节奏识别障碍。一个有节奏的声音，如转向灯，与我们看似无限的颅内播放列表中的任何一首歌曲之间的最轻微的相似性都会在一天的剩余时间里提示那首歌。[Joel]在他的转向灯中听到了“(不要害怕)收割者”，这自然导致了对更多牛铃的需求。所以有了一个汽车门锁执行器，一个继电器，一个临时制作的拍板，许多热熔胶和电缆带，他的滑板车的前面现在装饰有一个与转向灯同步的牛铃。下面的视频显示，它在流量上的吸引力有点有限，但至少[乔尔的爸爸]被它逗乐了。

(乔尔)在这件事上按照自己的节奏前进，这是值得称赞的。这可能有点奇怪，但没有牛铃网那么奇怪。
