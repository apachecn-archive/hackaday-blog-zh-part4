# 旧电锯改作厨房使用

> 原文：<https://hackaday.com/2019/08/19/old-chainsaw-repurposed-for-kitchen-use/>

有许多方法可以在停电时保持关键设备的运行。也许是一台电脑的 UPS，一组给手机充电的太阳能电池板，或者甚至是一台让你的冰箱或空调工作的发电机。对标准搅拌机的这种改进也能让你在断电时仍能制作美味的饮料。它靠汽油运行。

该建筑使用一个旧电锯来驱动搅拌机的叶片。[Bob]能够设计和制造一个全新的传动系统来让这个设备工作，首先是移除链锯链和杆，并将链轮连接到电机的主轴上。一条链条将它连接到一个定制的支架上，该支架支撑着角磨机的一部分，角磨机支撑着搅拌机罐。为了安全起见，再加上一个链条护板，你就会拥有一个功率比普通厨房电器稍大的搅拌机。

制作视频值得一看，即使你的无聊的电动搅拌机已经满足了你的需求。[Bob]工作的商店里有我们能想到的所有工具，包括电焊机、3D 打印机、带锯，甚至还有数控等离子切割机。它让我们想起了[这个老托尼]的商店。
