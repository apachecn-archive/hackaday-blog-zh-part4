# 公制和英制螺纹争夺超精确定位

> 原文：<https://hackaday.com/2022/09/21/metric-and-inch-threads-fight-it-out-for-ultra-precise-positioning/>

当你是一名机械师，你在贸易中的股票是精度，与你的首选单位千分之一的测量是常见的。但当你是一个模具制造商时，你的精密游戏需要更精细，能够以看似不可能的粒度定位工具和材料变得非常重要。

对于[亚当·德穆特]，也就是 YouTube 上的“机械师亚当”，对不会破产的超精细分辨率机械师千斤顶的需求导致了使用现成硬件和一些 3D 打印部件的设计。该设计围绕一个英寸公制螺纹适配器，你可以从麦克马斯特-卡尔拿起。适配器上的内螺纹是 M8-1.25，而外螺纹是 5/8 英寸-16 螺纹。这些螺纹的间距非常接近——只有 0.0063 英寸，或 161 微米。为了利用这一点，[亚当]打印了一个带有柔顺机构弹簧的笼子；保持架将螺纹部件固定在一起，并提供轴向预载荷以消除齿隙，并允许在每端安装精密钢球，以确保千斤顶的力通过每端的单点传递。千斤顶每转一圈，两端就会移动一个间距差，从而实现超精细的分辨率定位。需要更高的精度吗？尝试 M5 到 10-32 适配器，每转约 6 微米！

虽然我们之前已经看到过[不同的螺距用于精定位](https://hackaday.com/2020/03/21/screwy-math-for-super-fine-adjustments-differential-screws/)，【Adam】的方法需要加工。虽然这些千斤顶本身很有用，但[Adam]通过使用其中三个千斤顶来制作一个运动基座，可以在三个轴上进行微调。这不是一个纳米定位的 Stewart 平台，但你可以看到如何增加三个千斤顶和一些驱动器可以实现这一点。
