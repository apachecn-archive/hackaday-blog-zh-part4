# Mimic 巧妙地采用了时尚的 led

> 原文：<https://hackaday.com/2019/11/24/mimic-artfully-employs-leds-in-fashion/>

任何设定在不久的将来的科幻作品都涉及到会发光的衣服，或者里面有某种形式的电子设备。这还没有在主流时尚中发生，但[Amped Atelier]正在这个领域做严肃的工作。 *Mimic* 是他们参加 2016 年 MakeFashion Gala 的作品，[是 led 在时尚领域的成功典范。](https://www.ampedatelier.com/mimic)

模仿由两件组成，设计成模仿周围环境的鸡尾酒礼服，就像变色龙一样。led 由 Arduino 控制，配有颜色传感器。当被激活时，Arduino 可以改变 led 的颜色，以匹配呈现给发送者的任何内容。这项技术可以作为一种避免与朋友的服装发生冲突的好方法，或者向你的车发送一个你准备离开的秘密信号。

led 隐藏在吸引人的几何散射器下面，这些散射器直接 3D 打印在服装的面料上。这给服装一个有吸引力的，完成的外观，并允许扩散自然地与作品的线条流动。

这些作品表明，创造既时尚又高科技的夜光睡衣是可能的。然而，如果你想要更前卫的东西，我们也有。休息后的视频。
