# 这台电脑是一台装在红木盒子里的树莓皮笔记本电脑

> 原文：<https://hackaday.com/2022/06/09/the-compu-tor-is-a-raspberry-pi-laptop-in-a-mahogany-case/>

由[亨利·爱德华兹]设计的计算机是不属于任何一类的东西之一。这是一台翻盖式便携式电脑，虽然与大多数笔记本电脑不同，它没有内置电池。它有一个定制设计的光滑外壳，但缺乏典型的未来科幻外观。键盘可以作为输入设备，但也可以变成一种乐器。

简而言之，所有这些东西都有一点，但最引人注目的部分是加工精美的红木外壳。这两半通过两个结实的摩擦铰链和一根硅胶带状电缆连接在一起:下半部分包含键盘、扬声器、USB 端口和电源连接，而上半部分包含一个 Raspberry Pi 和一个 10 英寸的触摸屏。显示器边框具有典型的 CRT 显示器的弯曲形状，很好地符合 20 世纪 70 年代深色木材散发出的氛围。

另一个复古之处是各种电路板和前面板开关之间的连接:[Henry]使用了绕线，这是我们很久没有看到的东西了。键盘是带有手写标签的相同按键的简单网格。其他标签，如电源连接器的标签，由传统的压花带制成。

这台电脑运行的是 Debian，看起来像一台小巧的笔记本电脑一样好用。它甚至配有 USB 端口来连接外部设备，有了简单的 12 V 输入，为它找到外部电源应该不成问题。木材似乎是制造基于树莓派的笔记本电脑的流行材料:我们已经看到它们被装在任何东西里，从[木制雪茄盒](https://hackaday.com/2017/03/21/cigar-box-opens-to-raspberry-pi-laptop/)到[激光切割胶合板](https://hackaday.com/2018/03/15/a-plywood-laptop-for-your-raspberry-pi/)，甚至[难以置信的小盒子](https://hackaday.com/2019/05/28/tiny-wooden-laptop-packs-raspberry-pi/)。
