# 木制金属板冲压工具制造钢鞋头

> 原文：<https://hackaday.com/2020/05/07/wooden-sheet-metal-press-tools-make-steel-toecaps/>

如果你曾经仔细观察过典型的大规模生产的汽车，你会对压制钢板很熟悉。复杂的曲线可以重复数千次，方法是将一片钢板放在压力机的成型工具之间，施加巨大的力。同样的工作，一个熟练的打板工几个星期才能手工完成，只需一秒钟。当他想在车间穿一套 Crocs 时，这是[这里制造的东西]要解决的问题，而[需要在他的液压机](https://www.youtube.com/watch?v=IcjgrB9vTec)中制造工具来生产它们。我们在休息时间下面发布的视频展示了他的学习曲线，同时也是钣金冲压方面的一本方便的入门书。

我们看到他发现了金属板在压力下的特性，它是如何起皱和折叠的，以及工具如何需要精心设计，金属板如何需要安全地夹紧以防止这种情况发生。令人惊讶的是，他的工具是由数控加工的木材制成的，尽管我们确信它会在重复使用后磨损，但工具上的力似乎不会破坏这种材料。最后，他制作了一个多部分工具，包括冲压工具的两个部分，移动部分的加工导向，以及一套坚固的金属板来约束材料。当谈到金属板成型时，钢鞋头的应用可能不是每个人的第一想法，但我们确信这种技术可以在许多其他项目中找到应用。这是一个我们在过去已经涉足的领域，但从未如此复杂。
