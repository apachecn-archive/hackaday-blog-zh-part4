# 黑客日超级会议:尼克·普尔谈 Boardhouse 陷入困境

> 原文：<https://hackaday.com/2019/12/18/hackaday-superconference-nick-poole-on-boggling-the-boardhouse/>

到现在为止，我们都习惯了印刷电路板在艺术品中的作用，因为会议和非官方#BadgeLife 电子徽章场景中蓬勃发展的创造力。当电子设计工具的大师们出于审美而非技术原因转向生产时，结果总是会很特别。

尼克·普尔(Nick Poole)是为 SparkFun 工作的电子领域的王牌辩论家，因此在 PCB 设计方面也有着令人印象深刻的血统。[戴着一顶贝雷帽，带着令人敬畏的帽徽走上舞台，他在最近的 Hackaday Superconference](https://www.youtube.com/watch?v=rAXmzt-_NZ4) 上的演讲讲述了他在 PCB 制造中拓展可能性的经历。这是在印刷电路板介质中创建特殊作品所需技术的入门书，对于对该领域感兴趣的任何人来说都应该是必不可少的。

虽然他从将图形导入 PCB 设计包的基础开始，但他的核心内容在于通过创建互锁的 PCB 或堆叠电路板，超越单个 PCB 的二维空间，进入三维空间。

 [https://www.youtube.com/embed/rAXmzt-_NZ4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/rAXmzt-_NZ4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



例如，任何曾经将两块印刷电路板以直角焊接在一起的人都会知道，当热量使印刷电路板分层时，抬起一块铜的痛苦。他关于放置过孔来锚定层的技巧是演讲中包含的典型专业知识。

[![](img/ea5f5a174c1dc7f959b6244f5e154fed.png)](https://hackaday.com/wp-content/uploads/2019/12/nick-poole-pcb-flex-as-switch.jpg) 他继续用他的太空入侵者徽章作为例子，通过鼠标的叮咬，半透明的显示器，以及利用 PCB 手指的弹性进行的开关。这个徽章有三层，一个 APA102 可寻址 led 阵列，通过铣出的中间层和半透明的顶层发光。在这个过程中，我们学到了更多有价值的技巧。当界限被打破时，将板房技术人员的笔记作为文本放在层的非板房部分对 Gerbers 有很大帮助，因为员工经常会质疑一些明显偏离他们规范的东西。

无论是解决内角问题，确保电镀边缘的锯齿，还是将多块电路板连接在一起，我们认为在您将 PCB 工作提升到下一个水平时，这里应该有足够的东西让您忙碌。
