# 由乐高制成的模拟计算机预测潮汐

> 原文：<https://hackaday.com/2022/02/01/analog-computer-made-from-lego-predicts-tides/>

虽然海洋中的潮汐是由太阳和月亮的运动引起的，这两者都很容易观察到，但提前几天准确预测潮汐却相当困难。潮汐运动背后的数学是如此复杂，以至于一些最早的模拟计算机是专门用来进行潮汐计算的。早在 19 世纪晚期，威廉·汤姆孙爵士(更为人熟知的名字是开尔文勋爵)就设计了一台这样的“潮汐预测机”，一种令人印象深刻的齿轮和滑轮的排列。

[Pepijn de Vos]用乐高零件建造了汤姆森机器的现代版[，它给人的印象不亚于原版。共有 96 个乐高齿轮与海洋的自然节奏完美同步，而一组滑轮将四组齿轮连接在一起，以创建组成频率的总和。超声波传感器读取输出值，并将结果发送回 PC。](http://pepijndevos.nl/2022/01/30/predicting-the-tide-with-an-analog-computer-made-from-lego.html)

[Pepijn]遇到的一个有趣的问题是，乐高齿轮只能提供非常有限的齿轮比，他在博客中对此做了详细的解释。为了使潮汐计算达到任何精度，他需要将许多齿轮串联起来，而不会在机械装置中产生太多的摩擦和反冲。优化这种设置是一项重要的任务，它本身就需要大量的计算能力。

正如你在下面嵌入的视频中看到的，这台机器做出了非常流畅的动作，非常准确地对应了潮汐的实际运动。如果你对模拟潮汐预测器背后的科学感兴趣，我们有[篇关于那个](https://hackaday.com/2015/10/08/how-analog-tide-predictors-changed-human-history/)的深入文章。
