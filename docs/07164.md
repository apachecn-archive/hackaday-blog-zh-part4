# 3D 打印机能为自己打印出更好的细丝吗？

> 原文:[https://hack aday . com/2020/07/20/can-a-3d-printer-print-better-filament-for-self/](https://hackaday.com/2020/07/20/can-a-3d-printer-print-better-filament-for-itself/)

3D 打印部件的强度通常远不及同等的注塑部件和持续热处理等技术，尽管这些技术很有效，但通常会使部件变形而无法使用。

【CNC Kitchen】正在[调查最近一篇论文的结果](https://www.youtube.com/watch?v=z3aUGmfnmIg)(视频，嵌入下方)，该论文描述了一种由“星形”聚碳酸酯芯增强的新型 ABS 长丝，作者声称这种排列在退火过程中对变形具有弹性，退火过程通常是增加零件强度所必需的。虽然研究人员可以获得制造这种复合材料所需的专业设备，但[CNC Kitchen]的解决方案是简单地使用他的双挤出机设置来直接打印所需的混合长丝，我们认为这与现在的旧学派 RepRap“打印你的打印机”的观点产生了强烈的共鸣。

印刷的细丝看起来具有合理的尺寸精度，并且在没有连接喷嘴的情况下将印刷的线轴通过加热器块，确保不会有明显的堵塞。视频的其余部分侧重于在退火过程后，对普通聚碳酸酯、ABS 和这种新的混合长丝之间的强度和变形进行非常彻底的比较。虽然他的结论是混合的结果，但能够组合和打印自己的混合细丝是超级酷的，这本身就是一个成功！
