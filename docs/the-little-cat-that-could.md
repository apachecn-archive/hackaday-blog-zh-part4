# 这只小猫可以

> 原文：<https://hackaday.com/2018/10/10/the-little-cat-that-could/>

大多数人类花一年时间来学习他们的第一步，他们是出了名的笨拙。[Hartvik Line] [用更少的时间教会了一只机器猫走路](https://www.youtube.com/watch?v=zNXgT2csQ7A) [YouTube 链接],但这只猫比一个学步前的孩子有几个优势。第一个优势是它有四条腿，而第二个优势来自一种叫做遗传算法的机器学习技术，这种技术在两个小时内超过了人类的微调。这是一个非常好的基准。

机器人本身是一件令人印象深刻的作品，灵感来自瑞士 EPFL 研究所的机器人。所有这些瑞士工程对一个人来说都不容易编程，更不用说对一个学生来说了，但事实就是如此。她被称为“谢妮”，是挪威斯塔万格大学[哈特维克]硕士论文的一部分。机器学习的效率很快超过了人类的干预，如果底盘损坏，它甚至可以重新学习走路。

我们已经为[观察了](https://hackaday.com/2013/01/29/genetic-algorithms-become-programmers-themselves/)[遗传算法编程](https://hackaday.com/2015/01/07/genetic-algorithm-programmer-gets-functions/)超过五年，天网还没有出现，然而我们有了一只迈出第一步的机器人小猫。
