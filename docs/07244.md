# 激光切割您的 RGB LED 表

> 原文:[https://hack aday . com/2020/07/24/laser-cutting-your-way to-an-RGB-led-table/](https://hackaday.com/2020/07/24/laser-cutting-your-way-to-an-rgb-led-table/)

你有 RGB 键盘，甚至可能还有 RGB 鼠标。但是，如果你没有一个 RGB 表来玩游戏，你真的可以称自己为彩色 LED 大师吗？我们认为你已经知道答案了。幸运的是，[正如【ItKindaWorks】在他的最新项目](https://github.com/ItKindaWorks/smartTable)中所展示的，构建自己的项目很容易。假设你有一个足够大的激光切割机…

[![](../Images/db660cfb33f87953ba9e295f15779889.png)](https://hackaday.com/wp-content/uploads/2020/07/rgbtable_detail2.jpg) 表的构造相当简单明了。使用 80 瓦的激光切割机，他在一张 MDF 中放入一个通道以接受 RGB LED 灯条，一个口袋以容纳 Qi 无线充电器，以及一个孔以穿过所有电线。然后用第二层坚固的中密度纤维板作为背衬。

接下来，一块薄木板进入激光切割机。在休息后的视频中，你可以看到它卷起的自然趋势给[ItKindaWorks]带来了一点麻烦，但当战略加权时，它最终会展开。然后，他用激光在单板上打出一系列小孔，当它被粘在中密度纤维板上时，led 发出的光就会穿过这些小孔。铺在塑料条上的几条塑料条既能漫射光线，又能支撑顶部表面。

[![](../Images/2acb79a22fb7c21cbf560e1c7e407c69.png)](https://hackaday.com/wp-content/uploads/2020/07/rgbtable_detail1.jpg) 最终的结果是真正的华丽和非常未来的感觉。假设你已经有了设备，这也是一个相对简单的实验概念。当你[给你的军火库](https://hackaday.com/2018/03/15/a-plywood-laptop-for-your-raspberry-pi/)增加一个高功率激光器的时候，这是[独特建筑技术可能](https://hackaday.com/2019/04/02/custom-inflatables-are-only-a-laser-beam-away/)的又一个例子。
