# Game Boy Advance SP Case Mod 增加了电池容量和现代接口

> 原文:[https://hack aday . com/2021/08/24/game-boy-advance-sp-case-mod-adds-battery-capacity-and-modem-interfaces/](https://hackaday.com/2021/08/24/game-boy-advance-sp-case-mod-adds-battery-capacity-and-modern-interfaces/)

虽然没有什么比在原始硬件上运行复古游戏更好的了，但由于与现代充电器和耳机不兼容，在当今时代使用旧游戏机可能会很麻烦。[蒂托]和[凯尔]一起合作，用一整套新功能更新了任天堂的 Game Boy Advance SP，紧紧地装在一个 3D 打印的替代后壳中。(视频，嵌在下面。)

原来的 600 毫安时电池已经被一个 1600 毫安时的电池包取代，增加了几个小时的屏幕时间。Qi 标准无线电源模块以及 USB-C 连接器允许在不携带原始电源适配器的情况下为电池充电。蓝牙模块可以使用无线耳机，3.5 毫米插孔也可以使用经典耳塞，当 SP 取代最初的 GBA 时，这一功能就失去了。

新的后壳正好适合原来的控制台，使其厚约 10 毫米。虽然这使得它的便携性稍差，但对于那些手大的人来说，它显然更舒适。新功能是使用现成的电路板实现的，这些电路板通过焊接到 GBA 电路板上所需点的飞线连接在一起。原来的连接器和开关仍在原处并起作用，如果您想将设备恢复到原始状态，可以撤消整个操作。

还有人[在原来的 GBA 上加了 USB 充电](https://hackaday.com/2021/02/06/gba-gets-homebrew-usb-c-charging-upgrade/)，甚至[把那个手持的](https://hackaday.com/2021/08/19/the-stretch-limo-of-game-boys/)拉长了一倍。但是增加重要的新功能需要替换手持设备的全部内容。
