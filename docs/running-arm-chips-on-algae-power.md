# 运行藻类能源的手臂芯片

> 原文：<https://hackaday.com/2022/05/14/running-arm-chips-on-algae-power/>

AA 电池有多大，能让 ARM Cortex M0+运行六个月？嗯…可能是 AA 电池，但显然，那不值得一提。但是剑桥大学的研究人员已经建立了一个蓝绿色海藻细胞来完成这项工作。

正如你所料，藻类需要光，因为它们通过光合作用产生能量。然而，与传统的太阳能电池不同，这种藻类至少在一段时间内可以在黑暗中持续产生能量。据推测，藻类在白天储存能量，并在晚上释放能量，以度过自然发生的黑暗时期。

光合作用发电并不是一个新想法，因为光合作用会释放电子。典型的电池有金电极和某种质子交换膜。你可以在下面看到一个来自剑桥的关于光合作用发电的视频。当然，请记住，Cortex M0+能够以非常低的功耗工作。短期内不要去找那个海藻驱动的点焊机。

人们倾向于将电能作为能源，但还有其他方式来利用光合作用。例如，我们在过去已经看到藻类给鸡洞提供燃料。更不用说我们已经看到藻类以一种新颖的非电方式为机器人提供动力[。](https://hackaday.com/2022/01/27/thats-no-moon-its-an-algae-robot/)
