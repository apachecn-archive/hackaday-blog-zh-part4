# 对于这个球窝机器人，熟能生巧

> 原文：<https://hackaday.com/2022/05/20/practice-makes-perfect-for-this-ball-and-socket-robot/>

球窝关节是有用的，但是让一个零件在一个由磁铁固定的球体表面上滑动，需要大量的摆弄才能正确。我们佩服坚持，佩服钉死所有细节。[马修·芬利]已经用他的球窝机器人做到了这一点。他正在制作第六版，这证明了他对实现理想正义的渴望。幸运的是，他记录了每个版本。

[版本一](https://www.youtube.com/watch?v=v6GpraaxU2o)，由 DIY 圣诞装饰球制成，围绕径向轴没有稳定性，振荡严重。[版本二](https://www.youtube.com/watch?v=19vKO8CK4ag)演示了球内机构居中的问题。版本 3 修复了这个问题(在同一个视频中有涉及)。然后[版本四](https://www.youtube.com/watch?v=Cp9y-Nf8Hfs)修复了许多组装问题，并用 Arduino 取代了伺服控制器，但“手臂”部分太小，机械性能不稳定。

[版本五](https://www.youtube.com/watch?v=h_1I-x847ls)使用了装配式轴承。马修用气枪子弹做球。这不是个好主意。组装是一场噩梦。所以到第六版为止的所有这些进步都显示了他不断改进的技术。艺术家说“在你的过程中工作，而不是在你的作品中工作”。他对需要什么变得更加善于分析。他已经开始测量机器人的强度，并处理了一些问题，如增加限位开关，使它不会在行程极限时崩溃。

有趣的建造，让我们想起了[【斯蒂芬·杜弗兰】的 BB-8。](https://hackaday.com/2016/11/30/my-diy-bb-8-problems-solutions-lessons-learned/)
