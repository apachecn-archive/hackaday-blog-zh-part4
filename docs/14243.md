# 如何成为一名乐高潜水艇员

> 原文：<https://hackaday.com/2022/07/18/how-to-become-a-lego-submariner/>

潜水艇必然是一种复杂的、对安全至关重要的机器，但它所依赖的原理却很简单。浮力是可变的，通过将水泵入和泵出水箱，允许船只控制其所处的深度。[砖块实验频道]有一系列帖子详细描述了一艘工作潜艇的建造，其船体由塑料管制成，机械装置由乐高制成。

在这种潜艇中，浮箱是一个由乐高电机操作的注射器，推进力来自一对乐高电机，通过巧妙的磁耦合驱动，以避免轴封。为了监控深度，有一个激光距离传感器和一个压力传感器，还有一个 Raspberry Pi Zero 来控制整个节目。

在休息时间下面的视频中，你可以看到这艘船在不同深度的游泳池周围滑行，然后带着船上的镜头沿着一条阴凉的小溪开始更长的旅程。这是一艘非常实用的潜艇，我们希望能尝试一下。

这是 4.0 版本，它显示。我们去年[看过 3.0 版本](https://hackaday.com/2021/09/19/lego-submarine-gets-balloon-ballast-system/)，它绝不是我们展示的第一艘潜艇，[这是一艘由 PVC 管制成的](https://hackaday.com/2021/12/10/build-your-own-submarine/)。
