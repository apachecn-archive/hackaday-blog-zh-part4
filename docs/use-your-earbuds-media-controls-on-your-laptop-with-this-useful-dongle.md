# 通过这款有用的加密狗，在笔记本电脑上使用耳塞的媒体控制

> 原文：<https://hackaday.com/2019/11/09/use-your-earbuds-media-controls-on-your-laptop-with-this-useful-dongle/>

[David]带来了他设计得非常好的“Thumpware Media Controller ”,可以让你的手机耳机控制你电脑上的媒体播放。

我们意识到，一些电脑支持手机耳塞上的额外引脚，但至少我们中的一些人经历过习惯性地伸手触摸耳塞上的媒体控件，却听到一个非活动按钮发出的绝望的咔哒声的沮丧感(无论多么小)。这解决了这个问题，假设你[至少还拿着那副 3.5 毫米耳机](https://hackaday.com/2016/10/25/death-to-the-3-5mm-audio-jack-long-live-wireless/)。

媒体控制由 PIC16 截取，一个小电路板将信号分成一个 3.5 毫米插头和一个 USB 端口。真正给我们留下深刻印象的是专业外观的设计和外壳。布线、组装和应变消除都经过了精心的规划。总的来说，这是一种享受。

所有的文件都是可用的，所以通过一点焊接、黑客攻击和仔细打磨，有人可以为自己的设置组装一个专业外观的加密狗。
