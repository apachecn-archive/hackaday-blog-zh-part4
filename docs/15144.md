# 2022 年网络平台竞赛:Keezyboost40 是一个伪装成键盘的网络平台

> 原文:[https://hackaday . com/2022/10/20/2022-cyber deck-contest-keezyboost 40-is-a-cyber deck-masquerading-as-a-keyboard/](https://hackaday.com/2022/10/20/2022-cyberdeck-contest-keezyboost40-is-a-cyberdeck-masquerading-as-a-keyboard/)

对于超级强大的网络甲板来说有一些东西要说，但是对于挤进一个微小的形状因素中的不太强大的甲板来说也有一定的吸引力。[Christian Lo]设计了一个 cyberdeck，它看起来像一个简单的直线键盘，但运行在一个更加灵活的环境中。

你可以在 QMK 上玩游戏和动画[，但是【Lo】觉得一个不同的框架会给他更多的灵活性来真正扩展这个 Raspberry Pi Pico 驱动的平台所能做的极限。他决定使用基于 Rust 的固件和 keyberon 库，他说，“感觉就像是我在控制固件。”虽然董事会目前使用 Rust，但[Lo]表示，他愿意讨论其他固件选项来实现他的目标，比如为董事会开发一款虚拟宠物游戏。](https://docs.splitkb.com/hc/en-us/articles/360010533820-What-can-you-use-an-OLED-display-for-)

PCB 被描述为“bog 标准”,可能的例外是将 Pi 放在板上的切口中，以尽可能保持低调。代价是电路板刚度降低，与微控制器的连接应力可能增加。
