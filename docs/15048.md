# USB 电源隔离器保持烟雾在

> 原文:[https://hack aday . com/2022/10/12/USB-power-isolator-keep-smoke-in/](https://hackaday.com/2022/10/12/usb-power-isolator-keeps-smoke-in/)

任何做过电子项目的人都知道，任何好的设计最重要的部分是确保所有元件内部都有神奇的烟雾。有很多方法可以确保烟雾留在那里，但最重要的方法之一是确保电源被隔离。但是，如果你使用电脑上的 USB 端口作为电源，[将它与电脑](https://github.com/wagiminator/ATtiny412-USB-Power-Isolator)隔离可能会稍微复杂一些。

电源是基于一个小变压器和一组二极管作为整流器。当然，虽然变压器在隔离电源方面很棒，但在 DC 方面却不太好。这就是 ATtiny 微控制器的用途。它处理 MOSFETs 的高速开关，驱动变压器并处理一些功率调节。作为这个项目的一部分，还有两种不同的电源——第一种产生+5V，就像普通的 USB 插头一样，另一种产生+5V 和-5V。重要的是不要把这两者混为一谈，否则狡猾的蓝色烟雾可能会逃逸。

项目页面详细介绍了该设备的操作，因此如果对电学理论感兴趣，这绝对值得一读。将有价值的计算机与原型电路隔离当然很重要，但如果你正在寻找一种隔离完整 USB 连接的方法，看看这个版本，它包括 USB 到 FTDI 适配器的[隔离。](https://hackaday.com/2014/04/27/galvanic-isolated-ftdi-saves-your-computer/)
