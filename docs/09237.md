# 音叉让这个复古的数字时钟滴答作响

> 原文：<https://hackaday.com/2021/02/13/tuning-fork-keeps-this-throwback-digital-clock-ticking/>

无论你想建造哪种时钟，你都需要建造某种振荡器。无论是钟摆、摆轮，还是铯或铷的原子跃迁，都需要以可预测的方式来回运动，以形成时钟的时基。虽然音叉可能不是世界上最好的时钟，但它绝对符合要求，是一个非常有趣的时钟。

这个版本的一个好处是【Kris Slyka】从[一个音叉钟获得了灵感，我们前阵子讨论过这个音叉钟](https://hackaday.com/2019/08/29/impractical-clock-uses-tuning-fork/)——我们喜欢有人把一个很酷的概念变成他们自己的。虽然这两种时钟都使用 440 赫兹的音叉——对于喜欢音乐的人来说，音叉高于中音 C——但[克里斯]改变了它们的激发方法。她使用了一对现成的感应器，放置在每个臂的末端附近，并用一个强大的钕磁铁桥接，以感知 440 赫兹的振动，并提供保持叉子振动所需的踢力。

至于建筑的美学，我们认为克里斯真的做到了。使用通孔组件、老式七段显示器和自制蚀刻 PCB，她能够捕捉到真正有效的复古外观。RS-232 端口和钟罩外壳完善了这种感觉，尽管我们不确定设计的自定义字符集[Kris]——它很酷，但如果没有一点实践，其他人很难读懂。无论如何，这是一个有趣的构建，我们可以想象时钟发出的连续音调非常令人愉快。
