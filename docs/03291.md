# 机械震音以老派的方式做事情

> 原文：<https://hackaday.com/2019/06/12/mechanical-tremolo-does-things-the-old-school-way/>

“颤音”一词在音乐词典中有多种含义。至少在吉他界，颤音效果指的是振幅的周期性变化。这通常是通过固态电子设备实现的，但也会让人想起多年前哈蒙德风琴通过旋转 Leslie 扬声器发出的声音。[HackaweekTV]决定用老方法来做事情，[创造一个他自己的机械颤音效果(Youtube 链接，嵌入下面)。](https://www.youtube.com/watch?v=75kyOjfI_y4)

在电子方面，信号只需通过一个线性音频电位计。该效果是通过快速上下循环该电位计产生的。这种运动是通过一个从 Roomba 中抢救出来的齿轮电机实现的，它转动一个凸轮。弹簧从动件位于凸轮顶部，并连接到电位计。

发展中存在一些挑战。框架的刚性是一个问题，从动轮在凸轮上有阻碍的问题。然而，通过一些仔细的迭代，他们能够让一切正常运行。最终的项目听起来很棒，放大器开大了，就不用担心运动部件的声音了。

自然地，[你总是可以用 555 代替](https://hackaday.com/2011/11/29/fabbing-a-guitar-tremolo-stompbox/)来构建一个颤音。休息后的视频。
