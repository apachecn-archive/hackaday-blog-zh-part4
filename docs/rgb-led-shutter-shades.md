# RGB LED 快门灯罩

> 原文：<https://hackaday.com/2020/08/05/rgb-led-shutter-shades/>

[splat238]又回来了，带来了另一个很酷的 RGB LED 显示屏项目。在过去的几周里，我们在思考我们的读者是否已经受够了这些，但是这些年来我们已经了解到[你永远不会有太多的 LED 项目](https://hackaday.com/2020/05/18/jumbo-led-matrix-brings-classic-sprites-to-life/)。

不像我们在之前报道的那样制作一个很酷的面具，【spla决定设计一些百叶窗。真正酷的是他使用 PCB 本身作为框架，[类似于我们见过的另一个黑客](https://hackaday.com/2012/05/22/macetechs-led-glasses-prototype/)，我们确信这也使他的设计过程更加方便。

[splat238]将他的电路板预先组装好，因为手工焊接所有这些 led 非常困难。这种设计有 76 个。他向读者介绍了如何组装电路板，提供了他有过良好经验的可靠制造和组装工厂的信息，这非常有帮助。如果你还没有一站式服务，或者从未在之前为组装设计过[，这是非常可靠的信息。](https://hackaday.com/2017/05/10/designing-for-fab-a-heads-up-before-designing-pcbs-for-professional-assembly/)

这款眼镜使用了基于 ESP8266 的微控制器，因为它有足够的空间来存储 LED 图案，并有可能在以后的版本中包括 WiFi 控制。然而，我们认为你会很高兴用一个简单的按钮简单地切换模式。

led 最大使用 2.5 A，并依靠外部电源，所以在长时间佩戴时，你可能要非常小心。也许可以考虑做一点 [PWM](https://hackaday.com/2020/02/06/tool-writes-your-pwm-code-for-you/) 来帮助降低功耗。

又一个很酷的项目[splat238]！让他们继续来。
