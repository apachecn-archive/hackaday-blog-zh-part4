# 一个按钮的多种用途

> 原文：<https://hackaday.com/2018/12/05/many-uses-for-a-single-button/>

当在头脑中构建一个简单目标的项目时，随着项目经历最初的原型阶段，我们会添加越来越多的开关、按钮和复杂性，这种情况并不少见。像这样的特性蔓延往往会导致混乱的局面，而不是一个可用的项目。然而，只要足够专注，就有可能意识到事情正在发生，并坚持原来的计划。另一方面，[这个不止一次使用的单按钮项目](https://www.youtube.com/watch?v=LC8FoGw8q8I)似乎是特性蠕变的反义词。(YouTube，下面嵌入。)

[Danko]的项目只有一个目标:在只使用一个按钮和一个小屏幕的情况下尽可能地有用。现在，这种小型手持设备可以用作秒表、计数器，甚至可以播放《flappy bird》的初级版本。它使用 Arduino Pro Mini，这是一款在 I2C 上运行的 64×48 有机发光二极管屏幕，并有一个微型 100 毫安时 3.7V 电池为一切供电。如果你以前也从未使用过这么小的屏幕，那么这个视频值得一看。

从一个只有一个按钮的设备中获得三种功能是一个非常令人印象深刻的壮举，如果你能想到任何其他方式来获得更多的有用性，请务必在下面的评论中留下它。[Danko]对小屏幕的简单项目也不陌生。我们最近展示了他的[自制 Arduino 计算器](https://hackaday.com/2018/11/06/pocket-sized-arduino-calculator-makes-a-great-first-project/)，它使用了一个更小的屏幕。
