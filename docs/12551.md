# 在线工具将 STLs 转化为 3D ASCII 艺术

> 原文：<https://hackaday.com/2022/01/22/online-tool-turns-stls-into-3d-ascii-art/>

如果你足够努力，我们在这些页面上介绍的大多数项目都有一些实用价值。它们可能看起来很无聊，但是通常有一些事情迫使黑客投入时间和精力去做。当然，这并不意味着我们没有分享那些只是为了好玩的项目，这当然描述了这个在线 3D ASCII 艺术生成器。

但是等等——也许这并不完全正确。毕竟，[Andrew Sink]为此投入了大量时间，也为它的前任[他的自动 3D 低多边形生成器](https://lowpoly3d.xyz/)投入了大量时间。该项目导致了当前的工作，像以前一样，将 STL 模型作为输入，这次将其转换为 ASCII 艺术渲染。用于模型着色的字符集是可定制的；不过，使用默认设置时，阴影效果出奇的好。如果你愿意，你还可以切换到黑白主题，用鼠标在模型中导航，甚至将 ASCII 艺术导出为 PNG 或原始文本文件，无疑适合发送到[你的牵引式打印机](https://hackaday.com/2020/01/09/vintage-mini-inkjet-prints-on-demand-ascii-art/)。

[Andrew]的代码都在[GitHub](https://github.com/AndrewSink/STL-to-ASCII-Generator)上，它自由地使用了 three.js 库，所以也许扩展他的 3D JavaScript 技能真的是这一个隐藏的实用方面。并不是说它需要一个——我们认为它很酷，只是因为它的神奇之处。
