# 看看不同的空调有多么不同(在内部)

> 原文:[https://hackaday . com/2019/12/15/see-how-wild-different-air-conditions-can-on-the-inside/](https://hackaday.com/2019/12/15/see-how-wildly-different-air-conditioners-can-be-on-the-inside/)

空调容易想当然。从冰箱到气候控制，如果它们消失了，我们大多数人都会非常想念它们。这是吸引[Josh Levine]对空调感兴趣的部分原因，他提供了几个不同单元的有趣参观，以及它们可以有多么不同，尽管它们的工作方式基本相同。

[![](../Images/c47241d0ad81c49feaee3f3791bbcdc6.png)](https://hackaday.com/wp-content/uploads/2019/12/Air-conditioner-Tours-Square.png)

That white PCB is crucial (for running the bluetooth speaker and LED flashlight, that is.)

空调试图脱颖而出的一个方法是保持安静，大部分噪音来自风扇和压缩机。一个单元(海尔 Serenity)的目标是尽可能成为最安静的单元，但尽管这一努力充其量只能带来好坏参半的结果，但看到[Josh]介绍他们试图降低噪音的不同方法仍然很有趣](https://youtu.be/74IFbYYAhFM)(YouTube，嵌入下方)。限制噪音的元素包括为室内和室外风扇使用独立电机的不寻常步骤，甚至是确保它们完全平衡的小配重，就像汽车轮胎上的车轮配重一样。

另一个值得注意的空调是 Zero Breeze，这是一个便携式设备，是 Kickstarter 活动的产品。功能包括(无论是奇怪的还是可以预见的，你来判断)蓝牙扬声器和 LED 手电筒。[Josh]超过一半的人怀疑该产品永远不会真正上市，但却感到惊喜。它不仅交付了，而且被证明是一个非常好的设计，只有几个轻微的抓头时刻。

在[乔希]的网站上的综述中，还有一些可以查看，他还将它与我们过去介绍过的[自己的 DIY 单元](https://hackaday.com/2019/04/07/diy-air-conditioner-built-from-weird-donor-appliance/)进行了比较和对比。
