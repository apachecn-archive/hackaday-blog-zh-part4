# LED 衬衫用薄纱做的

> 原文:[https://hack aday . com/2020/03/13/led-shirt-doe-it-with-tulle/](https://hackaday.com/2020/03/13/led-shirt-does-it-with-tulle/)

鉴于我们生活在大多数人现在称之为“*未来”的时代，我们真的应该开始行动起来。我们在电动车方面做得不错，但在服装方面，我们还有一些地方需要弥补。帮助这一努力的是【艾米·古德乔尔德】，[，他为所有场合设计了一件别致的 LED 衬衫。](https://www.instructables.com/id/Led-Shirt-With-Button-to-Change-Modes-Arduino-Neop/)*

衬衫的基础是运行 FastLED 库的 ESP8266，连接到 WS2812B LEDs 串。这是一个很好的组合，可以快速简单地制作丰富多彩的动画，而不需要太多的麻烦。然后，将 LED 灯条缝在衬衫上，并在灯条上添加热缩材料，让线有所附着。薄纱织物用作扩散器，当它们关闭时隐藏起来，并提供更令人愉快的发光效果。一切都是由一个小盒子控制的，装有一个拱廊按钮和 7 段显示器。

这是一个有趣的作品，对于新手来说很容易实现，也是学习 led 和缝纫的好方法。我们以前见过其他类似的建筑，[比如这个发光的 LED 裙](https://hackaday.com/2019/10/23/led-skirt-is-stealth-by-day-party-by-night/)。休息后的视频。
