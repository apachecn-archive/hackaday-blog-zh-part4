# DIY 8 位电脑知无不言

> 原文:[https://hackaday . com/2020/12/31/DIY-8 位计算机百事通/](https://hackaday.com/2020/12/31/diy-8-bit-computer-knows-all-the-tricks/)

一些项目是他们各自领域的成人礼。对于计算机科学来说，从零开始建造自己的计算机当然是这些项目之一。当然，我们不是在谈论在线购买组件并组装一台现代 x86 机器。我们的意思是从头开始建立一个更接近完全可编程的 8 位计算机的东西，就像这个来自 Federico 的基于 74LS 逻辑芯片的计算机。

这台计算机是从零开始设计和制造的，这足以令人印象深刻，但[费德里科]也在大约一个月内完成了这个项目。它可以通过 DIP 开关或通过 USB 连接到另一台计算机进行手动编程，还包括一个可调时钟，可以执行 1 Hz 至 32 kHz 范围内的任何步骤。这款 8 位计算机配备了 1024 字节内存、一个 ALU、四个七段 led 和一个 2×16 LCD 显示屏(在第二代计算机中)。

这不仅是一台由清楚了解逻辑芯片的人设计的功能强大的机器，而且[Federico]还在他的 GitHub 页面上提供了代码和原理图。即使没有自己制作，也值得一读，但是如果你想走这条路而不印刷巨大的 PCB，你总是可以走试验板路线。
