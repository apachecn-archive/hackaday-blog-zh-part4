# 侵入我的有线心脏

> 原文：<https://hackaday.com/2019/06/12/hack-my-wired-heart/>

班轮笔记？过时了。在柏林，特别版 synth-wave 唱片的发行伴随着一个名为 [Wired Heart](https://hackaday.io/project/165700-wired-heart) 的实验合成器。

这个可爱的心形合成器由音乐技术爱好者[tobi tubbutec]设计，其核心是经典的 74HCT14 芯片，带有六个施密特触发振荡器。亮红色 PCB 有八个金色触摸和湿度感应垫，用于激活和调节这些振荡器。除了通过压力和导电液体来改变声音之外，您还可以使用板上的六组插头来插入自己的元件，进行嘈杂的实验。Wired Heart 配备了 led、光敏电阻和电位计，但我们也将自己的 DIY 织物压力传感器插入到这个合成器中，以制作一些出色的电子声音。

在上面链接的 Hackaday.io 帖子中，[tobi tubbutec]向我们介绍了他在原型制作“cardio tronic human-touch hex oscillator stereo snythespacer”时做出的一些电路设计决策。我们喜欢他的创意和有时非常规的设计，从他出于美学原因加入非功能性痕迹到他选择的硬同步方法——将一个振荡器的小脉冲注入另一个振荡器。如果你想更详细地研究他的布局，[tobi tubbutec]在他的文章中包含了 KiCad 原理图文件。

这个可爱的、可破解的合成器在今年的 SuperBooth 上引起了我们的注意——这是一个在柏林举行的年度独立电子音乐会议，如果你喜欢奇怪的噪音和手工制作的电子产品，那么它非常值得一试——但它最近也在 Tindie 上列出了。要听《连线之心》最初附带的欢快的合成波唱片，[请访问艺术家 Hyboid 的乐队营地](https://hyboid.bandcamp.com/album/wired-at-heart)。

如果你对实验乐器和 synthy 芯片调感兴趣，你也会喜欢[【jarek 319】的 Sega Genesis 合成器](https://hackaday.com/2017/02/17/sega-genesis-chiptunes-player-uses-original-chips/)。
