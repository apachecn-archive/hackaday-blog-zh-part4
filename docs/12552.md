# 用铜线和高电压探测α粒子

> 原文：<https://hackaday.com/2022/01/22/detecting-alpha-particles-using-copper-wire-and-high-voltage/>

如果你想测量放射性，没有什么比盖革计数器更好的了:紧凑，坚固，使用相当方便，它们是迄今为止最常用的检测电离辐射的工具。然而，其他一些方法在过去已经被使用过了，虽然它们在今天可能不太实用，但是重新创建它们可以成为一个有趣的实验。

[Mirko Pavleski]使用容易获得的组件建造了一个这样的设备，称为[阿尔法辐射火花探测器](https://hackaday.io/project/183599)。发明于 1945 年的火花探测器包含一个强电场，电离辐射在其中触发放电。与盖革-米勒管不同，它使用普通空气，这使得它仅对阿尔法辐射敏感；β和γ射线在环境压力下不会引起足够的电离。幸运的是，阿尔法辐射是老式烟雾探测器中镅片发出的主要辐射类型，所以找到一个可用的辐射源应该不会太难。

这种装置的构造非常简单:几根细铜线悬挂在一个圆形金属罐上方，而廉价的高压电源在它们之间提供强大的电场。当阿尔法源被带到附近时，火花从电线飞向罐子；一个串联电阻限制电流，以确保电线不会过热和熔化。

尽管火花探测器作为一种测量装置并不实用，但它仍然可以用来进行简单的放射性实验。例如，[Mirko]在下面嵌入的视频中演示了阿尔法粒子被一张纸阻挡，因此不会对人类造成直接危险。然而，器件中存在的高电压确实存在，因此对检测器比对辐射源更要小心。

我们见过几个自制的盖革计数器，有些[用大量的胶带](https://hackaday.com/2019/08/21/a-cheap-and-cheerful-geiger-counter-build/)制成，有些[用老式的 555 定时器](https://hackaday.com/2016/10/16/a-cheap-555-based-geiger-counter/)制成。但是你也可以[使用光电二极管](https://hackaday.com/2021/08/06/a-trio-of-photodiodes-make-a-radiation-detector/)或者甚至[某种塑料](https://hackaday.com/2021/01/12/visualizing-ionizing-radiation-with-diy-plastic-scintillators/)来可视化电离辐射。
