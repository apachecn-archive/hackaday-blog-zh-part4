# 入门级 SLA 打印机获得升级，打印效果更好

> 原文：<https://hackaday.com/2019/02/18/entry-level-sla-printer-gets-upgrades-prints-better/>

熔融沉积成型(FDM)打印机拥有 3D 打印市场的最大份额，廉价、易用的打印机每年消耗数千公斤细丝。那么，3D 打印的挑战在哪里呢？还有修改的余地吗？[Physics Anonymous]认为是这样的，并已经开始研究对业余爱好者来说可能是增材制造领域的下一个大挑战:黑掉廉价的立体平版印刷(SLA)打印机。也就是说，[任意立方光子打印机](https://www.youtube.com/watch?v=o--30L7hHRc)的拆卸和改进。

售价低至 450 美元的光子在简单性方面有很大优势。没有必要担心细丝和挤压机的问题，因为印刷品是通过液态树脂的光聚合作用一次一层地形成的。由于只有一个移动部件，即在步进驱动的丝杠上从树脂罐中逐渐上升的构建平台，SLA 打印机不会受到三个独立轴的累积误差的影响。但是，Anycubic 在运动控制领域做出了一些设计妥协，以满足光子的价格点，为升级留下了一个完美的目标。[Physics Anonymous]在 OEM 立柱的每一侧添加了优质的线性轴承，并为构建平台加工了一个托架。结果是更好的垂直定位精度和降低的斜率。这是一个简单的修复，大大提高了打印质量，几乎看不见的层。

可悲的是，光子的 LCD 屏幕受到了一次不相关的重大损伤，但看起来[PA]将能够从中恢复。我们希望如此，因为我们发现 SLA 打印非常吸引人，并且想马上投入进去。但是也许我们应该先从小处着手。
