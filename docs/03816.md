# 用紫外激光消除字迹揭示了无数产品中的增白剂

> 原文:[https://hack aday . com/2019/08/06/用紫外激光消失书写/](https://hackaday.com/2019/08/06/disappearing-writing-with-a-uv-laser/)

许多初露头角的制造商已经试验了隐形墨水，并为此目的开发了各种各样具有可行特性的解决方案。然而，[本·克拉斯诺]在摆弄紫外激光时，偶然发现了一种完全不同的方法。

![](../Images/9c84efaf8cd7df4e219e4241a30d4beb.png)

The effect is subtle, but remains visible for several minutes.

所讨论的激光器是 MNL100 紫外激光器，被配置为产生纳秒级的 20 kW 脉冲，频率高达 24 Hz，工作波长为 337 nm，深紫外光。将激光通过光纤，对准一些普通的白纸后，可以观察到暗斑，几分钟后暗斑消失得无影无踪。

经调查，暗斑似乎是纸张中荧光增白剂的结果。它们似乎过载了，或者被激光改变了化学性质，然后随着时间的推移慢慢恢复正常。进一步的实验表明，过氧化氢能够立即去除痕迹，氩气环境会减缓痕迹褪色的速度。

这是对一种奇怪化学效应的有趣观察，得益于装备精良的光学实验室来分析正在发生的事情。根据这一现象，我们可以得到一些关于如何从普通洗衣液中提取荧光添加剂的提示。无论是纸张、塑料还是纺织品，如果你的眼睛看起来真的是亮白色，它可能含有二苯乙烯有机化合物作为荧光增白剂，这是一种你以前从未想过的隐藏特性。休息后的视频。
