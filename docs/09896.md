# 问 Hackaday:为什么要做模块化硬件？

> 原文：<https://hackaday.com/2021/04/27/ask-hackaday-why-make-modular-hardware/>

在电影中，一切都是模块化的。飞船坠毁的时候有一把大枪掉了下来？好在你可以拿起它，照原样发射(看着你，[银河护卫队 2](https://www.youtube.com/watch?v=23ctjvE95Vo) )。超光速引擎死亡？没问题，因为在《星球大战》宇宙中，你只需投入一个新的就可以上路了。

当然，现实世界中的事情并不那么简单。大多数系统，无论是宇宙飞船还是手机，都非常复杂，包含成百上千个相互连接的部件。如果我的三星手机的相机坏了，我就不能从我女朋友的 iPhone 上偷相机了。它们不能互换，因为系统设计不同。即使我们有相同的手机，摄像头也可以互换，它们也不容易互换。我们必须撬开电话，小心翼翼地进行转换。说到开关，任天堂 Switch 就是一个很好的反例。Joycon break？买个新的戴上。

如果更多的产品像任天堂 Switch 一样会怎样？它的模块化只是冰山一角吗？

## PocKit 大小的模块化

PocKit 项目通过全力投入模块化来解决这个问题。这个令人印象深刻的平台由一个中央“大脑”模块组成，该模块包含一个 ESP32 和一个 STM32，以及大约 24 个其他模块，这些模块可以直接连接到平台上。这些模块包括一个屏幕、一个键盘、一个扬声器、各种各样的传感器和其他一些小部件。大脑模块甚至有一个 DDR 连接器，让您可以将 Raspberry Pi 计算模块放入组合中，以获得更大的功能。

所有模块都使用相同的电气接口，并由磁铁固定，允许几乎无限的重新配置。他们有一个很好的系统运行演示，为便携式电池供电的 Linux 系统添加了摄像头、GPS 等:

你们中的一些人可能在这里感受到了熟悉的火花，并回想起几年前谷歌宣布已经取消的 Ara 项目。这是谷歌在 2013 年推出的一款实验性手机。这应该是一款低成本的裸机，用户可以通过交换和添加模块到框架上来轻松升级，包括电池、处理器、摄像头和显示器。不幸的是，Project Ara 从未进入我们的口袋，因为谷歌早在 2016 年就将其封存。

当然，这些并不是我们看到的模块化电子产品的唯一例子。我们中的一些人甚至和他们一起长大。

## 模块化教育

![](img/a25540b9991c75fcc661634056f8675d.png)

A true classic: the Snap Circuits kit

当我还是个孩子的时候，我最喜欢的玩具之一是一个电路套装。它由数百个发光二极管、马达、蜂鸣器和刚性“电线”组成，所有这些都卡在一个塑料底座上——有点像面包板。我花了几个小时以不同的方式连接这些模块，偶尔会遇到一个实际上做了一些事情的配置。Snap Circuits 套件的模块化特性是对电子学的完美介绍——毕竟，7 岁的我还没有准备好设计和制作 PCB。相反，带有丝网印刷示意图符号的大型彩色模块是一个很好的教学工具。

如今，你仍然可以买到全新的 Snap Circuits 套件(去年圣诞节，我非常兴奋地为一个朋友的孩子买了一个)，但正如你所料，市场上还有更先进的模块化套件。Brick 'R' Knowledge 就是这样一种产品。这是对经典 Snap 电路的现代改造，带来了先进的组件，包括 Arduino 和网络模块。

你可以通过查看这些工具包和其他工具来判断，模块化电子设备是一种非常棒的教育工具。它们可以用来介绍复杂的概念，但不会使材料变得“愚笨”,而只是以一种友好、平易近人的方式呈现出来。让我们回到消费电子产品，我们在日常生活中需要多少模块化？

## 轮到你了

你知道该怎么做。是时候在下面插话了，让我们知道你的想法。一切都应该模块化吗？据推测，制造具有如此高程度的客户可定制性的东西比仅仅制造一两种可能的配置要花费更多的成本。你会花更多钱购买个性化产品，还是乐于购买现成的产品？如果你像我一样，你会购买模块化选项，找到你喜欢的配置，然后永远不要改变它——那么，能够重新配置你的小发明有什么意义呢？
