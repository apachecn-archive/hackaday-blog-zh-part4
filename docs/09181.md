# 从零开始建造的帽枪打火机

> 原文：<https://hackaday.com/2021/02/06/cap-gun-lighter-built-from-scratch/>

对我们大多数人来说，打火机是一个便宜的 2 美元的塑料工具，有一定的用途，仅此而已。我们中的一些人可能会有一个漂亮的 Zippo，或者一个防风打火机，以获得更好的户外性能。但是如果你是一名机械师，你可以考虑给自己做点特别的，比如这个由[W&M .莱弗沙]制造的。

对于那些喜欢制作芯片的人来说，这里有很多值得喜欢的东西。主体由黄铜和铜制成，通过喷灯焊接在一起。这个打火机通过一种不寻常的机制工作。液体罐里塞满了棉花和较轻的液体，这些液体供给灯芯，灯芯本身相当普通。然而，点火是通过一个装有弹簧的铝锤，点燃纸帽，点燃灯芯。然后可以通过吹灭来熄灭火焰。

这是一个肯定会成为话题的打火机，尽管我们想知道它的破解报告在一个安静的，保留的雪茄吧会有多受欢迎。这种机制可能比一般的打火机有更多的消耗品，但这是真正独一无二的代价。还有其他有创意的设计，比如这个打火机使用铂催化剂点火。休息后的视频。
