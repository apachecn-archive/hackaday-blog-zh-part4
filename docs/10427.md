# 智能灯泡运行厄运是一个非常聪明的想法

> 原文:[https://hack aday . com/2021/06/15/a-smart-light-bulb-running-doom-is-a-pretty-bright-idea/](https://hackaday.com/2021/06/15/a-smart-light-bulb-running-doom-is-a-pretty-bright-idea/)

一个灯泡可能看起来不像是一个游戏平台，但我们现在生活在未来，所以一切皆有可能。有了足够的专业知识，事实证明一个 RGB 灯泡确实可以被改造成运行 Doom 。

[编辑注:项目页面和视频在本书付印时被删除。尼古拉收到了撤下通知。当我们做的时候我们会让你知道更多。主链接已经更新到 Wayback 机器。]

这并不是说，Ikea tr DFRI 灯泡是完成这项工作所需的唯一物品。但是灯泡，特别是[这种可寻址的 GU10 RGB LEB 灯泡](https://www.ikea.com/us/en/p/tradfri-led-bulb-gu10-400-lumen-wireless-dimmable-white-spectrum-80408608/)，捐赠了最关键的组件，一个硅实验室 MGM210L 无线微控制器，具有足够的处理能力来运行香草厄运。添加到微控制器上的是一个 TFT 显示器，一个由几个按钮和一个移位寄存器组成的控制器，以及一些将它们缝合在一起的零碎东西。然而，还需要更多的内存，所以[Nicola]使用了 8 MB 的 QSPI 闪存和一些巧妙的技巧来减少延迟和提高带宽。这个有很多巧妙的技巧，但最酷的事情可能只是整个建筑的占地面积并不比原来的灯泡大多少。看看下面视频中令人惊讶的流畅游戏。

这是对似乎永无止境的“它会毁灭吗？”系列。我们已经看到这款经典游戏被移植到各种东西上，从[GPS](https://hackaday.com/2019/09/01/running-doom-on-a-doomed-gps/)到[一台厨房“撞吧”电脑](https://hackaday.com/2021/04/17/kitchen-bump-bar-plays-doom-between-orders/)，甚至到[一台示波器](https://hackaday.com/2018/10/18/playing-doom-on-keysight-oscilloscope-via-windows-ce/)。
