# 3D 打印工具将台钳变成便捷的压弯机

> 原文：<https://hackaday.com/2020/01/06/3d-printed-tools-turn-bench-vise-into-expedient-press-brake/>

很有可能我们大多数人都使用过台钳来做远远超出其预期用途的事情。这是可以理解的，因为老虎钳可能是许多商店中最强大的手动工具，能够通过扭转你的手腕施加数吨的压力。不利用这种力量没有任何意义，不是吗？

尽管如此，老虎钳的夹紧力有时可能需要一点技巧，这就是这些 3D 打印压弯工具背后的想法。[Brauns CNC]想出了这些工具，它们由一个冲头和一个具有匹配轮廓的模具组成。通过磁性法兰安装在虎钳的钳口上，使用虎钳将冲头驱动到模具中，在金属中形成整齐的弯曲。[Braun]详细介绍了冲头几何形状和工件回弹管理，以及处理比虎钳钳口更宽的工件。这些工具是用标准的 PLA 或 PETG 打印的，非常坚固，尽管他确实提到使用[他的钢筋增强 3D 打印方法](https://hackaday.com/2019/04/22/steel-reinforced-3d-prints/)来制作鹅颈冲头和其他可能需要加固的工具。我们可以想象[碳纤维增强丝](https://hackaday.com/2016/09/07/3d-printering-xt-cf20-carbon-fiber-filament-review/)也会增加强度。

可以肯定的是，不管你用什么工具，台钳都不是真正折弯机的好替代品。这种机床能够以尽可能少的设置将金属板和其他材料加工成复杂的形状，并带来临时设置无法比拟的功率和精度水平。但是用自制的工具和额外的努力在更轻的材料上制作小弯曲的能力本身就是一个强大的工具。
