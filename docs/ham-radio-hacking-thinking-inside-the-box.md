# 业余无线电黑客:盒子里的思考

> 原文：<https://hackaday.com/2022/08/02/ham-radio-hacking-thinking-inside-the-box/>

有两种方法可以改进业余无线电接收机，或者任何类型的接收机。您可以过滤和修改无线电频率，包括收音机的中频，或者您可以改变音频输出。历史上，RF 和 IF 技术最受重视，因为尽早剔除无用噪声和信号可以让无线电的其余部分专注于实际感兴趣的信号。然而，音频滤波器更容易使用，直到最近，能够处理 RF 频率的 DSP 还很昂贵且不常见。然而，[water Stanton]向我们展示了如何制造有史以来最便宜的音频增强器。它只不过是一个经过改造的纸箱，你可以在下面的视频中看到和听到结果。

一方面，你不应该期待奇迹。另一方面，您可能有现成的盒子，可以在接下来的三分钟内尝试一下，为什么不试一试呢？使用盒子和不使用盒子时，你可以听到一点不同。

特别是，你也可以看出盒子位置的不同。他还鼓励你尝试不同的材料。他把这比作把你的手放在耳朵周围来引导声音流动。实际上，我们已经看到了与此相差无几的手机无源放大器。
