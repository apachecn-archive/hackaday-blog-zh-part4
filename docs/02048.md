# 制作自己的磷光材料

> 原文：<https://hackaday.com/2019/02/06/make-your-own-phosphorescent-material/>

荧光粉是整个显示和照明技术的关键。阴极射线管，真空荧光显示器，甚至一些白色发光二极管都使用荧光粉发光。[【氢时间】决定做一个绿色磷光材料，已经在 Youtube](https://www.youtube.com/watch?v=NGdVCIGHeQE) 上分享了过程，嵌在下面。

目的是生产掺杂铜杂质的硫化锌晶体。这产生了具有熟悉的绿色发光的磷光体。[氢气时间]首先指出，确保所有使用的化学品质量良好非常重要，因为即使是轻微的杂质也会破坏最终产品。

在加入高度稀释的硫酸铜溶液之前，将硫化锌制成水溶液，同时加入氯化铵作为助熔剂。搅拌混合物，然后在用氩气冲洗的管中加热。烧制后，用水洗涤磷光体并冷却。

最终产品在紫外光下呈现出鲜艳的绿色，表明该工艺是成功的。[氢时间]打算在未来的工作中使用家酿磷光体来生产显示器。这让我们想起了[Jeri Ellsworth]，在家里制作自己的 EL 电线。休息后的视频。
