# 回到未来，我们带着这个通量电容器 PCB 徽章

> 原文:[https://hack aday . com/2021/12/05/back-to-the-future-we-go-with the-flux-capacitor-PCB-badge/](https://hackaday.com/2021/12/05/back-to-the-future-we-go-with-this-flux-capacitor-pcb-badge/)

[Arnov]是《回到未来》系列的超级粉丝，所以他想要一些电影中的纪念品来装饰他的工作区。成功电影特许经营的官方纪念品可能相当昂贵，所以[Arnov]选择自己制作一些东西，创造了他自己的[通量电容器 PCB 徽章](https://www.hackster.io/Arnov_Sharma_makes/flux-capacitor-pcb-badge-2b77eb)。 [![Doc Flux Capacitor Schematic from Back to the Future](../Images/01e278162f1dda0dcb9f4d3d96ff6b7d.png)](https://hackaday.com/wp-content/uploads/2021/11/Doc-flux-capacitor-schematic.webp)

幸运的是，[Arnov 的]设计不像电影中 Doc 的设计那样复杂(见右图)，所以复制起来应该容易得多。我们有一个简单的 LED 电路，由 8205S MOSFET 驱动，由 ATtiny 微控制器控制。有一个小二极管用于在 USB 和电池电源之间自动切换，还有几个限流电阻用于 led。幸运的是，[Arnov 的]项目只需要 0.017 W 的功率，所以没有钚核反应堆是必要的，你可以很容易地用标准的硬币电池或 USB 供电。那真是一种解脱。

和许多[Arnov]的项目一样，其设计之美在于他在 PCB 布局上的细节。在这种情况下，布局比他的其他作品要简单一些，只需要将闪烁的 led 排列成“Y”形，以反映电影中看到的通量电容器。他还为丝网印刷添加了一些细节，以帮助完成美感。

我们认为这值得添加到您的 [PCB 徽章收藏](https://hackaday.com/2019/11/18/add-a-bit-of-pcb-badge-glamour-to-your-boring-id-badge/)中。
