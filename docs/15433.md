# 是 MacOS。在未经改装的 Wii 上！

> 原文:[https://hack aday . com/2022/11/24/its-MAC OS-on-an-unmodified-Wii/](https://hackaday.com/2022/11/24/its-macos-on-an-unmodified-wii/)

我们习惯了所谓的“Hackintoshes”，即运行 MacOS 的非苹果硬件。我们最近推出的一款甚至内置在了任天堂 Wii 的外壳中。但是[杜丹]比这做得更好，在未经修改的 Wii 上运行 MacOS，原始的任天堂硬件(法语，[谷歌翻译链接](https://www-journaldulapin-com.translate.goog/2022/11/21/wii-mac-linux/?_x_tr_sl=auto&_x_tr_tl=en&_x_tr_hl=en&_x_tr_pto=wapp))。

这个看似不可能完成的任务是如何实现的？经验丰富的 Mac 爱好者会记得苹果电脑使用 PowerPC 处理器的日子，Wii 使用的 PowerPC 芯片与 Mac G3 系列电脑中使用的芯片非常相似。由于 Wii 可以运行基于 Linux 的操作系统，因此它可以运行 [Mac-on-Linux](https://www.maconlinux.net/) ，理论上提供了一个可以运行 PowerPC 版本 MacOS 的环境。

安装过程困难重重，但最终他还是让 Wii 运行了最后一款经典的 MacOS 9。它甚至可以在这些有限的资源上运行 DOOM、Internet Explorer 5 和 iTunes，尽管最后一个软件包有显示和声音问题。然后他尝试了一个 MacOS X 版本，但是没有成功。

公平地说，这并不完全是一种获得廉价 Mac 的方式，更多的是一种推动控制台超越其原始功能的练习。但这仍然是一个有趣的转移，也许有人会及时让 MacOS X 版本也能在 Wii 上工作。如果你对激发这部作品的 Mac-in-a-Wii 感兴趣，你可以在这里看到它。
