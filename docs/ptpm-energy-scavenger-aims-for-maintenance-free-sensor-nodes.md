# PTPM 能源清道夫致力于免维护传感器节点

> 原文：<https://hackaday.com/2018/08/01/ptpm-energy-scavenger-aims-for-maintenance-free-sensor-nodes/>

[Mile]的 [PTPM 能量收集器](https://hackaday.io/project/159484)认真对待收集的想法，其设计不仅能收集太阳能，还能从温差、振动和磁感应中收集能量。这个想法是制造可以自我供电并且需要最少维护的无线传感器节点。这个想法不仅仅是简单地抛弃电池；如果设备坚固耐用，不需要维护，它们可以安装在不切实际或不方便的地方。[Mile]说目标是减少供应链中最昂贵的部分:人力。

该原型与太阳能和超级电容器一起工作良好，但[Mile]看到了收集其他来源的潜力，例如通过将这些单元安装到主动机械上来收集压电能量。凭借可选的输出电压、用于长期存储的可选电池以及带外壳的参考设计，PPTM 能量收集器旨在为无线传感器平台提供强大的电源解决方案。
