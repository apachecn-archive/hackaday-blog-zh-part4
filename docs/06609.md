# 来吧，宝贝，点燃我的火焰按钮

> 原文:[https://hack aday . com/2020/05/22/come-on-baby-light-my-fire-button/](https://hackaday.com/2020/05/22/come-on-baby-light-my-fire-button/)

虽然任天堂 GameCube 在它的日子里站在 PS2 的阴影下，但它的控制器今天仍然是各种修改的热门目标——由于半透明的底部和按钮选项，许多修改都涉及 led。作为超级粉碎兄弟系列的狂热玩家，[goomysmash]当然也是同一个控制器的所有者，这促使他编写了 [*GoomWave*](https://github.com/goomysmash/GoomWave) ，这是一个“多功能和可黑客攻击的 LED 库”。在一个令人印象深刻的详细说明中，[他展示了如何以两种不同的方式修改你自己的控制器](https://www.instructables.com/id/Goomwave-Reactive-GameCube-Controller-LEDs/)来为你自己利用这个库。

最初的灵感来自于 [Shinewave](https://hackaday.com/2015/08/02/shinewave-gamecube-controller-is-shiny/) mod，它以与 Smash Bros 中预定义动作相关的颜色点亮 RGB LEDs，【goomysmash】旨在从一开始就对其进行改进并添加更多的多功能性。它的最新版本是使用 ATtiny85 的简化 ABXY 按钮，以及使用 Arduino Nano 的全按钮。它们都直接由控制板供电，并有不同的模式，它们要么对控制器的交互作出反应，要么只是定制的灯。在休息后的视频中可以看到所有不同模式的简要展示，在[一个旧版本的视频](https://www.youtube.com/watch?v=jQGL3Kr40-o)中也有一些更多的细节，也嵌入在下面。

撇开令人着迷的 LED 闪烁不谈，我们只是不得不钦佩[goomysmash]在布线和安装控制器内的所有东西时表现出的勤奋和清洁。但是如果你不喜欢轻型模式，或者你正在寻找其他 GameCube 控制器的修改，[增加蓝牙](https://hackaday.com/2019/04/29/esp32-adds-bluetooth-to-gamecube-controllers/)怎么样？
