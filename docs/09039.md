# 黄铜制成的游戏男孩复制品

> 原文:[https://hack aday . com/2021/01/21/game-boy-replica-built-in-brass/](https://hackaday.com/2021/01/21/game-boy-replica-built-in-brass/)

任天堂的 Game Boy 是手持游戏革命中的传奇，无论是在海湾战争中被儿童或士兵监护，它都几乎坚不可摧。然而，[Jiri]决定试着用黄铜而不是塑料制作一个他自己的贡品。

硬件基于用于 ESP32 的 Odroid GO 仿真器固件，在 2.2 英寸彩色 TFT 屏幕上运行。这是定制构建的一个很好的基础，可以避免损坏任何珍贵的经典硬件。然后，它被组装在黄铜制成的前板后面，精致的点对点黄铜线赋予它艺术的电路雕塑外观。黄铜有时确实很难使用，充当散热器，妨碍了支座的焊接。为了解决这个问题，[Jiri]使用一个电热板从下面加热盘子，保持足够的温度，这样一个手动熨斗就可以完成这项工作。

最终的结果是一个有趣的 Game Boy 模拟器，装在一个时尚的盒子里——尽管你不应该把它放在背包里，以免它短路暴露的导体。对于任何一个终身的任天堂粉丝来说，这都是一份很棒的礼物。[Jiri]对电路雕塑并不陌生，正如我们所知-[我们在](https://hackaday.com/2020/06/24/3d-printed-tools-make-circuit-sculpture-a-little-easier/)之前已经介绍过他的工具和方法。休息后的视频。
