# 升级宝马 I3 的电池

> 原文：<https://hackaday.com/2021/02/01/upgrading-the-batteries-in-a-bmw-i3/>

宝马 i3 于 2013 年底上市，是该品牌的首批电动汽车之一。早期型号还提供可选的增程发动机，配备 60 Ah 电池，充满电后可行驶 130 公里。然而，时过境迁，多年来，宝马在这些年里用更大容量的电池更新了车型。那么，用更新、更胖的榨汁机 120 Ah 包来改造一个老型号需要什么呢？

[丹尼尔](https://www.youtube.com/watch?v=EH02scyrCaE&ab_channel=Daniel%C3%96ster)在一段视频中很有帮助地展示了这一切，他指出这不是一份适合心脏虚弱或装备不良的人的工作。好消息是，在机械方面，新电池与旧电池具有相同的外部尺寸，这意味着它们可以用螺栓固定，而不需要任何切割、焊接或适配器。但好消息就这些了。电池由空调系统冷却，这意味着移除和更换意味着使用非常昂贵的专用硬件来排空系统中的制冷剂。此外，由于安全原因，当安全气囊被触发时，许多碰撞汽车中的电池被禁用，需要通过宝马的专有软件解锁或更换内部电池控制器。然后是通常的渐变清单，发生在任何汽车生产线上，这意味着某些车型年和装饰包可能有不兼容的插头和连接器或其他功能。

总的来说，这可能是相当多的工作要做，并且需要工具，需要专门的技工车间的服务。然而，找一家经常与电动汽车合作的有经验的商店，你可能会发现他们可以方便地为你升级，而不必太大惊小怪。我们以前见过[丹尼尔]解决升级问题，比如在日产 Leaf 上进行更容易的更换。休息后的视频。
