# 陌生事物留言板通过拼写来消磨时间

> 原文:[https://hackaday . com/2021/09/05/stranger-things-message-board-passes-the-time-by-spelling-it-out/](https://hackaday.com/2021/09/05/stranger-things-message-board-passes-the-time-by-spelling-it-out/)

网飞的怀旧热门剧《陌生人的事情》会很快回归第四季吗？我们可以拿出一块显灵板去问神灵，但是我们更愿意问离源头更近的地方，比如一个颠倒的灵魂。你知道最好的方法是用发光二极管——一个代表字母表中的一个字母，这样灵魂就可以拼出它们的信息。

[![Arduino, ESP01, and real-time clock powering this Stranger Things word clock](../Images/1a7ffe559e304686bab0ca7b117cdf30.png)](https://hackaday.com/wp-content/uploads/2021/09/stranger-things-clock-wiring.jpg) 虽然接触狄摩高根的世界不太可能与[【danjovic】的开源*陌生人的东西*板](https://hackaday.io/project/167567-stranger-board)联系，但你保证会得到为你拼出的每一分钟的时间，就像“现在是四点二十五分(或六点)”。如果你想吓吓不知情的朋友，你可以偷偷用手机给它发信息。

现在有两个版本——最初的桌面版本，和一个挂在墙上并使用高质量照片打印作为背景的版本。两者都使用 ESP-01 和 Arduino 来帮助驱动 26 个 RGB LEDs，并使用 DS2321 实时时钟来计时。我们喜欢壁挂版本的漆包线工作，但最酷的部分必须是英语和巴西葡萄牙语的双语支持。休息之后你可以看看两者的演示。

我们在这里见过很多单词钟，但这可能是少数几个滴着流行文化的之一。如果你想要的是令人惊叹的现代主义，那就来看看这种精心打造的美吧。
