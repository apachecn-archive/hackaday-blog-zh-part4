# 以一流的质量复制老式塑料零件

> 原文：<https://hackaday.com/2019/04/08/reproducing-vintage-plastic-parts-in-top-notch-quality/>

塑料是一种非常有用的材料，但随着年龄的增长，它也可能成为一种痛苦。世界各地的老式设备所有者正在遭受痛苦，因为旋钮断裂，挡板破裂，零件扭曲，变色和故障。通常，策略是从其他损坏的硬件中抢夺好的部件，并祈祷供应不会枯竭。[Eric Strebel]向我们展示了这并不是唯一的解决方案—[你可以用合适的工具自己复制老式塑料部件。](https://www.youtube.com/watch?v=K3ZaPlV10z8&feature=youtu.be)

在唱片业，没有什么可以替代老式设备，所以一个家庭手工业已经围绕着保持旧硬件运转而形成。[Eric]的任务是为经典的 Neve 音频控制台复制 VU 仪表挡板，因为自 20 世纪 70 年代以来就没有生产过替代零件。

第一步是获得高质量的复制母盘。移除原始表圈，并抛光以去除 40 多年磨损造成的划痕和瑕疵。然后在胶合板箱中制作一个硅胶模具。激光切割零件用于快速轻松地创建底座、流道和通风口。然后用树脂填充模具以生产最终零件。[Eric]演示了整个过程，使用透明硅胶和染色树脂使观众更容易看到。

不幸的是，由于使用了硅树脂和硬化剂，最初的结果很差。这些部件在尺寸上是可用的，但是具有模糊的表面光洁度，给出非常差的光学质量。这通过返回到已知良好的硅酮化合物得以纠正，该化合物能够第一次产生完全透明的部件。令人印象深刻的是，唯一需要完成的是折断流道和通风口。然后，该零件就可以安装了。作为最后一项表演，【Eric】然后将零件装在一个定制的激光雕刻硬纸板箱中。正如他们所说，展示就是一切。

有了现代设备，复制像[旋钮](https://hackaday.com/2019/01/20/cloning-knobs-for-vintage-testing-equipment/)和[徽章](https://hackaday.com/2019/03/28/casting-car-emblems-via-3d-printing/)这样的古董零件比以往任何时候都容易。休息后的视频。
