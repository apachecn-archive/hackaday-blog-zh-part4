# 磁性角度传感器模块使编码器更适合爆破

> 原文：<https://hackaday.com/2021/03/05/magnetic-angle-sensor-mods-make-encoder-better-for-blasting/>

我们看到的围绕这些部件的大部分黑客行为都与获取现有组件并以有趣的新方式将它们拼凑在一起有关。我们很少看到现有组件被破坏和重新利用，但当它发生时，就像[这个重新想象的旋转编码器](https://hackaday.io/project/178005-magnetic-rotary-encoder)，它肯定会抓住我们的注意力。

你可能还记得克里斯·G 最近的[激光小行星*T2*](https://hackaday.com/2021/02/18/laser-galvos-and-an-esp32-recreate-old-school-asteroids/)游戏。如果没有，你真的应该去看看——它的建造非常漂亮。制造过程中的一个小问题是控制，他使用的现成旋转编码器的分辨率远远不够。[Chris]没有选择商品替换零件，而是从原始编码器的机械零件(如轴和面板衬套，以及 AS5048A 传感器板)中推出了自己的零件。磁性角度传感器的分辨率为 14 位，在原始轴的底部粘有一个小型钕环磁体，改进后的编码器比原始的接触式编码器分辨率高得多。

传感器分线板的尺寸正好适合这项工作；所有[克里斯]需要做的就是 3D 打印一个小适配器来把这两个部件组装在一起。我们不得不承认，当我们第一次在 Hackaday.io 上看到这一点时，我们没有看到黑客是什么——修改后的部分看起来非常像一个普通的编码器。下面的视频显示了设计和建设过程中的一些精确的岩石爆破。
