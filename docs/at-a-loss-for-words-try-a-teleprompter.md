# 不知所措？试试提词器

> 原文：<https://hackaday.com/2022/11/19/at-a-loss-for-words-try-a-teleprompter/>

如今每个人都在做视频，你可能想用提词器来升级你的旁白游戏。【现代爱好者】[可以帮忙](https://www.modhobbyist.com/projects/24)。因为他制作视频——就像下面那个关于提词器的视频——我们假设他是出于自己对这个设备的需要而制作的。实际上，这是他的第二台提词机。第一个更大，不是电池供电，所以这个新版本提供了更多的便携性。摄像机通过提词器屏幕拍摄，这样你就可以直视摄像机，仍然保持在脚本上。

该项目重用了一些原始的提词器代码，通过 Raspberry Pi 显示一个文本文件。还有一个控制键盘，可以让你远程控制滚动速度。这个项目的真正关键是 3D 打印的房屋。嗯，还有反射玻璃屏幕。考虑到这一点，您可以用多种方式进行实际的文本显示。

显然，相机的重量在一定程度上限制了这款相机的便携性。当然，你可以用更轻的东西或者增加一些相反的重量来至少平衡一下。3D 打印文件在 [Thingiverse](https://www.thingiverse.com/thing:5436768) 上，其余的在 [GitHub](https://github.com/modern-hobbyist/modern-vlogging-teleprompter) 上，所以如果你想的话，你可以很容易地做出改变。

你可能会认为我们会看到更多的提词器项目，我们[确实看到了一些](https://hackaday.com/2020/10/26/a-teleprompter-for-the-rest-of-us/)。我们还见过一个黑客让你在视频会议上通过你的[笔记本电脑屏幕看东西。](https://hackaday.com/2020/05/29/two-way-mirror-improves-video-conferencing/)
