# 盒式合成器播放速度控制

> 原文：<https://hackaday.com/2020/12/07/cassette-synth-plays-with-speed-control/>

与黑胶唱片相比，磁带听起来可能没那么好，但在制作自己的音乐时，卡带播放器可以带来无穷的乐趣。例如，参见 Mellotron，或这种相对容易的 DIY 替代品，[【Rich Bernett】的 Cassettone 卡带播放器 synth](https://www.youtube.com/watch?v=o1GdNkX1ewU) 。

卡塞通的工作原理是用几个电位计代替控制磁带放音机电机速度的微调电位器。这些都是由位于木键下方的瞬时按钮激活的。在休息后的视频中，[Rich]给出了一个完整而详细的指南来构建您自己的。还有一份打磨过的谷歌文档，包括制作橱柜的示意图和样板。

说起来，表壳设计不好看吗？它由工艺胶合板制成，但涂有清漆和老式电子杂志上的零碎物品。这看起来真的是一个有趣的小乐器。

你宁愿用 MIDI 键盘控制你的磁带合成器吗？[只需添加 Arduino](https://hackaday.com/2020/10/01/reel-in-the-years-with-a-cassette-player-synth/) 。
