# 亚克力灯光拼图有重点

> 原文：<https://hackaday.com/2022/11/30/acrylic-light-puzzle-has-a-point/>

[JBV 创意]最近自豪地拥有了一台激光切割机，像我们大多数人一样，开始思考他们能用它做什么。答案很简单，一件聪明的小艺术品或由堆叠的丙烯酸树脂制成的拼图。

他创造了一些文字，并从一个点挤出来，但不是每个部分都与每个板块相交，给每个板块一个难以辨认的外观。这使得一个小光源(比如你手机背面的 LED)可以在墙上投下阴影。通过一些 3D 打印的支架和垫片，它被安装在一块漂亮的樱桃胶合板上。总的来说，这项技术非常简单，容易理解。[JBV 创意]没有包括过程的更多细节，这是一个遗憾，因为它看起来像是为一些谜题重新创建的美丽效果。

如果你在寻找带光源的雕刻亚克力材质，这些发光杯垫非常棒。或者是[这个拼图](https://hackaday.com/2021/07/01/jigsaw-puzzle-lights-up-with-each-piece/)，当棋子摆好的时候会亮起来。
