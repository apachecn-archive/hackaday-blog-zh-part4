# 开源生物学和生物黑客聊天

> 原文:[https://hack aday . com/2019/02/11/开源-生物学和生物黑客-hack-chat/](https://hackaday.com/2019/02/11/open-source-biology-and-biohacking-hack-chat/)

加入我们太平洋时间周三中午的[开源生物学和生物黑客聊天](https://hackaday.io/event/163454-open-source-biology-and-biohacking-hack-chat)！

贾斯汀·阿特金的名字可能听起来没什么印象，但你可能已经看过他的 YouTube 热门频道[The think Emporium](https://www.youtube.com/c/thethoughtemporium)，致力于定期发布开源科学。贾斯汀的兴趣广泛，从[天上的](https://hackaday.com/2017/03/23/see-satellites-with-a-simple-radio-telescope/)到[微观世界](https://hackaday.com/2018/12/14/a-lecture-by-a-fun-guy/)。

他目前的兴趣是对酵母进行基因改造以生产蜘蛛丝，甚至可能用酵母来酿造啤酒。他和 the Thought Emporium 团队一直忙于建立一个完整的 DIY 生物实验室来支持这项工作，并一直在进行各种测试实验。

请[加入我们的黑客聊天](https://hackaday.io/event/163454-open-source-biology-and-biohacking-hack-chat)，我们将讨论:

*   酵母基因工程的方法和原因；
*   从零开始建立一个有效的生物学实验室需要什么；
*   蜘蛛丝酵母项目现状的最新情况；和
*   开源生物学领域在哪里，它将走向何方。

当然，我们鼓励你在讨论中提出自己的问题。你可以在[开源生物学和生物黑客 Hack Chat 活动页面](https://hackaday.io/event/163454-open-source-biology-and-biohacking-hack-chat)上发表评论，我们会把它放在 Hack Chat 讨论的队列中。

[![join-hack-chat](../Images/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是在 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)上的现场社区活动。本周我们将在太平洋时间 2 月 13 日星期三中午坐下来。如果时区让你心烦意乱，[我们有一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20190213T12&p0=224&msg=Open+Source+Biology+and+Biohacking+Hack+Chat&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
