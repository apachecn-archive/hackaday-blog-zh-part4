# Arduino 用…水灭火？

> 原文:[https://hack aday . com/2019/01/08/arduino-fights-fire-with-water/](https://hackaday.com/2019/01/08/arduino-fights-fire-with-water/)

我们不认为我们会把我们的消防安全托付给一个携带几盎司水的机器人，但作为一个演示或科学项目，[Tinker Guru]的[消防机器人](https://create.arduino.cc/projecthub/130797/arduino-based-fire-fighting-robot-using-falme-sensor-4e0556?ref=user&ref_id=564578&offset=0)是对这个问题的一个有趣的回答:“我该如何处理我从中国购买的 Arduino 传感器的大盒子里的火焰传感器？”你可以在下面看到这个设备的视频。

你可以看到，这是一个非常标准的两轮机器人，驱动轮在后面，滑板在前面。还有一个火焰传感器和一个水泵。你大概能猜到，这个设备注意到了一个火焰，冲过去向它喷水。

不过，这让我们开始思考。建造一个真正的机器人消防员需要什么？事实证明，你不用费力就能发现已经有好几个这样的人了。铝热剂机器人似乎有很大的牵引力——在市场上，也就是说，尽管它的超大胎面可能也给了它很好的牵引力。大多数机器人都不自带水，甚至还有一个看起来像人类的机器人——[托尔](http://seb199.me.vt.edu/trec/thor/)。不管怎样，馅饼看起来像蛋糕一样。

有趣的是，似乎没有人携带任何种类的化学灭火器。当然，我们已经[看到过水是最好的情况，](https://hackaday.com/2018/12/06/fail-of-the-week-1975-the-browns-ferry-nuclear-incident/)无论如何。如果你想要一个稍微实用一点的房子——但只是一点点——看看能装一升水的[【伊万的】机器人](https://hackaday.com/2018/12/28/can-this-fire-fighting-robot-take-the-heat/)。
