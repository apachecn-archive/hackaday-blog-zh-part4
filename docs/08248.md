# 安全倾倒 N64 显影盒

> 原文:[https://hack aday . com/2020/11/02/dumping-a-n64-development-cartridge-safely/](https://hackaday.com/2020/11/02/dumping-a-n64-development-cartridge-safely/)

复古游戏爱好者一直对常见商业游戏之外的稀有游戏有着浓厚的兴趣。无论是早期测试版、评论版，还是从未发布的接近完整版本的游戏，这些发现都非常有价值。[Modern Vintage Gamer]最近发现了任天堂 64 版的 Turok 3 预发布版本[，并着手丢弃和保存这个发现](https://www.youtube.com/watch?v=T2Mk-d6QLio)。(视频，嵌在下面。)

对于像这样的一次性磁带盒，务必要非常小心，以便保存机载数据。简单地将它放入普通的控制台可能会启动游戏，但随之而来的是损坏购物车的非零几率。相反，采取的第一步是出于存档的目的而丢弃购物车。当使用原型手推车时，像 Retrode 这样的商品翻斗车不足以完成这项工作。[Modern Vintage Gamer]注意到带有并行端口的 Doctor V64 或 Gameshark 也可以工作，但选择使用更现代的解决方案，即 Ultrasave 和 64drive。

通过将盒式磁带备份并复制到 64 硬盘上，代码可以在真正的控制台上运行，而不会有损坏原始代码的风险。乍一看，这款游戏看起来和最终的零售版本很相似。使用文件比较工具对转储进行的分析表明,“80%完成”ROM 和零售版之间的唯一区别是标题，这使得[现代老式游戏玩家]猜测游戏可能是匆忙发布的。

虽然在这种情况下，转储并没有净一个令人惊讶的复古游戏的罕见版本，[现代复古玩家]做了一个伟大的工作，解释如何和为什么保存一个复古弹夹的过程。我们期待下一个撼动复古世界的罕见点滴；我们已经看到 Capcom arcade boards 的努力取得了巨大的成果。休息后的视频。
