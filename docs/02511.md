# 你现在能做的多通道野外记录仪

> 原文：<https://hackaday.com/2019/03/22/the-multichannel-field-recorder-you-can-build-right-now/>

野外录音机，或带有几个 XLR 插孔和一个 SD 卡插槽的背包式录音机，是一种利基设备，无论你选择什么样的商业野外录音机，你总是会在你想要的功能和你将得到的功能之间做出妥协。[Ben Biles]不想妥协，所以他建造了自己的多通道音频 DSP 现场记录器。它有一个四通道平衡主输出，两个立体声耳机输出，八个或更多的输入，数字 I/O 和足够的多轨道录音路由。

在机械方面，该系统的设计是一个 3D 打印的盒子，每一面都布满了各种连接器和贴片点。这就是当你需要大量 I/O 时得到的，是的，这些是面板安装连接器，所以准备好支付连接器的价格。模拟前端是一块 perfboard 上的背板，包含一个 8 通道差分 I/O。

当然，任何录音机用起来都很糟糕，除非有一个很棒的用户界面，为此，没有比手机上的高分辨率触摸屏更好的了。这导致[Ben]使用蓝牙连接到一个显示增益、电平、幻像电源开关和线路或麦克风复选框的应用程序。如果这还不够，还有一些用于音量的 MIDI 旋钮，因为 MIDI 对于用户输入来说仍然很棒。这是你在便携式录音设备中想要的一切，是的，[有一个 soundcloud 演示](https://soundcloud.com/benbiles/bbbox-proto1-recording-48khz16bitwav)。你也可以看看下面的演示视频。
