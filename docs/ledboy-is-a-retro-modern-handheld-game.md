# LEDBOY 是一款复古现代的掌上游戏

> 原文：<https://hackaday.com/2021/11/11/ledboy-is-a-retro-modern-handheld-game/>

早在 20 世纪 70 年代，市场上有一些基于 LED 的游戏，很快被液晶显示器和其他更先进的技术取代。然而，[grosofabian]想要用更现代的硬件重现经典风格的游戏。其结果是 LEDBOY，一款向那个时代致敬的彩色掌上游戏。

该手持设备基于 ATtiny 1614 微控制器，驱动 10×10 的 NeoPixel Nano 2427 LEDs 阵列，因其 2.4 mm x 2.7 mm 的小尺寸而得名。它们也是 RGB 的，所以有很多美妙的颜色可以玩。

包裹在一个整洁的外壳中，有一个可充电的 130 mAh 锂离子电池和一些简单的触觉按钮，这是一个整洁的小型手持游戏机。添加 CH340C 芯片用于 USB 串行任务，使用 Arduino IDE 编程也很容易。

Github 上的代码可供那些热衷于近距离观察的人使用。有趣的是，该项目与我们在 12 年前报道的一个同名建筑有着惊人的相似之处。时间是一个扁平的圆圈，视频，我的朋友们，在休息之后。
