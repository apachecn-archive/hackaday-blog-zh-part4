# 超级电容再战电池

> 原文：<https://hackaday.com/2021/12/01/supercapacitors-vs-batteries-again/>

超级电容肯定和电池不一样，我们都知道。它们倾向于具有非常低的工作电压，并且由于它们在平行板上存储电荷的工作原理，它们的放电曲线对于现代微控制器设备来说非常不友好。与现代锂聚合物(LiPo)电池相比，每单位体积的能量存储效率也较低，因此总的来说，它们对我们的许多项目来说看起来都不太有用。然而，正如[Andreas Spiess '][最新视频展示的](https://www.youtube.com/watch?v=Orf9segRLv0)，它们确实有一些可取的功能，可能使它们对某些嵌入式应用有用。

低工作电压最初看起来像是在典型的 3.3V 下工作的器件的问题，很容易将几个串联起来并随之滚动。但正如[Andreas]以其典型的清晰方式解释的那样，需要一个复杂的功率级，在电容电压高于所需水平时以降压模式工作，当电容电压低于所需水平时以升压模式工作。太复杂——简单地坚持使用并联超级电容的低压组，并且总是在升压模式下工作要容易得多。即使这样做，你实际上也不可能用一个总是活动的设备获得超过几个小时的工作电压。

那么，为什么要用超级瓶盖呢，使用吸脂器肯定更容易、更好。在许多情况下，答案肯定是肯定的。但是 LiPo 电池不能在冷冻温度下充电(除了某些特殊的低温产品)，否则电池会由于锂金属在阳极的沉积而迅速损坏。此外，你需要小心充电，尤其是当它们大量放电时，因为如果没有适当的处理，它们很容易损坏。LiPo 电池基于化学原理运行——锂离子不得不在结构内部移动，最终电池会耗尽。

超级电容器的优点是非常长的寿命(但有时，他们确实会泄漏)*更积极的充电和放电行为，并将在非常低的温度下工作。这使得它们在偶尔有大量电力可用时(超级快速充电循环)或在温度持续保持非常低的地方非常有用，例如在山上，太阳能可以工作，尽管速度很慢，但脂肪电池肯定不适合。*

其他电池化学物质也是可用的，比如可以耐寒的磷酸铁锂。此外，您也可以使用集成加热器来隔离电池，并将电池预热到安全的充电温度。因此，就像所有电子产品一样，为您的应用选择正确的器件非常重要，这一切都从电源开始。超级电容器可能会为你想到的特殊应用找到一个合适的性价比点。

超级电容器并不真正适合许多应用，比如给一辆电子自行车供电，或者给你的笔记本电脑供电，但不管怎样，他们还是做到了。
