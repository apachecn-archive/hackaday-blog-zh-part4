# 电子木偶控制器跳过操纵杆或键盘

> 原文:[https://hack aday . com/2021/09/04/animatronic-puppet-controller-skips-joystick-or-keyboard/](https://hackaday.com/2021/09/04/animatronic-puppetry-controller-skips-joystick-or-keyboard/)

电子动画的主要挑战之一是创造自然的运动。你可以为每一个可能的自由度建立一个驱动器，但如果你不能控制它平稳地发挥作用，它仍然会令人失望。[伏特先生]对电子动画项目产生了热情，但发现编程很乏味，用键盘或控制器进行手动控制很难做好。作为替代方案，他正在制造电子木偶控制器 Waldo。

沃尔多钻机正在与《传送门 2》游戏中健谈的球形机器人[惠特利](https://www.youtube.com/watch?v=mOeJAuE33gg)一起开发。木偶操纵装置由一系列[伏特先生]手上的环组成，每个环的位置由角度传感器读取。这使他能够控制惠特利的身体和眼球，眼睑和手柄的方向。惠特利和瓦尔多仍然需要一些改进，但我们期待看到完成的项目在行动中。

[门户](https://hackaday.com/2017/01/12/portal-ported-to-the-apple-ii/)游戏激发了几个特色项目，包括 [GLaDOS](https://hackaday.com/2013/03/25/installing-glados-in-the-ceiling-of-your-house/) 、[炮塔](https://hackaday.com/2019/08/12/locating-targets-with-charm-courtesy-of-a-life-size-portal-turret/)，当然还有更多[惠特利](https://hackaday.com/2015/11/30/robotic-wheatley-from-portal-2/)建造。
