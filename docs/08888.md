# 派对独木舟点亮水面

> 原文:[https://hack aday . com/2021/01/07/party-canoe-lights-up-the-water/](https://hackaday.com/2021/01/07/party-canoe-lights-up-the-water/)

一般来说，夜间进行的任何水上活动都更加危险。危险不太明显，很容易陷入困境。[Xyla Foxlin]的派对独木舟帮不上忙，[但天黑后它看起来确实是*生意*。](https://www.youtube.com/watch?v=Qg-Exm8oRP8&feature=youtu.be)

独木舟由玻璃纤维制成，直接在现有的独木舟上形成，使形状变得容易。它由两部分组成，特别注意使最终结果尽可能清晰。显然，玻璃纤维永远不会是完全透明的，但[Xyla]在获得漂亮的半透明磨砂外观方面做得很好。最终效果意味着它是装满可寻址 led 的完美独木舟。挂在 Arduino 上的一串 WS2812Bs，在夜间划船时构成一个吸引人的灯光秀。

玻璃纤维的扩散性质在这里真的很重要。我们之前已经讨论过这个话题——[这是让你的项目真正流行起来的关键。](https://hackaday.com/2017/04/25/ask-hackaday-what-about-the-diffusers/)休息后的视频。
