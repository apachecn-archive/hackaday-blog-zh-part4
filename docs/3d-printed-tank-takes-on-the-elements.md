# 3D 打印坦克呈现元素

> 原文：<https://hackaday.com/2021/05/16/3d-printed-tank-takes-on-the-elements/>

市售的无线电控制坦克很有趣，但有时你必须自己建造。[让我们打印]就是这么做的，在雪地里拿出来之前，在他的 3D 打印机上打了一个坦克。

坦克是一个相当简单的建筑，依靠一对有刷电机推进，由连接到标准无线电控制硬件的双速度控制器控制。然而，其他一切都是定制的，从 3D 打印的变速箱，到底盘和颇具侵略性的赛道。当坦克在泥泞中行驶时，后者的尖齿会留下很深的凹痕，尽管这不足以阻止小坦克在深雪中向高处移动。

然而，这种构建并不适合没有耐心的人。[让我们打印]注意到，光是轨道就花了 80 多个小时在 PETG 运行，更不用说其余的框架和变速箱。然而，我们确信这是一次很棒的学习经历，在户外驾驶也很有趣。现在，下一步肯定是走得更大。休息后的视频。
