# 泡沫飞艇是廉价的遥控玩具

> 原文：<https://hackaday.com/2018/11/15/foam-airboat-is-cheap-rc-fun/>

阿彻迷和路易斯安那州的居民已经熟悉了飞艇的概念。把一个驱动飞机螺旋桨的强大引擎放在平底船体上，你就有了一个穿越美国南部沼泽的绝佳方法。虽然一艘完全成熟的汽艇可能会花掉你 10 万美元中的大部分，但这种简单的无线电控制版本以低得多的价格提供了巨大的乐趣。

船体是建立在一张泡沫板上的，这种泡沫板便宜，容易获得，并且具有合适的浮力。然后它配备了一个无刷电机来运行道具和一个伺服系统来控制方向舵。带上无线电接收器和速度控制器，你就可以走了。

这个建筑可以在几个小时内完成，而且由于其机械结构简单，非常适合孩子们玩。还有发挥额外创造力的空间——如果你觉得饿了，你可以用西瓜代替。休息后的视频。
