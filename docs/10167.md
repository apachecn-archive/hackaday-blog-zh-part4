# 换刀打印机拥有独一无二的喷嘴手帕

> 原文:[https://hack aday . com/2021/05/18/tool changing-printers-get-a-nozzle-hanky-like-no-other/](https://hackaday.com/2021/05/18/toolchanging-printers-get-a-nozzle-hanky-like-no-other/)

当谈到更换工具的 3D 打印机时，闲置的喷嘴往往会流口水。不过，在投入使用之前，清理喷嘴上的粘性物质是至关重要的。由于每次打印切换喷嘴可能会发生数百次*，因此拥有坚如磐石的清洁溶液是制作干净整洁零件的关键。[Kevin Mardirossian]对现有的清洁解决方案并不太满意，所以他开发了[*Pebble Wiper*](https://youtu.be/EdZrv_zDask)，这是一款值得生产的喷嘴芯吸部件，可以完美地吸走喷嘴数千次。*

受[big brain 3d]的[可伸缩吹扫装置](https://www.bigbrain3d.com/product/retractable-purge-mechanism/)的启发，[Kevin]首先将吹扫工具放到黄铜布拉德上。它不是将细丝挤压到自由空间中，而是聚集成一个小水滴状的“卵石”,迅速冷却成可控的形状。从这里开始，用伺服臂轻轻一弹，用硅胶涂抹刷轻轻一擦，喷嘴就可以使用了。这种设置听起来可能很简单，但它是成千上万次测试的结果，目的是不让残余的软泥附着在实际打印的零件上。这是*在*【凯文】花时间从头开始——建造他自己的工具更换 3D 打印机来测试它。最后，他好心地[将文件放在 Github](https://github.com/KevinMar1/Pebble_Wiper/) 上，供其他黑客修补和恶作剧。

那么它的效果如何呢？从他分享的结果来看，我们想得很壮观。自从采用了它，他完全放弃了床上的任何牺牲打印人工制品，并能够始终如一地完美地完成令人惊叹的多材料打印，没有残留喷嘴流口水的迹象。虽然换刀系统已经成为黑客攻击和探索的伟大平台，但[Kevin 的] Pebble Wiper 使这些机器更接近达到“生产水平”的可靠性，从而最大限度地减少浪费。谁知道呢？也许所有这些鹅卵石都可以被磨碎，[再制成细丝](https://hackaday.com/2020/07/17/make-your-own-filament/)，[再捣碎](https://hackaday.com/2014/01/30/the-filawinder/)成可用的细丝？
