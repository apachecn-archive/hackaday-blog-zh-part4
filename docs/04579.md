# 赛马游戏集乐趣、技巧和竞争于一身

> 原文：<https://hackaday.com/2019/10/26/horse-racing-game-hits-trifecta-of-fun-skill-and-competition/>

在霓虹闪烁的拉斯韦加斯沙漠，如果你知道去哪里找，你可以找到一个古老的，20 世纪 80 年代的机电赛马游戏，叫做适马德比。在这个团体游戏中，你和几个喝醉的陌生人围坐在一个台球桌大小的机器旁，以每匹 25 美分的价格在小马上下注。没有什么技巧可言，全凭运气。这不是游戏。

[Alex Kov]的[机电赛马游戏](https://www.instructables.com/id/Electromechanical-Horse-Racing/)与适马德比相比是一只独角兽，或者至少是一只斑马。这个游戏需要耐心、技巧和狡猾。不像适马德比，你可以在家里用旧垃圾箱摇几下就可以轻松复制它。你只需要几个马达、晶体管、电解电容和一些无源器件。

这个想法很简单——提前出马，成为第一，赢得奖品——但这并不容易。当开关未被按下时，电路给电容器充电。按下它，马鼻子向前，排水帽。帽子里永远没有足够的巧克力到达终点线，所以真正的游戏是积累比其他人更多的果汁，然后保持领先或在下一次冲刺时超过他。请下注，并在休息后观看比赛。

记分牌将是这场比赛的一大亮点。如果你想保持机电，我们有一些[托特板灵感](https://hackaday.com/2015/11/04/tote-boards-the-impressive-engineering-of-horse-gambling/)给你。
