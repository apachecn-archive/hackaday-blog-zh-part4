# AI 相机知道它的 S**t

> 原文:[https://hackaday.com/2022/01/17/ai-camera-knows-its-st/](https://hackaday.com/2022/01/17/ai-camera-knows-its-st/)

大多数养狗的人都有一个问题。狗会在你的院子里到处留下它们的…副产品。有些人马上就捡起来，有些人就不管了。但是如果你的狗跑出了院子呢？你怎么知道这些东西藏在哪里？安全摄像头和[人工智能图像检测是答案](https://www.youtube.com/watch?v=uWZu3rnj-kQ&t=2s)，但可能不是你想的那样。

你可能会像我们一样认为，你可以训练系统来识别——嗯——堆。但相反，[Caleb]选择让人工智能做动物姿态估计，以检测狗的姿态，同时产生目标。这可能比识别出一堆难以描述的东西更容易，然后不管它是否被雪覆盖。

狗的姿势既表明事件已经发生，又暗示了事件发生的地点。地图会在网页上更新一个红圈，这样你就能正确识别地雷的位置。我们期待着一个机器人来捡起它，但也许这是今年晚些时候的一个项目。

虽然这个系统是为 Twinkie 调整的，但它可能也适用于许多其他狗，尽管我们知道至少有一只狗有一个标志性的姿势，但在整个过程中都会移动，所以我们打赌你必须重新训练这种行为。

看起来这个系统在室内是物有所值的，并且[连接到你的机器人吸尘器](https://hackaday.com/2016/08/24/roomba-vs-poop-teaching-robots-to-detect-pet-mess/)。我们之前实际上已经看过[类似](https://hackaday.com/2014/11/08/dog-tracker-knows-where-the-dirt-is/)的东西(视频还在，但是链接死了)。
