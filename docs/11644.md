# 在这种新颖的挤出机设计中，旋转的线对长丝产生咬合力

> 原文:[https://hack aday . com/2021/10/20/spinning-threads-put-the-bite-on-filament-in-this-novel-extruder-design/](https://hackaday.com/2021/10/20/spinning-threads-put-the-bite-on-filament-in-this-novel-extruder-design/)

当谈到 FDM 3D 打印的创新时，似乎没有多少空间可以移动指针。几乎所有关于灯丝印刷的东西都已经付诸实践，几乎所有组件都是现成的。即使是商业终端——挤出机——也是如此优化，以至于没有多少创新的空间。

还是有？在[大卫·莱特纳]看来，的确如此，这就是为什么他建造了[这个滚动螺杆挤出机](https://www.thingiverse.com/thing:4973115)(如果你可以访问 Thingiverse 链接，【大卫】[也在 reddit 上交叉发布](https://www.thingiverse.com/thing:4973115))。标准挤出机的工作原理是压带轮，相对较软的细丝通过一个连接在步进电机上的弹簧齿轮。步进器转动齿轮，齿轮将灯丝推进热端或从热端收回灯丝。[David]的设计使用了安装在两个环之间的三个螺杆。杆相对于环的中心轴成一定角度，形成一个大小正好适合细丝的通道。当环旋转时，杆上的螺纹与细丝接合，围绕细丝的整个圆周抓住细丝，并根据细丝旋转的方向推进或缩回细丝。下面的视频显示它正在工作；我们不得不承认看起来很迷人。

[David]自己也承认这并没有太大的优势，也许除了更低的跳跃倾向之外，因为力会分布在灯丝的整个表面，而不仅仅是一个小的夹点。不管怎样，我们喜欢导致这种事情的那种想法，而且我们敢打赌它可能有看不见的好处。也许挤压机实际上是一个创新的地方。见证[这个模块化喷嘴交换系统](https://hackaday.com/2021/04/13/removable-extruder-pulls-out-the-stops-on-features/)。
