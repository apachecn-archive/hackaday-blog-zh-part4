# 鼠标控制的鼠标控制器是愚蠢的，但可能是有用的

> 原文：<https://hackaday.com/2020/11/29/mouse-controlled-mouse-controller-is-silly-but-could-be-useful/>

无用的机器通常是作为一种有趣的消遣而建造的，因为根据它们的定义，它们没有任何价值。最流行的类型通常包括一个自动取消开关。然而，还有很多其他无用的机器要制造，我们认为(杰弗瑞的)制造特别有创意。

该构建由移动标准计算机鼠标的 XY 机架组成。为了控制机架，Raspberry Pi 向系统提供与插入单板计算机的第二个鼠标的运动相关的 g 代码。总体来说，这是非常标准的做法，Pi 向 Arduino 发送命令，Arduino 通过 CNC 控制器屏蔽运行各种步进电机。

是的，这是一个移动鼠标的鼠标——从表面上看，这似乎是一个非常无用的机器。然而，我们可以想象它对于使用非标准鼠标的非常旧的系统的远程控制是有用的，否则很难模仿。此外，将 XY 机架转变成一个合格的笔式绘图仪并不需要太多额外的工作—[我们已经见过很多这样的](https://hackaday.com/tag/pen-plotter/)。休息后的视频。
