# 独特的平板显示器在 CRT 游戏机中投入使用

> 原文：<https://hackaday.com/2018/11/13/unique-flat-screen-display-put-to-use-in-crt-game-boy/>

从最早的电视时代开始，阴极射线管一直统治着显示世界，直到最近，平面屏幕技术开始占据主导地位。在那段时间里，CRT 一直在变大，直到它们达到极限，超过这个极限，显像管就变得太大而不实用了。

但是 CRT 低端市场也有所动作。微型 CRT 出现在各种产品中，从便携式摄像机到著名的索尼 Watchman。这个小组中的一个漂亮的 CRT，一个视频对讲系统的扁平(tish)管，最终出现在[bitluni]的实验室中，在那里他正在把它变成一个带有 CRT 显示器的复古游戏男孩克隆体。这个曾经显示来自安装在门上的摄像机的视频的显示器是一个观众送的礼物。显示屏上的日期代码显示这是一个令人惊讶的新设备；单色 TFT 显示器在 2007 年很难获得吗？无论如何，这是一个简洁的设计，电子枪向上射向弯曲的荧光屏。通过一点谷歌辅助的逆向工程，[Bitluni]能够跟踪完成使用他的复古游戏机所需的视频连接，该游戏机使用的是[一个输出复合视频的 ESP32】。他收获了用于游戏音频的对讲机扬声器，添加了一个临时的任天堂游戏手柄，很快他就可以在平板屏幕上以辉煌的单色玩俄罗斯方块了。](http://hackaday.com/2018/02/22/software-defined-television-on-an-esp32/)

下面的视频只是一个系列中的第一个，原型将被塞进一个漂亮整洁的包中。它当然还需要一些调整，但这是一个很好的开始。我们等不及要看成品了。
