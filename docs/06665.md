# 启动你的引擎！

> 原文：<https://hackaday.com/2020/05/23/start-your-engines/>

又来了:[2020 年黑客日大奖刚刚揭晓](https://hackaday.com/2020/05/19/2020-hackaday-prize-reveals-four-open-challenges-and-new-dream-team-program/)！像往常一样，我们希望看到你们都善用自己的力量，帮助这个世界变得更美好。今年的转折是，四个非营利组织已经被选中，你的工作是帮助他们实现他们的目标:开发帮助海洋保护的解决方案，为脑瘫患者创建或重新设计开源辅助工具，为有需要的社区设计模块化住房，以及设计可以在实地轻松构建的开源医疗和技术工具。

你有多少次想帮忙，但是由于缺乏解决问题的背景知识，或者从哪里开始而止步不前？这就是与非营利组织合作的意义所在，这些组织现在已经有了切实的需求。

哦，我们提到奖金了吗？你不仅能做好事，而且还会做得很好！最佳全能解决方案获得 50，000 美元，有四个 10，000 美元的奖项，每个非营利组织一个，3，000 美元的荣誉奖，5，000 美元的外卡，20 个 500 美元的社区选择奖，然后是 12 个为期两个月的梦之队奖金。

Pshwew！每个人都有适合自己的东西，这是由我们的赞助商实现的:Supplyframe、Digi-Key、Microchip 和 ARM。

我们有四个好方法让你做好事。出去玩黑客吧！
