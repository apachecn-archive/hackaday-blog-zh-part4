# 建造沉降物的超级雪橇

> 原文：<https://hackaday.com/2021/05/31/building-fallouts-super-sledge/>

辐射系列游戏有各种各样可笑的武器，尤其是超级雪橇——一种火箭推进的大锤，看起来对使用者和对手一样危险。[所有的睚鲁]决定他必须在现实生活中重建这个建筑，不管风险有多大。

不愿意走一次性使用的固体火箭建造路线，[JAIRUS]转而选择使用电动管道风扇，并辅以丙烷供应来增加火焰。它实际上不是任何形式的火箭，燃烧的丙烷也不太可能增加任何真正的推力，但它确实从后面射出巨大的火焰，这很可怕。EDF 空转速度可以通过连接到速度控制器的伺服测试器上的电位计来设置，同时还有一个用于调节丙烷流量的阀门。然后可以使用开关将 EDF 速度提高，并增加丙烷流量，从而增加锤后部的流量强度。

值得注意的是，[睚鲁]实际上并没有演示如何挥动锤子。我们有点高兴，因为我们怀疑它可能会以一次相当大的爆炸结束，或者至少是燃烧。尽管如此，在你参加的任何万圣节派对上，它都很容易成为最可怕的道具武器。它与同样设计了的火焰漩涡加农炮【睚鲁】一脉相承。休息后的视频。
