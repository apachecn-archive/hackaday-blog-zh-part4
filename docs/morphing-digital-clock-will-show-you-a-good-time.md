# 变形数字钟会给你一个美好的时光

> 原文：<https://hackaday.com/2018/07/10/morphing-digital-clock-will-show-you-a-good-time/>

几周前，[HariFun]开始用 LED 矩阵模拟 7 段显示屏。看起来很简单，对吧？没错。他还想想出一种在数字之间转换的新方法，这是一项困难得多的任务。但他做到了，这真的很酷。在一位观众的建议下，[Hari]将这种转变作为[一个迷人的时钟的基础，它将模拟二手货平稳地带入数字时代](https://www.instructables.com/id/Morphing-Digital-Clock/)。

这是自沙漏以来最酷的观看时间流逝的方式。当一个数字滑入下一个数字时，你几乎可以听到光在移动。每个过渡都是完全唯一的，因此根据手指的不同，这涉及一个或多个从右向左滑动的垂直段，或者多个逆时针旋转的段。

你也可以通过一个 64×32 RGB LED 矩阵、一个 NodeMCU 和[【Hari】的数字转换代码](https://github.com/hwiguna/HariFun_166_Morphing_Clock)来观看时间流逝。它只需要花费 25 美元，你真的不能击败他的教学质量。休息之后，花一两秒钟来看看。
