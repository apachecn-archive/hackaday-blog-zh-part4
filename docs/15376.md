# 乳齿象来到 IBM 个人电脑

> 原文:[https://hack aday . com/2022/11/15/mastodon-comes-to-the-IBM-PC/](https://hackaday.com/2022/11/15/mastodon-comes-to-the-ibm-pc/)

埃隆·马斯克(Elon Musk)以令人垂涎的价格收购了 Twitter，他在社交媒体公司混乱管理中的现场冒险已经成为我们其他人的引人注目的表演。当我们嚼着美味的爆米花，享受节目时，许多 Twitter 用户已经转而使用开源的替代产品乳齿象。它为逃脱者提供了很多东西，包括为特定社区定制的实例，但除此之外，它还有 Twitter 从未有过的东西。你现在可以在 IBM 个人电脑上使用乳齿象客户端了。

毫无疑问，你们中的许多人正以怀疑的眼光看着我们，因为你们已经在电脑键盘后面嘟嘟叫了好几年了。但很可能你正在使用的 PC 是一台普通的现代 x86 机器，运行最新的操作系统，如 GNU/Linux 风格或微软 Windows，相比之下，这里我们指的是原始的，它们的父亲。因为我们正在谈论的客户端是 DOStodon，它被设计成在真正的 IBM 个人电脑上运行，就好像又回到了 20 世纪 80 年代初。

撇开特技黑客不谈，不管你是否在 16 位机器上启动 DOStodon 来获得你的 Fediverse 补丁，它都是一个有趣的软件，因为它是用 Javascript 编写的。这又把我们带到了同一个作者的 DOjS，一个带有声音的 DOS Javascript 画布。不是每个人都渴望在 20 世纪 80 年代早期的 PC 上运行他们的 Javascript 代码，但它的存在证明了旧平台仍有大量的生命。
