# 这种便利的缩微胶片阅读器照明复古数据表

> 原文：<https://hackaday.com/2020/12/20/this-expedient-microfiche-reader-illuminates-retro-datasheets/>

你必须是某个年代的人才能记得做过微缩胶片和缩微平片的研究。在公共记录、期刊和其他模糊的历史信息大规模数字化之前，枯树记录被光学压缩到精细的胶片上，要么是卷状，要么是平板状，后来被放大并在专门的阅读器上显示。这极大地减少了文件所需的存储空间，但一旦计算机时代到来，这最终成为一个技术上的死胡同。

这就是[CuriousMarc]最近遇到的问题——缩微胶片上的惠普元件信息宝库，却没有小型数据表的阅读器。所以很自然地，他建造了自己的缩微胶片阅读器。幸运的是，他得到了一台似乎非常适合这项工作的低价数码显微镜。不过，这台带有高清摄像头和 5 英寸液晶显示屏的显微镜更适合反射使用，而不是透射使用，所以[马克]很难获得缩微胶片的像样照片，即使有白纸做背景。

2.0 版本使用了废弃的 DVD 播放器屏幕上的背光，作为显微镜载物台的一种灯箱。这几乎是缩微胶片的完美尺寸，显微镜能够像任何专用的缩微胶片阅读器一样放大微小的字符，而价格只是它的一小部分。查看下面的视频，了解构建的详细信息，以及[Marc]从数据表中了解到的关于他的惠普零件累积奖金的信息。

有了储存在缩微胶片上的大量数据，我们惊讶地发现以前从未见过这样的阅读器。不过，我们已经讨论过微观的战时邮件。
