# 不给糖就捣蛋的大炮会给他们糖果壳冲击

> 原文：<https://hackaday.com/2020/10/23/trick-or-yeet-cannon-will-give-them-candy-shell-shock/>

关于万圣节，我们最喜欢的事情之一是每年由此而来的黑客数量。今年，黑客几乎是保持物理距离的要求，但这并不意味着我们不能玩得开心。你想找乐子？从五十英尺外向不给糖就捣蛋的人发射糖果炮弹的糖果大炮怎么样？

[保罗·麦凯布]的大炮使用喷水阀和空气压缩机发射一对有趣大小的糖果棒，每个糖果棒都被包裹在一个薄膜罐壳中。每个“不给糖就捣蛋”的人都会在车道尽头 50 英尺远的地方踩一个脚踏开关，只要有人在那里按住主点火器，大炮就会发出美妙的回应，听起来就像大型管乐器演奏 D 音。

得知降落伞的想法没有实现，我们很难过，但是荧光棒是夜晚的一个很好的补充。休息后看看演示，随后是构建视频，然后是更多有趣的发布。没有足够的时间来制造这种口径的大炮？你可以很快组装一个怪异的六英尺长的滑梯。
