# 年轻的制造商将传统的日本建筑与现代艺术相结合

> 原文:[https://hack aday . com/2022/01/25/young-maker-mixes-traditional-Japanese-construction-with-modern-art/](https://hackaday.com/2022/01/25/young-maker-mixes-traditional-japanese-construction-with-modern-art/)

我们是创造者。根据定义，我们制造东西。我们中的一些人喜欢从碎片中构建，而另一些人喜欢在他们的车库中制作他们自己的 IC。另一方面,[用英里制作]更喜欢最古老的制作方式之一:木工。在这次建造中，他更进一步，使用一种非常古老的日本木工方法 Kumiko 建造了一把 Stratocaster 风格的电吉他。结果绝对令人震惊[，正如你在下面的视频中看到的](https://www.youtube.com/watch?v=U5r_PE2OznM)。

受[ [现代创客播客](https://www.stitcher.com/show/the-modern-maker-podcast) ]提出的挑战的启发，建立一个与另一个与木工无关的爱好相联系的木工项目，[Miles]在这个一次性的战略中加入了几种艺术形式，使它出类拔萃。

这款吉他的核心是琴身采用的 Kumiko 风格。Kumiko 是一种不使用紧固件组装木材的日本方法。Kumiko 发展于公元 600-700 年，既是一种艺术形式，也是一种建筑方法。[Miles]更进一步，用黑化的环氧树脂填充 Kumiko 框架，然后打磨抛光。贴花将床头带入主题，但对细节的关注远远不止于此。请务必观看视频，这样你就可以欣赏这位年轻人展示的高水平工艺。

没错——【Miles】不是有几十年经验的创客。事实上，在 2017 年，他的一个 YouTube 视频是“ [12 岁造划艇！！！](https://www.youtube.com/watch?v=xBBnSBIv2cs)“[Miles]，我们向您致敬，我们期待看到您的艺术进步，因为您确实赢得了创客社区的关注，您理应成为其中的一员。

正如[Miles]吉他隐藏了一些伟大的黑客一样，这款带热插拔拾音器的吉他也是如此。感谢[Keith]将这篇文章发送到[的举报热线](https://hackaday.com/submit-a-tip/)！
