# 仅使用发光二极管的光通信

> 原文:[https://hack aday . com/2019/12/22/optical-communication-using-LEDs-alone/](https://hackaday.com/2019/12/22/optical-communication-using-leds-alone/)

我们都习惯于将不起眼的 LED 作为无处不在的光源，但有多少人知道这些元件也可以用作光电二极管？Giovanni Blu Mitolo 向我们展示了一个简单的数据链接，它只使用了一对 led 和几个 Arduinos。这是展示他的 PJON 网络层，虽然对于真实世界的应用程序，您需要的不仅仅是试验板上的几个 led，但我们仍然认为这是一个简洁的演示。

PJON 本身非常值得一看，它是 Arduinos 和其他小型微控制器平台的健壮和容错网络的实现。它有各种不同媒体的各种通信策略，正如这个 LED 演示所显示的那样，它的优势在于它能够通过其他网络不愿通过的媒体工作。无论是通过金属加热管道控制家庭自动化，还是在 433 MHz 上提供 LoRa 的替代方案，都绝对值得再看一眼。我们在之前提到过它，但仍然惊讶的是，从那以后我们并没有更经常地看到它。看看吧，视频在休息下面。
