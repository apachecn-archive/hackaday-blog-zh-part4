# 一种实用的双臂 SCARA 3D 打印机

> 原文：<https://hackaday.com/2020/03/18/a-practical-dual-arm-scara-3d-printer/>

我们在 Hackaday 看到了很多 3D 打印机，但随着这些年来 3D 打印机已经从一个令人兴奋的项目变成了一个日常工具，我们越来越少将 3D 打印机作为一个项目。我们很少看到 3D 打印机不是 XYZ 笛卡尔设计或德尔塔打印机的变体，但这就是[bondus]对基于平行 SCARA 机制的打印机所做的。如果你不熟悉 SCARA，它是工业机器人世界中使用的一种设计，其中一个几乎人形的关节臂在二维空间中工作，第三维空间通过升高或降低整个结构来提供。与笛卡尔设计相比，它具有速度更快的优势，但代价是需要更高质量的关节来保持定位的准确性。

这是他制造的第二台 SCARA 打印机，有一套坚固的铝臂和坚固的轴承。驱动力通过一对皮带传到一些非常大的滑轮上，校准对于确保两臂完全处于同一平面非常重要。圆形床位于提供 Z 轴的丝杠上。

结果当然令人印象深刻，无论是速度还是打印质量。我们在广告下方放了一段视频。SCARA 打印机是否会发展到无处不在的地步，我们无法给出答案，但我们在过去展示过其中的一小部分。特别令人难忘的是[这个使用了工业机械臂](https://hackaday.com/2013/10/09/scara-arm-becomes-enormous-3d-printer/)。
