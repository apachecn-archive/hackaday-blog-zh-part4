# 在 2022 年建造一艘全新的 Commodore 64，配备所有新部件

> 原文：<https://hackaday.com/2022/12/28/building-a-new-commodore-64-in-2022-with-all-new-components/>

称它为假的或只是新的，但当[DusteD]着手建造一个只有新零件的全新 Commodore 64 时，它导致了 Project [MaxFake64](http://dusted.dk/pages/c64/MaxFake64/) 与任何真正的 C64 在电气和二进制上兼容。虽然从 C64 模拟器是假的这个意义上来说，它不是真的“假”，但从这个意义上来说，它没有使用这个千年之前生产的部件。这实际上可能比现在得到一个完全工作状态的二手 C64 更容易。

总的来说，该项目包含一个由 Electroware 提供的售后 C64 电源、一个全新的 C64C 外壳、一个基于正版主板的 C64(ASSY NO . 250407)主板、一个通用 RF 模块化模块、一个基于 FPGA 的 *Kawari* VIC-II 替换件、一个使用 Monotech PCs 的 6502-to-6510 适配器的 6502 MPU、一个基于双 GAL 的 PLA 替换件、用于内核、字符和基本 rom(带有插座式 hacks)的 EPROMs 以及一个 SinSID

在此过程中发现的问题包括(透明的)C64C 外壳的一些裂缝和缺乏 CIA 替代品，如 [J6526](https://1nt3r.net/j-cia/) 。键盘也将在以后的某个时候被替换，像操纵杆端口这样的项目是从旧的 C64 中抢救出来的，而不是购买全新的。这些都不是根本性的问题，而且在未来寻找替换零件时，这些问题实际上可能具有经济意义。
