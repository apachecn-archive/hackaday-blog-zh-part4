# Nerf 爆能枪变成光枪控制器

> 原文：<https://hackaday.com/2020/10/31/nerf-blaster-becomes-light-gun-controller/>

传统的光枪依赖于阴极射线管技术的怪癖，因此不适合现代液晶电视和显示器。然而，铁杆复古玩家并不以脱离经典游戏而闻名，他们坚持不懈地开发新的硬件来适应旧游戏。在这种情况下，[布里特利夫]拿了一些 Nerf 爆能枪，[造了一对可以用今天的硬件工作的轻型枪械。](https://www.instructables.com/Nerf-Light-Gun-Controller/)

该版本依赖于 Ultramarc 的轻型枪械套件，其工作方式与最初的 Wiimote 类似。blaster 内部的摄像头用于对屏幕顶部的 LED 条进行三角测量，以实现清晰准确的跟踪。[BrittLiv]将 Ultramarc 套件与一些巧妙的黑客技术结合成一个 Nerf DoubleStrike blaster，悄悄地将按钮隐藏在里面，以便与原始的触发器和击发机构以及导轨中的锁定片进行交互。

有有线和无线两种版本，这种设置看起来是享受经典的方式，如*猎鸭*和*直射。*由于 Ultramarc 硬件模拟标准 USB 鼠标，blasters 可与 MAME 和 RetroPi 等常见平台配合使用。

我们以前见过一些疯狂的光枪黑客，就像这个建筑，它使用相机和数学让东西在没有 LED 灯的情况下工作！休息后的视频。
