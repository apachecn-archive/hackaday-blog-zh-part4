# 回到黑客空间

> 原文：<https://hackaday.com/2021/09/04/getting-back-into-hackerspaces/>

上周，我第一次有机会走出去和黑客们在一起，感觉就像永远一样。德国的黑客空间终于能够像往常一样重新开放，允许合理数量的人进入，前提是他们已经免疫或经过测试，当然还要戴口罩。这意味着我要接受安德烈亚斯的邀请，亲自来看他的立体忍者检查显微镜项目。

Stereo Ninja 基本上巧妙地利用了两个 Raspberry Pi 相机，交换了光学系统以获得更大的放大，并在 3D 显示器上显示结果——用快门眼镜观看。这是一个你必须亲自去看才能“理解”的项目。他仍在致力于简化构建，使之更简单、更实惠，让这个项目更容易被普通黑客接受。

我们谈到了 3D 显示器狄莺。事实证明，快门眼镜很便宜，看起来像是通过红外脉冲同步到显示器上的。应该有一个黑客的 3D 解决方案，至少可以与快速游戏显示器一起工作。[Andreas]还向我介绍了这款用于 Raspberry Pi CM4 的[出色分线板，该分线板将两个相机通道分开，便于立体/ 3D 拍摄。我参观了](https://www.waveshare.com/cm4-io-base-a.htm)[私人实验室](https://hackaday.io/fablabmuc)，我们谈论了焊接、金属 3D 打印、软件、硬件和各种无聊的东西。[ [阿历克斯](https://hackaday.io/tinyledmatrix) ]周末在他出城的路上出现了——我们已经很久没一起出去玩了。

简而言之，我记得以前是怎样的，那时可以访问其他黑客，访问其他黑客空间。这是一种自发的、相互启发的聊天，这种聊天在远程是不可能的，而且非常重要。

恐怕我们还没有完成 COVID 疫情，世界不同地区有完全不同的轨迹。如果你两年前告诉我，我会带着口罩和疫苗证明去黑客空间，我会认为你疯了。但与此同时，这次短暂的访问给了我一点对未来的希望。我们将度过这一切，我们将在我们当地的黑客空间再次相聚。
