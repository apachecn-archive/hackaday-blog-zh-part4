# 人机

> 原文:[https://hackaday.com/2021/07/31/the-man-machine/](https://hackaday.com/2021/07/31/the-man-machine/)

本周，我们看到了几个用于小型家庭制造的 DIY 工具，如果你正在爬出[快乐桶，进入绝望坑](https://www.sparkfun.com/news/909)——当你制造了足够多的物品，不再有趣，但你仍然没有能力将制造任务留给别人，这些工具会让你的生活变得更容易。

第一个是由 3D 打印机制成的自动[通孔焊接机。这实际上是有意义的，即使你正在为你组装电路板，因为通孔焊盘比 SMT 器件贵得多，而且它们通常按每个引脚收费。在你的项目上放一个 2×20 的引脚头，最终可能会花很多钱。或者你可以将解决方案自动化。](https://hackaday.com/2021/07/22/modified-3d-printer-solders-through-hole-components/)

[![](../Images/0898a80b4add285067fdf23047fbcc98.png)](https://hackaday.com/wp-content/uploads/2021/07/pnpassist_thumbnail.png) 本周的第二个解决方案确实引起了我的注意。 [PnPassist](https://hackaday.com/2021/07/23/pnpassist-a-smart-build-platform-for-manual-pcb-assembly/) 是一种机器，它可以转动你的 PCB，将激光十字准线定位在你需要放置的下一个 SMT 片上，甚至还有一个有机发光二极管屏幕，告诉你应该在那里放置什么。这里有许多伟大的机械设计选择，但真正吸引我注意的是这台机器如何很好地填补了手动和全自动取放之间的空白。

我知道你不喜欢在电路板和原理图或器件清单之间来回寻找，试图找到 Q23 在该死的电路板上的位置，或者查找电阻值。使用 PnPassist，您仍然需要进行放置，但需要机器的指导。如果你没有钱或空间买一个全自动的 PnP，这是一个明显的胜利，但对于短时间运行来说，加载卷轴比填充电路板需要更多的时间，这可能是一个巨大的胜利。

我喜欢这种增强人类能力的机器，我总是很高兴看到这样的设计。这让我想起了[非常聪明的牛头刨床原点](https://hackaday.com/2016/05/20/shaper-tools-will-blow-your-mind/)，甚至只是这个[方便的自动 XY 工作台，用于钻许多精确的孔](https://hackaday.com/2015/02/04/semi-auto-pcb-drill-press-makes-drilling-semi-painless/)。在所有这些情况下，都有一部分问题很难解决，需要非常庞大或昂贵的机器，或者可以通过肉袋更简单地完成。但是，将机器精度与人的因素结合起来，产生的不仅仅是这些部分的总和。
