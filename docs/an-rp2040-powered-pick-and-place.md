# RP2040 动力抓放器

> 原文：<https://hackaday.com/2022/11/07/an-rp2040-powered-pick-and-place/>

拾取和放置机器是一个值得一看的奇迹，因为它们精细而准确地放置一个又一个零件。不幸的是，它们必须有一个同样令人惊叹的价格标签。幸运的是，制作它们并不太困难，因为它们共享 3D 打印机的许多属性，但有一些额外的限制。[Stargirl Flowers]发布了 [Starfish，这是一款基于 RP2040](https://blog.thea.codes/starfish-a-control-board-with-the-rp2040/) 的开源取放控制板，帮助人们制作自己的取放板。

她买了一辆 LumenPnP，修理的渴望变得无法忽视。股票控制器上的 STM32 也碰巧被烧毁，留下了一个明显的开口来创建一个定制板。[Stargirl]选择 Trinamic TMC2209 电机控制器来驱动三个步进电机。令人印象深刻的是，电源电路内置了一个 3A 保险丝、一个用于分流电压尖峰的 TVS 二极管、一个用于反极性保护的 P 沟道 MOSFET、一个用于交流纹波的低通滤波器和一个 100μF 大电容。

RP2040 是一个很好的选择，因为它很容易获得，并且有大量的数字 I/o。USB 将电路板连接到外部工作，并且包括 ESD TVS 二极管，以便在连接和断开 USB 端口时保护电路板。真空电机由 74HC2G34 缓冲器控制，该缓冲器驱动两个 MOSFETs 的使能线。螺线管是相似的，但是具有高的电流峰值和小得多的电流来保持它们打开。DRV120 符合要求，因为它是带电流调节的单通道继电器。I2C 真空传感器与 Lumen 主板上的传感器相同；他们只需要一个 I2C 多路复用器。

这是一个记录非常完整的项目，解释了为什么选择每个部分以及为什么。如果你想创建一个需要持续的 RP2040 项目，我们认为这是一颗导航星。GitHub 上的[都在等你来看一下](https://github.com/wntrblm/jellyfish-and-starfish)。

这不是我们第一次看到 [RP2040 作为电机控制器](https://hackaday.com/2022/06/14/the-rp2040-doth-a-motor-controller-make/)的一部分，我们怀疑我们会看到更多。
