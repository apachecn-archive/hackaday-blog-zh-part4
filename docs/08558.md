# 菠菜照片印花

> 原文:[https://hackaday.com/2021/02/13/spinach-photo-prints/](https://hackaday.com/2021/02/13/spinach-photo-prints/)

有些人喜欢在沙拉里放菠菜。其他人更希望它永远不要靠近他们的叉子。然而，其他人，如 Almudena Romero，用它来打印图片，他们是我们今天要关注的人。

Anthotypes 是由植物染料制成的正面图像。想象一下，你在野餐时弄脏了你的衬衫，然后把它放在阳光下，用叉子盖住部分污渍。当你回来的时候，没有被餐具遮挡的污渍不见了，但现在你有了一个用贝芙阿姨的烧烤酱制成的永久的叉子形状的标志。这种版画背后的科学在休息时间下面的视频中得到了很好的阐述。你看，有些植物染料不适合轻度漂白，如果你没有耐心，就更少了，因为像蓝莓这样的污渍可以在阳光下晒一个月。

这段视频展示了如何制作你自己的植物染料，这种染料除了打印之外还有其他用途。由于染料在阳光下褪色，它可以是一种临时的油漆，或者你可以在你的花园里到处寻找样本，因为暴露最严重的样本褪色最严重。想象一下一个低技术含量的紫外线测量仪，它有测井功能，但却是靠菠菜运转的。

如果科学不能引起你的兴趣，艺术的可能性也同样很酷。所有的图片都有一个独一无二的 wabi-sabi 耀斑。你拿你最喜欢的照片，把它做成单色，印在透明的塑料片上，墨水会屏蔽染料，让其余的曝光。我们刚刚给了你一个在户外寻找阳光最充足的地方的建议，所以去染色吧。

Anthotype 印刷与电路板印刷过程中的[抗蚀剂有一些相似之处，但也许有人可以用](https://hackaday.com/2011/02/28/direct-to-pcb-etch-resist-printing/)[激光曝光](https://hackaday.com/2017/07/14/laser-exposing-pcbs-with-a-blu-ray-laser/)重新混合菠菜印刷！
