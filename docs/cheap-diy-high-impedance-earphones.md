# 廉价的 DIY 高阻抗耳机

> 原文：<https://hackaday.com/2020/12/03/cheap-diy-high-impedance-earphones/>

水晶收音机能感受到神奇，因为它们能从远处的电台收听音频，除了无线电信号本身的能量之外什么都没有。然而，为了实现这一壮举，他们通常依赖高阻抗耳机来产生非常小电流的可听声音。这些耳机很难找到，因此可能很贵。然而，[比利]想出了一种廉价制造它们的方法。

建造从一个普通压电蜂鸣器开始。它被拆掉了，里面多余的电路被移除了。压电元件本身然后直接连接到单声道音频插孔，用于[比利的]水晶收音机之一。为了使它成为一个可用的耳机，笔尖被切掉并粘在蜂鸣器的塑料外壳上。然后，使用常规现代耳塞中的橡胶耳罩来确保耳朵紧密舒适。

这是一种很好的方式来建造一些现在很难找到的东西，我们打赌[Billy]的设计比 90 年代在 Radio Shack 工具包中运送的硬塑料模型更舒适。当然，还有其他方法来构建高阻抗驱动器，[我们在](https://hackaday.com/2016/09/18/high-impedance-headphones-theyre-in-the-can/)之前已经介绍过。而且，如果你正在寻找建立一个水晶收音机，很难通过[【比利的】信用卡芯片建设](https://hackaday.com/2016/09/18/high-impedance-headphones-theyre-in-the-can/)。休息后的视频。
