# 橡胶和电脑接触的地方

> 原文:[https://hack aday . com/2021/12/21/橡胶在哪里遇到电脑/](https://hackaday.com/2021/12/21/where-the-rubber-meets-the-computer/)

如果你有机会去莱顿，就去吧。这是一个美丽的小城市，隐藏着一些高功率的大学研究。它还拥有世界上第一台[橡胶电脑](https://phys.org/news/2021-12-floppy-rubber.html)。不过，你不会很快在上面运行*孤岛危机*。这台初出茅庐的计算机有记忆，可以数到二——实际上更像是一台状态机。观看下面的视频比试图完全解释它更容易。或者你可以通读[的实际论文](https://www.pnas.org/content/118/50/e2111436118)。

如果你看视频，你会看到波纹橡胶结构中的变形显然是可重复的，代表着机器中的比特。按压和释放结构上的压力会形成输入和时钟，压缩时材料可能会从状态 A 进入状态 B，但当释放压力时，它会进入状态 c。压缩和压力角度允许不同的输入条件。一个例子橡胶状态机计算你压缩橡胶多少次。

你用一块智能橡胶做什么？我们不知道。也许如果你想让鞋子计数，这样你就可以每分钟发送一次计数，以节省电池？研究人员承认，他们心中也没有任何具体的应用，但假设有人会想使用他们的工作。

当然，视频的标题:“橡胶电脑”有点夸张，但我们可以原谅它。大多数人不会得到“橡胶有限状态机”虽然机械计算可能看起来有点过时，但事实证明[在分子水平上](https://hackaday.com/2015/10/19/rod-logic-and-graphene-elusive-molecule-scale-computers/)它可能变得非常重要。此外，你可以用硬纸板把[做成一台电脑(或者](https://hackaday.com/2014/10/14/a-clever-cardboard-computer/)[用 FPGA](https://hackaday.com/2017/10/13/computers-that-never-were/) 或电子表格模拟那台电脑，如果你喜欢的话)。
