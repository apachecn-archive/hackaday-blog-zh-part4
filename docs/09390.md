# 一种新的开源农业机器人成形了

> 原文:[https://hack aday . com/2021/03/05/a-new-开源-农耕-机器人-成形/](https://hackaday.com/2021/03/05/a-new-open-source-farming-robot-takes-shape/)

自动化农业的世界对那些没有投资于它的吸引力的人来说可能是一个乏味的世界，但就像机器人自己悄悄地在后台照看庄稼一样，那些遵循这条道路的人花了许多季节来完善他们的设计。Acorn 是一个来自加州研究农场 Twisted Fields 的[新开源机器人，它提供了一个从萌芽到农业机器人设计的迷人视角。](https://github.com/Twisted-Fields)

Acorn 不是像 Farmbot 这样设计的用于小型集约化花园的数控龙门起重机，而是一种用于大型农场的自主太阳能漫游车，它将在田地中持续巡航，照顾其地块中的植物。这是一项正在进行的工作，所以我们看到的是完成的火星车，带有工具和机器视觉。它追求低成本的轻型平台，太阳能电池板覆盖的铝制底盘，每个角落都有山地车前叉衍生的车轮。它有四轮驱动和四轮转向，这意味着它可以穿越最崎岖的农田。我们可以看到它自 2019 年原型以来的进展，尽管它似乎像季节本身一样缓慢地成熟，但我们可以看到最终版本可能是一个小农场上非常有用的机器。

这不是我们这些年来看到的第一个自主农业机器人，例如这个稍微更强壮的澳大利亚模型。我们猜测这是自主农业可能采取的方向，一些制造商设计的更传统的基于拖拉机的机械[承担重复装载和运输的角色](https://hackaday.com/2016/04/05/autonomous-tractor-brings-in-the-harvest/)。
