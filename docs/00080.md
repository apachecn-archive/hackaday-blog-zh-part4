# 香蕉反对种族主义

> 原文:[https://hackaday.com/2018/07/20/bananas-against-racism/](https://hackaday.com/2018/07/20/bananas-against-racism/)

这是一个古老的故事。不是爱，而是保护你创造的东西不被那些想要破坏美好事物的人破坏。在这种情况下，美丽的东西是一个阅读 Twitch 和 Youtube 评论的会说话的香蕉。ne'er-do-wells 是试图通过强迫 anana 背诵受限单词来禁止其帐户的巨魔。

问题源于[Greekgodx]的一次访问，其追随者倾向于黑暗面。当他们把目光投向[迈克·尼科尔斯]的黄色自动机时，事情陷入了一个惨淡的境地，24 小时的禁令落到了水果上。一系列的过滤工作已经完成，但这还不足以阻止钓鱼者，理货员增加了对账户的第二次永久打击。一场诋毁和过滤的军备竞赛随之而来，直到机器人能够拒绝所有种族主义的企图。

香蕉已经从饲料中被剥掉，但是如果愚蠢的机器人是你的[一杯茶](https://hackaday.com/2017/03/11/tea-for-two-a-tiny-tea-timer/)，看看[【西蒙·吉尔兹】](https://hackaday.com/2015/12/29/simone-does-strange-things-with-motors-and-servos/)把[汽车变成电脑鼠标](https://hackaday.com/2018/02/18/turning-a-car-into-a-computer-mouse/)。
