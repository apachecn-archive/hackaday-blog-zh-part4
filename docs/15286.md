# 带有一两个动圈的墙壁艺术

> 原文：<https://hackaday.com/2022/11/05/wall-art-with-a-moving-coil-or-two/>

几乎每一种复古指示器技术，从数码管到 flipdot，以及中间的所有东西，都以某种艺术作品或装饰的形式出现在这些页面上。很高兴看到这么多年来，特别是在我们的同事(Voja Antonic)手中，没有出现这么多。他[拿起一大堆动圈式面板仪表，将它们连接到一个由 PIR 传感器](https://hackaday.io/project/187998-useless-retro-panel-meters)触发的微控制器板上。正常情况下，读数是随机的，但是如果太靠近，所有的指针都会开始移动，从而在电子墙上显示出非常不同的效果。

他没有给我们他使用的控制电路的细节，但从某种意义上说，这无关紧要。我们认为，任何知道烙铁一端和另一端的黑客读者都应该能够从 DAC 产生一个小的 DC 电流来驱动电表，我们认为随机读取数据的软件也不会给你们带来麻烦。

与此同时，[Voja]这些年来已经产生了这么多有趣的项目，尤其是 2022 年超级大会徽章。这是几年前的一张照片。
