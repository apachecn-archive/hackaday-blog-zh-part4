# 用这种微型的 Arduino 动力加农炮杀死所有的可口可乐罐

> 原文:[https://hack aday . com/2019/12/08/death-to-all-Coca-Cola-cannon-with the-miniature-arduino-powered-cannon/](https://hackaday.com/2019/12/08/death-to-all-coca-cola-cans-with-this-miniature-arduino-powered-cannon/)

[MJKZZ]发来这个有趣的小教程，教你如何用注射器制作一个[小型自动加农炮。](https://www.youtube.com/watch?v=21PT2IwS2eY&feature=youtu.be)

他首先改装了一个电弧打火机，一种在刮风天用来点火的奇特的打火机，这样它就可以被一个微控制器控制了。小心地使用电线和热胶将电弧移动到注射器的针头端。当注射器充满一点酒精，原来的活塞被压回时，一个小火花会让它以一种非常令人满意的方式飞回来。

当然，如果没有 Arduino 的加入，那就不算是一个真正的黑客，没有任何理由，除了这样做的乐趣。[MKJZZ]在混合物中添加了一个超声波传感器，当被入侵物体[适当触发时，它会使用簧片继电器点燃电弧点火器](https://hackaday.com/2019/10/27/making-a-bronze-cannon-from-scratch/)。

他通过消除工作台上一个闯入的可乐罐展示了这一构建。休息后的视频里可以看到。总而言之，这是一次非常有趣的黑客攻击。
