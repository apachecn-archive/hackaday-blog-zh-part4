# 一系列的训练让这辆电动多功能车变得更加出色

> 原文:[https://hack aday . com/2019/01/06/a-quartet-of-drills-put-the-spurs-this-electric-utility-vehicle/](https://hackaday.com/2019/01/06/a-quartet-of-drills-put-the-spurs-to-this-electric-utility-vehicle/)

低腰裤款式。四轮驱动。全电动传动系统。在一角硬币上转动。扭断脖子加速。真皮座椅表面。是特斯拉的最新产品吗？没有。是[一辆钻动力的电动多功能车](https://www.youtube.com/watch?v=95wsA5CCTfg)，开起来好像爆炸。

令人惊讶的是，这并不是一个仅仅为了娱乐的版本。[Axel Borg]的小型车外形小，航程远，这实际上是有实际原因的。我们将把背景故事留给下面的第二个视频，但可以说这将是 NASA 用来将火箭推出发射台的履带的较小版本，而不是用来运输[他那看起来极其危险的载人多翼机](https://hackaday.com/2017/07/01/manned-multirotor-flies-again-electric-style/)。这辆车的传动装置很有趣:四个巨大的电钻，每个轮子一个。电钻由一个大的串联电池组供电，充满电后输出 260 伏的电压。钻机的通用电机与 DC 配合良好，每个电机的速度通过来自一对无绳钻机的 PWM 信号控制。下面的第一个视频显示[阿克塞尔]通过它的步伐；他一点也没有退缩，但是那辆车不断回来要更多。

我们知道这辆车正在为另一个项目服务，但如果我们有潜力坐在店里享受这么多乐趣，我们将很难专注于任何事情。尽管如此，我们希望多旋翼飞机能很快进行一次良好的试飞，一切顺利。
