# 泡沫板是制造轻型气垫船的材料

> 原文：<https://hackaday.com/2020/05/12/foamboard-makes-for-a-light-hovercraft/>

如果我们相信许多科幻电影，有一天成群穿着紧身银色弹力连体裤的人将乘坐气垫船四处游荡。气垫船还没有真正风靡世界，但[菲蒂姆-哈利米]建造了自己的模型版本，并向你展示他是如何做到的。你可以在下面的视频中看到这个小飞行器在移动。

理论上，气垫船很简单，但实际上它们并不像看起来那么简单。首先，你需要大量的空气来填充增压室以获得升力。那通常是一个嘈杂的操作。解决办法？在这种情况下，一个吹风机放弃了电机的原因。此外，一旦漂浮在几乎无摩擦的气垫上，你必须在不接触地面的情况下实际移动。像许多真正的气垫船一样，这种设计使用另一个风扇推动它前进。你可以在视频中看到，设计师使用绝地手部动作来控制车辆。

好消息是电机只是单向运行，因此不需要 H 桥来反转电机。一个简单的 FET 开关就可以打开或关闭它们。有一个伺服电机来移动方向舵，从而改变推力风扇的空气输出方向。

如果你想自己尝试一下这个，在 [GitHub](https://github.com/fitim-halimi/RC_Hovercraft) 上有原理图和源代码。我们想知道，在机器人上安装一个传感器，并对其进行有趣的扭转，或者防止它撞到墙上，会有多难。也许反推力风扇也会有用。事实上，两个额外的推力风扇，每个相隔 120 度，可以给你有趣的选择。

我们喜欢泡沫板为身体减肥的想法。我们以前见过肉类包装托盘。如果你没有读过我们关于气垫船历史的帖子，你可能会惊讶这个想法有多古老。我们继续等待我们的[翼气垫船](https://hackaday.com/2013/10/10/hovercraft-plane/)。
