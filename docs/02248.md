# 8 位的家伙建造了一台 16 位的电脑

> 原文:[https://hack aday . com/2019/03/02/the-8-bit-guy-builds-a-16 bit-computer/](https://hackaday.com/2019/03/02/the-8-bit-guy-builds-a-16-bit-computer/)

YouTube 上最好的复古历史学家之一是 8 位的家伙，在多年想做这样的事情后，它终于发生了。这个 8 位的家伙正在建造他梦想中的电脑，他的灵感很大程度上来自于 Commodore 64。

在我们深入了解这台电脑将会做什么和它将会有什么功能之前，重要的是要注意到这个 8 位的家伙在投入一年或更长时间在这个项目上之前实际上正在做一些市场和用户研究。他询问了其他著名的逆向计算 YouTubers，询问他们关于他们的“梦想”逆向计算机应该做什么的意见，他们提出了一个基本的要求清单。梦想中的计算机就像在 1957 年的雪佛兰上工作一样，所有的寄存器都可以立即被窥视和窥探。计算机将是完全可以理解的，到目前为止，一个人可以完全理解一切，从 CPU 内部的单个逻辑门到内核的结构。它会运行基本的。

在树莓派盛行的时代，有人可能会问，“为什么不搭配树莓派呢？”。对于 8 位的人来说，Pi 只是一台 Linux 计算机。与这台梦想中的计算机类似的其他逆向计算项目也失败了:[mega 65](http://mega65.org/)，一个复活 Commodore 65 的项目，将过于昂贵。[基本引擎](https://basicengine.org/)失败是因为它只进行合成，而且它运行在 ESP 上，所以你被屏蔽在真正的硬件之外。同样的问题也存在于 [Maximite](http://geoffg.net/maximite.html) 中，因为硬件是远离接口的一个抽象层。C256 Foenix 可能是最接近设计目标的，但它太贵了，即使没有 MIDI 端口、SID 芯片和其他有趣的硬件，它仍然高于理想的价位。

这款梦想电脑的“要求”是只使用现代部件，有 VGA 或 HDMI 视频输出，一个真正的 CPU，最好是 6502，不使用 FPGA 或微控制器，可以运行 Commodore Basic。此外，这台计算机的成本大约为 50 美元，100 美元是绝对的最高限额(意味着 BOM 成本大约为 15-25 美元)。这是绝对的，完全的，惊人的不可能。如果我没有提到这个项目不可能实现既定目标，那我就是在欺骗你。这个项目将达不到售价低于一百美元的目标。

也就是说，尝试一下没有坏处，所以 8 位的家伙目前正在使用一些开发板，特别是围绕 65816 CPU 设计的开发板。65816 是一个有趣的芯片，因为它*是*一个 6502，直到你翻转寄存器中的一位。它有比 6502 更大的地址空间，来自 Commodore 世界的一切应该(相对)容易移植到 65816。为什么这种 CPU 从未用于 Commodore 硬件？因为一个西方设计中心的销售人员告诉一名 Commodore 工程师，苹果公司正在他们的下一台电脑(Apple IIgs)中使用它。准将使用 816 年的选择当场死亡。

如果你想在这台电脑上帮忙，[有一个脸书小组负责组织构建](https://www.facebook.com/groups/396408204443613/)。这个脸书群组是一个*封闭的*群组，也就是说你需要一个脸书账号才能登录。不幸的是，但我们期待着围绕这台梦想中的电脑的一年的更新。建造一台符合规格的计算机是不可能的，但是我们非常渴望看到社区的尝试。
