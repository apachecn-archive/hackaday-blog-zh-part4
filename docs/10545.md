# 游戏机颜色得到一个可充电电池

> 原文:[https://hack aday . com/2021/06/25/game-boy-color-gets-a-charged-battery/](https://hackaday.com/2021/06/25/game-boy-color-gets-a-rechargeable-battery/)

任天堂的经典游戏男孩长期以来一直是掌机界的宠儿。然而，随着许多粉丝将他们的手持设备改造成像现代背光液晶显示器这样的耗电设备，使用 AA 电池运行可能会成为一种令人沮丧的练习，因为它们会很快耗尽。[【深奥的肖恩】通过给他的游戏机安装 USB 充电电池来解决这个问题。](https://www.youtube.com/watch?v=tGUx15ZgTeQ)(视频，下面嵌入。)

黑客是一个简单的，但执行相当整洁。[深奥的肖恩]首先从 Game Boy 主板上移除原来的 DC 插孔，并挖出外壳上的洞来安装微型 USB 端口。原来的电池外壳也是类似的，以适应 2000 mAh 锂聚合物袋状电池。单电池充电板用于管理电池，其原始连接器被移除，并替换为一个看起来更整洁的面板安装微型 USB 端口。然后，电子产品被包裹在 Kapton 胶带中，并被塞在外壳中，因为所有东西都被放回了一起。

结果是一个 USB 充电游戏男孩持续年龄。[深奥的肖恩]报告说，连续一周每天玩几个小时的游戏机而不会没电。这种黑客技术可能会在芯片调谐器中流行起来，他们经常在电视机更热情的部分将 AA 电池从手持设备中敲出来。我们也看到了其他 Game Boy 型号的类似黑客攻击。
