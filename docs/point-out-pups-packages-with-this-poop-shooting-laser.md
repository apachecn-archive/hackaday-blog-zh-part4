# 用这种射便便的激光指出小狗的包裹

> 原文：<https://hackaday.com/2022/05/24/point-out-pups-packages-with-this-poop-shooting-laser/>

当你足够幸运在生活中拥有一只狗时，你往往会忽略关系中一些更片面的方面。当你被严格限制在清理垃圾的地方时，你的毛茸茸的朋友可以自由地在院子里漫步，随心所欲地分发他或她的鸡块，并完全期待你跟着一起打扫卫生。看到我们在那里做了什么吗？

所以养狗的人有时会反抗这种不平衡的权力结构，把清理工作留到以后——通常是很久很久以后，当定位令人讨厌的狗堆可能有点困难的时候。所以很自然地，我们现在有了这个发射便便的激光炮塔来帮助指导你清理后院。它来自于[Caleb Olson]，他利用[他最近的便便姿势监视器](https://hackaday.com/2022/01/17/ai-camera-knows-its-st/)作为数据来源，来确定每个粪便在院子里的确切位置。为了指出它们，他在一个廉价的机器人手臂上安装了一个激光笔，并使用 OpenCV 来帮助排列每个便便上的亮绿色斑点。

但是等等，还有更多。[Caleb]的代码还优化了它的便便巡逻路线，最大限度地减少了它为访问每一堆便便而必须走的讨厌的路。同样的姿势估计算法，看着可爱的[Twinkie]存款，跟踪哪些[Caleb]经过，依次从工作列表中删除每个。所以现在，他不是让一只狗来控制他的生活，而是让一只狗和一台电脑来控制他的生活。完美。

我们开玩笑，因为便便，但实际上，这是机器学习中一个非常整洁的练习。不过，看起来机器人手臂确实有点矫枉过正——我们本以为简单的双伺服转台会很容易制造出来。
