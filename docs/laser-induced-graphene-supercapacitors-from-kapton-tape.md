# Kapton 胶带激光诱导石墨烯超级电容器

> 原文：<https://hackaday.com/2020/11/11/laser-induced-graphene-supercapacitors-from-kapton-tape/>

从媒体的报道来看，石墨烯是一种神奇的材料，可以治愈世界上所有的疾病。它将使电池变得更好，给太阳能电池板充能，并革新医学。虽然碳单层的许多应用实际上已经出现在市场上，但在这种材料应用于所有领域之前还有很长的路要走，部分原因是石墨烯很难制造。

然而，这并不一定很难，正如[Zachary Tong]用[他的激光诱导石墨烯超级电容器](https://www.youtube.com/watch?v=RKcUgdXUf9Y)向我们展示的那样。他的生产方法再简单不过了，而且现在你很有可能已经拥有了在自己的店里复制这种方法所需的一切。它只需要一个 405 纳米的激光器，一台 3D 打印机或 CNC 路由器，以及一卷 Kapton 胶带。正如[扎克]解释的那样，激光能量将用作 Kapton 基础材料的聚酰亚胺薄膜转化为一种石墨烯泡沫。这种泡沫不具备单层石墨烯的所有常见属性，但它有自己有趣的属性，如极高的表面积和适中的电导率。

为了制作他的超级瓶盖，[扎克]把一些 Kapton 胶带粘在载玻片上，用激光蚀刻出图案。他的图案有紧密间隔的交叉电极，当被一种弱硫酸电解质覆盖时，显示出非常高的电容。他尝试了不同的模式和配置，包括将胶带层层堆叠，并设计出了一些相当大的电容器。作为一个附带项目，他用同样的方法生产了一种非常有效的 Kapton-tape 加热元件，这种加热元件可能有大量的应用。

这里希望[扎克]的快速简单的石墨烯方法能激发进一步的实验。首先，请查看我们对 Kapton 的深入研究，并了解并非每种神奇材料都能兑现承诺。
