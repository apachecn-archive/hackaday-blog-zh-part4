# 用 Arduino 制作你自己的 MIDI 控制器

> 原文：<https://hackaday.com/2019/04/06/make-your-own-midi-controller-with-an-arduino/>

工程师无中生有，这一点在定制计算机硬件的创造中表现得最为明显。做一个简单的 MIDI 控制器，你需要固件设计和计算机架构的知识，你需要机械设计的知识，你需要知道电子设计。然后你需要实际的工作知识和经验来使用工具，不管是锤子、激光切割机还是 IDE。[Mega Das]集合了所有这些技能来构建一个 MIDI 控制器。当然，这是为了扬声器发出的哔哔声和滴滴答答的声音，但是退一步，意识到任何一个人都可以想象的有多棒，然后实现这样一个设备。

这个建筑的电子设备包括一个印刷电路板，用于断开 Arduino nano 与十几个街机按钮、四个滑动锅、两个旋转锅和几个螺丝端子的连接，以便将所有东西连接在一起。从机械上来说，这是一个激光切割的盒子，上面刻有一些花哨的图形，大小完全可以放入所有东西。

是的，这些年来我们已经看到了许多围绕 Arduino 构建的 MIDI 控制器，但这一个是自成一类的。这是把现成的零件定制成你想要的样子，也是 DIY 硬件创造的一个很好的例子。你可以看看下面的视频。
