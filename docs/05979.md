# 乐高能破钢吗？

> 原文：<https://hackaday.com/2020/03/17/can-lego-break-steel/>

贝特里奇标题定律认为，任何以问号结尾的标题都可以用响亮的“不”来回答。但是正如下面的视频所示，[一个将钢铁扭曲成碎片的乐高机器](https://www.youtube.com/watch?v=jRn5waE0qfk)不仅是可能的，它还是金属疲劳的实物课程。Touché，[Betteridge]。

在让塑料与金属相抗衡的过程中,[Brick Experiment Channel]依赖于[早期的工作](https://www.youtube.com/watch?v=7YGp5zebQFE),一台机器能够从像软面条一样的零件生产线上扭曲一根普通塑料轴。目前作品中的钢制车轴是一个显然不再可用的配件，不会被证明是一个容易的目标。

即使在用额外的工艺支柱来加强测试台以承受张力，并且用额外的销来加强齿轮之后，单个马达也无法克服轮轴的强度。第二个马达和一个复杂的齿轮组开始使车轴变形，但最终证明塑料无法承受钢的重量。第二轮有点作弊:同样的装备有一个新的轴，但这一次电机旋转不断切换。累积的金属疲劳从一个小裂纹开始，一直发展到车轴扭曲成两半。

[砖块实验频道]是一个很有趣的频道，我们以前也做过专题报道。除了像这样的破坏性项目，他们还得到了有趣的构建，如[这个乐高扑克牌发射器](https://hackaday.com/2020/03/07/vanquish-your-foes-with-lego-playing-card-machine-gun/)、[一个技术无人机](https://hackaday.com/2020/01/15/lego-drone-finally-takes-off/)和[一个乐高潜水艇](https://hackaday.com/2019/07/28/lego-powered-sub-built-in-a-water-jug/)。
