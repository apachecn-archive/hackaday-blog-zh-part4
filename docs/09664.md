# 用价值数十亿美元的机器人在玉米地里作弊

> 原文：<https://hackaday.com/2021/04/04/cheat-at-cornhole-with-a-bazillion-dollar-robot/>

虽然对我们大多数人来说，户外野餐的日子可能还有几个月，但肯定有足够的时间为那一刻做准备。虽然有些人可能会在剩下的隔离时间里花时间完善 recipies 或做各种家庭改善项目，但其他人则在这些活动中的各种游戏中练习技能。具体来说，这个来自[戴夫军械库]的小组训练了一个机器人，它帮助玩完美的 cornhole 游戏。(视频，嵌在下面。)

虽然有问题的机器人是一个工业级的 KUKA KR-20 机器人，价格高达 32，000 美元，但该小组建造的软件和控制系统对大多数人来说相当容易获得。计算机视觉由 Nvidia Jetson 板处理，这是一台具有额外并行计算能力的单板计算机，运行 OpenCV。有了这种设置和一只用来拿玉米袋的定制手，以及大量的训练，该软件可以很容易地识别玉米洞板，并指导机器人玩一场完美的游戏。

虽然我们并不是所有人都有昂贵的工业机器人坐在我们的垃圾抽屉里，但 OpenCV 和可访问计算机的使用可能会使这个项目成为任何对计算机视觉感兴趣的人的有用介绍，该小组在他们的 GitHub 页面上公开了代码[。OpenCV 还可以用于机器人之外的许多其他事情，](https://github.com/davesarmoury/Cornhole)[比如识别田地里的杂草](https://hackaday.com/2021/01/31/opencv-and-depth-camera-spots-weeds/)或者[使用树莓 Pi 进行面部识别](https://hackaday.com/2013/03/04/using-opencv-with-the-raspberry-pi/)。
