# Tindie Seller 评论他自己产品的仿制品

> 原文:[https://hack aday . com/2019/11/13/tindie-seller-reviews-a-knock-off-of-your-own-product/](https://hackaday.com/2019/11/13/tindie-seller-reviews-a-knock-off-of-his-own-product/)

如果模仿是最真诚的奉承形式，那么如今网络创作者正以惊人的速度受到真诚的奉承。我们黑客抄写员总是看到这一点，因为我们的文章的直接复制粘贴出现在其他网站上不同的署名下。这很烦人，但是考虑到对知识产权的普遍态度，再为此烦恼就没什么意义了。但是如果被侵犯的是硬件呢？

Hacker 和 Tindie 商店老板 Brian Lough 最近在他的一个产品中遇到了这个问题，但是他没有沮丧，而是对这个山寨货做了一个非常公平和周到的评论。有问题的板子，[是一个 D1 迷你矩阵盾](https://www.tindie.com/products/brianlough/d1-mini-matrix-shield/)，使得在像[他的俄罗斯方块主题的 YouTube 子计数器](https://hackaday.com/2018/06/19/a-youtube-subscriber-counter-with-a-tetris-twist/)这样的项目中使用 LED 矩阵面板变得轻而易举。山寨版来自阿里快递，最“讨人喜欢”的是阿里快递清单上的文字和图片，其中一些直接来自[Brian]的 Tindie 商店。虽然板子的布局不同，但很明显它受到了原作的强烈启发。他们所做的改变——如终端选择和缩小一些痕迹——只会降低仿制品的质量。这肯定是一个削减成本的举动，所以他们可以降低原来的销售，对不对？显然不是——山寨货更贵。是的，[布赖恩]的董事会是一个套件和模仿者完全组装，但它仍然回避了为什么的问题？

向[Brian]脱帽致敬，因为他不仅制造了一个有用的产品，而且还花时间对其进行了适当的设计，并有雄心将其投入市场。很遗憾有人觉得有必要偷他的作品，但这似乎是这些天来的一个成人礼。
