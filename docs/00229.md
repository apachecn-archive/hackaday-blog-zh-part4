# 那东西有多少电流？

> 原文:[https://hack aday . com/2018/08/05/how-much-current-does-that-thing-draw/](https://hackaday.com/2018/08/05/how-much-current-does-that-thing-draw/)

如果你问我们如何测量某物的电流消耗，我们会弄断一根电源线，把一个万用表串联在电源上。如果这不方便的话，我们已经知道把保险丝从电源中拿出来，换成电表。粗糙，但有效。但是如果你手头有大约 8000 美元，你可以去买一个 Keithley 2460 SourceMeter。

什么是源表？嗯，据我们所知，这是一个具有非常精确的内置电流监控和微处理器的电源，可以显示许多有趣的统计数据和图表。平心而论，这看起来像是一个加强版的模型，但它们的起价只有一半，这仍然比大多数黑客的预算高得多。

你可能想知道为什么我们在谈论一个 8000 美元的仪器。嗯，很高兴看到在某个时候过剩市场上会出现什么。但更重要的是，这似乎是一个非常可行的黑客项目。有许多方法可以测量从分流电阻到霍尔效应器件的电流。我们当然知道如何制作带有漂亮触摸屏的酷嵌入式系统。或者你可以把数据传输回电脑，用你最喜欢的语言进行处理。

我们想不出任何能与 SourceMeter 竞争的“智能电源”项目。我们已经看到了对负载和[转换器](https://hackaday.com/2018/05/24/a-smarter-psu-converter-leaves-the-magic-smoke-inside/)的监控。我们也看到很多项目[用于监控 AC](https://hackaday.com/2010/07/07/whole-house-current-monitoring/) 。不过，在浏览了这些源表的视频后，我们想知道为什么我们不总是在我们正在构建的任何基准电源上安装数字监控接口。
