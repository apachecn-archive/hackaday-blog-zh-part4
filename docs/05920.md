# 一个时尚的咖啡桌，展示了康威的生活游戏

> 原文:[https://hacka day . com/2020/03/10/a-style-coffee-table-featured-con ways-game-of-life/](https://hackaday.com/2020/03/10/a-stylish-coffee-table-featuring-conways-game-of-life/)

咖啡桌是放置热饮的好地方，但除此之外，大多数咖啡桌都没有其他功能。对于大多数人来说，这是完全可以接受的，但是黑客们经常想要更多。[Andrei Erdei]就是这样一个人，他将康威的生活游戏融入到这件漂亮的家具中。

建筑从一张现有咖啡桌开始，选择它的玻璃桌面。不幸的是，顶部的一面被涂成黑色，完全不透明，无法用作 led 的扩散器。取而代之的是用烟熏玻璃代替，用磨砂窗膜使最后的部分更暗。这种对细节的关注使得[Andrei]能够将 WS2812B LED 灯带安装在下方，这样在关闭时就看不见了。led 本身由基于 ESP8266 的 WEMOS D1 mini 运行。这个微控制器有大量的咕噜声，这使得运行 led 和生命代码游戏变得轻而易举。

如果你想做一些有吸引力和闪光的东西，康威的《生活游戏》很难出错。你甚至可以用它让[成为时钟](https://hackaday.com/2014/01/11/extremely-slick-game-of-life-based-clock/)或者[成为合成器](https://hackaday.com/2011/03/27/music-synthesized-from-the-game-of-life/)；或者[在里面建一个完整的俄罗斯方块游戏](https://hackaday.com/2017/09/17/building-a-working-game-of-tetris-in-conways-game-of-life/)。休息后的视频。
