# 用乐高将圆周率照相机聚焦

> 原文:[https://hack aday . com/2020/07/22/用乐高将 pi 相机对焦/](https://hackaday.com/2020/07/22/bringing-the-pi-camera-into-focus-with-lego/)

自从几个月前树莓派的高品质相机发布以来，它一直是许多黑客关注的焦点。在这个快速构建中，[马丁·曼德]向我们展示了如何完全用乐高制作一个[伺服驱动的聚焦装置。](https://www.instructables.com/id/Focus-the-Pi-High-Quality-Camera-With-Lego-and-a-S/)

这个项目的灵感是在他制作 1979 年的 Merlin Pi 相机时产生的，他发现要获得高质量的照片，设置正确的焦距是至关重要的。因此，他为自己设定了一个目标，那就是建造一个能让他精确和远程聚焦相机的机械装置。

正是大量现成的乐高兼容部件使得这个项目在不使用任何 3D 打印组件的情况下成为可能。他不仅发现了一个乐高兼容的连续旋转伺服系统，还发现了一个乐高兼容的 Pi 外壳，以及一个几乎完全适合相机镜头的乐高齿轮。他还在外壳上加了一个三脚架，让他可以在任何地方安装相机。相机和对焦机制由基于 [guizero](https://lawsie.github.io/guizero/) Python 3 库的自定义 GUI 控制，相机可以通过 VNCViewer 远程访问。

如果你更喜欢 3D 打印而不是乐高，还有其他[时尚的 Raspi HQ 相机](https://hackaday.com/2020/07/21/a-stylish-raspberry-pi-camera/)构建。
