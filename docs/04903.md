# 在家铸造齿轮

> 原文:[https://hackaday.com/2019/11/21/casting-gears-at-home/](https://hackaday.com/2019/11/21/casting-gears-at-home/)

自动门和大门是很棒的，除了它们失败的时候，在我们的经验中，这似乎是大约每三天一次。[疯狂的轮子]就是这样一个失败，一个塑料齿轮是罪魁祸首。他们没有购买新的驱动装置，而是开始用金属铸造替代品。

该视频轻于说明，重于渐进式摇滚，对于初学者来说可能有点难以理解。这个过程开始于将原来的塑料部件粘在一起，并用环氧树脂油灰填充缝隙。然后将齿轮放入明胶/甘油混合物中制成模具。然后在这个模具中注入蜡，制作出原始零件的蜡模。蜡制齿轮配有圆柱形杆，作为熔融金属的流道，然后在蜡阳模周围制作石膏模具。制作两个石膏模型，放在烤箱里融化蜡。

目的是铸造一个铝制的替换零件。第一次尝试失败了，因为铝冷却得太快。这意味着像齿轮齿这样的细节不能正确铸造，产生了无用的金属斑点。在第二次尝试中，石膏模具首先被加热，这样可以保持足够的温度，让铝填充更精细的细节。完成后，移除转轮、清理轮齿和在车床上改进齿轮形状就成了简单的后期处理工作。

由此产生的零件很好地完成了工作，与传动系统中的其他齿轮正确啮合，并有效地移动闸门。许多人在评论中表示，最初的塑料齿轮可能是一种安全措施，在大门堵塞的情况下脱下。虽然这可能是真的，但在出现问题时，使用易碎的塑料钥匙，而不是破坏整个齿轮，这是一种更稳健的设计实践。

铸造是相当容易的专门家庭制造商。一旦你学会了基本原理，这是制作定制金属零件的好方法！休息后的视频。
