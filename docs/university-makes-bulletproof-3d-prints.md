# 大学制造防弹 3D 打印

> 原文：<https://hackaday.com/2019/11/20/university-makes-bulletproof-3d-prints/>

莱斯大学的研究人员正在研究模拟管状的 3D 打印塑料结构——理论上的纳米管结构预计具有非凡的强度。其结果是非常坚固和非常可压缩的结构，可以真正抵御子弹。

作为一项实验，研究人员以 5.8 公里/秒的速度向一块塑料和一块模拟管材发射弹丸。管状块的结构将子弹阻挡在第二层，第二层以外没有明显的结构损坏。参考块具有一个大洞和贯穿其体积的裂缝。

有趣的不是材料的强度，而是拓扑结构，因为如果需要更大或更小的东西，像这样的材料应该可以很好地扩展。正如你在下面的视频中看到的，网格也使得材料非常可压缩。

研究人员希望在航空航天、汽车、体育、包装、土木工程和生物医学应用等领域找到这种材料的用途。与此同时，你只能想知道如果你真的可以用碳纳米管制造这种结构会发生什么。

你可能会认为防弹材料是一个新想法，但事实证明根本不是。凯夫拉尔——现代防弹背心中常见的那种——能防护的远不止 T2 子弹。
