# 自制废车场安全机甲赋予叔叔超能力

> 原文：<https://hackaday.com/2022/01/03/homemade-scrapyard-security-mech-gives-uncle-super-powers/>

[Handy Geng]又回来了，带着另一个疯狂的建造，这是我们不能不考虑的。有一天，他的叔叔来参观车间，说他愿意每天都去，甚至可以在[Handy 耿]不在的时候照看车间。但是作为一个老伙计，需要一根拐杖才能走动，如果“坏人”决定去拜访他，他就没什么用了。显而易见的解决方案是建造一个[骑乘式安全机甲，叔叔可以骑在上面](https://www.youtube.com/watch?v=mrPcFiGEnoE)(视频，嵌入在下面)，用来保护商店免受强盗的袭击。

建造从他从卡车上卸下一对大履带轮开始，当我们试图想象他刚刚参观的废料场时，这引起了周围的窃笑！构建视频更像是点焊装配日志，省略了不太有趣的子装配构造。如果他包括了所有的细节，这个视频会有几个小时那么长。不过，我们可能还是会看。

最终结构的特征包括但不限于，用于现场转弯的双马达，夜间巡逻灯，用于攻击模式的双气动拳头，用于更温和接近的双水炮，以及用于终极威慑的后置扬声器。实用的触摸包括一个集成的眼镜盒为准备好的读者，和一个步行手杖持有人，所以械甲怪是叔叔准备好了。他脸上的笑容似乎给他留下了深刻的印象！
