# 黑客日播客 002:好奇的小工具和 FPGA 智囊团

> 原文:[https://hack aday . com/2019/01/18/hack aday-podcast-ep2-好奇-小工具-和-FPGA-大脑-信任/](https://hackaday.com/2019/01/18/hackaday-podcast-ep2-curious-gadgets-and-the-fpga-brain-trust/)

在本周的播客中，编辑埃利奥特·威廉姆斯和迈克·斯奇斯回顾了本周最受欢迎的黑客和文章。亮点包括谷仓门望远镜跟踪器的深度潜水，监听主电源，超级计算机发明者的背景故事，以及新喷气发动机设计的疯狂测试实践。我们讨论了一些我们最喜欢的电路雕塑，并查看了一台基于纺织品的新计算机和一台基于服务器的旧计算机。

本周，开源 FPGA 运动的名人圆桌会议讨论了 2019 年的下一步。大卫·沙阿、克利福德·沃尔夫、彼得·埃斯登-坦普斯基和蒂姆·安塞尔在《35C3》节目中采访了埃利奥特。
