# 灵活的印刷电路板黑客与奥什公园聊天

> 原文：<https://hackaday.com/2019/05/20/osh-park-hack-chat/>

太平洋时间周四中午加入我们，与来自奥什公园的[德鲁](https://hackaday.io/hacker/113761-drew-fustini)和[克里斯](https://hackaday.io/hacker/131216)一起参加[灵活 PCB 黑客聊天](https://hackaday.io/event/164402-osh-park-hack-chat)！
*请注意与我们通常的黑客聊天日程不同的一天！印刷电路板已经存在了几十年，它的大规模生产给电子工业带来了不可估量的好处。但是，扭转 PCB 生产的经济状况并使其为小规模生产商甚至家庭实验者所用是相对较新的发展，从长远来看，这可能会对该行业产生更广泛和更深刻的影响。*

现在，似乎价格荒谬的专业印刷电路板还不够，家庭游戏玩家现在可以获得灵活的印刷电路板。从可穿戴设备到传感器应用，柔性印刷电路板具有广泛的应用，并将为硬件黑客开辟新的领域。我们甚至在[柔性 PCB 竞赛](https://hackaday.io/contest/163267-flexible-pcb-concept-contest)中与 OSH Park 合作，特别是为了伸展您的柔性翅膀，让您的思维超越扁平、刚性的 PCB。

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)中的社区直播活动。本周，我们将在太平洋时间 5 月 23 日星期四中午 12:00 坐下来讨论。如果时区让你烦恼，我们有[一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20190515T17&p0=224&msg=IoT+and+Agriculture+Hack+Chat+with+Akiba&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周四；随时加入，你可以看到社区在谈论什么。
