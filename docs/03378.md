# 这款尖端树莓皮表壳上的格栅设计

> 原文：<https://hackaday.com/2019/06/21/grate-design-on-this-cutting-edge-raspberry-pi-case/>

爱他们或恨他们，你不得不承认苹果:他们真的知道如何用设计来触动人们的按钮。他们的工业设计师可以制造出如此不可抗拒的产品——他们的营销团队可以充分炮轰炒作列车——以至于人们会排队几天来购买新产品，并为这一特权支付巨额资金。

但是，如果你是一个贫穷的大学生，没有购买这些工业设计珍品的预算，该怎么办呢？很简单——你自己动手，把一个覆盆子馅饼塞进奶酪刨丝器。这是一群来自葡萄牙阿威罗大学的工程学学生做的，他们称之为“书呆子-AETTUA ”,显然是为了向世界上最昂贵的奶酪刨丝器[致敬。下面这个名为 RasPro 的视频没有苹果 Mac Pro 的宣传片那么流畅，但它仍然传达了基本信息。就像 Mac 上精心加工的拉丝铝外壳一样，RasPro 上的宜家奶酪刨丝器只是一层皮。它覆盖了一个 3D 打印的机箱，里面有一个结实的电源和风扇，可以搭配树莓 Pi 3。还有一个扬声器用于播放音乐，这似乎是 RasPro 的主要用途。](https://www.apple.com/mac-pro/)

考虑到所有的事情，奶酪刨丝器的设计对于 Pi 机箱来说并不是那么糟糕。不过，如果这不吸引人，你可以选择:[激光切割胶合板](https://hackaday.com/2014/08/15/pivena-the-open-source-raspberry-pi-case/)，一个 [Altoids 锡](https://hackaday.com/2017/04/23/this-hacker-fit-an-entire-retropie-in-an-altoids-tin/)，甚至是你电脑里的。
