# ESP32 吃豆人时钟会记录你看了多久

> 原文:[https://hack aday . com/2021/11/23/esp32-PAC-man-clock-keep-track-of-how-long-you-watch-it/](https://hackaday.com/2021/11/23/esp32-pac-man-clock-keeps-track-of-how-long-you-watch-it/)

有这么多的因素使得一个好的时钟值得看更长的时间。这只钟不仅看起来很酷，它还可以随时玩吃豆人游戏！是的，你当然可以与吃豆人互动——触摸屏幕的边缘会使他相应地向左、向右、向上或向下移动。你也可以换成吃豆小姐，让所有的动画正常速度，快速，或疯狂快速。

[TechKiwiGadgets]几年前造了一个吃豆人时钟，很受欢迎，但不便宜，也不容易模仿。从那时起，他们将代码移植到了 ESP32 上，并制作了一个组件更少、更友好的新版本。不仅如此，他们还提供了在 protoboard 上构建 ESP32 屏蔽的详细说明，并提供了作为开源可制造 PCB 的屏蔽。还是工作量太大？完整的套件版本可在 Tindie 上获得。请务必在休息后查看疯狂速度模式动作。

虽然这不是我们看到的第一个吃豆人时钟，但它对时间和游戏给予了同等的关注，[而这一个更多的是关于游戏本身的](https://hackaday.com/2015/02/05/pac-man-clock-eats-time-not-pellets/)。
