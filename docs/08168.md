# POV 领导的工作人员带着艺术旋转

> 原文:[https://hack aday . com/2020/10/25/POV-led-staff-take-art-for-a-spin/](https://hackaday.com/2020/10/25/pov-led-staff-takes-art-for-a-spin/)

人体有很多很酷的把戏，但最容易利用的是视觉暂留(POV)。在光线消失后，我们的眼睛会在几分之一秒内继续看到光线，我们可以利用这一点来制作有趣的闪光灯玩具，如 POV 五线谱。当然，你可以购买 POV 人员和其他设备，但它们相当昂贵，你不会学到任何东西。自己构建东西通常是更昂贵的路线，但有了[【shurik 179】优秀的开源 POV 人员](https://www.instructables.com/Persistence-of-Vision-LED-Staff/)就不是这样了。

从详细的说明开始，这个项目有很多令人喜欢的地方。它基于 ItsyBitsyM4 Express 和 Adafruit 的 Dotstar LED 灯条。你可以使用蓝牙版本，但它已经很容易加载图像到工作人员，因为它显示为一个 USB 大容量存储设备。我们喜欢[shurik179]增加了一个 IMU，并对标尺进行了编码，这样无论标尺旋转得多快，图像看起来都是一致的。未来，[shurik179]可能会推出可折叠的蓝牙版本。这听起来像是一个壮举，我们迫不及待地想看到它的行动。

尽管向 POV 员工挥手致意很酷，但并没有真正的实际应用。有什么比时钟更实用？
