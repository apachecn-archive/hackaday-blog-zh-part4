# 自制玩具风洞吹(真的很好)

> 原文：<https://hackaday.com/2022/02/18/homemade-toy-wind-tunnel-blows-really-well/>

有时候，一个孩子在圣诞节早上醒来，跑下楼，只希望看到一样东西:一个闪亮的新风洞。去年 12 月，这正是[SparksAndCode]的儿子在树旁的下找到的~~，还带了一袋围巾、乒乓球和其他好玩的东西(当然是以物理学的名义)。~~

真实的故事开始于圣诞节前一周，当时[SparksAndCode]的儿子在一家科学博物馆被一个类似的设备迷住了。在妻子的建议下，[【SparksAndCode】开始用五金店的零件设计和建造一个风洞](https://www.youtube.com/watch?v=_BjSQnM58p0)，他的最后期限即将到来。隧道的基本结构是三根支撑胶合板模板的杆。这些壁是由塑料片卷在套环内形成的管子。下面是一个港口货运风扇，为无尽的娱乐提供稳定的气流。

在圣诞节早上他儿子的首次测试中发现了一些错误后，[SparksAndCode]将风洞带回商店进行了一些调整和升级，包括在进气口上安装一个网状盖子，以防止东西被吸入风扇。最后的结果是一个非常实用(而且有趣！)空气柱。寻找更多的功能(但不一定不好玩)？几年前他自制的研究风洞也为你带来了报道。
