# 竖锯电机使用 PCB 线圈产生径向磁通

> 原文:[https://hack aday . com/2019/07/16/jigsaw-motor-uses-PCB-coils-for-radial-flux/](https://hackaday.com/2019/07/16/jigsaw-motor-uses-pcb-coils-for-radial-flux/)

电动机很容易制造；还记得那些用线绕钉子做的实验吗？但是容易制造的东西往往很难加工，制造一个小、轻、功率大的马达可能很困难。然而[Carl Bugeja]并不是一个在挑战面前退缩的人，他的微型“拼图”PCB 马达是他马达制造实验的最新成果。

我们习惯于看到[Carl]的无刷 PCB 电机，但主要是轴向磁通电机，其中定子线圈的排列使其磁力线平行于电机轴—[他的微型 PCB 电机](https://hackaday.com/2018/01/24/this-tiny-motor-is-built-into-a-pcb/)是这种几何形状的一个很好的例子。虽然这些可以完全打印出来，但它们远非最佳。因此，[卡尔]开始寻找制造径向磁通 PCB 电机的方法。他的设计有六个六层 PCB 线圈，垂直焊接在六边形端板上。端板有迹线以星形配置连接线圈，与匹配的顶板一起为微型轴承提供支撑。与此同时，转子是一个 3D 打印的立方体，带有压配合的钕磁铁。看看下面的视频。

连接到一个电子稳定控制系统，电机工作正常，但不壮观。[卡尔]承认更多的调整是有序的，我们毫不怀疑他会继续优化设计。我们喜欢它的外观，我们渴望看到它的改进。
