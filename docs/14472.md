# 一个漂亮的 DIY 导向锯

> 原文：<https://hackaday.com/2022/08/30/a-beautiful-diy-guided-saw/>

[JSK-库布]很有创作木制品的天赋，特别是建造有趣的工具或改进现有的工具。这个最近的项目(下面有多个构建视频:[第 1 部分](https://www.youtube.com/watch?v=6NAHL_AJJ6E)、[第 2 部分](https://www.youtube.com/watch?v=jbUACGNowPA)和[第 3 部分](https://www.youtube.com/watch?v=hQP0vHSPegw))是一种非常灵活的导向拉锯，其中细齿锯片固定在床身上，工件在上面被拉动。通过将其固定在一个小角度，并使刀片能够通过床身上升，只需将工件拉过刀片，然后稍微卷起，直到达到最终切割长度，就可以逐步切割工件。

从构造角度来看，该工具是全金属的，由现成的零件和厚的手工切割铝合金板组成，甚至没有数控工具。唯一不寻常的部件是锯条本身，如果你想复制这个构造，要找到它可能有点困难。它看起来(如果你相信谷歌镜头的自动翻译，无论如何)是至少在日本可以买到的商业导向锯的备用刀片。

我们发现它特别令人高兴地看到使用自制的阳极氧化设置，给它一点爵士色彩的一些板，只是因为。就像这个 YouTuber 上的许多构建视频一样，如果只是为了收集一些关于自己工作空间的想法，如果你能阻止自己分心去看他在路上使用的所有其他整洁的工具，那么所有这些视频都非常值得观看。

DIY 工具构建在这些地方并不罕见，这里有一个整洁的[基于 DIY 组合表的项目](https://hackaday.com/2020/04/29/a-custom-saw-designed-for-close-quarters-making/)，这里有一个自制的[自动降低金属带锯](https://hackaday.com/2021/05/07/homemade-metal-band-saw/)，它可以真正节省时间。
