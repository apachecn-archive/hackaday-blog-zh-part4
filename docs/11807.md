# Focus Flower 通过往脸上喷水来激发灵感

> 原文：<https://hackaday.com/2021/11/04/focus-flower-motivates-by-squirting-water-in-your-face/>

当你需要完成一些任务，注意力不集中时，很难打败计时器。但无论何时，你都会感觉非常好。问题是当你没有及时完成的时候，惩罚是什么？除了良好的罪恶感之外，番茄工作法没有其他结果。如果有一点负面强化的话不是更好吗？

[【硬件未知】的焦点花](https://www.youtube.com/watch?v=W6ikFC172cY)从来不需要浇水，至少不是以正常的园艺方式。你将不得不装满一个蓄水池，因为这朵花提供液体动力。不，这不是烈酒，虽然我们认为当水的新鲜感消失后，你可以把它变成一个前卫的伏特加喷泉，让这款番茄鸡尾酒变得更像血腥玛丽。这是一个自然的下一步，特别是如果你已经进入辣酱的想法。

操作 Focus Flower 很简单:只需按下 easy 按钮启动任务定时器，附带的 Arduino Nano 就会开始倒计时。当你完成后，再按一下按钮，但如果你在倒计时结束前没有这样做，植物会从螺线管驱动的牙线器尖端向你喷射稳定的皮肤喷射水流。附近需要一个空气压缩机，这有点打破了极简主义的氛围，但是你可以把它藏在你的桌子下面。

“焦点之花”看起来确实对消极强化很有效。它不会让你完全一无所知——底座周围有一圈发光二极管，显示还剩多少时间。每当你成功地及时按下按钮，它就会用 12 种语言中的一种说“这很容易”,因此有了所有的标志。不要错过下面完全免费的电视广告。
