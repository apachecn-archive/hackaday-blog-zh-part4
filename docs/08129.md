# 如果你需要的话，光学中心打孔是一个简单的工具

> 原文：<https://hackaday.com/2020/10/24/optical-centrepunch-is-an-easy-build-if-you-need-one/>

厌倦了在那里挨他的重拳，但还没到那里，[Uri]决定必须做些什么。解决这个问题的一个常用工具是光学中心打孔机，但是出售的模型对于如此基本的东西来说有点太贵了。取而代之的是，[【乌】当选为他自己建造的。](https://www.youtube.com/watch?v=JtLvEjR0M1w)

光学中心打孔机是一种简单的工具，可以帮助机械师一次又一次地击中目标。使用一根导杆，导杆上有一根透明的塑料杆，杆的中心有一个点。这个点在要中心打孔的点上排成一行。然后将塑料杆取下，换上真正起作用的冲头。不满足于建造一些实用的东西，[Uri]而是把这个工具雕刻成一个类似佩珀中士(黄色潜水艇的名字)的东西。看到这块裸露的黄铜在镜头中迅速成为一个可识别的人物，证明了(尤里)作为一名雕塑家的技能。

这是一种任何人都可以用车床，或者至少是一台像样的钻床来制造的工具。我们认为这对那些视力不佳的人来说特别有用，可以让他们更容易有规律地击中目标。[Uri]以前也给这些页面增色不少——[他以前制作了一个华丽的工具，让所有其他的锤子嫉妒不已](https://hackaday.com/2019/03/05/making-a-hammer-with-beautiful-engravings/)。休息后的视频。
