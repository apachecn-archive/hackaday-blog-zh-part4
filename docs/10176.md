# 使用手钻部件的全轮驱动自行车

> 原文：<https://hackaday.com/2021/05/26/all-wheel-drive-bicycle-using-hand-drill-parts/>

一个熟练的山地自行车手可以穿越一些极端的地形，但[The Q]认为可能还有改进的空间，所以他[将一辆肥胖的自行车改装成全轮驱动](https://www.youtube.com/watch?v=MJpqJFE5L-g)。

这里的主要挑战是将踏板动力传递到前轮，尤其是耳机周围。[The Q]通过从一个非常古老的手钻的零件上有效地构建一个差速器解决了这个问题。由于前轮需要以与后轮相同的速度旋转，一条长链从后轮到耳机，由一对拨链器拉紧。这个前链轮转动一系列围绕耳机排列的直齿轮和伞齿轮，耳机通过另一个链条将动力向下传递到前轮。

感受一下自行车在松软的沙地、泥地和岩石上骑行的感觉会很有趣。我们可以看到它在这些条件下有一些优势，但不确定它是否足以抵消重量和复杂性方面的损失。这些额外的链条和齿轮看起来确实像是在抓树叶、衣服，甚至可能是皮肤。然而，我们怀疑[Q]这么做更有可能是为了挑战构建，我们当然可以理解。随着电动自行车的兴起，给前轮增加一个轮毂电机似乎是一个更简单的选择。

这些年来，我们已经看到了几个有趣的自行车黑客，包括一个 [strandbeest 后端](https://hackaday.com/2016/06/30/a-strandbeest-bicycle/)、 [3D 打印轮胎](https://hackaday.com/2018/05/07/3d-printed-bicycle-tire-not-full-of-hot-air/)和一个[自动换挡器](https://hackaday.com/2020/09/04/automatic-arduino-bicycle-shifter/)。
