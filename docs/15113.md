# Barilla 的完美意大利面开源工具

> 原文:[https://hack aday . com/2022/10/17/barillas-open-source-tool-for-perfect-pasta/](https://hackaday.com/2022/10/17/barillas-open-source-tool-for-perfect-pasta/)

煮意大利面可能是你在厨房里能做的最简单的事情之一，仅次于看着一壶水沸腾。但是正如意大利面制造商 [Barilla 在他们的网站上指出的那样，](https://www.barilla.com/en-gb/passive-cooking)如果你只是让意大利面在热水中浸泡，而不是一直主动煮沸，你就可以减少 80%的 CO₂排放量——这种技术被称为被动烹饪。

[![](../Images/9a8cb8e8c5df5cb35ac848c7fec681f4.png)](https://hackaday.com/wp-content/uploads/2022/10/barilla_detail.png) 诀窍在于掌握正确的时机，所以在一个相当令人惊讶的举动中，Barilla 发布了一个开源设备的设计，它将帮助你掌握这种节能技术。诚然，它不是一个非常复杂的硬件，只不过是由一个 Arduino Nano 33 BLE、一个 NTC 探头和一些包装在 3D 打印外壳中的无源组件组成。但是文档是伟大的，我们不得不承认 Barilla 在这一点上走出了他们的舒适区。

3D 打印外壳中的磁铁可以让它粘在你的锅盖上，当它检测到水沸腾时，这个小工具会提醒你的手机(至少对于这个版本的设备，需要一个 Android 或 iOS 应用程序)是时候放入意大利面了。几分钟后，它会告诉你什么时候可以关掉炉子，之后就只需要等待被动烹饪的意大利面可以拿出来的通知了。
