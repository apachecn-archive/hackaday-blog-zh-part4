# 一些经典街机的新鲜空气

> 原文：<https://hackaday.com/2022/06/07/a-breath-of-fresh-air-for-some-arcade-classics/>

据说好的东西来自小的包，当我们看到所有构建在 Altoids tin 中的漂亮项目时，这是很难否认的。现在，如果常规尺寸的盒子已经是这样了，我们可以为塞进他们的*小号*各种各样的盒子里的任何东西而加倍兴奋，这就是【凯登·科赫】决定用作他的 [mintyPico 的外壳，这是一个运行自制版本的 Snake，Breakout，Pong 和其他一些](https://www.youtube.com/watch?v=b9Obk1vu7JE)的小型游戏控制台。

正如“Pico”可能已经泄露的那样，该项目是围绕 Raspberry Pi Pico 板构建的，旨在作为便携式设备，[Kayden]推出了一个版本，其中还包含 LiPo 电池充电电路。一组 3d 打印部件将电路板、匹配的电池和按钮面板整齐地包装在锡本身中，而大小合适的 SSH1106 有机发光二极管则放入盖子中。所有设计文件以及游戏的 MicroPython 代码都可以在该项目的 GitHub 页面上找到[。](https://github.com/kkehe/PicoAltoids)

当你读到这个项目的名字时，你可能会有这种奇怪的熟悉感，事实上，[mintyPi 游戏机](https://hackaday.com/2017/04/23/this-hacker-fit-an-entire-retropie-in-an-altoids-tin/)是这里【Kayden】的主要灵感，正如[的 Pico Snake 项目](https://hackaday.com/2021/05/11/play-your-favorite-nokia-game-on-the-raspberry-pi-pico/)一样。考虑到这是他初中时的项目，这无疑是这两个项目令人印象深刻的完美结合。
