# 忍者艺术:PCB 夜光 Jutsu！

> [https://hacked ay . com/2021/12/10/忍者艺术-pcb-nightlight-jutsu/](https://hackaday.com/2021/12/10/ninja-art-pcb-nightlight-jutsu/)

这是由[Arnov]提供的最新 PCB 作品。他的[火影忍者小夜灯](https://www.hackster.io/Arnov_Sharma_makes/naruto-themed-night-lamp-a0f8e2)绝对会引起你的动漫爱好者朋友的注意。

LED 照明方式由 ATtiny13A 微控制器控制。他可能会选择 555 定时器，但也许他想要容易编程的闪烁模式。他还将 ATtiny 编程为读取一个小按钮，用于在不同的照明风格之间循环。最后，一个小的 LiPo 电池使这个项目非常便携，所以你可以在你的工作区域自由地重新放置它。

综上所述，这个项目的核心是 PCB 的物理尺寸设计。[Arnov]能够设计出火影忍者头部形状的电路板，他的头发、眼睛和头带都有很好的细节。如果你曾经尝试过自己的 PCB 艺术，[你会知道这可能是一项相当繁重的任务](https://hackaday.com/2017/08/30/how-to-do-pcb-art-in-eagle/)。他创造性地使用了 PCB 中的铜迹作为特征，在这种情况下，鸣人的忍者头带。我们认为将 led 放在 PCB 背面的微妙决定也是明智的。通过这样做，他将阻焊膜用作自然光的消除器，这真的给 PCB 带来了凉爽的黄色光芒。小心地去除铜层并且不使用铜浇注确实有助于美观。因为鸣人的头发是黄色的，所以他选择黄色的阻焊膜也很聪明。

总而言之，竖起两个大拇指【Arnov】。当你在这里的时候，[看看 Hackaday](https://hackaday.com/2020/05/24/flexible-pcb-earrings-put-the-art-in-art-deco/) 周围其他一些伟大的 PCB 艺术。
