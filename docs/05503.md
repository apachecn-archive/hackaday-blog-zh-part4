# 用火焰灯泡假装壁炉闪烁

> 原文:[https://hack aday . com/2020/02/10/fake-that-fire-flicker-with-flame-bulbes/](https://hackaday.com/2020/02/10/fake-that-fireplace-flicker-with-flame-bulbs/)

问问任何一个在寒冷的冬夜收看壁炉电视的人——尽管你感觉不到热量，也不能用它来烤棉花糖，但假火几乎和真火一样令人心旷神怡。如果你有孩子或宠物，那就安全多了。但是，当你可以自己做一个发光的插件时，为什么还要花钱去买呢？

你甚至不需要对微控制器和 RGB LEDs 产生兴趣——就像[火腿制造]做的那样，拆除一些 LED 火焰灯泡。它们已经拥有您需要的一切，而柔性 PCB 使它们易于使用。

[![](../Images/e6ff1194dd071818c3bb15df8df7d525.png)](https://hackaday.com/wp-content/uploads/2020/01/LED-fireplace-inset.png) 【火腿制造】用双面胶把相当于三个灯泡的东西粘在一块泡沫板上，把所有的引线焊接在一起，接上拨动开关和 2xAA 电池组。每个灯泡都有一个倾斜开关，这样无论方向如何，“火焰”都会向上流动，但[火腿制造]移除了这些开关，以避免闪烁的连接性，并与拨动开关斗争。

一旦它全部连接起来，[火腿制造]将一些磁铁热粘在泡沫板上，并将其附着在壁炉的底部，以防止它受到原木和灰坑的影响，同时仍然允许辉光从正确的位置发出，以获得真实感。唯一缺少的是爆裂声和爆裂声，而且[火腿制造]渴望听到你的实现想法。

[火腿制造]没有以传统方式使用他的壁炉，因为房子很小，而且是中央供暖的。但是如果你依靠它来保持温暖和舒适，为什么不把它做成声控的呢？
