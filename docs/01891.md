# 机器人无法将视线从瓶子上移开

> 原文:[https://hack aday . com/2019/01/25/robot-cant-take-its-eyes-off-the-bottle/](https://hackaday.com/2019/01/25/robot-cant-take-its-eyes-off-the-bottle/)

按照我们目前的理解，机器人倾向于依靠电力运行。只有在《未来》的幻想世界里，机器人才会寻求酒精作为燃料和娱乐的来源。[直到【莱斯·赖特】和他的啤酒寻找机器人出现。](https://www.youtube.com/watch?v=Y5V331kbSvY&list=PLB2Z43zIP_3a8-eNjE5i-EajiCa-bXxgM&index=1) (YouTube，休息后的视频。)

Raspberry Pi 3 为大脑提供了一个英特尔神经计算棒，作为神经网络任务的加速器。该硬件与 OpenCV 图像检测软件相结合，使被跟踪的机器人能够识别物体并相应地跟踪它们的位置。

选择一个啤酒瓶仅仅是一个有趣的插曲——该软件可以很容易地识别许多不同的物体类别。[Les]还实现了搜索功能，机器人将扫描房间，直到识别出目标瓶子。[GitHub 上提供了所需的软件和脚本供您阅读](https://github.com/leswright1977/RPi3_NCS2)。

在过去的几年里，我们已经看到了用于深度学习和神经网络计算的加速器硬件的爆炸式增长。当然，这对于无法链接到云服务的机器人应用来说尤其有用。我们期待看到这一领域的进一步发展——特别是一旦机器人能够打开冰箱，识别啤酒，并一举将其送到沙发上。未来会很辉煌！
