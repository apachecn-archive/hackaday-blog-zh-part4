# 黑客日播客 091:路易斯维尔爆炸器，产生日本细木工，继电器反电脑集会，砍机器人

> 原文：<https://hackaday.com/2020/10/30/hackaday-podcast-091-louisville-exploder-generating-japanese-joinery-relay-retrocomputer-rally-and-chop-the-robopup/>

Hackaday 的编辑迈克·斯奇斯和埃利奥特·威廉姆斯挖掘了本周不应错过的最伟大的黑客。有一个疯狂的人，用一个充满爆发力的球棒，而不是肌肉，来打破本垒打距离记录。这些工程印章的更优雅的使用显示在一个 CNC 软件工具中，它产生复杂的木材细木工，而不需要过于花哨的机器来制作它。如果你的血肉宠物跟不上你的兴趣，有一种新的机器狗出现了，它远远胜过它的组成部分，这些组成部分是 3D 打印的，有 Pi 和 Arduino 品种。就在你认为你已经看到了所有最疯狂的逆向计算机时，这里有一个基于机电继电器的机器，它花了六年时间来建造(尽管这里发生了这么多事情，它应该花了十六年)。
