# 吸血鬼充电器是一个坚固的任何东西到 5v 直流转换器

> 原文：<https://hackaday.com/2018/07/29/vampire-charger-is-a-rugged-anything-to-5vdc-converter/>

提供 5 VDC 的 USB 插座作为电源无处不在，几乎任何现代便携式技术都可以使用它们来运行或充电。事实上，USB 电源是如此常见，以至于人们很容易认为这是理所当然的。但在紧急情况下或灾难发生后，一部工作正常的手机或 GPS 可以拯救生命，不要指望干净、可靠的 USB 电源是明智的。

这就是由 Matteo Borri 和 Lisa Rein 设计的吸血鬼充电器 T1 出现的原因。它是一个硬件，专注于将人们可能接触到的任何电源或电源转换为可靠的 5v 直流电源，用于任何可以通过 USB 插入的设备。这不仅仅是一个具有宽输入范围的 DC-DC 转换器；当他们说它可以接受任何东西作为输入时，他们是认真的。找到了工作电源但不知道是什么电压？不知道哪根线是正的，哪根线是负的？都不知道是 AC 还是 DC？只要挂上鳄鱼夹，让吸血鬼充电器自己想办法；当灯是绿色的时候，电源是干净的。

吸血鬼充电器最近[被选中进入 Hackaday 奖的最后一轮](https://hackaday.com/2018/07/24/twenty-power-harvesting-projects-headed-to-the-hackaday-prize-finals/)，在这个过程中净赚 1000 美元现金。下一个挑战(将有另外 20 名决赛选手每人获得 1000 美元)是[人机界面](https://hackaday.io/prize/details#four)挑战。你需要输入的只是一个想法和一些文档，所以把那个一直在等待机会的项目掸掉灰尘，因为它就在这里。
