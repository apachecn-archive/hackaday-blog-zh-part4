# 电镀 3D 打印零件，强度极高

> 原文：<https://hackaday.com/2021/03/01/electroplating-3d-printed-parts-for-great-strength/>

树脂 3D 打印机与细丝打印机相比具有显著的优势，因为它们能够打印更小的零件，具有更精细的细节。主要的缺点是树脂部件通常不如灯丝部件坚固耐用。由于这个原因，它们通常更多地用于小型模型，而不是工作部件，但[Breaking Taps]希望通过电镀为它们添加金属来提高这些构件的强度。

铜和镍涂层都用于这些测试设置，每一种对树脂印刷都有不同的效果。镍大大增加了硬度，铜似乎增加了树脂部件可以承受的应变——尽管[Breaking Taps]讨论了这一结果的一些问题。

虽然电镀树脂的结果令人鼓舞，但他指出这是一个繁琐的过程。这是一个多步骤的考验，用一种特殊的油漆涂在树脂上，帮助金属附着，然后电镀。与他在这个视频中使用的简单样品相比，在更复杂的印刷品上也很难确保均匀的金属涂层。

然而，在说了和做了所有事情之后，如果一个工作部件需要比细丝打印机能够生产的更小，或者需要更精细的细节，这是一个给这些部件增加更多强度或刚度的非常方便的方法。然而，仍然有一些调查要做，因为用他的装置很难测试电镀丝印刷，但它确实显示出了希望。也许有一天，我们将能够直接使用金属而不是用它覆盖塑料来打印出如此高的精度[。](https://hackaday.com/2021/01/21/getting-closer-to-metal-3d-printing/)
