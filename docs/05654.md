# 从头开始做一套耳机

> 原文：<https://hackaday.com/2020/02/16/make-a-set-of-headphones-from-scratch/>

欣赏音频有多种方式，耳机就是其中之一。制作一套耳机是一个足够简单的项目，但[madaeon]通过从零开始构建耳机驱动器而不是使用现成的耳机，将这项技术提升到了一个新的水平。

结果是一组动圈驱动器，其构造技术包括使用外壳的半透明薄窗口作为隔膜和线圈的线圈架。氰基丙烯酸酯粘合剂将一切固定到位。振膜悬挂在一个纸板管的开口处，线圈位于磁铁上方，从而使移动质量最小，以获得尽可能好的声音再现效果。自己判断吧，我们在休息时间下面放了一个视频。

这些驱动器被放置在一套 3D 打印的耳机支架中，虽然它们可能不会与一套昂贵的商业耳机相匹配，但我们猜测它们的质量不会太差。至少，这是一个有趣的设计，可以作为进一步实验的基础。

令人惊讶的是，很少有国产扬声器或耳机驱动程序出现在这些页面上，这可能是因为现成的文章无处不在。一个例外是[这个柔性 PCB 扬声器](https://hackaday.com/2018/09/21/the-diaphragm-is-the-coil-in-these-flexible-pcb-speakers/)，当然我们也谈到了[自制的静电扬声器](https://hackaday.com/2016/08/03/electrostatic-loudspeakers-high-end-hifi-you-can-build-yourself/)。
