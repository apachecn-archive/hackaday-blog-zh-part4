# 猫不是一般的喵星人

> 原文:[https://hack aday . com/2022/04/09/cat-is-not-your-average-meow-cropad/](https://hackaday.com/2022/04/09/cat-is-not-your-average-meow-cropad/)

你是否已经完全放弃了任何平面形式的键盘，而在寻找一种完全不同的输入数据的方式？或者你想要一个 mega macropad 来进行 3D 设计、GIMP 或 Inkscape 工作，或者在与一个不错的第一人称射击游戏一起放松时使用？那么[这款名为 CAT](https://www.reddit.com/r/ErgoMechKeyboards/comments/twsgwr/new_way_of_keyboarding/) 的符合人体工程学的双拳键盘/控制器混搭可能就是你正在寻找的。

在这些苗条的猫科动物中，几乎都有你期望的东西——25 个左右的开关和一个 Arduino Pro Micro。有趣的是，这些开关都是杠杆作用的，而不是按钮。有两个品种的猫可以建造或购买:一个有 25 个按钮，另一个在拇指上的两个上部和两个下部按钮之间有一个操纵杆或轨迹球。你可以每只手一种类型！

Lynx Workshop 网站提供了更多信息，在那里您还可以找到从 3D 打印到电子设备到组装和编码的所有教程和说明。甚至还有额外的 3D 建模教程。不想投资时间来制作自己的猫？这些小猫也可用于[预购](https://www.lynx-workshop.com/preorder)。爪过去休息，检查他们的行动。

在找带常规按键开关的东西吗？哦，我们有很多这样的东西。
