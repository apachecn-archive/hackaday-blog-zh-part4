# 乐高爬杆者是垂直向上攀爬的绝佳研究对象

> 原文：<https://hackaday.com/2021/07/23/lego-pole-climbers-are-great-study-in-what-it-takes-to-go-vertically-upwards/>

用机器人爬电线杆听起来可能很复杂和困难，但事实并非如此。[这段来自[Brick Experiment Channel]的视频从机械的角度展示了在保持简单的同时完成工作的多种方法。](https://www.youtube.com/watch?v=MzmbooQJ9ko)(视频，下面嵌入。)

第一种方法使用重力锁定设计，其中电池组的重量放在杠杆臂上，以增加车轮抓紧杆的法向力。增加杠杆臂的长度，减小履带的角度，或者增加更防滑的轮胎都可以用来增加这种设计的抓地力。这种类型的最终设计能够毫不费力地爬上 8 米长的旗杆。

下一个版本使用橡皮筋来增加抓杆的张力。这也很好地工作，并取得了旗杆顶端。最后的构建是一个循环的设计，看起来真的很狂野，并且蜿蜒到旗杆的顶端。

很高兴看到设计这些乐高作品的实验方法，以及它们在野外的表现。[【砖头实验频道】之前也在这里精选过](https://hackaday.com/2021/01/23/obstacle-climbing-rover-built-with-the-power-of-lego/)。
