# 满足您黑客需求的公文包计算机

> 原文：<https://hackaday.com/2019/05/31/a-briefcase-computer-for-your-hacking-needs/>

体面的笔记本电脑已经存在几十年了。尽管如此，藏在公文包里的电脑还是有一些迷人和神秘的地方。[MakeFailRepeat]想要的就是这样一个钻机，[于是制造者做了一个。](https://www.instructables.com/id/Briefcase-Computer-With-Raspberry-Pi/)

该项目始于[MakeFailRepeat]收到的一台 15 英寸的 12V 显示器。自然地，它需要被用在一个令人敬畏的项目中，并且构建开始了。MDF 面板被切割以将屏幕安装在铝制公文包内，并覆盖黑色毛毡以获得令人愉悦的外观和感觉。Logitech 无线键盘和触摸板组合用于输入。该操作的大脑是一个树莓 Pi，配备了一个 UPS 帽子来处理电池和电源，以及一个 Adafruit 扬声器帽子来发出声音。

这个项目的灵感来自于 1990 年在 Amiga、Atari 和 MS-DOS 平台上发布的经典视频游戏 Captive T1。虽然我们很确定[MakeFailRepeat]没有被困在空间站，但他的公文包电脑应该还是有用的。[虽然](https://hackaday.com/2013/09/09/graffiti-briefcase-for-stealth-tagging/)，电脑并不是你唯一可以放进公文包的东西。休息后的视频。
