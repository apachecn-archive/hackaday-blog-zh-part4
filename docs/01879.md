# 泻盐修复铅酸蓄电池

> 原文：<https://hackaday.com/2019/01/20/epsom-salts-restores-lead-acid-battery/>

尽管电池技术取得了许多进步，但由于成本和提供大量浪涌电流的能力，铅酸电池仍被用于许多应用中。但它们不会永远持续下去。然而，[AvE]表明，在某些情况下，失效的电池可以用——尤其是——泻盐修复。如果当爷爷背痛时，用他泡的东西让你觉得好笑，你可以叫它硫酸镁。

你可以在下面的视频中找到完整的解释(其中包括[AvE]非常丰富多彩的语言)，但从根本上说，硫酸镁溶解了电池极板上堆积的硫酸铅。这种修复通常是暂时的，因为这种累积会与其他故障机制一起发生，如极板材料脱落和聚集在电池底部。显然，泻盐不能修复受损的盘子或进行任何其他神奇的治疗。

我们真的很喜欢[AvE]撕开电池来展示电池板和里面到底发生了什么。他还解释了为什么泻盐会有帮助。

我们很惊讶他把盐直接倒进了电池里。我们总是被教导加热一些蒸馏水，用泻盐浸透它。然后你会过滤掉任何剩下的固体，倒入水/盐混合物，直到它不能再多了。

这不仅不总是有效，而且也不是立竿见影。我们听说过用泻盐或苛性钠处理过的电池在几个星期后会复活。然而，即使你不想用盐来恢复电池，视频中也有很多有趣的电池事实和知识，你会觉得很有趣。

每个人都喜欢指出几乎任何项目都可以使用 555。那个芯片[可以在你修好后给你的电池](https://hackaday.com/2013/02/14/555-timer-charges-lead-acid-batteries/)充电。顺便说一下，这不是第一次[我们考虑给电池](https://hackaday.com/2017/08/25/blast-your-batterys-sulphates-is-it-worth-it/)加盐。另一方面，你可以用熔盐制作[这种奇特的电池。](https://hackaday.com/2015/08/03/building-a-battery-from-molten-salt/)
