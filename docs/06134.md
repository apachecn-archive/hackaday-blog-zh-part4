# 肉馅卷饼变成了不切实际的美味合成器

> 原文:[https://hack aday . com/2020/04/01/empanada-becoms-unactual-delicious-synth/](https://hackaday.com/2020/04/01/empanada-becomes-impractical-delicious-synth/)

Vegemite 是澳大利亚的主食——一种富含蛋白质的三明治，带有浓郁的咸味。它是为任何菜肴添加一点鲜味的好方法，这是近十年来澳大利亚烹饪中最热门的公开秘密。它也是一种可利用的导电涂料，亚历克斯曾用它将烘烤过的食物做成一种音乐装置。

该设备的基础是在 Adafruit Circuit Playground Express 上运行的基本音频示例草图。代码被调整为演奏 7 个音符的 c 大调音阶。然后用牙签通过每个衬垫将 PCB 连接到肉馅卷饼上，烘焙食品本身似乎连接到接地层。穿过垫子的牙签然后涂上 Vegemite，另一根牙签以同样的方式处理并用作指示笔。用牙签接触肉馅卷饼和其中一个垫子，电路接通，音符奏出。

当然，这是一种不切实际的音乐表演方式。但这是电子产品中使用的食品的有效展示。加尔文·哈里斯在 2009 年也做了同样的事情，只不过用人类代替了美味的糕点。休息后的视频。
