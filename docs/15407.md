# 简单模式让四足机器人站立和行走

> 原文：<https://hackaday.com/2022/11/19/simple-mod-lets-quadruped-robot-stand-and-walk/>

当谈到运动时，机器人通常不会一次做一件以上的事情。步行者坚持走，滚轴坚持滚。然而，这个让猎豹式四足动物站立甚至行走的简单方法相当聪明。

只需在后腿的胫骨上安装几个刚性支柱，机器人就有可能把自己提升到一个稳定的站立位置，甚至还能走动一下。对于一对没有活动部件的螺栓连接的钻头来说还不错！

机器人风格对我们的一些读者来说会很熟悉。它确实类似于 [Boston Dynamics 的 Spot](https://hackaday.com/2020/06/18/boston-dynamics-spot-robot-gets-a-price-tag/) ，但它更接近于麻省理工学院的*迷你猎豹*，其设计和无刷电机为[带来了引人注目的灵活性和速度](https://hackaday.com/2019/09/22/watch-legged-robot-run-circles-around-its-bigger-brethren/)。它不仅激发了无数的 DIY 努力，也激发了来自海外卖家的成套零件。
