# 电力是如何到达插座的

> 原文：<https://hackaday.com/2019/08/28/how-the-power-gets-to-the-outlet/>

【实用工程】准备在他的[最新视频](https://www.youtube.com/watch?v=7Q-aVBv7PWM)中向你解释变电站是如何获取电力的，你可以在下面看到。当我们与社区内外的人交谈时，我们总是注意到一件事，那就是大多数人不知道现代世界是如何运作的。

请你的非技术类朋友解释手机如何工作或硬盘如何存储数据，你不太可能得到一个很好的答案。然而，即使是我们中的大多数人，也只是专注于电子产品的某个特定方面。有很多人黑机器人或收音机。不过，交流电网并不是很多人的业余爱好。你知道每天上下班经过的那个变电站到底发生了什么吗？如果没有，你会在视频中学到一些东西。

我们当然知道变压器和断路器是什么。但是你知道什么是自动开关吗？你知道变电站可能具有的所有不同功能吗？如果没有，视频是给你的。

如果你想了解更多你在头顶上看到的电线的种类，我们会在前面的中做[。事实上，我们有一个关于](https://hackaday.com/2019/06/11/a-field-guide-to-transmission-lines/)[电网](https://hackaday.com/2017/02/14/the-electrical-grid-demystified-part-ii/)的完整系列，智能和非智能。
