# 真空动力旋转工具 Redux，这次加工

> 原文：<https://hackaday.com/2019/02/28/vacuum-powered-rotary-tool-redux-this-time-machined/>

我们喜欢看到项目被重新审视，尤其是当新的材料或方法使得第一个设计值得再次尝试的时候。这个双涡轮真空动力 Dremel 工具是一个更好的工具可以做什么的完美例子。

你可能还记得[JohnnyQ90]第一次尝试真空动力旋转工具。这个化身在设计上与目前的作品非常相似，完全是 3D 打印的，并在评论中引起了不小的争议，即在 FDM 打印机上以 43，000 RPM 的速度旋转任何东西是否明智。尽管有反对意见，[约翰尼]似乎从他自己的创造中幸存了下来。但是涡轮工具确实有它的局限性，包括有些贫血的扭矩。这个版本，机械加工而不是印刷，几乎完全由铝制成，似乎解决了这个问题，也许是由于旋转部件的质量增加了。双转子和定子是用 5 轴数控机床加工的，这对[Johnny 90]的商店是一个很大的补充。涡轮轴看起来像一个微型喷气发动机，使用安装在车床主轴箱和尾座的磁铁精心平衡。下面的视频展示了构建和一些测试；我们并不热衷于将工具挂在那根粗大的软管末端的人体工程学，但它看起来确实工作得很好。还有那个声音！

我们第一次注意到[JohnnyQ90]是在他加工汽水罐中的铝来制造迷你特斯拉涡轮机的时候。从那时起，他的构建已经走了很长的路，我们期待着他接下来会拿出什么。
