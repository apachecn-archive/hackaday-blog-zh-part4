# 数字厨房勺子让您轻松称量配料

> 原文:[https://hack aday . com/2022/10/27/digital-kitchen-spoon-makes-weighing-your-ingredients-a-snap/](https://hackaday.com/2022/10/27/digital-kitchen-spoon-makes-weighing-your-ingredients-a-snap/)

谈到食谱，似乎有两个阵营:基于体积测量的阵营和基于成分重量的阵营。重力测量的优势在于精确度更高，但代价是无法快速获取一点点这种和一点点那种。这将是很好的获得方便的体积测量与称重您的成分的准确性，不是吗？

是的，这正是[企鹅 DIY]用这个数字厨房勺子秤所做的。也许不足为奇，建造始于一个大搅拌勺和一个非常小的厨房尺度。勺子的碗被从手柄上切下来，连接到应变仪上，应变仪与 LCD 显示器和电路板一起从秤上拆下来。为了保持一切，一个有点敦实的手柄是由夹在铝垫之间的环氧树脂制成的。原始电子部件的隔间，以及 LiPo 电池和 USB 充电器模块，都是从树脂块中雕刻出来的，电子设备安装在显示器和控制器上，便于使用。下面的视频展示了厨房中勺子秤的制作和运行。

我们认为这不仅是一个伟大的想法，而且是一个了不起的执行。黑色环氧树脂和铝在手柄上看起来非常棒，几乎就像一个商业产品。当然，从头开始构建一个秤是很容易的——见鬼，你甚至可以[去掉应变仪](https://hackaday.com/2019/01/27/quartet-of-smd-resistors-used-to-sense-z-axis-height/)——但是在这里拆开现有的秤似乎是正确的。
