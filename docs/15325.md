# 去放风筝吧

> 原文：<https://hackaday.com/2022/11/13/go-fly-a-kite/>

三十年来，从风中获取能量一直是一种商业上可行的产生清洁能源的方式。风力涡轮机是一种可靠的、经过验证的技术，但它们也有一些缺点，其中之一是，由于地面以上的风力更大，这通常意味着更高、更昂贵的塔。不过，有一种方法可以解决这个问题，那就是用风筝代替固定的涡轮机来产生能量。

虽然风筝发电机不是一个新的想法， [[Benjamin]一直在研究这个风筝发电机](https://hackaday.io/project/188136-kites-for-future-flying-wind-turbine)，它比现有的风筝发电机有很多改进。像其他风筝发电机一样，这一个使用系绳来旋转位于地面上的发电机。虽然这与其他风筝系统相似，但这个原型的设计更简单，在飞行中可以扫过更大的区域。它还有一个带有多个独立转向系统的自动驾驶仪，[本杰明]说，如果风足够大，这将允许它一次飞行几个月。如果没有，它可以可靠地着陆，发射它也相对快速简单。

虽然与固定涡轮机相比，风筝确实有一些明显的缺点，包括系绳处的单点故障和大量的空地可供操作，但它们也有很多优点。它们更小，更简单，不需要复杂的偏航系统，并且可以很容易地在地面上维护。事实上，仅仅用一个业余爱好的风筝和一些现成的电子元件就可以制造出非常简单的风筝发电机。
