# 炉顶牛奶蒸锅漂亮、有效

> 原文：<https://hackaday.com/2018/12/12/stovetop-milk-steamer-is-beautiful-effective/>

摩卡壶是一种工业设计经典，起源于 20 世纪早期的意大利。直到今天，它仍然是一个在家里用负担得起的设备酿造高品质咖啡的极好方法。然而，如果你更喜欢拿铁咖啡，而不是浓缩咖啡，那你就不走运了——除非你有一杯这样的咖啡。

【创造】从这个项目的一个经典摩卡锅开始，着手打造一个炉顶牛奶蒸锅。顶部的蓄水池很快被切掉，一个水龙头安装在底部蓄水池的顶部。这使得来自下部容器的蒸汽流量得到控制。然后在龙头上安装一根钢管，钢管被弯曲、挤压、焊接成一个喷嘴，用来蒸牛奶。然后用漂亮的木质把手装饰，营造出一种美妙的美感。

虽然我们不确定焊接过程或使用的水龙头是食品级的，但有解决办法，这是一个可以在一个周末轻松完成的项目。更重要的是，你可以用一杯美味的热卡布奇诺来庆祝你的新创作。还有比这更好的吗？现在你所需要的就是你自己的特色烤肉。休息后的视频。
