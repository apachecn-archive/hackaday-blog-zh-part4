# 模板真空辅助有助于避免涂抹锡膏的心碎

> 原文：<https://hackaday.com/2021/12/22/stencil-vacuum-assist-helps-avoid-the-heartbreak-of-smeared-solder-paste/>

虽然使用模板应该使焊膏应用到 PCB 上成为一件简单的事情，但有许多“陷阱”使它更像艺术而不是科学。幸运的是，你可以制作一些工具，比如[这个 3D 打印的真空辅助模板夹具](https://hackaday.io/project/182116-3d-printed-vacuum-solder-stencil-printer-box-v3)，这需要一点技巧。

对于那些没有享受过这种乐趣的人来说，焊锡膏模板通常用于将适量的焊锡膏涂在 PCB 的焊盘上，并且只涂在焊盘上。问题是，一旦锡膏被挤过模板上的孔，不涂抹就不容易移除模板。[Marius Heier]的模板盒本质上是一个连接到商店真空吸尘器的小室，还有一个两件式穿孔工作台面。顶部平台的中心部分是固定的，而外部部分在 3D 打印的弹簧上上下移动。

在使用中，PCB 被放置在中心固定平台上，而模板位于其上。吸力将模板牢牢地吸到 PCB 上，并在焊膏涂敷时保持在那里。释放吸力会使平台的外部垂直弹起，形成漂亮、整洁的焊料覆盖焊盘。[Marius]在下面的视频中展示了盒子，并展示了一些可以与不同尺寸的 PCB 配合使用的适配器。

如果你认为你最近在附近见过手动真空模板盒，那你就对了——不久前，我们推出了一款由[意外制造者] 制作的模板盒。
