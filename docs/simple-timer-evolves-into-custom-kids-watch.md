# 简单的计时器演变成定制儿童手表

> 原文：<https://hackaday.com/2019/02/27/simple-timer-evolves-into-custom-kids-watch/>

第一次带着新手表去学校是孩子一生中的重要时刻。如果是你爸爸定制的数字模拟手表，那就完全是另外一回事了。

正如[克里斯·奥莱利]所说，他为儿子[弗拉德]制作的手表最初只是一个简单的日常刷牙计时器，除非给予适当的激励，否则任何忙碌的小伙子都不会理会这项家务。这就演变成了一个用发光二极管代替指针的通用模拟时钟的想法。[Chris]认为五分钟的分辨率对于一个九岁的孩子来说已经足够了，这大大减少了所需的 led 数量。ATtiny841 告诉 28 通道 I2C 驱动器点亮哪个 led，RTC 芯片让[Vlad]按计划运行。美丽的印刷电路板生活在数控加工铝外壳；我们实际上对[克里斯]评论说，丙烯酸原型本身看起来很棒，但[弗拉德]想要金属。手表没有外置按键；相反，稍微柔软的聚碳酸酯晶体靠着安装在 PCB 上的按钮来控制功能。

戴上时髦的腕带，[弗拉德]将在校园里尽情摇摆。这是一件非常漂亮的作品，需要广泛的技巧来完成，就像所有的手表一样。看看其他的手表制造，比如[这款可爱的纯模拟](https://hackaday.com/2018/03/27/attempt-at-wristwatch-is-a-solid-success/)，[另一款数模混合](https://hackaday.com/2018/07/05/leds-make-an-analog-wristwatch/)，或者[这款内置英格玛机的怀表](https://hackaday.com/2018/12/03/an-arduino-wrapped-in-an-oled-wrapped-inside-an-enigma-pocket-watch/)。
