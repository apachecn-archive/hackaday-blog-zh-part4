# 多音量旋钮给你所有的程序一个转弯

> 原文:[https://hack aday . com/2020/07/06/multi-volume-knob-gives-all-your-programs-a-turn/](https://hackaday.com/2020/07/06/multi-volume-knob-gives-all-your-programs-a-turn/)

我们都经历过。你在战斗岗位上，在模拟赛车或其他游戏区的深处，突然有人大声说话，打破了你的耳膜。不得不停下手头的工作来寻找合适的 Windows 音量滑块是一种侮辱。那么为什么要这么做呢？[打造【T3knomanzer】简单而优雅的多音量旋钮](https://maxmixproject.com/)，让自己沉浸其中。

[![](../Images/fb772bd3d749d48bc2ece33d6c202284.png)](https://hackaday.com/wp-content/uploads/2020/07/vol-knob-guts.jpg) 很简单，只需转动旋钮循环播放你的节目，直到小屏幕上出现不和谐的声音，然后按下旋钮就可以变成音量旋钮。如果您需要更改另一个音量，只需再次单击它。因为没有 Alt+Tabbing 到桌面，所以没有方格旗会从你的手指间溜走。

在这个精心设计的盒子里，你会发现一些常见的疑点——Arduino Nano、旋转编码器、有机发光二极管显示器和 LED 环，每一个都有自己的位置。

这个完全开源的旋钮看起来很棒，我们喜欢它非常容易复制，通过建立一个简单、描述良好、一步一步说明的网站。休息后看他们跳舞。

想要触手可及的不仅仅是音量？[这里有一个 DIY USB 旋钮，也可以做快捷方式](https://hackaday.com/2019/03/19/a-classy-usb-knob-for-the-discerning-computerist/)。
