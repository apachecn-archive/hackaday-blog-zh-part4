# 可视化涡流

> 原文:[https://hackaday.com/2019/02/19/visualizing-eddy-currents/](https://hackaday.com/2019/02/19/visualizing-eddy-currents/)

如果[Electroboom]放弃制作视频，决定成为波科诺斯的一只睡椅蜥蜴，我们希望他采用艺名涡流。然而，他在他最近的[视频帖子](https://www.youtube.com/watch?v=u7Rg0TcHQ4Y)中谈论了涡流，你可以在下面看到。

我们知道他并不真的认为他可以用一张铝箔让磁铁慢下来，他至少上演了他的大多数小电力事故，但我们仍然喜欢看它。同时，他也很好地解释了为什么铜管会使磁铁减速，以及涡流如何影响变压器效率。

他的跳舞线圈演示很有趣，但我们希望你不要亲自尝试，当他拿出隔离变压器时，我们非常放心。你可能想用一卷铝箔和一块磁铁来看视频——这似乎足够安全了。

了解一些涡电流可以解释一个常见的超导体技巧，以及感应炉灶等等。当然，在电机和变压器中，这些涡电流是不希望出现的，这一事实会影响这些设备中电磁铁芯的设计。

我们喜欢娱乐性和教育性的视频。【电爆】总是[对两个](https://hackaday.com/2017/04/06/that-sucks-death-of-a-tesla-coil/)都好。他确实倾向于发出一些更强烈的语言。这导致一些观众自动享受 T2 的困境。
