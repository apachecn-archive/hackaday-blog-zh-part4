# 从打印机到乙烯切割机

> 原文：<https://hackaday.com/2021/07/02/from-printer-to-vinyl-cutter/>

有些人可能会看着廉价的喷墨打印机，看到一个笨重的设备，更换墨水的成本比买一个新的还要高。[阿布舍克·维尔马]看到了一台旧的喷墨打印机，取而代之的是一个光滑的门架和进给机构，这是一个完美的平台来制作他自己的 DIY 乙烯基切割机。

这台打印机被小心地拆开了。进给机构被改造成由步进电机驱动，带有一些 3D 打印的转接板。用于切割刀片的基于螺线管的推/拉机构增加了 3D 打印外壳以及继电器模块。Arduino Uno 借助 CNC GRBL 防护罩接收来自计算机的命令。

我们喜欢的是这款产品的独创性和旧打印机内部零件的重复使用。例如，旧的 PCB 被切割，连接器被重新使用。从外面看，很难相信惠普没有把这个制造成乙烯切割器。

如果你手头没有打印机，你可以随时[用你的 CNC 作为乙烯基切割机](https://hackaday.com/2015/10/07/turning-your-cnc-into-a-vinyl-cutter/)。但是如果你没有 CNC，[Abhishek]共享他的刀具的所有 [STL 文件以及原理图](https://www.youtube.com/watch?v=StF1HVZoEus)。休息后的视频。
