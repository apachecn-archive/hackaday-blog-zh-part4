# 抖动让一切更酷:现在甚至是动画

> 原文：<https://hackaday.com/2022/07/04/dithering-makes-everything-cooler-now-even-animated/>

[dukope]正在写一个游戏，Obra Dinn 的回归，有着梦幻般的视觉风格。其中一个选择是让所有东西都变成辉煌的一位颜色，也就是众所周知的黑白，然后抖动回单色。就像他们以前在 Mac Plus 上做的那样。

如果犹豫不决是你的审美，那么认真对待它是很有意义的。它非常漂亮——看看下面的视频。

但更令人惊讶的是[杜科普]对抖动细节的关注。例如，[TIG 论坛上的这个帖子](https://forums.tigsource.com/index.php?topic=40832.msg1363742#msg1363742)详细介绍了当你有一个抖动的图像需要动画时的问题和解决方案。当虚拟摄像机在场景中平移时，你希望物体上的点保持相对恒定，这就需要一个定制的算法。如果你认为这很酷，[看看游戏中心的书是如何被动画化的](https://forums.tigsource.com/index.php?topic=40832.msg1406158#msg1406158)。

我们能说什么。我们以前喜欢犹豫不决，但这篇文章让我们的爱更深了。
