# 无线发光二极管不是第一个，但是你可以自己制作

> 原文：<https://hackaday.com/2019/05/28/wireless-leds-arent-a-first-but-you-can-make-your-own/>

无线 led。这就是斯科特·艾伦在日本发现的东西，如果你发现了什么，你只需要复制它。

[Scotty]在模型制造商和 gunpla 的展示架上发现了这些无线发光二极管。因为你不想在给塑料模型照明的时候跑电线，钻孔，和光纤打交道，所以模型公司想出了*无线*led。只要把它们粘上，它们就会眨眼。它需要一个基站，但这些是无线发光二极管。

在购买了一些这样的 LED 并采购了一个基站后，[Scotty]发现这些 LED 是由三个组件精心焊接在一起的:一个电感、两个电容和 LED 本身。基站只是两个线圈，实际上是一个无线电话充电器。哦，一些实验表明，如果你把这些无线发光二极管中的一个放在无线手机充电器上，它就会亮。

下一步当然是复制，所以[Scotty]前往秋叶原，拿了一些电线、电阻和发光二极管。电线被绕成一个线圈，焊接上一个 LED，一切正常。这绝不是第一个 DIY 无线 LED，因为有这么多技术，这也是最先流行的，你可以购买内嵌无线 LED 的按压式指甲。看看下面的视频。
