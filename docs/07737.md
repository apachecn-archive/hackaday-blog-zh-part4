# 第三次是这个接篮球机器人的魅力

> 原文:[https://hackaday . com/2020/09/09/third-times-a-charm-for-this-basketball-catching-robot/](https://hackaday.com/2020/09/09/third-times-a-charm-for-this-basketball-catching-robot/)

我们都知道项目的第一个版本通常是令人讨厌的，至少在回顾的时候是这样。当然，它将基本思想转化为具体的形式，但它真正做的只是为第二个版本做准备。好多了，但还是不够。第三版是神奇的地方。

至少这是[谢恩·威顿]寻求建造完美篮球机器人的过程。他的[的第一个版本](https://hackaday.com/2020/04/20/a-basketball-hoop-that-never-lets-you-brick/)是一个被动的篮板，它根据抛物面的形状改变传入的球的方向。尽管决定棋盘形状的数学很酷，但它明显缺乏任何复杂的系统，如电机和机器视觉——你知道，有趣的东西。[第二版](https://hackaday.com/2020/05/19/robotic-basketball-hoop-v2/)拥有所有这些细节，并且更好地捕捉脱靶镜头，但它仍然有一个有限的工作范围。

进入第三版，在下面的视频中可以看到。从[【马克·罗伯】的剧本](https://hackaday.com/2017/03/22/dartboard-watches-your-throw-catches-perfect-bullseyes/)【谢恩】中吸取经验，制造了一个邪恶的过度设计的科里西风格的机器人来覆盖他的商店墙壁。一切都是用尽可能轻的材料建造的，以保持惯性最小，并确保目标尽快到达正确的位置。[谢恩]甚至想出了如何安装电机倾斜篮板在框架上，而不是在马车上。Kinect 对传来的球——或建筑者的头——进行深度探测，并消耗掉它能到达的几乎每一个镜头。

[谢恩]已经做了一些伟大的工作，自动化了职业运动员的工作。除了篮球，他还攻克了[高尔夫](https://hackaday.com/2020/06/11/a-robotic-golf-club-to-possibly-boost-your-game/)和[棒球](https://hackaday.com/2020/08/12/a-special-baseball-bat-with-explosive-hitting-power/)，给两者都带来了爆发力。我们也期待这两个版本的第二版和第三版。
