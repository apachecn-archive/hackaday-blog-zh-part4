# 漂亮的夏季项目从废料中获取太阳能声音

> 原文：<https://hackaday.com/2021/08/17/spiffy-summer-project-sources-solar-sounds-from-scraps/>

[Gijs Gieskes]在制作电子艺术和声音装置方面有着悠久的历史，他的 [Zonneliedjes (sunsongs)](https://gieskes.nl/music/?file=zonneliedjes) 项目无疑是他声波创作的一个有趣延续。sunsongs 的目标是利用阳光创作音乐，它最突出的特点是太阳能电池板。

虽然还不清楚光子是如何转变成有节奏的碰撞和随机的哔哔声的，但结果相当令人满意。我们强烈怀疑，把随机垃圾变成光束机器人的原理也在起作用，也许还加上了一些电路弯曲。我们能够从他称之为“移动”的设备的照片中收集到的一个细节是一个 40106 振荡器，[Gijs] [已经在以前的项目](https://hackaday.com/2016/04/27/autonomous-electro-musical-devices/)中使用过。

[Gijs]使用的建筑风格让我们想起了业余无线电自制社区喜欢的“曼哈顿”建筑风格。方形的铜印刷电路板直接粘在太阳能电池的背面，电路就建在上面。仔细观察这些图片，我们还可以看到看起来像截止线的东西，这表明为了获得我们都可以涉及到的预期结果，进行了大量的实验。

休息之后一定要看看视频，还有[Gijs]网站。很长一段时间以来，他一直在从事这样的项目，我们甚至还特别报道了他 15 年前的项目。感谢持续的攻击，[Gijs]。我们期待看到你接下来的作品！

如果你对“光束机器人学”和“电路弯曲”这两个术语不熟悉(或者如果需要更新)，请务必查看我们最近的[光束机器人学再介绍](https://hackaday.com/2021/05/29/living-robots-revisiting-beam/)和经典的[电路弯曲介绍](https://hackaday.com/2011/01/11/intro-to-circuit-bending/)来熟悉一下。
