# 最后，一个开源的 MIDI 脚控制器

> 原文：<https://hackaday.com/2018/10/21/finally-an-open-source-midi-foot-controller/>

MIDI 比 Hackaday 的大多数读者存在的时间都长，你可以离开我的草坪了。尽管如此，MIDI 仍然普遍用于音乐表演的几乎每一个方面，并且有许多工具和应用程序可以让 MIDI 控制现场表演。也就是说，如果你想要一个 MIDI 脚控制器，你最好的选择可能是 90 年代后期使用的东西，尽管百灵达制造了一个可以接受的脚控制器，但没有一大堆功能。显然需要一个功能齐全、开源的 MIDI 脚踏控制器。这就是 Pedalino 的用武之地。它是今年 Hackaday 奖乐器挑战赛的冠军，如果你想要一个 MIDI 脚控器，这是你应该看的第一个地方。

使用 Pedalino，您可以更改吉他装置的预设，将旧的 MIDI 设备转换为 USB 兼容的设备，在现场表演期间让您用免提或脚踏方式控制装置，并且可以通过 WiFi 或蓝牙进行扩展。这是一个全功能的 MIDI 控制器，有三个用户配置文件，它可以控制最多 48 个脚踏开关。对于这样一个小设备来说，这是一个令人印象深刻的套件数量；通常你需要花费数百甚至数千美元来购买一个简单的 MIDI 控制器，而 Pedalino 用非常便宜的硬件来做所有的事情。

虽然 Pedalino 现在还处于原型阶段，但功能丰富的 MIDI 脚控制器显然有市场。它可能只是一个试验板和一个烧结图，但在软件方面有重要的工作要做，我们期待着它被塞进一个巨大的铝制外壳和一个踏板。
