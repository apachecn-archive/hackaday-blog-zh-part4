# 这个定制工作台会让你翻跟斗

> 原文：<https://hackaday.com/2022/06/26/this-custom-workbench-will-make-you-flip/>

在最近的一个视频中，[[someskilstudio]为他们纤细的车库工作台创造了一个整洁的工具储存系统](https://www.youtube.com/watch?v=kdnd06kkFFM)。我们以前见过这里使用的“五关节”270 度铰链，知道它们可以实现一些很酷的黑客攻击。在这里，您将看到他如何将这种独特的硬件用于构建一个密集的工作面。对于任何一个在车库里开店的人来说，你会知道有一个地方存放所有东西并使之变得容易是多么重要。

该视频有几个很好的技巧来确保所有的东西都组合在一起，这对任何用自己的工具集复制这些东西的人来说都是很关键的。如果你的空间更小，我们有一些很棒的过去的车间建筑，从[便携式](https://hackaday.com/2016/09/17/portable-workbench-crams-an-entire-workspace-into-one-box/)，到[微型](https://hackaday.com/2019/09/18/tiny-woodshop-is-packed-with-space-saving-hacks/)，到[精致](https://hackaday.com/2022/01/24/how-to-optimize-your-workspace-analyze-how-you-work/)。即使你已经建立了一个工作场所，我们也有关于[组织你的商店](https://hackaday.com/2022/01/24/how-to-optimize-your-workspace-analyze-how-you-work/)的建议，在[阴影板](https://hackaday.com/2022/06/21/no-tool-left-behind-with-the-help-of-homemade-shadow-boards/)或跨越[无限网格](https://hackaday.com/2022/04/18/gridfinity-3d-printed-super-quick-tool-storage-and-retrieval/)给每个工具一个家。显然，创造一个工作空间是我们最喜欢的项目之一。
