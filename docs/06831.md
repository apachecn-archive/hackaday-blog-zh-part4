# 自主哨兵枪包含一拳和一吨的建设技巧

> 原文:[https://hack aday . com/2020/06/10/autonomous-sentry-gun-packs-a-punch-and-ton-of-build-tips/](https://hackaday.com/2020/06/10/autonomous-sentry-gun-packs-a-punch-and-a-ton-of-build-tips/)

什么东西有双压缩空气炮，500 个滚球除臭球，还有一个态度恶劣的机器学习大脑？我们也不知道，直到[Leo Fernekes]把这个视频放在他的自动机器人哨兵枪上，亲眼看到它的行动。

现在，我们已经在这些页面上看到了成吨的哨兵枪，射击从[水](https://hackaday.com/2018/01/26/auto-tracking-sentry-gun-gives-deer-a-super-soaking/)到[各种形式的 Nerf](https://hackaday.com/2019/06/13/well-built-sentry-gun-addresses-the-menace-of-indoor-micro-uavs/) 的一切。而且很多这样的建筑都使用了某种形式的机器视觉来瞄准目标。因此，虽然看起来[Leo]在这里重操旧业，但这个版本充满了有趣的提示和技巧。

当[Leo]从我们的朋友[Edje Electronics]那里看到一个关于 TensorFlow 基础知识的视频时，这给了他投入人工智能项目所需的动力。他最终使用的控制器在场景中寻找人类，并将炮塔转向目标，在那里空气炮可以做它们的事情。沉重的弹药由压缩空气推动，压缩空气通过一个带有有趣驱动器的电磁阀注入弹膛，驱动器可以最大化打开速度。风格点指向[噬菌体 T4 灵感设计](https://en.wikipedia.org/wiki/Escherichia_virus_T4)，以及从 1:34 开始的序列，这让我们想起了*机械战警* 中的[工厂场景。](https://www.youtube.com/watch?v=IvDNfFVpvZI)

[Leo]在这个项目上投入了大量的工作，结果显示。他希望得到一个美术馆或博物馆来展示它，作为一个互动的作品来评论一个可能的机器人-人类的未来，大概是在让客人签署一份新闻稿之后。不管发生什么，这个机器人看起来很棒，[利奥]从它身上学到了很多，我们也是。
