# 定制迷你四轮驱动蒸汽运行

> 原文：<https://hackaday.com/2019/01/02/custom-mini-4wd-runs-on-steam/>

田宫的迷你四轮驱动玩具系列主要包括由 AA 电池供电的 1:32 比例的小型玩具车，它们没有遥控器，由水平定向的~~驱动~~导向轮围绕塑料轨道引导。调试和比赛这些车在世界上很多地方都很受欢迎，[但是这款车有点不同。](https://www.youtube.com/watch?v=XCe5iiKojno)

在最初的实验与修改田宫底盘不成功，一个新的建立使用定制的铝底盘开始。一个坚固的锅炉被制造出来，输入到一个活塞中，活塞通过一系列传动轴驱动所有四个轮子。

观看迭代设计过程解决各种问题(如活塞磨损和齿轮传动)很有趣。对于那些习惯了电动玩具的巨大速度的人来说，性能并不令人印象深刻，但我们很乐意看到使用蒸汽动力赛车的比赛系列。

我们在这里看不到太多蒸汽机车，但是哈德斯皮斯蒸汽机车是令人惊叹的东西。休息后的视频。
