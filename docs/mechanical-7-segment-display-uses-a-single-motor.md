# 机械 7 段显示器使用单个电机

> 原文：<https://hackaday.com/2021/06/27/mechanical-7-segment-display-uses-a-single-motor/>

七段显示器已经存在很长时间了，而且似乎有无数种方法来构建它们。最新的是 3D 打印机械大师【gzumwalt】的[机械七段](https://www.youtube.com/watch?v=ukRGbEUAVtM)，可以使用一个电机循环所有十个可能的数字。

诀窍在于一对同步的旋转圆盘，一个用于顶部的四段，另一个用于底部的三段。每个圆盘都有一系列同心的凸轮槽来驱动从动轮，将红色部分翻转到视野内外。在转盘的一次旋转中，显示屏可以循环显示所有十种状态，因此凸轮路径被分成 36 个增量。[gzumwalt]向我们展示了一个完整的实体版本，但从 CAD 设计和单个部分的工作原型来看，我们非常有信心它会。虽然它没有在设计中显示，但我们怀疑它将由步进电机驱动，并与皮带或中间齿轮同步。

我们最近看到的另一个 3D 打印机械显示器是 DIY flip dot，array，它使用了与商业版本相同的电磁系统。[gzumwalt]拥有围绕单个马达设计迷人的机械机器人的天赋，包括一个[边缘回避机器人](https://hackaday.com/2020/12/27/mechanical-edge-avoiding-robot/)和一个[磁性冰箱爬行器](https://hackaday.com/2020/12/10/a-walking-rover-destined-explore-your-fridge-door/)。
