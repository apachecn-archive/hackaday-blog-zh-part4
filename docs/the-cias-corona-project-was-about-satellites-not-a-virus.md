# 中情局的电晕项目是关于卫星，而不是病毒

> 原文：<https://hackaday.com/2020/08/23/the-cias-corona-project-was-about-satellites-not-a-virus/>

如今，我们认为轨道图像是理所当然的，但曾几何时，它是高科技和高度保密的。[斯科特·曼利]很好地概述了中情局的科罗纳间谍卫星，以及该计划早期的解密图像。

今天看来很奇怪，但间谍图像需要高分辨率，当时唯一实用的技术是胶片。这颗卫星装有高达 3000 英尺的胶片，一旦拍摄完毕，一个胶囊或桶将返回地球进行回收和冲洗。他们没有成功着陆——或者至少他们不应该着陆。中情局不希望反对者扫清这部电影，所以一架飞机应该在它降落到降落伞上时接住水桶，这是[汤姆·纳尔迪]关于从太空中接住落下的东西的历史的文章中的一个主题。

早期的照相机可以看到 40 英尺以下的细节。到 20 世纪 70 年代该项目结束时，改进的摄像机可以看到 3 英尺或更低。后来的卫星有一个 3D 相机和多个返回桶。这些卫星——官方说法——是一个将生物样本暴露在太空环境中并返回进行分析的项目。“发现”计划纯粹是一个幌子，整个事情在 1992 年被解密。

当然，从飞机上拍摄的电影也有一定的作用。一些间谍卫星试图[扫描胶片](https://hackaday.com/2018/08/02/the-photo-lab-that-flew-to-the-moon/)并发回数据，但这在月球任务中使用得更多，因为将太空舱返回地球要困难得多。
