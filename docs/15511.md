# 音频放大器让 VFD 以一种不寻常的方式工作

> 原文：<https://hackaday.com/2022/12/08/audio-amp-puts-vfds-to-work-in-an-unusual-way/>

可以肯定地说，大多数以 VFD 为特色的项目都更强调“D”方面。毕竟，真空荧光显示器是可靠的表演者，它们凉爽的蓝绿色光芒正是许多复古和不那么复古的建筑的正确外观。但这并不意味着没有利用“V”方面的应用，例如[这个使用 VFD 作为有源元件的漂亮音频前置放大器](https://www.instructables.com/Audio-Preamp-Built-From-VFD-Indicator-Tubes/)。

[JGJMatt]建造背后的灵感来自基于 VFD 的低压双三极管真空管的 [Korg Nutube](https://hackaday.com/2017/06/26/hackaday-prize-entry-messing-around-with-new-vacuum-tubes/) 系列。发现这些特定的组件有点昂贵，[JGJMatt]转向旧的备用 DM160 VFD 指示管，它基本上只是一个三极管，看看它如何作为放大器。该电路通过将 VFD 楔入 2N3904 输入级和 2N2007 MOSFET 输出之间，利用了 VFD 的低电流和低电压要求，整个电路通过 USB 升压转换器运行。在定制蚀刻的 PCB 上混合了 SMD 和通孔组件，有一个单独的 riser 卡，通过 3D 打印外壳的前面板稍微展示一下 VFD。

总而言之，我们觉得这个小放大器很酷，我们喜欢它给古老的 VFD 带来的变化。我们以前见过[类似的 VFD 放大器](https://hackaday.com/2013/11/09/vfd-display-becomes-an-amplifier/),但这一个的适合和完成真的值得。
