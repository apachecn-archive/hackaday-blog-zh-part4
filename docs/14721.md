# 用 TreeSheets 组织你的黑客:一个开源的分层电子表格

> 原文:[https://hackaday . com/2022/09/07/organize-your-hacks-with-tree sheets-an-open-source-hierarchical-spread sheet/](https://hackaday.com/2022/09/07/organise-your-hacks-with-treesheets-an-open-source-hierarchical-spreadsheet/)

有些人可能听说过荷兰程序员[Wouter Van Oortmerssen],因为他是 Amiga-E 编程语言的创造者，并且参与了几个游戏引擎。听说过*模拟城市*？*边陲之地 2* 或者*孤岛惊魂*怎么样？作为一名为各种客户服务的程序员，他有着漫长而辉煌的职业生涯——包括在谷歌的长期工作，从事网络组装——许多人至少会熟悉他的一些工作。但是你可能没有听说过他的 [TreeSheets 生产力工具](https://strlen.com/treesheets/)。这将是一个遗憾，因为你已经错过了一些非常有用的东西。

TreeSheets 被描述为一个分层的电子表格，它旨在替代几种不同的工具；想想电子表格、思维导图和文本编辑器等等。用[Wouter]的话说:

> 它就像一个电子表格，非常熟悉，但更适合复杂的数据，因为它是分层的。它就像一个思维映射器，但更有条理、更紧凑。
> 它就像一个大纲视图，但不止一个维度。
> 这就像一个文本编辑器，但是有结构。

经过大约十年的发展，TreeSheets 可能看起来有些过时，但设计是清晰的，不受干扰的，这正是当你试图专注于手头的任务时所需要的。为什么不试一试，看看它是否对你有用？休息过后，[可以看到 YouTube 用户【DrilixProject】](https://www.youtube.com/watch?v=UB-saQZfrsw)的视频教程。

对于生产力黑客来说，这并不陌生，Pomodachi 是一只可爱的 pomodoro 定时器和一只电子鸡的杂交。有时候软件工具不能解决生产力问题，但更多的是与你对项目的态度有关[。最后，有时候提高生产力的途径离家更近一点，](https://hackaday.com/2018/09/25/productivity-unfinished-projects-and-letting-go/)[就像在你的头脑里](https://hackaday.com/2016/10/14/program-your-brain-hack-your-way-to-productivity/)。
