# 视差更新黑客聊天

> 原文：<https://hackaday.com/2019/08/26/parallax-update-hack-chat/>

加入我们太平洋时间 8 月 28 日星期三中午的[视差更新黑客聊天](https://hackaday.io/event/165855-parallax-update-hack-chat)与[芯片](https://hackaday.io/ChipGracey)和[肯格雷西](https://hackaday.io/kgracey)！

对于我们中的许多人来说，我们第一次接触微控制器世界是通过 [Parallax 公司](https://www.parallax.com/)的产品。也许你对做一些小而轻的事情感兴趣，并希望利用你在 IBM-PC 或苹果电脑上的编程技能。或者你有一个老师围绕着一个 Boe-Bot 建立了一个机器人课程，或者你加入了第一个使用视差传感器的机器人团队。

无论你与 Parallax 产品的关系如何，毫无疑问，它们是业余爱好者微控制器革命的前沿。你也不能怀疑视差不仅仅是基本的邮票。自 2006 年推出以来，其广受欢迎的多核推进器芯片已经获得了热情的追随者，并已进入大量项目，其中许多项目[我们已经在 Hackaday](https://hackaday.com/2015/09/04/hacklet-73-parallax-propeller-projects/) 上做了专题报道。现在，人们期待已久的继任者 Propeller 2 也即将上市。

Gracey 兄弟从一开始就是 Parallax 的幕后黑手，Chip 设计所有产品，Ken 经营业务。他们将加入我们的黑客聊天，让我们了解视差怪的一切新情况，并告诉我们 P2 的内幕。请务必停止你的视差问题，或只是说你好。

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)中的社区直播活动。本周，我们将于太平洋时间 8 月 28 日星期三中午 12:00 坐下来讨论。如果时区让你烦恼，我们有[一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20190828T12&p0=224&msg=Parallax+Update+Hack+Chat&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
