# 3D 打印的大理石音乐机器看起来已经不错了

> 原文：<https://hackaday.com/2021/11/16/3d-printed-marble-music-machine-looking-good-already/>

受到才华横溢的[Wintergatan]的巨大大理石音乐机器和[Daniel de Bruin]的大理石跑道的启发，[Ivan Miranda]再次忙于建造一个基本上是 3D 打印的装置，以测试他从头开始建造自己的大理石音乐机器的想法。(视频，嵌在下面。)

利用他最近在树脂打印方面的经验和他自己的[巨型 3D 打印机](https://www.youtube.com/watch?v=KhBdMpEGH0o&t=1s)，他毫无困难地从他的车间生产出他需要的一切，即使设计工作显然需要很长时间。

这个构建显示了这个项目处于开发的早期阶段，因为显然有相当多的问题需要处理，但是到目前为止进展看起来令人鼓舞。要明确的是，计划是“做大”，这个小的八通道测试床只是为了探索球导向，运输和球释放到第一个音频测试设备 Korg Nano Pad 2 上的问题。

一些重要的初期问题被确定，例如当[Ivan]设计球升降机时，他打算从后面装载球，但后来需要切换到从前面装载。没什么大不了的，只需反转电机方向，在机械装置的另一侧装载球。可悲的是，这也意味着直接耦合的纸币鼓现在也以错误的方式旋转来释放球。哎呀。一次快速的黑客攻击后，[伊万]又开始工作了。各种零件需要用盘子垫起来，但有了工作台上的 3D 打印机，敲出这些零件几乎不需要时间或精力。这只是显示了 3D 打印机有多么有用，允许你在短时间内迭代，并将你的想法反馈到最终版本中。

[Ivan]显然会对这个有很多“乐趣”，因为[[winter gatan]肯定会证明](https://www.youtube.com/watch?v=WN90HYiFpAw)，这些大型音乐大理石机器的建造是相当了不起的事业。我们以后一定会继续关注，看看这部电影会走向何方！

当我们谈到[Wintergatan]大理石机器时，这里有一个向最新大理石机器 X 致敬的[，如果你需要一个 3D 打印大理石时钟，](https://hackaday.com/2020/08/22/mini-marble-powered-synth-pays-homage-to-its-bigger-cousins/)[那么先试试这个](https://hackaday.com/2018/09/12/a-3d-printed-marble-clock/)。
