# 激光切割一系列生成的雪花

> 原文：<https://hackaday.com/2022/12/12/laser-cutting-a-flurry-of-generative-snowflakes/>

现在是假期，还有什么比在你的激光切割机上雕刻出一些雪花更好的庆祝方式呢？[Bleeptrack]开发了一个基于网络的工具[来创建可生成的雪花饰品，这些饰品可以导出为现成的 SVG 文件，用于激光或乙烯切割。](https://fr0zensystem.bleeptrack.de/)

正如它们的名字一样，每个生成的雪花装饰物(很可能)都是独一无二的，创建的多层可以相互堆叠。[Bleeptrack]展示了一些实现，使用半透明纸夹在两个顶层剪纸之间，由木头或纸板制成。

雪花是最小设计的一个伟大平衡，同时仍然是美丽和丰富的细节。它们可以很容易地在你手边的任何激光切割机或乙烯切割机上生产出来。对于那些想深入了解 web 工具细节的人来说，可以在 GitHub 上找到源代码。切一个自己的会成为一个 [Neodriver 饰品](https://hackaday.com/2022/10/21/neodriver-ornament-brightens-up-christmas/)或者一个小小的[DOOMplaying 饰品](https://hackaday.com/2022/10/21/neodriver-ornament-brightens-up-christmas/)的完美附加物。休息后的视频！
