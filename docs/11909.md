# 推进纸飞机，用电磁铁将香肠一分为二

> 原文:[https://hacka day . com/2021/11/14/propel-paper-planes-bi section-香肠-电磁铁/](https://hackaday.com/2021/11/14/propel-paper-planes-bisect-sausages-with-electromagnets/)

你还在用手放纸飞机吗？那像是婴儿的玩具！[汤姆·斯坦顿]和他自制的[电磁轨道发射器](https://www.youtube.com/watch?v=_4TGb3MsSjE)一定会把你的纸飞机游戏带入 21 世纪。

公平地说，这种线性马达不仅仅可以用于发射纸飞机，而且已经可以在利基工业应用、公共交通系统和过山车中找到。是的，世界上许多超级大国也在积极探索在战场上利用电磁的潜力，比如高斯步枪和轨道炮。与此同时，视频(休息后)证明了完全有可能在你的创客空间里，使用相对基本的组件和基本的物理学，建造一个基本而有效的线性马达。

简而言之，这些发射系统使用电磁和适时的电子设备推动大量磁性材料沿着直线(有时是曲线)轨道前进。多对线圈沿着轨道放置，随着有效载荷的接近，每对线圈随后被高电流激励。通过连续使用许多线圈，质量及其有效载荷可以被加速到高速。

虽然自制的轨道发射器不太可能扭转战争的趋势，但[汤姆·斯坦顿]通过一项涉及高速视频和超市香肠的实验，探索了它们的致命潜力，得出了可怕的结果。
