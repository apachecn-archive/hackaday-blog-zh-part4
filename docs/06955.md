# 一个小三角钢琴得到 MIDI

> 原文:[https://hackaday.com/2020/06/25/baby-grand-gets-midi/](https://hackaday.com/2020/06/25/baby-grand-gets-midi/)

像许多人一样，[雅克]不认为一大块轻到可以夹在腋下的塑料是钢琴，即使它有 88 个琴键。钢琴应该是一件很重的家具，你得给别人买披萨来帮你搬。所以他买了一架二手的婴儿三角钢琴。尽管它的形状不是很好，所以在修复它的时候，他也给它添加了 MIDI。你可以在下面的视频中看到最终的结果。

100 美元，这个价格是合适的，尽管移动它要花更多的钱。在水渍、虫蛀和车库存放之间，这架旧齐默曼钢琴需要精心呵护。当谈到 MIDI 时，[Jacques]发现了一个二手的 Disklavier 一个非常昂贵的套件-但它不适合 Zimmerman 或手边的另一架钢琴。螺线管和光学传感器是为特定的钢琴设置的，那么你能做什么呢？轻松点。重建固定电磁阀和传感器的杆。

做任何你必须重复 88 次的事情肯定是一个挑战。但比那更糟。按键间隔紧密且不规则。螺线管必须非常紧密地包装，任何错误都意味着重做整个装配。幸运的是，新的条形设计将螺线管分组，因此您可以移除一些螺线管，而不必取出所有螺线管。

到目前为止，只有输出螺线管在工作。解读密钥是未来的工作，但我们认为这可能比他已经完成的更容易。这些年来，我们已经看到了很多对钢琴的模仿。我们甚至已经看到螺线管把戏在小规模上完成。
