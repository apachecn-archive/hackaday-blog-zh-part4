# 纸板在产品设计中的价值

> 原文:[https://hack aday . com/2018/11/14/纸板在产品设计中的价值/](https://hackaday.com/2018/11/14/the-value-of-cardboard-in-product-design/)

前阵子，【Eric Strebel】做了一个背包挂钩。结果很棒——只要把这个背包挂钩栓在墙上，他就能让背包远离地板，不碍事。甚至还有一个地方可以给他的手机充电。[Eric]正在考虑将这个想法变成一个产品，[刚刚发布了一个视频，展示了他制作纸板模型的过程](https://www.youtube.com/watch?v=8w2poOzRZeg)。

由于这是一项工业设计研究，任何模型都需要记住成品将如何构建。在这种情况下，[埃里克]将使用 4-5 毫米厚的铝，在喷水器上切割，弯曲到位，最后阳极氧化。成品将由弯曲的铝板制成，所以这一点点产品设计将使用 mat board——一种厚而重的纸板，常用于将图片安装在框架中。板坯将取代铝，因为它被仔细切割，弯曲，并粘合成型。

这个构建的工具很简单，只需要一把业余爱好刀、刀片、尺子和一支笔。但是使用 Matboard 有一些技巧。为了完美地弯曲这些作品，[埃里克]正在用水涂抹一面。这使得板坯中的纤维变得松散，从而在一层建筑被粘合在一起之前形成完美的折痕。

一旦几层板坯粘合在一起，成品就变得不像纸板，而更像非常柔软的木材。这使得[Eric]可以使用砂带打磨器和埋头钻头对这个一次性的原型进行一点点打磨。这篇完成的文章非常棒，现在[Eric]正在考虑将这个想法投入生产。
