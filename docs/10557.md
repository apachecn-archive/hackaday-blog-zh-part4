# 这么老的手机不应该运行安卓系统

> 原文：<https://hackaday.com/2021/06/28/a-phone-that-old-shouldnt-be-running-android/>

汽车和智能手机有一些奇怪的共同点，就像来自不同制造商的大多数日常轿车都趋于相似一样，智能手机也是如此。无论你的智能手机是最新最好的，还是只花了你 50 美元从超市买来的，看起来都没什么关系，因为这两款手机从表面上看都是几乎相同的黑色平板。

然而情况并不总是这样，在过去的几十年里，来自不同制造商的手机都有自己的风格，并且有各种各样的外形来适应所有的口味。不过，对那些日子的粉丝来说，还有一线希望，这就是 2000 年的索尼翻盖手机。它运行安卓系统。是的，你没看错，小屏幕上是 Android 9。

当然，这款手机所配备的处理器和电子设备早已不复存在，取而代之的是，这款手机采用了现代中国手表的内部结构——智能手机取代了原来的手表。整个电子产品包适合屏幕开口，虽然它需要一些 USB-C 插座和一些其他部件的布线，但从外面看起来，它似乎是为了运行 Android。可以看看休息下面的视频。

他高兴地承认，还有一段路要走，例如让原来的键盘工作，但即使有一个小小的触摸屏，它也足以成为一个日常司机。它可能有点小，但对于我们这些怀念旧手机的人来说，也许它有希望成为新的东西。

同时，这不是我们最近看到的第一次重复使用旧手机。
