# 射电望远镜伴随着 GNU 电台

> 原文:[https://hack aday . com/2019/01/20/radio-telescopes-horn-in-with-GNU-radio/](https://hackaday.com/2019/01/20/radio-telescopes-horn-in-with-gnu-radio/)

谁不喜欢仰望夜空？但是如果你对无线电感兴趣，使用射电望远镜会有一个完全不同的视角。[John Makous]在 GNU 无线电会议上讲述了他是如何致力于制造一个甚至对更年轻的学生来说都很实用的射电望远镜。

这个建筑唯一真正的高科技部分是低噪声放大器(LNA ),这个项目是一个典型的教师可以接触到的，他可能不是电子专家。它使用像油漆、稀释剂、罐子和木材这样的东西。[John]还在 GNU Radio 中构建了一些模块，方便其他教师处理来自望远镜的数据。正如他所说，“这是我喜欢做的那种无聊的事情。”我们能理解。

这台望远镜是用来接收 21 厘米波段来探测来自银河系的中性氢的。它可以绘制星系中的氢，还可以利用多普勒频移测量星系的旋转速度。对于一个升级的油漆稀释剂罐来说还不错。这些足够便宜，你甚至可以建造一个舰队。

对于任何对射电望远镜或太空感兴趣的人来说，这将是一个伟大的项目。然而，它是专门为课堂使用而设置的。学生们可以在数学、工程、编程，当然还有天文学和物理学方面灵活运用他们的技能。

我们已经看到[旧卫星 lna](https://hackaday.com/2016/09/28/listen-to-the-sun-saturn-and-the-milky-way-with-your-own-radio-telescope/)被改装成射电望远镜。如果你认为你没有地方放射电望远镜，[再想想](https://hackaday.com/2016/07/31/the-tiny-radio-telescope/)。
