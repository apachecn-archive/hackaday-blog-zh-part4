# 机器人和保龄球瓶

> 原文：<https://hackaday.com/2018/10/12/retrotechtacular-robots-and-bowling-pins/>

在最近的一次保龄球游览中，我们想到这是大多数美国人将直接与之互动的最先进的机器人系统之一。这在今天是一个大胆的主张，但在几十年前肯定是正确的。让我们漫步回到 1963 年，看看当时保龄球的艺术水平，AMF 自动保龄球手。

由于他们在工业自动化方面的基础，保龄球对于美国机器和铸造公司(AMF)来说是一个完美的问题。他们的业务始于 20 世纪初的自动化卷烟制造，二战后才开始关注保龄球瓶。这项挑战比你想象的要复杂得多，因为保龄球手被限制在一个狭窄的区域内，需要使用形状奇怪的球瓶、保龄球本身，还要处理设置新的框架，还要在第一次滚动后清理场地。

通过重力和摆动活塞将球从球销上分离，摆动活塞将错误的球销推回到传送带上。该传送带延伸通道的宽度，并将销钉移回销钉升降机，该升降机是一个垂直于地面移动的轮子，它将销钉定向并提升到一个旋转传送带上，该传送带可以将销钉放入设置夹具中，等待下一次完整的框架设置。

这部宣传片中的一切都有令人愉快的行话。我们特别喜欢机器如何“从钉板上清除朽木”的非机械性描述。我们可以整天观察这种自动化，事实上，在寻找的过程中，我们发现了一些其他的珍宝。这里是 AMF 82-70 的最新外观(与宣传片中的型号相同)。我们也想知道手动定位，并发现[这种手动机械辅助设置](https://www.youtube.com/watch?v=5NHm178lOpw)很有趣，尽管有音频。

令我们惊讶的是，我们以前在一篇回顾性文章中提到过 AMF。一旦他们的保龄球自动化开始起步，他们就把目光投向了餐厅自动化。看起来[布莱恩·本考夫参观机器人汉堡店](http://hackaday.com/2018/10/11/i-ate-a-robot-hamburger-before-the-restaurant-went-out-of-business/)实际上是一次复古体验！
