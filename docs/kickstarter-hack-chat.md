# Kickstarter 黑客聊天

> 原文：<https://hackaday.com/2019/08/05/kickstarter-hack-chat/>

加入我们太平洋时间 8 月 7 日星期三中午[的 Kickstarter Hack Chat](https://hackaday.io/event/165484-kickstarter-hack-chat) 与 [Beau Ambur](https://hackaday.io/hacker/179865-x) 和 [Clarissa Redwine](https://hackaday.io/hacker/134637-clarissa-redwine) ！

对我们许多人来说，神奇的事情就发生在我们的长椅上。我们混合一点这个，一点那个，和一些其他的东西，突然世界有了下一个大东西。是吗？它会重见天日吗？你会围绕你的项目建立一个社区吗，这样魔法就可以逃离商店，在市场的强光下存活下来？也许最重要的是，你有能力将你的项目推向市场吗？

众筹通常是这些问题以及更多问题的答案，而 [Kickstarter](https://www.kickstarter.com/) 是黑客可以将他们的项目转化为产品的地方之一。Beau 和 Clarissa 都是众筹公司的外联负责人，他们将在 Hack Chat 上回答你所有关于让你的项目离开工作台并进入市场的问题。加入我们的讨论，从建立一个对你的想法有足够热情的社区来资助它，到分享你的设计故事的正确方法。

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)中的社区直播活动。本周，我们将在太平洋时间 8 月 7 日星期三中午 12:00 坐下来讨论。如果时区让你烦恼，我们有[一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20190807T12&p0=224&msg=Kickstarter+Hack+Chat&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
