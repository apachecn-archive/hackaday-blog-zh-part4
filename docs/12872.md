# 3D 打印的 CPU 支架可降低温度

> 原文:[https://hack aday . com/2022/02/26/3d-printed-CPU-bracket-reduces-temperatures/](https://hackaday.com/2022/02/26/3d-printed-cpu-bracket-reduces-temperatures/)

当你的主板把你的 CPU 压弯了你会怎么做？如果你是[Karta]或[Luumi]，你 [3D 打印一个新的固定支架来解决问题](https://www.youtube.com/watch?v=IzBTyvaFYIg)。[Karta]首创了这一设计，[Luumi]也进行了尝试，并制作了您可以在下面看到的视频。

我们认为我们总是能找到平面，但实际上很难创造出真正平面的东西。当你试图最大化两个表面之间的热传递时，你通常会学到这一点。让两个本应平坦的表面接触是相当困难的。CPU 支架使用压力和某种热介质或浆糊的组合来填充 CPU 外壳和散热器之间的任何间隙。英特尔的 LGA1700 支架就是一个例子，但有一个问题。显然，对于最近的 CPU，支架有点太紧了，它弯曲了 CPU 的外壳。不伤 CPU，但确实抑制热传递。

其他人通过添加一些垫圈来稍微抬高支架，从而“解决”了这个问题。在这两种情况下，CPU 温度都有一些非常小的改善。[Luumi]说，部分问题是他的水冷块不完全平坦，需要研磨。然而,[Karta]报告说气温下降了 7 度，这是相当显著的。

我们喜欢看到 3D 打印如何修复或改善你所拥有的东西。他们在视频中谈论了很多关于研磨的事情，在某些情况下，人们实际上冒着[研磨 IC 芯片本身](https://hackaday.com/2020/07/21/die-lapping-for-better-cpu-performance/)的风险来使它变得更平。可以有帮助，但是风险比较大，收益比较低。
