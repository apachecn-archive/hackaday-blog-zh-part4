# 在没有试管测试器的情况下测试试管

> 原文:[https://hack aday . com/2022/07/03/testing-a-tube-without-a-tube-tester/](https://hackaday.com/2022/07/03/testing-a-tube-without-a-tube-tester/)

[M Caldeira]需要测试一个试管，但没有备用试管来做旧的交换测试。他手边也没有试管测试器。从 2015 年的一个视频中汲取灵感，他设法利用[工作台](https://www.youtube.com/watch?v=hD1lVtpNHHo)周围的东西拼凑出了一个*特设*试管测试器。你可以在下面看到这个过程的视频。

要重复他的努力，你需要几米。还好现在它们相对便宜。通常，电子管测试仪有一种方法来调整电子管的所有不同参数，但没有理由不使用测试平台电源来设置这些参数。

当然，这需要高压电源。你将无法使用 USB 端口来为大多数电子管供电。一个简单的测试板提供正确的偏置和连接电源的便捷方式。一个电源为加热器供电，而其他电源驱动其他电压。一个小型直流-DC 转换器从 12V 输入提供高压 B+线路。

观看电子管的运行当然是测试它的一种方法。试管测试员随处可见的日子已经一去不复返了。但是如果你真的需要测试很多试管的话，你还是可以在二手市场上找到一些有趣的。如果你真的有动力，你甚至可以[自己制作电子管](https://hackaday.com/2018/12/31/the-art-of-vacuum-tube-fabrication/)。
