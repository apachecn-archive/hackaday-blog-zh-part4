# 定制钻床工作台消除了麻烦

> 原文：<https://hackaday.com/2020/03/29/custom-drill-press-table-eliminates-hassles/>

用所有合适的工具打造一个完美的工作室是许多人的梦想。我们很多人用一个 dremel 工具，一个烙铁，以及厨房或备用卧室里任何可用的工作台面来拼凑东西。但是，即使我们最终找到了一个永久的车库或商店来工作，我们的工作空间仍然有一些挑战需要克服。[Workshop From Scratch]在他的钻床[上遇到了问题，并通过这个定制版本](https://www.youtube.com/watch?v=ABXhES_J0t8)解决了这些问题。

他没有改造现有的压力机，而是先用方管从头开始焊接了一张桌子。从那里，他开始着手解决这些问题。首先是在处理较大的工件时，必须上下做大量的调整。为此，他增加了一个电动可调操作台面，使他不必不断调整印刷机本身。对标准压机工作空间的第二个改进是为切割工具增加了一个冷却系统，节省了钻头费用，并加快了钻孔速度。

成品看起来很专业，这要归功于高质量的喷漆工作，当然，首先要有车间里所有合适的工具来组装这样的东西。我们脑子里都有一个满足自己需求的完美工作室的想法，但是当谈到自己建造一个工作室时，不要忘记跳出框框思考。
