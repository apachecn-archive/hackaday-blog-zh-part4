# 把 Arduino Enigma 放进你的口袋

> 原文:[https://hack aday . com/2019/03/28/put-an-arduino-enigma-in-your-pocket/](https://hackaday.com/2019/03/28/put-an-arduino-enigma-in-your-pocket/)

德国的 Enigma 设备一直是黑客们着迷的小玩意。我们已经看到了多年来创造的各种复制品和模拟器，最近它甚至成为我们每周黑客聊天的主题。但是如果你仔细想想，这并不是一个惊喜；英格玛完美融合了历史意义和工程魔法，还带有一丝健康的神秘感。为什么坏人总是有最酷的玩具？

[![](../Images/5beb1d0cebbcca6c23190865b8f0810c.png)](https://hackaday.com/wp-content/uploads/2019/03/arduenigma_detail.png) 如果你曾经想要[探索你自己的小谜复制品，【马克·库罗斯】已经组织了一个项目，使它比以往任何时候都更容易](https://github.com/mjculross/Enigma_Visual_Simulator)。事实上，它是如此的简单，以至于有些正在阅读这篇文章的人可能会在你阅读完这篇文章后，从你已经放在零件箱里的材料中拼凑出一个来。你需要的只是一个 Arduino Uno，一个 Adafruit 2.8”TFT 触摸屏，以及对二战技术的爱好。

多亏了分辨率相对较高的触摸屏，[Mark]能够为他的 Enigma 开发一个用户界面，让你真正感受到原始机器是如何工作的。显然，这是从现实世界的版本中大大简化的，但使用手写笔点击你想要旋转的转子或你想要插入的电线，比我们之前看到的许多尝试都更加身临其境。只需轻轻一点，你甚至可以加载历史机器配置，比如 1942 年盟军截获潜艇 U-262 上的密码信息时，该机是如何配置的。

[Mark]说这个项目总是关于开发软件，他把实际的硬件实现留给用户练习。仅仅是玩玩这个软件，把 Arduino 和触摸屏连接起来就足够了，但我们更希望看到有人真的接受这个想法并使用它。添加一些电池，一个充电电路，并把它们都放在一个小木箱里，以获得真正的英格玛外观。也不能忘记标志性的皱纹漆面。

多年来，我们见过各种形状和大小的英格玛机复制品。从可以戴在手腕上的，到使用现代组件的[全尺寸复制品](https://hackaday.com/2016/05/24/a-modern-but-classic-enigma-machine/)。我们甚至看到了一种变体[，你可以在几张纸上打印出来](https://hackaday.com/2016/04/02/paper-enigma-machine-2/)。娱乐游行没有停止的迹象，我们不会有任何其他方式。
