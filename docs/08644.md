# 钢筋混凝土割草机不受惩罚地削减

> 原文：<https://hackaday.com/2020/12/16/rc-lawn-mower-cuts-with-impunity/>

草是一种可爱的东西，但不幸的是，如果不加以注意，它会长得过长。显然，人们可以使用普通的割草机，但[rctestflight]的[Daniel]决定制造一种遥控的东西。(视频，下面嵌入。)

首先，[Daniel]将 X-acto 刀片安装到无刷 outrunner 电机上，测试它们割草的能力。由于对性能感到满意，他建造了一辆拖车来牵引安装在这种装置上的 RC 坦克，并取得了一些成功。随着这个概念开始开花结果，他采用了一种简洁的设计来获得最佳性能。

最终的建造依赖于钢筋混凝土岩石履带底盘，装有无刷电机，使用[磁场定向控制](https://en.wikipedia.org/wiki/Vector_control_(motor))在低速时获得最大扭矩。这使得遥控割草机能够缓慢地穿过草地，而不会压倒刀头。至于刀头，最终的装备有八个马达，每个都有两个刀片，可以不受惩罚地砍长草。

[Daniel]注意到以这种方式割草非常有趣，并且很惊讶在这一地区没有看到更多的建筑。(编者注:何[需](https://hackaday.com/2015/08/31/rc-lawnmower-has-no-grass-to-cut/)到[读](https://hackaday.com/2014/02/22/remote-controlled-lawn-mower-lets-you-sit-back-and-enjoy-the-show-2/) [更需](https://hackaday.com/2020/06/01/rc-lawn-mower-keeps-the-grass-greener-on-your-side-of-the-fence/) [Hackaday](https://hackaday.com/2010/05/03/remote-control-reel-mower/) 。)当然，[我们也看到了大量的自主构建。](https://hackaday.com/2019/04/23/mowerbot-keeping-the-lawn-in-check-since-1998/)
