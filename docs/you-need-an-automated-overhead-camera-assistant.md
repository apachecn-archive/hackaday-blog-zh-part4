# 你需要一个自动的头顶摄影助手

> 原文：<https://hackaday.com/2021/04/14/you-need-an-automated-overhead-camera-assistant/>

现在是 2021 年。每个人和他们的母亲都在拍摄自己做的事情，其中很多都是非常酷的内容。但是，由于我们大多数人还必须自己操作视频捕捉设备，所以很难用一个固定的俯视图制作引人注目的镜头，尤其是在涉及许多步骤的时候。滑块装备是一个好的开始，但以编程方式在三维空间移动相机的能力才是真正的优势。

[KronBjorn]出色的[自动头顶摄像助手](https://www.instructables.com/Automated-Overhead-Camera-Assistant-for-Instructio/)运行在 Arduino Mega 上，通过在串行监视器中键入命令来操作。它可以从直线向下平移 20°，并在 NEMA-17 步进电机上沿三个轴移动。你可以在休息后的视频中看到，它移动得非常平稳。最小整形设计很有趣，让我们想起了一个~~检眼镜~~ [综合屈光检查仪](https://en.wikipedia.org/wiki/Phoropter)——那是眼科医生的主要装备。只有一件事能让它变得更好，那就是一个专用的宏板。

如果你想建造自己的，你很幸运——这个项目有很多细节，包括完整的 BOM，所有的 STL，代码，甚至 3D 打印零件和电子设备的组装视频。滑过休息时间，观看几个简短的演示视频。

没有足够的空间来布置这样的场景？[试试受电弓版本](https://hackaday.com/2017/09/01/a-compact-portable-pantograph-camera-slider/)。
