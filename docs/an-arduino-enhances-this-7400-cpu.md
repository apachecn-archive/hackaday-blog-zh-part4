# Arduino 增强了这款 7400 CPU

> 原文：<https://hackaday.com/2019/09/22/an-arduino-enhances-this-7400-cpu/>

从零开始，用 74 系列逻辑制造一台完整的计算机要多快？理查德·格拉夫顿离英国复古计算机节只有 30 天了，他开始在这段时间里设计和制造他的剑桥 1 号计算机。结果是一台机器分布在几个试验板上，布线整齐，角落里出人意料地放着一个 Arduino 微处理器。小意板不是骗子吗？他说，事实并非如此，因为它不是计算机本身的一部分，而是作为一个程序加载器，使从个人电脑上下载软件变得尽可能容易。

机器本身很简单，4 位设计，8 位数据和地址总线。只有 16 条指令，时钟速度是比较行人的 40Hz。然而，这确实允许许多闪光灯以更清晰的方式显示机器的状态。我们在休息时间下面放了一个视频，如果你有更多的问题，你可以去看看 GitHub 库。

我们喜欢剑桥 1 号，我们认为 Arduino 作为它的一部分没有问题。这并不影响机器 74 驱动的本质。相反，它通过促进设备上的编码来增强设备的有用性。这些年来，我们这里有大量的 TTL 计算机，所以很难挑选一台送给你，但是你可能会想把 7400 作为系列中的原型机。
