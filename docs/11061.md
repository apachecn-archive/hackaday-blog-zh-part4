# 游戏机的加长豪华轿车

> 原文:[https://hack aday . com/2021/08/19/the-stretch-limo-of-game-boys/](https://hackaday.com/2021/08/19/the-stretch-limo-of-game-boys/)

在 Hackaday，我们看到了各种各样的项目，有些是出于一种深刻的需要或无法解决的渴望。其他的是从一个“为什么不”的世界中诞生的，也许正是这些项目让我们笑逐颜开。Retro Future 的[Elliot]的[wide boy Advance 就是这样一个项目](https://www.youtube.com/watch?v=NrDCjcAo9mk)。

从一个正常工作的游戏机和一个主板坏了的捐助者开始，科学怪人的出现就开始了。Dremel 将一个箱子一分为二，并移除另一个箱子的侧面，而可靠的旧车身填充物有助于填充和平滑缝隙。一个特别聪明的技巧是使用 Dremel 为填充物创造更容易附着的通道。有几个地方必须用填充物和粘在一起的塑料作为基底。正如你在下面的视频中看到的，无数个小时的打磨，涂底漆，打磨，再涂底漆导致了一个漂亮光滑的表面。紫漆的选择真的是卖了一个工厂的印象——清新的 Game Boy Advance。

将工作电路板脱焊，并将供体板切成片以适合延伸的边。使用一些电磁线，通过 PCB 上的测试点将连接桥接到原始主板。[Elliot]没有选择将屏幕换成 IPS 显示屏或添加背光。这些生活质量的改善是好的，但一个死亡赠品，任天堂没有做到这一点。目标是让用户想知道，即使只是一秒钟，如果任天堂只是碰巧制造了这种宽的一次性手持游戏机会怎么样。

[Elliot]制作它只是因为他觉得有趣，喜欢他制作的东西的形式。是黑客吗？是艺术吗？可能两者都有一点。这也不是他第一次修改任天堂掌机。他之前制作了一款[长的任天堂游戏机 DMG-01](https://www.youtube.com/watch?v=2zPehVx_J0M) 。我们喜欢看到所有野生黑客和调整游戏男孩线，如这个[游戏男孩颜色内的 DMG-01](https://hackaday.com/2021/08/06/game-boy-color-makes-itself-at-home-in-a-dmg-01/) 。
