# 铰链给柔性 PCB 带来新的意义

> 原文:[https://hack aday . com/2020/06/26/hinge-bring-the-new-mean-to-flexible-PCB/](https://hackaday.com/2020/06/26/hinge-brings-new-meaning-to-flexible-pcb/)

柔性 PC 板可以弯曲不是秘密。但是尽管基底具有弹性，你也不能真的把它们完全折叠起来。这让[Carl]很困扰，所以他开发了一种铰链设计,这样他就可以将木板完全对折。你可以看一个视频，展示一个例子，FlexBox，如下。

普通电路板可以折叠，但铜走线不能承受非常小的弯曲半径。【卡尔】的诀窍就是让折叠部分一点痕迹都没有。只有一个小桥承载两部分之间的走线，它可以像互连电缆一样弯曲。

的确，也可以使用两个 PCB 之间的板级互连，但将整个组件作为一个整体还是很有吸引力的，尤其是在使用柔性 PCB 的情况下。我们没有任何数据，但我们认为柔性 PCB 比插座和插头更可靠，并且不会增加额外成本。

[卡尔]是柔性板的大师。他制作了关于使用加强筋的混合柔性/刚性 PCB 的[廉价原型](https://www.youtube.com/watch?v=P51te12Oux4)的视频。他将它们用于[跳跃机器人](https://hackaday.com/2020/03/31/magnets-turn-flexible-pcb-into-electric-grasshopper/)和[某种全息光学显示器](https://hackaday.com/2020/02/28/latest-flexled-milestone-refines-the-pov-display/)。如果你想了解更多关于柔性印刷电路板的想法，可以看看去年柔性印刷电路板大赛的结果。
