# Blimpduino Hits 版本 2

> 原文：<https://hackaday.com/2018/11/27/blimpduino-hits-version-2/>

我们总是认为乘坐软式飞艇穿越大西洋会非常平静——至少一旦他们在船上安装了加热器。兴登堡号、R-101 号和谢南多号终结了飞艇的时代，至少对于商业客运来说是这样。但是你仍然可以用一个氦气球和一些电子设备自己飞行。一个著名的项目——Blimpduino——已经发展成了 [Blimpduino 2](http://www.blimpduino.com/) 。开源软件在 [GitHub](https://github.com/mRoboticsIO/Blimpduino2) 上。我们找不到 PCB 布局，所以我们不确定它是否会打开。不过，[的 3D 打印部件](https://www.thingiverse.com/thing:3093187)已经上市。

PCB 是问题的核心，它是一个四层板，带有 ARM M0 处理器、ESP8266 WiFi 模块、四个电机输出、两个伺服电机输出、一个 9 轴惯性导航系统、一个高度计和一个前方物体检测系统。车上还有一个电池充电器。

标准设置使用三个道具:两个用于推力，一个用于高度。有一个智能手机应用程序，显然，你甚至可以用第二部手机拥有一个副驾驶。升力体是一个带有氦气的聚酯薄膜气球，他们说这种控制甚至适用于一个非常大的气球。高度计数据来自飞行时间传感器，如果你想测量更高的高度，还有一个带温度传感器的压力传感器。

我们无法嵌入视频，但网站上有一个人在传东西。不过，你可以在下面看到一段宣传视频。正如你所料，有效载荷能力非常低，因此 3D 打印零件的填充量很低，电路板做得很轻。

当然，我们自己的[索菲·克拉维茨]已经建造她的无人驾驶软式飞艇部队有一段时间了，我们现在随时都在等待她统治世界的企图。如果你想玩些科技含量较低的东西，你可以拆开[一辆玩具车，装上自己的气球](https://hackaday.com/2013/11/16/13-homemade-rc-blimp/)。
