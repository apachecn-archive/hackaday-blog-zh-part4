# 珠穆朗玛峰灯重现著名的高峰

> 原文：<https://hackaday.com/2019/10/05/mt-everest-lamp-recreates-the-famous-peak/>

任何去过遥远山峰的人都会惊叹于自然、崎岖地形的美丽。也没有什么不同，他创造了一盏灯来表达对地球上最大山峰的敬畏。

当要复制一个精确的地貌几何图形时，最简单的方法是使用各种数控机器。在这里，3D 打印机被用来创建一个半透明的外壳复制山。然后在底部系上一个反向的贝壳，创造出一种类似于湖中倒影的效果。该组件安装有由 Arduino Nano 运行的 WS2812 LEDs，并悬挂在不锈钢框架上，具有吸引人的漂浮外观。

这是一件很棒的作品，放在任何桌子或咖啡桌上都会令人印象深刻。在生产有吸引力的家居照明时，3D 打印机为许多制造商提供了很好的服务。我们已经看到了许多伟大的构建——从 [8 位](https://hackaday.com/2019/06/02/animated-pixel-lamp-is-a-must-for-any-chiptuners-bedroom/)到[受花卉启发的](https://hackaday.com/2016/10/11/blooming-flower-lamp-will-test-your-3d-printer/)。休息后的视频。
