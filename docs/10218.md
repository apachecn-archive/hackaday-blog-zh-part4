# 薄涂层需要大量的设备和技术

> 原文：<https://hackaday.com/2021/05/31/thin-coatings-require-an-impressive-collection-of-equipment-and-know-how/>

老实说，我们中没有多少人需要以可控的方式在基底上沉积纳米厚的薄膜。但是如果你真的发现自己处于这种情况下，你可以做得比跟随[耶鲁安·弗勒格尔]的领导更糟，他建造了一个物理气相沉积设备来做这件事。

令人欣慰的是，[耶鲁安]在这方面有特殊的专长，并愿意分享。PVD 用于在基底上涂上一层极薄的金属或有机材料——想想透镜涂层或镜面镀银，以及半导体制造。该方法包括在真空中加热涂层材料，使得涂层材料以受控的方式蒸发并积聚在基底上。听起来很简单，但实际完成它所需的设备和技术令人望而生畏。[耶鲁安]的购物清单包括加热涂层材料的大电流电源，抽空涂层室的涡轮分子泵，以及监控涂层室内条件的仪器。大部分焊接室都是自制的，对于一个初学 TIG 焊接的人来说，这是一个勇敢的举动。下面的视频展示了制作过程中的亮点，也展示了 PVD 在玻璃盘上镀上一层薄薄的银。

这座建筑充满了精美的细节；我们特别喜欢通过测量连接到室内晶体的振荡器的频率变化来监测沉积过程的技术，因为它积累了成本材料。我们不确定[耶鲁安]对此有何看法，但我们怀疑这与他在谈论他的光学逻辑门实验时给出的一些暗示有关。我们期待看到这是不是真的。
