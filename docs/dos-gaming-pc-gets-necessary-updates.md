# DOS 游戏电脑获得必要的更新

> 原文：<https://hackaday.com/2021/04/17/dos-gaming-pc-gets-necessary-updates/>

PC-104 是一种标准的计算机外形，工业环境之外的大多数人可能从未见过。它本质上是一个英特尔 486 处理器，支持大多数计算机早已消失的标准，但这使它在两件事情上非常棒:控制旧的工业设备和在本机硬件上运行经典的 DOS 游戏。对于后者，我们再一次转向[Rasteri]，他在之前的基础上改进了一个更小的 DOS 游戏装备，[这次基于一个比 PC-104](https://www.youtube.com/watch?v=aJEp4ZUG7BI) 更小的平台。

像这样的构建的关键是它需要对早已过时的 ISA 总线的本机支持，以便能够与 SoundBlaster 卡接口，sound blaster 卡是那个时代视频游戏的黄金标准。这台更小的计算机在一个更小的封装中仍然有这个功能，但是有一些重大的改进。首先，它有一个浮点单元，所以它可以运行像雷神之锤这样的游戏。它也比 PC-104 系统快得多，而且耗电更少。最后，它适合一个更小的盒子。

这种构建不仅仅是在 SoM 计算机上运行软件。[Rasteri]还为这个项目定制了一个接口板，配有所有必要的端口和一个 ISA 声音芯片，同时保持尺寸最小。新版本也让他给这个版本起了一个比旧版本更好的名字(尽管他对这个升级的表述略有不同)，并且也让他在将来扩展一些功能。如果你也是这个传奇故事新手，一定要去看看第一个建筑。
