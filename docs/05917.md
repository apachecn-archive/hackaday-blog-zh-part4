# 提醒不要摸你的脸

> 原文：<https://hackaday.com/2020/03/09/a-reminder-not-to-touch-your-face/>

2020 年，全世界都在关注一种名叫新冠肺炎的新病毒的猖獗传播。像许多传染病一样，良好的卫生习惯可以减少传播。为了帮助战斗，尼克·比尔德组装了一个他称之为“哨兵”的装置。

概念很简单。当检测到这种行为时，通过发出警告灯来减少用户触摸自己的脸。这是通过使用 Arduino 来实现的，Arduino 通过超声波近程传感器的反馈来控制 LED。LED 被放置在用户的周边视觉中，当传感器检测到手(或其他物体)接近面部时会发光。

虽然不太可能大规模推广，但这个项目提醒我们要练习良好的日常自我保健。正如谚语所说，预防胜于治疗。当政府和行业都在努力解决当前的问题时，想想你的供应链会如何暴露在危机中。休息后的视频。
