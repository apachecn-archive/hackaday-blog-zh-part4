# 黑客日播客 024:捣碎智能手机按钮，声爆，垃圾打印，和一个可笑的织机

> 原文：<https://hackaday.com/2019/06/21/hackaday-podcast-024-mashing-smartphone-buttons-sound-blastering-trash-printing-and-a-ludicrous-loom/>

Hackaday 的编辑埃利奥特·威廉姆斯和迈克·斯奇斯(Mike Szczys)介绍了本周有趣的黑客活动。看起来谷歌被抓到剽窃歌词(他们是如何被抓到的是黑客)，电动汽车变得人为噪音更大。我们直接从使用过的塑料中进行 3D 打印，并用数百个 3D 打印零件建造一台织机。声霸 1.0 的重生要归功于一些(解释得很好的)逆向工程电路。你的智能手机将获得更多无需任何额外电子设备就能工作的按钮，我们将完成黄铜蚀刻和可下载的核反应堆计划。
