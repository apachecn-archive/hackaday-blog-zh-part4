# 50 岁的 8 轨换轨器修理和黑客

> 原文：<https://hackaday.com/2022/07/25/50-year-old-8-track-changer-repair-and-hack/>

出于尚不清楚的原因，[Techmoan]已经采购了一个 RCA 8 轨道转换器，它可以在一个定制的载体中容纳五个磁带盒。它有点工作，但有一点机械问题，这里和那里需要一些维护。此外，播放器是为美国市场和 60 赫兹电源设计的，但[Techmoan]在英国是 50 赫兹。

虽然基本的磁带放音机部分使用了电子设备，但其他部分都是由机械齿轮、杠杆和马达操作的。系统播放每个磁带盒的两面，直到播放完毕，然后自动切换到堆栈中的下一个磁带盒。每个弹夹可以长达 90 分钟，播放时间超过 7 个小时。奇怪的是，在第五盘磁带结束后，系统不会自动重复，这需要操作员干预。尚不完全清楚这些旋转木马主要是为了在企业内部播放背景音乐，还是为利基消费者应用而建。

在发现没有设置来调整 50/60 Hz 运行的磁带速度后，[Techmoan]可以为电机驱动轴订购或制造一个更大直径的皮带轮。但在真正的黑客风格中，他用透明胶带解决了问题。通过反复试验，他通过缠绕一定长度的胶带来增大滑轮直径，直到音乐听起来“足够好”为止。然后，他拿出“哇”和“颤动”测量仪，开始真正地调零——并让它开始工作。他说未来的视频需要这种改变，所以我们期待看到它将如何被采用。
