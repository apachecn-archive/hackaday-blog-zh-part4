# 炮弹模具为激光测量制造了一个漂亮的积分球

> 原文：<https://hackaday.com/2022/02/06/cannonball-mold-makes-a-dandy-integrating-sphere-for-laser-measurements/>

这是一个古老的谜:如果你有一个完美的球体，其内表面具有完美的反射性，光会永远在里面反弹吗？当你仔细思考时，答案是显而易见的，但这并不意味着你不能将这一原理付诸实践，正如我们看到的[这个自制的乌布里切特球体](https://www.youtube.com/watch?v=BtMqx5P74n4)用于光学测量。

如果你从未听说过 Ulbricht 球，不要担心——它也被称为积分球，这使它的功能更加明显。正如[Les Wright]解释的那样，积分球是一种光学元件，它有一个中空的球形空腔，表面涂有漫反射涂层。球体上有两个端口，一个用于接收光线——通常来自激光器——另一个用于发射光线。光线在球体内部来回反射，形成完美的漫射，并在出口处形成均匀的光束。

[Les]“需要一个积分球是因为他想用他的基于树莓π的光谱仪来测量他的一些激光器的输出。与其花钱买一个昂贵的商业积分球，或者在车床上转动一个，Les 转向了一个不太可能的来源:炮弹模具。模具内部涂有一种同样不可能的超白涂料，由硫酸钡和聚乙烯醇胶混合而成。通过在模具中加工几个端口，它可以完美地漫射他的染料激光器发出的光，以进行适当的测量。

激光可能是一个昂贵的爱好，但[Les]似乎总能找到一种方法，让东西更实惠，而且一样好。无论是自制的高压电源门把手帽还是 T2 用廉价的 CCD 相机拆除拜耳滤镜，他似乎总能找到办法。
