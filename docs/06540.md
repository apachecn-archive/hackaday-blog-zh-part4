# 橱窗里的那个绘图仪多少钱？

> 原文:[https://hackaday . com/2020/05/14/橱窗里的绘图仪多少钱/](https://hackaday.com/2020/05/14/how-much-is-that-plotter-in-the-window/)

我们确实生活在一个奇怪的时代。曾经回避与人类直接互动的人们现在渴望这种互动，但效果有限。在为数不多的几家商店中，几乎每个收银员都被密封在塑料保护罩后面，这意味着逗留的时间越少越好。把一个内向的人变成一个外向的人就足够了，至少在障碍消失之前。

我们认为，在[【尼克拉斯·罗伊】的“请留言”](https://www.niklasroy.com/pleaseleaveamessage/)背后，隐藏着伸出手去触摸某人的需求，这是他在柏林店铺前窗设置的一个互动艺术装置。他的商店位于市中心的一条街道上，地理位置优越，吸引了大量的行人，他的展示设计吸引了人们的眼球，或许还能让人会心一笑。该设备包括一个大型木制画架，上面放着一台旧的 X-Y 笔式绘图仪、一台 Arduino 和一台 ESP-8266，以及绘图仪步进器的几个驱动器。路人被鼓励扫描二维码，进入由 ESP-8266 提供的网页，在那里他们可以输入简短的信息。绘图仪尽职尽责地在一卷纸上拼写出来，让所有人都可以看到，使用了非常好的字体，[Niklas]设计成既可读又容易绘制。下面的视频展示了它与真人的行动；这似乎很受欢迎。

[Niklas]非常多产，我们已经报道了他的许多互动艺术装置。只要搜索他的名字，你就会找到一切，从压力清洗机跳舞水显示器到加大号弹球机。
