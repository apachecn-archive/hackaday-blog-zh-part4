# 你能帮助 3D 打印一个电动球吗？

> 原文：<https://hackaday.com/2020/01/26/can-you-help-3d-print-a-selectric-ball/>

IBM 电动打字机改变了我们所知道的打字机。他们独特的球元素取代了笨重的打字排，使大多数人打字更快。当[史蒂夫·马利克夫]想到用 3D 打印一个铅字球时——俗称高尔夫球——这似乎是一个很棒的主意。

问题？就是不太好用。根据[Steve]的说法，很可能是因为打印机的分辨率太低。然而，不清楚这些字符的纬度是否正确。还有一些其他的问题。树脂打印机可能会做得更好，有人呼吁在那里尝试一下，并报告回来。我们猜测一个更好的喷嘴和非常低的层高度可能有助于 FDM 打印机。

从图像来看，看起来有些球做得相当好，但在倾斜角度下没有完全命中。所以可能是别的原因。然而，这听起来像清理打印，所以它适合是一个主要问题。

Selectric 的出名有几个原因——你可以在下面的视频中看到这款机器的广告。铅字球意味着你不能卡住钥匙。因为你不需要拆下钥匙，而且墨盒里有色带，所以你必须非常努力才能让手指沾上墨水，即使你用的是布带而不是可能更好的碳膜色带。Selectric II 甚至可以使用一种特殊的胶带将碳带从纸上撕下来，以纠正错误。没有白色液体，也没有用小纸条涂改液。球移动的事实意味着你不需要在机器的侧面清理出空间来让压盘来回移动。

你能帮忙吗？如果你有一台 Selectric I 或 II 和一台高质量的打印机，这将是一个有趣的项目，可以尝试向[Steve]报告你的结果。如果你熟悉后来发行的铅球，你可能没有见过[史蒂夫]用来固定球的金属线夹。但是，你可以在下面的视频广告中看到它们。更现代的球使用塑料杆作为手柄，所以即使有布带，你也不太可能把墨水弄到手上。

虽然有一些选择器是用来与计算机接口的，但是你可以[改装其中的任何一个来做一些工作。电动自行车还在有史以来最伟大的科技间谍故事之一中扮演角色:](https://hackaday.com/2012/06/13/turning-an-ibm-selectric-into-a-printer/)[枪手计划](https://hackaday.com/2015/11/06/retrotechtacular-cold-war-era-hardware-keyloggers/)。
