# 技术革新:Amiga 为 NASA 的关键任务计算提供个人电脑

> 原文:[https://hackaday . com/2021/08/16/retro tech tacular-amiga-pips-the-PC-for-mission-critical-computing-at-NASA/](https://hackaday.com/2021/08/16/retrotechtacular-amiga-pips-the-pc-for-mission-critical-computing-at-nasa/)

1986 年，一群美国宇航局的工程师在解决他们的数据处理困境时面临着一个艰难的选择:继续忍受 PC 体系结构的低性能，还是支付现金购买异国情调的工作站。事实证明，准将阿米加是一个有趣的第三选择，除了一个事实，矛盾的是，[它的价格不够](http://obligement.free.fr/articles_traduction/amiganasa_en.php)。哦，苹果不想与此有任何瓜葛。

历史悠久的美国宇航局 AE 机库是运载火箭遥测和其他任务通信的枢纽，主要是在卡纳维拉尔角发射的发射前阶段。在整个 20 世纪后期，AE 机库支持 NASA 各种形状和大小的运载火箭，从 Atlas-Centaur 进化到强大的 Titan 家族。它甚至支持来自航天飞机项目的用户数据。来自这些任务的遥测数据在发送给美国宇航局的其他研究人员之前，在 AE 机库进行处理，甚至发送到世界各地的其他参与太空机构。

从几十年天文数字般的资金水平下来，20 世纪 80 年代是勒紧裤腰带的年代，NASA 需要预算解决方案，而不是在任务安全上吝啬。事实证明，Commodore Amiga 是处理运载火箭遥测技术的正确选择。因此，当阿米加亚特兰大集团的摄像机获准在 AE 机库内拍摄时，一切都静止了。

下面的视频拍摄于 1998 年，即第一台 Amiga 计算机在 AE 机库安装十多年之后。听说苹果 Macintosh 是计算硬件的首选，这很有趣(也不奇怪)。然而，作为一个封闭的系统，工程师们无法以所需的级别访问 Macintosh，也无法开发支持其操作所需的定制硬件。相比之下，准将更愿意给 NASA 送去一大堆文件来帮助他们。他们真好！

时任美国国家航空航天局首席系统工程师的盖瑞·琼斯继续说，阿米加对他的雇主来说是一个不受欢迎的选择。“他们希望我们购买个人电脑，运行 Windows 95 和 nt。我们一直试图告诉他们速度不够快，所以他们告诉我们买 DEC Alphas。我们告诉他们太贵了。他们不喜欢阿米加；还不够花。”呀。

该视频发生在 STS-89 及其前往和平号空间站的任务期间。看起来有些已经被时间遗忘了，然而[一篇旧的博客文章填补了一些知识空白](http://obligement.free.fr/articles_traduction/amiganasa_en.php)，它很值得一看，并且本身就是复古的。报告显示，这些机器直到 2006 年还在使用，其中一台[实际上是在不久前出售的](https://www.popularmechanics.com/space/a14390318/amiga-nasa-ebay/)。

所以有句顺口溜，“只有阿米加让它成为可能”。
