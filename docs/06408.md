# 用卷尺时钟观察一天

> 原文：<https://hackaday.com/2020/04/27/watch-the-day-inch-along-with-a-tape-measure-clock/>

如果我们让你快速背出所有你可以随意使用的工具，你可能会把你的钟表从清单上去掉。但是我们说时钟绝对是工具——以无数种形式出现的酷工具，给无尽的日子赋予了意义。

我们没有想到的时钟形式是一个实际的工具。所以当我们看到用卷尺制作的[【scealux】的时钟时，我们无比欣喜。至少，钟的报时部分是用卷尺制成的。这个箱子并不是真的用卷尺测量的——它完全是印刷的，粘合的，打磨的，并且涂得非常好，很容易让人误认为是真的。](http://www.instructables.com/id/Time-Measure-Tape-Measure-Clock/)

在这件功能艺术品中紧密包装的是 Arduino Nano 和 DS3231 precision RTC 模块，我们认为它非常适合基于工具的时钟。纳米获取时间，并驱动一个步进电机，只是勉强适合内部。有足够的胶带缠绕在印刷的轮毂上，以每英寸一小时的增量测量时间。花 1/16 英寸左右，休息后观看演示和简短的演示视频。

不是所有的工具都是锋利的，也不是所有的钟表都是精确的。这里有一个《泰晤士报》的时钟，给你要点。
