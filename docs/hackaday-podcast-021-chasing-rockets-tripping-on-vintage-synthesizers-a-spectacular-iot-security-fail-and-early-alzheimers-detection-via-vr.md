# 黑客日播客 021:追逐火箭，被老式合成器绊倒，一次壮观的物联网安全失败，以及通过虚拟现实进行的早期阿尔茨海默氏症检测

迈克·斯奇斯(Mike Szczys)本周正在享受他应得的假期，所以特约撰稿人丹·马洛尼(Dan Maloney)和执行主编埃利奥特·威廉姆斯一起看看本周所有伟大的黑客。在这一集里，我们将讨论 MIDI 2.0 的许可费用，窥探员工时的双向失败，以及用虚拟现实诊断阿尔茨海默氏症的潜力。我们还深入探讨了机器猎豹的精心设计的内部结构，这是一个对孩子来说足够安全的个人助理，以及如何通过监听你的监视器来揭示比你想象的更多的关于你的信息。你不会想错过一个太空迷对火的探索或一个黑客对焊接和锡焊的指导。你必须抓住一个倒霉的黑客从老式合成器接触高的故事。这是一次不寻常的旅行。
