# 不用看就能收到你的推文

> 原文:[https://hack aday . com/2018/12/22/get-your-tweets-without-looking/](https://hackaday.com/2018/12/22/get-your-tweets-without-looking/)

头戴式显示器从笨重到有点像玻璃洞。智能手表有其独特之处，但它们仍然会将你的视线从你正在做的事情上移开，比如开车。语音助手可以为你朗读，但它们需要一个车里其他人都必须听的扬声器，或者一个屏蔽重要声音的耳机。忽略收到的信息是不可能的，所以答案可能是使用不同于视觉的感觉。脸书公司和麻省理工学院的一个联合项目提出了一个解决方案，利用前臂的体感接收。

几年前，一个类似的想法出现在我们的办公桌上，看起来很有希望，但很难推销比当前技术更困难的东西，即使从长远来看是有利的。2013 年，一名佩戴者的背部被振动器马达覆盖着[，它的行为就像是频谱分析仪的触觉版本。现在，振动器的数量已经减少，通过利用图案来安装在套筒下。它是为有听力或视力障碍的人开发的，但哪些司机在看手机时不会受到影响？](https://hackaday.com/2013/12/09/future-tech-and-upgrading-your-brain/)

模式是这个版本与众不同的地方。这 39 个英语现象被赋予了一个独特的振动序列，足以对任何单词进行编码，而不是在手指上传递一个离散的音符，或在背后传递一系列数值。一个~~现象~~音位是最小的语音单位。下面的视频展示了这些音素是如何转化为触觉反馈的。希望，我们可以[发送推文](https://hackaday.com/2018/09/14/speech-recognition-without-a-voice/)而不用我们的手或嘴升级到完全的心灵感应。
