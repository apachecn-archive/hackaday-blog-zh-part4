# [本·克拉斯诺]建造了一个单分量干涉仪

> 原文:[https://hack aday . com/2018/12/13/Ben-krasnow-builds-a-one-component-interferometer/](https://hackaday.com/2018/12/13/ben-krasnow-builds-a-one-component-interferometer/)

当我们想到物理实验时，我们倾向于想象洞穴般的房间，充满了像光学工作台，真空室中的巨大线圈，以及一架又一架的放大器和数据采集硬件。但事实并非如此——你实际上可以用单个组件进行激光干涉测量，测量亚微米位移等等。

下面[Ben Krasnow]视频的精明观众会注意到，为了将激光二极管这一组件用作干涉仪，他需要一整套支持设备，如电源、信号发生器和非常非常好的混合信号示波器。但实验的原理是重要的一点，它使用一个内置监控光电二极管的激光二极管。旧的激光二极管通常使用这些光电二极管来控制激光结发出的光。但它们也会对反射回激光二极管的光做出反应，由于相长干涉和相消干涉，它们实际上可以产生与反射器的微小位移相对应的信号。[Ben]用它来测量小型扬声器的振动、电机轴的旋转，并在设置上稍加改变，以亚微米精度测量固定目标的范围。这是令人着迷的东西，事实上，你可以从一个组件中提取如此多的信息是非常酷的。

我们真的很喜欢[本]的演讲风格，以及他找到方法探索的有趣的物理小角落和缝隙。他最近研究了氦如何杀死 MEMS 传感器，这是一个同样有趣的话题。
