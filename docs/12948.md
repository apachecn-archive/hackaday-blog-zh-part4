# 苹果试图在 45 年后为家用电脑申请专利是否为时已晚？

> 原文:[https://hackaday . com/2022/03/04/are-apple-试图给家用电脑申请专利-45 年太晚了/](https://hackaday.com/2022/03/04/are-apple-trying-to-patent-the-home-computer-45-years-too-late/)

在我们最近纪念 Raspberry Pi 10 周年的文章中，我们称赞了他们的一体式 Raspberry Pi 400 电脑，因为迄今为止它没有成功吸引任何竞争产品。这种评估似乎还为时过早，因为有消息称，苹果公司已经提交了一项专利申请，申请专利的是一种输入设备中的[计算机，它看起来非常像 Pi 400。事实上，我们会走得更远，它看起来非常像过去的许多经典家用电脑，以至于我们想知道苹果到底认为什么是足够新颖的专利。](https://appft.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&u=%2Fnetahtml%2FPTO%2Fsearch-adv.html&r=42&p=1&f=G&l=50&d=PG01&S1=(apple.AANM.+AND+20220224.PD.)&OS=aanm/apple+and+pd/20220224&RS=(AANM/apple+AND+PD/20220224))

[![A Raspberry Pi 400 all-in-one keyboard console computer](../Images/5f060d969005445ecd8f708ba29dee7f.png)](https://hackaday.com/wp-content/uploads/2020/11/DSCF1980.jpg)

Looks pretty similar to us.

阅读该专利，它似乎是一个适用于多合一计算机的透明总括，可能的例外是“*一个单一输入/输出端口*”，这意味着设备上的唯一端口将是一个单一的 USB-C 端口，可以获取电源，与外围设备通信，并驱动显示器。无论如何，这似乎是一个非常弱的新颖性声明，如果只是因为我们认为一些最近的带有键盘的 Android 手机可能构成现有技术。

我们确信苹果的律师们已经准备好了他们的论点，但是我们不禁想知道他们是否已经陷入了一个老笑话，即苹果粉丝声称该公司发明了一些东西，而事实上他们在竞争多年后终于采用了它。

回想一下 8 位计算机的辉煌时代，我们很好奇哪一个是第一个拥有比键盘稍大的外形的计算机。苹果自己的苹果][不算，因为机器的大部分在键盘后面，但例如 Commodore 的 VIC-20 或 Sinclair 的 ZX 光谱等机器可以说是一体化键盘电脑。有人能提供比那两个更早的一体机型号吗？

如果你对一体机感兴趣，你可以阅读[我们的 Raspberry Pi 400 评测](https://hackaday.com/2020/11/02/new-raspberry-pi-400-is-a-computer-in-a-keyboard-for-70/)。
