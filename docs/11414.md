# 制作金属门把手

> 原文：<https://hackaday.com/2021/09/24/making-a-metal-hand-doorknob/>

普通的门把手因其简单而广受诟病，但总的来说，社会有太多其他问题，以至于它从未真正出现在日常对话中。然而，不要害怕，因为[马修]创造了一个更特别的东西:一个门把手，形状像他自己伸出的手。

这个建筑的灵感来自芝加哥 WNDR 博物馆的一个类似的门把手，你也可以再造一个。这是通过一个多阶段的模具制作过程实现的。[马修]的第一步是用完美的藻酸盐材料做一个他的手的柔性模具。

一旦凝固，[马修的手]被删除，模具充满了蜡。[马修]的手的蜡复制品然后被用来创造一个铸造金属的熔模石膏模型。模具中每个指尖的末端都添加了通气孔，以使熔融金属有效地填充整个空腔。

一旦熔模变得坚实干燥，蜡就被熔化掉，可以进行铸造了。丙烷炉用于熔化铸造金属，并使用简单的重力铸造方法填充模具。[马修]最后做了两只手，一只铝制的，一只铜制的。用研磨机和钢丝轮做了一些清理，一只手的复制品出现在他手中！

成品看起来非常适合挂在门把手上，我们相信每次开门时和你厚重的金属握手是非常令人满意的。值得注意的是[同样的技术也可以用于 3D 打印](https://hackaday.com/2021/06/25/casting-a-simple-3d-print-in-aluminium/)！[如果你完成了自己的家庭演员项目，一定要给我们写信。](http://hackaday.com/submit-a-tip)休息后的视频。
