# 死虫 Arduino 活泼，盾兼容

> 原文：<https://hackaday.com/2019/06/09/dead-bug-arduino-is-lively-and-shield-compatible/>

Arduino UNO 等微控制器演示板在 Hackaday 上无处不在，是许多项目的大脑，这些项目不可避免地会做出一些令人印象深刻或不寻常的事情。有时有人会构建一个特别小的演示板，或者一个非常大的演示板。在这里介绍的电路板的例子中，Arduino 是一个华丽的爱的劳动，它不能真正被称为电路板，因为没有 PCB。代替传统的玻璃纤维，【jiří·普劳斯】[在电路中形成黄铜棒](https://www.hackster.io/jiripraus/a-skeleton-arduino-uno-89bdd6)并用焊料将它固定在一起。

这种对项目的专注会给人留下印象。他的笔记显示，他看到了操作 ATMega328 的最简单的方法，建造了它，测试了它，然后转移到电源上，使它能够自我维持，然后转移到通信电路上，最后是灯。下面的视频显示了一个功能齐全的 Arduino 愉快地运行着 blink 程序。他计划将黄铜部分包裹在树脂中以增强其韧性，大概可以防止每次撞击造成短路。由于使用了定制的夹具，这些组件处于相同的位置，这意味着标准的屏蔽罩将正好安装到位。

Arduino 开始时远没有现在这么华丽，但也几乎一样脆弱，而且它已经成长了。[又缩小了](https://hackaday.com/2014/10/22/now-lets-see-the-worlds-largest-arduino/)。
