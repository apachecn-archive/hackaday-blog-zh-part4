# 变形键盘让你拨入恰到好处

> 原文：<https://hackaday.com/2022/12/21/morphing-keyboard-gets-you-dialed-in-just-right/>

所以你厌倦了矩形的、砖墙交错的键盘，想要去分离和/或因此。但是怎么做呢？哪种风格？做什么？你要做的是:你[在这里建造一个 LHM 变形板](https://www.youtube.com/watch?v=j5VzlZ-F2JQ)并定制它，因为这就是它的用途。

这到底是什么东西？它是一个键盘吗？嗯，只要你能按下开关并向计算机发送键盘命令，它对我们来说就一定闻起来像键盘。现在我们已经解决了这个问题，这里发生的事情是[LifeHackerMax]已经建立了一个高度可定制的 LHM 版本，他们的 26 键分裂。LHM 变形可以微调到几乎任何可以想象的程度，包括隆起的角度。按键按模块分组，可以前后滑动，以适应不同的手指长度。由于它们是半圆形，这些模块也可以倾斜和旋转，直到它们刚刚好。

但是 LHM 变形的超级酷之处在于它的组合方式——就像乐高一样。它是完全模块化的，如果您没有准备好，甚至不必拆分。但是所有的部分都通过铜线杆连接起来。如果你想给自己做一个，[3D 文件在 Thingiverse](https://www.thingiverse.com/thing:5682866) 上，[固件在 GitHub](https://github.com/lifehackermaxyt/LHM-Morph) 上。休息后请务必观看视频。

这个键盘让你想起什么了吗？或许是“挤压箱”？
