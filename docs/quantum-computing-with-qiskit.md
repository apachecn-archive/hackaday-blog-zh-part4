# 使用 QISKit 的量子计算

> 原文：<https://hackaday.com/2019/04/02/quantum-computing-with-qiskit/>

我们都知道量子计算要来了，但是很难知道如何入门。[Mtreinish]建议使用 [Qiskit](https://qiskit.org/) —一个 Apache 许可的 SDK，用于开发量子应用程序。他在新加坡做了一个演示，你可以在下面看到，还有一个笔记本，你可以在 [GitHub](https://github.com/mtreinish/qiskit-workshop) 上浏览。如果你不耐烦，你甚至可以通过谷歌在线运行[笔记本。](https://colab.research.google.com/github/mtreinish/qiskit-workshop/blob/master/Workshop.ipynb)

这些工具可以针对几个后端工作，包括模拟器或 IBM 提供的真实硬件。官方网站有[一个不同的笔记本，你可以用它作为教程](https://mybinder.org/v2/gh/QISKit/qiskit-tutorial/master?filepath=index.ipynb)。有趣的是，所有 Qiskit 程序的基础是“Terra”(地球)，渗透所有 Qiskit 元素的是 Aer 或 air。还有火元素和水元素。在官方笔记本的底部，你会发现很多社区笔记本，深入到特定的主题。

这种笔记本是一个很好的例子，可以用一种有文化的方式来分享计算。我们同意[Mtreinish]的观点，有一个模拟器是很好的，但真正令人兴奋的是使用 IBM 通过互联网提供的真实设备之一。

但是，模拟器确实有一定的魅力。我们喜欢《怪癖》,去年我们就此制作了一个三集系列。您可能会发现这篇文章对学习基本转换和其他基本概念很有帮助。
