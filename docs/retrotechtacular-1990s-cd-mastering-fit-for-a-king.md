# 复古:20 世纪 90 年代适合国王的 CD 母盘制作

> 原文：<https://hackaday.com/2022/07/07/retrotechtacular-1990s-cd-mastering-fit-for-a-king/>

在音乐转变成短暂的 1 和 0 流之前，它曾经有某种物理形式。从蜡筒到乙烯基光盘，再到不同外壳中各种尺寸的磁带，最后到激光唱片，每一种新媒体都被宣传为是对以前格式的技术飞跃，每一种都证明了购买它们需要越来越多的钱。

但这就是问题所在——每一次购买都会导致你获得一件实物，而这背后有一个广泛的制造和分销过程。很少有艺术家比迈克尔·杰克逊在他的全盛时期要求更多的制造工作，正如[这一对流行音乐之王在 1995 年发行的*历史*双碟集的 CD 制造过程的深入观察所揭示的那样](https://youtu.be/suQmNjH92YA)。

该视频是作为新泽西州皮特曼的索尼音乐光盘制造厂的员工和管理层写给迈克尔的一封情书而制作的。这个过程从主人来到工厂开始，奇怪的是以 U-matic 录像带的形式出现；3/4″连续循环磁带通常用于模拟视频，但也可用于[录制数字音频](https://en.wikipedia.org/wiki/U-matic#Digital_audio)。然后，数字音频被发送到玻璃母盘制作，在洁净室条件下，在一个大玻璃盘上制作实际的凹坑。事实上，许多生产过程与半导体制造非常相似，从对洁净室的需求(尽管条件没有晶圆厂严格)到使用等离子蚀刻、气相沉积和金属电镀操作。

一旦母盘制作完成，复制工作就真正开始了。在那里，压模盘进入注射成型机，热聚碳酸酯在压力下被压向表面。这些复制品被镀铝、旋涂 UV 固化漆，然后被送到生产线上进行测试、丝网印刷和包装。索尼额外雇佣了 40 名全职工人，他们似乎已经处理了所有繁琐的手工任务，如组装珠宝盒，以处理此次发布的额外负载。

尽管这个感谢视频可能很俗气，但它可能是有充分理由制作的。那时候，迈克尔·杰克逊的一张专辑本质上是一大群工人充分就业的保证。该团队每天能够生产大约 50，000 份，考虑到历史销量超过 2000 万份，这对皮特曼的优秀员工来说是一个很长的工作日。
