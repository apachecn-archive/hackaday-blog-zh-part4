# 可印刷的单向驱动器跳过离合器的棘轮

> 原文：<https://hackaday.com/2022/08/25/printable-one-way-driver-skips-ratchet-for-a-clutch/>

棘轮螺丝刀可以帮助你更快地工作，即使它们的体积意味着它们不是在狭小空间工作的最佳选择。[ukman]决定自己建造一个类似的装置，依靠一个稍微不同的机制——超越离合器。

该设计类似于自行车上使用的飞轮，允许在一个方向上自由移动，同时在另一个方向上抵制它。当螺丝刀朝一个方向转动时，轴被一系列锁定它的圆柱体楔住。然而，离合器的几何形状允许轴在另一个方向转动，而不会楔入到位。其结果是一个螺丝刀，可以转动，回滚，并进一步转动。因此，可以在不松开工具的情况下拧紧螺钉。

由于其 3D 打印结构，它可能不是重型、高扭矩工作的最佳工具，但它看起来比处理简单的组装任务更有能力。[我们还在这些部件上看到了其他一些漂亮的螺丝刀](https://hackaday.com/2020/09/04/the-screwdriver-you-dont-need-but-probably-want/)。
