# 部分审查了 E3D 换刀装置

> 原文：<https://hackaday.com/2019/10/04/e3d-tool-changer-partially-reviewed/>

[设计原型测试]在邮件中收到一个盒子。里面？一个 [E3D“刀具更换器和运动系统”](https://www.youtube.com/watch?v=oooJSzczehU)从表面上看，它看起来像一台 3D 打印机，但它被吹捧为一台可以安装几种不同类型工具的机器，包括一个 3D 打印头。在下面的视频中，你可以看到看起来很重的机器的组装。

在一个廉价 3D 打印机价格远低于 200 美元的世界里，这种机器要坚固得多，所有部件的价格约为 3000 美元。[设计原型测试]是一个有点突出的价格，但你不得不怀疑他们是否试图考虑到最终的 CNC 头，特别坚固的构建可能是一个优势。然而，运动腰带的使用让这看起来像是一个漫长的拍摄。

我们很想知道这台机器的最终结果。评论者对价格和其他几个项目很不满意。然而，换刀机制似乎让他更开心了。由于缺少鲍登管，他没有将四个打印头和挤出机组装在一起。也许我们会看到第二部分的视频。

即使价格很高，这些机器也在等候名单上，所以你很可能不能马上买到。如果你对这款机器的更多细节感兴趣，你可以查看 E3D 的网站。我们无法判断这是一台价格过高的 3D 打印机，还是一台廉价的高质量打印机，或者是否有一些工具可能需要运动系统的坚固性。我们确实喜欢分离设计的想法，这样你就可以购买你想要的东西，甚至以后可以扩展到多头系统。

市场会怎么想？时间会证明一切。我们喜欢将类似 3D 打印机的系统用于 3D 打印之外的事情。当然，我们真正想要的是一台用于[PCB](https://hackaday.com/2015/11/20/review-voltera-v-one-pcb-printer/)或者至少是[金属](https://hackaday.com/2019/04/08/metal-3d-printing-a-dose-of-reality/)的 3D 打印机。
