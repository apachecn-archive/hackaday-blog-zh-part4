# 3D 打印的柠檬电池

> 原文：<https://hackaday.com/2019/02/05/a-lemon-battery-via-3d-printing/>

有一大堆高中科学实验对教授学生生物、物理和化学的基础知识很有用。其中一个经典就是柠檬电池。[iqless]决定尝试一下这个想法，并为他的学生们做了一个小东西。

基本的柠檬电池非常简单。柠檬汁提供电解质，而铜和锌充当电极。这种电池没有希望为你的特斯拉充电，但你可能会有足够的电量来点亮一个 LED 或小灯泡(双关语)。

[iqless]认为将电极直接插入柠檬是相当简单的。相反，电解质托盘是 3D 打印的。托盘可以装满柠檬汁(手榨的或者直接从瓶子里装的),托盘上有固定装置可以固定铜钱和镀锌机器螺丝作为电极。该托盘允许构建几个电池并串联或并联，从而提供进一步的教学机会。

这是课堂上的一个有趣的转折，我们认为这里有很大的可能性来进一步实验替代电解质和电极材料。我们也希望看到一个成熟的版本，在基于柠檬的高电压实验中串联大量的级联电池，但这可能要求过高。在课堂科学中使用现代制作技术有很大的空间——[我们在](https://hackaday.com/2015/10/23/hacking-and-kids-the-one-hour-egg-drop/#more-173220)之前已经讨论过鸡蛋下落的变化。休息后的视频。
