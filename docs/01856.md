# 你是你自己的触觉反馈

> 原文:[https://hack aday . com/2019/02/15/you-are-you-own-you-own-your-own-tactile-feedback/](https://hackaday.com/2019/02/15/you-are-your-own-tactile-feedback/)

[Maurin Donneaud]显然在制作大型柔性触摸感应布方面投入了大量工作，提供了一个干净直观的界面，并让任何人都可以将其集成到自己的项目中..这种压力感应织物被吹捧为电子音乐接口，但如果你只想着控制音乐，你就是在限制自己。你可以教人工智能让直升机更平稳地着陆，检测装甲中的拳击/拉平打击，通过将其集成到衬垫中来保护运动员，或者测量高尔夫挥杆中的张力点，这只是《六十秒》作者头脑风暴中的几个例子。这个自制的电子纺织品可以测量三维尺寸，你可以用导电线、导电织物和压阻织物自己制作。如果你之前被这个想法吓倒了，那就没有理由退缩了。

这个想法并不新鲜，我们已经看到了一些 T2 的简洁的迭代，但是这个想法一分钟就能产生一英里(公里)的想法。观看线框界面让我们想起时空中的黑洞模拟，但这些更像是地球上的实时响应。最重要的是，当一堆硬币放在表面上时，它们显示出一致的结果。像大多数其他人一样，这是一个三明治，面包片是普通的织物和压阻材料，冷切是排列在网格中的导电带。[Maurin]设计了一个定制的 PCB，它可以在青少年和电阻网络之间制作一个方便的适配器，以了解哪条网格线受到了挤压。

如果您不需要柔性触摸表面，[我们也可以在这方面帮您](https://hackaday.com/2017/05/18/everythings-a-touch-surface-with-electrick/)。
