# 简单的设计提升了这种机械点阵显示器

> 原文：<https://hackaday.com/2021/12/07/simple-design-elevates-this-mechanical-dot-matrix-display/>

不要误解我们——我们和任何人一样喜欢独特的展示。但有时我们会偶然发现一个如此独特的例子，以至于我们缺乏基本的词汇来描述它。这就是[这个弹珠凸起点阵字母数字显示器](https://www.thingiverse.com/thing:5154001)的情况。但这很酷，所以我们要试一试。

[Shinsaku Hiura]设计的核心——字面意思——是一个 3D 打印的圆柱体，在其外圆周上有一个螺旋槽。圆筒在有竖杆的笼子里旋转；杆和凹槽的尺寸被设计成捕获 6 毫米气枪 BBs，其通过显示器的固定底座中的端口被送入凹槽。BBs 被送入正确位置的凹槽中形成字符，字符随着圆柱体的旋转而向上移动。看看下面的视频吧——它比语言更好地解释了这个问题。

聪明的一点是 BBs 是如何进入凹槽的。不是有一个单独的机构来控制进料口，而是有一个反冲机构，当驱动滚筒的电机反向运行一段时间时，它会打开进料口。这是一个聪明的使用凸轮来完成工作，而不增加额外的伺服系统，这让我们想起了在[他的单伺服七段显示器](https://hackaday.com/2021/11/13/a-one-servo-mechanical-seven-segment-display/)中展示的设计简约。

目前还不清楚这是否会是一个非常实用的显示器，但这并不妨碍它变得很酷。虽然，[Shinsaku Hiura]刚刚发布了一个后续视频,展示了一个用于显示谷歌日历中即将到来的事件的更大版本，所以也许我们错了。
