# PalmPilot 返回，这次是在您的浏览器中

> 原文：<https://hackaday.com/2022/12/04/the-palmpilot-returns-this-time-in-your-browser/>

PalmPilot 似乎没有得到多少追溯计算的爱，但也许它应该。毕竟，它可能不是最早的掌上电脑，但它可能是最成功的，并最终导致了智能手机的时代。无论您是否怀念以前的 Palm 应用程序，或者从未体验过它们，都不用担心。多亏了互联网档案项目，你现在可以在你的浏览器中重温它们所有的荣耀。

有 500 多个应用程序和游戏都运行在一个基于浏览器的模拟器中。有些程序似乎不太好用，有些在虚拟环境中毫无意义。但是很多都很好，如果你想要经典的应用程序，只需打开任何东西，然后按下 home 键。如果你想回顾 1999 年的 Palm IIIe PDA，可以看看下面的视频。

Grafitti 手写识别系统是当时最先进的。关键是，如果印刷体主要是以相同方式工作的单个笔画，那么系统可以更容易地识别印刷体。例如,“A”没有横杠,“F”缺少底部水平线。尽可能用一个笔画来构成字母，每个字母只有一种构成方式。美好时光！

当时的高科技是什么，现在你可以用备件来建造。如果你碰巧有一个 Palm，你可以考虑给它一个[急需的背光](https://hackaday.com/2022/05/17/led-backlight-brings-vibrant-colors-to-classic-palm-pdas/)。
