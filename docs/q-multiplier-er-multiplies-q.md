# Q 乘数——呃……乘以 Q

> 原文：<https://hackaday.com/2021/01/18/q-multiplier-er-multiplies-q/>

如果你不到某个年龄，你可能从未听说过 Q 乘数。这是一种增加无线电接收机中频“Q”的装置，因此提供了更高的选择性。如果你喜欢怀旧，你可以在下面杰夫的信息视频中看到一个 20 世纪 60 年代的希思基特 QF-1 Q 倍增器。

Q 倍增器是一个再生放大器，工作在振荡点以下。这为感兴趣的频率提供了非常高的放大，而为其他频率提供了较小的放大。一些收音机有一个像这样的内置舞台，但 QF-1 是为了添加到一个外部收音机。对于一些希斯基特接收机，有一个直接插入中频级的插头。这些无线电需要一些黑客攻击才能工作。

QF-1 有几种工作模式，可以作为带通滤波器或陷波滤波器。你也可以使用主旋钮来调节频率。电路显示并不过分复杂，也没有印刷电路板。有源器件是一个双三极管。

如果你想更深入地讨论这个电路，几年前 Orange County 业余无线电俱乐部 newletter 有一篇由[AF6C]撰写的关于这个设备的文章。它解释了每种模式的工作原理。它还提到了该设备的一些后代，如 HD-11 和 GD-125，这些产品一直销售到 1971 年。

如果你计划建立一个这样的电路，请记住，当时大多数无线电的中频是 455 kHz，这是 QF-1 所期望的。出于各种原因，如今大多数通信接收机使用明显更高的中频(例如，10.7 MHz 很常见)。

我们总是喜欢(杰夫的)旧接收机和设备的视频。并不是说他是唯一一个这样做的人。
