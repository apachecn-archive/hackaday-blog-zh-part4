# 卧床病人监护旨在改善护理

> 原文:[https://hack aday . com/2022/01/26/monitor-for-bedrobed-patients-aims-to-improve-care/](https://hackaday.com/2022/01/26/monitor-for-bedridden-patients-aims-to-improve-care/)

做创客和黑客的乐趣之一就是解决问题，满足需求。当你两者都能做到的时候，那就很特别了。[rodrigo.mejiasz]的项目无疑属于用他的卧床病人监护仪解决问题和满足迫切需求的特殊类别。

虽然[Rodrigo]的项目页面没有说明他创建这个项目的动机，但人们只需要看看他们当地的医院病房或老年护理机构，就可以理解为什么这个设备如此神奇。医护人员和护理人员被拉伸得像纸一样薄，他们的注意力经常被打断。

这就是卧床病人监护仪的用武之地。一个健康的人如果感到不舒服，可以调整自己的位置，但是卧床不起的病人却不能。不仅仅是卧床不起的病人无法下床，而是他们在没有帮助的情况下无法自己移动。结果是巨大的痛苦。如果任其发展，可能会导致褥疮。这些不仅令人非常不愉快，而且对病人的健康增加了危险。

卧床不起的病人监视器介入并提供不仅仅是类似鸡蛋计时器的警报，而且帮助护理人员跟踪病人在床上的位置，甚至跨越几个工作班次。这确保了护理的连续性，否则很容易错过。

这个建筑的美丽在于它的应用，但也在于它的简单:它只是一个 Arduino Mega，一个带有 Micro SD 卡的 TFT 盾牌，以及触摸屏本身。几个 LED 和一个蜂鸣器负责报警。精心配置的界面使设备的使用变得显而易见，以便员工可以立即使用监视器。

制造者潜入医疗领域的历史悠久，比如[这款获得 2021 年戴森奖的刺伤治疗设备](https://hackaday.com/2021/08/29/dyson-award-goes-to-emergency-medical-device-for-treating-stab-wounds/)。
