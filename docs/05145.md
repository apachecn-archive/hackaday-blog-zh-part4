# (非常)便宜的环氧树脂脱气

> 原文：<https://hackaday.com/2019/12/19/degassing-epoxy-resin-on-the-very-cheap/>

任何试图用环氧树脂封装东西的人都知道气泡有多麻烦。如果你想得到一个非常清晰的结果，你最不想要的就是一堆凝固在时间里的微小气泡。防止这种情况的最佳方法是将零件放在真空室中，这样在环氧树脂固化之前所有的空气都会排出，但对于一次性项目来说，这是一笔相当大的投资。

但是假设你的部件足够小，[Jasper Sikken]有一个很棒的提示，让你只花几美元就能建造一个简单的真空室。休息后，他在视频中展示了他自制的密室，我们认为你会同意前后的变化是相当戏剧性的。最好的部分是，如果你想建立自己的版本，你只需要两个部分。

第一个是一个密闭的容器，大到足以容纳你正在制作的作品。请记住，真空室越大，抽真空需要的时间就越长，所以要避免使用不必要的大容器。[Jasper]使用了一个带锁盖的玻璃罐，它不仅便宜易得，而且内部容积相当大。

显然，第二个部件是真空泵本身。这通常可能是一个很高的要求，但[Jasper]最近发现，你可以从通常的进口网站上以 5 美元的价格买到小型电池供电的小工具，这些小工具旨在从食物容器中吸取空气。你所需要做的就是在容器的盖子上戳一个洞，将设备放在洞的上方，然后观看魔术表演。

这种方法对任何比镇纸小的东西都很有用，但是如果你有比镇纸大的东西，[你就需要加强你的室内游戏](https://hackaday.com/2019/04/12/build-your-own-vacuum-chamber-for-degassing-and-more/)。幸运的是，在紧要关头，即使是[更大的真空室也能廉价建造。](https://hackaday.com/2018/08/04/hacked-vacuum-chamber-wont-suck-a-hole-in-your-budget/)
