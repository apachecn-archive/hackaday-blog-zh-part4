# 翼手龙的飞行:受侏罗纪启发的扑翼机

> 原文:[https://hacka day . com/2021/04/06/飞行的翼龙-一架受侏罗纪启发的扑翼飞机/](https://hackaday.com/2021/04/06/flight-of-the-pterothopter-a-jurassic-inspired-ornithopter/)

扑翼机看起来很傻。它们看起来像是不应该工作的东西。一架没有螺旋桨，翅膀不停拍打的飞机？那东西不可能飞起来。然而，有许多爱好者、研究人员和鸟类会强烈反对这种观点，因为扑翼机确实会飞。当他们这样做的时候，他们看起来几乎很迷人，这也是我们喜欢[【Hobi Cerdas】制作的翼手龙](https://www.youtube.com/watch?v=j6IXumMkYOk)的一个原因，这是一种模仿翼手龙的橡皮筋动力扑翼机。

玩笑归玩笑，扑翼机背后的科学和研究，以及相关的生物如何飞行本身就很吸引人——这就是为什么[Lewin Day]写了那篇关于蜜蜂如何设法在空中飞行的文章。我们可能会花上几个小时阅读这些东西，观看原型的 T2 视频。虽然我们目前可以建造的大多数模型都不如螺旋桨驱动的模型有效，但进化完善的飞行机制的潜力是无穷无尽的。仅此一项就足以在未来几年推动这样的建设。

正如你在下面的视频中看到的，[Hobi Cerdas]经历了他自己的研发过程，因为他让他的翼龙翱翔。该模型在它的首次飞行中被证明太重了，但这并不是一点点提高尾部和快速现场斩首不能解决的问题。在更成功的第二次飞行后，他换上了更薄的橡胶带，并修改了机翼的前缘以获得更大的推力。这使得小巴尔萨恐龙能够真正起飞，飞得足够长，与建筑物和树木有一些非常近距离的接触。

对于那些已经渴望建造自己的扑翼飞机的人来说，这些计划来自于 2017 年夏季版的*，[扑翼飞机协会](https://www.ornithopter.org/)(我们今天很高兴了解到的一个组织)的官方通讯。你也可以找到更多[深入的扑翼机建造日志](https://hackaday.com/2018/05/11/scratch-built-ornithopter-heres-how-i-flapped-my-way-to-flight/)来帮助你入门。老实说，没有理由限制自己不受控制的飞行；过去，我们遇到过一些令人印象深刻的遥控扑翼飞机。*
