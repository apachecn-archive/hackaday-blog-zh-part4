# Hackaday 播客 105:486 FPGA 上的厄运，你的灯丝有多粗，树莓 Pi 讲 Android Auto，我们向火星进发

> 原文:[https://hackaday . com/2021/02/12/hackaday-podcast-105-486-doom-on-FPGA-how-thick-of-your-filament-raspberry-pi-speaks-Android-auto-and-were-head-to-mars/](https://hackaday.com/2021/02/12/hackaday-podcast-105-486-doom-on-fpga-how-thick-is-your-filament-raspberry-pi-speaks-android-auto-and-were-headed-to-mars/)

Hackaday 的编辑迈克·斯奇斯和埃利奥特·威廉姆斯整理了过去一周的一些伟大的黑客作品。我们喜欢看到 til 311——DIP 封装中的复古显示器——用 SMD 电子器件和树脂铸造精致地再现。你可能永远不需要连续测量你的 3D 打印机灯丝的直径，但以防万一有一个聪明的霍尔效应传感器机制。我们都很钦佩 FPGA 领域的工作，本周我们看到一个 RISC-V 内核深入到 FPGA 堆栈中，运行最初在 486 计算机上运行的 Doom 版本。下周，美国国家航空航天局(NASA)的“坚忍”号火星车将在火星表面着陆，这是一场三环杂技工程，我们对此感到兴奋不已。
