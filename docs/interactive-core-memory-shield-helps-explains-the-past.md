# 交互式核心记忆盾有助于解释过去

> 原文：<https://hackaday.com/2019/12/19/interactive-core-memory-shield-helps-explains-the-past/>

[Andy Geppert]送来了他极其聪明的互动核心记忆盾。

在一个黑客的工作成为另一个黑客的基础的伟大展示中，[Andy]从[Jussi Kilpelainen]为 Arduino 设计的[核心记忆盾开始。](https://hackaday.com/2018/04/28/core-memory-upgrade-for-arduino/)当他在玩盾牌时，他希望“看到”内核存储器翻转，并产生了在单个内核后面添加 LED 矩阵的想法。

第一次迭代成功了，但它只显示了 Arduino 认为核心内存所处的状态。他真正想要的是对实际情况的现场解读。他意识到一个 Adafruit Featherwing 8×8 矩阵显示器也适合放在核心存储器后面。现在，指示灯会根据核心内存的读取状态进行更新。这使得他可以用磁性笔翻转单个的比特并看到结果。非常酷。
