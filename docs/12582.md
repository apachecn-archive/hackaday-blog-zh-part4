# 用框架木材制作华丽的东西

> 原文：<https://hackaday.com/2022/02/06/making-something-gorgeous-from-framing-lumber/>

在 Hackaday，我们通常会报道那些会眨眼、发出哔哔声的东西，有时它们甚至会发出声音。然而，该网站的名称是 Hackaday。我们是聪明的，以新的方式重复使用东西，最重要的是庆祝有趣的项目。虽然这不是一个会给头版增色的传统项目，但我们认为这个由框架木材制成的床头柜显然属于。

框架木材是臭名昭著的古怪，怪异和严重打结。大多数严肃的家具制造商避免使用便宜的材料，而选择更贵更硬的木材。在美国，大盒子五金店出售廉价的快速生长的软松木，这种松木具有大量的木材固有的翘曲和扭曲。将它弄直并做成直角的过程是复杂的，即使它被切割，木材内部的内应力也会释放，随着时间的推移，木板会再次扭曲变形。视频中的刨削、接合和切割的延时过程几乎具有治疗效果。结果是两件非常有用的家具，放在大多数房间里都像在家里一样。

建筑中明显的工艺值得注意，但更令人印象深刻的是用廉价和不合适的材料制作出美丽的东西的过程。或许，如果你受到启发，决定在这个周末制作自己的床头柜，[你可以添加一些触摸感应电子产品。休息后的视频。](https://hackaday.com/2013/11/21/beautiful-touch-sensitive-furniture/)
