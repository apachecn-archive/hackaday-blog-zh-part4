# 你期待已久的游戏男孩色彩配件

> 原文:[https://hack aday . com/2022/11/30/the-game-boy-color-accessory-you-waiting-for/](https://hackaday.com/2022/11/30/the-game-boy-color-accessory-youve-been-waiting-for/)

有时愚蠢的项目会吸引我们的眼球，我们就是忍不住要报道它们。在 Hackaday.io 上，[solderking]意识到在游戏男孩颜色的多游戏管理硬件中有一个明显的遗漏。显然，把一大堆游戏放在口袋里随身携带太单调了，一个硬件多转换器肯定会提高可用性。这种 [方便，口袋友好的解决方案](https://hackaday.io/project/188430-i-made-a-weird-gameboy-color-accessory) 允许你一次对接四个弹夹，只有一点轻微的不便，旋转整个组件，锁定游戏并加载它。还有什么比这更简单的呢？

该组件由一个可笑的三层 PCB 堆叠构成，带有一个旋转中心接头，完全是无源的，来自所选游戏盒的连接通过一系列连接器传递，最终通过通常的 edge 连接器进入 Game Boy。仅仅是这个工作的事实就显示了旧游戏平台曾经是多么的宽容(我们猜测，是多么的缓慢)，以及你能得到什么！尽管如此，这是一个有趣的构建，而且它确实有效，这只是表明，因为你可以，那么你应该。

我们对游戏机黑客并不陌生。这里有 [一个有用的插件](https://hackaday.com/2022/08/16/squareboi-is-the-diy-game-boy-cart-youve-always-wanted/) 来帮助你开发第一个程序。如果旧平台对你来说有点太有限，那么我们已经为你准备了一个 hack，它在机箱内插入了一个 iCE40 FPGA 和一个 Pi Zero ，以提供更多的活力。
