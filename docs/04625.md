# Hackaday 播客 041:呼吸器、链轮、气垫船和数据记录的“如何避免”一集

> 原文:[https://hackaday . com/2019/10/25/hackaday-podcast-041-the-how-not-to-episode-of-rebreathers-chain-链轮-气垫船-and-data-logging/](https://hackaday.com/2019/10/25/hackaday-podcast-041-the-how-not-to-episode-of-rebreathers-chain-sprockets-hovercraft-and-data-logging/)

黑客日报的编辑迈克·斯奇斯和埃利奥特·威廉姆斯揭示了真正的黑客周。似乎本周所有的项目都在以“错误”的方式做这件事，看到他们在这个过程中学到了什么是令人高兴的。气垫船*可以利用外部鼓风机产生的二氧化碳效应工作。你*可以*将你的 Linux 日志转储到焊接的 eMMC 内存中，而链轮*可以*从建筑支架上切下来。如果你真的想制造自己的呼吸器，你可以*做*。所有这些黑客技术都有效，看到如何以不同的方式做一些事情是对硬件黑客艺术的鼓舞人心的致敬…你可以通过问自己为什么这些特定的技术不是最常用的来学习很多东西。*

 *另外，迈克上周末在罗马的创客大会上遇到了亚历山德罗·拉内鲁西。除了是 slic3r 的原始作者之外，Alessandro 在过去几年中一直是意大利开放源码的领导者。他谈到了今年早些时候通过的立法，该立法要求政府委托的软件现在必须开源，并以开放许可的方式发布。
