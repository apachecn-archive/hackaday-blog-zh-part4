# Hackaday 播客 151:最黑客的 VR 手套，用鞋带画波巴·费特，ECU 黑客，氨从哪里来？

> 原文：<https://hackaday.com/2022/01/14/hackaday-podcast-151-the-hackiest-vr-glove-plotting-boba-fett-with-shoelaces-ecu-hacking-and-where-does-ammonia-come-from/>

Hackaday 主编埃利奥特·威廉姆斯和执行主编汤姆·纳尔迪再次回来谈论所有来自我们科技世界角落的怪异和精彩的故事。由于零件短缺，佳能不得不暂时放弃打碎他们的墨盒，这太完美了，不能忽视。对于国际空间站来说也有一些好消息，因为白宫表示他们准备支持这个轨道前哨站直到 2030 年。

我们还将关注一个非常有前途的项目，为 VR 提供触觉反馈，使用 Arduino IDE 对裸机 x86 进行编程，以及在 2010 年大众高尔夫上添加 DIY 自动驾驶系统所涉及的令人难以置信的逆向工程。最后，我们将发现为什么这个星球上的大多数人类生活依赖于一个许多人从未听说过的过程，并了解制造比实际需要更重的汽车的悠久历史。
