# 大气高压马达产生有用的能量

> 原文:[https://hack aday . com/2022/07/26/atmospheric-high-voltage-motor-makes-used-power/](https://hackaday.com/2022/07/26/atmospheric-high-voltage-motor-makes-useful-power/)

虽然这看起来像是一位才华横溢的发明家疯狂的狂热梦想，但尼古拉·特斯拉直接从大气中获取能量并将其释放的计划比小说更现实。出于几个显而易见的原因，从大气层中获取这种能量通常是非常困难的，尽管仍有可能做到，正如[lasersaber]用他最新的大气发动机展示的那样。

为了帮助解决从大气中获取电力的一些后勤问题，[lasersaber]正在使用范德格拉夫发电机作为高电压梯度的替代品，当在空中悬挂一根长电线时，可以发现这种高电压梯度。他已经对这样的高压电机进行了一段时间的实验，并完善了他对电晕放电电机的设计，使其足够大，并有足够的扭矩来驱动钻头。电机有一个导电的转子，定子上有一系列放电管，将电线上的金属点(大气电线会附着的地方)暴露在足够高的电压下会导致旋转。这种情况下，电压约为 30，000 伏，但电流极低。

有许多视频记录了他的最新成果，包括这个[后续视频](https://www.youtube.com/watch?v=_Bc12ocFwcM)，他在各种材料上钻了任意数量的孔，以展示其有效性。尽管他在这些作品中使用了范德格拉夫发电机，但他也展示了他们用无人机悬挂的电线工作的概念证明。他也成为了[高效](https://hackaday.com/2019/03/25/magnetic-bearings-might-keep-this-motor-spinning-for-millennia/)和[低功率](https://hackaday.com/2017/09/01/how-low-power-can-you-go/)电机方面的专家，并基于这些概念开发了许多其他有趣的产品。
