# 会开车的鸭子戴夫做油炸圈饼

> 原文：<https://hackaday.com/2021/10/06/dave-the-drive-able-duck-does-donuts/>

[Hey Jude]厌倦了从大多数现代电子玩具中散发出来的大男子主义，尤其是像怪物卡车和警察/军队布景这样的东西。他是在奇怪的东西中长大的，而不是攻击性，并且[想和他的儿子](https://www.instructables.com/Dave-the-Remove-Control-Duck/)分享一起敲打新玩具的经历。

这实际上是一艘被塞进诱饵鸭子里的遥控船，但事实远不止如此。在移除了让他保持直立的压舱物后，[Hey Jude]对鸭子戴夫的两侧进行了整形手术，在底部创建了一个船形的洞，并在顶部创建了一个铰链式的帽子，作为船只内部的访问面板。一切都用 Sugru 密封，尽管你可能会使用填缝剂，甚至是热熔胶(如果你想要更临时、更便宜的东西)。

最聪明的一点是戴夫背上的环——这使得他很容易从人行桥下放到池塘里，或者在他停在水中时营救他。休息过后，看看戴夫首航的录像。

遥控东西永远不会老。你有一台旧的任天堂 Zapper 吗？为什么不让它做你的家庭自动化投标呢？
