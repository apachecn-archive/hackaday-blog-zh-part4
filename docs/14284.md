# 诺基亚 5110 获得 Android Stowaway 和一个键盘

> 原文:[https://hackaday . com/2022/07/21/Nokia-5110-gets-Android-stowaway-and-a-keyboard/](https://hackaday.com/2022/07/21/nokia-5110-gets-android-stowaway-and-a-keyboard/)

尽管诺基亚现在在手机市场上基本上是一个后起之秀，但曾经有一段时间，他们的产品代表着移动设备的最先进水平。他们的一些手机甚至配备了滑出式键盘和发送电子邮件的功能；对于 90 年代末的设备来说，这几乎是闻所未闻的。那时[befinitiv]还是个孩子，买不起这些革命性的设备，所以[他建造了自己的现代版](https://github.com/befinitiv/nokia5110_communicator),看起来和感觉都像原版。

为了做到这一点，他借用了诺基亚 5110 手机的外壳和结构，但对其进行了修改，以在旧的电池盒中放置一个小型 Android 设备，以及一个小型蓝牙键盘(也是由[befinitiv]从头开始构建的)，该键盘连接到 Android 手机，以模仿旧的滑出式风格。然而，这不仅仅是一个案例模型。他还对手机的原始 PCB 进行了逆向工程，并在其中加入了蓝牙模块，使手机的屏幕和键盘基本上按照最初的设想工作。

这个项目很好地解决了 90 年代手机怀旧之痒，同时在现代世界中仍然可以作为真正的手机使用。假设你不太在意字面意义上的手机，Notkia 项目也是给这些旧手机带来新生命的令人印象深刻的努力。
