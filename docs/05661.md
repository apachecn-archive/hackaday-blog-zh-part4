# 对危险有感觉的狼人耳朵

> 原文:[https://hack aday . com/2020/02/12/有危险感的狼人耳朵/](https://hackaday.com/2020/02/12/werewolf-ears-with-a-sense-for-danger/)

当走在黑暗的街道上时，通常会感到有人在跟踪。当然，检查是值得的，但是如果我们能从未知中获得比简单的模糊感觉更好的数据呢？[caitlinsdad]建立了一个项目可以做到这一点，[还有一双可爱的耳朵。](https://www.instructables.com/id/ISO-Standard-Werewolf-Perky-Ears-Alert-System/)

狼人的耳朵声称是 ISO 标准，虽然我们还没有看到相关的文件来证明这一点。不管怎样，他们使用阿达弗洛·杰玛·M0 微控制器来运行节目，连接到红外接近传感器来检测运动。当被触发时，Gemma 通过抽动头带上的狼耳朵来响应信号，提醒佩戴者有人正在靠近。

建造和校准得当，这可能是一个有用的发明，对于那些经常发现自己被那些躲在人行道上的人跟踪的人，以及那些搬到另一个街区是一个更昂贵的选择的人。[我们也看到了其他反应灵敏的可穿戴设备](https://hackaday.com/2012/08/13/flutter-dress-vibrates-when-it-hears-loud-noises/)。休息后的视频。
