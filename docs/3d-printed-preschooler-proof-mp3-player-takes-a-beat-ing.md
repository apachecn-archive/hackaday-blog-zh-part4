# 3D 打印学龄前儿童证明 MP3 播放器需要一个节拍

> 原文：<https://hackaday.com/2021/10/08/3d-printed-preschooler-proof-mp3-player-takes-a-beat-ing/>

建立新想法的原型可能很有趣，但是将新项目放在一个耐用的外壳中可能是一个困难。尤其是当这个产品的使用者是自然界中最具破坏性的力量之一时:一个蹒跚学步的孩子！这就是[blue blade]想要为他的孙子制造一个耐用的 MP3 播放器时发现自己所处的环境，你可以在休息时间下面的中看到[他的工作成果。](https://www.youtube.com/watch?v=ow6FAjuPrN0)

硬件很简单:一个 16850 锂离子电池为 MP3 解码器/放大器模块供电，播放存储在微型 SD 卡上的 MP3。扬声器、电源开关和微型 USB 电池充电器完成了构建。最突出的是圈地。为什么？

当涉及儿童时，耐用性不是产品寿命的问题，而是安全问题。容易破碎的物品不仅没用，还可能很危险。考虑到这一点，[蓝刃]建造了一个色彩鲜艳的外壳，用金属螺栓连接超厚的墙壁。从外部来看，一个圆形的盖子栓在充电器连接器和微型 SD 卡插槽上。唯一的另一个突起是一个发光的摇杆开关，用于打开和关闭 MP3 播放器。

对耐用性的关注也延伸到了 MP3 立方体的内部。电子设备的切口防止它们水平移动，同时一个夹子栓在它们上面，使它们牢牢地固定。拉链将电池固定在底座上，热熔胶为电路板提供了额外的安全元素。甚至连扬声器也用四个螺栓牢牢固定。[blue blade]已经通过知识共享许可让获得了该印刷品。

更重要的是，[蓝色刀锋]报道说，他的孙子喜欢这个玩具，而且这个立方体已经经受住了六个多月的幼儿虐待。显然，对细节的关注使这款玩具几乎坚不可摧！
