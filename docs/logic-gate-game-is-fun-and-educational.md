# 逻辑门游戏既有趣又有教育意义

> 原文：<https://hackaday.com/2022/12/30/logic-gate-game-is-fun-and-educational/>

你对你的逻辑门了解多少？在 STEM 项目类的最终提交中，[BKriet]使用了树莓 Pi Pico、一些闪光灯和大量的 3D 打印将情况游戏化。结果是[名！那个！大门！](https://www.instructables.com/Logic-Gate-Game/)，一个有趣又有教育意义的玩具，最终被【BKriet】捐赠回班级(这是我们书中的一个热门举动)。

这个游戏的目的是在给定 A、B 和/或 C 作为输入的情况下，找出哪个逻辑门用于产生屏幕上显示的输出。游戏有十个阶段，每个正确的阶段奖励玩家 14 分，满分为 140 分。虽然每个阶段都会加载一个随机门，但是代码可以确保在一个游戏中没有门会重复出现。

这个项目是完全开源的，所以大门是大开的。没有 3D 打印机？这里有一大套 PCB 逻辑门，但实际上，你可以[用几乎任何东西制作逻辑门](https://hackaday.com/2017/01/03/make-logic-gates-out-of-almost-anything/)。
