# 15 伏到 110，000 伏

> 原文:[https://hackaday.com/2020/12/14/15-volts-to-110000-volts/](https://hackaday.com/2020/12/14/15-volts-to-110000-volts/)

制造高电压有一种满足感。当然，它有实际用途，比如霓虹灯或做某些实验，但是说实话——你真的只是想看到一些巨大的电弧照亮你黑暗的疯狂科学家的巢穴。[Mircemk]有治疗你病痛的药方。使用两阶段方法，他展示了一个简单的设置，即[从一个非常温和的 15V 电源产生大约 110KV](https://www.instructables.com/DIY-Flyback-Trafo-Cockroft-Walton-Voltage-Multipli/) 。

从 15V 开始，有一个使用反激式变压器和开关的级来产生合理的高电压。最后一级是 Cockroft-Walton 电压倍增器，可以产生相当大的电压。你可以在下面的视频中看到令人印象深刻的弧线。

显然，乘法器电路因科克洛夫和沃尔顿的实验而出名，但实际上是在 20 世纪初由一位名叫格雷纳彻的物理学家发明的。该电路使用二极管作为开关，并为一组并联的电容器充电。然而，放电使电容器串联。忽略损耗和负载，输出电压等于峰峰值输入电压乘以当前级数。现实世界的考虑意味着你不会完全得到它的电压，但它仍然可以提供一个有力的打击。点击休息时间观看电路运行视频！

有很多方法可以使[产生高电压](https://hackaday.com/2020/08/23/a-miniature-power-supply-for-high-voltage-hacking/)。小心[进行测量](https://hackaday.com/2016/12/08/measuring-high-voltage-in-millimeters-and-other-hv-probe-tricks/)。
