# 平面 PCB 线圈替代绕组变压器

> 原文:[https://hack aday . com/2022/01/29/planar-PCB-coils-as-a-alternative-to-winding-transformers/](https://hackaday.com/2022/01/29/planar-pcb-coils-as-an-alternative-to-winding-transformers/)

那些尝试过自己缠绕电感的读者会知道，这不是一件容易的事情，当这些电感处理高电压时，保持绕组层之间的充分绝缘尤其棘手。[Open Frime TV]有一个视频以一种新颖的方式解决了这个问题，通过使用堆叠的 PCB 线圈而不是电线为开关模式电源变压器[创建绕组(俄语；你必须启用 YouTube 的字幕自动翻译功能。](https://www.youtube.com/watch?v=4dqFR2q71rQ)

断点下方的视频有助于读者了解 PCB 线圈结构，提醒观众所有线匝必须位于同一方向，以及绕组之间绝缘的重要性。我们讨论了 PCB 线圈的特性与开关频率的关系，一旦变压器组装完毕，我们就会看到它被连接到电源板上进行测试。经验丰富的变压器绕组工可能对接下来发生的事情很熟悉；什么都不工作，变压器变热。在制作 PCB 时，他在每块板上留了一些铜，这相当于一个短路匝——切除这些铜可以让变压器完美地工作。

这种技术可能不能解决所有变压器的问题，但是如果你的工作需要缠绕变压器，这是一个有趣的选择。如果 PCB 线圈引起你的兴趣，[使用它们的特斯拉线圈怎么样](https://hackaday.com/2019/04/25/a-tesla-coil-from-pcbs/)？
