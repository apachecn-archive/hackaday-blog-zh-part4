# 滚动放射性骰子真正随机 D&D 发挥

> 原文：<https://hackaday.com/2022/11/02/roll-the-radioactive-dice-for-truly-random-dd-play/>

当你让一群人围着一张桌子玩“龙与地下城”游戏时，你必须预料到事情不会总是一帆风顺。毕竟，那些心甘情愿地创造并沉浸在另一个现实中的人可能会受到创伤，在这个现实中，一次糟糕的掷骰子可能会导致他们相处数月或数年的角色的虚拟死亡。伴随着创伤而来的是对有罪者的搜寻——这就是骰子！总是骰子！

消除这个借口，或者至少让它在统计学上不可信，是这个放射性随机骰子滚动器背后的想法。它来自[科学小屋]并使用放射性衰变产生真正的随机数，而不是大多数微控制器中内置的伪随机数发生器。该设计基于[【阿尔法凤凰】的μ子动力 RNG](https://hackaday.com/2020/01/20/random-numbers-from-outer-space/) ，但有一个重大转折:不是依赖背景辐射，【科学小屋】把铀的力量带到了派对上。

他们获得了一份 autunite 的样本，这是一种外形怪异的磷酸盐矿物，含有相当数量的铀，非常适合刺激骰子滚动器内置的盖革计数器。Autunite 还有一个优点，就是在紫外线下看起来非常酷，就像荷马·辛普森(Homer Simpson)所说的幽灵般的“燃料棒辉光”。这个建筑的核心是衰变能量 RNG，用于模拟从 D4 到 D100 的每个标准 D&D 骰子的投掷。激光切割的硬纸板外壳容纳了所有的控制和显示器，还具有一些战略性的开口，可以凝视它发光的内脏。

我们真的很喜欢这个设计，但我们不得不挑剔铀矿石的处理；诚然，autunite 的比活度可能相当低，但看起来至少有些手套是合适的。
