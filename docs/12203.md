# DIY 眼镜旨在改善色觉

> 原文:[https://hack aday . com/2021/12/15/DIY-glasses-aim-to-improve-color-vision/](https://hackaday.com/2021/12/15/diy-glasses-aim-to-improve-color-vision/)

通常，为了改善视力，我们会借助矫正镜片或激光眼科手术等工具来提高视力。然而，[凯西·康纳 2]发现了另一种方法，利用曝光来改善色觉，并试图在家里达到同样的效果。

最近发表在《自然》 杂志上的一项研究表明，暴露在 670 纳米的光线下 3 分钟会导致色觉改善，持续时间长达一周。原因是随着年龄的增长，眼睛中的视锥细胞产生 ATP 的能力越来越差，随着这种重要分子向眼睛细胞提供能量的减少，我们的颜色感知能力下降。暴露在 670 nm 的光线下似乎会导致眼睛中的线粒体在一种相当复杂的物理相互作用中产生更多的 ATP。

在[凯西的]建造中，led 被用来产生所需的 670 纳米红光，安装在乒乓球上，乒乓球被粘在一副太阳镜上。在计算出正确的曝光水平并每天早上定期向眼睛发射光线后，[Casey]计划在晚上用定制的 Python 脚本进行色度测试，以测量颜色感知。

[Casey]显示了对科学过程的正确理解，并解释了测试中使用的廉价监视器和设备。我们的期望是，即使结果可能无法直接与行业级测量进行比较，也应该有可能显示相对正或负的漂移。
