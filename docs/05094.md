# 螺纹插件物有所值吗？

> 原文:[https://hack aday . com/2019/12/12/you-get-you-money-worth from-threaded-inserts/](https://hackaday.com/2019/12/12/are-you-getting-your-moneys-worth-from-threaded-inserts/)

你有没有想过在你的 3D 打印项目中安装螺纹插件是否值得花费时间和金钱？来自 CNC Kitchen 的【Stefan】做到了，并决定[用科学](https://www.youtube.com/watch?v=G-UF4tv3Hvc)一劳永逸地回答这个问题。

如果这听起来很熟悉，那是有充分理由的:我们在三月份报道了[Stefan]在评估螺纹插件方面的最后一次尝试。然后，他主要感兴趣的是确定螺纹嵌件是否优于螺纹切割或直接打印到零件中。目前的工作是关于不同螺纹插件设计的相对价值。他看了三种不同风格的压入式插页，价格从每张几便士到 25 美分不等。外部滚花的复杂性似乎与价格无关；带有相对螺旋滚花的刀片看起来比那些在枪管外侧带有简单倒钩的刀片更难制造，但成本更低。事实上，在拔出测试中，中等价位的插件表现优于昂贵的插件。令人惊讶的是，最便宜的插件实际上在拔出阻力方面比印刷小孔和将 M3 螺钉直接拧入塑料中要差得多。

[Stefan]还研究了扭矩阻力，并发现三种插入类型之间没有实质性差异。事实上，没有一个插件被证明是薄弱点，因为所有扭矩测试的故障模式是 M3 螺栓本身。当然，这不适用于直接拧入塑料的螺栓；就抗扭矩而言，任何嵌件都比没有要好。

我们很高兴看到[Stefan]的测试，并感谢这些数据可以帮助我们成为知情的消费者。[John]在 Project Farm 做了类似的面对面测试，比如对不同环氧树脂粘合剂的测试。
