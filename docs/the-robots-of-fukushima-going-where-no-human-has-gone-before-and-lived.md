# 福岛的机器人:去人类从未去过(生活过)的地方

> 原文：<https://hackaday.com/2022/12/08/the-robots-of-fukushima-going-where-no-human-has-gone-before-and-lived/>

将机器人送入人类无法生存的环境是一个非常古老的概念。机器人不理会氧气、食物或任何其他无数的人类需求。它们也可以被视为一次性的，它们也可以抗辐射，它们可以在物理上适合小空间。如果你碰巧拥有一座发生过多次熔毁的核电站，你就需要机器人。很多都是。在休息时间下方的[视频中，【](https://www.youtube.com/watch?v=-ZJiVQ61MpQ) [Asianometry](https://www.youtube.com/@Asianometry) 】提供了福岛机器人的精彩简介。

从为三里岛事件开发的机器人开始，然后是切尔诺贝利，[Asianometry]进入了让机器人出现在现场背后的技术甚至政治，以及注定要用于太空和战争的机器人与注定要在熔毁后清理的机器人之间的交叉。

该视频进一步探讨了将机器人放入高辐射环境的挑战。同样有趣的是促进国内进一步创新的准备状态，或者说缺乏准备状态。

显然，清理熔化的反应堆需要高度专业化的机器人。此外，在一个反应堆上工作的机器人不能在其他反应堆上工作，这就产生了对更多定制机器的需求。该视频讨论了每一个问题，甚至触及了未来完全退役福岛设施所需的机器人。

要想再看看一些投入工作的早期机器人，可以看看我们十多年前发表的帖子“[福岛机器人日记](https://hackaday.com/2012/01/28/the-fukushima-robot-diaries/)”。
