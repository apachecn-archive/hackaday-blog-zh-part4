# Snap-Together Macropad 无需焊接即可实现

> 原文:[https://hack aday . com/2021/12/29/snap-together-macro pad-do-it-without-solder/](https://hackaday.com/2021/12/29/snap-together-macropad-does-it-without-solder/)

也许我们有偏见，但我们认为每个人都有使用 macropad 的必要。一套高度个性化的速度控制开始听起来像是一个非常节省时间的设备，这只是时间问题。

麻烦的是，购买 macropads 通常有点贵，而且不是每个人都喜欢构建键盘。好吧，如果你什么都不用焊接呢？这就是[【Jan Lunge】的手控宏指令](https://blog.heaper.de/3d-printed-macropad/)背后的想法。

如果你建造了这个，你仍然会想要打开一个通风的窗户，因为这个 macropad 需要大量的 3D 打印。它不需要胶水或螺丝，因为一切都扣在一起。

当然，这个版本的明星是[Jan]的热插拔插座设计。我们特别喜欢这个小夹子，它将列线固定在适当的位置，同时还在列线和行线之间提供了一个间隔。所有东西都用非绝缘线连接到 Pro Micro，并通过末端的弯曲和张力的魔力固定住。请务必在休息后观看构建视频。

渴望六包以上的开关？这种设计很容易扩展，直到微控制器输入耗尽。此时，您可能想要[添加屏幕来跟踪您所有的宏](https://hackaday.com/2021/10/30/a-hackable-keyboard-that-even-has-screens/)。
