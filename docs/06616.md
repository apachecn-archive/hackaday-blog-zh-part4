# 以一种好的方式砖砌你的 3D 打印机

> 原文：<https://hackaday.com/2020/05/20/bricking-your-3d-printer-in-a-good-way/>

在我们的方言中，砌砖几乎从来都不是好事。这意味着某些东西确实出了大问题，曾经有用而且可能很贵的小部件现在几乎和砖头一样有用。考虑到它们对文明的重要性，这似乎对砖头有点不公平，但它表达了这一点。

然而，事实证明，就噪音控制和打印质量而言，[砖块可以在 3D 打印](https://www.youtube.com/watch?v=y08v6PY_7ak)中发挥重要作用。正如[Stefan]在下面的视频中指出的那样，生活在一台 3D 打印机上不停地进行长时间打印会令人不安，特别是当步进电机的振动被传输到固体表面(如桌面)并被放大时。他发现将打印机与共振表面隔离是关键。虽然他最初的 Prusa i3 Mk 3S 上的 stock felt pad 脚有所帮助，但最好的结果是通过建立一个闭孔包装泡沫和混凝土铺路块的平台来实现的。弹性泡沫和摊铺机的减震质量相结合，将噪音水平降低了近 8 分贝。

[Stefan]还仔细测试了他的打印质量设置。机床通常有更多的质量来抑制不必要的振动，表现更好，因此有理由认为将打印机放在沉重的混凝土板上会提高性能。尽管质量上的差异不是很大，但还是很明显，再加上降噪，这使得在打印设置中包含摊铺机和一些泡沫碎片变得显而易见。

不满足于只花几块钱买一台摊铺机来减震？然后为你的机器铸造一个复合环氧树脂底座——要么是铝的[要么是花岗岩](https://hackaday.com/2017/03/27/casting-machine-bases-in-composite-epoxy/)的[。](https://hackaday.com/2016/05/01/precision-cnc-with-epoxy-granite/)
