# 乐高抢劫机器人偷钥匙

> 原文：<https://hackaday.com/2021/01/09/lego-heist-bot-steals-keys/>

私闯民宅是重罪，闯入者可能因此受到严重的身体伤害。派机器人代替是一个有吸引力的选择。虽然我们怀疑他们是否真的打算利用他们的发明为非作歹，但[砖块实验频道]还是用乐高建造了一个可行的抢劫机器人。

问题中的机器人实际上是两个，它们一起工作来执行它们的任务。一旦在房子里降下，剪式升降机机器人就会开到靠近咖啡桌的位置。然后，它通过伸出电机驱动的线性致动器将它的同伴机器人提升到位。然后，恢复机器人开车到桌子上，用手臂抓住一串钥匙，并在离开房子之前返回电梯机器人。这一切都是通过使用第三方乐高配件 [SBrick](https://www.sbrick.com//) 实现的，该配件允许通过蓝牙远程控制乐高电机。无线摄像头也有助于平台的视觉。

乐高真的很容易建造快速、实用的机械装置，而不需要大费周章。我们已经看到它被用在各种各样的方面，比如这个方便的电影扫描仪。休息后的视频。
