# 真空 Dragster 使用注射器推进

> 原文：<https://hackaday.com/2020/09/17/vacuum-dragster-uses-syringes-for-propulsion/>

大气压力无处不在，如果使用得当，它能提供巨大的压力。正如奥托·冯·葛利克在 350 多年前用他的马格德堡半球演示的那样，简单地从一个小室中抽走空气来创造一个真空就能产生令人震惊的结果。最近，[【汤姆·斯坦顿】已经使用真空为一辆小型 3D 打印的卡车提供动力。](https://www.youtube.com/watch?v=uH6-GQ1qFH0)

在 dragster 构建中，典型的柱塞注射器在末端被塞住，柱塞被拉回。大气压力对抗真空，想要将柱塞推回其原始位置。为了利用这一点，一根绳子系在柱塞上，使它在前进时转动齿轮，通过皮带驱动后轮。通过皮带传动装置上正确的齿轮比，dragster 能够快速旋转轮胎并向前射击。

这项工作是[汤姆]早期真空实验的伟大延续，使用注射器作为小型火箭。它让我们想起了高中竞赛中的经典二氧化碳赛车，对任何科学课来说都是一个伟大的项目。休息后的视频。
