# 液态甲烷火箭即将起飞

> 原文：<https://hackaday.com/2019/11/05/liquid-methane-rocket-is-set-to-soar/>

固体火箭是火箭学的一种有趣的入门方式。即使在家庭实验室里，酿造一批火箭糖也是可以实现的，任何人都可以用正确的材料试一试。建造一个能够飞行的液体燃料火箭发动机完全是另一回事，但是普渡太空计划能够胜任这项任务。

他们努力工作的结果是 Boomie Zoomie，一个 15 英尺高，130 磅重的火箭。峰值推力为 800 磅，它有足够的咕噜声来帮助东西离开地面。它以液态甲烷为燃料，这在大学建造的火箭中尚属首次。该飞行器由 6″铝管段构成，这是重量、成本和可加工性之间的最佳权衡。在设计过程中，我们特别注意模块化设计，既考虑到未来的设计修改，又便于现场准备。这使得团队的不同部分可以独立工作，简化了火箭发射的准备过程。

为了参加远火星液体火箭竞赛，该火箭已经经历了两次成功的点火。该团队估计，第一次发射将在未来几个月内进行。发射拖车和辅助支持设备的准备工作仍在继续，以使一切正常运行。目标是到达 45，000 英尺的高空。

对于那些对火箭事业感兴趣的人来说，普渡大学可能是最好的地方，因为它的太空项目有 300 多名成员。我们也看到了其他顶尖的大学火箭项目，比如波士顿大学旨在到达太空的努力。休息后的视频。
