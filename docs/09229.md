# 如何烤出完美光泽的核仁巧克力饼

> 原文:[https://hack aday . com/2021/02/14/how-to-bake-brownie-with-a perfect-glossy-skin/](https://hackaday.com/2021/02/14/how-to-bake-brownies-with-a-perfect-glossy-skin/)

如果你和这位作者有任何相似之处，你不会在最初的物体识别阶段花太多时间去检查核仁巧克力饼，而更喜欢在匆忙之后直接把它们放进嘴里。但那些更注重审美的人会非常小心地制作出有光泽、有吸引力的核仁巧克力饼。亚当·拉古塞决定一次又一次地找出获得完美核仁巧克力饼皮的方法。

在吃了许多许多盘核仁巧克力饼后，[Adam]在 Pia Sö rensen 博士]的帮助下，确定关键似乎是用蔗糖形式的非常精细溶解的糖制作核仁巧克力饼混合物，并仔细控制混合物中的水量。这就产生了一种粘稠的混合物，可以在烹饪过程中抵抗气泡，并产生一种光滑的表皮。水太多，混合物的粘性不足以支撑，导致核仁巧克力饼布满麻子，而果糖和葡萄糖等替代糖可能会破坏蔗糖分子的有序结构，而蔗糖分子是有光泽的表面所必需的。

[Adam]和[Pia]一起在探索这一过程背后的分子化学方面做了大量工作，并排除了在邪恶的孤立布朗尼亚文化中一直存在的几个神话。他们所缺少的是一套用 Arduino 和一些光电二极管执行的标准化反射率测试，但我们会假设这只是为了节省时间。我们以前见过分子美食领域的其他黑客，[像这个家酿厨房离心机](https://hackaday.com/tag/kitchen-centrifuge/)。休息后的视频。
