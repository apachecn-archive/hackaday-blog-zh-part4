# 你口袋里的 SSB

> 原文:[https://hackaday.com/2022/08/22/ssb-in-your-pocket/](https://hackaday.com/2022/08/22/ssb-in-your-pocket/)

在过去，短波收音机是主要的桌面设备。如今，你可以买到真正小巧的收音机。然而，它们中的大多数只具有 AM 功能(即，没有接收单边带或单边带信号的简单方法),并且可能具有接收 FM 广播的能力。小型收音机通常没有外置天线，而外置天线对于短波收音机来说至关重要。[Farpoint Farms]展示了 Raddy RF 7860，这是一款掌上收音机，但它具有难以捉摸的边带模式以及外部天线端口和有线天线。它甚至有一个可充电电池。

阅读评论，似乎这是一个汉荣达 HRD 747 收音机的改装版。当然，也有其他带边带接收的小型收音机，如 Tecsun PL368，但它们不会这么小。如果你在市场上寻找一个非常小的短波收音机，这可能是你想要的。

当然，问题是这些天你想在短波波段听什么。短波广播公司越来越少，尤其是那些面向普通观众的广播公司。然而，如果有你想听的东西，将这台收音机与一个[良好的便携式天线](https://hackaday.com/2020/04/12/homebrew-loop-antenna-brings-the-shortwave-world-to-you/)配对，就能完成任务。
