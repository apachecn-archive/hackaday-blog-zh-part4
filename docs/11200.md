# 用旧无人机零件建造的整洁的小飞艇

> 原文：<https://hackaday.com/2021/09/02/neat-little-airboat-built-from-old-drone-parts/>

多旋翼无人机往往需要高质量和匹配良好的零件，以便在空中保持平衡。然而，坠毁足够多的无人机，你可能会发现你有很多不匹配的碎片。在这种情况下，[杰森·苏特]决定洗劫他的垃圾箱，用备用零件给自己造了一艘小 FPV 汽艇。

这艘飞艇由一个 3D 打印的船体组成，配有一个独立的动力模块。动力模块容纳了飞行控制器，并在后部安装了双马达。装有三叶推进器，它们推动船只，并允许它用不同的推力而不是方向舵来驾驶。然后给它安装一个摄像头，让它可以通过 FPV 耳机进行操控。

操作仍然不完美，FPV 天线上的水导致视频传输出现一些问题。然而，这是一个有趣的项目，很好地利用了旧零件。当然，如果你自己的 FPV 项目有振动问题，可以考虑建造一个减震支架。休息后的视频。
