# 莫尔斯键盘与眨眼交流

> 原文：<https://hackaday.com/2022/03/18/morse-keyboard-communicates-with-the-blink-of-an-eye/>

我们大多数人用手与计算机进行交互，但是除了手指和脚之外，人体还能够进行多种类型的输入。但是那些不能使用四肢和没有声音的人怎么办？为了他们，是时候发挥创造力了。

迈克尔·保罗·科德发明了一种简单的打字方法，只需闪烁莫尔斯电码。那些记忆力好的人可能还记得[清醒抄写员](https://hackaday.com/2012/06/20/communicating-from-inside-your-dreams/)，他试图通过用加速度计检测快速眼球运动并触发他的电脑播放音乐来记录清醒梦实验。这反过来会通知[迈克尔]他实际上是在做梦，在脖子上系上斗篷，从高楼上纵身一跃是安全的。

虽然[迈克尔]的创造之前需要一个商业脑电图设备，但由于新的跨意识信息协议，他现在只需要一个旧的网络摄像头就可以工作，该协议通过使用面部检测，然后解释视频帧之间的像素变化量来操作。请务必在休息后查看它的运行情况。

你知道我们有多喜欢这里的辅助技术——就在两年前，[Byte 在 Hackaday Prize](https://hackaday.com/2020/11/07/the-byte-is-the-grand-prize-winner-of-the-2020-hackaday-prize/) 中获得了最高荣誉。
