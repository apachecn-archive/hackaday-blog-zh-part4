# 复古控制台升级赋予雅达利天赋

> 原文:[https://hack aday . com/2018/09/05/retro-console-upgrade-gives-atari-flair/](https://hackaday.com/2018/09/05/retro-console-upgrade-gives-atari-flair/)

如果你极度渴望对昔日视频游戏的怀旧感，但又不想花大价钱买一部 NES 经典，你总是可以找到一款可以直接插入电视的街机风格游戏。当然现在已经不是 1980 年了，玩太空入侵者或者小行星过一段时间就能老了。当这种情况发生时，只需用该系统的所有游戏取代升级版复古雅达利 2600 的内部组件。

正如预期的那样，这个升级是基于一个 Raspberry Pi Zero。不过，这可不像扔上 RetroPi 就收工那么简单。首先，[Blue Okiris]仍在使用最初的双按钮控制器/操纵杆，这是该版本所基于的吃豆人女士游戏，它增加了自己的一系列挑战。另一方面，RetroPi 没有他需要的一切，所以他转向了另一个叫做 Recalbox 的操作系统。它还包括 Kodi，因此也可以用作媒体中心。

从真正的意义上来说，这个版本看起来像是一个黑客。电路板的底部有点突出，但这更多的是一个功能，而不是一个错误，因为那里有一些额外的按钮和电源开关。总的来说，这是一个伟大的复古雅达利系统，拥有所有真正的经典，应该让[蓝色 Okiris]娱乐，直到有一天雅达利发布一个官方系统。不过，如果你想更深入地了解雅达利的世界，你总是可以[恢复一个而不是](https://hackaday.com/2018/07/15/incredible-atari-800xl-case-restoration/)。
