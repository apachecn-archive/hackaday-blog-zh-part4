# 远程屏幕查看器只有文本

> 原文:[https://hack aday . com/2022/05/31/remote-screen-viewer-is-text-only/](https://hackaday.com/2022/05/31/remote-screen-viewer-is-text-only/)

自从 2007 年第一次安装 Ubuntu 以来，你是否一直在慢慢地陷入对电脑的斯托曼式偏执？你现在厌恶任何有 GUI 的东西吗，包括浏览器？你用命令行检查你的邮件吗，即使你在七个代理后面？但是，你还想演《我的世界》吗？如果是这样的话，[这个只有命令行的屏幕浏览器可能只是一个使用 GUI 的工具，而不是技术上使用一个](https://github.com/louis-e/cli-screenview)。

这个远程屏幕查看器是由[louis-e]用 Python 构建的，一旦安装，允许客户机查看服务器的屏幕，即使客户机是纯文本控制台。[louis-e]从 Windows 命令提示符中演示了这一点。该脚本轮询服务器屏幕，然后使用各种可用的颜色和纹理在控制台中显示它。因此，分辨率和刷新率都很低，但只要没有精美的文本可供阅读，它仍足以播放《我的世界》和执行其他基于 GUI 的任务。

下面的视频只展示了远程屏幕查看器的演示，除了这个概念验证游戏演示之外，我们还可以想象很多其他用途。安装桌面环境和窗口管理器并不是所有计算机都必须的，所以如果您不想浪费时间和资源安装这些组件，这是一个功能性的解决方法。但是，如果你正在寻找一个更具体的机器的远程桌面软件，看看[这个软件，它可以在古董苹果电脑上实现远程桌面](https://hackaday.com/2022/03/07/remote-desktop-fun-for-your-old-macintosh/)。
