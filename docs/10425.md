# 数字 X 射线扫描仪拆卸产生大量工程好东西

> 原文:[https://hack aday . com/2021/06/14/digital-x-ray-scanner-tear down-yields-bounty-of-engineering-goodies/](https://hackaday.com/2021/06/14/digital-x-ray-scanner-teardown-yields-bounty-of-engineering-goodies/)

我们就直截了当地说:我们喜欢拆卸。拆开旧设备，看看工程师是如何解决问题的——或者没有解决——是一件非常迷人的事情，即使是对于打印机和收音机这样的日常设备。但拆卸真正有趣的地方是，当目标是如此奇怪和如此特殊的东西，以至于你通常不会指望看到外面，更不用说穿过它的内脏了。

【Mads Barnkob】偶然发现了一个这样的项目，[一台富士 FCR XG-1 数字射线照相扫描仪](https://kaizerpowerelectronics.dk/high-voltage/fujifilm-photomultiplier-tube/)。这种曾经昂贵且仍然非常笨重的医疗设备是一种“数字胶片系统”，从业者可以用它来取代放射摄影术中使用的老式银基胶片，而无需全部使用全新的数字 X 射线套件。这是一件复杂的设备，其工程产生了许多非常有趣的细节。

下面的视频是[Mads]系列的第三部分，在这里他瞄准了他想要的物体:机器的光电倍增管。然而，至少对我们来说，围绕在管子周围的东西才是真正的恒星；光是那个弯曲的丙烯酸光管就值入场费了。之前的视频集中在机器内部的激光扫描单元，以及运输成像板和扫描成像板所需的机电一体化装置。下面的视频还展示了 PM 管的实验，当与一块闪烁塑料结合时，它可以作为一个很好的辐射探测器。

我们之前已经介绍了一些关于 X 射线产生的信息，以及一些用于探测 X 射线的传感器 T2。我们还展示了一些有趣的科技内部 x 光照片，从[一个 Starlink 碟](https://hackaday.com/2021/01/11/starlink-satellite-dish-x-rayed-to-unlock-rf-magic-inside/)到[山寨适配器](https://hackaday.com/2020/07/15/x-ray-sleuthing-unveils-the-fake-in-your-adaptors/)。
