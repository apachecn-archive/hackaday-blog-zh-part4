# 为了乐趣和利益检测辐射

> 原文：<https://hackaday.com/2022/11/09/detecting-radiation-for-fun-and-profit/>

以前每个库存充足的末日地堡都有盖革计数器。现在，你不需要一个大的电子管仪表。你可以便宜地得到一个紧凑的数字仪器来处理你的辐射检测需求。【DiodeGoneWild】[从 FNIRSI](https://www.youtube.com/watch?v=TynpfXSBdhU) 回顾并推倒这样一个单元。这个外壳看起来像我们最近看到的其他几个类似的仪器，所以可以推测，有人正在大规模生产这些手持仪表外壳。你可以看下面的视频。该仪表可读取绝对放射性，也可测量累积照射量。

在测量了一些常见的放射性物质后，我们开始拆卸。里面当然是一根普通的管子。几个螺丝揭示了一个典型的可充电电池，一个相当简单的带有微控制器和实时时钟备用电池的 PCB。盖革管所需的高达几百伏的倍增电压需要大量的电路板。

PCB 的另一面只有按钮、振动马达，当然还有 LCD。除了与已知良好的血糖仪进行比较，我们不知道如何测试相对准确度。当然，没有塑料盖的裸管更灵敏，但也可以校准。

一个盖革计数器[不需要有很多部件](https://hackaday.com/2020/08/26/this-geiger-counter-has-few-parts/)。无论哪种方式，[数量惊人的东西](https://hackaday.com/2022/03/11/is-your-tape-dispenser-radioactive/)将引发它们。
