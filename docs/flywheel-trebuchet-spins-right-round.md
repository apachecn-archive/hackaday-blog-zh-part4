# 飞轮投石机正转

> 原文：<https://hackaday.com/2020/10/02/flywheel-trebuchet-spins-right-round/>

我们大多数人都熟悉帝国时代的攻城武器，科技的进步意味着这些遗迹在现代战场上并不常见。然而，发烧友社区的发展仍在继续，汤姆·斯坦顿(Tom Stanton)的[最新投石机设计](https://www.youtube.com/watch?v=RVT5i4nhIGs)为快速发射弹丸带来了不同的旋转。

该设计利用了飞轮作为储能装置的优势。飞轮使用手动曲柄，通过同步带和一套混合 3D 打印和 CNC 铝齿轮旋转加速。一旦旋转到足够的角速度，一个触发器就会从吊索上释放网球载荷，以每小时超过 180 英里的速度抛出。

在木材和钉子等传统材料的基础上，【汤姆】的最新设计依赖于铝，试图建造一种在户外淋雨时不会腐烂的东西。铝型材的使用也使调整和重新设计变得容易，同时提供必要的调整，如释放点和皮带张力。这些年来，我们推出了一些不同的设计；步行臂投石机可能是所有投石机中最古怪的。休息后的视频。
