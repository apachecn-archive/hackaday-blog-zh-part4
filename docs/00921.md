# 垃圾电视获得 RGB LED 背光

> 原文:[https://hack aday . com/2018/10/11/trashed-TV-gets-RGB-led-back light/](https://hackaday.com/2018/10/11/trashed-tv-gets-rgb-led-backlight/)

除非你把它拆开，否则它可能并不明显，但大多数列为“LED”的电视和显示器只是液晶面板，使用一组 LED 从后面照亮它们。类似地，通常被称为“LCD”的是使用荧光灯管照明的 LCD 面板。要获得没有独立背光的真正 LED 显示屏，你需要有机发光二极管。迷茫？欢迎来到消费科技的世界。

考虑到这些区别，【Zenodilodon】最近在一台坏掉的“LED 电视”上表演的黑客技术确实相当出色。通过[移除无用的白色 LED 背光，并用 RGB LED 条取代它们](https://www.youtube.com/watch?v=HRFS5VKMILA)，他不仅让电视再次工作，还赋予了它变色的能力。非常适合显示音乐视觉效果，或者用一场真正迷幻的*七武士*表演来开启你的下一个电影之夜。

在广告之后的视频中，[Zenodilodon]开始了他的 RGB 移植，把电视拆成了它的主要部分。最初的发光二极管是烘烤过的，所以它们也可以直接和它们的驱动电子设备放在一起。但 LCD 面板本身工作正常(通过用激光笔照射它来测试是否有图像)，扩散 LED 背光的塑料片很容易被回收。

旧的发光二极管被移除后，[Zenodilodon]展示了他的新灯带，并将它们焊接到外部控制器上。他小心翼翼地使用所有白色电线，因为他担心彩色电线可能会反射白光，并在显示器上引起注意。重新扣上电视后，他进行了几次演示，展示了白色 led 打开时的图像，以及当 led 在颜色之间循环时可以看到的一些有趣的效果。

RGB 条纹没有像原来的背光那样照亮显示器，因为有一些明显的暗点，你可以在条纹处看到一些水平线。但是[Zenodilodon]说，在现实生活中效果并不太差，考虑到这是一台便宜的电视，图像质量可能从来没有那么好。

另一方面，如果你在垃圾桶里发现一个屏幕破裂的 LED 电视或显示器，也许值得把它带回家[为你的照明项目](https://hackaday.com/2018/07/21/recycle-lcds-into-leds/)回收它的超高亮白色 LED。
