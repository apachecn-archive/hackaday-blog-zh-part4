# 乐高月球着陆器动画电影发布

> 原文：<https://hackaday.com/2021/03/18/lego-lunar-lander-animatronic-movie-released/>

退休科学家[马克·豪]花了几个月的时间制作了一部电子动画电影，在一部再现阿波罗 11 号登月的视频中展示了他的乐高月球着陆器。[Mark]不仅是制片人，还是技术总监、布景设计师和摄影师。他为这个场景设计并 3D 打印了一个定制的特效舞台。它使用步进电机、同步带、滑轮和一个线性导杆来驱动登月舱，所有这些都隐藏在一个离散的舞台塔内。他用水泥浆、喷雾粘合剂和少量小石头模拟月球风化层。

[Mark]在 Arduino Nano 中实现了特效音序器，并使用 Adafruit audio 声卡提供音效，他在该声卡上加载了真实阿波罗 11 号着陆时的声音文件。地板条形照明由 Neopixels 阵列提供，背光照明的地面从飞行空间降低一个切口。他做了一个定制的 PCB 主板来容纳 Arduino、声卡和电机驱动器。

由此产生的结果令人印象深刻。这并不是[马克]第一次试图缓解退休和冠状病毒隔离的双重无聊——早在 12 月他就制作了一部类似的[电子动画电影](https://youtu.be/q7emW9_gui4)，再现了土星五号的发射。感谢[jhookie55]的提示。
