# 乘坐星球大战陆地飞车得到一个真正的喷气发动机

> 原文：<https://hackaday.com/2022/09/20/ride-on-star-wars-land-speeder-gets-a-real-jet-engine/>

说到儿童乘坐玩具,《星球大战》陆地飞车是其中一个比较酷的例子。然而，使用杂草般的 12 伏电机，它们肯定不会快速移动。[乔尔创造]决定解决这一切，[跳上他装有真正喷气发动机的陆地飞车。](https://www.youtube.com/watch?v=OBdk_faHL6Q)

首先，原来的传动系统被删除，新的车轮安装在下面。最初，它被设置为前轮转向，后轮自由转向。遥控喷气发动机安装在陆地飞车背部的中央发动机槽中，并通过标准的双通道遥控发射器进行控制。

喷气发动机工作正常，但是轮子的配置导致了飞行摩托只能做油炸圈饼。由于重新配置了后轮锁定的飞行摩托，飞行摩托的操控比预想的要好得多。测试空间仅限于一个停车场，所以高速跑是不可能的。然而，基于有限的测试所取得的成就，它看起来似乎可以在油门最大化的情况下进行一次像样的剪辑。

这不是一个实用的构建，但它看起来肯定很有趣。[Joel Creates]有一个伟大的梦想，就是再增加两个喷气发动机，并把它带到跑道上进行高速测试，这是我们希望看到的。

遥控喷气发动机现在有点像 YouTube 上的时尚，出现在从[遥控汽车](https://hackaday.com/2022/08/28/high-speed-rc-jet-car-is-a-harsh-teacher/)到[特斯拉](https://hackaday.com/2021/12/16/tesla-model-s-gets-boost-with-jet-engine-upgrade/)的所有东西上。休息后的视频。
