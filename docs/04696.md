# 这种 RGB 树植根于 PCB

> 原文：<https://hackaday.com/2019/11/18/this-rgb-tree-has-its-roots-in-a-pcb/>

[Paczkaexpress]的 RGB 树融合了巧妙的建筑技术和艺术形式，形成了一个非常漂亮的雕塑。

他的树的树枝是由漆包线制成的，上面有一个 RGB LED，末端是一个母头。单独的电线都被缠绕并雕刻成[树形](https://hackaday.com/2013/12/15/ledmas-tree/)。电线上覆盖着一层非常薄的塑料，我们强烈建议在显微镜下观察，这样可以保持均匀的铜色，不会短路，增加效果。

我们发现一个特别令人满意的形式和功能的组合是树根如何在 PCB 基座上点击。PCB 装有 STM32、电源元件和一个 LED 驱动器。它没有隐藏魔法是如何起作用的，这棵树确实从它种植的土壤中获取养分。这将是一个有趣的工具。非常聪明，休息后你可以看到[的最终效果](https://www.youtube.com/watch?time_continue=49&v=UbRa6d_wOm8)。
