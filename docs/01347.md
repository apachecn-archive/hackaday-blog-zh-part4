# 咖啡杯上的马克杯自动绘图

> 原文：<https://hackaday.com/2018/11/25/mug-o-matic-plots-on-coffee-mugs/>

观看一台自主机器是一件令人着迷的事情。自动洗车机、从窗户后面拿起产品的汽水自动售货机、绘图仪，或者像 CNC 或 3D 打印机这样的机器人都是值得一看的。虽然[EngineerDog]称 Mug-O-Matic 是一个微型数控系统，但我们认为它更像是咖啡杯的绘图仪。正如你在下面的视频中看到的，看起来仍然很有趣。

该设计有大约 60 个打印部分，并在业务端使用了一支记号笔。它接受 gcode，甚至可以用我们自己的 Jolly 扳手在你最喜欢的杯子上装饰图案，所以你知道我们喜欢它。

当然，记号笔的印记不会永远留在马克杯上，但文章说你可以烘烤马克杯，使印记永久存在。起初，该项目使用廉价的 Arduino Nano 分线板。然而，这是为了用 Arduino 的电源驱动伺服系统而设置的，所以现在有了一个定制的 PCB，而不是电路板手术。

最终的图纸有点摇晃，我们不确定这是因为设备安装在杯子顶部的方式，还是因为钢笔所在的支架的杠杆臂。然而，对于一个有趣的项目来说，它做得很好。

当我们看到这个的时候，我们不禁想到了蛋机器人和 T2 球机器人。我们还想知道这是否可以在玻璃上绘制，然后用氢氟酸或盔甲蚀刻。
