# 自制 FPV 座舱中的高速飞行

> 原文:[https://hackaday . com/2021/03/19/high-speed-flight-in-a-home brew-fpv-cockpit/](https://hackaday.com/2021/03/19/high-speed-flight-in-a-homebrew-fpv-cockpit/)

第一人称视角，或称 FPV，在遥控飞机飞行圈子里非常流行。一副视频护目镜让飞行员可以从飞机上看到自己在飞行。为了让事情变得更加真实，在[FliteTest] [的团队建造了一个带有工作控制的复制驾驶舱，并将其带到实地进行测试。](https://www.youtube.com/watch?v=r43zuFWtdaI)(视频，下面嵌入。)

由于飞行员戴着视频护目镜，美学在这里不是关键。一个遥控发射器的万向节和电位计被掏空。前者安装了一个长杆用于副翼和升降舵控制，后者连接到方向舵的踏板上。甚至还有一个合适的油门手柄。这是一个低预算的建筑，PVC 管和蹦极带做了很多工作，但这丝毫没有降低有趣的因素。该团队后来加大赌注，用钻机以高达 120 英里/小时的速度飞行一个更快的模型。

有趣的是飞行员在坠机时的本能反应，因为第一人称视角视频和现实控制的结合给人一种强烈的感觉，即一个人实际上是在一架真实的飞机上。事实上，FPV 飞行的历史比你想象的要长，可以追溯到 20 世纪 80 年代。
