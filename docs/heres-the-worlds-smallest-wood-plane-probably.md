# 这是世界上最小的木飞机…可能是

> 原文：<https://hackaday.com/2022/08/10/heres-the-worlds-smallest-wood-plane-probably/>

诚然，我们无法确定由丹尼尔·德恩特蒙特组装的微型木飞机是否是世界上最小的，但我们愿意相信他的话。最起码，我们肯定没见过更小的。

在下面的视频中，[Daniel]用一小块木头制作了这个小工具，首先用带锯切下一个正方形，然后切换到一个小手锯来切割各个部分。这些被粘在一起成为飞机的机身，一个小钻头的柄被用来压住楔子和刀片。总的来说，它大约有 1/2 英寸长，功能齐全……或者至少和 1/2 英寸的木刨一样功能齐全。
