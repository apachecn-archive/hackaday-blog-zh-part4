# 润滑机器人的双手:可变摩擦使机器人手套更像我们自己的

> 原文：<https://hackaday.com/2018/09/23/greasing-robot-hands-variable-friction-makes-robo-mitts-more-like-our-own/>

除非你是在机器人学或修复学领域，否则你可能会认为我们的手拥有良好的运动技能是理所当然的。对于一个没有专用抓笔装置的机器人来说，拿起并使用一支笔可不是一件容易的事。用同样的夹子拿着手机同样令人生畏，更不用说拿着手机四处移动了。我们的手的神奇之处在于它的形状和质地，这使得笔和手机在某一时刻可以滑动，而在下一时刻又可以握得紧紧的。耶鲁大学的抓取实验室已经建造了一个抓取器，它通过改变机械手的摩擦力来解决这个问题。

一组带有低摩擦表面的弹簧加载板条允许被夹持的物体自由移动，但当机器人施加更多压力时，板条缩回，高摩擦表面接触物体。这类似于我们的手指表面是圆形的。当我们用手轻轻擦过某物时，它们会擦到表面，但当我们握紧时，我们柔软的肌肉接触到物体的表面，我们就可以握紧了。Grab 实验室在演示解决方案和采取措施开发更有能力的机器人方面做得很好。天网万岁。

我们不缺乏可供选择的手爪设计，包括[气动硅胶](https://hackaday.com/2013/06/30/air-powered-gripper-design-makes-them-easy-to-produce/)和一种[符合物体表面的](https://hackaday.com/2013/12/03/compliant-robot-gripper-wont-scramble-your-eggs/)，类似于我们的手。
