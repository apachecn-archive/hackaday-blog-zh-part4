# 这件艺术品的解释和建造都很美

> 原文:[https://hack aday . com/2019/01/15/both-explain-and-build-for-this-artwork-are-beautiful/](https://hackaday.com/2019/01/15/both-explanation-and-build-for-this-artwork-are-beautiful/)

有时你会遇到难以描述的项目，就像这个项目一样。所以也许最好从这个项目不是什么开始。它不是一个球体。它不是一个永久的能源装置。它既没有草率的构建，也没有草率的视频。这是一个电动菱形八面体，由[Wolfram Glatthar]精心设计，具有高质量的零件和对细节的关注。它的核心是建造一个摩擦最小的移动装置。没有机械装置的研磨，电子设备可能会先磨损。低摩擦也意味着低功耗，一小时的阳光可以让设备运行两天半。看看休息下面的视频。

沿着侧面是一个平衡环，带有螺纹螺丝插座和承重磁铁，利用排斥力悬挂菱形八面体的大部分。所有东西都是通过一个陶瓷球接触蓝宝石玻璃板来稳定的，这是一些非常坚硬的材料之间的单点接触。透明的蓝宝石加深了一切都在漂浮的错觉，但真正的磁悬浮需要更多的能量。

声悬浮不能被忘记是漂浮的另一种动力来源，否则你可以欺骗和使用[闪光灯诡计](https://hackaday.com/2017/05/22/gravity-defying-drips-of-a-bike-pump-controlled-fountain/)。
