# 范设计的乐高空间站

> 原文：<https://hackaday.com/2020/02/02/lego-space-station-designed-by-fan/>

大多数人都喜欢玩乐高，这已经不是什么秘密了，但是有些人真的是喜欢到了极致。乐高的 Idea 平台让人们提交设计供审查，也让用户对这些设计进行投票。如果被接受，该公司与设计师合作，把一个工具包投入生产，他们分享利润。克里斯托夫·鲁格提交了他为国际空间站设计的作品，三年后，你可以在乐高网站上买到。

该套件有 864 个部件，成品模型为 12 英寸×19 英寸×7 英寸——可能需要比喝咖啡休息更长的时间才能完成。该模型甚至包括两个旋转太阳能阿尔法旋转接头，允许太阳能电池板与太阳对齐。如果你有 3 个小时的时间，你可以在下面的直播录像中看到[斯科特]正在建造他的。

70 美元的价格还不错。你可以花更多的钱去买一个可能需要更多工作的工具包。它看起来也很准确。

我们怀疑这个构建可能超出了我们的平均乐高技能水平。然而，乐高网站上的构建说明链接还没有拉起任何东西，所以现在说还为时过早。如果它不是那么大，我们可能会做[纸质版](https://issfanclub.eu/build-your-own-iss/)来代替。

我们不应该对乐高空间站感到惊讶。毕竟，我们已经看到了[无人机](https://hackaday.com/2020/01/15/lego-drone-finally-takes-off/)和[潜艇](https://hackaday.com/2019/07/28/lego-powered-sub-built-in-a-water-jug/)。
