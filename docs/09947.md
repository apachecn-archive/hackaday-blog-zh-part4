# 直升机里充满了压缩空气

> 原文：<https://hackaday.com/2021/04/27/helicopter-is-full-of-compressed-air/>

[汤姆]喜欢建造小型直升机，并决定建造一架用压缩空气运行的直升机。(视频，嵌在下面。)原来比他想象的要难一点。最初，他尝试的是压缩空气四轴飞行器。他已经研究过空气涡轮机，但是让一辆车自己升到空中却发现有很多隐藏的问题。

[Tom]经过了大量的设计考虑，得出了直升机的设计方案。他考虑过反向旋转道具，但这涉及到一系列问题。他最终选定了一个带尾桨的探测器，尾桨安装在一根长吊杆的远端，以使杠杆臂减少尾桨所需的功。

这个构建的一些最好的部分是测试设置。我们喜欢他用来测试螺旋桨推力的试验台。这让他明白他的四轴飞行器设计在他手头的道具下是不可行的。他的纹影摄影装置也很有趣，提供了一些很酷的气流照片。

最终，结果有些令人失望。然而，有时你从失败中学到的比成功更多。此外，一个敏锐的黑客读者可能会有一些关于如何使飞行器更好的建议。我们觉得已经很接近了。

我们已经看到[Tom]以这样或那样的形式在这方面工作了一段时间。他的一些[气动设计](https://hackaday.com/2020/12/02/diaphragm-air-engine/)相当复杂。
