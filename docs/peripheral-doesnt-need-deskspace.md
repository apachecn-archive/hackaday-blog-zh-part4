# 外设不需要桌面空间

> 原文：<https://hackaday.com/2020/09/11/peripheral-doesnt-need-deskspace/>

我们中的一些人喜欢新硬件。抽屉里堆满了游戏手柄、可卷起的键盘和那些形状滑稽的符合人体工程学的鼠标，这绝对没有什么可耻的。MyTeleTouch 不会满足你对新硬件的渴望，因为[Dimitar Danailov]没有设计你持有的硬件，因为它将你的手机用作一个包罗万象的人机界面设备。一个加密狗插入标准的 USB 端口，你的 Android 手机可以通过蓝牙模仿 USB 键盘、鼠标或游戏手柄。

很有可能你已经用你最喜欢的硬件安装了你的主电脑，但是我们认为我们已经找到了一个实用的极简附件。还记得上次你启动一个过时的 Windows 桌面，并找出一个有问题的 USB 插头的旧鼠标吗？当您只想在智能电视上键入密码时，您有多长时间在移动盒子的底部四处摸索，试图找到一个专有的无线键盘加密狗？RetroPi 和一个游戏控制器呢？MyTeleTouch 不会改变你的日常体验，但当你不想带着全尺寸键盘走下三段楼梯，在一台自动忘记它有触摸屏的机器上按下{ENTER}时，它就会出现。如果你没有机会经常扮演英雄，你可以选择扮演反派。把这个藏在同事的 USB 接口上，当他们认为你在发短信时，你可能在摆弄他们的光标。

我们喜欢每个人都能一笑置之的恶作剧，我们喜欢 T2 的小键盘，这个提高了(空间)标准。
