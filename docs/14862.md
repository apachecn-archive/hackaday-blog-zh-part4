# 机器人在有趣的户外游戏中追逐球

> 原文:[https://hack aday . com/2022/09/22/robots-chase-down-balls-in-fun-outdoor-game/](https://hackaday.com/2022/09/22/robots-chase-down-balls-in-fun-outdoor-game/)

艺术装置并不总是静态雕塑或纯粹的美学。在 *[的情况下，Kiba](https://www.niklasroy.com/kiba/) ，*他们可以成为有趣的游戏供大家享受。

Kiba 行动的目的是让玩家收集游戏场上所有的“球”,这些球代表几勺冰淇淋。收集球是通过机器人完成的。每个球员表面上的任务是收集一种颜色的球或其他颜色的球，但球员往往决定和谐地一起工作。在游戏开始时，通过翻转一个大碗来释放球。这是一半的乐趣，这是通过拉一根线来实现的，这根线将容器倒置并分散球。

遥控机器人本身来自该组织建造的名为 [*Bubble Blast 的早期艺术装置。*](https://niklasroy.com/bubbleblast/) 它们是使用 3D 打印底盘建造的，两侧的车轮由 DC 齿轮马达驱动。通过坦克式转向，它们可以就地旋转，提供良好的机动性。一台 Arduino Nano 操控着这场表演，通过 433 MHz 无线电链路接收命令。电源通过 DeWalt 无绳电钻电池提供，机器人通过街机操纵杆控制。它们被标上颜色以匹配游戏中的球。

就艺术装置而言，它可能不花哨也不装腔作势，但它看起来肯定很有趣。我们确信它最终会引导许多玩家走向激动人心的[重量级战斗机器人世界。](https://hackaday.com/2022/08/22/small-combat-robots-pack-a-punch-in-antweight-division/)休息后的视频。
