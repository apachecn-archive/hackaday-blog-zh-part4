# 不是很多的真空捡拾工具

> 原文:[https://hack aday . com/2022/09/18/a-vacuum-pick-up-tool-for-not-a-lot/](https://hackaday.com/2022/09/18/a-vacuum-pick-up-tool-for-not-a-lot/)

当使用微尘表面贴装元件时，真空拾音器笔是一个非常重要的工具。不是试图用镊子移动零件，只成功地将其弹到中间距离，而是一个带有真空馈送的小橡胶吸盘，让您可以将它捡起来，并准确地放置在需要的位置。不幸的是，好的真空吸尘工具是有价格的，非常便宜的工具不值得花费。

[这就是[ TDG(贝拉) ]的 SMD 真空吸取工具的用武之地](https://hackaday.io/project/187342-smd-vacuum-pickup-tool)。廉价工具的问题仅仅在于它们的手动真空是无效的，它们需要带有吸盘的探针阵列。解决方案是将一个带有低压电机的小型真空泵与一个 3D 打印适配器连接到一个廉价真空工具的业务端，并使一个有用的工具成为结果。

我们在休息时间下面放了一个工具运行的短片。有点吵，但很明显表现不错。控制是通过 3D 打印侧面的气孔进行的，将手指放在上面，全部吸力都指向尖端。其结果是比之前在预算真空皮卡赌注中的竞争者更简单、更便宜。
