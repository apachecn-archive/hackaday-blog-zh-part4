# 手工制造世界上最小的喷气涡轮机

> 原文：<https://hackaday.com/2018/12/07/building-the-worlds-smallest-jet-turbine-by-hand/>

很少有机器像涡轮喷气发动机那样复杂。商用客机上的涡轮叶片是由单晶金属生长而成的。工程公差是疯狂的，一切都转得非常非常快。所有这些问题对伊戈尔来说都不是问题，他正在建造可能最终成为世界上最小的涡轮喷气发动机。他在自己的商店里做，很多工作都是手工完成的。我们不知道“拿着我的啤酒”的俄语翻译，但[伊戈尔]肯定知道。

据我们所知，这种涡轮喷气发动机的设计是一种离心流涡轮，或者与我们看到的将涡轮增压器从柴油发动机变成喷气式飞机的项目没有太大不同。这里的创新是使用车床加工压缩机级，方法是将端铣刀安装到主轴箱上，并将压缩机毛坯安装在转盘中的十字滑块上。这很奇怪，但你真的无法反驳看起来有用的东西。

[伊戈尔]通过创造一些疯狂的装置而出名。迄今为止，最令人印象深刻的是一架巨大的遥控飞机，由手工制作的喷气发动机驱动。这是一架巨大的玻璃纤维飞机，有一个自制的引擎，以每分钟 90，000 转的速度旋转[，不会分离](https://www.youtube.com/watch?v=HxjMJ9L_KbQ)。无论从哪方面来看，这都令人印象深刻。

[Igor]在 YouTube 和 Instagram 上发布了很多他的建造过程，包括用喷灯对压缩机级进行热处理。这是一个惊人的项目，即使这个微型涡轮机能够自我维持，那也是一个惊人的成就。你可以看看下面[Igor]的几个视频。
