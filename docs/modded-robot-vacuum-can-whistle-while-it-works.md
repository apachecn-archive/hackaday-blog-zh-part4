# 改进型机器人吸尘器在工作时会鸣笛

> 原文：<https://hackaday.com/2021/01/28/modded-robot-vacuum-can-whistle-while-it-works/>

在修理他的 Neato Botvac D85 时，[elad]注意到这个小家伙装了一个真正的扬声器，而不仅仅是一个压电蜂鸣器。考虑到这对于偶尔的哔哔声和滴滴答答声来说有点过分，他决定[用蓝牙接收器和第二个扬声器](https://eladorbach.blogspot.com/2021/01/neato-speaker-hack-stereo-bluetooth.html)来完善这些东西，这样机器人就可以在变脏时播放一些立体声音乐。

这不是一个非常昂贵的修改。在 VHM-314 蓝牙接收器、3 瓦 PAM8403 放大器和一个匹配的扬声器之间，[elad]说他只花了几美元。对于一个机器人真空吸尘器来说，这确实是一个很小的代价，它可以在房子里四处走动时播放自己的主题音乐。休息之后，可以在视频中听到 Neato 新音乐天赋的小演示。

也许不出所料，音频硬件在最大音量下消耗了机器人足够多的电池，运行时间明显减少。他现在并不太担心这个问题，但[elad]提到，如果它最终使吸尘器无法完成整个清洁周期，他可能会考虑添加一个专用电源来保持音乐继续播放。

尽管 iRobot 在早期给予了一些鼓励，但我们并没有看到像你想象的那么多机器人真空黑客。窥探这些自动化管家的内部总是很有趣[，尤其是当](https://hackaday.com/2016/04/08/robotic-vacuums-get-torn-down-for-design-showdown/)[是一台定制的机器](https://hackaday.com/2018/02/24/this-3d-printed-robotic-vacuum-sucks/)时。
