# 你必须有很高的智商才能理解这个瑞克和莫蒂传送门枪的复制品

> 原文：<https://hackaday.com/2018/09/02/you-have-to-have-a-very-high-iq-to-understand-this-rick-and-morty-portal-gun-replica/>

现在才刚刚九月，但这仍然意味着你已经开始准备你的万圣节服装了。如果去年有任何迹象的话，那么今年最受欢迎的服装将会是，到目前为止，里克和莫蒂的服装。关于这一点有很多话要说，但让我简化一下:如果你装扮成*瑞克和莫蒂*中的瑞克，你就不是瑞克。你是个毛蒂。

然而，万圣节是一些角色扮演和道具制作活动的绝佳机会，今年[【达人】已经完成了](https://hackaday.io/project/101776-interdimensional-portal-gun)。他正在用投影仪从*瑞克和莫蒂*那里组装传送门枪。是的，无论你指向哪里，它都会显示入口。它实际上是在建造一些东西，而不是买一顶蓝色的假发和一件实验服。瑞克会很骄傲的。

这个传送门建筑复制品的关键是与那些用喜讯照亮房屋四周的圣诞投影仪相同的技术。这些只是旋转框架中的微小的 gobos，用高亮度 led 照明。这很容易安装在 3D 打印的门户枪外壳中，当你添加一些 18650 LiPOs，一个声音扬声器和一个冷却的 PC 风扇时，你就有了一个真正的投影门户枪的气质。

虽然这只是一项正在进行中的工作，但迄今为止这是一项了不起的成就。万圣节就要到了，这对外面那些毛蒂们来说是一个很好的建筑。
