# 小小的木工车间堆满了节省空间的工具

> 原文：<https://hackaday.com/2019/09/18/tiny-woodshop-is-packed-with-space-saving-hacks/>

公平的警告:一旦你看了[【Stephen】的小作坊之旅](https://www.youtube.com/watch?v=gCtRWwvGHeY)，你就正式没有借口解释为什么你需要扩大你的作坊了。而且，一旦你看到他的存储和组织技巧，你会羞愧地在任何你称之为家的地方复制一些。

[Stephen]的木工车间是一个舒适的 6 英尺 x 8 英尺(1.8 米 x 2.4 米)花园小屋。前墙几乎完全被门和窗户占据，减少了可用的墙面空间，但提供了充足的自然光，并防止小空间诱发幽闭恐惧症。剩余空间的每一平方英寸都得到了优化和组织。[Stephen]明智地避开了笨重的橱柜，转而使用悬挂式工具架，所有工具架都用法式夹板灵活地安装在墙上。每样东西都有一个位置，因为每件手工工具都在伸手可及的范围内，所以它会一直存放到需要的时候，用完后又会放回去。不过，这家商店拥有的不仅仅是手工工具；车床、钻床、厚度刨床、砂光机、空气压缩机、线锯、带锯，甚至台锯都可以放在里面。甚至还有由“野兽”[Stephen]的 DIY 吸尘器提供的灰尘收集功能。

不管你是在木材、金属还是硅行业工作，我们都可以从[斯蒂芬]的店里学到一些东西。这是一个效率和组织的模型，虽然他不太可能在那里建造一个全尺寸的[安妮女王]梳妆台，但从他的博客中可以明显看出，他用它做了很多事情。可惜我们上次去[逛小店的时候错过了这个。](https://hackaday.com/2017/09/06/ask-hackaday-how-small-is-your-shop/)
