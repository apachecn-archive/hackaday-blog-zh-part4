# 复古控制台的新型控制器

> 原文：<https://hackaday.com/2018/10/12/new-controller-for-retro-console/>

在复古游戏的世界中，当使用模拟器和非本地硬件时，使用任何碰巧可用的 USB 控制器是很常见的。这允许我们在使用可配置控制器时获得怀旧的外观。一件不常见的事情是使用原始硬件，同时仍然找到一种方法来使现代控制器适应旧的控制台。这正是你所需要的，当你在一个有着臭名昭著的糟糕控制器的平台上玩复古游戏的时候。

[Scott]喜欢他的 Atari 5200，但不居中和一般可怕的操纵杆即使在 80 年代初控制台处于鼎盛时期也没有得到很好的接受。他认为使用 Playstation 2 的双震动控制器可以提供更好的游戏体验，并开始构建适配器。他发现，在某种程度上，双震动控制器几乎是雅达利的完美搭档，因为它已经内置了两根模拟控制杆。还有一系列关于双冲击控制器与 AVR 微控制器配对的信息，所以他不必重新发明轮子。从那时起，这只是两个硬件之间的配对通信协议的问题。

项目页面详细介绍了 SPI 通信协议以及 Atari 和 Playstation 控制器的需求。如果你是一个复古游戏迷，真的很喜欢通信协议，或者一直对你的雅达利爱恨交加，因为控制器*就是那么糟糕*，这是值得一试的。不过，如果这太过分的话，[还有其他方法可以让雅达利怀旧](https://hackaday.com/2018/09/05/retro-console-upgrade-gives-atari-flair/)。
