# 免提触觉盲文显示器正在掀起波澜

> 原文：<https://hackaday.com/2020/06/07/hands-free-haptic-braille-display-is-making-waves/>

在过去的几个月里，世界上大多数人都尽量避免接触公共物品。不幸的是，任何依赖盲文标志、地形图和音频插孔的弱视者都没有这种奢侈——至少目前没有。

德国拜罗伊特大学的一组研究人员正在想办法解决这个问题。他们正在开发 [HaptiRead，这是一个半空中的触觉反馈系统](https://techxplore.com/news/2020-06-digital-braille-speaker-mid-air-pulses.html)，可以用作盲文或 3D 形状的无触摸、可刷新显示器。HaptiRead 基于 Stratos Explore 开发套件，该套件包含 256 个超声波传感器。当一个人靠近显示器时，Leap motion 传感器可以在 2.5 英尺远的地方检测到他们的手，并开始通过声波提供信息。每个焦点都用不同的频率调制，以帮助区分它们。

HaptiRead 可以以三种方式显示信息:常数，模仿静态盲文显示器，逐点，逐行。研究人员声称，在逐点法领先的情况下，试验的准确率高达 94%。该系统仍在发展中，因为它只能做四个细胞的点组合，需要做六个才准备好。休息后查看简短的讲解者视频，或者[阅读小组论文](https://arxiv.org/pdf/2005.06292)【PDF 下载】。
