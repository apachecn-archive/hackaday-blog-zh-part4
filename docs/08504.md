# 制造一把“无法撬开”的锁

> 原文:[https://hackaday.com/2020/11/29/making-a-unpickable-lock/](https://hackaday.com/2020/11/29/making-a-unpickable-lock/)

每次制造商把一种新的“无法撬开”的锁推向市场，业余和专业锁匠都会用新产品来证明他们是错的。[Stuff Made Here]的[Shane]决定尝试设计和制造一把无法撬开的锁，并发现那个特别的兔子洞比预期的要深得多。(视频，嵌在下面。)

大多数普通的弹子锁都可以被撬开，这要归功于轻微的松配合和微小的制造缺陷。通过在圆柱体上施加张力的同时提升或撞击销，可以使销在剪切线上一个接一个地结合。一旦所有的插销都被固定在正确的位置，就可以解锁了。

[Shane]的设计旨在通过将所有锁销锁定在设定的任何位置，防止锁销一个接一个地被设定在解锁位置，并防止在转动锁芯测试密码时进一步操作。从理论上讲，这应该阻止进行挑选的人知道是否有任何一个销子处于正确的位置，迫使他们采取简单地尝试不同组合的困难且耗时的方法。

[Shane]对挑战性项目并不陌生，这次也不例外。许多零件不得不重复制造多次，即使是在他的[设备齐全的家庭机械车间](https://hackaday.com/2020/05/16/cnc-plasma-cutter-filter-gets-the-slag-out/)。当滚筒旋转时，将销保持在设定位置的机构特别难以可靠地工作。他明确表示，这种锁纯粹是一种教育练习，由于其机械复杂性和难以加工，在商业上不可行。

一名当地锁匠用标准技术撬开了锁，但真正的考验还在后头。许多读者可能已经想到了[【LockPickingLawyer】](https://hackaday.com/2019/10/03/pistol-safes-poor-design-means-biometric-sensor-bypassed-in-seconds/)这个名字。[Shane]已经与他取得了联系，并将在进一步完善后给他发送一把锁进行测试，我们期待看到结果！
