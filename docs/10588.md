# 车库半导体厂获得反应离子蚀刻升级

> 原文：<https://hackaday.com/2021/06/29/garage-semiconductor-fab-gets-reactive-ion-etching-upgrade/>

这是一个我们很少有人会面临的问题:一旦你制造出第一个自制集成电路，下一步你会做什么？如果你是山姆·泽洛夫，答案很清楚:制造更好的集成电路。

至少这是[Sam]的计划，他的新反应离子蚀刻装置旨在使其成为可能。虽然[的 Z1 双差分放大器芯片](https://hackaday.com/2018/05/03/more-details-on-that-first-home-made-lithographically-produced-ic/)取得了巨大成功，但他用来制造芯片的光刻工艺有其局限性。他使用的化学蚀刻工艺有点繁琐，如果蚀刻剂渗透到掩模下面，容易对掩模进行底切。顾名思义，RIE 使用高活性离子的等离子体来进行蚀刻，从而获得更精细的细节，并为使用更先进的材料打开了大门。

[Sam]的 RIE 钻机看起来像水管工的不锈钢噩梦，中间是一个真空室，用于蚀刻晶片。排空空气后，将少量氟化气体——四氟化碳或总是令人愉快的六氟化硫——加入到腔室中。高压馈通提供产生等离子体所需的射频能量，等离子体将氟离子从工艺气体中分离出来。带负电荷且极其活泼的氟离子被吸引到晶片上，在那里它们攻击并蚀刻掉不受光致抗蚀剂层保护的表面。

这一切听起来足够简单，但下面的视频揭示了复杂性。有很多细节，像正确测量真空，避免触电，防止真空泵油爆炸，以及处理有毒废物。向[山姆的爸爸]脱帽致敬，感谢他安全地将废气通过车库门。这与[【惠更斯光学】的最新努力](https://hackaday.com/2021/06/20/tiny-tesla-valves-etched-in-glass/)有关“与氟有关的最酷的事情”奖。
