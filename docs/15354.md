# 欢迎回来，超级大陆！

> 原文:[https://hackaday.com/2022/11/12/welcome-back-supercon/](https://hackaday.com/2022/11/12/welcome-back-supercon/)

最近两个 11 月，Hackaday 的年度聚会以远程模式举行:Remoticon 而不是 Supercon。当我还在从时差中恢复过来的时候，我在思考现场活动和虚拟活动的利弊。想知道明年我们如何结合两者的优点。来和我一起头脑风暴吧！

拥有一个直播的超级电脑的明显好处是可以方便地与在场的每个人交谈、交易代码提示、生活经历和必看项目。您可以亲自实时交易徽章附加产品，无需等待海关清关。简单地一起出去玩有一种真正的魅力，在一起吃墨西哥玉米卷时这样做更好。自发的合作既容易又自然。最后，虽然你可以在 YouTube 上看到有人用霓虹灯变压器电死了一个 twinkie，但你闻不到臭氧的味道。

与此相反，所有昂贵的旅行，前面提到的一些人的时差，以及现实世界的限制，只有这么多人可以在一个给定的物理空间一次适应。

Remoticon 最大的好处是能听到那些无法亲临现场的人的声音，不管是因为地理原因还是金钱原因。因为所有东西都在网上，所以没有遗漏，任何人都可以自由地参加这样或那样的谈话。在线聊天频道在 Remoticon 期间也有更好的参与——也许因为它们是镇上唯一的游戏——但那是一个更加全球化的社区。

对于墨西哥玉米卷我们可能无能为力，但是我们能做些什么来整合 Remoticon 的优势呢？我们*做了*一个阶段的直播，我们有两个聊天频道一直开放评论。如果你远程参加了 Supercon，请告诉我们进展如何，如果你有任何建议来改善我们下次的远程体验。因为最终，我们希望 Hackaday 像黑客社区本身一样包容和全球化。
