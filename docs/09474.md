# 超大型气象站使用老式模拟仪表

> 原文:[https://hack aday . com/2021/03/10/super size-weather-station-uses-antique-analog-meters/](https://hackaday.com/2021/03/10/supersized-weather-station-uses-antique-analog-meters/)

对于我们大多数人来说，获取天气信息就像解锁智能手机或打开电脑，将应用程序或浏览器指向自己选择的天气网站一样简单。这一切都很好，但它缺乏老式气象站的模拟表盘和彩色木箱所具有的某种派头。[BuildComics]创建的气象站将这种古老的美学与现代气象数据可用性结合在一起，然后将其提升到这个巨大的模拟气象站建筑的水平。

气象站使用 16 个独立的刻度盘，每个刻度盘都有不同的标签，用于显示特定类型的数据。他们中的一些人需要新的玻璃，其他人也需要修改线圈，以便用比设计更低的电流驱动，因为在这个项目中，每个线圈都将由两个 Arduinos 中的一个驱动。每一个都通过一个电位计连接到一个微控制器的输出端，电位计控制指针的位置，以适应各种不同设计的仪表。微控制器本身从[BuildComics]家门外的真实世界传感器和互联网上获得天气信息，这使得人们能够收集到尽可能多的最新天气信息。

这些旧米的定制量令人印象深刻，更令人印象深刻的是项目的最终重量。[BuildComics]报道称，需要两个人才能将它抬到壁挂支架上，考虑到这些老式模拟仪表中的铁含量，这并不奇怪。而且，尽管在现实世界中不再常见，这些古老的古董仪表在气象站之外还有很多用途。
