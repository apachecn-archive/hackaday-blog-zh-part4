# Furby 加 Borges 等于 Borgy

> 原文：<https://hackaday.com/2018/12/08/furby-plus-borges-equals-borgy/>

如果你和一个 20 世纪文学大师杂交，你会得到什么？[博格](https://www.hackster.io/roni-bandini/furby-turns-into-a-writer-s-animatronics-bot-132780)。阿根廷黑客[Roni Bandini]发现了一个旧的 Furby，并决定通过改变它的个性来入侵它。他的灵感来自阿根廷作家豪尔赫·路易斯·博尔赫斯，超现实主义写作的先驱之一。这个想法是，在一天中的任意时间，Borgy 将分享来自 Borges 的一点智慧来激励和启发。

[Roni]黑掉了 Furby，用一个更强大的扬声器替换了它，并建造了一个底座来放置更大的扬声器和一个可以激活 Borgy 的开关。他还使用了一个 Arduino Nano 和一个 Sparkfun MP3 播放器 shield，里面装有博尔赫斯的样本。

当 Furby 说话时，它分享了博尔赫斯的一些智慧。这是一个简单，但却非常有效的方法，对于寻求灵感的人来说非常有用。或者，正如博尔赫斯自己曾经说过的:*“不要说话，除非你能改善沉默。”*
