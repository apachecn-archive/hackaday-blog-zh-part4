# 将旧的拨动开关变成复古技术的展示品

> 原文：<https://hackaday.com/2019/10/09/turning-old-toggle-switches-into-retro-tech-showpieces/>

虽然我们这些黑客社区的人通常专注于创造新的东西，但恢复旧的东西也有很多可说的。找到一件硬件，让它看起来和工作起来都像新的一样，这是非常令人满意的，几十个 YouTube 频道和博客的存在只是为了满足对更多修复内容的需求。

名副其实的[开关和杠杆]已经在复古浪潮中骑行了一段时间，而[他关于恢复和修复老式拨动开关的视频](https://www.youtube.com/watch?v=ihTscox0Ems)表明他已经学会了一两个值得分享的技巧。这些开关都是跳蚤市场上找到的，都是曾经风光过的矮胖的野兽。但是旧零件是为了持久耐用而制造的，事实证明它们足够坚固，能够承受任何修复的第一步:拆卸。大多数开关很容易被撬开，但有几个需要先钻掉铆钉。接下来的清洁和抛光步骤非常简单，尽管我们喜欢关于微网磨料和抛光剂的提示。另一个很好的建议是使用酚醛树脂多氯联苯作为破损酚醛塑料体的修复材料；它们在化学成分上很相似，虽然它们可能与原来的不完全匹配，但当它们与 CA 胶和小苏打一起作为填充物时，它们可以进行很好的修复。

3D 打印修复也可行，但保持历史一致性也令人满意。毕竟，庆祝工程历史才是修复工程的真正意义所在。即使你正在建造新的东西，你也可以用[开关和控制杆]制造的这些酸蚀黄铜饰板让它看起来复古酷。
