# 从零开始的乳胶波纹管

> 原文:[https://hackaday.com/2019/08/12/latex-bellows-from-scratch/](https://hackaday.com/2019/08/12/latex-bellows-from-scratch/)

如果你认为来自 Bellowphone 频道的半球形灯泡(Len)是一个抽水马桶浮子，这是可以理解的。这是他设计的风箱，类似于自行车喇叭的可挤压部分，但更结实，在早上六点时不那么烦人。这些橡胶挤压球是老派的最好的方式，工艺从他的视频的每一秒钟推出。[Len]视频的背景充满了工具、材料、例子和仪器，就像我们的办公室和创客空间充满了烙铁、led 和无源器件一样。

他的视频介绍了制作乳胶波纹管的所有步骤，从一个硬茎灯泡开始，然后用乳胶涂上颜色。这需要一堆外套和相关的干燥时间，所以如果你需要很多风箱，你会想要多个灯泡。涂上乳胶后，我们转向被称为猪嘴大师 5000 的装置。SM5K 看起来像一个固定在台钳上的木制夹具，但它是一个专门制造的过度设计的卡盘，有四个球轴承固定在台钳上。当乳胶足够厚时，模板被移除，灯泡被修复，然后，更多的涂层。每个球大约有二十层，每层之间有三个小时，这至少是一个周末的工作。好事降临在穿大衣的人身上。最后的步骤是煮沸球茎，并加入硅酮防腐剂。如果保养得当，它们可以使用十年。

我们在这里看到许多电子仪器和自动化仪器，球形球肯定是人机界面的一部分，但是我们从[Len]那里看到的技术将允许任何人[设计他们自己的更有利于机械化的风箱。[Len]说他的灵感之一是[Harry Partch]和他的 Blo-Boy，一种由壁炉风箱驱动的风琴。我们认为这些挤压球甚至更好。](https://hackaday.com/2017/02/12/blade-runners-voight-kampff-test-gets-real/)
