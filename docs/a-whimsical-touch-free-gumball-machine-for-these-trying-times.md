# 一个异想天开的免触摸口香糖贩卖机

> 原文：<https://hackaday.com/2020/12/10/a-whimsical-touch-free-gumball-machine-for-these-trying-times/>

糟糕的是，现在某些东西在公共场合是禁止的，比如饮水机和投币糖果口香糖贩卖机——尤其是那种有趣的，你可以看着你的口香糖球从塔上下来，然后进入收集箱。希望有一天会出现接受 NFC 支付的商业免接触机器。在那之前，我们必须自己用纸板、奇思妙想和微小的碎片来制作它们。

【棕狗小工具】还用了他们的一个疯狂电路位板，这是一个 Micro:bit-to-LEGO 接口模块，用导电胶带搭建电路。火箭底部有一个距离传感器，还有一个伺服系统来分发口香糖。这整个建筑非常棒，但是我们特别喜欢乐高技术梁的巧妙使用，既抓住了口香糖球，又防止了下一个口香糖球跑到任何地方。休息之后你可以看到它的实际效果。

挥挥手，接收口香糖球对最终用户来说是再简单不过了。三个机器人的方法需要更多的工作。
