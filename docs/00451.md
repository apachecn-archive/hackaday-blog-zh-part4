# 机器人侵犯你的个人空间

> 原文：<https://hackaday.com/2018/09/05/robots-invade-your-personal-space/>

如果你曾经不得不完成一项任务，比如通过远程连接搭建一个乐高模型，你就会知道这些挑战就像一个荒谬的小学小组项目。给予指导的人常常难以描述他们在想什么，而做这项工作的人也难以解释指导者想要什么。“把蓝色积木翻过来。不，只有一半。回去吧。现在转过去。不，另一边。不要！不是顺时针，是向下。那是向上！天啊。你甚至 listening‽“好时光。

虽然你可能不会每天都遇到这种情况，但日本庆应义塾大学有一种直观的方式，通过摩尔/斯威兹体验，给教师一种与被指导者进行身体互动的方式。教练肩上挂着一个典型的海盗鹦鹉式的照相机。两个手臂由教练控制，教练可以通过立体摄像机看到世界各地的第一人称视角。这种与用户环境交互的自然方式允许肌肉记忆从教练传递给佩戴者。

对于一些其他风格的远程呈现，请参见这个深海机器人和一个圆柱形屏幕，它看起来像是有人直接从全息甲板传送过来的 T2。
