# 这款基于 Arduino 的喷墨打印机无需速度

> 原文:[https://hackaday . com/2020/06/13/不需要借助基于 arduino 的喷墨打印机提高速度/](https://hackaday.com/2020/06/13/no-need-for-speed-with-this-arduino-based-inkjet-printer/)

说到计算机，似乎唯一重要的就是速度。一般来说，越多越好，外设也一样。我们需要最快的网络适配器、最快的显卡和最快的打印机。那么，为什么世界上有人会故意制造一台非常慢的喷墨打印机呢？当然是为了艺术。

至少这是下面视频中[HomoFaciens]告诉我们的故事。他的努力是为了支持一个朋友的艺术项目，该项目寻求在一卷纸上缓慢但连续地打印。[HomoFaciens]的打印机基于惠普 C6602 喷墨墨盒，这是一种高价耗材，这使得购买新打印机比在耗尽后更换打印机更有吸引力。在弄清楚如何驱动打印头之后——通过 ULN2803 达林顿阵列驱动芯片的 5 到 6μs 18 伏脉冲似乎就能做到这一点——他将所有东西安装到一台旧的 3D 打印机的机架上。有趣的是，看着图像慢慢地建立起来——这是打印机通常隐藏起来不让别人看到的——并且看到如何通过隔行扫描每一行来增加打印机的 DPI 计数。

在视频快结束的时候，我们看到了他的[“纹身枪打印机”](https://hackaday.com/2016/06/16/homofaciens-shows-off-with-diy-paper-printer/)，这让我们想起了他这些年来做的所有其他很酷的事情。从[由回形针和纸板制成的数控机器](https://hackaday.com/2016/04/27/cardboard-and-paperclip-cnc-plotter-destined-for-self-replication/)到[由电阻轮制成的编码器](https://hackaday.com/2016/06/15/wheel-of-resistors-form-unique-rotary-encoder/)，【HomoFaciens】有一些有趣的设计，你真的应该去看看。
