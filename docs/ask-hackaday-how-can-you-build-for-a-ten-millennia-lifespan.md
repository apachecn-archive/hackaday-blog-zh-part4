# 问哈卡黛:你如何能创造一万年的寿命？

> 原文：<https://hackaday.com/2019/07/01/ask-hackaday-how-can-you-build-for-a-ten-millennia-lifespan/>

最近有很多关于 Long Now 基金会和杰夫·贝索斯花费 4200 万美元建造一个预计能运行 10000 年的巨型机械钟的新闻。我们不确定我们是否真的同意这是一个真正的万年钟，因为它从参观它的人那里吸取了部分能量。据我们所知，发明家丹尼·希尔斯已经制作了这种时钟来储存来自几个来源的能量，当它有足够的能量时偶尔会发出钟声，所以我们不确定它是如何真正维持自身的。然而，它确实让我们想到了一个有趣的问题:你如何设计出一种能真正工作一万年的东西？

## 为什么？

第一个问题可能是你为什么想要？我们不在时钟上出售。但至少有两个简单的答案:安全储存非常糟糕的东西和世代星际飞船。我们当然在制造需要长期封存的核材料和生物材料。如果我们今天想去另一个星系，我们就必须建造一艘飞船，让我们的后代去最近的恒星。在这两种情况下，事物都必须持久，要么不需要修复，要么是可持续的。

## 多大才算老？

这座钟似乎主要是机械的，我们确实有一些纯粹机械的东西持续很长时间的例子，尽管并不总是处于最佳状态。时钟一年滴答一次也无妨。

马耳他的巨石庙宇可以追溯到 5000 年前，比埃及金字塔或巨石阵还要古老。可以追溯到大约同一时期的是霍华德庄园，一个古老的苏格兰农庄和新庄园，也是爱尔兰的宗教场所。

当然，那些不是机器，也没有一万年的历史。在土耳其，有一些近 10，000 年历史的古老房屋和一些巨大的巨石，尽管它们几乎没有得到很好的保护。甚至耶里科的部分城墙也是同样的年代。

对于长寿命的机器来说，这些数字要糟糕得多。一些教堂的钟可以追溯到 17 世纪。在一万年的时间线上，这甚至算不上一个亮点。仍然存在的最古老的蒸汽机甚至更新，可以追溯到 1725 年。因此，建造真正的机器来维持这种规模是一个相对未经证实的想法。诚然，今天的材料更好了，但事情也更复杂了。

## 问题#1:电源

这将是一个大问题。挥挥手呼吁使用核能或电池是很容易的，但是让它们持续很长时间却是一个更大的问题。诚然，核电池可以使用一个世纪或更长时间，但离一万年还差得很远。如果你可以制造一个持续足够长时间的反应堆，你仍然需要给它补充燃料，尽管铀的半衰期是数百万年甚至数十亿年(取决于同位素)，所以这是可行的，但你必须携带很多，并有一个可靠的方式来补充燃料。

如果你被困在地球上，太阳能、地热甚至风能都有可能。然而，这些都不适用于星际飞船。众所周知，熔盐电池的保质期很长，但一旦被激活，通常不会持续很长时间。

在牛津大学，有些钟已经用一节电池响了将近 200 年，但这是一个特殊而不寻常的例子。所以权力似乎是一个关键问题。但这不是唯一的一个。

## 问题 2:机械和其他老化的东西

现实世界的零件磨损。弹簧变得不那么有弹性了。磁铁会消磁。电解电容器变干。集成电路中的金属电迁移或生长枝晶。湿气进入包装。我们不需要经常处理这种情况，因为它发生在我们正常使用的很长一段时间内。但是这些东西——可能还有更多——在几千年后会成为问题。

 [https://www.youtube.com/embed/COMTYzqe_XE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/COMTYzqe_XE?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



想象一下[有开关的一代船](https://www.popsci.com/realistic-generational-spaceship/)。机械开关。你不得不携带大量的备件或商店来修理或制造新的开关，以及这样做的原材料。你能做得更好吗？触摸屏可能太复杂了。一个 LED 和一个手指大小的孔的光传感器怎么样？没有机制，但你可能仍然不会从 LED 中获得那么长的寿命，尤其是如果它几乎一直亮着的话。

核废料库甚至更成问题，因为即使没有人在旁边照顾它，它也应该继续运作。你如何为下一波人类或蟑螂或任何继承了剩下的东西的东西留下警告？
