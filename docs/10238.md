# 带紫外光的持久性显示器

> 原文:[https://hack aday . com/2021/05/24/persistent-displays-with-uv-light/](https://hackaday.com/2021/05/24/persistent-displays-with-uv-light/)

今年的 Hackaday 奖将“反思显示器”作为其首要主题之一，而[Tucker Shannon]用一组令人印象深刻的显示器[向我们展示了他在这个主题上的最佳表现，这些显示器使用紫外光](https://hackaday.io/project/179950-uv-glow-in-the-dark-laser-displays)在黑暗中发光的材料上“书写”。这些材料在紫外线照射后会发光一段时间，因此在表面上移动像紫外线 LED 这样的光源会绘制发光的文本或简单的图形，可以很容易地被消费。

[![](../Images/5e8e61e3d43aa40b34252dcf883dc1aa.png)](https://hackaday.com/wp-content/uploads/2021/05/uv-display-thumbnail.jpg) 其中一个例子[这是我们在 2018 年](https://hackaday.com/2018/03/22/arduino-clock-jots-down-the-time-in-uv/)第一次迷恋的时钟。这是一个相当有吸引力的 3D 打印事件，这些伺服系统安装在屏幕下方，通过一对连杆移动 UV LED。在相同的 UV 触控笔介质上播放的其他产品包括由 Raspberry Pi Zero 控制的 az-el 底座上的激光器。这是一个非常有效的好主意，我们可以看到它有很大的潜力。

但迄今为止最令人印象深刻的是本文顶部图片和底部演示视频中显示的模型。磷光材料的环是显示表面本身。这个用两个滚轮移动那个环来组成 X 轴，并上下移动 Y 轴的 UV 源。与所有这些设计一样，无论写了什么都会很快褪色，让表面为下一条信息做好准备。
