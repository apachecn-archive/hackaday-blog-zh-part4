# DIY 电动滚弯机可以处理较厚的材料

> 原文：<https://hackaday.com/2019/12/22/diy-electric-roller-bender-can-handle-the-thicker-stuff/>

每个严肃的金属工人最终都会在某个时候得到一台辊弯机，但是如果你像(在车库里的这段时间)那样专注，你可能会开始自己动手制作这些东西。他的[重型电动滚弯机](https://www.youtube.com/watch?v=g3ZG2HlqLMQ)，在休息后的视频中演示，非常适合他的[小型手动机器](https://hackaday.com/2019/09/25/bend-some-bars-with-a-flywheel/)无法处理的更厚的钢和更大的半径。

这两种机器的基本概念是相同的，都有两个固定辊和位于它们之间的第三个可调节的相对辊。大多数部件都是金属碎片，每个轴都在安装在自制轴台的轴承上运行。两个固定轴通过链传动连接在一起，废工业电机通过蜗轮齿轮箱提供旋转动力。每个轴上有两个可调衬套，以保持工件对齐。旧汽车千斤顶上的丝杠用于调整移动滚轮的位置。

我们从视频中获得了一些有趣的提示，比如如何在钻床上正确对齐圆柱形工件以钻径向孔。他还以一种非常巧妙的方式使用拨动开关作为限位开关，并在工件到达终点时使用 F 形夹来激活它们。

在家里制作自己的工具是由来已久的黑客传统，我们从未在 Hackaday 上看到过这种情况。看看这个 [DIY 钻床](https://hackaday.com/2019/12/15/the-home-made-drill-press-of-your-dreams/)和[立式数控铣床](https://hackaday.com/2019/11/27/steel-tubes-and-ground-plates-form-the-skeleton-of-this-diy-vertical-cnc-mill/)。
