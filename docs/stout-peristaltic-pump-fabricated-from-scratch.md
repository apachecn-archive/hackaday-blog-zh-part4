# 从头开始制造结实的蠕动泵

> 原文：<https://hackaday.com/2022/01/12/stout-peristaltic-pump-fabricated-from-scratch/>

蠕动泵最广为人知的可能是其在泵机构不与工作流体接触的情况下泵送流体的能力。这对于食品安全应用和泵可能污染流体的其他情况至关重要。[Maciej Nowak]已经建造了一个很好的这种泵的例子，从头开始用铝制造。

构建视频详细介绍了加工过程，展示了在安装轴承和硅胶软管之前，铝制车身是如何在车床上制造的。然后制造泵轴，连同一组黄铜滚子一起沿着管子挤压，产生泵送作用。还对滚筒进行了润滑，以减少管道上的摩擦。为泵提供动力的是一个小型 DC 电机，通过一个小齿形带传送驱动力，使成品看起来颇具工业气息。

我们习惯于在这个地方看到大量的 [3D 打印泵](https://hackaday.com/2014/09/16/a-3d-printed-peristaltic-pump/) [。虽然这种结构需要一个设备齐全的机械车间，但它比任何塑料都要坚硬得多，在紧急情况下，你也可以很容易地用它来打破窗户，尽管这是一些挑剔的泵客户要求的一个模糊的功能。](https://hackaday.com/2021/05/28/3d-printing-a-centrifugal-water-pump/)

[Maciej]展示了通过泵送一些绿色液体的构建，注意到蠕动设计不需要灌注，这使得操作更加容易。它也是双向的，如果需要，可以运行得非常慢。

总的来说，它展示了金属加工的好处以及蠕动泵设计的伟大特点。休息后的视频。
