# 健谈的考斯特以一种友好的方式煽动

> 原文:[https://hack aday . com/2020/03/31/chatty-coaster-agitates-in-a-a-friendly-way/](https://hackaday.com/2020/03/31/chatty-coaster-agitates-in-a-friendly-way/)

尴尬的沉默会让人非常不舒服。谢天谢地，这是一个可以通过技术解决的问题。健谈的 Coaster 旨在做到这一点，检测谈话中的停顿，并插入有用的问题以推动事情向前发展。

杯垫是围绕一个 Arduino Micro 构建的，它使用麦克风来检测房间内的音频水平。当它检测到长时间的沉默时，它就会使用 SparkFun 音频分线板发出一个声音剪辑。这些问题从简单的到政治敏感的都有，所以你很有可能因此得到一些辛辣的话题。任何会说话的设备都有让人讨厌而不是有帮助的风险，而且聊天杯垫也有可能属于这一类。选择正确的内容似乎是关键。

总的来说，虽然这可能不是无聊公司的最终解决方案，但它可以获得一两个笑声，并作为一个学习如何在微控制器上处理音频的好方法。休息后的视频。

我们承认，当我们读这本书的时候，我们觉得似曾相识。但这一次可没那么糟。
