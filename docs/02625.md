# 垃圾箱迷笛

> 原文:[https://hackaday.com/2019/04/27/junkbox-midi/](https://hackaday.com/2019/04/27/junkbox-midi/)

你曾经窥视过你的五金废料箱的空处吗？这可能是一个拥有无限潜力的零件的仙境，它们只需要被组装起来，并取决于你囤积的东西，被编程。使用激光雕刻机也没有坏处。Sobolak 先生的箱子里的东西比一般人的要凉，至少以黑客作家的标准来看是这样。他的声音调色板项目是接口、硬件、通道和颜色的混合。有街机按钮，滑动电位计，旋转电位计，微型激光竖琴，触摸钢琴，鼓垫，赢得了垃圾盒建造非凡的称号。

在引擎盖下，我们发现常见的铜带、电线和焊料将操作员连接到一个微小的 3.2。在 BOM 中更深奥的部分，我们发现一些看起来很好玩的花式软罐。所有的代码都链接在 Instructable 中，但绝对没有理由进行精确的复制。MIDI 来自 80 年代，这个协议的库比比皆是，所以构建适合你角色的界面可能是最难的部分。Instructable 中的一些技术可能会对你有所帮助，比如如何连接压电元件，这样它就可以读取比一个破坏球或激光竖琴轻的东西，大约有你的手掌大小。

如果你想让[自己制作](https://hackaday.com/2019/04/06/make-your-own-midi-controller-with-an-arduino/)或[成为真正随机的](https://hackaday.com/2019/03/09/truly-random-midi-control/)，我们不缺 [MIDI 接口](https://hackaday.com/2018/09/04/make-the-surface-dial-do-more-things-such-as-midi/)。
