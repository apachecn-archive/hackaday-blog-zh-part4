# 带投影镜头的终极散景

> 原文：<https://hackaday.com/2022/08/02/ultimate-bokeh-with-a-projector-lens/>

散景是一个摄影术语，有点难以定义，但基本上是柔和的、美学上令人愉悦的背景模糊，通常用于使主体突出。也称为“背景分离”或“对象隔离”，在光学上实现它需要孔径低于 2.8 或优选更低的快速镜头。这些镜头可能会变得非常昂贵，但在休息后的视频中，来自[DIY Perks]的[Matt]将所有商业可用的选项打得落花流水。利用一台旧的 episcope 投影仪，他建造了一个背景分离度相当于不存在的 35mm f0.4 镜头的[摄影装置](https://www.youtube.com/watch?v=9cT0jXI7l4E)。

与大多数用于投影预先录制的图像的传统投影仪不同，瞄准镜用于投影物理对象(如书籍)的图像。由于镜头背面投影出的成像圆的尺寸，直接在相机中使用这种镜头是不可能的。直径为 500mm 时，根本没有成像传感器可以捕捉到它。相反，[马特]为图像搭建了一个投影屏幕，用普通相机从对面拍摄。

投影屏幕是通过将一片漫射膜夹在两片保持在铝挤压框架中的透明丙烯酸之间而制成的。为了阻挡所有其他光线，[马特]在屏幕两侧添加了铝罩，这也可以用来安装镜头和摄像头。镜头一侧的遮光罩安装在一个单独的铝框架上，通过调整屏幕和镜头之间的距离使图像聚焦。3D 打印支架上的线性杆和轴承允许平稳运动，而连接到有线遥控器的电机驱动丝杠进行实际调整。两半之间的缝隙用黑纸做的风箱盖住。

最初，投影图像的外围非常暗，因此[Matt]在屏幕的两侧添加了菲涅耳透镜。这在光线到达屏幕之前将来自附加镜头的光线拉直，并将其聚焦到相机的镜头中，几乎消除了渐晕。他在自己的 DIY 4K 投影仪项目中使用了非常相似的解决方案。

结果看起来几乎像做梦一样，背景和前景都很模糊，而主体却清晰地保持在焦点上。尽管[马特]正在拍摄钻机内部的投影图像，但这是不可能的。这有一个额外的好处，允许使用任何相机，包括智能手机。
