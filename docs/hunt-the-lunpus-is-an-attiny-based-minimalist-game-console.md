# Hunt The Lunpus 是一款基于 ATtiny 的极简游戏主机

> 原文：<https://hackaday.com/2022/07/22/hunt-the-lunpus-is-an-attiny-based-minimalist-game-console/>

在一个游戏控制台拥有越来越高的分辨率和越来越快的帧速率的世界里，看到有人反其道而行之令人耳目一新:[道格·麦金尼斯]的最新项目是[一个微型手持游戏控制台，可能拥有最低分辨率的图形](https://github.com/dmcinnes/lunpus)。硬件方面，它是一个小 PCB，包含一个 ATtiny84，两个七段 LED 显示屏，一个扬声器和几个按钮。正是这个软件赋予了这个项目魔力，所有这些都可以在 GitHub 上找到，还有原理图和 PCB 布局。

这款游戏名为 *Hunt the Lunpus* ，顾名思义，它的灵感来自于 20 世纪 70 年代的经典游戏 *Hunt the Wumpus* 。玩家通过一个相互连接的房间迷宫，试图避免粘液坑和掠夺蝙蝠，同时寻找 Lunpus，这是一个睡觉的怪物，会吃掉玩家，除非他们先用箭射死它。四个按钮提供方向控制，第五个作为“动作”按钮开始游戏并发射箭头。

虽然 *Wumpus* 最初是一款基于文本的冒险游戏，但 *Lunpus* 完全是图形化的:七段显示器显示洞穴的墙壁，并以不同的方式闪烁，以提醒玩家各种危险。[Doug]在下面嵌入的视频中解释了事件的发生过程；虽然一开始可能需要一点练习才能找到路，但我们已经可以想象自己带着装满箭的箭袋在洞穴中游荡，准备猎杀一些伦普斯。不管怎样，谁需要 4K 图形？

如果你喜欢极简主义的游戏机，有很多选择:LEDBOY 只需要几个 LED 就可以呈现太空入侵者，而 [TWANG 只需要一个 LED 条](https://hackaday.com/2018/01/24/diy-dungeon-crawler-game-plays-on-single-led-strip/)。你还可以[在这个 8×8 的 LED 矩阵](https://hackaday.com/2013/12/19/fubarino-contest-led-matrix-game-console/)上探索更多的迷宫，甚至[在分辨率稍高的](https://hackaday.com/2019/01/15/arduino-hunts-and-sees-the-wumpus/)上寻找迷宫。
