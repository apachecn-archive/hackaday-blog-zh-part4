# 创造你自己的极光

> 原文：<https://hackaday.com/2019/02/02/create-an-aurora-of-your-own/>

在我们的日常经历中，我们会遇到或利用许多我们可能没有意识到的科学原理，即使我们在描述它们时立即就能认出它们。其中一个令人好奇的是焦散线，它不仅指腐蚀性物质，还指当光穿过透明物体时可以观察到的光的行为。举个例子，拿着玻璃对着光源会产生这种效果，虽然这肯定很有趣，但也有使用激光操纵这些图案的[方法，这产生了类似极光的效果。](https://www.element14.com/community/community/project14/merryboxesleds/blog/2019/01/15/light-modulated-xmas)

这个项目的第一部分是寻找光源。led 被证明对于良好的分辨率来说太宽了，所以[神经调节器]将激光从一些 DVD 驱动器中取出作为点光源。从那里开始，他用来产生[腐蚀性图案](https://en.wikipedia.org/wiki/Caustic_(optics))的水面需要被搅动，因为这些图案在通过光滑表面时不会形成。为此，他使用了一个小型扬声器和驱动电路，可以精确控制水面上的波纹。

该项目的最后一部分是将激光器固定在从投影仪上取下的特殊透镜上，并将所有东西连接到激光器的驱动电路上。从那里，焦散图案可以被产生和控制，尽管[神经调制器]指出，这种设备在胶片上的效果与现实生活中人眼和大脑感知它们的方式截然不同。如果你对这种效果着迷，即使是通过相机的镜头，还有其他的[基于光的艺术装置](https://hackaday.com/2018/01/01/flowing-light-art-inspired-by-plankton/)也可能吸引你的眼球。
