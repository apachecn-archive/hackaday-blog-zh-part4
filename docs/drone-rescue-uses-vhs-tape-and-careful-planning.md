# 无人机救援使用 VHS 磁带和仔细的计划

> 原文：<https://hackaday.com/2022/12/24/drone-rescue-uses-vhs-tape-and-careful-planning/>

如果你经常在户外放飞你的无人机，你可能会担心你的骄傲和快乐会在某个时候卡在一棵大树上。但是，正如斯科特·威廉姆森发现的那样，在室内飞行也不能保证你安全。有一次，他的 65 毫米微型 Mobula 6HD 四轴飞行器卡在一个室内体育馆的屋顶横梁上，不得不开始一次大胆的营救。

第一项工作是侦察，[斯科特]派出另一架无人驾驶飞机调查情况。在那里，他试图用安装在一架更大的无人机前部的临时长矛将卡住的四轴飞行器捅出来。但这最终只是让大鸟也卡住了。它最终设法挣脱了，尽管它在下落时被斯科特接住，严重受损。正如告诉 Hackaday 的那样，[Scott]因此决定他需要建立一个家中情况的模型，以帮助他设计一种救援技术。

最后，[斯科特]选定了一个用回形针制成的抓钩。一架无人驾驶飞机在屋顶横梁上安装了一段很长的 VHS 胶带，然后他在地面上安装了抓钩。然后用 VHS 录像带把钩子绕到椽子上，套住无人机，把它带回地面。

这需要一些毅力，但[斯科特]最终从高高的监狱中救出了他的小无人机。然而，我们最喜欢这个故事的部分是[斯科特]像抢劫或[洞穴救援行动一样计划恢复。](https://hackaday.com/2018/07/17/lets-talk-about-elon-musks-submarine/)
