# 100 秒内看 Fortran

> 原文:[https://hack aday . com/2022/07/16/100 秒内看 fortran/](https://hackaday.com/2022/07/16/looking-at-fortran-in-100-seconds/)

通常，当我们谈论旧计算机时，我们想到的是 BASIC 解释程序。但是[Fireship]提醒我们，它最初是 Fortran，并承诺在 100 秒内给你[要点。我们认为你在这么短的时间内做不了什么，但是我们不得不承认他们做得很好。](https://www.youtube.com/watch?v=NMWzgy8FsKs)

当然，我们知道 Fortran 也无妨——看完这个视频后，你可能不会把它写进你的简历。另一方面，我们对他们挤进来的数量印象深刻。如果你从六七十年代就没接触过 Fortran，你应该知道它已经变了。指针、动态内存分配，甚至对象都是可能的。它仍然是一种非常强大的语言，非常擅长处理大量的数字。此外，你可以从几十年的 Fortran 开发中借鉴许多复杂的算法。

如果你决定试一试，当然有 [GNU Fortran](https://gcc.gnu.org/fortran/) 。老实说，尽管过去几年我们花了很多时间编写 Fortran，但我们不建议新的就业前景使用它。但是，如果你有一些时期的硬件，想要一个真实的体验，它可能只是一条路要走。或者，如果你想玩，只需[启动浏览器](https://www.onlinegdb.com/online_fortran_compiler)。

然而，并不是每个人都同意 Fortran 正在衰落。人们努力使它更符合时代潮流。你甚至可以用它来进行[网站开发](https://hackaday.com/2016/12/27/fortran-for-the-web/)。
