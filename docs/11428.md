# 可缝合的碳纳米管线可以纺出很多很棒的

> 原文:[https://hack aday . com/2021/09/28/sew-able-carbon-nanotube-thread-can-spin-a-lot-of-awesome/](https://hackaday.com/2021/09/28/sew-able-carbon-nanotube-thread-could-spin-a-lot-of-awesome/)

很多人就是不喜欢戴珠宝，甚至(或者尤其是)智能手表。然而，他们想和其他人一样接受生物反馈。好吧，我们这些不看手表的人有东西可以期待了，因为莱斯大学的一群研究生已经创造了由碳纳米管编织而成的极其坚固的导电线，这种线可以被缝进标准的运动服中，并被用作电极、天线或简单地用作弹道防护。

最初的碳纳米管只有 22 微米宽，太细了，不能用作线。取而代之的是，该团队将三束七英尺长的管子编织在一起，每束管子都使用了模型船建造者用来制作微小索具的那种机器。然后，他们将线之字形缝合成衬衫，这增加了缝线的灵活性。这种线可能像金属一样坚固和导电，但纤维柔软而有弹性，最重要的是，可以机洗。在它的强度和导电性之间，这种线可以有一长串的应用，从军事到民用。休息后看看视频中的介绍。

目前，衬衫必须非常贴身，但未来的衣服可以很容易地含有更高浓度的纳米线，以获得更好的信号。这是好事，因为我们仍然带着我们的 COVID 也就是我们从任何人一生中最长的行军以来增加的体重，而且从来不喜欢紧身衬衫。
