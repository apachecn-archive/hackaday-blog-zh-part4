# 耀眼的桌面十二面体

> 原文：<https://hackaday.com/2021/01/31/dazzling-desktop-dodecahedron/>

就像我们一样，[阿格巴伯]挖掘所有已经存在的无限多面体，但遗憾的是，几乎所有的多面体都太大了，不能用作桌面上分散注意力的话题。这可能是因为把它们做大要容易得多，但这并没有阻止[阿格巴伯]的尝试，取得了惊人的成功，[为任何想接受挑战、打造自己的炫目桌面玩具的人铺平了道路。](https://www.instructables.com/Desktop-Sized-Infinity-Dodecahedron/)

[![](img/66616cfd2eaa2446d7487444019e245f.png)](https://hackaday.com/wp-content/uploads/2021/01/dodeca-jig.png) 我们都知道，所有那些小小的发光二极管必须以某种方式连接在一起。在更大的版本中，电线会工作得很好，但在垒球大小的时候，它们会变成一个乏味而复杂的噩梦。那么[阿格巴伯]做了什么呢？没错，他们设计了两种不同类型的定制角 PCB。将 led 和面板固定在一起的 3D 打印支架也不是轻而易举的事情——[AGBarber]建议如果你有树脂打印机，就使用树脂打印机，尽管这不是严格要求的。

关于这个项目的一切都是开源的，包括一个额外的可打印夹具，用于以正确的角度将支架粘合在一起。所有的步骤都有很好的记录，从应用镜面薄膜到控制灯光的 Wemos D1 迷你程序[AGBarber]也编入了大量的动画，你可以在休息后观看。

想要建造一个小的无限的东西，不是很难吗？打开一个冰镇的杯垫，看看这些很酷的杯垫。
