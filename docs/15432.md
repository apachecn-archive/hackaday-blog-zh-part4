# 如果你的无人机飞了，就吃了它！

> 原文:[https://hackaday.com/2022/11/22/if-your-drone-flies-eat-it/](https://hackaday.com/2022/11/22/if-your-drone-flies-eat-it/)

多年来，我们在 Hackaday 展示了无数无人机项目，包括固定翼、旋翼、多旋翼等。我们认为其中可能有一种我们从未见过的类型，但这将会改变，因为这是我们第一次给你带来可食用的无人机。

你可能会问，为什么你可能需要一架可食用的无人机？这并不是为了在关闭机场后隐藏证据——相反，这是瑞士联邦理工学院的一个研究项目，旨在为被困的登山者提供有效的食物。圣伯纳德狗失业了，现在是用现代方式！

玩笑归玩笑，这显然是一个实验性的飞行器，一个固定翼单翼机，它的翅膀是由年糕和明胶制成的。一个被困的登山者当然可以大嚼这些空气油，但我们猜测一个真正的设备需要一些更有营养的东西，同时保留轻型细胞结构。

这可能是我们的第一架可食用无人机，但它不是我们带给你的第一件[可食用技术](https://hackaday.com/2022/04/19/edible-electronics-let-us-hear-the-lamentations-of-the-chocolate-bunnies/)。
