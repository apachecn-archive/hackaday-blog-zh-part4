# 光束蜻蜓引起了一个抖动

> 原文：<https://hackaday.com/2019/10/31/beam-dragonfly-causes-a-flap/>

正常人会把坏掉的东西扔掉。但不是像我们这样的人。或者，很明显，[纳米机器人]。一只便宜的机器人蜻蜓死了，他把它拆成机器人零件。但他保留了变速箱，希望能造出一架新的蜻蜓，并且用一些黄铜棒，他[做到了](https://www.instructables.com/id/Flapping-Dragonfly-BEAM-Robot-From-a-Broke-RC-Toy/)。

蜻蜓的电路使用太阳能电池板和几个闪烁的发光二极管供电。这是一个光束机器人，所以看不到微控制器。你可以看到蜻蜓如何移动的简短视频。

真的，虽然，这是使用焊接黄铜棒的机翼制造的整洁的部分。头部有闪烁的发光二极管作为眼睛。秘方当然是变速箱。如果你要复制这个项目，你可能必须找出自己的变速箱修改，除非你只是碰巧得到完全相同的玩具。

除了 LED 眼睛，蜻蜓的尾巴上还有一个电容器组。这不美观。机翼依靠太阳能引擎电路运行。发光二极管和电容器工作来积累和释放太阳能，即使没有足够的光线直接为马达提供动力，蜻蜓也可以扇动翅膀。

事实证明，蜻蜓只需要多一点点的果汁，所以[nanobotgeek]隐藏了一个额外的电容器来增加存储体的大小。

我们想知道[凯瑞]能否让他的蜻蜓徽章飘动起来？我们知道这是作弊，但我们更希望光束机器人有[一些智能。](https://hackaday.com/2013/10/13/turbot-is-a-beampicaxe-hybrid/)
