# 你的控制台，你的弹夹，你选择？任天堂面临挑战

> 原文:[https://hackaday . com/2022/07/09/your-console-your-cartridge-you-choose-Nintendo-faces-a-challenge/](https://hackaday.com/2022/07/09/your-console-your-cartridge-you-choose-nintendo-faces-a-challenge/)

如果你读了我们的文章，你会注意到我们通常会以与我们谈论的主题相关的图片为特色。如果它们来自其他来源，并且没有 CC 许可或类似许可，那么它们是其他人的财产，但我们是在被称为*合理使用*的原则下使用它们的。在合理使用的情况下，版权材料的摘录可以用于新闻报道等目的，例如，如果我们要写漫威超级英雄，我们可以使用美国队长的版权图片。一些内容所有者仍然试图阻止这一点，这是其中之一，[Linus Tech Tips]在他们的视线中作为[他们出版了一份指南，在 Steam Deck](https://www.youtube.com/watch?v=oIYvPNtWZ34) 上运行任天堂 Switch 游戏，而他们认为没有给这个臭名昭著的诉讼游戏巨头任何合理的理由采取行动。里面全是精心模糊的任天堂 IP，没有盗版软件下载的报道。

一方面，这是关于一个强硬的游戏机开发商搞垮小型在线内容生产商，但另一个角度与阅读 Hackaday 的硬件社区更相关。它在软件仿真领域也有应用，因为虽然控制台制造商更愿意停止除他们自己的未经修改的硬件之外的所有硬件运行游戏，但是不应该阻止合法拥有的软件或硬件以其所有者选择的任何方式运行。这是视频最后探讨的中心论题，试图让任天堂在这个问题上公开的噱头是他们宣传它的方式。

尽管[Linus Tech Tips]可能是 YouTube 上观看次数最多的技术频道之一，但很明显，如果任天堂选择上钩，它将拥有更雄厚的资金。所以我们确信在我们写作时，他们的律师都在关注这件事，但我们很有兴趣看看视频中的说法是否足以维持下去。想到这可能会导致任天堂重新考虑他们的一些政策，这是件好事，但我们不会屏住呼吸。
