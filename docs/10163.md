# 霓虹灯的历史

> 原文:[https://hackaday.com/2021/05/17/the-history-of-neon-lights/](https://hackaday.com/2021/05/17/the-history-of-neon-lights/)

我们总是喜欢[历史学家]的历史视频，但它们并不总是涵盖技术历史。然而，当它们出现时，我们对它们的喜爱是他最近在霓虹灯历史上发布的视频的两倍。当然，正如他指出的那样，许多霓虹灯中并没有真正的氖——它们根据你想要的颜色使用各种惰性气体。当然，有些有霓虹灯，但是这个名字已经被记住了。

视频的后半部分更多的是关于标志本身，但前半部分谈到了[威廉·拉姆赛]，一位苏格兰化学家，他开始从大气中提取气体成分。首先发现的是氩，然后是氦。氪和氖紧随其后。另一种惰性气体氙也落入了他的实验中。他和另一位科学家因此获得了诺贝尔奖。

[历史学家]还提到了盖斯勒电子管，这个名字我们以前只在旧的广播书籍中读到过。有趣的是，几乎所有的真空管和充气管都是从这项技术发展而来的。毕竟，霓虹灯实际上是一种充气管，普通的霓虹灯也是如此。

霓虹灯的商业化不得不等待乔治·克劳德学会如何有效地液化空气。他成立了一家公司，你可能听说过:液化空气公司，它与卡尔·冯·林德早期生产大气气体的系统竞争，如果你购买任何一种工业气体，你可能也听说过林德公司。当然，接触高电压也是一项要求。

奇怪的是，[克劳德]没有想到用它们来做醒目的标志。他想用它们作为人工照明的一种形式。是一位合伙人[Jaques Fonseque]发现了利用电灯制作广告标志的可能性。1913 年，Cinzano 标志出现在巴黎，到 1919 年，巴黎歌剧院的入口处有了霓虹灯管照明。

霓虹灯的商业用途远不如过去普遍。然而，霓虹灯作为艺术正在卷土重来，这是周期性发生的事情。如果你想在不吹制玻璃的情况下制作自己的复制品，试试 [EL wire](https://hackaday.com/2016/09/18/forge-your-own-neon-signs-with-el-wire/) 。我们已经在[看过不少人造霓虹灯项目](https://hackaday.com/2019/08/12/el-wire-makes-for-a-great-faux-neon-sign/)，它们看起来棒极了。
