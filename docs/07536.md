# 树莓派是一个实用的三重录音机

> 原文:[https://hack aday . com/2020/08/24/raspberry-pi-makes-a-practical-tricorder/](https://hackaday.com/2020/08/24/raspberry-pi-makes-a-practical-tricorder/)

当您在树莓 Pi 中添加热感摄像头、软件定义的无线电加密狗和电池时，您会得到什么？如果你是[saveitforparts],你可以制作一个三重记录器，用于嗅探无线电信号和观察热信号。他承认，这些视频(见下文)并不完全是“如何做”,但它仍然会给你一些关于下一个版本的想法。

你可以感觉到对一些 Linux 配置问题的失望，但是[saveitforparts]承认他不是 Linux 或 Raspberry Pi 的专家。版本 1 看起来有点像原型，但是版本 2 更加完善。我们仍然不确定是否会看到斯波克带着这样的箱子，但一些 3D 打印技术可以马上把它分离出来。

当然，一台真正的三录机是一台麦戈芬，它能做任何剧情需要的事情。这个更加实用，但是它可以监控热能和射频能量，并且可以容纳更多的传感器。这是一个很好的例子，这个项目在过去是很难做的，但是现在变得容易多了。廉价计算机和现成模块以及相关软件的出现开辟了许多可能性。

如果你想做你自己的三录黑客，你可以接手一个商业模式。然后，有一个[官方复制品](https://hackaday.com/2020/07/22/cbs-announces-functional-tricorder-replica-for-2021/)正在路上，看起来它可能有一些类似的功能。
