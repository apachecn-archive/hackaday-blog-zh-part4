# 终于到了圣诞装饰的时候了吗？

> 原文：<https://hackaday.com/2021/12/03/is-it-finally-time-for-christmas-decorations/>

[Arnov]正试图融入节日气氛，并以他知道的方式这样做。他想为他的圣诞树做一些很酷的装饰品，并决定最好的装饰品是你自己做的，所以他自己做了一个闪闪发光的圣诞树装饰品。

著名的“ [blinky circuit](https://hackaday.com/2018/09/25/minimal-blinky-project-makes-the-chip-the-circuit-board/) ”当然是我们在 Hackaday 并不陌生的一条赛道。我们的一些读者会非常高兴地看到，他确实使用了 555 定时器，而不是 Arduino。555 定时器连接到 CD4017 十进制计数器的时钟引脚，十进制计数器的输出连接到 led。在时钟脉冲的每一次低至高转换时，LED 会依次亮起，不过您可以尝试创造性地使用 LED 布线方案来实现不同的闪烁效果。

读者可能真正从这个版本中学到的是[Arnov]详细介绍了如何将图像导入他选择的 CAD 工具，或者在他的情况下导入 CAD。我们知道这有时会有点棘手。最后，我们喜欢这个项目兼做 [PCB 艺术](https://hackaday.com/2020/01/13/the-way-of-the-pcb-artist-how-to-make-truly-beautiful-circuit-boards/)和[焊接挑战](https://hackaday.com/2016/06/19/soldering-challenge-to-challenge-you/)。在你的下一个初学焊接研讨会上，它肯定会成为一个很好的演示项目。
