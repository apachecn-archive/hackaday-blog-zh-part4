# SmallKat:一个可爱的动态机器人

> 原文:[https://hack aday . com/2019/11/03/small kat-an-obervable-and-dynamic-robot/](https://hackaday.com/2019/11/03/smallkat-an-adorable-and-dynamic-robot/)

[SmallKat 是一个可爱的小机器人，拥有许多功能](https://hackaday.io/project/164727-smallkat-an-adorable-dynamics-oriented-robot-cat)围绕教学和实验动态机器人控制而设计。很遗憾我们还没有报道 SmallKat，因为它不仅入围了 [2019 年黑客日奖](https://hackaday.com/2019/09/10/meet-the-20-finalists-in-the-2019-hackaday-prize/)，也是今年 [Bootstrap 获奖者之一](https://hackaday.com/2019/06/14/meet-the-bootstrap-winners-of-the-2019-hackaday-prize/)。

许多业余爱好机器人通过重复预编程的动作序列来移动。例如，大多数六足动物都遵循这种思路。然而，像 Spot 和 MiniCheetah 这样的机器人展示了一个不同的世界，在这个世界中，机器人通过它们的当前状态、它们环境的测量状态和一些想象的未来来确定它们肢体的位置。这些机器人比它们的前辈能干得多。

然而，即使是这些机器人的高性价比版本(T1 ),价格也会大幅攀升至数万美元。SmallKat 将在这方面有所帮助:基于业余爱好伺服系统和 ESP32 的硬件保持在可承受的范围内。数据可以传输到更大的计算机上进行实验，这样可以节省一些支持更大设备(如 Pi)所增加的重量。

这种设备将让学生在不倾家荡产的情况下实验各种动态模型，甚至基于机器学习的运动。甚至有一个很好的软件工作室来帮助学习过程。休息后它来回走动的视频。
