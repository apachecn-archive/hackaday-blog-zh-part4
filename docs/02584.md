# Mods 让 WaveBird 在 Switch 时代踢个不停

> 原文：<https://hackaday.com/2019/04/13/mods-keep-the-wavebird-kicking-in-the-switch-era/>

任天堂在 21 世纪初做出了一些有问题的决定，但开发 WaveBird 肯定不是其中之一。在无线控制器成为家用游戏机标准的几年前，WaveBird 让 GameCube 的所有者看到了未来。它成功地提供了无延迟的游戏，而没有求助于容易被阻挡的红外，并且电池寿命和范围足够长，除了缺乏隆隆声支持之外，切掉电线真的没有任何负面影响。

[![](img/1818cb98467e6b7a1e3ada2feabc5e1e.png)](https://hackaday.com/wp-content/uploads/2019/03/switchbird_detail.jpg) 事实上，WaveBird 是一个非常好的控制器，即使在 2019 年，一些粉丝也无法放下它。[比尔·帕克斯顿]非常喜欢他的，所以他[决定修改它，这样他就可以在任天堂最新的印钞机上使用它，Switch](https://bitbuilt.net/forums/index.php?threads/late-night-modding.1686/page-3)，而不必摆弄任何适配器。当他在做的时候，他决定修复控制器唯一的严重缺陷，并黑进一些隆隆的马达；可以说，这使得他的重新设计的 WaveBird 几乎在所有方面都优于原版。

这可能是反直觉的，但这里的技巧是[Bill]实际上从标准有线 GameCube 控制器中取出了内部部件，并将其全部安装到 WaveBird 的外壳中。这就是他如何重新获得隆隆声支持的，但标志性的无线功能来自哪里？

为此，他拆开了一个“GBros”。无线适配器”。这个小工具旨在让你在交换机上无线使用现有的 GameCube 控制器，所以他所要做的就是将它的 PCB 推到控制器内部，并直接将其连接到控制器板上的衬垫上。值得庆幸的是，WaveBird 一开始就相当笨重，所以内部有足够的空间来添加所有额外的硬件，没有太多的麻烦。

在这样的修改和对控制器的无线协议进行逆向工程的努力之间，黑客们不会让这个革命性的配件轻轻地进入那个美好的夜晚。[你可能会看到 GameCube 因为一个迷因而被屠杀](https://hackaday.com/2018/03/16/gamecube-dock-for-switch-mods-nintendo-with-more-nintendo/)，但是 WaveBirds 永远不会死。
