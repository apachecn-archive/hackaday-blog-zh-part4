# 线性时钟加快了动作

> 原文:[https://hack aday . com/2020/09/26/linear-clock-ratchets-up-the-action/](https://hackaday.com/2020/09/26/linear-clock-ratchets-up-the-action/)

从表面上看，制作一个通过沿线性刻度移动指针来显示时间的时钟应该不会太难。毕竟，步进器和线性驱动器应该可以在瞬间完成这项工作。扔个 Arduino 进去，Bob 就是你叔叔了，对吧？

不对。至少这不是[Leo Fernekes]决定建造这个独特的棘轮线性时钟的方式，这是一个明智的决定，使这个项目绝非一般。这个想法已经在[Leo]的脑海中萦绕多年，直到灵感以[这个老 Tony]的不太可能的形式出现，他是我们最喜欢的 YouTube 机械师之一。【老托尼】做了[一个视频](https://www.youtube.com/watch?v=3_wPH904a_8)关于简单天才的闩锁机制，就像伸缩笔里的那种，那是一个“啊哈！”[狮子座]的时刻。对于棘轮，他使用一条带锯条，使齿朝上。一个复杂的弹簧钢片，弯曲与刀片的齿啮合，形成一个棘爪，使指针向上移动，直到它到达顶部。

[Leo]很早就决定这将是一个脉冲时钟，就像学校和工厂使用的那种。他用伺服系统每分钟将一条胶带向上推一次；带子与爪啮合，爪拖动指针，将棘爪沿棘轮向上移动一个齿，并将指针提升一分钟，接近顶部。指针在顶部释放，然后回落，重新开始循环；为了阻止它的自由落体，[Leo]想出了一个天才的主意，在铝框架上安装磁铁并利用感应的涡流来完成这项工作。这款时钟采用 3D 打印的装饰艺术刻度，是一款独一无二的时计，绝不乏味。

我们非常欣赏[Leo]在项目上的独特和创造性，以及他的作品范围。看看他的[永恒连续性测试器](https://hackaday.com/2020/07/13/build-an-everlasting-continuity-tester/)和他的[噬菌体般的哨兵枪](https://hackaday.com/2020/06/10/autonomous-sentry-gun-packs-a-punch-and-a-ton-of-build-tips/)的一些整洁的建造细节。
