# 克隆控制台婴儿床本赫克的经典 SNES 卡奇

> 原文：<https://hackaday.com/2020/12/02/clone-console-cribs-ben-hecks-classic-snes-cache/>

“当人们敲我竹杠时，我不会生气。实际上，我认为这很酷，因为模仿是最真诚的奉承形式。”—本·黑肯登

对一些人来说，“把东西拼凑在一起”可能意味着大量借用他人的作品来创作新的衍生作品。尽管资深硬件黑客本·黑肯登并没有想到他早期的 SNES 掌上电脑项目会成为 Famicom 式克隆游戏机的灵感来源。多年来，网上已经出现了许多克隆人控制台，而且都有不同程度的构建质量。问题中的主体克隆控制台被称为 Easegmer 12 位复古控制台，因此[Ben]决定[记录他借用自己的原始设计拆卸手持设备的过程。](https://www.youtube.com/watch?v=I6Oz2fqutw8&ab_channel=BenHeckHacks)(视频，下面嵌入。)

Easegmer 手持设备根据其包装列出了一系列“令人惊讶”的功能，包括:体育游戏、逻辑游戏、memoyr 游戏、USB 充电器管理、双电源选项和 dirunal 双背光选项。所有大的(和稍微拼错的)承诺，尽管最惊人的声明是，“没有暴力游戏，你孩子的身体和大脑得到充分锻炼。”。这种说法可能有一点真实性，除了 220 场比赛中的第 84 场比赛字面上被命名为“暴力”。除了扣篮之外，这款手持设备还配备了标准尺寸的可充电电池，以及用三节 AAA 电池为设备供电的选项。甚至还有一个“有趣的尺寸”螺丝刀和几个替换螺丝，这比大多数现代电子产品都要多。

自从[Ben]最初在他的网站上发布他的 SNES 便携项目以来，已经过去了将近 20 年。因此，作为期待已久的后续措施，[本]计划制造一种“超便携”。这个元便携版将从他 2001 年在 SNES 便携版中保存的 Adobe Illustrator 文件开始，并整合 Easegmer 克隆控制台的各个部分。从而产生了他的克隆体的克隆体的新克隆体……或者那个项目最终肯定是值得重复的。
