# 降低八木元件隔离的吊杆

> 原文：<https://hackaday.com/2020/05/10/lowering-the-boom-on-yagi-element-isolation/>

至少可以说，天线设计可能会令人困惑。天线中有如此多的黑魔法，以至于新手们经常看着设计，却不知道这东西到底是如何工作的。两个元件之间的长度或角度的微小变化会导致谐振频率的巨大差异或天线阻抗的显著变化。它能让人分心。

特别令人担忧的是馈线的两个导体之间似乎经常出现完全短路，这是[【安德鲁·麦克尼尔】用一对 WiFi 八木天线](https://www.youtube.com/watch?v=fao1KKwmps0)探索的。这些高方向性天线具有一个驱动元件和多个寄生元件，特别是驱动元件后面的反射器和前面的一个或多个导向器。基于元件间距的相长和相消干涉以及基于元件长度的电容或电感耦合决定了天线的特性。[Andrew]的测试天线的 12 个导向器要么与吊杆隔离，要么与馈线的屏蔽短接在一起。在一个已知信号源的并行测试中，两个天线的表现完全相同，这意味着如果你选择建造一个八木，你可以在选择什么材料以及如何将元素连接到吊杆上有很大的灵活性。

如果你想更深入地了解八木天线的工作原理，并了解为什么它被更恰当地称为八木宇田天线，请查看我们关于其历史和工作原理的故事。向[andrew]致敬，他提醒我们，天线设计通常是一项实用性的练习；毕竟，在适当的情况下，[一把伞和一些铁罐](https://hackaday.com/2018/04/28/umbrella-and-tin-cans-turned-into-wifi-dish-antenna/)甚至[一颗生锈的钉子](https://hackaday.com/2020/04/23/the-rusty-nail-award-for-worst-wifi-antenna/)都可以。
