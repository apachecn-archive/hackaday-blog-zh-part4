# 压力下的优雅:谢莉·格林庆祝卷曲的连接

我们认为，可以很有把握地假设，我们的读者在那里制作的大多数电气连接都涉及到焊料或焊膏。但是在我们的一生中，我们都有过一两次曲折的联系。也许你在这里或那里挤了一个对接连接器，或者做了一根以太网电缆。除了在以太网电缆中获得正确的布线顺序，您对连接器内部发生的事情有多好奇？

看起来焊料似乎是制作低电阻电气连接的最佳选择。毕竟，你是在把金属和另一种金属焊接在一起。对于久坐室内的电路板来说，这通常都很好。但是，如果一个关节需要机械稳定，并能在潜在的恶劣环境中生存，你就不需要一种合金来把东西固定在一起。你需要金属与金属的接触，而压接就是这样。

一个制作精良的卷曲应该可以持续几十年，但是[正如 Shelley Green 在 2019 年 Hackaday Superconference](https://www.youtube.com/watch?v=PoaArqWcp_w) 的演讲中解释的那样，优质的卷曲不是偶然发生的。好的卷曲是精心设计的，并且从头到尾都是精心制作的。

## 卷曲动力学

雪莱将失败的卷边比作困扰挑战者号航天飞机的 O 型圈灾难。像 O 型圈和压接接头这样的东西可能看起来很小，无关紧要，但如果不注意它们的物理特性，就可能意味着灾难。

那么，怎样才能做好卷曲呢？这完全是物理问题。压接的要点是以尽可能低的电阻进行连接。永久性机械压接包括将导体压在一起，以便电流可以从 A 流向 b。宏观结构上，当连接器变形超过金属的屈服点时，会出现良好的压接，并且导线变形均匀。

[![](../Images/a6473fc151ba2af2ec88150e4c56f0c0.png)](https://hackaday.com/wp-content/uploads/2020/04/crimp-profiles.png)

A handful of crimp profiles. Image via [Pico Tools](http://www.picotools.com)

正如 Shelley 解释的那样，这种体积变形不是低电阻的原因，而是导体和连接器之间的残余应力将系统保持在一起。

如果你在扫描电子显微镜下观察压接连接的微观结构，表面相当粗糙，只有几个点实际上相互接触。实际发生接触的高调位置——在材料科学中称为粗糙点或 a 点——是电阻等级的真正指标。

良好压接的设计有许多方面。首先，你必须考虑应用、压接所处的环境以及压接的预期结果。接下来是实施细节——压接方式、电线、端子以及每个人都喜欢的工具。电缆越粗，所有这些东西就变得越重要。

Shelley 还讨论了不同应用和环境下各种压接方式的优点，并指出压头的形状可能意味着耐用性的巨大差异。

## 真相在测试中

最后，Shelley 谈到了通常用于压接连接的测试方案。正如您所料，该测试旨在回答压接绘图板上提出的问题。压接电阻低吗？这可以在连接处于负载下时通过毫伏跌落测试来验证。这种连接会被证明是机械坚固的吗？是时候使用电力拉力机测试导线断裂和拔出情况了。

如果你真的想判断一个卷曲，采取一个横截面，并在扫描电子显微镜下检查。理想情况下，导体形成了楔形六边形的蜂窝。任何可见的空隙都表明存在气穴，这些气穴会增加电线中已经存在的不可避免的削弱性的微膜。

卷曲的连接比你想象的要多得多，谢莉的演讲肯定会让你的神经元围绕这些基础发生大规模变形。
