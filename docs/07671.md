# 无雾口罩黑客解决口罩与眼镜的难题与高超的密封

> 原文：<https://hackaday.com/2020/09/02/fog-free-mask-hack-solves-mask-versus-glasses-conundrum-with-superb-seal/>

如果你戴着口罩和眼镜的时间超过了四分之一秒，你可能会为我们没有一个神奇的解决镜片模糊的方法而感到恼火。潮湿的空气也是未过滤空气逸出的一个很好的指示器。大多数面具的鼻梁上都有一些柔性金属，用来密封顶部，但这远远不够。威斯康星大学麦迪逊工程学院的[David Rothamer]和[Scott Sanders]创作的 [*獾印*](https://making.engr.wisc.edu/mask-fitter/) 在新冠肺炎疫情期间可以免费复制，甚至用于商业目的。它的工作原理是在下巴下面放一根弹性线，在鼻子上放一根可变形的线，以促进嘴和鼻子的接触。 

你可以通过三种方式建立自己的网站。每种配置都适合不同的情况。第一种设计是最容易制作的，应该对大多数人都适用。第二种最适合那些需要在下半部脸上有更好的印记的人，比如留着胡子的人。它还可以有耳挂，这意味着你的 3D 打印护耳器有了另一种用途。威斯康星大学麦迪逊校区也有好玩的[开锁](https://hackaday.com/2013/05/14/master-lock-auto-cracker-built-as-coursework-at-university/)和[石墨烯实验](https://hackaday.com/2015/10/14/graphene-grown-on-semiconductors-big-step-toward-manufacturability/)。
