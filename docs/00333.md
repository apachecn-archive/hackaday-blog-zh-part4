# 半自动绕线机旋转电机转子

> 原文：<https://hackaday.com/2018/08/17/semi-automated-winder-spins-rotors-for-motors/>

你的秘密邪恶计划是什么？你想通过制造一台可以自我复制的机器来统治世界吗？或者你只是厌倦了手工缠绕电机转子和其他线圈？不管怎样，[这个自动绕线机](https://youtu.be/05jpsSEF_Qs)可能是你需要的。

我们在一定程度上是在开玩笑，但这是真的，自我复制机器的闭环意味着能够制造像马达这样的东西。无论是有刷电机还是无刷电机，这都意味着将线轴变成某种线圈。[Mr Innovative]的绕线机使用 3D 打印的管子将电磁线绕在转子铁芯上。步进电机将旋转臂转动指定的次数，并在终点暂停，以便操作员可以移动焊线，为下一个弧线腾出空间。转子然后旋转到它自己的步进电机上的下一个位置，缠绕继续。这一手动步骤需要注意，以使这是一个全自动系统，我们认为需要解决电线的张力，以便绕组更紧一点。但这仍然是一个很好的开始，它给了我们一些相关线圈缠绕项目的思路。

当然，并不是每个电机都需要缠绕线圈。毕竟[蚀刻线圈的无刷 PCB 电机](https://hackaday.com/2018/08/09/friday-hack-chat-motors-made-out-of-pcbs/)是一个东西。
