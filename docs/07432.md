# 会说话的头教拉普拉斯变换

> 原文:[https://hack aday . com/2020/08/09/talking-head-teaches-la place-transform/](https://hackaday.com/2020/08/09/talking-head-teaches-laplace-transform/)

大多数和电子学打交道的人都听说过傅立叶变换。这一数学过程使计算机分析声音和视频成为可能，它还为从模式匹配到频率合成的任务提供了重要的数学见解。拉普拉斯变换不太为人所熟悉，尽管它是傅立叶变换的推广。史蒂夫·布伦顿在最近的一个视频中对拉普拉斯变换背后的数学原理做了很好的解释，你可以在下面看到。

拉普拉斯变换有许多应用，包括变换各种类型的微分方程。这种情况经常出现在电子器件中，在这些器件中有电感和电容等时变元件。您可以执行拉普拉斯运算，使用普通代数求解，然后进行逆变换以获得正确的答案，而不必求解微分方程。这类似于对数如何将一个更难的问题——乘法——转化为一个更简单的加法问题，但规模要大得多。

我们认为这是一种选择，但[史蒂夫]穿着黑色背景的全黑衣服，所以我们发现自己把他想象成一个数学知识的浮动头。玩笑归玩笑，这个话题的解释真是太棒了！你可能需要一些微积分来从视频中获得最大的收益，但是看着他穿过方程式是很有趣的。

一旦你解决了这个变换，你会发现了解 [S 平面](https://hackaday.com/2015/12/24/beyond-control-maths-of-a-control-system/)是拉普拉斯变换的图形表示是很有趣的。如果你需要温习一下我们之前提到的微积分，那么[微积分其实并不难](https://hackaday.com/2015/12/22/calculus-is-not-hard-the-derivative/)。我们也喜欢关于这个主题的一本[非常古老的书](http://calculusmadeeasy.org)。
