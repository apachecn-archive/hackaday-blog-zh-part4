# Ken Shirriff 谈论了整个芯片去盖的世界

> 原文：<https://hackaday.com/2018/10/11/ken-shirriff-chats-about-a-whole-world-of-chip-decapping/>

逆向工程硅是一个黑暗的艺术，当你刚刚开始，最好坚持较小的咒语，诅咒和巫术。Hackaday 在去年的 Supercon 上采访了 Ken Shirriff，讨论了芯片解封装和逆向工程的情况。他的建议是从一个老朋友开始:555 定时器。

Ken 因其拍摄集成电路(IC)核心的硅芯片并绘制结构以创建电路原理图的工作而闻名。我们期待着 Ken 在几周后的超级黑客大会上的演讲。在下面的采访视频中感受一下。

你会惊讶地发现，养成这个爱好是多么容易。肯关于使用 555 定时器的建议是明智的，因为你可以用钢锯而不是像发烟硝酸这样的化学物质来做。显然，如果你去易贝，你仍然可以将这些芯片封装在金属罐中，而不是通常想到的黑色环氧树脂。拿一把钢锯打开罐子，给露出来的模具拍一张很棒的照片，然后你就可以开始研究这个结构了。这些芯片不是用非常小的数字芯片制造的，电路本身也不复杂。Ken 甚至有一本带你完成这个过程的入门书，包括模具照片，如果你想跳过钢锯这一步的话。

你可能不会以设计芯片为生，但可以把这当成电子工程师的拼图游戏。你有原理图，你只需要找到所有部件的位置。

有许多知名人士与肯在同一个领域工作，他提到了其中的几个人:Siliconpr0n 的约翰·麦克马斯特、T2 Project 5474 的罗布·巴鲁克和 T4 Zeptobars 的优秀员工。所有这些都是很好的学习资源，并且可以让你接触到这些照片进行进一步的研究。

肯使用 Hugin 软件包将他显微镜中的多张照片拼接在一起。这是拍出每个部分都非常清晰的照片的秘密，我们认为这和逆向工程一样了不起。要深入了解，请查看他在[2016 年黑客日超级大会](https://www.youtube.com/watch?v=aHx-XUA6f9g)上的演讲。我们迫不及待地想看看他今年会给[带来什么](https://www.eventbrite.com/e/hackaday-superconference-2018-tickets-47386813234?aff=1011com)！
