# 为 122 GHz 操作加工波导是精细的工作

> 原文:[https://hack aday . com/2022/02/12/machining-waveguides-for-122-GHz-operation-is-precious-work/](https://hackaday.com/2022/02/12/machining-waveguides-for-122-ghz-operation-is-delicate-work/)

现代汽车中用于巡航控制和防撞的毫米波雷达通常被设计为工作在 100 米左右的范围内。然而，凭借一些工程常识，实验人员在一些测试中已经让这些设备在高达 60 公里的范围内发送信号。[加工和微波]决定看看他是否能把船推得更远，并开始加工一些波导组合器腔，这样他就可以把雷达芯片与一些非常高性能的天线一起使用。

![](../Images/a31f16613289653b02a6408cb003859b.png)

Precision-machined components are required to successfully use these 122 GHz components for long-range transmissions.

该项目的最终目标是为汽车应用中常见的毫米波雷达芯片输出的 122GHz 信号生产 53 dBi 天线。在该频率下工作需要获得公差*以便制造性能良好的天线。*

大量的精细车床工作和愉快的加工戏谑之后，精密波导完成了。对于未经训练的人来说，它可能看起来不太像，但经过精心设计和加工，它不仅易于连接到雷达和抛物面天线系统，而且性能足够高，有望打破其他进取的无线电实验者创下的记录。如果这还不够困难的话，最后的工作包括制作 24 个这样的东西！
