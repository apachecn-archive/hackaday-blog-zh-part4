# 用成吨的锡焊把一辆毁坏的游戏机车复活

> 原文:[https://hacka day . com/2021/09/15/用数吨焊锡让一个被毁掉的游戏机起死回生/](https://hackaday.com/2021/09/15/bringing-a-ruined-game-boy-cart-back-to-life-with-tons-of-soldering/)

对于许多人来说，复古游戏机和手持设备充满了怀旧和快乐的回忆。然而，保持这些机器和它们的介质运行有时是一项困难的工作。[泰勒]接受挑战，为原版游戏《男孩》[抢救出一份科比的梦想之地，并开始着手这项任务](https://hackaday.io/project/181682-soldering-kirbys-dreamland-back-to-life)。

[![](../Images/15f81e2a46dbaf6f0e8c64cfe695d27c.png)](https://hackaday.com/wp-content/uploads/2021/09/repairing-game-boy-cartridge.jpg) 墨盒被严重腐蚀，许多痕迹都被腐蚀掉了，致使游戏无法运行。首先，移除所有元件，并清洁电路板。这样就可以方便地访问整个电路板上的走线。然后，工作是小心翼翼地从仍然残留的部分走线移除一些阻焊膜，并用细铜线桥接间隙。更糟糕的是，几个过孔被损坏，[Taylor]通过将跳线穿过电路板并在每一侧进行修复来解决这个问题。

对于有经验的人来说，这是一个足够简单的修复，但对于一个不知道如何焊接的复古游戏迷来说，这几乎是一个魔法。[Taylor]给了我们一个很好的例子，告诉我们如何正确处理腐蚀的手推车，足够的细节对初学者来说很有教育意义。

我们也看到了这方面的其他伟大作品，比如从可怕的绿色死亡中复苏的 Amiga 2000。如果你自己做过复古拯救，[一定要给我们写信！](http://hackaday.com/submit-a-tip)
