# 晶体管基础动画

> 原文：<https://hackaday.com/2018/09/02/transistor-fundamentals-animated/>

当我们在学校的时候，每一个关于晶体管如何工作的描述都是非常枯燥的，并且包含了大量的数学知识。我们假设你可能有一个伟大的导师，他能够更直观地解释事情，但那是运气，统计上不太可能。如今，互联网上有很多很棒的视频来解释事情，即使你知道主题，看一些很棒的动画也很有趣。例如[Sabin]有这个关于 MOSFETs 如何工作的漂亮的动画解释，你可以在下面看到。

它使用了与他早期关于双极晶体管的视频(下面是第二个视频)相同的基本图形和风格，这也是一个值得观看的视频。平心而论，对你的电子老师来说，这些视频中的图形在 20 世纪会花费很多钱——只要看看我们在[一些历史帖子](https://hackaday.com/2017/05/29/retrotechtacular-the-transistor-1953-film/)中谈论的一些视频就知道了。

即使你精通器件物理，如果你想得到一些帮助来解释电子和空穴在半导体中的运动，这些视频也很棒。我们很高兴看到的一件事是，当像双极晶体管一样掺杂时，笑脸电子不会皱眉。我们不愿想到我们友好的电子不快乐。

在当今世界，你可以不知道设备是如何工作的。毕竟，树莓派在不知道这个级别发生了什么的情况下工作得很好。当然，你可以在不了解发动机的情况下驾驶汽车，但你可以打赌印第 500 的每个车手都知道发动机是如何工作的。

如果你想更进一步了解如何使用场效应晶体管，我们做到了。一旦你理解了在电子层面上到底发生了什么，阅读关于[半导体进展](https://hackaday.com/2015/11/23/new-magnetic-semiconductor/)的文章就会变得更加有趣。
