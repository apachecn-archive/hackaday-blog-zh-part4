# “Glixie”为夜光显示器注入了新的活力

> 原文：<https://hackaday.com/2020/03/20/glixie-puts-a-new-spin-on-glow-in-the-dark-displays/>

对于我们看到的以新的和不寻常的方式使用谢妮管的许多项目来说，有一个规模较小但通常非常有趣的展示队列可以归入“类谢妮”类别。在这些项目中，除了释放惰性气体之外的一些东西被用来形成角色。这个滚动磷光单字符显示器就是这样一个项目，我们认为它看起来棒极了。

遵循这些构建特有的*ixie 命名约定，[StephenDeVos]将其称为“Glixie”这与手工制作的谢妮的尺寸相当，但还不至于大到难以操作。显示模式是在黑暗中发光的薄膜，它旋转经过一串垂直的紫外发光二极管，随着圆柱体的旋转，这些发光二极管依次点亮，逐列建立点阵字符。当整个角色建立起来的时候，第一栏有些褪色，但还不至于令人反感。我们喜欢整个建筑，用激光切割的木头，黄铜和钢硬件。看看下面的视频吧。

如果这种磷光显示策略看起来很熟悉，那是因为我们以前见过它。还记得[这种磷光显示的持续性](https://hackaday.com/2018/04/05/persistence-of-phosphorescence-clock-displays-youtube-stats-too/)吗？或者也许[这个会写时间的机器人时钟](https://hackaday.com/2018/03/22/arduino-clock-jots-down-the-time-in-uv/)？这不是一个新的想法，但(斯蒂芬)的执行力是不可战胜的。
