# 极简主义伴侣制造者让你保持咖啡因

> 原文:[https://hack aday . com/2019/12/24/minimalist-mate-maker-keep-you-caffeinated/](https://hackaday.com/2019/12/24/minimalist-mate-maker-keeps-you-caffeinated/)

美国人喜欢他们的咖啡。英国人喜欢喝茶。在南美洲，度过一天的最佳方式是喝巴拉圭茶，一种由巴拉圭茶制成的茶。它通常在社交场合分享，由一个人准备饮料供大家享用。虽然咖啡因当然值得一个仪式，但它从来不需要。走到街上，你会看到到处都有人胳膊下夹着热水瓶，保持水的温度，并随时准备给手中的马黛茶续杯。

斯坦利真空热水瓶是旅途中饮酒者的一个相当受欢迎的选择，但是阿根廷政府最近对外国进口品实施了新的限制。[Roni Bandini]决定建造一个最小可行的交配机器,这样他就可以一直有热水供应。

Arduino Nano 将水加热，并在 LCD 屏幕上显示上升的温度。当温度刚刚好的时候，显示器要求你的杯子。超声波传感器检测杯子并分配草图中确定的一定量的水。在失去味道之前，耶巴叶可以使用几次，所以机器会跟踪并让他知道什么时候该更换它们了。休息之后你可以品尝一下简短的演示。

比方说，你没有准备完美的伴侣，结果总是很糟糕。那总比太冷好，但还是不理想。为什么不做一个温度感应杯垫，当它冷却到最佳状态时会提醒你呢？
