# 模拟坦克驾驶模拟器巡逻一个微小的物理景观

> 原文:[https://hack aday . com/2022/10/18/analog-tank-driving-simulator-patrols-a-tiny-physical-landscape/](https://hackaday.com/2022/10/18/analog-tank-driving-simulator-patrols-a-tiny-physical-landscape/)

在 20 世纪 70 年代，当 3D 计算机生成的图形仍然只是未来的幻想时，你如何建立一个实用的坦克驾驶模拟器？如果你是欧洲的一所加油机学校，解决办法就是用一台[大型数控机床驱动一台摄像机围绕一个微型地形模型](https://www.festungsmuseum.ch/fasip/)(德语，[翻译](https://www-festungsmuseum-ch.translate.goog/fasip/?_x_tr_sl=auto&_x_tr_tl=en&_x_tr_hl=en&_x_tr_pto=wapp))。在广告之后的视频中，[汤姆·斯科特]试驾了这辆车。

![The old computer was replaced with a Raspberry Pi](../Images/299e5b8def9376570d15d0fa23f6600e.png)

The original computer was replaced with a Raspberry Pi

该模拟器在法国开发，为新学员提供了一种更安全、更具成本效益的方法来教授驾驶 Centurion、Leopard 2 或 Panzer 68 坦克的基础知识。受训者坐在安装在液压操作运动平台上的逼真“驾驶舱”中，他的面前有一个电视屏幕，它与安装在大型龙门式数控平台上的摄像头相连。

相机的镜头安装在一个旋转金属底座的正上方，该底座可以滑过 12 米长的地形模型，并将它的角度发送到液压系统。它甚至会根据坦克在模型上的当前位置来改变坦克的操作，以模拟不同的表面，如泥土、雪或沥青。

这些系统中的最后一个在瑞士图恩的军事训练中心一直使用到 2004 年，直到被瑞士军事博物馆保存下来才被废弃。最初的 70 年代的计算机、电子设备和液压系统最终失灵了，因此博物馆对该系统进行了彻底的翻新，以使其恢复正常工作，供博物馆游客使用。它尽可能保持原样，但找不到原始计算机的部件，所以它被替换为一个 Raspberry Pi 和自定义接口板。

三十年来，这些模拟器可能训练了几千名坦克驾驶员，甚至在技术有限的情况下，也很好地训练了学员进行实战。除了为操作员、驾驶员和飞行员提供培训，模拟器也很有趣。我们已经看到了一些令人印象深刻的自制模拟器，包括一个 [A-10 疣猪](https://hackaday.com/2021/09/30/the-ultimate-brrrt-simulator-fully-featured-a-10-warthog-cockpit/)，一个 [F-15 模拟器](https://hackaday.com/2019/09/04/eagle-reborn-f-15-simulator-from-a-wreck/)用真实的残骸建造，甚至还有一个[星际飞船的舰桥](https://hackaday.com/2018/09/03/unleash-your-inner-starship-captain-with-this-immersive-simulator-console/)。
