# 从废料堆中建立一个边缘照明标志

> 原文:[https://hack aday . com/2022/04/22/building-a-edge-lit-sign-from the scrap-pile/](https://hackaday.com/2022/04/22/building-an-edge-lit-sign-from-the-scrap-pile/)

无论是在商店橱窗里还是安装在美国宇航局任务控制中心的控制台顶部，边缘照明亚克力都是制作高能见度标志的流行选择。部分原因是它们惊人的全息影像外观，但也因为它们非常便宜且易于生产。只是*如何*便宜又容易？看看[最近从【Hack Modular】](https://www.youtube.com/watch?v=7SgkAXmq56o)获得的这个视频，这是一个完美的例子。

现在你可能会认为你需要像 CNC 路由器这样的东西来制作这样的标志，对于更复杂的图像来说，这是有争议的。但是如果你只关心文本，并且有一只相当稳定的手，你可以用一个印刷模板和一个剃刀片来完成蚀刻步骤。当然，[Hack Modular]为这个标志选择的 LCD 风格字体特别适合手工切割——如果你对边缘字体感兴趣，这种方法可能不是你想要的。

[![](../Images/1d3cd3ecad88d5ea90e3bc5150fc2aa0.png)](https://hackaday.com/wp-content/uploads/2022/04/edgelit_detail.jpg)

This linear LED provides a more consistent light.

随着文字被刻在丙烯酸树脂上，唯一缺少的成分就是光。为此，[Hack Modular]使用了 12 伏线性 LED 条形灯。也就是说，它不是像传统的灯条那样点缀着单独的 led，而是提供一个连续的光带，非常适合这种应用。它被粘在一块废木头上，一个旧的 Meccano 设备上生锈的角支架被用来将丙烯酸树脂固定在中心线上。如果你认为最终产品看起来像是从垃圾中创造出来的东西，不要难过，这就是我们的意图。

最终结果看起来很棒。事实上，如果我们说实话，这比我们想象的用手工工具要好得多。诚然，字体的选择与此有很大关系，但话又说回来，我们不会介意我们所有的边缘照明丙烯酸标牌最终看起来像大的七段显示屏。
