# 手势控制的拍摄装置非常直观

> 原文:[https://hack aday . com/2022/10/28/手势控制-拍摄-齿轮-作品-超直观/](https://hackaday.com/2022/10/28/gesture-controlled-filming-gear-works-super-intuitively/)

如果你独自工作，拍摄好的视频可能是一项艰巨的任务。[Pave Workshop]开发了一系列手势响应工具来提供帮助，[重点是创建一个简单直观的界面。](https://www.instructables.com/Controlling-Devices-With-Your-Hands-Automated-Film/)

该系统基于使用 Kinect V2 来感知用户做出的手势，然后控制场景中的各种对象。例如，一个招手的动作可以指示相机滑块向前或向后移动，一个停止的手势会告诉它停止。双手合拢或分开的特殊手势表示相机应该放大或缩小。灯光也可以通过将拳头拉向或远离它们来控制，以改变它们的亮度。

对于一个如此顺利的项目来说，困难在于细节。[铺平道路研讨会]展示了一切节点的细节。JS 被用来编织从定制相机滑块到飞利浦 Hue 灯泡和其他 Arduino 组件的所有东西。

在 YouTube 上的演示视频[中，该项目看起来令人印象深刻。我们以前也见过其他一些令人印象深刻的](https://www.youtube.com/watch?v=uXg5oKD287A)[自动拍摄设备](https://hackaday.com/2019/09/17/speed-up-filming-with-this-jawdropping-8-axis-camera-crane/)。
