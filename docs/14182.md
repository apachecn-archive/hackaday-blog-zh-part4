# 甚至毁灭战士现在也可以运行毁灭战士了！

> 原文:[https://hackaday.com/2022/07/12/even-doom-can-now-run-doom/](https://hackaday.com/2022/07/12/even-doom-can-now-run-doom/)

多年来，对任何新入侵的硬件的标准测试都是这样的:它能运行 *DOOM* 吗？id Software 1993 年的经典第一人称射击游戏已经出现在所有东西上，但这里有一个来自[kgsws]的有点特别。[是*末日*，在*末日*本身](https://github.com/kgsws/doom-in-doom)内部运行。

那么这一壮举是如何实现的呢？在最初的 DOS *DOOM II* 可执行文件中有一个代码执行漏洞，它已经被用来在最初的文件中运行更现代的*巧克力 Doom* 。它作为游戏中的纹理出现，给人一种奇怪的效果，就像在电影院观看一样。

广告下方的视频展示了游戏中的游戏，但真正的价值在于它对漏洞的深入描述，它带我们了解了游戏的一些内部工作原理，并巧妙地解释了正在发生的事情。它以一个特制的电影胶片结束，在里面可以玩*毁灭战士* -in- *毁灭战士*，甚至*己烯* -in- *毁灭战士*。拿起你值得信赖的电锯，这将是一个漫长的夜晚。
