# 适用于任何车间的起重机

> 原文：<https://hackaday.com/2019/09/26/a-crane-fit-for-any-workshop/>

有时，我们会在车间遇到比我们预想的要大一点的物品。发动机缸体、铁砧或特别坚固的机床。举起这些东西也许是可能的，但是这样做有受伤的风险，也许会疝气或者更糟。对于这些时刻，一个设备特别齐全的车间将包括一台小型起重机，并且[车间从零开始]已经发布了一个视频，我们将它放在休息时间的下方，展示了一个特别好的模型的构造。

制造一台起重机本身并不是一件困难的事情，因为大多数对金属加工有兴趣的读者都可能制造一台。这个视频吸引人的地方是看着金属制品做得很好的满足感，虽然他确实使用了带锯和钻床，但视频中没有很多东西是用更基本的工具做不到的。其结果是一个漂亮的项目，可能比许多商业产品更好，尽管直觉告诉我们，枢轴点用套筒和销钉比用螺栓更好。提升力来自现成的液压油缸。

令人惊讶的是，这里很少出现起重机，但至少我们给你带来了一台阳台起重机。
