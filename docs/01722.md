# 口袋妖怪哭了，他们是如何工作的

> 原文：<https://hackaday.com/2019/01/04/pokemon-cries-and-how-they-work/>

如果你是看着神奇宝贝电视连续剧长大的，你自然会熟悉所有你最喜欢的口袋妖怪的叫声。动漫里的生物大多倾向于说自己的名字，一遍又一遍。为那些听到远处“鼠兔，鼠兔！”，至今仍不由自主地抽搐。

然而，游戏在这方面有很大的不同。神奇宝贝的第一代是在 Game Boy 上发布的，它根本不具备提供完整比特流音频的声音功能。相反，声音是根据各种参数为各种神奇宝贝合成的。这是一个非常复杂的系统，但是不要害怕-[通过](https://youtu.be/gDLpbFXnpeY)可以得到帮助。

该视频按位分解了每个神奇宝贝叫声的参数是如何存储的，以及它们是如何合成的。它被分成几个容易理解的部分，首先解释了 Game Boy 的声音硬件是如何工作的，有两个脉冲通道和一个噪声通道，然后再详细解释为什么一些神奇宝贝会发出相同或相似的叫声。

这是复古游戏逆向工程中的杰作，并以高质量的图形指南专业地展示了软件层面上正在发生的事情。甚至有一个模拟器，你可以用它来探索原始游戏中的各种叫声，并生成你自己的叫声。

既然我们已经跟上了神奇宝贝的速度，那么修复一个 37 年前的游戏中的错误怎么样？休息后的视频。
