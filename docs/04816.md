# 这个 Arduino 会一直盯着你

> 原文:[https://hack aday . com/2019/11/13/this-arduino-keeps-its-eyes-on-you/](https://hackaday.com/2019/11/13/this-arduino-keeps-its-eyes-on-you/)

[威尔]想要制造一些不需要高精度 3D 打印的电子眼睛。他最终得到了一个宽容的设计，使用了一个 Arduino 和六个伺服电机。你可以在下面的视频中看到眼睛四处移动的视频。

物料清单非常简单，包括一个 Arduino、一个驱动板和一个操纵杆。3D 打印零件无需支架即可轻松打印，并可与 PLA 配合使用。除了开孔之外，不需要太多的后期处理，尽管他确实打磨了实际的眼球，这听起来很痛苦。

结果是一个很好的紧凑的包装来容纳六个电机，并且眼睛运动的响应时间非常令人印象深刻。这将是一个很好的道具，甚至是机器人的一部分，来代替传统的眼睛。

虽然操纵杆很好，但我们希望看到连接一个超声波传感器，这样当你走过房间时，眼睛可以跟踪你。也许它们可以被挂在明年万圣节的旧画像后面。话说回来，也许一个[头骨](https://hackaday.com/2019/02/01/this-creepy-skull-shows-time-with-its-eyes/)会更好。如果你想重温伺服系统，从[激光炮塔教程](https://hackaday.com/2018/01/07/afroman-teaches-intro-to-servos-builds-laser-turret/)开始。
