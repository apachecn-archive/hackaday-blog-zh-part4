# 看看在象棋比赛中作弊是否可行

> 原文:[https://hacka day . com/2022/10/11/看着-如果-在象棋中作弊-很难-甚至是可能的/](https://hackaday.com/2022/10/11/seeing-if-cheating-at-chess-the-hard-way-is-even-possible/)

所有关于作弊丑闻的淫秽故事震撼了冠军级别的国际象棋世界，你会认为我们已经在 Hackaday 上至少深入研究了这个故事，特别是考虑到这个故事的技术角度。但我们没有，这不是因为我们对所谓的欺骗的细节感到不安；更确切地说，这是因为在这样一个故事中用双关语来表达太容易了。挂得最低的水果不总是最甜的。

但是，我们将试一试，并且在我们观察[一个实验以确定是否有可能以被指控的特定方式欺骗](https://www.youtube.com/watch?v=D4YZnJLmb_A)时，完全诚实地玩这个游戏。对于外行人来说，19 岁的特级大师汉斯·尼曼被指控作弊，可能是通过使用藏在他直肠里的遥控性玩具[。这个想法是让共犯使用这个玩具，它包含一个由应用程序通过蓝牙或 WiFi 控制的振动电机，根据下棋人工智能对游戏的分析向[尼曼]发送建议的步骤。](https://www.newsweek.com/hans-niemann-magnus-carlsen-chess-sex-toy-scandal-1749903)

[尼曼]是否作弊并不是这里的关注点，而是[钢铁队长]的实验只是第一次看是否有可能使用拟议的技术作弊——最重要的是，不被抓住。他试图根据指控复制[尼曼]现在在锦标赛上接受的扫描制度，看看是否可以通过各种厚度的肌肉检测到性玩具的替身——连接到 ESP32 的触觉马达。[钢铁队长]没有表现出[尼曼]据称表现出的那种对工艺的执着，而是用大香肠片代替人肉。然后，他尝试通过不断增加的午餐肉层来扫描该设备的射频辐射。我们不会破坏结果，除了说 baloney 实际上对某些事情有好处。

我们之前已经介绍过另一种[侵入性较小的作弊方法](https://hackaday.com/2022/09/07/how-to-be-a-stinkin-chess-cheat-sockfish/)，鉴于上述结果，这种方法可能更容易被发现。
