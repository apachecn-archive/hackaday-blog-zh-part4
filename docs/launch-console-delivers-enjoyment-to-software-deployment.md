# 启动控制台为软件部署带来乐趣

> 原文：<https://hackaday.com/2020/03/26/launch-console-delivers-enjoyment-to-software-deployment/>

有时感觉好像所有与机器的良好物理交互都消失了。鉴于我们目前的细菌战形势，这可能是一件好事。但是如果只有不到十个人会接触到某个东西，那么找点乐子，为这些东西制作你自己的界面可能是没问题的。

有趣似乎是[sethvoltz]复古风格的发射控制台背后的一些灵感。这种基于双因素授权令牌的系统负责一项通常不引人注目的重要任务——将代码部署到产品中。

控制台以 Yubikey 为中心，这是一种用于 2FA 的硬件加密狗。扳动防护开关将启动发射序列，然后是时候将 Yubikey 插入 3D 打印的锁芯并等待授权。如果树莓派决定所有系统都准备好了，那么钥匙可以转 90 度，蘑菇按钮捣碎。休息后我们允许你看解密的演示。逗留一会儿，看看锁芯内部的 CAD 视图。

游戏机文化是伟大的，但旧的全尺寸橱柜肯定占用了很多空间。如果你是一个硬件爱好者，[看看这个测试多个伺服系统的迷你控制台](https://hackaday.com/2020/01/05/space-saving-servo-tester-console-looks-space-worthy/)。
