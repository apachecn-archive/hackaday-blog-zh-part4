# Synthfonio 让音乐变得像周日早晨一样简单

> 原文:[https://hack aday . com/2020/02/02/synthfonio-makes-music-easy-like-Sunday-morning/](https://hackaday.com/2020/02/02/synthfonio-makes-music-easy-like-sunday-morning/)

这首歌献给所有热爱音乐的人，他们在灵魂中感受到音乐，但不一定在头脑中理解它。任何乐器都不应该阻碍表达，但似乎它们都是如此(也许除了卡祖笛)。

[FrancoMolina]的[混合合成器-MIDI 控制器](https://www.instructables.com/id/The-Synthfonio-a-Musical-Instrument-for-Everyone/)是想要演奏音乐和实际演奏之间的捷径。基本上，你按下 Synthfonio 脖子上的一个按钮来设置音阶，并通过按下安装在身体上的控制器中的限位开关来演奏实际的音符。如果您感觉忧郁，您可以通过在按下根音的同时按下相对次要音符的颈部按钮来切换到次要音阶，例如 A+C=Am。想换八度？只需上下滑动整个控制器，总共三次。

所有这些开关都复用到两个 arduino——一个 MKR1010 用于 USB MIDI 控制，一个 bare '328 用于提供内置合成器声音。电力来自一个升压的 18650，它可以用一个非常便宜的电路板充电。[Franco]有所有可用的代码和文件，所以去享受制作音乐的乐趣吧，不要被一堆理论关掉。按下那里的按钮查看演示。

如果“便携”对你来说意味着口袋大小，那么让[这个迷你木管乐器 MIDI 控制器](https://hackaday.com/2017/11/13/pocket-woodwind-midi-controller-helps-you-carry-a-tune/)让你大吃一惊。
