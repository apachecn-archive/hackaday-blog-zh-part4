# 疯狂的火腿:2019 年业余无线电场日

> 原文：<https://hackaday.com/2019/06/19/hams-gone-wild-amateur-radio-field-day-2019/>

在业余无线电给人的所有印象中，伟大的户外活动通常并不重要。人们似乎认为哈姆坐在一个黑暗的房间里，桌子上堆满了无线电设备，敲打着莫尔斯电码直到深夜，对窗外的世界漠不关心。所有这些听起来都有点像硬核游戏，真的。

虽然这种形象在很多情况下都适用，但火腿确实喜欢每年至少外出一次。随着 2019 年业余无线电场日的到来，这一天即将到来。北美各地的火腿会在每年六月的第四个完整周末，离开他们的棚屋，来到这个世界，在某种公共场所，通常是公园或其他绿地，开展业务。部分野餐，部分社区拓展，部分睡衣派对——持续整个周末，昼夜不停——hams 利用野外日向公众展示业余无线电真正的亮点:在严峻条件下的实时全球通信。

这也是一个让人们对获得执照感到兴奋的机会，许多现场日地点举办“广播”站，以便无执照的人可以在有执照的运营商的监督下尝试联系。获得许可但装备不足的火腿也有机会操纵别人的装备，也许还能获得第一次钻机采购。而且也有很多机会学习新的模式，比如 [FT8](https://hackaday.com/2018/11/02/ft8-saving-ham-radio-or-killing-it/) 和 [WSPR](https://hackaday.com/2013/03/21/wspr-transmitter-shows-true-value-of-raspberry-pi-for-hacking/) 。例如，你的书记员正在寻找一些关于开始使用 [APRS](https://en.wikipedia.org/wiki/Automatic_Packet_Reporting_System) 的指导，这是一个自动数据包报告系统，用于像[高空气球跟踪](https://hackaday.com/2015/03/24/aprs-tracking-system-flies-your-balloons/)这样的事情。

如果你对学习如何正确操作无线电设备有任何兴趣，你应该找到最近的场地并停下来。美国无线电接力联盟(ARRL)有大量的比赛日信息，从定位 1500 个比赛日地点的地图到将在那个周末举行的比赛的规则，再到设置和操作有效的比赛日设置的指南。今年会有 40，000 只火腿上市，如果你能顺道过来问几个问题，他们都会很兴奋的。
