# 2022 科幻大赛:你的家庭助手，哈尔 9000

> 原文:[https://hackaday . com/2022/04/23/2022-科幻竞赛-你的家-助手-hal-9000/](https://hackaday.com/2022/04/23/2022-sci-fi-contest-your-home-assistant-hal-9000/)

任何看过《2001:太空漫游》的人都会很容易记起 HAL 9000，这台有感知能力的计算机在“发现一号”飞船上背叛了它的人类同伴。[Ben Brooks]决定重新创造预感数字生物，[，并把它作为智能家居助手。](https://hackaday.io/project/184869-hal-9000)

该构建由 3D 打印组件组成，看起来非常像电影中的 HAL。它作为一个独立的设备运行，连接到[Ben]的家庭助理实例，这是一个自托管的家庭自动化解决方案。在 ESP8266 和 DF 播放器迷你模块的帮助下，该设备能够播放电影中的声音剪辑。它由一个按钮或运动传感器触发，但它也连接到家庭助理，以获得一些额外的智能。这种设置可以确保 Chromecast 在电视上播放内容时 HAL 保持沉默，以免干扰重要的观看。

总的来说，这是一个有趣的电影致敬建设是非常真实的源材料。让我们只希望这个哈尔不会有任何疯狂的想法，迫使[本]拆开它的处理器来停止它的危险阴谋。
