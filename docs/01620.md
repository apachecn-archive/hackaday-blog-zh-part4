# 发光二极管和 Pi 让你虚拟装饰这个在线圣诞树

> 原文：<https://hackaday.com/2018/12/23/leds-and-pi-let-you-virtually-decorate-this-online-christmas-tree/>

任何装饰过圣诞树的人都知道，灯光才是真正的装饰。但是无论你在它周围缠绕多少线，似乎永远都不够。此外，标准设置要么坐在那里什么也不做，要么只是定期闪烁。打哈欠。

但是黑客们的目标更高，【leo.currie】的[交互式“可绘画”圣诞树](https://hackaday.io/project/162892-christmas-tree-paint)将灯光游戏向前推进了一步。标准灯串被连接到 ESP8266 的 WS2811 RGB LEDs 灯串取代。一个连接到树莓派的摄像头被设置成将树的图像传输到互联网上的所有人，但有一个特殊的转折:它还创建了树上每道光的地图。这允许响应于 Pi 上的网页上的用户输入来单独控制灯。结果是，你可以实时地用你喜欢的任何颜色来绘制这棵树，或者上传各种动画 gif 来显示在树上。你可以[直接玩树](https://hecknswell.com/tree.html)，或者在下面的视频中观看 Pi 不可避免地被抱死的回放。

想象一下这个的可能性。为什么不在你家的屋檐上垂直悬挂许多 LED 灯串，做成一个巨大的低分辨率显示屏呢？我们以前展示过大量的[大型互动 LED 圣诞显示屏](https://hackaday.com/2017/12/23/touch-panels-make-this-christmas-tree-interactive/)，我们很想看看你们会推出什么。
