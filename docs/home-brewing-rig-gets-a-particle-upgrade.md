# 家庭酿造装备获得粒子升级

> 原文：<https://hackaday.com/2019/03/15/home-brewing-rig-gets-a-particle-upgrade/>

家庭酿造是一种消遣，它可以是一门艺术，也可以是一门科学，取决于你的喜好。[布兰登·萨特伦]是一个非常倾向于科学的人。有很多可以做的来监测和控制酿造，和[布兰登]是许多谁建立了自定义硬件，以帮助获得最佳结果之一。现在，硬件应该升级了。

[布兰登]最初的 BrewBuddy 系统依赖于粒子光子，这是一个有用的平台，尽管如此，它仍在不断发展。随着新粒子 Argon 的推出，[Brandon]将目光投向了增加马力后可能实现的新功能。LCD 屏幕上添加了图形，压电传感器检测发酵过程的开始。这是对第一次构建的原始温度监控和绘图功能的补充。

从一个微控制器平台升级到另一个平台可能会令人头疼，但在这种情况下，只需要进行一些小的更改。修改了 3 行代码来说明不同的 pin 分配，其余的代码都很整齐。这证明了粒子平台的兼容性，这次升级是如此容易。

在之前，我们已经讨论过第三代粒子板，我们希望在未来的更多版本中看到它们。休息后的视频。
