# LEDCard:便携式环形灯

> 原文:[https://hack aday . com/2023/01/01/led card-the-pocket able-ring-light/](https://hackaday.com/2023/01/01/ledcard-the-pocketable-ring-light/)

有多少次你发现自己在试图获得一个物体的清晰特写镜头时，在摸索照明？尽管如今智能手机配备了非常好的相机，但正确照亮物体并拍摄令人印象深刻的微距照片并不完全是它们的强项。这就是[MisterHW]的[led card](https://github.com/MisterHW/LEDCard)非常受欢迎的地方。它不仅提供了一个信用卡大小的环形灯，还允许插入一个模压的丙烯酸透镜来拍摄高质量的微距照片。

该项目目前由一个带可充电锂离子纽扣电池(LIR2430)的 PCB 和一个 USB 供电的充电控制器组成。给 LED 卡充电(或插入新充电的锂离子硬币电池)后，只需按一下按钮，就会通过 LM3410 LED 驱动器 IC 点亮 SMD LEDs。轻轻按下(半按)ON 按钮可获得中等亮度，完全按下可获得最高亮度。最后，在 led 灯亮起的情况下按下测试按钮，执行电池电量测试，如果电池正常，则关闭 led 灯。如果灯一直亮着，就该给 led 卡充电了。

正如[MisterHW]指出的，LEDCard 足够小巧，无论你去哪里都可以随身携带，这也使它适合用作应急手电筒。这也不是设计的最终迭代。未来的(增量)改进包括环形灯的扩散器等等。即便如此，在目前的状态下，LEDCard 已经是一个经过验证的设计。
