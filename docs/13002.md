# 介绍通用雅达利键盘外壳

> 原文：<https://hackaday.com/2022/03/10/introducing-the-universal-atari-keyboard-case/>

[10p6]想知道如果 Atari 在 1985 年在其 16 位和 32 位计算机生产线上使用标准化键盘会是什么样子。想象是有趣的，但建造东西更好，因此他们开始创造这样的东西。[进入通用雅达利键盘盒](https://www.youtube.com/watch?v=c4bg_00VENQ)。

外壳设计灵活，可以接受包括 Atari ST 和 Falcon 在内的各种型号的键盘。该键盘可用于雅达利 Mega、TT 或台式雅达利电脑，无需 mods。它还将现代外设带到这些旧的 Atari 平台上，支持使用现代 USB 鼠标，同时还使用两个板载操纵杆端口。电源和软盘指示灯存在，但巧妙地隐藏在外壳下，只有在照明时才可见。它还包括 5 瓦立体声扬声器，以充分利用雅达利的声音硬件。

最终的零件全长 473 毫米，由树脂 3D 打印而成，具有高质量的表面光洁度。结果是如此之好，它几乎看起来像一个真正的工厂键盘。

如果你经常玩你的老式雅达利机器，你想要一个很棒的键盘来使用它们，这可能是你的设计。[10p6]承诺很快将设计文件上传到 Thingiverse，供那些渴望复制这项工作的人使用。

我们也看到复古雅达利键盘被转换成现代机器。休息后的视频。
