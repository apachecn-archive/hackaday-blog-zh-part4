# 乐高蒸汽机复制品

> 原文:[https://hack aday . com/2021/01/25/steam-engine-replica-from-Lego/](https://hackaday.com/2021/01/25/steam-engine-replica-from-lego/)

如果一百年前的工程选择只是略有不同，我们可能会在一个充满蒸汽机而不是内燃机的世界中结束。不过，就目前而言，蒸汽机仅限于少数小众应用，当然还有由发烧友打造的模型。例如，这个[完全由乐高建造，是 1907 年最初生产的蒸汽机的比例复制品](https://www.youtube.com/watch?v=PWD5hDkim6Y)。

该车型基于 2500 马力的三倍膨胀四缸发动机，该发动机在 20 世纪上半叶实际使用。由于该模型仅使用乐高(和一些橡皮筋)建造，因此它使用真空而不是工作蒸汽进行操作，但原理基本相同。它还包括 Corliss 阀门，这是一种来自 1850 年的技术，使用旋转阀门，极大地提高了当时蒸汽机的效率。

这种构造是对原始机器的令人印象深刻的再现，由于顶部有一个工作阀，甚至可以以极慢的速度运行，从而可以详细查看其操作。最大转速约为 80 rpm，非常接近原机的 68 rpm 运行速度。但是，如果你更喜欢你的蒸汽机在现实世界中的应用，一定要看看这个蒸汽动力割草机。
