# Agora，一个可破解的电子纸时钟

> 原文：<https://hackaday.com/2019/11/26/agora-a-hackable-e-paper-clock/>

[Daniel Zilinec]欣赏电子纸的美感，并认为它会成为一个伟大的时钟。

电子纸的自然出现当然吸引了很多黑客。我们已经看到了从[打字机](https://hackaday.com/2018/10/20/e-ink-typewriter-is-refreshingly-slow/)到虚拟[货币追踪器](https://hackaday.com/2019/08/12/paperledger-an-e-ink-cryptocurrency-ticker/)的一切。Agora 时钟设计为电池供电；一款经典的床头柜闹钟。凭借其广角视野，甚至对光的反应，即使在黎明时分也很容易看到它。

通过预先设计 PCB，他为用户节省了大量时间。它有一个内置的充电 IC，背光 led 和按钮垫。你所需要做的就是从现有的 thingiverse 文件中打印出案例，然后组装起来。原理图和固件也可供更有事业心的黑客使用。

还有一个有点令人费解的手表版本的时钟。穿上这样一件肯定会成为一种时尚。尽管如此，有机的感觉和可能的字体还是值得考虑的。
