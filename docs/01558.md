# 这个消防机器人能承受高温吗？

> 原文:[https://hack aday . com/2018/12/28/can-this-fire-fighting-robot-take-the-heat/](https://hackaday.com/2018/12/28/can-this-fire-fighting-robot-take-the-heat/)

消防是一项艰难而危险的工作，它将人类置于第一线，定期拯救生命和财产。它是一些机器人助手的首选，而[【Ivan】已经带着一个有趣的构建](https://www.youtube.com/watch?v=badfpajRI5Q)介入，尽管它不会很快在你的市政部门服务，但它让我们思考各种可能性。

这是一个无线电控制的机器人，它的大脑有一个 Arduino Uno。几个电机驱动板用于驱动四个挡风玻璃刮水器电机。早在网上购物时代之前，刮水器电机就是黑客的主食——一种便宜、容易获得的高扭矩电机，可以很容易地驱动用于一系列爱好项目。他们说只有 90 年代的孩子记得。

就送水而言，这个机器人的证书有点不足，只能携带 1 升水。然而，我们很欣赏[Ivan]使用特百惠容器作为水箱——通过一些附加配件，这可能是在其他项目中盛水的一种很好的方式。小型 DC 动力泵由工业固态继电器控制——对于可能淋湿的机器人来说是个不错的选择。还有一个车载二氧化碳灭火器，但遗憾的是它还没有插入任何东西。

这个建筑是一个[伊凡]的经典——大，有趣，3D 打印的规模比我们习惯的要大得多。这是我们之前看到的他令人印象深刻的坦克建造的有力延续。休息后的视频。
