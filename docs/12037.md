# 用吸力来测量建筑物

> 原文:[https://hack aday . com/2021/11/28/scale-buildings-with-power-of-suction/](https://hackaday.com/2021/11/28/scale-buildings-with-the-power-of-suction/)

城墙挡不住【以利亚·西里奥利】。这位未来的超级英雄一直在忙着用他的[独立真空攀爬器](https://www.youtube.com/watch?v=9Lz9TDEoBJQ)攀爬建筑物的侧面。(视频插播在休息之后。)

在受到空军设计挑战赛的[获奖项目的启发后，我们的便衣十字军战士开始研究一对原型真空攀岩器。](https://phys.org/news/2012-06-usu-team-personnel-vacuum-climber.html)[木制原型](https://www.youtube.com/watch?v=n5RRYKWgesI)获得了意想不到的成功，所以休息后很快就开始制作视频中展示的模型。第二个版本中的主要改进包括使用 1/2 英寸的丙烯酸树脂代替胶合板，以及改进的垫圈，以更好地密封许多建筑墙壁的不完美外部。

虽然该系统最终仍会与砖墙(和其他不完美的表面)斗争，但当攀爬更光滑的混凝土墙时，它的性能已经足够了。虽然丙烯酸是一个比胶合板好得多的选择，但其中一个丙烯酸板仍然出现了裂缝。即便如此，结果还是不言自明，我们不得不为发明者对他的设备看似无条件的信任鼓掌。

我们最近没有看到[以利亚·西里奥利]的后续报道，所以希望他正忙于第三版，而不是被困在某处。与此同时，看看有人如何利用回收的微波变压器完成类似的爬壁壮举。
