# 空线轴是有用的工具，比如计数器

> 原文：<https://hackaday.com/2022/09/25/empty-spools-make-useful-tools-like-counters/>

把事情做好是怎么回事？有一个宋飞的轶事可以归结为:拿一个日历，做一件事，在你做这件事的每一天打上一个大大的 X。很快，你会渴望一连串的 x，然后你会想把整个月都涂黑。这是可靠的建议。

【3D Printy】也喜欢条纹，2022 年初做了几个分辨率。作为全年制作的 30 个视频中的第一个，他们展示了[这个巨大的 3D 打印计数装置](https://www.youtube.com/watch?v=Dy3pF2HQYqo)(视频，嵌在下面)，它使用空的细丝线轴，一些 3D 打印，没有其他东西。这些都是孵化线轴，它不会对每种类型都有效，但设计应该放大和缩小以适应其他口味。

这不是[3D Printy]的第一次计数器竞技——他已经做了几个正常大小的，并在这个过程中完善了一个聪明的结转机制，这当然是开源的。因此，每个线轴代表一个数字，核心中有印刷部分，使计数结转到下一个线轴。虽然早期的计数器使用螺纹杆，但这个巨大的版本骑在 2.5 毫米的光滑杆上，所以线轴可以很容易地分开。但是一切是如何保持在一起的呢？当然是由 TPU 细丝制成的巨大松紧带——因为答案总是在房间里。
