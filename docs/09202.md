# 用糖果和工程表达你的爱

> 原文：<https://hackaday.com/2021/02/13/express-your-love-with-candy-and-engineering/>

情人节还没有礼物吗？我们衷心建议你启动打印机，准备好再次爱上工程学，因为[JBV 创意]已经设计了一个超级可爱的机器，它会转动任何人的心。他称之为有史以来最过度设计的糖果自动售货机，我们不得不同意。这肯定是我们见过的最美丽的景色之一。

这种优雅的设计中没有任何电子设备，只有纯粹的机械手摇乐趣。转动曲柄可以同时做两件事——它可以在控制糖果数量的滑槽下方来回移动一个小观察板，同时移动传送带，将货物送到收货区。

这整个设计是绝对的天才，尤其是分离机制，切断了糖果的流动，但允许带继续移动。请务必观看构建视频，其中[JBV 创意]在没有任何工具的情况下轻松地将机器组装在一起，并观看后续视频，其中他讨论了工程挑战并展示了为此付出的努力。

当然，有不止一种方法可以过度设计糖果分配器。这里有一个找到了花生巧克力豆的圣杯——那些没有得到花生的巧克力豆。
