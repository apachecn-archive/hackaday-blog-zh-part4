# 打造一流的驻极体话筒

> 原文:[https://hack aday . com/2020/11/02/building-a-top-notch 驻极体麦克风/](https://hackaday.com/2020/11/02/building-a-top-notch-electret-microphone/)

驻极体话筒能够提供高质量的输出，并因其平滑的频率响应而备受推崇。然而，与其他类型不同，它们不能简单地直接插入混音台。相反，它们需要特殊的高阻抗电路来提取音频信号进行录音。[DJJules]是这些话筒的忠实粉丝，[他决定打造一款高品质、易于使用的电路，并与社区分享。](https://www.instructables.com/OPA-Based-Alice-Microphones-a-Cardioid-and-a-Figur/)

该项目的目标是创建一个与 TSB2555B 驻极体膜盒相匹配的电路，该电路可以与幻像电源一起使用，并且可以用容易获得的器件构建。[DJJules]过去使用过场效应晶体管，但厌倦了经常寻找过时的部件。相反，这种设计依赖于双通道运算放大器 OPA1642，其低静态电流意味着它非常适合使用幻像电源。这意味着麦克风不需要电池，使用双运算放大器使电路能够正确驱动平衡的音频连接。

该电路设计用于安装在普通的 BM700 或 BM800 话筒主体内，[如果有兴趣自己制作 PCB，可以从 PCBWay 订购。](https://www.pcbway.com/project/shareproject/Dual_OPA__Alice__Microphone_Circuit.html)Shapeways 上还有一个[鞍座](https://www.shapeways.com/product/7Q5MC658X/tsb-2555bxz3-saddle-for-bm-700-bm-800)，用于将驻极体膜盒整齐地安装在外壳内。

最终的结果令人印象深刻，这个项目将成为任何渴望开始制作自己的设备的人进入 DIY 麦克风领域的一个好机会。当然，如果你正在寻找一种更简单的方式开始，还有更简单的构建。休息后的视频。
