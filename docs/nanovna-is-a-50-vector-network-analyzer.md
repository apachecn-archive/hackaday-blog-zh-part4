# NanoVNA 是一款价值 50 美元的矢量网络分析仪

> 原文：<https://hackaday.com/2019/08/11/nanovna-is-a-50-vector-network-analyzer/>

曾几何时，示波器又大又贵。现在你几乎可以用任何预算买到各种尺寸和功能的瞄准镜。矢量网络分析仪(VNA)还没有同样的发展，但是 NanoVNA(T1)可能会改变这种情况。[IMSAI Guy]花 50 美元左右买了一个，还做了一系列关于它的视频。剧透提示:他喜欢。你可以在下面看到他发布的几个视频中的一个。

NanoVNA 很小，但扫描频率从 50 千赫到 900 兆赫，并且有一个触摸屏。例如，如果你需要把它拖到天线塔上，这种设备会使用可充电电池。作为一个快速测试，你可以在视频的早期看到一个橡胶鸭天线的分析。该设备将回波损耗显示为曲线图，您可以使用光标精确测量这些值。它还显示了电抗的史密斯圆图。

如果不喜欢触摸屏，还可以通过 [USB](https://www.youtube.com/watch?v=zw7Dp1nwvD8) 控制设备。你需要从 [Google Drive](https://drive.google.com/drive/folders/1-JViWLBOIzaHTdwdONX2RP8S4EgWxoND) 获得软件，你也可以从那里获得手册。此外，还有额外的固件文件，因此您可以刷新仪器。例如，您可以将最高频率限制为 300 MHz 或选择更大的字体。

你可能想知道为什么要把频率限制在 300MHz。根据手册，该仪器在 300 MHz 下具有更好的性能。也有在现场校准设备的规定。

对于 50 美元来说，这是相当可观的。[IMSAI] guy 还做了一个关于它的[精度](https://www.youtube.com/watch?v=ggcQ7GsGc8s)的视频，用示波器和频率计数器检查它的输出。

我们已经研究过其他[廉价的 vna](https://hackaday.com/2018/05/06/ham-reviews-minivna/)，尽管与我们之前见过的一些相比，这款 vna 的频率范围很有趣。我们也测试了[模拟探索 2 号的 VNA](https://hackaday.com/2018/06/20/analog-discovery-2-as-a-vector-network-analyzer/) 。它的成本要高一点，但也能做其他事情，但 VNA 的最高频率是 10 MHz。
