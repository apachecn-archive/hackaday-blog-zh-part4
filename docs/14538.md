# 麦克纳姆轮式机器人底盘接受 PS4 控制器的命令

> 原文:[https://hack aday . com/2022/08/19/me canum-wheeled-robot-chassis-takes-commands-from-ps4-controller/](https://hackaday.com/2022/08/19/mecanum-wheeled-robot-chassis-takes-commands-from-ps4-controller/)

从机器人到机场行李处理设备，麦克纳姆轮都是受欢迎的选择。根据它们的旋转方向，它们可以在任何平面方向上产生力，提供很大的机动性。[[ATOM]着手建造这样一个机器人底盘，并在这个过程中学到了很多东西。](https://www.instructables.com/PS4-Controlled-Holonomic-Drive/)

这个设计和我们过去见过的相似。这个机器人有四个麦克纳姆轮，每个都由自己的马达驱动。根据不同轮子的旋转方向，机器人可以前进、后退，甚至左右扫射。此外，由于每个车轮上的滚轮[，它可以有效地转弯而不会过度打滑。](https://hackaday.com/2018/04/04/laser-cut-mecanum-wheel-for-the-budget-roboticist/)ESP32 充当“机器人”的大脑，允许它通过蓝牙通过 PS4 游戏手柄轻松遥控。

如果你想建造一个小型机器人底盘，在狭小的空间里移动自如，这可能是一个很好的学习项目。所有必要的部件都相对容易获得，PCB 文件[可以在 GitHub 上获得。](https://github.com/maker-ATOM/PS4-Controlled-Holonomic-Drive)

如果你喜欢麦克纳姆车轮的想法，但需要更大的东西，可以考虑从一套气垫板车轮马达开始。
