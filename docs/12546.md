# 硅胶器件:DIY 可拉伸电路

> 原文：<https://hackaday.com/2022/01/23/silicone-devices-diy-stretchable-circuits/>

基于聚酰亚胺薄膜的柔性电路现在很常见，你可以在多家工厂制作原型，其成本几乎是普通黑客可以接受的。聚酰亚胺薄膜对于如此薄的东西来说非常坚韧，但最终它会撕裂，对于较大的元件，弯曲半径非常有限。但是 T2 可拉伸电路又如何呢，因为在电路中你可以弯曲、扭曲和拉伸？让我们来介绍一下[硅胶器件](http://www.raframakers.net/wiki/Main/SiliconeDevices)。比利时 Hasselt 大学的一个研究小组一直在制作真正灵活的硅基电路基板原型，成功地将各种 SMT 元件类型与双层互连、过孔和外部触点集成在一起。

使用普通的激光切割机和一些容易制作的特殊工具——承诺提供指南——应该可以复制这个过程，这纯粹是收集一些特殊材料，并使用你周围的边角料来完成其余部分。互连使用 [Galinstan](https://www.rotometals.com/low-melting-point-alloy-galinstan-68-5-ga-21-5-in-10-sn-50-grams/) ，这是一种镓、铟和锡的低熔点合金。不幸的是，这种材料相当昂贵，并且由于镓的含量而不能空运，没有专门的处理，花费相当大。但除此之外，除了一些丙烯酸片，一些乙烯树脂，铜箔和一些喷雾，没有什么是遥不可及的。

建造过程与我们通常看到的相反，首先将元件和铜接触板放置在涂有底漆的乙烯板上。这张纸上用激光标有部件轮廓，以便能够正确放置。对，没错，他们是用激光切割机给乙烯基做标记，一种含氯的塑料。坚持一下这个想法。

绝缘层和基板层是通过[用一层透明硅胶刮涂](https://www.youtube.com/watch?v=XwZbvwbY8WM)制成的。互连层的形成是通过将新鲜的乙烯基片粘贴到暴露的触点上，并通过激光切割来暴露焊盘和互连迹线。下一步，用刷子刷上花俏的 Galinstan，去掉乙烯基模板。冲洗并重复下一层绝缘硅胶，更多的电路走线，然后使用激光切割机精确蚀刻通孔区域，以增加更多的金属化。最后，在整个组件上涂上一层硅胶，再次用激光将硅胶从接触垫上蚀刻掉，再涂上一点焊锡，就大功告成了。很简单，如果我们的制造商空间没有禁止激光切割乙烯基的规定。

这显然是一个非常简短的概述，这里有一个非常详细的指导指南为你准备好，以及一份正式的研究论文，详细说明为什么会这样，为什么你可能想自己尝试一下。

如果你对定制的可穿戴设备感兴趣，你可能会记得[这篇关于硅电路的早期文章](https://hackaday.com/2016/01/07/conductive-silicone-makes-flexible-circuits/)，以及同一时期的这篇[看起来怪异的有机东西](https://hackaday.com/2016/03/15/stretchable-traces-for-flexible-circuits/)。
