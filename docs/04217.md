# 比萨饼烤箱建立练习遗忘健身球

> 原文:[https://hack aday . com/2019/09/18/pizza-oven-build-exercises-forgot-gym-ball/](https://hackaday.com/2019/09/18/pizza-oven-build-exercises-forgotten-gym-ball/)

看，这就是披萨爱好者追寻梦想的结果。对于[phammy57]来说，这可能是一个足够天真的开始——他得到了一块披萨石，然后可能是一个巨大的摇摆披萨切割器。也许他甚至学会了如何把面团抛向空中。但是每次[phammy57]把一个自制的馅饼放进电烤箱，这种不安的感觉就变得更强烈了。最终，他屈服于披萨的制作方式，建造了一个烧木头的烤箱。

关于这个建筑最有趣的事情也是最重要的:这个比萨饼制作机在一个健身球上旋转，这个健身球是形成烤箱的基础。为了做到这一点，[phammy57]将球推过一大块胶合板上的一个洞，有效地创造了世界上最大的 Pogo Bal ( [还记得那些](http://www.retroland.com/pogo-ball/)？).然后，他用塑料包裹球作为脱模剂，并铺上蛭石、水泥和水的浓稠混合物。

[phammy57]用轻质砌块建造了底座，为木材储存区的顶部雕刻了一个漂亮的拱门。一旦穹顶被固定在底部，开口被切开，用砖勾勒出轮廓，他就切了一个通风孔，建造了烟囱。最后，是时候添加绝缘毯材料，鸡丝，更多的蛭石，和石膏涂层完成。休息之后简单看一下里面。

这是一个漫长的建造、固化和烧制过程，但最终结果看起来棒极了。我们打赌它也像冠军一样吃披萨。可能会让这个 45 秒披萨烤箱物有所值。
