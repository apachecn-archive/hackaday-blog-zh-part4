# SLA 打印便携式 Wii 掌机的考验和磨难

> 原文:[https://hack aday . com/2021/11/28/the-trials-and-traits-of-SLA-printing-a-portable-Wii-hand held/](https://hackaday.com/2021/11/28/the-trials-and-tribulations-of-sla-printing-a-portable-wii-handheld/)

G-Boy 套件彻底改变了围绕建造便携式家用游戏机的亚文化，允许将整个 Wii 塞进原始 Game Boy 大小的形状因子中。[克里斯·唐宁]对这个领域并不陌生，他自己找来了一套 G-Boy 套件，尽自己最大的能力来打造它。(视频插播在休息之后。)

然而，他想超越原始套件的 FDM 印刷部分。因此，他联系了套件开发商 Gman，后者向他提供了 3D 模型文件，以便他自己尝试更高质量的 SLA 打印。[Downing]在过去的表单 2 中有一些 SLA 打印的经验，但是在表单 3 的构建中发现了一些独特的挑战。

SLA 印刷的好处是它提供了更精细的细节和表面光洁度。这尤其适用于用户经常使用的外壳和按钮。然而，标准的树脂，船舶与形式 3 有翘曲的问题，特别是在薄平的墙壁，构成了大部分的 G-Boy 案件。

其他问题包括标准固化 SLA 树脂比更软的 FDM 塑料更难拧入螺钉，这阻碍了设计的组装。它也很脆，容易破碎。

作为一个正确成品的粉丝，[唐宁]决定无论如何都要打磨和油漆外壳。悲剧发生了，因为超过一年了，喷雾罐开始吐出大块的东西。然而，它意外地变成了一个胜利，产生了一个有吸引力的花岗岩石般的完成，实际上看起来相当不错。

G-Boy 套件使 Wii 便携版成为主流，并吸引了许多新的开发者进入亚文化。[唐宁]可能是一个坚定的场景，但仍然在建造的过程中学习新的技能。

我们迫不及待地想看到场景中接下来会发生什么，尽管我们怀疑在我们说话的时候有人已经在那里砍一个罕见的 PlayStation 5 了。
