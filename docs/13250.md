# 详细的大屏幕万用表回顾

> 原文：<https://hackaday.com/2022/04/04/detailed-big-screen-multimeter-review/>

看来大屏幕廉价米真的流行起来了。[TheHWcave]对一款 [KAIWEETS KM601](https://www.youtube.com/watch?v=2XAa2By9Xg8) 做了非常详细的评测，这款车与你能从常规渠道获得的其他几十个中国品牌完全一样。你可以在下面的视频中看到评论。

如果说我们从这个视频中学到了什么，那就是我们确实学到了你可以用一个标尺来识别没有标记的引信。里面的保险丝没有标记，所以他想知道它们看起来是否是正确的值。我们本想在受控条件下引爆它们，但我们知道他不想在测试后才销毁库存保险丝。

拆卸和分析非常详细，有一些有趣的花絮，特别是电流探针如何通过电路板。一个有趣的观察是，内部的控制器芯片具有一次性可编程存储器，他推测校准数据被编程到其中一次。如果是真的，这将使仪表不可能有任何方法来现场校准它。当然，电表可能会在启动时使用某种已知的内部基准电压源进行自校准，但这很难判断，因为似乎没有该器件的数据手册。

里面有几个让人惊讶的细节。内部有两个独立的处理器。一个可能是温度函数。焊接工艺——主要是去除多余的焊料——还有待改进。尽管他建议使用一套更好的探头，但这种仪表对于它的价格来说已经足够好了。

这款仪表看起来与我们之前看到的[不同品牌的仪表](https://hackaday.com/2021/12/07/multimeters-go-big-screen/)相同。或者你可以调查一下更便宜的[米](https://hackaday.com/2018/10/23/3-multimeter-teardown/)。
