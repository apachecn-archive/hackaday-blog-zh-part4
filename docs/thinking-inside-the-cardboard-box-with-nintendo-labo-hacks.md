# 用任天堂 Labo Hacks 在(纸板)盒子里思考

> 原文：<https://hackaday.com/2018/07/19/thinking-inside-the-cardboard-box-with-nintendo-labo-hacks/>

硬纸板是最简单的制作实物的方法之一，比我们通常在这里提到的 3D 打印和激光切割要简单得多。因此，当任天堂发布他们的 Labo 纸板配件系列时，并不需要天才就能预测到官方产品将会有大量的用户创作。任天堂很聪明，不仅为这种创造力的聚集提供了一个互联网论坛，他们还举办比赛来突出一些最好的作品。

winner's circle 中最令人印象深刻的项目是将独一无二的纸板创作与使用 [Toy-Con Garage](https://labo.nintendo.com/invent/) 编写的定制软件相结合，这是任天堂 Switch 控制台内置的可视化软件开发环境。用户运行任天堂 Labo 的“[发现](https://labo.nintendo.com/discover/)”活动后，就可以进入车库，这些活动将带领用户了解他们购买的 Labo 配件是如何工作的。因此，这个学习和发现过程也可以作为一个介绍性的编程教程，教用户如何创建软件来点亮他们的定制纸板作品。

任天堂开放了 Labo 活动背后的一些机制供用户创建自己的活动，这很酷，但这只是任天堂 Switch 功能的一小部分。我们对不同的人有不同的方法。我们中的一些人喜欢逆向工程那些小小的快乐骗局是如何运作的细节。其他人[破解一些东西来避免一个更令人沮丧而不是有趣的游戏谜题](https://hackaday.com/2017/10/25/teensy-script-plays-nintendo-switch-strikes-out/)。还有一些人不满足，直到[完全脱离沙盒](https://hackaday.com/2018/07/01/nintendo-switch-gets-internal-trinket-hardmod/)。
