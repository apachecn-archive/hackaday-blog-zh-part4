# Flipper 零黑客工具获得自定义应用程序的用户界面编辑器

> 原文：<https://hackaday.com/2022/12/10/flipper-zero-hacker-tool-gets-ui-editor-for-custom-apps/>

[Mikhail]为 Flipper Zero 多用途黑客工具发布了一个方便的 GUI 编辑器/生成器工具,使得布局和 UI 元素的制作更加容易和直观。

那些决定钻研滚动自己的应用程序或附加组件的人会发现这是一个方便的资源，特别是当它为可视元素生成必要的代码时。它也不局限于放置图标。框、线、点、文本等可以自由布局，使事物看起来恰到好处。

要使用它，只需将不同大小的图标拖放到屏幕区域。非图标 UI 元素，如框架、线条、文本和其他元素，可以通过单击按钮来放置。若要四处移动元素，请先点按“选择”按钮，然后根据需要拖移内容。要微调定位(或更改字符串的文本)，可以在模拟屏幕的右侧访问和修改所选元素的属性。当事情看起来不错的时候，切换到代码标签并复制，以便在你的 *Flipper* 应用程序中使用。

不熟悉零号脚蹼？这是一种无线多功能工具。一个非常有趣的装置旨在让无线探索和实验像它的吉祥物海豚一样触手可及，非常可爱。
