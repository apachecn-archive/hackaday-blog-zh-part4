# 伺服绘图仪不需要任何外来的东西

> 原文:[https://hack aday . com/2022/01/05/servo-plotter-needs-nothing-exotic/](https://hackaday.com/2022/01/05/servo-plotter-needs-nothing-exotic/)

尽管 3D 打印机的广泛使用使线性轴承和丝杠等物品变得更加常见，但你仍然无法跑到当地的大盒子五金店买到它们。然而，你可以得到抽屉滑轨，任何爱好商店都可以卖给你一些遥控伺服。这和 Arduino 可以使[成为一个简单易用的绘图仪](https://www.instructables.com/Two-Servo-and-Drawer-Rail-Pen-Plotter/)。问问[吉姆德]就知道了。你也可以在下面的视频中看到它的动作。

当然，伺服系统通常不是你在绘图仪中使用的。但是滑块将伺服系统的旋转转化为线性运动。一个 X 伺服和一个 Y 伺服是你所需要的。另一个 microservo 使用铰链上下移动笔，你也可以从五金店买到。

好看吗？不。它能做出令人惊叹的艺术品吗？不，再来一次。但这是那种你可以从你碰巧挂在周围的东西中拼凑出来的东西，尤其是如果你打算扔掉一张放有幻灯片的旧桌子或旧柜子。

这将是一个伟大的雨天项目。我们对简单的绘图仪项目很感兴趣，即使你可以把一支笔和你的 3D 打印机或 CNC 机器连接起来。不过，这些不适合你的[白板](https://hackaday.com/2021/01/03/blackboard-becomes-tidy-pen-plotter/)。
