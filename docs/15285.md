# 只有一辆 TO92 就够了

> 原文:[https://hackaday.com/2022/11/05/when-only-a-to92-will-do/](https://hackaday.com/2022/11/05/when-only-a-to92-will-do/)

随着通孔元件被表贴元件取代，我们开始看到曾经很常见的元件外形的消失。许多东西，如金属罐晶体管，几年前就变得稀有了，而其他的仍然存在，尽管在越来越少的地方。其中之一是曾经无处不在的 TO92 模制塑料晶体管，我们在 2022 年根本看不到它。[Sam Ettinger]是 D 型塑料泡泡的粉丝，他甚至为新一代人创造了这些泡泡。

虽然 TO92 在当时是一种相对微型的封装，但它仍然足够大，可以轻松地将 SOT23 或类似的 SMD 封装器件安装在小型 PCB 上。因此，只有足够空间放置器件和三条线的小电路板就制作好了，可以封装了。环氧树脂模制 T092 的效果很差，因此改为制作 T092 外壳的 SLA 印刷。它可以整齐地放在 PCB 上，形成完美的 TO92 封装。我们确信半透明的粉红色包装在 20 世纪 60 年代会引起一些人的注意。

总有一天，旧电子产品的修复者会使用并改进这种技术来替换报废的部件。毕竟，我们以前见过这种技术。
