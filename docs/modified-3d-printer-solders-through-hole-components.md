# 改进的 3D 打印机焊料通孔组件

> 原文：<https://hackaday.com/2021/07/22/modified-3d-printer-solders-through-hole-components/>

表面贴装技术对于一般的电子产品，尤其是业余爱好者来说，是一种神奇的力量倍增器。但有时你别无选择，只能使用通孔元件，这意味着即使你可以利用 SMD 进行大部分设计，你可能仍然需要花一点时间拿着烙铁。[或者不，如果你有一台备用的 3D 打印机。](https://www.youtube.com/watch?v=7pUrcxzmpQs)

我们在这里得到的只是来自[hydrosys4]的一个相当简短的视频，所以没有很多构建细节。但是这里发生了什么已经很清楚了。从看起来更长的 LK4 打印机开始，【hydrosys4】增加了一个支架来固定烙铁，以及一个焊丝导向器。焊料由一个或多或少标准的挤压机处理，一旦被烙铁加热，就被送入接头。这里的秘密调料可能是夹具，用 3D 打印的夹具将通孔连接器固定在打印机的底座上。由于 PCB 位于连接器的顶部，只需教授每个接头的 X-Y-Z 位置，加热，并用挤压机推进焊料。

下面的视频显示它在高速运行；我们把它放慢到 25%，以了解它在现实中的情况，虽然它可能不快，但它很精确，而且不会累。对于一次性电路板，它可能没有太多的应用，但如果你正在制造小型 PCB，它是一个天才的解决方案。在之前，我们已经见过[类似的焊接机器人，但是为了保持这个机器人的简单，我们向【hydrosys4】致敬。](https://hackaday.com/2019/04/26/3d-printer-becomes-soldering-robot/)
