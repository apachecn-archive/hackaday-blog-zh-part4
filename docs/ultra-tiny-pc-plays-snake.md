# 超微型电脑玩蛇

> 原文：<https://hackaday.com/2019/02/23/ultra-tiny-pc-plays-snake/>

[史蒂夫·马丁]曾经演过一个关于“让我们变小！”你不得不怀疑[保罗·克林格]是否是这种套路的粉丝，因为他最近完成了一个非常小的 3D 打印电脑，可以玩蛇。好吧，它不是一台真正的个人电脑，也不太实用，但它确实运行良好，会成为一个很好的桌面谈资。你可以在下面的视频中看到它所有的小荣耀。

3D 打印机打印出了一个微型电脑外壳、一个显示器和一个操纵杆。PC 包含一个 ATtiny1614、一个 RGB LED 和一些光纤，看起来像机箱照明。这个显示器真的是一个小有机发光二极管屏幕。一个 5 向开关变成操纵杆。

一些不太明显的细节给我们留下了深刻的印象。例如，电脑外壳有一个用抛光透明 PLA 印刷的实际窗口，这样你就可以看穿它。另一侧的一个小孔可以让你用一些弹簧针进入设备进行重新编程。甚至有螺纹的热固性插入物，让螺钉拧入它们的配合孔，尽管[Paul]承认你可以把孔做得更小，只让螺纹咬入塑料。

如果您正在为 at tiny 寻找一个开发环境，我们刚刚谈到了这一点。我们喜欢在小小的硬件上玩经典游戏。
