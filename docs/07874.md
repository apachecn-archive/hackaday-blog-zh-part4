# 用于 Raspberry Pi 4 的优雅模块化外壳系统

> 原文:[https://hack aday . com/2020/09/28/an-elegant-modular-enclosure-system-for-the-raspberry-pi-4/](https://hackaday.com/2020/09/28/an-elegant-modular-enclosure-system-for-the-raspberry-pi-4/)

[NODE]长期以来一直在试验 Raspberry Pi 服务器和迷你电脑，非常清楚布线会如何迅速变成老鼠窝。他的最新创造是[迷你服务器版本 3](https://n-o-d-e.net/node_mini_server3.html) ，这是一个用于 Raspberry Pi 4 的模块化外壳系统，旨在将其变成实用的计算盒。

基本外壳是一个 92 毫米 x 92 毫米 x 26 毫米的 3D 打印框架，带有定制的 PCB 顶盖。其中一个主要目标是将所有主要连接器集中在一侧，使 micro SD 插槽易于使用。为此，[NODE]创建了一组定制的 PCB 适配器，将 USB-C 和 HDMI 端口与其他 USB 端口路由到同一侧，并将 micro SD 插槽移动到机箱底部。还设计了一个薄型适配器来将 mSATA SSD 连接到 USB 3 端口中的一个，并且外壳内部有一个或两个冷却风扇的空间。与以前版本的迷你服务器不同，不需要对 Pi 本身进行任何硬件修改。

我们可以看到的唯一缺点是，它不允许外部访问 GPIO 端口，但整个项目是开源的，专门允许人们进行自己的修改。

[NODE]热衷于将 Raspberry Pis 转变为定制计算设备，从小型终端设备和[袖珍服务器](https://hackaday.com/2015/02/26/file-sharing-in-your-pocket/)，到完整的[笔记本电脑](https://hackaday.com/2016/11/05/turn-that-old-tablet-into-a-sub-100-linux-laptop/)。
