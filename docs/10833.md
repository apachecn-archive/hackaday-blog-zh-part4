# 使用 Game Boy 摄像头控制视频通话

> 原文:[https://hack aday . com/2021/07/25/dominate-video-calls-with-game-boy-camera-web cam/](https://hackaday.com/2021/07/25/dominate-video-calls-with-game-boy-camera-webcam/)

我们不能保证这一切都是积极的，但毫无疑问，当你使用 Game Boy Camera 加入视频通话时，你会得到很多关注。假设他们能认出你。1998 年玩具的分辨率和视频质量肯定没有很好地老化，这是在它被压缩并通过互联网发送之前。

[![](../Images/dbd7fd9ea6da30298e9110bce13f26e2.png)](https://hackaday.com/wp-content/uploads/2021/07/gbcam_detail.jpg) 从技术的角度来看，这个其实很简单，虽然有点复杂。[RetroGameCouch]没有以任何方式修改 Game Boy 相机，他只是将它连接到超级 Game Boy，而超级 Game Boy 又被插入到超级任天堂中。从那里，SNES 的视频输出通过 HDMI 转换器，最后[在一个便宜的 HDMI 捕获设备](https://hackaday.com/2020/12/21/heavy-raspberry-pi-user-keep-an-hdmi-to-usb-capture-device-around/)中终止。他特别的 SNES 已经用分量视频进行了修改，但在普通硬件上，你必须满足于复合视频。

所有这些适配器和电缆的最终结果是，Game Boy 摄像头的实时馈送，以及超级 Game Boy 的屏幕边框，都可以作为标准的 USB 视频设备在计算机上使用，可以用于任何您希望的程序。如果你对恢复静态图像更感兴趣，我们最近看到了一个项目，它可以让你通过 WiFi 从游戏机摄像头中提取图像。
