# 过度驱动真空管并释放其中的魔法之光

> 原文:[https://hack aday . com/2021/07/15/over driving-vacuum-tubes-and-releasing-the-magic-light-within/](https://hackaday.com/2021/07/15/overdriving-vacuum-tubes-and-releasing-the-magic-light-within/)

我们都见过被哄骗释放出少量魔法烟雾的电子元件，这当然是让它首先工作的原因。但在过去，零件是由玻璃和金属制成的，而且更加坚硬——你几乎可以对它们做任何事情，它们不会释放出神奇的烟雾。非常无聊。

当然，除非你知道“红色电镀”的秘密(大卫·洛维特在下面的视频中探索了这个秘密)。我们一直在关注[【大卫】在真空管](https://hackaday.com/2021/06/19/vacuum-tube-magic-comes-to-the-741/)方面的工作，前面提到的基本上无烟的元件被他用来构建[一个简单的一位微处理器](https://www.youtube.com/playlist?list=PLnw98JPyObn1GUapiXLlGm8RrpQF-J_c1)。他的电路倾向于相当温和地驱动电子管，但在一个有趣的扭曲中，他让他的破坏性的一面出来一点，真的推了几个电子管，看看会发生什么。发生的事情非常戏剧性——当足够多的电子从阴极流向阳极时，它们的集体动能将板加热到樱桃红，因此有了“红色电镀”这个术语。

[大卫]为他的刑讯室选择了一些受害者，尽管在电板上施加了大约 195 伏的电压，但并不是所有的受害者都合作。然而，有些管子却合作无间，很快呈现出一种非常不健康的光芒。一个电子管，一个 6BZ7 双三极管，真的上演了一场表演，管内的东西变得如此之热，以至于扭曲和短路在一起，导致一些令人印象深刻的烟火。把它想象成释放魔法之光而不是魔法烟雾。

看过 X 射线管是如何工作的后，我们不禁想知道大卫在制作这部电影时是不是有点超出了他的预期。可能不会——医用 X 射线管涉及的能量比这高得多——但是，看看红色电镀会产生什么样的意外排放可能还是很有趣的。
