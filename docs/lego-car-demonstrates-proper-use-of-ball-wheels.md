# 乐高汽车展示如何正确使用球轮

> 原文：<https://hackaday.com/2022/03/23/lego-car-demonstrates-proper-use-of-ball-wheels/>

车轮通常只是简单的圆柱体，尽管《机器人 2004》的粉丝可能会羡慕主角驾驶的漂亮的球形车轮车。YouTuber[砖块技术]决定用乐高汽车设计来评估球形轮子[的使用。](https://www.youtube.com/watch?v=5p-B51FKEps)

球轮的好处是，当在不同的轴上驱动时，它们可以转向多个方向，从而提高机动性。一组装有橡皮筋的驱动滚轮推着 52 毫米的 Duplo 球体，球轮可以向前、向后、向左和向右旋转。这给了乐高汽车一个相当整洁的扫射能力，以及就地旋转或以更传统的方式转向的能力。由于 BuWizz 模块允许远程控制乐高马达，汽车可以通过智能手机控制。

球形车轮不太可能在主流汽车中流行；驱动它们所需的机械复杂性使得这种设计对汽车来说不切实际。然而， [omniwheels](https://hackaday.com/2019/03/12/omni-wheels-move-this-cnc-plotter/) 和类似的设计已经在[叉车](https://www.google.com/imgres?imgurl=https%3A%2F%2Fi.ytimg.com%2Fvi%2F0DBXuZv38l8%2Fmaxresdefault.jpg&imgrefurl=https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3D0DBXuZv38l8&tbnid=CeAFbkIVYZUuDM&vet=12ahUKEwiB55qjjNn2AhUZy3MBHUKFCqAQMygAegUIARC5AQ..i&docid=EHTbf7M_sG_JWM&w=1280&h=720&q=forklift%20omniwheel&ved=2ahUKEwiB55qjjNn2AhUZy3MBHUKFCqAQMygAegUIARC5AQ)和其他此类低速应用中得到了一些应用，在这些应用中，向任何方向移动的能力都非常有用。休息后的视频。
