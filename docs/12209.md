# 3D 打印零件可承受蒸汽加热

> 原文：<https://hackaday.com/2021/12/18/3d-printed-parts-holds-up-to-steam-heat/>

汽轮机是各种工业机械的核心，尤其是用于发电的机械。[Integza]认为他需要更好地了解这项技术，并决定自己建造一个—[使用 3D 打印。](https://www.youtube.com/watch?v=RExAzZ9xNf0)

首先，需要一个蒸汽源，电炉上的高压锅被压入服务。蒸汽通过树脂印刷的喷嘴喷出，这种树脂比大多数 FDM 印刷的零件更耐热。类似地，涡轮机叶轮也是用树脂印刷的，蒸汽出口直接指向其叶片。

为了对零件进行真正的压力测试，需要更多的蒸汽。为了实现这一点，过氧化氢被泵入浸渍在钢丝绒中的二氧化锰催化剂中以产生蒸汽。这造成了绝对的混乱，但打印的部分仍然存在。

蒸汽轮机没有做任何有用的工作，但能够在高温下运行。我们很想看到这样一种设备实际上用来承受一些负载，也许是在某种 3D 打印发电涡轮机设计中。

或者，如果你更喜欢你的蒸汽涡轮机更经典的驱动，[考虑这个版本](https://hackaday.com/2020/10/06/making-a-modern-version-of-a-steam-engine-from-antiquity/)。休息后的视频。
