# 制作自己的柔性面板灯

> 原文：<https://hackaday.com/2019/08/03/make-your-own-flexible-panel-lights/>

在这个时代，生产价值就是一切。这些天，即使是最底层的内容创作者也在包装 4K 的智能手机和数码单反相机，所以如果你想竞争，你就需要硬件。照明是制作好视频的关键，所以你可以在手边找到一组灵活的面板灯。谢天谢地， [[DIY Perks]在这里向你展示如何打造你自己的。](https://www.youtube.com/watch?v=qlMtakaa7u8)(视频嵌入下方。)

构建一个好的视频灯光装置的关键是获得正确的 CRI，或者说是显色指数。在低 CRI 灯光下，颜色会看起来不自然，或者在视频中有奇怪的投射。[DIY Perks]已经努力寻找一家供应商，提供一系列不同色温的高质量 LED 灯条，这些灯条具有高 CRI 值，非常适合严肃的视频工作。

为了建造这种柔性面板，LED 灯条被粘在一个假皮革衬垫上，然后给它一个钢丝骨架，使它能够弯曲成各种形状。面板的每个角落都有皮革环，这样就可以使用灵活的铝制支架将灯安装到支架上。led 略微欠压，以帮助它们持续更长时间，并使它们能够通过笔记本电脑电源运行。

这个版本关注的是光的质量和可用性，而不是仅仅用一束明亮的发光二极管来解决问题，然后说它很好。结果很好，小组显示[比【DIY Perks】的早期版本](https://hackaday.com/2018/09/25/diy-studio-lights-to-improve-your-videos/)有了显著的改进。
