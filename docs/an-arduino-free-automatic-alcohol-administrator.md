# 无 Arduino 自动酒精管理器

> 原文：<https://hackaday.com/2020/04/05/an-arduino-free-automatic-alcohol-administrator/>

随着所有免提分配器设计的出现，制造商世界可能会面临 Arduino 短缺。我们开玩笑地说，但是使用 Arduino 来设计原型，然后让它做所有的工作，这太容易了，即使你知道这样做是多余的。

[ASCAS]接受了这一挑战，制造了一种廉价而简单的分配器，它依靠回收部件和必要的电子设备。它使用红外接近传感器模块来检测脏手指，并使用一个小型潜水泵将异丙醇、消毒剂或肥皂推送到你悬停的手上。该电源来自一根牺牲 USB 电缆，并通过一个晶体管进行切换，因此它可以插在墙上或便携式电源组中。

我们钦佩这个项目中的重复使用量，尤其是喷嘴变窄的圆珠笔件。请务必在休息后观看构建视频。

希望你们都还在规定的 20 秒内洗手。如果你开始下滑，为什么不制作一个数字沙漏，看着像素消失呢？
