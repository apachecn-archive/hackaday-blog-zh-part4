# 用于超导体悬浮的莫比乌斯带轨道

> 原文：<https://hackaday.com/2021/03/23/a-mobius-strip-track-for-superconductor-levitation/>

超导体是有趣的东西，尽管我们在日常生活中并不太依赖它们。它们会非常有用，只要它们不需要这么冷。尽管科学家们在实验室里辛苦地研究这个问题，但还是有一些乐趣，正如伊萨卡学院的莫比乌斯带悬浮轨道所展示的那样。(视频，下面嵌入。)

这个装置利用了超导体可以悬浮在磁铁上的事实，反之亦然。在某些条件下，由于磁通钉扎，超导体甚至可以锁定在磁体上方的位置，其中来自磁场的磁通“管”穿透超导体，并被超导体中的电流钉扎在适当的位置。这是一种令人惊叹的效果，超导材料似乎神奇地漂浮在磁性表面上方的锁定高度，与传统的磁悬浮截然不同。

赛道的建设并不简单。早期生产扭曲 540 度的莫比乌斯带的尝试在钢中没有成功。然后，研究小组改变了策略，使用一种更加柔韧的柔性塑料。这是覆盖在钕磁铁，以创造必要的领域，并由此产生的视觉效果是一个银砖磁路。

这是一个伟大的展示，非常直观地展示了莫比乌斯带和超导悬浮的概念。如果室温半导体成为现实，这很有可能成为一个永远在线的装置。[这也是我们见过的最酷的悬浮滑板之一背后的诀窍](https://hackaday.com/2015/08/05/secrets-of-the-lexus-hoverboard-revealed/)。休息后的视频。
