# 减少金属 3D 打印中的翘曲

> 原文：<https://hackaday.com/2022/04/13/reducing-warping-in-metal-3d-prints/>

我们习惯于在使用 ABS 等热塑性塑料打印时处理翘曲问题，但金属打印机也有这个问题。密歇根大学有一项新技术 [SmartScan](https://me.engin.umich.edu/news-events/news/smarter-3d-printing-makes-better-parts-faster/) ，有望减轻这一问题。你可以在下面看到一个关于这项技术的视频。

这个想法是在激光烧结之前开发一个打印部件的热模型，然后以一种热量不会积累的方式移动激光。该视频展示了以传统方式雕刻金属是如何随着激光加热区域而导致金属翘曲的。使用 SmartScan 热模型，他们能够将变形减少近一半。

听起来他们还没有把这个应用到 3D 打印机的烧结中，所以你只能假设结果是相似的。当然，通过粉末的热分布可能与通过固体金属的热分布不同，所以当然有必要进行进一步的测试。

我们不禁想知道，如果你只是简单地选择模式中的随机片段来运行，而不是完全按顺序运行，结果是否会一样好。当然，基于模型的方法应该更好，但我们想知道它是否比简单的掷骰子方法好得多。

与此同时，我们仍在等待我们的金属 3D 打印机。[铜](https://me.engin.umich.edu/news-events/news/smarter-3d-printing-makes-better-parts-faster/)似乎触手可及。如果事实如此，那就是，如果你不介意[进行一些后处理](https://hackaday.com/2021/07/09/3d-printed-copper-rocket-nozzle-costs-under-two-grand/)。
