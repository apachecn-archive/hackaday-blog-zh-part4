# 问 Hackaday:只有电脑能做什么？

> 原文:[https://hack aday . com/2021/09/22/ask-hack aday-what-can-only-a-computer-do/](https://hackaday.com/2021/09/22/ask-hackaday-what-can-only-a-computer-do/)

应用计算机来改进我们已经理解的东西是很容易的。例如，今天你可能会买一个合成器，而不是一架钢琴。它看起来和工作起来——有时——都像钢琴。但是它也可以做很多其他的事情，比如吹喇叭，给你伴奏或者录制和播放你的音乐。有很多这样的例子:文字处理器代替了打字机，MP3 播放器代替了录音机，PDF 文件代替了印刷材料。但是一些全新的东西呢？我是在看[音速 Pi](https://sonic-pi.net) 的时候想到这个的，这是一种通过编码来演奏的乐器。

但是回到键盘、文字处理器和 MP3 播放器。这些东西与其说是革命性的，不如说是进化性的。甚至像数码摄影这样的东西也不完全是革命性的。当然，我们大多数人不能在黑屋子里用 PhotoShop 做所有你能做的魔术，但是有些巫师可以。我们中的大多数人也不能设计出一个适合拍照的小册子，但是人们在没有电脑的帮助下每天都在做。那么我们用电脑做的那些事情是全新的呢？没有电脑的帮助你能做什么？

我不确定这个问题是否有明确的答案。它更像是一个光谱。sonic Pi——你可以在下面看到一个视频——使用编码来创建音乐，但它仍然是音乐。也许完全革命性的应该是直接做[大脑夹带](https://hackaday.com/2019/04/08/brain-hacking-with-entrainment/)。尽管如此，我认为通过实时编程进行创作的想法是一个更大的飞跃，但并不是每个人都欣赏这一点。例如，有些人喜欢 OpenSCAD，有些人希望使用鼠标来绘制 3D 模型。其他人使用 FreeCAD 并来回切换，因为 FreeCAD 可以使用 OpenSCAD。

因此，看起来计算机所做的大部分事情都是让事情变得更快或更容易。你不再需要西尔斯目录，因为你在网站上购物。甚至西尔斯也有一个。隔壁那对制作和销售定制婴儿毯的夫妇也是如此。当你订购时，你不必邮寄支票，小企业主也不必去邮局。一切都可以通过联网的电脑来完成。

但是有哪些东西是计算机独有的呢？以我的理解，比如现代的喷气式飞机，之所以能飞，只是因为有了电脑。你可能知道如何处理负面稳定性，但你永远无法足够快地做出调整。这可能是关键。毕竟在基础层面，计算机不会做什么神奇的事情。但是他们做事情的速度比我们想象的要快。但是如果有足够的时间，你可以手工计算，比如说，断层扫描或者 X 射线晶体学。只是工作量太大了。

通过编码制作音乐可能和大多数其他编码任务一样合格，尽管当时有演奏钢琴和没有电脑的提花织机。再说一次，像 Sonic Pi 这样实时处理的东西可能更适合计算机。当然，我想你可以假设某种机器，它不是接受代码的传统计算机，但如果你这么说，你真的只是在重新发明计算机。

所以问题是:有哪些事情你只能用电脑来做？似乎很容易，直到你开始思考它。当然，有些东西会更加实用或者做得更好。手工 3D 打印会很糟糕。在计算机出现之前，制作可供照相的拷贝、数学表格或 3D 动画是非常昂贵的，然而这些都是由某人完成的。完全由计算机实现的新领域在哪里？
