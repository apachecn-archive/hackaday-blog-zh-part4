# 安德 3 遇到米格焊机，使金属板凳-有点

> 原文：<https://hackaday.com/2021/11/23/ender-3-meets-mig-welder-to-make-a-metal-benchy-kind-of/>

当你能在 Aldi 买到 3D 打印机时，你就很清楚 3D 打印已经付诸实践了。至少对于 3D 打印的塑料版；金属印刷完全是另一回事。以可控的方式挤出一点熔化的塑料很容易，但当你试图对金属做同样的事情时，事情会变得更有活力。

至少这就是[卢卡斯]在他的 Cranktown City YouTube 频道上尝试建造一台金属 3D 打印机的经历。诚然，他从一开始就为自己设置了一个挑战，试图将一台米格焊机安装到安德 3 上，这是一个无论如何都没有准备好承受即将遭受的虐待的平台。[下面视频系列的第一部分](https://www.youtube.com/watch?v=L7sv3HkDyok)展示了第一次尝试，对于印刷商和印刷品来说，都以失败告终。

但是第一个原型，尽管有熔化的零件，给了[卢卡斯]足够的时间继续改进第二版，包括更好的构建板，为打印机的投标位提供热屏蔽，以及合法的 MIG 焊机送丝机。[Lucas]还内置了电流控制，与焊机控制有一个智能的非破坏性接口。这些改进足以尝试 Benchy 打印，开始时相当不错，但到了最后有点下垂。

尽管不完美，Benchy 是对第一版的无形小块的巨大改进，打印机对未来有很大的希望。你不能指责(卢卡斯)的一件事是太容易放弃一个项目；毕竟，他从零开始造了一个激光切割机，我[包括激光管](https://hackaday.com/2021/03/31/scratch-built-co2-laser-tube-kicks-off-a-laser-cutter-build/)。
