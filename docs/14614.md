# 自己做醋

> 原文:[https://hackaday.com/2022/08/26/make-your-own-vinegar/](https://hackaday.com/2022/08/26/make-your-own-vinegar/)

让发酵为我们工作是让人类让食物保存更久，让节日更有趣的原始方法之一。[迈克·G]一直在尝试自制醋，并发现最终产品是他烹饪的美味补充。

第一步类似于制作酒精饮料。拿一些含糖的东西，比如水果，和水混合，静置。野生酵母会以糖为食，制造酒精。一旦酒精含量达到 6-12%的范围，产生的液体可以从固体中分离出来并暴露在空气中。这使得醋杆菌将酒精转化为醋酸，产生醋。整个过程大约需要 30 天。

[迈克]的第一轮实验主要是新鲜水果，加入了葡萄干。为了防止白色霉菌的形成，混合物应该每天搅拌，但是生活阻碍了它，霉菌在所有的水果上都失去了控制，除了葡萄干。这给了[迈克]另一轮尝试干果，这是显着不容易发霉，并产生美味的风味醋。[Mike]还展示了它们在一些令人垂涎的菜肴中的应用。

DIY 醋生产过程只是乞求一些[发酵监测和自动化技术](https://hackaday.com/2018/06/27/raspberry-pi-tracks-starter-fermentation-for-optimized-sourdough/)。我们已经看到了大量的[酸面团](https://hackaday.com/2021/07/03/e-ink-equipped-sourdough-starter-jar/)和[啤酒生产](https://hackaday.com/2015/02/07/brewing-beer-with-a-sous-vide-cooker/)项目，我们怀疑这些项目经过一些小的改变也可以应用于醋的生产。
