# 一种死亡的摄影形式从灰烬中升起

> 原文：<https://hackaday.com/2022/12/02/a-dead-photographic-format-rises-from-the-ashes/>

有时，我们会偶然发现一个并不全新但仍然非常特别的黑客。因此，这是与[Hèrm Hofmeyer]的[指南，以再造一个胶卷盒为柯达光盘摄影格式](https://sites.google.com/view/fresh-kodak-disc-film/home)。它写于 2020 年，但描述了一个大约十年前的项目。

光盘格式是柯达在 20 世纪 80 年代的巨大希望，是摄影便利的极致，胶片是一个装在薄墨盒里的 16 张照片的圆盘。虽然相机是在市场的消费端，但它们比肉眼看到的更复杂，有当时最新的电子设备和一些创新的塑料多元素非球面透镜。面对更好的紧凑型 35 毫米相机，它失败了，因为光盘的便利性不足以弥补相对较小的底片，而且很少有实验室拥有专门的打印设备来获得这种格式的最佳效果。相机逐渐淡出人们的视线，这部电影在 20 世纪 90 年代末停止生产。

制造盒式磁盘的最大障碍来自盒式磁盘外壳本身。解决办法是从专门冲洗过期摄影胶片的国际电影救援组织那里购买二手胶片。这些阶段是在切割胶片盘片、在其边缘打孔并将其装入胶片盒之后进行的。在照片中，这是一个足够精确的过程，值得记住的是，在真正的墨盒中，这必须在黑暗中进行。

对于任何对摄影感兴趣的人来说，这都是一件有趣的作品，虽然盘式相机一直是一种消费快照相机，但我们可以看到它会吸引那些受 Lomography 影响的人。我们希望我们能得到一个磁盘盒，也许能设计出一个 3D 打印版本，让它更容易使用。
