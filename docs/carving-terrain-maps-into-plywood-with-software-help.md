# 在软件的帮助下将地形图雕刻到胶合板上

> 原文：<https://hackaday.com/2022/11/28/carving-terrain-maps-into-plywood-with-software-help/>

数控机床是非常通用的工具。在机械加工车间，他们可以加工各种金属和塑料零件。除此之外，他们可以雕刻包括玻璃在内的各种材料，甚至制作 PCB。[Steve]在他的店里有一台自己创造的数控机床，虽然他可能会将它用于那些普通的用途，但他的艺术创作今天与这些 3D 地形图一起展示。

制作一幅好的地形起伏图的关键是要有好的材料储备。[Steve]正在使用胶合板，因为材料中的自然层次感非常好地模拟了地形线，尤其是他正在使用的高质量海洋级桦木胶合板。确保选择没有结的作品可以极大地改善最终产品，花时间填充任何空隙也是如此。选择好的股票只是过程的一部分。[Steve]正在使用堪萨斯大学的一款新软件来创建模型，并输入到数控机床中。该软件名为 [TouchTerrain](https://touchterrain.geol.iastate.edu/) ，是对其他类似软件包的巨大改进。

凭借良好的库存和轻松创建 3D 地形图的能力，任何人都可以用这样的 CNC 机器轻松复制他们选择的地形。我们认为，如果分辨率足够高，这个过程可能很容易移植到 3D 打印机等其他工具上。[我们也看到过使用激光切割机](https://hackaday.com/2015/12/29/laser-cut-map-taken-to-the-next-level/)的类似构建，尽管使用的方法略有不同。
