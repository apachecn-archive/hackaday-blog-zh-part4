# DIY 变成点击购买了吗？

> 原文：<https://hackaday.com/2021/12/11/has-diy-become-click-and-buy/>

我们生活在 DIY 的伟大时代，尽管具有讽刺意味的是，其中一些是因为我们不需要自己动手的所有步骤。印刷电路板可以很容易地以低廉的价格订购，我们项目的机械部件可以方便地在网上订购，[制造的数量不超过一首歌](https://hackaday.com/2021/12/08/made-to-spec-the-coming-age-of-prototyping-as-a-service/)，或者在家里 3D 打印塑料就可以了。如果一切都外包出去，这*真的是 DIY 吗？是，不是，也有可能。*

这完全取决于你认为 DIY 的真正价值在哪里。它存在于想法、概念和设计中吗？或者在其实现中，制造？我认为大部分的价值实际上在于前者，正如我个人喜欢物理地构建许多项目的各个部分的许多过程一样。

例如，我最近设计并制造了一台热线数控泡沫切割机。或者更好，我设计了一系列改进的版本，因为我从来没有在第一次尝试中做对任何事情。一路上，我 3D 打印了新的和改进版本的塑料部件，尽可能多地消除了我耐心等待的小故障。这可能花费了好几个周末的时间，分散在几个月的时间里，但与测试、修复和重新设计所花的时间相比，在物理建筑上花费的时间或精力非常少。

而且，大部分零件都是在五金店买的。电机控制器防护罩和廉价的 Arduino 克隆来自易贝。甚至那些我自己做的 3D 打印的钻头也是由机器制造的——如果我订购的话，我对整个过程的体验不会有任何不同。

当然，工艺仍然存在，我们在黑客日项目中一直看到这一点。见鬼，我承认我仍然很享受亲手制作东西的过程。很平静。但是，如果说 3D 打印和廉价的短期印刷电路板服务促进了想法和项目的快速扩散，那就是许多项目的真正价值在于想法和文档。也就是说，我得开始写那个泡沫切割机了…
