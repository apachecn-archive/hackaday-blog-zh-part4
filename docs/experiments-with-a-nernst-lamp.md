# 能斯特灯实验

> 原文：<https://hackaday.com/2021/12/22/experiments-with-a-nernst-lamp/>

爱迪生的每部传记都谈到白炽灯的秘密是如何去除灯泡中的空气。当你使用传统灯丝时确实如此，但一位名叫能斯特的人发现，使用已经氧化的灯丝可以让你制造出一种在正常大气中也能正常工作的灯。在最近的一段视频中，你可以看到这些可以追溯到 19 世纪的怪事。

这种灯使用陶瓷灯丝，但缺点是灯丝需要很热才能让灯工作。这个实验用一根氧化锆棒，并试图点燃它。热源是丙烷火炬。

我们不建议像你在视频中看到的那样将鳄鱼夹挂在电线上。电阻、保险丝和隔离变压器会使设置更安全一些。如果这一切还不够危险，别忘了还有一个燃烧的丙烷火炬和燃烧的绝缘材料。什么会出错？

经过几次尝试，棒传导更多的热量，但最终，电流增加了。我们将不得不等待下一次分期付款，看看它是否真的点亮。

我们看到的最后一个这样的灯使用了一个陶瓷刀片作为灯丝。如果你想了解更多能斯特的历史，可以去看看我们的[博物馆](https://hackaday.com/2020/06/16/retrotechtacular-the-nernst-lamp/)。
