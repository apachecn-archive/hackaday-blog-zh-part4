# ingenius 土著巧妙螺旋输送机

> 原文：<https://hackaday.com/2022/06/04/ingenious-indigenous-artful-screw-conveyor/>

我们很多人都听说过阿基米德螺旋这个名字，但并不是每个人都知道螺旋输送机这个术语。这些人(遗憾的是，[Breeze Media]的摄像师没有告诉我们他们的名字或公司名称)将建造螺旋输送机的过程归结为一门艺术。

螺旋输送机是有用的，但许多人回避他们，因为他们看起来很难制造。在这个视频中，我们看到它是如何做到的。这个视频中的工作人员是在大型设备的金属上做的，但同样的方法也可以在小规模的塑料片或纸上使用。

它从切割垫圈和径向切割开始。当它们扭曲成最终形状时，洞会闭合，所以洞比形成中心的管子大一点。然后用大锤猛击，给它们一个轻微的螺旋(想象一个锁紧垫圈)。它工作了。狭缝边缘焊接在一起，形成一个“压缩”螺旋，末端焊接到管道上

现在是巧妙的部分。他们有一个高大的门架，只有几根带横杆的钢管杆，架设在工厂的院子里。在它下面，他们钻了一口井。管子的自由端向下延伸到井里。螺旋的底部被夹在井周围的基板上。接下来，管子被吊起形成最终形状。最后，一切都焊接到位。

在广告之后的视频中，他们正在制作一个螺旋喂食器。料斗下方的部分需要较低的螺距。于是他们夹了几圈，把主要部分拉出来，焊接，然后移动夹子，做支线部分。
