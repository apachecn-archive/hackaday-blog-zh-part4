# 用废金属制造踏板车尾气

> 原文：<https://hackaday.com/2020/06/23/building-a-scooter-exhaust-from-scrap-metal/>

当汽车上的一个零件出现故障时，通常的反应是从箱子里拿出一个新的零件。然而，有时，不管是出于需要还是仅仅出于爱好，手工制作一个解决方案是可能的。当[用废金属](https://www.youtube.com/watch?v=ZpoW93G9VZY)为他的摩托车制造一个新的排气管时【Samodel】就是这么做的。

这是经典后院金属加工技术的一个很好的例子。使用在排气口上摩擦的纸板模板重新创建法兰，残留的油留下清晰的印象。使用研磨机和钻头的艰苦工作让事情开始，用大量的锉刀来完美地完成这件作品。然后在电脑上画出一个调整好的管道草图，并创建一个纸模板。这些模板是从旧冰箱上剪下来的，用来制作主要的消声器部分。

还有很多其他的技巧——从快速肮脏的弯管到方便的板材成型技术。这也不是我们第一次看到用废料制作伟大的金属制品。休息后的视频。
