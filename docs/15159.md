# Velomobile 获得电动助力

> 原文：<https://hackaday.com/2022/10/22/velomobile-gets-electric-assist/>

当你把所有被接受的自行车设计扔出窗外，重新开始，你会得到什么？嗯，它可能看起来有点像[索基的] [velomobile](https://www.youtube.com/watch?v=YE58GWC1LKE) 。

大多数自行车都有一个相当标准的、一眼就能认出来的形状，这种形状已经流行了一个多世纪了。虽然这是对其前身一分钱自行车的巨大改进，但除了一个自命不凡的自行车竞赛标准组织说他们必须要有这种两个三角形的框架形状之外，没有理由一辆自行车*需要*有这种形状。但是，如果你想把他们完全武断的规则扔出窗外，你可以建造更高效、更快的自行车，比如横卧自行车，甚至全整流罩自行车。如果你想跑得更快，你可以随时添加一个标准的电子自行车马达套件。

这比在普通自行车上安装马达要难得多。自行车往往有标准化的零件和尺寸，而[Saukki]的 [velomobile](https://en.wikipedia.org/wiki/Velomobile) 与标准自行车相差甚远。首先，他需要为显示器和电池定制支架，他需要将电池做得特别宽，这样它的重量就不会穿透碳纤维机身。紧急制动杆电机切断需要拆除，以配合他的控制系统，最后中间驱动电机也需要一个定制的安装。这是一个 TSDZ2 电机，带有扭矩感应踏板辅助。

变化不止于此。velomobile 的最大速度远远高于标准自行车。这需要一些齿轮比的变化，在一个 60 齿的巨型链环的形式。

这导致了这种构造的一个主要问题，即 velomobile 本身可以实现如此高的速度，以至于电动辅助系统在大部分行驶过程中无法工作。欧洲大部分地区都有法律要求，电动自行车只有踏板辅助(没有油门)，超过特定速度时就会停止辅助。但是如果你想制造一辆突破法律界限而不是严格遵守法律的电动自行车，[看看这辆使用洗衣机马达的电动自行车。](https://hackaday.com/2020/07/06/the-spin-cycle-washing-machine-motor-converts-10-speed-to-e-bike/)
