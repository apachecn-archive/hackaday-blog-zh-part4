# 黑客日链接:2021 年 1 月 3 日

> 原文：<https://hackaday.com/2021/01/03/hackaday-links-january-3-2021/>

上周，我们特别报道了美国无人机识别新规则的生效。如果你错过了文章中的[，简短的故事是，几乎所有的无人驾驶飞机将很快需要传输它们的位置、高度、速度和序列号，以及操作员的位置，可能通过 WiFi 或蓝牙。](https://hackaday.com/2020/12/29/federal-aviation-administration-announces-major-drone-rule-changes/)[美国联邦航空局的规则变化并没有被 megacorporation Alphabet 的无人机送货子公司 Wing](https://www.theverge.com/2021/1/1/22209558/google-wing-faa-drone-remote-id-broadcast-rule-privacy-security) 接受。在他们看来，本地广播飞行细节将是对隐私的侵犯，因为窥探远程 ID 流量的观察者可以推断，比如说，一架无人机在药房和邻居家之间穿梭可能意味着有人生病了。他们说得有道理，但一家谷歌公司如何设法穿透厚厚的讽刺云层，抱怨隐私问题和监视状态的兴起，令人难以置信。

说到监管负担，获得业余无线电执照似乎不再像过去那样容易了。联邦通信委员会对新的业余无线电执照、执照更新和现有执照的变更，如虚荣心呼号，收取 35 美元的费用。虽然 35 美元并不便宜，但也不是世界末日，而且比 FCC 最初提议的 50 美元要好。尽管如此，对于大部分自动化的东西来说，这似乎有点高了。无论如何，看起来我们仍然可以继续我们的[、【50 美元火腿】系列](https://hackaday.com/2019/03/08/the-50-ham-getting-your-ticket-punched/)。

停留在业余无线电的话题上一分钟，看起来很快就会有一种新的数字模式被探索。这种变化将在 WSJT-X 的 2.4.0 版本发布时出现，该程序构成了 WSPR 和 FT8 等数字模式的核心。新来的叫做 Q65，基本上是目前 QRA64 弱信号模式的后续。Q65 针对 VHF 频段和更高频段的微弱、快速衰落的信号进行了优化，因此它可能会受到地球-月球-地球爱好者以及那些喜欢做像从流星尾反射信号这样的事情的人的欢迎。我们认为 Q65 应该也能让[客机反弹](https://hackaday.com/2015/12/18/bouncing-radio-off-of-airplanes/)。无论何时发行，我们都渴望尝试一下。

听着，我们知道一旦新的一年到来，很难习惯于写出正确的年份，而在《疫情时报》上，这个时间已经呈现出一种相对的感觉。但是我们很确定现在还不是四月，这是对声称游戏电脑和大众油炸食品结合的广告最合理的解释。我们强烈怀疑这只是酷派大师和百胜之间的营销噱头。品牌，但从表面上看，KF console——它不是游戏主机，它充其量是一台预建的游戏 PC——应该在你玩的时候使用多余的热量来保持你家门口肯德基的温暖。在充满难以置信的蠢事的一年里，这一件显然排在前五名。

最后，看起来我们都可以松一口气了，我们的飞行员，或者至少是他们中的一部分，没有看到任何东西。最近不断有飞行员报告进出洛杉矶，有人在喷气背包里嗡嗡作响。好吧，[有人终于捕捉到了夜魔侠](https://www.cbsnews.com/news/jetpack-flying-video-los-angeles-area/)的视频，尽管它不稳定且不清晰——看起来像所有神秘生物的视频一样——但它确实看起来像一个人类大小的两足动物站着飞来飞去。视频描述称，这是由一名飞行教练在帕洛斯贝尔德斯附近 3000 英尺(914 米)的高度拍摄的，背景是卡特琳娜岛。那离大陆大约 20 英里(32 公里)，所以无论这个人在飞什么都有惊人的范围。而且，飞行员对设备有着难以置信的信心——这是一段很长的路，以同样的滑翔比坠入一个像砖块一样的东西。
