# 这种双重挤压系统非常棒

> 原文：<https://hackaday.com/2021/03/07/this-dual-extrusion-system-rocks/>

3D 打印机的双挤压系统已经存在了好几年，但额外的成本、复杂性和打印的麻烦让它们远离了大多数黑客的工作台。来自 *Proper Printing* 的【Jón Schone】现在抛出了他自己的帽子，带有一个[定制的双挤压摇杆系统](https://www.youtube.com/watch?v=OBB5WuhNhWk)，它可以在没有任何额外致动器的情况下更换挤压机。

这两个挤压机安装在一个弹簧加载的摇杆机构上，该机构将不活动的挤压机保持向上并远离印刷表面。通过将支架移动到 x 轴的任意一端来交换挤出机，在 x 轴的一端，v 形轮滚动一个斜坡并弹出摇杆，将新的挤出机放在支架的中心线上。车厢顶部有 3 个轮子，但任何时候只有两个轮子与铁轨接触。虽然该系统比简单地并排安装两台挤出机更复杂，但它减少了不工作的喷嘴渗透到零件上或刮擦表面的机会。每个挤出机的高度可以通过一个螺钉进行调整，喷嘴之间的任何水平偏移都可以通过校准程序进行检查，并在固件中进行校正。休息后看完整视频。

[【jón】正在提供设计文件和修改后的固件](https://properprinting.pro/product/dual-extrusion-system-for-creality-printers-aka-the-rocker/)以在您自己的 Ender 3 Pro 上执行此 mod(尽管他指出其他 Creality 打印机应该是兼容的)，但您仍然需要为第二台挤出机采购带有额外步进驱动器和加热器输出的控制板。这是他在这款受欢迎的入门级打印机上进行的一长串黑客攻击中的又一个，例如[允许你将机器折叠起来并随身携带](https://hackaday.com/2019/12/26/folding-an-off-the-shelf-3d-printer/)。
