# 当市场下跌时，类似班克斯的股票追踪器会撕碎你的钱

> 原文:[https://hack aday . com/2021/09/20/banksy-like-stock-tracker-shred-your-money-when-the-markets-down/](https://hackaday.com/2021/09/20/banksy-like-stock-tracker-shreds-your-money-when-the-markets-down/)

对于任何玩股票市场的人来说，或许对投资加密货币的人来说更是如此，看着你的投资组合价值上下波动可能是一种充满压力的体验。如果你想有一个实时显示你的投资，甚至增加更多的压力，[路易斯马克思]已经满足了你。他的最新项目是一个[有机玻璃盒子](https://www.youtube.com/watch?v=cXswTe__ZQY)(德语视频)，当你的投资组合上升时，里面装满钞票，当它下降时，撕碎同样的钞票。

受一件声名狼藉的班克斯艺术品的启发，[路易斯]开始制作一个适合挂在他办公室墙上的木头和有机玻璃陈列柜。然后，他安装了一个小型碎纸机，用伺服系统进行了改造，这样它就可以由 Arduino 操作了。由于找不到现成的纸币分发器，他设计并 3D 打印了一个，由弹簧托盘和电机驱动的轮子组成。

![A plastic box that dispenses a banknote](../Images/c6c6ee7436a589dae77e67945be97a0c.png)

该项目还包括一个 Raspberry Pi，通过编程从在线来源获取市场数据，并计算[Luis]投资组合的净利润或亏损。由此产生的系统是一个相当令人不安的市场起伏的可视化:不得不清扫你地板上的绿色纸条增加了亏损痛苦的侮辱。

如果你想要一种不那么痛苦的方式来跟踪你的投资，试试这个 [Rocketship](https://hackaday.com/2021/02/11/gme-tracking-rocketship-for-the-diamond-handed-traders/) 。对于那些对传统股票报价机感兴趣的人来说，[这款基于 ESP32 的](https://hackaday.com/2021/04/26/build-an-esp32-stock-ticker-to-watch-your-gme-gains/)可能更合你的口味。
