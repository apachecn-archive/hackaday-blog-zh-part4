# 开尔文探针回顾展示 4 线电阻测量的工作原理

> 原文：<https://hackaday.com/2020/10/31/kelvin-probes-review-shows-how-4-wire-resistance-measurement-works/>

你可能会认为图中的探针只是看起来很滑稽的鳄鱼夹。但是如果你看[tomtektest]最近的视频，[你会了解到它们实际上是开尔文探测器](https://www.youtube.com/watch?v=Y5SYuXyRO2I)。开尔文探针是一种特殊类型的探针，用于使用四根导线进行精确的电阻测量，事实上，探针的钳夹是相互电绝缘的。

我们喜欢[汤姆]从他的老教练那里得到的建议:你永远也不会真正测量阻力。你在测量电压和电流。对于四线测量，一对导线将电流传输到被测设备，另一对导线测量电压降。

如果你想知道为什么这比两个探针好，这都归结于测试探针的电阻。通过具有一定电阻的探针线牵引电源电流会导致压降，从而影响测量。虽然感测线对也将具有电阻，但是感测电流可以非常小，这意味着测量中的误差将相应地更小。

我们已经看到这些探测器[也是从零开始建造的](https://hackaday.com/2020/08/28/creating-kelvin-test-leads-for-four-wire-measurments/)。你确实需要一个能进行实际四线测量的电表，尽管电源和电压表也能完成这项工作。如果你不想在现实生活中探索，你可以在[虚拟现实](https://hackaday.com/2019/06/05/circuit-vr-resistance-measurement-with-four-wires/)中进行。(嗯，反正是电路模拟器。)
