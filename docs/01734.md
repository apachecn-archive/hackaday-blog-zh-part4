# 这台机器教手语

> 原文:[https://hack aday . com/2019/01/06/this-machine-teaches-sign-language/](https://hackaday.com/2019/01/06/this-machine-teaches-sign-language/)

如果你没有沉浸其中，或者至少没有向一个流利的人学习，手语就像其他语言一样很难学。知道自己什么时候犯了小错误或遗漏了细微差别并不容易。这是一种在学习时有其独特问题的媒介，所以如果你想学习，并且没有机会接触懂这种语言的人，你可能想接触下一个最好的东西:[一台可以教你的机器](http://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2018/aac94_kd333_rp426/aac94_kd333_rp426/aac94_kd333_rp426/main.html)。

这个项目来自[Bruce Land]的三名电子和计算机工程专业的高年级学生，[Alicia]、[Raul]和[Kerry]，作为他们在康奈尔大学最后设计课的一部分。希望学习手语字母的人戴上一只手套，每只手指都装有位置传感器。该设备内部的计算机在屏幕上显示每个字母的正确符号，然后检查手套上的传感器，以确保手处于正确的位置。两个字母还包括做出手势，该设备能够通过使用陀螺仪和指南针来跟踪这一点，以确保字母已经正确签名。它似乎只涵盖了字母表，而不是更广泛的词汇，但作为概念的证明，它是非常有效的。

学生们表明，使用这种机器作为教学工具来可靠地学习字母表是完全可能的。这种类型的技术也可以用于其他应用，例如人机界面设备的手势识别。如果你想看到更多这些有趣的和有参考价值的高级设计，我们已经精选了不少，从[测谎仪](https://hackaday.com/2017/01/03/retro-style-diy-polygraph-believe-it-or-not/)到[自行车声纳系统](https://hackaday.com/2017/01/11/this-bike-sonar-is-off-the-chain/)。
