# 在这个由步枪外壳制成的电子箔上拍摄海浪

> 原文：<https://hackaday.com/2021/04/07/shoot-above-the-waves-on-this-e-foil-made-from-a-rifle-case/>

所以你说你想乘坐电动水翼艇在海浪之上飞行，但你没有办法购买商业板。或者，你没有时间和技能来雕刻一块木板，并为它配备马达和翅膀，让它在水面上滑行。你运气不好吗？如果你按照这个有黑客价值的电子箔制作使用防水步枪外壳作为…外壳的话，就不会了？董事会？不管怎样，漂浮的部分。

如果你以前没有遇到过电子箔，准备好突然需要一些你从来不知道存在的东西。e-foil 基本上是一个冲浪板，带有一个强大的无刷电机，安装在各种龙骨上，离吃水线相当远。随着发动机是一个水翼，以提供升力，足以提高董事会出水面的董事会获得速度。它们看起来很有趣。

大多数电子箔是围绕相当于冲浪板的东西建造的，带有容纳电池、电机控制器和其他电子设备的隔间。[弗兰克]和[朱利安]通过购买一个防水的步枪外壳来解决困难的冲浪板制造问题。它可能不是很有流体动力，但它的形状因素是正确的，它已经可以漂浮，而且它有足够的空间容纳电子设备。上面的链接有很多关于建造的细节，从用铝制内骨架加固外壳开始，但在一天结束时，他们只花了大约 2000€，大部分是现成的零件。下面的视频展示了步枪外壳的首航；我们惊讶地看到，当步枪外壳离开水时，马达使用的功率下降得多远多快。

与我们见过的一些电子箔片相比，这个看起来更像快照。向[Frank]和[Julian]致敬，他们找到了一种方法，让这成为另一种我们负担得起但却没有时间去做的爱好。
