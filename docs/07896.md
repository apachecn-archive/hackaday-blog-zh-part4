# 阴影灯故意缓慢移动

> 原文：<https://hackaday.com/2020/10/02/shadow-lamp-moves-delibrately-slowly/>

受到著名的熔岩灯的启发，[mojaptix]想要打造一个自己的作品，具有类似的有机、变化的灯光效果。然而，他没有让加热的蜡流动，而是用自己的方式创造了一盏具有伪随机效果的灯。

该灯本身是围绕皮影戏概念构建的，使用一对[mojaptix]3D 打印的旋转图形。这些图案一前一后旋转，由宜家 LED 灯从后面照亮。随着光圈的旋转，它们为 LED 发出的光提供了缓慢变化的路径，这些光被投射到组件前面的纸屏幕上。为了产生长周期旋转，旋转组件由普通时钟机芯的分针转动。这是一个很好的方式来获得一个低成本的慢速旋转电机和变速箱设置，只要你的扭矩要求绝对微乎其微。

这是一种产生缓慢变化的灯光效果的巧妙方法；我们以前也报道过关于这个话题的其他讨论。休息后的视频。
