# 因为传统的激光竖琴不够危险

> 原文:[https://hack aday . com/2019/09/10/因为-常规-激光-竖琴-不-危险-够了/](https://hackaday.com/2019/09/10/because-conventional-laser-harps-arent-dangerous-enough/)

20 世纪 80 年代末，法国音乐家让·米歇尔·雅尔带着一把激光竖琴巡回演出。十几岁的粉丝中流传着这样的话:他必须戴上特殊的手套来防止他的手被烧伤，因为 20 世纪 80 年代的青少年既易受影响又不太了解激光。事实上，我们社区的一名成员告诉我们，他是道路工作人员的一部分，手套是反射性的问题，所以激光竖琴仍然相对无害，法国竖琴手保留了他们的手指。为了给激光竖琴体验增加一点趣味，[詹姆斯·科克伦] [将激光测距仪连接到特斯拉线圈](https://www.youtube.com/watch?v=ohGsdv1wRZk)上，使乐器在性能上比[贾雷]模型有更多的爆裂能量。

它从一个激光卷尺[开始，该卷尺被修改为 Arduino 测距仪](https://www.hackster.io/iliasam/making-a-cheap-laser-rangefinder-for-arduino-4dd849)，与[定制的 MIDI 代码](https://github.com/squirtifier/X-40-Laser-Rangefinder-aMIDI-Controller)耦合，以制作一个激光竖琴 MIDI 控制器。讨论中的特斯拉线圈碰巧也是一种 MIDI 乐器，所以一个可以轻松控制另一个。增加了一个接地的锁子甲手套，使它可以在靠近线圈的地方玩，他奖励我们一个《T4》星际迷航主题的表演。特斯拉乐趣和游戏在我们身后，然后他给我们演示了一个更传统的 MIDI 乐器。

这些年来，我们在这里有无数个特斯拉线圈项目，如果你渴望更多，我们建议从这个不寻常的[平面 PCB 线圈设计](https://hackaday.com/2019/04/25/a-tesla-coil-from-pcbs/)开始。与此同时，你可以在视频中看到激光竖琴线圈下方的休息。
