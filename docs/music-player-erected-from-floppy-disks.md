# 用软盘搭建的音乐播放器

> 原文：<https://hackaday.com/2020/02/07/music-player-erected-from-floppy-disks/>

因为怀旧而保留着一堆旧软盘？我们也是，这可能是他们最酷的用途。虽然想象他写了一个压缩算法来在 1.44Mb 的杯垫上无损复制 Coltrane 的 Blue Train 很有趣，或者以某种方式滚动他自己的迷你光盘，[【迪诺·菲佐蒂】的 Diskplayer 使用软盘来提供 Spotify 专辑](https://www.dinofizzotti.com/blog/2020-02-05-diskplayer-using-3.5-floppy-disks-to-play-albums-on-spotify/)。

软盘上到底有什么？对应的 Spotify 相册 URL。他只需在驱动器中弹出一张磁盘，Pi 就会完成剩下的工作——它会检测软盘事件并执行一个脚本来启动开源 Spotify 客户端。没有曲目跳跃和无序播放，只是整张专辑，要么接受，要么退出。如果你仔细想想，他实际上已经设法改善了黑胶唱片的体验，因为所有的歌曲都在一面。演示在休息后排队等候，它包括[Dino]的简单网络接口用于写软盘。

当这个项目在 7 个月前开始时，[Dino]打算用 RFID 标签将他的乙烯基收藏带入 21 世纪，但我们很高兴他决定使用一种相当过时的媒介。没有一个驱动器或一堆软盘在壁橱里积灰？[迪诺]也没有。但是他发现很多人在 ebay 上卖漂亮的彩色软盘，亚马逊有大量便宜的外置硬盘。我们认为专辑封面贴纸是一个很好的接触，因为是匹配专辑封面的颜色，以软盘。他把那些坏男孩关起来是对的。

有一堆软驱吗？制作一个电子琴，制作你自己的音乐。
