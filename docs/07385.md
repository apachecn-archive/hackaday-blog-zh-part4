# 制造超导体

> 原文:[https://hack aday . com/2020/08/04/nilered-makes-超导体/](https://hackaday.com/2020/08/04/nilered-makes-superconductors/)

我们总是喜欢[尼尔勒的]视频。他的最新作品展示了他是如何制造出一些相对高温的超导陶瓷的。在互联网上找到一些看起来很好的指导后，[NileRed]发现论文中有些东西没有意义。可以看视频，下面。

超导体是 YBCO，由于其成分的比例，有时被称为 123。结果大多数材料都可以在网上买到，除了一种他不得不从更传统的来源购买的外来化学物质。

一些化学物质的纯度有疑问，所以他提纯了一些，并在其中发现了一些污染物。如果你想自己动手，你需要一些化学处理工具，我们建议你阅读所有相关材料的安全说明。

如果你没有工具，你将不得不安排真空室、8 吨压力机、熔炉和更普通的东西，如搅拌热板和实验室玻璃器皿。你甚至可能需要一个搅拌机。

当然，你还需要处理液氮，让东西冷却到足以看到超导效应。[NileRed]实际上必须为这项工作购买一个炉子，花费大约 2000 美元，并且正如您将看到的那样，要将它制成一个整体并以两倍的额定电压撞击它是一个挑战。

即使我们可能不需要修理熔炉，我们也怀疑我们会在短期内制造出超导体。但令人欣慰的是，如果你愿意，你可以在车库里做任何事情。

我们已经讨论了一些[NileRed]找到灵感的[项目](https://hackaday.com/2018/02/28/cook-up-your-own-high-temperature-superconductors/)。虽然我们对这个过程印象深刻，但我们不确定这种材料是否能够制作出[实用的悬浮滑板](https://hackaday.com/2015/08/10/diy-superconducting-hoverboard/)。
