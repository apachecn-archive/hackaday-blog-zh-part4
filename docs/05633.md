# 一个廉价和容易的 GoPro 安装模型火箭

> 原文:[https://hacka day . com/2020/02/16/a-廉价且简单的 go pro-mount-for-model-rocket ry/](https://hackaday.com/2020/02/16/a-cheap-and-easy-gopro-mount-for-model-rocketry/)

发射模型火箭很有趣，但这项爱好的实质在于你下一步做什么。一些人选择给他们的火箭安装仪器或携带其他先进的有效载荷。[seamster]喜欢拍摄他的飞行，并为此制作了一个鼻锥相机包。

GoPro 是[seamster]执行任务的首选相机，其动作摄像头设计使得只需按一下按钮即可轻松发射。为了将它安装在火箭上，头锥被设计成几个部分。顶部和底部是 3D 打印的，与从汽水瓶切割的透明塑料圆柱体相匹配。在圆柱体内部，GoPro 和高度计硬件用泡沫块固定在适当的位置，泡沫块是从旧的地垫切割而成的。火箭的降落伞连接在头锥的顶部，这使得相机在飞行的上升和下降阶段都可以悬挂在正确的方向上。休息之后，看看用这个设置创建的高飞视频。

这是一个简单的设计，[裁缝]能够在短短几个小时内在 Tinkercad 中快速完成，并且一个普通的制造者可以在家里轻松复制的设计。拍摄飞行从未如此简单—[从 20 世纪 70 年代开始，我们已经走过了漫长的道路。](https://hackaday.com/2012/07/12/launching-rockets-and-recording-video/)
