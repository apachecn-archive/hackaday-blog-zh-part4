# 机器人用红外线灭火

> 原文：<https://hackaday.com/2020/03/08/robot-fights-fire-with-ir/>

用机器人救火可能会夺走人类的工作，但也能拯救生命。[梅尔·贝尔电子公司]制造了一个(受监督的)[儿童友好型消防机器人](http://www.instructables.com/id/Firefighting-Robot-Using-PICO/)，它通过追逐火焰并将其吹灭来灭火。

这个高度警惕的机器人总是在寻找火灾，不会在其他任何事情上浪费时间。一旦它检测到火焰的存在，它就将自己定位在火源上，并加速通过由螺旋桨和 DC 发动机组成的风扇将其扑灭。

科学来了:火发出红外线，业余爱好者的火焰传感器使用红外线来探测火。这个消防机器人的前端有三个这样的火焰传感器，将数字数据输出到世界上最小的 Arduino——基于 ATmega32U4 的 PICO 板，而[Mell Bell]恰好出售这种板。捂住你的嘴和鼻子，沿着地板向裂口爬去，看看这个东西有多灵敏。

消防员并不是唯一参与保护森林的勇敢的人，也不是唯一感受到自动化压力的人。Hackaday 自己的[汤姆·纳尔迪]写了一篇关于一个叫“火眼金睛”的濒临灭绝的物种的文章，无疑会点燃你的兴趣。
