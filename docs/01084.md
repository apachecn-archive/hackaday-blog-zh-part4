# 这遥控堡垒之夜火箭是一个皇家胜利

> 原文：<https://hackaday.com/2018/10/27/this-rc-fortnite-rocket-is-a-victory-royale/>

《《我的世界》》已经结束，而《红色死亡救赎 2》还没有开始。现在这个星球上最大的东西是堡垒之夜，哦，天哪，是不是太棒了。这是有史以来最好的比赛，我们希望每一天都是皇家战役。但是如果堡垒之夜是真实的生活呢？这正是[Giaco]和[David]在制造 RC 堡垒之夜泡沫火箭时[所做的。这是堡垒之夜，在现实世界中！如果你不介意，我们要去角落里摆个姿势。](https://www.youtube.com/watch?v=unKHPNmVEY8)

这个构建的核心是用[制作刀](https://www.kickstarter.com/projects/1496249101/maker-knife-the-edc-pocket-tool-you-want-to-carry)完成的，最初是由[Giaco]作为 Kickstarter 推出的，作为一种日常携带的实用刀，具有陶瓷刀片。这对于一个美工来说令人印象深刻，但更令人印象深刻的是，这个神奇的工具可以用来制作现实生活中的堡垒之夜火箭。

这种火箭，就像我们这些天看到的许多其他遥控飞机一样，是由泡沫板建造的，这种技术是由 Flite Test 的人们推广的，并且使用了你通常在泡沫板模型飞机中看到的所有建造技术。控制表面的铰链被倒角并用包装带加固，伺服系统只是热粘在车身上，控制喇叭只是一些纸板。

令人印象深刻的是这个堡垒之夜火箭实际上是在 T1 飞行。[Giaco]带着[David]的 rcexplorer 名声把这架飞机带了出去，尽管这架“飞机”实际上没有任何升力面，尽管重心不明显，而且事实上油漆比飞机本身还重，但它能飞。也很好，直到它卡在树上。我猜，制作对 12 岁孩子有吸引力的内容是要付出代价的。
