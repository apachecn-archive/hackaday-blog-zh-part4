# 增加钢丝滚道改善了 3D 打印轴承

> 原文:[https://hack aday . com/2021/12/13/adding-wire-races-improves-3d-printed-bearings/](https://hackaday.com/2021/12/13/adding-wire-races-improves-3d-printed-bearings/)

像许多动力传输部件一样，轴承比以前更容易获得。过去很难找到您到底需要什么，但现在只需点击几下鼠标就能找到高质量的轴承。但是它们并不总是便宜的，尤其是当你得到更大的尺寸时，所以知道如何打印自己的轴承会是一项有用的技能。

当然，3D 打印的轴承并不能在所有的应用中发挥作用，但是[Eros Nicolau]对此有一个计划。他使用钢丝环作为磨损表面，而不是冒着在塑料跑道上运行塑料或金属球的摩擦热损坏的风险。下面的第一个视频显示了轴承的早期版本，其中一对钢丝环排列在 3D 打印的内圈和外圈上。这些工作还可以，但遭受偶尔粘点，有点在噪音方面。

第二个视频显示了版本二，它使用了相同的钢丝环比赛安排，但增加了一个印刷球笼来限制球。这使事情更安静，并消除绑定，使轴承运行更顺畅。[Eros]还在轴承中加入了一点润滑剂，以液体聚四氟乙烯的形式，更好的说法是聚四氟乙烯。这看起来确实让事情变得顺利了。我们会想象聚四氟乙烯会比石油基润滑脂更适合大多数印刷塑料，但我们很想看看轴承长期使用的情况。

也许你还记得以前在这里见过大型 3D 打印轴承？你是对的。如果你需要了解更多关于[轴承如何工作](https://hackaday.com/2018/02/14/mechanisms-bearings/)——或者[润滑油](https://hackaday.com/2016/05/17/beyond-wd-40-lubes-for-the-home-shop/)，我们会帮你搞定。
