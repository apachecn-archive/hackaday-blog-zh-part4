# 用旋转工具和烙铁解决买家的后悔

> 原文:[https://hacka day . com/2020/06/09/solving-buyers-remour with-a-rotary-tool-and-soldering-iron/](https://hackaday.com/2020/06/09/solving-buyers-remorse-with-a-rotary-tool-and-soldering-iron/)

在这一点上，很明显 USB-C 已经成为越来越多应用的新标准连接器，但主要是充电。这次甚至苹果也加入了进来，由于向后兼容，你不必放弃使用旧标准的设备，因为它们可能更简单或更有弹性。对于[Mat]另一方面，现在已经是 USB-C 一路了。然而，当他买笔记本电脑的时候，价格标签说服了他，他开始后悔，因为超薄设计的所有便利都被笔记本电脑专有充电端口的笨重充电器抵消了。

嗯，俗话说，对于这种情况:喜欢它，离开它，或者[拿出工具，返工那个笨蛋](https://notenoughtech.com/featured/best-usb-c-hack-ever/)。幸运的是，原装充电器提供 20 V 电压，非常符合 USB 电源传输(PD)规范，打开笔记本电脑后，[Mat]很高兴地看到内部提供了足够的空间来安装他打算使用的 USB-C 模块。更好的是，充电端口本身是一个连接到电缆的独立组件，因此没有必要对主板进行修改。一旦 USB-C 模块被焊接到同一根电缆上，剩下唯一要做的事情就是在笔记本电脑外壳上挖一个更大的洞，并对过时的充电器说再见。

缺点当然是缺乏实际的 USB 功能和闪亮的新充电端口，但这从来不是这里的目标。随着越来越多的 USB-C 设备的出现，我们以前看到过类似于的[修改也就不足为奇了，而且](https://hackaday.com/2019/05/07/add-usb-c-to-your-laptop-almost/)[不仅仅是笔记本电脑](https://hackaday.com/2019/06/17/wireless-mouse-power-up-logitech-mx-master-gets-usb-c-and-big-battery/)。如果你想把你自己的一个设备升级到 USB-C，并且*希望得到真正的 USB 功能，[别担心，我们也会为你提供服务](https://hackaday.com/2020/05/12/neat-and-tidy-usb-c-conversions-for-legacy-devices/)。*
