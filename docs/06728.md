# 一个自制的翻斗车，你会发誓来自一家工厂

> 原文:[https://hackaday . com/2020/06/01/a-home-made-dumper-youd-sweet-from-a-factory/](https://hackaday.com/2020/06/01/a-home-made-dumper-youd-swear-came-from-a-factory/)

当谈到 YouTube 视频时，我们最喜欢的莫过于一些高质量的车间活动，看着一些人掌握着他们的工具，用原材料制造出一台机器，并取得令人惊叹的结果。[这是【从零开始的车间】用他自制的迷你翻斗车](https://www.youtube.com/watch?v=tvsxwDl5HPo)交付的东西，他从零开始制造一辆小型翻斗车，结果看起来就像是从他的农业供应商那里买来的工厂制造的。

它的核心是一个由焊接在一起的双箱形截面管制成的坚固底盘，他用螺栓将一个二手液压传动装置固定在底盘上，这种传动装置是你会在大型手扶式场地管理机械上找到的类型。在后面是一个来自移动滑板车的前转向轴，它以福特蒙迪欧的轴承和轮毂为枢轴，以确保在崎岖地面上的稳定性。当小型本田四冲程发动机驱动它时，操作员可以站在一个平台上。水桶是等离子切割焊接的，可以肯定的说他的焊接能力超过了我们。

结果是一台看起来非常有用的机器，我们敢承认，我们不介意试一试。[可能没有这辆电动自制自卸车](https://hackaday.com/2019/08/13/electric-dump-truck-tricycle-is-no-toy/)那么强大，但是我们喜欢。
