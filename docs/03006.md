# Sega Genesis 的 MIDI 合成器

> 原文：<https://hackaday.com/2019/05/15/midi-synthesizer-from-a-sega-genesis/>

[Aidan]对用于创建音频的 FM 合成芯片非常感兴趣，在 Sega Genesis 上发现了那个时代最有趣的芯片之一。任何参与当时游戏机战争的人肯定都记得那些视频游戏系统能够产生的经典、独特的声音，所以[Aidan]使用 Genesis 的声音芯片建造了一个设备，可以播放任何游戏的任何音乐。然而，该项目的第二次迭代能够将这些相同的声音文件用作 MIDI 合成器。

这些芯片的有趣之处在于它们如何使用寄存器来改变音频输出。本质上，有一个复杂的音域映射(他的文章中有一部分被简单地称为“音域地狱”)，可以调用它来访问合成器上常见的各种效果。这一点也不简单，一旦[Aidan]开始添加 MIDI 功能，事情就变得更加复杂了。然而，一旦他完成了对世嘉创世纪技术手册和一堆寄存器的筛选，他就有了一个独特的合成器，它听起来不像你听过的任何东西，除非你曾经玩过创世纪。

如果你想看看他的第一个项目，MegaBlaster，它直接播放旧的 Genesis 游戏的声音文件，[我们不久前曾专题介绍过这个项目](https://hackaday.com/2018/10/04/we-got-your-sega-chiptunes-right-here/)。请记住，尽管他的最新项目不仅仅是一个更新的 MegaBlaster。他从头开始建造了这一切。
