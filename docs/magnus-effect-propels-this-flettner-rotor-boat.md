# 马格纳斯效应推动了这艘弗莱特纳转子船

> 原文：<https://hackaday.com/2021/11/27/magnus-effect-propels-this-flettner-rotor-boat/>

马格努斯效应是一个有趣而有用的现象。来自“空气计划”的詹姆斯·沃姆斯利决定把它用在一艘小型无线电控制的船上，成功地利用了这种效果。(视频，插播后断。)

马格努斯效应是一件有趣的事情，流体流过旋转物体时产生一个与流动方向和旋转轴成直角的空气动力。(这就是为什么弧线球会弯曲。)通过旋转一个叫做弗莱特纳转子的高圆柱体，这可以用于船上的推进。这利用了马格努斯效应来产生推力。

这艘船使用 3D 打印的船体，用泄漏密封喷雾和大量喷漆密封，以避免泄漏。在双体船设计的中心，有一个由无刷电机驱动的旋转转子。在用于推力的转子外部，一个简单的方向舵用于转向。

随着转子转动，船只能够借助风力产生的推力成功航行。然而，有一个初期问题，强风很快掀翻了船。[詹姆斯]意识到添加一些适当的龙骨将有助于避免船倾覆。

我们以前在这些地方见过[詹姆斯]，也就是在这个建造之前的马格努斯效应飞行器。
