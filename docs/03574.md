# 数控设计的新方法

> 原文：<https://hackaday.com/2019/07/17/a-fruity-approach-to-cnc-design/>

[弗兰克·豪沃思]发现他的餐厅需要一盏灯。作为制造商的信念，商店购买根本不行。在一家严肃的木材店的支配下，[【Frank】去了趟超市寻找灵感](https://www.youtube.com/watch?v=C3dCSzBeF9E)。

在为他的项目挑选了一颗形状特别好的杨桃后，[Frank]不想花费过多的时间试图在建模软件中重新创建有机的肿块，而是使用[mesh room](https://alicevision.org/)通过摄影测量来创建一个模型。在几次失败的尝试后，成功是通过使用一个有纹理的旋转桌子作为背景，用哑光灰色画杨桃，最后撒上黑色斑点。这给了软件足够的视觉线索来精确地模拟水果的几何形状。

有了一个 3D 模型，Fusion Slicer 就可以用来生成一个可以用平面激光切割片构建的模型。然后生成切割轮廓，并传递给 Rhino 进行最终调整。一切准备就绪后，从胶合板上切下零件，一个潜在灯具设计的小模型就制作出来了。[Frank]在最终建造之前，目前正在与餐厅的居民讨论设计。

摄影测量和现代 CAD 工具使得处理自然形态变得快速而简单。我们也看到这项技术被用于其他目的，Eric Strebel 提供了一个很好的例子来说明如何将它用于逆向工程。
