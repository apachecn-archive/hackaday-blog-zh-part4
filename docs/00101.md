# 对象检测，使用张量流

> 原文：<https://hackaday.com/2018/07/31/object-detection-with-tensorflow/>

让计算机识别物体一直是计算机科学中的一个历史难题，但随着机器学习的兴起，它变得更容易解决。[物体识别中可以使用的工具之一](https://github.com/EdjeElectronics/TensorFlow-Object-Detection-on-the-Raspberry-Pi)是一个名为 TensorFlow 的开源库，[Evan]又名[Edje Electronics]就是为了这个目的而开发的。

他的物体识别软件运行在一个配有摄像头的 Raspberry Pi 上，还利用了 Open CV。[Evan]指出，这为 Pi 开辟了许多创造性的低成本检测应用，如设置一个摄像头，检测宠物何时在门口等待被放入或放出，[计算进出蜂箱的蜜蜂数量](https://hackaday.com/2018/05/30/counting-bees-with-a-raspberry-pi/)，或监控办公室的停车位。

这个项目还使用了许多其他的工具包，包括 Protobuf。它还大量使用了 Python 脚本，但是如果你对此感到满意，并且你有一个计算机视觉的应用程序，[Evan]的教程将带你入门。
