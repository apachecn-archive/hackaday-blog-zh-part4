# 烙铁加摄像头万向节有助于抵消黑客的手抖

> 原文：<https://hackaday.com/2021/06/04/soldering-iron-plus-camera-gimbal-helps-cancel-out-hackers-hand-tremors/>

焊接需要稳定的双手，所以当乔纳森·格利希不幸患上了一种叫做“特发性震颤”的疾病，影响到他的双手时，焊接变得更加困难。但是有一天，当[乔纳森]和一个朋友聊天时，他们被好主意仙女拜访了，在真正的黑客时尚中，他结束了[重新使用手持相机稳定万向节来固定烙铁](https://www.instructables.com/Solder-Station-Stabilizer/)而不是相机或智能手机。现在，万向架不是抵消手的运动来保持相机稳定，而是帮助保持烙铁稳定。

虽然廉价的万向架单元的内部工作不需要修改，但在项目开始之前还有一些事情需要做。第一个是建立一种方法，快速方便地连接和断开烙铁与万向节。由于燕尾形连接器，熨斗可以安全地存放在常规皮套中，仅在需要时连接。

另一个修改更微妙。稳定器马达预计将管理类似智能手机的东西，但烙铁更轻，平衡能力也不同。这意味着该系统工作正常，但还没有达到需要的水平。在使用一些小铅块调整烙铁的质量和重心后——就万向节而言，使其感觉和移动更像 iPhone——结果得到了改善。

烙铁稳定器目前工作得足够好，但我们不怀疑[乔纳森]已经有了进一步的调整。这是一个将消费类设备转变为辅助设备的绝妙方法，请观看下面的视频短片。
