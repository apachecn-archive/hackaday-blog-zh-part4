# 建造一个乐高碎纸机

> 原文：<https://hackaday.com/2022/03/11/building-a-lego-paper-shredder/>

出于安全原因，有时我们需要在扔掉文件之前销毁它们，碎纸机是实现这一目标的主要方式。如果你没有自己的，你可以考虑自己做一个，就像[砖块实验频道]用乐高做的那样。

第一次尝试用互锁的齿轮粉碎一张小纸条失败了，只是以一种吸引人的波纹方式揉皱了纸张。正如“绝密”纸上所说，“如果你能看到这个，那就是碎纸机坏掉了。”添加更多的齿轮设法在纸上凿出几个洞，但仍然远远没有效果。继续沿着这条路走下去只会让乐高马达停滞不前。

使用不同尺寸齿轮的重新设计最终成功地将纸张撕成了大块。一组齿轮会抓住纸张，而另一组会撕开一部分。进一步的改进将这种方法与使用锥齿轮作为一种刀片相结合，进一步提高了切碎性能，达到了将纸张撕成令人满意的微小碎片的程度。

这是一个有趣的小构建，即使它不会接近一整页 A4 纸。这是一个很好的例子，当你设定一个简单的目标，并有容易衡量的结果，在这种情况下，纸上原始信息的易读性。
