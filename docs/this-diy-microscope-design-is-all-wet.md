# 这个 DIY 显微镜设计大错特错

> 原文：<https://hackaday.com/2022/01/14/this-diy-microscope-design-is-all-wet/>

罗伯特·默里-史密斯想要重现一些古代显微镜的工作方式:用一滴水作为透镜。这个想法是，一滴水的弯月面将作为一个透镜。这是因为表面张力，通过控制水对表面的吸引力，你实际上可以形成凸面和凹面。

有趣的是，这并不需要很多设备。一些塑料，一个打孔机，一些钢笔，一个手电筒，和一些其他零碎东西。接下来就是抓一些水坑里的水，检查里面的生物。当然，对于单透镜，这些是更合适的放大镜。有人声称中国人在几千年前就制造了这种仪器。[罗伯特]提到[安东尼·范·列文虎克]是显微镜之父，尽管他不是第一个制造这种设备的人。他确实用一种他保密的方法创造了令人惊叹的玻璃镜片，但这种方法是用现代科学研究出来的。

通过相机很难看到太多，但它显然是放大了。这是一个不错的小雨天孩子的项目，因为你手头可能有你需要的一切。我们想知道你能用这样的设备想象出其他什么现成的东西。

当然，如果你想建造一个真正的显微镜，设计就在那里。你甚至可以用[乐高](https://hackaday.com/2021/06/28/lego-microscope-aims-to-discover-future-scientists/)做一个。
