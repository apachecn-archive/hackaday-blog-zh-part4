# 树莓派的预算时间

> 原文：<https://hackaday.com/2020/10/02/bullet-time-on-a-budget-with-the-raspberry-pi/>

《子弹时间》在《黑客帝国》( 1999)中一炮而红，成为最热门的新电影特效。当时，尖端特效需要强大的硬件和强大的处理能力来完成这项工作。当然，这些天来，事情有所进展。【埃里克·巴累】对高端系统[并不陌生，但他想看看低端市场能做些什么。](https://www.youtube.com/watch?v=YIQP61xfFzA)(视频，下面嵌入。)

与其依赖一堆昂贵的数码单反相机，【Eric】决定尝试用 15 台 Raspberry Pi 和标准的 Raspberry Pi 相机构建一个子弹时间相机装备。然而，在他的专业设置中，仅仅一台相机的成本就可能超过 1000 美元，而整个钻机的成本可能低于这个数字。

最初的结果不稳定且不吸引人，但[埃里克]坚持下来了。最大的问题之一是相机组件的不准确性，因为它们是用导热膏粘上去的。通过一些定制的修改和调整，[Eric]最终能够让事情达到一个可接受的状态。与 DSLR 钻机相比，它还有一个好处，即由于尺寸小，摄像机可以安装得更近。

我们已经在着手将钻机升级到新的 Raspberry Pi HQ 摄像机，这是我们之前讨论过的。
