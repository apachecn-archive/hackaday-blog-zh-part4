# 再利用塑料保护多氯联苯

> 原文：<https://hackaday.com/2019/06/06/repurposed-plastic-protects-pcbs/>

一根错误的电线剪断了错误的电插脚，就会释放出你的魔法烟雾。即使你很幸运，散落的零件也是无数故障的根源，从破坏性的到致命的。[TheRainHarvester]分享了他用我们大多数人扔在回收箱里的废塑料覆盖 Arduino Nano 的技巧。视频也是休息之后。他称之为盆栽，但我们认为这是一个定制的封面。

诀窍是从食物容器盖上切下一点塑料，通常是高密度聚乙烯或塑料 2 号。修剪一块比你没有保护的木板大一点的木板，找个方法把它固定住，这样你就可以用热风枪炸开它了。当我们在我们的一个 Hackaday 远程实验室尝试这种方法，并在电路板和一些绿色塑料之间涂上少量热胶时，效果很好。视频建议使用金属夹具，这在制作多个夹具时是合乎逻辑的。YouTube 评论者和提示提交者[Keith o]建议使用真空成型器来实现更紧密的贴合，我们不介意看到定制的窗口切口来访问关键的电路板部分，如 DIP 开关或修剪器。

我们理解为什么电线短路是一个问题，特别是当你[菊花链三个电源](https://hackaday.com/2018/12/10/laptop-chargers-team-up-to-get-the-juice-flowing/)时，就像[TheRainHarvester]以前的一个视频中发生的那样。
