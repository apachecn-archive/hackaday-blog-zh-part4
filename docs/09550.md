# 时钟的时钟增加了美观和功能的光管指针

> 原文:[https://hackaday . com/2021/03/19/clock-of-clocks-adds-light-pipe-hands-for-beauty and-function/](https://hackaday.com/2021/03/19/clock-of-clocks-adds-light-pipe-hands-for-beauty-and-function/)

我们已经习惯于看到“元时钟”，这种时钟使用一系列模拟钟面，并用时钟的指针拼凑字符。他们非常聪明，我们总是喜欢看到他们，尤其是当他们有详细的构建指令时，就像这个一样。

[Erich Styger]的“MetaClockClock”显示的另一个优点是对原始概念的扭曲。大多数时钟的时钟依赖于模拟运动的手和脸之间的对比，[Erich]增加了混合的光线。隐藏在每个时钟表圈里面的是一条 RGB 发光二极管；再加上时钟的透明丙烯酸指针，这些指针可以充当光导管，每个时钟都可以为显示器贡献不同的形状和不同的颜色。每个时钟都是围绕一个双轴步进电机构建的，这种电机用于汽车仪表板仪表；每个电机都安装在定制的 PCB 上，而 led 则安装在自己的环形 PCB 上。二十四个时钟安装在一个非常漂亮的胡桃木面板上，与光管指针配合得非常好。下面的视频展示了一些可能的展示方式。

[Erich]非常详细地记录了他的构建过程，[在 GitHub](https://github.com/ErichStyger/MetaClockClock) 上有所有的设计文件。我们不会说重建他的建筑会很容易——这里需要很多技能，从电子到木工——但至少所有的信息都在那里。我们认为这是对[【埃里希】早期版本](https://hackaday.com/2020/01/01/building-a-giant-meta-clock-made-of-smaller-clocks/)的一次漂亮升级，我们希望看到更多这样的版本。
