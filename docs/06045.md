# 一个非常垃圾的项目

> 原文:[https://hackaday.com/2020/03/28/a-really-garbage-project/](https://hackaday.com/2020/03/28/a-really-garbage-project/)

不管你是谁，你都会制造这样或那样的垃圾。两名学生决定，他们想创造一个智能垃圾桶，当垃圾桶满了，甚至有臭味时，它会提醒他们。

我们将继续记录:我们不知道酒精传感器可以告诉你的垃圾是否是臭的，所以如果这行得通，那是我们的一个新发现。然而，它有某种意义，因为垃圾发酵。我们认为垃圾有气味是因为硫化氢和甲烷。

垃圾桶有一个艰难的生活，所以如果你真的想复制这一点，你可能会想安装的东西更安全一点。然而，该软件通过云服务运行一切，从那里可以使用手机应用程序 Blynk 和 IFTTT 将东西发送到电子表格，如果你想跟踪你的垃圾历史统计数据。

你可以在视频中看到这是一个小型的概念验证罐。我们几乎想建造一个，只是为了看看传感器是否真的知道罐头什么时候有气味。

诚然，该项目可能不是最实用的，但令人惊讶的是，由于有大量廉价的电路板和服务可用，现在构建具有高度连接性的设备是多么容易。

这不是我们第一次看到 [Blynk](https://hackaday.com/2016/12/02/blynk-with-joy/) 。就此而言，它甚至不是我们的第一个[智能垃圾桶](https://hackaday.com/2016/03/17/presenting-the-internet-of-trash-cans/)。
