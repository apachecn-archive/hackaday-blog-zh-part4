# 给自己造一对发光的翅膀

> 原文：<https://hackaday.com/2019/10/21/build-yourself-a-set-of-glowing-wings/>

人类并不是从工厂里就带着翅膀来的，而且事后大部分安装翅膀的努力至少在飞行方面是徒劳的。然而，这并不意味着你不能切割出极具破坏性的美学效果，[而【娜塔莉娜的】光纤翅膀就是一种很好的方式来做到这一点。](https://www.instructables.com/id/Fiber-Optic-Wings/)

翅膀是一个皮革制品项目，由缠在躯干上的背带组成。这是光纤的安装点，也是用来驱动灯光表演的 RGB 生物手电筒的安装点。皮革部分通过激光切割成合适的形状，从而在设计中轻松打造精致的羽毛形状。然后将这些部分适当染色，并缝合成最终的形状。然后，光纤束缠绕在背带上，从佩戴者的任一肩部伸出。EVA 泡沫用于帮助创建翅膀的正确形状，允许不同的层保持分离，以创造更多的视觉深度。

这是一个在晚上看起来绝对引人注目的建筑，不像其他基于翅膀的角色扮演，没有那么多的缺点，因为人群和交通。如果你今年万圣节要扮成黑客天使，那会让你看起来像个杀手。
