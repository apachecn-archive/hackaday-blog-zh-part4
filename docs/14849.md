# 3D 打印的大理石跑道具有整洁的电梯联动装置

> 原文:[https://hack aday . com/2022/09/21/a-3d-printed-marble-run-features-neat-elevator-linkage/](https://hackaday.com/2022/09/21/a-3d-printed-marble-run-features-neat-elevator-linkage/)

很少有什么能像简单的弹珠赛跑那样令人愉悦和放松。当然，让弹珠在重力作用下落下的原因是，你最终需要把它们重新举起来。[提肌有这样一种机制。](https://www.instructables.com/Marblevator-Pick-and-Place/)

总的来说，建筑的特点是相对简单的大理石运行。它由六个 3D 打印的斜坡组成，大理石在几秒钟内就可以滚下来。然而，真正神奇的是将弹珠从底部一直恢复到顶部的机制。

一个马达转动一个齿轮，然后齿轮转动一个曲柄，导致一个多连杆菱形。菱形的一个角上有一个小突起，上面有一块磁铁，可以从底部吸起弹珠。当机械装置转动时，菱形块移动，将弹球运送臂带到弹球运行的顶部。在那里，它被另一个磁铁吸住，磁铁在弹球落下之前会吸住它一会儿。

这是一个简单的项目，但仍然可以成为一个出色的桌面玩具。这也是学习连锁分析和自己设计这类系统的好方法。如果你很喜欢大理石纹，[你也可以考虑程序化地生成它们](https://hackaday.com/2020/03/04/procedurally-generating-marble-runs/)。休息后的视频。
