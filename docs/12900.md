# Hackaday 播客 157: Airtag 安全，扭曲的 3D 打印，用 DIY 机械臂缝合葡萄，以及巫师的计算器

> 原文：<https://hackaday.com/2022/02/25/hackaday-podcast-157-airtag-security-warped-3d-printing-suturing-grapes-with-a-diy-robot-arm-and-the-wizards-calculator/>

本周 Hackaday 主编埃利奥特·威廉姆斯和执行主编汤姆·纳尔迪关注本周最有趣的故事和项目，从反乌托邦新闻开始，几个人的仿生眼植入物在没有警告的情况下被关闭。然后，我们转到一个稍微不那么令人沮丧的讨论，关于苹果的空中标签网络的安全性差，以及如何在个人不知情的情况下使用它来跟踪个人。但这并不都是悲观的。我们将看看旨在推动桌面 3D 打印信封的新项目，并惊叹于一个 DIY 机械臂的构建如此精确，以至于它可以在葡萄皮上缝针。你还会听到自制水听器惊人的低成本，鹿蹄草背后令人不安的化学物质，以及看起来像来自霍格沃茨魔法学校的早期便携式计算机。
