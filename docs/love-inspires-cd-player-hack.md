# 爱情启发了 CD 播放器黑客

> 原文：<https://hackaday.com/2019/02/20/love-inspires-cd-player-hack/>

心脏是一个令人印象深刻的硬件。这是一种罕见的泵，在某些情况下可以连续运行 80 多年。出于某种原因，它也成了人类爱和感情的普遍象征。在这种情况下，[[迪帕克·卡特里]用基本的、容易获得的部件建造了一个跳动的电子心脏。](https://hackaday.io/project/163135-lors-electronic-heart)

建造的核心(双关语)是从 CD 播放器中抢救出来的镜头组件，它使用线圈和永磁体来移动镜头，以便读取光盘。在这种情况下，线圈改为从一个非稳态多谐振荡器电路馈送脉冲，该电路由黑客最喜欢的 555 定时器 IC 构建。这些都是在试验板上组装的，这是构建这种依赖于通过交换组件值进行实验的项目的好方法。

最终结果相当令人满意。[Deepak]还试验了一个 Arduino 驱动版本[，节奏略有不同。](https://youtu.be/tQZ-EJYry_k)

我们还没有看到太多使用光驱镜头组件的项目，但我们确信肯定还有其他应用。如果你最终用它来搅动生物样本或建造一个令人敬畏的激光投影仪，[一定要打电话给 tips line](http://hackaday.com/submit-a-tip) 。休息后的视频。
