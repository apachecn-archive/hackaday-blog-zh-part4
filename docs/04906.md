# 为 RC2014 计算机构建前面板

> 原文:[https://hack aday . com/2019/11/21/building-a-front-panel-for-the-RC 2014-computer/](https://hackaday.com/2019/11/21/building-a-front-panel-for-the-rc2014-computer/)

RC2014 是一个漂亮的 Z80 电脑套件，在过去多次为这些页面增色不少。它允许任何一个有烙铁和 USB 转串行适配器的人体验 20 世纪 80 年代早期桌面计算的刺激。但是，如果你正在寻找一种更加复古的体验呢？在这种情况下，[这款来自【詹姆斯·史丹利】](https://incoherency.co.uk/blog/stories/rc2014-frontpanel.html)的定制 RC2014 前面板可能正是让牵牛星挠痒痒的东西。

[![](../Images/c3d8ff9c2ebe17825bfd9b0174f4d37d.png)](https://hackaday.com/wp-content/uploads/2019/11/rc2014case_detail.jpg) 前面板让你只需拨动开关和发光二极管就能查看和更改内存内容，就像 20 世纪 70 年代早期的微型计算机一样。如果你曾经想了解计算机如何在最基本的层面上工作，单步执行指令并以二进制形式读出它们是一个很好的方法。

[James]说，他是在阅读了 1978 年的《Kilobaud》杂志(正如一个人所做的那样)并看到一篇关于制造带前面板的家酿 Z80 机器的文章后，受到启发而开始这个项目的。显然，他必须对这种方法稍加修改，以适应这个古老的 CPU 上相对现代的变化，但想法本质上是相同的。

他为这个项目所写的文档肯定会让那些迷恋于昔日标志性计算机的人着迷，但即使是对现代更敏感的读者也可能会发现一些有趣的细节。[James]将套件计算机中的数据和各种状态混合在一起的方式占了大部分篇幅，但之后他讲述了他是如何设计 PCB 的，并总结了他创建专业外观面板的技巧。

这不是我们第一次看到现代 Z80 电脑配备闪光灯，但它肯定是最专业的外观之一。如果你将 Tindie 上可用的 [RC2014 工具包与【詹姆斯】提供的细节结合起来，你很快就会学会用拨动开关编程电脑的艺术。](https://www.tindie.com/stores/semachthemonkey/)
