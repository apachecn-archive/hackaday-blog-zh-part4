# OpenChronograph 可让您滚动自己的智能手表

> 原文：<https://hackaday.com/2020/02/26/openchronograph-lets-you-roll-your-own-smart-watch/>

起初，智能手表就像你戴在手腕上的小平板电脑或手机。最近，我们注意到更多的“混合”智能手表，看起来像普通的手表，但使用手来交流数据。例如，你可能会听到一条短信，然后看到手摆向 1，表明这是你的另一半。想自己卷吗？ [OpenChronograph](https://github.com/snietfeld/OpenChronograph) 项目应该是你的第一站。

这些手表是 Fossil 和 Skagen 手表板的替代品(请记住 Fossil 和 Skagen 实际上是同一家公司)。有一个兼容 Arduino 的 Atmega328p，一个超低功耗的实时时钟，一个磁力计，压力传感器，温度传感器，以及对三只手的支持。你甚至可以[使用 Python 创建 PCB 艺术品，作为表盘](https://github.com/snietfeld/OpenChronograph/tree/master/Dials)。

你可以观看一段视频，了解如何用新版本替换现有手表的 PCB。你还需要一个特殊的微型弹簧针编程器，因为这个小盒子里没有太多空间。

老实说，打开手表对我们来说有点可怕，但至少混合手表通常不会那么贵。你通常可以在 100 美元以下找到一台哈根或一台男用机器。一个化石 Q 活动家看起来好像它可能进来只是多一点点。

当然，你可以跳过现有的手表，[从头开始打造一款](https://hackaday.com/2019/05/01/scratch-built-smartwatch-looks-pretty-darn-sharp-with-3d-printed-case-and-round-lcd/)。有些[比其他](https://hackaday.com/2019/01/01/a-smartwatch-you-can-easily-build-yourself/)更容易建造。
