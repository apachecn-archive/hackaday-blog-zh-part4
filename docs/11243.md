# 在家庭实验室制作有机发光二极管显示器

> 原文：<https://hackaday.com/2021/09/04/making-oled-displays-in-the-home-lab/>

只是一个一般的观察:当你的项目的 BOM 包括镱金属时，它很可能是有趣的东西。我们会说[在家里制作自己的有机发光二极管展示品](https://youtu.be/qg8pMUd-tSk)绝对属于这一类。

当然，制造有机发光二极管需要的不仅仅是一种稀土金属，更重要的是[耶鲁安·弗勒格尔]给这个项目带来的该领域的经验。在飞利浦工作多年后，[耶鲁安]已经做好了应对复杂过程的准备，包括物理气相沉积[和协同喹诺酮的有机化学。更不用说量子物理学了，这在下面视频的前十分钟左右做了很好的总结。从那里开始，所有的一切都是关于使用光刻技术和前面提到的 PVD 在 ITO(氧化铟锡)玻璃基板上建立 Alq3(一种电致发光有机化合物)夹层来制作几个有机发光二极管显示器。我们特别欣赏使用树脂 3D 打印机来创建光致抗蚀剂掩模，以及 PVD 工艺的细节。](https://hackaday.com/2021/05/31/thin-coatings-require-an-impressive-collection-of-equipment-and-know-how/)

这些展示本身看起来棒极了——至少在一段时间内。有机片段从材料中的针孔开始快速氧化；洁净室可以解决这个问题，但这毕竟只是一个演示。另外，[耶鲁安]显示器的蓝绿色光芒让我们强烈地想起了[本·克拉斯诺]不久前建造的阿波罗·DSKY 显示器的复制品。
