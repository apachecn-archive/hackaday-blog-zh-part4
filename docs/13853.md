# 菲涅耳透镜的起源

> 原文:[https://hack aday . com/2022/06/07/菲涅耳透镜的原点/](https://hackaday.com/2022/06/07/the-origin-of-the-fresnel-lens/)

如果你是一个业余读者，你可能知道什么是菲涅耳透镜。从高射投影仪到虚拟现实耳机，你可以在任何东西中找到它们。虽然现在看起来很普通，但菲涅尔透镜在当时是一项重要的发明，因为它彻底改变了海上导航，根据 IEEE Spectrum 的一篇帖子，这是其发明背后的驱动力。事实上，镜头已经被称为“[拯救百万船只的发明](https://www.bbc.com/travel/article/20190620-the-invention-that-saved-a-million-ships)”。

这个问题源于导航方面的问题。通过太阳和星星导航是好的，但当你有厚厚的云层或其他原因看不到它们时就不可行了。一座灯塔通常标志着一个重要的点，你要么想驶向这个点，要么有时想远离这个点。当然，今天，我们有全球定位系统，但在很长一段时间里，灯塔是你最好的选择。

问题是，在那个年代，灯塔就是一盏油灯，一面凹面镜，一个普通的镜头。这使得灯塔很难被发现。拿破仑开始委任灯塔作为桥梁和道路军团的一部分。这是一个雇佣了光学天才奥古斯丁·让·菲涅尔的团队。尽管一些灯塔已经在使用透镜，但他们没有使用特殊的菲涅耳式透镜。曾经有过关于建造这种透镜的猜测，但是当菲涅耳在 1819 年提出他的透镜用于灯塔时，他显然没有意识到这一点。他提出的镜头也与早期的提议有些不同。

这种透镜像一系列棱镜一样工作，边缘的棱镜使光线弯曲得更厉害，而中心的棱镜几乎不弯曲光线。与传统透镜相比，菲涅尔透镜更薄更轻，或者相反，对于相同的厚度和重量，菲涅尔透镜可以具有更好的特性。然而，这种扭曲使它们不太适合常规镜头仍然占主导地位的成像。当然，更薄的透镜应该让更多的光通过，这在你试图远距离发射光束时很重要。菲涅尔透镜让 98%的灯的输出通过，这可以向 32 公里外的船只发出信号。

到 1823 年，透镜出现在灯塔上。到 1860 年，美国所有的灯塔都使用了改进的光学系统。这是一个你不会真的想太多的东西，但它曾经是主要的高科技。你一定会想，200 年后，我们今天使用的东西会被归入普通和平凡的范畴。
