# (重新)做一个 ColecoVision

> 原文:[https://hackaday.com/2020/07/12/remaking-a-colecovision/](https://hackaday.com/2020/07/12/remaking-a-colecovision/)

[含铅焊料]在跳蚤市场发现了一些 ColecoVision 游戏盒，像我们大多数人一样，想，“我要从头开始构建一个 ColecoVision 控制台来玩它们！”好吧，也许我们大多数人会想到这一点，但不会真的去做。他做到了，你可以看到结果的细节，因为他写了两篇文章，一篇是关于设计的，一篇是关于 T2 的建设的。

众所周知，ColecoVision 是一款游戏控制台，可以扩展成一台不错的个人电脑。在这种配置下，它甚至还有一台雏菊轮式打印机。然而，在这两种配置中，游戏控制台都是操作的大脑。根据[含铅焊料]的说法，尽管生产了 200 多万个，但正常工作的产品价格很高，因为几个设计问题使得它们不太可能存活几十年。比起修理和改造一个原始的单元，建造新的更便宜也更有教育意义。

设计目标是使用复合视频、单电源，并减小电路板尺寸，主要是尽可能使用表面贴装器件。[含铅焊料]承认，在开始这个项目时，他没有完全理解地址和数据总线如何在这样的系统中工作，他记录了他所学到的东西，并以一个非常简洁的总线操作总结结束。然而，我们认为，“……一条总线上一次只能启用一个芯片……”这种说法有点误导。一次启用两个芯片，一个写入器和一个读取器。此外，同时读取几个器件的情况并不少见(例如，一些 CPU 的每个寄存器都有两个副本)。但这是吹毛求疵，尤其是在这里使用 Z80 CPU 的情况下。

当印刷电路板到达时，总价格标签已经超过 68 加元。我们记得在电路板上打乱 EPROMS，使布局更容易。毕竟，你并不关心 EPROM 中的位置 0 和 1 实际上是在相邻的单元中。你只在乎当你想要的时候，你会得到。ColecoVision 显然对其视频和音频芯片做了同样的事情。数据总线反向运行。也就是说，Z80 的 D0 引脚连接到 D7，D7 连接到 D0。不幸的是，最初的 PCB 没有考虑到声音芯片，所以新的电路板版本解决了这个问题。

这些帖子直截了当地指出了所犯的错误，也许我们都能从阅读中有所收获。例如，尺寸上的细微差异会导致前两个版本的电路板出现一些问题。最后，棋盘转了 6 圈。幸运的是，PC 板比以前便宜了。

即便如此，仍然有一些控制器的问题，但我们毫不怀疑他会解决它，并告诉我们什么时候他做到了。如果你曾经处理过这种规模的项目，你可能对调试新板的过程太熟悉了。但是我们中的大多数人不会记录到这种程度。如果你曾经想看别人做这样的新设计，你会想读这些帖子。

也许在制作电路板之前，[一块试验板会有所帮助](https://hackaday.com/2016/03/07/breadboard-colecovision/)。将游戏控制台转换成个人电脑的想法不仅仅是为了科尔科威视和亚当(科尔科威视的电脑个性)。流行的[雅达利 2600](https://hackaday.com/2020/02/29/converting-an-atari-2600-into-a-home-computer-did-that-ever-work/) 也曾有过同样的尝试。
