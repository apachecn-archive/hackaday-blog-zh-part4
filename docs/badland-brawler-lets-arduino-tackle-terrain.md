# Badland Brawler 让 Arduino 处理地形

> 原文：<https://hackaday.com/2019/01/27/badland-brawler-lets-arduino-tackle-terrain/>

对于一个电子人来说，制造一个机器人——尤其是一个健壮的机器人——可能有些令人生畏。[Jithin]从一个现成的四轮驱动底盘开始建造[一个越野 Arduino 机器人](https://rootsaid.com/off-road-robot/)他称之为 Badland Brawler。该套件略高于 100 美元，但正如你在下面的视频中看到的，它非常结实，有一个封闭的框架和大型泥胎。

其余部分包括一个 Arduino、一个电池和一个电机驱动器 IC。Arduino 是一个带 WiFi 的机器人(事实上是一个 MKR 1000)，还有一个控制机器人的手机应用程序。

老实说，一旦你搞定了底盘，剩下的就很容易了。当然，手机应用程序需要更多的努力，但是你可以用很多方法来代替它。例如，我想到了 Blynk。

电机驱动器很容易搞清楚。对于一些传感器来说，这将是一个很好的平台，允许更多的自主性。我们喜欢这个框架有很多不同的电路板和传感器的安装点，并且可以在内部容纳大部分东西。对于将要穿越崎岖地形的机器人来说，这可能是个好主意。

如果你决定用 Blynk 开发你自己的应用，我们已经用一种非常不同的机器人完成了[。四轮驱动机器人不需要](https://hackaday.com/2016/12/01/the-joy-of-the-esp8266-and-blynk/)[那么大](https://hackaday.com/2017/06/16/esp32-mini-robot-packs-sensors-and-4wd/)，就像我们过去看到的那样。
