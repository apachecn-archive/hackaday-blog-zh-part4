# 用绳子支撑的桌子教物理

> 原文:[https://hack aday . com/2020/03/25/table-hold-up-by-strings-teaches-physics/](https://hackaday.com/2020/03/25/table-held-up-by-strings-teaches-physics/)

如果你从未听说过张拉整体结构，你应该停下来看看下面的视频。在其中，[行动实验室]展示了一个 3D 打印的桌子，它是用绳子支撑起来的。我们没说被绳子吊着而是举起来了。至少看起来是这样。这个模型来自 [Thingiverse](https://www.thingiverse.com/thing:3970291) ，但它是你必须亲眼所见才能相信的东西之一。

基本的想法很简单。绳子有很大的抗拉强度，但在最轻微的压力下就会倒塌。琴弦的排列将力施加在悬挂着的中心琴弦上——这个力将琴弦向下拉。然而，其他三根弦不仅仅是为了展示，它们防止结构向任何一个方向倾斜。

实际上有这种结构的真实例子。视频展示了英国节上的天空之桥作为一个例子和一座澳大利亚的桥。该视频还指出，人体使用张拉整体，因为肌腱非常类似于模型中的弦。

对于在家上学的人，甚至是被隔离的孩子来说，这将是一个伟大的实验。打印本身看起来不是很有挑战性，尽管组装可能有点棘手。

这不是我们见过的第一个这样的结构。如果关于肌腱的讨论让你认为这可能在机器人学中有用，[你将是正确的](https://hackaday.com/2018/11/11/robot-never-misses-leg-day/)。
