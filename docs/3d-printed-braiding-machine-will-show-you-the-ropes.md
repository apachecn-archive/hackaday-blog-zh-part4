# 3D 打印编织机将向你展示绳索

> 原文：<https://hackaday.com/2021/11/04/3d-printed-braiding-machine-will-show-you-the-ropes/>

五月柱编织机，又名圆形编织机，是一种制作编织管的角齿轮编织机，就像那些编织的手指陷阱或可爱的开膛帕拉科德，使你无聊的 DIY USB 电缆变得有趣。之所以这么叫，是因为这个动作模仿了一群五月柱舞者的动作，他们在复杂的舞蹈中上下摆动，相互交织，这导致了同样复杂的杆装饰工作。

[![A 3D-printed model showing the motion of a maypole braider. ](img/05d72f51f517b6deec72d7be2cce7cbe.png) ](https://hackaday.com/wp-content/uploads/2021/10/maypole-braider-inner.gif) [像[kmatch98]这样的五月柱编织工负责各种绳索、细绳和其他编织品，比如飞鱼线](https://hackaday.io/project/181960-braiding-machine-maypole-braider-and-bobbins)。他们使用三个或更多的线轴，每个线轴上都装有一股纱线。

编织任何东西，包括头发，最重要的部分之一是在编织的过程中保持辫子的张力。在这里，每个线轴都骑在一个线轴载体内，它执行许多任务-它将线轴固定在适当的位置，在应该释放纱线的时候释放纱线，并在沿着轨道滑行时保持纱线的张力。

这台迷人的机器由直齿轮、一个 8 字形轨道的框架、一对喇叭齿轮和一只脚组成，脚是线轴底部的导轨，沿着轨道移动。早期，[kmatch98]制作了一个坐立不安的旋转器版本来可视化基本功能。他研究了商业线轴载体的图片，不仅成功地对它们进行了逆向工程，还通过取消两个弹簧中的一个并用重力代替它来改进设计。剩余的弹簧用于释放梭芯。

令人惊讶的是，这不是我们展示的第一台塑料编织机。这里有一个用奇异的乐高积木做的。
