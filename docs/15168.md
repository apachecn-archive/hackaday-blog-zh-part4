# Turbo Button 向早期个人电脑致以迷人的敬意

> 原文：<https://hackaday.com/2022/11/03/turbo-button-pays-charming-homage-to-early-personal-computers/>

PC turbo 按钮和 LED 时钟速度显示是早期个人电脑的常见功能。想要为他的现代战斗站添加一点复古的时尚，[马修·弗罗斯特]组装了一个[迷人而实用的涡轮按钮控制面板](https://youtu.be/LzP6FxHc00o)。

在过去的日子里，这个汽车术语意味着激活时的性能提升。相反，“turbo 模式”会以其额定速度为您的 x86 处理器计时。禁用“turbo”会抑制 CPU，通常会一直降到 4.77MHz。继承自最初的 IBM PC，一些早期的计算机程序依赖于这种特定的时钟速度，否则会在更快的硬件上运行得太快(或根本不运行)。在奔腾时代，个人电脑营销团队和工程师们都不再包括涡轮按钮和发光的时钟速度数字。

这种对 turbo 按钮的现代重新想象使用 Arduino 微控制器、七段显示器和触觉开关来模拟原始硬件的外观和感觉。不是直接调整 CPU 时钟速度，而是在平衡和高性能 Windows 电源计划之间切换。七段显示屏以 GHz 为单位测量时钟速度，精确到两位小数。我们承认，当切换到 turbo 时，看到这些数字一英寸一英寸地增加是非常令人满意的。

 最右边的按钮在测量 CPU 速度、GPU 利用率、网络负载和内存利用率之间切换，在原来的灵感基础上有所改进。管状钥匙锁也是早期电脑上常见的一种，它可以启用和禁用整个系统的网络，这对于让孩子们远离网络非常有用(至少在他们知道如何从系统中移除 5.25 英寸**驱动器托架并使用回形针热连线网络适配器之前)。**

 **在 GitHub 页面上有更多的细节，如果你想自己开发的话。这个项目可能看起来特别吸引人的 PC sleeper 版本，其中新的组件被“隐藏”在旧的机箱硬件中。如果这让你感到怀旧，你可能想听听我们对[的想法，为什么它都是关于奔腾](https://hackaday.com/2021/11/04/weird-als-monster-battlestation-is-now-just-a-reasonably-fast-pc/)。
