# 听到无法忍受的

> 原文:[https://hackaday.com/2021/08/07/hearing-the-unhearable/](https://hackaday.com/2021/08/07/hearing-the-unhearable/)

我的妻子正在观看一部犯罪剧，其中一个情节转折涉及一名证人的助听器出现故障，以便他能够听到周围的电磁波。如果你仔细想想，这并不是那么不可信。许多助听器都有一个 t 形线圈，本质上是一个感应器，用于与电话中的扬声器耦合。如果那出了乱子，也许你可以听到你周围所有变化的磁场，如果你可以摆脱主电源线的持续嗡嗡声，那甚至可能是有趣的。

当然，她转向我说“我们需要做一个！”这应该一点也不难——一个大电感和一个放大器就够了。事实上，这可能很容易，对我儿子来说，这将是一个很好的电子入门项目。但这里也有足够多的未知，这将是有趣的。多大的线圈？有多近？有多敏感？电源频率位呢？铁氧体磁芯还是不是？

当然，这些都不是火箭科学，但它可能充满了组装、发现和直接探索。总之，完美的周末项目。最后，它会揭露一些通常看不见的东西，这就是乐趣所在。

这一定是驱使法拉第和马可尼、伏打和麦克斯韦的同样的冲动。直接感知、看到、听到和理解一些在我们有限的听觉和视觉范围之外，却又在我们周围的东西，是一件令人惊奇的事情。我可以写下描述它的方程式——毕竟我在学校里学过——但是没有什么可以代替在你自己的家里四处闲逛。谁知道呢，也许再过几个周末，我们就能给自己造一个全波段接收器了。
