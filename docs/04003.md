# 空气曲棍球台是一件轻而易举的事

> 原文:[https://hack aday . com/2019/08/26/air-hockey-table-is-a-breeze-to-build/](https://hackaday.com/2019/08/26/air-hockey-table-is-a-breeze-to-build/)

我们中的许多人都考虑过购买一个空气曲棍球表，但推迟了价格。而且就算有钱，那些东西也占了很大空间。你真的打算多长时间使用一次？

[这个 DIY 的空气曲棍球桌就是答案](https://www.instructables.com/id/DIY-Low-Cost-Air-Hockey-Table/)。它够大够好玩，但又够小够轻，可以在淡季轻松收藏。只要你有一个可以切换到吹风机模式的吸尘器，它的价格大约是 50 美元，是一个便宜的产品。前锋、球门、护角和记分牌外壳都是 3D 打印的，而冰球和操场是激光切割的丙烯酸材料。[Technovation]将丙烯酸树脂脚粘在罢工者身上，帮助他们坚持更久。

记分牌是一个 Arduino Uno 加上一个 LCD，它会改变颜色以匹配当前的获胜者。得分必须通过按键手动输入，但我们认为用力或重量传感器或其他东西检测球门中的冰球是相当容易的。目前，边缘周围的 RGB LEDs 是用遥控器单独控制的。最终目标是让 Arduino 做到这一点。拍摄过去的突破和交叉检查出来。

已经有桌子了？过了这么久，就没人再玩你了？给自己造一个机器人对手。
