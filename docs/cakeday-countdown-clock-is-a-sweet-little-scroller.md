# Cakeday 倒计时钟是一个可爱的小卷轴

> 原文：<https://hackaday.com/2020/07/08/cakeday-countdown-clock-is-a-sweet-little-scroller/>

如果你想让陌生人在你生日的时候给你美好的祝愿，你必须戴上一顶尖尖的帽子或者一个写着“今天是我的生日，给我礼物”的纽扣来引起人们的注意。但是在你的 reddit cakeday，也就是你加入的那一天，你会被自动挑选出来，在你的用户名旁边加上一块 8 位蛋糕。你的 cakeday 的伟大之处在于，你几乎肯定会得到一次报应，尤其是如果你做了一些与 cakeday 相关的事情，比如[【ScottyD】的 cakeday 倒计时钟](https://www.thingiverse.com/thing:4521908)。但是很多人忘记了他们的蛋糕日是什么，错过了乐趣。

这个倒计时钟的工作方式和你想象的一样——每一天都有一条信息显示离下一天还有多少天。当大喜的日子到来时，信息变成今天是你的蛋糕日。这两条信息都以可爱的像素化小蛋糕片为书挡，如果我们做了其中的一个，我们会在当天大量应用。

这个简单但有趣的项目不应该对您的部件盒产生太大的影响，因为它本质上是一个 Arduino，一个实时时钟模块和一个 32×8 LED 矩阵来显示文本。我们喜欢一体式外壳设计，因为它为 Arduino 创造了一个搁板，便于从后面粘贴显示屏。如果出于某种原因你不 reddit，那么无论如何做一个，用它来倒数你的 IRL 生日或什么的。休息之后，我们为您准备了构建视频剪辑版供您欣赏。

如果 2020 年给你带来了足够多的生存危机，我们会理解，但如果没有，考虑建造一个倒计时你剩余寿命的时钟。
