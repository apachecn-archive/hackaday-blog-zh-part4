# 原型制作原型

> 原文：<https://hackaday.com/2022/11/29/prototyping-the-prototype/>

对于基本的原型制作，拼凑功能电路的首选工具是试验板。在花钱花时间在 PCB 上之前，这是证明一个概念可行的好方法。对于更复杂的任务，我们可以利用模拟软件，如 SPICE。但是还没有真正的工具将这两个概念融合在一起。这就是 CRUMB 希望解决的问题，它是一个允许模拟试验电路板电路的工具。

目前，大多数基本电路功能都适用于 1.0 版本。这包括电阻、电容、开关、一些发光二极管和电位计等无源元件，以及晶体管和二极管等有源元件。有一些逻辑芯片可用，如 74XX 系列芯片和 555 定时器，这打开了一个庞大的电路建设阵列。甚至还有一个示波器功能，加上音频输出，将蜂鸣器纳入电路模拟。目前正在开发的是 LCD 显示模块和对示波器的改进。

除了原型制作，这对任何人都有用，包括学生，他们正在学习电路而不需要购买任何硬件。这个项目的主要缺点是，它似乎没有免费或试用版本，源代码不可用，而且它只在 Steam，Apple Store 和 Google Play 上销售。也就是说，有一个论坛可供用户讨论问题和未来版本的需求，因此有可能围绕它建立一个社区。我们已经看到[以前的非自由版本的电路仿真软件](https://hackaday.com/2020/09/20/ti-and-cadence-make-pspice-free/)在一段时间后变得更加开放，所以这不是不可能的。
