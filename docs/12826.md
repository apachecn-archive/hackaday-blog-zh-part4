# Hackaday 播客 156: 3D 打印彩虹、分体挂钟、电动汽车电池交换和软盘时间

本周，Hackaday 主编埃利奥特·威廉姆斯和作业编辑克里斯蒂娜·帕诺斯(Kristina Panos)对一款漂亮的意大利分体挂钟趋之若鹜，这款挂钟价格不菲，还有一款由软盘制成的时钟，可以几乎不花钱就能重新制作。我们还将为原型的无焊电路唱赞歌，并对一个有足够多的传感器来控制整个房子的灯丝干燥箱感到惊叹。我们将讨论那些闪闪发光的衍射光栅的细微之处，在展览结束时，你会知道我们每个人在显微镜下喜欢什么。
