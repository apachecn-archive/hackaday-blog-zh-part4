# 瘦客户机和智能手机取代了 3D 打印机的 Raspberry Pi 和触摸屏

> 原文：<https://hackaday.com/2022/11/04/thin-client-and-smartphone-step-in-for-3d-printers-raspberry-pi-and-touchscreen/>

众所周知，树莓派现在有点难找了。除非你有先见之明，在供应枯竭之前囤积货物——如果你有，我们想和你谈谈——很有可能你已经有相当数量的项目无限期搁置了无处不在的 SBC。也许这让你开始考虑 Pi 的替代方案。

这显然是最近[Crimson Repair]的想法，其结果是发现一台旧的瘦客户机 PC 至少在某些情况下是树莓派的不错替代品。诚然，下面的视频很长，但它充满了如何获得 Dell Wyse 3040 的逐命令说明，这是一款瘦客户端，可以在二级市场上以 25 美元左右的价格找到，并作为 3D 打印机的 Klipper 替代品运行。这些机器通常用于销售点应用等，配备了 1.4 GHz 的英特尔凌动处理器和几千兆字节的内存，其外形正好适合塞进 Ender 3 的底座。

安装并运行一个就是把 Debian 镜像放到 USB 盘上，并配置瘦客户机从 USB 启动。之后，安装 [Klipper](https://hackaday.com/2017/12/26/fast-3d-printing-with-raspberry-pi-but-not-how-you-think/) 和连接降压转换器为机器供电就很简单了。这不完全是火箭手术，但是当有人已经走在你前面的时候，为什么还要蒙混过关呢？如果你想更进一步，下面的[第二个视频](https://youtu.be/TcgTrkA8Oj0)将带你完成使用旧安卓手机添加触摸屏所需的所有步骤。有了 3D 打印支架，整个事情就是一个非常完整的打印机控制解决方案。
