# 你想戴的 BCD 手表

> 原文：<https://hackaday.com/2018/07/24/a-bcd-wristwatch-youd-want-to-wear/>

时计是日常生活中的必需品，这些年来我们已经拥有了太多这样的时计，对于我们来说，一个新的时计一定是特别特别的。简单地爆发 Nixies 就足够的日子已经过去很久了。

所以这个由[斯维德工业]的二元手表项目绝对吸引了我们的眼球。不是因为特别新颖，毕竟二进制 LED 时钟本身并不难，而是因为它的构造质量非常高。这是一个足够简单的设计，在一个圆形 PCB 上有一个实时时钟芯片和一个处于最省电模式的 ATmega328，led 阵列作为显示器，所有这些都包含在一个 3D 打印的外壳中。

这种设计具有真正的品质，分立元件塞在电路板下方，只留下顶部的 IC 和 led 陪伴。玻璃正面被粘合到位，外壳是专业的 3D 打印。电源来自单个 CR2032，为了节省电池寿命，led 仅通过按下隐藏的按钮来激活。我们会戴上这块手表。就此而言，你会戴上这只表。看看休息下面的视频，我们相信你会同意。看起来有几个甚至可以在 Tindie 上找到[。](https://www.tindie.com/products/SverdIndustries/binary-wrist-watch/)

这不是我们推出的第一款二元手表，所以很难进行比较。这个[极低 BoM 的例子](https://hackaday.com/2015/12/10/the-three-week-three-dollar-binary-watch/)可能缺少这里展示的例子的一些修饰，但是它同样有能力抓住我们的眼球。
