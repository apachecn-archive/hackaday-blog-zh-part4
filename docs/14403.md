# 黄铜牌匾纪念哥哥

> 原文：<https://hackaday.com/2022/08/03/brass-plaque-honors-brother/>

黄铜牌匾之所以引人注目，是因为没有人会心血来潮制作它们。它们比木材或塑料更昂贵，加工它们也相对困难。[Becky Stern]选择了这个媒介来纪念她喜欢咖啡、摩托车和手工制作的哥哥。她做了一些扑克牌大小的东西来装饰他最喜欢的热豆汁品牌，还做了一个大的挂在他的纪念地。

主要部件是立式盐浴、DC 电源、待蚀刻的金属、大约相同大小的废钢和水搅拌器，在这种情况下，水搅拌器是气泵和扩散石。你可以手动搅拌两个小时，尽情享受你的表演，但是请相信我们，走简单的路线。该视频没有明确要求使用柔性电线，但[Becky]明智地选择了一些高股连接导线，这将减少令人头疼的问题，因为 stiff copper 有自己的想法，你不希望双方发生碰撞。

有几种方法可以将绝缘掩模转移到金属上，我们在视频中看到 ole' magazine paper 方法失败了，但切割乙烯基材料是一种享受。你可能更喜欢激光或[树脂打印机](https://hackaday.com/2018/11/24/put-that-dlp-printer-to-use-making-pcbs/)，也没关系。一旦你的面具被分类，连接正极引线到黄铜，负极引线到你的钢。现在，它进入搅动的盐水浴，应用直流电，让电使你的设计不朽。
