# 音乐电脑平台是部分合成器，部分 MIDI 控制器，一切都很酷

> 原文:[https://hackaday . com/2021/09/09/09/musical-cyber deck-is-part-synth-part-midi-controller-and-all-cool/](https://hackaday.com/2021/09/09/musical-cyberdeck-is-part-synth-part-midi-controller-and-all-cool/)

当一个新的项目类型开始获得大量曝光时，通常不久我们就会看到人们分叉出基本概念，并在新的方向上取得突破。POV 显示器发生了这种情况，谢妮钟表也发生了这种情况，现在，这种情况似乎正在电脑平台上发生。这是我们可以得到的东西，特别是为了适应特殊任务而构建的 cyberdeck，比如这个音乐 cyberdeck/synth。

像许多音乐家一样，[本杰明·卡西亚]在和他的乐队“大杀器”一起表演时，觉得自己需要一个工具来帮助他主要需要在飞行中触发音轨回放，但也想要一些东西作为巨型效果踏板和独立的合成器。虽然大部分工作可以用 iPad 来完成，但它看起来没有 cyberdeck 那么酷。该建筑以 Raspberry Pi 4 和 7 英寸液晶显示器为中心。它们位于一个 25 键 USB MIDI 键盘和一个小型混音器的顶部。键盘旁边是一个 USB 小键盘，它有自定义映射，允许快速访问 cyberdeck 的 Patchbox 操作系统中的隐藏菜单功能。所有的东西都被绑在一个 3D 打印的框架上；下面的视频展示了它的实际应用，它听起来和看起来一样好。

我们认为[本杰明]的网络平台非常棒。需要看看其他一些专门的网络平台吗？为什么不看看[这个准备好战斗的虚拟平台](https://hackaday.com/2021/04/08/three-piece-cyberdeck-plays-the-role-of-military-computer-that-never-was/)、[一个旨在不受干扰的](https://hackaday.com/2021/02/09/stay-focused-with-this-distraction-free-cyberdeck/)，或者[一个用于巡逻放射性荒地的虚拟平台](https://hackaday.com/2020/11/24/over-the-top-cyberdeck-is-really-a-geiger-deck/)。
