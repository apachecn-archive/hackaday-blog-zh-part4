# Z80 单板电脑看起来像是黑仔的产品

> 原文:[https://hackaday . com/2022/08/20/z80-单板电脑-看起来像是可能是杀手的产品/](https://hackaday.com/2022/08/20/z80-single-board-computer-looks-like-it-could-have-been-a-killer-product/)

大多数逆向计算机构建似乎都集中在恢复旧机器或从头开始重建它们。无论哪种方式，目标都是尽可能接近原始机器，虽然我们当然尊重这些构建，但还有其他方式来庆祝昨天的计算机，正如[这台 Z80 单板计算机](https://github.com/solisted/z80sbc)很好地展示的那样。

[Ivan Farafontov]的 SBC 有点像“从未出现过的 Z80 ”,在 8 位计算的全盛时期，这几乎是可能的，如果它出现的话，会引起很大的轰动。大多数外围芯片来自 Zilog，在当时的许多 Z80 机器中都可以找到，如 TRS-80 和 ZX 频谱。它与老派路线不同的地方是视频部分，它使用 Atmel CPLD 芯片和双端口 RAM 来驱动 VGA 显示器。不过，它看起来仍然很像，配有 256×192 像素的 16 色显示屏。紧凑的视频部分有助于保持这台机器的整体占用空间非常小，至少按照旧机器的标准来看是这样。这台机器比它的定制键盘大不了多少，键盘上有机械开关和非常好看的定制键帽，一切都可以装进 3D 打印的盒子里。

从下面视频的 4:30 分开始的演示对许多读者来说将是一场怀旧风暴，因为它是由[Ivan]从头开始编写的一个版本的 *Boulder Dash* ，以及他用来创建游戏精灵的磁贴编辑器。当然，如果你想自己构建，所有的设计文件和代码都是可用的。我们最近推出了[的另一款 Z80，它从来就不是](https://hackaday.com/2022/07/19/modular-z80-really-racks-up-the-retrocomputer-cred/)，但是【伊万】的机器确实以其紧凑的尺寸和强大的功能做出了声明。
