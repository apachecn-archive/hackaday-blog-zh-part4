# 哥伦比亚决定 3D 打印食物尝起来像鸡肉

> 原文：<https://hackaday.com/2021/09/26/columbia-decides-3d-printed-food-tastes-like-chicken/>

哥伦比亚大学的研究人员已经使用多波长激光烹饪 3D 打印的鸡肉。显然，它尝起来像鸡肉。我们对 3D 打印的鸡肉蛋白烹饪出鸡肉的味道并不感到惊讶，但是，话说回来，你必须做科学研究。

虽然增材制造是各种制造业的最新流行语，但也有各种 3D 打印食品的尝试。我们也见过披萨打印机和假牛排打印机。你不想打印生食是有道理的——成品需要煮熟。你可以在下面看到几个关于这个过程的视频。

从好的方面来看，用蓝光和红外激光烹饪的鸡肉比传统烹饪技术收缩得少。据称，它还保留了两倍的水分含量，因此，与仅仅将一些油炸食品放在烤架上相比，3D 打印的鸡肉可能有一些真正的优势。

我们不确定我们是否准备好将我们的打印机转换成制作鸡块，我们现在将坚持更传统的烹饪方式。但是很高兴知道当我们准备打印下一餐时，有科学数据可以帮助我们。

如果你的口味范围是牛肉模拟品，我们以前见过这种打印机。[转换打印机](https://hackaday.com/2021/06/18/3d-printing-food-university-style/)来处理食物似乎并不困难，尽管你的结果可能会有所不同。
