# WiFi 会杀死室内植物吗？

> 原文:[https://hackaday.com/2019/02/28/does-wifi-kill-houseplants/](https://hackaday.com/2019/02/28/does-wifi-kill-houseplants/)

剧透警告:没有。

为了得出这个与最近几个 YouTube 视频的综合智慧背道而驰的结论，[安德鲁·麦克尼尔]做了一个非常巧妙的小实验。[Andrew]作为一名射频工程师和许多自制 WiFi 天线的多产制造商，在这一领域拥有相当多的专业知识，其中一些[我们以前在这些页面上介绍过](https://hackaday.com/2018/04/28/umbrella-and-tin-cans-turned-into-wifi-dish-antenna/)。他的实验集中在堆肥中发芽的水芹种子。准备两个相同的容器，其中一个从上方浸泡在来自三个独立的 2.4 GHz 发射器的 RF 能量中。每个发射器都耦合到一个放大器和一个 PCB 双象限天线，在 WiFi 频谱的略有不同的部分辐射约 300 mW。两个装置都被放置在朝东窗户的独立房间里，每隔一天在房间之间交换一次，以平衡微环境的影响。

仅仅过了几天，水芹在两个花盆里都发芽了，并继续生长。射频处理过的豆芽没有明显的抑制作用——事实上，它们看起来比未处理过的花盆更茂盛。[Andrew]指出，在量化之前，这不是真正的科学，所以他的下一步是重复实验，并仔细测量生物量。他还计划在下一轮加大马力。

我们认为这将终结“WiFi 杀死了我的室内植物”的无稽之谈——毕竟，WiFi 甚至可以帮助你的植物存活。但不知何故，我们怀疑这场辩论会很快消失。
