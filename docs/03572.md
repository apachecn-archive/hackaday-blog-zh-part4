# 堆叠的珀耳帖云室结构

> 原文：<https://hackaday.com/2019/07/15/a-stacked-peltier-cloud-chamber-build/>

亚原子粒子非常难以看到，但通过正确的技术，它们是可以被看到的。用干冰建造一个云室是实现这一目标的常用方法，但获取材料可能很困难。[The Thought Emporium]想要一个更容易接近的建筑，[转而采用了基于帕尔贴的设计(Youtube 链接，嵌入在下面)](https://www.youtube.com/watch?v=-Cu3wW8rNAs)。

通过级联堆叠几个帕尔贴冷却器，可以增加产生的温差。在这种设计中，试验箱的铜板被冷却到零下 33 华氏度(-36.11 摄氏度)，这对于实验的进行来说已经足够冷了。酒精被添加到玻璃容器中，当它到达冷板时，它会产生过饱和蒸汽。当受到从放射源中冲出的带电粒子的干扰时，蒸汽凝结，留下可见的痕迹。

云室是一种在家中尝试的流行实验。这是一个很棒的科学展览项目，而且可以很容易地用旧电脑部件和几个来自易贝的廉价模块来建造。做放射源实验的时候小心点就好。休息后的视频。
