# 这台喷气式发动机会帮你度过难关

> 原文：<https://hackaday.com/2022/07/14/this-jet-engine-will-see-you-through/>

你是否曾希望能在一台复杂的机器还在运转的时候窥视它的内部？我们可以用今天的模拟和 CAD 工具来做，但这和做 IRL 不一样。【扭曲感知】做了一个[透视喷气发动机体验一下感觉](https://www.youtube.com/watch?v=MgL0GW248mE)。效果，我们敢说，比任何模拟都要好。

[扭曲的感知]对喷气发动机很有经验，以前曾在他的车上安装过。第一步是平衡，虽然他没有[使用示波器](https://hackaday.com/2022/06/18/balancing-a-motor-with-an-oscilloscope/)，他可以在几千克的范围内获得平衡。然后经过一些轻 CAD 的工作，都是机械加工。制造了支架，并用激光切割了垫圈，以将大而厚的透明盖固定在一起。有一些令人兴奋的东西可以看(和听)。由于压力和热量，发动机膨胀和收缩明显，但有趣的是，当它上下倾斜时，看到它在物理上移动。

此外，声音，因为它通过各种推力水平是相当令人印象深刻的。但是，当然，有气流测试的喷气发动机测试是什么？令人惊讶的是，发动机吸入的空气并没有他想象的那么多。八十磅的推力并不意味着八十磅的空气。

这个 [3D 打印的水冷喷气发动机](https://hackaday.com/2022/05/30/3d-printing-a-water-cooled-jet-engine/)并不完全透明，但看到制造发动机本身的完整过程很有趣。休息后的视频。
