# 如果你有时间，你可以成为一名制造工程师

> 原文：<https://hackaday.com/2020/01/03/you-could-be-a-manufacturing-engineer-if-you-could-only-find-the-time/>

老实说，露丝·王君馨不可能在 20 分钟的演讲中教会你如何成为一名制造工程师。但是没人能做到。这是关于在不遵循传统教育途径的情况下获取新职业的技能，这需要一些时间。但是格蕾丝成功了，她在 2019 年 Hackaday 超级大会上的演讲分享了她在不完全打乱你的生活的情况下重塑你的职业道路的经验。

当露丝意识到成为一名创造者让她快乐，她想做更多的事情时，她开始了疯狂的旅程。See 希望“大规模制造产品”,这是制造业的定义。她采取了黑客的方式，通过利用她的个人项目来揭开制造业的面纱。她做了一些众筹活动，这些活动让她认识到生产多种产品的困难。同时利用这些项目的收入进行培训和寻求指导。

这些导师可以有多种形式。她的成功很大程度上源于她在会议上通过在线互动建立的人际关系，以及在她位于旧金山的本地黑客空间 Noisebridge 建立的人际关系。她提到的最有趣的故事之一是一个导师黑客，她在那里找到了拥有她想要学习的技能的人，并说服他们如果她做了所有的组织跑腿工作，就给他们开一堂关于这方面的课。

当开始在一家初创公司从事制造项目的机会到来时，她开始学习普通话。但她的技能习得阶段从未结束。露丝提倡个人好奇心，用自己的好奇心作为雷达，去发现别人可能错过的新事物。

但是在这次演讲中，我们可能受益最大的是她在最后关于寻找时间的讨论。露丝说“你的生产力永远不会提高”，因为有生产力的人已经在利用他们的可用时间了。这并不意味着你不能兼职学习新的职业技能。但这确实意味着你需要扔掉一些东西。从迟到，到用一个干净的衣箱，露丝当然是用了很多生活窍门才做到这一点的。她成功了，在最初的顿悟两年后，她终于找到了一份制造工程师的工作。
