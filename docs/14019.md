# Hackaday 播客 174:闯入巢穴，最便宜的 3D 打印机，你的 HDMI 中的间谍，到处都是人工智能

> 原文：<https://hackaday.com/2022/06/24/hackaday-podcast-174-breaking-into-the-nest-the-cheapest-3d-printer-a-spy-in-your-hdmi-and-ai-all-over-the-place/>

刚刚度假归来的主编埃利奥特·威廉姆斯成功回归播客！本周，主编汤姆·纳尔迪加入了他的行列，他很高兴自己不用再一个人做所有的事情。在这一集里，我们将讨论如何处理定制印刷电路板中的 BGA 组件，这是一个针对谷歌 Nest Hub 的特别出色的黑客攻击，以及为什么你可能并不真正想要世界上最便宜的 3D 打印机。我们还将看看一个令人难以置信的项目，将诺基亚 1680 变成一台 Linux 驱动的手持电脑，第一个 HDMI 防火墙，以及一个非常擅长制作墨西哥玉米卷的机器人。对人工智能感兴趣的听众也将是一种享受，就像那些梦想将地位低下的汽车交流发电机提升到黑客世界中更突出位置的人一样。
