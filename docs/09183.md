# 对 TIL311 显示屏的现代致敬

> 原文：<https://hackaday.com/2021/02/08/a-modern-homage-to-the-til311-display/>

早在 20 世纪 70 年代，市场上就有各种各样深奥的 LED 显示屏。其中之一是德州仪器(Texas Instruments)的 DIP 封装 TIL311，能够显示 16 进制，从 0-9 到 A-F。虽然这些不再容易获得，但深红色的塑料封装有一些美感，因此[Alex]开始制作一种现代娱乐。

该建筑由一个装有 20 个 led 的小 PCB 和一个运行该节目的 STM8S 微控制器组成。这可以用来仿真 TIL311 上的原始解码器逻辑，或者用其他固件编程，以便测试显示或启用其他显示功能。然而，该项目真正的亮点在于视觉展示。[Alex]一直在尝试用半透明的红色树脂封装硬件，以恰当地模仿原始部件的外观，这对于获得 70 年代的酷美学有很大的帮助。对细节的关注是一流的，[Alex]甚至仔细选择了与 TIL311 原件上的方形切割设计最匹配的引脚。

这是一个有趣的构建，当你不能使用新的和旧的库存时，它可能对一个项目有用。过去，我们已经看到了谢妮电子管的类似努力。休息后的视频。
