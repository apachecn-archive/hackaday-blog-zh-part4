# 单轮摩托车拖车

> 原文：<https://hackaday.com/2021/08/02/single-wheel-motorcycle-trailer/>

一辆摩托车可能不会占用太多的停车空间，但是对于运输摩托车的拖车来说就不是这样了。为了解决这个问题，[Make It Extreme]制造了一辆定制的[单轮摩托车拖车](https://www.youtube.com/watch?v=a7PVPVbyK9k)，宽度仅比摩托车本身宽一点。

拖车的框架是由几节大直径钢管焊接而成的，后端的 C 形部分装有一个车轮。一个标准的球形挂接装置可以让整个拖车向一侧倾斜，因此使用了一个销钉挂接装置，带有一个枢轴，可以上下移动。另一个枢轴被添加到后轮前方的框架上，以允许拖车降低到地面进行装载。它通过一个手动泵压的液压缸上升和下降，后面的一个小旋转坡道也可以作为摩托车后轮的止动装置。有了刚性车架和无悬挂，我们只是想知道它在高速行驶时应对颠簸的能力如何。

[Make It Extreme]非常喜欢他的休闲车，其中包括一辆[单轨摩托车](https://hackaday.com/2019/09/11/rideable-tank-tread-its-a-monotrack-motorcycle-that-begs-you-to-stop-very-slowly/)、一辆[独轮车](https://hackaday.com/2019/01/16/this-monowheel-is-bright-orange-and-we-want-one/)和一辆[全地形悬浮滑板](https://hackaday.com/2016/07/28/all-terrain-hoverboard-junk-pile-build/)等等。
