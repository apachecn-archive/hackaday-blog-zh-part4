# 轻松 DIY 壁虎胶带

> 原文：<https://hackaday.com/2019/05/23/easy-diy-gecko-tape/>

壁虎是一种神奇的生物，能够在各种表面上行走和粘附。如果你曾经醒来在你的天花板上看到蜥蜴，你已经熟悉它们的能力。这背后的机制是最近许多研究的领域，并且[[The think Emporium]决定尝试自己重现这种效果(Youtube 视频，嵌入下面)。](https://www.youtube.com/watch?v=vpTX32KdVBQ)

壁虎粘在物体表面的方式是通过它们脚上纳米级的毛发。这些毛发极大地增加了壁虎与物体表面的接触面积。这使得通常很小的分子间作用力累积起来，使壁虎保持粘附。

几个团队已经成功创造出能够重现这种能力的合成物质。的确[我们以前在这里展示过一些。](https://hackaday.com/2010/04/15/stanfords-stickybot3/)在这种情况下，实验开始尝试通过在微孔过滤器上浇铸 RTV 硅树脂来生成所需的纳米结构。这是不成功的，因为创建的材料表面上的毛发太稀疏，并且处于随机的角度。下一步包括尝试使用纹身枪，针，最后是锋利的钨丝来制作蜡像，然后可以在上面浇铸硅胶来获得几何形状。这也是不成功的，因为不可能生成足够小的特征来产生效果。

最后的实验包括在每毫米 1000 线的衍射光栅上浇铸硅树脂。这在硅树脂的表面产生了微小的隆起，大大提高了它的粘性。虽然生成的脊不像壁虎脚或专业制作的电影那样有能力，但它们确实具有令人印象深刻的承重能力。一小部分硅胶在测试中能够长时间承受超过 20 磅的重量。

这是一个很好的例子，说明了如何用家庭作坊里容易获得的材料来做看似复杂的科学。我们很想看看在这种方法上做了更多的工作后，壁虎胶带能有多结实。休息后的视频。
