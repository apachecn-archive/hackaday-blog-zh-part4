# 加工木质镶嵌物，不需要 CNC

> 原文：<https://hackaday.com/2021/10/17/machining-wood-inlays-no-cnc-required/>

几乎很难记得有哪个时候，对大多数制造业问题的显而易见的答案不是“扔进数控系统”数控机床已经变得如此根深蒂固，以至于首字母缩略词已经成为一个动词；很少有人会误解类似“我们就这样吧”这样的陈述。

但是在 CNC 机器变得如此普遍之前，有许多以可控方式切割材料的巧妙技巧，正如[Pask]用[这个工具向我们展示的那样，它可以加工木材用于镶嵌](https://www.youtube.com/watch?v=005_HNv_v4w)。该工具被称为解析器(或传递器)钻，旨在与钢模板结合使用。[Pask]的版本似乎很容易制作；一对软钢棒被锻造成扁平的铁锹形状，然后在上面磨出一个切割面。钻头的两半焊接在一起，并打磨成适合手钻的卡盘，这是一种现代的认可，因为很少有人愿意使用驱动原始解析器钻头的传统弓和胸甲。

在使用中，确定嵌体形状的钢模板被固定到工件上。钻头的切削刃插入模板切口中以加工出木材；钻头的突出部分起到深度限位和导向的作用。只需要几秒钟就可以制作出一个整洁的、无 CNC 的镶嵌物。下面的视频展示了正在制作和使用的工具。

很高兴看到不需要花哨的数控机床就能完成的事情。当然，我们并不反对他们，但是当用一些废钢和一点小聪明就能得到同样的结果时，这是非常令人印象深刻的。想找一个介于手工工具和 CNC 之间的东西做木工？pantorouter 可能就是你的速度。
