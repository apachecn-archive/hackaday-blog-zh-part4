# 在苏联，门铃响了

> 原文:[https://hack aday . com/2020/01/13/in-Soviet-Russia-门铃-铃声-你/](https://hackaday.com/2020/01/13/in-soviet-russia-doorbell-rings-you/)

我们可以想象门铃的起源确实很古老。如果你住在一个山洞里，你可能会在附近放一根棍子和一块石头，让人们在不侵犯你的山洞的情况下引起你的注意。1817 年，一个名叫威廉·默克多的苏格兰人在房子里装了一个钟，游客可以通过压缩空气系统按铃，但电动门铃不得不等到 1831 年。自那以后，基本理念几乎没有改变。住在荷兰而不是俄罗斯的 Erientes 想要一个更智能的门铃。特别是，他读到过老年人被按门铃开门的人所害。所以【Erientes】用一个树莓 Pi 做了[一个支持面部识别的门铃](https://www.instructables.com/id/Doorbell-With-Face-Recognition/)。

这个练习实际上更像是一个操作挑战，而不是一个技术挑战，这要归功于由 DLib 支持的用于[人脸识别](https://github.com/ageitgey/face_recognition)的高质量 Python 库。然而，我们确实喜欢面向非技术用户的用户界面。这个比喻是一个交通灯，红灯意味着不允许进入。这些灯是按钮，因此您可以使用它们将特定的人列入白名单或黑名单。

我们可以看到它与一个小键盘相结合，构成一个双因素门禁系统。要打开门，你必须出示你的脸并输入密码。虽然面部识别系统确实不完美，但有人学习你的代码并能够复制你的脸以欺骗算法的可能性似乎很小。

如果你想在线玩面部识别，你[不需要安装任何软件](https://beta.deepnote.com/project/a72491db-ee65-4a47-9843-575e1afc71bd)。这是一个伟大的使用 [Jupyter](https://hackaday.com/2019/02/22/drops-of-jupyter-notebooks-how-to-keep-notes-in-the-information-age/) 。至于愚弄面部识别，也许[这个项目需要更多的工作](https://hackaday.com/2019/10/31/be-anyone-or-anything-with-facial-projection-mask/)。
