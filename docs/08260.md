# 开源乐高控制器

> 原文:[https://hack aday . com/2020/11/05/open-source-Lego-controller/](https://hackaday.com/2020/11/05/open-source-lego-controller/)

泰勒·科林斯(Tyler Collins)白天是一名机械和制造工程师，他在业余时间自学电子和固件开发，并创造了一个名为*evl no One*的[开源乐高控制器](https://www.kickstarter.com/projects/spacecadet/evlono-one)。它基于 STM32 和 Arduino 生态系统，兼容现有的各种乐高控制器、传感器和执行器。[Tyler]在帮助一个课后项目时遇到了乐高思维风暴，并开始想知道他是否可以制作一个更灵活的控制器。我们不得不说他成功了，令人惊讶的是他在这个 4 x 4 的单高砖块格式中装入了多少东西。

Evlō no One 基于 ESP32 双核 MCU，具有 WiFi、蓝牙和用于无线连接的红外发射器。它还拥有 USB-C 电源传输，三个电机控制器，扬声器，led 和一个按钮。深入 Kickstarted 页面，了解关于这些接口和规范的更多详细信息。固件和硬件都将在 GitHub 上开源发布。

虽然[Tyler]的原型都在运行，但他指出这是他的第一次大制作。FCC 认证测试和生产模具是推动 2021 年 2 月出货的两个最大项目。如果计算机驱动的乐高模型是你的爱好之一，一定要看看[泰勒]的项目。如果你错过了我们[Daniel Pikora]关于乐高和开源的交集(碰撞)的 FOSSCON 2018 演讲，[我们的文章](https://hackaday.com/2018/09/07/fosscon-2018-where-open-source-and-lego-collide/)是乐高(AFOL)社区成年粉丝的必读之作。
