# slippy slapper 无用地使用所有 arduino

> 原文:[https://hack aday . com/2020/03/13/slippy-slapper-useless-uses-all-the-arduinos/](https://hackaday.com/2020/03/13/slippy-slapper-uselessly-uses-all-the-arduinos/)

想让那个烦人的高效同事降一级吗？是啊，我们也是。他们怎么敢做这么多事，还这么开心？当你努力整理一封电子邮件时，他们怎么可能整天敲键盘呢？

Slippy Slapper 是一个无用的机器，它使用无休止的分心和轻微的身体暴力等策略将人变成无用的机器。它通过让他们毫无理由地按下按钮来按下你的按钮，而不是杀死他们的生产力。当他们想走开的时候，你猜怎么着？那是另一个闹剧。如果滑溜溜的小子不那么可爱的话，他会通过代理激怒我们。

你说得对，你不需要 Arduino 来做这个。对于峰值低效率和功耗，实际上需要四个。其中一个充当主人，并根据它从 Slippy 的超声波鼻孔获得的反馈向其他三个发出命令。另外三个控制 slappin 的伺服系统、扬声器和从 SD 卡上读取 WAV 文件。拍打你的方式过去的休息，看看 Slippy 斯莱普的闹剧演示。

需要同时惹恼一群同事吗？当会议室正在安装时，把一大堆无用的机器偷偷放进会议室。
