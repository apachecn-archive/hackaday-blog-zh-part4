# 任天堂 64 通过游戏鲨鱼自制

> 原文:[https://hackaday . com/2019/01/11/Nintendo-64-home brew-via-game-shark/](https://hackaday.com/2019/01/11/nintendo-64-homebrew-via-game-shark/)

任天堂 64 现在是一款经典的游戏机，很受欢迎，尽管在商业利益上输给了索尼主导的 PlayStation。与 NES 和 Game Boy 等平台相比，它在自制软件和黑客领域并不总是受到那么多关注。这意味着使用控制台所需的工具并不广为人知。然而，如果你有合适的硬件，有一个非常简单的方法可以将自制软件加载到任天堂 64 上。

要实现这一点，你需要一个 N64 Gameshark，尤其是高于 3.0 的版本。这些包括一个并行端口和相关的板载逻辑，以允许控制台从连接的计算机接收数据和命令。[Nathan]演示了使用 gs_libusb 实用程序将自制代码传送到控制台，使用 usb 转并行适配器使其在现代计算机上变得容易。

如果你想亲自尝试一下，可以在 github 上找到这些工具。我们看到的平台[的大多数黑客都是便携式的](https://hackaday.com/2013/10/27/aluminum-unibody-nintendo64/)，但是如果你有新的东西，[你知道该找谁。](https://hackaday.com/submit-a-tip/)
