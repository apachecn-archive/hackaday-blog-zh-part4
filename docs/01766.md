# 用木头制作的眼镜框

> 原文:[https://hack aday . com/2019/01/09/glasses-frames-crafted-out-of-wood/](https://hackaday.com/2019/01/09/glasses-frames-crafted-out-of-wood/)

市场上的大多数眼镜和太阳镜都使用金属或塑料框架。创建各种有趣的框架几何形状相对容易，安装光学元件的公差可以很容易地控制，并且它们是可以承受日常使用的坚固材料。木材在所有这些方面都有所欠缺，但这并不意味着你不能用它来制作一副漂亮的眼镜。

ZYLO 是一家制作木制眼镜的公司，这段来自[Paide]的视频详细展示了制作过程。现代工具被用来尽可能提高工作效率。零件经过激光切割和雕刻，形成镜架的主要部分和镜腿(位于耳朵上方的手臂，用于将耳朵固定在脸上)。在进行进一步组装之前，使用特殊的夹具在层压木材部件上赋予曲线。像大多数其他现代太阳镜一样，金属预制铰链和螺钉用于将所有东西栓在一起，但这涉及到大量的手工加工，包括精致的镶嵌和突出的标志特征。

相比之下， [Manuel Arroyave 在制作 Cedoro 眼镜](https://www.youtube.com/watch?v=IYwOsvOb19M)时表现得非常不同。首先将板材层压在一起，然后用特殊的水平轴铣削装置加工出大致的形状。即使是像铰链这样的小细节，也是用木头精心手工制作的，并配有微小的木榫。

这表明完成一项工作总有不止一种方法。我们很想拿出激光切割机，自己动手制作一些定制的色调。也许，你太累了，不能自己戴上你的太阳镜？没关系，这也有解决办法。休息后的视频。
