# 可调活塞阻尼锤

> 原文：<https://hackaday.com/2021/04/19/adjustable-piston-damped-hammer/>

当你只有一把锤子时，每个问题都是对更好的锤子的不断追求，正如流行的说法。至少，这似乎是[Ebenisterieélose]的情况。她想要一把无差速锤,不仅要有美观的木质和黄铜结构，还要有可调节的减震器，以确保每次锤子摆动都尽可能高效。

对于那些不熟悉专用锤的人来说，死锤通常在锤头内有一些可移动的物质，如沙子或铅弹。当锤撞击物体时，质量向前移动，减少锤从物体上反弹，并将更多能量转移到每次撞击中。这个锤子省略了一个被动质量，有利于四个定制加工的铜管，每个都装有加重流体，弹簧和黄铜重量。每个活塞都以类似于车辆减震器的方式充当阻尼器，每个活塞顶部的螺钉和 o 形环允许根据需要添加不同重量的流体来调整它们。活塞的一些详细测试也显示出比任何被动质量种类都有显著的改进。

这不仅对于一个经常以不精确的方式使用的工具(至少对于我们这些不是熟练工匠的人来说)来说是令人难以置信的细节和精度，而且它由木头、皮革和黄铜制成，这使它的外观和感觉优于塑料和玻璃纤维锤子，这是最现代的死锤的典型特征。单就工艺而言，它甚至可以凭借其复杂的定制雕刻与这款[工程师锤相媲美。](https://hackaday.com/2019/03/05/making-a-hammer-with-beautiful-engravings/)
