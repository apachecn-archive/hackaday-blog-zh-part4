# [弗兰]今年在帮圣诞老人杀人

> 原文:[https://hack aday . com/2019/12/22/Fran-is-helping-Santa-slay-this-year/](https://hackaday.com/2019/12/22/fran-is-helping-santa-slay-this-year/)

我们知道至少有一个人应该在今年圣诞老人的“好”名单上。[Fran]可能已经快成功了，但是给圣诞老人寄一张手工制作的贺卡，里面有一个功能齐全的吉他放大器可能会让他用闪光笔或银色笔重写她的名字。

[![](../Images/c9596a0278429f67ec9ce9726799bb7f.png)](https://hackaday.com/wp-content/uploads/2019/12/card-amp-guts.png) 这款填充料大小的放大器基于 LM386 和使其摇滚所需的最少组件。一切都是死虫焊接，夹在两片卡股票。第一个版本有一个 386 听起来不错，但[Fran]希望声音更大，所以她增加了另一个阶段，有第二个 386。[Fran]将扬声器的边缘直接粘在卡片上，这样它就可以像一个圆锥体一样，发出比扬声器本身更好的声音。

圣诞老人需要摇动的只是他的斧头和一个由 1/4”插孔和 9 伏电线连接到插在卡上的 3 针插头的小接口。他可以在圣诞音乐中休息一会儿，让那些饼干在他演奏的时候消化一下。如果你想远离“淘气”名单，请务必在休息后查看构建视频。

想制作自己的音乐贺卡吗？如果你能设计一个 ATtiny85，[你不需要更多就能发送一个微笑](https://hackaday.com/2011/11/22/musical-greeting-card-with-minimal-parts/)。如果视觉艺术更适合你， [3D 打印一张 2D 的照片](https://hackaday.com/2016/12/20/3d-printed-greeting-cards/)。
