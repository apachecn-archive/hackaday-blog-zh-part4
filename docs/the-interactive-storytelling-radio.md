# 互动讲故事电台

> 原文：<https://hackaday.com/2020/09/13/the-interactive-storytelling-radio/>

[8BitsAndAByte]回来了，这次他们使用人工智能创造了一个互动的讲故事者。在树莓皮的帮助下，他们升级了他们挖出的一台旧的冷战时期的收音机，结果令人印象深刻。

收音机的主控制板完好无损，因此很容易使用所有预先存在的硬件来控制扬声器，并使用收音机前面板上的按钮和开关来触发一些 Pi 的 GPIO。为了增加一些人工智能，他们使用了[谷歌的 AIY 语音工具包](https://hackaday.com/2018/04/26/google-lowers-the-artificial-intelligence-bar-with-complete-diy-kits/)，允许他们接入谷歌的[看似无穷无尽的人工智能平台](https://hackaday.com/2018/06/21/make-a-natural-language-phone-bot-like-googles-duplex-ai/)。这可能是一个“风水轮流转的时刻”，但我们可能有点过于乐观了。

无论如何，他们使用了一个非常有趣的叫做 Dialogflow 的软件，它创造了一个有点自然的对话互动，类似于一个聊天框。如您所料，Dialogflow 将语音转换为文本，但也可以解释上下文语音并提供上下文响应。相当整洁… [但也可能有点令人毛骨悚然](https://hackaday.com/2017/10/24/listen-to-the-netherworld-with-artificial-intelligence/)。谁知道呢？[陪审团还没有出来](https://hackaday.com/2017/02/13/the-future-of-artificial-intelligence/)。

无论如何，如果你像我们一样，有时需要脱离人类，那么这个项目可能正适合你。
