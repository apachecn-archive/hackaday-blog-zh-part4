# Commodore TED 系列的最新数据集

> 原文:[https://hack aday . com/2020/10/12/retro-dataset-for-commodore-ted-series/](https://hackaday.com/2020/10/12/retro-datasette-for-commodore-ted-series/)

复古电脑爱好者[Steven Combs]记录了他建造 te duino 的冒险经历，te duino 是 Commodore Datasette 的现代替代品，它使用 SD 卡而不是录音带。他的设计基于【彼得·爱德华兹】的[塔普伊诺项目](https://hackaday.io/project/1985-tapuino-the-20-c64-tape-emulator)，该项目在 2014 年由[黑客日推出。[Steven]将美学设计提升到了一个新的水平，并对其进行了修改，以与他的 Plus/4 和其他 TED 系列 Commodores 配合使用。我们很惊讶他能够在 SketchUp 中设计这个围栏，并且对他的创意 3 的结果印象深刻。他竭尽全力去搭配 Plus/4 的颜色和款式，而且做得相当不错。[Steven]还在这个外壳中应用了一些有趣的设计功能。PCB 模块是搭扣配合的，按钮是作为一个整体制成的——与活动铰链没有什么不同。电缆的 3D 打印应力消除是一个很好的点睛之笔，我们不能不同意[Steven]的明智建议——“大猩猩任何东西都很酷”。](https://hackaday.com/2014/12/12/hacklet-26-arduino-projects/)

这只是该项目的第一部分。敬请关注未来的改进、调整和修饰。
