# 打印您的键盘稳定性之路

> 原文：<https://hackaday.com/2020/08/20/print-your-way-to-keyboard-stability/>

键盘按键稳定器，或者发烧友圈子里所谓的 stabs，完全符合你的预期——它们稳定较长的按键，如 Shifts 和空格键，这样它们就不必被击到正中心才能均匀地启动。Stabs 的工作原理是在钥匙开关的侧面安装两个非功能性开关致动器，用一根粗线棒连接。有些人喜欢针刺，并坚持稳定每一个大于 1u 的按键，而其他人则认为针刺会带来更多麻烦，因为各种原因，如卡嗒声。

[![](img/8eb97abcbb0c53567d68f48483458f39.png)](https://hackaday.com/wp-content/uploads/2020/08/stab-magnet.png) 【有风险的 3D 打印】一直在研究[一种用于 Cherry MX 瓶盖的参数化可打印稳定系统，该系统使用小型磁盘磁铁](https://www.youtube.com/watch?v=FFjcZLgZK2g)来保持电线的位置。正如你在视频中看到的(插播在休息之后)，结果是一个清脆的噼啪声，没有格格作响。磁铁能稳定电线，所以它能很好地弹回。

虽然打印很容易，但[Riskable]表示，设计过程并不像看上去那样千篇一律。稳定器杆的中心点不应该在切口的中心，即使肉眼看起来是这样。在此之后，痛点转移到了电线上，在进行必要的弯曲之前，尽可能使其变直。[Riskable]将制作一个矫直器来帮助解决这个问题，[我们建议像这样的东西](https://hackaday.com/2019/09/28/arduino-wire-bender-probably-wont-kill-all-humans/)。

Clacker hacking 相当于兔子洞，尤其是在与 3D 打印相结合的时候。[我们最近看到了一个完全 3D 打印的宏垫、弹簧和所有的东西](https://hackaday.com/2020/07/09/3d-printing-a-macro-pad-switches-and-all/)。
