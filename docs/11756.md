# 带着人工肌肉机械臂漫步神秘谷

> 原文:[https://hack aday . com/2021/10/28/带着人工肌肉机械臂漫步神秘山谷/](https://hackaday.com/2021/10/28/taking-a-stroll-down-uncanny-valley-with-the-artificial-muscle-robotic-arm/)

维基百科上说“恐怖谷假说预测，一个看起来像人类的实体会有在观众中引发冰冷、怪异感觉的风险。”是的，我们必须承认，尽管难以置信，但看到[自动机器人]的手和前臂以几乎人类的方式移动，有点令人不安。不要只相信我们的话，让你自己被休息时间下面的[视频所吸引和迷惑。](https://www.youtube.com/watch?v=guDIwspRGJ8)

虽然人造肌肉机器人手臂的创造者对其工作原理相当保密，但仔细阅读 YouTube 频道的确会对此有所了解。手臂和手的运动是由人造肌肉实现的，人造肌肉本身由加压到 130 磅/平方英寸(9 巴)的水赋予生命。肌肉本身似乎是水密的纤维编织，但这些细节没有提供。柔性钢丝网里的膀胱，就像手指陷阱？

[Automaton Robotics]的目标是最终利用他们的人工肌肉技术创造一个人形机器人。展示的演示令人印象深刻，因为手有力量举起 7 公斤(15.6 磅)的哑铃，即使它的一些最强的人工肌肉尚未安装。

几年前，我们[发表了一篇关于人工肌肉](https://hackaday.com/2016/06/06/artificial-muscles-to-bring-relief-to-robotic-tenseness/)的文章，其中提到了当施加气压时会收缩的气动人工肌肉，似乎[自动机器人]已经采用了相同的方法来代替水。你有什么想法？请在下面的评论中告诉我们。此外，感谢[穿苏格兰短裙的瑞典人]给了我这个好建议！请务必[也发送您自己的提示](https://hackaday.com/submit-a-tip/)！
