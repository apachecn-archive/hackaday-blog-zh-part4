# 为过去的平台设计的门户端口

> 原文：<https://hackaday.com/2018/12/05/a-portal-port-programmed-for-platforms-of-the-past/>

如果你还有一台 Commodore 64，而且它正在积灰，现在还不要把它卖给易贝的收藏家。仍然有一些自制游戏开发发生在致力于这个经典系统的一小组程序员身上。最新的是一款由杰米·富勒开发的类似于门户网站的游戏，看起来很棒。

Commodore 没有与 Playstation 相同的规格，但没有理由不玩这个版本。它有相同风格的谜题，玩家需要射击入口和操纵物体以达到目标。GLaDOS 甚至露面了。[Del Seymour]的图形和[Roy Widding]的音乐也将硬件推向了极限。

如果你没有 C64，有一些模拟器可用，如 VICE，可以让你玩这个游戏，而不必找到一台 80 年代的工作电脑。如果你真的很专注，你也可以[构建自己的模拟器](https://hackaday.com/2018/03/28/emulating-a-complete-commodore-64/)，或者[恢复一个已经积满灰尘的](https://hackaday.com/2017/12/21/can-this-commodore-64-be-saved/)。最后，我们知道严格来说，它不是一个门户网站，但有时可以在标题中使用一些艺术许可。
