# Twitch 玩战斗机器人？

> 原文：<https://hackaday.com/2018/08/08/twitch-plays-battling-robots/>

随着 *Twitch Plays Pokemon* ，观众互动在 2014 年达到了历史新高，这是一个在线游戏流，观众可以协作指挥一个模拟的游戏男孩玩口袋妖怪 Red。从那以后，这个概念开始流行。今天，我们看到这扩展到现实世界中的机器人，[与【theotherlonestar】的 Twitch 聊天控制的机器人。](https://hackaday.io/project/159572-twitch-chat-controlled-robot)

该建筑利用了现代现成的组件 ESP8266 提供了大脑，而 Pololu Zumo 提供了现成的机器人底盘，以节省建筑机械方面的时间。L298N 双电机控制器则处理动力。

然而，真正的独创性在于教会机器人对来自 Twitch chat 的命令做出反应。聊天以易于解析的 IRC 格式提供，这使得围绕它的编程变得容易。[theothelonestar]创建了一个命令集，使机器人能够被流媒体观众远程驱动，然后给机器人配备了锤子和一顶软呢帽，如果有人愿意的话。

这是一个很酷的构建，随着 Twitch 继续减少流和聊天延迟，它显示了进一步的前景。我们期待看到未来的战斗，但是第一场已经让我们兴奋了。
