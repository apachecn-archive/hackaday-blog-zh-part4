# 探索德州仪器被遗忘的 CPU

> 原文：<https://hackaday.com/2022/09/23/exploring-texas-instruments-forgotten-cpu/>

你通常不会听到德州仪器这个名字与第一个微处理器联系在一起。但是 TI TMX 1795 是 8008 芯片，比 8008 早生产几个月。不过，它从未上市，所以大部分人都已经忘记了它。[而不是【肯·希尔里夫】](https://www.righto.com/2015/05/the-texas-instruments-tmx-1795-first.html)。你也可以在下面的视频中看到该设备 2015 年的演示。

芯片具有相同架构的原因是，它们是为了取代 Datapoint 2200 可编程终端内的相同大型电路板而构建的。这些庞然大物可以用 BASIC 或 PL/B 编程。

由于加热问题，Datapoint 要求英特尔将电路板缩小为芯片——但在延迟后，他们反而更换了电源，并对该设备失去了兴趣。我听说了这件事，想参与这笔交易。然而，Datapoint 并不为所动。该芯片不能很好地承受电压波动，因为他们已经更换了电源，并采用了比芯片更快的新 CPU 设计。他们也不关心要得到一个完整的系统需要添加多少东西。

那么，为什么英特尔 8008 在市场上行得通，而 TI 芯片却不行呢？毕竟，Datapoint 也决定不使用 8008。但正如[Ken]指出的那样，8008 比 TI 芯片小得多，因此生产成本更低。

像往常一样，[Ken]的帖子总是很有趣，很有启发性。他看过很多旧电脑。他甚至钻研老的[太空硬件](https://hackaday.com/2021/06/19/apollo-shift-register-is-discrete/)。很棒的东西！
