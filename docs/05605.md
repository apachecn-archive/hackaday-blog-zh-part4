# 旧的旋转式电话投入使用

> 原文：<https://hackaday.com/2020/02/04/old-rotary-phone-gets-called-into-action/>

我们用指尖敲击的玻璃越多，我们就越怀念像旋转式电话这样有趣的物理界面。当然，他们花了很长时间拨号，你不想成为那些在过渡到 DTMF 期间被卡在其中的孩子之一，特别是如果你试图成为广播电台的第九个呼叫者，但这一切的坚实机电体验只是很酷，好吗？声音和重量让他们看起来很成熟。

[Tal O]明白了。他几乎已经把这个老姑娘带进了 21 世纪，却没有给她留下任何东西。里面有一些你所期望的东西，如用于电话部分的 SIM800 GSM 模块，以及用于计数拨号器脉冲并在它和 GSM 模块之间通信的 ESP32。但它也有一些我们以前没见过的东西。整个旅程在一个五部分的视频系列中概述，休息后我们将为您拨入第一部分。

虽然[Tal]让响铃器工作以证明这是可行的，但他不希望有一个单独的 12V 电路来运行铃声。此外，铃铛和它们的电磁铁占据了很大的空间，所以他妥协了，用旋转铃声的 mp3。[Tal]还希望有一种方法可以获得已拨号码的反馈，而不需要切开电话来添加屏幕，所以他找到了一个文本到语音库，让电话一拨号就大声说出每个号码。它使用相同的内部扬声器作为铃声，但我们认为如果反馈通过手机扬声器来进行会更好。

如果[Tal]正在为这款手机寻找另一种现代便利，那么[快速拨号](https://hackaday.com/2019/09/24/finally-a-rotary-cell-phone-with-speed-dial/)如何？
