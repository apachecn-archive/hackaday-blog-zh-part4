# 水坝爆破，遥控式

> 原文：<https://hackaday.com/2019/11/23/dambusting-r-c-style/>

声明:在制作下面的视频时，没有水坝被破坏。当然，这并不意味着向二战大坝爆破者和他们的“弹跳炸弹”致敬的比例模型不值得做。

在一场充斥着黑客的战争中，[巴恩斯·沃利斯]的弹跳炸弹概念可能是最黑客的。在下面的视频中，[汤姆·斯坦顿]解释说，[沃利斯]在跳过水面上的弹珠后，想到了跳过湖面上的炸弹来摧毁敌人的基础设施。他使用桶状炸弹建造了一个装置，可以给它们适当的旋转力，并在适当的时候释放它们，让它们在轰炸机逃跑时跳过湖面。一旦击中大坝的边缘，炸弹会下沉到底部附近爆炸，造成最大的破坏。

[汤姆]的比例钻机最终成为一个聪明的设计，装有弹簧的手臂在由无刷电机旋转后释放 3D 打印的桶。他与 R/C 建造者[詹姆斯·沃姆斯利]合作，后者设计出了一架很棒的泡沫纸板兰开斯特轰炸机，就像英国皇家空军第 617 中队使用的一样。在他们选择进行测试的平静的天气和平静的湖面上，R/C Lanc 在释放第一枚桶装炸弹之前进行了几次测试。第一次运行有点太陡，导致炸弹只是潜入水中没有跳过。技术问题和紧急着陆挫败了第二次飞行，但第三次飞行非常完美——炸弹跳过三次，而飞机优雅地倾斜离开。[汤姆]还尝试了重型四轴飞行器和炸弹装置，这是[巴恩斯·沃利斯]以前想都不敢想的。

向[Tom]和[James]致敬，感谢他们在这个项目上的合作和让跳绳工作起来。这让我们想起了设计好的跳岩方法，尽管没有那么致命的意图。
