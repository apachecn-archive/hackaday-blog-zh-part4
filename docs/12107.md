# OpenDog 第三版已经准备好了

> 原文:[https://hack aday . com/2021/12/18/open dog-version-3-is-ready-to-go-walkies/](https://hackaday.com/2021/12/18/opendog-version-3-is-ready-to-go-walkies/)

我们关注[詹姆斯·布鲁顿]的开放狗项目已经有一段时间了，随着他的工作节奏加快——不管是疫情还是疫情——发展速度惊人。最新的里程碑是版本 3 【视频，嵌入下方】的[公开发布。)此次系统升级增加了 3D 打印摆线齿轮箱，取消了之前的皮带传动。[詹姆斯]有巨大的乐趣调整这些电机控制器参数，并承认他们还没有完全拨入。他指出，更宽的变速箱体意味着机器人的几何形状需要改变一点，以前的皮带传动版本可能有一点优势，但他有信心他可以做到这一点(鉴于他以前令人难以置信的机器人制造，我们完全相信他会搞定它！)](https://youtu.be/eKZIJwJBjEs)

![](../Images/d781b4d097ba06fae2c5d0024021664a.png)

Silicone overmolding around a 3D printed former, using a 3D printed mould

旧版本与光滑的塑料脚斗争；圆形足的可预测平滑接触形状的优势在某种程度上被有限的接触面积大小抵消，这意味着在某些表面上没有太多的抓地力。[James]解决方案显而易见——只需学习如何制作 3D 打印硅胶模具，并在塑料模具周围浇铸一个漂亮的橡胶脚，问题就解决了！不幸的是，他忽略了添加一些凹槽，以便在模具半模之间插入杠杆，所以固化后要分开有点困难。我们确信这是一个不会再犯的初学者错误。

openDogV3 的完整源代码现在可以在 GitHub 页面上找到。这里是整个项目的[播放列表](https://www.youtube.com/playlist?list=PLpwJoq86vov8uTgd8_WNgBHFpDYemO-OJ)，以及摆线传动开发的直接链接( [part1](https://www.youtube.com/watch?v=pWMB5VbLb6w) 、 [part2](https://www.youtube.com/watch?v=tgEOpl880KM) 、 [part3](https://www.youtube.com/watch?v=IVpYtyS5Q-k) )。)但在你们开始 3D 打印自己的狗狗之前，[詹姆斯]告诉我们总成本将在 2000 美元左右，其中大部分是电机和 ODrive 单元，所以这一款只适合认真的建设者！

我们已经介绍了相当多的机器狗，一个特别好的例子是[眩晕的狼](https://hackaday.com/2019/03/30/a-pet-robot-just-like-boston-dynamics-makes/)，如果你想知道你到底为什么想要一只机器狗，那么[问 Hackaday](https://hackaday.com/2019/09/25/ask-hackaday-what-good-is-a-robot-dog/) 你也介绍了。
