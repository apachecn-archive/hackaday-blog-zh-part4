# 2019 年在 PDP-7 上运行的 UNIX 版本 0

> 原文:[https://hack aday . com/2019/11/17/UNIX-version-0-running-on-a-PDP-7-in-2019/](https://hackaday.com/2019/11/17/unix-version-0-running-on-a-pdp-7-in-2019/)

随着 UNIX 操作系统的 50 岁生日出现在最近的新闻中，人们开始关注它最早的起源。在西雅图的活体计算机博物馆，他们已经超越了一点历史调查，因为他们已经有了 unix(或者我们在这种情况下应该说 UNIX？)在 DEC PDP-7 小型计算机上运行的版本 0。在原始硬件上的这个原始版本更加引人注目，因为与它的兄弟姐妹不同，很少有 PDP-7 幸存下来。

运行 UNIX 版本 0 的机器属于[Fred Yearian]，他是一名前波音工程师，在 20 世纪 70 年代末从公司的盈余渠道购买了他的机器。他让它恢复了工作状态，它在他的地下室里放了几十年，而老式计算机界一直认为，包括博物馆现有的机器在内，只有四台幸存下来——其中只有一台能用。因此，[Fred]带着一台可能正常工作的第五台机器出人意料地出现，让人有些吃惊。

为了加载操作系统，一个磁盘模拟器被连接到机器上，这可能是几十年来第一次编写一个新的 UNIX 版本 0 设备驱动程序来使它能够被使用。第一个登录是用户“dmr”，这是对 UNIX 的共同创造者 Dennis M. Ritchie 的敬意。
