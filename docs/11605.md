# 3D 打印机切割金属

> 原文:[https://hackaday.com/2021/10/22/3d-printer-cuts-metal/](https://hackaday.com/2021/10/22/3d-printer-cuts-metal/)

我们不时会看到 3D 打印机，它可以用混凝土打印出整个房子，或者用金属打印出整个火箭。但通常情况下，出于我们预算友好的爱好者需求，我们大多数 3D 打印机将打印小塑料零件。不过，如果你有耐心和一点点盐水，看看这个 3D 打印机，它已经被修改过，可以从任何类型的金属上切割零件，由[Morlock]制造，他已经把[打印机变成了 5 轴数控机器](https://www.youtube.com/watch?v=KieJN-J4s38)。

当然，这种修改不是 3D 打印金属。它将 3D 打印机的 CNC 功能转化为使用电化学加工(ECM)的加工工具。这一过程通过在盐水中使电极通过金属来快速腐蚀金属，从而从工件上去除金属。这是一种非常精确的切割金属的方法，不需要昂贵或笨重的加工工具，这些工具使用可以轻松 3D 打印或容易获得的零件。通过使用 3D 打印机轴并将打印床修改为耐盐水，可以切割高达 3 毫米的金属零件，而不管所用的金属类型。[Morlock]还为切割工具增加了两个额外的轴，允许它以奇怪的角度切割金属。

像这样使用 3D 打印机来执行 CNC 加工是一种获得机床性能的极好方法，而不需要花费一台机床的费用。当然，它需要对 3D 打印机进行一些重大修改，但它不需要标准 CNC 机器所需的强度和刚度，以便在可接受的公差内获得零件。如果你有兴趣使用更传统的方法来引导一个这样的机器，我们最近推出了一款数控机器[，它可以用普通材料制成，以最低的成本组装在一起](https://hackaday.com/2021/08/19/beginning-the-machine-shop-journey-with-a-diy-cnc/)。
