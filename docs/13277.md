# Raspberry Pi 测试台会告诉您使用哪种胶水

> 原文:[https://hack aday . com/2022/04/07/raspberry-pi-test-stand-tell-you-that-glues-to-use/](https://hackaday.com/2022/04/07/raspberry-pi-test-stand-tells-you-which-glues-to-use/)

不是所有的胶水都是一样的；或者说，并不是每种胶水都适合每种应用。但是如何知道哪种胶水用在哪种接头上呢？这个问题的答案并不总是很清楚，但是关于不同胶水的相对强度的确切数字是一个很好的起点。

要量化通常可能有点主观的过程，可能没有人比木工和黑客【马蒂亚斯·万德尔】更好了，因为他配备了[他的 DIY 力量测试仪](https://hackaday.com/2021/11/22/shop-built-rig-measure-strength-of-wood-accurately/)。利用其步进驱动力来炸开胶合搭接接头，[马提亚斯]使用连接到树莓 Pi 的应变仪来测量各种粘合剂的屈服点。

他的第一轮测试有一些有趣的结果，包括通常被吹嘘的建筑粘合剂最终出现在一个遥远的地方。基于聚氨酯的大猩猩胶也表现不佳，至少相对于其声誉和可能造成的混乱而言。整体强度中令人惊讶的突出是热熔胶，尽管它似乎有一种塑性屈服模式。曾经细心的经验主义者，[马提亚斯]重复了他的测试使用硬木，有明显不同的结果；看起来胶水在密度更大的木材上表现得更好。他还重复了一些测试，以确保每种粘合剂都得到公平的对待。看看下面的视频，看看最终的结果。

看到这样的实验来检验我们通常认为理所当然的东西总是好的。YouTube 上的 Project Farm 频道的约翰也做这种事情，甚至做了环氧粘合剂的面对面测试。
