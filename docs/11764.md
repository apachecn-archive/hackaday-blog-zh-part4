# 改进的拨动开关优雅超详细的驾驶舱模拟器面板

> 原文：<https://hackaday.com/2021/10/30/modified-toggle-switches-grace-hyper-detailed-cockpit-simulator-panels/>

在驾驶舱模拟器爱好者的世界里，没有一个细节是小到不能忽略的。让每一个驾驶舱控制的外观和感觉都恰到好处是很重要的，这通常意味着要为驾驶舱精确的部件买单。但并非总是如此，正如[这些 DIY 磁性捕捉拨动开关](https://www.youtube.com/watch?v=OCmDc2_VuGI)所示。

你很有可能已经看过[疣猪计划]的极其详细的 A-10 雷电 II 驾驶舱模拟器；[我们最近对它进行了特别报道](https://hackaday.com/2021/09/30/the-ultimate-brrrt-simulator-fully-featured-a-10-warthog-cockpit/)，来自正在进行的构建的视频定期出现在我们的提要中。sim 解决了最微小的细节，包括使用特殊的拨动开关，利用电磁铁自动锁定到位。它们在市场上有售，但只针对那些非常有钱的人——取决于供应商，每台高达几千美元*！*

幸运的是，自制替代品既便宜又容易制造——一个瞬时 DPST 拨动开关被部分挖空，用一根钉子代替了它的一个电极。钉子从开关的背面伸出，那里有一个支架支撑着一个小电磁铁。通电后，当开关打开时，电磁铁牢牢抓住钉子；模拟飞行员仍然可以手动关闭开关，或者通过线圈断电自动释放。每个开关的制造成本不到 20 美元，包括驱动线圈所需的 MOSFETs 和提供逻辑的 Arduino。他们装饰的面板看起来很棒，开关增加了功能细节，正好适合整个建筑。
