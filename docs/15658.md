# 差动转弯驱动具有很高的机动性

> 原文：<https://hackaday.com/2022/12/17/differential-swerve-drive-is-highly-maneuverable/>

有各种各样的轮子设计可以提供不同方向的旋转和平移。然而，正如[WildWillyRobots]所展示的那样，差速转向驱动在一个复杂的支架上使用常规车轮[来实现令人印象深刻的方向灵活性。](https://www.youtube.com/watch?v=TP8wQRZUuk4)

该设计使用一个安装在车轴上的规则圆形车轮，车轴一端有一个齿轮。这允许车轮被驱动。车轮和车轴安装在一个圆形托架上，然后在轴承上安装一对环绕的齿轮。差动驱动这些齿轮改变了驱动器的行为方式。当两个齿轮以相同的方向驱动时，车轮绕其垂直轴旋转，指向不同的方向。如果两个齿轮以相反的方向驱动，则车轮本身被驱动。相对改变两个齿轮的速度允许控制车轮的方向和驱动。其结果是一个轮子可以旋转到任何角度，然后被向前或向后驱动。

给机器人安装一套这样的轮子，就创造了一个高度可操作的平台。另外，它没有各种全轮设计常见的抓地力差的缺点。
