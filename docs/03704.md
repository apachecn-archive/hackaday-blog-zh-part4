# 3D 打印机遇上 CNC 路由器制作木版画

> 原文：<https://hackaday.com/2019/07/25/3d-printer-meets-cnc-router-to-make-wood-prints/>

我们以前见过很多胶合板 3D 打印机；毕竟，许多早期的业余机器都是由激光切割的胶合板制成的。但是[这台胶合板 3D 打印机](https://shane.engineer/blog/plywood-3d-printer)不是*用胶合板制作的*—*打印的是*胶合板。嗯，算是吧。

是的，我们知道——印刷商使用的不是胶合板，而是刨花板，这种材料也是宜家所有商场的平板包装仓库所用的材料。称它为打印机也有点夸张。这个由[Shane Whigton]和他的 Formlabs Hackathon 团队创造的东西，更像是一台混合加减型数控机器。一个安装在机架上的刳刨机从一块新的方形材料上雕刻出每一层印刷品，这块材料可以是胶合板，也可以是刨花板。一旦切好一层，起重机就在上面涂上胶水，在上面放上一层新材料，然后夹紧。然后路由器切割下一层，依此类推。层的高度受限于材料的厚度——在这种情况下是 3/4 英寸(19 毫米)——而且有相当多的浪费，但这不是重点。在下面的视频中，看看正在运行的打印机和由此产生的巨型板凳。

看到刨花板上的灰尘和胶水让我们想到:用 3D 打印机挤出锯屑和胶水混合的糊状物怎么样？我们认为这有点像那些挤压混凝土建造房屋的巨型打印机。
