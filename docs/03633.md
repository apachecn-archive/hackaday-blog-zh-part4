# DropController 为文档设置了标杆

> 原文：<https://hackaday.com/2020/08/18/dropcontroller-sets-the-bar-for-documentation/>

dropController 拥有那种我们希望在我们构建某个东西的时候自动生成的文档。[Martyn Currey]为水滴摄影建立了一个强大的平台，我们不想忽视硬件，但最令人印象深刻的部分可能是网站。这可能不是很花哨，但它是彻底的和逻辑组织。您可以找到零件列表、装配手册、教程、草图和原理图。要是我们遇到的所有项目都这么详细就好了。

水滴摄影非常酷，尽管徒手拍摄会让你的耐心下降到 9.81 米/秒以上。这个概念是一个电磁阀将闪烁打开以释放一滴水，等待一定数量的微秒，然后通过有线遥控电缆触发你的 DSLR。棘手的部分来自控制多达六个阀门和三个闪光灯。我们没有足够的手指和脚趾去按所有那些按钮。

物料清单包含许多常见的部件，如 Arduino Nano、LM2596 稳压器、一些 MOSFETS、HC-06 蓝牙模块，以及连接一切的标准音频连接器。没有什么会让银行破产，但如果钱不是问题，[马丁]销售套件和完整的单位。

水滴控制器并不是最新的产品，而[闪光灯摄影](https://hackaday.com/2015/10/05/frozen-time-photography-with-a-100w-led/)是由来已久的传统。
