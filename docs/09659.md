# 滑移转向机器人底盘受到冲击

> 原文：<https://hackaday.com/2021/03/26/skid-steer-robot-chassis-takes-a-beating/>

[下班后的工程]想要一个具有小转弯半径的高机动性机器人底盘。打滑转向似乎是完美的解决方案，但可用的商业选项并不符合他的喜好。因此，定制构建就是答案—[带来令人印象深刻的结果](https://www.youtube.com/watch?v=63go-gQNn5o)。

该建筑装有两个大型钢筋混凝土电机，每侧一个，每个电机通过皮带驱动两个车轮。这将滑移转向所需的电子设备减少到最低限度。它全部装配在等离子切割的金属底盘内，这种底盘足够坚固，可以承受一些猛烈的撞击。

建造的主要目标之一是消除振动和冲击损坏电机和齿轮箱的风险。许多现成的设计将车轮直接连接到变速箱输出轴上，随着时间的推移，可能会损坏昂贵的部件。在这种设计中，一个单独的轴承组件被用来代替车轮承受负载。

这是一个很好的例子，说明了工程第一的方法如何能以最少的忙乱制造出一个坚固的“机器人”。配备了一些脂肪越野轮胎的性能令人印象深刻，机器人在泥，雪和水中毫不费力地撕裂它。
