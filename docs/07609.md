# 脑瘫工具助手

> 原文:[https://hack aday . com/2020/09/03/脑瘫-工具-助手/](https://hackaday.com/2020/09/03/cerebral-palsy-tool-assistant/)

我们都应该创造。有些人似乎把灵感藏在口袋里，但当他们以自己选择的艺术形式表达自己时，每个人都会受益。我们每个人都有工具，从 Dremels 到画笔，这里的许多人都有自己的工具。即使我们没有支持宏功能的机械键盘或双面挤压打印机，我们也能应付。但是如果你不能操作你的钻子，或者鼠标，或者甚至是一支铅笔呢？对我们来说，这将是极其痛苦的，对一些人来说这是现实。[Laura Roth]和[Christopher Sweeney]是艺术教师，他们为患有脑瘫的学生设计了一个工具架，以便他们能够独立表达自己。

[![](../Images/4e5b5077179d3f3634afd027d9e1171e.png)](https://hackaday.com/wp-content/uploads/2020/08/2020-08-26-Cerebral-Palsy-Tool-Assistant-Thmnl.jpg) 在这个横幅图像的两边，你可以看到来自患有痉挛性脑瘫的【萨拉】的铅笔画。她在戴着模仿她手的工具架时画了这些画。现在，这个设计服务于其他学生，是 2020 年黑客日奖的一部分。工具架像一个宽手镯一样缠绕在手腕上。罗纹保持其形状，管接受圆柱形物体，如铅笔，笔尖和画笔。结果是，铅笔的尖端与握在手中的位置相差不远，但这回避了手和手指的抓握和精细控制问题。

印花以 STL 的形式提供，应该用柔韧的细丝印花，以确保穿着舒适。请注意数字笔，它可能需要在笔管和用户之间有导电的东西。

黑客们很熟悉大脑麻痹带来的挑战，这些年来我们很高兴看到各种各样的解决方案，比如[开门器](https://hackaday.com/2018/01/27/iot-doorman-eye-controlled-door-for-a-girl-with-cerebral-palsy/)、[相机万向架](https://hackaday.com/2017/03/21/scratch-built-camera-gimbal-for-photographer-with-cerebral-palsy/)，还有完全[支持](https://hackaday.com/2011/11/11/dad-hacks-his-son/)。
