# 开源 PowerPC 笔记本越来越近

> 原文：<https://hackaday.com/2022/12/09/an-open-source-powerpc-notebook-edges-closer/>

回到 2020 年，我们报道了使用 PowerPC 架构创建全新开源笔记本电脑平台的努力。当时他们有很大的计划和 PCB 设计，我们很高兴地报告，在这中间的两年里，他们已经取得了进展，现在已经有一些真正的原型准备好进行测试。

有些人可能会问为什么这是必要的，毕竟有很多笔记本电脑和不止一个常用的处理器平台。但这忽略了开源硬件的要点，即它的多元化和功能性一样重要。但是如果你只在稍微老一点的 MAC 和一些游戏主机上遇到过 PowerPC 架构，那么驱动这款设备的芯片是什么呢？答案是，不是那些古老的芯片，而是[恩智浦 T2080](https://www.nxp.com/products/processors-and-microcontrollers/power-architecture/qoriq-communication-processors/t-series/qoriq-t2080-and-t2081-multicore-communications-processors:T2080) ，一款 1.8 GHz 四核设备，拥有相当于笔记本电脑的功率。

当然，在原型和最终设备之间还有许多障碍需要跨越，但考虑到功能笔记本电脑的挑战，他们已经达到了这一里程碑，这令人印象深刻。我们期待着看到进一步的迭代，也许，只是也许，有一天一个完整的设备。[我们最初的报道在这里](https://hackaday.com/2020/08/28/open-hardware-laptop-built-on-power-pc-isa/)。
