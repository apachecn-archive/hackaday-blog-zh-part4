# 克努茨堡——松鼠完成所有不可能完成的任务

> 原文:[https://hack aday . com/2022/04/26/fort-knutz-squirrels-go-all-mission-impossible/](https://hackaday.com/2022/04/26/fort-knutz-squirrels-go-all-mission-impossible/)

在他的后院有一个喂鸟器。还有吃种子的松鼠。因此，有人做了一件事，他建造了一个由九部分组成的松鼠障碍赛，最后奖励了核桃，然后[拍摄了它们战胜障碍赛](https://www.youtube.com/watch?v=DTvS9lvRxZ8)。

(剧透——这在视频中要好得多，我们把它放在了休息时间的下面)。

他家后院的四只松鼠进入了一个“赌场”,避开了重重“安检”。从那里，它穿过安装在轴承上的杆，跃入直升机下的板条箱，该板条箱用滑索连接到随机移动砖块的砖墙，并进入他们的藏身之处。

![A squirrel at a model buffet in a casino](../Images/8a3242bc3f964be51697825fb6329533.png)

Security is about to get him.

隐藏的电梯井通向下水道，下水道通向《碟中谍》中著名的房间，在那里[汤姆·克鲁斯]必须避开地板，但要到达顶部的舱口，他们必须通过“侵入”控制系统(通过按下键盘形状的按钮)和放下绳梯来放下梯子。

接下来，他们穿过一个管状迷宫，来到一个充满激光束(3D 打印机灯丝)的房间，最后他们可以和 Fort Knutz 一起跳上平台。如果他们打开了保险库的门，他们会得到一堆核桃作为奖励。

是的，这完全是疯了。

这是一个了不起的建筑，大部分是用气动执行器完成的。有些地方的工艺很美。障碍物比例匀称，赏心悦目。他运用了可爱的原则。它也是松鼠安全的。例如，每个封闭空间都有一个简单的单向紧急出口。这个视频是电影制作的精彩片段。有可爱的角色，一个情节，最终一个关于冲突的迷人故事变成了一种有趣和给予的关系。

这是他早期的松鼠迷宫[忍者战士课程的重建。](https://www.youtube.com/watch?v=hFZFjoX2cGg)
