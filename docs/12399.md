# 电容器是如何工作的？

> 原文:[https://hackaday.com/2022/01/06/how-do-capacitors-work/](https://hackaday.com/2022/01/06/how-do-capacitors-work/)

如果你像[科学精神病院]一样，你可能会想知道电容器是如何工作的，因为在它们的核心，它们只不过是一个充满空气或另一个绝缘体的间隙。他解释了在最近的一个视频中，你可以看到下面。

当然，在 DC，电容器的导电性并不比用作电介质的绝缘体好多少。然而，DC 电压必须在某个时候开始，当它开始时，它短暂地看起来像交流。视频用简单的语言解释了这一切。当然，如果你精通数学，你可以从普通的 C=dQ/dV 方程中得到同样多的东西。

如果这还不能说明问题，视频中关于费用的解释将会给你一些启发。他甚至展示了一个经典“水力模型”的动画，这有助于培养对过程的直觉。

我们通常不会考虑电荷、电压、电流和能量之间关系的细节，但正如[科学庇护]指出的，他是一个理论物理学家，而不是工程师。因此，他深入探讨了实际的解释，这不是我们在设计电容时通常会想到的，因为我们通常只对实际结果感兴趣。不过，还是很高兴知道。

虽然大多数真实电容器看起来不像露天间隙，但有些看起来像，你甚至可以把它们做成露天间隙。你也可以只是[把一些电线](https://hackaday.com/2018/07/03/these-capacitors-are-a-cheap-gimmick/)拧在一起。
