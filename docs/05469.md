# Carbide3D 路由器拆卸和测试

> 原文：<https://hackaday.com/2020/01/27/carbide3d-router-teardown-and-testing/>

从表面上看，你可能会认为一个小的路由器很简单。毕竟，除了旋转的马达，它是什么？然而，该电机必须处理一些相当严重的扭矩，这取决于你是路由。[Baki1]让他的 Carbide3D 路由器在一个项目中死去，所以他做了我们任何人都会做的事情。他[撕开](https://www.youtube.com/watch?v=PLuOmIOzqaI&feature=youtu.be)。

除了展示它的内部，他还试图找出它的问题所在。它看起来像一个熔断的三端双向可控硅开关是罪魁祸首，我们假设第二部分将是修复和如何实际工作。

马达在跳跃，事实上，如果没有钳子轻轻敲几下，马达就不会开始转动。一旦脱离外壳，你可以在相对较小的空间内看到数量惊人的电路。

三端双向可控硅开关暴露在外时大多会从印刷电路板上脱落。一些测试表明，路由器在机械方面完好无损。然而，三端双向可控硅开关测试糟糕，我们将不得不等待第 2 部分，看看是否真的修复它。

这不禁让我们想起了 2016 年看到的 [Dremel triac 修复](https://hackaday.com/2013/11/26/simple-dremel-triac-hack-repair/)。这也提醒我们，我们想建立我们的[大红色开关](https://hackaday.com/2018/02/11/push-big-red-button-receive-power/)。
