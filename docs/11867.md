# 3D 打印绝对编码器绝对精彩

> 原文：<https://hackaday.com/2021/11/10/3d-printed-absolute-encoder-is-absolutely-wonderful/>

当您需要记录旋转物体的角度时，无论是旋钮还是机械臂中的关节，绝对旋转编码器几乎总是最佳选择。它们很便宜，很容易买到，事实证明[你可以用一个磁传感器、一个拉链和一个滑板轴承](https://github.com/scottbez1/AS5600Knob)做出一个非常棒的东西。

当[Scott Bezek]拿到 AS5600 磁传感器分线板时，他就是这么做的。传感器本身是位于电路板中间的集成电路，在斯科特的设计中，它位于 3D 打印的载体上。它的顶部有一个轴承架，上面有一个轴承——你猜对了。特别是一个标准的 608 滑板轴承，它卡在底座上，并由一个拉链牢牢地固定在底座的标签上。最后一部分是一个 3D 打印的旋钮，内置一个微小的磁铁，垂直于旋转轴。旋钮滑入轴承，AS5600 读取磁铁的方向。

当然，如果你只是想要一个旋钮，你可以只购买一个编码器，并与它做，但这种方法有其优势。也许您无法在设计中安装商用编码器。也许你需要轴承提供的超平滑旋转。或者也许你正在[实际建造那个机器人手臂](https://hackaday.com/2020/09/15/open-source-robotic-arm-for-all-purposes/)——像这样的定制磁性编码器在致动器设计中非常常见，虽然更工业化的版本(通常)有更少的拉链，但【Scott】的设计将非常适合。
