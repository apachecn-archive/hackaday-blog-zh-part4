# DIY 模拟合成器看起来很有趣

> 原文:[https://hack aday . com/2021/12/10/DIY-analog-synth-looks-like-fun/](https://hackaday.com/2021/12/10/diy-analog-synth-looks-like-fun/)

构建构成模拟合成器的单个组件相对容易，这使得自己动手制作非常诱人。这就是[Albert ny strm]所做的，结果是这个[看起来很棒，听起来很棒的模拟合成器](https://www.youtube.com/watch?v=ttWqUQ5hmU8)。

他的 monosynth 中的 VCO 基于 AS3340 VCO 芯片，这是一个克隆的 [柯蒂斯电子音乐专业](http://www.curtiselectromusic.com)‘CEM 3340 芯片(用于 Oberheim OB-Xa、Roland Jupiter-6 和时序电路 Prophet-5 等机器。)压控滤波器基于 Moritz Klein 基于 VACTROL 的 VCF 电路，包络基于 Thomas Henry 的 555 包络电路(谷歌搜索会很快找到这些，以及使用 CEM 芯片的建筑示意图。)最后，键盘是来自 Arturia Keystep 的施主。虽然没有一步一步的构建说明，或原理图，但我们确实有一些关于仪器的信息。正如你可以从一些重要的照片中看到的，很容易就能看出[艾伯特]是如何把所有东西组合在一起的。或者不是。

 [![Synth_Insides02](../Images/bbef5a74d14391452bf09bc3a8f8fcbf.png "Synth_Insides02")](https://hackaday.com/2021/12/10/diy-analog-synth-looks-like-fun/synth_insides02/)  [![Synth Gutshot 1](../Images/cd76dbabd9b36382e74543bd0749b1b6.png "Synth_Insides01")](https://hackaday.com/2021/12/10/diy-analog-synth-looks-like-fun/synth_insides01/) 

即使内部有点疯狂，最终结果是一个看起来很棒的单声道合成器，它几乎做了你需要的一切。如果你很想把一堆组件连接在一起，然后自己做一个，还有更复杂的方法。或者也许你更喜欢走[的数字路线](https://hackaday.com/2016/05/15/hackaday-prize-entry-a-cute-synthesizer/)？无论哪种方式，synths 的构建和使用都是一大乐趣。
