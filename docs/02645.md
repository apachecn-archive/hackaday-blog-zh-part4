# 一瞥典型的英国住宅电源板

> 原文：<https://hackaday.com/2019/04/14/a-peek-inside-a-typical-british-residential-power-panel/>

无论你在哪个领域，了解别人是如何实践的都是有趣且有益的。配电系统尤其如此，不同国家甚至不同地区的标准和做法都不一样。[这次典型的英国住宅配电板之旅](https://www.youtube.com/watch?v=2R3L_nvNP6Y)是一个很好的例子，展示了解决相同工程问题的不同方法，以及任何设计中都会出现的妥协。

我们习惯于看到[大克莱夫]把有趣的设备撕成碎片，但请放心，这个配电板在泄露秘密时基本上保持完好。与北美住宅建筑中常见的配电盘和断路器相比，英国的消费单元是整洁和简单的奇迹。诚然，展出的装置还没有投入使用，一旦电工完成它，事情无疑会有所改变，但事实上一切都是安装好的。[Clive]解释了该面板的一些奇怪之处，例如看起来像主断路器的东西实际上只是一个隔离开关，还有一对剩余电流设备(rcd)，我们在北美称之为接地故障断路器(GFCIs)，尽管看起来像断路器，但它们也不充当断路器。提供一个坚固的母线将 rcd 连接到相邻的断路器，形成两组独立的接地故障保护。

[Clive]沮丧地注意到，汇流条的接线片实际上可能会插在断路器上升的夹钳终端后面，导致连接不良和过热。尽管如此，我们并不介意将这些概念引入北美的面板，这一点我们在不久前关于电路保护的讨论中有所涉及。
