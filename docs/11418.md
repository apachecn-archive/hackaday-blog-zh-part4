# G10 是你渴望的 3D 打印表面

> 原文：<https://hackaday.com/2021/10/13/g10-is-the-3d-print-surface-you-crave/>

从一开始，打印表面就是 3D 打印机开发和实验的主要部分。[制造商缪斯]一直在试验 G10，一种廉价的高压玻璃纤维层压板，并发现它是大多数 FDM 印刷需求的绝佳选择。(视频插播在休息之后。)

你可能更熟悉 G10 的耐火版本 FR-4，这是大多数 PCB 使用的玻璃纤维基板。它也被称为品牌名称 Garolite。[制造商缪斯]测试了解放军，PETG(对他的[耳机建设](https://hackaday.com/2021/09/27/how-good-are-the-headamame-3d-printed-headphones/))，ABS，ASA，[宠物](https://hackaday.com/2021/06/29/petbot-turn-pet-bottles-into-filament/)， [PCTG](https://hackaday.com/2021/03/24/pctg-filament-has-interesting-properties/) ，和尼龙。当加热到适当的温度时，所有材料都表现出优异的床粘附性，并且当其冷却时，通常会自释放部件。对 TPU 来说，床没有加热是为了防止它粘得太紧。测试了 0.5 毫米、1.5 毫米和 3 毫米 G10 纸张厚度，[制造商 Muse]发现 1.5 毫米是去除特别粘的印刷品的刚性和灵活性之间的完美平衡。

G10 已经在一些商业 3D 打印机中使用，但除了像尼龙这样的高温材料之外，关于它的使用信息很少。只要你小心不要刮伤床，它会在零件底部留下极好的表面光洁度。与玻璃相比，其较轻的重量有利于床身沿 Y 轴移动的打印机。另一个主要优势是成本低，尤其是与一些更奇特的床材料相比。

结果当然看起来很有希望，我们渴望得到一些 G10 为我们自己的打印机。如果你很难找到它出售，看看你当地的制刀供应商，他们把它作为手柄材料出售。
