# 空气炮喷出一股二茂铁火花

> 原文:[https://hack aday . com/2020/10/15/air-cannon-serves-up-a-blast-of-ferrocerium-sparks/](https://hackaday.com/2020/10/15/air-cannon-serves-up-a-blast-of-ferrocerium-sparks/)

好吧，看起来我们有了一种新的方式来娱乐孩子们和破坏邻居，如果这个发射火花的二茂铁炮像它看起来那样容易建造的话。

这是由一个著名的眉唱项目供应商[NightHawkInLight]带给我们的，他的丙烷火炬似乎从未中断过。这里的想法是一个显然很受欢迎的把戏的大规模版本，打火机中的“火石”实际上是铁铈合金(一种恰当命名的铁铈合金)的棒，被加热到几乎熔化的状态，然后掉落在坚硬的表面上。熔化的合金从那里爆发出一阵火花，让在场的人欢笑不已。

[NightHawkInLight]版本的把戏放大了一切。他没有使用更轻的火石，而是使用野营用的那种点火器上的铬铁棒。该杆被塞入由钢制动线形成的桶中，该桶连接到 PVC 空气室的输出。他最著名的丙烷火炬是以这样一种方式连接的，即火焰作用在装有引火塞的物体上，在空气从燃烧室内释放出来之前，把它加热到熔化状态。在下面的视频中看到的大量火花令人印象深刻，但我们已经厌倦了性别展示派对和森林大火。我们只希望他手边有灭火器。

说真的，小心这种东西。从[他的等离子推进汽水瓶](https://hackaday.com/2017/10/12/propane-powered-plasma-rifle/)到[用电弧焊机制造人造红宝石](https://hackaday.com/2020/03/06/a-quick-and-easy-recipe-for-synthetic-rubies/)，夜鹰之光在这类项目上有着丰富的经验。我们肯定他不想看到任何人受伤。
