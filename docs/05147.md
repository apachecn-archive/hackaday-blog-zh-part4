# Raspberry Pi 上的 Commodore 磁带机模拟器

> 原文:[https://hack aday . com/2019/12/17/commodore-tape-drive-emulator-on-a-raspberry-pi/](https://hackaday.com/2019/12/17/commodore-tape-drive-emulator-on-a-raspberry-pi/)

我们想象大多数阅读 Hackaday 的人身边都有一两个旧的树莓皮。这有点不太可能，你仍然有一个 8 位的 Commodore 在工作状态，但我们敢打赌，观众中有不少人可以把自己算在这两个群体中。那么为什么不介绍他们呢？

[RhinoDevel]写信告诉我们关于 [CBM Tape Pi 的情况，这是一个用于 Raspberry Pi](https://github.com/RhinoDevel/cbmtapepi) 的开源 Commodore 磁带机仿真器，只需要几个无源组件就可以连接起来。更好的是，该项目针对的是更老的 Pis，他们更有可能在零件箱中无所事事。在休息后的视频中，可以看到一个 Commodore 宠物正高兴地用它古雅的小复合视频连接器从原始的 Raspberry Pi 加载内容。

Commodore 本身没有任何特殊软件，该项目允许用户在 Pi 的 SD 卡上加载和保存 PRG，并遍历目录。不要期待一流的 I/O，因为[RhinoDevel]指出目前还没有实现快速加载器。当然[如果你是一个足够狂热的爱好者，在 21 世纪的今天还在四处闲逛](https://hackaday.com/2019/05/02/twenty-five-years-since-the-end-of-commodore/)一辆 VIC-20 或 C64，那么我们认为你有足够的耐心度过难关。
