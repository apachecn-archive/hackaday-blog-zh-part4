# 在家模拟外星球探索

> 原文:[https://hack aday . com/2020/05/24/模仿-系外行星-探索-在家/](https://hackaday.com/2020/05/24/mimicking-exoplanet-exploration-at-home/)

人类总是想知道我们在宇宙中是否是孤独的。外面有什么？当然，在过去的几周里，我们越来越多地对我们自己的直接接近感到疑惑，但那是一个不同的故事。直到两年前，我们有开普勒太空望远镜帮助我们探索银河系内的系外行星来寻找答案。[poblocki1982]从小就对太空着迷，最近发现 3D 打印是他的新事物，他认为没有什么比找到一种结合你的爱好的方法更好的了，并建造了[一个模拟望远镜主要概念](https://www.instructables.com/id/Exoplanet-Transit-Demo-Apparatus/)的简化模型版本。

总的想法是，当一颗恒星的一颗行星经过它时，检测恒星亮度的轻微变化，并利用这种变化来分析每颗行星的特征。他通过连接到 Arduino 的 LDR 实现了这一点，允许实时读取和记录 SD 卡上的数据。不幸的是，火箭科学还不在他的爱好清单上，所以[poblocki1982]不得不把外层空间带回家。他用一个 DC 马达驱动两个不同大小、转速和距离的“行星”绕着它们的“恒星”旋转，他有了完美的模型行星系统，可以很容易地兼作装饰灯。

显然，这并不意味着像真正的开普勒太空望远镜那样探测真正的行星，而是展示它的一般概念，因此这是一个很好的小科学实验。为了更实际地利用我们自己的太阳系，[poblocki1982]最近建造了[这个自动校准的日晷](https://hackaday.com/2020/05/06/sun-seeking-sundial-self-calibrates-in-no-time/)。如果你喜欢行星的旋转模型，那就去看看[之前关于那个](https://hackaday.com/tag/orrery/)的一些项目。
