# 看看这款智能压力调节器的精湛工艺

> 原文:[https://hack aday . com/2018/08/28/see-the-fallious-work-in-this-smart-pressure-regulator/](https://hackaday.com/2018/08/28/see-the-fabulous-workmanship-in-this-smart-pressure-regulator/)

对于许多需要控制气压的项目来说，通常的选择是连接一个泵，也许是用一个电机控制器来打开和关闭它，然后用它来工作。如果一个人的要求不能由那个水平的设备和控制来满足，那么是时候看看商业监管机构了。[克雷格·沃森]正是这样做的，但发现结果令人失望，因为它们很昂贵。他发现，商业产品——尤其是在低压下——往往会漏气，偶尔会报告不正确的压力，而且总体上并不十分精确。出于一种必要的感觉，他开始设计自己的电子控制闭环压力调节器。金属块是一个定制的歧管，其上安装有阀门硬件，安装在顶部的 PCB 支撑着控制系统。[项目日志](https://hackaday.io/project/148274/logs)有一些很棒的图片和原型制作过程的细节。

这个项目是[克雷格]在[微流体控制系统](//hackaday.io/project/27511)上的工作成果，因为他发现这些有用系统中的许多设备对于小型实验室或个人来说过于昂贵。在开发电子压力调节器的过程中，他意识到它可能有超越微流体控制的应用，并将其创建为一个模块化设备，可以很容易地集成到其他系统中，并处理正压或负压。它特别适用于对空气要求低、供应量有限，但需要精确控制的场合。
