# 对着麦克风唱情歌

> 原文：<https://hackaday.com/2019/02/12/love-songs-to-the-microphone/>

弗兰克·辛纳屈(Frank Sinatra)的传记作者曾评论说，对于辛纳屈这样的歌手来说，他们的乐器就是麦克风。我们倾向于将话筒视为理想的传感器，忠实地拾取声音。但是和大多数电子元件一样，麦克风也是不完美的。它们有不同的频率响应。当我们说“爆米花”这样的词时，他们会听到砰砰的声音，而现场听众通常听不到这些声音。

[Cheddar]有一个有趣的视频(见下文)，讲述了像辛纳特拉、宾·克罗斯比和比莉·哈乐黛这样的表演者如何学会利用麦克风。他们认为麦克风改变了人类唱歌的方式，他们是对的。

早期的录音是蜡缸或磁盘上的机械事件。这意味着人们不得不对着喇叭尖叫，以收集足够的声音来移动录音指针。麦克风的发明让歌手可以更自然地表演，但正如视频所示，最优秀和最持久的歌手学会了利用麦克风。

不过，该视频忽略了另一个关键因素，即音响工程师，他经常调整东西以获得尽可能好的声音。如果你想知道“鲁迪·贴身男仆”是谁，我们认为他们指的是鲁迪·瓦利。

我们观察了[带状话筒如何改变录音](https://hackaday.com/2016/05/02/blue-ribbon-microphone/)。虽然视频中提到了宾·克罗斯比，但事实证明他在[将磁带录音](https://hackaday.com/2018/08/07/recorded-programming-thanks-to-bing-crosby/)引入广播业——以及后来的电视业——中发挥了重要作用。
