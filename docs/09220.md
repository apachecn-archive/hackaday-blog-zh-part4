# 带空调的电脑超频

> 原文:[https://hack aday . com/2021/02/11/PC-用空调超频/](https://hackaday.com/2021/02/11/pc-overclocking-with-an-air-conditioner/)

我们从不坚持认为黑客是实用的。[Tech Ingredients]是活生生的例子，因为他们改装了一个电脑机箱，使用窗式空调对电脑进行超频。他们认为他们还没有达到上限，并让他们的 AMD 锐龙 8 核处理器达到 4.58 GHz。

将空气从空调中强制排出的一个优点是，强制进入系统的空气非常干燥和干净。诀窍是创建一个简单的管道连接到一个 5000 英国热量单位的空调。它实际上并不与 CPU 冷却块相连接，相反，它只是迫使冷空气进入机箱，这往往会冷却内部的一切。不可否认，这并不比把你的电脑投入液氮中更糟糕，我们也承认空调设备是为了保持大面积的低温和高负荷运转而制造的。空调开着，他们至少关掉了一些风扇。即使在高速行驶时，气温也保持凉爽。

我们非常喜欢热界面测试设备，他们用它来测试界面介质。不过，我们想知道，你是否可以接入冷却回路，并实际推动制冷剂通过水冷块。另一种选择是像核反应堆一样使用双回路，在中央处理器上使用水回路，在空调中使用制冷剂回路。尽管如此，他们这样做的优势是简单和可逆的。

这让我们想到只是 3D 打印我们空调的面板，以容纳几个[树莓派板](https://hackaday.com/2020/11/11/adventures-in-overclocking-which-raspberry-pi-4-flavor-is-fastest/)。比一个[杜瓦瓶](https://hackaday.com/2013/08/18/liquid-nitrogen-finally-makes-an-arduino-project-cool/)便宜。
