# 一个打开你黑客空间的机器人游戏

> 原文:[https://hack aday . com/2022/03/25/a-robot-game-to-open-your-hacker space/](https://hackaday.com/2022/03/25/a-robot-game-to-open-your-hackerspace/)

欢迎新的 hackerspace 加入总是好的，因此我们很高兴听到位于荷兰东北部边缘的 Hackerspace Drenthe 即将开业。在全球疫情期间开创一个新的空间是一项壮举。作为开幕式的一部分，需要为好奇的公众展示一个机器人，还有什么比玩井字游戏的机器人手臂更容易接近呢！

可以正确地说，机器人精确地移动积木不一定是一项突破性的成就，但它的目的是为黑客空间的开放提供养眼的东西，同时也作为空间附近学校的一些学生的实验，这是成功的。界面是令人愉悦的复古*战争游戏*风格的终端，软件是用 Python 编写的。好奇的人可以在[的 GitHub 仓库](https://github.com/Hackerspace-drenthe/dobot)上找到一切，如果你在欧洲的那个地区，你可以在荷兰边境城镇科沃顿找到 [Hackerspace Drenthe](https://www.hackerspace-drenthe.nl/) ，并参加他们 4 月 2 日的开幕式。
