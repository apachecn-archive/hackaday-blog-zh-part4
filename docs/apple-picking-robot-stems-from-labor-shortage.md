# 摘苹果机器人源于劳动力短缺

> 原文：<https://hackaday.com/2021/05/01/apple-picking-robot-stems-from-labor-shortage/>

在疫情造成的所有与工作相关的问题中，还有一个是人们通常呆在家里的结果:几乎没有任何背包客做传统的短暂背包客工作，如摘苹果。莫纳什大学机械和航空航天工程系的研究人员找到了填补这一空白的方法，他们建造了一个气动机械臂，可以以最高速度每 7 秒钟收获一个苹果。

一套相机和算法在树叶中寻找水果，并像人类一样通过轻轻抓住并扭动来小心翼翼地摘下它。为了做到这一点，机器人必须考虑水果的形状，悬挂的方式，以及在保持损害最小的情况下将水果从树上分离的位置。吸力系统有助于将苹果拉进柔软的四指抓地力，然后手臂扭曲和旋转，将苹果放入垃圾箱。

这个机器人有很多优点，包括它可以在任何光线和天气条件下工作，可以在不到 200 毫秒的时间内识别一个苹果。唯一的问题是，这种操作会导致偶尔缺失果梗——这是一个听起来很挑剔的化妆品问题，但肯定会阻止一些商店购买这种水果。嗯，那个，而且到目前为止似乎只有一个这样的机器人。

休息之后有两个视频——一个给你要点的短视频，和一个提供吸盘视图的长视频，吸盘从手指中间出现，像一个异形的小嘴。

一些读者可能想知道为什么当振动收割机存在时，苹果仍然被单独采摘。[“摇一摇一抓”往往会碰伤苹果](https://reeis.usda.gov/web/crisprojectpages/1005200-shake-and-catch-harvesting-for-fresh-market-apples.html)，使它们不受农产品销售商的欢迎，然而，正如这个有趣的收获视频所示，被采摘者[粗暴对待的苹果注定要榨汁。机器人抓取器很温和，我们见过适合特定需求的所有形状和尺寸。当需求更加普遍时，](https://www.youtube.com/watch?v=20aop341JOE)[滚轮](https://hackaday.com/2020/09/24/roller-based-robot-hand-grasps/)或[柔软的球体](https://hackaday.com/2020/06/12/rolling-out-a-new-robot-arm/)可能就是答案。
