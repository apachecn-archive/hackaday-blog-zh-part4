# 水电站模型是一种有启发性的教育工具

> 原文:[https://hack aday . com/2020/07/21/model-hydro electric-plant-is-an-illumination-educational-tool/](https://hackaday.com/2020/07/21/model-hydroelectric-plant-is-an-illuminating-educational-tool/)

点亮一条发光二极管的方法不止一种。你有没有尝试过建立自己的水力发电厂来做到这一点？现在你可以了。复制[【Matic markovi】进入 2020 年黑客日奖](https://hackaday.io/project/171863-3d-printed-hydroelectric-power-plant-model)肯定会教你一些东西，如果不是很多东西的话，关于水力发电的方式和变量在其中发挥作用的方式。

在[Matic]的模型中，水从一个高度可调的水库流入一个 3D 打印的水斗式涡轮机。水射流以 90 度角撞击涡轮的杯形翅片，使组件快速旋转。这种机械能为连接到 Arduino Nano 的无刷 DC 电机充电，Arduino Nano 对来自发电机的交流电进行整流，并使用它照亮 RGB 条，就像均衡器显示器一样，代表正在产生的功率。

这无疑是我们见过的最酷的教育展示之一。通过三向拨动开关，水库可以在 55 厘米(21.6 英寸)的范围内上下移动，很容易看出水库越高，产生的功率越多。[Matic]如果您想自己制作 STL 和 INOs，它们通常都在常见的地方。流过演示的间隙，然后是由看不见的手重新组合在一起的分解渲染。

你的水力发电装置不需要很花哨，只要能用就行。[一个人的垃圾桶可以成为另一个人的离网手机充电器](https://hackaday.com/2016/08/15/trash-heap-water-wheel-recharges-iphone-in-the-woods/)。
