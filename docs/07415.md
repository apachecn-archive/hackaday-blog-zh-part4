# 用等离子打印！

> 原文：<https://hackaday.com/2020/08/08/print-with-plasma/>

多年来，计算机有各种各样的方法将自己的想法写在纸上。无论是菊花轮、点阵、激光还是喷墨打印机，我们都曾诅咒过不听话的打印机。还有另一种类型的打印机，也许我们不常想到，但却是百万台收银机和停车售票机中的主力:热敏打印机。这些机制可以很容易地作为剩余物品找到，并且这些年来已经进入了不止一个项目。[HomoFaciens]通过从零开始建造一台利用热敏纸的等离子打印机，将热敏打印向前推进了一步[。](https://homofaciens.de/technics-machines-plasma-printer_en.htm)

热敏打印机顾名思义是通过将图像烧入纸张来完成工作的。它可能无法提供最好的打印质量，但却不需要色带、墨盒或碳粉。这个 DIY 版本使用现成的电池供电的等离子打火机来完成这项工作，安装在由两个步进电机驱动的 3D 打印 XY 打印机机构上。幕后是一个 Arduino Uno，它通过 USB 从 Linux 机器上的命令行程序接收指令。众所周知，这算不上打印技术的巅峰，但至少是一个迷人的项目。你可以在休息时间下面的视频中看到它的运行。

这不是第一台打印机，我们马上会想起几年前的墨滴打印机。
