# SpaceX 星际飞船复制品尝试失败

> 原文：<https://hackaday.com/2020/12/15/spacex-starship-replica-attempts-the-bellyflop/>

SpaceX 最出名的可能是其垂直着陆的可重复使用火箭技术。最近的这种努力是星际飞船，它最近在上个月进行了一次激烈的测试，试图进行腹部机动。[Nicholas Rehm]想在当地公园尝试类似的飞行模式，并着手建造自己的遥控星际飞船。

[建造就像我们之前见过的几个](https://hackaday.com/2019/05/07/supercapacitors-propel-rocket-to-the-skies/)，其中电力被用来推动一个火箭般的飞行器，使用螺旋桨和主动稳定。在这种情况下，有一对带反向旋转螺旋桨的双发动机，可以旋转以引导推力，以及四个外部控制表面。这些都在[尼古拉斯]的定制飞行控制员的指挥下。

在测试该装置时，[Nicholas]能够执行平稳上升，随后是腹部翻转，然后平稳返回垂直飞行和下降。由于地面粗糙，垂直降落在草地上是不可能的，但我们想象如果连接正确的起落架，这是可行的。

虽然以这种方式没有机翼的飞行不是特别有效，但很高兴看到通过智能使用控制系统工程来保持飞行器稳定可以实现什么。我们也期待看到[尼古拉斯]的下一次尝试。休息后的视频。
