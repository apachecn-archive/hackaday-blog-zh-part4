# 不是它能驾驭厄运，而是厄运能驾驭它吗？

> 原文：<https://hackaday.com/2022/12/25/not-can-it-run-doom-but-can-doom-run-it/>

这是对硬件黑客的标准测试，半认真半开玩笑地问道:“它能运行 DOOM 吗？”。id software 公司 90 年代初的标志性射手已经出现在从牙刷到乐高积木的所有东西上，但还没有人提出相反的问题:*厄运能跑得动吗？*”。丹尼·斯潘塞回答了这个问题，他已经证明了通过在末日关卡中制造一台可以工作的加法机[，在游戏中执行计算任务是可能的。](https://blog.otterstack.com/posts/202212-doom-calculator/)

如果你熟悉在《我的世界》制造工作电脑的人，这也是类似的情况。游戏元素被用来创建逻辑元素，从那里可以组装更复杂的系统。《毁灭战士》没有《《我的世界》》那样的游戏逻辑，但通过将怪物的行为与游戏中涉及房间、按钮和门的动作巧妙结合，有可能创造出最简单的积木——与非门。

休息下面的视频显示了行动中的加法器，首先在操作中(我们喜欢怪物驱动的显示！)，然后参观充满计算怪兽的逻辑区。重要的是要注意到这不是一台计算机，他还没有证明它是图灵完备的，而且毁灭级别的最大大小无论是什么都会对所能做的事情施加一个上限。但它确实表明，理论上至少一台计算机是可以制造出来的，我们确信人们会继续这项工作。
