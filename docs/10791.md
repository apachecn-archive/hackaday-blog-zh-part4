# 开尔文勋爵的发明将水滴变成了火花

> 原文:[https://hacka day . com/2021/07/22/Lord-kel vins-paraffin-turns-drips-into-sparks/](https://hackaday.com/2021/07/22/lord-kelvins-contraption-turns-drips-into-sparks/)

人们很容易认为产生数千伏电压的设备一定涉及相对现代的技术，但事实是，能够在户外点燃火花的机器早于爱迪生的灯泡。这意味着用现代工具、建筑技术和零件可用性来重建它们，可能比大多数人意识到的要容易得多。[Jay bow les 为他最新的*等离子频道*视频组装的迷人机器是一个完美的例子](https://www.youtube.com/watch?v=8IuwxVmcSM0)，因为它能够在没有任何电子元件的情况下产生 6000 伏的电压。

尽管[杰伊]可能很聪明，但他不能把这个想法归功于自己。这项荣誉属于开尔文勋爵，他在 1867 年发明了这种特殊样式的静电发生器。该机器被称为“开尔文滴管”或“开尔文勋爵的雷暴”，它能够在不使用任何移动部件的情况下从下落的水中产生高压电荷。

[![](../Images/4da3d8dd987f3d936a5f11fd29d40dd7.png)](https://hackaday.com/wp-content/uploads/2021/07/kelvinwater_detail.jpg)

Diverging streams means a charge is building up.

我们自己的[【Steven Dufresne】写了一篇关于这些设备如何操作的深入研究](https://hackaday.com/2017/03/27/getting-sparks-from-water-with-lord-kelvins-thunderstorm/)，但简短的版本是，正负电荷在两组金属感应器环和桶中建立，水流本身充当一种电线，将电荷带到头顶的蓄水池。正如[Jay]在视频中演示的那样，当水流被通过的感应器吸引时，你就会知道事情在起作用。

与其将一个单独的火花隙连接到他的滴管底部的水“接收器”上，[Jay]发现他使用的金属杯子上的把手也工作得很好。通过将杯子移近或移远，他可以调整间隙，第二次调整可以让他移动感应器的垂直位置。这听起来像是需要一些拨弄才能让一切就位，但是一旦它工作了，整个事情就非常令人印象深刻。

当然，如果你想认真对待高压实验，你会希望很快升级到一些不那么古怪的设备。幸运的是，[Jay]已经证明了[组装一个可靠的高压电源并不需要昂贵或复杂](https://hackaday.com/2020/08/23/a-miniature-power-supply-for-high-voltage-hacking/)。
