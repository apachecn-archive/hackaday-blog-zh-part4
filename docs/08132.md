# 我们现在不想成为帝国冲锋队

> 原文:[https://hack aday . com/2020/12/24/we-won-won-want-to-be-storm borders-right-now/](https://hackaday.com/2020/12/24/we-would-not-want-to-be-stormtroopers-right-now/)

感谢 Hacksmith 的 Hackaday 校友[James Hobson],人类离幻想精确的光剑又近了一步。他们的原型军刀可以穿透(角色扮演)冲锋队的盔甲，(泡沫)墙壁，以及(合法的！)1/4 英寸(6.35 毫米)钢板。出于这么多的原因，我们想把[重点放在刀刃和刀柄](https://www.youtube.com/watch?v=xC6J4T_hUKg)上。(视频，嵌在下面。)

叶片是一种为玻璃加工设计的等离子流，燃烧丙烷/氧气混合物，几乎没有残余物，但“叶片”保持紧密的圆柱形状。使用定制的 PCB 作为混音控制器，刀片可以像电影中一样伸缩。手柄不是一个技术奇迹；这是一个艺术奇迹，如果你想看一些令人赏心悦目的加工，请在休息后观看第一个视频。第二个视频展示了一管 4000 华氏度的便携式等离子体能造成多大的伤害。

你还不会和任何人决斗，因为没有磁场像卢卡斯想象的那样塑造刀刃。不幸的是，你不能阻挡任何比气球剑更坚固的东西，因为固体物质会穿过它，但它会在这个过程中遭受巨大的烧伤。光剑是一种幻想武器，但书呆子们的集体热情让它变得和以往一样真实，吉尼斯的人给了这种建造可信度。
