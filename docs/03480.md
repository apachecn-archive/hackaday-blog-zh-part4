# 项目出口:铸造舱口手柄

> 原文：<https://hackaday.com/2019/07/05/project-egress-casting-the-hatch-handle/>

每扇门都需要一个把手，甚至——特别是——宇宙飞船的门。来自“保罗车库”的[保罗]获得了为阿波罗 11 号指令舱舱门制造把手的许可，该舱门是“出口项目”的一部分。

对于那些不熟悉[项目 Egress](https://wp.me/paBn4l-1wXM) 的人来说，这是第一次登月 50 周年的庆祝活动，旨在重现任务中的一件重要艺术品:阿波罗 11 号指令舱*的统一乘员舱门，或 UCH*。来自不同学科的 44 名制作者被指派制作 UCH 的各种作品，每个人都可以自由使用他或她想要的任何材料和方法。[Paul]选择了可能成为共识的材料——铝——并决定通过扮演这个角色来发挥他的优势。

手柄本身是一个矮胖的东西，正如人们所期望的那样，它是为宇航员设计的。[Paul]从手柄的 3D 打印版本开始，用铸造砂制作了一个两件式模具。原始零件可能是机械加工的，这意味着它没有铸造零件应该具有的拔模斜度，以使从成型介质中移除更容易。[保罗]运气好，得到了一个完美的模具，和一个完美的硅铝浇注到启动。所有的铸造需要的是一点清理和一些洞栓在门上。

[保罗]的把手将和其他零件一起被运到史密森尼博物馆，比如[[弗兰布兰奇]的闩锁组件](https://hackaday.com/?p=365200&preview=true&preview_id=365200)，这样[亚当]就可以在本月晚些时候的 50 周年庆典上现场组装舱门。请继续关注更多的项目出口覆盖面，因为零件不断滚动。
