# Arduino 电缆跟踪器有助于诊断损坏的 USB 电缆

> 原文:[https://hack aday . com/2021/07/18/arduino-cable-tracer-helps-diagnose-broken-USB-cables/](https://hackaday.com/2021/07/18/arduino-cable-tracer-helps-diagnose-broken-usb-cables/)

这些年来，我们都发现自己在太多看起来相似的 USB 电缆中游泳。有些具有所有的导体和功能，有些是奇怪的只有电力的怪人，有些可以快速给我们的手机充电，而有些则不能。这是一个令人头痛的问题，也是[TechKiwiGadgets]希望通过 Arduino 电缆跟踪器解决的问题。

追踪器与 USB-A、Mini-USB、Micro-USB 和 USB-C 电缆配合使用，以确定连接是否断开，并识别布线配置。它是围绕 Arduino Mega 2560 构建的，这是提供大量 GPIO 引脚的理想选择，非常适合此目的。探测结果显示在 2.8 英寸的 TFT LCD 显示屏上，可以很容易地判断出哪些电缆做了什么。

这是一个整洁的构建，我们可以想象这对于快速获取从某个地方的垃圾箱中挖出的任何电缆的通/断状态非常有用。只要记得将你发现的任何坏的电缆做好标记，这样它们就不会再来打扰你了。休息后的视频。
