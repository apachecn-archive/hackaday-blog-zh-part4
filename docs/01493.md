# 几十个伺服系统翻转这个 3D 打印的数字时钟的各个部分

> 原文：<https://hackaday.com/2018/12/10/dozens-of-servos-flip-the-segments-of-this-3d-printed-digital-clock/>

基于七段显示的数字钟？不刺激。一个有七段显示屏的数字钟，真的很大，可以在一个足球场上看到？那更有趣一点。[采用机电七段显示的大幅面数字钟？](http://www.otvinta.com/download14.html)这下可有好戏看了。

这个钟是通过[Otvinta]来到我们这里的，它是一个很好的例子，展示了你可以用 3D 打印和一点想象力做什么。显示器的每一部分都连接到一个可以翻转 90 度的小型业余伺服系统上。安装在一个印刷的塑料框架中，这些段可以根据需要翻转，组成显示时间所需的数字。28 个伺服系统需要两个 Pololu 控制器板，它们可以与运行 Windows IoT 的 Raspberry Pi 对话，这是一个有趣的设计选择，我们不常看到。你可能会认为 28 个伺服系统来回碰撞可能是无法忍受的，但下面的视频显示，显示器实际上相当安静。我们希望看到这种全黑白色的印刷，甚至是荧光塑料；在紫外线下看起来有多酷？

我们并不是说这是我们见过的唯一一个[七段伺服时钟](https://hackaday.com/2013/07/03/twenty-two-servos-and-an-awesome-clock/)，但它是一个非常光滑的构建。当然[使用伺服系统报时的方法不止一种](https://hackaday.com/2018/03/22/arduino-clock-jots-down-the-time-in-uv/)。
