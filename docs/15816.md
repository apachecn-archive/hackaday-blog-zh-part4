# DIYson 灯将线缆隐藏在接缝之间

> 原文：<https://hackaday.com/2023/01/02/diyson-lamp-hides-cables-between-the-seams/>

[史蒂文·贝内特]非常喜欢戴森的新灯，以至于他决定以原版的精神克隆他自己的版本。然而，戴森知道是什么让他们的灯如此特别——以至于他们申请了隐藏电源线技术的专利。[Steven]毫不气馁，他面临的最新挑战是创建一个既能捕捉原作的优雅，又不会完全复制的线缆管理解决方案。

[Steven]的最新更新从原始模型的专利细节开始。简而言之，戴森的优雅来自于扁平电缆(也许是柔性印刷电路板？)和在两个主要结构梁之间传递能量的磁性界面。后半部分讨论了[Steven]的替代解决方案:可以 3D 打印成任意长度的微型牵引链。像扁平柔性电缆一样，这种电缆位于灯的两个结构梁的凹槽中；但与原来的不同，它在行程的一端向外绕成一个环。总的来说,[Steven]对这个结果非常满意，我们认为这个解决方案赋予了这款灯独特的魅力。

捕捉设计精髓并不仅仅停留在电线管理上。看看系列中的一些[以前的视频日志，了解一下散热和机械感方面面临的一些其他挑战。](https://www.youtube.com/watch?v=uHMxSkKujzI)

电线管理做得好的话，会触动我们头脑后面某处的设计之痒。如果你对更多的电缆管理解决方案感到好奇，看看其他一些使用[卷尺](https://hackaday.com/2017/12/17/diy-cable-chain-looks-great-stays-cheap/)或涉及[DIY 盘绕方法](https://hackaday.com/2022/01/23/cables-too-long-try-cable-management-via-diy-coiling/)的技巧。
