# 遥控半履带车是激光切割的杰作

> 原文：<https://hackaday.com/2020/08/29/rc-halftrack-is-lasercut-masterpiece/>

半履带车是一种车辆设计，在二战后的几十年里逐渐失宠。将轮式车辆易于驾驶和操纵的优点与履带式车辆的强泥地和雪地性能结合起来，它们在随着装甲运兵车的兴起而逐渐淘汰之前发挥了作用。[JackCarter]希望建造他自己的，[所以制造了一个激光切割遥控版的 *SdKfz 251 22。*](https://www.instructables.com/id/Lasercut-RC-PakWagen/)

这项工作令人印象深刻，[JackCarter]根据车辆的照片和插图在 Solidworks 中创建了设计。移动部件是激光切割的，包括轨道本身，由许多微小的激光切割 MDF 部件组装而成。使用激光切割制作模型的好处是,[杰克·卡特]很容易制作简单的夹具来简化轨道拼接的过程。带有电机屏蔽的 NodeMCU 控制用于驱动履带的齿轮电机，并驱动用于转向的伺服系统。由于 Blynk 框架使得为定制项目构建应用程序变得容易，因此可以通过智能手机进行控制。

成品真正展示了[JackCarter]的 3D 设计技能，看起来建造和驾驶起来非常有趣。我们希望看到它用一点油漆和一些时期的贴花来真正完成外观。黑客喜欢好的履带式车辆，我们以前也见过一些令人印象深刻的建造。休息后的视频。
