# 陀螺仪水平是数字的

> 原文:[https://hackaday.com/2020/11/15/gyroscope-level-is-digital/](https://hackaday.com/2020/11/15/gyroscope-level-is-digital/)

水平仪是一种基本的建筑工具，你知道，就是那种在一管液体中有一点气泡的水平仪。[DesignBuildDestroy]用一个 Arduino，一个陀螺仪芯片和一个有机发光二极管，做了一个没有气泡的 [3D 打印水平仪](https://designbuilddestroy.com/digital-level/)，但它确实有一个不错的数字显示屏。

当你意识到陀螺仪曾经是为导弹和飞机保留的高科技项目时，这是很有趣的。现在你可以用六轴传感器来测量硬币。更好的是，项目中使用的代码可以卸载 Arduino 进行大量处理。

最初，该设备安装在试验板上，这总是一个解决问题的好主意。多亏了有机发光二极管，Arduino 可以在没有电脑的情况下进行自我校准，还可以做其他一些事情。显示屏很容易阅读，但我们认为应该有一个模式，显示一个用 O 字符制作的小气泡。看起来这将是一个有趣的雨天项目。不过，我们确实喜欢自动屏幕旋转。
