# 铜:一个世纪前的交流电整流

> 原文：<https://hackaday.com/2022/04/20/copper-rectifying-ac-a-century-ago/>

[Robert Murray-Smith]为我们展示了一个有趣的电子设备，它来自硅半导体出现之前的年代，[不起眼的金属氧化物整流器](https://www.youtube.com/watch?v=vvx1PTYyTSk)。在 19 世纪末残酷的 AC/DC 电流战争——涉及爱迪生、特斯拉和西屋电气等几个主要领导者——之后，电子尘埃落定，AC 成为最终的赢家。但是有一个问题。用变压器将高压交流电从配电网降压到更易管理的水平，并直接馈入灯泡和电加热器等消耗交流电的设备，这很简单。但是其他设备真的需要 DC，为了得到它，你需要一个整流器。

事实证明，即使在早期，我们也有可以完成这种操作的半导体设备，不是基于硅或锗，而是基于铜。氧化铜(I)是一种天然存在的 P 型半导体，它可以通过在火焰中加热铜片，并刮掉氧化铜(II)的外层，留下下面的活性层来容易地构建。简单地接触一块钢就足以完成该装置。

显然，一个实用的整流器有点难以制造，需要一定程度的控制，但你得到的想法。CuO 金属整流器可以整流，也可以作为热电堆运行，甚至可以作为太阳能电池，一旦我们对硅感到兴奋，它就被遗忘了。

其他类似的金属整流器也发挥了一些作用，例如![](img/5a7159a4386aa3890e26e1a21187e8d2.png)硒整流器，它基于硒化镉-硒界面的特性，形成了一个 NP 结，尽管它不能处理像旧铜一样多的功率。最后一种设备是对最初的 CuO 整流器的一点改进，它是基于一堆硫化铜/镁金属板，但它们出现得太晚了。一旦我们发现了锗和硅的奇迹，在它真正被广泛采用之前，它就被丢进了历史书里。
