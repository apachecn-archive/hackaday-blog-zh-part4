# 黑客日链接:2022 年 1 月 16 日

随着冬季真正笼罩北半球，是时候再次泡在特斯拉上[让一些车主在寒冷中——字面意思是](https://electrek.co/2022/01/12/tesla-owners-losing-heat-extreme-cold-heat-pumps-failing-badly/)。似乎一些 Model 3 和 Model Y 的车主发现他们的车的热泵并不能完全胜任泵送热量的任务。这似乎主要发生在美国东北部和加拿大南部，那里的极地涡旋再次主导天气，并将温度降至-30°C(-22°F)范围内，这可能更多地说明了热力学定律，而不是特斯拉气候控制系统的工程。毕竟如果车外热量不多，很难往里面抽。但话说回来，这些都是昂贵的机器，其中一些已经进行了广泛的维修，以解决去年出现的完全相同的问题。在我们看来，车主对特斯拉的抱怨是合理的，他们可能会从联邦政府那里得到一些帮助，联邦政府从安全的角度对 T2 的情况感兴趣。毕竟，没有热量可能意味着车窗起雾，这对安全旅行几乎没有好处。但是，嘿，这就是自驾的目的，对吧？

詹姆斯·韦伯太空望远镜上缺乏工程相机，我们一直依赖动画来说明展开天文台并为其任务做好准备所需的几十个部署，这一事实已经被说得很多了。先不说给飞船增加额外的相机没有什么意义，因为有趣的事情都发生在没有太阳照射的一侧，我们确实有幸看到了被宣传为[“人类最后一次看韦伯”](https://www.youtube.com/watch?v=U5VtQ957py0)的东西，这要感谢阿丽亚娜 5 号火箭上的工程相机。但没那么快——一位名叫[伊森·戈恩的天体摄影师设法在 JWST](https://youtu.be/qfKSeGTf0jk) 发射后第二天过境 L2 时发现了它。诚然，这个光点没有阿丽亚娜的照片那么壮观，而且它需要大量的天文摄影器材来完成，但看着韦伯优雅地穿过猎户座仍然令人激动。

当谈到保护元件和组件免受静电放电(ESD)的损害时，似乎有两种思想流派:要么你认真对待它并采取预防措施，要么你认为这是无事生非。Dan Kollen (AI6XG)似乎属于前一个阵营，他与我们分享了他对 ESD 损害风险的想法以及他采取的减轻风险的方法。这篇文章很简短，但充满了有趣的信息，如相对湿度对 ESD 产生的影响以及各种组件对电击的相对敏感度。他还分享了他的 ESD 保护策略——剧透:你并不总是需要戴腕带。ESD 术语表也是一个方便的参考。

如果亨利·福特今天还活着，并在 159 岁高龄时设法在宝马公司找到了一份工作，他可能会忍不住说:“你可以想要任何颜色，只要是黑色或白色，或者有一天可能是红色。”这要归功于德国公司在 CES 上发布的电子墨水喷漆变色汽车。如果宣传视频是可信的，它实际上看起来很酷，尤其是表面上隐约可见的六边形电极网格。虽然我们可以看到这个想法的实用性——夏天用白色的车，冬天用黑色的车——但我们不禁想知道像这样匆忙改变汽车外观的合法性。

最后，显然没有什么能躲过谷歌无所不知的眼睛。至少这可能是 Gioacchino Gammino 现在的想法，谷歌街景被用来帮助寻找和捕获在逃二十年的黑手党成员。这条线索来自西班牙加拉帕戈斯一家杂货店的街景图像，Gammino 已经逃到那里，用假名开始了新的生活。显然，警察已经知道他在镇上；像素化的照片与嫌疑人非常相似，足以让他们找到他的具体下落。快速查看该公司的脸书主页，可以看到一张 Gammino 的清晰照片，以及一个可以辨认的面部伤疤。我们认为这里的教训是犯罪不值得，它真的不能与社交媒体很好地融合。
