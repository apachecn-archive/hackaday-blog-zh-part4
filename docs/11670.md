# 重金属车床制造不节省钢材

> 原文：<https://hackaday.com/2021/10/27/heavy-metal-lathe-build-doesnt-spare-the-steel/>

众所周知，车床是最基本的机床，也是唯一能自我复制的机床。虽然我们不会争论车床在机械加工车间的首要地位，但这台靠刮痕打造的重型车床几乎证明了后一种观点是错误的。

当然，我们已经习惯了看到自制车床，而且我们以前也展示过不少。但是有两点让[Jornt]的构建脱颖而出:构建它只需要很少的专业工具，以及成品的庞大体积。在大多数自制车床倾向于台式品种和功能铸铝部件的地方，[Jornt]用钢来建造，而且很多。机器的底座和床身由废弃的工字钢焊接而成，导轨由角铁制成，角铁已经用一个巧妙的夹具磨平，以支撑角磨机。角磨机在建造中起着突出的作用，手钻、锉刀和焊机等简单工具也是如此——是的，还有未完工的车床本身，它用于钻出主轴箱的轴承座。

完成的车床由跑步机电机驱动，毫无疑问[【杰里米·菲尔丁】会支持](https://hackaday.com/2021/10/18/making-your-projects-move-jeremy-fielding-to-deliver-remoticon-keynote/)，重达 450 公斤。老实说，它看起来就像你可以从目录中买到的东西，并且具有商用机器的大部分功能。我们希望在这台车床上看到的一件事是[为他现成的车床开发的电子丝杠](https://hackaday.com/2019/04/17/benchtop-lathe-gets-an-electronic-leadscrew-makeover/)。
