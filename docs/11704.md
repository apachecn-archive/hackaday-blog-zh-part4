# 3D 打印降压真空成型教程

> 原文:[https://hack aday . com/2021/10/22/真空成型-3d-印刷-降压-教程/](https://hackaday.com/2021/10/22/vacuum-forming-with-3d-printed-buck-tutorial/)

[Matterhackers]有一个很好的视频教程，介绍如何使用[真空成型来制造塑料物品](https://www.matterhackers.com/articles/how-to-use-a-vacuumformer)。当然，你有一台 3D 打印机，但如果你正在快速制作薄而柔韧的物品，真空成型有一些优势。但是不要难过。流程中的主项目来自 3D 打印机。就像一个模子，成型不会产生一个复制的主，称为降压。相反，巴克提供了类似塑料包裹的骰子的东西。

虽然明显的真空成型物品包括外卖食品容器和零售物品的塑料泡罩包装等，但您也可以制造更坚固的物品。显然，原三部曲中的所有星球大战电影都使用真空成型来制造冲锋队盔甲。

像许多制造技术一样，有特殊的设计规则。例如，你不能有任何突出部分，否则塑料将不会从降压释放。事实上，理想情况下，降压器的所有侧面都有一个轻微的斜度，这样底部比顶部宽，以简化降压器与成品的分离。

3D 打印巴克的另一个问题是塑料将完全符合表面。因此，如果你有层线或其他不规则，它们也会出现在成品中。你可能不在乎，或者你可能想砂和可能密封的打印。当然，你可以用任何能承受高温的东西来赚钱，包括——据《华盛顿邮报》报道，包括生土豆。虽然 PLA 因为热而不确定，但帖子表明他们通过在打印时增加顶层的数量成功地使用了普通 PLA。但是，建议使用温度更高的高级 PLA、ABS 或 PETG。

Makyu Formbox 连接到商店真空吸尘器或家用真空吸尘器，以提供负压。不出所料,[Matterhackers]出售的这款设备并不便宜，但我们记得花了更多钱买了一台 3D 打印机(最便宜的价格约为 700 美元)。

如果你更喜欢 DIY，拿一些 MDF 和[检查这些计划](https://hackaday.com/2014/10/19/diy-vacuum-former-on-the-cheap/)。或者，去找一些[大一点的](https://hackaday.com/2011/08/31/how-to-build-a-vacuum-form-table-that-gets-it-right-every-time/)。
