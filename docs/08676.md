# 我们打赌你从来没有见过粉红色的假牙合成器

> 原文：<https://hackaday.com/2020/12/21/we-bet-youve-never-seen-a-pink-denture-synth/>

在合成器世界的一端，矗立着为专业音乐家的需求量身定制的最佳音质和性能的商业乐器。另一方面，有一些怪异、古怪的原型和艺术构建，旨在挑战我们对合成器应该是什么的概念。[Love hultén 的 VOC-25 就属于后一类。](https://www.lovehulten.com/voc25.html)

合成器是围绕着 Axoloti 内核 T1 构建的，这是一个用于音频实验的微控制器板。封装立体声 DAC 和 ADC，以及 MIDI 输入和输出，是此类项目的完美基础。加载了声乐样本，在相当典型的意义上，它是由键盘演奏的。有趣的是面板上有 25 组塑料牙齿。由于一个螺线管，当用户弹奏相应的音符时，牙齿就会开合。随着机器和珍珠白本身的噼啪声，它给这件作品增添了相当令人毛骨悚然的氛围。

凭借其干净柔和的外壳，我们可以想象这件作品是前卫电影剪辑的明星，或者只是在制造者集会上吓唬孩子的东西。毫无疑问，这是一个有趣的建筑。这些年来，我们也看到了一些其他伟大的实验合成器——[这个 48 Game Boy build 浮现在脑海中](https://hackaday.com/2018/11/29/an-englishman-and-48-gameboys-walk-into-a-bar/)。休息后的视频。
