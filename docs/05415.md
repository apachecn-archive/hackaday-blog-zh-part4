# BinDayCator 让你知道什么时候该倒垃圾

> 原文：<https://hackaday.com/2020/01/23/bindaycator-lets-you-know-when-to-take-out-the-trash/>

城市垃圾清运是一种美妙的奢侈。通常一周一次，大卡车会从你的住处经过并取走垃圾箱。你所要做的就是记得把它们熄灭！以人类特有的方式，这很难做到。[输入 BinDayCator。](https://www.youtube.com/watch?time_continue=1&v=zLU1cOYf1dY)

该项目包括一个典型的市政会有轮垃圾桶的 3D 打印模型。印刷在白色灯丝上，当被强大的 WS2812B 发光二极管照亮时，垃圾箱是半透明的，足以发光。四个发光二极管有助于为整个模型提供均匀的光线，并允许垃圾桶在不同的部分显示多种颜色。这意味着如果是绿色垃圾箱日，垃圾箱会发出绿光。如果是红色和蓝色箱子的日子，指示器会点亮红色和蓝色的部分。

不幸的是，目前还没有一个全球标准，让地方议会用来在互联网上提供垃圾日数据，所以配置起来并不像在当地垃圾管理局的网站上指向垃圾日图标那么简单。取而代之的是，一些节点红色代码被用来每天一次抓取议会网站，并告诉控制 BinDayCator 的 ESP8266 哪些垃圾箱应该被放置在路边。更高版本有一个自定义日历用户界面，允许手动配置相关日期。

BinDayCator 是一个可爱的设备，如果有一种简单而容易的方法来配置必要的数据馈送，它可能会在商业上获得成功。我们希望前门有一个，这样我们就不会错过另一个募捐日了。视觉指示器总是有用的，即使只是检查邮件。休息后的视频。
