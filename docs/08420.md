# 艺术机器人有颜料，会旅行

> 原文:[https://hack aday . com/2020/11/22/art-robot-has-paints-will-travel/](https://hackaday.com/2020/11/22/artistic-robot-has-paints-will-travel/)

创造力是人类的一个特征，也是许多机器人试图模仿的一个特征。一些专注于事物的大脑方面，与神经网络和机器学习合作，产生新的艺术输出。其他人从事机械方面的工作，建造可以在现实世界中出于艺术目的操纵工具的机器人。[Technovation]的最新产品属于后一类——一个由 Arduino 驱动的小型“机器人”，喜欢画画。

机器人在两个轮子上移动，每个轮子都由步进电机驱动，以实现精确移动。画笔本身由另一个步进器控制，步进器在颜料罐和画布之间旋转画笔。一个伺服系统被用来将画笔浸入盆中，并将其应用到画布上。一个 Arduino Uno 负责表演，机器人目前被编程为在画布上随机绘制各种颜色的线条。

凭借其粗纱设计，理论上它可以在任意大的画布上作画。这是一个平台，当与神经网络或者一些机器视觉配对时，可以证明它非常有能力创作更复杂的艺术品。[我们以前也见过其他的绘画机器人](https://www.instructables.com/Arduino-Powered-Painting-Robot/)。休息后的视频。
