# 现代磁带回声变得容易

> 原文:[https://hackaday.com/2021/08/21/modern-tape-echo-made-easy/](https://hackaday.com/2021/08/21/modern-tape-echo-made-easy/)

现代流行音乐越来越依赖越来越复杂和错综复杂的设备和算法来生成朗朗上口的曲调，但即使在几十年前也仍然如此。当时和现在唯一的区别是，过去的大多数设备是模拟的，而不是数字的。例如，不起眼的磁带回声最初是通过在录音头上绕一圈磁带，然后立即回放而制成的。那个时代的老式模拟机器越来越难找到，[所以【亚当·保罗】决定自己做](https://hackaday.io/project/181052-janky-tape-echo)。

起初，[Adam]计划在各种配置中使用标准盒式磁带，以达到预期的效果，但这被证明太麻烦了，他最终将他的设计改为在定制的磁带走带设备中使用盒式磁带内部。最终设计包括外壳内的一个小胶带环，电机驱动主轴。磁带通过记录磁头，然后是读取磁头，然后是擦除磁头，以获得回声。所有这些都是在设备内部完成的，提供了 1/4”插孔，这样音乐家就可以像配置标准效果踏板一样插入他们选择的乐器。

整个构建被设计为可使用现成的部件进行构建和修复，这解决了几十年前维护(或甚至找到)专用磁带 echo 机器的部件的问题。我们喜欢模拟设备的声音，也喜欢它在一个充满数字替代品的世界里仍然是模拟设备的事实。很像我们一年前推出的这款[磁带合成器](https://hackaday.com/2020/10/01/reel-in-the-years-with-a-cassette-player-synth/)。
