# 乙烯基切割机从剪贴簿转移到垫圈

> 原文：<https://hackaday.com/2020/03/30/vinyl-cutter-migrates-from-scrapbooks-to-gaskets/>

我们太了解这一切了:车库里又一个平稳运行的夜晚被一个坏掉的部件轻易中止了。在深夜，没有一家五金店会开门营业。虽然等待可能会回报病人，但创造力可能会回报那些现在就行动的人。一天晚上[贾斯汀]发现自己就在那里:垫圈破了。他并不沮丧，而是转向他的未婚妻[Amy],他们两人设法在仅仅几分钟内就在[Amy 的]乙烯基切割机上设计并切割出一个完全合适的替换垫圈。

在休息后的视频中，两人向我们详细介绍了他们的过程。从现有垫片的图像开始，他们捕捉参考图像。photoshop 中的一些简单工作清理了所有东西，除了他们正在寻找的结果垫片。最后，在测量现有尺寸后，在乙烯基切割机的软件中“通过眼睛”测量尺寸，使他们有足够的精度来重新制作一个与原始垫圈完全相同的重复垫圈。

我们似乎经常在路过或评论中听到乙烯基切割垫圈，但很高兴看到一个团队发布了这样一个神话般的成功故事，让它们得到了很好的利用。如果普通的老式乙烯切割刀片不能达到目的，为什么不试试用超声波速度运行呢？
