# econet——英国早期教育网络

> 原文：<https://hackaday.com/2020/04/16/econet-britains-early-educational-network/>

如果你比较美国和英国早期的个人电脑市场，你会注意到一个很大的不同。虽然许多美国学校有苹果电脑，但学校里也有大量其他电脑。在英国，几乎每所拥有电脑的学校都有一台 Acorn BBC Micro。[RetroBytes]带我们回到记忆中，解释这些学校如何以及为什么使用[Econet](https://www.youtube.com/watch?v=3GrtpuWxhHY)——一个在英国以外几乎不为人知的早期网络。你可以看到视频，其中包括对 Econet 所涉及的 Acorn 工程师之一的采访。

如今，你不必让人们相信社交网络的价值，但在当时，这并不是一件容易的事。大多数学校采用网络的驱动力是在计算机之间共享一个非常昂贵的硬盘驱动器。该网络使用 RS-422，这是苹果电脑、航天器和工业控制应用中常见的选择。

Econet 有很多我们今天认为理所当然的东西。电子邮件、聊天、远程桌面等等。理论上，速度可能高达 100 kB/秒。如果你想连接到远处的学校或主机，甚至还有一个 X.25 网关。奇怪的是，Econet 需要一个“时钟盒”，为网络上的所有站点提供数据时钟。

很难记得在以太网出现之前有这么多网络解决方案。每个人都有自己的赛马，从 ARCnet、Token Ring 和 Omninet，不一而足。有一些 IBM PC Econet 卡生产，但根据[RetroBytes]他们是昂贵的，很难找到。

然而，像所有网络一样，Econet 最终会屈服于以太网。不过，花了点时间。直到最近，Linux 内核甚至还支持它。

虽然 RS-422 在工业环境中很常见，但我们在这里并没有看到太多。当然，有一个 X 射线传感器。然后是[热感相机照相亭](https://hackaday.com/2017/12/01/purely-functional-selfies-thermal-printer-speaks-haskell/)，但那可能是 RS-485，它与 RS-422 相关，是 RS-422 的双向版本。
