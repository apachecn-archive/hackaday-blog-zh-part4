# 在 3 万美元的 8 GHz 示波器内

> 原文：<https://hackaday.com/2020/07/29/inside-a-30000-8-ghz-scope/>

互联网最大的好处之一——尤其是视频部分——就是你可以接触到很多你可能看不到的东西。以示波器为例。如果你幸运的话，你可能有一两个非常好的仪器在工作，如果它们工作正常，你肯定不会被允许把它们拆开。[信号路径]作为一个恰当的例子，[拆掉一个 30，000 美元的 MSO6 8 GHz 示波器](https://www.youtube.com/watch?v=8WNuGUQStq8)。

实际上，基本价格还不到 30，000 美元，但是当你装备了一个的时候，你可能会突破 30，000 美元的大关。与我们通常使用的示波器相比，这些是非常不同的。当然，屏幕更大，密度更高，但看电路板，它们更像某种高端计算机，而不是示波器。当然，在某种程度上，确实如此。

构建昂贵的 8 GHz 的真正诀窍是信号完整性。但设计中最明显的部分是热量管理。整个盒子里都是散热器和其他热管理设备。

内部的电路板实际上可以容纳六个输入，尽管示波器只设置了四个输入。不过，这里没有软件黑客。这些板缺少连接器和管理前端的特殊 IC。

这段视频将近一个小时，讲述了很多细节。查看模拟前端设计令人惊讶地具有启发性，特别是因为有两个未填充的部分，因此您可以在不移除任何器件的情况下轻松推断布线。

上世纪 70 年代到 90 年代，我们曾经认为我们在购买多余的泰克或惠普示波器。我们想知道这要多久才会成为节日庆典和易贝的主要商品？

如果你想将它与一个更普通的范围进行对比，看看这个 OWON 的[内部。如果你在市场上，你也可以去](https://hackaday.com/2020/01/15/owon-oscilloscope-teardown/)[买一些更实惠的东西](https://hackaday.com/2016/09/20/ask-hackaday-help-me-choose-a-scope/)。只是不要期望它看起来或执行起来像这个范围！
