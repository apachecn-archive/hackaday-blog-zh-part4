# 使用这款双耳麦克风进行身临其境的立体声录音

> 原文:[https://hack aday . com/2022/02/20/immersive-stereo-sound-recording-with this-dural-microphone/](https://hackaday.com/2022/02/20/immersive-stereo-sound-recording-with-this-binaural-microphone/)

录音已经成为消费技术很长时间了，现在它无处不在，伸手拿起移动设备，启动一个应用程序只需要一瞬间。任何对录音感兴趣的人都会发现，虽然制作简单的录音相对简单。但是，正如你们当中已经用斜带将一对舒尔 SM58s 做成 X 型配置的人所知，在录音棚外录音时，创造立体声效果是一项挑战。为了完善这一点，[Kevin Loughin] [创造了一个双耳麦克风，](https://www.youtube.com/watch?v=WV7bch5u8zU)它模拟了一个人的头部，将麦克风放在耳朵上，以产生几乎身临其境的立体声图像。

商用双耳麦克风的价格可能高达数千美元，但这款麦克风选择了更经济的设计，使用了为理发师出售的现成人体模型头。它充满了高密度泡沫，在它的耳朵里[凯文]放置了 3D 打印的耳道和驻极体麦克风胶囊。背面是电池和一个偏置电路盒。

正如你在休息时间下面的视频中听到的结果令人印象深刻，当然成本也是如此。这并不是我们向您展示的第一个这样的麦克风，请与使用纯泡沫头的麦克风进行比较。
