# 野猫强化喂食器

> 原文：<https://hackaday.com/2019/10/11/fortified-feeder-for-feral-felines/>

互联网上随处可见的大多数商用宠物喂食器都是廉价制造的，电机很弱，界面也很烦人。想在户外使用？祝你好运。如果你需要重型户外猫喂食器，[你必须加热你的烙铁，自己动手](https://www.instructables.com/id/Industrial-Strength-Cat-pet-Feeder/)。

[jplanaux]根据合同喂养一群在附近游荡的野猫，但他经常一去就是几个星期。他的双馈电线故障转移系统有一个薄弱环节，那就是这些商业馈电线——它们功率不足，完全不可靠，即使在为 Raspi 控制进行改装后也是如此。他需要的是一个完全根据他的情况定制的工业级自动喂食器。

一个简单的网络界面让他可以设置自动喂食时间，或者在顾客出现但没有食物的情况下按需推送狗粮。该系统对碗进行拍照，以验证食物是否流出并随后被吃掉。它应该是防浣熊的，所以[jplanaux]可以看到谁或什么东西在吃。除此之外，进料器是非常标准的，在 NEMA17 驱动的螺旋驱动装置顶部有一个大漏斗。步进器是继电器驱动的，所以它只在驱动螺杆的时候使用能量。

[jplanaux]有 STL 文件和代码可用，甚至为想要建造一个碗和底座并在室内使用的人设计了一个碗和底座扩展。休息之后，慢慢品尝 kibble 大小的演示视频。

我们在这里咀嚼的自动喂食器大部分是为干粮设计的。提供湿食物是一个更困难的问题，但是绝对有可能完成。
