# 您不知道自己需要的 3D 打印绘图仪

> 原文：<https://hackaday.com/2018/10/15/the-3d-printed-plotter-you-didnt-know-you-needed/>

最近，我们已经看到了重新设计的 3D 打印机的涌入。节俭的黑客一直在利用廉价的 3D 打印机作为一种方式来引导他们的构建，从激光雕刻机到拾取和放置机器。这并没有什么错，老实说，由于规模经济，当你可以以低于单独部件成本的价格获得一台便宜的 3D 打印机时，你不这样做就是愚蠢的。

但是，对于使用打印部件和光滑的杆子来建造东西的经典 RepRap 思维，还是有一些东西要说的。举个例子，[【达斯·维德】送来给我们观赏的大型 3D 打印绘图仪](https://grabcad.com/library/drawbot-plotter-1)。这不是有人在他们的开箱独价特别挤出机上吐笔，这是一个专门建造的绘图仪，它显示。在休息后的视频中，你不仅可以看到它画得有多好，还可以看到与修改后的 3D 打印机相比，它的工作区域有多大。

如果你熟悉 3D 打印机，你应该对它非常熟悉。使用 3D 打印机中普遍存在的相同的 GT2 带、步进器、终点停止开关和线性轴承，应该不难找到零件来构建自己的打印机。它甚至使用运行 Marlin 1.1.9 的带有 RAMPS 1.4 的 Mega 2560 进行控制。

最大的区别是物理布局。由于没有沉重的热端或挤压装配四处移动，绘图仪有一个悬臂设计，使其更大的范围。因为它只需要轻轻将笔从纸上提起，所以也不需要复杂的带丝杠的 Z 轴；一个简单的伺服系统安装在手臂的末端，用来提升和举起笔。我们尤其喜欢使用卷尺来减轻他的布线压力，这是一个我们(和你们中的许多人)去年爱上的绝妙技巧。

虽然就便利性而言，很难打败仅仅将一支笔扔进你的桌面 3D 打印机的商业端，但我们认为从这个构建中可以很清楚地看出，结果并不完全相同。如果你想要一个真正的绘图仪，就造一个真正的绘图仪。
