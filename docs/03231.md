# 机甲战争:就像驾驶一个建筑大小的机器人穿过一个繁忙的城市

> 原文:[https://hack aday . com/2019/06/07/mech-warning-like-driving-a-building-size-robot-through-a-busy-city/](https://hackaday.com/2019/06/07/mech-warfare-like-driving-a-building-sized-robot-through-a-busy-city/)

一组机器腿敲击混凝土发出的声音是惊人的。不为别的，休息后的视频就为这个值得一看。当我第一次在 Maker Faire 的[机甲战](http://rteamrobotics.weebly.com/mech-warfare.html)区闲逛时，这引起了我的注意，因为在设置日期间，一个竞争对手让他们的机器人四处闲逛作为演示。

 [![Installing target plate](../Images/6db18ad95368030319a155c3bd1a67cd.png "mech-warfare-installing-target-plates")](https://hackaday.com/2019/06/07/mech-warfare-like-driving-a-building-sized-robot-through-a-busy-city/mech-warfare-installing-target-plates/) Installing target plate [![Making sure the ammo is set to go](../Images/7a2f3e322a3a4b28a44d5279a6e823db.png "mech-warfare-target-plates-installed")](https://hackaday.com/2019/06/07/mech-warfare-like-driving-a-building-sized-robot-through-a-busy-city/mech-warfare-target-plates-installed/) Making sure the ammo is set to go

这是真正的黑客机器人竞赛。有约束，但也有很大的自由空间。满足一打左右的要求，剩下的你就可以尽情发挥创造力了。我最喜欢的部分是，这不是一个破坏性的事件，不像许多基于战斗的机器人电视节目，往往会让我反胃。相反，这些机器人每个都携带一把电动气枪，并试图击中竞争对手机器人上的四个目标面板中的任何一个。

[![](../Images/2436964ee1fb16d6e64949ecf007f1b5.png)](https://hackaday.com/wp-content/uploads/2019/06/mech-warfare-operators-during-battle.jpg)

Robot operators are behind a sheet and only view through the robot’s cameras.

这些让我想到了《龟壳里的幽灵》中的巨大四足坦克 Tachikoma。不仅仅是腿的数量——限制在四条或更少——具有相似性。每一轮，机器人都被放置在一个城市的微型模型中，操作员使用我习惯在飞行无人机中看到的 FPV 硬件。这种设置给了他们驾驶建筑大小的机甲的效果，观众可以跟着安装在记分牌旁边的监视器观看比赛。
