# 查看气体调节器内部

> 原文：<https://hackaday.com/2021/07/26/seeing-inside-a-gas-regulator/>

我们被有趣的工程所包围，但其中一些被密封在一个外壳内，远离容易检查的地方。一个恰当的例子；不起眼的气体调节器。它存在于我们周围的设备中，从丙烷烤架到焊接设备。这是一个密封的装置——你见过它的内部，知道它是如何工作的吗？感谢[农场工艺 101]，在休息之后的视频中，[我们可以这样做](https://www.youtube.com/watch?v=--ylSk8nTbs)。

从本质上说，这是一个液压杠杆。大型隔膜由调整器的低压侧加压，并由弹簧支撑。当与环境大气相比的压力高到足以克服弹簧张力时，杠杆倾斜，关闭高压阀。因此，压力由弹簧强度决定。我们也看到了系统是如何失败的——在这种情况下，似乎是一些砂砾干扰了阀门。我们发现隐藏的工程学非常令人满意，特别是当我们像在这里一样如此清楚地理解它的时候。尽情享受吧！
