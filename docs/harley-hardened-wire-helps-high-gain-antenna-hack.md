# 哈雷硬化线帮助高增益天线黑客

> 原文：<https://hackaday.com/2018/07/18/harley-hardened-wire-helps-high-gain-antenna-hack/>

哈雷戴维森摩托车和建造天线有什么关系？绝对没有，除非你碰巧有一个，需要[加工硬化铜线，为 LoRa](https://www.youtube.com/watch?v=lVD-dnYLv-U) 建造一个共线天线。

我们会解释的。安德烈斯·斯皮斯从来就不是一个安于现状的人，他需要一个更好的天线来进行他的劳拉实验。为了寻求高增益和全向模式，他购买了一种商用共线天线，这是一种带有精确间隔环的导线，其作用就像一堆偶极子。可悲的是，在一次面对面的测试中，[Andreas]发现商用天线在覆盖范围方面并不比低增益天线好，所以他决定自己开发。

铜线是一种很好的天线材料，因为它不需要特殊工具就可以很容易地成型，而且焊接起来像冠军一样。但是你在家庭中心得到的东西还远远不够硬，不足以支撑一个独立的垂直鞭子。这就是哈雷机车的用武之地:[Andreas]用他的猪拉出 1.75 毫米直径(比#14 AWG 稍大)的铜线。加工硬化不仅强化了导线，还将其直径减小到了天线设计所需的 1.4 毫米。[他的矢量网络分析仪](http://hackaday.com/2018/03/29/putting-a-poor-mans-vector-analyzer-through-its-paces/)告诉他，为了让天线在 868 MHz 谐振，需要接地层元件和稍微调整一下环路直径，但最终看起来天线有望提供 5 dBi 的增益。

当然，还有很多其他方法来延长电线——你可以用悬挂的重物来延长它，或者如果你有野心的话，甚至可以用[卡丁车电动绞盘](https://hackaday.com/2011/03/01/super-winch-makes-sledding-100-more-fun/)来延长它。但是如果你有一辆那样的自行车，为什么不炫耀一下呢？
