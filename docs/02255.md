# 与你的视野交谈，它会服从

> 原文：<https://hackaday.com/2019/03/03/talk-to-your-scope-and-it-will-obey/>

示波器是一种我们很多人都使用的设备，当我们的手忙于测试探针或其他工具时，我们经常不得不使用它。【詹姆士·威尔森】[通过将它连接到运行](https://github.com/jmwilson/ollie) [snips.ai](https://snips.ai/) 语音助手的树莓 Pi 3，解决了如何控制他的“范围徒手的问题。这是一个有趣的软件，与 Alexa 或 Google Assistant 等公司提供的云服务相比，它可以在设备上本地运行。

我们正在讨论的示波器是一个是德科技 1000-X，可以在下面的视频中看到，但是查看 Python 代码，我们可以想象同样的技术被带到其他仪器上，例如一两年前我们看到的通过 USB 控制的 Rigol 1054z。snips.ai 软件的使用为我们社区中的语音控制项目如何超越云服务提供了一个指针，有趣的是，尽管他们没有小题大做，但他们的软件[似乎是开源的](https://docs.snips.ai/additional-resources/terms-of-use)。

示波器不一定只能通过语音遥控。不用手进行测量似乎是一种普遍的愿望——我们过去介绍过的一个项目[用脚踏开关](https://hackaday.com/2018/12/02/keep-both-hands-on-the-probes-with-this-oscilloscope-footswitch/)完成了这项工作。
