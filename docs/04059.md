# 蒸汽朋克收音机看起来很不错

> 原文:[https://hack aday . com/2019/08/30/steam punk-radio-looks-the-business/](https://hackaday.com/2019/08/30/steampunk-radio-looks-the-business/)

总的来说，收音机不是由蒸汽驱动的。有人可能会说，我们的大部分市政电力供应确实来自蒸汽涡轮机，但这可能是一个长期的问题。不管怎样，蒸汽朋克仍然是一种受欢迎和有吸引力的审美，[，这也是【克里斯汀】为她的收音机设计选择的审美。](https://www.instructables.com/id/SteamPunk-Radio/)

建造婴儿床从[克里斯汀的]早期工作的 VFD 闹钟，使用类似的电子管和驱动芯片来运行显示。调频收音机和扩音功能由便捷模块提供。灯管出于美观的目的而安装，巧妙地用少量变色发光二极管点亮。也许最简洁的触摸是使用阀门手柄来控制调谐和音量。步进电机转动一系列齿轮，这是任何真正的蒸汽朋克制作所必须的，甚至还有一个电磁致动器来使莫尔斯键移动。为了运行这一切，一对 Arduino Megas 负责处理所有不同系统的 I/O 需求。

这是一个奇特的建筑，它展示了在追求一个特定的外观和感觉时，你可以走多远。这是一台可以成为任何黑客咖啡桌上的绝佳谈资的收音机。如果这还不够的话，[考虑买一台完整的笔记本电脑](https://hackaday.com/2019/06/09/homebrew-laptop-makes-a-statement-with-a-steampunk-theme/)。休息后的视频。
