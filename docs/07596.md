# 在鸡蛋上激光蚀刻照片

> 原文：<https://hackaday.com/2020/08/28/laser-etching-photos-on-to-eggs/>

数控机床是每个制造商的梦想。能够通过最少的手工劳动从 CAD 设计中生产出精确的零件，它们是车间里的一个很好的工具。或者，你可以把它们用于更有趣的追求。[Leo]的项目是后一种——在鸡蛋上蚀刻灰度照片。

你首先需要的是一台兼容鸡蛋的数控机床。Eggbot 是一个受欢迎的选择，否则现有机器上的第四轴也可以像[Leo]的情况一样完成这项工作。耦合鸡蛋是一项微妙的任务，为此，一些橡胶纸卷从一台旧打印机中抢救出来并投入使用。然后，需要在 CNC 头上安装一个激光器，并用探针绘制鸡蛋深度，以确保整个蚀刻过程都在焦点上。接下来，只需加载一张图像，将灰度数据转换成相关的 g 代码，然后将其刻录到鸡蛋上。

使用涂有黑色墨水的鸡蛋，结果令人印象深刻。这些彩蛋将会成为一个有趣的复活节礼物，或者作为一种廉价的方式来教授学生数控技术。显然，眼睛保护是必须的，并且确保安装你的激光器安全，以避免任何无意的暴露。休息后的视频。
