# 自制液相色谱仪

> 原文：<https://hackaday.com/2021/03/22/build-it-yourself-lc-meter/>

一个基本的数字万用表(DMM)通常是有抱负的电子修理工购买的第一个测量工具。即使是廉价的数字万用表也能很好地测量电压、电流和电阻；检查连续性；甚至可以具有测量晶体管增益的模式。每个工具箱至少需要一个数字多用表，但大多数都有一个关键的限制——它们不能测量两个基本的电学量:电感和电容。在 Hackaday.io 上，[core weaver]开发了一款开源 LC 电表，允许您构建自己的工具来测量电感和电容。

[core weaver]的设计都是通孔的，所以即使只是组装一个也是对初学电子的人的一个很好的锻炼。然而，他不仅仅是发布了一个设计，[在一系列视频](https://www.youtube.com/playlist?list=PLkaYf9fE05ZFBNix39xXy-Ew_vBT5_2YM)中，他讲述了设备操作的理论；解释电路、固件和机箱的设计；并向您展示如何将它们组合在一起。当你需要测量很多零件的时候(例如，如果你必须整理一袋便宜的电容器来寻找特定的价值)，他甚至开发了一个桌面程序来帮你省去一些麻烦！

成品米看起来不可思议！如果你想为自己制作一个，他已经把所有的文件上传到了 GitHub 上，我们强烈建议你在休息之后看看他的第一个视频。如果你想为自己打造一个 6.5 位数字多用表来搭配我们的 LC 电表，可以考虑[的这款，它甚至有一个自制的 ADC。](https://hackaday.com/2020/08/02/a-diy-6-5-digit-multimeter-is-a-lesson-in-clever-circuitry/)
