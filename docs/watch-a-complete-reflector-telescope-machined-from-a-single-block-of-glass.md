# 观看由一块玻璃加工而成的完整的反射望远镜

> 原文：<https://hackaday.com/2022/04/21/watch-a-complete-reflector-telescope-machined-from-a-single-block-of-glass/>

如果这是用单片玻璃制作一个完整的反射望远镜的简单部分，我们迫不及待地想知道最难的部分！

对于那些不关注 YouTube 上[耶鲁安·弗勒格尔]的*惠更斯光学*频道的人来说，一点背景故事可能是必要的。几个月前，他发布了[一个讨论单片望远镜](https://www.youtube.com/watch?v=HxwhCmO90UQ)的视频，其中所有的反射和折射表面都被研磨成一块厚玻璃。光学工程师同事[里克·特尔·霍斯特]已经制作了几个微型单片施密特-卡塞格林反射镜，用于立方体卫星，所以[耶鲁安]决定自己制作一个放大版。

建造从一块 45 毫米厚的冕状玻璃开始，用金刚石孔锯从中钻出一个 50 毫米的圆柱体。然后，坯件的表面被打磨成复杂的曲线，以反射入射光，首先反射出抛物线形的后表面，然后反射到打磨到前表面中心的双曲线副镜上。最后通过背面中心的折射表面，完成光子通过玻璃块的旅程，将 275 毫米的焦距压缩到一个紧凑的封装中。

当然，所有这些都大大低估了实现这一目标所需的工作量。从最初计算表面形状到加工玻璃这种众所周知的无情材料，每一步都充满了危险。因为设计是整体的，任何错误都意味着从头开始。看看下面的视频，惊叹于获得这样的结果所需的技能。

[耶鲁安]的视频给我们印象最深的是高科技和古老的光学制作方法和材料的混合，我们已经看到他用这些方法和材料制作了从[微型特斯拉阀](https://hackaday.com/2021/06/20/tiny-tesla-valves-etched-in-glass/)到[可变表面镜](https://hackaday.com/2020/06/27/variable-mirror-changes-shape-under-pressure/)的一切东西。
