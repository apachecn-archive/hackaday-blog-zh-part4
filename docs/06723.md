# 美味的输出设备

> 原文：<https://hackaday.com/2020/05/30/a-tasty-output-device/>

我们为你的耳朵准备了耳机，为你的眼睛准备了监视器。有些电脑甚至有触觉反馈。现在研究人员有了味觉输出装置。这个非常奇怪的装置使用了五种凝胶，每种凝胶代表一种人类可以感知的味道。如果我们理解这篇论文，诀窍是电离凝胶抑制凝胶的味道。通过控制每种凝胶的电离水平，你可以合成任何味道，就像你可以用三个 led 产生颜色一样。

这五种凝胶由琼脂和甘氨酸(甜)、氯化镁(苦)、柠檬酸(酸)、盐(咸)和谷氨酸钠(鲜味)制成。如果你在学校没有学过鲜味，那是一种类似肉汤或肉的美味，通常与谷氨酸钠联系在一起。

该设备的形状像寿司卷，这样当凝胶接触舌头时，铜箔阴极也可以连接。使用这个会让你看起来比戴谷歌眼镜的人更奇怪，但我们认为这是站在科技前沿的代价。似乎没有任何理由你不能复制这样的东西，尽管我们怀疑在聚会上传递它是否卫生。也许你的下一部家庭电影可以展示一顿饭，让观众也品尝一下。

如果你想知道嗅觉，那是另一组研究人员的研究。你会认为这是我们见过的第一个味觉输出装置，但不…令人惊讶的是，[它不是](https://hackaday.com/2014/02/27/taste-o-vision-is-now-a-thing/)。
