# 1983 年贝尔实验室的集成电路制造

> 原文：<https://hackaday.com/2022/08/23/integrated-circuit-manufacturing-at-bell-labs-in-1983/>

随着技术进步的永无止境，可以说最复杂的技术变得如此接近魔法，以至于对他们所在行业之外的人来说是不可理解的。我们已经看到了现代半导体晶圆厂运作的一小部分的预排视频快照，但让我们面对它，你看到的一切都非常谨慎，出于环境控制等原因隐藏在大型密封盒中，很难真正看到里面发生了什么。

让我们回到几十年前的 1983 年，带着[在默里山的贝尔实验室](https://www.youtube.com/watch?v=TiOjevNkK9I)有趣地参观 IC 制造工厂(视频，嵌入下面),你可以更好地了解这个过程是如何工作的，尽管当时芯片只托管几万个有源设备，而今天有数十亿个。这个工厂在三英寸的晶片上操作，每个晶片生产大约 100 个管芯，每个管芯都由手工处理和加工，而现代晶片要大得多，管芯通常小得多，每个晶片的总管芯数千个，并且从不由肮脏的人处理。

以现代标准来看，每立方英尺 100 个的粒子数似乎很可笑，但当时的设备几何尺寸相对较大，因此缺陷率并不严重。看到操作人员都穿上防护服，但却露着脸，很多胡子在微风中露出来，我们不禁有些窃笑。很简单，完全没有必要穿兔女郎套装。无论如何，虽然外套主要是为了保护环境，但我们确实偶尔看到一个操作员在执行一些高风险任务时戴着适当的防护面罩，如晶片清洗，毕竟正如旁白所说的“这些酸足以腐蚀皮肤”，这肯定会毁了你的下午。

如果不提及[Sam Zeloof]和[他自己动手制作芯片的方法](https://hackaday.com/2021/08/14/the-zeloof-z2-intergrated-circuit-has-100-transistors/)的进展，任何关于集成电路处理的故事都是不完整的，尽管他只管理数百个设备，但这只能在一定时间内改善。
