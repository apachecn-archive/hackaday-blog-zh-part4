# 收拾茶几，弹球时间到了

> 原文:[https://hack aday . com/2021/12/28/clear-off-the-coffee-table-its-pinball-time/](https://hackaday.com/2021/12/28/clear-off-the-coffee-table-its-pinball-time/)

像我们许多人一样，[BuildXYZ]一直想拥有一台弹球机，但没有足够的空间来购买这样一个又大又重的玩具。但是有志者事竟成。[BuildXYZ]认为[如果他们能把弹球机做成咖啡桌形状的](https://www.buildxyz.xyz/pinball-coffee-table/)，他们至少已经成功了一半。

[BuildXYZ]没有选择任何引脚。在做了大量研究后，他们选定了 1981 年的 Bally Centaur，因为它是早期的固态机器，也是最好的机器之一。它没有二级游戏场水平来处理，使它更容易做这个项目。

我们从哪里开始描述这种美丽的爱的劳动呢？有太多的细节要列出来，但要知道，这似乎是等量的修复工作和定制工作，使这张桌子在一起。休息后的构建视频绝对值得你花时间，你会更好地理解为此花费的时间，从 FPGA 上构建的自定义乐谱解码器芯片到 3D 打印的替换掉落目标和新的丙烯酸位，以替换操场上变黄的那些。

[![](../Images/1a948cd3fc434c2ff22ab601f1dfd2e4.png)](https://hackaday.com/2021/12/28/clear-off-the-coffee-table-its-pinball-time/pinball-rotisserie/)[![Pinball circuit boards built into a set of drawers in a steel frame.](../Images/3716d7e998ed192fdf803f91e5b990e3.png)](https://hackaday.com/2021/12/28/clear-off-the-coffee-table-its-pinball-time/pinball-table-drawers/)

[BuildXYZ]从建造一个 pintisserie 开始，这听起来就像是一个旋转烤肉叉，用于弹球机的内脏，使其工作变得轻而易举。这种易于维护的特点再次体现在新的机柜设计中，电路板被固定在一对抽屉上。

不，你不需要弹平。但你必须在按下开始键之前清理顶部，因为一对微型工业线性致动器根据设置将后端升高 5-7 度。我们对缺少 plunger 感到有点难过，但是[BuildXYZ]是对的——它会敲你的膝盖骨。从好的方面来看，[BuildXYZ]重用了“自由球”螺线管作为球发射器，它由那个闪亮的金属按钮驱动。再次强调，请务必在休息后查看。

老实说，自从这台完全由 K ' NEX 建造的弹球机以来，我们还没有见过如此多的工作进入弹球机。
