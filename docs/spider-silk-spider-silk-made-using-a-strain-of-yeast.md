# 蜘蛛丝，蜘蛛丝，用一种酵母菌制成

> 原文：<https://hackaday.com/2020/11/06/spider-silk-spider-silk-made-using-a-strain-of-yeast/>

公司花费数千美元为市场开发一个项目，希望他们的投资能有大回报。这样的投资每天都在发生，不会让任何人震惊。让你感到惊讶的是，有人花了十几年时间，自己花了几千美元，制作了一个非常畅销的产品的开源版本。在这种情况下，我们谈论的是生产蜘蛛丝的转基因酵母。如果这听起来像是一些蜘蛛侠笑话和科幻参考的前奏，那么你在这两个方面都是正确的。[Justin Atkin]开始时有一些遗传学家的工作，所以他计划遵循熟悉的程序，如提取黑寡妇 DNA，分离和复制丝基因，并将它们粘贴到酵母菌株中。很简单，对吧？自然，好的科学不会在一夜之间发生。

最强蜘蛛丝有几个竞争者，其中金色丝线织球者最受关注，但黑寡妇的织网几乎一样强，[贾斯汀]很乐意穿黑寡妇启发的珠宝，而金色织球者看起来像是从 *星河战队* 中爬出来的。他第一次尝试提取 DNA 是从一小瓶保存的~~噩梦燃料~~蜘蛛标本开始的，因为这种东西你可以在网上买到。可悲的是，它们被酒精制成蜜饯，这抹杀了 DNA，所以他从饲养员那里转移到干燥的标本，这也没有产生结果，这些只是里程碑式的难题。

在经历了所有的挫折和死胡同之后，【Justin】和他的实验室团队另辟蹊径，设计了一个按订单生产的质粒。哎，你可以在网上买到任何东西。构建这些序列的实验室不像一个定制 t 恤的生产商，因为你不能只给他们发*【Gattaca*。TXT 和一张支票。实验室将拒绝具有重复片段的序列，蜘蛛丝是长的重复链。[贾斯汀]必须让他的重现丝绸看起来独一无二，所以他必须调整自然的配方。

丝并不是一个单一的东西，而是从喷丝头出来的东西的一种分类。[蜘蛛](https://hackaday.com/2018/04/04/its-a-spider-its-a-droideka-its-both/)产的丝多达十个品种，而且是单体的组合。有些是粘性的，有些是有弹性的，有些是坚固的，但是对于商业上可行的东西，我们感兴趣的是坚固而没有弹性的拖丝。可以肯定的是，这就是彼得·帕克在他的[网兜](https://hackaday.com/2017/10/07/spidermav-drone-shoots-webs-to-perch-and-stabilize/)中装载的东西。这种多样性来自于对强拉伸序列的排序和再排序，以增加可变性，但仅此而已。[Justin]为这种混合添加了一种新的贡献，这种贡献以生物矿化的形式出现，我们在珍珠质(珍珠之母)、甲壳质和骨骼中可以看到这种形式。这是大自然赋予动物的最强壮的东西之一，所以改进的配方在混合中有一点额外的力量。另一个因素是告诉酵母分泌蚕丝的序列，而不仅仅是在体内产生。这使得收割区别于挤奶和屠宰。

当实验室的质粒到达时，他们将它混合到一些巴斯德毕赤酵母[酵母](https://hackaday.com/2020/01/08/engineering-your-way-to-better-sourdough-and-other-fermented-goods/)中，这种酵母通常是粉红色的，但经过改造的菌株是骨白色的。这是一种易变的菌株，不想接受新的基因，但 Youtube 评论员提供了有用的协议。这本身就是一种成就，就好像你能得到 YT 评论版块提供有用的数据，你值得一座奖杯。在新的程序之后，一系列的测试表明基因改造起了作用。[贾斯汀]的酵母以可观的速度生产丝绸。哇哦。

现在，计划是想办法用大规模生产的丝绸做点什么。我们喜欢种植坚韧的工装裤或黑寡妇网络摄影师的想法。不包括蜘蛛感官。
