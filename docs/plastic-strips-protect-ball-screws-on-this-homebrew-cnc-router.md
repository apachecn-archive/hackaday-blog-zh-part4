# 塑料条保护这个自制数控路由器上的滚珠丝杠

> 原文：<https://hackaday.com/2020/10/26/plastic-strips-protect-ball-screws-on-this-homebrew-cnc-router/>

这是 CNC 路由器拥有者的生活事实。无论是一张中密度纤维板上的细小灰尘，还是一块铝片上的脂肪碎片，你的工具产生的碎片到处都是。你可以试着从源头上控制它，但斯沃夫总会想办法逃脱并引发问题。

Nikodem Bartnik 不愿意处理他自制的数控镂铣机昂贵的滚珠丝杠上堆积的碎屑，他自己动手制作了这些 DIY 伸缩滚珠丝杠套。是的，商业滚珠丝杠盖是可用的，但它们针对的是专业机器，因此不仅对像他这样的自制机器来说太大，而且价格也超出了专业预算。所以[Nikodem]重新创造了他们的基本设计:将薄材料条缠绕成一个紧密的弹簧，形成一个可以伸缩的管子。第一个原型是用纸做的，可以工作，但被证明有太多的摩擦。第二版是由聚酯薄膜制成的，足够光滑以完成工作，另外，它是透明的。它们看起来非常锋利，正如你在下面的视频中看到的，似乎表现很好。

很高兴看到构建进展到可以解决这种细节的地步。我们已经跟随[【niko DEM】的 CNC 构建很多年了](https://hackaday.com/2018/11/16/turning-a-rotary-tool-into-a-cnc/)，它真的走了很长的路。
