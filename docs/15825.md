# 在 LCD 树脂打印机上连续打印:不再浪费剥离时间？可能吗？

> 原文：<https://hackaday.com/2023/01/03/continuous-printing-on-lcd-resin-printer-no-more-wasted-time-on-peeling-is-it-possible/>

任何使用 SLA 打印机进行过 3D 打印的人可能都很清楚每一层的剥离步骤。这包括将新印刷的层从附着在树脂槽底部的 FEP 薄膜上拉开。由于所涉及的力，构建板在 Z 轴上的缩回速度必须仔细调整，以避免发生可怕的事情，例如物体被拉离构建板。最终这是限制 SLA 打印速度的原因，然而[Jan Mrázek]认为用富氧层[替换 FEP 可以有所帮助。](https://blog.honzamrazek.cz/2022/12/continuous-printing-on-lcd-resin-printer-no-more-wasted-time-on-peeling/)

原理相对简单:氧气的存在会抑制树脂的固化，这就是为什么要在低氧环境中快速固化树脂部件，例如浸没在水中。Carbon 的商业打印机使用一种称为“连续液体界面生产”(CLIP)的专利方法，其他公司的树脂打印机使用各种其他(也获得专利)方法，减少或消除剥离的需要。理论上，通过使用透氧层代替 FEP 薄膜，即使是消费级 SLA 打印机也可以跳过剥离步骤。

[Jan]最初试图制造一种能渗透氧气的硅树脂薄膜来代替 FEP 薄膜，大约 10 层效果很好，直到树脂可用的氧气耗尽，剥离力变得太大。接下来的尝试包括试图创造一种氧气补充机制，但不幸的是至今没有太大的成功。

不管这些挫折，这是一个有趣的研究方向，可以使廉价的消费级 SLA 打印机更加有效。
