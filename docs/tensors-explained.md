# 张量解释道

> 原文：<https://hackaday.com/2020/01/29/tensors-explained/>

你可以在不接触张量的情况下做很多电子学的工作，但是在某些情况下，张量是绝对必要的。问题是大部分数学课本给你的描述非常干巴巴，很难内化。[这就是【科学庇护】的由来](https://www.youtube.com/watch?v=bpG3gqDM80w)。他们最近的视频(见下文)从干巴巴的定义开始，然后向你展示它的含义和原因。

根据视频，教科书的定义是:

> m 维中的秩 n 张量是具有 n 个指数和 m ^n 个分量的数学对象，并且遵守某些变换规则。

这听起来很像一个数组，但我们不确定“某些转换规则”对任何人来说到底意味着什么。

维基百科做得好一点:

> 描述从一组代数对象到另一组代数对象的线性映射的代数对象。

这些构造是任何电磁事物(包括天线)的关键，并在应力计算和量子力学中大量出现。甚至爱因斯坦的相对论也使用张量。

这段视频花了十几分钟。你甚至可以得到一些四面体骰子和篮球的应用。

令人遗憾的是，许多数学和电子学似乎变得比必要的更难。威尔·斯韦特曼的微积分系列很好地揭示了这个主题。话又说回来，你可以用[一个电脑拐杖](https://hackaday.com/2019/05/11/mathics-how-to-do-hard-math-when-youre-not-an-mit-janitor/)，但那并不能帮助你理解，真的。
