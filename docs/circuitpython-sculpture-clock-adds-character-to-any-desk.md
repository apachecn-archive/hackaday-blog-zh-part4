# CircuitPython 雕塑时钟为任何桌子增添个性

> 原文：<https://hackaday.com/2019/11/21/circuitpython-sculpture-clock-adds-character-to-any-desk/>

我们可能都与这个故事的起源有关。[亚历克斯]正在做一个更大、更复杂的时钟项目，这时这个可爱的小座钟创意引起了他的注意。谁不想要一个有个性和一点点态度的钟呢？

这个小家伙的大脑是一个智能的 M0 快车，他从一个羽翼式 RTC 记录仪中获取时间数据。这些天线不仅仅是为了看——[ Alex]选择了 ItsyBitsy，因为它可以在没有额外组件的情况下轻松地进行电容式触摸。在休息后的简短演示中，[Alex]展示了在切换到时钟脸之前，触摸它们是如何触发动画脸或静止脸的。

我们喜欢功能性电路雕塑，尤其是具有如此多特征的。[Alex]受到了[[Mohit bho ite]令人惊叹的赛道雕塑](https://hackaday.com/2018/11/19/flywire-circuits-at-the-next-level/)的启发，似乎效仿了他先在纸上展示一切的伟大榜样。顺便说一下，我们在 Supercon 主演[Mohit]和他的电路雕塑之前的最后一次聊天。错过了？在这里阅读文字记录。
