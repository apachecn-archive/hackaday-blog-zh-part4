# 把激光换成涡流加农炮，你就有了…升力？

> 原文：<https://hackaday.com/2022/02/28/swap-the-laser-for-a-vortex-cannon-and-you-have-lift/>

当人们思考太空旅行的未来时，一个想法是一艘带有巨大太阳帆的宇宙飞船，由巨大的激光推动。受到这个概念的启发，但缺少巨型激光器，[汤姆·斯坦顿]建造了一艘由涡流加农炮驱动的小型飞船。

创造一个漩涡已经够难了，但是创造一个足够强大的漩涡来移动更长的距离并推动某物需要一些努力。[Tom]从一些便宜的螺线管开始，但是有一些问题。它们的内部喷嘴很小，限制了气流。他用四个阀门连接在一起来提供所需的空气量。此外，他发现他们的响应时间不足。他们不能足够快地关闭，所以它不是喷出一股空气，而是喷出更接近溪流的东西。为了弥补这一点，[Tom] 3d 打印并尝试了几种不同尺寸的锥形喷嘴，看看是否有帮助。不幸的是，事实并非如此。所以他把喷嘴和一个膨胀室结合起来，膨胀室可以让压力波变短，然后它变窄，再次加速。这提供了一个像样的漩涡。

接下来[汤姆]转向他的手艺。在设计了一个 3d 模型后，他有了一个模板，可以从纸上剪下一些形状，并用胶带粘在一起，形成一个可以捕捉漩涡的轻型车辆。最初的测试并不太有希望，因为飞行器扭曲了，它行进的绳子有太多的摩擦。切换到垂直测试显示了更多的希望，但试图快速产生多个漩涡是不成功的，因为先前环的湍流打破了新的环。

那么，我们能从中学到什么呢？他似乎没什么可展示的。[汤姆]调整和迭代他的工作方式，并继续完善他的工艺。希望在未来，我们能看到一个功能齐全的版本。教训是不断列举各种可能性。就像[这个基于网络摄像头的姿态传感器不断迭代，走向成功](https://hackaday.com/2014/05/06/a-webcam-based-posture-sensor/)。休息后的视频。
