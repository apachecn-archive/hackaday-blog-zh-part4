# Remoticon 视频:KiCad 到 Blender PCB 渲染

> 原文:[https://hack aday . com/2020/12/23/remoticon-video-kicad-to-blender-PCB-renders/](https://hackaday.com/2020/12/23/remoticon-video-kicad-to-blender-pcb-renders/)

如今，我们似乎希望我们的 PCB 设计软件做所有的事情，而且它*几乎*做到了。您不仅可以布置所有内容，检查电气和设计规则，甚至可以列出材料清单，而且许多 PCB 工具可以生成足够好的 3D 模型来检查零件间隙，或者在设计外壳时非常有用。但是，当涉及到制作照片级逼真输出时，无论是为了广告还是仅仅为了赏心悦目，您可能需要求助于 3D 设计工具。

[![](../Images/856ef1d6407d645e0d9654b1e7bbacbe.png)](https://hackaday.com/wp-content/uploads/2020/12/tiny_blinky_06_283_featured.png) 在这个工作坊中， [Anool Mahidharia 将 KiCad 的 VRML 输出，在 Blender](https://www.youtube.com/watch?v=mVpT5dw4sew) 中渲染，然后开始调整结果，直到你几乎不能确定这是真实的东西还是 3D 模型。他从 KiCad 中的一块板开始，包括在[项目的 GitHub repo](https://github.com/Hack-a-Day/Remoticon_KiCad_Blender) 中，你可以跟随基本的导入，或者一直复制 ATtiny85 顶部的图形，并确保通孔的内部与顶部匹配。

如果你不了解 Blender，也许你不知道它是一个多么全面的 3D 建模和动画工具。伴随着不可思议的力量而来的是众所周知的攀登高山的陡峭的学习曲线。Anool 甚至没有试图把你变成一个搅拌机专家，而是专注于你需要做出好看的 PCB 渲染的调整和技巧。你可以在网上到处找到通用搅拌机教程，但是如果你想要 PCB 专用的，你来对地方了。
