# 光束驱动的抛球光束让我们发出光束

> 原文：<https://hackaday.com/2021/04/13/beam-powered-ball-flinging-beam-has-us-beaming/>

我们对光束项目情有独钟，因为我们喜欢看到太阳在大型电容器的帮助下做有趣的事情。[【nano robotgeek】的弹珠机是一个非同寻常的例子](https://www.instructables.com/Solar-BEAM-Marble-Machine/) —一旦充满电，两个 4700 μF 的电容器就会将电能注入一个自制的螺线管，螺线管将滚珠弹射向两条轨道的边缘。

[NanoRobotGeek]从免费提供的 [Suneater 太阳能电路](http://faq.solarbotics.net/suneater/suneater.html#suneater1)开始。这是光束机器人的一个主要部分，稍微修改一下以适应这个特殊项目的需要。首先是验证杠杆(或者梁，如果你愿意的话)原理是否可行，而[NanoRobotGeek]正是从这一点出发，以令人钦佩的细节构建了它。事实上，它的交替之间的漩涡轨道和之字形轨道是迷人的。

这里有几个学科在发挥作用，我们认为它在各个方面都做得很漂亮，特别是因为这是[NanoRobotGeek]首次涉足轨道弯曲。我们喜欢它扔球的方式，变轨杆也非常令人满意。休息过后，你可以在视频中查看一下。

虽然这是[NanoRobotGeek]的处女大理石轨道，但它不是他们的第一个电路雕塑— [看看这个拍打着翅膀、以光束为动力的蜻蜓](https://hackaday.com/2019/10/31/beam-dragonfly-causes-a-flap/)。
