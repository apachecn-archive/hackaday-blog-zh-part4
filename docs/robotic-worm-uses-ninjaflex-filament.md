# 机器人蠕虫使用 NinjaFlex 灯丝

> 原文：<https://hackaday.com/2020/09/24/robotic-worm-uses-ninjaflex-filament/>

如果你想建造一个移动的机器，你可能会考虑轮子或轨道，甚至可能是一个带你飞行的道具。当[nwlauer]在花园里发现一条蚯蚓时，它启发了[一个采用蠕动运动的 3D 打印机器人](https://www.instructables.com/id/How-to-Make-a-Robotic-Worm/)。你可以在下面看到它移动的视频。

这个机器人使用气动技术和软塑料，而且显然是防水的。你的打印机的进给路径必须非常严格，以支持灵活的灯丝而不堵塞。还有一些聚乙烯醇丝和硅胶管参与。

对于控制端，Arduino 控制一个隔膜泵和六个小电磁阀。当我们读到电磁阀经常用于眼部按摩器时，我们觉得很有趣。代码在 [GitHub](https://github.com/nwlauer/3D_Printing_Soft_Robotics) 上。

这些片段具有 Ninjaflex 以及 PVA 支撑插入物。你把每一段的中心粘在一起，但不是边缘，然后溶解聚乙烯醇，在里面形成空气室。

这个动作看起来不是很有效，但它可能处理了很多地形选项。我们不确定这种技术在目前的形式下是否有真正的实际用途，但我们打赌它可以被改编，而且它只是一个除了印刷手机壳之外与柔性细丝有关的很酷的例子。

我们已经看到 Ninjaflex 用于更实际的用途，如[定制皮带](https://hackaday.com/2017/11/03/3d-printing-belts-for-vintage-hardware/)。[小键盘](https://hackaday.com/2017/08/15/print-a-flexible-keypad/)是另一个实用项目。
