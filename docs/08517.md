# 水下无人机面临试水

> 原文：<https://hackaday.com/2020/12/03/underwater-drone-faces-trial-by-water/>

水下遥控潜水器，通常被称为 ROV，通常通过系绳操作。这是由于无线电波在水下传播不良。[西蒙]想要建造这样一架无人机，但选择了一种附加条件更少的替代设计，可以这么说。到目前为止，一路上都有挑战。(视频，下面嵌入。)

这种水下无人机使用 3D 打印的底盘，充满了眼睛，这大大有助于将建筑拟人化。四个马达用于控制，两个用于水平面内的推力，两个安装在垂直面内用于姿态控制。这使得无人机可以在中性浮力下设置，并通过推力而不是复杂的压载机制穿过水柱。该建筑旨在避免使用绳索，而是使用较短的电缆连接到一个浮动装置，该装置使用无线电与岸上的操作员通信。

该产品面临的主要挑战是密封机箱以防进水。这就是 3D 打印的分层性质的缺点。即使经过几次油漆和密封剂处理，[西蒙]也无法阻止水进入无人机。进一步的问题是需要过多的压载物来抵消无人机因位移而产生的自然浮力。

不管斗争如何，我们期待看到下一个版本纠正当前版本的一些缺点。我们确信[西蒙]制作电动冲浪板的经验会派上用场。休息后的视频。
