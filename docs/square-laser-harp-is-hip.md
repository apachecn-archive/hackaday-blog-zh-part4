# 方形激光竖琴很时髦

> 原文：<https://hackaday.com/2019/08/17/square-laser-harp-is-hip/>

你知道，我们没有意识到我们对垂直激光竖琴有多厌倦，直到我们看到[Jonathan Bumstead]进入 2019 年 Hackaday 奖。模仿这种鼓舞人心的仪器的设计当然很好。但是合成器的巧妙之处在于，它们并不局限于它们所模仿的声学仪器的物理特性。这个项目将激光竖琴提升到了功能性雕塑领域。是一件产生艺术的艺术品。

这种艺术竖琴完全独立，内置 MIDI、放大器和扬声器。这个美人的大脑是一个 Arduino Mega 和一个 Adafruit 音乐制作人 shield，这给了它二十种不同的乐器声音。六层中的每一层都有两个激光器、两个镜子和两个光敏电阻安装在胶合板框架的角落里。激光器和光敏电阻背靠背安装在相对的角上，在另外两个角上安装镜子来完成路径。[乔纳森]巧妙地用胶片罐塑料的乳白色碎片来扩散激光。

这不是[乔纳森]的第一次光学竞技。以前的经验告诉他能够在飞行中重新调整激光的重要性，因为每次他移动它，激光模块就会失去对准。这一次，他建造了运动学支架，让他可以使用四个螺丝重新定位激光器，每个螺丝推动一个角落。

这里有很多不错的触摸，尤其是仪器选择器轮。[Jonathan]在一个精彩的演示/构建视频中解释了它和竖琴的其余部分，在休息后，它只是在空间中烧了一个洞。
