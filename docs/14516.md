# 万圣节镜子提供了一个强大的恐惧

> 原文:[https://hack aday . com/2022/08/16/Halloween-mirror-offers-a-mighty-fright/](https://hackaday.com/2022/08/16/halloween-mirror-offers-a-mighty-fright/)

在恐怖世界里，无论你谈论的是电影还是电子游戏，跳楼恐惧都是有争议的。你也可以把这种刺激的快感带到现实生活中，用这个来自[jasonwinfieldnz]的万圣节镜子。

白天，或者只是当里面很亮的时候，镜子看起来是正常的，就像其他的镜子一样。然而，在特殊的双向镜面玻璃表面后面是一个怪异的 3D 打印，如一个头骨或一个令人讨厌的黄色卡通人物。当光线变暗时，一切都变了。连接到 Digispark 的光敏电阻检测到这种变化，然后点亮一些 5V 的 led 灯来照亮这个可怕的图像，将它显示在镜子后面。更好的是，它可以在 DFplayer MP3 模块的帮助下播放响亮的尖叫声。

我们也希望看到这个概念更进一步。如果当一个过路人走近时，房间的灯突然熄灭，镜子完全亮了起来，那该有多好。

这些年来，我们已经看到了一些很棒的万圣节作品。如果你渴望在这一季得到一个，你可能现在就想得到黑客！休息后的视频。
