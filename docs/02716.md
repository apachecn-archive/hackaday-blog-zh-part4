# FabGL 拥有为 ESP32 编写游戏所需的一切

> 原文：<https://hackaday.com/2019/04/17/fabgl-has-everything-you-need-to-write-games-for-the-esp32/>

通常，当一个人考虑写一个视频游戏时，平台是首先要做的决定之一。从 PC 开始很容易，移动开发也相当容易。当然，你也可以开发微控制器平台。[Fabrizio Di Vittorio]以 FabGL 的名字[构建了一套完美的工具来完成 ESP32。](https://github.com/fdivitto/fabgl)

这个库包含了一系列非常适合开发游戏的特性。有高达 64 色的 VGA 输出，PS/2 鼠标和键盘输入，以及强大的图形库和游戏引擎。如有必要，它甚至可以充当 ANSI/VT 终端。

[Fabrizio]已经对硬件进行了测试，通过简单的球、多边形和精灵，各种基准显示出令人印象深刻的性能。你可以很容易地制作一个 90 年代早期风格的 2D 游戏，而不会遇到任何硬件限制——尽管考虑到 ESP32 时钟高达 240MHz，这在某种程度上是可以预料的。

这是一个令人印象深刻的项目(休息后的视频)，我们希望看到更多的游戏在平台上开发。一旦你有了 VGA 连接器，你应该[尝试一些 ESP32 VGA 技巧](https://hackaday.com/2019/02/05/back-to-video-basics-with-an-esp32-vga-display/)。对于那些 ESP8266 的铁杆玩家来说,[也有一个针对该芯片的游戏引擎](https://hackaday.com/2019/03/11/esp8266-gets-its-game-on-with-open-source-engine/)！
