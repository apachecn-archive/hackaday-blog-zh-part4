# 载人辅助直升机

> 原文：<https://hackaday.com/2021/09/03/coaxcopter-to-carry-man/>

从业余爱好市场上买到的所有负担得起的飞行控制器和马达的主要好处之一是，你真的可以尝试一些疯狂的飞机设计。[amazingdiyprojects]正在试验一种同轴直升机设计，目标是将来可能用于载人版本。(视频链接，嵌入下方。)

该飞机使用一对同轴反向旋转电机，配有大型螺旋桨，螺旋桨下方有几个冗余控制面。与更传统的四轴飞行器类型设计相比，这种布置的理论优势之一是冗余性。虽然四轴飞行器会在单个发动机出现故障时开始翻滚，但这种设计仍然能够在只有一个发动机的情况下安全下降。

它也不依赖于主电机进行偏航、俯仰和滚转控制。在多旋翼飞机中，发动机需要保留大量的发动机可用功率，以便在接到姿态控制通知时增加扭矩。这种飞行器可以利用发动机提供的所有推力来提升，因为控制是由操纵面提供的。螺旋桨下面有五组冗余控制面，每组都连接到一个独立的飞行控制器。

这种设计的另一个优点是对于给定的占地面积是有效的，因为一个大螺旋桨总是比多个小螺旋桨更有效。[amazingdiyprojects]的目标之一是将全尺寸的船装进集装箱或拖车里，以便运输时不需要拆卸。

[amazingdiyprojects]以前制造过有人驾驶的无人机，同时使用了[电动机](https://hackaday.com/2017/07/01/manned-multirotor-flies-again-electric-style/)和[内燃机](https://hackaday.com/2016/07/28/repaired-manned-multicopter-flies-without-horrifying-crash/)。不要错过下面视频中 7:00 的[最诡异的风洞](https://youtu.be/tEEXPxN7qq4?t=421)。
