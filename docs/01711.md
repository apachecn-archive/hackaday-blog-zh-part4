# 热熔胶使这些片段发光

> 原文：<https://hackaday.com/2019/01/13/hot-glue-makes-these-segments-glow/>

可以肯定地说，热熔胶是我们在 Hackaday 看到的项目的主要组成部分。没有胶枪的读者不会很多，一滴粘性物质可以保证很多项目。但我们并不经常看到它被用作除粘性之外的一种属性的组成部分，所以[DusteD]的[反应计时器项目](https://dusted.dk/pages/reactionTimer/)很有趣，因为它将热熔胶用作其 LED 七段显示器的半透明光导和扩散器。

计时器非常简单，由 Arduino 板驱动，而显示器则预先形成在 3D 打印的外壳中。热胶填充每个部分后面的外壳，经过几次实验，发现最好的填充方法是从后面贴着一片 Kapton 胶带。led 被连接到一个公共的阴极阵列中，与拱廊式按钮和 Arduino 一起，整个装置都整齐地安装在盒子中。您可以在休息时间下方的视频中看到实际效果。

当然，这种显示器因其使用热胶而不同寻常，但不是唯一的。在之前，我们已经看到了[对热胶光管显示器的不同看法。](https://hackaday.com/2011/08/25/large-7-segment-display-made-from-glue/)
