# 时尚闹钟摇滚 VFD

> 原文：<https://hackaday.com/2019/06/14/stylish-alarm-clock-rocks-a-vfd/>

如果你想做一个数字钟，有很多种显示技术可供选择。许多黑客似乎对柏林墙东侧的光鲜食物情有独钟。[ChristineNZ]就是这样的一个黑客，[他成功地获得了一些合适的苏联设备来制作闹钟。](https://www.instructables.com/id/VFD-Alarm-Clock/)

该时钟采用 IV-27M 真空荧光显示屏，由现已解体的苏联制造。它拥有 13 个七段数字，拥有迷人的蓝色光芒，这是其他技术无法提供的。MAX6921AWI 芯片用于驱动 VFD，Arduino Mega 是操作的大脑。还有一个兼容 HD44780 的 LCD，可以显示更多的字母数字信息，还有一个 4×4 键盘用于控制设备。

不过这个建筑最好的部分是围栏。VFD 封装在玻璃管中，两端由 90 度铜管连接器支撑。这些使 VFD 保持在高处，并且也作为从管子两端出来的电线的导管。这一切都是建立在一个木制底座的顶部，这个底座支撑着其余的电子设备。

这是一个吸引人的建筑，我们喜欢玻璃管建筑创造的漂浮外观。这不是我们第一次看到旧的俄罗斯 VFD，我们怀疑这将是最后一次。休息后的视频。
