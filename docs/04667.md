# Raspberry Pi NAS 在捐助者 PC 中如鱼得水

> 原文:[https://hack aday . com/2019/11/01/raspberry-pi-nas-makes-in-home-in-donor-PC/](https://hackaday.com/2019/11/01/raspberry-pi-nas-makes-itself-at-home-in-donor-pc/)

可以肯定地说，我们大多数人都至少有一个挂在某个地方的 USB 电缆上的树莓派，默默地完成一些你不愿意在“真正的”电脑上做的乏味任务。尽管它们很便宜，但它在哪里开店并不是一个大问题。但是如果你像杰里米·s·库克一样，想让你 35 美元的 Linux 电脑成为家庭的永久成员，那么他关于把一台旧电脑变成大材小用的个人电脑的建议可能会让你感兴趣。

[![](../Images/84d6f8da6bb8bf9dd3133af7e423b6ca.png)](https://hackaday.com/wp-content/uploads/2019/10/pinas_detail.jpg) 从旧的联想台式个人电脑中抢救出来的主要部件显然就是机箱本身。除去原来的组件，这个箱子给他足够的空间来安装 Pi 以及几个硬盘和一个带电源的 USB 集线器。为了防止树莓派的底部与金属电脑外壳短路，他为它设计并 3D 打印了一个底座。其他东西都用钩环扣固定，方便快捷地移动和调整。

虽然这可能不是绝对必要的，但[Jeremy]也花时间抢救电脑的旧散热器。由于实在太大了，以至于不能按原样放在码头上，他用他的磨在码头后面拉了一条线，然后把码头掰成了两半。他用一点热胶带将对分的散热器固定在 Pi 的 SoC 上，用几片电工胶带确保它不会短路。

Raspberry Pi NAS 构建非常受欢迎，这些年来我们已经看到了数不清的东西。你可以用宜家的零件建造一个，如果你不介意塑料，[你可以随时 3D 打印整个东西](https://hackaday.com/2018/03/21/3d-printed-raspberry-pi-nas-with-dual-drive-bays/)。如果你真的想做到最小化，你甚至可以把一些文件挂在网络上，只需要把一个圆周率为零的圆周率插入 USB 接口。
