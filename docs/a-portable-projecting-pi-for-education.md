# 一种便携式教育投影墩

> 原文：<https://hackaday.com/2022/02/02/a-portable-projecting-pi-for-education/>

我们在 Hackaday 涵盖了许多 cyberdeck 项目，定制便携式计算机通常围绕 Raspberry Pi 构建。我们并不经常报道一台计算机完美地实现并超越了一个虚拟平台试图做的事情，但这正是[苏比尔·巴杜里所做的。为了满足印度学生补上两年被 COVID 中断的学校教育的需要，他创造了 pπ，一个树莓 Pi，投影仪和键盘一体化的计算机，在一个整洁的金属片外壳中，看起来好像只是另一套扳手或类似的东西。他不费吹灰之力就实现了*终极*电脑平台，因为这台机器不是科幻道具，相反它有一个令人钦佩的明确用途。

你可以在 GitLab 库的[中找到你自己构建的所有文件。这个案例是激光切割金属板，他做了一个成本分析，得出一个相对健康的 17200 印度卢比，大约 230 美元。我们希望它能很好地服务于自己的目的，并为一代人提供一个坚固可靠的教学辅助工具，COVID 从这一代人身上获得了太多。你可以在休息时间下面的视频中看到更多。](https://gitlab.com/subir_bhaduri/p_pi)
