# 灯泡制成的 50W 扬声器

> 原文:[https://hacka day . com/2020/06/08/a-50w-灯泡做的扬声器/](https://hackaday.com/2020/06/08/a-50w-speaker-made-of-light-bulbs/)

当我们想到扬声器时，我们很可能会想象一个纸锥，其底部某处有一圈线圈，悬浮在磁场中。这是一项有一百多年历史的技术，已经接近完善。然而，动圈并不是将电流转化为声音的唯一手段。当暴露在音频中时，许多组件会发出声音，包括让[Eric]惊讶的是，不起眼的白炽灯泡。他在为他们制作可寻址驱动器时发现，当他们点亮时，他可以听到 PWM 频率，所以[他开始利用这种效应作为扬声器](http://www.riderx.info/the-worst-50-watt-speaker-in-the-world/)。

使用一个 ESP32 板，由于便宜的元件，他从 MIDI 文件开始，以 PWM 频率结束。这是一次从 ESP32 创建多个 PWM 通道的有趣旅程，他详细介绍了一路上遇到的一些问题。结果是在休息时间下方的视频中可以看到一组唱歌的灯泡，他坦率地承认这可能是他可能制作的最糟糕的 50 W 扬声器。然而，这不是这样一个实验的目的，我们为他这样做而鼓掌。

更多基于 MIDI 的蠢事，请看一下[PCB 特斯拉线圈](https://hackaday.com/2019/04/25/a-tesla-coil-from-pcbs/)。
