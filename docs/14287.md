# 使用 Buildify 快速设计城市

> 原文:[https://hack aday . com/2022/07/21/design-cities-in-a-snap-with-build ify/](https://hackaday.com/2022/07/21/design-cities-in-a-snap-with-buildify/)

设计 3D 环境很难，但也不尽然。一周前，如果你决定在 Blender 中设计一整个城市，比方说为一个游戏或动画，你可能会下载一些充满建筑形状和纹理的资源包，并在几天内煞费苦心地放置它们，修改模型并根据需要制作新的模型。现在，你只需要下载 [Buildify](https://paveloliva.gumroad.com/l/buildify) ，给它一个资源包，然后看着奇迹发生。

由[Pavel Oliva]制作的 Buildify 是我们很久以来看到的最令人印象深刻的 Blender 内容之一。它可以让你通过绘制建筑物的轮廓来生成整个城市。您可以抓取墙壁并调整单个结构的大小，墙壁、窗户、门、纹理和其他一切都将根据需要自动重新排列。您甚至可以在开放的街道地图上选择一个区域，然后观看 Buildify 使用您选择的资源包在 Blender 中重建该区域(也许一个 [KiCad PCB 设计](https://hackaday.com/2020/12/23/remoticon-video-kicad-to-blender-pcb-renders/)也可以用作源材料？).这真的是令人难以置信的事情，你只要看看下面的视频就能明白这个工具有多有用。

想付多少就付多少。你可以从[Pavel]的网站上抓取的混合文件并不包括你在视频中看到的所有美丽的资产，而是生成简单的灰色块状建筑。他制作了视频中使用的一个包，并将很快在网上免费发布。与此同时，他链接到你可以购买的其他网站，或者你可以雄心勃勃地创建自己的网站。我们知道，用不了多久，我们就会看到由 Buildify 生成的城市的动画和游戏。
