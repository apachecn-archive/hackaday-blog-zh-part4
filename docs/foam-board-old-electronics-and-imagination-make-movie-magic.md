# 泡沫板、旧电子产品和想象力让电影变得神奇

> 原文：<https://hackaday.com/2019/12/31/foam-board-old-electronics-and-imagination-make-movie-magic/>

当谈到为电影和电视搭建布景和道具时，很容易犯科幻小说的错误——尤其是低成本制作。对于布景部门来说，回到“拿一堆东西，把它涂成银色”的模式一定很诱人，这对于观众中精通技术的人来说是一种乏味的体验。

但是，如果有合适的人参与，低预算并不一定意味着低产值。以[【乔尔·哈特劳布】最近的工作为一部众筹科幻电影](https://www.youtube.com/watch?v=FitxuWNgnJM)搭建布景，这部电影叫做*无限极*。这是一个后世界末日的故事，需要一个带有*余波*氛围的地下掩体，而【乔尔】欣然接受了这个机会，将场景组合在一起。他主要使用老式电子设备和泡沫绝缘板，通过数控加工制成令人信服的面板，为地堡建造了非常精细的控制台。语音通信器是由一个旧的电子管型台式收音机外壳和一些七段显示器组成的，旧的液晶投影仪的底盘是一个令人信服的便携式计算机终端。最棒的黑客攻击是针对气闸门的控制面板。他们使用一种老式的 TDD，或者说是为聋人设计的电信设备。有了键盘和 VFD 显示器，它正好符合布景的感觉。但是[Joel]做了额外的努力，使它成为一个实用的作品，通过记录来自声学耦合器的调制音调并播放它们，使它看起来好像有信息进来。气闸门看起来也很棒。

像许多黑客一样，用一大堆垃圾和一点想象力就能完成的事情令人印象深刻。但是，如果你有一个更大的预算，你需要一些电脑显示器，我们知道合适的人选。
