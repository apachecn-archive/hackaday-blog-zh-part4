# 阴凉的空气伞赋予新的生命

> 原文：<https://hackaday.com/2021/12/04/shady-air-umbrella-given-new-lease-on-life/>

许多臭名昭著的 Kickstarter 项目最终失败了，让支持者感到沮丧和愤怒。通常以引人注目的令人信服的视频为卖点，视频中的关键部分恰好在屏幕之外。[艾伦·潘]回忆起一个这样的项目，空气伞，[决定通过制作自己的](https://www.youtube.com/watch?v=Yt_6P6gK2Q4)来挽回这个项目。

空气伞的基本概念是一种可以在你的头上产生一个快速移动的空气锥来偏转空气的装置。根据 Kickstarter 主页上列出的规格，[Allen]制作了一个简单的原型，但什么也没做。怀疑得到了证实，他决定继续购买一台大功率电动吹叶机。一个喷嘴被 3d 打印出来，可以将空气导入所需的圆盘。在花园软管上进行的薄雾功能的早期测试似乎很有希望，他们的工作方式是逐渐增大雨滴。

最后，加利福尼亚的云向他们微笑，下雨了。[艾伦]欣喜若狂，他的伞工作。有几分钟，他拿着吹叶机，一只耳朵听不到太多声音，但这是用空气伞保持干燥的小小代价。

如果你对更多虚假的 Kickstarter 声明感到好奇，为什么不读一下这个[小 Arduino 兼容板做出的一些可疑的声明](https://hackaday.com/2017/07/30/false-claims-on-kickstarter-whats-new/)。
