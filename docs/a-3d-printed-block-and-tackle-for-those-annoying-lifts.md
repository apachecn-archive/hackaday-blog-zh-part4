# 一个 3D 打印的滑轮组，适合那些烦人的电梯

> 原文：<https://hackaday.com/2021/08/19/a-3d-printed-block-and-tackle-for-those-annoying-lifts/>

也许简陋的滑轮组——多个平行滑轮以减少提升的努力——不再像液压装载机时代那样常见，但它仍然是每当有提升任务时的有用机制。为此，[semi]已经生产了一个 3D 打印的滑轮组系统，正如我们在下面的视频中看到的那样，它使提升中等重量的货物变得轻而易举。

这是一个足够简单的机制，3D 打印机提供滑轮、楔块和连接点，以及将所有东西固定在一起的钢螺栓。它的最大重量为 20 公斤(44 磅)，虽然在信任它拥有 200 公斤的发动机之前可能会有一些犹豫，但我们猜测它的能力远远超过我们展示的。如果你想试一试，这些文件可以在 Thingiverse 上找到。

滑轮组应该在各地工程师的心中占有特殊的位置，因为[这是第一个使用大规模生产技术制造的产品](https://en.wikipedia.org/wiki/Portsmouth_Block_Mills#Blocks)。这座 19 世纪早期的工厂来自伊桑巴德·金德姆·布鲁内尔[的父亲](https://hackaday.com/2017/12/15/hardware-heroes-isambard-kingdom-brunel/)[马克·布鲁内尔](https://en.wikipedia.org/wiki/Marc_Isambard_Brunel)的作品，这并不奇怪，我们已经把他作为之前一篇黑客文章的主题。
