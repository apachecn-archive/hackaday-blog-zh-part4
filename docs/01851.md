# 你口袋里的架子鼓

> 原文:[https://hackaday.com/2019/01/22/a-drum-set-in-your-pocket/](https://hackaday.com/2019/01/22/a-drum-set-in-your-pocket/)

如果你不介意鸡腿突出来，工装裤可以把鸡腿放进口袋里。他们还可以拿着这个架子鼓，并且仍然有足够的空间放一副耳机、一些笔和一个小笔记本。至少，盖伊的工装裤可以适应这一切。现在你的口袋里已经有了足够的音乐设备，可以作曲和打鼓，甚至可以写一些笔记。我们不能说[托马什 Ghz]带着笔记本，但他非常想要一个装在口袋里的鼓，足以制作一个定制电路板，带到 2017 年雅典 Fasma 节上。他为一个 Teensy 3.2 编写了代码，这个代码贴在他的 PCB 背面，靠近 9V 电池。不要害怕，最小的组件是 0805，所以即使是笨拙的手指也可以自己制作。Gerber 文件和 BOM 都是可用的，所以没有什么可以阻止你。

在板上，我们可以找到一系列运算放大器来支持耳机和线路电平输出，四个大按钮来激活每种类型的鼓:鼓、鼓、小军鼓和鼓帽。然后，我们有四个电位计来改变每个声音，如音高、衰减/长度、调制和失真。一旦完美的模式被记录下来，它可以被保存在非易失性存储器中，以防你电量耗尽，尽管它可以在一节电池上运行长达 7 个半小时。如果你发现自己投资于硬件，也有一个关于使用鼓机的视频演练，所以拿起你的笔记本，打败它。

我们在更简单的芯片中见过更简单的鼓，甚至在完全不同类型的芯片中见过更简单的鼓。
