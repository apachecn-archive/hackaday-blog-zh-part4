# 有史以来最糟糕的键盘之一，现在是 Arduino 外设

> 原文：<https://hackaday.com/2022/10/24/one-of-the-worst-keyboards-ever-now-an-arduino-peripheral/>

对于一定年龄的英国孩子来说，他们第一次接触电脑很可能是在 Sinclair ZX81 电脑前。作为大获成功的 ZX 系列的前身，它鲜为人知，售价不到 100 美元，配有 Z80 处理器和高达 1k 的内存。在 Sinclair 产品的悠久传统中，为了达到这个价位，它做出了一些妥协，其中最明显的是 40 键薄膜键盘。那些学会了对其令人沮丧的缺乏触觉反馈进行编码的人可能会惊讶地看到[一个 Arduino 项目将其作为轻松将键盘连接到 Arduino](https://create.arduino.cc/projecthub/sl001/read-a-zx81-keyboard-with-arduinos-and-build-things-with-it-0189bd) 的完美方式。

像许多逆向计算部件一样，ZX81 '板已经重新制造，这让许多 Sinclair 爱好者感到高兴。因此，它很容易买到，而且相对便宜(我们认为它们的售价不到 20 欧元！)，所以令人惊讶的是它是 Arduino 项目的合理选择。试图通过触摸来定义 ZX81 键厚度的细微差异的任务将为新一代带来真正的逆向计算体验。也许[如果可以在 Mbed 上完成](https://hackaday.com/2014/12/22/zx81-emulated-on-an-mbed/)那么有人甚至会在 Arduino 上制作 ZX81 仿真器。

在 Hackaday ，我们是 ZX81 的忠实粉丝，对我们中的一些人来说，这是第一台电脑。愿它继续让粉丝们开心！
