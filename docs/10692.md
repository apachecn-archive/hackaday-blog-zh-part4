# DIY 胸肌板用图片交流

> 原文：<https://hackaday.com/2021/07/15/diy-pecs-board-uses-pictures-to-communicate/>

与自闭症患者和非语言人士交流的一种方式是通过使用图片交换交流系统或 PECS 板，他们可以用它来指出他们一天中需要或想要什么。然而，这些板的商业版本也有它们的问题——它们很贵，而且就画面而言它们相当僵硬。[【Alain Mauer】创建了一个更加个性化的开源 PECS 板，并且有音频引导](https://hackaday.io/project/180642-pecs-communication-board)。

这里的首要条件是坚固，因为[阿兰]的儿子[斯科特]已经砸碎了两部智能手机和一台平板电脑。[Alain]选择了激光切割的中密度纤维板外壳，应该可以使用相当长一段时间。里面有一个 Arduino Pro Mini 和一个 DF Player Mini，每当[Scott]按下 16 键铜箔电容式键盘上的一个按钮时，它就会播放 micro SD 卡中相应的片段。这种 PECS 板也很智能——它会在几分钟不活动后发出关机提醒，并发出声音低电池警告。

到目前为止，[斯科特]对物体照片的反应比对图画的反应更好。休息之后看他和董事会互动。

这远远不是(阿兰)为帮助(斯科特)而做的第一件事。[一定要看看这款基于 Pi 的媒体播放器，它是为了吸引人而不是激怒人而设计的](https://hackaday.com/2016/04/18/custom-media-player-helps-hackers-autistic-son/)。
