# 艾梦想着时间旅行

> 原文:[https://hackaday.com/2022/09/28/ai-dreaming-of-time-travel/](https://hackaday.com/2022/09/28/ai-dreaming-of-time-travel/)

我们喜欢艺术和技术的交叉，一个由 AI(稳定扩散)[制作的视频想象了一次穿越时间的旅行](https://twitter.com/xsteenbrugge/status/1558508866463219712) ( [Nitter](https://nitter.net/xsteenbrugge/status/1558508866463219712) )就是一个可爱的例子。这个项目相对简单，但和大多数艺术项目一样,[史云光·斯廷布鲁格]花了无数个小时调整和摆弄这个过程的不同部分，直到他喜欢为止。他提到尝试了数千种不同的提示和种子——其中一个提示的例子是“一个有小屋的小部落村庄”在视频中，每个提示有 72 帧，强度慢慢增加，然后随着下一个提示的出现而降低。

YouTube 上还有其他人工智能视频，通常是[将一首歌的歌词转换成人工智能生成的形式](https://hackaday.com/2022/09/14/ai-midjourney-images-stairway-to-heaven/)。但如果你曾与人工智能系统合作过，你会注意到背景在[史云光]的视频中保持非常稳定，因为它经历了几十次反馈循环。这很难做到，因为你想改变图像的内容而不改变外观。因此，他不得不写大量的代码，试图随着时间的推移保持视觉上的时态一致性。希望我们能看到他的一些改进的开源版本，就像他在 Twitter 上提到的那样。

同时，我们可以坐下来享受美好的事物。如果你仍然不相信稳定扩散没什么大不了的，也许我们可以再做点什么来说服你的观点。
