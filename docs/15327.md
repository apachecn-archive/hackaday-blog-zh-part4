# 哦，太好了！3D 打印无破损的抓手零件

> 原文:[https://hack aday . com/2022/11/11/oh-snap-3d-printing-snapping-parts-without-breaking/](https://hackaday.com/2022/11/11/oh-snap-3d-printing-snapping-parts-without-breakage/)

塑料的一大优点是它可以相对灵活。我们总是能看到咬合在一起的东西，但当 3D 打印时，你不会经常遇到咬合设计。[工程师成长]有一个视频可以帮助你[设计不会损坏](https://www.youtube.com/watch?v=-jgbd8RUhhY)的卡扣配件。

在下面的第一个视频中，他介绍了三个有帮助的参数。第一个是捕捉元素的长度。其次，可以减小底切尺寸。你也可以试着抓拍；尽可能的薄，尽管在这个例子中他太薄了，最终还是折断了。

最后一个建议，在下面的第二个视频中有详细介绍，就是改变你使用的材料。关键参数被称为断裂伸长率。对于 PLA，此值的典型值为 8%。ABS 是 10%，PETG 是 24%，尼龙是 100%。简单地说，你可以假设一个 PETG 碎片在断裂前可以变形 25%。那可能是真的，但是在那之前它会永久变形。视频建议使用 10%或 15%的值来确保零件不会变形。

在第三个视频中，您也将了解到打印方向也很重要。如你所料，让钩子脱离构建板会导致一个弱钩子。
