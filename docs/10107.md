# 随机机器人创造随机艺术

> 原文:[https://hack aday . com/2021/05/14/random-robot-makes-random-art/](https://hackaday.com/2021/05/14/random-robot-makes-random-art/)

对于一把牙刷和一个抵消重量的小马达的价格来说，刚毛机器人本质上是可以制造的最便宜的机器人。马达摇动牙刷，刷毛模式允许机器人移动，尽管是完全随机的模式。虽然这可能看起来不像一个真正的机器人，可以以任何有意义的方式与环境互动，【scanlime】展示了这个机器人的多功能性——它看起来只是随机移动——[实际上可以用来以非随机的方式制作艺术](https://www.youtube.com/watch?v=3a4-ZxISiRM)。

该项目没有使用单个刚毛机器人，而是将三个机器人内置于一个 3D 打印的柔性外壳中，每个外壳偏移 120 度，并且可以在中心的开口中容纳一支笔。这使得他们可以控制机器人的运动方向。从那里，定制软件试图解决刚毛机器人的随机性，以产生给定的图像。当然，作为一个刚毛机器人，它很容易受到外部环境的影响，例如桌子的水平，甚至是电源/通信系绳施加的小作用力。

随着设计的一些迭代，如修改手臂和控制系统，她有了一个有趣的艺术制作机器人，它固有的随机运动相当可靠。对于那些想要尝试这种东西的人来说，运行机器人的代码和 3D 打印零件的 CAD 文件[都可以在该项目的 GitHub 页面](https://github.com/scanlime/wiggler)上获得。如果你正在寻找其他不仅仅在桌面上游荡的刚毛机器人，一定要看看这个[循线刚毛机器人](https://hackaday.com/2016/05/25/black-line-follower-a-modern-bristlebot/)。
