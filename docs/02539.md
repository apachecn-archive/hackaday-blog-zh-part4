# Python 与物联网黑客聊天

> 原文:[https://hack aday . com/2019/04/01/python-and-the-internet-of-things-hack-chat/](https://hackaday.com/2019/04/01/python-and-the-internet-of-things-hack-chat/)

加入我们太平洋时间周三中午的 [Python 和物联网黑客聊天](https://hackaday.io/event/164062-python-and-the-internet-of-things-hack-chat)！

关于现在最常用的编程语言是什么，众说纷纭，但是很难否认 Python 的流行和多功能性。自从它被发明以来的近 30 年里，它已经从一种小众语言发展成为一种成熟的开发环境，如今似乎无处不在。这包括我们现在使用 MicroPython 的心爱的微控制器，以及 Adafruit 的 [CircuitPython](https://learn.adafruit.com/welcome-to-circuitpython) ，大大降低了入门级黑客的门槛，简化并加快了老手的开发，并提供了一条通往 Python 驱动的物联网 [![](../Images/208aa0b0bfb72b29d650663b8c4062bc.png)](https://hackaday.com/wp-content/uploads/2019/04/pythonandtheiothackchat-01.jpg) 的道路。

来自 Adafruit Industries 的 CircuitPython 团队——[丹·哈尔伯特](https://hackaday.io/dhalbert)、[卡特尼·伦博](https://hackaday.io/hacker/277997-kattni)、[利莫尔·“lady ada”·弗里德](https://hackaday.io/ladyada)、[菲利普·托伦](https://hackaday.io/pt)和[斯科特·肖克罗夫特](https://hackaday.io/tannewt)——将参加黑客聊天，回答您所有关于 Python 和物联网的问题。加入我们的讨论:

*   CircuitPython 是如何出现的；
*   支持 Python 的物联网产品范围；
*   物联网设备 Python 入门；和
*   Python 驱动的物联网前景如何？

作为额外的诱惑，我们将赠送给 [Adafruit.io](http://adafruit.io/) 五张一年的免费入场券！我们将从黑客聊天参与者的名单中随机抽取五个名字。停下来就有机会赢。此外，Adafruit 团队也将在黑客聊天期间[进行视频直播](https://www.youtube.com/watch?v=RMNeRk3mcyA&feature=youtu.be)。

当然，我们鼓励你在讨论中提出自己的问题。你可以在 [Python 和物联网 Hack Chat](https://hackaday.io/event/164062-python-and-the-internet-of-things-hack-chat) 上发表评论，我们会把它放在 Hack Chat 讨论的队列中。

[![join-hack-chat](../Images/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是在 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)上的现场社区活动。本周我们将在太平洋时间 4 月 3 日星期三中午坐下来。如果时区让你心烦意乱，[我们有一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20190403T12&p0=224&msg=Python+and+the+Internet+of+Things+Hack+Chat&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
