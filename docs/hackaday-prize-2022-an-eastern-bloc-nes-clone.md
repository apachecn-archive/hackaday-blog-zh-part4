# 2022 年黑客日奖:一个东方集团的 NES 克隆

> 原文：<https://hackaday.com/2022/07/31/hackaday-prize-2022-an-eastern-bloc-nes-clone/>

如果任天堂除了他们的角色和公认的顶级视频游戏之外还有其他什么出名的话，那它在使用他们的知识产权时对粉丝是无情的。他们对仅仅因为展示非任天堂硬件模拟他们游戏的人采取法律行动，甚至试图关闭像超级粉碎兄弟这样的游戏的竞争场景。为了摆脱任天堂法律团队的窥探，需要采取极端的措施——比如在铁幕后面建造你的任天堂游戏机克隆版。

[Marek wice k]就是在这样一个地方长大的，所以玩 Famicom(又名 NES)游戏的唯一方法是使用一个克隆系统，就像当时在东方集团流行的一个叫做 Pegasus 的系统，它可以通过一些修补来完成任务。[Marek]最近从这个克隆控制台中发现了 CPU 和 GPU 芯片，并开始构建自己的芯片。使用 perf 板和电线，他能够测试芯片并确认它们功能正常，但视频内存有一个问题，他花了一段时间来跟踪和修复。

之后，他基本上有了一个全功能的 Famicom，可以使用任何弹夹。虽然我们希望住在东欧仍然能让他远离任天堂，以避免被任天堂骚扰，但我们永远不能太确定。当然，除非你使用这个可以合法模仿 SNES 游戏的设备。
