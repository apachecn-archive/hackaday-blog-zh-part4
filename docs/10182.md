# DIY 绝缘螺母和螺栓

> 原文：<https://hackaday.com/2021/05/22/diy-insulating-nuts-and-bolts/>

[Rudi Schoenmackers]设计了一套聪明的定制 3D 打印夹具，可以轻松制作自己的木制六角螺母和螺栓。嗯，如果你有一个木工车间，里面有刳刨机、带锯和砂带磨光机，那就简单了。

然而，你不会用这些来安装你的印刷电路板。它们相当大——UNC 1-6 螺纹(最接近的公制螺纹可能是 M36-4)。[鲁迪]指出，这些夹具可以很容易地适应产生不同尺寸和螺距的螺纹，甚至是左旋螺纹，但我们怀疑制造#4-40 或 M3-0.5 是不可能的。制造螺纹的商业夹具是有的，但正如[Rudi]指出的那样，它们相当昂贵。假设你有一台 3D 打印机，[Rudi]的夹具价格相当低。

我们不知道如何在普通的黑客项目中最好地利用这些螺母和螺栓，但[Rudi]喜欢将它们作为酷玩具赠送出去，或者用它们制作大型夹具和老虎钳。如果你有木制螺纹紧固件可以派上用场的应用，请告诉我们。如果你对木螺纹感兴趣，那就来看看几年前我们报道过的制作简单水龙头的项目。
