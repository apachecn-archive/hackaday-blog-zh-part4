# 黄铜色的经典 WiFi 时钟也能显示天气

> 原文:[https://hack aday . com/2020/11/01/brassy-classy-wifi-clock-shows-weather-too/](https://hackaday.com/2020/11/01/brassy-classy-wifi-clock-shows-weather-too/)

电路雕塑是将你的焊接和电子技能提升到三维艺术领域的好方法。在这种情况下，艺术也可以是实用的。[以【romaindurocher】的互动 WiFi 时钟为例](https://hackaday.io/project/175515-wifi-clock)。不管怎样，时钟会循环显示时间、温度，如果适用的话，还会显示降水预报。

但是[罗曼]不需要观察和等待他们想要的信息。多亏了红外接近传感器，[romain]可以中断循环，并根据手经过传感器的次数获得日期、时间、温度，甚至是笑脸动画。时钟本身是基于一个 Adafruit 羽毛 HUZZAH ESP8266 和一个羽翼显示。它使用 OpenWeather API 来检索所有信息。

我们真的很喜欢这个样子，这个角度让我们想起了示波器和其他实验室设备。如果你想制作自己的版本，这个项目是完全开放的，尽管最困难的部分是让它看起来像[罗曼]做的那样干净。休息之后，请花点时间观看简短的演示。这是我们的赛道雕塑挑战赛中一个精彩的参赛作品，该挑战赛正在接受为期十天的参赛作品。

不太喜欢直线和实用性？[电路雕塑有多种形式，其中一些是人形的](https://hackaday.com/2020/10/27/circuit-sculpture-breathes-life-into-discrete-components/)。
