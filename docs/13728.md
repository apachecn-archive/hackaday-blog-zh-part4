# 512 千兆字节的软盘

> 原文:[https://hack aday . com/2022/05/26/the-512 GB 软盘/](https://hackaday.com/2022/05/26/the-512-gigabyte-floppy-disk/)

有时候，一项技术几乎在一夜之间消失得无影无踪，而另一些时候，它们会随着时间的推移逐渐消失，以至于几乎没有人注意到它们的消失。可移动介质也是如此，虽然我们偶尔还会有 USB 闪存盘或 SD 卡，但它们的数量和普及程度远不及过去的软盘、Zip 盘和 CD-r om。如果软盘对你来说只是一个保存图标，那么你仍然有机会体验它们的复古魅力，感谢[Franklinstein]。他制造了一个 3.5 英寸的软盘，避开了 720 k、1.44 M，甚至 2.88 Mb，并且声称拥有 512 Gb 的容量。我们肯定不记得以前的这些了！

当然，正如我们在下面的视频中看到的，他既没有实现令人震惊的数据压缩壮举，也没有在单个氧化铁分子中存储比特的尖端方法。取而代之的是软驱铰链打开，在磁盘本身所在的地方有一个微型 SD 卡的支架。这很有趣，我们不得不同意他的观点，这使得微型 SDs 非常方便，可以承载这么多数据。这让我们不禁想知道，是否有可能以某种方式将 14 个微 SDs 多路复用到一个 PCB 上的微控制器上，以便装入软盘外壳中。或许 ESP32 可以通过 web 界面成为一个慢速文件服务器？

他指出，如果把 512 Gb 的软盘堆在一起，它们将轻松超过最高建筑的高度，因此至少这代表了空间的节省。如果你在寻找功能稍微多一点的东西，并且不介意修改驱动器，总有一种经典的方法可以将软盘与 SD 卡结合起来。
