# 业余无线电单边带收发器适合放在口袋里

> 原文:[https://hack aday . com/2021/12/09/ham-radio-ss b-收发器-口袋装/](https://hackaday.com/2021/12/09/ham-radio-ssb-transceiver-fits-in-pocket/)

谈论这款中国业余无线电收发机需要一系列名副其实的首字母缩写词:HF、SSB、QRP 和 SDR。[Paul]做了很好的工作，打开了钻机的箱子，并检查了一下。该收音机是一个德国项目的克隆，提供了一个带有可充电电池的低功率收音机。下面可以看到他关于齿轮的视频。

SSB 对于低功耗操作来说是一个奇怪的选择，尽管我们想知道是否可以使用 PSK31 这样在低功耗下具有良好性能的模式来馈送数字数据。这种收音机有多种型号，价格一般不到 200 美元，有时还会低很多。

收音机前面没什么东西。有几个按钮，一个旋转编码器，一个 LCD 以及内置的扬声器和麦克风。如果您不想使用电池，可以使用电源端口来运行收音机，还有一个单独的电池充电端口。还有按键端口、外部麦克风和扬声器，以及音频连接，看起来它们可以用于数字模式。据评论者称，这款收音机没有内部充电电路，所以你必须小心你插入充电端口的东西。

往里面看，收音机看起来制作得出奇的好。在视频的结尾，你可以看到电台也做了一些联系。看起来很有趣。对于(丹·马洛尼的)50 美元的火腿系列来说，这有点贵，但也不算太贵。至少，你可以从他那里借鉴天线的想法。如果你喜欢更模拟的东西，[抓住七个晶体管，建立这个单边带收发器](https://hackaday.com/2021/11/20/pssst-heres-a-novel-ssb-radio-design-with-only-seven-transistors/)。
