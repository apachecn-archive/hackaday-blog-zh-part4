# 什么进入高压二极管？

> 原文：<https://hackaday.com/2021/10/03/what-goes-into-a-high-voltage-diode/>

当我们使用一个电子元件时，我们对其内部的情况有所了解。我们知道，在晶体管内部有一小片半导体，其结由不同掺杂区域蚀刻而成，在电容器中，某种电介质表面会有金属化板。逆向工程给了我们大量的集成电路芯片照片，但仍有一些组件的秘密有待发现。其中一张是由[WizardTim]拍摄的，当时他正切开一个 20KV 的高压二极管。

传统低压硅二极管的正向压降约为 0.7V，最大反向电压相对较低，例如 1N4001 整流器为 50V。对于规格更高的 1N4007，反向电压额定值为 700V。这个二极管有一个 25KV 的反向电压，其结构的线索来自其引用的 45V 正向电压。果然，当安装在树脂中并仔细打磨和抛光时，它的内部显示为一堆串联的二极管，以正向电压为代价增加反向电压。

揭示一个不寻常的组件的内部工作是令人着迷的，所使用的研磨技术绝对值得一看。这是我们以前见过的，例如在[减少 CPU 厚度以提高性能](https://hackaday.com/2020/07/21/die-lapping-for-better-cpu-performance/)中。
