# 带摩擦驱动的电动小轮车

> 原文：<https://hackaday.com/2020/11/26/electric-bmx-with-friction-drive/>

电动自行车在过去几年里越来越受欢迎，虽然你可以很容易地从一家知名的自行车制造商那里买到一辆，但我们大多数人可能会倾向于至少买一套工具，并把它绑在我们已经有的自行车上。然而，并没有适用于每种自行车几何形状的套件，所以如果你想要一辆电动 BMX 自行车，你可能会想要[尝试一些定制的东西，就像【谢伊·奈奎斯特】在他的最新作品](https://www.youtube.com/watch?v=FCMLuXoypsQ)中所做的那样。(视频，嵌在下面。)

BMX 车架的前三角比大多数自行车都要小，所以他的体型需要非常紧凑。为此，它使用了两个用皮带连接在一起的小型电机，这些电机共同驱动一个摩擦驱动装置，该装置夹紧后轮胎，使其直接旋转。与电机安装在后轮上的轮毂驱动相比，这也使自行车的重量分布更加平衡。它还使用了一个更紧凑的锂聚合物电池组，而不是大多数电动自行车使用的典型的 18650 锂离子电池组，尽管它的续航里程只有大约 3 英里，但它的电量足以推动它在一个滑板公园内行驶。

该建筑也拥有令人印象深刻的数字，每台电机的峰值功率为 2.5 千瓦。这使它进入了电动摩托车的领域，尽管它的身材很小，但它确实很快。然而，为了获得真正的高速电动自行车体验，你需要一个稍微大一点的车架和马达，即使这意味着将安全抛到窗外。
