# 乐高钻机制造漂亮的水漩涡

> 原文：<https://hackaday.com/2022/09/01/lego-rig-makes-pretty-water-vortexes/>

乐高和它的技术系列是学习各种机械的好方法，但是玩起来也很有趣。我们怀疑是后一个原因让[砖块技术]追求用建筑玩具创造出一个令人神往的水漩涡。

第一个设计使用一个透明的塑料球作为水容器。球体内部放置了一个小涡轮，通过磁耦合从外部转动。这意味着球体可以保持完整，不需要孔，也不需要可能泄漏的复杂密封。它工作得很好，当一个普通的乐高马达带动涡轮旋转时，会产生一个涡流。第二次尝试选择旋转整个球体本身。然后用小乐高轮子将球体固定在快速旋转的转盘上。结果令人印象深刻，创造了一个大的和相对良好形成的漩涡。

当然，这个视频把最好的留到了最后。这个透明的大球体被安装在一个完全包围它的装置中。球体本身由于安装在两个不同轴上的轮子而旋转。这使得球体可以在 PlayStation 控制器的命令下向各个方向旋转，从而产生更复杂的涡流和流动模式。提供了一组旋转脚轮以在球体向各个方向旋转时保持其位置，并用弹簧和橡皮筋缓冲，以防止钻机晃动。

这有什么意义吗？嗯，不是表面上的水平，不。但是，它肯定会成为一个摇动鸡尾酒的装置。或者，如果出于某种原因，你需要一个噪音大但有吸引力的离心分离器，也许这可以做到这一点。
