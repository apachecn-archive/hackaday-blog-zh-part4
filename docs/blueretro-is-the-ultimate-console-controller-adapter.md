# 蓝色复古是终极控制台控制器适配器

> 原文：<https://hackaday.com/2020/07/01/blueretro-is-the-ultimate-console-controller-adapter/>

复古游戏机非常有趣，充满了经典游戏和大量的怀旧元素。然而，它们也缺乏更现代的硬件的抛光和人体工程学。严肃的游戏玩家也会经常发现他们原来的游戏手柄也用坏了。BlueRetro 是所有这些问题以及更多问题的解决方案。

BlueRetro 是一款蓝牙控制器适配器，适用于各种老式控制台平台。由雅克·加尼翁开发，它使用 ESP32 实现强大的无线功能。ESP32 的一个内核用于蓝牙通话和处理控制器接口，而另一个处理器内核处理与连接的控制台的通话。

对细节的关注程度是这个项目真正的亮点。[Jacques]实现了许多高级功能，如将轴映射到按钮，反之亦然——这是在不同系统之间交换控制器时必不可少的。BlueRetro 的输出是 DB25 连接器，然后使用适配器电缆连接到各种控制台的控制器端口。它甚至能够模拟多达 7 名球员的行动多点击适配器。

[在一段视频中，](https://www.youtube.com/watch?time_continue=266&v=xBVsEUAiUrs&feature=emb_title) [Jacques]展示了他收藏的老式游戏机使用的硬件，这些游戏机挂在一个架子上，带有令人印象深刻的 A/V 切换器设置。很明显，这是一个不吝惜以正确方式做事的黑客。我们以前也看过他的作品，[为 CRT 电视设计了整洁的 RGB 输入模式。](https://hackaday.com/2020/04/09/adding-rgb-to-a-crt/)休息后的视频。
