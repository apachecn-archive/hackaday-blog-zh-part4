# 项目出口:铰链

> 原文：<https://hackaday.com/2019/07/18/project-egress-the-hinges/>

门的铰链可以说是它最重要的部分。毕竟，没有铰链的门只是一面墙。或者舱壁，如果我们说的是宇宙飞船上的无铰链舱门。

因此[为](https://www.youtube.com/watch?v=DDttNgCpquo)[进展出口](https://hackaday.com/2019/07/03/forty-four-hackers-and-a-hatch-progress-egress-takes-off/)制作铰链的任务，通过制作指挥舱舱口的复制品来庆祝阿波罗 11 号着陆 50 周年的任务，交给了【吉米·迪瑞斯塔】。铰链是复杂的联动装置，不仅设计用于处理发射台 225 磅(102 公斤)的舱口，还允许在前往月球的途中进行扩展的舱外活动(EVA)。[Jimmy]是一名多媒体制作人，他很可能会像在木头上工作一样加工金属，他的铰链是对 20 世纪 60 年代航空航天工程的研究，由 ipe、极其坚硬致密的热带硬木和黄铜制成。

[Jimmy]的建造始于一个全尺寸的 3D 打印铰链模型，这一举措获得了回报，因为打印既可以作为加工木质组件的模板，也可以作为测试夹具，以确保一切都能正确连接。黄铜片被弯曲并焊接到铰链支架上，而黄铜棒在车床上转动以模拟原始的液压缸铰链支撑。黑色的 ipe 和黄铜配合得非常好，应该会和组装好的舱口上的[【弗兰布兰奇】的胡桃木和黄铜插销](https://hackaday.com/2019/07/04/project-egress-fran-makes-a-latch/)很好。

随着[亚当·沙维奇]的所有部件的最后组装定于 18 日星期四，我们对阿波罗和至少部分由此诞生的创造者运动的庆祝活动已经接近尾声。
