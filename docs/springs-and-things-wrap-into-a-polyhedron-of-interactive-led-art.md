# 弹簧和东西被包装成一个互动 LED 艺术的多面体

> 原文：<https://hackaday.com/2020/08/02/springs-and-things-wrap-into-a-polyhedron-of-interactive-led-art/>

抖动球和最近流行的某种病毒之间的任何相似之处纯属巧合。虽然又一个项目在新冠肺炎锁定期间进行，我们可以看到灵感来自哪里。

《摇摆球体》是另一件互动艺术作品，它来自于[罗宾·鲍姆加滕]显然是春天驱动的想象力，他的[量子花园](https://hackaday.com/2019/03/06/door-springs-and-neopixels-demonstrate-quantum-computing-principles/)作品去年曾登上我们的版面。更早、更平的版本使用了一系列弹簧门挡——那种路过的脚拨动时听起来很棒的弹簧门挡——每个弹簧门挡周围都有一个新像素环。弹簧充当触摸传感器，以无穷无尽的迷人方式改变 LED 环上的图案和颜色。

对于 Wobble Sphere，[Robin]采用了相同的 spring 和 LED 单元，将它们分解成一组六边形和五边形的 PCB，并将整个东西包装成一个 72 面的多面体。从 2D 到 3D 空间的过渡涉及到一些令人印象深刻的机械和电气工程，尤其是解决如何在提供方便的结构支撑的同时连接一切的问题。前者是通过大量带状电缆实现的，而后者是通过 3D 打印骨架和相邻 PCB 之间的焊接连接来实现的。其结果是一个显示器，邀请触摸，并奖励它美丽的光图案围绕球体追逐。休息之后，请观看视频中的实际操作。

以免有人认为弹簧是[Robin]盒子里的唯一工具，我们不能忘记他曾经让一个挥舞着刀子的 Arduino 驱动的游戏面对不知情的公众。检查一下；这比听起来有趣多了。
