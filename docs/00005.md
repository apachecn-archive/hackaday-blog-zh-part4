# 不要放盐和黄油，这爆米花是给机器人的

> 原文:[https://hackaday . com/2018/07/11/hold-the-salt-and-butter-this-popcorn-is-for-a-robot/](https://hackaday.com/2018/07/11/hold-the-salt-and-butter-this-popcorn-is-for-a-robot/)

爆米花！它又轻又蓬松，是一种非常灵活的零食。我们可以把它们做成原味的，用一些盐和黄油做一道美味的点心，或者如果你喜欢吃甜食，可以裹上焦糖。现在，康奈尔大学向我们展示了另一种享受爆米花的方式:利用它们的爆裂动作作为机器人致动器的机械力。

乍一看，这可能不太正统，但很有道理。我们通过加热水来爆裂玉米，直到它变成蒸汽，引发体积的快速膨胀。这与我们的发动机燃烧空气燃料混合物以产生体积的快速膨胀没有太大区别。或者利用热能将水煮沸并引发其膨胀成蒸汽。因此，爆米花的玉米粒可以作为一个小型、简单、独立的引擎，将热能转化为机械能。

显然，这将是一个单一用途的机制，但这对合适的利基市场来说是完全合意的。当爆米花如此便宜时，一次性使用更容易吞咽，而且可生物降解，残留最少。这篇研究论文展示了三种利用爆裂玉米机械能的方法，但这并不是一份详尽的清单。公开邀请大家集思广益，将其他创意添加到菜单中。

当然，如果你更喜欢糖果而不是爆米花，你可以用甘草做一个[机器人执行器。](https://hackaday.com/2017/02/07/33c3-edible-soft-robotics/)

不管怎样，机器人起义都会很好吃。
