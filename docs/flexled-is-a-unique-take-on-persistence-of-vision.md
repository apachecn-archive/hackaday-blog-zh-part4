# FlexLED 是视觉暂留的独特体现

> 原文：<https://hackaday.com/2019/07/09/flexled-is-a-unique-take-on-persistence-of-vision/>

很多黑客都做过视觉暂留效应的实验。挥动一束发光二极管，在合适的时间让它们闪光，就有可能让图像看起来悬在空中。有很多方法可以做到这一点，无论是用手摇动 led，旋转它们，甚至是把它们放在你的自行车车轮上。[Carl Bugeja]走了一条不同的路，[利用了柔性印刷电路板创造的可能性](https://hackaday.io/project/164585-flexled)。

[卡尔]的项目被称为 FlexLED。这恰如其分地描述了这一构建，在原型形式中，在柔性 PCB 的末端安装一个 LED。PCB 本身具有形成线圈的迹线图案，这使得它能够与磁场更强地相互作用。通过让正确的电流流过线圈，可以使柔性 PCB 上下摆动，快速移动末端的 LED。然后通过控制 LED 的闪烁，就有可能创造出视觉暂留的效果。

目前只安装了一个能够显示 3 种颜色的 LED，FlexLED 的视觉显示有些受限。然而，[Carl]报告说，这种效果在真人身上比在摄像机上更令人印象深刻，他已经在计划将该项目扩大到多 LED 显示屏。

POV 技术可以做一些令人印象深刻的事情，甚至可以进行立体显示。如果你自己正在做什么，一定要让我们知道。休息后的视频。
