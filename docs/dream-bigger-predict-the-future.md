# 梦想更大，预测未来

> 原文：<https://hackaday.com/2021/10/30/dream-bigger-predict-the-future/>

我很想告诉你，我从来没有错过，但我错了很多次。记得 Arduino 吗？当它还是全新的时候，我认为它只是一些愚蠢的库和下拉菜单，供那些懒得打出自己的`#include`语句的人使用。不用说，它发起了大约一百万次黑客攻击，并将微控制器编程带入了主流。哎呀。

类似地，大约十五年前，我看到麻省理工学院媒体实验室的一个教育项目。它由一堆上面有液晶显示屏的积木组成，当它们放在一起时会相互作用。然而，真正吸引人的是每个积木内部都有一个加速度计，所以你可以把水从一个积木里“倒”到另一个积木里。

那时，加速度计很贵，即使是批量生产。即使是其中的一个立方体在当时也要 100 美元，更不用说一整套了。加速度计是如此昂贵，以至于我不会考虑在一个项目中加入一个，更不用说十几个了，所以出于黑客的目的，我忽略了它们。然后是手机和规模经济。今天，即使在芯片短缺时期，它们也很容易买到，每个大约 2 美元，这使得它们在这种“无聊”的用途中非常有用。

从 Arduino 的经历中，我学会了永远不要低估在我看来是“小”便利的影响。(也许更重要的是，来自社区的巨大共同努力的价值。)麻省理工学院加速度计的故事告诉我们，一些部件在未来会变得非常便宜，所以你不应该把这种很酷的新传感器排除在你的设计之外。毕竟，十年前，没有人会想到我们会以不到一个汉堡的价格拥有激光飞行时间测距仪。

哪些新组件非常有用，或者充满潜力，在未来可能足够便宜，值得研究？明天早上来 Hackaday，加入我们的讨论吧！
