# 富米克:一只 Arduino 壁画机器人水母

> 原文：<https://hackaday.com/2022/03/10/fumik-an-arduino-wall-drawing-robot-jellyfish/>

如果你曾经想建立一个大画幅绘图仪，但没有足够的空间，也许可以把它挂在墙上，让它变得可爱。这是壁绘机器人 Fumik 背后的想法。正如你所料，这个小装置只是一个带笔的运动基座。我们希望墙上贴着纸，因为不是每个人都想在他们的干墙上贴上电脑生成的艺术作品。

最大尺寸显然是 5 米宽，3 米高，有足够的空间来表达自己。控制器是 Arduino Mega，带 CNC 护罩的步进电机驱动整个组件。有趣的是，马达和电子设备都在水母身上，而不是在墙上。

该设备一次只能容纳一支笔，但您可以使用一种颜色进行绘制，然后手动更换笔。GitHub 上的文件很好，但是你需要从视频中直观地了解一些机制。然而，因为它使用现成的硬件，所以应该很容易弄清楚。这看起来像一个廉价而欢快的墙壁绘图仪，结果不言自明。

我们见过类似的[墙壁绘图仪](https://hackaday.com/2021/01/19/pen-plotter-draws-maps-directly-on-the-wall/)。[不止一次](https://hackaday.com/2018/10/08/another-drawbot-uses-a-pi-and-web-sockets/)，甚至。
