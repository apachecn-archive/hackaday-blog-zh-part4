# 雄心勃勃的 LED 立方体提供了无尽的视频游戏滚动；扮演恶魔城

> 原文:[https://hack aday . com/2019/11/04/ambition-led-cube-provides-endless-video-game-scrolling-plays-castlevania/](https://hackaday.com/2019/11/04/ambitious-led-cube-provides-endless-video-game-scrolling-plays-castlevania/)

LED 立方体现在风靡一时，考虑到投入的大量工作和人们发现用它们做的有趣的事情，这是理所当然的。然而，[Greig Stewart]并不满足于制作另一个位置敏感显示器或抽象设计，而是选择了更具雄心的东西:[一个 LED 立方体，上面有一个可玩的游戏*恶魔城*。](https://www.element14.com/community/community/halloween/blog/2019/10/31/nes-castlevania-on-an-led-matrix-cube)

正如雄心勃勃的项目经常做的那样，这个项目需要利用以前的艺术，其中一些我们以前已经介绍过了。[Greig]从立方体建造者那里获得灵感和信息，如[【polyloyd】](https://hackaday.com/2019/04/30/opengl-shaders-and-an-led-cube/)[【Greg Davill】](https://hackaday.com/2019/09/05/handmade-led-cube-is-a-work-of-art/)[【kbob】](https://hackaday.com/2019/09/05/this-led-cube-is-one-heck-of-an-icebreaker/)，将六个 64 LED 矩阵面板投入使用。弄清楚结构元素是早期的绊脚石，但[Greig]用 3D 打印的支架和铰链完成了它，这本身就是一件艺术品；整个东西看起来像是博格人建造的。里面的 Raspberry Pi 使 Gameboy 模拟器成为可能，他的第一次尝试是同时运行六个不同的游戏，每个面板上一个。他选定了一个游戏，经典的侧滚游戏*恶魔城*，只在四个面板上玩。需要一些魔法来解卷游戏，这样角色可以绕着立方体走，而不是让背景滚动；你可以在下面的视频中查看结果。

目前，立方体坐在一个旋转的转盘上，一个小马达控制旋转以响应脚的控制。【Greig】希望将电机置于游戏的控制之下，以便物理滚动与游戏性同步；我们衷心赞同这一计划，并期待着结果。
