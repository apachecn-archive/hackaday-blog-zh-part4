# 引导扇区 Pong 作为组装速成班

> 原文：<https://hackaday.com/2020/09/28/boot-sector-pong-as-a-crash-course-in-assembly/>

你有没有想过开发一个足够小，可以放入磁盘 512 字节引导扇区的游戏？看某人用汇编语言开发一个程序将近两个小时怎么样？如果你对其中一个问题的回答是肯定的，或者两个问题都是肯定的，你将会爱上这个来自[Queso Fuego]的项目。

无论你只是想查看公共领域的源代码，还是看着他从一个空白文件开始，为你的阅读乐趣编写每一行代码，你都有可能从这个项目中学到一两个技巧。例如，他解释了游戏中的所有“图形”是如何在 80 x 25 文本模式下完成的，只需设置角色单元格的背景颜色，而无需打印任何文本。

我们真的很喜欢休息后视频中的演示，这是在多天的过程中录制的，通过背景中不断变化的光线水平来判断。当他打出每一行代码时，他解释它的功能是什么，并给出任何必要的背景信息来解释它如何适应更大的程序。如果你曾经想知道你是否有能力在 ASM 中编程，看这个视频是一个很好的决定。

[Queso Fuego]提到这个项目，以及他对这种低级编程的[研究](https://hackaday.com/2020/05/10/512-bytes-ought-to-be-enough-for-dinosaurs/)，是由于我们许多人都感觉到的[社交距离无聊](https://hackaday.com/2020/08/14/tech-at-home-winners-who-made-the-best-of-their-quarantine/)而产生的。虽然我们当然不主张让他永远被锁在家里，但有了这样的项目，你必须承认这对我们其他人来说似乎是一个胜利。
