# 黑客 USB 黑客聊天

> 原文:[https://hackaday.com/2020/02/24/hacking-usb-hack-chat/](https://hackaday.com/2020/02/24/hacking-usb-hack-chat/)

加入我们太平洋时间 2 月 26 日星期三中午与[凯特·坦金](https://hackaday.io/hacker/254157-kate-temkin)的[黑客 USB 聊天](https://hackaday.io/event/169792-hacking-usb-hack-chat/)！

尽管通用串行总线有着令人不快的特质，但近四分之一世纪以来，它一直是外设连接的游戏规则改变者。曾经只是将鼠标和键盘连接到计算机的一种手段，现在已经被扩展和增强成比最初的设计者想要的要多得多的东西。导致 USB 这些创新用途的灵活性也导致了它的无处不在，几乎每一种可以想象的设备都出现了某种形式的连接器。

Kate Temkin 非常熟悉通用串行总线的复杂性。作为 Great Scott Gadgets 的软件负责人，凯特为 GSG 的产品开发了软件和固件，比如 T2 的 GreatFET 和 T4 的 HackRF。Kate 还参与并维护了许多开源项目，包括 FaceDancer 项目。当她不忙于这些事情时，她会在 USB 安全培训课程中分享她深厚的知识，展示 USB 如何容易受到攻击，以及如何防止攻击。

加入我们本周的黑客 USB Hack Chat，在这里 Kate 将讨论任何关于 USB 的事情。来了解 USB 标准的未来，以及如何让您的 USB 项目走上正轨。

[![join-hack-chat](../Images/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)中的社区直播活动。本周，我们将于太平洋时间 2 月 26 日星期三中午 12:00 坐下来讨论。如果时区让你烦恼，我们有[一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20200226T12&p0=224&msg=Hacking+USB+Hack+Chat&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
