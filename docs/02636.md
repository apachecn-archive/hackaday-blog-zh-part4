# 使用 3D 打印设备数控加工您自己的印刷电路板

> 原文:[https://hack aday . com/2019/04/05/CNC-your-own-PCBs-with-a-3d-printed-mill/](https://hackaday.com/2019/04/05/cnc-your-own-pcbs-with-a-3d-printed-mill/)

是的，你可以快速设计出一个印刷电路板，把它送到一个工厂，然后在一两周内收到一个完成的电路板。有一些快速组装工厂会制造电路板并为你组装。但有时你现在需要一块电路板，这就是我们进入家用 PCB 制造的时候。你可以用蚀刻或铣削来做到这一点，但[伦佐]有一个很好的解决方案。[他建造了一台 3D 打印铣床，将制作印刷电路板。](https://www.mischianti.org/2019/04/03/cnc-cyclone-pcb-factory-mechanics-part-1/)

这个微型研磨机的设计是基于一个手持旋转工具，也称为 Dremel，但这就像面巾纸，所以只需购买 Proxxon。该轧机采用 3D 打印 T 型轨道设计，在光滑的杆上安装线性轴承，配有标准的 NEMA 17 步进电机和人字齿轮，几乎没有齿隙。这里发生了很多事情，但是幸运的是[伦佐]有一个完整构建过程的视频教程可以在下面看到。

我们之前已经看到了[伦佐]在自制 PCB 制造方面的一些努力，[直到并包括在烧结的帮助下使用绿色阻焊膜](https://hackaday.com/2019/03/22/create-green-soldermasked-pcbs-with-fritzing/)。这很好，非常好，唯一真正区别于制造 PCB 的是没有电镀通孔。那只是一点石墨和电镀，我们期待着[伦佐]在国内制造 PCB 的进一步冒险。
