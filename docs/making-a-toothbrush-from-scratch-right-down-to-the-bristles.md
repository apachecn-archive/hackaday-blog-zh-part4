# 从头开始制作牙刷，一直到刷毛

> 原文：<https://hackaday.com/2021/02/16/making-a-toothbrush-from-scratch-right-down-to-the-bristles/>

我们中的大多数人可能会用一把最多值几美元的牙刷，这种牙刷由注射成型塑料制成，用于娇嫩的口腔。也许如果你是一个真正的洁牙者，你会有一个花哨的电动牙刷。我们敢打赌，很少有人会从零开始加工他们自己的定制牙刷，但是如果你想要一些灵感，T2 正在这么做。

许多工作对硬模加工爱好者来说是熟悉的。木柄的制作非常精细，包括多种染色和涂漆木材的堆叠——在这种情况下，犀木是两者中较浅的一种，苋菜红提供了丰富的红色。阀杆是时尚的不锈钢件，优雅地弯曲成有品位的曲线。最后，光是刷头的组装就值得一看。它是定制的——用步进电机用自动夹具定制切割钢垫板和钓鱼线刷毛。我们怀疑钓鱼线不适合牙科使用，但是尼龙线至少和普通牙刷差不多。

虽然我们可能不会在没有咨询牙科专业人士的情况下将它放在我们的嘴唇之间，但这是一个很好的视频，可以让我们了解如何在工作室制作美丽的定制物品。我们以前也看到过[W&M·莱夫沙]的优雅作品——[一个令人愉快的折衷式帽枪打火机。](https://hackaday.com/2021/02/06/cap-gun-lighter-built-from-scratch/)休息后的视频。
