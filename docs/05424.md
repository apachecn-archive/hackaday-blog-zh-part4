# 买三十年没造的芯片要知道什么

> 原文:[https://hack aday . com/2020/01/15/what-known-when-known-when-buying-the-thinks-that-been-that-thinks-been-that-been-study-study-the-thinked-thinks-thinked-thinked-30 年来一直没有生产的芯片/](https://hackaday.com/2020/01/15/what-to-know-when-buying-chips-that-havent-been-made-for-three-decades/)

近年来，我们中与雅马哈 FM 合成器等老式声音发生器芯片打交道的人很可能遇到过“假”或“有标记”的芯片，有时会被重新标记为与封装内的芯片完全不同的芯片。来自 Plogue 的 David Viens 经过 3 年的研究终于发布了他的发现。(视频，嵌在下面。)

首先要确定的是这些芯片是以什么方式“造假”的？显然，2015 年雅马哈没有生产新的 YM2612，但这并不意味着这些只是另一家模具厂生产的未经许可的克隆产品。[David]解释了这些芯片通常是原始样品，来源于回收的电子废物，这些废物来自中国大多数环境不安全的操作，然后被翻新并被经销商说成是“新的”。令人欣慰的是，截至 2017 年，他解释说，这些工厂中的大多数现在都被关闭，并转移到一个工业园区，在那里可以以污染更少的方式进行工作。

[大卫]要研究的下一件事是如何发现这些有标记的芯片。他解释了如何在 IC 封装中使用指示性标志来识别哪个芯片厂生产了它们，以及从电路板上拆下并修复的芯片的可见指示。有不同的方式可以做标记，有时可以解开黑色的顶部，因为它被称为，并显示下面的原始标记与简单的应用丙酮与棉签。

我们之前已经讨论过[假芯片](https://hackaday.com/2017/08/22/fake-ram-identifying-a-counterfeit-chip/)和[如何导致硬件故障](https://hackaday.com/2017/05/31/counterfeit-hardware-may-lead-to-malware-and-failure/)，但是在这些已经不再生产的芯片的情况下，我们除了 [FPGA](https://hackaday.com/2016/11/24/giving-the-world-a-better-sid/) 或[软件重新实现](https://hackaday.com/2012/08/31/propeller-turned-into-chiptune-player-with-a-software-sid/)之外没有太多选择。休息之后，请看[David]对这些筹码的 40 分钟回顾。
