# 焊接练习工具包在完成后仍然有用

> 原文：<https://hackaday.com/2020/09/26/soldering-practice-kit-remains-useful-after-completion/>

[Jim Heaney]不满足于标准的焊接练习套件，这些套件一旦制成就几乎没有任何用途，他决定亲自动手，为他的班级设计一个易于组装的套件，一旦组装起来，它将成为任何制造商工作台上最方便的工具:[一个功能性电压表](https://hackaday.io/project/174663-voltmeter-soldering-kit)。

该套件的核心是一个标准的 Atmega 328P 微控制器。虽然他本来可以选择更小或更便宜的产品，但不仅体积大的部分更容易焊接，[Jim]还认为，如果他的学生想制作像 Arduino 这样的试验板，这种芯片很容易重新利用。电压表有一个从 0 到 100 VDC 的固定测量范围，板上仅有的开关是用于给它通电和一个保持按钮，该按钮冻结当前显示在三位七段显示屏上的值。

除了向其他制造商和教育工作者出售他的套件，[Jim]还希望他的项目鼓励其他人设计类似的焊接套件，这些套件有利于某种功能，而不是在所有焊盘上都有焊料时就被丢弃，以及零件品种和文档。如果你是在焊接光谱的另一端，那么为什么不挑战你的技能焊接时间限制？
