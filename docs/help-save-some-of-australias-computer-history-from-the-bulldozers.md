# 帮助从推土机中拯救一些澳大利亚的计算机历史

> 原文：<https://hackaday.com/2018/08/04/help-save-some-of-australias-computer-history-from-the-bulldozers/>

当多个线人写信告诉我们一个故事时，我们可以说这是一个重要的故事。今天早上我们收到消息，位于悉尼郊区维拉伍德的澳大利亚计算机博物馆协会的仓库即将被拆除，他们迫切需要拯救仓库中的文物。他们需要有相当大的备用存储容量的澳大利亚人来帮助他们保存和存储藏品，而他们只有几天的时间来做这件事。

来自 EEVblog [的热情洋溢的 Dave 发布了一个视频，在视频中他进行了一次旅行](https://www.youtube.com/watch?v=mK4M8dv4NNU)，和我们一样，他不断地对他发现的物品惊叹不已。一台 EAI 模拟计算机，全套 DEC PDP-11 技术文档，一个巨大的英特尔开发系统，Tektronix 打印机，巨大的 DEC 机架，VAXen 的存储卡，成堆成堆的文档箱，等等。

因此，如果你是一个澳大利亚人，在悉尼附近，碰巧有一个目前未使用的仓库、谷仓或工业单位，可以存放一些这种东西，[尽快与他们联系](http://www.acms.org.au/)。其中一些很可能是垃圾，但在这个宝库中无疑蕴藏着许多需要拯救的东西。我们自己也会在那里，但是很遗憾我们在世界的另一边。
