# 你可以把软饮料瓶变成方便的太阳能灯

> 原文：<https://hackaday.com/2022/03/29/you-can-turn-soft-drink-bottles-into-handy-solar-lamps/>

太阳能灯是一种流行的花园装饰。当然，它们在大多数五金店和花园商店都很便宜，但是如果你更喜欢自己动手做，你可能会喜欢自己动手做。[opengreenenergy]已经做到了，使用回收材料进行廉价而简单的设计。

这个设计的灵感来自于莫泽瓶，这是一个装满水的瓶子，用来在白天将阳光散射到房间里。然而，这种设计使用 LED 代替阳光来提供照明，用于装饰花园或外出露营或旅行时使用。

在本设计中，太阳能电池板用于在白天使用 LP4060B5F 充电控制器 IC 为锂聚合物电池充电。它与 AP6685 电池保护 IC 配合使用，以确保电池不会过度放电或在使用中损坏。当天黑时太阳能电池板停止供电，LED 就会自动打开。它可以设置为较低或较高的亮度，以根据需要提供更多的运行时间或更多的光线。

所有的电路都被包裹在一个整洁的 3D 打印外壳中，允许硬件直接拧在常规软饮料瓶的顶部。再加上瓶中的一些水，也许还有一点漂白剂来阻止藻类的生长，结果是一个方便、便携的灯，而且有足够的质量来避免它被轻易吹倒。
