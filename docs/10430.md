# 印花双体船

> 原文：<https://hackaday.com/2021/06/15/printed-catamaran/>

如果你想把一些仪器送到湖上或海上，你会想要一些能漂浮的东西。当然，如果你需要在水下，或者如果你可以在水上飞行，还有其他选择，但有时你想在水面上。就稳定性而言，很难击败双体船——一艘有两个船体的船，每个船体支撑甲板的一侧。如果这听起来像是你梦想中的海洋传感器平台，试试[打印来自【electrosync】的](https://www.thingiverse.com/thing:4884505)。

该船看起来超级稳定，有一个无刷电机推进系统。设计目的是携带环境和水质监测装置。它可以在船体中容纳超过 5 公斤的有效载荷，并且有一个可选的甲板系统，尽管 STL 文件中尚未包含该计划。

从照片中很难看出比例，但这是一米长，所以它不适合大多数人的浴缸。你可以在下面的视频中看到这只野兽。这些视频大约有一年了，但是 STLs 是这个月才发布的。

我们总是喜欢看到人们在普通 3D 打印机上 3D 打印出的非常大的东西。我们也喜欢看[实用的印刷品](https://hackaday.com/2021/03/25/speaker-is-fully-3d-printed/)。当然，我们见过[更大的船](https://hackaday.com/2021/01/30/3d-printing-a-full-scale-fiberglass-speedboat/)，但不是用消费级打印机。
