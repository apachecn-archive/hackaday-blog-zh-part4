# 连续硅胶模制

> 原文:[https://hackaday.com/2021/03/26/serial-silicone-molding/](https://hackaday.com/2021/03/26/serial-silicone-molding/)

正如凯文·霍姆斯(Kevin Holmes)所发现的那样，当你需要制造数十个定制模制零件时，制造个位数数量的定制模制零件的技术并不适用。他需要制作 80 多套硅胶马达支架，而一模制工艺是行不通的。他探索了几种解决方案，但都因过于复杂而遭到拒绝。最后[Kevin]想出了一个主意，用库存金属棒的轨道将模具夹在一起，形成菊花链。观看是一个非常漂亮的过程，你可以看看下面的视频，它与非常慢的 7495 四位移位寄存器没有什么不同。

![](../Images/d9a0d67176e2abaa85e53d3f513910cd.png)

尽管他使用的硅胶是透明的，但注意，当它在模具之间传播时，你仍然可以看到它的存在。他手动执行从一个库到下一个库的半字节进位操作，我们想知道他是否可以级联这些库，并在一次真正的大挤压中注入所有 80 个库？

你可能会问，为什么有人需要 80 套硅橡胶电机支架？嗯，你可能还记得我们[在 2018 年](https://hackaday.com/2018/08/27/time-stretching-zoetrope-animation-runs-longer-than-it-should/)写过的 [4-mation 3D 窥镜](https://4-mation.co.uk)。[Kevin]是这个迷人项目的创始人之一，看起来他们的 Kickstarter 项目已经成功了。正如他在下面的视频中演示的那样，如果没有某种类型的降噪装置，马达发出的隆隆声会在 zoetrope 的舞台上被放大。如果你有任何喜欢的小批量制造的模具制作技巧，请在下面的评论中告诉我们。感谢[乔治·格雷夫斯]给我们发来这条提示。
