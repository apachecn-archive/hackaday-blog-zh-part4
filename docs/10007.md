# 用倒计时器改装卡西欧 W800-H

> 原文：<https://hackaday.com/2021/04/30/modding-a-casio-w800-h-with-a-countdown-timer/>

股票，卡西欧 W800-H 腕表配备了双时间模式，多种闹钟和秒表-对一些人来说很有用的功能。然而，如果你知道在哪里找，更多是可能的。[[Ian]决定潜入引擎盖下，启用工厂隐藏的倒计时功能。](https://ianrrees.github.io/2019/09/13/casio-w-800h-extras.html)

黑客攻击包括打开手表的外壳，露出主印刷电路板的背面。在那里，一系列跳线控制各种功能。[Ian]的理论是，这使得卡西欧可以在各种手表之间共享一个基本的 PCB，并通过跳线选择来实现功能，从而节省制造成本。只需一点点焊芯，就可以断开跳线焊盘，实现隐藏倒计时功能。其他功能，如多个报警，可以用其他跳线以同样的方式禁用，这表明低功能型号也使用相同的板。

这是一个有用的技巧，这意味着当他需要的时候，他的手腕上总会有一个倒计时器。现在很难找到煮过头的借口，但我们确信他能应付。当然，手表黑客不一定是电子的—[正如这个苹果手表的定制透明外壳所展示的](https://hackaday.com/2021/01/27/apple-watch-gets-custom-transparent-case/)。休息后的视频。
