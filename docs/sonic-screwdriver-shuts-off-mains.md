# 声波螺丝刀切断电源

> 原文：<https://hackaday.com/2019/05/30/sonic-screwdriver-shuts-off-mains/>

在神秘博士的世界里，声波螺丝刀是一种功能广泛的多功能工具。[哈特利]想为自己做些那样的事情，于是自己造了一个装置。

由于无法再现该节目中的各种特征，他选择了一些更简单的东西。该设备配有 ATTiny85 和 433MHz 发射机。它被编程来开关无线控制的电源插座。这让他只需轻轻一点螺丝刀就能控制家中的电器。电源由传统的 AA 电池提供，升压转换器将其升压至 5V 以运行电子设备。

它被包裹在一个 3D 打印的盒子里，这个盒子经过精心设计，可以容纳里面的所有部件。PCB 布局的纸质模型也用于设计阶段。[Hartley]充分利用了 CAD 软件，确保每样东西第一次就能完全吻合。

这是一个有趣的项目，就像音速起子一样。休息后的视频。
