# 激光皇冠夜间闪耀

> 原文：<https://hackaday.com/2019/09/16/laser-crown-shines-at-night/>

有时候你需要一些非常华丽的东西来完成一套服装。无论是闪亮的耳环还是时尚的项链，配饰都是在时尚战场上竞争的关键。对于那些想要带来一些严重火力的人来说，激光冠可能正是医生所要求的。

首先，我们应该声明，表冠仅在制造过程中使用激光切割毛毡和亚克力材料。光源是 Adafruit 的 Neopixel 戒指，能够带来鲜艳的色彩，而没有伤害眼睛的风险。然后，戒指由一系列咬合在一起的丙烯酸部件和一顶毡帽组装而成，帽子上有发夹槽，用于将皇冠固定在佩戴者的头上。一个小饰品驱动着这场表演，一个 LiPo 电池被用作轻型电源。

这是一座很好的建筑，肯定会引起很多人的注意。我们在这些部件周围看到了一些很棒的可穿戴设备；[这件 EL 夹克是特别受欢迎的](https://hackaday.com/2019/01/12/cyberpunk-jacket-is-the-garment-of-choice-for-the-streets-of-2019/)。休息后的视频。
