# 吉他英雄控制器获得新的音乐生活

> 原文:[https://hack aday . com/2020/01/07/guitar-hero-controller-gets-a-new-musical-life/](https://hackaday.com/2020/01/07/guitar-hero-controller-gets-a-new-musical-life/)

吉他英雄曾经是一件大事，直到现在。视频游戏行业尽了最大努力也无法重新唤起人们对节奏游戏的热情，全球各地的旧货店现在都堆满了小塑料乐器。[Analog Sketchbook]决定为 Wii 赋予其中一把吉他新的生命，[将其重新用作合成器控制器。](https://www.instructables.com/id/Keytar-Hero-Using-a-Wii-Guitar-Controller-As-a-Syn/)

由于现代制造商对电子问题的解决方案的流行，这种构建非常简单。与吉他挂钩是一个解决的问题，Adafruit Nunchucky 分线板允许吉他英雄控制器通过跳线连接到树莓 Pi 的 IO 引脚。

通信是通过 I2C 进行的，很容易在 Pi 上运行纯数据。[Analog Sketchbook]创建了一个运行合成器的补丁，由吉他本身的按钮和控件控制。使用这种设置，您可以创建任意数量的不同套路，让吉他以不同的方式演奏。我们希望看到一个 chiptune 风格的琶音补丁，或者像《创世纪》一样播放胖胖的 FM 合成音的东西，但这只是我们的看法。天空真的是这里的极限，在 Pi 上有大量的咕噜声用于各种形式的合成。

这是一个有趣的构建，为一个被遗忘的游戏配件赋予了新的生命。[早在 2010 年](https://hackaday.com/2010/02/04/guitar-hero-as-an-instrument-or-midi-controller/)，我们就已经看到它们被重新利用。休息后的视频。
