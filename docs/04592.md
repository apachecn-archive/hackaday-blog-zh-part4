# 跳舞的牛仔数码管

> 原文:[https://hack aday . com/2019/10/27/a-跳舞-牛仔-谢妮-tube/](https://hackaday.com/2019/10/27/a-dancing-cowboy-nixie-tube/)

如果有四个你从未想过会依次听到的单词，它们可能是“跳舞的牛仔数码管”。但是[这正是【Glasslinger】所做的](https://www.youtube.com/watch?v=9r5NCVR_zYU&)，而且听起来也确实如此——一个跳舞的胶囊牛仔。

我们已经将结果视频放在了休息时间的下方，在视频中，我们看到了一个引人注目的谢妮建造之旅，以及所需的专业工具。小接触，如需要绝缘与玻璃毛细管的电线不应该发光，建设的信封和干，并精心布局的各种牛仔元件在云母片解释。

电子管在我们面前成型，一个驱动 PCB 被蚀刻，整个排列被放置在一个定制的木箱中。这是最好的老派建筑，唯一的现代感来自安排各个部分的 Arduino Uno。不过，不难想象，随着时间的推移，霍尼韦尔机械定序器可能会被用于同样的任务。

当然，我们之前已经给你带来了[Glasslinger]的作品，但我们也看到了[一些更传统的自制 Nixies](https://hackaday.com/2013/11/07/homemade-nixie-tubes/)。
