# 永不遗忘的金属:镍钛诺和形状记忆

> 原文：<https://hackaday.com/2018/11/23/the-metal-that-never-forgets-nitinol-and-shape-memory/>

你可能以前听说过镍钛诺线，但我们怀疑共同的基础知识只不过是重复它是一种形状记忆合金。工程师比尔·哈马克带领我们进行了一次关于镍钛诺和形状记忆合金的快速旅程。

这个名字本身就像你说纸巾的时候说面巾纸，或者你说钩环扣的时候用 V 字。*Nickel**Ti**tanium**N**aval**O**rd nance**L**aboratories*的前几个字母组合起来形成了 1962 年开发的一种镍钛合金的名称:镍钛合金。它被称为形状记忆，因为你可以在室温下拉伸或弯曲它，当加热到大约 75°C(167°F)时，它会恢复到原始形状。这种特殊的金属可以做到这一点，因为它的键形成了菱形的“孪生结构”——弯曲或拉伸会移动这些菱形(或菱形，随你选)，但不会改变哪些原子相互键合。

这次材料科学之旅让你厌烦到流泪了吗？这就是我们热爱[比尔]工作的原因。他总是做得很棒，去神秘化普通的神秘主义，这次也不例外。下面的视频更好地说明了我们上面描述的内容，但也拉出了一个镍钛诺引擎，增加了惊喜因素。一块直的镍钛诺绕着两个滑轮弯曲成一个环。较低的滑轮浸没在热水中，导致镍钛诺想要伸直，但它会循环回到顶部滑轮，在空气中弯曲和冷却，并产生驱动发动机的杠杆效应。去年我们看到了这个概念的更复杂版本。

你知道那些你可以以任何方式弯曲的眼镜框，它们会弹回原来的形状吗？他们利用了镍钛诺的超强弹性。[Bill]还讲述了在医疗应用中作为支架的用途，以及汽车工业中古怪的工程技巧。
