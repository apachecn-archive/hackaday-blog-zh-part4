# 便携式比萨饼烤箱的温度超过 900 度

> 原文：<https://hackaday.com/2022/01/24/portable-pizza-oven-has-temperature-level-over-900/>

虽然可以在家里从零开始制作披萨，直到面团本身，但如果没有披萨烤箱，复制这种味道和精致的口感将是一场斗争。大多数家用烤箱的温度都超过了 260 摄氏度/500 华氏度的上限，比萨饼的烹饪效果最好，而比萨饼烤箱的温度通常会比这个温度高得多。我们大多数人都没有足够的资源在家里放置一个商业级的烧柴砖烤箱，[但是下一个最好的东西就是这个来自[Andrew W]](https://www.youtube.com/watch?v=yPzFmZpNd9o) 的便携式披萨烤箱。

建造从一些金属板开始，形成烤箱的外壳和内盖。[Andrew]通过一些测试发现，弯曲的形状似乎可以产生最好的效果，所以金属板在焊接在一起之前要经过辊子来获得它的形状。烤箱的大致形状完成后，他制作了两个不同的燃烧器。一个放在烤箱的后面，有自己的扩散器来保持烤箱尽可能热，另一个放在堇青石下面，从底部加热。两者都由定制的铜管道输送气体，当它点火时，温度足以在几分钟内烤好一个比萨饼。有了一些可折叠的桌腿，烤箱也变得相当便携，它的小尺寸意味着它可以比传统烤箱加热得更快。

这是[安德鲁]的第三个原型烤箱，看起来他已经完善了配方。事实上，[我们在大约两年前推出了他以前的一个版本](https://hackaday.com/2020/02/11/portable-pizza-oven-does-the-job-and-fast/)，并且很高兴看到他在这个版本中取得的进步。拥有这样的东西的唯一缺点是总是能够做出美味的披萨对健康的潜在影响，但这是我们愿意冒的风险。
