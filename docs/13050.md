# 2045 年:半人马座阿尔法星

> 原文:[https://hackaday.com/2022/03/15/in-2045-alpha-centauri/](https://hackaday.com/2022/03/15/in-2045-alpha-centauri/)

我们已经讨论了“突破星照”项目，该项目旨在 20 年内将太阳帆探测器发送到半人马座阿尔法星。一点基本的数学知识，并知道半人马座阿尔法星距离我们 4.3 光年，这意味着你需要以超过光速 20%的速度在这段时间内旅行。一些新论文提出了解决一些工程问题的方法。

基本想法很简单。一个非常小的探测器系在一个非常大的帆上。但是称它为太阳帆有点用词不当。帆的动力将是一个强大的激光器，它为微型探测器的推进系统提供更可靠的动力。问题是什么？首先，薄帆在持续的压力下会撕裂。根据其中一篇论文的说法，答案是将船帆做成降落伞的形状，这样它就可以在压力下翻滚。

另一个问题是不要烧掉帆。太空是一个很难倾倒废热的环境，因为辐射是转移废热的唯一途径。另一篇论文提出，船帆上的纳米级图案将允许它向星际环境释放废热。

拟议中的帆宽 3 米，由超薄的氧化铝和二硫化钼制成。这艘帆船将携带一个普通微芯片大小的探测器。在 T1 之前，我们已经看过他们的计划，但是新的文件表明工程正在设计上取得进展。

当然，我们认为会有更多的问题需要解决。但是进入太空变得越来越容易，所以真正的测试在不久的将来是完全可能的。我们也不确定一个微芯片大小的探测器如何将它所发现的信息传回地球。反射激光似乎很困难，你不能逆转方向，沿着光束往回走。就这一点而言，你能在这么小的东西里装多少仪器——大概——从激光或外星太阳获得能量。

再说一次，我们欢迎大胆的新想法，我们希望看到某样东西能飞向另一个星球，至少在我们的孩子能看到的时候。帆已经有了一些实际用途。那么，也可能有人已经从其他地方[送了一个到这里](https://hackaday.com/2018/11/08/did-we-just-get-buzzed-by-alien-space-junk/)。
