# 值得骄傲的 Pi 相机

> 原文:[https://hackaday.com/2022/08/28/a-pi-camera-to-be-proud-of/](https://hackaday.com/2022/08/28/a-pi-camera-to-be-proud-of/)

自 2020 年推出以来，Raspberry Pi HQ 相机已经出现在各种版本中，并为摄影项目带来了许多与专业人士竞争的机会。[我们收到的最新消息来自【Kevin MC aleer】](http://www.kevsrobots.com/blog/pikon-camera.html)，他用一个全尺寸的 Pi 把相机装在一个非常类似于无镜紧凑型相机的盒子里。

盒子里面是一个 Waveshare 触摸屏，可以安装在 GPIO 接头上，还有一个 NanoWave 5000 mAH USB 电池组。相机模块安装在设备的前面，C 型支架可以容纳镜头。软件仍在开发中，有望成为控制各种相机程序的 Python 脚本。有足够多的 Pi 相机项目让软件成为一个选择和品味的问题。

我们喜欢它的外形，也喜欢使用非常紧凑的 NanoWave 电池，所以我们认为这是一个有一些可能性的设计。也许在面对日常摄影时，Pi 端口上的盖子可能会有用。但问题仍然存在，它是否能接近甚至是一个预算的无镜紧凑型相机的性能，我们猜测这将取决于操作者的技能，镜头质量和软件功能，就像它在 Pi HQ 模块上一样。我们期待着看到这个项目的成果，但同时你可以看到一个视频，所有的细节如下休息。
