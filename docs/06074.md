# 自动化洗手液——如果你能找到的话

> 原文:[https://hack aday . com/2020/03/25/automating-hand-sanitizer-if-you-can-find-any/](https://hackaday.com/2020/03/25/automating-hand-sanitizer-if-you-can-find-any/)

我们曾经看过一部《罗密欧与朱丽叶》,里面的两个家庭是现代的暴民家庭，拥有三件套西装和手枪。如果他们在本周推出国王理查德 III，那句著名的台词可能是:“洗手液，洗手液，我的洗手液王国！”即使你在你的预备者缓存里藏了一瓶，你也必须接触瓶子，这样你才能和其他使用者交叉感染。公共场所经常有自动配药器来打击这一点，[现在你也可以](https://www.instructables.com/id/DIY-Easy-Non-Contact-Automatic-Hand-Sanitizer-Disp/)。[Just Barran]在一段视频中展示了该设备，您可以在下面看到。

为项目采购零件有时是一个问题，但现在我们打赌洗手液将是最难的部分。当然，互联网上有很多自制的啤酒，这些啤酒可能是有效的，也可能是无效的，比如啤酒、谷物酒精或其他各种原料。

[巴兰]有一个大垃圾箱。所以他偷了一个 Arduino 和一个超声波传感器。有点棘手的部分是拉下泵。基本的想法是用一个伺服电机拉一些钓鱼线。为了接合瓶子，有一小块来自笔记本封面的塑料，钓鱼线延伸到它的两侧。钓鱼线的一边是固定的，另一边是伺服拉的。

我们可能会使用螺线管来推动按钮，但我们喜欢伺服方法，因为它简单。最后，它看起来确实工作得很好。更换瓶子可能需要一点手术，因为有一个螺丝固定塑料支架，你可能需要更新钓鱼线的长度。对于商业项目来说，这可能是一个障碍，但是对于你自己的使用来说，这似乎不是一个问题。

钓鱼线比你想象的更有用。我们甚至看到它被用作 [3D 打印机](https://hackaday.com/2018/05/30/the-tantillus-reborn/)的皮带。
