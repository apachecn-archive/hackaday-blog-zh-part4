# 超声波阵列为这个万圣节幽灵作家提供能量

> 原文：<https://hackaday.com/2021/10/12/ultrasonic-array-powers-this-halloween-spirit-writer/>

恐怖季节即将来临，随之而来的是想出最怪异的方式来吓唬孩子们的竞赛。动作激活的跳跃惊吓设置总是很受欢迎，但在我们看来是一种廉价的刺激。因此，如果你正在为你的万圣节恐怖地板寻找一些不同的东西，你可能会考虑使用超声波的“精神写作”。

丹·比文的想法是我们在过去几年中见过的超声波悬浮项目的一个变种。虽然观察悬浮在半空中的聚苯乙烯泡沫塑料碎片是很酷的，但[Dan]似乎要将这个概念提升到一个新的水平。很大程度上仍然是一个原型，该装置有一个悬挂在黑暗表面上的 256 传感器矩阵。发酵粉被洒在书写表面以代替灰尘，灰尘很容易被坚硬表面反射的声波干扰。可以控制这个阵列，让它看起来像是一只看不见的手在尘土中描绘出一个图案，这种效果非常令人信服。我们会选择“REDRUM”而不是五角星形，但笔画不同。

[Dan]显然，在为这个重要的夜晚做好准备之前，还有很长的路要走，但概念验证是合理的。在我们等待成品的时候，我们将把它归档，作为一种可能有其他应用的技术。毕竟，SMD 元件很小很轻——也许是超声波拾放器？在这种情况下，[声波镊子](https://hackaday.com/2021/08/02/levitation-by-sound/)可能正是你想要的。
