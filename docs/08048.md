# 厄运土豆

> 原文:[https://hackaday.com/2020/10/13/the-potatoes-of-doom/](https://hackaday.com/2020/10/13/the-potatoes-of-doom/)

多年来，1993 年的经典《毁灭战士》已经获得了一个近乎迷因的地位，它似乎可以在任何东西上运行。从打印机到智能手表的所有东西都已经在现在标志性的第一级毁灭战士上展示过了。展望未来，[Equalo]开始在土豆上运行厄运。然而，在我们开发出完整的生物计算机之前，他不得不满足于在土豆驱动的设备上运行 Doom。(视频，嵌在下面。)

正如我们之前看到的其他黑客一样，土豆是一种不错的能源，只需要土豆、锌和铜。一些人试图让扩大土豆能量变得更容易，另一些人则专注于让单个土豆变得更强大的 T2。将土豆用作电池的最大障碍是，尽管每个土豆可以输出几乎一伏的电压，但电流却小得可笑。

电流不足是[Equalo]大幅提升典型土豆电池的原因。目标器件 Raspberry Pi Zero 在 4.5V 时需要约 100 mA 电流，这意味着他需要 700 多片马铃薯切片。在煮了数百个土豆后，在朋友和家人的帮助下，这个巨大的土豆电池建成了，我们不禁惊叹于其巨大的规模和胆识。扩大土豆电池的挑战在于，当你连接第 400 个土豆时，你的第一个土豆已经开始腐蚀了。

下次你在为一项巨大的任务寻找灵感时，也许可以看看[Equalo 的]巨型土豆电池的故事，并记住只要有决心和一百磅土豆就能完成什么。
