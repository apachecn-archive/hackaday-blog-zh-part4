# 也许最简单的云室

> 原文：<https://hackaday.com/2022/02/11/maybe-the-simplest-cloud-chamber/>

你见过威尔逊云室吗——一个让你可视化电离辐射的科学实验？造一个有多难？如果你按照[斯托皮的]例子，[一点也不难](https://stoppi-homemade-physics.de/radioaktivitaet/nebelkammer/)(德语，[谷歌翻译链接](https://stoppi--homemade--physics-de.translate.goog/radioaktivitaet/nebelkammer/?_x_tr_sl=de&_x_tr_tl=en&_x_tr_hl=en&_x_tr_pto=wapp))。一个塑料瓶。一些胶带，一个手电筒，一些水，热胶水，还有——唯一奇特的部分——一点镅 241。你可以在下面的视频中看到这个设计，这个页面也有一些更复杂的设计，包括一个使用 CPU 冷却器的设计。即使你不会说德语，视频也会很有帮助。

如果您构建简单版本，您需要调整您的期望，但是它看起来很有效。塑料瓶是必须的，因为你必须挤压它来改变容器中的压力。

如果你想知道哪里可以找到镅-241，可以去看看废弃的烟雾探测器。下面的第二个视频详细介绍了如何从这个共同的资源中获取信息。你需要能装进瓶盖的东西。

在视频上很难看到，但不同类型的电离辐射在室内留下了蛛丝马迹。α粒子倾向于留下粗直的轨迹，而β粒子倾向于更加精细和弯曲。虽然在现代，这些或多或少是科学珍品，但曾经有一段时间它们是真正的实验室设备，它们参与了正电子、μ介子和介子的发现。

如果你想尝试一个更大的房间，我们已经看过了。一些房间使用[酒精和干冰](https://hackaday.com/2011/01/23/your-very-own-cloud-chamber/)。
