# 手鼓桌上有一个令人费解的秘密

> 原文：<https://hackaday.com/2022/11/01/tambour-table-with-a-puzzling-secret/>

有些人真的很喜欢拼图。西蒙·吉尔兹是这些严肃的谜题爱好者之一，[建造了一个变形桌](https://www.youtube.com/watch?v=G5VQUDpK9Iw) (YouTube)，让她可以轻松地在谜题和更平凡的任务之间切换，比如吃饭。

虽然有可拆卸桌面游戏桌的商业解决方案和铰链盖等更简单的解决方案，但[Giertz]决定“让它变得更加复杂和过度设计。”向外滚动的铃鼓桌面已经使它成为一件独一无二的家具，但第二个拼图桌面可以与桌子的两侧齐平，这真的使它达到了一个新的水平。

如果这还不够，桌子上的黄铜把手也是定制的。在 grand maker 的传统中，[Giertz]听从了她内心的 MYOG(制作你自己的 Gnome ),得到了一台车床来学习自己制作把手，而不是仅仅从货架上购买一些。

如果你不太喜欢拼图，你可能想看看[拼图是如何被打败的](https://hackaday.com/2022/08/04/jigsaw-puzzles-are-defeated/)。如果你担心丢失棋子，看看这些 [3D 打印滑动拼图](https://hackaday.com/2021/08/28/never-lose-a-piece-with-3d-printed-sliding-puzzles/)。
