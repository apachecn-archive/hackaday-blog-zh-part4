# 用这一个简单的技巧节流你的固体火箭发动机！

> 原文：<https://hackaday.com/2022/03/18/throttle-your-solid-rocket-motors-with-this-one-simple-trick/>

几十年来，人类满足于将有效载荷发射到轨道上，然后看着数十万小时的血汗和泪水就这样坠入海洋。然后，部分是因为能够节流火箭发动机的巨大进步，我们开始着陆我们的第一级助推器。[乔]在 *BPS.space* YouTube 频道厌倦了观看 SpaceX 公司所有的助推器着陆乐趣，但他还没有完全处于节流液体发动机阶段。所以[在视频下面的休息](https://www.youtube.com/watch?v=8sksiYfkses)他问了这个问题:你能节流*固体*火箭发动机吗？是的。不。算是吧。

通过限制燃料和氧化剂的数量，节流液体火箭发动机实际上与节流任何其他发动机没有什么不同。这本身就具有挑战性，因为这是火箭科学。不过，对于液体火箭发动机来说，这个概念至少很简单。但是模型火箭爱好者只使用极端高端的液体燃料发动机。大多数人转而使用固体燃料火箭，燃料是预先混合的，一点也不变。

这些明显的障碍并没有阻止[乔]的尝试。再次尝试。然后，再一次。再一次为好的措施。然后再重复一次。一路上肯定有一些失败，我们为[乔]甚至承认他不知道如何正确使用电钻而鼓掌。任何年龄的黑客都可能有不知道如何做某事的时候，尽管我们也倾向于不太谈论那部分。

我们不会破坏结局，只是说这个视频绝对值得一看，看看[Joe]如何在不实际节流的情况下从根本上解决了限制固体火箭发动机有效推力的问题，并了解了一个他从未见过的新问题。

当然，你也可以在家里用过多的原料制作火箭发动机，但一定要在*别人的厨房里做！*
