# 低功耗气象站击败竞争对手

> 原文:[https://hack aday . com/2019/05/03/low-power-weather-station-blows-the-competition-away/](https://hackaday.com/2019/05/03/low-power-weather-station-blows-the-competition-away/)

对于任何从事电子项目的人来说，建立一个气象站并不是什么难事。网上有很多计划，如果符合一定的标准，你甚至可以把你的站放在天气地下。不过，这些设备通常可以获得可靠的电源，而且像任何电子项目一样，一旦需要在偏远地区可靠地工作，就会很快面临挑战。[【tegwyn☠twmffat】气象站通过](https://hackaday.io/project/163922-low-power-high-accuracy-weather-station)迎接了这一挑战，目前已经可靠工作了三年。

从任何由不可靠的能源(太阳能、风能等)供电的电路中获得这种可靠性。)而电池是一个相当大的挑战。你不仅需要解决电源管理问题，确保你的应用在冬天能得到足够的阳光，还需要对电路进行一些极低功耗的改造。这个气象站通过使用 LoRa 进行通信来实现所有这些，并且还配备了一个单独的硬件看门狗定时器，如果它断电或由于某种原因挂断，它可以重新启动气象站。

如果你一直在寻找一个气象站建立，这是一个伟大的地方开始。[Tegwyn☠Twmffat]也经历了气象站的组装，完成了一个钢索支撑的平台来安装它。如果你需要更多关于偏远地区节能的想法，还有其他气象站。
