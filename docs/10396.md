# 气球天线不需要塔

> 原文：<https://hackaday.com/2021/06/11/balloon-antenna-doesnt-need-a-tower/>

你用美元树的花线和气球做什么？如果你是【业余无线电速成班】，[你做一个业余无线电天线](https://www.youtube.com/watch?v=EdB5U3L0YwQ)。花线是导电的，用一块作为字面上的天勾，另一块作为地线，应该有所作为。如你所料，他确实使用了一个调谐器来匹配随机的电线长度。

第一次尝试气球太少，风太大。他最终改用了非美元商店的氦气罐。那个气球膨胀到大约 36 英寸，看起来有很大的升力。看起来最后他用了两个。

风带来的一个困难是，这根线一直缠绕在其他天线上。但是如果你真的想使用气球天线，我们猜测你还没有一些大天线。不过，我们确实希望你在院子后面看到的公用事业管线比它们看起来要远。纠结于这些会对你的健康和你的收音机有害。

看起来天线做得很好，尽管使用 PSK31 可以覆盖大量的 sins，因为你不需要大量的信号来与该模式进行接触。

我们见过风筝和气球升起的紧急天线，尤其是用于军事用途。这些[氦气罐](https://hackaday.com/2016/07/14/ham-antenna-rises-to-the-occasion/)也不是第一次出现了。
