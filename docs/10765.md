# 360 度显示器上的新旋转

> 原文：<https://hackaday.com/2021/07/21/a-new-spin-on-360-degree-displays/>

早在 2018 年，[Salah]就用一个品客薯片罐和一个快速马达创造了一个原型展示，似乎违背了逻辑。虽然不是立体的，但这张图片确实显示了 360 度范围内从任何有利位置拍摄的相同的 2D 图像。

纸板怎么能制造出这种效果？有点像 [zoetrope](https://en.wikipedia.org/wiki/Zoetrope) 使用狭缝来创建快门效果，这种显示器使用一个细狭缝来限制图像的视图，一次只能看到一个窄的垂直切片。当移动足够快时，[视觉暂留](https://en.wikipedia.org/wiki/Persistence_of_vision)开始将这些切片组合成一幅完整的图像。我们认为这项技术最酷的地方在于，无论从哪个角度看，同时被多个观众观看，效果都是一样的。

休息后的项目页面和视频演示没有太多细节，尽管这个想法非常简单，不需要额外的解释。我们假设在下面的视频中看到的明亮的 LED 是为了克服通过窄缝观看时图像相对暗淡的外观，并不是严格要求的。
