# 一根触手，那是一件艺术品

> 原文：<https://hackaday.com/2020/09/15/a-tentacle-thats-a-work-of-art/>

当我们在黑客作家面前构建项目时，我们都将自己的专业领域带到了工作中，例如，软件开发人员将聪明的大脑带到了他们的微控制器上，或者电子工程师可能会带来一个设计良好的电路。[伊沃·德·哈斯]是一名机械工程师，从他的电动触须可以很明显地看出，他已经将自己在该领域的专业知识发挥到了极致。

如果你认为它看起来很熟悉，那么一些读者可能会记得[我们在 2020 年 2 月在 Hacker Hotel】见过一个原型模型。在疫情给我们带来锁定和取消之前的最后几周，他收集了一个非常有价值的概念证明，从他的文章和下面的视频中我们可以看到，他利用所有的 COVID 时间在成品中取得了巨大的效果。早在二月份，控制是通过一对操纵杆来实现的，我们特别感兴趣的是他现在使用一个迷你触手作为控制器。](https://hackaday.com/2020/03/06/from-hacker-hotel-2020-badges-sharks-tentacles-old-school-hacking-and-much-more/)

它的核心是由齿轮连接的 3D 打印的反平行四边形，电缆保持张力，并通过一组绞盘控制触手的运动。设计过程从一开始就很详细，读起来很吸引人，随着它的夹子放在最后，我们迫不及待地想看到一个没有取消的事件，在这个事件中我们可以真正看到触手。

如果你想看更多[Yvo]的作品，也许你会记得[他的可穿戴和功能的皮普男孩](https://hackaday.com/2013/07/01/3d-printed-pip-boy-geiger-counter-not-included/)，和[他的工作*门户*炮塔](https://hackaday.com/2019/08/12/locating-targets-with-charm-courtesy-of-a-life-size-portal-turret/)。
