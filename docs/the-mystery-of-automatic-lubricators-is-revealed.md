# 自动润滑器的秘密被揭开

> 原文：<https://hackaday.com/2022/06/30/the-mystery-of-automatic-lubricators-is-revealed/>

工业机器有各种需要定期润滑的运动部件，以防止磨损和损坏。从历史上看，这需要维护人员的定期访问，以保持润滑和光滑。自动润滑器通过定期给机器添加新鲜油脂来消除这一工作，而[大克莱夫] [决定看看是什么让它们滴答作响](https://www.youtube.com/watch?v=Be9RU3PU1bw)。

![](img/a6d26fcce02b49b80eea1f9654edebcf.png)

The device can be set to deliver a full load of grease over a period of 1-12 months.

最简单的模型仅仅使用一个弹簧，随着时间的推移慢慢挤出油脂。更换弹簧会改变润滑脂的分配速度。化学版本也存在。选择一种化学药丸，并将其放入装有液体的容器中，液体会随着时间的推移释放出气体。随着气体的释放，它会产生压力，迫使柱塞向下，随着时间的推移分配油脂。

也许最奇特的型号是电子型号，然而，它的背面有一个用于选择油脂输送速度的刻度盘。转动刻度盘改变连接在两个密封的锌空气电池上的电阻。根据一项专利[大克莱夫]发现，显然，当电流被迫通过这些细胞，并且它们被排除在氧气之外时，这些细胞释放出氢气。然后，这迫使柱塞向下，分配润滑脂。转动转盘会改变阻力，从而改变润滑脂的分配速度。

工业上对节省劳动力的追求产生了多种自动润滑器的设计，所有这些设计都非常简单，并针对用途进行了优化。它显示了一些元件和一些创造性思维可以实现多少功能，其中一个人的第一个冲动可能是用定时器或微控制器来完成这项工作。
