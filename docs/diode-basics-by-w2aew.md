# 二极管基础知识[w2ew]

> 原文：<https://hackaday.com/2019/11/11/diode-basics-by-w2aew/>

我们不认为我们需要[一个二极管](https://www.youtube.com/watch?v=Pp5AAj2aIzc)的基本指南，直到我们看到它来自【w2 aw】，然后我们知道我们会学到一些新东西。名为“从理想到现实的二极管”的 18 分钟视频并没有令人失望，它混合了[笔记](https://www.qsl.net/w2aew/youtube/Diode_Basics.pdf)和带有曲线跟踪器的时间，以了解这些设备的所有信息。

作为一个典型的[w2ew]视频，这不仅仅包括二极管的简单操作。它包括动态电阻、结电容等主题，并讨论各种二极管类型。

事实上，该视频最大的优点之一是，它可以作为他对不同类型的二极管所做的许多其他详细视频的索引。齐纳二极管在视频 289。隧道二极管出现在视频 204 中。还有很多其他的。

二极管用途非常广泛。它们可以作为开关、整流器、检测器和保护器。特殊的二极管类型也各有其独特之处。一个特殊的二极管可以发光(或者甚至吸收黑暗)或者调节电压。它们也是其他更复杂设备模型中常见的构建模块。如果你想了解基本情况，我们之前已经详细研究过[二极管](https://hackaday.com/2016/12/09/diodes-the-switch-you-never-knew-you-had/)。如果你想在[建造自己的](https://hackaday.com/2016/05/01/home-made-diodes-from-copper-oxide/)，找一些铜，看看你的厨房就知道了。
