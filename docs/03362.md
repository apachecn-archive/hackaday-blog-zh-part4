# 咖啡桌、电脑和铁路

> 原文:[https://hack aday . com/2019/06/22/coffee-tables-computers-and-railways/](https://hackaday.com/2019/06/22/coffee-tables-computers-and-railways/)

从 20 世纪 50 年代到 80 年代，如果你是一个英国孩子，你的玩具店很可能有一辆火车。这不是普通的火车模型，而是一个完整的铁路模型，放在一个玻璃盒子里，大约有台球桌那么大，有一个按钮，按下后会启动一个计时器，并启动一个小型坦克引擎，在一个有两个车厢的电路上运行。对于在黑白电视中长大的一代人来说是不可思议的，但对于今天的年轻人来说可能不是这样。现代的玻璃柜布局来自[杰克·弗林]，他创造了一个咖啡桌[，里面有一个自动化和计算机化的 N 轨距铁路布局](https://hackaday.io/project/161540-not-just-another-coffee-table)。而且这绝对是一条*铁路*而不是*铁路*，主要机车是一辆 Brush Type 4，一辆英国铁路 47 级柴油。

这个模型是一件艺术品，在岩石山坡的背景下，在一个椭圆形的双轨上有一个稍微理想化的英国街道场景。在山上有一个意想不到的惊喜，你可以在我们放在裂缝下面的视频上看到，在它下面是电子设备。一个 Teensy 处理轨道切换和板上所有的各种 LED 灯，一个 Sprog DCC 控制器负责列车，监督一切的是一个 Raspberry Pi 运行[一些用 Python 编写的定制软件](https://github.com/JackFlynn90/PiTrainManager)和一个用于控制的 web 界面。如果我们的咖啡桌有这样的布局，我们可能无法抗拒一点遥控铁路的动作！
