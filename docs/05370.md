# 工程过度将日常工作机械化

> 原文：<https://hackaday.com/2020/01/09/engineering-overkill-motorizes-the-daily-grind/>

你不能拽超人的斗篷，也不能夹在工程师和他的咖啡之间。事实上，如果你办公室里所有的咖啡都是制造含咖啡因垃圾的老古董，你不会阻止你的同事追求更好的咖啡。

[乔治]每天都在他的办公桌前像对待科学实验一样对待咖啡:用水壶，Chemex 和手动咖啡研磨机。问题是，这种安排吸引了越来越多的关注，而[乔治]作为办公室咖啡师的无薪兼职开始让他筋疲力尽。他考虑过买一个电动研磨机，只是时间长到足以嘲笑这个想法，然后去车间为他的手动研磨机建造了[这个漂亮的过度设计的电动支架。](https://studiored.com/shop-built-electric-coffee-grinder/)

将这些手磨机机动化或者甚至用电钻驱动它们的唯一麻烦是，它们中的许多都有一个五角形轴。但是这个问题与[乔治]和他档案不相符。他很快就在 100 转/分的马达和研磨机轴之间建立了连接，并着手建造过于合适的框架。

整个事情是华丽的，但我们特别喜欢气弹簧，当他倒豆子时，它支撑着马达。[乔治]也许应该开始一个豆类基金，如果他还没有。这种噪音加上咖啡的味道一定会吸引更多的新加入者。抓住一个新的杯子，磨过去的休息，看看这个婴儿粉碎一些 beanage。

不是每个人都喜欢边喝咖啡边喝，但是在咖啡的各个方面都有一些窍门。[看看【Maya Posch】完全改造浓缩咖啡机的冒险经历](https://hackaday.com/2019/04/26/make-that-special-cup-of-coffee-by-completely-tweaking-the-coffee-machine/)。
