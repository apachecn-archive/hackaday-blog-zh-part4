# 用于机器人移动的 3D 打印全方位球

> 原文：<https://hackaday.com/2021/05/20/3d-printing-omni-balls-for-robot-locomotion/>

轮子对于出行来说很好，但是它们只能绕着一个轴旋转。全向轮是存在的，但对[詹姆斯·布鲁顿]来说还是有点太单调了。他的最新项目包括 3D 打印定制全方位滚动球。(视频，下面嵌入。)

omniball 的概念来自大阪大学的早期工作，该大学也生产了一种名为 OmniCrawler 的坦克车。球形设计，配有轴和脚轮，允许在多个方向上旋转，允许装有这种全向球的平台容易地在所有方向上旋转和平移。

[James]开始创造他自己版本的设计，依靠 grippy TPU 细丝作为抓垫，给 3D 打印半球一些急需的抓地力。内部还有轴承，以允许半球和内部脚轮之间的相对旋转，这是必要的，以允许车轮在半球的任一极上平稳移动。然后使用滑动轴承将三个全向球组装到一个平台上，这证明了球在所有方向上平滑滚动的能力。

虽然现在这只是一个基本想法的演示，但我们可以想象这些球在机器人平台上的巨大作用，该平台需要轻松地在光滑的表面上的狭小空间中导航。然而，omni-balls 的机械复杂性可能会使其在更脏的越野环境中无法有效使用。

我们之前也看到过[James]的工作——[比如他为行走机器人设计的柔顺腿、](https://hackaday.com/2020/03/22/compliant-quadruped-legs-using-servos/)以及上周他的[主动陀螺仪平衡器](https://hackaday.com/2021/05/17/actively-balancing-a-robot-with-a-gyroscope/)。[詹姆斯]什么时候睡觉？
