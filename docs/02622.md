# 带 LED 字幕的街机展示您的游戏

> 原文:[https://hack aday . com/2019/04/04/arcade-with-led-marquee-shows-off-your-game/](https://hackaday.com/2019/04/04/arcade-with-led-marquee-shows-off-your-game/)

我们已经在 Hackaday 上看到了很多街机版本。说真的，很多。更重要的是，自从树莓派占领了世界，让你再也不用拆旧笔记本来造一台了。这是具有巨大吸引力的项目之一:要么你是建造了自己的拱廊的人，要么你是希望拥有自己的拱廊的人。但是，即使在看到所有这些建筑之后，我们偶尔也会遇到一个值得特别认可的标本。

[![](../Images/56eb778fdb596d82670dc4dd063f8839.png)](https://hackaday.com/wp-content/uploads/2019/04/ledarcade_detail.jpg)

LED display controller

[阿尔·克林]最近写信告诉我们他的拱廊建筑，[，我们认为你会同意它值得仔细看看](https://www.instructables.com/id/Vertical-Bartop-Arcade-With-Integrated-PIXEL-LED-D/)。核心构建实际上是对以前发布的设计的修改，但使这一个独特的是在顶部添加了一个可编程的 LED 矩阵，它实际上显示了你当前正在玩的任何游戏的徽标和插图。这种展示真的有助于销售整体外观，并立即使体验更加真实。当然，你不需要*用*你家街机的遮蓬来展示适合那个时代的艺术品……但是我们知道你想要它。

那么，如何将他们的 Raspberry Pi 与这款漂亮的 64×32 LED 字幕显示器连接起来呢？碰巧的是，[Al]在做很酷的 LED 显示屏，甚至在 Kickstarter 上有几次成功的活动来证明这一点。他开发了一种电路板，可以让你轻松连接到低成本的 HUB75 LED 面板，就像街机中使用的那种。距离我们上次看到解决这些特定 led 显示屏的项目已经过去了几年，看到自那以来取得的进展令人鼓舞。

即使你不知何故不喜欢 LED 字幕，这个版本确实是一个桌面街机的绝佳例子。[Al]花了很长时间来记录他的构建，包括在构建的不同阶段将几个视频放在一起。如果你对家庭拱廊建筑艺术的开始感到好奇，这个项目将是一个很好的晴雨表。

无论是[你年轻时站在](https://hackaday.com/2016/10/06/galaga-cabinet-is-out-of-this-world/)前的机器的全尺寸复制品，还是[根据你的严格规格](https://hackaday.com/2015/04/07/scratch-built-bar-top-mame-cabinet/)进行的全新设计，很少有项目比[在你的客厅里眨着眼睛的这些美女中的一个](https://hackaday.com/2018/01/14/bartop-arcade-cabinet-build-skips-the-kit/)更好的话题了。
