# 3D 打印智能眼镜将 Linux 放在你的面前

> 原文:[https://hack aday . com/2021/08/14/3d-printed-smart-glasses-put-Linux-in-your-face/](https://hackaday.com/2021/08/14/3d-printed-smart-glasses-put-linux-in-your-face/)

对由微型控制器驱动的 DIY 可穿戴设备不感兴趣，[【Teemu Laurila】一直在研究 3D 打印头戴式计算机](https://hackaday.io/project/181167-raspberry-pi-zero-w-smart-glasses)，它将一个成熟的 Linux 桌面放在你的视野中。它可能不像谷歌眼镜那样纤薄和符合人体工程学，但它在原始潜力方面远远弥补了它。

这款可穿戴设备配有超频的 Raspberry Pi Zero W、运行频率为 60 Hz 的 ST7789VW 240×240 IPS 显示屏和前置摄像头，是一个非常低成本的增强现实实验平台。[Teemu]已经制作了一个令人印象深刻的手部跟踪演示，可以近乎实时地找出所有十个手指的位置。处理工作必须在他的台式电脑上完成，因为 Zero 还不能胜任这项任务，但正如你在下面的视频中看到的，整个工作进行得很好。

[![](../Images/807a2816ae270947f212b6184a2e72d4.png)](https://hackaday.com/wp-content/uploads/2021/08/diyglass_detail.jpg)

Precision optics, courtesy of a hacksaw

在结构上，头戴式装置由九个 3D 打印部件组成，这些部件夹在一副标准眼镜上。[Teemu]表示，这些部件可能需要调整以适应你的特定框架，但这种设计足够模块化，应该不会花费太多精力。他使用 0.6 毫米的 PETG 塑料作为前反射镜，主镜头是从一副廉价的虚拟现实护目镜上取下的[，并被手工切割成矩形。](https://hackaday.com/2021/03/30/google-calls-it-quits-with-vr-but-cardboard-lives-on/)

构建的发展已经在几个视频中记录下来，看看硬件在相对较短的时间内取得了多大的进步是很有趣的。最初的版本让[Teemu]看起来像是在扮演《星际迷航》中的博格人无人机，但最新的版本似乎更加实用。我们仍然不会尝试在飞机上戴它，但它在黑客大会上也不会显得格格不入。
