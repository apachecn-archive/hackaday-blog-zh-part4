# 科学饮料架

> 原文:[https://hackaday.com/2018/07/17/beverage-holder-of-science/](https://hackaday.com/2018/07/17/beverage-holder-of-science/)

[K&J Magnetics]的人可以使用精确的磁力计，这是多年经验积累的丰富知识，但当涉及到像[magnetic koo zie](https://www.kjmagnetics.com/blog.asp?p=koozie)这样的愚蠢项目时，他们会直接进行试错，而不是模拟和计算。当然，这与关键任务相反。

一旦实验结束，他们就开始解释他们的结果，这样我们就能学到更多的东西，而不仅仅是如何把啤酒放在工具箱边上。他们清楚地描述了与磁保持相关的三个因素，这三个因素是这个实验的核心。第一，磁体和表面之间的任何东西都应该很薄。第二个因素是，它应该是抓地力，而不是滑。最后一个要素是考虑被悬挂的饮料的杠杆作用。快速说三遍。

从帮助[可视化气体原子](https://hackaday.com/2017/08/15/a-gas-model-made-of-magnets/)、[机械师的工具](https://hackaday.com/2016/11/02/magnets-for-a-machinist/)到[绕过枪支安全特征](https://hackaday.com/2017/07/25/smart-gun-beaten-by-dumb-magnets/)，磁铁在任何事情上都是如此酷。
