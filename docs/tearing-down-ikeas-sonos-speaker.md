# 拆除宜家的 Sonos 扬声器

> 原文：<https://hackaday.com/2019/10/11/tearing-down-ikeas-sonos-speaker/>

了解一件电子产品最好的方法莫过于拆卸它。在引擎盖下偷窥可以揭示一个设备的设计、制造和起源的所有方式。[这不算]在这方面做得很好，[挖掘宜家的 *Symfonisk* 扬声器。](http://youtube.com/watch?v=ZB413S8KDmo)

Symfonisk 是一款支持 WiFi 的扬声器，与 Sonos 生态系统合作。拆开这款设备可以发现它与宜家早期的 *Eneby* 扬声器有一些相似之处，两款设备共享相似的扬声器驱动程序，显然来自 GGEC。然而，深入挖掘后，发现 *Symfonisk* 与另一个制造商的扬声器有更多的共同点。

该视频不仅很好地调查了该设备的制造来源，还剖析了它的工作方式。这显示了扬声器如何依赖于 Atheros WiFi 专用芯片组，从而解释了蓝牙功能的缺乏，并讨论了诸如电缆管理的简洁解决方案等问题。有趣的是，该扬声器使用双通道 DAC 和 D 类放大器，但只能在单声道下工作。相反，这两个通道用于分别驱动高音扬声器和低音扬声器，允许 EQ 在主 CPU 上的软件中完成，无需模拟分频电子设备。

这是一次拆解，对于任何考虑制造消费电子产品的人来说，都是一个很好的入门，尤其是那些涉及高保真领域的人。要想知道很久以前是怎么做的，[也许可以试试这个 8 轨拆卸来代替](https://hackaday.com/2017/04/12/retro-teardown-inside-an-8-track-stereo-player/)。休息后的视频。
