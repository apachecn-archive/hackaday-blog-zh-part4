# Pi 动力 1:35 比例豹式坦克

> 原文：<https://hackaday.com/2022/01/09/powering-up-a-135-scale-panther-tank/>

坦克迷[丹尼尔·扎勒加]多年来一直喜欢在数字领域摆弄装甲战车，但直到最近才意识到他拥有将自己的热情带入现实世界所需的技术和技能。尽管规模略有缩小。所以他从田宫买了一套 1:35 的德国二战黑豹坦克塑料模型，然后[开始研究让它移动的方法](https://github.com/dbeef/tank)。

对丹尼尔来说幸运的是，组装好的模型本质上是中空的。这给了他足够的空间来安装齿轮驱动电机、电池、电机控制器、电压调节器、转台的伺服系统和控制整个节目的树莓 Pi Zero。那些厌恶热胶的人最好不要太靠近这里的建筑，但它完成了工作。另外，这只小黑豹又不会去看什么前线战斗。

[![](img/900c195822d71ebbc66c98fac64c17f1.png)](https://hackaday.com/wp-content/uploads/2022/01/panther_detail.jpg) 模型套件的另一个非常适合机动化的因素是它有真正的橡胶胎面。这意味着[丹尼尔]只需在油箱侧面钻几个洞，并想出如何将驱动链轮安装到他的齿轮马达上。不幸的是，在这个模型上看起来车轮是静止的，这意味着胎面必须被拖过它们。这肯定会降低坦克的动力和速度，但在休息后的视频中，你可以看到它的运动仍然相当逼真。

为了控制坦克，他将手机浏览器指向树莓派上运行的一个简单页面。只需在屏幕上拖动一个手指，就可以操作坦克的两个独立踏板，转动炮塔。[丹尼尔]说他最初的计划更加详细，网页显示了来自机载摄像机的实时视频以及来自各种传感器的读数。但至少现在，事情尽可能简单明了。

这肯定不是我们在 Hackaday 看到的第一辆加大马力的玩具坦克。从华丽的蒸汽动力机器到这款带有激光辅助瞄准系统的[虎式坦克](https://hackaday.com/2013/06/11/giving-an-rc-tank-a-fire-control-computer/)，这些小型履带式平台一直是硬件黑客的最爱。
