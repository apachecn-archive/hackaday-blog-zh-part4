# 闹鬼的电视用树莓派来反映恐惧

> 原文：<https://hackaday.com/2020/11/13/haunted-tv-does-mirror-scares-with-raspberry-pi/>

万圣节可能又要过去一年了，但这只是意味着你有更多的时间为下一次做准备。[gocivici]对经典的镜子恐慌进行了有趣的改变，这可能正合你的胃口。

建造从一台旧的黑白电视开始，连接到一个树莓 Pi 3。Pi 通过隐藏在屏幕耳机插孔中的摄像机拍摄电视前的场景，并在屏幕上显示出来。摄像头馈送通过 OpenCV 运行，OpenCV 运行面部和眼睛检测算法来确定一个人何时在看屏幕。基于一个基本的计时器脚本，当一个观众看了足够长的时间，一个幽灵般的幽灵就会出现，潜伏在观众的身后。当使用者回头看时，幽灵很快消失，就像经典的恐怖比喻一样。

这是一个有趣的建筑，会成为你下次万圣节派对的绝佳布景。为了获得额外的效果，一定要把它藏在黑暗的走廊里，用一些红外发光二极管照亮摄像机的场景。如果你更喜欢有点奇思妙想的东西，可以考虑这些会唱歌的动画南瓜，而不是 T1。休息后的视频。
