# 你想往哪个方向拧紧这个螺栓都行

> 原文：<https://hackaday.com/2021/01/04/tighten-this-bolt-in-any-direction-you-want/>

金属车床是有能力的机器，在工业革命中发挥了很大的作用，是一个不可思议的工具。但这并不意味着它们不能用来找乐子，正如[【Oleg Pevtsov】所展示的那样，他制作了一个双向螺栓作为加工练习](https://www.youtube.com/watch?v=cDfMI5ahbJI)，只是因为他可以这样做。

休息之后的两个视频都是俄语的，但是视频和自动生成的字幕足以表达要点。螺栓尺寸为 M42，间距为 40 mm，在两个方向上都开有凹槽，以便旋入左旋和右旋螺母。大螺距意味着需要为每个螺纹方向切割十个单独的凹槽来覆盖螺栓表面，而不是像普通螺栓那样只有一个连续的凹槽。因为这些都是在手工车床上加工的，所以需要一个千分表来保持精确的间距。(奥列格)花了四次艰苦的尝试才把它做好，但最终结果看起来非常好。他没有使用固定刀具，而是使用了安装在定制夹具上的修整刳刨机。

[Oleg]还加工了三种不同的黄铜螺母，用固定刀具安装在螺栓上。首先制造了左旋和右旋螺纹螺母，接着是双向螺母。由于螺距较大且经过精心加工，所有三个螺母都将在重力作用下向下旋转螺栓。虽然双向螺母不像其他两个那样平滑移动，但它可以随机改变旋转和平移方向。

虽然这是一个独一无二的小动作玩具，但我们的读者有谁在野外见过双向螺栓或丝杠吗？我们可以想象，在单个丝杠上向相反方向移动两个螺母的能力可能有一些实际应用。

在手动车床上制造令人难以置信的零件是可能的。一台[手工制造的 V10 发动机](https://hackaday.com/2020/09/20/1-3-scale-hybrid-rc-car-with-a-scratch-built-125cc-v10-engine/)和一架[气动六角直升机](https://hackaday.com/2020/08/10/true-craftsmanship-pneumatic-powered-drone-wasnt-made-to-fly/)只是两个例子，说明了在足够的技能、知识和耐心下什么是可能的。可悲的是，这是一种逐渐消失的工艺形式，在业余爱好项目之外，大多被数控机床淘汰。
