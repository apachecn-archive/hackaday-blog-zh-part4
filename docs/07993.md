# 令人印象深刻的模块化模具箱

> 原文：<https://hackaday.com/2020/10/07/an-impressive-modular-mold-box/>

多产的制造商和产品设计师[Eric Strebel]拥有多年制造可重复使用的硅树脂和树脂铸模箱的经验。他以前总是使用 3/4”胶合板，但它带来了一些问题，如不准确，螺钉最终会滑出，根本没有扩展的空间。现在[Eric]决定设计一个非常棒的模块化模具箱系统,它甚至可以堆叠。休息之后，请观看视频中的设计和构建过程。

[Eric]利用添加制造技术，制作了带有凹位的别致梯形墙壁，允许这种模块化系统所依赖的魔力——少量 M6 内六角螺钉和用于张紧的匹配螺母。一旦打印好了，[埃里克]就把坚果钉在墙上，每隔 10 毫米做一个填充线。像往常一样，[Eric]的视频附带了额外的知识，比如他使用一个简单的刮卡器来清理指纹，平滑侧面，并倒角所有边缘。

如果你想塑造混凝土和石膏之类的东西，[你最好使用柔性细丝](https://hackaday.com/2019/01/27/casting-concrete-with-3d-printed-molds/)。
