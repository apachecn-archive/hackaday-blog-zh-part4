# 你实验室的圣诞树

> 原文:[https://hack aday . com/2018/12/06/a-圣诞树-为你的实验室/](https://hackaday.com/2018/12/06/a-christmas-tree-for-your-lab/)

似乎节日装饰品一年比一年来得早。你的实验室可能没有足够的空间来放置一棵成熟的树，但是如果你有一个任意的波形发生器和一个示波器，Tektronix 有办法让你用电子方式展示你的灵魂。

你可以看下面的视频。当然，它的特点是 Tektronix gear，但我们非常确定您可以让它与任何至少有两个通道和一个 XY 模式示波器的任意波形发生器一起工作。

让这一切变得简单的关键不是泰克应用程序。就是 [WebPlotDigitizer](https://automeris.io/WebPlotDigitizer/) ，一个帮助你加载数据并从中提取 X 和 Y 坐标的网页。这实际上是一个非常有用的工具，可以用来提取你可能需要的图像或绘图数据，所以这是一个非常方便的工具。

一旦有了坐标，就必须将 X 值分成一个信号，将 Y 值分成另一个通道。很难想象它会做出你定义的形状，直到你切换到 XY 模式，它就在那里。

当然，他们还不满足，还用了第三个频道来添加一些花环。我们已经看到了大量的[函数生成器](https://hackaday.com/2014/08/09/an-open-source-1mhz-abritrary-waveform-generator-with-an-awesome-ui/),你可以将它们压入这种服务中。我们甚至[推出了自己的](https://hackaday.com/2015/09/15/how-to-build-a-pocket-sized-mbed-signal-generator/)。
