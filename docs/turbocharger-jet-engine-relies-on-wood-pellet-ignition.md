# 涡轮增压喷气发动机依靠木质颗粒点火

> 原文：<https://hackaday.com/2021/12/22/turbocharger-jet-engine-relies-on-wood-pellet-ignition/>

汽车上使用的涡轮增压器与喷气式发动机有一些相似之处。从根本上说，两者都包含一个从热气体中获取能量的涡轮机，用它来旋转压缩机，压缩机吸入新鲜空气进行燃烧。因此，将涡轮增压器改造成喷气发动机是完全可能的，[和【HRom】决定尝试一下。](https://www.youtube.com/watch?v=Pe0arGcacY8)

这款车从一个涡轮开始，它似乎被用于大众集团的柴油发动机上。第一步是从涡轮壳体上切下整体排气歧管。然后增加一个燃烧室，从压缩机壳体吸入新鲜空气，并将热的燃烧产物输送到涡轮入口。自制的喷气发动机燃烧丙烷作为燃料，通过喷嘴引入燃烧室。

最初的试验失败了，因为燃烧发生在涡轮排气处，而不是燃烧室内，这可能是由于燃烧室内缺乏合适的点火源。重新设计使用了一个更大的由灭火器建造的燃烧室，里面插入了燃烧的木球以使注入的丙烷燃烧。

重新设计起作用了，涡轮增压器喷气发动机以越来越快的速度转动时发出雷鸣般的尖叫。然而，由于没有加油系统或任何控制发动机中空气或燃料流动的方法，它最终在一股巨大的烟雾中停止。不管怎样，即使点火方法还不成熟，发动机也能持续运转。

在之前，我们已经看到过类似的建造，这种基本的构造意味着它们远不是飞行重量级的发动机。然而，它们非常酷，是学习喷气发动机工作原理的好方法。休息后的视频。
