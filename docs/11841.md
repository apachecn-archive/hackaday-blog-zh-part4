# 磁锁思想的发展显示了巨大的进步

> 原文：<https://hackaday.com/2021/11/09/development-of-magnetic-locking-idea-shows-great-progress/>

不管怎么做，不管有多假，磁悬浮看起来就是很酷。我们不知道你，但只要走过最俗气的小玩意店，漂浮和旋转物体的展示总能吸引我们的眼球。抛开超导体不谈，这些设备几乎都以同样的方式工作；一个带有永久稀土磁体的物体被固定在一对电磁铁之间的稳定位置，一个在上面，一个在下面，用一些控制电子设备来调整磁场强度并闭合回路。

但是，可能有另一种方式，尽管是一种相当特殊的情况，磁铁不仅可以悬浮，还可以使用旋转磁场锁定到位[。该视频演示了如何利用磁铁的质量对旋转磁场进行锁相。本质上，磁体将希望旋转以与旋转磁场对齐，但其质量将意味着力的作用和旋转的发生存在时间延迟，这将滞后于旋转磁场，如果相位正好如此，旋转将被取消，磁体将被锁定在稳定的位置。实质上，可以利用磁体的惯性来抵消磁体快速旋转的趋势，以在磁场中找到稳定的位置。](https://www.youtube.com/watch?v=V5FyFvgxUhE)

虽然这个想法并不新鲜，但土耳其的实验者[ Hamdi Ucar ]已经研究这个课题有一段时间了(查看他的 [YouTube 频道](https://www.youtube.com/user/sudanamaru/videos)上的许多内容)，甚至在[发表了一篇关于这个课题的非常详细的学术论文](https://www.mdpi.com/2073-8994/13/3/442)。为了简洁起见，我们在这里的解释试图简化这个主题，但是因为这篇论文有很多血淋淋的细节给你们当中的物理学家，如果你能处理数学，你可以得出你自己的结论。
