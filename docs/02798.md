# 使用 3D 打印机对纺织品进行静电纺丝

> 原文:[https://hack aday . com/2019/04/27/use-a-3d-printer-to-electro spin-textiles/](https://hackaday.com/2019/04/27/use-a-3d-printer-to-electrospin-textiles/)

我们都习惯于桌面 3D 打印机，这些打印机将熔化的塑料挤压成层来构建成品。卡耐基梅隆大学人机交互研究所的两位研究人员[迈克尔·里维拉]和[斯科特·哈德森]为他们的打印机增加了另一项功能:[纺织品静电纺丝](https://mikeriv.com/research/desktop-electrospinning/)。

静电纺丝是一种技术，其中挤出的材料通过静电荷从挤出机中加速形成极细的纤维。通过在挤出机和床之间施加数千伏的电荷，他们可以制造出一种纤维，并从高处将其铺成网状，从而制造出类似毛毡的织物。同一台挤出机也可以生产传统的固体印花，允许生产复合织物和固体物品。他们展示了各种各样的印刷品，包括一个折叠式手机支架，一个编织灯，一个互动的羊毛绵羊，这些和其他东西一起可以在休息时间下面的视频中看到。

整篇论文可以下载成 PDF 格式，阅读起来非常有趣。所涉及的电压意味着你的 Prusa 克隆可能不会很快具备这种能力，但我们期待桌面静电纺丝成为负担得起的 3D 打印机的一项功能。
