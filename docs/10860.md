# 液氮不适合蒸汽机

> 原文：<https://hackaday.com/2021/07/28/liquid-nitrogen-isnt-suitable-for-steam-engines/>

液氮是一种有趣的玩物，只要你小心，避免冻掉自己的手指并在工作台上打碎它们。当液态氮在-196 摄氏度左右变成气态氮时，[行动实验室]认为它可以在室温下用来推动一台简单的蒸汽机。测试得出了有趣的结果。

接受测试的设备是英雄的引擎，也就是众所周知的 aeolipile。它由一个装满水的空心球组成，装有一系列喷嘴，当容器受热时会喷出蒸汽。通过火箭原理，这导致设备绕其轴旋转。

当装满水并用蜡烛加热时，aeolipile 以每分钟 2520 转的速度旋转。[行动实验室]接下来测试了它在真空室内装满水，低压使水在室温下沸腾。然而，随着发动机以更慢的速度旋转，效果就不那么令人印象深刻了。

最好的结果是发动机内有液氮。随着氮气在室温下迅速沸腾，气溶胶迅速旋转到很高的速度。在发动机顶部的密封因过压而爆炸之前，发动机支架必须保持稳定，以免翻倒。

我们很想看到同样的实验也在活塞式蒸汽机上进行。休息后的视频。
