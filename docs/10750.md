# 为最新的黑客奖挑战重新想象支持技术

> 原文:[https://hackaday . com/2021/07/19/re imagine-supported-tech-for-the-new-the-new-hackaday-prize-challenge/](https://hackaday.com/2021/07/19/reimagine-supportive-tech-for-the-newest-hackaday-prize-challenge/)

从现在开始，2021 年黑客日奖挑战你[重新想象支持性技术](https://prize.supplyframe.com/#section-challenges)。非常坦率地说，这都是关于成功的捷径。我们能让人们更容易了解科学和技术吗？我们能打破一些阻碍人们将 DIY 作为一种爱好(或生活方式)的障碍吗？我们能做些什么来利用彼此的经验和技能？

[![](../Images/716c70d3940473f3409fe5bda739130c.png)](https://hackaday.com/wp-content/uploads/2021/07/m_hehr_portable-workbench-mini-lab.jpg) 举个例子，要建造自己的电子产品，你需要一个巨大的专用电子实验室，对吗？当然这是无稽之谈，但我们之所以知道这一点，是因为我们已经深入到焊台和真空镊子中。在外人看来，这就像一座无法攀登的山。如果我告诉你，你可以在任何桌子上建立电力，并在黑客会议之间轻松存储所有东西，会怎么样？这听起来像是[M.Hehr 的]便携式工作台&迷你实验室项目的工作。这里有一个蓝图，可以让初学者从零开始，一边抽烟，一边享受过程中的乐趣。

把复杂的话题分解成我们这些新手能接受的东西怎么样？无线电信号就在我们周围，但是进入 SDR 的障碍又是多种多样的。一个很好的支持技术将是一个展示简单硬件的项目，并与已经准备好的开源软件工具链共享一个虚拟机。一个初学者可以拿起这样的东西，在几个小时内就能听到从身边飞过的飞机的应答器。

如果你正在读这篇文章，很可能你已经花了无数快乐的时间学习如何做一些困难而有趣的事情。分享财富！采用现有的硬件概念，并使其模块化和易于使用。改进现有设计，使其更容易为各种行动不便的用户所接受。或者组织一个初学者友好的项目来推进 STEM 教育。
