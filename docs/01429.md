# 一个 Arduino 包在一个有机发光二极管包在一个英格玛怀表

> 原文:[https://hack aday . com/2018/12/03/an-arduino-wrapped-in-an-OLED-wrapped-inside-an-enigma-pocket-watch/](https://hackaday.com/2018/12/03/an-arduino-wrapped-in-an-oled-wrapped-inside-an-enigma-pocket-watch/)

一只怀表，塞在马甲口袋里，拖着一条长长的链子，是一件复古时髦的配饰。一只去掉了机械内部零件、升级为智能设备的怀表可能是一件令人厌恶的钟表，但仍不失为一件很酷的事情。由怀表改装成的数字密码机独树一帜。

[Simon]承认他对怀表情有独钟，收藏了相当多的旧表和不太旧的表，有些甚至可以日常携带。问题是，它们最终会坏掉，合格的制表师越来越难找到。所以改装废弃的手表已经成了他的爱好，这个例子太棒了。它使用一个运行在 Arduino 上的 Enigma 模拟器，类似于几年前他塞进一个[有点过大的手表](https://hackaday.com/2015/03/23/enigma-machine-wristwatch/)中的模拟器。将它装入怀表外壳需要一些技巧，包括一个 0.5 毫米厚的主 PCB，它可以稍微弯曲以适应外壳的轮廓。一个小有机发光二极管屏幕透过前挡板偷看，这是一个有吸引力的黑色褶皱完成，黄铜按钮一个很好的复古外观。封面上甚至有一个酸蚀的黄铜徽章，上面有他的特殊标志，以及最初的英格玛转子的轮廓。

非常令人印象深刻的做工，我们甚至不在乎它不显示时间。需要一点原始谜的背景知识吗？不久前,[Steve Dufresne]在基础知识方面做得很好。
