# 原始的辛克莱·ZX 光谱出现在橱柜里

> 原文：<https://hackaday.com/2019/03/04/the-primordial-sinclair-zx-spectrum-emerges-from-the-cupboard/>

位于英国剑桥的计算历史中心收到了许多捐赠，这些捐赠可以丰富他们的收藏和博物馆展示。许多都很有趣但很平凡，但他们最新视频的主题远非如此。他们从折叠的防静电垫下炫耀地展示的缠绕着电线的原型板不是普通的电脑，因为它是辛克莱·ZX 光谱的原型。

它来自当地的咨询公司九块瓷砖，该公司在 20 世纪 80 年代早期与辛克莱研究公司签约，生产基本的只读存储器，用于替代他们流行的 ZX81 家用微型计算机。我们放在 break 下面的文章和视频详细介绍了 rom 项目的历史，来自 Sinclair 传奇般的成本削减的压力，以及发布未完成 ROM 版本的决定，这意味着后来的外设必须携带更新例程的影子 ROM。

该板本身是一个标准的绕线原型板，所有主要的频谱元件都以某种形式存在。这是一个 16k 的型号，没有扩展连接器，布局和最终机器是前后颠倒的。ULA 芯片是陶瓷封装的预生产产品，键盘通过 D 连接器连接。质量不错的按键开关与橡胶按键和薄膜形成了鲜明的对比，频谱所有者后来在播放戴利·汤普森的十项全能比赛时会将橡胶按键和薄膜捣碎。

这台机器是一个非凡的人工制品，我们都应该感谢九块瓷砖，确保它被保留下来，供那些对计算感兴趣的人研究和欣赏。这看起来可能没什么，但 protoboard 在启动大量技术人员的职业生涯方面发挥了作用，我们怀疑其中一些人会成为黑客读者。下次我们在剑桥时，一定会顺便去看它。

如果你还没有去过计算历史中心，我们建议你看一下我们几年前的评论。如果原型家用电脑是你的东西，[这肯定不是第一次美化这些页面](https://hackaday.com/2015/02/15/hackaday-retro-edition-the-second-most-valuable-home-computer/)。
