# 用于软细丝的更好的鲍登驱动器

> 原文:[https://hack aday . com/2019/05/16/a-better-bow den-drive-for-floppy-filaments/](https://hackaday.com/2019/05/16/a-better-bowden-drive-for-floppy-filaments/)

你可能不会想到用“刚性”这个词来描述大多数 3D 打印机细丝，但大多数塑料细丝实际上在很短的长度内非常坚硬，坚硬到足以被推入挤出机。不过，尝试用 TPE 等更软的塑料做同样的事情，你可能会发现自己在看[这种用于弹性细丝的改进型鲍登驱动器](https://www.thingiverse.com/thing:1752766)。

一些 3D 打印机设计者青睐的鲍登驱动背后的想法很简单:将细丝夹在电机驱动的轮子和惰轮之间，将它推上管道，进入挤出机的热端。但对于 TPE 和类似的弹性细丝，[Tech2C]发现他的 Hypercube 打印机上的鲍登驱动导致打印成品出现卡纸和挤压不足的现象。对原料驱动的仔细分析显示了一些弱点，例如有多少细丝没有被支撑在轮的输出侧。[Tech2C]修改了驱动器，以缩小该间隙，并将输出管开口移近驱动器。库存驱动轮也被更小直径的滚花轮取代。新驱动器固定在步进器上，结果显著改善——新驱动器上的 TPE 花瓶几乎完美无瑕，而旧驱动器上有大量斑点和伪像。和收回测试打印显示没有与解放军串线，这意味着新的驱动器不只是好的软东西。

总的来说，这是一个伟大的多功能和可黑客的小型打印机升级。当然，我们以前见过超立方体—[这种床高探针使用 SMD 电阻作为应变计](https://hackaday.com/2019/01/27/quartet-of-smd-resistors-used-to-sense-z-axis-height/)连接到波顿驱动的另一端。
