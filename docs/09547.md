# 用遥控装置制造的喷气式卡丁车

> 原文：<https://hackaday.com/2021/03/16/jet-powered-go-kart-built-with-rc-gear/>

尽管在 20 世纪进行了许多试验，涡轮机汽车从未真正出现过。尽管相对于它们的尺寸来说，它们的功率输出很高，但它们不太适合陆地运输应用；甚至 M1A1 坦克也因其涡轮动力装置而备受指责。尽管如此，这并没有阻止在卡丁车后面扔喷气式飞机的[扭曲的观念]，而且看起来很有趣。(视频，下面嵌入。)

建造从一辆普通的小型赛车开始，活塞发动机和所有相关的传动装置都被匆忙拆除。钢筋混凝土大小的涡轮喷气发动机安装在一个优雅的铝制支架上，整齐地焊接在汽车的后部。它与电子控制器相连，油门由遥控发射器控制。试图单手驾驶而另一只手握驾驶杆并不理想，但这是零件不能及时到达时所做出的牺牲。

早期测试揭示了颠簸时空气进入燃油管路的问题，但总体性能令人印象深刻。未来的计划包括我们迫不及待想看到的极速跑。当然，如果对你来说还不够离谱的话，可以考虑[柯林·福尔泽的]pulse jet build。
