# 黑客日超级会议:一位模拟工程师投身 RF

> 原文：<https://hackaday.com/2019/12/09/hackaday-superconference-an-analog-engineer-dives-into-rf/>

我们这些从事电子工作的人通常会通过我们掌握的特定途径来接触艺术，同时从周围的人那里吸取我们需要的东西。例如，随着项目变得更加复杂，对音频电路的兴趣可能会扩展到 DSP 和微控制器。一些领域仍然保留着一种不可能的气氛，一种黑暗艺术的名声，对许多人来说其中最主要的是无线电频率。无线电电路往往简单得令人惊讶，但这种简单性掩盖了大量的复杂性，因为这种介质不会以相对静态的模拟电压或一组低频逻辑电平的有序方式运行。

Chris Gammell 对许多 Hackaday 读者来说是一张熟悉的面孔，因为他掌握了许多电子诡计，所以当发现 RF 一直是他知识中的空白之一时，有点令人惊讶。在他在 Hackaday Superconference 的演讲中，他带我们经历了他进入 RF 工作的旅程，其结果是任何对无线电电路有好奇心但不知道从哪里开始的人必看的。

对于那些通过业余无线电爱好者介绍电子学的读者来说，这次演讲将遵循一条熟悉的道路，因为它在一个简单的层次上处理信号如何在 RF 频率下通过电路的非常基本的问题。我们了解匹配和增益，最重要的是了解级间损耗以及精心设计如何将损耗降至最低。他指出，集成电路的进步已经使无线功能进入了几乎任何设计师的能力范围，值得注意的是，这种情况加上大多数地区对频谱使用的更宽松的方法，意味着越来越高的频率已经成为物理设计变得越来越重要的标准。

讲座继续讲述他在测试设备、PCB 布局以及高频常见元件缺点方面的发现。这是一个我们在之前超级会议的报告中提到过的主题，正如射频设计大师迈克尔·奥斯曼给我们的建议一样。

如果你是一个实验者，这不应该是一个害怕的领域。如此多的设备和模块的低成本意味着在不倾家荡产的情况下有很大的尝试空间。在更简单的层面上，也可以通过更传统的射频获得一些经验，例如，尝试试验电视天线前置放大器，70 厘米业余波段的东西，甚至是用废弃的超高频电视调谐器进行试验。对于粗心的射频实验者来说，可能有很多陷阱，但仍有很多学习经验的空间。
