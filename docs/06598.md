# 键盘开关真的是一个变压器

> 原文：<https://hackaday.com/2020/05/17/keyboard-switch-is-really-a-transformer/>

我们不知道为什么[TubeTime]决定以一系列 Twitter 帖子的形式展示这个[古怪的键盘开关](https://twitter.com/TubeTimeUS/status/1260688848104771586)，但我们很高兴在某个地方看到它们。起初，这个开关看起来很传统。但是当图片揭示内部时，你会注意到一些不寻常的东西:一个铁氧体圆环！这些开关作为变压器工作，被称为电磁阀开关。

这些开关有两组两个引脚，一组用于缠绕在铁氧体磁芯上的变压器的初级线圈，另一组用于次级线圈。那个变压器保持静止，但是一对永久磁铁移动。当琴键向上时，磁铁靠近铁芯，导致变压器饱和，因此次级线圈几乎没有输出。当您按下该键时，磁铁从磁芯移开，允许信号通过变压器。这意味着没有机械接触，有利于开关寿命。在小火花可能导致爆炸的环境中，这一点也很重要。你可以在下面观看一个关于使用这些开关的键盘的视频。

有时被称为 Cortron 开关，它们在 1974 年作为专利出现。虽然它们确实在相当多的产品中得到了应用——尤其是那些不需要机械开关的产品，但是霍尔效应开关的生产变得更加经济，并在 2005 年将磁性阀门开关逐出了市场。

令人惊讶的是，这是那些互联网搜索只出现少数页面的罕见话题之一。在一个[键盘](https://deskthority.net/wiki/ITW_magnetic_valve)网站上有一个关于他们历史的简短介绍(那里也有一个关于接口的有趣的[讨论)。我们还在这一页](https://deskthority.net/viewtopic.php?t=22219)找到了一些关于他们在哪里出现[的信息。这些，加上一些零散的讨论帖子和专利，似乎就是关于它的。](http://telcontar.net/KBK/ITW/)

奇怪的是，这并不是我们第一次在键盘中看到[变形金刚，尽管并没有以同样的方式使用。如果你的变形金刚理论生疏了，我们有](https://hackaday.com/2019/06/03/tearing-apart-pulse-transformer-switches/)[为你准备的视频](https://hackaday.com/2018/03/27/everything-you-wanted-to-know-about-transformers-but-were-afraid-to-ask/)。
