# 按键式 MIDI 电话和声码器覆盖蠢朋克

> 原文:[https://hackaday . com/2022/09/06/touch-tone-midi-phone-and-vocoder-covers-daft-punk/](https://hackaday.com/2022/09/06/touch-tone-midi-phone-and-vocoder-covers-daft-punk/)

[poprhythm]的[按键式 MIDI 电话](https://www.instructables.com/Touch-Tone-MIDI-Phone/)是一个将旧的按键式电话转换成完整的 MIDI 乐器的奇妙转换，带有完整的麦克风，但这个项目不仅仅是炫耀结果。[poprhythm]详细描述了他如何与键盘交互，他如何使用 Arduino 创建一个工作的 MIDI 界面，以及他如何决定(从音乐角度来说)每个按钮应该做什么。手机上的 led 甚至可以根据正在发生的事情快乐地闪烁，这是一种很好的触摸。

[![](../Images/b78c5b6db8789cbbd87bf5281bd7c755.png)](https://hackaday.com/wp-content/uploads/2022/09/FUL6DDGL7BWZ1DX.jpg) 当然，事情并没有就此结束。[poprhythm]还利用了手机听筒中的麦克风。由于手机现在是一种具有麦克风和音符输入的 MIDI 乐器，因此可以将它们一起用作声码器软件的输入，他通过覆盖全球*的[和蠢朋克](https://www.youtube.com/watch?v=tBrKK4lQfPI)* 的进行了演示(视频)。

我们喜欢[poprhythm]解释他如何与一切事物交互的方式，因为硬件工作都是关于这样的细节，以及找到正确的资源。这里是 Arduino 代码的 GitHub 库和一些其他资源的链接。

我们以前见过 MIDI 电话项目，每个项目都有自己独特的方式:[这里有一种不同的方法将键盘电话转换为 MIDI](https://hackaday.com/2021/06/09/old-phone-becomes-midi-controller/) ，而[这种旋转脉冲拨号电话](https://hackaday.com/2022/03/15/rotary-phone-midi-controller-still-makes-calls/)走的是完全不同的方向，电话本身完全没有修改，只使用外部接口。

您可以在下面嵌入的短视频中欣赏[poprhythm]的按键式 MIDI 手机，每个视频都展示了该产品的不同方面。很棒的作品！
