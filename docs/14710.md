# 纳米粒子从水中夺取氢

> 原文:[https://hack aday . com/2022/09/05/nanopols-rip-hydrogen-from-water/](https://hackaday.com/2022/09/05/nanoparticles-rip-hydrogen-from-water/)

氢燃料很有前途，虽然空气和水中有大量的氢，但问题是如何提取。研究人员已经开发出一种方法，在没有额外能量输入的情况下，使用铝纳米粒子从水中提取氢。然而，它确实需要镓来进行反应。这种反应并非未知(见下面的视频)，但新的研究有一些有趣的转折。

当然，铝既便宜又丰富。镓，没有那么多，但该过程允许回收和再利用镓，所以它更具成本效益。该工艺正在申请专利，当然，真正的诀窍是制造铝纳米颗粒。但是如果你有，这是一个简单的方法，从水中提取氢，不需要额外的能量，在室温下。因为生成氧化铝和释放氢与镓的反应是众所周知的，所以这里真正的研究似乎是确定铝的最佳性质和铝与镓的比例。

虽然镓在典型的黑客工作室中并不常见——除非你把半导体中的东西算在内——但它并不昂贵，而且相对容易处理。但是氢气并不多，所以如果你决定用这种方法生产氢气，要小心了！

我们已经看到了镓[机器人](https://hackaday.com/2022/05/29/better-robots-through-gallium/)甚至是[天线](https://hackaday.com/2015/08/31/liquid-metal-changes-shape-to-tune-antenna/)。因此，如果你真的得到了一些液态金属，有大量的实验可以尝试。
