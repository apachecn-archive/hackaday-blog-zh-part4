# 用树脂复制乙烯基唱片

> 原文：<https://hackaday.com/2022/11/01/reproducing-vinyl-records-in-resin/>

虽然大多数只是普通的，但黑胶唱片可以有各种各样的颜色、形状，有些甚至在黑暗中发光。[Evan 和 Katelyn]决定用明亮的夜光树脂复制这张普通的旧唱片。

通过首先制作乙烯基唱片的硅胶模具，然后将几种不同颜色的树脂倒入最终的模具中，[埃文和凯特琳]能够制作出一张看起来很棒的唱片，它仍然保留了传输唱片原始声音所需的纹理。由此产生的作品有一些静态，但音乐仍然是可识别的。也就是说，音响发烧友可能更喜欢把它挂在墙上，而不是放在他们的留声机里。

丙烯酸环被激光切割并用螺栓固定在一起，以建立硅胶模具的模型，原始记录放在底部。为了防止气泡，在浇注到唱片上之前，硅树脂在真空室中脱气，并且在浇注到所得模具中之后，树脂在压力罐中固化。

如果你对记录最初是如何产生的感兴趣，看看这一期的[回溯技术](https://hackaday.com/2013/12/31/retrotechtacular-wax-on-wax-off-how-records-are-made/)。树脂更实际的应用可能是这种技术在[复制老式塑料部件](https://hackaday.com/2019/04/08/reproducing-vintage-plastic-parts-in-top-notch-quality/)。
