# 看到橡子阿基米德得到修复和翻新，在辉煌的细节

> 原文：<https://hackaday.com/2021/06/21/see-acorn-archimedes-get-repaired-and-refurbished-in-glorious-detail/>

想看一台 90 年代的 Acorn Archimedes A3020 家用电脑被打开、翻新和试驾吗？不要错过[drygol]关于 retro hax 的精彩文章，因为它包含了所有这些，甚至更多！

[![](img/222c6bd79097889be77dc55b0afe097b.png)](https://hackaday.com/wp-content/uploads/2021/06/CQ8WILc-1.png)

A modern upgrade allowing the use of a CF card in place of an internal hard drive, via a CF2IDE adapter and 3D-printed fixture.

阿基米德是 Acorn Computers 的一系列基于 ARM 的个人电脑，于 80 年代末发布，90 年代停产，因为 Macintosh 和 IBM PC 兼容机最终占据了主导地位。他们在当时是有能力的机器，并且[drygol]在安装一些升级的同时，翻新了一个原始的回到工作状态。

第一件事是打开机器检查内部。可见的腐蚀用草酸清除，旧的电解电容器理所当然地被替换，任何被腐蚀的痕迹得到仔细的修复。去除插座上的腐蚀需要拆焊零件进行清洗，然后重新焊接，因此整个过程可能需要大量的工作。幸运的是，老式五金件通常在设计时就考虑到了手工组装，因此零件往往易于维修，并且在维修过程中具有良好的可视性。键盘被完全拆卸并去黄，产生了一个令人瞠目结舌的吸引人的结果。

一旦计算机本身正常工作，就该进行一些现代升级了。一个是给机器一个适配器，以使用 CF 卡代替内部 IDE 硬盘，而[drygol]在使用 3D 打印件方面做得很好，使 CF2IDE 适配器看起来像是工厂提供的。内部软盘驱动器也被 GOTEK 软盘仿真器(也带有 3D 打印适配器)取代，进行了另一次现代化升级。

完全翻新和升级的机器看起来很光滑，所以看看 Acorn Archimedes A3020 在视频中展示它可以做什么(嵌入在下面)，也许会有点怀旧的感觉。
