# 发电机依靠天然气运行，配有现场权宜尖端

> 原文:[https://hack aday . com/2021/03/03/generator-runs-on-natural-gas-with-field-advetect-tip/](https://hackaday.com/2021/03/03/generator-runs-on-natural-gas-with-field-expedient-tip/)

许多小型汽油发动机可以通过更换化油器和增加调节器的套件安全地改装为使用天然气或丙烷，从而在紧急情况下难以获得汽油的情况下提供可靠的替代燃料来源。[由[HowToLou]提供的这款裤子座椅绝对不是用天然气驱动发电机的安全方式](https://www.youtube.com/watch?v=sDHGCzOZn_M)*，但如果你发现自己处于恢复供电可能是生死攸关的情况下，这是一个值得记住的提示。*

 *这里的基本思想是，通过连接到空气过滤箱的软管，将天然气(尽管丙烷也应该工作)直接输入发动机的进气口。当起动发动机时，气体管线上的阀门用于手动调节空气-燃料混合物，直到它点火。这是一个极其简单的方法，必要时，你可以用手头的零件完成。但是正如你所料，这种简单是有代价的。

这种方法有几个大问题，但最大的问题肯定是，当发动机停止运转时，没有什么可以切断气体的流动。所以，如果发电机停转，或者你在关闭后忘记关闭阀门，就有可能出现非常危险的情况。此外，手动燃气阀将与根据负载自动上下节流的发电机不一致。虽然公平地说，肯定有发电机只是运行发动机全速前进。

[很像我们上个月报道的他的 DIY 发电机联锁板](https://hackaday.com/2021/02/22/how-to-design-a-custom-generator-interlock-plate/)，我们想象这个项目可能会以错误的方式惹恼一些人。不可否认，这不是一个你应该掉以轻心的修改，当然也不是你想要长期使用的东西。[但是最近德克萨斯的局势提醒了我们](https://hackaday.com/2021/02/16/trouble-with-the-texas-power-grid/)，非常时期有时需要非常手段。
