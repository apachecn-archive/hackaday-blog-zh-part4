# 你可以自己制作一个巨大的 7 段显示器

> 原文：<https://hackaday.com/2022/07/27/you-can-build-a-giant-7-segment-display-of-your-very-own/>

有时你需要显示一个漂亮的大数字，使它在远处容易阅读。[Lewis]正好有这个东西:[一个大的可扩展 7 段显示屏。](https://www.instructables.com/How-to-Build-a-Super-sized-Expandable-Seven-Segmen/)

构建是模块化的，允许它从 2 位扩展到 10 位甚至更多。数字本身是由装配在丙烯酸树脂上的 3D 打印部件制成的。这些可以在一个木框中组合起来，显示更多的数字。各个元素由可寻址的 led 点亮，该项目可以使用 Arduino Nano 或 ESP8266 进行控制。后者开辟了通过 WiFi 控制屏幕的可能性，这可能证明是有用的。

[刘易斯]已经为当地的一家游泳俱乐部制作了自己的版本，在那里它将被用作计时器。其他应用可以是各种体育比赛中的记分牌，或者通过在你的前院显示随机数字来迷惑你的邻居。

我们已经看到[伊万·米兰达] [的一个类似的构建，它也很好地充当了车间时钟](https://hackaday.com/2020/11/01/big-workshop-clock-is-3d-printing-done-right/)。休息后的视频。
