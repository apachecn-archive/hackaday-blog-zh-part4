# 马尼托巴省制作专利疫苗验证系统的开放软件演示

> 原文：<https://hackaday.com/2021/08/19/manitoban-makes-open-software-demo-of-proprietary-vaccine-verification-systems/>

[Mark Jenkins]对马尼托巴省要求餐馆使用的 Covid 19 疫苗接种验证系统印象不佳。顾客必须出示一个二维码，这个二维码必须通过苹果或谷歌提供的移动应用程序进行验证。在当地黑客空间的帮助下，他提出了一个 [bash 脚本解决方案](https://github.com/markjenkins/immunizedshellscriptmb),只需要千字节，而移动应用需要 50 MB。[Mark]对应用程序可用性的排他性和缺乏开放 API 感到不满意。他的担忧也不完全是理论上的——谷歌神秘地将他们的应用从 Play Store 下架了一个多星期。

如下面的视频所示，中间结果是一个名为 Alexandra 的演示系统。它由收据打印机、用作 QR 扫描仪的网络摄像头和运行该脚本的 2005 年时代的笔记本电脑组成。正如[Mark]明确指出的，这仅仅是概念的证明。仍然有一些工作要做——例如，用于验证 Google 服务器的方法是暂时的。但最终[马克]希望很快有一个自由软件替代品，适合餐馆老板在他们的机构中使用。
