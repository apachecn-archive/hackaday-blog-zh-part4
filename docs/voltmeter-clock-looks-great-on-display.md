# 伏特计时钟在显示器上看起来很棒

> 原文：<https://hackaday.com/2020/01/03/voltmeter-clock-looks-great-on-display/>

电压表很便宜，而且有很好的工业美感。这使得它们成为黑客建造时钟的首选。[Brett Oliver]走上了这条路，一路走来，他打造了一款非常时尚的腕表。

[Brett]最初想用 240 度的电压表，但是成本太高，所以选择了更常见的 90 度型号。新的表盘是通过首先打磨旧的表盘，用老式的米白色重新喷漆，然后用喷墨转印纸涂上新的图形制成的。

对细节的关注延续到了本案。[Brett]的目标是用老派的实验室设备美学建造时钟。一大块红木被制作成底座。一个透明的塑料盖来自易贝，这才是真正的作品。选择大按钮和拨动开关来完成外观。

在电子方面，它全部由 PIC16F628A 运行，通过 PWM 控制电压表。运行与 20 兆赫晶体，PIC 不是一个伟大的计时器。相反，整场演出都与我们几年前推出的[Brett]的主时钟同步。

对于一个黑客来说，建造一个时钟是一个必经之路，而[Brett]的例子向我们展示了在这种追求中，工艺是如何真正获得回报的。休息后的视频。
