# 用键帽机器人制作艺术品

> 原文：<https://hackaday.com/2020/05/07/making-art-with-keycap-bots/>

机器人有各种形状和大小，从遥远星球上的远程着陆器到在汽车工厂努力工作的装配臂。当然，这个定义很宽泛，也可以包含更多无关紧要的实体。[smdavee]的水彩机器人可能不是特别复杂或有感觉，但它们是一个有趣的创意构建。

该设计类似于刚毛机器人，带有一个寻呼机振动马达，允许机器人用不稳定的脚摇晃。在这种情况下，使用键盘帽，在底部插入棉头作为腿。然后将它们浸在水彩颜料中，连接的马达打开，使“机器人”在页面上振动。

这是一个简单的构建，特别适合教小孩子基本的电子概念。此外，还有额外的乐趣，让水彩乱七八糟。如果你的车库里藏着一个有趣的艺术机器人，[一定要让我们知道](http://hackaday.com/submit-a-tip)。休息后的视频。
