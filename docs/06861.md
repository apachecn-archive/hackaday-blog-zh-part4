# 古董军队剩余接收器恢复

> 原文：<https://hackaday.com/2020/06/13/antique-army-surplus-receiver-restored/>

如果你去过美国的 hamfest，你可能至少见过 ARC 5 命令集。这些都是非常坚固的接收器，有很多这样的接收器。哈姆多年来一直在改造它们。在最近的一个视频中，[Tom N3LLL]分享了他的一些恢复它们的技巧。

你可能会认为这些就像一个普通的旧收音机，但有一些独特的挑战，包括充满蜂蜡的电容器和奇怪的螺纹螺钉。[Tom]制作了几块定制板，用固态逆变器替换发电机，替换奇数电容器，并提供了面板。他还 3D 打印了一些替换螺柱，以替换发电机经常腐烂的防震螺柱。

结尾的拆卸说明了这些东西有多粗糙。汤姆的修复哲学是在保持外在美感的同时使布景现代化。接收器性能良好，正如你所料，像坦克一样建造。

如果你想尝试修复，这些都不是很贵，因为有这么多。通常运费和收音机的价格差不多，但是一个好的价格可能会高一点。我们认为真正的乐趣是得到一个不太好的形状，并使它变得更好。

每个人都有自己的风格，我们知道一些修复者更纯粹，但实际上，[汤姆]的修复看起来很棒，听起来很棒，并保留了这些伟大的旧收音机，以便有人可能在另一个 75 年后仍在使用它们。

不出所料，我们之前已经报道过 ARC 5 T1，如果你喜欢的话，这种修复会更传统一点。如果你需要在 AM 波段听些什么，试试[一个匹配的发射机](https://hackaday.com/2018/01/12/what-to-do-with-all-of-those-mid-century-and-pre-war-vintage-am-radios-build-your-own-am-radio-station/)。
