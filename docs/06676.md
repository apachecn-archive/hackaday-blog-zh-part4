# DMCA 撤下发布的卡西欧代码不是

> 原文:[https://hack aday . com/2020/05/24/dmca-take down-issued-over-casio-code-that-wasnt/](https://hackaday.com/2020/05/24/dmca-takedown-issued-over-casio-code-that-wasnt/)

本月早些时候，我们发布了一个巧妙的计算器黑客的报道，该黑客拿走了一个卡西欧计算器，并在其太阳能电池占据的空间中放置了一个 ESP8266 模块和一个有机发光二极管显示器。由一对不显眼的霍尔效应设备控制的计算器可以被用作一个巧妙的作弊设备，但对我们来说，它是一个精心实施的黑客的缩影。我们可能喜欢它，但似乎卡西欧的人不喜欢，因为他们已经发布了一个关于该项目的 GitHub 库的 DMCA 关闭通知。

**编辑更新:**【Tom Fleet】[报道 GitHub 已完成 DMCA 审查](https://twitter.com/tomfleet/status/1265393150249631749)并认定[代码回购](https://github.com/Neutrino-1/Hack-Casio-Calculator)不侵犯卡西欧的知识产权。然而，YouTube 视频的版权问题似乎还没有解决，该视频仍然不可用。然而，那段视频仍然可以在互联网上找到。

我们不是律师，但如果你愿意访问我们的原始报道并观看完整的视频，你会发现 ESP 没有以任何方式利用计算器的功能。卡西欧处理器上的环氧树脂块完好无损，没有电线连接到计算器主板上，因此很难想象任何卡西欧代码是如何进入充满 Arduino IDE 的 ESP8266 代码的存储库中的。在 GitHub 上快速搜索“Hack-Casio-Calculator ”,在发布时，尽管 Casio 已经关闭，但还是找到了相关代码，我们看不到他们在讨论什么。也许你可以？

多年来，在我们的社区项目中，有很多尝试使用 DMCA。有些是合法的，有些是[试图压制可悲安全事件的曝光](https://hackaday.com/2014/08/05/hardware-security-and-a-dmca-takedown-notice/)，更多的是[可笑的荒谬](https://hackaday.com/2011/02/20/thingiverse-receives-first-dmca-takedown/)。在我们看来，这一个似乎是最后一类，因为很难看出所描述的项目怎么可能包含任何卡西欧代码。如果环氧树脂斑点被移除，卡西欧的代码从计算器芯片中被检索出来，那么发布 DMCA 下线是完全合法的(我们当然会报道这个故事！)，但据我们所知，把手术刀拿到计算器的盒子里，然后把一个模块塞到太阳能电池板窗口后面，并不太可能。

显而易见，卡西欧不喜欢将他们的计算器变成作弊工具，我们理解为什么会这样。但是走 DMCA 路线只会让这件事更加引人注目，我们这些记性好的人都知道这只能导致一个结论。
