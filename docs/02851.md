# 飞行棒现在是一个东西

> 原文:[https://hack aday . com/2019/04/29/flying-sticks-are-now-a-thing/](https://hackaday.com/2019/04/29/flying-sticks-are-now-a-thing/)

固定翼飞机和直升机不再是遥控世界的宠儿。甚至四轴飞行器和其他多旋翼飞行器也开始显得过时了，因为社区期待着更离谱的设计。[rctestflight]通过这种同轴双旋翼结构将事情简化到了极致，[也被称为飞行棒(Youtube 视频，嵌入下方)。](https://www.youtube.com/watch?v=N7mSE_ST1gs)

最初的设计包括两个装有支柱的无刷 outrunner 电机，它们以相反的方向旋转，以抵消各自的扭矩。每个都安装在一个万向架上，用来提供控制权限。iNav 被用作飞行控制器，选择它是因为它有多种多样的马达混合设置。建造该飞行器是为了测试其从自由落体中恢复的能力，作为早期建造无刷“火箭”飞行器的后续尝试。

对于从根本上来说是一根棍子上的两个道具来说，性能出奇的好。最初的测试没有成功恢复，但修复后的单框架版本几乎实现了这一壮举。[一般来说，多旋翼都在努力进行自由落体回收](https://hackaday.com/2014/02/06/free-falling-quadcopter-experiments-end-with-splat/)，因此在这一领域进行更多的研究绝对是值得的。休息后的视频。
