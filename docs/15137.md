# 用钢筋焊接的 DIY 自行车车轮

> 原文：<https://hackaday.com/2022/10/24/diy-bike-wheels-welded-with-rebar/>

[Liebregts]正在进行一项三轮车设计，需要一对轮子到前面去。普通的自行车车轮不行，因为它们不是为单侧支撑设计的。他们还希望能够安装盘式制动器。因此，他们着手制造一套定制车轮来完成这项工作。

建造始于一套常规的 20 英寸自行车轮圈，所有辐条都被去掉了。焊接在内圆周上的一圈钢筋增加了轮圈的强度。然后用焊机和车床组装了一套轮毂和轮轴，并为盘式制动器部件提供了螺栓连接。钢筋长度然后被焊接成不可调节的辐条。接下来，是时候刷一层漆了。最后[Liebregts]装上轮胎和刹车，宣布任务完成。

显然，这是有可能购买车轮专门为三轮车建设。然而，[liebregt]发现很难找到他们真正想要的，尤其是在盘式制动器选项方面。最好的选择是定制版本。由此产生的车轮显然比传统的自行车车轮要重得多，但它们也可能更坚固。例如，如果你需要压下一块防水布，这些轮子可能就能解决问题。
