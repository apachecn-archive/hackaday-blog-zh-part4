# 隐藏的铁路模型被放大到 n 倍

> 原文：<https://hackaday.com/2022/01/06/hidden-model-railroading-taken-to-the-nth-scale/>

如果我们为自己的家做了一件整洁的“环氧树脂河”类型的艺术品，我们大多数人都会非常满意。多产的制作人[彼得·瓦尔德拉夫]却不是这样。[Peter]是一位熟练的工匠，他喜欢将模型轨道隐藏在日常家具和固定装置中。为此，他创造了他所谓的隐藏环氧树脂铁路，你可以在休息时间下方的[视频中看到。](https://www.youtube.com/watch?v=bYujwMY8_4Q)

该项目从基本的木工开始，用彩色环氧树脂浇注来建造框架、铰链和铰链环氧树脂河。没有就此打住，[Pete]继续建造了一个全 N 比例的铁路模型。该框架兼作背景，配有一幅画得很漂亮的场景，我们猜测这也是一个 N 尺度的热气球。

当然，我们中的怀疑论者可能会怀疑模型铁路关闭时是否完全隐藏起来。但是皮特和我们分享的一个细节是，直到他们的女儿指出来，他的妻子才知道这个建筑的火车部分。她认为这只是餐厅里的一件艺术品。真是一个伟大的黑客！

整个版本都记录在[【皮特】的 YouTube 频道](https://www.youtube.com/c/PeterWaldraff)，[的 15 集版本系列](https://www.youtube.com/playlist?list=PLWtBDw_gr9AT9K-Z0DX4A6EluIlqbHWJI)中。绝对值得一探究竟。在之前，我们已经报道了[的一个【皮特】隐藏的铁路建筑，所以一定要去看看！](https://hackaday.com/2021/04/10/coffee-table-railroad-flips-to-hide-the-fun/)
