# [本·克拉斯诺]看着胶卷相机里的日期戳

> 原文：<https://hackaday.com/2019/10/15/ben-krasnow-looks-inside-film-camera-date-stamping/>

老实说，我们从来没有想过那些老式胶片相机是如何把日期戳放在相框的右下角的。幸运的是，[Ben Krasnow]并没有遭受这种可悲的缺乏好奇心的痛苦，他的视频拆除了一个盖日期戳的胶片相机的背面(嵌在下面)不仅回答了这个问题，还提供了价值工程的一个有用的教训。

对于可能相当一部分从未在胶片上拍过照片的观众来说，廉价的 35 毫米相机曾经是一件大事。除非你想购买单镜头反光相机以及所有的镜头和配件，否则它们真的是一个人拥有的所有家庭快照之类的东西。它们比早期的盒式相机好得多，比如 110 或——令人不寒而栗——盘式胶片相机(T1)，而且这些相机也开始具备一些简洁的电子功能。一个是小小的红橙色日期戳，从颜色来看，我们和[本]认为它是某种贴在薄膜上的 LED，但它最终比那要酷得多。

深入到一个旧相机的背面，[Ben]发现实际上有一个微型投影仪，它使用一面镜子来折叠胶片和麦粒白炽灯泡之间的光路。LCD 滤光器位于光路中，但因为它不完全在胶片的平面上，所以它实际上必须将图像投影到胶片上。白炽灯泡充当点光源，镜子使光路足够长，使得日期戳图像在胶片上清晰可见。它便宜，容易适应其他相机，而且可靠。

像这样的拆卸不是[本]通常的视频节目的素材，它更倾向于[自制的 CT 扫描仪](https://hackaday.com/2013/01/09/ben-krasnow-builds-a-ct-scanner/)和[阿波罗级电致发光显示器](https://hackaday.com/2019/05/19/ben-krasnow-makes-a-dsky/)，但这也是有益和有趣的。
