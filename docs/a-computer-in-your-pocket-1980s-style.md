# 口袋里的电脑，80 年代风格

> 原文：<https://hackaday.com/2021/01/22/a-computer-in-your-pocket-1980s-style/>

如今，口袋里装一台小电脑是再正常不过的事情了。但是四十年前，这是一个新的高科技想法。[8 位的家伙]有一个很棒的视频[，涵盖了 20 世纪 80 年代和 90 年代口袋电脑](https://www.youtube.com/watch?v=d3NIe1jTZMc)和个人数字助理的最新技术。你可以看下面的视频。

视频中有很多熟悉的面孔，包括 Radio Shack 袖珍计算机、Palm Pilots，以及一些质量不一的更不知名的机器。

它可能会给你留下深刻印象，知道无线电夏克 TRS-80 PC-1 袖珍计算机实际上有两个 CPU。当然，每个 CPU 都是一个运行频率为 256 kHz 的 4 位处理器，所以可能没有听起来那么令人印象深刻。尽管如此，在 24 字符液晶显示器上进行 BASIC 编程在当时还是一个奇迹。

1983 年的卡西欧 PB-700。不到 200 美元，你就得到了一个 4 行显示器，每行有 20 个字符。但它也有一些图形功能。你甚至可以得到一个可选的扩展坞，提供彩色绘图仪和微型录音机。不幸的是，它们并没有被广泛使用，至少在美国是这样。

如果你想要超便携，精工 UC-2000 实际上是一款可以连接键盘和打印机的手表。不幸的是，[Guy]找不到电池，但我们确实很喜欢看他把它拆开，取出没电的电池。

我怀疑结尾的个人轶事对许多上世纪七八十年代的学龄儿童来说会有些熟悉。时代确实变了。

当然，今天你可以在口袋里装更多的计算能力。这段视频并不是我们看到的唯一一封关于那个时代的[电脑的怀旧情书。](https://hackaday.com/2019/10/02/where-did-pocket-computing-start/)
