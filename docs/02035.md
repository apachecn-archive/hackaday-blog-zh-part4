# 小 FPV 机器人用 ESP32 保持简单

> 原文:[https://hack aday . com/2019/02/11/little-fpv-bot-keeps-it-simple-with-an-esp32/](https://hackaday.com/2019/02/11/little-fpv-bot-keeps-it-simple-with-an-esp32/)

当谈到机器人时，似乎趋势是让它们尽可能复杂——如果你对此有任何疑问，请看看波士顿动力公司的任何东西。但是简单的机器人也有很多值得一提的地方，比如这个可爱的 ESP32 驱动的直播机器人。

现在真的是[Max。k 的创造更多的是遥控汽车而不是机器人，将它与波士顿动力公司的一个噩梦般的创造相比较可能是不公平的。但是【Max。k 的新项目本身就是对他早期更大的“ZeroBot”的简化和重新构想。顾名思义，ZeroBot 是由 Raspberry Pi Zero 控制的，对于一个旨在播放 FPV 视频的移动平台来说，这是一个显而易见的选择。ESP32 bot 避开了 Pi 平台，转而使用 ESP32。为了尽可能节省空间，[Max。K]为微控制器及其支持元件定制 PCB。3D 打印的外壳设计得很好，可以容纳主板、两个电机、一个小型 VGA 摄像头和一个电池组。在 160×120 的分辨率下，视频并不惊人，但它可以从 ESP32 以足够好的帧速率传输，以使用简单的网络界面驱动机器人的事实令人印象深刻。

这是一个有趣的项目，非常干净、流畅。我们喜欢这个小机器人的线条，并不介意自己建造一个作为快速周末项目。
