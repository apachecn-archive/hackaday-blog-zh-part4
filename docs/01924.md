# SNES 便携式利用闪光车为更多的游戏

> 原文:[https://hack aday . com/2019/01/25/snes-portable-leverages-flash-cart-for-more-games/](https://hackaday.com/2019/01/25/snes-portable-leverages-flash-cart-for-more-games/)

掌上游戏机必须做出许多设计选择，而他们的电视连接兄弟不必担心这些选择。电池寿命很重要，屏幕可视性也很重要，如果你要整天带着它们，游戏不能太笨重。[Chris]对构建手持版本的家用游戏机并不陌生，[并在他最新的便携式 SNES 构建中吸取了一些经验。](https://downingsbasement.com/2019/01/22/the-snes-box-go-perhaps-a-first-for-console-modding/)

主板由小 SNES 提供，这是一款在游戏机统治末期发布的轻量级紧凑型产品。这是足够小，它不需要调整，但是[克里斯]选择取代效率低下的 7805 与一个更现代的开关稳压器。表壳是在典型的 FDM 设备上 3D 打印的，而按钮是在 Form 2 上生产的，以获得更好的尺寸精度和表面光洁度。

![](../Images/81207d419c90808e8c3696aace91776e.png)

The flash cart PCB is permanently wired to the motherboard.

然而，真正的聚会作品是使用 SD2SNES flash cart。这允许将大量不同的 rom 加载到一个 SD 卡上，并在原始的控制台硬件上播放。这在便携式构建中特别有用，因为它可以携带你想要的所有游戏，而不是必须摆弄几个全尺寸的 SNES 墨盒。SD2SNES 在控制台内部永久布线，主板和盒式 PCB 之间有大量跳线。尽管引线很长，[Chris]报告说连接没有问题。

有一些限制 flash cart 对于使用额外芯片的游戏来说不能正常工作，例如 Star Fox 中的 SuperFX。尽管如此，这是一个优秀的，高质量的建筑，我们相信在外面玩会很有趣。
