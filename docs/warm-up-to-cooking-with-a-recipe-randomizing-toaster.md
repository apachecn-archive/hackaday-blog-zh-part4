# 用随机食谱烤面包机做饭前热身

> 原文：<https://hackaday.com/2020/04/08/warm-up-to-cooking-with-a-recipe-randomizing-toaster/>

你有没有在热敏打印机很热的时候买了一台，但是当你不知道如何处理它的时候，你的兴趣就冷却了？类似的事情也发生在了[Sunyecz22]身上，这台可怜的打印机一直闲置着，直到神奇的一天，它的完美用途出现了——一个烤面包机形式的随机食谱接收器。

【Sunyecz22】厌倦了每周去杂货店前搜索食谱。在互联网上数以百万计的食谱选择和之前 1000 字的论文之间，这个过程就像一份兼职工作。现在他们所要做的就是按下小杠杆，等待食谱被烤成热敏纸。它不打印完整的食谱，只打印要点，我们喜欢这样。你可以得到名字、准备时间、评分和链接到食谱页面的二维码。

这台烤面包机运行在 Raspberry Pi Zero W 上，它使用 Spoonacular API 获取食谱，并将这些食谱发送到打印机。杠杆利用一些旧的笔式弹簧来启动限位开关，开始获取食谱的过程。我们认为，如果它一直保持到食谱弹出来，那将会非常酷。休息时涂上黄油，观看简短的演示视频。

我们必须说，这个烤面包机比《红矮星》里的有声烤面包机有用多了。
