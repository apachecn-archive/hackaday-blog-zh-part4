# 玩超级挂起与黑客控制器给爪子的理由

> 原文:[https://hack aday . com/2022/01/07/playing-super-hang-on-with-hacked-controller-give-reason-for-paws/](https://hackaday.com/2022/01/07/playing-super-hang-on-with-hacked-controller-gives-reason-for-paws/)

当你在二手商店购物时，会发生一件事。你知道这是怎么回事:你看到一件令你心动的东西，你的思维立刻锁定，而这件东西必须是你的。[Tom Tilley]在当地旧货店看到 Paw Patrol 儿童玩具时经历了这一点，你可以在休息时间下方的视频中看到[他假期黑客会议的结果。](https://www.youtube.com/watch?v=kHApOIClPoA)

[汤姆]如何使用爪子巡逻游戏？看起来像摩托车驾驶舱让他几乎没有选择。没过多久，他就把游戏翻了过来，拆了内部零件，拼凑出了我们最近见过的最完美的基于玩具的界面。

使用一个树莓 Pi Pico 控制器和一些精心的手术，[汤姆]把爪子巡逻游戏变成了 1987 年世嘉摩托车比赛街机游戏超级挂机的控制器。看[汤姆]打球是一种享受，但万一整件事是一个失败的前景，它也是完全可逆的！

当然，我们对超级玩家不能使用爪子按钮感到有点失望。

另一方面，这是一辆变成游戏控制器的真实汽车。有一个热门控制器黑客分享？一定要让[线报](https://hackaday.com/submit-a-tip/)知道！
