# 使用此开发板了解 Touch

> 原文：<https://hackaday.com/2022/12/31/get-to-know-touch-with-this-dev-board/>

在中国零件供应商 LCSC 的目录中，可以找到许多美国或欧洲供应商没有的零件，因此任何想要评估它们的人都会发现自己处于不利地位。[昏昏欲睡的小马实验室]就有这样一个部件抓住了他们的眼球，Sam&Wing AI08 8 通道电容式触摸控制器。如何评价一个信息很少的芯片？[当然是设计开发板](https://www.sleepyponylabs.com/projects/ai08devboard.html)！

被测试的芯片属于一个家族，该家族提供相似的功能，但是具有多种接口选项。被测器件有八个触摸输入和一个 BCD 输出。所述输出用于馈送 74 系列解码器芯片并驱动一些 led。触摸板的设计参考了[一份微芯片应用笔记](https://www.microchip.com/en-us/application-notes/an2934),这份笔记附带地让读者对这个主题产生了兴趣，因为它涵盖的远不止是简单的触摸按钮。

你是否需要这种触摸芯片是你自己设计的问题，然而，这个项目表明，随着廉价定制 PCB 和意想不到的部件的现成可用，仅仅为了评估目的而创建电路板并不是没有道理的。

也许以前的黑客文章的主题会发现这个板很有用。
