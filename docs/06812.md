# 独特的 Strandbeest 站在高与线的腿

> 原文:[https://hack aday . com/2020/06/08/unique-strand beest-stands-tall-with-line-of-legs/](https://hackaday.com/2020/06/08/unique-strandbeest-stands-tall-with-line-of-legs/)

多排错综复杂的关节腿是 Strandbeest 的定义特征，但[James Bruton]想知道他是否可以将自平衡两轮机器人中使用的相同原理减少到单排[。虽然称他的实验完全成功可能还为时过早，但他(相对)苗条的 Strandbeest 迈出的第一步看起来很有希望。](https://github.com/XRobots/BalancingStrandbeest)

[![](../Images/b6eb928ab2ca9f519b3a5986dee1d817.png)](https://hackaday.com/wp-content/uploads/2020/06/balancebeest_detail.jpg) 最初这个机器人只有两对腿，但在测试中【詹姆斯】发现这种排列有点不稳定。通过将总数量增加到每侧四条腿并改进配重布置，机器人已经能够行走车间的长度。不幸的是，腿部设计的一个问题似乎阻止了 Strandbeest 后退。

通常这不会是一个大问题，但在这种情况下，它使 Strandbeest 无法在静止时保持自我平衡。换句话说，机器人需要不断向前移动，否则就会摔倒。尽管如此，[詹姆斯]认为这个想法有希望，并希望继续在更大的领域试验机器人。

具体来说，他想看看双电机机器人是否可以通过改变两组腿的速度来转弯。如果它能在一个足够小的圈子里行走，它就能一直前进，直到能量耗尽。对我们来说听起来有点可怕，但我们还是想看看。

读者可能会想起另一个机器人项目中的[詹姆斯]，比如惊人的开放式狗。我们不知道他对腿式机器人的痴迷从何而来，但我们当然不会抱怨。
