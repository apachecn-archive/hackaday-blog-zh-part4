# 合成红宝石的快速简易配方

> 原文：<https://hackaday.com/2020/03/06/a-quick-and-easy-recipe-for-synthetic-rubies/>

考虑到制造人造钻石所需要的压力、灼热的温度，你会认为任何人造宝石都需要类似的条件。然而显然不是，因为[夜鹰之光]揭示了他制作人造红宝石的简单方法。

像它们的宝石表亲蓝宝石一样，红宝石只是刚玉或氧化铝的一种变体。蓝宝石的蓝色主要来自铁，而红宝石的粉色到血红色则来自铬。所以[NightHawkInLight]的配方从氧化铝喷砂粉末和氧化铬(III)开始，氧化铬是一种常见的绿色颜料，也是一个家族中较安全的化合物之一，该家族包括像六价铬化合物这样的剧毒物质。当混合在一起时，使用带有碳电极的电弧焊机在石墨坩埚中加热这两种粉末。坩埚似乎是由 EDM 电极制成的；我们以前见过它们被用于空气轴承，但是小坩埚是这种材料的另一个重要用途。需要一些技巧来防止新生的红宝石散落一地，但最终，[夜鹰之光]得到了一颗巨大的深粉色红宝石。

这看起来像一个有趣的，快速的小项目，可以尝试一下。我们想知道这种方法是否可以被改进来制造红宝石激光器的内部，或者是否可以用来制造蓝宝石。
