# 团队在弹球决赛项目中获得高分

> 原文：<https://hackaday.com/2019/12/25/team-scores-big-points-with-pinball-final-project/>

对于他们在[Bruce Land]的 PIC32 微控制器设计课上的最终项目，[Sujith]、[Julia]和[Andrew]想做一些有趣的事情。还有什么比随着弹球机的机电警报曲弯腰更有趣的呢？

[这台机器看起来很棒](http://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2019/jen67_aht53_sn438/jen67_aht53_sn438/jen67_aht53_sn438/index.html)，正如你在休息后的演示视频中看到的，它的播放和声音也很棒。我们特别喜欢回旋镖障碍和游戏状态驱动的 LED 条。你得的分越多，它们就越亮。我们还喜欢这台机器结合了传统的得分方法和一些非常聪明的方法，如顶部附近的回旋镖目标和由铜带制成的得分触发器。

该团队从设计任何弹球机的核心——鳍状肢开始。虽然我们已经看到汽车门锁驱动器用于家酿机，该小组继续用传统的螺线管来驱动它们。不幸的是，螺线管造成了很多干扰，但团队用过滤电容器和铝箔法拉第笼绕过了电线。

如果所有这些弹球谈话已经点燃了你的电路，[为什么不试着做你自己的机器呢](https://hackaday.com/2017/11/11/a-sandbox-for-diy-pinball-design/)？
