# 适合户外使用的双耳话筒

> 原文：<https://hackaday.com/2022/12/17/a-binaural-microphone-for-the-great-outdoors/>

双耳录音旨在尽可能模拟通过人耳收听的体验，因此双耳麦克风的形状通常类似于人的头部，麦克风拾音器放置在耳朵所在的位置。不过，这并不是制作双耳麦克风的唯一方法，皇冠 立体声环境采样系统、 或 SASS 也用一对压力区麦克风做了同样的事情，用于室外录音。[菲利普·穆里尔]没有原版的麦克风，但他已经尽了最大努力[制作了一个他自己的类似萨斯的麦克风](https://hackaday.io/project/188609-sass-style-stereo-microphone-for-nature-recording)。

这种设计吸引人的地方在于它的简单性，主体使用泡沫板，包装板作为挡雨板，几层无纺布作为挡风层。也许最好的是录音，在录音中我们可以听到环境录音的最佳效果。用耳机听，我们建议[黎明合唱](https://freesound.org/people/filipmu/sounds/591464/)。

如果你对双耳录音和立体声感兴趣，[我们过去曾近距离观察过](https://hackaday.com/2022/09/21/know-audio-stereo/)。
