# 这个平视显示器是全湿了

> 原文：<https://hackaday.com/2019/08/29/this-heads-up-display-is-all-wet/>

运动员使用他们能找到的任何东西来提高他们的成绩或改善他们的训练有着悠久的历史。虽然健身追踪器手表并不新鲜，但游泳运动员已经用它们来追踪自己的分段时间、距离和其他参数。然而，健身追踪器的问题是你必须看手表。FORM 的泳镜承诺解决这一问题，他们的[智能泳镜](https://www.formswim.com/products/form-swim-goggles)为游泳者提供了一个平视显示器。你可以在下面看到一段关于他们的视频。

屏幕只在一只眼睛上，虽然你可以从左向右切换。这款设备有惯性导航系统，当然也是防水的。据说它可以承受 32 英尺的深度，充电后可以持续 16 个小时。除了显示屏之外，它还可以使用蓝牙将您的数据发送到手机。

所有这些都是有代价的，护目镜的价格约为 200 美元。这不是一元店的泳镜，但即使是一副漂亮的泳镜也可能卖 30 美元。

据报道，该设备可以跟踪分段时间，间歇时间，休息时间，总时间，中风率和计数，每次中风的距离，步速，距离，长度计数，甚至燃烧的卡路里。

这让我们想起了[谷歌眼镜](https://hackaday.com/2014/07/28/a-better-google-glass-for-60-this-one-folds/)。然而，我们见过的最相似的展示是[汽车](https://hackaday.com/2017/07/20/heads-up-display-turns-car-into-fighter-jet/)。
