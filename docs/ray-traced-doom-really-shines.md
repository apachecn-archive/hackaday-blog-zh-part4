# 光线追踪的毁灭战士真的闪闪发光！

> 原文：<https://hackaday.com/2022/04/18/ray-traced-doom-really-shines/>

我们是复古游戏和添加新图形功能的超级粉丝，所以你必须知道当[Sultim Tsyrendashiev]发布[他的光线追踪毁灭引擎](https://github.com/sultim-t/prboom-plus-rt)时，我们必须报道它。现在这确实打破了传统——这个版本需要 AMD 或 Nvidia 光线跟踪功能卡，而不是在每个可以想象的平台上运行 Doom。另一方面，厄运的精神肯定是活着的，因为光线跟踪的厄运已经在蒸汽甲板上演示过了。请观看下面的视频进行演示，休息后回来了解更多信息。

这一图形壮举最令人兴奋的部分可能是[ray stredgl 1 库](https://github.com/sultim-t/RayTracedGL1)，它“简化了将具有固定功能管道的应用程序移植到实时路径跟踪的过程。”除了《毁灭战士》,还有《严肃的山姆》和《半条命 1》的演示。甚至还有实验性的 Linux 支持！我们设法在我们的系统上编译和[测试它，运行 6700 XT 和 Fedora 35，带有前沿台面。有一些视觉上的小问题需要解决，但这是迄今为止一个出色的项目。我们唯一的抱怨是，它是基于 prboom，而不是仍然保持的 GZDoom，尽管有足够的关注，谁知道这个项目会去哪里。如果这让你渴望更多，检查更多的](https://www.youtube.com/watch?v=mV-KMCf7Q3s)[复古升级](https://hackaday.com/2019/04/26/snes-mode-7-gets-an-hd-upgrade/)，或更多设备上的[毁灭。](https://hackaday.com/2021/08/13/doom-on-a-desk-phone-is-just-the-tip-of-the-iceburg/)
