# 导盲鞋将计算机视觉与触觉反馈结合起来

> 原文:[https://hack aday . com/2021/05/12/seeing-eye-shoes-pair-computer-vision-with-haptic-feedback/](https://hackaday.com/2021/05/12/seeing-eye-shoes-pair-computer-vision-with-haptic-feedback/)

多年来，我们已经看到许多项目使用超声波或飞行时间传感器作为视觉障碍者的物体检测方法。超声波传感器像声纳一样探测物体——它们发送声音脉冲，并测量信号从物体反弹回来所需的时间。飞行时间传感器基本上做同样的事情，但用的是红外光。在任一情况下，通知通常作为触觉反馈出现在手腕或头部或超声波模块所附着的任何肢体上。我们经常想为什么没有商业制造的鞋子能做到这一点，但是事实证明有，而且它们会变得更好。

The shoe’s view — safe walking space is delimited with color. Image via [TU Graz](https://www.tugraz.at/en/tu-graz/services/news-stories/media-service/singleview/article/aus-der-fussperspektive-tu-graz-entwickelt-algorithmus-fuer-schuhbasiertes-blindenassistenzsystem0/)

今天，Tec-Innovation 生产脚趾上带有超声波传感器的鞋子，可以探测到 4 米远的物体。穿鞋者通过鞋内的触觉反馈以及通过蓝牙的音频电话通知来获知障碍物。该公司与奥地利格拉茨科技大学合作，赋予鞋子机器人视觉，提供更好的细节。

超声波有很大的帮助，但是它不能探测障碍物的地形，也不能区分坑洞、岩石和墙壁。但是如果你的双脚上都有摄像头，你就可以利用这些数据来确定障碍物的类型，并相应地通知用户。这些新型号仍将使用超声波传感器进行初始目标检测，并使用摄像头进行分析。

无论何时出现，传感器都将通过应用程序连接起来，这为各种城市的众包障碍地图铺平了道路。鞋子也会很贵。你能以更少的钱做同样的事情吗？想想这个挑战吧！
