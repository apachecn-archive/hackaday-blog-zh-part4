# 伍德·SCARA 的手臂被抓住了

> 原文:[https://hackaday.com/2019/03/14/wood-scara-arm-gets-a-grip/](https://hackaday.com/2019/03/14/wood-scara-arm-gets-a-grip/)

[Ignacio]的 *VIRK I* 是一个 SCARA 设计的机器人手臂，具有令人难忘的木质身体，其[新手爪允许它进行简单的拾取和放置演示](http://automaticartisan.com/2019/01/29/virk-i-early-2019-progress-report/)。设计机器人手臂是一项艰巨的任务，基本的机械设计只是整体的一部分。即使 SCARA 分支的基本框架是一个已解决的问题，但构建它的挑战和永无止境的实施细节使它成为一个长期项目。

当我们[第一次看到 *VIRK I*](https://hackaday.com/2018/09/13/wood-shines-in-this-scara-robotic-arm-project/) 闪亮的澳大利亚布莱克伍德时，它没有任何末端效应器，[Ignacio]不确定控制它的最佳方式。从那以后，【Ignacio】已经试验了[马林](http://marlinfw.org/)和[旺萨马斯](https://github.com/wangsamas/wangsamas-firmware)对 SCARA 手臂的支持，并设计了一个基于业余爱好伺服的手爪。看到这个项目向前推进，就像看到手臂移动~~乒乓球~~球一样美好，嵌在下面。
