# 你用喷雾器安装的天线

> 原文：<https://hackaday.com/2018/09/27/antennas-that-you-install-with-a-spray-can/>

随着手机、WiFi、蓝牙和其他无线电技术的迅猛发展，对天线的需求也在不断增加。一切都变得越来越小，甚至可以佩戴，因此传统天线比以往任何时候都不实用。你可能已经在 ESP8266s 之类的东西上看到过 PCB 天线，但德雷克塞尔大学的研究人员现在正在研究使用碳化钛-被称为 MXene 来构建薄、轻、甚至透明的天线，其性能优于铜天线。与 3D 打印的趋势相反，这些天线像墨水或油漆一样被喷涂在表面上。

使用金属的传统天线在皮肤上承载大部分电流(我们在之前已经讨论过[)。例如，在 WiFi 频率下，铜天线的趋肤深度约为 1.33 微米。这意味着天线必须至少足够厚，以在该深度从所有表面传输电流——实际上，5 微米是最薄的厚度。这听起来并不多，但当你试图做一些薄而柔韧的东西时，它就相当厚了。使用 MXene，研究人员制作了 100 纳米厚的天线——这是 1 微米的 10%，只有传统天线的 2%。](https://hackaday.com/2017/11/22/skin-effect-in-the-game/)

还有其他材料可以做成薄天线，但它们都面临着挑战，要么是因为它们的导电性不好，要么是因为难以制造。MXene 是德崇大学开发的一个相当新的材料系列。为了制造它，你从钛、铝和碳的混合物 MAX 开始。铝的去除过程需要酸和搅拌 24 小时，氯化锂和离心机。氢氟酸很难处理，但是一个细心的家庭实验室是做不到的。你可以在下面看到一个关于制作 MXene 的 Drexel 视频。研究人员将天线喷涂在薄塑料基板上。

对我们来说，唯一棘手的是所用的特定 MXene 薄片在空气中会因氧化而降解。这意味着生产需要氩气，最终产品必须用一些东西来保护它免受空气的影响，所以这将增加实际设备的厚度。

当然， [PCB 天线](https://hackaday.com/2018/06/12/teardown-locates-fractal-antenna/)并不是什么新鲜事。但是如果你读了这篇论文，你会发现这些天线可以轻易地超越传统的薄天线。
