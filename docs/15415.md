# 新零件日:用于射频介电结构的奇特灯丝

> 原文:[https://hack aday . com/2022/11/20/new-part-day-exotic-filament-for-RF-dielectric-structures/](https://hackaday.com/2022/11/20/new-part-day-exotic-filament-for-rf-dielectric-structures/)

在门外汉看来，微波 RF 设计的世界充满了难以想象的奇异器件，因为工程师们利用物理定律按照自己的意愿驯服无线电信号。他们武器库中的武器包括已知介电特性的材料，用这些材料可以制造出对遇到它们的射频具有预期效果的结构。这在传统上一直是一个困难和昂贵的过程，但现在由于有了具有一系列已知介电值的 3D 打印机细丝而变得容易多了。

最好是把这些材料设计成类似于我们在光领域都习惯的菲涅尔透镜的结构。由[微波杂志](https://www.microwavejournal.com/articles/39143-nanoe-launches-a-line-of-dielectric-filaments-for-rf-applications)给出的例子是一个用于可控天线的超曲面，用更传统的方法，这将是一个更困难的工作。

通常[当我们告诉你一种新的特殊灯丝](https://hackaday.com/2021/03/24/pctg-filament-has-interesting-properties/)时，我们会预计它会比标准的 PLA 更贵，但这种灯丝是独一无二的，每公斤 275 欧元。因此，大多数读者的兴趣可能更多的是这项技术，而不是使用的预期，但即使如此，我们也可以看到，在我们的范围内，仍然会有微波实验者可能会被其独特的属性所诱惑。我们期待着使用它开发的东西。
