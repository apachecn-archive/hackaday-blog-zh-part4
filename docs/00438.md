# 四轴飞行器抛弃电池；仅靠太阳能飞行

> 原文：<https://hackaday.com/2018/08/25/quadcopter-ditches-batteries-flies-on-solar-power-alone/>

当你思考这个问题时，这似乎是显而易见的:为什么不在四轴飞行器上安装一块太阳能电池板，这样它就可以依靠太阳能飞行了？不幸的是，物理学是一个残酷的女主人，当你考虑重量功率比、面板效率和类似的繁琐技术细节时，它变得更加复杂。然而，新加坡国立大学的这群学生已经在一定程度上克服了这些技术问题:他们刚刚制造了一架仅由太阳能供电的无人机，没有电池或其他电源。

他们的创作是一架定制的四轴飞行器，由碳纤维制成，重量仅为 2.6 千克(约 5.7 磅)，但具有约 4 平方米(约 43 平方英尺)的太阳能电池板。通过测试和手工选择效率最高的面板，他们能够产生足够的电力来驱动四个转子，并成功实现了高达 10 米的高度。自 2012 年以来，学生们一直在研究这种原型，当时他们的第一个版本只能产生飞行所需电力的 45%。因此，在下面的演示中达到 100%的飞行动力是一个重要的步骤。

不过，这个原型看起来确实有点摇摇晃晃:所有的太阳能电池板表面都像帆一样，可以捕捉到风，所以我们怀疑它在微风中是否稳定。但是我们确信未来会有所改善。
