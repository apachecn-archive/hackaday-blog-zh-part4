# 适合家庭商店的旋转浇铸机

> 原文:[https://hacka day . com/2022/06/03/a-旋转浇铸-家用机器尺寸/](https://hackaday.com/2022/06/03/a-rotocasting-machine-sized-for-the-home-shop/)

如果你曾经想知道像坦克和鼓这样的大型中空塑料结构是如何形成的，你很幸运:[安迪]不仅向我们介绍了旋转浇铸和成型的细节，而且他还建造了这个可爱的小旋转浇铸机来帮助他的 DIY 项目。

当然，[安迪]的体型不会太大，比如汽车油箱或皮划艇。不仅它的尺寸更适合小型零件，而且这些结构通常是用旋转模塑的相关工艺制成的。这两个过程都使用了一个封闭的多部件模具，其中部分填充了塑料树脂，然后围绕两个轴旋转模具，以在模具内部分布一层薄薄的树脂。区别在于滚塑使用热塑性树脂，而滚塑使用聚氨酯和硅树脂等在室温下固化的树脂。

这台机器看起来很简单，但只是因为他煞费苦心地对它进行了优化。下面的视频详细介绍了构建过程——如果您只是想看看它的运行情况，请随意跳到第二个视频的 11:38 标记处。虽然你会错过一些有趣的花絮，比如在方管中焊接一个完美的 90°接头。还有一个定制工具[Andy]用来拼接他用来驱动模具旋转的串珠链，这是纯粹的天才。

使用这台机器和一个复杂的九件式模具，[安迪]能够用聚氨酯树脂制造出非常精细的钢筋混凝土汽车轮胎。我们很想看看这个装置还有什么用处——几乎和我们想看看模具是如何制造的细节一样多。我们以前见过其他的旋转浇铸机，但是这一台是最适合的。
