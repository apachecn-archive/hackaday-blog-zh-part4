# Omniwhegs 是可怕的两倍

> 原文：<https://hackaday.com/2022/12/14/omniwhegs-are-awesome-times-two/>

最奇怪的轮子是什么？全方位轮。除非你把轮子也算作轮子。这份来自上海科技大学的研究论文探索了这两种想法的融合，其中车轮作为标准的全向轮滚动，直到车轴上的伺服系统将它们展开成车轮配置。结果呢？OmniWhegs！

最终的车辆与最初的车轮概念有些不同，最初的车轮概念使用了被动平衡腿部力量的柔性机构。在这里，omniwhegs 是刚性的，实际上使用了一个同步例程，你可以在下面嵌入的视频中看到。

如果你不能得到足够的全方位轮，你并不孤单。这里有一辆[稀有三轮](https://hackaday.com/2013/06/08/omniwheel-robot-build-uses-a-bit-of-everything/)，这里有一辆[MDF](https://hackaday.com/2016/02/20/mdf-omniwheel-uses-no-metal-or-plastic/)全向轮。我们还没有看到足够多的基于 whegs 的机器人，但是 [OutRunner 非常令人震惊](https://hackaday.com/2014/05/10/robot-runs-on-6-legs-but-never-more-than-2-at-a-time/)，我们认为值得再看一眼。

我们以前也见过[轮子变成轮子](https://hackaday.com/2020/11/04/wheels-or-legs-why-not-both/)，但是没有泛光灯。我们也不知道它是否能走出机器人渲染阶段。

因此，上海团队获得了荣誉，他们采用了最奇怪的车轮，并实际建造了它们！
