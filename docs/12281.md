# DVD 驱动部件的微型线性致动器

> 原文:[https://hack aday . com/2021/12/25/mini-linear-actuators-from-DVD-drive-parts/](https://hackaday.com/2021/12/25/mini-linear-actuators-from-dvd-drive-parts/)

多年来，一些最小和最便宜的国产 CNC 机械装置的来源一直是似乎永无止境的剩余 CD 和 DVD-ROM 驱动器的供应。移动激光的线性致动器可能不是最长或最强的，但它是免费的，我们已经看到许多使用 CD 驱动器的小 X-Y 工作台。正是这些机制[Nemo404]走得更远，将丝杠和电机从驱动底盘中解放出来，并将它们放置在一个 3D 打印的外壳中，形成一个完整的线性致动器，可用于其他项目。(视频，嵌在下面。)

似乎没有位置反馈，甚至没有限制开关，这将优雅一个典型的 CD 驱动器，但除此之外，它使一个紧凑的单位。有两个版本，一个是线性轴承，另一个是 CD 驱动器中的黄铜衬套。目前还不清楚这一结果有多强，但它似乎足以证明举起一个装有螺钉的小容器。

如果你需要制作自己的致动器，那么除了容易获得的旧 CD 驱动器[之外，文件可以在 Thingiverse](https://www.thingiverse.com/thing:5154778) 上找到。通过参观这个工厂，向你自己介绍数控机床[的光盘驱动器世界。](https://hackaday.com/2020/08/25/mini-cnc-mill-goes-horizontal-to-reuse-cd-drives/)
