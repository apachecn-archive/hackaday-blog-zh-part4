# 将宜家灯罩改装成令人惊叹的二色性灯

> 原文：<https://hackaday.com/2021/10/14/hacking-an-ikea-lampshade-into-a-stunning-dichroic-lamp/>

通常，当我们看到一个彩色灯项目时，它利用了 RGB LEDs 和各种可爱的动画和褪色效果。[这个来自【Raymond Power】](https://www.instructables.com/Dichroic-Lamp-LEDs-But-No-RGBs/)的项目以美丽的变色为特色，但为了二向色薄膜的魔力，放弃了花哨的 led。

二色性薄膜与薄膜干涉一起工作，通过薄膜的光的波长根据入射角而变化。因此，当观察者的视角改变时，薄膜的表观颜色也会改变。当几层薄膜叠放在一起时，会产生特别美丽的效果。

[Raymond]碰巧从一个别致的宜家灯罩中获得了一些影片。当时，他一直在尝试折叠纸立方体和类似的结构，并决定融合这两种想法。

结果是一个立方体二向色灯罩，看起来真的很棒。这座建筑位于一盏简单的白色 LED 灯之上，以丰富的互补色和变换色点亮。

这是一件漂亮的东西，也是我们希望在自己家里拥有的东西。二色性材料也被用于一些更科学的用途。休息后的视频。
