# 枯燥幽默的典范

> 原文：<https://hackaday.com/2021/06/18/a-model-of-dry-humor/>

如果你想看树屋中模型香蕉与许多半开玩笑幽默的辉煌组合，你会欣赏[【Studson】的建造视频](https://www.youtube.com/watch?v=2k4ZK9p8fwc)。视频也在休息之后。他正在向 1999 年的大金刚 64 致敬，这对于一些人来说可能是很久以前的事情了。抓起一块你最喜欢的香蕉味水果，坐着等着听笑话，就像他的烤树皮一样干。

树屋混合使用发现的材料和工艺用品。在一个多彩的转折中，所有挥舞着棕色树皮的树枝都是绿色的，而装饰性的绿色植物来自一个模特商店的货架。这一切都始于从后院修剪下来的分叉树枝和光滑的容器盖，这可能会让你下次在购买什锦果仁时多看两眼。如果你认为咖啡搅拌器不能在它们的预期目的之外使用，准备睁开眼睛，但记得戴上眼睛保护装置，因为一些木屑看起来可以达到逃逸速度。让这个看起来像猿的住所，而不是鸟舍的关键是颜色的选择和装饰技术。从结果来判断，对比步骤，做出这种口径的模型是行家的标志。

如果你想尽情享受[木制大金刚](https://hackaday.com/2012/09/22/mechanical-donkey-kong-features-laser-cut-mario/)，我们可以满足你的愿望，但如果你更喜欢真人大小的树屋，这可能[会激发你的想象力](https://hackaday.com/2011/02/21/amazing-backyard-rocket-ship-tree-house/)。
