# 重现了费里斯·布勒休假时的对讲机

> 原文:[https://hack aday . com/2021/06/11/recreating-the-intercom-from-ferris-buellers-day-off/](https://hackaday.com/2021/06/11/recreating-the-intercom-from-ferris-buellers-day-off/)

《费里斯·布勒的休息日》是一部流行文化经典，也是那个时代杰出的青少年电影之一。值得注意的是，名义上的角色费里斯自己也是一个黑客，这部电影展示了几个少年用来逃离一天学校的装置。其中就有对讲机，[【亚伦】用现代技术忠实地再现了它。](https://www.youtube.com/watch?v=9Vd-yOBCuxE)

对于那些没有看过这部电影的人来说，对讲机被连接到一个卡带播放器上，以假装对任何在 Ferris 不在时参观过房子的人做出可信的反应。[Aaron]没有用传统的方式，而是用一个装有声音芯片的 ESP32 来制作他的复制品。当访客按动对讲机时，它会播放电影中的声音片段，同时也会向[亚伦]房子里的另一个 ESP 微控制器发出信号，让他知道有人来访。

该建筑是对经典电影的迷人致敬，而且更有趣的是[亚伦]努力使它看起来也是一部分，选择在一个正确的时期内建造内部通信外壳。然而，为了避免那些没有看过这部电影的人感到困惑，他小心翼翼地放置了一个标志，以澄清对讲机并不像它看起来那样。

我们以前也见过其他类似的电影——《星际迷航》中的这个版本。休息后的视频。
