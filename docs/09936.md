# 3D 打印浓缩咖啡零件

> 原文:[https://hackaday.com/2021/05/15/3d-printing-espresso-parts/](https://hackaday.com/2021/05/15/3d-printing-espresso-parts/)

事实上，任何爱好都有无尽的陷阱，每个角落都有新的细节需要学习。养蜂、微控制器、自行车和园艺(仅举几个例子)都是如此，但那些参与复杂的咖啡烘焙和酿造的人将这一细节发挥到了极致。制作咖啡的方法数不胜数，既有虔诚的追随者，也有反对者，而且每种方法都有其独特的设备。为了探索其中的一种方法并酿造出完美的浓缩咖啡，[Eric]求助于他值得信赖的 3D 打印机和一些压缩气瓶。

[![](../Images/d6bb4bb9072f3b838bed4de391049fb2.png)](https://hackaday.com/wp-content/uploads/2021/05/3dpespresso_detail.png) 一台浓缩咖啡机利用高压迫使热水通过精细研磨的咖啡。这种压力通常由电泵产生，但也有手动咖啡机。这些需要能够承受高压力的昂贵部件，因此[Eric]没有建造带杠杆的重型机器，而是转向压缩二氧化碳来提供所需的高压。

为了建造压力/冲泡室，他 3D 打印了除盛放咖啡的金属篮之外的大部分部件。3D 打印帽需要承受大约 9 个大气压的压力，因此它相当厚，用四个大螺栓固定，并容纳一个小二氧化碳罐，安全阀和压力表。

对[埃里克]的品味来说，与传统的浓缩咖啡机相比，这种精巧的装置能以最低的成本做出一杯上好的咖啡。一次性二氧化碳筒只增加了 0.15 美元的总成本，因为它的简单性和小尺寸，这是一个很好的权衡。他计划随着时间的推移改进设计，我们迫不及待地想看到他的发现。与此同时，我们将专注于确保[我们的咖啡豆是最高品质的](https://hackaday.com/2021/05/01/coffee-cupping-is-a-grind-spectroscopy-could-brew-better-beans/)，这样它们就可以为下一杯浓缩咖啡做好准备。
