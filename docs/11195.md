# 3D 打印您自己的坚固镜头盖

> 原文：<https://hackaday.com/2021/08/31/3d-printing-your-own-sturdy-lens-caps/>

镜头盖对于保护昂贵的相机镜头免受损坏非常重要。灰尘、砂砾和其他讨厌的东西都会很快破坏照片的质量，如果你运气不好，甚至会永久损坏镜头。然而，镜头盖也很容易丢失。因此，能够自己制作是很有用的，而且【DSLR 数控 DIY】知道如何去做。

打印自己的镜头盖的好处是定制。不管你的镜头的大小和形状如何，当你 3D 打印你自己的帽子时，你可以设计它来适应。该视频还展示了能够将文本直接嵌入帽体的好处，因此您永远不会混淆哪个帽与哪个镜头相配。该帽使用来自活页夹的金属杆，以提供挂在透镜上所需的夹紧力。这是对一些随着时间推移变得越来越弱的活动铰链设计的改进。

总的来说，如果你有一堆镜头需要一个新的镜头盖，这可能是你的项目。这也可能比在网上寻找隐形眼镜的替换帽更便宜、更容易。或者，想想你可以用[花哨的镜头适配器](https://hackaday.com/2021/04/23/an-anamorphic-lens-adapter-for-very-pretty-video/)做些什么。休息后的视频。
