# 3D 打印布料比你想象的要容易

> 原文：<https://hackaday.com/2022/05/24/3d-printing-fabrics-is-easier-than-you-think/>

由编织线制成的传统纺织品是非常有用的材料。[Sara Alvarez]已经取得了一些成功[通过 3D 打印创造了类似织物的材料](https://www.youtube.com/watch?v=6MQSxgcn9i0)，尽管它们不完全相同，但它们有一些相似的属性，使它们独特而有用。

织物是由许多线编织成一个紧密的整体。[Sara]的 3D 打印织物是不同的，因为打印机不能轻易地将单个纤维编织在一起。取而代之的是，使用多种方法来创建类似的材料。

最简单的也许是锁子甲方法，在这种方法中，许多单独的小链环连接在一起，形成一种相对坚硬的材料。或者，可以使用 g 代码或精细建模来创建织物状图案，这些图案直接打印在柔性材料上，成为织物状片材。最后，填充方法利用切片器内置的代码来创建一个可以 3D 打印的图案，通过移除打印的顶层和底层来创建类似织物的材料。

[Sara]演示了使用切片器方法创建一个简单的“织物”样本，并演示了成品的质量。她还展示了可以利用这种技术的各种应用程序。

如果你是一个 3D 打印爱好者，也喜欢制作衣服和服装，考虑打印一些鞋子—[就像我们之前见过的这些](https://hackaday.com/2022/03/02/3d-printed-shoes-make-bigfoot-tracks/)。休息后的视频。
