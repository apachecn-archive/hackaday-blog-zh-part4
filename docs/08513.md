# 从零开始建造洗碗机

> 原文:[https://hack aday . com/2020/11/30/从头开始建造洗碗机/](https://hackaday.com/2020/11/30/building-a-dishwasher-from-scratch/)

[Billy]不喜欢洗碗，但也发现缺乏商业解决方案。市场上的选择根本不适合他的炊具和餐具。他没有妥协，而是着手制造一台自己设计的洗碗机。

这个建筑由一大堆硬件组成，全部集中在一个相当大的塑料盆里。洗衣机的螺线管让水进入系统，它由浴盆底部的一个元件加热。然后，它被泵送通过花园洒水喷头，给盘子一个良好的全面喷洒。在洗涤周期结束时，排水泵将水排出，让所有东西都变干。ESP8266 和一组继电器用于运行该节目，用户通过按钮和小屏幕选择洗涤程序。

可能需要几年的时间才能组合在一起，但[比利的]洗碗机似乎可以完成这项工作。感兴趣的人可以在 Github 上找到文件，但是我们建议不要尝试这样的构建，除非你熟悉管道和主电源。[建造自己的洗碗机的另一个好处是，你不太可能需要修补它来抵御广泛的攻击](https://hackaday.com/2017/03/28/2017-the-year-of-the-dishwasher-security-patch/)——安全性反而取决于你。休息后的视频。
