# 病态节拍:用音乐和智能手机攻击生物安全室

> 原文:[https://hack aday . com/2022/11/23/sick-beats-using-music-and-smart phone-to-attack-a-biosafety-room/](https://hackaday.com/2022/11/23/sick-beats-using-music-and-smartphone-to-attack-a-biosafety-room/)

想象一部电影，场景设在绝密的生物武器研究实验室。这个恶棍穿着一套兔女郎服装，大步走进该设施的内部密室——生物安全室之一，这里只处理传染性最强、最致命的微生物。当他拿出电话时，紧张感增加了；他肯定会用它来影响一些戏剧性的攻击，或者引爆一个爆炸装置。相反，他调出他的播放列表……播放一首歌？这是什么样的小人？

事实证明，也许有人读过一篇关于利用音乐黑进生物安全室的可能性的新论文。这项工作是由加州大学欧文分校的研究人员[ Anomadarshi Barua]，【约纳坦·吉萨乔·阿查迈尔勒】和【穆罕默德·阿卜德 乌拉·阿尔·法罗克】完成的，重点关注各种设施中的负压室，但特别关注的是它们被用来防止病原体逃逸到整个世界的情况。

负压房间使用复杂的 HVAC 系统来保持室内压力低于室外压力，并竭尽全力保持这种压力。 这种房间的控制系统依靠压差传感器，它检测由薄隔膜隔开的两个端口之间的压差。由于两个端口之间的压力差引起的隔膜的偏转可以电容性地或压阻性地被感测。

问题是，振膜往往在音频范围内具有共振频率，这使得它们容易受到欺骗。用音频扫描评估了几种不同的常用传感器，显示了 700 至 900 Hz 的共振最佳点。这对于嵌入音频轨道来说是正确的，允许攻击者隐藏在简单的视线中——或者声音中，视情况而定。以这种频率调整传感器可能会说服控制系统进行调整，从房间中去除空气及其包含的任何病原体。剩下的你可以想象。 

我们已经变得非常喜欢寻找和报道一些最古怪的侧道攻击媒介，比如[薯片袋](https://hackaday.com/2020/03/09/fear-of-potato-chips-samy-kamkars-side-channel-attack-roundup/)和 [clicky 键盘](https://hackaday.com/2022/05/06/audio-eavesdropping-exploit-might-make-that-clicky-keyboard-less-cool/)。这一攻击尤其可怕，因为它看起来更有道理，风险也更高。
