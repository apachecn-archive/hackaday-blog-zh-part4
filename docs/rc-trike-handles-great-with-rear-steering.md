# 钢筋混凝土三轮车处理伟大的后转向

> 原文：<https://hackaday.com/2019/07/02/rc-trike-handles-great-with-rear-steering/>

有三个轮子的小型机器人非常多。最典型的配置是让两个前轮驱动并以滑移转向配置使车辆转向。第三个轮子通常是一个简单的脚轮。然而，这不是唯一的出路，[和[markus.purtz]已经构建了一个以不同方式做事的版本](https://www.instructables.com/id/RC-Trike-With-Rear-Steering-Wheel/)。

该建筑是一个小型无线电控制的 FPV 三轮车。与通常的滑移转向设置不同，后轮安装在一对水平轴承上，这使得后轮可以左右转动。伺服系统用于控制后轮位置，一对横拉杆端用于将喇叭连接到后转向总成。这也不是唯一的非传统设计选择——磁铁被用来将顶板固定在车辆底盘上，而不是螺丝或夹子。对于视频，用户可以安装一个小型的专用 FPV 相机，或附带安装的 GoPro。

没有任何代码或控制细节发布，我们不能 100%确定它是如何工作的。然而，从视频来看，似乎两个前轮以相同的速度行驶，转向仅由后轮控制。当在光滑的路面上行驶时，这一点很明显，在这种路面上，可以看到车辆在转弯时滑动。虽然这种设置不太可能比带有脚轮的简单差速转向系统有更多优势，但它确实表明后转向系统本身也是有效的。

它展示了在设计中使用合适的轴承和机械零件的好处。随着今天的在线市场，它从来没有更容易找到你需要的东西。对复制设计感兴趣的人可以在 Thingiverse 上找到零件。与此同时[小 FPV 机器人仍然很受欢迎](https://hackaday.com/2019/02/11/little-fpv-bot-keeps-it-simple-with-an-esp32/)，我们确信我们会继续看到它们的到来。休息后的视频。
