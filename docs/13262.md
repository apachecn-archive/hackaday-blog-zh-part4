# 修复每一个破损的通道，让这个游戏重现生机

> 原文:[https://hack aday . com/2022/04/05/fix-every-broken-via-to-return-this-game-to-life/](https://hackaday.com/2022/04/05/fix-every-broken-via-to-return-this-game-to-life/)

我们都知道水在错误的地方会对电子产品造成严重破坏，因此我们可能都有过无法修复的设备损坏。[Solderking]应该扔掉任天堂口袋妖怪红宝石墨盒中被水损坏的 PCB 吗？他当然应该这么做，但是当面对一块所有通孔都被腐蚀的电路板时，他选择了一条不太明显的道路，修复了它们。

除了视频中一些非常精细的焊接之外，没有什么出乎意料的。他拆下零件并尝试返工，但重新组装的主板无法启动。所以他又把它们拿走了，这次是把它们打磨成铜。接下来是对电路板上的每一个小瓶进行修复，将细线穿过小孔插入海绵中，焊接顶部，然后将顶部翻转过来，在另一侧固定密集的电线。固定 ROM 导致一个相当具有挑战性的装配，包括以一个角度安装芯片和额外的电线到它的焊盘，这证明了这个故事的价值。这不是一个货币价值，而是坚持一些史诗般的返工，以实现最终启动的 PCB。当然，更换主板会更有意义。但这不是重点，对吧？
