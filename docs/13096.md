# 带着自制街机经典旋风转圈

> 原文:[https://hack aday . com/2022/03/20/going-around-in-circuits-classic-cyclone/](https://hackaday.com/2022/03/20/going-around-in-circles-with-homemade-arcade-classic-cyclone/)

经典街机游戏 *Cyclone* 吸引了许多玩家，以及他们的硬币，这要归功于它简单却令人上瘾的游戏性。最基本的形式是一束光绕着一个圆形轨道跑，然后玩家必须停在正确的地方。Arduino 爱好者[mircemk]将[制作成了这款游戏](https://hackaday.io/project/184467-diy-arduino-cyclone-game-with-ws2812b-led-ring)的家庭版，它可以让上瘾者永远玩下去，而不会耗尽硬币。

这个较小的版本没有街机机柜，而是有一个直立的 3D 打印环，可以容纳 60 个 WS2812 LEDs。环中心的另外六个作为计分器。底座中的 Arduino 驱动 led 并运行游戏，这是基于[oKeeg] 构建的早期迭代[。一个有趣的补充是一个大型自制的“街机按钮”，它足够大，足够坚固，可以承受沮丧的玩家对它的任何虐待。](https://www.instructables.com/CycloneArduino-LED-Game/)

复古风格的音效和闪烁的灯光序列让游戏有点街机的感觉，即使没有一个大柜子和成堆的硬币。像这样简单的 LED 游戏在任何家庭或办公室总是很吸引眼球；如果你喜欢这个，一定要看看其他的 LED 游戏，如[手持 LEDBOY](https://hackaday.com/2021/11/11/ledboy-is-a-retro-modern-handheld-game/) 、[一维地牢爬虫 TWANG](https://hackaday.com/2018/01/24/diy-dungeon-crawler-game-plays-on-single-led-strip/) 和这个 [LED 赛车游戏](https://hackaday.com/2021/08/12/virtual-racers-battle-it-out-on-portable-ws2812-track/)。
