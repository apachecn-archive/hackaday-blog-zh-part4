# 老学校紧固件教程是铆接

> 原文：<https://hackaday.com/2021/09/10/old-school-fastener-tutorial-is-riveting/>

不管你是在一起制作、修理还是砍东西，我们都需要紧固件。螺钉、螺母和螺栓以及抽芯铆钉有时很方便。各种树脂甚至热胶都同样有用。然而在某些情况下，我们找不到合适的紧固件，我们需要另一个锦囊妙计。

[罗伯特·默里·史密斯]发现自己处于这样一个位置。他的目标是将两面都需要抛光的两片铝连接起来。胶水、铆钉、螺丝、螺母或螺栓都不合适。[Robert]总是有很多新的和旧的想法，他采用了一种老式的紧固件，正如他在视频“[如何制作和使用铆钉](https://www.youtube.com/watch?v=Gn0DqqQTrqk)”中所解释的那样。

在 break 下面的视频中，[Robert]详细介绍了如何用 5 毫米(3/16 英寸)的扁钢制作一个简单的铆钉模具，用黄铜棒制作铆钉，然后用平头铆钉连接两片铝。他使用的简单工具使得任何人都可以用丙烷火炬、老虎钳、一些基本工具和简单的羊角锤来使用这项技术。我们也欣赏[Robert]对冷铆接、热铆接以及根据需要对铆钉进行退火的讨论。

铆接不仅是一项有着数千年历史的技术，它在工业革命期间的进步和应用也使得原本不可能存在的技术成为可能。Hackaday 自己的[Jenny List]做了一篇关于 2018 年铆钉的[精彩报道，你一定不想错过！](https://hackaday.com/2018/08/20/the-forgotten-art-of-riveted-structures/)
