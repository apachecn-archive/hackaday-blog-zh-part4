# 木制坦克的运动取决于铰链

> 原文：<https://hackaday.com/2020/09/09/wooden-tanks-movement-hinges-on-hinges/>

当我们第一次看到这个坦克时，我们觉得它很酷。侧面是未上漆的 1/2 英寸(12 毫米)胶合板，所以它不浮华。这些尺寸来自于谷歌搜索二战时期的海泽，并将它们缩放到 1:6。让我们大吃一惊的是[Bret Tallent]对你在五金店或自行车店里能找到的零件的利用程度。他使用电动自行车的双马达，轮子看起来像替换的购物车轮子。最好的部分是踏板，这是几十个铰链，用成对的螺栓和尼龙螺母固定。知道修理你的宝宝不会比骑自行车走得更远，这是令人欣慰的。

我们不知道是什么让[Bret]走上人行道优越感的道路，但我们怀疑他和我们一样被禁锢着，并寻求表达自己。Mini-Hetzer 没有得到 Power Wheels 的许可，也永远不会得到许可，因此它可能不会很快成为一项业务。有一个完整的画廊，从一个空的胶合板底座开始，照片讲述了这个院子是如何发展到这一步的故事。有计划增加一个彩弹枪和流媒体视频，所以我们建议你今年不要乱动他街区的南瓜灯。参观一下他的画廊，看看你是否会受到启发，也从五金店中拼凑出一些巧妙的东西。然后，[给我们讲讲](https://hackaday.com/submit-a-tip/)。

另一个有创意的黑客用[木材做他们的坦克车体和踏板](https://hackaday.com/2018/07/09/on-the-right-tracks-electric-wheelchair-guts-find-new-life-as-tank/)。如果你喜欢你的履带式车辆的功能，我们有一个打算[滑行小型飞机在停机坪](https://hackaday.com/2018/06/19/an-arduino-powered-tank-built-to-pull-planes/)。
