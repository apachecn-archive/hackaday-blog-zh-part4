# 通过建造一个过度设计的滑板来测试碳纤维增强细丝

> 原文：<https://hackaday.com/2019/12/03/testing-carbon-fibre-reinforced-filament-by-building-an-over-engineered-skateboard/>

在过去的几年里，FDM 3D 打印机的细丝取得了突飞猛进的进步，碳纤维(CF)增强细丝正在成为一种常见的景象。非凡机器人[James Bruton]得到了一些 CF 增强的 PLA，并最终建造了一个完全过度设计的 3D 打印滑板。(视频，下面嵌入。)

[James]首先用一个 0.5 毫米和一个 1.2 毫米的大喷嘴打印了一些试件，有和没有 CF，他对这些试件进行了悬臂偏转测试。有碳纤维的那块比没有碳纤维的要硬 20%。

[James]然后用 CF PLA 和增韧 PLA 的交替部分建造了一个非常坚固和看起来很酷的滑板甲板，总共 2.7 千克的细丝。它非常坚固，所以在装上一套卡车和轮子后，他在当地的一个滑板公园做了一些温和的骑行，在那里它没有任何问题地存活了下来。他承认这完全是过度设计，但他指出，甲板上的内部空腔是长板上电池的最佳位置。

牢记 3D 打印的优点和缺点，从头开始设计一些东西，会产生一些非常有趣和创新的设计，这是一个完美的例子，我们希望看到更多这样的例子。我们展示了许多[詹姆斯]的项目，包括他为[马克·罗伯]制作的[遥控保龄球](https://hackaday.com/2019/11/24/cheating-at-bowling-the-hacker-way/)和他令人印象深刻的[开放式狗](https://hackaday.com/2018/06/11/james-bruton-is-making-a-dog-opendog-project/)和[星球大战机器人](https://hackaday.com/2016/12/10/speed-run-james-brutons-star-wars-builds/)。
