# 生活的实时字幕

> 原文：<https://hackaday.com/2022/06/08/live-subtitles-for-your-life/>

个人平视显示器是一项技术，它的时代现在应该已经到来，但尽管有谷歌眼镜等尝试，但它一直拒绝流行起来。虽然[巴塞尔·萨利赫]的 CaptionIt 项目中有一个有趣的可能性，[一个为日常情况提供字幕的平视显示器](https://www.youtube.com/watch?v=DIHrooFKeLU)。

硬件是一个小小的 I C 有机发光二极管屏幕，带有一个反射镜和一个连接到一副眼镜的 3D 打印支架，据称它可以与几乎任何 ARM v7 SBC 一起工作，包括最近的 Raspberry Pi 板。它使用 [vosc](https://alphacephei.com/vosk/) 语音识别工具包从 USP 音频设备读取音频，结果文本显示在屏幕上。

在休息时间下面的视频中显示了该设备的运行情况，没有亲自尝试过，我们无法评论它的实用性，但除了新奇之外，我们可以看到它作为辅助工具可能会产生重大影响。但是，我们希望看到电子巴别鱼和翻译软件一起发展，这样无意中却令人捧腹的国际误解就可以被所有人分享。

普通读者会知道我们过去给你带来了很多愚蠢的事情。
