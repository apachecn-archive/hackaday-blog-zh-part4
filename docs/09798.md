# 名片上的激光切割阻焊膜

> 原文:[https://hack aday . com/2021/04/09/名片上的激光切割阻焊膜/](https://hackaday.com/2021/04/09/laser-cut-solder-masks-from-business-cards/)

有很多方法可以在家里制作印刷电路板，但对于某些功能来说，最好还是去电路板商店。这些功能在数量上继续减少，但不是很多人能在家里建造像四层板这样的东西。对一些人来说，添加阻焊膜可能是这些功能之一，但如果你碰巧有一台激光切割机和几张名片，那么[这个过程对房屋建筑商来说也是触手可及的](https://www.youtube.com/watch?v=022UD5y8Iww)。

[Jeremy Cook]很幸运身边有一台激光切割机，他有一个想法，用它来帮助改善他的表面贴装焊接工艺。通过用激光切割机将阻焊层切割成名片，它可以被固定在 PCB 的顶部，然后用作模板，比其他方式更容易添加焊膏。它极大地减少了在这部分过程中花费的时间，特别是当涉及多个电路板时，因为模板可以多次使用。

虽然激光切割机肯定不是一个严格的要求，但它肯定有助于像 X-acto 刀这样的东西。[Jeremy]还指出，这个过程有时是用透明胶片甚至 Kapton 来完成的，[我们以前也见过几次](https://hackaday.com/2014/06/22/one-off-kapton-solder-masks/)。
