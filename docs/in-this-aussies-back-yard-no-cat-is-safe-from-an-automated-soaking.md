# 在这个澳大利亚人的后院，没有一只猫能幸免于自动浸泡！

> 原文：<https://hackaday.com/2019/05/14/in-this-aussies-back-yard-no-cat-is-safe-from-an-automated-soaking/>

我们 Hackaday 的一些人是爱猫人士，但我们也明白，在郊区的后花园里有太多不需要的猫可能会令人烦恼，几年前，我们介绍了一个来自澳大利亚 YouTuber(克雷格·特纳)的项目，在这个项目中，他创造了一种运动检测喷水器，用作相对无害的驱猫剂。现在他带着更新版本的回来了，这个版本更光滑，也更容易制作。

其核心是相同的 PIR-turns-on-water 操作，但这一次有一个电磁阀和专用喷嘴，而不是汽车中控锁执行器和草坪喷雾器。做电子工作是现成的 PIR 模块，因此不再需要侵入安全 PIR 检测器。添加一些管道部分和带有一点热胶的聚四氟乙烯带，结果是一个更加专业和流线型的设备。该视频给出了一个完整的结构，尽管我们注意到他忽略了强调他的保护二极管的极性，所以如果你以他为榜样，请保持警惕。

所以，如果你不喜欢你的邻居不断提供免费的猫咪陪伴，那么现在你可以让他们打包走人了。该项目的最新视频在休息时间以下，但如果你在寻找原始的，那么你可以回到我们当时的报道。
