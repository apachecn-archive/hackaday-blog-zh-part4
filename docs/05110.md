# 一把由垃圾制成的惊人的射线枪

> 原文：<https://hackaday.com/2019/12/12/a-stunning-ray-gun-built-from-junk/>

如果说科幻小说中有什么典型的武器，那一定是射线枪。从他令人印象深刻的垃圾收藏中，建造了这个独一无二的尤物。它围绕着一个老式胶木焊枪、一个真空管和一个便携式火炉燃烧器，所有这些都有助于营造梦幻般的中世纪外观。

内部是 MAKE 的射线枪音效电路的略微修改版本:它从 lo-fi 合成器制造商最喜欢的 IC 40106 hex 反相施密特触发器中挤压方波。[lonesoulsurfer]能够重复使用焊枪的触发器来启动皮尤-皮尤-皮尤，他可以用电位计调整死亡射线的输出。这把枪由世界上最大的虚拟拍卖行提供的旧手机电池和组合锂离子充电器/升压模块驱动。爆炸过去休息观看建设视频。

如果一个小小的绿色 LED 对你来说还不够，也许你会更喜欢这个光绘枪。
