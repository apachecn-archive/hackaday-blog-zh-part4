# 超级马里奥晒伤模式照亮了一个游戏立方体的最爱

> 原文：<https://hackaday.com/2021/03/27/super-mario-sunburn-mod-shines-up-a-gamecube-favorite/>

当谈到 3D 马里奥游戏时，超级马里奥阳光总觉得有点欠火。无论是不稳定的相机控制，加重硬币配额，还是包含一个有知觉的超级酒鬼，这款游戏都没有完全满足粉丝的期望。为了洗刷这一名声[【韦德】创造了一个 mod 来复兴经常被贬低的 GameCube 游戏](https://gamebanana.com/maps/209335)。经过两年的制作，超级马里奥日炙打破了超级马里奥阳光大开新的水平，更多的硬币，和现代开放世界游戏的自由。收集游戏中的闪光收藏品不再自动将马里奥扭曲回岛屿中心，而是允许马里奥继续填充这些口袋。

为了应用晒伤 mod 补丁，任天堂 GameCube 的超级马里奥阳光需要一个干净的 rip。抓取 GameCube 光盘最简单的方法实际上是用任天堂 Wii — [只要它能通过家酿频道](https://dolphin-emu.org/docs/guides/ripping-games/)运行 CleapRip。有了干净的游戏镜像，晒伤补丁可以通过运行 Delta Patcher 应用到 Windows 上。从那里可以通过可选的高清纹理包，甚至是真正的任天堂硬件模拟器来欣赏晒伤的图像。像这样一个全面的国防部肯定值得一些 WaveBird 时间。

[韦德 mod 的到来对于许多马里奥粉丝来说是一个关键时刻。去年年底，任天堂发布了一款名为《超级马里奥 3D 全明星》(Super Mario 3D All-Stars)的 3D 马里奥游戏。该版本带来了最轻的触动，未能提供一个合适的现代化的超级马里奥阳光。该公司甚至不允许玩家在 16:9 宽屏下玩([不像晒伤](https://www.moddb.com/mods/super-mario-sunburn/downloads/super-mario-sunburn-manual))。3 月底，任天堂将把超级马里奥 3D 全明星塞进“布瑟的金库”，从而将其从商店货架上撤下。更有理由尝试一下超级马里奥晒伤。
