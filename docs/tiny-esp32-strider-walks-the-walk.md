# 小小的 ESP32 Strider 走着走着

> 原文：<https://hackaday.com/2021/09/06/tiny-esp32-strider-walks-the-walk/>

对于机器人来说，轮子可能是最简单的移动方式，但是步行器看起来绝对更令人满意。由一个 ESP32 摄像板控制的小小漫步者步行者【陈亮】[肯定就是这种情况。](https://www.instructables.com/Strider-Camera-Robot/)

Strider 机制可能看起来类似于 [Strandbeest](https://hackaday.com/2018/08/16/theo-jansen-invents-a-faster-simpler-wind-powered-strandbeest/) 步行者，但它将脚抬得更高，允许它穿越更崎岖的地形。[陈]的小型 3D 打印版本由一对齿轮传动的 N20 电机驱动，每侧有三条腿。ESP32 相机板允许使用 WiFi 进行控制和 FPV 视频馈送，电源来自 14500 LiFePO4 电池。马达、腿部机构和轴承所需的宽度意味着机器人相当宽，以至于它可能会卡在相机视野之外的东西上。[陈]正在通过使用连续旋转伺服系统和线驱动轴使其变得更窄。

我们已经看到了许多腿步行器的不足或重复，如由[韦德]和[本·瓦格]开发的[trot bot 和 Strider](https://hackaday.com/2018/11/23/move-over-strandbeest-heres-strider/)机制，他们的[网站](https://www.diywalkers.com/walker-abcs.html)是未来建筑者的优秀资源。
