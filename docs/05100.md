# 边缘发光乒乓球拍点亮战斗

> 原文:[https://hack aday . com/2019/12/13/edge-lit-ping-pong-paddle-lights-up-the-fight/](https://hackaday.com/2019/12/13/edge-lit-ping-pong-paddle-lights-up-the-fight/)

[乔治]和他的同事喜欢在公司的乒乓球桌上发泄午餐时的一点蒸汽。我们可能会做同样的事情，除了只有我们对着墙，大部分的练习都是弯腰从地板上捡球。当他在他们的店里发现一块丙烯酸碎片时，[乔治]有了一个好主意，制作了一个带有公司标志的侧光球拍。

这种桨不仅看起来很酷，而且工作起来也很好，尽管与标准桨相比，它更重，表面更光滑。首先，[George]找到一个规定大小的桨轮廓，并将其导入 SolidWorks，然后为 led 和其他电子设备设计所有必要的切口。他还设计并印刷了符合人体工程学的把手来保护货物。

继续手头上的东西这个主题，[George]使用了通孔 led，并挖掘了他们周围的大量电池夹和弹簧来设计原型，而不是用 SMT LEDs 和可充电电池组来制作它。戴上吸汗带，因为休息后我们会提供制作视频。

我们在这里看到的乒乓球比球拍多，这可能是因为它们是很好的 LED 扩散器。
