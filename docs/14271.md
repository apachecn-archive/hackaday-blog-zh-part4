# 黑掉一个玩偶盒是非常令人惊讶的

> 原文:[https://hack aday . com/2022/07/20/hacking-a-jack-in-the-box-to-be-extra-surprise/](https://hackaday.com/2022/07/20/hacking-a-jack-in-the-box-to-be-extra-surprising/)

盒子里的杰克第一次就够吓人的了。然而，如果你以前见过小丑跳出来，它不会有同样的影响。[弗兰克林斯坦]决定用另一种有效载荷*替换掉小丑，这将带来他所追求的恐惧。*

 *在玩具内部，一个 Arduino Nano 在表演。它与一个空气喇叭配对，安装在一个特殊的框架中，与一个遥控伺服系统一起。当时机成熟时，遥控伺服系统会按下喇叭，发出巨大的噪音。还有一个五彩纸屑爆破器，内置一个充满压缩空气的小室。当一个螺线管被释放，压缩空气通过一个装满五彩纸屑的漏斗冲出，喷射到房间里。

当玩具上的曲柄转动时，就会播放典型的歌曲。当盒子的盖子打开时，它会释放一个开关，Arduino 就会发出五彩纸屑和喇叭。这对[弗兰克林斯坦]本人来说已经够令人震惊的了，对那些期待玩具中典型的蹦蹦跳跳的小丑的人来说更令人惊讶。

我们希望看到一个更令人讨厌的版本，至少在一分钟内不关闭喇叭。我们以前也见过他们在一些很棒的万圣节黑客活动中被雇佣。休息后的视频。
