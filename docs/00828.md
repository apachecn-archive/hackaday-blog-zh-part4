# 蚂蚁，泥土，雨水，还有不肯放弃的准将 64

> 原文:[https://hack aday . com/2018/10/03/ants-dirt-rain-and-the-commodore-64-that-won-won-quit/](https://hackaday.com/2018/10/03/ants-dirt-rain-and-the-commodore-64-that-wouldnt-quit/)

一些电子设备是为最恶劣的条件而制造的。凭借坚固的钢制外壳、防风雨垫圈和穿过密封压盖的电缆，这些机器可以承受大自然母亲扔给它们的最坏情况，摆脱灰尘、泥土、雨水和冰。然而，从家用个人电脑时代开始的消费级电脑显然不是这样的机器。

制造成本很高，很容易受到溢出的激浪的影响，那个时代受到任何虐待的机器很少有人能活着讲述这个故事。这个勇敢的准将 64C 却不是这样，它在恶劣天气中存活了几十年。正如[Adrian Black]在下面的视频中所述，这台机器是在俄勒冈州的一个垃圾场上，与真人秀节目非常喜欢的那些“采摘者”之一的其他好吃的东西堆在一起。这台机器是一场灾难。它没有被[浸过油](https://hackaday.com/2018/05/25/bringing-a-vic-20-back-from-an-oily-grave/)，但是它装载了松针和一个蚁群。然而，最糟糕的是铁锈。射频屏蔽有些地方已经腐蚀成粉末，到处都是红色的锈迹。[阿德里安]毫不气馁，给机器洗了个好澡，先是在水中，然后在异丙醇中。Deoxit 的自由应用有助于头部连接，足以看到机器奇迹般地启动。这需要一些欺骗，特别是 6526 I/O 控制器，但[阿德里安]最终能够让主板上的一切工作，甚至是声音芯片。

这台机器幸存下来是因为好的工程还是好的运气值得商榷，但看到它起死回生是一种享受。我们希望全面修复正在进行中，不仅仅是为了弥补几十年来的忽视。
