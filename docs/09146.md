# 简单的 3D 打印七段显示器

> 原文:[https://hack aday . com/2021/02/02/simple-3d-printed-seven-segment-displays/](https://hackaday.com/2021/02/02/simple-3d-printed-seven-segment-displays/)

7 段 LED 显示屏是革命性的，最终提供了清晰、易读和低功耗的数字显示解决方案。我们现在有很多其他便宜的显示器选择，但有时你只需要旧的零到九，在一个大的，可见的包装中，启动。在这种情况下，[考虑制作一套 3D 打印的七段显示器](https://nerdiy.de/en/howto-elektronik-sieben-segment-ziffer-pxldigit-aus-ws2812-leds-bauen/)。

该建筑由一个 3D 打印框架组成，每个部分包含两个 WS2812B 可寻址 led。然后，将每个 7 段组件连接起来，使它们能够以菊花链形式连接，将数据传递给链中的下一位数字。纸张用于扩散 led 以获得更平滑的外观，每个数字都印有白色 3D 打印封面，以进一步传播光线并提供干净的外观。

基于 WS2812Bs，很容易用任何微控制器或配备 GPIO 的 Linux 板驱动这种显示器。我们喜欢大而漂亮的展示——[,越艺术化越好](https://hackaday.com/2020/02/10/huge-seven-segment-display-made-from-broken-glass/)。休息后的视频。
