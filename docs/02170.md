# 野马破折号成为书架艺术品

> 原文：<https://hackaday.com/2019/02/24/mustang-dash-becomes-bookshelf-art-piece/>

尽管大多数通用仪表在过去 60 年中保持不变，但过去的汽车仪表板使用了与今天非常不同的技术。电缆驱动的速度计是常见的，以及里程表的机械驱动。燃油和温度表通常直接连接到它们的发送器上，一些油压表实际上有一条油管一直通到仪表板后面。现在，大部分事情都通过 CAN 总线处理，[这启发了【托马斯】基于书架的野马构建。](https://www.rc-cam.com/forum/index.php?/topic/4169-ford-mustang-instrument-cluster-project-introduction/)

这个项目背后的想法是建立一个很好的书架艺术，使用一个现代的可以驱动的野马仪表板。通过研究和反复试验，[托马斯]能够找出与 2009 年野马仪表板接口所需的 CAN 消息。在这个过程中有无数的小问题——[ Thomas]不得不 3D 打印自己的连接器，刷新 CAN 总线接口，并进行多次有根据的猜测才能让事情正常运行。

dash 与一个带有 MP3 屏蔽和 30 瓦音频系统的 Arduino 相结合，该系统既提供驱动 dash 的 CAN 信号，也提供野马在城镇中奔跑时必须的声音效果。它最后配备了一个点火钥匙开关和 3 个传统野马颜色的 LED 灯按钮。

这是一个有趣的构建，它很好地展示了与现代汽车子系统接口所需的基本工具和技术。[抢救一个仪表组也是给你的家庭赛车 sim 增加沉浸感的好方法](https://hackaday.com/2012/04/20/real-bmw-dash-cluster-for-your-racing-games/)。休息后的视频。
