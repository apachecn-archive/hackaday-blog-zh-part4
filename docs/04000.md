# 铅模型使 LED 立方体更容易建造

> 原文:[https://hack aday . com/2019/08/25/lead-former-makes-led-cubes-a little-easy-to-build/](https://hackaday.com/2019/08/25/lead-former-makes-led-cubes-a-little-easier-to-build/)

毫无疑问，一个制作精美的 LED 立方体很有吸引力；同样，毫无疑问，构建它们会非常痛苦。毕竟，工作量是你希望每面都有 LED 数量的立方，让我们面对现实吧 LED 立方体越大越好。如何应对所有这些乏味的销售线索形成过程？

[TylerTimoJ]的解决方案是[一个定制设计的销售线索形成工具](https://www.youtube.com/watch?v=zU-193ZnEQs)，我们不得不说它给我们留下了深刻的印象。[他的 LED 立方体](https://hackaday.com/2016/05/22/real-time-driving-of-rgb-led-cube-using-unity3d/)使用分立的 RGB LEDs，这种 LED 有四根引线，每根引线都通过焊接到电线上悬浮在空中。为了制作这样一个电路雕塑作品所需的整洁外观，导线必须修剪并弯曲成合适的角度，手工制作时这确实是一项繁琐的工作。他的工具有伺服控制的夹钳，可以夹紧导线，螺线管驱动的导线成形器从下方进入，将每根导线弯曲到合适的程度。lead former 及其配套的 trimmer 显然在[TylerTimoJ]获得一切正确之前经历了许多迭代，但我们要说的是，能够处理数千个 led 而不感到单调乏味可能值得付出努力。

我们期待着这个工具将实现的巨大的 LED 立方体。也许这个数控弯丝机和自动剪丝机会对支撑线派上用场？
