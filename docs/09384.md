# 用电动冰鞋切碎冰块

> 原文：<https://hackaday.com/2021/03/01/shredding-the-ice-with-powered-skates/>

小而大功率的无刷电机的出现有助于所谓的微型机动车辆的发展。但是如果你的通勤包括穿越一个冰冻的湖，你可能会发现选项有点缺乏。幸运的是，来自[RCLifeOn]的[Simon]现在为你提供了一个解决方案，那就是[电动溜冰鞋](https://www.youtube.com/watch?v=aUBJxFs8hko)。

[Simon]使用 3D 打印支架将 outrunner 无刷电机安装到一双溜冰鞋的背面。马达的旋转外壳被用作车轮，一串螺栓拧在上面，用于挖掘冰层并提供牵引力。起初[西蒙]试图使用一对钢筋混凝土汽车弹簧来保持电机与冰接触，但弹簧力不足以完成这项任务，所以他最终刚性安装电机。在冰面上从静止状态获得适当的牵引力仍然很棘手，所以他最终向后倾斜推动马达向下，这也使他失去了平衡，限制了实际的加速度。跟踪问题最明显的解决方案似乎是更强的弹簧，但我们假设他手头没有。电池装在背包里，电缆连接到冰鞋，无线电动滑板控制器用于油门控制。

这些溜冰鞋的明显风险是，如果你摔倒，钉头马达会无意中变成绞肉机。这看起来仍然是一个有趣的项目，我们不介意尝试一下这些溜冰鞋。

[西蒙]喜欢摆弄无刷电机，做过从[喷气动力冲浪板](https://hackaday.com/2020/09/08/electric-surfboard-doesnt-need-waves/)到[陀螺稳定遥控“摩托车”](https://hackaday.com/2019/09/06/who-needs-four-wheels-when-youve-got-a-gyro/)的各种东西。
