# 用于 PCB 清洗的廉价替代溶剂

> 原文：<https://hackaday.com/2020/04/29/cheap-alternative-solvents-for-pcb-cleaning/>

如果你习惯在焊接后用异丙醇清洗 PCB，你可能有一大罐这种东西。如果你没有，你可能就不走运了，因为新冠肺炎疫情已经把 IPA 清除出了零售市场。但是不要担心:根据你居住的地方，[替代的 PCB 清洗溶液可能就在离你最近的汽车配件商店](https://www.youtube.com/watch?v=d3Qmm6Tegb4)附近。

[Steven]寻找一种更便宜、或许更容易获得的替代品来代替他常用的专用焊剂清洁剂，这使他尝试在一些测试板上使用汽车制动清洁剂。他怀疑它们可能含有丙酮，丙酮容易对 PCB 上的阻焊和丝网印刷产生不良结果，所以一些测试是有序的。他尝试的品牌是 norm fest*Bremsenreiniger*MC-1，这是一个德国品牌，根据其[的安全数据表](http://www.sdbpool.de/Pdf/DownloadDfpByArtNr?t=f8850c13a0486b9b93d4b26ad22738ee&artnr=2897333500&lc=de_de)只含有碳氢化合物，如烷烃、丁烷和丙烷。它做了一个很好的工作，几乎没有附带损害。

在下面的视频中，[Steven]又看了几个有类似结果的品牌，我们被他的结果所鼓舞，去检查为美国市场生产的刹车清洁剂。唉，几乎所有廉价易得的气雾剂都以丙酮为主要成分，混合了甲醇、乙醇和各种成分，这些成分加在一起可能会让一天变得糟糕。我们能找到的唯一一个在美国销售的不含丙酮的品牌是[Keller-heart](https://www.kellerheartt.com/collections/brake-cleaner/products/non-chlorinate-brake-cleaner-case-14-oz)，它在[的 SDS](http://www.sdbpool.de/Pdf/DownloadDfpByArtNr?t=f8850c13a0486b9b93d4b26ad22738ee&artnr=2897333500&lc=de_de) 上只列出了石脑油和乙醇。可能还有其他的，但是一定要测试你发现的任何东西。

当然，气溶胶溶剂并不是清洗 PCB 的唯一方法。超声波清洁器做得很好，正如[Steven]发现的那样，[它们通常对大多数部件都是安全的](https://hackaday.com/2019/12/05/how-safe-is-that-ultrasonic-bath-for-flux-removal/)。
