# 黑掉大自然的音乐家

> 原文:[https://hackaday.com/2018/10/20/hacking-natures-musicians/](https://hackaday.com/2018/10/20/hacking-natures-musicians/)

我们刚刚完成了 Hackaday 奖的乐器挑战，对于大多数项目来说，这意味着复制人类发出的声音，或者为人类创造音乐家。然而，音乐不仅仅是一个黎明就能创造的东西；世界被音景所包围，你只需要在乡间走一走就能听到。

为了她的 Hackaday 奖参赛作品，[Kelly]正在黑《自然》的音乐家。她在用晶体管和印刷电路板复制乡村的声音。这是模拟电子学的惊人之作，听起来也很棒。

最令人印象深刻的板[凯利]一直在工作的是大自然母亲板，一种不同动物电路的自然电子合唱。这完全是随机的，基于一个真正的随机数发生器，并使用一组晶体管和 555 个定时器来产生发送到压电的脉冲。这种电路对噪声非常敏感，在制作时[Kelly]发现她的 2N3904 晶体管并不都是一样的；他们中的一些人为噪音发生器工作，一些人没有。这是一个很难设计的电路，但结果是令人愉快的。

那么，模拟电子产品听起来能像充满蟋蟀的森林吗？令人惊讶的是，是的。[本次演示](https://vimeo.com/294641101)展示了几块布满晶体管、电容、电阻和 led 的试验板所能实现的功能。这是一个声音的电子雕塑，灵感来自弗吉尼亚乡村的夜间声景。你有蟋蟀、蝉、螽斯、青蛙、鸟和世界上所有其他非人类的音乐家。太美了。
