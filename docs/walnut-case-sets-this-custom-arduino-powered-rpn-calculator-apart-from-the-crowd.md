# 胡桃木盒将这款定制的 Arduino 驱动的 RPN 计算器与众不同

> 原文：<https://hackaday.com/2022/11/03/walnut-case-sets-this-custom-arduino-powered-rpn-calculator-apart-from-the-crowd/>

我们中有多少人拥有真正独一无二的日常工具？可能我们中没有多少人；看看你的桌子周围，翻出你的口袋，但通常情况下，你会发现你拥有的每样东西几乎都是这个星球上的其他人也能买到的。但如果你有这个装在木箱里的漂亮的定制 RPN 计算器，情况就不一样了。

这个来自[Shinsaku Hiura]，他通常用独特的机械时钟[和](https://hackaday.com/2022/07/30/whats-the-time-its-casinoclock/)[显示器](https://hackaday.com/2021/11/13/a-one-servo-mechanical-seven-segment-display/)让我们眼花缭乱。这个计算器解决了一个更实际的问题——市场上缺乏具有正确键盘感觉的 RPN 计算器，特别是他想要的大按键和轻触。恰当地说，构建从一个数字键盘开始，这个数字键盘一旦从 USB 接口中解放出来，就会被逆向工程处理，以找出矩阵是如何连接的。接下来，一个连接键盘和 Arduino 的定制 PCB 和一个 20×4 LCD 显示屏被磨出来，同时一个测试用例被设计和印刷以检查安装。最终的表壳由一块实心胡桃木磨制而成，并配有一个丙烯酸窗户，线条简洁，颜色悦目，外观鲜明。

至于计算器本身，下面的演示展示了它的运行速度。该代码很聪明，因为它通过将所有科学和工程功能隐藏在一个“秘密银密钥”后面来利用最少数量的可用密钥，这个“秘密银密钥”曾经是 equals 密钥，显然在 RPN 中是不需要的。向[Shinsaku]致敬，这是他办公桌上又一个漂亮又独特的东西。
