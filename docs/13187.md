# 用偏光器显示的应力

> 原文：<https://hackaday.com/2022/04/01/stresses-revealed-with-a-polariscope/>

压力有很多表现方式，至少在讨论材料科学的时候。混凝土中的裂缝是一个很常见的例子，但是材料中的任何灾难性故障通常都是由于一些无法承受的应力造成的。如果你有兴趣在这些应力对底层材料造成损害之前观察它们，看看[这个 DIY 偏光镜，它可以观察玻璃](https://www.youtube.com/watch?v=W29-6HUcfoU)和其他透明物体的内部应力。

偏振镜得名于它使用偏振光来观察透明物体(如玻璃)的内部结构。当偏振光以某种方式穿过玻璃时，应力显示为较亮的区域，这是由于受应力的玻璃将光弯曲回来进入视野。这个是由放置在 LCD 屏幕前的偏振滤光器构成的，用来显示完全白色的图像。当玻璃被放在屏幕和滤光器之间时，除非玻璃上有应力，否则通过偏振器看不到光。甚至在一个没有受力的玻璃管上施加一个力也能显示出这种效果，这个项目的创造者[高级修补]有几个其他的作品以惊人的细节显示了这种效果。

这种效果也可以在其他塑料材料的着色区域中观察到。这是一个有趣的工具，可以帮助任何经常使用玻璃的人，但它本身也很有趣，可以看到各种家居用品制造过程中留下的线索。我们已经看到了一些其他的调查方法来确定其他家居用品是如何大规模生产的，就像[这个项目分解了注射成型过程](https://hackaday.com/2017/12/17/the-miracle-of-injection-molding-how-does-it-work/)。
