# 又一次磁控管拆卸

> 原文：<https://hackaday.com/2022/12/22/another-magnetron-teardown/>

[电子人]对微波炉内部的电压和陶瓷有着健康的尊重。但他仍然鼓起勇气撕开一个盒子，给我们看里面的东西，并描述一些部件的特征。你可以在下面看到拆机的视频。

电压的危险是显而易见的。但是，里面还有一个陶瓷绝缘体。其中一些是由氧化铝制成的，但另一些是由氧化铍制成的。你可能不想吸入任何一种，但氧化铍，如果制成粉末，会导致严重的健康问题。显然，如果你决定撕开烤箱，你需要小心。当然，另一个危险是如果你把烤箱放回一起，并试图使用它。你需要确保所有的屏蔽都回到正确的位置。

视频显示了使用测试设备的几个组件的操作，在某些情况下，还显示了一些替代组件。LC 振荡器的动画非常容易理解。然而，当他实际上用旋转工具切入磁控管时，你真的可以看到该设备是如何工作的。一些动画让它更加清晰。

我们已经有几年没有看到磁控管拆卸了。你可以用磁控管做很多事情，从雷达到[真空沉积薄膜](https://hackaday.com/2019/07/23/vacuum-sputtering-with-a-homemade-magnetron/)。
