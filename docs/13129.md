# 乐高分拣真空击败了这个问题

> 原文：<https://hackaday.com/2022/03/24/lego-sorting-vacuum-defeats-the-problem/>

乐高最糟糕的是什么？大多数人都会同意这一事实，即那些鲜艳多彩的腹肌不知何故是你赤脚和坚实地面之间最邪恶的东西。【不必要的发明】从手柄上做了一百八十度大转弯，做了一个相当有用的发明——[一个乐高分类吸尘器](https://newatlas.com/around-the-home/lego-sorting-vacuum/)叫做 Suck It。

T2:嗯，从技术上来说，这是一个商店真空附件，但它做得很漂亮。【不必要的发明】从一台从未用过的机器和一个切成四分之一的透明塑料圆筒开始。然后，他设计了一些 3D 可打印支架，这些支架有两个作用:它们将圆柱体再次固定在一起，并在收集和按大小分类乐高积木的阶段这样做。

分类支架没有顶层和底层，它们都是分类孔和填充物，以实现最大吸力。但是等等，它变得更好，因为支架与嵌入的磁铁令人满意地卡在一起。唯一能让这个造型更好的是某种头部附件，它可以一次聚集更多的窄条。休息之后，请务必查看构建和演示视频。

当然，用这种方法，你还是要把各部分打开，把你的乐高放好。你可以直接用吸尘器把它们吸进盒子里。
