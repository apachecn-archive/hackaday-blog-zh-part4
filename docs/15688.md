# 摄影，立体的方式

> 原文：<https://hackaday.com/2022/12/19/photography-the-stereo-way/>

至少自 20 世纪 60 年代以来，大多数消费级音频设备都是立体声的，使听众能够以三维视角体验声音，就好像声音最初产生时他们就在现场一样。然而，立体摄影已经稍微落后于立体声音频趋势，大多数技术都只是昙花一现，或者需要笨拙的硬件才能完全体验。这种由 3D 摄影爱好者[团体](https://photo-3d.groups.io)生产的 [DIY 立体相机就不一样了，他们也有一些方法来观看 3D 照片，而不需要任何额外的硬件。](https://www.youtube.com/watch?v=wcuBFhz_7H8)

该相机使用两个成像传感器来产生立体图像。一个传感器是固定的，另一个在滑块上，允许用户调整任何特定照片所需的 3D 效果的“量”。[Jim]使用这款相机主要是为了进行微距摄影，这意味着他只需要两个传感器之间有几毫米的间距就可以达到理想的效果，但对于更远的物体，可以使用更大的间距。该相机使用双 Raspberry Pi 处理器、锂电池和触摸屏界面。它包括了一大堆功能，比如焦点堆叠，但是为了更全面地体验这个版本，我们强烈建议在休息之后看看视频。

至于观看照片，这些立体 3D 图像只需要一点点练习就可以观看。[本指南](https://stereoscopy.blog/2019/08/04/how-to-view-stereoscopic-3-d-images-basic-tutorial/)提供了一些简单的入门示例，虽然一开始感觉像是 90 年代后期的魔眼拼图，但很快就变得直观了。[另一个指南](http://www.minerbits.com/how-to-view-3D.html)在结尾也有一些更复杂的 3D 地图供练习。这比需要使用特殊的眼镜或某种可佩戴的 3D 浏览器进步了很多。[也有一些方法](https://hackaday.com/2012/02/27/make-any-photo-3d-using-the-gimp/)可以从普通 2D 相机拍摄的照片中创建 3D 图像。
