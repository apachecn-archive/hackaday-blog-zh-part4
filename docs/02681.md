# 便携式 Pi 电源包适合小型项目

> 原文:[https://hack aday . com/2019/04/09/portable-pi-power-pack-makes-for-petite-projects/](https://hackaday.com/2019/04/09/portable-pi-power-pack-makes-for-petite-projects/)

自从 Pi Zero 发布以来，已经有许多尝试来添加电源组。毕竟，手机电池的大小和 Pi Zero 差不多，增加一个 USB 充电端口并在 Pi 上焊接几根电线很容易。PiSugar 可能是我们见过的 Pi Zero 最可爱的电池组，它有多种与 Pi 兼容的帽子，能够成为小型显示器、键盘或任何其他小型便携式 Linux 机器有用的东西。

这个构建的核心是一个π0 大小的小电路板。连接到该板上的是一个 900mAh 电池，整个组件通过一组两个弹簧夹连接到 Pi Zero，弹簧夹与 Pi 背面的一对焊盘相匹配。将这两块电路板拧在一起，你就有了一个完美的无线解决方案来增加 Pi Zero 的功率。

但是皮苏加尔并没有就此止步。也有这样的情况，1.3 英寸的 LCD 显示器、2.13 英寸的 ePaper 显示器、有机发光二极管显示器、摄像头、4G 模块，以及只是提供来自 Pi GPIO 头的引脚的东西。这是一个完整的平台，如果你用白色塑料打印这些部件，它们看起来就像装满了电子产品和 Linux 优点的小方糖。

是的，你以前见过 3D 打印的 Pi 案例，但没有一个完整的平台可以在一个可扩展的平台上给你一个 Pi Zero，它可以放在你的口袋里，看起来像甜甜的蔗糖。
