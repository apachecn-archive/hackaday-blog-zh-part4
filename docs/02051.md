# 可下载的 3D 驾驶舱增强 FPV 赛车

> 原文:[https://hack aday . com/2019/02/17/可下载-3d-cockpits-enhance-fpv-racing/](https://hackaday.com/2019/02/17/downloadable-3d-cockpits-enhance-fpv-racing/)

![](../Images/d2951e1273be10a3969b5883fe5964f3.png)

RC 中的第一人称视角(或第一人称视频)指的是通过视频链接驾驶遥控车辆或飞机，虽然严肃的赛车手会以任何方式安装相机以提供最佳优势，但将相机安装在微型飞行员头部的位置总是很有趣，因此可以获得更身临其境的行动视图。[SupermotoXL]显然是这种方法的粉丝，并分享了一些 RC 汽车模型的 3D 打印驾驶舱套件的可下载设计，包括用于其他车辆的更通用的组件。这些模型提供了仪表板、方向盘，甚至允许使用小型伺服系统来使方向盘的运动与实际发送的控制信号相匹配。通过添加另一个伺服系统，使观众可以转动摄像机，整体效果得到了进一步改善。
