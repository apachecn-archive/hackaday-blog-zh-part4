# “片场安静”也适用于物品

> 原文：<https://hackaday.com/2021/09/04/quiet-on-the-set-goes-for-objects-too/>

除非你精通声学，试图入睡，或者只是处于紧张状态，否则你可能不会意识到普通物品有多吵。几乎所有的东西都发出足够的噪音来破坏一个音响师的一天，或者至少，破坏他们听到两个角色之间对话的机会。你在片场需要的是无声但真实的普通发声器，比如纸袋、冰块，以及在较小程度上的台球。

如果你在 Reddit 上花过一点时间，你可能已经看过令人沮丧的简短 gif 图片，上面是[蒂姆·舒尔茨]快速解释这个或那个无声道具是如何制作的。下面嵌入的是道具黑客的概要,其中包含了更多的信息。谈理想工作！解决问题，然后靠拼凑一个解决方案谋生，这听起来既可怕又令人愉快。
