# 品尝电视:TTTV

> 原文:[https://hackaday.com/2021/12/27/taste-the-television-tttv/](https://hackaday.com/2021/12/27/taste-the-television-tttv/)

东京明治大学科学技术学院的副教授[霍梅·宫下]开发了一种在电视或监视器上再现味觉的新技术，一种叫做*品尝电视*(TTTV)的系统。研究小组使用味觉传感器对各种食物进行采样，并提出了 10 种不同气溶胶口味的调色板，可以以各种比例进行组合。这种味道是在屏幕顶部的一个暂存区产生的，然后被滚动到一个塑料薄膜上。

![](../Images/1473ccac798f1ef4f8d7aecaa352a31a.png)

休息时间下方的视频中显示的可能应用包括烹饪节目、餐厅菜单和品酒活动。我们不太确定这对消费者会有多受欢迎。如果不是不卫生的话，与朋友远程品尝烹饪节目会很不方便。我们也不知道目前的视频接口协议，如 HDMI 或 ATSC，包括除了视觉和声音以外的任何感官规定。如果你有学术期刊的话，[宫下教授]关于 TTTV 的研究论文可以在第 34 届 ACM 用户界面软件和技术研讨会上找到。
