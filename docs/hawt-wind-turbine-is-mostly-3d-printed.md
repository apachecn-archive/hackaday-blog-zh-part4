# HAWT 风力涡轮机大部分是 3D 打印的

> 原文：<https://hackaday.com/2020/08/07/hawt-wind-turbine-is-mostly-3d-printed/>

风力涡轮机是一种很好的可再生能源，也是一个很好的 DIY 项目。它们可以用各种材料建造，对初学者来说门槛很低。[Fab]利用现代建筑技术建造了这样一个装置，并将其命名为 WinDIY。

WinDIY 的设计大部分是 3D 打印的，带有熟悉的三叶设计。转子的直径为 1.2 米，这意味着在强风中需要制动和调节涡轮机以确保安全。[Fab]的目标是通过结合机械和电子制动以及变桨距叶片来实现这种控制。3D 打印设计的好处是，它允许快速迭代，特别是几何形状复杂的零件，否则加工这些零件会非常耗时或昂贵。

[Fab]的文章非常详细地介绍了诸如[桨距控制系统的设计和其他细节、](https://hackaday.io/project/172328/log/180102-first-design-of-the-pitch-control-actuator)等主题，这对从事类似项目的其他人来说应该是一个很好的参考。如果你在寻找更具科幻未来氛围的东西，[考虑尝试垂直轴建筑。](https://hackaday.com/2019/10/05/building-a-wind-power-generator-in-your-backyard/)
