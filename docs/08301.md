“革命”是一个被随意使用的术语，尤其是在电子领域。然而，这是可以理解的——由“晶体管革命”或“个人电脑革命”或最近的“人工智能革命”导致的社会变化是变革性的，往往是好的，有时是坏的。然而，一个共同的线索是，一旦这些革命发生，之后一切都不一样了。

软件定义无线电(SDR)和数字信号处理(DSP)就是这种情况。这两个相关领域可能看起来不像其他一些电子革命那样具有变革性，但当你仔细想想，它们确实改变了无线电通信的世界。SDR 意味着复杂的无线电发射机和接收机不再需要严格地用硬件实现，如滤波器、混频器、检波器和放大器的集合；相反，它们可以简化为一系列在计算机上运行的算法。

与 DSP 合作，SDR 带来了 RF 领域的巨大变化，强大的高带宽无线电链路几乎是事后才被内置到设备中。但这些概念可能很难让人理解，至少在深入基础知识并真正尝试了解 SDR 和 DSP 的工作原理时是如此。谢天谢地，马里兰大学的兼职教授马克·李奇曼博士写了这本书。[“py SDR:使用 Python 的 SDR 和 DSP 指南”](https://pysdr.org/index.html)是对 SDR 和 DSP 的精彩介绍，面向那些希望学习如何将 SDR 和 DSP 应用于实际系统的人。Lichtman 博士将在 Hack Chat 上讨论他的教材，回答您关于如何最好地学习 SDR 和 DSP 的问题，并讨论在掌握基础知识后的下一步行动。
