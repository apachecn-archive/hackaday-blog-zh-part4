# DNA 逻辑

> 原文:[https://hackaday.com/2022/11/17/logic-via-dna/](https://hackaday.com/2022/11/17/logic-via-dna/)

我们常说，你可以用几乎任何东西制造逻辑门。史蒂夫·莫尔德(Steve Mould)会同意这一点，因为他刚刚和一个装满 DNA 的托盘(T1)玩完十字游戏(如果你是美国人，就玩井字游戏)。你可以在下面的视频中看到最终的游戏以及它是如何工作的。

DNA 的使用并不真正重要，因为它只是为九个细胞中的每一个实现了一个逻辑方程。因此，举例来说，只有当某些其他方块被 O 占据或没有被 O 占据时，每个细胞才会被 DNA)占据。因此，您实际上是使用每个细胞的状态及其反转来创建一个 AND/OR 门。

当然，关键是你如何输入你的动作，DNA 如何处理这些方程？输入很简单。你为每次移动添加不同的解决方案，并转向每个单元格。正如视频中所解释的，处理过程需要一些实验室工作。

老实说，这让我们想起了 1976 年的一个二极管逻辑项目。它看起来像一个程序，但实际上它只是组合逻辑的巧妙运用。实用？也许不是，但它说明了生物化学世界的不同之处。正如我们常说的，[实际上任何东西都可以成为逻辑门](https://hackaday.com/2017/01/03/make-logic-gates-out-of-almost-anything/)。[真的](https://hackaday.com/2012/09/28/making-logic-gates-out-of-crabs/)。
