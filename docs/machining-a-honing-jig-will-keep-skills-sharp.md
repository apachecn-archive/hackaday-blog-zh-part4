# 加工珩磨夹具将保持技能敏锐

> 原文：<https://hackaday.com/2021/03/01/machining-a-honing-jig-will-keep-skills-sharp/>

[Amy make Stuff]长期以来一直使用一对金刚石磨块徒手磨利刨子、凿子和商店里所有其他沉闷的东西。虽然这种方法效果很好，但如果没有某种夹具在磨刀时牢牢固定住刀片，结果往往会不一致。这些类型的设备非常丰富，而且价格便宜，但是正如[Amy]在广告之后的视频中所说的，她不能加工任何东西。男孩，我们知道那种感觉。

[Amy]能够完全用她周围的材料来做这个，从一块废铝开始，最终切割成夹具的两半。视频中充满了技巧和窍门，近距离观察[Amy]的过程真的很有趣。我们最喜欢的部分必须是那个膨胀和收缩夹具的把手。[Amy]通过在一个圆的外边缘附近钻一些孔，然后铣掉边缘，直到她有一个完全有凹槽的旋钮。一旦她完成了夹具，她升级了她的珩磨条，用乳白色的高密度聚乙烯为它们磨出了一个新家。

磨是很好的工具，但是有点贵。如果你刚刚开始，为什么不把钻床改造成铣床呢？那不是比买一个更有趣吗？
