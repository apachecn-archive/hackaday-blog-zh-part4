# 异想天开的焊料台兼职作为玩具 3D 打印机

> 原文:[https://hack aday . com/2020/09/15/whickical-solder-stand-moonlight-as-toy-3d-printer/](https://hackaday.com/2020/09/15/whimsical-solder-stand-moonlights-as-toy-3d-printer/)

[![](../Images/19735bd540b505666c5ac8a9d50c01c5.png)](https://hackaday.com/wp-content/uploads/2020/09/Lego-pieces.png)

A few Lego pieces provide key functionality, like an articulated dispenser head.

我们大多数人在焊接时都会将一段焊料弯曲成更方便的形状和角度，只是将烙铁和工件推入悬挂的焊料中，而不是伸出第三只手。嗯，[yukseltemiz]似乎已经决定焊料分配器和微型 3D 打印机模型可以有很多共同点，并创建了一个 1/5 比例的 Ender 3 打印机模型，作为焊料支架和分配器。焊料卷轴挂在细丝卷要去的地方，焊料本身通过“打印头”分发。

这很可爱，我们也很喜欢[yukseltemiz]将一些乐高积木融入建筑的方式。一个旋转环和孔眼将焊料从卷上引导下来，一个小乐高球和插座为分配器提供了关节，这是将焊料弯曲到更方便工作的角度的一个重要特征。这让我们认为，将乐高积木与更传统的硬件如 M3 螺母和螺栓一起使用可能是一种尚未开发的技术。您可以在下面的简短组装视频中看到该装置的运行情况。
