# 自制 16 位计算机重新发明了所有的轮子

> 原文:[https://hackaday . com/2021/12/03/home brew-16-bit-computer-reinventions-all-the-wheels/](https://hackaday.com/2021/12/03/homebrew-16-bit-computer-reinvents-all-the-wheels/)

构建自己的计算机有许多可能的途径。人们可以制造他们自己的 Z80 或 MOS 6502 计算机，然后运行一个周期正确的操作系统。或者一个人可以像詹姆斯·斯坦利那样从零开始。[詹姆斯]发明了一种完全独特的电脑和 CPU，他称之为 SCAMP。SCAMP 运行一个名为 SCAMP/os 的定制操作系统，你可以在休息时间下面的视频中查看。

[James]将 CPU 和计算机描述为有目的的原始。它由分立的 74xx 系列逻辑芯片构成，运行速度快至足以自制 1 MHz。另外，它有很多闪烁的灯，这不禁让我们想起了最初的 Imsai 8080。但是 SCAMP/os 启动的不是一个用于编程的开关面板，而是一个通过串行终端呈现的外壳。程序是用定制的语言编写的，带有自己的编译器。该操作系统被描述为具有类似 Unix 的感觉和类似 CP/M 的功能。这是一个很好的组合！

除了简洁的外观和闪亮的灯光，我们最喜欢这个版本的是[James]在他的博客和[Github 上记录这个版本所做的大量工作，在那里可以找到源代码和设计](https://github.com/jes/scamp-cpu)。还公开邀请贡献者帮助推进该项目。我们确信他会到达那里，一次一点。

虽然[詹姆斯]目前使用的是压缩闪存卡存储，但我们不禁想知道[盒式磁带存储系统](https://hackaday.com/2020/11/10/proper-cassettes-for-your-fpga-retrocomputer/)是否是一个值得的未来升级。
