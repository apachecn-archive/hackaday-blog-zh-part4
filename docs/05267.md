# 牛郎星 8800 再次项目

> 原文：<https://hackaday.com/2020/01/04/altair-8800-again-project/>

[德克]发布了一个视频(你可以在下面看到)，标题是，“神秘复古项目开始。”这是他的 Altair 8800 模拟器的多部分系列的第一部分。前面板似乎是激光切割的，在未来的一些视频片段中，我们希望看到他用一个小东西模拟 CPU。

已经有大量的 8800 克隆，从重建原始 PCB 的项目，到只在内部运行树莓 Pi 的项目。中间地带将使用 Arduino 或其他小型 CPU 来模拟 8080 CPU。

开关是多路复用的，这并不罕见。然而，LED 实际上是一个可寻址的 LED 条。我们已经建立了一个类似的离散 led 面板，我们可以证明，使用带将更容易。除此之外，它允许你改变 LED 的颜色，以适应你的装饰，这是旧的 8800 没有烙铁做不到的。

当然，缺点是您可以同时更新 led。对于大多数东西来说，这可能没什么大不了的，但它仍然与最初的计算机有一点点不同。

即使没有仿真，这也将是一个面板连接到在 PC 上运行的 SIMH 的良好开端。LED 灯带和激光切割机的使用使这比我们见过的大多数其他建筑都要容易得多。

自从 1975 年 1 月版的《大众电子产品》(将于 12 月上市)中出现了牛郎星的原型，我们总是在年底左右想到它。

如果你没有耐心建造一个完整的前面板，你可以[把它放在 PCB](https://hackaday.com/2017/03/14/the-altair-shield/) 上。事实上，我们很想制造一系列安装在面板后面的 PCB。在之前，我们已经见过几次这种方式接近[。](https://hackaday.com/2015/04/12/altair-8800-front-panel-for-an-8080-emulator/)
