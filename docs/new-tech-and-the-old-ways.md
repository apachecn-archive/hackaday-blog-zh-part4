# 新技术和老方法

> 原文：<https://hackaday.com/2022/04/30/new-tech-and-the-old-ways/>

本周在 Hackaday 上，我们特别推出了一个项目，这个项目勾起了我的怀旧之情，证明了在将新技术应用于老问题时会有很好的机会。让我解释一下。

[Muth]与我们分享了一个项目，该项目将传统的模拟摄影打印与现代的液晶屏幕结合起来。基本的想法是用 4K 单色屏幕代替底片，通过将屏幕直接放在相纸上并在均匀的光源下曝光来制作接触印刷。就像老方法一样，但是用液晶显示器代替了胶片。

但是屏幕和电影的主要区别是什么呢？当然，你可以随意改变 LCD 上的图像。因此，当[Muth]校准曝光时，他突然意识到，他可以创建一个动态的、动画版本的图像，并逐步曝光纸张的不同部分，扩大可用的动态范围，并使他能够控制最终图像对比度的细微差别。

作为一个老摄影爱好者，这是那种我们会在暗房里一直手动完成的把戏。“闪避”就是在曝光时间的一部分用手或特殊工具遮住投射的光线，从而使图像的一部分变亮。通过[Muth]的过程，他可以在每像素水平上以编程方式减淡图像。以前我们会因为这种能力而杀人。

这里更大的故事是，通过尝试开箱即用的东西，将新工具应用于旧程序，[Muth]偶然发现了新功能。作为黑客，我们无时无刻不在玩弄我们能得到的最新技术。当你这样做的时候，你可能也会偶然发现新技术带来的新的可能性。睁大眼睛！
