# 拆卸:索尼新艾博正在接受手术

> 原文:[https://hackaday . com/2018/10/22/tear down-sonys-new-艾博-goes-under-the-knife/](https://hackaday.com/2018/10/22/teardown-sonys-new-aibo-goes-under-the-knife/)

完全出乎意料的是，索尼公司已经在北美发布了他们的机器狗系列的最新版本，艾博。这种设备已经在日本上市，那里有许多狗主人俱乐部，可以与任何专门的养狗俱乐部相媲美。感谢[Robot Start]团队，在他们拆卸艾博 ERS-1000 的过程中，我们现在可以一窥制造人类最好的朋友机器人的过程。

![Sony Aibo Battery Removal](../Images/8c290424d384f4d7c1bb3bfc009fbf4d.png)

据 Robot Start 的 Yoshihiro 称，艾博看起来使用了一种类似 Handycam 摄像机的专有电池。这三个金触点用于在地毯形状的电源底座上充电，艾博将定期返回以睡个“午觉”。小狗眼睛后面有几个方形的有机发光二极管屏幕。它们是一英寸左右的全彩 OLEDs。屏幕之间是一个电容式触摸传感器，它缠绕在头顶，也是压力敏感的。![Sony Aibo Teardown OLED Eyes 2018](../Images/d278616324dd37d52aa8cf5730518679.png)

根据索尼的新闻稿，艾博鼻子里的鱼眼镜头被用来识别人脸和导航空间。

将所有主要部件放在一起设计，无疑让人们了解了最新款艾博的复杂性。看到这款设备的进步会很有趣，因为它们都配备了 4G LTE 和 802.11 b/g/n WiFi，可以连接到索尼的服务器进行深度学习。

![Sony Aibo Teardown Complete 2018](../Images/49989629a98fd9066ff7f841b11e697d.png)

只要设备在订阅计划下，新的行为应该会自动下载。虽然索尼目前没有计划与任何语音激活的虚拟助手集成，但我们仍然可以期待 Hackaday 社区提供一些扩展功能的可能性。
