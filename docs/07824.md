# 声控沙发满足您的各种饮料需求

> 原文：<https://hackaday.com/2020/09/28/voice-controlled-sofa-meets-your-every-beverage-need/>

人们通常认为这很无聊，但现代世界充满了奢侈品。家庭自动化、食品配送，甚至上网都是手边的好工具，但对我们大多数人来说都是微不足道的。如果这些现代奇迹对你来说还不够，奢华的生活还缺少某种东西，请允许我们向你介绍终极便利:[一个语音控制的啤酒分发沙发，内置制冷系统](https://www.youtube.com/watch?v=OL8z9MNZb2k)。

这是一个来自【车库复仇者】的项目，在达到这种水平之前经历了多次迭代。第一个版本的金属工作没有像预期的那样装配在一起，在决定重新利用一个实际的冰箱之前，在实际的制冷方面进行了许多尝试。有了这些东西，他就能进入项目的核心部分了。沙发冰箱可以装 12 瓶啤酒，它们在传送带上，传送带会自动将下一瓶啤酒放到自动抽屉里。当(通过语音、应用程序或遥控器)发出命令时，沙发会打开抽屉，这样使用者就可以轻松地抓住一个抽屉，而不必移动超过一只手臂。一切，包括语音识别模块，都由 Arduino 控制，这是传统。

对细节的关注也非常出色。遥控器包含一个内置的开瓶器，其中一个是冰箱的背光和玻璃盖，抽屉在感应到啤酒已被获取时会自动缩回。我们不能对自己的沙发要求太多，除了它们可能会带我们去我们想去的地方。但是也许现在最好把这两个沙发用例分开。
