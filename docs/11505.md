# Furter Burner 就是这样做香肠的

> 原文：<https://hackaday.com/2021/10/06/furter-burner-cooks-the-wieners-just-so/>

有时候你会饿得想吃两根香肠，而不是更多。[韦斯利]设计了他的 [Furter Burner](https://www.youtube.com/watch?v=9EAzrofKDtc) 来精确处理这些情况，它看起来像是在烹饪一对香肠。(视频，嵌在下面。)

这个过程从几个木制特技香肠和一些泡沫板开始，[韦斯利]用这些来完成设计。在那里，CAD 设计被绘制出来，零件被从压缩板中取出以进一步解决装配问题。后来转向胶合板版本，拥有一个木制原型很快就会发现许多需要改进的地方，从在烤架表面增加把手到允许燃烧的气孔。

这个设计在完成之前还需要在 metal 上反复几次。最终的结果令人印象深刻——产生了一个燃烧煤炭的双香肠炊具，配有便于香肠处理的串肉扦，并带有[韦斯利]自己的标志。

视频展示了迭代设计过程的好处。这也证明了为什么在使用重型材料之前，用更便宜的材料粗略设计通常是有意义的，特别是像[韦斯利]这样的情况，金属部件只能在场外切割。首先在内部完善设计，省去了很多麻烦。

我们以前也看过[Wesley]的作品，比如这个令人印象深刻的工作室储物解决方案。
