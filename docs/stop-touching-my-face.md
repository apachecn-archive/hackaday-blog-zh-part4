# 别碰我的脸

> 原文：<https://hackaday.com/2021/07/13/stop-touching-my-face/>

我们都有一两个不太引以为豪的习惯，并且可能想了很多方法来帮助我们摆脱它们。嗯，[Friedlc]想知道他是否可以创造一种机制，让他停止用一点负面条件反射来触摸自己的脸。他装配了一个头部支架，每当伸手够到他的脸时，它就会拍打他的前额。

他需要做的第一件事是检测一只手靠近他的脸。他决定使用一些廉价的红外运动传感器，但注意到它们有一些局限性。他很难调整运动传感器的灵敏度，以防止误报，而且在阳光直射下，由于太阳辐射使光电探测器饱和，这些传感器完全没有用。尽管存在这些问题，[Friedlc]认为他在室内更需要他的设备，所以他坚持使用红外探测器。

对于他所谓的“击球手”，他想到了一些不同的想法。也许是一个带拍子的旋转鼓会砸到他的手，也许是一个连杆上的打击臂。他承认，他的基本机械设计知识让他想到完美的“击球手”有点挑战，但像任何优秀的黑客一样，[Friedlc]只是继续努力。他决定使用一个[凸轮机构](https://en.wikipedia.org/wiki/Cam)，这使他能够将马达与击打动作分开。这个选择实际上减少了马达的负载，避免了马达停转和给他带来其他麻烦。

[Friedlc]对他的发明非常自豪，并指出这真的帮助他不再摸自己的脸，因为连续击打头部绝对是一种威慑。这当然不是我们第一次在 hack aday 上看到巴甫洛夫条件反射项目[。我们可能都需要一些帮助来改掉一些坏习惯。](https://hackaday.com/2015/05/12/super-simple-cat-feeder/)[但也许你更喜欢正面强化而不是](https://hackaday.com/2020/10/31/using-open-source-to-train-your-dog/)。
