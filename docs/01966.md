# 最微小的倒退

> 原文:[https://hackaday.com/2019/01/29/the-tiniest-retropie/](https://hackaday.com/2019/01/29/the-tiniest-retropie/)

RetroPie 项目是一个用于 Raspberry Pi 的软件套件，允许用户通过模拟器轻松地玩经典视频游戏。它已经存在一段时间了，所以用一个基本的控制器和视频输出来设置它相对来说比较简单。这意味着实现 RetroPie 的新方法的竞赛已经开始，[Christian]将此视为一种挑战，尽可能地建造最小的 RetroPie。

他为自己设定的约束条件是将项目控制在 100 毫米以下。为此，他使用了一个装有 RetroPie 软件的 Pi 零点仪，并将其与 1.44 英寸的屏幕配对。有一个微小的脂肪电池藏在那里，以及一个小型音频放大器。几乎所有其他东西都是 3D 打印的，包括外壳、D-pad 和按钮。如果你想推出你自己的，整个版本也可以在 Thingiverse 上找到。

虽然这可能是我们见过的最小的回顾，但仍然有一些值得尊敬的提及。还有一款[我们见过的尺寸更小的另一款手持设备](https://hackaday.com/2018/04/07/the-better-retropie-handheld/)，还有一款[被塞进了一个带有翻盖屏幕的 Altoids 罐](https://hackaday.com/2017/04/23/this-hacker-fit-an-entire-retropie-in-an-altoids-tin/)。活着是一个激动人心的时刻！
