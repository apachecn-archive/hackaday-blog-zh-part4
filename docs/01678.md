# 马自达心爱的“旋转火箭”的 3D 打印汪克尔发动机

> 原文：<https://hackaday.com/2018/12/30/3d-printing-wankel-engine-from-mazdas-beloved-rotary-rocket/>

虽然有一个短暂的公司叫做 Rotary Rocket，但这个词更为人所知的是作为马自达 RX-7 的昵称，这是少数使用汪克尔或转子发动机的汽车之一。如果你想知道这些是如何工作的，为什么不打印一个模型呢？【工程解释】就是这么做的。他们打印了一个 [1/3 比例的模型](https://www.youtube.com/watch?v=Ny4-Jc_qW60)，并制作了一段视频解释和演示其操作。模型本身来自 [Thingiverse](https://www.thingiverse.com/thing:3006071) ，由【EricThePoolBoy】创作。

我们非常喜欢这个模型的一点是用灯光来展示燃烧的不同阶段。冷气进气是蓝光，热气是红光，以此类推。这真的有助于想象正在发生的事情。可以看下面的视频。

如果你以前没有见过 Wankel，这是一个巧妙的设计。它有很少的运动部件，提供非常平稳的动力传输和高功率重量比。不过，缺点是发动机故意燃烧机油来润滑和密封，因此很难达到排放标准，需要大量的机油。当前设计的燃料效率也不是很好，特别是因为制造商经常用燃料效率来换取更好的排放。

如果你想了解更多关于 Wankel 的内容，请查看我们之前的[帖子](https://hackaday.com/2016/03/03/broken-promises-of-the-wankel-engine/)(以及所附的 165 条评论)。今年早些时候，我们还关注了——或者更确切地说是通过——[的另一个汪克尔。](https://hackaday.com/2018/05/01/see-through-rotary-engine-reveals-wankel-magic/)
