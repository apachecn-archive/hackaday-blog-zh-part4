# 那不是小行星…哦，事实上它是

> 原文：<https://hackaday.com/2022/06/05/thats-no-asteroid-oh-actually-it-is/>

在杀手小行星袭击你的星球之前识别它们有多重要？问问任何恐龙。哦，等等……当然你也需要一种方法来改变它们的方向，但是最近人们对寻找它们的兴趣有所增加，包括一个新的私人资助的项目，叫做 T2 小行星研究所。

该计划隶属于 B612 计划以及包括华盛顿大学在内的其他计划，利用一个名为 ADAM 的开源云平台，已经发现了 104 颗新的小行星，并绘制了它们的轨道。

有趣的是，该研究所本身并不获取任何图像。相反，它使用新技术来搜索现有的光学记录，以识别以前未被注意到的小行星，并计算它们的轨迹。

你不得不怀疑有多少其他数据集还在漂浮，它们包含着未知的发现，等待着正确的算法和计算能力。当然，一旦你发现了下一颗灭绝的小行星，你就必须决定如何处理它。激光？炸弹？隔空轻轻一推？还是希望外星方尖碑产生偏转射线？你会怎么做？

美国宇航局正在试验移动小行星。如果你想自己找到一些，你可能想看看现有的的[地图集。](https://hackaday.com/2021/09/10/nasa-are-squaring-up-against-the-asteroid-threat/)
