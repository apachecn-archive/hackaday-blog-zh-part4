# 黑客日链接:2021 年 4 月 4 日

> 原文:[https://hackaday . com/2021/04/04/hackaday-links-April-4-2021/](https://hackaday.com/2021/04/04/hackaday-links-april-4-2021/)

我可以说，在包括愚人节的一周内做一篇链接综述文章不是一件有趣的工作吗？因为它不是。我的意思是，当他们在这样的一天发布报告时，你怎么能认真对待类似天王星发出 X 射线的报告呢？不过，这看起来确实是一个合理的故事，而且非常有趣。行星发射 X 射线并不是什么新鲜事；几十年来，我们已经知道木星和土星都是强大的 X 射线源。尽管天王星是我们太阳系的一个奇怪的孩子，但在钱德拉天文台 2007 年捕获的数据中发现 X 射线辐射的证据是出人意料的。天文学家认为 X 射线可能来自天王星环，或者它们可能是太阳发出的 X 射线的反射。或者，这可能是气体巨星磁场的奇怪排列导致了在光谱的 X 射线部分发光的强大极光。不管它是什么，它是怪异而美丽的，总的来说，这并不是一件坏事。

另一个潜在的基于笑话的故事出现在本周，关于看似不可能的“EmDrive”。看起来，当你似乎违反了物理定律时，你可能做错了，仔细的实验室测试表明，无燃料推进还没有出现。人们可能会认为，用微波填充一个封闭的不对称室绝对不会产生推力，这是不言而喻的，但 EmDrive 的支持者多年来一直报告说，这个不太可能的引擎产生了少量但可测量的推力。然而，德累斯顿大学的一个研究小组发现情况并非如此。尽管他们能够测量发动机的位移，但这似乎是由于测试台在射频能量流入驱动室时变热并翘曲造成的。通过改变发动机的支撑方式，他们能够抵消尺寸变化，使其看起来像 EmDrive 实际工作。

[![](../Images/594a6509acd1a5e5cdacb49e08dc4e05.png)](https://hackaday.com/wp-content/uploads/2021/04/Ex0jWqTXMAkARzE.jpg) 想使用表面贴装器件，但又不想费力组装一个 SMD 板？没问题，至少如果你跟随大卫·布坎南的领导，执行[无表面表面贴装原型](https://twitter.com/David3141593/status/1377295985798438917)。我们在 Twitter 上偶然发现了这个，觉得它看起来很酷——有点电路雕塑的感觉，我们喜欢普通 0.1 英寸 perfboard 的老派外观。大卫报告说，飞线只是漆包线磁线；在焊接之前，我们已经完成了刮擦和清洁电磁线的工作，我们认为建造的那部分一定很痛苦。我们联系了大卫，问他是否有任何准备磁线的捷径，但是，唉，他说他只是用了一滴热焊料和一点耐心，而搪瓷煮掉了。我们仍然真的很喜欢这种风格的建设，我们鼓掌的努力。

说到偶然遇到的事情，这是这份工作的最大乐趣之一——当我们四处搜寻最新的黑客时，掉进算法生成的兔子洞。一个这样的意外发现是这个 YouTube 频道记录了一个非常好的喷气发动机建造。我们以前见过很多喷气式发动机，但很少有像这样带加力燃烧室的。Praendy 为发动机建造的可变喉道喷嘴也有令人深感满意的地方——这种复杂程度在业余爱好者的喷气发动机中并不常见，但其机制非常简单，易于理解。

[![](../Images/f046e638853a36f4d32247b1a8a770f9.png)](https://hackaday.com/wp-content/uploads/2021/04/automatic_tube_polishing_sanding_machine-hi7igncl_9emp4-shot0001_thumbnail.png) 我们发现的另一个兔子洞是在[报道了这款酷炫的 TIG 钨磨具](https://hackaday.com/2021/03/29/put-the-perfect-point-on-your-tungstens-with-this-die-grinder-attachment/)之后。这让我们进入了 Metalist 的后备目录，在那里我们发现了许多有趣的东西。但是真正的享受是[这个自动管子抛光机](https://www.youtube.com/watch?v=hi7iGNcl_9E)(视频)，我们不得不说它让我们一直猜测到最后。如果你有 12 分钟的时间，并且你喜欢金属加工制作，看看它，看看你是否对这个工具的聪明之处感到惊讶。

最后，我们以前听说过 Anatoli Bugorski 的痛苦，但从未在这个令人不安的视频中详细介绍过。(嵌在下面。)

你会问谁是阿纳托利·布戈尔斯基？他是一名俄罗斯粒子物理学家，1978 年在一个加速器实验室工作时，设法让自己的头直接进入 76 GeV 质子束的路径。尽管受到了大剂量的辐射，布戈尔斯基不仅在事故中幸存下来，还设法完成了他的博士学位，并开始了漫长的核物理生涯。他也结婚了，有了一个儿子。他肯定受了伤——主要是面部瘫痪和部分失聪——但没有遭受像切尔诺贝利消防员或其他受到大剂量辐射的人那样可怕的命运。该视频讲述了事故发生的一些细节——事实证明，两个灯泡比一个要好。我们喜欢这个视频，但无法停止思考布戈尔斯基是俄罗斯原子能时代的菲尼亚斯·盖奇。
