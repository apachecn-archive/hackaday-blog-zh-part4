# 与国际空间站对接并不像你想象的那么容易

> 原文：<https://hackaday.com/2020/05/15/docking-with-iss-isnt-as-easy-as-you-might-think/>

复杂是一件有趣的事情。在史前时期，一个穴居人可能会靠着一根木头漂浮在湖面上。这很简单。但是当你添加一个舵，一个帆，甚至一个马达，它变得越来越复杂。但是如果你增加足够的复杂性——比如 GPS 和自动驾驶仪，它又变得简单了。SpaceX 龙太空舱实际上与国际空间站对接。但是，如果需要，空间站上的工作人员可以手动接管。那会是什么样子？试试[模拟](https://iss-sim.spacex.com/)就知道了。如果你第一次做不到，试试看，下面的(斯科特·曼利的)视频可能会帮助你。

这不是一个浮华的星球大战式的模拟器。多想想 2001。动作慢，容易失控。与旧的阿波罗时代相比，用户界面无疑是现代的

即使你设法对接，在现实生活中你也不能只是打开舱门进入空间站。温度需要一段时间来升温，所以当温暖潮湿的空气冲击寒冷的对接通道时，你不会有大量的冷凝。

我们听说,[吉姆·布里登斯蒂]——美国国家航空航天局的家伙— [第一次尝试就成功了。但是因为他是飞行员，我们认为他可能比我们有更多的实践经验。](https://twitter.com/JimBridenstine/status/1260271033489260548)
