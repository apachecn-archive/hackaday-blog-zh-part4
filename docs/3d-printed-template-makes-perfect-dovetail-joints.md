# 3D 打印模板制造完美的燕尾接头

> 原文：<https://hackaday.com/2022/06/03/3d-printed-template-makes-perfect-dovetail-joints/>

一件家具上的燕尾榫是让它看起来像“木匠大师”而不是“宜家”的特征之一。传统手工制作的燕尾榫需要精确的尺寸，甚至更精确的锯和凿，这些技能可能需要数年才能掌握。一个稍微不那么手工的方法是使用刳刨机和燕尾模板；路由器做出完美的直线切割，而模板确保它只去它需要去的地方。

如果你还没有这样的模板，看看[Guy Perez]的设计，这是一个可调节的燕尾模板，很容易用 3D 打印机生产。它由安装在 T 型轨道上的十个独立部件组成，这使它们能够侧向滑动，从而产生不同宽度的引脚和尾部。T 型轨道安装在一个木制主体上，带有一个集成的夹具来固定目标工件，以及一个终点挡板来为所有测量提供参考。

正如你在下面的视频中所看到的，最终的夹具易于使用，每次都能得到近乎完美的吻合。[Guy]提供了 CAD 文件以及关于其设计的详细说明，因此如果您需要不同尾部角度的工件或想要使用更厚的木材，您可以轻松调整它们。

虽然这种夹具会使切割普通的燕尾接头变得容易得多，但你仍然可以通过将[做成不可能的木槌](https://hackaday.com/2020/02/22/this-mallet-has-backwards-dovetails-thats-impossible/)来炫耀你的手工技能。想加入金属块吗？看看[这个可爱的小燕尾立方体](https://hackaday.com/2019/10/10/diy-tiny-dovetail-cube-needs-diy-dovetail-cutter/)。
