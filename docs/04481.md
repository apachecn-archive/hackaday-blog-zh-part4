# 煎饼的 CPU 对决

> 原文：<https://hackaday.com/2019/10/11/cpu-showdown-for-pancakes/>

如果你问人们如何评价一名司机，大多数人会说他们比平均水平要好。起初，这似乎不太可能，直到你意识到一件事:人们用不同的标准来评价自己。所以莎莉认为她是个好司机，因为她开得很快。汤姆从未出过车祸。爱丽丝从不迷路。你可以在 CPU 上看到同样的效果。有些更快，或者有更多的内存带宽，或者每个周期有更多的指令发出。但是 Tom 硬件公司的 Andrew 和 Scharon 想对 CPU 进行真正的测试。[它能做多好的煎饼](https://www.tomshardware.com/reviews/intel-amd-hedt-pancakes,6366.html)？想知道的话，看下面的视频。

虽然你的 CPU 可能很适合玩视频游戏，但它的烹饪面积却小得惊人，所以人们需要一个非常小的锅。平底锅上有凹槽，所以他们在上面涂了厚厚的热油脂。我们也怀疑那是食品级油脂。

我们费了这么大劲把热量从中央处理器带走，你会认为它们会煮得更快。厨师们在英特尔 CPU 上设置了 BIOS，将温度调节到 105 摄氏度。即便如此，在运行压力程序的同时，做一个蛋糕也花了 16 分钟。

AMD 会怎么样？显然，不太好。虽然 Threadripper CPU 有一个更大的炉灶面，但当它太热时，某些东西会使它关闭，因此，煎饼几乎没有煮熟。诚然，英特尔芯片关闭过一次，但在外部风扇的帮助下，它完成了任务。

老实说，我们更喜欢在数控机床上做煎饼。如果你搜索煎饼，你会惊讶地发现有许多[类似的项目](https://hackaday.com/2016/01/16/edible-art-from-a-robot-pancake-printer/)。但是没有一个使用 CPU 作为灶台。也许我们应该试试树莓派 4。
