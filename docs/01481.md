# 氢动力 Nerf 爆能枪是危险的可怕

> 原文：<https://hackaday.com/2018/12/10/hydrogen-powered-nerf-blaster-is-dangerously-awesome/>

Nerf 爆能枪已经存在了几十年，让孩子们兴奋不已，也让家长们忧心忡忡。大多数由弹簧或压缩空气驱动，是将精致的传家宝置于危险中的理想节日玩具。不满足于通常的扔泡沫的手枪，[[彼得·斯里波尔]决定把事情提升一个档次](https://www.youtube.com/watch?v=pvuASNhxxcA)。

建造从一个巨型旋风冲击波开始，它使用更大的红色 NERF 飞镖作为弹药。水箱安装在外面，装有不锈钢电极。原始的弹簧和柱塞发射组件然后被移除，为由铜管制成的发射室腾出空间。一个类似泰瑟枪的小装置被用作点火器。当按下充电开关时，电流通过水中的电极，将水分解为氢气和氧气。这然后被传递到发射室，在那里它可以被泰瑟枪模块点燃，由触发器激活。

尽管爆能枪偶尔会因为过压而破坏飞镖，但它比普通爆能枪有更高的射速。尽管它很复杂，但性能多少有些不稳定，但手持式氢鼓泡器的凉爽因素很难忽视。[Peter]确实注意到爆炸性气体和危险的催化剂化学物质的结合使得这个建筑可能最好留给成年人。

如果这个 NERF hack 还不够危险，你可能更喜欢这些泰瑟枪。休息后的视频。
