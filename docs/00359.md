# 自定义对开显示是一种独特的方式来显示天气

> 原文:[https://hack aday . com/2018/08/18/custom-split-flap-display-is-a-unique-way-show-the-weather/](https://hackaday.com/2018/08/18/custom-split-flap-display-is-a-unique-way-to-show-the-weather/)

毫无疑问，翻盖展示很有魅力。无论是在 70 年代的复古时钟收音机上，还是在机场或火车站的大型到达和离开显示器上，观看显示器通过翻转卡片建立清晰、易读的信息都是令人着迷的。但是一个有对开显示的气象站呢？那是你不常见到的。

我们通常会看到一些项目使用从某种商业展示中获得的分体式单元，但[gabbapeople]决定定制并从头开始建造这些展示。每个显示器的框架和机械装置都是由激光切割的丙烯酸树脂制成，活页夹也是如此。每个单元可以在 36 张卡上显示完整的字母数字字符集，每个显示器由其自己的步进器驱动。Arduino 从天气 API 获取当前状况，并将天气描述翻译成四个字符的代码。下面的视频中显示的代码看起来有点神秘，但是随项目发布的缩写列表让事情变得更加清晰。如果你不用看清单就能猜出“HMOO”是什么，那就加分了。

我们喜欢它的外观和感觉，但是我们想知道对开图标是否也是一种显示天气的好方法。看起来按照[gabbabeople]的详细说明去做很容易。或者，你也可以看看我们精选的其他[定制对开展示](https://hackaday.com/2018/02/20/easy-modular-alphanumeric-displays-are-full-of-flappy-goodness/)来获取更多灵感。
