# 给你的猫做一个电动滑板

> 原文：<https://hackaday.com/2018/09/21/make-an-electric-skateboard-for-your-cat/>

你有没有看着你的猫想“你知道，我的小猫真的需要一个电动滑板！“很可能没有，但这似乎发生在【Kim Pimmel】看他的猫 MIDI 的时候，所以他决定造一只。这个过程包括建造一个简单的低功率滑板，带有一个[羽毛主板](https://www.adafruit.com/product/3046)和[电机控制器](https://www.adafruit.com/product/2927)以及一个激光切割开关机制。当[金]把一份礼物放进机器时，猫拉下开关，滑板向前移动，进入一个勇敢的新的电子滑板猫的未来。米迪看起来对这整件事有点不以为然，但我怀疑，只要食物源源不断，他会很乐意继续运送。现在，如果他能想出如何说服猫奥利，我们将真正取得进展。

在这些页面上，猫科动物的胡闹似乎是一种常规的消遣，而不仅仅是寻求简单的猫驱动的点击诱饵。很多[猫喂食器](https://hackaday.com/2015/11/05/automatic-cat-feeder-dispenses-noms-wants-cheezburger/)和[猫发现者](https://hackaday.com/2018/08/28/wheres-the-cat-locating-your-moggy-without-eating-batteries/)比比皆是，但这个项目并不是唯一一个[猫操作的](https://hackaday.com/2016/09/01/cat-operated-cat-food-dispenser/)。我们读者的宠物现在可能已经能在一英里外认出一个 Arduino 了。
