# 大 F 键表示敬意

> 原文：<https://hackaday.com/2021/09/01/big-f-key-to-pay-big-respects/>

所以你的盟友被杀了。你的同志倒下了。不知何故，那个小键上的大写字母 F 应该表达你的敬意？求你了。你需要的是一个巨大的，专用的 F 键，与你尊重的大小相匹配。[和[Jaryd_Giesen]将教你如何建立自己的](https://www.thingiverse.com/thing:4946549)。嗯，算是吧。在 Thingiverse 构建指南和下面搞笑的构建视频之间，你会得到要点。

[![Making a custom spring using a drill and a 3D printed dowel.](img/5e7eb8941bfb408c4a7196f4266bf779.png)](https://hackaday.com/wp-content/uploads/2021/09/giant-F-key-spring.jpeg) 这个建筑最酷的地方之一是定制的弹簧。在生日时间紧张和封锁之间，根本没有办法在两天内找到一个巨大的弹簧，所以[Jaryd]打印了一个上面有孔的圆柱体，可以夹在一个钻子里，代替车床。十次尝试后，完美的春天就在某处。

我们喜欢这里的细节层次——制作一个布丁风格的键帽来匹配主键盘是这个噼啪蛋糕上的糖衣。但最精彩的部分藏在里面:巨大开关的杆驱动一个普通大小的按键开关，因为这样更有趣。因为它是一个巨大的红色，它不会发出噼啪声，但听起来也不是线性的，主要是因为你可以听到印刷的碎片相互摩擦。休息之后看看构建视频，如果你只是想听听这个东西，就点击第二个视频。

在这里，我们最喜欢的事情之一就是观赏美景。有些东西只是为了好看，但[其他时候它们是有用的工具](https://hackaday.com/2020/07/10/its-time-for-watch-clocks-to-make-a-comeback/)。
