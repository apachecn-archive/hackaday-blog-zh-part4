# 水开关灯照明电流

> 原文：<https://hackaday.com/2020/01/31/water-switch-lamp-illuminates-current-flow/>

他们总是告诉你不要把水电混在一起。虽然是的，这是关于这两者的很好的一般建议，但如果你负责任地去做，你就不会在时空的结构上撕开一个洞。[水能导电，为什么不用它来开灯呢](https://www.instructables.com/id/Hydro-Lamp/)？

[Manvith Subraya]的水电灯是一个提醒，不要让大开关模糊了你对简单组件的想法。开关不必很复杂，一些最可靠的开关非常简单——簧片开关和水银倾斜开关就是很好的例子。通过以 1:1 的比例将水加盐，[Manvith]确保电力将流经丙烯酸罐，完成电路并点亮两端的 20W LEDs。

休息后的简短演示视频揭示了使用水作为倾斜开关的一个有趣方面——它不是瞬时的。当他慢慢地将灯从垂直移动到水平，然后再移动回来时，光线随着电子的潮流而变亮和变暗。我们认为建造一个电动框架来利用这一点来营造氛围照明会很有意思，尤其是如果在水的后面有几个 led 灯的话。

水经常被用来解释电流流动的基本原理以及电压、电流和电阻的关系动力学。[正如我们在这台水计算机中看到的，这个概念一直流向逻辑门](https://hackaday.com/2018/12/03/andersons-water-computer-spills-the-analog-secrets-of-digital-logic/)。
