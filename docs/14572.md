# 激烈的塑料:外壳重建使用捐赠材料

> 原文:[https://hack aday . com/2022/08/21/drastic-plastic-enclosure-rebuild-uses-donor-material/](https://hackaday.com/2022/08/21/drastic-plastic-enclosure-rebuild-uses-donor-material/)

虽然 3D 打印机很棒，但人们倾向于把它们当成万能锤子，几乎所有东西都变成了一颗等待被敲击的钉子。因此，随着黑客的胃口变得对同样的老东西过于挑剔，看到以这样一种老派的方式进行围栏修复令人耳目一新。也就是说:[【doido Santos】“在老式放大器上对摇摇欲坠的侧整流罩进行经典修复](https://www.youtube.com/watch?v=3qd6TUwBcUQ)。

[![](../Images/486752fb364606580a3b57d7744cd534.png)](https://hackaday.com/wp-content/uploads/2022/08/drastic_plastic_inner.jpeg) 是的，不是设计替换件，印刷它们，用油漆或丙酮模糊来隐藏分层的证据，[多伊多]调用了一个损坏的音响系统，其底盘在角落里有一个类似于放大器的浮雕。

在切割出两块匹配的捐赠塑料后，[doido]用胶带将它们粘在一起，并用烙铁将它们焊接起来，烙铁配有弯曲但扁平的铲尖，看起来非常适合这种用途。虽然捐赠者的外壳提供了急需的浮雕，但这种美感中缺少一个角落，所以[doido]用浮雕作为模具铸造了一点熔化的塑料。

一旦这些碎片被钉在一起，[doido]就把它们锉平，用砂纸打磨，擦亮，然后把它们安装在放大器上。他们看起来很棒，没有人会知道。但是如果我们处在(多伊多)的位置，我们会告诉所有人我们做了什么。休息之后一定要去看看。
