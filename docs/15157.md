# 改装机器人

> 原文:[https://hackaday.com/2022/10/22/retrofitting-robots/](https://hackaday.com/2022/10/22/retrofitting-robots/)

Al Williams [写了一篇简洁的思想文章](https://hackaday.com/2022/10/17/in-our-own-image-do-we-need-humanoid-robots/),讲述了为什么我们如此着迷于人的形状的机器人，而不是以他们应该做的任何事情的形状出现的机器人。艾尔部分相信科幻有部分责任，它塑造了人们对机器人长相的期望。

引发整个思维列车的是轰鸣(机器人在轨道上)式的机器人手臂，至少在媒体上是这样的，就像机器人油炸厨师一样。顾名思义，这是一个在轨道上的机器人手臂，它在煎炸表面上来回移动，并使用 CV 算法来感应和翻转汉堡。是的，一个翻汉堡的机器人手臂。Al 问他们为什么不像其他装配线一样，将脚蹼设计成炉顶。

在这种特殊情况下，我认为这是一个经济问题。汉堡连锁店已经有了一个为操作人员翻动汉堡而设计的环境。轨道上的机器人手臂是自动化任务的最便宜的方式，符合当前的人体工程学。机械臂的工作方式类似于人类手臂，因为它必须在为人类手臂设计的环境中工作。

你能重新设计一个新的自动汉堡系统，让它更节省空间或者更可靠吗？大概吧。如果你做了，你会有一个人形手臂吗？不一定。但这是关于将机器人技术修补到非机器人流程中，这意味着他们必须遵守我们的规则。我不否认机器人手臂翻转汉堡很酷，但我猜想这实际上是阻力最小的途径。

想到一个科幻时间线，看起来像人类的机器人首先出现，最终被专门建造的智能机器取代，这些机器看起来一点也不像我们，因为环境得到了彻底的检修，这感觉有点奇怪，但这可能是它将要上演的方式。生活并不总是模仿艺术。
