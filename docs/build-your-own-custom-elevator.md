# 打造你自己的定制电梯

> 原文：<https://hackaday.com/2020/12/28/build-your-own-custom-elevator/>

在我们的日常生活中，有很多东西是早期遗留下来的，我们只是因为惯性而继续使用，即使它们在现代没有太多意义。例子包括 60 赫兹的电网，铁轨之间的间距，当然还有独立的阁楼梯子，这种梯子可以安装在标准间距的天花板托梁之间。它不够宽，无法将大或重的东西放进阁楼，建筑标准不会因为这一点不便而改变，所以如果你想将空间变成更有用的东西[，你需要建造一个定制电梯](https://www.youtube.com/watch?v=CgdsamzPN8w)。

这个阁楼电梯来自[Brian],他最近搬进了一个只有他以前家一半面积的房子，但仍然需要存放他所有的东西。这意味着巧妙利用可用空间。对于电梯，他用 2x 木材建造了一个平台，用螺栓和钢支撑固定在一起。车厢在由 1 5/8”超级支柱制成的轨道上上下运行，并由额定 550 磅的绞车马达提升，这足以提升包括大型工具箱在内的大多数家用物品。

我们唯一希望在视频中看到的是开场是如何进行的。据推测，这可能涉及到切割天花板托梁，使开口比标准的阁楼梯子更宽，并且需要注意确保天花板/地板不会被削弱。无论哪种方式，这都是一个常见问题的很好的解决方案，也许可以用定制控制器使[在两个以上的级别上工作。](https://hackaday.com/2012/04/01/build-an-elevator-controller-gain-a-friend-for-life/)
