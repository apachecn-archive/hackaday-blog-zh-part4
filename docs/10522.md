# 经济型程控 DC 电源的回顾与拆解

> 原文:[https://hack aday . com/2021/06/24/review-and-tear-down-of-economic-programmable-DC-power-supply/](https://hackaday.com/2021/06/24/review-and-teardown-of-economical-programmable-dc-power-supply/)

[凯瑞·王]不怕弄脏自己的手，而且总是非常愿意打开东西，看看是什么让它们滴答作响。这一次，[他检查并拆除了 Topshak LW-3010EC 可编程 DC 电源](http://www.kerrywong.com/2021/05/28/teardown-of-an-lw-3010ec-programmable-power-supply/)，首先测试该设备，然后将其打开，看看它的内部情况。

[![](../Images/3f0df4e2c497219cbba6051a2d99d626.png)](https://hackaday.com/wp-content/uploads/2021/06/LW3010_1.jpg)Topshak LW-3010 EC 是由众多制造商制造的一系列相当经济的电源中的一个，这些制造商共享许多相同的内部部件和基本结构。这款电源既可编程，又美观紧凑，[Kerry]在测试功能和检查内部结构时，将它与同一范围内的其他电源进行了比较和对比。

总的来说，[凯瑞]似乎对这支部队很满意。你可以在下面嵌入的视频中看到他测试这款设备，视频最后是他打开设备并解释里面的东西。如果你曾经对这些电源中的一个内部有什么以及它们的预期性能感到好奇，请务必打开分页符下方的视频。

说到电源，我们大多数人都很容易接触到 ATX 的电源。它们是非常强大的硬件，可以用自己的方式破解。我们自己的 Jenny List 将告诉你你需要了解的关于 ATX 电力供应的一切，以及如何将它用于新的用途。
