# 锡膏模版印刷不吸

> 原文：<https://hackaday.com/2021/10/13/solder-paste-stencilling-that-doesnt-suck/>

使用焊锡膏模板可能是一件真正的麻烦事，当你对着它们呼吸时，它们很少会平躺着动来动去。[意想不到的制造商]表达了他的沮丧，并提出了一个简单的解决方案，他简单地[制作了一个 3D 打印的夹具来对齐 PCB 板，并使用他的商店真空吸尘器](https://www.youtube.com/watch?v=hbHJ6JBdSCs)，嘿，很快！

如果你曾经被诱惑从无框架切换到有框架的焊接模板，那么你会注意到它们可能会很难使用。通常的在线供应商有大量的模板支架清单，但在我们看来，它们都完全一样，更适合模板印刷 t 恤，而不是处理微小的 PCB 足迹。

无框架模板的问题是与 PCB 的夹紧和配准，当使用夹具时，框架模板固定，夹具可以拨入旋转和平移误差。

但问题是，除非 PCB 周围有一个非常平坦的支撑区域，否则框架的重量会使模板在 PCB 上弯曲，导致部分模板脱离焊盘。这导致浆糊没有被推到你想要的地方，而是粘在模板的孔上，当你把它拿起来的时候就脱落了。最气人。

你可以试着通过在周围缠绕相同厚度的备用 PCB 来抵消它，但这并不总是非常成功，因为这位抄写员有丰富的手工经验。[意想不到的制造商]通过制作一个 3D 打印夹具来解决这个弯曲问题，该夹具用螺栓固定在模板支架上，采用一个带孔的定制顶板，从而允许从下面施加真空。这将 PCB 吸到夹具上，使其保持平坦(以防它也翘曲)，并将模板直接拉到 PCB 上，使其完全平坦。

这种设置的第二个好处是，你现在可以在同一个夹具上使用无框架模板，因为真空足够强，可以保持它在适当的位置，但没有强到你不能在粘贴前进行微小的对齐调整。

这个简单的想法有助于使一个共同的 PCB 原型任务，吸少了很多。
