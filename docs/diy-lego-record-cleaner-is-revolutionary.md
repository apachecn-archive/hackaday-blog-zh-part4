# DIY 乐高唱片清洁器是革命性的

> 原文：<https://hackaday.com/2021/01/16/diy-lego-record-cleaner-is-revolutionary/>

当谈到保持黑胶唱片清洁时，有许多学派。这是一个几乎和接下来的仪式一样重要的仪式——排队记录和放下唱针。我们见过人们使用各种东西，从 Windex 和超细纤维毛巾到价格数百甚至数千美元的超声波清洗机。在建造一个更结实的超声波唱片清洁器和等待零件的过程中，[【Baserolokus】环顾了一下房子周围所有的乐高积木，决定在此期间建造一个塑料原型](https://hackaday.io/project/176889-lego-edgedriven-ultrasonic-cleaner-spinning-device)。

超声波清洗背后的想法很简单——高频声波通过蒸馏水产生大量的微小气泡。这些气泡轻轻地将所有的污垢和尘垢从凹槽中敲出，无需使用任何刷子、抹布或刺激性清洁剂。[Baserolokus]做了两件挂在洗衣盆边上的衣服。在一边，一个技术马达以每分钟不到一转的速度旋转着唱片，它对着嵌在另一边的 3D 打印机轮子旋转。休息之后，请查看实际情况。

清洗你的黑胶唱片是很好的第一步，但是你可能会用一个低于标准的唱盘毁掉你的唱片。[深入阅读【Jenny List】关于该主题的详尽介绍](https://hackaday.com/2017/03/03/record-players-explained-for-the-streaming-generation/)。
