# 经典的芯片阵容为这个有趣的配音警报器合成器提供动力

> 原文:[https://hack aday . com/2021/09/24/classic-chip-line-up-powers-this-fun-dub-siren-synth/](https://hackaday.com/2021/09/24/classic-chip-line-up-powers-this-fun-dub-siren-synth/)

有一组精英芯片属于“冰冷、死亡之手”类别，它们往往是几十年来证明其价值的部件，而不是几年。其中最主要的是无处不在的 555 定时器芯片，它在发布近 50 年后仍能在最奇怪的地方找到自己的路。再加上 741 运算放大器和 LM386 音频放大器等其他硅芯片，你几乎可以为任何项目打造名人堂阵容。

这正是为这个有趣的小配音警报器提供动力的芯片组合。正如[lonesoulsurfer]解释的那样，配音警笛最初是作为音乐表演的一部分使用的警车等的实际警笛。震耳欲聋的版本最终被录音棚和 DJ 使用的采样或合成警笛效果所取代，这使我们想到了当前的项目。下面的视频以一个演示开始，很难相信这个盒子产生的声音的多样性仅仅来自一对 555 和一个 741 缓冲器。主 PCB 上的五个盆控制效果，而第二个商业混响模块——经过修改也支持回声效果——增加了深度和存在。我内置扬声器和一个漂亮的木质外壳完成了构建，老实说，这听起来比任何基于 555 的 synth 都好。

对这个版本背后的芯片感兴趣吗？我们已经谈论了[555 以及它是如何成为](https://hackaday.com/2018/10/10/the-555-and-how-it-got-that-way/)的，带[看了看 741](https://hackaday.com/2015/10/31/a-peek-under-the-hood-of-the-741-op-amp/) 的内部，并且给[上了一堂关于 LM386 忠诚度](https://hackaday.com/2016/12/07/you-can-have-my-lm386s-when-you-pry-them-from-my-cold-dead-hands/)的课。
