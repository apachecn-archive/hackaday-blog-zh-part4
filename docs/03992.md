# 精细加工的阀门控制微型遥控液压装置

> 原文：<https://hackaday.com/2019/08/28/finely-machined-valve-controls-miniature-rc-hydraulics/>

液压元件是乐高的工业动力传输版本。泵、汽缸、阀门——几乎所有的东西都是标准化的，装配一个工作系统就是挑选正确的部件，然后把所有的东西连接在一起。如果你想制造一台挖掘机或一辆自动倾卸卡车，这很好，但如果你想缩小规模呢？

微型液压系统需要微型元件，其中【TinC33】生产的这个自制液压阀就是一个很好的例子。(视频嵌入下方。)如果你对为什么有人会需要这些感到好奇，看看[不久前建造的微型液压缸](https://hackaday.com/2018/02/18/mini-lathe-makes-tiny-hydraulic-cylinders-for-rc-snow-plow/)，其中你会了解到微型遥控扫雪机是一件东西。下面的视频首先简要但清晰地解释了液压回路的工作原理，以及他设计的旋转双动比例阀。所有的零件都是由铝和黄铜原料在车床上手工加工而成的。加工操作值得一看，但如果你不喜欢这些东西，跳到 13:44 的最后组装和测试。阀门工作良好，提供了非常精细的气缸控制和出色的负载保持，没有泄漏被看到。令人印象深刻。

[TinC33]用一个单一阀体中的多个阀门的设计来结束视频。这看起来可能是一个有趣的加工挑战，也是我们希望看到的。
