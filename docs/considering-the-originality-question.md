# 考虑到创意问题

> 原文：<https://hackaday.com/2019/11/09/considering-the-originality-question/>

许多黑客读者对旧技术感兴趣，从古董摩托车到电子管收音机到复古电脑，你拥有、保存和修复它们。有时你会使用新的部件，因为原件要么不可用，要么非常糟糕，但当你这样做时，你真的是在修复物品，还是在创造一个没有原件灵魂的合成赝品？这是铁路电影和纪录片制作人[Chris Eden-Green] [关于蒸汽机车](https://www.youtube.com/watch?v=8KwdwhuyZWE)考虑的一个问题，作为一个辩论的主题，我们认为它对更广泛的关注旧技术的社区有兴趣。

一路上，这部电影为非铁路专家提供了一个有趣的视角，让他们深入了解一辆运行中的蒸汽机车的检修计划，主线铁路有巨大的车间，但这对一条保存完好的小铁路提出了更大的挑战。一两年前，我们写了世界上第一条保存下来的铁路，威尔士窄轨铁路，作为一个例子，下面的视频中令人惊讶的是，在 20 世纪 50 年代重建后，最早的两个机车中几乎没有原始金属。

这部电影应该会在铁路爱好者中引发一些思考和辩论，毫无疑问也会在黑客读者中引发一些思考和辩论。我们倾向于同意他的结论，即机器是用来运转的，而不是在博物馆里收集灰尘，一个大部分修复甚至复制的机车没有任何害处。毕竟，就像一台复古计算机通过它运行的软件来区分一样，乘坐蒸汽火车更多的是视觉和嗅觉，而不是确切地知道哪种金属构成了机车。
