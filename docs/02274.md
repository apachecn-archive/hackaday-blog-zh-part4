# 用煤减少碳排放

> 原文:[https://hack aday . com/2019/03/04/用煤减少碳排放/](https://hackaday.com/2019/03/04/reducing-carbon-emissions-with-coal/)

这似乎是一个悖论，但煤炭可能是解决碳排放问题的答案。关键不是燃烧它，而是利用大气中的二氧化碳制造它。虽然这在理论上一直是可能的，但高温使其在实践中变得困难。然而,[最近发表在《自然通讯》](https://www.nature.com/articles/s41467-019-08824-8)上的一篇论文展示了一种特殊的液态金属电催化剂如何将气体转化为固体形式的碳，除其他外，这种碳适用于制造高质量的电容器电极。这个过程——你可以在下面的视频中看到更多——在室温下工作。

从空气中提取二氧化碳并不难，问题是如何处理它。以气体或液体的形式储存效率低且成本高，而将其转化为固体则更容易储存，甚至在实际应用中重复使用。

液态金属催化剂是镓基的，并且包含金属铈纳米颗粒等。液态金属抗焦化——用产生的碳材料覆盖催化剂。

这个过程不会产生适合坏孩子长袜的煤块。相反，它会产生大约 3 纳米厚的薄片。我们已经看到了在空气中生长碳纳米纤维的提议，但是这需要高温。也许这项技术可以制造出[钻石电池](https://hackaday.com/2016/11/29/diamond-batteries-that-last-for-millennia/)，一石二鸟。
