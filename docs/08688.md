# 杯架自动消毒器

> 原文:[https://hack aday . com/2020/12/16/automatic-sanitizer-for-your-cup holder/](https://hackaday.com/2020/12/16/automatic-sanitizer-for-your-cupholder/)

为什么在进店和开车回家之间很难记住使用洗手液？我们试着在挡风玻璃雨刷杆上挂一个瓶子，但它很碍事，很难使用并与乘客分享。[最理想的是在汽车上安装一个免提泵，提醒你使用它](https://www.instructables.com/Automatic-Hand-Sanitzer-for-Your-Car/)。

你不需要将它连接到点火器或任何东西上——你所要做的就是用点烟器(或者直插式插座，如果你幸运的话)给它通电。每次你转动钥匙，这个泵就会启动并播放一首小曲来提醒你使用它。从电子学角度来说，这再简单不过了 Arduino UNO 从距离传感器读取你的手，并激活伺服系统，分配三个短泵的异丙醇。休息之后，请查看实际情况。
