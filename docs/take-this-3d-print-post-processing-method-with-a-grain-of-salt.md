# 对这种 3D 打印后处理方法持保留态度

> 原文：<https://hackaday.com/2020/10/27/take-this-3d-print-post-processing-method-with-a-grain-of-salt/>

关于 FDM 印刷厂的印刷品后处理有很多民间传说。支持者对他们的方法深信不疑，这些方法通常旨在强化零件或改善其外观，或者两者兼而有之。但是它们真的有用吗？

CNC Kitchen 的[Stefan]知道一堆轶事不能代替实际数据，因此再次进行了一些有价值的实验，这次是测试在盐中退火的零件的强度。这是他最近用石膏掩埋后的[烘焙印迹实验的后续，在强度增益方面产生了复杂的结果。观众评论说，食盐是一种很好的照片退火介质，所以他开始寻找合适的盐。事实证明，颗粒越细越好——粉状盐包裹得越紧，软化的塑料流动的空间就越小——但粉状盐并不容易找到。最后，他用搅拌机粉碎食盐来制作自己的食物。他还不得不反复试验温度和时间，直到想出一个好的工艺。](https://hackaday.com/2020/09/24/plaster-annealing-3d-prints-for-strength/)

结果令人印象深刻。在垂直于印刷层施加力的情况下测试的 PETG、ABS 和两种 PLA 印刷都显示出断裂后强度的显著增加，达到几乎与印刷有平行于应力的层的部件的强度相匹配的程度。与石膏一样，零件以 100%填充率打印；以 20%印刷的基准在退火后明显不适航。退火零件的表面光洁度是点蚀和白色残留物的有趣组合——不是不好看，但仍然有点奇怪。

盐退火可能有点麻烦，但这是一个很好的方法，可以添加到人们为 3D 打印想出的所有其他后期处理技巧中。
