# 第一款折叠式 IPhone 不是来自苹果

> 原文：<https://hackaday.com/2022/11/11/first-folding-iphone-doesnt-come-from-apple/>

折叠手机如今风靡一时，许多主要的智能手机制造商都有这种外形的产品。苹果在这一细分市场明显缺席，因此[KJMX]决定通过[“iPhone v”.](https://www.youtube.com/watch?v=k1DxL-vyjfs)(YouTube-中文字幕通过 [MacRumors](https://www.macrumors.com/2022/11/08/foldable-iphone-motorola-razr/) )将事情掌握在自己手中。

这些制造商没有试图将现有的折叠手机屏幕与 iPhone 相结合，而是将实际的 iPhone X 屏幕分层用于 mod。他们进行了 37 次尝试，才制造出一种分层的屏幕，既灵活又实用。几款不同的折叠手机被拆卸下来，[KJMX]发现摩托罗拉 Razr 折叠机构最适合 iPhone X 屏幕。不幸的是，由于 iPhone 屏幕不是为折叠而设计的，在相对较少的折叠次数后，它仍然会失败。

还做出了其他牺牲，比如移除 Taptic 引擎和更小的电池，以使一切都符合预期的外形。“iPhone V”拥有迄今为止最差的电池寿命。然而，经过近一年的努力，[KJMX]可以真正地宣称已经做出了苹果没有做到的东西。

好奇其他黑客让 iPhone 做的比苹果预想的更多吗？看看如何给 iPhone 添加 USB-C(T1)，[试着给它充电更快(T3)，或者给它一个](https://hackaday.com/2022/11/03/just-how-fast-could-you-charge-an-iphone/)[大内存升级(T5)。](https://hackaday.com/2018/04/13/the-ultimate-iphone-upgrade/)
