# 简单的 Wi-Fi 猫门解决了额外的动物问题，仅此而已

> 原文:[https://hack aday . com/2022/11/02/simple-wi-fi-cat-door-solves-the-extra-critter-problem-and-not-more/](https://hackaday.com/2022/11/02/simple-wi-fi-cat-door-solves-the-extra-critter-problem-and-nothing-more/)

任何生活中有户外猫的人都知道他们倾向于把宠物带回家，在成为前宠物的不同阶段。任何生活中有黑客的人都知道，有一种趋势是用技术来解决这个问题。但有时，最简单的解决方案是最好的。

就拿这个简单的[步进电动猫门锁](https://hackaday.io/project/188040-wifi-controlled-stepper-motor)来说吧。对于[杰森·温菲尔德]来说，他的户外猫深夜要求重返的根本问题是，在快速目测检查没有夜宵后，不得不手动解锁猫门。这样的活动会让你很难再入睡。对此的一个自然反应是用机器学习来完全自动化这个过程，以识别产品并拒绝进入；毕竟，我们以前见过[和](https://hackaday.com/2022/08/26/machine-learning-gives-cats-one-more-way-to-control-their-humans/)一模一样。但认识到破坏性的部分是起身检查位，[Jason]只是用 ESP8266 微控制器制造了一个简单的步进驱动锁。有了 3D 打印的外壳和电池组，以及附近的 Wi-Fi 摄像头，这种锁拒绝猫进入，直到他看到它，这时他只需点击锁的网页来开门。下面的视频将显示锁的行动，除了那只猫一旦闻到气味就溜走了。猫要猫了。

我们欣赏这个项目的简单性。它解决了最小特征集的问题，这是我们有时看得太少的东西。它也有一些不错的想法，比如如果电池没电了，可以拆下非系留螺栓来开门。思维敏捷，[杰森]，好梦。
