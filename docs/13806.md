# 用激光消灭臭虫

> 原文:[https://hacka day . com/2022/06/06/激光灭虫器/](https://hackaday.com/2022/06/06/bug-eliminator-zaps-with-a-laser/)

蚊子往往被视为一个几乎普遍的负面因素，至少在人类的生活中是如此。虽然它们是许多其他动物的食物来源，甚至可能为一些植物授粉，但它们也携带疟疾和寨卡病毒等疾病，更不用说令人发痒的叮咬了。多年来，人们发明了各种蚊子威慑手段来解决这些问题，但其中一个更有趣的是[Ildaron] [的这个项目，它试图建造一种蚊子跟踪激光器](https://hackaday.io/project/185652-laser-device-for-neutralizing-mosquitoes)。

该设备使用神经学习算法来识别附近飞行的蚊子。一旦检测到蚊子，就会用激光瞄准并激活它，以“热中和”害虫。控制系统以及神经网络和机器学习都托管在 Raspberry Pi 和 Jetson Nano 上，这赋予了它大量的计算能力。这个特定项目的唯一主要缺点是高能激光也可能对人类有害。

理想情况下，像这样的设备的市场会降低价格，甚至可能通过使用像专门为这些蚊子瞄准机器开发的 ASIC 这样的东西。同时，[Ildaron]在他的 [GitHub 页面](https://github.com/Ildaron/Laser_control)上复制了这个项目。我们在[之前也看到过类似的建筑，它们对非飞行昆虫](https://hackaday.com/2019/10/05/tracking-ants-and-zapping-them-with-lasers/)有效，所以它被更广泛地采用似乎只是时间问题——要么如此，要么就是审判日！
