# 用飞行视点显示器照亮夜空

> 原文：<https://hackaday.com/2018/12/19/lighting-up-the-night-sky-with-a-flying-pov-display/>

我们以前已经看到过大量的视觉暂留显示，但[这种天空书写的视点显示](https://www.youtube.com/watch?v=pyHJnP_3FHQ)似乎是第一次。我们不得不同意它的创作者，这是非常酷的。

这个创意人是伊万·米兰达(Ivan Miranda)，他构想了一个飞行视点，作为他的机器人点阵沙滩打印机的变形。但是由于没有任何遥控飞行经验，他向 YouTuber 的同事汤姆·斯坦顿寻求合作，他最近的航拍作品包括[这架气动飞机](https://hackaday.com/2017/12/11/a-remote-controlled-air-plane/)。[Ivan]最初的概念是一个长条状的新像素，它将被附着在一个大翼展飞机的下面。没有太多考虑大多数遥控飞机的有效载荷限制，他想出了一个 3 米长的工作显示器。他下面的视频展示了它在他店里的使用，还有一些令人印象深刻的长曝光图像。

[Tom]的职责是使 POV 显示可飞行。他将长度削减到 2 米，并将重量削减到足以安装到四轴飞行器上。尽管这台机器很笨拙，但他能够很好地控制它，开始在暮色苍茫的天空中绘画。他的视频结尾的图像实际上令人震惊——我们特别喜欢 [*【雷鸟 2】*](https://en.wikipedia.org/wiki/Thunderbirds_machines#Thunderbird_2)，这让我们回到了童年。

我们不确定这有什么实际用途，但这不是重点。这是一个有趣的项目，来自一个不太可能的二人组，这就足够了。
