# TOBOT 是你态度恶劣的井字游戏对手

> 原文：<https://hackaday.com/2020/09/16/tobot-is-your-tic-tac-toe-opponent-with-a-bad-attitude/>

【3dprintedlife】显然有点无聊。他没有玩井字游戏来消磨时间，而是建造了一个令人印象深刻的井字游戏机器人，名为 [TOBOT](https://www.youtube.com/watch?time_continue=1&v=OLSQ40sogNU&feature=emb_logo) 。该机器人使用一个 Rasberry Pi Zero 和一根羽毛来控制一个双轴机器人手臂，该手臂可以使用笔来绘制棋盘和移动。它还使用一个简单的计算机视觉系统来观察棋盘，以了解你的移动，它也有声音。

TOBOT 的另一个特点是态度不好。机器人想要赢。很糟糕。看看下面的视频，你就会明白我们的意思了。

代码在 [GitHub](https://github.com/DDeGonge/TicTacToe-Bot) 上。我们对机器人用来观察你移动的简单算法印象深刻。3D 打印的机器人手臂看起来和工作起来也很棒。最初的 PC 板工作不太好，但更换做得很好。

我们喜欢结合机械、电子和软件的项目。奇思妙想的元素也能加分。我们想知道 TOBOT 是否会学习玩一些其他游戏，如 dots。

你并不真的需要一台电脑来玩井字游戏，尽管它很有用。我们已经看到软件部分减少到一个[对 printf](https://hackaday.com/2020/06/05/tic-tac-toe-implemented-in-single-call-to-printf/) 的荒谬调用。
