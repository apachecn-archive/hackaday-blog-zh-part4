# 3D 打印旋风除尘器

> 原文：<https://hackaday.com/2021/12/17/3d-printing-a-cyclonic-dust-seperator/>

[rctestflight]最近购买了一台大型 CNC 路由器，这意味着是时候在车间安排一些除尘工作了。很自然地，他开始着手自己建造它！

在小规模生产中使用车间真空吸尘器是没问题的，但是在大规模生产中它们会很快被填满。为了防止它很快被填满并浪费真空袋，[rctestflight]想要建造一个 3D 打印的旋风分离器，以收集并倾倒来自路线选择过程的比空气重的颗粒到一个连接的桶中。

[rctestflight]试验了各种设计，从四旋风分离器到大型单旋风分离器，甚至三系列设计。还建造了扩散器设计，旨在将空气流动减慢到颗粒脱离气流的程度。最终，大型单气旋设计被证明是去除气流中颗粒的最有效方法。

从根本上说，如果你正在制造大量的灰尘，旋风分离器是处理这个问题的好方法。我们也看到过类似的构建被放大[来处理整个车间的需求](https://hackaday.com/2018/01/05/automatic-dust-collection-for-the-whole-shop/)。休息后的视频。
