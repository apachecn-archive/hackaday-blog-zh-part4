# Arduino 带来了 MicroPython IDE

> 原文:[https://hack aday . com/2022/11/14/arduino-bring-a-micropython-ide/](https://hackaday.com/2022/11/14/arduino-brings-a-micropython-ide/)

Arduino 和 MicroPython 都是电子教育领域的巨头，它们中的每一个都代表了作为教育工作者不能错过的东西。Arduino 为你提供了一个广泛的廉价硬件生态系统和一个初学者友好的 IDE，论坛帖子解释了你可能会遇到的每一个问题。另一方面，MicroPython 提供了一个强大的成熟的编程环境，如果您试图稍微错误地解析 JSON，它不会像机关枪一样扫射三角括号。他们看起来像是天造地设的一对，而今天，来自天堂的 Arduino 实验室降临了。

这不是一个 Arduino IDE 扩展——它是一个独立的 Arduino IDE 形状的应用程序，可以进行 MicroPython 编辑，并从友好的环境中将代码上传到您的主板。它通过串行端口工作，因此，古老的基于 ESP8266 的板不应该被排除在外-它甚至提供文件管理器功能！Arduino 声明这是一项实验性的工作——例如，它还没有语法检查，也没有做出任何承诺。也就是说，对于初学者来说，它已经是一个非常棒的 MicroPython IDE，而且绝对是朝着正确的方向迈出的一步。想试试吗？[在这里下载，](https://labs.arduino.cc/en/labs/micropython)甚至还有一个 Linux 版本！

高级语言让你更快地构建项目——非常适合对微控制器感兴趣的人。希望接下来是一个 MicroPython 库管理器和存储库！我们在 2016 年第一次[试用了 MicroPython，](https://hackaday.com/2016/07/21/micropython-on-the-esp8266-kicking-the-tires/)从那以后[走了很长的路](https://hackaday.com/tag/micropython/)——我们已经看到了不少初学者友好的 MicroPython 介绍，从[游戏手持编程课程、](https://hackaday.com/2021/05/25/simple-micropython-game-is-a-30-minute-game-dev-course/)到[双足机器人编程 MicroPython 探索。](https://hackaday.com/2020/11/14/micropython-on-microcontrollers/)当然，你可以[带上你的 C 库。](https://hackaday.com/2019/08/31/micropython-and-c-play-together-better/)
