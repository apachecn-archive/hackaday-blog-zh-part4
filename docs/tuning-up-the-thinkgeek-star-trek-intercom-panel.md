# 调整 ThinkGeek Star Trek 内部通信面板

> 原文：<https://hackaday.com/2019/12/06/tuning-up-the-thinkgeek-star-trek-intercom-panel/>

在《星际迷航》中，柯克和他的朋友们所要做的就是按下位于方便位置的对讲机面板上的按钮，开始通话，无论他们在船上的什么地方，预定的接收者都会立即回复。它是如何工作的？谁知道呢。尽管，或者甚至可能是因为，这项技术被轻描淡写地解释过的本质，樱桃红的墙壁对讲机仍然对这场开创性表演的粉丝们有着一定的魅力。

一位观众从 ThinkGeek 那里发来了一个缩小版的对讲机复制品，虽然它看起来非常接近原始道具，但它有一些令人讨厌的设计元素。当被侧面安装的运动传感器触发时，面板将播放自动门的标志性 *swoosh* 或“红色警报”音效。对于孩子的卧室来说，这可能是一个可爱的想法，但对于经常录制 YouTube 视频的人来说，这不太理想。

[![](img/95f78da3a9c40a68ca81326be17b3854.png)](https://hackaday.com/wp-content/uploads/2019/12/tospanel_detail.jpg)

Peak 23rd century technology

所以当务之急是将运动传感器从电路中移除，代之以一个按钮。[Fran]绘制了一个快速图表来解释这些传感器是如何工作的，并展示了它们可以很容易地被瞬时开关旁路，因为它们通常会在被触发时将线路拉高。然后，她将面板右侧的指示灯转换成一个按钮，以启用警报声音效果，无论如何，这更准确地反映了它在节目中的工作方式。

另一个问题，也可能是最令《星际迷航》粉丝震惊的问题是，面板顶部的“红色警报”指示灯实际上并没有像剧中那样闪烁。设计和制造这种面板，而不是在那块磨砂塑料后面放几个 led，似乎有点像生产一辆火柴盒汽车，却忘记让车轮旋转。通过几个红色发光二极管和一些新的线路，这个疏忽很快就被纠正了。

虽然它可能并不完美，但至少 ThinkGeek 这次确实生产了一款功能性产品。它可能会像他们愚人节的“产品”一样永远不会投入生产，[迫使一个绝望的旅行者不情愿地建立自己的版本](https://hackaday.com/2016/11/16/star-trek-phone-dock-might-as-well-be-from-picards-night-stand/)。
