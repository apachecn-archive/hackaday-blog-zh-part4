# 油浸覆盆子酱在重载下保持凉爽

> 原文:[https://hack aday . com/2018/08/13/油浸覆盆子-pi-keep-its-cool-under-heavy-loads/](https://hackaday.com/2018/08/13/oil-immersed-raspberry-pi-keeps-its-cool-under-heavy-loads/)

一般来说，液体和电子不能混合。然而，有一种液体却与这种趋势背道而驰，它能极大地延长某些电路的寿命:石油。从电线杆上的大型电源变压器到变电站的开关设备，绝缘油可以冷却和绝缘所有东西。但是对于更小的电路来说，油又如何呢？

[骨之王]很想知道油冷树莓酱是否可行，简短的回答是:在很大程度上，是的。下面视频中看到的实验设置有些粗糙——只是一个 Pi 运行 Quake 3 一个小时，以真正提高 CPU 温度，这是远程监控的。无论是否安装散热器，在自由空气中，Pi 范围从空闲时的约 50°C 到负载时的约 70°C，这相当热。他承认，将 Pi 泡在纯植物油中是一个糟糕的选择，但这极大地改变了这些数字:闲置时为 37 摄氏度，一个小时的游戏后只有 48 摄氏度。他还测试了 Pi 后清理，这是他遇到一个小问题的地方。clean machine 开始运行良好，但此后不久就出现了一系列重启问题。12 小时后，圆周率是好的，但是，所以他认为几滴还没有蒸发的水是罪魁祸首。

油浸是冷却 Pi 的实用方法吗？大概不会。当然，这并不意味着人们以前没有尝试过，但我们为这种努力和细致的实验喝彩。
