# 3D 打印不仅仅是肉眼能看到的

> 原文:[https://hackaday . com/2019/10/19/there-to-the-3d-print than-the-eye-can-see/](https://hackaday.com/2019/10/19/theres-more-to-the-3d-print-than-the-eye-can-see/)

如果你认为为 3D 打印设计 CAD 已经够难了，那就等着听这个`.stl`技巧吧。

Maker's Muse 的[Angus]最近[演示了一种在`.stl`文件中创建隐藏几何图形](https://www.youtube.com/watch?v=ElJHM6IMIMg)的方法，这种方法只会在 3D 打印之前的切片过程中显示出来。(视频，嵌在下面。)这个过程包括创建厚度小于 3D 打印机喷嘴尺寸的几何图形，这些几何图形在`.stl`编辑器中仍然看起来是实心的，但不会被 FDM 切片器渲染。

大多数 3D 打印机都有 0.4 毫米厚的喷嘴，因此创建壁薄于此值的几何图形将会产生您想要的效果。这种技巧的一些可能的用途是创建复活节彩蛋，甚至是扰乱其他 3D 打印爱好者。当然，[安格斯]建议不要使用这种“犯罪或恶意欺骗”，我不得不同意。

他还透露了一些其他技巧，包括创造一个实际上很薄但看起来很坚固的物体的方法:非常适合制作不可打印的信件，揭示隐藏的信息。

然而，这是一个很酷的技巧，也许是切片软件中的“功能而非缺陷”之一。
