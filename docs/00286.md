# 就连 PlayStation 2 也逃不过 Java

> 原文:[https://hack aday . com/2018/08/07/even-the-PlayStation-2-cant-escape-Java/](https://hackaday.com/2018/08/07/even-the-playstation-2-cant-escape-java/)

不管你喜不喜欢，你都不能否认 Java 在支持的平台方面有着令人印象深刻的记录。从翻盖手机到 DVD 播放器，任何东西都可以使用 Java，更不用说电脑了，甲骨文曾宣称 Java 在 30 亿台设备上运行。事实上，目前的估计可能偏低。尤其是当(迈克尔·科恩)不断想出如何在越来越深奥的设备上运行它的时候。

[![](../Images/246439490d88a58a12fff0cb1d9a776f.png)](https://hackaday.com/wp-content/uploads/2018/08/ps2java_anim.gif) 【迈克尔】来信告诉我们[他已经为 Java Grinder](http://www.mikekohn.net/micro/playstation2_java.php) 增加了对 PlayStation 2 控制台的支持，Java Grinder 是他的软件，用于获取 Java 代码并将其转换成各种意想不到的平台的原生二进制文件。他以前的征服包括 TRS-80 和雅达利 2600，所以相比之下，PS2 是一个几乎驯服的名单。

老实说，你可能不想在索尼近 20 年的游戏系统上运行 Java 程序。但是没关系。无论如何,[Michael]写的文档很吸引人，涵盖了 PS2 相当独特的硬件的细节，以及他在仿真器上开发和在真实硬件上部署时遇到的奇怪情况。即使你永远不会将这些发现付诸实践，它也绝对值得一读。

在休息后的视频中，你可以看到演示[迈克尔]想出了在一个真正的 PS2 上启动，以证明该软件的工程。为了真正打上他的印记，他提到他写并表演了小样的歌曲，甚至在纸上画了一些艺术作品，并扫描到他的电脑里。

我们之前报道过他让 Java 在 Sega Genesis 上运行的工作，以及 T2 令人尊敬的 6502 CPU。哦，还有一次他[将数据编码到一个煎饼上](https://hackaday.com/2017/02/21/pancake-rom-eat-only-memory/)。我们喜欢这家伙的风格。
