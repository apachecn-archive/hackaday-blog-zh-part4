# 低于 50 美元的注塑硅胶部件

> 原文：<https://hackaday.com/2021/09/05/injection-molding-silicone-parts-for-under-50/>

你可能已经在网上看过很多使用 3D 打印模具制作硅胶零件的教程。这些方法中的绝大多数使用简单的浇注方法来填充模具。这依赖于仔细的脱气和温和的浇注来减少最终产品中气泡的存在。[Jan Mrázek]一直在研究一种替代方法，[这种方法允许在家庭商店中以低成本进行注射成型。](https://blog.honzamrazek.cz/2021/08/a-better-way-of-making-silicone-components-using-a-resin-printer-injection-molding-for-less-than-50-usd/)

该工艺依赖于印刷树脂模具的使用。[Jan]指出，这通常需要使用缩合固化硅酮，因为添加剂类型在树脂模具中固化不好。将缩合硅酮混合、脱气并倒入标准筒中。从那里，它被安装在一个硅胶输送气枪中，该气枪使用压缩空气迫使硅胶从喷嘴中出来，进入等待的模具中。

它基本上是使用一堆家庭 DIY 设备来创建一个廉价的硅胶零件注射成型解决方案。[Jan]指出，模具设计需要几个模块来适应这一过程，400-800 kPa 是注射硅胶的合适压力。

在压力下注射硅胶对于复杂的模具设计来说非常好，因为它迫使材料进入所有小的困难的角落和缝隙。当然，我们以前也见过制造硅树脂部件的其他方法。一定要在评论中说出你自己喜欢的生产优质硅胶零件的技术。休息后的视频。
