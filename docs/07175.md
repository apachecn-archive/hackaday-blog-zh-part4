# 苏联核心记忆实验

> 原文:[https://hack aday . com/2020/07/14/Suzuki-core-memory-experiments/](https://hackaday.com/2020/07/14/soviet-core-memory-experiments/)

当你在易贝上买了一些旧的苏联核心内存模块时，你会怎么做？如果你是[CuriousMarc]，你把它连接到一些测试连接器上，用你的测试台看看核心内存是否还能工作。剧透:的确如此。

虽然以今天的标准来看这似乎很粗糙，但曾几何时，这些内存模块可能是他们那个时代令人惊叹的微型技术。每个小磁环代表一个比特，模块由 1024 个和 4096 个小甜甜圈串成一个网格。

核心内存确实有几个有趣的好处。首先，它是非挥发性的。位的状态不依赖于阵列的有功功率。第二，核心内存明显抗辐射。这是最初的航天飞机使用核心内存的一个重要原因，直到测试表明升级到固态内存是可行的。

当然，也有不利的一面。手工制作成本很高，读取一点是破坏性的，需要重写。虽然当时的位密度令人印象深刻，但我们现在看看这些 1K 位和 4K 位器件，再看看最小的 SD 卡，就会发现时代已经发生了变化。

为什么要做这样的事？为什么不呢？我们喜欢为模块制作的外壳，它既有吸引力，又能防止学生或游客戳到精致的小铁氧体环。

你认为核心记忆是最奇怪的旧记忆吗？绝不可能。想要 Arduino 上的核心？[没问题](https://hackaday.com/2018/04/28/core-memory-upgrade-for-arduino/)。
