# 总是软绵绵的时间！

> 原文：<https://hackaday.com/2022/02/09/its-always-floppy-time/>

在[Jan Derogee]的桌子上放着一堆电脑杂志，上面放着一盒 3.5 英寸的软盘，这对我们大多数人来说都不会显得格格不入。因果观察者只会看到一个逆向计算机爱好者存在的碎片，但会留下来。事实上，这是一个时钟，也是我们很久以来见过的最不寻常的时钟之一。

一盒软盘怎么能告诉我们时间？选定的磁盘有定制的标签，看起来好像是真实的游戏收藏，但实际上是带有数字和游戏艺术的赝品。隐藏在掏空的杂志堆中的一个巧妙的凸轮系统会升起正确的软盘来显示时间，左边一排是小时，右边是分钟，最接近的五分钟。软盘是通过移除一些塑料和磁盘本身来修改的，因为早期版本有粉碎磁盘的习惯。最后一点是桌子上时钟旁边的诺基亚手机，它会播放一首曲子来代替整点的钟声。你可以在休息时间下方的视频中看到时钟的运行。
