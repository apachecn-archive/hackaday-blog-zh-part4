# 一个芯片就能完成这一切

> 原文:[https://hack aday . com/2020/01/27/one-chip-do-it-all-in-this-macguyver-synth/](https://hackaday.com/2020/01/27/one-chip-does-it-all-in-this-macguyver-synth/)

当您想到简单的合成器时，您会想到哪些组件？你真正需要做的是一个振荡器，一个放大器，和一些输入，这样你就可以演奏不同的音符。我们最喜欢用来产生方波的可能是 40106 IC，它有六个反相施密特触发器，通常还有一个 386 来放大输出。

但有可能比这更简单，学校正在上课，由(朱丽)授课。[【Jule】的小模拟合成器使用一个集成电路作为振荡器和放大器](http://instructables.com/id/Make-an-Awesome-Analog-Synthesizer/)——一个 TL072 运算放大器。其余部分由纯粹的分立元件组成。

[Jule]说这些瞬时开关是低于标准的，如果按下时适当摆动，会增加颤音效果。对我们来说，这些按钮看起来很漂亮，比直径为 1/8 英寸的驱动器更容易拔出来。此外，每当你按下多个按钮时，附加的阻力会解锁合成器内部的 R2D2 声音。我们真的没有看到这里的缺点。

默认情况下，这是一个调到 c 大调的八按钮合成器。但是有一个惊喜——你可以将不同的电容器插入一个头部，并动态改变八度音程。休息之后来看看。

制造音高正确的频率需要奇怪的电阻值，我们通常可以用两个串联电阻来满足。但是等等，电阻值到底是怎么回事？为什么它们有颜色代码？
