# 微型激光切割机使微型步进机开始工作

> 原文：<https://hackaday.com/2020/06/03/tiny-laser-cutter-puts-micro-steppers-to-work/>

来自中国的廉价激光切割机的涌入对创客运动来说是一个福音，即使是以大量的修补工作为代价。所以有些人更喜欢自己动手，认为从零开始意味着你会得到你想要的。显然[迈克·兰金]想要的是一个非常非常小的激光切割机。

[迈克]亲切地称他的发明为 ESP32 Burninator，它小到有被意外放错地方的危险。这个平台依靠微小的步进驱动线性驱动器，从全球速卖通可以便宜地买到。整个机械结构是两个 PCB——一个垂直的部件，容纳 ESP32、有机发光二极管显示器、X 轴电机和来自旧 DVD 刻录机的激光驱动器；较小的底板支撑 Y 轴和载物台。“Stage”实际上是对这种切割机邮票大小的工作区域的一个相当宏伟的术语，但下面的视频显示，它确实切割黑纸。

削减是有点摇摆不定，但这肯定是预期的考虑到运行齿轮，我们喜欢它无论如何。这让我们想起了肖恩·哈金斯不久前做的那台小到可以放进圣诞装饰品的树脂 3D 打印机。不过，我们建议不要把它挂在树上。
