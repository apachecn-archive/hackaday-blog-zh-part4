# 使用电子路由器升降机为木工车间带来精确性

> 原文：<https://hackaday.com/2022/04/20/bring-precision-to-the-woodshop-with-an-electronic-router-lift/>

木工从金属加工人群中得到的一个打击是他们选择的材料有点…顺从。测量只需要在 1/16 英寸左右，或者大约 1 毫米的范围内，这取决于你在大西洋的哪一边。如果你偏离了一点呢？别担心，这就是砂纸的用途。

[这款电子刳刨机升降机](https://www.42pursuit.com/projects/electronic-router-lift)旨在缩小精度差距，让木工活不那么主观。[GavinL]的构建说明显然是针对那些没有涉足 Arduinos 和步进电机领域的木工，他做了一项令人钦佩的工作，解决了这个群体在处理这样一个构建时可能会感到的犹豫。幸运的是，这个项目的许多机械方面可以通过商用刳刨机升降机来解决，该升降机连接到桌面安装的插入式刳刨机，并允许从桌面上方微调切割工具的高度。

剩下的就是添加一个 NEMA 23 步进器来驱动路由器升降机，加上一个 Arduino 来控制它。[GavinL]提出了一些不错的功能，如快速点动控制，微调编码器，以及快速将工具向上或向下移动的能力。另一个非常好的接触是接触传感器，这是一对临时附着在工具上的磁性探针和一个指示触发的高度规。查看下面的视频，了解所有的操作。

我们对[GavinL]的设置有一个异议，那就是步进机将承受的灰尘量。他可能需要尽快换上防尘踏步机。即便如此，我们认为他在弥合机电一体化和木工之间的差距方面做了很大的工作——这也是[【马蒂亚斯·万德尔】在](https://hackaday.com/2017/09/26/smooth-and-steady-cuts-with-an-improvised-power-feeder/)方面一直在做的工作。
