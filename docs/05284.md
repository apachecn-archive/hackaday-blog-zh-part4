# 探索这个以 ASCII 艺术呈现的 3D 世界

> 原文：<https://hackaday.com/2020/01/01/explore-this-3d-world-rendered-in-ascii-art/>

像素化 RPG 在像《塞尔达传说》和《口袋妖怪》这样的游戏中是相当标准的，但是你见过像 [ASCIICKER](http://asciicker.com/y0/) 这样的东西吗？这是一个用 ASCII 艺术渲染的全彩三维世界，可以在你的浏览器中播放。

就目前而言，这款游戏是作为一项实验而存在的。除了探索世界之外，没有故事情节或目标，尽管你可以遇到(或跟随)其他探索游戏的人——尽管所有的精灵看起来都一样，所以可能很难进行互动。游戏由[Gumix]创建，完全用 JavaScript 构建，没有使用任何其他游戏引擎。

所有之前的迭代也已经在线发布，可以通过在 URL 末尾添加 X1 到 X13 来访问。随着游戏开发从 2017 年开始，它已经经历了相当大的变化。有一个完整的[子编辑](https://www.reddit.com/r/asciicker/)致力于游戏，开发者定期更新开源开发工具的开发，为环境建立新的水平和功能。目前的引擎能够渲染像栅栏一样薄的物体以及水体中的倒影。

你可以[亲自尝试一下这个游戏](http://asciicker.com/y0/)看看你有什么想法！
