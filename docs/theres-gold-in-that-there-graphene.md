# 石墨烯里有金子

> 原文：<https://hackaday.com/2022/08/23/theres-gold-in-that-there-graphene/>

我们周围都是金子，嵌在我们的电子产品中。还有人收集电子垃圾，用各种方法从中提炼黄金。然而，很难将其称为“快速致富”计划，因为回收的黄金数量通常很少。尽管如此，如果你能批量生产，你可以赚一些钱，回收利用总是一个好主意。在曼彻斯特大学，他们有更好的方法利用石墨烯从电子垃圾中提取黄金。你可以在下面看到关于这个过程的简短视频，或者[阅读全文](https://www.nature.com/articles/s41467-022-32204-4.pdf)。

过程相对简单。你将电子垃圾溶解在一种溶剂中，加入一些氧化石墨烯，金就会与石墨烯结合在一起。你取出石墨烯，烧掉它，得到你想要的黄金。一克石墨烯可以抓住 2 克黄金，与黄金相比，每克石墨烯相对便宜。

使用抗坏血酸将氧化石墨烯纳米片加工成胶体悬浮液。这个化学过程将与氯结合的金转化为元素金。在探究了这一过程的工作原理后，他们能够通过控制 pH 值来增加这一过程的选择性，从而使大部分残留物实际上是金。

该团队认为，他们可以建立一个连续的过程，在液化电子垃圾流经系统时提取黄金。如果你更愿意用传统的方法，[这里有一个适合你的开始](https://hackaday.com/2013/11/18/salvaging-gold-from-old-electronics/)。话又说回来，除了黄金，还有其他[金属可以回收。](https://hackaday.com/2020/11/30/recovering-metal-from-waste/)
