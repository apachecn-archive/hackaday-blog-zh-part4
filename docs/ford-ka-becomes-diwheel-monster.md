# 福特·卡变成了迪轮怪兽

> 原文：<https://hackaday.com/2018/10/24/ford-ka-becomes-diwheel-monster/>

如果你参加过正确的赛事，你以前就见过它们——带有外部笼子的赛车，让赛车在紧急制动下向前完成一个筋斗。它们令人印象深刻，但还可以更进一步。[输入【mastermilo82】和 RollKa](https://www.youtube.com/watch?v=orNqEGhs8F4) 。

RollKa 是 RollGolf 的延续，roll golf 是一款简单的 roll 汽车。围绕福特 Ka 建造，它避开了外部笼子，设计更加激进。Ka 已经被缩短，设计成适合两个巨大的钢圈，包裹着汽车的每一侧。额外的滚轮被焊接到 Ka 的车顶，使其能够有效地在外部钢圈内滚动。

这是一个相当古怪的设计，被称为 diwheel。在之前，我们已经看到了令人印象深刻的电动版本，但至少在现阶段，这个项目似乎缺乏任何先进的控制系统，完全依靠运气和焊接技术。建造仍处于早期阶段，随着《T2》第三集《T3》开始在 power 下进行一些早期动作测试。这证明了一个宽敞的车库和一些想象力可以实现什么，我们迫不及待地想看看接下来会发生什么！休息后的视频。
