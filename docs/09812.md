# 从政府仓库里挖出一个古老的游戏

> 原文:[https://hack aday . com/2021/04/12/掘出一个来自政府仓库的古代游戏/](https://hackaday.com/2021/04/12/exhuming-an-ancient-game-from-a-government-warehouse/)

许多读者将会熟悉《夺宝奇兵》的最后一幕，在这一幕中，被印第安纳·琼斯找回的约柜被放在一个板条箱里，然后被推走，消失在一个布满灰尘的政府仓库的看似无限的深处。谁知道这种虚构的纳税人资助的仓库里隐藏着什么宝藏，似乎是为了证明它们的巨大潜力，[亚瑟·奥德怀尔]讲述了[一个数字考古的故事](https://quuxplusone.github.io/blog/2021/03/09/castlequest/)，在一名公务员的帮助下，一款被认为早已失传的游戏的完整源代码被重新载入。

这款游戏名为 *Castlequest* ，他曾于 20 世纪 80 年代在现已倒闭的 [GEnie](https://en.wikipedia.org/wiki/GEnie) 在线服务上玩过。极少的网上参考之一来自[美国版权局](https://cocatalog.loc.gov/cgi-bin/Pwebrecon.cgi?Search_Arg=TXu000091366&Search_Code=REGS&CNT=10&HIST=1)版权目录中的一个条目，版权所有者可以选择注册他们的作品。最终，经过一些侦查工作和与游戏作者之一的交谈，他收到了参赛作品的副本。但是，他惊喜地发现了游戏的完整 Fortran 代码，而不是预期的摘要。问题在于，它是打印页面的 PDF 扫描版，而不是代码本身，因此在发布到 GitHub 库和[最终成为可编译的](https://quuxplusone.github.io/blog/2021/03/11/castlequest-update/)之前，会有一个冗长的转录过程。该代码作为其故事的重要组成部分仍受版权保护，但如果你感兴趣，你可以回到 40 年前，在文本冒险中试试运气。

也许在那些满是灰尘的版权仓库里会有更多的东西被发现，寻找它肯定比挖垃圾填埋场更令人愉快。
