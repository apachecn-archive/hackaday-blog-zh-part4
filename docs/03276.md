# 小坦克检查你的爬行空间

> 原文：<https://hackaday.com/2019/06/18/tiny-tank-inspects-your-crawlspace/>

如果你有一些无人机或 FPV 部分躺在周围，这是为你建立。这是一个遥控坦克，带有摄像头和视频发射器，大小只有 65 毫米×40 毫米×30 毫米。你究竟为什么要造这么小的东西？我想你可以在你的爬行空间四处看看。无论怎么看，这东西都很小。

该坦克通过两个无刷电机实现传统的坦克滑行转向。电池是一个电池，因为这是你可以放在这么小的车辆中的最大电池，相机只是现成的四轴飞行器，放在 3D 打印的外壳中。有几个 led 灯。除此之外，它实在是太小太可爱了。

这辆坦克的建造者[honnnest]发布了一个视频，展示了这款坦克的建造过程，并展示了你可以从这么小的坦克中得到什么样的视频。对于一辆坦克来说，这有点快，而且这还没有考虑到比例效应，但如果底盘是 3D 打印的，你也可以打印一些减速齿轮。
