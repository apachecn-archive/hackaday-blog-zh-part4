# 周五黑客聊天:玩火

> 原文:[https://hack aday . com/2018/09/12/Friday-hack-chat-玩火/](https://hackaday.com/2018/09/12/friday-hack-chat-playing-with-fire/)

我们很确定所有的黑客、修补者和制造者在他们年轻的时候都有一点点纵火狂。这就是为什么本周的黑客聊天如此令人兴奋:我们正在谈论[用火](https://hackaday.io/event/161057-hacking-with-fire-hack-chat)黑客。

我们本周 Hack Chat 的嘉宾是[Brice Farrell]，他像我们大多数人一样，一生都对火感兴趣。他利用这种兴趣，把业余爱好变成了半专业的东西。他是 PGI 认证的烟火技师、电气工程师和冰雕师。今年，他出现在*战斗机器人*上，在那里他用奶酪为《皇家战役》建造了火焰系统。

[![](../Images/7d90915d831f5c3ded526a29d0a662a9.png)](https://hackaday.com/wp-content/uploads/2018/09/hackingwithfirehackchatv1-01.png) 鉴于[Brice]丰富的专业知识，本次黑客聊天将涵盖与动火作业相关的安全问题，如何保护自己的安全，以及如何合法行事。我们将讨论各种尺寸的火球射手，点火技术，以及战斗机器人中火的使用(和介绍)。最后一点非常有趣:对战斗机器人开火真的有用吗？你能做些什么来保护你的机器人免受其害？

本次黑客聊天的关注点包括:

*   消防安全
*   产生火焰和火球的区别
*   点火技术
*   消防安全
*   战斗机器人中的火
*   消防安全
