# 会说话的洗衣机是视力受损者的一个干净的解决方案

> 原文：<https://hackaday.com/2019/05/04/talking-washer-is-a-clean-solution-for-the-visually-impaired/>

你最近购买过电器吗？它们都是发光二极管，到处都是发光二极管。你可能会说制造商不了解触觉控制的效用。【小翼翔】的花式新洗衣机就是用这种现代布料裁剪的。虽然它有一个很好的选择循环的大旋钮，但你选择的唯一指示是一个 LED。这对[小翼翔]来说不是问题，但对他视力有障碍的妻子来说却是一个阻碍。

他们试图用那些用来保持柜门安静的胶粘橡胶脚为她最常用的自行车制作触觉路标。但是在机器的 14(！)不同的洗涤周期和无休止旋转的选择器旋钮，触觉地图的想法是洗。是让机器说话的时候了。

[对于他的第一个微控制器项目，【小翼】为这个问题设计了一个完全非侵入性且非常棒的解决方案](https://www.instructables.com/id/Talking-Washing-Machine-Interface-for-the-Blind-AK/)。他使用排列成环形的 ldr 来检测哪个 LED 被点亮。回收的鼠标垫泡沫和黑色苯乙烯防止环境光产生假阳性，并兼作传感器和支撑板的外壳。当[小翼夫人]用旋钮循环时，安装在附近项目盒中的 Arduino 克隆体决定选择哪个程序，Velleman KA02 音频屏蔽播放[小翼]的录制剪辑，宣布循环数和描述。

这个被称为 SOAP(语音输出通知程序)的系统在过去的一年里给了小翼夫人很大的帮助。休息之后，观看她带着它转一圈，并留下来观看 SOAP 的起源故事和演示视频。

令人困惑的是，很少有洗衣机和烘干机会让你知道它们什么时候洗完。不要浪费时间一遍又一遍地检查— [洗衣间谍等待振动结束，然后给你发短信](https://hackaday.com/2018/05/20/internet-of-laundry-let-the-esp8266-watch-your-dirty-drawers-get-clean/)。
