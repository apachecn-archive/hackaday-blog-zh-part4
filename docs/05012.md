# 用 3D 打印将史密森尼博物馆带回家

> 原文：<https://hackaday.com/2019/12/09/bring-the-smithsonian-home-with-3d-printing/>

如果你去过华盛顿 DC，你会知道史密森尼不仅仅是一座建筑，而是由 19 个博物馆、21 个图书馆、9 个研究中心和一个动物园组成的集合。即使有成百上千的附属博物馆，也有一种方法可以让你至少了解一部分博物馆。史密森尼博物馆有一个 3D 数字化门户,目前展示了收藏中的 124 个物品模型。其中近 100 家有你可以下载并打印的模型——或者让人帮你打印。

如果你已经有了打印机，自己打印可能是最划算的选择。根据史密森尼博物馆的说法，如果你想要一个 1/20 比例的霸王龙头盖骨模型，Shapeways 将会以大约 21 美元的价格提供。如果你想要尼尔·阿姆斯特朗(Neil Armstrong 英寸版本的宇航服，价格大约在 130 美元左右。

有些模型相当复杂。例如，阿波罗 11 号舱门有相当多的细节。这些模型从太空到观世音菩萨，再到超新星遗迹的模型。还有费城炮艇和一只蓝色螃蟹的模型。

当然，您可能需要缩放和支持模型以进行打印。我们还怀疑他们中的许多人会在异国情调的细丝或印刷更好看。例如，哈里发的陪葬浮雕半身像可能会受益于一些石头或大理石状的细丝。

我们跟踪了[项目出口](https://hackaday.com/2019/07/03/forty-four-hackers-and-a-hatch-progress-egress-takes-off/)，但不知道史密森尼提供了这么多可打印的模型。如果你真的想要 NASA 的模型，你可以随时联系 [NASA 本身](https://hackaday.com/2016/11/08/nasa-puts-its-3d-models-up-on-github/)。
