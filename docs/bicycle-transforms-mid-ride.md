# 自行车改变了骑行过程

> 原文：<https://hackaday.com/2019/11/13/bicycle-transforms-mid-ride/>

对于我们这些 80 年代末 90 年代初的孩子来说，我们可能梦想有一天拥有一辆巨大的牵引拖车，它可以变成一个巨大的战斗机器人。或者简单地拥有一个玩具，可以近似这种从一种形式到另一种形式的变化。然而，作为成年人，我们已经意识到这是一厢情愿的想法。也就是说，除非我们决定[建造这辆变形自行车](https://www.instructables.com/id/Transforming-Bicycle-Tall-Bike-to-Chopper-Bike/)。

起初看起来有点不寻常的低骑手风格的自行车通过固定在自行车后三角的气瓶毫不费力地变成了高自行车。自行车开始是一个全悬挂山地车，但后弹簧被删除，为这个气缸腾出空间。山地车后三角的枢转动作是这里的关键设计元素:在这种情况下，当被气缸推动时，它允许车架容易地改变形状。在前面加上一些更长的叉子和一层油漆就完成了建造。

这辆自行车参加了 Instrucables 的“让它动起来”比赛，到目前为止已经得到了广泛的认可。毫无疑问，这是一辆独一无二的自行车，我们已经见过几辆了。从[俄罗斯越野电动多用途自行车](https://hackaday.com/2019/07/28/russian-ebike-goes-everywhere-possibly-legal/)到[防止司机超速行驶的自行车](https://hackaday.com/2019/06/15/enforce-speed-limits-with-a-rusty-bike/)，出现了许多有趣的基于自行车的产品。
