# 一瓦的水培照明系统

> 原文：<https://hackaday.com/2022/10/31/a-single-watt-hydroponic-lighting-system/>

水培系统是一种越来越受欢迎的使用最少资源在室内种植植物的方法。甚至一些商业农业操作正在使用水培种植技术，因为这些方法比传统农业方法消耗更少的水、土地面积和其他资源。不利的一面是，所需的照明系统通常需要大量的能源。这就是为什么[ColdDayApril]设立了一个挑战，用不超过一瓦特的水栽培植物。

该系统被设置成在所谓的深水栽培中种植单一的辣椒植物，在深水栽培中，植物被悬浮在含有其生长所需一切的营养液中。lightning 系统基于三星 LM301B，它接近将电能转换为白光的物理极限，可以管理大约 220 流明。这些低功率二极管需要一个特殊的电源，并且使用专门建造的反射外壳将光有效地导向植物。通过将这个组件放置在非常靠近植物的地方，并随着它的生长进行调整，[ColdDayApril]能够在 92 天内使辣椒从种子到开花。

值得注意的是，系统的其余部分也消耗了一点能量。一个两瓦的风扇有助于溶液培养室内的空气循环，深水系统通常需要一个空气泵来给水充氧，这需要另外两瓦的功率。这仍然是一个令人印象深刻的成就，因为大多数业余建筑爱好者使用数百瓦的照明系统，并使用更多数量级的能源。但是，[如果你愿意在系统中添加一些鱼的话](https://hackaday.com/2012/06/27/urban-farming-uses-aquaponics-to-make-farmland-where-there-is-none/)你可以进一步减少管理水系统所需的一些能量需求。
