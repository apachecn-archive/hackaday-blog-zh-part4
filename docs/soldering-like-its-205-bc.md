# 像公元前 205 年一样焊接

> 原文：<https://hackaday.com/2018/09/18/soldering-like-its-205-bc/>

你有没有停下来想过焊接的发现是多么的不可能？很难想象是什么顺序的事件导致了它；毕竟，在适当的助熔剂存在下，将金属加热到合适的温度，同时使用正确比例的铅锡合金，这似乎不是偶然发生的事情。

幸运的是，Clickspring 的[Chris]目前正致力于重现古代可能会使用的工具和技术，他制作了一部精彩的视频，讲述了用传统方式进行精密软焊的过程。下面的视频是他正在制作的一个侧面系列的一部分，他正在制作一个 [Antikythera 机械装置](http://hackaday.com/2015/11/23/the-antikythera-mechanism/)的复制品，这是一个奇怪的古代模拟天文计算机。机制中的许多部分都是焊接的，并且[克里斯]使用已知的工具和材料探索了可行的方法，这些工具和材料是在机制建造时可用的(被不同的历史学家报道为公元前 205 年到公元前 70 年左右的任何时间)。他的铁是锻造的铜块，他的热源是炭火，他的焊料是铅和锡 60:40 的混合物，就像我们今天使用的一样。他生动地展示了表面预处理和通量是多么重要，并显示了主动和被动通量。他决定用松香做最后的接缝，结果非常光滑完美；我们怀疑这需要相当多的练习才能掌握这项技术，但是像往常一样，[克里斯]让它看起来很容易。

如果你想更深入地了解现代技术，我们已经深入讨论了[焊料和助焊剂的物理学](http://hackaday.com/2017/02/23/what-the-flux-how-does-solder-work-anyway/)。如果你需要更多那些甜蜜的点击春天的视频，[我们也为你准备好了](http://hackaday.com/2016/12/06/for-your-binge-watching-pleasure-the-clickspring-clock-is-finally-complete/)。
