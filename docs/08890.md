# 一只可以无线上网的狗！

> 原文:[https://hackaday.com/2021/01/07/a-wi-fi-enabled-dog/](https://hackaday.com/2021/01/07/a-wi-fi-enabled-dog/)

我们的犬类朋友已经在我们身边数千年了，它们的史前狼祖先和我们一起进化成了我们今天知道的品种。但是令人惊讶的是，直到现在还没有一只狗可以无线上网，至少根据[Entropy]的说法，他的狗[Kaya] [现在戴着一个彩色的无线上网项圈](https://randombits.eu/my-dog-has-wifi)。

发光狗项圈和挽具已经和我们在一起有一段时间了，通过让它们在晚上可见来保护动物免受事故的伤害，但[Kaya]的 colar 是一个特别令人失望的例子。它的单一光线和无光泽的光纤加上令人失望的电池寿命还有很多不足之处，所以当它坏了的时候，有足够的理由升级它。一条可寻址的 led 和一个 ESP32 模块，以及一个 18650 锂离子电池。我们有点不确定是否可以用手机控制灯光，但也许最大的好处在其他地方。如果[Kaya]走失，ESP32 的 Wi-Fi 热点可以作为一个信标，在短距离内找到她，正如任何狗主人都会告诉你的那样，当他们调查一些迷人的新气味而忽略你的呼叫时，这可能是一个福音。你可以在视频中看到她在制作衣领。

犬类黑客不时出现在这些网页上。我们最喜欢的是这只不太成功但非常有趣的遥控狗。
