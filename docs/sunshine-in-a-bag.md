# 袋子里的阳光

> 原文：<https://hackaday.com/2020/10/03/sunshine-in-a-bag/>

如果你有一台树脂 3D 打印机或使用紫外线粘合剂，紫外线(UV)固化灯至关重要。一些人在旧的亚马逊运输箱上贴上箔纸，然后在里面的某个地方放上一盏聚光灯。其他人在全天然光源下工作，索尔。这两种选择都有便携性和可靠性的问题，但[AudreyObscura]用一个衬有紫外线灯的反射垫覆盖了它。这个 HackadayPrize2020 的决赛选手体现了一个理想，即好的想法往往很简单，而且它的材料清单非常短。

箔泡绝缘是介质，因为它提供了结构和反射性，但它不与 LED 灯条的粘合剂配合。[AudreyObscura]演示了作为界面层的遮蔽胶带让每个人都玩得很好。一个有经验的制造商的好例子，他们的设计包括捆绑电线和绝缘连接，以保持一切整洁和隔离。通过不同的安排，这可以形成一个从上面点燃的隧道，一个从墙上点燃的烟囱，或者你可以把它挂在一些脚手架上。

如果你需要一些不太方便携带的东西放在你自己的店里，你可能会考虑一个装满镜子的房间。一个很好的触摸添加是[一个转盘，以帮助确保整个部分是固化](https://hackaday.com/2014/11/24/a-uv-lightbox-for-curing-prints/)没有任何遗漏的领域。
