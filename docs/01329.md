# 对不起，亚历山大，恐怕我不能那样做

> 原文：<https://hackaday.com/2018/11/24/im-sorry-alexander-im-afraid-i-cant-do-that/>

把人送上太空极其困难，虽然把机器人送上太空仍然很有挑战性，但这要容易得多。出于这个原因，几十年来，机器人和探测器一直在帮助我们探索太阳系。然而，现在，一个机器人助手在国际空间站上与宇航员一起工作，而不是像机器人手臂一样没有人情味的东西，这个机器人有一张脸，可以在船上导航，并可以对语音输入做出反应。

该机器人被称为 CIMON，是船员互动移动伴侣。由空中客车公司制造的这种交互式助手将与德国宇航员亚历山大·格斯特一起飞行，以测试像这样的机器人助手的概念。它能够在机舱内自由移动，并且能够了解自己所处的空间，而无需专门为其编程。它像智能手机一样处理语音输入，但仍然通过 IBM Watson AI 处理地球上的请求。这意味着它并不是完全独立的，未来这项技术的实现可能需要在低地球轨道以外的任务中更加独立。

虽然设计师们听取了 2001 年的警告，没有让它完全控制空间站，但他们也知道，创造一个互动机器人是有帮助的，它不会像一个令人毛骨悚然的红眼那样令人讨厌。这个机器人可以在屏幕上显示一张互动的脸，也可以使用同一个屏幕来显示示意图、程序步骤或宇航员需要的任何其他东西。如果令人毛骨悚然的设计更符合你的风格，你仍然可以让哈尔在你的房子里看着你。
