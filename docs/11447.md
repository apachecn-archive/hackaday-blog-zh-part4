# 多力多思太空计划的良好开端

> 原文:[https://hack aday . com/2021/09/25/a-promising-start-for-the-Doritos-space-program/](https://hackaday.com/2021/09/25/a-promising-start-for-the-doritos-space-program/)

火箭技术是棘手的东西，但只要你不试图进入太空，整个想法基本上可以归结为一个简单的概念:如果你给它足够的推力，任何东西都可以飞起来。至少在一段时间内。正是这一基本前提允许爱好者们有时称之为“古怪的人”飞行；这些不寻常的物体可能不是理想的火箭，但在那里放一个足够大的发动机，它就会离开发射台。

最近，[concretedog]想尝试组装他自己的古怪火箭，并着手为动力飞行改造一个多力多滋 STAX 管。把品客薯片管变成火箭有很多先例，但当然，这并不奇怪。毕竟，如果不是一个坚固而轻便的圆柱体，火箭算什么？但 STAX 管的圆形三角形有望成为一种有趣的节奏变化。而且看起来很酷，所以就这样了。

把零食容器变成火箭其实很简单。首先，[混凝土狗]在一张纸上画了一圈管子的外面，然后用他的手机拍了张照片。那张图片随后被导入 Inkscape，并转换成一个他可以在 CAD 中随意摆弄的矢量文件。

在他的激光切割的薄胶合板和载入他的 3D 打印机的 PETG 之间，他能够拿出一个足够强大的马达支架来携带 Estes D12-5。然后他创造了一些鳍粘在侧面，和一个三角形鼻锥。安装了一个简单的回收系统，整个过程以适合多力多滋的橙色和黑色配色方案结束。

火箭不寻常的形状意味着没有定制软件，在计算机上模拟它的飞行特性是行不通的，所以[concretedog]不得不使用老式的方法来检查稳定性，用绳子把它绕成一个圆圈。在修剪后，它会在系绳上正确地定位，他相当肯定它会在动力下直线飞行。果然，下面的视频显示了玉米片奶酪味道的火箭以令人印象深刻的速度和稳定性向天空飞奔。

这远不是我们最近见过的最先进的模型火箭，但我们真的很欣赏这种建造的简单性。这是一个很好的提醒，乐趣不一定是高科技，通过遵循一些基本的建设原则，你可以在周末敲出一个安全的公园传单火箭。
