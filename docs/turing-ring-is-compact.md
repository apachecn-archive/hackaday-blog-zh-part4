# 图灵环是紧的

> 原文：<https://hackaday.com/2021/12/28/turing-ring-is-compact/>

经典图灵机的一个问题是磁带必须无限长。【马克的】[图灵环](https://hackaday.io/project/183146-turing-ring)还是没有无限带，但是为了节省空间确实做成了圆形。再加上一个非常聪明和有能力的用户界面，使它成为我们见过的最有用的图灵机之一。你可以在下面的视频中看到一个演示。

该设备使用 Arduino Nano、Neopixel 戒指、编码器和激光切割外壳，看起来很棒。最小化用户界面有几种模式，下面的视频将带你了解所有的模式。

你甚至可以将磁带存储到 EEPROM 或串口上，这样就更实用了。当然，你可能想知道图灵机有什么实用性？诚实的答案是不多，但从好的方面来看，这款机器也有时钟模式，所以即使你不解释图灵机是如何工作的，它也会成为一个很好的话题。

除了简单的用户界面之外，外壳上的字体也给我们留下了深刻的印象。[Mark]在一次比赛中赢得了 LED 戒指，所以对他来说，它的总成本很低，但即使你买下所有的部件，也不会太多。

这个当然比我们看的最后一个图灵机更紧凑。如果你需要复习图灵机以及它们为什么重要，这里有[为你准备的一些读物](https://hackaday.com/2016/08/18/the-turing-tapes/)。
