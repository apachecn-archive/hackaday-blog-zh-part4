# 在你的办公桌上放一只蚂蚁，制造一些小型 PCB

> 原文:[https://hack aday . com/2019/07/27/add-an-ant-to-your-desk for some-compact-PCB-manufacturing/](https://hackaday.com/2019/07/27/add-an-ant-to-your-desk-for-some-compact-pcb-manufacturing/)

通常，当一个人想到使用数控机床生产 PCB 时，他会想到那些大而笨重的数控机床，它们几乎占据了整个桌子。但是，如果一台数控机床足够小，可以放在桌子上而不碍事，但仍然可以用来制造单面和双面印刷电路板，那会怎么样呢？这是 Ant 背后的想法，这是 Mattia 和 Angelo 设计并开源的紧凑型 PCB 制造机 T1。

除了上面链接的该项目的 Bitbucket 存储库,“Ant Team”还有一个 [YouTube 频道](https://www.youtube.com/channel/UCX44z-SSL7LzcB4xxgUdHHA),上面有一系列关于该项目的经过专业编辑的视频，从构建小机器到各种更新等等，还可以查看链接后附加的视频，了解该项目的可视化介绍。

支持和社区互动主要是通过这个项目的 Reddit 组来完成的，在那里这个小机器随着它的继续发展找到了一个受欢迎的社区。在这一点上，机器本身被指定为能够从商业上可用的 3D 打印部件中构建，不需要更多的切割或成型工具。精度约为 0.2 mm 走线间距。

使用 USB 微型摄像头和 bCNC 软件可以实现双面板的光学对准，与商业解决方案相比，材料成本据说非常便宜

老实说，看到机器运转后，你不会想要一台这么小又好看的数控机床放在你的办公桌上吗？如果有一件事可能要补充，这可能是一种处理产生多氯联苯时产生的铜粉尘的方法。我们可以想象，每次 PCB 制造会议结束后，都要清理桌子上的东西，这有点麻烦。
