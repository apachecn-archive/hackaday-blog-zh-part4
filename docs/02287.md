# DIY 回流炉被大量记录

> 原文:[https://hack aday . com/2019/03/10/DIY-reflow-oven-is-heavy-documentated/](https://hackaday.com/2019/03/10/diy-reflow-oven-is-heavily-documented/)

[DJ Legion]决定他想要一个回流炉，所以他买了一个烤箱和各种零件，包括一个固态继电器、一个扬声器、一个显示器和一个热电偶。让这个项目与众不同的是[视频文档](https://www.youtube.com/watch?v=3P6qYwsLcVo)的数量。下面的四个视频包含了大约 50 分钟的信息，他承诺会有更多的信息。

我们还没有找到他的软件——可能是因为他还在研究这个软件，但我们正满怀期待地关注着他的 GitHub 页面。我们非常喜欢将控制器集成到烤箱中的 3D 打印面板。它几乎看起来像一个商业单位。在 3D 打印的零件上使用木纹纸是一种很好的触摸。

[DJ]还没有完成，虽然。他想增加加热时间，并可能增加对流。他还计划制作一个新视频，展示实际制作电路板的过程，以及他如何改进校准曲线。

我们印象深刻，但也很想抓住一个[回流图](https://hackaday.com/2018/05/15/reflowduino-put-that-toaster-oven-to-good-use/)。工作已经完成了，你可以得到无线控制和许多其他好东西。当然，我们也有自己动手的冲动，所以不能真的去扔石头。如果你不想摆弄烤箱，你可以随时[突袭发廊。](https://hackaday.com/2014/03/30/reflowing-with-a-hair-straightener/)
