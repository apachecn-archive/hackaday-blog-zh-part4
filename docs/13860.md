# LTSpice 提示和一篇很长的教程

> 原文:[https://hack aday . com/2022/06/07/lt spice-tips-and-a-long-tutorial/](https://hackaday.com/2022/06/07/ltspice-tips-and-a-long-tutorial/)

我们总是喜欢来自[FesZ]的视频，所以当我们看到他关于 LTSpice 的最新[提示和技巧时，我们决定花 20 分钟来观看它。但是我们注意到在文本中，他有一整套关于 LTSpice 的视频教程，这实际上是第 30 集。所以有很多值得看的。](https://www.youtube.com/watch?v=xjj2XXDhZRQ)

像任何提示和技巧视频一样，你可能知道其中的一些，也可能不关心其中的一些——例如，第一个视频谈到了设置颜色，这是一个高度个人化的偏好。但很有可能你会在视频中找到喜欢的东西。

有没有遇到时间步长错误？[FesZ]对此有建议。想要热量信息？正在寻找一种简单的方法来计算均方根值？该视频涵盖了所有这些，甚至更多。注意，如果你在 Wine 下运行 LTSpice(像我们一样)，Alt 键鼠标快捷键可能不起作用，因为大多数 Linux 桌面将使用 Alt+鼠标来移动窗口。然而，如果你在你的设置中摸索，你可以把那个功能移到 Windows 键，这样 Alt 键就可以在 LTSpice 和其他程序中工作。例如，在 KDE 的最新版本中，在“窗口管理”、“窗口行为”和“窗口操作”下查找，将修改键从 Alt 改为 Meta。

我们真的很有兴趣看一些其他的视频。主题范围从简单的像使用。逐步引导到更高层次的主题，如模拟晶体、测量功率因数和使用变压器(我们已经讨论过的内容[)。).事实上，我们已经做了](https://hackaday.com/2016/03/02/transforming-spice/)[我们自己的(较短的)LTSpice](https://hackaday.com/2016/02/26/adding-spice-to-your-workbench/) 入门系列，所以如果你想狂看一些东西，有很多视频供你开始模拟。你可能也想看看[我们的虚拟现实系列](https://hackaday.com/2018/04/06/circuit-vr-the-dickson-charge-pump/)。
