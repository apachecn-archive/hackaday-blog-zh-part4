# 一个松松垮垮的选择和放置

> 原文:[https://hackaday.com/2020/01/23/a-ploopy-pick-and-place/](https://hackaday.com/2020/01/23/a-ploopy-pick-and-place/)

相当多的黑客在他们的职业生涯中达到了那个尴尬的年龄——对于手动拾取和放置来说太老了，但是对于成熟的 PnP 机器来说又太年轻了。显而易见的解决方案是构建自己的 PnP，这可以像在旧的 3D 打印机的 Z 轴上放置吸盘一样简单。然而，将零件送入取放机可能是一个棘手的问题。

或者不是，如果你像[林奕匡]那样思考并制造出这些半自动 SMD 带式送料器的话。为 8 毫米塑料或纸带建造的进料器是 3D 打印的组件，可安装在拾放机工作范围内的机架上。每个进纸器的顶部都有一个用于放胶带的插槽，通过使用 PnP 的 Z 轴按下机箱前面的杠杆来推进胶带。胶带槽中的长舌逐渐剥开胶带盖，露出一部分，然后被 PnP 吸盘吸住。任何机器都应该工作；[菲尔]使用他的[文字定位器](https://hackaday.com/2015/02/06/a-diy-pick-and-place-you-can-build-right-now/)。我们喜欢这样的想法:零件在需要之前一直受到保护；令人满意的点击杠杆动作也很酷。在下面的视频中简单看看它的实际应用。

看起来像是[菲尔]为了支持他广受欢迎的[老式轨迹球](https://hackaday.com/2019/11/01/ploopy-open-source-trackball-keeps-rolling-along/)而建造的，这款轨迹球既有成套的也有完全组装的。我们认为无论你是否使用 PnP，进料器的设计都很棒，尽管这里有一个[更简单的盒子设计](https://hackaday.com/2019/11/07/3d-printed-magazines-tame-the-smd-tape-beast/)用于纯手工 SMD 工作。
