# 使用建模粘土和摄影测量设计一个定制的外壳

> 原文：<https://hackaday.com/2021/07/14/design-a-custom-enclosure-using-modelling-clay-and-photogrammetry/>

当涉及到设计不是简单的盒子或其他基本形状的外壳时，设计过程往往会变得有些乏味和复杂，因为要传输到 CAD 程序中的测量值数量开始激增。Sebastian Sokolowski 详细描述了一个可能的捷径，他描述了一个将粘土模型和 T2 摄影测量结合起来的过程。

[Sebastian]介绍了手持控制器的设计，抓握时应符合人体工程学。这从必须安装在控制器内部的电子和机械组件开始:在 CAD 工具(在 Fusion 360 中演示)内部，这些组件周围有一个简单的盒子外壳。然后这个盒子被 3D 打印出来，用模型粘土在这个盒子周围制作出控制器的理想形状。

准备好控制器的模型粘土版本后，在这些照片被开源 [Meshroom](https://github.com/alicevision/meshroom) 工具处理成 3D 模型之前，从尽可能多的角度对其进行拍照。在修复了网格中的一些问题并减少了这个模型的顶点数，这样 CAD 工具就不会在导入它的时候卡住了，现在可以进行最后的处理了。

在 CAD 工具中，现在剩下的就是改进导入的模型，以改进其外部形状，并创建用于安装电子设备、开关和其他组件的内部细节。
