# 玉米淀粉修复游戏机屏幕

> 原文:[https://hack aday . com/2019/03/04/corn-starch-fixes-a-game-boy-screen/](https://hackaday.com/2019/03/04/corn-starch-fixes-a-game-boy-screen/)

任天堂的游戏机是 20 世纪 90 年代的掌机。像他们的许多产品一样，它以经得起愤怒的孩子的惩罚和军事打击而闻名。它最大的弱点可能是它没有背光的液晶显示屏。复古玩家和芯片调谐器一样，发现自己定期修改和更换这些。

在这些交换过程中，一个常见的问题是“牛顿环”，即偏振镜与 LCD 玻璃接触，导致难看的视觉假象。谢天谢地，有一个简单的解决办法。通过应用小到看不见的微观粒子，有可能将两者分开。【深奥的肖恩】使用玉米淀粉，而[【博加曼兹】偏爱硅藻土。](https://www.youtube.com/watch?v=1r34o5PYugU)为了获得最佳效果，可以使用化妆刷来涂抹精细的涂层，并使用压缩空气来清洁 Game Boy 并去除任何多余的涂层。

用家用订书钉解决一个微妙的屏幕问题是很少见的，但当它起作用时就令人满意了。在摄像机上很难看到结果，但许多人报告说这解决了令人沮丧的问题。所以，[如果你打算给你的游戏机](https://hackaday.com/2012/06/11/adding-a-backlight-to-the-ol-game-boy-brick/)打背光，把这个放在你的锦囊妙计里。它可以让你得到最好的结果，也可能对其他老式液晶显示器有用。休息后的视频。
