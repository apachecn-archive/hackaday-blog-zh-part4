# 西红柿不是棉花

> 原文：<https://hackaday.com/2021/11/22/tomatoes-are-not-guncotton/>

[Integza]讨厌西红柿，但喜欢火箭。因此，他决定试试能否将他最讨厌的水果加工成某种火箭燃料，或者至少是相对易燃的东西。实验以失败告终，但背后的科学很有趣。

基本的想法是，西红柿主要由水、糖和纤维素组成。因此，如果你硝化纤维素，它就变成了硝化纤维素，也就是所谓的枪棉。枪棉花曾经被用来代替火器中的火药，尽管今天它经常被魔术师用来制造无灰火焰。

为了实现这一目标，[Integza]首先尝试使用 50:50 的硝酸和硫酸混合物来制造普通的枪棉。然后用小苏打和水的混合物中和棉花以除去过量的酸，并干燥棉花。一旦经过测试，它会像你所期望的那样迅速燃烧。

用水、肥皂、过氧化氢和漂白剂去除番茄中的糖分后，在进行酸处理之前，将番茄干燥以去除多余的水分。然后对它们进行类似的中和、干燥和测试。一个西红柿确实烧得很快，而其他的只是嘶嘶作响。

这背后的一个原因可能是由于西红柿的成分。番茄通常由纤维素、半纤维素和木质素的混合物组成，后者是众所周知的产生劣质棉的成分。摇摇欲坠的准备工作可能对结果产生了一些影响。这么说吧，也不建议在没有防护装备[和通风橱](https://hackaday.com/2013/11/30/over-powered-fume-hood-is-awesome/)的情况下使用发烟酸。

视频的标题声称西红柿被变成了火箭燃料，这远远超出了实验的实际结果。然而，随着一些更先进的化学处理，我们肯定会看到水果变得比原来更易燃。不过，为了达到最佳效果，你最好还是坚持使用直筒棉。休息后的视频。
