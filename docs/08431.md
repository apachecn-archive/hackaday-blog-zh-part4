# 试验板上的全 8 位计算机

> 原文：<https://hackaday.com/2020/11/23/full-8-bit-computer-on-breadboards/>

进入一个大型电子项目通常需要使用专门的工具，即使用某种烙铁或其他方式将焊料应用于复杂、微小和热敏的部件。虽然在某些时候学会掌握这种技能是最好的，但它并不总是必要的，即使对于大型复杂的项目，如[de ruf 1]完全在试验板上构建的全 8 位计算机。

对于一个全功能的 8 位计算机，这个版本深入到计算机如何工作的细节。时钟允许程序一次通过一个周期，甚至可以通过一组开关单独访问存储器。还有很多其他有趣的特性，比如使用寄存器来访问额外的内存。它具有一个 SPI 端口和 PS/2 键盘控制器，还可以从 SD 卡加载程序。

该建筑的灵感来自于[食本者]的一些项目，这些项目以使用逻辑门和 TTL 芯片执行复杂任务而闻名，例如另一台在小显示器上玩贪吃蛇游戏的试验板计算机。这当然是了解计算机内部功能的好方法，更好的是不需要焊接。但是你可能需要一些额外的试验板。
