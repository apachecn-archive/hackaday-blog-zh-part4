# 看看老式同步示波器的内部

> 原文：<https://hackaday.com/2022/09/28/a-look-inside-an-old-school-synchroscope/>

没有什么东西比得上老式的电气设备，尤其是与配电相关的东西。有一些关于矮胖，沉重的结构，厚厚的胶木案件，你可以从房间的另一边阅读的表盘。这部令人愉快的同步示波器似乎为铁幕后的生活带来了双重意义。

你会问，同步示波器到底是什么？正如[DiodeGoneWild]所解释的(这是人类有史以来最好的口音)，同步示波器用于指示两个交流电源何时同相。这在发电和配电中很重要，在发电和配电中，将刚启动的发电机连接到稳定的电网并不是一个好主意。这个同步示波器内部有一个非常强大的机制，四个驱动线圈在一个圆形定子上相隔 90 度。里面是一个连接到仪表指针的移动线圈，这使它成为一个感应电机，当两个输入电流彼此同相时，它就会停止转动。

该仪表充满了工程上的好东西，如阻尼指针的磁制动器，以及用于向移动线圈提供电流的整洁的电感耦合方法。[DiodeGoneWild]很好地解释了电表的工作原理，并做了一些基本测试，向我们展示了这东西问世 60 多年来没有造成任何重大损害。我们渴望看到它很快接受全面测试。

这只是[DiodeGoneWild]一系列酷炫拆解中的最新一个。他最近带我们看了一眼老式瓦特计的内部，还看了看 T2 该死的激光头灯。
