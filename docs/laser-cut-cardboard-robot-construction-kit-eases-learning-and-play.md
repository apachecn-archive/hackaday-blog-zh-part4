# 激光切割纸板机器人构建套件简化了学习和游戏

> 原文：<https://hackaday.com/2018/10/18/laser-cut-cardboard-robot-construction-kit-eases-learning-and-play/>

将微控制器和其他电子设备放入一个简单的项目从未如此容易，这具有巨大的学习潜力。但是当涉及到外壳、框架和连接器等机械构件时，情况就不一样了。采购经济的伺服系统、电机和微控制器板比安排其他机器人部件更容易，这些部件允许廉价和可获得的定制和实验。

这就是安迪·福里斯特(Andy Forest)带着[激光切割纸板机器人建造套件](https://hackaday.io/project/157209)的原因，该套件始于[蒸汽实验室](http://www.steamlabs.ca/)，这是一家位于多伦多的非营利性社区创客空间。该设计使用激光切割瓦楞纸板制作模块化框架、外壳和基本硬件。这是一种既经济又有效的方法，可以创建机器人和电子动画所需的机械元素，同时还允许轻松定制。床单上有塑料吸管、筷子轴、SG90 伺服电机的打孔部分，当然，任何缺失的东西都可以很容易地用热熔胶添加或用刀切割。除了开源的设计之外，还有一个针对教育工作者的[活动指南](https://www.oercommons.org/authoring/28868-robot-inventors-activity-guide/view)，给出了使用所有东西的不同方式的视觉示例。

纸板是一种很好的原型材料，但让整个项目唱起来的是设计允许容易修改和播放，同时易于来源和生产的方式。
