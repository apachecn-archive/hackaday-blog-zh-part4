# 新零件日:Bouffalo Labs bl 602 RISC-V Wi-Fi/蓝牙 SoC

> 原文:[https://hack aday . com/2020/11/20/new-part-day-bouffalo-labs-bl 602-RISC-v-wi-fi-bluetooth-SOC/](https://hackaday.com/2020/11/20/new-part-day-bouffalo-labs-bl602-risc-v-wi-fi-bluetooth-soc/)

我们现在都应该习惯于微控制器上的无线硬件，Espressif 或 Nordic Labs 主宰着黑客界。这些年来，在这个舞台上还有其他几个竞争者没有真正引起我们社区的注意，通常是因为他们可用信息的不透明性。

一个新的竞争者应该值得再看一眼。【Bouffalo Labs 的 BL602】是一款支持 Wi-Fi 和蓝牙 LE 的微控制器，带有[32 位 RISC-V 衍生内核](https://www.sifive.com/press/bouffalo-lab-standardizes-on-sifive-risc-v-embedded)。如果你对此不感兴趣，也许 PINE64 的人正在带头努力对其进行逆向工程，以实现完全开源的无 blob 无线实现的消息可能会引起你的注意。

那么你在哪里能得到一个呢？沉住气，这个芯片还处于孕育的早期阶段。我们可以看到商店里有一些令人兴奋的可能性，但我们仍然在弄清楚硬件接口和其他软件需要使它工作。一个社区正在努力对它进行逆向工程，这让我们回到我们之前提到的 PINE64 的故事。

你可以从全球速卖通的供应商那里找到 BL602 模块，但是如果你加入他们的 blob 逆向工程，PINE64 的人[会给你一个免费的。不过请注意，这个提议是针对那些准备对项目做出承诺的人的，所以如果你不帮忙送货，不要寄希望于免费的东西给他们。](https://www.pine64.org/2020/10/28/nutcracker-challenge-blob-free-wifi-ble/)

我们可能会看到 BL602 在未来几个月获得开源工具链和内部 blobs，这要归功于那些致力于它的人的努力。就像 2014 年的 ESP8266 一样，它开始成为一个信息相对匮乏的黑匣子。但是，如果这种黑客努力得到回报，我们将拥有一个廉价的 RISC-V Wi-Fi 和蓝牙模块，从硅以上完全开源软件。活着是多么美好的时光啊！
