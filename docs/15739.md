# Spotify 播放器带回了实体媒体

> 原文:[https://hack aday . com/2022/12/24/Spotify-player-brings-back-physical-media/](https://hackaday.com/2022/12/24/spotify-player-brings-back-physical-media/)

数字音乐让你随身携带所有的音乐变得更加方便，但是我们是否失去了一些非物质化的东西？[Jordi Parra]觉得数字音乐体验有所欠缺，设计了一款带有触觉界面的 Spotify 播放器。

特定的播放列表是通过看起来像迷你光盘和黑胶唱片交叉的小 RFID 标签来选择的。由于这是一个原型，Arduino 读取 RFID 标签，但需要一台计算机来实际播放 Spotify 播放列表。未来的版本可能包括一个集成的扬声器，并运行 libspotify 来创建一个独立的设备。

虽然要实现完全无缝的体验还有很多工作要做，但我们喜欢这个项目工业设计中的细节。简洁的线条和木材与更现代材料的结合让它看起来像是一件永恒的科技产品。一定要看看[完整的照片库](https://flic.kr/s/aHsjsPEaXB)，包括令人印象深刻的包装照片。

想要更多带有触觉界面的数字音乐吗？看看这个 [MP3 播放器架](https://hackaday.com/2021/02/12/ikea-shelf-becomes-kid-friendly-mp3-player/)或者一个[简单的网络收音机移植](https://hackaday.com/2022/09/22/simple-internet-radio-transplant/)。
