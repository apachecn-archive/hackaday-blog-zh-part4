# 开一辆迷你车床兜风

> 原文：<https://hackaday.com/2018/12/31/take-a-mini-lathe-for-a-spin/>

[这个老托尼]对优质工具并不陌生，但他是从一台[迷你车床](https://www.youtube.com/watch?v=05vUCdzhoe4)开始的。怀旧并没有阻止他传播他一贯的 snark 品牌(实际上，除了 YouTube 下线之外，任何东西都无法阻止这一点)。他评价车床加工不同材料的能力，并让你决定这是一项投资，还是一个钱坑。

车床零件的范围从看起来像是在家里进行[动力轮修复](https://hackaday.com/2016/11/14/power-wheels-rescued-restored-and-enhanced/)项目的廉价启动/停止按钮组件，到尾架上方便的凸轮锁定机构，这是我们的解说员了解到的车床的改进。我们看到这个速度已经过测试，但很快就被证明不是营销宣传，除非你允许 40%的误差。它使用一个 500 瓦的 DC 电机，所以不要试图校正电源频率差异。对丝杠和螺纹表盘的判断是，你得到你所付出的，这是通过煞费苦心地将螺纹切割成铝来证明的。最后，我们看到冷轧钢上的酷刑试验。

也许有人从迷你车床社区将停止与他们的两美分。如果你喜欢这个车床入门，考虑一下【这个老托尼】的[数控机床](https://hackaday.com/2018/12/05/the-complete-beginners-guide-to-building-a-cnc-machine/)或[注射成型](https://hackaday.com/2017/12/17/the-miracle-of-injection-molding-how-does-it-work/)指南。但是对我们来说，[奎因·邓基的][系列机床](https://hackaday.com/2018/01/23/the-precision-upon-which-civilizations-are-built/)今年是一次真正的款待。
