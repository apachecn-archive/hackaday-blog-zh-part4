# 完全无用的咖啡自动售货机绝不是

> 原文：<https://hackaday.com/2021/04/05/totally-useless-coffee-dispenser-is-anything-but/>

当然，好咖啡是很好喝的，但坦率地说，咖啡因就是咖啡因，我们会以任何方式得到它。包括冻干的，如果你只有这些的话。我们不会根据人们对含咖啡因饮料的口味来评判任何人，称这种咖啡自动售货机“完全没用”显然是不真实的。它显然有它的用途，即使你不喜欢冻干咖啡，你也可以牺牲一罐雀巢咖啡，用彩虹糖或其他任何可以放进小收集篮的东西装满它。

在这台机器中，杯子是触发器——下面的 3D 打印板激活嵌在废木头底座中的微型开关，这触发了背面的微型钻头，以驱动步进电机旋转收集篮。虽然[smogdog]已经提供了所有的文件，但你还得想出自己的连接器来将东西悬挂在杯子上，并雕刻自己的底座。

当我们能看到一台机器在做什么的时候，我们喜欢它，因为它不仅有用，而且很漂亮。至少在一段时间内，这种方法奏效了。出于某种原因，它一直烧坏步进电机。休息之后，请在概念验证活动中查看。

我们已经看到 Micro:bit 做了很多事情，这个弹球机是最有趣的之一。
