# 最后，一套真正的游戏立方体游戏手柄

> 原文：<https://hackaday.com/2020/01/13/finally-a-real-set-of-gamecube-joycons/>

Gamecube 在当时的销量可能不如其竞争对手，但它至今仍保持着一个狂热的粉丝群。很大程度上由于 *Smash Bros.* 社区，其控制者仍然备受追捧。在《任天堂 Switch》发布后，随着大量粉丝的渲染,[Shank Mods]认为有人会创造一套带有 Gamecube 控件的游戏手柄。等了将近四年后，[他决定自己动手。](https://www.youtube.com/watch?v=Sb0-I8FkUWU)

构建从 Wavebird 控制器外壳开始，选择它是因为它的身体更大，恰好与开关的高度相同。外壳从中间切开，3D 打印组件被创建来将 Joycon 安装导轨连接到控制器的两个部分。大型控制器内部也有足够的空间，很容易将所有 Joycon 组件放入其中。兼容性是这个版本的一个关键目标，所以我们非常关注 Gamecube Joycons 的所有切换功能。在必要的地方增加了额外的按钮，以前的模拟触发器用插头进行了修改，以匹配开关组件的纯数字操作。

这是一个必须克服许多障碍的项目，从使一切都适合的机械重新设计，到解决 Joycon 硬件的神秘电气设计。然而辛勤的工作得到了回报，[Shank Mods]和几个有才华的社区成员一起，创造了一个漂亮的硬件。在之前，我们已经看过以 Gamecube 为主题的游戏手柄，但是这个版本真的很棒。如果你已经修改了原来的 Xbox 控制器来配合开关工作，[请务必让我们知道](http://hackaday.com/submit-a-tip)。休息后的视频。
