# 开放硬件狗治疗分配器是行为研究的踏脚石

> 原文：<https://hackaday.com/2022/02/06/open-hardware-dog-treat-dispenser-is-a-stepping-stone-for-behavioral-research/>

开源硬件的原理开始在科学研究领域大步前进。[Walker Arce]向我们讲述了他与[Jeffrey R. Stevens]合著的论文，内容是关于[一种为科学研究人员设计的狗零食分配器](https://openhardware.metajnl.com/articles/10.5334/joh.41/)——对于行为研究目的来说是不可或缺的，并且易于复制，因此我们的科学也可以复制。树莓派、NEMA 踏步机和大量 3D 打印部件的使用使这个建筑便宜(< $200 美元)，并且易于重复用于任何涉及狗或其他喜爱美食的动物的实验。

即使你不是科学家，你也可以为你自己的宠物训练做一个——这个设计是如此的简单和容易复制！大部分零件都是业余爱好者级别的，而且很有可能，你可以在你的车间里找到大部分零件。想知道这个分配器是如何工作的，最重要的是，狗是否满意？休息过后，请观看[一小段演示视频](https://www.youtube.com/watch?v=fIKlNvzwXUY)。

尽管这种分配器可以在市场上买到，但设计和验证一种新型分配器比你想象的更有价值——作者报告说，根据他们的经验，现成的分配器有 20-30%的错误率，而他们的分配器可能只有 4%,他们有[测试结果来支持这一点](https://github.com/unl-cchil/canine_precise_dispenser/tree/main/reliability%20testing)。我们不禁感到高兴的是，性能更好的产品可供我们任何人构建。[GitHub 库](https://github.com/unl-cchil/canine_precise_dispenser/)拥有你想要的一切——从 STLs 和 PCB 文件，到 Raspberry Pi SD 卡图像和[14 页的组装和设置指南 PDF](https://github.com/unl-cchil/canine_precise_dispenser/blob/main/documentation/user_guide/unl_cchil_precise_dispenser_guide.pdf) 。

开放硬件和科学[是天作之合](//hackaday.com/2015/04/08/open-hardware-for-open-science-interview-with-charles-fracchia/)，即使关系还在发展。黑客社区[在](https://hackaday.com/2018/05/09/friday-hack-chat-open-hardware-for-science/)之前已经聚集在一起讨论科学中的开放硬件，并且不时地，开源科学设备增光我们的页面，就像最近分类的[生物传感黑客使用再利用的消费级设备](https://hackaday.com/2021/12/24/when-hacking-and-biosensing-collide/)。
