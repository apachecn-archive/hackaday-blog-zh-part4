# 三向定向天线

> 原文：<https://hackaday.com/2022/01/21/directional-antenna-3-way/>

如果你阅读旧的天线书籍，你可能会看到相控垂直天线的想法。这些电缆使用一定长度的同轴电缆来控制信号的相位，以三角形配置传输到三个垂直电缆。根据相位，您可以使天线阵列指向三个方向中的一个。[DX Commander]设计了这种天线的非常现代的版本，并在最近的视频中展示了其背后的理论，您可以在下面看到。

似乎另一个火腿建立了天线和控制盒，他发送到[DX 指挥官]虽然他还没有设置它来创建一个 80 米的定向天线。我们将有兴趣看看它在实践中是如何工作的。

当然，相位延迟线和垂直线将被调谐到一个特定的频率，所以你不能指望这是一个宽带系统。它也不小，因为垂直放置相隔四分之一波长，你需要大约相同数量的同轴电缆用于定相电缆(考虑速度因素)。

控制盒可以选择哪个垂直方向获得延迟并控制方向。你也可以完全关闭延迟，用一个全向天线结束。

这可能不会成为廉价天线，尤其是 80 米长的天线。虽然你在高频下不经常看到相控阵，但它们在更高频率下更常见。
