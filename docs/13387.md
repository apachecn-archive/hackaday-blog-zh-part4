# 定制的小型车间起重机减轻了负荷

> 原文：<https://hackaday.com/2022/04/19/custom-fit-small-shop-crane-lightens-the-load/>

在车间奢侈品的候选名单上，我们敢打赌许多黑客会包括一个高架起重机。安全轻松地提升重物的能力打开了一个新项目的世界，并使商店进入一个完全不同的能力类别。

和我们中的许多人一样，[Jornt]在一家空间非常有限的商店工作，所以他建造的悬臂起重机必须是定制的。该建筑完全由钢管制成，首先制作一个桅杆来支撑起重机，并将其挤压到商店现有货架的一个小槽中，尽管是在现场焊接的，但不知何故没有着火。许多定制零件被安装到安装吊臂的回转装置中，吊臂本身是一个用棍子建造的空间框架，必须适应倾斜的天花板。沿着悬臂底部的双排管道允许一辆载有 500 公斤电动绞车的小车沿着它运行，提供了一个看起来覆盖了大部分车间的工作包线。为安全的黄黑色油漆工作脱帽致敬——非常工业。

从下面视频中的测试来看，该起重机完全可以胜任车间内提升发动机和其他重物的任务。如果[Jornt]再次处理另一个版本，比如他的毫不妥协的 DIY 车床,这应该会很方便。
