# Hackaday 播客 112:我们有一个 NFT，比赛一个莫比乌斯带，用 OpenCV 和 Blender 同步视频

> 原文：<https://hackaday.com/2021/04/02/hackaday-podcast-112-we-have-an-nft-racing-a-mobius-strip-and-syncing-video-with-opencv-and-blender/>

Hackaday 的编辑埃利奥特·威廉姆斯和迈克·什奇斯庆祝本周最聪明的项目。我们试图在周四通过我们的 Lightmode 和 NFT 文章抓住一些傻瓜——如果你还没有阅读这些文章，请确保你回去阅读它们并开怀大笑。

虽然这些被归入*而非黑客*，但本周的许多其他功能都是世界级的黑客，例如由 1.5 打真空管构建的 555 定时器，以及由 Hackaday 自己的 64 个磁性致动器围绕 PCB 线圈构建的机械单词时钟【Mortiz v. Sivers】。

听觉享受，[Linus Akesson]又名[lft]展示了一个听起来像教堂风琴一样大的 Commodore64。作为 OpenCV 和 Blender 的杰作，你不能错过[Matthew Earl]的项目，该项目将火星着陆的视频叠加在静止的卫星照片上…完美！
