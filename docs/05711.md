# 重温有线电视的光辉岁月与这个复古的天气饲料

> 原文：<https://hackaday.com/2020/02/21/relive-the-glory-days-of-cable-tv-with-this-retro-weather-feed/>

这可能会让年轻的读者感到惊讶，但曾经有一段时间，天气频道的真人秀节目只是天气。过去，等上“本地八分钟”不超过 10 分钟，通过简单的文本抓取本地情况和预报，看起来像是直接从国家气象局(National Weather Service)获取的。那些日子，可悲的是，它们似乎一去不复返了。

或许不会，如果[这个复古的天气频道反馈](https://github.com/probnot/wpg-weatherchan)对此有什么要说的话。它是[probnot]的产品，由一个运行在 Raspberry Pi 上的简单 Python 程序组成。来自温尼伯的[probnot]正在利用加拿大环境部获取当地的天气数据，但是应该很容易修改以使用当地气象提供商的 API。屏幕充满了复古的美好，从简单的配色到块状的白色文字；底部的数字时钟和本地新闻补充了老学校的体验。代码似乎不支持周期正确的平滑爵士萨克斯管，但这也应该是一个简单的修改。

抛开所有的嘲笑不谈，这将是早晨例行公事的一个可喜的补充。对于完整的复古骑行，为什么不考虑将[放在旧电视盒](https://hackaday.com/2017/12/12/old-tv-lends-case-to-retro-magic-mirror/)中？
