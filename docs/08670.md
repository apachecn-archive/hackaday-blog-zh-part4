# 可以吃的曲子

> 原文：<https://hackaday.com/2020/12/14/tunes-you-can-eat/>

本周，复古小工具收藏家和视频博主[Techmoan]展示了我们所知的可能是最美味的音频记录格式——巧克力留声机唱片。(视频，嵌在下面。)与他典型的媒体形式探索相比,《巧克力》这张唱片是最近的作品。他第一次听说它们是在 2015 年，当时塔斯马尼亚艺术家[Julia Drouhin]提供了[巧克力唱片](https://www.abc.net.au/news/2015-07-09/french-tasmanian-artist-spins-sweet-sounds-of-chocolate-records/6607014)作为她的艺术项目的一部分。[Techmoan]最终得到的那首歌来自一家[英国巧克力商](https://www.etsy.com/uk/listing/661867495/chocolate-record-a-record-made-in)，他为他们提供定制标签和两首歌曲供你选择。视频中有一些关于如何回放你的巧克力盘而不损坏它的提示(尽可能使用最轻的唱针跟踪力)。这些光盘只在一面以 45 转/分钟的速度录制，大小与标准的单张光盘差不多。但是由于厚度是普通唱盘的五倍，它们比普通唱盘含有更多的热量。

没有对图克斯伯里镇乐队的反思，但这可能是有史以来保真度最低的记录媒体，但至少你可以在听完之后吃它——标签和所有东西。我们希望《T2 碟中谍》的电影制作人能注意到这一点，这样我们就能在下一部电影中看到秘密的音频简报被吃掉，而不是化为乌有。
