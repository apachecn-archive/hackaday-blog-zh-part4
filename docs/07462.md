# 2020 年秋季宣布推出六个新的 HackadayU 课程

> 原文:[https://hackaday . com/2020/08/13/six-new-hackadayu-courses-announced-for-fall-2020/](https://hackaday.com/2020/08/13/six-new-hackadayu-courses-announced-for-fall-2020/)

刚刚公布了 [HackadayU](https://hackaday.io/u/) 课程的秋季阵容，[现在就拿票](https://www.eventbrite.com/o/hackaday-5851454453)！

每门课程都由专家讲师指导，他们将主题提炼为一套四节现场互动课，外加一节我们称之为办公时间的问答课。主题从提升你的 Linux 技能和学习串行总线到构建互动艺术和进入第一人称视角(FPV)无人机飞行。

查看下面列出的课程名称、教师和详细信息。如果你想从老师那里听到每堂课的内容，休息后会有他们的预告视频。

*   [使用光和传感器的互动媒体艺术](https://www.eventbrite.com/e/hackaday-u-interactive-media-art-with-light-and-sensors-tickets-115767601011)
    *   讲师:米拉贝勒·琼斯
    *   [课程概述:](https://hackaday.io/project/174179-interactive-media-art-with-light-and-sensors)本课程将涵盖如何开发基于传感器输入的互动艺术品、装置和体验。
*   [FPV 无人机简介](https://www.eventbrite.com/e/hackaday-u-introduction-to-fpv-drones-tickets-115769881833)
    *   讲师:Ayan Pahwa
    *   [课程概述:](https://hackaday.io/project/174180-introduction-to-fpv-drones)我们将熟悉无人机的多旋翼类别，包括物理学、空气动力学、电子学、数字信号处理(DSP)以及编写所涉及的软件。
*   [使用 Arduino 和 FastLED 的 LED 简介](https://www.eventbrite.com/e/hackadayu-intro-to-leds-using-arduino-and-fastled-tickets-115758892965)
    *   指导老师:凯西·拉夫林和米拉贝勒·琼斯
    *   [课程概述:](https://hackaday.io/project/174150-led-programming-with-arduino-fastled)学生将了解 LED 的工作原理，以及如何使用 Arduino IDE 对 LED 模式进行编程。
*   [Linux +电子学:树莓派课程](https://www.eventbrite.com/e/hackaday-u-linux-electronics-a-raspberry-pi-course-tickets-115783299967)
    *   教练:巴勃罗·奥亚尔佐
    *   [课程概述:](https://hackaday.io/project/174153-linux-electronics-a-raspberry-pi-course)本课程面向那些希望从 Arduino 升级到 Linux 计算机的人，这些计算机体积小，足以满足项目需求，但功能强大得多，可以完全满足项目需求，但不知道从哪里开始。
*   [嵌入式串行总线(第一部分)](https://www.eventbrite.com/e/hackaday-u-embedded-serial-buses-part-1-tickets-115774210781)
    *   讲师:亚历山大·罗塞尔
    *   [课程概述:](https://hackaday.io/project/173667-embedded-serial-buses)本课程将涵盖 I2C 和单线串行总线。我们将研究这两种总线类型的硬件层、协议层和软件/应用层。
*   [艺术+代码](https://www.eventbrite.com/e/hackaday-u-art-code-tickets-115772311099)
    *   讲师:凯西·亨特
    *   课程概述:学生将通过掌握 P5.js JavaScript 库来增长他们的技术技能，还将学习数字空间中的美学和艺术史。

HackadayU 课程是“按需付费”。为了确保现场座位不被浪费，每堂课的最低捐款额为 1 美元。收益将捐给慈善机构，我们很高兴地宣布 HackadayU 夏季会议向 Steam Coders 捐赠了 4，200 美元。秋季班将选择一个新的慈善机构，详情如下。

每堂课都将被录制下来，并在编辑后可供使用。现在就可以用 Ghidra 系列看看优秀的 [*逆向工程。*量子计算*和 *KiCad + FreeCAD* 课程的视频即将发布。*](https://www.youtube.com/playlist?list=PL_tws4AXg7auglkFo6ZRoWGXnWL0FHAEi)
