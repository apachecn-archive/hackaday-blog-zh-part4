# 旋转磁场，解释

> 原文:[https://hack aday . com/2020/11/12/旋转磁场-解释/](https://hackaday.com/2020/11/12/rotating-magnetic-fields-explained/)

如果你用一块磁铁、一个线圈和一些针做了一个马达，你可能会记得马达和发电机依赖于旋转磁场。一旦你知道它是如何工作的，这个概念就很简单了，但是你有没有想过是谁从一开始就解决了这个问题？不出所料，特斯拉也参与其中。但是迈克尔·多布罗沃斯基或者沃尔特·贝利呢？对大多数人来说不是普通的名字。【学工程】有一个华而不实的视频，内容涉及[旋转磁场机器](https://www.youtube.com/watch?v=wqrGHeuxUvI)的历史和理论，你可以在下面看。

当时，靠直流电运转的电机不太实用，会导致运动不平稳。然而，特斯拉和另一位名叫费拉里斯的发明家意识到，交流电流可以在没有移动换向器的情况下产生旋转磁场。

特斯拉的电机使用了他的两相发电机产生的两个 90 度相位差的交流电。法拉利使用单相和电感来产生相位差。然而，两相电机有局限性，Dobrowolsky 的三相设计很快取代了两相设计。

该视频有许多动画来帮助理解电机如何工作。这也解释了为什么电机绕组的磁极与永久磁铁的磁极相反。看完这个视频，你准备好设计下一个超高效电动机了吗？没有。但是你会对交流电机内部真正发生的事情有更好的理解。

如果你给磁铁附近的线圈通电，就能使轴转动。如果那个轴带动磁铁和线圈旋转，你就得到电。因此，虽然不理想，[许多发电机可以作为电动机工作，反之亦然](https://hackaday.com/2020/01/16/car-alternators-make-great-electric-motors-heres-how/)。如果你认为整个世界都靠三相交流电运转，你应该读读关于[两相](https://hackaday.com/2018/03/15/a-tale-of-two-phases-and-tech-inertia/)的顽固坚持。
