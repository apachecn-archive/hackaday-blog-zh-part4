# 武器化的坐立不安纺纱工

> 原文：<https://hackaday.com/2018/08/19/weaponized-fidget-spinners/>

坐立不安的纺纱工一度是最热门的新潮流，但是他们的 15 分钟名气已经完全过去了。他们很适合坐立不安，而不是其他。使用它们的主要目的之一是尽可能快地旋转它们。然而，在[寿司拉面]用压缩空气旋转后伤到自己之后，一个新的危险的想法出现在脑海里—[。](https://www.youtube.com/watch?v=MS4x8ThI_NI)

你现在看到的是一个由压缩空气驱动的坐立不安旋转器*剑*。这本身就是一件幸事，因为它防止了这个可怕的设备被人轻易携带。通过一个危险的构建蒙太奇，我们看到将近 50 个坐立不安的旋转器(在超色中，不多不少)被安装在一个轴上。然后将轴连接到刀柄上，一根塑料线巧妙地弯曲起来，在扳机的拉动下输送压缩空气，使坐立不安的旋转器以中等速度旋转。

的确，坐立不安的旋转器不会从压缩空气中获得大量扭矩，因此大多数损坏纯粹是通过在易碎的玻璃物体上摆动可能相当重的设备来完成的。也就是说，不入虎穴，焉得虎子，我们总是很高兴看到研究和开发继续在坐立不安旋转空间。
