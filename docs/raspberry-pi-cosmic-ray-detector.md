# Raspberry Pi 宇宙射线探测器

> 原文：<https://hackaday.com/2021/03/16/raspberry-pi-cosmic-ray-detector/>

[马可]有一个碘化钠探测器，通过闪烁显示宇宙辐射。这种材料被宇宙射线击中时会发光，传统上，光电倍增管会检测到检测到的照片。在你可以在下面的视频中看到的快速演示之后，他[建造了宇宙 Pi](https://www.youtube.com/watch?v=PCB8nv4fatc) ，这是欧洲粒子物理研究所的一个项目，旨在创建一个巨大的分布式宇宙射线探测器。[宇宙 Pi](https://ohwr.org/project/cosmic-pi/wikis/home) 使用闪烁，但不是来自晶体。它使用塑料闪烁体和硅光电探测器，因此比传统的探测器更容易使用。

使用一个四层板和一些收获的组件，该设备检测μ子。有两个闪烁探测器，撞击这两个探测器的μ子可能没有本地起源。该仪器有一个全球定位系统，以获得准确的时间和位置数据。船上还有其他传感器，用于收集每个探测到的事件的情况数据。

总的来说，这看起来是一个有趣的构建。有必要防止箱子漏光，因为漏光会干扰灵敏的测量。我们总是欣赏公民科学项目，这些大规模分布的仪器特别令人兴奋，因为你可以成为一些真正大的东西的一部分。
