# 一款你可以轻松打造的智能手表

> 原文：<https://hackaday.com/2019/01/01/a-smartwatch-you-can-easily-build-yourself/>

智能手表的概念被搁置了很长时间，直到这项技术真正取得成果。通过追求小型化，现代智能手表变得时尚、紧凑，并以其尺寸而言功能非凡。苹果和三星等公司在研发上投入了大量资金，但这并不意味着你不能创造自己的东西。通过这款基于 Arduino 的智能手表,[Electronoobs]做到了这一点。

手表的大脑是黑客的主食，古老的 ATmega328，因其在 Arduino Uno 和 Nano 平台上的使用而闻名。FTDI 模块用于 USB 通信，使电路板编程变得轻而易举。蓝牙通信由另一个预建模块处理，一个名为 Notiduino 的智能手机应用程序负责将通知传递给手表。

这是一个不做任何疯狂或难以理解的事情的构建，只是简单地以一种非常整洁的方式组合有用的部分。这款手表令人印象深刻地轻薄小巧，适合自己动手制作，拥有大量实用功能，但又不过分。
