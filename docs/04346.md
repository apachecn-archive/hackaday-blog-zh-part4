# 点击弹簧扑克牌压机是一件艺术品

> 原文:[https://hackaday . com/2019/09/30/the-click spring-playing-card-press is-a-work-of-art/](https://hackaday.com/2019/09/30/the-clickspring-playing-card-press-is-a-work-of-art/)

我们不知道什么是扑克牌压机，也不关心。我们所知道的是，在[看到来自 Clickspring 的【Chris】让他的扑克牌按下](https://www.youtube.com/watch?v=3PwAQZNLy0I)之后，我们想要它。

再深入一点，[Chris]主动提出为[Chris Ramsay]制作这张卡片印刷机，他是一位专攻心脏病学或卡片幻觉艺术的魔术师。扑克牌的手感对使用扑克牌表演至关重要，压牌机可以保持一副牌的形状。这不是一款常见的设备，[Clickspring Chris]精心设计了一款，从[Chris Ramsay]的标志中引入了元素。

[![](../Images/4578c15d59a645fcf6e30600203b54b9.png)](https://hackaday.com/wp-content/uploads/2019/09/The-Chris-Ramsay-Playing-Card-Press-9-7-screenshot.png) 像所有的 Clickspring 视频一样，这是一个观看的乐趣，但在离开时，没有叙述——只有 30 分钟的精密加工和金属加工。[Chris]已经大规模涉足金属雕刻，并利用他的技能为一切添加细节，从顶部风格化的橡子到复杂的压板，所有这些都是徒手完成的。还有那些蛇！它们由黄铜棒制成，手工弯曲成形，环绕在侧支架上，形成[克里斯·拉姆齐]的标志。所有的黄铜都镀了金，而所有的螺丝都镀了热蓝漆。安顿下来，欣赏下面的视频。

自从[点击弹簧骨架钟](https://hackaday.com/2016/12/06/for-your-binge-watching-pleasure-the-clickspring-clock-is-finally-complete/)完成已经有一段时间了，在这段时间里【Chris】一直致力于复制[安蒂基希拉机芯](https://hackaday.com/2015/11/23/the-antikythera-mechanism/)。然而，当他对这个机制有了一个新的发现时，他的视频输出明显变慢了，这个发现值得作为一篇学术论文写出来。我们不能吝惜他追求这一目标所需的时间，我们很高兴他也为这个项目找到了时间。
