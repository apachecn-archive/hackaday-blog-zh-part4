# 2022 年科幻竞赛:你一直想要的电子动画尤达宝宝

Simple robot parts make up the internals.

说到科幻，很难超越《星球大战》、《T1》和许多提交给我们竞赛的作品，它们都在那个确切的宇宙中。凯文·哈灵顿的参赛作品就是这样一个例子，他的电子动画尤达宝宝[就像你希望的那样可爱。](https://hackaday.io/project/175622-the-child-animatronic-hack-of-baby-yoda-plush-toy)

这款车基于 Pololu 罗米底盘，一个简单的两轮差动驱动机器人平台。它与赫淮斯托斯手臂 2 形式的机器人手臂配对，为早熟的小生物提供关节。一个 ESP32 微控制器作为操作的大脑，控制所有的伺服系统和电机，使小尤达移动。通过 RBE1001lib，使用 ESP32 [上托管的网站，通过 WiFi 进行控制。](https://github.com/WPIRoboticsEngineering/RBE1001Lib)

在任何科幻大会上，电子动画婴儿尤达肯定会成为热门人物。总的来说，这是一个简单的机器人，通过用一个可爱的玩具剥皮，它变得更加有个性。这也不是我们见到的第一个婴儿尤达(或格罗古)——这个受欢迎的角色[也曾启发过我们的建造！](https://hackaday.com/2021/01/12/baby-yoda-becomes-personable-robot/)休息后的视频。
