# 从头开始建造一辆铝制遥控卡车

> 原文：<https://hackaday.com/2021/09/05/building-an-aluminum-rc-truck-from-scratch/>

这些天来，你可以得到任何一种无线电控制的车辆作为一个现成的模型。汽车、卡车、挖掘机，应有尽有。打开盒子，给电池充电，你就可以开始了。即使有了所有这些现代化的便利，仍然有一种特殊的建模者，他们只使用一些现成的部件来创建自己的模型。

[里尼·安妮塔]正是这种罕见的品种，从零开始创造了这辆[铝制遥控卡车](https://www.youtube.com/watch?v=Tm4z_aFqOnk)。卡车本身是一个驾驶室-发动机驾驶室(COE)的缩写，这种风格可以在世界各地的本地交付中看到。他从梯形框架底盘开始，这是用挤压铝通道建造的。这种材料与零售店陈列柜中通常用于门轨的材料相同。电子和标准钢筋混凝土票价:接收器，电子速度控制，和伺服转向。电池是回收的锂电池。主变速箱和驱动轴看起来来自另一个钢筋混凝土车辆，而钢板弹簧和悬挂组件都是定制的。

卡车的车身是手工成型金属的一个很好的例子。首先，创造了一个木制的形式。窗户和门板的部分被雕刻出来。然后将铝片弯在木模板上。小心放置的锤子将金属弯曲成雕刻的部分——留下门、窗和其他面板线条的印记。

在整个建造过程中，我们对[里尼]的技能感到惊讶，事实上整个工作是用基本工具完成的。磨床、旧钻床和铆钉枪是首选工具；找不到焊工或 3D 打印机。这使得像这样的项目对任何黑客来说都是唾手可得的——尽管这可能需要一些时间来磨练您的技能！对于他的下一辆卡车，也许[里尼]可以[增加一个自动驾驶选项](https://hackaday.com/2020/10/31/self-driving-rc-truck-is-a-masters-thesis-in-cybernetics-and-robotics/)！
