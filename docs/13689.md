# HP-200LX 像 90 年代一样运行网站

> 原文:[https://hack aday . com/2022/05/21/HP-200 LX-runs-website-like-its-the-90s/](https://hackaday.com/2022/05/21/hp-200lx-runs-website-like-its-the-90s/)

HP-200LX 掌上电脑在当时是一台令人着迷的机器，鉴于有人负责[为*掌上电脑管*](https://www.palmtoptube.com/about.htm) 提供网站服务，这是一个致力于复古掌上电脑的网站和 YouTube 频道，【特伦斯·维高文】证明了它的时代尚未结束。

HP-200LX 本身并不具备充当现代 web 服务器的条件，但它不需要太多就能提供缺失的部分。PCMCIA 网络适配器提供以太网连接，NAS 包含网站内容，而网络和 web 服务器软件在本地运行。稳定的电力来自墙上的适配器，但 200LX 本身的两个可充电 AA 电池可以充当迷你 UPS，在断电时提供备用电源。

HP-200LX 是一款突破性的产品，它来得正是时候，领先于其他真正的掌上电脑，如 T2 的 IBM PC 110 T3。在 90 年代初，人们无法想象口袋里会有一台功能齐全的 MS-DOS 机器，更不用说一台能用几节 AA 电池维持几周的机器了。它没有一些专有的操作系统和奇怪的端口，这种功能是大约 30 年后，人们能够胜任提供网络流量的部分原因。
