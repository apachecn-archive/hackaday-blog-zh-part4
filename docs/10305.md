# 留意你自己的铋，使你的脱焊更容易

> 原文:[https://hack aday . com/2021/06/02/make-your-desolding-ease-by-minding-your-own-bismum/](https://hackaday.com/2021/06/02/make-your-desoldering-easier-by-minding-your-own-bismuth/)

任何以相图开始的视频都会立即引起我们的注意。诚然，我们对这类东西的要求很低，但是除了吸引眼球之外，[Robin Debreuil]对他在铋的帮助下进行脱焊技术的快速概述是值得一看的。

除了用于治疗胃部不适的粉红色粘稠溶液之外，铋还有很多商业用途。然而，就脱焊而言，它有降低锡和锡合金(如焊料)熔点的趋势，这使得它成为工具包中一个有价值的补充。[Robin]首先演示了一点铋能在多大程度上降低锡焊料的熔点——大约 135。这样就有足够的时间来工作，将导线从衬垫中释放出来变得轻而易举。他用一些大的 QFP 芯片证明了这一点，这些芯片实际上是从板上跳下来的。他还展示了一种清洁引线上铋锡混合物的巧妙技术，用一些辅助夹子将一段脱焊编织物夹在垂直方向上。编织物沿着芯片的一侧将铋-锡混合物从引线上吸走，而重力将它拉下编织物，安全地汇集在工作台上。相当狡猾。

以免含铅焊料爱好者烦恼，[罗宾]向我们保证，这也适用于铅锡焊料。你也不用担心破产；铋很便宜，也很容易获得。另外，它相当无毒，至少就重金属而言。但是，唉——[它显然不是很好加工](https://hackaday.com/2016/08/24/fail-of-the-week-machining-bismuth/)。
