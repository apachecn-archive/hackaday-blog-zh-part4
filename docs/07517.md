# 棍子上的纹影

> 原文：<https://hackaday.com/2020/08/25/schlieren-on-a-stick/>

纹影成像是一种使用相机和一些巧妙的光学设置来观察透明流体密度的技术。像空气这样的流体的密度可能会根据空气本身与各种气体的成分而变化，或者它可能会因声波或压力波而变化。听起来你可能需要一个复杂和/或昂贵的设置来查看这些东西，但通过一些常见的东西[你可以拥有自己的纹影设置](https://eladorbach.blogspot.com/2020/08/schlieren-on-stick.html)，如【elad】所示。

他的装置依赖于一部手机，连接在自拍杆上，另一端是一面球面镜。自拍杆使调整相机到镜子的距离变得容易，因为相机的特定距离是焦距的函数。对于手机相机来说，最好通过使用小 LED 作为点光源的实验来找到这个距离。一旦它被校准并开始工作，手机上就会显示一个圆形的视野，让观众可以看到镜子前面密度的任何变化。

[elad]指出，这个建筑的唯一缺点是自拍杆不够坚硬，无法防止图像稍微晃动，但总的来说，这是一个优秀的项目，展示了摄影/仪器世界中一个整洁而有用的技巧，可以用于许多其他项目。在 T1 之前，我们只见过一次纹影成像，它使用了一种稍微不同的方法来观察变化的密度。
