# 一个(几乎全部)新的准将 64

> 原文:[https://hack aday . com/2022/05/04/a-near-all-new-commodore-64/](https://hackaday.com/2022/05/04/a-nearly-all-new-commodore-64/)

Commodore 64 在推出 40 年后仍然是最有影响力的 8 位家用电脑之一，因此毫不奇怪有一个庞大的爱好者社区围绕着它。这么多年来生产了这么多，人们可能会认为不会缺少幸存的标本，但可悲的是时间和组件故障已经付出了代价，经典的微并不总是最可靠的孩子。因此，供应 C64 零件的家庭手工业已经兴起，导致[复古小屋]尝试完全从零开始制造一个新的。

正如你在下面的视频中看到的，这并不是一个完全新的 64 年，因为包括一些定制硅在内的部分来自失败的电路板。PCB 是原始产品的现代翻版，SID 声音芯片是 ARMsid，大多数部件来自一个方便的打包套件，使装配 BoM 更加容易。代替原始 RF 调制器的大银盒的是一个现代的复合电路板，并且有一些小的连接器部分差异的问题。

组装只是一个非常长的通孔焊接过程，一旦他完成了，就会拒绝工作。我们都经历过，最终他追踪到了一个错误安装的芯片。如果你认为你在这里见过一些全新的 c64，那你就错了，[其中一个甚至用乐高做了那些难以捉摸的键帽](https://hackaday.com/2019/03/12/its-raining-brand-new-commodore-64s/)。
