# 魔镜不是透明的金属

> 原文：<https://hackaday.com/2022/08/22/magic-mirror-isnt-transparent-metal/>

《星际迷航》电影中有一个麦戈芬叫“透明铝”虽然魔镜并不真正透明，但对于不经意的观察者来说，它看起来是透明的。如果你没见过这些，它们是抛光的金属镜，背面有浮雕图案。然而，当你将点光源照射在镜子上时，反射与镜子背面的相匹配。是透明的吗？不，下面由[史蒂夫·莫尔德]制作的视频解释了到底发生了什么。

事实是表面非常细微的变化产生了图像。你需要对光学和微积分有所了解，才能完全理解正在发生的事情。

然而，真正的谜团是这些零件是如何制造的。有一些关于压花厚度的猜测，但请记住，表面变化是几分之一微米。更有趣的是，并不是所有的镜子都反射背面的图像，这让你认为背面的图像只是一个借口，不一定是制造过程的一部分。

我们认为，如果有人能弄清楚如何制作这些，那就是我们的读者。我们会考虑除了使用 555 或 Arduino 以外的任何猜测。我们想知道你能否用环氧树脂浇铸这些东西。我们承认光学不是我们的强项，但我们偶尔也会钻研一下。总有一个[菲涅尔透镜](https://hackaday.com/2022/06/07/the-origin-of-the-fresnel-lens/)的用处。
