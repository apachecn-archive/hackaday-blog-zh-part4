# 奶酪磨碎机现在磨碎奶酪

> 原文：<https://hackaday.com/2019/09/09/cheese-grater-now-grates-cheese/>

如果你在苹果产品酷起来之前就一直在使用它们，你可能还记得 Power Mac G5。这是在苹果使用英特尔处理器之前的一段时间，所以兼容性问题很严重，苹果的用户数量很少。它们在一些地区仍然很受欢迎，但没有现在这样广泛的吸引力。钻孔铝制设计的高质量一直延续到英特尔时代，并越来越受欢迎，但这种外壳仍被通俗地称为“奶酪刨丝器”。尽管最初不能磨碎奶酪，但这台 Power Mac [确实能磨碎奶酪](https://www.youtube.com/watch?v=0rJBMtzr-94)。

未磨碎的奶酪被放在 CD 驱动器插槽中，在那里它穿过一系列 3D 打印齿轮，这些齿轮将奶酪磨碎成小块。当奶酪磨碎机通过树莓 Pi 检测到奶酪时，它会自动启动。Pi 4 还可以作为旧 G5 外壳中的工作台式计算机，配有定制的 HDMI I/O 端口，这些端口与外壳集成在一起，使其看起来像原始硬件。

有趣的是，Pi 4 的计算能力和内存比苹果当时的旗舰 Mac 更强，而功耗却低 100 倍左右。这是一个功能性的构建，详细阐述了硬件社区中的一个笑话，我们都可以欣赏。也许下一个版本应该是将蓝烟用于生产目的。同时，普通读者会记得[这不是我们给你看的第一集与苹果相关的奶酪光栅](https://hackaday.com/2019/06/24/does-the-cheese-grater-do-a-great-grate-of-cheese/)。
