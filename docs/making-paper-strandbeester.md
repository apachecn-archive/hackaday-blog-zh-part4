# 制作纸束

> 原文：<https://hackaday.com/2022/11/25/making-paper-strandbeester/>

我们喜欢纸质机械装置的艺术性。创造幽默、美丽和惊奇的简单工具和技术。

[Wolfcat Workshop]的[Federico Tobon]制作了令人惊叹的自动机，融合了剪纸艺术、折纸和传统木雕自动机。他推出了一个有用的新视频，内容是用纸零件制作链接。

在这个短视频中，【费德里科】向我们展示了如何为【西奥·詹森】的经典作品《Strandbeest》制作纸质版的腿部装置，我们已经在[的许多版本中介绍过](https://hackaday.com/2020/06/08/unique-strandbeest-stands-tall-with-line-of-legs/)。

纸张自动机中的旋转接头有时是用一个机械紧固件(如柱螺钉)来完成的，但这违背了事情的简单性，并且通常看起来很笨重。[费德里科]使用一个简单的自扣件。一个部件上的 5 毫米孔与另一个部件上的两个“瓣”紧密配合。他制作了一个单独的[视频，介绍如何制作紧固件](https://www.youtube.com/shorts/OSvs1zi9KY0)。他在用造纸工人的 Cricut 型机器切割零件。相当酷。

我们已经介绍了许多来自[Wolfcat Workshop]的其他很酷的填充材料。如果你想要更多他的自动机视觉糖果，看看[简单自动机盛会](https://hackaday.com/2019/02/02/simple-automata-extravaganza/)。
