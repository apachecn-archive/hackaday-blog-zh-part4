# 回收椅子把地铁带到你的客厅

> 原文：<https://hackaday.com/2022/11/06/recycled-chairs-bring-the-subway-to-your-living-room/>

公交座椅日子过得很坎坷。忍受一些会让你的沙发哭泣的磨损周期，它们会受到打击，必须保持看起来像样。火车、公交车退役，旧座椅何去何从？来自麻省理工学院爱好商店的一个团队调查了退役的 MBTA 红线汽车的座椅发生了什么，并将它们回收成时尚的椅子。

经过一些调查和许多电子邮件之后，MBTA 把一些旧的地铁座位让给了这个团队。由于地铁座椅没有腿，旧教堂长椅上的木头被用来做底座。它需要一个长凳末端支撑来创建每一组腿，这些腿是在带锯上切割出来的。旧的深色污渍被磨掉了，底部涂了三层凝胶面漆，让老橡木的自然美闪闪发光。
