# 巴克豪森效应:听觉磁铁的诞生

> 原文:[https://hack aday . com/2022/11/19/the-Barkhausen-effect-hearing-magnets-being-born/](https://hackaday.com/2022/11/19/the-barkhausen-effect-hearing-magnets-being-born/)

巴克豪森效应——以德国物理学家海因里希·巴克豪森的名字命名——是一个术语，指铁磁材料在外部磁场的影响下，由于其离散磁畴的大小和方向发生变化而产生的噪声输出。磁畴很小:小于形成磁性材料的微晶颗粒，但大于原子尺度。Barkausen 发现，当磁场靠近含铁材料时，局部磁场会随机翻转，因为磁畴会重新排列成最小能量配置，这种磁场噪声可以通过适当排列的拾波线圈和放大器来检测。在 下面的 [短演示视频中，这种巴克豪森噪声可以被馈入音频放大器，产生一个非常说明问题的效果示例。](https://www.youtube.com/watch?v=BLXVLDysroY)

这种效果的实际应用的一个例子是无损检测和磁性结构的鉴定，磁性结构在使用中可能会受到损坏，例如在核工业中。受检零件中的晶体不连续性或杂质会导致局部机械应力增加，从而导致意外故障。巴克豪森噪声效应可以很容易地用来检测这种不连续性，并让评估人员了解相关器件的状况。总而言之，这是一个需要了解的有用技术！

如果你认为巴克豪森是一个熟悉的名字，你很可能会想到 [巴克豪森稳定性标准](https://en.wikipedia.org/wiki/Barkhausen_stability_criterion) ，它是描述线性反馈电路振荡的一些必要条件的基础。我们以前讨论过这样的电路，比如这个 [潜入桥式振荡器](https://hackaday.com/2018/04/27/circuit-vr-oscillating-bridges/) 。
