# DIY 塑料单人快艇

> 原文：<https://hackaday.com/2020/06/17/diy-plastic-speedboat-for-one/>

Coroplast(波纹塑料的缩写)是一种有趣的材料。它的结构类似于纸板，但因为它是塑料的，所以它是防水的，可以用于一系列独特的应用。它通常用于政治庭院标志，但这种轻质材料有比广告更有趣的事情可做。例如，保罗·埃尔金斯用它来制造快艇。

[保罗]建造的船使用一块 coroplast，他切割并折叠成基本的船体形状。在那里，他开始组装完成这艘船所需的其他东西，包括用木头加强形状，添加方向盘，建造一个气窗来安装马达，并在驾驶舱中放置控制装置，如油门和转向装置。整个建筑足以推动一个人以大约 5 节的速度在水面上行驶，这令人印象深刻。

要自己做一个，你需要一个小型的舷外马达，但是在他的一系列视频中，所有其他的细节都有清晰的描述。如果你想建造自己的船，但不喜欢身后有噪音的二冲程发动机，你也可以考虑建造一艘静音推进模式的船。

如果你有很多 Coro，一定要看看[Paul]的其他项目，包括一个小房子。
