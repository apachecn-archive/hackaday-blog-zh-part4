# 黑客日超级会议:电子产品的 3D 打印

> 原文：<https://hackaday.com/2019/01/21/hackaday-superconference-3d-printing-for-electronics/>

在她的日常工作中，李正吉·D·丹斯比是一名软件模拟开发者，为视频游戏创建模拟。在空闲时间，她会把特斯拉的钥匙植入自己的手臂，制作 T2 的积木珠宝和 T4 的角色扮演剑，并寻求电子和 3D 打印的其他冒险。李正吉已经在 3D 打印社区中名声大噪，当她参加 RepRap meetups 和 Maker Faires 时，她被粉丝们包围。

她在今年的 Hackaday Superconference 上也很受欢迎，她在那里发表了关于 3D 打印和电子集成的演讲。李正吉的工作专注于道具和角色扮演，这本身就是一种技能，你只需要看看一些古老的《流言终结者》、来自 ILM 的纪录片镜头，甚至是模型制作者，就能意识到这是一种需要很多技能的神秘艺术。幸运的是，李正吉在那里向我们展示了她多年来学到的技巧，让制作一次性作品比你想象的更容易。

如果你把做某件事当作一份工作或一种爱好，你最终会想出一大堆窍门、交易工具或其他捷径来让下一件事变得更容易。电子道具也不例外。如果你需要一个电路，让一些 led 闪烁，让扬声器发出噪音，你可能会设计一次，然后在很多项目中重复使用。[李正吉]正在这样做，用一种她称之为“黑盒子”的技术。这些黑匣子可能是连接到微控制器的伺服系统，或者只是电池、灯和开关。不管怎样，你做得越多，你就越擅长做，制造硬件也不例外。

但对于李正吉来说，她对创建定制道具和角色扮演设备的大部分见解都是在图书馆管理领域。李正吉在她的许多工作中使用了 Fusion，你可以得到几乎整个麦克马斯特·卡尔目录的 Fusion 模型。将这本机械工程的圣经作为 3D 模型放在手边是一种力量倍增器；你不仅可以制作模型，还可以购买你需要的零件。麦克马斯特·卡尔目录确实是工程领域的伟大英雄之一，无论是谁决定将这些 3D 模型融入 Fusion，都应该被视为圣人。

不仅如此，李正吉还在使用 Autodesk 的“数据表到 Eagle part”工具。这也是一个游戏改变者。你再也不用在 Eagle 中制作零件了，它还会生成一个 CAD 模型。李正吉也在使用参数化工具设计零件，这是一种可以重新混合设计的伟大技术。

至于她建造了什么，你有 *Farscape* 枪和 LED 时装。你有塞尔达的剑，还有带齿轮的东西。李正吉是最具生产力和创造力的人才之一，我们非常高兴听到她在 Hackaday Superconference 是如何工作的。
