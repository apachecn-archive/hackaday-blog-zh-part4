# ATtiny85 贪吃蛇游戏是一个电路三明治

> 原文：<https://hackaday.com/2022/03/26/attiny85-snake-game-is-a-circuit-sandwich/>

如果说学习编程语言有什么隐现的不成文的规则，那就是必须通过以某种形式打印`Hello, World!`来打破语法。如果在一个新的微控制器上游戏编程存在这样的规则，那么毫无疑问[你应该实现 Snake](https://www.instructables.com/ATtiny85-Snake-Game/) 。

[![](img/1ec302bfa2545b001bca0444027b1094.png)](https://hackaday.com/wp-content/uploads/2022/03/ATtiny85-snake-inner.png) 这是[__cultsauce__]第一次尝试离开 Arduinoville，尽管他们确实用一台电脑对 ATtiny85 进行了编程，但一路上他们学到了很多东西。

用 85 年的游戏变出蛇来并不需要太多——主要是你需要一个屏幕来玩它(在这种情况下是一个有机发光二极管)，一些按钮来指引蛇走向食物点，一些无源器件和一个电源。

[__cultsauce__]从编写微控制器开始，然后在试验板上测试一切，这两种做法都令人钦佩。然后是制作这个胶合板和软木夹层的时候了，这给点对点的焊点一些喘息的空间，防止它们被压碎。请务必在休息后查看它的运行情况，如果您想吸引自己的“小蛇”，请从 GitHub 获取文件。

你可以用这个微型微控制器做很多事情，包括机器学习。
