# 巨大的数控路由器使用巧妙的技巧来提高性能

> 原文:[https://hacka day . com/2020/12/16/maximum-CNC-router-uses-clever-tricks-to-improve-performance/](https://hackaday.com/2020/12/16/enormous-cnc-router-uses-clever-tricks-to-improve-performance/)

由木材和 3D 打印部件制成的数控机器可能很受欢迎，但从精度和可重复性的角度来看，它们并不总是实用的。尤其是当机器的尺寸按比例增加时，其部件的符合性开始真正增加。但是这些问题能解决吗？[jamie clarke]认为是这样的，他正在建造一台能够处理一整块胶合板的数控刳刨机。(视频，嵌在下面。)

这是一项正在进行的工作，下面的视频只是这个过程的开始。但是我们发现[jamie]的构建即使在早期也很有趣，因为他包含了一些聪明的技巧来控制困扰大型 CNC 机器的正常溢出源。为了在预算范围内提供刚度，[jamie]为他的机器底座设计了一个木制扭力盒。这是根 CNC 项目所采用的方法，也是这个版本的灵感来源。该床由浅盒构成，通过应用于刚性轻质框架的受力表皮来实现其刚度。

扭力箱底座上是由普通长度的方形钢管制成的导轨。这些可能是很短的长度，但超过三米需要访问一个完整的胶合板，甚至钢会弯曲。[jamie]的解决方案是一个随车架移动的支架，它将横梁在所有行程点的无支撑长度减半。他正在使用类似的方法来对抗滚珠丝杠的抖动，在丝杠的中点有一个聪明的翻转支架。

到目前为止，我们对这个版本的质量印象深刻。我们期待看到它的发展方向以及这台机器的表现如何，所以我们正在密切关注播放列表的更新。估计建造成本为 1，500 英镑，这可能正是你一直在寻找的 CNC 版本。
