# 小鸭子狩猎看起来很有趣

> 原文：<https://hackaday.com/2020/03/08/tiny-duck-hunt-looks-like-big-fun/>

除非你有一台旧的显像管电视，或者你曾经通过 Wiimote 接收器和几个微控制器使用一个带液晶显示器的光枪，或者你住在成人游乐场附近，否则你现在可能真的很想玩猎鸭游戏。是时候更新狩猎许可证了，因为[【Danko】已经为 NodeMCU 板重新创建了游戏，而且现在是开放季节](https://www.reddit.com/r/arduino/comments/fdhb28/my_new_project_esp_game_on_oled_programed_using/)。

除了鸭子，你还可以拍摄各种大小和分值的可爱的 Twitter 式小鸟，如果你愿意，还可以拍摄一辆小型出租车卡车。有一个 60 秒的自由时间，然后时间到了，你的分数就会显示出来。作为一个特别的奖励，如果你什么都不打，就不会有自鸣得意的狗来嘲笑你。请务必查看演示，并在休息后制作视频。

这款掌上游戏机目前生活在一个布线良好的试验板上，而[Danko]则在一个定制的 PCB 上工作。他还计划在未来增加对 Arduboy 游戏的支持，也许是一个操纵杆，而不是一个 D-pad 按钮。

有很多关于老式 CRT 如何阅读 NES 轻机枪的神话流传着，但我们自己的[威尔·斯韦特曼]在他迷人的《猎鸭:重装上阵》中驳斥了这些神话。
