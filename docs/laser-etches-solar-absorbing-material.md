# 激光蚀刻太阳能吸收材料

> 原文：<https://hackaday.com/2020/02/06/laser-etches-solar-absorbing-material/>

如今拥有一台激光切割机并不是什么大事。但是罗切斯特大学的教授郭春雷有一个强大的飞秒脉冲激光器，并用它创造了可能是完美的太阳能吸收器。你可以在下面看到一个关于这项工作的视频。

显而易见，白色材料反射大部分光，因此比黑色材料吸收更少的能量——这是辐射计工作的一部分。特别是钨，是一种吸收太阳能的良好金属，但这种新的激光处理方法——在金属表面建立纳米结构——与未经处理的钨相比，效率提高了 130%。

该团队此前使用了一种类似的技术来使金属具有亲水性或疏水性。这一过程使金属呈现黑色，这也是高效率的来源。据报道，在一块普通的黑色钨上使用激光处理的表面仍然有 15%的增加。

我们想为我们的[太阳能泳池加热器](https://hackaday.com/2018/07/14/iot-solar-pool-heating/)提供一些这种材料。我们怀疑这是用钨开始的，所以这可能是第一次改造。非钨的[设计](https://hackaday.com/2014/08/07/easy-to-build-solar-pool-heater-saves-money-and-keeps-you-from-freezing/)有的是。我们更有可能在我们的[灯泡](https://hackaday.com/2018/09/26/bask-in-the-warm-glow-of-diy-incandescent-bulbs/)或[金属成型工具](https://hackaday.com/2018/09/07/the-how-and-why-of-tungsten-carbide-inserts-and-a-factory-tour/)中找到钨。
