# DIY 弧光灯使一个不必要的强大的自行车头灯

> 原文：<https://hackaday.com/2018/10/17/diy-arc-light-makes-an-unnecessarily-powerful-bicycle-headlight/>

还记得给自行车装上前灯意味着在你的车把上夹上一个巨大的铬合金子弹形灯，在你的前叉上栓上一个小发电机吗？打开车头灯意味着将发电机翻转到与前轮接触的位置，为白炽灯泡供电几英尺，这样引入的阻力会让你停下来。[这个可笑的弧光灯自行车大灯](https://www.youtube.com/watch?v=DwUC050kCAQ)不是那个。一点也不。

我们已经习惯了在他广受欢迎的 KREOSAN YouTube 频道上看到[Alex]做各种不可思议，有时甚至是不可能的事情。我们也习惯于看他的俄语视频，这丝毫没有降低手机的娱乐价值；然而，为不冒险的人提供了字幕。他的弧光灯的电极是电车上的石墨刷，而电池是由 98 个 18650 锂离子电池组成的看起来非常粗糙的集合。一个可怕的老鼠窝盘绕的电缆充当镇流器，以减轻电弧被击中时短路的影响。反射器是一个旧的卫星电视天线，上面覆盖着箔带，电极放在一个临时的支架上，原来是喇叭天线。它很亮，很吵，很危险，而且像恶魔一样冒烟，但是我们喜欢它。

当然，把它安装在自行车的前面只是为了好玩，尽管它的结构很简单，但它仍然可以工作。光线投射到邻居的公寓，无法联系到他们发表评论，但我们认为他们和我们一样感到有趣。
