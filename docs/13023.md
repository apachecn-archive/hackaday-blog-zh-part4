# 为自己建造一个天气预报模型

> 原文：<https://hackaday.com/2022/03/12/build-yourself-a-weather-reporting-diorama/>

如今，如果你不喜欢看早间电视，你可以随时从你的智能手机、电脑或一系列其他连接设备中的任何一个获得当天天气的更新。然而，如果你正在寻找一种更有趣的方式来看看商店里有什么，这个来自 *DIY Machines* [的[Lewis]的天气立体模型可能正是你正在寻找的。](https://www.instructables.com/WeatherBot-a-Motorised-Weather-Machine-3D-Printabl/)

构建使用 ESP32 作为项目的大脑，负责在互联网上查询最新的天气信息。这些信息随后显示在 2.9 英寸的电子墨水显示屏上，显示当地的温度、降雨几率和风速预测。到目前为止，一切都很简单。

然而，它真正有创意的地方是使用激光切割的“场景光盘”，上面有不同的图形来代表不同的天气条件。它们也可以是 3D 打印的。这些通过 ESP32 控制的伺服系统旋转，允许立体布景显示由当前预报通知的代表性场景。如果下雪了，你会看到雪人，但是如果天气很好，你更有可能看到一个带着狗散步的女人。

这是一种有趣的方式来了解大自然母亲在商店里有什么，并会在任何早餐吧启动看起来很棒。我们以前也见过[刘易斯]的一些伟大作品，比如这个令人惊叹的七段钟架。
