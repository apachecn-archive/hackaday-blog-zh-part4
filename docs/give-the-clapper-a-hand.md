# 为拍板者鼓掌

> 原文：<https://hackaday.com/2018/07/13/give-the-clapper-a-hand/>

虽然“拍板”可能首先让人联想到低成本广告的图像，但它仍然是一种遥控开关灯和房子周围其他东西的有用方式。但是如果你想开关的灯没有插在墙上，比如吊扇，铃锤就帮不了你。为了给这个臭名昭著的设备添加一些功能，[Robin] [从头开始构建了一个，它内置了你可能想要的所有额外功能](https://hackaday.io/project/159587-retrofit-clapper-light-switch)。

首先，新的舌板直接连接到电灯开关上，有利于开关本身的机械动作，而不是需要布线的机电继电器。通过这种设置，即使你租了一间公寓，也很容易安装，而且它的优点是可以切换任何设备，即使它没有插入墙壁。还有一个内置的麦克风来听掌声，但由于它是开源的，你可以对它进行编程，当它听到任何声音时就启动开关。它还包括连接到家庭自动化系统的能力。

如果你没有参与家庭自动化游戏的原因是你住在出租房里，无法对你的房子进行必要的改造，那么[罗宾]的拍板可能就是你最终实现生活空间自动化所需要的东西。所有文件都可以在项目网站上找到，包括 3D 打印计划和项目代码。然而，一旦你开始使用家庭自动化，你可以做更多的事情。
