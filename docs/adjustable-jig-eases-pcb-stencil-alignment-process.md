# 可调夹具简化 PCB 模板对齐过程

> 原文：<https://hackaday.com/2020/06/07/adjustable-jig-eases-pcb-stencil-alignment-process/>

印刷电路板模板使焊膏的应用变得轻而易举，但是在焊膏开始之前还有一个漫长而繁琐的过程。你必须想出一些方法来准确地在电路板上对齐模板，这往往涉及到使用胶带和旧 PCB 的应急设置，以及相当多的技巧和一点运气。

幸运的是，[Valera Perinski]提出了一个更好的处理模板的方法。模板印刷机是一种灵活、可调节的校准夹具，可以减少繁琐的调整工作。该夹具主要由铝挤压件和 3D 打印零件以及一系列现成的硬件组成。该机构有一个铰接框架，将模板固定在压板上方的固定位置，目标 PCB 放置在压板上。电路板由骑在螺杆上的夹具固定；随着模板在电路板上向下翻转，用户可以微调电路板和模板的相对位置，从而实现完美的对齐。下面的视频主要是一个建筑蒙太奇，但如果你跳到大约 29:00 标记，你会看到夹具通过其步伐。

诚然，这样的工具比磁带和备用 PCB 要多得多，但如果你做了大量的 SMD 工作，这可能是值得的。这肯定比锡膏分发机器人要省力。
