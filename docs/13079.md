# 开源 ASICs Hack Chat 重新定义了可能性

> 原文:[https://hack aday . com/2022/03/18/the-open-source-ASICs-hack-chat-redefines-possible/](https://hackaday.com/2022/03/18/the-open-source-asics-hack-chat-redefines-possible/)

曾几何时，电子爱好者所能得到的只有无源元件和真空管。然后集成电路出现了，它改变了一切。快进一点，负担得起的可编程微控制器出现了。电子学入门变得容易多了，硬件和软件之间的界限开始变得模糊。最近，业余爱好者社区了解了现场可编程门阵列(FPGAs)以及使用它们所需的工具。虽然不像 IC 或 MCU 那样广泛适用，但 FPGAs 在硬件黑客中的扩散再次打开了之前紧闭的大门。

我们目前正处于另一个范式转变的边缘，但如果你没有听说过，这并不奇怪。毕竟，过去几年有点不寻常，所以 2020 年谷歌与 SkyWater 和 Efabless 合作设计和制造开源专用集成电路(ASICs)的公告没有引起很多人的注意。但马特·文恩不是，他是本周黑客聊天的主持人。对他来说，这是他一直等待的机会。

马特和我们许多人一样，从制造电子设备和用废弃的旧硬件制造新玩意开始。他毕业于微控制器，当开源工具链开始出现时，他对 FPGAs 特别感兴趣。当然，在这一点上，它对他来说已经不仅仅是一个爱好了。当他看到来自 Efabless 的 Tim Edwards 演示一个用开源工具制作的芯片时，他正在瑞士 2019 年开源硬件周上发表演讲。不幸的是，所涉及的成本对于个人来说仍然太高，无法将他们的想法投入到硅中。

因此，当谷歌和 Skywater 宣布他们将为选择几个月后生产的开源 ASIC 设计买单时，Matt 说他处于一个很好的位置。[从那以后，他开始经营*零到 ASIC 课程*](https://www.zerotoasiccourse.com/) ，该课程旨在教你如何使用开源工艺开发套件生产自己的芯片，迄今已有 160 人接受了他的邀请。

正如你所料，聊天中的许多问题都与使用 130 纳米工艺实际能产生什么样的设计有关。特别是考虑到每个创造者的电路在每个多项目晶片上可以占据的物理空间的限制(MPW)。其他人想知道移植到现有 FPGA 设计上有多困难，或者该工艺在模拟应用中的效果如何。根据 Matt 在课程中看到的设计数量，他可以通过指出特定个人的 ASIC 来回答许多问题。例如，他举了 Harald Pretl 的数模转换器和 Thomas Parry 的 5 GHz 卫星收发器作为主要的模拟例子。

因此，让我们说，你把工作放在设计一个专用集成电路，它被批准生产在未来的 MPW，然后呢？首先，你必须希望一切都按计划进行。Matt 解释说，由于工具链中的时间问题，最初的运行几乎完全报废，尽管最后，[他基本上能够收回自己的芯片](https://www.zerotoasiccourse.com/post/mpw1-bringup/)。但是从那以后他们已经运行了几次，所以让我们假设没有生产问题。你家门口到底有什么？

[![](../Images/05c67858c1c633a730b2363883f1816b.png)T2】](https://hackaday.com/wp-content/uploads/2022/03/vennchat_chips_bright.png)

如果你期待一个方便的 DIP8，你可能会失望。虽然一些 DIY 友好的封装可能会很好，但现在 ASICs 采用的是晶圆级芯片尺寸封装(WLCSP)，间距为 0.5 毫米。如果你能相信的话，这实际上是对第一次运行的改进，第一次运行是以裸片的形式发布的。当然，正如 Matt 指出的那样，任何开始设计自己的定制 ASIC 的人可能都不会被一些精细间距焊接的前景吓倒。聊天中有些人想知道生产兼容的多氯联苯有多难，但是 Matt 说，根据他的经验[奥什公园已经接受了挑战](https://www.zerotoasiccourse.com/post/vga_clock_pcb/)。

就像之前的[金属 3D 打印黑客聊天](https://hackaday.com/2022/03/11/the-metal-3d-printing-hack-chat-brings-the-heat/)一样，本周的会议讨论了一个对硬件黑客和爱好者来说绝对前沿的话题。说实话，阅读 Hackaday 的绝大多数人不会像他们试图用自制电子枪烧结金属那样与 x 射线战斗一样，更有可能发送定制 ASIC。但这并不意味着有些人做这件事不重要，或者不鼓舞人心。也就是说，如果你*真的*成为少数几个可以吹嘘自己设计了定制芯片的人之一——别忘了给我们送一个过来。

我们非常感谢 Matt Venn 能够再次与我们分享他在开源专用集成电路领域的宝贵经验。如果你还没有查看它们，如果你想了解更多关于硬件黑客领域这个迷人的新前沿的信息，你必须查看他为 Remoticon 2020 举办的 [*零到 ASIC* 研讨会和他的演讲](https://hackaday.com/2020/12/29/remoticon-video-from-zero-to-asic-how-to-design-in-silicon/) [*开源 ASIC——一年透视*。](https://hackaday.com/2022/02/17/remoticon-2021-matt-venn-helps-you-make-asics/)
