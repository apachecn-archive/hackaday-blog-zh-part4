# 设计卫生纸卷保持架

> 原文：<https://hackaday.com/2019/01/17/designing-a-toilet-roll-holder/>

在某一点上，所有的东西都需要设计。设计厨房、体育场和星际飞船的人有工作可做。然而，也有工作给那些设计餐具，软管配件，甚至是卫生纸架的人。[[Eric Strebel]在这里分享这样一个故事。](https://www.youtube.com/watch?v=aitGh5M9_Bc&feature=youtu.be)

[Eric]涵盖了从开始到结束的整个过程。在开始时，各种各样的概念在纸上被起草和探索。各种各样的想法被相互评估，并被削减到少数几个。然后，纸板模型被创建，概念被进一步完善。这一过程还会持续几个阶段，直到进入为最终产品选择颜色和材料的有趣环节。

观察成本和制造过程对成品的影响，对于设计过程在现实世界中如何运作是有启发性的。卫生纸架本身也是一个有趣的单元——使用可调节的磁性棘爪来实现单手使用，还包括一个手机支架。

我们以前看过[Eric]的作品，比如他关于纸板在设计中的价值的初级读本。休息后的视频。
