# 磁铁让这只熊猫动了起来

> 原文:[https://hack aday . com/2019/09/23/magnets-make-this-panda-move/](https://hackaday.com/2019/09/23/magnets-make-this-panda-move/)

桌面上的单板计算机很适合快速演示，但如果将它带到野外(甚至是房子的其他地方)，您可能需要更多的安全保护，以防碎片、ESD 和跌落。随着单板机变得越来越有用，这成为一个越来越需要解决的相关问题，加上一个光滑的外壳可能是一个好的台式电脑黑客和一些看起来准备作为产品出售的东西之间的区别。[Chris](作为 ProjectSBC)一直在为 LattePanda Alpha SBC 开发一系列名为 [MagClick Case System](https://liliputing.com/2019/09/diy-case-turns-a-tiny-pc-into-a-modular-game-console-for-handheld-or-desktop-gaming.html) 的适应性案例，绝对值得一看。

![](../Images/811e7a679eb7282249663798c636ce5c.png)

LattePanda Alpha 不是一款普通的单板机；它本质上是一款低功耗超极本的主板，包含高达英特尔酷睿 M 系列处理器、8GB 内存和 64GB eMMC。更不用说板载 Atmega32u4、WiFi、千兆以太网等等。如果你把 PCIe 从一个 m.2 卡槽中取出来，它有足够的马力用作日常的台式电脑，甚至是一个轻度游戏系统。但是[Chris]意识到这种适应性正在成为一种痛苦，因为随着他的使用需求的变化，他不得不逐个案例地进行调整。于是 MagClick 案例系统诞生了。

顾名思义，MagClick 表壳设计用于在建造和安装过程中通过磁力将*与*卡合在一起。扣上 SBC 的侧面，加上顶部和底部，您就有了一个暴露所有端口的封闭式台式机系统。打开顶部，将它扣在 GPU/电源外壳的侧面，您就有了一个固定的游戏系统。再次将它取下，并将其连接到[Chris]的任天堂 Switch 风格便携盒(配有电池和游戏手柄),您就可以在旅途中玩游戏了。在实践中，其中一些转换仍然需要插入内部和外部线路，但我们认为这是一个很大的改进。

现在，一些 MagClick 变体的文件在 Thingiverse 上[，而【Chris】现在有一个](https://www.thingiverse.com/ProjectSBC/designs)[教程来组装桌面(“基本”)外壳](https://www.thingiverse.com/thing:3860552)。如果你有兴趣了解更多信息，请在休息后观看精彩的介绍视频。

这不是我们最近看到的唯一一例 SBC 病例。[Madeline Gannon]为 Jetson Nano 做了一系列类似的适应性案例,这样她就可以把它带到世界上来。
