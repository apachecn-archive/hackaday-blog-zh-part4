# 树脂堆积被证明是杂乱和困难的

> 原文：<https://hackaday.com/2021/12/12/resin-stacking-proves-messy-and-difficult/>

3D 打印机通常是您想要一次性快速原型时使用的工具。然而，它们越来越多地被用于大量生产。[鞠波叔叔] [决定尝试树脂堆积技术](https://www.youtube.com/watch?v=WNcTij80SFE)，以便在他的树脂打印机上快速制作许多小雕像。然而，并不是一切都完全按照计划进行。

技术很简单。这个想法是在树脂 3D 打印机上一次连续打印出一个物体的多个副本。为了实现这一点，对象被多次克隆，并且脚手架被创建以允许多个对象堆叠在彼此之上。这必须小心进行，以避免破坏物体的几何形状，与支撑材料类似，在该过程中使用更多的树脂。

[鞠波叔叔]试验了几次，但在试图打印一些小型磁电机雕像时，遇到了许多问题。当印刷物从构建板上脱落时，使用 raft 的初始实验失败了。随着木筏被移除，第二次印刷失败，因为脚手架印刷不太正确。进一步的调整和加强支架改善了情况，[鞠波]设法在一次操作中打印了 93 个小雕像。

如果你想在尽可能短的时间内在树脂打印机上打印出大量的模型，这是一项非常有用的技术。然而，当你改进这项技术时，预计会花费大量的树脂。您还需要一个[大清洗槽](https://hackaday.com/2021/09/17/add-on-lets-fdm-3d-printer-wash-and-cure-resin-parts/)来在后处理过程中清洗印刷品。休息后的视频。
