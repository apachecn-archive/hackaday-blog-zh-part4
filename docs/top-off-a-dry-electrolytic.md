# 给干电解液加顶

> 原文：<https://hackaday.com/2019/10/23/top-off-a-dry-electrolytic/>

制作电容器相当容易。只要让两个导体靠在一起。面积越大，距离越近，电容就越大。[BigClive]发现了一些假电容，它们本应价值很高，但事实并非如此。把它们拆开后发现，电容器内部没有电解质，而电解质赋予了这些单元它们的名字和高价值。他做了什么？[混合一些电解液，然后重新装满](https://www.youtube.com/watch?v=IAQClL3bZzg)看看会发生什么。你可以看下面的视频。

电解电容有一个秘密武器，可以让两个电极尽量靠近。电解质在一个电极上形成非常薄的绝缘层，电容在导电流体和该电极之间，而不是在两个电极之间。这使得导体之间的间隙非常窄，也解释了为什么小型电解电容比其他大多数技术在相似的外形下具有更大的电容。

还有其他的诡计在起作用。例如，电容器的阳极侧通常被蚀刻或粗糙化，以提供比类似的平坦材料更大的表面积。在制造过程中，成型电压导致氧化层形成，并且还在电容器上设置极化。通过器件的反向电流可能会导致短路并迅速损坏器件。

【克莱夫】自制的硼酸电解液和防冻液起作用了吗？有几分是，也有几分不是——看看视频就知道了。了解这一切是如何工作的可以帮助你修理旧的电解装置。每隔一段时间，我们会尝试制作自己的电容器，用于[教育目的](https://hackaday.com/2011/05/09/roll-your-own-capacitors/)或[超高压目的](https://hackaday.com/2015/08/02/homemade-high-voltage-caps/)。
