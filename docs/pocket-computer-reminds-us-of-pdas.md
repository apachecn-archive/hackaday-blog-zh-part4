# 掌上电脑让我们想起 PDA

> 原文：<https://hackaday.com/2022/06/23/pocket-computer-reminds-us-of-pdas/>

在 2000 年代末智能手机出现之前，人们对能够完成相对简单的计算任务的袖珍型电脑仍有合理的需求。Palm Pilots 和其他 PDA(个人数字助理)在 90 年代和 21 世纪初风靡一时，尽管对于那个时代的尖端技术来说，许多这些设备都有天文数字的价格标签。这款基于 Arduino 的 PDA 可以追溯到那个时代，尽管它的零件清单要容易得多。

这款车是基于一款带有有机发光二极管屏幕的 Arudino Nano 设计的，拥有 PDA 必备的五大功能:计算器、秒表、游戏、电话簿和日历。在如此小的微控制器上集成所有这些组件，当使用默认库时，内存很快成为一个问题。[Danko]使用他自己的定制库，以便充分利用内存，这些库都可以在项目的 GitHub 页面上找到。该构建还包括一个定制的 PCB，以保持整个袖珍计算机的口袋大小。

这个小小的版本中还包含了其他一些功能，比如可以用电位计玩的突围游戏。这是一个令人印象深刻的构建，它尽可能地利用了微控制器的功能，如果您喜欢像使用 PC 一样使用微控制器的项目，可以看看这个带有自己的命令行界面的 Arduino 构建。
