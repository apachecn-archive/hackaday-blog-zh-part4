# AMD 的 Threadripper 是“最强大”的 DIY 笔记本电脑的跳动、发热的心脏

> 原文：<https://hackaday.com/2021/01/22/amds-threadripper-is-the-beating-heating-heart-of-most-powerful-diy-laptop/>

如果你有钱花，市场上有很多功能强大的“替代台式机”的笔记本电脑。不过，有时候，这并不能满足你对真正的、无拘无束的计算能力的渴望。当你对失败有着无法满足的渴望时，你必须自己去闯一闯，就像[杰夫]用 Threadripper 笔记本电脑所做的那样。

目标是将 AMD Threadripper 处理器打包成名义上的便携式笔记本电脑格式。对于这一构建，AMD 1950X 因其经济性和巨大的计算能力以及 180 瓦的 TDP 而被选中。到目前为止，这种高热量输出阻止了芯片进入便携式制造，但[Jeff]不认为这是一个问题，而是一个挑战。

其结果是一个狂野的、捆绑在一起的高功率部件，可以被亲切地称为笔记本电脑——尽管我们建议不要把它放在你的膝盖上。惠普 Media Center 机箱中集成了 4K 18 英寸屏幕、键盘、触摸板和许多戴尔 Powerbanks，符合通常的外形规格，尽管暴露的热管和电缆比普通消费者可能习惯的要多。

[Jeff]声称这是目前世界上最强大的笔记本电脑，因为使用 3950X 的构建在他们的应用程序中抑制了它。我们没有数据进行比较，但我们肯定不会在战斗中用我们自己的便携式装备来对抗它。DIY 笔记本电脑在 Hackaday 有着悠久的历史，可以追溯到 2007 年。[如果你有自己的狂野身材，一定要给我们写信。](http://hackaday.com/submit-a-tip)休息后的视频。
