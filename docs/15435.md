# PCB 上的赛车

> 原文:[https://hackaday.com/2022/11/24/racing-cars-on-a-pcb/](https://hackaday.com/2022/11/24/racing-cars-on-a-pcb/)

委婉地说，卡尔·弗里德里希·高斯是一位博学的人，对现代世界中我们认为理所当然的大部分事情负有责任。作为一名物理学家和数学家，他开创了几个研究领域，包括磁学领域。但由于他在第一辆汽车制造前几十年就去世了，他不可能想象出这一发明，[一条由[杰夫·麦克布赖德]设计的名为高斯赛道](https://jeffmcbride.net/gauss-speedway/)的磁力老虎机赛道，它以这位著名科学家的名字命名。

高斯高速公路的灵感来自机器人技术的最新发展，许多小型机器人可以借助集成在其操作区域的电路轨迹在一个大区域内行驶。将正确的电流施加到这些轨迹上，就会产生推动机器人的磁场。[Jeff]想要制造类似的东西，直接集成到印刷电路板中，于是产生了吃角子老虎的想法。小型汽车中有微小的磁铁，它们与 PCB 中的轨迹相互作用，允许汽车在轨道上高精度地移动。他确实放弃了传统的老虎机控制器，转而选择了直接安装在 PCB 上的按钮式控制器，这意味着一切都是完全集成的。

虽然这更多的是一种演示或概念验证，但这种类型的机器人的一些特征可以在[这个视频](https://www.youtube.com/watch?v=uL6e3co4Qqc)中看到，它显示它们在不平坦的表面上，甚至在墙上以高精度快速移动。像这样的磁性机器人正在复兴，我们甚至已经看到[的一些机器人利用磁性来变形](https://hackaday.com/2022/05/14/attack-of-the-magnetic-slime-robots/)。
