# 动画南瓜在万圣节唱歌和惊吓

> 原文：<https://hackaday.com/2020/10/30/animated-pumpkins-sing-and-scare-on-halloween/>

![](img/fc477f14bfe9f84daae8ec8c91b3e195.png)

The animated video combined with the 3D-printed prop makes for an excellent effect.

用南瓜雕刻南瓜灯是许多人最喜欢的万圣节传统，但是依靠蜡烛和刀子显然是低技术含量的。[DIY Machines]的[Lewis]决定做一些更有活力的东西来吓唬当地的“不给糖就捣蛋”的人。

该建筑由 3D 打印的南瓜组成，用低成本投影仪从背后照亮。在树莓派的驱动下，投影仪播放视频文件，将动画脸投射到南瓜上。这种效果非常棒，给人一种真实的拟人化的杰克南瓜灯坐在你家门廊上的错觉。为了控制系统，一系列街机按钮被连接到树莓 Pi 上，允许游客激活一首歌、一个惊吓或一个故事。

这是一个有趣的建筑，是为你的万圣节装饰增加一些互动性的好方法。如果你想让你的工作更上一层楼，考虑把它投射到你的整个房子里。休息后的视频。
