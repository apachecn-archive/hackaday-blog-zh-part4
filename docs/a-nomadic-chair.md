# 游牧式椅子

> 原文：<https://hackaday.com/2022/12/29/a-nomadic-chair/>

不缺少不同类型的折叠椅或便携式椅子，但设计师[Jorge Penadés]制造了一款能够长期使用的背包椅。

不用螺丝或胶水组装的家具总是很吸引人，而这把椅子正符合这个要求。使用简单的金属连接件和细木工，它可以在两分钟内安装和拆卸，而不会有像包椅一样脆弱的感觉。木材经过自然处理，连接器呈现出亮丽的色彩，而不会让人感到力不从心。如果你点击链接，甚至有一些沙发版本的图片。

在背包模式下，这些部件由皮革补丁和棘轮带固定在一起。[Penadés]关注的是这件家具的便携性，而不是舒适性，但我们认为这种连接方式可以在未来用于更舒适的便携式家具。

如果你在寻找更有趣的家具，看看这张有着令人困惑的秘密的[铃鼓桌](https://hackaday.com/2022/11/01/tambour-table-with-a-puzzling-secret/)或者这些[数控座椅](https://hackaday.com/2013/09/18/revenge-of-the-glueless-stool-and-a-new-folding-chair/)。
