# 动力转向泵的用途是提高速度

> 原文：<https://hackaday.com/2018/08/29/power-steering-pump-repurposed-for-great-speed/>

电动自行车最近得到了很多关注。几乎任何人都可以在网上购买一套工具，并获得一个完全合法的足够大范围的街道骑行。但是，如果你不想走套件路线，而宁愿采取一种能让你在这些部件周围得到更多关注的策略，那么请从[Jule553648]最近的构建中做些笔记，即[绝对没有使用套件](https://www.youtube.com/watch?v=u5HlD77K3Jc)中的任何部件。

从建立的电机是一个电动助力转向泵从一个垃圾场的汽车。它被安装在一次性的自行车后架上，并在来自易贝的袖珍自行车变速箱的一些齿轮的帮助下驱动后轮。这一建造中的许多零件都是使用 CAD 和机械车间设计和制造的，电池和电源控制器的零件都是通过中国采购的，以节省成本。

整个建筑有一种我们无法抗拒的自制氛围。这辆自行车可以在平地上以 35 公里/小时的速度行驶，而不会流汗，行驶里程约为 40 公里，这是不容小觑的。它甚至可能是合法的，这取决于电机的瓦数以及你是否生活在欧洲(电动自行车上通常不允许使用油门)。不过，如果你缺少一家机械商店，[不久前，我们推出了一套制作精良的工具](https://hackaday.com/2018/08/08/electric-bike-from-the-ground-up/),你可以把它作为一个模型来试用。
