# 战斗机器人飞向天空！

> 原文:[https://hackaday.com/2019/05/01/battlebots-to-the-skies/](https://hackaday.com/2019/05/01/battlebots-to-the-skies/)

如果你太小，不记得电视上的战斗机器人，有两件事你应该知道。首先，YouTube 上有很多这部史诗般的机器人大战的精彩片段，其次，现在有了一个更好的版本，用无人机取代了仅仅局限于陆地上的机器人。叫 [DroneClash 2019](http://mavlab.tudelft.nl/droneclash-2019-high-tension-high-tech-and-good-humoured/) ，看起来很惊艳。

这些机器人不仅被设置在一个盒子里，并被要求相互战斗，他们首先必须通过一条带有反无人机措施的走廊。无人驾驶飞机必须通过并进入最后一个房间的皇家战役。如果这还不够好的话，这个活动是由一位荷兰王子发起的，由一所大学举办。

这是一项年度活动，旨在推动无人机和反无人机技术的发展，但我们很高兴看到它被选作电视节目。如果没有，你可能会满足于不久前的一场巨大的人类驱动的机器人比赛，或者可能只是一头钻进旧战斗机器人剪辑的兔子洞。
