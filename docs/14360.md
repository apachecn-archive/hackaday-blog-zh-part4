# 当[卡尔]说跳的时候，PCB 说“多高？”

> 原文:[https://hack aday . com/2022/07/30/when-Carl-says-jump-PCBs-say-how-high/](https://hackaday.com/2022/07/30/when-carl-says-jump-pcbs-say-how-high/)

我们注意到[Carl Bugeja]喜欢柔性印刷电路板。他的最新发明是制造基于 PCB 的弹簧，结合一些磁铁来创造跳跃的小装置。我们不确定这些东西会有什么实际用途，但是它们确实很新颖，你可以在下面的视频中看到它们跳来跳去。

[Carl]对弹簧的结构和设计做了很多实验。你可以在视频中看到几个迭代，但并不是所有的都很好。当电流流动时，底座中的 PCB 线圈被磁化，这排斥或吸引弹簧另一端的磁铁。你能用 PCB 弹簧做什么？我们不确定。也许这就是你的下一个微型机器人爬楼梯的方式？

增加加强筋会产生太硬的弹簧，使电磁铁无法吸引。我们想知道如果在底部有不同的线圈设计是否会更有效。就此而言，如果你真的想优化跳跃行为，你可能不必在那个位置使用扁平的 PCB 线圈。

通常，当我们与[Carl]联系时，他正在制作[基于 PCB 的马达](https://hackaday.com/2021/05/30/tiny-pcb-motor-robot-is-making-its-first-wobbly-moves/)。或者，有时，他在制作用于回流焊接的 [PCB 加热器](https://hackaday.com/2022/02/13/sticker-brings-the-heat/)。当然，我们之前已经看过[跳跃机器人](https://hackaday.com/2020/08/02/little-jumping-bot-can-now-stick-the-perfect-landing/)。我们会说磁铁似乎没有使用压缩空气的强度大。
