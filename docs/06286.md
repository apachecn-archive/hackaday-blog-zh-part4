# 模型火箭发射器是如此严重，它有一个公文包

> 原文:[https://hack aday . com/2020/04/17/model-rocket-launcher-is so-serious-it-has-a-公文包/](https://hackaday.com/2020/04/17/model-rocket-launcher-is-so-serious-it-has-a-briefcase/)

还有什么比发射自己制造的复杂火箭更令人激动的呢？首先，用任何比股票点火系统更好的东西来启动它将是朝着正确的方向迈出的一步。一个装满有趣的过度杀戮的公文包怎么样？

[FastEddy59]正在建造一个带有推力矢量控制(TVC)系统的模型火箭，以帮助稳定。令我们高兴的是，他设计了一个同样雄心勃勃的控制器，用安全代码和物理钥匙来增加发射序列的味道。没有一个巨大的紧急停止按钮来关闭一切的发射控制器是什么？不完整，如果你问我们。

在碳纤维包裹的丙烯酸罩下，有一个 Arduino 大型发动机和一个 NRF24 LoRa 模块，用于传输到火箭。甚至还有一个 DHT11 温度传感器来验证发射条件是否理想。它仍然是一项正在进行的工作，有许多功能即将推出，如更花哨的标签和大量适合隐藏扬声器的声音文件。这个案子有很多内容，休息之后，[FastEddy59]的视频摘要已经准备好了，正在 pad 上等待。

[FastEddy59]计划在几个月后进行第一次发射，我们真诚地希望他[给火箭装上照相机](https://hackaday.com/2020/02/16/a-cheap-and-easy-gopro-mount-for-model-rocketry/)。
