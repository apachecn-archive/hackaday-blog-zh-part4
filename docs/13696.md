# 虚拟现实频谱分析仪

> 原文:[https://hackaday.com/2022/05/21/vr-spectrum-analyzer/](https://hackaday.com/2022/05/21/vr-spectrum-analyzer/)

在某个时候，我们可能都希望我们有一个虚拟现实耳机，让我们可以围绕我们的设计飞行。虽然不完全相同，但[manahiyo831]有一些甚至可能更好的东西:一个 [VR 频谱分析仪](https://github.com/manahiyo831/VR_RTLSDR_AVATAR)。您可以在下面的视频中了解到它的样子，尽管那实际上是来自早期版本。

视频显示一台远程电脑使用 RTL 加密狗接收信号。新版本在 Quest 2 耳机上运行，因此您只需将加密狗连接到耳机上。当然，戴上这个你会看起来像一个太空学员，但是——老实说——如果你愿意戴着耳机被人看到，这没有更多的硬件。

不过，我们真正想看到的是一个定向天线，这样你就可以在你看的方向上看到信号。现在这将是一件大事。事实上，这是不可否认的酷，但我们不知道它的真正用途是什么。

你还想看哪些 VR 测试装备？一个类似 *Tron* 的逻辑分析仪？一个能让你在空中画波形的函数发生器？耳机示波器？又或许只是 VR 中的一个巨型工作台？

频谱分析仪[是 SDR](https://hackaday.com/2021/11/15/pluto-spectrum-analyzer-uses-command-line/) 的一个自然项目。或者[里面有特别提款权的东西](https://hackaday.com/2021/08/27/cable-modem-turned-spectrum-analyzer/)。
