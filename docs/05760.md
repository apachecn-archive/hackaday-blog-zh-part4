# 手势控制简单易行

> 原文:[https://hack aday . com/2020/02/22/手势控制轻松之路/](https://hackaday.com/2020/02/22/gesture-control-the-easy-way/)

手势控制是一项已经流行了很长时间的技术，但从未被主流接受。Wii 保龄球有一段时间很有趣，但我们还没有经常用手势开门或点披萨。然而，自己做起来可能相当容易，就像[RC Lover san]发现的那样，他有着一副瘦骨如柴的身材。

通常，当我们想到手势控制时，我们会想到物体跟踪相机或 MEMS 加速度计。相反，这种构建使用简单的倾斜开关，就像你可能在很久以前的弹球机中发现的那样。其中四个被放置在一个手腕安装设备上，允许用户倾斜他们的手臂，以不同的方向移动遥控汽车。倾斜开关很容易侵入玩具遥控汽车的控制器，因为它们只是取代了 PCB 上现有的按钮。

这个项目表明，并不是所有事情都必须用先进的传感器和复杂的算法来完成。有时，这一切都可以通过一些廉价的开关和一些独创性来完成。此外，使用手臂动作在地板上快速移动 BB-8 看起来很有趣。我们也看到了用花盆构建简单手势控制的其他尝试。休息后的视频。
