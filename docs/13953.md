# 走钢丝会作弊吗？

> 原文:[https://hack aday . com/2022/06/18/你能在走钢丝时作弊吗/](https://hackaday.com/2022/06/18/can-you-cheat-at-tightrope-walking/)

走钢丝不是一件容易的事——需要数年才能掌握(即使有平衡杆)——但这对(詹姆斯·布鲁顿)来说太难了。很明显，解决方案只是一些电子控制平衡技术的应用问题，而[詹姆斯]正是这个工作的人。看起来有点像《捉鬼敢死队》中的道具和中世纪刑具的结合体，[这个沉重的平衡作弊装置](https://www.youtube.com/watch?v=RhOrwjPkes0)几乎起作用了！

更严重的是，两足动物的平衡是一个复杂的问题。你有多个肢体部分，可以以多种方式独立移动，上身也有助于以难以预测的方式围绕重心移动。所以，一个简单的绑在躯干上的反扭矩装置帮不上什么忙也就不足为奇了，但看着他尝试肯定很有趣。底线是——我们的身体相当重，如果反作用轮本身不太重，就很难产生校正感兴趣平面倾斜所需的力，而额外的质量对整体平衡问题没有任何帮助。我们认为整体概念是合理的，只是所有这些额外的四肢使这种简单的传感和补偿策略只能部分有效。

稳定小型机器人可能比人容易一点，比如这个陀螺仪稳定的 monowheel，但有时你甚至不需要陀螺仪，因为你可以直接控制驱动轮。
