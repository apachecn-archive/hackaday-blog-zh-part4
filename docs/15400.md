# 来一杯新鲜的烘焙咖啡

> 原文:[https://hack aday . com/2022/11/18/pop-your-way-to-a-fresh-bake-coffee/](https://hackaday.com/2022/11/18/pop-your-way-to-a-fresh-roast-coffee/)

在我们的社区，咖啡的制作和饮用已经成为一种艺术，许多人不遗余力地追求完美的杯子。埃里克·索伦森热衷于烘焙自己的咖啡豆，但考虑到专用烘焙机的价格，他使用热空气爆米花机制作了自己的咖啡豆。

故事分几个部分展开，首先是禁用 popper 上的热断流器，代之以热传感器。Arduino 控制风扇和加热线圈，以随着时间的推移调节和改变烘焙温度。也许是软件和用户界面让这个项目大放异彩，在 320 x 240 像素的触摸屏上有一个图形界面，图形配置文件界面更让人想起回流炉，而不是厨房电器。整个东西和它的电源一起整齐地包装在一个细长的盒子里，虽然我们不是豆类专家，但我们欣赏成品均匀的棕色。

这些年来，烘焙咖啡的人在这里出现过无数次。最近的一个是[这个摆盘设计](https://hackaday.com/2021/07/13/wobble-disk-coffee-roaster-looks-good-in-wood/)。
