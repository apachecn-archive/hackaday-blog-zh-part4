# 声控眼镜和放大镜

> 原文：<https://hackaday.com/2018/11/03/voice-controlled-glasses-and-magnifying-lens/>

如果你正在阅读 Hackaday，你可能非常熟悉非常小的部分。0201 电阻很小，甚至更小的零件也不是没听说过。手机上的螺丝非常小，如果你想检查 3D 打印的细节，放大镜真的很方便。如果你有好的视力并且你还年轻，这很容易，但是我们中的很多人没有这种奢侈，而是必须依靠放大镜和放大镜。[【Mauro】的 Hackaday 奖项目通过增加声控伺服系统，使佩戴这些 loupes 和镜头变得更加容易](https://hackaday.io/project/160106-voice-controlled-glasses-and-magnifying-lens)。

这种设备背后的基本想法很简单——只需在一副眼镜上安装一个标准的业余爱好伺服系统，并在铰链上安装一副 loupes。有了 Raspberry Pi Zero W，控制这个伺服很容易。这里真正的技巧是添加语音控制，为此[Mauro]正在使用沃森语音到文本服务。将一双眼睛从眼睛上移开就像用 Watson 语音转文本服务建立一个帐户一样简单，并使用 NodeJS 发出 API 调用。

除了放大镜之外，[Mauro]还想出了一些其他的主意，让这个设备变得更加有用。它可以用来焊接护目镜，在你开车穿过隧道时摘掉太阳镜，或者它甚至可以被改造成那些疯狂吸管的改进版本，这些吸管可以在塑料眼镜的边缘吸取液体。这里的潜力几乎是无限的，这是今年 Hackaday 奖中较好的项目之一。

休息之后，你可以看到这些眼镜的运行视频(没有语音激活)。
