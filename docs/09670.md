# 用电移动东西

> 原文：<https://hackaday.com/2021/03/29/moving-things-with-electricity/>

我们一直用电在马达和磁铁的帮助下移动东西。但是如果你有足够的电压，你可以单独用电压移动东西。正如[詹姆斯]发现的那样，如果你的物体——在他的例子中是乒乓球——是导电的，效果会更好。

他想给他的“伟大的球机”增加一个范德格拉夫发电机，这个球机已经有一些移动乒乓球的酷方法了。然而，为了获得静电运动，[詹姆斯]不得不求助于用射频屏蔽喷雾喷洒这些球。

我们喜欢他在花大量时间 3D 打印最终项目之前做一些实验，看看球是如何工作的。不过，我们确实想知道，是否有更便宜的方法可以让球导电，比如石墨或导电墨水。

最终的结果是一个 3D 打印的低能乒乓轨道炮，或多或少。这个设计非常聪明，即使你不想让乒乓球加速，你也能找到一些想法。当然，一个 Arduino 控制着整个事件。如果你确实想复制它，你可以在 GitHub 上找到 [CAD 文件和软件](https://github.com/XRobots/Accelerator)。

我们总是对范德格拉夫发电机情有独钟，即使是很小的发电机。如果你[了解它们是如何工作的](https://hackaday.com/2017/02/16/how-a-van-de-graaff-generator-works/)，它们并不难制造。
