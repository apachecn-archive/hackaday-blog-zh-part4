# 通过在顶部安装 Jibo 使工业机器人人性化

> 原文:[https://hack aday . com/2019/05/25/humanizing-industrial-robots-by-sticking-a-jibo-on-top/](https://hackaday.com/2019/05/25/humanizing-industrial-robots-by-sticking-a-jibo-on-top/)

在我们的现代世界中存在着大量的机器人，其中绝大多数是高度专业化的机器。他们做一项工作，而且做得很好，但是他们没有太多的个性。[古伊列梅·马丁斯]正在做一个有趣的项目，建造一个可以制作巧克力艺术品的机器人手臂，但它需要一些东西来使它更加人性化。谢天谢地，Jibo 在那里帮忙。

对于外行人来说，Jibo 是一个由一家初创公司生产的伴侣机器人，这家公司后来倒闭了。依赖云意味着当资金耗尽，服务器关闭时，Jibo 基本上就死了。【古伊列梅】设法挽救了其中的一个单位，并给了它新的生命。

由于死去的公司无法提供 SDK，机器人的整个大脑被 LattePanda 取代，latte panda 是一台 Windows 10 单板计算机，集成了 Arduino 微控制器。这与一系列 Phidgets 电机驱动器相结合，以控制 Jibo 的所有关节，并与一些 Unity 软件相结合，以在原始屏幕上提供迷人的表情。

随着 Jibo 的身体安装在机械臂上，一个简单的巧克力装饰机器人现在有了个性。这个机器人可以向人类挥手，并在一天的工作中表达自己的情感。这是添加到项目中的一个有趣的特性，并且肯定会使它更有趣。我们以前看到过处理类似主题的项目，试图制造友好的机器人宠物作为伴侣。休息后的视频。
