# 天才或诅咒，这 USB-C 连接器是灵活的

> 原文:[https://hack aday . com/2022/01/01/genius-or-curled-this-USB-c-connector-is-flexible/](https://hackaday.com/2022/01/01/genius-or-cursed-this-usb-c-connector-is-flexible/)

自从第一个实验者将 PCB 安装到 USB-A 插座中以来，USB 连接器就一直致力于对其机械规格进行创造性的解释。到目前为止，USB-C 标准及其较小的连接器基本上避开了这一趋势，尽管由于[Sam Ettinger]的工作，这种情况可能会改变。他自己对[他的 USB-C 连接器使用柔性 PCB 和 BGA 封装的 ATTiny84A 微控制器](https://hackaday.io/project/183302-a-usb-c-connector-on-a-flex-pcb)的描述是“被诅咒的”，但我们无法决定它是否也应该被称为“天才”。

这款受启发的连接器制造的关键是实现了 BGA 和柔性 PCB 的总厚度达到所需的 0.7 毫米。BGA 提供了必要的刚度，尽管它是单面连接器，但它完全适合空间。有几个演示板作为概念验证，全部可以在 GitHub 库中找到[。](https://github.com/settinger/USB-C_PCB_experiments)

我们可以看到这种技术在各种小型 USB-C 项目中得到了应用，不管它有多邪恶或多么天才。我们喜欢看到一些项目，这些项目推动了利用这种媒介所能做的事情的发展，[向以前被诅咒的 USB-C 设备](https://hackaday.com/2021/03/22/cursed-usb-c-when-plug-orientation-matters/)致敬。
