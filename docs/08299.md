# 无限镜吉他碎片永远

> 原文:[https://hack aday . com/2020/11/09/infinity-mirror-guitar-shreds-forever/](https://hackaday.com/2020/11/09/infinity-mirror-guitar-shreds-forever/)

就在我们认为没有什么可以做成无限镜子的时候，[伯尔斯艺术]去做了一件现在看来显而易见的东西——一把无限镜子吉他。休息之后，请查看[构建视频](https://www.youtube.com/watch?v=lhx6ZCrspX0)，在那里【伯尔斯艺术】不浪费任何时间地直接进入主题。

他首先为琴身、一体式琴颈和琴身制作了一个 3/4 英寸的木头框架。顶部的亚克力有双向镜面膜，背面涂有镜面漆，以获得无限效果。[伯尔斯艺术]还为拾音器和电子产品制作了亚克力盒子。这两个都被打磨成霜状，所以灯光反射得很好。

没有什么超级花哨的电子产品，只是一些应用程序控制的 RGB LEDs。我们希望看到一个版本，其中的发光二极管实时响应音乐。效果还是很酷的，所以如果你不想看整个构建，[至少看看最后的演示，其中【伯尔斯艺术】播放了一个即兴重复片段](https://www.youtube.com/watch?v=lhx6ZCrspX0&t=8m9s)。从来没有一个延迟踏板如此合适。

如果你不是一个很好的制琴师，不要担心不能制作翻唱版。我们已经看到了很多无限镜子，但是如果你想要一些有用的东西，就做一些无限杯垫吧。
