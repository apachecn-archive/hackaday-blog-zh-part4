# 修复封装变压器内的电容器

> 原文:[https://hack aday . com/2018/07/22/repair-a-capted-transformer-inside-a-capted-transformer/](https://hackaday.com/2018/07/22/repairing-a-capacitor-inside-a-potted-transformer/)

我们总是喜欢看(卡尔森的)视频，因为他看起来像是在拍摄上世纪 50 年代免下车电影中的火箭船场景。在最近的一个视频中，他发现了一个旧示波器，它有一个变压器组件，内部装有一对电容器。电容器出了故障，所以[卡尔森]决定修理它。问题？变压器和电容器用某种焦油化合物封装在一起。你可以在下面的视频中看到结果。

他实际上并不确定电容器是否真的在变压器中，但它们在原理图中，通过排除过程，它必须在内部。一旦他解放了变压器，他做了一些测试，以确定在 depotting 电容器。去涂层需要大量的热量，可能会损坏变压器，所以他想确定它真的在那里。

几乎和去渍一样有趣的是准备工作。因为变压器仍在工作，他仔细测量了次级电压，并以一种不寻常的方式用标记(我们只是在电线上使用小标签)标记了电线。

这个过程的开始是有点机械移除索环，但主要部分需要一个带温度控制的烤箱。他实际上在烤箱上切了一个洞来安装变压器，这样他就可以在设备被加热时对其进行操作。他实际上在外面做了热的工作，这是明智的，因为没有人知道 1930 年代的灌封化合物在里面有什么。

我们对变压器的清洁程度感到惊讶。我们想知道一种现代环氧树脂灌封化合物是否能轻松、干净地释放这种物质。变压器装在一个小纸箱里，这可能使它更容易移动。

盒子里面？正如预测的那样，两个电容器。修理很快，然后只是用一个新的纸箱、一些铆钉和新的索环把它装回去。
