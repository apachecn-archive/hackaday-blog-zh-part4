# 一个工具扭曲电线，串烤肉串

> 原文:[https://hack aday . com/2022/02/11/one-tool-twists-wires-and-skews-shish-kebabs/](https://hackaday.com/2022/02/11/one-tool-twists-wires-and-skewers-shish-kebabs/)

在准备镀锡和/或焊接时用手指拧绞电线几乎是制作电子组件的人的反射。但是，如果电线离你太近，你的手指绕不过去，或者你有相扑运动员的手指，那该怎么办呢？那么[DIYDSP] [有一个解决方案给你](https://www.youtube.com/watch?v=aMlYvR4xEk8)(见休息时间下面的视频)很容易用串在你厨房抽屉里的烤肉串串做出来。[DIYDSP]想要绞合如此密集的导线的原因是将 0.1 英寸长的 O.C .绞合带状电缆直接焊接到 PCB 引脚接头图案上。

方法很简单。在工厂切割的平端钻一个长孔，然后用埋头钻钻头钻一个圆锥形，以便将电线导入。[DIYDSP]发现 1/16 英寸(1.6 毫米)的钻头有点太大，无法夹住他正在使用的电线类型，最终选定了 0.6 毫米的钻头。如果你用的是更大的线，你应该试验一下以得到合适的尺寸，或者只是做几个不同直径的线，因为它们很容易制作——只要清楚地标记出来，这样你就不会不小心用它们在 BBQ 上烤羊肉串了。

最终得到的工具与手持式绕线工具的商业端没有什么不同，但工作原理不同，而且只是成本的一小部分。如果你用绞合线做了任何数量的互连布线，那么你应该看看这个视频，然后做几个这样的东西放在你的工具箱里。
