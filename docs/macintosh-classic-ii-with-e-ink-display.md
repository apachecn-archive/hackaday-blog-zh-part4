# 配备 E-Ink 显示屏的 Macintosh Classic II

> 原文：<https://hackaday.com/2022/08/24/macintosh-classic-ii-with-e-ink-display/>

随着各种古董计算机的老化，随着硬件开始出现物理故障，操作它们变得越来越困难。保持这些系统正常运行通常需要从其他机器上清理零件，随着时间的推移，这些零件越来越难找到。但是如果你抛开只使用适合时代的组件的要求，有一些有趣的方法可以用一些现代技术来复兴旧设备，[比如这款带有独特显示屏的 Mac Classic】。](https://java.util.properties/blog/paper-mac.html)

麦金塔经典二代是苹果销售的第一台价格低于 1000 美元的麦金塔电脑的继承者。因此，这台机器有一些较低的规格，如单色 512×342 显示器。这一款已经改装了电子墨水显示屏，实际上赋予了它与原版相同的灰度美感。电子墨水显示器由 Raspberry Pi 驱动，它显示复制系统 7 环境和一组照片。

虽然在这一点上，计算机的唯一原创部分是外壳，但该项目的创建者[Dave]也通过 Arduino 内置了对苹果桌面总线的支持，因此可以使用原始的苹果鼠标和键盘。虽然这很大程度上是一个工作的 Mac 经典的幻觉，但我们仍然欣赏这种美感。

如果你是一个典型的苹果纯粹主义者，那么[看看这个 SE/30，除了树莓派之外，它几乎完全使用了原始部件](https://hackaday.com/2018/12/04/yet-another-restomod-of-the-greatest-computer-ever/),以允许它与现代互联网通信。
