# 小姐弹球桌得到摇滚改头换面

> 原文:[https://hack aday . com/2018/07/13/mademoiselle-pinball-table-gets-rock-n-roll-改头换面/](https://hackaday.com/2018/07/13/mademoiselle-pinball-table-gets-rock-n-roll-makeover/)

从前，在布鲁克林的威廉斯堡，有一家音乐场所/艺术家集体/效果踏板公司帮助重新定义了工业。那个地方被音频称为死亡。2014 年，Vice Media 买下了 DBA 辛辛苦苦改造的大楼，这让它遭遇了中产阶级化的死亡。从灰烬中升起的死亡由音频街机，展示了由独立艺术家 DIY 弹球柜。

他们最近的作品叫做乱葬岗乐队(APTBS)。它建立在 1959 年的 Gottlieb Mademoiselle 桌子上，以当地同名的噪音/鞋子乐队为主题，通过音频与死亡紧密相连。根据[马克·克莱布]的说法，这张桌子是对 APTBS 那种完全超负荷的弹球式表演风格的致敬。当玩这张桌子时，几乎没有什么感觉可以幸免，它有闪光灯，黑光灯，APTBS 的视频和音频剪辑，以及一台造雾机。是啊。

[Mark]从一个朋友那里得到了这个项目，他已经切断了一些线路并开始在上面进行黑客攻击。几乎桌子的每一部分都必须用原始设备制造商的零件升级，否则就要完全更换。现在有一个小孩跑保险杠，另一个小孩在开关上。一个 Arduino 驱动照亮操场的新像素条，另一个 Uno 在这些可爱的 VFD 管上显示比分。所有四个 micros 都用 Python 和一个 Raspi 3 捆绑在一起。

如果你住在纽约附近，你可以在 7 月 15 日亲自去红色泊松酒店玩夜光球。如果没有，不要翻转-只是轻推一下，看看她的行动。我们提到过有闪光灯吗？你被警告了。
