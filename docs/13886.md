# 用柔性密封件制作投影仪屏幕还行，算是吧

> 原文:[https://hackaday . com/2022/06/10/making-a-projector-screen-out-of-flex-seal-works-okay-kinda/](https://hackaday.com/2022/06/10/making-a-projector-screen-out-of-flex-seal-works-okay-kinda/)

在大屏幕上看电影很有趣，但去电影院或汽车电影院可能会很麻烦。只要有点创意，在家里也能获得同样的体验，正如[The Hook Up]制作的这个 DIY 投影屏幕所示。

建筑始于一个为庭院设计的巨型电动卷帘。它的得分很低，因为它是从原来的家搬走后被打捞上来的。看到屏幕门在 Flex Seal 的帮助下变成了一艘船，[The Hook Up]有信心 flyscreen 可以被密封起来，用于投影。

很快，事情变得艰难起来。轻型应用程序并没有真正填补 flyscreen 中的漏洞，而厚型应用程序在运行方面有很大问题。最终，屏幕被涂上了 3 加仑的白色柔性封条，并悬挂起来进行测试。

运行造成了问题，因为凹凸不平的屏幕纹理在观看电影时会分散注意力。此外，光滑的表面会产生难看的反光。经过反复试验后，问题得到了解决，用砂纸将柔性密封表面打磨平整，并使用哑光透明喷漆使光泽变暗。

结果是一个巨大的投影屏幕，一按按钮就可以滚下来，这种情况[我们以前见过，](https://hackaday.com/2014/03/31/magic-screwdriver-decides-if-you-watch-tv-or-not/)尽管成本很高。[The Hook Up]欣然承认，投入的几百美元本可以更好地用于[购买一个预制屏幕](https://twitter.com/TheHookUp1/status/1534874790565126144)。尽管如此，这是一个很酷的项目，我们尊重创作者投入的工作！休息后的视频。
