# 万圣节南瓜用邪恶的眼睛吓唬人

> 原文：<https://hackaday.com/2020/11/15/halloween-pumpkin-scares-with-an-evil-eye/>

如今，一盏经典的南瓜灯已经不够用了。孩子们预计至少会被高质量的电子动画吓到。由[零到无限]创作的这部令人难忘的作品可能会达到这个目的。

一个真正的南瓜被压入这个建筑中，带着通常的威胁性笑容和烛光内部。然而，它的不同之处在于它的单个动画眼睛。眼睛本身是由一个乒乓球构成的，用马克笔在上面画出令人毛骨悚然的血丝。在 Arduino Nano 的指挥下，一对伺服系统允许眼睛抽动和转动。为了进一步互动，超声波传感器被用于只有当它感应到有人接近时才触发南瓜。

这是一个有趣的假期建设，也是一个伟大的入门如何与伺服和微控制器。我们可以想象一个更先进的设置，使用更多的传感器和南瓜来训练对不知情的游客的多只眼睛。如果这还不够可怕，也许可以让你的南瓜喷火来代替。休息后的视频。
