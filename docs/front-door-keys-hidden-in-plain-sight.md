# 前门钥匙藏在显眼的地方

> 原文：<https://hackaday.com/2022/10/18/front-door-keys-hidden-in-plain-sight/>

如果说管理一堆密钥有什么问题的话，不管它们是用于 RSA、SSH 还是汽车，那就是大量的密钥会很麻烦。事实上，任何让生活变得更简单的东西都是我们经常看到的构建于之上的项目的概念，键也不例外。例如，这个项目通过将钥匙藏在一件珠宝中，消除了有意识地随身携带钥匙的需要。

这个项目源于[Maxime]之前的项目，允许用智能手机或平板电脑解锁前门。这并不比带钥匙好多少，因为贵重的电子产品必须随身携带。相反，这款产品避开了智能手机，而是采用了一个可以佩戴的戒指，只需挥挥手就可以打开门锁。该戒指包含一个 RFID，它由一个由 Wemos D1 迷你监控的天线读取。当它看到戒指时，一组伺服系统就会打开门。

整个装置安装在门的前面，大约在窥视孔通常所在的位置，机械致动器在里面。它似乎和随身携带金属钥匙一样安全(如果不是更安全的话)，我们也欣赏以这种方式展示电路板的美感，而不是藏在外壳里。这是一个有趣的建筑，让我们想起了其他一些独特的开门方式。
