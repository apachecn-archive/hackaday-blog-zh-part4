# 氛围灯像水一样荡漾

> 原文:[https://hack aday . com/2020/12/31/氛围灯-涟漪-似水/](https://hackaday.com/2020/12/31/ambience-lamp-ripples-like-water/)

在人类忍受了一年之后，我们都可以在生活中多放松一点。[这款氛围灯正是为工作、学习或闲逛营造轻松氛围的好东西](https://www.instructables.com/Water-Surface/)。只要触摸表面，光线就会荡漾开来，就像触摸时在静止的水面上形成的同心圆一样。当光线稳定下来时，它看起来就像一个准备好进行夜间游泳的诱人的游泳池。

里面真的没有任何惊喜——灯是通过触摸顶部的中心通过 capsense 操作的。三个 NeoPixel 环和一个 RGB LED 条提供照明，Arduino UNO 运行该节目。[Qttting_F]使用了一个便宜的陶瓷碗和一块丙烯酸树脂作为盖子，但这也可以很容易地用白色 PLA 或其他材料印刷。休息之后，请查看实际情况。

氛围很好，但有时你需要更实用的东西。[那些类型的灯也可以打印](https://hackaday.com/2020/11/04/3d-printed-workshop-lamp-uses-a-few-neat-tricks/)。
