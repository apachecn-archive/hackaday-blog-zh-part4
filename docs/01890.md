# 开源碳纤维杆端

> 原文：<https://hackaday.com/2019/01/22/open-sourced-carbon-fiber-rod-ends/>

已经在这个街区生活了几十年的模型制作者和制造者通常都有他们喜欢的材料。轻木、金属片、铜管……这些都是业余爱好者工作室的主要产品。复合材料在很大程度上是一个新生事物，并开始在业余爱好市场获得更多的立足点。[【Anthony】一直在做这方面的实验，并且已经为碳纤维管材创造了一些有用的附件。](https://www.youtube.com/watch?v=62OFivZV69c&feature=youtu.be)

这些配件是由铝激光切割或 3D 打印而成。杆端是一个简单的两件式设计，在插入碳纤维杆之前，将两个设计放在一起。[Anthony]展示了一系列用作步进电机连杆的杆，然后对连杆进行拔出测试。使用氰基丙烯酸酯胶安装，该链节可承受超过 180 磅的拉力。实力令人印象深刻，[安东尼]还谈到了如何安装适当的轴承，以使用运动项目的链接。

总的来说，这些链接可能会对任何在构建中使用碳纤维棒的人有用，而且有帮助的是，所需的文件都可以在 GitHub 上获得。这种原材料现在很便宜，在网上很容易买到，如果使用得当，它会很结实，很有弹性。[我们已经看到碳纤维最近也出现在更多的项目中。](https://hackaday.com/2016/05/15/super-strong-3d-component-carbon-fiber-parts/)休息后的视频。
