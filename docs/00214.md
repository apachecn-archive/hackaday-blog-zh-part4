# 建造终结者的手臂

> 原文：<https://hackaday.com/2018/07/31/building-the-terminators-arm/>

Cyberdyne 系统公司的 800 系列终结者是一个非常能干的机器人，碰巧看起来非常像阿诺德·施瓦辛格。它拥有先进的金属内骨骼，多年来一直是许多 DIY 道具制作的灵感来源。[[KenToonz]已经决定接手这样一个项目，并邀请观众一同参与。](https://www.youtube.com/watch?v=2mdcc5FaPhk) (YouTube，下面嵌入。)

这个项目特别有趣，因为它涉及到一个模仿人类手和手臂的机器人的再造。值得庆幸的是，由于敬业的人们的努力工作，原版电影的蓝图已经可以在网上找到了。这些功能本身并不完善，但为有抱负的建造者提供了一个强大的基础来获得正确的外观和感觉，同时为那些希望建造以预期方式移动和操作的东西的人留下了修改的空间。[KenToonz]打算做到这一点，并设想安装各种弹簧和其他机制，使关节能够适当地伸展和收缩。

[KenToonz]从手指开始，向后朝着前臂工作，然后开始添加各种填隙片，使其作为一个组件一起工作。所涉及的加工涵盖了从手指的小金属件到为活动部件生产定制弹簧的一切。一旦全部完成，我们迫不及待地想看成品！

我们以前也看到过一些很棒的道具——像这个三录相机配得上斯波克先生。
