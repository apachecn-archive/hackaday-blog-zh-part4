# BINA-VIEW:迷人的机械干涉显示器

> 原文：<https://hackaday.com/2020/04/07/bina-view-a-fascinating-mechanical-interference-display/>

[Fran Blanche]在下面的[视频拆除](https://www.youtube.com/watch?v=yFV4AOjrdO0)中拆除了这个迷人的展示。

这些显示器可以支持买方选择的多达 64 个字符，由 6 位控制，令人惊讶的是，每位仅需要 128 mW 来控制；对于它的时代来说，这是一盏相当亮的灯。除了字母数字组合，显示器还支持“色板”，我们发现这很有趣。这种全装修的模型按今天的价格计算每台只需 1206 美元，或者按最新的市场价计算五卷卫生纸。这只是一个数字。

如果你仔细阅读链接到这里的文件，并观看她的视频，你就能了解这个展示是如何运作的。有六个螺线管连接到装置后部的杆上。灯通过透镜照射到板组件的背面。每块板都是一个精心设计的穿孔网格。当螺线管激活时，选定的板倾斜，干扰固定的栅格。这导致光仅在某些区域被阻挡。

这似乎很清楚为什么从来没有起飞。与翻转显示器和谢妮管相比，排列这些看起来像是一场生产噩梦。尽管如此，这些角色对他们来说还是很有魅力的。我们不介意看到这种显示器的 3D 打印/激光切割版本。开始工作！
