# 你现在可以制作自己的发光二极管 D20(有多达 2400 个发光二极管)

> 原文:[https://hack aday . com/2021/04/03/you-can-buy-your-own-build-your-own-Luming-led-d20-with a whopping-2400-LEDs/](https://hackaday.com/2021/04/03/you-can-now-build-your-own-glowing-led-d20-with-a-whopping-2400-leds/)

D20，或称 20 面骰子，最常见的形状是正二十面体。这是一个奇妙迷人的几何图形，多年来，它在无数桌面角色扮演游戏中保持着命运的平衡。正是这种神圣的几何图形[Greg Davill]选择用发光的 RGB LEDs 的荣耀来祝福。现在， [[Greg]已经共享了这些文件，因此您可以创建自己的文件。](https://github.com/gregdavill/d20-hardware)

[![](../Images/8fb8b98aa86356f5b97a7701c07873e7.png)](https://hackaday.com/wp-content/uploads/2021/03/d20-600.jpg)[D20](https://gregdavill.com/blog/d20)的开发博客是一个很好的阅读材料，它强调了创造这样一个发光如此明亮的紧凑项目的挑战。该设计使用了 2400 个 1.5 毫米 x 1.5 毫米的 led，采用了老式的 RGB 风格，在 20 个侧面之间均匀分布。没错，这里没有花哨的自寻址智能 LED——每个 LED 都是由[Greg]的硬件直接手动控制的。SAMD51 和 ICE40UP5K FPGA 用于运行显示器。每个面板都被固定在一个几乎没有的 3D 打印框架中，用带状电缆连接在一起，以保持紧凑。索尼相机电池安装在框架的狭小空间内，以提供必要的电力。

我们第一次报道这个项目是在去年年底，现在很高兴看到它以一种易于复制的形式出现。然而，这种电路板的组装并不适合胆小的人，因为有大量的 SMD 零件需要处理。我们怀疑这只是正在进行的 LED 闪光灯军备竞赛中的又一次齐射，我们迫不及待地想知道[Greg]和社区的其他人接下来会推出什么。如果你有新的发光发热的线索，[让我们知道](http://hackaday.com/submit-a-tip)。休息后的视频。
