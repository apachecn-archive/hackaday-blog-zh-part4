# 蓝牙开发板持续发展

> 原文：<https://hackaday.com/2020/06/15/bluetooth-development-board-goes-the-distance/>

你有没有遇到过一个有趣的芯片或元件，你想用它做实验，却发现似乎没有开发板？如今，制作自己的电路板比过去容易多了，但如果只是为了个人使用，还是有点麻烦。这就是为什么[Nikolaj Andersson Nielsen]决定向社区发布他的定制远程蓝牙开发板 RF cat。

[![](img/31f22607e3fe5bc88114cd65473a344e.png)](https://hackaday.com/wp-content/uploads/2020/06/rfcat_detail.png) 该板基于 MeshTek 的一个模块，实质上是 Nordic nRF52832 的放大版本。根据[Nikolaj]的说法，这使该模块的发射功率是基本型号芯片的 30 倍。

RFCat 与 Arduino IDE 兼容，并使用 Adafruit nRF52 引导加载程序，从而可以轻松编写自己的代码来利用所有这些新发现的功能。主要是通过 USB-C 对电路板进行编程，但[它也支持串行线调试(SWD)](https://hackaday.com/2020/02/11/xbox-controller-provides-intro-to-swd-hacking/) 和空中更新，后者可以通过设备上的物理按钮来触发。

如果你想拥有自己的 RFCat，[现在 Tindie 上就有了](https://www.tindie.com/products/rfcat/rfcat-n32-long-range-bluetooth-development-board)。放大模块最初是为了建立蓝牙网状网络，但我们相信还有其他有趣的应用等待着被发现。
