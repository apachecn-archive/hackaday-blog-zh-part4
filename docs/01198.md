# Kinect Visualizer 演示让 Winamp 大受欢迎

> 原文：<https://hackaday.com/2018/11/11/kinect-visualizer-demo-gives-winamp-a-run-for-its-money/>

Winamp 吃定了，因为多亏了[Samarth]手中的微软 Kinect，有了一种让你的屏幕跟着你跳舞的新方法。他创建了一个音乐可视化演示，通过输出有趣的像素化轮廓和变色闪光灯，利用 Kinect 上的 3D 深度摄像头。当有大的高帽击中或低音重击时，相机馈送相应地反应(像任何好的可视化器会做的那样)。他甚至上传了他的项目代码，以防有人想看看。

可视化工具利用了 OpenKinect 处理库,该库为许多其他类似的 Kinect 艺术项目提供了基础。它是专门为编码人员提供一种更快捷的方式来访问 Kinect 输出的原始颜色和深度数据。它的创建者丹尼尔·希夫曼已经发布了一些教程，以帮助任何想要创建自己的实时动画的人。

可视化演示(见下面的视频)是作为周末举行的[海德拉巴 Maker Faire 的一部分创建的。该博览会是该市的第一个创客博览会，将展出 200 多个不同学科的创客展品。从地理上来说，在离你最近的社区之外看到创客社区总是很棒，所以希望我们将来会看到更多像[Samarth]这样的人参加更多的创客活动。](https://news.webindia123.com/news/articles/India/20181102/3466870.html)
