# 用光电倍增管进行单光子探测

> 原文:[https://hack aday . com/2022/12/20/单光子光电倍增管检测/](https://hackaday.com/2022/12/20/single-photon-detection-with-photomultipliers/)

除非你是一个音响发烧友，否则你可能会认为电子管主要是属于使用旧技术的人的。然而，与更现代的传感器相比，光电倍增管仍然是有用的，并且[Jaynes Network]已经[研究了它们如何工作](https://www.youtube.com/watch?v=cd79PTRwLM0)，尤其是与闪烁探测器一起工作。

他检查的 RCA 光电倍增管有十个阶段，甚至可以探测到一个光子。结合闪烁探测器，它们是很好的辐射探测器。

当我们听到有人明显爱上了这样一根管子背后的工程，我们不禁会心一笑。我们明白了。管的内部很拥挤，所以很难识别倍增电极和其他部分，但一些图表很容易看出管是如何工作的。

考虑到它的年龄，我们对它的文档看起来很好印象深刻。我们指的是它所处的状态。这份文件本身显然是一份带有手绘图形的打字文件的复制品。

我们希望有一些地铁运行的镜头，但我们必须等待未来的视频。不过，我们打赌这一天会到来。虽然有一些固态探测器，但它们并不适用于所有应用。然而，曾经有一段时间，电子管被广泛应用，包括 X 射线扫描仪和 T2 摄影设备。
