# 在不良法律开始之前阻止它们

> 原文:[https://hack aday . com/2020/07/25/stop-bad-laws-before-they-start/](https://hackaday.com/2020/07/25/stop-bad-laws-before-they-start/)

随着今年夏天发生的一切，你可能会被原谅没有跟上新提议的监管框架，但如果你对软件定义无线电(SDR)甚至是更新你的 WiFi 路由器感兴趣，你应该这样做。现在，在欧洲委员会之前，有一个提案从根本上阻止你将自己的固件/软件更新到任何带有收音机的产品上。这显然对欧洲人很重要，但由于制造商通常按照最严格的全球要求制造硬件，这可能会影响到每个人。什么算无线电设备？从 WiFi 路由器到可穿戴设备，从 SDR 加密狗到短波收音机，无所不包。

这个想法是为了防止流氓可重新配置的无线电互相交谈，并防止消费者阻止他们的路由器和无线电。在 SDR 成为标准、固件为王之前，监管机构很容易测试一些硬件并确保其合规，但现在任何人都可以刷新固件，他们如何确保无线电合规呢？自然会阻止用户运行他们自己的固件。Hackaday 很难支持这种方法。

[影响评估](https://ec.europa.eu/info/law/better-regulation/)听起来更像是提议裁决的广告文案，而不是诚实的评估，但是你应该读一读，因为它让你知道委员会从何而来。令人欣慰的是，他们明确提到开源软件开发是一种值得保护的商品，但他们的“可能的社会影响”包括“增加的安全性和安全性”，他们的结论是没有负面的环境影响。当制造商不再支持该设备时，您会怎么做？我有很多设备不再受固件更新的支持，因为开源固件，这些设备更加安全，而且根本不在垃圾填埋场。

类似地，“欧盟自主保护其产品的能力增强也可能有助于公民更好地保护他们的信息相关权利”来自一个奇怪的世界，在那里你可以[相信小米的家庭自动化固件不会给家里打电话](https://hackaday.com/2019/10/24/reverse-engineering-xiaomi-iot-firmware/)，但不能相信开源替代品。

公众意见仍然开放，不仅限于欧洲公民。如上所述，即使你不在欧盟，这也可能会影响到你，所以请畅所欲言。你有到九月的时间，如果你投诉的话，你将会在一些很棒的公司里。事实上，通读公众评论是相当振奋人心的:大学、研究人员和黑客都提出了避开拟议方法的理由。我们希望委员会听到我们的声音。
