# 由于一个快速的泡沫球，看一个 3D 显示

> 原文：<https://hackaday.com/2019/11/19/behold-a-3d-display-thanks-to-a-speeding-foam-ball/>

我们已经看到以各种不同方式尝试的 3D 图像投影，但这对我们来说是一种新的方式。苏塞克斯大学互动实验室的立体显示器通过将光线投射到一个微小的泡沫球上来创建 3D 图像，这个泡沫球在空气中快速移动，足以创建视觉暂留效果。(视频，嵌在下面。)这是怎么实现的？有一大组超声波传感器，执行研究人员所说的“声音捕捉”。

这与[声悬浮装置](https://hackaday.com/2018/11/28/72-tranducers-for-acoustic-levitation/)背后的原理相同，这种装置展示了轻质物体(如微小的聚苯乙烯泡沫球)如何克服重力。但是这种 3D 显示器不仅能够在 3D 空间中移动物体，而且能够以足够高的速度和足够的控制来移动物体，以产生视觉效果的持续性。[他们(尚未发布)论文的摘要](http://sro.sussex.ac.uk/id/eprint/86930/)声称被捕获的球可以以每秒几米的速度移动。

它还有其他一些锦囊妙计。该阵列能够同时产生声音，并且通过让用户触摸由换能器产生的高气压和低气压区域来提供有限形式的触觉反馈。当然，这些区域不可能是被高速球占据的区域，但这是一个巧妙的技巧。请观看下面的视频进行演示。
