# 欢迎来到 Ziptie 城

> 原文：<https://hackaday.com/2022/08/05/welcome-to-ziptie-city/>

重力、磁力、弱核力和强核力；当然他们把很多东西放在一起。但是另一种自然的约束力需要你最大的尊重？拉链。

或者至少这是[艾伦·赖纳]的[对电缆扎带和 3D 打印的结合](https://www.youtube.com/watch?v=qgt9P6jsyaY)。(视频，嵌在下面。)所以他设计了一个展示模型，[拉链城](https://www.printables.com/model/248291-zip-tie-city)，它展现了拉链渠道设计提供的许多可能性。

然后，他设计了一个多输出电池盒，带有大量的拉链固定器，用于检查和保持模块和电池的布线。(释放出一些神奇的烟雾，但即使是拉链也无法将这些东西藏在里面。)

我们喜欢在螺丝太多或者简单的解决方案就足够的情况下使用拉链。为拉链带设计通道是一个纯粹的胜利，并利用了 3D 打印的独特优势；试图用减法加工成一个零件是很可怕的。

然而，拉链并不是镇上唯一的游戏。如果你需要胶水，我们的[汤姆·纳尔迪]去了他当地的五金店，测试了这本必读经典作品中的选项。

感谢[Keith Olson]的建议，他想让你在心情好的时候看看[拉链男](https://www.youtube.com/watch?v=uTMr584t-nI)(视频)。
