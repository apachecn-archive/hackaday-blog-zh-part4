# 1903 年的一个小把戏让一台老式单色电视焕发了色彩

> 原文:[https://hack aday . com/2020/07/28/trick-from-1903-make-a-old-monochrome-TV-spit-some-colors/](https://hackaday.com/2020/07/28/trick-from-1903-makes-an-old-monochrome-tv-spit-some-colours/)

可以肯定地说，如今彩色电视被认为是理所当然的。被色彩动态范围、色彩空间准确性和深度等现代营销术语所消耗，当简单地用最轻微的色彩暗示重建图像需要一些严肃的独创性并赢得你应得的赞扬时，很容易忽略图像复制的卑微开端！

重温一项古老的技术，这项技术于 1903 年首次获得专利，并成功地使一台老式黑白电视产生了彩色图像。本质上，这个想法实际上类似于今天仍然使用的廉价图像传感器和液晶显示器。我们没有像 AMOLED 那样通过单独集成颜色源来产生真正的 RGB 颜色，而是采取了一种更简单的方法:产生一个更简单的单色图像，其中每个颜色像素由四个单色子像素物理表示，每个颜色分量对应一个子像素。现在，根据图像的颜色信息点亮每个子像素，并依靠外部[滤色器阵列](https://en.wikipedia.org/wiki/Color_filter_array)来组合并吐出正确的颜色。

他首先使用一些图像处理将标准的彩色视频转换成上述的单色子像素表示。接下来，使用喷墨打印机在一些醋酸纤维片上打印 Bayer 滤色器阵列(最初的发明者使用马铃薯淀粉！)，当覆盖在单色显示器上时，会神奇地产生彩色输出。

这种技术存在一些问题，主要是难以测量电视像素的尺寸，然后难以制作并完美地对准滤片。你应该看看[anfractuosus]是如何解决这些问题的。
