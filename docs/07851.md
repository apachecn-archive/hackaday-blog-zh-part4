# 狄莺高端相机臂

> 原文:[https://hackaday . com/2020/09/27/狄莺-a-高端-摄像头-arm/](https://hackaday.com/2020/09/27/diying-a-high-end-camera-arm/)

任何想要拍摄视频的人首先要买的东西之一应该是三脚架。关键是获得干净和稳定的拍摄，但他们可以限制他们的运动范围。想要更高端一点的东西，但对商业选择的高成本不满，[【亚历山大·查佩尔】决定建造自己的相机臂。](https://www.youtube.com/watch?v=V9e9ZWflCQU&feature=youtu.be)

建造基于方形铝管，高耐受性材料充当手臂的垂直和水平轨道。3D 打印的支架和适配器用于将所有东西栓在一起，还有几个打印的组件用作钻孔指南，以帮助精确加工铝管。调整内置于沿轨道行进的托架中，以帮助解决 3D 打印零件中的任何倾斜。然后安装配重系统，以确保摄像机不在锁定位置时不会撞到地板。

这是一个整洁的建筑，这给了亚历山大比他现有的三脚架更大的灵活性。此外，调整摄像机位置比以前快得多。当然，当你建造自己的钻塔时，天空是无限的。休息后的视频。
