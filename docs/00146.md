# 柔性 PCB 成为致动器

> 原文:[https://hack aday . com/2018/07/29/flexible-PCB-becoms-the-actuator/](https://hackaday.com/2018/07/29/flexible-pcb-becomes-the-actuator/)

电磁线圈枪采用一排电磁体共同工作，形成一个移动的电磁场。这些电场加速了一个项目，你可以用电来移动物质，通常是以惊人的速度。

[Carl Bugeja]利用他的[柔性 PCB 致动器](https://hackaday.io/project/159936-flexible-pcb-actuator)在某种意义上颠覆了这个想法。现在，一排电磁铁是运动的部分，磁性物体是静止的部分。在经典的线圈枪配置中仍然有一系列扁平的 PCB 电感，但正如标题所示，是在柔性基板上。

结果是一种奇怪的有机运动，让人想起一些蜥蜴、毛虫或蛇。它可以在磁铁上循环移动，或者在磁铁上方的空气中弯曲。这是一个新颖的活动部分，他给我们看了一个视频，我们把它放在了休息时间下面。

他计划将其用于某种形式的机器人，尽管它肯定有希望，但我们很想知道它能产生什么样的力，以及柔性 PCB 是否足够坚固，可以重复操作。我们向他致敬，他提出了一个简单的想法，并如此有效地证明了这个概念。
