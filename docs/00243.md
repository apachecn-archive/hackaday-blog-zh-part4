# 快速构建的 PowerWall

> 原文:[https://hackaday.com/2018/08/11/the-quick-build-powerwall/](https://hackaday.com/2018/08/11/the-quick-build-powerwall/)

埃隆·马斯克不仅仅是最伟大的人——他还是一位伟大的发明家。他发明了可重复使用的火箭、电动汽车等等。虽然这些都是了不起的成就，但埃隆最伟大的发明可能是 PowerWall。PowerWall 的想法很简单，并且已经存在多年了:只需要一堆电池，为你的房子建造一个巨大的 UPS。然而，埃隆把它带到了最前沿，世界各地的 DIY 者都在建造他们自己的。谢谢你埃隆。

当然，虽然构建自己的 PowerWall 的想法很简单，但细节是关键。你打算怎么买那些电池？你将如何把它们连接在一起？你如何把它连接到你的保险丝盒上？这是一个系统集成的噩梦，如果你做错了什么，锂电池可能会着火，这使得它变得更加困难。[jejugarcia]正在建造自己的 PowerWall，他可能会想到一个有趣的解决方案。他建立了一个模块化系统来储存和充电数百个 18650 电池。它看起来很棒，这可能是任何想要构建自己的 PowerWall 的人的答案。

除了获取数百个 18650 单元之外，构建 PowerWall 的最大问题是简单地将所有单元连接在一起。这可以通过 3D 打印的电池支架、焊料和汇流条来实现，一些人尝试将电线直接点焊到电池上。这个项目可能是一个更好的解决方案:它使用从你最喜欢的中国零售商那里容易获得的标准塑料电池架和一个 PCB 来将电池变成电池。

该电池模块的设计包括一个具有足够宽走线的 PCB、一个 XT60 电源连接器和几个用于充电器平衡连接器的接头。这是一个七单元设置，与传统方式制作 PowerWall 所需的数百小时相比，这些模块可以很快组装起来。

对这些模块的测试显示没有爆炸，一切都如预期的那样工作。然而，有一个问题:当汲取高负载时，这些廉价电池连接器的端子达到 150。这使得这些模块不适合像电动自行车这样的高负载应用，但如果你把数百个这样的模块放在一起为你的房子供电，应该没问题。不过，投资一些冷却设备可能是个好主意。
