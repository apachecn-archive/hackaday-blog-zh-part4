# 有了陀螺，谁还需要四个轮子？

> 原文：<https://hackaday.com/2019/09/06/who-needs-four-wheels-when-youve-got-a-gyro/>

你的普通车通常有四个轮子，后备箱里还有一个备用的。这是一个经过深思熟虑的数字，很少有车辆偏离常规。然而，这并不意味着没有其他可能性，[和【RCLifeOn】决定以这种方式进行实验。](https://www.youtube.com/watch?v=Rlnexcft2ns)

其结果是一个陀螺稳定的两轮遥控汽车，或者我们可以说，一种摩托车。无刷电机驱动后轮，而转向由控制前轮的伺服系统控制。一个大的旋转圆盘在车辆中心充当陀螺仪，它都被包装在一个简单的 3D 打印框架中。

结果令人印象深刻，陀螺仪对车辆的性能产生了明显的影响。虽然它可以在没有陀螺仪的情况下驱动，但它需要不断的转向校正以保持直立。随着陀螺旋转起来，它骑起来更像自行车，几乎没有稳定性问题。

这是一个有趣的项目，也是了解陀螺稳定性的好方法。当然，关于这个话题也有很棒的入门读物。休息后的视频。
