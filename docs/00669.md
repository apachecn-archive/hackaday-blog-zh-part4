# 黑客电动牙刷轻松破解锁

> 原文:[https://hack aday . com/2018/09/17/hacked-电动牙刷-击败-轻松锁/](https://hackaday.com/2018/09/17/hacked-electric-toothbrush-defeats-locks-with-ease/)

电影版的《开锁》倾向于强调这门手艺细致、精致的形象。英雄或恶棍拿出一个装有精美工具的细长钱包，熟练而精确地使用它们，很快就打开了锁。他们让这看起来很容易，虽然这种形象离现实不远，但还有其他方式撬锁。

[这个便利的电动牙刷开锁工具](https://www.youtube.com/watch?v=CEdJH2tBDfo)是一个更加暴力的开锁方法的惊人有效的例子。正如[Jolly Peanut]解释的那样，弹子门锁的工作原理是将每个弹子与锁芯的剪切线对齐，这使得锁可以转动。这可以用拨片一次一个销子来完成，或者通过振动销子来完成，直到它们随机地与剪切线排成一条线，刚好足够让锁转动。锁匠可能会使用专门制造的工具来完成这项工作，但一个简单的电池供电的电动牙刷也可以在必要时使用。[Jolly Peanut]移除了刷子通常的商业端，露出了高频振动的金属驱动杆。通过一点研磨，杆变细，以适合锁的键槽，并且通过施加一点扭矩，振动足以将销弹出到正确的位置。他在下面视频中的几把锁上试了试，每次只需要几秒钟。

当然，这种暴力方法有其缺点。它们并不微妙，它们产生的噪音可能会吸引不必要的注意。在这种情况下，用一把巨大的 3D 打印透明锁磨练你的手工开锁技能。
