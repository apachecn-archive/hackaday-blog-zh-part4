# 定制 40% F 型键盘 100%牛逼

> 原文:[https://hackaday . com/2022/06/16/custom-40-model-f-keyboard-is-100-awesome/](https://hackaday.com/2022/06/16/custom-40-model-f-keyboard-is-100-awesome/)

仔细看这个美女。不，那不是 IBM 的 M 型或者其他什么——它是一个定制的 40%电容弯曲弹簧键盘，由 durken 设计。很容易想象另一种现实，IBM 仍然以 IBM 的身份存在，并拥有强大的键盘游戏，或者 Unicomp 正在为那些不需要 101 或 104 键的人实现梦想。

[![](../Images/d4a12431a8f66446b50594e3b5c1019e.png)](https://hackaday.com/wp-content/uploads/2022/06/40-model-d-innner.jpeg) 屈曲现在怎么样了？这个可爱的板子用的是一个 IBM 老型号 f 的电容式[扣合弹簧开关](https://deskthority.net/wiki/IBM_buckling_spring)，基本上你每按一个键，就有一个小弹簧以连接的名义弯过来(或者扣上)。在电容式版本中，弹簧将锤子推至一对极板上，导致电容发生变化，从而被识别为按键。在这种情况下，这些按键由 TH-XWhatsit 控制器读取。

以 F XT 的 PCB 为例，[durken]在一个 PCB 上制作了一个电容焊盘场，并制作了第二个接地层 PCB 以避免干扰。为了真正向这些键盘致敬，[durken]决定稍微弯曲 PCB，这自然使几乎所有东西都变得复杂，尤其是桶板。

解决方案是制作一个单独的桶板，它可以滑入外壳，并通过安装支架拧到顶部。更有趣的是，[durken]在 IBM 标识下安装了一个 SKCL 锁开关，可以启用螺线管模式。请务必查看(更新！)休息后的视频。

弯曲弹簧键盘的最大好处之一是每个键听起来都略有不同。除非使用多个螺线管，否则螺线管模式[不会如此。](https://hackaday.com/2020/05/05/the-clickiest-keyboard-ever/)
