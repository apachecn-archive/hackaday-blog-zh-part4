# 遥控雪地车在冰上留下痕迹

> 原文:[https://hack aday . com/2022/02/18/RC-snowmobile-makes-tracks-on-ice/](https://hackaday.com/2022/02/18/rc-snowmobile-makes-tracks-on-ice/)

随着所有的未来技术目前在我们的支配下，似乎有点奇怪的是，大多数客车基本上是一样的东西，因为他们是一个世纪前。四个轮子，一个马达，和一些座位似乎是一个难以超越的公式。但在 3D 打印世界里，快速原型是游戏的名字，一些独特的车辆设计已经推出，特别是在钢筋混凝土世界。最近的一个来自[RCLifeOn] [的单轮遥控雪地车](https://www.youtube.com/watch?v=aaQxQ7DdVtE)向我们展示。

虽然不是传统的带轨道的雪地车，但这款车确实有一些相似之处。它的背面有一个驱动轮，印有 TPR 以提高灵活性，它还包括沿其整个圆周的螺柱，以使其在冰上具有更好的牵引力。前面有由旧溜冰鞋制成的转轮，车辆用于转向，它与一个 RC 控制器和一些锂电池绑在一起，以处理转向和驱动电机。

在这款车的第一次迭代中有一些设计缺陷，包括非常大的转弯半径，不必要的高扭矩齿轮装置，以及对于链传动来说过于灵活的框架。[RCLifeOn]也在一个湖中测试这个，这个湖看起来就要变成液态了，这就产生了一些有趣的视频片段，他用树枝和绳子找回了被困的车辆。总而言之，我们希望在未来进行第二次修订，届时一些问题将得到解决，这种独一无二的车辆可以真正穿越冰冻的荒原[与十年前的另一辆有趣的雪地车](https://hackaday.com/2013/02/15/pulse-jet-snowmobile-or-what-swedes-do-during-hibernation/)没有什么不同。
