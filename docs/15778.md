# 购物车为你做了繁琐的工作

> 原文:[https://hack aday . com/2022/12/29/shopping-cart-do-the-dragonary-work-for-you/](https://hackaday.com/2022/12/29/shopping-cart-does-the-tedious-work-for-you/)

由于现代微控制器，基本的家庭自动化任务，如开灯和关灯，打开百叶窗，以及其他各种简单的任务已经成为常见的 DIY 项目。但是随着人工智能和机器学习的出现，可以卸载到计算机上的任务数量急剧增加。这款自动消除杂货店结账队伍的购物车当然属于这一类。

该项目的灵感来自无收银员的亚马逊商店，顾客只需走进商店，拿走他们想要的东西，然后离开。这之所以成为可能，是因为电脑监控他们的购买行为并自动收费，但创造者[kutluhan_aktar]希望探索一种方法，在没有遍布商店的传感器和摄像头的情况下做到这一点。相反，通过将硬件安装到购物车上，传感器将与购物者一起旅行，并监控购物车中的内容，而不是从货架上拿走的内容。它是围绕 OpenMV Cam H7 构建的，OpenMV Cam H7 是一个微控制器，配有一个专门为这些类型的任务设计的摄像头，箱子内的定制电路还包括 WiFi 连接，以确保购物车可以正确报告其发现。

[kutluhan_aktar]还从头开始构建了整个软件堆栈，并在一组常见产品上训练该模型作为概念验证。这个想法是让较小的商店更有效地运营，而不需要一整套亚马逊硬件和软件来支持，这个原型似乎在这方面工作得很好。如果你想用更普通的硬件自己开发一个机器视觉项目，[看看这个项目，它用的是 Raspberry Pi 而不是](https://hackaday.com/2019/03/01/leigh-johnsons-guide-to-machine-vision-on-raspberry-pi/)。
