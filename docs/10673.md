# 3D 打印零件有助于快速制造电动滑板车

> 原文:[https://hack aday . com/2021/07/09/3d-printed-parts-make-for-a-quick-electric-scooter-build/](https://hackaday.com/2021/07/09/3d-printed-parts-make-for-a-quick-electric-scooter-build/)

有时候，如果散步成为你日常生活的一部分，即使是很短的距离也会变得很无聊。[亚历山大·查佩尔]发现自己正处于这样一个境地，所以他决定迅速制造一辆小型摩托车四处旅行。

这种构建很大程度上是“零件放在商店周围”的风格。由于一系列 3D 打印零件，旧滑板甲板上安装了漂亮的橡胶滑板车车轮和一套把手。不幸的是，第一次修订在滑板甲板的弯曲方面有问题，它不是为了承受一个成年人单腿站立的全部重量而设计的。另一个滑板甲板被投入使用，用金属管加固以增加强度。

从那里，[亚历山大]开始创建一个前轮驱动系统，使用电钻，几个轴延伸，和直角驱动。夹在车把管上，钻机的触发器由一根绳子连接的旋转节流阀控制。

这不是最容易骑的踏板车，站立起步时扭矩有点太大，有时操纵特性有点吓人。然而，我们确信通过一些练习和调整，[亚历山大]将会驾轻就熟。然而，如果你喜欢更狂野的东西，可以考虑这款代步车。休息后的视频。
