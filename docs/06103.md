# 经过调整的 PCB 打造终极便携式 N64

> 原文:[https://hack aday . com/2020/03/27/trimmed-PCB-makes-the-ultimate-portable-n64/](https://hackaday.com/2020/03/27/trimmed-pcb-makes-the-ultimate-portable-n64/)

我们在定制手持设备领域看到的最令人印象深刻的创新之一是使用“微调”PCB。这些是任天堂 Wii 和世嘉 Dreamcast 等流行视频游戏机的主板，实际上已经被缩小到更小的尺寸。可以想象，在系统停止运行之前找到可以切割的精确形状需要大量的研究和测试。但是如果你能完成它，一些真正不可思议的构建是可能的。

举个例子[这款由【GMan】](https://bitbuilt.net/forums/index.php?threads/n64-sp-finished.3282/#post-38367)打造的绝对不可思议的翻盖 N64。在将主板切割成手掌大小的尺寸后，他已经能够创建一个手持系统，只比控制台的原始墨盒大一点点。

[![](../Images/48d9baadc2b28c92ffc24d08a70622eb.png)](https://hackaday.com/wp-content/uploads/2020/03/n64sp_detail.jpg) 顺带一提的是那些原装的墨盒依然受支持，并且装进了系统后部的一个槽里 Game Boy style。对于扔进你的口袋来说，它仍然有点太大了，但是我们怀疑如果不借助仿真，你能造出一个更小的便携式 N64。

在休息后的视频中，[Gman]解释说，当发现系统的外围接口(PIF)芯片可以成功重新定位时，调整后的 N64s 才真正取得突破。由于该芯片位于 PCB 的外部边缘，能够移动它意味着电路板可以被切割得比以前更小。

但是在 3D 打印的外壳中不仅仅有一个被黑掉的 N64 主板。[Gman]还设计了一个定制的 PCB，处理 USB-C 电源传输，为手持设备的 4250 mAh 电池充电，[并通过 I2S 提供数字音频](https://hackaday.com/2019/04/18/all-you-need-to-know-about-i2s/)。这是一个非常专业的设置，如果你认为电路板是原始控制台的一部分，那也情有可原。

考虑到 N64 SP 的设计和制造是如此之好，发现这不是[Gman]第一次把这样的东西放在一起可能不会感到惊讶。去年，他用许多相同的技巧制作了同样令人印象深刻的便携式 Dreamcast。
