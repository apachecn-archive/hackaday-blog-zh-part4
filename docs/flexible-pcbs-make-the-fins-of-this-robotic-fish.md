# 这种机器鱼的鳍是由柔性多氯联苯制成的

> 原文：<https://hackaday.com/2018/09/14/flexible-pcbs-make-the-fins-of-this-robotic-fish/>

我们喜欢在这里有一点打破常规的想法，任何时候我们看到机器人不使用轮子和马达来移动，我们都会注意到。所以当[一个兜售使用软致动器鳍的机器鱼的项目](https://hackaday.io/project/160825-robot-fish-pcb-fins)越过了提示线时，我们不得不看一看。

原来这条机器鱼来自于[Carl Bugeja]丰富的思想，他的 [PCB 电机](http://hackaday.com/2018/01/24/this-tiny-motor-is-built-into-a-pcb/)和[柔性致动器](http://hackaday.com/2018/07/29/flexible-pcb-becomes-the-actuator/)之前已经在这里介绍过了。这些鱼鳍的基本概念来自后一个项目，该项目使用印刷在柔性 Kapton 基底两面的线圈。放置在磁铁附近，当电流通过时，致动器弯曲。下面的视频展示了两条原型机器鱼，每条都有四个鳍。第一个是嵌有磁铁的泡沫碎片；鱼鳍确实在拍打，但整个东西太重了。第二个版本轻得多，几乎可以工作了，但是驱动器的系绳太硬，不允许它真正弯曲它的鳍。

虽然看起来它有希望，但我们很兴奋地看着[卡尔]把它带到哪里。也许是成群的小机器鱼在巡逻寻找污染？
