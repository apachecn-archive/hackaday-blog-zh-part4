# 你能感觉到的时钟这个词

> 原文:[https://hack aday . com/2021/03/30/the-word-clock-you-can-feel/](https://hackaday.com/2021/03/30/the-word-clock-you-can-feel/)

到目前为止，几乎每个人都遇到过一个单词时钟项目，如果他们自己没有建立的话。看着时钟，以一种更人性化的方式而不仅仅是脸上的数字来看时间，这是一种吸引力。但是视觉之外还有感官。你听过单词钟吗？你有没有*感受过*一字钟？对于这些问题，Hackaday 自己的[Moritz Sivers]现在可以给出肯定的答案，因为他经历了极端的学习过程，参与设计和建造了[一个由磁铁驱动的触觉单词时钟](https://www.instructables.com/8x8-Haptic-Wordclock/)。

显示器的单个字母由定制印刷电路板上的磁线圈矩阵驱动。它们的工作方式与 LED 矩阵有些类似，只是它们产生的磁场可以推动或拉动磁铁，而不是产生光。因此，有各种不同的挑战需要解决:从线圈设计，到驱动增加的功耗，甚至考虑线圈如何与相邻线圈相互作用。受其他触觉显示器研究的启发，[Moritz]使用铁箔使磁铁锁定到位。这样，每个字母将停留在其向前或向后的位置，而无需给线圈供电来保持它在那里。此外，当附近的线圈被激活时，字母保持更稳定。

像单词钟这样的“无处不在”的项目的部分乐趣在于看到有创造力的黑客如何让他们自己的作品脱颖而出。无论是经典设计的缩小版[还是](https://hackaday.com/2019/03/27/leds-shine-through-pcb-on-this-tiny-word-clock/)[简单干净的东西](https://hackaday.com/2019/12/26/word-clock-does-the-job-with-laser-etched-acrylic/)，我们都喜欢看到它们。不出所料，[莫里茨]本人给我们留下了深刻的印象[他在过去对单词时钟的独特理解](https://hackaday.com/2019/04/02/a-word-clock-the-hard-way/)。(编者注:那跟[他的云庭](https://hackaday.com/2019/01/13/see-the-radioactive-world-with-this-peltier-cloud-chamber/)比起来不算什么！)
