# LED 矩阵成为有趣的俄罗斯方块时钟

> 原文：<https://hackaday.com/2019/06/27/led-matrix-becomes-fun-tetris-clock/>

有时一个项目的诞生仅仅是因为一些有趣的部分被搁置了太久。当然，这是一个和其他任何理由一样好的理由，并且经常会导致一些有趣的结果。[【Jorj Bauer】的俄罗斯方块展示就是这样一个项目](https://hackaday.io/project/166204-tetris-display)。

这个项目开始是因为[Jorj]有一个 8 x 32 WS2812 LED 阵列，是时候把它变成一个很酷的东西了。由此产生的显示器有几个特点，使它成为家中受欢迎的一件。它可以充当时钟，根据一天中的时间自动补偿夏令时和亮度控制。它还可以用作文本滚动条，当然，也是聚会的一部分——它可以玩俄罗斯方块。这一切都在一个 ESP-01 上运行，第二个设备充当遥控器来控制游戏。

不仅仅是另一个 LED 矩阵项目，[Jorj]还加入了一些小元素。开发了一种字体，允许时间以完全由俄罗斯方块(或俄罗斯方块)组成的像素字体显示。这使得时间可以通过从显示屏顶部掉落的碎片来显示。俄罗斯方块的实现也很可靠——实现了专业人士期望的正确的超级旋转系统。

[Jorj]报道说，这个建筑的灵感来自于这些页面中的早期俄罗斯方块时钟。这是一件整洁的作品，我们确信它是壁炉架的一大亮点。休息后的视频。
