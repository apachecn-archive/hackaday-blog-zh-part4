# LED 矩阵黑客聊天

> 原文：<https://hackaday.com/2021/06/07/led-matrix-hack-chat/>

加入我们太平洋时间 6 月 9 日星期三中午的 [LED Matrix Hack Chat](https://hackaday.io/event/179481-led-matrix-hack-chat) 与 [Garrett Mace](https://hackaday.io/hacker/192913-garrett-mace) ！

令人惊讶的是，发光二极管如此迅速地从物理实验室的好奇变成了数百万甚至数十亿的主流商品。这些年来，led 的一切都变得更好、更小、更便宜，在相对较短的时间内，从“你想要的任何颜色，只要它是红色”阶段发展到彩虹的所有颜色甚至更多。led 已经进入了不久前还不太可能出现的应用领域，如建筑照明、汽车应用，甚至是覆盖广告牌、建筑和体育场馆的大型彩色高分辨率显示屏。

正是后一种应用似乎为电子爱好者提供了一个福音，即廉价而丰富的 LED 矩阵模块。这些很容易在通常的地方获得，并且它们紧密包装的针尖可以以任何强度显示任何颜色，它们对黑客来说有很多有趣和有用的应用。但是你到底如何使用它们呢？通常情况下，电子端是非常简单的，但在计算如何处理所有这些 led 时涉及的一些数学问题可能有点令人费解。

为了帮助我们理清这一切，加勒特·梅斯会顺便来看看黑客聊天室。你可能见过加勒特的酷 LED 矩阵灯罩，它经历了大量的修改，是酷黑客中被大量复制的时尚配件。它们看起来很简单，但有一些技巧可以让它们正确工作，加勒特将分享他的秘密。带着你对 LED 矩阵模块工作的疑问，尤其是那些不符合简单笛卡尔坐标的奇数尺寸模块和奇怪排列。
