# EMOJO 聊天机器人会在那里等你

> 原文：<https://hackaday.com/2021/07/07/emojo-chatbot-will-be-there-for-you/>

我们有时都需要有人倾诉，而疫情只会让焦虑和抑郁患者的数量变得更糟。交换最简单的客套话可以让你再次感到完整，但面具让你很难与陌生人交流并判断他们的情绪，所以你去杂货店的大旅行会让你在人群中感到孤独。

所以你回到家，仍然感到孤独，也许你打开电视。观看人们互动可能是仅次于实际互动的最好事情，它甚至可能让你发笑。但是你曾经希望你能和电视上的人交谈吗？有了[aniketdhole]的 EMOJO 聊天机器人，你会觉得仿佛置身于朋友之间。从技术上来说，你是——所有的对话都来自电视节目*老友记*。

在《荒岛余生》中，汤姆·汉克斯没有给那个排球一张皱眉头的脸，不是吗？他也没有傻笑。相反，他面带苦笑，这表明了他的性格深度和对当前严峻形势的把握。但现在我们有了表情符号，它们在传达和唤起情感方面做得非常好。EMOJO 是一个视觉聊天机器人，它使用语音和表情符号进行简单的双向对话，以帮助赶走孤独。它使用 Raspberry Pi 和 TFT 显示屏从蓝牙耳机接收语音输入，将其转换为文本，然后以语音和文本的形式做出回应。[它入围了黑客日奖](https://hackaday.com/2021/06/28/ten-projects-won-the-rethink-displays-round-of-the-hackaday-prize/)的反思展示环节，我们迫不及待地想看到它的角色如何发展。请务必在休息后观看演示。
