# 电解槽除锈

> 原文:[https://hack aday . com/2018/11/24/电解-槽-除-锈/](https://hackaday.com/2018/11/24/electrolysis-tank-removes-rust/)

如果你有生锈的东西，你可以用钢丝刷和大量的油脂。或者你可以让电在电解槽中为你工作。[米勒的星球]向你展示如何[建造这样一辆坦克](https://www.youtube.com/watch?v=hpfwqItzjBY)，但更好的是，他以非常详细的方式解释了为什么它会工作。

这种水槽使用碳酸钠电解质——只有水和洗衣粉。在反应中，电解液中的自由电子取代了生锈金属片上的氧。一个玻璃容器、一根钢棒和一个电源构成了其余部分。

例如，一个生锈的滑动门被放在牢房里。当然，钢棒生锈了。这件作品出来看起来很糟糕，因为锈渣结块了，但一个小时后它就被擦掉了，留下了一个非常干净的作品。他认为把它放久一点会产生更好的结果。

你不应该使用不锈钢作为钢电极，因为不锈钢含有有毒形式的铬(尽管一位评论者提到这不是一个问题，我们不确定这是否是一个问题)。他还警告你一定要待在户外或通风条件下，因为这个过程会形成氢气，你不想让兴登堡灾难在你的店里重演。

你可以用电解作用做很多事情。甚至[钻孔](https://hackaday.com/2012/08/29/gilded-hello-kitty-ax-made-using-electrolysis/)。或者，你总是可以[制造氯气](https://hackaday.com/2018/08/19/cody-builds-a-chlorine-machine/)，它可能和氢气一样危险。
