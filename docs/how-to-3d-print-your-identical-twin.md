# 如何 3D 打印你的同卵双胞胎

> 原文：<https://hackaday.com/2019/11/23/how-to-3d-print-your-identical-twin/>

带着从某个广告展示中偷来的纸板剪影，在城市里度过一个愉快的周末是可能的。然而，现在是 2019 年，3D 打印意味着你可以走得更远。RCLifeOn 的[Simon]竟然打印出了一个真人大小的自己的替身，而且这只是有点令人毛骨悚然！(视频，下面嵌入。)

这个模型来源于西蒙对商用硬件进行的 3D 扫描。Optimus P1 工业级 3D 打印机用于打印零件，总打印时间约为 200 小时。粘合剂用于将各个部分连接在一起，然后将组件打磨并涂上底漆，准备上漆。

不愿意独自完成这项任务，[西蒙]请了一位专业画家来帮助完成这幅作品的最后润色。最终结果令人印象深刻，尤其是从远处看。[西蒙 2.0]随后被派往市中心，旨在向困惑的路人募捐。

我们怀疑，随着制造替身的技术越来越普及，定制替身的市场只会越来越大。如果你处理过类似的项目，[一定要让我们知道](http://hackaday.com/submit-a-tip)。休息后的视频。
