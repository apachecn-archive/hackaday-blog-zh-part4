# 以防你无法到达逃生室

> 原文：<https://hackaday.com/2019/11/27/in-case-you-cannot-make-it-to-an-escape-room/>

对于那些喜欢解谜、看事物如何运作或享受神秘的人来说，逃生室是很棒的。每个阅读这篇文章的人都至少属于其中一类。我们喜欢谜题和神秘，但我们也喜欢观察事物是如何运作的。为此，我们将你的注意力引向[doktorinjh]的[“手提箱中的炸弹解除谜题”游戏](https://imgur.com/a/8nmzieA)，这是一个他自己建造的神秘谜题盒。我猜其中的奥秘主要在游戏性上，你可以在下面观看，因为他向我们展示了他的构建照片，并描述了里面的硬件。

它的核心是一个 Arduino Mega，这是一个明智的选择，因为我们的背面估计他的 I/O 计数超过 45，Mega 可以用几个备用引脚来处理它们。在公文包的范围内工作有其自身的挑战，但我们喜欢他在顶部面板中使用六边形主题的方式，以允许旋钮间隙。这是如此微妙，我们几乎错过了它。

逃生室的主题令人愉快，我们欣赏多种形式的[游戏](https://hackaday.com/2019/11/04/ambitious-led-cube-provides-endless-video-game-scrolling-plays-castlevania/)、[美学](https://hackaday.com/2019/06/24/plasma-globe-reveals-your-next-clue/)和[科技诡计](https://hackaday.com/2019/06/13/let-the-cards-fall-where-they-may-with-a-robotic-rain-main/)的组合。
