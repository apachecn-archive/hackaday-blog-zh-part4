# 弹出式商店有助于充分利用小商店

> 原文:[https://hack aday . com/2019/08/09/pop-up-outlet-helps-make-the-most-of-a-tiny-shop/](https://hackaday.com/2019/08/09/pop-up-outlet-helps-make-the-most-of-a-tiny-shop/)

你不得不佩服一些人为了把商店挤进狭小的空间所采取的步骤。对于一些人来说，寻找方法来打包更多的工具和从事越来越大的项目成为他们自己的目的，他们发现这样做的整洁的小技巧真的很有启发性。

以[这个工作台弹出出口条](https://www.youtube.com/watch?v=h0Ky8jjGWck)为例。[木工店爱好者]占据的商店看起来是一个单车车库，首先规模较小，几乎全是多功能工作台。它在底部提供了大量的存储空间，在顶部提供了一个巨大的工作面，但使用小型电动工具意味着在已经有限的地板空间上拉伸延长线，并产生绊倒的危险。因此，他在工作台上为一扇巧妙的活板门留出了一点空间，隐藏了一个装有插座条的小托盘。

![](../Images/6dc5b4880bc3058153b1b530e6384de1.png)托盘骑在短抽屉滑轨上，借助一个小型气动弹簧，当门打开时，托盘会弹出。有一个小麻烦，一些下滑导致托盘堵塞，但这是一个简单的滚柱轴承照顾。下面的视频展示了它的构造，以及它如何在需要的时候完全不碍事。

尽管这个构建很酷，但与工作台相比，它只是小商店蛋糕上的糖衣。[Woodshop Junkies]有一个完整的播放列表，包括值得观看的版本。你可能想参考[我们的小商店综述](https://hackaday.com/2017/09/06/ask-hackaday-how-small-is-your-shop/)来获得更多在小空间里做很多事情的技巧。
