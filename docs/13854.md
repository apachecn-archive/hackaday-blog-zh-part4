# 非常奇怪的线框游戏男孩，实际工作

> 原文：<https://hackaday.com/2022/06/07/the-wonderfully-weird-wireframe-game-boy-that-actually-works/>

我们在 Hackaday 看到了许多伟大的硬件项目:有些非常聪明，有些制作精美，有些展示了它们的制造者的非凡技能。其他的只是普通的怪异，但是仍然设法包括一些甚至所有的上述类别。例证:[【kgsws】的线框游戏少年项目](https://www.youtube.com/watch?v=8YL36p2WiLM)。这可能是我们迄今为止见过的最怪异的游戏男孩 mod，但从技术角度来看也令人印象深刻。

基本的想法是拿一个 Game Boy 口袋，去掉它的外壳，换上一个由粗铜线制成的笼状结构。这听起来有点道理；想想那些透明的游戏男孩，只是没有透明的塑料。[kgsws]的视频(嵌入下方)显示，他弯曲了几根铜线，以匹配 Game Boy 的整体形状，然后为墨盒插座、显示器、D-pad 和四个按钮添加了支架。之后，你只需滑入印刷电路板，插入一些电池就可以了，对吗？

嗯，这对[kgsws]来说是不行的。相反，他所做的是使用热空气脱焊站从主板上移除所有芯片，并直接将它们安装在线框内，无需 PCB。然后，他用几十根细铜线连接墨盒插槽、CPU、RAM、按钮和其他所有东西，以重建主板的功能。当我们看到他残忍地用剪刀剪断显示器的扁平电缆时，我们有点退缩，那也是通过飞线连接到系统的其余部分，直接焊接到屏幕的触点上。

令人惊讶的是，在获得一对新电池后，该系统设法启动并运行其软件。尽管 D-pad 有点不可靠，但裸体游戏男孩实际上完全可用，尽管它可能需要比任天堂着名的防弹硬件更精细的操作。我们已经看到游戏机被改造成各种不同的形状和大小，但没有一个像今天这样不寻常。如果你喜欢线框结构，看看[这个诡异的声音发生器](https://hackaday.com/2022/01/15/the-eerie-sounds-of-ioalieia-an-esp32-valve-analog-hybrid-circuit-sculpture/)或者[这个漂亮的电路雕塑钟](https://hackaday.com/2021/11/10/a-breathtaking-circuit-sculpture-clock/)。
