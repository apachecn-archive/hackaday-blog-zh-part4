# 简约的门铃不需要互联网连接，甚至不需要电源

> 原文:[https://hack aday . com/2022/02/22/minimalistic-门铃-不需要互联网连接-甚至不需要电源/](https://hackaday.com/2022/02/22/minimalistic-doorbell-doesnt-need-an-internet-connection-or-even-a-power-supply/)

门铃是日常用品中的一种，开始很简单，但随着时间的推移变得越来越复杂。最初是一种将两块金属撞击在一起的机制，后来演变成各种有线和无线的电铃，最终发展到今天的智能门铃，即使主人在半个地球之外，也能向他们传送实时视频。

但有时，少即是多。[低技术痴迷] [用备用部件制作了一个门铃](https://www.youtube.com/watch?v=t_rXvGx3hEI)，它不需要互联网连接，甚至不需要电源。但它也不是一个纯粹的机械设备:游客转动安装在步进电机上的旋钮，产生交流电脉冲。这些脉冲然后被送入一个旧硬盘的音圈，使它的臂振动并敲击一个安装在盘片原来位置的铃。

除了是一个伟大的极简设计，门铃也是法拉第感应定律的一个巧妙的演示。步进电机显然足够坚固，能够承受破坏，尽管我们可以想象门铃的奇怪形状可能也会让一些善意的游客感到困惑。如果你对不寻常的门铃感兴趣，你可能会想看看这个由一个旧的墙上电话制成的门铃，还有 T2 这个机电装置。
