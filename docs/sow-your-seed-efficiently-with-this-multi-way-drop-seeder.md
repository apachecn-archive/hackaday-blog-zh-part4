# 用这种多路播种器有效地播种

> 原文：<https://hackaday.com/2019/11/16/sow-your-seed-efficiently-with-this-multi-way-drop-seeder/>

任何曾经用种子繁殖小植物的人都会知道，有效地播种是一个困难的过程。在每个点获得一致数量的种子同时达到任何速度几乎是不可能的，结果这变成了一个乏味的过程。要是有什么方法可以使它自动化就好了，也许有一种方法可以一次完成整个托盘！

幸运的是，迈克尔·拉特克利夫就在身边，带着他托盘大小的播种机。它由两块丙烯酸板组成，每块板上都有一个网格状的孔，彼此错开，可以通过杠杆对准。将种子在上表面摇动，直到所有的孔都含有一些，然后操作杠杆，让种子落入下面的土壤中。如果需要，有一个匹配的穴播器来推动土壤中所需的网格孔。

这是一个简单而巧妙的小工具，它真的会让园艺家的生活变得容易得多，尽管它可能并不适合所有类型的种子。他制作了一个视频，我们把它放在休息时间的下面，如果你想制作这个视频，我们已经介绍过了。
