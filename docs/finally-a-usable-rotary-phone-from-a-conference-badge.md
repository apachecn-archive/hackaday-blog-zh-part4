# 最后，一个可用的会议徽章旋转电话

> 原文：<https://hackaday.com/2019/11/09/finally-a-usable-rotary-phone-from-a-conference-badge/>

几周前，我们特别介绍了一个来自[Dan]的项目，这是一个正在进行中的项目，他正在将 EMF 2018 电子会议徽章贴在一部旋转式手机上。当时，我们期待着他的进步，希望在 2021 年的 EMF 大会上，我们能看到他的进步。我们不得不说我们当时帮了他一个大忙，因为他已经取得了很大的进步，现在[已经把它变成了一部功能齐全的蜂窝旋转式电话](https://trandi.wordpress.com/2019/10/28/rotary-dial-phone-revival-4-final/)。

当我们离开他的时候，他已经把表盘连接到了徽章上，但这足以激发我们的兴趣，因为我们认为旧的电子会议徽章应该有更多的重复使用。从那以后，[在电机驱动器和廉价的直流-DC 转换器的帮助下，逆向设计了最初的铃铛](https://trandi.wordpress.com/2019/09/15/rotary-dial-phone-revival-2/)，以及[带有蓝牙耳机的手机](https://trandi.wordpress.com/2019/10/03/rotary-dial-phone-revival-3-the-headset/)，因为在实验中，他设法杀死了徽章的音频电路。

结果可以在休息下面的视频中看到，我们不得不承认它看起来相当不错。取决于你在世界的哪个角落，你会喜欢或讨厌这种铃声，但这对这种设备的实用性没有什么影响。如果你有一抽屉积满灰尘的会议徽章，也许是时候再看一眼了。
