# ESP32 是这个艺术装置背后的大脑

> 原文:[https://hack aday . com/2021/08/22/esp32-is the brains-behind the-art-installation/](https://hackaday.com/2021/08/22/esp32-is-the-brains-behind-this-art-installation/)

由于其尺寸小、价格低和无线功能，ESP32 已经实现了无数的小型电子项目，甚至一些商业产品。大量的遥感器、照明设备，甚至家庭自动化项目现在都运行在这个小芯片上。但是，被归入一个电子外壳来控制一个小的电气设置并不是这些微小芯片所能做的全部，正如[ Eirik Brandal]用这个独特的视听艺术向我们展示的那样。 

该项目本质上是一个小型的自动化合成器，其中有一系列对应于各种音阶的阵列。这些中的任何一个都可以被选择用于乐器演奏。音阶的音符通过一些随机的变化被打乱，允许一个完全自动化的乐器。音乐生成也完全是模拟的，由一些振荡器、放大器和其他过滤和效果创建。ESP32 还控制着一个灯光雕塑，在音乐播放时点亮一系列 led。

艺术装置本身创造了令人难以忘怀的，令人着迷的曲调，在休息后链接的视频中有说明。虽然它并不完全属于人工智能领域，因为它使用预编程的模式，其中混合了一些随机性，但它确实给了我们一些线索[一些其他项目已经使用人工智能来创作新音乐](https://hackaday.com/2018/12/16/artificial-intelligence-composes-new-christmas-songs/)。
