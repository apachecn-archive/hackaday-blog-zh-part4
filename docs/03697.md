# 装在水壶里的乐高动力潜艇

> 原文:[https://hack aday . com/2019/07/28/Lego-powered-sub-built-in-a-water-jug/](https://hackaday.com/2019/07/28/lego-powered-sub-built-in-a-water-jug/)

潜艇被普遍认为很酷，但给遥控模型师带来了几个挑战，这些挑战在陆地和空中是没有的。进水会毁了你的项目，而且它永远有可能沉入水底，一去不复返。然而，这并没有使[砖块实验频道]相适应，于是一个乐高玩具潜艇就这样诞生了。(视频嵌入下方休息。)

潜艇用水罐作为船体。在处理密封旋转螺旋桨轴之前，该视频通过密封船体本身的过程。一个大的注射器被用作压载舱，乐高马达被用来驱动压载舱，并提供推进力和转向。一艘现有的遥控潜艇被用作零件，提供必要的无线电控制硬件。

在测试中，潜艇的表现令人钦佩，最后几个必要的调整，以提高螺旋桨的性能。它不会很快赢得任何比赛，但它是一个功能齐全的水下探险家，我们希望有一天自己能去湖上探险。

我们以前见过乐高潜艇，甚至包括导弹。
