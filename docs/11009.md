# 解决超高真空泄漏有一个基本的解决方案

> 原文：<https://hackaday.com/2021/08/16/solving-ultra-high-vacuum-leaks-has-an-elementary-solution/>

当我们想到真空泄漏时，我们通常会想到一辆不能正常行驶的汽车。大多数自然吸气式内燃机依靠活塞产生的真空来吸入化油器或燃油喷射系统产生的空气燃油混合物。确定泄漏通常包括在发动机运转时，在常见的故障区域周围喷洒可燃物质。发动机转速的变化表明可燃气体何时进入进气歧管，从而发现泄漏。

如果你的真空泄漏发生在一个高度专业化的科学设备中，那里的压力比大气压力低 12 个数量级，泄漏非常小，一次只能让几个原子进入真空室，那该怎么办？[AlphaPhoenix]在他的视频《[气密与真空密封》中深入探讨了这个主题。](https://www.youtube.com/watch?v=VD69crOFx10)“你可以在休息时间下面观看。

[AlphaPhoenix]不仅讨论了一个完美的压力容器是如何密封的，他还解释了所使用的专业故障排除方法，这些方法与汽车真空泄漏的故障排除没有什么不同，只是在这种情况下，本质上更加复杂和基本。

我们也很喜欢评论部分，在那里[AlphaPhoenix]回答了围绕视频的一些最常见的问题:扭矩模式，所用气体的稀缺性，以及放任自流。

谈论吸尘器会让你兴奋吗？也许你会喜欢这样的真空操作，比如把牙膏放回你的[自制真空室](https://hackaday.com/2020/06/29/an-affordable-home-made-vacuum-chamber/)的管子里。
