# 带面板的车间组织

> 原文:[https://hack aday . com/2021/08/10/workshop-organization-with-panels/](https://hackaday.com/2021/08/10/workshop-organization-with-panels/)

厌倦了所有已完成(或半完成)的项目把你的工作空间弄得乱七八糟？或者你把它们扔在一个盒子里，然后忘了是哪个盒子？[另一个制造者]aka[与 Dan 一起开发]想出了一个他称之为任务控制的解决方案——将你的项目面板化，并把它们存储在由假墙提供的许多格子中的一个格子中。

![](../Images/64abda4c2d13b8aa3aa33a18ab11a642.png)

Back view of an example project panel

每个项目都有一块面板，不用的时候可以整齐地存放起来。对于某些项目，这可能只是为了存储。对于其他项目，这可能是一个展示。占据任务控制中心的是一个大型监视器，大概是一个永久性的装置。看起来有两种不同大小的面板，但我们想知道更多和更小的面板是否会更有用。在他整理这些内容时，我们特别喜欢[Dan]对他的定制工具 *Cornerator 3000* 提出的一条建议:

> 当你想重复某件事时，不要犹豫要不要跳吉格舞。

[Dan]将在他的 GitHub 库上发布这个工作空间，以及他在 YouTube 上发布的各种项目的代码和文档。他还很自豪地用 100%回收材料建造了这个系统，或者如他所说，他去翻垃圾箱了。在你的实验室里，你有一个好的储存/展示项目的系统吗？请在下面的评论中告诉我们。
