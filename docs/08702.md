# 过度设计的开瓶器解除了喝酒的苦差事

> 原文:[https://hack aday . com/2020/12/18/over-engineered-bottle-opener-take-the-drugry-out-of-drinking/](https://hackaday.com/2020/12/18/over-engineered-bottle-opener-takes-the-drudgery-out-of-drinking/)

有些项目只需扫一眼，你就能知道最初是什么激发了它们。对于这个过度设计的机器人开瓶器来说，明显的影响是丰富的空闲时间和啤酒的结合。很多啤酒。

当然，根据不同的场合，有很多方法可以打开冰镇高杯的盖子。[马特·麦考伊]和他的同伴选择了“高脉冲”方法，当不是由机器人执行时，通常是通过将瓶盖的边缘放在工作台面上，并用手掌拍打瓶子来完成。这台毫无意义的机器做着同样的事情，除了风格不同。

瓶子被放在一个支架上，支架轻轻地但牢固地抓住瓶子，并以一种完全不必要的运动控制芭蕾将瓶子呈现给开启机构。一旦到位，丝杠向下移动支架，同时在一束弹性外科手术管中存储势能，同时释放盖子边缘的棘爪。一个杠杆在车厢底部移动，使棘爪向上飞行以释放酒，给机器人一个当之无愧的泡沫淋浴。在下面的视频中，看看 190 个定制零件和啤酒之间的奇妙互动。

向[Matt] *等人*脱帽致敬，感谢他们为各地陷入困境的啤酒商所做的不懈努力。这看起来像是与[精神控制啤酒乒乓](https://hackaday.com/2020/12/09/mind-controlled-beer-pong-gets-easier-as-you-drink/)游戏搭配的完美配件。
