# 用 3D 打印机了解历史上的枪械设计

> 原文：<https://hackaday.com/2020/09/12/learn-about-historic-firearm-design-with-a-3d-printer/>

在上个世纪，枪支子弹的基本设计几乎没有改变，但是枪支本身的机械结构经历了许多升级。扳机、保险装置、弹匣和操作机构的演变是一个令人着迷的研究领域。对大多数人来说，这些设备的实际操作经验很少，但多亏了像[zvc]这样的人，你可以[3D 打印历史火器的精确副本](https://www.youtube.com/watch?v=KUt549ytJ_U&list=PLePBcvcRk6xpi66lzzNmLKOqj32HNyOs2)，并看到所有部件如何为你自己装配在一起。

[zvc]正在慢慢建立一个 3D 模型库，目前有 9 个可用，从毛瑟 C96“扫帚柄”手枪到现代的 M4 步枪。除了弹簧和一些紧固件，几乎[zvc]模型的每一个零件都是 3D 打印的，包括拆卸销和提取器。除了能够发射实弹这一明显的例外，看起来所有的部件都像真枪一样配合工作。没有一个设计时考虑过 3D 打印，所以需要一台调整良好的打印机，大量的支撑结构和后处理才能使一切正常工作。表面光洁度会有点粗糙，一些较小的薄壁部件在重复操作或过度用力后可能容易断裂。这些型号不是免费的，但所有价格都低于€10。

这些模型确实展示了 3D 打印的真正超能力之一:功能模型和原型。快速迭代设计更新和在几个小时内获得最新设计的能力在产品开发中是非常宝贵的。[Giaco]在开发他的[动力驱动装置](https://hackaday.com/2020/09/04/the-screwdriver-you-dont-need-but-probably-want/)时广泛使用了这一技术。当你[在网上购买 3D 打印模型](https://hackaday.com/2020/08/26/3d-printering-the-world-of-non-free-3d-models-is-buyer-beware/)时，一定要确保可能存在的陷阱。
