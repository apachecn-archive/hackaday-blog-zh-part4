# 为全地形车配备垃圾桶升降器

> 原文：<https://hackaday.com/2022/04/29/equipping-an-atv-with-a-trashcan-lifter/>

[ITman496]是我们这些黑客中的一员，他的健康问题，在他的案例中，是背部受伤。他渴望解决他不得不处理的各种困难，在他制作的视频中，是关于移动一个大垃圾桶穿过他地产上被冰覆盖的道路。他不愿意再拿自己的健康冒险，也不满足于出售的脆弱解决方案需要他做繁重的提升工作，尽管如此，他还是设计并制造了一个绞盘驱动的垃圾桶提升机构——与叉车并不完全不同。他将它安装到他的 ATV 上，测试它，改进它，拍摄他的过程——然后为我们制作了一个详细描述整个构造的视频！

在他的手机上绘制了概念草图后，他在 SketchUp 中进行了建模和测试，然后切割和焊接零件，并描述了一种焊接对齐技巧——使用 3D 打印的接头将两个待焊接的零件固定在一起进行点焊，确保近乎完美的对齐。初始测试是成功的！在那里，他描述了一些令人惊讶但回想起来预期的易用性改进，这些改进在模拟过程中没有出现，比如给勺子添加倒角，这样他就不必非常精确地调整 ATV 的角度来捡起垃圾桶。最后，在使用了大约一个月之后，他告诉我们，对于他的目的来说，它非常有效！

谢天谢地，并不是所有这样的垃圾桶都需要拿出来——其中一些是自愿拿出来的，你甚至可以得到更小的垃圾桶，T2 可以接住你从房间另一边扔过来的东西。我们之前报道过[ITman496]的冒险，[从 2016 年一个失败的机器人制造中吸取教训。](https://hackaday.com/2016/10/07/fail-of-the-week-power-wheels-racing-series/)，2018 年[采用超轻型飞机](https://hackaday.com/2018/11/29/adopting-an-orphaned-ultralight/)！
