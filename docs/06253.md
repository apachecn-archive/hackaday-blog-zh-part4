# 用五金店的 UV-C 柜消毒 PPE，价格便宜

> 原文:[https://hack aday . com/2020/04/12/dissanct-PPE-on-the-price-with-this-hardware-store-uv-c-cabinet/](https://hackaday.com/2020/04/12/disinfect-ppe-on-the-cheap-with-this-hardware-store-uv-c-cabinet/)

当前的形势给了全世界的秘密细菌恐惧症患者一个走出去，清理一切的机会。有些可能过头了；当我们跑完步回到杂货店擦罐子和盒子的时候，我们自己可能会有一定的过度。但有时消毒是很明显的，有一种简单的方法来杀死口罩等物品上的细菌，可以延长一些通常是一次性物品的寿命，从而带来很大的不同。这就是[这个快速简便的 UV-C 杀菌柜](https://www.deeplocal.com/youvee)真正大放异彩的地方。

[Deeplocal]的“YouVee”背后的想法是，可以从任何大型家庭商店买到的零件快速拼凑起来，从而限制外出的次数。您甚至可能已经拥有了所有需要的东西，这将使它成为一个超级简单的构建。商业端是一个 UV-C 杀菌荧光灯，用于后院池塘的澄清器。一个荧光吊灯被改造成可以通过剪断一点塑料来接受这种灯，这种灯被固定在一个结实的黑色塑料手提包的盖子里面。手提袋的内部衬有铝带，用于待消毒物品的支架由油漆滚筒屏幕制成。巧妙之处在于安全联锁装置；为防止暴露在紫外线下，取下盖子前需要拔掉灯的插头。查看[完整的构建教程](https://makezine.com/projects/build-a-diy-uv-c-sterilization-cabinet-to-fight-covid-19/)了解详情。

我们不能保证 YouVee 的杀菌效果，但它似乎是一个坚实的设计。如果你有疑问，你可以随时[轻松测量 UV-C 通量](https://hackaday.com/2020/03/27/measuring-uv-c-for-about-5/)，或者你可以制造一个更小版本的[过氧化氢蒸汽个人防护用品消毒器](https://hackaday.com/2020/04/02/fda-says-ppe-can-be-reused-after-trip-through-shipping-container-decontamination-system/)，以防万一。
