# 失败:空气伞

> 原文：<https://hackaday.com/2020/01/25/failed-air-umbrella/>

大约五年前，一家 Kickstarter 为空气伞融资。没过多久，这个项目就告吹了，公司至少做了一些退款。旧闻，我们知道。但是【行动实验室】最近探索了空气伞背后的[物理学以及为什么它不太实用。(视频，嵌在下面。)](https://www.youtube.com/watch?v=-XwbLvhQnqA)

注意我们说的不太实际，不是不可行。使用压缩空气可以把雨从你身边射走。问题是你需要大量的空气压力。这意味着你还需要大量的电池。特别是，[行动实验室]使用了吹叶机，即使在这样的速度下，也只有很小的水流偏转。换句话说，你还是会被淋湿。

该视频计算平均雨滴的最终速度的估计值。我们查看了威斯康星大学的天气预报员网站，他们得出了一个略高的数字，但也不是很高。根据大学的说法，大约是每小时 20 英里，这取决于落差的大小。

根据视频，空气压缩机移动空气的速度必须比终端速度快一点。所以问题是，你想让一股时速 25 英里的气流正好掠过你的头顶吗？手柄里一个小电池能让压缩机运转多久？

正如你所看到的，这不是一件实际的事情，但它仍然设法产生了相当多的支持——超过 100，000 美元。我们想知道赞助者中是否有物理学家？

我们对[疯狂的 Kickstarter 声明](https://hackaday.com/2017/07/30/false-claims-on-kickstarter-whats-new/)并不陌生。当然，即使是伟大的产品[有时也会在众筹市场失败。](https://hackaday.com/2016/05/16/why-kickstarter-products-fail/)
