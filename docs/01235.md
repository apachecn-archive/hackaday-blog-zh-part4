# 不邪恶的树莓派看起来只是一个黑客

> 原文：<https://hackaday.com/2018/11/13/non-nefarious-raspberry-pi-only-looks-like-a-hack/>

我们要提前警告你，这不是一次黑客攻击。或者至少在[LiveOverflow]对一个藏在大学图书馆的神秘设备进行了一些数字取证后，事情变成了这样。他得出的结论是，没有任何不顺心的事情发生，这一结论很有趣，也很有启发性，因为这是未知艺术品的最终目的。

正如[LiveOverflow]在下面的视频中告诉我们的那样，他在 Reddit 上看到了一条线索——我们现在找不到任何线索——描述了一群藏在大学图书馆垃圾桶、自动售货机和桌子后面的奇怪设备。[LiveOverflow]将发布的图片识别为带有 USB WiFi 加密狗的 Raspberry Pi Zeroes 好奇心被激起，他向 OP 伸出手，主动提出帮助解开这个谜。

下面的视频讲述了随后的取证乐趣，包括一些可疑的做法，如将设备的 SD 卡插入发现者的 PC。在发现者看来非常“黑客化”的东西，在[LiveOverflow]进入远程诊断兔子洞以辨别这些设备的用途后，被证明是相当无害的。我们不会破坏揭示，但足以说，他们是一个非常聪明的系统的一部分，完全没有邪恶的目的。

我们认为这是一个有趣的信息安全游戏，并在几个层面上具有指导意义，尤其是要记住“平民”在野外可能会看到这样的设备。我们每天接触的硬件和软件对普通大众来说可能具有威胁性。也许下次大学应该花钱买一些描述这个装备的标签。
