# 让您的 3D 打印零件焕然一新

> 原文:[https://hack aday . com/2022/05/19/put-a-new-spin-on-your-3d-printed-parts/](https://hackaday.com/2022/05/19/put-a-new-spin-on-your-3d-printed-parts/)

一旦你厌倦了用 3D 打印机打印钥匙链和耳塞支架，你会想要设计一些更复杂的东西。旋转的东西呢？[3DSage]对如何将简单的电机和控制器集成到几个不同尺寸的盒子中有很好的指导。结合一些 3D 打印的链接，这些盒子可以将您的项目(打印或其他)变成旋转的东西。

为了演示，他创造了一些猫玩具，玩了一个魔术的想法，并把自拍灯改装成…什么的。我们毫不怀疑你能找到这些小马达模块的用处。这些盒子的区别主要在于电池组的大小。还有几个有趣的配件，如可充电纽扣电池及其充电器的 3D 支架。

此外，他还演示了如何使用电动机作为(相当差的)发电机。直到他用压缩空气来驱动水车，他才成功地安装了水车。你可能会认为水可以解决这个问题。

视频强调你应该焊接连接，但你不是必须的。老实说，我们认为如果你正在用 3D 打印机建造移动的东西，你可能应该继续学习焊接。这并不难，而且有很多学习的理由。

当然，你可以 [3D 打印马达本身](https://hackaday.com/2014/07/30/a-3d-printed-brushless-motor/)。[让电机模块适应不同的用途](https://hackaday.com/2017/06/22/converting-a-robotic-motor-for-lego-blocks/)当然不是一个新想法，但看到更多应用基本组件的方法总是很棒。
