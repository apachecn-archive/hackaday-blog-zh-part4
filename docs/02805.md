# 一部军用 TA-1024A 野战电话

> 原文:[https://hack aday . com/2019/04/24/house-training-a-military-ta-1024 a-field-phone/](https://hackaday.com/2019/04/24/house-training-a-military-ta-1024a-field-telephone/)

在博物馆发现一些有趣的军用手机后，[CuriousMarc]想知道如何将这些重型电信设备改造成民用。他知道大部分内部零件都是徒劳的，但他推断，如果他能对听筒和键盘等关键元件进行逆向工程，他也许能把它们连接到标准电话的电子设备上。对我们来说幸运的是，[他好心地记录了这个过程](https://www.youtube.com/watch?v=4h-TOQAOY_I)。

有许多有趣的问题需要解决，但第一个也可能是最大的一个问题是键盘不寻常的接线。它没有像我们这样的现代黑客可能期望的那样连接，而且[CuriousMarc]不得不最终做一些相当重要的重新布线。通过用 Dremel 切割 PCB 上的现有迹线，并在背面钻新孔来布线，他能够将其转换为当代按键式电话可以使用的布线方案。

需要制作一个适配器来安装一个基本的驻极体麦克风，以取代原来的动态麦克风，但原来的扬声器是可用的。他想采用磁传感器来检测手机何时摘机，但最终还是钻一个小孔并使用标准按钮要简单得多。

这款手机的主板是你所期待的军用通信设备中华丽的零成本结构的完美例子，但不幸的是，它不得不被扔进垃圾箱。取而代之的是一部价值 9.99 美元的 RCA 手机。[CuriousMarc]将无法再联系北美防空司令部，但至少他可以订一份披萨。键盘上的红色按钮最初用于设置军方 AUTOVON 电话网络的呼叫优先级，现在已经连接到电话的更普通功能，如重拨。

虽然这对于一次性项目来说很好，但我们希望看到这些电话的嵌入式 POTS 或 VoIP 转换[，而不涉及如此多的修改和重新布线](https://hackaday.com/2015/03/09/convert-a-rotary-phone-to-voip-using-raspberry-pi/)。现在我们已经有了一些关于键盘和挂钩传感器的文档，在定制 PCB 时考虑它们的特性应该不难。[将老式装备拖进现代总是黑客们最喜欢的消遣](https://hackaday.com/2018/07/09/vintage-headphones-bluetooth-conversion-goes-the-extra-mile/)，所以也许有人会受到[的启发，接受挑战](https://hackaday.com/2019/04/03/2019-hackaday-prize-begins-right-now/)。
