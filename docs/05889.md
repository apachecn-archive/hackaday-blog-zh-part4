# 一个防静电戒指让他们都很开心

> 原文：<https://hackaday.com/2020/03/12/one-anti-static-ring-to-delight-them-all/>

冬天最糟糕的是什么？如果你和我们一样喜欢呆在室内，那么静电可能是最重要的因素。它会毁了你的芯片，真的，但你总是戴着一个手腕接地带当你处理这些，对不对？但是远离长椅，每一个门把手和电灯开关都埋伏着，随时准备电击你。如果你有[一个像【LaPuge】一样的防静电环](https://www.instructables.com/id/Antistatic-Ring/)，你可能会看到一个微小的霓虹灯点亮，而不是你可怜的手指和放电点之间的空气。

戒指本身采用 TPU 95A 细丝印花，舒适灵活。电路并不复杂，只是一个氖灯、一个 1mω电阻和一些铜带，但这件功能性珠宝有可能引发大量带电对话。快速穿过缺口，看到它靠着门把手亮起来。

如果你想一整年都点亮霓虹灯，[建造一个霓虹灯场，在你的特斯拉线圈附近挥舞它们](https://hackaday.com/2020/03/04/visualizing-energy-fields-with-a-neon-bulb-array/)！
