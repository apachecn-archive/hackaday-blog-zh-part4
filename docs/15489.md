# 提高功能的美学 PCB 设计技巧

> 原文：<https://hackaday.com/2022/11/29/aesthetic-pcb-design-tips-for-improved-functionality/>

我们这些花费大量时间设计 PCB 的硬件类型的人经常会饶有兴趣地看着其他人的设计，在某些情况下，还会相当钦佩。他们的一些板子看起来太棒了。但是审美重要吗？毕竟，对于大多数产品来说，PCB 上的精密电子元件都被安全地藏在一个保护罩内。但是，正如[Phil's Lab]解释的那样，[美观的 PCB 设计可以带来功能上的改进](https://www.youtube.com/watch?v=D2UaRPkRExw)，例如，在可制造性(以及产量)、电磁兼容性(EMC)和其他几个重要因素方面，外观更好的设计表现也更好。

首先，使 PCB 易于阅读，并使用合理的元件和连接位置，可以减少错误，从而加快调试速度。保持一致且不太紧的放置网格可以使拾取和放置机器更容易完成任务，并减少回流期间的焊接问题。但也有更严重的问题，如设计分区的实施，如将模拟电路放在一起，远离高噪声电源和数字区域，这可能会影响正常工作和失败。

该视频还介绍了其他一些有趣的技巧，其中一个亮点是使用接地 PCB 周界区，通过拼接实现目标波长。这将降低电源层的 EMC 侧辐射，而且如果选择适当的表面光洁度，并保持阻焊层敞开，您就可以获得一个自由的全周长触点来接地示波器探头。哦，看起来也不错。

Hackaday 对漂亮的艺术印刷电路板并不陌生，比如[Saar Drimer]和许多其他人的作品。但是，如果一个 PCB 不能满足你的需求，总有[的“奥利奥”结构可以考虑](https://hackaday.com/2019/01/18/oreo-construction-hiding-your-components-inside-the-pcb/)。
