# 在家自己做榫钉

> 原文:[https://hack aday . com/2019/01/08/自制榫钉/](https://hackaday.com/2019/01/08/make-your-own-dowels-at-home/)

榫钉是一种有用的木工技术，它使连接几块木材变得容易，特别是在粘合剂的帮助下。然而，取决于你住在哪里，很难买到各种各样的股票。如果你关注外观，这一点尤为重要——例如，便宜的松木榫可能会破坏精致的硬木成品的外观。

谢天谢地，在家里自己做榫钉很容易。[【Pask Makes】之前一直使用简单的传力杆板，但这次，决定打造豪华版。](https://www.youtube.com/watch?v=-CG1XO08ByA)在一块厚钢板上钻一系列的孔，然后安装到一个木块上。然后，可以将方形原料压入孔中，以生产榫钉。

[Pask]注意到有几种方法可以使用销钉板。将木材原料锤入孔中最适合硬木，而将方形原料放入电钻的卡盘中，并在旋转时迫使其通过，可以在较软的木材上获得更好的光洁度。还有一些关于如何最好地生产榫钉的有用提示，包括强度和纹理方向的说明。

在你的工作室里，这是一个有用的工具，意味着你可以把任何木材变成你的木工项目的榫钉。如果你是木材世界的新手，不要担心—[我们已经为你准备了入门书。](https://hackaday.com/2017/01/23/woodworking-basics-for-the-hardware-hacker/)休息后的视频。
