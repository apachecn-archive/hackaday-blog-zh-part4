# 让棒球超音速飞行

> 原文：<https://hackaday.com/2020/09/11/making-baseballs-go-supersonic/>

当专业工程师像孩子一样咯咯笑时，你知道有趣的事情就要发生了。[每天都更聪明]的[Destin Sandlin]，[Jeremy Fielding]和其他几个志同道合的人已经建造了一个非常令人印象深刻的空气炮，[能够以超音速发射棒球](https://www.youtube.com/watch?v=cqidD7kVnxY)。

![](img/88bcc09278ead13b892f931bae6d0d7f.png)

Baseball execution. Not for sensitive viewers.

枪口加载佳能由一个大的压力室和真空室粘在一起，一个插头和棒球分开两者。枪管形成真空室的一部分，并在枪口端用塑料带密封，塑料带在发射时会破裂。击发机构贯穿压力舱的整个长度，从后面退出，由一个大型气动击发机构固定在适当的位置。当扣机被释放时，它“弹出”两个室之间的软木塞，将高压氮气送入真空室，迫使球向前移动。这导致塞杆从压力室的后部射出，在那里被气动活塞阻止。整个装置被永久安装在拖车上。一个看起来很专业的控制盒被用来在安全的钢防爆盾后面操作野兽。

请务必在休息后观看带字幕的视频。第一个是集锦，第二个是长达一小时的非常有趣的幕后花絮。令建造者惊讶的是，他们能够在第一次尝试中以 1.38 马赫(1050 英里/小时或 1690 公里/小时)的速度射出一个棒球，只有一个部分加压的系统和一个泄漏的真空室。当撞击厚钢靶时，球完全分解，在靶上留下缝线。[Destin]和他的同事用他常用的高速相机记录了结果，但也包括一个[纹影](https://hackaday.com/2016/01/09/hacklet-90-schlieren-videos-and-jigsaw-puzzle-robots/)装备，使他们能够拍摄高速球产生的冲击波和马赫锥。几个镜头之后，螺栓被从停止塞棒的气动活塞上取下，根据钢架在该区域的弯曲程度来判断，这并不奇怪。

这是一个非常雄心勃勃的项目，我们不得不承认它是值得发笑的。解决了初期问题后，我们期待看到更多精彩的实验。[Destin]因其内容丰富且有趣的视频而闻名，从[傅立叶分析](https://hackaday.com/2018/12/21/explaining-fourier-again/)到[倒着骑自行车](https://hackaday.com/2015/04/27/try-not-to-fall-off-the-backwards-brain-bike/)。[Jeremy Fielding]制造了一些令人印象深刻的工具，我们特别介绍了他的[数控台锯](https://hackaday.com/2020/09/08/measure-1024-times-cut-once/)和 [DIY 测力计](https://hackaday.com/2020/03/02/this-diy-dynamometer-shows-just-what-a-motor-can-do/)。
