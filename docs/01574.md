# 旧瓦特计使用磁性做数学

> 原文:[https://hack aday . com/2018/12/18/old-watt meter-uses-magnetics-to-do-the-math/](https://hackaday.com/2018/12/18/old-wattmeter-uses-magnetics-to-do-the-math/)

测量电路中的功率传输似乎是一项简单的任务。测量电流和电压，用[焦耳]和[欧姆]做一点数学运算，你就得到答案了。但是如果你想设计一个自动计算的仪器呢？如果你必须严格按照机电原理来做呢？

这就是[Shahriar]在拆除旧的实验室级瓦特计时要解决的问题。老实说，这个视频对他来说有些偏离；我们已经习惯了看到仪器经过他的工作台，如果是新买的，这些仪器会在[的钱包里打出一个七位数的洞](https://hackaday.com/2018/09/24/tearing-into-a-1-3-million-oscilloscope/)。这些瓦特表来自韦斯顿仪器公司(Weston Instruments)，是坚固的中世纪工业设计的美丽范例，似乎至少一直使用到 2013 年。沉重的胶木外壳和用于电流和电压输入的坚固接线柱使电表看起来像是摔在地板上也能一笑而过。

但是，正如[Shahriar]在拆除一个祭祀用的仪表时发现的那样，仪器背后的机电运动非常微妙。瓦特计使用动圈式电表，与其他面板电表非常相似，但用一对线圈取代了永磁定子。电压接线柱通过一个串联电阻连接到动圈的细线上，同时电流通过定子线圈的较重绕组。两个磁场共同作用，将电压乘以电流，并使指针克服弹簧预载而偏转，以显示功率。它非常聪明，内部工作是一种享受。

我们只是喜欢观察旧电子产品的内部，尤其是动圈式电表。它们是很棒的小玩意，而且重新利用也很有趣。
