# 环氧树脂 LED 立方体看起来很光滑，并随着节拍闪烁

> 原文:[https://hack aday . com/2019/07/07/epoxy-led-cube-looks-sleek-and-flash-to-beat/](https://hackaday.com/2019/07/07/epoxy-led-cube-looks-sleek-and-flashes-to-the-beat/)

如果说在这个两极分化的时代有什么东西是普遍流行的，那就是五颜六色的发光物体。led 在这一领域占据着至高无上的地位，我们习惯于看到各种有趣的闪光设备击中了 tips 线。今天也不例外，我们一直在看[【modu structural Maker】的时尚环氧树脂 LED 立方体](https://www.youtube.com/watch?v=np89oPvm0DY)。

建造始于黑色环氧树脂立方体的铸造，顶部附近有一个切口，led 将安装在其中。使用三聚氰胺形式，用铝箔胶带，填缝和糊蜡来帮助密封。从模型中释放铸件后，有一些难看的空隙，通过用台锯修剪下来，这些空隙很快被消除。将块切割成合适的尺寸，安装 LED 灯条，在白色环氧树脂作为扩散器注入之前，用热胶密封灯腔。剩下的只是打磨立方体和安装电子设备的简单工作。

这个立方体由一个单细胞脂肪电池供电，还有一个无线电源接收器和充电模块来保持电力流动。这个立方体可以用在大多数无线手机充电器上，也可以用在它自己的专用充电底座上。led 由现成的模块控制，该模块提供各种闪烁显示以及音乐反应模式。

虽然电子方面是用现成的部件完成的，但这件作品中真正的艺术是在立方体的构建中。它光滑迷人的外形放在任何茶几或床头柜上都会令人惊艳。

LED 立方体是你午休时的绝佳去处。这个支持 OpenGL 的构建给人留下了特别深刻的印象。休息后的视频。
