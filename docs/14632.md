# 暴风雨前的宁静？

> 原文:[https://hackaday.com/2022/08/27/the-quiet-before-the-storm/](https://hackaday.com/2022/08/27/the-quiet-before-the-storm/)

我和妻子正在读一本关于 20 世纪初物理学的书。一半是科学史，一半是一些最著名的物理学家的传记，这很有趣。但这让我想到了 120 年前的物理状态。

我们现在称之为经典力学的东西在相当长一段时间内被完全解决了，甚至神秘的电学和磁学最近也被麦克斯韦和亥维赛德所终结。似乎暂时没有什么需要解释的了。然后所有的门都敞开了。

尽管我个人很喜欢爱因斯坦的相对论，但我认为量子力学是视角上最具革命性的变化，也是这一世纪以来大多数研究的驱动力。这一切是如何开始的？以最奇怪的方式——尼尔斯·玻尔担心为什么氢气和氦气在电离时会发出特殊的颜色——这导致了他的原子模型和量子包中能量的想法。或者可能是[德布罗意的想法，即电子可以表现得像波](https://en.wikipedia.org/wiki/Matter_wave)或磁铁，分别来自狭缝和阴极射线实验，这导致了海森堡的测不准原理。

不管怎样，最奇怪、最深刻的物理学革命——量子力学——的诞生来自于回答一些简单明了的问题。氦气为什么会发出粉红色，电视是怎么工作的？(我知道，那时他们还没有电视……)不管是分开看还是放在一起看，没有人会想到，回答这些问题需要我们彻底重新思考如何看待现实。然而它发生了。

我不禁想知道，除了大型强子对撞机或詹姆斯·韦伯太空望远镜等数十亿美元的项目之外，是否还有一些更简单的现象，我们应该问“为什么？”关于。我们是否处于暴风雨前的类似宁静中？或者说，通过这些大型项目来不断推回我们无知的边界是真的吗？
