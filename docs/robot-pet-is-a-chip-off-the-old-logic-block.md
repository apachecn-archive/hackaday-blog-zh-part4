# 机器宠物是旧逻辑块的翻版

> 原文：<https://hackaday.com/2021/08/15/robot-pet-is-a-chip-off-the-old-logic-block/>

当[埃兹拉·托马斯]的高级设计项目需要灵感时，他只需要看看他自己的机器人。[Ezra]在高中时期根据 1979 年 Frank DaCosta 的经典书籍“如何构建自己的工作机器人宠物”构建而成，他已经艰难地了解到围绕其 8085 处理器的绕线 74xx 系列逻辑芯片的许多限制和复杂性。

[以斯拉]开始寻求重新创造微型怪物，称之为[芯片宠物](https://ezrasrobots.wordpress.com/2021/07/07/pet-on-a-chip/)。使用现代 FPGA 芯片允许电子器件缩小一个数量级，并为未来的扩展提供灵活性。在足够大的 FPGA 上实现 8 位 CPU 为 VGA GPU、电机控制器、串行 UART 等留下了足够的空间。用 Python 编写的自定义汇编程序处理 CPU 编程。

结果呢？重量轻 12 倍，功耗低 13 倍，性能更好，还有很大的发展空间。[Ezra]暗示了 I2C 总线的扩展以及更高级的编程语言，使得软件开发不那么困难。

芯片上的宠物是一个非常棒的工程项目，我们希望随着时间的推移，我们会从[Ezra]那里看到更多这样的东西。在休息时间下面的视频中观看他的宠物在芯片上的活动。

如果[Ezra]的 FPGA 恶作剧让你不知道如何开始，你可以查看来自 [2019 Hackaday 超级大会](https://hackaday.com/tag/2019-hackaday-superconference/)的[这篇 FPGA 介绍](https://hackaday.com/2019/12/06/david-williams-is-fpga-curious/)。如果你有自己的 FPGA 作品要分享，请通过[提示热线](https://hackaday.com/submit-a-tip/)告诉我们！
