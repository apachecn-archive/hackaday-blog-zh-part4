# 可擦钢笔墨水为 3D 打印增添色彩

> 原文:[https://hackaday . com/2021/09/28/erasable-pen-ink-adds-colors-to-3d-prints/](https://hackaday.com/2021/09/28/erasable-pen-ink-adds-colors-to-3d-prints/)

众所周知，在 3D 打印过程中改变颜色非常困难。要么您需要多个打印头在打印期间准备就绪，这会增加打印机的操作和维护成本，要么您需要停止打印以切换灯丝，然后希望在恢复打印时一切都匹配。有一些解决这个问题的方法，但没有几个像[这个一样顺利轻松，它使用可擦笔墨水在飞行中给灯丝添加颜色](https://hackaday.io/project/181767-programmable-filament-reversible-thermochromic)。

可擦钢笔墨水是一种热变色材料，当它像铅笔上的石墨一样被擦掉时，不会从纸上消失。相反，摩擦产生的热量使它变得透明。通过将此属性用于 3D 打印，可以简单地通过改变热端的温度来操纵打印中的颜色。当然，[Autodrop3d]的团队在试验这种方法时有一个相当好的学习曲线，因为他们必须在比正常温度低得多的温度下运行挤压机，以控制墨水的颜色，必须以比正常速度慢得多的速度运行打印，并且使用非常粘的低温塑料进行打印。

由于对打印设置的所有这些修改，在材料和速度方面肯定会有一些限制，但项目的结果不言自明。这使得普通的 3D 打印机可以在不修改硬件的情况下使用这种方法，并且颜色的改变可以完全在软件中完成。虽然每个人都在追赶这种新技术，但是有多个打印头的 3D 打印机还有一些其他的好处，以及一些在没有太多中断的情况下进行切换的聪明方法。
