# 游戏耳机实现简单的无线充电

> 原文:[https://hack aday . com/2021/05/17/gaming-headset-gets-simple-wireless-charging/](https://hackaday.com/2021/05/17/gaming-headset-gets-simple-wireless-charging/)

尽管这项技术本身已经普及，而且相对便宜，但提供无线充电功能的设备仍不像许多人希望的那样普遍。当然，它不能提供像 USB-C 这样的东西那么多的功率，但对于不需要匆忙充电的低功耗设备来说，这种便利是不可否认的。

厌倦了每次会议后都要插上电源，[泰勒·伯利]决定自己动手，为他的海龟滩侦察 200 耳机增加无线充电功能。但最终，这个项目没有什么不能适应你自己特定的耳机选择。或者任何其他通过 USB 充电的设备。

[![](../Images/de4e00fa6706b71873a9ea409e357aa3.png)](https://hackaday.com/wp-content/uploads/2021/05/qiheadset_detail2.jpg) 为了简单起见，【泰勒】使用了现成的无线充电发射器和接收器对。发射器安装在 3D 打印的支架上，耳机悬挂在支架上，接收器简单地粘在耳机的顶部。接收器覆盖着一层薄薄的 3D 印刷板，但如果你不想设计一个全新的部分，几圈电工胶带也可以。

一旦一切就绪，他就沿着耳机的侧面拉一根电线，接入来自 USB 端口的 5 伏线路。因此，现在只要[泰勒]在演奏结束后记得挂好耳机，下次他伸手拿耳机时，电池就会一直充满。

[考虑到我们已经看到了多少项目](https://hackaday.com/2019/10/07/make-wireless-earbuds-truly-wireless/)[将无线充电添加到消费设备](https://hackaday.com/2014/06/10/stuff-wireless-charging-into-a-nooks-crannies/)，老实说，这在 2021 年仍然不是一个标准功能，这有点令人惊讶。在制造商[弄清楚他们想用这项技术](https://hackaday.com/2017/02/23/suddenly-wireless-power-transmission-is-everywhere/)做什么之前，看来黑客们只能继续自己做了。
