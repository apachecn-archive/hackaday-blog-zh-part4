# 如何将电锯变成切锯

> 原文:[https://hackaday . com/2019/07/05/how-to-turn-a-a-champsaw/](https://hackaday.com/2019/07/05/how-to-turn-a-chainsaw-into-a-chopsaw/)

如果你要做大量的金属加工，锯子是一个很好的工具。这是一种简单快捷的方法，可以进行大量整洁、准确的切割。[制造材料]想做的就是这个，但是没有锯子。[相反，一个旧的 Stihl 电锯被放在工作台上，黑客攻击随之而来(Youtube 链接，嵌入下方)](https://www.youtube.com/watch?v=StWLRmgQJUY&ab_channel=MakingStuff)。

为此，有必要采购一些零件，并对链锯进行一些改造。离合器钟被删除，并修改安装一个滚子链链轮。然后制造了一个臂，它在远端安装了一对轴颈轴承。另一个链轮安装在这一端，以及安装切割轮的轴。最后，在切割轮上安装了一个防护装置，给建筑一个安全的假象。

[Making Stuff]指出，链锯可以很容易地转换回它的标准用途，只需要改装原始零件，并用库存零件替换修改后的离合器钟。这是一个从一个工具中获得两个工具的好方法，我们确信它将在未来的项目中被证明是有用的。
