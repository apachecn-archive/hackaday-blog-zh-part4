# 可食用电子产品让我们听到了巧克力兔子的哀歌

> 原文:[https://hack aday . com/2022/04/19/食用电子产品——让我们听听巧克力兔子的哀歌/](https://hackaday.com/2022/04/19/edible-electronics-let-us-hear-the-lamentations-of-the-chocolate-bunnies/)

又一个巧克力兔子日过去了，你做了什么来纪念这个日子呢？你可能会放松，被你为大斋节放弃的任何东西包围，但如果你像[反复失败]那样混合巧克力和电子产品，对你有好处。他们创造了完全可食用的巧克力复活节兔子[，当被咬时会尖叫。](https://www.youtube.com/watch?v=z4N4V3HJxAI)

[![](../Images/65c09de8d83086bd3dc10440083ab9a3.png)](https://hackaday.com/wp-content/uploads/2022/04/screaming-chocolate-bunny-inner.jpeg) 很明显，最难的部分是找到既导电又能安全食用的东西来构建电路。【反复失败】花了很多时间用碳[油凝胶](https://onlinelibrary.wiley.com/doi/full/10.1002/adfm.202113417)膏，这是由天然油和蜡制成的。它不仅比[反复失败]的皮肤导电性差，而且出来后漆黑一片，尝起来一点味道都没有，仔细想想，这是一种额外的收获。

然后是蛋糕漆，里面掺了微量的银粉。虽然这样做效果更好，但一个成功的电路可能需要近乎致命的数量。呀！

胜出者竟然是食用银叶，类似金叶但更便宜。有过戈尔德施莱格吗？金叶是悬浮在里面的东西。silver leaf 的真正好处是它是薄片状的，可以很容易地用剪刀剪成电路走线，并用铜带连接到 I/O 引脚。一定要在休息后检查一下，包括[重复失败]的朋友对无辜咬下巧克力兔子耳朵的反应，就像一个人倾向于首先做的那样。

你觉得你更愿意听到植物咯咯的笑声吗？当然，这听起来很可爱，但实际上有点令人毛骨悚然。
