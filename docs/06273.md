# 树莓派可视对讲系统

> 原文:[https://hack aday . com/2020/04/27/a-raspberry-pi-video-intercom-system/](https://hackaday.com/2020/04/27/a-raspberry-pi-video-intercom-system/)

当谈到黑客时，我们总是对设计的美感感到惊讶，正如我们对电路的复杂性或软件的聪明感到惊讶一样。我们认为组装项目总是很有趣，这些项目只是在我们的商店里很快组装起来，并且只需工作，无需担心其他事情。但是，当你真正在美学上投入时间，将形式与功能结合起来时，结果总是令人惊叹。

这就是[Hacker Shack]的工程师在 Hackster 上用他们的基于树莓 Pi 的视频对讲系统所做的。现在[在](https://hackaday.com/2019/05/13/video-doorbell-focuses-on-quality-aesthetic/)之前，我们已经在 Hackaday 上看到了 RPi 门铃项目，但它是一个全双工视频对讲系统的实现，使【黑客小屋】的项目真正脱颖而出。([除非你想更隐秘一点](https://hackaday.com/2014/08/06/raspberry-pi-spies-on-your-front-door/))。他们使用了一个带有非品牌 Pi 摄像头的 Raspberry Pi 3 型号 B，但 R Pi 品牌的摄像头也可以正常工作。将摄像头与清晰的 LCD 显示屏、麦克风和扬声器连接起来，您就可以开始工作了！
