# 化学和激光把任何塑料表面变成印刷电路板

> 原文:[https://hack aday . com/2018/09/04/chemistry-and-lasers-turn-any-plastic-surface-into-a-PCB/](https://hackaday.com/2018/09/04/chemistry-and-lasers-turn-any-plastic-surface-into-a-pcb/)

从表面上看，PCB 生产似乎已经减少到实际生产。爱好者永远都在蚀刻自己的电路板，定制 PCB 制造市场上有很多供应商，他们的能力涵盖了从非常简单的单面通孔电路板到可以想象的最细间距多层 SMD 电路板的所有范围。

那么，我们究竟为什么还需要另一种方法来制造多氯联苯呢？因为正如[Ben Krasnow]所指出的，将几乎任何塑料表面转变成 PCB 的能力真的很方便，而且不一定是 fab houses 现在处理的事情。下面的视频展示了[Ben]是如何想出他的方法的，他走了一条不明显的道路，一部分是化学实验，一部分是材料科学。基本的想法是使用无电镀铜，一种不使用电解而将铜沉积到基底上的方法。

这使得非导电基板(Ben 使用 Formlabs SLA 打印机打印的小零件)能够镀上足够的铜，以形成可焊走线。这其中涉及的化学成分并不简单；还有催化剂、表面活性剂和硫酸铜饱和溶液要管理。即使他拨入了，他也必须弄清楚如何用激光切割机制作走线和过孔。它最终成功了，但需要做大量的工作。看看下面的视频，看看他是如何到达那里的，以及他下一步打算去哪里。

你必须把它交给[本]；当他决定探索某样东西时，他会全力以赴。我们欣赏他的奉献精神，无论他是用蜡烛探索磁流体力学还是用高速水流制造等离子体。
