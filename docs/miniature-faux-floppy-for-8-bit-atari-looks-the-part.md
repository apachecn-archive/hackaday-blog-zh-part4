# 微型仿软盘 8 位雅达利看起来的一部分

> 原文：<https://hackaday.com/2020/09/07/miniature-faux-floppy-for-8-bit-atari-looks-the-part/>

过去的复古电脑有很多乐趣，但对现代用户来说，这可能是一种文化冲击。这么多年后回到软盘，提醒人们存储技术在速度、可靠性，当然还有容量方面已经取得了多大的进步。幸运的是，有办法结合两个世界的优点。

经典计算机的软驱模拟器当然不是什么新东西，但是我们认为[c0pperdragon]组装的这个[值得仔细研究一下](https://github.com/c0pperdragon/SIO2SD)。基于 ATmega32U4 的仿真器不仅器件数量特别少，而且代码是在 Arduino IDE 中编写的。这两个特性使得新玩家可以很容易地复制和修改设计。必要时，你甚至可以用 Arduino 在试验板上实现它。

模拟器被装在一个 3D 打印的外壳中，看起来像一个时代合适的 Atari 1050 磁盘驱动器，只是你使用的是 SD 卡而不是软盘。该固件可以模拟两个物理驱动器，并支持每个 SD 卡上多达 100 个磁盘映像。用户界面非常简单，有两个按钮和一对七段 led 来指示当前加载了哪个磁盘映像。

这些年来，我们已经看到了一些非常复杂的磁盘模拟器，但是这个版本的简单明了令人信服。如果它能帮助更多的人体验逆向计算的独特乐趣，在我们看来这是一个胜利。
