# 机器人+小号=悲伤长号. mp3

> 原文:[https://hack aday . com/2018/10/22/robot-小号-sad-长号-mp3/](https://hackaday.com/2018/10/22/robot-trumpet-sad-trombone-mp3/)

[尤里·沙克德]真的很喜欢拉丁音乐。当他的兴趣渐浓时，他买了一把小号，以便自己演奏一些充满活力的曲子。当他意识到从这个东西中获得可靠的喇叭状声音有多困难时，他的热情有所减退，但他并不打算完全放弃。Geekcon 2018 临近，于是他想，[为什么不给我做一个会吹小号的机器人](https://medium.com/@urish/we-tried-to-build-a-robot-that-plays-the-trumpet-and-happily-failed-c9ca1d6063b8)？

他在网上搜了一下，发现 20 年前还有人煞费苦心地用一双乳胶唇模仿口型(想想橡胶手套手指盛满水)。另一个灵魂写过关于测量铜管乐器的气流。有了这些信息，[Uri]和合作伙伴[Ariella]和[Avi]花了几个小时摆弄气泵、乳胶和水，想出了一个听起来像——而且[Uri]的描述非常准确——一个坏掉的机器人迪吉里杜管的概念证明。它的工作，但声音断断续续。

快进到极客大会。灵光一现，[Avi]想到在方程中加入电容。他建议他们用一个塑料盒作为空气的缓冲，这很有效。[Ariella] 3D 打印了一些手指来驱动阀门，但该团队最终以伺服系统驱动的木制手指告终。机器人喇叭的设置只持续了足够长的时间来获得一个视频，然后一个伺服系统迅速烧毁。哇哇哇。休息之后，请闭上你的嘴。

如果[Uri]厌倦了这个东西，他总是可以把它变成一个游戏控制器，就像小号英雄一样。
