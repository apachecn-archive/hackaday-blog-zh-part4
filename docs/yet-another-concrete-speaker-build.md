# 又一个具体扬声器构造

> 原文：<https://hackaday.com/2019/05/17/yet-another-concrete-speaker-build/>

如果你想便宜地制造一些重的东西，混凝土是很好的选择。[Marek Unger]决定试一试，[用这种材料铸造一个家庭高保真音响设备的音箱(Youtube 链接，嵌入下面)](https://www.youtube.com/watch?v=KDnyZ2EA5xc)。

最初的尝试包括创建一个激光切割的中密度纤维板外部模具，内部有一个聚苯乙烯泡沫塑料芯，稍后将被移除。这并不成功，[Marek]进一步发展了这一设计。第二次修订使用了同样由激光切割中密度纤维板制成的内核，设计用于铸造后留在内部。这种内部模具已经包括扬声器驱动器的安装孔，使组装也更容易。

铸造完成后，外壳上安装了唐带 W4-1320SIF 驱动器。这是一个全范围的驱动器，这意味着它们可以在不需要分频器或其他扬声器来填充频率范围的情况下使用。每个机柜的重量刚刚超过 10 公斤，它们的端口在较低的频段有额外的响应。声音测试令人印象深刻，最终产品的粗糙美感在[Marek]的客厅里看起来很棒。

我们已经看到混凝土被用于各种项目，从[熔炉](https://hackaday.com/2019/05/04/how-to-build-a-small-metal-furnace-at-home/)到 [USB 集线器](https://hackaday.com/2019/04/16/concrete-usb-hub-isnt-going-anywhere/)。休息后的视频。
