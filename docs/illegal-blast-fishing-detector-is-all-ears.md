# 非法爆破捕鱼探测器洗耳恭听

> 原文：<https://hackaday.com/2020/08/05/illegal-blast-fishing-detector-is-all-ears/>

爆破捕鱼——在水下使用炸药，通过冲击波杀死整群鱼的行为——几十年来一直是菲律宾的一个普遍问题。虽然一些渔民在新地点的第一次爆炸中发了财，但从长远来看，这对任何人都没有好处，尤其是对珊瑚和其他海洋生物。许多爆破渔民使用自制炸药，经常冒着失去手指和四肢的危险。

地方当局尝试了许多策略来阻止这种活动。在关于爆破捕鱼收益递减的教育失败的地方，用战略性放置的圣母玛利亚雕像来吸引宗教已经取得了一些成功。[【Ifthekar ahammad】有另一个想法，它涉及到检测爆炸，三角测量爆炸的位置，并尽快向当局报告](https://hackaday.io/project/174076-cbobby)。

CBobby 系统通过分析音频频谱来工作。它从环境背景噪音水平中寻找瞬时变化，并分析持续时间和听到的频率，以确定是否发生了爆炸。在菲律宾进行实地测试的计划已经被疫情打破，但是[ifthekar]一直在德国努力工作，用水下扬声器发出爆炸噪音进行测试。该系统已经可以将爆炸与各种环境声音区分开来，如船只、大型海洋生物的吼叫、地震、下雨和打雷。

尽管测试设备被包裹在霓虹橙丙烯酸树脂中，但实际的爆破捕鱼塔特勒将被伪装成有毒的石鱼，使其像火蚁或黄蜂一样吸引人。

破坏性的爆破捕鱼是非常可怕的，尽管这是为了生存。但是我们如何看待使用无人机捕鱼作为运动呢？
