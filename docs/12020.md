# 真的需要干灯丝吗？

> 原文：<https://hackaday.com/2021/11/29/do-you-really-need-to-dry-filament/>

围绕 3D 打印材料的储存和干燥有很多观点和理论。有些人绝对相信，如果灯丝被存放在密封袋外，即使只有几天，你也必须烘烤灯丝。还有一些人“从未遇到过问题”。因此，是时候让一些知情人士做些测试，试图确定我们都在问的问题的答案了；[湿丝真的有多烂](https://toms3d.org/2021/11/23/how-bad-is-wet-filament-really/)？

[Thomas Sanladerer]建立了一个简单的实验，使用三种常见类型的细丝样品，特别是 PLA、PET-G 和 ASA。他将样本存放在三个环境中，桌子上，室外花园里，最后浸入水中整整一周。随之而来的是大量的印刷，但他们都*做了*印刷。

不同的细丝吸水的速度不同，这取决于它们的化学成分和环境，尼龙显然特别喜欢良好的浸泡。随着吸水率的增加，最明显的印刷缺陷似乎是拉丝，除了令人讨厌和降低表面质量之外，它在总体上并不那么严重。有趣的是，吸水性似乎并不影响最终部件的强度。
 从视觉上来说，老好人 benchy 是一个很棒的测试模型，它在一个可爱的小模型中提供了许多用于评估打印质量的功能。[Thomas]特别注意到，模型中挤压机需要减速的区域，打印质量会变差。他总结说，这是因为灯丝在高温下停留的时间更长，使得水沸腾的时间更长，对印刷的影响更大。

丝间性能似乎表明 PLA 受影响最小，而 PET-G 受影响最大。总之，PLA 在你的桌子上踢来踢去似乎很好，而你确实需要用 ASA 和 PET-G 做一点干燥护理。
