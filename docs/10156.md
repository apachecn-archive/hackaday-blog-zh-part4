# 一把链锯让这台绞盘发出一些咕噜声

> 原文：<https://hackaday.com/2021/05/31/a-chainsaw-gives-this-winch-some-grunt/>

对于一个令人满意的 Youtube 观看会话来说，没有什么比一些高质量的机械车间工作更好的了，这正是[波兰制造]提供的[他们将一个 12V 的小绞车转换成电锯的动力](https://www.youtube.com/watch?v=lctXHA5WgjE)。成品除了原始的齿轮箱和轴部件之外，并没有太多的东西，但生锈的钢材被改造成尺寸完美的部件，组装成一个全新的组件，这一迷人的景象总是引人入胜。

转换从移除和拆卸电机开始，以显示其轴和滚筒的锁定机构。然后将轴向下转动，并制造一个轴环将其连接到链锯上的驱动花键上。我们很高兴地看到，链锯在这个版本中没有修改，而是简单地拧开了刀片，并以可逆的过程连接了绞盘。最后，原始的鼓被认为对于应用来说太小，所以制造新的鼓。我们在一个波兰农场看到了结果，他们高兴地参与一些林业工作，甚至在他们的皮卡车卡住的时候拉着它。

这绝不是我们第一次在这些页面上介绍[Mad in Poland]，尤其是这个电磁切圆夹具。
