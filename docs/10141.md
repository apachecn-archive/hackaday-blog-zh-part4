# 一个可堆叠的行星齿轮箱，你可以在家里打印

> 原文：<https://hackaday.com/2021/05/14/a-stackable-planetary-gearbox-you-can-print-at-home/>

在 YouTube 的一个小角落里，有一个小型但充满活力的社区，分享关于他们自己设计的变速箱的视频，特别是那些传动比非常高或具有其他古怪功能的变速箱。亚文化的追随者被称为*齿轮头*，【让我们打印】就是其中之一。他的最新作品是 3D 打印的行星齿轮箱设计[，重点是易于组装和多种传动比选择。](https://www.youtube.com/watch?v=McfVmBNaNuA)(视频，下面嵌入。)

随着[让我们打印]越来越厌倦为他们的每个项目设计定制的齿轮系，齿轮箱应运而生。他们着陆的行星设计具有可堆叠的优势，每个减速块都为火车增加了 1:3 的减速。

出于测试目的，将四个阶段组合起来，总减速比为 1:81。由此产生的变速箱能够在输出耦合器失效前提升 40 公斤，这对一些从热喷嘴喷出的塑料来说是一个不小的壮举。这是由塑料制成的大传动比齿轮箱的常见问题——通常，齿轮箱的组件无法承受产生的巨大负载。

不考虑材料的限制，我们确信齿轮箱将在未来的项目中被证明是有用的。我们以前也见过其他坚固的 3D 打印变速箱，[比如这个名副其实的【为了什么放下齿轮】](https://hackaday.com/2017/07/27/3d-printed-gearbox-lifts-an-anvil-with-ease/)中的铁砧升降机。自己一头扎进网上变速箱亚文化。
