# 鹈鹕案件成为重击蓝牙扬声器

> 原文：<https://hackaday.com/2019/10/25/pelican-case-becomes-thumping-bluetooth-speaker/>

鹈鹕案件是伟大的，如果你需要一个坚固的外壳来保护一些敏感的齿轮。它们也是高度可修改的，[并且可以为蓝牙扬声器的构建做一个很好的基础。](https://www.instructables.com/id/DIY-Pelican-1050-Bluetooth-Speaker-Dayton-Audio/)

像许多现代版本一样，这在很大程度上是将一系列现成的模块连接成一个更大的整体。Tinyshine 蓝牙音频板连接到代顿音频 D 类放大器。D 类放大器具有紧凑的尺寸和良好的功效，是任何便携式音频应用的绝佳选择。电源由手工制作的 3 芯 18650 电池组提供，同时标准降压转换器和电池保护板被替换以确保电池保持良好状态。

不想克扣音频质量，安装了一对代顿音频全频驱动程序，消除了交叉安装或每个通道多个驱动程序的需要。在背面还有第三个被动驱动器，尽管我们不是 100%清楚它的用途。如果你有任何线索，请在评论中告诉我们。

这是一个项目，作为一个伟大的蓝图，任何人都想建立自己的高保真蓝牙扬声器。相关模块都是现成的——只需将它们连接到一个漂亮的功放和一套像样的扬声器上。设计完全由你决定——无论你是选择烟斗、T2、包包还是其他什么。黑客快乐！
