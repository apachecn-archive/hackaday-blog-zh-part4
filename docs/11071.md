# 任何东西都不应该影响这个易操作的天气小部件的构建

> 原文:[https://hackaday . com/2021/08/23/nothing-should-cloud-the-build-of-this-wyew-weather-widget/](https://hackaday.com/2021/08/23/nothing-should-cloud-the-build-of-this-wieldy-weather-widget/)

天气是黑客们似乎永远感兴趣的事情之一。我们可能会谴责可以准确预测两天天气的想法，更不用说七天了，但是如果有一个更长的预测可用，天哪，我们会看一眼的。

那么，为什么每次你想查看温度时，都要拿起手机或打开浏览器标签呢？如果你真的很喜欢它，你应该做一个桌面天气小工具。[opengreenenergy]已经写了一个很好的指南来整洁地构建这个经典且非常有用的项目，涵盖了从焊接到获取 API 密钥的所有内容。里面是一个 ESP8266 和一个 2.8 英寸的触摸屏显示器，通过打开的天气地图显示当地的情况。主屏幕显示时间、日期、当前天气、7 天预报和每天的月相，随后的屏幕显示进一步的细节。不忙也能增长见识。

我们喜欢[卡扣式外壳](https://www.thingiverse.com/thing:4928974)的流线型外观。这可能是一个相当简单的项目，但由于内部空间的限制，按设计建造是具有挑战性的。休息之后，请观看视频，其中展示了古老的老虎钳。
