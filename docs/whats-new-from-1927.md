# 有什么新鲜事，1927 年

> 原文：<https://hackaday.com/2023/01/02/whats-new-from-1927/>

新年伊始，对于互联网档案馆来说，这意味着刚刚进入公共领域的记录。1927 年最好的以前受版权保护的材料现在在公共领域公开出售，这意味着有一个电影、书籍和音乐的宝库可以自由复制和混合。

他们的文章突出了 1927 年流行文化的几个值得注意的部分，如果你对科幻感兴趣，我们建议你一定要注意一下弗里茨·兰的*大都市* ，但对于普通读者来说，文章本身没有太多其他与技术相关的内容。[钻研 1927 年](https://archive.org/details/books?query=date%3A%5B*+TO+1927%5D&sort=-date)的档案仍然是一种迷人的消遣，因为除了看看现在免费的东西之外，它还引领我们进入了 20 世纪 20 年代的技术状态。在这里，我们发现自己在英语语言中导航就像在图书馆中导航一样多，因为我们做的很多事情都使用了几十年前的词汇。

1927 年的电子学很大程度上涉及到使用真空管的无线电，因此找到当年的教科书相对简单，如[](https://archive.org/details/in.ernet.dli.2015.493157/page/n3/mode/2up)**和 [*无线电波的传播*](https://archive.org/details/in.ernet.dli.2015.212389/mode/2up) ，以及一叠与之配套的[无线电期刊](https://archive.org/details/periodicals?query=date%3A%5B*+TO+1927%5D+radio&sort=-date)。我们所知的计算机根本就不存在，我们用来描述自动化任务的机器的一些词汇当时似乎也没有使用。像“机电”或“序列器”这样的词没有任何用处，但是有大量关于重型电气工程的书籍。 [*电厂工程*](https://archive.org/details/engineeringpowe00orrogoog/page/n12/mode/2up) 布景，我们不得不承认在 *[电动火车](https://archive.org/details/in.ernet.dli.2015.211638/page/n5/mode/2up)上花了不少时间。*如果我们花了一些时间浏览存档并发现了许多有趣的内容，我们猜想您也会感兴趣。**
