# 用感性的太阳镜挑战你对现实的认知

> 原文：<https://hackaday.com/2018/10/06/challenge-your-perception-of-reality-with-emotional-sunglasses/>

因《搭车人指南》而出名的对危险敏感的太阳镜直接影响使用者对刺激的反应，在面对危险时变得完全不透明。这是个好主意，但是如果太阳镜会影响你的情绪呢？[这就是今年黑客奖](https://hackaday.io/project/160615-emotiglass)中的情绪眼镜项目正在做的事情。这个概念允许计算机改变用户对现实的情感感知。

“情绪眼镜”背后的关键思想来自伦敦大学的一名研究人员今年发表的一篇论文。显然，你对图像的情绪反应是可以控制的，这取决于图像出现在你心跳周期中的时间点。例如，研究人员发现疼痛的感知取决于刺激在心动周期中的传递点。

为了用一些开源硬件来测试这一假设，[David Prutchi]和[Jason Meyers]创造了一副带有液晶镜片的太阳镜，这种镜片可以是透明的，也可以是不透明的。加上检测心动周期的心电图传感器和将一切联系在一起的微控制器，你就得到一个在情感上相当于对危险敏感的太阳镜的设备。

这是一个伟大的项目，它赢得了 1000 美元，进入了 Hackaday 奖的决赛，我们很自豪这个项目有望获得 50，000 美元的大奖。
