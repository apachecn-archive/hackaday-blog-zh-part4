# 一元店焊接帮手

> 原文:[https://hack aday . com/2018/10/21/dollar-store-helping-hands-for-soldering/](https://hackaday.com/2018/10/21/dollar-store-helping-hands-for-soldering/)

虽然[我爱制造]似乎有中文文本，但他们最近的视频(见下文)就像一个无言的研讨会，所以不管你是否精通普通话。焊接钳看起来主要来自一元店(或者雅恩店)。

据我们所知，该组件是两个实用夹，就像你可能会在软木板上使用或密封芯片，微型 SIM 卡切割器和电视兔耳朵。哦，还有一个注射器。兔子耳朵在构建过程中大多会被破坏。你还必须做一些切割和塑料熔化工作(我们可能用了钻头)，但是用一些简单的手工工具，你就可以做任何事情。他们没有表现出来，但很明显，他们在模拟刀具上钻了一个孔，所以你无论如何都需要一个钻头。

对于垃圾盒构建来说，结果看起来相当不错。我们喜欢它可以折叠存放的方式。大约在 3 点 50 分的时候，我们以为音乐结束了，但是——唉——它又开始了。你可能想让你的扬声器静音。

如果你想知道注射器是干什么用的，它的一半装着焊线轴，另一半作为焊料的导向。我们不确定为什么 SIM 卡切割器是底座。看起来你可以使用任何重的东西，甚至是可以夹在工作台面上的东西。然而，你不太可能完全按照他们的方式来做，因为你的零件会有一点不同。

我们喜欢工件夹具，如果它们[是 3D 打印的](https://hackaday.com/2018/01/24/printed-it-rubber-band-pcb-vice/)或者[非常简单](https://hackaday.com/2015/06/13/review-stickvise-needs-a-place-on-your-bench/)。毕竟，如果你要建造一个东西，在你工作的时候，它需要保持静止。
