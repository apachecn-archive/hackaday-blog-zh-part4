# 为无线电控制汽车制造锻造碳纤维机翼

> 原文:[https://hack aday . com/2022/02/02/02/building-forged-cars-cars 的碳纤维翼/](https://hackaday.com/2022/02/02/building-forged-carbon-fiber-wings-for-radio-control-cars/)

当涉及到建造像样的空气动力学装置时，你要专注于获得精确的几何形状，并确保你的部件足够强大，以应对它们产生的力。来自[下班后工程]的这款车在这些方面做得很好，[由一个小型遥控赛车的高下压力翼组成。](https://www.youtube.com/watch?v=3kPTGFGMoZI&feature=emb_title)

该视频指出，即使是一辆像样的遥控车，在最好的情况下，工厂也会有非常糟糕的空气动力学部件，升阻比(L/D)最多为 2-3:1。这意味着，虽然它们可能会产生少量的下压力，但同时也会产生大量的阻力。

这里设计的双元件机翼效率更高，L/D 比接近 17:1，这是一个巨大的进步。即使是漫不经心的人也能注意到，这个设计看起来更像你在全尺寸汽车上看到的东西，而不是玩具上看到的一些古怪的设计。

机翼采用 3D 打印模具，采用锻造碳纤维工艺制造，赋予机翼足够的强度。鉴于它是为一辆时速超过 100 英里的遥控汽车制造的，确保机翼足够坚硬以适应高速行驶是关键。

[下班后的工程]很好地展示了如何准备模具，用碳纤维填充模具，并浇注树脂，并讨论了大量关于如何通过锻造碳工艺获得良好结果的有用提示。

结果是一个令人难以置信的令人印象深刻的尾翼，其空气动力学性能与其良好的外观相匹配。这可能比 3D 打印更复杂，但这项工作的结果要艰难得多。
