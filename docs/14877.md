# Python 黑客的电子表格

> 原文:[https://hack aday . com/2022/09/23/a-spread sheet-for-the-python-hacker/](https://hackaday.com/2022/09/23/a-spreadsheet-for-the-python-hacker/)

你可以写一个 Python 程序或者使用 Jupyter 笔记本来做几乎任何事情。但是你也可以使用电子表格快速完成很多事情。Grist 是一个“黑客”的电子表格，它融合了这些领域。它看起来像一个电子表格，但下面是 SQLite 表，公式语言是 Python。

代码是[开源的](https://github.com/gristlabs/grist-core)，如果你想托管它，有免费和付费的计划。你甚至可以[在不登录的情况下试用](https://docs.getgrist.com)，要么从空白屏幕开始，要么使用模板。下面可以看到一个介绍视频。

如果你擅长电子表格，请做好准备。Grist 不完全是一个电子表格——它更像是数据库上的一个类似电子表格的界面。因此，如果您有一个包含数量和价格的行，您可以添加一个计算总价的列。这很好，但是您现在已经对表中的每一行都这样做了。根据你的观点，这可能是一个特点。如果你想要底部的总数呢？然后添加一个小部件，它实际上是一个单独的表。

您可以派生电子表格并协调副本，有点像合并受源代码控制的程序。您可以轻松地制作仪表板和不同的数据视图。例如，如果你想知道哪只山羊生了哪只山羊… [这里有一个关于这个](https://docs.getgrist.com/iqhzhQM66qg7/Pedigree-Example/m/fork/p/2#a1.s3.r11.c2)的例子。因为这是黑客日，你可能更喜欢一个[莫尔斯电码测验](https://docs.getgrist.com/qctFijX9HUZu/Morse-Code/m/fork/p/2)。

这看起来像是在编码一切和被大肆吹嘘的“无代码”工具之间的一个很好的平衡。老实说，我们和朱庇特在一起会很舒服。但是这种类似于电子表格的事实会使它在某些情况下更受欢迎。

我们确实喜欢古怪的电子表格软件。另一方面，我们也喜欢[滥用传统的电子表格](https://hackaday.com/2020/11/13/dsp-spreadsheet-the-goertzel-algorithm-is-fouriers-simpler-cousin/)。
