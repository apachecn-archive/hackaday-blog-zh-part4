# 适合各种场合的摇旗帽子

> 原文：<https://hackaday.com/2020/04/28/a-flag-waving-hat-for-all-occasions/>

当[品尝代码]看到他的 YouTube 频道接近 1000 名订户时，是时候做点特别的事情了。但是庆祝不是浪费的理由。这顶挥舞着旗帜庆祝的帽子对未来有着无限的可能性。

建造很简单，正适合这些奇怪的稀缺时期。一个 Arduino Uno 热粘在帽子后面，直接驱动前面的一对 9g 伺服系统。[品尝代码]通过将两个贴纸背靠背贴在一起，中间夹一根竹签来制作旗帜。代码的特色是旗帜会以三种随机选择的模式中的一种挥舞——左右摇摆、反向摇摆和狂野手势。

在 1k subs 的新鲜感消失后，[品尝代码]可以将旗帜换成 Jolly Rogers，以帮助保持社交距离。将来有一天，当情况真的好转时，它们可以被换成新型冠状病毒胜利旗帜，或者悬挂当地运动队的旗帜。我们认为将某种真正的信号量消息编程到标志中会很酷，尽管移动性可能太有限了。休息之后，请观看构建视频，这是一个画中画的过程，因为[品味代码]推出了一个频道回顾，并为未来奠定了基础。

即使 YouTube 搞乱了用户数，我们认为它仍然值得做一个很酷的计数器。这里有一个扭曲的俄罗斯方块。
