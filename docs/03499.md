# 黑客日链接:2019 年 7 月 7 日

> 原文：<https://hackaday.com/2019/07/07/hackaday-links-july-7-2019/>

比如模块化合成器？你当然需要，而且你需要另一个洞来投钱！在过去的几个月里，Supplyframe 一直在旧金山举办合成器和电子音乐聚会。本周，HDDG/Piqued meetup 将与 VCV Rack 的创造者进行一场精彩的会谈。VCV 机架是一个开源的，虚拟的，模块化的合成器——基本上是一堆 Eurorack 模块*在一台计算机*里面，它的成本要低得多。这个演讲是本周四晚上在旧金山举行的。你应该来！

W600 是一个新模块([你可以从 Seeed](https://www.seeedstudio.com/w600.html) 获得，尽管它是由 Winner Micro 以各种格式生产的)，它基本上是一个 ESP32，只是它使用了 ARM Cortex-M3 而不是 Tensilica 内核。[ultratechie]最近得到了其中一个模块，[开始使用 MicroPython](http://www.ultratechie.com/projects/w600-micropython/) 。这似乎是一个功能强大的模块，它只有 3 美元，但这足以赶上 ESP32 吗？

紫色大猩猩进入美术馆。在鹿特丹 Het Nieuwe 研究所[有一个新展览，展示“计算机病毒的破坏性之美”](https://malware.hetnieuweinstituut.nl/en/press/malware-exhibition-reveals-destructive-beauty-computer-virus)。策展人正在详细介绍计算机病毒的历史进程，从无害的 DOS 病毒到*梅丽莎*到*震网*和勒索软件。

USB C 已经存在了一段时间，但 2019 年是一切开始成为 USB C 的一年。恰当的例子是:树莓 Pi 4。[唯一的问题是 Raspberry Pi 基金会搞砸了他们对 USB C](https://hackaday.com/2019/06/28/power-to-the-pi-4-some-chargers-may-not-make-the-grade/) 的实现。没问题，因为[这里是你如何设计一个 USB C 电源接收器](https://medium.com/@leung.benson/how-to-design-a-proper-usb-c-power-sink-hint-not-the-way-raspberry-pi-4-did-it-f470d7a5910)。基本上，你给每个 CC 线自己的电阻。想都别想，直接抄 USB C 规范。你对 USB C 的了解并不比设计它的人多，删除一个电阻并不能真正节省一大笔钱。照抄说明书就行了。
