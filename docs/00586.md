# 只有两个电机的鞋带系紧机器人

> 原文：<https://hackaday.com/2018/09/08/shoelace-tying-robot-with-only-two-motors/>

人类做的很多事情，机器都很难。例证:系鞋带。想想手指交叉的复杂舞蹈，从一只手到另一只手传递花边。所以当加州大学戴维斯分校的五名学生组成的团队一起建造了一台机器，它用两个钩子，一些非常聪明的齿轮和两个马达完成了工作，我们不得不说我们印象深刻。[在 Youtube 上观看它的运行](https://www.youtube.com/watch?v=erNi07dH5pw)(也嵌入在下面)。

双电机约束起初似乎是一个阻碍，但现在我们已经看了大约一百遍视频，我们非常确信一个足够聪明的机械工程师可以用两个电机和足够的齿轮做几乎任何事情。你看，秘密在于，一个马达专门用于将驱动齿轮来回移动到多个目的地，另一个马达提供动力。

今天是黑客日，我相信你们中的一些人会说“我可以用一个马达做到这一点！”把这当成一个挑战。

同时，如果你需要看更多的齿轮色情，看看这个蜂鸟机器人。或者看看凸轮驱动机器的奇迹，看看【弗兰布兰奇】用 [Maillardet 自动机](https://hackaday.com/2016/12/15/fran-blanche-goes-in-depth-with-the-maillardet-automaton/)做的工作。
