# 酷酷的工具:Pantorouter 打开跟踪功能

> 原文:[https://hack aday . com/2018/10/01/cool-tools-the-panto router-turns-tracing-on-the-side/](https://hackaday.com/2018/10/01/cool-tools-the-pantorouter-turns-tracing-on-its-side/)

不久前，我们写了一个小的 CNC 工具，用于自动化木工过程的某些部分。当时，它故意限制范围似乎不太寻常，但一些评论者提到，这让他们想起了另一个设备，[【马蒂亚斯】的 Pantorouter](https://woodgears.ca/pantorouter/) 。不需要太多的调查就能看出[的评论者](https://hackaday.com/2018/08/28/a-cnc-woodworking-tool-that-does-the-hard-parts/#comment-4953697)是对的！火柴棍设备确实感觉有点像 CNC 版的 Pantorouter，而且它似乎比一篇帖子本身更有价值。Pantorouter 是另一种小型手动但自动化工具的迷人范例，它经过优化，可加速和改善某些木工作业。

![](../Images/0ac41402b9a24d8afce43501edcd7ea3.png)panto router 是一种称为[缩放仪](https://en.wikipedia.org/wiki/Pantograph)的机械装置的具体应用，该装置是一种通用连杆，可用于两种用途:引导线性运动和跟踪图案，同时放大或缩小图案。(剪式升降机是第一次使用的例子。)不需要在维基百科条目中进行太多搜索就可以看出，Pantorouter 肯定不是第一种使用这种机制的机器工具，甚至不是第一个这样做的路由器。在你最喜欢的搜索引擎上搜索[“受电弓路由器”](https://duckduckgo.com/?q=pantograph+router)，会出现一大堆工具、模板和产品来建造这样的设备。但是这些通常用于放大 2D 印刷模板以印刷精美的字体等。这就是受电弓与众不同的地方；它是为切割细木工而设计的。

典型的缩放仪的工作原理是引导一个节点围绕一个形状的周边放大或缩小，同时另一个节点以不同的幅度跟随。因此，操作者可以在工具架跟随的同时引导模板周围机构的一部分。Pantorouter 基本上是一个缩放器，其工作方式相同。用户插入一个他们想要切割的形状的模板，打开刳刨机，引导其多次通过，去除余量，直到余量与模板的形状匹配，并且没有什么要切割的。Pantorouter 的切割头不是平放在表面上，而是直立在边缘上，刳刨机刀头平行于滑板平面延伸。很难描述，但很容易看到。看看这张照片，你就知道了。模板放在手柄和路由器之间，照片中[马提亚斯]指向的地方。此图显示了一个原始的 Pantorouter，带有一个用于切割榫眼的模板。当用户移动手柄时，它沿着模板件的内部或外部引导轴承。组件的这一部分安装在由第二个杠杆控制的滑板上，允许它被精确地导入和导出以控制切割深度。床下有可配置的挡块来调节最大和最小深度。原始机器的几何形状设计为从模板到库存固定缩减一半。

![](../Images/d196b13201c5e8b7507a5cd68fecd2c2.png)排屑器最初演示切割[榫眼和榫头](https://en.wikipedia.org/wiki/Mortise_and_tenon)接头。[马提亚斯]指出，用圆柱形钻头切割榫眼的简单方法是留下一个圆形的槽。一个矩形的凸榫将适合这样一个空隙，但并不紧密，为了完美的配合，凸榫必须被切割成一个匹配的圆形突起。使用 Panorouter 可以轻松完美地切割匹配的榫眼和榫头。一旦一个模板对被完善，它可以被无限地复制并保持同样的精确匹配。在这一点上，你甚至不需要测量它！只需夹一块原料，然后切割。但是没有理由就此止步。多重凸榫和榫眼？没问题！这只是一个新的模板，可以重复使用，每次都非常适合。需要定位销伸出你的股票？你可以在 Panorouter 上将它们切成圆柱形突起。因为模板的尺寸是成品的两倍，所以比以 1:1 的比例切割模板更容易制作。

从最初的 Pantorouter 开始，已经有过几次迭代。[马提亚斯]制造了一个加大号版本的,它更大一点，旨在使用在北美以外更容易找到的路由器。他还有一个[更传统的定向刳刨机/缩放仪](https://woodgears.ca/pantograph/index.html)，它更适合描摹模板，尽管它也是铰接的，以便切割 3D 设计。所有的计划都可以用来建造你自己，一些挖掘发现了一些预制的设计，可以购买组装或作为套件。
