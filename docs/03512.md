# 这个 Chiptune 演奏者有 Nintendon 没有的

> 原文：<https://hackaday.com/2019/07/06/this-chiptune-players-got-what-nintendont/>

当谈到 chiptunes 时，最初的任天堂娱乐系统和 Game Boy 获得了所有的赞誉。OPL 合成器拥有所有的乐趣。但是，在布满灰尘的旧机器中还有另一个芯片，它至少与《超级男人 2 OST》一样有趣，至少与《超级男人 2 OST》一样有影响力。是 YM2612，世嘉 Mega Drive/Genesis 里的芯片。

[natalie]创造了一个便携式设备，能够播放针对这台古老机器中声音芯片的文件。这是 megagrll，这是 iPod 的原始创世纪音轨。

在 MegaGRRL 内部是一个 ESP-WROOM-32 模块形式的 ESP32。当然，里面有一个 YM2612 芯片，还有一个耳机放大器和一个电池充电器。显示屏是一个相当标准和便宜的东西，全彩色 240 x 320 像素，这个设备上有七个按钮，因为当然你需要一个 A，B，*和* C 按钮。

结合 3D 打印外壳，GameGRRL 确实做到了它所说的:它播放所有来自旧世嘉游戏的音乐。现在，当你不可避免地与某人争论迈克尔·杰克逊创作了《索尼克 3》的原声带时，证据就在你的口袋里。如果你想在脚趾果酱和厄尔原声带果酱，这也是正确的。你可以看看下面的视频演示。
