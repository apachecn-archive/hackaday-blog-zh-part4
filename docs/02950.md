# LED 面板灯是使用原型板的好方法

> 原文:[https://hack aday . com/2019/05/13/led-panel-lamp-is-a-great-way-to-use-proto board/](https://hackaday.com/2019/05/13/led-panel-lamp-is-a-great-way-to-use-protoboard/)

现在可以在一个简单易用的封装中采购具有巨大光输出的板上芯片 LED 模块。然而，它们可能有较大的功率需求，并且较便宜的模块也容易出现死点。[Heliox]用老派的方式组装了一个很棒的 LED 灯设计，展示了完成工作的方法不止一种。

标准的 SMD 发光二极管是今天的订单。led 整齐地排列在原板上，便于焊接。这也使得电力分配变得不在话下，铜走线将电力输送到每一行。电力由安装在 3D 打印外壳背面的 18650 锂电池提供。GoPro 风格的底座作为外壳的一部分印刷而成，允许灯以多种方式轻松安装。

这是打造多功能 LED 灯的一种快速、廉价且简单的方法。随着扩散器的安装和集成 USB 充电，我们可以看到这是一个非常好的便携式设备，适合移动摄像师或技术人员。[我们以前也见过[Heliox]的 LED 产品](https://hackaday.com/2018/12/29/infinity-cube-is-gorgeous-yet-simple/)。休息后的视频。
