# 寻宝与少数常见组件

> 原文：<https://hackaday.com/2021/11/15/treasure-hunting-with-a-handful-of-common-components/>

有时候越简单越好——当你不需要板载微控制器的计算能力时，通常最好依靠简单的电路来完成工作。由于到处都是廉价的 Raspberry Pis 和 ESP32s，人们很容易忘记许多更简单的项目可以在没有一行代码的情况下完成(随着芯片短缺的持续，记住这一点可能比以往任何时候都更重要)。

当[mircemk]建造他的简单感应平衡金属探测器时，他有了正确的想法。它使用一对 555 定时器、晶体管和无源器件，通过一圈电线来感应金属物体的存在。他能够探测到 15 厘米远的硬币和 60 厘米远的更大的物体——对于你现在可能在你的工作台的备件抽屉里的一堆组件来说，这已经不错了！检测机的选择性可以通过几个电位计来调节，在真正的金属检测机中，一旦发现什么东西，它会有一个蜂鸣器大声向你鸣响(还有一个 LED，以防蜂鸣器变得太烦人)。

总而言之，这个金属探测器看起来是一个非常有趣的项目——非常适合初学者和经验丰富的黑客。这是一个很好的提醒，不是每个项目都需要 WiFi 或有机发光二极管显示器才有用，但不要让这阻止你做得太多！如果触摸屏更适合你， [[mircemk]也为你提供了智能手机集成版](https://hackaday.com/2020/05/02/a-smart-diy-metal-detector/)。
