# 任何东西都会变成时钟

> 原文：<https://hackaday.com/2020/08/15/anything-becomes-a-clock/>

钟表在这里是一个很受欢迎的项目，而且有很好的理由。有大量的选择，而且总是有新的看时间的方式。使用激光、文字甚至滚珠轴承的时钟都是显示时间的非典型方式，但就像数学家在寻找一个长期理解的想法的一般证明一样，这个来自[juldozer][的时钟向我们展示了一种将任何物体变成时钟](https://www.thingiverse.com/thing:4557021)的方法。

他的建筑使用 AA 供电的时钟机芯，你可以在任何典型的挂钟上找到，而不是达到他的 Arduino 和步进电机的解决方案。驱动指针进行这些运动的电机扭矩极低，功率极低，这使得它们能够在如此小的电源下持续如此长的时间。他使用其中的两个，一个用于小时，一个用于分钟，他在上面安装了一个定制的 lazy Suzan。转盘需要极低的摩擦，以避免他不得不每天更换电池的情况，因此在一些 3D 打印后，他有两个可以容纳任何物体的旋转板，以告诉他当前时间。

虽然他没有从零开始设计时钟或重新发明任何其他轮子，但这个项目的亮点在于他能够利用这样一个低功率电机来转动如此沉重的东西。这可能在计时领域之外有很好的用途，这让我们想起了去年 Hackaday 奖的这个 [3D 打印齿轮组](https://hackaday.com/2019/05/08/pushing-3d-printed-wheels-and-transmissions-to-the-limit/)。
