# 你总是可以用 ATtiny 来代替 555

> 原文:[https://hack aday . com/2021/12/15/you-can-always-use-an-attini-inst-a-555/](https://hackaday.com/2021/12/15/you-can-always-use-an-attiny-instead-of-a-555/)

这是为 Hackaday 写的一个常量，每当一个项目使用 555 定时器出现时，有人会说“你可以使用微控制器来做到这一点！”。这就是[Shranav Palakurthi]在 ATTiny555 中采用的方法，[这是一个通过巧妙利用不起眼且无处不在的微控制器芯片来模拟整个 555](https://hackaday.io/project/183013-attiny555) 的项目。我们都曾为此感到内疚，但现在，ATTiny85 的爱好者们终于有了确凿的证据，证明他们最喜欢的一块廉价硅可以证明它的勇气。

巧妙的 555 替代品[的全部细节可以在其 GitHub 库](https://github.com/shraiwi/ATTiny555)中找到，对于那些愿意冒险的人来说，这就像添加一个电阻和更新固件一样简单。它不是 555 的完美替代品，模拟性能不完美，重置和接地引脚互换，但它确实带来了较低电源电压的优势。

你可以在休息时的 YouTube 视频中看到该设备的运行，但同时也为终于有办法用你最喜欢的廉价 8 引脚芯片取代所有不必要的 555 而高兴！

当我们谈到 555 的主题时，不要忘记[我们将再次进行我们的 555 竞赛](https://hackaday.com/2021/12/01/the-555-timer-contest-returns/)。
