# 观看一艘船被焊接在一起是 ASMR 工作室

> 原文:[https://hack aday . com/2022/07/11/watching-a boat-get-welded-together-is-workshop-asmr/](https://hackaday.com/2022/07/11/watching-a-boat-get-welded-together-is-workshop-asmr/)

如果你已经在互联网上呆了足够长的时间来了解 Hackaday，我们敢打赌你一定熟悉自主感觉经络反应(ASMR)的概念——头皮上的刺痛感觉，据说是由某些听觉刺激触发的。YouTube 上有无数视频承诺给你“刺激”，使用的东西从鸡毛掸子到过于浆洗的衬衫，但对我们来说，选择的工具显然是[鲍勃]手中的[林肯电动马格南 PRO 100SG 短管枪。](https://www.youtube.com/watch?v=JbH2SPmVmFM)

[![](../Images/18ad0caafeb8863a3438e5dd3fb93212.png)](https://hackaday.com/wp-content/uploads/2022/07/jetboat_detail.jpg)

You’ll want a friend to help wrangle the panels.

诚然，我们不能保证最新的*制作材料*视频会给所有观众带来愉悦的身体感觉，但至少，我们认为你会同意，看着【鲍勃】和他的兄弟有条不紊地将 12 英尺长的船体焊接在一起，最终将成为一艘定制的喷气船，是一种奇怪的放松。

虽然我们通常将[Bob]与 scratch builds 联系在一起，但这次他实际上正在通过一个商业工具包来工作。该套件由 Jet Stream Adventure Boats 出售，包括构成船体、桁条和顶部甲板的预切割铝板——挡风玻璃和座椅等细节作为额外物品提供。引擎和喷气驱动需要从现有的个人船只(PWC)上抢救，但这将不得不等待未来的视频。现在，有一艘*满载*(明白吗？)进行点焊。

建造过程看起来非常顺利，除了当他们试图把船头放在一起的时候。由于无法使两块侧板正确贴合，[Bob]最终不得不联系制造商。经过一些来回，原来一定是有一点打破了数控时，船体面板通过，作为一个关键削减了近 8 英寸(20 厘米)太短。他能够用拼图完成切割，并继续建造，但我们仍在挠头，为什么在它运出之前没有被发现。

[这不会是我们报道过的第一艘自制船](https://hackaday.com/2022/06/03/recycled-speed-boat-beats-the-barnacles-out-of-your-average-rebuild/)，但是考虑到[Bob]对细节的关注，我们特别期待在未来的视频中看到这艘船如何发展。*尤其是*，因为他是~~愚蠢地~~勇敢地要求评论者为他的新手艺想出一个名字。
