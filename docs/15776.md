# 工作台电脑，50 度扭转

> 原文:[https://hack aday . com/2022/12/28/work bench-PC-with-a-50s-twist/](https://hackaday.com/2022/12/28/workbench-pc-with-a-50s-twist/)

[HolGer71]有一个迷你 ITX 英特尔凌动主板，他发现它对 COM 和 LPT 等老式接口很有用。他突发奇想，决定给它一个更复古的外观——将它改造成一个更类似于 50 年代家用电器的设备，配有合适的显示器、鼠标和键盘。这个名为“Legacy-PC Computer Case”的项目，在其 3D 打印设计中巧妙地模仿了金属板材的结构。这还不是全部——所有的东西都是开源的，并且有足够的文档可以让你自己构建！

![](../Images/dc8a6ae0aa1b4d8135d9fd78c8f20e5f.png) [HolGer71]从[一般打印和装订建议、](https://gitlab.com/hzahnlei/legacy-pc-computer-case/-/blob/main/docs/README.md)开始，并从那里开始检查设置的每个部分。[主板固定盒](https://gitlab.com/hzahnlei/legacy-pc-computer-case/-/blob/main/docs/computer/README.md)围绕一个小型 miniITX 盒框架构建，将它包围起来，并为连接器和灯泡添加扩展。对于显示器，他在一个旧的配备 VGA 的 17 英寸桌面屏幕周围建立了一个新的框架——当然很容易找到。[键盘](https://gitlab.com/hzahnlei/legacy-pc-computer-case/-/blob/main/docs/keyboard/README.md)不贵，但配有机械开关，鼠标[是老式的富士通-西门子，但你会看到它是以不同的标签生产的。总之，这结合了非常通用的组件到一个可靠的和时尚的设备，为您的车间的需要。](https://gitlab.com/hzahnlei/legacy-pc-computer-case/-/blob/main/docs/mouse/README.md)

显然，这台机器配备了 Windows 7，这是最早支持的版本，现在可以在办公桌上工作了——准备好为旧的编程加密狗运行晦涩的软件，并且在这样做的时候看起来绝对令人难以置信。我们很少看到这样的努力投入到从零开始创造设计并与社区分享它们——大多数时候，我们看到 PC 内置于现有的设备中，如[这个老式收音机](https://hackaday.com/2016/10/29/vintage-radio-transformed-into-custom-pc-case/)，或[一个台式逻辑分析仪](https://hackaday.com/2017/12/31/old-logic-analyzer-becomes-new-pc-case/)。
