# 自适应宏垫使用微小的有机发光二极管屏幕作为键帽

> 原文：<https://hackaday.com/2020/12/17/adaptive-macro-pad-uses-tiny-oled-screens-as-keycaps/>

当我们第一次看到自适应宏键盘 Keybon 时，我们有点想知道这有什么大不了的。老实说，它看起来像任何其他的 USB 宏键盘，在厚重的按键上有各种常见任务的大图标。但是外表可能具有欺骗性，[麦克斯·克恩]给了凯本几个惊喜。

首先，Keybon 的每一个按钮实际上都是一个微型有机发光二极管显示器，使得键帽可以通过软件定制。九个 0.66 英寸显示器的分辨率都是 64 x 48 像素，这对于所有类型的图标来说都足够了，每个显示器都安装在一个 SMD 按钮开关上。他必须处理键帽在开关按钮上晃动而不按下它的问题；这个问题通过 3D 打印的悬臂框架解决了，它迫使键帽只能在一个轴上旋转，从而产生干净、令人满意的点击按键。

Keybon 的另一个技巧是交互性。它自己用一组标准图标启动，并通过 USB 发送相应的按键。但是当与它的伙伴 Windows 应用程序一起使用时，整个宏集可以被切换出来以适应正在使用的任何应用程序。这使用户能够访问 web 浏览器、EDA 套件、CAD 应用程序或 IDE 的自定义宏。该应用程序支持多达八个宏集，可以在下面的视频中看到。

我们喜欢它的外观和功能[Max。K]内置于 Keybon 中，但我们想知道电子墨水显示器是否也是键帽的一个好选择。它们现在以废弃商店货架价格标签的形式出售，而且它们可能很好，因为图标在没有电的情况下会持续存在。
