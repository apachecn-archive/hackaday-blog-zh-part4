# 每天照一面镜子，以积极向上的方式开始新的一天

> 原文:[https://hackaday . com/2020/05/28/begin-your ' s day-on-an-uplifting-note-a-daily-affirmation-mirror/](https://hackaday.com/2020/05/28/begin-your-day-on-an-uplifting-note-with-a-daily-affirmation-mirror/)

我们已经在这里看到了几十个“魔镜”建筑，其中大多数显示各种信息——日历、天气、新闻。它们是很好的构建，但是它们往往有点忙，不能真正激发一天平静的开始。但是如果你足够优秀，足够聪明，你可以建造[这个电子肯定镜](https://www.youtube.com/watch?v=kueeMawNy_0)，然后把它搞定，人们会喜欢你的。

[Becky Stern]在这个版本中把魔镜的概念降到了最低，只使用一组 14 段字母数字显示器来滚动令人振奋的信息。她使用的玻璃是部分反射的，当背面覆盖黑色胶带，并有一个小入口用于显示时，它就成为了一面像样的镜子。这些显示是由一个使用存储在草图中的静态断言的饰品驱动的；具有 WiFi 连接的微控制器也可以用于即时获得确认。或者，你知道，股票价格和交通更新，如果你不喜欢[斯图尔特斯马利]的东西。

那么前面提到的那些魔镜建筑呢？我们有[大的](https://hackaday.com/2015/12/30/magic-mirror-on-the-wall-is-pi-or-esp-fairest-of-all/)、[小的](https://hackaday.com/2016/02/28/79-smart-mirror-uses-raspberry-pi/)、[复古的](https://hackaday.com/2017/12/12/old-tv-lends-case-to-retro-magic-mirror/)，甚至还有[以孩子为中心的](https://hackaday.com/2018/05/27/magic-mirror-tirelessly-indulges-childrens-curiousity/)。你挑吧！
