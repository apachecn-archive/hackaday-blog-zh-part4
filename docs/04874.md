# 启示录的收音机

> 原文:[https://hackaday.com/2019/11/15/a-radio-for-the-apocalypse/](https://hackaday.com/2019/11/15/a-radio-for-the-apocalypse/)

过去几周，出现了大量与世界末日相关的文章，但当我看到由手绕线圈和氧化的英国便士制成的调幅收音机时，我不禁被深深打动了。我们已经介绍过[散兵坑无线电](https://hackaday.com/2016/04/21/hacking-when-it-counts-pow-canteen-radios/)，这是战时用找到的零件拼凑而成的老式无线电。

本例使用可变电容进行调谐，但这在技术上是可选的。真正需要的是一个线圈和一个类似二极管的东西。令人惊讶的是，氧化铜是一种半导体，一便士上的表面氧化足以形成一个简陋的二极管。但是，请注意，并不是所有硬币都有必要的铜镀层。如果一便士有绿色氧化物，它很可能是候选物。

需要一个快速拼凑的调幅收音机？有一些铁丝和一便士吗？是的，看看休息时间下面的视频，你会知道如何让它发生。当世界末日来临时，你会感谢我们的。
