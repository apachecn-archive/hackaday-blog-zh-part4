# 555 个定时机器人将统治世界

> 原文:[https://hack aday . com/2018/09/21/555-timer-robots-will-rule-the-world/](https://hackaday.com/2018/09/21/555-timer-robots-will-rule-the-world/)

每当一个项目包含 Arduino 或 Raspberry Pi 时，我们都会在 Hackaday 读者的评论中看到一个笑话，说“我本来可以用 555 定时器来完成它！”如果这个项目相当于一盏闪烁的灯或任何振荡的东西，那就更是如此了。在他最新的 *Volos 项目*视频中，Danko Bertovi 用 555 定时器电路制作了一个完整的[机器人。](https://www.youtube.com/watch?v=A_hUYwNvWhU)

好的，这实际上是一个机器人形状的死虫电路，但是它有闪烁的灯。我们也喜欢电池的底座，尽管它下面的一些不平似乎使整个东西有点不稳定，正如你在下面的视频中看到的。也有一些部分只是装饰性的。不过很可爱，是 555 定时器电路，外形像机器人。这一切使它成为一个胜利。

我们确实想知道如何进一步发展。毕竟，行走周期是一种振荡，所以 555 定时器电路可以运行一些伺服电机或至少一些压电脚。有人有想法吗？

另一方面，如果你在寻找一个属于美术博物馆的死 bug 电路，那么你只需要看看时钟就可以了。
