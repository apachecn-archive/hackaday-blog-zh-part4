# 本周失败:吹叶机不能飞

> 原文：<https://hackaday.com/2018/11/12/fail-of-the-week-leaf-blowers-cant-fly/>

吹叶机是郊区星期六交响乐团的主要乐器，也是最令人讨厌的乐器之一。用耙子并不需要做太多的工作，就能说服即使是最讨厌噪音的房主去大盒子商店买一个耙子来加速草坪家务。一旦你真的买了一个，感受到这些手持女妖产生的推力，你可能会想，如果我放开这个东西，它会飞吗？

[Peter Sripol]也有这样的想法，并着手建造了几架以叶片鼓风机为动力的飞机来回答这个问题。报告答案是否定的，这可能不是一个剧透警报，但下面的视频是一个有趣的观看。令人惊讶的是两架飞机差一点就成功了。第一架飞机是一架简化的 Ryobi 二冲程叶片鼓风机，悬挂在一个巨大的机翼和尾翼上，几乎要离开地面。1.1 版本增加了一个可伸缩的电动助推螺旋桨——严格用于起飞——并减少了大量多余的重量。那架飞机实际上是跳到了空中，但是，唉，伺服系统的问题使[彼得]无法关闭电源，独自在 Ryobi 上飞行。即使是伺服修复也无法挽救下一次飞行，那次飞行在起飞后就坠毁了。一个 2.0 版本，这一次使用了一个残酷修改的电动吹叶机，稍微更适合飞行，但在变得不稳定之前预示了几次。

我们能从这一切中学到什么？除了要花很大力气才能让吹叶机飞起来之外，没什么别的了。我们感谢[彼得]在这里的辛勤工作，但我们认为他最好专注于他漂亮的自制超轻。
