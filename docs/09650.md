# Hackaday 播客 111: 3D 图形是超声波的，Alexa 的脑叶切除，3D 打印泄漏的火箭，以及字体系统的游戏

Hackaday 的编辑迈克·什奇斯和埃利奥特·威廉姆斯策划了为期一周的伟大黑客活动。在 3D 空间中创建的物理显示器是一个圣杯，你可以用 200 个超声波传感器、4 个 FPGAs 和大量数学运算来制作一个。智能扬声器中有一个麦克风阵列，如果你只是推出自己的固件，它就是你的黑客。业余爱好伺服系统可能很糟糕，但本周我们看到，通过打开 DC 电机添加一个简单的 DIY 位置传感器，它们可以变得非常棒。激光正在进入汽车前灯；我们在这一集阐明了情况。
