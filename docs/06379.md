# 自动化您的 Xbox

> 原文:[https://hackaday.com/2020/04/24/automate-your-xbox/](https://hackaday.com/2020/04/24/automate-your-xbox/)

机器人先是抢走了我们的工作，然后又来抢我们的电子游戏。这个反乌托邦式的未来是由[小法国 Kev]带给你的，他设计了这个可爱的 3D 打印机械臂来与 Xbox One 控制器操纵杆进行交互。他在休息后的视频中展示了这一点，用 Unity 编写了一个控制球平衡的物理演示。

向他的设计质量致敬。拇指操纵杆的旋钮两侧各有两个部分。他用螺丝将这些部件相互连接起来，紧紧抓住棍子。关节处使用了轴承，以使控制机械臂的两个伺服电机平稳运行。机器人附件的底座是用拉链绑在控制器上的。

该构建以机器学习实验为目标。由于计算机可以通过 Arduino 控制手臂，并且计算机可以访问虚拟环境中发生的事情的指标，因此这是训练神经网络的完美选择。你和我们想的一样吗？这是硬件速度运行你最喜欢的视频游戏的开始，就像五年前[SethBling] [为*超级马里奥世界*](https://hackaday.com/2015/06/14/neural-networks-and-mario/)所做的那样。这将更加令人印象深刻，因为这将通过自动化控制器的机械部分来完成，而不是纯粹在软件领域中操作。你只需要自己做[来实现按钮控制](https://hackaday.com/2015/01/24/arduino-servo-scotch-tape-an-interesting-conversation/)。
