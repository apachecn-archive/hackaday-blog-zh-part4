# Z8000 王牌需要您的帮助

> 原文：<https://hackaday.com/2022/12/19/z8000-trump-card-needs-your-help/>

[Smbakeryt]需要您的帮助。他为 PC 买了一个 1984 年的 Z8000 协处理器卡，但软件在运行中丢失了。显然，协处理器——被称为王牌——在著名的[史蒂夫·西亚]的帮助下出现在了 *Byte* 杂志上。[的原理图已经发布](https://worldradiohistory.com/Archive-Byte/80s/Byte-1984-05.pdf)，如果你给【史蒂夫】发送你建造它的证明，他会给你发送[软件](https://worldradiohistory.com/hd2/IDX-Consumer/Archive-Byte-IDX/IDX/80s/82-83/Byte-1984-06-OCR-Page-0117.pdf)。该产品后来被商业化，但似乎没有人拥有该软件，所以[Smbakeryt]正在寻找它。

董事会本身在当时是相当惊人的。它增加了一个 16 位 Zilog Z8000 CPU，内存为 512 K。1984 年的大铁，比当时的 IBM 个人电脑性能好一点。

我们怀念计算机设备伴随着大量文件的日子。如今，你更有可能得到一个带有网址的贴纸。Z8000 是一个很好的处理器，可以模仿 Z80，但它从来没有变得非常受欢迎。除了 Zilog 的 System 8000，CPU 也进入了一些 Unix 计算机，包括 Onyx C8002 和几台 Olivetti 计算机。准将计划在一个被取消的项目中使用 CPU。Z8000 以不使用微码而闻名，因此，它适合于具有 17，500 个晶体管的相对较小的芯片(相比之下，8086 的晶体管为 29，000 个)。

我们希望有人能在软件方面提供帮助。如果你想要你自己的 Z8000 系统，你最好选择 Clover。或者，坚持用一辆便宜的上的 [Z80。](https://hackaday.com/2018/07/28/the-4-z80-single-board-computer-evolved/)
