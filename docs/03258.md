# 电路弯曲那些可爱的声音

> 原文：<https://hackaday.com/2019/06/08/circuit-bending-those-adorable-voices/>

Leapfrog 制造一些非常棒的儿童电子产品。尤其令人钦佩的是这些设备的低成本、电池寿命和音频质量。这种电路弯曲黑客利用这些音频电路，将字母朋友变成你的主唱。演示视频中的表演以一些令人印象深刻的技巧开始，但请耐心等待，因为到最后，这只紫色的小毛毛虫证明了自己是一件配得上你一直在组装的花哨 Eurorack 的乐器。

上面的图片提供了动物内部的一个很好的视角。[Jason Hotchkiss]提到他对构建质量印象深刻，我们不得不同意。此外，看看所有这些投入——这是乞求离开玩具王国，加入乐队。凭着只能通过大量电路弯曲经验获得的直觉，他猜测 PCB 上的单个通孔电阻是用来调节时钟速度的。这使得很容易投入一个 trimpot 弯音，他继续找出个人的注意控制。

所有这些毛虫的脚排列在一个键盘矩阵中，以检测按钮的按压。在拿出示波器进行了一点逆向工程后，[Jason]抓起一个 PIC 微控制器，将其添加到与库存带状连接器相同的焊点上。结果就是脚上的按钮还能用，但是现在 Alphabet Pal 也有 MIDI 控制了。

看一看完整的细节，休息后的视频听听它的行动。如果你是弯曲电路玩具的粉丝，[这个漂亮的粉红色键盘总是给我们留下深刻的印象](https://hackaday.com/2018/06/24/circuit-bent-keyboard-is-pretty-in-pink/)，尤其是新增的弹簧混响！
