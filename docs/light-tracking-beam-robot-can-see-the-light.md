# 光跟踪光束机器人可以看到光

> 原文：<https://hackaday.com/2021/11/21/light-tracking-beam-robot-can-see-the-light/>

BEAM robotics 代表生物学、电子学、美学和力学，是一种专注于用简单模拟电路制造机器人的精神。[NanoRobotGeek]建立了一个很好的例子，创造了一个光跟踪机器人，不使用电池和微控制器。

机器人的目标是追踪它能看到的最亮的光源。这是通过将来自四个光电二极管的信号输入一些模拟逻辑来实现的，然后模拟逻辑将电压输出到两个电机，这两个电机将机器人瞄准，引导它走向光明。还有一个声音检测电路，当它通过连接的麦克风检测到哨声时，会提示机器人摆动。

整个电路是使用黄铜线自由形成的，结果是一个令人难以置信的巧妙构建。展示在一个钟罩里，这个建筑看起来像是融合了过去和未来的精致艺术品。它既不是蒸汽朋克，也不是赛博朋克，而是结合了复古铜管和现代发光二极管。

这是一个伟大的建筑，它让我们想起了我们最近看到的一些伟大的赛道雕塑。休息后的视频。
