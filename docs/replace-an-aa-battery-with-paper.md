# 用纸替换 AA 电池

> 原文：<https://hackaday.com/2022/08/08/replace-an-aa-battery-with-paper/>

纸是社会无处不在的一部分；以至于其背后令人难以置信的工程设计经常被忽视。然而[罗伯特]却不是这样，他对这种材料及其远不止记录信息的许多用途有着深刻的理解。在这个特殊的视频中，他重现了研究人员发现的一种方法，即将一张纸变成一种电池，其性能与 AA 型碱性电池相当。(视频，嵌入休息下方。)

这一过程包括创造几种不同类型的墨水，每一种都可以用相对常见的材料制成，如虫胶、乙醇、聚乙二醇和石墨。这些材料中的每一种都以不同的比例混合以产生墨水。一旦制成阴极墨水和阳极墨水，就需要称为集电器墨水的第三种墨水，其基本上起到导线的作用。纸张被浸入盐溶液中，然后让它干燥，形成部分防水涂层，当需要时，可以通过润湿它来激活它，这允许电池的离子流动发生。

一旦你看到它的运行，这种电池的化学性质就很有意义，而且只要电池保持干燥，这种电池生产方法还具有长保质期的额外好处。它们也不会像不可充电的碱性电池那样破坏环境，至少除非你想采取一些极端措施来重复使用它们。
