# 还记得 DAB 电台吗？心灵探测器被拆卸

> 原文:[https://hack aday . com/2022/08/13/remember-da B- radio-the-psion-wave finder-gets-a-tear down/](https://hackaday.com/2022/08/13/remember-dab-radio-the-psion-wavefinder-gets-a-teardown/)

随着数字音乐在 20 世纪 90 年代横扫几乎所有的收听媒体，令人惊讶的是，有一个领域的模拟支持仍然非常活跃。我们在这里谈论的不是黑胶唱片的复兴，而是调频广播，它已经成功地有效地抵挡了几十年的数字竞争。二十年前，它的日子似乎屈指可数，在欧洲，第一代 DAB 数字收音机似乎准备征服电波。其中有一个真正奇怪的东西，也是 Psion 最后的重要消费产品之一，WaveFinder USB DAB 无线电接收器。[Backofficeshow]有一个，[并且为了我们的娱乐](https://www.youtube.com/watch?v=jaGFjZhIYrA)把它给拆了。他将其描述为第一个消费 SDR 产品，这可能有点夸张，但无论如何，这是对将成为计算死水一潭的有趣审视。

在 peak-90s 风格的半透明蓝色外壳中，有一个带有许多屏蔽的 PCB，上面有一个 USB 控制器和一系列 DSP 芯片。无线电解调在硬件中完成，但信号解调显然是在主机 PC 上完成的。当时其 299 英镑的价格使其成为 DAB 接收的负担得起的终端， *The Register* 认为[下载广播宽带数据的能力使其成为一种革命性的产品](https://www.theregister.com/2000/10/14/broadband_for_free_radio_kills/)，但遗憾的是，消费者和广播公司都不同意，在不光彩地退出之前，它被大打折扣。DAB 本身将很难满足这些期望，广播公司基于多路传输的许可模式使其对地方电台没有吸引力，这意味着即使现在 FM 仍然充满了电台。也许随着收听无情地转向流媒体，它的时代已经过去了，事实上[爱尔兰已经完全放弃了 DAB](https://hackaday.com/2021/03/07/the-digital-radio-era-partially-ends-in-ireland/)。

如果你想了解更多关于 DAB 的信息，[我们看了一下不久前的技术](https://hackaday.com/2017/01/09/anatomy-of-a-digital-broadcast-radio-system/)。
