# 小猫妨碍了频率计数器的恢复

> 原文:[https://hack aday . com/2022/08/24/frequency-counter-restoration-buddy-by-kitties/](https://hackaday.com/2022/08/24/frequency-counter-restoration-impeded-by-kittens/)

我们认为数字显示器是你在相对现代的设备上看到的东西。但一些旧设备有像 nixies 或 numitrons 这样的东西，以获得看起来很酷的复古数字显示器。然而，HP 521A 频率计数器使用四列十个分立的氖灯来制作一个明显科技含量低但有效的数字显示器。[天兔电气]一段时间以来一直在恢复其中的一个，但是在第二个和第三个视频之间有一个间隙，因为他的工作室变成了一个小猫托儿所。下面可以看到最后一个视频。

在之前的视频中，他让大部分设备工作，但仍有一些奇怪的行为。这个视频展示了走向成功的最后步骤。有趣的是，由于四列中的每一列都是相同的，因此可以比较十年间的读数。

然而，最后发现霓虹灯泡被高度腐蚀，更换所有的霓虹灯泡使事情变得更好。然而，本应读取 60 Hz 线路频率的自检读数却是 72 Hz，因此需要重新校准。但这相对容易，因为从后面板上可以拿到锅。如果你想看到更多关于修复的细节，请务必查看之前的视频。

我们喜欢这个旧设备，以及聪明的设计师如何用我们认为很少的东西做了这么多。我们讨厌鼓励你潜在的上瘾，但我们之前已经给出了如何获得旧装备的建议。如果你想看看在 WS2812 面板出现之前有什么可能，你可以建造这个[霓虹灯装置](https://hackaday.com/2020/07/18/forget-led-matrices-how-about-neon/)。
