# [本·克拉斯诺]做了一个 DSKY

> 原文：<https://hackaday.com/2019/05/19/ben-krasnow-makes-a-dsky/>

来自阿波罗计划的数百件甚至数千件文物散落在全球各地，一些扭曲的残骸沉入海底，另一些则被亲切地保存在博物馆或私人收藏家手中。剩下的几乎都是纯稀有元素，所以如果你想要类似阿波罗的东西，你可能得自己动手了。

[Ben Krasnow]接受挑战，从零开始制作了一个电致发光的阿波罗时代的 DSKY 显示器,并取得了出色的成果。DSKY，或“显示器和键盘”，是阿波罗制导计算机的用户界面，这是一种专门建造的数字导航系统，共载有 24 人往返。[Ben]说重建显示花了很长时间，我们可以看到原因。他需要掌握相当多的技能，包括让玻璃面板显示器工作的丝网印刷。该面板是磷光涂料、电介质和导电墨水的夹层结构。油墨被丝网印刷在背面以形成字符，全部应用于氧化铟锡(ITO)导电玻璃。一个具有相同字符段模式的 PCB 位于其后，通过三个 HV507 高压移位寄存器以 300 伏左右的电压驱动每个字符段。这是一个令人印象深刻的工程，给人一种绝对不是自制的感觉。

在下面的视频中，[Ben]详细讲述了他在通往这个神奇终点的路上所经历的考验，尤其是由于移位寄存器中无效的保护二极管而导致的一片又一片的油炸。这是一个史诗般的调试故事，本身就值得门票的价格。当然，这并不是镇上唯一的 DSKY——有一段时间[【弗兰·布兰奇】也在设计一个——但是那种蓝色的光芒有一些东西是我们真正喜欢的。](https://hackaday.com/2017/05/29/re-creating-the-apollo-dskys-display/)
