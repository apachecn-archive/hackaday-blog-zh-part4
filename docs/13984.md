# 不介意我滑雪吧

> 原文：<https://hackaday.com/2022/06/20/dont-mind-if-i-ski-do/>

黑客们有一个古老的传统，那就是自己动手制作。无论是真品太贵，还是你认为你可以做一个更好的，瘙痒袭来，它可以迅速蔓延。[自制的疯狂]在建造自己的摩托艇时非常兴奋。

除了顶部有壳的船之外，什么是水上摩托？在早期的一个视频中，他用胶合板制作了一艘船，在经过玻璃纤维和密封的常规步骤后，他自豪地乘坐他相对普通的船四处漂浮。但是现在他有了一个工作底，是时候回到 CAD 了。他打印出他需要的各种形状的模板，每一个都标有不同的代号，然后把它们粘在胶合板上。这里没有花哨的 CNC，只有稳稳的手和拼图。我们喜欢他为自己编写的专业构建说明，这些说明以类似乐高积木的质量详细说明了每一个部件如何插入哪里以及以什么顺序完成它们。除了水上摩托艇的顶层，他还设计了一个支架，让船在制作过程中可以休息，这是额外的一步。安装在天花板上的绞盘使得把船吊到位变得很容易。接下来，他用 PU 胶把各种框架连接起来。薄胶合板作为骨架顶部的覆盖层。填充，打磨和玻璃纤维覆盖了结构，使其防水。更多的砂光和一些底漆后，这是为另一个水测试做好准备。

他设计了一个带有舷外发动机的版本，但他正试图建造一个内置喷气驱动的版本。因此，我们期待着看到下一步，他在他的定制船只上飞来飞去。但是他已经做的事情令人印象深刻。如果你正在寻找一个小一点的东西来拉你绕着水转，为什么不看看这个小小的 3d 打印拖船呢？休息后的视频。
