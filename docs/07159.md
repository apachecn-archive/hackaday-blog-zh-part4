# 555 对恋人的圣地

> 原文:[https://hacka day . com/2020/07/15/a-555 情侣圣地/](https://hackaday.com/2020/07/15/a-shrine-for-all-the-555-lovers/)

对于我们的许多读者来说，经典的 555 定时器在他们心中有着特殊的地位，谁敢用 Arduino 代替它，谁就该被诅咒。对于那些认真投入的人，或者那些只是喜欢新奇装饰品的人来说，[acerlaguinto7]建造了一个恰到好处的东西:[一个巨大的、实际上很实用的纸板 555 定时器 IC](https://www.instructables.com/id/Giga-Max-555-the-Giant-555-Timer/) 。

[acerlaguinto7]对原始 IC 进行了所有的测量，将其放大了 22 倍，并开始切割纸板——也考虑了方向凹槽——并添加标记以模仿 TI 的 NE555P。接下来，他把一堆铝罐拆开，把它们做成大头针的形状，再次尽可能接近原来的形状。最重要的是，他把一个真正的 NE555 放在了巨大的对应物中，并将其连接到汽水罐引脚上，将其变成了一个完全可操作的超大定时器 IC。

显然，像这样的巨大导电引脚为一些[死虫闪烁的光](https://hackaday.com/2018/09/25/minimal-blinky-project-makes-the-chip-the-circuit-board/)尖叫，即使是最颤抖的手也能设法焊接，而【acerlaguinto7】当然提供了，正如你在休息后的视频中看到的。我们可以看到这种进一步发展的一种巧妙方式是集成[这种试验板实现，作为](https://hackaday.com/2019/06/17/modeling-the-classic-555-timer-on-a-breadboard/)内部 555 的替代物——或者只是将它连接到[巨大的树莓派](https://hackaday.com/2016/06/11/the-raspberry-pi-infinity-is-a-fully-functional-huge-raspberry-pi/)。
