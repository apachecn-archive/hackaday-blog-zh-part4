# 气象站得到急需的升级

> 原文:[https://hack aday . com/2019/11/09/weather-station-gets-much-need-upgrades/](https://hackaday.com/2019/11/09/weather-station-gets-much-needed-upgrades/)

气象站是一个受欢迎的项目，部分原因是它有助于(也很有趣)了解你所在位置的天气，而不是你的邮政编码中模糊的天气预报。它们也很受欢迎，因为它们是获得微控制器、传感器、I/O 和通信协议经验的好方法。随着时间的推移，你自己的构建也可以很容易地升级，【Tysonpower】向我们展示了几年前他对流行的 Sparkfun 气象站进行的一些[升级。](https://tysonpower.de/blog/esp8266-weather-station)

Sparkfun station 是一个很好的构建基础，只是需要一些更新。第一个问题是，传感器包并不容易买到，但在全球速卖通上的一些搜索发现了一套来自中国的类似传感器。一台 Wemos D1 Mini 被用作替换控制器，它被扣上扣子并进行编程，结果比原来的 Sparkfun 站略便宜(也更现代)。

这个新空间站的所有部件和代码都可以在[【Tyson power】的 Github 页面](https://gitlab.com/Tynet.eu/esp8266-thingsboard-weatherstation)上找到，如果你想看看我们之前在这里介绍过的类似空间站，三年前的[有一个也是太阳能的](https://hackaday.com/2016/08/17/solar-powered-weather-station-has-the-complete-suite-of-sensors/)。
