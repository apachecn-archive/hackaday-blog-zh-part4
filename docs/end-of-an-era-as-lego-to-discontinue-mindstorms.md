# 随着乐高停止头脑风暴，一个时代的终结

> 原文：<https://hackaday.com/2022/11/03/end-of-an-era-as-lego-to-discontinue-mindstorms/>

当有如此多的单板计算机和其他产品旨在为儿童提供学习编程和其他技能的方法时，人们很容易忘记在 Arduino 或 Raspberry Pi 及其模仿者出现之前，计算机是一个非常昂贵的封闭盒子。

在 90 年代末 8 位家用电脑革命之后留下的真空中，出现了乐高的 Mindstorms 套件，这是一个带有特殊可编程砖块的连锁物品盒，让孩子们有机会制作自由形式的计算机化机器人项目。最近有消息称[24 年后，该公司将在今年年底停止 Mindstorms 系列](https://www.brickfanatics.com/lego-discontinuing-mindstorms-end-of-2022/),这对任何从那时起就乘坐无障碍微控制器列车的人来说，就像是一个时代的终结。

Mindstorms 的起源是麻省理工学院媒体实验室的[可编程积木项目](https://www.tech-insider.org/lego-mindstorms/research/1997/1222.html)，一系列带有微控制器的矮胖乐高积木和用于电机和传感器的 Mindstorms 乐高积木触点。他们的 Logo 编程语言实现被乐高避开了，取而代之的是主机上的图形系统，于是 Mindstorms 工具包诞生了。该品牌已被用于控制器的一系列迭代，以及一系列不同的机器人套件。

1998 年，一台家用电脑已经从 BASIC 中可编程的东西演变成一台运行 Windows 和微软 Office 的机器。像 Parallax's BASIC Stamp 这样的电路板是可以买到的，但价格昂贵，而且没有任何控制功能。Mindstorms 套件是革命性的，它在玩具中提供了一个可访问的完全可编程的微控制器，以及一整套乐高玩具，包括电机和传感器。

我们猜测，2022 年，Mindstorms 已经被更好更便宜的单板计算机淘汰，但这并没有剥夺它在为 90 年代的孩子们提供第一次按照他们的方式制作一个合适的机器人的机会方面的特殊地位。工具包[已经在 Hackaday](https://hackaday.com/2017/11/17/mindstorms-forkliftbots-gonna-take-your-job/) 找到了自己的位置，但也许我们展示的大多数使用它们的项目已经有几年的历史了，现在强调了它们为什么会走到尽头。这么长时间的思想风暴，你不会被遗忘！
