# 家庭监控，没有所有的传感器

> 原文：<https://hackaday.com/2020/07/17/home-monitoring-without-all-the-sensors/>

智能家居有很多额外的好处，其中最重要的是能够监控你家里的情况，跟踪它们，并做出趋势。每件监控设备，如传感器或摄像机，都是需要在您的系统上运行的另一组电线和需要维护的另一个“东西”。然而，有时候有一些聪明的方法可以避开传感器，同时仍然保留它们的有用性。

在这个建筑中，[squix]使用现有的传感器进行电力计量，以便在他的烤箱预热时提醒他。传感器是一个 Shelly 3EM，它与他的家庭自动化接口的方式是通过意识到他的电烤箱一旦达到所需的温度就会停止向加热元件供电。他可以用家庭控制器监控家中电力需求的突然大幅下降，并利用这种下降来提醒他烤箱已经准备好了，而不必安装额外的东西，如温度传感器。

虽然这种特殊的传感器可能只在欧洲的一些地区可用，但我们认为这种想法将适用于许多不同的传感器，甚至其他设备。即使是一个[小型机器学习设备](https://hackaday.com/2018/12/25/machine-learning-on-tiny-platforms-like-raspberry-pi-and-arduino/)也应该能够知道什么负载在什么时间到来，然后根据这些数据进行编程以执行功能。
