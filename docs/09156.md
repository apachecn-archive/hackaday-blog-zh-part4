# 自行车车轮灯闪烁恰到好处

> 原文：<https://hackaday.com/2021/02/04/bike-wheel-light-flashes-just-right/>

当谈到在汽车周围安全骑自行车时，灯越多越好。理想情况下，这些灯会自己亮起来，所以你不必每次都记得开关它们。这正是[杰里米·库克]最新产品背后的想法——这是一种自动自行车灯，它可以检测振动并点亮一些 led 作为响应。

构建非常简单——一个硬币电池供电的 ATtiny85 从弹簧振动传感器读取输入，并使 led 闪烁。这是为了补充[Jeremy]的主自行车灯，它是手动操作的，一直亮着。我们特别喜欢这里的形式追随功能——板子的形状被设计成拉链状绑在辐条上，所以它尽可能地贴近动作。他巧妙地利用纸板和激光切割机模拟出一个适合辐条之间的木板原型。如果你问我们，你的第二个专业制造的 PCB 非常酷。骑过去休息，看看建立视频。

如果你认为辐条上的萤火虫不足以保证你的安全，那么[去全彩虹派对自行车](https://hackaday.com/2020/11/22/rgb-party-bike-flashes-with-the-beat/)。
