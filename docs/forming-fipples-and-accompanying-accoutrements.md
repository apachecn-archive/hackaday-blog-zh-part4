# 形成乳头和伴随的装备

> 原文：<https://hackaday.com/2019/08/06/forming-fipples-and-accompanying-accoutrements/>

[苏斯博士]用最少的文字和明亮的插图创作了令人难忘的书籍。他启发了儿童和成人，其中一个人[Len]长大后为 YouTube 上的 Bellowphone 频道创作了管乐器。在他奇思妙想的创作背后是重要的工程学，而这一次，我们将看到一个苹果的建造过程。视频也在休息后播放。尽管 fipple 听起来像是一个[苏斯博士]发明的词，但它是一个合法的音乐术语，意思是像哨子一样的喉舌。在这种情况下，它将空气吹过玻璃罐，为[Len]的管风琴创造声音。请看下面的第二个视频，是来自*魔笛*的表演。

[Len]使用透明的硬质聚氯乙烯材料制作薄膜，并使用定制的成型模具在薄膜变软时使其成型。剩下的是用剃刀、手锉和干湿砂纸进行的精密手工工具工作。一旦完成，fipple 看起来就像任何通过严格的建造技术生产的乐器部件。制作吹口是一回事，但如果方向不对，它就不会发出任何声音，所以我们还学习了如何将钢带变成风琴瓶组件。如果加上一些油管和橡胶挤压球，就可以自己做仪器了。

Bellowphone 频道存在的部分原因是[Len]在[管风琴](https://hackaday.com/2016/10/01/tiny-pipe-organ-needs-tiny-church/)社区中找到了很多支持，他们向他展示了[他们生计的秘密内部运作](https://hackaday.com/2011/07/20/amazing-pipe-organ-desk-features-secret-compartments-and-an-all-wood-logic-board/)，现在是他与[分享这种热情](https://hackaday.com/2014/05/25/pvc-percussion-pipe-organ-sounds-surprisingly-good/)的机会。
