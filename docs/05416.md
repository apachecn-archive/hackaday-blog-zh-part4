# 弹簧夹是完全 3D 打印的

> 原文:[https://hack aday . com/2020/01/23/spring-clamp-is-complete-3d-printed/](https://hackaday.com/2020/01/23/spring-clamp-is-completely-3d-printed/)

对于门外汉来说，双丝打印机可能看起来像是一个噱头，但它们为高级设计提供了一些强大的选择。[[Darren tar bard]展示了一个漂亮的弹簧夹，它是在一次操作中 3D 打印出来的。](https://www.thingiverse.com/thing:3214038)

这个夹子和你在任何五金店都能找到的类似。标准的 PLA 或 ABS 细丝可用于夹具的主体，其具有集成的铰链。然而，该元件不是典型的金属弹簧，而是 3D 打印的。弹簧由 TPU 细丝制成，并在适当的位置印刷。弹簧组件上不同的填充百分比可以改变弹簧的特性，从而形成更柔软或更牢固的握持。

这是双丝打印应用的一个很好的例子——比用它打印双色皮卡丘要有用得多。3D 打印机在工具领域大有可为；他们甚至能把台钳变成有效的压弯机。休息后的视频。
