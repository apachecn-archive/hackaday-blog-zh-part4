# 从头开始的泡沫冲浪板

> 原文：<https://hackaday.com/2021/12/29/foam-surfboard-from-scratch/>

你是否曾经想自己制作冲浪板，但却因为缺少工具、材料或成本而止步不前？几乎完全借鉴了一家知名家居装饰零售商的产品，[Andrew 1997][详细介绍了制作你的木板所需的步骤](https://www.instructables.com/How-to-Make-a-Home-Depot-Surfboard/)。

在他的指南中，他详细介绍了 XPS(发泡聚苯乙烯)和 EPS(挤塑聚苯乙烯)之间的区别，以及每种产品的闭孔和开孔性质如何影响最终的纸板。从两张粉红色的 XPS 板开始，他用胶水将它们层压在一起，形成他的空白板。桁条是冲浪板中间的一根长木头，可以提供额外的弹性和强度。一些地板胶合板用曲线锯弯曲成所需的形状。不幸的是，坯料需要被分成两半来安装桁条。然而，他有一个窍门，可以在不弯曲的情况下将坯料粘在一起，这个窍门就是棘轮带。

他用热金属丝将泡沫切成大致合适的形状。清理工作是用砂块、刨子和水平仪完成的。下一步是用环氧树脂和玻璃纤维层压电路板。接下来，[Andrew]详细介绍了该流程的一些注意事项，并开始制作玻璃。打磨至 2000 粒度和一些抛光化合物使董事会华丽。经过一段时间的最终固化，你就可以乘风破浪了。

YouTube 上有一个[方便的播放列表，详细介绍了这个过程，所以你可以跟随](https://www.youtube.com/playlist?list=PLHeE1mcmnNBQ00siGWuGXLIjvFkLY0B1P)。一旦你完成了你的冲浪板，[也许你的下一个项目将是用喷气驱动装置给它加电](https://hackaday.com/2021/11/25/surfboard-gets-jet-upgrades/)。休息后的视频。
