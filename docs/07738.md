# 世界电台让你的手指行走

> 原文：<https://hackaday.com/2020/09/12/world-radio-lets-your-fingers-do-the-walking/>

过去，收听来自遥远国家的广播需要短波设备，但由于有了互联网，我们现在可以在自己舒适的桌面上收听来自全球各地的广播。只需点击几下鼠标，你就可以在本地新闻电台和卡萨布兰卡最新流行趋势之间切换。尽管在线流媒体可能很方便，但一些人仍然渴望传统的广播体验。

对于这些人来说，由亚伯拉罕·马丁内斯·细川玉子主持的树莓派世界电台可能是解决办法。这款互联网收音机内置于 20 世纪 60 年代的 Invicta 收音机中，使用了非常独特的界面。您可以使用前面的旋钮在世界地图上平移和缩放，而不仅仅是从频道列表中进行选择。流媒体频道由位于其来源国的气泡表示，因此您实际上必须“旅行”到那里才能收听。休息后的视频简要演示了它在实践中是如何工作的。

我们承认它最终可能会变得有点乏味，但是从视觉的角度来看，它绝对是奇妙的。[亚伯拉罕]甚至给了地图一个适当的复古外观，以更好地匹配整体审美。通常我们会说用 Raspberry Pi 4 驱动一个流媒体播放器有点大材小用，但是考虑到这里使用的 GUI 组件，这可能是正确的选择。

当然，我们以前见过将[互联网收音机嵌入老式外壳](https://hackaday.com/2016/02/27/an-internet-radio-in-a-classic-cathedral-case/)，我们甚至见过[使用地球仪来选择电台](https://hackaday.com/2020/07/27/radioglobe-takes-the-world-of-internet-radio-for-a-spin/)，但是将这两个概念结合到一个有凝聚力的项目中确实是一项相当大的成就。
