# PS2 得到生姜便携式治疗

> 原文:[https://hack aday . com/2021/08/23/PS2-gets-the-ginger-portable-treatment/](https://hackaday.com/2021/08/23/ps2-gets-the-ginger-portable-treatment/)

关于这款[便携式 PS2](https://www.gingerofmods.com/eclipse) 我们首先注意到的是，塑料看起来像一个消费级外壳，而不是 3D 打印的外壳。它来自[GingerOfOz]，他有很多便携式转换，所以我们并不惊讶这看起来像一个真正的索尼设备。当你像何一样有经验的时候，像塑料质地，按钮选择这样的细节，都是解决了的问题，但不应该被我们这些凡人认为是理所当然的。

当然，这不仅仅是漂亮，如果它没有功能，我们就不会谈论它。该系统从 USB 存储器播放几乎所有 PS2 的歌曲。值得注意的例外是那些拒绝加载没有双休克控制器。粗鲁。如果你想知道它是否全速玩游戏，是的。它实现了真正的速度，因为它使用了一个 PS2 超薄主板得到削减了 Dremel。定制的印刷电路板提供硬件的其余部分，如音量按钮和电池充电。没有光驱，因为它们很耗电，所以你的电影剪辑画面可能会滞后，加载时间会稍长。
