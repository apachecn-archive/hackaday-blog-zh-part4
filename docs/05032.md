# 用 880 节 AA 电池烹饪火鸡

> 原文:[https://hack aday . com/2019/12/02/cooking-a-turkey-with-880-aa-电池/](https://hackaday.com/2019/12/02/cooking-a-turkey-with-880-aa-batteries/)

在一年中的这个时候，正确烹饪火鸡是一件严肃的事情。随着重大节日的临近，容不得任何差错——任何差错都可能让客人感到不满和饥饿。然而，斯蒂芬·法恩斯沃思冒险尝试用 AA 电池烹饪火鸡。

AA 对于这样一个任务的吸引力正是因为它是一个如此糟糕的选择。它是为便携性而设计的，而不是高功率输出，它从未被设计成主要烹饪工作的能源。为了让事情顺利进行，[史蒂夫]计算出需要多少电池。这包括计算烹饪效率、电池热性能和鸟本身的比热。随着这些数字汇集在一起，一个 300 瓦的慢炖锅被投入使用，以避免电池过度消耗。

随着 880 原子吸收光谱仪被装入一个定制的载体中，[史蒂夫]接上了功率计和炊具，并密切关注着温度。几个小时后，电池组开始变热，因此采取了额外的冷却措施以避免火灾。就在六个小时之前，火鸡已经煮透了，可以吃了。据估计，这些电池仍有足够的电量维持几个小时。

这不是一种快速有效的烹饪火鸡的方法，但肯定是可以做到的。我们完全期待[史蒂夫]明年也能提交硬币细胞火鸡烹饪比赛。记住，一点工程总是有帮助的，尤其是在厨房里。休息后的视频。
