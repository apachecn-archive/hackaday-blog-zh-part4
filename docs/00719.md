# 振膜是这些柔性 PCB 扬声器中的线圈

> 原文:[https://hack aday . com/2018/09/21/the-diaphragm-is-the-coil-in-this-flexible-PCB-speakers/](https://hackaday.com/2018/09/21/the-diaphragm-is-the-coil-in-these-flexible-pcb-speakers/)

扬声器曾经是大型机电设备，由磁铁、动圈和纸锥共同工作，以令人愉悦的方式移动空气。当然，它们已经变得越来越小，小到可以直接拧进你的耳朵，或者放在最纤薄的智能手机里，但仍能提供合理的音质。基本机制没有太大变化，但这并不意味着没有其他方法可以将电信号转换成声波。

以这些由柔性印刷电路板制成的扬声器为例。在研究他的[柔性 PCB 软致动器](http://hackaday.com/2018/07/29/flexible-pcb-becomes-the-actuator/)时，【Carl Bugeja】注意到，当线圈被放置在磁铁上时，PWM 信号穿过薄 PCB 材料上的线圈时会发出一种可以听到的蜂鸣声。他决定利用这一点，努力从印刷电路板制作一个像样的扬声器。一个连接到简单放大器的早期原型显示出了希望，所以他 3D 打印了一个环来支撑 PCB，就像一个小钕磁铁上的隔膜一样。音质还不错，但音量很低，所以[Carl]试验了一种附着在印刷电路板上的纸盆来提高音量。这没有多大帮助，但普通物体作为共鸣器似乎工作得相当好。看看下面视频中的结果。

这在很大程度上是一项正在进行的工作，但鉴于[卡尔]在 PCB 创作方面的记录，从[机器鱼](http://hackaday.com/2018/09/14/flexible-pcbs-make-the-fins-of-this-robotic-fish/)到内置在 PCB 中的[步进电机，我们可以说他会做出实质性的改进。关注他和其他人在 2018 Hackaday 奖](http://hackaday.com/2018/01/24/this-tiny-motor-is-built-into-a-pcb/)[乐器挑战赛](https://hackaday.io/prize/details#five)部分的进展。
