# 在蒸汽甲板的帮助下测向

> 原文:[https://hack aday . com/2022/10/21/从蒸汽甲板上借助帮助测向/](https://hackaday.com/2022/10/21/direction-finding-with-help-from-the-steam-deck/)

测向或猎狐是业余无线电圈子里的一项流行活动，在这个圈子里，一群手持无线电的人试图找到一个广播源。除了是业余爱好者的爱好之外，它也是试图追踪侵犯领空者的监管者的必要工具。有很多方法可以确定无线电传输的精确位置，但是这个方法成功地使用了一艘船和一个蒸汽甲板，每个都配备了软件定义的无线电。

这个项目来自于[Aaron],他在业余无线电圈以专注于 SDR 的 Linux 发行版 DragonOS 而闻名；它拥有优质 SDR 体验所需的所有工具，在本例中是 KrakenSDR 和 DF 聚合器。他把所有东西都装在蒸汽甲板上，放在湖边的一个安全的地方，同时他在船上带着装有相同软件的第二个设备。由于这两个设备监听特定的信号，他能够快速锁定他在岸上的朋友，他的朋友正在用 70 厘米的波段广播，这要感谢所有这些软件包的帮助。

虽然业余无线电并不总是以年轻和令人兴奋的活动而闻名，但软件定义的无线电和其他数字模式的出现似乎正在动摇这个世界。当然，乘船绕湖飞驰本身也很有趣，像这样的猎狐也可以用树莓派这样小而简单的东西来完成。
