# Pi-Cast 为 KVM 增加了 ATX 信令

> 原文:[https://hack aday . com/2022/12/29/pi-cast-adds-atx-signaling-to-KVM/](https://hackaday.com/2022/12/29/pi-cast-adds-atx-signalling-to-kvm/)

KVM 是一个很好的工具，用于管理许多不同的计算机，而无需在桌子上堆满额外的外围设备，或者根据需要将键盘、显示器和鼠标重新连接到每台新计算机。对于本地管理，这可以节省大量的时间和头痛。然而，对于远程管理来说，需要一个虚拟 KVM，尽管这些解决方案价格昂贵，但只需一小部分成本就可以围绕 Raspberry Pi 构建一个虚拟 KVM。[这款电脑还增加了更多功能](https://hackaday.io/project/188805-pi-cast-kvm-switch),它还可以切换主板上的 ATX 信号，并简化了引导电缆的管理。

虽然我们之前介绍的 PiKVM 确实包括切换这些信号以远程打开或关闭或重置计算机的能力，以及监控状态 led，但[VI lear]的这个项目在某种程度上简化了设计。通过使用内置的 USB 3.0 多路复用器，这些信号可以与键盘和鼠标所需的 USB 2.0 集成到同一根 RJ45 电缆中，从而消除了对 USB 电缆的需求，简化了电缆管理。当然，这意味着 PiKVM 帽子不会以同样的方式工作，所以[VI lear]已经创建了一个新的分线板来解决这些问题。

像这样的真正联网 KVM[或从](https://hackaday.com/2020/11/24/true-networked-kvm-without-breaking-the-bank/)中获得灵感的 PiKVM 都是不可或缺的工具，因为它们不仅允许计算机远程重置或重启，还允许计算机与其 BIOS 或引导设置进行交互，而不必实际访问计算机。能够使用 Raspberry Pi 有效地做到这一点无疑降低了许多人部署此类解决方案的成本和复杂性。
