# 启动到基本盒包黑仔图形引擎

> 原文：<https://hackaday.com/2020/06/23/boot-to-basic-box-packs-a-killer-graphics-engine/>

在家用电脑时代的早期，许多机器会直接启动一个 BASIC 解释程序。这是向大众教授编程的好方法。然而，在大多数平台上，图形程序慢得令人难以置信，这极大地限制了所能实现的东西。到 2020 年，这种限制将成为过去，颜色 Maximite 2 。(视频，嵌在下面。)

Color Maximite 2 是一款基于 STM 32h 743 it 6 微控制器的计算机，采用 Cortex-M7 32 位 RISC 内核和 Chrom-ART 图形加速器。它运行在 480MHz，有大量的咕噜声，使它能够向屏幕提供生动的图形，让人想起 16 位游戏机时代的最佳产品。Maximite 2 将该芯片与一个基本的解释器结合在一起，并配有高效的图形例程。这使得游戏的开发具有快速流畅的移动，大量的巨大精灵和详细的背景。

[cTrix]很好地展示了由[Geoff Graham]和[Peter Mather]设计的机器。通过一系列演示来测试计算机，它展示了硬件令人印象深刻的视觉和听觉能力。它是阿米加时代 BlitzBASIC 的优秀精神继承者。特别令人愉快的是看到一个基本的解释器增加了语法高亮——使得解析代码更加容易！

我们很乐意看到这成为一个现成的套件，因为很明显，这个平台可以为复古爱好者提供很多东西。与近十年前的最大极限相比，这无疑有了很大的进步。休息后的视频。
