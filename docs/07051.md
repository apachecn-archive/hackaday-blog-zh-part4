# 梅林皮相机是一个摄影奇才

> 原文:[https://hack aday . com/2020/07/01/merlin-pi-camera-is-a-photographic-wizard/](https://hackaday.com/2020/07/01/merlin-pi-camera-is-a-photographic-wizard/)

[【M 先生】非常兴奋地摆弄新的高质量树莓 Pi 相机，并围绕它建立一个项目](http://www.instructables.com/id/1979-Merlin-Pi-High-Quality-Camera/)。不幸的是，禁闭迫使他在手头的旧技术中翻找，而不是在野外寻找一个新的引人注目的外壳。只要有六节 AA 电池可以为饥饿的 20 世纪 70 年代 led 供电，我们就会花很多时间玩这些梅林玩具，所以我们认为[M 先生]应该更经常地探索他的个人商店。

在我们忘记之前——这很酷；这个已经坏了。Merlin Pi 相机的魔法在两个层面上发挥作用——[ M 先生]可以通过为触摸屏打造的何炅拍摄静态照片和录制视频，或者回到过去，使用 Merlin 控制面板上的小按钮。[M 先生]在 GUI 中加入了 Dropbox 上传程序，这样他就不用担心在拍摄过程中 SD 卡被后院的鸟类电影填满了。

[M 先生]说他在试图浸湿贴纸时不小心弄弯了梅林的电池盖，不得不使用一块丙烯酸。虽然这很不幸，但我们认为这可能是因为相机镜头需要一个巨大的洞。休息之后，请观看构建视频。

如果你到现在才听说这个结实的新相机模块，我们自己的[Jenny List]几个月前让它成为焦点，最近[亲自尝试了一下](https://hackaday.com/2020/07/01/why-are-digital-cameras-still-boring/)。
