# 建造你设计的卡坦板

> 原文:[https://hack aday . com/2021/07/30/build-that-catan-board-you-designed/](https://hackaday.com/2021/07/30/build-that-catan-board-you-designed/)

几个月前，我们发布了关于卡坦板的电子定居者的一天设计。现在他发布了一段视频。他的第一个视频是关于游戏的设计，特别是电子元件。在此视频中，[Sam]将带我们了解主板的物理构造。

[![](../Images/2837c1a4d24bf3e72f4f63fb3f0fb9d0.png)T2】](https://hackaday.com/wp-content/uploads/2021/07/Catan2_Play.png)

去过几次他当地的创客空间后，他可以切割木板的木质部分，以及每块木板顶部的丙烯酸六边形。即使有了数控机床，仍有一些清理工作需要完成。用凿子把木头的边缘清理干净并染色后，就该把电路板放进去，给它们接线并编程了。该构建包括一个掷骰子——按下按钮显示掷出的数字，以掷出的数字的形式点亮瓷砖。最后一点是邀请一些朋友来玩游戏。

在上一篇文章的设计过程和这篇文章的构建过程之间，我们可以很好地了解[Sam]从开始到最终产品的设计方式。看看我们之前关于[Sam]设计的文章以及一些[其他](https://hackaday.com/2020/02/09/why-settle-for-the-standard-catan-board/) [卡坦](https://hackaday.com/2019/08/20/get-ready-for-games-night-with-3d-catan/) [文章](https://hackaday.com/2015/02/28/laser-cut-settlers-of-catan-board-best-christmas-gift-ever/)。
