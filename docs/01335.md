# 3D 打印有刷电机易于可视化

> 原文:[https://hack aday . com/2018/12/01/3d-printed-brushed-motor-is-easy-visualized/](https://hackaday.com/2018/12/01/3d-printed-brushed-motor-is-easy-to-visualize/)

一台电动机——或一台发电机——需要一些普通磁铁和一些电磁铁。通常的安排是有一个电刷换向器，它既给电磁铁供电，又在电机旋转时切换它们的极性。永磁体不会旋转，当它们摆动时，不会吸引或排斥电磁体。这可能有点难以想象，但如果你 3D 打印[米勒星球的] [工作模型](https://www.youtube.com/watch?v=L6Ey8AkqEeQ)——或者只是看下面的视频——你就可以看到它是如何工作的。

我们认为最难的部分是缠绕大电磁铁。让轮轴(钉子)居中也很难，但从视频来看，这似乎并不重要。到 3D 模型文件的链接有问题，但是看起来[这个可以工作](https://www.tinkercad.com/things/h8oxTd0DE44#/)。

你真的可以看到电刷和换向器如何通过线圈改变电流方向。作为一个简单的演示，电刷可能不像真正的电机那样坚固，但为了理解它是如何工作的，这是一个很好的演示。

如果你更愿意建造一个无刷电机，我们在之前已经[见过了。如果你想要一个没有 3D 打印的更快的有刷电机，你可以用](https://hackaday.com/2016/04/23/a-simple-and-educational-brushless-motor/)[和 3 根电线](https://hackaday.com/2018/02/01/three-wires-one-motor/)来建造一个，但是在那个设计中很难看到刷子。
