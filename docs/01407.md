# 将您的电暖器连接到互联网(简单而便宜)！

> 原文:[https://hack aday . com/2018/12/01/将你的电暖器连接到互联网——简单而便宜/](https://hackaday.com/2018/12/01/connect-your-electric-heater-to-the-internet-easily-and-cheaply/)

冬天已经来临，现在大多数家庭应该已经不再使用白炽灯泡，所以我们不能用那种方式取暖。为了避寒，edent 投资了一条电热毯。这不是普通的电热毯——不，[这是一个连接到互联网的电热毯](https://shkspr.mobi/blog/2018/11/building-an-alexa-powered-electric-blanket/)，由 Alexa 供电。

这是一个[edent]和他妻子的项目，由于需要双加热区，事情变得稍微复杂了一些。是的，双区电热毯是存在的(就像两个电热毯和缝纫机一样)，但真正的问题是找到一个插上电源就能打开的毯子。谁会想到一个简单的电阻加热元件会如此复杂？

对于这个项目面向互联网的一面，[edent]正在使用一个 [Meross 智能插头](https://shkspr.mobi/blog/2018/08/review-meross-smart-plugs-and-surge-protector/)和一个 [Sonoff S20 智能插头](https://shkspr.mobi/blog/2017/05/review-sonoff-s20-wifi-plug/)。这些都是通过与 Alexa 的工作，并配置为一个'电热毯'组。简单地说，“Alexa，打开电热毯”就能打开床。

有几个问题需要进一步改进。Alexa 不识别声音，所以说‘转到我这边的床’不起作用。毯子也会在一小时后关闭，但插头插座仍然通电。也有可能黑客会闯入这个 Alexa 并烧毁房子，但这是一个互联网上的设备；这种事情实际上从来没有发生过。
