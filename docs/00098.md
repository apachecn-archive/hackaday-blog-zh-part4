# 夏普斯和胶棒战斗巨美金属加工蓝调

> 原文：<https://hackaday.com/2018/07/20/sharpies-and-glue-sticks-fight-the-gummy-metal-machining-blues/>

“巨美”可能不是描述金属时脑海中浮现的形容词，但任何一个经历过铝堵塞钻头或端铣刀的人都会告诉你，某些金属确实表现出无益的行为。但是一篇新的研究论文试图揭示胶粘金属现象，可能只是机械师们在储备办公用品。

有点违反直觉的是，像钢这样较硬的金属往往比较软的金属更容易切割；尤其是铝，还有铜、镍合金和一些不锈钢合金。但是它确实发生了，普渡大学的 Srinivasan Chandrasekar 和他的同事们想找出原因，以及可以做些什么。因此，第一项工作是近距离接触切削工具和金属材料之间的界面，观察切削动态。在一段有趣的视频中，他们看到较软的金属倾向于以弯曲的方式折叠，而不是在确定的剪切平面上断裂。

之前已经注意到，切割普通机械师的标记液 Dykem 会改变软金属中的碎片形成，研究人员测试了从 Sharpies 到胶带甚至修正液的所有东西，发现它们都在一定程度上有助于减少胶粘作用。在他们的显微镜下，他们可以清楚地看到，一旦切削刃碰到处理过的表面，碎片会以不同的方式形成，倾向于表现得更脆，并弹出而不是折叠。他们还注意到，经过处理的金属的切削力明显下降，而且靴子的表面光洁度也大大提高。

记号笔和胶棒会像[量块绞拧](https://hackaday.com/2017/01/09/machinist-magic-gauge-block-wringing/)一样进入老机械师的戏法书中吗？只有时间能证明一切。但是现在，这是一个非常有趣的研究，你可以在你的店里进行测试。让我们知道你在评论中发现了什么。
