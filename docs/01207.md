# 乐高·塔迪斯在虚空中旋转

> 原文：<https://hackaday.com/2018/11/12/lego-tardis-spins-through-the-void/>

使用乐高技术齿轮和棒似乎是一个很好的方式来为你的常规乐高创作带来动画。使用齿轮和曲轴，你可以制作你最喜欢的电视节目或电影中的模型，就像乐高运动雕塑制作人，[乔希·大卫]在制作一个[旋转的 TARDIS](https://www.youtube.com/watch?v=nP1uaxbEd_w) 时所做的那样。转动把手，雕塑就会在空间和时间中旋转。

大齿轮保持在原位。由曲柄转动的隐藏齿轮从下方转动轴，该轴穿过大齿轮，使 TARDIS 绕主轴旋转。与 TARDIS 型号相连的是一个较小的齿轮，以一定角度与较大的固定齿轮啮合。这个更小的齿轮导致 TARDIS 绕着自己的轴旋转，而整个东西都绕着主轴旋转。如果你的手太累了，你可以用乐高玩具代替。

这是一个整洁的效果，你可以得到计划[Josh]的[Etsy 页面](https://www.etsy.com/listing/636366710/lego-phone-box-kinetic-sculpture?ref=shop_home_feat_4)。然而，最棒的是，你可以得到一套包含所有部件的！TARDIS 在这里很受欢迎，我们有很多以它为焦点的项目:从[树顶](https://hackaday.com/2010/12/27/tardis-tree-topper/)到[低音扬声器](https://hackaday.com/2016/04/06/this-tardis-is-an-infrasonic-subwoofer/)的一切。当然，我们唯一的问题是，‘它的内部更大吗？’？'
