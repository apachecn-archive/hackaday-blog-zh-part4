# 另一个闪亮的灯光项目——带有新冠肺炎风格

> 原文：<https://hackaday.com/2020/03/24/another-blinky-light-project-with-a-covid-19-twist/>

似乎所有人现在都在谈论病毒恐慌，这让我们大多数人都有了一点额外的时间。[保罗·克林格]——一个我们以前见过的名字——建造了一个闪烁的 LED 项目来打发时间。那又怎样？嗯，这些灯看起来像新型冠状病毒病毒，发光二极管闪烁着病毒 RNA 代码。你可以在下面的视频中看到结果。

当你考虑到我们已经看到【保罗】制造[微小的东西](https://hackaday.com/2019/02/23/ultra-tiny-pc-plays-snake/)甚至[闪现出他自己的 DNA](https://hackaday.com/2020/02/01/art-imitates-dna/) 时，这并不奇怪，所以他显然在这个领域有一些特殊的兴趣。

该项目需要 22 个不同颜色的发光二极管。一个微小的 CPU 使它变得实用。正如您可能预料的那样，真正的问题是在您构建它的时候让所有的东西都呆在一个地方。线框有一个临时铰链，所以你可以破解虚拟病毒，更换电池或重新编程控制器。

即使有所有的发光二极管，[保罗]说，他得到一个单一的 CR2032 硬币电池一天。我们认为在危机结束后，你可以把它作为圣诞树的装饰品。或者为一个婴儿手机复制几份(只要它离孩子太远而够不着)。
