# Turbo Rascal 是我们一直想要的复古 Pascal 编译器

> 原文:[https://hack aday . com/2021/11/30/turbo-rascal-is-the-retro-Pascal-compiler-we-always-wanted/](https://hackaday.com/2021/11/30/turbo-rascal-is-the-retro-pascal-compiler-we-always-wanted/)

Pascal 并不是当今最重要的编程语言之一；随着世界转向 C#、Python 和 Java 等新奇事物，它已经退居幕后。然而，这种语言有它的粉丝，其中一个人开发了一个新的针对复古平台的编译器—[,它被命名为 Turbo Rascal。](https://lemonspawn.com/turbo-rascal-syntax-error-expected-but-begin/)

支持的平台列表非常广泛，Turbo Rascal 能够为 C64、Amiga 500、BBC Micro、IBM PC、Atari ST、Game Boy、Amstrad、NES、ZX Spectrum 等编译高度优化的二进制文件。有一个可用的 IDE，甚至包括图形编辑器，可以快速地将项目放在一起。也被称为涡轮流氓语法错误，或 TRSE，这是一个人的工作。

该编译器可以在 64 位 Windows、Linux 和 OS X 上运行，YouTube 上也有大量的教程视频[。因此，没有借口不立即开始开发一个新的复古游戏。看看下面的演示视频，记住——只要我们继续使用它，](https://www.youtube.com/watch?v=nWCy9zowrGw&list=PLngJUKDQkcxbZllq72bKk5WuK8uvjcJia)[帕斯卡就不会死！](https://hackaday.com/2018/04/08/orbital-mechanics-on-a-vintage-kaypro/)
