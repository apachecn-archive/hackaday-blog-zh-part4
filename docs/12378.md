# 用 Kabooms 保持菲律宾地表水的清洁

> 原文:[https://hack aday . com/2022/01/02/keeping-the-Philippine-surface-waters-clean-with-kabooms/](https://hackaday.com/2022/01/02/keeping-the-philippines-surface-waters-clean-with-kabooms/)

YouTube 上[热带海洋清理](https://www.youtube.com/channel/UCXEq1v110PGMQ3fDqZpIqdg/videos)的[Rich]一直在努力阻止塑料垃圾进入菲律宾附近的水域。即使作为一个几乎只有一个人的团队，他也在使用一个吊杆系统收集大量的塑料垃圾，这个吊杆系统是由垃圾制成的:旧轮胎和空塑料瓶。这种 Kaboom 系统是一种低成本的废物收集方法，可以收集和妥善处理。此外,[Rich]还安装了容器，当地人可以在那里处理他们的塑料垃圾。

Kaboom 系统由【Rich】在[本视频](https://www.youtube.com/watch?v=XSwN2kEPbZ4)中详细介绍(休息后也有链接)。作为一个小本预算项目，它严重依赖捐款和当地的支持来安装更多的围油栏。然而，这是一种非常有效的方法来防止这种常见的塑料垃圾进入海洋。这些围栏是由人类经常出没的废弃物品制成的，这应该很容易做到。

理想情况下，地方政府应该安装这种捕获系统和简单的废物处理选项，但有时像这样的基层努力似乎会带来最快的变化。
