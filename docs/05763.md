# 悬浮星际飞船模型即将着陆

> 原文:[https://hacka day . com/2020/02/25/悬浮-星际飞船-模型-来着陆/](https://hackaday.com/2020/02/25/levitating-starship-model-comes-in-for-a-landing/)

如果一切按计划进行，埃隆·马斯克表示，SpaceX 的第一代巨型星际飞船将在 2020 年底前进行轨道飞行。这是一个相当大胆的说法，但当你像老科幻杂志中那样让火箭尾部着地似乎成了家常便饭时，我们认为你有权利虚张声势。我们很高兴看到未来几个月内飞行器的发展，但即使真的保持停飞，我们也很乐意接受[Chris Chimienti]的这个“飞行”星际飞船模型作为安慰奖。

[![](../Images/6f36b3998a46ae50cb9c9aedd8a87b1e.png)](https://hackaday.com/wp-content/uploads/2020/02/3dpstarship_detail.jpg)

A magnetic levitation module, we’re officially in the future.

对他在网上找到的星际飞船的 3D 打印模型感到有点失望，[克里斯]开始建造自己的模型。但是仅仅让他更大、更强、更精确地适应星际飞船目前的设计是不够的；他还想让它更刺激一点。火箭坐落的“云”支架上嵌入的一些 RGB LEDs 和 Arduino 是一个好的开始，受 SpaceX 的真正自主太空港无人机船*启发的着陆垫看起来很棒*都亮了。

但这是我们正在谈论的星际飞船，一种可以真正推动人类成为多星球物种的交通工具。说句公道话，你真的要把它打得落花流水。所以[克里斯]在网上找到了一个可以支撑几百克的磁悬浮模块，并着手让他的塑料星际飞船真正悬停在着陆台上。

正如你所想象的，这有点棘手。火箭的第一个版本看起来很棒，但太重了，所以他改用所谓的“螺旋花瓶模式”打印模型，使其完全中空。现在它轻多了，底部装有磁性板，足够稳定，可以自己漂浮。作为最后的润色，[克里斯]在星际飞船的底座上添加了一些红色发光二极管和一个硬币电池，这样看起来就像这艘光滑的飞船正在用它“不可能”的全流式分级燃烧发动机进行最后一秒钟的着陆燃烧。

这[不是我们第一次看到模型火箭](https://hackaday.com/2018/01/13/led-stand-for-lego-saturn-v-boldly-goes-where-no-lego-has-gone-before/)和[下面的电子发光云](https://hackaday.com/2019/07/19/custom-bases-make-lego-spacecraft-even-cooler/)，但这肯定是我们第一次看到可以悬浮在半空中的火箭。虽然这个小火箭可能无法一路到达火星，但在不久的将来，我们不会惊讶地看到它落在其他黑客和制造商的办公桌上。
