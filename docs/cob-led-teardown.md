# COB LED 拆卸

> 原文：<https://hackaday.com/2019/02/15/cob-led-teardown/>

[大克莱夫]拿起一些非常不寻常的用于水培的板上芯片(COB)发光二极管，并开始用视频检查它们。尽管几乎马上就损坏了电路板，他还是设法在这些阵列上做了一些测试，你可以在下面的视频中看到结果。他还将其与旧的 LED 模块进行了比较。

144 个发光二极管产生大量的光。除了给设备通电之外，他还在放大的情况下查看了 led 的结构，比较了使用微小焊线进行连接的旧版本和直接焊接在电路板上的新版本。

电路板的右侧有一个交流连接、一个防止过压的金属氧化物变阻器和一个桥式整流器。另一侧包含一个 LED 吸电流电路。[Clive]做了一点数学计算，决定 led 的排列方式可以让您选择是否让 led 簇接受 220V 输入或 110V 输入。

当前接收器中的设备在互联网搜索中没有显示。[Clive]做了一些探查和分析，以了解电路中发生了什么。有一个集成电路，一个齐纳二极管，和一个结实的晶体管。

板上芯片并不是很新，但它给实验者带来了表面贴装曾经带来的同样的问题。这些似乎使用“翻转”方法，电路板上放置有焊球，当所有东西都经过烤箱时，放在倒置的组件上。然而，在某些情况下，使用激光或超声波方法焊接精细的接合线或带，其方式与传统 ic 连接到其引线框架的方式相同。

自 2014 年或 2012 年[以来，LED 照明已经走过了漫长的道路。你一定想知道我们还要多久才能看到《星际迷航》风格的](https://hackaday.com/2012/04/13/stair-accent-lights-made-from-cheap-led-strips/)[灯光面板](https://memory-alpha.fandom.com/wiki/Lighting_panel)。
