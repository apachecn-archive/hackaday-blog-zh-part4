# 坚固的水火箭发射器让工程恰到好处

> 原文：<https://hackaday.com/2020/11/05/robust-water-rocket-launcher-gets-the-engineering-just-right/>

通常，当我们遇到一个声称过度工程化的项目时，我们承认我们会有点兴奋。这样的项目总是拥有娱乐性的夸张设计、材料和方法的潜力。不过，在这种情况下，我们将恭敬地不同意[扎克·希普斯]对[他的遥控汽水瓶火箭发射器](https://www.youtube.com/watch?v=7p8Ur9hALEs)的评估，认为它“过度工程化”。对我们来说，似乎刚刚好。

这并不是要剥夺这个版本所完成的任何东西。事实上，我们对建造的完整性印象深刻，这是为了建立一个充电和发射空气动力水火箭的站。这一过程始于一个原型，主要由 3D 打印的零件建造，但也有一些车间废料将其组装在一起。这使得[Zach]能够测试零件的几何形状、机械装置的操作以及它如何与 2 升汽水瓶颈部的法兰连接。

老实说，原型本身就很好，可能是我们许多人会停下来的地方，但[扎克]继续前进。他将大部分打印部件变成了机械加工的铝和 Delrin，制成了一个非常坚固的气动支架。我们不得不说，钳口围绕瓶子边缘闭合的力度有点可怕——看起来它很容易夹住一个任性的手指。但是，如果他设法避免这种命运，这样一个丰盛的钻机应该让[扎克]飞行很长一段时间。或许它甚至可以发射两级水火箭。
