# 去除内脏并升级无冷却的激光“冷却器”

> 原文：<https://hackaday.com/2022/10/29/gutting-and-upgrading-laser-chiller-with-no-chill/>

得到一个便宜的二氧化碳激光切割机非常适合你的车间需求，虽然你可能会得到一个比声明的激光管弱，但它仍然会切割你需要切割的任何东西。然而，正如[RealTimeKodi]在项目后的博客文章中所展示的那样，你旁边的冷却设备可能就不是这样了。他们买了一台 CX3000“冷却器”，[发现它没有冷却器组件](https://twitter.com/RealTimeKodi/status/1547014413147938820) ( [Nitter](https://nitter.net/RealTimeKodi/status/1547014413147938820) )，只配备了一个散热器、一个风扇和一个泵。

让你的激光管水接近环境温度是你已经可以用一个水族箱泵和一桶水来实现的——而且不值得花 100 美元。带着沉没成本和未满足的对合适冷却器的需求，[RealTimeKodi]开始寻找可行的途径——首先是使用 TEC 元素。升级过程很有趣，但结果却不尽如人意，因为元件消耗了大量能量，却几乎没有任何有用的输出。

[RealTimeKodi]没有放弃，最终找到了一台旧的饮水机，里面有类似冷水机的组件，售价 200 美元。他们本来可以按原样使用喷泉，但一些设计问题和对冒险的渴望阻碍了他们，无可争议地迫使他们将喷泉的内脏塞进 CX3000 的外壳。

购买一台激光切割机有时会感觉像十年前购买一台 3D 打印机——你得到一台 K40，[学会使用它](https://hackaday.com/2018/09/27/laser-noob-getting-started-with-the-k40-laser/)，[添加缺失的安全功能](https://hackaday.com/2017/11/22/hacking-a-k40-laser-cutter/)，[自动对焦中的 mod](https://hackaday.com/2018/12/02/spring-loaded-bed-for-k40-laser-acts-as-an-auto-focus/)，[升级控制板](https://hackaday.com/2017/06/10/drop-in-controller-for-ebay-k40-laser-engraver-gets-results/)，[扩展工作面](https://hackaday.com/2018/07/19/expanding-the-k40-laser-cutter-with-aluminum-extrusion/) …也就是说，我们的经验表明，如果 A4 大小的 3 毫米木材切割适合你，你不需要任何这些东西，但一台合适的冷却器仍然值得其重量的镀金丙烯酸树脂。
