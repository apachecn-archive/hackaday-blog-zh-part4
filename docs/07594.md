# 你从来没有见过这个 RetroPie 模拟器控制台:西瓜

> 原文:[https://hack aday . com/2020/08/26/youve-never-seen-this-retro pie-emulator-console-西瓜/](https://hackaday.com/2020/08/26/youve-never-seen-this-retropie-emulator-console-watermelon/)

Raspberry Pi 是一个非常受欢迎的平台，用于模拟旧的控制台，RetroPie 框架使其很容易立即开始使用。通常，这些单板计算机会内置在有趣的街机盒或复制控制台外壳中，以增加魅力。这些都已经完成了，所以相反，[Cedishappy]决定走自己的路——这导致了精彩的[西瓜游戏机。](https://www.youtube.com/watch?v=dqgjozNR3W8&feature=youtu.be)

听起来像是在一个独特的外壳中建造一个 RetroPie 钻机的琐碎练习，实际上伴随着一些工程挑战。基本功能都很标准——GPIO 引脚连接按钮、扬声器和屏幕，模拟 Gameboy 的进步。但是机械实现更复杂。西瓜首先被切开，去掉红色的果肉，只留下瓜皮。然后用纸和纸板模板为按钮和屏幕打孔。不幸的是，热胶对西瓜不起作用，因此，牙签被用来固定屏幕和扬声器。为了保护电子产品免受内部潮湿环境的影响，树莓皮和其他需要的组件上使用了透明的食物包装。

[Cedishappy]超越了项目视频，迷人地展示了旁观者对令人困惑的游戏系统的反应。电子产品与水果和蔬菜的结合是一个我们不常探索的领域；我们自己的[Mike Szczys]制作了一个华丽的 LED 南瓜灯，看起来真的很棒。休息后的视频。
