# 自主多任务执行机器人

> 原文：<https://hackaday.com/2020/09/14/autonomous-multi-task-performing-robot/>

[Ruchir]对机器人技术很感兴趣已经有一段时间了，并且总是被越来越受欢迎的避障机器人逗乐，但是想要一些可以做更多事情的东西。因此，像任何优秀的黑客一样，他决定自己开发一些东西。

他想将所有流行的初学者机器人功能整合到一项发明中。他的机器人可以跟随一条线，检测一个障碍，并检索一个对象，而无需在模式之间切换。它甚至可以跟随另一个机器人，这是非常整洁的。

他的机器人有很多你能想到的硬件。它使用 Raspberry Pi 来处理所有繁重的图像处理，具有光学传感器来跟踪线路和避免障碍，并包括一个用于音频反馈的扬声器。特别酷的是令人印象深刻的界面，称为 Regbot GUI，这是[Ruchir]与他的机器人一起使用的。根据维基页面的消息，Regbot GUI 似乎伴随着丹麦技术大学的 Jens Christian Andersen 教授开发的教育机器人平台，用于向工程专业学生教授控制。[Ruchir]能够让 GUI 适应他的特定机器人，没有问题。

使用 Regbot GUI，[Ruchir]可以实时监控机器人的所有传感器数据(加速度计、陀螺仪、距离传感器、伺服、编码器等。)，根据需要动态调整其校准设置，甚至提供通用的 killswitch [，以防不可思议的事情发生](https://hackaday.com/2016/06/27/taking-killer-robots-seriously/)。我们会说，在你着手下一个机器人项目之前，它绝对值得一看。
