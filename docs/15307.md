# 带着神奇宝贝散步

> 原文:[https://hackaday.com/2022/11/09/taking-pokemon-on-a-walk/](https://hackaday.com/2022/11/09/taking-pokemon-on-a-walk/)

模仿旧电脑或视频游戏系统并不总是为了重现童年怀旧或在已经不存在的硬件上玩经典游戏。很多时候，这是一个学习视频游戏编程技巧的好方法。很多老游戏都有源代码，任何人都可以随意修改，其中之一就是神奇宝贝翡翠。这是[Inkbox]玩的第一个神奇宝贝游戏，[，他用这个定制的 ROM 文件](https://www.youtube.com/watch?v=2tpBooD_X7k)给它添加了一些现代特征。

这个游戏的第一个特点是可以让你的神奇宝贝在地图上跟随他们的角色。这在后来的游戏中很常见，但在祖母绿和红宝石首次出现时还不是一个功能。[Inkbox]需要将后期游戏中的精灵导入到 Emerald 游戏文件中，转换它们的调色板以匹配游戏的调色板，然后开始处理机制。在一切都完成后，神奇宝贝不仅跟随玩家在地图上移动，还会被激活，进入和退出它们的神奇宝贝球，甚至以可信的 32 位方式从壁架上跳下。

像这样的老游戏的一个优点是，它们已经存在了足够长的时间，可以获得源代码或反编译代码，它们通常有大量的文档，并且它们运行的平台现在也是众所周知的。神奇宝贝翡翠在这方面并不孤单；事实上，有一个巨大的 [Game Boy Advance homebrew 场景并不太难涉足](https://hackaday.com/2022/08/29/a-primer-for-the-homebrew-game-boy-advance-scene/)。
