# 用胶棒使渗透性调谐振荡器稳定

> 原文：<https://hackaday.com/2022/10/19/permeability-tuned-oscillators-made-stable-with-a-glue-stick/>

一个多世纪以来，业余无线电爱好者已经用一个线圈和一个可变电容器制作了调谐电路。近几十年来，随着 SDR 技术取代传统调谐电容器，可变电容器的供应已经减少。业余无线电爱好者不再需要调谐电路了？不尽然，[正如[Bill Meara N2CQR]在 break](https://www.youtube.com/watch?v=QTdQqCmw_0U) 下方的视频中向我们展示的那样，通过使用磁导率调谐制作可变电感。这算不上什么高科技，主要部件就像胶棒一样简单。

磁导率调谐电感器具有通过螺钉移入和移出其中心的磁芯。胶棒在其末端的旋钮上的丝杠上具有胶芯，因此用铁氧体环代替胶水的旧胶棒形成合理的磁导率调谐线圈架。线圈缠绕在它的外面，当组装成振荡器时，它提供了一个有用的调谐范围。这不是一个新的想法，因为渗透率调谐可以在汽车收音机和电视调谐器以及其他应用中找到，但它仍然是一个值得记住的好技巧。
