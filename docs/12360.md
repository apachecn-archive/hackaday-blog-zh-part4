# 内置 AMD 处理器的华丽书桌

> 原文:[https://hacka day . com/2022/01/04/a-huarly-desk-with-amd-inside/](https://hackaday.com/2022/01/04/a-gorgeous-desk-with-amd-inside/)

我们第一个承认我们在 Hackaday 没有看到太多木工活。但是这张内置电脑的桌子实在是太漂亮了，不能不与人分享。

该建筑主要是樱桃贴面半英寸胶合板和真正的樱桃。有几十个角度和复杂的作品，都适合在一个宝贵的和强大的办公桌。桌子的中心是进气格栅，采用 2019 年苹果 Mac Pro 风格的饰面。[John]提到他以前曾用一个停着的钻头和一些耐心手工完成过，但他更喜欢自动化的方式。两个小房间位于中央通风口的两侧，由胶合板制成，上面粘有樱桃贴面。一个 USB 集线器隐藏在其中一个小房间的后面，暴露了隐藏在里面的 AMD 驱动的桌面 PC 的所有 I/O。桌子的顶部装有铰链，方便拿取电脑。[约翰]声称他做了已知宇宙中最酷的桌子。我们不知道我们是否能说它是最酷的，但我们肯定很欣赏制作它的过程和专业知识。

在你完成了受[约翰]项目启发的新办公桌后，也许你会对[一个悬浮涡轮办公桌玩具](https://hackaday.com/2021/09/05/building-a-levitating-turbine-desk-toy/)感兴趣以达成交易。休息后的视频。
