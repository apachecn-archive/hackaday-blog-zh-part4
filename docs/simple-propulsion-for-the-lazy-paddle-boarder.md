# 懒人划水边界的简单推进

> 原文：<https://hackaday.com/2022/08/01/simple-propulsion-for-the-lazy-paddle-boarder/>

健康的户外活动的一个缺点是所有的运动。如果你能造出一些东西来为你做艰苦的工作，谁会愿意去做呢？这似乎是[Bitluni]最新作品的主题，这是一个用于立式桨板的简单(且粗糙)的推进系统。

在获得一个充气站立式桨板并尝试了几次后，[Bitluni]决定跳过“站立”和“划水”部分。他设计并打印了一个非常简单的螺旋桨，他打算用一个无刷电机和速度控制器来驱动它。在钻出支柱来安装轴的过程中，他意识到自己把事情搞得太复杂了。所以他决定用电池供电的电钻来代替。对于轴管，他修改了一个旧拐杖，在手柄上为轴钻了一个孔，并在另一端添加了一个带轴承的管道。他还在手柄上附加了一个钩环来固定在桨板上。

在湖上进行的测试表明，推进系统在概念验证方面表现相对较好，但存在一些缺陷。为了正确地潜入水中，[比特鲁尼]必须面朝后坐在桨板的后面。如果离地面太近，它会吸入空气，失去推力，或者用水喷他和他的钻头。当然，在这个过程中也存在淹没他的钻头的真实风险。

项目不需要复杂到令人愉快，通过快速创建概念证明，而不是花很长时间来提出“完美”的设计，您通常可以学到更多。

如果你想看一些更先进的水上项目，可以看看由[【RCLifeOn】](https://hackaday.com/2020/09/08/electric-surfboard-doesnt-need-waves/)和[【安德鲁 W】](https://hackaday.com/2021/11/25/surfboard-gets-jet-upgrades/)建造的喷水动力电动冲浪板。
