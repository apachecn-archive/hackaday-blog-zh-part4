# 物联网磁带滚动器从不需要铅笔

> 原文：<https://hackaday.com/2020/03/10/iot-cassette-scroller-never-needs-a-pencil/>

20 世纪 80 年代和 90 年代的透明电子产品热潮显然对[MisterM]产生了影响，我们完全可以理解。正如我们可以证明的那样，里面那些糖果色的元件无疑让几千个孩子朝着电子学的方向前进。

尽管可能性似乎对他非常不利，但[MisterM]能够在标准盒式磁带中安装[滚动物联网通知器的所有必要组件。树莓派 Zero W 和捐赠者的盒子都做了一些手术，目的是让所有的组件都适合这样一个狭小的空间。我们很高兴他坚持了下来，因为它看起来很棒。](https://www.instructables.com/id/Cassette-Pi-IoT-Scroller/)

拉斯皮用的是阿达果。IO 和 IFTTT 获取各种通知——推文、天气、土壤湿度，你能想到的——并在 11×7 LED 矩阵上滚动它们。振动圆盘电机首先发出嗡嗡声，因此[MisterM]不会错过任何东西。点击中断按钮，把这个东西翻过来，因为构建视频都在 B 端排队。

如果你更喜欢玩卡式录音机，[添加一些回放速度电位计来扰乱声音](https://hackaday.com/2019/09/25/hacked-tape-player-makes-for-a-unique-instrument/)，或者[全力以赴，做一个电子管](https://hackaday.com/2013/09/10/the-melloman-mk-ii/)。
