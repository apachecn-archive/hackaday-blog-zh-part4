# 家酿 16 位计算机是一个电线包裹的艺术品

> 原文：<https://hackaday.com/2020/07/24/homebrew-16-bit-computer-is-a-wire-wrapped-work-of-art/>

如今，试验板 8 位计算机构建似乎风靡一时，这是有充分理由的:使用离散逻辑芯片从板上构建自己的 CPU 是真正了解微处理器如何工作的一个好方法。更不用说这是一个令人难以置信的弯曲。但是一旦你征服了八位，你会怎么做？简单:用 74HC 逻辑芯片制造一台 16 位计算机。

细心的读者可能会记得这台计算机的建造者【保罗·康斯坦丁诺】，因为[他以前在 8 位试验板计算机上的工作](https://hackaday.com/2019/05/09/cpu-made-from-74hc-chips-is-a-glorious-mess/)。尽管那团乱糟糟的电线非常熵，但对保罗来说，维护它肯定是一场噩梦。因此，当升级的时候，他明智地选择了一种更综合的建造方法。构建方法是绕线，将多个卡插入背板，通过带状电缆连接。整个卡笼比之前的版本要整洁得多，而且看起来适合快速修改。笼子里最上面的卡现在充当控制面板；最终，[保罗]计划在笼子上放一个真正的前面板，以支持所有的开关和闪光灯等建筑需求。延伸目标包括支持音频和视频，并使机器在线，以便任何人都可以登录。

下面的视频是机器当前状态的概述；播放列表中的早期视频更详细地介绍了设计和构建。我们希望很快看到原理图，我们很想知道在哪里可以得到一些我们自己项目的绕线印刷电路板。
