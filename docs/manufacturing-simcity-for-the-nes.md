# 为 NES 制造模拟城市

> 原文：<https://hackaday.com/2019/04/19/manufacturing-simcity-for-the-nes/>

去年年底，新闻披露了任天堂娱乐系统时代的极其罕见的人工制品。《模拟城市》是 PC 和几乎所有其他主机的经典模拟游戏，它是为 NES 编写的，但从未发行。现在有一个人终于找到了他的副本，它被丢弃在波特兰复古游戏博览会上，并最终被放到了互联网上。这是一个未完成的游戏，但它基本上是可玩的，即使它比 PC 版本更原始一点。

[Matt]想为 NES 建造他自己的模拟城市，这就是他所做的。这个项目花了几个月的时间和大量的研究，但最终有了一个看起来很专业的卡带版本。

随着模拟城市的 ROM 在互联网上的松散，这部分的建设相对容易。你仍然可以得到 EPROMs 或 EEPROMs，UV 橡皮擦，一个好的程序员会通过通常的供应商花 50 美元卖给你。互联网上甚至有地方将仿真器兼容的 ROM 文件分成两个文件，分别用于每个 NES 卡带中的角色和程序 ROM。这里的诀窍是用正确的映射器找到正确的墨盒。事实证明，只有四种游戏，你可以简单地将 SimCity ROM 芯片放入其中，并期望一切正常。所有这些游戏都很贵，但它们的 Famicom 版本很便宜。

在小心翼翼地拆下 Famicom 游戏，焊接上新的芯片，并贴上花哨的专业标签后，[Matt]为 NES 制作了他的卡带版《模拟城市》。不过，这是给一个家庭用的，不过你可以买适配器。看看下面的视频。
