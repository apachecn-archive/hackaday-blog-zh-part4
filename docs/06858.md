# 从源头编译一个水培系统

> 原文：<https://hackaday.com/2020/06/16/compile-a-hydroponics-system-from-source/>

只要有好的土壤，照料花园通常是一项有益的工作。如果没有，要么很快就不得不处理像沙子或硬粘土这样的土壤，要么每年不得不运送堆肥，成本会很高。或者，也可以建立一个不需要任何土壤的植物种植系统，尽管这需要一个自动化的系统来管理输送到植物的水和养分。

由[Kyle]设计的这个装置是独一无二的,因为它使用了他自己的开源软件 Mycodo 来控制水培系统。它被加载到 Raspberry Pi 4(他注意到现在可以从 USB 驱动器而不是 SD 卡启动[)上，它控制所有必要的外围设备，以确保水有正确的营养成分和化学成分。](https://hackaday.com/2020/05/25/boot-your-pi-over-usb/)

然而，这个建筑不仅仅是一个软件控制面板。[Kyle]介绍了设置小型水培系统的每个部分，该系统能够在室内有效地种植 15-20 株植物。他种植各种莴苣和罗勒，但是这个系统也可以用于更多种类的植物。稍加改变，一个类似的系统不仅可以种植这样的植物，也可以养鱼。
