# 建造一辆强大的手推车

> 原文：<https://hackaday.com/2021/07/10/building-a-big-ol-powerful-wheelbarrow/>

有时候你必须在很大的范围内搬运很重的货物。普通的手推车很好，但是当一个人不得不多次旅行时，它会很快变得很累。[车间从零开始]而不是[选择建立一个电动独轮车](https://www.youtube.com/watch?v=gVqFvvYOFgM)，有大量的咕噜声来转移负载。

建造完全是从地面开始的，由钢制 RHS 部分焊接而成，并提供了足够的机动性后转向。实际的转向工作是由一个从汽车上改装的齿条来完成的，该齿条安装在后轮组件的单侧附件上。这是一种非常干净利落的工作方式，看起来效果不错。驱动力通过静液压剪草机传动装置传递到前轮。一台 17 马力的发动机为手头的工作提供了足够的动力，甚至配备了电动启动装置，以便于使用。

令人印象深刻的是看到多少钻机是由原材料组装而成的；甚至燃料箱也是钢制的。我们以前见过类似的建造，比如这个整洁的带锯。休息后的视频。
