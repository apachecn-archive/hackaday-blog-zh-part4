# 另一个 Drawbot 使用 Pi 和 Web 套接字

> 原文:[https://hack aday . com/2018/10/08/another-draw bot-uses-a-pi-and-web-sockets/](https://hackaday.com/2018/10/08/another-drawbot-uses-a-pi-and-web-sockets/)

关于艺术。穴居人在墙上画画。人们不停地给绘图机器人编程。我们最近看到的一个是[Andy 的]draw bot,它使用 WiFi 和 WebSockets 在任何平面上绘画。此外，约翰逊县图书馆[有一篇很棒的关于他们如何建造图书馆的文章](https://www.instructables.com/id/Drawbot/)，如果你想尝试一下，你会发现他们的说明非常有用。下面的视频也非常鼓舞人心。

让这个建筑特别有趣的是，它使用了一个驱动系统，两个固定点连接着吸盘。有各种各样的 3D 打印零件——有些只是为了构建，有些是重新利用的旧零件。

由于该设备可以移动到不同的表面，软件需要知道线轴之间的距离以及笔(吊篮)的位置。一旦设置完成，SVG 文件就可以开始绘图过程。自从穴居时代以来我们已经走了多远。

驱动机制让我们想起了我们见过的一些[电缆机器人](https://hackaday.com/2017/10/20/cable-bots-arise-domination-of-the-universe-is-at-hand/)。我们最近还看到了一个[的 drawbot 徽章](https://hackaday.com/2018/09/16/drawbot-badge-represents-the-cnc-world-in-badge-design/)(没有关系)。
