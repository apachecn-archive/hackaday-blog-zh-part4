### Twitter 的提示(#5 会让你震惊)

Twitter 上最优秀、最聪明的人聚集在一起，给我们一些他们最好的建议。从经典的建议，如“经常修改密码”，到更现代的真理，如“只有被媒体报道才是关键的”。不要公布任何细节，以保持低严重性。”我个人最喜欢的是这样一条真理:漏洞缓解是一笔浪费的开支，钱最好花在 Web 应用程序防火墙上，它肯定会让你远离麻烦。

其他热门话题包括使用 Stackoverflow 解决所有安全补救问题的建议。另一个用户建议在你的网站上保留营业时间，这样它只能在你工作的时候被攻击。也许最好的建议是，你给你的狗起个至少 16 个字符的名字，混合使用，因为你不可避免地会用宠物的名字作为你的密码。查看`#cisotips`标签，了解更多这些令人敬畏的技巧！

### 如何知道何时举报家人电脑犯罪

这是一个旧的，但很好的。一个如何知道你的孩子何时越线的快速指南，你应该报警以遏制他们的电脑黑客行为。一些要注意的危险信号是 Tor、VM 技术或令人畏惧的 Kali Linux 的使用。同样令人担忧的还有专用黑客硬件，比如 WiFi 菠萝。特别值得注意的是 Metasploit，因为它是“渗透软件”。最后，小心不和谐的使用，因为这是黑客技巧主要分布的地方。

### 病毒警报

著名的计算机安全专家艾尔·扬科维奇[给我们带来了病毒警报](https://www.youtube.com/watch?v=zvfD5rnkTws)。(视频链接。)这个特别恶心，通过邮件传播。主要 PoC 是主题行“臭奶酪”。让这一个特别令人讨厌的是它如何跨越数字鸿沟，造成我们通常在恶意软件中看不到的破坏。不幸的是，补救步骤相当复杂。一旦发现感染，通常会立即关机。在此基础上，建议的解决方案是将机器重新安置到一个地质安全的位置，然后应用硅基缓解措施，直到问题完全解决。最后一步是彻底热消毒任何接触到感染的材料。传话下去，注意安全！我们下周见！
