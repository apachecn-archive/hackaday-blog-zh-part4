# 需要可破解的旋律？这是俄罗斯方块主题和更多

> 原文：<https://hackaday.com/2019/12/25/need-hackable-melodies-heres-the-tetris-theme-and-more/>

[Robson Couto]开始对音乐项目感兴趣，并且作为副作用，为各种各样的主题、歌曲和旋律创建了带有简单符号的可下载代码。它们都是用于 Arduino 的，并且只使用内置的`tone()`功能，但是不要让这分散你的注意力。如果你看过去，你会发现每一个小品都是由单个音符和持续时间组成的旋律；容易适应其他目的或简单地按原样使用。毕竟，[罗布森]希望每首曲子的来源易于理解，易于修改，并且没有外部依赖性。

所有这些听起来可能有点像 MIDI，但 MIDI 与硬件事件的共同点要比乐谱多得多，因为它包括(除其他外)作为独立元素的音符开始和音符结束。将 MIDI 转换成更有用的格式是一个项目的重要组成部分，该项目将巴赫音乐输入神经网络，并获得了令人惊讶的好结果。

在做音乐项目时，有时用音符和持续时间简单地表示一段可识别的旋律会是一个非常方便的资源，你可以在 GitHub 上找到它们。休息之后有一个俄罗斯方块主题的简短视频(实际名称:[](https://en.wikipedia.org/wiki/Korobeiniki)*)正在播放。*
