# 人类最好的机器人朋友

> 原文:[https://hackaday.com/2018/09/17/mans-best-robotic-friend/](https://hackaday.com/2018/09/17/mans-best-robotic-friend/)

当谈到机器人时，一些最有趣的工作——当然也是最搞笑的——来自波士顿动力公司，以及他们的实习生团队踢翻了机器狗。这是一个令人印象深刻的工程壮举，即使这些机器人驮骡对于它们在战场上的预期用途来说太大声了，但这也是一个展示一堆马达实际上有多酷的伟大展示。

它与波士顿动力公司的机器人不太一样，但[Dimitris]'[hack aday Prize](https://hackaday.io/project/158251-quadruped-robot-with-22-dof)的项目几乎是一个同样令人印象深刻的电机、3D 打印零件、SLAM 处理和反向运动学的组合。我想你也可以把它踢过来，看着它努力笑。

这只机器狗最初是在 Fusion 360 中建模的，设计有 22 个 Dynamixel AX-12A 机器人致动器:大而结实的串行可控伺服系统。当然，把一堆马达栓在一个框架上是比较容易的部分。这里真正的挑战是弄清楚运动学和教这只机器狗如何走路。这仍是一项正在进行的工作，但迄今为止[Dimitris]能够移动脊柱，保持双脚与地面水平，并让机器人稍微行走。还有工作要做，但是已经做了大量的工作。

这款机器人即将推出的功能包括安装在头部的 RealSense 摄像头，用于周围环境的 3D 可视化。还有一个尾巴的计划，大致基于我们见过的一些触手机器人。它完成后将会是一个伟大的项目，它已经是 Hackaday 奖的优秀参赛作品。
