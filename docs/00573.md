# 创建您自己的 ESP8266 盾牌

> 原文:[https://hack aday . com/2018/09/07/create-your-own-esp8266-shields/](https://hackaday.com/2018/09/07/create-your-own-esp8266-shields/)

ESP8266 在相对较短的时间内变得令人难以置信地受欢迎，这也难怪。价格低廉，功能强大，兼容 Arduino，最棒的是，开箱即用就有 Wi-Fi。但是尽管它的能力和受欢迎程度，它至少在一个方面仍然落后于 Arduino。也就是说，大量的附加“屏蔽”可以插入 Arduino，以添加从试验板到 GPS 接收器的所有东西。

[![](../Images/0e6746d9235f30da1dafe8669d954213.png)](https://hackaday.com/wp-content/uploads/2018/09/espboard_detail.png) 在自由市场决定加快步伐，开始为各种 ESP8266 开发板制作标准化屏蔽之前，看来黑客们将不得不收拾残局。[【Rui Santos】为广受欢迎的 Wemos D1 迷你板](https://randomnerdtutorials.com/esp8266-multisensor-shield/)制作了一个非常详细的一步一步的指导，这应该会给你很多灵感来旋转你自己的定制附加模块。

本指南以书面教程和两部分视频的形式呈现，涵盖了从在试验板上开发和测试电路到在 KiCad 中设计 PCB 并发送出去进行制造的所有内容。最终结果是一个专业外观的 PCB，与库存 D1 Mini 的尺寸相匹配，并添加了 DS18B20 温度传感器、PIR 运动检测器、光敏电阻和一些拧紧端子。

[Rui]继续展示如何通过 ESP8266 上的 web 界面利用新的传感器屏蔽，甚至将整个东西包装在 3D 打印的外壳中。如果你想生产出更有凝聚力的成品，这些都是值得一试的技能。

如果你正在为 ESP32 寻找一个类似的项目， [[Rui]你也在那里进行了报道](https://hackaday.com/2018/02/10/esp32-weather-station-on-a-pcb/)。你可能也对我们最近强调的的[系列 ESP8266 教程感兴趣。](https://hackaday.com/2018/08/26/demystifying-the-esp8266-with-a-series-of-tutorials/)
