# 黑客日链接:2020 年 10 月 25 日

> 原文：<https://hackaday.com/2020/10/25/hackaday-links-october-25-2020/>

Siglent 一直在大举进军中档测试设备市场，制造商的仪器出现在各地的工作台上。因 Kurokesu 而出名的 Saulius Lukse 发现自己拥有一个 Siglent SPD3303X 可编程电源，至少从硬件方面来看，它看起来是一个非常好的单元。然而，它附带的软件并没有完全点燃他的热情，所以 Saulius 想出了[一个 Python 库来控制电源](https://github.com/Kurokesu/siglent_psu_api)。这个库让他可以通过以太网端口控制电源的几乎所有方面。仍然有一些功能不太好用，到目前为止，他只使用他的特定电源进行了测试，但很有可能在其他 Siglent 仪器的命令集中至少有一些交叉。我们渴望看到其他人捡起它并加以利用。

从“每个人都需要一个爱好”部门，我们发现[这个 IBM 1401 大型机系统](https://www.youtube.com/watch?v=cM-1Un7r89I)的超详细模型非常吸引人。我们可能在它开发的早期写了这个，但现在看来，模型制造商 6502b 已经完成了整个设置，所以它有另一种外观。细节层次令人瞠目结舌——从操作员控制台到行式打印机，每件设备的最小特征都被重现。甚至带有系统文档的三环活页夹也在那里。不要让我们开始那些磁带驱动器，或在正确的收获黄金时期的小椅子。

说到娱乐，你有没有想过现在有多少人在太空中？或者有多少人有幸搭上了上楼的顺风车？这有一个数据库:[超级星团](https://www.supercluster.com/astronauts?sort=&ascending=false&life%20form=human&)上的宇航员数据库。它列出了几乎所有有意发射到太空的东西——人类和非人类——从 1961 年的尤里·加加林开始，到该俱乐部的最新成员[谢尔盖·库德-斯韦尔奇科夫](https://www.supercluster.com/astronauts/sergey-kud-sverchkov?sort=launch%20order&ascending=false&life%20form=human&)，他上周刚刚从他的家乡拜科努尔起飞进入国际空间站。所有人和所有东西都在那里，包括撞上月球的[【某缓步动物】](https://www.supercluster.com/astronauts/some-tardigrades-(4)?sort=launch%20order&ascending=false&life%20form=micro-creatures&)。他们甚至包括[这个家伙](https://www.supercluster.com/astronauts/starman?sort=launch%20order&ascending=false&limit=25&life%20form=robots&)，这让我们很好奇为什么他们不包括[那个臭名昭著的井盖](https://en.wikipedia.org/wiki/Operation_Plumbbob)。

最后，对于机械师来说，如果你曾经想知道颤振是什么样子的，就不要再想了。Breaking Taps 对端铣刀颤振进行了有趣的慢动作分析，结果有点出乎意料。这个镜头真的很酷——看着四槽立铣刀剥掉低碳钢并将细小的花饰扔到一边是非常令人满意的。当他引起喋喋不休时，高速投篮的价值是显而易见的；主轴、工件、老虎钳和几乎所有的东西都开始摆动，导致低质量的切割，最终，当超过其极限时，立铣刀的寿命戏剧性地结束。有趣的东西——让我们想起了[本·克拉斯诺在他的电子显微镜](https://hackaday.com/2015/09/05/better-image-capture-for-your-scanning-electron-microscope/)中对切屑形成的近距离个人观察。
