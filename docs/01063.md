# 周五黑客聊天:空中黑客

> 原文：<https://hackaday.com/2018/10/24/friday-hack-chat-air-hacking/>

软机器人领域肯定比你的标准伺服电机和金属框架机械臂要不成熟得多。也许这是因为制造一个灵活的机器人更难，也许是因为制造软机器人的最好方法只有十年左右的时间。也许，虽然，这是因为它很难控制空气。

[![](img/c665f9d56b87f5a8da199c91ff562472.png)](https://hackaday.com/wp-content/uploads/2018/10/airhackinghackchatposterv1-3.jpeg) 在本周的黑客聊天中，我们将和【阿米塔布·什里瓦斯塔瓦】一起讨论[空中黑客。[Amitabh]是 NYU ITP 的一名研究生，研究创新技术，在那里他制作互动艺术、研究工具以及用各种材料做实验。最近他一直在开发可编程空气，一种用于软机器人的气动控制器。我们已经在 ThiMaker Faire](https://hackaday.io/event/161903-air-hacking-hack-chat) 看到了他的作品，这是今年 Hackaday 奖中一个令人敬畏的项目。

在本次聊天中，我们将讨论 DIY 软机器人。软机器人是一个不断发展的领域，业余爱好者可以获得许多唾手可得的成果。除了分享关于建立你自己的软机器人的经验和资源，我们将谈论驱动！收听看看你如何在你的下一个项目中使用气动技术。

在本周的聊天中，我们将讨论:

*   气体力学
*   可编程空气
*   软机器人
*   将气动添加到项目中的方法

当然，我们鼓励你在讨论中提出自己的问题。你可以在 [Air Hacking Hack Chat](https://hackaday.io/event/161903-air-hacking-hack-chat) 上发表评论，我们会把它放在 Hack Chat 讨论的队列中。
