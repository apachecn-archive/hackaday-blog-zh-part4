# 实时物体检测，59 美元

> 原文:[https://hack aday . com/2021/03/26/59/实时物体检测/](https://hackaday.com/2021/03/26/real-time-object-detection-for-59/)

曾经有一段时间，让一台机器识别相机中的物体是很困难的，即使不尝试实时识别也是如此。但是现在，你可以用不到 60 美元的杰特森纳米板来做这件事。效果如何？[看下面【穆尔塔扎的】视频](https://www.youtube.com/watch?v=mB025B7KpeE)，看看你怎么看。

视频的前几分钟激起了我们的兴趣，这也是好事，因为 50 行代码得到了一个 50 多分钟的视频！不过，这值得一看，因为有很多关于如何在自己的项目中应用这种技术的好信息。

当然，完成这项工作的代码远不止 50 行。Jetson 代码和 OpenCV 做了很多繁重的工作。但是，利用强大的工具让艰难的工作变得简单是一件好事。

我们对神经网络推理图像的能力印象深刻。有一个人打开沙发变成床的图像，系统正确地将它识别为沙发和床。相当令人印象深刻。

我们已经看到[专用物体识别系统](https://hackaday.com/2018/11/09/pixy2-is-super-vision-for-arduino-or-raspberry-pi/)在这个价位，但 Jetson 有很大的灵活性。我们很高兴看到 Jetson Nano，它拥有 128 个内核和良好的内存，我们正在等待看到更多围绕它构建的真正强大的项目。
