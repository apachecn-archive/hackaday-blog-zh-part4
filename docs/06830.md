# 焊接至铝

> 原文:[https://hackaday.com/2020/06/11/solder-to-aluminum/](https://hackaday.com/2020/06/11/solder-to-aluminum/)

如果你曾经试图焊接到铝上，你就会知道没有某种特殊的技术是不容易的。[SimpleTronic]最近展示了一种方法，用化学方法将铜镀在铝上，让你可以轻松焊接。我们不是化学家，所以我们不确定这是不是最好的方法，但这些化学物质包括盐、硫酸铜(在游泳池商店里可以找到)、你用来蚀刻 PCB 的氯化铁和水。

一旦你有了裸露的铝，你就可以用硫酸铜和一点点氯化铁来制备溶液。在溶液中使用盐显然可以去除铝的氧化。然后用同样的不含盐的溶液在金属上镀一层铜，你可以用它来焊接。你可以在下面看到这个过程的视频。

我们做了一些阅读，发现艺术家们使用硫酸铜和盐来蚀刻铝，有时会加入硫酸氢钠。当然，你也可以使用[特殊焊剂化合物](https://superiorflux.com/papers/TWJ.May.2018.Tech.Solder.Al.pdf)和高温来焊接铝。这也取决于铝的类型，我们想知道这种方法是否适用于罐头以外的东西。

也许这个技术会让我们更多的利用[铝箔](https://hackaday.com/2015/11/30/conjuring-capacitive-touch-sensors-from-paper-and-aluminum-foil/)。历史上，[焊接](https://hackaday.com/2019/04/15/nanoparticles-make-mega-difference-for-unweldable-aluminum/)也一直是个问题。
