# 用等离子切割台远离火花

> 原文：<https://hackaday.com/2021/12/05/keep-the-sparks-away-with-a-plasma-cutting-table/>

对于一次性的项目或原型，我们经常会用手头的工作空间来凑合。例如，使用甲板栏杆作为临时锯木架，对于拥有圆锯的人来说可能是熟悉的，但是[丹尼尔]的情况略有不同。为了使用他的等离子切割机，他一直在随机的砖块上安装金属工件，但就像厌倦了用锯子切割甲板的家庭杂工一样，[他决定提出一个更持久的解决方案，并建造了一个定制的等离子切割台](https://www.youtube.com/watch?v=ndC37ZyQWG0)。

等离子切割有产生大量火花的趋势，因此大多数等离子切割台的商业产品都包括一个水浴来收集切割过程中产生的所有碎屑。[Daniel]在一个金属盆上搭建了一张桌子，用来盛水。桌子本身由铝制成，设计成不需要焊接，尽管大多数有等离子切割机的人可能也有电焊工。框架设计非常坚固，包括弯曲的板条，增加了桌子的强度。桌子也设计成可移动的，所以当桌子移动时，弯曲的板条会保持不动。

虽然乍一看这似乎是一张普通的金属桌子，但这张桌子实际上是用[丹尼尔]正在制作的自制数控机床设计的。数控等离子切割机需要一个坚固、平坦的表面，不能放在车道上的砖块上，所以这张桌子既满足了[Daniel]每次切割东西时不被火花溅到的迫切需求，也满足了他未来的数控机床对坚固、平坦工作空间的需求。我们期待着看到建造完成，但同时看看这个[电动等离子切割机](https://hackaday.com/2019/12/23/motorizing-a-plasma-cutter-on-the-cheap/)，它有一个数控机床的开端，如果只是在一个方向。
