# 自制磁控管真空溅射

> 原文：<https://hackaday.com/2019/07/23/vacuum-sputtering-with-a-homemade-magnetron/>

常言道，“你永远不会太富或太薄”，当谈到涂料时，确实往往越薄越好。真正薄的涂层，有时只有几个原子厚，是物理气相沉积，或 PVD，一种将物质转化为蒸汽并凝结在基底上的技术，有时使用磁控管产生等离子体。

这听起来很复杂，但通过一些合理的工具和对高电压的健康尊重，[一个用于等离子溅射的 DIY 磁控管](https://www.youtube.com/watch?v=Cyu7etM-0Ko)可以让你开始。公平地说，[Justin Atkin]在他的设置上工作了多年，最初受到不得不满足于为他的构建找到零件和一般废料的阻碍。与许多事情一样，接触车床和使用它的技能被证明是可行的，使他能够制造定制零件，如真空室的馈通以及液冷底座，这可以防止热量破坏将等离子体集中到目标金属上的磁铁。使用由旧微波部件制成的高压 DC 电源，[Justin]已经能够在玻璃片上溅射铜膜，使用其他金属取得了有限的成功。他还意外地通过溅射氧化铜而不是纯铜制造了一对分色镜。下面的视频有一些鬼魅般的绿色和紫色光芒的美丽镜头。

从光学到 DIY 半导体，像这样的装置打开了许多可能性。它可能不像我们看过的一些 PVD 场景那么精致，但我们仍然印象深刻。
