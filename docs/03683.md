# 俄罗斯 EBike 无处不在，可能是合法的

> 原文：<https://hackaday.com/2019/07/28/russian-ebike-goes-everywhere-possibly-legal/>

电动自行车可能会席卷全球，但世界本身并没有单一的方式来监管电动自行车在公共道路上的使用。你的自行车在你居住的街道或人行道上骑行是否合法主要取决于…你住在哪里。然而，如果你足够幸运，生活在一个自行车被法律定义为少于四个轮子并且能够由人力驱动的地方，[这辆来自俄罗斯的有趣的自行车可能是我们见过的最好的自制自行车](https://www.youtube.com/watch?v=p0uvX4tNH1A)。(视频嵌入休息下方。)

虽然对于我们这些不懂斯拉夫语的人来说，这种构造的一些细节可能会丢失，但视频本身很好地展示了这种电动汽车构造的特征。它有一个定制的框架，前面有两个轮子，每个轮子都有自己的独立悬架，使它能够轻松穿越极其崎岖的地形，即使是山地车也无法实现。它似乎是由一个相对简单的单后轮后轮毂提供动力，在搭载一名乘客和可能的一些货物的同时，可能可以实现 20 公里/小时的速度。

这款车令人印象深刻的地方不是电气化，而是悬挂部件。任何寻找越野车的人都可以从这款车中获得一些灵感。如果你对动力传动系统更感兴趣，有很多其他车辆使用独特的电力传动系统，如[这艘电动船](https://hackaday.com/2018/12/04/go-up-a-creek-without-a-paddle/)。如果你碰巧懂俄语，看到了这个建筑中其他一些有趣的细节，而这些细节可能是以英语为母语的人错过的，请在评论中给我们留下。
