# 在 ESC 的帮助下测试无刷电机

> 原文:[https://hack aday . com/2019/06/27/testing-small-help-from-the-ESC/](https://hackaday.com/2019/06/27/testing-brushless-motors-with-a-little-help-from-the-esc/)

如今，无刷电机是紧凑型封装中需要高功率应用的首选。有可能购买现成的各种不同配置的电机，并且可用的范围只会越来越好。然而，有时获得真正的最优需要一点定制。对于电机，这可能涉及交换磁铁或手动缠绕线圈。在这些情况下，测试改造后的电机以确定其性能可能是有用的。[【JyeSmith】的 ESC 测试仪能够做到这一点](https://github.com/JyeSmith/dshot-esc-tester)。

基本上，ESC 测试仪是一个简单的硬件。它使用一个微控制器来说 Dshot 协议。该协议通常用于多旋翼飞行控制器和 ESC 之间的通信。在这种情况下，Dshot 遥测显示在一个小的有机发光二极管屏幕上。这使用户能够读取 KV 值，以及其他有用的数据，如电流消耗和 RPM。这有助于量化对电机进行的任何修改的影响，并证明对了解虚假起源的部分是有用的。

这是一个设备，应该证明对那些试图从他们的多旋翼构建中挤出最后一滴性能的人有用。随着无人机比赛越来越受欢迎，我们预计会看到更多类似的项目出现。如果你还在努力学习这项技术背后的理论，你可以随时制作自己的无刷电机。休息后的视频。
