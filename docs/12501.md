# 旧防火墙重生为复古 PC

> 原文:[https://hack aday . com/2022/01/17/old-firewall-reborn-as-retro-PC/](https://hackaday.com/2022/01/17/old-firewall-reborn-as-retro-pc/)

我们喜欢让旧设备获得新生的项目。[Splashdust]有一个 20 年历史的企业防火墙，就像一辆坦克。他打开它，发现里面有一个完整的 x86 嵌入式主板，并着手恢复它，将其变成一台复古的游戏电脑(见他的*奇数&过时* YouTube 频道下面的[视频)。](https://www.youtube.com/watch?v=RjvLe4ghn8U)

![](../Images/c3defbbb9af5560657b08b6a3e627faa.png)

这个企业防火墙和路由器盒子来自瑞典的一家小公司 Clavister，是他们 2000 年代早期 S 系列的一部分。主板似乎是一个用于其他设备的通用主板，由一个[通过 Eden ESP 4000](https://en.wikipedia.org/wiki/VIA_Eden) 以 400 MHz 运行。x86 处理器的 Eden 系列是面向嵌入式应用的低功耗芯片。显卡芯片是威盛 2000 年收购的 S3 Graphics 的 Twister T。在更换电解电容器和制作一些电缆后，[Splashdust]插入 PCI 声卡，并从 CF 卡启动到 Windows 98(我们喜欢他使用的紧凑型 PCB 卡)。

在接下来的两个视频中(这里的和这里的，他建立了一个外壳(Thingiverse 上的[指令](https://www.thingiverse.com/thing:5156657)，并尝试了其他几个操作系统。他能够让 [Tiny Core Linux](http://tinycorelinux.net) 发行版与 [NetSurf 浏览器](https://en.wikipedia.org/wiki/NetSurf)一起运行，但无法让 Windows 2000 或 XP 运行。回到 Windows 98，他调整了驱动程序和设置，最终为自己的努力做出了一台体面的复古游戏电脑。下次你清理垃圾箱的时候，先看看那些披萨盒子里的小玩意——你可能会找到类似的宝贝。
