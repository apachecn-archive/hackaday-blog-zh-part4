# 3D 打印完整的风力发电机

> 原文：<https://hackaday.com/2019/09/19/3d-print-a-complete-wind-generator/>

对我们许多人来说，我们的风景中点缀着风力涡轮机，其中绝大多数都是水平排列的，就好像它们是巨大的飞机螺旋桨。一个更加罕见的景象是垂直风力涡轮机，它仍然是风力实验者的主食。[Troy]和他的兄弟[发布了一个视频](https://www.youtube.com/watch?v=_d_H_goHZHg)，展示了一个小型 3D 打印的立式风力涡轮机，它不同寻常地包括一个从头开始制作的交流发电机以及转子本身。

该机器采用带有三个戽斗的 [Savonius 转子](https://en.wikipedia.org/wiki/Savonius_wind_turbine)设计，与一些替代产品相比，该设计简单且转速较低，扭矩较高。铲子由许多 3D 打印的部分组装而成，直接驱动发电机，发电机使用定子上的大量线圈，定子被包含磁体阵列的转子包围。一个简单的整流器和三端调节器产生 5 伏输出。

可悲的是，没有足够的风给它一个体面的视频测试，但他们用一个非常大的风扇站在演示。我们喜欢交流发电机的设计，但我们有兴趣了解分段转子在户外条件下的表现，或许调节器可以受益于开关模式组件。如果你想试试，他说如果有足够的兴趣，他会把这些文件作为开源文件发布。我们感兴趣[特洛伊]，请做！

这些年来，许多风力涡轮机已经通过了这些页面，作为对比，这里有一个水平的 3D 打印示例。
