# 喷气式客机机舱成为独特的野营车

> 原文:[https://hack aday . com/2020/07/25/jet-airliner-nacelle-been-a-unique-camper/](https://hackaday.com/2020/07/25/jet-airliner-nacelle-becomes-a-unique-camper/)

很可能你们中的一些人会考虑为自己做一个定制的野营车。你们中的一些人甚至可能做了一个泪滴大篷车。虽然你不太可能像[史蒂夫·琼斯]走得那么远，他[从一架退役的前英国皇家空军 VC-10 客机上取下一个外部发动机舱，并将其改造成一辆真正独一无二的野营车](https://www.facebook.com/The-VC10-caravan-pod-108438714152064/)。

从表面上看，对于这样一个项目来说，喷气发动机机舱应该是一个简单的外壳，但这样一个简化的视图可能没有考虑到发动机在飞行中所需的许多通风口、管道和舱口。将它改造成露营者的防水房屋是一项重要的工作，他设法做到了这一点，同时留下了一套发动机检修门，作为一个可以看到风景的房间的大开口。

机舱安装在一个狭窄的前大篷车底盘上，其前进气口形成了一个引人注目的窗口，内部装修非常好，这使得我们许多人都想亲自尝试一下。你可以在休息时间下面看到它的视频，我们希望有一天能有幸在露营地遇到它。

我们已经向你展示了这些年来我们的露营者，但也许这个 3D 打印的最有吸引力。
