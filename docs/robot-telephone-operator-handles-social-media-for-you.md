# 机器人电话接线员为你处理社交媒体

> 原文：<https://hackaday.com/2019/03/22/robot-telephone-operator-handles-social-media-for-you/>

社交媒体在现代生活中已经变得无处不在。如果不在各种科技公司面前提供你的个人数据，你就不可能得到一个聚会的邀请。[David]想要避免看到无数人爬山和吃玉米卷的照片的压力，但也不想通过完全避开社交媒体来排斥自己。答案自然是自动化。

[大卫]恰当地命名他的机器人电话接线员，这正是它所做的。步进电机和伺服系统允许机器人的电容附件与[大卫 iPhone 的触摸屏互动。安装摄像头，结合 OpenCV，机器人能够完成许多重要的任务。

喜欢 Instagram 帖子？完成了。转发无聊的推文？简单。问你的朋友迈克怎么了？是的，电话接线员已经搞定了。鉴于此类平台上人类交互的低质量，完全有可能[大卫]甚至不用尝试就击败了图灵测试。这个机器人甚至可以在周日早上懒洋洋地向下滚动。很恐怖的东西。

当然，如果你太喜欢社交媒体而不相信机器人，你可能会更喜欢把你喜欢的穿在袖子上。休息后的视频。
