# 触觉游戏为视障人士带来乐趣

> 原文：<https://hackaday.com/2019/11/12/haptic-games-bring-fun-to-the-visually-impaired/>

不管你的父母怎么说，游戏对我们有好处。它们教会我们管理资源，并给予我们多巴胺奖励，就像进食和交配一样。即使你不擅长游戏，它们仍然是生活中一种有趣的消遣。

有很多游戏可以让视障人士享受，只是它们依赖视觉效果。例如，你可以只用五个骰子、一个杯子、铅笔和纸，以及规则和计分的知识来玩 Yahtzee。最大的障碍是区分骰子和保持分数。

我们尊敬的 [2019 Hackaday 奖前 20 名决赛选手](https://hackaday.com/2019/09/10/meet-the-20-finalists-in-the-2019-hackaday-prize/)之一是【JanThar】的[触觉游戏](https://hackaday.io/project/165949-hapticgames)。[JanThar]越来越多的游戏使用 3D 打印、振动马达和 RFID 来用感官反馈取代视觉线索。Yahtzee-wise，有一套印刷骰子和记分卡。记分卡使用球形磁铁和算盘布局。[JanThar]也在研究一种记忆游戏来教授盲文，尽管它可以被改编成供视力受损者使用的纯盲文。每个游戏棋子都包含一个 RFID 芯片，所以玩家可以把它举到阅读器前检查他们拥有什么。

我们最喜欢的可能是基于[JanThar]2017 年 Hackaday 奖参赛作品的 PONG 游戏，即 [HaptiVision](https://hackaday.io/project/27112-haptivision) 背心。通过一个 16×8 振动电机场的魔力，球员可以跟踪球在他们躯干上的运动，并用滑块控制球拍。休息之后有一个简短的游戏演示。
