# 可调，低冲击 Keeb 是最舒适的

> 原文:[https://hack aday . com/2021/04/02/adjustable-low-impact-keeb-is-as-as-comes/](https://hackaday.com/2021/04/02/adjustable-low-impact-keeb-is-about-as-comfortable-as-it-gets/)

在不完全放弃的情况下，缓解打字的重复性压力的最酷的方法是什么？除了手指什么都不要动，尽量少动，不要有任何拉伸或伸手。自从我们第一次看到这种古怪奇妙的 DataHand 键盘，我们就已经是它的粉丝了，但是[Ben Gruver]实际上已经用这些停产的键盘很多年了，就像日常驾驶一样。而当我们爱着稀缺的东西时，我们该怎么办？像本一样，用纸板做我们自己的改良版。

[![](../Images/ce875399d1267d25f60894a66a4a60e0.png)](https://hackaday.com/wp-content/uploads/2021/03/lalboard-inner.png)[【Ben】使用 lalboard 已经有两年了，并且有一个改进第二版](https://hackaday.io/project/178232-lalboard)的清单，我们很自豪能在 IO 上主持这个项目。许多改进都是为了让这项庞大的工程更容易印刷和组装。第一个版本使用铜带走线，但[Ben]正在开发一种可制造的 PCB，它将使用一对 Teensy 2.0s 之外的东西，也许还有 QMK 固件。

不会改变的是奇妙的光学按键开关设计，它使用红外 LED 和光电晶体管来捕捉按键，并使用微小的方形磁铁将按键返回到原始位置，并发出我们非常确定的令人满意的咔嗒声。

这款键盘最酷的地方在于它的可调性。每个按键组可以在 6 个方向上调节，包括每个手指的不同高度，如果这是最好的工作方式。一旦这一切都想通了，那么是时候打印一些完美的永久对峙。想为自己制作一个这样的科幻噼啪机吗？[Ben]在 GitHub 上有[BOM，一些打印说明和提示，以及制作铜带 PCB](https://github.com/JesusFreke/lalboard/tree/v2)的指南。休息之后，看看[本]以每分钟 120 秒的速度改写卡夫卡的变形记。

有兴趣了解更多关于原始 DataHand 键盘的信息吗？这是我们的拍摄。
