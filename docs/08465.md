# Cornhole 董事会播放胜利的歌曲

> 原文：<https://hackaday.com/2020/11/26/cornhole-boards-play-victory-songs/>

你如何立刻让游戏变得更好？点亮它，在晚上播放。我们通常会说“喝酒”,但我们非常确定喝酒已经是 cornhole 的先决条件——这是一种在有角度的木板上向洞里扔豆子袋的游戏。

[硬件未知]喜欢 cornhole，并被赠予一套便携式，折叠板，照亮了环周围的夜间行动。[这些被证明是反应板的完美基础，每当得分时，反应板就会亮起并播放声音](https://www.reddit.com/r/arduino/comments/jzl65v/cornhole_boards_that_react_to_impacts_and_bags/)。两块板都有一个振动传感器来检测袋子撞到顶部，还有一对红外光束传感器穿过孔。Arduino Nano 从传感器读取数据，并控制放大器和测向播放器发出声音。

玩家将袋子放在板子上可以得到一分和一首歌，将袋子放进洞里可以得到三分和一首不同的歌。我们喜欢复活节彩蛋——任何设法同时触发振动传感器和光束中断探测器的人都将听到一群鹅的叫声。休息之后，请查看构建之旅。
