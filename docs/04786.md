# 观看 3D 打印机的全新设计

> 原文：<https://hackaday.com/2019/11/10/watch-a-3d-printer-get-designed-from-the-ground-up/>

很多时候，当你看到一个构建视频时，你只能看到最终的产品。即使有构建本身的镜头，通常也只是作为主要组件完成的高光部分。但幸运的是，[罗伊·伯恩森]一直在研究的“V-Baby”CoreXY 3D 打印机并非如此。

通过观看他的视频播放列表，你可以看到他在努力实现他的各种设计目标。例如，他希望最终的设计既可加工*又可印刷*，这是可能的，但这无疑增加了复杂性和时间。他还在某一点上从三角形基底过渡到矩形基底。这些决定，以及背后的原因，都被记录和讨论。

在这个系列的最后，我们可以看到最终的测试和折磨过程，因为他上升到最终的设计版本。对于任何试图从零开始设计他们的第一台 3D 打印机的人来说，这肯定会揭开这个过程的神秘面纱。
