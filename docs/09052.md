# MouSTer 将 USB 引入复古电脑

> 原文:[https://hack aday . com/2021/01/22/mouster-brings-USB-to-retro-computers/](https://hackaday.com/2021/01/22/mouster-brings-usb-to-retro-computers/)

喜欢偶尔在周日开着老款阿米加出去兜风的人通常会这样做，因为他们对更简单的时代有着渴望的记忆。以前，你可以在内存以千字节而不是千兆字节计量的机器上编辑文档或查看电子表格。但是即使是最狂热的复古电脑爱好者通常也会考虑到一点现代的便利。

进入捕鼠器。这个微小的设备[将一个普通的 USB HID 鼠标转换成旧电脑可以理解的东西](https://retrohax.net/the-mouster-project-is-here/)。它甚至支持将索尼的 PlayStation 4 控制器用作通用游戏手柄。虽然固件仍在调整中，但该团队已经确认它可以在几台经典机器上工作，并相信它应该可以在更多机器上工作。考虑到这些旧外设在二手市场上的价格，在你的老式电脑上使用 USB 鼠标或控制器不仅更方便，而且可能会便宜得多。

[![](../Images/0158fb85bb74b53fb8c04dfa4b48717b.png)](https://hackaday.com/wp-content/uploads/2021/01/mouster_detail.jpg) 证实 retro computing super fan【dry gol】是 mouSTer 工作团队的成员，在最近他的 *retrohax* 博客的一篇文章中，他谈了一点自从[他在夏天](https://hackaday.com/2020/07/14/modern-mice-on-old-computers/)最后一次更新以来发生的事情。他还谈到了他们在将其投入生产时所面临的挑战。即使你对老式电脑不感兴趣，这里也有一些经验教训，告诉你如何从少数几个原型变成可以真正卖给公众的东西。

我们特别喜欢 mouSTer 围栏的细节，或者说它的缺失。最初[Drygol]表示，他们将让外壳注塑成型，但尽管他们谈过的一些公司最初有兴趣，但最终没有人上钩，因为这需要用相对不常见的低压注射来完成。虽然 3D 打印仍然是一种选择，但该团队最终使用透明热缩管在电子设备上创建了一个简单的保形保护壳。就个人而言，我们认为这看起来很棒，但听起来这只是一个临时的解决方案，直到可以实现更强大的解决方案。

正如你可能想象的那样，我们已经看到了旨在将现代输入设备引入 Atari ST 等老式计算机的 DIY 项目，但 mouSTer 的小巧比例以及它是一款交钥匙产品的事实肯定会吸引那些希望在使用经典设备时最大限度地减少头痛的人。
