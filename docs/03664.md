# 像 1969 年一样控制您的网络浏览器

> 原文：<https://hackaday.com/2019/07/23/control-your-web-browser-like-its-1969/>

想象一下，你的任务是开发一个与全球互联设备网络连接的设备。你会故意设计一个弹簧表盘，除了打开和关闭一组触点 1 到 10 次之外什么都不能做吗？我们将不得不生活在一个什么样的疯狂世界里，在这个世界里，这样的东西是技术的顶峰？

显然，这样的世界曾经存在过，现在我们已经将日历向前滚动了半个世纪左右，我们的网络和界面都变得更加复杂，如果可以说是更好的话。但是[Jan Derogee]认为倒退一步是合理的，所以他设计了这个旋转式手机网络浏览器。想法很简单:拿起听筒，拨你想连接的服务器的 IP 地址。DNS？呸，谁需要？

当然还有一个小问题，大多数网站不能通过 IP 地址直接访问了，但是不用担心——[ Jan]有一个令人难以置信的模糊解决方案。它依赖于这样一个事实，即许多数字在中文发音时听起来像普通短语，所以最终会有许多网站拥有基于数字的 URL。他提供了一个例子，使用数字 517，听起来有点像“我想吃饭”，来访问麦当劳的中文网站。听起来像“吃”和“老婆”的数字 7 是如何解决的，留给读者做练习。

这里我们认为[Jan]的[旋转数字键盘](https://hackaday.com/2019/03/13/3d-printed-rotary-dial-keypad-is-wonderfully-useless/)的价值值得怀疑。尽管如此，我们还是很欣赏这个版本，而且[让旧手机重新投入使用](https://hackaday.com/2015/11/18/old-school-rotary-phone-gets-gsm-upgrade/)无论以何种方式，都是值得赞赏的。
