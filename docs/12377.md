# 找回那个 YouTube 不喜欢按钮

> 原文：<https://hackaday.com/2022/01/02/getting-back-that-youtube-dislike-button/>

啊，谷歌。很少有公司掌握了创造惊人技术和产品的艺术，然后如此有效地放弃和错误处理它们。YouTube 就是一个很好的例子。引用“不喜欢的攻击”，这是对特定视频的协调投票，YouTube 选择隐藏所有视频的不喜欢计数器。可以指出的是，不喜欢仍然以他们一贯的方式影响推荐算法，并且创建者仍然在他们自己的仪表板上看到他们的不喜欢计数。

YouTube 不喜欢他们的倒带视频的恶名，这种想法可能有一定道理，2018 年的不喜欢人数接近 2000 万。企业合作伙伴不喜欢视频中可见的不喜欢号码的理论甚至可能是正确的。不管怎样，YouTube 做出了改变，人们讨厌它。他们的平台，所以你对此无能为力，对吗？

引用我最喜欢的虚构数学家的话，“生活，呃，呃，会找到出路的”。是的，一个黑客，我们中的一员，已经想出了一个变通办法。[Dmitrii Selivanov]已经整合了[“返回 YouTube 不喜欢”浏览器扩展](https://returnyoutubedislike.com/)，它做了几件事。首先，它正在提取关于视频的存档数据，利用官方宣布和删除不喜欢的 API 之间的时间间隔。

但是对于新视频呢？这就是事情更难的地方。如果你安装了这个扩展，你的视频喜好和厌恶会被跟踪，综合的用户数据会被用来推断任何给定视频的厌恶次数。[Dimitrii]还在研究一种方法，允许各个频道与该项目共享他们的统计数据，为他们的视频提供更多的官方数字。

该扩展是开源的，Chrome 网上商店显示有超过一百万的用户。Linus Tech Tips 和许多其他渠道都涉及到了这一点，所以请查看他们的视频了解更多信息。
