# 建立自己的真空室脱气和更多

> 原文：<https://hackaday.com/2019/04/12/build-your-own-vacuum-chamber-for-degassing-and-more/>

在商店里有一个真空室是很有用的。它可以用于各种目的，从科学实验到油漆和环氧树脂的脱气。它们不是你在每个车间都能找到的东西，但是不要害怕——[你总是可以用你得到的废料来建造一个！](https://www.youtube.com/watch?v=bv3tYR4ROBs&list=PLOzaysTbaKi6jcKZyiLBV_i42_6SgkRe8&index=10) (YouTube 视频，下面嵌入。)

[VegOilGuy]从一个简单的胶合板箱开始建造，它被拧在一起，然后用身体填充物和油漆开始。这有助于使盒子密封，以及提高美观。然后在盖子上切一个槽，然后填充过量的硅酮密封剂。将一个用铝箔覆盖的平板放在上面，让硅树脂固化几天。这用于为稍后放置的盖子创建平坦的密封表面。

一旦密封完成，这是一个简单的管道工作来完成室。[VegOilGuy]很好地演示了铜焊接以及安装必要的水龙头和止回阀的正确方法。结合电动泵，真空室成功通过测试，在这个过程中完全破坏了一些棉花糖。

只需在网上花上几美元购买一些特别的东西，任何一个方便的制造者都可以在一个周末内组装好它。不过，你总是可以走另一条路——比如用旧冰箱压缩机来完成工作。
