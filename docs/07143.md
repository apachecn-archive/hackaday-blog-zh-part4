# 使用物理静音按钮快速静音和取消静音

> 原文:[https://hacka day . com/2020/07/15/快速静音并使用物理静音按钮取消自己的静音/](https://hackaday.com/2020/07/15/quickly-mute-and-unmute-yourself-using-the-physical-mute-button/)

随着今年许多会议转向完全虚拟化，在可预见的未来，视频会议将继续成为我们生活中的主流。[Elliot]希望稍微增加他的视频会议体验，让他的体验更符合人体工程学。我们都有过这样的问题:寻找隐藏在其他应用程序后面的缩放窗口，拼命寻找静音按钮。此外，当我们被叫到的时候，我们会拼命给人留下我们一直在专心听讲的印象，即使我们并没有。

为了解决所有这些问题，[他建立了一个物理静音按钮](https://www.instructables.com/id/Zoom-Meetings-Physical-Mute-Button/)来轻松地在缩放通话期间打开和关闭静音选项。该设备利用了他的 Digispark 板的原生 USB 功能，以及 Zoom 中的一些内置键盘快捷键。有了原生 USB，Digispark 板可以像键盘一样工作，使得使用微控制器模拟键盘按压变得非常简单。加入一个街机风格的按钮，做一点手工制作，你就有了自己的物理静音按钮。

简单的设计和优雅的机械装配给我们留下了深刻的印象。[Elliot]甚至做了一个改进的版本，带有第二个按钮，允许他控制他的视频。酷按钮[埃利奥特]！
