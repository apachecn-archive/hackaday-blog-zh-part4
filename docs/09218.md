# 简单的滑动口哨得到一个 MIDI 改造

> 原文:[https://hacka day . com/2021/02/10/simple-slide-whistle-gets-a-midi-make up/](https://hackaday.com/2021/02/10/simple-slide-whistle-gets-a-midi-makeover/)

在今天的“迷笛会吗？”我们有[常见的滑行哨](https://mitxela.com/projects/slide)。剧透:是的，会的，结果就在迷人的边缘——嗯，有点奇怪。

正如 maker [mitxela]指出的那样，尽管滑笛很简单，但它是一种很难演奏的乐器。或者，至少是一个很难重复的音符。它有点像一个微小的塑料长号，因为两者都没有按键或停止键，将振动的空气柱限制在特定的长度。实际上，下面视频的开头显示了一个使用磁铁解决滑动口哨问题的巧妙方法，但这主要是一个附带项目。

[mitxela]滑动哨子的 MIDI 化需要的不仅仅是几块磁铁。为了将载玻片移动到规定的位置，一对高精度伺服系统通过激光切割胶合板剪刀连杆连接。音乐家的肺动力被一个小的电动鼓风机取代，安装在远离哨子的地方，通过一根长软管供气。风扇的速度可以改变，因此气流的速度也可以改变；这可以防止低音在音域上移，防止过度吹，如果这是正确的术语的话。另一个伺服系统控制一个阻尼器，它可以切断哨子口的气流，从而控制音符，而不必完全关闭风扇。主要文章详细介绍了控制电子设备和校准过程。

该视频有几首 ~~YouTube 版权罢工~~演示歌曲，我们不得不说，我们对该机制的响应速度印象深刻。有些人会反对多余的伺服噪音，但我们发现它很好——几乎像吉他弦吱吱声。我们也喜欢[【米切拉】的伺服弹拨音乐盒](https://hackaday.com/2018/03/23/servos-do-the-plucking-in-this-midi-music-box/)加入的曲调。
