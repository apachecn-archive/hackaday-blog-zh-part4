# 今天的辛克莱基础课

> 原文:[https://hackaday.com/2021/05/10/sinclair-basic-for-today/](https://hackaday.com/2021/05/10/sinclair-basic-for-today/)

如果你到了一定的年龄，你对计算机编程的第一次接触可能是基础的。几年来，没有比辛克莱·ZX 系列计算机更便宜的 BASIC 编程方法了。如果你渴望那些日子，你可能会发现 1980 年左右的 BASIC 版本有点局限。或者您可以使用来自[Paul Dunn]的[spec basic](https://github.com/ZXDunny/SpecBAS)。

SpecBasic 显然与 Spectrum 相当兼容，但允许您使用更好的硬件。例如，SpecBas 不是 256×192 的 8 色屏幕，而是更大的屏幕，多达 256 种颜色。但是，这确实会导致某些不兼容性，您可以在项目的自述文件中了解到。

自述文件还包含配置系统以使用更大字体的说明，这可能有助于在大型显示器上提高可读性。还有一个文件，里面全是你可能想看的演示。

虽然今天学习 1980 年代的 BASIC 并不是什么职业发展，但我们确实认为在一个简单整洁、交互性很强的环境中学习简单的编程概念并不是一件坏事。毕竟，它适用于整整一代开发人员。

如果你更喜欢你在网络上的基本编码，[那是可能的](https://hackaday.com/2018/09/17/its-the-web-basically/)。或者，也许你只是想要更现代的。
