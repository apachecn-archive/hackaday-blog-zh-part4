# 3D 打印变得越来越小

> 原文：<https://hackaday.com/2022/10/26/3d-printing-gets-small-in-a-big-way/>

如果你的车间里有一台 3D 打印机，你可能会更担心如何用它打印出更大的物体。然而，阿姆斯特丹大学有一种新技术，可以实现亚微米分辨率的快速大规模打印。这项技术是光刻技术和立体光刻技术的混合。

印刷精细细节的问题之一是印刷时间变得非常长。然而，这项新技术声称具有“可接受的生产时间”显然，生物打印应用对该技术的第一个被许可方非常感兴趣。例如，有传言说几个小时就能打印出一个肾脏支架，或者不到一天就能打印出一个完整的心脏支架。

另一个示例应用是生产具有 200 微米通道和 20 微米限制的色谱仪器。这需要一台能打印非常精细细节的打印机。在半导体和机械超材料中也有应用。当然，我们总是注意到光刻工艺，因为我们用它来制造 PC 板，甚至集成电路。一台可以进行光刻的台式打印机可能会为生产电子电路开辟新的思路。

如果你今天想玩光刻，【Ben Krasnow】[有一些建议](https://hackaday.com/2020/03/26/ben-krasnow-rolls-old-school-camera-out-for-photolithography/)。当然，有几种方法可以生产 PC 板，甚至用普通的 3D 打印机。
