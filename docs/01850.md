# 早餐机器人做鸡蛋做得很完美

> 原文：<https://hackaday.com/2019/01/16/breakfast-bot-does-eggs-to-perfection/>

早餐是充满矛盾的一餐。以一顿充满能量和营养的丰盛大餐开始新的一天是很重要的，但是当你仍然睡眼惺忪、半睡半醒的时候做饭也是很困难的。正如生活中的许多问题一样，自动化是答案。詹姆斯·布鲁顿拥有煮鸡蛋的装备，让你的一天有一个好的开始。

基本设备使用恒温控制的电热板加热一壶水。[James]然后使用以前项目中的编码器控制的线性致动器来升高和降低网筛，将鸡蛋放入锅中。Arduino 用于测量水温，只有当水温超过 90 摄氏度时，才会开始烹饪过程。此时，一个 6 分钟的计时器开始计时，鸡蛋被从水中取出，并由伺服控制的扭转机构倾倒出来。

未来的工作将包括伺服控制热板的旋钮，并建立一个斜槽来捕捉鸡蛋，以进一步减少人工干预的需要。虽然在房子里有一个自动加热板有一些危险，但这可以与 RTC 同步，以确保你的煮鸡蛋每天按时准备好。

早餐机是这些地区和[的一大传统，我们在这个时代已经见过一些了](https://hackaday.com/2009/09/28/the-incredible-breakfast-machine/)。休息后的视频。
