# 惠斯通电桥配你的锅

> 原文:[https://hack aday . com/2022/12/07/a-wheatstone-bridge-matches-your-pots/](https://hackaday.com/2022/12/07/a-wheatstone-bridge-matches-your-pots/)

有时候，最简单的方法可能是最有用或最巧妙的，例如[Keri Szafir]的[方法，确保音频设备中使用的电位计匹配](https://hackaday.io/project/188546-stereo-pot-testing-hack)。如果你考虑一下典型的立体声放大器，你会看到两个放大器在一个盒子里，只有一个音量控制。两个频道，一个旋钮？音量旋钮是联动立体声电位计。

并非所有电位计都是相同的，特别是在较便宜的器件中，它们在两个电位计之间以及在整个行程中可能没有一致匹配的电阻。这打乱了立体声平衡，因此自然值得选择一个匹配良好的部分。[Keri]选择它们不是用他的金耳朵，而是通过将两个罐子连接在一起作为惠斯通电桥。由于不匹配，两个电刷之间的电表可以检测到任何电流。

惠斯通电桥是一种方便的电路，在交流和 DC 测量中有很多用途。我们可能最常看到的是应变仪中的 T2。
