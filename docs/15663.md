# 用这个电子墨水泡茶计时器及时注满你的茶壶

> 原文:[https://hacka day . com/2022/12/15/用这个电子墨水茶定时器给你的茶壶充值/](https://hackaday.com/2022/12/15/top-up-your-teapot-in-time-with-this-e-ink-tea-timer/)

无论你是一个茶迷还是一个普通的饮者，注意你的冲泡时间是很重要的:太短了，你只是在喝热水，太长了，你的冲泡变得又苦又涩。[Bob]想帮助他的父母避免后一种情况，他给他们做了一个方便的小计时器,显示他们最后一次装满茶壶的时间。

操作计时器再简单不过了:只需按下茶壶盖，它就会在电子墨水显示屏上存储当前时间。里面是一个 Pimoroni Badger 2040，带有实时时钟子板，由一组 AAA 电池供电。Badger 是一个由 RP2040 供电的板，集成了电子墨水显示器，非常适合这种使用情况:当按钮被按下时，显示器只需更新一次，之后就不再使用任何电源。

自然地，茶定时器被装在一个茶壶形状的外壳里。它内部有一个聪明的机制，当你按下盖子时，它会按下獾的一个按钮，并提供你在下面嵌入的视频中听到的令人满意的点击。在[Bob]的 Creality Ender 3 上打印花了超过 13 个小时，但最终结果看起来确实如此。

从功能上来说，这款煮茶计时器非常简单:大多数其他设计都专注于最初的冲泡过程，并包括当你的茶准备好了时提醒你的功能。
