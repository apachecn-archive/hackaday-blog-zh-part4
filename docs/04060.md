# 力敏电阻消除了床调平的痛苦

> 原文:[https://hack aday . com/2019/09/01/force-sensitive-resistor-take-the-pain-out-of-bed-leveling/](https://hackaday.com/2019/09/01/force-sensitive-resistor-takes-the-pain-out-of-bed-leveling/)

你如何知道你的 3D 打印机床是否水平？哦，别担心，你会知道的。没有一个水平的床，细丝不会正确地粘在建造表面，你会遇到各种各样的其他问题。知道把床弄得恰到好处有多难， [[Antzy]制造了一个工具来帮忙。](https://www.instructables.com/id/FS-Touch-Bed-Levelling-Tool/)

这款被他称为 FS-Touch 的设备是基于一个安装了力敏电阻的 Arduino Pro 微型电脑。这允许基于电阻器放置在床和喷嘴之间时读取的力来测量床和喷嘴之间的距离。

使用该工具很简单。首先，使用典型的纸方法将床粗略对齐[。](https://all3dp.com/2/3d-printer-bed-leveling-step-by-step-tutorial/)然后，从床的一角读取读数，并保存测量结果以供参考。其他角可以设置为相同的水平，在 led 的帮助下，指导用户向哪个方向转动调节旋钮。

用这种方法测量力比有些困难的纸法具有更高的可重复性。它有望减轻那些正在努力让自己的床保持合适形状的用户的负担。当然，[自动调平使事情变得更加简单](https://hackaday.com/2015/06/29/semi-automatic-bed-leveling-your-3d-printer/)。休息后的视频。
