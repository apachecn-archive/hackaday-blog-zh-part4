# 快速黑客:使用钉书钉时，重新盖主板

> 原文:[https://hack aday . com/2022/03/22/quick-hacks-using-staples-when-recapping-motherboards/](https://hackaday.com/2022/03/22/quick-hacks-using-staples-when-recapping-motherboards/)

[马尔西奥·特谢拉]需要翻新一个旧的苹果麦金塔主板，在热风返工过程中，他遇到了一个简单的方法[用普通的纸钉作为临时隔热板](https://www.youtube.com/watch?v=PuA8G8iDBaI)(视频，嵌入下方)。热气返工的问题是尽量减少附带损害；你正在使用温度足以熔化焊料的空气，要想知道如何最好地保护更精密的部件不受损坏，可能需要相当多的经验。较大的物品由于其热质量需要更长的时间来加热，但是较小的部件在试图移动附近的目标时会很快被过多的热量损坏。

塑料连接器的锋利边缘特别容易损坏，良好的保护至关重要。由聚酰亚胺(Kapton)、PET 以及金属选项(铝带是有用的)制成的胶带通常被用于临时遮蔽有过热危险的区域。但是它们会引起其他问题。Kapton 胶带虽然很耐热，但在返工铅笔的冲击下，往往会变形和弯曲一点。更不用说一些品牌的胶带在受热时会留下令人讨厌的粘性转移残留物，需要额外的清理。

也许一盒或两盒订书钉可能值得加入一个人的锦囊妙计，毕竟更多的选择总是好的。如果你对用黑掉*一个热风工作站不感兴趣，而对黑掉一个热风返工站更感兴趣的话，[给你](https://hackaday.com/2017/05/25/hack-your-hot-air-station/)，当我们正在返工劣质电脑时，下面是一个[黑客作家试着修理他儿子的 Xbox](https://hackaday.com/2021/12/08/fail-of-the-week-hackaday-writer-attempts-xbox-repair/) 时发生的事情。*
