# 镜像音乐机器反映昼夜节律

> 原文:[https://hack aday . com/2020/09/15/mirrored-music-machine-reflects-昼夜节律/](https://hackaday.com/2020/09/15/mirrored-music-machine-reflects-circadian-rhythms/)

互动艺术家[Daric Gill]来信分享了他在过去一年中创作的令人难以置信的电子雕塑。它被称为昼夜节律机器，它是一个由传感器启动的正念音乐和灯光事件，根据一天中的时间和五度周期播放各种原创作品。这台机器执行一些稳定的动作，如在每个小时的顶部播放编钟，以及在太阳正午播放特殊序列。

[![](../Images/2a9cef6feed2e6faddd8cdbda2c580ca.png)](https://hackaday.com/wp-content/uploads/2020/09/circadian-machine-guts.jpg) 这个赛博朋克风格的截顶六角形双金字塔首先对自己进行地理定位，然后学习当地日出和日落的时间。一个由羽毛 M4 快车和音乐制造商 FeatherWing 制成的音乐模块获取天文数据，并控制灯光，扬声器和几个运动传感器，当被触发时，会在飞行中改变灯光和声音。一个独立的羽毛 Huzzah 和 DS3231 RTC 处理 WiFi 协商并跟踪时间。

除了每小时的灯光和声音，昼夜节律机器还做了一些非常有趣的事情:它根据日出和日落执行另一组动作，基本上是在这两个事件之间填充一整天的动作，这似乎是对人类每天所做的事情的致敬。休息后查看[构建笔记](https://dgillart.wordpress.com/2020/09/08/making-circ/)和走查视频，然后留下来观看完整的构建视频。

互联网上充斥着乞求被转化为艺术的信息。例如，[世界各地有足够多不安全的闭路电视摄像头，它们占据着最佳的有利位置，你可以每天每小时观看不同的日出*和日落*](https://hackaday.com/2020/08/25/sunrise-sunset-repeat/)。
