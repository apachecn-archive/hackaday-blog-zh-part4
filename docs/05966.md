# 热丝剪切机隆重升温生产力

> 原文：<https://hackaday.com/2020/03/16/hot-wire-ribbon-cutter-ceremoniously-heats-up-productivity/>

任何剪过缎带、罗缎或其他东西的人，可能会对磨损的边缘感到沮丧。有一些方法可以避免这种情况，比如斜着剪开丝带，或者双斜着剪成分叉点，或者直着剪开，用打火机烧灼丝线。但是，如果你有十三打面包师的几十个价值的东西来装饰，没有人有时间。

[IgorM92]为他的妻子制作了[这款热线剪彩刀](https://www.instructables.com/id/Cutter-for-Textile-Ribbons-DIY/)，她的妻子经营着一家看起来很不错的烘焙公司。它通过使用旧烙铁的加热元件将切割和热封合为一个步骤。如果你没有，你可以很容易地使用旧吹风机、烤面包机或线绕电阻器上的镍铬合金线。

因为这个想法本质上是短路电源来加热电线，所以应该安全地完成。[IgorM92]使用手机充电器将主电源调节至 5 V。电路中没有太多其他东西，只有一个摇臂开关、一个电源指示 LED 及其电阻，但这个简单的项目无疑将节省大量时间和人力。燃烧过去的休息，看着它增加产量。

镍铬合金丝也适合切割泡沫。这里有一个可以在几分钟内完成的基本版本。
