# 一个甚至有屏幕的可破解键盘

> 原文：<https://hackaday.com/2021/10/30/a-hackable-keyboard-that-even-has-screens/>

如今，世界上有大量可用的键盘，满足所有最简单和最高级的需求。然而，如果你想要的东西*恰到好处，*有时候打造自己的东西是值得的。[[扎克]就是这么做的。](https://www.youtube.com/watch?v=Fg0V5M0llaE)

[Zach]构建的一个关键特征是它背离了 Cherry MX switch 的外形。该设计使用了薄型开关，这有助于保持键盘足够低，以避免导致手腕问题。键盘还使用 IO 扩展器来连接所有的按键开关，有助于减少鬼键的发生。该板也可以分成两半，允许它在需要时重新用作更小的宏垫。

它被包裹在一个很酷的 3D 打印外壳中，右手边甚至有三个有机发光二极管显示器。它们焊接在 PCB 上的特殊切口上，允许显示器弯曲并触发触摸开关，充当巨大的可按压按钮。

[Zach]很好地解释了他做出的所有巧妙的工程决策，以将最大的功能塞进设计中。我们也看到了其他一些很棒的 DIY 人体工程学设计。休息后的视频。
