# 莱姆 SDR(冥王星也是)发送电视

> 原文：<https://hackaday.com/2019/01/19/lime-sdr-and-pluto-too-sends-tv/>

如果您使用无处不在的 RTL 软件定义无线电加密狗体验过软件定义无线电(SDR ),那么您就错过了它的一半。虽然这些特别提款权并不昂贵，但它们只能获得。下一步是传输。[腐蚀性]展示了他如何使用 DATV Express 以及 Lime SDR 或 Pluto(ADI 公司的评估设备)来传输视频。他展示了如何在业余无线电的背景下设置这一切。之前的视频展示了如何使用 SDR 和一些 Windows 软件接收信号。接收器也可以与 RTL SDR 或 HackRF 板一起工作。你可以在下面看到这两段视频。

DATV 快递软件有很多选项，因为 SDR 如果频率捷变，你应该能够在任何频率(在 SDR 范围内)上使用它。最后，他提到，要真正将这些应用于广播，您需要一个滤波器和放大器，因为输出有点原始且功率较低。

如果你年纪够大，还记得电视发射机是一个装满电路的大盒子的时候，看到视频从一个小小的电路板中涌出是相当令人惊讶的。更重要的是，在发送和接收时，您可以用软件进行大量处理，这在使用传统硬件时是非常先进的。

奇怪的是， [RTL SDR](https://hackaday.com/2017/09/05/19-rtl-sdr-dongles-reviewed/) 本来是用来接收电视的。你实际上可以用[除了树莓派](https://hackaday.com/2016/04/03/dvb-s-from-a-raspberry-pi-with-no-extra-hardware/)什么都不用做传输，而且【腐蚀剂】提到他很快会做一个关于这个的视频。
