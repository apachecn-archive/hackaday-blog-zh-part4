# TréPhonos 在休斯顿唤起历史

> 原文：<https://hackaday.com/2019/02/08/trephonos-calls-up-history-in-houston/>

休斯顿历史悠久的第三区，又名“The Tre”，充满了历史，其中一些历史被数字化保存下来，可以通过一个艺术装置进入，这个艺术装置的形式是[改装的付费电话](https://projectrowhouses.org/calendar/trphonos-love-we-sound-so-good-on-the-phone)。显而易见，我们喜欢付费电话，看到它们充满活力，我们心里暖洋洋的。把它们和当地历史放在一起，就意味着更多的箱子。24 个人合作重建了三部手机，我们可以在休息后的视频中看到，包括三位视觉艺术家、三位大使和 18 位居民，他们努力使手机不仅与选区相关，而且与社区相关。一部手机播放曾经或仍然住在病房里的音乐家的声音片段，另一部手机播放单词故事，第三部手机播放 Tre 中重要位置的现场录音。

每部手机都由太阳能电池和 USB 电池组供电，连接到一个带有音频适配器板和 20 瓦放大器的 Teensy。按钮 1-9 播放每部手机独有的录音信息，星号键将录制一条信息，零键将播放用户录制的信息。对于年轻人来说，智能手机的应用程序很容易理解，但付费电话确保这些时间胶囊可以被任何年龄的人欣赏，不管他们有多精通技术，这是明智的，也是有吸引力的。与普通付费电话不同，硬币返回杆和硬币槽也有相关的声音剪辑，因此艺术家可以获得额外的积分。

我们说过[我们喜欢付费电话](https://hackaday.com/2016/12/01/payphone-boombox-straight-out-of-the-1990s/)吗？是的，[是的，我们做了](https://hackaday.com/2015/01/05/breathe-new-life-into-payphones-with-asterisk/)。在 Hackday 上[的第一个帖子是为了一个红盒子，这就开始了。](https://hackaday.com/2004/09/05/radioshack-phone-dialer-red-box/)
