# 未来的电池，现在由你自己建造

> 原文:[https://hack aday . com/2021/03/13/battery-of-the-future-now-build-yourself/](https://hackaday.com/2021/03/13/battery-of-the-future-now-buildable-yourself/)

理论上，电池和电容相当简单。一个用化学方法储存能量，另一个用电场储存能量。然而实际上，建造一个具有实用能量密度的能量存储装置是一项精细复杂的工作。但是如果你能获得一些化合物，用离子液体生产有用的电池和电解电容器实际上并不困难。

离子液体是具有一些其他重要性质的导电液体。几乎所有显示的产品都可以用相对普通的化合物制造，并且大多数产品都具有有利的物理性质，使它们稳定且使用相对安全。利用化学实验室中的一些设备，可以没有太多麻烦地生产各种各样的这些液体(尽管概述的一种方法使用惰性气体室)，然后通过让离子液体被吸收到设备中，可以从那里制造电池和电容器。

下面的视频展示了其中几个设备的生产过程，然后通过运行一盏小型 LED 灯展示了它们的效果。虽然它们可能不会很快被用来制造 DIY 电动汽车，但非典型储能设备的生产和改进将是满足现在和未来社会大部分能源需求的关键，特别是像这样的铝电池。
