# 人工智能为杀蜂机器提供动力

> 原文:[https://hack aday . com/2019/07/16/artificial-intelligence-powers-a-wasp-killing-machine/](https://hackaday.com/2019/07/16/artificial-intelligence-powers-a-wasp-killing-machine/)

在本文发表时，Hackaday 了解到在美国或国外没有亲 wasp 的游说团体活跃。为什么？嗯，黄蜂是一种被认为不能对社会做出任何可行的经济贡献的昆虫，因此很少有甚至没有支持它的人。事实上，许多人积极寻求击败黄蜂，[，【Tegwyn☠Twmffat】就是其中之一。](https://hackaday.io/project/161581-ai-equiped-wasp-and-asian-hornet-sentry-gun)

[Tegwyn]的项目是寻求在黄蜂和亚洲黄蜂是入侵害虫的栖息地消灭它们。为了在不伤害其他物种的情况下实现这一目标，目的是训练一个神经网络来检测这些生物，然后使用激光蒸发它们。

最初的计划包括一个万向哨兵枪式的设置。然而，对在户外发射激光的安全担忧，加上对飞行昆虫成像的困难，共同导致了这个想法的搁置。目前的系统包括引导昆虫沿着蜂箱入口处的一根小管向下。在这里，如果它们被检测为黄蜂或大黄蜂，它们可以很容易地在近距离和非常详细地成像，并且可以被安全地包含在管中的激光蒸发。

这是一个令人兴奋的项目，可以作为如何处理野生入侵昆虫物种的一个很好的模型。我们以前也见过昆虫装点我们的页面。休息后的视频。
