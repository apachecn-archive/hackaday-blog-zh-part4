# 锂矿到电池线:特斯拉电池日和电动汽车的未来

> 原文：<https://hackaday.com/2021/09/14/lithium-mine-to-battery-line-tesla-battery-day-and-the-future-of-evs/>

在去年的特斯拉电池日展示和随之而来的一系列信息之后，[限制因素] [花了几个月的时间研究特斯拉宣布的计划背后的无数主题](https://www.youtube.com/watch?v=5v-DTS-ibow)，从锂到铜和钴的所有资源市场，以及所有这些对电动汽车(ev)以及电池电动汽车(bev)和储能电池的意义。

其中一些变化是立竿见影的，例如使用电池组作为结构元件以减轻支撑结构的重量，而其他一些变化，例如电池阴极不再使用钴，以及特斯拉在美国建立自己的锂粘土采矿业务的计划，都是更长期的前景。同样无法确定的是:特斯拉计划用来取代目前 18650 电池的著名“tabless”4680 电池何时将大规模生产，以及何时它们将实现承诺的 16%的增长。

即便如此，在一个多小时的视频中(休息后也有链接)，整体前景似乎相当乐观，LFP 电池也获得了欢呼。需要指出的一个明显的过程迹象是，无钴电池[已经用于](https://www.teslarati.com/tesla-model-3-lfp-battery-explains-elon-musk/)Model 3 Tesla，最常见于中国车型。
