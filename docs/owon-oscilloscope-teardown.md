# OWON 示波器拆卸

> 原文：<https://hackaday.com/2020/01/15/owon-oscilloscope-teardown/>

我们同情[learneletronic]的声明:“我感到羞耻。我可能又买了一个示波器。”我们明白了，我们喜欢看他拆下 OWON SDS1102 。(视频，嵌在下面。)正如你可能猜到的，这是一个 100 兆赫，双通道示波器，与你能以低价买到的许多其他中国示波器非常相似。

下面这段视频的最后十分钟是他移走箱子的。里面只有三块小木板。一个明明是电源。另外两个身上没多少。电路板的一部分上有一个微小的 RF 屏蔽，因此您认为那是输入部分。

较大的电路板看起来像是安装了用户界面控件。中央板似乎具有所有前端和微处理器组件。令人惊讶的是几乎没有什么可看的。

新的瞄准镜大约 250 美元左右。像这个价位的大多数示波器一样，它有一个相对较小的屏幕和一个 8 位前端。声称的采样率是 1GS/s，然而，屏幕比 Rigol DS1102E 大，这与 OWON 相当，比 OWON 贵 50 美元左右。

我们请读者[帮我们挑选一款便宜的示波器](https://hackaday.com/2016/09/20/ask-hackaday-help-me-choose-a-scope/)，发现每个人都有自己的看法。上次我们看 OWON 示波器时，它是一个令人困惑的[数字示波器模仿模拟示波器](https://hackaday.com/2019/09/10/digital-oscilloscope-does-its-best-analog-impression/)。
