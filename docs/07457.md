# 为视障人士 3D 打印交互式地图

> 原文：<https://hackaday.com/2020/08/11/3d-printing-interactive-maps-for-the-visually-impaired/>

大多数用于地理教学的地图和教材在本质上都是高度可视化的。对于那些有视觉障碍的人来说，如果没有合适的资源，学习会变得更加困难。在参观了莫斯科的一所寄宿学校后， [[Sergei]着手制作一张交互式地图，不管学生的视力状况如何，都可以教他们地理知识](https://hackaday.io/project/173875-interactive-maps-for-visually-impaired)。

在看到学校里使用的凹凸不平的纸质地图后，[Sergei]决定一定有更好的方法。解决方案是 3D 打印，这使得制作具有物理轮廓的地图变得容易。最初的尝试包括打印带有凸起特征的街道地图和世界地图，这样学生可以感觉到线条而不是看到它们。

更进一步，[Sergei]全力以赴，生产了一种交互式教育设备。该版本由一张世界地图组成，并包含有关国家、文化等信息的音频文件。当超声波传感器检测到范围内的用户时，它会邀请他们在地图上按下或拉出可移动的大陆。该设备可以感应触摸，这要归功于一对用于触发音频文件的 MPR121 电容式触摸传感器板。

在视觉可能缺乏的地方，用触觉来教学是一个很好的方法。以前的获奖作品也在这一领域发挥了作用，[像这种触觉手套可以帮助视力受损的用户解读相机数据](https://hackaday.com/2019/11/06/giving-sight-to-the-blind-with-a-wave-of-the-hand/)。随着技术的进步，我们迫不及待地想知道接下来会发生什么！
