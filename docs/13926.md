# 珠算合成器真的加起来

> 原文:[https://hack aday . com/2022/06/15/abacus-synthesizer-really-adds-up/](https://hackaday.com/2022/06/15/abacus-synthesizer-really-adds-up/)

算盘自古以来就有，并且在接受它的数百种文化中采用了类似的形式。它可能是第一批被认为在现代背景下拥有“用户界面”的设备之一——至少对于简单的算术计算来说是如此。但是使用算盘作为音乐合成器的用户界面似乎是一件全新的事情。

部分艺术概念项目和部分乐器，由[Elias Jarzombek]制作的“Abacusynth”是一种为控制合成器带来更多视觉和触觉体验的方式，而不是通常的旋钮和开关。synth 的控制部分由跨越两个胶合板立柱的四个水平杆组成。每个杆对应于复音合成器的一个声音，并固定一个安装在低摩擦轴承上的菱形旋转器。每个旋转器可以在它的控制杆上左右移动，控制声音的出现；旋转开槽旋钮通过立柱中的光电传感器控制通道的调制。每根杆的一侧都有一个旋钮，可以激活编码器来控制每种声音的波形及其谐波。

在使用中，合成器是电子音乐和动态雕塑的完美结合。旋钮似乎永远都在旋转，所以 Abacusynth 结合了一点坐立不安的旋转体验和对内置扬声器发出的新声音的探索。synth 也有一个 MIDI 接口，所以它可以很好地与其他乐器配合使用。下面的视频展示了 Abacusynth 的硬件版本；在构建之前，还有一个基于网络的模拟可以尝试。
