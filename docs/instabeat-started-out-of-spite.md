# InstaBeat 开始是出于恶意

> 原文：<https://hackaday.com/2019/11/24/instabeat-started-out-of-spite/>

[Tom]用这个[小型可编程 MP3 播放器](https://hackaday.io/project/165020-instabeat-le-mp3-player)教电子学，但它并不是作为教学工具开始的。

像所有父母有时被要求做的那样，[汤姆]充当他女儿和朋友的司机。当他播放甲壳虫乐队时，他的一位乘客告诉他，她完全没有品味，一点也不喜欢他们。所以他决定这个世界需要一个甲壳虫乐队的设备。这样，所有无知的平民都可以按下按钮进行自我教育。

该机器基于一些 SEED studio 部件和一个简单的 PCB。它可以保存所有 12 张原创专辑，甚至可以用人工生成的声音宣布它们的名称。由于该工具包易于组装，它很快就被重新用作教学辅助工具。他们开始学习[激光切割机](https://hackaday.com/2019/04/08/a-laser-cutting-101/)并做一些通孔焊接。

他计划把它变成一个任何人都可以复制的更正式的操作研讨会。他还想开发一个小的软件套件，用于文本到语音的转换，并让扬声器扮演其他角色，比如万用表。
