# 旧机器人真空获得新的钢筋混凝土租赁生活

> 原文:[https://hack aday . com/2022/12/22/old-robotic-vacuum-gets-a-new-RC-lease-on-life/](https://hackaday.com/2022/12/22/old-robotic-vacuum-gets-a-new-rc-lease-on-life/)

按照我们的想法，机器人吸尘器背后的全部目的是它们的自主性。他们并不特别擅长吸尘，但他们坚持不懈，最终在尽可能少的人工干预下完成工作。那么，你究竟为什么想要将机器人真空转换成无线电控制呢？

对卢卡斯来说，答案很简单:这是一个价值 20 美元的旧货大甩卖，为什么不呢？此外，他有一些秘密的邪恶计划，重新利用吸人机器人进行自主房间测绘，这听起来像一个很酷的项目，将受益于对这个小家伙的解剖和生理的彻底了解。有问题的机器人是胡佛的追求。像[卢卡斯]一样，我们不知道胡佛制造了机器人吸尘器(旁白:他们可能不知道)，但尽管用户在网上普遍持负面评论，他发现这是一个坚固的建造，非常模块化和可修复的单元。

在对机器人的主板进行逆向工程的最初勇敢尝试之后——我们鼓励[卢卡斯]最终回到这个项目——他满足于仅仅描述机器人的电机和传感器，并构建自己的控制器。他选择的 Raspberry Pi Zero 可能看起来有些过火，但他已经设置了它来与 PS4 游戏控制器对话，所以它是有意义的——直到他在其中释放了魔法烟雾。一个备用的 Pi 解决了这个问题，正如下面的简短视频所示，他终于能够控制这个机器人了。

[卢卡斯]对他的新小伙伴有更多的计划，包括集成原有的传感器和添加新的传感器。鉴于它的预期任务，我们会说激光雷达传感器会是一个很好的补充，但这只是一个猜测。不管他为此准备了什么，我们都很想知道会发生什么。
