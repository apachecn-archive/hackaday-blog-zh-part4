# 经过训练的可穿戴式咳嗽传感器

> 原文:[https://hack aday . com/2022/11/15/wearable-sensor-trained-to-count-coupes/](https://hackaday.com/2022/11/15/wearable-sensor-trained-to-count-coughs/)

有很多问题对人类来说很容易解决，但对计算机来说几乎是不可能的困难。尽管看起来以现代计算能力，我们应该能够解决很多这样的问题，但像识别图像中的物体这样的事情仍然相当困难。类似地，识别音频样本中的特定声音仍然是个问题，正如[Eivind]发现的那样，这阻碍了大量的医学研究。为了解决一个特殊的问题，他发明了一个计算病人咳嗽次数的系统。

这是为了帮助患有慢性阻塞性肺病(COPD)的人而建造的。大多数研究这种疾病和治疗患者的现有方法都是通过录音人工计算咳嗽次数。虽然有一些软件解决这个问题以节省一些时间，但这个设备试图在咳嗽发生时实时识别咳嗽。它通过使用 tinyML 训练一个模型来识别咳嗽并拒绝类似咳嗽的声音来实现这一点。一切都运行在一个 Arduino Nano 上，与 BLE 进行通信。

虽然模型训练的唯一数据是来自[Eivind]的声音，但现有的原型似乎显示出了希望。有了更多可靠的数据，这可能成为这种疾病患者的有力工具。而且，即使这在一个小平台上使用机器学习，我们之前已经看到过【Arudinos 有足够的能力成为有效的机器学习解决方案使用正确的工具。
