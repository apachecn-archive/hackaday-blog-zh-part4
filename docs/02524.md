# 螺纹 3D 打印零件比较

> 原文:[https://hack aday . com/2019/03/26/threaded-3d-printed-part-comparison/](https://hackaday.com/2019/03/26/threaded-3d-printed-part-comparison/)

如果你想用 3D 打印零件制造出真正的组件，你最终需要处理螺纹。最简单的方法是制作一个螺母陷阱，您可以在打印后或甚至在打印过程中插入标准螺母。然而，这种方法有其局限性。如果您想要一个真正的螺纹零件，您可以打印螺纹，用丝锥或螺栓切割螺纹，或者使用螺纹嵌件。[Stefan]进行了一些测试，看看这些方法中的每一种在实际使用中如何。(YouTube，下面嵌入。)他用了五十个测试零件生成数据进行对比。

我们喜欢螺纹插入法，在这种方法中，黄铜插入件在热的时候被推入塑料中。嵌件中的特殊特征导致黄铜部分抓住塑料，使得难以将嵌件拉出或将其拧入孔中。我们喜欢的另一件事是，测试使用了打印在水平和垂直平面上的孔。你可以清楚地看到，方向确实改变了孔的工作方式和失败方式。

测试孔是 M5。[Stephan]指出，用这些方法无法将螺栓拧紧到推荐的扭矩，但可以非常接近。有道理的是，螺纹通常不是失效的部分，但周围的材料会开裂或分层。

尽管印刷线的表现比其他方法稍差，但它们仍然出奇地好。当然，如果你计划多次拆卸和重新组装零件，你会想象螺纹会比黄铜嵌件磨损得更快。

作为最后一个技巧，[Stephan]展示了他如何有选择地增加洞周围的外壳厚度来增加强度。这取决于您使用的切片机，但大多数现代切片机可以在指定的打印量内更改参数。

我们已经看到了用螺母制成的[定制插件，它们应该工作得很好，尽管它们没有抓取塑料的功能。如果你要进行一次大规模的运行，你可能想先凑合](https://hackaday.com/2015/08/31/custom-threaded-inserts-for-3d-printing/)[一台热压机](https://hackaday.com/2015/03/06/diy-thermal-insert-press/)。
