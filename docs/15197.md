# 本周安全:IOS、OpenSSL 和 SQLite

> 原文：<https://hackaday.com/2022/10/28/this-week-in-security-ios-openssl-and-sqlite/>

本周早些时候，iOS 新版本发布，[修复了一些安全问题](https://support.apple.com/en-us/HT213489)。其中一个特别指出它“可能被积极利用了”，并被匿名举报。这个[通常意味着一个漏洞在野外被发现](https://arstechnica.com/information-technology/2022/10/apple-patches-high-severity-0-day-for-iphones-and-ipads/)，被用作一个积极活动的一部分。匿名信贷也很有趣。根据有根据的猜测，这是一次相当有针对性的攻击，发现它的安全公司不想泄露太多信息。

其他感兴趣的是与 GPU 相关的修复，归功于[【Asahi Lina】](https://twitter.com/LinaAsahi)，VTuber 正在将 Linux 移植到苹果 M1/M2 平台，并特别关注 GPU 驱动程序。她是一个有趣的案例，正在做一些令人印象深刻的工作。Linux 内核将如何处理来自化名的拉请求，这个问题仍然没有答案。无论如何，更新你的 iOS 设备。
