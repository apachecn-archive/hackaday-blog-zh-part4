# 当黑客攻击和生物传感发生冲突时

> 原文：<https://hackaday.com/2021/12/24/when-hacking-and-biosensing-collide/>

丹麦技术大学的 Edwin Hwu 教授写信给开放获取科学杂志*生物传感器*的特别版，呼吁投稿。在这一过程中，他链接了三场演讲的视频，这些演讲都是关于入侵消费电子设备以实现生物传感和纳米打印的。他们中的许多人关注于蓝光光盘读写头的巧妙使用(但这还不是全部！)这里有很多好的黑客。

例如，在原子力显微镜(AFM) 中使用光学拾波器的视频[是疯狂的。原子力显微镜通过将一个非常尖锐、非常微小的探针放在振动臂的末端，并在有问题的表面上扫描来分辨亚微米级的特征。通过反射光线并测量其变化来测量手臂的偏转，这正是这些光学传感器的设计目的。除了惊人的分辨率，Hwu 博士的]原子力显微镜可以用很少的预算制造！](https://www.youtube.com/watch?v=5bqujaldaCQ&t=419s)

说到原子力显微镜，看看这个视频中他基于简单压电圆盘的版本，但是也不要忽视其他黑客。这是一个旨在向科学家介绍消费电子黑客的演讲，所以在开始的几分钟里，你绝对会发现自己在点头。但后来他记录了使用无线“间谍相机”笔将 DVD 播放器变成高速微流体显微镜的微型闪光灯。最后，Hwu 博士的]实验室还对[纳米级 3D 打印](https://www.youtube.com/watch?v=LqVUkdZKM58)做了一些非常有趣的工作，记录在这个视频中，再次使用不起眼的蓝光驱动器，既用于曝光光聚合物，也用于用介质旋涂光盘。非常聪明！

如果你正在做任何生物传感科学黑客，一定要让 Hwu 博士]知道。或者直接打开你衣柜里积满灰尘的蓝光光驱。
