# 间谍技术:中情局面具五分钟或更少

> 原文：<https://hackaday.com/2021/05/18/spy-tech-cia-masks-in-five-minutes-or-less/>

你知道一个古老的比喻:詹姆斯·邦德被杀了，但结果是戴着好看的肖恩·康纳利面具的另一个人。《碟中谍》和《史酷比》的主题经常有所变化。但是，很明显，事实比小说更奇怪。中情局有——或者至少曾经有——一个伪装的首领。一位前任官员现在为国际间谍博物馆工作，他有一些非常有趣的故事，关于中情局特工在执行任务时使用的真实面具。

根据下面的视频，该机构获得了好莱坞的帮助，特别是《人猿星球》的面具制作人，来帮助他们完成这个项目。当然，在电影中，你可以花几个小时来应用一个遮罩，并控制它如何被照亮，摄像机如何仔细检查它，如果出现问题，你只需重做场景。如果你正在购买秘密计划，你的鼻子掉了，这可能很难解释。

虽然制作这样一个面具的技术本身就很有趣，但前首席执行官(Jonna Mendez)的评论是视频中最有趣的部分。她说:“我们(博物馆)被允许展示它(面具)的事实告诉我，中情局已经向前迈进了。”

所以你只能好奇中情局和其他间谍机构现在有什么技术来伪装一个特工。在威廉·夏特纳短暂的 TekWar 系列中，人们使用看起来像黑色兜帽的全息面具，表面上有某种发射器，可以投射出不同的面孔。或者你把你的脸放在专门的 3D 打印机里，屏住呼吸。我们将在评论中接受猜测。

显然，这类东西的行业术语是超现实面具，它们被用于从万圣节服装到抢劫银行的各种场合。尽管如此，我们想象即使是最贵的也不能在雨中跑进小巷的时候在两分钟内穿上。

如果你想知道这些面具会对人脸识别系统产生什么影响，[你不是唯一一个想知道的人](https://hackaday.com/2013/11/09/detect-disguises-with-a-raspberry-pi/)。我们不建议你自己制作硅胶面具，但是既然你不听我们的，这里有一些关于[制作模具](https://hackaday.com/2021/01/21/making-silicone-molds-big-ones/)的建议。
