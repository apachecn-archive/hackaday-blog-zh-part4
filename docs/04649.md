# 在损坏电池之前保护好你的电池

> 原文：<https://hackaday.com/2019/10/30/protect-your-batteries-before-you-wreck-your-batteries/>

[Jan]正在解决一个我们很多人都遇到过的问题，[深度放电我们项目的电池](https://hackaday.io/project/164742-universal-undervoltage-lockout-up-to-34v-input)并潜在地损坏电池。

他的主板可以处理 6 到 34 伏的电池，支持 LiPo 或 Lion 电池。该板可以灵活调整其截止电压。它还有一个我们很喜欢的特点；用户可以设置关断电池前的延迟时间:当峰值电流过大导致电池在低于阈值的电压下工作几秒钟时非常有用。一旦电路板被关闭，需要手动复位，以允许电源再次被提取。

他的最新版本是令人印象深刻的 1 平方英尺。英寸大小！这可以适用于任何项目，甚至可以灵活选择电池连接器。下一次，我们有一个高电流消耗的项目，需要昂贵的电池或可能需要长时间运行的监控设备，为了安全起见，我们可能会在那里放一块这样的板。
