# 电动直排轮滑鞋依靠牧田动力运行

> 原文：<https://hackaday.com/2020/01/16/motorized-inline-skates-run-on-makita-power/>

直排轮滑鞋可能很有趣，但像大多数有轮子的装置一样，它们在机动化时甚至更好。怀着这样一个目标，[【现实生活中的家伙们】决定制造一套电动溜冰鞋，用牧田的动力来跑！](https://www.youtube.com/watch?v=Sva_1gwH9vY)

为了给地面提供动力，每只冰鞋上的第三个轮子被改装成带有链轮齿。然后在冰鞋上安装一个 Makita 钻头，通过 90 度齿轮箱和链传动将动力传递给轮子。通过将扳机从弹壳上取下并用一根延长的电缆将其连接起来来控制钻头。

这是一个俗人的设置，可能需要认真练习才能有效使用，但如果你真的想炫耀，也允许像差速转向这样的花式技巧。这是一个很好的例子，使用电钻作为一个包罗万象的运动设置，电池，电机和传动系统已经集成在一个整洁的包装中。我们也不是第一次看到电动滑冰鞋了！休息后的视频。
