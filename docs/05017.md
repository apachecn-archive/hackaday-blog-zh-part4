# 你能给标致压电吗？

> 原文:[https://hackaday.com/2019/12/09/can-you-piezo-a-peugeot/](https://hackaday.com/2019/12/09/can-you-piezo-a-peugeot/)

当谈到气候变化时，汽车制造商有一个问题。在大气层外二氧化碳的各种来源中，他们的产品可能是最受公众关注的，因此他们的营销部门正采取更加绝望的措施，用绿色光环来美化它们。其中包括法国品牌标致，其 208 车型[的新电动版本出现在一个精美的视频中，旁边还有一个未来主义的能量收集广告牌](https://www.youtube.com/watch?amp;v=6_lVGiokmNM)。

这不是一个普通的广告牌，也不是传统的风力涡轮机或太阳能电池阵列，而是在巴黎最繁忙的地区之一收集环境噪音，并通过一系列压电能量捕获单元将其转化为电能为汽车充电。这在 Hackaday 引起了我们的注意，因为它似乎好得令人难以置信。这是一个营销噱头，还是你可以做一个压电广告牌作为一个实用的绿色能源设备？让我们仔细看看。

## 压电到底是什么？

正如大多数读者所知，压电效应是某些材料的一种特性，通过这种特性，当材料变形时，电荷会在其表面积聚。反过来，当外部电荷施加到压电材料上时，压电材料也会变形。

常见的压电物质有多种，包括晶体材料和聚合物，大多数人都会在家庭应用中遇到它们，如扬声器、蜂鸣器和气体打火机，就像点燃烤架的 clicky 按钮。这些气体打火机产生的火花表明，压电材料产生的电压可能大得惊人，但它们天生具有极高的阻抗，因此它们通常不能输送很大的电流。是的，理论上它们可以用来产生有用的能量，但是对于一个不经意的观察者来说，它们似乎不是一个显而易见的选择。因此，标致的广告牌值得仔细检查。

[![That's a lot of greeting cards](../Images/6a71e816f124d59d48fd0b23d50992d6.png)](https://hackaday.com/wp-content/uploads/2019/11/piezo-pug-array.jpg)

That’s a lot of greeting cards

我们可能期望广告牌使用一种先进的材料，也许是一种具有大表面积的压电聚合物。令人惊讶的是，在视频中看到它是由大量常见的压电圆盘发声器组成的，这些发声器可以在许多廉价玩具和音乐贺卡中找到。它们由激光切割板固定在一个阵列中，每个后面都有一个小的印刷电路板，用于固定电源连接器，将其输出馈送到另一个可能包含某种调节器的 PCB。

这些组件的使用确实令人惊讶。虽然它们无疑会从环境声音中产生一些电力，但这肯定不是它们的预期用途。这有几个原因，它们在于它们所面对的声音的性质以及它们与用于能量收集的压电装置的不同。

## 精心调整的能量收集它不是

压电能量收集器确实存在，它们可以产生可用的能量，但要做到这一点，它们必须被轻微扭曲或弯曲。它们最适合于从特定频率的持续振动中获取能量，并且通常被选择和安装成在接近其所在系统的共振频率下工作。

如果你弯曲廉价的压电发声器圆盘，它们也会发电，并且也有共振频率，但是在这里它们没有以导致它们这样做的方式安装，并且被馈送相当于宽频谱的频率，而不是仔细调谐的振动源。因此，它们不太可能在接近峰值效率的任何地方运行，无论如何，峰值效率都不会像专门制造的设备那样高。

[![The system also has a mains power connection.](../Images/082549dd6715f29ecb94c367409f5b81.png)](https://hackaday.com/wp-content/uploads/2019/11/piezo-pug-diagram.jpg)

The system also has a mains power connection.

疑问的决定性时刻出现在标致视频本身的短暂瞬间。整个项目在技术细节上非常轻，但我们确实看到了一个显示城市电网连接的框图。在这一点上，故事变得清晰了，这不是一个声音采集充电点，而是一个普通的电源充电点，可以从声音采集面板获取一些输入。因此，可以说这是一个宣传噱头，而不是一个获取能量的严肃尝试。

这是一个遗憾，因为它仍然是一个有趣的项目，它的创造者显然投入了大量的工作。我们怀疑标致是否会公布其设计、构造和效率的完整技术报告，但我们认为这至少不会损害他们在我们社区的形象(恰恰相反，我们想看数据！).电动汽车仍然是一种小众产品，对技术感兴趣的人更有吸引力，因此围绕它们的任何技术都会带来大量读者。

如果有最后一点要从这里收集的话，那就是环境声音中包含了大量的能量，那么它是如何被收集的呢？我们年轻时对高阻抗动铁耳机的实验的直觉会带我们走向音频带宽更宽的东西，但我们不确定一系列多余的 20 世纪 60 年代电话部件是否会比廉价的贺卡扬声器更有效或更合适。我们相信你会有自己的观点，请在评论中畅所欲言。
