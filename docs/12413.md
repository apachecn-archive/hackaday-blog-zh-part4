# 酸损坏的游戏男孩恢复

> 原文：<https://hackaday.com/2022/01/17/acid-damaged-game-boy-restored/>

最初的 Game Boy 是有史以来销量最大的掌上电子游戏系统，只是被它的继任者超越了。它仍然以很大的优势保持着第二的位置，但即使如此，他们现在已经相处多年，找到一个处于完美工作状态的可能比你想象的更难。更有可能的是，你会发现一台电脑缺少组件，屏幕出现故障，[或者它的电子元件被几十年前的 AAs](https://hackaday.io/project/183423-restoring-a-battery-acid-eaten-gameboy) 电池酸液腐蚀了。

后一种情况就是[泰勒]发现自己并决定对这部经典作品进行全面修复的地方。首先，他从受损区域移除了所有组件，这样他就可以看到痕迹的路径。在对损坏部分进行清理并移除阻焊膜后，他使用 30 号线桥接 PCB 的受损部分，然后将所有部分重新组装到正确的位置。有几个需要被替换，但最终 Game Boy 恢复了 90 年代的辉煌。

这种构造是一个很好的例子，说明了尖端精细的烙铁可以做什么，同时也提醒人们不要将 AA 电池长时间留在任何设备中。AA 电池对于最初的游戏玩家来说总是一个弱点，所以如果你决定放弃任何类型的电池[，你可以建立一个这样的电池](https://hackaday.com/2020/09/11/game-boy-plays-forever/)。
