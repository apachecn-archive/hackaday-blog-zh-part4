# 为拥有一切的人准备的药片分配器

> 原文:[https://hack aday . com/2022/09/16/a-pill-dispenser-for-the-person-with-everything/](https://hackaday.com/2022/09/16/a-pill-dispenser-for-the-person-with-everything/)

整理药片是一项相当单调乏味的任务，自动化的时机已经成熟。这是一项[Mellow]热情承担的任务，其结果是[一个设计非常好的分配器](https://www.mellowfire.com/post/automated-pill-dispenser-finally-finished)，它有一堆由 ESP8266 控制的伺服料斗，可以按时分配所需的药丸。

有一系列的视频，我们把最新的放在休息时间下面，展示了这个项目的各种迭代。早期版本使用多个微控制器，而不是单个 ESP，他的传感器选择既简单又巧妙。单个振动传感器检测落在上面的药丸，从而实现极其紧凑的电子设置和 3D 打印堆栈的基础。

这种设计给我们留下了深刻的印象，它的简单性、独创性和令人愉悦的美感，使用一块 perfboard 和大量热收缩材料来制作一个非常整洁的布线织机。我们不确定我们是否会需要这样的药片分配器，但如果我们需要，我们不认为我们能提出更好的设计。

你可能会惊讶地发现，药丸分配器[在](https://hackaday.com/2020/05/16/did-grandma-remember-her-pills-this-dispenser-tells-you/)之前就已经出现在这里了。
