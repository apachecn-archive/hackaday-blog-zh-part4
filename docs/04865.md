# 二十面体发光是其中最好的

> 原文:[https://hack aday . com/2019/11/22/二十面体-与最好的人一起发光/](https://hackaday.com/2019/11/22/icosahedron-glows-with-the-best-of-them/)

Glowables 有各种形状和大小，我们总是渴望看到黑客找到各种不同的方法来很好地利用大量的 led。[cabrera.101]想参与其中，[并迅速做出了一个相当华丽的二十面体。](http://wiki.lvl1.org/Neopixel_Icosahedron)

该建筑的边缘使用了每米 144 个 LED 的高密度条带，每米 60 个 LED 的条带用于连接中间不锈钢球的管子。Arduino Mega 控制 Neopixel 条，布线经过精心规划，以确保所有 led 都有足够的功率和信号来正常工作。[cabrera.101]不是一个吝啬鬼，他给钻机配备了 5V、60A 的电源——这在 1992 年似乎很可笑，但在今天几乎不会引起人们的注意。

这将是一个完美的科幻电影的背景。边缘光线在中央球体上的反射特别闪烁。如果你是 glowables 领域的新手，这很容易开始——[也有很多工具可以帮助你。](https://hackaday.com/2019/10/05/how-to-get-started-with-fadecandy-and-leds/)休息后的视频。
