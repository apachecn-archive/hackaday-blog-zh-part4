# Cosplay 道具制作教程和蓝图的宝库

> 原文：<https://hackaday.com/2019/02/09/a-trove-of-cosplay-prop-making-tutorials-and-blueprints/>

[CutTransformGlue]最近发布了一个“[制作雷伊的星球大战爆破手](https://twitter.com/cuttransform)”的构建视频，在休息后嵌入。建筑使用多层中密度纤维板来建造爆破器，看到它成形是一种享受，以惊人的油漆工作结束。这是一个很好的方法来学习将这些道具运用到生活中的技巧，并帮助你磨练你的技能。但是深入挖掘让我们陷入了一个可怕的兔子洞。

[CutTransformGlue]从[惩罚道具学院](https://punishedprops.com)获得了 Rey 的爆破器的计划，这是一个来自西雅图的道具和服装制作团队，致力于“将热情的粉丝转变为自信、熟练的制作者”。这些人建造了[各种各样的项目](https://punishedprops.com/category/blogs/)，从枪支、武器、服装、道具等等，显然他们对自己的工作非常熟练。但是他们并没有把这些技能留给自己，在一系列的帖子和视频中，他们与我们分享了各种各样的[技能](https://punishedprops.com/category/techniques/)，如泡沫铸造(必须爱上这个铸币)、成型、铸造、绘画、3D 打印、真空成型和电子技术。如果你想了解更多关于供应品的信息，请查看工具和材料部分。如果所有这些都让你想造一把 Skyrim Wuuthrad 或 Halo4 狙击步枪，那就去惊人的[免费蓝图区](https://punishedprops.com/free-blueprint-files/)找一个装满下载的宝箱吧。

就像我们之前说的，如果建造这样的东西是你的事情，这是一个兔子洞，你会发现很难从中脱身。玩得开心。
