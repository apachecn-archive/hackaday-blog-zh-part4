# 印刷零件将尺子变成标记量规

> 原文：<https://hackaday.com/2018/11/07/printed-parts-turn-ruler-into-marking-gauge/>

对于那些花更多时间在烙铁上而不是锯子上的业余读者来说，划线规是一种用来在一块木头(偶尔是金属或塑料)上划平行线进行切割的工具。该工具在要标记的工件边缘运行，用户可以通过调整来设置划线的距离。例如，如果您想将纸板切割成更小的长条，划线规将是提前布置线条的理想选择。

但是和许多小众工具一样，它不是你每天都会用到的东西。对[chaosbc]来说，这意味着他想看看自己能否找到一个便宜的 DIY 解决方案。此外，他现在就可以拥有它，而不是等着它从海外乘慢船而来。通过添加一些聪明的 3D 打印组件，他能够[将他可信赖的铝尺变成一个可用的标记量规](https://hackaday.io/project/162187-lets-hack-my-ruler)，只需花费细丝和一些硬件。

划线规的一般设计相当简单:有一个可以上下移动刻度轴(称为主轴箱)的块，它可以让你设置线的深度，然后在末端有一块可以固定你的划线工具。如果你用的是足够软的东西，那么标记工具可以是一把刀片，但是对于木头来说，通常是一支铅笔。

[chaosbc]为他的 DIY 标记量规提供了所有的 STL 文件，尽管它们可能需要修改，因为它们是为他的特定尺子创建的。幸运的是，这些部分并不复杂，所以解决起来应该不会太难。他还有一个有用的提示给那些想要复制他的作品的人:在用来锁住主轴箱的螺栓上滴几滴强力胶就足以创造一个无损伤的表面，这样你就不会撕毁你的尺子。
