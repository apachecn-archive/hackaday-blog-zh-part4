# 向量收集器检查器

> 原文:[https://hackaday.com/2021/09/13/vektor-kollektor-inspector/](https://hackaday.com/2021/09/13/vektor-kollektor-inspector/)

随着世界再次开放，[Niklas Roy]和[Kati hyypp]一直忙于制作一个公共的合作项目。[体验 Vektor Kollektor](https://niklasroy.com/vektorkollektor/project.html) ，这是一种便携式绘图机体验，配有芯片调谐配乐。很高兴看到公共艺术与创客社区相遇，毫无矫饰，充满乐趣！

[![](../Images/f646c80e4737354b68ddc1c2ec14c143.png)](https://hackaday.com/wp-content/uploads/2021/09/niklas-roy-vektorkollektor-thumb.gif) 建造从 80 年代的 HP7475A 笔式绘图仪开始，这是一台 DOA(或在初始测试中烧毁)的绘图仪。[Niklas]和[Kati]保留了机械装置，但重新设计了控制装置，便于与 Arduino Nano 集成，并由摩托车电池供电。

神奇之处似乎不在于垃圾箱的构建(这很好)，而在于这个团队扩展项目的方式。使用带有街机按钮的操纵杆作为输入，他们将 Vektor Kollektor 带到公共公园和街道，邀请其他人在那里进行艺术创作。Kollekted 绘画可以在画廊网站上以非常酷的动画形式免费下载，可以在 t 恤、3D 打印和咖啡杯上使用，因为，为什么不呢？

一些精选的图纸甚至用大型绘图仪喷涂在墙上，我们真的希望[Niklas Roy]和[Kati hyypp]尽快分享这个建筑的细节。当然，这是紧随我们从这两位硬件艺术家那里看到的[车间窗口机器人](https://hackaday.com/2021/09/07/pandemic-gives-passersby-a-window-on-cyborg-control/)之后出现的。
