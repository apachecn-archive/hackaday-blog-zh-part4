# 一台看着你前进方向的割草机

> 原文：<https://hackaday.com/2020/10/15/a-lawnmower-that-looks-where-youre-going/>

小时候，我爸爸给我讲过一个故事，讲的是在农场里用一台气动割草机割草。一天，某种农具被留在地里，旧的工业割草机将它撕碎，一大块锋利的金属飞向他的腿。对我爸爸来说幸运的是，大塑料轮设法阻止了金属片，破坏了轮子。我的祖父很沮丧，因为他需要修理割草机，但很感激我的父亲仍然双脚连着。

当然，这个故事给了我一个教训，让我不要抱怨轮到我修剪草坪了，但也给了我一个教训，剪草机可能很危险。[胶带机械师]自己尝试是否能完全防止这种事故，并为他的家庭割草机创造了一种[自动安全关闭机制。(视频嵌入下方。)](https://www.youtube.com/watch?v=eDZeLPgYBvg)

这使用了一个感应传感器，可以在金属被吸进割草机之前检测到金属。传感器触发继电器，通过点火线圈接地强制关闭割草机。虽然它不像[其他安全机制](https://hackaday.com/2017/06/22/ask-hackaday-sawstop-bastion-of-safety-or-patent-troll/)那样物理地停止刀片，但它确实通过尽快关闭刀片的电源来防止情况升级。接近点火线圈并不容易，因为它需要深入发动机本身，但现在[DuctTape Mechanic]有一种割草机，可以通过电容传感器或更多智能来进一步扩展，以确定它是在探测地下还是地上的金属。

总有一天我们会有机器人割草机，但在那之前，我们会称赞黑客们努力让世界变得更安全。
