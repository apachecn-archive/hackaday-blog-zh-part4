# 液压系统变得简单

> 原文：<https://hackaday.com/2021/04/07/hydraulics-made-simple/>

收集电子是伟大的，也是我们大多数人非常擅长的，但最好的项目都有某种与现实世界的接口。通常，这涉及到某种流体，如水或空气在管道中流动。如果你没有直观地掌握水力学，[实用工程]有一个你会喜欢的视频。它解释了[流量和压力如何在管道](https://www.youtube.com/watch?v=ZQKpu-obzlU)中工作。

当然，并不是每个项目都涉及管道，但是管道、洒水系统、冷却系统，甚至机器人通常都有液压元素。此外，正如视频所指出的那样，管道中的流体流动与流经电线的电流非常相似。

关于摩擦损失的讨论涵盖了一些数学知识，但我们惊讶的是没有听到太多关于测量湍流的雷诺数的信息，尽管视频在最后确实涵盖了湍流。Hazen-Williams 方程的 C 项不依赖于雷诺数，但仅对室温下有限流速范围内的水有效。如果你正在使用其他流体或在其他操作条件下工作，你将需要更多的数学，但视频仍然给你一些直觉，这将有助于不管流体。

然而，真正的宝石是实验测试装置，有清晰的管道和一些流量计和压力计。我们不禁也注意到了仪器上的眼睛。

虽然你可能每个项目都需要液压直觉，但多了解它是一件好事。[流体机器人](https://hackaday.com/2018/10/03/giant-robot-arm-uses-fluid-power-not-electronics/)有一定的优势。通过一些工作，你甚至可以 [3D 打印液压](https://hackaday.com/2015/12/20/3d-printed-hydraulics/)。
