# 老鼠在这个自动化的虚拟现实竞技场中学习玩毁灭战士

> 原文：<https://hackaday.com/2021/12/29/rats-learn-to-play-doom-in-this-automated-vr-arena/>

当我们发表一篇标题为“DOOM”的文章时，这通常是另一个让古老的游戏在某个极简平台上运行的例子。然而，这款基于毁灭战士的老鼠虚拟现实装备与其说是在黑毁灭战士，不如说是在黑老鼠。

起初是[维克多·托特]的一个附带项目，现在已经发展成为一个相当复杂的装置。在设备的中心是一个全向跑步机，由一个保龄球大小的聚苯乙烯球组成。这个球可以自由旋转，传感器可以检测两个轴的旋转——这基本上是一个倒置的大型机电鼠标。老鼠骑在球的顶部，戴着一个挽具以防球滑落。一个巨大的曲面显示器位于老鼠的正前方，显示虚拟环境，这是一个定制的末日地图。

随着虚拟现实装备的建立，[维克多]致力于训练的自动化。糖果分配器提供了适当的动力，而动力驱动轮与球接合，如果老鼠被困在虚拟世界中，就轻推它。[维克多]说他已经用这种自动化的方法训练了三只老鼠——[罗梅罗]、[卡马克]和[汤姆]——在笔直的走廊上行走。至于游戏的核心——射击怪物——[ Viktor]也涵盖了这一点，它有一个传感器，可以检测到老鼠何时用后腿直立起来进行射击。

让老鼠达到视频中所看到的状态的总训练时间约为六周，据[维克多]报道，整个训练花费了他大约 2000 美元。这花费了大量的时间和金钱，但是结果却非常有趣。如果你对极简主义的末日建筑更感兴趣，我们理解——看看灯泡上的[末日，或者](https://hackaday.com/2021/06/15/a-smart-light-bulb-running-doom-is-a-pretty-bright-idea/)[恒温器](https://hackaday.com/2017/05/22/doomed-thermostat/)，或者[甚至 GPS](https://hackaday.com/2019/09/01/running-doom-on-a-doomed-gps/) 。
