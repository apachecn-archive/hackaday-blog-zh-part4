# 用经济实惠的复合装甲为您的汽车提供防弹保护

> 原文：<https://hackaday.com/2020/01/18/bullet-proofing-your-car-with-an-affordable-composite-armor/>

还记得像《速度与激情》这样的动作电影吗？在这些电影中，汽车不断被高速飞行的子弹撞得粉碎。怎样才能保护车辆免受 AK-47 的攻击？在[PrepTech]的[三部曲](https://www.youtube.com/watch?v=5Ih1drMkOjY) [DIY 复合材料](https://www.youtube.com/watch?v=HC_-GHuP8TM) [车辆装甲](https://www.youtube.com/watch?v=E6b3v3KJwBs)教程中，他展示了自己是如何从头开始制作防弹装甲的。即使你认为整个文明的彻底崩溃有点牵强，你也不得不承认这很酷。

第一部分处理实际构建组合。他使用多层不锈钢、陶瓷马赛克瓷砖、玻璃纤维以及环氧树脂来建造复合材料。选择树脂是因为其高三维交联密度，而玻璃纤维恰好是最便宜的复合织物。考虑到切割玻璃纤维产生的微小碎片的性质，必须非常小心，以免这些碎片最终进入你的衣服或脸上。戴上呼吸器和手套，以及防护外层，会有所帮助。

层压后的织物，它硬化到一点，个别股变得僵硬。下一层是硬陶瓷，它的作用是使抛射体变形并减速，使其在撞击时损失约 40%的动能。他在瓷砖之间注入硅树脂以增加弹性。[PrepTech]不是使用一个只能承受一次冲击的大瓷砖，而是使用瓷砖的马赛克，允许多个瓷砖受到冲击而不影响周围瓷砖的完整性。虽然工业装甲使用硼或碳化硅，但陶瓷成本要低得多。

不锈钢来自一个废弃的垃圾场，被切割成适合其他瓷砖的尺寸，然后用环氧树脂粘合到复合材料的其余部分。最终结果允许放置一周，以使环氧树脂在接受弹道测试之前完全硬化。一枚格洛克---- kor pion vz ----遗留下来的子弹穿透了钢板。61，和 AK-47，但被德拉古诺夫狙击步枪击穿。将不锈钢的厚度增加到至少一厘米的弹道级钢可能有助于保护钢板免受更大口径的攻击，但[PrepTech]解释说，他无法在自己的国家获得这种材料。

然而，较低的口径仍然无法穿透甚至钢铁，所以除非你计划在高口径武器上测试钢板，否则它肯定是低成本防御工具的成功。
