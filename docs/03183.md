# 微型叉车让无用的空间变得可用

> 原文:[https://hack aday . com/2019/05/30/tiny-铲车-使-不可用-空间-可用/](https://hackaday.com/2019/05/30/tiny-forklift-makes-unusable-space-usable/)

在某些地理区域，带有爬行空间的房屋相当普遍。这种爬行空间可以使管道和电线等物品更容易接近，甚至可以在没有(或不能)地下室的家庭中用作储物空间。随着建筑通风的改善，与建在混凝土板上的房屋相比，这些都是额外的好处。虽然这些爬行空间不太容易进去，但[戴夫]有一种更容易的方法来进出这些有用但很小的空间。

叉车进入爬行空间。该机器人主要由现成的组件制成，包括几个标准电机和线性致动器，以移动和操作前叉。这都是非常标准的，但这个建筑真正的亮点在于它使用了 FPV 相机，监视器和发射器，允许飞行员通过远程控制在小空间内导航机器人。对于我们当中那些有安全意识的人来说，船上还有一个[灭火球](https://www.youtube.com/watch?v=L_0sawCkIk4)，万一机器人在他的房子下面着火了，它会自动启动。

这是一个伟大的、高质量的构建，展示了普通部件如何通过正确的想法来实现革命性的东西。发现问题，然后制定解决方案，同时不要忘记购买一些安全设备，即使是像家中闲置空间这样简单的事情，也能真正发挥作用。他们也可以处理家务。
