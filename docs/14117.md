# 用电子观察原子

> 原文：<https://hackaday.com/2022/07/05/viewing-atoms-with-electrons/>

当我们在学校的时候，他们总是告诉我们我们看不见原子。如果你有电子显微镜，那么他们错了。[AlphaPhoenix]获得了一台[扫描~~隧道~~透射电子显微镜](https://www.youtube.com/watch?v=eYVNZgnQ8gE)，他向我们展示了一片非常薄的晶体中的一些原子。

当然，你没有直接对原子成像。你看着原子的影子，但仍然。如果你以前从未使用过 SEM 或 STEM，这里有很多有趣的小细节，比如样品架和真空系统。

让我们震惊的一件事是，与我们过去使用的 SEM 相比，计算机确实改变了一切。在 20 世纪 80 年代，我们的 SEM 看起来就像某人心目中的 25 世纪航天飞机，有旋钮、刻度盘、仪表、杠杆，甚至踏板。现在，它看起来像一台有一堆显示器的电脑。

比图片更好的是，我们真的很喜欢显微镜如何工作的解释和宏观演示。如果你以前没有使用过这项技术，它是惊人的，他甚至没有涉及到像能量色散光谱和其他有用的技术。

建造自己的电子显微镜并不容易，但有些人确实在尝试。还有[试试](https://hackaday.com/2016/12/02/nascent-project-open-source-scanning-electron-microscope/)。你最好还是抓住一个盈余。
