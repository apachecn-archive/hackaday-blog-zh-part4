# 卡带上的塞尔达汇编专辑

> 原文：<https://hackaday.com/2019/04/05/a-zelda-compilation-album-on-a-cartridge/>

塞尔达系列游戏因其令人兴奋的游戏性、引人入胜的故事以及它们的原声音乐而闻名。从快节奏的 boss 战，到情绪混乱的场景，这些曲调多年来已经在数百万台任天堂游戏机上播放。[Tyler Barnes]已经是一个很长时间的粉丝了，并决定制作其中一些曲目的汇编-[当然是用卡带来发表。](http://nintendoage.com/forum/messageview.cfm?catid=22&threadid=186475&StartRow=16&fbclid=IwAR2CblFNVQTzKe_nhzgoqYSNOMMe88Trv4yTUNO7ig4NKtSXVqjjVYFDT3E#bottom)

音乐是使用[音乐宏语言](https://en.wikipedia.org/wiki/Music_Macro_Language)创作的，并编码成 NSF 格式，准备在任天堂娱乐系统上播放。[泰勒]编码了一个菜单系统，允许用户选择他们想听的曲目。还有一些漂亮的视差动画，以及一个复活节彩蛋，供那些足够了解游戏的人解锁。

[Tyler]希望刻录一些 EEPROMs 并发送一些定制的购物车，无论是使用捐赠者的购物车还是新构建的购物车。如果你是 NES 音乐的粉丝，但也需要 C64 补丁，[也有一个解决方案](https://hackaday.com/2014/07/12/commodore-64-and-nintendo-make-beautiful-music-together-with-syncart/)。休息后的视频。
