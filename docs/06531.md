# 用 LE STRUM 在树枝上摇摆

> 原文：<https://hackaday.com/2020/05/23/rocking-out-on-a-limb-with-le-strum/>

很难想象没有 keytar 的 80 年代 Synth-pop，然而这种庄严的半人马乐器很少得到太多的爱，他们的球员经常成为嘲笑的目标。似乎挂在脖子上应该是专为弦乐器保留的特权。好吧，[midierror]至少有一部分是正确的，MIDI 腿上的完整部分保证让每个键盘手相比之下看起来像一个有声望的大提琴手。

看起来像 1987 年电影*中的人体模型*进行了一次黑暗的、门格尔斯式的转弯，这既令人敬畏又很怪异，这要归功于将[乐乱弹](https://www.tindie.com/products/hotchk155/le-strum-midi-strummed-chord-controller-kit/)的概念构建到一条腿中。LE STRUM 本身是一种由 Jason Hotchkiss 打造的[开源](https://github.com/hotchk155/Voici-Le-Strum) MIDI 乐器，他将其描述为“唱针和全弦的结合体”。它由一组按钮组成，可以选择不同的和弦组合，然后通过在一组触板上刮擦连接的唱针来弹奏。然而，[midierror]还发行了预组装版本的 LE STRUM 的[,它使用琴弦而不是接触垫，并使用拨片进行实际的弹奏，将它变成了一种足够接近的弦乐器。](https://www.tindie.com/products/midierror/le-strum-midi-chord-controller-constructed/)

现在唯一缺少的是一个功能正常的膝关节，也许从这个 MIDI 控制的手风琴中得到一些灵感，我们已经准备好用 kneetar 来彻底改变手风琴世界？由于它是围绕一张照片而设计的，这款拍大腿游戏甚至不会花费你一只胳膊，只需要一条腿——但这些令人毛骨悚然的双关语已经足够了。
