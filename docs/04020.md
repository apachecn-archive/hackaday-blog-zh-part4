# 另一种在家制造多氯联苯的方法

> 原文：<https://hackaday.com/2019/08/27/another-way-to-make-pcbs-at-home/>

推出您自己的定制 PCB 的一种更受欢迎的方法是，在您选择的 CAD 程序中简单地创建模型，并将其发送给电路板制造商，他们会为您处理脏工作。这样就不需要处理化学品、铜尘或维护昂贵的工具。不过，电路板制造商和家庭蚀刻系统之间的中间地带可能是[igorfonseca83]一直在做的事情:[使用廉价的激光雕刻机为他制作 PCBs】。](https://hackaday.io/project/167325-custom-pcb-using-a-low-power-laser-engraver)

激光雕刻机基本上是一个低功率的激光数控机器，只是稍微太弱，无法切割大多数通常用激光切割机切割的东西。事实证明，10W 系统是从标准 PCB 坯料上移除掩模的完美能量。这实际上取代了旧的墨粉转移方法中的打印机，铜仍然必须溶解在化学溶液中，但结果比试图修改打印机来完成这项任务要稳健得多。

如果你不熟悉自制 PCB 涉及标准桌面打印机的时代，[许多人仍在使用这种方法](https://hackaday.com/2016/09/12/take-your-pcbs-from-good-to-great-toner-transfer/)，尽管结果可能因打印机的可靠性而有所不同。如果你想跳过中间人，并且需要化学浴，一种[更强大的激光实际上也可以为你切割痕迹](https://hackaday.com/2013/01/22/laser-cut-pcbs/)。
