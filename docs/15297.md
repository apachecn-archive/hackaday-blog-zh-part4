# 绳芯滚筒机

> 原文:[https://hackaday.com/2022/11/06/rope-core-drum-machine/](https://hackaday.com/2022/11/06/rope-core-drum-machine/)

我们最喜欢的音乐黑客之一，[看，妈妈，没有电脑]正在危险地接近制造一台电脑。他的追求是创造一种独特的鼓机，灵感来自[，一种苏联自动拨号器，使用绳芯存储器存储号码](https://hackaday.com/2022/01/13/soviet-era-auto-dialler-uses-magnetic-rope-core-memory/)。Rope 内存是磁心内存的只读兄弟，磁心内存是 60 年代和 70 年代初用于构建一些受欢迎的计算机的内存技术。绳芯不是通过磁化陶瓷甜甜圈来编程的，而是通过在其中编织一根电线来编程的。当[看妈妈]看到自动拨号器使用用户可编程接口的技术时，自然地，他只需要建立一个 synth 音序器。

基本技术是利用连接到比较器的初级绕组来监控铁氧体环。然后将信号线穿过一些环，并使用 555 通过它发送振荡信号。单个位，也称为铁氧体环，拾取信号并触发比较器。(用变形金刚当比特，多好玩啊！)所以对于一个工作的音序器来说，每个铁氧体环控制一个乐器。环 1 是底鼓，环 2 是小军鼓，等等。每根电线都是不同的步骤，它们被依次供电。将位输出连接到您的乐器，然后编织您的节拍。这已经是我们见过的最独特的音序器，但还有更多。

因为他使用非常大的铁氧体磁芯，所以有足够的空间从他的社区获取节拍提交，所以有八个不同的节拍“编程”到这个音序器中。并在它们之间切换？看起来[看妈妈]正在使用旧电话系统的一步一步的模块。打电话！
