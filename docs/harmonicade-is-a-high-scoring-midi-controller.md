# Harmonicade 是一个高分的 MIDI 控制器

> 原文：<https://hackaday.com/2020/01/29/harmonicade-is-a-high-scoring-midi-controller/>

当[KOOP 乐器]开始学习钢琴时，他没有准备好在标准键盘布局上学习和弦及其关系的单调乏味。但这并没有扼杀他逗钢琴的欲望，而是激发他探索更容易演奏的替代布局。他转换到同构，开始建立 MIDI 控制器，一直没有回头。

最新的咒语是 [Harmonicade](https://www.koopinstruments.com/instrument-projects/harmonicade) ，一个双层数字排列[威奇-海登风格](https://en.wikipedia.org/wiki/Wicki-Hayden_note_layout)。两个甲板都有 5 个八度音阶，是(电气)相同的，并运行了一个单一的 Teensy 3.6。我们钦佩[KOOP]使用 DB25 连接器来解决甲板和甲板之间的布线问题——这是一个非常巧妙的解决方案。几乎和他的[注释优美的代码](https://github.com/KOOPInstruments/harmonicade/blob/master/harmonicade.ino)一样简洁。

虽然按钮板和控制盒都是打印的，并且是开源的，但是它们被设计成容易由丙烯酸或胶合板制成。[KOOP]将继续迭代，直到他完全满意控制位置和布局，以及分解和重组的容易程度。休息之后，我们为您准备了一个双镜头视频——一个是[KOOP]演奏的 Harmonicade，另一个更长的是探索和演奏它的前身 Melodicade。

厌倦了传统外观的 MIDI 控制器？我们听到你的风箱，并提供这个[在手风琴](https://hackaday.com/2019/08/18/midi-controller-in-a-concertina-looks-sea-shanty-ready/)的 MIDI 控制器。
