# 用 Nerf 无人机获得近距离空中支援

> 原文：<https://hackaday.com/2021/07/26/get-some-close-air-support-with-a-nerf-drone/>

在家工作的最大优势是有更多的时间和爱人在一起，但是这种时间有时会导致摩擦。[Cory]发现 Nerf 和他的孩子打架是发泄情绪的有效方式，但觉得他在这个过程中做了太多的锻炼。相反，他为 FPV 四轴飞行器配备了 3D 打印 Nerf 枪来取代他的位置。

由于每次射击后手动给 Nerf 枪重新装弹是不可行的，他需要创造一个自动装弹机。飞镖由一对并排安装的无刷无人机电机推动，中间的空间刚好够一个飞镖挤压。马达被允许旋转起来，然后从现成的 Nerf 弹匣中取出一个装在伺服操作活塞上的飞镖。发动机 ESCs 和伺服系统由 Arduino Nano 控制，Arduino Nano 从无人机飞行控制器上的一个备用输出接收开火命令。nerf 设备很容易从无人机上拆卸下来，所以[Cory]也可以驾驶无人机执行更和平的任务。休息后看其中一场战斗的视频。[Cory]可能需要找到一个替代的控制位置，以防止自己被对手用作掩护。

Nerf 枪是一种有趣而无害的方式来实现你的科幻战争幻想，尤其是在我们这些天可用的技术下。从 [FPV 哨兵枪](https://hackaday.com/2021/07/23/a-nerf-ball-turret-complete-with-fpv/)到[自动瞄准步枪](https://hackaday.com/2021/07/12/a-capable-nerf-launcher-robot/)，还有[重炮](https://hackaday.com/2021/07/12/a-capable-nerf-launcher-robot/)，一切皆有可能。
