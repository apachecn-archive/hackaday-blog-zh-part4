# 代码经典来了

> 原文:[https://hack aday . com/2019/12/11/code-the-classics-is-coming/](https://hackaday.com/2019/12/11/code-the-classics-is-coming/)

我们为今天的年轻人感到遗憾。如果你花几个小时玩一个现代的视频游戏，并决定自己写一个，有一个大工程在等着你。现在的游戏和程序一样都是表演，有电影般的 3D 渲染，复调的声音和音乐轨道，以及详细的故事情节。40 年前不是这样，那时你可以玩 Pong，然后考虑编写自己的版本。树莓派的人必须同意，因为他们正在接受一本名为“[编码经典](https://www.raspberrypi.org/blog/pre-order-code-the-classics-today/)”的书的预订在节目中，他们采访了几款经典街机游戏的设计师，然后展示了你可以自己运行和破解的游戏的 Python 版本。你可以看看他们关于标题的视频，在下面。

代码来自 Raspberry Pi 创始人[Eben Upton]，正如你所料，这些游戏不一定是忠实的复制品，而是受旧街机标准的启发。

这本 224 页的书被列为第一卷，显然，在两卷之间，这本书将处理不同类型的经典游戏，从乒乓球到明智的足球。从视频中，我们认为其他游戏是蜈蚣，青蛙和泡泡。

如果你对复古游戏感兴趣，这应该是一本好书。我们不知道第一卷有多少，第二卷什么时候会出现。

如果你不想玩 Python 写的游戏，你随时可以写 Python 程序让[替你](https://hackaday.com/2012/02/07/web-game-bot-coded-with-python/)玩游戏。如果你最喜欢的编程语言是 solder，你仍然可以[玩 Pong](https://hackaday.com/2019/09/17/building-video-pong-with-discrete-components/) 。
