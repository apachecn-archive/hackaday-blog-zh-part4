# 用三叉神经刺激模拟虚拟现实应用程序中的温度

> 原文:[https://hack aday . com/2022/10/24/simulating-temperature-in-VR-apps-with-triangle-neural-stimulation/](https://hackaday.com/2022/10/24/simulating-temperature-in-vr-apps-with-trigeminal-nerve-stimulation/)

虚拟现实系统一直在变得越来越好，但它们仍然主要是视觉和听觉设备，可能还添加了一点触觉反馈。这仍然使五种典型感觉中的 40%被排除在外，当然，除非[这种刺激三叉神经的虚拟现实配件](https://github.com/humancomputerintegration/trigeminal-based-temperature-illusions)流行起来。

虽然你可能会把这看作是一个简单的“嗅觉视觉”式的嗅觉反馈，但芝加哥大学人机集成实验室的[Jas Brooks]、[Steven Nagels]和[Pedro Lopes]的工作旨在提供一个虚拟现实用户可能在模拟中体验的不同热状态的模拟。没错，添加到现成的 Vive 耳机中确实会使用三个带振动网状雾化器的微流体泵将化学物质吹进佩戴者的鼻子，但这是化学物质的选择及其目标。所用的兴奋剂是无味的，所以它们不是触发鼻子中的嗅球，而是针对三叉神经，三叉神经也支配着鼻子的内层，并引起更多的全身感觉，像辣椒的普遍辣味和薄荷的清凉感。耳机利用这些感觉来改变模拟中的热状态。

下面的视频显示了为该实验开发的自定义模拟。除了辣椒素的热量和桉树脑的冷却作用，该团队还添加了第三个通道，其中含有 8-巯基-p-薄荷烷-3-酮，这是一种有机化合物，旨在模拟游戏中启动的发电机产生的烟雾。该论文详细介绍了可以被刺激的各种受体和所需的不同混合物，完整的构建信息可在 GitHub repo 中找到。我们会饶有兴趣地关注这一次。
