# 通过机器学习产生积极性

> 原文：<https://hackaday.com/2020/08/06/generate-positivity-with-machine-learning/>

随着人们对手势识别和机器学习的理解越来越多，并开始开发在许多不同平台上实现它们的方法，这些天手势识别和机器学习得到了大量的播放时间。当然，这允许更容易地接触到那些能够在严格的学术或商业环境之外使用新工具的人。例如，带着[nate.damen]戴在头上的[手势识别流媒体电视在亚特兰大的街道上滑行](https://hackaday.io/project/173777-gesture-recognition-wearable)。

他被称为[atltvhead]，他佩戴的电视正面有一个功能性的 LED 屏幕。整个场景让我们想起了《深思熟虑》。该屏幕可以显示各种动画，这些动画是通过 Twitch chat 控制的，他可以在镇上流式传输他的旅程。他想给动画增加一点互动，简化他的用户界面，所以他设置了一个手势感应套，可以根据他如何移动手臂来增强动画。他使用手臂传感器中的 Arduino 和背包中的 Raspberry Pi 将所有这些联系在一起，他深入杂草中解释如何使用 Tensorflow 来识别手势。下面链接的视频显示了他使用的机器学习系统的大量训练。

[nate.damen]也没有停留在欢快的电视头上。他还背着一个背包，当他穿着旱冰鞋经过人们身边时，这个背包向他们展示令人振奋的信息，不想漏掉那些没有看到他到来的人。我们认为这是一个非常令人振奋的项目，为了让手势识别机器学习算法正确，所做的大量工作本身就令人印象深刻。不过，如果你是 Tensorflow 的新手，我们已经推荐了一些项目，这些项目可以使用树莓 Pi 和相机进行[可靠的对象识别。](https://hackaday.com/2018/07/31/object-detection-with-tensorflow/)
