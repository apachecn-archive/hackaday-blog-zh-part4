# 学院项目网 360 度视点显示

> 原文：<https://hackaday.com/2019/07/25/college-project-nets-360-degree-pov-display/>

大学毕业设计是多年理论学习最终付诸实践的高潮。对许多学生来说，这是他们第一次体验真正的工程。[Melangeaddict]选择为他的最后一个项目[进行视觉暂留展示，并在此过程中学到了很多东西](https://www.youtube.com/watch?v=O9l8S8WyD6Q&list=PLeMr3aXmTkuoWbZJV1MeUtPTRN7RUbBVq&index=2&t=0s)。

该显示器由一排 48 个 RGB LEDs 组成，安装在一个能够旋转 360 度的臂上，带有一个简单的纸张漫射器。这个手臂是由一个电动马达的皮带驱动旋转起来的，所以靠近这个机器是非常不明智的。由于优质轴承和精心制造，旋转阻力最小。红外 LED 安装在框架上，光由旋转臂上的光电二极管接收，使图像保持固定在空间中，不会随时间漂移。图像可以通过蓝牙接口无线加载到显示器上，这对于 2011 年的 DIY 项目来说是非常先进的。

我们是 POV 显示器 360 度方法的粉丝，通过正确的旋转速度和快速的数据速率，有可能从设备中获得一些非常高的分辨率。小心别把手伸进机器里。

当谈到视点显示时，有一口深井可以探索，从[三维构建](https://hackaday.com/2019/07/20/a-multi-layered-spin-on-persistence-of-vision/)到[振动灵活设置](https://hackaday.com/2019/07/09/flexled-is-a-unique-take-on-persistence-of-vision/)。休息后的视频。
