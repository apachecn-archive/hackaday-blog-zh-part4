# 生物黑客的微流体技术

> 原文：<https://hackaday.com/2021/07/05/microfluidics-for-biohacking-hack-chat/>

加入我们太平洋时间 7 月 7 日星期三中午与 Krishna Sanka 的[微流体生物黑客聊天](https://hackaday.io/event/180466-microfluidics-for-biohacking-hack-chat)！

“微流体学”听起来像是一个怪异而奇妙的领域，但它不会过多地触及常规生活。但是想想看，每次你启动喷墨打印机，你都在让微流体工作，因为纳升大小的墨滴在空间中喷出，准确地撞击你的纸张。

喷墨可能是平凡的，但是它们背后的原理却是不同的。微流体机制已经在各种产品和过程中找到了自己的路，也许最有趣的用途是用来探索和利用生命的微观领域。微流体可以用来重现细胞中发生的一些纳米级生化反应，不仅提供了观察生物世界的新方法，而且常常可以操纵它。微流体设备从可以针对成千上万个目标快速筛选候选药物的“DNA 芯片”到可以快速筛选接触毒素或病原体的临床样本的设备。

微流体技术在生物黑客中有很多应用，Krishna Sanka 正在积极地整合这两个领域。作为一名工程学研究生，他的重点是开源、DIY 微流体，可以帮助生物黑客提升他们的游戏，他会停下来参加黑客聊天，向我们介绍基本知识。带着你关于如何——以及为什么——建造你自己的微流体设备的问题，来看看现代生物黑客是如何学习“随波逐流”的。
