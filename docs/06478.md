# 乐高显微镜做研究

> 原文:[https://hack aday . com/2020/05/05/Lego-microscope-does-research/](https://hackaday.com/2020/05/05/lego-microscope-does-research/)

我们见过很多用乐高建造的实用机器。为什么不呢？砖块便宜又丰富，所以如果它们能完成任务，谁会在乎它们看起来像是孩子的玩具呢？显然，不是[尤克塞尔·特米兹]。他是 IBM 的一名工程师，他的工作包括拍摄微观流体回路的照片。当他对自己价值 10，000 美元的高倍显微镜不满意时，他自己造了一台。[使用乐高](https://github.com/IBM/MicroscoPy)。照片怎么样？好到能出现在很多科学期刊上。

显然，这款显微镜不仅仅包含乐高，但它的价格仍然低于 300 美元。根据来自 [Futurism](https://futurism.com/engineer-published-scientific-papers-lego-microscope) 的采访，目标设备是反射性的，这使得直接拍摄它们变得困难。在对三脚架上的相机进行实验后，[Yuksel]决定他可以建立自己的专用设备。你可以在下面看到一段视频和一些照片。

根据同一个采访，需要几个原型才能做好。第一个原型没有使用乐高，而是 3D 打印的。然而，为了使显微镜更加模块化和可配置，[Yuksel]突袭了玩具箱。

开源显微镜是完全机动化的，模块化的，使用带有 800 万像素摄像头的 Raspberry Pi 来捕捉图像。Arduino 控制步进电机和灯光。下面的第二个视频展示了这种构造，您可以在 IBM 的 GitHub 存储库中找到文档。

我们之前并不是没有见过[定制显微镜](https://hackaday.com/2019/06/14/gigapixel-microscope-reveals-tiny-parts-of-the-big-picture/)的构建。如果你更喜欢 3D 打印，[这可能会让你开始](https://hackaday.com/2019/01/26/3d-printed-microscope-stage-offers-precise-movement/)。
