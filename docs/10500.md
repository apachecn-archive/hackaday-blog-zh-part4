# 螺线管发动机能为汽车提供动力吗？

> 原文:[https://hack aday . com/2021/06/26/can-the-solenoid-engine-power-a-car/](https://hackaday.com/2021/06/26/can-the-solenoid-engine-power-a-car/)

[Emiel]又名[实用工程师]在他功能齐全的商店里做各种有趣的项目，他的一个切线是建造一系列螺线管发动机。这些发动机模仿内燃机的功能，每个螺线管充当一个活塞。然而，[Emiel]的概念引擎的唯一问题是，他从未真正把它们放到汽车上来证明它们的有效性。这个建筑最终证明了他们可以为车辆提供动力。

这个项目从一个新的引擎开始。[Emiel]选择了使用四个螺线管和基于 Arduino 的控制器的 V4 设计。在让它正常运行遇到一些麻烦后，他清理了一个他在 V8 螺线管引擎中构建的小电路板，以帮助计时。安装后，螺线管发出咔哒声，以单一恒定速度旋转曲轴。车辆本身大部分是 3D 打印的，用两根铝管作为支撑结构来安装发动机。甚至车轮都是 3D 打印的，上面涂有特殊的橡胶涂层。组装好一个小型传动系统后，它就要出发去参加这个小型原型车的比赛了。

虽然这款小型汽车没有方向盘，只能以恒定的速度行驶，但这些微型电动引擎实际工作的概念证明是[Emiel]关于这些奇怪引擎的视频收藏的一个受欢迎的补充。当然，它们不如用电动机直接驱动车轮效率高，但我们都知道这没什么意思。如果你没有见过他最复杂的构造，[V8 当然值得一试](https://hackaday.com/2020/03/20/this-v8-makes-a-shocking-amount-of-power/)，并且展示了他为这辆车重新设计的计时电路。
