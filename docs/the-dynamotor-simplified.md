# 发电机简化了

> 原文：<https://hackaday.com/2021/04/11/the-dynamotor-simplified/>

[Robert Murray-Smith]不喜欢将 DC 转换为交流电的逆变器的价格。这使他建造了一台发电机，或有时被称为电动发电机组。这些装置只是 DC 发动机驱动的发电机。当然，马达也可以用作发电机，罗伯特有一堆无刷马达，以电脑风扇的形式出现。[双风扇发电机诞生](https://www.youtube.com/c/RobertMurraySmith/videos)。

无刷电机之所以吸引人，是因为传统上，电刷通常是发电机的故障所在。将作为发电机的风扇需要一些手术，但这很简单。他刮掉了所有的控制电子设备，将电线连接到线圈上，形成了一个三相发电机。这种配置也不需要风扇叶片。如果你使用普通的马达和发电机，使轴同心将是一项重要的任务。有了风扇，只需对准安装孔就很简单，而且你可以免费获得完美的对准。

它是如何工作的？[Robert]有第二个视频显示了示波器上的[输出。下面两个视频你都可以看。发电机产生一个好看的正弦波，可能比大多数价格合理的固态逆变器好得多。他没有提到他能成功提取多少电流，但可能不多。你还需要一个变压器来代替输出线电压的商用逆变器，这样会增加系统损耗。另一方面，如果你想要一个较低电压的交流电，你可能只需要更换所有的变压器，如果你自己制造一个齿轮。](https://youtu.be/W7QRBk85aUw)
