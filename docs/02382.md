# 下雨了全新的准将 64s

> 原文：<https://hackaday.com/2019/03/12/its-raining-brand-new-commodore-64s/>

显然，现在是建造你自己的准将 64 的最佳时机。在一天之内，我们得到了三(3！)分开来自两个不同黑客的 C64 版本。

这是通过一系列完全不同的项目实现的，这些项目分别重建了整个机器的一部分。替代主板存在，像[终极 64](https://ultimate64.com/) 和 [C64 重装 Mk2](https://icomp.de/shop-icomp/en/produkt-details/product/c64-reloaded-mk2.html) 。Pixelwizard 提供了新的案例[。](https://shop.pixelwizard.eu/en/commodore-c64/4/c64c-case-retro-black?c=12)由于 Mechboard 64 项目，甚至可以拥有新的键盘底座[。](https://www.breadbox64.com/blog/the-mechboard64/)

[埃里克·希尔]拿走了所有这些零件，[从零开始建造了他自己的 C64](https://amigalove.com/viewtopic.php?f=13&t=967)。不满足于一个，[他重复了这个过程，建造了另一个](https://amigalove.com/viewtopic.php?f=13&t=968)。

这两台机器作为两种不同主板选项的演示。综上所述，它们展示了有多少老式 Commodore 组件被粉丝群体重新制造:除了键帽和可能的一些硅，两台机器中的所有部件都是新的。

我们刚才是不是说了“键帽”这成了[Perifractic]的得意之作，他发现某些乐高技术件与最初的 Commodore 64 键有相同的十字形槽。经过一番实验，[全套乐高键帽制作完成。](https://www.youtube.com/watch?v=5UX-gqylYgQ) (YouTube，下面嵌入。)这绝不是一套随意拼凑的物品，这套物品可以和印有娱乐图案的瓷砖一起订购。这让你可以用准将工厂生产的零个零件来制造 C64。这证明了世界上最畅销的电脑的受欢迎程度，现在再次有可能用全新的零件来制造一台电脑。

如果你想复制这一壮举，[[perifactic]的网站可以让你轻松订购所需的一切。从几年前在 Kickstarter 上推出的第一个复制案例](https://www.perifractic.com/Perifractic/Build_a_new_C64.html)[到现在，事情已经有了很大的进展](https://hackaday.com/2015/03/11/hackaday-retro-edition-a-new-commodore-64-case/)。
