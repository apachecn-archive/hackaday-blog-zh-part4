# 钉这个棘手的久美子模式

> 原文：<https://hackaday.com/2022/03/11/nail-this-tricky-kumiko-pattern/>

[Pask Makes]之前已经在木头上制作了 Kumiko 图案，并对结果感到满意，但他想知道是否可以用金属而不是木头制作出视觉上相似的东西。

为此，他伸手去拿钉子，因为这是一种廉价的均匀的小金属棒。有趣的是，Kumiko 是一种不用钉子就能连接小块木头的技术。有许多不同的模式使用这种技术，大多数是从大自然中获得灵感。是木材本身的压力将它结合在一起，需要专门的计划和数千分钟的调整。因为[Pask]使用 MIG 焊机将钉子焊接在一起，所以从技术上讲这不是 Kumiko，而是 Kumiko 图案。

第一步是去除指甲上的涂层，这是一种用一点酸就能很好地完成的东西。滴了一点酸后，他的指甲已经准备好了，他准备把它们钉在一起。他在一张纸上打印了一个模板，用一把直尺和一个带凹槽的手掌刨削器在纸上切出一个个小通道，让每个钉子都能嵌入。在一个小夹具的帮助下，钉子被修剪到合适的宽度。在他把钉子钉在一起后，他回来在中间填上。

这是一个简单的小项目，创造了一个美丽的图案，这是一个很好的提醒，简单的材料可以制作复杂的东西。如果你喜欢木头的外观，这把久美子吉他可能更合你的口味。休息后的视频。
