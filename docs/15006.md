# 价值 60 美元的激光器通过新控制器获得成功

> 原文:[https://hack aday . com/2022/10/05/60-使用新控制器进行激光切割/](https://hackaday.com/2022/10/05/60-laser-makes-the-cut-with-new-controller/)

如果你正在阅读 Lightburn 论坛，你可能已经有了某种激光切割机。但是，如果你像我们大多数人一样，你总是会被诱惑去做另一笔“交易”[Dkj4linux]有一个帖子，他在那里买了一台 79 美元的激光雕刻机(我们注意到，现在售价在 59 美元到 65 美元之间)。像大多数便宜的雕刻机一样，这台机器有一个专有的控制器和 Windows 专用软件。毫不奇怪[Dkj4linux]会想用…嗯… Linux。答案？把主板拆下来，换上旧的备用主板。

正如你在下面的视频中看到的，这台机器看起来构造得很好。以这个价格，你可以得到一个 3 瓦的激光头(就光功率而言，这可能要小得多)，以及 220x290mm 毫米的建筑面积。控制器在一个小的金属外壳中，很容易简单地拔下两轴和激光控制电缆。

这将是一个很好的使用旧的 3D 打印机控制器，你已经在你的垃圾箱，因为上次升级。事实证明，如果你需要的话，JL1 板确实有标准的 GRBL 固件[。看起来固件并不完美，但是有解决办法。无论如何，我们可能会扔掉廉价的控制器，使用我们自己的控制器。](https://forum.lightburnsoftware.com/t/jl1-jl3-grbl-firmware-avialable/72113)

老实说，调用这些刀具有点夸张，因为在这个功率水平下，你将很难切割非常厚的东西。但看起来它做了一个可信的雕刻工作，它可能有一些耐心可以切割一些材料。

当然你也可以[自己做框架，加几个步进电机](https://hackaday.com/2022/09/14/mokeylaser-a-diy-laser-engraver-that-you-can-easily-build/)买个激光模块就行了。或者做[更高级的](https://hackaday.com/tag/diy-laser-cutter/)。但是，如果你想在没有大笔投资的情况下尝试激光，这种东西可能正是你想要的。
