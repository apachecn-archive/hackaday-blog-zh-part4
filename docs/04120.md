# 用声致发光捕捉瓶子里的星星

> 原文：<https://hackaday.com/2019/09/06/capture-a-star-in-a-jar-with-sonoluminescence/>

不说别的，[贾斯汀·阿特金]就是执着。否则你如何解释用简单的工具创造声致发光的五年探索？

那么到底什么是声致发光呢？简而言之，顾名思义:由声音引起的光的释放。在[Justin]的案例中，他用一个超声波传感器在一瓶水的共振频率下建立了一个驻波。一滴水被用来夹带一个小气泡，这个小气泡在烧瓶中保持稳定的位置，就像聚苯乙烯泡沫珠在[声波悬浮器](https://hackaday.com/2017/01/05/acoustic-levitation-with-a-twist/)中一样。关掉灯，你会看到泡泡发出幽灵般的蓝光。

是什么导致了发光？好问题。根据[Justin]的说法，我们只是不确定是什么导致了它，尽管主要的理论是气泡的空化导致被捕获的气体剧烈压缩和加热，转变成短暂的等离子体。但是这个理论存在问题，这也是他想展示这个过程有多简单的原因之一——现在他已经用五年的努力解决了这个问题。安装传感器和正确调整驱动电路并不容易，但只需一个信号发生器、一个音频放大器和一卷磁线，你也可以制作自己的“瓶中之星”

我们赞赏[Justin]成功完成这个项目的决心。这与他坚持不懈地努力使[成为冷等离子火炬](https://hackaday.com/2019/04/28/cold-plasma-torch-produces-a-cleansing-flame-that-never-consumes/)，甚至[成为他的桌面射电望远镜](https://hackaday.com/2018/06/20/desktop-radio-telescope-images-the-wifi-universe/)没什么不同。
