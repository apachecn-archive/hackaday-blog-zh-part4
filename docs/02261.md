# 从软件到 tindie hack 与 brian laugh 聊天

> 原文:[https://hack aday . com/2019/03/04/from-software-to-tindie-hack-chat-with-Brian-lough/](https://hackaday.com/2019/03/04/from-software-to-tindie-hack-chat-with-brian-lough/)

加入我们太平洋时间周三中午的[从软件到 Tindie Hack Chat](https://hackaday.io/event/164058-from-software-to-tindie-hack-chat) ！

Brian Lough 走了一条迂回但可能并不罕见的路线来到硬件黑客领域。Brian 接受过电子和计算机工程教育，是一名软件开发人员，当 ESP8266 上市时，他迷上了 Arduino 开发。他意识到像这样的微控制器以低廉的价格提供了令人难以置信的功能，于是这个错误咬了他一口。

从那以后，Brian 完全接受了硬件黑客的方式，甚至以一种与他的观众合作的“黑客方式”来直播完整的构建。他还把自己的一些作品变成了合法产品，在他的 Tindie 商店出售，甚至在发货前在 T2 自动测试以发现错误并提高质量。

请[加入我们的黑客聊天](https://hackaday.io/messages/room/2369)，我们将讨论:

*   软件黑客如何导致硬件黑客；
*   创作过程以及直播如何帮助或阻碍它；
*   从项目到产品的含义；和
*   我们很快会看到什么样的新项目？

当然，我们鼓励你在讨论中提出自己的问题。你可以在从软件到 Tindie Hack Chat 的[上发表评论，我们会把它放在 Hack Chat 讨论的队列中。](https://hackaday.io/event/164058-from-software-to-tindie-hack-chat)
