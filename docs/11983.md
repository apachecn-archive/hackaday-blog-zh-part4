# 电动单轮车

> 原文:[https://hackaday.com/2021/11/24/e-monowheel/](https://hackaday.com/2021/11/24/e-monowheel/)

总的来说，独轮车是一种古怪的，疯狂的未来交通工具，我们把它和气垫自行车和喷气背包归为一类:奇怪，目前不切实际，但非常酷。山姆·巴克不满足于等待遥远的未来，他制作了自己的电动独轮车。(视频，下面嵌入。)

任何单轮车最难的部分是外轮缘需要承受巨大的磨损。它支撑车辆的重量，并提供大部分的结构完整性，但也是推进的手段。[Sam]首先想到的是使用蹦床框架，因为它是一种圆形且相当坚固的钢管。他 3D 打印了连接副框架和蹦床框架的滚轮。扁钢用于制作副框架内的角钢，直钢管将内框架连接成梯形。蹦床框架是焊接在一起的，在第一次旋转测试中，它因应力而断裂。它只是不够坚固。

不听劝阻，他找到了一家将钢材弯曲成定制形状的公司。他从他之前改装的另一辆自行车上偷了电动自行车套件，轮子正在转动。一些手柄和脚踏板之后，是时候进行正式的试驾了。总的来说，结果相当令人印象深刻，当骑着车在镇上的街道上行驶时,(山姆)的双摄非常棒。如果你正在寻找搔单轮痒处，看看这个木制的单轮。
