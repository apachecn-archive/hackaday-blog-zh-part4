# 这台塞特斯打印机是静音运行的

> 原文:[https://hack aday . com/2018/07/26/this-cetus-printer-is-rigged-for-silent-running/](https://hackaday.com/2018/07/26/this-cetus-printer-is-rigged-for-silent-running/)

入门级 3D 打印机市场非常丰富，许多供应商提供的产品都非常好。但是没有什么是完美的，为了达到 200 美元的价位，一些妥协是不可避免的。当然，这并不意味着你必须接受这些工程选择，这使得这些廉价打印机成为售后改装的一个很好的出发点。

[Linas K]走的是这条路，在这个过程中[让他的塞特斯 3D 打印机基本上是无声的](https://www.youtube.com/watch?v=uWTyraokJP8)。下面视频的第一部分回顾了 stock machine 的缺点和[Linas]所做的机械改变，包括 Z 轴滑轨的新支架，将 WiFi 天线重新定位到合理的地方，以及为每个滑轨添加限位开关。在机箱内，电子设备得到了彻底的返工，定制的 PCB 容纳 Trinamic 步进驱动器，实现超静音操作。新板还支持限位开关以及挤出机风扇和平台加热器垫的恒温控制。作为奖励，新的 PCB 比原来的小得多，留出了将电源塞进外壳的空间，这是一个很好的触摸。这并不便宜，这意味着基本上掏空了打印机，但结果令人印象深刻的安静。

如果[Linas]发布 BOM 和 PCB 设计，我们很想在我们自己的塞特斯上尝试这些静音模块(提示，提示)。塞特斯黑客在这里越来越流行了。从三位塞特斯专家到[将一台塞特斯变成一台 PCB 机](https://hackaday.com/2017/11/27/entry-level-3d-printer-becomes-budget-pcb-machine/)，这个小打印机可以提供很多东西。
