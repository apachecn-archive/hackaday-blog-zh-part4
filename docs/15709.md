# 氮化镓革命

> 原文:[https://hack aday . com/2022/12/20/the-氮化镓-revolution/](https://hackaday.com/2022/12/20/the-gallium-nitride-revolution/)

[Asianometry]一直在学习氮化镓半导体，并在下面的视频中分享他所知道的信息。这种半导体材料比更常见的硅具有更高的带隙电压。这使得它适用于需要更高效率和更少热量的应用。

这种材料最初用于发光二极管，但我们看到这种材料越来越多地用于充电器等高功率应用。使用这种技术的手机充电器尤其常见。当你想到全世界每天需要多少手机充电器时，这并不奇怪。

其他需要节能设备的地方包括数据中心、电动汽车和电池供电设备。不过，如果这种材料变得非常受欢迎，我们是否能制造出足够的材料来满足全球需求还不清楚。这一点尤其正确，因为用于制造硅器件的机械和工艺不适用于氮化镓。碳化硅是一个竞争对手，它可能更容易制造，尽管它不如氮化镓有效。

在之前，我们已经[研究过氮化镓，我们确信我们将再次看到它。碳化硅也许有一天会在金星表面运行。你甚至可以用它来制作](https://hackaday.com/2019/05/14/the-amazing-new-world-of-gallium-nitride/)[自制 led](https://hackaday.com/2012/06/05/homemade-silicon-carbide-led/)。
