# 蜘蛛侠的摇摆更接近现实

> 原文:[https://hack aday . com/2021/10/26/蜘蛛侠-荡秋千-更接近现实一点/](https://hackaday.com/2021/10/26/spider-man-swings-a-little-closer-to-reality/)

尽管每天阅读 Hackaday，看到人们做的不可思议的事情，但偶尔还是会有一些事情让你大吃一惊。有时这只是技术上令人印象深刻，但通常只是，“我不认为有人会尝试这个，甚至不会想到这个。”[Joel Creates]就是这样一个例子，他的[蜘蛛侠腕戴式网络摄像机](https://www.youtube.com/watch?v=HaP_OeRuNew)。

此前，[Joel]已经围绕一个像背包一样佩戴的加压热胶罐建立了一个网络投掷系统。它在小型化方面的不足，在功能性方面得到了弥补。然而，Build IRL 的 JT]创造了一个基于抓钩的蜘蛛侠系统，可以发射绳索，这让[Joel]想到，也许可以将热熔胶和抓钩系统结合起来，形成一个更小的整体包装。

他的解决方案很简单。装满胶水的旧二氧化碳筒和一个钻在里面的小喷嘴被装入一个快速连接装置。在装载之前，热熔胶通过小工具带上的感应线圈加热。罐子上的隔热涂料层和微真空球有助于[Joel]将其放入手腕射击器中，而不会烧伤自己。一个带有杠杆启动系统的自行车轮胎充气机构成了射手的主要组件。使用压缩空气，该系统在一个表面发射一团热胶，一个带孔的金属网状圆盘和一根绳子连接到热胶上。随着胶水迅速冷却，金属盘提供了大量的表面供粘合剂附着。总的来说，结果令人印象深刻，但工程挑战是一个令人兴奋的旅程。从失败的原型到失败的电源似乎都发生在这个版本上。

再加上一些电磁铁，你真的可以拥有整个蜘蛛包。
