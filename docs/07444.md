# RetroPlug 将 Gameboy 模拟器与您的 DAW 同步，以应对 Chiptunes 过载

> 原文:[https://hack aday . com/2020/08/12/retro plug-syncs-game boy-emulators-with-your-Daw-for-chip tunes-overload/](https://hackaday.com/2020/08/12/retroplug-syncs-gameboy-emulators-with-your-daw-for-chiptunes-overload/)

Gameboy 是 chiptune 领域最大的平台之一。虽然可以用一个手持设备来播放一场演出，但许多艺术家选择使用两个甚至更多的手持设备来增强他们的声音并震撼观众。为了简化为这种设置创作歌曲的工作流程， [[tommitytom]创建了 Retroplug](https://github.com/tommitytom/RetroPlug) ，休息之后，您可以在演示视频中看到他演示这些功能。

Retroplug 是一个用于 Gameboy 模拟器的 VST 包装器。这使得在像 Ableton 或 Fruityloops 这样的数字音频软件中运行多个仿真 Gameboy 实例成为可能。而不是必须变戏法多个 30 岁的游戏机和相关的电池和连接线，相反，这一切都可以在一个托管 VST 窗口。

目前，该软件仅适用于 64 位 Windows 和 VST2，但是源代码可供那些渴望一窥究竟的人使用。它完全实现了对 mGB 的 MIDI 支持，与 LSDJ 和 Arduinoboy 设置配合良好。也为每个仿真实例创建了文件，所以当你完成作曲时，你可以在外出表演时将你的歌曲放到真正的硬件上！
