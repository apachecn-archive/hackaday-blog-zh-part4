# 黑客日播客 110:一个统一码来统治他们，黑客焦点堆叠，虚拟打字，僵尸天气频道

> 原文：<https://hackaday.com/2021/03/19/hackaday-podcast-110-one-unicode-to-rule-them-hacking-focus-stacking-virtual-typing-and-zombie-weather-channel/>

黑客日编辑埃利奥特·威廉姆斯和迈克·斯兹西斯报道了硬件黑客的伟大一周。我们看到了一次故障注入攻击，它使用电动苍蝇拍和手绕线圈来旋转 AVR micro 内部的位。当使用显微镜对电路板成像时，焦点叠加是你想要的，并且有一个针对 Eakins 相机的黑客使其自动化。在我们的“不能错过的文章”中，我们反复讲述了如何在气候变暖的情况下给城市降温，然后带着安静的钦佩注视着 Unicode 标准所取得的成就。但是，当谈到令人头晕的黑客时，你不能击败逆向工程的努力，这是机架安装箱展示的，使天气频道在 80 年代和 90 年代令人敬畏。
