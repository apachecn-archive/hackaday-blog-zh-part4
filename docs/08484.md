# 从废物中回收金属

> 原文：<https://hackaday.com/2020/11/30/recovering-metal-from-waste/>

提炼贵金属不是打磨从地下挖出来的石头那么简单。通常，需要复杂的化学过程来适当地或大量地处理这些材料，但是这些过程会留下相当多的废物。通常，这些废物中会留下有价值的金属，而[NerdRage]已经收集了他的化学设备来展示如何回收这些金属。

该工艺旨在从硝酸铜中回收铜和硝酸，硝酸铜是金属加工过程中常见的废弃副产品。虽然存在一种称为[热分解](https://en.wikipedia.org/wiki/Thermal_decomposition)的过程来实现这一点，但它并不是特别有效，所以这种替代方法看起来可以提高产量。第一步是硝酸铜与硫酸反应，生成硝酸和硫酸铜。从那里，硫酸铜被放置在电解槽中，使用铂阴极和铜阳极使电流通过。该过程完成后，所有的铜将沉积在铜电极上。

除了可回收的铜量之外，这一过程的另一个有趣之处在于硫酸和硝酸是可回收的，并且能够在其他过程中再次使用。这个过程比热分解效率高得多，也不涉及任何有毒气体。当然，如果你喜欢从废物中收集有价值的金属，你也可以看看[回收一些黄金](https://hackaday.com/2013/11/18/salvaging-gold-from-old-electronics/)。
