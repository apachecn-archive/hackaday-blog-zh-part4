# DIY 宝贝麻省理工学院猎豹机器人

> 原文:[https://hackaday.com/2020/07/05/diy-baby-mit-cheetah-robot/](https://hackaday.com/2020/07/05/diy-baby-mit-cheetah-robot/)

如今，3D 打印机已经成为大多数制造商的主要产品，使黑客能够快速生产简单的机械原型，而不需要专门的机械车间。我们在 Hackaday 上看到了许多有创意的 3D 设计，麻省理工学院的小猎豹机器人也不例外。你肯定看过[麻省理工学院的猎豹机器人](https://hackaday.com/2014/09/15/mits-robotic-cheetah-is-getting-even-scarier/)。嗯，jegatheesan 的黑客对猎豹机器人进行了个人旋转，他的结果非常酷。

机器人的身体是 3D 打印的，这使得定制设计和更换损坏的零件变得很容易。腿被设计成一个五连杆机构，两个伺服电机控制四条腿中的每一条。额外的伺服电机用于旋转 HC-SR04，这是一种流行的超声波距离传感器，用于自主模式的避障机制。该机器人还可以通过蓝牙使用麻省理工学院应用程序 Inventor 开发的应用程序[jegatheesan]进行控制。

总的来说，机制可能需要一点工作——(jegatheesan 的)小猎豹可能不会很快超过麻省理工学院的机器人——但这是一个很酷的黑客，我们期待着第 3 版。也许猎豹会成为一个很酷的同伴机器人？
