# 特雷门琴有了声音

> 原文：<https://hackaday.com/2019/07/09/the-theremin-gets-a-voice/>

每隔一段时间，我们都会遇到一个项目，它在现有的设计上添加了一个可笑的好创意。这正是肖骁和法国达朗贝尔研究所 LAM 研究小组所做的。他们的项目 T-VOKS 是一种唱歌和说话的特雷门琴，肯定会让办公室里的每个人都疯狂。(下面嵌入了 YouTube 链接，供您查看。)

对于外行来说，特雷门琴是一种不需要身体接触的电子乐器。相反，它使用两个天线来感应操作者手的距离，并利用这一点来调节输出音频的音高和音量。从音乐会到电影背景音乐，甚至是吓唬邻居，这种乐器都可以。

T-VOKS 是一种不同的乐器，它与语音合成器接口来唱歌。还有一个额外的传感器用于音节排序，下面的视频显示了这个小工具的运行。锦上添花的是演奏的乐器，或者应该是在真正的音乐会上演唱。如果你需要细节，还有一篇研究论文详细介绍了 Dropbox[ [PDF](https://www.dropbox.com/s/h4bn1qbpcjjab4b/NIME2019_Paper.pdf?dl=0) ]的操作。

我们想知道 TTS 引擎将如何与这个想法一起工作，并希望在未来看到更多类似的项目。在那些想要开始的人之前，看看 DIY theremin 的[构建指南。](https://hackaday.com/2017/09/09/theremin-in-detail/)
