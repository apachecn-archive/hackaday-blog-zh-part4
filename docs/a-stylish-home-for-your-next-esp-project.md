# 为您的下一个 ESP 项目打造一个时尚的家

> 原文：<https://hackaday.com/2019/09/20/a-stylish-home-for-your-next-esp-project/>

ESP8266 和 ESP32 正迅速成为各种应用的首选微控制器。但是我们最近看到很多活动的一个特殊领域是家庭自动化；这些板使你的项目在线变得如此容易和便宜，以至于现在组装你自己的自动化系统比以往任何时候都更有吸引力。利用这一趋势，[hwhardsoft]一直在研究一种非常适合安装在客厅墙壁上的 [ESP 外壳](https://hackaday.io/project/162060-top-wall-mount-touchscreen-for-esp8266)。

当然，这个项目不仅仅是一个公认的非常漂亮的塑料盒。该系统还包括一个 ILI9341 2.4 英寸触摸屏 LCD、一个集成电压调节器，甚至还有一个“perfboard”部分，让您可以轻松连接特设电路和传感器。你甚至不需要切换到裸模块，因为 PCB 的设计是为了适应常见的开发板，如 Wemos D1 迷你和 NodeMCU。

尽管它的外表，这个项目是非常友好的初学者。利用通孔组件，拧紧终端，和一个完美的标签丝网印刷，你不需要成为一个硬件专家来生产一个非常光滑的小工具，全家人都可以欣赏。

[很像我们几个月前报道的 HestiaPi 项目](https://hackaday.com/2019/06/11/hestiapi-a-stylish-open-hardware-thermostat/)，这个项目采用了一个便宜且容易获得的开发板，并将其转变为具有商业产品所有特征的东西。这些项目提醒人们[随着时间的推移，建造和购买之间的界限变得越来越模糊。](https://hackaday.com/2019/09/15/scratch-built-media-player-channels-1980s-design/)
