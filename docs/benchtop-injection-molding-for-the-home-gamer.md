# 面向家庭游戏玩家的台式注塑成型

> 原文：<https://hackaday.com/2020/10/16/benchtop-injection-molding-for-the-home-gamer/>

当我们想到注塑成型时，首先想到的是高度自动化的生产线，每小时生产成千上万个零件。然而，正如“工匠”所展示的那样，同样的技术也可以缩小到 DIYer 可以达到的水平。

使用一个紧凑的手动注射成型机，[克拉夫斯曼]展示了小规模注射成型背后的基本技术。正在讨论的 PIM-Shooter 150 a 型设计用于处理低熔点塑料，如聚丙烯和低密度聚乙烯，并且可以使用比工业中使用的典型钢模具制造成本低得多的铝模具。

然而，真正改变游戏规则的是当这个工匠拿出他的硅胶模具制作技术，并应用到注射成型中。通过用硅树脂制作模具，它们可以更便宜、更容易地制造出来，而不需要重型数控机器来制造所需的几何形状。只要对细节给予正确的关注，就有可能获得好的结果，而根本不需要投资定制铝模具。

注射成型是一种可以实现 3D 打印和其他技术无法实现的事情的工艺；它甚至可以用来生产可存活的晶状体。休息后的视频。
