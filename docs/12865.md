# 级联故障、计算机问题和欧姆定律:理解 2003 年东北大停电

> 原文：<https://hackaday.com/2022/02/25/cascade-failures-computer-problems-and-ohms-law-understanding-the-northeast-blackout-of-2003/>

我们都经历过某种形式的停电，无论是断路器在不方便的时候因暴风雨而跳闸，导致树木和电线之间缺乏分隔。这种影响通常是局部的，很少会有生命损失，尽管它可能发生。但是在下面的视频中，实际工程的 Grady 分解了 2003 年的东北大停电，这是北美有史以来最大的停电。在某些情况下，停电几天，近 100 人死于停电。

[Grady]详细介绍了监控系统、软件模拟以及运营大规模电网的应急计划。该视频解释了感性负载如何引起电抗，以及这种效应如何加剧了一个已经很复杂的问题。不知道什么是感性负载和电抗？没关系，视频很好地解释了这一点，它为理解交流电子学，甚至是围绕电感、电容和电抗的射频电子学理论提供了一个极好的基础。

那么，是什么导致了真正的停电呢？复杂的级联故障是一步一步解释的，视频当然值得一看，即使你已经熟悉这个事件。

提起 2003 年的停电而不谈论一年前的[德克萨斯 ERCOT 停电是不负责任的行为](https://hackaday.com/2021/02/16/trouble-with-the-texas-power-grid/)——这篇文章的评论部分几乎引起了 Hackaday 数据中心的停电！
