# [本·克拉斯诺]用电钻很小的孔

> 原文：<https://hackaday.com/2019/03/14/ben-krasnow-drills-really-small-holes-with-electricity/>

钻孔很容易；近 40，000 年来，人类一直在以这样或那样的形式这样做。在坚硬的材料上钻非常小的孔更具挑战性，但仍然可行。在坚硬的材料上钻又深又直的孔完全是另一回事。

幸运的是，现在我们有了放电加工(EDM ),这种技术带来了各种可能性。同样幸运的是， [[Ben Krasnow]得到了一些 EDM 设备来试用](https://www.youtube.com/watch?v=rpHYBz7ToII)，结果令人着迷。正如[Ben]解释的那样，EDM 的核心就是利用一个小电弧从表面烧蚀金属。电弧通过电弧控制器精确控制其频率，通过 CNC 运动控制精确控制其位置。电弧控制器一直是家用电火花加工的症结所在，但[本]试用的这款 [BaxEDM BX17](https://baxedm.com/product/bx17-arc-generator/) 正是瞄准了小型商店市场。[Ben]建造的整个测试平台具有明显的自制外观，一个数控龙门装配在一个水箱上，一个 EDM 钻头缓慢旋转钻杆，一个无空气喷枪提供高压工艺流体。下面的视频显示，它仍然工作得非常好。

虽然我们当然渴望看到[本]承诺的电火花铣削和切割视频，但我们怀疑我们会排队支付€2950 英镑购买他使用的电弧控制器。如果你的勇气大于金钱，这台电源供电的 EDM 可能更适合你。
