# 钢铁营史诗机甲驾驶舱建造

> 原文:[https://hack aday . com/2019/03/02/an-epic-mech-cockpit-build-for-steel-campaign/](https://hackaday.com/2019/03/02/an-epic-mech-cockpit-build-for-steel-battalion/)

钢铁军团于 2002 年在 Xbox 上发布，至今仍是最铁杆的机甲模拟器之一。它因其覆盖着按钮的巨大双棒控制器而成为传奇，如果你没有及时弹出，它会删除你保存的游戏。它将巨型机器人游戏带到了一个新的水平，但从根本上说，你仍然是在家里的电视机前玩游戏。事情在 2015 年变得真正严重，随着[大型钢铁营盒子的完成——你梦想中的战斗机甲驾驶舱。](http://www.bsbb-mkii.com/)

![](../Images/c25b77e6cced6d4f667e6e7e7dd79f08.png)

Coaching the player is a key part of BSBB gameplay, with a manual created specifically for this purpose.

如果你认为这只是一台放在黑屋子里贴着贴纸的电视机，那你就大错特错了。Imgur 线程涵盖了构建过程，这是一次很棒的经历。事情始于一个定制橱柜的建造，它的尺寸有意制造出幽闭恐惧症。有一个带 4 点式安全带的旋转座椅，还有一个将玩家密封在里面的舱口。在对盒子进行初步测试以确定它有多暗的过程中，一名制作者被困在里面，不得不呼救。这应该突出了构建的严肃性。

控制器被修改并连接到定制的电子设备上，以增加逼真的效果。被击中？由于底座上的马达和低音炮，您可以感受到座椅的隆隆声。机甲彻底损坏了？整个驾驶舱沐浴在闪烁的红光中。甚至还有烟雾效果，让战斗中的事情更加紧张。

整个系统与外界相连，教练可以通过 Xbox 和几个内部摄像头的视频观看内部的活动。提供了一个基本的手册来帮助教练让玩家在战斗的第一时间保持活力。这是一个定制的对讲机设置，使用剩余的中国航空耳机建造的。甚至还有一部红色电话，给人一种真正的军人的感觉。

它涵盖了你能想到的所有细节。如果你热衷于尝试，它将被永久租借给加州奥克兰的艺术和数字娱乐博物馆。它唤起了人们对[的记忆，一个为扮演阿耳忒弥斯而建造的相似建筑。](https://hackaday.com/2018/09/03/unleash-your-inner-starship-captain-with-this-immersive-simulator-console/)休息后的视频。
