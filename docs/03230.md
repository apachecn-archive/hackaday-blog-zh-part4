# 阿都博伊得到了一个古怪的模型

> 原文:[https://hack aday . com/2019/07/02/the-arduboy-gets-a-crank-mod/](https://hackaday.com/2019/07/02/the-arduboy-gets-a-crank-mod/)

你已经看到了虚拟现实耳机和可穿戴视频游戏控制器和飞行轭以及所有其他类型和种类的视频游戏控制器，但曲柄？是的，[ardu boy 现在有了一个古怪的改造](https://community.arduboy.com/t/arduboy-crank-mod/7785)以纪念(或公然抄袭)由恐慌和青少年工程创造的视频游戏控制台 [PlayDate](https://play.date/) 。

这个构建的基础是 Arduboy，一个信用卡大小的微型游戏系统。这款游戏主机具有糖果般的按钮，与 Arduino IDE 兼容，并且有一个已经制作了几十款游戏的社区。哪里有软件开发人员，哪里就不可避免地有一些硬件工程师在等着，这也不例外。[bateske]为 Arduboy 创建了一个曲柄模型，让这个微型的、类似玩具的游戏主机有了一个曲柄。准备写一个鲈鱼钓鱼模拟器？这是你的机会。

该构建的硬件包括用于设备内部的 360°旋转编码器。对于手柄，[bateske]在亚马逊上找到了一个有趣的[“草药和香料的高级研磨机”。令人震惊的是，这个手摇曲柄](https://www.amazon.com/gp/product/B018K6A4TC)[只是和旋转编码器](https://twitter.com/bateskecom/status/1133547164725141505/photo/1)一起工作。

至于游戏，这是 Arduboy 游戏机的全新用户界面，所以当然有一些有趣的可能性。有一种钓鱼模拟器比真正的钓鱼更有趣，还有一种类似于 Flappy Bird 的东西，只是它不是在拍打翅膀，而是在无底的深渊上跳跃。你可以看看下面这个曲柄控制台。
