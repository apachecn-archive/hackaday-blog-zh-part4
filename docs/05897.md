# DARPA 挑战自主机器人团队导航未完工的核电站

> 原文:[https://hack aday . com/2020/03/06/DARPA-challenge-autonomous-robot-teams-to-navigate-unfinished-nuclear-power-plant/](https://hackaday.com/2020/03/06/darpa-challenge-autonomous-robot-teams-to-navigate-unfinished-nuclear-power-plant/)

机器人可能会在地面上找到立足之地，但今天的自主机器人很难在地下工作。DARPA 希望推动技术的发展，所以他们正在进行一项地下(SubT)挑战，刚刚结束了最新一轮。IEEE Spectrum 发表了对本次城市赛道比赛(以及部分参赛队伍)[的精彩回顾。这是三个地下问题子领域中的第二个，分别在六个月前呈现给参与者，让他们为结合三种类型的最终事件做准备。](https://spectrum.ieee.org/automaton/robotics/robotics-hardware/late-nights-cool-hacks-and-more-stories-from-the-darpa-subt-urban-circuit)

如果你[错过了直播](https://hackaday.com/2020/02/20/darpa-subterranean-challenge-urban-circuit-now-livestreaming/)或者喜欢剪辑过的精彩视频，它们都是 [DARPAtv 地下挑战播放列表](https://www.youtube.com/playlist?list=PL6wMum5UsYvYpbhQALOcbhzXYTt3qnzqA)的一部分。今天，它从城市赛道集锦开始，继续到其他视频。包括团队简介，比赛过程的视频演练，实际的比赛镜头，编辑的回顾视频，以及颁奖仪式。播放列表中有一半是六个月前隧道赛道的视频，所以我们可以比较一下各队的表现以及[他们在这一路上学到了什么](https://www.youtube.com/watch?v=KIabWbqv2K4)。在刚刚完成的城市赛道上，我们学到了更多的经验，车队将在接下来的六个月里改进他们的机器人。到那时，我们将举行洞穴巡回赛，参赛队伍准备学习在地下操作机器人的新知识。

系统轨道(真实世界环境中的真实机器人)竞赛在华盛顿州埃尔马一座尚未完工的核电站举行。它不那么原始的混凝土和金属环境代表了复杂的城市地下基础设施。如果这个设施看起来很熟悉，那可能是因为它也偶尔被娱乐业租用来拍摄。这场 SubT 竞赛可能是几个月前 DARPA 发出[一些听起来不祥的推文](https://www.engadget.com/2019/08/28/darpa-wants-abandoned-tunnels/)的无辜原因。正如该请求中所概述的那样，这两个竞赛课程挑战机器人在不规则形状的区域中导航，从幽闭恐怖的狭小空间到礼堂大小的空间，跨越多个楼层。

所有这些都只有昏暗的灯光(如果有的话)，所以使用相机的机器人需要自带明亮的灯光。为了解决地形多变的挑战，许多团队派出了不同的机器人小组，每个小组都有自己的专长。为了让这些机器人进行协调，它们还必须自带网络基础设施。我们看到几个机器人设置网状网络节点，这些节点也可以作为定位信标，就像汉瑟和葛丽特的面包屑一样。

阅读 IEEE Spectrum 的故事，了解更多关于最后时刻黑客攻击的故事。看看那个从机器人轮子上砍下大块来帮助它爬楼梯的团队，那个冲出去买了一台笔记本电脑来替换一个被烧坏的机器人大脑的团队，还有那个把一名成员送上跨太平洋航班作为关键替换零件的夜间快递员的团队。
