# 环氧树脂夜灯是一个令人惊叹的海洋主题建筑

> 原文：<https://hackaday.com/2022/02/12/epoxy-resin-night-light-is-an-amazing-ocean-themed-build/>

我们都见过那些“河”表，其中一棵可爱的老树填充了一些蓝色树脂，以创造一种水一样的美感。来自[smartyleowl]的这个项目采用了这一基本理念，但又进一步发展，结果是一个美丽的建筑，既是一个简单的灯，也是一个立体布景。

首先，选择一块合适的未准备好的粗糙木头来为水下场景制作一个悬崖。紫外线活性蓝色粉末的斑点散布在木头上，一些小塑料珊瑚和海洋植物也被粘住。然后用丙烯酸树脂在木头周围制作一个模子。小鲸鱼和潜水员雕像悬挂在原地，蓝色树脂注入完成水下场景。一旦树脂硬化，它就会被抛光成清晰的光泽，边缘被很好地斜切。然后，它被放置在一个照明底座上，从下面照亮场景，给它一个有点缥缈的水下质量。

无论如何，这不是一个复杂的项目，但它是一个很好的例子，展示了人们可以通过创造性地应用彩色树脂来创造美丽的东西。当然，生产这么好看的灯显然需要一些技巧——获得无气泡的树脂浇注和漂亮闪亮的木材表面并不容易。然而，没有理由你不能从今天开始学习！休息后的视频。
