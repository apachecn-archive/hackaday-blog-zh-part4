# 混凝土扬声器既美观又实用

> 原文：<https://hackaday.com/2019/05/17/concrete-speakers-are-attractive-and-functional/>

在很多领域——赛车运动、太空探索、可穿戴设备——越轻越好。但事实并非总是如此。当你想阻尼振动，阻止东西移动，给东西一个很好的重量感时，更重是最好的方法。像机床、铁砧和扬声器就是这种情况。利用这一理念，[SoundBlab]建造了一套混凝土扬声器。 (Youtube 链接，嵌入下方)

混凝土扬声器外壳的尺寸适用于 3”驱动器，使用两个测量罐作为模具进行铸造。由于所用塑料的表面，这使得最终产品具有光滑和有光泽的表面。混凝土在浇注过程中也要搅拌，以尽量减少混合物中的气泡。

铸造完成后，外壳配有胶合板端盖，用于安装 Fountek FE85 扬声器驱动器。这些是全范围驱动器，这意味着不需要分频器或其他驱动器。然后，扬声器被安装在由木质边饰构成的支架上，这些边饰被染成对比色，具有很好的美感。毡垫放置在底座上，外壳内部填充了聚四氟乙烯，以进一步减少任何不必要的振动。

[声音测试证实扬声器性能良好，](https://www.youtube.com/watch?v=BuQK82Fwb1U)作为休息室音频系统的一部分，它们看起来非常棒。我们认为它们是一对出色的书架式扬声器，非常适合中等音量的舒适聆听。
