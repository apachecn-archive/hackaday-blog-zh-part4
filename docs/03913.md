# 通过坏掉的电视获得巨大的人造日光

> 原文：<https://hackaday.com/2019/08/17/great-artificial-daylight-via-broken-tvs/>

[DIY Perks]长期以来一直热衷于精确模拟真实日光的灯光。他经常为自己的项目选择高质量的 led，但最近他采取了不同的策略——[用坏电视制作吸引人的家庭照明解决方案。](https://www.youtube.com/watch?v=8JrqH2oOTK4)

黑客攻击包括移除损坏的电视或显示器的背光。这些内部有一个强大的白光，但真正的关键是它们还具有菲涅耳透镜。这有助于背光看起来非常类似于真实的天光，因为它将光线散射到房间的周围。

由于驱动大多数 LED 和 CCFL 背光的困难，该项目将原来的照明设备剥离出来，代之以一套在易贝很容易获得的高 CRI LED 灯。与大多数背光相比，它们很容易从 12 伏驱动，并产生更接近实际日光的白光。led 就位后，显示器原有的漫射器和菲涅尔透镜放回原位，用铝框将灯光装饰一新。

安装在有角度的天花板上，这种灯看起来确实像是真正的阳光在雨天透过窗户照射进来。这是一个令人愉快的效果，在照亮房间方面做得很好，我们怀疑它也非常适合一般的视频工作。毕竟，对于一个好的灯光工作室来说，DIY Perks 并不陌生。休息后的视频。
