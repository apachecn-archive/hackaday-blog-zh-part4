# 识别夜间鸣叫的生物

> 原文：<https://hackaday.com/2020/09/08/identifying-creatures-that-go-chirp-in-the-night/>

众所周知，蝙蝠利用回声定位来导航和寻找猎物，但你知道吗，不同种类的蝙蝠制造的超声波芯片非常不同，可以用来识别身份。[Tegwyn☠Twmffat]做到了，这就是为什么他想出了这个令人印象深刻的设备[,能够对夜间飞行的不同蝙蝠进行分类。](https://hackaday.io/project/169854-intelligent-wildlife-species-detector)

现在这可能看起来像是一个奇怪的小玩意，但如果你从事野生动物保护工作，不难想象这种功能会有多么有用。这种装置可以用来很容易地估计特定地区蝙蝠种群的数量和多样性。[Tegwyn☠Twmffat]还提到，至少在理论上，核心概念应该适用于其他类型的嘈杂动物，如啮齿动物或海豚。

该设备由 NVIDIA Jetson Nano 提供动力，通过高端超声波麦克风倾听蝙蝠的叫声。然后这些数据被软件处理，并与 Tegwyn☠Twmffat 个人在当地自然保护区收集的样本数据库进行比较。在休息后的视频中，你还可以看到他如何使用一组叮当作响的房子钥匙作为控制，以确保系统正常运行。

作为 4 月份 Train All Things 竞赛的获胜者，我们迫切希望看到智能野生动物物种探测器在 2020 年 Hackaday 奖竞赛中的表现。
