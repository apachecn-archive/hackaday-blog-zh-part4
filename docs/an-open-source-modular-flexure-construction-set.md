# 一个开源的模块化挠曲结构集

> 原文：<https://hackaday.com/2022/11/22/an-open-source-modular-flexure-construction-set/>

弯曲是一种看起来无害的机制，几乎在任何一种消费设备中都可以找到。提供具有小位移的受限运动，完成受控张力，它们的设计可能相当棘手。GrabCAD 设计师[Vyacheslav Popov]来自乌克兰，由于那里目前的局势，出售一系列弯曲积木的计划变得很困难。最终，[维亚切斯拉夫]决定慷慨地 [发布他的作品开源](https://grabcad.com/sl_st-1/models) ，供大家欣赏。这个集合非常广泛，看起来它可以解决各种各样的弯曲设计问题。(链接前三集:[set 00](https://grabcad.com/library/sl_st-modular-flexure-0-00-1)[set 01](https://grabcad.com/library/sl_st-modular-flexure-0-01-1)[set 02](https://grabcad.com/library/sl_st-modular-flexure-0-02-1)但查看作者收藏页面查看其他很多)

不仅仅是那些超便宜的机械装置利用了弯曲，还有更多。火星探测器使用基于弯曲的悬架，科学仪器(干涉仪等)在需要特定轴约束的小运动中使用它们，最后，MEMS 加速度计和陀螺仪完全基于它们。我们甚至不打算在自然界中举出弯曲的例子。他们无处不在。而且，现在我们有更多的设计例子可以使用，所以为什么不弯曲你的弯曲肌肉，把一个送到 3D 打印机上玩一玩呢？

我们在这里看到很多弯曲，就像这个 [很好地展示了可达到的精确度](https://hackaday.com/2022/06/03/3d-printed-flexture-shows-precision-in-action/) 。Flexures 可以 [做出一些好吃的机制](https://hackaday.com/2021/12/14/fabulous-flexure-mechanism-makes-for-resetting-cat-calendar/) ，还有 [整齐的 3D 可打印输入设备](https://hackaday.com/2021/05/25/3d-printed-joystick-using-spherical-flexure-joint/) 。
