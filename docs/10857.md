# 视频反馈效果创造了高清的壮观景象

> 原文：<https://hackaday.com/2021/07/28/video-feedback-effects-make-a-glorious-spectacle-in-hd/>

视频反馈最广为人知的可能是它在电影《波西米亚狂想曲》中的出现。这不是你经常看到的特别受欢迎的效果，因为用摄像机拍摄屏幕什么的设置起来相当混乱。不管怎样，可能的效果是辉煌的，正如[戴夫·布莱尔]和他惊人的视频反馈动态雕塑所展示的。

在这个过程中没有任何电脑参与——这只是经典的、老式的视频反馈。它是通过将摄像机对准屏幕并将图像反馈到同一个屏幕上而产生的。三个摄像头与双视频开关和分束玻璃相结合，通过 HDMI 输入提供源图像。

通过转动和旋转各种相机，[戴夫]能够使用源图像生成美丽的弯曲分形效果，当他与雕塑互动时，颜色的彩虹融化并扭曲在一起。这是一个美丽的效果，我们很惊讶我们在视频行业没有看到更多。

希望[戴夫]被招募来让他的机器为下一部[Doja Cat]电影剪辑工作，这样我们就能得到更多这种好处。休息后的视频。
