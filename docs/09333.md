# 微控制器开发板可以有多小！

> 原文：<https://hackaday.com/2021/02/21/how-tiny-can-a-microcontroller-dev-board-be/>

市场上有数不清的微控制器板，可以肯定的是，每个可以想象的应用或用户都会有一个。其中有一些非常小的，但这对(阿伦·莫瑞斯)来说还不够。[想要看看没有定制 PCB 的 ATtiny 板能做多小](https://www.instructables.com/Insanely-Small-AVRArduino-Microcontroller-Developm/)，他拿了一个流行的极简处理器的 SOIC-8 版本，并将其与一块 6mm×8mm 的 0.05 英寸原型板配对，以创建一个与其连接器相比相形见绌的设备。

这是一个非常简单的电路，几乎没有什么是以前没有做过的，但这里的价值在于制作它的复杂焊接，而不是它的新颖性。ATtiny402 和三个无源 SMD 元件安装在尽可能小的原型板上，母接头和一组编程引脚对器件体积的贡献远远大于电路板本身。在决定采用更传统的布局之前，他还尝试了两个更小的木板侧面设计。在视频中看到的系统运行演示是一个磁通量检测器，与其所在的 40 引脚 DIP Z80 相比相形见绌。

很多主板都声称很小，但很少有这么小的。这台 ESP32 是更常见的竞争者。
