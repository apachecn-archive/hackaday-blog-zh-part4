# 金属电镀塑料或金属零件

> 原文：<https://hackaday.com/2021/01/01/metal-plating-plastic-or-metal-parts/>

像我们大多数人一样，[Clem]想要 3D 打印金属。金属 3D 打印机确实存在，但对于我们大多数车库黑客来说，它们通常遥不可及。作为替代，[Clem]使用[自制电镀系统](https://www.element14.com/community/docs/DOC-95956/l/episode-477-metal-plate-your-3d-prints-with-a-diy-galvanizing-machine?CMP=SOM-YOUTUBE-PRG-E14PRESENTS-EP477-METAL-PLATE-3D-PRINTS-DIY-GALVANIZING-COMM)来获得带有金属涂层的印刷品。

设置非常简单。小玻璃罐充当电镀槽，机器使用 Arduino 控制器和 PCB 来控制继电器之类的东西，以控制用于电镀的 24V 电压。为了保持一切整洁，[Clem]设计了一个 3D 打印盒，当你不使用它们时，它可以储存所有的电缆和化学品。由于零件可能会变热，塑料是 PETG。

诀窍是零件需要导电才能使用电镀——通常塑料是不导电的。[Clem]给塑料部件上漆以赋予其导电性。石墨涂料并没有产生很大的效果。然而，铁基涂料效果更好，但模糊了印刷的细节。除了电镀(镀锌或钢)之外，您还可以在大约 12 分钟时看到钉子镀铜，一分钟后是塑料电镀演示。这台机器甚至可以使用昂贵的含金电解液镀金。在视频评论中，有人还提到，在不使用涂料的情况下尝试电镀导电细丝会很有趣。[Clem]试图清除一个大部件上的铁锈，但电源无法胜任这项任务。

镀铜通常是使零件导电的一个步骤，这样你就可以镀上另一种金属。除了硫酸铜，可以用[醋酸铜](https://hackaday.com/2013/11/01/copper-electroplating-the-cheap-and-safe-way/)。有时，将金属加工成精细的细节可能会很困难，使用[笔](https://hackaday.com/2019/05/08/make-an-electroplating-marker-because-plating-complex-objects-is-hard/)直接电镀这些区域会更容易。
