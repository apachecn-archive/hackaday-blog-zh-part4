# 主动球节采用球形齿轮

> 原文：<https://hackaday.com/2021/06/24/active-ball-joint-uses-spherical-gear/>

常见的 CAD 操作是采用 2D 形状，并将其挤压成 3D 形状。但是如果你拿一个齿轮，沿着一个球体复制它，然后旋转它，再重复一次，会发生什么呢？正如你在下面的视频中看到的，你最终得到了一个豪猪一样的球，你可以从几乎任何角度向它传递能量。有一篇论文将这个球形齿轮描述为主动球节机构的一部分，即使你没有机械倾向，它也是值得一看的。

球形齿轮——从技术上来说是十字球形齿轮——是由 PEEK 制成的，看起来并没有那么难制造。在驱动系统中还有一个更简单的版本，称为单极齿轮，提供三个自由度。

这看起来正是你的下一个机器人手臂项目的东西。论文涵盖了你需要的运动学方程，但没有提供 3D 打印模型。然而，它确实提到了零件是 SLA 打印的，因此有可能创建具有 3D 打印齿轮的系统。

随着最近将机器人手臂与 3D 打印机融合的趋势，这可能是及时的。上一次我们看到机器人手臂中使用的[球，并不是出于同样的原因。](https://hackaday.com/2020/06/12/rolling-out-a-new-robot-arm/)
