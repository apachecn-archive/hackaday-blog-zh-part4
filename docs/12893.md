# 仿生植入物也会过时和不受支持

> 原文:[https://hack aday . com/2022/02/26/bionic-implants-can-go-obsolete-and-unsupported-too/](https://hackaday.com/2022/02/26/bionic-implants-can-go-obsolete-and-unsupported-too/)

当一个硬件得不到公司的支持时，可能会令人沮丧。错误可能不再被修复，或者在最坏的情况下，完美的硬件可能会随着软件许可超时而完全停止工作。可悲的是，对于一个依赖于第二视觉公司的视网膜植入物的团体来说，该公司已经停止生产和支持给他们提供原始仿生视觉的设备。

这些设备本身由植入视网膜的电极组成，这些电极可以向神经系统发送信号，这些信号以光点的形式呈现给用户。摄像机输入用于捕捉图像，然后将图像转换成信号发送到视网膜电极。至少可以说，结果是低分辨率的，提供的视觉是粗糙的，但它给盲人用户一种他们从未有过的基本感觉。这在视觉上相当于人工耳蜗技术。

这个故事太熟悉了；第二视觉医疗产品公司推出了一种尖端设备，筹集了资金并将其推向世界，但最终却破产了，让其用户陷入困境。超过 350 人在一只眼睛里安装了植入物，而一个叫特里·拜兰的人是唯一一个在他的左右眼球里都植入了植入物的人。该设备的性能参差不齐，一些用户对该设备赞不绝口，而其他人则质疑其实用性。

然而，第二视觉植入物的所有用户现在都靠自己了。任何失败或问题都将得不到纠正，因为公司及其员工不再支持或服务植入物或相关硬件。对一些人来说，植入物仍然有效。对于像芭芭拉·坎贝尔这样的其他人来说，该设备有一天突然断电，从此她就失明了。医疗问题也可能突然出现——公司倒闭后，医生无法保护设备上的信息，导致一名患者无法进行核磁共振成像。

这是一个发人深省的故事，讲述了当一家科技公司倒闭时会发生什么。对于任何从一家未经测试的新公司购买医疗设备的人来说，这是一个值得思考的故事。休息后的视频。
