# 黑客日播客 131:发发善心，晶片机 Pi，只有翅膀，和一个翻转电缆薯条收音机

> 原文：<https://hackaday.com/2021/08/13/hackaday-podcast-131-have-a-heart-transputer-pi-just-the-wing-and-a-flipped-cable-fries-radio/>

《黑客日报》的编辑迈克·斯奇斯和埃利奥特·威廉姆斯讲述了过去一周的硬件黑客事件。有一个新的 Tamagochi hack 可以在 STM32 这样的普通老式微控制器上运行原始 ROM。你知道吗，你可以用一束强激光把相机传感器上的 Bayer 滤镜炸开，而传感器仍然会工作。我们没有。本周有很多关于商用喷气式飞机设计改变的争论，这将移除窗户——但这是为了让飞机更有效率。我们惊叹于用人工心脏输送血液所需要的一切，并在收音机放出神奇的烟雾后进入故障检修兔洞。
