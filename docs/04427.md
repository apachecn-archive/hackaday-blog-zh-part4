# 树莓皮火腿无线电遥控评论

> 原文:[https://hack aday . com/2019/10/07/raspberry-pi-ham-radio-remote-reviewed/](https://hackaday.com/2019/10/07/raspberry-pi-ham-radio-remote-reviewed/)

如今火腿收音机的一个问题是，由于城市法律和房主契约，大多数火腿住的地方不能装一个又大又旧的天线。如果你只是在甚高频或超高频的地方电台工作，这可能不是一个大问题。但是对于高频使用，使用低剖面天线是一件大事。然而，大多数现代无线电可以远程操作。著名的业余无线电公司 MFJ 现在有了 RigPi 站服务器[【业余无线电 DX】有了一个早期版本，并做了审查](https://www.youtube.com/watch?v=GWrLlMP9CsA)。

顾名思义，盒子里装的是树莓派。还有一个音频接口。这个想法是巩固钻机控制连同其他电台控制(如旋转器)以及反馈音频来回到电台。它还向无线电发送莫尔斯电码键控信号。这个想法是，这个盒子将把你的收音机放到网络上，这样你就可以在个人电脑或移动设备上使用网络浏览器来操作它。

据 MFJ 称，你可以轻松地远程操作语音、莫尔斯电码或数字模式。这个盒子使用开源软件，可以控制 200 多种不同的无线电和 30 个转子。当然，你可以自己构建所有这些，并使用相同的开源软件，但它包装得很好。[业余无线电爱好者 DX]说，你不需要了解太多关于 Pi 或 Linux 来使用这个盒子，尽管很明显，如果你愿意，你可以进入 Linux 并使用普通的应用程序。

即使你不想传输，我们也可以看到这样的设置用于远程监控。我们希望看到一个远程终端的配套盒子，它有音频硬件，一个键盘，也许还有一个旋钮来充当各种遥控器。当然，你可以自己想出怎么做。我们想知道一些火腿俱乐部是否会开始通过这样的接口提供远程无线电——我们以前见过这样做，但不太好。

你的 [$50 的收音机](https://hackaday.com/2019/03/15/the-50-ham-entry-level-transceivers-for-technicians/)可能不能用这个，如果你用 [FT8](https://hackaday.com/2018/11/02/ft8-saving-ham-radio-or-killing-it/) ，你可以说你不需要在那里。
