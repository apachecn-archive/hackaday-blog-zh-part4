# AI 在边缘黑客聊天

> 原文:[https://hackaday.com/2019/04/29/ai-at-the-edge-hack-chat/](https://hackaday.com/2019/04/29/ai-at-the-edge-hack-chat/)

加入我们太平洋时间周三中午的 [AI at the Edge Hack Chat](https://hackaday.io/event/164563-ai-at-the-edge-hack-chat-with-nvidia) 与 NVIDIA 的约翰·韦尔什！

机器学习曾经是像 IBM 的沃森或云的几乎无限的计算能力这样的大铁的业务。但是人工智能的力量正在从数据中心转移到边缘，在那里物联网设备正在做以前闻所未闻的事情。能够运行现代人工智能工作负载的嵌入式系统现在足够便宜，几乎任何黑客都买得起，为曾经只是科幻小说梦想的应用和功能打开了大门。

[约翰·韦尔什](https://hackaday.io/hacker/399260-john)是边缘计算领域的领先公司英伟达的开发技术工程师。他会顺便来参加 Hack Chat，讨论 NVIDIA 的 Edge 产品，比如我们最近评论的 Jetson Nano[。加入我们，讨论 NVIDIA 完整的 Jetson 嵌入式人工智能产品线，开始使用 Edge AI，以及 Edge AI 的发展方向。](https://hackaday.com/2019/03/18/hands-on-new-nvidia-jetson-nano-is-more-power-in-a-smaller-form-factor/)

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
