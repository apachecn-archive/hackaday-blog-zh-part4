# 一副死耳塞的无线扬声器

> 原文：<https://hackaday.com/2021/05/05/a-wireless-speaker-pair-from-dead-earbuds/>

有了廉价的蓝牙接收器，制作蓝牙扬声器变得很容易，但令人惊讶的是，没有一种简单的方法来制作一对真正的无线立体声扬声器。[Matt]从 *DIY Perks* 意识到现代的[蓝牙耳塞包含了所有的电子设备来做到这一点](https://www.youtube.com/watch?v=zNHDbXAmY_0)。

由于这种耳塞的流行，在易贝可以很便宜地买到一副坏掉的耳塞。通常，只有电池或扬声器单元出问题，这两者都不是这个版本所需要的。[Matt]将一对耳塞拆开，然后焊接到电池和扬声器电线上。扬声器电线连接到音频放大器，驱动中音和高音扬声器驱动器和低音炮。放大器的输出也经过滤波，以匹配扬声器。电源由一组四个 18650 电池提供。

[Matt]将驱动器和电子设备安装在一些有吸引力的 CNC 加工木质外壳中。在视频中，他非常强调正确地密封所有间隙，以获得最佳的音频质量。和他所有的项目一样，最终的结果看起来和执行起来都像一个高端的商业产品。我们几乎感到惊讶的是，他没有给扬声器添加任何黄铜，正如他在他的 [USB-C 显示器](https://hackaday.com/2021/02/25/diy-usb-c-touch-monitor-is-all-polished-brass/)或 [PS5 外壳构建](https://hackaday.com/2021/03/30/high-end-case-mods-to-a-playstation-5/)上所做的那样。
