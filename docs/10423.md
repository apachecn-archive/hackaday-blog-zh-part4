# 每个极客都必须知道的

> 原文:[https://hackaday.com/2021/06/12/what-every-geek-must-know/](https://hackaday.com/2021/06/12/what-every-geek-must-know/)

怎么可能会有极客文化呢？我的意思是，假设所有书呆子都知道一点欧姆定律，可以通过足够多的量子力学假装至少在鸡尾酒会上很有趣，甚至可能有一个最喜欢的电阻颜色或正弦、余弦和正切角的记忆方法是一回事。但是为什么我们都知道生命、宇宙和一切的答案呢？

几周前，迈克和我在播客上发现他从未玩过星际争霸。我惊呆了！尤其是自从他迷上了电子游戏之后，他就再也没有玩过*这个开创性的*没有石头剪刀布的三方游戏了！我的思维陷入了困境。不过话说回来，在我的人生中，有一段时间我并没有真正读完《T4 沙丘》或《密码游戏》的全部内容，这可能会让迈克的下巴掉在地上。

[![](../Images/27d6eb6a0f54639fbfba3f3d780eb726.png)](https://hackaday.com/wp-content/uploads/2021/06/cthulu_pendant_thumbnail.png) 无论你更喜欢《星际迷航》还是《星球大战》、《黑客帝国》还是《霍比特人》，更令人惊讶的是，我们竟然有这么多共同点！仔细想想，我很确定我们的交流就是原因——这是一种口口相传的文化。黑客空间的一些人正在谈论 Cthulu，你可能会读到一些 Lovecraft。关于《火星人》中黑客的[似是而非的争论已经让至少几个极客去了电影院或图书馆。事情就是这样。](https://hackaday.com/2017/06/30/hacking-on-mars-in-the-martian/)
