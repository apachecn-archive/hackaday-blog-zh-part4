# 用准将熏肉 64

> 原文:[https://hack aday . com/2020/07/27/smoking-meat-with a-commodore-64/](https://hackaday.com/2020/07/27/smoking-meat-with-a-commodore-64/)

当[Deadline]找不到替换他的特制电动吸烟器的控制模块时，他可以直接把它扔进垃圾桶。相反，他决定想出自己的系统来接管吸烟者的原始大脑。基于近 40 岁的准将 64 可能不会是*我们的*的第一选择，但很难对最终结果提出异议。

[![](../Images/da52fc758c2aa12e2f1f970a197e0793.png)](https://hackaday.com/wp-content/uploads/2020/07/c64smoker_detail.jpg)

Connectors to control the smoker’s hardware.

在最基本的层面上，像这样控制电动吸烟器只需要一个温度传感器，一个控制加热元件的继电器，以及一些让这两个设备相互通信的东西。但是为了获得最佳效果，你还需要某种计时器，以及一种简单的方法来动态改变目标温度。将继电器和温度传感器连接到 C64 的背面非常容易，他所要做的就是编写基本代码将它们粘合在一起。

由于 Masterbuilt 的原始控制器通过几个相对完善的连接器与 smoker 接口，这种攻击变得相当容易。因此，他不必摆弄任何电源电压电子设备，只需将连接器中的电线拉高，就可以点燃吸烟者的加热元件。这对于任何想用 C64 或其他东西替换类似吸烟者的控制器的人来说是个好兆头。

过去，我们已经看到了一些令人印象深刻的定制吸烟控制器，看起来它们似乎可以很容易地适应这些商业单位的使用。尽管真正的烟迷[可能更喜欢按照他们严格的规格来建造整个建筑。](https://hackaday.com/2017/12/04/oil-barrel-smoker/)
