# 这三个贝壳之谜终于解开了！

> 原文:[https://hackaday . com/2020/04/16/the-three-shell-mystery-finally-solved/](https://hackaday.com/2020/04/16/the-three-shell-mystery-finally-solved/)

虽然我们当然承认开放硬件社区在帮助缓解冠状病毒危机方面的宝贵贡献，但我们也期待有一天人们开始重新构建除 3D 打印面罩、疫情追踪器和自动皂液器之外的其他东西。然而，[阿吉斯·威彻特]设计的这个洗手计时器是一个非常有创意的版本，它也试图解决长期悬而未决的神秘问题,[如何使用三个贝壳。](https://youtu.be/gdnuOa7tDco)不幸的是，与原电影中的贝壳不同，这三个贝壳并没有取代卫生纸，而许多人此刻似乎非常迫切地需要卫生纸。

构建非常简单，只需要几个现成的组件，包括 Neopixel 条、IR 近程传感器和 Arduino Nano。塑料贝壳取自经典的德国“Schleckmuschel”糖果，因此给这个项目增添了一点复古的味道。如下面嵌入的视频所示，计时器通过连续调暗每个贝壳下的 led 灯来工作，直到所有 led 灯短暂闪烁指示 20 秒的推荐持续时间过去。

洗手定时器项目并不总是可视化的，正如这个播放你最喜欢的 Spotify 音乐的项目所证明的。我们真正想看到的是有人制造一个由脏话触发的卫生纸机。
