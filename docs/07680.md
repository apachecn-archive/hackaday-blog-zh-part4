# 醒目和面包屑悬浮

> 原文:[https://hack aday . com/2020/09/10/吸睛加面包屑悬浮/](https://hackaday.com/2020/09/10/eye-catching-and-crumb-suspending/)

印刷电路板曾经是绿色或褐色的，而且总是隐藏着。现在，它们可以是巧妙的、结构化的，就像电子会议徽章一样，它们是整个项目。在这种情况下，我们发现了 Open LEV，这是一个马蹄形的桌面小玩意，上面布满了支持[声悬浮器](https://hackaday.io/project/169024-open-lev)的模拟电路。[John Loefler]是佛罗里达州一所大学 3D 打印实验室的机械工程师经理，所以，当然，他需要在他的工作空间里放一些最无聊的东西。他没有求助于微控制器，而是用模拟元件填写了一份零件清单。我们不得不假设他剩下的时间都用来准备他的 PCB 展示室了。丝网印刷层的部分也是功能性的。如果你仔细观察超声波传感器(银色圆柱体)连接的地方，有深度计帮助定位。这很聪明。

![](../Images/3ac7b416aee84209b65f446c90c1468b.png)

声悬浮利用了声波中的节点。节点是被高压区域包围的点。换句话说，如果某个东西停留在一个节点内，无论它向哪个方向移动，都会遇到把它推回去的压力。也有什么都不想挂的二律背反。如果有什么东西扰乱了波动，即使是从上面，漂浮的东西也会掉下来。

我们以前看过[声悬浮](https://hackaday.com/2013/09/20/diy-ultrasonic-acoustic-levitation/)和[磁悬浮](https://hackaday.com/2015/11/02/magnetic-levitation-with-arduino/)，但是我们永远不会厌倦。
