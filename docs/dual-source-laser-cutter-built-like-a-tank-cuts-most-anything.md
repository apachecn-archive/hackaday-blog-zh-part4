# 双源激光切割机像坦克一样，几乎可以切割任何东西

> 原文：<https://hackaday.com/2018/07/15/dual-source-laser-cutter-built-like-a-tank-cuts-most-anything/>

激光切割机不是那种你可能会考虑在家里制作的东西，但是如果你仔细并做了研究，你没有理由不这样做。这就是丹尼尔·因格拉西亚用[激光二人组](http://laserduo.com/)所做的，这是一种开源激光切割机，有两个光源用于切割各种材料。他的最终产品不是一个小设备:它有一个压制成型的铝制外壳，看起来更像一战时期的坦克，而不是一件精密机械。但这有一个很好的理由:你不能乱用激光，尤其是激光二人组使用的 130 瓦 CO2 和 75 瓦 Yag 激光。

 [https://www.youtube.com/embed/EAzdC5dfosA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/EAzdC5dfosA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



[Daniele]将开源整个项目，从他使用的定制电机控制器 [Satshakit-grbl](https://github.com/satshakit/satshakit-grbl) 开始。他希望在汉堡大学复制完成后，在 8 月份发布切割机的最终计划。这两种激光意味着它可以切割比大多数激光更广泛的材料:CO2 激光可以切割或雕刻木材，织物或中密度纤维板，而 75W Yag 激光可以烧穿更硬的材料，如黄铜，不锈钢，铜或大理石。这为激光切割机开辟了新的用途:它可以制造多氯联苯，雕刻金属，甚至制作精美的墓碑。150 x 100 x 50 厘米(约 60 x 40 x 20 英寸)的工作区域意味着你也可以在一块完成整个墓碑。

[Daniele]表示，这些零件大多是 3D 打印、CNC 加工或压制成型的。后者可能会超出典型的家庭黑客的能力，但大多数像样的黑客空间将拥有所需的能力，或者知道有人拥有。我们以前见过很多[建造自己的](https://hackaday.com/2018/03/04/k40-laser-cutter-meet-raspberry-pi/) [自己的激光切割机](https://hackaday.com/2016/08/20/bachelor-builds-enormous-laser-cutter-nobody-complains/)项目，以及[改进来自阿里快递](https://hackaday.com/2017/02/22/tales-of-a-cheap-chinese-laser-cutter/)的廉价模型 [，但是这一个坚实的设计和能力使它成为一个值得观看的项目。在此期间，你可以在 7 月下旬在图卢兹举行的](https://hackaday.com/2017/04/24/laser-surgery-expanding-the-bed-of-a-cheap-chinese-laser-cutter/) [Fab14](http://www.fab14.org/) 大会上看看 DuoLaser。
