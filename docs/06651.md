# 激光切割拼图盒是安全破解的乐趣

> 原文:[https://hack aday . com/2020/05/25/laser cut-puzzle box-is-safe-cracking-fun/](https://hackaday.com/2020/05/25/lasercut-puzzlebox-is-safe-cracking-fun/)

如果你走进现实世界，开始随意摆弄保险箱的把手，你可能会发现自己被穿制服的警察带走了。一个更安全的消遣可能是在家里玩你自己的拼图盒，[这正是[thediylife]在这个版本中所做的。](https://www.instructables.com/id/Crack-the-Code-Game-Arduino-Based-Puzzle-Box/)

该设计实现了一个基本的保险箱破解游戏，玩家在一系列回合中尝试猜测保险箱的组合。输入是通过一个旋转编码器，连接到 Arduino Uno 内部。这个项目真的赢了，因为完成看起来如此惊人。保险箱是由 3 毫米的 MDF 制成的，它是用激光切割成型的——在普通的制造商那里很容易组装。该界面由一个小有机发光二极管屏幕和一些发光二极管组成，而一个伺服系统则充当着锁门的角色。当你看到面板下面的组件被热粘在孔里时，你真的会对业务面的整洁感到苍白。

这是一个简单的构建，它将成为一个很棒的隐藏着奖品的派对游戏。我们以前也见过其他的拼图盒构建，比如基于 GPS 的反向地理缓存构建。休息后的视频。
