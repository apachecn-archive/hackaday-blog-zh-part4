# 万圣节轻松闪烁的 LED 眼睛

> 原文:[https://hack aday . com/2018/10/28/easy-blinking-led-eyes-for-Halloween/](https://hackaday.com/2018/10/28/easy-blinking-led-eyes-for-halloween/)

现在时间不多了。如果你要做些什么来给年轻人一些夜惊，以此来换取所有的糖果，你最好快点做。游戏进行到这么晚，你可能没有时间做任何过于复杂的事情，但幸运的是，我们已经发现了一些快速的万圣节技巧，可以让你获得一些非常酷的效果，即使它距离重要的夜晚只有几个小时。

[![](../Images/0d8a6f13ff306bfa1672afb1ce9fef83.png)](https://hackaday.com/wp-content/uploads/2018/10/ledeyes_thumb.gif) 作为一个完美的例子，这些 [LED“闪烁的眼睛”是由【威尔·莫泽】](https://www.youtube.com/watch?v=nbWs9cv3cqM)创造的。使用一些光秃秃的 led、一个 Arduino 和一个纸板盒，这些小工具可以快速轻松地部署在你的窗户或灌木丛中，在太阳下山后产生令人不安的效果。由于 Arduino 代码中的伪随机数发生器，这些“眼睛”甚至有一点可变性，这有助于推销这样一种想法，即你的万圣节游客正在被适当的夜间生物监视。

这个项目的硬件方面非常简单。[Will]拿一个容器，比如一个小纸箱，在上面割两个洞作为眼睛。他指出，内部为白色或反光的容器效果最好。你会想在这里得到一点艺术，想出一些不同形状的眼睛，这是在休息后的视频演示。每个盒子里都有一个彩色 LED，连接到 Arduino。

对于软件来说，[Will]使用一个浮动模拟引脚作为随机噪声源，由此得出每个 LED 闪烁的频率和持续时间。这个项目的硬件和软件对初学者来说都是完美的，所以这可能是一个让小黑客参与今年庆祝活动的好方法；如果你是那种除了让小人爬出来还喜欢复制他们的人。

led 似乎是黑客在万圣节选择的装饰品，从可佩戴的 LED 眼睛到 T2 遥控发光南瓜。
