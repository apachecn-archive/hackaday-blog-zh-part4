# 用于厚皮动物保护的 USB 鼠标黑客

> 原文：<https://hackaday.com/2021/08/13/usb-mouse-hack-for-pachyderm-protection/>

当我们大多数人想到地震仪时，我们的脑海中会浮现出破碎的建筑、弯曲的道路以及搜救队在废墟中挖掘的画面。但是，当[Subir Bhaduri]的团队面临尽可能节俭地解决现实世界问题的挑战，作为 2020 年节俭科学课程的一部分时，他想到了印度农村的农民，对他们来说，由于大象的袭击而损失庄稼是一个现实。这种袭击能够并且已经造成了人类和大象的生命损失。他如何运用科学手段来防止这种冲突，并且成本低廉？

灵感是否来自使用光标速度调到“轨道速度”的计算机鼠标仍有争议，但[Subir]开始寻找这种灵敏度是否可以用于上述大象的地震探测。他的概念证明是一个极其节俭的低成本地震仪，使用一个光学鼠标和一些便宜的 PVC 管和配件。

我们邀请你观看休息时间下面的视频来了解它是如何工作的。你会和我们一样对[Subir]工程原理的实际应用印象深刻。睁大你的眼睛寻找美丽的磁阻尼黑客。真是一种享受！

如果自命不凡的讨厌厚皮动物 p 波激起了你的兴趣，也许你会欣赏以前的项目，这些项目用压电拾音器和管道部件产生数据。
