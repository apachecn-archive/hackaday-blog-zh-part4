# 欢迎来到我们的新机器人…护士

> 原文：<https://hackaday.com/2021/08/25/welcome-to-our-new-robot-nurses/>

汉森机器人公司想要制造机器人，但不是“迷失在太空中”[Robby]的机器人。多想想《星际迷航》机器人的【数据】。他们宣布了[【Grace】，一个栩栩如生的机器人被制造来承担医生和老人的护理职责](https://www.cnn.com/2021/08/19/asia/grace-hanson-robotics-android-nurse-hnk-spc-intl/index.html)。结合奇点工作室，该机器人类似于该公司的[Sophia]机器人，鉴于当前的技术，该机器人被制作得尽可能逼真，并且显然具有沙特公民身份。

这个机器人有热敏摄像头和其他传感器，所以它可以直接读取病人的数据。它使用该公司的 Frubber 作为面部。该公司表示:

> [Frubber 是]一种专有的纳米技术皮肤，模仿真实的人类肌肉组织和皮肤。这使得我们的机器人能够展示高质量的表情和交互性，模拟类似人类的面部特征和表情。

正如你在视频中看到的，Frubber 技术非常好，但不会很快愚弄任何人。但是你可以说机器人静止不动时看起来最真实。当解说员谈论人工智能时，我们在画面中看到一个 Arduino Uno，感到很有趣。

[Grace]会说英语、普通话和广东话，这并不奇怪，因为 Hanson 在香港工作。你会被机器人护士吓跑吗？我们的猜测是，大多数 Hackaday 的读者会很好，甚至好奇。然而，普通大众可能会有所不同。

话说回来，至少她有手而不是触角。她看起来比一些改装过的[儿童玩具](https://hackaday.com/2018/09/22/turn-your-teddy-bear-into-a-robot-with-yales-robotic-skin/)更逼真。
