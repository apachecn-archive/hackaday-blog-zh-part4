# 自制振荡器单独在一个玻璃杯里

> 原文：<https://hackaday.com/2019/09/07/homebrew-oscillator-is-in-a-glass-by-itself/>

当我们挑战自己时，伟大的事情就会发生。但是当别人说‘我打赌你不行’而你成功了，回报会更大。在[威尔科尔]成功制造出音叉振荡器后，他的兄弟向他挑战，要他用酒杯做一个音叉振荡器。我们为此干杯！

首先，[威尔科尔]需要找到一种方法，使酒杯不断振动，而不必站在那里用湿手指触摸边缘。一个安装在附近的压电扬声器做到了这一点。然后他必须探测声波，放大它们，并反馈回来。

在考虑了制造激光麦克风的想法，并放弃了普通麦克风的想法(因为反馈很尖锐)后，他决定使用光。led 没有起作用，可能是因为光线太发散了。但他发现，通过将激光对准合适的位置，酒杯的曲线会调制光线，而光波可以被光电晶体管探测到。然后就是放大声音并反馈给压电元件。

在休息后的演示视频中，你可以看到一旦他倒入一些水，玻璃杯中的振动就会显现。任何玩过水杯的人都会告诉你，这也会改变频率。[编者按:我原以为音高会有大得多的变化。不知道这是怎么回事。]

说到这个，[这里有一个蒸汽酒杯 armonica](https://hackaday.com/2011/10/21/hydrocrystallophone-probably-wont-make-you-insane/#more-59238) ，它使用一个旧的转盘电机来旋转酒杯，并使用一个气压缸来升高和降低水位。
