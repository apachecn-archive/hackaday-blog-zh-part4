# 生产出一种可以射出子弹的道具枪

> 原文：<https://hackaday.com/2020/07/31/producing-a-prop-gun-that-actually-ejects-cases/>

电影《战争之人》在塞浦路斯拍摄时，出现了一个问题。他们需要看起来逼真的道具枪，开火时可以弹出弹壳，但这也是该国枪支法律允许的。这项任务落在了【帕拉迪姆】的肩上，[他开始着手制作一个能够胜任这项工作的道具。](https://www.youtube.com/watch?v=F7YriE5UX3A)

根据塞浦路斯的法律，使用像气枪这样现成的东西是不允许的。相反，他不得不从头开始，创造一个外观上类似于科尔特 1911 的设计，以适应电影时代。使用绿色煤气罐作为动力，第一个重点是获得一个现实的半自动点火循环发生。完成后，下一个目标是让子弹从武器中射出。为了实现这一点，使用了一个杠杆，该杠杆由射击后向后移动的滑块驱动，将“用过的”子弹推出端口。

[Paradym]进行了大量的工作，包括 3D 打印零件的设计、弹簧的加工以及支柱的最终组装。我们以前也见过其他道具枪。休息后的视频。
