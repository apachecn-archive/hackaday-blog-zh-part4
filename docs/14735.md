# 通过刮胡子改善 CD 声音？

> 原文:[https://hack aday . com/2022/09/07/improve-CD-sound-by-shafting/](https://hackaday.com/2022/09/07/improve-cd-sound-by-shaving/)

我们总是喜欢人们为了获得更好的音频再现而做的奇怪的事情。奇特的电缆、特殊的放大器和更高分辨率的数字化想到了。这其中大部分都是可疑的，但[Techmoan]提出了我们过去肯定错过的东西:用一种被称为“CD 声音改善器”的小工具刮 CD 该理论认为，糟糕的 CD 再现来自激光的光散射。据这种老式设备的制造商称，解决方案是切割一个 36 度的斜面，作为一个光阱。你可以在下面的视频中看到这个小工具。

该设备声称它减少了振动，改善了音频，甚至帮助 DVD 播放更好的视频。正如你可能想象的那样，这实际上没什么希望。这个盒子本质上是一个电机驱动的转盘，一个剃须刀刀片，和一个吸尘器的端口，用来吸走脏乱的东西。你也被告知用马克笔给边缘上色。

一方面，这似乎很荒谬。当然，CD 上的标记能击败某些形式的复制保护似乎也很疯狂，但它[做到了](https://www.wired.com/2002/05/cd-crack-magic-marker-indeed/)。显然，相当多的人花了大约 500 美元来修整他们 CD 上的塑料。也有很多积极的评价。但是正如你所料，做这个手术并没有什么实际的好处。然而，[Techmoan]确实以科学的名义牺牲了一张[戴夫·布鲁贝克]的 CD。我们认为你不会对结果感到惊讶。

人们总是在争论设备的微小变化会给你的音频带来多大的变化。不幸的是，大多数简单的事情可能不会产生你能听到的影响，而要产生这种影响[要困难得多](https://hackaday.com/2021/10/20/know-audio-it-all-depends-on-the-dac/)。
