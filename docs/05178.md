# 折叠现成的 3D 打印机

> 原文:[https://hack aday . com/2019/12/26/folding-an-the-off-the-shelf-3d-printer/](https://hackaday.com/2019/12/26/folding-an-off-the-shelf-3d-printer/)

大多数 3D 打印机不会占用很多空间，但它们可能非常笨重，不便于携带。[Jón Schone]在他的 YouTube 频道 ProperPrinting 上的一个秘密项目需要一台[紧凑型折叠式 3D 打印机](https://www.youtube.com/watch?v=6GTCkd7rDe4)，所以他决定修改一台 Creality Ender 3 Pro 来实现这一点。

从一个全新的 Ender 3 Pro 开始，他的第一步是将显示器和电源单元移到底部框架中，为可折叠的顶部框架腾出空间。对于折叠机构，他选定了一个四连杆机构，该机构允许垂直框架向下折叠时平移到打印机的前面，这使它成为一个真正紧凑的包装，浪费的空间最小。接头由碳纤维增强尼龙 3D 打印配件和轴用螺栓组成。通过使用螺杆和挤压框架上的滑动安装点，整个机构可进行微调。框架两侧的小支架允许打印机安全地卡入直立和折叠位置。

总之，我们认为这是一个设计良好、执行漂亮的黑客，足以让我们真的想为自己建立一个。随着重复折叠和使用，是否会出现任何溢出仍有待观察，尤其是在卡扣式末端止动装置处，但如果出现这种情况，应该还是很容易解决的。

我们之前展示过[Jón]的定制 [3D 打印 D-sub 连接器](https://hackaday.com/2019/12/13/feeding-both-filament-and-electrons-through-a-custom-d-sub-connector/)，也是为他的 Ender 3 设计的。也看看另一个折叠打印机， [X 打印机](https://hackaday.com/2019/10/02/x-printer-fits-in-a-backpack/)。
