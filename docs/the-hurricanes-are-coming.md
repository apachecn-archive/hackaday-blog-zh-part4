# 飓风要来了

> 原文：<https://hackaday.com/2020/08/31/the-hurricanes-are-coming/>

现在是北半球的飓风季节，大量的新闻和气象组织仍然致力于提醒人们，如果风暴即将影响他们的地区。接收这些信息的方法也不缺乏。我们都有自己喜欢的天气应用程序或预报网站，手机、电视和广播电台也有紧急警报。不过，如果这些都不适合你，你还可以推出自己的天气预报阅读板。

[受损海豚]用树莓 Pi 和 64×128 LED 矩阵制作了一个天气警报阅读板。Raspberry Pi 运行 Raspbian 并使用 Adafruit 的帽子，一旦连接到互联网，就使用自定义 python 代码下载特定区域的天气信息。从那里，它可以在阅读板屏幕上立即显示任何紧急天气警报，包括飓风警报。它依赖于国家气象局的数据，所以如果你所在的地区没有这种数据，就需要对代码进行一些修改。

虽然他指出，在获取天气信息时，你可能不应该完全依赖他的非专业 python 代码，但这仍然是一种检索天气事件信息的好方法，而不必一直刷新浏览器。一旦风暴过去，[确保你为接下来的日子做好准备。](https://hackaday.com/2014/06/03/solar-energy-provides-an-outlet-for-hurricane-related-woes/)
