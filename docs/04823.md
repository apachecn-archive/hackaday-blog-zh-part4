2019 年 11 月 15 日星期五——帕萨迪纳。[在我写这篇文章的时候，2019 年黑客日超级大会](https://hackaday.io/superconference/)正在进入高潮。坐在注册柜台外的 Supplyframe 总部是一种无休止的娱乐，因为与会者涌入并获得他们的礼品袋和徽章。这就像观看硬件黑客世界的杰出人物游行，每个人看起来都准备好工作了。研讨会正在开始，SMD 焊接挑战正在进行，每个角落似乎都有人俯身查看[令人惊叹的 Hackaday Superconference 徽章](https://hackaday.com/2019/11/04/gigantic-fpga-in-a-game-boy-form-factor-2019-supercon-badge-is-a-hardware-siren-song/)，试图将其变成更令人惊叹的东西。[会谈](https://hackaday.com/2019/09/27/heres-your-first-look-at-the-talks-of-the-2019-hackaday-superconference/)将于周六开始，如果你不是本周末数百名幸运儿中的一员，请确保收听[直播](https://hackaday.com/2018/11/03/livestream-the-hackaday-superconference/)，这样你就不会错过任何活动。

普通人能够用激光从天而降的那一天显然已经到来。住在阿根廷的 Pablo 一直在关注邻国智利的大规模抗议活动。过去几周，大量人群定期聚集抗议不平等。周三晚上，聚集在首都圣地亚哥的人群对头顶上突然出现的一架警用无人机表示不满。在一次令人印象深刻的合作中，[他们将 40 到 50 个绿色激光指示器对准了违规的无人机](https://interestingengineering.com/chilean-protesters-used-lasers-to-bring-down-police-drone-and-it-worked)。显示绿色光束划破空气的视频非常令人惊讶，更令人惊讶的是无人机显然是被激光击落的。很难说是不是 FPV 相机让操作员眼花了，还是几十个激光的累积热量对无人机造成了某种损害，我们猜测，无人机在中央着陆时，抗议者并没有对它太友好，所以飞机可能没有留下多少东西来做法医分析，这很遗憾。我们会注意到，抗议者还将他们的激光对准了一架警用直升机，这种行为对人类飞行员来说是极其危险的，我们不能宽恕。

Chris Petrich 报道说[你零件箱里的 DS18B20 温度传感器芯片很有可能是假冒的](https://github.com/cpetrich/counterfeit_DS18B20)，这应该不会让任何人感到震惊。他从易贝的 20 多家供应商那里购买的 500 个传感器几乎全部被检测为假货。他的 Github readme 有一个广泛的列表，将假冒产品分为四类，问题从不准确的温度偏移到没有 EEPROM 的传感器，无法与寄生电源一起工作。更糟糕的是，许多假货的测试几乎有点像真正的芯片，这意味着它们可能在你的设计中工作，但你显然没有得到你所付出的。简单来说，辨别真假芯片的方法是，马克西姆芯片上有激光蚀刻的标记，而冒名顶替者则展示印刷的数字。如果你需要真正的交易，克里斯建议坚持与有经验的供应链的知名供应商合作。顾客小心上当。

几周前，我们发布了[一个链接](https://hackaday.com/2019/09/29/hackaday-links-september-29-2019/)到恩智浦自制 RF 设计挑战赛，要求参与者用恩智浦的新型 LDMOS RF 功率晶体管制作一些很酷的东西。刚刚宣布了挑战赛的三个获胜者，我们很自豪地看到 Razvan 的出色设计的宽带 RF 功率放大器[，我们最近推出的](https://hackaday.com/2019/11/07/well-engineered-rf-amplifier-powers-ham-radio-contacts/)，获得了第二名。第一名是吉姆·维奇为[设计的另一款宽带放大器](https://sites.google.com/site/rfpowertools/home/nxp-mrf-101)，它可以用现成的 CPU 散热器进行散热管理，售价 80 美元。三等奖授予了韦斯顿·布劳恩领导的团队，他们发明了一种用于立方体卫星微推进器等离子体腔的开关模式射频放大器[，被亲切地命名为口袋火箭。我们最近推出了类似的推进器，我们将在 12 月就这个主题进行一次黑客聊天。祝贺获奖者的优秀设计。](http://superlab.stanford.edu/rfchallenge.html)
