# 发射井发射模型火箭升空

> 原文:[https://hack aday . com/2020/03/16/silo-launch-model-rocket-goes-thoomp/](https://hackaday.com/2020/03/16/silo-launched-model-rocket-goes-thoomp/)

虽然从发射井发射的火箭通常是战争武器，英国石油公司的乔·巴纳德。太空]认为模型火箭技术仍然可以做得更好。于是他建造了一个功能型[管发射模型火箭](https://www.youtube.com/watch?v=6G_o5Idmqns&feature=emb_title)。

像[乔]的[其他火箭](https://hackaday.com/2018/06/20/would-you-look-at-that-yaw-control/)一样，它具有伺服驱动推力矢量系统，而不是稳定的鳍。发射器由一个 98 毫米的纸板管组成，内部有一个气动活塞，在火箭在半空中点燃引擎之前，将火箭从管中弹出。当一切正常时，在发动机启动之前，可以看到火箭静止不动地悬在空中一瞬间。

该发射器还具有一个伺服控制的舱口，它在火箭弹出前打开，然后在火箭离开时关闭，以保护管子。火箭本身是用降落伞回收的，为了搞笑，他增加了一个带降落伞的小型特斯拉跑车。

像这样复杂的项目很少能在第一次尝试中成功，Thoomp 也不例外。让[信号飞行计算机](https://bps.space/signal)在正确的时刻点燃火箭发动机被证明是具有挑战性的，并且需要对如何使用加速度计输入来识别发射事件进行一些调整。飞行计算机也是一个非常强大的数据记录器，所以每一次发射尝试，无论成功与否，都成为一次学习的机会。休息之后，请观看第二个视频，了解所有这些数据是如何分析的。
