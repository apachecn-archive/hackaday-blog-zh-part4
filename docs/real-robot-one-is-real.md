# 真正的机器人一号是…真的

> 原文：<https://hackaday.com/2022/08/07/real-robot-one-is-real/>

我们看到的大多数机器人手臂都很酷，但比玩具强不了多少。通常，他们使用 RC 伺服系统来做运动，这对于做一些基本的运动来说很棒，但如果你想要更工业化和更有能力的东西，请查看[Pavel]RR1-真正的机器人一号。粗壮的手臂有六个自由度，由步进电机和定制的行星齿轮箱驱动。每个关节都有一个编码器，用于精确的位置反馈。正如你在下面的视频中看到的，第一个原型已经在工作了。第二版即将推出。

当你看到这个东西在工作时，你可以马上分辨出它不是玩具。有四个 NEMA23 步进器和三个较小的 NEMA17 电机。虽然有 3D 打印的零件，但你也可以在构建中看到许多金属。可以看到手臂举起 1 公斤的杠铃，拿起提神的软饮料的视频。

从[帕维尔]对第二版的评论来看，听起来他试图将反弹最小化，这对于第一版来说肯定是个问题。我们没有看到该设备任何部分的任何设计文件，因此，就目前而言，它只能作为灵感。但是由于这个项目已经在 Hackaday.io 上发布，我们希望最终能看到第二次修改的设计。

与此同时，如果你想做一些更简单的东西，也许 [raid 一个 CD 驱动器](https://hackaday.com/2022/05/26/robotic-hand-uses-old-cd-rom-parts/)。虽然基于伺服的手臂名声不好，但事实上，[可以通过一些工作让它们变得非常能干。](https://hackaday.com/2021/03/19/robot-arm-achieves-amazing-accuracy-with-just-servos/)
