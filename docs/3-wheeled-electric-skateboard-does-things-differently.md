# 三轮电动滑板做的事情不同

> 原文：<https://hackaday.com/2022/02/09/3-wheeled-electric-skateboard-does-things-differently/>

通常，电动滑板用无刷电机驱动一个或多个轮子，同时保持所有东西都安装在相当标准的卡车上，以保持机动性。然而，[swedishFeetballs]决定走一条不同的路线，使用一些有趣的部件建造一个三轮设计[。](https://www.instructables.com/DIY-3-Wheeled-Electric-Skateboard-With-a-Hub-Motor/)

该建筑依靠一个大型组合轮毂电机和车轮，类似于你在悬浮滑板或一些电动滑板车上看到的那些；这一个是来自易贝的小米零件。它通过现成的电动滑板速度控制器来控制，该控制器配有自己的遥控器。

硬件全部用螺栓固定在一个定制的滑板甲板上，从零开始建造，以接受大的单后轮。在前面，一个普通的滑板卡车被使用。电池安装在甲板下。据报道，该板的最高速度为 15 英里/小时，不出所料，这与小米 M365 的轮毂电机相匹配。

这是一种制作电动滑板的简洁方式，老实说，我们对它的运行方式非常好奇。不幸的是，只有[几秒钟](https://www.youtube.com/watch?v=iqYuhOdaKz4)的镜头可用，但我们已经将它嵌入下面，供您观看的乐趣！
