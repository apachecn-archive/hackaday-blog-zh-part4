# 手持式游戏控制台将处理能力放在盒式磁带中

> 原文:[https://hack aday . com/2019/07/12/hand held-game-console-puts-processing-power-in-the-cartridge/](https://hackaday.com/2019/07/12/handheld-game-console-puts-processing-power-in-the-cartridge/)

随着用于微控制器的廉价屏幕的激增，我们已经看到了小型手持游戏项目的相应增长。选择你最喜欢的芯片，从通常的嫌疑对象中抓取一个屏幕，添加一些按钮，你就可以开始了。[【bobricius】用一种非常规的基于弹药筒的构建对此进行了独特的旋转](https://hackaday.io/project/164736-retro-game-console-with-cpu-on-fake-sd-cards)。

手持设备的主体由吸引人的黑色和金色印刷电路板构成，具有一个屏幕，一些控件和一个开/关开关。板上还有一个 microSD 插座，它与携带微控制器的墨盒连接。改变推车，就能改变游戏。

[bobricius]为各种常见的微控制器平台开发了手推车，从 Attiny85 到古老的 ATmega328。由于 microSD 插槽除了共享屏幕和控件的引脚之外没有什么作用，因此几乎可以将任何平台连接到手持设备上。甚至还有一个树莓派手推车的设计，只是为了好玩。

这是一个有趣的微控制器手持概念，我们迫不及待地想看看它的下一步。这让我们想起了 Arduboy，[如果你真的按下它](https://hackaday.com/2019/01/07/star-fox-comes-to-arduboy/)，它甚至可以做 3D 图形。休息后的视频。
