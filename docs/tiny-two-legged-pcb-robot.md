# 微型两条腿印刷电路板机器人

> 原文：<https://hackaday.com/2019/08/19/tiny-two-legged-pcb-robot/>

YouTuber 和电子工程师(Carl Bugeja)擅长为柔性印刷电路板找到创造性的用途。在过去的一年里，他一直在试验 PCB 马达，将其用于无人机、机器鱼以及最近的蜂群机器人。这是他在振动机器人系列中的最后一个视频，他取得了迄今为止最好的成绩。(嵌在下面。)

他从安装在 3D 打印外壳中的机器人腿和磁铁等柔性 PCB 致动器开始。柔性 PCB 致动器就像低效的电磁铁一样工作，当电流通过时，其效率足以对磁铁做出反应，但其效率不足以使其不能立即释放。

最新的设计使用了刚性的 0.6 毫米 FR4 PCB，作为框架，防止机器人的中部弯曲。机器人的“大脑”位于其中心，与柔性 PCB 致动器相连。由于他过去的机器人的最大限制是重量，他移除了两条腿以减轻 20%的重量，从而使行走更直。他还增加了一个蓝牙模块来无线控制机器人，并用新的更轻的电池(28 mAh，15 C，420 mA)取代了他的旧 LiPo。

他的最新视频显示，机器人能够向前、向后和左右移动。这比他以前的尝试有了巨大的进步，以前的尝试大多导致机器人在原地振动或在工作台周围扑腾。它不会给你买啤酒，但它真的很酷。
