# RGB 派对自行车随着节拍闪烁

> 原文：<https://hackaday.com/2020/11/22/rgb-party-bike-flashes-with-the-beat/>

对骑自行车的人来说，最大的危险之一是晚上不被人看见。为了抵消这一点，各种照明和反光装置都有助于确保在街上看到自行车。当然，你也不必止步于纯实用。[TechnoChic]决定给她的骑行找点乐子，[用许多许多的发光二极管装饰她的派对自行车。](https://www.instructables.com/NeoPixel-Party-Bike-Music-Reactive-Animations-With/)

正如你所料，RGB 照明得益于 WS2812B LED 灯条。表演的是三个 Arduino Nano 33 IoTs 一个用于自行车框架上的 led，另外两个分别安装在前轮和后轮上。这使得可以轻松控制辐条上的 led，而不必将数据和电源线传递给旋转的车轮。框架上的 led 甚至可以对音乐做出反应，Arduino 通过其模数转换器之一对音乐输入进行采样。

与自行车上的音箱搭配，这种建筑是在夜晚宣传城市集体骑行的好方法。我们可以想象这样一辆自行车在临界质量时绝对成功，尽管如果你想吸引燃烧人的注意，你可能需要添加一个激光或闪光炮。如果你厌倦了踩踏板，[你也可以考虑电动转换。休息后的视频。](https://hackaday.com/2019/08/26/the-trials-and-tribulations-of-e-bike-drivetrain-design/)
