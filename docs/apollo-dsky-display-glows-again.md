# 阿波罗 DSKY 显示器再次发光

> 原文：<https://hackaday.com/2021/08/02/apollo-dsky-display-glows-again/>

我们喜欢看到旧技术复活，尤其是在设备最初打算如何使用的情况下。当它是太空装备时，就像[好奇的马克]和他通常的快乐乐队所做的那样，当他们设法点燃几个真正的阿波罗 DSKY 显示器时。

“显示器和键盘”形成了阿波罗制导计算机的人机界面，这是一种专门制造的机器，允许阿波罗任务飞往月球，安全着陆，并返回地球。完整的 DSKYs 很难获得，但一位名叫[Marcel]的幸运收藏家能够获得一对电致发光面板，一个是原型，一个是符合飞行条件的备用。他把它们交给 AGC 专家(Carl Claunch)，由他来解决让显示器重新工作的所有细节——对于一个需要 250 伏、800 赫兹电压的设备来说，这是一个不小的任务。

下面视频的前三分之一主要是关于 DSKY 展览的背景故事和文物的历史方面；跳到 12:30 左右，进入技术细节，包括令人惊讶地使用继电器来驱动显示器的各个部分。一旦你意识到 60 年代中期的晶体管无法胜任这项任务，这就说得通了，这肯定使阿波罗飞船成为一个非常棒的点击场所。我们还对将总中继数保持在最低水平的巧妙方法感兴趣，因为我们意识到并不是每个段的组合对每个七段显示都有效。

该视频有几个客串，像[本·克拉斯诺]，谈到[电致发光显示器](https://hackaday.com/2018/11/26/applied-science-rolls-an-electroluminescent-controller/)和[DSKY 复制品](https://hackaday.com/2019/05/19/ben-krasnow-makes-a-dsky/)时，他自己也毫不懈怠。我们还可以一睹[知名组件切片机](https://hackaday.com/2019/04/02/components-cut-in-half-reveal-their-inner-beauty/)和[MOnSter 6502](https://hackaday.com/2016/05/24/how-the-dis-integrated-6502-came-to-be/)-tamer【tube time[】](https://hackaday.com/2019/04/02/components-cut-in-half-reveal-their-inner-beauty/)的风采。
