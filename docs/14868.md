# 大 3D 打印 BMO 也是 OctoPrint 服务器

> 原文:[https://hack aday . com/2022/09/23/big-3d-printed-BMO-is-also-an-octoprint-server/](https://hackaday.com/2022/09/23/big-3d-printed-bmo-is-also-an-octoprint-server/)

OctoPrint 是 3D 打印机的一个有用工具，基本上可以通过 USB 端口远程访问每台 3D 打印机。【艾莉·卡茨】决定建造一个真人大小的 BMO 形状的 OctoPrint 服务器，来自*冒险时间*，[，结果非常可爱。](https://www.instructables.com/Life-Sized-Talking-BMO-From-Adventure-Time-thats-A/)

Raspberry Pi 4 是构建的核心，[Allie]为此选择了 8 GB 的型号。它配有一个覆盆子 Pi 触摸屏，作为 BMO 的脸。Pi 也有一个立体声音频输出板，并连接到一个定制的 PCB，运行 BMO 的所有按钮。打印 BMO 本身相当简单，但需要一些使用较大 PETG 零件的经验。对于那些在家玩的人来说，一个有用的注意是，Polymaker PolyLite PETG 在蓝绿色方面几乎是 BMO 真实车身颜色的完美复制品。

一点 Python 代码使 BMO 的脸变得生动，并在按下按钮时发出有趣的俏皮话。不过，到了工作时间，触摸屏就成了 OctoPrint 的直观界面。最终的建筑既有趣又实用，是 3D 打印真正可以实现的一个很好的例子。这是一个集可爱的小雕像和功能性印花于一体的产品，[这是我们不常见到的](https://hackaday.com/2016/07/16/pokedex-case-keeps-your-phone-powered-up-so-you-can-catch-em-all/)！
