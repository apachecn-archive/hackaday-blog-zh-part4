# [本·克拉斯诺]推出旧的学校相机用于照相平版印刷

> 原文：<https://hackaday.com/2020/03/26/ben-krasnow-rolls-old-school-camera-out-for-photolithography/>

在一个相机被简化为微芯片的时代，具有讽刺意味的是，带有风箱和黑布覆盖在取景屏上用于对焦的老式取景相机，却作为摄影的象征而存在。这种技术似乎过时了，在现代世界中没有应用，但正如[Ben Krasnow]向我们展示的那样，[当你想制作自制微芯片时，一台老式摄像机正是你所需要的。(视频，嵌在下面。)](https://www.youtube.com/watch?v=YAPt_DcWAvw)

诚然，本在下面的视频中展示的光刻过程是芯片制造的上游。但是在更大范围内掌握这一过程是前进的一步。这个想法是创建一个图案的高分辨率照片——[Ben]选择了一个测试图案和一个 IRS 税单，这是对这个季节的认可——可以用作面具。他选择的相机是一台 4×5 视角的相机，镜头和胶片通过可调节的风箱连接。他发现需要进行修改以保持胶片固定在焦平面上，因此他在胶片包装上增加了一个真空口，以将胶片吸平。冲洗胶片一直是神奇的，在暗房的红光下看着潜像出现在胶片上真的让我们回到了过去——我们几乎可以闻到醋停止解决方案的味道。

[Ben]还逐步完成了余下的光刻过程——用光致抗蚀剂旋涂载玻片，在紫外光下制作底片的接触印版，显影印版，并用钛对其进行溅射。这是一个令人着迷的过程，事实上[本]提到了车库芯片制造商[萨姆·泽洛夫]和来自思想商场的[[贾斯汀·阿特金]，这意味着我们最喜欢的三个 YouTube 疯狂科学家正在合作。可能性是无限的。](https://hackaday.com/2019/07/23/vacuum-sputtering-with-a-homemade-magnetron/)
