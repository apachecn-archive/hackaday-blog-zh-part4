# 无论晴雨，这个气压计看起来都非常好

> 原文:[https://hack aday . com/2020/03/13/this-barometer-looks-mighty-fine-rain-or-shine/](https://hackaday.com/2020/03/13/this-barometer-looks-mighty-fine-rain-or-shine/)

神话传说中，罗马掌管风暴和突发天气的女神坦皮斯在领事西庇阿的船队在科西嘉岛遭遇风暴时救了他。作为回报，她要求为她修建一座神庙。

[![](../Images/cf4626613402e754fa43efc9b0cd9426.png)](https://hackaday.com/wp-content/uploads/2020/03/tempestas-guts.png)【SephenDeVos】[被称为 Tempestas II](https://hackaday.io/project/170292-tempestas-ii-a-clock-style-barometer) 的美丽气压计对观看者没有任何要求，但无论如何都可能会引起注意，因为它看起来太酷了。如果天气一点也不晴朗，合适的遮挡太阳的天气因素，无论是云、更多的云、雨还是闪电，都会摆动到位，层层遮挡蓝天，就像真实生活一样。

总共有五个天气服务伺服系统，它们都由 Arduino Nano 通过 16 通道 PWM 驱动器控制。Nano 从 BMP280 气压/温度传感器获得消息，并相应地驱动伺服系统。

九层装饰精美的树脂玻璃隐藏了机翼中的云和东西，而外面天气很好。我们非常喜欢它的外观——甚至在没有阳光照射的背面也很漂亮。这是一个新的正在进行的项目，所以看起来很有可能[Sephen]会在这个项目结束之前发布代码。与此同时，休息之后看看演示。

我们在这里没有看到太多气压计——也许压力太大了。这个告诉你当压力太低时停止喝咖啡。
