# 阿尔戈斯恐怖之书

> 原文：<https://hackaday.com/2020/07/26/argos-book-of-horrors/>

如果你住在英国以外，你可能不熟悉 Argos，但它基本上是美国人会有的，如果西尔斯在互联网普及后没有成为一个彻底的灾难。虽然他们经营着许多实体店，是一家强大的在线零售商，但他们仍然有一个大的实体目录，出人意料地受欢迎。事实上，它是如此之大，以至于使用机器学习可以用它来做有趣(和令人毛骨悚然)的事情。

来自[克里斯·约翰森]的这个项目被称为“恐怖之书”,它是通过将 Argos 目录的全部 16000 页输入到机器学习算法中而制成的。计算机获取所有页面并生成一个模型，该模型将页面连接成一系列动画，将整个目录融合成一个流动的、不断变化的目录。它近乎令人毛骨悚然，无论是在视觉上，还是在我们无法确切知道计算机在生成这些图像时“在想什么”这一事实上。

模型被训练的步数越多，图像也变得越恐怖。要了解该项目的更多信息，你可以[在 Twitter](https://twitter.com/defaced/status/1278232180523773952) 上关注它，那里会不时发布新的图片。这也让我们想起了其他一些机器学习项目，这些项目最近被用于[制作具有同样迷人图像的短片](https://hackaday.com/2020/05/29/creating-surreal-short-films-from-machine-learning/)。
