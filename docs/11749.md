# EMC 教程让您了解最新情况

> 原文:[https://hack aday . com/2021/10/29/EMC-tutorial-puts-you-in-the-loop/](https://hackaday.com/2021/10/29/emc-tutorial-puts-you-in-the-loop/)

有一次，一个学生问他的实验室老师，为什么他的放大器在振荡。在看了它并注意到它的不规则结构后，老师说，“一个更好的问题是为什么它不应该振荡？”事实是，我们的电路会产生噪声，尤其是在振荡的情况下。失真和非线性会产生谐波，其它元件缺陷也会产生影响。

[FesZ Electronics]有一个很棒的关于开关电源噪声的视频系列，以及关于热环路的最新讨论。如果你想提高下一个设计的噪声性能，这些视频非常值得一看。下面可以看到热循环视频。

我们真的很喜欢家酿噪音探头。除了现实世界的探测。视频还观察了模拟条件下的电路操作。即使您不关心噪声性能，这里也有许多关于基本开关电源设计的有用信息。

您可以看到小热环路 PCB 与极小热环路 PCB 的区别。下次布置电源板时要考虑的一些事情。

如果你想更深入地研究[噪声模拟](https://hackaday.com/2020/11/03/make-some-noise-or-simulate-it-at-least/)，我们有一个很好的读物给你。或者放弃模拟，用 SDR 加密狗把[变成你自己的廉价探测器](https://hackaday.com/2019/04/24/a-diy-emc-probe-from-semi-rigid-coax-and-an-sdr/)。
