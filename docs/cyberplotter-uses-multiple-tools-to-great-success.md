# 数码绘图仪使用多种工具获得巨大成功

> 原文：<https://hackaday.com/2020/09/17/cyberplotter-uses-multiple-tools-to-great-success/>

CNC 故障是这些地区常见的疾病。通常，患者表现出对计算机控制机器的持续痴迷，最常见的是在初次接触 3D 打印机或基本绘图仪等网关硬件之后。一旦诊断出来，就没有治愈的方法——只是建造，再建造的欲望。[亚当·黑尔]对这一点太了解了，[并建造了网络绘图仪来服务于苦难。](https://maniacallabs.com/2020/09/03/cyberplotter/)

电脑绘图仪是[Adam]的 CNC 愿望清单的顶点——一个具有非常大的构建面积的双轴构建，以及为不同的工作安装一整套不同工具的能力。根据所安装的物品，其工作面积可达 800 毫米 x 750mm 毫米，可以产生一些非常大的输出。在 5X 的主持下，[亚当]做了各种巧妙的工作，让系统可以在商业领域使用不同的设备。有一个用于雕刻的激光，和一个顶级的笔式绘图仪，具有高质量的线性轨道，用于精确移动。但有趣的还不止于此——[ Adam]甚至安装了 Z 轴、相机，甚至是我们以前从未想过的基于 Sharpie 的喷枪。

这不是[Adam]的第一次构建，过去的经验在这里大放异彩——在先前知识的武装下，该构建在不完全损害功能的情况下做了许多事情。你可能会觉得[Adam]的名字很熟悉——在之前，我们已经在这些页面上[介绍过他的雕刻师。休息后的视频。](https://hackaday.com/2019/11/06/an-open-hardware-laser-engraver-for-everyone/)
