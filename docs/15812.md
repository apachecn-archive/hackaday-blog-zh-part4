# 一种适用于重型运输的自行车拖车

> 原文:[https://hack aday . com/2023/01/02/a-自行车-拖车-适合重载运输/](https://hackaday.com/2023/01/02/a-bicycle-trailer-fit-for-heavy-haulage/)

作为一个骑自行车的人的一个问题是，自行车的设计不能承载比人多得多的东西。你可以从车把上拿些篮子和购物袋，但迟早会有装不下的东西。在这种情况下，只有一种方法可以让你保持两个轮子:找一辆自行车拖车。如果你想自己建造一辆，那么有 [La Charette](https://charrette.bike/) (法语，[谷歌翻译链接](https://charrette-bike.translate.goog/?_x_tr_sl=auto&_x_tr_tl=en&_x_tr_hl=en&_x_tr_pto=wapp))，一个来自法国的开源三轮设计。

结构是一个坚固的焊接箱形截面管空间框架，前轮提供转向，牵引杆连接到自行车的座位杆。伴随着令人印象深刻的负载能力而来的是牵引它的问题，对于不具备超人力量的自行车手来说，还有一个电力驱动前轮的选择。对于这种大小的负载，停止整个事情是一个基本特征，为此，有一个由牵引杆上的力操作的惯性制动系统。

总的来说，它似乎是一个有用的拖车，尽管在不使用时，它的存储空间较大。这无疑是我们见过的最大的自行车拖车之一，尽管可能不是最时尚的。
