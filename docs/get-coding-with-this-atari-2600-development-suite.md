# 使用 Atari 2600 开发套件进行编码

> 原文：<https://hackaday.com/2019/04/22/get-coding-with-this-atari-2600-development-suite/>

有时，迫切需要忙于为旧的复古系统编码，但不幸的是，入门的门槛可能很高。需要找到一个可行的编译器，更不用说试图找出如何将代码加载到原始的老式硬件上了。不过，没必要这么难。[[heat sync Labs]的团队建造了一个 Atari 2600 开发站，这样黑客空间的成员就可以简单地摇摆起来开始工作。](https://hackaday.io/project/164627-atari-2600-development-suite-for-a-hackerspace)

使用这个平台，开发是一个多步骤的过程。手头有一本纸质手册，提供如何为 Atari 编码的细节。一台 IBM PC 就在手边，允许初露头角的开发人员汇编代码。这个文本文件然后被编译成一个 Atari ROM，然后通过一个特殊的工具将它转换成一个音频文件。这是为了让它与 Starpath Supercharger 一起使用，这允许游戏通过盒式磁带加载到 Atari 上，或者在这种情况下，原始数字音频。通过在连接到增压器盒的 PC 上播放音频文件，可以在 Atari 2600 上运行任意代码。

对于一个新手来说，在 6502 汇编中编程并不是最容易爬的山，但是有经验的程序员可能会喜欢这种简单的开发环境。它为进入复古控制台编程的世界提供了一个简单的入口，没有什么比看到您的代码在原始硬件上运行更有趣的了。
