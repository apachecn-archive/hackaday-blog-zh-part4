# 电子动画尼古拉·特斯拉设置记录直

> 原文：<https://hackaday.com/2020/07/08/animatronic-nikola-tesla-sets-the-record-straight/>

虽然普通读者可能对尼古拉·特斯拉和他令人难以置信的作品了如指掌，但对普通路人来说就不一定了。甚至一个孩子都知道托马斯·爱迪生和亚历山大·格雷厄姆·贝尔的名字，但正如(丹尼尔·斯普林沃尔德)所哀叹的那样，特斯拉这个名字更多地与时髦的电动汽车系列联系在一起，而不是与以其命名的杰出的塞尔维亚发明家联系在一起。

为了公平竞争，[【丹尼尔】想出了一个让这位伟人为自己辩护的方法](https://blog.springwald.de/post/feature/Nikola-Tesla-Bust)。这个定制设计的交流电爱好者的机器人复制品能够以互动的方式讲述特斯拉的生活和成就，尽管这种方式令人毛骨悚然。

关于这一点还没有太多的技术细节，但我们可以从下面的图库和视频中收集到的是，在特斯拉先生的大脑中有数量惊人的 OpenSCAD 设计的 3D 打印部件。在这种组合中加入适量的弹簧、连杆和伺服系统，你就只差一点博物馆展品的小胡子了。

我们过去报道过的大多数电子动画项目[都是基于动物的](https://hackaday.com/2018/07/27/animatronic-puppet-takes-cues-from-animation-software/)，所以看到机械地模仿人类的行为举止肯定很有趣。我们不确定这个会说话的特斯拉头是否有助于教育大众，但它肯定是一项令人印象深刻的技术成就。
