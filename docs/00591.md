# 用于 CAD、游戏和视频编辑的模块化键盘

> 原文:[https://hack aday . com/2018/09/08/用于 cad 游戏和视频编辑的模块化键盘/](https://hackaday.com/2018/09/08/modular-keyboards-for-cad-gaming-and-video-editing/)

在所有的输入设备中，键盘是最伟大的。不过，这也是有代价的:以前，视频和音乐编辑应用程序都有定制键盘。有 Pro Tools 键盘、Final Cut 键盘和无数的 Adobe 键盘。这个问题的解决方法是什么？显然，更多的键盘，这一次我们将使它们模块化。

为了他的 Hackaday 奖参赛作品，[【Cole B】正在建造模块化的可编程 USB 键盘](https://hackaday.io/project/160095-modular-usb-hotkey-controller)。它拥有一切:一个标准的 3×3 键盘，一个只有四个电位计的键盘，一个旋转编码器的键盘，一个推子的键盘。

这些键盘的设计本质上是模块化的，这意味着需要有一种方法将所有这些模块连接在一起，最好没有一堆 USB 电缆散落在各处。目前，[科尔]与[合作的最佳想法是弹簧针和磁铁](https://hackaday.io/project/160095/log/151283-813-pogo-connectors-and-dial-module)。这是一个伟大的想法，尽管苹果公司有不同的想法，可能不会太热衷于看到整个“磁铁和大头针”的想法被偷走。

然而，这是一个很好的项目，它展示了在有限的预算下，你能在制造业上走多远。这些已经是很棒的键盘模块了，连接器方案已经将这个项目推向了键盘黑客的上层。
