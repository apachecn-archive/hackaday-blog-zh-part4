# 逆向技术:理解结构形状的强度

> 原文：<https://hackaday.com/2022/02/03/retrotechtacular-understanding-the-strength-of-structural-shapes/>

实力。刚性。可靠性。承受重量而不弯曲的能力。这些都是我们在建造一个机械结构时要寻找的东西。在今天的回顾性文章中，我们仔细看看一个问题的答案:“什么是形状？”

事实证明，相当多。在 20 世纪 40 年代由多产的 [Jam Handy 组织](https://en.wikipedia.org/wiki/Jam_Handy)制作的一部精彩电影中，我们从科学的角度看了形状如何影响梁的承载能力。比较由相同厚度的金属制成的单面金属片、角铁、C 形槽和盒装管，不仅可以看出它们可以承受多少载荷，还可以看出它们是如何失效的。

然后，这些概念被实际应用到我们日常处理的事物中:桥梁、汽车、飞机和建筑。飞机梁、桥梁、车架和建筑大梁都受益于这部电影时间胶囊中讨论的工程学。

这部影片中的概念没有一个突然过时，因为尽管自这部影片制作以来，我们对工程的理解肯定有所进步，但这些基本概念仍保持不变。因此，它们将适用于我们制造的任何结构或机械设备，无论是 3d 打印、CNC 布线、焊接、粘合、真空成型、拉链捆绑、管道胶带、线包还是热粘合。

睁大你的眼睛，欣赏罕见的[波音 314 快船](https://en.wikipedia.org/wiki/Boeing_314_Clipper)在水上降落和 [1920 年的布法罗斯普林菲尔德蒸汽压路机](https://rootsofmotivepower.com/heavy-equipment/road-roller-buffalo-springfield-steam-roller/)的精彩景象和声音，展示这部电影的赞助商雪佛兰如何制作他们的汽车框架。
