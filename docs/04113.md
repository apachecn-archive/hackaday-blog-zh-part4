# 通过 Arduino 的大而闪亮的俄罗斯方块

> 原文：<https://hackaday.com/2019/09/06/big-and-glowy-tetris-via-arduino/>

俄罗斯方块 T1 在 1989 年为任天堂游戏机发行时获得了巨大成功，就像 1976 年 T2 突破 T3 在街机上获得巨大成功一样。尽管如此，今天的游戏玩家期望的不仅仅是一个有严重运动模糊问题的微型单色 LCD。[进入 LED 俄罗斯方块构建从【电子人】。](https://electronoobs.com/eng_arduino_tut104.php)

该产品依赖于黑客最喜欢的 WS2812B LED 灯串。发光二极管以 8×16 的矩阵排列，创造出人们熟悉的俄罗斯方块游戏场。按钮和操纵杆安装在前面板上，允许玩家控制动作。一个 Arduino Mega 运行该节目，一个 DFPlayer 用来播放著名的主题音乐，如上面的樱桃。

这是一个有趣的构建，对于任何黑客来说都是一个不错的补充。毕竟，巨大的发光二极管让一切变得更好—[这个乒乓球显示屏就是这种形式的一个很好的例子。](https://hackaday.com/2019/08/29/giant-led-display-is-1200-balls-to-the-wall/)休息后的视频。
