# 软件定义无线电变得简单

> 原文：<https://hackaday.com/2020/04/11/software-defined-radio-made-easy/>

就在几十年前，进入业余无线电爱好意味着许多专业硬件，改变你的设置以在各种频率上工作并不是特别容易。由于软件定义无线电(SDR)以一种我们大多数人都可以使用的方式出现，这一准入门槛大大降低，并使广播过程变得更加容易。不言而喻，它确实需要一些软件，但[Aaron]的最新项目使得获得该软件变得极其简单。

他所做的是基于 Debian 创建了一个定制的 Linux 发行版，名为 DragonOS，拥有启动和运行所需的整套 SDR 程序。开箱即用，它支持 RTL-SDR，HackRF 和 LimeSDR 包，甚至包括其他有趣的工具，如 Kismet。他发布了几个视频演示，包括使用 RTL-SDR 接收 ADS-B，还在他的 YouTube 频道上展示了该操作系统在各种场景下的几个定制实现。下面链接的视频还展示了如何在虚拟机中设置发行版，因此即使您没有专用于 SDR 的计算机，您也可以运行它。

进入 SDR 从未如此容易，而且在垃圾抽屉里有一些东西可以用来开始的几率相当高。使用[Aaron]的软件套件，这一过程变得异常简单。不过，如果你在硬件方面有点欠缺，没有比几年前的经典[电视调谐器转 SDR 黑客](https://hackaday.com/2012/03/30/working-software-defined-radio-with-a-tv-tuner-card/)更好的开始了。
