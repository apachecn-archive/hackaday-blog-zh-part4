# 引脚排列书在这里，这正是你需要的

> 原文：<https://hackaday.com/2021/12/28/the-pinouts-book-is-here-and-its-just-what-you-need/>

不幸的是，这些天来自谜一般的[节点]的更新很少。事实上，他的最新帖子只是我们在 2021 年第二次听到黑客的消息。但正如我们对他的黑白版本的期待一样，它肯定不会让人失望。

正好赶上你可能庆祝的任何节日，[【NODE】发布了*引脚书*](https://n-o-d-e.net/pinouts.html) 。他与同事[Baptiste]合作了一段时间的一个项目，免费的 PDF 下载包含 300 多页高对比度硬件图及其各自的引脚排列。这是你所能得到的最简单明了的东西，除了开头的那一页，整篇文档中没有一个字是不完整的。这是黑客极简主义的最佳作品，我们都在关注它。

[![](img/e2bfa5ca84ef7196d5f834a0c339b2ce.png)](https://hackaday.com/wp-content/uploads/2021/12/pinouts_detail.png)

从音频/视频连接器一直到开发板和单板计算机，*引脚书*坚持使用相同格式的图表和随附的图表，使您非常容易找到您想要的东西。如果你需要更多的信息，这种简化的布局无法提供，每个条目都包括一个链接，指向[书的配套网站上的一个专门页面。](https://pinouts.org/)这将引导您进入补充数据，如制造商网站、零件完整数据表等。

根据[NODE]的说法，Creative Commons BY-SA 许可作品的最初计划是以实体书的形式发布，但该项目的规模急剧扩大，他们意识到作为数字文档导航和使用会容易得多。虽然我们不反对，但实体版本放在我们的书架上肯定会很好看。与此同时，那些想在经济上支持这项努力的人可以购买印有直接从书中摘录的图表的衬衫。

[我们一直认为大幅面电子纸设备](https://hackaday.com/2020/04/13/will-2020-finally-be-the-year-of-electronic-paper/)将是黑客工作台的理想工具，因为它允许浏览原理图和数据表，眼睛疲劳最小。现在我们也可以在我们的黑客友好的电子墨水目录上安装的东西[列表中添加一本*的书*。](https://hackaday.com/2021/06/21/review-inkplate-6plus/)
