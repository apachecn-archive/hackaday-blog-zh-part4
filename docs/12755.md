# 这辆迷你摩托车会让你陷入困境

> 原文：<https://hackaday.com/2022/02/13/this-minibike-will-land-you-in-hot-water/>

迷你摩托车是一种美国现象，它让我们这些来自那些这种装置在路上是非法的国家的人着迷；他们似乎提供了很多我们无法参与的低级趣味。[HowToLou]有一个，因为它是在户外使用的东西，自然需要一些方法来固定酿造。他如何解决在偏僻的地方需要一大杯开水的问题？[用于排气管](https://www.youtube.com/watch?v=LA6UZkOVYpM)的重力供给热交换器，由倒置的瓶子制成的容器供给。

从下面的视频中可以看出，这是一个足够简单的工作，但效果惊人。一根小口径铜管穿过软木塞进入瓶中，然后缠绕在一根形成排气口的管子上。由此产生的热交换器被隔热，发动机启动，软木塞松动到足以使少量水流出。结果——早上喝咖啡的热水流量很大。

这可能不是最实用的热水器，但它肯定有点意思[，即使它可能无法与我们介绍过的所有迷你摩托车一起使用](https://hackaday.com/2020/10/03/electric-volkspod-takes-you-on-an-eco-friendly-beetle-cruise/)。
