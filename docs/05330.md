# 安装在芯片上的粒子加速器

> 原文:[https://hack aday . com/2020/01/14/适合芯片的粒子加速器/](https://hackaday.com/2020/01/14/particle-accelerators-that-fit-on-a-chip/)

如果让你想象一个粒子加速器，你可能会想象一个由数百名工程师和研究人员操纵的一公里长的设施中包含的高能电子束。你可能想不到比指甲还小的芯片，然而这正是 SLAC 国家加速器实验室的 T2 芯片加速器国际项目(ACHIP)T3 所完成的。

斯坦福大学的团队开发了一种设备，使用激光沿着硅片上蚀刻的通道加速电子。自 1960 年激光发明以来，微型加速器的想法就一直存在，但对产生电子的设备的要求使得早期的概念验证难以批量生产。

![](../Images/5e3f0076ddf66802ccaf5e1e2d102c49.png)

via Scientific American

激光产生的电磁波比全尺寸加速器中使用的微波波长短得多，这使它们能够在更有限的空间中加速电子——通道可以缩小到千分之三毫米宽。为了正确地耦合激光和电子，光波必须以尽可能多的能量将粒子推向正确的方向。这也需要设备产生电子并通过适当的通道传输它们。有了刻在硅上的[加速器，多个组件可以安装在同一个芯片上。](https://www.scientificamerican.com/article/new-particle-accelerator-fits-on-a-silicon-chip/?text=New%20Particle%20Accelerator%20Fits%20on%20a%20Silicon%20Chip#)

在最新的原型中，激光从芯片上方撞击光栅，将能量导入波导。电磁波向外辐射，随着波导移动，直到它们到达一个蚀刻图案，形成一个聚焦的电磁场。当电子穿过磁场时，它们会加速并获得能量。

结果表明，原型可以将电子提升 915 电子伏，相当于电子在一米内获得 3000 万电子伏。虽然这种变化没有 SLAC 的规模大，但它确实更容易扩大规模，因为研究人员可以在未来的设计中安装多个加速路径，而无需大型全尺寸加速器。该芯片作为加速器的单级存在，允许更多的研究人员进行实验，而无需在昂贵的全尺寸粒子加速器中预留空间。
