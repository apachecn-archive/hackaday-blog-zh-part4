# 这个无线电控制的帆船使用 2X4s

> 原文：<https://hackaday.com/2019/11/08/this-radio-control-sailboat-uses-2x4s/>

当[PeterSripol]还是个孩子的时候，他用一块 2×4 的碎片和一些餐巾纸帆做了一个简单的帆船。他已经不是 8 岁了，但是他决定用他从小学到的技术制作另一艘 2X4 帆船。

你必须克服故事时间和老鼠，但是建造技能是显而易见的。有一个钢筋混凝土舵，一个铅球龙骨，总的来说，这是一个这么简单的建立一个很好的船。

看这个视频你不会学到任何电子学，但是让木头产生流体动力的机械构造很有趣。他们将这艘船与一艘更原始的自制船、一艘商用船进行了比较，在某些情况下，这艘船还是一根木头。

我们不会想到用碳纤维箭杆做桅杆。它们容易获得、坚固且重量轻。好主意。伺服安装技术——虽然不漂亮——当然简单有效。显然，从 8 岁开始，他已经学会了一些东西。

虽然这个视频有点无聊，但帆船对于一些电力非常宝贵的海上传感来说可能很重要。当然，我们最喜欢的帆船在[另一种海洋](https://hackaday.com/2019/06/24/small-lightsail-will-propel-cubesat/)。这让我们想知道你还能从 2×4 中得到什么。
