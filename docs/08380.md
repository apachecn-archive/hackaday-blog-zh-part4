# Vectron 增加了基本和圣诞树控制

> 原文:[https://hack aday . com/2020/11/16/vectron-adds-basic-and-Christmas-tree-control/](https://hackaday.com/2020/11/16/vectron-adds-basic-and-christmas-tree-control/)

[Nick Bild]并不满足于就此罢休，他又一次更新了他的[几乎实用的试验板 6502 Vectron 项目](https://github.com/nickbild/vectron_65),增加了微小的基础和家庭树自动化。没有像上次一样使用[LCD 模块](https://hackaday.com/2019/04/19/a-nearly-practical-6502-breadboard-computer/)，或者他的[使用 7400 系列逻辑](https://github.com/nickbild/vectron_vga)定制的 VGA 输出，【尼克】这次选择走现代路线，使用 TinyFPGA BX 实现了 VGA 输出。

1976 年，比尔·盖茨发表了著名的《致业余爱好者的公开信》之后，Tiny Basic 是 Basic 发布的第一个版本之一。当 Altair Basic 售价为 150 美元时，汤姆·皮特曼为 6800 编写了 Tiny Basic，并仅以 5 美元的价格出售(别担心，汤姆·T2 已经免费使用了)。我们从浏览小小的 Basic 手册中得到了极大的乐趣，并了解到我们的序列号可以在纸带引导上找到，而且电传打字机通常至少在得到 X-OFF 控制信号后会多接收一个字符。

在视频中，你可以看到[Nick]运行一个简短的 Basic 程序，并从 Vectron 操作他的圣诞树灯，尽管它只是开关控制。他暗示 PCB 版本正在开发中，但他很难决定何时停止添加功能。这是一个我们非常熟悉的难题。
