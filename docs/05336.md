# Ondophone On Point

> 原文：<https://hackaday.com/2020/03/28/ondophone-on-point/>

Ondophone 这个名字是两种乐器的混搭，一种是 Marxophone，另一种是 ondes martenot。从 Marxophone 中，[Wintergatan]借用了装有弹簧的锤子，一旦被激活，就会反复敲击琴弦。ondes martenot 借用了特雷门琴般的声音，并能够依靠西方的半音音符。将如此不同的乐器结合在一起需要一个团队，就像它的名字一样，它产生了一种美妙的融合。

在 Ondophone 的左侧，每当琴颈上下移动时，我们都会看到弹簧锤敲打着一根钢弦。旁边是一个 Ebow，它用电磁铁振动琴弦，只要有电就能保持音符。隐藏在琴颈内的是划分半音位置的磁铁，所以你可以轻拂过它们，听到滑动的声音，或者停留在它们上面，跟随曲调。

断断续续的敲击声和嗡嗡声的结合很好地配合了这首歌的“令人毛骨悚然”的阶段，这导致了几乎将这个原型留在绘图板上的范围蔓延。视频讲的是可以用这个设计做的所有事情，这是我们熟知的痛苦/自由。和 Ondophone 头条节目吻别吧。

ondes martenot 是一种早期的电子乐器，所以我们有一些[高科技的迭代](https://hackaday.com/2013/06/25/the-ondes-martenot-better-than-a-theremin/)，如果你还没有听说过 [DIY Ebow 的可能，我们会告诉你](https://hackaday.com/2012/05/09/electromagnetic-harp-with-home-made-ebows/)。
