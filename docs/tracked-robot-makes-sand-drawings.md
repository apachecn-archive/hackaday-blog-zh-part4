# 履带式机器人制作沙画

> 原文：<https://hackaday.com/2020/06/24/tracked-robot-makes-sand-drawings/>

[Ivan]似乎很喜欢制造带轨道的 3D 打印车辆。他的最新作品使用 50 个伺服电机在海滩的沙子上画出图案。你可以在下面的视频中看到它的工作原理。嗯，更准确地说，你可以看到它不工作，然后工作，因为第一次迭代没有完全按照计划进行。

Arduino Mega 2560 提供了大脑，整个单元重约 31 磅，包括电池。我们没有看到伊万的设计文件，尽管你自己对机器人做些改动并不难。

说到重量，我们被他建造的又快又脏的拖车逗乐了。我们想知道他是否有这些轮子，或者他是否必须从某个地方为这个项目找到它们。

机器人或多或少地沿着一条直线移动，伺服系统要么将一个尖尖的部分拖进沙子，要么将尖尖的部分提起，这样沙子就不会在那个区域受到干扰。机器人并不完美。它不仅第一次没有工作，而且在第二次测试中，它看起来至少掉了一个尖角。轨道似乎提供了良好的牵引力，但我们不想打赌运动是完全直线的。

另一方面，它确实完成了任务。有很多电线，我们怀疑这就是为什么它被做成一整块的原因。把它分成几个部分对运输来说会更好。如果你做得对，你甚至可以让它包含不同数量的部分。然而，这需要大量的连接器和一种方法来支撑梁的重量，所以这将是一个更困难的问题。

我们希望设计文件被发布，但我们仍然认为这是一个足够简洁的想法，也足够容易理解。我们不太可能制造一个 30 磅重的机器人，但我们可能会考虑复制一个小尺寸的机器人，明年夏天带到我们当地的海滩上。

我们不禁想起了 [Skryf](https://hackaday.com/2015/10/15/sandy-poems-drawn-by-a-robot-named-skryf/) ，那个没有在沙子里画画而是用沙子画出了*的机器人。然后还有[的沙机器人](https://hackaday.com/2018/10/18/sandbot-happily-and-tirelessly-rolls-patterns-in-sand/)。*
