# 将液晶显示器回收成发光二极管

> 原文：<https://hackaday.com/2018/07/21/recycle-lcds-into-leds/>

当我们看到现代 LED 电视的广告时，我们总是觉得很有趣。这些电视不使用发光二极管来显示画面。他们无非是用 LED 背光的液晶屏代替了冷阴极荧光灯。[Akshaylals]有一些废弃的 LCD 笔记本电脑和手机面板，他决定回收它们[以便在](https://www.instructables.com/id/4-Digit-7-Segment-Display-From-Trash/)内使用 led。

大多数面板用一排发光二极管从一个或两个边缘照亮。你只需要撕下一些胶带和塑料。如果你想知道这些塑料片是做什么的，请看下面这个工程师的视频。

【Akshaylals】担心试图去焊接表面贴装 led 会损坏它们，所以他只是把这些条切成更小的块。为了证明 LED 的实用性，他制作了一些大型的 7 段 LED 显示器，在每段的每一侧使用两个“自由”LED。

一些强力胶和热熔胶以及一些小电线把所有东西都粘在一起。你只需要小心 led 的极性，可以用欧姆表检查。在你把它们粘起来之前，确保它们正常工作可能是个好主意。

为了测试，有一个 LED 驱动板，让 Arduino 用 8 个引脚驱动整个事件。计划是最终在闹钟项目中使用亮白色显示器。

你再也看不到很多非 LED 背光的 LCD 了，但是有可能[转换](https://hackaday.com/2011/03/04/led-backlight-conversion-using-recycled-ccfl-inverter-parts/)。请记住，[有机发光二极管面板](https://hackaday.com/2008/04/17/sony-xel-1-oled-teardown/)是完全不同的，所以撕掉它们不太可能产生 LED 灯条。
