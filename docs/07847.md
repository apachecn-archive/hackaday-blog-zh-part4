# 数字留言簿是完美的黑客结婚礼物

> 原文:[https://hackaday . com/2020/09/22/a-digital-guest book-is-a-perfect-hacker-wedding-gift/](https://hackaday.com/2020/09/22/a-digital-guestbook-is-a-perfect-hacker-wedding-gift/)

他哥哥的婚礼快到了，[塞巴斯蒂安]需要一份结婚礼物。他没有购买任何东西，而是选择建立一个电子留言簿，这样客人们就可以和这对幸福的夫妇分享他们的祝福。

这个留言簿有一个简单的基于网络的界面，可以通过一个域名(Sebastian)进入，这个域名是在婚礼之前用这对夫妇的名字注册的。在那里，用户可以输入文本，并为电子留言簿绘制友好的信息。这本留言簿本身包括一个运行电子墨水显示的 ESP32，包装在一个整洁的 3D 打印外壳中，上面有这对夫妇的首字母。它定期查询网络服务器，并在屏幕上显示找到的消息。

这是电子墨水显示器的一个很好的用途，因为它使在明亮的日光下阅读信息变得容易，而其他技术可能会动摇。[Sebastian]还很聪明地为招待会的夜晚部分安装了一些发光二极管。在之前，我们已经在这些页面上展示了一些结婚礼物，包括这个特别有趣的[方糖。](https://hackaday.com/2012/02/17/wedding-gift-fail-has-happy-couple-cursing-your-name/)休息后的视频。
