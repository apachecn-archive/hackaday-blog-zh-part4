# 机电时代的终结

> 原文：<https://hackaday.com/2021/12/18/the-end-of-the-electromechanical-era/>

从长远来看，21 世纪的最初几年可能会被视为人类技术发展的一个短暂时代的结束。在 20 世纪初，几乎所有的东西都是机械的。当然有一些电子设备，但是像留声机和“电影”相机这样的消费产品纯粹是机械产品。你把它们摇起来，它们就在弹簧上跑。如今，你购买的几乎所有消费设备都将完全电子化。在这之间，有大约 50 年的时间，我称之为机电时代。

珍妮·李斯特本周拆解了一台 1972 年的老式富士胶片电影相机，完美地捕捉到了这个时代的中期。有一个小的印刷电路板和一个电动马达，但控制装置中的大部分重物实际上都放在杠杆、轴承和非常聪明的机械装置上。电气和机械系统是松散耦合的，电气由机械控制。

我愿意就具体细节进行辩论，但我初步认为机电时代的巅峰在 1990 年左右。去年，我不得不更换一台[索尼随身听 WM-D6C](https://walkmancentral.com/products/wm-d6c) 中所有腐烂的橡胶传动带，这是一台 1984-2002 年生产的专业便携式磁带播放器和录音机。

这不是一个简单的磁带录音机——电机是电子调节的，以保持这样一个小设备荒谬的恒定速度，我的内置了杜比 B *和* C 降噪电路，以及一些像样的麦克风前置放大器。但是，当你按下快进按钮时，它会物理地推开涂有橡胶的驱动轮，滑动的金属片会通过建立和断开电接触来改变操作模式。它的精度不仅取决于电子设备，还取决于机械组件和电机。这是真正的半电子半机械。

但那个时代早已过去。CD 机的出现标志着结束，虽然当时我们没有看到。当然，有一个马达，但所有的按钮都是电子的，所有的“机制”几乎都是在硅中实现的。数码相机可能是机电时代棺材上的最后一颗钉子:由于不需要处理物理胶片，对任何机械产品的最后需求都消失了。如果你不明白我的意思，请打开 GoPro。

虽然我很高兴再也不用更换盒式录音机的驱动橡胶了，但我想到早期的 iPods 和旋转的金属硬盘，以及它们如何让位于我现在使用的全硅 Zoom H5 录音机，我感到有点悲伤。它有一个信噪比和安静的前置放大器，没有哇或颤动，以及一个我买 WM-D6C 时简直难以置信的质量。

然而，如果你发现自己在旧货店，而你以前从来没有这样做过，买下并拆开这些过去时代的奇迹之一。一台卡式录音机，即使是便宜的，也隐藏着丰富的机电设计。
