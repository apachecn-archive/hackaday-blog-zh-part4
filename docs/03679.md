# Popstick 风扇车是一个有趣的蓝牙建筑

> 原文：<https://hackaday.com/2019/07/25/popstick-fan-car-is-a-fun-bluetooth-build/>

Archer 的粉丝们已经知道了，但对于世界上的其他人来说，值得一提的是——船很好，但粉丝的船更好。陆地车辆也是如此。【修补技术】也有同感，[造了一辆蓝牙控制的风扇车，可以在地板上跑来跑去。](https://www.youtube.com/watch?v=CN5wAmYsRMc&feature=youtu.be) (YouTube，下面嵌入。)

建造从一系列粘在一起的棒棒糖开始，形成一个底盘。扭带然后被用来作为瓶盖车轮的轴，而转向是由一个伺服控制的纸板方向舵操纵的。推进是通过一对装有风扇的寻呼机大小的马达实现的。一只名为 Adafruit Bluefruit Feather 的 M0 负责表演，它通过蓝牙接收命令，并通过车辆中央的 H 桥芯片驱动电机。

这是一个有趣的手工艺风格的建筑，对于对电子和制作感兴趣的孩子来说是一个很好的项目。它教授基础电子学，也是对微控制器世界的一个很好的介绍。这是我们见过的较小的无线电控制建筑之一，[但如果你喜欢的话，你可以随时进行全尺寸建造](https://hackaday.com/2018/03/16/go-big-or-go-home-this-arduino-rc-car-can-take-you-there/)。
