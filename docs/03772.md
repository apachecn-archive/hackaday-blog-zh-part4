# 用树莓派驱动 P5 面板

> 原文:[https://hack aday . com/2019/08/01/driving-P5-panels-with-a-raspberry-pi/](https://hackaday.com/2019/08/01/driving-p5-panels-with-a-raspberry-pi/)

用 led 制作显示器可能很有趣，但随着像素数量的增加，很快就会变得乏味。在这一点上，投资于现成的面板是值得的，这些面板已经预先焊接好，随时可以使用。[汤姆·哈蒙德]想要运行一个 P5 面板，[并让一个树莓派开始工作。](https://www.instructables.com/id/P5-LED-Panel-With-Raspberry-Pi/)

P5 面板之所以被称为 P5 面板，是因为它们的 led 之间的间距为 5 mm。有问题的面板由 RGB LEDs 制成，允许显示全色图形。在这个建筑中，一个 64 x 32 的面板总共使用了 2048 个像素。Raspberry Pi Zero 运行显示器，使用其 GPIO 输出像素数据。为圣诞灯光显示设计的软件用于对灯光表演进行编程，在这种情况下选择 xLights。所有这些都被包裹在一个整洁的 3D 打印框架中，最终产品看起来组装得非常好。

这是熟悉 P5 面板的一个很好的方式，如果你打算在赛道上建造一个更大的建筑，这也是一个很好的起点。如果你全力以赴，[从架构角度考虑如何充分利用你的安装](https://hackaday.com/2019/02/10/hidden-led-video-wall-at-the-oregon-museum-of-science/)。休息后的视频。
