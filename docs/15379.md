# 把铰链带给一个 90 年代的游戏男孩

> 原文:[https://hack aday . com/2022/11/16/把铰链带到一个 90 年代的游戏男孩/](https://hackaday.com/2022/11/16/bringing-the-hinge-to-a-1990s-game-boy/)

手机的新热点是它似乎是一个铰链，一个非常脆弱的柔性液晶显示屏。当然，铰链并不是什么新东西，几十年前，它们在功能手机中风靡一时，当然，任天堂也加入了这一行列，推出了非常可爱的 Game Boy Advance SP。这种铰链设计引起了[Allison Parrish]的注意，他用[Game Boy Pocket SP](https://posts.decontextualize.com/pocket-sp/)将它带到了早期一代的 Game Boy 上。这是一款上世纪 90 年代末的 Game Boy Pocket，其 PCB 被小心地切成两半，装在一个定制的外壳中，看起来就像 Advance SP 的铰链外壳。

该案例是一个整洁的 CAD 工作非常漂亮的 3D 打印，并配备了一套先进的 SP 铰链，但也许这一建设的最整洁的部分是一套灵活的 PCB 连接在一起的两个单位的一半。这看起来就像是从工厂出来的一样，这是一个超越许多主机模块的成就。

与 SP 相比，整个东西有点粗糙，以便为口袋的全尺寸墨盒腾出空间，但不会太难看。任何一个拥有这种手机的 90 年代的孩子都会成为他们所有同学羡慕的对象！

你可能不会惊讶，这并不是任天堂游戏机第一次被改编成 SP 风格的游戏。
