# 足够先进的技术和公正

> 原文:[https://hack aday . com/2020/12/05/足够先进的技术和司法/](https://hackaday.com/2020/12/05/sufficiently-advanced-technology-and-justice/)

想象一下，你是陪审团的一员，你得到了一个从监控摄像头拍摄的图像。它看起来非常像嫌疑人，但图像被原始图像的人工智能“增强”了。你定罪了吗？这在合理怀疑的天平上如何衡量？你应该要求看原作吗？

人工智能增强、放大或修改的图像非常逼真。但是他们给你看的不是现实。当我们上周写下这一点时，我们强调的一种方法的作者之一，[Denis Shiryaev]在评论中指出，这些修改不是原始的“恢复”。举例来说，虽然他们可能会添加难以置信的细节，但他们不会重现或还原现实。神经网络从它所学习的数百万张面孔中创造出自己的现实。

[![](../Images/b5e0e30e2098ce5e6f6033cfdff1f2b5.png)](https://hackaday.com/wp-content/uploads/2020/12/obama_depixel.png) 出于识别的目的，这正是问题所在:数百万*其他人*的面部特征被用来提高分辨率。你能认出像素化图像中的人吗？你能在产生的上采样中识别出同一个人吗？如果陪审团面前的问题是“被告是美国前总统吗？”根据呈现给你的图像，你会有不同的回答。你会对自己判断人工智能修饰照片的能力产生误导性的信心。显然，陪审团需要知情的怀疑态度。

不幸的是，我们都在电影和电视节目中看到过无数“放大、增强”的例子，它们被成功地用来逮捕罪犯并给他们定罪。我们还没有看到关于对抗性神经网络如何从少量像素中创建人脸的详细分析。这一点，再加上最终产品近乎神奇的分辨率，肯定会动摇正常人的陪审团。另一方面，故意误导的“深度假货”的流行可能有助于教育公众，当涉及人工智能时，相信他们看到的东西是危险的。

这仅仅是一个例子，但是让公众对运行我们世界的技术的深层运作和局限性保持兴趣并接受教育比以往任何时候都更重要，但是有些材料确实很难。我们如何将科学与魔法分开？
