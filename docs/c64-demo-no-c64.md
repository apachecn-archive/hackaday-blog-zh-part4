# C64 演示，无 C64

> 原文：<https://hackaday.com/2021/07/08/c64-demo-no-c64/>

永远不要低估场景的独创性。自我施加的限制导致了难以置信的创造力，并且，他们设法让他们的演示运行的设备的范围从未停止过让我们惊讶。但是我们从来没有想到我们会看到一个没有核心组件的 C64 演示:C64 T1。

充分披露:[马蒂亚斯·克拉姆]的演示，称为“自由旋转”，确实需要一个 C64 才能入门。古老的基于 6502 的计算机在 1541 磁盘驱动器上运行加载程序。但从那以后，都是软驱。马提亚斯已经暴露了他所有的诡计。

下面的视频展示了完整的演示，包括一个令人心跳停止的摄像头电缆模块。通过增加一个 100ω电阻，[Matthias]将串行时钟和数据线变成了一个两位数模转换器，足以为黑白像素产生信号，并产生显示器所需的同步脉冲。

没有声音的演示是不完整的，Freespin 的曲调来自控制驱动器的步进电机，就像一个单音软盘晶体管。

看着除了软驱以外的东西运行一个很酷的演示是非常令人惊奇的。是的，我们知道软盘里有一台成熟的计算机，但是让它工作所需的比特碰撞仍然令人印象深刻。看着你用多个驱动器能做些什么可能很酷，但我们也理解极简美学。说到小小的演示:256 字节的[HellMood]“T2”记忆或[Linus akeson]“T4”一个心灵诞生仍然让我们无语。
