# 免提翻页

> 原文：<https://hackaday.com/2021/01/07/hands-free-page-turning/>

对于那些动不了一根手指就能翻开电子书的人来说，一个解决方案就在眼前。首尔的科技公司 Visual Camp 已经将他们的眼球追踪算法应用于电子书阅读器。(视频，嵌在下面。)据报道，这是电子书阅读器第一次如此配备。

如果你的眼睛停留在翻页按钮上，它就会翻页。虽然这种特殊的应用程序似乎无害，但其他一些被吹捧的应用程序似乎有点做作，如果不是侵入性的话。比如在你看书的时候应用凝视分析，他们声称可以对其他书籍进行有针对性的推荐。

在之前，我们已经讨论过眼球追踪设备[，但是它们使用了硬件。Visual Camp 声称，他们基于人工智能的技术只需要一个彩色摄像头，可以集成到现有的配备摄像头的设备中，例如这款电子书阅读器。他们还为希望在应用程序中添加眼球追踪控制的开发人员提供了一个 SDK。然而，眼球追踪是很难的，困难在于细节。很想看看他们在做什么。](https://hackaday.com/2020/10/02/seek-and-ye-shall-command/)
