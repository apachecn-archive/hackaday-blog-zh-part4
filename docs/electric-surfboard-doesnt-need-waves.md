# 电动冲浪板不需要浪

> 原文：<https://hackaday.com/2020/09/08/electric-surfboard-doesnt-need-waves/>

冲浪是一项伟大的运动，但它严重依赖于环境来提供合适的波浪进行练习。如果你没有住在正确的海岸附近，你只是运气不好。当然，你也可以给自己造一个电板，而不是去破坏当地的湖泊—[,这正是[西蒙]所做的。](https://www.youtube.com/watch?v=HG_EgJwo_WU)

继早期 3D 打印船的实验之后，冲浪板由一对喷气驱动装置驱动，来源于[【青少年喷气】](https://www.youtube.com/watch?v=-dTv7zzceto)。西蒙不是一个吝啬马力的人，他选择了双 5000 瓦的发动机来提供充足的动力。当然，在这样的功率水平下运行会带来一些令人头痛的问题，而且一路上会有大量的电子稳定控制系统损坏和电机损坏。坚持不懈和大量的现金支出之后，[和冲浪板以超过 45 公里/小时的速度在水中雕刻出优雅的线条。](https://www.youtube.com/watch?v=ImoUkJRDheM)

未来的更新旨在解决水进入电子设备的问题，特别是手控器，西蒙已经破坏了两个。我们期待着在未来几天看到更多这样的飞行器出现在水中，特别是因为它们的噪音和烦人程度至少比普通的喷气式滑水艇低 30%。休息后的视频。
