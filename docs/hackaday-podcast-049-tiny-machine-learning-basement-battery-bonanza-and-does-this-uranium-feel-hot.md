# Hackaday 播客 049:微型机器学习，地下室电池富矿，这种铀感觉热吗？

> 原文：<https://hackaday.com/2020/01/10/hackaday-podcast-049-tiny-machine-learning-basement-battery-bonanza-and-does-this-uranium-feel-hot/>

Hackaday 的编辑迈克·什奇斯和埃利奥特·威廉姆斯对所有的黑客进行了分类，以找出本周你可能错过的最有趣的硬件项目。你知道没有神经网络也可以使用机器学习吗？这是一个在 ATtiny85 上做的项目。我们还围绕 3D 打印的压弯机进行思考，研究 ESP32 的节能功能，使其更好地使用电池，并讨论物联网咖啡机黑客，这是如此之好，以至于可以成为一个股票功能。此外，我们深入自然发生的核反应堆，欣赏常见的，但不可思议的条形码。
