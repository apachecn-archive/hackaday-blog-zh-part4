# 你们到底有多少人？

> 原文：<https://hackaday.com/2020/08/01/how-many-of-you-are-there-really/>

我们现在习惯于听到，“我们都有自己独特的方式。”但是如果我们真的不那么独特呢？许多人认为不超过两种政治观点，也许是少数宗教信仰，当然也不超过一种方式来描述一个黑客。但尽管在生活等其他方面存在这种争议，至少我们都可以依赖于我们个体名字的唯一性。还是可以？

你有没有想过有太多的人叫[在此插入姓名]？嗯，[妮可]认为在她的祖国比利时，有太多的人分享她的名字，她决定把它做成一件艺术品。

她能够找到比利时人名字的数据，并写了一个 Python 脚本…呃…用 Excel 找到每个邮政编码中 Nicoles 的数量。然后，她创建了一个比利时的 3D 地图，分成每个省，每个省的高度与该地区尼科尔的数量成比例。这是一项非常简单的打印工作，目前任何标准的 3D 打印机都可以完成。

算不上什么“做点什么”的技巧，但可能会成为一个很酷的令人沮丧的装饰品，它会不断提醒我们[我们是多么独特](https://hackaday.com/2020/07/18/the-sincerest-form-of-flattery/)。
