# 在湖上度过一整天

> 原文：<https://hackaday.com/2018/12/06/spend-all-day-on-the-lake/>

随着太阳能电池板价格的下降，太阳能汽车变得越来越普遍，所有这些汽车的电机和控制器的可用性上升。用一套工具制造一辆太阳能电动自行车是一回事，但这艘太阳能船是各个层面的黑客大师，从太阳能驱动系统到浮桥，甚至是锚。

[J Mantzel]在他的频道上有许多关于他的船的视频，看完所有这些视频可能会让你想要建造自己的船。他的船上几乎所有的东西都是从他周围的东西开始建造的。例如，锚由玻璃纤维手工制成，然后填充混凝土，他的转向系统是一个由绳索、滑轮和轴组成的半复杂系统。这艘船的大部分外壳也是由玻璃纤维手工制成的，所有可以重新利用的东西都留到以后使用。

该系统的十个面板、电池、逆变器和其他杂项部分约占整个船只成本的一半，但他报告说，他还将该船用作他家的备用电源，并可以使用该系统运行其他东西，例如电锯。他还把船用于露营和建筑，而且不用担心燃料，这对他非常有用。

如果你看了频道上的视频，你会发现这不是他唯一的太阳能船。他最近完成了一艘[太阳能快艇](https://www.youtube.com/watch?v=-AdKmOpP1x8)以及一个定制的螺旋桨，可以真正在水中移动。他的视频显然也很受欢迎，因为他们已经被[的读者反复链接到我们最近发表的一些](https://hackaday.com/2018/12/04/go-up-a-creek-without-a-paddle/) [太阳能汽车](https://hackaday.com/2018/11/28/pedal-far-with-a-solar-powered-tricycle/)的报道。
