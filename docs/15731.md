# 经典 HP-16C 计算器的现代致敬

> 原文:[https://hack aday . com/2022/12/21/a-modern-tribute-to-the-classic-HP-16c-calculator/](https://hackaday.com/2022/12/21/a-modern-tribute-to-the-classic-hp-16c-calculator/)

HP-16C 计算机科学家深受喜爱，因为它是惠普公司制造的唯一一款专用程序员计算器。世界上大多数幸存下来的例子都被很好地使用了，从 1989 年起你就不能从惠普订购一台了。因此， [[K Johansen]开始使用现代硬件建造一个向 HP-16C 致敬的产品。](https://hackaday.io/project/188722-the-pico-16c-an-homage-to-the-venerable-hp-16c)

构建依赖于一个树莓派 Pico 作为操作的大脑。与许多经典的惠普计算器一样，它以[反向波兰符号、](https://hackaday.com/2021/06/05/an-open-source-scientific-rpn-calculator/)进行运算，并包括惯常的堆栈操作。为了更好地为程序员服务，它被设置为接受十六进制、八进制、十进制和二进制格式的输入，并可以很容易地在它们之间转换。除此之外，它还配备了常见的算术运算符，以及像 NOT、AND 等位运算。

然而，也许我们最喜欢的是键盘。这一切都是由廉价的全球速卖通键盘、标签制作机和激光打印机组合而成的。这是一个完全 DIY 的工作，有点粗糙，但它使计算器更容易使用。

它不是 HP-16C 的精确复制品，但操作上的差异很小。那些希望构建自己的文件的人可以从项目的 [Github 页面获取所需的文件。](https://github.com/someyob/Pico-16C) [我们以前也见过其他经典惠普计算器的复制品。](https://hackaday.com/2021/11/24/build-your-own-hp41c/)如果你有自己的数学项目正在实验室酝酿，请不要犹豫，把它们发送到[的 tipsline！](http://hackaday.com/submit-a-tip)
