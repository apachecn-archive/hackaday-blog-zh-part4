# 手工锻造的表壳让谢妮钟表成为艺术品

> 原文：<https://hackaday.com/2018/09/17/hand-forged-cases-make-nixie-clocks-into-works-of-art/>

“谢妮”和“蒸汽朋克”都有点被过度使用了。很难统计我们最近看到的结合了两者的时钟项目的数量，通常我们不愿意在没有好的理由的情况下展示这个主题的另一个变体。[这是个很好的理由](https://www.youtube.com/watch?v=Mj6AW-_kHSM)。

简单地说，[Claes Vahlberg]制造的一位数谢妮钟是艺术作品。这款钟有一个小版本，配有一个 16 英寸的谢妮，还有一个大版本，使用了一个 Dalibor Farny 定制的谢妮，这本身就是一件艺术作品。每个时钟都有时间和日期、温度和气压等功能，甚至还有当前月球周期的剩余天数。不过，这些钟表盒才是真正的享受。它们由钢手工锻造而成，让我们想起锅炉顶部的汽笛。

[Claes]没有太多关于构建过程的细节——我们一直在联系，他说他正在编写文档——但这没关系。好像这还不够，这些钟是由遥控器控制的，它有自己的 IN-16 电子管，是运动控制的。最后一点是一个很好的触摸，因为没有按钮来分散对金属外壳的光滑线条的注意力。

我们滔滔不绝，但我们认为这个真的很棒。这并不是要从以前的谢妮-蒸汽朋克混搭中拿走任何东西，比如这个一位数的时钟或太阳能电表。但是这些时钟是一种超越。
