# 窥视最新的回声秀 10

> 原文:[https://hack aday . com/2021/04/09/taking-a-peek-inside-the-new-echo-show-10/](https://hackaday.com/2021/04/09/taking-a-peek-inside-the-newest-echo-show-10/)

当亚马逊发布最初的 Echo 时，这是一件非常简单的事情。圆柱体，顶部有一些发光二极管，完成。然后他们推出了回声点，基本上是一样的东西，但更短。有一段时间，似乎有一个非常明确的主题，但后来在某个时候，亚马逊决定开始生产各种形式的 Echo 设备是一个好主意，从墙上的插头到字面上的太阳镜，事情变得复杂得多。作为一个完美的例子，看看这个[由【txyzinfo】](https://hackaday.io/project/178825-amazon-echo-show-10-3rd-generation-teardown)拆解的第三代 Echo Show 10。

诚然，基地仍然看起来有点像旧的回声，但家族的相似之处仅限于此。正如你可能从名称中收集的那样，该节目采用了高分辨率的 10.1 英寸液晶面板，大大提高了亚马逊能够向用户强制投放的广告的数量和类型。在真正的*黑镜*时尚中，底座中甚至有一个无刷电机，允许机器将显示器转向用户，无论他们如何试图逃脱。

[![](../Images/df8acf2136373e6884136f5dac274450.png)](https://hackaday.com/wp-content/uploads/2021/04/show10_detail.jpg)

A salvageable part if there ever was one.

拆卸过程没有任何评论；在下面的视频和黑客日。IO 页面，你会发现设备内部清晰明亮的图像。但是对于那些只是对这些 250 美元的小玩意的内部感兴趣的人来说，这才是你真正需要的。

在这一点上，[txyzinfo]似乎没有试图对该节目进行逆向工程或弄清楚它是如何工作的，看看主板的复杂性，我们并不感到惊讶。尽管如此，看看他们将所有的硬件都装进了这么一个相对较小的设备中，还是令人惊叹。

如果你在寻找更新的 Echo 设备的技术测试，[【Brian Dorey】在 2019 年](https://hackaday.com/2019/08/15/uncovering-the-echo-dots-hidden-usb-port/)对第三代 Dot 进行了一些令人印象深刻的探索，【electronupdate】甚至[对 Flex](https://hackaday.com/2021/01/14/is-your-echo-flex-listening/) 内部的一些芯片进行了解封。在软件方面，看看最近[为最初的 Echo](https://hackaday.com/2021/03/22/amazon-echo-gets-open-source-brain-transplant/) 制作开源固件的努力。
