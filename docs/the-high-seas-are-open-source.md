# 公海是开源的

> 原文：<https://hackaday.com/2020/03/01/the-high-seas-are-open-source/>

拥有一艘旧船的最大问题之一(除了是一个钱坑——无论船龄如何，所有船都有这个问题)是缺乏零件和设备，如果你能找到这些零件，也缺乏对它们的支持。像大多数事情一样，这是一个可以从一些开源解决方案中受益匪浅的领域，德国的 [Open Boat 项目已经展示了这一点。(](https://open-boat-projects.org/)[谷歌翻译自德语](https://translate.google.com/translate?sl=de&tl=en&u=https%3A%2F%2Fopen-boat-projects.org%2F))

该小组为几乎任何尺寸的船只提供各种设备问题的解决方案。在他们最近的展览会上，许多人对目前只有昂贵的专有选项的情况下的开源解决方案感兴趣，例如对各种绘图设备的支持。然而，这并不是这个项目的唯一部分。它包括许多独立的项目，如他们的自动驾驶和导航解决方案。甚至还有完整的硬件包，都有完整的文档记录。

出于多种原因，像这样大型、昂贵的东西的开源解决方案通常很少。其他开源传输模式的选择也很有限，因为似乎大多数大公司都不愿意轻易放弃他们的秘密。然而，像这样的社区给了我们希望，人们将有其他选择来修理他们的汽车，而不必支付太多的钱。
