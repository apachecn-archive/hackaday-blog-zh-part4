# 为朋友打造的电容式鼠标是一个感人的故事

> 原文:[https://hackaday . com/2021/08/30/电容式鼠标为朋友打造，为感人故事打造/](https://hackaday.com/2021/08/30/capacitive-mouse-built-for-a-friend-makes-for-a-touching-tale/)

你鼠标内部的那些小开关可能会被额定为 5000 万次点击或更多，但你的手指可能会比这更快磨损。相信我们——鼠标手臂和/或手疲劳一点也不好玩。如果你从未有过不愉快，试着想象一下不能点击或移动鼠标而没有极度不适。

[![TTP223 touch sensor modules and the modifications necessary for this project.](../Images/4f00bc902b8d79d65aefbd99c3b1979c.png)](https://hackaday.com/wp-content/uploads/2021/08/capacitive-mouse-touch-sensor-mod-cropped-1.jpg) 为了今年的黑客大奖，[【孙膑】为一个患有 ALS 的朋友一起黑了一个电容鼠标](https://hackaday.io/project/181300-capacitive-touch-button-computer-mouse-hack)。它不使用微型开关，而是使用触摸传感器来检测左右点击，并使用 led 来指示点击何时发生。这让我们认为触觉反馈可能很酷，但它可能很快就会过时，甚至更糟的是，你可能过一段时间就习惯了，不再感觉到它。

这种鼠标对于任何条件下行动受限的人来说都是一个很好的选择——ALS、关节炎、扳机指或腕管/肘管综合征。这也将有利于任何想更隐秘地使用鼠标的人，比如在图书馆、小的共享空间或深夜。我们可以看到的唯一缺点是，你要么必须习惯悬停你的手指，要么学会让它们远离电容按钮。否则，你会比你真正想要的更频繁地启动它们。

如果您想要构建其中的一个，您会在 IO 上找到一套很好的指令，其中包括一些必要的小修改，以使 TTP223 电容模块足够敏感，能够检测手指的存在。你真正需要做的是桥接几个焊盘，增加一个电容，并删除 SMD 发光二极管。[孙斌]说这是一个正在进行的项目。在这一点上，他已经得到了一些 beta 测试人员的参与，并计划很快做出一个专用的 PCB。在短暂的演示中勉强过关。

正确的老鼠可以拯救你的四肢，当然。组装一只不同类型的老鼠，你也许能从大象袭击中拯救你的庄稼。
