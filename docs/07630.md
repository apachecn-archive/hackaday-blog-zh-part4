# 袖珍电视现在全天候显示检查频道

> 原文：<https://hackaday.com/2020/08/31/pocket-tv-now-shows-the-inspection-channel-24-7/>

那些小口袋电视在 20 世纪 80 年代和 90 年代是非常酷的小玩意，但今天它们几乎没有用处，至少对于它们观看模拟电视的预期目的来说。(如果有人在为这些东西制作微型数模转换器，请告诉我们。)

现在模拟口袋电视已经过时了，它们终于足够便宜，可以侵入一个有用的工具，比如检查摄像头。[technichenews]找到了一台不错的卡西欧电视和一台合适的模拟针孔摄像头，它也可以进行红外成像。由于相机有 RCA 插头，电视的视频输入是一些早已过时的专有 3.5 毫米电缆，[technichenews]通过将黄色 RCA 线焊接到旧耳机的电缆上，制作了一种新的视频专用电缆。相机的电源来自设置为 12V 的通用壁式电源插座。

这个项目中我们最喜欢的部分是[technichenews]利用电视中可能是最没用的部分——天线——进入恒星的方式。他们的计划是使用相机窥视小型引擎，因此通过将它安装在天线的末端，它将成为一个可伸缩的，球形关节的，全方位的眼睛。休息之后，您可以查看构建视频。

需要一种更快、更简单的方法来近距离观察，而不需要倾家荡产？我们听说那些超薄的耳垢检查相机相当不错。
