# 一种紧凑的应变波齿轮组件

> 原文：<https://hackaday.com/2019/05/20/a-compact-strain-wave-gear-assembly/>

应变波传动是一种在小空间内生产高效率、高传动比齿轮箱的聪明方法。它包括一个外部固定的轮齿环和一个内部柔性的轮齿环，通过一个使轮齿环扭曲的椭圆形转子使轮齿环与外部轮齿环啮合。它们并不便宜，所以[Leo Vu]已经尝试生产一些 3D 打印的应变波齿轮箱，你可以在你的机器人项目中使用。

他创造了三种传动比的变速箱，1:31，1:21 和 1:15。它不是直径 145 毫米、重量超过 1 千克的最微型设备，但我们仍然可以想象它的许多令人兴奋的应用。我们很好奇 3D 打印齿轮能有多坚固，但我们预计你会对中等大小的机器人感兴趣，而不是一级方程式赛车。有一个展示变速箱的视频，我们把它放在了休息区的下面。

这当然不是我们带给你的第一个应变波齿轮组，不止一个 [3D 打印项目](https://hackaday.com/2018/06/19/printing-strain-wave-gears/)为这些页面增光添彩。我们甚至为你[带来了乐高版](https://hackaday.com/2016/12/09/lego-strain-wave-gear-is-easy-on-the-eyes/)。
