# 定制应变仪有助于保持滑翔伞在高空

> 原文:[https://hack aday . com/2020/09/20/custom-strain-gauges-help-keep-paraglider-aloft/](https://hackaday.com/2020/09/20/custom-strain-gauges-help-keep-paraglider-aloft/)

不管他们在飞什么，优秀的飞行员对他们的飞机有一种“感觉”。无论是听到奇怪的声音，还是感觉到明显的振动，他们都能立即知道什么时候出了问题。发展这种第六感有时对保持起飞次数与着陆次数相等的目标至关重要。

同样的事情也发生在非传统飞行器上，比如滑翔伞，失败的惩罚同样很高。飞行员安德烈·班德拉设计的这款滑翔伞绳索张力监控器背后的理念是在高空中远离麻烦。滑翔伞，以及它们的动力兄弟动力伞，看起来有点像降落伞，但实际上最好的描述是一个可充气的翅膀。机翼通过来自前缘开口的空气加压来保持其形状。如果飞行员没有保持正确的迎角，机翼会减压并塌陷，有时会导致可怕的后果。

幸运的是，大多数飞行员最终会产生一种崩溃的感觉，这种感觉是通过连接机翼和安全带的线的张力变化来感知的。[Andre]的“Tensy”在下面的视频中使用了一系列应变仪来观察机翼前缘线条的张力释放，发出声音警报。此外，Tensy 还能捕捉机翼上的线张力数据，用于监控飞机和飞行员的表现。

这里有很多很棒的设计元素，但就我们的资金而言，我们发现轻量级自制应变仪是这种设计的真正瑰宝。这也不是[安德烈]第一次飞到这些页面上了——他的巨型遥控滑翔伞在一月份大受欢迎。
