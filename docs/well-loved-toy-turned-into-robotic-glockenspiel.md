# 深受喜爱的玩具变成了机器人钟琴

> 原文：<https://hackaday.com/2019/02/04/well-loved-toy-turned-into-robotic-glockenspiel/>

如果有一个比“钟琴”更快乐的词被引入英语，我们不知道它是什么。并且[用一堆伺服系统和一个 Arduino](https://www.youtube.com/watch?v=zi-oPoGKWmc) 控制所述仪器让我们同样快乐。

当[莱昂·范·登·布克尔]在旧货店发现一个玩具钟琴时，他知道该做什么——把它销毁。他的第一次尝试是在一对万向伺服系统上安装一个锤子，除了这个深受喜爱的玩具发出的声音质量不佳之外，它还能工作。一次只能有一个音符的事实可能是版本二的灵感来源，版本二看到了从原始基础上移除的音调条，清除了它们有点花哨的油漆，并贴在新的共鸣板上。改进后的乐器配备了八个伺服系统，每个音符一个，每个都有一个 3D 打印的手臂和木槌。Arduino 操作伺服系统，Android 应用程序通过蓝牙控制乐器，因为谁不想用智能手机应用程序控制电子钟琴呢？下面的视频显示，它工作得相当好，即使有几个音符需要一些调整。我们甚至不觉得伺服噪音会让人分心。

没错，我们以前也有过更加出色的机器人钟琴手，但是这个版本的简单有它自己的魅力。
