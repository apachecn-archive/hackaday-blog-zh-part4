# 西瓜数控用的是懒苏珊

> 原文：<https://hackaday.com/2022/10/12/watermelon-cnc-uses-lazy-susan/>

这是一年中世界上某些地方的许多人雕刻南瓜的时候。[Gonkee]正在雕刻一个西瓜，我们假设它是相似的。他决定制造一台数控机床来为他雕刻。不寻常的部分是用两个懒人 Susans 做了一个[旋转雕刻机](https://www.youtube.com/watch?v=na_BnqqMIvg)。你可以在下面的视频中看到结果。

硬件很聪明，有软件可以让你画图，尽管我们希望能处理 gcode 或者切片 STL。这将是一个有价值的附加项目。在甜瓜雕刻机 3000 令人满意地工作之前需要一些迭代。看到一个雕刻工具在两个圆上操作，给了我们很多想法。我们不确定坐骑有多坚固，所以不要计划不做一些改变就雕刻铝，但我们怀疑这是可能的。

话说回来，安装在框架上的激光头可能会很快完成甜瓜的工作，并且不需要太多的机械刚度。然而，这需要一点努力来保持它的焦点。这么多想法可以尝试！

显然，西瓜是一种[流行的黑客媒介](https://hackaday.com/2012/12/17/watermelon-air-boat/)。甚至还有一个[拿着游戏机](https://hackaday.com/2020/08/26/youve-never-seen-this-retropie-emulator-console-watermelon/)。
