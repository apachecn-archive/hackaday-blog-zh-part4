# 用算法机器给彩虹分类

> 原文：<https://hackaday.com/2020/02/09/sort-the-rainbow-with-an-algorithm-machine/>

当你试图了解一个算法是如何工作的，并不总是很容易将正在发生的事情可视化。嗯，也许除了二进制排序，感谢电话簿。教授[thatguyer]是一名计算机科学教师，他希望找到一种方法来帮助他的学生可视化算法的过程，同时掌握他们的资源成本。

[![](img/89cc7786743e4b03553c2973637ba2ce.png) ](https://hackaday.com/wp-content/uploads/2020/01/button-harness.png) [算法机器](http://www.instructables.com/id/The-Algorithm-Machine/)可以使用两个 100 计数的 RGB LEDs 条演示 8 种不同的搜索和排序算法——一个代表整数阵列，另一个创建指向被检查的整数的指示器。

这种功能美也是完全互动的。一旦用户选择了值和算法，并开始了这个过程，他们就可以用旋转编码器加速或减速，或者停下来讨论，然后用那个光滑的三角形播放按钮重新开始。我们特别喜欢控制按钮线束[guyer]创造了保持一切整洁和热插拔。

这种迭代使用 3D 打印面板来赋予 led 形状，但在早期版本中，[那个 guyer]从黄铜管道中切割并打磨了一吨圆，并折叠了从一次性烤盘中切割的许多三角形。这个世界需要更多像那个家伙一样忠诚的老师。这看起来真的像是这些概念的一个方便的教具，我们希望我们在课堂上有一个这样的教具来玩。这是你观看演示的算法:点击休息，按播放，享受。

如果你仍然感到困惑，有[种其他方法可以通过可视化](https://hackaday.com/2018/05/06/algorithms-for-visual-learners/)来理解算法。如果做不到这一点，那就[看看这些匈牙利民族舞蹈演员表演各种算法节奏](https://www.youtube.com/user/AlgoRythmics/videos)。
