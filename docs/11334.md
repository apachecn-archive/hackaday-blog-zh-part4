# 重新充电的除湿机重新投入使用

> 原文:[https://hack aday . com/2021/09/19/充电-除湿器-重新投入使用/](https://hackaday.com/2021/09/19/recharged-dehumidifier-put-back-into-service/)

对于普通消费者来说，维修成本相对较低的家用设备，比如微波炉和电视，在经济上是不可行的。当修理费很可能比买一个新型号的成本还高时，你很难责怪他们。幸运的是，对于我们这样的人来说，这意味着你经常可以在二手市场上找到便宜甚至免费的电器，只需一点故障排除和一些备件就可以重新上线。

就拿[无功能除湿机【HowToLou】最近碰到的](https://www.youtube.com/watch?v=FhaZJTZJMIg)来说吧。如果你尝试，你可能找不到一家愿意为这些事情费心的专业修理店，但正如他在下面的视频中所示，这并不意味着 DIY'er 不能通过一些可能的故障模式运行，并使设备恢复运行。作为奖励，他还向观众展示了典型的压缩机除湿机是如何工作的。

[![](../Images/d439dab908722c5a93e5b686bfb26f6f.png)](https://hackaday.com/wp-content/uploads/2021/09/dehumidifierfix_detail.jpg)

The failed thermal switch.

除了收集室缺水之外，这台除湿机出问题的第一个迹象是压缩机没有运转。经过更仔细的检查，[HowToLou]确定热切断开关出现故障，并卡在打开位置。幸运的是，它有一个可见的零件号，所以他可以订购一个替代品，同时，他所要做的就是切断电路的开关，直接连接压缩机的电源。

不幸的是，即使压缩机在运转，也没有收集到水。注意到蒸发器盘管没有变得很冷，[HowToLou]认为该装置可能制冷剂不足。通常这些系统不打算再充电，但通过一个聪明的穿刺龙头阀，你可以在低压侧添加一个快速连接端口。这种特殊的除湿机碰巧装满了汽车空调系统中使用的 R134a，所以他去了一趟汽车配件商店，买了一罐制冷剂，还带了一个方便的压力表。

在变得多汁后，[HowToLou]显示线圈上形成了冰，大量的水被倒入水箱。汽车空调加油罐通常包含一些物质来阻止或减少系统中的泄漏，所以希望这将成为一个长期的解决方案。这可能不是我们见过的最复杂的除湿机维修，但肯定是最简单的。如果你看到这些东西躺在路边，也许你应该把它捡起来，看看是什么让它不舒服。
