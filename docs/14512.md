# 这个简单的媒体播放器将启发初学者，并邀请实验

> 原文:[https://hackaday . com/2022/08/16/this-simple-media-player-will-inspire-beginners-and-invite-experimentation/](https://hackaday.com/2022/08/16/this-simple-media-player-will-inspire-beginners-and-invite-experimentation/)

虽然这在几十年前被认为是科幻小说，但在一个适合你口袋的小平板电脑上观看几乎任何电影或电视节目的能力在今天都不是什么大事。但是对于一个电子初学者来说，能够组装一个像这样的便携式视频播放器将是非常令人兴奋的，甚至可能成为进入更大的电子设计世界的大门。

为了寻找灵感，YouTube 上 *Super Make Something* 的【Alex】向[寻求帮助，这是我们在一月份推出的 Rickrolling 钥匙链媒体播放器](https://hackaday.com/2022/01/10/hitclips-custom-cartridge-hack-will-never-give-up-let-down-or-turn-around/)。他的播放器更大，功能更强，PCB 设计允许播放器以多种配置构建，从纯音频到全视频和 LiPo 电池。该播放器的核心是一个 ESP32 模块，带有一个音频放大器和扬声器，外加一个 1.8 英寸的 LCD 屏幕，带 SD 读卡器，用于存储媒体文件。添加一些控件、开关和一点代码，您就可以快速播放媒体文件了。建立信息和演示视频如下。

这可能是一个简单的设计，但我们觉得这才是重点。[亚历克斯]已经尽力使这成为一个尽可能平易近人的建设。所有的部件都很便宜，很容易获得，将它们组装在一起所需的技能也很少——可能的例外是焊接 ESP32 模块，它没有城堡形边缘端子。对于初学者来说，通过混合几个模块就能得到一个可用的媒体播放器将是不可思议的，而且它在之后仍然非常容易被破解的事实只是锦上添花。
