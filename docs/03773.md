# 为预算有限的黑客设计的外骨骼手臂

> 原文:[https://hack aday . com/2019/08/02/an-exoskeleton-arm-for-a-hacker-on-a-budget/](https://hackaday.com/2019/08/02/an-exoskeleton-arm-for-a-hacker-on-a-budget/)

无论是出于机甲套装赋予超人力量的梦想，还是为了肌肉功能受损者的机动性，动力外骨骼在工程师中发挥了奇妙的魅力。通过机器增强人体来实现伟大目标的想法受到了阻碍，尽管任务的难度很大，但小到足以舒适佩戴的致动器和电源是一个不容易克服的重大挑战。这是一个让[ 克里斯詹·贝斯着迷的主题，因为在他年幼时看到他的祖母挣扎着举起东西，[他展示了一个工作动力外骨骼手臂](https://hackaday.io/project/166841-affordable-exoskeleton-arm-exoarm)来证明他的想法。

在低技术含量的建筑中，用手工工具和钻床在铝片和木块上做这个练习很棒。动力来自汽车挡风玻璃刮水器电机，电力来自连接在该设备线束上的一个巨大的脂肪。肘关节中有一个反馈电位计，Arduino 在一对安装在手套上的按钮的指导下监督操作。在下面的视频中看到它举起一辆自行车当然令人印象深刻，尽管我们想知道它的重量会如何影响肌肉功能低于平均水平的人。

像这样的项目非常值得一看，因为有可能有人会通过建造这样的项目得到帮助。然而，总有一点需要注意，因为最好的解决方案来自那些需要它们的人，而不是那些仅仅认为他们有解决方案的人。在过去的几年里，我们已经在这里写了关于工程师救星陷阱的文章。

这不是我们看到的第一个假肢，我们报道过英国的一个黑客空间为当地的一个年轻人打印了一个。
