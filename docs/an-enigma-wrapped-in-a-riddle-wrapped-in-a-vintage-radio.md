# 包裹在一个老式收音机里的谜语中的谜

> 原文：<https://hackaday.com/2018/09/30/an-enigma-wrapped-in-a-riddle-wrapped-in-a-vintage-radio/>

谜题盒是黑客攻击的绝佳机会。你可以从原本用于其他用途的盒子开始。你要设计电路和控制，为玩家提供一个复杂的难题。你可以为解决问题的人提供丰厚的奖励。哈拉德博士为一款原创派对游戏设计的梦境传输盒拥有所有这些元素。

这个盒子是一台破旧的 1948 年国家 NC-33 业余无线电，是在多次竞标失败后从易贝购买的。除了扬声器，大部分都被移除了。电子设备是基于 Arduino 的，所以大部分的智能都是以代码的形式存在的。电位计和开关为玩家提供了输入密码的机制。当输入正确的密码时，一个继电器会触发一个外部烟雾机，并开启一个照亮派对舞会的激光器，奖励胜利者。当然，还有声音效果和录音信息。

当我们说字谜盒是伟大的黑客时，我们不是在开玩笑。这里有[一个点燃烟花](https://hackaday.com/2015/07/23/puzzle-box-is-rigged-to-blow/)，[一个仅由分立元件制成](https://hackaday.com/2014/05/14/electronic-puzzle-box-uses-only-discrete-components/)，还有[一个基于情人节的](https://hackaday.com/2013/02/18/valentines-puzzle-box-makes-you-work-for-whats-inside/)让你的另一半为他们的礼物而工作。
