# 推出光滑的旋转相位转换器

> 原文：<https://hackaday.com/2018/08/06/rolling-out-a-slick-rotary-phase-converter/>

家庭机械师经常可以在二手工业设备上找到很好的交易，许多南本德车床或布里奇波特铣床已经跟随某人回家。接下来就是给它插上电源的时候了，新主人发现店里找不到给新野兽供电所需的三相插头。因此开始哭泣和咬牙切齿。

幸运的是，[手工极限]在车间电力方面走在了前面，[建造了一个旋转相位转换器来为他的机器](https://www.youtube.com/watch?v=uGKZ51zg0X8)供电。工业通常在三相交流系统上运行，主要是因为三相电机比等效的单相电机更高效、更紧凑。但住宅电力服务要么是分相的，要么在[手工极端]的总部所在地英国是单相的。旋转相位转换器是一种可以产生缺相的机电设备，本质上是一种三相电机，可以在一个绕组上运行，并在其他绕组上产生缺相。为此，它需要一些支持控制电路，如定时器和接触器，以便在电机启动时切换绕组连接，以及用于电机启动和平衡各相电压的电容器。控制装置安装在 DIN 导轨上，并整齐地连接到一个外观漂亮的控制面板上。所有东西都装在一个坚固的外壳里，这个外壳足够大，可以用作移动工具车。这是一个非常好的工作——在下面的视频中观看整个构建过程。

如果你对配电感兴趣，我们有一本涵盖基础知识的入门书。如果你在机床市场上，[【奎因】的机床买家指南](https://hackaday.com/2018/03/21/lathes-foreign-and-domestic/)将让你决定一台三相机器是否值得额外的努力。
