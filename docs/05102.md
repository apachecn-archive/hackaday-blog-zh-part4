# 时钟里的小气泡

> 原文:[https://hackaday.com/2019/12/20/tiny-bubbles-in-the-clock/](https://hackaday.com/2019/12/20/tiny-bubbles-in-the-clock/)

当[DonHo]唱到微小的气泡时，他可能没有想到它们嵌在甘油里。但那就是[ShinodaY]的[时钟](https://www.instructables.com/id/Glowing-Air-Bubble-Clock-Powerd-by-ESP8266/)中的气泡所在之处。粘性液体更好地保持气泡，使时间更容易读取。你可以在下面的视频中观看放松的展示。

操作原理很简单，让我们想起了一个反向的俄罗斯方块游戏。底座上的电磁阀释放气泡，形成一排显示器。上升的气泡为下一排腾出了空间。显示器的列数与底部的出风口数一样多。分隔气泡像素就像调整气泡之间的时间一样简单。

由于有一个 I/O 扩展器，ESP8266 控制着整个系统。一些新像素发光二极管使整个东西看起来更酷。

这是时钟的第二个版本。第一个版本(见下面的第二个视频)使用水，我们认为你会同意甘油使所有的差异。

该项目既是水族馆工程，也是电子学工程。我们还想知道你还能像这样展示什么？也许是一些粗糙的图形或推文？或许把它作为一种有趣的游戏形式会很酷，尤其是如果它的规模显著扩大的话。

比如我们记得有一个[超大气泡显示](https://hackaday.com/2014/08/14/bubble-displays-are-increasing-in-resolution/)(注:出站链接[变了](http://www.taomc.com/pipe-dream))。你甚至可以做一个 [3D 显示器](https://hackaday.com/2014/01/09/3d-bubble-display/)——有点像。
