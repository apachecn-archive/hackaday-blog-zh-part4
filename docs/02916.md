# 为这款 CRT 掌上游戏机竖起大拇指

> 原文：<https://hackaday.com/2019/05/06/thumbs-up-for-this-crt-handheld-gaming-console/>

尽管视频游戏图形取得了进步，但可以肯定地说，我们不会很快看到老派 8 位游戏的任何衰落。对一些人来说，这是怀旧，对另一些人来说，这只是进入游戏开发本身的一个伟大而简单的第一步。对于[gocivici]来说，当他建造这种便携式摄像机风格的一键游戏控制台时，两者兼而有之。

以运行 PICO-8 的 Raspberry Pi Zero 为核心，[gocivici]抢救了一台旧摄像机的取景器用于显示，这样就把它变成了另一种手持控制台。对于完全符合人体工程学的处理，一个单一的拇指操作按钮作为唯一的控制选项。这当然使得重用需要更多输入选项的现有游戏变得有点困难，所以他和一些朋友[决定自己写一个合适的游戏](https://www.lexaloffle.com/bbs/?tid=33071)，希望其他人可以效仿。

不幸的是，我们没有看到很多项目使用这些旧的摄像机取景器，考虑到现代的液晶显示器和有机发光二极管选项，这并不奇怪，但这些微小的 CRT 有一些有趣的东西。因此，如果你想看到更多，可以看看这个微型雅达利显示器和几年前的 T2 DIY 夜视单片眼镜。为了保护你的眼睛安全和健康，[【gocivici】也会保护你](https://hackaday.com/2019/02/08/ya-only-get-one-set-of-eyes/)。
