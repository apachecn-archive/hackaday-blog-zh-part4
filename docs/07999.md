# 制作适合 2020 年的万圣节服装

> 原文:[https://hack aday . com/2020/10/06/making-a-Halloween-costume-fit-for-2020/](https://hackaday.com/2020/10/06/making-a-halloween-costume-fit-for-2020/)

全国各地的家长们都在考虑如何应对即将到来的不给糖就捣蛋的季节。社交距离、无接触款待或在家简单地做这些措施都被认为是乐趣和安全的平衡。[BuildXYZ]已经决定加入今年的挑战，并在他的男孩们的服装中加入面具。

它始于一个 3d 打印的面具，打印成两半，并用硅填缝剂和 N95 过滤材料密封在侧面的入口和出口孔中。面具的真正魔力是安装在前面的小有机发光二极管屏幕，它与面具内的小驻极体麦克风一起工作。通过对麦克风进行采样并应用滚动平均值，Arduino Nano 可以确定显示器上绘制的嘴是张开还是闭合。腰带夹上的一个小电池组(带有一个在屏幕上闪烁“不给糖就捣蛋”的按钮)为整个系统供电，可以很容易地隐藏在斗篷或服装下。

这不是我们今年万圣节看到的第一个黑客，比如这个[社交距离糖果滑梯。](https://hackaday.com/2020/09/25/candy-slide-keeps-halloween-spooky-and-socially-distant/)我们有一种感觉，随着这个月的流逝，人们开始将他们的聪明才智应用到这个季节，还会有更多。
