# 如何开始使用 Fadecandy 和 led

> 原文：<https://hackaday.com/2019/10/05/how-to-get-started-with-fadecandy-and-leds/>

互联网上充斥着数百万令人惊叹的 LED 项目，为此，我们都非常感激。对于那些在黑客/制造者矩阵之外的人来说，可能很难知道如何实现这样的构建。不要担心，因为[Amy Goodchild]已经整理了一份初学者指南，用 Fadecandy 和 Processing 制作漂亮的 glowables。

Fadecandy 是一个专门为艺术目的驱动 WS2812B LEDs 而设计的平台。这使得用户可以专注于事物的视觉方面，而不会陷入选择正确的微控制器和选择适用的库的麻烦中。它与 Processing 结合使用效果很好，Processing 是一款为尝试视觉艺术的程序员设计的软件。通过 USB 链接，任何由处理绘制的图形都可以映射到附属于 Fadecandy 控制器的 led 上。

[Amy]很好地解释了如何做所有需要做的事情，从购买正确的设备，到连接所有设备，然后使用正确的软件使其正常运行。如果你曾经想用一吨的发光二极管建造一个大的浮华的项目，这将是一个很好的开始。
