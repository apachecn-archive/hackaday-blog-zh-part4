# 致动器打开干燥碗碟的门

> 原文：<https://hackaday.com/2020/11/18/actuator-opens-the-door-to-drier-dishes/>

大多数时候，洗碗机在洗碗甚至漂洗盘子方面都很棒。它们容易失效的地方是干燥部分。不知何故，这些东西干得足够热，以至于可以扭曲炻器盘子，但不会干到当你打开门的时候，东西实际上是干的。归咎于空气流动不足。

理想情况下，洗碗机周期在晚饭后不久开始，这样就可以在睡觉前完成并打开。但是如果你这样做，你就会错过所有的深夜小吃和偶尔的酒杯。等到睡觉的时候再启动它，它必须在里面有湿气的情况下坐几个小时。[显然，答案是在循环结束时聆听胜利的哔哔声，并使用缓慢但有力的致动器将门推开](https://hackaday.io/project/172112-automatic-dishwasher-opener)。

[Ivan Stepaniuk]正在用麦克风监听洗碗机的频率，用可靠的 LM386 放大它们，并使用 STM32 蓝色药丸处理音频。[Ivan]计划为物联网集成一个 ESP8266 板，大概是为了在门成功打开时收到通知。休息后请欣赏演示。

是的，洗碗机是伟大的，直到他们不是，一些小零件打破。但是，既然你可以打印一个，为什么还要花钱买一个新的洗涤剂盒盖呢？
