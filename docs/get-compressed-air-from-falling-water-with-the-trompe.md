# 用长号从下落的水中获取压缩空气

> 原文：<https://hackaday.com/2020/01/17/get-compressed-air-from-falling-water-with-the-trompe/>

如果你和我们一样，理解早期工业革命的过程和方法需要一些手势。以鼓风炉为例，它依靠稳定的压缩空气供应来给火添加燃料，并提供从矿石中炼铁所需的氧气。在有电之前，空气究竟是如何被压缩的？我们假设它来自一组由水轮驱动的风箱，当然这种方法被使用了，但事实证明还有另一种从水中获得压缩空气的方法:伸缩仪。

正如《实用工程》的 Grady 在下面的短片[中解释的那样，这个望远镜是一个聪明的装置，用来产生稳定的高压压缩空气供应。为了演示这一过程，他拿出看似取之不尽的透明丙烯酸管道建造了一个小型错视仪。这个想法是利用一系列管子周围的水来创造一个局部真空，并夹带气泡。气泡被水的湍流拉下垂直管，然后进入水平部分，在那里流动变得均匀。气泡上升到水平管的顶部，在那里它们被另一个垂直管排出，脱气水继续进入第二个垂直部分，其高度决定了储存空气的压力。它很巧妙，不需要电力和移动部件，并且可以很好地扩大规模——[ Grady]讲述了一个关于一个为加拿大的矿井提供商业压缩空气的错视仪的故事。](https://www.youtube.com/watch?v=uvf0lD5xzH0)
