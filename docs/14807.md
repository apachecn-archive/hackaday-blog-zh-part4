# 刮建钢筋混凝土挖掘机是一个模型制作的绝技

> 原文：<https://hackaday.com/2022/09/16/scratch-built-rc-excavator-is-a-model-making-tour-de-force/>

一些项目对细节的关注程度让你瞠目结舌。这个临时搭建的遥控模型挖掘机不仅细节令人惊叹，而且制作它所用的材料和工具也令人惊叹。

我们不得不诚实地说，几个星期以来，我们一直在关注[王哈]在这方面的进展。下面的第一个视频是一个完整的项目参观，这是一个精心忠实于原来的卡特彼勒 390F 履带式挖掘机。虽然这令人印象深刻，但你必须看看下面第二个视频中从制作轨道开始的构建过程。大部分模型的原材料是普通的灰色 PVC 管，它被切片并切成平板，用临时装配的台锯切成小块，然后加热成型制成弯曲的块。查看[完整的播放列表](https://www.youtube.com/playlist?list=PLbJ9MRuGCKMnjmQm4WTzRCLRZFekaArlJ)以获得大量的制造乐趣，如微型铰链和工作插销。

我们不可能对[王哈]的工艺赞不绝口，但这不是我们喜欢的全部。这里有大量有用的提示，并且为更实际的构建提供了大量的思考素材。我们正在考虑操作动臂、铲斗和铲斗的全套工作液压缸，以及伺服操作的液压控制阀。当然，所有这些都是从零开始制作的，而且大部分是由聚氯乙烯制成的。对于电动马达或线性致动器不能满足需求的项目，请记住这一点。

如果这种施工技术对你来说似乎很熟悉，那可能是因为我们在六月份展示了一个由类似加工的 PVC 管制成的工具箱。
