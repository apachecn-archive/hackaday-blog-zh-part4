# 公约发挥口袋妖怪对巨型彩色游戏男孩服装

> 原文：<https://hackaday.com/2021/09/29/convention-plays-pokemon-on-giant-color-game-boy-costume/>

标准的角色扮演很有趣，但是崇拜者除了上下打量你和挑剔细节还能做什么呢？互动角色扮演，这就是它的价值所在。[Jaryd Giesen]知道这一点，并设法在几天内完成了一套彩色游戏服。

最初的计划是在手臂上使用一个小型投影仪，就像那些帮助你看到屏幕的蠕虫灯之一，但[Jaryd]最终得到了一个二手显示器，并将其绑在胸前。然后他从那里拿走了剩下的部分。在所有这些纸板的下面，事情非常简单:有一个运行 RetroPie 仿真器的 Raspberry Pi，一个处理输入的 Pico，和两节电池——一节结实的 12，000 mAH 电池用于显示器，一个常规的电源组用于 Pi 和 Pico。

正如你将在休息后的构建和演示视频中看到的，近 100 人停下来按[Jaryd]的按钮。他们并没有在游戏中走得很远，但看起来他们很乐意尝试。

既然我们仍然穿着疫情，你可能会考虑在今年的万圣节服装中加入面具。只是一个想法。
