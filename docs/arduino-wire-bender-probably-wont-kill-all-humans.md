# Arduino 弯丝机可能不会杀死所有人类

> 原文：<https://hackaday.com/2019/09/28/arduino-wire-bender-probably-wont-kill-all-humans/>

你想自己做弹簧吗？是的，我们也是这么想的。好吧，吹掉备用 Arduino 上的灰尘，继续读。几个月前，我们让你知道著名的电路雕塑家[jiří·普劳斯]正在制作一台精密的弯线机来帮助他磨练技艺。现在是真的，很壮观，[完全开源](https://www.instructables.com/id/Spring-Making-Wire-Bending-Machine)。

除了这个“duino ”,你还需要一个 CNC 保护罩和几个 NEMA 17 步进器——一个用来送电线，一个用来帮助弯曲电线。在被弯曲或卷成弹簧之前，金属丝必须非常直，所以从线轴架上下来的金属丝在进入折弯机之前要通过两组辊子。

[Jiří]'s 这次建造的主要目标是精确，这一点我们完全可以做到。如果你要造一台机器为你做一些事情，理想情况下，它也应该比你一个人做得更好。这是他的第二个目标，使这个建筑如此不同寻常。[Jiří]希望用普通的硬件和 3D 打印机就能轻松建造。每一部分都设计成无支撑印刷。跳过休息时间观看构建视频。

你也可以[在车床](https://hackaday.com/2016/05/29/making-springs-at-home/)上制作自己的弹簧，或者[用黑出来的 g 代码](https://hackaday.com/2018/11/14/3d-print-springs-with-hacked-gcode/)打印出来。
