# 可印刷碳化物开辟了有趣的可能性

> 原文：<https://hackaday.com/2022/04/15/printable-carbide-opens-up-interesting-possibilities/>

总部位于瑞典的大公司山特维克(Sandvik)显然已经生产硬质合金很长时间了——据他们称，自 1932 年以来。众所周知，这种材料非常耐磨。现在该公司表示，他们已经有了 3D 打印材料的流程。你可以在下面看到一段关于这种新材料的视频。

如果你没有遇到过这种材料，它本质上是金属中结合的细小碳化物颗粒。你会发现这种材料广泛用于切割工具。“自由设计从未如此艰难”的口号既聪明又令人困惑，但我们接受了他们的观点。

该工艺或多或少类似于其他金属粘合剂技术。碳化钨和钴的粉末与胶水混合形成一个生坯，你仍然需要烧制它来完成零件。

你能做什么样的东西？这是山特维克的一位工程师的话:

> 例如，在拉丝过程中，生产率通常受到在保持质量的情况下拉丝速度的限制，而拉丝速度又取决于拉丝模的温度。几十年来，人们一直在试图解决这个问题，但难度极大。一个 3D 打印的冷却金属丝笔尖就是这个谜题的答案。由于我们的材料和专有工艺，从第一个基本草图到完全烧结的产品仅用了四天时间。

不要计划在你的 Ender 3 上安装硬质合金细丝。这毕竟是金属材料。然而，3D 打印可以提供传统方法难以获得的几何形状。因此，即使你不得不求助于专业的 3D 打印店，也很高兴知道你可以在这种超硬材料中进行创作。

与使用塑料相比，金属印刷有一系列不同的问题。如果你真的想让你目前的打印机做金属，[它可以](https://hackaday.com/2021/02/25/3d-printering-makerbot-prints-stainless-you-do-half-the-work/)，但你必须作弊一点。或者试试[电镀](https://hackaday.com/2021/03/01/electroplating-3d-printed-parts-for-great-strength/)。
