# 改装的汽车让学步的孩子四处玩耍

> 原文:[https://hack aday . com/2020/02/10/converted-car-let-幼儿-tool-around/](https://hackaday.com/2020/02/10/converted-car-lets-toddlers-tool-around/)

几年前，特拉华大学的一名教授启动了一个名为 Go Baby Go 的项目。它旨在为残疾儿童带来乐趣和实惠的移动性。这个想法是修改电动轮汽车，让残疾儿童更容易操作，并在这个过程中教会尽可能多的人如何操作。[South Eugene Robotics Team] [更进一步](http://www.instructables.com/id/GoBabyGo-Make-a-Joystick-controlled-Ride-on-Car/)，用一个控制两个 Arduino Nano 电机的操纵杆代替方向盘。

在第一种情况下，你用一个按钮代替脚踏板。该计划还要求使用 PVC 框架、高背座椅和安全带，以使其更加安全。最终结果是一个有趣的乘坐，孩子们可以控制自己，功能很像电动轮椅，但更实惠。它的额外好处是成为其他孩子的有趣话题，而不是一件怪异可怕的事情。

他们还用 5 英寸的脚轮取代了前轮，因为能够转圈真是太棒了。他们的项目非常详细地展示了如何进行整个转换，从一辆标准的代步车开始，该车带有一些所需的组件。汽车过去休息，看看一个非常快乐的孩子在消防车周围工具短演示。

如果这些孩子变得太野，他们将需要[牵引力控制来控制这些东西](https://hackaday.com/2019/06/23/this-upgraded-power-wheels-toy-is-powerful-enough-to-need-traction-control/)。
