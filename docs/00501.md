# 周五黑客聊天:GNU 电台

> 原文:[https://hackaday.com/2018/08/29/friday-hack-chat-gnu-radio/](https://hackaday.com/2018/08/29/friday-hack-chat-gnu-radio/)

当人们发现廉价的 USB 电视调谐器是可以成为 SDR 的功能性硬件时，软件定义无线电受到了广泛的欢迎。是软件让这成为可能，说到 SDR 软件，没有比 GNU Radio 更好的工具了。在本周的黑客聊天中，我们将与这款出色的软件工具背后的一些人坐在一起，向他们请教。

我们本周黑客聊天的嘉宾是德里克·科泽尔和内特·坦普尔，GNU 无线电项目的官员。他们也是今年 GNU 无线电会议的组织者。社区经理、PyBOMBS 维护者和 GNU 无线电基金会官员 Martin Braun 也将加入 Hack Chat。

GNU Radio 可能是任何软件定义的无线电工具链中最重要的部分。该软件提供信号处理模块来实现软件定义无线电。GNU radio 就是你拿一个电视调谐器 USB 加密狗，从卫星上获取图像。您可以使用它进行模拟，GNU Radio 被业余爱好者、学者和工业界人士广泛使用。

在本周的黑客聊天中，我们将谈论 GNU Radio。你能用它做什么？界面*真的是受 MaxMSP 启发的*吗？所有这些和更多的[在本周的黑客聊天](https://hackaday.io/event/160465-gnu-radio-hack-chat)。

*   使 GNU 无线电工作的各种硬件
*   编写模块的核心过程
*   GNU Radio 即将推出的功能
