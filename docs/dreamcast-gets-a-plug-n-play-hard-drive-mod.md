# Dreamcast 获得一个即插即用的硬盘模块

> 原文：<https://hackaday.com/2019/07/29/dreamcast-gets-a-plug-n-play-hard-drive-mod/>

对于世嘉来说，Dreamcast 是一个值得骄傲的时刻，至少在最初，它是推出新一代产品的第一台主机。不幸的是，这并没有转化为大规模的销售，插头被拉远早于预期。然而，这款游戏机至今仍有一群忠实的粉丝，他们继续修补硬件。【DreamcastChannel】就是其中之一，[并拼凑了一个俏皮的即插即用的硬盘 mod。](https://www.youtube.com/watch?v=Lizu54URYmE&feature=youtu.be)

mod 基于早期的工作，包括手动将 IDE 电缆的 44 条线焊接到主 Dreamcast 主板上。这使得 IDE 硬盘可以整齐地安装在外壳内，但[DreamcastChannel]知道还有可能做得更好。

从头开始，GDROM 光驱组件被拆除，只剩下金属外壳和 PCB。然后将硬盘的 IDE 电缆焊接到 PCB 上的焊盘上。3D 打印支架用于将硬盘固定在金属外壳上。这使得整个组件可以整齐地插入 Dreamcast，使用 GDROM 的原始连接器。

这是一个让在 Dreamcast 中安装硬盘变得干净整洁的方法。结合被黑的 BIOS 和 Dreamshell，它让玩备份游戏变得轻而易举。我们以前也见过很多 Dreamcast 的黑客——[VMU 经常是引人注目的重要候选人](https://hackaday.com/2017/07/17/completely-owning-the-dreamcast-add-on-you-never-hadcompletely-owning-the-dreamcast-add-on-you-never-had/)。休息后的视频。
