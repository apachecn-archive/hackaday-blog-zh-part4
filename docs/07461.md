# 迷你星球大战街机让你在旅途中炸毁死星

> 原文:[https://hackaday . com/2020/08/12/miniature-star-wars-arcade-let-you-blow-up-the-death-star-on-the-go/](https://hackaday.com/2020/08/12/miniature-star-wars-arcade-lets-you-blow-up-the-death-star-on-the-go/)

如果你对下午在当地商场度过的时光有美好的童年记忆，那么你偶尔会做白日梦，想找到一个旧橱柜，把它放在客厅里。但是这些机器的尺寸、成本和稀有性使得实际上拥有一台对大多数人来说不切实际。

虽然这个由[杰米·麦克尚]创作的经典*星球大战*街机游戏的功能齐全的 1/4 比例复制品可能不是原版的完美替代品，但不可否认的是，它会更容易通过你的前门进入。几乎这部 1983 年的标志性机器的每一个方面都被精心再造，甚至连接受微型硬币的投币口都是如此。坦率地说，如果他只做了一半的细节工作，这个构建就足够令人印象深刻了，但是我们当然不会抱怨他做了额外的工作。

[![](../Images/5af31453dd3713c380e99d2c81f7d30f.png)](https://hackaday.com/wp-content/uploads/2020/08/swmini_detail.png) 【杰米】为了这个造型，很大程度上依赖于树脂 3D 打印部件，而且理由充分。很难想象他是如何利用传统制造技术为他的橱柜生产出一些微小的工作部件的。这款游戏的标志性控制手柄和硬币接收器机制真是令人难以置信的小型化壮举，证明了使用相对便宜的工具在 DIY 水平上的可能性。

橱柜本身是由中密度纤维板切割而成，使用的平面图是从实物中适当缩小的。在里面，你会发现一个树莓 Pi 3 型号 A+运行 RetroPie 直接连接到一个集成放大扬声器的 4.3 英寸液晶显示器的背面。[Jamie]正在使用 Arduino 来处理与光学硬币检测器和控制器的接口，光学硬币检测器和控制器通过 USB HID 与 Pi 通信。他甚至还添加了一对 3000 毫安的 LiPo 电池组和一个专用的充电控制器，这样你就可以在旅途中炸毁死星。

还是觉得你的公寓装不下一个？不用担心，早在 2012 年，我们实际上看到有人[以 1/6 的比例](https://hackaday.com/2012/04/26/micro-star-wars-arcade-game-is-a-work-of-art/)重建了这个相同的橱柜。
