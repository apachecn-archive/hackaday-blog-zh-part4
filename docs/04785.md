# 集 Arduino 和 Enigma 于一身

> 原文:[https://hack aday . com/2019/12/12/an-arduino-and-an-enigma-all-rolled-into-one/](https://hackaday.com/2019/12/12/an-arduino-and-an-enigma-all-rolled-into-one/)

这个黑客想设计一个[英格玛机器模拟器](https://hackaday.io/project/165559-at-mega-enigma-a-2560-pro-mini-enigma-simulator)已经有一段时间了，但是直到他们意识到有一个具有剩余 I/O 的紧凑 Arduino，他们才迈出了这一步。

原木在机器上经历各种变化。涵盖了从与原始插件板相似的插件板变化到 16 段 LED 测试仪的所有内容。在其中一个帖子里，你甚至可以看到它解码了一条真正的 U 艇信息。

早期版本采用极具吸引力的激光切割外壳，但最新设计采用了更加优雅的外壳解决方案。模拟器使用 16 段显示屏和按键瞬时按钮。其核心是一款 2560 Pro mini。这篇文章包含了很多关于谜背后的密码的细节，读起来很有趣。有趣的是，PCB 是用很多人又爱又恨的 EDA 软件 [Fritzing](https://hackaday.com/2019/04/30/fritzing-is-back-and-this-time-its-written-in-javascript/) 设计的。

我们喜欢这个项目中的工艺和关注，可以看到它在设计周期中变成一个非常有吸引力的套件。
