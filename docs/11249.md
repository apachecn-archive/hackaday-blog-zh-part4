# 在 FreeCAD 中制作的高度可配置的开源显微镜

> 原文:[https://hack aday . com/2021/09/09/09/highly-configurable-open-source-microscope-cooked-up-in-free CAD/](https://hackaday.com/2021/09/09/highly-configurable-open-source-microscope-cooked-up-in-freecad/)

当你作为一名对 3D 打印和编程感兴趣的医学组织病理学专家，你会得到什么？你得到了一个完全烘焙的开源显微镜，特别是[便携式可升级模块化经济型](https://github.com/TadPath/PUMA)(或 PUMA)，就是这样。这不是玩具显微镜。通过将随处可得的现成电子设备、一两磅灯丝和少量高质量光学部件结合起来，PUMA 很可能创造出我们所品尝过的最好的开源显微镜体验之一。

GitHub 用户[TadPath]是一名医学病理学家，显然对如何制造一台伟大的仪器略知一二，所以看到这个美味的项目以如此完整的方式展示出来，我们真的很高兴。很多时候，我们研究一个高调的项目，只找到一堆 STL 文件和一些难以找到来源的特殊部分。但不是在这里。这是特意设计的，几乎任何一个拥有 3D 打印机和易贝账户的人都可以建造它。

该项目目前尚未获得医学诊断用途的认证，但这可能只是一个金钱和时间的问题。教育和研究的价值(尤其是在发展中国家)怎么强调都不为过。

![](../Images/c6994ceae5d325b0e3cb5d708c4b7775.png)

A small selection of the fixed and active aperture choices

模块化允许多种配置，从简单的环境光照明，单个物镜，非常适合在无电的野外使用，到带有基于 TFT 的空间光调制器的三目镜设置，支持先进的方法，如纹影相位衬度(例如，允许活细胞内流体流动的可视化)和平视显示器，用于对样品进行测量。此外，PUMA 经过专门设计，可在野外快速轻松地进行分解，这有助于忙碌的研究人员外出工作。

GitHub repo 拥有您构建自己的配置和适当的附加组件所需的所有细节，从 CAD 文件(FreeCAD source，因此您可以将其重新混合到您的核心内容中)到采购零件的详细物料清单。
