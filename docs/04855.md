# 菲尔德基特是 2019 年 Hackaday 奖的大奖得主

> 原文:[https://hackaday . com/2019/11/16/field kit-is-the-grand-prize-winner of-the-the-2019-hackaday-prize/](https://hackaday.com/2019/11/16/fieldkit-is-the-grand-prize-winner-of-the-2019-hackaday-prize/)

FieldKit 是一个开源的模块化传感器系统，用于在恶劣环境下进行研究，刚刚被评为 2019 年 Hackaday 奖的大奖得主。在这一为期九个月的全球工程计划中，获得“最佳产品”的第一名和称号的奖励是 125，000 美元。在今晚于加州帕萨迪纳举行的 Hackaday 超级大会期间举行的 Hackaday 颁奖典礼上，还提名了另外五名最高获奖者和五名荣誉奖。

今年的 Hackaday 奖侧重于产品开发。从一个好的想法和一个工作原型，参赛者被鼓励重复他们的 UX，工业设计，人体工程学，软件和机械计划，因为他们致力于一个既可制造又满足用户需求的产品。在 20 名决赛选手中，前 11 名如下。作为今年计划的一部分，已经分发了超过 200，000 美元的现金奖励，数千名硬件黑客、制造商和艺术家竞相创造更美好的未来。

![](../Images/a0c2b1389d93521397db8482167a950c.png)

## [最佳产品:现场工具包](https://hackaday.io/project/26354-fieldkit)

菲尔德基特是 2019 年 Hackaday 奖的大奖得主。[这款多功能、加固型传感器套件](https://hackaday.io/project/26354-fieldkit)专为困难环境中的研究项目而设计。通过在定制设计的传感器和现成的系统之间取得平衡，FieldKit 采用模块化概念(包括硬件和软件)并使设计开源，从而解决了成本和可用性问题。FieldKit 被命名为“最佳产品”,因在今年的奖项中拔得头筹而获得 125，000 美元的奖励。

进行科学研究不应该要求你也有电子和计算机工程的背景。FieldKit 已进行了广泛的 UX/用户界面测试，因此硬件易于设置，使用模块满足您的传感器和通信需求，以及网站、应用程序和导出选项，使收集和利用数据成为直观和无挫折的体验。作为一个开源项目，设计自己的定制硬件模块的研究人员也被邀请分享这些设计，以造福他人。

### [带有 CV、BCI 和 EMG 的 3D 打印假体](https://hackaday.io/project/165075-3d-printed-prosthesis-with-cv-bci-and-emg)

这个[义肢有自己的思维](https://hackaday.io/project/165075-3d-printed-prosthesis-with-cv-bci-and-emg)，能够帮助用户超越基本动作。通过在假肢的手掌中嵌入摄像头，计算机视觉被用来识别物体并从存储的抓握方法中进行选择。当试图抓住一个饮料罐时，CV 系统将识别它并调整手腕角度，使其在手臂角度变化时保持对齐。硬件设计是模块化的，其他输入包括肌电图和 BCI。这个项目被认为是最佳概念，被授予 10，000 美元的现金奖励。

![](../Images/53fd814449c4023da8a2d8456b239c40.png)

### DLT 一号——一个该死的 Linux 平板电脑！

当一些人还在等待 Linux 桌面时代的到来时，这个项目超越了这个时代，实现了基于 Linux 的平板电脑的开放设计。该项目的目标是避免许多消费平板电脑中存在的操作系统锁定，并提供可修复和可升级的硬件设计——目前所有消费平板电脑都不具备这些特征。这个项目被公认为最佳设计，可获得 10，000 美元的现金奖励。

![](../Images/4d3f5cac9aba1b4e49c37ec572f98053.png)

### [公理:100+kW 电机控制器](https://hackaday.io/project/164932-axiom-100kw-motor-controller)

为了提供大功率电机控制的非专有选项， [Axiom 能够驱动 100 千瓦级的三相电机](https://hackaday.io/project/164932-axiom-100kw-motor-controller)。Axiom 能够在 400 V 时提供 300 A 的电流，是用于电动汽车的强大开源选项。使用开源工具链将硬件控制分配给 ice40 FPGA，使这一控制器真正成为统治所有硬件的控制器。如果标准功能不能满足您的需求，可以利用 FPGA 的灵活性来满足您的需求。这个项目被公认为最佳作品，可获得 10，000 美元的现金奖励。

![](../Images/9cc85ca4bcbf6a41beeda7be9fe0da47.png)

### [Knobo:学习盲文的键盘](https://hackaday.io/project/166947-knobo)

对于视障人士来说，阅读盲文的能力可以带来更好的就业机会、更高的收入以及更多的阅读爱好。但是学习布里尔仍然是全世界的一个热点。 [Knobo 是一款 USB 盲文外设，目标是提高盲文识字率](https://hackaday.io/project/166947-knobo)。Knobo 使用盲文字符的六个位来帮助练习学习字符集，直到他们可以过渡到全盲文键盘或其他传统盲文书写器。作为公认的最佳基准，该项目将获得 10，000 美元的现金奖励。

![](../Images/80b31cab73e4321c2392badb9e9c61d9.png)

### SmallKat:一只可爱的动态导向机器猫

SmallKat 旨在将一个先进机器人平台交到开发世界下一代机器人的人们手中。这是一个动态机器人平台，它收集关于自身运动的反馈，并拥有实时纠正问题的处理能力。这是一种让机器人站起来而不是堆在地板上的东西，SmallKat 让学生们在建造会走路的机器人时把脚放在下面。该项目被公认为最佳沟通项目，将获得 10，000 美元的现金奖励。
