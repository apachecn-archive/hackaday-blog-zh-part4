# 四频段数字 HF SDR 收发器提供高性能，仅售 60 美元

> 原文：<https://hackaday.com/2021/12/08/four-band-digital-hf-sdr-transceiver-offers-high-performance-for-only-60/>

业余无线电爱好通常被认为是那些有可观可支配收入的人的专利。然而，最近几年，潮流发生了转变。廉价的微控制器和信号发生器有助于扭转局面，来自 QRP 实验室的[60 美元 QDX](https://qrp-labs.com/qdx.html)甚至更进一步，将性能/价格比送上了屋顶。你可以在休息时间下面的视频中看到更多细节。

QDX 是[Hans Summers]的发明，他以生产专注于低功率传输的平价高性能业余无线电套件而闻名，在业余无线电术语中称为“QRP”。这是什么？这是一款口袋大小的四频段(80、40、30、20 米)软件定义无线电(SDR)，设计用于一些最流行的数字无线电模式:FT8 和 JS8Call，以及任何其他基于 FSK 的模式，如 RTTY。它还被测试在 60 米的距离上工作良好(在规格范围内)。

传统收音机必须通过特殊的硬件接口连接到计算机，而 QDX 则设计为通过标准 USB A>B 电缆直接连接到计算机。CAT 控制、PTT 和音频都由 QDX 直接处理，不需要特殊的接口。虽然无线电基本上是即插即用的，但配置、测试和故障排除可以通过连接到 QDX 独特的串行控制台来完成，该控制台包含一个基于文本的瀑布。对于那些想要运行自己的 SDR 接收器的人，可以通过声卡直接发送 I/Q 输出。

现在是坏消息:由于全球芯片短缺，QDX 目前缺货，没有人知道他们什么时候会再次开始发货。QRP 实验室正在尽可能地寻找零部件来源，以制造更多的装置，但是当然，其他人现在也是如此。
