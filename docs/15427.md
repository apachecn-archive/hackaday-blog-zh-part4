# 便携式 ESP32 RGB 激光显示有所有的装饰品

> 原文:[https://hack aday . com/2022/11/21/portable-esp32-RGB-laser show-has-all-the-trimmings/](https://hackaday.com/2022/11/21/portable-esp32-rgb-lasershow-has-all-the-trimmings/)

也许曾经有一段时间，花哨的激光效果超出了那些没有最大的银行账户，但今天他们可以很容易地创建。[Corebb]向我们展示了如何使用现成的 RGB 激光模块和镜子模块的整洁单元，[由 ESP32 驱动，其软件旨在尽可能方便使用](https://hackaday.io/project/188322-bblaser-esp32-rgb-laser-galvo-projector)。

休息时间下面的视频是中文的，所以如果你是英语国家的人，你必须打开字幕，它会带我们看整个过程。它安装在 SLA 3D 打印外壳中，该外壳整齐地容纳所有部件。ESP32 模块驱动一对 DAC，DAC 通过一对放大器驱动检流计电机。

然后，该软件允许为你的创造性表达提供各种定制显示，包括通过 WiFi 上传快速草图。除了漂亮的图案，我们还看到它被安装在自行车上，用于平视显示速度和导航信息。即使它确实在某一点脱落和断裂，我们也可以看到它可能是一个非常有用的配件。

如果您想亲自尝试，可以在 GitHub 资源库中找到所有代码。同时，我们过去在这里报道过很多激光投影仪项目，包括最近的[这个使用步进电机代替检流计的项目](https://hackaday.com/2022/11/02/laser-project-relies-on-steppers-rather-than-galvanometers/)。
