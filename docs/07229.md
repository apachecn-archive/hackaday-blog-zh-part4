# 用神奇的微控制器魔方带回烦躁玩具热潮

> 原文：<https://hackaday.com/2020/07/21/bringing-back-the-fidget-toy-craze-with-the-magic-microcontroller-cube/>

[rickysisiodia]有一些废弃的 ATmega128 芯片，他不想就这样扔掉，所以他决定把它们变成自己的[发光小动作玩具](https://www.instructables.com/id/Magic-Cube-or-Micro-controller-Cube/)。这种玩具是一个六面骰子，非常小，你可以把它挂在钥匙链上。他将 ATmega128 焊接在立方体的两边，并添加了一些圆点，使他的玩具看起来像一个功能性骰子。我们对他令人印象深刻的灵巧程度感到非常惊讶。如果你问我们，将这些 0.8 毫米间距的引线焊接在一起似乎很乏味。

然后，他在 perfboard 上连接了一个简单的电池供电的倾斜开关 LED 电路，他可以偷偷地将它放在立方体内。他使用了一个水银开关，正如你可能想到的，使用少量的水银来短路开关内的两个金属触点，完成电路并点亮 LED。我们建议使用[无汞倾斜开关](https://hackaday.com/2011/12/02/cheap-diy-tilt-switches-are-great-in-a-pinch/)，以避免任何可能的污染。你知道我们，任何减轻不必要灾难的方法都是一条好路线。但无论如何，骰子会根据立方体的方向点亮不同颜色的 LED，甚至会闪烁。

这是一个非常酷的技巧，可以让你的朋友在下一次 PCB 艺术聚会上赞叹不已。我们可能会把它放在[电子艺术类别](https://hackaday.com/2020/01/13/the-way-of-the-pcb-artist-how-to-make-truly-beautiful-circuit-boards/)中，这样它就不会和[那些曾经受人喜爱的小动作玩具](https://hackaday.com/2017/12/28/fidget-spinner-becomes-a-brushless-motor-remains-useless/)混为一谈。
