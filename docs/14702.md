# 机器学习应用程序会记住名字，所以你不必记住

> 原文：<https://hackaday.com/2022/09/03/machine-learning-app-remembers-names-so-you-dont-have-to/>

根据你的观点，现实生活中与陌生人的对话可能会令人耳目一新，也可能令人恐惧。我们中的一些人在新的社会环境中油嘴滑舌，游刃有余，而另一些人确信最轻微的失误会永远困扰他们。在这些谈话失礼中，最主要的可能是忘记了几秒钟前刚刚自我介绍的那个人的名字。

在这种情况下，[Caleb]没有把自己关进耻辱的监狱，而是用这个稍微有点令人毛骨悚然的回忆名字的智能手机应用程序进行了反击。非零蠕变因素来自于这样一个事实，正如[Caleb]所指出的那样，如果谷歌或亚马逊这样做，这款应用越过了我们大多数人都会觉得不可接受的界限——比如监听你的每一次对话。它这样做不是根据你的对话向你投放广告，而是从自然的对话流中找出你的对话者的名字。

事实证明这是一个棘手的问题，即使有命名实体识别(NER)的帮助，它基本上是在自然文本中寻找事物的名称。这里使用的 NER 库 Apache OpenNLP 在提取名字方面工作得很好，但弄清楚它们是介绍的一部分还是关于第三方的一点八卦是[Caleb]投入大量编码工作的地方。这一点，并试图使整个事情至少有点尊重隐私。观看下面的视频进行演示。

可以肯定的是，这只不过是一句简单的“再提醒我一下你的名字？”会的，但不会尴尬。不过它仍然很酷，我们特别热衷于了解 NER 和它的大量应用。不过，这些都是未来的计划。我们很高兴看到[凯勒]已经从监测他的狗和[的孩子](https://hackaday.com/2022/07/23/machine-learning-baby-monitor-prevents-the-hunger-games/)的身体机能中走了出来。至少现在是这样。
