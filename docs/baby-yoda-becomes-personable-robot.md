# 尤达宝宝变成了风度翩翩的机器人

> 原文：<https://hackaday.com/2021/01/12/baby-yoda-becomes-personable-robot/>

尤达宝宝是迪士尼电影《曼达洛人》中的热门角色，但据我们所知，这个角色在现实生活中并不存在。相反，[【曼努埃尔·阿胡马达】着手建造一个机器人复制品，配有人工智能。](https://www.youtube.com/watch?v=PEj8jWapGt4&feature=youtu.be)(视频，下面嵌入。)

第一步是建造一个尤达宝宝的基本机器人仿真模型，这是通过给一个玩具配备伺服系统、马达和树莓派来实现的。一切都连接好了，尤达宝宝能够移动他的头和胳膊，并在轮子上快速移动，所有这些都在蓝牙游戏手柄的控制下。解决了这个问题后，[Manuel]以智能手机的形式添加了大脑，运行英特尔的 OpenBot 机器学习平台。这使得尤达宝宝可以跟踪和跟随它在智能手机摄像头上看到的人，甚至有可能通过未来的升级来导航现实世界。

这是一个有趣的构建，我们希望看到机器人在一个会议上探索和交朋友。我们之前已经介绍过 OpenBot，期待看到它在更多版本中使用。休息后的视频。
