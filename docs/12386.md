# 开源 Openwheel

> 原文:[https://hackaday.com/2022/01/03/diy-onewheel-on-the-cheap/](https://hackaday.com/2022/01/03/diy-onewheel-on-the-cheap/)

这个故事是我们都经历过的:我们看到一件商业技术，我们想得到它，但是它的价格标签让我们怀疑它是否是由黄金压制的拉丁姆制成的。[扎克]的欲望对象？一家名为 Onewheel 的公司出售的单轮驱动滑板。但是正如你在休息时间下面的[视频和](https://www.youtube.com/watch?v=A0AmxffnrwU)[他的优秀网站](https://www.bytesizedengineering.com/projects/openwheel)中看到的，扎克采用了钱包轻但时间重的方法，建造了他自己的原型，他称之为 Openwheel。

从一个电动轮开始，[扎克]使用铝，非常大的 3D 打印件，和一个非常光滑的现成控制器包来控制开式轮。平衡由控制器处理，而一个巨大的 48 伏脂电池通过一个结实的电子速度控制器，允许再生制动等先进功能。

我们不会破坏结果，但[扎克]的 Openwheel 出来很好，甚至超过了商业单位的一些规格。你可能会想看看他在 YouTube 上关于这款产品的系列视频，从而了解这款设备的所有工作，甚至是原型。

如果你更喜欢坦克履带板，看看这个[坦克履带滑板](https://hackaday.com/2020/12/25/tank-track-skateboard/),我们前段时间展示过的！
