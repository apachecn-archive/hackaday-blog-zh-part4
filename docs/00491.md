# Retrotechtacular:建造喷气时代的钢铁巨人

> 原文:[https://hackaday . com/2018/09/01/retro techtacular-the-iron-giants-that-build-the-jet-age/](https://hackaday.com/2018/09/01/retrotechtacular-the-iron-giants-that-built-the-jet-age/)

在第二次世界大战的最后几个月里，轴心国和同盟国互相倾尽所有。形势开始向有利于盟军的方向转变，但是德国人表现出了惊人的适应力，至少在替换被击落的战斗机和轰炸机方面。当盟军检查这些飞机的残骸时，他们发现了令人不安的事实:这些飞机包含由单块金属锻造而成的大块部件，这表明盟军没有这种制造能力，这使得德国人能够快速廉价地制造出更好更快的飞机。

战争结束后，盟军开始寻找德国人用来制造飞机的工具，并发现了大型闭式模锻压力机，可以一步将铝和镁合金挤压成零件。苏联人运走了一台 3 万吨的机器，而美国人带着一船小型印刷机回家，并且知道俄罗斯人比他们有优势。由此开始了重型冲压计划，这是美国军方最终成功缩小战略制造能力巨大差距的尝试，下面的精彩视频详细介绍了[【机器思维】。](https://www.youtube.com/watch?v=hpgK51w6uhk)

人们不会马上把像梅斯塔 50，000 吨压力机这样的庞然大物与飞机制造相提并论，这种机器有九层楼高，一半埋在地下，直接固定在岩床上。但是如果没有它和来自这个项目的类似机器，从 B-52 到波音 747 的飞机都不可能制造出来。无论如何，这都不是过时的技术；在运营了几十年后，1982 年卖给了美国铝业公司，在一些铸件出现裂缝后，“五十”最近花了 100 美元进行了改造，媒体和它的复古兄弟们仍在为最近的 F-35 战斗机挤出零件。
