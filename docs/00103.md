# 构建 Wi-Fi 智能秤

> 原文:[https://hackaday.com/2018/07/23/build-a-wi-fi-smart-scale/](https://hackaday.com/2018/07/23/build-a-wi-fi-smart-scale/)

有很多方法可以监测你体重的变化。你可以从你裤子的合身程度或者你腰带上的刻痕得到一个模糊的概念。但是，任何认真想保持身材的人都必须站在磅秤上，以磅或公斤为单位来获得冷酷的事实。

[[igorfonseca83]决定燃烧一些卡路里，并建造自己的智能体重秤](https://hackaday.io/project/159086-diy-wi-fi-smart-scale-with-esp8266)，它使用 IFTTT 将体重数据发送到他的健身追踪器。它由夹在两片胶合板中间的四个 50 千克的称重传感器制成。HX711 传感器模块读取称重传感器，FireBeetle ESP8266 将数据上传至 Adafruit IO。他的体重同时显示在一个发光二极管矩阵上。

我们赞扬[igorfonseca83]努力使这成为一个任何人都可以复制的简单的教育项目。说明很棒，图片很清晰，甚至还有一个所有部分组合在一起的 CAD 动画。慢跑过休息时间，观看构建视频，并在下面称重。
