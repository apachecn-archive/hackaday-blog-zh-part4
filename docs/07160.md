# 一种更简单的滚动 LED 球的方法

> 原文：<https://hackaday.com/2020/07/14/an-easier-way-to-roll-your-own-led-ball/>

是的，电路雕塑很神奇。但是所需的耐心和技巧使得我们所看到的大多数设计对于普通初学者来说遥不可及。我们完全理解——不是每个人都喜欢复杂的结构焊接。

[哈里·威古纳]被[jiří·普劳斯]去年圣诞节制作的 LED 球迷住了，他认为有比打印底片用作焊接夹具更不痛苦的方法来在闪光灯下覆盖球体。事实证明，至少有一种方法——只需[设计结构，用 PCB 代替黄铜棒，将所有东西组装在一起，就像用 FR4](https://hackaday.io/project/173461-led-ball) 制成的 3D 拼图。

这个贴片 LED 球几乎准备好了黄金时间。[Hari]希望每个人都可以访问它，并且完全参数化，所以他仍在解决一些问题。休息后，看看当前的形式，因为[Hari]使用 Arduino 在各种显示模式中滚动球，并谈论沿途的失败，比如不得不锉掉 LED 插槽，因为它们第一次设计得太紧了。[Hari]还在努力使部件摩擦配合，这样球更容易组装，尤其是在开始的时候。

3D 打印作为电路雕刻焊接夹具是很好的工具，不要误解我们。除此之外，你还能如何在曲线上焊接黄铜棒？
