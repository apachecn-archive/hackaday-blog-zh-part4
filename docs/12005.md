# 令人印象深刻的黑客把螺栓变成气动雕刻机

> 原文:[https://hack aday . com/2021/12/02/impressive-hack-turns-bolt-into-pneumatic-engraver/](https://hackaday.com/2021/12/02/impressive-hack-turns-bolt-into-pneumatic-engraver/)

你有没有看过这样的视频，它能让你用新的眼光看待日常物品？[这是其中一个视频](https://www.youtube.com/watch?v=AHSIS6iwaQU)(嵌在下面)。对我们来说幸运的是，[如果你不总是明白发生了什么，还会有一篇报道。](https://www.instructables.com/Turning-Stainless-Steel-Bolt-Into-Pneumatic-Engrav)

在这种情况下，所发生的事情是【AMbros Custom】正在熟练地将一个不锈钢 M20 螺栓变成一个气动雕刻工具。是的，你没看错。但是这个黑客最令人惊奇的地方是它使用了最少的工具。首先，看不到一台车床——[ AMbros Custom]只是把它塞进钻头，或者加几个螺母，然后用老虎钳夹住。

那么，它是如何工作的呢？[AMbros Custom]将它挂在压缩机上，使内部的活塞上下移动，搅动雕刻钻头。如果你不想看视频，在报道中有大量的图片。

你还能用螺栓做什么？如果你有工具，你可以做很多事情。你甚至可以把它变成一个秘密的现金仓库，用来购买更多的大型螺栓。
