# 看我的嘴唇，在这个无缝的面具下

> 原文：<https://hackaday.com/2020/08/30/read-my-lips-under-this-no-sew-mask/>

人类不断与我们的身体交流，口罩覆盖了最具表现力的部分之一。对一些人来说，这是陌生人的消声器，但对依赖唇读的人来说是毁灭性的。有几个面具有一个明确的窗口，正是为了这个目的，但他们的专业和高需求。Adafruit 的 Erin St Blaine 展示了她如何用你家里已经有的材料制作窗口式口罩。即使你的缝纫机被锁在当地的制造商空间，你也是幸运的，因为你不需要一针一线。对于倾向于线程的人来说，调整配方是很容易的。

面膜接触脸的部分是毛巾布，但是任何透气的棉毛巾都应该可以。说明书里有一个 PDF，你可以在里面打印四种尺寸的模板。你还会发现一个从你的冷饮杯里抢救出来的塑料窗口的剪影。水瓶应该也可以。柔韧的胶水把布料粘在一起，但是要把耳挂粘上，我们要依靠我们的老朋友，红色的摇摆线。如果你没有那个颜色和牌子，任何订书机在紧要关头都可以。别忘了加点除雾器，保持微笑。

自豪地戴上你自制的面具，系好，但不要太快。
