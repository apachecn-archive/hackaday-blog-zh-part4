# 直升机用八个旋转的水平机翼飞行

> 原文：<https://hackaday.com/2021/08/06/cyclocopter-flies-with-eight-spinning-horizontal-wings/>

对于传统的垂直起飞和着陆，垂直轴上的旋翼是最常见的解决方案，如在直升机和多旋翼飞机中所见。一个不太受欢迎的解决方案是 cyclocopter，它由一对绕着水平轴旋转的转子和水平叶片组成。作为研究项目的一部分，Nicholas Rehm 建造了一架遥控直升机，并在休息后的视频中向我们展示了这种独特的飞行器。

也被称为回旋陀螺，这个想法并不新鲜，第一个建于 1909 年。第一次飞行是在很久以后的 20 世纪 30 年代，但人们很快发现它们太不稳定，无法由人类手动驾驶，所以这个想法被搁置了。多亏了现代微控制器，研究人员最近能够制造小规模版本，就像德克萨斯大学的小例子。

在两个摆线转子的每一个上使用四个或更多的翼型产生升力。在旋转的顶部和底部，它们有一个正的攻角，侧面有一个中性角。叶片的攻角可以调整，以产生向前或向后的推力。一个带有传统螺旋桨的额外马达安装在机头上，以抵消主旋翼产生的扭矩，类似于直升机的尾旋翼。

与多旋翼飞机不同，直升机不需要向前倾斜来水平移动。叶片也不需要像传统的旋翼飞机那样变尖和扭曲，因为相对气流速度沿着叶片的长度保持恒定。然而，它们有一些明显的缺点，在可预见的未来，这些缺点可能会阻止它们超越实验阶段。转子在机械上非常复杂，并且需要非常轻，因为这种设计本身没有很大的结构强度。[Nicholas]证明了这一点，当时一次轻微的碰撞折断了其中一个旋翼。然而，它很好地展示了[Nicholas]开源的[dRehmFlight 飞行控制器](https://hackaday.com/2021/02/15/drehmflight-customizable-flight-stabilisation-for-your-weird-flying-contraptions/)的适应性，他还用它驾驶过[垂直起降 F-35](https://hackaday.com/2021/07/15/foam-f-35-learns-to-hover/)和[腹部朝下的星际飞船](https://hackaday.com/2020/12/15/spacex-starship-replica-attempts-the-bellyflop/)。
