# 便携式 3D 打印机变得更小、更快、更好

> 原文:[https://hack aday . com/2022/05/20/portable-3d-printer-gets-even-small-fast-better/](https://hackaday.com/2022/05/20/portable-3d-printer-gets-even-smaller-faster-better/)

如何改进一台快速、功能强大的 3D 打印机，它拥有创新的设计，并且足够便携，可以放在打印机的卷轴盒中？从[正电子 V3 便携式打印机](https://github.com/KRALYN/PositronV3)(视频，嵌入下方)的内容来看，它需要大量的辛勤工作和不愿意满足于折衷的设计。加上一些幸运的突破和一些设计魔法。

当[我们第一次报道](https://hackaday.com/2021/08/03/strangest-upside-down-3d-printer-fits-in-a-filament-box/)关于【Kralyn】的创新“正电子”打印机时，它的主要卖点是它的便携性和独特的布局。通过向下折叠的 Z 轴和底部的 CoreXY 风格的驱动器，加上有趣的 90°热端和透明的加热构建板，正电子成功实现了大多数设计目标。但是总有改进的空间，在下面的视频中显示的正电子 V3 已经在最初的概念上做出了一些相当大的飞跃。

V3 的设计保持了原始版本的基本布局，但大大提高了可用性和可移植性，同时提高了性能和构建体积。加热硼硅酸盐建筑板现在举行的 Z 轴驱动器更坚固的支柱，并通过高温 MagSafe 连接器获得其果汁。X 轴和 Y 轴现在由扁平步进器驱动，该步进器加上与驱动滑轮同轴的惰轮，使 CoreXY 驱动器更加紧凑，从而使打印机的底座更加紧凑。该打印机仍然比大多数传统的机架设计要快得多，打印质量与任何市售产品相当。是的，当折叠起来的时候，它仍然可以装进一个标准的 1 公斤长的线轴盒里。

我们喜欢这种设计，V3 的诞生和中间产品 V2 的失败是一个有趣的设计案例。作为奖励，[Kralyn]将开放 V3 设计的源代码，所以一旦他发布文件，你就可以构建自己的版本。
