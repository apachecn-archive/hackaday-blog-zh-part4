# 用纸板制作音乐

> 原文:[https://hack aday . com/2019/10/08/making-music-from-cardboard/](https://hackaday.com/2019/10/08/making-music-from-cardboard/)

MaKey MaKey 的粉丝可能会发现这个项目很相似，但这个迷你自动装置比用水果制作音乐要多得多。

[![](../Images/80184ad70bd2e3b662556038fcbe874a.png)T2】](https://hackaday.com/wp-content/uploads/2019/10/automat-1.png)

Mini Automat(它是[Dada Machines]最初的[Automat](https://www.kickstarter.com/projects/1481744082/dadamachines-music-machines-for-everyone)项目的分支)的想法是让任何人都能欣赏音乐。该设备的功能相当于一个即插即用的 MIDI 控制器，它连接到计算机、MIDI 工作站(键盘和序列器)或 DAW 以进行输入，并触发输出上的致动器来创作音乐。

通过使电路板与 Arduino 和 Circuit Python 兼容，以及添加用于连接传感器、按钮或灯光系统的数字和模拟引脚，这些修改使最初的 Automat 更容易被黑客攻击。

该团队已经发布了所有原理图、固件和软件，只有电路板布局未向公众发布。从推、拉、摇、砸和敲打鼓的螺线管到振动螺丝和珠子的表面，有大量不同的家用物品可以用来制作复杂的分层音乐作品，即使是一个人的音乐家。

柏林的团队致力于开源音乐技术硬件，希望将环境和财务上可持续的想法推向市场。
