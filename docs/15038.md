# TRX-杜是软件无线电的红色火龙果克隆

> 原文:[https://hack aday . com/2022/10/09/Trx-duo-is-a-red-pitaya-clone-for-software-defined-radio/](https://hackaday.com/2022/10/09/trx-duo-is-a-red-pitaya-clone-for-software-defined-radio/)

如果你已经注意到 TRX-DUO 软件定义无线电收发器，你可能会想知道它如何与 Red Pitaya 或 HackRF 等其他选择相媲美。[技术头脑]要求对下面视频中的红色设备进行[审查。](https://www.youtube.com/watch?v=teks8v3u8F0)

虽然这台机器不便宜，但也不像它的一些竞争对手那么贵。当然，你可以用很低的价格买到 RTL-SDR 加密狗，但这样你就错过了传输的机会。与廉价的软件定义无线电相比，该设备非常强大:

*   频率:10 千赫至 60 兆赫
*   CPU:采用双核 ARM Cortex A9 的 Zynq SoC
*   RAM: 512 MB
*   操作系统:Linux
*   连接:以太网和 USB 连接(带加密狗的 WiFi)
*   模数转换器:16 位，125 毫秒/秒(2 个通道)
*   数模转换器:14 位，125 毫秒/秒(2 个通道)

该板启动 SD 卡，有几个可供选择。视频显示了两幅不同的图像。其中一个有很多应用程序可以在这个设备上运行，也可以在一个红色的火龙果上运行。该设备显示了一个带有各种选项的浏览器菜单，结果令人印象深刻。使用盒子作为 WSPR 信标，在低输出功率的情况下听起来相当不错。然而，它能够轻易地听到这个世界。

你可以用比现行价格低 100 美元的价格买到一个性能稍差的红色火龙果模型。然而，对于类似的东西，你将为红色火龙果支付更多的钱——取决于能力——也许更多，尽管你确实为增加的价格获得更多的能力。

你可以用一个[传输 SDR](https://hackaday.com/2020/04/14/pluto-might-not-be-a-planet-but-it-is-an-sdr-transceiver/) 做更多的事情——既有传输又有接收开启了许多新项目。当然，固定的应用程序很棒，但是如果你得到了其中的一个，你会想要尝试一下 [GNU Radio](https://hackaday.com/2015/11/11/getting-started-with-gnu-radio/) 。
