# 这(几乎)是两个键盘手合二为一！

> 原文：<https://hackaday.com/2019/04/13/its-almost-two-keytars-in-one/>

所有最好的 20 世纪 80 年代怀旧音乐表演都应该有一个键盘手。这是法律，或者别的什么。[特雷门英雄]通过一个新视频提醒了我们这一点，我们在下面展示了一个他几年前与[萨姆·雷]和[西达尔特·瓦德加马]一起参与创作的乐器。[Blade](https://siddv.net/projects/blade)是一个 3D 打印的 keytar，具有两个吉他英雄颈部和一对集成的 Game Boys，以提供来自真正硅的声音。

然而，用这些术语来描述它就错过了大量的其他组件和特性。键盘本身来自一个摇滚乐队 keytar，它将 MIDI 提供给一个运行 PD Extended 的 Raspberry Pi，该 PD Extended 处理所有的按钮映射。Arduino Mega 为两个吉他英雄颈部执行相同的任务。来自不同来源的 Midi 由 Arduino 男孩处理，然后提供给发出声音的游戏男孩。哦，还有一个 Leap Motion 3D 运动控制器，尽管它似乎没有直接用于 chiptune synth 功能。

这些年来，我们这里已经有几个 keytars 了，但是这个让我们想起了由[Jeri Ellsworth]创造的[Commodore 64 乐器](https://hackaday.com/2012/07/09/jeri-ellsworth-on-making-her-c64-bass-keytar/)。
