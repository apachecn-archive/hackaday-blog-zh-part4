# 机器人紧贴天花板

> 原文:[https://hackaday.com/2021/06/09/robot-clings-to-ceiling/](https://hackaday.com/2021/06/09/robot-clings-to-ceiling/)

想象你在电影院，你看到一个类似 Roomba 的机器人在爬墙或抓着天花板。这怎么可能呢？如果你和我们一样，你可能会想到吸盘或墙壁上的机械或磁性的东西。话说回来，这是一部电影，所以也许这只是一个相机的把戏。然而，来自加州大学圣迭戈分校生物创新机器人和设计实验室的机器人并不是相机特技。正如埃文·阿克曼在 [IEEE Spectrum](https://spectrum.ieee.org/automaton/robotics/robotics-hardware/robot-sticks-to-ceilings) 上的一篇帖子中提到的，“这要么是某种模糊的流体效应，要么是黑魔法。”你可以在下面观看一段关于机器人的视频。

事实证明，答案比你想象的更接近吸盘。根据来自实验室的论文,一个小的柔性盘以 200 赫兹振动。这会在磁盘和下面的表面之间产生一层薄薄的(小于 1 毫米)低压空气。机器人可以抵抗来自磁盘的高达 5 牛顿的吸力。

这个圆盘的直径只有 14 厘米，大约相当于一个普通碟子的大小。这似乎很容易用你自己的机器人来复制。最大的问题是噪音。在 200 赫兹，你会听到这个机器人来了。看起来向上或向下扩展也是一个问题。太小，驱动偏心重块的电机占用空间太大。越大，你需要支撑的就越多。我们想知道压电盘是否可行。

我们期待看到基于这项技术的自制机器人。我们当然见过其他的[爬壁机器人](https://hackaday.com/2012/05/29/climbing-robot-glues-its-own-feet-to-the-wall/)。有些甚至可以[飞](https://hackaday.com/2011/09/12/wall-climbing-bot-does-it-with-ease/)。
