# 用微制造的透镜阵列获得一个蝇眼视图

> 原文：<https://hackaday.com/2021/09/07/getting-a-flys-eye-view-with-microfabricated-lens-arrays/>

原子力显微镜，激光烧蚀，以及用有毒化学物质的女巫混合物蚀刻:听起来像是[Zachary Tong]又在实验室里玩了，这一次他发现了一种制造显微透镜阵列的方法。

像许多最好的项目一样，[扎克]的微制造之旅始于一次愉快的意外。这发生在他研究金属激活化学蚀刻(MACE)时，该技术使用贵金属催化剂在硅上选择性地雕刻高纵横比的特征。在用激光对镀银的硅晶片进行喷砂处理后，他注意到蚀刻后的烧蚀坑非常光滑和均匀。这使他对正在发生的事情有了几个假设，所有这些假设他都能够验证。

实验本身非常有趣，但真正酷的是[扎克]意识到硅中光滑的半球形坑可以作为一系列微观凸透镜的模具。他能够通过旋涂将少量透明硅树脂沉积到模具中，并(最终)将微透镜阵列转移到玻璃载玻片上。这些透镜非常小——几百个透镜只有几百平方微米——而且形状非常好。当然，总有改进的空间，但对于基于偶然发现的初步尝试，我们称之为胜利。至于这些镜头有什么好，你的猜测和我们的一样好。但像这样的新过程往往会找到一种有用的方式，这是从家庭实验室出来的事实并不能改变这一事实。

我们发现这种微型制造很吸引人。无论是[制作有机发光二极管显示器](https://hackaday.com/2021/09/04/making-oled-displays-in-the-home-lab/)、[用等离子微加工玻璃](https://hackaday.com/2021/01/09/micromachining-glass-with-a-laser-very-very-slowly/)，甚至是[自己卷半导体](https://hackaday.com/2021/08/14/the-zeloof-z2-intergrated-circuit-has-100-transistors/)，这玩意我们都看不够。
