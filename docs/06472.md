# 《毁灭克隆人》展示了另一个现实世界的阿米加可能拥有的一切

> 原文:[https://hack aday . com/2020/05/04/doom-clone-shows-what-a-alternate-reality-amiga-could-had/](https://hackaday.com/2020/05/04/doom-clone-shows-what-an-alternate-reality-amiga-couldve-had/)

你能在阿米加上运行毁灭战士吗？不，不完全是，可以说这是电脑在 90 年代中期消亡的原因之一，因为它没有赶上个人电脑世界的 FPS 热潮。Altair demogroup 的[Krzysztof Kluczek]并没有成功地用原始文章来补救这种状况，而是向我们展示了一个潜在的竞争者如何能够在过去为未扩展的 Amiga 硬件设计。

许多开发人员试图模仿 id 软件射手的刺激和氛围，但他们都需要更快的处理器和更大的内存的高端 Amigas，这限制了他们在已经减少的人口统计中的玩家基础。不仅如此，即使有更好的硬件，也没有一个能比得上当时 Doom 在普通的 486 上的表现。[Krzysztof]并没有试图移植 Doom 本身，而是创建了一个定制设计的引擎，以利用 1987 年存在的 OCS Amiga 的局限性。其结果是恐惧，一个 2.5D 引擎，类似于 SNES 的末日港，并使用 Freedoom 项目的资产，以保持版权。

这可能不是厄运，但它是一个很好的窥视 33 岁的硬件可以在当时的权利。从技术上来说，它已经超过了 Wolfenstein 3D 引擎所能做的，所以如果有人想要制作一个直上端口而不是他们自己的游戏，那就有一个想法。如果你喜欢看《毁灭战士》在它本不该运行的机器上运行，男孩[做](https://hackaday.com/2019/09/01/running-doom-on-a-doomed-gps/)我们[有](https://hackaday.com/2019/08/29/how-to-play-doom-and-more-on-an-nes/)一些[帖子](https://hackaday.com/2018/10/18/playing-doom-on-keysight-oscilloscope-via-windows-ce/)给[你](https://hackaday.com/2014/07/26/playing-doom-on-an-atm/)。否则，休息后留下来看两个展示恐惧引擎的视频。
