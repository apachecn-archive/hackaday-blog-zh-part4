# 工作模型揭示了韦伯的镜子驱动器惊人的工程

> 原文：<https://hackaday.com/2022/02/08/working-model-reveals-amazing-engineering-of-webbs-mirror-actuators/>

我们最终在 Hackaday 上覆盖了很多太空主题，这不是因为我们是超级太空迷——剧透一下:我们是——而是因为当你有有效的无限预算和制作不会失败的东西的权限时，往往会产生令人惊叹的工程。詹姆斯·韦伯太空望远镜上的镜面致动器是这种极端工程的完美例子，为了更好地理解它们是如何工作的，[Zachary Tong]建造了这些神奇机器的工作模型。

JWST 的主镜由 18 个独立的六边形部分组成，每个部分的位置都必须精确调整，以形成完美的反射镜。每个镜子都有七个致动器，可以在七个自由度上移动它——Stewart 平台机构通常提供的六个自由度，加上稍微改变镜子曲率的能力。[扎克]的模型致动器是从镜像承包商鲍尔航空航天公司提供的[公开信息](https://www.esmats.eu/amspapers/pastpapers/pdfs/2006/warden.pdf) (PDF)逆向工程而来的。虽然 OEM 部件是由通常的太空合金和材料制成的，但模型是 3D 打印的，由廉价的步进电机驱动。

这种简单掩盖了模型揭示的巧妙机制。致动器允许在很大的行程范围内进行粗调和微调。一个聪明的不倒翁机制意味着只需要一个电机进行微调和粗调，一个弯曲机制用于进行更精细的微调-步长只有 8 纳米！

向[Zach]致敬，感谢他为我们挖掘这个问题，并向[提供他所有的文件，以防你想打印自己的文件。你可能不会很快建造一个太空天文台，但是这些机制可以为你的设计提供很多信息。](https://www.thingiverse.com/thing:5232214)
