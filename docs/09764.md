# 三件式电脑甲板发挥了从未有过的军用电脑的作用

> 原文：<https://hackaday.com/2021/04/08/three-piece-cyberdeck-plays-the-role-of-military-computer-that-never-was/>

虽然我们还没有厌倦网络平台运动，但我们不得不承认，我们看到的一些构建开始落入似乎有点熟悉的类别。我想到的是蛤壳式美学，但实际上，有了显示屏和键盘的空间，这种形式对于网络装饰来说是非常自然的。这就是为什么我们如此喜欢这个 cyberdeck 概念的三件套。

像许多网络甲板建筑一样，令人惊叹的大胡子[Max]甲板的灵感来自军事剩余世界。据说，他有一个小小的蛤壳式箱子，里面曾经装着德国国防军的无线电工具和供给品。他认为这将是半个分离式键盘的完美外壳，于是又找了几个坚固的铝制外壳，开始工作。作为一个机械键盘爱好者，[Max]已经有了可以放入两个盒子的 PCB，所以他在这些盒子上安装了合适的咔嗒开关，设计了看起来很酷的面板，并用可伸缩的电缆连接两个盒子。第三个案例得到了一个树莓 Pi 4，它有一个修剪过的散热器，一个电池和电源管理，以及一个宽大的触摸板和液晶面板显示器。Kali Linux 安装完成了 tacticool 的外观。

三件套的电脑平台连在一起看起来很酷，但[Max]还需要一件才能真正卖出去。所以他 3D 打印了三个单元的书套；漆成军颜色，并适当地进行了处理，整个东西真的很好用。最近我们已经在[的](https://hackaday.com/2021/03/25/data-blaster-is-a-hip-rpi-cyberdeck/)看到了很多[网络甲板，各种风格](https://hackaday.com/2021/01/10/cyberpunk-mazda-mx-5-packs-onboard-cyberdeck/)，但是这个真的很令人高兴。
