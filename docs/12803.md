# 从车头灯回收的高功率激光

> 原文：<https://hackaday.com/2022/02/17/high-power-laser-salvaged-from-headlights/>

[DiodeGoneWild]的最新视频名副其实。他[拆开一个激光头灯](https://www.youtube.com/watch?v=2-zPgkk8PLM)来恢复一个相当强大的蓝色激光。你可以看下面的视频。

头灯与蓝色激光二极管一起工作，激发磷光体产生白光。去掉外面的装饰，露出了一个三针激光二极管(外壳是第三个针)。还有一个坚固的散热器。从组件中移除二极管很困难，但将其留在散热器中并使用现有的连接器是很容易的。

当然，磷光体和滤光器必须去除。一些破坏性的工作用螺丝刀和钳子打破了他试图移除的二极管的光学元件。然后，他用修改过的外壳替换了剩余二极管上的光学元件。

在低电流测试中，二极管没有发出激光，但起到了普通 LED 的作用。不过，更大的电流起了作用。没有光学系统的激光产生了一条线而不是一个点，但仍然有足够的能量熔化一些塑料和火柴。为了获得平行光束，内部透镜需要靠近二极管，钻头允许这种情况发生，这大大降低了光束的发散度，但没有创造出最好的结果。

随着廉价激光模组的泛滥，报废一个大灯真的值得吗？也许吧。但不管怎样，这是现代车头灯内部的有趣外观。我们之前已经看过这些车头灯的内部。也许你可以把那些旧前灯[变成一个烤箱](https://hackaday.com/2021/09/28/microwave-ovens-need-more-power-use-lasers-instead/)。
