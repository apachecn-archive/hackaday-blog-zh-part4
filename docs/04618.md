# 在冲浪窗口的帮助下悬挂十个

> 原文:[https://hack aday . com/2019/10/24/hang-ten-with-help-from-the-surf-window/](https://hackaday.com/2019/10/24/hang-ten-with-help-from-the-surf-window/)

除非你住在一个特别、独特的地方，比如夏威夷或哥斯达黎加，否则你不可能每天都冲浪。因为天气条件需要恰到好处，所以在大多数地方计划冲浪活动甚至冲浪旅行并不容易。不仅浪高，风速和风向、潮汐、水温和气温，甚至海洋生物的数量和种类都会影响你的冲浪活动。如果条件好的话，你会想要一个很容易马上告诉你的东西。

这个来自[luke]的项目叫做冲浪窗口，只需一瞥就能显示当地海滩的冲浪情况。由各种木头制成，每个部分代表海滩的一种天气状况。例如，一只旋转的海鸥给出了风向，波浪高度由 3D 的移动波浪表示。所有部件都与各种电机相连，并连接到 Arduino Mega +WiFi R3，该 R3 从冲浪预测网站 Magicseaweed 获取所有信息。

冲浪窗口可以显示世界上几乎任何可冲浪海滩的当前状况，所以如果你真的想知道[大白鲨](https://www.youtube.com/watch?v=krsqU2HFJkw)、[小牛](https://www.youtube.com/watch?v=gS6LJgfR1AM)甚至[礁石之路](https://www.youtube.com/watch?v=Zf35gn22UTw)现在是如何破裂的，你可以用这个给你一个更细致的观察。不要忘记根据条件选择[正确的板！](https://hackaday.com/2019/01/28/shred-the-gnar-without-paddling/)
