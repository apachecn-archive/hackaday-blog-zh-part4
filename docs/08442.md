# 现实生活中我们最讨厌的任务

> 原文:[https://hack aday . com/2020/11/22/现实生活中我们最讨厌的任务/](https://hackaday.com/2020/11/22/the-most-annoying-among-us-tasks-created-in-real-life/)

《我们中间》是一部关于欺骗和阴谋的热门游戏。玩过这个游戏的人都知道在*号上试图完成一些故意设定的困难任务的挫败感。*【扎克·弗里德曼】决定[在现实生活中重现其中一些](https://www.youtube.com/watch?v=NJTJqJyBLic)。

[Zach]构建了游戏中最令人沮丧的三个任务。有一个极其缓慢的上传/下载站，由一个旧的三星平板电脑和一个 NFC 标签组成，还有一个反应器启动控制台，使用树莓 Pi 3B、Teensy 3.2 和一个定制的机械键盘创建。但也许最令人讨厌的是臭名昭著的读卡器。它由另一个 Teensy 构建，要求用户以*恰到好处的速度刷他们的 ID 卡，*除了每次刷的速度是随机产生的。此外，机器失败了 20%的好刷卡*只是因为。*也许我们最喜欢的是[Zach]通过将有机发光二极管显示器放在深绿色塑料后面，并使用 14 段字体，重新创造了经典的 VFD 外观。

这是对一款大获成功的独立游戏的一种有趣的致敬，我们可以想象这些道具会在 makerspace 派对上大受欢迎。[我们以前也展示过其他*的*主题建筑](https://hackaday.com/2020/11/21/among-us-emergency-meeting-button-becomes-reality/)。休息后的视频。
