# 多亏了物理学，从简单的充气玩具到复杂的动作

> 原文:[https://hack aday . com/2022/07/24/complex-movements-from-simple-inflatables-thanks-to-physics/](https://hackaday.com/2022/07/24/complex-movements-from-simple-inflatables-thanks-to-physics/)

基于注入压力改变形状的充气致动器可能很强大，但它们的大限制是它们总是以相同的方式变形。

[![](../Images/1b5ceb774132193afbcf21857d85810d.png)](https://hackaday.com/wp-content/uploads/2022/07/Our-Origami-World-0-28-screenshot.png)

The Kresling pattern, which inspired the actuator design.

但是通过从折纸中获得结构灵感，研究人员创造了 3D 打印的致动器，表明仅由单一压力源驱动的致动器实现复杂运动是可能的。这是怎么做到的？通过使致动器在物理上双稳态，在某种程度上不需要额外的压力源。

关键是基于 [Kresling 模式](https://thekidshouldseethis.com/post/the-kresling-pattern-and-our-origami-world)的修改设计，每个致动器都有一个特别设计的部分(上图中的彩色三角形)，设计为在一定量的正压下弹出，并在弹出后保持稳定。这个部分保持其形状，直到施加一定量的负压，并且该部分弹回。

无论该部分是否弹出，都会改变致动器的形状，从而改变其变形方式。这使得简单的致动器是双稳态的，并且能够仅使用单个压力源进行不同的运动。将一堆这样的致动器堆积起来，通过仔细的压力控制，复杂的运动就成为可能。在两个短视频中，就在分页符的下面，看看它是如何工作的。

这里有一个视频伴随着[研究论文](https://onlinelibrary.wiley.com/doi/10.1002/adfm.202201891)演示了致动器的设计。这些致动器是由 TPU 和解放军 3D 打印的。在它的正下方嵌入了一个视频，展示了一个由 12 个致动器组成的圆柱的复杂运动，所有这些运动都是由一个压力源驱动的。

折纸术和在我们的自然世界中发现的设计[与一般的机器人学](https://hackaday.com/2022/03/09/bringing-the-art-of-origami-and-kirigami-to-robotics-and-medical-technology/)配合得很好，看到如此简单的结构变化如何对设计产生如此大的影响令人着迷。
