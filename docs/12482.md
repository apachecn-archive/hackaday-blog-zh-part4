# 巨大的轮子造就了激动人心的电动滑冰鞋

> 原文：<https://hackaday.com/2022/01/15/giant-wheels-make-for-exciting-powered-rollerskates/>

轮滑鞋很有趣，但是它们只能在相对光滑的表面上滚动。[火球工具]想要一些更坚固的东西，所以着手建造他自己的。

设计的挑战是使用尽可能多的轮椅部件，包括轮子，来制造这些冰鞋。大约 22 英寸高，轮子内部有很大的轴承，并被设计成在单侧轴支撑上运行，非常适合溜冰鞋。然后用一个金属支架来固定滑雪靴固定器，这样轮子就可以适合穿着者的脚。训练轮安装在后部，使骑手更容易，而链锯发动机被压入服务，以提供一些受欢迎的推进力。

在平坦的车间地面上进行的短暂测试中，车轮表现出色。希望大直径的轮子能比传统的滚轴溜冰鞋在粗糙的表面如草地或泥土上表现得更好。我们期待看到这项测试的实际效果，并与我们见过的其他电动滑冰鞋进行比较。休息后的视频。
