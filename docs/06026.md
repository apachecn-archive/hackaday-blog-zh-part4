# 时尚的麦克风是金属印刷做得对吗

> 原文：<https://hackaday.com/2020/03/21/stylish-mic-is-metal-printing-done-right/>

[Eric Strebel]想要一个麦克风来录制画外音，而作为一名设计师，想要一个适合这项任务的令人印象深刻的东西。受经典的 Unidyne 55 的启发，他开始设计自己的麦克风，并使用了一些非常奇特的技术来制作。

麦克风是围绕一个带状元素，提供良好的动态响应。这个设计是用 CAD 创建的，最初打算由三个独立的部分组成。然而，[Eric]意识到通过使用粘合剂喷射 3D 打印机，这就没有必要了。

粘合剂喷射是一种技术，其中喷嘴将粘合剂喷射到粉末床中，以创建 3D 打印零件。在这种情况下，使用钢粉，然后用液态青铜在烘箱中进行后处理。青铜渗入成品中，增加了强度。这是一个制造金属零件的过程，没有什么限制；主要关心的是最小壁厚。

有了活页夹喷墨打印机，[Eric]可以自由设计最终产品的时尚几何形状。将六边形与经典的 50 年代曲线混合，最终的结果令人印象深刻。[Eric]现在经常使用麦克风来录制画外音，名副其实的 Hexavox 甚至在 NAMM 亮相。
