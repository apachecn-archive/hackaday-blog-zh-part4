# 海军陆战队 3D 打印零件修复数百万美元的战斗机

> 原文：<https://hackaday.com/2018/08/23/marines-3d-print-part-to-repair-multi-million-dollar-fighter/>

好消息是:完成你正在进行的修复所需要的只是一个小零件。坏消息是:它只能在更大、更贵的组件中使用。最糟糕的消息:运输时间是永远。我们都有过这种经历，对于 DIYer 来说，这是一颗难以下咽的药丸。似乎是 3D 打印的一个很好的用例。

现在想象一下，你是一名美国海军陆战队员，你没有修理洗碗机或电视遥控器，而是在商店里买了一架价值 1.23 亿美元的 F-35 战斗机。你需要的零件是起落架舱门的一个小塑料保险杠，但它只能作为整个舱门组件的一部分，这需要花费 7 万美元纳税人的钱。从美国发货的时间是以周计算的。你甚至能想到用 3D 打印来代替它吗？事实证明你可以做到，而且看起来世界各地的部队修理站将会有更多的增材制造。

由于显而易见的原因，打印部件的细节还没有公布，但该部件是在 Blender 中建模并在 PETG 的一台似乎是消费级的打印机上打印的。该部件在快速获得适航性批准后安装，停飞的战斗机在几天内就恢复了服役。令人鼓舞的是，这不是一次性的；其他部件已经被海军陆战队批准用于飞行，地面车辆的可打印部件的整个目录也是可用的。这就是《迷失太空》 中的 3D 打印小说[建立的现实。](https://hackaday.com/2018/06/25/lost-in-space-gets-3d-printing-right/)

谁知道呢？也许在一次性无人机[中有可现场打印的部件](https://hackaday.com/2017/07/03/automate-the-freight-front-line-deliveries-by-drone/)是军队用于远距离补给任务的。
