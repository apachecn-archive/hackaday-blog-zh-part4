# 复杂和无用相互抵消

> 原文:[https://hacka day . com/2021/03/03/复杂无用-相互取消-相互淘汰/](https://hackaday.com/2021/03/03/complicated-and-useless-cancel-each-other-out/)

我们都知道拖延意味着什么，但是积极地花时间去建造一台无用的机器呢？毫无疑问，你已经看到了顶部有一个诱人的小开关的装饰盒。当你不可避免地扳动开关时，一个致动器从外壳的一半弹出，唯一的目的是解开它自己的电源开关。[Paz Hameiri]更进了一步，添加了一些[Rube Goldberg]的味道，在微控制器的帮助下，他的控制杆花了很长时间关闭电源。(休息后的视频。)

我们没有为这个项目找到任何代码或图表，但如果你知道这个无用机器的内部结构，为你的办公桌重新创建一个应该不会有任何麻烦。最重要的设计因素是开关。它们的触点必须并联，这样，只要一个触点处于活动状态，控制器就能通电。你将如何给这台无用的机器增添趣味？

尽管这些被称为[无用的机器](https://hackaday.com/2020/09/20/finally-a-differently-useless-machine/)，但它们服务于装饰、开始对话的目的，或者炫耀你的木工和[编程技能的一种方式](https://hackaday.com/2020/11/03/useless-box-with-attitude-isnt-entirely-useless/)。
