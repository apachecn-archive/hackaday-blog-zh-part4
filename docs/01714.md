# DIY 丝带元素升级录音室麦克风

> 原文：<https://hackaday.com/2019/01/02/diy-ribbon-element-upgrades-a-studio-microphone/>

对于那些有一些专业音频经验的人来说，“带状麦克风”一词往往会让人联想到广播黄金时代的那些又大又粗的麦克风，这种麦克风装饰着电台的呼号，由最新的万人迷梦舟歌手哼唱。这个 DIY 缎带麦克风不是这些东西中的任何一个，但它仍然非常酷。

当然，带状麦克风并不总是巨大的，其背后的技术也远未过时。[Frank Olsen]的带状话筒从去除普通录音室话筒的元素开始，只留下主体和连接器。他构建的元件主要由小铝片和丙烯酸块组成，看起来非常像商业麦克风中的带状元件:一对磁铁，中间悬挂着一条薄的波纹状箔片。箔片通过[弗兰克]制造的夹具起皱，这是一个很好的工具，但他说用于手工制作的卷纸机也可以工作。在拾音器制作的背后有一些非常繁琐的工作，但一切都很好地融入了旧的麦克风主体。下面的视频是用麦克风叙述的，所以我们知道它是有效的。

有趣的事实:带状麦克风是由沃尔特·肖特基发明的。[那个沃尔特肖特基](https://hackaday.com/2016/09/12/diodes-with-hats-zener-and-schottky/)。需要更多关于这些话筒工作原理的信息吗？我们的同事[Al Williams]已经为您介绍了[这篇关于基础知识的文章](https://hackaday.com/2016/05/02/blue-ribbon-microphone/)。
