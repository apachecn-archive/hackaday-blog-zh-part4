# 给予控制台控制器鼠标般的精确瞄准

> 原文：<https://hackaday.com/2021/09/04/giving-a-console-controller-mouse-like-precision-aim/>

控制器 vs 键盘和鼠标是游戏世界中永无止境的战斗之一，双方都有铁杆支持者。[Tech Yesterday]一直致力于打造一款两全其美的控制器。他最新的 [Mouse Pro 控制器 V5](https://www.youtube.com/watch?v=kerK52IRGjs) 的特点是一个倒置的鼠标骑在滚珠轴承上。

[Mouse Pro 控制器 V1-3](https://hackaday.com/2020/12/15/mouse-controller-hybrid-aims-to-dominate-in-first-person-shooters/) 的主要目标是为光学拇指鼠标创造尽可能大的移动表面，以实现精确瞄准。然而，[Tech Yesterday]发现，一个人的拇指不太适合穿越一个大的平面，但在凹面上工作得更好。[在 V4 上，他翻转了](https://www.youtube.com/watch?v=4sJAD3TuJzk)周围的光学传感器，将其嵌入控制器中，拇指上还连着一个圆形的小“鼠标垫”。凹面由大型 LED 灯泡的扩散器制成。对于[Tech Yesterday]来说，它的摩擦力有点太大了，所以他用磁铁在表面嵌入了一系列小滚珠轴承。

虽然这种“拇指鼠标”具有出色的精度，但当您需要进行大幅度移动时，如为片段执行 360 度无范围操作时，它可能会有点慢。对于这些情况，[Tech Yesterday]在控制器的背面嵌入了一个拇指棒，允许使用他的中指进行快速的横向移动。

[Tech Yesterday]已经在做 V6 了，但是感觉已经接近他技能的极限了。如果你有兴趣和他一起工作，一定要[联系](https://www.youtube.com/channel/UCNLJd7YD8ysETf5zAfJObjA/about)！为了乐趣和性能改装控制器是很棒的，但是[对于有假手的游戏玩家来说这是一个要求](https://hackaday.com/2020/12/15/console-controller-mod-gets-amputee-back-in-the-game/)。
