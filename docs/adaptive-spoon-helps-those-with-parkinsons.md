# 适应性勺子帮助帕金森氏症患者

> 原文：<https://hackaday.com/2019/01/09/adaptive-spoon-helps-those-with-parkinsons/>

在医疗条件下生活有很多副作用，而且不是所有的副作用都很明显。对于帕金森病来说，其中一个条件是持续的手部震颤。这显然会导致对任何涉及精细运动技能的事情感到沮丧，但也包括吃饭，这可能比其他日常任务更麻烦。有一些产品可以帮助缓解震颤，但价格如此之高，[Rupin]决定用现成的组件制作一个[震颤补偿器皿。](https://www.hackster.io/RupinC/assistive-spoon-for-patients-with-parkinson-s-3628b6)

这个项目的主要灵感来源是 [Liftware Steady](https://www.liftware.com/steady/) ，但是 200 美元左右的价格对很多人来说是遥不可及的。为了降低成本，这种辅助勺子的核心有一个材料清单，我们大多数人都已经有了。它是围绕一个 Arduino 和一个 MPU6050 惯性测量单元以及两个通用伺服电机构建的。让这个器皿正常工作确实需要一些 3D 打印和大量的数学运算，但代码可以在项目网站上找到，任何人都可以看一看。

这个项目解决了一个我们经常看到的问题:一个经济有效的、开源的医疗问题解决方案，唯一的替代方案要昂贵得多。通常这出现在假肢领域，但也可以通过以低于医疗公司提供的价格直接制造胰岛素等生物化合物来帮助解决这个问题。
