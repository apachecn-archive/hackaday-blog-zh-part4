# 链接耦合天线调谐器无字工作坊

> 原文：<https://hackaday.com/2020/03/15/link-coupling-antenna-tuner-wordless-workshop/>

还记得科普里的“无字工坊”吗？[罗伊·多蒂]展示了一个家庭问题，以及在主角的车库工作室里拼凑出来的解决方案。我们想知道罗伊会对 YouTube 做些什么？也许类似于来自[VE2TAE]和[VE2AEV]的视频，展示了他们的[链路耦合天线调谐构建](https://www.youtube.com/watch?v=nh4zWR1SQ3I)。你可以在休息后观看视频，如果你不是爵士乐爱好者，你可以将音量调至静音。

就像[Doty]的漫画一样，该视频假定你将对尺寸和组件值有自己的想法，以满足你的需求。但是建筑本身就很美。缠绕成巨大线圈的管道令人印象深刻，并带回了过去的记忆。然而，可变电容器的构造确实让我们兴奋不已。大空气可变帽可能很难找到，但视频使它们看起来很容易制作。

几个好看的旋钮和面板仪表使一个伟大的期待调谐器。有了这个空间，我们可以想象它将毫无困难地处理全部法律权力。如果你想了解更多关于这种类型的调谐器，[VK1OD]有一个关于它的很棒的页面，现在似乎已经不存在了。但是[互联网档案馆](https://web.archive.org/web/20130321175120/http://vk1od.net/tx/concept/lctr.htm)一如既往地拯救了我们。

这个设计相当古老，所以即使是一部 [1934 年的《无线电》](https://www.americanradiohistory.com/Archive-Radio/30s/Radio-1934-04.pdf)副本也能解释它(请看第 6 页)。如果你想看一个更冗长的制作可变电容的例子——虽然它们更小，但原理是一样的——[N4DFP]有一篇关于这个的[好文章](https://www.qsl.net/n4dfp/buildcaps.html)。

当然，这些天来，大多数人期望他们的[天线调谐是自动的](https://hackaday.com/2014/01/06/automatic-antenna-tuner/)。不过，如果你愿意，你可以用一些乐高[改装你的手动玩具](https://hackaday.com/2009/07/03/lego-modded-antenna-tuner/)。
