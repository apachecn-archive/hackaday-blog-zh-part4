# 机器人反叛带回 BBC 摄像师

> 原文：<https://hackaday.com/2022/12/30/robot-rebellion-brings-back-bbc-camera-operators/>

现代电视新闻工作室是现场视频和 CGI 的杰作，因为各网络都在争夺最闪光的展示。伦敦的 BBC 新闻也不例外，它在 2013 年拥抱了未来，用机器人摄像机取代了有血有肉的摄影师。从表面上看，这是有道理的；它更便宜，新闻编辑室的摄像机最有可能记录一系列非常相似的镜头。十年后，他们将在人类的胜利中退休，因为公司厌倦了病毒式的失败，让主持人争先恐后地赶上。

对于普通读者来说，媒体报道可能显得微不足道，但对于有技术头脑的人来说，其中有值得思考的内容。似乎摄像机有一套预编程的操作，制作团队可以为他们的不同镜头选择，错误的操作很容易被启用。还有一种说法是系统的年龄可能与此有关，但这在某种程度上被他们的例子破坏了，我们把这个例子放在下面，是相机只有一年的时候。

鉴于现代电视演播室是一个受到严格控制的空间，检测主持人的位置以及他们是否在镜头中在 2013 年应该不是遥不可及的，所以我们很好奇他们为什么没有走这条路。也许 OpenCV 可以检测人，或者在行动前简单地检测麦克风的音量就可以了。不管怎样，我们欢迎摄像师回来，即使我们永远见不到他们，尽管我们会错过这些病毒式的搞笑镜头。
