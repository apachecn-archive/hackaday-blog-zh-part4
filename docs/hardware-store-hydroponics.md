# 五金店水培

> 原文：<https://hackaday.com/2020/10/05/hardware-store-hydroponics/>

科幻电影经常描绘未来的园艺，无论是在陆地上还是在飞船上，水培花园里满是绿叶蔬菜和色彩鲜艳的水果。这里没有土壤，只有清澈的水，暗示着未来——人们创造了一个植物的乌托邦，脱离了他们尘世的根源。

如果你放弃了聚碳酸酯管、霓虹灯和园艺机器人，这种明星般的食物制作方法就不是虚构的了。在他的 2020 年 Hackaday 奖参赛作品中，[AVR]分享了他如何用附近家装商店采购的零件为 16 株植物制作了一张床。它可能缺乏好莱坞版本的视觉活力，但它将以黑客的预算种植无土作物。

这个建筑的起点是一个坚固的木制底座。上面的 PVC 管和围栏部分很轻，但里面的水会变重，如果你种植大型植物，它们会变得出奇的重。说到水，这属于水培的子类是营养薄膜技术，或称 NFT，它使用一条充满植物生长所需营养的浅水流。方形栅栏柱为植物生长的地方提供了一个安装网杯的平顶，也为水流持续流动的地方提供了一个平底。一个盆和[泵](https://hackaday.com/2018/04/17/diy-drill-powered-water-pump/)保持[植物的新鲜](https://hackaday.com/2020/04/29/raspberry-pi-waters-your-lawn-serves-html/)和喂养，直到它们准备好收获。
