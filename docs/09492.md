# 一种不同的 3D 打印机:桌面全息图

> 原文：<https://hackaday.com/2021/03/12/a-different-kind-of-3d-printer-desktop-holograms/>

全息图像并不新鲜，但一种能把它们吐出来的台式机器可能很快就会出现，假设 LitiHolo 的 Kickstarter 成功的话。这款机器的零售价将为 1600 美元，适合两英尺见方的空间。它可以用 1 毫米的全息元(相当于一个像素)产生 4×5 英寸的全息图。

这台机器允许每个全息元有 23 个视区，可以在几秒钟内创建移动全息图——就像著名的吹吻全息图一样。

当然，你还需要一种特殊的自显影胶片和一种将 3D 图像输入打印机的方法，如软件或进行 3D 扫描的相机。在 4×5 的尺寸下，一张胶片大约 13 美元，这将产生一个全息图。

由于 5 英寸是 127 毫米，因此结果的 hogel 分辨率约为 101×127，网站上的样本和下面的视频看起来肯定不像是高清的。

人们会为低分辨率全息图支付 1600 美元吗？更重要的是，粒状全息图是否有市场，能让你赚回投资？也许不是，但这并没有阻止我们购买 3D 打印机和其他车间玩具。另外，如果这个流行起来，十年后会有什么可用的呢？

当然，如果你有激光设备，你已经可以[制作你自己的全息图](https://hackaday.com/2018/07/23/hope-xii-make-your-own-holograms/)。你甚至可以得到有[大部分你需要的](https://hackaday.com/2020/12/24/hello-holograms/)的装备。
