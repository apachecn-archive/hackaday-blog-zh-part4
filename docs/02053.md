# 用于电子纺织品的三导体枢轴优于电线

> 原文：<https://hackaday.com/2019/02/13/three-conductor-pivot-for-e-textiles-is-better-than-wires/>

电子纺织品的枢轴似乎是一个微不足道的问题。毕竟，电线和织物可以很好地弯曲和伸缩。然而，穿在身上的东西可能有更棘手的需求。卡扣连接器是获得电气连接和支点的常用方法，但它们仅提供一根导线。当[KOBAKANT]需要一个带有三个电导体的枢轴连接时，[他们提出了一个设计，通过使用集成到单个按钮扣的柔性电路板来实现这一点。](http://www.kobakant.at/DIY/?p=7435)

这个有趣的设计是对一个特定要求的解决方案的一部分，该要求是精确测量手的运动。照片显示了两个连接在一起的条带，它们作为一个整体旋转。靠近中心的金属盘是磁铁，下面是霍尔效应传感器。当手腕弯曲时，磁体移近或移离传感器，该单元相应地平滑地弯曲和枢转。下面嵌入的简短视频清楚地表明了整个事情是如何工作的。
