# 走进宝丰的内心

> 原文:[https://hack aday . com/2022/11/15/get-to-the-heart-of-a-baofeng/](https://hackaday.com/2022/11/15/getting-to-the-heart-of-a-baofeng/)

在业余无线电圈里，几乎没有任何一件设备比不起眼的宝丰手持收发器更能吸引争议。这是可以理解的——收音机是价值工程的一个光辉典范，它有足够的零件来完成工作，同时又处于 FCC 规则的边缘。每台大约 25 美元，这种收音机非常便宜，以至于实验几乎是拥有它的必要条件。

但是，尽管宝丰可能被剥开，但它所包含的秘密甚至比收音机本身更有吸引力。还有谁能比[HB9BLA]更好地带领我们通过宝丰 sa 818 的核心射频模块呢？HB9BLA 是一个有着令人生疑的熟悉瑞士口音的家伙。只需大约 8 美元，你就可以从全球速卖通买到一个这样的小玩意，并且拥有几乎所有 VHF 或 UHF 无线电的重要部件 SDR 收发器、功率放大器和使其工作的所有粘合逻辑。

在下面的视频中，[Andreas]在电路板的帮助下测试 SA818 模块，电路板将模块与一些附件配对，如音频放大器和低通 RF 滤波器。使用 Raspberry Pi 和 Python 库来控制模块，这是对 Baofeng 功能的一个不错的模仿。但这仅仅是开始。通过给 Pi 添加一个 USB 声卡，该设置能够进入每个火腿最喜欢的分组无线电系统，APRS。SA818 模块还有许多其他应用，其中一些[Andreas]在视频的最后提到了。口袋大小的中继器，小得离谱的 EchoLink 热点，甚至是 Altoids tin 中的 AllStar 节点。

当然，如果你想玩得开心，你需要一个业余无线电执照。别担心，这很简单——我们会帮助你到达那里。
