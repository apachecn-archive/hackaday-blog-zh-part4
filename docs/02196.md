# 所有的事情谜黑客聊天

> 原文:[https://hack aday . com/2019/02/25/all-things-enigma-hack-chat/](https://hackaday.com/2019/02/25/all-things-enigma-hack-chat/)

加入我们太平洋时间周三中午的[所有的事情谜黑客聊天](https://hackaday.io/event/163974-all-things-engima-hack-chat)！

本周的 Hack Chat 对我们来说有点与众不同，因为我们的主持人 Simon Jansen 处理了太多有趣的项目，以至于很难确定一个话题。西蒙是一个多学科的黑客，他的兴趣涵盖了从建造[一个弹药罐苹果】【](https://hackaday.com/2011/12/26/ammo-tin/)到[一个真正的蒸汽朋克摇滚者](https://hackaday.com/2010/12/02/steam-powered-rickrolling/)。一杯啤酒怎么样？还是[一个 TARDIS](https://hackaday.com/2011/07/01/a-mame-cabinet-fit-for-a-doctor-who/) 里的 MAME？或者也许[让一部旧手机播放音乐通过](https://hackaday.com/2012/03/05/antique-phone-provides-a-soundtrack-perfect-for-restoring-old-cars/)还原一辆汽车？哦，还记得[《星球大战:第四集](http://www.asciimation.co.nz/index.php) 那部超赞的 ASCII 动画吗？那是西蒙。

因此，选择一个话题有点困难，但我们请西蒙谈谈他最近的英格玛手表。他已经成功地将二战时期的英格玛密码机的电子模拟装置放入手表和最近的怀表中。它们都是华丽的构建，需要大量的技能来完成。我们将从那里开始，看看对话会把我们带到哪里！

请加入我们的黑客聊天，我们将讨论:

*   迷上英格玛的原因；
*   工具、技术和车间设置；
*   融合多种不同的技能组合；和
*   我们很快会看到什么样的新项目？

当然，我们鼓励你在讨论中提出自己的问题。你可以在 [All Things Enigma Hack Chat](https://hackaday.io/event/163974-all-things-engima-hack-chat) 上发表评论，我们会把它放在 Hack Chat 讨论的队列中。

[![join-hack-chat](../Images/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是在 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)上的现场社区活动。本周我们将在太平洋时间 2 月 27 日星期三中午坐下来。如果时区让你心烦意乱，[我们有一个方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20190227T12&p0=224&msg=All+Things+Enigma+Hack+Chat&font=cursive)。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
