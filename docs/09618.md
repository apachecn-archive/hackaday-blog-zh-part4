# Data Blaster 是一个时髦的 RPi 网络平台

> 原文:[https://hack aday . com/2021/03/25/data-blaster-is-a-hip-rpi-cyber deck/](https://hackaday.com/2021/03/25/data-blaster-is-a-hip-rpi-cyberdeck/)

Cyberdecks 曾经是未来计算机可能的样子的科幻近似。最终，消费电子产品走上了一条截然不同的道路。不过没关系，因为创客社区认为 cyberdecks 太棒了，不能忽视，并开始制作自己的。在对已经存在的一些令人惊叹的构建垂涎三尺之后，【扎克·弗里德曼】决定是时候开始他自己的构建了，[产生了 Data Blaster。](https://www.youtube.com/watch?v=guGffGw3uDg)

[![](../Images/99d127891c8280e74ff284cafa985e6f.png)](https://hackaday.com/wp-content/uploads/2021/03/datablaster_detail.jpg)

Epoxy holds the printed parts to the Pi 400

Raspberry Pi 在新生的 cyberdeck 场景中一直很受欢迎，它在一个紧凑的便携式包中提供了真正的 Linux 计算能力。现在，我们有 Raspberry Pi 400，正是它，内置在一个外壳中，大约是半个网络平台。这构成了[扎克]的构建基础，它的全尺寸键盘很方便。

除此之外，[他还增加了一个 1280×480 的宽屏液晶显示器](https://hackaday.com/2021/01/11/retro-terminals-bring-some-style-to-your-desktop/)，可穿戴显示器和一个 USB 电源组，把它变成了一个真正的无处不在的终端。3D 打印的手柄手感特别好，便于站着使用，这是笔记本电脑做不到的。作为一个额外的奖励，甚至还有一个小小的软件无线电，配有一个可折叠的天线，增加了*的酷因子。*

这是一个有趣的构建，也是一个有用的构建。我们怀疑厚重的塑料和可抓取的设计可能会使 Data Blaster 在恶劣的野外使用中比笔记本电脑更受欢迎，而不是更传统的笔记本电脑。[我们也看到了这一领域的其他优秀作品](https://hackaday.com/2020/12/08/devterm-beats-cyberdeck-builders-to-the-punch/)。休息后的视频。
