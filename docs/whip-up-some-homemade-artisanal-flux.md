# 搅拌一些自制的手工熔剂

> 原文：<https://hackaday.com/2022/10/18/whip-up-some-homemade-artisanal-flux/>

你不会想太多电力公司，直到你打开灯，他们不亮。焊剂也是如此。通常，你不会想太多，直到你尝试不用它焊接。助焊剂对金属表面有清洁作用，有助于正确焊接。问题是，你知道你用的助焊剂里有什么吗？我们也不知道。[Catsndogs]有一个 r[recipe 来制作自己的 flux](https://hackaday.io/project/187703-rosin-soldering-flux-diy) 然后你就知道了。

松香焊剂的核心基本上是树汁。如果你住在松树附近，你可以自然地获得它。如果没有，你可以在乐器店找到。弦乐器用松香，所以很容易买到。如果你自己去找的话，[Catsndogs]报道说不管它是旧的还是干净的。不过，你确实想尽可能多地挑出树皮和死蚂蚁。你基本上把它溶解在酒精中(至少 80%的异丙醇或乙醇)。然后用滤纸或咖啡过滤器过滤。

你可以通过让酒精蒸发使混合物变稠或者通过加入更多的酒精使其变稀来调整粘度。较厚的焊剂有利于固定 SMD 零件。正如你所料，这不是“不干净”的焊剂。此外，助焊剂非常易燃，所以要小心。

这不是我们第一次听说这个食谱。甚至还有第二次[的](https://hackaday.com/2012/05/10/what-the-flux-buy-it-or-brew-it-yourself/)。但这是一个很好的提醒，你可以自己制作不含任何商业制剂中古怪化学物质的产品。
