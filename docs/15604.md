# 终极游戏和手表支持 NES

> 原文:[https://hack aday . com/2022/12/09/ultimate-game-and-watch-has-support-for-nes/](https://hackaday.com/2022/12/09/ultimate-game-and-watch-has-support-for-nes/)

我们已经多次讨论过特性蠕变，它通常被认为是设计原型时应该避免的事情。在一个构建中包含许多特性听起来可能不错，但是这通常会导致更复杂和更困难，当实际完成一个项目时。[Brendan]对这款最初为游戏和观看游戏设计的定制手持设备有着相反的体验，他最终也添加了 NES 和 Game Boy 功能。

由于这个版本最初只是为了游戏和观看游戏，屏幕大约是这些旧游戏的大小，虽然它可以很容易地模仿这些 80 年代手持设备上出现的单色 LCD 风格的视频，但它也支持彩色，这意味着它也是模仿其他主机的完美候选。它是基于一个树莓派零 2W 和外壳是定制印刷和绘画。不过，由于不支持本地模拟输出，必须找出一些音频的变通办法，但它仍然拥有所有这些系统的几乎所有功能。

虽然我们已经看到了大量的定制便携式产品，从复古游戏机到更现代的游戏机，但游戏和手表目录经常被忽略。有一些在外面，但在这种情况下，我们很欣赏允许这个版本支持 Game Boy 和 NES 游戏的功能蠕变。
