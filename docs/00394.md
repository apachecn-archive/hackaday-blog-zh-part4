# 你从未见过像这样的固态示波器

> 原文:[https://hack aday . com/2018/08/25/youve-never-see-a-solid-state-oscope-like-this-one/](https://hackaday.com/2018/08/25/youve-never-see-a-solid-state-oscilloscope-like-this-one/)

还记得示波器有大脑之前的时代吗？这很容易忘记，因为我们已经习惯了一类简单的固态示波器，它使用微控制器作为信号处理器，使用小型 LCD 显示器来显示结果波形。它们通常以廉价套件的形式提供，虽然带宽不是很大，但在低频应用中表现出色。当然，最初信号处理的方式要简单得多。

[SimpleTronic]提醒我们，小型固态示波器不需要微控制器，在试验板上有一个[示波器，它以更传统的方式在 LED 矩阵上显示波形。这在很大程度上是一个模拟示波器，其中 CRT 的 X 偏转电路由步进通过显示器上的 led 列的十进制计数器代替，Y 偏转电路由一些模拟信号调理电路代替，随后是驱动显示行的 LM3914 条形图显示芯片。有一些改进，如触发电路，但它仍然是一个非常容易理解和令人惊讶的简单设备。](https://hackaday.io/project/160577-mini-oscilloscope)

其声称带宽为 40 kHz，由其扫描范围而非模拟带宽定义，输入电压范围为 50 mVpp 至 50 Vpp。由于其低带宽，它几乎不是一种有用的工具，但它的优势在于新颖性和理解传统示波器，而不是它的实用性。你可以在我们放在休息区下面的视频中看到它的运行。

有限使用的范围不时出现在这些页面上。我们最喜欢的是这个烙铁。
