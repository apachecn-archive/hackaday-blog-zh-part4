# 复古游戏领结

> 原文:[https://hackaday.com/2020/08/04/retro-game-bow-tie/](https://hackaday.com/2020/08/04/retro-game-bow-tie/)

[格雷格]喜欢砍他的领结。回到高中，他在参加舞会时戴的领结上添加了一些明亮的 RGB LEDs，甚至获得了男性最佳着装奖。最近，他决定尝试另一种领结黑客技术，这一次[让他的领带有一些复古的街机游戏的感觉](https://www.hackster.io/gpro1/boredom-busting-breakout-bowtie-c3f999)。

他决定使用 ATtiny85 并尝试做一些更低级的编程来更新他的技能。他从头开始写他所有的库，这真的帮助他在这个过程中学到了很多东西。这也有助于他确保他的代码尽可能高效，因为他在使用 ATtiny85 时有相当多的内存限制(只有 512 字节的 RAM)。

他用木头设计了领结的主体。他将所有的电子设备都安装在机身内，同时允许 ATtiny 伸出机身，让他的领结具有黑客的美感。当然，他需要使用拨动开关来玩游戏，所以他也做了一个插槽。

Hackaday 上的[电子产品领结系列的一个不错的补充。如果你问我们，这是真正的美学设计。你也知道](https://hackaday.com/2013/08/15/bow-tie-kindly-suggests-that-you-back-off/)[我们有多爱复古游戏](https://hackaday.com/2020/02/26/developing-retro-games-for-conference-badges-with-kate-morris/)。
