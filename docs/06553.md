# 编织线的扭曲故事

> 原文:[https://hack aday . com/2020/05/12/a-twisted-tale-of-woven-wires/](https://hackaday.com/2020/05/12/a-twisted-tale-of-woven-wires/)

一年多前，我们举办了一场竞赛，挑战读者离开舒适的 PCB 和试验板。我们希望看到三维的赛道，为创造力和艺术天赋加分。说实话，最初有人担心“电路雕塑”比赛对于 Hackaday 社区来说有点太抽象了，但大量绝对华丽的参赛作品无疑打消了这些疑虑。

在最近的一个视频中， [[Michael Aichlmayr]向观众展示了他令人着迷的作品 *Wonderlandscape*](https://www.youtube.com/watch?v=e83CkBysqUY) 的创作过程，该作品最终在电路雕塑大赛中获得了最佳金属作品荣誉奖。尽管这不仅仅是一个项目的简单走查。当然，你会看到黄铜棒原料是如何巧妙地扭曲和包装，以创造一个金属冬季景观，看起来像是来自鲍勃·罗斯迄今为止未知的赛博朋克时期，但这只是故事的一半。

[![](../Images/104ea5a4dbfcdfb8402eeff7bf7c346f.png)](https://hackaday.com/wp-content/uploads/2020/05/wirestory_detail.jpg)

The brass bar was twisted by hand with pliers.

在视频中，[Michael]回忆了他是如何发现蓬勃发展的电子雕塑社区的，并指出了一些让他着迷于发现通常隐藏在塑料外壳内部的美丽的特殊例子。最终，他听说了电路雕塑比赛，并决定这是一个完美的机会来建立自己的东西。没错， *Wonderlandscape* 是他第一次尝试将电子产品变成艺术。

但最好的可能还在后头。[Michael]解释说，由于比赛的时间限制，他不得不使用从工艺品商店购买的金属原料。但他的最终目标实际上是熔化回收的黄铜和青铜部件，制造自己的线材和棒材。我们迫不及待地想看到当他开始用自己定制的金属工作时，他将能够完成什么，并急切地等待他说将会重温他一直在工作的技术的未来视频。

这个故事很好地提醒了我们，偶尔走出自己的舒适区是一件好事。毫无疑问，毫无经验地参加比赛是有风险的，但是[迈克尔]从另一边出来时更有经验，而且在社区里有了一些新朋友。那么，为什么不[参加我们最新的比赛，看看它会带你去哪里](https://hackaday.com/2020/04/30/new-contest-making-tech-at-home/)？
