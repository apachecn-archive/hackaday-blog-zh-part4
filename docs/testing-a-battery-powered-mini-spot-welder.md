# 测试电池供电的小型点焊机

> 原文：<https://hackaday.com/2019/03/25/testing-a-battery-powered-mini-spot-welder/>

你见过粘在电池端子上的薄金属片有小麻点吗？那是点焊机的工作。点焊是一种没有太多选择的工艺；要么使用点焊机来做好这项工作，要么干脆不做。正是这种需求促使[欧文·里德]从韩国的一家制造商那里购买了一台小型电池驱动点焊机，并在镍条上进行测试。

使用的点焊机【Erwin】是[一个名叫【aulakiria】](https://blog.naver.com/aulakiria/220992039512)(链接是韩语，机器翻译[这里是](https://translate.google.com/translate?hl=en&sl=auto&tl=en&u=https%3A%2F%2Fblog.naver.com%2Faulakiria%2F220992039512))的用户的作品，被设计成便携式，由 RC 常用的电池供电。[Erwin]对结果很满意，并在下面嵌入的视频中演示了该设备。

点焊机项目看到很多 DIY，[其中一些是成功的](https://hackaday.com/2017/07/18/beautiful-diy-spot-welder-reminds-us-we-love-3d-printing/)而[其他的就不那么成功了](https://hackaday.com/2018/09/21/the-un-economy-of-building-your-own-spot-welder/)。我们自己的 Sean Boyce 甚至试着让[制造一台太阳能点焊机](https://hackaday.com/2018/03/15/building-a-portable-solar-powered-spot-welder-nearly-practical/)，他将结果描述为“几乎实用”
