# 拉面笔让你用面条涂鸦

> 原文:[https://hack aday . com/2019/11/06/ramen-pen-lets-you-doodle-with-noodle/](https://hackaday.com/2019/11/06/ramen-pen-lets-you-doodle-with-noodles/)

不要把你的古怪想法一笔勾销——把它们变成现实。多年来，木工一直使用笔体作为画布来展示美丽的木材。但是这有什么意思呢？[JPayneWoodworking] [用拉面做了一支笔](https://jpaynewoodworking.com/making-a-pen-using-ramen-noodles/)就是想看看他能不能。

正如他在休息后的构建视频中解释的那样，这个过程非常简单。他把未煮过的面条块锤成足够小的碎片，以适合钢笔空白模具，但又不会小到无法辨认。然后他注入橙色、银色和黑色的环氧树脂。[JPayneWoodworking]为万圣节选择了这些颜色，但我们认为它看起来并不怪异，而是让钢笔看起来像一碗来自高档拉面店的牛肉汤。

加入~~味包~~色素后，他将它放入压力罐中，去除所有的气泡。一旦它安装好了，过程就和其他笔坯一样了——把它放在车床上旋转，抛光，铰出，然后用笔套件的零件装配它。我们想看看下一个向[JPayneWoodworking]要笔的人脸上的表情。

想进入木工行业只是为了做这种奇怪的东西？我们不怪你。但是像你这样的硬件黑客怎么入门呢？[【丹·马洛尼】为你报道](https://hackaday.com/2017/01/23/woodworking-basics-for-the-hardware-hacker/)。
