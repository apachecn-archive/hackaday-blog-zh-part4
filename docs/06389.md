# 你需要更多的怪异

> 原文:[https://hackaday.com/2020/04/25/you-need-more-weird/](https://hackaday.com/2020/04/25/you-need-more-weird/)

当你需要创造性地解决一个问题时，你会怎么做？我，我去散步，最好是在树林里。这是离桌子和电脑尽可能远的地方，不知何故，走出我办公室的框框也帮助我跳出这个隐喻的框框去思考。也许是新鲜空气，也许是运动。或者，可能是把我的物理头脑放在一个不同的(头脑)空间，帮助我以不同的方式思考。

心理学家发现，置身于外，成为一个局外人，甚至只是接触到直截了当的陌生人会让你觉得更奇怪(T2)，也就是说，更有创造性。艺术家、作家和其他超有创造力的人常常有点古怪，这几乎是老生常谈了。想想那些在药物、精神疾病或极度混乱的影响下做出最佳作品的艺术家。

好消息是你可能不用走那么远。心理学家能够简单地通过让人们暴露在怪异的环境中来衡量创造性解决问题能力的提高。你也不需要去一个神奇的蘑菇之旅。例如，在一项研究中，在回答问卷之前，这是在一个颠倒的虚拟世界中进行的。当雷·威尔逊建议建造一个愚蠢的合成器有助于你思考时，他是开玩笑的，但是现在有了科学的支持，谁还会笑呢？

所以，如果你发现自己像我一样被困在同样的四堵墙内，一定要不时地跳出这个框框。为了你自己的创造力，把你的大脑暴露在怪异中。为一个完全古怪的项目腾出一些时间。当然，阅读更多 Hackaday！(我们有古怪。)
