# 再制造旋转飞机发动机

> 原文：<https://hackaday.com/2019/04/13/remanufacturing-a-rotary-airplane-engine/>

如果有人告诉你他们见过转子发动机，你很可能会立即想到汪克尔发动机，就像你可能在一些更具异国情调的马自达跑车中发现的那样。但是还有另一种转子发动机拥有优先权，它可以作为许多 20 世纪早期飞机的动力装置。在这些旋转式发动机中，气缸围绕固定的曲轴呈放射状排列，发动机本身旋转。它们的优点是极其简单，动力平稳，零件数量少，代价是润滑完全损耗，旋转质量相对较大，控制动力有些困难。到 20 世纪 20 年代，这些旋转发动机大部分已经过时，但是最近对 WW1 时代的飞机的兴趣高涨，导致了对它们的少量需求。位于新西兰的经典航空加工服务公司已经介入填补了这个空白[，并且正在重新制造那个时代数量最多的 Gnome 径向引擎](https://www.youtube.com/watch?v=TtMu8mrwM2k)。

对于任何对内燃机感兴趣的人来说，[Gnome](https://en.wikipedia.org/wiki/Rotary_engine#Gnome)是一项令人着迷的研究。它是一种九缸设计，运行四冲程奥托循环，但它只有一个阀门，而不是你可能熟悉的汽车上的两个或更多阀门。所谓的单气门设计将气门用于燃料和排气，在进气冲程和排气冲程都打开气门。单个气门和没有化油器的简单性因此被改变其功率的困难所抵消，所以旋转式发动机经常减少点火气缸的数量来代替节流。

CAMS 侏儒是原版的忠实复制品，但采用了现代冶金技术，并增加了电子点火系统。最初的蓖麻油仍然在使用——这似乎是经典的航空爱好者喜欢的味道——但因为它在发动机中留下粘性沉积物而臭名昭著，他们正在评估现代替代品。他们的网站上有一些技术细节，有一天你很有可能在你附近的航展上听到他们的引擎。
