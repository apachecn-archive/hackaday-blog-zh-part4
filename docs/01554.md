# 搬家不一定要这么辛苦

> 原文：<https://hackaday.com/2018/12/15/bust-a-move-doesnt-have-to-be-such-a-grind/>

电脑游戏玩家受益于鼠标和键盘，它们在游戏输入设备方面非常灵活。更重要的是，平台的自由意味着你可以很容易地设计出任何你能想到的最适合你正在玩的游戏的 mad 控制器。[带着名副其实的潮人游戏手柄](http://tomtilley.net/projects/hipster-controller/)进入【汤姆·蒂利】。

是的，这是一个用咖啡研磨机做成的控制器。手摇曲柄设置使得它非常适合模仿经典益智游戏《半身像移动》中的游戏板。[Tom]以一种完美的方式连接控制器——鼠标的滚轮从机箱中取出，拉链系在研磨机的轴上，让研磨机旋转来转动滚轮。然后鼠标滚轮被映射为游戏中的目标控制——真的就这么简单。一个街机按钮也安装在一个杯子里，并连接到鼠标左键上作为触发器。

[Tom]指出，虽然你可以在玩游戏时给研磨机加满油，但这让玩游戏变得非常困难。这是一个粗糙的构建，表明有时你不需要微控制器和代码行——只需要一个鼠标、一些电线和适当的创造力。

而且，事实证明， [[Tom]几年来一直是“半身像搬家”的粉丝。休息后的视频。](https://hackaday.com/2012/11/08/bust-a-move-physical-controller/)
