# Hackaday 播客 124:用石墨烯艰难驾驶，被激光愚弄，用毒酸蚀刻，以及所有的 Linux 命令

> 原文：<https://hackaday.com/2021/06/25/hackaday-podcast-124-hard-drivin-with-graphene-fooled-by-lasers-etching-with-poison-acid-and-all-the-linux-commands/>

Hackaday 的编辑埃利奥特·威廉姆斯和迈克·什奇斯对本周展出的危险项目感到惊讶，包括用氢氟酸蚀刻玻璃和将 20 万安培的电流注入 5000 安培的保险丝。一种将 Raspberry Pi 变成 SDR 的新主板展示了 Broadcom 芯片中存在的二级内存接口(SMI)的强大功能。我们还讨论了石墨烯在硬盘驱动器中的潜力，并以拆除非常早期的电子节拍器结束。
