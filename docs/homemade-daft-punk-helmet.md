# 自制蠢朋克头盔

> 原文：<https://hackaday.com/2018/12/29/homemade-daft-punk-helmet/>

你可能不是法国人，你可能没有一系列热门唱片，但你仍然可以拥有闪亮的 LED 头盔，这要归功于[Electronoobs]的这一构建。他们组装了一个整洁的[蠢朋克头盔](https://www.electronoobs.com/eng_arduino_tut62.php)，由 3D 打印部件、Arduino、蓝牙模块、一串 WS2812 可寻址 led 和一个简单的应用程序组成。头盔本身是 3D 打印的，Arduino、蓝牙和电池都安装在下巴里。LED 面板是一系列串联在一起的 WS2812 LED 灯条。整个事情是通过蓝牙连接到一个 Android 应用程序来控制的，这个应用程序是用麻省理工学院的应用程序 Inventor 开发的。

这是一个很好的，简单的构建，但正如我们之前讨论过的[选择扩散器是很难的](https://hackaday.com/2017/04/25/ask-hackaday-what-about-the-diffusers/)。我们不确定覆盖 LED 条的更厚面板，或者翻转 LED 并添加反射层是否是改善漫射外观的正确举措。无论哪种方式，这都是一个开始自己制作的好地方，也是了解如何享受 LED 灯带乐趣的好方法。

我们在过去已经看到了很多类似的构建，包括[手工焊接的](https://hackaday.com/2018/09/28/gaze-upon-this-daft-punk-helmets-rows-of-utterly-perfect-hand-soldered-leds/)、 [iPhone 控制的](https://hackaday.com/2014/11/03/iphone-controlled-daft-punk-helmet/)，以及一个[酒吧，当你戴着它的时候可以在里面闲逛](https://hackaday.com/2013/06/24/how-to-build-a-tron-bar-that-daft-punk-would-hang-out-at/)。不过，这个比大多数都简单，使用[麻省理工学院的应用程序发明者](http://appinventor.mit.edu/explore/index-2.html)也使得从移动设备上控制它变得简单。
