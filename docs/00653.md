# 别让戴在手腕上的喷火器打断你的眉毛

> 原文：<https://hackaday.com/2018/09/14/let-no-eyebrow-go-unsinged-with-a-wrist-mounted-flamethrower/>

我们只说一次，就在前面:腕戴喷火器是一个坏主意。鼻子发痒和短暂的健忘会让你度过糟糕的一天。也就是说，[这场燃烧的末日挑战](https://hackaday.io/project/161126-flame-o-tron)看起来很有趣。

我们必须把它交给[史蒂夫·埃尔南德斯]——他在 Flame-O-Tron 9000 上做了很多工作。基于他在该领域的现有技术，史蒂夫在这个设计上更进一步。原理是一样的——丁烷加火花等于乐趣——但这个喷火器的内脏是全新的。一个~~管式炸弹~~定制油箱被用来代替库存丁烷罐，一个电磁阀控制燃料流量。一切都生活在一个时髦的丙烯酸外壳中，骑在一个手工制作的皮革护腕上，手柄上的控制加上一个 Arduino 允许用户发射短暂的火焰或充电成为一个真正的火球。休息后看看大家对短视频中最终产品的看法；听起来好像即使燃料耗尽，高压也能成为一把漂亮的眩晕枪。

也许我们应该停止对这些腕式火箭的安全唠叨。毕竟，我们已经看到[很多](https://hackaday.com/2017/08/01/be-the-firebender-you-want-to-see-in-the-world/)，[很多](https://hackaday.com/2017/12/05/3d-print-yourself-a-flame-thrower/)，[很多](https://hackaday.com/2010/12/13/shooting-fireballs-from-your-wrists-hadouken/)他们当中，连一个受伤的报道都没有。
