# 在盒子里随着这个机器人鼓手的节拍演奏

> 原文：<https://hackaday.com/2019/06/08/play-to-the-beat-of-this-robotic-drummer-in-a-box/>

没有鼓手？没问题！稍加巧妙，你可以[将架子鼓的基本部件装进一个盒子](https://www.instructables.com/id/DrumCube-an-Arduino-Robot-Drummer/)，并自动化你的节奏部分。

请注意，[弗兰科·莫利纳]的“鼓面”没有人类鼓手的闪光，但它确实保持了稳定的节奏，有自己的魅力。鼓机大部分是机械的，这让我们想起了 Wintergatan 大理石机，它不仅听起来迷人，看起来也很迷人。DrumCube 有一个由两个伺服控制棒演奏的小鼓，一个使用泡沫在两个连接到低通滤波器的压电传感器之间来回摇摆的底鼓，以及一个用于踩镲的反向偏置晶体管白噪声发生器。可悲的是，大型齿轮似乎只是为了展示。Arduino 运行一切，并确保机械鼓与电子钹同步，这不是一个简单的壮举。

下面的视频展示了它在[Franco]吉他伴奏下的动作，看起来和听起来一样好。喜欢更紧凑的全电子架子鼓吗？这里有一个适合你放在口袋里的。
