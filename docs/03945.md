# LED 三角看起来很酷；有人告诉三角乐队

> 原文:[https://hack aday . com/2019/08/24/led-triangle-looks-cool-someone-tell-alt-j/](https://hackaday.com/2019/08/24/led-triangle-looks-cool-someone-tell-alt-j/)

对于普通人来说，家居装饰就像从宜家《家居指南》中挑选几样一样简单。然而，制造商是一个不同的品种——更喜欢定制和发光的东西。[这个 LED 三角形是形式](https://www.instructables.com/id/Making-a-great-LED-triangle/)的一个特别好的例子，也是来自[scan lime]工作室的最新卓越基准。

来自刚刚过去的 2014 年，这是一个非常适合普通创客的设计。它由多层激光切割硬纸板和丙烯酸树脂制成，为 led 创造了 16 个独立的口袋，中间几乎没有出血。一个黑色挡板配合完成的效果，随着磨砂白色丙烯酸扩散每个三角形元素。

该建筑使用 WS2812B LEDs，由[scanlime 的] Fadecandy 控制器控制。Fadecandy 是专门为 LED 艺术项目设计的硬件和软件的组合，提供对抖动和其他效果的高质量控制，以帮助使 glowables 更漂亮。它往往会在需要令人瞠目结舌的可视化效果的地方出现。在这个应用中，它做得很好，像素的伪随机闪烁在本质上几乎是催眠的。

这是一个伟大的赛博朋克艺术作品，我们希望在家里的咖啡桌上有一个。如果你厌倦了 LED 立方体，[基于三角形的建筑可能会重新点燃你的热情](https://hackaday.com/2012/05/31/triangle-grid-led-display/)。休息后的视频。
