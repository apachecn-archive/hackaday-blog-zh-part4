# 五十架机动滑翔伞部分飞行，部分滑行

> 原文：<https://hackaday.com/2022/03/29/fifty-motored-paraglider-partly-flies-partly-glides/>

如果有一件事你可以指望(彼得·斯里波尔)做，那就是定义“不要在家里尝试这个”的航空类别在休息时间下面的视频中，[彼得]展示了他最新的空中恐怖:[一个装有*五十个*电动机](https://www.youtube.com/watch?v=pXcVTsuXxOA)的动力滑翔伞背包。它会飞吗？是的。成功了吗？呃……大部分时间。

正如[彼得]在视频中所说:不要在家里尝试这个。[Peter]已经制作了一个滑翔伞，它本质上是一个非刚性的织物机翼，在未经训练的人看来，它类似于一个降落伞，并用来自其他项目的 50 个无人机马达为它提供动力。两个电机分别以推/拉配置安装在一个 5×5 阵列的 3d 打印管道内。

虽然实验基本上是成功的，但它也是一个失败，因为没有足够的电力，电池寿命太短，总的来说不是那么好。每一个实验都需要以绝对的成功而告终，以便从中获得乐趣并学到可以应用于下一次迭代的经验教训吗？肯定不是！我们为[彼得]愿意失败而鼓掌——尽管，我们不得不承认，当你已经有了一顶展开的降落伞时，失败会容易得多！
