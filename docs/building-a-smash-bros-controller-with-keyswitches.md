# 用按键开关构建 Smash Bros 控制器

> 原文：<https://hackaday.com/2020/12/18/building-a-smash-bros-controller-with-keyswitches/>

当涉及到竞技格斗游戏时，手中拥有正确的控制器可以决定胜败。正是因为这个原因，许多锦标赛对控制器有严格的规定。[Akaki Kuumeri]最近组装了一个定制控制器，[旨在最大限度地提高《超级粉碎兄弟:任天堂 Switch 终极版》的性能。](https://www.youtube.com/watch?v=tCPDhHOb5hM)(视频，下面嵌入。)

该建筑由一个有吸引力的 3D 打印体组装而成，让人想起最初的任天堂娱乐系统控制器。在内部，一个廉价的第三方 Gamecube 控制器用于与控制台接口。机械键盘开关被用来取代按钮，甚至模拟棒，还有一个特殊的修饰键，可以在舞台上行走和奔跑。这是通过一些模拟模拟棒中间位置的电阻来实现的，并使实现高级组合更加容易。

这是一个有趣的构建，我们可以想象精确的数字按键输入比模拟控制有一些好处。值得注意的是，由于自定义键盘亚文化，如果没有现成的机械按键开关，这样的构建就不会那么容易。[我们已经看到这些令人满意的开关出现在最近的许多控制器版本中](https://wp.me/paBn4l-1TF1?preview=true)。
