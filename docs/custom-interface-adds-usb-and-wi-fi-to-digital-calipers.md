# 自定义接口为数字卡尺增加了 USB 和 Wi-Fi

> 原文：<https://hackaday.com/2022/10/17/custom-interface-adds-usb-and-wi-fi-to-digital-calipers/>

虽然老派机械师通常更喜欢他们可靠的卡尺上的机械游标刻度，但现在许多用户购买带数字读数的卡尺。这些模型通常带有额外的功能，如差分测量，或“保持”功能，用于那些您必须在机器深处操纵仪器的情况。另一个有用的功能是一个数据链接，让您可以直接在计算机上记录您的测量值，而不是手动输入所有的值。

[Liba2k]买的 VINCA 品牌卡尺就有这样的数据链接功能，需要一个单独出售的 USB 适配器。工具本身有一个微型 USB 连接器，但它不是实现 USB 接口，而是用于携带专有串行协议——如果你问我们，这应该被归类为重罪的设计决策。[Liba2k]没有购买官方的 USB 适配器，而是解码了协议并构建了自己的接口，名为 VINCA Reader，可以通过 USB 或 Wi-Fi 连接。

串行格式原来是一个简单的串行总线，每次输出 24 位。为了使其 1.2 V 信号电平适应 ESP32 使用的 3.3 V 信号电平，[Liba2k]设计了一个简单的电平转换器电路，使用了一些分立元件。ESP 可以通过其 Wi-Fi 接口与电脑进行通信，为此[Liba2k]编写了一个类似电子表格的应用程序；或者，可以连接一根普通的 USB 电缆来模拟键盘，以便与任何其他软件一起使用。

由于增加了 Wi-Fi 功能，VINCA 阅读器实际上比官方的 USB 适配器更完整，而且可能也更便宜。串行接口似乎是所有卡钳制造商共有的，尽管许多人选择了比微型 USB 更明智的连接器。如果你必须进行[成千上万次类似的测量](https://hackaday.com/2020/11/17/sorting-thousands-of-drill-bits/)，自动读数系统会特别方便。
