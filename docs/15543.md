# 用耳朵呼吸？

> 原文:[https://hackaday.com/2022/12/04/breathe-through-your-ears/](https://hackaday.com/2022/12/04/breathe-through-your-ears/)

尽管心率监测和计步备受关注，但呼吸频率监测却经常被忽视。如今，智能手表开始越来越多地集成呼吸频率监测功能。然而，目前的设备通常只是查看每分钟的呼吸次数，而没有提取呼吸波形中更有趣的特征，这些特征可以让我们比每分钟呼吸次数本身更深入地了解我们的身体。【Davies】和他的团队决定，他们想通过制造一种能够测量呼吸频率的[耳塞来改变这种情况](https://spiral.imperial.ac.uk/bitstream/10044/1/96220/9/breathing_earPPG_final.pdf)。

我们已经看到了一些商业耳塞和耳机，可以测量心率，并根据你的心跳节奏调整你的音乐。[Davies]的项目使用[光电容积描记法](https://hackaday.com/2015/01/11/simple-and-inexpensive-heartbeat-detector/) (PPG)测量每次心跳的血容量变化，通常用于确定心率。事实证明，PPG 也可以用来测量呼吸频率。当我们吸气时，胸腔内的压力降低，这导致中心静脉压降低，并随之降低 PPG 波形的 DC 和交流分量。这些由呼吸引起的 PPG 波形调制比我们看到的由心脏跳动引起的调制要慢得多，这使得它们在普通 PPG 测量设备中经常被忽略。正如你可以想象的那样，[戴维斯的]项目不仅可以用于确定呼吸频率，还可以用于检查呼吸信号的时域变化，这些变化可能表明哮喘、睡眠呼吸暂停和慢性阻塞性肺病(COPD)。

有趣的是，他使用类似的硬件来提取 PPG 信号，就像我们在其他心率监测设备中看到的那样。一个简单的模拟前端，带有一个红外 LED 来照亮皮肤下的血管，一个光电二极管来测量从皮肤下的组织反射的 LED 信号的变化。由于血容量随着呼吸而轻微变化，所以从组织反射的 LED 信号也轻微变化。通过分析反射的 LED 信号中的这些微小变化，他可以提取呼吸波形，并查看患有慢性阻塞性肺病等呼吸系统疾病的人和没有疾病的人之间的信号变化。

我们建议通读一下戴维斯的论文。也许我们可以将他们的分析整合到我们在 Hackaday 上看到的其他 PPG 测量设备中。
