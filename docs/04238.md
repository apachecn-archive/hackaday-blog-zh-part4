# 微型真空吸尘器很烂(好的方面)

> 原文：<https://hackaday.com/2019/09/19/tiny-vacuum-cleaner-sucks-in-a-good-way/>

有时候，一件事的非凡之处不在于它的内容，而在于它的规模。例如，一团纱线并不令人兴奋，除非它是一团巨大的纱线。在天平的另一端，写下你的名字是一件司空见惯的事情，但把它放在一粒米上，在商场就要 10 美元。[托比·巴特森]从 8 岁起就开始制作真空吸尘器——他现在看起来比那个时候老多了。这本身没什么大不了的，但他的机器很小。事实上，他拥有最小真空吸尘器的吉尼斯世界纪录。他的最新成果？Altoid 罐子里的真空！

从电子学上来说，这只是一个开关，一个电池，一个马达。但是如果你正在寻找一个 Dremel 工具项目，你是幸运的。看看下面这个小设备的视频。除了 Altoids 罐，还有一根金属管和一些碎苏打罐。

如果你在美国，你可能不确定材料清单中的某一项。punnet 是杂货店里用来装草莓或西红柿之类东西的小塑料筐之一。

我们一直在想我们能给这个项目增加些什么，但是我们一无所获。它真的不需要任何发光二极管，一个带互联网连接的微控制器就太大了。也许一个 555 可以给你一个高，低设置与一些 PWM 的电机。

也许这可能是一个微小的机器人真空的开始。如果你认为越大越好，总有这个[戴森克隆](https://hackaday.com/2019/05/17/a-cyclonic-vacuum-cleaner-on-a-hackers-budget/)。
