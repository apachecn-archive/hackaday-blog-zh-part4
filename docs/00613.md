# 分解用于跟踪舌头的鼠标传感器

> 原文：<https://hackaday.com/2018/09/11/disassembling-mouse-sensors-for-tracking-tongues/>

我们刚刚在今年的 Hackaday 奖中完成了人机界面挑战，随之而来的是一系列有趣的鼠标和键盘新设计，它们使用看似不太可能的组件，推动了你认为应该可能的事情。最好的例子之一是来自[oneohm] 的项目[Bit。这是一个电脑鼠标，它用一个小小的轨迹板以你从未想过的方式使用。它是一只适合你舌头的老鼠。](https://hackaday.io/project/154493-the-bit)

这个钻头背后的想法是为四肢使用受限的人创造一种输入设备。这有点像 2015 年 Hackaday 奖的获奖者 [Eyedriveomatic](https://hackaday.io/project/5426-eye-controlled-wheelchair) ，但完全是为了放在舌尖上而设计的。

舌头控制鼠标的第一次实验是用黑莓手机上的光学触控板/导航按钮进行的。像现在所有的鼠标传感器一样，这些模块实际上是非常小的，非常蹩脚的相机。[oneohm]拿起一对这样的模块，发现它们有完全不同的内部跟踪模块，所以实验转向 PixArt Imaging 的表面跟踪模块，该模块也用作 Prusa 3D 打印机中的细丝传感器。这个模块很容易连接到一个微控制器，并小心翼翼地应用塑料，嵌入到一个奶嘴。是的，它跟踪舌头并将其转化为光标运动。这是一个舌头跟踪鼠标，它工作正常。

这是 Hackaday 奖的一个很棒的项目。它不仅为人机界面带来了新的技术，而且以一种所有人都可以使用的方式实现。
