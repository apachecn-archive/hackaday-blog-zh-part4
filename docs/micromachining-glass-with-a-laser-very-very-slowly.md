# 用激光微加工玻璃——非常非常慢

> 原文：<https://hackaday.com/2021/01/09/micromachining-glass-with-a-laser-very-very-slowly/>

当谈到机械加工时，首先想到的材料可能是铝、钢或塑料。我们不一定认为玻璃是一种适合加工的材料，至少不是在车床上夹紧的那种意义上。但是玻璃也是一种需要成型的材料，有很多不同的方法来完成。然而，很少有像用激光诱导等离子气泡对玻璃进行微加工这样有趣的事情。(视频，嵌在下面。)

以下视频来自【扎卡里童】。它有点长，但我们发现它充满了信息。这一过程，正式名称为“激光诱导背面湿法蚀刻”，使用激光轰击一箱硫酸铜。当一块玻璃悬浮在溶液表面，激光从顶部透过玻璃聚焦时，一些有趣的事情发生了。

激光的第一个脉冲使溶液蒸发并分解硫酸铜。铜吸附在保护性蒸汽泡内的玻璃表面上，这种保护性蒸汽泡持续足够长的时间，以便第二个激光脉冲出现。这种脉冲加热了吸附的铜和原始气泡中的蒸汽，足以熔化一小块玻璃。随着这一过程的重复，小特征慢慢地被蚀刻到玻璃的下侧。[Zachary]在视频中演示了所有这些，以及当设置有点偏离时会出现什么问题。也有一些伟大的过程中高速镜头，值得门票本身的价格。

我们怀疑这种方法会很快成为主流方法，尤其是因为它需要 50 瓦的 Nd:YAG 光纤激光器。但这是一个有趣的过程，让我们想起了[Zachary]的其他激光探索，如[使用激光和 Kapton 制造石墨烯超级电容器](https://hackaday.com/2020/11/11/laser-induced-graphene-supercapacitors-from-kapton-tape/)。
