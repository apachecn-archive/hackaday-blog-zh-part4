# 3D 打印桥梁各付各的

> 原文：<https://hackaday.com/2018/10/24/3d-printed-bridge-goes-dutch/>

如果你去过阿姆斯特丹，你会知道那里有很多运河，因此也有很多桥。明年，老城中心一座独特的人行桥将投入使用。[不锈钢桥](https://mx3d.com/smart-bridge/)将被 3D 打印，并嵌入许多传感器，这些传感器将收集打印机 MX3D 及其合作伙伴 Autodesk，Alan Turing Institute 和阿姆斯特丹高级都市研究所希望在未来帮助生产更好的 3D 打印结构。这座桥将穿过 Oudezijds Achterburgwal，它靠近该市臭名昭著的红灯区。

由于这座桥与用来打印它的模型完全匹配，科学家们希望能够非常容易地将传感器数据映射到这座桥的虚拟双胞胎上。你可以在下面看到一些关于大桥建设的视频。本月，在[荷兰设计周](https://www.ddw.nl/en/programme/140/mx3d-bridge)期间，参观者有机会走过这座桥，生成一些首批实时数据集。

如果你有一台 3D 打印机，你就会知道很多很酷的东西需要很长时间才能打印出来。这座桥耗时 6 个月，用了 1100 公里的钢丝和 4500 公斤的钢材。你可以想象告诉一个朋友，“我们检查了印刷，灯丝在三月份卡住了。”

我们之前见过 MX3D，最近的[打印自行车](https://hackaday.com/2017/02/03/3d-printed-bicycle-from-stainless-steel/)。如果你想建造小桥，你可以随时使用[技术](https://hackaday.com/2017/11/22/functioning-technic-slj900-bridge-builder/)。
