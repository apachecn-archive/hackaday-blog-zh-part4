# 振动簧片逆变器:可能是你能做的最简单的逆变器

> 原文：<https://hackaday.com/2021/03/29/the-vibrating-reed-inverter-possibly-the-simplest-inverter-you-can-make/>

我们这些在路上工作的人总是害怕被困在没有电的地方，面临着出版截止日期和快速耗尽的笔记本电脑电池之间的竞赛。我们非常幸运地生活在一个便宜、轻便、高效的固态开关模式逆变器可以从汽车点烟器插座为我们提供主电源并扭转局面的时代。在这些逆变器之前，出现了更重的器件，其晶体管以 50Hz 的线路速度开关，在它们之前出现了机电设备，如旋转转换器或振动簧片逆变器。[罗伯特·默里-史密斯(Robert Murray-Smith)研究的就是这最后一种类型](https://www.youtube.com/watch?v=L-vC9SvU2Lo)，使他所定位的最简单的逆变器成为可能。

如果你曾经玩过继电器，你可能会意识到继电器可以像蜂鸣器一样接线，振动簧片逆变器就是利用了这一特性。他拿起一个八进制继电器，用一个小的电源变压器连接起来，就可以立即得到一个非常便宜的逆变器。这并不完美，因为他指出频率不对。除非通过增加电容器来改善电弧问题，否则继电器最终会磨损。但是，如果你发现自己处于一个 MacGyver 式的困境，只有你的垃圾盒来拯救，它确实是一个粗糙和现成的逆变器。

如果你对逆变器感兴趣，了解它们的工作原理可能会对你有所帮助。
