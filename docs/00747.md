# 一小口半的木片

> 原文:[https://hack aday . com/2018/09/25/a-nibble-and-half-of-wood-bits/](https://hackaday.com/2018/09/25/a-nibble-and-a-half-of-wooden-bits/)

如果你熟悉二进制，你需要教一个只懂十进制的人什么？如果你不知道如何计算二进制数，请告诉我们下面的视频是否能帮助你理解二进制数系统的工作原理。如果你对学习或计算二进制不感兴趣，也许你可以欣赏制作一个循环所有 1 和 0 的计数器所涉及的机制(链接到下面显示的视频)。这个机制很简单。每个“1”面板的角上有一个偏心连接的杠杆，所以当它倒置时会悬挂起来，当它直立时会垂到一边，所以它可以旋转相邻的面板。

也许这是一个桌面小玩意，用来炫耀你的木工手艺，或者激光切割机或 3D 打印机的技能。不管它是由什么制成的，它都不会帮助你完成任何工作，除非你是一个想演示二进制离散本质的老师。如果你喜欢木头和碎片，我们有一个[华丽的二元浮木钟](https://hackaday.com/2018/07/02/driftwood-binary-clock-is-no-hollow-achievement/)让你大饱眼福。与此同时，如果处理数字的模拟方法适合你，我们有用纸模型进行的[二进制数学。](https://hackaday.com/2017/06/12/paper-circuit-does-binary-math-with-compressed-air/)
