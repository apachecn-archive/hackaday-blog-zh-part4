# 移植厄运，但方向相反

> 原文:[https://hack aday . com/2022/07/20/porting-doom-but-in-the-opposite-direction/](https://hackaday.com/2022/07/20/porting-doom-but-in-the-opposite-direction/)

DOOM 最初是为 MS-DOS 发布的，是更广泛的第一人称射击游戏类型的支柱游戏之一。它也成为了一种被移植到世界上任何一个奇怪平台上的迷因。现在，哥斯达黎加的一群开发者已经找到了一种方法来扭转这个笑话——《T2 》,将一款旧的手机游戏《毁灭 T3》《T4》移植到 PC 上。

这款游戏名为 *DOOM RPG* ，是 2005 年为 BREW 和 Java 兼容手机开发的。一个名为 GEC.inc 的小组已经将该游戏移植到 Windows 上，并在世界末日论坛上概述了他们的工作[。](https://www.doomworld.com/forum/topic/129997-doom-rpg-port-reverse-engineering-update-version-02/)与许多此类项目一样，该端口免费提供，但由于版权原因，不包括原始游戏文件本身。你必须自己找到游戏数据，并将其与该组织在论坛上发布的文件相结合，以使其在现代 PC 上工作。

对于那些错过了基于*毁灭*宇宙(*毁灭宇宙)的回合制角色扮演游戏的人来说？)，*今天是个好日子。你再也不用怀念过去那种古老、笨重的 Java 智能手机了。现在，你可以在一个不那么糟糕的平台上玩游戏，并聆听独特而引人注目的 MIDI 风格的配乐。

毁灭端口现在很热门，不管是被遗忘的苹果操作系统还是 T2 的世嘉街机硬件。休息后的视频。
