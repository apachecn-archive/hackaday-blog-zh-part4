# 了解 vans

> 原文:[https://hackaday.com/2020/06/11/learning-about-vnas/](https://hackaday.com/2020/06/11/learning-about-vnas/)

我们生活在一个成为电子极客的好时代。过去，只有我们当中最富有或最精明的人才拥有一台真正好的示波器，而现在，拥有一台几十年前会花费一大笔钱的示波器、一台逻辑分析仪、任意波形发生器，以及一台曾经是超级计算机但仍不负债的东西是完全可行的。对于从事射频电子工作的人来说，一个比较酷的设备是矢量网络分析仪(VNA)，它曾经是舶来品，但现在可以以很低的价格买到。但是你用它做什么呢？[【w2ew】有答案](https://www.youtube.com/watch?v=Sb3q8f0NBZc)。

我们一直期待来自【W2AEW】的视频。即使我们知道他所涉及的主题，我们通常也会发现一些新的或有趣的东西。像他所有的视频一样，这个非常实用。没有太多的绘画，但大量的范围拍摄和实验。

毕竟，你知道电容和电感会引起不同的相移，但实际上在现场切换元件时看到它，是令人难忘的。此外，当他做一些“图表”时，我们真的很喜欢他的指针。

我们看了大约 50 美元的 NanoVNA。如果你有花不完的钱，我们喜欢昂贵的[模拟探索 2](https://hackaday.com/2018/06/20/analog-discovery-2-as-a-vector-network-analyzer/) ，它不仅是一个 VNA，但也有很多其他仪器。
