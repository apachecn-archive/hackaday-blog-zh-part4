# 无人机能推自行车吗？

> 原文:[https://hackaday.com/2022/06/19/can-a-drone-push-a-bike/](https://hackaday.com/2022/06/19/can-a-drone-push-a-bike/)

这听起来像是中西部工程师可能会问的一个修辞性问题，类似于“你能修复这个糟糕的 PCB 旋转吗？”[汤姆·斯坦顿]开始回答题目，并最终制造了一辆带有无人机马达的电动自行车。

你可能会想，*马达就是马达；有什么大不了的？*但是无人机马达和普通电动自行车马达的用途完全不同。无人机马达的转速为 30，000 RPM，电动自行车轮毂马达的转速通常为 200-300 RPM，但要大得多。此外，无人机马达在大风扇的吹拂下会短暂加速，而电动自行车马达几乎可以连续运转。

第一步是使用齿轮和滑轮来降低电机的转速，以提供更大的扭矩。稍加 CAD 和 3D 打印之后，[Tom]已经准备好了一个可以尝试的设置。然而，马达很快就烧坏了。第二代发动机略大，齿轮减速更多，表现非常出色。在正式的电动自行车和无人机自行车之间的比赛之后，线圈几乎融化了。

如果你正在考虑让你的自行车电动化，我们有一些建议。我们将免费提供第二条建议:使用比无人机马达更大的马达，即使它在技术上是可行的。休息后的视频。
