# 用自制的光纤耦合器正确地传输激光

> 原文：<https://hackaday.com/2022/12/09/properly-pipe-laser-light-around-with-homebrew-fiber-couplings/>

很少有人能拿起一个便宜的激光笔，而不是像光剑或相位器一样挥舞它，并配有音效。“皮尤-皮尤”因素让投射激光束变得有趣，即使这不是最安全的做法，也不是将光从一个地方传递到另一个地方的最有效方式。

我们怀疑[Les Wright]已经在他的实验室里完成了不止几个激光项目，包括他最新的激光光纤耦合实验。下面的视频充满了连接廉价通信级光纤组件的技巧，尽管它们的终端是标准化的，但并不总是容易与他收集的激光器一起使用。挑战之一是包层内的光纤通常非常小，只有 9 微米。在没有对齐帮助的情况下，这是一个很小的目标，而[Les]使用了一系列的攻击来完成。

视频的核心部分演示了如何使用廉价的光纤故障定位器和简单的光具座设置来将任何激光器与光纤精确对准。一对可调节的镜子允许他精确地重叠故障定位器和目标激光的光束。效果可能很有趣；我们不知道通信级光纤会通过包层泄漏如此多的光，弯曲半径限制非常明显。[Les]在后续视频中，我们期待看到一些实用的传感应用。
