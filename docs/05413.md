# 机器人小马迎接黑客空间的访客

> 原文:[https://hack aday . com/2020/01/24/robo-pony-greets-hacker space-visitors/](https://hackaday.com/2020/01/24/robo-pony-greets-hackerspace-visitors/)

机器动物伴侣曾经风靡一时，尽管它们有限的个性和恼人的声音效果经常很快让它们被扔进垃圾箱。这使得他们更适合黑客攻击。[David Bynoe]有一只小奶油糖果需要一个新家，他决定让小马在他当地的 hackerspace 工作。

乳脂糖小马是一种迷人的家畜，但是一旦它的皮被剥掉，就会变得非常危险。安装在一个牌匾上，小马有三个 PIR 传感器来检测运动。这些传感器被用来让小马充当迎宾员，当人们进入黑客空间时醒来，并在房间里跟着他们。额外的硬件通过使用浮动电容器和继电器来激活原始的电容式触摸传感器，从而与 pony 的库存电子设备进行接口。最后一件作品以金色涂料和一些 RGB 眼睛完成。

这是一个有趣的项目，给[温哥华 Hack Space](http://vanhack.ca/wp/) 增添了一点个性，我们确信它会受到会员们的喜爱。我们以前见过其他的配套玩具黑客，[Furby 总是项目的成熟目标](https://hackaday.com/2017/01/21/taking-control-of-your-furby/)。休息后的视频。
