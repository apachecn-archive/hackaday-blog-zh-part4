# 使用熔接机维修三星电视的电缆

> 原文：<https://hackaday.com/2022/05/23/using-a-fusion-splicer-to-repair-a-samsung-tvs-cable/>

一些三星电视配有一个名为 One Connect 的系统，所有外部电缆都连接到一个单独的盒子上，这样只有一根小信号线连接到电视。在一些版本中，连接电视和连接盒的电缆是纯光纤电缆，几乎透明，因此易于隐藏。

然而，细光纤电缆很脆弱；当[Elecami Wolf]以非常低的价格获得其中一台电视时，原来这是因为它的一根连接线断了。替换电缆非常昂贵，所以[Elecami Wolf]继续研究光纤电缆的内部工作原理，并找出如何修复损坏的光纤电缆。

电缆由四对塑料涂层玻璃纤维组成，分别连接到两端粗连接器内的接收器和发射器。修复电缆需要两件事:弄清楚哪些光纤应该相互连接，以及一种可靠的方式将它们连接在一起。

第一个问题相当困难:简单的 1:1 连接不起作用，所以需要花点功夫来找出正确的连接设置。一个聪明的技巧是将一台摄像机对准一根工作中的电缆，比较两端的闪光灯；这有助于确定四对中的两对的正确顺序。对于其他两个，结合逆向工程的电子电路和一些系统的试验和错误产生了一个完整的电路图。

对于第二部分，[Elecami Wolf]拜访了一位光纤专家，他借给他一个熔接机。这是一台相当精巧的设备，它半自动地将两根光纤连接在一起，并用电弧将它们焊接起来。一旦这一切完成，这是一个问题，涵盖拼接，以保护他们免受急弯，和花式电视再次工作。

虽然不是每个人都有机会接触到多模式融合拼接机，但[Elecami Wolf]的视频提供了对基于现代光纤的消费电子产品工作原理的迷人见解。这可能是我们见过的第一次光纤拼接尝试；但是如果你想把光纤接到你的电路上，[这个球形透镜设置](https://hackaday.com/2022/02/21/a-ball-lens-for-optical-fiber-coupling-on-the-cheap/)是一个巧妙的技巧。
