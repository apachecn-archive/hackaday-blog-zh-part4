# 以下是 Hackaday 奖通配符挑战赛的十名决赛选手

> 原文:[https://hackaday . com/2021/11/03/以下是 hackaday-prize-wild card-challenge 的十名决赛选手/](https://hackaday.com/2021/11/03/here-are-the-ten-finalists-for-the-hackaday-prize-wildcard-challenge/)

2021 年 Hackaday 奖的每个阶段都要求设计师在各个领域重新想象传统解决方案，从机器人到辅助设备。但是对于重新激活通配符，[今年奖项](https://prize.supplyframe.com/#section-challenges)的第五次也是最后一次挑战，*反思、刷新、重建*的主题可以应用于参赛者想要的任何东西。今天，我们很高兴地宣布十个通配符项目已被选中，赢得 500 美元，并进入决赛。谁会赢得今年的冠军？我们将在几周后的[黑客大会](https://www.eventbrite.com/e/hackaday-remoticon-2021-tickets-172183193567)上找到答案！

MetaSense 项目是一个很好的例子，说明如何使用新技术来重新思考我们通常认为已经解决的问题。该项目利用多材料 3D 打印来生产导电电池，这些电池可以根据物理变形改变其电容。通过一些巧妙的几何结构，这些电池可以链接在一起，产生单个部件的设备，可以代替传统的拨动开关，操纵杆，压力传感器，甚至加速度计。

[![](../Images/16f4aa4f545f9738d633130c78eb3ccd.png)](https://hackaday.com/wp-content/uploads/2021/11/wildcard21_detail.jpg) 谈到 3D 打印，直接颗粒挤压机项目想象未来[桌面打印机不再局限于使用人造细丝卷](https://hackaday.io/project/181165-my-machinery-2021-direct-granules-extruder)。关键是一个强大的挤出机设计，可以足够快地研磨塑料颗粒，直接将其送入传统 3D 打印机的热端。这不仅意味着可观的运营成本节约，因为按重量计算，生塑料颗粒比细丝便宜得多，而且可能允许使用更奇特的塑料混合物甚至回收材料进行打印。

一些项目甚至让我们重新思考对个人黑客来说什么是可能的。WiFiWart 利用了一台微型单板 Linux 计算机，它是由一个人从头开始设计和建造的，只使用免费和开源软件。无论是这个渗透测试工具已经将一台完整的 Linux 计算机和两个 WiFi 适配器打包到一个手机充电器大小的盒子中，还是它是由一个专门的黑客使用免费工具完成的，你都会忍不住对它留下深刻的印象。
