# 使用 STEM 学习工具包学习水净化技术

> 原文：<https://hackaday.com/2019/12/07/learn-water-purification-techniques-with-this-stem-learning-kit/>

我们看到许多伟大的 STEM 教育项目。这些项目有办法变成更大的东西。有多少商业设备和机器是基于 Raspberry Pi 和 Arduinos 的？[Ryan Beltrán]正在用普通的材料教人们如何净化水。这个特殊的套件演示了一种叫做电凝聚的水净化过程。

当电流通过悬浮在水中的两个电极时，会改变悬浮固体的表面电荷。这使得固体、金属和油凝结在一起，这使得它们更容易处理和清洗。

该套件由一个罐子、电极、一些 3D 打印部件和一个预闪存 Arduino 组成。还有盐和过滤器来完成净化过程。学生可以立即开始实验，如果他们受到启发，他们将拥有尝试更先进技术的所有工具。

STEM 工具包通常严重倾向于机器人学或计算机科学，但是还有许多广阔而有趣的领域需要解决问题。
