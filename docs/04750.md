# 当你可以制造玩具时，为什么还要买呢？

> 原文:[https://hack aday . com/2019/11/08/why-buy-toys-when-you-can-build-them-instead/](https://hackaday.com/2019/11/08/why-buy-toys-when-you-can-build-them-instead/)

像许多突然发现自己是父母的有创造力的人一样，[Marta]想为他的孩子做一些特别的东西来玩。任何人都可以购买现成的电子玩具，但如果你有能力按照自己的方式设计一个，为什么不去做呢？但是，即使与黑客父母设定的相当高的标准相比，我们也不得不承认在“Marta Musik Maschine”上投入的时间、思想和努力绝对是惊人的。

[![](../Images/9cf9f2a17cac184d1213898478d98f0b.png)](https://hackaday.com/wp-content/uploads/2019/10/marta_detail.jpg)【Marta】受到各种商业产品的启发，这些产品使用 RFID 和其他技术来识别孩子正在玩耍的角色并做出相应的反应。但由于他不想被局限在某个特定公司的生态系统中，而且摆弄玩具似乎会遭到创造者的反对，所以他决定推出自己的版本。

在 Musik Maschine 的专用网站上的许多帖子中，[Marta]以绝对精致的细节解释了他对玩具的每个设计考虑的思考过程。每篇文章都被有益地分解为最终玩具的每个子系统，可以说是足够详细和完整，足以作为它们自己的独立项目。即使你不想进入 DIY 电子玩具的世界，这里几乎肯定会有一个让你着迷的帖子。从 Python 代码与 arcade 按钮的接口到设计 3D 打印外壳的技巧，这里真的有适合每个人的东西。

黑客的[孩子经常是邻居羡慕的对象](https://hackaday.com/2018/02/07/motorized-mini-excavator-rises-from-sheets-of-plywood/)感谢[他们父母提供的独一无二的玩具](https://hackaday.com/2017/11/03/mini-millennium-falcon-is-envy-of-the-neighborhood/)考虑到【玛尔塔】对一个蹒跚学步的玩具的投入程度，我们迫不及待地想看看他下一步会做出什么。
