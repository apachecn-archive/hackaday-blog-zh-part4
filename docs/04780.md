# 易于组装的 LED 立方体

> 原文:[https://hack aday . com/2019/12/15/a-led-cube-designed-for-easy-assembly/](https://hackaday.com/2019/12/15/a-led-cube-designed-for-easy-assembly/)

LED 立方体既迷人又有趣，但通常很难建造。对于[【伯克瑟斯】设计简洁的立方体来说，情况并非如此。](https://hackaday.io/project/165465-cubex8)

许多立方体被放在一起，形成一种精致的雕塑风格。传统上，led 的引线被艺术地弯曲，然后[个小时被花费在未来的彩虹博格立方体上。这种构造更像是主板或背板设计。led 是表面安装单元，回流到耙形 PCB 上。在每个“耙子”的底部都有一个直角的公头。然后焊接到基板上，形成可靠的机械结合。](https://hackaday.com/2016/02/05/building-the-worlds-smallest-rgb-led-cube/)

这种方法有一些缺点。例如，PCB 会在某些视角遮挡 led。然而，这可以通过在房间中仔细放置来缓解，或者在一种变化中，将立方体安装在不同的方向上，这样耙是水平的而不是垂直的。

无论如何，我们欣赏这个老项目的新成果，并且可以肯定地看到它比那些需要几个星期的下午才能完成的工具包更有普遍的吸引力。
