# 向任何方向飞行的无人驾驶飞机

> 原文：<https://hackaday.com/2020/04/14/the-drone-that-flies-in-any-orientation/>

现代无线电控制的多旋翼无人机可以非常敏捷，但只能围绕偏航轴改变方向，同时保持在大致相同的位置。苏黎世联邦理工学院的研究人员再次建造并测试了六自由度可控运动的多转子，这次极大地提高了效率。

我们之前报道过苏黎世联邦理工学院的[类似设计](https://hackaday.com/2017/06/13/harrier-like-tilt-thrust-in-multirotor-aircraft/),它是一架带有有限旋转臂的六轴直升机。这种新设计也是一种六轴直升机，但每个旋转臂上有两个同轴电机。与之前的设计相比，每个臂的旋转范围都有所增加，超过了 360 度。凭借旋转范围和非常复杂的控制系统，无人机可以有效地在任何方向上飞行，同时仍然能够在任何方向上施加有效的扭矩或线性力。这为无人机可以执行的任务开辟了许多可能性，如近距离工业检查，使用工具或拉动电缆，同时保持转子清洁。

在拉紧电机电缆之前，臂确实有有限的旋转量，但控制系统会跟踪这一点，并可以在运动过程中或运动后松开。休息后请观看视频，了解它的运行情况。完整的[科学论文](https://arxiv.org/abs/2003.09512)不是轻松阅读，但绝对有趣。我们期待着看到这些类型设计是否以及何时在现实世界中得到应用。

毫无疑问，我们的未来会有很多无人机，迄今为止最成功的项目可能是卢旺达和加纳的[滑索固定翼无人机](https://hackaday.com/2017/05/12/automate-the-freight-medical-deliveries-by-drone/)，自 2016 年以来，它已经运送了超过 35000 次紧急医疗用品。
