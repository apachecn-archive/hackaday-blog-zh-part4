# 将您的电脑排放到室外

> 原文:[https://hackaday.com/2022/07/10/venting-your-pc-outside/](https://hackaday.com/2022/07/10/venting-your-pc-outside/)

随着现代游戏机中 CPU 和 GPU 的功耗需求不断上升，它们正迅速变得越来越像一个碰巧玩游戏的空间加热器。如果你在一个关着门的狭小空间里使用你的电脑，你可能会发现你办公室的温度上升得相当快。对此的一些解决方案包括风扇、窗式空调设备，或者将计算机移动到其他地方并将电缆布线回办公室。[Linus Tech Tips]的优秀人员尝试了一种有点另类的东西，他们把整个电脑放在一个盒子里。

在 Hackaday，我们通常不涉及[Linus 技术技巧],但我们认为这种方法有些新颖。电脑机箱有许多排气扇和孔，所以很难从一个点提取热空气。因此，在购买了一个滑稽的大而便宜的“植物”生长帐篷后，他们可以封闭 PC 并通过一些绝缘管道散热。激光切割的转接板和 3d 打印的软管连接器允许软管位于窗口中，以向外通风。一个内置风扇将帐篷里所有需要的空气抽到外面。最终，当一些基准运行时，房间内的温度保持寒冷，但有猜测认为风扇从公寓的其他地方吸入空气以释放 PC 的热量。我们希望看到一个更封闭的系统，外面有一个热交换器。

或许他们可以借用[【Diy Perks】的贝罗 PC build](https://hackaday.com/2021/07/06/bellow-cooled-pc-is-a-well-engineered-display-piece/) 并把水管直接连到上面，摆脱帐篷。休息后的视频。
