# 末日回顾:金属 25 年

> 原文:[https://hack aday . com/2018/12/28/doom-retrospect-金属 25 年/](https://hackaday.com/2018/12/28/doom-retrospective-25-years-of-metal/)

金属是许多东西。一种坚硬粗糙的材料，在火中锻造后变得足够锋利，可以切割任何东西。与其同名的音乐同样尖锐，散发出一种不屈的态度，试图将装腔作势的人与真正的追随者区分开来。金属是一种不盲目遵循规则的情绪，是一条不太通向黑暗面的道路。在电子游戏中，没有什么比毁灭更具金属元素了。

《毁灭战士》的创造者 id Software 在 20 世纪 90 年代一直致力于改变人们对电脑游戏的看法。与主机游戏相比，当时的游戏既死板又缓慢。图形保真度在 PC 上技术上更胜一筹，但没有其他开发者能在像 id 这样的游戏中确定运动。该团队以他们的指挥官基恩系列([在一次失败的超级马里奥兄弟 3 PC 演示](https://www.pcgamer.com/take-a-look-at-id-softwares-1990-super-mario-bros-3-demo/)之后出现)以及定义 Wolfenstein 3D 的类型而闻名，但没有什么能超过 Doom。在一个已经充斥着“态度”的时代，毁灭战士建立了自己的身份。忧郁的灯光，怪诞的怪物设计，标志性的推进战斗，以及 Soundblaster 可以处理的所有 MIDI 吉他；1993 年，厄运看上去并感觉到比其他一切都重要。

那年 12 月，参议员乔·利伯曼和赫伯·科尔举行听证会，公开谴责在美国销售的视频游戏中包含暴力内容。大部分的争论试图把视频游戏产业和它的开发者描绘成试图腐蚀国家年轻人的离经叛道者。Id Software 的反应就像是竖起了可以想象的最大的中指，第二天就向全世界发布了《毁灭战士》。四分之一世纪后，人们仍在谈论它。

> “我感到非常自豪的是，所有 32 位处理器都运行了 Doom，我认为这是它开源的一个重要方面。”
> 
> *–**id 软件**联合创始人约翰·卡马克*

## 但是它会运行毁灭吗？

![Lost Episodes of DooM Book Cover Square](../Images/c5e28a00d79de7d8e98e713c95135ddb.png)

*The Lost Episodes of Doom* by Jonathan Mendoza came with a floppy disc featuring 24 episodic Doom WADs.

甚至在将公司总部设在德克萨斯州梅斯基特市东塔楼 666 号套房之前，id Software 就已经看到了用他们的商业理念颠覆个人电脑领域的潜力。除了最初的邮购版本之外，《毁灭战士》还发布了多个零售版本，id 还将《毁灭战士》游戏引擎授权给了其他开发者，结果是《异教徒》(金属)、《己烯》(金属)和《Chex Quest》(肯定不是金属)。

Id 热衷于他们的粉丝们想要为他们之前的游戏 Wolfenstein 3D 创建自定义关卡的愿望，所以一个更友好的设计是 Doom 代码库不可或缺的一部分。游戏的 WAD(数据都在哪里？)包文件系统允许轻松安装自定义级别。只需要安装一个单独的文件就可以让粉丝们在当时的各种 BBS 和 FTP 站点上分享关卡。这些作品的链接随着《毁灭战士》的续集来来去去流传了好几年，但这款游戏最持久的影响是该公司如何让那四张原版软盘寿终正寝。

到 1997 年厄运降临世嘉土星的时候，这款游戏已经出现了十几个版本，id 软件也进入了他们的雷神之锤系列。12 月 23 日，公司[在开源许可下发布了 Doom](https://archive.org/details/Doom_source_DOOM_Source_Code_License) 的原始源代码，这是前所未有的举动。把一切都还给让 id Software 成为精英级开发者的社区似乎是另一个时间线之外的努力。在一个充满秘密的行业，赠送如此有价值的知识产权的行为可能是 id Software 公司最强硬的举动。

从相机到计算器，从打印机到钢琴，或者任何有显示屏和 12 MB 存储空间的东西都遭遇了厄运。将 Doom 移植到另一个设备的行为已经成为黑客社区中的一种伪通行权。它已经到了每一个时代的口语短语，“它运行厄运吗？”据说另一场黑客胜人一筹的元游戏随之而来。足以说明的是，同样的“我们对抗世界”的心态已经超越了游戏本身。

> “我相信 Doom 最重要的遗产是它的社区，是那些通过创造 mod 和工具让它存活了 25 年的人们。”
> 
> *–**id 软件**联合创始人约翰·罗梅洛*

## 这么多年过去了还在破碎

![id Software Creators Group Photo from John Romero Archive](../Images/ad4bdb3d331c0e2f7efc8a9275060f17.png)

Promotional photo of id Software’s original crew with the Spiderdemon and Mancubus models.

自 1993 年以来，《毁灭战士》系列已经进行了几次重塑(2019 年《毁灭战士:永恒》上映时还会有更多)，但原版仍然是其创作者之一的首选。《毁灭战士》最初的程序员之一约翰·罗梅洛最近宣布，他将发布自己的《毁灭战士》精神继承者，名为 SIGIL 。用罗梅罗的话来说，这九个新级别的免费 MegaWAD 扩展是“爱的劳动，提醒我们在 id 工作的所有美好时光。”它承诺在《毁灭战士》第四集结束后将玩家直接送回地狱。

此外，这个项目来自一个与 id 软件早期紧密相连的人，这证明了他们的游戏对文化的影响。动作平台游戏有《指挥官基恩》,在线多人游戏有《雷神之锤》,但《毁灭战士》巩固了该公司作为第一人称射击游戏最重要设计师的地位。鉴于目前的游戏格局，如果曾经存在一个既有影响力又好玩的游戏名单，那么它在没有厄运的情况下是不完整的。它继续激励着人们追求各种形式的改装，永远不要满足于“库存”。最金属的游戏，就像它的许多角色一样，根本不会死。它将继续存在，就像 25 年前它重新定义了 PC 游戏一样。
