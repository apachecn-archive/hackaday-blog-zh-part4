# 小规模的疯狂的麦克斯:Danny Huynh 的反乌托邦电子动画

> 原文：<https://hackaday.com/2021/09/06/small-scale-mad-max-danny-huynhs-dystopian-animatronics/>

黑客精神在后启示录小说中一直活跃，因为角色们从他们能找到的任何垃圾中拼凑出新玩意儿。虽然这些在现实中可能并不总是实际可行的，但它们的主要目的通常是为了看上去像个角色。对于[【Danny Huynh】的后启示录电子动画创作](https://www.youtube.com/channel/UCIO3WzVUHOEkJSvy14LtT1w/videos)来说，这绝对是一个例子，看起来它们可以直接插入《疯狂的麦克斯》或《辐射尘》。

[Danny]是一个狂热的 RC 爱好者，所以许多模型都是[高度定制的现成 RC 汽车](https://www.youtube.com/channel/UCIO3WzVUHOEkJSvy14LtT1w/videos)。然而，真正吸引眼球的是这些模型中栩栩如生的人物。他们的手和脚随着方向盘和油门移动，在[摩托车制造](https://www.youtube.com/watch?v=r3GdRBoW4Uw)中，他们经常在转弯时倾斜。其他值得注意的建造包括[六脚出租车](https://www.youtube.com/watch?v=gpyzpR9iDso)和几个[电子动画乐队](https://www.youtube.com/watch?v=OPulCe0DjTg)。

所有的车辆都是电动的，但看起来[丹尼]经常包括一个音频模块来模拟轰鸣的发动机。他大量使用伺服系统和联动装置来控制角色的动作，电线和电子设备被油漆或车身小心翼翼地隐藏起来。

有了今天所有可用的 CGI 技术，伟大的电子动画制作就像一个怪异的[栩栩如生的心脏](https://hackaday.com/2020/01/19/be-still-my-animatronic-heart/)或[一个会说话的尼古拉·特斯拉](https://hackaday.com/2020/07/08/animatronic-nikola-tesla-sets-the-record-straight/)都让人印象深刻。
