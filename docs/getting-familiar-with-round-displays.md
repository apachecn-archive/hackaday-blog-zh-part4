# 熟悉圆形显示器

> 原文：<https://hackaday.com/2021/12/11/getting-familiar-with-round-displays/>

曾几何时，创客项目仅限于使用简单的字符显示器或那些从流行的诺基亚中回收的显示器，主要是由于成本的原因。如今，只需几块钱就能买到一台小型有机发光二极管或液晶显示器。然而，你可以变得更好，而[伏特先生]正是通过对漂亮的圆形显示器的探索做到了这一点。

使用圆形显示器并不一定很难，市场上有很多很棒的产品。[伏特先生]经历了各种选择，从你可以连接到微控制器的廉价裸屏，到直接用于树莓 Pi 甚至直接 HDMI 输入的更大型号。许多是相当高的分辨率，看起来特别漂亮，当适当的艺术品驱动。

然而，当你远离传统的矩形屏幕时，有一些小技巧会派上用场。某些型号上的屏幕支架可能没有与圆形显示屏的中心直接对齐，如果没有考虑到这一点，可能会导致结果看起来*与*不一致。

同样重要的是要记住，使用笛卡尔坐标，圆形显示器仍然像方形显示器一样被驱动。尝试使用边缘带有界面元素的软件是很难的，因为很多都是在屏幕外呈现的。相反，圆形显示器最适合用于显示专门制作的内容，而不是用于传统软件的矩形屏幕。

从根本上说，圆形显示器是一个整洁的东西，因为它们允许[建造优雅的项目](https://hackaday.com/2021/10/25/tackle-the-monkey-raspberry-pi-gets-round-screen/)，而不必用模糊的挡板或外壳等欺骗手段来抽象一个圆。休息后的视频。
