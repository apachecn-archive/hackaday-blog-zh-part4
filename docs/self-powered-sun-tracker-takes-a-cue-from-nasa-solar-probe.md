# 自供电太阳追踪器从美国宇航局太阳探测器得到启示

> 原文：<https://hackaday.com/2018/11/18/self-powered-sun-tracker-takes-a-cue-from-nasa-solar-probe/>

让太阳能电池板跟踪太阳一直是一个有趣的问题，它导致了一些复杂的解决方案。如今，使用 GPS 和伺服系统的控制器似乎很受欢迎，但正如美国宇航局启发的太阳追踪器所示，这项任务不必过于复杂。

从下面的视频中可以明显看出，[NightHawkInLight]的太阳能追踪器目前只是一个概念验证，但它肯定显示出了前景。它是基于美国宇航局的掠日探测器帕克太阳探测器，该探测器使用探测器后部的传感器来操纵飞行器，以防止阳光从防护罩的侧面偷看。[NightHawkInLight]的设计通过使用太阳能电池作为四个传感器，进一步简化了这一方案。安装在遮阳篷后面的电池直接连接到控制方位角和仰角的小齿轮马达。当一个电池看到太阳时，它给电机供电，电机将电池板向右移动，再次遮挡太阳，从而切断电机的电源。

[NightHawkInLight]提到了一个显而易见的问题，当太阳升起，阵列在上次日落后指向完全相反的方向时会发生什么，但我们仍然不确定他的解决方案——一个更大的阵列，跟踪单元安装得更远——是否可行。我们也不确定它将如何扩展到需要更大电机来移动的更大阵列。当然，我们已经见过用更复杂的追踪器来处理这样的阵列，但是我们希望这种设计的简单性可以在现实世界中应用。
