# 最差 WiFi 天线的锈钉奖

> 原文:[https://hack aday . com/2020/04/23/the-rusty-nail-award-for-worst wifi-antenna/](https://hackaday.com/2020/04/23/the-rusty-nail-award-for-worst-wifi-antenna/)

一般来说，一分钱一分货，如果你买的是一元店的 WiFi 天线，声称可以提供 12 dBi 的信号增益，那么当[一个生锈的钉子比它](https://www.youtube.com/watch?v=uObC2VHCFKU)表现得更好时，你不应该感到惊讶。

在罗马一家商店引起安德鲁·麦克尼尔注意的平板天线是营销天才的奇迹。他说，引起他注意的是 Windows Vista 兼容性标签，这是一个真正让这颗宝石过时的策略。当它被设计成插入 WiFi 适配器上的 SMA 插孔时，完全不相关的指示也是如此。[Andrew]的拆卸并不令人振奋，只露出一块印刷电路板，上面有一些明显随机的痕迹，用作偶极子的元件。我们发现有趣的是，PCB 丝印标签的通孔为 H1 到 H6，这是一个伟大的方式，使一个不拥挤的板似乎更重要一点。

测试结果并不比拆卸更令人印象深刻。网络分析仪扫描显示，天线根本没有调谐到 2.4 GHz 的 WiFi 频段，天线连接到适配器的实际测试无法嗅出任何本地热点。为了强调这种天线有多糟糕，[Andrew]用一个 SMA 连接器和一个生锈的钉子拼凑了一个简单的天线，它轻松地胜过了平板天线。

我们之前已经看过很多[安德鲁·麦克尼尔]的 WiFi 天线视频，比如[他的雨伞和铁罐碟](https://hackaday.com/2018/04/28/umbrella-and-tin-cans-turned-into-wifi-dish-antenna/)。我们喜欢他对 WiFi 爱好者和批评者的疯狂主张所带来的理智，尤其是当他表明 [WiFi 不会杀死室内植物](https://hackaday.com/2019/02/28/does-wifi-kill-houseplants/)。我们不禁想知道他是怎么看待[现在的 5G 傻逼](https://hackaday.com/2020/04/08/on-5g-and-the-fear-of-radiation/)的。
