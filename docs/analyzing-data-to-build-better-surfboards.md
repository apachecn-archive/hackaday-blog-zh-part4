# 分析数据以制造更好的冲浪板

> 原文：<https://hackaday.com/2019/09/14/analyzing-data-to-build-better-surfboards/>

在冲浪的世界里，可用的设备就像爱好者本身一样多种多样。不同的主板针对不同的条件和风格进行了优化，业界不断创新，以实现更高的性能。 [[DARK-labs]旨在将数据分析引入该领域，帮助创建个性化的电路板。](https://hackaday.io/project/166977-data-generated-surfboards)

目标是使用嵌入冲浪板的传感器网络来分析特定冲浪者的风格。这些数据随后被用于识别诸如姿势和脚的偏好等特征，这些特征随后可被用于优化板的设计以适应需要。一旦一个 CAD 模型按照这些指导方针被创建，它就可以被 CNC 加工并变成一个成品板，准备好迎接挑战。

这是一个项目，我们预计将捕获许多冲浪者的兴趣，我们不会惊讶地看到这个概念在未来几年取得更大的进步。我们还展示了一些其他的主板黑客——[这种电子构建尤其引人注目。](https://hackaday.com/2019/01/28/shred-the-gnar-without-paddling/)
