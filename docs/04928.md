# 黑客日播客 044:超级 2019 特别节目

> 原文：<https://hackaday.com/2019/11/22/hackaday-podcast-ep-044-superconference-special/>

Hackaday 一半的写作人员都在本周末的 2019 Hackaday Supercon 上，我们自己的 Kerry Scharfglass 借此机会采访了所有人。与此同时，埃利奥特在徽章黑客仪式前两个小时左右在烙铁周围闲逛，收集成功和失败项目的故事。

把这两者放在一起，你就有了一个音频拼贴画，让你至少可以窥见超现实生活的一个方面，并让你有机会为你每天在这里读到的文字配上声音！
