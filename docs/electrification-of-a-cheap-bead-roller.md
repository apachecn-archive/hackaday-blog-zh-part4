# 廉价胎圈辊的电气化

> 原文：<https://hackaday.com/2020/02/17/electrification-of-a-cheap-bead-roller/>

我们都有旧的项目，这些项目可能并没有真正交付使用，有时它们会尘封多年。不过它们也有用处，当你真的需要那部分的时候，你可以把它从被遗忘的项目中取出来。这就是[Mustie1]在一个被遗忘的电动自行车项目中所做的，[他拿走了它的马达，并用它来自动化他的钢丝圈滚轮](https://www.youtube.com/watch?v=kANvNkBwfnU)。

胎圈辊是汽车车身领域使用的一种工具，用于将胎圈(连续凹陷)压入一块金属板。他在台虎钳上安装了一个便宜的滚轴，通过手柄操作。不幸的是，工具的尺寸意味着它很难在滚动精确珠子的同时操作，因此需要改进。

他先是考虑用无绳电钻，后来又想起了电动自行车项目。它的齿轮马达来自一个电动轮椅，当然拥有正确的速度，但他需要一个合适的链轮。这是由他买的一辆报废的引擎辅助自行车提供的，事实证明非常适合这项工作。最终的自动化滚筒使用安装在脚踏开关上的无线钻机的触发控制器，滚筒安装在由一件健身设备改造的支架上。结果是一个有用的，最重要的是可控的工具，可以运行一个完美的珠在任何形状的金属片上。

令人惊讶的是，这是我们在这里介绍的第一个珠辊，但钣金工作是硬件黑客项目中的一个常数。首先，请阅读我们的金属板弯曲指南。
