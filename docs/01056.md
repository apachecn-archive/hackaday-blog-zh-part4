# 轮子上的暗箱出现在开阔的路上

> 原文:[https://hack aday . com/2018/10/24/camera-obscura-on-wheels-hits-the-open-road/](https://hackaday.com/2018/10/24/camera-obscura-on-wheels-hits-the-open-road/)

暗箱是一种非常简单的装置。由一个暗室组成，只有一个针孔让光线进入，它将图像聚焦在其内表面。如果你想拍永久拷贝，只要在里面装上感光胶片，管理好曝光时间就可以了。听起来像普通相机，对吧？区别在于规模——暗箱足够大，人类可以站在里面观看图像。通常，它们是大型的固定房间。[【物理女孩】用租来的箱式卡车](https://www.youtube.com/watch?v=qIp9kItDUh8)做了一个暗箱，在路上大出风头。

[![](../Images/cf94c1c01f1f123f7c2bddd85a511b8e.png)](https://www.youtube.com/watch?v=qIp9kItDUh8)

[The optics of the camera obscura project an image upside-down.](https://www.youtube.com/watch?v=qIp9kItDUh8)

基本的概念是一个伟大的——租用一辆箱式卡车，并用纸板盖住后开口。在纸板上剪一个小洞，你就创造了一个带轮子的暗箱。该视频很好地解释了正在发生的事情背后的光学原理，甚至还有关于如何通过修改光圈来改变曝光水平和焦距的实验。

在视频上观看暗箱的唯一缺点是，你无法欣赏现实生活中可见的分辨率和细节。请相信我们，它比当今市场上的任何高清电视都要好。

滚动的暗箱是一个伟大的实验，只需要一些纸板、胶带和一个晴朗的日子。在学校或创客空间里作为一种教育活动来执行会非常有趣。一旦你解决了这个问题，也许可以考虑数字版本。休息后的视频。
