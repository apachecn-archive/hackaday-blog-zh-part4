# 黑客日链接:2022 年 3 月 20 日

> 原文：<https://hackaday.com/2022/03/20/hackaday-links-march-20-2022/>

嗯，降级很快！不到一周前，由于世界电子制造中心新冠肺炎的复兴，中国深圳市被封锁。这显然在电子产品供应链上上下下引起了不小的恐慌，有望再次让寻求从 PCB 到元件到完整电子组件的制造商感到不安。但是仅仅几天后，[中国政府宣布深圳封锁已经结束。至少部分是这样的——深圳五个区的工厂和公共交通已经重新开放，iPhone 制造商富士康是深圳较大的厂商之一，获准部分重新开放。这对业余爱好者快速获得廉价 PCB 的能力意味着什么？这很难说，至少在这一点上。请在下面的评论中分享你的供应链中断经历。](https://www.bloomberg.com/news/articles/2022-03-18/shenzhen-lockdown-eases-as-xi-seeks-to-soften-covid-impact)

来自万里之外的好消息，美国国家航空航天局宣布，詹姆斯·韦伯太空望远镜完成了复杂的镜面对准程序的第一部分。该过程使用内置在 18 个六边形镜面中的复杂致动器，轻微移动每个镜面，将它们全部对准一个虚拟光学表面。结果不仅是[我们一直看到的令人惊叹的“自拍”图像](https://www.msn.com/en-us/news/technology/james-webb-space-telescope-s-latest-selfie-marks-a-milestone-for-the-space-observatory/ar-AAVa8r5)，也是[明星韦伯一直作为目标聚焦的美丽照片](https://www.livescience.com/james-webb-telescope-image-limits-of-physics)。下面的视频详细解释了这一过程，并分享了下一步是将镜子移入和移出，或“活塞”它们，以便 18 个独立的波前都对齐，以完美的相位将光发送到仪器。谈精度！

对于您的应用程序来说，普通的树莓派不够坚韧吗？你需要在一个可以承受几个 g 的振动而仍然保持工作的平台上运行 *DOOM* 吗？厌倦了当天气变得有点潮湿或太热时，你的基于 Pi 的气象站打破自己的？然后你会想看看[的 DuraCOR Pi](https://www.curtisswrightds.com/news/press-release/raspberry-pi-powered-ultra-small-form-factor-rugged-mission-computer.html) ，这是一款包含 Pi CM4 的加固型机箱，专为极端环境打造。该机器安装在一个防护等级为 IP67 的小型外壳中，在振动、温度、湿度和电磁干扰条件方面符合 MIL-STD 规范。这看起来不太像是针对业余爱好者市场的东西——它是由国防承包商柯蒂斯-赖特国防解决方案公司(Curtiss-Wright Defense Solutions)销售的，该公司的根源可以追溯到俄亥俄州的一对学会如何飞行的自行车机械师。所以这个 Pi 可能更像是你在建造一架无人机或者类似的东西时所需要的。尽管如此，知道这样的事情存在还是很酷的。

BrainLubeOnline 收集了一组有趣的 x 光照片。除了一只鼠标——T2，另一种是 T3——所有的东西不是电子的就是机械的，这造就了真正有趣的照片。看到齿轮上的齿或螺丝上的螺纹，并看穿物体，以一种全新的方式展示了机械世界——真的。

最后，你会买一辆不能打开引擎盖的车吗？我们大多数人可能不会，但话说回来，我们大多数人可能不会购买奔驰 EQS 580 电动轿车。YouTube 上 Sarah -n- Tuned 的 Sarah[不知何故得到了其中一个宝贝](https://www.youtube.com/watch?v=9FNhw2c8ijQ)，她恰如其分地将其描述为“德国飞船”，并试驾了它，包括“全速”加速测试。就在那次脖子折断的骑行之后，大约在下面视频中的 7:20 分，她要求汽车的内置助手打开引擎盖，汽车拒绝了这一要求，称“引擎盖只能由专业车间打开。”不管怎样，莎拉设法打开了它，它不是一个破房子——它是为汽车提供动力的两个马达之一的家，还有各种各样的其他好东西。
