# 这个闪电探测器非常灵敏

> 原文：<https://hackaday.com/2019/10/03/this-lightning-detector-is-remarkably-sensitive/>

雷击是非常高能量的事件，当它们在大气中爆炸时会释放大量的无线电频率能量。这使得它们很容易被探测到，能量释放的幅度意味着它可以在令人印象深刻的范围内完成。[[Jay]决定制造一台属于自己的设备，并对其探测性能印象深刻。](https://www.youtube.com/watch?v=VacGFyoY2vI&feature=youtu.be)

该装置是一种简单而有效的设计。天线用于捕捉 RF 信号，然后通过单个晶体管级放大这些信号。这是连接到一个基本的晶体管闪光灯电路，这是偏置的，只有当一个传入的信号超过边缘闪烁。在建立了电路之后，[Jay]注意到这个设备不仅仅接收来自闪电的信号，还接收来自许多其他较小放电的信号。该设备能够检测到穿着袜子在木地板上受到的电击，以及范德格拉夫发电机的放电，甚至是从椅子上站起来！

闪电探测器已经存在很长时间了；在之前，我们已经看到其他人美化了这些页面。休息后的视频。
