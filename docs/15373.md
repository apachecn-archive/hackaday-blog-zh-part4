# 传真机会停止唱歌吗？

> 原文:[https://hack aday . com/2022/11/16/the-fax-machine-ever-stop-singing/](https://hackaday.com/2022/11/16/will-the-fax-machine-ever-stop-singing/)

在整个 80 年代和 90 年代，你不可能在任何大小的办公室里挥动订书机而不撞到传真机。但是是什么让传真机成为如此多嘲笑的对象呢？是哔哔声吗？垃圾传真？或者，在云存储和拇指驱动器的世界里，它们看起来已经过时了？也许以上都是真的。虽然我可能是 Hackaday 的常驻老学校办公室工作人员等等，但你可能会惊讶地发现，我没有传真机。事实上，上一次我不得不发传真的时候，我记得为了免费发一份传真，我不得不把我的电子邮件地址给了某个网站。

在大洋彼岸，[英国政府已经决定根据所谓的普遍服务令(USO)立法取消对传真服务的要求，这基本上确保了全英国的居民都能以他们能够承受的价格获得电话服务。英国通信办公室，又名 Ofcom，最近宣布他们与政府达成一致。由于该行业正从公共交换电话网(PSTN)转向 IP 电话，传真机的工作方式将会不同。](https://www.theguardian.com/technology/2022/nov/01/fax-for-the-memories-ofcom-backs-death-warrant-for-80s-office-staple)

## 只有传真

[![](../Images/46f106d976e89a4888a05d521a69d6f0.png)](https://hackaday.com/wp-content/uploads/2022/11/elisha-gray-telautograph.webp)

Elisha Gray’s telautograph. Image via [RedOrbit](https://www.redorbit.com/education/reference_library/technology_1/inventions/2583648/telautograph/).

你可能已经知道了，但是传真比电话早。起初，传真通过电线传输，然后通过无线电波，最后通过电话系统。当然，现在它们倾向于通过互联网传输到传真服务器，然后被分发到个人电脑。

早期的传真设备是机械的和化学的，例如亚历山大·贝恩的电打印电报，他在 1843 年获得专利。1865 年，意大利物理学家乔瓦尼·卡塞利发明了 Pantelegraph，由此诞生了巴黎和法国里昂之间的第一个电传服务。

1880 年，谢尔福·彼德维尔制造了第一台无需人工绘图就能扫描任何 2D 原作的机器。八年后，以利沙·格雷发明了 telautograph，这使得签名可以远距离传送。下一项创新出现在 1924 年，当时美国电话电报公司通过电话从克里夫兰向纽约发送了 15 张照片。同年，RCA 发明了今天传真机的前身，无线照相电报机。

## 生活传真

Ofcom 给相关公民一个月的时间在修正案生效前陈述他们的情况。少数发表意见的支持者指出，传真机在法律、医疗和旅游领域仍然广泛使用。更重要的是，这意味着不再支持其他语音频带数据应用，如为需要者提供的远程护理警报。但是 Ofcom 保证，他们希望服务提供商在这些情况下与客户合作，提供替代方案。根据对法律、医疗、旅游和能源部门的调查，Ofcom 得出结论“传真的使用很少，并且在仍然使用的地方正在寻找替代品，”议会辩论的官方记录 Hansard 指出。

有趣的是，2017 年，[英国国家医疗服务系统被报道为世界上最大的传真机购买者](https://www.nationalhealthexecutive.com/Health-Care-News/digital-doldrums-nhs-remains-worlds-largest-purchaser-of-fax-machines)，2018 年有近 12000 台在使用。但卫生和社会保障部在 2019 年宣布购买新机器是非法的，该部表示，现有的机器必须在 2020 年 3 月底之前用电子邮件取代。

那么，这真的是传真机末日的开始吗？虽然现在我们有很多发送照片和文件的方式，但在电子签名被广泛接受之前，也许会，也许不会。就在几年前，我们看到了[一个可以通过传真](https://hackaday.com/2019/05/04/faxsploit-exploiting-a-fax-with-a-picture/)远程执行代码的漏洞，所以他们仍然受到坏人的关注。
