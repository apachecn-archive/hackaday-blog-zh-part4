# 最奇怪的黑客

> 原文:[https://hackaday.com/2022/02/05/the-weirdest-hack/](https://hackaday.com/2022/02/05/the-weirdest-hack/)

我在 FLOSS 播客上(为野兽的第集)!)我们一直在谈论黑客日。其中一个主持人，秘密 Hackaday 自己伪装的乔纳森·本内特，问我在 Hackaday 上见过的最奇怪的黑客是什么。奇怪吗？！？！

我像被车灯照着的小鹿一样被抓住了。我们的黑客都不奇怪！或者他们都是？我不知道，这当然取决于你的观点。做一个能发出周期性喵喵叫声的盒子藏在朋友的衣柜里是不是很奇怪？[设计新的有趣的轮子让杂技演员在](https://hackaday.com/2022/02/02/3d-printed-circular-prototype-performance-prop-captivates-circus-spectators/)里滚来滚去会不会很奇怪？想要一个[彩虹色 USB 拨码开关](https://hackaday.com/2022/02/01/rainbow-dip-switch-is-the-coolest-way-to-configure-your-project/)会不会很奇怪？这些都是上周的帖子，奇怪吗？

好吧，也许我们*是*有点奇怪。但这是我们喜欢的方式。保持它的怪异和美妙，哈卡代。你一天八小时已经有足够多正常的事情要做了！
