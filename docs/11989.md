# 构建您自己的 HP41C

> 原文:[https://hackaday.com/2021/11/24/build-your-own-hp41c/](https://hackaday.com/2021/11/24/build-your-own-hp41c/)

曾经有一段时间，工程师们带着计算尺。然后有一段时间我们都带着计算器。当然，计算器仍然存在，但你更可能使用手机。如果你真的需要严肃的数字运算，你会求助于一台完整的计算机。但有一段尴尬的时间，计算器非常重要，计算机非常昂贵，计算器试图成为我们需要的成熟的计算机。HP41C 可能是这一趋势的顶峰。如果你曾经有过一个，你知道这是当今科技的奇迹，它有字母数字功能和四个用于更多内存或 rom 的插入端口。它真的是一台小型的手提电脑。没有吗？别担心，你现在可以[建造你自己的](https://youtu.be/wpQaDlrgTQk)。事实上，如果您愿意，HP 模拟器也可以像 HP15C 或 16C 一样工作。

你可以在下面的视频中看到这个设备的运行。正如你所料，这个版本使用通孔 ATMEGA328，即使在 8 MHz 下，仿真速度也比原来的计算器快。这台机器的内存是原来计算器的两倍多，还有一个内置的实时时钟。显示屏也是背光的，这是我们在原版中想要的。

不过，我们真正喜欢的是你可以下载的用于计算器的 3D 打印盒。你也可以在视频中看到。我们有点难过的是，该套件是在 HP15-C 一样的横向不像 HP41 的纵向格式。此外，原理图就在那里，因此可能有人会设计出一个很好的 HP41 专用电路板。

当然，惠普计算器使用的是 RPN，但是如果你是在计算尺的环境下长大的，那也没关系。对于其他人来说，这通常是一个学习曲线。此外，尽管 HP41C 很棒，但它没有二进制或十六进制甚至八进制数学，除非你使用“合成编程”，尽管我们想象所使用的仿真器可以处理合成编程(HP41C 的一种汇编语言)，但用这台机器处理非十进制问题从来都不有趣。尽管如此，这是一个多么伟大的计算器，看到它以现代形式重生是很有趣的。

我们已经看到了另一个[假 HP41C](https://hackaday.com/2018/07/04/nonpareil-rpn-hp-41-calculator-build/) ，我们认为它使用了相同的仿真软件，但需要 SMD 焊接。我们想知道[金宇野或 1802 宇野](https://hackaday.com/2017/07/25/kim-1-to-cosmac-elf-conversion-sort-of/)是否可以在相同的硬件上运行？似乎会的。
