# Kid Rover 是牛逼的六个轮子

> 原文：<https://hackaday.com/2019/03/22/kid-rover-is-six-wheels-of-awesome/>

学习 TIG 焊接有很多方法。大多数涉及一系列的实践部分，制作对接接头和将各种管道段焊接在一起。[克里斯·泰默曼]决定走得更远一点。结果是一辆 kid rover，它肯定会成为附近所有孩子羡慕的对象。

底盘是全铝的，这使得 TIG 焊接成为这项工作的最佳选择。当然，[克里斯]不满足于简单地建造一个基本的卡丁车或童车。这种甜蜜的乘坐方式受到美国宇航局火星探测器摇臂转向架设计的启发，使其具有乘坐高度和灵活性，可以在严重的障碍上漫游。自然，有六轮驱动和四轮转向来完成动态包。还应该指出的是，黄色车轮是一个令人惊叹的设计选择，我们只是没有看到足够的。

这是一辆做工精美的汽车，是对[克里斯]的加工和设计技能的证明。我们迫不及待地想看到它在比利时泥泞的土地上试驾。如果你渴望开始建造自己的摇臂转向架，美国国家航空航天局已经为你准备了开源设计。休息后的视频。
