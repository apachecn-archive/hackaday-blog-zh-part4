# 仅次于网络卡车的最好的东西

> 原文:[https://hackaday . com/2020/05/21/下一个最好的东西到一辆网络卡车/](https://hackaday.com/2020/05/21/the-next-best-thing-to-a-cybertruck/)

虽然特斯拉 Cybertruck 的生产要到 2021 年才会开始(最早)，但你随时可以尝试建造自己的车。除非你有一个非常大的备件抽屉，否则它可能不会是全尺寸的，但如果你有一个像[Emiel]一样库存充足的商店，你至少可以建立一个模型。他花了一些时间用一片铝片做了一个电脑卡车模型。(视频，嵌在下面。你可能想打开字幕。)

这个项目是一个很好的例子，说明了一些表面上看起来简单的项目需要一些专门的工具才能做得恰到好处。首先，用激光将铝片切割成合适的形状，包括窗户等细节，并用雕刻师标记弯曲点，以帮助弯曲过程。[Emiel]缺少的一个工具是[制动器](https://en.wikipedia.org/wiki/Brake_(sheet_metal_bending))，但是他用一套金属弯曲钳得到了很好的结果。

完成这个模型也不是特别顺利。他计划将金属钎焊在一起，但所需的热量使车身板件不断翘曲。解决办法是用环氧树脂把多余的部分打磨掉，结果有望比铜焊更坚固，因为他在环氧树脂中加入了一块布来增加强度。窗户由聚碳酸酯制成(在耐久性测试中没有破裂)，我们希望当[Emiel]准备好安装马达时，他会使用[一个他定制的电动马达](https://hackaday.com/2020/03/20/this-v8-makes-a-shocking-amount-of-power/)。
