# 与 Arduino 兼容的无人驾驶地面车辆

> 原文:[https://hack aday . com/2018/10/02/an-unmanned-ground-vehicle-compatible-with-an-arduino/](https://hackaday.com/2018/10/02/an-unmanned-ground-vehicle-compatable-with-an-arduino/)

制造自己的机器人是每个人都应该做的事情，而且[Ahmed]已经制造了一些设计用于室内驾驶的机器人。不过，室内机器人很简单:你可以在平坦的表面上滚动，最坏的情况是你要担心楼梯。户外机器人是完全不同的东西，这使得这个项目如此壮观。这是 M1 漫游者，一种无人驾驶地面车辆，围绕 Arduino 平台建造。

M1 漫游者的设计目标不仅仅是一辆可以在室内驾驶的遥控汽车。这个机器人是为崎岖不平的地形设计的，是一个可以被*编程*的机器人，也可以被电脑、视频游戏控制器或定制操纵杆驱动。

为此，M1 漫游者围绕高质量的激光切割胶合板进行设计，由几个通过双 H 桥控制的 DC 电机提供动力，并装载了传感器，包括一个前置超声波传感器。所有的电子设备都藏在机箱里，软件也棒极了。事实上，通过将智能手机巧妙地安装在底盘顶部，这个小机器人可以成为一个自主漫游车，并配有网络摄像头。这是我们见过的最好的机器人漫游者项目之一，也是今年 Hackaday 奖的惊人之作。
