# 现在有一个乐高悬浮 Dyno

> 原文：<https://hackaday.com/2022/06/16/theres-a-lego-suspension-dyno-now/>

当涉及到性能悬架的开发和测试时，有一个测试仪器让你可靠地重现某些条件是很有帮助的。[这款乐高悬挂 dyno 就是这么做的](https://www.youtube.com/watch?v=UuQxctEQhLA)，它显然对那些在迷你图摩托车悬挂上做研发的人有很大的帮助。

该建筑依靠四个马达来克服转动矮胖传送带的阻力，传送带就像一条滚动的道路。由于传送带由技术梁制成，各种乐高积木可以添加到传送带上，作为测试悬架的凸起或扰动。

该视频演示了测力计的使用，展示了一个简单的乐高摩托车设计如何应对各种大小的颠簸。很容易更换前叉和弹簧，改变几何形状来调整悬架，当在相同的测试条件下运行时，可以很容易地看到这些变化。

虽然我们不认为有太多的人在这个特定领域工作，但这里教授的课程是有价值的。这种设置使人们能够快速直观地看到车辆参数的变化如何影响操控。很难想象有比这样的东西更好的车辆动力学教学工具，[让你直接看到真正发生了什么](https://hackaday.com/2020/03/11/retrotechtacular-automotive-suspension-is-all-about-waves/)！
