# 您可以自己打造一款价格实惠的参考话筒

> 原文：<https://hackaday.com/2022/07/02/an-affordable-reference-mic-you-can-build-yourself/>

参考话筒是音频工作的重要工具。它们以平坦的频率响应而闻名，通常用于描述房间或空间的音频响应。OpenRefMic 的目标是成为一个开源设计，在不支付高昂零售价格的情况下生产参考话筒。

该产品的核心是一个前置放大器，采用标准 48 V 幻像电源供电，负责偏置驻极体麦克风元件，并作为麦克风信号的缓冲器。它是专为配合 PUI AOM-5024L-HD-F-R 话筒振膜使用而设计的，因其良好的性能和低噪音特性而被选中。然而，其他电子麦克风应该也能工作。硬件被包裹在一个 3D 打印的盒子里，可以很容易地在最基本的打印机上制作。它配有一个压配合格栅，可将麦克风炭精盒固定到位。

这个项目的主要目标是降低噪音；项目创建者[loudifier]指出，大多数商用参考话筒首先关注平坦的频率响应，然后才是降低噪声。OpenRefMic 在这方面表现出色，它缺乏完美平坦的频率响应，这与校准均衡相得益彰。它还可以使用普通的专业级 XLR 电缆和幻像电源，而不需要昂贵的实验室专用电缆和接口。

最终结果归功于[loudifier]，他对参考话筒设计的原理有着深刻的理解。最近，我们还看到了其他一些出色的低成本参考话筒！
