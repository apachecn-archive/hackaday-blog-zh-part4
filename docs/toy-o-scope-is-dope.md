# 玩具 O-Scope 是兴奋剂

> 原文：<https://hackaday.com/2020/09/12/toy-o-scope-is-dope/>

除了一个锥形瓶(如果他们是“科学家”的话)之外，我们童年的玩偶和活动人偶没有多少附件是围绕着实验室设备的不，他们倾向于烤面包机，车辆和枪。当我们年轻的时候，我们的英雄做食物，开性感的汽车，和坏人战斗。现在我们长大了，我们的一些英雄挥舞着烙铁、键盘和示波器。【阿德里安·赫布兹】制造了一个[比例模型示波器](https://www.youtube.com/watch?v=m0cg3zCVKhM)，它比过去的烧杯和试管架更加耀眼。视频也显示如下。

在这个望远镜的左侧是一个可旋转的旋钮，它可以驱动几个内部杠杆来弯曲一个氖绿色的橡皮筋。杠杆和背景是黑色的，所以它看起来像一个屏幕，而乐队接近正弦波。[阿德里安]的范围将上升到任何面包烘烤，蓝宝驾驶，战术酷忍者猎人的游戏。在更大的尺度下，薄橡皮筋看起来更像锯齿而不是正弦，但也许这对你的维京电子人 DJ 理发师来说是完美的。他们的实验室里会有什么缩小版的你？

我们喜欢看到像这样具有现代吸引力的新玩具，但我们尊重老式玩具。黑客父母在为他们的孩子保持[新鲜和活泼](https://hackaday.com/2019/11/08/why-buy-toys-when-you-can-build-them-instead/)方面做得很好。
