# 贴纸带来热度

> 原文:[https://hackaday.com/2022/02/13/sticker-brings-the-heat/](https://hackaday.com/2022/02/13/sticker-brings-the-heat/)

[Carl]一直在寻找 PCB 回流和其他应用的加热板。在他最新的视频中，他展示了他是如何使用背面有粘性的薄柔性印刷电路板作为变热的标签的。你可以在 [GitHub](https://github.com/CarlBugeja/HeatStick) 上找到 gerber 文件和设计文件。

你可能认为这对于柔性 PCB 来说是一件非常简单的事情，但事实证明，虽然 PCB 可能是柔性的，但走线不是柔性的，因此你在加热器中看到的典型长走线不会允许标签弯曲，如果你想把它缠绕在咖啡杯上，这是一个问题。

答案是创造一种轨迹频繁改变方向的模式，这就是[卡尔]所做的。他还谈到在木板边缘使用加强筋。

视频显示了相当多的测试，表明基板在长时间高温后变色。在这些条件下，粘合剂也易于熔化。因此，建议在需要长时间低温或短时间高温的情况下使用它们。当然，取决于你的应用，只要不影响长期可靠性，颜色的变化可能并不重要。

你会用柔性加热贴纸做什么？自动加热马克杯是显而易见的。想到了在寒冷的日子里贴在裸露管道上的贴纸。一个薄的 3D 打印机床加热器或不规则形状的爬行动物栖息地也可能有用。

只要你以任何方式放置图案，我们想在这样的贴纸上制作[热敏数字](https://hackaday.com/2021/06/17/using-heaters-to-display-time/)。似乎每次我们检查[卡尔]时，他都在[加热东西](https://hackaday.com/2021/10/06/using-a-pcb-to-reflow-pcbs-take-2/)。
