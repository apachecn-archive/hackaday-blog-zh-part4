# 焊接机器人承担了一项又热又脏又危险的工作

> 原文：<https://hackaday.com/2018/11/06/welding-robot-takes-on-a-hot-dirty-dangerous-job/>

他们过去常说机器人会接管对人类来说太脏或太危险的工作。这正是[乔尔·沙利文] [创造这个焊接机器人](https://www.youtube.com/watch?v=YOt0LxMMp2M)时的想法。[Joel]为 OSB 行业设计了机器人。不，那不是新的操作系统，是定向 Strand Board 的简称。OSB 是一种工程木材，由木片制成。它类似于胶合板，但不需要大而薄的木材。为了制造 OSB 板，将 5 英寸厚的木屑与胶水混合，并在 7500 磅/平方英寸和 400 华氏度下压缩至 5/16 英寸

用于制造 OSB 的压机是大规模并行操作。一次可以压 20 块以上的板。你的新闻也是一个主要的损害领域。隐藏在木材中的螺母或螺栓会钻入压机，造成凹痕，这将在通过该部分的每张纸上显现出来。修复压机的唯一方法是将其关闭，部分拆除，并用焊机填补空隙。[Joel 的]机器人通过在仍然热的、仍然组装的压力机上进行焊接，消除了大部分停工时间。

这个机器人看起来像是受到了战斗机器人的启发，这很合适，因为它工作的环境更像是战场。这是一台又低又宽的机器。前面是两个铰接臂，一个带有焊接机，一个带有模具研磨机。焊机填充压板中的任何空隙，模具磨床将新焊缝磨平。一个英特尔 NUC 控制的东西，有大量的电机驱动，电源和继电器在板上。

[乔尔的]机器人是拴着的，有供应氩气、电力和压缩空气的脐带。空气穿过底盘的通道，使热压机上的机器人保持凉爽。一切都是为高温设计的，甚至是车轮。[Joel]尝试了几种橡胶，但最终选定了实心铝车轮。机器人移动速度不快，所以牵引力很大。一些微型步进电机驱动轮子。当需要焊接时，气动支腿会将机器人锁定在狭窄的压力机内。

带有数字十字准线的摄像机允许操作员通过网络界面控制一切。一旦所有的参数都设置好了，操作员点击 go，机器人开始焊接时火花四溅。
