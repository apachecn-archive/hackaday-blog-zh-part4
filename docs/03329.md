# 构建一个 Mag Lev 光学表

> 原文：<https://hackaday.com/2019/06/16/building-a-mag-lev-optical-table/>

当你谈论光学时，事情往往发生在纳米尺度上。这意味着即使是最轻微的振动也会破坏精细的工作。[The Thought Emporium]正在进行一项生产巧克力全息图的大规模项目，需要一个稳定的表面来安装一些光学组件。因此，他决定[建造一个磁悬浮激光工作台](https://www.youtube.com/watch?v=ML4FHNGzAn0)。

构建从一系列八个加工的 delrin 衬套开始，每个衬套安装一个强钕环磁体。四个放在底座上，一根细钢棒向上突出。然后放置另外四个衬套，使磁铁的磁极彼此相对，使它们悬浮起来。然后一个丙烯酸板被降低到顶部，由悬浮磁铁支撑。

这是一种非常简单的制造磁悬浮桌子的方法，在一个基本的干涉测量测试中，似乎在隔离桌子振动方面做得很好。我们也想知道通过使用某种涡流阻尼器是否有进一步改进的空间。它应该为进一步的实验提供了一个极好的平台，我们期待在不久的将来看到一些巧克力全息图。
