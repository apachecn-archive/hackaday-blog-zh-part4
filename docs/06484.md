# 数控等离子切割过滤器去除炉渣

> 原文:[https://hack aday . com/2020/05/16/CNC-plasma-cutter-filter-gets-the-slag-out/](https://hackaday.com/2020/05/16/cnc-plasma-cutter-filter-gets-the-slag-out/)

无论你在店里使用什么样的工具和材料，很有可能某个过程会释放出你不想呼吸的东西。台锯？最好处理一下木屑。3D 打印机？我们已经令人厌烦地讨论了烟尘控制。焊接？最好不要吸入那些助焊剂烟雾。但是也许没有什么地方比金属车间的排烟更重要了，在那里蒸发的金属碎片会造成呼吸系统的严重破坏。

降低这种风险是[Shane Wighton]设计这种免清洗等离子切割过滤器的基本原理。他的数控等离子切割机不是一个收集切割残渣的水台，而是一个吸走残渣的吸风口。然而，倒吸式风扇喷射出的生动火花掩盖了它的无效性。谢恩的想法是基于旋风原理，这是[木工车间吸尘器](https://hackaday.com/2009/07/19/cyclonic-dust-seperation/)和[昂贵的真空吸尘器](https://hackaday.com/2019/05/17/a-cyclonic-vacuum-cleaner-on-a-hackers-budget/)的共同点。塑料管部分纵向分成两半，外面包着铝带，以防被热火花点燃，垂直安装在空气通道中。这些管道被排列成一系列嵌套的“S”形，当充满火花的空气离开下降气流时，为其提供了一条曲折的路径。

下面的视频显示，大部分夹带的固体减速并落到过滤器底部；一些仍然可以通过，但是用粘合片测试表明废气中的金属颗粒减少了很多。我们喜欢这种设计，尤其是没有任何东西会阻塞或严重限制气流。
