# 干净利落的快捷方式

> 原文：<https://hackaday.com/2019/01/23/crisp-clean-shortcuts/>

人们总是告诉我们，他们使用电脑最喜欢的部分是一遍又一遍地重复相同的按键序列，日复一日。还有像[本尼]这样的人，他们宁愿让一个微控制器在触摸一个时髦的 USB 外设时做重复的工作。那些喜欢额外打字的人似乎也喜欢一直在他们的电脑上添加新的专有软件，但他们又不走运了，因为这个表盘充当了键盘和鼠标，所以当他们在朋友的电脑上工作时，他们甚至不能安装那个臃肿的软件。抱歉，伙计们，你们有些人运气不好。

作为计算机输入的旋转编码器并不是新的，商业版本已经存在多年了，但它们足够小众，对最终用户来说非常昂贵。短 BOM 和巨大的多功能性将使一些人重新考虑增加一个自己的工作站。在下面的视频中，屏幕图像被旋转以获得正确的角度，然后绘制一条线，就像有人在纸上做的那样。另一个演示让我们想起 XKCD，它通过撤销和重做功能循环，给你一个可逆的工作时间表。

如果你想让你的副手宏使能器有更多的变化和按钮，我们可以满足你，或者你只是想让它们在某些时候有 T2。
