# 步行臂投石机不同，但有效

> 原文:[https://hack aday . com/2019/09/15/walking-arm-trebuchet-is-different-but-effective/](https://hackaday.com/2019/09/15/walking-arm-trebuchet-is-different-but-effective/)

对于我们许多人来说，我们第一次接触著名的投石机是在*帝国时代 II、*或者是一部关于历史上的攻城机械的纪录片。然而，许多人继续追求他们自己的构建，探索新的和旧的设计。步行臂投石机[就是一个很好的例子，它用一种非传统的设计达到了很大的效果。](https://www.instructables.com/id/Worlds-Simplest-and-Newest-Trebuchet-Walking-Arm-T/)

该设计避开了刚性框架，而是简单地由一个臂和一个三角形腿组件组成。手臂被倒置，通过让投石机向前折叠并停留在三角形腿上来发射。三角形的腿上安装有长钉，可以钻入地面，然后手臂绕轴旋转，发射抛射物。据报道，该设计非常有效，类似于浮臂投石机，设计非常简单。表现如此之好，[它在 2018 年佛蒙特州南瓜投掷节](http://vtpumpkinchuckin.blogspot.com/2018/10/results-of-10th-annual-vermont-pumpkin.html)上大获全胜。

有各种各样的方法来建造投石机，[我们在](https://hackaday.com/2018/09/10/trebucheting-tennis-balls-at-124-mph/)之前已经介绍过一些。[你甚至可以测量你的有效载荷来量化性能](https://hackaday.com/2019/05/20/make-physics-fun-with-a-trebuchet/)。休息后的视频。
