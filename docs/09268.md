# DIY 面包片纸违背了纹理

> 原文：<https://hackaday.com/2021/02/17/diy-bread-slice-paper-goes-against-the-grain/>

现在是北半球的严冬，在疫情和极地涡旋之间，我们已经厌倦了盯着同样的四面墙不停地吃东西。这是尝试新的低热量食物的完美配方，比如用回收站里的东西烤一条面包形状的便条纸。

[SusanLand]喜欢用她手头的任何废弃物品造纸，包括旧牛仔裤。当她试着只用卫生纸卷筒来造纸时，效果并不理想，但这给了她一个想法来制作一些另类的文具。她用碎办公用纸和玉米淀粉来增强纸浆，用一小撮黄色和橙色的纸来调和全麦色调。一旦果肉准备好了，她就把它倒进用塑料牛奶壶做成的面包形状的模子里。

这份关于自制纸张的简明介绍涵盖了从制浆技术到干燥方法的所有内容。切片干燥后，[SusanLand]用划线工具、彩色铅笔和一把种子修饰它们，完成外观。休息之后，请观看视频中的过程。
