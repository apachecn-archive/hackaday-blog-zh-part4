# 垃圾箱机密

> 原文:[https://hackaday.com/2021/04/17/junkbox-confidential/](https://hackaday.com/2021/04/17/junkbox-confidential/)

托马斯·爱迪生有句名言:“要发明，你需要丰富的想象力和一堆垃圾。”阿门，兄弟。我个人的垃圾堆(咳咳，收集二手机电古董)当然是零部件的来源，但也是惊喜和奇迹的来源。有时垃圾本身激发想象力，但有时垃圾就是垃圾。

有一些我为某个特定计划买的二手装备，可能是十年前买的？，而且完全忘记了。虽然重新发现它们很有趣——我买了六个二手超级吸水泵组件，夏天就要到了——但令人悲伤的事实是，被遗忘的部分往往因为某种原因而被遗忘。无论我当时有什么古怪的想法都已经消失了，剩下的只有零件了。

但是在这些悲惨的生物中，有一些绝对的宝石。不断需要使用的零件。甚至会激发托马斯·爱迪生的想象力。难以忘怀的垃圾。大多数情况下，是他们的身体在召唤我。我有一堆旧的 5 英寸硬盘，被掏空了，变成了一个旋转编码器。多年来，我一直把它当作桌子上的 USB 滚轮，但最近它又出现在了其他愚蠢的项目中——给我儿子的音乐盒，它根据你旋转它的速度连续播放音符，最近是一个一米线性运动项目的滚轮，它还没有真正找到它的完整表达，但可能会成为相机滑块。无论如何，当我需要一个漂亮的物理旋转编码器旋钮时，硬盘就在那里等着被使用。

老式的旋转式电话也属于这一类。它以桶形插孔而不是 RJ45 结束，因此很容易合并到其他项目中。以前，它是 USB 拨号密码设备的接口，但它被占用更少桌面空间的软件所取代。我肯定会想出别的事情让它做。

[![](../Images/c88c390fc48ac6d86bd1af92ebd5ac90.png)](https://hackaday.com/wp-content/uploads/2021/04/Spacemouse_Plus_thumbnail.png) 我年复一年打捞出来的令人难忘的垃圾中无可争议的王者，是两只[麦哲伦太空老鼠](http://spacemice.org/index.php?title=Spacemouse_Plus)。它们是老式的串行端口，对任何微控制器都是 9600 波特，并且易于接口。他们推出了六个控制轴和几个按钮，坦率地说，他们是一个技术奇迹。这些已经被用于遥控车，甚至有一次被用于乘坐车辆，但只是短暂的，因为它太危险了。我从未尝试过同时使用四把以上的斧子，但这种隐含的挑战是它们魅力的一部分。

最奇怪的是， [Linux Spacenav 驱动程序项目](http://spacenav.sourceforge.net/)的维护者 John Tsiombikas 在 12 月将系列太空鼠标纳入了代码库。所以现在我把它们都用于它们的预期目的——在 CAD/CAM 软件中操纵 3D 设计——一个在办公室，一个在车间。不是黑客！

垃圾箱里有太空老鼠真是太棒了；他们真的很鼓舞人心。一方面，提升到日常使用是最终目标，我想，但它仍然在我心里留下了一个洞。我想，我还会多留意几天。垃圾箱饿了。
