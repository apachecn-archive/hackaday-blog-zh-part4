# 新零件日:SD NAND 是像 SD 卡一样工作的表面贴装芯片

> 原文:[https://hackaday . com/2020/12/29/new-part-day-SD-NAND-are-surface-mount-chips-the-work-like-an-SD-card/](https://hackaday.com/2020/12/29/new-part-day-sd-nand-are-surface-mount-chips-that-work-like-an-sd-card/)

SD 卡一直是微控制器爱好者的最爱。它们便宜、容易获得、易于接口，仍然是需要存储大量数据的小型项目的主要选择。现在，它们以芯片的形式出现了！这些被称为模拟 SD 卡接口本身的 SD NAND 部件。

[【lady ada】最近在发现](https://blog.adafruit.com/2020/12/26/sd-card-on-a-chip/)[来自【Greg Davill】](https://twitter.com/GregDavill/status/1332136806284886016)(我们熟悉他是因为[他出色的 LED 立方体](https://hackaday.com/2019/09/05/handmade-led-cube-is-a-work-of-art/))的一条推文后，给了他们一次试驾。这些设备由 XTX 科技公司制造，可从 LCSC 获得，尺寸范围为 1、2、4 和 8gb。[Ivan Grokhotkov]在回复[Greg]的原始推文中也提到了另一家制造商的类似设备[，因此可能会有更多的消息来源。](http://www.longsto.com/product/30.html)

这些芯片采用标准 LGA8 表面贴装封装，可以轻松焊接到电路板上，与在槽型连接器中使用普通 SD 或 microSD 卡相比，具有机械和制造优势。此外，与其他 SMD 闪存部件不同，它们处理所有文件系统细节，并为您进行损耗均衡！随着 SD 卡尺寸的膨胀，如今在普通卡的货架上也很难找到这些。

[Adafruit]计划很快为这些部件开发一个突破，包括一个易于使用的电平转换器。我们可以想象这些芯片会出现在各种数据记录器项目中，因为它们可以与其他部件一起订购，并永久焊接到设计中。如果你有一个特别好的想法来证明这些芯片在哪里有用，请在评论中提出来。休息后的视频。
