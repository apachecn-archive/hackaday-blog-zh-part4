# 摆盘咖啡烘焙机得到的咖啡豆恰到好处

> 原文:[https://hack aday . com/2020/07/23/wobble-disk-coffee-baker-gets-the-beans-just-right/](https://hackaday.com/2020/07/23/wobble-disk-coffee-roaster-gets-the-beans-just-right/)

烘焙咖啡是一门艺术还是一门科学，取决于你和谁交谈。然而，两大阵营都认为关注细节是关键。众所周知，许多顽固的笨蛋，会走得更远，以至于创造他们自己的烘烤硬件来完成工作。[Larry Cotton]就是这样一个建筑商，他创造了一个优雅的烘焙器，让他的酿造恰到好处。

该构建基于摆动盘设计。它由一个与旋转轴成 45 度角的圆板组成。随着轴的旋转，圆盘轻轻扫过并搅动烤肉，让整批烤肉均匀加热，而不会烧焦豆子。这是一个由两部分组成的设计，热风枪部分位于底座，为烘烤过程产生热空气。豆篮位于顶部，由磁铁固定，磁铁也作为摇摆盘电机电源的导管。

这是一个整洁的建筑，允许准确的烘焙，一旦完成就可以很容易地倾倒豆子。[如果你自己也是一个严肃的傻瓜](https://hackaday.com/2018/01/23/build-an-excellent-coffee-roaster-with-a-satisfyingly-low-price-tag/)袖子里藏着一些黑客，[一定要让我们知道](http://hackaday.com/submit-a-tip)！休息后的视频。
