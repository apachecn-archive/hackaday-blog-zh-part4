# 3D 打印你自己的电动螺丝刀

> 原文:[https://hack aday . com/2019/03/05/3d-print-your-own-electric-driver/](https://hackaday.com/2019/03/05/3d-print-your-own-electric-screwdriver/)

对于偶尔的零星工作，一把普通的螺丝刀就可以了。然而，在类似小型生产操作的情况下，使用起来会很快变得不舒服。在这些情况下，电动螺丝刀非常有用。然而，没有必要急着去商店——你可以自己做一个,[Electronoobs]就是这么做的。

构建很简单，由一系列连接在一起的廉价模块组成。一个 18650 电池通过升压转换器向 H 桥电机控制器供电，该控制器允许 DC 齿轮电机向前和向后驱动。还有一个 USB 电池充电模块，允许螺丝刀从标准的微型 USB 充电器充电。这一切都整齐地包装在一个 3D 打印的外壳中，带有几个用于前进和后退操作的触觉按钮。

这是现代供应链质量的一个证明，人们可以在家里用预包装的模块组装一个像这样的可用工具。所有零件，包括轴承和钻夹头，都来自易贝。
