# 盯着奖品！

> 原文:[https://hackaday.com/2022/04/02/eyes-on-the-prize/](https://hackaday.com/2022/04/02/eyes-on-the-prize/)

今年的黑客日奖有了一个良好的开端。这太棒了，因为今年的挑战特别重要:通过更好的能源收集、更好的资源利用来减少人类在地球上的足迹，并让我们已经拥有的东西运行得更久一点。这不仅是下个世纪的主要挑战，也是像我们这样的黑客的游乐场。

[![](../Images/fed9adefe78f1f2775f437a7c857c883.png)](https://hackaday.com/wp-content/uploads/2022/04/planet-friendly_power.png) 第一期，[行星友好力量](https://hackaday.io/contest/184555-supplyframe-designlab-2022-hackaday-prize/log/204304-challenge-1-planet-friendly-power)正在如火如荼的进行中，第一天我们就看到了一些参赛作品！他们作弊了吗？他们有内部消息吗？没有。成千上万的黑客一直在研究节能的方法来推动他们的项目。如果你的 Raspberry Pi 数据记录器可以以太阳能为燃料，这不仅对世界更好，而且是一个你不必记得更换电池的项目。

我们面临着回收利用的挑战，一个是防止垃圾填埋的逆向工程，一个是环境监测和通信基础设施。这些都是很棒的黑客话题，展示了像我们这样的人如何做我们的小部分来保持世界运行而不使它陷入困境。

因此，你们所有人都在制作网状网络，优化太阳能项目，破解开放的封闭物联网网络以防止它们过时，或者建造塑料分类机器人，这是你们获得一些钱和对你们出色工作的一些认可的机会。

再次感谢我们的 Supplyframe overlords 对 Hackaday 奖宗旨的一贯支持和信任，也感谢 DigiKey 多年来一直是该奖的赞助商！没有他们，我们不可能成功。
