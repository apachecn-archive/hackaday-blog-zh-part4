# 裸机给了这个 Pi 一些经典的合成器

> 原文：<https://hackaday.com/2022/04/19/bare-metal-gives-this-pi-some-classic-synths/>

我们习惯于在这里展示的许多项目中看到 Raspberry Pi，但公平地说，它们通常以某种操作系统为特色。使用 Pi 还有另一种方式，更类似于使用 Arduino 等微控制器:直接对其编程，即所谓的裸机编程。MiniDexed 就是一个例子，它复制了 20 世纪 80 年代经典的雅马哈专业合成器，在一个单元中模拟了相当于该公司八个著名的 DX7 合成器。它几乎可以接受任何 Pi，再加上一个音频板、一个旋转编码器和一个 LCD 显示器，就成了一个现成的单元。下面是一段视频。

公平地说，我们不是 Raspberry Pi 裸机编程方面的专家，但值得转移到 20 世纪 80 年代的合成器世界来探索 DX7。这种乐器是整个 20 世纪 80 年代流行音乐的主要组成部分，也是雅马哈作为一种负担得起的 FM 合成器的主要商业成功。这是 20 世纪 70 年代斯坦福大学的一项专利技术，随后获得了该公司的许可，与当时的其他合成器不同，它完全以数字方式产生声音。很难高估 DX7 的影响力，因为它的声音随处可见，即使在今天，如果你有声卡，你也不是不可能拥有一台雅马哈 FM synth。

对 DX7 好奇？去年底，芯片逆向工程大师[Ken Shirriff] [曝光了它的秘密](https://hackaday.com/2021/11/25/a-teardown-of-80s-era-music/)。
