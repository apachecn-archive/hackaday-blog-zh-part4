# 电动汽车充电器可能是黑客的一个严重目标

> 原文:[https://hack aday . com/2022/11/28/ev-chargers-be-a-serious-target-for-hackers/](https://hackaday.com/2022/11/28/ev-chargers-could-be-a-serious-target-for-hackers/)

电脑！如今他们无处不在。从恒温器到冰箱，甚至百叶窗，所有东西现在都在互联网上，这使得它们都适合黑客攻击。

电动汽车充电器正在成为日常生活的一部分。它们也是连接设备，因此如果设计和维护不当，会带来安全风险。正如物联网上的许多其他设备一样，[真相绝非如此。](https://www.theregister.com/2022/11/15/ev_charging_infrastructure_sandia/)

## 妥协了！

有时，保护某个系统或设备就像断开它与网络的连接一样简单。例如，当谈到电灯开关和门锁时，我们多年来都没有在线使用它们。然而，就电动汽车充电器而言，这并不实用。至少，运行支付系统需要连接。此外，能够远程监控电动汽车充电器的状态和健康状况对保持它们的可用性和可操作性有很大帮助。

鉴于电动汽车充电器必须连接，保护它们非常重要。然而，桑迪亚国家实验室的研究表明，到目前为止，电动汽车充电器公司在保护他们的系统方面并没有做到最好。研究人员调查了各种攻击媒介和漏洞，发现现有系统在许多方面根本达不到要求。

车辆-充电器接口被作为主要目标进行研究。EV 充电器通常通过充电电缆传递的信号与车辆通信。这种沟通包括协商功率水平和充电时间等细节。然而，如果车辆的响应没有得到正确处理或消毒，它也可能是恶意软件感染电动汽车充电器的途径。研究人员发现，不仅可以从这些连接中嗅探数据，而且使用软件定义无线电(SDR)的低功率攻击可以在 47 米之外停止车辆的充电。这些接口通常也是完全不加密的，这使得它们容易受到中间人和欺骗攻击。

电动汽车充电器的用户界面也很脆弱。最简单的攻击反映了在加油站使用的攻击，在那里读卡器被安装了套卡器来获取卡数据。其他直接的黑客攻击包括对依赖支付和账户管理系统的 RFID 克隆攻击。充电网络使用的智能手机应用程序也可能成为黑客的目标。

就像打印机等其他网络硬件一样，电动汽车充电器通常带有基于网络的配置界面。而且，就像打印机一样，许多打印机最终不可避免地会出现在更广阔的互联网上。研究人员发现所有类型的电动汽车充电器都可以公开其配置页面。更糟糕的是，许多人的凭证非常薄弱，通常配置有基本密码，或者只需要一个序列号就可以访问。

即使 web 服务是安全的，管理界面仍然会带来严重的风险。通常，电动汽车充电器带有某种形式的诊断和维护端口。这可以通过串行、WiFi、以太网、USB 或蓝牙实现。在许多情况下，充电器被发现有不必要的服务，如通过这些接口访问的 Telnet 和 FTP，从而带来了更广泛的安全风险。在某些情况下，充电器很容易允许固件转储或未签名的更新，或日志数据的泄漏。更糟糕的是，在许多情况下，物理端口的安全性很差，为恶意行为者提供了方便。

## 潜在后果

电动汽车充电器中的漏洞可能会在设备受损时带来各种后果。其中许多后果性质轻微，范围有限。例如，一个装有套卡器的充电器可能会导致犯罪分子盗取数百名用户的信用卡。打开网络配置页面可能会让黑客禁用充电器或取消用户正在进行的会话。

然而，在某些情况下，后果可能要严重得多。固件受损的充电器可能会被配置为禁用某些安全功能，从而给用户带来风险。充电器可能会被重新编程，以便在安全插入电缆之前给电缆通电，从而导致触电死亡的风险。欺骗性的车辆-充电器通信可能导致充电器向车辆输送过多的电力。在这种情况下，适当安全的车辆可能会关闭与流氓充电器的连接。然而，理想情况下，车辆和充电器都足够安全，不会首先造成风险。

还有更大规模的担忧。现代电动汽车充电器需要来自电网的大量电力。虽然 7 千瓦和 11 千瓦的家用充电器曾经是标准配置，但今天的 DC 快速充电器的功率水平超过 350 千瓦。在这样的功率水平下，研究人员担心，如果攻击得当，可能会对电网造成重大破坏。如果攻击者可以控制足够多的车辆充电器，简单地让它们立即停止就可能威胁到电网的频率和电压稳定性。在目前的电动汽车使用水平下，这不是一个主要的风险。根本没有足够多的车辆一次给电网带来足够的负荷来造成威胁。

然而，随着电动汽车使用率的上升，威胁也在增加。随着车辆到电网充电器也成为一种东西，也有可能出现振荡攻击方法。大量车辆突然要求电网供电，然后几秒钟后试图反馈电力，这可能使当局难以维持对电网的控制。控制电网就是平衡电网的发电量和电力需求。电动汽车充电器僵尸网络的供需突然变化可能会导致大范围停电，甚至暂时导致某些基础设施离线。这样做可能需要几乎同时控制数十万个充电器，因此目前，电网工程师可以在晚上继续睡个好觉。

随着电动汽车充电器在未来几年的推广，现在是确保安全的时候了。谢天谢地，桑迪亚国家实验室为电动汽车充电器行业提供了一份关于最佳实践的文件。大部分都很简单，比如物理保护管理端口，验证固件更新，以及将 web 配置接口放在严格的防火墙后面。然而，在一个地方有一个最佳实践的列表，是公司确保他们的产品没有不必要的不安全的一个简单方法。
