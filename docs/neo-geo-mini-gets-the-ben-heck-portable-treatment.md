# 近地天体地理迷你获得本赫克便携式待遇

> 原文：<https://hackaday.com/2018/11/21/neo-geo-mini-gets-the-ben-heck-portable-treatment/>

感觉像是近地小行星来了又走了。当最初系统的粉丝第一次触摸“非点击式”操纵杆时，围绕拥有一台小型可用街机的所有宣传都消失了。虽然看到包含 USB-C 电源是令人鼓舞的，但没有内部电池允许玩家在没有束缚的情况下使用系统。对产品目前的状态不满意[Ben Heck]拍摄了一段视频，详细介绍了他使用 Neo Geo Mini 内部部件的最新便携式创作。

便携式设计的重点是融入近地轨道 MVS 拱廊系统的各个方面，而近地轨道迷你缺乏这些方面。D-pad 包括微型开关，或者像[Ben Heck]称之为纳米开关，以获得更明显的触觉。他能够重新利用原来 PCB 上的扬声器和耳机插孔以及 4:3 纵横比的 LCD。定制面板包裹着熟悉的红色和白色徽章，而 3D 打印的面部按钮则有经典的红色、黄色、绿色和蓝色。不要担心，他们终于在这里正确的按钮配置。

很高兴看到[本·赫克]自今年早些时候他的 YouTube 秀结束后仍在制作便携式魔术。这些年来，他为改装社区做出了很多贡献，在这个 Neo Geo Mini 便携式视频中也有很多有用的提示。请注意，构建被分成两个独立的视频(第二部分在下面)。我们期待将来有更多像[本·赫克]这样的项目。
