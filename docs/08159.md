# 双冲击飞行模拟器轭架

> 原文：<https://hackaday.com/2020/11/04/dualshock-flight-simulator-yoke/>

飞机控制界面可分为操纵杆和轭架，操纵杆在飞行模拟器中更受欢迎。[Akaki Kuumeri]一直在为游戏控制台控制器设计一些巧妙的 3D 打印适配器，他的最新作品是 PlayStation DualShock 控制器的 [yoke 适配器。](https://www.youtube.com/watch?v=ylIe6uNZf6c)

像他以前的[操纵杆/油门组合](https://hackaday.com/2020/09/25/xbox-controller-gets-snap-on-joystick-from-clever-3d-printed-design/)一样，这个轭利用一系列球窝连接将轭的推/拉和旋转运动转换成控制器拇指杆上的适当输入。除了提供弹簧张力的橡皮筋之外，所有组件都是 3D 打印的。在不同部件之间的滑动接触表面上，[Akaki]特别设计了沿着纹理(层线)滑动的部件，以允许在不借助轴承的情况下平稳运动。

如果你想要一个绝对的极简轭，用胶带将一些电位计粘到书桌抽屉上。或者你可以去规模的另一端，建立一个[完整的驾驶舱](https://hackaday.com/2020/01/22/p-51-cockpit-recreated-with-help-of-local-makerspace/)。随着微软飞行模拟器 2020 的到来，我们将看到许多控制器的构建。
