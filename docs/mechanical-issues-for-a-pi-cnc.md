# Pi 数控系统的机械问题

> 原文：<https://hackaday.com/2018/08/08/mechanical-issues-for-a-pi-cnc/>

树莓派平台已经在创客社区流行起来，用于各种数控项目。单板计算机很适合充当小型 CNC 设置或 3D 打印机的服务器，尽管公平地说，对于繁重的工作，它们可能无法胜任直接驱动严重钻机中的步进机的任务。[【Danny】开始尝试使用 Pi Zero 构建自己的数控绘图仪，并在此过程中学到了一些东西。](https://hackaday.io/project/160248-pi-cnc-plotter-fail)

绘图仪使用 3D 打印零件，并结合了减速的 DC 刷电机。添加电位计是为了让 Pi 跟踪笔的位置。这使得能够通过反馈来校正位置。

虽然绘图仪可以移动并接受命令，但它确实存在一些问题。齿轮系中存在明显的游隙，[Danny]怀疑这是导致输出结果不佳的原因。如果你有任何关于如何改进或克服这一点的想法，请在评论中提出来！
