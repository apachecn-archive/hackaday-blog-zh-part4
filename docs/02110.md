# 黑客日播客 005:不死灯泡、家庭化学和 3D 打印的力量

> 原文：<https://hackaday.com/2019/02/08/hackaday-podcast-005-undead-lightbulbs-home-chemistry-and-the-strength-of-3d-printing/>

与 Hackaday 的编辑迈克·什奇斯和埃利奥特·威廉姆斯一起了解过去一周有趣的黑客活动。本周，我们将讨论 LED 灯泡寿命额定值下降背后的故事，了解增强 3D 打印部件的有限元分析，欣赏锻造之美，并惊叹开源激光雷达的发展。我们深入研究了上周播客引发的蓝药丸项目的伟大读者建议，讨论了 V2 火箭的一些历史，并涵盖了 Chromecast 控制硬件、发光家庭化学、K40 激光切割机附件等。
