# 令人敬畏的视频超失误的自主船

> 原文:[https://hack aday . com/2019/12/21/autonomous-boat-for-awesome-video-hyperlaps/](https://hackaday.com/2019/12/21/autonomous-boat-for-awesome-video-hyperlapses/)

随着智能手机、行动相机和手持万向架功能的不断增强，在 YouTube 和 Instagram 等平台上，最佳照片的争夺日益激烈。超驰序列是军械库中最受欢迎的武器之一，[丹尼尔·赖利]又名[rctestflight]意识到他的自主船可能是一个令人敬畏的[超驰平台](https://www.youtube.com/watch?v=3OVgsIb7ED8&feature=emb_title)。

这是他的自主船的第三个版本，第一个版本的[遭受海藻袭击，第二个版本的](https://hackaday.com/2019/08/18/ardurover-boat-uses-to-float/)[几乎和鱼睡在一起。新版本是一艘平底船，几乎完全由粉红色的隔热泡沫制成，使其稳定且不会下沉。它使用了与第二版相同的电子设备和空气船推进装置，并增加了一个安装在智能手机万向节上的 GoPro，以拍摄超级失误。由于安装在高处的马达，它有一种将船首全速推入水中的趋势，但通过在船首下方添加泡沫凸起进行了纠正，代价是一些效率。](https://hackaday.com/2019/11/06/autonomous-air-boat-vs-lake-washington/)

最困难的部分是调整万向架的设置，在没有平移跳跃的情况下产生超重叠。在平静的水面上，船足够稳定，足以欺骗 IMU 相信它没有转动，所以万向节控制器使用电机编码器来保持位置，这不允许它吸收船不断进行的所有小的航向校正。关闭编码器集成后，情况有所改善，但它仍会偶尔碰到死区的边缘，在死区内万向节不会随船转动。最后[丹尼尔]满足于慢慢地将万向架向左移动，同时绘制出一条经过仔细计算的左转路线，以使船本身不被拍摄到。虽然并不完美，但这些镜头仍然很好地捕捉到了西雅图联合湖的夜景。让它达到这个水平需要很多小时的午夜测试，因为[丹尼尔]尽最大努力避免其他船只通行，我们相信这是值得的。

我们期待他的下一个视频，包括他的太阳能飞机的更新。
