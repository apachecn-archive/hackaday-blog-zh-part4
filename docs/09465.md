# Arduino Micro 按下动物穿越的按钮

> 原文：<https://hackaday.com/2021/03/08/arduino-micro-pushes-animal-crossings-buttons/>

视频游戏中的重复性任务经常会触动我们的神经。[无脸科技]厌倦了在《动物穿越》中捕捉流星时将“A”混为一谈，所以[他开始自动解决他的问题](https://facelesstech.wordpress.com/2020/08/29/animal-crossing-automation/)。在短暂考虑安装一个伺服系统来为他工作后，他回忆起之前的一项努力，即[在*塞尔达:野性的呼吸*中使用 Arduino Teensy 来自动化保龄球迷你游戏，并决定使用微控制器来为他捕捉星星。](https://hackaday.com/2017/10/25/teensy-script-plays-nintendo-switch-strikes-out/)

[Facelesstech]编写了一个 Arduino Pro Micro 来伪造控制器按钮的按下。它先按几下按钮，向开关表明自己的身份，然后不断按下按钮，自动捕捉每一颗流星。连接起来很容易——一个移动适配器允许交换机的 USB-C 端口直接连接到 Arduino 的 Micro-USB 端口，甚至供电！

[Facelesstech]还设计了一个紧凑的 3D 打印外壳，将 Arduino Pro Micro 与 ISP 头一起打包，以便于更新。外壳甚至让 Arduino 的电源 LED 发光，让你知道它在工作！

如果你也需要自动操作视频游戏按钮，[Facelesstech]已经友好地上传了源代码和 3D 设计供你尝试。如果你喜欢更低技术含量的东西，也许你可以试试机械按钮。
