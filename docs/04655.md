# 蜜蜂看护者证明，即使是蜜蜂也不会在监视状态下安全

> 原文:[https://hack aday . com/2019/10/31/bee-minder-provides-not-even-bees-are-safe-from-surveillance-States/](https://hackaday.com/2019/10/31/bee-minder-proves-not-even-bees-are-safe-from-surveillance-states/)

我们都知道蜜蜂对我们的生态系统有多重要，而[Kris Winer]的[蜜蜂监测器](https://hackaday.io/project/164810-bee-minder)提供了一种很好的方法来监测这些令人惊叹但又脆弱的生物的栖息地，有望在灾难袭击重要的蜂巢之前提醒我们。

该板是基于 LoRa 传感器瓦称为蝉，但重新设计，使其更小，更便宜。LoRa 是一种流行的低功耗广域网，运行在亚 Ghz 频段，专为类似这样的[应用而设计。](https://hackaday.com/2019/09/23/lora-based-plant-monitoring/)该板有一套漂亮的传感器。它可以探测长波紫外线、中波紫外线和可见光谱。它还可以观察温度、压力和湿度。对蜜蜂来说重要的是，加速度计可以检测蜂箱的各种振动以及破坏等灾难事件。

所有数据都被记录到一个 [Cayenne](https://developers.mydevices.com/cayenne/features/) 仪表盘中，未来的农民可以在任何地方查看和分析。[Kris]提到，电路板相对容易根据应用使用不同的传感器套件重新旋转。像这样的技术可以走向更可持续的未来。
