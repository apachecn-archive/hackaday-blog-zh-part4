# 一个带热敏打印机的地下城主

> 原文：<https://hackaday.com/2022/12/24/a-dungeon-master-with-a-thermal-printer/>

热敏打印机在当今世界无处不在，每当我们必须从某个地方拿到收据时，我们都能找到它。它们便宜、快速、易于使用。不仅如此，正如[丹尼尔]发现的那样，除了药店三英尺长的收据之外，它们还非常容易重新编程和用于其他用途。他将它们改编成了 D & D 游戏中地下城主的关键工具。

虽然他采用了最常见的热敏打印机标准，爱普生标准代码，但这个项目的真正乐趣在于用户界面。他使得通过 HTML 快速构建模板和其他面向 D&D 的表单成为可能，因此地下城主可以打印出角色表单、游戏中的物品、地图或他们当时可能需要的任何其他东西。这一切都是高度可配置的，可以满足任何需求，并且界面可以在 Mac、Windows 和 Linux 上工作。

所有的项目代码都可以在丹尼尔的 GitHub 页面上找到。大多数热敏打印机也使用这个标准，所以只要有一卷热敏纸，就可以很容易地找到便宜的打印机并投入使用。如果热敏纸的感觉勾起了一些童年的怀旧之情，那可能是因为你年轻时有过 Game Boy 打印机，并且[正在寻找重新体验热敏纸打印机魔力的方法](https://hackaday.com/2021/10/22/an-open-source-game-boy-printer-that-doesnt-print/)。
