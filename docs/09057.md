# Hackaday 播客 102: Raspberry Pi 微控制器、麦克风 Killswitch 和 45 度 3D 打印机

> 原文:[https://hack aday . com/2021/01/22/hack aday-podcast-102-raspberry-pi-微控制器-麦克风-killswitch-and-a-45 度-3d-printer/](https://hackaday.com/2021/01/22/hackaday-podcast-102-raspberry-pi-microcontroller-microphone-killswitch-and-a-45-degree-3d-printer/)

Hackaday 的编辑埃利奥特·威廉姆斯和迈克·什奇斯筛选了一周的优秀黑客。大新闻当然是 Raspberry Pi 微控制器，Elliot 在宣布之前已经在工作台上玩了几个星期——它内置了一些有趣的可编程模块(PIO )!飞利浦设计了一种 LED 灯泡，可以驱动 LED 以提高效率和延长寿命。亚马逊为 Echo Flex 中的麦克风添加了一个漂亮的小硬件禁用电路——一个相当极端的拆卸展示了他们是如何做到的。此外，我们还讨论了开源远程 RC 协议、墙壁大小的笔式绘图仪艺术，以及使喷嘴成角度以避免需要支持的 3D 打印机。
