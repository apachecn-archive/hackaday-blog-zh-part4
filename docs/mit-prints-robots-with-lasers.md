# 麻省理工学院用激光打印机器人

> 原文：<https://hackaday.com/2021/02/12/mit-prints-robots-with-lasers/>

麻省理工学院的计算机科学和人工智能实验室(CSAIL)希望将激光切割机转换成更为 T1 的东西。通过将头部连接到商用激光切割机并添加软件，他们将切割机、导电打印机和拾放系统的功能结合在一起。这个想法是为了能够建造完整的设备，如机器人和无人机。

这个概念被称为 [LaserFactory](https://hcie.csail.mit.edu/research/laserfactory/laserfactory.html) ，听起来像是星际迷航式的复制器，但它并没有创造出电路元件和马达之类的东西。它只是简单地把它们捡起来，放置好，然后用银导电墨水把它们连接起来。通过观看下面的视频，你可以很好地了解它是如何工作的。

虽然今天所有这些技术确实都存在，但是将它们结合在一起还是有一些协同作用的。例如，视频显示用激光在丙烯酸树脂上蚀刻凹槽，用银墨水填充凹槽，然后用激光固化银，形成一种印刷电路板。然后，抓放器可以放下机器可以激光焊接到位的元件。

当然，这种结构的耐用性还有待观察。此外，很明显，根据可用组件的不同，您的选择余地也很有限。就是不能用四轴飞行器零件造坦克。

在之前，我们已经见过[激光焊接设备。自制的](https://hackaday.com/2014/08/27/thp-semifinalst-laser-solder-paste/)[取放](https://hackaday.com/2015/02/06/a-diy-pick-and-place-you-can-build-right-now/)机器也随处可见。你可以用一个激光模块、一个糊状物挤压机和一些其他零碎的东西来制作你自己的版本，这似乎不是太难的事情。
