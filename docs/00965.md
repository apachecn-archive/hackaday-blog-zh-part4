# 弹出通知恐龙

> 原文：<https://hackaday.com/2018/10/18/popup-notification-dinosaur/>

在我们的虚拟空间中有很多事情在发生，任何有智能手机的人都可以证明这一事实。你能想到的每件事都有弹出通知，有时一个真正重要的通知可能会迷失在细枝末节的海洋中。为了确保你不会错过那个重要的通知，你可以把这个任务交给你的私人恐龙。

3D 打印的恐龙有一个齿条和小齿轮齿轮组，允许它在命令时向上延伸。它还有一组用于眼睛的发光二极管，当它弹出时就会打开。这两个伺服系统和发光二极管由恐龙底部的一个小 Arduino 控制。这个 Arduino 可以被编程为在你喜欢的任何时候激活恐龙，比如来自特定人的电子邮件，对 Reddit 上评论的回复，或来电等等。一定要看看休息下面的视频。

有这只恐龙在你的桌子上，你不太可能会错过它的激活。如果你想要功能相同但动作更少、灯光更多的东西，还有一个由 led 制成的[通知 3D 立方体](https://hackaday.com/2015/05/10/led-notification-cube-is-a-good-first-project/)也一定会吸引你的眼球。
