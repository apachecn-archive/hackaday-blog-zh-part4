# 老式电流表成为工厂水分测量仪

> 原文:[https://hack aday . com/2020/08/15/vintage-电流表-变-工厂-水分计/](https://hackaday.com/2020/08/15/vintage-ammeter-becomes-plant-moisture-gauge/)

在当地的跳蚤市场或车库拍卖中遇到老式测量设备并不少见。这些工具通常具有不可抗拒的美感，并能使用几十年，但当现代万用表出现时，它们往往会靠边站。[Build Comics]手头就有这样一个作品，[并决定用一些现代硬件来替代它。](https://buildcomics.com/meters/arduino-hero-toolbox-the-plantmeter/)

构建从哈特曼&布劳恩 60 安培电流表开始。装满了一个漂亮的木箱，它是现代改装的完美候选。该装置使用动铁型指示器。为了将该设备变成土壤湿度监测器，[Build Comics]从移除原始的重绕线圈开始。取而代之的是安装了一个定制的线圈，使用改良的缝纫机缠绕在 3D 打印的线轴上。这使得电表很容易由 Arduino 驱动，只需在 GPIO 引脚上安装一个晶体管。为了检测湿度，使用了 Iduino ME110 湿度探头。配有布面金属丝，保持复古外观。最初的仪表盘也被拍照、修改和重印，以读取湿度而不是电流。

如果你对这些仪表修复技术感兴趣，但没有园艺技能，不用担心。[Build Comics]用类似的技巧[组装了一个华丽的气象站](https://hackaday.com/2020/08/08/vintage-gauges-turned-classy-weather-display/)，放在你的桌子上会很好看。
