# Don’t Tase Me, Keeb!

> 原文：<https://hackaday.com/2021/03/02/dont-tase-me-keeb/>

好吧，所以这不是真的使用泰瑟枪-这只是点击诱饵，我们道歉。一个真正的泰瑟枪将是一个可怕的方式来训练自己成为一个更好的打字员，因为根据你选择在哪里传递电击，你可能会在 5 万伏的几毫安电压下很快损坏你的打字神经。

[nobody6502]没有用泰瑟枪，而是从亚马逊上买了一包恶作剧口香糖，这种口香糖提供了一种更可行的电击，这种电击足以让用户更小心地打字。[nobody6502]建立一个简单的没有痛苦、没有训练的[网站](https://nobody5050.github.io/taser/)，一次呈现随机的英语单词，对照[近 50 万条目的开源列表](https://github.com/dwyl/english-words)检查错别字。拼错一个单词，口香糖电路就会发出继电器驱动的电击。

这种疼痛训练器的主要大脑是一个粒子氩板，它有 I/O 引脚，可以通过网络控制。当网站检测到一个打字错误时，它会向氩气发送一个信号，氩气会开启一个继电器，激活电击机制。最让人印象深刻的是【nobody6502】没有一台成熟的电脑，把所有东西都编程在一台 iPad 上。休息之后，请观看构建视频。
