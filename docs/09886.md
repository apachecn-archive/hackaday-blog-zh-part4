# 不，谢妮·谢妮钟

> 原文:[https://hackaday.com/2021/04/20/no-nixie-nixie-clock/](https://hackaday.com/2021/04/20/no-nixie-nixie-clock/)

在[Techmoan]的 YouTube 频道[上，他对一个经过数月等待终于到来的新玩意](https://www.youtube.com/watch?v=ygjonCPPcGM)感到兴奋 EleksTube IPS 假数码管时钟。这是一个使用六个 135×240 像素 IPS 显示面板的数码管时钟的重新想象。它们像小广告牌一样安装在玻璃灯泡内，模仿复古的外观。根据[Techmoan]对这些显示器的测量，它们似乎与 TTGO ESP32 模块中使用的 16:9 IPS 显示器相同。效果相当令人印象深刻，每个数字都是一个完整的显示，这一事实带来了相当大的灵活性。例如，如果你不喜欢谢妮的外观，你可以从一套风格中选择或制作自己的自定义数字。

虽然这是 EleksMaker 推出的成品，但这个概念显然也适用于 DIY 项目。拥有所有这些显示器的一个有益的副作用是，改变时钟的设置不再是一种神秘的体验——显示器用于以明文显示和选择选项。这对许多人来说是一个卖点，因为这款设备不需要手机应用程序来控制。我们希望更多的产品能跟上这一趋势。感谢[IanS]的提示。

![](../Images/94d564a05edfe25868b37b47977a6346.png)

Menu settings and options made easy

这不是 EleksMaker 第一次尝试更换数码管。这个[早期基于胶合板的时钟](https://wiki.eleksmaker.com/doku.php?id=elekstube)使用边缘照明的丙烯酸数字面板来模仿数码管效果。早在 2019 年，我们报道了 [FIXIE(仿谢妮)项目](https://hackaday.com/2019/11/26/turning-oleds-and-acrylic-into-faux-nixie-tubes/)，该项目以类似的方式使用有机发光二极管屏幕，但不像这种 IPS 面板方法那样紧凑。
