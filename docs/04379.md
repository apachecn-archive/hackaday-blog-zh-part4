# 单色 CRT 和液晶快门联手打造彩色视频

> 原文:[https://hackaday . com/2019/10/01/monochrome-CRT-and-liquid-crystal-shutter-team-up-for-color-video/](https://hackaday.com/2019/10/01/monochrome-crt-and-liquid-crystal-shutter-team-up-for-color-video/)

如果你的任务是设计一个彩色视频显示器，你该怎么做就很清楚了。但是如果 20 年前你被要求这么做呢？从工程的角度来看，这是不是已经板上钉钉了？显然不是，正如这个混合 LCD-CRT 视频监视器所展示的。

老实说，我们从来没有听说过这种特殊的设计，被称为“LCCS”，或液晶彩色快门，直到[技术连接 JVC 监视器的部分拆卸及其操作的解释。这个想法很简单，可以追溯到美国彩色电视的早期，当时广播公司正忙着试图以一种最大化利润的方式将彩色带入单色世界。一个方案是在黑白阴极射线管前旋转一个色轮，并使二者同步，这基本上就是 LCCS 系统中正在发生的事情。液晶面板在红色、蓝色和绿色色调之间及时循环，CRT 的图像在它后面，产生全色图像。“但是等等！”你哭了。" 2000 年的时候肯定有小型彩色显像管！"当然有，但是有点糟糕。只要看看下面视频中彩色 CRT 和 LCCS 的比较，你就会明白为什么这个系统在专业视频市场上开辟了一个利基市场，特别是在数字电影技术出现之前的视频辅助显示器。泰克公司也将类似的系统用于彩色示波器。

像往常一样，[科技联系]已经设法挖掘出一点有趣的技术化石记录，并以一种迷人的方式呈现出来。从[黑胶视频](https://hackaday.com/2019/09/14/rca-created-video-records-too-late/)到[20 世纪 80 年代版权保护](https://hackaday.com/2018/05/27/rolling-old-school-with-copy-protection-from-the-1980s/)再到[烤面包机的内部部件](https://hackaday.com/2019/03/27/the-surprising-tech-of-a-cheap-toaster/)，我们享受着被遗忘的技术背后的样子。
