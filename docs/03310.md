# 机器人白板清洁器让白板随时可用

> 原文:[https://hack aday . com/2019/06/15/a-robotic-white board-cleaner-keep-the-board-ready-to-go/](https://hackaday.com/2019/06/15/a-robotic-whiteboard-cleaner-keeps-the-board-ready-to-go/)

擦拭白板可能是一项乏味的工作。没有人想在长时间的会议后留下来清理，敏感信息经常被公开传播。不过，不要害怕，这个机器人是来帮助 T1 的。

这个小装置被称为 Wipy，是一个机器人清洁器，可以快速移动以保持白板清洁并为工作做好准备。借助 Arduino Uno 的大脑，它使用红外线跟踪传感器来瞄准要擦拭的区域，而不是浪费时间擦拭已经干净的区域。它还装有一个用于测距的飞行时间传感器，使它能够避开障碍物或在板上写字的忙碌的人。

如果说 Wipy 缺少什么，那大概就是判断力。尽管它有一张可爱的表情符号般的脸，但它并不真的能够机智，或者知道什么时候不需要它。建议在您完全完成之前关闭 Wipy，以免它在您完成之前闯入并开始清除重要的计算。

从根本上说，这是一个有趣的构建，也是学习如何使用各种传感器的好方法。如果你做过类似的事情，一定要让我们知道。另外，[也考虑一下写作方面的自动化。](https://hackaday.com/2018/04/26/writing-on-a-whiteboard-performed-by-a-robot/)休息后半开玩笑的电视广告。
