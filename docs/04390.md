# 交互式 LED 圆顶发光与他们最好的

> 原文:[https://hack aday . com/2019/10/02/interactive-led-dome-lumings-with-the-best-them/](https://hackaday.com/2019/10/02/interactive-led-dome-glows-with-the-best-of-them/)

随着价格和组件的可用性，现在比以往任何时候都更容易将一大堆 led 扔在一个建筑上，让它们闪烁。难的是把它做好。[Amy Goodchild]是一名艺术家，擅长制作漂亮的 LED 项目。[*在穹顶*安装时也不例外。](https://www.instructables.com/id/Interactive-LED-Dome-With-Fadecandy-Processing-and/)

该建筑基于一个大型的网格圆顶，安装有发光二极管面板，可以对里面的居住者做出反应。使用微软 Kinect 作为传感器，使 dome 能够绘制出 3D 空间中正在发生的事情，并使用这些数据来指导其动画。使用 WS2812B LED 条，结合 Fadecandy 控制器进行处理。这是一个强大的组合，使设计有吸引力的 LED 效果更容易，而不必强迫用户去努力编写自己的库或优化他们的微控制器代码。

对于那些对穹顶本身更感兴趣的人来说，你会很高兴地知道[艾米]也没有忽略那里的细节。该构建实际上是作为一个商业可用的工具包开始的，尽管仍然需要大量的手工切割、拧紧和绘画。她通过一系列视频出色地记录了穹顶的建造，并向读者介绍了她所做的一些设计决定(如果有机会，她会重拍)。

人们在最好的时候也喜欢网格球顶；增加一个交互式 LED 装置会让事情更上一层楼。[我们也看到它们被用作温室](https://hackaday.com/2019/07/01/building-a-geodesic-dome-greenhouse/)，它们也使[成为一个伟大的黑客空间项目。](https://hackaday.com/2016/03/07/geodesic-dome-build-at-rev-space-den-haag/)休息后的视频。
