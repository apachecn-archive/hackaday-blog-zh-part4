# 噩梦般的燃料远程呈现机器人可能成为你最后的朋友

> 原文:[https://hack aday . com/2020/05/09/nightman-fuel-tele presence-bot-may-been-your last-friend/](https://hackaday.com/2020/05/09/nightmare-fuel-telepresence-bot-may-become-your-last-friend/)

在疫情事件尘埃落定之后，历史学家将从许多不同的角度回顾这一时期。我们最感兴趣的当然是在焦虑、压力和无聊的培养皿中蓬勃发展的创造力，这些已经成为在家点菜的多余配菜。

[亨特·欧文]和他的兄弟真的很想念他们的朋友，所以他们举办了一场非常独家的黑客马拉松，并建造了一个可怕的远程呈现机器人，它看起来像是《T2》中的威尔森和《T4》中的那个骑三轮车的东西的混搭。哦，更糟糕的是，它是由夜光聚乳酸制成的。

现在，当他们与朋友视频聊天时，TELEBOT 会让他们感觉那个人就在房间里。伺服操纵的老式娃娃眼睛后面的 Arduino Uno 使用朋友的语音输入来控制基于音量水平的发条牙齿。正如你所想象的，他们的朋友对 TELEBOT 有一些不可思议的问题，所以他们打印了一套小帽子，实际上让一切变得更好。如果你认为你能处理好，休息后看看构建/演示视频。

对你来说还不够恐怖吗？试着从头开始打造你自己的眼睛。
