# 厨房里的焊锡锅

> 原文:[https://hack aday . com/2022/04/05/solder-pot-from-the-kitchen/](https://hackaday.com/2022/04/05/solder-pot-from-the-kitchen/)

我们并不害怕危险的项目，但是，话又说回来，[一个装满铅焊料的大锅](https://www.instructables.com/Low-Cost-Solder-Pot-for-the-Hobbyist/)可能有点多，即使对我们来说。不言而喻，你应该非常小心，你不会想使用任何炊具再次为任何其他目的。你可以在下面的视频中看到它的构造。

一方面，做一个焊料锅并不难。你只需要一个不会融化的容器和一个热源。但是看起来熔化的金属应该放在更难翻倒的东西里。这里真正的故事是使用焊料锅的技术，因为制作非常简单:只需要一个便宜的热板和一个铁锅。

你为什么想要一个焊料锅？它们是有用的。正如[Coalpeck]所展示的，你可以用它们很容易地浸焊一个通孔 PCB。如果你想给很多电线上锡，它们也很棒。他们还可以很好地从电路板或连接器上拆卸零件。检查旧的，但很好的视频商业单位删除主视频后的 PCB 连接器。

我们认为让报纸变成褐色的温度测量技术很有趣。当然，一个足够大到有用的商用焊料锅并不便宜。不过，你可以花不到 50 美元买到更小的罐子(50-80 毫米)。这些通常会有一个托盘来收集溢出物，并且很难被意外打翻。不过，并不是说你不想小心。如果你尝试这样做，我们建议你使用一个没有把手的锅，并把它放在一个外锅里来接溢出的水。但是如果你在工作台上洒了几磅熔化的焊料，不要说我们没有警告你。

这些年来，我们已经报道了几个自制焊料罐，但是神秘的是，所有的原始网站都不见了。我们希望他们没事。我们确实研究了大量的[脱焊技术](https://hackaday.com/2015/12/28/desoldering-doesnt-necessarily-suck/)，其中包括焊料罐。或者扔掉热铅锅，试试[【比尔·赫德】的一种方法](https://hackaday.com/2014/08/21/smt-and-thru-hole-desoldering/)。
