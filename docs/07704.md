# 曲速飞行的无限远镜

> 原文:[https://hack aday . com/2020/09/11/infinity-mirror-at-warp-speed/](https://hackaday.com/2020/09/11/infinity-mirror-at-warp-speed/)

发明常常把一个以上的旧想法结合成一个新想法。即使融合的东西是相似的，结果也可能比其各部分的总和更有价值。与那些表盘下方有数字时钟的模拟手表不同，当[mojaptix]将无限远镜子的[再反射特性与游乐园镜子的图像扭曲特性](https://www.thingiverse.com/thing:4587038)结合起来时，我们得到的不仅仅是一对镜子。由此产生的图像看起来很有趣。扭曲两个平行镜子的一个表面不仅仅会改变一点点结果，因为这两个平面会互相影响对方的视角，最终的结果是一场指数级的扭曲秀。

我们的主持人在一面普通的玻璃镜子上安装了一个 3D 打印环，上面有一条面向轮毂的发光二极管。除此之外，他还设计了四块配对的板，用来固定不同形状的半反射膜。第一个是双曲抛物面，但可能更容易认为它的形状像品客薯片(薯片)。一旦光线被应用，它看起来就像一个精神错乱的上帝做的领结或者一个光和反射的无限兔子洞的开始。为了进一步疯狂，他一次用四个形状打我们，所以我们希望你能花点时间欣赏下面的视频。

这个人对光学并不陌生，我们已经报道过几个通过演示来教授概念的其他酷发明。他的[精密卡尺](https://hackaday.com/2018/03/07/0-05-mm-precision-thats-a-moire/)展示了莫尔效应，他的[数字日晷](https://hackaday.com/2015/10/28/we-didnt-know-the-sun-could-do-digital/)利用了平行光束。
