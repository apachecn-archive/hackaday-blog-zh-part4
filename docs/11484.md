# 用于工作台的迷你照相机起重机

> 原文：<https://hackaday.com/2021/10/05/mini-camera-crane-for-your-workbench/>

如果你曾经试图在你的工作台上用照片或视频记录一个项目，你就会知道不断移动三脚架以获得正确镜头的挑战。【机械论者】对这种挫败感很熟悉，所以他造了一个小型[桌面相机起重机](https://www.youtube.com/watch?v=A9Z6XS3IQbg)。

受到[【伊万·米兰达】的大型摄像机起重机](https://www.youtube.com/watch?v=A9Z6XS3IQbg)的极大启发，该建筑将其缩小，并主要使用 3D 打印部件。起重机的臂可以围绕底座沿两个轴旋转，使用平行杆机构在垂直运动范围内保持相机方向不变。相机支架本身允许额外的 3 个自由度来捕捉任何角度，并可以安装 DSLR 或智能手机。为了抵消相机的重量，可调节的配重被添加到臂的后部。每个旋转轴都可以用指旋螺钉锁定。

我们当然可以看到像这样的起重机在我们的工作台上不仅仅是用于摄像工作。您可以创建附件来放置灯、显示器、万用表或一些辅助设备。关于制作一个[引人入胜的项目视频](https://hackaday.com/2020/10/09/how-to-shoot-a-great-project-video/)的一些技巧，请查看【Lewin Day】关于这个主题的精彩视频。
