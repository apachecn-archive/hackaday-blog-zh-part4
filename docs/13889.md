# 3D 打印凸轮泵转移水井

> 原文：<https://hackaday.com/2022/06/10/3d-printed-lobe-pump-shifts-water-well/>

凸轮泵最广为人知的用途可能是用于罗茨型增压器，但它们也能抽水。[让我们打印]用一个 3D 打印的设计[巧妙地展示了这一点，它可以发挥它们的最佳性能。](https://www.youtube.com/watch?v=MqrZm67GphY)

该设计使用两个 8 字形反转转子，或叶片。当转子转动时，它们将流体截留在转子和外壳之间，迫使流体流向出口。这是一种正排量设计，这意味着它在每次旋转中捕获固定体积的流体，将其从入口移动到出口。

该设计需要两个旋转叶片的适当定时，以确保它们保持封闭的容积，并且不会相互影响。这是通过泵背面的一对正时齿轮实现的。外壳、叶片和齿轮都是 3D 打印的，这使得任何人都可以在家里用自己的打印机复制它。

ABS 用于转子，因为它能更好地处理摩擦，不容易熔化。然而，树脂印刷的凸角也因其较高的公差而被采用，两种设计在实践中均可接受。

该泵仍需进一步改进；希望能减少泵后部的泄漏。[让我们打印]还打算给泵本身增加一个马达，而不是使用电钻来运行设备。很高兴看到这些 3D 打印的泵制造[继续认真进行](https://hackaday.com/2020/09/29/eight-motors-can-sure-pump-a-lot-of-water/)。休息后的视频。
