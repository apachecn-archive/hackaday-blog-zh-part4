# 机械复用倒装点

> 原文：<https://hackaday.com/2021/06/24/mechanically-multiplexed-flip-dot/>

翻转点显示器是永恒的经典，但驾驶大型显示器可能会很快变成一个重大挑战。电磁体需要大量的电流来运行，并且驱动电路会变得非常昂贵。[詹姆斯·布鲁顿]想要建造自己的，但遵循了一条有点不同的路线，建造了一个[机械复用翻转点(球？)显示](https://www.youtube.com/watch?v=tzm8KB9QgJg)。

[James]的 5×3 概念证明上的每个点都是双稳态机械装置，可以显示或隐藏乒乓球大小的半球。不是使用电磁铁，而是通过安装在显示器后面移动支架上的一排微型伺服系统来翻转这些点。该机制源于[James]之前的一个项目，一个[机械多路复用器](https://hackaday.com/2021/05/04/prioritising-mechanical-multiplexer/)。每个点机制在机制的后面有一个钩子，用于伺服推动或拉动来翻转点。这种设计的一个主要缺点是，伺服喇叭在移动通过钩子之前必须与点的状态匹配，否则它可能会碰撞和损坏某些东西，这也降低了托架的移动速度。

这个构建只是为了对这个概念有一个感觉，而且[詹姆斯]已经有了一些改变和改进的想法。挂钩的设计当然可以改变，皮带传动确实可以加快速度。我们认为这个机械展示是一个非常有趣的设计挑战，我们很想听听我们的读者会如何应对？请在下面的评论中告诉我们。

最近，我们首次报道了一款 [3D 打印翻转点显示器](https://hackaday.com/2021/05/29/3d-printed-flip-dots/)。它仍然很小,[Larry Builds]正在解决一些问题，但我们希望看到它最终能与[早餐的大型装置](https://hackaday.com/2016/11/29/flip-dot-displays-appear-with-modernized-drivers/)的迷人效果相媲美。
