# Glowtie 非常适合那些化妆舞会

> 原文:[https://hack aday . com/2019/02/25/glow tie-is-perfect for the-fancy-dress-raves/](https://hackaday.com/2019/02/25/glowtie-is-perfect-for-those-fancy-dress-raves/)

你厌倦了你传统的领结吗？你希望它有 RGB LEDs、WiFi 和可以从智能手机访问的网络界面吗？如果你像我们一样在 hack aday…也许不是。但这并没有阻止斯蒂芬·霍斯(Stephen Hawes)创造 Glowtie，这是一款公认的非常漂亮的开源电子领带，你可以自己制作，甚至作为一个组装单元购买。我们真的生活在未来。

[![](../Images/82d74b003c71a431729b6a9fcae5347e.png)](https://hackaday.com/wp-content/uploads/2019/02/glowtie_detail.jpg)

Evolution of the Glowtie

虽然我们不是这些地方的时尚专家(请查看“关于”页面以获取证据)，但我们绝对可以欣赏[Stephen]在设计上投入的时间和精力。特别是考虑到他决定将硬件和软件作为开源软件发布，同时仍然将设备放在 Kickstarter 上。我们看到太多的 Kickstarters 承诺在获得资金后开放源代码，所以我们总是很高兴看到一个项目愿意从一开始就把一切都放在那里。

对于硬件，[Stephen]已经采用了越来越受欢迎的 ESP8266 模块和 PCB 边缘周围的 WS2812B LEDs 阵列。底部还有一个微型电源开关，Glowtie 背面有一个 USB 端口，用于为两个 1S 300mAh lipo 电池充电。3D 打印的后面板为冲浪板提供了一些支撑，并带有一个集成支架，允许它夹在衬衫的顶部按钮上。对于那些不太喜欢裸露的 PCB 外观或暴露的 led 使人失明的人来说，有一个布面板覆盖在 Glowtie 的前面，不仅可以漫射光，还可以使它看起来更像一条真正的领带。

要控制 Glowtie，用户只需将智能手机连接到设备的 WiFi 接入点，并使用基于网络的界面。用户可以改变 led 的颜色和亮度，以及从不同的预装闪烁和渐变模式中进行选择。最终的结果，尤其是布扩散，真的看起来很华丽。即使这不是你日常穿的那种东西，我们也毫不怀疑你每次夹上它都会得到很多关注。

应该说【斯蒂芬】对可穿戴技术并不陌生。我们之前已经报道过他的[略显恐怖的腕装喷火器](https://hackaday.com/2014/09/25/a-wrist-mounted-flamethrower-sure-why-not/)，所以如果他[设法建造了它而没有引爆自己](https://hackaday.com/2018/10/17/flamethrower-gets-update-retains-some-sketchiness/)，我们想象相比之下建造一个点燃领带应该是小菜一碟。
