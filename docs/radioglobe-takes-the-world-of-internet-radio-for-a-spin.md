# RadioGlobe 带着网络电台的世界兜了一圈

> 原文：<https://hackaday.com/2020/07/27/radioglobe-takes-the-world-of-internet-radio-for-a-spin/>

不可否认，网络电台的覆盖范围和多样性非常酷。问题是，没有一个可用的接口真正给予这个庞大的东西应有的公正。我们渴望有一个更实际、更令人满意的界面来收听全球各地的电台，而[Jude]正好满足了这一需求。

[![](img/0c83e2c21b439d409f875153280f0239.png) ](https://hackaday.com/wp-content/uploads/2020/07/radioglobe-selector-detail.jpg) [ RadioGlobe 让用户通过旋转一个真实的地球仪](https://www.instructables.com/id/RadioGlobe-Spin-to-Search-Over-Web-Radio-2000-Stat/)来收听来自世界各地的 2000 多个电台。它通过使用两个绝对旋转编码器工作，每个编码器有多达 1024 个位置可用。一个编码器插在南极，当用户旋转地球仪时，它读取经度线。

另一个编码器在地球仪的左侧，读取十字线上聚焦的纬度。两个编码器都连接到 Raspberry Pi 4，尽管如果你想使用令人难以置信的详细指令复制这个开源项目，他说 Raspberry Pi 3 B+也可以工作。

在底座上有一个 LCD 显示坐标，城市和站点 ID。该区域的其他电台可通过底座上的滚轮进行调谐。还有一个 RGB LED，在电台调谐时闪烁红色，完成后变成绿色。我们完全喜欢这个建筑干净简约的外观——尤其是令人惊讶的透明底部面板，让你看到所有的内脏。

休息之后有三个视频——一个简短的演示，向您介绍它的工作原理，一个较长的演示，以及对绝对旋转编码器的详细解释。这些只是冰山一角，因为[Jude]保持了构建的每日 vlog。

也许你只是渴望一个网络收音机，拨在复古的吸引力。这台古董网络收音机有很多功能，但从外面看你是看不出来的。
