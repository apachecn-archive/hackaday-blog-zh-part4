# 与弗兰·布兰奇的聊天

> 原文：<https://hackaday.com/2021/08/09/vintage-displays-hack-chat-with-fran-blanche/>

加入我们太平洋时间 8 月 11 日星期三中午的[复古展示黑客聊天](https://hackaday.io/event/180822-vintage-displays-hack-chat-with-fran-blanche)与[弗兰布兰奇](https://hackaday.io/hacker/220960-fran-blanche)！

就集成的容易程度和可以显示的信息密度而言，很难否认这样一个事实，即像 LCD 面板这样的现代显示器并不比过去基于字符的显示器优越。把它扔进一个项目，从一些现成的库中添加一点代码来驱动它，你就可以开始下一项工作了。

高效，是的，但是这种方法对工程师的灵魂有什么作用呢？它挠什么设计痒；它颂扬的是什么审美？怀旧的问题，没错，并不是每个项目都适合探索旧的显示技术。但有些人仍然这样做，谢天谢地，当情况需要时，我们很高兴有那些仍然积极参与复古展示社区的人，确保曾经最先进的技术仍然能够被添加到现代项目中。

毫无疑问，Fran Blanche 是将古董展示的火炬传递给下一代的人之一。你肯定会从她在 YouTube 上受欢迎的[弗兰实验室频道](https://www.youtube.com/channel/UCMLgHbpJ8qYqj3CkdbvC0Ww)中认识弗兰，在那里，除了大约一百万个其他兴趣，她还探索了一些非常酷的老式显示器:[的尼莫](https://hackaday.com/2017/10/19/the-nixie-tube-killer-that-never-was/)阴极射线管、[超亮的白炽灯七段显示器](https://hackaday.com/2021/04/20/incandescent-7-segment-displays-are-awesome/)，令人愉快的奇怪的[“Bina-View】、](https://hackaday.com/2020/04/07/bina-view-a-fascinating-mechanical-interference-display/)等等。Fran 将在 Hack Chat 上停下来谈论所有这些复古展示，她从收集它们中学到了什么，以及它们如何塑造了我们这些天来习以为常的展示。哦，也许我们还会谈到[她即将乘坐的“G-Force 1”](https://www.youtube.com/watch?v=745YqMF8GBY)。
