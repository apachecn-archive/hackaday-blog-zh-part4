# 切向摆动切刀使零件不受起伏

> 原文：<https://hackaday.com/2020/02/27/tangental-oscillating-cutting-knife-makes-parts-from-the-ups-and-downs/>

如果你认为手动使用美工刀是一种累赘，你并不孤单。[鲁本 111]主动把你手上的磨损卸下来，放进一台定制的机床里，他们称之为*TOCK*T3，或*T*angental*O*scilling*C*utting*K*NiFe。TOCK 螺栓连接到典型的 CNC 路由器上，使其能够短时间加工薄材料，如纸板。然而，TOCK 不是施加恒定的向下压力，而是以高速垂直振动，在材料上打孔，同时以适当的速度切割。

TOCK 的振荡由径向对称的凸轮机构驱动，允许刀片完全旋转一整圈，同时仍然进行振荡。虽然传统的将刀片栓接到 CNC 机器上的廉价方法是被动地沿着它们被指引的路径旋转，但[luben111]已经采取了为该轴提供动力的慷慨的额外步骤，通过将 PLT 文件转换为 g 代码的自定义脚本来命令刀片在 cutting director 中主动旋转。最终的结果是一个工具，保留了笨重的厚材料，如纸板大量的细节。最重要的是，整个设置都记录在 Thingiverse 上，带有 CAD 文件和灯光说明。一些人甚至走得更远，甚至复制了他们自己的 T1！

很高兴看到一些涉足各种学科，以生产一个工作的机床。就刀具而言，我们开始看到其他实用刀具增强和用例的良好传播，无论是传统数控改造还是 T2 自制超声波模块的坚实尝试。
