# 模块化 Z80 真正提升了复古计算机的信誉

> 原文：<https://hackaday.com/2022/07/19/modular-z80-really-racks-up-the-retrocomputer-cred/>

很少有逆向计算项目不是一项热爱的工作。与今天的入门级电脑相比，实在没有任何实际理由来建造一台几乎不能胜任任何任务的电脑。但是缺乏做某事的实际理由很少能阻止黑客，就像这个漂亮的基于 Z80 的模块化机架式计算机一样。

事实上，与现代多核千兆赫电脑相比，复古电脑至少在一个领域表现出色——那就是怀旧。这就是[Ricardo Kaltchuk]在他的构建中要做的事情，从在他的零件箱中找到 Z80 和 Intel 8251 USART 开始。这些形成了后来成为“质子”计算机的核心，这是一个模块化的美丽建筑，由大约 7 厘米×10 厘米的印刷电路板组成，插入由铝角制成的机架内的背板。除了电源和 Z80 CPU，其他模块包括一个带有零插入力 EPROM 插座的 RAM 卡，一个带有 128 MB 压缩闪存卡的大容量存储模块，以及标准串行和 I2C 通信模块。

合身度和光洁度都很出色，性能令人印象深刻。质子运行 CP/M，并拥有大量的旧应用程序，这些程序将带回一些记忆，如*超级计算*和*数据库*。我们敢打赌, *WordStar* 就在某个地方，或者很可能就在那里。下面的视频有点粗糙，但很好地展示了一切。

在某些方面，Proton 让我们想起了 RC2014 ，但它的贴合性和光洁度才是它的亮点。不过，这并不是说李嘉图已经投入了大量的工作。这份 62 页的手册包含了每个模块的每个细节，以及构建自己的模块的说明。
