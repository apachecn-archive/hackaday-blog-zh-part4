# 使用这款 8 轴摄影起重机加快拍摄速度

> 原文：<https://hackaday.com/2019/09/17/speed-up-filming-with-this-jawdropping-8-axis-camera-crane/>

如今，除非你在网上发布了一个关于这个项目的视频，否则你会觉得这个项目并不存在。[mingul]正在制作自己的视频，但发现不得不反复移动和设置摄像机很累。自然，一个完全多余的八轴运动控制机器人成为了解决方案。视频嵌入下方休息。

建筑的规模是值得一看的。它在 X 轴上的行程为 4.5 米，在 Y 轴上的行程为 6.5 米，在 Z 轴上的行程为 2.1 米，能够穿越[mingul]车间的整个长度。倾斜，平移和滚动轴都具有 540 度的旋转，还有电机来控制相机的变焦和对焦。通过像 Dragonframe 这样的软件，可以对复杂的相机移动进行编程，而像经典的 dolly zoom 这样的技术对于这样一个多功能的钻机来说是不在话下的。还可以用无线 Xbox 控制器实时控制运动。

[mingul]报道称，该建筑花费了整整三个月的 CNC 加工、3D 打印和组装时间。这比简单的电动相机滑盖高了一大步，但我们都必须从某个地方开始。
