# 动手操作 Voltera V-One PCB 打印机

> 原文：<https://hackaday.com/2021/02/05/hands-on-with-the-voltera-v-one-pcb-printer/>

对许多人来说，创建自己的 PC 板是一个必经之路。不过，现在你可以订购超级便宜的主板，而且很快就能拿到，所以自己动手做并不总是有意义的。尽管如此，一些人喜欢挑战，而另一些人甚至不想等几天。可能每个人都梦想有一台类似 3D 打印机的机器，可以生产出漂亮的印刷电路板。Voltera V-One 还没有达到那种复杂程度，但也差不了太远。[Great Scott]在下面的视频中向我们展示了他如何使用系统构建两种不同的电路板。虽然结果令人印象深刻，但您也可以看到有一些限制，尤其是如果您不是在设计电路板时考虑机器的话。

有一点是显而易见的，那就是这台机器确实需要你的帮助。除了对准孔，你还需要为过孔安装小铆钉，为通孔元件安装稍微小一点的铆钉。[上次我们看机器](https://hackaday.com/2015/11/20/review-voltera-v-one-pcb-printer/)，它根本没有钻孔，但【Scott】展示了钻孔附件，它允许机器生产通孔并支持有引线元件。

该系统依靠可互换的头和导电墨水。在机械上，它和 3D 打印机没有太大区别。然而，这种墨水似乎有点不稳定。在第一个测试板上，一面没有完全固化，因为[Scott]忘记了一个步骤。虽然它看起来不错，但焊接有问题。因为走线不是铜，机器也有自己推荐的焊料。

然而，最有趣的部分是当[斯科特]试图将现有的电路板移植到系统中时。他必须对铜填充和通孔尺寸做一些改变。紧密的间距需要一些手工返工，因此他可能应该在设计中使走线间距更宽。

打印机不便宜，耗材也很贵。很难证明仅仅为了得到快速原型而花费的成本是合理的，因为它不能很好地代表最终的电路板。请记住，导电油墨不像铜那样导电，也不会像阻焊膜或丝网印刷那样精细。如果你有一个活跃的黑客空间，一所学校，或者一个繁忙的实验室，购买一次性的主板可能是值得的。然而，对于原型制作板，您不太可能希望将最终设计局限于打印机所需的限制。此外，如果你不介意等待电路板——请记住，如果你不介意支付更多费用，你可以花钱快速制作和运送电路板——打印机的盈亏平衡点非常高。

另一方面，3D 打印机过去很贵，现在到处都是。3D 打印的质量不如其他一些生产方法，但它仍然有一席之地。因此，如果这项技术改进一点，变得便宜很多，它可能有一天会出现在你的车间里。

早在 2015 年，我们预测[制造自己的 PCB 变得越来越难以证明](https://hackaday.com/2015/09/21/why-are-you-still-making-pcbs/)。当时，许多人对这一预测感到不满，但为你制造主板的成本进一步下降，继续削弱了你自己制造主板的价值主张。当然，有时候你只是想不顾经济状况做点什么——我们都这么做过。但是这确实使得印刷电路板打印机的经济性更加难以计算。
