# 用胶合板建造分布式扬声器

> 原文：<https://hackaday.com/2020/12/16/building-distributed-mode-loudspeakers-with-plywood/>

分布式扬声器的工作方式与 99%的应用中使用的典型驱动器截然不同。他们不是使用类似活塞的运动来产生声波，而是依赖于激发整个面板振动，从而产生声音。[【JGJMatt】决定造一对书架大小的单元，效果很大。](https://www.instructables.com/Plywood-DML-Speakers/)

构建从一对 44 毫米 DML 激励器开始，这在网上很容易买到。这些必须修改，以消除他们的股票金属安装板，降低了早期测试的声音输出。相反，3D 打印件被用来将激励器安装到 3 毫米的胶合板上，这些胶合板被激光切割成主要的 DML 面板。此外，为了进一步提高扬声器的高频响应，在面板上安装了 whizzer 锥。扬声器支架由更多的 3D 打印件和铝棒组装而成，给最终产品带来了干净、现代的外观。

根据测试视频，扬声器的性能令人钦佩，尽管[JGJMatt]指出，它们应该与低音炮配合使用，因为 DML 单元不容易产生 100Hz 以下的频率。我们以前也见过规模更大的类似建筑。休息后的视频。
