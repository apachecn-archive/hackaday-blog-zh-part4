# 柔性印刷电路板机器人来回移动

> 原文:[https://hack aday . com/2019/02/27/flexible-PCB-robot-flops-around-to-get-around/](https://hackaday.com/2019/02/27/flexible-pcb-robot-flops-around-to-get-around/)

在他不断寻求尽可能减少机器人部件数量的过程中，[卡尔·布格贾]偶然发现了一个不寻常的设计:[几乎只由 PCB 制成的机器人](https://hackaday.io/project/162529-pcb-robot)。

诚然，在这一点上，称这些松软的四条腿的生物为机器人还是有点牵强。下面的视频显示，虽然它们确实在自己的力量下移动，但对运动没有太多的控制——目前还没有。[Carl]的设计使用了由 FR4 制成的看起来极其脆弱的上臂组件。每个手臂都有一个小钕磁铁，悬挂在一个柔性 PCB 线圈的中心，就像我们以前见过他用作致动器和[扬声器](http://hackaday.com/2018/09/21/the-diaphragm-is-the-coil-in-these-flexible-pcb-speakers/)一样。线圈由位于四条腿交叉处的微控制器控制。在一些不令人振奋的拴系测试揭示了过度顺从的 FR4 磁体支撑的一些问题后，[Carl]做了一些改变并提高了腿部运动的频率。这导致了实际的运动，并最终实现了自由操作，机器人愉快地嗡嗡作响。

仍然存在磁铁臂缺乏刚性的问题，但我们乐观地认为[卡尔]可以克服这些问题。我们非常喜欢这个想法，可以看到各种灵巧的拍打和扑腾运动的应用。
