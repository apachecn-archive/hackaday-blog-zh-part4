# 三角洲机器人正在整理高尔夫球和取名字

> 原文：<https://hackaday.com/2019/01/09/delta-robot-is-sorting-golf-balls-and-taking-names/>

这是每个辛勤工作的美国人都面临的普遍情况——你在钙矿工作了一整天后回到家，发现自己被困在一堆彩色高尔夫球中，这些球根本不会自己分类。最后，你可以收起你的分拣漏斗和运球手套——[【安东尼】有解决方案](https://youtu.be/Tg7quTMk-_k)。

没错——这是一个 delta 机器人，任务是按照颜色对高尔夫球进行分类。一个像素 2 的物体跟踪相机被用来观察桌子，三角形的手臂四处抽动，以使相机获得无障碍的视野。一旦知道了球的位置，就进行一次冒泡排序，将球重新排列成正确的颜色顺序。

[Anthony]欣然承认冒泡排序在这项任务中效率很低；这是一个有意的选择，以便以后可以与其他排序方法进行比较。[Anthony]还详细介绍了吸入式抓具的开发过程，并讨论了降低噪音的减震方法。

Delta 机器看起来总是很有趣，是分拣机的好选择。我们也见过一些非常小的。休息后的视频。
