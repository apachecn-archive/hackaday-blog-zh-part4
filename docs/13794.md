# 是抖音，在你的手腕上！

> 原文:[https://hackaday.com/2022/06/01/its-tiktok-on-your-wrist/](https://hackaday.com/2022/06/01/its-tiktok-on-your-wrist/)

20 世纪 80 年代初，精工电视腕表是人们渴望的终极目标之一。它需要在你的口袋里装一个大盒子来装它的电子设备，这没关系，它是你手腕上的电视，未来就在这里！当然，现在我们有技术让腕戴式视频成为现实，但令人难过的是，我们选择了用手机看视频，却从未真正兑现过腕戴式电视的承诺。然而希望总是存在的，这就是[戴夫·班尼特]的[ESP32 驱动的抖音手表](https://hackaday.io/project/185568-esp32-tiktok-watch)。

手腕上是带有音频 DAC 和放大器、LCD 屏幕和电池的 ESP32 本身，但遗憾的是，这个组合没有足够的能力直接与抖音通话。相反，这是在一台配套的 PC 上使用 Python 完成的，最终的视频通过 WiFi 上传到设备上。不是精工电视那种笨重的电子产品，我们就要了。所有信息[都可以在 GitHub](https://github.com/DaveBben/unofficial_tiktok_smartwatch) 上找到，在休息时间下面有一个 YouTube 视频。

因此，一代人的病毒式视频现在可以在移动中拍摄，而不需要借助稍微不太便携的手机。这可能不是最不显眼的钟表，但它肯定会让你注意到。
