# 简单的 3D 打印机械臂使用柔顺机构

> 原文:[https://hack aday . com/2020/02/09/simple-3d-printed-robotic-arm-uses-compliant-mechanism/](https://hackaday.com/2020/02/09/simple-3d-printed-robotic-arm-uses-compliant-mechanism/)

通过玩耍学习对所有年龄的人都有效，自 2016 年以来,[ sland concepts]一直在设计 STEM 套件，帮助教孩子们建立他们未来的霸主。他们正在推出他们的 LittleArm 机械臂的[版本 3，从版本 1 的进展是一项在不牺牲功能的情况下简化和减少零件数量的有趣研究。](https://hackaday.io/project/169809-littlearm-v3)

在所有的 LittleArm 版本中，主要的机械部件都是 3D 打印的，由 3 个伺服系统驱动运动，外加一个额外的伺服系统来运行夹持器。这些套件专门用于重复构建和拆卸，教室是小螺钉容易消失的好地方，因此减少螺钉的数量是 v3 的一个大目标。与以前的版本相比，夹持器/前臂表现出了最显著的改进，通过使用一种柔性机制，从 8 个独立的组件简化为单个 3D 打印部件——这种弯曲的模式允许夹持器弯曲到位。夹持器尖端还具有一个简单的“切口”，使其更容易抓住水平物体。

基于 Arduino Nano 的扩展板用于控制手臂，HC-06 蓝牙模块允许通过智能手机应用程序控制它。还可以添加各种传感器来扩展套件的功能。不幸的是，机械设计不是开源的，但它仍然可以成为你自己设计项目的灵感来源。

希望这个工具包能激励一些未来的黑客建造一个更先进的 3D 打印版本，甚至是一个巨大的液压动力臂，比如 T2。
