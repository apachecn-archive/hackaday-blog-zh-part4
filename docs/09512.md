# 木制的你爱做一个缎带麦克风？

> 原文:[https://hack aday . com/2021/03/18/wooden-you-love-to-build-a-ribbon-microphone/](https://hackaday.com/2021/03/18/wooden-you-love-to-build-a-ribbon-microphone/)

当考虑建造音频设备时，来自枯树的碳水化合物泡沫不是第一个想到的材料。但实际上，没有理由不探索新材料来完成通常为金属或塑料保留的工作，当成功完成时，就像[这款木丝带麦克风](https://www.youtube.com/watch?v=1DsqWRe4nMc)一样，结果看起来和听起来都很棒。

公平地说，在[弗兰克·奥尔森]的经典 RCA 型麦克风的复制品中，有许多非木材部件。毕竟，很难让木材表现出将声能转化为电流所需的电磁特性。但这并不意味着木材，特别是胡桃木贴面，在这个设计中不是最重要的。[弗兰克]用薄单板工作；用商用乙烯基切割机切割成型，并与交替的纹理堆叠在一起，用大量的氰基丙烯酸酯粘合剂将木材粘合起来，形成各种胶合板。狗骨形的机身上安装了两块钕磁铁，留出了一个足够容纳麦克风带状振膜的间隙。它是由一片薄铝箔制成的，使用 DIY 卷曲辊形成波纹。悬挂在磁铁之间并连接到导线，麦克风元件用木头和织物挡风玻璃装饰，并悬挂在临时框架中的松紧带上进行测试。下面视频上的旁白是用麦克风录制的，听起来相当好听。

我们以前见过[丝带话筒](https://hackaday.com/2019/01/02/diy-ribbon-element-upgrades-a-studio-microphone/)，以及[木质话筒](https://hackaday.com/2020/04/18/wood-and-carbon-rods-used-for-this-handsome-and-effective-microphone/)，但这是我们第一次见到木质丝带话筒。看起来似乎[弗兰克]有更多的工作要做，以完成它，我们急切地等待着成品。
