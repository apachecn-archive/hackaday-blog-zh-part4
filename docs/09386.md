# 蒂姆·亨金带着零件的秘密生活再次上路

> 原文：<https://hackaday.com/2021/02/28/tim-hunkin-rides-again-with-the-secret-life-of-components/>

长期读者可能会记得偶尔出现的一个*工程英雄*系列，该系列聚焦于英国工程师、发明家兼电视节目主持人蒂姆·亨金，因其复杂的创作、不寻常的街机和 1990 年左右的*机器的秘密生活*电视连续剧而闻名。随着他带着一个新系列通过 YouTube 回到我们的屏幕上，我们现在似乎有了新的享受。 [*《零部件的秘密生活》*](https://www.youtube.com/watch?v=6JAgXz6xO0s) 将是他试图将长期职业生涯中积累的知识传授给我们大多数人的尝试。

该系列将于 3 月 4 日推出八个视频，从休息时间下方的预览视频片段来看，他将涵盖广泛的内容，包括弹簧、粘合剂、链条、皮带、开关等。他的娱乐风格和漂亮的工作模型保证会成为一些非常好的内容，同时为真正的工艺大师的工作室提供了一个独特的视角。

作为一个开胃菜，我们对蒂姆·亨金的简介值得一读。它的特色是参观他在伦敦霍尔本的[新奇自动化](https://www.novelty-automation.com/)拱廊，这应该是任何一个在那个城市旅行的休闲读者必去的地方。
