# BlinkBox:可寻址 led 的调试工具

> 原文：<https://hackaday.com/2018/07/25/blinkbox-debugging-tool-for-addressable-leds/>

您有多频繁地发现自己不得不暂停项目来制作测试电路或编写一些测试代码来找到问题的根源？对同一个测试做足够多的变化，你最终会做出一个专用的测试工具。这正是[德文·布雷]发现自己在做的事情。

[Devon]对不同类型的可寻址 led 做了大量的工作，经过大量的经验，创造了[blink box，一个可寻址 led 的专用测试工具](http://www.esologic.com/blinkbox/)。它支持多个 LED 芯片组，你可以给它一个你想要点亮的 LED 的数量，并且你可以选择一个测试动画。它甚至可以将你的设置写入 EEPROM，这样你下次开机时就不用重复了。

他还做了非常好的包装工作，创建了一个 3D 打印的外壳，使用背光按钮在黑暗中工作，甚至为 LCD 屏幕添加了一个对比度旋钮。他为制作这个所付出的努力值得称赞。复制它所需的一切都可以在他的网页上找到，还有好奇者的示意图。观看它的实际操作，或者在下面的视频中欣赏他的杰作。

我们在 Hackaday 上找不到任何类似的黑客，但这里有一个漂亮的 LED 测试仪[，它还会告诉你使用哪个电阻值](https://hackaday.com/2016/01/29/led-tester-royale/)。
