# shape shifter——一个开源的鼓机

> 原文：<https://hackaday.com/2019/10/13/shapeshifter-an-open-source-drum-machine/>

随着微控制器每一代都变得越来越强大，以前仅仅是白日梦的事情现在已经很容易实现了。变形鼓机器就是一个很好的例子。

变形人的设计是开源的，[Github](https://gitlab.com/Faselunare/shapeshifter)上为好奇的音乐修补者提供了一切。该器件围绕一个只有通孔元件的 PCB 构建，即使是经验最少的发烧友也能轻松组装。然后将一个 3.6 兆赫的天线插入板上的插座，提供 180 兆赫的咕噜声来运行节目。这是一个很好的选择，因为 Teensy 平台有大量的库，这使得处理音频变得很简单。

作为开源软件，不仅自己制作很容易，而且有足够的空间根据你的个人喜好重新组合设计。甚至还有一个试验板区域，可以添加扩展卡来实现更多可能性。一些用户甚至走得更远，添加显示和过滤模式来真正打开东西。

我们喜欢 Hackaday 的好鼓机，从阿姆斯特拉德的到 T2 的。[如果你有自己的身材，一定要把它放在提示栏里。](http://hackaday.com/submit-a-tip)
