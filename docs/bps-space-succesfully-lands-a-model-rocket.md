# BPS。太空成功着陆模型火箭

> 原文：<https://hackaday.com/2022/08/05/bps-space-succesfully-lands-a-model-rocket/>

如果你在过去几年里一直关注[乔·巴纳德]的火箭项目，你就会知道他的主要目标之一是像 SpaceX 一样推进着陆模型火箭。现在，经历了 7 年的过山车之旅，他终于用他最新版本的侦察火箭实现了这个目标。

![Rocket touching down](img/8658a52bea2d57d4816968d23efc5ea2.png)

We have touchdown!

使用标准的业余爱好级固体燃料火箭发动机来发射和着陆火箭需要很多东西。一个核心组成部分是火箭在整个飞行过程中的稳定性，这是通过使用火箭发动机的推力矢量控制(TVC)装置和装有精心调整的制导软件的定制飞行计算机来实现的。直到最近，TVC 坐骑还是 3D 打印的，但[乔]将其升级为[加工铝](https://www.youtube.com/watch?v=Mq5alUV-AHI&t=325s)以尽可能消除弯曲和间隙。

由于固体燃料火箭在技术上不能被节流，[乔]最初试图以这样一种方式确定下降发动机的点火时间，即它会在火箭着陆时烧尽。点火时间和确切的推力数字根本不够可重复，所以在他的 [2020 着陆尝试](https://hackaday.com/2020/11/27/so-close-to-landing-a-model-rocket-on-its-tail/)中，他通过左右摆动 TVC 实现了一些节流效果，减少了垂直推力分量。这最终让位于最终的解决方案，一对[陶瓷钳子](https://hackaday.com/2022/03/18/throttle-your-solid-rocket-motors-with-this-one-simple-trick/)，根据需要阻止马达的推力。

另一个有趣的组件是着陆腿。它们由轻质碳纤维棒制成，通过用镍铬合金线熔化橡皮筋来释放，并在弹簧张力下折叠到位。他们还必须小心翼翼地改进，以吸收尽可能多的冲击而不反弹，这扼杀了以前的几次着陆尝试。

回放[Joe]的视频，看到他在工程方面的进展绝对令人振奋，我们期待着他未来的计划。其中包括腹部下垂的星际飞船的功能比例模型，一个神秘的“肉火箭”，和一个大的，超过 100 公里高度的空间拍摄。
