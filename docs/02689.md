# 这个自行车打气筒现在糟透了

> 原文:[https://hackaday.com/2019/04/11/this-bike-pump-now-sucks/](https://hackaday.com/2019/04/11/this-bike-pump-now-sucks/)

抽真空是每个自豪的制造者偶尔必须做的事情。无论是对环氧树脂脱气还是对零件进行真空成型，您都需要一个可靠的泵来完成这项工作。[drcrash]正好有指南可以帮助——[如何将普通的手泵转换成真空吸尘器。](https://www.instructables.com/id/make-a-manual-vacuum-pump-for-under-%252420-by-convert/)(视频中断后嵌入。)

[drcrash]建议从粘液品牌 2060-A 泵或类似产品开始。这是一个基本的手泵，没有压力表或其他多余的东西。它也有一个坚固的钢轴，可以承受重复使用。你还需要一些管道和止回阀来完成这项工作。

基本概念是重新配置泵，将空气从物体中吸出，而不是吹入。通过移除原来的止回阀并在相反的方向安装一个，并反转泵的活塞，可以毫不费力地获得良好的真空。[drcrash]报道说，这种设置可以达到低于大气压 11 psi，这对于广泛的应用来说是足够的。如果你需要更进一步，你可以试着建造你自己的涡轮分子泵。
