# 抬头:智能玻璃万用表

> 原文：<https://hackaday.com/2021/04/18/heads-up-smart-glass-multimeter/>

有时很难探测电路，然后查看仪表。[electronio OBS]决定通过使用有机发光二极管屏幕和蓝牙模块制造一款类似谷歌眼镜的万用表来解决这个问题。

定制 PCB 没有太多惊喜。一块小型板有一个控制器、一个电池充电器、一个显示器和一个蓝牙模块。不过，他确实忘记了一个开关，所以除非你安排一个外部开关，否则这块板总是开着的。

CPU 是从一个现有的 Arduino 模块中取出来的，我们认为它的优点是已经安装了引导加载程序，但是如果我们正在构建它，我们可能会使用一个新的芯片。光学系统需要一面镜子和一个镜头，这样你就可以清楚地看到显示器。矛盾的是，显示器在后面，让你可以很容易地专注于它。

如果你以前没有使用过蓝牙模块，这是一个很好的入门项目，展示了两种连接模块的方法。

我们应该提到的是，该项目从[Alain Mauer]在[hack aday . io](https://hackaday.com/2016/10/05/hackaday-prize-entry-head-up-for-high-voltage/)上的一个类似项目中获得了灵感。我们已经看到了很多这种眼镜式显示器的设计，包括[一个 13 岁孩子的好设计](https://hackaday.com/2015/11/10/the-simplest-smart-glasses-concept/)。
