# 用一些巧妙的技巧改进一台小型车床

> 原文:[https://hackaday . com/2021/11/13/用几个聪明的黑客改进一台迷你车床/](https://hackaday.com/2021/11/13/improving-a-mini-lathe-with-a-few-clever-hacks/)

像许多预算机械师一样，这位令人愉快地乐观命名的[我们可以做得更好]在他的进口迷你车床上遇到了一些更好的控制问题。但是，他没有忍受痛苦，而是选择纠正机器的缺点，并在这个过程中，教给每个人一些很棒的技巧。

[我们可以做得更好]的车床改造集中在车厢手轮上，它似乎缺乏适当的轴承，并以最不精确的方式摆动。最重要的是，驱动装置的传动装置使得手轮每转一圈，滑架移动 19 毫米，这是不令人满意的。一个单一的齿轮变化，使每转均匀 20 毫米，当加上一个校准和索引手轮环大大简化了托架行程测量。

虽然最终结果的建设是相当大的权利，我们的钱最好的视频部分是其丰富的机械师的提示集合。使用木制销钉和印刷纸模板来代替合适的分度头是非常聪明的，就像使用车床的尾座来驱动雕刻工具来切割索引线一样。我们以前见过使用安装在刀架上的 Dremel 工具来代替铣床，但是看到使用这种技巧总是很好的。将刻度盘锁在手轮上的机制也非常巧妙。

考虑一个迷你车床？尽管[我们可以做得更好]的经历可能令人鼓舞，但深入了解这种机器的利弊可能是明智的。
