# 极坐标求积仪通过绘制周长来量化面积

> 原文：<https://hackaday.com/2022/03/25/polar-planimeter-quantifies-area-by-plotting-perimeter/>

如今，如果不精通某种 CAD 技术，很难被贴上“制造者”或“黑客”的标签——即使 C 代表纸板。但是在 CAD 出现之前，已经有了制图和相关的艺术，人们不能仅仅选择一个形状，然后在你选择的正方形单位中看到它的面积。那么，一个老派的绘图员怎么能算出复杂形状的面积呢？[Chris Staecker]向我们介绍了 polar 求积仪，这是一种专门为此目的而设计的测量工具，在休息时间下方的视频中对[进行了详细说明。](https://www.youtube.com/watch?v=jMvEOmpy8Kw)

正在讨论的极坐标求积仪是 20 世纪 60 年代的高端设备。有趣的是，第一个极坐标求积仪是在 19 世纪早期发明的，甚至在描述其功能的数学完成之前。一根杠杆放在一端的固定位置，另一端插入求积仪。求积仪本身有另一个臂，上面有一个十字线。该单位是零了一个按钮，和轮廓的形状是在顺时针方向与十字线跟踪。

使极坐标求积仪能够进行多维测量的是固定臂。固定臂绕枢轴转动，使求积仪能够跟踪影响输出的角度变化。所以，求积仪不仅仅测量周长，还测量周长的大小。最终测量值以平方英寸为单位输出。

总的来说，这是一个我们不知道存在的非常巧妙的工具，在所有事情都可以通过点击一两次鼠标来完成之前，看到这样的问题是如何被解决的是非常有趣的。一定要看看这本 [100 多年前的参考书，来充实你过去的知识](https://hackaday.com/2021/06/04/old-textbooks-galore/)。感谢[赞]的伟大提示！
