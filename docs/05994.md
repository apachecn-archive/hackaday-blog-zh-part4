# 激光雷达系统不再仅仅是测距仪

> 原文:[https://hack aday . com/2020/03/24/lidar-system-not-just-a-range der-more/](https://hackaday.com/2020/03/24/lidar-system-isnt-just-a-rangefinder-anymore/)

对于任何项目来说，质量和成本之间通常都有一个权衡，因为更高质量的零件、更多的功能或项目的任何方面都可能推动其价格上涨。似乎[伊利亚萨姆]在他的项目中成功地完全避免了这种范式。他的新激光雷达系统在精度、采样和质量方面都出类拔萃，不知何故，部分成本仅约 114 美元。

激光雷达系统的工作原理是向不同方向发出许多光脉冲，在光返回时测量光的反射。因此，激光雷达系统随着用于激光输出和接收数据的更高频率脉冲和更快控制电子设备而改进。该系统能够精确到几厘米以内，并在每秒 15 次扫描的情况下工作长达 25 米。关键是一个高功率激光模块，它可以在极短的时间内输出高达 75 瓦的功率。更多细节可以在这个页面找到 ( [谷歌翻译自俄语](https://translate.google.com/translate?hl=en&sl=auto&tl=en&u=https%3A%2F%2Fhabr.com%2Fru%2Fpost%2F485574%2F))。

这个项目的另一个好处是，[iliasam]已经从他的 GitHub 页面提供了所有可用的东西[，包括硬件规格，所以只要你有一台 3D 打印机，这也不会花很长时间来制作。甚至还有激光驱动电路如何工作的详细说明，以及如何内置安全功能来防止任何人的视力意外受损。不用说，这不仅仅是一个激光测距仪模块，但如果你想知道如何重新利用这些模块，](https://github.com/iliasam/OpenTOFLidar)[【伊利亚萨姆】也可以向你展示这些](https://hackaday.com/2018/05/22/hacking-a-cheap-laser-rangefinder/)。
