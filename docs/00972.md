# 弯曲电视线路以获得更好的输入

> 原文:[https://hack aday . com/2018/10/15/circuit-bending-a-TV-for-better-input/](https://hackaday.com/2018/10/15/circuit-bending-a-tv-for-better-input/)

如果你没注意到，CRT 越来越难找了。你不能善意地得到它们，因为旧货店不再接受巨型显像管电视了。你不可能在为收垃圾的人设置的路边找到它们，因为他们不会把它们捡起来。在易贝很难找到它们，因为没人想运送它们。这是一个遗憾，因为享受老式电脑和游戏系统的最佳方式是使用带有 RGB 输入的 CRT。如果你还没有，你能期待的最好的就是一台带有复合输入的老式 CRT 显示器。

但是有一个办法。[8 位家伙] [刚刚打开了 90 年代末的 CRT 电视](https://www.youtube.com/watch?v=DLz6pgvsZ_I)，并对其进行了修改以接受 RGB 输入。这是你的苹果显示器，你的准将显示器，也是你的 Sega Genesis 更好的显示器。

在拆开一台旧的 CRT 显示器并摆弄电路之前，有一些事情需要了解。每个(彩色)阴极射线管有三个电子枪，分别用于红色、绿色和蓝色。这需要高电压，在具有 RGB 输入的 CRT 中，你看到的是一个电路路径，它接收这些输入，放大它们，然后将它们发送到电子枪。如果电视只有一个复合输入，有一点电路将复合信号分开，并将其发送到枪。在[8 位家伙]的电视中——以及从 90 年代中期到后期你会发现的几乎所有 CRT 电视中——都有一个处理这种转换的“丛林 IC ”,大多数时候都有用于屏幕显示的 RGB 输入。通过简单地接入这些输入，你可以在背面用花哨的 RCA 插孔添加 RGB 输入。

虽然为 90 年代末的 CRT 添加 RGB 输入的实际过程会因每个品牌和型号的不同而略有不同，但过程几乎是相同的。这真的只是一点点焊接，然后坐下来玩旧电脑，最终在正确的屏幕上显示正确的颜色。
