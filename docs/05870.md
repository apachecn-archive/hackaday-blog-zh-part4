# 相机和代码联手制作不可能的悬停激光效果

> 原文:[https://hackaday . com/2020/03/05/camera-and-code-team-up-to-make-impossible-homping-laser-effect/](https://hackaday.com/2020/03/05/camera-and-code-team-up-to-make-impossible-hovering-laser-effect/)

马上，我们会说[这个显示激光束停在半空中的视频](https://www.youtube.com/watch?v=tozuzV5YZ7U)只不过是一个相机的把戏。但诀窍就是诀窍，你必须承认它看起来真的很酷。

从[汤姆·斯科特]的视频开始，休息后的第一个视频。[Tom]非常擅长以一种优雅而吸引人的方式展示迷人的话题，他在这里确实做到了。在一个黑暗的房间里，一个扭曲的[汤姆]用一束看似缓慢移动的光束摆姿势，类似于一百万部科幻电影中激光武器似乎总是无视物理定律。他甚至成功地用“强制停止”和有点笨拙的*矩阵*风格的子弹时间镜头在慢动作光子上拉了一个【凯洛·伦】。这是娱乐性的东西，这种效果完全归功于卷帘式快门效应。激光束与相机的快门同步快速调制，随着相机转动 90 度，效果是减慢甚至停止光束。

黑客的棘手部分是激光，这是[Seb Lee-Delisle]的杰作。下面的第二个视频详细介绍了他的效果。我们之前已经看过[Seb]的作品，用[一个巨大的激光*小行星*游戏](https://hackaday.com/2017/03/08/light-replaces-electrons-for-giant-vector-graphics-asteroids-game/)和[一个诡计 NES 激光爆破器](https://hackaday.com/2016/08/31/nes-light-gun-fires-awesome-laser-effect/)对抗这个效果。
