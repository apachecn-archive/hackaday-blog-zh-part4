# 对所有人来说，这是一个非常公平的交易

> 原文：<https://hackaday.com/2022/07/07/a-mostly-fair-deal-for-all-with-a-raspberry-pi/>

成为一名专业的发牌员需要相当的技巧，这甚至不是每个人都能熟练掌握的。幸运的是，即使是最笨手笨脚的发牌员也有一个解决方案，那就是大卫·斯特恩的自动发牌员使用一个带有摄像头和模式识别功能的 Raspberry Pi 4。

它采用伺服控制臂的形式，末端有吸盘，能够拾取卡片并将其呈现给相机。然后，可以通过价值来识别它们，并且可以分发预先确定的手牌或者随机的手牌。看起来预定的手牌不是扑克作弊的辅助手段，而是桥牌玩家艺术的一部分。你可以在休息时间下面的视频中看到它的作用。

我们喜欢这个项目，但遗憾的是，在这一点上，我们必须让[Dave]承担责任，因为尽管他提供了足够的细节来吸引我们的兴趣，但他却拒绝向我们展示代码。想到来自心怀不满的黑客读者的叫嚣可能会刺激他扔给我们一两块面包屑，那就太好了。

你可能不会惊讶地发现[这不是第一个发现自己发牌的树莓派](https://hackaday.com/2020/07/20/raspberry-pi-shuffler-is-computerized-card-shark/)。
