# 无绳电钻长出翅膀，开始飞行

> 原文:[https://hack aday . com/2019/10/16/cordless-drill-sprouts-wings-and-take-flight/](https://hackaday.com/2019/10/16/cordless-drill-sprouts-wings-and-takes-flight/)

无刷电机和锂电池是遥控飞机的一场革命。硝基发动机将不再独领风骚，因为飞行几乎在一夜之间变得更加便宜和容易。同样的技术也在电动工具中找到了自己的路，导致[Peter Sripol]决定[在下面嵌入的视频中](https://www.youtube.com/watch?v=5EiZ0NuDpoQ)在飞行的飞机中建造一个电钻。

未经改造的德瓦尔特钻是建造的核心，作为飞船的推进装置。一个伺服系统被用来启动钻机的扳机作为油门。由于与典型的业余爱好无刷电机相比，电钻齿轮传动明显下降，因此有必要使用比通常情况下大得多的道具。这是在[威廉·奥斯曼]的帮助下用木头定制加工出来的，尽管有一些意外，但(大部分)还是完整无缺的。机身由带有杨木翼梁的泡沫机翼组成，铝挤压件用作尾梁。几个 3D 打印的零件将所有东西连接在一起。

尽管钻孔机很重，但组装起来的飞机能够很容易地飞行。宽大的机翼和螺旋桨有助于弥补动力系统的不足。不幸的是，没有足够的剩余升力来携带智能手机的有效载荷来捕捉飞行中的镜头，但总的来说，该项目可以被认为是一个巨大的成功。
