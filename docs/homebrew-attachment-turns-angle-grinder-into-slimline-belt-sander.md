# 自制附件将角磨机变成细长带式砂光机

> 原文：<https://hackaday.com/2018/11/05/homebrew-attachment-turns-angle-grinder-into-slimline-belt-sander/>

如果有像角磨机这样的小型电动工具，我们还没有找到。这些多用途的工具让你的手掌充满力量，即使没有经过打磨，它们也有很大的功能范围，从切割到研磨到抛光和清洁，只需选择心轴上的东西。

通过一个简单的自制附件，[Darek] [将他的角磨机变成了微型砂带磨光机](https://www.youtube.com/watch?v=LTC1qJrGXnM)非常适合那些难以到达的地方。夹紧圆盘保护装置通常所在位置的附件为研磨机的心轴增加了一个驱动辊；惰辊骑在一个小气动弹簧的末端，使皮带保持张紧状态。砂带本身是从较宽的砂带上切割下来的，附件可以承载各种宽度的砂带。最棒的是，他没有任何复杂的机械工具。没有车床？没问题——使用角磨机本身将驱动辊打磨成保持皮带居中所需的适当凸度轮廓。我们看到的唯一问题是附件不容易从研磨机上拆下来，不过没关系。毕竟，研磨机就像薯片，你不能停在一个地方。

当然，这不是[Darek]的第一次[角磨机黑客竞技](https://hackaday.com/2018/07/02/simple-jig-uses-electromagnet-for-clean-angle-grinder-cuts/)。如果你正在寻找如何打造自己的灵感，别再犹豫了:[一台地板打磨机](https://hackaday.com/2017/08/18/its-an-angle-grinder-no-its-a-floor-sander/)、[一台精密平面磨床](https://hackaday.com/2015/03/13/diy-surface-grinder-for-making-precision-parts-at-home/)，甚至[一辆电动自行车](https://hackaday.com/2017/08/24/possibly-the-most-up-cycled-hacked-e-bike-youll-see-all-week/)都可以被制造出来。
