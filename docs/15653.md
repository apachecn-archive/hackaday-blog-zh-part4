# 转盘悖论是如何工作的

> 原文：<https://hackaday.com/2022/12/14/how-the-turntable-paradox-works/>

把大多数物体放在转盘上，让它旋转，它们很快就会飞走。然而，对一个球做同样的事情，它不知何故设法在上面滚了一段时间而没有掉下来。[Steve Mould]在最近的 YouTube 视频中着手解开这个“转盘悖论”。

在基本情况下，球滚动的事实是它保持在转盘上的原因。根据牛顿第一运动定律，当转盘旋转时，球向相反的方向旋转。只要球被允许以与转盘相同的速度滚动，在没有任何其他干扰力的情况下，它将基本上停留在原地。然而，当球沿着转盘移动时，它很快就会以一种更复杂的圆周运动结束，以转盘本身的速度为比例旋转。[Steve]解释了起作用的机制，并深入研究了事情背后的数学原理。

有时，像这样的演示可能看起来只是好奇。然而，理解像这样的物理效应是开发各种复杂而神奇的机械的关键。休息后的视频。
