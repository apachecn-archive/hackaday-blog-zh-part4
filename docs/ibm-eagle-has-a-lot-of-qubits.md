# IBM Eagle 有很多量子位

> 原文：<https://hackaday.com/2022/02/08/ibm-eagle-has-a-lot-of-qubits/>

量子计算机需要多少量子位？很多，如果你想要有用的东西。然而，今天，我们不得不满足于比我们想要的少得多的东西。但是 IBM 的新 Eagle 拥有最先进的量子计算机:127 个量子位。自然，他们计划做更多的工作，你可以在下面的视频中看到“系统二”的预览。

127 个量子位的数字既大得令人印象深刻，又小得令人沮丧。每一个量子位元都会让传统电脑模拟机器的工作量增加两倍。希望有一天能制造出用传统计算机模拟不切实际的量子计算机。这就是众所周知的量子优势，虽然几个团队已经声称它，但实际实现它是一个争论的话题。

像任何计算机一样，一般来说，更多的比特或量子比特比更少的比特好。然而，它对现代量子系统尤其重要，因为大多数实用方案要求冗余和纠错在量子计算机硬件的现代实现中是可靠的。未来会怎样？IBM 声称，他们将使用与 Eagle 相同的 3D 封装技术来构建超过 1000 个量子位的 Condor 处理器。Condor 计划于 2023 年推出，2022 年将推出一款 433 量子位的中间芯片。

将任何东西放大通常需要的不仅仅是复制较小的东西。在 Eagle 和至少一个它的前辈的例子中，部分缩放是使用可以读取不同量子位的读出单元。只有几个量子位的旧处理器会为每个量子位配备专用的读出硬件，但一旦你得到数百或数千个量子位，这就站不住脚了。

量子位并不是衡量计算机能力的唯一标准，就像比特数多的传统计算机可能不如比特数少的计算机。你还必须考虑量子位的质量以及它们是如何连接的。

谁将赢得量子霸权的竞赛？还是已经赢了？我们有一种感觉，如果它还没有被完成，它在未来不会很远。如果你想想计算机在 1960 年的状态，并与大约 60 年后的今天进行比较，你不得不怀疑在这个领域是否也会有如此大的进步。

你听到的关于量子计算的大多数公告来自谷歌、IBM 或微软。但是还有霍尼韦尔和其他一些玩家。如果你想为量子冲击做好准备，也许可以从[这本教程](https://hackaday.com/2018/01/24/quantum-weirdness-in-your-browser/)开始，它将主要在模拟器上运行。
