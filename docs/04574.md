# 令人毛骨悚然的万圣节娃娃可能会让你变成贝琪·韦兹

> 原文:[https://hack aday . com/2019/10/22/creepy-Halloween-doll-may-make-you-Betsy-wetsy/](https://hackaday.com/2019/10/22/creepy-halloween-doll-might-make-you-betsy-wetsy/)

如果你想在这个万圣节吓吓你的邻居，你可能会选择像骷髅或僵尸这样的老古董。但你不一定要去血淋淋地寻找荣耀。想想那个会说话的娃娃。那些东西已经够吓人的了，对吧？嗯，[cabuu]说不行，[这个娃娃应该是有伺服系统的动画，并且有遥控器](https://www.instructables.com/id/Animated-WiFi-Doll)。她应该还能说话，只是不像你期望的那样。

[![](../Images/05c9da0fa177b0e2a287a388135d8a05.png)](https://hackaday.com/wp-content/uploads/2019/10/doll-eye-bracket.png) 忘记推她的肚子，因为宝宝现在有了一个 Wemos D1 迷你和她自己的 Blynk 应用程序。应用程序中的一组滑块控制着一个使她的眼睛活动的微型伺服系统，以及另一个将她的头从一边扭到另一边的伺服系统。她的脑袋不太清醒，但这可能是最好的结果。有预设的惊吓模式[cabuu]可以设置和忘记，直到她通过运动传感器激活。

我们特别喜欢设计和印刷的支架[cabuu],它将眼球与伺服系统连接在一起，同时他巧妙地利用印刷的配对支架将伺服系统固定在头部的适当位置。如果你认为你能忍受，休息后有一个演示视频。请继续关注之后的娃娃解剖，因为[卡布]建立并插入了可怕的花絮。

我们喜欢既天真又疯狂的黑客。你见过坦克工程师托马斯演唱里克·阿斯特利的歌曲吗？
