# 大 LED 墙看起来很酷，可以吸引超过 170 安培

> 原文:[https://hack aday . com/2019/07/31/big-ol-led-wall-looks-cool-can-draw-over-170-amps/](https://hackaday.com/2019/07/31/big-ol-led-wall-looks-cool-can-draw-over-170-amps/)

建造巨大的 LED 墙伴随着一系列严峻的挑战。无论是电源、电缆布线，还是找到一种足够快地为所有像素输出数据的方法，都需要做一些事情来构建一个像样的显示器。[菲尔]想要一份办公室声明，[所以卷起袖子开始工作](https://medium.com/@incubation.ff/how-to-build-your-very-own-large-led-jumbotron-video-wall-and-go-insane-in-the-process-beef53363ba1)。

这个建筑使用 P5 面板，[，我们之前已经见过在较小规模上使用的](https://wp.me/paBn4l-1yc2)。最初的测试是用 Raspberry Pi 3 完成的，当构建达到 28 个面板时，grunt 就开始用光了。刷新率很低，任何有运动的东西看起来都很乱。在这一点上，一个专用的驱动程序是为了处理完整的 48 面板显示器。其他挑战包括应对巨大的电力需求——在 5 伏电压下超过 170 安培——以及建立一个框架来安全地固定所有面板。

最终的产品令人印象深刻，2 米宽，1.2 米高。分辨率为 384 x 256。有了 Mac Mini 通过现成的驱动程序将视频运行到显示器上，所有形式的内容都是可能的。[Phill]甚至为用户创建了一个松弛的频道，向显示器发送 gif 和文本消息。当然，我们确信没有人会利用这一功能。
