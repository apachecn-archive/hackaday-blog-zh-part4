# 透明化油器给出了一个清晰的演示

> 原文：<https://hackaday.com/2021/07/06/see-through-carburetor-gives-a-clear-demonstration/>

大多数汽车上的化油器已经被逐步淘汰，但一个世纪以来，化油器一直是标准配置，现在仍用于许多小型发动机上。在他父亲的帮助下，带着高速摄像机，[每天都更聪明]通过试验一个连接到真实引擎的 DIY 透明汽化器来研究这些装置。

化油器的作用是将汽油和氧气以正确的比例混合，以便在发动机内燃烧。汽油从油箱流到碗状部，从那里被吸入文丘里管。阻气门调节进入 carb 的空气量，而节气门控制进入发动机的空气-燃料混合物的量。化油器似乎是由树脂 3D 打印体和歧管制成的，带有丙烯酸盖和 PLA 节流阀和阻风门。它安装在一个单缸发动机上。

高速镜头是令人难以置信的，清楚地显示了化油器的运作，使它非常容易理解。如果你感兴趣的话，他还上传了第二个视频，里面有将近 80 分钟的详细镜头。

[每天都变得更聪明]的传染性好奇心已经导致了许多迷人的项目，包括[超音速棒球佳能](https://hackaday.com/2020/09/11/making-baseballs-go-supersonic/)和[倒向自行车](https://hackaday.com/2015/04/27/try-not-to-fall-off-the-backwards-brain-bike/)。
