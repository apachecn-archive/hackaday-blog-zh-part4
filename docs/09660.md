# Mega Man Hack 放弃冲锋投篮，增加经典风格

> 原文:[https://hack aday . com/2021/08/04/mega-man-hack-drops-charge-shot-adds-classic-style/](https://hackaday.com/2021/08/04/mega-man-hack-drops-charge-shot-adds-classic-style/)

当谈到六个原始的超级男人游戏时，前三个和最后一个有明显的分界线。Mega Man 4 引入了充电射击机制，允许玩家按住发射按钮，以便从 Mega Man 的臂炮中启动单次爆炸。由[Peter] 制作的《超级男人 4:免费》ROM hack 被恰当地命名为[，它试图通过完全去除充电射击机制来实现与《超级男人》游戏前三部曲的衔接。为了补偿这个变化，敌人的生命值也被调整了，这样敌人就不会像子弹一样软了。](https://www.romhacking.net/hacks/5764/)

《百万富翁 4:免费下载》以 IPS 补丁文件的形式提供。有一些免费的实用程序，如 Floating IPS，可以将补丁文件应用到 NES 盒式磁盘的干净转储中。这个 ROM hack 可以通过 flashcart 设备在原始的任天堂娱乐系统硬件上播放，也可以通过任何常见的 NES 仿真器如 FCEUX 或 Nestopia 播放。

《超级男人 4》最令人讨厌的部分之一(减去难度)是不断的冲锋声淹没了精彩的配乐。有了像[Peter]这样的补丁，这不再是一个持续的问题，玩家可以让他们的拇指休息一下，不需要在整个跑步过程中持续按住火力。抛开所有受欢迎的变化不谈，这仍然不会改变日本的游戏电视广告比美国的平面广告更酷的事实。
