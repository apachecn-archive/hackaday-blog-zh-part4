# 观看这个乐高缩放雕刻巧克力信息

> 原文:[https://hack aday . com/2019/03/07/watch-this-Lego-graphic-carve-chocolate-messages/](https://hackaday.com/2019/03/07/watch-this-lego-pantograph-carve-chocolate-messages/)

[Matthias Wandel]以他非常有趣的木工项目而闻名，所以你可能会原谅你没有想到这个由乐高制成的可爱的[巧克力雕刻受电弓。他用它在一块巧克力上刻下了令人愉快的情人节信息，但并没有就此止步。他更加努力地将巧克力小心地切成心形，然后用热风枪快速击打，将粗糙的边缘去掉，最终得到脆而光滑的巧克力。](http://woodgears.ca/pantograph/lego.html)

切割端是一个插在乐高积木内部的小刀片，但这是整个组装中唯一的非乐高零件。获得良好雕刻的关键是在雕刻前冷却巧克力，你可以在下面的视频中看到整个过程。
