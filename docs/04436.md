# 复活顽固的 SE/30

> 原文:[https://hack aday . com/2019/10/08/resursing-a-recurrent-se-30/](https://hackaday.com/2019/10/08/resurrecting-a-recalcitrant-se-30/)

麦金塔 SE/30 被认为是紧凑型 Mac 系列的最佳选择。它配备了强大的 68030 处理器，能够使用高达 128 MB 的内存，在紧凑的外形中带来了严重的问题。这些机器现在已经使用了 30 多年，它们经常磨损得很厉害。[[这不算]让这个特殊的例子运行起来，对他来说是一项艰巨的工作。](https://www.youtube.com/watch?v=FVZJGeNpbtY)(视频嵌入下方。)

随着计算机在启动时显示著名的【SimasiMac 屏幕，它在打开时很遗憾地不起作用。[这不计算]进行了所有常见的尝试来解决这个问题——清洗电路板，重新加盖，检查潜在的断裂痕迹——都无济于事。在经历了许多惊愕之后，修复并不那么困难——一组新的 RAM 帮助治愈了 Mac 的问题。

随着 Mac 现在显示出一些生命的迹象，还有更多的事情要做。软驱拒绝启动，弹出磁盘，无法读取任何内容。头部清洗被证明是有帮助的，但还不够。只有当磁头电机的蜗轮重新润滑，使其能够正确搜索时，驱动器才能成功启动。硬盘被证明抵抗任何试图让它工作，所以被替换为 SCSI2SD。

随着一系列维修工作的完成，SE/30 再次投入使用。稍加努力，机箱和键盘也变得很不错。[这还不算]现在有一台空前经典的苹果电脑，状态极佳。

我们在过去已经看到了一些伟大的修复——[这个装满泥土的准将 64 是一个特别引人注目的故事](https://hackaday.com/2018/10/03/ants-dirt-rain-and-the-commodore-64-that-wouldnt-quit/)。休息后的视频。
