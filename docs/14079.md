# 2022 年 Hackaday 奖:MasterPi 是一个拥有漂亮轮子的机器人

> 原文：<https://hackaday.com/2022/07/01/hackaday-prize-2022-masterpi-is-a-capable-robot-with-fancy-wheels/>

当制造移动机器人时，机动性往往比速度更重要。MasterPi 机器人很好地证明了这一点，它使用别致的轮子来帮助它向任何需要的方向滑行。

四个 DC 齿轮马达安装在金属底盘上，每个驱动一个麦克纳姆轮。这些是特殊的轮子，轮子周围装有一定角度的滚轮，允许机器人向各个方向移动，并根据每个轮子的驱动方式以各种方式旋转。

在这个高机动性底盘的顶部放置了一个 5 自由度的机械臂。该机器人还拥有一个超声波传感器，用于躲避物体，以及一个摄像头，用于循线任务。相机还允许机器人捡起石块并识别它们的颜色，然后将它们分类放入盒子中。这一切都是由一个 Raspberry Pi 驱动的，它运行一堆 Python 代码来实现一切。

这是一个很好的项目，展示了机器人在一些智能设计选择和现代计算硬件的帮助下可以有多强的能力。[我们以前也见过其他一些智能块分类器](https://hackaday.com/2011/04/08/nxt-machine-sorts-lego-blocks-automatically/)。
