# Perler Printer 推动了像素艺术的发展，这是任何精灵艺术家都做不到的

> 原文：<https://hackaday.com/2019/10/23/perler-printer-pushes-pixel-art-like-no-sprite-artist-could/>

在一个休闲的周末放下 Perler 珠子制作你最喜欢的经典任天堂角色，这种做法有一种奇怪的抚慰。但是说真的，当我们可以利用机器为我们做重物时，为什么要像穴居人一样使用我们肮脏的手呢？这正是[knezuld11]所做的！他们建造了一台 64 色珠子精灵打印机，包括自动烹饪功能，用于融合结果。(视频，嵌在下面。)

从顶部开始，多达 64 种独特的珠子颜色被藏在顶部的墨盒中。大型搅拌器负责将这些珠子传送到试管中，以供低级珠子选择器使用。在这一级别，珠子颜色被序列化到一个管中，该管被馈送到输出“喷嘴”指导珠子图案的整个过程由 Python 脚本驱动，该脚本将图像作为输入，并将它们的颜色近似为可用的珠子调色板。当珠子“打印”完成后，机器将它的加热床向上倾斜，将珠子底部煮熟，以一种[knezuld11]称实际上比典型熨烫方法更有效的方式将它们融合在一起。

我们只是喜欢这个系统的完整功能。虽然[knezuld11]提到 Bead Sprite 打印机是试图打破世界纪录，但我们可以想象这台机器还有许多其他方式可以导致一些异想天开的约定。非常坦率地说，我们希望在艺术家巷看到这台机器制作按需艺术。

如果你在看这个视频的时候，因为极度兴奋而弄洒了你所有的珠子，不要担心！这台来自我们过去的[自动珠粒分拣机](https://hackaday.com/2018/10/07/robot-sorts-beads-by-color/)正是能帮你解决问题的东西。
