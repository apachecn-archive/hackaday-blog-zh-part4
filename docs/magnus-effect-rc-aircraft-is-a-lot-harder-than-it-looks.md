# 马格努斯效应遥控飞机比看起来难多了

> 原文：<https://hackaday.com/2021/08/05/magnus-effect-rc-aircraft-is-a-lot-harder-than-it-looks/>

在过去的一个世纪里，传统的翼型机翼在让飞行器升空方面脱颖而出，但也有一些其他有趣的设计来来去去。其中之一是马格努斯效应飞机，它利用旋转圆柱体产生的升力。来自[Project Air] [的[James Whomsley]决定建造一个作为副业项目](https://www.youtube.com/watch?v=UG2O_GK7-R8)，但它最终比他最初怀疑的要有挑战性得多。(视频，嵌在下面。)

马格纳斯效应在几年前获得了一点病毒式的名声，当时[【多可笑】用一个下旋球把一个篮球扔下了大坝墙](https://www.youtube.com/watch?v=H9SF2YIKRY8&t=0s)。[James] T 形马格纳斯效应飞机在顶部有一对旋转气缸来产生升力，由无刷电机通过皮带驱动。第二个带螺旋桨的无刷电机位于中心碳纤维管上，提供向前的推力，方向舵提供偏航控制。电池固定在浴缸底部以保持稳定。

第一次飞行看起来非常有希望，但[詹姆斯]很快遇到了一系列与重心、动力、俯仰控制和阻力有关的问题。经过建造-坠毁-重建循环的迭代，他最终得到了更大的发动机和方向舵，更短的“翅膀”，和更高的推力发动机位置。这导致了飞行器仍然只能勉强控制，但在空中停留了相当长一段时间。因为他们的意图是永远不要把它变成一个长期项目，詹姆斯称之为成功，避免了更多的牦牛毛，并继续他的飞艇和 T2 火箭飞机的工作。

如果你有兴趣自己造一个，他会把他所有的实验发现写在一个简短的报告里。为了获得更多的灵感，看看我们报道过的另一架马格努斯效应飞机，它的机翼使用了[肯德基桶](https://hackaday.com/2017/04/14/kfc-winged-aircraft-actually-flys/)。
