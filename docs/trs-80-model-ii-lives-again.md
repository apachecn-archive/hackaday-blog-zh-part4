# TRS-80 II 型复活了

> 原文：<https://hackaday.com/2022/09/17/trs-80-model-ii-lives-again/>

许多人都有一台 TRS-80 I 型收音机，这是一台内置在键盘上的“家用电脑”,需要外接显示器或电视机。后来，Radio Shack 将计算机升级到型号 III，这是一种流行的“多合一”选择，带有显示器，甚至还有存储软盘的空间。但是第二种型号并不常见。原因？它的目标是商业和相应的定价。[Adrian]有一辆 [Model II，它的状况很糟糕，但一直在让它恢复生机](https://www.youtube.com/watch?v=Qq9oRcgFnX0)。你可以在下面的视频中看到他是如何做到的。

令 Radio Shack 惊讶的是，II 型类似于企业经常使用的老式“Trash 80”。但它有更复杂的功能，包括一个 4MHz 的 CPU——在那个时代是极快的速度。它还有一个 80×25 的文本显示器和一个 500K 的 8 英寸软驱。还有标准的串行和打印机端口。

有几个有趣的特点。软驱的主轴靠交流电源运转，如果电脑开着，磁盘就会旋转。此外，还有银行切换，所以你可以超过 64K，而且你不必与视频显示器共享你的运行内存。理论上，这台机器可以超过 64K，因为一半的内存是可转换的。实际上，早期的型号没有足够的扩展空间来处理超过 64K 的物理数据。

机器必须有一张软盘才能启动，在之前的视频中，计算机无法正确启动。鉴于它多年来保存不佳，这并不令人惊讶。你将看到机器被拆卸，原理图检查，集成电路测试。然而，最终，电脑似乎会自我修复。如果你对这种事情有经验，你可以猜猜接下来会发生什么。当然，它又失败了。

理论上，测试良好的部分通过加热暂时愈合，[阿德里安]更换了芯片，但需要一点工作才能发现这是一个坏的磁盘控制器。一旦他知道了坏的区域，找到坏的芯片就相对容易了。

如果你对 model I 有美好的回忆，为什么不[造一辆](https://hackaday.com/2020/01/19/trs-80-clone-uses-modern-parts/)？我们喜欢看到这些旧机器被修复而不是被掏空，尽管有时几乎没有选择。
