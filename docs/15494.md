# 我需要一个黑客马拉松

> 原文：<https://hackaday.com/2022/11/27/i-need-a-hackation/>

最近,“居家度假”成为了一种受欢迎的选择，取代了去远方的旅行:你可以休息一段时间，享受当地的环境，而不必在护照上盖章。但我不需要去博物馆或游乐园，更不用说去了解陌生的事物了。我脑子里有一个项目在燃烧，我需要的是在地下室工作几天来取得一些进展。我需要的是一个黑客组织。

有些项目会让你下班后分心，但这个项目很难，需要我全力以赴。这不是一个啤酒和一个项目的项目。所以这一周都不在。这样就剩下周末了，但那是和家人一起出去玩的黄金时间。当然，当我儿子和朋友出去玩的时候，我可以在周六或周日中午思考/工作的几个小时内找到工作，但是花一整天或更多的时间去解决一个棘手的问题是有意义的。

当然，我充分意识到我可能不会在一天之内完成它，我会希望有另一天，或者又一天。就这样吧。夏天你在海边不也是这样吗？黑客行为难道不应该和去迪斯尼乐园一样重要吗？

你乘过出租车吗？因为那是我需要的。请告诉我有更好的名字。
