# 斯珀从现实生活中创造虚拟现实

> 原文：<https://hackaday.com/2020/03/02/esper-makes-virtual-reality-from-live-reality/>

《刀锋战士》中有一个场景，迪卡将一张照片放入一台神奇的机器中，这台机器可以让他无限制地放大和增强，甚至可以看到周围的障碍物。在今天的气候下，这开始看起来更合理，因为到处都有摄像头。贾斯珀·范·洛宁在[斯珀探索了这一概念，这是一个他在韩国首尔的艺术家驻地创作的技术艺术装置](https://jaspervanloenen.com/#)。

《斯珀》由两部分组成，通过在虚拟现实中展示真实的现实，彻底颠覆了虚拟现实。它覆盖了两个相邻的房间，一个用于记录现实，另一个用于通过耳机进行实时虚拟观看。第一个配备了 60 个安装在定制支架上的 ESP32 摄像机，所有摄像机都从不同的栖木和天花板上指向不同的方向。[Jasper]使用了一个基于 openFrameworks 的 Android 应用程序来绘制摄像头在 3D 空间中的位置。隔壁房间是如此的空旷，甚至没有 FOMO。你不会想错过这一集的，休息后再来看看吧。

在反乌托邦到来之前，再造科幻道具都是乐趣和游戏。话又说回来，[我们都可以轻松访问大约 70，000 个不安全的监控摄像头](https://hackaday.com/2020/02/18/insecure-surveillance-cameras-provide-dystopian-peep-show/)这一事实是一个很好的开始。
