# 一种简单的螺旋宠物喂食器

> 原文：<https://hackaday.com/2020/01/17/a-simple-auger-pet-feeder/>

宠物喂食器是一个受欢迎的创客项目。人们可以推测，这显示了创造者们普遍的巨大自信，他们愿意相信自己的工作可以让他们的动物同伴一次存活很多天。[达伦·塔巴德]就是这样一位制造者，他组装了这个简单的螺旋钻。

该项目包括一个干燥的宠物食品料斗，其中插入一个螺旋钻。这两个部分都是 3D 打印的，对于普通制造商来说，很容易在家里生产。建造是专门围绕[达伦]必须手动操作的部件设计的，即一个 28BYJ-48 步进电机，它负责转动螺旋钻。运行这个节目的是一个 Arduino，它可以用任何合适的定时代码来运行，这是喂养特定宠物所必需的。还有一个由[szuchid]创造的混合版本，它在顶部添加了一个更大的食物储存盘，用于更长时间的无人值守操作。

这是一个基本的构建，但对于大多数制造商来说，只需要一些废弃的组件和一卷灯丝就可以轻松实现。当然，如果你的宠物喜欢湿食物，你可能需要一个不同的设计。休息后的视频。
