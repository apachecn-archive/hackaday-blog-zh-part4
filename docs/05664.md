# 飞轮储存能量，最终为飞机提供动力

> 原文：<https://hackaday.com/2020/02/13/flywheel-stores-energy-to-power-an-airplane-eventually/>

问题:飞轮能储存足够的能量来驱动飞机吗？回答:是的，它可以，对于“飞轮”和“飞机”的某些值

我们能想到的唯一一个试图建造飞轮动力飞机的人是汤姆·斯坦顿。他是一个非常棒的想法，这些想法经常会有回报，比如[的协同效应气垫船](https://hackaday.com/2019/10/17/coanda-effect-makes-a-better-hovercraft-than-a-quadcopter/)，以及[从未离开地面很远](https://hackaday.com/2019/01/24/reaction-wheels-almost-control-this-unusual-drone/)，或者[突然再次遇到它](https://hackaday.com/2019/06/08/unconventional-drone-uses-gas-thrusters-for-control/)。对于下面的大部分视频，似乎他的飞轮动力飞机注定要牢牢地留在最后一类，事实上，一个巨大的飞轮飞行的想法似乎违反直觉。但是[汤姆]提醒我们，由于飞轮储存的动能随着角速度的平方增加，所以它转动的速度比它的质量更重要。复合碳纤维和铝飞轮通过 3D 打印的锥齿轮与最小飞机的螺旋桨相连，并通过外部 BLDC 电机旋转。

可悲的是，不管重量减了多少，飞机都没飞多远。但是[汤姆]通过把螺旋桨变成飞轮，从失败的虎口中夺取了胜利——他打印了一个连接螺旋桨叶片的环，并设计了一个飞轮离合器，将其连接到马达上。飞轮推进器储存了足够的能量来完成一些体面的飞行，以及遭受一些令人满意的壮观的解体。

像往常一样，向[汤姆]脱帽致敬，因为他不羞于分享他的失败，这样我们都可以学习，并坚持让他的想法起飞。
