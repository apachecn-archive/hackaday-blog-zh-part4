# 借助喷墨头进行全彩色 3D 打印

> 原文:[https://hackaday . com/2021/12/04/借助喷墨头的全彩 3d 打印/](https://hackaday.com/2021/12/04/full-color-3d-printing-with-the-help-of-an-inkjet-head/)

3D 打印是一个受欢迎的过程，尽管其中一个难题是零件通常只用一种单色生产。[Aad]一直在对他的 Prusa i3 打印机进行简单的改造，然而，这使得全彩色部件的生产成为可能。

![](../Images/f3c19cae82402676ec8ec150cd757cd2.png)

Note the dual vertical assemblies – one carrying the extruder, the other carrying an inkjet print head.

破解很简单，只需在打印机上添加第二个垂直框架。而不是安装第二个挤出机，但是，有一个喷墨打印机头提供 CMYK 水基油墨。在主挤出机放下每一层透明 PLA 塑料后，印刷品被移动到喷墨头下，喷墨头在印刷下一层之前放下彩色油墨，[如本印刷视频所示。](https://www.youtube.com/watch?v=VZEYCzkyc7w)

[Aad]注意到目前的结果并不完美。墨水似乎会在层与层之间渗出，在放下后与塑料混合在一起。然而，使用不同墨水和细丝的进一步测试正在进行中，旨在提高结果的质量和对比度。
