# Janksy 机器人一次画一个点

> 原文：<https://hackaday.com/2021/11/02/janksy-robot-paints-murals-one-dot-at-a-time/>

[这里制造的东西]有一个新的商店，有一面巨大的空白墙。空白的白墙是不行的，所以他没有自己粉刷墙壁，而是设计了一个机器人替他粉刷。(视频，嵌在下面。)

结果是 Janksy。由金属、木材和 3D 打印零件制成的巨大机器。Janksy 是一个巧妙的设计，因为它有两套 X 轴和 Y 轴。一个大型的缓慢移动的轨道和电缆系统将机器人大致定位在墙壁的正确区域。从那里，一个更小，但更快，更精确的运动系统做最后的动作。

Janksy 的“业务端”当然是喷漆工；在这种情况下是港口货运模型。选择的媒介是丙烯颜料，因为 Janksy 要画几天，他不想用传统颜料的挥发性溶剂来熏自己。

Janksy 主要是喷点颜料。凑近了你只会看到点，但退后一点，一个完整的图像就成形了。这是一种叫做[点彩法](https://en.wikipedia.org/wiki/Pointillism)的技术，这种技术将 Janksy 置于乔治·修拉和
文森特·梵高等艺术家的行列中。

虽然人类艺术家混合颜色以产生他们想要的色调，但机器人无法轻易做到这一点。[这里制作的东西]花了相当多的时间解释基本的颜色理论，以及青色、品红色和黄色的点如何在眼睛中结合产生颜色——就像显示器使用红色、绿色和蓝色光的像素一样。

做完这些工作后，你可能会想知道[这里制造的东西]会想在他的墙上挂什么。好吧，我们只能说他爱他的妻子，尽管他在这里的恶作剧经常引起愤怒的目光。休息后观看完整的故事视频。
