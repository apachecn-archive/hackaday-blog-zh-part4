# 由怪异的捐赠设备制成的 DIY 空调

> 原文：<https://hackaday.com/2019/04/07/diy-air-conditioner-built-from-weird-donor-appliance/>

在世界的一些地方，没有空调的生活几乎是不可想象的。但是在气候较为温和的地方，这并不罕见。[乔希的]公寓没有中央空调——那种连接到强制通风加热/冷却系统的空调。不过，它确实有空调水回路，所以他决定改装几台实验性空调。

他不是完全从零开始。他制造 AC 的两次尝试都来自捐赠的零件。[成功的那家是从热水器](https://www.youtube.com/watch?v=Q5M-he35mTk)起家的。[的第一次尝试](https://www.youtube.com/watch?v=XOlNXSCh86A)并不太成功，它使用了冰箱压缩机和底板加热器的蒸发器。通过热交换器的流量控制被证明是非常棘手的，所以[Josh]声称他大部分在入口处得到了冰，而通过蒸发器得到了最小的冷却。

更成功的一个工作得更好，但仍然有一个蒸发器冻结的问题，他试图解决。他正在寻找如何使它更好地工作的建议。尽管我们喜欢一个好的住处，我们的建议是搬到一个不同的公寓大楼。

我们见过其他自制的冷却器，但它们更像是[沼泽冷却器](https://hackaday.com/2014/08/08/a-do-it-yourself-air-conditioner-with-evaporative-cooling-5-gallon-bucket/)。如果你只是需要冷却你的桌子，你可能只需要用金属罐装一些[冰。](https://hackaday.com/2014/03/02/this-desktop-air-conditioner-is-really-cool/)
