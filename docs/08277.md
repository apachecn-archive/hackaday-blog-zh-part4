# 阐明捕蝇草的内部工作原理

> 原文:[https://hack aday . com/2020/11/10/illuminating-the-inner-work-of-a-venus-fly trap/](https://hackaday.com/2020/11/10/illuminating-the-inner-workings-of-a-venus-flytrap/)

作为一种食肉植物，捕蝇草一直是一个迷人的研究课题。他们的许多谜团之一是他们如何区分昆虫的访问和营养较少的刺激物，如被风吹动的鹅卵石。现在，科学家们离破译潜在机制更近了一步，这得益于一种实时可视化钙变化的新能力。

长期以来，钙一直被怀疑在捕蝇草的关闭/不关闭决策过程中发挥重要作用，但科学家们之前无法验证他们的假设。钙的标准化学测试需要将植物切开，这只会产生静态的快照。软件类比将会因为内存转储而终止进程，但是无法在运行时调试进程。有诱人的迹象表明，生物钙基模拟计算机正在工作，但大自然没有理由在其上进化出 JTAG 测试点。

由于缺乏在线调试头，科学家们转向了下一个最好的办法:增加诊断指示灯。但是，代替闪烁的发光二极管，基因被加入以产生一种在钙存在下发光的蛋白质。一旦成功，他们可以与工程植物一起工作，并获得视觉反馈。立即看到钙水平的变化和传播，以响应不同时期的各种刺激。证实了陷阱只有在刺激模式将钙水平推至阈值以上时才会突然关闭。

有了这些发光的蛋白质，研究人员发现钙可以解释一些行为，但不是全部。还有其他东西，被怀疑是一个快速的电网络，可以感知猎物的运动并触发钙的释放。这将是需要深入研究的东西，但至少我们有更多的经验[处理电脉冲](https://hackaday.com/2018/12/12/cyborg-or-leafy-sensor-array/)和[而不仅仅是植物](https://hackaday.com/2019/07/24/brain-computer-interfaces-separating-fact-from-fiction-on-musks-brain-implant-claims/)。
