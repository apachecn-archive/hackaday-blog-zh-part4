# 压电陀螺显示了它们是如何在白天回滚的

> 原文:[https://hack aday . com/2019/02/07/压电陀螺展示他们如何在一天内滚回/](https://hackaday.com/2019/02/07/piezoelectric-gyro-shows-how-they-rolled-back-in-the-day/)

毫无疑问，微机电系统(MEMS)技术给世界带来了奇迹。使用 MEMS 芯片，您的手机可以检测到最轻微的移动，将其变成一个敏感的传感器平台，几乎可以预测您接下来要做什么。实际上，想想就有点毛骨悚然。

但是在纳米级 MEMS 惯性感测出现之前，许多产品需要知道它们的起伏，许多人转向诸如[这种振动压电陀螺仪](http://www.kerrywong.com/2019/02/03/teardown-of-a-piezoelectric-vibrating-gyroscope/)之类的产品，这是[Kerry Wong]在一台旧摄像机中发现的。下面的视频显示了传感器的拆卸，按 MEMS 标准来看是巨大的，但仍然是微工程的奇迹。该设备被归类为科里奥利振动陀螺仪(CVG)，顾名思义，它使用科里奥利效应来检测旋转。在这个装置中，[Kerry]发现一个狭长的压电元件横跨传感器的长轴，悬挂在似乎是四个柔性臂上。[Kerry]通电时探测了传感器的内部，在压电元件上发现了 22 kHz 的信号；这使杆在一个平面内振动，因此当它旋转时，它在支撑臂上施加一个可以被检测到的力。事实上，[凯瑞]将传感器的输出连接到一个奇妙的老式 VOM 上，它的指针随着传感器的轻微移动而摆动。

可悲的是，MEMS 使这种传感器过时了，但我们欣赏引擎盖下的外观。事实上，MEMS 芯片也在使用同样的原理来检测运动，只是规模小得多。想要了解 MEMS 基础知识吗？[【Al】你报道过](https://hackaday.com/2017/06/26/mems-the-biggest-word-in-small/)吗？
