# 用废弃零件 DIY 神经分析仪

> 原文：<https://hackaday.com/2020/05/15/diy-neuralyzer-from-scrap-parts/>

在 Hackaday，角色扮演和道具制作是我们最关心的事情。这就是为什么每当我们看到科幻技术被带到生活中，我们都会忍不住密切关注。[进入【如何制作】DIY Neuralyzer，来自黑衣人系列](https://youtu.be/T-ZCe4GM-Vw)。不幸的是，这个[不会像现实生活中的神经分析仪](https://hackaday.com/2014/10/13/uc-davis-researchers-use-light-to-erase-memories-in-genetically-altered-mice)那样抹去你的记忆，但它会成为你下一次角色扮演活动的一个很酷的道具。

让这个项目值得分享的是它使用了非常简单的家庭工具和一些废金属，一些 PVC，一个 LED，一个开关，也许还有一些其他的东西。该设计的基础由两块中空的杆状废金属和一根弹簧组成，该弹簧使整个装置机械化。

这段视频是几个月前的了。最近在 Reddit 上发了一个帖子才把它发到我们的 feed 上，但我们很高兴看到了它。

很棒的项目【如何制作】！我们可以再建议几个 led 吗？
