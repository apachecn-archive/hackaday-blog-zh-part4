# K40 激光器的弹簧加载底座可用作自动对焦

> 原文：<https://hackaday.com/2018/12/02/spring-loaded-bed-for-k40-laser-acts-as-an-auto-focus/>

激光雕刻和切割与用放大镜聚焦太阳光线有一些共同之处:良好的聚焦对结果至关重要。如果使用不同厚度的材料，每次材料变化时都需要重新设置焦距，手动对焦很快就变成了一件苦差事。[斯科奇作品]有一个聪明的解决方案来避免不断重新聚焦，不涉及任何类型的传感器或电机。结果是[一个自动调节床，补偿材料高度变化](http://www.scorchworks.com/Blog/spring-loaded-k40-laser-platform/)，确保材料的顶部表面始终与激光头保持固定距离。

[斯科奇工厂]的做法是用角铝和一些五金件制作两个弹簧夹。当一张材料被放入机器中时，边缘会被塞进铝“唇”的下面，同时被从下面向上推。通过固定角铝顶层的高度，无论材料的厚度如何，任何板材最终都与激光头保持相同的距离。

[Scorch Works]在下面嵌入的视频中展示了组装过程，以及适应不同材料和特殊情况的几种不同方式，请务必查看。

对于 K40 激光来说，这是一个非常聪明的模块，并且相对来说需要很少的硬件。它甚至可以用从[扩展 K40 到一个更大的切割区域](https://hackaday.com/2018/07/19/expanding-the-k40-laser-cutter-with-aluminum-extrusion/)的剩余物来完成。
