# 自制的 CAT 扫描仪不应该扫描猫

> 原文：<https://hackaday.com/2022/08/07/homemade-cat-scan-shouldnt-scan-cats/>

[烟火师]考虑买一台 CAT 扫描仪，发现它们要花上百万美元。因此，他决定用一个废弃的 X 射线管和其他一些杂七杂八的零件，花大约 200 美元建造一个。闪烁探测器提供图像，以便用照相手机拍摄。控制？一个 Arduino，还有什么？你可以观看下面的视频，但是由于大量的 NSFW 语言，如果你不想让任何人感到震惊，你可能需要带上耳机。

当然，在使用高能 X 射线时，你需要小心。为了远离 X 射线源，[烟火技术]使用了 Roku 遥控器和红外传感器来远程控制设备。电子学很简单。你只需要旋转一个转盘，并在点亮 X 射线管的同时触发相机。

真正的问题是执行断层扫描(CAT 中的 T)将平面图片转换为 3D 表示。当然，有软件可以做到这一点，这就是他所使用的。老实说，这让我们想起了使用手机和转盘的廉价 3D 扫描仪。如果你选择其中一个，加上闪烁体和 X 射线管，它应该也能工作。

如果您决定复制这一点，请确保您了解 X 射线的安全性。你不应该让任何活着的东西受到光束照射——不管是猫还是别的什么。“拼凑”的构建质量让我们想起了我们见过的另一台自制 x 光机。令人惊讶的是，这并不是我们见到的第一台[自制 CAT 扫描仪](https://hackaday.com/2019/02/20/diy-x-ray-machine-becomes-ct-scanner/)。
