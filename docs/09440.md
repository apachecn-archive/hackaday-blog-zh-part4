# 喷水切割的精密糕点

> 原文:[https://hack aday . com/2021/03/06/water jet-cut-precision-pastry/](https://hackaday.com/2021/03/06/waterjet-cut-precision-pastry/)

我们的生活中需要更多高端、几何的糕点。这种洞察力来自一个相当古老的视频，嵌入在下面，[展示了一个极其聪明的 2D CNC 机制，它在蛋糕盘上切割出形状](https://www.youtube.com/watch?v=EOOtx4T7OcA&feature=youtu.be)，打开了一个任意蛋糕拓扑的宇宙。

对我们来说，这台机器最酷的地方是驱动机构。一个巨大的圆形齿轮被夹在两条齿形带之间。当两条带子一起移动时，整个物体会平移，但当它们向相反方向移动时，物体会转动。它似乎漂浮在一个塑料平台上，因为这种设计允许水射流切割头保持完全固定，只需要在下面开一个小孔，这无疑简化了高压水的输送和收集。机器的外围是由垂直板条组成的蛋糕盘，就像激光或等离子切割台上的一样，它们滑入定位销，让水通过。

这台机器的运动学是一场梦，或者也许是一场噩梦。为了切割一条直线，它做了一个摆线形状的平移和旋转的舞蹈，你只需要看到它在运动。由于这种复杂的路径，蛋糕进给速度沿途变化，所以这种机器不会对所有应用都是完美的，并且依赖于薄切口。我们不禁会想，蛋糕在这个过程中会变得多么令人眩晕。

[![](../Images/b505e658727207a844bd57055c91ead4.png)](https://hackaday.com/wp-content/uploads/2021/03/watercut-xs-0uteql-ip6imp4-shot0001_thumbnail.png) 的确，同一家公司出了一款[相对行人的双臂运动切割机](https://www.youtube.com/watch?v=0uTEQL-Ip6I)(另一个视频！)提出了不同的运动学问题。它本质上是一个双臂绘图仪，下面有一个移动的桌子，有助于增加工作区域。细节很少，但看起来他们尽量减少了移动桌子的运动，用僵硬的手臂做高频的小事情。据推测，有人将前一台机器的速度调到 11，并旋转蛋糕进入房间，导致他们重新思考整个移动蛋糕的设计。

当然，水切割糕点并不局限于奇特的数控机械装置。这个(第三！)视频演示了一个简单的笛卡尔 XY 机器人也能完成这项工作。

如果你仔细想想，使用高压纯水切割食品在很多层面上都是一种胜利。我们只是错过了舔刀的机会。感谢 Adam G DeMuri 的精彩评论，带领我们进入一个含水食物的新世界。
