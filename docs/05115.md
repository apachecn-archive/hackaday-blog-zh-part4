# 燃烧的丙烷完美地展示了特斯拉阀的工作原理

> 原文:[https://hack aday . com/2019/12/13/burning-丙烷-beautifully-illustrated-a-Tesla-valve-works/](https://hackaday.com/2019/12/13/burning-propane-beautifully-illustrates-how-a-tesla-valve-works/)

当你听到“特斯拉”这个名字的时候，很有可能你会立刻想到这家公司，这家公司正试图重新发明汽车，以及使之成为可能的每一个行业。虽然我们为他们的努力喝彩，但他们选择使用一位塞尔维亚学者的姓氏作为他们的企业品牌是一种耻辱，因为老(尼古拉)在他的时代做了许多有趣的事情，他理应被人们记住。

拿特斯拉阀来说。从本质上说，特斯拉阀是一个流体二极管，它使用曲折的路径来允许一个方向的流动，但严重限制另一个方向的流动。然而，理解它如何工作并不一定是直观的，这就是为什么[NightHawkInLight]选择用爆炸的丙烷来演示特斯拉阀原理。这对他来说不是新的领域；我们过去报道过[的丙烷动力步枪](https://hackaday.com/2017/10/12/propane-powered-plasma-rifle/)。

在这些实验中，旋转的蓝色和绿色火焰前锋使得燃烧的丙烷成为展示特斯拉阀如何工作的完美工作流体。下面的视频讲述了这个故事，高速镜头显示了限制逆流的湍流。令人惊讶的发现是，在向前的方向上，燃烧的气体实际上似乎在沿着阀门向下移动时加速了；高超音速特斯拉等离子炮，有人吗？

我们以前见过特斯拉阀门，包括由“Shrinky Dink”制成的。这在可视化使阀门工作的流动模式方面做得很好，但在充满有色水的微小通道和燃料空气混合物的爆炸分解之间存在巨大的表演差距。这有点冒险，标准的“不要在家里尝试这个”免责声明适用，但幸运的是[夜鹰之光]仍然有他的眉毛，所以他一定是在做正确的事情。
