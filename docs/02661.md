# 当您需要 Nixies 时，该显示器将为您提供一切便利

> 原文：<https://hackaday.com/2019/04/09/the-display-for-when-you-want-nixies-without-all-the-hassle/>

如果你想显示数字，就选择数字。没有比这更好的工作方式了，仅仅因为它们看起来太酷了。不幸的是，Nixies 需要高电压，控制它们有点奇怪，而且它们消耗大量的能量。这些事实给了我们几个谢妮备选方案，而[戴夫]在这里带来了另一个。这是一个光管谢妮，由丙烯酸棒制成。

用光线照射到一块丙烯酸树脂上来显示数字的想法可能和谢妮一样古老。60 年代有一些工具使用侧光塑料面板来显示数字，最近我们看到了激光切割版本，[谢利](https://hackaday.com/2017/01/14/smoothly-modernized-nixie-display/)。这个显示器仅仅是十张压克力板，上面刻有数字 0 到 9。用光穿过右边的亚克力板，那个数字就亮了。

你可以用丙烯酸做任何事情，它已经被用来做光管了，所以[戴夫]拿了一些丙烯酸棒，把它弯曲成几个数字的形状。只需一点点工作，他就能制作出自己的 FauxNixie，方法是将这些数字安装在一个精心改造过的灯泡上，灯泡由十个独立的发光二极管连接而成。结果是巨大的，巨大的，巨大的谢妮风格的数字，对于有眼光的 glowey 爱好者来说是完美的时钟。
