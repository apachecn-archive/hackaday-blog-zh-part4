# 总是有一个正方形备用

> 原文：<https://hackaday.com/2019/01/19/always-have-a-square-to-spare/>

人性的某些方面会在我们人生的某个时刻影响我们所有人。无论是在没有雨伞的情况下淋雨，还是在上班的路上爆胎，或者是升级一个不知何故破坏了整个安装的 Linux 包，有些经历确实是普遍的。其中之一是从纸卷上扯下几格卫生纸，结果用力过猛，整卷卫生纸都散开了。尽管如此，还是有可能[采用一点技术，这样我们就都不用再经历这种麻烦了](http://holdenart.com/home/polarity-gear/)。

[威廉·霍尔登]和[埃里克·斯特雷贝尔]决定用一种创新的轴承代替典型的卫生纸架来解决这个问题。嵌入机械装置中的是一组磁盘，它提供了比普通卷筒固定器更高的阻力。慢慢拉出正方形的纸是可能的，但就像非牛顿流体在施加更大的力时变成固体一样，当在卫生纸上执行更高速度的拖拉时，磁体将提供足够的阻力。这使得纸被撕开而不是展开整个纸卷，并且还允许使用者单手操作卫生纸。

这是一个很好的解决方案，可以解决我们都曾面临过的问题，但可能在我们经历了一分钟后就忘记了。而且，它还能托住你的手机，防止它掉进马桶里！如果你想看看他们的 Kickstarter，他们正试图筹集资金将产品推向市场。而且，如果你想进一步升级你的卫生纸机，当然也有一个[物联网设备。](https://hackaday.com/2017/11/27/iotp-the-internet-of-toilet-paper/)
