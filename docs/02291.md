# 简单的改装完全改变了这架钢琴的声音

> 原文：<https://hackaday.com/2019/03/04/simple-hack-completely-changes-the-sound-of-this-piano/>

我们偏爱这里的乐器黑客，主要是因为我们发现乐器是迷人的机器。很少有东西比钢琴更复杂，事实证明，也很少有东西如此容易掌握。尽管如此，我们不得不承认这个拉格泰姆钢琴黑客让我们大吃一惊。

我们总是认为，从一架钢琴中可以哄出丰富多样的音调，从老式西方沙龙钢琴的微弱声音到大型音乐会的丰富音调，主要是由于乐器的结构和演奏方式。显然并非如此，正如[测量工作室]通过在一架小型立式钢琴上安装“曼陀林导轨”所展示的那样。这台仪器曾经风光过，所以第一步是拆卸和清洗。增加了一个横跨整个弦板宽度的木栏杆，一个垂到琴锤高度的织物窗帘。窗帘被剪成与锤子等间距的流苏，用玉米淀粉标记锤子的位置是一个很好的技巧，每个流苏上都有金属夹。完成的曼陀林轨道可以使用新的脚踏板升高和降低，当锤子用金属夹而不是柔软的毡头敲击琴弦时，完全改变了音调。它让钢琴听起来有点像大键琴，或者前面提到的沙龙乐器，只要用脚一碰，它就会回到原来的音调。

我们提供的大多数钢琴黑客倾向于电子种类，所以很高兴看到纯粹的机械钢琴黑客有所改变。如果被黑掉的钢琴不能用作乐器，你可以随时[把它变成工作台](https://hackaday.com/2019/02/03/recycled-piano-becomes-upcycled-workbench/)。
