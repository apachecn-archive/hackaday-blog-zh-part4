# 制作气凝胶，不适合胆小的人

> 原文：<https://hackaday.com/2020/04/03/making-aerogel/>

气凝胶——空气和二氧化硅的混合物——是一种看起来像奇迹的材料。它几乎不存在，因为材料是 99%的空气。[NileRed]想要制作自己的作品，他在最近的一个视频中记录了他的作品，你可以在下面看到。

如果你决定复制他的结果，小心使用原硅酸四甲酯。他是这样说的:

> 最棒的是，当它进入你的眼睛时，它进入表面之下，颗粒太小而无法去除。因此，你可能会永久失明。

它还会弄乱你的肺，所以你可能需要一个通风罩来真正解决这个问题。它也不便宜。你需要的其他东西更容易处理:甲醇、蒸馏水和氨水。

这个过程包括开发硅胶，然后让它干燥。听起来很容易，但你会在视频中看到，这并不像听起来那么简单。[NileRed]使它变得更难，因为他想制造出形状漂亮的气凝胶，尽管它非常脆弱。模具的构造其实很巧妙，需要去一元店一趟。

干燥材料需要更多的甲醇来取代水。然后你可以去除甲醇，留下材料中的空气。这样做需要一氧化碳排出甲醇，然后迅速将液态一氧化碳转化为气体。在这个过程中，一个真空烹饪器找到了新的用途。

这不是一个快速的项目。该视频长达 43 分钟，除了在最后几分钟推广其他视频之外，我们想不出你在哪里删除了任何内容。

考虑到你可以花 40 美元左右买一个气凝胶圆盘或圆柱体，我们不确定我们会急于复制这个实验。另一方面，我们对这项工作印象非常深刻，看到困难问题的解决方案也很有趣。

还有其他可以形成气凝胶的材料，包括[陶瓷](https://hackaday.com/2019/02/27/ceramic-aerogel-meets-stretch-goals/)。[Ben Krasnow]过去也做过类似的视频，使用了[非常相似的流程](https://hackaday.com/2011/11/16/making-aerogel-at-home/)，但是他没有[NileRed]使用的聪明的模型。
