# 不管你怎么说，一套毛皮衣服是很费工夫的。

> 原文:[https://hack aday . com/2019/12/14/say-what-you-will-a-fur suit-a-lot-of-work/](https://hackaday.com/2019/12/14/say-what-you-will-a-fursuit-is-a-lot-of-work/)

[蟑螂博士]的建造日志显示的一件事是，制作一件[毛皮服并不是一件容易的事情。](https://hackaday.io/project/163655-my-first-fursuit-yes-i-am-a-furry)

Furries 来自于早期的美国漫画，并成长为当今互联网无法独善其身的亚文化。许多人在参与这种亚文化时会扮演自己毛茸茸的化身，其中一个获奖的成就是设计、委托或制作一套毛皮服装。[蟑螂博士]的体型说明了为什么有些套装能轻易卖到 1 万美元。这真的是一项爱的工作。这也带来了他在这个项目中的一个目标，发现更便宜的方法来建造这些服装，这样其他分享他爱好的人就有一个更经济的过程来加入。

我们对建造过程非常着迷。一个底座是由模拟头部周围的软泡沫制成的。在这个基础上，在他的化身马库斯开始成形之前，更多的泡沫被分层和雕刻。他的妻子发现测试过程特别幽默，但当他对安排和可移动的下巴感到满意时，他开始制作图案。

图案制作过程非常巧妙。他用胶带在泡沫基底上涂了一层，然后把它撕下来。然后很容易有策略地切割胶带，直到它变平。我们可以肯定地看到自己用这个技巧做任何事情，从金属片到复制塑料模型。

然后是相当多的困难缝合。我们以前从来没有想过这个问题，但是如果你试图模拟毛皮，必须注意毛皮的方向；进一步增加了难度。

无论您的观点在哪里，没有黑客可以拒绝详细的构建日志，如果足够重视，任何地方都可以学到一些技巧。
