# 60 千兆赫相控阵

> 原文：<https://hackaday.com/2020/01/31/a-60-ghz-phased-array/>

我们的朋友[亨特·斯科特]在过去的一次超级会议上做了一个关于相控阵天线的演讲。他提到他正在寻找合作者，用 SiBeam SB9210 芯片制作天线。这是一个专门用于 WirelessHD 的芯片，WirelessHD 是一个或多或少失败的视频流协议，它本质上是一个芯片上的整个 60 GHz 相控阵，具有发射和接收功能。15 美元，看起来很划算，而且[亨特]仍然想让这个设备工作。

不利的一面是，Lattice 收购了 SiBeam，并扼杀了这个芯片——考虑到 WirelessHD 从未真正起飞，这并不奇怪。然而，[亨特]说这种芯片存在于一些旧的智能电视和笔记本电脑中。如果你能在过剩的市场上找到这些设备的替代板，你就能买到芯片和支持电路。

情况有点棘手。[Hunter]有这些器件的数据表，但仍受保密协议的约束。他仍在努力消除那个障碍。但在此之前，他对类似的芯片有一些建议，这些芯片有公共数据表和控制器芯片，如果你在工作系统中嗅它的总线，可能很好地揭示了如何设置廉价的天线芯片。

还有一堆公开链接，应该能把事情说清楚。然而，我们希望 Lattice 允许 NDA 在数据表中过期。与此同时，你可以在下面的视频中看到开始这一切的 Supercon 谈话。

你可能对可视化相控阵感兴趣。这些在[雷达系统](https://hackaday.com/2015/04/07/build-a-phased-array-radar-in-your-garage-that-sees-through-walls/)中真的很常见。
