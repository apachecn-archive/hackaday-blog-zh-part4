# SCADA 安全黑客聊天

> 原文：<https://hackaday.com/2021/07/12/scada-security-hack-chat/>

加入我们太平洋时间 7 月 14 日星期三中午与 ireann Leverett 的 [SCADA 安全黑客聊天](https://hackaday.io/event/180476-scada-security-hack-chat)！

作为一个社会，在过去的一年半左右的时间里，我们已经学到了很多惨痛的教训。但我们面临的最大教训之一是我们基础设施的真正脆弱性。摇摇欲坠的建筑和桥梁及其悲惨的后果是一回事，但随着对食品和能源供应链的攻击，随着复杂性的增加，我们的系统显然是最脆弱的。

天哪，我们擅长制造复杂的系统。仅在美国，数百万英里的电缆和管道将这个国家从一个海岸连接到另一个海岸，其中大部分安装在偏远崎岖的地方。这样的远程系统需要监测和控制，这是监控和数据采集(SCADA)系统的工作。这些网络随着基础架构的发展而发展，通常是以某种特定的方式发展，鉴于其性质，它们可能成为威胁行为者的诱人目标。

寻找保护此类系统的方法是艾瑞·莱弗里特非常关心的事情。作为剑桥大学的高级风险研究员，他了解我们的基础设施面临的威胁，并努力寻找减轻这些威胁的方法。他的书 [*解决网络风险*](https://www.wiley.com/en-us/Solving+Cyber+Risk%3A+Protecting+Your+Company+and+Society-p-9781119490920) 从总体上为保护 IT 基础设施设计了一个框架。在这次黑客聊天中，ireann 将阐述 SCADA 系统的特殊需求，以及如何最好地保护这些网络。请带着您关于基础设施自动化、降低网络风险以及如何保护我们赖以生存的无尽管道和电线网络的问题来访。
