# USB-C 在它从未打算在的地方

> 原文：<https://hackaday.com/2020/07/24/usb-c-where-it-was-never-intended-to-be/>

USB-C 革命正在进行中，首先是你的新手机，然后是你的单板电脑，现在你的笔记本电脑很可能会配备熟悉的可逆圆角连接器。我们仍然处于需要随身携带微型 USB、专有笔记本电脑和 USB-C 电源的交叉时期，但只有 USB-C 的世界的承诺已经近在咫尺。对于[Purkkaviritys]来说，现在更接近了，因为[他改装了他的 Thinkpad T440s，用 USB 充电器](https://www.youtube.com/watch?v=_y_tpfEfKWc)代替了其专有的联想方形插头部分。(视频，嵌在下面。)

它的核心是一个 USB-PD 仿真器模块，完成与电源协商的所有艰苦工作，为笔记本电脑提供它需要的 DC 电压。但这并不那么简单，因为需要一个电阻来保证笔记本电脑有真正的电源。该模块封装在精心设计的周围，巧妙地占据了原始连接器腾出的空间，由于这款笔记本电脑的内部电源连接器位于一根短电缆上，因此非常容易安装到机箱中。如果你不知道这是国产升级，你会认为这台笔记本电脑出厂时配有 USB-C 电源插座也情有可原。

这里使用的 USB-C 模块是一个通用部件。我们之前在烙铁转换中见过[。](https://hackaday.com/2019/11/01/adding-usb-c-to-the-ts100/)
