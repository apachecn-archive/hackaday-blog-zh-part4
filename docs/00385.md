# LED 化吉他

> 原文:[https://hackaday.com/2018/09/01/led-ifying-a-guitar/](https://hackaday.com/2018/09/01/led-ifying-a-guitar/)

假设你有一把吉他，一把昂贵的吉他——三把这样的吉他中的一把。然后说这把吉他听起来很棒，但是缺少了什么。它需要一些东西，但不会破坏完成。在 Sparkfun 上，[Englandsaurus]被要求为一把三件套的吉他设计一个看起来非常酷的模型—[用 LED 灯条覆盖吉他主体](https://www.sparkfun.com/news/2754)在吉他上创造灯光图案。

为了不损坏或修改吉他[Englandsaurus]将琴身夹在两块有机玻璃板之间，通过 3D 打印的夹子连接在一起。夹子有双重用途——它们将有机玻璃片固定在吉他上，同时也作为一对光纤管的导管，这对光纤管环绕在琴身的边缘。为了让颜色沿着吉他的边缘一路传播而不中断光线，光纤电缆被偏移了。每一个夹子都有光线射入。一根电缆在两个夹子之间穿过，跳过其中一个夹子，第二根电缆在跳过的夹子之间穿过。这允许光线在吉他的身体周围流动。

这些 led 在全白时接近 500W，消耗了很多功率，然而，在全亮度时，它们过于明亮，所以[Englandsaurus]使用了一些 [WonderFlex](https://www.wonderflexworld.com/) ，一种可成型的漫射塑料片来覆盖它们。即使这样，发光二极管也不能以全亮度运行。然而，光缆由于其覆盖层，需要全亮度。

大约 1600 个发光二极管安装在这个模块中，吉他本身没有被修改。所有的东西都是可拆卸的，如果带子被拿掉，吉他会回到原来的样子。看一看 [Strumbot](https://hackaday.com/2018/01/31/strumbot-the-guitar-that-strums-itself/) ，另一个没有修改原始吉他的项目，或者一个非常酷的[废金属吉他](https://hackaday.com/2015/12/08/gitaar-van-schroot-the-scrap-metal-guitar/)。
