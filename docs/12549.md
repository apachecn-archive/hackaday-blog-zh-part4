# 关于麦克纳姆的一切

> 原文:[https://hackaday.com/2022/01/20/all-about-mecanum/](https://hackaday.com/2022/01/20/all-about-mecanum/)

如果你和机器人或其他轮式项目打过交道，你可能听说过麦克纳姆轮。这些看似神奇的轮子有能力向任何方向移动。如果你曾经见过，它是如何工作的就很明显了。它们看起来或多或少像普通的轮子，但它们也有与正常运动轴成 45 度离轴旋转的滚轮。这导致车轮的驱动力以 45 度角移动。然而，有很多细节不是一眼就能看出来的。为什么滚轮是锥形的？你如何用这些轮子控制车辆？[Lesics]在最近的一个视频中[很好地解释了轮子是如何工作的](https://www.youtube.com/watch?v=noqBUEgyQ8A)，你可以在下面看到。

有了四个轮子，你可以有一对轮子——一个在右前方，一个在左后方——它们的净力矢量为+45 度。那么另一对轮子可以不同地制造，以具有-45 度的净力矢量。该视频展示了向不同方向移动部分或全部车轮如何使车辆向多个不同方向移动。

很容易混淆麦克纳姆轮和全向轮，但它们是不同的。虽然两者都有一个带滚轮的主轮，但 omniwheel 的滚轮与主轴成 90 度角，因此它们可以在那个方向滑动。这些通常用在一个三角形的配置中，称为猕猴桃驱动。麦克纳姆车轮并非没有缺点，包括成本。然而，你可以[制作自己的](https://hackaday.com/2018/04/04/laser-cut-mecanum-wheel-for-the-budget-roboticist/)。
