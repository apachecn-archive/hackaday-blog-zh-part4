# 赋予 3D 打印塑料一种经久耐用的金属外观

> 原文：<https://hackaday.com/2021/06/04/give-3d-printed-plastic-a-well-worn-metal-look/>

负担得起的 3D 打印机让我们无需昂贵的大车间就能将想法变成现实，但伴随着它们的强大而来的是一些缺点。FDM 打印机的性质赋予他们生产的零件层线和喷嘴脊。它们可以通过优化打印设置最小化，但永远不会消除。[艾米丽·维拉斯科]喜欢 3D 打印的力量，但不喜欢零件的外观。所以她努力让 3D 打印的塑料看起来像劣质金属，并向我们展示了她是如何做到的。(视频也在休息后嵌入。)

这个视频是她的宠物眼睛项目的后续，以回应 Twitter 上的反馈。她曾提到，回收的 Pet Eye 金属盒不够大，无法容纳所有东西，所以她不得不在背面安装一个 3D 打印盒来扩展其内部体积。它非常适合，以至于这个即兴的评论让许多想知道更多关于它是如何完成的人感到惊讶。所以她设计了一个覆盖着机械特征的演示立方体，并给我们演示了它的变形。

类似的技术可以在其他领域找到，如精细比例建模。但从历史上看，这些领域不必应对 3D 打印塑料的层线和脊带来的挑战。在任何情况下，我们都喜欢看到这种交叉授粉:就像 3D 打印零件被用来定制精细比例的模型一样，我们可以将成熟的金属加工技术融入 3D 打印零件。这是我们盒子里的另一个工具，用于 [3D 打印整理](https://hackaday.com/2017/11/15/visual-3d-print-finishing-guide/)和[工具，如热](https://hackaday.com/2018/05/03/smooth-pla-through-the-fire-and-flames/)和[甚至盐](https://hackaday.com/2020/10/27/take-this-3d-print-post-processing-method-with-a-grain-of-salt/)。
