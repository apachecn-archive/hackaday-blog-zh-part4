# 一字钟，来硬的

> 原文：<https://hackaday.com/2019/04/02/a-word-clock-the-hard-way/>

我们都见过单词钟，它们很棒，但用单词显示时间的方式就这么多。这个有 114 个伺服系统的字钟是最难做到的。

我们不确定[Moritz v. Sivers]用这个投影钟的目的是什么，但他肯定是对的。基本想法是将组成时间信息所需的字符投射到半透明的 PVC 屏幕上，这当然可以通过简单的字符掩模和一些 led 来实现。但是为了获得额外的效果，[莫里茨]将每个字符安装在一个新像素上的信箱里。信箱安装在由微型伺服机构驱动的齿条和小齿轮上。他们离屏幕越近，焦点越清晰，人物越小。加上一点颜色的变化，时间似乎从一个混乱的、没有聚焦的背景中浮了出来。它非常引人注目，值得花 200 多个小时来制作所有部件。[完整的构建说明](https://www.instructables.com/id/Word-Clock-Controlled-by-114-Servos/)可用，演示视频在休息后。

我们几乎喜欢任何单词钟——大的[、](https://hackaday.com/2016/07/10/walnut-guitar-back-yields-wood-for-classy-word-clock/)[小的](https://hackaday.com/2019/03/27/leds-shine-through-pcb-on-this-tiny-word-clock/)，甚至是[宽屏](https://hackaday.com/2016/04/27/slim-and-classy-word-clock-shows-the-weather-too/)。不过，这个真的触动了我们所有的按钮。
