# 用二手银器纺一些马铃薯

> 原文:[https://hack aday . com/2022/02/12/spin-some-spudgers-from-二手货-银器/](https://hackaday.com/2022/02/12/spin-some-spudgers-from-secondhand-silverware/)

尽管它不是这项工作的合适工具，但我们都曾将平头螺丝刀用于其他用途。承认吧——你已经用工具箱里的那一个撬开了一两件东西，反正都是脏的，被弄坏了。但通常情况下，螺丝刀不够薄。你需要的是一个土豆，出于某种原因，似乎只有塑料的。归咎于我们的一次性时代。

在一次与建筑电子相关的休息中， [[lonesoulsurfer]花时间用二手银器制作了一套土豆钳之类的东西。这些都是从旧货店买来的勺子和黄油刀制成的。对于勺子，[lonesoulsurfer]用角磨机去除头部，在砂带机上成型，然后变薄，直到它们变得细长。在把手末端做了同样的工作后，[lonesoulsurfer]用一些化合物打磨了轮子上的新工具。](https://www.instructables.com/How-to-Make-Spudgers-prying-Tools-Out-of-Cutlery/)

并非所有这些都是 spudgers 一些注定要刮，其他的为解除徽章和贴花。但它们都和谐地生活在一个方便携带的箱子里。休息之后，请观看构建视频。
