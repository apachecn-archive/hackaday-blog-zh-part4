# 这是你八岁时想让你爸爸给你做的自行车！

> 原文:[https://hackaday . com/2019/11/16/this-the-bike-you-want-you-dad-to-make-you-when-you-bad-岁/](https://hackaday.com/2019/11/16/this-is-the-bike-you-wanted-your-dad-to-make-you-when-you-were-eight-years-old/)

足智多谋的[涡轮征服巨鹰]有一个开始项目的极好借口，他为他的孩子建造它们并制作视频。我们不太确定他的小摩托车是不是造出来的，因为爸爸也想试一试，因为他似乎从测试中获得了很多乐趣。

这个建筑从一个中国的自行车汽油转换套件开始。有一个小的~~两个~~四冲程发动机和一个基本的链条驱动一个大链轮，旨在安装在自行车车轮的相对侧踏板链轮。他使用了一对气动独轮车车轮，为其制作了一个新的衬套，并在其上焊接了链轮。这些进入一个相当简单的硬尾框架，他为它做了一个带衬垫的摩托车座椅，从那时起他就准备好出发了。

其结果是一个相当酷的小非道路合法摩托车，我们怀疑大多数读者会渴望拥有。不过，我们不太确定它似乎没有刹车。自己判断，视频在休息下面。

这不是我们带给你的第一辆国产小自行车，尽管它比第一辆安全得多。
