# 贴现工具公司的组织启示

> 原文：<https://hackaday.com/2022/10/15/organizational-inspiration-from-the-discount-tool-company/>

当需要任何工具来快速完成工作，或者只需要少量时间时，在当地有一个当地的“折扣工具”公司来解决一些工作，但通常是低质量的解决方案来解决可能出现的任何问题。虽然有一些宝石，但这些工具中的大多数不会像更昂贵的同类产品那样持续大量使用。另一方面，在这些折扣店还可以找到其他东西，比如解决储物问题的灵感。

这种特殊的存储系统来自 Harbor Freight，并使用一组链接的十字线，其中心是中空的。一组可移动的隔间位于顶部，支脚可以在十字准线内互锁。这使得工具箱内的空间得到了更有效的利用，但是[Alan]希望它能有更多的用途。他设计并实现了存储案例基础模板(SCBT)，允许任何大小的容器安装类似的十字准线网络。

随着这个非专有系统的实施和打印，减少[Alan]工作空间混乱的最初目标已经实现。3D 打印文件可以在任何空间轻松修改，在 [Thingiverse](https://www.thingiverse.com/thing:5527247) 和 Printables 上均可获得。对于在小空间里打包很多东西的其他方法，我们推荐了[这个小工作室](https://hackaday.com/2019/09/18/tiny-woodshop-is-packed-with-space-saving-hacks/)，不久前，这里堆满了存储工具。
