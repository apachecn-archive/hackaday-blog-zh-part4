# 速 8 电影编辑器重生为 YouTube 终端

> 原文：<https://hackaday.com/2022/10/30/super-8-film-editor-reborn-as-a-youtube-terminal/>

我们喜欢为旧设备赋予新生命的黑客，而[edwardianpug]的 [YouTube 终端](https://www.reddit.com/r/cyberDeck/comments/uxfhq6/my_youtube_terminal_not_entirely_sure_if_it/)通过将新硬件放入 Super 8 电影编辑器中，无疑符合这一要求。

[edwardianpug]本可以将这段看起来很经典的影音历史放到架子上展示，但她决定更新它的组件，这样它就可以显示任何 YouTube 视频，而不是一次只显示一段电影。Boost-Box 通过使用终端通过 [Ytfzf](https://github.com/pystardust/ytfzf) 搜索和播放视频来保持复古未来主义的主题。

原来的屏幕已经被一个 800×600 的 LCD 取代，黄色的 USB 线提供了一个漂亮的颜色，可以将直形键盘连接到设备上。以免你认为这“毁了”一个复古技术的作品，[edwardianpug]说 [20 分钟](https://www.reddit.com/r/RASPBERRY_PI_PROJECTS/comments/ychs75/comment/itpztls/?utm_source=share&utm_medium=web2x&context=3)就能让这个设备回到观看老电影的状态。

你在寻找更现代和复古的混搭吗？看看这些建在漂亮复古外壳中的[骰子塔](https://hackaday.com/2022/07/26/hackaday-prize-2022-digital-dice-towers-built-in-beautiful-retro-cases/)，一个[真空管和微控制器火腿发射器](https://hackaday.com/2022/02/27/retro-and-new-tech-combine-in-this-hybrid-ham-transmitter/)，或者这个[电脑平台在一个复古扬声器](https://hackaday.com/2022/10/05/retro-speaker-becomes-the-perfect-micro-pc/)。
