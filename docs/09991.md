# 储蓄罐吃角子老虎机让储蓄变得有趣

> 原文:[https://hack aday . com/2021/05/12/piggy-bank-slot-machine-puts-a-spin-on-saving/](https://hackaday.com/2021/05/12/piggy-bank-slot-machine-puts-a-spin-on-saving/)

存钱本来就没什么意思，直到你有机会把钱花在更棒的事情上。如果有可能立即获得超过你投入的金额的回报，你会不会更愿意把硬币投入存钱罐？我们知道我们会的。最棒的是，如果你把这样一个储蓄罐吃角子老虎机放在户外，你的朋友和邻居可以玩它，你可能会赚更多的钱。就像他们说的，房子总是赢。

[![](../Images/d66f9615758cfe616476aa94a99ee1e4.png)](https://hackaday.com/wp-content/uploads/2021/05/piggy-bank-slot-machine-inner.jpeg) 将一枚硬币投入投币口，硬币穿过一对电线，电线就像一个简单的开关，启动卷轴旋转。里面是一个 Arduino Uno 和一个巨大的印刷螺旋进料器，由一个小型步进电机和一对印刷齿轮驱动。卷轴已经现代化，显示器由四个独立的 LED 矩阵组成，由于一些烟熏粘合膜，它们看起来像一个单元。

这个漂亮的小机器花了整整一周的时间进行 3D 打印，其中包括浪费在一个失败两次的巨大部件上的 32 个小时。[Max 3D Design]尝试在切片机中将模型旋转 180°，谢天谢地，这解决了问题。然后是无数个小时的打磨，用身体填充物抹平，涂底漆，上漆，让它看起来棒极了。

如果你想自己制作，所有的文件都在 Thingiverse 上。代码没有显示出来，但是我们知道 Arduino 老虎机代码已经存在了。休息之后，请观看构建和演示视频。

尽管我们很喜欢这个建筑简单性，但如果有手柄可以拉的话，它会更像老虎机。[原来你也可以打印这些东西](https://hackaday.com/2020/12/05/slot-machine-has-a-handle-on-fun/)。
