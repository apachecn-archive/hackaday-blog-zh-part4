# 这个尤克里里做芯片音乐，不仅仅是因为它是由一个游戏机做成的

> 原文:[https://hack aday . com/2018/10/24/this-ukulele-does-chip tunes-and-not-just-based-out-of-a-game-boy/](https://hackaday.com/2018/10/24/this-ukulele-does-chiptunes-and-not-just-because-its-made-out-of-a-game-boy/)

当你想到创作型歌手时，鲍勃·迪伦的名字可能会浮现在你的脑海中。你可能会想到杰夫·巴克利，你可能会想到汉克·威廉斯，斯普林斯汀，大卫·鲍依，或者普林斯。你错了。有史以来最伟大的创作歌手是小蒂姆，这个看起来像怪异艾尔的家伙穿越时空，在贝克时代的《神秘博士》中客串了一把。小蒂姆有天使般的嗓音，因为我想曼蒙和彼列也是天使。小蒂姆也是当前尤克里里琴复兴背后的灵感来源，这种*一样东西*让弦乐器行业今天还活着。

尽管小蒂姆在 1996 年就去世了，他还是很想看到这个将尤克里里琴带入 20 世纪末的项目。这是一个游戏男孩，DMG-01，变成了一种可玩的乐器。这是一个功能性的 uke，但它也有电子设备将它变成一个 chiptune 机器。

这个项目的第一个目标是从一个游戏机盒子里造出一把实用的尤克里里琴。这很简单——颈部是 3D 打印的，桥是拧进去的，游戏机的外壳是用一些 PCB 材料加固的。到目前为止，这不是什么新鲜事；你可以在 Thingiverse 上找到 3D 打印的尤克里里琴的模型。

这个项目的第二个目标是把这个尤克里里变成一个 chiptune 机器。这意味着要为琴弦设计一个拾音器，因为这些是尼龙的，所以你不能在尤克里里琴上做磁性拾音器。第一种解决方案是红外反射传感器，它可以工作，但功耗太高。更好的解决方案是一个标准的弯曲压力传感器，它工作得足够好。这个信号被扭曲成方波，发出令人惊讶的游戏男孩般的声音。你可以看看下面的视频演示。
