# 迷人的时钟用高高的电子管报时

> 原文：<https://hackaday.com/2019/10/08/captivating-clock-tells-time-with-tall-tubes/>

时间可能是我们最重要的社会结构。我们对流逝时间的感知随着我们所做的一切而改变，归根结底，时间是我们真正拥有的一切。你可以选择明智地使用它，或者袖手旁观。如果你想两者兼得，就造一个这样的钟，看起来时尚精致。

[【ChristineNZ】的世纪中期遇见蒸汽朋克时钟](https://www.instructables.com/id/ILC1-18L-Clock/)使用了八个 il C1-1/8l，这很可能是有史以来最大的 VFD 电子管(仍然有新旧库存)。除了时间，它还显示日期、相对湿度以及摄氏和华氏温度。每隔 15 分钟就会有悦耳的钟声提醒你时间在浪费。

秒针以 HH/MM/SS 的格式滑过，每一个刻度都被一根专门用来跳舞打发时间的管子隔开。迷人的显示器由 Arduino Mega 和 MAX6921 VFD 驱动器驱动，内置于红木框架中。除了 Mega 之外，没有一个 PCB 所有的 VFD 都安装在木头上，所有的东西都是点对点连接的。扫过休息时间，观看渐进式幻灯片构建视频，视频以所有功能的演示结束。

那些发光的蓝绿色显示屏并不局限于计时。它们可以[取代液晶显示器](https://hackaday.com/2019/06/12/prusa-printer-gets-an-lcd-ectomy-gains-a-vfd/)，或者成为[滚动字幕](https://hackaday.com/2018/04/19/marquee-display-uses-six-dozen-surplus-vfd-tubes-to-great-effect/)。
