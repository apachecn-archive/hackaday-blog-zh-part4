# 绿色黑客:超频光合作用

> 原文：<https://hackaday.com/2019/01/05/green-hacking-overclocking-photosynthesis/>

我们认为黑客是按照我们的意愿来控制技术。但是一些系统是生物性的，我们也开始看到在那个领域有更多的黑客攻击。这应该会让科幻迷们感到兴奋，他们习惯于阅读与生物技术有关的文化，所以也许我们也会在现实世界中实现这一点。入侵农作物和动物可以追溯到几个世纪前，尽管我们在这方面确实越来越好。一个恰当的例子是:科学家们已经找到了一种让光合作用更好的方法，这应该会带来更多的农作物。

我们在学校学过，植物利用二氧化碳和阳光来产生能量和氧气。但是没有人向我们解释这到底是怎么发生的。似乎是一种名为 rubisco 的蛋白质导致了这种情况，但不幸的是，它并不太挑剔。除了将碳(从二氧化碳)转化为糖，它还将氧转化为称为 ROS(活性氧)的有毒化合物，然后大多数植物不得不花费能量来消除这些化合物。科学家估计，如果你能恢复在这个过程中损失的卡路里，你可以在目前的生产水平上养活全球额外的 2 亿人。

下面的视频解释了不同种类植物的光合作用机制。植物必须做一些事情来抵消这些有毒化合物，在 C3，植物占物种的 85%，这需要一些产生的能量来对抗。玉米、高粱和甘蔗是 C4 植物，它们处理光合作用的方式不同，自然减少了有毒物质的产生，相应地产量也更高。C3 植物还有一个额外的问题，除了用于清除毒素的能量外，大约 20%的产糖机制被转移到制造毒素上。

外行人对这项研究的解释是，正常的植物有一个很长的路径来清除毒素——光呼吸——科学家们设计了较短的路径，并开发了基因来实现黑客攻击，然后将它们拼接到烟草植物中，并在真实的田地中种植两年。这种转基因植物长得更快更高，同时产生了 40%以上的生物量。烟草很容易种植，但现在他们正转向粮食作物，看看能否复制他们的成功。

这项研究部分由比尔和梅林达盖茨基金会赞助，是一项名为 RIPE(实现提高光合效率)的全球努力的一部分。虽然我们可能无法达到哈里·哈里森在《伊甸园之西》中想象的生物技术，但生物工程可以产生巨大的影响。例如，读一读诺曼·博洛格如何因为拯救了大约 10 亿人而获得诺贝尔奖。当然，也有常规的技术方法[使农业更有效率](https://hackaday.com/2018/10/25/vinduino-water-smart-farming-now-with-lora/)。
