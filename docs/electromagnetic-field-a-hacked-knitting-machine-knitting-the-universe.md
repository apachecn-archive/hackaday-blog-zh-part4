# 电磁场:一台被黑掉的针织机，编织着宇宙

> 原文：<https://hackaday.com/2018/09/06/electromagnetic-field-a-hacked-knitting-machine-knitting-the-universe/>

一个大型黑客营吸引了来自世界各地的与会者，在最近的英国电磁场展上，当然有大量的国际游客。可能其中一个旅程最长的人是来自澳洲的[莎拉·斯潘塞]，她值得我们钦佩，不仅因为她的工作，也因为她将自己微薄的行李空间用于她为活动带来的装置。在休息室的帐篷里，你可以找到编织的宇宙，这是一幅夜空地图，点亮的新像素星座覆盖了整个墙壁，在演讲中，你可以找到她如何通过将一台 20 世纪 80 年代的兄弟针织机黑进网络打印机来创建它的深入描述。

她从针织机黑客入侵的盆栽历史开始，导致了模拟软盘驱动器的使用，取代了用于在原始硬件上存储扫描设计的机械项目。她拿了一台 16 位兄弟针织机的现有黑客软件，并为她后来的 32 位 mode l 重新编写了它，然后为它创建了一个名为 [Octoknit](https://github.com/chixor/octoknit) 的网络界面，它在树莓 Pi 上运行。然后，我们通过一台针织机的操作和她在逆向工程文件格式的进一步冒险。她最终得到了抖动的四色图像，但仍有一个问题。在 Brother 上，颜色变化是通过按一个按钮来实现的，所以需要一些自动化的过程。这个任务被她的丈夫接手，他创造了[一个 Arduino 驱动的机械按钮按压器](https://github.com/mage0r/AutoChanger)，这已经成为一个团队的努力。有了这个，她唯一的手工任务就变成了定期调整重量，以保持针织成品的张力。

最后，她转向编织的宇宙本身，这在当时已经成为一种病毒式的感觉。我们这些创建了黑客营装置的人会欣赏这个作品的工作量，她真的应该在演讲结束时得到掌声。在休息时间下面观看，这是一个引人入胜的半小时。
