# 制作古代蒸汽机的现代版本

> 原文:[https://hack aday . com/2020/10/06/making-a-modern-version-a-a-steam-engine-from-ancient/](https://hackaday.com/2020/10/06/making-a-modern-version-of-a-steam-engine-from-antiquity/)

想象一下，回到 2200 年前，没有什么比肌肉或风的速度更快了。想想看，如果看到像第一个已知的蒸汽轮机的例子——希罗的发动机，那该是多么令人震惊。看着一个球体呼啸着拖曳着一缕蒸汽，同时火焰在它周围舔来舔去，这可能是一种近乎神秘的体验。

当然，我们不能像那样回到过去，但是看到一个现代的 Hero 引擎复制品并测试它可能离这样的经历不远了。这个引擎，也被称为 aeolopile，是由[Make It Extreme]的工作人员制造的，他们的金属加工视频总是一种享受。发动机的转子由一对焊接在一起的半球形碗制成，由穿过大壶盖的管子支撑着。【做到极致】煞费苦心让发动机安全，有了原来不可能有的安全阀和压力表。aeolopile 外观很棒，与对该设备的描述非常相似，该设备可能是也可能不是由希腊数学家[亚历山大的 Heron 发明的，正如下面的视频所示，当它旋转起来时，它会上演一场精彩的表演。

人们不禁会想，这样的东西是如何在没有人——任何人——采取下一个合乎逻辑的步骤的情况下被发明出来的。它只是被当作一种好奇心，并没有引发两千年前的工业革命，这令人难以置信。虽然我们以前见过比 Hero 的引擎简单得多的版本，但这一款确实在金属加工能力上占了上风。
