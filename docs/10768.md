# 简单的尖端有助于在困难的零件上实现粉末喷涂的完美

> 原文:[https://hack aday . com/2021/07/24/simple-tip-helps-with-powder-coating-perfection-on-different-parts/](https://hackaday.com/2021/07/24/simple-tip-helps-with-powder-coating-perfection-on-difficult-parts/)

说在一元店可以买到的花园小径灯很便宜是对它们毫无价值的夸大。这些太阳能灯非常便宜，几乎没有必要购买，这一事实导致[马克·普雷斯林]陷入了一个制造兔子洞，并以[一些关于复杂几何形状粉末涂层零件的伟大技巧](https://www.youtube.com/watch?v=L8UiK909Tpg)结束。

对于像庭院灯这样平凡的东西来说，粉末涂料似乎有点大材小用，但[马克]有一个观点——如果你买的东西在阳光下晒了几周就坏了，你还不如自己动手做。一个合适的装饰不仅是获得正确外观的重要部分，也是让这些完全不像 Tardis 的灯具在天气中持续存在的重要部分。下面的视频系列涵盖了整个设计和建造过程，最终形成了带有一些深槽的铝格栅。这种特征被证明很难用粉末涂层达到，在粉末涂层中，涂层的微小颗粒由于它们之间的高电位差而被吸引到工件上。涂覆后，零件被加热以熔化颗粒并形成坚韧、美丽的表面。

但是对于凹槽和其他高纵横比的特征，颗粒倾向于避免聚集在角落和缝隙中，导致不均匀的光洁度。[Mark]的解决方案是转向“热植绒”，在将不带电荷的涂层应用到深层特征之前，先加热零件。这使得角落和凹槽在其余涂层以标准方式涂覆之前被很好地涂覆，从而导致更好的光洁度。
