# 迪士尼幻想工程的“奇异果项目”熊格鲁特

> 原文：<https://hackaday.com/2021/04/26/disney-imagineerings-project-kiwi-bears-groot/>

总有一天，我们可能会原谅波士顿动力公司垄断了步行机器人市场。他们(和其他玩家)正在他们的领域取得令人难以置信的进展，但三年前，迪士尼试图为他们的一些更小的电影角色创造自主、自由行走的机器人演员，发现现有的平台都不合适。所以他们成立了他们的想象工程部门，致力于“奇异果项目”，我们现在看到了这些努力的成果。

关于双足机器人的研究已经积累了多年，正如俗话所说，如果这些想象工程师看得更远，那是因为他们站在更大的机器人平台的肩膀上。然而，项目 Kiwi 的设计师们在他们的小型化过程中列出了一系列创新，从迫使空气通过中空骨骼的“骨髓导管”冷却系统，到允许执行器甚至在关节之间共享电机的传动装置。电子元件通过带状柔性电缆分布在各个 PCB 上的骨架周围，以减少布线，几乎每个组件都是定制的，以满足复杂的尺寸和重量要求。

即使在这个早期的原型中，迪士尼在栩栩如生的电子动画中的根基也是显而易见的。格鲁特的动作是情绪化的，如果有点小心的话，软件可以通过他的步态和姿势表达各种各样的个性。眼睛和脸就像我们期望的那样富于表情(尽管对接缝的敏锐眼光会消除一些明显的西方世界的感觉)。据报道，这个版本可以处理温和的推挤和接触，但我们确实发现一个安全电缆仍然连接到头部。因此，在我们看到它在公园里与公众互动之前，可能还有一段路要走。

迪士尼的幻想工程部门已经在机器人方面做了一些令人惊叹的工作，他们继续在更传统的电子动画领域做出重大创新。它现在看起来肯定是最酷的工作场所之一，现在我们渴望[建造我们自己的两足朋友来和](https://hackaday.com/2017/10/20/hackaday-prize-entry-two-leg-robot/)一起玩。
