# faxsploit–利用带有图片的传真

> 原文:[https://hack aday . com/2019/05/04/faxsploit-exploining-a-fax-with-a-picture/](https://hackaday.com/2019/05/04/faxsploit-exploiting-a-fax-with-a-picture/)

安全研究人员发现了一种通过向传真机发送特制文档来远程执行传真机上的代码的方法。所以…谁在乎传真？显然，许多人仍在许多机构、政府和行业中使用它，包括医疗保健行业、法律、银行和商业。官僚主义和旧程序往往难以根除。

出于许多原因，这是值得适当关注的利用之一。这是有据可查的，是一个很好的老式黑客和逆向工程的例子。[Eyal Itkin]、[Yannay Livneh]和[Yaniv Balmas]在一篇做得很好的文章中向我们展示了他们的过程，你可以在这里阅读。如果你对安全黑客感兴趣，这真的值得一读，也值得[看 DEFCON 视频](https://www.youtube.com/watch?v=qLCE8spVX9Q)。他们将注意力集中在多功能打印机/扫描仪/传真机上，结果非常好。

> 我们的研究旨在探究，如果攻击者只需一根电话线，除了目标的传真号码之外一无所有，就能通过向多功能一体打印机发送恶意传真来攻击该打印机，将会发生什么。
> 
> 事实上，我们在多功能一体打印机中发现了几个严重的漏洞，这些漏洞使我们能够“利用”多功能一体打印机，并通过发送恶意制作的传真来完全控制它。

正如研究人员所指出的那样，一旦多功能一体打印机遭到破坏，它就可能被用于各种恶意活动，从渗透内部网络到窃取打印文件，甚至到挖掘比特币。理论上，他们甚至可以制造传真蠕虫，通过电话线复制。
