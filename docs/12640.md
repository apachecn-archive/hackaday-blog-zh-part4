# Remoticon 2021 // Vaibhav Chhabra 和 M19 集体制造了一百万个面罩

> 原文：<https://hackaday.com/2022/03/10/remoticon-2021-vaibhav-chhabra-and-the-m19-collective-make-one-million-faceshields/>

[Vaibhav Chhabra]，印度孟买创客庇护黑客空间的联合创始人，[通过讲述一个关于黑客空间如何发展到目前地位的小故事开始了他的 Remoticon 演讲](https://youtu.be/-RTtVraPVQg)。它诞生于对办公室天花板倒塌的失望，经历了 8 年的搬迁和重组，积累了一个忠诚的参与者基础——这在管理良好的黑客空间中并不罕见。当新冠肺炎到达印度时，这种环境为 M19 的努力提供了一个完美的滋生地，将“我们能做什么”和“我们应该做什么”的询问混合成一场完美的风暴，并开始了 49 天的工作会议，在物理和组织上迅速超越了黑客空间。

当 2020 年无限期两周隔离的前两周在印度宣布时，一群人决定在黑客空间等待，而不是把自己限制在家里。随着我们社会的各个方面在新冠肺炎的直接影响下开始崩溃，消息传来——个人防护装备短缺，尤其是对一线工人而言。各国普遍没有准备好个人防护设备，印度也不例外。因此，制造者收容所里的人们站了出来，发现自己处于一个制造防护设备的绝佳位置，而其他人都不准备提供帮助。

这些年来，我们已经看到了类似的集体项目——这个项目从规模上来说是非常宏伟的。所有这些都值得我们学习——例如，在各种黑客空间基础设施的背景下，可制造性优化的细微差别。对于任何希望成功的项目来说，一个经常被遗忘的需求得到了扩展——将最终用户，医疗保健专家，保持在一个紧密的迭代设计循环中。这种努力的数量，加上媒体的报道，没有被其他黑客空间忽视，这是分享知识的挑战出现的地方，并被迅速处理。最终结果？仅在一个半月多一点的时间里，印度的黑客空间和其他配备激光切割机的地方就生产了一百万个面罩。

在对面罩的需求开始最终得到满足后，黑客空间联盟的新能力和优势正在等待下一个目标的开启——至少，那些不需要转移到质量保证和保持努力运行的目标。期间开发的经验和框架现在是研究案例研究和 QA/QC 原则形成工作的重要组成部分，由剑桥大学领导和资助。反过来，制造商的可用资源被用于设计和制造氧气浓缩机，以及维修那些正等待投入使用的设备。

如果你曾经寻找无数黑客空间在一个社会项目上合作的故事，这就是它——对于任何希望激励其他黑客攻克大型社会问题的人来说，有很多东西可以学习。这个演讲涵盖了这个过程中你想学习的每一个部分，用一个激动人心的故事包装，你可以把它拍成纪录片。这种努力还在继续，无论何时下一次风暴袭击我们，我们都会看到它的原则在起作用。
