# 自制的 Onewheel 使用悬浮滑板部件

> 原文：<https://hackaday.com/2020/03/08/homebuilt-onewheel-uses-hoverboard-parts/>

自从 1989 年《回到未来 II》首次上映以来，人们一直在等待悬浮滑板成为现实。相反，我们得到了一个危险的双轮装置，同名。想要更酷一点的东西，[【巴特克·普洛内克】决定将他的改装成单轮设计。](https://www.youtube.com/watch?time_continue=1&v=Z2dxsSuPQGs&feature=emb_logo)(视频，下面嵌入。)

黑客首先加工悬浮滑板的轮毂电机。它们被栓在一起，用作一个大轮子的轮毂。正如[Bartek]在第一次尝试中发现的那样，在此过程中必须小心避免马达外壳破裂。然后将轮子安装在钢架的中心，在钢架上安装两个滑板作为踏板。还是用原来的悬浮板控制器；我们很想知道新的电机配置是否需要修改固件。

这是一个经典的车库黑客，导致一个可行的个人电动汽车。此外，转弯更容易，DIY onewheel 能够很好地来回雕刻。我们已经看到其他人打算使用类似的建筑来通勤。
