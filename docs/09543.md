# LTSpice 中的自定义组件

> 原文:[https://hack aday . com/2021/03/16/custom-components-in-lt spice/](https://hackaday.com/2021/03/16/custom-components-in-ltspice/)

如果你喜欢模拟电路，你可能用过 LTSpice。这个程序有很多我们不太用的强大功能，包括定制相当复杂的组件的能力。为了说明它是如何工作的，[asa pro]制作了一个电位计组件，这不仅是一个很好的说明，也是一个有用的组件。

当然，元件只是两个电阻。但是，使用参数，组件将获得两个值，总阻力和百分比。然后实际电阻值会自行调整。

为了简化符号的制作，视频展示了如何读入一个现有的符号(在这种情况下是一个电阻)，然后修改它以适应需要。当然，最后，您将看到如何实际应用新组件。当然，真正的价值将是制造更复杂的元件，但电位计向您展示了您需要的所有技术，如计算内部值和将引脚映射到变量。

这是一件并不太难的事情，你只需要花时间去理解它一次。视频可以有所帮助。如果你需要更基本的帮助，我们有自己的视频。或者查看一些[电路 VR](https://hackaday.com/2018/08/23/circuit-vr-measuring-with-ltspice/) 系列。
