# 用人工智能助母鸡一臂之力

> 原文：<https://hackaday.com/2022/09/26/lending-a-helping-hand-to-hens-with-ai/>

正如任何以前养过鸡或其他家禽的人会告诉你的那样，这可能是非常累人的工作。那么为什么不造一个机器人来替我们做所有的艰苦工作呢？这正是[Aktar Kutluhan]用人工智能驱动的物联网系统展示的内容，该系统可以自动喂食小鸡并监控未孵化的蛋。

毫无疑问，母鸡是可爱的、长有羽毛的动物，但它们也很挑剔。鸡蛋的重量、大小和频率可以决定母鸡的整体健康状况，如果像喂食时间表这样简单的事情过于分散，它们可以完全停止下蛋。这正是启发[Aktar]创建一个系统的原因，该系统可以每天在一致的时间给母鸡喂食，同时记录下的鸡蛋，以确保鸡舍快乐健康。

令人印象深刻的是，这一构建不仅仅是摆脱日常琐事的智能自动化，它完全是由物联网设备构建的，包括人工智能。该设置使用[边缘脉冲](https://www.edgeimpulse.com/)作为 OPenMV Cam H7 微控制器上的对象检测模型，以识别鸡笼中的鸡蛋。从那里，一个 WizFi360-EVB-Pico 板被连接起来，这样数据可以通过 WiFi 发送，一个 DHT22 被放入以监控和记录鸡舍的整体温度。

这已经是一个令人惊叹的设置，但当谈到物联网设备时，天空是无限的。你可以控制大鸡舍里的加热灯，如果母鸡的水少了，自动重新装满水碗，或者甚至建造一个自动孵化器。我们才刚刚开始看到人工智能帮助监控我们宠物健康的聪明方法。看看另一个黑客如何使用人工智能来监控猫屎，以确保他们毛茸茸的朋友没有吃塑料。感谢[Aktar Kutluhan]向我们展示了更多使用人工智能帮助宠物的方法！
