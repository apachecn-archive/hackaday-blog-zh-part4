# 树莓派洗牌机是电脑化的老千

> 原文:[https://hack aday . com/2020/07/20/raspberry-pi-shuffler-is-computerized-card-shark/](https://hackaday.com/2020/07/20/raspberry-pi-shuffler-is-computerized-card-shark/)

如果你和一小群人一起玩德州扑克或其他纸牌游戏，你可能会厌倦一次又一次的洗牌。[3dprintedLife]正处于这样一个位置，他意识到在他的预算中没有好的自动洗牌机。相反，他选择构建一个，并加入一些额外的功能来破坏游戏。

机器的机械结构发展很快，因为准确处理和分发卡片是一项挑战，特别是在 3D 打印零件公差宽松的情况下。在开发了一个可靠的传输机制后，它能够用一些基本的命令很好地洗牌。

然而，真正的魔力来自于安装一个摄像头和运行 OpenCV 的 Raspberry Pi。这能够读取每张牌的价值和花色，然后按照特定的顺序堆叠这副牌，以符合庄家的意愿。这一切都是通过网络界面控制的，能够在 21 点和德州扑克中创造有保证的胜利。Github 上的文件是为那些渴望更深入了解机器工作原理的人准备的。

这种机制做了如此漂亮的洗牌工作，以至于你的朋友可能甚至没有注意到这个诡计。这说明，在借助机器赌博时，你应该时刻保持头脑清醒。当然，如果你只想制造混乱，[这种乐高牌机枪可能更适合你。休息后的视频。](https://hackaday.com/2020/03/07/vanquish-your-foes-with-lego-playing-card-machine-gun/)
