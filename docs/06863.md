# Arduino 磁性板一点也不无聊

> 原文:[https://hack aday . com/2020/06/17/arduino-magnetic-board-is-than-than-boring/](https://hackaday.com/2020/06/17/arduino-magnetic-board-is-anything-but-boring/)

磁铁(尤其是那些球磁铁！)都是无穷无尽的迷人之处，不是吗？将它们与 led 结合在一起几乎是危险的，因为像[andrei.erdei]的 [Arduino 磁性板](http://www.instructables.com/id/Arduino-Magnetic-Board/)这样的东西在键盘之外招手，你怎么能完成任何事情呢？

当 RGB LEDs 在性感的树脂玻璃场地上滚动时，这个有趣的板使用球形磁铁来点亮它们。在 LED 矩阵下面是由 36 个簧片开关组成的管弦乐队——这些充满气体的小玻璃米粒有轴向引线，在磁场的存在下可以合在一起或分开。led 由 Arduino Pro Mini 控制，用于音效的 8ω扬声器也是如此。

[andrei.erdei]已经为这个可爱的桌面玩具开发了一些应用程序，它们都在 GitHub 上。有一个追逐游戏，涉及倾斜棋盘，用磁铁抓住下一个红点，一个灯光绘画游戏，以及一个模仿色调矩阵的音序器。滚动到休息时间，查看一系列简短的演示视频。

想玩簧片开关，但目前找不到？你可以自己做。
