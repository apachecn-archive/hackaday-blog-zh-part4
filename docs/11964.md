# Minty Tunes 是 Altoids 罐中的无线音频

> 原文：<https://hackaday.com/2021/11/18/minty-tunes-is-wireless-audio-in-an-altoids-tin/>

如今，许多手机都不再有音频插孔了。如果您最喜欢的耳机尚未启用无线功能，可能会让您很难听音乐。Minty Tunes 结合一点 Altoids tin flair 解决了这个问题。

是的，最重要的是，这是一个内置在薄荷糖罐中的蓝牙音频接收器。建造相对简单，将一个便宜的蓝牙音频解码器模块连接到锂聚合物电池上。这与 TP4056 电池充电器 IC 配合使用，使电池能够轻松充满。然后，所有这些都被连接起来，卡在每个人都喜欢的黑客电子设备外壳里。

对于那些不喜欢 AirPods 之类的无线耳塞的人来说，这是一个从智能手机上听音乐的有用解决方案。正如建造者所说，这也是一种通过高保真音响或带辅助端口的汽车音响播放音乐的好方法。或者，你也可以从头开始制作自己的袖珍 MP3 播放器。休息后的视频。
