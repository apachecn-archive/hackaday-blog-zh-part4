# 一个 Arduino 掌机来统治他们

> 原文:[https://hack aday . com/2019/05/28/one-arduino-hand-hand-hand-to-rule-them-all/](https://hackaday.com/2019/05/28/one-arduino-handheld-to-rule-them-all/)

没有什么比重复的努力更烦人的了。不得不一次又一次地经历同样的困境是让自己精疲力竭的一种完美方式，甚至可能让你无法处理一直萦绕在你脑海深处的项目。[Alain Mauer]发现他已经构建了足够多的 Arduino 小工具，这些小工具足够相似，他可以通过创建一个标准化的硬件来节省自己的时间，他可以在上加载他的代码。

他开始将这个设备称为 Arduino Nano QP(代表快速项目)，现在它是 2019 年 Hackaday 奖的一部分。[Alain]并没有承诺它是一切的完美契合，但估计他提出的简单 Arduino 项目中约有 85%可以在 QP 上实现。这要归功于设备底部的螺丝端子，它让您可以轻松地连接任何尚未安装在板上的硬件。

QP 板本身有无处不在的 16×2 字符 LCD 显示屏(配有对比度控制微调器)，七个触觉按钮以模糊的游戏机风格布局排列，当然还有一个焊接你的 Arduino Nano 的点。所有这些都受到一个非常光滑的激光切割丙烯酸案件的保护，完成保留按钮和蚀刻标签。

我们已经看到了很多手持 Arduino 设备，但是我们不得不承认，这款设备的实用性引起了我们的兴趣。下次我们想做快速测试时，我们不介意在实验室里放一个这样的东西。
