# 宝丽金帮你激光切割美丽的图案

> 原文:[https://hack aday . com/2019/03/08/polygonia-helps-you-laser-cut-beautiful-patterns/](https://hackaday.com/2019/03/08/polygonia-helps-you-laser-cut-beautiful-patterns/)

激光切割机是令人惊讶的多功能工具，如果你能把它们分解成一系列平面部件，它可以帮助你建造各种复杂的结构。[大卫]在史密森尼发现了一些艺术品，这些艺术品吸引了他的眼球，它们使用了带有有趣重复图案的平面形状。想要自己制作类似的作品，他开始开发软件来帮助实现。

![](../Images/47c84523cecf1eb692b9892d54c3a7fe.png)

The tool makes it easy to generate complex profiles and export them for lasercutting.

[David]的工作成果是 Polygonia 设计套件。这是一个工具，旨在使激光切割创建几何图案变得容易和简单。web 界面设计者有各种各样的选项来绘制形状和图案，并且还可以控制框架大小。[大卫]用各种创作展示了这些特征。这个项目的 Instagram 页面以基本的矩形面板为特色，带有嵌入的立方体和三角形图案，一直到华丽的八面体，由许多面板用 3D 打印的夹子固定在一起。

这些都是用 Javascript、手工 CSS 和 HTML 编写的。云存储和认证都是通过 Amazon Web Services 处理的，这使得处理这些功能变得很容易。[David]特别努力创建了一个单独的测试站点，这样代码实验就不会影响到现场站点——这对任何商业努力都是至关重要的。

如果你想创作一些几何形状的激光切割艺术品，请在线试用这款工具。前 3 个导出是免费的，有多种订阅模式可供重度用户使用。我们完全有望在未来几年的周末市场上看到花式激光切割家居用品的爆发。

如果你对激光切割艺术的渴望还没有得到满足，试试这种多彩的压克力艺术。
