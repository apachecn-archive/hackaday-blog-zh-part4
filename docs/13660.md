# 彩色电脑不需要电池

> 原文：<https://hackaday.com/2022/05/18/coloring-computer-needs-no-batteries/>

虽然 Radio Shack 确实有彩色电脑，但我们不认为他们有这个想法。[Pepepépepe]有一些涂色书页和简单的规则，让你[用蜡笔](https://ipfs.io/ipfs/QmYz7DPRWypGQcbAHr7Mi8EKB6ntSPsEnUsCXbAhBiHQZP/)模拟逻辑电路。可下载的杂志有手写的说明和几个例子。

请记住，这是一台计算机，就像 20 世纪 60 年代的旧逻辑工具是计算机一样。它们实际上是数字逻辑电路的演示。为了操作“计算机”，你选择两种颜色，一种是正方形，另一种是圆形。你给路径涂上颜色，直到你到达一个“诺里”看起来可疑地像有眼睛的弹弓的诺里有一个特殊的规则。如果 nory 的两个分支上都有您的圆形颜色，则 nory 的输出将是方形颜色。否则出来的颜色就是圆圈色。

显然，这个规则生成了一个 NOR 门，其中圆形的颜色是 0，方形的颜色是 1。由于颜色沿着路径传播，您可以绘制包括 7 段显示在内的显示元素并使它们工作。

还有一些规则要学习，你可以建立一台电脑，甚至有一个游戏垫。不开玩笑。这样实用吗？当然不是。但对于一个聪明的孩子来说，这可能是一个有趣的数字逻辑入门，谁知道这会导致什么呢？

像这样再造一些旧的 Geniac“计算机”会很有趣。如果你喜欢纸电脑，有[心脏](https://hackaday.com/2019/11/22/cardboard-computer-replica-is-cheap-and-easy-to-make/)(你甚至可以[在 FPGA](https://hackaday.com/2017/10/13/computers-that-never-were/) 上模拟纸电脑)。
