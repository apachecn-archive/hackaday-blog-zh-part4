# 用砂盘焊接 SMDs

> 原文：<https://hackaday.com/2019/06/05/solder-smds-with-a-pan-o-sand/>

对于那些从小就与通孔元件打交道的人来说，表面贴装器件可能很难处理。然而，有很多技术是 DIY 一族无法掌握的。有了合适的设备，焊接 SMD 板是一件轻而易举的事——[只需给自己一个热沙锅(Youtube 链接，嵌入在下面)](https://www.youtube.com/watch?v=aEn3Wb_zrts) [！](https://www.youtube.com/watch?v=aEn3Wb_zrts)

这一过程始于专业制造的 PCB 和附带的模板。目前，所有主要的 PCB CAD 软件包都能够生成模板文件，许多制造商会在 PCB 订单中以最小的额外成本提供激光切割模板。电路板首先安装在一个稳定的表面上，并涂上焊膏，然后用镊子夹住元件。完美的放置是不必要的，因为熔融焊料的表面张力会将元件拉向正确的方向。然后将组装好的板放在煎锅中的沙层上，煎锅放在电磁炉上。然后加热电路板，直到焊料熔化，所有的元件都干净利落地回流。一旦允许冷却，板就完成了！

诀窍在于沙子有助于均匀加热电路板，同时使其与热源保持安全距离。结果很好，而且这个过程比手工焊接要快得多。关注这个过程也很容易。当然，传统的方法仍然是使用简陋的烤箱，但是新技术总是有用的。[我们也见过用本生灯做的](https://hackaday.com/2014/02/15/smd-soldering-on-hot-sand/)。休息后的视频。
