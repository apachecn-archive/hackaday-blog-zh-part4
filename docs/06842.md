# Marian Croak 是 VoIP 应用的 MVP

> 原文:[https://hack aday . com/2020/07/07/Marian-croak-is-the-MVP-of-VoIP-adoption/](https://hackaday.com/2020/07/07/marian-croak-is-the-mvp-of-voip-adoption/)

如果你曾经使用过 FaceTime，Skype，拥有一个神奇的 Jack，或者在灾难发生后通过发送短信捐款，那么你要感谢 Marian Croak。她的领导能力和前瞻性思维改变了马贝尔利用其影响力的方式，使所有这些事情成为可能。

Marian Croak 是一个说话温和的女人，自称不爱说话，但她的行动响亮地支持了互联网协议(IP)作为通信的未来。人类总是在寻找下一个最好的交流媒介，最快的清晰了解对方的途径。我们今天仍然在打电话，但是语音已经和文本和视频一起成为了第二好的存在方式。所有这一切都依赖于一个植根于 Marian 作品的多功能网络。

## 修理小姐

大多数孩子，尤其是女孩，在童年的某个时候都想帮助父母。但是年轻的玛丽安对学习烹饪或洗衣服不感兴趣——她想成为当炉子坏了或洗碗机开始漏水时，她妈妈会叫她的人。每当修理工上门时，玛丽安都会紧跟在后面，问他们关于设备和故障排除程序的问题。

其他 20 世纪 60 年代的家庭可能会阻止这种行为，但玛丽安得到了家人的大力支持，特别是她的父亲。她甚至在 10 年级就离开了天主教学校，去一所窗户上有栅栏的纽约公立高中追求更科学的教育。玛丽安的父母不高兴，但他们没有试图阻止她。事实上，她爸爸在地下室建了一个化学装置来表示他的支持。毕业后，玛丽安去了普林斯顿大学，然后搬到全国各地的南加州大学读研究生。在那里，她获得了定量分析(现在称为数据科学)和社会心理学的博士学位。

## 拉着马贝尔进入互联网时代

玛丽安在 1982 年接受了贝尔实验室的一个职位，并在她南加州大学毕业帽落地之前搬回了东海岸。当公司的大部分精力都放在语音上时，Marian 却在开发消息应用程序，并负责发现已经部署的消息应用程序是否相互兼容。

[![](../Images/b63dda0124ae0008db4558cfeac51ec1.png)](https://hackaday.com/wp-content/uploads/2020/06/Marian-Bell-Labs.jpg)

Marian at Bell Labs. Image via [Greater Diversity News](http://greaterdiversity.com/meet-black-woman-created-technology-behind-skype-magicjack/marian-croak-2/)

到 20 世纪 90 年代初，Marian 进入了数据服务和网络领域。这两者都是新基础设施的到期日，而美国电话电报公司正计划耗资数十亿美元转换到异步传输模式(ATM)协议。Marian 和她的团队开始研究 TCP/IP，并相信这是未来的趋势。他们支持这一协议，并使主管们相信 IP 是一条可行之路，尽管这项技术还不能完全满足集成语音、数据和视频的要求。

美国电话电报公司合并了 IP 和语音网络工程团队，并指派 Marian 管理这个由大约 2000 名工程师组成的团队，以便为网络做好准备。由于她的背景，Marian 处于一个独特的位置，既了解快速进入市场的 IP 工程师的困境，也了解网络方面缓慢、稳定、乐于测试的方法。但玛丽安不是什么袖手旁观的副总裁——她白天、晚上和周末都在战壕里。到 Marian 在美国电话电报公司 32 年的职业生涯结束时，她已经获得了 200 多项专利，其中一半以上与 IP 语音(VoIP)产品有关。

## 用你的手机投票

在 21 世纪初，短信还处于婴儿期，由 T9 输入统治——使用手机上的数字键盘拼出信息，~~通常通过~~(利用预测算法来避免)重复按键来选择每个字母。尽管如此，玛丽安还是看到了这项服务的潜力。AT & T 与*美国偶像*合作，由 Marian 的团队创建了一个基于文本的投票系统。数百万人学会了如何发短信，以便每周称体重。

[![](../Images/c9042d7d32e0d0d90491b18c1f7b8dee.png)](https://hackaday.com/wp-content/uploads/2020/06/Marian-with-drawings.jpg)

Image via [AT&T](https://www.youtube.com/watch?v=ejZfo1fLZZ8)

当 2005 年卡特里娜飓风袭击墨西哥湾沿岸时，Marian 看到了更多的潜力，于是[创建了一个平台，让人们通过发短信](https://www.fastcompany.com/3018870/the-surprising-link-between-american-idol-and-text-to-donate-fundraising)向救济基金捐款，实质上是授权在下个月的电话账单上多付一点钱。2010 年地震后,“捐赠”再次被用于为海地提供救援，取得了巨大成功。

Marian 于 2014 年被谷歌聘用，目前是一名工程副总裁。今天，她负责在发展中国家扩大互联网接入。她继续她的人道主义工作，在飓风玛丽亚的破坏后，她领导了 Loon 项目的倡议，为波多黎各的居民提供基于气球的紧急蜂窝服务。

在许多方面，她仍在做着她整个职业生涯中一直在做的事情——“试图以简单、务实的方式改变世界”，正如她在 2012 年写给科技界年轻女性的一封信中所说。玛丽安一直是我的灵感来源。你可以在下面观看她在谷歌 2017 年女性科技创客山景峰会上的主题演讲。
