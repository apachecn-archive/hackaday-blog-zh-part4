# 家酿门把手帽高电压乐趣

> 原文：<https://hackaday.com/2021/02/19/homebrew-doorknob-caps-for-high-voltage-fun/>

Mouser 和 Digi-Key 非常适合满足大多数需求，而且它们提供的零件范围令人眼花缭乱。但是考虑到硬件黑客社区的广泛利益，很少有公司能够满足所有人的需求。

当一个人的爱好涉及高电压和自制激光器时，就像[Les Wright]一样，对于需要的深奥部分来说尤其如此。他最近想出了一个 DIY 门把手电容设计,使得难以获得的高压电容更容易获得。我们已经看到[Les]在他的[横向激发大气(TEA)激光器](https://hackaday.com/2020/08/09/how-about-a-nice-cuppa-tea-laser/)中使用这些帽，这是一个简单的设计，使用高压放电穿过一个充满室内空气或氮气的狭长通道。高压电源需要大陶瓷电容，虽然[Les]有很多，但很难在网上买到。他尝试了[后续使用普通径向引线陶瓷电容器](https://hackaday.com/2020/12/26/no-doorknobs-needed-for-this-nitrogen-laser-build/)，当激光器工作时，他确实在电容器引线之间得到了一些闪络。

[Les]的解决方案是将厚重的瓶盖浸泡在丙酮中一周左右，以去除其环氧树脂覆盖层。一旦被剥去，引线被弯曲成更轴向的结构并焊接到黄铜机器螺钉上。然后将电介质块放入一小段塑料管中，并用环氧树脂封装，螺栓从两端伸出。结果很难与真正的门把手帽区分开来；下面的视频展示了构建过程和一些测试。

向[Les]脱帽致敬，他同情我们这些想要复制他的作品却发现自己没有这些必需品的人。很高兴知道有一种方法可以在你需要的时候制造出 unobtanium 部件。
