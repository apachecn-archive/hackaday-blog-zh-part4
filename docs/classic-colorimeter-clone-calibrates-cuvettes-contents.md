# 经典比色计克隆校准比色皿的内容物

> 原文：<https://hackaday.com/2021/12/06/classic-colorimeter-clone-calibrates-cuvettes-contents/>

对于任何涉足家庭化学的人来说，拥有精确的测量设备意味着成功与失败的区别。但是，由于许多仪器昂贵且难找，家庭化学家能做些什么呢？很自然地，建造他们自己的设备。[Abizar]着手用木头和备用电子元件为自己制作了一个色度计。

色度计(化学术语)是一种通过测量吸收多少特定波长的光来确定溶液浓度的仪器。[Abizar]的设计灵感来自 20 世纪 50 年代的经典 Klett-Summerson 比色计，它使用灯泡和滤色器来选择波长，加上光敏电阻来测量样品吸收的光量。当然，一个更现代的解决方案是使用各种颜色的 led，这正是[Abizar]所做的，尽管他确实通过使用模拟仪表作为读出设备给它带来了复古的感觉。

比色计的主体由激光切割的木头制成，堆叠在一起形成一个刚性外壳。色轮包含 11 个不同的 LED，并由一个巧妙的棘轮机构制成，以保持与比色皿对齐，以及一个滑动触点，以驱动电流进入所选的 LED。所有部件都被涂成黑色，以防止仪器内部的杂散反射，但也使它看起来足够酷，适合任何邪恶天才的实验室。在下面嵌入的视频中，[Abizar]演示了该仪器，并展示了它是如何组装的。

虽然我们以前没有见过任何人制作自己的比色计，但我们已经看到了 [DIY 分光光度计](https://hackaday.com/2016/08/01/hackaday-prize-entry-a-visible-spectrophotometer/)(测量溶液的整个吸收光谱)，甚至还有[积木来制作完整的生物化学实验室](https://hackaday.com/2016/08/17/lego-like-chemistry-and-biology-erector-set/)。
