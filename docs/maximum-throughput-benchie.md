# 最大吞吐量基准

> 原文：<https://hackaday.com/2021/12/26/maximum-throughput-benchie/>

你曾经需要快速制作几百个东西吗？[Roetz 4.0]让您了解了他大规模平行参加快艇比赛的情况。

快艇赛背后的想法是你可以多快地打印一艘 Benchy——一艘用作 3d 打印机测试打印的小船。加快打印速度是相当棘手的，因为这意味着移动头更快，给层更少的时间来沉积和一系列其他问题。所以[Roetz]借鉴了 CPU 设计师的剧本，他没有增加延迟，而是提高了吞吐量。最初的计划是 20 个热端，但由于冷却问题，不得不减少到 18 个。也许比机器的规模更令人印象深刻的是，它唯一现成的部件是冷却风扇。其他的都是[罗茨]自己打印或者加工的。整个运行是在不到一个小时内完成的，从技术上来说，他每次替补的时间不到 3.6 分钟，即使考虑到一些失败的情况。

这并不是(罗茨的)第一台定制 3d 打印机。不久前，他将一台坐标测量机改造成了一台 3d 打印机，在一大片建筑区域内提供了难以置信的精度。感谢[Jan Roetz]发送此邮件！休息后的视频。
