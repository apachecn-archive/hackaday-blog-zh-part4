# 维多利亚女王的秘密(电传打字机和宇宙精灵)

> 原文:[https://hack aday . com/2021/04/12/queen-victorias-secret-teletype-and-cosmac-elf/](https://hackaday.com/2021/04/12/queen-victorias-secret-teletype-and-cosmac-elf/)

我们真的不认为维多利亚时代的人有 COSMAC Elf——基于 RCA 1802 CPU 的家酿计算机。但如果他们这样做了，它可能看起来像是[丹尼尔·罗斯] [对该系统的蒸汽朋克再创造，包括一个看起来合适的电传打字设备](https://mycosmac1802project.blogspot.com/2021/03/victorian-teletype-aka-steampunk-themed.html)。你可以在下面的一系列视频中看到这个东西。实际上，有相当多的视频展示了该系统的不同部分，还有几个月前的博客帖子。

一个[魔眼管](https://hackaday.com/2021/01/22/meet-the-magic-eye-vacuum-tube/)在这个造型中看起来并不格格不入。我们特别喜欢玻璃管显示器和扬声器，尽管套用斯波克先生的话，我们认为*号航空母舰企业号*与基于石刀和熊皮的技术格格不入。在积极的一面，VFD 显示器有正确的发光外观，虽然谢妮也很好。

视频没有太多细节，但如果你想尝试类似的东西，博客帖子会有。老实说，1802 系统设计非常简单，因为它的片内 DMA 允许您从交换机加载存储器，而无需监视器等实际软件。电传打字机大约在 1900 年作为雷明顿#7 开始使用，尽管另一台更新的机器捐赠了部件来使一切正常工作。这是一个证明，当时的东西是如何建造的，它采取了尽可能多的滥用，仍然有工作的部分。

我们对 1802 年的设计情有独钟——它在当时是一个非常好的设计。我们甚至到了模拟它的地步。
