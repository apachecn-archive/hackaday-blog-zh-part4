# 2022 科幻大赛:非常星球大战门

> 原文：<https://hackaday.com/2022/04/22/2022-sci-fi-contest-a-very-star-wars-door/>

每一个原《星球大战》三部曲的粉丝都知道汉·索洛的困境，他被帝国军队残忍地冻成了炭。[erv.plecter]得到了一个 Solo 的复制品，这次是用聚氨酯冷冻的，[，并开始用它作为家庭影院系统的门。](https://hackaday.io/project/184860-star-wars-interactive-home-cinema-door)

就像电影中一样，门的一侧有一系列的控制装置和灯，显然是为了表现碳化砖和被困在里面的走私者的状态。这是通过使用 SAMD51 微控制器实现的，该微控制器控制 5 米长的 WS2812B LED 灯条和一个小型有机发光二极管显示器。

还有一个惊人的小烟雾效果，使用 vape 吸入器制作。事实证明，这些设备在各种主题造型和服装中都很受欢迎。它们是在一个小而紧凑的包装中制造可视烟雾的好方法。

[erv.plecter]非常友好地分享了许多关于建造的细节，包括聚氨酯铸件是如何组装到门上的。最终的结果看起来非常真实，肯定会在任何一场《星球大战》电影之夜大受欢迎。只是不要强迫每个人看完*西斯的复仇来破坏事情。*休息后的视频。
