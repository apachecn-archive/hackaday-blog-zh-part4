# 从一个廉价的闪光球吸取一次性设计的教训

> 原文：<https://hackaday.com/2018/10/30/lessons-in-disposable-design-from-a-cheap-blinky-ball/>

有计划的淘汰，当你成为它的受害者时，虽然令人讨厌，但仍然值得钦佩。你不禁会对设计者肃然起敬，他设法优化产品，使其寿命比保修期多一天。说真的，为什么总是第二天？

设计那些永远不会长寿到被淘汰的产品肯定同样具有挑战性，而[electronupdate]拆了一个便宜的 LED 发光玩具来看看其中涉及到什么。毫无疑问，你以前见过这些引发癫痫的硅胶球，大多是在收银台之类的地方，它们的售价是制造它们的价格的几百倍。这个特殊的设备，似乎是这个物种的代表，有两个明亮的发光二极管，一个小的控制器芯片，三个纽扣电池供电，以及一个弹性开关来激活它。所有这些都被安装在一块廉价的酚醛树脂印刷电路板上，控制器芯片和一个发光二极管被一滴透明的环氧树脂覆盖。

当[电子更新]脱掉芯片并把显微镜对准芯片时，这种拆卸比大多数其它的都要简单；下面的视频展示了使用的晶体管数量，并提出了一个可能的电路。这个球的所有东西都很便宜，很可能这些东西建造起来基本上不需要任何成本。这对于一周左右就要被填埋的东西来说是有道理的。

没错，这个烦人的 blinky-thing 是低端垃圾，但还是有设计课可以从中吸取的。任何为广阔市场建造的东西都必须有一个价格点，理解这些限制对于理解计划淘汰是如何工作的很重要。
