# 通风机背后的真实工程

> 原文:[https://hack aday . com/2020/04/07/real-engineering-behind-ventilator/](https://hackaday.com/2020/04/07/real-engineering-behind-ventilators/)

认知专家告诉我们，大多数人认为他们知道的比他们实际知道的多。一个特别的指标是，如果某人是一个领域的专家，他们觉得所有其他领域都与他们相关(例如，一切都可以归结为数学、化学或物理)。这导致他们对自己并不了解的事情过于自信。当人们清楚当前的病毒危机可能导致呼吸机短缺时，许多电子专家开始设计低成本、易于复制的呼吸机。几秒钟挤一次包，到底有多难？但事实证明，要做好这件事，你需要知道很多细节。[真正的工程]和[真正的科学]联手创造了一个优秀的视频，它涵盖了你需要知道的很多东西。你可以看下面的视频。这段视频展示了几个设计，虽然是出于利他主义，但如果用在真正的病人身上，可能弊大于利。

该视频的创作者是一名生物医学工程师，他曾在 Medtronic 工作过——顺便说一下，Medtronic 是一家通风机制造商，最近[开源了他们的一项设计](https://hackaday.com/2020/03/30/professional-ventilator-design-open-sourced-today-by-medtronic/)。他们还让 Rohin Francis 博士]检查视频，他有一个[医疗 YouTube 频道](https://www.youtube.com/c/medlifecrisis)。提供一些屏幕背景，我们学习了一些新的医学术语，发现一家工厂生产的高端呼吸机每周售价约为 225 英镑。他们认为可以增加到 500。

实际上有两个不同的问题。首先，呼吸机的医疗功能当然很重要。在某些情况下，最近几乎所有“紧急设计”呼吸机使用的核心袋实际上可能会对人体的呼吸机制产生不利影响。过多的压力会导致气压伤——由于过压而损伤脆弱的肺细胞。

另一个问题是可靠性。如果我的 Arduino 时钟一天需要重启一到两次，那就不太理想了，但也不会打扰到任何人。如果为我呼吸的机器挂了，那是一个非常严重的问题。

[Real Engineering]认为许多利用通风机设计来吸引公众注意力的商业实体是在盗用麻省理工学院已经存在了几年的成果。设计中有一些微妙的特点，对于那些可能抄袭或试图简化设计的人来说并不总是显而易见的。

如果你想试着做一个简单的紧急呼吸机，你真的应该看看这个视频。它主要集中在医学方面。关于安全关键问题的更多信息，我们也谈到了其他人对这个话题的关注。

我们都赞成[黑呼吸机](https://hackaday.com/2020/03/12/ultimate-medical-hackathon-how-fast-can-we-design-and-deploy-an-open-source-ventilator/)。然而，我们需要确保我们理解这个问题不仅仅是一个机电问题。
