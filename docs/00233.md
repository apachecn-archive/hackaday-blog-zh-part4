# 周五黑客聊天:通过触摸训练机器人

> 原文:[https://hack aday . com/2018/08/01/Friday-hack-chat-training-robots-by-touch/](https://hackaday.com/2018/08/01/friday-hack-chat-training-robots-by-touch/)

说到训练机器人，你可以抓住一个操纵杆，或者用代码仔细编写动作程序。不过，更好的方法是自己移动机器人，让机器人无限回放所有这些动作。这是通过触摸来训练机器人，也是本周 hackaday.io 上的 Hack Chat 的主题。

[![](../Images/1f59f16ccfa2317f723497718b418f06.png)](https://hackaday.com/wp-content/uploads/2018/07/trainingrobotshackchatv1-01.png) 我们本周 Hack Chat 的嘉宾是[Kent Gilson]，发明家、连续创业者、可重构计算的先驱。[Kent]是 Viva 的创始人，Viva 是一种面向对象的编程语言和操作环境，可将 FPGAs 的强大功能运用到通用计算中。他创办了八家企业，赢得了多个奖项，并创造了用于全球众多行业的产品。

[Kent]在 hackaday.io 上成名的是 [Dexter](https://hackaday.io/project/158779-dexter) ，这是一种具有 50 微米可重复性和模块化末端执行器的低成本机械臂。它通过三个谐波驱动器和光学编码器来实现这一点，从而赋予其极高的精度。手臂也是可训练的，这意味着你可以手动控制它，并回放它走过的确切路径。这是通过触摸来训练机器人，正是这个黑客聊天的全部内容。

在这次聊天中，我们将讨论:

*   建造可训练的机器人
*   发展机器人触觉学
*   训练机器人制造
*   异质直接数字制造
