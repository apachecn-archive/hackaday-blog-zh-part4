# 廉价 3D 打印机中不一致图层的简单修复方法

> 原文：<https://hackaday.com/2021/10/17/an-easy-fix-for-inconsistent-layers-in-cheap-3d-printers/>

如果你对 CNC Kitchen 的[Stefan]有一点可以说的话，那就是他在改进 3D 打印流程或解决打印机问题时有条不紊。例证:[这是一台入门级 3D 打印机的挤压不一致的根本原因分析](https://www.cnckitchen.com/blog/fixing-inconsistent-extrusions-systematically)。

问题中的打印机是塞特斯 MK3，这种打印机因其低得离谱的价格和高质量的线性轴承而被许多人使用。不幸的是，打印机仍有许多需要改进的地方，它的图层不一致的倾向是[Stefan]抱怨的主要问题。这种“虚张声势”可以归咎于任何数量的问题，但[Stefan]没有猜测，而是通过系统的排除过程来找到根本原因。我们不会破坏结局，但足以说，这个问题是微妙的，可能是其他打印机类似问题的原因。修复也很容易，而且完全是机械的——只需要更换几个零件。下面的视频展示了整个诊断过程，以及前后对比。[Stefan]还调侃了即将到来的关于他如何从股票所有权控制委员会转换塞特斯的治疗，我们有兴趣看到。

如果你还没有看过[Stefan]的其他 3D 打印视频，你真的应该看看。无论是用混凝土铺路机减振、[盐退火打印增强强度](https://hackaday.com/2020/10/27/take-this-3d-print-post-processing-method-with-a-grain-of-salt/)，还是[利用有限元分析优化填充物](https://hackaday.com/2019/02/06/finite-element-analysis-results-in-smart-infill/)，他总是对 3D 打印感兴趣。
