# 湿软的机器人硬件在压力下表现良好

> 原文：<https://hackaday.com/2021/12/08/squishy-robot-hardware-does-well-under-pressure/>

如果你对 Festo 机器人的嫉妒正在恶化，不要烦恼！[mikey77]向我们展示了，即使没有巨大的存钱罐，我们仍然可以用 3D 打印机和去五金店构建一些奇妙的软机器人项目。要开始，只需通过这个 [3D 打印人造肌肉的过程:安装者在器械上设置](https://www.instructables.com/3D-Printed-Artificial-Muscles-Erector-Set/)项目。

简而言之，[mikey77]慷慨地为我们提供了一个系统，用于设计基于基本关节机制(Omega 肌肉)构建的软体机器人。根据它的名字，这个基本单元包含一个充气膜，随着压力的增加而膨胀，并与另一个欧米茄肌肉协同工作，产生向上和向下的角运动。每块肌肉还包含两个端点，以连接到一个基座、一个抓手或更多ω肌肉。只需根据需要缩放它们，并堆叠它们以产生定制的软机器人肢体，或者使用现有的 STLs 来制作关节式软夹持器。

这个项目实际上分为机器人肌肉和大脑两部分。[mikey77]不仅带我们经历了制造欧米伽肌肉的过程，我们还得到了一份建立压力系统的指南[用来控制它们。总的来说，这是一个功能完整的设置，用于探索您自己的软机器人项目和一个伟大的开始项目。休息之后请继续关注一段演示视频。没有声音，但是我们确信你会满意地发出一声*嘶嘶*的声音。](https://www.instructables.com/Air-Muscles-Make-an-Artificial-Muscle-Robot-Contr/)

我们并不是每天都能看到 FFF 的 3D 打印机制造出需要密封的零件。[Mikey 77]的成功让我们乐观地看到未来项目中更多的空中力量。与此同时，看看我们之前[抓到的抽水铁](https://hackaday.com/2016/06/06/artificial-muscles-to-bring-relief-to-robotic-tenseness/)的硅-硅混血儿。
