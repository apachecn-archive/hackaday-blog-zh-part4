# 业余无线电爱好者接受了大脑移植

> 原文：<https://hackaday.com/2021/11/27/ham-radio-gets-brain-transplant/>

旧收音机没有多少智能。但随着数字合成变得越来越普遍，收音机中的数字电子设备往往和射频电路一样多。问题是数字电子产品每年都变得越来越好，所以今年看起来高科技的东西明年就过时了。[IMSAI Guy]有一个 Icom IC-245，他决定[用一个 Arduino](https://youtu.be/b52L_VcNKxo) 代替里面的数字电子设备。

他在视频的第一部分花了很多时间解释设计需要做什么，你可以在下面看到。Arduino Nano 很适合，他使用一些额外的部件来获得移位寄存器、0-1V 数模转换器和有机发光二极管显示器的接口。

除非你有这个确切的无线电，否则你可能无法直接应用这个项目。尽管如此，当有人设计这样的东西时，特别是当他们边走边解释他们的理由时，从他们的肩膀上看过去是很棒的。

当然，PCB 必须与它所取代的电路板尺寸完全相同，包括安装孔和接口连接器。看起来他第一次就做对了，这并不容易。有用吗？第一段视频结束时我们还不知道。您将不得不观看下一个(也在下面)他实际填充 PCB 和测试一切。
