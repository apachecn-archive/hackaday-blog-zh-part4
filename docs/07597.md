# 你可以在家里搭建一个小小的雅各布梯子

> 原文：<https://hackaday.com/2020/08/29/a-tiny-jacobs-ladder-you-can-build-at-home/>

巨大的嘶嘶电弧很有趣，如果处理得当，也不会太危险。Jacob's Ladder 就是这样一种产生电弧的设备，在老式科幻电影中经常出现。它仍然是高压电原理的迷人演示，[你可以在家里轻松制作自己的迷你版](https://www.instructables.com/id/Mini-High-Voltage-Jacobs-Ladder/)。

构建从一个基本的高压升压套件开始，该套件在输出端将 4V DC 变为 15 kV。在这个电压水平下，有可能在空气中产生电弧。为了制造雅各布的梯子，该套件被连接到一对间距很小的电极上，随着垂直上升，电极之间的距离会慢慢变远。当电弧从一个电极跳到另一个电极时，它电离空气，电压由于电流而下降。流动的电流加热空气，空气开始上升，带着电流路径，造成我们都知道和喜爱的熟悉的攀爬弧。随着电极之间的距离向顶部增加，电弧最终不再持续。没有电流流动时，电压再次上升，在设备底部形成新的电弧，重复这一过程。

这是一个有趣的构建，可以成为一个优秀的桌面玩具，尽管最好在玻璃下展示，以避免意外的电击。如果你愿意的话，你甚至可以用微波部件做一个更大的。休息后的视频。
