# 使用开源工具升级扁平废料

> 原文:[https://hack aday . com/2022/09/09/up cycling-flat-scraps-with-open-source-tools/](https://hackaday.com/2022/09/09/upcycling-flat-scraps-with-open-source-tools/)

如果你有任何像数控铣床或激光切割机那样的切割设备，你可能会产生许多看起来奇怪的废料。我们大多数人都不喜欢扔掉任何东西，但是你打算如何使用这些奇怪的形状呢？[Caddzeus]有一个答案。他用一台相机和一些软件将形状精确地数字化成他选择的 CAD 软件包中可用的形式。

这个过程包括设置一些将出现在照片上的已知尺寸的目标。这使得拍摄照片时不必过分担心与相机的距离或角度。

[![](../Images/af4c68900e43438dcfc59cd858100603.png)](https://hackaday.com/wp-content/uploads/2022/09/scrapimage_detail.jpg) 使用 GIMP，他调整图像以去除透视。然后，他将图像放入 Inkscape，在那里他可以精确地缩放和描绘形状。

有详细的说明——包括视频——但是如果你知道如何使用这些工具，你大概可以自己弄清楚。如果你想得到一个你想要加工的东西的轮廓，或者作为参考切入你的 CAD 程序，这个技术也是有用的。我们喜欢用激光切割机来做这件事，这样你就可以在开始之前在纸上烧出你正在雕刻或切割的东西的轮廓，并轻松地将工件对准激光。

当然，一小部分可以将其图像交给传统的扫描仪，您可以使用类似的技术来调整比例。如果你开始经常使用 Inkscape，你会想要更多的插件。你永远不知道你能从一些[的零碎废料中造出什么东西。](https://hackaday.com/2022/08/30/making-a-pipe-crimper-from-scrap/)
