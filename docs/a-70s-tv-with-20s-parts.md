# 带有 20 年代零件的 70 年代电视机

> 原文：<https://hackaday.com/2022/07/24/a-70s-tv-with-20s-parts/>

随着年龄的增长，保持旧技术的工作变得异常困难。我们大多数人都经历过电容器瘟疫、电线绝缘瓦解、计划报废，甚至是原公司倒闭，不再提供零件或服务。要让一台古董运转起来，通常需要大量的零部件，或者以[Aaron]的 70 年代索尼电视机为例，大量的现代技术让它看起来像是属于一台半个世纪前的机器。

这台电视上的原始反激变压器是这台机器故障的原始原因，获得一个新的变压器需要基本上摧毁一个工作集，所以这是一个 resto-mod 的完美候选人，而不会扰乱任何纯粹主义者。首先，[Aaron]订购了一个带控制装置(和一个遥控器)的 LCD，它几乎可以与现有的边框相匹配，然后开始将现代控制装置与电视上的旧模拟转盘相集成。这意味着使用大量的旋转编码器和编程一个微控制器来做翻译。

在这个版本中还有许多其他细节，包括音频集成，添加 HDMI 等现代视频和音频输入，以及添加 led 以背光照明原来的(现在正在工作的)UHF 和 VHF 频道指示器。在他的 70 年代主题展示墙中，这台电视机看起来非常自然。如果你自己的展示墙突出了一个更古老的时代，[看一些旧收音机的修复图来代替](https://hackaday.com/2014/09/09/welcome-to-the-old-school-restoring-antique-radios/)。
