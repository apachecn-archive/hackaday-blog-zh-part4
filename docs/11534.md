# 疯狂的乐高机器人在你的院子里敲敲打打

> 原文:[https://hack aday . com/2021/10/02/wild-Lego-bot-pronks-about-your-patio/](https://hackaday.com/2021/10/02/wild-lego-bot-pronks-about-your-patio/)

腿式机器人有各种形状和大小。从由研究生组成的团队构建的范式设定四足动物，到一定会娱乐你的初级黑客的 Kickstarter 犬类，入口点非常广泛。然而，[奥拉西德]不是一个简单的旁观者，他想加入建造四足动物的乐趣，并带我们一起去。结果是 5BQE2，一个灵活的预算四足动物,它可以以每秒 1 米的速度在院子里来回摆动。

没有系绳，重量就成了让这种生物以相当快的速度移动的额外费用。让这一切成为可能的部分原因是奥拉西德轻盈的双腿。围绕五连杆机构设计腿部，将原本沉重的致动器从腿部转移到体内，从而使肢体能够更快地运动。更重要的是，5BQE2 是由我们全盛时期的乐高塑料积木制成的。带着一份完整的[材料清单](https://github.com/oracid/5BQE2-5Bars-Quadruped-with-extension/blob/main/5BQE-2-BOM.xlsx)，我们正准备前往父母的车库，掸去那些零件的灰尘，开始第二次人生。

5BQE2 的一些动作镜头，休息后看看视频。由于没有组装说明，任何一套都不完整，请观看完整视频，一步一步地完成组装过程。

在 Hackaday，我们对行走机器人并不陌生，但并不是所有的机器人都用腿行走。沿着记忆之路旅行，看看[卡尔·布格贾]的嗡嗡作响的振动机器人和加州大学伯克利分校的跳跃机器人[萨尔托](https://hackaday.com/2018/10/09/one-legged-jumping-robot-shows-that-control-is-everything/)。
