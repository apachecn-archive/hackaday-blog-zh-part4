# 给自己弄个木偶 _ 为了教育)

> 原文:[https://hack aday . com/2021/07/12/get-yourself-a-pupper-for-education/](https://hackaday.com/2021/07/12/get-yourself-a-pupper-for-education/)

虽然从 DARPA 和其他实验室出来的一些机器人令人不安的外观可能会让我们对我们想要给这些机器人多少智能有所保留，但在它们不可避免的全球接管之前，还有很多东西要向它们学习。这个被称为迷你 Pupper 的小型四足动物正是这项工作的机器人。凭借低成本和熟悉的平台，它是学习一些行业技巧的理想机器人。

对于这么小的四足动物，必须做出一些独特的改变来确保机器人的功能。自从一年多前首次上映以来，已经有了一些进展。首先是设计一个定制的伺服系统，可以处理这个机器人的独特特征。从那时起，对机器人底盘进行了一些其他改进，例如使用螺纹杆，以便于组装和维护。其他一些事情保持不变，但像使用树莓 Pi 来处理控制系统和自我导航。

当然，自己制造这个机器人所需的一切都是开源的，从代码到原理图。对于试验四足动物，甚至自动导航，这将是一个很好的开始方式，并且小尺寸也将限制其天网式接管的能力。那是一笔不错的奖金。
