# 时尚的树莓派相机

> 原文:[https://hack aday . com/2020/07/21/a-style-raspberry-pi-camera/](https://hackaday.com/2020/07/21/a-stylish-raspberry-pi-camera/)

Raspberry Pi HQ 相机模块是一款令人兴奋的产品，它首次将质量接近可更换镜头相机的东西交到了硬件黑客的手中。它已经吸引了那些希望探索相机外形的人的注意。我们在这一领域的最新参与者来自[BBまどーし]，他选择了[，一种非常好的传统紧凑型相机的 3D 打印模拟物](https://hackaday.io/project/173836-turning-your-raspberry-pi-into-a-camera)。

正如你所料，在前面是模块，隐藏在一个智能塑料环后面。在那后面是一个电池盒，隐藏的不是你可能期望的 18650s 的支架或裸露的脂肪袋，而是一个 10，400 mAH 的 USB 电源组。这背后是一个接近传统树莓皮的东西，旨在采用超像素屏幕。电池可能看起来是一个不太冒险的选择，但它凸显了紧凑型电源组的性价比。它可能没有黑客美学，但你不能否认它的成本和简单性。

细节是这个设计有趣的部分，例如，它有一个标准的配件鞋印在它的顶部。还有一个快门按钮，但他们承认自己不是一个足以让它工作的软件向导。也许快速浏览一下 20 世纪 70 年代梅林游戏中的 Pi 相机会更好。
