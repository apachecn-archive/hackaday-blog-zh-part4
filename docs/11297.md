# GLaDOS 语音助手被动-主动自动化家庭

> 原文:[https://hack aday . com/2021/09/13/glados-voice-assistant-passive-aggressive-automates-home/](https://hackaday.com/2021/09/13/glados-voice-assistant-passive-aggressively-automates-home/)

有了现代语音助手，我们可以告诉电脑播放我们最喜欢的音乐、查询天气或开灯。像我们许多人一样，[nerda sic]屈服于谷歌和亚马逊制造的各种家庭自动化产品的便利性和可感知的简单性。也像我们许多人一样，他发现接受围绕这种云连接设备的隐私暗示有点困难。但是在卖掉他的家和 Echo 之后，[nerda sic]错过了通过语音命令控制他的智能家居的能力。[【讷达西奇】没有屈服，也没有买回他留下的封闭生态系统，而是决定向一个凶残、被动、好斗、讽刺、有点疯狂的人工智能开放他的家:GLaDOS](https://www.youtube.com/watch?v=Y3h5tKWqf-w) ，你可以在休息后看到它的行动。

使用来自另一位 YouTube 创造者 Volt 先生]的开源设计，[nerda sic]3d 打印了尽可能多的 GLaDOS 电子动画模型，并实现了相同的硬件使其工作。[nerda sic]使用了更多的开源软件，并创造了一个功能强大但有点有限的家庭人工智能，可以管理他的家庭自动化，提供天气预报，讲笑话等。GLaDOS 在听从[讷达西奇]的命令的同时，也提供了一些受原版门户游戏启发的精彩的一行程序。

Seeed Studio 的一名重新发言人清理了发送到 Raspberry Pi 4 的音频，并允许未来扩展，这将允许 GLaDOS 看向对其说话的人的方向。凭借其红外摄像头，另一项增强功能将允许 GlaDOS 在人们走动时盯着他们看。一点都不恐怖，对吧？[nerda sic]还计划在内部进行语音到文本的处理，而不是当前迭代中使用的谷歌云语音到文本 API，他的[将一切都放在 GitHub](https://github.com/Nerdaxic/GLaDOS-Voice-Assistant)上，这样你也可以有一个邪恶的人工智能挂在你的每句话上。

当然，如果 GLaDOS 的出现还不够，你可以建造一个[功能实体大小的传送门炮塔](https://hackaday.com/2019/08/12/locating-targets-with-charm-courtesy-of-a-life-size-portal-turret/)或者用你自己的[传送门收音机](https://hackaday.com/2018/01/17/recreating-the-radio-from-portal/)收听广播。
