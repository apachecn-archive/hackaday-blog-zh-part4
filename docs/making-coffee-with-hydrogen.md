# 用氢气制作咖啡

> 原文：<https://hackaday.com/2021/10/09/making-coffee-with-hydrogen/>

对于对低碳未来感兴趣的工程师来说，用氢气取代化石燃料气体的想法是一个圣杯。有各种各样的方案，但是它们都有一个问题，那就是氢很难储存或运输。它不容易液化，而且它的分子很小，这意味着许多适合甲烷的容器材料根本就不能容纳它。

[同位素学家]有一个想法:不是以氢而是以金属铝的形式传输能量，并通过与氢氧化钠水溶液反应产生氢。正如你在休息时间下面的视频中看到的，他通过产生足够的氢气来制作一杯咖啡来展示这一点。

这显然是非常成功的，但从绿色角度来看，它是如何实现的呢？原料是铝和氢氧化钠，除了氢气，它还生产铝酸钠。铝是通过电解熔融铝土矿生产的，生产过程中需要消耗大量能源，但由于使用水力发电通常是最经济的，因此它可以成为零碳能源储备。氢氧化钠也是通过电解过程生产的，这次使用盐水作为原料，因此它也有可能通过低碳电力生产。同时，铝酸钠溶液是一种碱性物质，但在环境中容易降解成惰性氧化铝和氢氧化铝。因此，虽然不能保证他使用的原料是低碳的，但这肯定是一种可能性。

因此，有了废铝和各种各样的罐子，就有可能做出一杯热咖啡。很明显，这项技术不会以这种方式在家里使用，但这就意味着它没有用了吗？不难想象，能源作为重但无害的铝金属被远距离运输，我们已经看到[一种具有相同目标的不同化学物质被用于为车辆提供动力](https://hackaday.com/2020/05/26/aluminium-pucks-fuel-hydrogen-trucks/)。
