# USB 网络摄像头缺货？用树莓派和 HQ 相机模块做一个

> 原文：<https://hackaday.com/2020/11/30/usb-webcams-out-of-stock-make-one-with-a-raspberry-pi-and-hq-camera-module/>

越来越多的人在家工作已经对 USB 网络摄像头的成本和可用性产生了影响，所以[杰夫·吉尔林]用一个大约 100 美元的 DIY 解决方案解决了这个问题。它由一个 Raspberry Pi 和 HQ 相机模块组成，充当 USB 网络摄像头，并且没有伪装成相机设备或任何东西的 ffmpeg 在网络上的混乱流。它就像 USB 相机一样工作。

[![](img/27252e413c37d33202084853d5e6a8d8.png)](https://hackaday.com/wp-content/uploads/2020/11/Raspberry-Pi-Zero-is-a-PRO-HQ-webcam-for-less-than-100-3-27-screenshot.png) 【杰夫】为他的单元选择了一个 Raspberry Pi Zero 和 HQ 摄像头模块，制作了一个整洁的包装，它可能不像商业网络摄像头那么小，但肯定是一个非常值得尊敬的 USB 摄像头。话虽如此，但也有一些缺点，即缺少麦克风或自动对焦，更高分辨率下的延迟问题，以及需要彻底关闭 Pi。

[查看 GitHub 库](https://github.com/geerlingguy/pi-webcam)以获得安装所需的一切，包括完整的硬件列表和一些安装选项。[杰夫]还测试了相机是否能与新的[键盘嵌入式树莓 Pi 400](https://hackaday.com/2020/11/02/new-raspberry-pi-400-is-a-computer-in-a-keyboard-for-70/) 一起工作，它绝对能。下面嵌入的是整个项目的视频演练和演示，请查看。
