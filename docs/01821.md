# 切割智慧和塑料

> 原文：<https://hackaday.com/2019/01/14/cutting-wit-and-plastic/>

如果你曾经用手术刀切割过比橡皮擦更硬的东西，你可以使用热刀或者更好的超声波切割器。电锯也能工作，但它们有自己的问题。[这个老托尼]使用一个业余爱好商店的工具来切割一些塑料和木头，然后演示一个商用的[超声波切割机，以显示刀片如何用较少的蛮力穿过](https://www.youtube.com/watch?v=FA46h7Db1BY)。前者需要一些肌肉和技巧，最后还需要一点抗菌剂。休息之后也可以看视频。

这不仅仅是一篇工具评论，[Tony]用螺丝刀把它拆开，并给出了他尖刻的评论。有利的一面是，它可以很好地切割聚苯乙烯，而普通的刀只会划伤或粉碎它。与此同时，在负面类别中，我们没有听到确切的价格，但它们的价格似乎只有他的迷你车床的一半。如果你需要一个估计的投资回报，考虑 2000 X-acto 刀片的价格，但你也可能希望考虑到减少手的老茧。当你在购物的时候，也许还可以考虑一副耳塞；当视频进行到 17:30 时，他试图切割一个陶瓷配件，并设法发出一声震耳欲聋的尖叫。我们警告过你。

这是他的[不成功的尝试](https://hackaday.com/2018/02/27/fail-of-the-week-the-little-ultrasonic-knife-that-couldnt/)将超声波清洁器变成超声波切割器的恰当后续，但我们已经看到了成功的[将洁牙器变成切割器](https://hackaday.com/2019/01/07/making-an-ultrasonic-cutter-for-post-processing-tiny-3d-prints/)。
