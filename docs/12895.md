# 从悬浮滑板到滑板车

> 原文:[https://hackaday.com/2022/02/28/from-hoverboard-to-scooter/](https://hackaday.com/2022/02/28/from-hoverboard-to-scooter/)

我敢肯定，当“悬浮滑板”开始出现时，任何看过《回到未来》的人都会有点失望。它们不会漂浮，对于 12 岁以上的人来说，它们看起来相当可笑。但它们有一个巨大的优势，就是便宜且容易找到。[由疯子制造]分解一块悬浮滑板的零件[来制造一辆令人难以置信的定制电动滑板车](https://www.youtube.com/watch?v=FyYmJzMMRJI)。

分解零件后的第一步是分解轮毂电机。他拔出车轴，开始用车床和铣床加工一个新的车轴。经过一段时间的磨炼，他有了一个坚固的钢轴。磨制出一个盘式制动器的适配器，可以连接到车轮上。TIG 焊机出来用一些铝焊出一个盒子来装电子设备。车轮上焊接了一个支架，上面装有弹簧减震器，有助于平稳行驶。叉子是在车床和砂带机上加工的，但实际的冲击来自一辆旧自行车。要把叉子装到车架上，[疯子]把一块棒料弯成形状；像个疯子一样。车把是从自行车上取下的，车叉被延长到成人的高度。

在巷子里快速试乘显示后震不够强，所以他换了一个强的。所有的零件都涂了一层粉末。在电子方面，它有一个标准的速度控制器和一个定制的电池，由 18650 个 13s6p 配置的电池组成，并捆绑在一起。经过大量的布线后，他试驾了这辆车，我们喜欢看他在雪地里在街上飞驰。

这里的许多零件都加工成压配合公差，然后焊接上去。这其中的技巧、录像和努力是不可思议的。如果你有灵感并且手头没有车床，也许[这款 3d 打印的踏板车可能更适合你](https://hackaday.com/2021/09/15/3d-printed-scooter-zips-around/)。休息后的视频。
