# 从 18000 英尺的高空扔下一架滑翔机

> 原文:[https://hacka day . com/2020/04/17/从 18000 英尺的高空放下滑翔机/](https://hackaday.com/2020/04/17/dropping-a-glider-from-18000-feet/)

[塔里克和凯末尔]心中有一个目标:从高空气球上扔下一架自制的自主滑翔机，并安全返回家园。为了激励他们，[塔里克]决定在到达 18000 英尺之前不理发。鉴于他们项目的雄心，他的头发现在变得相当长也就不足为奇了。

T2】

当他越来越多毛时，[塔里克]正在从几个角度研究这个项目，利用钓鱼线和镍铬合金线创建一个将 2.2 米(7.2 英尺)宽的滑翔机从气球上放下的装置。当该设备到达一定高度时，镍铬合金线被激活并升温，融化了钓鱼线并使滑翔机下降。他一直在低空使用四轴飞行器测试这一点，结果喜忧参半:通常滑翔机以机头朝下的姿态下降，以高速冲向地面。当他们到达 18，000 英尺时，这可能不是什么问题，但从 90 英尺的四轴飞行器上看，这是个问题。

塔里克和 T2【凯末尔对这个项目采取了非常诚实和开放的态度，记录了他们的失误和失败，详细程度不亚于他们的成功。这是一个值得称赞的态度，他们目前已经取得了很大的进展，创造了一个以更稳定的姿态降落滑翔机的系统，并成功着陆了几次。

接下来，他们想开始计划更高高度的发射，这意味着他们需要考虑控制空域，并获得联邦航空局的飞行许可。因此，他们正在寻找一个拥有太空港或类似设施的合作伙伴，他们可能会使用这些设施。有人想帮助他们吗？
