# x 光和高压黑客聊天

> 原文：<https://hackaday.com/2019/02/18/x-rays-and-high-voltage-hack-chat/>

太平洋时间周三中午，请加入我们的 X 射线和高压黑客聊天。

弗兰·皮尔纳斯喜欢挑战别人可能会回避的项目。快速浏览一下他的 [Hackaday.io](https://hackaday.io/Neukyhm) 个人资料，可以发现他最近正在从事的几个令人兴奋的项目，包括[一台 DIY X 光机](https://hackaday.io/project/162928-3d-printed-x-ray-machine)和[运行这台机器所需的高压驱动器](https://hackaday.io/project/162935-piernass-fast-zvs-mazzilli-driver)。不仅如此，他最近将自己自制的 X 射线装置提升到了一个新的高度——一台[计算机断层扫描仪](https://hackaday.io/project/163791-x-ray-ct-scanner)。他的 [YouTube 频道](https://www.youtube.com/channel/UCUqwTU8jrILvUqx2c6qnRXA)也有一些使用潜在致命电压和电离辐射的令人兴奋的东西。

请加入我们的黑客聊天，我们将讨论:

*   如何在高压和电离辐射下安全工作；
*   采购不常见的组件，如 X 射线管；
*   弗兰是如何决定开始在危险区边缘打球的；和
*   他对未来有什么样的想法。

当然，我们鼓励你在讨论中提出自己的问题。你可以在[X 射线和高压黑客聊天](https://hackaday.io/event/163583-x-rays-and-high-voltage-hack-chat)上发表评论，我们会把它放在黑客聊天讨论的队列中。

点击右边的那个发言气泡，你会被直接带到 Hackaday.io 上的黑客聊天群，不用等到周三；随时加入，你可以看到社区在谈论什么。
