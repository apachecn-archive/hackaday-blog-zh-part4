# Remoticon 2021 // Debra Ansell 以你意想不到的方式连接 PCB

> 原文:[https://hack aday . com/2022/01/20/remoticon-2021-debra-ansell-connects-PCB-in-ways-you-not-expect/](https://hackaday.com/2022/01/20/remoticon-2021-debra-ansell-connects-pcb-in-ways-you-didnt-expect/)

“led 改善了一切。”生活的准则。[GeekMomProjects]的 Debra Ansell 制作的几乎所有东西都是明亮、大胆和闪亮的。但是如果你在寻找一串简单的 WS2812s，那你找错人了。在过去几年中，Debra 一直在制造更大、更复杂的组件，这意味着要深入模块化 PCB 的机械设计。在这个过程中，Debra 提出了一些很棒的技术，你可以在自己的构建中使用，她在 2021 年 Hackaday Remoticon 期间的[演示中与我们分享了这些技术。](https://www.youtube.com/watch?v=vzc96or9RLY)

她首先快速概述了 PCB 艺术的发展状况，特别是她最近喜欢的风格:三维结构，其中物理 PCB 本身就是该项目的雕塑元素。她用流行的三角形壁挂雕塑和她自己对“内部发光”的迷恋——侧面发光的丙烯酸散射器——来跨越这个界限。然后她开始带我们走上[创造她自己的墙壁艺术](https://hackaday.com/2021/06/07/triangle-tiles-form-blinky-networks-using-clever-interconnects/)的道路，这是你需要倾听的地方。

例如，用白色阻焊膜制作 PCB 也许是因为你喜欢它的反光特性——比标准的绿色甚至黑色更贵。黛布拉的黑客？丝印免费。在我们看来，黑色面罩上的纯白丝绸就像一块白色 PCB！她为我们分析了材料的零售价格，比较了 PCB 和亚克力。每平方厘米多氯联苯的价格比原丙烯酸树脂贵 50%,但除非丙烯酸树脂的切割和成型是免费的，否则这很可能是白费力气。另外，如果你有*任何*需要安装的电子设备，你也可以选择 PCB。

[![](../Images/dcabf3e1bf31bbfc2eff45ddd8c4fe87.png)](https://hackaday.com/wp-content/uploads/2022/01/connector_contenders.png) 这个演讲的核心是关于一个并不那么高科技的东西，但是对于任何涉及多个工作部件的项目来说，它都是成败的关键:连接器。(看看她对[为什么寻找合适的联系人就像网上约会](https://www.youtube.com/watch?v=vzc96or9RLY&t=16m49s)！)在完成了所有的设计权衡之后，她提出了自己的设计。

剧透警告:定制 PCB 与更多定制 PCB 连接在一起！丙烯酸支架、压力连接器和 PCB 材料的夹层符合所有要求:机械稳定性、预算友好性和美观。

黛布拉最后概述了她最近的工作。您将了解为什么不使用玩偶之家铰链作为可穿戴手镯上的连接器，将 PCB 焊接在一起与使用定制连接器的利弊，如何用定制滑环连接的电路板制作蛇，甚至看到一个全新的设计:她的 Light Tiles 2.0。如果你没有灵感去做一些闪光的东西，或者用多块电路板做一个模块化的结构，我们不知道会发生什么！

如果你想在家跟进的话，黛布拉还上传了演讲的幻灯片。
