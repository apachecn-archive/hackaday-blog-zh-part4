# 3D 打印的滑板卡车出奇的好

> 原文：<https://hackaday.com/2022/07/20/3d-printed-skate-trucks-do-surprisingly-okay/>

如果你能买到现成的东西，很有可能有人已经尝试过 3D 打印他们自己的版本。丹尼尔·诺里(Daniel Norée)在滑板卡车上就做到了这一点，并推出了自己的设计。

![](img/6a525865650a45a80170c99c5d48792d.png)卡车的主体是 3D 打印的，吊架也是。然后使用 195 mm M8 螺杆穿过卡车的中心，以便为安装车轮和轴承本身提供轴。他使用碳纤维增强尼龙 3D 打印了零件，切片器设置为最大化强度。在测试中，它们在附近滚动得很好。

[Mayer Makes]在网上找到了这个设计，并用他自己的透明高抗冲树脂 3D 打印了一些[，制作了一套很酷的透明卡车。这是一种坚韧的材料，](https://www.youtube.com/watch?v=pv8RbB5g9iM)[我们以前在这个网站上介绍过。](https://hackaday.com/2022/05/06/super-tough-resin-is-literally-as-tough-as-nails/)

这些卡车最终落入了[盲文滑板]的手中，他们测试了这些卡车的性能。卡车松了，但在公园里被好好揍了一顿。最终，其中一辆卡车在水泥地上摔了许多跟头后屈服了。

其他伟大的滑板技巧包括[在 3D 打印模具中铸造自己的轮子](https://hackaday.com/2020/12/26/casting-skateboard-wheels-with-a-3d-printed-mold/)。休息后的视频。
