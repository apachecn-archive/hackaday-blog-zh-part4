# 辅助键盘是模块化和实用的

> 原文:[https://hackaday . com/2020/09/07/accessibility-keyboard-is-modular-and-practical/](https://hackaday.com/2020/09/07/accessibility-keyboard-is-modular-and-practical/)

我们还没有从[Dario case]那里得到很多关于他的 K-能力开发套件的细节，但在他的 HackadayPrize2020 条目上有足够的线索，我们可以梳理出关键点。计划是提供一个具有蓝牙 HID 功能的控制模块作为鼠标和键盘。它将插入用户专用板上的插座。每种风格都将适合神经肌肉疾病患者，并允许他们以适合自己需求的方式与计算机互动。例如，如果有人缺乏精细的运动控制，他们可能需要大按钮，而肌肉薄弱的人可能需要彼此靠近的衬垫。从下面的视频来看，原型板并不比纸板和电线更花哨。开发最好的设备并不意味着一打迭代打印或浪费激光切割丙烯酸板。

示例代码支持三种鼠标移动，左、右和下，但有计划开发一种工具来重新编程。鉴于名称和突出的 LCD，我们怀疑将来会有键盘支持。ESP32 的处理和蓝牙功能取决于它的功能，它还支持触摸感应，因此定制的衬垫可以对轻微擦伤或钝拳做出反应。

我们并不缺少定制键盘，从[舒适的光荣元素](https://hackaday.com/2020/05/28/bluetooth-takes-keyboard-from-diy-to-super-fly/)到[令人困惑的打字工具](https://hackaday.com/2020/05/23/more-terrible-keyboards-that-nonetheless-work/)。
