# [Dana Sibera]创造了不存在的设备

> 原文：<https://hackaday.com/2022/11/01/dana-sibera-creates-devices-that-dont-exist/>

在 Twitter 上被称为[[@ nano raptor】](https://twitter.com/NanoRaptor)的【Dana Sibera】让我们对可能出现的设备感到好奇，并对永远不会出现的设备感到担忧——这些设备是由她可敬的照片编辑和 3D 建模技能召唤出来的。有没有想过看一辆侧面装有绿色阴极射线管的 M 型车[？现在你有了。也许，](https://twitter.com/NanoRaptor/status/1446632590253780992)[的一种“自攻”式 DE-9 插头用木螺丝作插销？运气不好，反正就是这样，不过如果实在难以忍受，你可以吃一个口感清洁剂。一年多前，她开始了经典的](https://twitter.com/NanoRaptor/status/1518905163146162176)[“辛辣的枕头，但实际上是枕头”](https://twitter.com/NanoRaptor/status/1417786425169219593)设计，她一直用她想象深处的技术和技术相关物体的描绘来吸引我们。

她展示的东西在现实生活中并不存在，有些遗憾，有些庆幸，但当你在推特上偶然发现一个内置均衡器的立方体形状的 iPod 时，这并不是你脑海中的第一件事。像这样的图片[【卡带 ROM】](https://twitter.com/NanoRaptor/status/1426400262621515777)或[豪华木纹 386DX](https://twitter.com/NanoRaptor/status/1559841940811956224) 在他们是什么是很明显的。另一方面，像这种[“迷你 VGA”加密狗](https://twitter.com/NanoRaptor/status/1567412728050122752)或[琥珀色 CRT 装饰的 TI92 Plus](https://twitter.com/NanoRaptor/status/1483952190654156800) 这样的设备可能会让你在意识到发生了什么之前伸手去拿钱包，并且伴随着照片证明的关于[早期软盘驱动器是基于穿孔卡的](https://twitter.com/NanoRaptor/status/1419093676056186883)的断言可能会让你相信你只是没有跟上你的逆向计算琐事。

众所周知，旧的硬件更具表现力和实验性，而[Dana]的设计充分利用了这一点——从斜切外壳的 SGI Indy】现在配有匹配的显示器，到[的 L 显示屏 Powerbook。](https://twitter.com/NanoRaptor/status/1575103481886621702)新硬件也没有通过——这里是[最新的带点拨轮的 iPod Nano，](https://twitter.com/NanoRaptor/status/1524133952180858882)一个苹果从未达到过的逻辑结论，[一个带有十几个 USB-C 输入的主板](https://twitter.com/NanoRaptor/status/1520564916629151744)代替了 ATX 电源连接器。

有些设计会引起怀旧，有些会让 T2 感到害怕，而【黛娜】会不断给我们带来新的惊喜。观察多产的艺术家是一种乐趣——上次，我们报道了[佩珀对【约书亚·艾林森】、](https://hackaday.com/2022/05/03/blending-peppers-ghost-synths-and-vintage-tvs/)的鬼魂实验，他[一直实验](https://twitter.com/ellingson/status/1583667251575521280)到今天。
