# 从零开始自制操作系统？雪花莲展示了它是如何做到的

> 原文:[https://hackaday . com/2022/06/11/home brew-an-OS-from-scratch-snow drop-shows-how-its-done/](https://hackaday.com/2022/06/11/homebrew-an-os-from-scratch-snowdrop-shows-how-its-done/)

有没有想过推出自己的操作系统需要什么？[Sebastian]的[snow drop OS](http://sebastianmihai.com/snowdrop/)可能会让你对这个过程有所了解，甚至可能会给你一些启发。

[![](../Images/0a31ebf3a6a9d81dad81e4b33eb68904.png)](https://hackaday.com/wp-content/uploads/2022/06/Snowdrop-OS-v29-homebrew-operating-system-from-scratch-in-assembly-language-0-0-screenshot.png) 【塞巴斯蒂安】完全从零开始，只用 x86 汇编语言，创造了 Snowdrop。它不仅仅是一个框架，它拥有许多有用的工具和程序，包括一个基本的解释器和链接器(用于创建独立的基本可执行文件)。)这还没有触及有用的基本要素，比如多任务处理和 GUI 框架。甚至有许多资源专门用于简化游戏开发。因为正如[塞巴斯蒂安]所说，没有游戏的操作系统算什么？

有兴趣试试 Snowdrop，或者偷看一下源代码？[二进制和源代码部分](http://sebastianmihai.com/snowdrop/binaries-sources.html)有你需要的所有内容，页面顶部的其他标题会带你去各种相关的好东西。如果你有几分钟的时间，我们建议你在[的视频之旅](https://www.youtube.com/watch?v=E3uusC4exy0)中观看雪莲花的各种元素和特征的演示(嵌入在分页符之后)。)

Snowdrop 是一个雄心勃勃的项目，但我们对[塞巴斯蒂安]成功了并不感到惊讶；我们以前见过他的低级软件技能，他围绕经典单口机游戏[](https://hackaday.com/2021/06/08/the-epic-saga-of-hacking-knights-of-the-round/)*的骑士所做的梦幻般的努力。*
