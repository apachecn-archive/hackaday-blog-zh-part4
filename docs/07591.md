# 卡丁车免提

> 原文:[https://hackaday.com/2020/09/01/karting-hands-free/](https://hackaday.com/2020/09/01/karting-hands-free/)

我们有些人的电脑鼠标按钮比手指还多，分辨率跟踪比肉眼能辨别的还要精细，还有强制通风的通风口。所有这些特征都预先假定了一件事；用户有一只功能正常的手。[Federico Runco]知道肌萎缩性侧索硬化症(ALS)或卢·格赫里希氏病会剥夺一个人使用标准计算机输入或电动轮椅操纵杆的能力。他正在为 2020 年 Hackaday 奖建造[eye drive](https://hackaday.io/project/166905-eyesdrive-steering-with-eyes-movement)，以恢复 ALS 患者的行动能力。已经有了一些解决方案，但是这个方案只关注一个简短的物料清单。

现有的系统很昂贵，并且经常跟踪瞳孔位置，这可以返回精确的数据，但是 EyesDrive 只能辨别左、右和静止。为此，我们需要三个非侵入式电极、一个带有放大器的定制电路板、信号处理电路和一个微控制器。他在定制的印刷电路板上安装了蓝牙插座，这是主要的通信方式。在下面的视频中，他驾驶一辆虚拟卡丁车在一个棘手的赛道上行驶，以证明他的系统能够胜任城市轮椅的任务。

不应将[费德里科·朗科]的 EyesDrive 与 2015 年黑客大赛的冠军 Eyedrivomatic 相混淆，后者由两位杰出的黑客[史蒂夫·埃文斯](https://hackaday.com/2017/04/24/steve-evans-passes-away-leaves-an-inspiring-legacy/)和[帕特里克·乔伊斯](https://hackaday.com/2017/04/27/assistive-technology-pioneer-patrick-joyce-has-passed-away/)领导。
