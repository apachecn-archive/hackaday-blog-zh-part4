# 用橡子自动化农场

> 原文:[https://hack aday . com/2021/07/01/automate-the-farm-with-acorn/](https://hackaday.com/2021/07/01/automate-the-farm-with-acorn/)

在过去的几年里，农业经历了一场巨大的变革。自第二次世界大战以来，大多数工业化农业依赖于合成肥料、大型机械和种植单一作物的大型农场。现在，越来越多的成功农民逆潮流而动，他们在小农场种植多种作物，使用不需要太多工业的自然施肥方法。当然，即使是这些类型的农场，一些机械仍然是很好的，所以这位农民一直在开发一种开源的自动化农业机器人。

这个机器人被称为 Acorn，是加利福尼亚农场的泰勒的项目。该平台由一个 800 瓦的太阳能电池阵列供电，为一组超级电容器提供能量存储。它使用山地车车轮和装有电动轮毂电机的轮胎，这使它能够四轮驱动和四轮转向，使其即使在泥泞的场地也能行驶。耕作工具，以及任何计算机视觉和自动化硬件，都可以放在太阳能电池板下面。这个原型使用 Nvidia Jetson 模块来处理机器学习和自动化的繁重工作，使用 Raspberry Pi 来处理机器人的基本操作，并可以使用高度精确的 GPS 单元在农场周围导航。

虽然机器人的开发目前正在进行中，但[taylor]希望开发一个社区，建立他们自己的版本，并帮助开发平台。随着越来越多的农民从不可持续的单一作物转向更生态友好的方法，包括多种作物同时种植、碳封存和反季节覆盖作物，这样的农业改进当然是必要的。这当然是一段很长的路要走，但是很多人已经开始努力了。
