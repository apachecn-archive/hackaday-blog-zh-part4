# 忘记智能手表；打造一顶智能帽子

> 原文：<https://hackaday.com/2021/06/21/forget-smart-watch-build-a-smart-hat/>

智能手表在今天相当普遍，但你知道有多少人戴着智能帽子吗？[奥利弗]建造了[威尔逊](https://www.instructables.com/Wilson-the-IoT-Hat/)，他称之为“物联网帽子”我们想知道这个名字的灵感是否来自同名的家装角色，他只是作为一顶帽子出现在围栏线上。你可以在下面看到这个项目的视频。

该项目对于硬件来说非常简单。一个 LED 灯条、一个 Arduino 和一个蓝牙模块。哦。和一顶帽子。如你所料，这个软件有点复杂。它允许你显示短信到你的帽子上。

随着可穿戴设备的发展，你对这个项目的满意程度将取决于什么对你来说是重要的。如果你想看短信，你可能会喜欢戴着手表，因为戴着帽子很难看清。但是，如果你想向世界展示你的想法——和你的手机——这可能正是你想要的。

不考虑实用性，我们认为这是一个有趣的项目，现在面对面的聚会又回来了，这是在下一次超级演唱会上穿的好东西。它看起来肯定比[平视棒球帽](https://hackaday.com/2012/03/21/heads-up-display-mounts-on-brim-of-your-cap/)更好，尽管你可以自己阅读。奇怪的是，我们看到的大多数帽子黑客都与[棒球帽](https://hackaday.com/2019/07/21/a-baseball-cap-that-films-the-past/)有关。
