# 你希望拥有的 Raspberry Pi 便携式控制台

> 原文：<https://hackaday.com/2019/05/16/the-raspberry-pi-portable-console-you-wish-you-had/>

复古游戏机是一个有趣的全能项目。你有电子，机械设计和软件方面的考虑。今年的黑客日奖，全押了。[7.9 英寸显示屏的便携式复古游戏机](https://hackaday.io/project/164930-portable-retro-game-console-with-79-inch-display)是一件艺术品，复古游戏机应有尽有。

出于节省空间的考虑，该构建基于 Raspberry Pi 3 A+而不是 B 模型。屏幕是一块漂亮的 7.9 英寸 IPS 面板，分辨率为 2048 x 1536。立体声 3 W 扬声器播放音乐，8000 毫安时提供 3 到 6 小时的播放时间。

虽然使用 Raspberry Pi 3 进行复古游戏很有趣，但由于更强大的单板计算机，模仿更大更差的控制台有很多机会。Nvidia Jetson Nano 比 Raspberry Pi 3 强大得多，可以想象它可以模仿 N64 和 PlayStation 游戏。[原子 Pi](https://hackaday.com/2019/05/08/this-atomic-pi-eats-other-pis-for-lunch/) ，这台完全不是工业剩余的神奇计算机被重新包装成一台教育计算机，已经被证明可以模仿 N64 游戏。想象一下，从背包里拿出一个便携式游戏机，在公交车上玩 Conker 的坏毛日。哦，这是厚脸皮，但这是可能的，感谢硬件创造者的惊人工作。
