# 停双走

> [https://hacked ay . com/2020/03/03/stop-n-go-double/](https://hackaday.com/2020/03/03/stop-n-go-duplo/)

[besh ur]2 岁的孩子痴迷于交通工具，所以他从垃圾桶里拿起几块 DUPLO 积木，制作了这个玩具交通灯作为生日礼物。嘿，还不如让他习惯交通的现实，对吧？这也给黑客们上了一堂很好的早期课:既然自己能做，为什么还要买呢？

交通模式由位于切割后块内的 Arduino Nano V3 决定。侧面有一个按钮，以防有溢出，灯需要闪烁红色，直到问题得到解决。没有尝试在原位焊接所有东西，冒着熔化塑料的风险，[beshur]在助手的帮助下将 led 和电阻器固定在 Nano 上，然后将所有东西都装入外壳。5 毫米的发光二极管完全适合第二块的钻孔柱，并产生一个漂亮，柔和的光芒。小心行事，休息后再来看看。

当然，塑料积木也可以做真正的工作。这个乐高巧克力缩放仪很可爱。
