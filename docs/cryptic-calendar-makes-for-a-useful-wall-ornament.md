# 神秘的日历是一种有用的墙壁装饰品

> 原文：<https://hackaday.com/2020/10/30/cryptic-calendar-makes-for-a-useful-wall-ornament/>

黑客喜欢好的时钟构造，但它的长期表亲日历，在野外很少见到。无论如何，它们可以是一个有用而优雅的项目，正如这个来自[Wolfspaw]的神秘设计所展示的。

该项目由一系列旋转的轮子组成，展示了一系列神秘的符号。当转盘上的标记与视窗正确对齐时，它们分别显示日期、月份和星期几。车轮本身装有 3D 打印的齿圈，这些齿圈由 Arduino Nano 控制下的步进电机转动。霍尔效应传感器和磁铁用于保持一切适当对齐，而 DS3231 实时时钟处理计时任务。

这是一个整洁的建筑，我们认为神秘的设计增加了一点神秘感，使这成为一个很好的话题。这个版本实际上是我们在之前介绍过的一个项目的混搭，经过缩放和赋予独特的风格，以适应【Wolfspaw】自己的个人审美。休息后的视频。
