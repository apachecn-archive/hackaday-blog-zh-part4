# 通过 Excel 进行三维图形实验

> 原文:[https://hack aday . com/2019/08/29/experiments-in-3d-graphics-via-excel/](https://hackaday.com/2019/08/29/experiments-in-3d-graphics-via-excel/)

3D 图形曾经是大学研究小组和大型专业计算系统的领域。最终被驯服，成为主流。你的手机、平板电脑和家用电脑都完全能够生成移动的 3D 图形。顺便说一下，微软的 Excel 也是如此。

这是[solly]的工作，他已经用这种方式用 Excel 做了很长时间的实验。从[伪 3D 图形](https://github.com/s0lly/Pseudo3DEngineInExcel)开始，项目进展到[真实 3D 引擎的开发。](https://www.youtube.com/watch?v=bFOL9kantXA)自然，事情不能就此打住。下一个合乎逻辑的步骤是前进到光线追踪，这是顺利完成的。[毫无特色的飞机上闪亮的球体在这里是意料之中的事。](https://www.youtube.com/watch?v=m28jJ7CMp8A)

图形必须是基本的，分辨率大约为 256×144。输出是通过改变电子表格中不同单元格的颜色来实现的。相关文件可以在 Github 上找到，对于那些渴望修补自己实验的人来说。我们已经看到其他人尝试过类似的工作，[用【C Bel】为平台](https://hackaday.com/2018/02/18/hack-excel-for-3d-rendering/)编写一个完整的游戏引擎。休息后的视频。
