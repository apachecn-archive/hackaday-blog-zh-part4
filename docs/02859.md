# 由于 CNC 和停止运动，光绘非常精确

> 原文：<https://hackaday.com/2019/05/04/utterly-precise-light-painting-thanks-to-cnc-and-stop-motion/>

光绘是在拍摄长曝光照片的同时移动光线的过程，它从光源的路径上创建了一种绘画。至少从 1900 年代早期到中期，人们就已经以这样或那样的方式做到了这一点，但是现代的硬件和方法已经允许在这个旧想法上进行各种新的旋转。【乔希·谢尔登】[展示了他为一个口香糖广告](https://www.jshel.co/things/light-painting-animations-for-lucy)所做的光绘是多么真实，展示了在 CNC 控制下结合定格动画技术的单个多色 LED 的可能性。其余的魔力来自软件。【乔希】在 Blender 中设计动画，然后路径被导出并用作他自制的*光绘机*的指令。因此，这台机器用灯光和摄像机再现了原始动画，而不是单一的计算机生成的图形。

[Josh]对这种光线绘画并不陌生。在之前，我们已经看过他的神奇机器在工作，我们很高兴他分享了他最新作品背后的细节。下面嵌入的是一个简洁的视频，展示了整个过程，但如果你很着急，只是想看看最终产品，这里有一个[结果](https://youtu.be/_zCRO0HVo0g?t=196)的捷径。

对于那些想了解更多的人来说，GitHub 上有大量关于【Josh】的 [*光绘机*的细节，以及对工作流程和软件的更深入的描述，所以请查看一下。](https://github.com/Defaultio/LightPaintingMachine)
