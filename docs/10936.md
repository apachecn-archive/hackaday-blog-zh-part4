# 检修战斗机器人

> 原文:[https://hackaday.com/2021/08/06/overhauling-a-battle-bot/](https://hackaday.com/2021/08/06/overhauling-a-battle-bot/)

旧的战斗机器人会死在哪里？在车间的~~天空~~角落里的巨大零件箱，它们在那里等待拆卸并在其他项目中使用。但是偶尔，如果一个战斗机器人真的很幸运，它们会被再次取出并恢复正常工作。[Charles]讲述的关于《全面检修 1》的故事也是如此，这是一个机器人浩克，他最后一次出现在战斗状态[是在该剧 2015 年第三季](https://battlebots.com/season-1-robots/)中。

在更新的设计(大修 2 和大修 3)取得成功后，看到一些工作被注入这些老骨头是一个惊喜。它没有毫发无伤地逃离零件箱，因为它的轮子被另一种叫做 sadbot 的设计取代了。现在到位的是“shuffle drive pods”，一种基于凸轮的系统，可以让机器人爬行。休息后在视频中观看它们的动作很有趣，只是要确保先把音量调小。难怪[查尔斯]计划用新设计的车轮模块取代它们。

[![](../Images/9f9bc2f9e8c2f1c98c2937ffdc212727.png)](https://hackaday.com/wp-content/uploads/2021/08/overhaul-1-battle-20-ton-jack.jpg) 在一场激烈的比赛中这些东西受到了很大的损伤，而机修 1 的车架仍然扭曲变形。液压轮胎千斤顶是首选工具，因为损坏是由外部造成的，需要从内部推出。作为这些东西是如何建造的证明，任何旧的千斤顶都不行，为此需要一个 20 吨重的装置。前面的一组叉状物(称为浮桥)也向内弯曲，需要一条链条和一个随从才能把它们拉出来。

多年后重访项目的好处是技术往往会向前发展。我们可以想象[Charles]正在进行的一套新车轮模块的设计工作要容易得多，零件(电机、驱动器、电池等)的质量也比五年前第一次制造时高得多。这是大修系列的第一部分，我们将密切关注。
