# 3D 打印钓鱼诱饵模具

> 原文：<https://hackaday.com/2020/04/10/3d-printing-fishing-lure-molds/>

每个渔夫都有一个秘密。秘密地点、秘密技术或秘密诱饵。也许这就是为什么绑苍蝇这么受欢迎。[史蒂夫]当然有他自己特殊的诱惑，尽管他并不保守秘密。(视频，嵌在下面。)他用 Simplify3D 设计鱼饵，3D 打印模具，然后浇铸。

3D 打印部分很有趣，但看到他用来激发灵感的诱饵和自然猎物也很不错。如果你想抓鱼，你必须使用看起来像真正食物的诱饵。

[史蒂夫]有其他视频，涵盖了更基本的诱惑，但一个涵盖了波浪分裂线乌贼模拟和四尾蛴螬。grub 的模型分为四个部分，每个部分都花了六个小时来打印，所以这不是我们这些注意力不集中的人的项目。

使用中的塑料在 Plastix 品牌“plastisol”上是死的。颜色是一个插件，所以你可以创建不同颜色的诱饵。在视频中，你可以看到视频中的紫色和绿色。结果看起来像是你可能会买的东西，但当然更好。有一个模具需要一些调整，但即使是需要调整的模具，结果看起来也很好。

我们认为这些可以和无人机捕鱼设备一起使用。理论上，你可以用它来进行[遥控冰钓](https://hackaday.com/2012/07/04/the-excitement-of-ice-fishing-now-from-anywhere-in-the-world/)，但是我们不确定你会如何通过遥控来完成它。
