# 旋转拨号键盘是复古手机爱好者的完美恶作剧

> 原文：<https://hackaday.com/2022/12/05/rotary-dial-number-pad-is-the-perfect-prank-for-retro-phone-enthusiast/>

我们不确定你们其他人，但是对我们来说，一个没有数字键盘的键盘在右边看起来有点——赤裸裸的？我们可能不是会计师，但有键盘在那里是令人欣慰的，当你需要比键盘顶部的一排数字键更快地输入数字时，它已经准备好了。

不过，我们可以肯定的是，快速数字键并不是这个旋转式拨号盘数字键盘的全部。事实上，这实际上是一个愚人节恶作剧，捉弄了一个痴迷复古手机的同事，而且效果很好。从一个肯定是库存异常丰富的零件箱中的旧电话拨号盘开始，[Squidgeefish]首先研究出拨号盘与廉价机械键盘接口的电气方面。事实证明，在这些旧表盘中有很多接触反弹，导致一些软件黑客让 Arduino 高兴。

还需要一点黑客技术来将 USB 集线器插入键盘，以及对键盘的 PCB 进行字面上的黑客攻击。一个 3D 打印的外壳可以让旋转转盘放在普通数字小键盘的位置，看起来非常好。我们也喜欢通过用一个巨大的恶作剧键替换整排数字键来解决这个问题。

虽然这都是为了好玩，但这里有一些很酷的技巧，比如在 Dremel 工具中夹一点打印机细丝来搅拌焊接零件。尽管我们之前已经见过参数键帽生成器,但看到它运行起来还是很酷的。
