# 万圣节，骑自行车的骷髅在街上游荡

> 原文：<https://hackaday.com/2022/11/11/bike-riding-skeleton-stalks-the-streets-on-halloween/>

如今，静止的南瓜和静止的骷髅不足以吓唬人。如果你想在万圣节真正制造一种惊吓，你需要一些更有说服力的东西。[这个来自[rc 绝地]的骑自行车的骷髅也许能做到这一点。](https://www.instructables.com/The-Bike-Riding-Skeleton/)

几个巧妙的技巧让这个令人印象深刻的建筑在本质上出奇的简单。推进是通过隐藏在边车中的报废电动滑板车传动系统。这不仅推动钻机，但第三个轮子意味着没有必要做任何花哨的平衡工作来保持自行车直立。转向是通过安装在自行车车架上的一个大而粗的伺服系统来控制车把的。常规钢筋混凝土齿轮处理远程控制的转向和驱动。

骨架本身是现成的，经过修改，关节更加灵活。手连在车把上，脚连在踏板上，所以当自行车在路上行驶时，它看起来像是在踩踏自行车。一个狗骨架骑在边车上，作为一个幽灵般的伙伴。

一具骷髅在小轮车旁的街道上徘徊，这是一个令人毛骨悚然的景象。我们以前也见过一些其他伟大的骨架构建，从[犬科动物](https://hackaday.com/2017/11/21/reanimating-boney-the-robot-dog/)到[音乐剧](https://hackaday.com/2017/11/30/skelly-the-skeleton-is-a-scary-good-musician/)。休息后的视频。
