# 摸脸，灯光追逐，警笛赛跑

> 原文：<https://hackaday.com/2020/04/13/touch-face-lights-chase-sirens-race/>

面对现实吧——人们会摸自己的脸。有时脸会痒，尤其是在过敏季节。但是要戒掉类似的东西，第一步是要认识到你这样做的频率。

[戴上[ Mauricio Martins ]的这款手镯，你的面部接触频率将很快变得明显](http://www.hackaday.io/project/170868)。把它绑在你最喜欢的抓脸手臂上，开始你的一天。该代码不断轮询加速度计，以查看您的手是否在您的面部附近。如果是这样的话，红灯会环绕，紧急车辆式的警笛会响起，让你周围的人知道你犯了罪。

这款免触摸手环非常棒，因为它很简单，而且很有效。它使用用 Make 代码编程的 Circuit Playground Express，但是将它移植到 Arduino 或 CircuitPython 会很容易。如果你想做一些更优雅的东西，我们都支持，但你可以同时用这个来帮助自己摆脱这个习惯。休息后请欣赏演示。

有时候你必须后退一步，做一些不需要花里胡哨就能工作的东西。你听说过一个关于天体物理学家的笑话吗？他把磁铁卡在鼻子里试图解决这个问题。
