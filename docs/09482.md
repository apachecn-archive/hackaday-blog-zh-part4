# 红外滤光器使激光聚焦更容易

> 原文:[https://hack aday . com/2021/03/09/激光对焦-使用红外滤镜更轻松/](https://hackaday.com/2021/03/09/laser-focus-made-easier-with-ir-filter/)

如果你用过二极管激光雕刻机或切割机，你就会知道聚焦是至关重要的。你可能认为获得清晰的焦点会相对简单，但事实并非如此简单。[Makers Mashup]在一个视频中展示了如何使用一个[可调红外滤光器](https://www.youtube.com/watch?v=-RjK8P_ZPDE&list=WL&index=1)来消除所有的光泄漏，以获得清晰的图像，使对焦更简单。

他展示的滤光片可以从 530 纳米调整到 750 纳米，可以拧入 72 毫米的镜头，但它对你的眼球也很好。(Makers Mashup)表示，他最终会为它表明立场，这样他就可以腾出双手来浏览它。

激光不是点光源，焦点也不是尖点。即便如此，在低功率下观察激光显示一个亮点被稍微不太亮的亮点所包围。很难找出精确的最小点。

我们之前已经注意到，使用黑色目标会有所帮助，我们的一副激光护目镜比另一副更好地切断了条纹，因此毫不奇怪，适当调谐的过滤器会使事情变得更容易。

聚焦不好的激光无法以最高分辨率雕刻，切割时会失去动力。这个话题每隔一段时间就会出现。我们甚至看到了对焦，不是用光学方法，而是用弹簧做的。
