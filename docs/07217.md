# 您手机上的 HDMI 显示器

> 原文:[https://hack aday . com/2020/07/18/an-HDMI-monitor-from-your-phone/](https://hackaday.com/2020/07/18/an-hdmi-monitor-from-your-phone/)

数字视频已经发展到这样一个地步，我们已经拥有了接近广播质量的高清制作能力，而且价格往往低得惊人。不过，有一个领域的好处还没有完全体现在我们的钱包里，那就是小型高清显示器，你可以把它放在相机上进行拍摄。这是[Neon Airship]注意到的一个问题，他提出了一个解决方案，允许[使用 Android 手机作为 HDMI 监视器](https://www.youtube.com/watch?v=PZLEUOG9k4Y)。由于我们中的许多人现在都将拥有一部功能完美的老式手机，这是一个很有吸引力的提议，可能成本很低。

secret 并不是最优秀的黑客，因为它使用所有现成的硬件，但有时这并不是对像这样的项目感兴趣的唯一原因。[Neon]正在使用一种 HDMI-to-USB 捕捉卡，这种卡最近以惊人的低价从普通渠道获得。当与合适的 USB OTG 电缆配对时，该适配器可以被手机视为另一个网络摄像头。

我们看到他尝试了几个网络摄像头浏览器应用程序，其中一个相当令人担忧地要求直接从 APK 下载，结果是他的相机上有一个质量非常好的 HDMI 监视器，真的没有破产。有时最简单的解决方案会带来最有用的结果。

这是我们这些试验自己相机外形的人特别感兴趣的事情。
