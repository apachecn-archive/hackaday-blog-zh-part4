# 廉价的 FPV 护目镜变成口袋大小的显示器

> 原文:[https://hack aday . com/2018/09/08/cheap-fpv-goggles-turned-pocket-size-display/](https://hackaday.com/2018/09/08/cheap-fpv-goggles-turned-pocket-sized-display/)

由于第一人称视角(FPV)遥控飞机在过去几年中的爆炸式流行，相关硬件的成本迅速下降。今天，你可以在各种进口网站上以低于 40 美元的价格买到入门级的 FPV 护目镜。花点钱你就能得到一个 5.8 GHz 的接收器、电池和一个液晶显示器；即使组件本身并不高端，在这个价格上，它本质上是一个冲动购买。

[![](../Images/09c2923e058e750c175286d471c8e241.png)](https://hackaday.com/wp-content/uploads/2018/09/minifpv_detail.jpg) 【诺曼德】不一定需要廉价的 FPV 耳机，但他*喜欢*拥有一个口袋大小的显示器的想法，他可以传递给其他人，这样他们就可以看到他在飞行中看到的东西。因此，他[从 Eachine VR006 耳机中收获了主要组件，并为它们设计了一个新的 3D 打印外壳](https://github.com/nomand/EachineVR006)。最终的结果看起来棒极了，而且比市场上的商业替代品便宜得多。

他创建了一个非常详细的分步指南，指导您如何在项目的 GitHub 资源库中自己执行转换，还制作了一个视频，他在视频中回顾了修改并讨论了最终结果。[nomand]很明显，这是一个供其他人复制的项目，而不是一次性的建设，考虑到价格和最终结果，如果这种转换在 FPV 圈子里流行，我们不会感到惊讶。

也许这个项目最好的部分是它几乎不需要修改原始硬件；只焊两根线，因为原来的连接器太大了。否则只需要小心地将耳机拆开，并将组件移植到[nomand]精心设计的 3D 打印外壳中。这个箱子设计得非常好，它甚至不需要任何紧固件，它可以一起滑动，所有东西都用一些有策略地放置的泡沫片固定住。

在这次修改和我们最近报道的定制观众显示器之间，看起来对低于 50 美元的便携式 FPV 显示器有明确的需求。奇怪的是，到目前为止还没有制造商试图填补这一空白。
