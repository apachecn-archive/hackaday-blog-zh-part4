# 模型铁路的转盘

> 原文:[https://hack aday . com/2019/01/11/a-转盘换模型铁路/](https://hackaday.com/2019/01/11/a-turntable-for-model-railroads/)

很久以前，在柴油电力机车出现之前，火车并不能很好地倒退。此外，有时有必要在狭小的空间内调转车厢。为此，发明了铁路转盘。如果你想在模型布局上实现一个，这个来自 DIY &数字铁路世界的项目就是为你准备的。

该项目处于早期阶段——到目前为止，展示了如何使用电位计来控制步进电机的速度，从而旋转转盘，从而设置 Arduino Uno。转盘本身是一个来自 Thingiverse 的 [3D 打印部件，旨在适应所使用的特定步进电机。](https://www.thingiverse.com/thing:933854)

这就解决了简单的部分——将一段轨道旋转 360 度，以正确定位列车。然而，前面还有大量的工作要做。电源需要连接到轨道上，并且需要设计一个系统来精确地将转盘与输出轨道对齐。这对于 N 规设置尤其重要，因为公差就是一切。

我们很想知道你会如何解决各种问题，以建立一个工作模型转盘在评论中。我们以前在这些地方看到过一些严肃的铁路模型。休息后的视频。
