# 树莓乌鸦 Pi 安全摄像头有双重功能

> 原文：<https://hackaday.com/2020/11/03/raspberry-raven-pi-security-camera-does-double-duty/>

节日装饰品最糟糕的一点是，虽然你可以一整年都不放，但你的邻居可能不会因此而非常喜欢你。你家的圣诞灯是一回事，但就万圣节装饰而言，[【MisterM】的乌鸦安全摄像头](https://www.instructables.com/Raven-Pi-Security-Camera/)是这一规则的少数例外之一。

T2 永不超生会想知道谁去了那里。一旦这只乌鸦把它的珠状红色 LED 眼睛放在花园里潜伏的任何东西上，它就会转动一下头，发出一些随机的声音。这只鸟要么哇哇叫！或者引用克里斯托弗·李对埃德加·爱伦·坡的《乌鸦》的解读。

这只鸟的胸腔内有一个树莓 Pi 2 和标准摄像头，一个旋转头部的伺服系统，一个音频放大器和扬声器。这只鸟在 Raspi 操作系统上运行 MotionEye，因此它可以在感应到运动时运行脚本。

我们喜欢[MisterM]能够找到大小合适的塑料片来安装颈部的伺服系统和头部的喇叭。这只是表明，并不是所有东西都需要 3D 打印机、数控机床或木工。休息之后看看这个恐怖的演示。
