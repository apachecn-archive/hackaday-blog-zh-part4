# 用热胶枪注射成型

> 原文：<https://hackaday.com/2019/06/11/injection-molding-with-a-hot-glue-gun/>

注射成型是一种工业过程，在世界范围内用于快速和经济地生产塑料零件。[Nikodem Bartnik]想在家里试验一下这个，[于是他做了一些模具并开始工作(Youtube 链接，嵌入在下面)](https://youtu.be/ZXzZvT6VH4I)。

[Nikodem]使用基于 Dremel 的 CNC 平台生产铝模具。这使得设计可以在 CAD 软件中创建，并有助于零件以及浇口和通风口的几何图形的制作。通过早期的尝试了解了热问题后，模具被夹在老虎钳中。木头被用作绝缘体，以减少热量向虎钳的散失。

有了这种装置，就可以用热胶模制出表面光洁度好的 M5 垫圈。后来，由于胶水在穿过整个模具之前就冷却了，所以用更大的模具的尝试没有成功。[Nikodem]已经决定改进他的设置，我们期待看到接下来会发生什么。我们之前也看到过其他人在这方面进行实验。休息后的视频。
