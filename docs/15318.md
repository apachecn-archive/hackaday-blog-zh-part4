# 旧的 6809 计算机再次生活在试验板上

> 原文:[https://hacka day . com/2022/11/09/old-6809-computer-lifes-again-on-bread boards/](https://hackaday.com/2022/11/09/old-6809-computer-lives-again-on-breadboards/)

在旧的 CPU 中，6809 从来没有像它的一些兄弟一样受到如此多的关注。无线电室 CoCo 使用了它，无线世界杂志上的一篇建筑文章也使用了它。现在[戴夫]已经在试验板上重建了那台电脑，它看起来棒极了。文件在 GitHub 上，甚至还有一系列关于这台机器的视频。可以看下面第一个。

你甚至可以阅读 1981 年 1 月无线世界的原始文章，在那里主板使用了 6802。a 6809 的升级版出现在 1981 年 7 月号的[上。杂志承诺你可以用 100 美元建立这个系统。除了 6809，只有几个芯片。一个 PROM、两个 RAM 芯片、一个 6821 PIA 和一个用于地址选择的 74LS138 解码器。MC1413 晶体管阵列还允许 7 段显示器和键盘以及 7442 BCD 解码器。](https://worldradiohistory.com/UK/Wireless-World/80s/Wireless-World-1981-07.pdf)

显然[戴夫]早在 80 年代就已经开发了一台类似的计算机，并对其进行了修改，以适应无线世界的项目内存图。听起来好像他没有完成，但他找到了旧木板，并决定在试验板上重新制作。

像当时的许多计算机一样，这台机器有一个盒式接口。我们真的很喜欢 7 段 led 灯的美感和整体外观。

6809 确实在一些特定的工业和视频游戏应用中得到了应用。还有一台基于 6809 的新西兰[教育计算机，以及其他一些家用计算机，如 SuperPET 和 Dragon。](https://hackaday.com/2017/01/17/new-caps-and-ram-save-another-poly-1/)
