# 建造你自己的潜水艇

> 原文：<https://hackaday.com/2021/12/10/build-your-own-submarine/>

如果你尝试建造会飞的东西，为什么不像 DIYPerks 那样尝试建造潜水艇呢？正如你在下面的视频中看到的，关键是控制浮力，使用的机制令人印象深刻。潜艇前后有两个巨大的注射器来压缩或解压水。柱塞现在是 3D 打印的致动器，在丝杠上运行。两个高扭矩电机和一些夹在丙烯酸盘中的电池组成了其余部分。这是一个大容器——你不会在你的浴缸里尝试，甚至不会在你的游泳池里尝试，除非它是一个大容器。

当然，一切都需要滴水不漏。这款潜艇没有尝试对电源开关进行防水处理，而是使用了一个簧片开关，这样附近的磁铁就可以打开它。不是什么原创的想法，但我们一直认为它比密封和灌封化合物更优雅。

不过，最好的部分是修改伺服控制板，用线性滑动电阻代替旋转电位计来闭合环路。一个聪明的黑客，似乎很适合这个应用程序。缺点是射频控制。正如真正的潜艇所知，2.4 GHz 信号在水下传播不了多远。像一艘真正的潜艇一样，这艘潜艇在水面上漂浮着一个天线阵列，这样它就可以从岸上接收控制信号。它还能实时上传视频。

一堆水泵作为推进器工作。潜艇的船体是一个直径与灌溉管道相同的管子，因此潜艇可以使用现成的部件来实现这一目的。

海上试验是在一个湖上进行的，有一个朋友的小得多的潜艇。有一些常见的问题需要解决，但最终，添加一些压舱物和预防性添加一些密封使一切正常工作。

最近我们已经看到了几个替补队员。这其中的一些让我们想起了 T2 的 Subnautica。
