# 路易吉的豪宅第一人称模式带来了怪异的新观点

> 原文：<https://hackaday.com/2022/10/28/luigis-mansion-first-person-mod-brings-spooky-new-perspective/>

任天堂 GameCube 在很多方面都出乎意料。它是紫色的，有豆子形状的按钮，而且它不是用马里奥游戏启动的。取而代之的是由马里奥的哥哥*路易吉的豪宅*主演的恐怖相邻的幽灵冒险游戏。这个游戏在当时是一个图形展示品，然而，摄像机角度都是固定的，就像早期的*生化危机*游戏一样。不满足于在这些范围内玩，modder [Sky Bluigi]为游戏创建了[第一人称摄像机补丁，最终让玩家看到为什么 luigi 总是如此惊慌失措。](https://www.romhacking.net/hacks/6460/)

这个被命名为*路易吉的豪宅 FPO* (第一人称优化)的补丁在很大程度上体现了游戏对儿童友好、怪异的审美。除了用新镜头探索环境的能力，它还提供了手动打开和关闭手电筒的能力。尽管*路易吉的豪宅 FPO* 最令人印象深刻的部分是它运行在真正的硬件上。玩这个 mod 需要的只是北美发布的 *Luigi's Mansion* 的清晰图像和一个像 Delta Patcher 这样的. xdeta[补丁实用程序。GameCube 游戏可以使用运行 Clean Rip 或类似备份工具的软调制任天堂 Wii 控制台直接抓取到 USB 拇指驱动器上。](https://www.romhacking.net/utilities/704/)

Luigi's Mansion FPO 实际上提供了一系列补丁，根据使用的补丁，这些补丁提供了修改后的控制和增加的视野。最初的游戏有反向控制瞄准 Luigi 的幽灵真空，所以“反向 C-Stick 控制”补丁将安装一个更现代的瞄准方案，在右边的棍子上向上瞄准，反之亦然。“更好的 FOV”将镜头拉得离路易吉的头部更远一点，同时保留了最初的瞄准方案。虽然不管你决定用哪个补丁，像这样的一个 mod 总是一个重温经典的好借口。
