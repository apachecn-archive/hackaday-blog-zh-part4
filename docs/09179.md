# 由于 Gigaleak，超级马里奥原声音轨获得高质量的重制

> 原文：<https://hackaday.com/2021/02/06/super-mario-original-sound-tracks-get-high-quality-remaster-thanks-to-gigaleak/>

2020 年，许多千兆字节的任天堂内部数据在更广泛的互联网上泄露。被称为“Gigaleak”的它包含了许多游戏的源代码和资产。利用这次泄露的数据，[一群狂热者已经将 SNES 超级马里奥世界原声音轨(OST)](https://twitter.com/lebrickster/status/1356013269513863171) 进行了高质量的翻唱。

当在泄露的数据中发现《超级马里奥世界》的 Gameboy Advance 翻拍版的源代码时，这项工作才成为可能。源代码包括样本的名称，这些名称与最初的 SNES 游戏中使用的名称相同。这使得该团队能够在数千兆字节的泄露文件中找到原始样本。

我们想知道那些代码会有什么效果，推测这将是模拟器场景的一颗毒丸。这种类型的黑客甚至没有在我们的雷达上，我们很高兴看到这个项目曝光。复制的歌曲与 SNES 的原声音乐有着完全不同的质量。这在很大程度上是因为样本不必被压缩或剪切以适合于一个卡带，并与控制台的声音芯片一起工作。声音中的其他差异也来自于这样一个事实，即与游戏中不同，这些重现中的样本与原始游戏中的播放长度不匹配。

不管变化如何，听到这些经典视频游戏曲调更完整、更圆润的声音是很有趣的。这多少让我们想起了后来的 CD 控制台时代，那时音响设计师能够摆脱早期硬件的限制。当然，我们仍然对 chiptune 的改变俯首称臣——如果你好奇的话，这个 MIDI Gameboy mod 是一个很好的起点。休息后的视频。
