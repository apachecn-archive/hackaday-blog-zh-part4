# CD 播放器供电的 555 钢琴去手风琴计划

> 原文：<https://hackaday.com/2021/12/30/cd-player-powered-555-piano-goes-accordion-to-plan/>

啊对了，555 钢琴项目。无论是 Atari Punk 控制台，还是其他一些基于 555 的合成器，黑客都喜欢听 555 在连接几个无源器件和一个扬声器时可以做什么。这是一种值得关注的声音。但是对伯纳来说，这还不够！在幕间休息之下，你可以看到他的作品，名为 Acordeonador。

Acordeonador 是西班牙语中“手风琴”和“发电机”的组合词，它做了迄今为止我们还没有见过的项目能做的事情:它将 CD 驱动器变成基于 555 的合成器的发电机。

为了给音乐伴奏者一种更加模拟的感觉，一个 4700uf 的大型电解电容器储存的能量刚好足以让音乐的产生不仅仅是开/关的事情。效果很棒，而且效果很好！作为一个不遗漏任何细节的人，[Berna]在 perf 板上制作了原型，然后在板上覆盖了一层似乎是木纹的接触纸，给了它 20 世纪 70 年代双键盘电子琴的感觉。

这真的证明了 555 项目可以成为美好时光的源泉！Hackaday 有 555 个项目，但如果你喜欢这个，一定要看看宇宙中最重要的设备，它当然是由 555 驱动的。
