# 不要走过这个 3D 打印的人行横道灯

> 原文:[https://hack aday . com/2021/12/01/don-walk-past-this-3d-printed-人行横道灯/](https://hackaday.com/2021/12/01/dont-walk-past-this-3d-printed-pedestrian-crossing-light/)

按比例缩小的电子复制品令人愉悦，这个由[sjm4306]制作的[可爱的 3D 打印行人过街灯](https://hackaday.io/project/182720-3d-printed-mini-pedestrian-crossing-light)也不例外。

虽然比现实世界中的同类产品小一点，但由于在美国无处不在的使用，这种行人灯上的亮黄色外壳和发光指示器很容易辨认。这些打印出来的零件仅仅通过摩擦力结合在一起，这使得组装变得轻而易举。具有许多角度的“安全格栅”最终成为构建过程中最乏味的部分之一，但这种努力绝对是合理的，因为没有它看起来就不太对。

一个合适的微型 ATtiny85 驱动一对 LED 灯条，有效地模仿人们熟悉的“行走”和“不行走”的符号。[sjm4306]在设计电路板和外壳时考虑到了各种选项。例如，如果你想在旅途中给这个装置供电，那么只有足够的空间来塞进一个薄电池。如果你手头没有 ATtiny85，该板还支持 ATmega328p 甚至 ESP8266。

所有的构建细节都可以在 [Hackaday.io](https://hackaday.io/project/182720-3d-printed-mini-pedestrian-crossing-light) 上找到。虽然它被宣传为“夜灯”，但我们认为这可能是一个办公室玩具的绝佳平台，类似于这个[办公室状态灯项目](https://hackaday.com/2020/06/13/office-status-light-turns-do-not-disturb-to-busy/)。或者如果你已经有了一个全尺寸的步行灯，为什么不[把它连到互联网上呢？](https://hackaday.com/2014/07/26/control-this-pedestrian-walk-signal-online/)
