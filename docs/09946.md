# 逻辑流，字面上，在这个水加法器

> 原文:[https://hack aday . com/2021/04/26/logic-flows-literal-in-this-water-adder/](https://hackaday.com/2021/04/26/logic-flows-literally-in-this-water-adder/)

许多基本的电子文本用水来比喻电。要知道，压力是电压，流量是电流，管径是电阻。具有讽刺意味的是，有些人用液体来建造逻辑门，而事实上，你可以只用流过一些结构的水来建造任何你喜欢的逻辑电路。不这么认为？看看下面[史蒂夫·莫尔德]的视频。

流体逻辑并不是什么新东西，但它总是有点奇特。通常，用水甚至空气——这是一种流体——代替电子意味着你试图在一个艰苦的环境中工作或有其他一些特殊的需要。据我们所知，[史蒂夫]这样做只是因为他可以，我们明白。

历史上，流体逻辑使用小型压力射流来改变流体流动的方向，这是美国陆军在 1957 年发现的。然而，史蒂夫使用一些依靠水位和虹吸作用工作的电池。他也有一些不实用的逻辑门，非常能说明问题，即使它们工作得不是很好。

第一次尝试有点失败，但所有问题最终都解决了。当然，两个四位数字相加和一台完整的计算机相差甚远，但你不得不承认，这是一个开始。旧计算机使用水星中的声波进行存储，因此使用这种类型的结构来构建复杂的东西，比如说 [EDSAC](https://hackaday.com/2018/09/28/retrotechtacular-heres-how-they-programmed-the-edsac-computer/) ，将是一个有趣的配对。

事实证明，如果你开始观察，[许多东西](https://hackaday.com/2017/01/03/make-logic-gates-out-of-almost-anything/)可以形成逻辑门，甚至[多米诺骨牌](https://hackaday.com/2015/05/07/learning-logic-gates-with-dominos/)。你不得不怀疑古埃及人是否有这样的想法，他们是否可以用砖块做一台流体计算机，并用尼罗河为其提供动力？可能不比金字塔难，你只能想知道他们会用它做什么。
