# 灯闪烁模块非常适合汽车使用

> 原文：<https://hackaday.com/2022/10/01/lamp-flashing-module-is-perfect-for-automotive-use/>

现代汽车往往有相当先进的照明系统，全部集成在汽车电脑的控制之下。然而，在过去，像刹车灯和指示器之类的东西都是用模拟电子设备完成的。如果你的老爷车需要一个很好的老式闪光器模块，你可能会发现这个来自[DIY Guy Chris]的构建很有用。

这是一个全模拟构建，不需要微控制器或其他先进的现代发明。相反，一个小的双极 PNP 晶体管和一个更强大的 NPN MOSFET 作为振荡器，对电容器进行充电和放电，以产生所需的闪烁行为。改变主电容的大小会改变闪光频率。选择 MOSFET 是因为运行 12 伏灯泡需要相当大的电流。所绘制的设计旨在运行多达八个典型的汽车灯泡，如您可能会在指示灯中找到的灯泡。然而，[Chris]只用四个演示了电路。

闪光器电路在 20 世纪 90 年代被广泛使用。最初的[马自达 Miata](https://hackaday.com/2021/06/27/miata-sci-fi-digital-dash/) 有一个非常相似的电路藏在仪表板下，用来控制转向灯。对于旧车来说，这些电路可能很难找到，所以如果你发现零件很难得到，建立自己的电路可能是一个有用的变通办法。休息后的视频。
