# 重力-水-空气动力发动机

> 原文：<https://hackaday.com/2022/03/28/gravity-water-air-powered-engine/>

在 Hackaday，空气发动机是常见的事情。它们相对新颖，而且相当容易 3D 打印，不需要任何液体或支持机械。例如，[汤姆·斯坦顿]采用了以前的空气发动机设计，去掉了空气压缩机，[改为利用重力和水产生几磅/平方英寸的压力来驱动发动机](https://www.youtube.com/watch?v=1Zf_2CZV6Zw)。

基本的设置是在高处放一大壶水。软管向下延伸至[Tom's]定制的丙烯酸压力舱。一点 cncing 和一些环氧树脂做了一个固体室，我们很高兴地报告，[汤姆]在建造前做了一些初步模拟，以确保他不是意外地建造了一个炸弹。餐巾纸背面的数学计算显示，他的设置可以达到大约 0.6 巴(大约 8 磅/平方英寸)。他的第一次测试几乎完全证明了这一点。不幸的是，尽管早期取得了成功，[Tom]还是遇到了一些问题。他的引擎会因为吸入空气和压力下降而停止，压力的补充速度受到他钻的相对较小的入口孔的限制。

为了解决这个问题，他[为引擎](https://hackaday.com/2020/12/02/diaphragm-air-engine/)打印了一个更大的隔膜，因此较低的气压有更多的推力。这使得发动机在油箱加满之前可以运转一段时间。此外，他平滑和抛光一切，所以它是尽可能低的摩擦。我们知道我们经常在这里陈述，但这是令人难以置信的是，这些天 3D 打印部件可以实现什么。

我们喜欢在这个视频中看到明显的迭代。桌子上展示的各种引擎版本提供了一个关于[汤姆]坚持不懈的有力故事。给发动机提供动力是给整个家庭提供动力的一小步。
