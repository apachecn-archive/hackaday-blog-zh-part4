# DIY 闪烁探测器非常灵敏

> 原文：<https://hackaday.com/2019/07/19/diy-scintillation-detector-is-mighty-sensitive/>

盖革计数器是一个受欢迎的黑客项目，如果核灾难来临时，它可能会被证明是有用的。然而，它们并不是检测辐射的唯一技术。闪烁探测器是完成这项工作的另一种方法，[，[亚历克斯·伦古]已经建造了他自己的一种。](https://hackaday.io/project/166505-diy-scintillation-detector)

与更常见的盖革-弥勒计数器相比，闪烁探测器有几个优点。它们通过使用在电离辐射下发光或闪烁的晶体来工作。然后，这种光被传送到光电倍增管，光电倍增管相应地发出一连串的电子。这个信号代表检测到的放射性水平。它们对少量辐射更敏感，比盖革-弥勒管对伽马辐射更敏感。然而，它们通常被认为更难使用，建造成本更高。

[Alex]的构建使用了一个 2 英寸的碘化钠闪烁体，并结合了一个他在跳蚤市场以低价购得的廉价光电倍增管。[Jim Williams]的高电压、低噪音电源用于运行电子管，所有这些都包裹在一个整洁的 3D 打印外壳中。通过设备背面的 BNC 连接器输出。

测试表明，该设计是可行的，并且比预期的盖革-米勒计数器灵敏得多。如果你对精确测量少量辐射感兴趣，这可能是你的选择。我们已经看到这种技术也用于伽马射线光谱学。
