# 磁铁让电子纺织品的原型制作变得轻而易举

> 原文：<https://hackaday.com/2020/02/26/magnets-make-prototyping-e-textiles-a-snap/>

你如何制作电子纺织品的原型？只要不把你逼疯或者不浪费宝贵的导电线。我们想象不出比这个名副其实的 [ThreadBoard](https://www.instructables.com/id/The-ThreadBoard-Microbit-E-Textile-Prototyping-Boa/) 更简单的测试可穿戴设备的方法。

如果你从来没有玩过电子纺织品，它们的原型会非常复杂。当然，铜线也是松软的，但是如果你弯曲它们，至少它们会成形。导电线只是想躺在那里，柔软而展开，用它卷曲的末端嘲笑你疲惫的状态。线板的神奇之处在于磁性连接点，无论你把线挂在哪里，它都能把线扣到位。

板本身是由硬毡制成的，孔可以激光切割或冲压，以适应您的磁盘。这些吸引人的连接点用毛毡背面的胶带固定，尽管经典的双面胶也可以。我们希望看到有人用电源和接地轨制作更大的电路板，甚至在衬衫上制作可穿戴的线板。

尽管[chrishillcs]正在用 micro:bit 进行演示，但任何有大孔的电路板都应该可以工作，他计划在未来进行扩展。现在，把针埋起来，休息一会儿，看着[克里斯]建立一个电路，点亮一个 LED，比你说钕的速度还快。

电子纺织品的复杂乐趣并不仅限于原型制作——实现最终产品可能要困难得多。如果你需要绝对平行的线而没有太多的麻烦，在你的缝纫机上放一个锁线脚。
