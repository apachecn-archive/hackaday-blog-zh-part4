# 任天堂的纸板钢琴成为真正的工作乐器

> 原文:[https://hackaday . com/2019/05/06/nintendos-cardboard-piano-becoms-a-real-working-instrument/](https://hackaday.com/2019/05/06/nintendos-cardboard-piano-becomes-a-real-working-instrument/)

任天堂的 LABO 钢琴是一种奇怪的乐器。它由纸板制成，并与一些先进的软件结合使用，完全是被动的，所有的声音产生和智能都来自插入机身的开关控制台。[喜鹊西蒙]认为这根本不行，[并着手将 LABO 钢琴变成一个真正的合成器(Youtube 链接，嵌入下文)。](https://www.youtube.com/watch?v=0CjLOElL0rY)

为了实现这一壮举，[西蒙]采购了一个 [OKAY synth 套件](https://www.oskitone.com/product/okay-synth-diy-kit)——一个基本的单声道合成器，旨在安装在 3D 打印的盒子里。相反，它建在 LABO 宽敞的纸板房里。键盘用胶带加固，并调整以接受那些常见而可怕的红色 SPST 按钮，前面板装有控制转盘，通常是开关所在的位置。

经过精心制作，这架钢琴已经可以摇摆了。这不是最灵敏的仪器，柔性纸板难以可靠地触发安装的按钮，但它确实有效。[Simon]在鼓轨上演奏了一首小的器乐作品，以证明你不需要任天堂 Switch 就能在 LABO 钢琴上找到乐趣。

预计在未来几年的 chiptune 展会上，将会看到类似的建筑——至少在 mold 战胜它们之前。还有其他方法可以破解 LABO 钢琴。休息后的视频。
