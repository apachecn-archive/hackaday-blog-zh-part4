# 36 C3:Phyphox–使用智能手机传感器进行物理实验

> 原文：<https://hackaday.com/2019/12/29/36c3-phyphox-using-smartphone-sensors-for-physics-experiments/>

众所周知，如今的普通智能手机都有大量适合放在口袋里的小配件，几十年前，这些小配件可以轻松装满一辆汽车的后备箱。在这里，我们喜欢考虑摄像机、音乐播放设备，甚至电话，但我们不要忽视如今我们随身携带的微型传感器形式的测量设备的数量。[Sebastian Staacks]在 36 C3 关于[phy phox 移动实验室](https://phyphox.org/)应用的演讲中介绍了如何将这些传感器用于教育目的来教授物理。

虽然访问移动设备的传感器数据通常可以通过一些 API 调用非常直接地完成，但 phyphox 应用程序不仅是在屏幕上很好地绘制所有可用传感器数据的快捷方式，它还可以导出数据，以便以后进行额外的可视化和处理。附带的*实验编辑器*允许定义从数据捕获到分析的自定义实验，这些实验以基于 XML 的文件格式存储，并可能通过 QR 码共享。

除了演示应用程序本身，如果你想知道手机中的加速度计、磁力计或气压传感器等传感器实际上是如何工作的，你可以使用其中的哪一个来检测飞机上的厕所冲水和测量电梯速度，以及如何验证你的硬盘正确旋转，你会喜欢这个演讲。如果你只是想为自己摆弄传感器数据打下一个良好的基础，这些数据都是开源的，在 GitHub 上可用于 Android 和 T2 iOS。
