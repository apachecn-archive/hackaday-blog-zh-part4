# 这款电脑鼠标内置一台鼠标电脑

> 原文：<https://hackaday.com/2019/01/17/this-computer-mouse-houses-a-mouse-computer/>

大家之前都听说过电脑鼠标，那么[鼠标电脑呢？](https://www.youtube.com/watch?v=bLjFe7Lzjs8)

诚然，[电子手榴弹]的一体化电脑在一个超大的鼠标形状的情况下几乎没有实用价值。但这不是重点，这只是为了做一些很酷的事情。灵感来自塞满树莓皮的键盘，以制造一台几乎是多功能一体的机器；这种不寻常大小的啮齿动物是下一个合乎逻辑的步骤。在 3D 打印的外壳中有一个 Pi Zero W 和一个 LiPo 电池以及一个鼠标机构——唉，目前市场上没有真正的鼠标可以容纳一切——这款计算机不仅有一个几乎可用的微型液晶显示屏，还有一个滑出式蓝牙键盘。键盘与显示屏成直角的人体工程学设计让我们犹豫，但可用性并不是重点。不要对表演部门抱太大期望——在演奏*《我的世界》*几秒钟后，钻机就会呕吐。

尽管有其局限性，这种鼠标电脑还是有一定的魅力。我们总是喜欢“仅仅因为我能”的项目，不管它们是游戏机尤克里里还是喷火的电子鸭子 T2。这类项目的价值通常不在于它们生产了什么，而在于它们推进了前人未曾涉足的领域。
