# 一架 136 欧元的脉冲喷气式飞机来点无烟火锁定的乐趣

> 原文：<https://hackaday.com/2021/01/15/a-136-euro-pulse-jet-for-some-no-firework-lockdown-fun/>

为了减少疫情期间因集会导致的病毒传播，荷兰政府已经禁止燃放烟花。荷兰人民喜欢他们吵闹的东西，所以我们被告知禁令被广泛藐视，但[建立漫画]是一群守法的车间工具英雄。对于他们的锁定噪音，他们创造了一个完全合法的脉冲喷射。有趣的部分是，它完全是用相当基本的工具在最低限度的预算下制造的，TIG 和 MIG 避开了普通的电焊机。

脉冲射流的形式可能是人们熟悉的，因为它取自其他已发表的设计。一根长管向后弯曲，在其一个臂上放置一个燃烧室，这样射流形成一个共振室，产生连续的废气脉冲。这个是由不锈钢管制成的，详尽的文档应该值得任何想自己制作的人一看。用电焊机焊接薄板需要相当多的技巧，在一些地方他们能烧出一两个洞。其中一个需要一个补丁，但历史悠久的沿边缘运行珠子的技术成功地关闭了另一个。

他们第一次尝试使用带有 3D 打印适配器的叶片吹风机点燃它失败了，但在建造了更有弹性的部件和更有效的气体注射器后，发动机启动了。然后，它被带到一个农场，制造一些严重的噪音，没有太多愤怒的邻居，正如你在休息时的视频中看到的那样。

制作漫画的英雄工具以前也在这里出现过，最近的一次是用模拟时钟。
