# 3D 打印逻辑门

> 原文:[https://hackaday.com/2019/03/02/3d-printing-logic-gates/](https://hackaday.com/2019/03/02/3d-printing-logic-gates/)

这似乎是一个悖论，但在未来，微型计算机可能会抛弃电子设备，回到它们的机械根源。在宏观尺度上，机械计算机繁琐而缓慢，但当你的领域缩小到几个分子时，电子设备就难以工作，而机械系统却运转良好。此外，这些设备不直接使用电力，不生成电子签名，可能对辐射等损坏电子设备的东西不太敏感。《自然通讯》最近的一篇论文讨论了[如何使用宏观尺度的 3D 打印技术和微立体光刻的小得多的版本来 3D 打印普通逻辑门](https://www.nature.com/articles/s41467-019-08678-0)。你可以在下面看到盖茨的视频。

这些门使用双稳态柔性机构。较大的大门使用 ABS 塑料，面积约为 250 平方毫米。较小的门不到 25 平方毫米。他们还使用一种特殊的技术使门在理论上可以小到 100 微米，尽管其中一些是该团队未来的工作。

这些并不是第一个提出的微机械门。然而，许多其他机械设计包括一个旋转关节来实现反转，而这种设计并不要求这样。这意味着它更容易制造，而且不会像其他方案那样有摩擦和磨损的问题。

有了功能反相器，下一步就是“或”门。通过组合非和或门，你可以很容易地制造与非门、与门、或非门以及任何你能想象出的逻辑。

机械计算绝对是老派，但不是在这些规模上。事实证明，你几乎可以从任何事物中找到逻辑。[连扫雷舰](https://hackaday.com/2017/01/03/make-logic-gates-out-of-almost-anything/)。
