# 紧急 DNS 需求的 Pie Stop

> 原文：<https://hackaday.com/2022/11/24/pie-stop-for-emergency-dns-needs/>

随着广告拦截者和广告创作者之间的军备竞赛不断升级，互联网广告战争愈演愈烈。为了让现代互联网体验更加令人愉快，许多人转向 DNS 级别的过滤器来阻止广告进入网络。不过，这种解决方案也有附带的损害，因为可用的黑名单有时会过滤掉一些本应提供给用户的内容。[为了应对这些紧急情况，【克里斯托弗】创造了 Pie Stop，](https://planetkris.com/2022/11/pie-stop-for-pi-hole/)一个物理按钮，在他的 Pie-Hole 上执行临时通过。

虽然[Kristopher]能够识别问题，并为这些事件创建适当的白名单，但他家里的其他人并不认为这项任务是直截了当的。当他不在现场解决问题时，任何人都可以按下这个紧急停止按钮来暂时停止 DNS 过滤，并允许所有流量通过网络。它基于 ESP-01S，一种更小的 ESP8266 板，只有两个 GPIO 引脚。当按下时，它会向 Pi-Hole 发送自定义命令，以禁用广告拦截。盒子里的电池可以让它方便地放在任何需要的人身边。

使用此按钮，即使采用最激进的 DNS 级广告拦截，也能有效防止网络混乱。如果您还没有考虑在自己的网络上部署其中一个，一旦您看到它们有多么强大，就很难离开它们。[看看这个也能捕捉垃圾邮件的](https://hackaday.com/2021/06/15/spam-caught-in-a-tin-of-spam/)。
