# 用 Raspberry Pi 构建的多光谱成像系统

> 原文：<https://hackaday.com/2022/08/25/multispectral-imaging-system-built-with-raspberry-pi/>

多光谱成像是一种有用的工具，可以揭示隐藏在人眼中的各种秘密。[elad orbach]使用不起眼的 Raspberry Pi 建立了一个平台来执行这样的成像。

该项目建在一个暗箱内，防止外界光线污染结果。摄像机安装在顶部，用于拍摄安装在下方的标本，Pi 使用它在各种光照条件下拍摄照片。该建筑依靠各种彩色发光二极管提供干净、准确的光输出，以实现精确成像的目的。led 全部安装在一个大型铝制散热器上，可以通过 Raspberry Pi 打开和关闭，以各种不同的照明设置捕捉图像。相机周围放置了一个护套，以确保只有从样本反射的光才能到达相机，从而切断 led 本身的泄漏。

当对植物材料成像时，多光谱成像特别有用。在不同的光线下拍照可以揭示疾病、营养缺乏和其他影响植物的异常情况。我们甚至见过它被用来研究绘画。休息后的视频。
