# 难以置信的重装饰品可能不适合挂在树上

> 原文：<https://hackaday.com/2018/12/20/incredibly-heavy-ornament-likely-inappropriate-to-hang-on-tree/>

又到了一年中的这个时候，圣诞节黑客们蜂拥而至。为了融入圣诞节的气氛，FoxGuard 团队想在树上挂一个定制的装饰品。他们得到的可能比他们预想的要多。

这是一个简单的构建，以迷人的假日方式展示了使用 DAC 和示波器的基本技术。通过在 XY 模式下操作，Tektronix T932A 模拟示波器被用作显示器。Teensy 3.5 随后被选为其板载数模转换器，并用于输出信号，以在屏幕上绘制圣诞树和星星。

老派的编码人员会欣赏在绘图纸上绘制图形的努力。虽然这个黑客没有做任何尖端或疯狂的事情，但令人印象深刻的是，由于现代开发方法，这是多么快速和容易。虽然实现这一点的技术已经存在了几十年，但 1998 年的黑客可能已经花了几个小时用 DAC 对 PIC 微控制器进行试验，更不用说所需的编码了。我们已经走了很长一段路。

这有点好玩，但是我们强烈建议你不要试图在家里的树上挂一个模拟示波器。[这些 WiFi 控制的饰品或许更合适](https://hackaday.com/2017/01/01/wifi-controlled-christmas-ornaments/)。休息后的视频。
