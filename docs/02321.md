# 发射射弹的气囊装药

> 原文：<https://hackaday.com/2019/03/16/an-airbag-charge-to-launch-a-projectile/>

买小炸药包不是特别容易。至少，它不在英国，那里是[涡轮征服巨鹰]的发源地。但是想得到一个却出奇的容易，因为大多数人开车的时候都带着一个在他们面前的汽车安全气囊。在一阵要么天才要么疯狂，我们无法决定哪一个，他[决定使用安全气囊充电发射弹丸](https://www.youtube.com/watch?v=3mTDBs8xAwc)。

正如你在下面的视频中所看到的，他直接开始拆除雷诺方向盘的中心，然后似乎放弃了谨慎，拿起一台研磨机向里面充电。这是一个迷人的解构，因为它揭示了不是一个，而是两个不同大小的电荷，被一个似乎含有某种填料的空间分开。

他的抛射物是一根钢管，上面有一个旋转的钢尖，通过放置在炮尾前的管子发射出去，他在炮尾放置炸药。发射管里面焊接了一块金属，他告诉我们不能从里面发射炮弹，这是合法的。在向废铁杰里罐开火时，它有足够的能量轻易穿过它的钢铁墙壁，所以它是一种相当可怕的武器。

他向观众保证，利用龙头发射的设计，他没有违反法律，但我们不确定我们是否愿意向英国警察解释这一点。他确实指出，虽然这是一个令人印象深刻的奇观，但它也是一个相当危险的设备，所以也许不要在家里这样做。

如果你不喜欢把安全气囊拆成碎片，从头开始做一个怎么样？
