# 咸吗？小费罐愤怒退出小游戏

> 原文：<https://hackaday.com/2019/07/05/salty-tip-canister-to-rage-quit-games/>

你渴望一种更明显的方式来戒掉电子游戏吗？砸按钮当然是自然而然的事，但是这样会伤到控制器或者键盘。你可以放弃你的渴望，因为[在此插入控制器]有一个优雅的解决方案，值得它的盐。

[咸怒戒控制器](https://www.reddit.com/r/Overwatch/comments/c88n6r/i_built_a_controller_that_allows_you_to_literally/)很简单。杯子里装满了蒸馏水。当你往里面倒盐时，两个螺栓终端会告诉 Arduino Micro 水中的电阻已经降低。微处理器发送你想要的任何按键，所以你可以在辞职前叫出你没出息的医生，或者干脆离开。[在此插入控制器]在他的网站上有[示例代码来帮助您入门。冷静地点击休息时间观看演示和建立视频，否则我们将不得不禁止你的暴力行为。](https://www.insertcontrollerhere.com/example-code)

有了合适的工具，你可以把任何东西变成游戏控制器。看看[这个使用 Python 和 CAN 总线嗅探](https://hackaday.com/2017/12/22/turn-a-car-into-a-game-controller/)的汽车控制器。
