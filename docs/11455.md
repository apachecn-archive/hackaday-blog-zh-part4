# 微波炉:需要更大的功率？用激光代替！

> 原文：<https://hackaday.com/2021/09/28/microwave-ovens-need-more-power-use-lasers-instead/>

你知道这是怎么回事，你下班晚了，你被堵车堵了好久，你饿了。你伸手去拿微波炉餐，心想，要是我不用等那三分半钟就好了，900 瓦根本不够用。你需要的是一个激光微波炉，幸运的是，[【Styropyro】已经造了一个，所以你不需要](https://www.youtube.com/watch?v=fM9hYzJnao0)。不，真的，不要。

在他观察到微波只在半波的基础上工作，并在 50%的时间里提供功率后，他试图通过加倍高压变压器和整流二极管将其转换为全波。虽然这种方法奏效了，但可怜的磁控管没有走完全程，过早地死去了。

不要灰心，显而易见的是放弃用无聊的旧无线电波烹饪的整个概念，而只是用一堆该死的激光来代替。现在我们不确定他是如何设法得到他使用的一些部件的，至少可以说激光阵列模块看起来很粗略，坦率地说，鉴于他们可以聚集的可笑的光束功率，我们不认为他们应该很容易得到。

随着建造完成，达到了通常的[Styropryo]的优秀建造质量水平，他继续制作一些令人垂涎的美食，如激光烧焦的 poptart，烧成灰烬的牛排和没有真正焦糖化的洋葱，以及我们最喜欢的激光爆米花。好吧，他承认光束有太多的能量，真的应该是红外的，并且更加扩散以至于模糊地实用，但是我们不关心这些部分的实用性。谁不想因为在错误的时间走进厨房而立刻失明呢？

我们之前已经报道了相当多的微波炉相关黑客，包括一个[整洁的微波炉窑](https://hackaday.com/2019/07/20/a-microwave-kiln-from-scratch/)，以及使用微波部件的黑客，例如 [janky Jacob's ladder](https://hackaday.com/2019/02/02/microwave-parts-become-quick-and-nasty-jacobs-ladder/) ，但这可能是我们遇到的第一个激光微波炉。希望是最后一次:)
