# 神秘的惠普齿轮拆卸

> 原文：<https://hackaday.com/2021/10/22/mystery-hp-gear-teardown/>

什么东西是圆的，中间有一个看起来像真空管的东西，是惠普在 20 世纪 50 年代制造的？我们也不知道，但[信号路径]恢复了一个，并在最近的视频中向我们展示了这个神秘的仪器，你可以在下面看到。我们不会破坏这个设备是什么的惊喜，但我们会分享他在视频中很早就透露了它是什么，所以没有太多的取笑。

然而，我们会给你一些提示。看着它，你可以猜测它是用于高压的，事实上，它的额定电压高达 25 千伏。我们也将放弃暗示，这是为使用 AC，而不是 DC。我们认为，电线末端插头的形状也是一个线索。

这个不寻常的圆形盒子里面没有太多东西(顺便说一下，这是另一个线索)，但是有一些我们已经很久没有看到的古董部件。最后一个线索:为什么有一个金属棒和球从设备的一侧伸出来？

老实说，内部有点乏味，所以不像我们看过的一些拆卸视频，这个视频的真正明星是这个不寻常的设备，而不是它的内部工作。如果你渴望更复杂的惠普探索，看看我们之前报道的 [HP3458A 修复](https://hackaday.com/2021/10/18/the-hp3458a-king-of-multimeters-for-three-decades/)。或者去守旧派[偷看 HP 150A](https://hackaday.com/2021/08/01/the-oscope-restoration/) 的内部。
