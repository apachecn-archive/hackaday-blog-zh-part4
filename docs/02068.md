# 清洗桶有助于多材料打印

> 原文:[https://hack aday . com/2019/02/07/purge-buckets-to-help-with-multi material-printing/](https://hackaday.com/2019/02/07/purge-buckets-to-help-with-multimaterial-printing/)

3D 打印很酷，但大多数基本的熔融沉积打印机只是以单一颜色打印。这意味着，如果你想要一个更漂亮，更有活力的打印，你需要油漆或执行一些其他类型的整理过程。可以在飞行中转换细丝的多材料打印机是存在的，但是它们经常有浪费的问题。[【3d Mn】决定尝试建造一个净化桶作为解决方案。](https://www.youtube.com/watch?v=XDZzQ2S6jt4)

[3DMN]以前熟悉在运行多材料打印时使用清除块。一个基本块模型与实际所需的零件一起打印。印刷该块，使其与所需部件处于相同的层高度，这样喷嘴可以干净地吹扫，而不会在印刷床上布满塑料。

厌倦了浪费，[3DMN]设计了一个清洗桶，它随着 Geeetech A20M 打印机的 Z 轴移动。桶通过锁紧螺母连接到 Z 轴上，并且无论打印处于哪个阶段，桶相对于喷嘴始终处于同一高度。当需要更换材料时，喷嘴移动到桶中，清洗灯丝，然后移回打印位置。该桶配有一个 3 毫米的硅胶擦拭器，有助于确保清洗完成后喷嘴上没有材料残留，还有铝带，有助于防止清洗后的细丝粘在桶壁上。

[3DMN]注意到一些打印速度也有所提高，因为不再需要将清除对象与主要部分一起打印。对于那些希望尝试自己的设置的人来说，这些器件可以从 Thingiverse 上获得。

多材料打印可以产生一些很好的视觉效果，很高兴看到社区提供解决方案来改进流程并减少相关的浪费。我们还看到了[细丝拼接](https://hackaday.com/2018/03/24/hands-on-with-filament-splicing-robots/)，这是多材料打印的另一种独特方法。休息后的视频。
