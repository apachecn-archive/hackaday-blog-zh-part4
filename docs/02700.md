# 用垃圾建造一艘遥控空气船

> 原文:[https://hack aday . com/2019/04/12/building-an-RC-air-boat-from-garbage/](https://hackaday.com/2019/04/12/building-an-rc-air-boat-from-garbage/)

当你把一个旧光驱、一些空汽水瓶和一个微控制器组合在一起，你会得到什么？嗯…没什么，真的。那仍然只是一堆垃圾。但是如果你加上电池、遥控接收器和一些马达，你就危险地接近了[一个有趣的小玩具，可以在池塘里踢来踢去，就像【安东尼奥·里佐】最近展示的](https://www.instructables.com/id/UChip-RC-Boat-Out-of-Plastic-Bottles-and-CD-ROM-Pl/)。

[![](../Images/9e7b590990af00f50ffcf635b6939ec6.png)](https://hackaday.com/wp-content/uploads/2019/04/trashboat_detail.jpg) 几个塑料瓶绑在一起组成了船身，【安东尼奥】用一个弯曲成 90 度角的旧光驱的内部框架来支撑两个小 DC 马达。在一个特别好的接触中，驱动器的橡胶防振衬套被重新用作电机安装，尽管他承认这只是运气不好，电机是一个完美的配合。

对于电子设备，[Antonio]已经将定制电机控制器与 uChip 配对，uChip 是一种小型的 Arduino 兼容微控制器，采用窄 DIP-16 封装。无线通信由现成的 cPPM 接收器提供，例如您可能会在小型飞机或四轴飞行器中看到的那种。

整个建筑由一个普通的 18650 锂离子电池供电，考虑到它们在笔记本电脑电池中的常见程度，这也很容易从垃圾中回收；不过，如果你在这栋楼里扔了一个新牢房，我们也不会因此而反对你。所有的东西都放在一个高科技的塑料夹层袋里，以最小的努力提供最小的防水。

如果使用商业遥控接收器和发射器对你的口味来说有点太主流了，[你可以随时滚动自己的收音机](https://hackaday.com/2019/03/19/diy-six-channel-arduino-rc-transmitter/)。
