# 花式线环游戏是一个美丽的黄铜

> 原文:[https://hack aday . com/2022/03/30/fancy-wire-loop-game-is-a-beauty-in-brass/](https://hackaday.com/2022/03/30/fancy-wire-loop-game-is-a-beauty-in-brass/)

简单的线环游戏通常是作为一个有趣的项目来教授学生有关电子学的知识。[W&M·莱弗沙]制作了他们自己的版本，展示了他们精细的制作和加工技能，并将其称为一种清醒度测试，还有一个好玩的名字“酒精测试仪”

这个游戏的机制很简单。玩家必须在不接触拼图的情况下引导一个金属环。当玩家失败时，蜂鸣器和灯用于向他们指示，该项目由一个通过 USB 端口充电的小型锂聚合物袋电池供电。

这个版本真正的亮点在于展示，W&M·莱夫沙展示了他们真的有能力在铜管乐器上做出伟大的作品。我们看到的不是一根简单的弯曲金属线，而是一根由单片金属切割而成的形状精巧的矩形截面梁。它配有一个做工精致的带滚花手柄的魔杖。

我们以前也见过类似的展示他们精致工艺的展示——比如定制牙刷和 T2 粉末打火机。休息后的视频。
