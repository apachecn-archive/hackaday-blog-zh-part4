# 手术机器人是一个真正的机器人

> 原文：<https://hackaday.com/2020/05/07/surgery-robot-is-a-real-cut-up/>

做手术的机器人是一件严肃的事情。控制系统中的一个漏洞可能会导致灾难。当然，除非你是[迈克尔·里维斯]，在这种情况下，灾难是乐趣的一部分。(视频，嵌在下面。)

从达芬奇手术系统中获得灵感，[迈克尔]开始建立一个更快的系统，同时仍然保持精度。他创造了一个皮带驱动龙门系统，与许多 3D 打印机、激光切割机或木工数控机器没有什么不同。像这样的机器经常使用步进电机。[Michael]决定改用[Oskar Weigl 的] ODrive 和无刷电机。 [ODrive](https://odriverobotics.com/) 在[开源](https://github.com/madcowswe/ODrive)控制器上，它关闭架子上的无刷电机——比如那些在 R/C 飞机或悬浮板上发现的，进入精密工业伺服系统。听起来熟悉吗？ODrive 是 2016 年黑客日奖的[参赛者。[Michael]甚至能够通过监控 ODrive 的电流消耗来取消无处不在的限位开关。](https://hackaday.com/2016/05/23/hackaday-prize-entry-industrial-servo-control-on-the-cheap/)

这一切加起来就是一个严重的建设。但是[这毕竟是【迈克尔“激光眼”里维斯】](https://hackaday.com/2017/04/18/robot-targets-eyeballs-fires-lasers-oshas-gonna-love-this-one/)。该视频旨在娱乐，具有教育和启发的潜在有效载荷。当他用一把巨大的菜刀武装机器人，并对一个菠萝进行“手术”时，有趣的事情开始了。如果你想知道当人体模型和假血进入画面时会发生什么，那么休息后观看视频。

我们的一些读者可能会对这次介绍感到厌烦。但是在所有的跳转剪辑中，[迈克尔]提供了大量有用的信息。一个例子是，他展示了他的铝挤压龙门横梁是如何下垂和摇晃的，就像一根湿面条，一旦他给它装上 10 磅的硬件来驱动刀具。如果他把活动横梁设计在龙门框架的短边上，事情会变得更加棘手。

最终的机器人不会很快给我们做手术，但它肯定会成为一个有趣的视频。我已经把这个给几个医疗专业人士看过了，这给了他们一个在艰难的一天后非常需要的笑声。
