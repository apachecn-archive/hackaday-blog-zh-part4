# 从头开始打造智能音箱

> 原文：<https://hackaday.com/2019/07/05/building-a-smart-speaker-from-scratch/>

自从本世纪初首次推出以来，智能扬声器已经激增。这些设备将语音识别和助理功能结合在一起，并带有一种不祥的预感，即向公司付费以获得窃听你谈话的特权，有一天可能会反咬一口。为此，[【易慧】正试图从零开始打造一款开源的智能音箱](https://hackaday.io/project/164221-smart-speaker-from-scratch)。

最初的原型使用了一个树莓派 3B 和一个 ReSpeaker 麦克风阵列。为了降低成本，开发计划包括用定制的麦克风阵列 PCB 和 NanoPi 板替换这些组件，然后实施基本的触摸控制来帮助与设备交互。

已经有了很大的进步，构建展示了一些漂亮的特性。特别令人印象深刻的是[使用声音向设备发送 WiFi 设置的能力](https://github.com/voice-engine/wave)，以及在线和离线语音识别功能的实现。如果你的网络中断了，但是你仍然希望你的数字朋友在睡觉的时候关灯，这是很有用的。

这不是我们第一次看到关注隐私的虚拟助手，我们希望这不是最后一次。休息后的视频。
