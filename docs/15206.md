# 看起来像桌面的网站集合

> 原文：<https://hackaday.com/2022/10/27/a-collection-of-websites-that-look-like-desktops/>

自从 Web 1.0 的黄金时代以来，网页设计已经走过了漫长的道路。关于如何创建一个干净高效的网站，有很多规则，但是有时候把它们丢掉，变得更有创意会更有趣。本着这种精神， [[Simone]策划了一系列精彩的网站，模仿电脑桌面的在线体验。](https://simone.computer/#/webdesktops)

该系列的网站非常符合这一主题。访问时，会出现一个奖励 BIOS 屏幕。从那里，我们得到了一个类似 Windows 95 的界面，上面有很多链接，可以链接到其他模拟计算机桌面布局的网站。如果你鼠标离开太久，甚至会弹出一个 3D 屏幕保护程序。

还有 [Browso.app](https://www.browso.app/) ，它以一种让人想起 MacOS 9 的主题半准确地告诉你关于你的电脑的信息。同时，点击“好像你在工作”会带你去[一个奇怪的游戏](https://pippinbarr.com/itisasifyouweredoingwork/)，它复制了办公室的平庸。 [Nightwave Plaza](https://plaza.one/) 提供精致的 vaporwave vibes，而其他人只是打算在浏览器窗口中忠实地再现[各种](https://taws.ch/WB.html) [操作系统](https://macos-web.app/)。

这是一个有趣的网站集合，从古怪的熟悉网站到令人印象深刻的前现实再现网站。四处点击一会儿，看看外面有什么，这当然很有趣。我们确实喜欢围绕这些部分的一些好的[网络蜉蝣](https://hackaday.com/2016/12/03/ask-hackaday-whats-your-favorite-internet-relic/)！
