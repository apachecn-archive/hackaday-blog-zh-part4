# 一个不太贵的蒸箱

> 原文:[https://hackaday.com/2022/08/10/a-steam-box-for-not-a-lot/](https://hackaday.com/2022/08/10/a-steam-box-for-not-a-lot/)

如果你曾经惊叹于造船者或一些家具制造商使用的复杂木质曲线，那么你可能在某个时候渴望得到一个蒸汽箱。顾名思义，这是一个小室，一块木头在里面蒸，直到它变得柔韧，在这一点上，它可以被压成新的形状，一旦冷却，它将保持不变。足智多谋的【Xyla Foxlin】向我们展示了[如何使用容易找到的部件](https://www.youtube.com/watch?v=55eIkfR0UYY)制作蒸汽箱，这可以在休息时间下方的视频中看到。

蒸汽供应来自装饰者用于墙纸剥离的那种商用蒸汽锅炉，蒸汽箱本身由一段 PVC 管制成。在管道内部是一系列的铝销，形成一个架子，木头坐在上面，远离任何冷凝物，整个东西坐在一个斜面上，蒸汽入口和底部的冷凝物排放口。

使用时，将一块木头放入管子中蒸熟，然后用老虎钳中的一套模具将其弯曲。这个过程看起来足够简单，甚至我们也可以试一试，所以我们确信 Hackaday 的读者会发现它很有趣。

我们认为这可能是我们为您带来的第一个蒸汽箱，但[这不是我们第一次讨论弯曲木材](https://hackaday.com/2017/12/06/print-a-plywood-bending-jig/)。
