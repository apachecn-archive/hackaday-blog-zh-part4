# 3D 打印机重新用于轻型实验室自动化任务

> 原文：<https://hackaday.com/2022/11/11/3d-printer-repurposed-for-light-duty-lab-automation-tasks/>

实验室自动化设备非常昂贵，以至于小型实验室经常因价格过高而无法进入市场。这很遗憾，因为有很多繁琐的手工任务，即使是普通的实验室也会从自动化中受益。哦，好吧——这就是研究生的作用。

但实际上，给实验室带来一点自动化并不难，如果你跟随[Marco]、[Chinna]、[Vittorio]和[的领导，将 3D 打印机变成一个简单的实验室机器人](https://www.hardware-x.com/article/S2468-0672(22)00115-8/fulltext)。这就是 his Ender——bog-standard Creality Ender 3，经过一些特殊的修改，使其成为自动化组织切片制备的工具。组织学是对组织解剖的研究，使用各种固定和染色技术使微观特征可见。在实践中，这意味着在不同溶液的广口瓶之间来回移动载有载玻片的篮子，这项工作非常适合简单的笛卡尔龙门实验室机器人，工作范围小，负载轻。

没有一个打印机修改是永久的；3D 打印附件——用于载玻片篮的挂钩和标准组织学染色瓶的载体——可以快速从打印机上取下，使其恢复正常功能。运行 HistoEnder 所需要的只是一些定制的 g 代码，以及在打印床上小心地对齐 jar 载体。我们认为床用加热器甚至可以用来加热固定液和染色液。下面的推文中嵌入了 HistoEnder 的一个简短视频。
