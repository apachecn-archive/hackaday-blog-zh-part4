# 电动车窗马达变成了迷你电锯

> 原文:[https://hackaday . com/2021/04/02/电动窗-马达-变成迷你电锯/](https://hackaday.com/2021/04/02/electric-window-motor-becomes-mini-chainsaw/)

[由[Make it Extreme]制作的这款迷你手持电锯](https://www.makeitextreme.com/en/newsroom/entry/VIDEOS/minichainsaw)是基于车门上的电动马达，与用来升降车窗的电动马达相同。它们是常见的抢救部件，经过正确的修改和一些备用的电锯钻头，事实证明，马达完全有能力切割各种木材。加上无绳工具电池组，便携式迷你手持电锯就诞生了。

[![](../Images/82a433e4865a53d8cdd0443aad8f2714.png)](https://hackaday.com/wp-content/uploads/2021/03/minichainsaw-small-e1617259107657.jpg) 关于构建视频(嵌入在下面，休息后)真正值得注意的不仅仅是它展示了构建过程，并设法让它看起来很容易。不，真正值得注意的是，在视频中，发生了什么总是很清楚，而且没有说一个字。没有旁白，没有看别人说话，只有扎实的搭建和演示。在这里,“展示，不要说”的原则是绝对牢记在心的。
