# 观看康威的生命游戏飘过一个翻转点显示

> 原文:[https://hack aday . com/2020/07/02/watch-con ways-game-of-life-flutter-cross-a-flip-dot-display/](https://hackaday.com/2020/07/02/watch-conways-game-of-life-flutter-across-a-flip-dot-display/)

[![](../Images/4a8bad5b7f2af303ac61d9f6a91fd37b.png)](https://hackaday.com/wp-content/uploads/2020/06/Game-of-Life-flip-dot-anim.gif) 和我们许多人一样，【约翰·惠廷顿】对约翰·何顿·康威今年早些时候去世的消息感到悲伤，为了纪念他的工作，他[将生命的游戏加入到他一直在做的翻转点显示](https://engineer.john-whittington.co.uk/2020/04/game-of-life-on-a-flip-dot-display/)中。机电显示器的实体似乎特别适合细胞自动机。

喜欢你看到的吗？如果你想知道是什么让这一切发生的，显示的是一个[阿尔法-泽塔 XY5 28×14](https://flipdots.com/en/products-services/flip-dot-boards-xy5/) 显示器，但[约翰]目前正在努力将它们建成一个更大的 256 x 56 显示器。GitHub 托管了 [flip-dot 模拟器和【约翰】正在使用的驱动软件](https://github.com/tuna-f1sh/flipdot)，而[的游戏人生功能在这里](https://gist.github.com/tuna-f1sh/9e6ff4552f75de3705cae6d3c044b1cc)。

如果你是人生游戏的新手，不确定你在看什么，[埃利奥特·威廉姆斯] [在他的文章](https://hackaday.com/2020/04/18/the-game-that-launched-1000-hackers/)中告诉你你需要知道的一切，庆祝它的深远影响和持久遗产。观看下面嵌入的视频中的 flip-dot 显示。
