# 调试电子设备:要知道它为什么不工作，首先要找到它实际上在做什么

> 原文:[https://hackaday . com/2020/03/05/debugging-electronics-to-know-why-it-not-work-first-find-what-it-be-actually-do/](https://hackaday.com/2020/03/05/debugging-electronics-to-know-why-it-didnt-work-first-find-what-it-is-actually-doing/)

恭喜你，你已经完成了你的电子项目。在检查了明显的问题后，你接通电源，但…它没有按你的要求运行。他们几乎从来没有在第一次尝试中工作过，因此[我们在 2019 年 Hackaday 超级大会上与丹尼尔·萨马林(Daniel Samarin)一起步入了电子调试的世界](https://www.youtube.com/watch?v=oULeLtcv4n4)。下面嵌入了新发布的谈话视频。

刚开始接触闪烁的 led 和预制套件的新手会从这里的信息中受益最多，但也有一些对更有经验的老手有用的花絮。重点是理解 ***实际上是什么*** 在电路内部发生，这就解释了讲座的题目:*调试电子:你搞不定地道理！*这样我们可以将观察到的行为与设计意图进行比较。没有准确的理解，任何尝试性的解决都注定要失败。

要真正做到这一点，你需要使用那些经常在库存充足的电子工作台上找到的工具。丹尼尔深入探讨了超越 printf 和闪烁 LED 的行业技巧，以形成一个处理任何调试任务的计划。

因为电子流是肉眼看不见的，所以理解我们的电路需要电子行业的测量工具。这种设备可能会非常昂贵，但丹尼尔指出了几个“物有所值”的工具，在这些工具中，适度的投资可以让我们获得实质性的洞察力。我们很少需要顶级的专业仪器。一个不起眼的万用表可以带我们走很远！这不是关于我们使用的工具，而是关于我们如何以严格和有纪律的方式使用它们。不管是基础米还是[最新增强现实助手 hotness](https://hackaday.com/2020/02/05/debugging-pcbs-with-augmented-reality/) ，基础都是一样的。

毫不奇怪，DMM 和示波器是最先推荐的两种工具，但他继续推荐使用热传感器——你可以用手指，但红外热感枪和热感相机会更有用。当提出新的主板设计时，检查是否有任何东西变得过热或过热是故障排除的首要任务。丹尼尔列出的必备工具还包括电流传感器和逻辑分析仪。

别忘了仔细检查你的工具；那个长凳上满是噪音吗？它真的输出了正确的电压吗？你的示波器探头损坏了吗？调试你自己的测试平台和调试你正在使用的电路板一样重要。

隔离问题是关键。如果不能重现问题，就不能解决问题。这其中很大一部分是在过程中做好笔记，这样你就不会浪费时间一遍又一遍地检查设计的相同部分，试图找出是什么导致了失败。如果你凭记忆做任何事情，笔记会让你看起来不像是在做任何事情。

我们可以戳戳机械问题来理解问题，而软件开发为我们指尖上的内部洞察力提供调试器。电子调试的工具可能不同，症状可能更难以发现，但基本原理适用于任何地方。
