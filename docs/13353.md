# 我们听说过用砖砌硬盘，但是…

> 原文：<https://hackaday.com/2022/04/14/weve-heard-of-bricking-a-hard-drive-but/>

自从个人电脑问世以来，海量存储已经走过了漫长的道路。[Tech Time traveler]有一个关于 PC 硬盘黎明的有趣视频，聚焦于一家名为 [MiniScribe](https://www.youtube.com/watch?v=OWwzQyXAvXI) 的公司。在一个充满希望的开端之后，他们失去了一份 IBM 的合同，陷入了困难时期。

显然，该公司伪造了高达 1500 万美元的库存，因为如果没有利润，高管们会担心自己的工作。一旦他们发现不正确的库存，他们不仅着手修改公司的记录来匹配它，而且他们还闯入外部审计公司的记录来改变那里的事情。

高级管理层策划了一项计划，以小额冲销假库存，以逃避投资者和政府监管机构的注意。但要做到这一点，他们需要能够解释不存在的库存的余额在哪里。所以他们租了一个仓库来存放欺诈库存，并用砖头将其填满。像你用来盖房子的真砖。大约 26，000 块砖被包装在盒子里，分配了序列号，放在托盘上。审计员将看到产品准备发货，甚至有计划假装将它们发送给 CompuAdd 和 CalAbco 这两个客户，这两个客户同意接受并返回纸面上的砖块，允许他们一次吸收一点 1500 万美元的注销。

不幸的是，虚假的优秀财务业绩导致了对未来甚至更好业绩的预期，这就需要进一步的欺诈。该公司已经扭亏为盈，但只是纸上谈兵。电脑行业的低迷和信用透支标志着末日的开始。供应商和员工拿不到工资。一位高级经理违反了内幕交易者规则，抛售了大量股票。

扭亏为盈的首席执行官最终辞职，一位新的首席执行官发现了欺诈行为，并公布了他们亏空 1 亿美元的调查结果。破产将公司的资产推向 Maxtor，随后对 16 人的刑事指控以罚款和监禁告终。尚不清楚这些盒装砖块是意外运送给了谁，还是由一名拿着橡胶薪水的不满员工运送的。[科技时间旅行者]推测，如果有人有一个，这将是相当收藏家的项目。

当然，我们听说一些公司在做一些有问题的事情，但是这确实在范围上令人印象深刻。至少他们没有像 T2 的一些科技公司一样欺骗终端用户。
