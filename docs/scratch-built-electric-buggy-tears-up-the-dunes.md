# 白手起家的电动汽车撕裂了沙丘

> 原文：<https://hackaday.com/2022/07/05/scratch-built-electric-buggy-tears-up-the-dunes/>

可以肯定地说，未来的个人交通工具很可能是电动的。为了支持这一点，每个主要汽车制造商要么现在就有电动传动系统可供选择，要么他们正在努力开发一个。虽然有朝一日你的郊区食品杂货店将由太阳能驱动是件好事，但对仅仅是驾驶乐趣的电动汽车的迫切需求呢？

为了填补有趣的空白，至少现在，[詹姆斯·比格]建造了你买不到的东西:[一辆全电动沙滩车](https://youtu.be/xBl3KHxCyQI)。为了避免你认为这是一个工具包，请放心，下面的总结视频显示，这个小沙轨是 100%从头开始建立。底盘由弯管制成，并使用巧妙的胶合板模板焊接起来，以获得恰到好处的角度。该车有四轮独立悬架和一个广泛的，积极的姿态来处理崎岖的地形。车身面板是在定制的制动器上弯曲的铝板，这也是用来形成热枪帮助下的树脂玻璃挡风玻璃。

虽然车身使越野车看起来很恶心，但传动系统也同样令人印象深刻。[詹姆斯]用了一个 ME1616，一个液冷 55 千瓦的野兽。链传动将马达连接到本田 CR-V 的差速器上，该差速器安装了限滑装置。这些电池也令人印象深刻——32 个定制的磷酸铁锂电池由 32650 个真空成型的 ABS 塑料外壳制成，紧密地嵌套在一起。这一切都增加了泥土中的乐趣；视频跳到 23:37，看看这个东西能干什么。

老实说，这里的工艺水平是一流的，更令人印象深刻的是，它不是花哨的——只是良好、可靠的方法和大量的辛勤工作。我们很乐意有时间和资源投入到这样的事情中——虽然[一个嵌入式机箱电动汽车](https://hackaday.com/2020/11/15/an-ev-conversion-engineered-as-a-drop-in-replacement/)可能也是一个令人满意的构建。
