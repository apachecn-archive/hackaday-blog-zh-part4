# 机器人在 IPhone 的帮助下绘制房间地图

> 原文:[https://hack aday . com/2018/07/14/robot-maps-rooms-with-help-from-iphone/](https://hackaday.com/2018/07/14/robot-maps-rooms-with-help-from-iphone/)

自从苹果开始使用英特尔芯片以来，Unity 引擎就一直存在，并在游戏世界中引起了不小的轰动。Unity 允许开发者创建 2D 和 3D 游戏，但这个游戏引擎还有一些其他有趣的应用。例如，[matthewhallberg]用它来[建造一个可以绘制 3D 房间地图的机器人](https://hackaday.io/project/159675-room-mapping-arduino-robot-with-unity-3d)。

推动这个项目的是一家机器人公司，该公司在其业务中使用了一系列机器人。机器人使用计算机视觉导航，但不能从头开始绘制房间地图。他们雇佣了[matthewhallberg]来解决这个问题，这个机器人是一个初步的成果。使用 [Unity](https://en.wikipedia.org/wiki/Unity_(game_engine)) 引擎和 iPhone，机器人可以在三种模式中的一种模式下工作。第一个是用户控制模式，第二个是对象跟踪，第三个是 3D 映射。

这个机器人似乎很容易建造，只携带一部 iPhone，一个节点 MCU，一些马达和一块电池。大多数计算工作都是远程完成的，机器人只需从另一台计算机接收移动命令。这里有很多软件方面的东西，有很多工具包和软件包需要安装和相互通信，但是下面的视频很好地展示了你需要什么以及它们如何一起工作。如果这一切都太多了，还有其他具有计算机视觉形式的机器人可以让你开始进入计算机视觉和地图的世界。
