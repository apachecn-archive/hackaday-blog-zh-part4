# 华丽的标本是单词钟的最后一个单词

> 原文：<https://hackaday.com/2021/06/23/gorgeous-specimen-is-the-final-word-in-word-clocks/>

在这一点上，可以肯定地说，单词钟已经不像以前那样令人兴奋了。我们已经看到一些版本将这个概念归结为一个部件箱的构建，无论是好是坏，它都带走了很多魔力。你只需要得到一个发光二极管阵列，在它前面放一些字母，写一些代码，就大功告成了。

但是之后[马克·西德勒]送来了他的作品，我们记起了当初为什么我们会集体爱上这些钟表。吸引我们的不是最终的结果，虽然最终的钟确实很华丽，但它苦心设计和建造的故事。为这个项目创建的文档无疑是我们在很长一段时间内看到的最好的一些，无论你是否有任何建立自己的单词时钟的愿望，你都不会后悔坐下来通读它。

如果你能以某种方式从阅读构建日志中走出来，并且没有留下深刻的印象，那么时钟的功能集肯定会让你崩溃。以五分钟为增量显示时间的能力使它成为我们见过的最实用的单词时钟之一，生活质量功能，如基于环境光水平的自动亮度控制，以及智能手机控制的网络界面配置时钟，只是它的几个突出功能。

顺便提一下，时钟背后的发光，由专用的 WS2812 RGB LEDs 阵列提供，不仅仅是为了营造氛围。它指示由 Python `astral`包计算出的太阳在天空中的位置，并模仿日出和日落的颜色。船上甚至有一个指南针，以确保发光二极管与他们的宇航同行正确对准。

[Mark]实际上做了几个这样的钟，大部分是作为礼物送人的。一些幸运的接收者住得很远，以至于钟必须被运送，所以他设计了一个定制的装运箱，以便在旅行中安全地保存所有东西。这也意味着他必须想出一种方法，在没有用户干预的情况下远程维护这些时钟上的代码，所以他创建了一个固件更新和遥测收集后端，亚马逊网络服务会定期检查这些后端。老实说，对这个项目每一个细节的关注是惊人的。

如果你有兴趣看看这些单词钟有什么大惊小怪的，但不太符合[马克]的水平，不要担心。正如我们之前所说的，你可以用一个 LED 阵列和一个微控制器来构建一个小型版本。如果最后变成了困扰，不要责怪我们。
