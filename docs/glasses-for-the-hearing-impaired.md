# 听障人士的眼镜？

> 原文：<https://hackaday.com/2018/10/14/glasses-for-the-hearing-impaired/>

如果你没有听力损失，很容易忘记你是多么依赖你的耳朵。如果你能负担得起，助听器是很好的，但它们不像眼镜那样能立即在几乎所有方面提高你的感觉。除了不得不习惯助听器之外，你还会经常发现噪音甚至反馈增加。如果你最近去过剧院，你可能会注意到附近某个地方有一个闭路字幕显示系统，如果你听力不好，你可以坐在它的可视范围内。这限制了你的座位选择，需要你将注意力分散在舞台和设备上。伦敦国家剧院正在使用爱普生智能眼镜[将字幕放在你个人的视线中](https://www.nationaltheatre.org.uk/your-visit/access/captioned-glasses)(见下面的视频)。

爱普生眼镜类似于几年前引起轰动的谷歌眼镜，它似乎是一个如此伟大的应用程序，我们很惊讶它花了这么长时间才被创造出来。我们也很惊讶听到这个项目的长度，令人惊讶的是它花了四年时间。爱普生眼镜可以接受 HDMI 或 USB-C 输入，因此看起来好像树莓 Pi、电池和眼镜可以使这成为一个周末项目。

增强现实是那些应该很棒的技术之一，但它的“杀手级应用”似乎还没有出现。然而，如果你听力不好，这肯定是它。这让我们想知道像这样的头饰还能做些什么。语音到文本的技术每天都在进步，所以也许可以用眼镜来描述现实生活？我们甚至可以看到，仅仅显示一个红色条来指示大声音的方向可能是一个方便的事情。

当然，你不需要智能眼镜只是为了[得到一个大噪音的警报](https://hackaday.com/2017/09/05/hackaday-prize-entry-vibhear/)。我们甚至看到 [Alexa 学习美国手语](https://hackaday.com/2018/07/25/talking-to-alexa-with-sign-language/)。
