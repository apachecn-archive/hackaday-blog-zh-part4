# 用运动中的磁铁煮蛋

> 原文：<https://hackaday.com/2018/11/02/cooking-eggs-with-magnets-in-motion/>

或许找些干柴生火会更容易些，但是如果你真的陷入困境，而碰巧有一堆磁铁和一个跑步机马达，[这个 DIY 电磁炉](https://www.kjmagnetics.com/blog.asp?p=induction-stove)可能是你热早餐的关键。

对于那些不熟悉它们的人来说，电磁炉是真实存在的。这个想法可以追溯到上个世纪初，包括使用强磁场在烹饪器皿的金属中感应涡流。正如[K&J Magnetics]解释的那样，通过改变附近的磁场，在导体中感应出涡电流。电流产生自己的磁场，与产生磁场的方向相反。由此产生的电流流过导体，加热导体。对于他们的炉灶面，他们选择使用一台旧的跑步机电机旋转一堆极性交替的强有力的钕磁铁。第一次尝试加热到只能勉强煮熟一个鸡蛋。增加更多的磁铁导致更多的热量，但突破来自一个更小的锅。下面的视频展示了操作中的炉灶面。

值得注意的是[商用感应炉灶](https://hackaday.com/2016/02/19/what-can-we-learn-from-a-cheap-induction-cooktop/)使用线圈和高频交流电代替旋转磁铁。他们对炊具的挑剔也是出了名的。因此，为[K&J]找到这样一个权宜之计的成功而喝彩。下一步，我们希望看到永久磁铁被可以电子换向的小线圈取代，也许是用[无刷电机控制器](https://hackaday.com/2016/05/18/hacking-rc-brushless-motor-controllers-for-use-in-big-robots/)。
