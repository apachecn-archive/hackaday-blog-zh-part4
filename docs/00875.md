# 把你的 3D 打印机搬到 2D

> 原文:[https://hack aday . com/2018/10/08/knock-your-3d-printer-down-to-2d/](https://hackaday.com/2018/10/08/knock-your-3d-printer-down-to-2d/)

黑客喜欢 3D 打印机。事实上，他们可能太爱他们了。我们~~希望~~知道我们不是唯一不能拒绝一家(或两家)海外印刷商的好交易的人。但是，当你不抽塑料船和其他解放军除尘器，你应该做些什么呢？

[![](../Images/29be111bf202023af4539775e53acfce.png)](https://hackaday.com/wp-content/uploads/2018/10/3d2d_detail.jpg) 如果你像[尤里·沙克德]一样，你可以递给他们一支笔，告诉他们开始写作。假期快到了，总得有人在这些卡片上签名。[在他的详细报道中，他展示了他如何能够在不做任何永久性改动的情况下，给他的 Creality CR-10](https://medium.com/@urish/how-to-turn-your-3d-printer-into-a-plotter-in-one-hour-d6fe14559f1a) 打印机添加一支笔，将其改造成一台简洁的信函书写机器。

这种技术的物理方面非常简单:只要想出一些方法将笔保持在打印机热端以下一点。这里的定位有点关键，因为你不想在写邮件的时候把喷嘴撞到床上。[Uri]发挥想象力，设计了一个小支架，可以夹在 CR-10 上，甚至有一个 M3 螺丝来固定钢笔，但如果你只是想尝试一下，你可以用拉链带。

[Uri]深入到软件方面的更多细节，这很好，因为让打印机执行所需的特定舞蹈动作确实需要一些 Inkscape 的技巧。他一步一步地(通过屏幕截图)解释了如何设置方向点和配置工具参数以获得最佳性能。即使你不打算在下一次 hackerspace 会议之前使用 3D 打印机为你的 8x10s 签名，这也是一个很好的使用 Inkscape 制作 GCode 的指南，对制作 PCB 等任务很有帮助。

这里的一般过程[非常类似于给你的 3D 打印机](http://hackaday.com/2018/09/29/turn-a-cheap-3d-printer-into-a-cheap-laser-cutter/)添加一个激光模块，但是你的眼球做出最佳死星印象的风险要低得多。
