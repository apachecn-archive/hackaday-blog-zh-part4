# 用木屑把树皮从混响中分离出来

> 原文:[https://hacka day . com/2021/11/21/用木屑将树皮从混响中取出/](https://hackaday.com/2021/11/21/taking-the-bark-out-of-reverb-with-wood-scraps/)

在过去的几年里，许多人已经习惯了在家里举行虚拟会议。厨房餐桌、沙发、备用卧室和院子里的吊床等空间都变成了“办公室”。你可以想象，许多这样的空间并不以其音质而闻名。[Zac] [用碎木片做了一个声音扩散艺术品](https://www.youtube.com/watch?v=yzCFF5D5gPw)帮助他的办公室在录音时声音更好。

混响是由声音从坚硬平坦的表面(如干墙)反射回来引起的。这些反射会被话筒拾取，导致音质明显下降。通常有两种方法来消除空间中的混响:扩散和吸收。扩散是[Zac]追求的技术，有成千上万张不同角度和位置的脸，它把粗糙的反射分解成数百万个微小的反射。吸收通常是用泡沫和其他典型的软物质来完成的。

[Zac]碰巧有一大堆各种木材的下脚料和额外的材料，很容易制作出视觉上有趣的作品。他用台锯将它们锯成一致的宽度，然后用鼓式打磨机将它们打磨成相同的深度。接下来，用斜切锯将长棍切成 5 种不同的长度，给他留下了成千上万的小木片。最困难的部分开始于他必须用 CA 胶将几千块碎片粘到一个胶合板背板上。打磨，聚精会神，法国夹板使这三件作品可以挂在墙上。

总的来说，效果是惊人的。虽然我们希望有更多关于改善的硬数据，但听起来确实更好。如果你对更多的木工感兴趣，[看看如何不用 CNC](https://hackaday.com/2021/10/17/machining-wood-inlays-no-cnc-required/) 来制作镶嵌物。休息后的视频。
