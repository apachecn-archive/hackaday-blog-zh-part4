# 先进的传感器和 Arduino 节省燃油

> 原文：<https://hackaday.com/2022/08/03/saving-fuel-with-advanced-sensors-and-an-arduino/>

当[机器人酒吧]不忙着调整他们本田 Insight 的 420cc *大块*发动机时，他们可能正在研究其他一些完全不同的汽车暴行。在休息时间下面的视频中，你会看到他们将 Insight 的“稀薄燃烧”系统概念和~~的“嫁接”~~ [融入他们的 1997 年土星跑车](https://www.youtube.com/watch?v=SACCy4KBUKs)。

什么是稀薄燃烧系统？简单地说，它欺骗汽车在轻载行驶时燃烧更少的燃料，以提高车辆的平均里程。土星的电子设备不够复杂，无法简单地实现稀薄燃烧系统，所以[机器人酒吧]做了我们任何人都可能做的事情:用 Arduino 黑进去。

该视频在细节方面做得很好，但本质上是通过使用分辨率更高的氧传感器(宽带)，然后向 ECU 输出适当的窄带信号，[Robot Cantina]只需一个电位计就可以微调空气/燃料比，而汽车的 ECU 并不知道。结果如何？嗯…它们并不像预期的那样，这意味着更多的实验，更多的零件，希望有更多的视频。我们喜欢看到科学方法得到有趣的应用！

人们总是在寻求尝试有趣的新(有时是旧)想法，比如这个用割草机汽化器改装的热棒。
