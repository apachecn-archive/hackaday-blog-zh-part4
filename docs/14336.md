# 悬浮滑板驱动的沙发很有趣，也有点危险

> 原文:[https://hack aday . com/2022/08/02/hover board-powered-sofa-is-fun-and-bit-dangerous/](https://hackaday.com/2022/08/02/hoverboard-powered-sofa-is-fun-and-a-bit-dangerous/)

废弃的悬浮滑板是黑客免费高扭矩电机的一个很好的来源。这可能包括疯狂但有趣的项目，比如这个由悬浮滑板驱动的宜家沙发，正如[Bitluni]和他在柏林 xHain Hackerspace 的朋友们所展示的。

有了几个在各种条件下报废的悬浮滑板和一个可用的电动自行车电池，该小组开始探索不同的选项来组装一个可用的传动系统。第一次尝试是通过拦截来自基于陀螺仪的控制器的通信来直接命令电机驱动器。9 位通信协议是一块难啃的骨头，所以他们尝试(并且失败了)直接使用陀螺板作为控制器。在研究过程中，他们发现有人为悬浮滑板控制器创造了替代固件，允许用 Wii 双节棍进行控制。甚至还有一个基于网络的[配置](https://pionierland.de/hoverhack/)工具用于编译固件。

将一些木质垫片拧到沙发底部，只需将悬浮滑板电机的外壳拧到沙发底部，并在两半之间添加一段 PVC 管进行布线，就可以连接悬浮滑板电机。脚轮被添加到沙发的后角，以完成底盘。马达对双节棍上的控制输入非常敏感，所以坐在沙发上往往会很快变成一场竞技活动。沙发也不是用来在外角承重的，所以在它开始开裂后，必须用胶合板加固。

我们已经看到了大量涉及悬浮滑板电机的黑客，包括一个带麦克纳姆轮的电动滑板和一个令人惊讶的实用电动自行车转换 T2。
