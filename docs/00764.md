# DIY 工作室灯改善您的视频

> 原文:[https://hack aday . com/2018/09/25/DIY-studio-lights-to-improve-your-videos/](https://hackaday.com/2018/09/25/diy-studio-lights-to-improve-your-videos/)

现在是 2018 年，距离 YouTube 成立整整十三年。随着在线分享服务的建立和运行，以及几乎每部手机都配备了高分辨率摄像头，视频制作已经民主化。可悲的是，对于那些有眼光的市民来说，*好的*视频的制作并没有那么广泛。好的视频需要什么？良好的照明—[你可以自己动手建造](https://www.youtube.com/watch?v=DhbMnQt14_o)。

这种来自[DIY Perks]的构建依赖于现成的组件，并使用普通制作者也能理解的简单构建技术。使用廉价的 LED 灯条(尽管是摄影级的)，加上现成的塑料和调光器模块，就有可能建造一个既保持色彩完整性又重量轻、体积小、易于使用的灯。最终产品非常优雅——一眼看去，它可能是一个商业产品。像菊花链电源和组合不同光谱的 led 以更好地控制这样的巧妙技巧增加了功能。

总的来说，这是一个需要一些时间的构建，但它可以很容易地在一个周末完成，结果是一流的。这也不是我们在这里看到的第一个 lightbox。休息后的视频。
