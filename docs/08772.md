# 用现代硬件逆向计算，不需要仿真

> 原文：<https://hackaday.com/2020/12/27/retrocomputing-with-modern-hardware-no-emulation-required/>

x86 处理器家族目前是 PC 世界中最普遍的处理器类型，自 20 世纪 80 年代 IBM PC 问世以来一直如此。如果你想玩一个老的 LucasArts 游戏或再次体验 Windows 3.1，模仿这些旧设备很容易，但真正的体验是在原始硬件上找到的。而且，由于工业设备兼容性的需要，你可以用新的硬件建造一个全新的 486 机器，它将运行这个复古的软件，就像它本身是新的一样。

[Rasteri]策划了这一建设，这是 NES 经典和其他怀旧游戏机重新发布的回忆。它基于 90 年代初推出的 PC/104 标准，主要用于工业控制应用。该平台非常小，并且为该构建选择的板承载运行在 300 MHz 的 486 处理器。它有板载 VGA 兼容显卡，但没有声霸卡，所以他设计并制作了自己的 ISA 兼容声卡，适合 PC/104 的可用扩展端口。

在添加了一些更小的外围设备并将其安装在定制的机箱中后，[Rasteri]在新的裸机 486 硬件上有了一台工作正常的 DOS 机器，可以按照最初的意图播放 DOOM。如果你在滑雪时感觉像被雪怪吃掉了，它还可以运行早期版本的 Windows 来玩微软娱乐包中的游戏。(拉斯特里人)对这种密集的复古计算也不陌生，因为他是那个注定要在最初的 NES 硬件上运行的人。
