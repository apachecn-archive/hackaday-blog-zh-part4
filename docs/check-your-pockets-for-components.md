# 检查你口袋里的组件

> 原文：<https://hackaday.com/2020/09/28/check-your-pockets-for-components/>

理想的元件测试仪就像电子设备的三录仪——它可以随时测量你需要的任何东西。也许你有几个设备，如欧姆表，也许我们的万用表上有晶体管插座。但是，当你需要查看晶闸管是否有故障时，你会怎么做呢？[阿克谢·巴韦贾]想要一个万事俱备的测试器，所以他正在建造一个能装进口袋的综合设备。它会识别类型和大小:

*   电阻
*   电容器
*   感应器
*   二极管
*   晶体管
*   半导体闸流管
*   翠克家族

其小尺寸的关键是 USB-C 电源端口，而不是板载电池和附带的充电电路。在[定制 PCB](https://www.tindie.com/products/akshaybaweja/component-tester-keychain/) 的背面是一个 ATMEGA328，正面有一个有机发光二极管屏幕、三个测试引脚插座和一个按钮。

我们很欣赏控制器如何不在乎您连接引脚的顺序，它会计算出测试对象是什么以及每根导线的功能。结果会一直显示在屏幕上，直到您加载新组件并再次按下按钮。一个函数发生器功能将它与一般的测试器区分开来，但它没有 T2 听觉连续测量仪。
