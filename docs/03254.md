# 乐高在水下，有模型潜艇和导弹

> 原文：<https://hackaday.com/2019/06/08/lego-goes-underwater-with-model-submarines-and-missiles/>

用乐高制作一辆玩具车很有趣，但制作一辆可以实际工作的玩具车更有趣。【PeterSripol】做了[两艘乐高潜艇](https://www.youtube.com/watch?v=dolTvEnjjy4)，你可以在下面的视频中看到。没有很多建造信息，但是看着潜艇发射导弹，然后被深水炸弹摧毁是值得的。

其中一艘潜艇更大，使用方向舵驾驶。很明显，它比另一个较小的潜艇更难控制，后者使用两个相反的马达来操纵。看起来很有趣。

[Peter]发现他必须使用较低频率的 RC 控制器来穿透水。较小的那个似乎拖着一根电线，所以我们认为它可能甚至没有使用无线电控制。有许多定制的 3D 打印部件可以帮助电机和其他机电设备。

不过，有几件事我们想知道。所用的电子产品非常传统，浸泡在水中可能对它们的寿命没有好处。我们也不清楚潜艇是否有任何浮力控制。我们认为困在里面的空气只是让它们在某种程度上漂浮，直到空气逸出。

我们最喜欢的[型号潜艇](https://hackaday.com/2009/12/28/star-trek-submarine/)也是我们最喜欢的星舰。如果你想要浮力控制的例子，看看这个过去的 [Hackaday 奖参赛作品](https://hackaday.com/2017/05/24/hackaday-prize-entry-underwater-glider-offers-low-power-exploration/)。
