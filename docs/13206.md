# 3D 打印吉他琴颈

> 原文：<https://hackaday.com/2022/04/01/3d-printing-a-guitar-neck/>

许多第一次制作吉他的人专注于制作琴身，而忽略了琴颈，如果你想要一个可以很好演奏的乐器，琴颈有很多复杂的尺寸。然而， *Proper Printing* 的【Jón Schone】想要从他的吉他制造之旅中最困难的部分开始，[并着手 3D 打印一整块吉他琴颈。](https://www.youtube.com/watch?v=-HobdwFHNtU)

从表面上看，设计琴颈可能听起来很难，但是 FreeCAD 的 Marz 吉他设计器插件有助于使制作琴颈变得轻而易举。一旦导入到 Fusion 360 中，几何图形将进行调整以进行 3D 打印，特别是为了适应内部的桁架杆。用绿色 PLA 在 Creality CF30 带式打印机(有趣的是，它被安装在墙上)上打印，由此产生的颈部可以在一英里外被视为非传统设计。随着一个桁架杆被敲入，品被安装，硬件被连接，它被安装到一个便宜的吉他上进行测试。

印花的脖子很管用，而且用一些适当的重复片段给了它一个适当的安定感，让它通过它的步伐。据报道，这是一个有点灵活的一面，但仍然可以玩，在它的表现令人惊讶的正常。[Jón]现在计划通过 3D 打印吉他的其余部分来继续该项目。

与此同时，如果你厌倦了给自己的吉他调音，可以考虑制作一个机器人调音师来帮忙。休息后的视频。
