# Live Jam Kit 帮助电子音乐家保持同步

> 原文:[https://hack aday . com/2021/11/23/live-jam-kit-helps-electronic-musiciers-stay-in-sync/](https://hackaday.com/2021/11/23/live-jam-kit-helps-electronic-musicians-stay-in-sync/)

用合成器和鼓机现场演奏可能很有趣，但对[Christian]来说，还缺少一点东西。他在寻找一种方法，让团队中的每个人都跟上节奏，尽情摇摆，并决定建立一些帮助。

构建的精神是让一个人最终负责使用 Ableton 的混音。这就停止了音量竞赛，因为每个音乐家都把自己的音量调大了，堵塞变成了嘈杂的混乱。每个音乐家还会得到一个同步按钮，如果他们的乐器已经过时，他们可以按这个按钮。果酱中的每个人也在耳机中获得自己的监听信号，以及一个活套。

电子 jam 中的单个玩家可以创建一个很酷的小循环，并使用 looper 将它发送给运行 Ableton 的主控制器。然后，他们可以在不破坏主混音的情况下，在耳机中混合其他东西，然后再次循环播放。

[【Christian】的演示视频](https://www.youtube.com/watch?v=e-3O1OGSID0)很好地展示了它是如何工作的。我们特别喜欢 sync 按钮，当 jam 中的音序器被速度信号绊倒时，它摆脱了通常的挫败感。

这一切都是由一个小东西构建的，看起来是用一堆不同的合成器和鼓机组织即兴演奏的好方法。我们当然很想加入这场欢乐。
