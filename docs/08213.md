# JavaScript 中的 TTL 模拟器

> 原文:[https://hack aday . com/2020/11/02/TTL-simulator-in-JavaScript/](https://hackaday.com/2020/11/02/ttl-simulator-in-javascript/)

你如何庆祝你的 YouTube 频道突破 7400 用户大关？如果你是【低级 JavaScript】，答案很明显:你用 JavaScript 创建一个 [7400 TTL 逻辑模拟器。门模拟从简单的门发展到触发器和寄存器。你也许可以用这段代码构建一台基于 7400 的计算机。](https://github.com/LowLevelJavaScript/Digital-Logic-Simulator)

除了好玩有趣之外，视频和评论中还有很多有趣的链接。其中一个原因是，观看该频道的某个人获取了代码，并制作了一个令人印象深刻的类似 Verilog 的 IDE。

当然，你可以去 [EDAPlayground](https://hackaday.com/2015/07/21/learn-fpgas-in-your-browser/) ，写一些 Verilog，然后用普通工具模拟它。然而，不知何故，这似乎没有虚拟连接 7400 个芯片有趣。不要问我们为什么。

还有一个令人印象深刻的开源网络逻辑模拟器的链接。如果你想修改或者只是看看它是如何工作的，你可以在 [GitHub](https://github.com/renato-bohler/logossim) 上找到它的代码。

视频提到[Ben Eater]的[试验板计算机](https://hackaday.com/2017/04/10/8-bit-breadboard-computer-is-up-to-8-hours/)是这个项目的灵感来源，我们希望看到有人使用这种技术为浏览器做模拟。如果你想在你的浏览器中看到一台更原始的计算机，试试 [GENIAC](https://hackaday.com/2019/08/20/circuit-vr-advanced-falstad-logic-with-geniac/) 。
