# 彩虹糖分类器让长任务变短

> 原文:[https://hack aday . com/2020/12/16/skitle-sorter-makes-long-task-shorter/](https://hackaday.com/2020/12/16/skittle-sorter-makes-long-task-shorter/)

Big Candy 的代理商最常见的抱怨之一是他们产品的非同质性。彩虹糖和 M&Ms 是这一领域的两个令人震惊的违规者，这就要靠坚定的消费者自己来处理这些糖果的分类了。当然，你总是可以用手来做，但是众所周知，机器将会完成这项工作。

这个彩虹糖分类器是[DIY 机器]的[Lewis]的创造，它是一个针对初学者水平的构建。它由纸板制成，使用一对伺服系统来处理糖果的运输，通过旋转圆盘和滑槽将糖果运输到必要的彩色垃圾箱中。彩虹糖由连接到 Arduino Nano 的 TCS34725 彩色扫描仪扫描，该扫描仪改变输出斜槽的角度，将糖果倾倒在适当的位置。料斗能够毫无问题地处理一袋标准的 180 克的彩虹糖。

[Lewis]很好地解释了构建的每个阶段，从机械和电子方面，到所需的校准，以使一切都很好地配合。该项目向建筑商教授了许多有用的课程，比如如何使用限位开关和其他自动化概念。我们以前也在这些页面上介绍过[刘易斯];他的时尚挂钟尤其令人喜爱。休息后的视频。
