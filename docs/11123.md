# 用于免提视频的踏板操作电缆摄像头

> 原文:[https://hack aday . com/2021/08/25/脚踏式有线摄像头免提视频/](https://hackaday.com/2021/08/25/pedal-operated-cable-cam-for-hands-free-video/)

[老式后院钢筋混凝土]已经建立了一个很好的小钢筋混凝土轨道在他的后院，并希望一个机动化的小车系统来捕捉镜头沿着主直道与他的 GoPro。仅使用废物箱零件，他创造了一个简单的[踏板操作钢筋混凝土电缆小车](https://www.youtube.com/watch?v=lHZS-Cxf-VE)。(视频，嵌在下面。)

[老式后院钢筋混凝土]首先试验了一辆高速汽车在一段模型火车轨道上运行。但是，它在高速上颠簸，轨道昂贵，需要 50 V 通过开放的轨道。新的电缆凸轮给了一个更平稳的乘坐，几乎没有成本与他的旧钢筋混凝土齿轮供应。电缆摄像头由遥控飞机上的有刷电机驱动，通过一些杂草修剪机线上的塑料轮运行。控制由一个旧的 27 MHz RC 系统提供，控制器的内部被移植到一个旧的哇哇吉他踏板上。

非齿轮电机可以驱动电缆比要求的快得多，所以[老式后院钢筋混凝土]需要练习一些小心的脚控制，以合理的速度运行它。这说起来容易做起来难，同时还用手控制 RC 汽车，所以他计划用更新的 2.4 GHz 系统软件端点限制来替换 RC 系统。我们可能会接触到 ESP32 或任何其他我们熟悉的无线微控制器，但值得记住的是，大多数人并不熟悉这些工具。

这绝对是我们今年报道过的最简约的有线摄像头，但只是展示了它们的制作有多简单。你可以随时从 3D 打印部件升级到[光滑的折叠框架，并添加](https://hackaday.com/2020/07/19/rc-car-becomes-cable-cam/)[机器视觉和远程视频流](https://hackaday.com/2021/08/04/cablecam-is-an-exercise-in-system-integration/)。
