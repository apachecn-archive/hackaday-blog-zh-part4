# 哦，天啊！我们是无线电工程师！

> 原文：<https://hackaday.com/2022/09/06/retrotechtacular-oh-boy-were-radio-engineers/>

很可惜，周围的“书呆子店”越来越少了。弗莱的不见了。无线电小屋不见了。但是最好的总是那些有垃圾的地方。硅谷对这些地方来说很棒，但它们无处不在。通常，他们通过向维修行业出售零件来赚钱，但他们也有像我们这样的人。在德克萨斯州的休斯顿地区仍然有一家这样的商店。两个原始电子零件出口之一，简称 EPO。穿过那里就像一个旧设备和零件的博物馆，我并不羞于承认我有时会从我家开一个小时的车，只是在过道里逛逛，什么也不需要买。在一次旅行中，我发现了一些以前没有注意到的东西。Remco Caravelle 发射机/接收机。

盒子显然是旧的，收音机的样式也明显是复古的。你可以看出它没有迎合现代市场，因为它提到:“播放业余无线电操作员”，这肯定会迷惑今天的大多数孩子。该装置是一个调幅接收器和一个发射器，配有一个莫尔斯电码键和麦克风。[在](https://www.youtube.com/watch?v=UKA6TNZSdWU)下方的视频中，你可以看到 Remco 公司一款类似产品的当代广告。

这个玩具出现在 1962 年，虽然你“自己制造”，但它不是你必须焊接的东西。不知何故，我小时候错过了这个，尽管我确实有通用电气公司的类似产品(Y7060 CB 基站)。你一定想知道有多少业余无线电爱好者和广播公司推出了这样一个玩具？我们已经了解到[拉什·林堡]因为这个套件而对广播产生了兴趣。就像广告里的小孩说的，“哦，天啊！我们是无线电工程师！”

今天，无线电比那时更加重要。但是像所有成功的技术一样，普通人已经看不到它了。不是这样的，当跨越大西洋打电话的时候是一个奇迹。更不用说[20 世纪 40 年代的车载电话](https://hackaday.com/2020/03/13/retrotechtacular-mobile-phones-1940s-style/)。
