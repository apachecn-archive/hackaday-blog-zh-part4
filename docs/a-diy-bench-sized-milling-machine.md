# 一台 DIY 台式铣床

> 原文：<https://hackaday.com/2019/10/15/a-diy-bench-sized-milling-machine/>

在网上的机械加工社区闲逛，你会更加熟悉破旧的布里奇波特工厂，在这一点上，它实际上是一个迷因。但是磨坊有各种形状和大小，从很久以前的结实的旧铁，到更小、更紧凑的建筑。【Honus】决定建造后者，[并分享了该项目的细节。](https://www.instructables.com/id/Benchtop-Micro-Milling-Machine/)

[Honus]建造的目的是制造一个小型台式磨机，能够处理较小的任务。轧机的框架是由 80/20 挤压材料制成的，并配有大量铝板。Igus 线性滑轨处理 X、Y 和 Z 轴。一个旧的刷过的 Makita 钻机马达作为主轴驱动，由一个连接到 Arduino 的旧的 R/C 速度控制器控制。[Honus]然后根据需要编造各种零碎的东西把它们组合在一起。

这个工厂干净整洁，看起来很适合加工铝。我们认为它在[Honus 的]工作室中会非常有用。如果你在考虑给自己买一些书桌大小的工具，或许也可以考虑雕刻师！休息后的视频。
