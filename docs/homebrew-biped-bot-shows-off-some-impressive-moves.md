# 自制双足机器人展示了一些令人印象深刻的动作

> 原文：<https://hackaday.com/2022/08/12/homebrew-biped-bot-shows-off-some-impressive-moves/>

我们已经在 Hackaday 上看到了足够多的 DIY 机器人平台，知道它们中的大多数都采取了阻力最小的字面和比喻的方式。也就是说，它们往往是某种轮式漫游车。当然，如果你想做一些更有挑战性的事情，还有很多其他形式的运动。

来自[Tast's Robots]的这个[双足机器人就是一个完美的例子](https://hackaday.io/project/185729-upkie-homemade-wheeled-biped-robot)。虽然它在技术上仍然是轮式的，但它的自平衡特性让事情变得更加复杂。它不仅可以直立，还具有独特的下蹲能力，可以通过旋转其电动膝盖和臀部。正如下面的视频所示，它甚至可以穿越相对不平坦的地形——对于这样一个自平衡机器人来说，在硬木和地毯之间实现如此平滑的过渡并不容易。

[![](img/ab3396358315468bdd6fa320b9fa5371.png)](https://hackaday.com/wp-content/uploads/2022/08/upkie_detail.jpg) 但是最好的部分呢？它不仅仅是完全开源的，它还被设计成只使用普通家庭游戏玩家可用的工具和功能来构建。这意味着 3D 打印组件、木榫钉和钢筋混凝土汽车零件。即使是电源，一个 Ryobi 18 V 工具电池，也很容易获得，并且相对黑客友好。

与硬件一样令人印象深刻的是为处理平衡、运动和反向运动学等事情而开发的软件包套件。每一个都是作为它们自己的 Apache 许可的项目来维护和记录的，这使得它们比作为一个整体系统来实现要容易得多。

如果你*真的*想抛弃轮子，[我们在过去见过一些两足步行机](https://hackaday.com/2020/07/03/robotic-biped-walks-on-inverse-kinematics/)。但坦率地说，它们都无法与这个项目的能力和范围相提并论。
