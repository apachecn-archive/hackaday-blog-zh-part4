# 用于脉冲拨号代的移动电话

> 原文:[https://hacka day . com/2020/07/29/a-用于脉冲拨号生成的移动电话/](https://hackaday.com/2020/07/29/a-mobile-phone-for-the-pulse-dial-generation/)

蜂窝网络数据模块无处不在的可用性的一个有益的副作用是，它们可以用于创建定制的移动电话。令人惊讶的是，我们并没有像预期的那样看到很多这样的项目，但通过纠正这种不足，我们很高兴看到[质子游戏玩家]，[的工作，他用一部老式的旋转拨号手机升级了 Arduino 和 GSM shield](https://hackaday.io/project/173968-arduino-mobile-dial-phone)，做出了一个非常出人意料的手机项目。

从报道中不能完全确定哪家制造商生产了捐赠手机，或者它是为哪个国家的网络生产的，但它似乎是你在 20 世纪 60 年代可能在世界各地发现的典型类型。我们得到了各种元件的分类以及如何与它们接口，例如 ringer 就是使用电机驱动板运行的。有全面的转换说明，虽然可悲的是，它们涉及到掏空手机和删除原来的硬件。结果可以在休息下面的视频中看到，完成的项目从最不可能的硬件打了一个手机电话。

这当然不是我们推出的第一款旋转式拨号手机，包括基于会议徽章的。
