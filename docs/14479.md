# 虚拟破解你的引擎

> 原文:[https://hackaday.com/2022/08/11/hack-your-engine-virtually/](https://hackaday.com/2022/08/11/hack-your-engine-virtually/)

众所周知，我们喜欢在构建某些东西之前模拟电路，这方面有很多很好的工具。但是我们这些从事汽车工作的人呢？嗯，你可以试试 engine-sim，这是一个实时的内燃机模拟软件。老实说，该程序坦率地承认，它不够准确，无法做工程或引擎调整。但从好的方面来看，它有音频输出，至少是一个很好的教育工具，可以显示发动机的运行以及不同的参数可能会如何影响它。你可以在下面看到这个工具的视频。

[Ange-Yaghi]提到该代码主要是为 YoutTube 演示提供动力。然而，自述文件暗示它可能会更好——或者至少有所不同——并且欢迎通过合作来使它变得更好。

我们从第一手资料中知道，模拟器不一定要完全准确才能让你感受到事物是如何工作的。毕竟，大多数电子仿真并不总是模拟导线电阻和寄生效应之类的东西。我们将让更懂发动机的人来告诉我们这个模拟器是否足够接近有用。

键盘命令让你切换点火，改变齿轮使用离合器，并改变油门。我们希望看到这样的数据屏幕，用于我们实际的汽车引擎，也许使用其中一个蓝牙适配器，你可以更接近。

我们记得[一个时钟，它也是某种引擎模拟器](https://hackaday.com/tag/engine-simulator/)。如果你的模拟梦更多地集中在电子产品上，那里有很多选择。
