# 这个自制的电动钢锯切割又快又干净

> 原文：<https://hackaday.com/2019/02/15/this-home-made-power-hacksaw-cuts-quick-and-clean/>

如果你在车间里切割金属，你可能会使用台式切割锯，或者为了更精细的工作使用带锯。电动钢锯是一种笨重的装置，看起来和感觉起来都很老式。尽管这种设计有其固有的缺点，但[[Emiel]还是决定建造一种在钻机动力下运行的机器人，它获得了成功。](https://www.youtube.com/watch?v=mrk6pN53kxk)

该建筑使用一个基本的电池供电的钻机作为其电源。它与一根轴相连，轴带动一个联动装置旋转，这与蒸汽机车上看到的联动装置没有什么不同，只是方向相反。在这种情况下，连杆机构将钻头的旋转运动转化为钢锯的线性运动，钢锯在 3D 打印轴承的引导下沿着金属轨道移动。

胶合板和塑料移动部件的主体，这可能不是你的高容量，快节奏工作的选择工具。然而，正如[Emiel]所指出的，这比手工制作要快，而且这是一个有趣的构建，基本上使用了车间里已经存在的东西。[这也不是我们第一次看到电动钢锯使用 3D 打印零件](https://hackaday.com/2016/04/08/3d-printed-linkage-produces-automatic-hacksaw/)。休息后的视频。
