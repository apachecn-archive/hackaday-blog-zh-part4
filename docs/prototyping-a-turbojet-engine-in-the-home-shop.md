# 在家庭作坊制作涡轮喷气发动机原型

> 原文：<https://hackaday.com/2021/03/16/prototyping-a-turbojet-engine-in-the-home-shop/>

涡轮喷气发动机的发展改变了航空业的游戏规则，因为飞机设计师不再需要与越来越大、越来越复杂的活塞发动机作斗争，螺旋桨也不会让飞机停留在音速以下。然而，涡轮喷气发动机是一种精密的设备，为了成功地工作，需要最大限度的材料。在他寻求在家里建造一个机器人的过程中，他发现了这一点。

与大多数家用喷气发动机不同，这款发动机没有使用涡轮增压器，也没有采用更简单的脉冲喷气设计——尽管[Integza]也制造了这些发动机。这是一种合适的径流涡轮喷气发动机设计。该建筑使用 3D 打印的压缩机，这是可能的，因为它不必处理太多的热量。然而，对于涡轮机，[Integza]意识到塑料不能切割它。在使用陶瓷树脂的实验失败后，3D 打印夹具被替代，以允许金属片容易地制作成可工作的涡轮机。其他内部部件由耐热混凝土制成，燃烧室由钢焊接而成。

经过几次尝试后，发动机确实运行了，尽管在部件开始熔化前只运行了十秒钟。虽然发动机离飞行就绪还有很长的路要走，但它显示出建造一台小型涡轮喷气发动机有多困难。多年来，就连世界主要大国也一直在努力解决这个问题。休息后的视频。
