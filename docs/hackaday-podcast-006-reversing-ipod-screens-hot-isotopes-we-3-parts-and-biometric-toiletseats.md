# 黑客日播客 006:反向 IPod 屏幕，热同位素，我们< 3 部分，和生物识别厕所座椅

> 原文：<https://hackaday.com/2019/02/15/hackaday-podcast-006-reversing-ipod-screens-hot-isotopes-we-3-parts-and-biometric-toiletseats/>

本周黑客圈有什么热门话题？Hackaday 的编辑埃利奥特·威廉姆斯和迈克·什奇斯回顾了过去七天他们最喜欢的黑客和文章。在第六集中，我们讲述了迈克·哈里森在 iPod nano 替换屏幕上所做的令人难以置信的逆向工程工作。我们涉足深空电源的放射性世界，花一些时间崇拜零件和零件制造商，并以非常高雅的眼光看待马桶座圈技术。在我们的 quickfire hacks 中，我们讨论了连贯的声音(可以认为它类似于层流，但用于音频)、嵌入式的最小 ide、金属加工的手工工具以及可以实现这一点的小型 ESP32 机器人。
