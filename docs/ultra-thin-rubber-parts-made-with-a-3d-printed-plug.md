# 用 3D 打印插头制成的超薄橡胶部件

> 原文：<https://hackaday.com/2022/09/09/ultra-thin-rubber-parts-made-with-a-3d-printed-plug/>

我们通常认为 3D 打印组件是硬塑料，因为在大多数情况下，这就是我们在台式机中装载的东西。但是除了普通的 PLA、PETG 和 ABS 之外，你还可以用各种柔性细丝如 TPU 进行打印。这对于生产定制密封件或坚固外壳非常方便。

但是如果你想制造非常薄的橡胶部件呢？在这种情况下，大多数台式机上的 0.4 毫米喷嘴将成为您的限制因素。但丹尼尔·鲍恩在他最新的[*视频*](https://www.youtube.com/watch?v=jWt2ynIMgoM)*中展示的方法却并非如此。这里的技巧是，打印机没有生产最终产品——它正在制造一种水溶性插头，对于手头的应用来说，这种插头的尺寸略小。*

 *[![](img/70b2c7367d18a96c9691cab692a5470f.png)](https://hackaday.com/wp-content/uploads/2022/09/thinrubber_detail.jpg) 一旦塞子印好，【丹尼尔】就会在上面喷上几层塑料蘸酱。这在印刷部分建立了一个橡胶涂层，一旦它达到了期望的厚度，整个东西被扔进超声波清洁器来打碎细丝。你剩下的是一个类似硅胶的部分，它的形状与你的原始印刷品相同，但比你正常挤出的任何东西都薄得多。

那么[丹尼尔]打算用这项技术完成什么呢？我们承认这个物体的形状很有启发性，但在这种情况下，尺寸留给我们的问题比答案更多。也许我们会在下一个视频中了解更多，我们被告知会看到插头被浸入乳胶中。

如果减法制造更适合你，你总是可以[冻结一片橡胶，然后用 CNC](https://hackaday.com/2014/09/22/cryogenic-machining-custom-rubber-parts/) 从上面切割出定制的零件。
