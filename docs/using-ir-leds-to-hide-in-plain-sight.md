# 使用红外发光二极管隐藏在显眼的地方

> 原文：<https://hackaday.com/2020/02/28/using-ir-leds-to-hide-in-plain-sight/>

如今，不被监控摄像头监视似乎是不可能的——从商场到街角，组织使用监控摄像头来检查顾客变得越来越普遍。虽然集会自由在《美国状况》和《世界人权宣言》等文件中被视为一项基本人权，但它并不是一项在世界各地都受到尊重的权利。通常情况下，执行命令的政府会使用图像识别程序识别个人，防止他们集会或示威反对政府。

工程师[Nick Bild]建造的自由之盾试图打破现状，让人们选择是否愿意被看到。人类可见的辐射光谱在 740 纳米左右达到最大，这使得正常的观察者无法检测到红外波。

该项目使用嵌入衣服中的 940 纳米红外(IR)发光二极管，以压倒用于监控的红外敏感相机中的光电二极管。由于光的波长对人类来说是不可见的，它们不会妨碍正常的行为，这使得它成为一种隐藏在众目睽睽之下的理想方式。当然，使用 SMD LEDs 而不是更大尺寸的 led 也有助于使光更少被肉眼看到。

这个结果并没有完全遮住你的脸，但作为一个概念验证，它肯定是一个如何避免被跟踪的例子。
