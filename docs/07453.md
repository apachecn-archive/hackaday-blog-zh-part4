# 移动冰箱贴打造独特的时钟

> 原文:[https://hack aday . com/2020/08/14/moving-ICEF-magness-make-for-unique-clock/](https://hackaday.com/2020/08/14/moving-fridge-magnets-make-for-unique-clock/)

我们看到围绕这些部分的大量时钟项目，作为黑客，我们喜欢展示它们。但是，我们偶尔会发现一种伟大的新方法来显示时间，这种方法真正引起了我们的注意，并需要更仔细的观察，例如[这个移动的冰箱磁铁时钟](https://hackaday.io/project/174246-refrigerator-magnet-clock)。

克雷格·科尔文(Craig Colvin)制作的这个独特的时钟周围的冰箱磁铁是五颜六色的塑料，多年来一直装饰在蹒跚学步的家庭的冰箱下部。[克雷格]没有把磁铁放在冰箱上，而是把一片白色丙烯酸层压到一片薄钢板上，让磁铁有东西可以吸附。移动数字是 CoreXY 式机械装置 T1 的工作。皮带驱动的笛卡儿运动将头部移动到正确的位置来拾取数字；头部的伺服系统将两块强力磁铁移动到数字下方的位置。然后，头部将数字移动到正确的位置，释放磁铁，数字就留在板上。休息过后，您可以在视频中看到它的运行。

虽然我们喜欢这样，但它让我们想起了一些伟大的 mod。人们可以想象添加字母来制作一个合法的单词时钟，或者只是添加一个日历显示。我们也希望通过将这种磁铁安装到冰箱门上，在它们的自然栖息地看到它们。
