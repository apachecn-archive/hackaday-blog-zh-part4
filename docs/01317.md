# 观看低成本机电一体化实验室分发糖果，分拣杯子

> 原文:[https://hackaday . com/2018/11/26/watch-the-low-cost-mechatronics-lab-dispose-candy-sort-cups/](https://hackaday.com/2018/11/26/watch-the-low-cost-mechatronics-lab-dispense-candy-sort-cups/)

当简单的电机和直线运动与正确的机械设计和控制系统匹配时，可以做很多事情。教授这些原则是 [LCMT(低成本机电一体化培训师)](https://www.maryland.design/lcmt)背后的目标，它主要是作为一种教育工具。LCMT 采取“边做边学”的方法，通过创建一个系统来教授各种原则，该系统从漏斗中取出一个杯子，从分配器中装满糖果，然后根据颜色对杯子进行分类，所有这些都是通过使用相对简单的系统的适当组合来完成的。

这种低成本的机电一体化训练器造价不到 1000 美元，是美国马里兰州安妮阿伦德尔社区学院一个团队的杰作。LCMT 显然不是一次性项目；网站上有完整的 CAD 文件和构建文档，以及针对教育工作者的完整实验指南。

下面嵌入了组装好的系统的演示视频，由[Tim Callinan]完成演示。值得一看的是，这个系统的设计有多简洁，你们当中的视觉学习者可以通过观察这个系统的动作来学习一二。
