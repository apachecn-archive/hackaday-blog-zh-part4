# 定制 RC 控制器设计指南

> 原文：<https://hackaday.com/2021/11/12/a-guide-to-designing-a-custom-rc-controller/>

这些天来，有各种形状和大小的钢筋混凝土控制器吨。然而，如果您想为您的应用程序构建一些带有适量按钮和控制杆的东西，您可能想自己设计一些东西。这正是[塞巴斯蒂安]所做的。

该项目实际上在一段时间前就开始了，【Sebastian】分享了他通过使用粘土和摄影测量建造定制人体工程学外壳的过程，[我们在](https://hackaday.com/2021/07/14/design-a-custom-enclosure-using-modelling-clay-and-photogrammetry/)之前已经报道过。

在这个造型优美的外壳内，该建筑依靠 STM32 微控制器，连接着一系列电位计、按钮和拇指操纵杆(更多电位计)。一个 NRF24L01 模块是用来处理无线电传输方面的事情。

总的来说，[Sebastian]已经制作了一个很好的指南来从头开始设计一个定制的 RC 控制器，而不是简单地指导一个人如何复制他自己的构建。有了这些技能，任何制造商都应该能够制造出完全定制的控制器。休息后的视频。
