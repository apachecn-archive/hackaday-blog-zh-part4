# 一把模糊的光剑

> 原文：<https://hackaday.com/2019/01/04/a-foggy-lightsaber-build/>

自 1977 年光剑出现在第一部《星球大战》电影中以来，它就一直吸引着观众。不幸的是，自那以后的几年里，乔治·卢卡斯没有与更广泛的公众分享这项技术，所以我们只能靠苍白的仿制品生存。这就是这样一个版本。

人类最接近绝地技术的模拟是激光，这个建筑使用了 8 个激光和两个发光二极管。它们的目的是在剑柄上方的固定距离重合。然后用二氧化碳自行车打气筒吹电子烟来制造烟雾。这使得红色激光很容易被人眼看到。

这把仿制光剑确实有其局限性——快速运动往往会驱散雾气，使其再次隐形，而且它在垂直方向上确实处于最佳状态。焦点之外也有一些分歧。也就是说，当它静止不动，闷烧着的时候，看起来确实有些令人印象深刻。

在我们更好地掌握等离子物理学之前，也许你可以用这种基于火的建筑？休息后的视频。
