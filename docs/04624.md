# 从零开始制作青铜大炮

> 原文:[https://hack aday . com/2019/10/27/从零开始制作青铜炮/](https://hackaday.com/2019/10/27/making-a-bronze-cannon-from-scratch/)

在家铸造金属有点棘手，但不可否认的是，结果可能是相当有益的。[FarmCraft101]展示了他令人难以置信的工艺，并在这个过程中学到了一些新东西，通过 [scratch 建造了一个比例复制品青铜大炮和马车](https://www.youtube.com/watch?v=9kJxOn9NU9w)。

从足够数量的废金属开始，他首先生产了青铜锭。得到真正的选角权需要多次尝试。首先尝试了消失模铸造，失败的很惨，但提供了一个样本金属，通过拉伸强度测试。第二次尝试是使用一个木桶模型和一个组合模具，水平铸造，导致桶的顶部收缩。第三次尝试，垂直排列，几乎导致了一个高风险的游戏“地板是熔岩”，在铸造过程中模具裂开后，熔化的青铜从他的车库地板上倾泻而出。

第四次尝试终于成功了，再次使用垂直模具，但夹紧更加牢固。然后，在车床的帮助下，这个毛坯桶被钻了出来，并加工成一面漂亮的镜子。然后他把注意力转向了马车，它本身就是一个真正的美人，拥有定制的马车车轮，带有烧焦的木材饰面和亚麻子油涂层。

您可以在休息后查看构建视频，但我们现在要警告您，[FarmCraft101]实际上从未发射过这个华丽的创作。如果你想尝试自己动手做炮弹工程，并拥有一台 3D 打印机，你可能[想尝试一下 lost PLA 铸造](https://hackaday.com/2017/01/02/lost-pla-casting-brings-out-the-beauty-of-macromolecules/)，或者进入[批量生产一些自己动手做的硅胶模具](https://hackaday.com/2017/08/09/silicone-molds-for-stove-top-metal-casting/)。
