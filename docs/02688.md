# WiFi 控制的飞机是廉价的飞行乐趣

> 原文:[https://hack aday . com/2019/04/12/wifi 控制的飞机是廉价的飞行乐趣/](https://hackaday.com/2019/04/12/wifi-controlled-plane-is-cheap-flying-fun/)

无线电控制飞机的世界曾经是一个昂贵的专属爱好，仅限于那些有时间和金钱投资困难建造和昂贵无线电设备的人。最近，这种爱好变得更容易获得，可以在郊区公园等较小空间飞行的廉价现成飞机已经上市。Ravi Butani 制造了这样一架飞机，你甚至可以用你的智能手机驾驶它！

ESP8266 在这里有双重功能，既是大脑又是通讯系统。一个定制的智能手机应用程序通过 WiFi 与飞机通信。触摸屏幕增加油门，而转向是通过倾斜手机来实现的。还有对信号强度和电池电量的监控，如果飞机超出范围或电池电量低，手机就会振动。

飞行控制是通过差动推力实现的，动力来自由微型 SMD MOSFETs 控制的两个小型 DC 发动机。这架飞机在静止的条件下飞行得非常好，在开放的公园环境中，WiFi 连接也很稳定。[Ravi]报道称，使用摩托罗拉 G5S 智能手机可以在大约 70 米的范围内进行控制。

尽管构建简单，组件成本低，但最终产品的表现令人钦佩。这将是一个伟大的周末项目，在结束时，你可以去驾驶你的新飞机！如果你担心电池充电，不要担心——[有一个解决方案。休息后的视频。](https://hackaday.com/2018/10/03/keep-em-flying-with-this-monster-diy-battery/)
