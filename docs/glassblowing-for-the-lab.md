# 实验室玻璃吹制

> 原文：<https://hackaday.com/2022/03/04/glassblowing-for-the-lab/>

曾经有一段时间，从经销商那里订购一些玻璃器皿意味着下一个相当大的最小订单，邮寄一张支票，并等待数周，等待一箱完整的玻璃器皿到来。在那些日子里，从玻璃管中吹制自己的玻璃器皿是相当普遍的，[惠勒科学]已经就如何做到这一点做了一系列的报道。即使你对建立一个化学实验室不感兴趣，你可能会发现最新一集的[制作气体放电管](https://www.youtube.com/watch?v=MYEVXMYKBAc)值得一看。有几个视频，你可以在下面看到其中的几个。

当然，吹玻璃是字面上的玩火，一定要小心。最重要的规则？不要吸气。话说回来，对于很多事情来说，吹玻璃并不涉及你实际上吹，但它更像是弯曲和成型，从技术上来说，他展示的是灯具，而不是实际的玻璃吹制，但这是一个技术问题。

当然，你需要一个火炬，视频中的火炬是一个丙烷燃料的火炬，还有一个氧气罐。制造气体放电管还需要一些额外的硬件，包括一个氩气瓶和一个两级真空泵。你还需要一种方法来激励电子管，但既然你可能已经有了一个特斯拉线圈，那就不成问题了。

如果你想手工制作试管和一些其他物品，还有其他视频。我们等着看接下来会发生什么。如果你对玻璃吹制更感兴趣，看看[【火炬吉米】](https://hackaday.com/2013/06/05/jimmy-the-torch-making-blown-glass/)。小心点。玻璃吹制可能是让[成为自己的电子管](https://hackaday.com/2016/05/04/home-brew-vacuum-tubes-are-easier-than-you-think/)的入门药品。
