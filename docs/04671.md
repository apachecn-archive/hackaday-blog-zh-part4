# 在这个飞行建筑中，注射器变成了火箭

> 原文：<https://hackaday.com/2019/10/30/syringes-becomes-rockets-in-this-flying-build/>

注射器在车间里有各种有用的应用，从注射液体到帮助拾取微小的元件。然而，总有一点轻浮的余地，于是[汤姆·斯坦顿]决定[尝试一些注射器火箭建造](https://www.youtube.com/watch?v=Rbxa-3QVNU4)。

基本的想法是堵住注射器的末端，然后拉动活塞在管内形成真空。当被释放时，柱塞将从抵消真空的大气压力中向前冲，撞击管子的端部并向前推出注射器。

[Tom]最初尝试使用小注射器很有趣，但较大的注射器在破损、密封问题和超重等问题上举步维艰。真空加农炮要幸运得多，它能够将炮弹发射到一个合适的高度，尽管没有太多的稳定性。[Tom]设计了一个小型的 3D 打印发射器，可以安装 10 毫米的注射器，让你在车间里随意射击。

看到这个概念被详细探究是很有趣的，因为[汤姆]在解释这一现象背后的基本物理学方面做得很好。如果你渴望更多，考虑使用注射器作为模型构建的基本液压致动器。休息后的视频。
