# 你会被这辆火箭联盟的遥控车吓到的

> 原文:[https://hackaday . com/2021/01/15/you-to-flip-out-over-this-rocket-league-RC-car/](https://hackaday.com/2021/01/15/youre-going-to-flip-out-over-this-rocket-league-rc-car/)

火箭联盟是一款以广受欢迎而闻名的视频游戏，尽管没有几个小时的练习几乎无法玩。它包括超高速汽车踢足球，完成翻转，跳跃和火箭助推到球的能力。[mrak_ripple]决定他想要一辆真正的遥控汽车，[并开始工作。](https://www.youtube.com/watch?v=bPFBVjJP97c)

虽然火箭助推超出了这个版本的范围，[mrak_ripple]很有信心他可以建造一辆模仿*火箭联盟*辛烷车的跳跃、翻转的遥控汽车。最初的实验涉及一个定制的 3D 打印弹簧机制，但结果并不令人满意。相反，本着真正的黑客精神，从现有的玩具中取出一个跳跃机制，安装在汽车上。这与一个由无刷电机和飞轮组成的机制相结合，在半空中产生翻转力矩。

最终的结果令人印象深刻，汽车翻转相对干净，一旦改进和减轻其原始设计。我们希望看到一个两轴建筑，也可以前空翻和后空翻。与我们上次看到的[mrak_ripple]，[有趣的土豆泥投石机](https://hackaday.com/2020/11/12/quick-and-dirty-trebuchet-flings-mashed-potato/)相比，这是一个复杂的升级。休息后的视频。
