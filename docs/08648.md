# 自制的赛车游戏几乎可以从头开始做任何事情

> 原文：<https://hackaday.com/2020/12/12/homebuilt-racing-sim-does-almost-everything-from-scratch/>

如果你想要一个 sim 游戏装备，市场上有现成的选择，只要你的预算能够承受。然而，有些人选择避开这条路线，从头开始建立自己的路线。然而，很少有人走得像[Popicasa POPStuDio]这样远。

第一个版本的钻机是你所能得到的最大的黑客攻击，很高兴看到它是从废料中建造的。车轮本身和踏板都是用旧 PVC 管制成的，用一堆旧木头拧在一起作为框架。一个便宜的 USB 游戏手柄用来处理输入到 PC 的踏板和 H-shifter。H-shifter 使用简单的电源开关，以一种巧妙的方式重新用于感应档位。旋钮本身是由看起来像是热胶水的东西铸造而成的。转向是通过将车轮连接到一根柔性轴上来实现的，这根柔性轴可以使智能手机前后倾斜，使用其内部的加速度计和陀螺仪来感应旋转。目前还不清楚这是如何与运行项目汽车的 PC 联系在一起的，但这仍然令人印象深刻。

[版本 2 更上一层楼](https://www.youtube.com/watch?v=cHVQUGKmQe8)，使用 Arduino Leonardo 作为人机界面设备来处理转向和踏板功能。还有力反馈，通过皮带传动连接到转向轴的大电机。这个版本实现了一个 H-shifter 以及一个更现代的体验换档。

这两种构造在现代都是独一无二的，因为它们避开了 CNC 或 3D 打印零件。这都是手工完成的，需要几天的努力，并且只使用基本的工具。看到这样一个复杂的建筑只用简单的材料和纯粹的投入就完成了，真是令人耳目一新。我们确信[Popicasa POPStuDio]喜欢这个平台，我们迫不及待地想知道它下一步会走向何方。也许下一个版本[甚至会有一个运动平台，](https://hackaday.com/2018/10/04/homebrew-linear-actuators-put-the-moves-on-this-motion-simulator/)可能是用旧的叉车零件建造的？只有时间能证明一切。休息后的视频。
