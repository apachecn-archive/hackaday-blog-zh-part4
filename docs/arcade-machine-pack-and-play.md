# 街机打包播放

> 原文：<https://hackaday.com/2021/11/14/arcade-machine-pack-and-play/>

街机的大木箱给房间增添了自信。又高又重的盒子的问题是，它占据了相当多的空间，很容易吸引眼球。所以[Alexandre Chappel] [着手避免这种情况，并建造了一台可以隐藏在众目睽睽之下的街机。](https://www.youtube.com/watch?v=OB8VauGEvqg)

[![](img/5795fddc66a36f91bc3febbd4029ef1b.png)](https://hackaday.com/wp-content/uploads/2021/11/hiddenarcade_detail.jpg)

Extra points awarded for neat wiring on the inside.

这个创意是一个挂在墙上的木箱，不用时可以折叠起来。[Alex]从切割成面板的波罗的海桦木胶合板开始。接下来，他使用封边条(一种背面涂有胶水的薄单板)，使所有暴露的边缘看起来像天然木材。接下来，在面框上打一个筛孔，让液晶显示器可以舒适地放在里面。口袋孔和饼干的组合让[Alex]不用任何可见的螺丝或紧固件就能组装所有东西。

在 3D 打印机的帮助下，他很快制造了一个锁定机制，以便在折叠时保持前面板的连接。铰链也是 3D 打印的。典型的 Raspberry Pi 4 为这台特殊的机器提供动力。两个法国夹板将盒子固定在墙上，一旦系统安装在墙上，我们不得不说它看起来令人难以置信。

如果你正在寻找一个更小但更传统的街机柜，为什么不看看这个适合初学走路的孩子的街机柜呢？或者，如果你喜欢隐藏的拱廊的实木外观，[这个全尺寸的实心橡木橱柜会是你喜欢的东西](https://hackaday.com/2021/01/08/solid-oak-arcade-cabinet-when-particle-board-wont-do/)。休息后的视频。
