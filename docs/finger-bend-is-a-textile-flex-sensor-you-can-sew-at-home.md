# 手指弯曲是一种纺织品弯曲传感器，你可以在家里缝

> 原文：<https://hackaday.com/2021/04/02/finger-bend-is-a-textile-flex-sensor-you-can-sew-at-home/>

因此，我们经常使用电子设备的控制装置，涉及到我们的手指直接抓取、触摸或移动另一个物体或表面。我们使用直接检测身体运动的界面不太常见。柔性传感器是实现这一点的一种方式，[这正是[WillpowerStudios]在手指弯曲方面的目标。](https://www.instructables.com/Finger-Bend/)

该传感器的结构很简单，使用压阻织物，当变形时其电阻会发生变化。通过将它缝合到一个可以放在手指上的护套中，并用导电线将其连接起来，它可以用于检测佩戴者手指的弯曲，方法是通过任何普通微控制器上的模数转换器对电阻进行采样。将这项技术扩展到全手，只需在每个手指上弯曲一个手指，然后将每个手指连接到各自的 ADC 通道。如果你想变得非常有趣，你甚至可以用多路复用器快速扫描它们。
