# 用激光进行微加工

> 原文：<https://hackaday.com/2022/04/19/micromachining-with-a-laser/>

[Breaking Taps]有一个很好的脉冲光纤激光器，并决定尝试用硅进行[微加工。你可以在下面的视频中看到结果。硅能很好地吸收激光的红外线，尽管硅的物理特性还有待改进。他还在改进钢、铜和黄铜的加工工艺，这可能会更实用一些。](https://www.youtube.com/watch?v=TrYhy8Pll-0)

激光的脉冲持续时间很短，但脉冲具有很大的能量。这是实验性的，所以有些测试不太好，但有些——比如齿轮——看起来很棒。

这可能对你 200 美元的中国激光雕刻机不起作用。但是如果你有合适的激光器(他的是普通的，价格在 4000-6000 美元之间)，关于焦距和定时的提示可能会帮你省去一些自己的实验。

我们必须承认我们有点嫉妒。他不仅可以使用相当好的激光，还可以使用扫描电子显微镜，它比我们过去使用的质量更好，我们甚至再也无法使用它们了。你可以在视频中看到在 SEM 下对零件进行的大量分析。

如果你想证明光纤激光器的价格是合理的，他们确实制造了漂亮的印刷电路板。我们甚至已经看到了一点这些[激光器如何在内部工作](https://hackaday.com/2019/12/02/fun-with-a-200-kw-fiber-laser/)。
