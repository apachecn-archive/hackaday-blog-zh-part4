# 测量原子

> 原文：<https://hackaday.com/2021/04/28/measuring-an-atom/>

你需要一个设备齐全的实验室来[测量一个原子的大小](https://stoppi-homemade-physics.de/atomgroesse/)(德语，[机器翻译](https://translate.google.com/translate?sl=auto&tl=en&u=https://stoppi-homemade-physics.de/atomgroesse/))吗？根据斯托皮的说法，不。你需要葵花籽油、一些熊藓孢子和一点汽油。你还需要一些日常用品，比如注射器、烤盘和罐子。你可以在下面的视频中看到整个过程。这个测量并不是真的针对一个特定的原子，但它是一个脂质分子的平均值，这仍然令人印象深刻。

你实际上是在测量一滴油滴在水面上的直径。由于石油主要是油酸，层的高度是 167 个原子。之后，是一些简单的测量和数学，得到高度，找到平均原子高度。

该测量依赖于油酸是一种脂质并且部分疏水部分亲水的事实。这意味着当油酸分子遇到水时，它会以特定的方向站立。一滴油会扩散到一个分子的厚度。如果你知道你引入的油的体积，考虑到圆盘的体积必须与你开始时的体积相同，你可以从数学上推导出油滴扩散的高度。

虽然你的里程可能会有所不同，但[stoppi]通过取薄膜的高度并除以薄膜中的原子数，得到了 1.76 埃。也就是每原子 176 微微米。如果你找不到孢子，已知滑石粉在这种应用中也有效，你也可以用酒精代替汽油。你可能会发现把粉末放在盐罐里分发会很方便。你要轻敲托盘，以确保油滴尽可能地扩散。这也是一个好主意，采取平均直径测量，因为你的油滴可能不会是完美的圆形。

这个过程并不新鲜。1932 年，[欧文·朗缪尔]因为包括这一方法的工作获得了诺贝尔奖。如果你更想看到你的分子，[试试 3D 打印](https://hackaday.com/2017/01/02/lost-pla-casting-brings-out-the-beauty-of-macromolecules/)。如果你是一个测量物质宇宙的爱好者，为什么不试试寻找光速的 T2 T3 呢？
