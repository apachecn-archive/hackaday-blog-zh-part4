# 竹签发射飞艇

> 原文：<https://hackaday.com/2019/11/19/bamboo-skewers-launch-airship/>

我们不得不承认，我们喜欢飞艇。一艘庄严的齐柏林飞艇漂浮在曼哈顿上空的形象让我们想象未来。飞艇已经不多了，但是你可以自己建造。[狡猾的机器人]展示了如何使用他们的一块板来制作一个简单而容易控制的气球。老实说，他们没有给你很多细节，但我们知道如何转动电机和伺服系统。我们喜欢他们用热胶水和竹子做的建筑。有效，而且说起来很有趣。

竹串很容易找到，是一个轻量级的框架。一些无人机马达提供推力，一些简单的遥控伺服系统控制道具的角度。简单明了。

我们真的很喜欢框架如此之小，而且你可以在几分钟内将它安装在任何类型的控制器板上。氦离子也很容易得到。你甚至可以让 goop 放在里面(或者让它放在里面),让它们比以前更耐用。

如果你想自主飞行，任何控制板都能帮你实现。该视频使用了一个手机应用程序，我们认为这是他们推广的智能机器人的一部分，但老实说，有很多其他方法可以实现这一点。

想到了 Blynk 。或者用 [websockets](https://hackaday.com/2017/12/23/control-a-quadcopter-over-websockets/) 自己卷。我们在想，如果你安装三台相隔 60 度的电机，你可能会得到非常好的控制，特别是如果你有一个光学流量传感器或类似的东西来闭合环路。
