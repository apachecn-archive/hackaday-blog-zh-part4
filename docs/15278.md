机械键盘爱好有各种各样的途径，其中一种比较常见的途径是在同事之间进行。[crsayen]又名[DrJamesOIncandenza]就有一段这样的关系，不久前，他的一个同事打开了 CRKBD。对于外行人来说，这是 Corne keyboard 的简称，它是一个列交错的 3×6 分离式键盘，每只手有三个拇指键。

虽然[JOI]非常喜欢 Corne，尤其是 chocs，但他偶尔会渴望拍打(这就是我们从现在开始对 linears 打字的称呼——拍打与敲击)一些无声的 MX 开关[,因此建造了这个 Corne 衍生的单体分离](https://www.reddit.com/r/ErgoMechKeyboards/comments/y9gfe4/i_designed_my_first_keyboard_and_couldnt_wait_for/),称为 Keyboard One。除了更明显的差异之外，

我们认为对于第一个键盘来说，这看起来相当快，有点像稍微小一点的爱丽丝什么的。但是你不需要大费周章去获取【JOI】的知识，因为[所有的东西都在 GitHub](https://github.com/crsayen/keyboard-one) 上。[JOI]说他们有点后悔使用 3×6，但是已经在计划另一个有更多钥匙的版本。看，这就是精神。
