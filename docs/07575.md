# 暴死彩虹排序游戏露出你的真面目

> 原文：<https://hackaday.com/2020/08/30/__trashed-38/>

当[马茨曼]八岁的女儿想出一个他们可以一起玩的彩虹游戏时，他全神贯注。毕竟，她是认证的彩虹专家。他们在 R&D 开了几次会，想出了[一款彩虹排序速度测试游戏，看起来玩起来很有趣，构建起来也相当简单。](http://www.instructables.com/id/Tap-Tap-Rainbow-a-2-Player-Quick-Reaction-Game/)

按下侧面蓝色按钮，顶部的 RGB LEDs 灯会随机排列。这个游戏的目标很简单——只要按下每个 LED 对应的街机按钮，将彩虹排在其他玩家之前。谁的排序更快，谁就会在他们的一组酷酷的透明按钮后面得到一个彩虹动画的奖励。

在激光切割的盒子里是一个 Espruino，这是一个方便的小型微控制器，会说 JavaScript。所有的街机按钮都连接成一个按键矩阵。你们当中精明的人已经注意到每样东西都有六个，那是因为靛蓝光太难与蓝色区分开来。休息之后看看紧张的游戏。

如果[马兹曼]的女儿想学计算机科学，[彩虹也是个好主意](https://hackaday.com/2020/02/09/sort-the-rainbow-with-an-algorithm-machine/)。
