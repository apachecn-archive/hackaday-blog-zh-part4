# 新手程序员可以创建经典的游戏男孩游戏

> 原文:[https://hack aday . com/2019/05/22/新手-编码者-可以创建-经典-游戏-男孩-游戏/](https://hackaday.com/2019/05/22/novice-coders-can-create-classic-game-boy-games/)

制作一个现代电子游戏需要大量的工作。通常整个公司会花(至少)几个月的时间来开发游戏，选择或编程引擎，并解决问题。不过，这种工作量对于旧的视频游戏系统来说并不是绝对必要的，自制软件开发者通常能够独自为经典系统开发完整的游戏。在过去，这可能需要一些特殊的软件、编程知识，可能还需要硬件，但现在任何人都可以以最低的门槛为最初的 Game Boy 开发游戏。

该项目被称为 GB Studio ，允许人们在没有编程知识的情况下为 8 位手持系统开发自制游戏。一旦建成，游戏可以在任何模拟器上玩，甚至可以加载到一个插件上，在原始硬件上玩(如果有 flash cart 的话)。可以用任何可以创建`.png`图像的东西来创建图形，也有一些功能允许在网络浏览器或移动设备上玩游戏。

虽然游戏似乎仅限于 RPG 风格的游戏，但这仍然是一个令人印象深刻的壮举，对任何对游戏开发感兴趣的人都非常有用。如果它让游戏代码对用户可用，它也可以成为更复杂的游戏编程的入口。它甚至可能导致类似于[在原始硬件](https://hackaday.com/2014/12/30/game-boy-cartridge-emulator-uses-stm32/)上模拟整个墨盒的事情。
