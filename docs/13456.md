# 手工打造的金属鼠标雕刻精美

> 原文:[https://hackaday . com/2022/04/26/手工打造金属鼠标雕刻精美/](https://hackaday.com/2022/04/26/hand-built-metal-mouse-is-beautifully-engraved/)

电脑鼠标就像电脑本身一样，过去几乎都是用难看的米色设计制造的。这些天来，事情变得更时髦了，但他们仍然在很大程度上遵循一个简单的塑料公式。[Uri Tuchman]决定打造一款别致的金属雕刻电脑鼠标[,让桌面更加时尚。](https://www.youtube.com/watch?v=JrICvFcPJ_A)

构建从一个简单的三键滚动鼠标开始，因为在电子产品方面重新发明轮子真的没有意义。内部的 PCB 被拉出并组装在黄铜基板上，同时根据需要为鼠标滚轮提供支架和支撑。它搭配了一个沉重的黄铜外壳，有一个很好的缓坡，可以很好地握在手中。或者，尽可能地，考虑到成品的方形金属边缘。

建造充满了有趣的细节，像[Uri]试图用手形成一个六角形轴，雕刻的工作也同样令人印象深刻。无论如何，这是一个可以和蒸汽朋克键盘完美搭配的版本。或者，如果你讨厌手工雕刻，考虑一下[制造自己的数控机床。](https://hackaday.com/2019/07/23/desk-sized-cnc-engraver-does-the-job/)休息后的视频。
