# 在爆发力的帮助下争取本垒打记录

> 原文:[https://hack aday . com/2020/10/27/going-for the home-run-record-with-explosive-help/](https://hackaday.com/2020/10/27/going-for-the-home-run-record-with-explosive-help/)

疯狂工程师的棒球本垒打距离挑战真的升温了，两个主要的(只有？)竞争对手联手。[Stuff Made Here]的[Shane]和[Smarter Every Day]的[Destin]深入研究了[Shane]最新的粉末充电棒球棒，该球棒旨在击球 600 多英尺。

[谢恩]这次制造了两个新版本的球棒，利用了他从以前的 V1 和 V2 爆炸球棒中学到的经验。它仍然使用空白墨盒，但这次最大容量从三个增加到四个。在 V3 中，球棒的一部分被移除了，代之以四连杆机构，这使得球棒的整个前端都可以移动。该联动装置集成了一个用于四个空弹壳的弹膛，可以像双管猎枪一样装填弹药，并以令人满意的咔哒声关闭。不幸的是，移动部分的质量对于焊接来说太大了，整个前部在第一次测试中就断裂了，所以设计被废弃了。

V4 回到了最初版本的活塞概念，除了 V4 包含两个平行的活塞，在金属球棒中，具有更大的击球面。使用两个墨盒时，它工作得很好，但使用三个和四个墨盒时，零件开始损坏，需要多次设计更新才能修复。[【Destin】报道了项目](https://www.youtube.com/watch?v=6zHegXvEwdI)的物理原理，并拍摄了一些非常酷的高速视频。他和[杰里米·菲尔丁]用他们疯狂的击球手保持着目前 617 英尺的距离记录。不幸的是，在[谢恩]的最后一次远距离尝试中，球棒再次折断，球消失在 600 英尺标记之外的一片高草中，所以他们无法确认记录是否真的被打破了。

[Destin]和他的团队仍然是无可争议的棒球速度记录保持者，他们拥有[超音速棒球佳能](https://hackaday.com/2020/09/11/making-baseballs-go-supersonic/)。听起来[Destin]和[Shane]未来可能会有另一次合作，我们非常期待那次疯狂冒险的结果。
