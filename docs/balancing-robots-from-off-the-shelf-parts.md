# 用现成的零件平衡机器人

> 原文：<https://hackaday.com/2018/10/17/balancing-robots-from-off-the-shelf-parts/>

在这个时代，就电子爱好而言，我们真的很幸运。陀螺仪和电机控制器等先进模块随时可用，不只是作为单个部件，而是作为预焊接模块，可以以最少的麻烦和低成本连接在一起。这个简单的平衡机器人是一个很好的例子，展示了用这些部件可以做什么 ( [谷歌翻译链接](https://translate.google.com.ar/translate?hl=&sl=ja&tl=en&u=http%3A%2F%2Fwww.love-hokuto.com%2F%3Fp%3D7069))。

这个机器人有一个运行节目的 ESP32，它提供了所需的处理能力，以及用于从智能手机控制机器人的 WiFi 接口。这是通过使用开源机器人教学资源 JJRobots 的应用程序实现的。步进电机由来自亚马逊的 DRV8825 模块控制，MPU6050 陀螺仪完成主要组件。自然地，源代码可以在 GitHub 上[获得，以满足你的阅读乐趣。](https://github.com/ghmartin77/B-ROBOT_EVO2_ESP32)

值得注意的是，在今天这个时代，有可能建立这样一个项目，几乎不需要焊接。有了信用卡和充足的补丁程序，就有可能很快完成复杂的数字项目。我们以前也见过类似的方法。休息后的视频。
