# 未能制造铁磁流体

> 原文：<https://hackaday.com/2018/10/26/failing-at-making-ferrofluid/>

[NileRed]承认，虽然铁磁流体有实际用途，但他只是想玩玩它，不想支付他在加拿大发现的高昂价格。他发现的很多说明都不是关于制造真正的铁磁流体的。他开始创造真实的东西，但他并不完全成功。你可以在下面的视频中看到[结果](https://www.youtube.com/watch?v=uLAeHVRxhAU)——一点也不差。

我们总是说你从失败中学到的比成功更多。制造铁磁流体的过程包括两个关键步骤:制造包覆的磁铁矿纳米粒子和去除过大或包覆不当的粒子。在第一次不完全令人满意的尝试后，[niler]试图使用溶剂和磁铁来纯化材料，以创造更高质量的粒子。尽管如此，即使是“坏”材料，和一块强有力的磁铁一起玩看起来也很有趣。

你会看到这种材料明显有磁性，只是不像普通铁磁流体那样有尖峰。[NileRed]有商业铁磁流体用于测试，并发现如果他将其稀释到足够的程度，它就像他自制的液体一样。所以虽然不是结论性的，看起来他把这一批稀释得太多了。

我们希望不久能看到他更好的作品。他在第一个补丁中使用的基础材料是自制的——他在另一个视频中谈到了这一点。然而，对于第二批，他将从商业氯化铁开始，也就是我们所知的 PCB 蚀刻剂。

尽管实验并不完全成功，我们还是很喜欢看到这个过程，观看自制的铁磁流体和商用铁磁流体的性能。他在视频评论中得到了很多建议和猜测，很有可能一个 Hackaday 读者也能提供帮助。

我们已经看到其他关于不成功的铁磁流体生产的报道。如果你需要一个实际的理由来制造或购买一些，那么[一个时钟](https://hackaday.com/2015/08/18/ferrofluid-clock-is-a-work-of-art/)怎么样？
