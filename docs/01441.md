# 一款被遗忘的早期个人电脑时代的经典之作

> 原文：<https://hackaday.com/2018/12/06/a-scratch-built-forgotten-classic-of-the-early-pc-age/>

对 Commodore 机器的所有逆向计算机的喜爱似乎都落在 C64 和 Amiga 上，对 VIC-20 还剩下一点点。这些机器确实很棒，但是为它们铺平道路的 Commodore 机器呢？20 世纪 70 年代末，学区急于将杂物室改造成新的“计算机实验室”,而第一批被吞并的那台机器呢？

宠物 2001 可能有点难以爱上它，因为它集显示器、录音机和可怕的 chiclet 键盘于一身，但有些人仍然对它情有独钟。[格伦]显然对这台机器有足够强烈的感觉，以至于[用目前生产的零件](http://www.glensstuff.com/pet2001/pet2001.htm)造了一只宠物，结果相当不错。当然，当试图从零开始再造一台 40 岁的机器时，必须做出一些让步。这款保护套并没有试图复制一体式设计，原始键盘幸运地被标准 PS/2 键盘所取代。除此之外，使用新的 65xx 生产芯片和 74HCT 系列逻辑芯片忠实地复制了该架构。[Glen]不得不穿过一些铁环才能到达那里，但正如下面的视频所示，成品机器玩了一场像样的太空入侵者游戏。

我们已经看到[一只宠物被 FPGA](https://hackaday.com/2011/10/27/recreating-the-commodore-pet-with-an-fpga/) 从坟墓里带回来，[一只 C64 在树莓派](https://hackaday.com/2018/03/28/emulating-a-complete-commodore-64/)上模拟，但是回到基础，从头开始建造这是对 PC 历史上一台重要机器的恰当致敬。
