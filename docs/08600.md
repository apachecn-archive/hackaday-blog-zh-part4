# 自制摩托车油门支架

> 原文:[https://hack aday . com/2020/12/08/home-crafting-a-motorcycle-throttle-hold/](https://hackaday.com/2020/12/08/home-crafting-a-motorcycle-throttle-hold/)

巡航控制是汽车上的常见特征，尽管在摩托车市场上不太常见。考虑到长途骑行中持续的油门应用会使手腕疼痛，许多骑行者渴望这种便利。作为一个作弊的解决方案，螺栓锁保持油门在一个设定的位置是可用的，虽然质量不同，一般来说，他们需要由油门手激活。[谢妮]想要一个解决方案，让右手完全自由，握住而不是锁住油门。

[谢妮]发明的装置本质上是一个制动器，它安装在油门手柄内并固定住它。这是通过一种机制实现的，这种机制将一对小制动蹄压入油门内部，当骑车人松手时，阻止它旋转回到空档。刹车由左车把上的控制器通过鲍登线缆激活，允许[谢妮]在高速公路上激活油门保持，并使用右手检查口袋或简单地休息。

这是一个整洁的建筑，[谢妮]做了大量的工作来解释各种设计选择和鲍登拉索驱动机制的复杂性。这绝不是一个一刀切的建设，但其他有进取心的机械师肯定可以复制其他摩托车的设计没有太多问题。
