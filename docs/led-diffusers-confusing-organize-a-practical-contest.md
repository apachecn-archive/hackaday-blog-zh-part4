# LED 扩散器令人困惑？组织一次实践竞赛

> 原文：<https://hackaday.com/2022/11/03/led-diffusers-confusing-organize-a-practical-contest/>

我们都想要一个漂亮而闪亮的 LED 灯条，它看起来不像是由单个 LED 组成的——一束均匀的光更有吸引力。有各种各样的扩散选项，但它们可能会令人困惑——有时你只是想知道，哪一个更好？如果有一件事可以轻松解决这个问题，那就是一个实际的测试，这就是[The Hook Up] [为我们](https://www.youtube.com/watch?v=DKOWeuV1FWE)设计的学习内容。

首先，他谈到了可用的 LED 灯条——每米有 30、60 和 144 个 LED 灯，后者比前者更容易扩散。从那里，你可以得到一些不同种类的扩散器盖和铝型材，然后[挂钩]将它们组合成对，从远处拍摄它们，给我们提供每个组合在不同占空比下如何工作的简明视觉效果，并不时进行亮度测量，以评估不同扩散层的损失。他提出了一个简单的规则——当选择漫射器时，LED 和漫射器之间的距离必须大于 LED 之间的距离，实验证实了这一点。最后，一个要点是，60LED/m 和 144LED/m 条带之间的差异不够显著，无法通过合适的扩散器进行补偿。

似乎你不必总是选择最昂贵的方案。当然，你也不必去找预制的扩散器——你可以 [3D 打印它们，](https://hackaday.com/2019/02/09/3d-printed-diffusers-make-more-natural-light/)或者，事实上，使用[几十个可用选项中的一个。](https://hackaday.com/2017/04/25/ask-hackaday-what-about-the-diffusers/)然而，如果您只是需要快速扩散几米长的 LED 光带，现在您知道哪些组合适合您了。
