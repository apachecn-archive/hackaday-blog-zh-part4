# 用乐高盒子里的电路块模拟城市街区

> 原文：<https://hackaday.com/2019/11/12/simulate-city-blocks-with-circuit-blocks-in-a-lego-box/>

你有没有环顾过你所在城市的布局，觉得自己可以做得更好？也许你一直想看看她如何依靠核能或风能运行，或者只是玩玩市政设施，看看你的选择如何影响市民。

[Robbe Nagel]为一个工业设计项目中的创意编程课程制作了这个物理数字模拟器。我们不知道所有的细节，但正如[Robbe]在休息后的视频中解释的那样，每个块的底部都有一个电阻器，每个格子都有一对准备与之配对的触点。一个 Arduino 安全地躺在下面的乐高碉堡里，读取不同的电阻值来决定哪块积木被放在哪里。

[Robbe]编写了一个程序，可以评估各种布局，并提供人口、整体健康、教育水平、污染等方面的统计数据。正如您在 break 之后看到的，一旦添加或删除了块，这些值就会改变。让这个模拟器如此酷的部分原因是它可以用于严肃的目的，或者它可以完全游戏化。

我们喜欢乐高已经不是什么秘密了，尤其是作为一种外壳材料。打扮一下或[打扮一下](https://hackaday.com/2019/10/26/lego-and-minecraft-team-up-for-custom-gaming-pc-case/)，只是不要在地板上留下任何碎片。
