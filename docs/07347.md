# 使用 PongMate CyberCannon Mark III 完善您的啤酒乒乓游戏

> 原文:[https://hack aday . com/2020/08/05/perfect-your-beer-pong-game-with-the-pong mate-cyber cannon-mark-iii/](https://hackaday.com/2020/08/05/perfect-your-beer-pong-game-with-the-pongmate-cybercannon-mark-iii/)

[格兰特]受到启发，帮助他的派对客人提高他们的啤酒乒乓游戏。他创造的是一个令人印象深刻的装置，肯定会让他在下一场比赛中势不可挡。

该设备使用陀螺仪和飞行时间传感器来计算乒乓球的最佳轨迹。使用两个气泡水平仪和一系列指示灯 led 将用户引导到正确的发射位置，当到达最佳位置时，这些指示灯将变为绿色。

发射装置使用一个伺服电机将球推进圆轮式机器，然后圆轮式机器将乒乓球推向目标。圆形车轮机器由两个 DC 马达驱动，马达的速度由与目标的距离决定。[Grant]将 DC 马达速度校准到离目标的距离，并发现一个非常可重复的关系，有利于立方根函数。你可以在他的 Instructable 页面上看到他的校准数据，以及一个很酷的演示视频，展示该设备如何根据与目标的距离自动调整电机速度。
