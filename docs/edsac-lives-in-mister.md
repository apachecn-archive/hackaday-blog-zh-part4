# 埃德萨克住在米斯特

> 原文：<https://hackaday.com/2020/05/15/edsac-lives-in-mister/>

关于哪一台是第一台现代计算机有很多争论。虽然还有争论的空间，但莫里斯·威尔克斯博士的研究成果 EDSAC 无疑是第一批。虽然我们以前见过模拟器，但 MiSTer 平台基于 FPGA 的模拟器还有很多优点。请看下面的视频。

我们今天认为理所当然的许多东西都是首先在 EDSAC 上开发出来的。例如，“惠勒跳跃”(以研究生大卫·惠勒的名字命名)就是子程序思想的起源。

快速浏览一下 Verilog，我们认为内部的设置是不同的。毕竟，真正的 EDSAC 是一次处理一位数据，因为这些位是从水银延迟线的末端掉下来的。然而，复制是足够忠实地运行威尔克斯的程序来计算查普曼的掠入射积分，并得到与真实机器相同的结果。

mercury 延迟线设置的一个好处是，各个位相对于彼此在某个时间出现。连接一个示波器，你就可以实时查看机器的内存。这也意味着一个聪明的程序员可以将这些行中的比特排列起来，在示波器上形成图像，这就是一个程序员在 1952 年让计算机玩井字游戏时所做的。

EDSAC 既重要又简单，我们已经看到了一些 [FPGA 实现](https://hackaday.com/2017/10/05/learn-fpga-programming-from-the-1940s/)。不像一些[早期的计算机](https://hackaday.com/2017/01/31/eniac-the-way-we-were/)，EDSAC 足够接近现代机器，值得[学习编程](https://hackaday.com/2018/09/28/retrotechtacular-heres-how-they-programmed-the-edsac-computer/)。如果你想知道更多关于先生的事，这就是你想要的。
