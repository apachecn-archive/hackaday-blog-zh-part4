# 这支热风枪是天才之作还是致命之作，我们无法决定

> 原文：<https://hackaday.com/2021/01/31/this-hot-air-gun-is-either-a-work-of-genius-or-lethal-we-cant-decide/>

长椅上的必需品之一是某种形式的热风枪。无论是用于返工 PCB 的精密工具，还是用于脱漆的宽冲程工具，我们都有一个。脱漆剂相当便宜，但没有[Porcas Pregos e Parafusos]自制热风枪便宜。这种有点令人毛骨悚然的设备由各种垃圾零件制成，并提供热空气，尽管我们怀疑烧伤操作员的可能性仍然很高。

它的核心是一个由电源供电的水锅炉元件，设计成可以放入杯子或类似的东西中，因为如果不以某种方式冷却，这种设备会烧毁，所以微波炉中有一个风扇使空气通过它。整个东西放在一个从圆形蛋糕罐上切下的铝锥内，一起放在一个木制底盘上，一个废弃电钻的手柄和电源开关为操作者提供了一些可以抓住的东西。

正如你从下面的视频中看到的，它是一个有效的热风枪，但我们猜测你很快就会学会避免接触金属锥。尽管如此，作为一个社区，我们对烙铁已经习惯了这一点，正如 RevSpace 的 t 恤所言:“ [*如果它闻起来像鸡肉，你就拿错了*](https://revspace.nl/Herdruk_chicken_shirt) ”。

奇怪的是，[这不是我们见过的第一个 DIY 热风枪](https://hackaday.com/tag/diy-heat-gun/)。
