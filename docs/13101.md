# 用热变色箔庆祝圆周率日的无限

> 原文:[https://hack aday . com/2022/03/21/用热变色箔庆祝无限圆周率日/](https://hackaday.com/2022/03/21/celebrating-the-infinity-of-pi-day-with-thermochromic-foil/)

你可能需要一些时间来理解 Hackaday 校友[Moritz Sivers]与我们分享的视频中发生的事情。这是[Moritz]为[今年的圆周率日](https://hackaday.com/tag/pi-day/)做出的贡献——一台在无限循环中(从技术上讲，不完全是)显示圆周率数字的机器，向我们展示了一个我们想不到的巧妙把戏。

这台机器的两个主要元件是一个环形的热变色箔片和一个热敏打印机。当数字被打印机的加热元件标记在箔片上时，它们可以被看到几秒钟，直到箔片从视图中消失，只是最终被循环回去并重新热压印。“圆周率数字计算”部分被卸载到谷歌的`pi.delivery`服务中，这是一个π即服务的端点，它将传输多达 50 万亿个圆周率的第一位数字，以防你需要它们——一个 ESP8266 忠实地获取这些数字，并将它们发送到热敏打印机。

这台机器可以打印数字，直到有什么东西坏了，或者数万亿个可用的数字用完，这是对圆周率无限本性的恰当致敬，这是一个我们除了从根本上尊重之外别无选择的数字。几天前，我们展示了一个类似的圆周率日致敬，尽管更加自给自足——一个 Arduino [在字符显示器上计算并打印圆周率](https://hackaday.com/2022/03/15/simple-arduino-build-lets-you-keep-an-eye-on-pi/)的数字！如果[阿基米德能数得更好一点](https://hackaday.com/2018/03/14/archimedes-would-have-known-better-if-he-could-count-to-a-million/)，我们可能已经庆祝这一天一千年了。
