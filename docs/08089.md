# 具有多重人格的 TTGO ESP32 模块

> 原文：<https://hackaday.com/2020/10/19/ttgo-esp32-module-with-multiple-personalities/>

Volos 项目教育家[Danko berto VI]有一个 TTGO ESP32 委员会在寻找一个项目，所以他为这样一个小屏幕实现了一个功能惊人的气象站。大概这对于他来说太无聊了，所以他决定写一个经典 Atari 游戏 Breakout 的版本。[Danko]更喜欢在 ESP32 项目中使用 Arduino IDE，并且已经让[的突破软件](https://drive.google.com/file/d/1_lYi8ClOMExkd-qtNqBLwg-l2Y3UiV7F/view)成为 Arduino 草图。我们希望气象站的草图也能很快发布。TTGO 是一款小型 ESP32 板，采用 ST7789V 1.14 in (29 mm) TFT 彩色显示屏，可从您最喜爱的深圳市场供应商处购买。这个平台非常适合各种利基应用。我们很想知道您在项目中是如何使用或计划使用这些模块的。

我们在去年夏天写了一个这样的项目，一个类似的 TTGO 模块被用来显示阿波罗 11 号任务 50 年的广播延迟记录。[Danko]对 Hackaday 并不陌生——他已经完成了几个基于 Arduino 的计算器项目。也许最引人注目的是去年的[电路雕塑二进制数](https://hackaday.com/2019/12/25/circuit-sculpture-teaches-binary-plays-pong/)计算器，这是另一个演变成电脑游戏(Pong)的项目。
