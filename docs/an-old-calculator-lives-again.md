# 一个旧计算器又活了

> 原文：<https://hackaday.com/2020/08/24/an-old-calculator-lives-again/>

曾几何时，任何学电子学的学生都会在腰带上挂一把计算尺。到了 20 世纪 70 年代，计算尺变成了一个昂贵的电子计算器。今天你可以在一元店买到计算器。【JohnAudioTech】掏出一个[旧 Radio Shack 真空荧光显示](https://www.youtube.com/watch?v=A7bfVFG3LIk) (VFD)计算器，发现没用。显然，这意味着是时候打开它了。

看到这些旧设备中的一个再次被打开是很有趣的。大通孔 IC 消费电子！对该设备的故障排除最终是反气候的，因为电池盒的一根断线解释了整个事情。

作为一个拆机，虽然，这是一个有趣的视频。不仅所有器件都是通孔的，而且 PCB 显然是手工布局，蜿蜒的走线像某种艺术品一样流过电路板。

[John]甚至对计算器本身有需求，如果你碰巧有一个的话。显然，计算器 IC 提供了一个 pi 常量键，但没有用于该功能的键。同时按平方根和 4 个键会欺骗芯片提供圆周率。

另一个有趣的项目:[约翰]修改了一个便宜的相机，并删除了红外过滤器。利用这一点，他能够对显示器的阴极线进行成像，这通常很难用肉眼看到。

现在不太需要计算器了，因为你的手机可以轻松地进行数学运算，从简单的计算器应用程序到符号数学程序。但是，就像计算尺一样，我们对这些掌上电脑情有独钟。

如果你不想要老式的，一个 Arduino 与这些旧机器相比有足够的动力。如果你碰巧想要一个[现代计算器](https://hackaday.com/2020/03/02/the-last-scientific-calculator/)，他们现在也能做很多。
