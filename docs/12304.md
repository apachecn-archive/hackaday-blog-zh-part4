# 珀尔帖雪花玻璃球的特点是雪人穿着真正的霜冻

> 原文:[https://hack aday . com/2021/12/24/peltier-snow-globe-features-雪人-谁-dresses-his-in-real-frost/](https://hackaday.com/2021/12/24/peltier-snow-globe-features-snowman-who-dresses-himself-in-real-frost/)

我们怀疑很少有人会认为雪花玻璃球包含真正的雪，但现在我们已经看到了自己制造雪的雪花玻璃球，我们不得不承认充满水的节日装饰支柱看起来有点令人失望。

像肖恩·哈金斯(Sean Hodgins)多年来发明的许多圣诞装饰品一样，这个自结霜雪人设计巧妙，制作可爱。工作端是一块铝，向下弯成经典的雪人形状；通过用 CA 胶将一些滚珠轴承粘在一起，无车床也可以做同样的事情。用 3D 打印的配件装饰，雕塑坐落在珀尔帖冷却器的基座上，堆叠在一个大 CPU 冷却器的顶部。在这个还没穿好衣服的雪人的侧面是一对大功率电阻器，它们作为加热元件向地球充入水蒸气。[肖恩]选择的液体是异丙醇，它似乎非常有效，因为小雕像很快就被霜覆盖。

但是等等，还有更多——正如[Sean]指出的，这个仪器有 90%的可能成为云室。也许我们会在假期后看到一个不那么喜庆的版本。在那之前，享受[他的可以打印其他装饰品的装饰品](https://hackaday.com/2018/12/24/have-yourself-a-recursive-little-christmas-ornament-that-prints-ornaments/)，[他的闪光灯 PCB 衣架](https://hackaday.com/2016/12/24/these-ornaments-measure-christmas-cheer/)，或者[他播放节日广告的小电视](https://hackaday.com/2017/12/11/make-christmas-commercial-again-with-this-tiny-tv-ornament/)。
