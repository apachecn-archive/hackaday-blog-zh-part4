# 你喜欢听数控切割的唱片吗？

> 原文:[https://hackaday . com/2022/06/17/wooden-you-like-to-head-a-CNC-cut-photo-record/](https://hackaday.com/2022/06/17/wooden-you-like-to-hear-a-cnc-cut-phonograph-record/)

不管你怎么说[托马斯·爱迪生]，但很难否认他自称的个人最喜欢的发明的天才之处:留声机。在可延展的介质上以物理模式捕捉声音是真正的革命，作为记录声音的主要介质超过一个世纪并建立了几个主要行业的基本技术今天仍然活跃。

在如此多的技术历史背后，当制作自己的留声机唱片的冲动袭来时，有抱负的发明家该怎么办？简单— [用数控切割机从木头上切割下来](https://www.youtube.com/watch?v=X1N9oMS3lRM)。至少这是[alnwlsn]在用他的路由器切割 PCB 时受到“百分之一的灵感”打击后的反应。他认为铜片上的轨迹可能和唱片上的凹槽一样精细，于是他想出了一些数学方法来描述精细的螺旋凹槽，并覆盖声音文件中的数据，然后将整个过程转换成 g 代码。

为了寻找合适的介质，他转向了用于运输 PCB 模板的 MDF 废板，经过大约三个小时的研磨，获得了看起来相当毛的 78 RPM 记录。令人惊讶的是，这张唱片在发条留声机上播放得相当好。对于沉重的木制唱片来说，弹簧马达有点弱，需要手动辅助，但你或多或少可以清楚地听到 40 秒的录音。更令人惊讶的是，当钢针换成一大块牙签时，录音听起来要好得多。你可以在下面的视频中查看整个过程，你可以在 GitHub 的上找到 g 代码生成脚本。

所有这些关于用摇摆不定的线条再现音乐的讨论让你困惑了吗？哦，你好，自以为是的年轻人——看看[【Jenny】的 MP3 一代入门](https://hackaday.com/2017/03/03/record-players-explained-for-the-streaming-generation/)了解你需要的背景知识。
