# 焊料忍者涉足 USB 电源奥秘

> 原文:[https://hack aday . com/2019/09/09/solder-ninja-dubbles-in-USB-power-arcana/](https://hackaday.com/2019/09/09/solder-ninja-dabbles-in-usb-power-arcana/)

USB 在 20 世纪 90 年代首次出现，旨在简化外围设备与 PC 的连接，并消除各种传统接口的激增。20 多年后，它不仅实现了最初的目标，而且成为各种个人电子设备充电和供电的事实标准。如果你在 1995 年问某人是否能制造一个 USB 驱动的烙铁，他们会礼貌地请你离开 USB 实现者论坛。但是时代变了，[而焊料忍者也不过如此！](https://hackaday.io/project/167310-solder-ninja)

由于最大功率为 40 W，焊接忍者需要精心设计以确保实用性。它支持多种 USB 电源标准，包括 USB-BC 1.2、USB 快速充电和 USB 电源传输。这使得它能够汲取加热元件所需的大量电流。为了便于在野外使用各种充电器，它会显示当前协商的电压和最大电流消耗。这使用户能够了解设备的不同性能，这取决于它所插入的充电器。

鉴于众多不同的 USB 电源标准，我们想象[Nicolas]有圣人般的耐心来完善这样一个项目。我们以前也见过类似的版本。休息后的视频。
