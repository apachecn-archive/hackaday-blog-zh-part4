# 高速摄像机加割草机等于破坏性的乐趣

> 原文：<https://hackaday.com/2018/12/28/high-speed-camera-plus-lawnmower-equals-destructive-fun/>

我讨厌无端破坏视频。你知道，那些问“如果我们把一个炽热的钚球扔到一袋奇多上，会发生什么？”有大量的烟雾，火焰和一大堆广告收入背后的白痴。

不过，这个破坏视频有点不同。[特斯拉 500]想把他的[高速摄像机安装到旋转叶片](https://youtu.be/RtnQLc_NI0Q)上，但是*没有*破坏摄像机。在这个视频中，他记录了建造一个钻机的过程，这个钻机以每分钟几千转的速度旋转一台[3000 美元的相机](https://www.krontech.ca/)。这一切都是关于平衡，关于建立一个钻机，在高速下适当地平衡相机和刀片的重量。

为了把它做好，进行了几次尝试，而[特斯拉 500]展示了他如何测试和改进每个版本，包括改变重量以适应相机本身的不同密度，相机的一侧有沉重的电池。然后他把东西放到刀片上，看看切片后是什么样子。很自然。
