# 假人空气动力学

> 原文:[https://hackaday.com/2020/07/11/aerodynamics-for-dummies/](https://hackaday.com/2020/07/11/aerodynamics-for-dummies/)

我们不知道空气动力学是否真的是一个适合假人的主题，但如果你对飞行或建造无人机和模型飞机感兴趣，你可以在下面看到的[Jenny Ma]的新视频将帮助你简单介绍一些[的关键概念](https://www.youtube.com/watch?v=Z1FAAJ4hUaQ)。(嵌在下面。)

该节目从升力、推力和阻力开始，但转移到失速和棺材角等主题。如果你有一张飞行员机票，你可能不会学到很多新东西，但对于我们其他人来说，有一些有趣的金块，你可能不知道或可能已经从高中的物理课上忘记了。

事实上，我们中的一些人已经足够大，以至于我们在学校里学习了一个关于翅膀如何产生升力的错误理论，而[珍妮]涵盖了这一点，并解释了我们现在如何知道那个理论的因果关系是落后的。

说到不好的理论，视频里涵盖了亚音速飞行，虽然有一点超音速飞行的讨论。请记住，在 1947 年 X1 突破音障之前，许多人认为这是不可能的。

这个视频不会让你为设计下一架大型喷气式飞机或 X1 做好准备，但它可能会在你下次想要建造一架[新颖的四轴飞行器](https://hackaday.com/2019/10/17/coanda-effect-makes-a-better-hovercraft-than-a-quadcopter/)甚至是[纸飞机](https://hackaday.com/2018/10/25/paper-airplane-database-has-the-wright-stuff/)时帮到你。
