# 这台业余无线电在任何频率下都不安全

> 原文：<https://hackaday.com/2021/12/03/this-ham-radio-is-unsafe-at-any-frequency/>

当我们还是孩子的时候，我们骑自行车没有护垫和头盔。我们喝含糖饮料。我们玩化学设备和电动工具。我们还建造了直接使用交流电的东西。[Mike]记得并制作了一个，大概更多的是为了讨论可能让你震惊的事情的安全预防措施，而不是诱使你重复它。他称之为[复古 QRP 鳏夫制造者](https://youtu.be/7n-52exDO3w)，如果这是某种暗示的话。(下面还嵌入了这个不安全发射器的视频。)

这个设计不时出现在旧的电子杂志上。这款收音机建在一块开放式电路板上，没有地线，不需要复杂的电源。这也不仅限于发射机。一些电视和收音机有一个“热底盘”这就是为什么我们被教导要先用手背去摸一个不知名的底盘。电击会收缩你的肌肉，将你的手臂拉开，而不是让你抓住电活性部分。

为了安全起见，[迈克]使用了隔离变压器来避免灾难。一个大电阻降低了电路中给电子管供电的电压。有一个霓虹灯球来指示你是否把插头插反了，从而使事情变得危险。

我们喜欢[迈克的]优秀的代码使用旧的 J-38 风格的关键。不是每个人都用拨片或键盘。如今，小型发射机不需要[高电压。此外，今天的 50 美元可能还不如当时的 10 美元值钱。如果你渴望老式齿轮，试试](https://hackaday.com/2017/09/13/a-fully-featured-fifty-dollar-qrp-radio/)[老晶体管](https://hackaday.com/2017/03/28/vintage-transistor-powers-qrp-transmitter/)。
