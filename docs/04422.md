# 这场飓风使用了一种新颖的技术

> 原文:[https://hack aday . com/2019/10/05/this-hurricane-uses-a-novel-technique/](https://hackaday.com/2019/10/05/this-hurricane-uses-a-novel-technique/)

你可能听说过勇敢的飞行员，也就是所谓的“少数”，他们驾驶超级海上喷火战斗机升空，在不列颠之战中拯救了世界。这是一个包含了很多真相的故事，但正如经常发生的那样，它掩盖了一个更加复杂的故事。早在 1940 年夏天，这些飞行员确实在英格兰南部的机场上空盘旋，但他们中更多的人是在霍克飓风的控制下投入战斗，而不是更具魅力的马棚。

在公众眼中，飓风可能会因喷火而黯然失色，但对[Marius Taciuc]来说不是，他[制作了一个全功能的钢筋混凝土模型。通常这不值得我们注意，但在这种情况下，他采用了一种相当迷人的建造技术。他通过 3D 打印飞机的框架并在其上覆盖热收缩薄膜，重建了原始飞机的掺杂织物皮肤，使这确实是一种非常罕见的鸟。](https://hackaday.io/project/167876-full-rc-hawker-hurricane)

下面的视频向我们介绍了一些步骤，包括根据 2D 飞机图片的描摹，在 CAD 包中开发框架，安装胶片，以及最终尝试飞行，但不幸的是，由于选择了不合适的机轮而失败。但短暂的飞行和坠毁确实证明了这种建造方法是耐用的，这导致了我们对它的兴趣。虽然它显然是一架功能性飞机，但还有其他应用可以受益于这种轻质而坚固的材料组合。

[马里乌斯]实际上用类似的技术创造了一个更上镜的喷火式战斗机的模型，尽管据我们所知，那架还没有起飞。顺便说一句，这些页面已经被非解放军种类的飓风[增光。](https://hackaday.com/2019/05/07/the-queen-of-the-hurricanes/)
