# 这只 Arduino 喂狗

> 原文:[https://hackaday.com/2019/03/15/this-arduino-feeds-the-dog/](https://hackaday.com/2019/03/15/this-arduino-feeds-the-dog/)

养狗的乐趣之一是喂养它。你能让另一个生命如此快乐的机会有多少？然而，当你最好的朋友饿的时候，你有时不能在那里。[El Taller De TD]使用 Arduino 和步进电机建造了一个[自动喂狗器。视频和链接都是西班牙语的，但如果你的西班牙语不熟练，YouTube 的字幕自动翻译还不错，谷歌翻译可以帮你搞定网站。](https://eltallerdetd.wordpress.com/2019/03/09/alimentador-automatico-bluetooth-para-mascotas-con-arduino/)

电子设备相当简单:一个 Arduino、一个蓝牙模块和一个步进电机驱动器。机械方面，你只需要电机和一些 PVC 管。有一个小的手机应用程序使用应用程序 Inventor 来驱动蓝牙。

这将是一个非常简单的第一个项目，当然，对任何种类的动物都有用。对于狗的使用，我们可能已经硬化了外部电线和电路板。此外，你可以在软件中做很多事情，例如，你可以每 8 小时喂一次。看起来你可以添加一个传感器来告诉你什么时候没有食物，或者如果食物因为某种原因没有被喂食。

我们之前已经看过如何将 [App Inventor 与蓝牙](https://hackaday.com/2013/02/15/beginners-androidarduino-example-shows-the-power-of-app-inventor/)配合使用，这非常简单。为了有更多的交流选择，我们可能会倾向于选择 [Blynk](https://hackaday.com/2016/03/10/app-control-with-ease-using-blynk/) ，但无论哪种方式都很简单。
