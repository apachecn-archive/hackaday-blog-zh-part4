# 硬币清分机美观大方

> 原文:[https://hackaday . com/2022/01/16/硬币清分机-是优雅而美丽的/](https://hackaday.com/2022/01/16/coin-sorter-is-elegant-and-beautiful/)

数钱是教孩子们数学和金钱的好方法，但对于我们这些已经过了一年级的人来说，这变得令人厌倦。因此，正如丹尼尔·塔尔塔利亚所展示的，机器应该承担这项工作。

振动马达被用来摇动装满硬币的漏斗，让它们以稳定的速度通过进料口落入机器。然后，它们通过一个基于尺寸的分类器，该分类器根据硬币的物理尺寸将硬币送入不同的通道。硬币通过连接到 Arduino 的红外传感器进行计数，然后通过一个非常可爱的迷宫进入机器底部的分类箱。

这是一个整洁的建筑，如果你经常发现自己需要数零钱，这是一个很好的东西。我们以前没有见过太多的投币柜台，但我们见过[一家自助洗衣店用一些黑客技巧进行了大修](https://hackaday.com/2013/11/25/running-a-laundromat-with-an-arduino/)。休息后的视频。
