# 遥控汽车的地面效应空气动力学

> 原文：<https://hackaday.com/2021/12/21/ground-effect-aerodynamics-on-an-rc-car/>

地面效应空气动力学将在 2022 赛季大规模回归一级方程式，希望洗去最近争议性赛季决胜局后留在范嘴里的恶趣味。[下班后的工程]以前曾在遥控汽车上试验过 F1 空气动力学，并决定是时候自己尝试和实现一个合适的地面效应设计了。

地面效应空气动力学的目的是在汽车底部和地面之间产生一个气流收缩。这种收缩加速了赛车下方的气流，根据伯努利原理，会产生相应的压降，将赛车吸到赛道上。粘度也起了作用；从汽车的角度来看，汽车下方的道路正在以一定的速度向后移动，由于地面本身的边界层，拉动液体。这进一步有助于增加效果的强度。

RC 赛车上安装了真空成型的底架和侧裙，以产生地面效应下压力。用吹叶机进行的快速测试表明系统工作正常，侧裙是一个关键部件。

在安装了地面效应的测试中，横向加速度显著提高了约 20%，尽管[下班后工程]承认，没有风洞，结果不是最科学的。然而，由于底架相对较轻，我们怀疑空气动力学元素可能会提供大量的好处，而不会对加速或操控产生太大的负面影响。
