# 模型车显示门半开

> 原文：<https://hackaday.com/2019/03/09/model-car-indicates-door-is-ajar/>

现代汽车的科技含量确实令人震惊。除了最便宜的车型，加热座椅、无钥匙进入和一系列有用的摄像头在所有车型上都变得越来越普遍。[mathisox]开一辆稍微旧一点的大众面包车，已经改装成露营车。不幸的是，它缺乏一个适当的门半开显示。不过，没关系，这个问题有一个迷人的解决方案。

mathisox 没有坚持枯燥的指示灯和低分辨率液晶显示器的汽车标准，而是采取了一种更类似的方法。一辆与他的货车相匹配的小型模型汽车被找到，并很快为这个项目被掏空。然后它安装了伺服系统来打开和关闭门和后舱。伺服系统由 Arduino Nano 控制，Arduino Nano 读取车辆中的车门开关，并启动模型上的相应部件。

由于模型车突出地贴在仪表板上，它可以作为车门当前状态的清晰视觉指示器。这远没有那些老克莱斯勒公司一再坚持门就是一个罐子那么碍眼。
