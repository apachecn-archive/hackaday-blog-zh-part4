# 时钟监控深空网络，守护丢失的火星探测器

> 原文:[https://hack aday . com/2018/10/07/clock-monitors-deep-space-network-keep-vigil-over-lost-mars-rover/](https://hackaday.com/2018/10/07/clock-monitors-deep-space-network-keeps-vigil-over-lost-mars-rover/)

我们已经很久很久没有听到机遇号的消息了，这是一辆非凡的火星探测器，它打破了人们对耐力和生产率的所有预期，但自从一场全球性的沙尘暴遮住了太阳，使它缺乏电力以来，它一直保持沉默。现在，它栖息在火星上一个陨石坑的边缘，等待足够的阳光给电池充电，这样它就可以打电话回家了。我们能做的就是坐着等。

为了打发时间，直到*机会*再次蠢蠢欲动，【G4lile0】建造了[这个深空网络时钟](https://github.com/G4lile0/DeepSpaceNetworkClock)。围绕着一个 ESP32 和一个 TFT 显示屏，这个时钟监控着[深空网络(DSN)](http://hackaday.com/2017/07/21/serious-dx-the-deep-space-network/) 网站，看看任务控制中心是否正在使用它所拥有的任何一个巨大的天线来监听来自被困火星车的信号。如果 DSN 在监听，它会显示一个特殊的动画，鼓励漫游者打电话回家；否则，它会显示网络正在与众多遥远的探测器中的哪一个进行通信，同时还会显示火星任务照片的幻灯片，以保持乐观。当*机遇*入住的那一天终于到来时，警报会响起，这样【G4lile0】就可以开香槟和我们其他人一起庆祝了。

我们意识到,*号机遇号*在这场考验中幸存下来的几率正在减少。这是一场艰苦的战斗；毕竟，机器在陷入黑暗时是其最初 90 天设计寿命的 55 倍，所以这是一场艰苦的战斗。话说回来，它以前战胜过困难，所以还是有希望的。
