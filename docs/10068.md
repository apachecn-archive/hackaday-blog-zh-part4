# 精细结构常数在一眨眼之间

> 原文:[https://hack aday . com/2021/05/07/the-fine-structure-constant-in-a-blink/](https://hackaday.com/2021/05/07/the-fine-structure-constant-in-a-blink/)

电子学实际上是物理学的一个应用分支，所以如果你对电子学很认真，你可能也懂一点物理，这并不奇怪。如果你曾经听说过术语[“精细结构常数”](https://www.youtube.com/watch?v=9pebuHH4epg)并且不完全确定它的意思，[Parth G]想在一分钟内给你解释一下。他的视频解释出现在下面。

你可能知道常数，常以α表示，约为 1/137，但这是什么意思呢？答案与电子的轨道有关。你可能还记得在学校时，电子在原子核周围的壳层中运行。也就是说，一个原子可能在最内层有一些电子，在外层有更多的电子。

事实证明，如果你看得足够近，这些壳层中的每一个都被进一步分成子壳层，每个子壳层都有一个离散的能级。这些子壳层是常数所指的精细结构。

每个子壳与同一个主壳中的相邻子壳间隔开一点。相隔多远？距离取决于原子核中质子数与精细结构常数乘积的平方，你猜对了。

当然，也许你需要知道更多关于精细结构常数的知识，这可能就是为什么这个视频被标为第一部分。但是如果你喜欢小块的物理知识，[Parth]频道有很多，包括如何解薛定谔方程的[和关于向量微积分算子](https://www.youtube.com/watch?v=sPZWtZ8vt1w)[的直觉](https://www.youtube.com/watch?v=hI4yTE8WT88)。作为一个典型的物理学家，[Parth]甚至告诉我们，[欧姆定律并不像我们想象的那么简单，尽管在现实生活中，它几乎总是如此。](https://www.youtube.com/watch?v=Zao9JV1BLg8)
