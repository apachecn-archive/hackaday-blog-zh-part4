# 本周失败:铜钎焊熟铁

> 原文：<https://hackaday.com/2022/09/04/fail-of-the-week-bronze-brazed-wrought-iron/>

[威尔·斯特尔特]，一个在蒙大拿州工作的年轻有为的铁匠，有了一个极好的主意，用一种独特的复合材料来完成一把刀的制作。这件作品是多个铁匠合作完成的，作为这个团队中的年轻人，[威尔]真的很想全力以赴，最终制作出一件他多年来一直在考虑的材料，给长辈们留下深刻印象。他知道如果你试图在太低的温度下锻造熟铁，它会产生裂纹和裂缝。你能故意这样做，然后用青铜填充这些裂缝吗？这将是非常惊人的材料，明亮的青铜纹理贯穿黑暗的铁。他必须尝试。

不幸的是，我们年轻的实验者遇到了一些没有足够时间克服的问题。首先，让青铜流动并填充铁的空隙是一个挑战，特别是当用火炬加热时。将整个实验扔进熔炉导致青铜从外壳中渗漏。最有希望的尝试是一个加强的盒子，放在烤箱里大约 20 分钟，温度高到足以液化青铜。它看起来很棒，直到他切开它，发现对于一个可加工的坯料来说有太多的气穴。

尝试是失败的，但我们很高兴(威尔)继续前进，把视频放在那里。如果你知道如何做到这一点，去他的频道发表评论，我们都期待着第二部分，他最终钉技术。
