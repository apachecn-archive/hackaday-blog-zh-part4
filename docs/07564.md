# 同步焊接站

> 原文：<https://hackaday.com/2020/08/27/simultaneous-soldering-station/>

烙铁是一种个人工具。有些人需要凉的，有些人喜欢热的。把它做好需要一些实践和经验，但是当你找到一个有用的提示和临时方法时，你就要坚持下去。[Riccardo Pittini]用他的开源[焊接站，焊接 RT1](https://hackaday.io/project/171848-soldering-rt1) ，在中间的某个地方着陆。当你启动它时，它会问你想要什么温度，然后它就会升温。很简单。当你准备好开始幻想的时候，你可以插上第二个熨斗，耗尽汽车电池，记录预设的温度，限制你的工作周期，并打开一个串行连接。

控制器在 32u4 处理器上有一个 Arduino 引导装载程序，所以它看起来像你的计算机的微型计算机。该系统与 RT 系列韦勒吸头配合使用，该系列吸头产品阵容全面。[Riccardo]还再造了 SMD 镊子，你可以在他的 [Tindie store](https://www.tindie.com/products/pittinihub/soldering-station-rt1/?utm_source=hackadayio&utm_medium=link&utm_campaign=project_buynow) 找到一切。

焊接有一种从新手到大师的表达意见的方式。如果我们能采访年轻时的自己，我们会为那些自以为无所不知的人提供一些宝贵的智慧。如果人体工程学是你优先考虑的，看看 [TS100 3D 打印的箱子](https://hackaday.com/2019/10/28/3d-printing-new-cases-for-the-ts100-soldering-iron/)，在[我们看来](https://hackaday.com/2017/07/24/review-ts100-soldering-iron/)这是一个很好的熨斗。
