# 迷路的 PLA 铸造的迷宫螺栓玩具

> 原文：<https://hackaday.com/2022/05/05/maze-bolt-toy-by-lost-pla-casting/>

迷宫螺栓，这种螺栓的轴上有一个迷宫，螺母上有一个销穿过，非常有趣。[这里有一个由[Robinson Foundry]制作的非常漂亮的金属版本](https://www.youtube.com/watch?v=GJOOLH9ZP2I)，它是由更多制造商应该知道的工艺制作的——消失模铸造。

他的基本方法是在 PLA 中 3D 打印，然后使用或多或少与失蜡铸造相同的工艺。

他用 PLA 3D 打印了零件，以及伴随铸造的浇口和冒口，然后将零件浸入浆料中十(10)次。他在窑中加热到 500 华氏度(260 摄氏度)，PLA 熔化并耗尽或烧掉。随着 PLA 的消失，在修复了一些裂缝后，他将温度提高到 1500 F (815 C ),并将浆料玻璃化成陶瓷。他现在有了模具。

螺母是青铜的。螺栓是铝制的。他把金属和模具一起倒入热的沙子中，这样金属就可以流入所有的小细节。该项目的其余部分只是清理，但我们了解到，你可以通过改变气压来改变玻璃珠喷砂产生的光洁度。

一个有用的技术和一个有趣的玩具的伟大演示。

我们介绍了一个伟大的技术，使用微波进行[消失模铸造。](https://hackaday.com/2014/08/20/lost-pla-casting-with-a-little-help-from-your-microwave/)
