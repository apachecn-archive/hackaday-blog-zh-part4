# 用手旋转一个乐高轮子能有多快？

> 原文：<https://hackaday.com/2022/03/10/how-fast-can-you-spin-a-lego-wheel-by-hand/>

这不是你每天问自己的问题，但这是[砖块实验频道]着手回答的一个问题:你用手旋转一个乐高轮子能有多快？以他们典型的方式，他们开始建造一个越来越复杂的装置来优化最大转速这一非常特殊的情况。

建造从一个安装在轴上的乐高轮子开始，由两根乐高技术梁支撑。轴的一部分上还贴有白色闪光标记，用于用光电转速表测量转速。第一次尝试达到 1700 转/分。升级来得又快又厚，通过三级复合齿轮系，手动曲柄轮达到 6300 转/分。增加另一个阶段会导致塑料工艺轴在手的扭矩输入下开始扭曲的问题。

采用一种新的方法拉一根线来转动轮子，第一次尝试的转速为 8300 转/分。传动装置将转速进一步提高到 12，900 转，但增加更多的齿轮又会导致车轴在压力下弯曲的问题。不过，双向拉绳设计有所帮助，系统转速可达 13，100 转/分。

到目前为止，一些部件已经损坏，但用润滑良好的新部件进行重建提供了巨大的推动力。现在光滑的轴运行更顺畅，随着机械装置自行拆卸，车轮达到了 19，300 RPM 的转速。

与[Brick Experiment Channel]，[的一些早期作品相比，这是一个不太复杂的追求，比如我们之前看到的令人印象深刻的爬杆设计](https://hackaday.com/2021/07/23/lego-pole-climbers-are-great-study-in-what-it-takes-to-go-vertically-upwards/)。然而，这个视频展示了迭代设计的力量以及从这个过程中可能获得的收益。
