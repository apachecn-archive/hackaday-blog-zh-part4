# 借助 cnc 打造谷歌风格的标志

> 原文：<https://hackaday.com/2019/07/02/building-a-googie-style-sign-with-the-help-of-cnc/>

谷歌风格是美国战后时期的主要建筑趋势。回到这种风格仍然很流行，[和【Wesley Treat】得到了为当地一家拖车汽车旅馆设计这种风格标志的工作(Youtube 链接，嵌入下面)](https://www.youtube.com/watch?v=OK8Eyxx9xz4)。

数控工具使任何工作变得简单，这一个也不例外。标志的平滑曲线是由几块聚氯乙烯板雕刻而成，堆叠起来形成标志的主体。然后用砂纸打磨，涂上油灰，再涂上一点油漆。为了节省时间，像这样的步骤可能会被跳过，特别是考虑到一旦标志被安装，很少有人会看到那些部分。然而，[卫斯理]为他的作品感到自豪，最后一件作品因此更好。同样重要的是，作品要打动客户，而不仅仅是公众。

标志的正面也是用聚氯乙烯板制作的，并涂上一层油漆，用刷子刷出人造木材的表面。然后将乙烯基应用于文本和图形元素，以创造彩色背光效果。该标志由现成的 LED 灯条照亮，整个组件是全天候密封的，以保护它免受自然环境的影响。

最终产品是一件美丽的作品，回归了经典的谷歌美学，是对[卫斯理]技巧的证明。这是一个很好的例子，说明用正确的工具和对细节的适当关注来创造伟大的作品是多么容易。它还展示了 led 对于标牌的重要性，无论你是在[海滩](https://hackaday.com/2015/06/06/led-sign-brightens-up-the-beach-after-dark/)还是在[实验室](https://hackaday.com/2014/10/29/bluetooth-enabled-danger-sign-for-lab/)。休息后的视频。
