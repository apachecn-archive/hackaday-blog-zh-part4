# 在任天堂 Switch 上用 Ubuntu 构建 Dolphin 模拟器

> 原文:[https://hackaday . com/2021/03/28/building-the-dolphin-emulator-in-Ubuntu-on-a-a-Nintendo-switch/](https://hackaday.com/2021/03/28/building-the-dolphin-emulator-in-ubuntu-on-a-nintendo-switch/)

[LOE TECH]已经养成了在他的任天堂 Switch 上尝试各种仿真方法并记录结果的习惯。在这些测试中，他看到的一些最佳性能利用了 Ubuntu Linux 中运行的 Dolphin 模拟器，他制作了一个教程视频[，记录了如何构建项目，以及如何进行一些性能调整以充分利用 mod](https://www.youtube.com/watch?v=hYllL0I_4j8) 。

我们喜欢看到 Linux 运行在基本上任何有处理器的东西上。这是一个经典的黑客攻击。任天堂传统上一直保持其游戏机相当锁定，虽然，即使面对一些真正令人印象深刻的努力；因此，看到开源操作系统在游戏机上相对流畅地运行总是一件乐事。这个 Ubuntu 安装基于 NVIDIA 的 Linux for Tegra (L4T)软件包，它在相同的硬件上比[的 Android 安装提供了一些性能增益。正如我们已经看到的那些 Android 黑客，然而，这个软件模块也利用了 Switchroot 项目，当然，它只适用于特定的，未打补丁的硬件。但是，如果你已经赢得了序列号彩票，并且你愿意拿你心爱的游戏机冒险，[LOE TECH]也有](https://hackaday.com/2021/02/10/android-10-ported-to-the-nintendo-switch/)[一个视频，详细介绍了他用来启动和运行 Ubuntu 的过程](https://www.youtube.com/watch?v=SlWHtWq9d0o)。

看看下面的视频，看看 Gamecube 游戏测试的大杂烩。有些看起来运行得很好，而其他的，嗯…就没那么好了。但是我们真的很欣赏他没有删掉那些结巴和滞后的游戏。这样，我们可以更真实、更全面地了解交换机上的非官方仿真性能。另外，看赛车游戏以慢动作进行几乎是一种乐趣；差不多，也就是说，如果我们不能理解比赛有多么令人沮丧的话。
