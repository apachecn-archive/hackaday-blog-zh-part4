# 玻璃:经典，但神秘

> 原文：<https://hackaday.com/2022/06/11/glass-classic-but-mysterious/>

在人类历史的大部分时间里，人们用他们能找到的东西制造东西。有些石头可以做箭头。其他人制造火花。树木可以变成木材。但是真正的力量是当你可以把同样的材料做成具有非常不同特性的东西。例如，植物纤维变成布料，或者岩石释放出里面的金属。玻璃是最古老的工程材料之一。你可能会认为玻璃的历史悠久(至少可以追溯到 4500 年前)，到现在为止，我们已经了解了关于它的所有知识。根据乔恩·卡特赖特在《物理世界》上发表的一篇有趣的文章，我们没有。一点也不。

根据[乔恩]的说法，至少有五个“玻璃之谜”我们仍然不明白。当然，融化沙子、苏打和石灰很容易——这是我们之前谈论过的事情——但是，事实上，许多材料在从液体快速冷却到固体时会变成玻璃。问题是，我们真的不明白为什么会这样。

几乎任何东西都可以变成玻璃，这一点也非常有趣。你知道吗，音乐节上的蚁群和人群的行为就像玻璃一样？模拟退火——一种处理复杂优化问题的计算机算法——也利用了我们在玻璃中看到的同类行为。

把金属变成玻璃很难。你必须非常迅速地冷却它，有时大约每秒几十亿度。但是回报是巨大的。由于没有晶界，金属玻璃不容易磨损，也不容易吸收动能。比如一个滚珠轴承打在钢板上会反弹几下，但是钢板会很快吸收轴承里的能量。然而，金属玻璃板从轴承吸收的能量要少得多。想看看吗？看下面的视频。

我们已经讨论过玻璃是如何与其他古老的工程材料一起制造的。如果你有一台激光切割机，你甚至可以在不使用[疯狂温度](https://hackaday.com/2015/08/22/mits-glass-3d-printer/)的情况下 [3D 打印玻璃](https://hackaday.com/2022/05/28/3d-print-glass-with-a-laser-cutter/)(那个帖子上的链接已经死了，但视频还在)。
