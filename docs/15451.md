# 1350 万核心计算机

> 原文：<https://hackaday.com/2022/11/23/the-13-5-million-core-computer/>

如今，拥有双核或四核 CPU 并不稀奇，拥有 12 核甚至 16 核 CPU 也并不罕见。来自脑波强化系统的仙女座号是一台拥有 1350 万个核心的超级计算机。该公司声称，这是有史以来最大的人工智能超级计算机之一(但不是最大的)，可以执行 120 Petaflops 的“密集计算”。

我们不确定这种方法，但他们也声称“人工智能计算”超过一万亿次。该计算机有一个结构背板，可以在节点之间每秒处理 96.8 太比特。根据 Extreme Tech 上的一个帖子，核心技术是一个 3 平面晶片处理器，WSE-2。一个平面用于通信，一个容纳 40 GB 的静态 RAM，数学平面有 850，000 个独立核心和 340 万个浮点单元。

数据被发送到内核，并由一组 64 核 AMD EPYC 3 处理器收集。仙女座菌株被优化来处理稀疏矩阵计算。该公司声称性能“几乎呈线性增长”。也就是说，当使用的内核数量增加一倍时，总运行时间大约减少了一半。

这台机器可远程使用，造价约为 3500 万美元。由于它在高峰运行时间使用 500 千瓦，它也不能自由运行。Extreme Tech 指出，橡树岭国家实验室的前沿计算机更大、更精确，但它花费了 6 亿美元，所以你会认为它的能力更强。

我们看到的大多数自制“超级计算机”更多的是为了[学习如何使用集群](https://hackaday.com/2018/03/21/everyone-needs-a-personal-supercomputer/)而不是试图达到这种性能。当然，如果你有一个现代的图形卡，OpenCL 和 [CUDA](https://hackaday.com/2018/03/19/cuda-is-like-owning-a-supercomputer/) 也会让你做一些，但是规模要小得多。
