# 这只眼睛正通过它的微型显示器看着你

> 原文：<https://hackaday.com/2022/10/09/this-eye-is-watching-you-from-its-tiny-crt/>

阴极射线管(CRT)的时代已经一去不复返了，这通常是件好事。显像管笨重、体积大且易碎，需要复杂的高压电子设备才能工作。但实际上并不是所有的都很大:微型显示管也被生产出来，用于像摄像机取景器这样的东西，来自 Sideburn 工作室的 Tavis 决定把其中的一个变成 T2，一个有点令人毛骨悚然的艺术项目。

这个建筑的核心是一个从 RCA 摄像机中抢救出来的一英寸 CRT。[Tavis]将小管安装在 3D 打印底座上的丙烯酸盒内。在底座内有一个树莓皮，还有一个高压驱动器和一个电源管理板。Pi 连续播放一段视频，视频显示人的眼睛在眨眼并朝各个方向看。只是一只眼睛，漂浮在太空中，看着周围的世界。

当 Pi 启动时，魔力短暂消失，因为它显示了 Pi 的标准启动序列的微观版本，但一旦它运行，它会给房间增添一种怪异的氛围。它实际上看起来像是你在前卫艺术展览中发现的东西——在视频(嵌入在下面)中，它伴随着怪异的音乐，给它带来了更令人不安的感觉。电子眼总是有点吓人，尤其是当[它们真的在看着你的时候。](https://hackaday.com/2021/10/29/this-eyeball-watches-you-thanks-to-kinect-tracking/)
