# 开源实验室摇杆超级流畅

> 原文：<https://hackaday.com/2018/07/20/open-source-laboratory-rocker-is-super-smooth/>

实验室设备通常很昂贵，但预算可能会很紧张，并不总是能够满足小型实验室或研究人员的需求。这就是为什么[akshay_d21]设计了一个带有模块化托盘的[开源实验室摇杆](https://www.thingiverse.com/thing:2948680)，该托盘使用了常见的硬件和 3D 打印部件。该设备产生精确控制的平滑运动，通过以跷跷板运动倾斜连接的托盘来执行自动轻度到中度的样品混合。它可以容纳烧杯或试管，但由于托盘是模块化的，可以设计不同的托盘来满足特定的需求。

[源代码和示意图](https://drive.google.com/drive/folders/1HpQJ8lLGFQqquti2QErEgz5Tcm9hLckl)可从【akshay_d21】的 Google Drive 获得，3D 模型也可从[国立卫生研究院的 3D 打印交易所获得](https://3dprint.nih.gov/discover/3dpx-009288)。下面嵌入了一个演示视频，在视频中你可以看到这些动作是多么的流畅和可控。
