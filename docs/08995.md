# 自制中继计算机看起来像是一个商业产品

> 原文:[https://hackaday . com/2021/01/19/home brew-relay-computer-looking-like-it-can-a-commercial-product/](https://hackaday.com/2021/01/19/homebrew-relay-computer-looks-like-it-could-be-a-commercial-product/)

你可能没有注意到，但是我们在 Hackaday 真的很喜欢我们的 clicky 东西。Clicky 机械键盘，不必要的嘈杂的翻转点显示器，以及几乎所有由许多继电器组成的东西——它们都以不止一种方式抓住了我们的注意力。因此，毫不奇怪，我们似乎完全错过了可能是所有产品中点击量最大的产品:[一台完全运行的 8 位计算机，只使用继电器](https://relaycomputer.co.uk/)。

更令人惊讶的是，我们没能找到并展示(保罗·劳)的优秀作品，因为他在过去十年的大部分时间里都在从事这项工作。他的博客上的第一篇文章描述了 2013 年的构建，当时他正在测试算术逻辑单元(ALU)中的 led。从那以后，[保罗]取得了令人难以置信的进步，构建了一个又一个模块，每个模块都包含了计算机功能的一小部分。这些模块插入带有背板连接的卡盒中，整个组件位于由铝挤压和光滑的黑色面板制成的外壳中，外观非常时尚。对于使用 400 多个 DPDT 继电器来进行思考的东西来说，这台计算机非常紧凑。

除了博客，【保罗】还有一个订阅量不足的 YouTube 频道，里面有[一个最近的系列节目](https://www.youtube.com/playlist?list=PLLACadYlrOka4N5IEfm1cmSuf39ZQSQst)在电脑上深入播放。我们包括下面的整体游览，但你真的应该看看其余的视频，以了解有多少工作进入这个构建。我们已经看到了从[单板](https://hackaday.com/2017/09/18/single-board-relay-computer/)到[大小不等的中继计算机，只是简单可笑的](https://hackaday.com/2020/10/23/relay-computer-consumes-six-years-and-4-5-suitcases/)，但是这一台确实以适合度、光洁度以及功能性而获奖。
