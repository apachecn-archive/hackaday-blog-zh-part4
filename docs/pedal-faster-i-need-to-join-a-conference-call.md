# 踩快点！我需要参加电话会议！

> 原文：<https://hackaday.com/2019/07/14/pedal-faster-i-need-to-join-a-conference-call/>

如今很难找到一辆没有手机充电装置的汽车。毕竟，手机随时都需要充电，而且我们在车里花了很多时间。但是如果你花很多时间在自行车上呢？来自弗吉尼亚州林奇堡的五名青少年决定制造一种用脚踏电源给手机充电的东西。

当然，这不是一个新的想法。你的交流发电机在你的车里给你的手机充电，自行车上也连接了交流发电机，用于照明和其他用途。根据该团队的说法，你需要以每小时 4 英里的速度踩踏板，才能获得足够的电压为手机充电。你可以走得更快，因为电路有一个调节器。我们尤其喜欢他们如何使用转速表和电钻来确定速度与电压的关系。我们也喜欢 3D 打印的零件，比如车把支架，你可以把它用于其他用途。

我们唯一的想法是使用升压/降压转换器，这样即使更低的电压也能产生足够的电压给手机充电。当然，电流会少一些——没有免费的午餐——但至少你可以充电。

也许最好的事情是考虑这样的夏令营，看看下面视频中团队的热情。当一个项目开始工作时，你最后一次像这样容光焕发是什么时候？

我们以前见过这样的自行车充电器，它没有使用 DC 电压和升压转换器。我们还看到了一辆用太阳能电池驱动的自行车。将其中的一个连接到健身车上会很有趣，这样你就可以感觉到你的锻炼正在完成一些事情。
