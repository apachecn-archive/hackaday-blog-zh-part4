# 这条蛇有腿

> 原文:[https://hackaday.com/2022/08/14/this-snake-has-legs/](https://hackaday.com/2022/08/14/this-snake-has-legs/)

[艾伦·潘]喜欢蛇。他是如此爱它们，以至于他决定扮演上帝的角色，抛弃数百万年的进化——仅仅是为了让蛇重新拥有它们“失去”的腿。

好吧，这个黑客已经把舌头牢牢地插在了脸颊上，但它仍然非常酷。[Allen]设计并 3D 打印了一个可以被描述为蛇可以骑的机器人。

建造并不容易。艾伦第一次尝试使用基于[【杰米·曼策尔】的巨型机器人](https://hackaday.com/2013/08/08/six-years-a-giant-robot-and-a-kickstarter/)的玩具，并没有完全按照计划进行。谢天谢地，这些只是用长毛绒蛇测试假人测试的。谢天谢地，阿伦的第二个进球正中目标。

机器人本身由 4 条腿组成，每条腿有 3 个关节和两个伺服系统。足部关节可以自由转动，以应对任何不平坦的地形。机器人的步态源自艾伦在宠物店观察到的蜥蜴。机器人的主体是一根透明的塑料管。一旦~~新治~~蛇决定*进入机器人*，它就不会被绑住。事实上，这条蛇可以随时离开。

目前，整个系统只是向前走。[艾伦]似乎在使用一个带有硬编码行走序列的伺服控制器。我们很想看到下一步——找出一种方法让蛇控制机器人的方向。或许用一个带视线检测的摄像头？

在之前，我们已经介绍过由动物驱动的[机器人，我们也介绍过【艾伦】的一些作品——比如 mjlnir](https://hackaday.com/2022/02/15/so-long-said-all-the-tank-driving-fish/)上的这个[电磁再现。](https://hackaday.com/2015/10/14/thors-hammer-build-recognizes-its-masters-hand/)
