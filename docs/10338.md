# 扁平变压器给这个 PCB 特斯拉线圈一些踢

> 原文：<https://hackaday.com/2021/06/09/flat-transformer-gives-this-pcb-tesla-coil-some-kick/>

可以说，任何特斯拉线圈制作最乏味的部分是缠绕变压器。将细线缠绕到合适的形式上，使一切整洁，并确保它的电气和机械性能良好，这是一个棘手的问题，无论是从最终产品的功能还是美学角度来看，这都是一个成败的命题。因此，这种高输出印刷电路特斯拉应该带走一些沉闷和不确定性。

现在，PCB 线圈已经不是什么新鲜事了——我们已经看到了大量的例子，从[马达](https://hackaday.com/2020/12/11/microstepping-a-pcb-motor/)到[扬声器](https://hackaday.com/2018/09/21/the-diaphragm-is-the-coil-in-these-flexible-pcb-speakers/)无所不包。我们甚至看到了一些 [PCB 特斯拉线圈](https://hackaday.com/2017/10/30/pcb-tesla-coil-is-perfect-desk-toy/)，但正如【雷环】指出的，这些大多是输出较低的线圈，无法带来热量。他的印刷线圈产生了一些非常严重的流光——在某些情况下长达一英尺(30 厘米)。线圈的次级线圈有 6 密耳的走线，间隔 6 密耳，共 240 匝。初级是电路板另一侧的一条 240 密耳的走线，整个封装在透明的两部分环氧树脂中，以防止电弧。在它下面的 PCB 上的非谐振半桥驱动器的驱动下，线圈真的可以装一拳。完整的原理图和构建信息可以在上面的链接中找到，而下面的视频展示了它能做什么。

老实说，对于 PCB 线圈节省的工作量，我们很想尝试一下。它可能没有手绕线圈的经典外观，但它确实完成了任务。
