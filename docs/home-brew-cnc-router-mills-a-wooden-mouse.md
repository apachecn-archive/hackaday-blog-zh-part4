# 自制数控路由器铣木制鼠标

> 原文：<https://hackaday.com/2019/04/15/home-brew-cnc-router-mills-a-wooden-mouse/>

首先，我们承认没有真正的实际理由想要一只木制鼠标，除非 50 年前在“所有演示之母”的[中，这种有争议的纤维素啮齿类动物是开启这一切的动物。简单地在标准无线光电鼠标的内部套上一个外壳只是弯曲，但我们对此并不介意。](https://hackaday.com/2019/01/03/retrotechtacular-the-mother-of-all-demos-50-years-on/)

也就是说，[吉姆·克鲁姆]的设计展示了一些令人印象深刻的技能，无论是鼠标的设计还是机器的制造质量。从一块看起来像白橡木的东西开始，[吉姆]挖出上壳的粗糙形状，然后用一个小球头铣刀进行精炼，然后翻转过来雕刻另一面。他的登记似乎很准确，因为一切都匹配得很好，外壳只有几毫米厚。底板得到了同样的处理，以创建支持鼠标内脏和电池座所需的复杂形状。他甚至磨出了一个小电池盒盖。他用对比强烈的深色木头做滚轮，用装饰带把顶部和底部固定在一起，最后涂上一层薄薄的密封剂。

这是一个很棒的外观，也很实用，正如下面的视频所揭示的。我们之前已经见过其他一些奇特的鼠标，像[这个木头和铝的模型](https://hackaday.com/2016/02/19/turn-your-10-dollar-mouse-into-a-fancy-10-dollar-mouse-with-cnc/)或者甚至[一个会在【查尔斯·巴贝奇】的桌子上看着家的鼠标](https://hackaday.com/2016/11/23/victorian-mouse/)。
