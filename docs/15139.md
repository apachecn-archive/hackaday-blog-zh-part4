# 试图建造世界上最快的 Roomba

> 原文：<https://hackaday.com/2022/10/22/trying-to-build-the-worlds-fastest-roomba/>

很多人抱怨 Roombas 不可靠，工作能力差，或者就是让人讨厌。很少有人抱怨他们在直线上不够快。不管怎样，[electrosync]开始建造世界上最快的 Roomba [来满足他个人的需求。](https://www.youtube.com/watch?v=iLbMRJ9yrbg)

对于这个挑战，[electrosync]制定了自己的规则。该建筑必须看起来像一个 Roomba，使用两个驱动轮，每个轮一个电机。它还必须保持其真空功能。拆下一台二手的 Roomba 后，他开始在机箱中为升级的硬件腾出空间。刷 DC 775 电机被选为动力传动系统，这些运行通过一个 3:1 行星减速齿轮箱。然后，3D 打印的安装架被用来在现有的底盘上安装新的电机。新的 3D 打印车轮完善了传动系统。最初的镍氢电池被一对 3 芯锂聚合物电池取代，以获得更大的功率。

通过蓝牙 GPS 设备测量，升级后的 Roomba 达到了令人印象深刻的 36 公里/小时(22 英里/小时)。它采用了新的聚氨酯橡胶车轮设计和改进的防车轮打滑装置，时速高达 49 公里(30 英里)。添加 4 芯电池将车速进一步提高到 57 公里/小时(35 英里/小时)，但 Roomba 变得难以控制。
