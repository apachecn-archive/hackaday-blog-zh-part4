# 桌面条形音箱非常适合 PC 使用

> 原文:[https://hack aday . com/2022/04/13/desktop-sound bar-is-ideal-for-PC-use/](https://hackaday.com/2022/04/13/desktop-soundbar-is-ideal-for-pc-use/)

条形音箱是一种相当奇怪的扬声器，通常与电视一起使用，以增加一些司机塞进平板电视无法比拟的冲击力。 *DIY Perks* 的【Matt】想要一个更适合在电脑桌上使用而不是在休息室使用的条形音箱，[开始着手制造一个。](https://www.youtube.com/watch?v=nmU7u_RFFSo)

常规的条形音箱不太适合电脑桌，因为它们往往会将声音传递到人的胸部而不是耳朵。[Matt]的设计将扬声器稍微向上倾斜，对准用户的头部，这是应该的。这座建筑使用回收的木地板，这是一种廉价的漂亮木材，不像中密度纤维板那样丑陋或易剥落。

该设计充当显示器支架和键盘箱，将屏幕提升到舒适的观看高度。扬声器本身安装在两侧的隔音罩中，也有助于提供良好的立体声分离效果。架子上还内置了一个低音炮，以增加一些低音响应，具有令人印象深刻的整洁的低音室设计。最后是一些 LED 灯和一个 USB 集线器，该设计提供了出色的声音以及可爱的桌面环境，便于完成工作。
