# 这个 DIY 测力计展示了一个马达能做什么

> 原文:[https://hack aday . com/2020/03/02/this-DIY-测力计-展示-只是-一个马达能做什么/](https://hackaday.com/2020/03/02/this-diy-dynamometer-shows-just-what-a-motor-can-do/)

回到高中时代，所有严肃的齿轮头过去都吹嘘两件事:他们的阻力条票，和他们的测力计报告。前者展示了他们的肌肉车行驶四分之一英里的速度，而后者则记录了他们精心制作的机器能够提供多少功率。我能说什么；汽油很便宜，我们也没有网络来分散我们的注意力。

炫耀的权利并不是杰里米·菲尔丁想要的 DIY 测力计，也不是得到底特律 V8 引擎的细节。相反，他想描述中小型电动机的特征，着眼于将它们用于不同的项目。为此，他建造了一个简单的夹具来测量计算电机功率输出所需的两个参数:速度和扭矩。磁性转速表测量电机的速度，但扭矩证明更具挑战性。测试中的电机与一个独立的电动制动电机相连，当它没有动力时，可以自由旋转。已知长度的杠杆臂一端连接到制动电机，另一端连接到数字秤。随着测试中的电机加速旋转，制动电机逐渐通电，旋转其外壳，并通过杠杆臂在磅秤上产生一个力。神秘马达只需要一点数学知识就能揭示它的秘密。

[杰里米]的视频总是具有指导性，他在发现时明显感受到的喜悦是有感染力的，所以我们很惊讶地看到我们以前没有展示过他的任何东西。不过，我们以前也见过 dynos，从微小的[到](https://hackaday.com/2018/07/22/diy-tiny-dyno/)[计算机化的](https://hackaday.com/2017/12/18/a-dynamometer-for-measuring-motor-power/)到有时会爆炸的那种。
