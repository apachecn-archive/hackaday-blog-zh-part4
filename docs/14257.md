# IOT 车库门开启器为优秀的初学者 IOT 项目

> 原文:[https://hack aday . com/2022/07/19/IOT-garage-door-opener-makes-for-excellent-beginner-IOT-project/](https://hackaday.com/2022/07/19/iot-garage-door-opener-makes-for-excellent-beginner-iot-project/)

如果你住在一个有车库门开启器的家里，你可能经历过一个或多个不可避免的时刻。你把车停在你家门口，按下车库开门器上的按钮，然后……什么也没有。或者你找不到车库开门器。或者你必须反复按下按钮才能得到回应。或者……你明白了。然而，多亏了[Core Electronics],你现在有了使用一个更好的设备来控制你自己的车库门的基础:你的手机。你可以在网上或者在休息下面的[视频中](https://www.youtube.com/watch?v=bIBQKl_Sots&t=2s)[看到教程。](https://core-electronics.com.au/projects/wifi-garage-door-controller-with-raspberry-pi-pico-w-smart-home-project/)

[Core electronics]的[Michael]厌倦了为他的车库开门器处理一个劣质遥控器的不一致性和不方便性。当他检查车库门开启器本身的控制板时，他发现它已经被配置成允许连接三个按钮:上、下和停止。

意识到 MOSFET 可以配置为开关后，[Michael]用三个这样的设备构建了一个电路，然后将它们连接到 Raspberry Pi Pico W。一些 [MicroPython 代码](https://core-electronics.com.au/projects/wifi-garage-door-controller-with-raspberry-pi-pico-w-smart-home-project/)完成了所需的工作，现在[Michael]可以通过手机上的网站访问控制。俏皮！

我们还赞赏该教程给出了该项目的完整视图，以便初学黑客可以决定这是否是他们的项目，并且它讨论了制作车库门和 IOT 设备的一些安全方面。干得好！

鉴于视频中的警告，我们想知道[Michael]是否也看到了 Hackaday 年的文章“[建造 IOT 车库开门器的考验和磨难](https://hackaday.com/2019/07/15/the-trials-and-tribulations-of-building-an-iot-garage-door-opener/)”。
