# 受显微镜启发的刀具更换器可旋转多色 3D 打印

> 原文:[https://hack aday . com/2019/05/15/microscope-inspired-tool changer-spins-多色-3d-prints/](https://hackaday.com/2019/05/15/microscope-inspired-toolchanger-spins-multicolor-3d-prints/)

3D 打印社区只是对更换工具的打印机感到兴奋，但这些机器仍然是例外，而不是常态。这里有一个例外:[Paul Paukstelis]用一种新颖的换头解决方案制造了[一台五色打印机。](https://www.thingiverse.com/thing:3536311)

[Paul 的] 3D 打印机对任何在湿实验室呆过的人来说都是一个礼帽。首先，打印机是从以前的液体处理系统的残余物中诞生的，这是一个巨大的盈余。说到换头，[保罗]结合了一些来自 E3D 工具更换视频的真实灵感和一些从他实验室的显微镜借来的设计特征。其结果是，打印机的五工具头更换器的机械行为非常类似于复合光学显微镜中的鼻架。

因为打印机是从旧的实验室设备演变而来的，[保罗]将他的打印机命名为一个谱系，他称之为“回收快速原型打印机”，或 *RecRap* 。最棒的是，他把[的 CAD 文件上传到 Thingiverse](https://www.thingiverse.com/thing:3536311) 上，这样你也可以深入了解这个换头解决方案。

我们喜欢看到这些工具获得第二次生命，我们认为在这个废弃的实验室设备谱系中有很多新的后代。
