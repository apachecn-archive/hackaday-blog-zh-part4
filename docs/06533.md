# 奶奶记得她的药吗？这个饮水机告诉你！

> 原文:[https://hack aday . com/2020/05/16/did-奶奶记得她的药丸-这-分配器-告诉你/](https://hackaday.com/2020/05/16/did-grandma-remember-her-pills-this-dispenser-tells-you/)

如今，一切都必须智能，虽然智能是一个很好的激励因素，可以让你涉足整个物联网领域，但许多这样的事业往往只是寻找问题的解决方案。然而，最好的情况是，你实际上用它使某人的生活变得更容易，或者帮助一个需要帮助的人。对于[古力·莫拉德]和[德克·宾亚明]来说，当他们建造他们的自动药丸分配器时，两者都有一点:帮助那些依赖药物治疗的人，并缓解那些担心他们是否真的记得的人的焦虑。

使用 ESP8266 和一个相当简单的结构，包括一套附有塑料片的伺服系统，以及一个针对每种药丸类型有策略地放置切口的塑料管，可以使用 Node-RED web 界面自动将预定量的每种药丸分配到一个盒子中，或者在给定时间，或者根据需要。安装在盒子上的簧片开关然后监控它是否在设定的时间内实际打开，如果没有，则通过电报应用程序通知紧急联系人。当然，一个顽固的药物接受者可能很容易欺骗系统，但即使增加一个精确的秤来确保药片确实被取出，也可能对抗这种不愿吃药的患者，所以可以安全地假设这主要是为了防止简单的健忘。

他们的概念验证目前仅限于两种不同类型的药丸，但有足够的 PWM 输出来控制伺服系统，这应该很容易扩展到任何数量。虽然它可能不像几年前我们看到的一些药丸分配器那样复杂，但它仍然完成了它的主要任务。此外，当涉及到人们的健康时，一个足够好的解决方案总是比一个尚未实施的完美想法要好。
