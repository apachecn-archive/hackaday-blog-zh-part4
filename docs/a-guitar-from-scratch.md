# 从零开始的吉他

> 原文：<https://hackaday.com/2022/07/13/a-guitar-from-scratch/>

你可以用任何吉他追求你的摇滚巨星梦想，无论是从昂贵的琴师、当铺还是邮购目录。但是要加入摇滚巨星的行列，有一种方法可以拥有一件真正特别的乐器，那就是自己动手制作。[Bensoncraft]可能不是[Bryan May]的父亲，或者实际上不是[Eddie van Halen]，但他至少通过他的视频“[吉他从零开始:硬模式](https://www.youtube.com/watch?v=F71heUkSdGo)”加入了自制吉他制作人的专属行列。

这是一个相当长的视频，我们把它放在休息时间下面，但这是一个令人信服的视频，因为我们知道制作一把电吉他需要多少操作。当他说硬模式时，他不是在开玩笑，除了琴弦，没有预制的部件，他也没有遵循一套计划。包括调音器和拾音器在内的所有东西都是从头开始制作的，但对于一个非吉他手来说，制作乐器坚固的琴身和琴颈所用的大量不同木材才是最有趣的。即使你永远不会做吉他，你也应该看看。

我们以前从未见过像这样的吉他，但我们给你带来了 3D 打印的吉他琴身、T2 和琴颈。
