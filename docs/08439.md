# 无线鼠标获得充电房

> 原文：<https://hackaday.com/2020/11/23/wireless-mouse-gets-a-charging-house/>

鼠标电池似乎总是在最糟糕的时候没电，比如当你正在工作时或者正在做演示时。[AyhamAS]厌倦了四处寻找替换电池，决定为他们的鼠标建造一个漂亮的小充电底座。

这个构建的核心是坞站内的 TC4056A 充电板。由于这块板是为 3.7 V 电池充电设计的，【AyhamAS】去掉了充电限流电阻，换上了一对通孔电阻。基座背面的开关让[AyhamAS]在快速充电和慢速充电这两个值之间进行选择。

在鼠标方面，[AyhamAS]巧妙地使用了接收器储存格来存放联系人。鼠标和坞站中的磁铁以及弹簧触点为整个体验增加了更多的触觉反馈。坞站本身看起来也很棒——它是由经过磨砂处理的丙烯酸制成的。休息之后，请观看构建视频。

如果你的鼠标有电池组，只要还有空间，你就可以升级到更大的电池组。
