# 无线电力量的危险演示

> 原文：<https://hackaday.com/2020/04/16/a-dangerous-demonstration-of-the-power-of-radio/>

地面广播可能是一个垂死的媒介，但仍然有大量的听众。如果不检查“八号线的交通状况”以确定你是否需要改变路线，或者更新昨天比赛的比分，上下班会是什么样子？将信号发送给尽可能多的听众需要很大的能量，而这个危险但有趣的演示展示了一些无线电塔的能量有多大。

通过 reddit 帖子的方式向我们展示了一组工作人员在 15000 瓦调幅广播塔上工作的视频短片。他们似乎正准备进行塔台维护，这意味着切断天线电源。正如这位工程师解释的那样，中波波段调幅广播电台的天线通常是整个发射塔结构，而调频和电视台的发射塔通常只是将天线尽可能地高出地面。当工作人员断开跳线并在夹具和天线馈线之间形成电弧时，乐趣就开始了。由此产生的等离子球就像一个扬声器，让我们清楚地听到电台的节目。这就像我们以前见过的[等离子扬声器](https://hackaday.com/2016/04/12/rotating-plasma-vortex-speaker/)，尽管异常危险。

这是一个令人印象深刻的电力通过广播塔的展示，也是一个生动的提醒，不要惹他们。可悲的是，这样的警告经常被忽视，而年轻人和愚蠢的人为此付出了代价。毕竟，他们在无线电塔周围竖起栅栏是有原因的。
