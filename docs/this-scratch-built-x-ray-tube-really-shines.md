# 这个临时搭建的 X 射线管真的很棒

> 原文：<https://hackaday.com/2022/09/24/this-scratch-built-x-ray-tube-really-shines/>

无论如何，自制 X 射线管都不是一个好主意。但这并不意味着我们不会谈论它，因为它非常酷。

当我们说制造 X 射线管时，我们指的是它——[atominik]用玻璃试管、钨焊条和一些废金属等原材料来制造这种危险而有趣的管子。他的工具设置也很简约——我们可能会看到一台玻璃吹制工的车床，就像[Dalibor Farny]用来制作[他的定制谢妮管](https://hackaday.com/2016/10/04/the-art-of-making-a-nixie-tube/)的车床一样，[atominik]只有一个小的氧丙烷手火炬来工作。除了明显的真空泵之外，唯一的其他专用工具是自制点焊机，用于将金属部件粘合到用于玻璃-金属密封的钨丝上。

虽然[atominik]制造了几个版本，但最好的电子管是热阴极设计，在铜聚焦杯内有一个涂钍的钨阴极。与之相对的是阳极，一个铜块靶，有一个倾斜的面来引导 X 射线垂直于管的长轴。他还加入了一个钛电极来制造一种吸气剂来清除氧气和氮气，并提高管内的真空。总而言之，它看起来非常类似于商业牙科 X 射线管。

下面视频中的演示既令人信服又令人恐惧。他没有提到他在阳极上使用的电压，但从破裂声来看，我们猜测大约在 25-30 千伏之间。电子管真的让他的盖革计数器滴答作响。

这里希望[atominik]在这些实验中采取适当的预防措施，如果你决定复制这个，你也要这样做。你可能还想看看我们对商业医用 X 射线管内部工程的看法。
