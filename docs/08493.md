# 从头开始建造超大型数控路由器

> 原文:[https://hack aday . com/2020/11/28/scratch-building-a-super size-CNC-router/](https://hackaday.com/2020/11/28/scratch-building-a-supersized-cnc-router/)

我们中的许多人已经在新冠肺炎一级防范禁闭中度过了大半年，我们有什么可展示的呢？网飞队列有进展吗？也许是(稍微)清理过的车库或车间？如果你是*制造东西*的【鲍勃】就不会了。名声:他花了九个月的时间在一台[完全定制的数控刳刨机上工作，这台刳刨机大到足以加工一整块胶合板](https://makingstuff.info/Projects/Full_Size_CNC_Router)。

这一构建被记录在一系列近十几个 YouTube 视频中，第一个视频早在 2020 年 1 月就发布了。看到[Bob]去钢铁厂拿他的框架组件，眼前连一个面具都没有，这提醒了他在这个项目上工作了多久。如果有人想追随他的脚步，他还在自己的网站上列出了一份全面的材料清单。这款车的售价仅略低于 4000 美元，当然不是预算之作。但是当我们谈论这种规模的机器时，没有什么东西是便宜的。

[![](../Images/518e8198df95656c0316fd5ec521a2d0.png)](https://hackaday.com/wp-content/uploads/2020/11/bigcnc_detail.jpg)

Every component on this build is heavy-duty.

即使你没有建造你自己版本的路由器，你也不可能看着建造日志而不从这种机器的可能性中得到启发。在最后一个视频中，我们甚至看到了一点自我复制的动作，因为巨型 CNC 为它自己的电子外壳切割零件。

从视频中你可以看出,[鲍勃]为他的创作感到自豪，并不羞于向观众展示一路上的每一次胜利。即使事情没有按计划进行，当他解释问题以及问题最终是如何解决的时候，也有一些教训可以吸取。

当然，我们知道[一个自制的数控路由器不需要花费数千美元](https://hackaday.com/2015/01/05/resourceful-cnc-router-built-from-hardware-store-parts/)或者[占据台球桌](https://hackaday.com/2014/06/27/building-a-wood-cnc-router-from-scratch/)那么多的空间。普通的黑客读者可能不需要这样的怪物，即使他们需要，也不会有地方保存它。但这并不意味着我们不能羡慕地看着，因为我们等着看[鲍勃]用他的武库中这样一个不可思议的工具制造出什么样的项目。
