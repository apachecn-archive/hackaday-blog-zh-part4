# 这几乎是一个新的树莓 Pi 计算模块 4。但也不尽然

> 原文:[https://hack aday . com/2022/04/08/its-almosphere-a-new-raspberry-pi-compute-module-4-but-not-quite/](https://hackaday.com/2022/04/08/its-almost-a-new-raspberry-pi-compute-module-4-but-not-quite/)

我们知道，读者熟悉全球芯片短缺及其对产品可用性的影响。Raspberry Pi 的人没有逃脱它的阴影，因为即使他们设法保持了他们的 RP2040 微控制器的可用性，公平地说，他们的一些旗舰 Linux 功能的板已经很难找到了。所有这些都产生了一个不太可能的效果，那就是一款新的树莓派，但出乎意料的是，很少有最终用户会拿到它。

Raspberry Pi 计算模块从早期就已经是该系列的一部分，在其早期版本中采用了 SODIMM 外形。最后一个 SODIMM 计算模块有一个 Pi 3 处理器，这个意想不到的新模型据报道有非常相似的硬件规格，但具有 Pi 4 处理器。似乎芯片短缺已经影响了早期 SoC 的供应，为了让 SODIMM 计算模块的众多工业客户保持业务，他们不得不生产这种升级产品。到目前为止，它还没有单独上市销售，并且有可能只停留在工业电路板领域，但是随着故事的发展，有一个关于它的[树莓派论坛话题](https://forums.raspberrypi.com/viewtopic.php?t=332471)，你可以在休息时间下面的视频中找到相关信息。

当然，目前的计算模块仍然是新外形的 CM4，我们认为它[可能是目前所有 Pi 产品](https://hackaday.com/2021/10/14/the-compute-module-comes-of-age-say-hello-to-the-real-cutting-edge-of-raspberry-pi/)中最令人兴奋的。同时[这不是第一个在野外看到的定制工业树莓皮](https://hackaday.com/2021/07/03/a-custom-raspberry-pi-spotted-in-the-wild/)。
