# 科学与网络巨魔:测试另一种文件系统

> 原文：<https://hackaday.com/2021/11/24/science-vs-internet-trolls-testing-another-kind-of-file-system/>

不管你在网上做什么或说什么，你总是做错了。键盘突击队已经准备好扑向你，告诉你它应该如何去做。因此，当火球工具公司的[Jason]因其被认为有缺陷的锉金属方法而受到扶手椅型磨坊主的指责时，就不足为奇了。

但是[Jason]选择了用行动而不是语言来反击，建立了一个系统来测试其他的备案方法。他的锉削风格是让锉刀在前后两个冲程中都与枪托接触，这激怒了那些声称锉刀决不能在工件上向后拖的人，以免牙齿变钝。下面的第一个视频显示了测试设备的构建，它利用他巨大的 Cinncinatti 牛头刨床作为原动机，以及一个气动夹具来固定工件并模仿两种锉削风格。[下面的第二部分](https://www.youtube.com/watch?v=xbykic--SKA)展示了运行中的测试平台，而【杰森】用他的实验方法真正超越了自己。他测试了三种不同等级的 Pferd 文件——只有最好的，没有浪费——并使用互联网认可的风格和他更懒惰的风格重复了每一次运行。

结果呢？我们不会破坏你的想法，但是可以说蜂群思维并不总是正确的。更重要的是，[Jason]小心翼翼地打破神话，获得了一些有趣和意想不到的结果。他的频道充满了[伟大的商店提示](https://hackaday.com/2019/08/13/shop-made-tools-turn-cheap-steel-into-telescoping-tubes/)和[有趣的建造](https://hackaday.com/2020/11/05/giant-blacksmith-vise-from-start-to-finish/)，所以如果你想看看金属加工是如何完成的，就去看看他吧。
