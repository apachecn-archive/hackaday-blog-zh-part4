# LEDSpicer 是一款开源的街机灯光控制器

> 原文:[https://hackaday . com/2019/06/13/led spicer-is-a-open-source-a-light-controller-for-your-arcade-machine/](https://hackaday.com/2019/06/13/ledspicer-is-an-open-source-light-controller-for-your-arcade-machine/)

在这个廉价和容易模仿的时代，建造一个家庭拱廊橱柜比以往任何时候都更有诱惑力。如果你想炫耀，它必须有一个灯光表演，以真正吸引人群。为了让这变得更容易，Patricio 用 LEDSpicer 的名字组装了一个软件包。

这个项目是在[Patricio]制作基于 Linux 的 MAME 橱柜时产生的，他意识到控制他的 Ultimarc LED 板的软件选项有限。由于现有的解决方案缺乏功能，是时候进行编码了。

LEDSpicer 只在 Linux 上运行，并且需要编译，但是对于一般的 MAME 狂热分子来说，这并不是一个很大的障碍。它配有各种各样的动画，以及在游戏过程中创建吸引模式和管理 led 的工具。甚至还有音频反应模式可以让你享受游戏乐趣。它也是开源的，所以如果你想自己添加什么东西的话，很容易修改。

这是一个伟大的包，应该有助于许多商场建设者。led 可以用来对橱柜产生巨大影响；这个帐篷就是一个特别好的例子。休息后的视频。
