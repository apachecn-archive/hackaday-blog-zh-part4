# 3D 可打印运动联轴器，随时可用

> 原文：<https://hackaday.com/2020/08/11/3d-printable-kinematic-couplings-ready-to-use/>

时间可能会带来变化，但运动学耦合不会。[由[nickw]提供的这个方便的运动学耦合资源](https://pinshape.com/items/36874-3d-printed-an-introduction-to-kinematic-coupling)是几年前的一个设计竞赛，但最棒的是它包括了用于 3D 打印的现成模型，以及硬件的材料清单(和 McMaster-Carr 零件号)。这份简短的文件写得很好，并附有装配图和简明实用的理论。附带的 3D 模型可以随时复制并粘贴到任何你可能觉得有用的地方。

什么是运动联轴器？它们是一种确保两个部件以精确和可重复的方式物理连接、分离和重新连接的方法。下载内容包括开尔文和麦克斯韦系统运动耦合的现成设计，以及激光系统中常见的光学机械安装的更高级设计。

从 Pinshape 下载需要一个免费帐户，但是模型和文档是在[CC–Attribution](https://creativecommons.org/licenses/by/4.0/)下许可的，并且可以在设计中使用(当然，只要满足许可的归属部分。)下面是一个简短的视频，演示了使用麦克斯韦系统的耦合。开尔文系统是类似的。
