# 用硬币做一个小盒

> 原文:[https://hacka day . com/2022/03/11/用硬币制作一个吊坠盒/](https://hackaday.com/2022/03/11/making-a-locket-from-a-coin/)

有些国家对销毁或篡改发行的货币有严格的法律，但有些国家没有。属于后一类的公民可能会喜欢建造一个类似于[这个由[Elier Olivos]创造的吊坠盒，由一枚大硬币](https://www.youtube.com/watch?v=4nu-VYUybNQ)制成。

在用线锯将硬币切成两半之前，先用圆规在硬币周边划出一条线。一旦硬币的两半被弄平，接下来是时候对它们进行加热和淬火了，这样它们就更有延展性，可以被捣成稍微有点圆的形状。然后制作金属环，并添加到每一半，以赋予吊坠一定的深度。

然后，仔细制作一个铰链，并将其固定在硬币两半的边缘，使闭合的硬币具有几乎无缝的外观。安装在里面的小插销有助于保持小盒关闭。最后的修饰是为佩戴者在项链上系上一个环，并把项链盒的内部打磨成镜面光泽。

[Elier]通过结合他的双手和一些深奥的工具的帮助，使建造看起来很容易。观看大师工作时的视频可以让人非常放松，最近我们已经看到了一些很好的例子。休息后的视频。
