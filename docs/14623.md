# 介绍裂缝:射频黑客的工具箱

> 原文:[https://hack aday . com/2022/08/27/introducing-裂隙-a-toolbox-for-RF-hacker/](https://hackaday.com/2022/08/27/introducing-fissure-a-toolbox-for-the-rf-hacker/)

不管手头的工作是什么，如果你要处理它，你需要合适的工具。如果你的工作包括理解我们都生活在其中的射频能量虚拟汤中的任何信号，那么你将需要像[裂缝射频框架](https://github.com/ainfosec/FISSURE)这样的东西。

确切地说，裂缝是什么，从它的首字母缩略词就很清楚了，它代表频率无关的基于 SDR 的信号理解和逆向工程。这一切都很新——看起来像是几周前[Chris Poore]在 DEFCON 上发表了一篇关于使用裂隙分析半挂卡车和拖车之间的电力线通信的演讲，他们还计划在下个月的 GNU 无线电会议上发表演讲。我们已经查看了所有我们能找到的关于裂缝的材料，这似乎是一个射频黑客的梦想成真。他们在[推特](https://twitter.com/FissureRF)上已经有了一些例子，比如用十位拨码开关设置的安全码强行打开旧车库门，向汽车发送轮胎压力监测系统(TPMS)信号。他们还在 GitHub 自述文件中提到了该框架的一些功能；我们对各种协议的包制作特别感兴趣。下面的视频提供了更多关于裂缝功能的例子。

看起来裂缝会很有趣，并且对你的射频分析和逆向工程工作非常方便。如果你一直像我们一样使用通用无线电黑客,这看起来很相似，只是更相似。我们将很快下载并试用它，所以请留意实际操作报告。
