# 3D 打印机械装置显示实时天气预报

> 原文：<https://hackaday.com/2022/02/26/3d-printed-mechanical-contraption-shows-live-weather-forecast/>

"今天天气会怎么样？"这个问题几乎永远萦绕在那些生活在英国等地的人们的脑海中，那里灿烂的阳光可以跟随厚厚的云层，一小时后却变成了毛毛雨。如今，你只需要看一眼手机就能知道你是否需要带一把伞，但这有什么意思呢？为什么不用一个巨大的机械显示器向你展示今天天气的概要呢？

作为一个机器人和其他充满齿轮和滑轮的装置的粉丝，[【迈克】决定为他最新的*迈克制作*视频建造这样一台机器。它使用鲜艳的指示器，灵感来自 BBC 著名的“蓬松的云”符号，可以显示阳光、云、雨和雪的各种组合。这些符号由几十个齿轮、杠杆、摆臂和其他移动部件移动，这些部件都是 3D 打印的。我们尤其喜欢从云层后面折出阳光的系统；你可以在下面嵌入的视频中看到它的工作。](https://hackaday.io/project/184132-3d-printed-arduino-weather-api-contraption)

Arduino MKR WiFi 1010 通过开放的天气 API 获取实时天气数据。然后通过一对电机驱动器 IC 驱动机械系统。繁重的工作由步进电机和伺服系统完成，而微型开关和光学探测器决定每个运动的终点。

如果你对天气显示感兴趣，你很幸运:这些年来我们展示了许多不同的风格，包括[电子纸屏幕](https://hackaday.com/2020/11/03/e-paper-weather-display-is-a-great-base-to-build-from/)、[模拟仪表](https://hackaday.com/2020/08/08/vintage-gauges-turned-classy-weather-display/)、[分体式显示器](https://hackaday.com/2018/08/18/custom-split-flap-display-is-a-unique-way-to-show-the-weather/)，甚至还有一个[当地天气的微型再现](https://hackaday.com/2012/11/15/an-extemely-unique-weather-display/)。
