# 通过感应加热保持咖啡温暖

> 原文:[https://hack aday . com/2021/04/11/通过感应加热保持咖啡温暖/](https://hackaday.com/2021/04/11/keep-coffee-warm-through-induction-heating/)

变压器在提高或降低交流系统的电压方面有明显的用途，但它们还有许多其他神秘的用途。电动机和发电机功能相似，可以像变压器一样建模，但真正有趣的应用是在这些工业设置之外。无线充电本质上是一个空芯变压器，它允许电力流过否则是空的空间，感应烹饪使用类似的原理来感应锅碗瓢盆中的电流。[在这种情况下，还有咖啡杯](https://www.youtube.com/watch?v=BQpwUWkhlQA)。

[![](../Images/8c6ae891c3e2ca9f33e71048543b4040.png)T2】](https://hackaday.com/wp-content/uploads/2021/04/induction-heating-coffee-mug-schematic.png)

[Sajjad]的项目是当咖啡放在他的桌子上时努力保持它的温度。为了制造这种特殊的变压器，他将杯子放在一圈粗电线中，电线与方波发生器相连。一个电容器与线圈并联，使设备在特定的调谐频率下实现谐振。一旦达到该频率，线圈就能在咖啡杯的金属部分有效地产生涡流，并以最小的输入能量加热咖啡。

虽然这个项目不适用于陶瓷杯，但[Sajjad]确实用马克杯中的金属勺子演示了它。虽然它没有加热到足以熔化焊料的程度，但如果没有金属杯子，它可以在紧要关头保持咖啡的温暖。他还计划对它进行升级，这样它在他的桌子上占用的空间会稍微少一些。不过现在，当它放在他的测试台上时，它可以很容易地保持他的咖啡杯的温度。
