# 触觉手套无线控制机器人手

> 原文:[https://hack aday . com/2019/10/25/haptic-glove-controls-robot-hand-wireless/](https://hackaday.com/2019/10/25/haptic-glove-controls-robot-hand-wirelessly/)

[米勒]想用一些无线模块练习一下，最后创造了一个机器人手，他可以在触觉手套的帮助下遥控操作。它看起来高度可复制，正如你可以看到的视频，在休息。

该手套使用 Arduino 的模数转换器来读取一些 flex 传感器。商用柔性传感器非常昂贵，所以他试验了一些自制的传感器。锡箔和石墨的效果不好，但使用一些弯曲的金属罐效果更好，尽管分辨率不好。

得益于 NRF24L01 模块，无线通信设置非常简单。最难的部分是把弯曲传感器缝进手套里。我们认为手套上的一些电路看起来也不稳定。

对于机器手，他用轻木和铰链连接每个关节。柔性线像弹簧一样提供回复力。这只手以一种原始的方式表现出惊人的艺术性。

虽然这是一个很酷的演示，但这手牌不太可能实用。[神经冲动](https://hackaday.com/2016/12/31/move-a-robotic-hand-with-your-nerve-impulses/)更好但更难。这只手套让我们想起了我们在 T3 之前在 T2 见过的一只。
