# 用 SLS 树脂整理 FDM 印花

> 原文:[https://hack aday . com/2019/12/25/finishing-FDM-prints-with-SLS-resin/](https://hackaday.com/2019/12/25/finishing-fdm-prints-with-sls-resin/)

[Thomas Sanladerer]有一台基于细丝的 3D 打印机和一台树脂打印机。这两种原材料能结合起来做出更好的东西吗？[【Thomas】做了一些实验](https://www.youtube.com/watch?v=nOZaTB34RrI)用一些磁铁把零件吊起来，用热风焊枪把东西加热。

诀窍是用酒精切割树脂。当然，你也需要使用紫外光进行固化。

这些部分看起来相当不错，尽管他确实根据一些因素得到了不同的结果。为了观察它在实际应用中的效果，他拿了一个非常大的印刷外星人蛋。问题是，鸡蛋放不进腌制站。几分钟的时间，一个散热器，一个钻床，一个 LED 模块，就完成了一个手持紫外线固化灯。

好消息是[你不需要树脂打印机](https://hackaday.com/2019/08/05/resin-printers-are-now-cheaper-still-kind-of-a-hassle/)来利用这个过程——只需要树脂。他还指出，如果你有一些零件需要保持它们的尺寸，因为它们与其他东西匹配，你可以很容易地掩盖零件，使树脂远离这些区域。

如果这个视频(及其显示的结果)让你感兴趣，那么你会喜欢[唐纳德·帕普]去年写的关于他自己尝试用 UV 树脂平滑 3D 打印零件的深入报道。
