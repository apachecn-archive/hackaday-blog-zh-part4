# 1/3 比例的混合动力遥控汽车，配备临时制造的 125cc V10 发动机

> 原文：<https://hackaday.com/2020/09/20/1-3-scale-hybrid-rc-car-with-a-scratch-built-125cc-v10-engine/>

比例模型引擎是迷人的工程作品，遥控汽车总是很棒，不管你的年龄。[Keith57000]在这两个方面都超越了巅峰，创造了一辆令人印象深刻的混合动力 RC 汽车，它围绕着一台定制的 125 cc V10 发动机。

[Keith57000]在 2013 年[完成 1/4 比例的 V8](https://hackaday.com/2013/10/11/a-48cc-v8-with-injection-and-opensource-ecu/) 后，开始制造 V10 发动机。在[论坛的一个帖子](http://www.modelenginemaker.com/index.php/topic,3007.315.html)中记录了这一构建，并附有他精美工艺的大量照片。大多数机械部件都是在手工车床和铣床上加工的。没有 CNC，只有大量的图纸和测量，分度头的巧妙使用，和仔细的表盘读数。该发动机还配备了带有 MegaSquirt 控制器的电子燃油喷射系统。

汽车的其余部分和动力装置一样令人印象深刻。底盘是弯管，有机加工支架和碳纤维悬挂部件。两个电动滑板电机增加了一点点的力量。[三速变速箱](https://www.youtube.com/watch?v=5QerBHz739U)也是定制的，由从坑式自行车和角磨机中清理出来的齿轮制成。它使用两个小型气动活塞来进行换档，并带有一个智能的伺服机构，以机械方式切换电磁阀。查看[频道](https://www.youtube.com/user/Keith5700/videos)上的所有 14 个构建视频，了解更多详情。

一个如此复杂的业余项目绝不会没有障碍，Keith57000 在视频和 build thread 中详细介绍了这一点。到目前为止已经花了七年时间，但毫无疑问，这是我们见过的最令人印象深刻的遥控车。他对手动机床的熟练是我们在数控时代很少能看到的。我们期待着成品，满怀希望地在一条有着 [FPV 驾驶舱](https://hackaday.com/2019/02/17/downloadable-3d-cockpits-enhance-fpv-racing/)的赛道上尖叫。
