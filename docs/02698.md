# 一个 USB -C 烙铁

> 原文：<https://hackaday.com/2019/04/19/a-usb-c-soldering-iron-for-weller-tips/>

曾经有一段时间，一个像样的温控烙铁采取烙铁本身和一盒电子产品的形式，但现在它很可能是一个小型化的事情，将温度控制器内置在一个轻薄的手柄中。像 Miniware TS 系列这样的熨斗已经成为最受欢迎的产品，取代了传统的焊接台。

【Thomas . lepi】[结合了两个世界](https://hackaday.io/project/163827-usb-c-pd-soldering-pen-for-weller-rt-tips)的精华，采用 TS 风格的微处理器驱动手柄驱动熟悉的 Weller RT 元件。它的界面非常简单，但通过它的 USB 电源插座的串口提供了调整的机会。提供控制的是一个[STM 32 f 042g 6 u 6](https://www.st.com/en/microcontrollers-microprocessors/stm32f042g6.html)ARM Cortex M0 微控制器，USB 电源控制来自一个 [STUSB4500QTR](https://www.st.com/en/interfaces-and-transceivers/stusb4500.html) 。


如果你已经习惯了熨斗，比如迷你的 TS100，那么这款带有漂亮的 3D 打印外壳的熨斗将会非常容易使用。TS100 或[的 USB-C 兄弟](https://hackaday.com/2018/06/27/a-sneak-peek-at-the-ts100-soldering-irons-younger-sibling/)的现成可用是否会消除制造这种熨斗的需要取决于你，但这也不是重点。Weller 技巧是同类中较好的一些，所以也许这可能使这个项目值得再看一眼。
