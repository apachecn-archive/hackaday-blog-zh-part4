# 带轮子的全方位步行器

> 原文：<https://hackaday.com/2022/07/26/omnidirectional-walker-with-wheeled-feet/>

[詹姆斯·布鲁顿]正在探索机器人移动的所有怪异和奇妙的方法，在他的最新项目中，他创造了一个可以瞬间向任何方向移动的[全向行走器](https://www.youtube.com/watch?v=e1cMjbQnDwY)。

这种步行器实际上利用了三个独立的四条腿的 Strider 机制，以 120 度角连接成一个三角形。轮子连接到每条腿的底部，与腿的运动平面成直角，以允许脚滑动。改变每个机构的相对速度和方向可以让机器人向任何方向移动，类似于他的[球轮机器人](https://hackaday.com/2021/05/29/robot-moves-in-any-direction-on-ball-wheels/)。每个 strider 机械装置都使用单个马达，看起来类似于 [Strandbeest](https://hackaday.com/2018/08/16/theo-jansen-invents-a-faster-simpler-wind-powered-strandbeest/) 步行者，但它抬起脚来穿越更崎岖的地形。[James]用一些障碍证明了这一点，并发现在这样的方向上移动，所有三组腿都提供最佳结果。

[詹姆斯]计划建造一个更大的可供骑乘的版本，但我们认为他应该安装一个箱子来装他所有的东西，并命名为行李。
