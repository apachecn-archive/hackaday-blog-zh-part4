# 这种不锈钢刀的制作始于铁矿石原料

> 原文：<https://hackaday.com/2022/08/06/this-stainless-steel-knife-build-starts-with-raw-iron-ore/>

在家制作刀具已经成为一种流行的爱好，这在一定程度上要归功于电视真人秀，以及停工带来的空闲时间和闲散人员。根据你对这项爱好的兴趣程度，制作的范围可以从组装和完成一套预锻零件，到用锤子和铁砧进行真正的锻造。但几乎每座建筑都包含来自商业供应商钢材。

不是[这个](https://www.youtube.com/watch?v=G_haToAsS0s)。没有从通常的渠道购买金属，[Thoisoi]的第一站是意大利阿尔卑斯山的一个铁矿，在那里他获得了一大块铁矿石，准确地说是磁铁矿。由于需要高温，从原矿石中冶炼自己的铁并将其合金化成钢通常不是一个后院项目，这个问题[Thoisoi]用铝热剂的魔力解决了。铝热剂混合物中的氧化铁和铝以异常放热的方式[发生反应](https://hackaday.com/2013/05/23/laptop-vs-thermite-slow-motion-destruction/)生成元素铁，在受控条件下，这些元素铁可以被捕获为或多或少纯净的铸锭，准备用于锻造。

在用商业获得的氧化铁进行测试后，[Thoisoi]尝试了他的磁铁矿粉。由于添加了石墨、锰、镍、硅和铬等好东西，他最终能够制造出一块相当大的 402 不锈钢。他把金属交给真正的铁匠进行粗锻；它看起来就像铁砧上的钢铁。成品刀看起来很好，性能也很好，刀片具有不锈钢的特征外观。这是一个不错的结果，而且是以几个粘土花盆为代价的。
