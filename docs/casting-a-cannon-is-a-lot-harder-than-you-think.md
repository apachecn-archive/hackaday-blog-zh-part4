# 铸造一门大炮比你想象的要困难得多

> 原文：<https://hackaday.com/2019/06/02/casting-a-cannon-is-a-lot-harder-than-you-think/>

我们看过后院选角，大部分情况下，我们都知道是怎么回事。你用一两块四块的胶合板做一个框架，弄些沙子，把它压实，然后非常小心地在一个模型周围做一个模子。这完全是另一回事。[【farm craft 101】正在铸造青铜加农炮](https://www.youtube.com/watch?v=tqt7MjcyeAM)。当然，它缩小了一点，但这是理智决定一个人能从熔化的金属中铸造出来的极限。

这种铸造大炮的尝试或多或少是你从后院青铜铸造实验中所期待的。有一个木瓶和一个湿型砂模具，所有的东西都被捣实了，里面还涂了一层爽身粉。然而，这是一个大型铸件，这带来了一个问题:在浇注过程中，砂箱的两半仅用几个 c 形夹钳固定在一起。结局很糟糕，熔化的青铜挤压模具，最终流到车库的地板上。独自做这件事也许不是个好主意。

模具的失败意味着一些数学计算是必要的，在一些快速计算后，发现超过 300 磅的力将模具的侧面推开。第二次倒酒，用尼龙带把瓶子的两边绑在一起，用一个好看的青铜加农炮更成功，准备用一个钢丝轮进行一些虐待。

这只是该系列的第一个视频，接下来的视频将介绍桶的加工和钻孔。那是一艘很棒的飞船。
