# 3D 打印升力体飞机

> 原文：<https://hackaday.com/2019/04/29/3d-printing-a-lifting-body-aircraft/>

当你想到非传统的飞机时，在过去的几十年里，飞翼已经在太阳下呆了很长时间。在飞行线上有着 B-2 精神和 F-117A 夜鹰这样引人注目的设计，这并不奇怪。然而，这个升力体从未真正流行起来，直到今天它还在耻辱中煎熬着。尽管默默无闻，[【rctest flight】还是决定为自己 3D 打印几个升力体](https://www.youtube.com/watch?v=PmRX-CzJbd0)，带他们出去实地测试(YouTube 视频，嵌入下方)。

大多数飞机的机身设计成低阻力，机翼设计成提供升力。升力体飞机将机身设计的重点放在提供升力上，通常没有真正的机翼设计，只需要控制表面来配合机身。为了这个项目，建造了几个不同的设计，飞行器从一个多旋翼飞行器在相当高的高度投放。由于重心问题和不确定的空气动力学现象，最初的试验被稳定性问题所阻碍。早期的设计特别容易突然进入不可恢复的平尾旋。后来的修改包括添加更多的稳定剂，这在一定程度上有助于性能。

3D 打印是实验空气动力学现象的一种很好的方式，因为它很容易创建各种复杂的几何形状来修补。[rctestflight]在开发基本飞行器方面做了扎实的工作，我们希望看到动力测试和更多开发方面的工作继续下去。如果你更喜欢飞翔的翅膀，你也可以 3D 打印出来。休息后的视频。
