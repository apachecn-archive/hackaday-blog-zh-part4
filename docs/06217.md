# 谢妮时钟变成了音箱

> 原文:[https://hackaday.com/2020/04/07/nixie-clock-turns-boombox/](https://hackaday.com/2020/04/07/nixie-clock-turns-boombox/)

有了所有的谢妮时钟项目，想出新的独特的东西真的很难。尽管如此，[the bw]还是用他的[终极谢妮网络闹钟(UNIAC)](http://eequalspi.com/index.php/2020/04/04/uniac-the-ultimate-nixie-internet-alarm-clock/) 做到了这一点，它绝对不吝啬于酷功能。

尽管这款设备确实可以显示时间，但它实际上是一个便携式音箱，使用 Raspberry Pi Zero 运行 Mopidy 从 Spotify 播放音乐。外壳由烟熏丙烯酸制成，加上 IN-12A 谢妮管、IN-13 VU 米和照明按钮，使这款音箱具有 70/90 年代混搭复古风格。丙烯酸外壳很特别，因为它仅由两块弯曲成型的板组成，与常用的手指或 T 形槽设计形成对比，边缘光滑。

为了他的项目，[the bw]设计了一个通用的谢妮显示器，它不仅可以显示时间和日期，还可以显示已经过去或剩余的赛道时间。他还发明了一种 Python 生成的人工语音，可以为你朗读当前的播放列表。[the bw]遇到的唯一问题是试图为该设备设计合适的电池系统，因为启动期间的高电流消耗很容易导致停电。由于时间限制，他最终采用了 MacGyver 风格的解决方案，将亚马逊的 12 V 电池组贴在设备背面。

在种类繁多的谢妮项目中，我们认为我们以前从未在音频播放器中见过它们，除了一些将它们用作放大器的尝试。然而，众所周知 [IN-13 电子管成就了一个伟大的 VU 计](https://hackaday.com/2012/11/05/collection-of-nixie-bar-graphs-bump-to-the-beat/)。
