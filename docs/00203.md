# 微光，作为逻辑

> 原文：<https://hackaday.com/2018/08/03/glimmies-as-logic/>

[Jacob Christ]写了一篇黑客文章，将成为今年夏天的“坐立不安”。为什么？他那一代年轻人最喜欢的玩具是一个闪光玩具。虽然坐立不安的旋转器对旋转很有用，但这种小动物般的微光[似乎有一个意想不到的属性](https://hackaday.io/project/160046-glimmies-logic)，它们可以充当逻辑门。

它们形成了一个光学逆变器，在它们的头部是一个光电晶体管，在它们的腹部是一个 LED，当头部在黑暗中时它就会发光。他通过实验发现，它们可以组合成一个与门，再加上一个反相器就成了与非门。由于所有的逻辑功能都可以用与非门来实现，因此应该有可能制造任何基于逻辑的器件，甚至是全功能的计算机。他估计单扇门的成本为 16.30 美元。一台计算机需要大约 80，000 个微光才能工作，但也许某个财大气粗的人会有足够的胆量去试一试。
