# 2019 年唯一值得拥有的卡带播放器

> 原文:[https://hack aday . com/2019/04/02/the only-cassette-player-worth-own-in-2019/](https://hackaday.com/2019/04/02/the-only-cassette-player-worth-owning-in-2019/)

黑胶唱片有音响发烧友来保持它的相关性，CD 仍然有害怕流媒体音乐的人，但谁会为盒式磁带而悲伤呢？昨天，我们会说没有人，但现在[伊戈尔·阿法纳西耶夫]将他的最新创作释放到一个毫无防备的世界，我们不再那么肯定了。得益于 3D 打印和 led 的广泛应用，一台便携式磁带播放器从善意的 5 美元发现开始，现在已经成为一件非常华丽的电子艺术品。

[![](../Images/e74c3de619e4ddb66e0554f919ce48bd.png)](https://hackaday.com/wp-content/uploads/2019/03/retrotape_detail-1.jpg) 将磁带机构从原始外壳和 AM/FM 调谐器等外部电子设备中解放出来后，【Igor】开始为硬件设计复古风格的外壳，展示传统上隐藏的复杂机电设备。通过添加一个聪明的 3D 打印支架，他甚至能够在原始播放器的按钮下添加微型开关，这样他就可以检测到播放器的当前状态，而不必修改电子设备。这使得完成的播放器可以根据它当前正在做的事情改变 RGB LEDs 的颜色。

[Igor]想出了一个非常聪明的方法，将发光图标集成到表壳中，方法是将明亮的 led 放在特制的印刷薄片后面。在他的测试中，它看起来很棒，但在经过大量的打磨、涂底漆和喷漆后，3D 打印部件变成了生产质量的外壳，最终产品上不再能看到 led。尽管它们在这个特殊的案例中不起作用，但我们认为这是一个值得进一步研究的绝妙技术。

 [![retrotape_icon1](../Images/2037997c082548943156e57742fe2b7c.png "retrotape_icon1")](https://hackaday.com/2019/04/02/the-only-cassette-player-worth-owning-in-2019/retrotape_icon1/)  [![retrotape_icon2](../Images/5429a352a48fb5dab3e555b54c64238e.png "retrotape_icon2")](https://hackaday.com/2019/04/02/the-only-cassette-player-worth-owning-in-2019/retrotape_icon2/) 

这个建筑的细节是惊人的。看到他不得不设计和打印的所有单个组件，以使最终产品组合在一起，这真的是一种激励。像这样的项目是 3D 打印真正发光的地方，因为试图用传统制造技术复制这种建筑绝对是一场噩梦。

如果你不能摆脱以前见过这个名字或对细节的关注的感觉，这是有充分理由的。去年，我们报道了另一个版本，展示了伊戈尔将平凡变成非凡的诀窍。
