# 使用 3D 打印零件升级线修剪器

> 原文:[https://hack aday . com/2022/11/10/upgrading-a-line-trimmer-with-3d-printed-parts/](https://hackaday.com/2022/11/10/upgrading-a-line-trimmer-with-3d-printed-parts/)

许多人抱怨用新的修剪器线重绕除草机的麻烦。制造商的回应是用实心塑料叶片制作模型。不过，其中一些很糟糕，比如这个属于[随机序列]的 Ozito 模型。3D 打印是未来的发展方向，[改造刀片修剪器以使用传统生产线。](https://www.randseq.org/2019/09/line-adapter-for-ozito-blade-trimmer.html)

设计很简单。[随机序列]创建了一个小塑料标签，与 Ozito trimmer 的塑料刀片的附件标签相匹配。在拉环的末端，代替刀片的是一个圆形槽，一段修剪线可以插入其中。诀窍是用点烟器将一个灯泡稍微融化在一段修剪线上，这样它就不会从槽中拉出。离心力(评论里争论一下)让线不掉出来。

[随机序列]在 PETG 打印它们，但注意到该部件可以受益于额外的强度。它们在碰到坚硬的物体时会断裂，就像股票修剪刀片一样。此外，不同于冲击进给修剪器头，没有办法自动进给更多的线。相反，人们必须简单地用新的线手工组装更多的接头适配器。

总的来说，虽然，这是一个伟大的方式来适应更强大，更有能力的修剪线，以除草机，否则由薄弱的刀片。据报道，它可以与 Ozito 和潜在的 Bosch tirmmers 一起工作，并且那些希望自己打印的人可以在 Thingiverse 上看到一些部件。

就像 string trimmer line 曾经被用作 3D 打印灯丝一样，你也可以反其道而行之，[把旧塑料瓶变成 trimmer line。](https://hackaday.com/2022/07/12/weed-eater-string-made-from-plastic-bottles/)如果你在花园里找到了自己的有趣工具，[请不要犹豫，告诉我们。](http://hackaday.com/submit-a-tip)
