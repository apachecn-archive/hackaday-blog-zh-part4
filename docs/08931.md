# 危险的贺卡，你可能不应该给任何人

> 原文：<https://hackaday.com/2021/01/11/dangerous-greeting-cards-you-probably-shouldnt-send-anyone/>

常规的纸质贺卡价格昂贵，当然烦人的音乐种类更是如此。然而，如果你真的想全力以赴，你必须去定制。这就是[Xyla]和[Ian]所做的，[尽管我们不会建议你制作或发送这些给任何人！](https://www.youtube.com/watch?v=bdPggtD6khc&t=2s)

“危险卡”的两种变体被制造出来。一个人将手持蜂鸣器的电击电路重新利用，将其塞入一张闪亮的迪士尼贺卡中。使用音乐贺卡上的开关，当开关打开时，电击电路被激活。电路连接到隐藏在闪亮箔设计中的铝带电极，当用户打开卡时，他们会受到痛苦的电击。火焰投掷的变体是由相同的机制触发的，但不是向电子火柴发送电力，而是在卡片打开时用闪光棉花发射火球。

这些卡片是很好的恶作剧，尽管出于显而易见的原因，我们会对邮寄它们保持警惕。我们以前也看过[Xyla 的]作品，她发光的皮艇是一个特别的亮点。休息后的视频。
