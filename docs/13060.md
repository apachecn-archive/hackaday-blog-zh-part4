# 廉价点焊机拆卸

> 原文:[https://hackaday.com/2022/03/16/cheap-spot-welder-teardown/](https://hackaday.com/2022/03/16/cheap-spot-welder-teardown/)

过去很难通过两块金属释放足够的电流将它们融合在一起。但是锂离子电池可以做到。问题是，应该吗？【信号路径】将[一台廉价的电池点焊机拆开](https://www.youtube.com/watch?v=mt0229lU9sU)看看里面是什么，并试图回答这个问题。你可以在下面的视频中看到拆卸。

廉价的焊机有一些明显的安全问题，所以第一件事是修剪下来一些电线，也是一些 PCB 走线，以确保它们是最低的电阻。当然，线路中的电阻越小，焊接可用的电流就越大。

这位焊工在将金属条固定到电池上方面做得很好。成功焊接后，视频展示了如何使用具有浪涌功能的钳形电流表测量电流。由此产生的脉冲是 960 安培，虽然它立即下降超过 50 毫秒的脉冲，但仍然令人印象深刻。

安全问题令人担忧，但很容易处理，我们建议你买一个也做同样的检查。一些评论者提到，他们版本的类似焊机没有同样的问题，但在你烧毁你的商店之前，值得努力检查。

之前，我们也有自己对廉价点焊机的担忧。这不是我们看到的第一个在首次使用前通过安全检查的[廉价装备](https://hackaday.com/2022/03/10/reflow-hotplate-teardown-uncovers-the-bare-minimum/)。
