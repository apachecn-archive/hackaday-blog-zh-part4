# 简单的手工工具把黄铜和钢变成了一个神奇的星盘

> 原文：<https://hackaday.com/2019/02/09/simple-hand-tools-turn-brass-and-steel-into-an-amazing-astrolabe/>

古代工具和仪器有一种迷人的魅力。我们的祖先只用最简单的手工工具就能制造出精密的机械装置，这个想法很吸引人。看着某人重现这一壮举，比如用手[制作星盘](https://www.youtube.com/watch?v=8U7ED6k5NLg)，也会非常吸引人。

星盘是一种古老的天文工具，具有令人难以置信的多功能性，允许用户做任何事情，从计算太阳何时升起到预测夜空中几十颗星星的位置。它只用几个运动部件就完成了这一切，这让它变得更加迷人。[尤里·塔奇曼]只用一些手工工具就开始了下面视频中的星盘制作。不过，他很快就厌倦了手工线锯的工作，并发明了一个由旧缝纫机脚踏板驱动的简单的线锯来加快工作速度。不过真正的享受是手工雕刻，这是一项[Uri]显然已经掌握的技能。我们不禁想到，CNC 路由器可以更快地做同样的事情，但看着[Uri]做这件事更令人满意。从对比鲜明的黄铜和钢部件到复杂的阿拉伯文字标记的选择，建筑的每一部分都很有表现力。[Uri]有另一个视频，讲述了星盘的基础知识和他的设计过程，也很值得一看。

虽然它远不是一个复杂的仪器，但这个星盘让我们有心情再次观看整个点击发条时钟的构建。[克里斯]目前正在研究他自己的古代乐器，重建安提基西拉机制。我们迫不及待地想看那部电影。
