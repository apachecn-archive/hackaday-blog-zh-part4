# 36C3:从 A 到 Z 的 SIM 卡技术

> 原文:[https://hack aday . com/2019/12/30/36 C3-sim-card-technology-from-a-to-z/](https://hackaday.com/2019/12/30/36c3-sim-card-technology-from-a-to-z/)

SIM 卡就在我们身边，随着物联网的持续增长，NB-IoT 等技术的诞生，这可能很快就会成为现实。但是我们对它们、它们的内部结构和它们的通信协议真正了解多少呢？推而广之，他们的安全？为了阐明这些问题，开源和移动设备巨头 LaForge 在德国莱比锡的 36C3 上做了一个关于 SIM 卡技术的介绍性演讲。

从基于德国 C-Netz 的早期蜂窝网络的简短历史课以及 SIM 卡本身的起源开始，[LaForge]讲述了从 2G 到 5G 的每一代产品的主要规范和技术部分。涵盖物理基础、I/O 接口、通信协议和 SIM 卡上的文件系统，您将得到“PIN2 到底是干什么用的？”一路上。

当然，在 CCC 活动中，如果没有对安全方面进行深入和批判性的审视，这样的演讲是不完整的。考虑到软件和功能方面的无线更新越来越普遍——由于现在大部分都在运行 Java 这肯定有值得关注的地方。
