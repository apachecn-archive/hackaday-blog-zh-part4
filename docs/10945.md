# 建造一个太阳能游戏机口袋

> 原文:[https://hack aday . com/2021/08/08/building-a-solar-powered-game-boy-pocket/](https://hackaday.com/2021/08/08/building-a-solar-powered-game-boy-pocket/)

光线一直是经典游戏男孩体验的关键部分。我们中的一些人有着美好的回忆，晚上坐在汽车后座上，暂停和取消暂停游戏，因为头顶上的路灯短暂地提供了足够的光线，可以看到没有照明的显示器。这些经典手持设备的第三方 IPS 显示器的出现在很大程度上消除了这个问题，但正如您所料，更现代的屏幕对功率的要求增加了，这降低了系统的运行时间。

[![](../Images/c5c2bf3507c500df8d94bfcd85afdfc6.png)](https://hackaday.com/wp-content/uploads/2021/08/solargbp_detail.jpg)

Installing the USB-C charge controller.

作为能源生产研究的一部分，[【休斯顿自然科学博物馆】着手研究他们是否可以通过在已经改装了 IPS 显示屏的 Game Boy 口袋](https://www.youtube.com/watch?v=eKTeAgDiR-c)的背面添加太阳能电池板来改善情况。Game Boy 的口袋版被选中，因为它有一个很好的平坦背部，可以很容易地连接太阳能电池板，事实上，这个 mod 的电池板尺寸很好，看起来就像是这样的设备。

在下面的视频中，你可以看到修改开始时切掉了 Game Boy 的后面板的一大部分，以适应 1000 mAh 的 LiPo 电池。然后用强力胶将太阳能电池板贴在背面。一个二极管焊接在太阳能电池上，然后连接到带有 USB-C 输入的充电控制器。充电控制器的位置最终比预期的要复杂，但是用一点热胶水，它就工作得很好。总的来说，这是一个简单的模型，但却是一个绝妙的主意。

这不是我们见过的第一个太阳能手持游戏系统，但很高兴看到这个想法被重新审视和扩展，特别是在人体工程学方面。此外，我们喜欢这个黑客慢慢成型时叙述的令人难以置信的细节。休息后的视频。
