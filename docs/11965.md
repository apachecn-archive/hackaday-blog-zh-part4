# 机器人手臂的廉价大伺服系统

> 原文：<https://hackaday.com/2021/11/19/cheap-big-servo-for-robot-arm/>

[Skyentific]正在寻求推动业余爱好者机器人技术的发展。马达和它们的齿轮，即致动器，通常是最昂贵的部分。对于他的体型，他意识到他需要能够提供足够扭矩的大伺服系统。因此，他着手创造一种 3D 打印设计，以便在预算范围内完成这项工作。(视频，下面嵌入。)

与无刷解决方案相比，步进电机的成本较低，尤其是在考虑控制硬件时，因此受到青睐。在这种设计中，步进电机驱动一个太阳齿轮，作为一个更大的高传动比行星齿轮箱的一部分。交叉滚子轴承用于允许伺服有效地处理径向和轴向载荷。伺服系统作为一个整体被设计成整齐地安装到机器人手臂本身的关节中，并具有这样的外部安装点。

这是一个整洁的伺服系统，有点像全尺寸工业设计中使用的伺服系统，至少在某种意义上是机器人手臂关节的一个组成部分。根据[Skyentific]使用的材料，它的价格也相对便宜，只有 32 美元。
