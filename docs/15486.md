# 简易的 BBS 指南

> 原文:[https://hack aday . com/2022/11/29/a-handy-guide-to-the-huble-BBS/](https://hackaday.com/2022/11/29/a-handy-guide-to-the-humble-bbs/)

我们中的一些人很早就上网了，他们深情地记得过去的网络——闪烁的横幅、令人分心的背景和自动侧滚的文本。但是，在我们今天所认识的万维网和互联网出现之前的一段时间里，交流的方式是通过公告板系统，或 BBS。今天仍然有一些人可以引用这种深奥的魔法，这一页也许是这种逆向计算风格的权威指南。

本指南由[Blake 负责。 Patterson】他使用各种古董机器和一些现代硬件来访问仍在使用的 BBSes。他在本指南中指出，使用 telnet 和现代计算机访问它们是可能的，但使用 Amiga 或 Atari 之类的东西会给你完整的体验。有一些工具可以将电话调制解调器信号从原始硬件转换为现代网络设备可以理解的信号，虽然体验可能会稍微快一些，但它似乎很好地保留了怀旧因素。

对于那些寻找更具体指南的人来说，我们在过去已经介绍过几次[布雷克]的作品，[一次是用一台老式的爱普生 PX-8 笔记本电脑](https://hackaday.com/2018/04/18/browsing-modern-day-bbs-on-the-epson-px-8-laptop/)和[一次是用一台现代的 ESP8266](https://hackaday.com/2017/06/17/bbsing-with-the-esp8266/) 。连接到这些旧服务不需要太多的计算能力，所以抓住你能抓住的任何东西，开始 BBSing 吧！
