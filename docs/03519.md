# 面条做的吉他在黑暗中会发光

> 原文:[https://hack aday . com/2019/07/06/面条做的吉他会在黑暗中发光/](https://hackaday.com/2019/07/06/guitar-made-from-noodles-glows-in-the-dark/)

木头。具体来说，某些类型的色调木；赋予某种色调的木材。吉他就是由这种材料制成的。偶尔，塑料，或者金属，或者玻璃纤维，或者任何东西。[_forwardaudio_]用面条打造了他的[，因为，为什么不呢？](https://imgur.com/a/Q1WO1Ws)

嗯，不是完全没有面条。环氧树脂被用来给面条增加一些强度，因为，尽管面条给吉他带来了美妙的音色，但它们的强度并不为人所知。环氧树脂有助于保持面条的位置，突出其面条的味道。

为了给吉他的外观增加一点冲击力，琴身的背面和正面混合了紫外线粉，正面是蓝色，背面是绿色。一旦吉他被组装好，一组紫外线弦也被加入，以增加更多的光辉。

在视频中(休息后),显示了构建过程以及简化的仅卷布线。最后，[_forwardaudio_]在吉他上转了一会儿。

我会出去的。
