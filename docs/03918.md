# 黑客日播客 031:完整的驱动器，DEF CON 的徽章，我们不做片上调试，和小规模制造混乱

> 原文：<https://hackaday.com/2019/08/16/hackaday-podcast-031-holonomic-drives-badges-of-def-con-we-dont-do-on-chip-debugging-and-small-run-manufacturing-snafus/>

Mike Szczys 和 Kerry Scharfglass 在 DEF CON 现场录制了本周的播客。在众多讨论主题中，我们探索了一些更有趣的移动机器人的方法。从 BB-8 到完整的驱动器，凯瑞希望能及时证明超级计算机的概念。你在项目中使用片上调试吗？我们也不是，但也许我们应该。动态内存分配也是如此；但是，当你已经击败了诸如 Teensy 4.0 上的芯片等微处理器时，你为什么还需要这样做呢？我们以对徽章制造商的一些采访来结束本周的展览，这些制造商推出了数百种设计，并在此过程中遇到了制造问题。没有问题要解决就不是工程了。
