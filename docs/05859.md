# 喷气战斗机玩具吸塑包装是一张名片

> 原文：<https://hackaday.com/2020/03/05/blister-pack-with-jet-fighter-toy-is-a-business-card/>

在名片的世界里，对一些人来说，一张白色的长方形卡片似乎不再有用了。名片不仅仅是显示你联系方式的一种方式，相反，它也是一种展示你的工作和向世界展示你能力的方式。对[agepbiz]来说，这些是 3D 设计专家的技能，那么还有什么比分发他的作品的 3D 打印样本更好的方式呢？如何把它变成一张名片？[当然是放在零售式的泡罩包装里](https://www.youtube.com/watch?v=mc6YA-sxSL8)。看看休息下面的视频。

这是一个有趣的过程，因为肯定有读者会想到出售他们的作品，这是一种很有吸引力的方式来展示一个小组件，同时仍然保持它的安全不受损坏。这个玩具—[是一个小型的 3D 打印喷气式战斗机](https://www.thingiverse.com/thing:2887652),带有工作的摇摆翅膀，本身就是一个杰作——被放在一张底牌上，上面粘着一个定制的泡罩。示出了用 CNC 卡片切割机制造印刷的背卡，接着是用定制的 SLA 印刷模具制造泡罩，该模具用于真空成形透明塑料片。令人惊讶的是，整个组装只是一个胶棒，我们原本以为会有更多的抓取。结果是一个专业外观的泡罩包装产品，如果你在商店里看到它，你不会眨一下眼睛，这是非常有用的事情之一，了解一个产品是如何制作的..

这张卡片放在你的钱包里可能有点大，但这并不是这些年来我们带给你的唯一一张新颖的卡片。一些我们最近最喜欢的运行 Linux 的[或者玩俄罗斯方块的](https://hackaday.com/2019/12/24/now-even-your-business-card-can-run-linux/)或者。
