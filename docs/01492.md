# 电子人，或多叶传感器阵列？

> 原文：<https://hackaday.com/2018/12/12/cyborg-or-leafy-sensor-array/>

有些植物反应迅速，足以让我们的感官注意到，如捕蝇草或含羞草。大多数时候，我们至少需要延时摄影才能注意到，而更奇特的传感器可以测量像微孔打开和关闭这样的事情。与任何传感器读数一样，这些测量可以通过一个我们称之为自动化的小技巧转化为行动。麻省理工学院的[Harpreet Sareen]和[Pattie Maes]以一种我们以前从未见过的方式将这两种想法结合在一起，在一个名为 elo wan 的项目中，植物占据了主导地位。细节不多，但概念很容易理解。

我们不确定这是否是一个成熟的电子人，或者这是一个使用生物传感器的机器人。也许这一切都取决于你从哪个角度来呈现这个工厂和机器的混合体。也许这真的是该项目所宣称的共生关系。没有植物，机器人将不会接收到任何指令，并且没有机器人，植物将接收到次优的光。还有哪些方法可以将植物整合到机器人技术中，使其成为真正的机器人(T2)？
