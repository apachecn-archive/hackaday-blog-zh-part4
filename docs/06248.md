# 脉搏可视化是一个真正的心脏工作

> 原文:[https://hack aday . com/2020/04/15/pulse-visualizer-is-a-real-work-of-heart/](https://hackaday.com/2020/04/15/pulse-visualizer-is-a-real-work-of-heart/)

有些项目非常简单，比如柠檬电池或者土豆钟。其他项目植根于简单的想法，但它们的设计和执行将它们提升到另一个水平。[【Sharathnaik】的心脏可视化仪在电子方面可能并不那么复杂，但其执行过程却令人心跳加速](https://www.instructables.com/id/Heart-Visualizer-See-You-Heart-Beat/)。

我们大多数人最接近自己心跳的方式是观察脖子或手腕上的皮肤抽动。你知道你的心在让你活着，但很难体会它是如何发挥作用的。只需几个组件和打印部件，当你的脉冲驱动单 x 剪刀机构推拉塑料板时，心脏的泵送动作就变得栩栩如生。

这种心脏可视化仪不像它模拟的器官那样复杂，对于任何刚开始接触电子产品的人来说都很容易构建。将手指放在底座的心率传感器上，Arduino Nano 会根据您的个人心跳驱动一个伺服系统。我们希望看到有人激动的时候它加班加点。现在，休息之后有一个平稳的演示，然后是组装视频。

心跳感应也可以很浪漫。这里有一个可爱的电路雕塑，它以接收器的速率运行。
