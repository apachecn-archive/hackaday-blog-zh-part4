# 黑客日播客 114:眼睛在看着你，外星艺术，数控电锯和 Galvie 流感

> 原文:[https://hackaday . com/2021/04/16/hackaday-podcast-114-eye-is-watching-you-alien-art-CNC-chasaw-and-the-gal vie-flu/](https://hackaday.com/2021/04/16/hackaday-podcast-114-eye-is-watching-you-alien-art-cnc-chainsaw-and-the-galvie-flu/)

Hackaday 的编辑埃利奥特·威廉姆斯和迈克·什奇斯对过去一周出现的黑客攻击感到惊讶。一个令人瞠目结舌的网络摄像头黑客以电子动画的形式出现，给你屏幕上方的摄像头一个环顾四周的眼球，一个眨眼的眼睑，以及皮肤，睫毛和眉毛来完成幻觉(同时让我们不寒而栗)。

丹深入研究了锌流感——焊接镀锌金属等含锌部件时要避免的东西。给了一个机械臂一把电锯，导致了很多 hijinks 其中最头疼的就是路径规划这样的机器。我们听到了一个非常棒的故事，通过使用版权局的主要工具之一，复活了一个默默无闻的电脑游戏。
