# 一个 Lambda 8300 重生了

> 原文：<https://hackaday.com/2020/12/10/a-lambda-8300-lives-again/>

如果你跟踪回溯计算——或者你已经足够大，可以记住那些日子——你会一遍又一遍地听到相同的名字。例如，准将、苹果、无线电室和辛克莱。但是 [Lambda 8300](https://www.youtube.com/watch?v=JrV4uEMKkX8) 呢？大多数人没有听说过这些，但[迈克]听说过，他有不少。这台计算机类似于 Sinclair ZX81，但不是一个精确的克隆。他的所有机器都需要一些维修(他承诺的维修视频正在路上)，但对于下面的视频，他将显示器直接连接到 PCB 以获得稳定的输出，因此显然 RF 调制器是这种情况下出现故障的子系统。

一旦视频清晰，您可以看到运行一个简单的 BASIC 程序的演练。在那个时代，计算机通常使用盒式录音机来存储数据。[迈克]拿起一些专用于计算机使用的录音机，但都不在工作状态。然而，消费者播放器工作正常。

磁带不太可靠，所以[Mike]将音频加载到他的手机中，然后用于将来的加载。我们被快速的拆卸弄得很兴奋，想看更多，但是[Mike]说至少还有两个视频在路上，所以我们想我们必须冻结视频，呆呆地看着公告板，直到他们到来。

这些机器制造得很便宜，所以 CPU 做了所有的事情。键盘输入/输出、屏幕输出和盒式磁带接口都直接与 CPU 相连，尽可能少用硬件辅助。辛克莱机器也因此而闻名，这使得它们的操作很特别，就像你在视频中看到的那样。在美国，这些电脑的商标是“你的电脑 PC 8300”和“Unisonic Futura 8300”它们在不同的市场上以其他的名字出售，但是没有辛克莱和它的直系亲属成功，据报道他们卖了 150 万台。

一方面，这些廉价机器大概推出了很多崭露头角的年轻程序员。但是你也想知道有多少在圣诞节早上被打开，然后被塞在壁橱后面，在新年的时候被遗忘了？

我们还没有看到任何 Lambda 8300 仿真，但它足够接近 ZX81 和[那些确实存在的](https://hackaday.com/2014/12/22/zx81-emulated-on-an-mbed/)。你可以发现人们仍然在这些机器上工作，[将它们连接到互联网](https://hackaday.com/2016/11/23/zx81-connects-to-the-network/)并生产[附加板](https://hackaday.com/2016/11/21/sinclair-io-board-completed-over-30-years-later/)。
