# 恢复一个老式的 IBM I/O 测试器

> 原文:[https://hack aday . com/2021/12/19/restoring-a-vintage-IBM-I-o-tester/](https://hackaday.com/2021/12/19/restoring-a-vintage-ibm-i-o-tester/)

到目前为止，[CuriousMarc]和他的志愿者团队已经精通 20 世纪 60 年代的硬件修复。因此，当一台老式 IBM I/O 测试机落入他们手中时，一次全面的机器改造几乎是不可避免的。

I/O 测试器可以追溯到 1965 年左右，这大致与 IBM 广受赞誉的 System/360 计算机主机的推出相吻合。除了计算机本身之外，企业客户还可以为他们的计算系统订购各种外围设备。这些包括存储设备、打印机、附加的操作员控制台等等。由于这些外设共享相同的 I/O 设计，便携式硬件测试平台是明智的设计选择。一台便携式低压测试仪可以与任意数量的 IBM 外设配对，这样就不需要在每一个计算硬件上都有独特的调试面板。

快进到今天，IBM I/O 测试器看起来肯定是老式的，带有闪光灯面板和开关。要使用测试仪，只需将它的一个(或两个)104 针连接器连接到您选择的 IBM 外设上，插入附带的纸覆盖层，然后*瞧。*然后，操作员可以观察许多灯的状态，以评估所连接外围设备的内部数字工作情况。根据连接的硬件，测试仪可以显示数据寄存器的内容、打印状态、磁盘和磁带传输状态，甚至更多。测试仪的 90 个指示灯的用途完全取决于所连接的外围设备，成对的纸覆盖对于理解它们的含义是必不可少的。

在[Ken Shirriff]破译了文档之后，没过多久，测试仪就可以使用 24 VAC(通常由被测设备提供)通电了。注意到几个烧坏的灯需要更换。由于不确定的设计选择，灯组件需要小手术，并且至少一个拨动开关需要新的导轨和大剂量的触点清洁剂才能恢复使用。

目前，[CuriousMarc]正在使用 blinkenlights 面板作为一个令人惊讶的复古时钟。随着一卡车的 IBM 旧硬件被存放在他的仓库里，看看这个修复的测试器是否有一天会重新投入使用，这将是令人兴奋的。读者也应该看看我们对他之前主要项目的报道，修复一台[阿波罗制导计算机](https://hackaday.com/2018/11/17/restoring-an-apollo-guidance-computer/)。
