# 用聚氨酯树脂制造柔性包覆成型部件

> 原文：<https://hackaday.com/2019/07/19/making-flexible-overmolded-parts-with-urethane-resin/>

树脂铸造视频最近在社交媒体上掀起了风暴。从镶嵌的浮木桌子到别致的钢笔，所有的东西都得到了处理。倒一些颜色漂亮的环氧树脂很简单，但这只是冰山一角。[Eric Strebel]在树脂浇铸方面有一些重要的技能，[最近一直在研究一些用聚氨酯树脂包覆成型的电子产品(Youtube 链接，嵌入在下面)](https://www.youtube.com/watch?v=iTY2ABhtTKg&feature=youtu.be)。

构建从使用 3D 打印的 SLA 母版创建硅胶模具开始。该器件用于原型医疗设备，需要包覆成型，其中柔性 PCB 覆盖有柔性聚氨酯。木制引脚用于将柔性 PCB 卡入模具以实现精确定位，并且在 PCB 的金属触点上放置一个小屏蔽，以避免它们被硅树脂覆盖。

在准备生产实际零件之前，先用空模具进行初步测试，以确定要使用的正确材料。[Eric]非常关注最终产品，因为任何错误都会浪费客户提供给他的昂贵的 PCB 原型。将电子元件放入模具后，使用注射器将树脂脱气并小心注射，以最大限度地减少气泡的出现。经过一些精细的手工清理，完成的零件就可以交付了。

这一过程涵盖了原型零件包覆成型的基本知识，也展示了[Eric]生产高质量原型零件的技能。我们以前也看过[Eric]的作品，比如他对纸板在产品设计中的价值的讨论。休息后的视频。
