# 无人机用小型汽油发电机样机的设计

> 原文：<https://hackaday.com/2019/07/20/designing-compact-gasoline-generator-prototype-for-drone-use/>

锂电池和无刷电机帮助多旋翼无人机成为可能，但电池只能持续这么久。液体燃料具有高得多的能量密度，但还没有广泛应用于这些方面。[Tech Ingredients]一直在试验一种紧凑型汽油燃料发电机，[旨在延长无人机的飞行时间，远远超过目前使用电池可能实现的时间(Youtube 链接，嵌入下方)](https://www.youtube.com/watch?v=1yB49G756OI)。

建造始于单缸四冲程发动机。然而，扭矩峰值和振动使事情变得困难。经过一些迭代，设计决定采用两个单缸二冲程发动机，装有同步带，以保持它们 180 度异相。结合一对平衡飞轮，这使振动降到最低。无刷电机被用作发电机，结合整流二极管和电容器来平滑电压输出。该发电机旨在与锂电池组并联使用，以确保无人机始终有电力可用，即使在临时故障的情况下。

这是一个充满希望的建造，我们迫不及待地想看看一旦系统完成并准备好飞行，能达到什么样的飞行时间。我们已经看到其他人也在试验混合动力无人机。
