# 机械键盘结束了，这个设备赢了

> 原文：<https://hackaday.com/2022/06/24/mechanical-keyboards-are-over-this-device-has-won/>

在 21 世纪 20 年代，任何自尊的技术爱好者的办公桌都离不开某种特殊的键盘，无论是老式的 IBM 型、深奥的布局或形状因素，还是由咔嗒作响的机械开关制成的标准键盘。但是也许我们已经找到了一个深奥的键盘来统治它们，以[【HIGEDARUMA】的 8 位键盘](https://hackaday.io/project/185991-the-8bit-keyboard)的形式。你们现在都可以回家了，这场比赛已经由这个具有最简单前提的输入设备赢得了胜利；通过在一行拨动开关上将 ASCII 值设置为二进制来输入文本。没有键盘比最早的微型计算机上的键盘更复古了！

玩笑归玩笑，也许这个键盘对许多读者来说有点深奥，但它仍然是一个执行良好的项目。除了二进制输入行之外，还有一个按键按钮，可以将任何值发送到计算机，还有一个股票按钮，可以将多个输入作为一个输入进行存储和发送。如果你暂停一会儿，想想你使用 Ctrl-C 和 Ctrl-V 的频率，这是一个必不可少的函数。在日本网站 ( [谷歌翻译链接](https://protopedia-net.translate.goog/prototype/2593?_x_tr_sl=auto&_x_tr_tl=en&_x_tr_hl=en&_x_tr_pto=wapp))上有更多的信息[，它揭示了引擎盖下是一个运行在 ESP32 上的蓝牙设备。](https://protopedia.net/prototype/2593)

我们可以想象，稍微使用一下，很快将 ASCII 记忆为二进制是可能的，事实上，当发现读者已经拥有这种技能时，我们一点也不会感到惊讶。但不知何故，我们无法想象它曾经是一个特别快速的文本输入设备。你们自己看看吧，就在休息时间下面的视频里。
