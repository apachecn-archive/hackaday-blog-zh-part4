# 新冠肺炎统计:解读茶叶

> 原文:[https://hackaday.com/2020/03/28/reading-the-tea-leaves/](https://hackaday.com/2020/03/28/reading-the-tea-leaves/)

如果你像我们一样一直在追踪新冠肺炎疫情在世界各地的传播，你无疑已经看到了大量的统计数据。原始数据看起来令人震惊，在许多情况下确实如此，但像往常一样，问问自己这些数字意味着什么是至关重要的。

例如，我们自己的汤姆·纳尔迪组装了一个计数器，显示美国的新冠肺炎病例总数。这是一个很酷的项目，它将一些网络抓取、一个漂亮的有机发光二极管屏幕和一个 3D 打印的网络显示结合在一起。当这一切结束时，可以很容易地对它进行重新训练，以显示一些其他感兴趣的统计数据，这是对许多 web APIs 的一个很好的介绍。然而，它看错了号码。

让我解释一下。疾病呈指数级传播:患病人数越多，传播疾病的人就越多。当你画出它们的瞬时值——新冠肺炎病例的原始数量时，指数曲线看起来都是一样的。相反，将一个指数与另一个指数区分开来的是增长参数，这与每天新增病例的数量有关，或者更准确地说，与新增病例的日常变化有关。

如果不加以控制，特别是在传播的早期阶段，新病例的数量每天都在增加。但是随着控制措施的实施，主要是社会距离的减少，新病例的数量会减少，甚至是减少。总之，计划就是这样。

这个视频很好地解释了这一点，如果这是一种自然传播的流行病，那么新病例开始下降的时间点就标志着疾病进程的中点。在这里，我们希望特别严格的隔离程序将缩短这一过程，但如果你对疾病的传播方式感兴趣，每天新感染的转折点就是你要寻找的。

那么，为什么不把每日差异放在你的桌面上呢？这些数字是有噪声的，差异在四周跳跃。说正经的，你可能会想在新的案例数字上加上一个移动平均值，看看有什么不同。或者只是简单地展示新的案例，然后期待它连续几天下降。

尽管如此，这并不是一个完美的措施。首先，新冠肺炎似乎在没有症状的情况下潜伏了大约一周。这意味着无论我们有多少数据，它们都可能比实际情况落后一周。我们至少在一周内不会看到社交距离的影响，也许更久。

更复杂的是测试的可用性，人为因素，如周末，更多的人接受测试，但更少的政府报告办公室开放，时区等。(2 月 13 日发生了什么？)

我不打算说我们看到的新冠肺炎统计数据毫无用处——事实上远非如此。但如果你要在这个疫情上独领风骚，那就做对了。画出每天的新案例，至少在你的脑海中，也许应用一点平滑，并意识到你现在看到的可能代表上周发生的事情。

当你终于看到转折点的时候，你可能会稍微庆祝一下，因为那意味着中点是在一周之前。我们已经看到它在 2 月 2 日左右在中国发生，我期待它在这里发生。我希望无论你在哪里，它都会发生，而且越快越好。

我们会挺过去的。大家注意安全。并且让自己不被感染才能让别人不被感染。
