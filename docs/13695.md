# 构建您自己的 8088 XT 主板

> 原文：<https://hackaday.com/2022/05/21/building-your-own-8088-xt-motherboard/>

曾经有一段时间，XT 级主板——就像配备 8088 CPU 的旧 IBM PC——是一项高科技成就。现在，像这样的东西在一般的爱好实验室里很容易就能得到。[【家酿 8088】做到了](https://github.com/homebrew8088/8088-PC-Compatible)，它看起来出奇的简单，尤其是与如今的主板相比。

该板将采用 8088 或 NEC 芯片之一，默认情况下配备 512 K 的 RAM，几个 ISA 插槽，一个 PC 扬声器，一个 USB 硬盘驱动器和一个 PS/2 键盘连接器。该板将适合在 ATX 的情况下。还不错。你可以在下面看到该板的视频。

事实上，该频道有许多相关视频，主网站也有许多有趣的话题，如从树莓派驾驶 8088 或 8086 [。在](https://www.homebrew8088.com/home/raspberry-pi-second-project)[的 GitHub](https://github.com/homebrew8088/8088-PC-Compatible) 网站上有 KiCad 的设计文件以及许多其他信息。即使你只是想修理一个旧的主板或者想设计一个新的 ISA 卡，其中的一些也会很有趣。
