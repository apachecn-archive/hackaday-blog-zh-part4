# 3D 打印试验板助手使布线更整洁、更容易

> 原文：<https://hackaday.com/2020/02/23/3d-printed-breadboard-helper-makes-wiring-neater-and-easier/>

试验板使电路原型制作和测试变得简单。如果你用带针的软线来连接，通常会导致老鼠窝。对于我们中的许多人来说，使用实心电线也是一个老鼠窝。然而，我们中非常整洁的人会将实心电线切割成合适的长度，剥去适量的电线，并沿着电路板将电线非常平整整齐地放置。[moono nuration]做了一个 3D 打印，使得后一种方法更容易。你可以在 Thingiverse 上找到他的[试验板电线助手，并观看下面的视频。](https://www.thingiverse.com/thing:3862775)

这个想法很简单:从一根一端剥开的金属丝开始，然后数出它需要穿过的孔的数量，并将剥开的一端穿过这个孔。修剪电线使之合适。要完成另一侧，将金属丝沿工具平放至边缘。现在你可以看到在哪里剥去电线的那一边。移除绝缘层后，您可以向下弯曲电线并切割电线以使其适合。现在你有一个完美的尺寸和形状的电线放置在实际的试验板上。

当然，如果现有的试验板不太拥挤，这并不难做到。你甚至可以使用一个备用的试验板。但是用这种夹具将电线修剪到合适的尺寸更容易一些。如果你不想 3D 打印它，你可能会从一个便宜的板的背面撕下胶带，并移除弹簧，以获得类似的效果。

因此，虽然这个小工具可能不会改变你的生活，但它可能会让你的生活变得更轻松。试验板制作时，你还使用什么工具？在评论里让大家知道。
