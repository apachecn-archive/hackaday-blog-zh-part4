# 学习弹吉他，数码

> 原文:[https://hack aday . com/2022/03/02/数字化学习演奏吉他/](https://hackaday.com/2022/03/02/learn-to-play-guitar-digitally/)

学习演奏乐器需要投入大量的时间。不过，如果在过去的两年里，你碰巧被困在家里，你可能有机会(Dmitriy)不得不拿起一把吉他并学习演奏。然而，【Dmitriy】没有坚持使用传统吉他，而是选择[打造自己的数字吉他](https://hackaday.io/page/11876-how-i-learned-to-play-guitar-and-create-my-own-digital-one)，它拥有各种你在 Fender 或 Gibson 中找不到的功能。

这种独特的乐器的物理主体完全由[Dmitriy]用 3D 打印零件设计而成，并对原本是吉他指板的每个音符使用电容式触摸传感器。琴弦也被一组六个开关取代，可以像普通吉他一样弹奏，并用于记录何时弹奏音符。在几个原型之后，一切都被连接到一个定制的 PCB 上。这个项目的软件方面也令人印象深刻；它包括创建自定义固件来注册所有的按钮按压，并将信息传输到 MIDI 控制器，以便吉他可以与任何支持 MIDI 的设备进行数字通信。

为了完成这个项目，[Dmitriy]还添加了一个无线设备以及一些其他额外的功能，如加速度计，可以用来以他能想到的任何方式来增强吉他的声音。这是自去年 Noli 智能吉他原型亮相以来，我们见过的最具创新性的吉他之一，这款吉他也正在从[原型走向市场](https://playsensy.com/)。
