# AV Synth 是迷幻模拟混乱

> 原文：<https://hackaday.com/2018/10/24/av-synth-is-psychedelic-analog-mayhem/>

数字视频很酷，但在流畅、细腻的故障和失真方面，它无法与模拟视频竞争。[【吉斯克斯】开发了一个模拟视听合成器](http://gieskes.nl/instruments/?file=3TrinsRGB1)，这是一个很好的老派复古视觉效果的例子，你可以用一些简单的组件来制作。

它被称为 3 trisrg b+1c，有组装和套件两种形式。可能最好从[手动](http://gieskes.nl/instruments/files/3TrinsRGB1/3TrinsRGB1c-manualV1.pdf)开始。合成是通过使用 HEF40106 hex 反相缓冲器实现的，这是一种廉价且易于获得的器件，但仍能提供出色的结果。视频可以在 RGB 振荡器和一系列输入之间切换，并且有各种控制来创建那些经典的滚动效果和其他视觉怪异。

此外，到底层电路的一系列连接在头部连接器上断开。这允许插入额外的模块，并且有几种设计可以用来扩展单元的能力。

模拟视频在日常生活中已经不常使用了，但它是一项可以修补和试验的伟大技术。我们以前也在这个舞台上看过一些[吉斯克斯]的实验——比如这个 [Arduino 视频采样器。](https://hackaday.com/2011/07/15/arduino-video-sampler/)休息后的视频。
