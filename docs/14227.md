# 从字面意义上来说，乘火车

> 原文：<https://hackaday.com/2022/07/16/riding-the-rails-in-a-literal-sense/>

数百英里的铁轨散布在美国和其他国家。不管它们看起来如何，许多都没有被遗弃。但是在一条真正废弃的铁轨上，让一辆铁路自行车去探索铁轨似乎很有趣。

[凸轮工程]住在加州中部，想看看赛道上的生活是什么样的。他的系统包括一个由橡胶长板轮制成的前定位轮，两侧有定位盘。他的侧面还有一个吊杆，可以作为支架伸出来。最终，这提供了一个相当稳定的乘坐，它沿着轨道平稳滑行，没有人来平衡它。然而，前轮确实有一些问题，因为当履带穿过路面时，往往没有足够的间隙给车轮。此外，由于[接合线](https://en.wikipedia.org/wiki/Track_circuit)连接到轨道上，他已经不得不将前轮做得比需要的宽一点。整个东西折叠起来，使紧凑和时髦的乘坐。

这不是我们看到的第一辆轨道自行车，我们希望看到更多。休息后的视频。
