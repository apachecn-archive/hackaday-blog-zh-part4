# 空中足球看起来很专业

> 原文:[https://hackaday.com/2022/03/23/air-football-looks-pro/](https://hackaday.com/2022/03/23/air-football-looks-pro/)

如果你是美国人，你可能会认为塞拉斯·汉森的项目是“空中足球”，但大多数人更喜欢空中足球。无论哪种方式，它都像空气曲棍球，但更像足球场。这个项目看起来很棒——如果你在当地玩具店的货架上看到这个，你不会想到它。你可以在下面看到游戏的视频。

不出所料，这个游戏的大脑是一个 Arduino。由于激光切割和 3D 打印，外壳看起来很好。罗兰打印机生产的贴纸真的打扮的情况下，但你可以找到另一种艺术的方式来做装饰。

你可能不需要所有花哨的制作工具就能完成这项工作，但是手工钻这些气孔会很痛苦。空气来自一个 3,000 RPM 的无刷风扇，一对线跟踪器被重新用于感应冰球(呃，球)何时到达球门。触摸显示器处理记分。

总的来说，这是一个看起来很棒的项目，也是那些不使用任何高科技的东西，但看起来仍然很棒，似乎工作得很好的项目之一。

当然，我们以前见过曲棍球桌。如果你太反社会而没有对手，你可以[总是建造一个](https://hackaday.com/2016/12/15/smartphone-will-destroy-you-at-air-hockey/)。
