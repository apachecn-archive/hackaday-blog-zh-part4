# 任何房间都有天窗

> 原文:[https://hackaday.com/2020/10/04/skylight-in-any-room/](https://hackaday.com/2020/10/04/skylight-in-any-room/)

尽管有大量内向的迷因，人类还是需要阳光。如果维生素 D 不是你关心的问题，太阳是一个强大的加热器，它有助于植物生长。对哑剧演员来说，不幸的是，他们的房子没有定位好来捕捉所有这些美味的阳光。对我们来说幸运的是，他们进入 2020 年 Hackaday 奖的是他们的[太阳跟踪装置](https://hackaday.io/project/5142-reflecting-the-sun-into-your-home)，它将这些强大的光线重定向到整个房子。它使用几面镜子来改变小屋周围的光线方向，使其进入室内。对于那些在昏暗的办公室工作的人来说，为了一瞥自然的阳光，再多的工作也不为过。

**电影剧透预警:**我们在 1985 年的电影《T2》*《传奇》* 中见过这一招，足以击败黑暗之王。

这个项目始于 2014 年，中断了五年多，但它回来了，是五年经验推动的改进的主要因素。不太可能改变的部分是调节定位镜角度的螺纹支柱、驱动电机和电源电路。他们的第一个计划是用 Arduino、DC 电机和太阳遥测数据建造一个太阳能控制器，但现在他们倾向于用一根长电缆在房子里安装步进电机和计算机。他们是今年的决赛选手，所以我们会密切关注他们的进一步发展。
