# 小光枪可以:辛登做好

> 原文:[https://hack aday . com/2022/06/22/the-little-light gun-that-can-sinden-make-good/](https://hackaday.com/2022/06/22/the-little-lightgun-that-could-sinden-makes-good/)

早在 2018 年，我们报道了(安德鲁·辛登)正在做的工作，即创造一种可以在现代电视上工作的光枪。该项目正在通过 Kickstarter 寻求资金，但至少部分由于对所涉及技术的怀疑，该活动远未达到其目标。当时，故事似乎就此结束了。

但是我们最近看到了几个月前刊登在*《卫报》* 上的[对【安德鲁】的精彩采访，它不仅讲述了故事的其余部分，而且以一个快乐的结局结束——经过多年的努力，](https://www.theguardian.com/games/2022/mar/07/duck-hunt-light-gun)[辛登光枪现在可以购买了](https://www.sindenlightgun.com/)。这并不完全是一些人喜欢的交钥匙产品，因为在*猎鸭*中为了抓住一些同名的水鸟，人们必须经历相当多的关卡，但没有什么会吓走普通的黑客读者。

[![](../Images/7cd20e7f4e432cf6ced5234a140e98f3.png)](https://hackaday.com/wp-content/uploads/2022/06/sinden_detail2.jpg)

Limited technical details about the 2018 prototype may have kept backers away.

硬件的最终版本抛弃了从 Wii 枪支配件继承的现实主义枪支美学，现在具有一个色彩鲜艳的手枪外壳，看起来不会与连接到一台 *Virtua Cop* 机器的地方不相称。它还获得了一个可选的力反馈反冲螺线管，尽管它在基础型号已经高达 100 美元的价格标签上又增加了 60 美元。

我们很高兴地看到[Andrew]认识到了[获得 Linux 对软件方面的支持的重要性，因为它支持为 Raspberry Pi 4 开发预配置的 Retropie 映像。虽然您不需要在 Pi 上进行模拟，但是对于那些希望在桌面上偶尔消灭僵尸的人来说，Windows 和 x86 Linux 也是受支持的。](https://www.sindenlightgun.com/details/)

很多时候，当我们在 Hackaday 上报道一个项目时，这是一次性交易:有人有特殊的需求或愿望，制造了一个小工具来满足它，然后继续前进。这没什么不好，但是当我们看到这个社区的项目发展成更多的东西时，我们会有一种自豪感。虽然并不是我们展示的每一个拼凑起来的硬件[都有可能成为下一个 Arduboy](https://hackaday.com/2014/03/01/the-credit-card-sized-gameboy/) 或 [Sinden Lightgun](https://hackaday.com/2018/06/28/a-lightgun-for-lcds-thanks-to-maths/) ，但我们喜欢认为我们已经报道了下一个重大的*项目转化为产品*的成功故事，只是我们还不知道而已。
