# 在 VIC-20 里面

> 原文:[https://hackaday.com/2021/05/19/inside-the-vic-20/](https://hackaday.com/2021/05/19/inside-the-vic-20/)

Commodore 机器在这里很受欢迎，但通常当您想到 Commodore 时，您会想到 Commodore 64，或者 PET 或 Amiga。但 64 号准将有个姐姐叫 VIC 20。这是第一台销售 100 万台的电脑，与其更知名的继任者有许多共同之处。这台机器才制造了几年，[可疑工程]一直在恢复一个超过几个视频。在下面的视频中，他[打开它看了看里面的](https://www.youtube.com/watch?v=2QAL2aiDrAA)，还有其他东西。

如果你想直接进入开场，你需要快进大约 5 分半钟。键盘拔出来，一套由单独的电线制成的漂亮的老式电缆连接到超薄的主板上，上面有所有的智能。没有带状电缆或柔性印刷电路板！

看到 20 世纪 80 年代的设计选择很有趣。例如，caps lock 键是一个实际的闩锁键，它以机械方式保持其位置。有两个 ROM 芯片，一个用于系统软件，一个用于基本。两个 RAM 芯片给你一个巨大的 5K 读/写内存。

当时 300 美元的电脑和今天 800 美元的电脑差不多。你仍然需要一台显示器或一台电视机和一台盒式录音机。与此同时，内存可以有所扩展，但显示屏停留在 176×184 和你选择的 8 种颜色。

VIC20 可能不是当时最有影响力的电脑。但这显然是从宠物到非常有影响力的准将 64 的垫脚石。很高兴看到一只又跑了。

这个视频里的机器看起来比我们上次看到的[还原的](https://hackaday.com/2018/05/25/bringing-a-vic-20-back-from-an-oily-grave/)好多了。我们自己的【比尔牧群】是准将故事的一部分，他出现在[的同名纪录片](https://hackaday.com/2018/02/23/the-commodore-story-premieres-february-23rd/)中，非常值得一看。
