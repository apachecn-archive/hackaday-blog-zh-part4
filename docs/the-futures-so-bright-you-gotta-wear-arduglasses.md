# 未来是如此的光明，你必须戴上硬质眼镜

> 原文：<https://hackaday.com/2021/03/25/the-futures-so-bright-you-gotta-wear-arduglasses/>

微型有机发光二极管显示器是现代零件箱中的绝对必需品，那么还有什么比一副兼容 Arduino 的有机发光二极管眼镜更好的方式来表明你对创客运动的忠诚呢？由 Arduboy 策划人[Kevin Bates]创造的这些数字眼镜可能不会帮助你看得更清楚——事实上，你会看得更糟——但它们肯定会让你在下一次黑客大会上鹤立鸡群。)

这个项目的关键是一对透明的 CrystalFonts 有机发光二极管显示器，就像最近肖恩·哈金斯用来制作他华丽的体积显示器的那对一样。事实上，[凯文]说，正是这些展示的成功激发了他追求自己的项目。通过一些巧妙的 PCB 设计，他想出了一些可以由 OSH Park 制造的电路板，并用首饰盒铰链组装在一起。同样来自 OSH Park 的小型柔性电路将电路板连接起来，并允许框架在不佩戴时折叠起来。

Arduglasses 使用与 Arduboy 相同的 ATmega32U4 微控制器，通过几个基本的控制和一个 100 mAh 的小充电电池，他们可以在技术上运行来自开源手持设备的广泛软件库的任何东西。当然，*技术上来说*是这里的关键词。虽然硬件能够玩游戏，但[Kevin]报告说，有机发光二极管的显示器离佩戴者的眼睛太近，实际上无法集中注意力。也就是说，为这些眼镜轻松创建软件的能力为迷因提供了大量机会，正如我们在下面的视频中看到的那样。

出于可能显而易见的原因，[Kevin]认为 Arduglasses 是一项实验，并不打算将其转化为商业产品或套件。但是如果有人感兴趣，他愿意把设计文件放在 GitHub 上，给任何想给自己的赛博朋克衣橱添加一副 Arduino 眼镜的人。
