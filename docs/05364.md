# 连接在电极上的不断生长的人类神经元

> 原文：<https://hackaday.com/2020/02/02/growing-human-neurons-hooked-up-to-electrodes/>

哲学家们长期以来一直在思考“罐子里的大脑”的概念，但迄今为止，这仍然是科幻小说的保留地，而不是现实。然而，在阅读了一些科学论文后，[Justin]想亲自尝试这一壮举，[于是着手在电极阵列上培养一些人类神经元。](https://www.youtube.com/watch?v=V2YDApNRK3g)

该项目建立在[Justin]早期工作的基础上，使用他的 DC 溅射设备在玻璃显微镜载玻片上涂上电极。第一层是银，具有高导电性，并添加了金层，具有生物相容性。然后用环氧树脂将 Falcon 管的螺帽粘合起来，作为神经元培养基的容器。最后，添加空气过滤器以允许生物混合物呼吸。

这是[贾斯汀]第一次尝试培养神经元，一路上有很多障碍。定制培养组件存在环氧树脂粘合剂泄漏或完全失效的问题，导致只有一个载玻片通过了灭菌过程。此外，意外添加的神经元数量过多。虽然在显微镜下观察到一些生长，但[Justin]无法从系统中检测到任何真实的信号。

尽管最终结果不佳，但一路上还是学到了很多东西。[Justin]已经制定了计划来修复最初实验中的一些缺陷，我们期待看到该项目未来的更新。休息后的视频。
