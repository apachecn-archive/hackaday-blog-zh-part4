# 小小变成小小的手持电脑，玩模拟器

> 原文：<https://hackaday.com/2022/09/29/teensy-becomes-tiny-handheld-computer-plays-emulators/>

科幻小说预言，有一天我们都会随身携带功能强大的微型计算机。虽然智能手机很棒，但这些预测更多的是基于更可爱的系统，更接近现有的电脑，有键盘和屏幕。让-马克·哈文格特已经按照这些思路制造了一些东西，它被称为 T 型计算机。

这个版本以强大的 Teensy 4.1 为中心。这意味着它有一个 800 MHz 的 Cortex-M7 处理器，1 MB 的内存和 8 MB 的闪存-使过去许多复古计算机的规格黯然失色。[Jean-MarcHarvengt]将 Teensy 与 42 键键盘和 TFT 屏幕配对，制成了一个紧凑的手持计算机平台。它还有 VGA，可以在更大的屏幕上显示，还有 USB 和一个老派的 Atari 操纵杆端口！电源通过背面的小型可充电锂电池提供，16 位立体声音频通过标准的 3.5 毫米插孔提供。如果你需要与某些东西接口，还有一个小的 GPIO 可用。

它能够模仿 Commodore 64 和超级任天堂，以及 Atari Lynx 等更不知名的系统。在你问之前，是的，它可以运行*毁灭。* 这是一个有趣的小平台，可以在旅途中玩复古游戏和黑客。如果你想构建自己的系统，Github 上的[文件可以很容易地用来重建系统。](https://github.com/Jean-MarcHarvengt/T-COMPUTER)

手持电脑越来越受欢迎，因为一个小小的开发板就能拥有如此强大的计算能力。如果你已经建造了你自己的整洁的小装备，[一定要让我们知道](http://hackaday.com/submit-a-tip)！休息后的视频。
