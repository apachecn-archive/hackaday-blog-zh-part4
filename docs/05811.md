# 夹持器使用皮带夹紧和抓取

> 原文:[https://hack aday . com/2020/02/27/夹持器-使用-带-夹-抓/](https://hackaday.com/2020/02/27/gripper-uses-belts-to-pinch-and-grasp/)

尽管自机器人技术诞生以来已经做了很多工作，但在灵活性和适应性方面，仍然没有人的手可以与之匹敌。Korea tech IRIM 实验室的研究人员凭借其巧妙的 [BLT 夹钳](https://www.youtube.com/watch?v=_erUF2E8oJQ)又向前迈进了一步，这种夹钳可以精确地夹紧或以均匀分布的力抓住更大的物体。(视频嵌入下方。)![](../Images/1676d60e5632188b057b6a156d27dfdf.png)

三指手爪在技术上被称为“带和连杆驱动的可变形自适应手爪，具有主动转换能力”。每个手指都是刚性“指尖”和驱动链接的有趣组合，以及作为抓握表面的皮带。驱动连杆在其底部有一个小齿轮箱，用于打开和关闭手，带有“指尖”的铰链由弹簧加载到打开位置。柔性带在指尖和夹持器的基部之间伸展，其可以被拉紧以驱动指尖进行夹紧，或者在夹持器的内部提供均匀的力以进行抓取。两个手指还可以在基座上旋转，以提供各种夹持器配置。这使得夹持器可以以多种方式使用，包括在不掉落物体的情况下在夹紧和抓取之间平稳切换。

我们喜欢这种相对简单的机制，并且可以看到它被用于通用机器人和假手，特别是如果集成了力感测的话。使用 3D 打印组件、一条齿形带和两个廉价的伺服系统，该机制应该很容易复制，所以开始破解吧！

夹子有各种形状和大小。[软夹子](https://hackaday.com/2019/08/14/jessica-is-soft-on-robot-grippers/)近年来开始流行，[折纸](https://hackaday.com/2019/03/18/origami-gripper-is-great-for-soft-and-heavy-objects/)也可以是一种选择。
