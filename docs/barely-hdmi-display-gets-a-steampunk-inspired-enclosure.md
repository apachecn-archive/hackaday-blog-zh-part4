# HDMI 显示屏几乎没有蒸汽朋克风格的外壳

> 原文：<https://hackaday.com/2022/05/13/barely-hdmi-display-gets-a-steampunk-inspired-enclosure/>

这是一个有趣的问题:建造世界上最差的 HDMI 显示器的后续工作是什么？简单——把它放在一个很酷的蒸汽朋克风格的盒子里，然后收工。

这似乎是[mitxela]的解决方案，请不要将我们的评估视为对最初版本或后续版本[的抨击。[mitxela]自己对他的“相当愚蠢的项目”获得的关注表示有点惊讶，该项目在 HDMI 接口中使用 I2C 接口来驱动一个微小的单色有机发光二极管屏幕。低刷新率，低分辨率——它拥有你不想在显示器上看到的一切，但仍然是一个值得关注的酷黑客。](https://hackaday.com/2022/04/01/making-your-own-technically-hdmi-oled-monitor/)

正如下面的视频所揭示的那样，目前的工作是为狡猾的显示器创建一个外壳，金属加工远比其他任何东西都重要。显示器本身放在一个由黄铜加工而成的小盒子里，而 HDMI 插头则有一个看起来很坚固的黄铜外壳，使更常见的模制塑料插头看起来不可原谅地脆弱——尽管有热熔胶。连接两者的是一个柔性杆，允许它插入计算机的 HDMI 端口，并让用户可以灵活地将几乎无用的显示器放置在可以看到的最佳位置。

但同样，我们的判断可能过于苛刻；而 *DOOM* 基本上无法在微型显示器上播放，“坏苹果！!"相当具有观赏性，尤其是伴随着[mitxela]的[伺服控制的 MIDI 音乐盒](https://hackaday.com/2018/03/23/servos-do-the-plucking-in-this-midi-music-box/)。从什么时候开始，可用性成为评判一个黑客酷不酷的标准了？
