# 如果你能设计自己的铝制手会怎么样？

> 原文：<https://hackaday.com/2020/10/22/what-if-you-could-design-your-own-aluminum-hand/>

[伊恩·戴维斯]决定在他的手上重新开始。【伊恩】左手缺了四根手指，一年来[一直在他的 YouTube 频道上展示 DIY 假肢](https://www.youtube.com/channel/UCv1xxFkEiAdCVy6foEEUIvw/videos)。早在七月份，我们就报道过【伊恩】的[铝手](https://hackaday.com/2020/07/05/hackaday-links-july-5-2020/)。

为什么是铝？[Ian]发现自己多次重印以前版本的 3D 打印塑料零件，因为铰链接头损坏，或紫外线损坏使它们变脆。凭借一个巧妙的张开机制和一些由 Arduino 驱动的传感器，[Ian]每天都戴着定制的加工铝手。

然而，和许多制作人一样，他渴望重新审视和完善这个项目。即使上一个版本是生活质量的一大飞跃，他仍然找到了改进的空间。一个特别的问题是传感器容易移动，很难得到准确的读数。为了克服这一点，[伊恩]转向了成型工艺。然而，添加稳定硅层意味着假体的设计需要改变。考虑到一些改进，[Ian]开始制作他手掌的石膏正片，努力制作硅负片。从这里开始的下一步是制造一个玻璃纤维外壳，它可以通过嵌入玻璃纤维外壳中的传感器线来覆盖硅树脂。

看到过去几年里 3D 打印假肢的爆炸令人惊讶，希望这一趋势继续下去。我们期待看到[Ian]迈向理想义肢的下一步！
