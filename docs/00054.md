# 从微波爆米花中获取能量

> 原文：<https://hackaday.com/2018/07/15/harvesting-power-from-microwave-popcorn/>

今年 Hackaday 奖的挑战之一是[能量收集](https://hackaday.io/prize/details#three)我们要求每个人创造一个从某物中收集能量的东西。可能是太阳能，也可能是从下落的重物中获取能量。如果你想做一个 TED 演讲，可以从声波中获取能量。它可以从环境射频中获取能量，哪里是获取环境射频的最佳位置？[没错，旁边有个微波炉](https://hackaday.io/project/159319-mwessenger)。

[法学家]今年 Hackaday 奖的能量收集挑战的参赛作品是一个安装在微波炉前门的简单装置。该设计使用一个简单的 PCB 天线来收集能量，一个 LTC3108 DC/DC 转换器被放在一个垃圾抽屉里，一堆无源器件用来吸收一些从微波炉中逃逸的光子。这整个装置的想法是当微波炉完成时，利用收集的能量通过蓝牙(或其他什么)发送信息。不过，真的，这属于“因为我能”这一类奇怪的构建。

那么，这个能量收集 PCB 有用吗？最初的测试是不确定的，因为没有调整天线和电路。然而，在[法学家]将电路板连接到电压表上并煮了一些豆子后，他看到电路上有整整一伏的电压。这是一个开始，一个真正“智能”的微波炉附件的开始。不过，真的，看到一个电路从一个泄漏的法拉第笼中获取能量很酷。
