# Monowheel Mayhem:当良好的陀螺进动变坏

> 原文：<https://hackaday.com/2022/01/19/monowheel-mayhem-when-good-gyroscopic-precession-goes-bad/>

自从汽车时代开始以来，汽车迷们就一直痴迷于使用尽可能少的车轮。不满足于被摩托车不完全致残的前景，独轮车诞生了。这些未来的毁灭之轮登上了流行杂志和其他期刊的封面，让各种黑客目瞪口呆。[詹姆斯·布鲁顿]就是这样一个黑客，在休息时间下面的视频中，你可以看到他的第二个 3d 打印单轮机器人。

[詹姆斯]“奇妙的独轮车设计精美。轴承表面，齿轮，惰轮，电机，是的，双陀螺仪都包含在轮胎的圆周内。陀螺仪由一个相当大的伺服系统驱动，并由一个保持位置同步的齿轮连接在一起。他们的工作是始终保持单轮车的平衡。

但是正如[詹姆斯]发现的，只有一个轮子的主要困难不是横向平衡。问任何一个单核细胞增多症患者，他们都会向你保证这是可能的。真正的诀窍是使机器前后平衡。与两轮自行车不同，除了一点重力，单轮自行车没有什么可以施加扭矩。

正如[詹姆斯]艰难地发现的那样，正是在这种前后平衡的行为中，陀螺进动露出了丑陋的头。这个概念在视频中解释得很好。我们不会破坏惊喜的结局，因为解释和结论都很好，所以一定要看完！

如果你想看看[詹姆斯]的第一个版本，我们在这里讨论了它。如果你是不怕死的类型，也许我们可以对你在[两冲程人类大小的单轮车感兴趣，这可能会在急诊室结束访问](https://hackaday.com/2019/01/16/this-monowheel-is-bright-orange-and-we-want-one/)。至少他们戴了头盔。感谢[Baldpower]的提示！
