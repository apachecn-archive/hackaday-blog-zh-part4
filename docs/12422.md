# DE10-Lite-Ful FPGA 开发板 Hack 播放 1981 年经典后卫

> 原文：<https://hackaday.com/2022/01/08/de10-lite-ful-fpga-dev-board-hack-plays-the-1981-classic-defender/>

我们不确定作业是什么，但是[Garret Carter]在田纳西理工大学的数字系统设计课的作业结果不禁引起了我们的注意。在广告下方，你可以看到[加勒特]所描述的 1981 年街机热门游戏“防御者”的“简化风格化版本”。

为了保持低价格但尽可能保持高性能，[Garrett]开始用 VHDL 对 DE10-Lite FPGA 开发板进行编程。结果令人信服，虽然不完美，但在预算之内。

DE10-Lite 板给了[Garrett]发挥更大创造力的机会，充分利用了开发板的板载开关、按钮、7 段 LED 和*加速计*。在这种情况下，dev 板不仅是游戏，也是控制器和状态显示器。确实是一次非常巧妙的黑客攻击！

如果你想自己制作，你可以在[【Garrett】的 Github 页面](https://github.com/w3arycod3r/fpga-defender)获得完整的项目细节。加勒特，我们不知道你的教授会给你多少分，但我们给你一个 A+，肯定会再玩一次。

虽然 FPGA 开发板不一定便宜，但我们自己的 Jenny List 显示，如果你知道去哪里找，你可能能够[找到一个二手但可用的 FPGA 板，只要花一小部分钱。](https://hackaday.com/2020/12/10/a-xilinx-zynq-linux-fpga-board-for-under-20-the-windfall-of-decommissioned-crypto-mining/)
