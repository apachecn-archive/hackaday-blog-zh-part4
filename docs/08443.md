# 你好，全息图

> 原文：<https://hackaday.com/2020/12/24/hello-holograms/>

全息图很难描述，因为科幻小说给任何三维图像都起了这个名字。基于科学事实的版本没有那么花哨，但仍然很酷。合法的全息图是存储在摄影介质上的图像，它们保留了从某些角度拍摄的对象的照片。换句话说，当[【贾斯汀·阿特金】制作一个模型建筑的全息图时，](https://www.youtube.com/watch?v=aTB2ryoWIFU)(视频，嵌入下方)你可以看到钟楼的东侧，但当你重新定向时，你看到的是西侧，或者是屋顶，如果你指向下方。全息术不同于体视术，体视术使用两个摄像头向您显示 3D 图像。对于立体图像，你不能倾斜它并看到主体的新部分，所以每种方法都有合适的位置。

在家制作全息图有几种不同的方法。首先，你可能需要一个 DIY 全息工具包，因为它会附带曝光板和一个已知良好的光源。我们不可能告诉你，你买不到盘子和激光笔来走更少人走过的路。接下来，你需要一个不会动的东西，所以我们担心你不能让你的小猫咪永生。最后一个必要条件是一个稳定的平台，因为你将执行长曝光拍摄，甚至在设置上呼吸都会破坏图像。不同的颜色来自相干光源，因此获得视频中广告的“彩虹全息图”是一个混合光的问题。因为你可以花很少的钱买到红色、绿色和蓝色的激光笔，所以你可以对你的内容进行颜色混合。

另一种类型的全息图出现在交易卡等物品上，是超级英雄或抽象形状的极度变色(彩色，而不是令人讨厌)的图像。它们是不同的品种，可以一起印刷，不像贾斯汀给我们展示的那种一次性的。

如果你渴望立体显示器，我们很高兴向你介绍这种能够展示令人瞠目结舌的 3D 模型的美[或这种全彩色块状鸭子](https://hackaday.com/2019/10/25/giant-leds-ruby-lasers-hologram-displays-and-other-cool-stuff-seen-at-maker-faire-rome/)[。](https://hackaday.com/2020/09/12/3d-pov-display-has-the-shakes/)
