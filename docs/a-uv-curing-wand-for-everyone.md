# 每个人都有一个紫外线固化棒

> 原文：<https://hackaday.com/2020/12/06/a-uv-curing-wand-for-everyone/>

普通人使用紫外线棒的经历是在牙医那里得到一个洞。然而，任何拥有树脂基 3d 打印机的人都知道紫外线固化系统有多重要。通常印刷品上的一些斑点需要一点额外的紫外线来加强。[Mile]已经着手[创造一种名为 Photon 的开源紫外线固化棒，这种棒具有成本效益并且易于制造](https://hackaday.io/project/174558-photon)。

有趣的是，从 5 美元的 LED 手电筒到更大的工业泛光系统，即使没有上百个紫外线固化系统，也有几十个。[Mile]直入主题，展示了那些更便宜的模块正在进行的权衡，以及商业系统正在做而他没有做的事情。[Mile's] Photon wand 试图在保持较低成本的同时，提高辐射功率的能效。这是通过仔细选择 CSP LEDs 而不是传统的引线键合，并确保光源正确聚焦和冷却来实现的。从干净的 PCB 和光滑的外壳来看，很明显[Mile]已经采取了额外的措施来使这款产品更有利于生产。由于树脂固化有两种行业标准波长(364 纳米和 405 纳米)，因此 Photon 中的 LED 模块可由用户更换。

我们喜欢这个项目，是因为它忽略了现成的东西，而是深入研究。首先了解现有事物的缺点和局限性，然后设定一个目标，朝着不同的方向努力。这不是我们最近看到的第一个紫外线固化工具，所以显然需要更好的东西，这就是今天的产品。
