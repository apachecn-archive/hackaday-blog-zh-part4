# 自制锯停止附件只是粗略的，因为它的声音

> 原文：<https://hackaday.com/2022/08/02/homemade-sawstop-attachment-is-just-about-as-sketchy-as-it-sounds/>

TL；当[柯林·福尔泽]是你的“安全检查员”时，你真的应该重新考虑你的项目目标。

我们大多数人现在可能都见过 SawStop 品牌的自停式台锯，当比木头更肉的东西有异味接触到旋转的刀片时，它会检测到，或多或少地立即停止它，并防止突然的创伤性截肢。这是一个杰出的想法，我们希望看到所有台锯内置的技术。但是，唉，锯子的价格让许多木工难以承受，这让露丝·阿莫斯只能自己动手制作这个系统。

应该马上声明，[Ruth]在这里所做的一切都不是一个好主意，所有展示的东西都只是概念证明。她建造的基础是一个看起来有些脆弱的承包商风格的锯子，[露丝]在锯子上安装了一个 Arduino 装置，用于检测导电的东西何时接触到锯条。她没有分享感测方法的细节，但我们猜测是电容耦合。然后，她开始试验一系列桌子上的小发明来阻止锯条，但效果有限，加上所有的附件会使锯子基本上无用。但是在原型阶段，在桌子上工作确实是有意义的，并且允许她找出什么是不可行的。

最终，电动剪草机的电磁离合器似乎做到了这一点，尽管代价是锯子的沉重负担和系统角动量的显著增加。尽管如此，在旧的热狗测试中，刀片几乎立即停止。当然，它不会将刀片掉到桌子下面，热狗的穿着也有点糟糕，但仍然令人印象深刻。

如果你感兴趣的话，我们之前已经讨论过 SawStop 的技术以及为什么它没有得到应有的广泛应用。
