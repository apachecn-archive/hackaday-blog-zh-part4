# 移过去，斯特雷德来了！

> 原文：<https://hackaday.com/2018/11/23/move-over-strandbeest-heres-strider/>

父子团队[Wade]和[Ben Vagle]已经开发并广泛测试了两种伟大的步行机设计: [TrotBot](https://www.diywalkers.com/trotbot-ver-3.html) 和全新的 [Strider](https://www.diywalkers.com/strider-linkage-plans.html) 。但这还不够:他们的网站详细介绍了他们在模拟和建造这些小动物方面来之不易的实践经验，规模从乐高技术到[车库填充](https://www.youtube.com/watch?time_continue=1&v=0QJjtfX0-OQ) (YouTube，嵌入下方)。他们的[沃克 ABC 的页面](https://www.diywalkers.com/walker-abcs.html)就充满了对这个问题极其深刻的见解，是必读的。

这些机构被设计得比詹森联动装置更简单，比科兰 T4 更平稳。特别是，当他们不在海滩上散步时，步行者的脚经常需要清除障碍，而[Vagles']的设计将脚趾[抬得比其他设计](https://www.diywalkers.com/blog/walking-tanks-prof-shigleys-pioneering-study-of-mechanical-walkers)更高，同时还保持重心以恒定的速度移动，不需要脚打滑或撞到地面。他们做了一些聪明的事情，比如给机器人添加脚趾以平衡它们的步态，甚至提供了 Python 中的[模拟器](https://www.diywalkers.com/python-linkage-simulator.html)和 Scratch 中的[模拟器](https://www.diywalkers.com/scratch-linkage-simulator.html)，这将帮助你改进自己的设计。

如果你想要一个简单移动的机器人，你会用轮子。我们喜欢步行者，因为他们看起来很棒。当我们写道[韦德]说 Trotbot 的一种步态看起来像动物时，他指出 TrotBot 的工作名称[来自一种马式步态](https://www.youtube.com/watch?v=XpmqqpXvrIU) (YouTube)。与 TrotBot 相比，Strider 家族没有太多的个性，但他们跑得更平稳、更快、更强壮。已经有一款 [3D 打印友好型机器人模型](https://www.diywalkers.com/3d-printed-trotbot-by-scott-anderson.html)问世。谁来为 Strider 做点什么？

我们有多爱机械步行者？足以张贴关于[用詹森联动装置](https://hackaday.com/2016/06/30/a-strandbeest-bicycle/)制造的自行车，遥控玩具 Strandbeests 既有[带武器](https://hackaday.com/2018/06/29/r-c-rocket-beest-burns-up-fuses-out-there-alone/)又有[不带](https://hackaday.com/2016/02/03/motorized-strandbeest-is-remote-controlled-and-awesome/)，[电钻驱动的步行滑板车](https://hackaday.com/2015/07/10/drill-powered-scooter-walks-with-legs/)，当然[基本上是西奥·詹森做的任何事情](https://hackaday.com/2018/08/16/theo-jansen-invents-a-faster-simpler-wind-powered-strandbeest/)。

如果访问[Wade]和[Ben]的网站没有让你在 walker 项目上工作，不管是实体的还是虚拟的，我们不知道什么会。
