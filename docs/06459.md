# 定制固件使劳拉启用 HAB 跟踪手表

> 原文:[https://hack aday . com/2020/05/06/custom-firmware-makes-a-lora-enabled-hab-tracker-watch/](https://hackaday.com/2020/05/06/custom-firmware-makes-a-lora-enabled-hab-tracker-watch/)

[![](../Images/a84fcb6ab8f092d2575f8389b6428209.png)](https://hackaday.com/wp-content/uploads/2020/04/HAB-Watch-logo.png) 高空气球(HAB)是获取各种数据和拍摄精彩照片和视频的绝佳方式，但飞上去的东西必须下来。一旦设备着陆，人们必须跟踪它。GPS 和 LoRA 具有很长的无线范围和易用性，是跟踪返回地球的有效载荷的无价工具。[戴夫·阿克曼]已经制造了手持接收器来引导他到达有效载荷，但是想要更小的东西；理想的是可以戴在手腕上的东西。

有一天，他偶然发现了价格实惠的 LilyGo T-Watch，其中包括 GPS 和 LoRA 功能，他开始有了想法。手表有功能，但股票固件不符合标准。【戴夫】[没有被吓住，他写了新的固件，把这个设备变成了一个戴在手腕上的 GPS 和 LoRA chase 手表](http://www.daveakerman.com/?p=2683)。

新固件不仅功能强大，而且用户界面也很棒。[新固件的 GitHub 库在这里](https://github.com/daveake/S7xG)，你可以在下面嵌入的简短视频中看到运行中的 UI。
