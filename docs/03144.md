# 创造最佳环氧花岗岩的实验

> 原文：<https://hackaday.com/2019/06/01/experiments-in-creating-the-best-epoxy-granite/>

如果你想要一台大型数控机床，你需要一个坚固、抗振动的底座。他们用金属制造铃铛，所以如果你想要不会摇晃的东西，那可能不是最好的。环氧花岗岩是你最好的选择，但是什么环氧花岗岩是最好的呢？这就是[Adam Bender] [在一系列实验](https://www.adambender.info/post/2017/03/25/epoxy-granite-machine-frame-how-to)中回答的问题，这些实验产生了一台漂亮的数控机床。

虽然这是一个项目，导致了一个完整的数控机床基地，这也是一个实验，以确定最佳配方，创造自己的环氧花岗岩。实验的目的是有效地确定最好看的环氧花岗岩，并在该复合材料的组成中使用四个变量。游戏沙、砾石、染料(氧化铁和液态环氧树脂染料形式)和双组分环氧树脂用于创建七个不同的样本。使用岩石的样品没有那么好，仍然有截留的空气。即使将环氧树脂放在真空室中脱气也是如此。获胜的组合原来是 80%的沙子和 20%的环氧树脂以及一点黑色染料的混合物，在 DIY 摇床上振动 30 分钟。

有了环氧花岗岩的正确配方，[亚当]安装好他的模具，给所有东西都涂上蜡。内部骨架，或者数控机床将被螺栓固定的部分，被组装在模具内，环氧树脂被灌注进去。其结果是不可思议的，也是一台将金属转化为芯片的机器的绝佳基础。你可以看看下面的视频。
