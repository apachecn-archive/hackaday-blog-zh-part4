# 头脑风暴

> 原文：<https://hackaday.com/2022/11/19/brainstorming/>

和其他黑客一起玩的最好的事情之一就是自由的头脑风暴会议。举个例子:我参加了电子产品交易会，最后和[Stephen Hawes]和[Lucian Chapar]在一起，他们是 LumenPnP 开源取放机的幕后推手，我们过去已经讨论过很多次了。

在许多很酷的功能中，它有一个安装在零件移动头上的摄像头，可以找到 PCB 上的基准标记。但是当然，这意味着摄像机安装在一个几乎通用的双轴机架上，这让极客们神魂颠倒。[Stephen]谈到了将它变成一个照片拼接微距摄影设备是多么容易，它可以产生令人惊讶的高分辨率照片。

与此同时，[卢西恩]和我在想这个机架与 3D 打印机有多相似，[卢西恩]问为什么 3D 打印机没有在热端安装摄像头。他甚至在东海岸说唱节上兜售这个想法，并让一些人对此感到兴奋。

所以这里的想法是:挤出机附近的计算机视觉给你实时过程控制。你可以用它来在 z 轴上归位喷嘴，你可以用它来判断灯丝何时用完，或者步进器何时跳过了步进。如果你真的对它进行了改进，你可以用它来弥补其他印刷缺陷。简而言之，这将是一个简单的硬件添加，将开启一个计算机视觉软件改进的宇宙，最重要的是，它对家庭游戏玩家来说足够简单——你可能只需要一台 3D 打印机。

现在我已经和你分享了我的灵感。希望它能激发一些 DIY 3DP 创新，或者至少鼓励你在下面进行头脑风暴。
