# 迷笛老鼠制造美妙的音乐

> 原文：<https://hackaday.com/2021/07/18/midi-mouse-makes-marvelous-music/>

这是一种古老的误解，认为数字音乐家只是使用鼠标和键盘进行艺术创作。这通常与事实相去甚远，因为许多电脑音乐艺术家在创作过程中使用了各种各样的键盘/合成器、MIDI 控制器和“模拟”乐器。但是如果其中一个工具只是一只老鼠呢？

嗯，当[kzra]把一个旧的 ps/2 滚球鼠标变成一个电子仪器时，他一定是这么想的。出于对音乐的热爱和对浪费的厌恶，鼠标是一个全功能的 MIDI 控制器。音符音高映射到指针的 x 坐标，音量(在 MIDI-speak 中称为力度)映射到 y 坐标。滚轮可以用作调制轮，用户可以配置，但最常用于改变音符的音高。鼠标按钮用于演奏音符，根据乐器设定的模式，其行为会略有不同。

不满足于简单地输出 MIDI 音符，[kzra]还设计了一个直观的用户界面来配合鼠标。一个漂亮的小有机发光二极管显示模式、音量、音符和鼠标坐标，一个 8×8 的 LED 矩阵也显示音符和音量。这是一个奇妙而多才多艺的小乐器，休息之后你一定要看看视频，亲自看看。在之前，我们已经看到了一些[令人敬畏的复古技术 MIDI 控制器，这非常适合。](https://hackaday.com/2021/06/19/turning-gamecube-n64-pads-into-midi-controllers/)
