# 电子 Covid 测试拆除显示令人沮丧的一次性使用浪费的例子

> 原文:[https://hacka day . com/2021/10/17/electronic-covid-test-tear-down-shows-floating-example-of-1-time-use-waste/](https://hackaday.com/2021/10/17/electronic-covid-test-tear-down-shows-frustrating-example-of-1-time-use-waste/)

来自[TheSignalPath]的最新视频是他购买家用新冠肺炎测试的结果。他发现了一个电子版本，可以连接到你的手机，并在手机上显示结果。该设备是一种抗原测试，在内部，像家庭测试一样工作，使用类似于妊娠测试的线路显示结果。因此，不知何故，手机版读取线路并与手机通信。但是怎么做呢？这就是视频的重点，你可以在下面看到。

在传统的测试中，必须出现一条控制线来表明测试已经正确完成。下面的一条线表示检测到了病毒。电子测试内部的电路板上有一个塑料单元，它包含一个类似的条带，并具有用于参考线和检测线的光学传感器。因为它本质上是一种光学设备——在带组件中有一些透镜，看起来像是在检测染料，因为它通过板上的 led 移动，从而照亮这种情况。

显微镜下，CPU 是典型的北欧产的支持蓝牙的 ARM 芯片。电路板确实通电了，但由于测试条的原因，设备只能运行一次。视频指出——我们也同意——创建一个完整的带光学组件的支持蓝牙的微控制器板，只是为了读取一次本来就很容易读取的纸条，这似乎是一种浪费。我们将坚持使用简单的测试条。尽管如此，看到内部还是很有趣的。

如果你想了解更多关于抗原测试的信息，我们已经介绍过了。我们还谈到了 [PCR 检测](https://hackaday.com/2020/03/24/coronavirus-testing-just-the-facts/)。
