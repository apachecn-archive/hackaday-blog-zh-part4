# 透镜状透镜使事物隐形

> 原文:[https://hack aday . com/2021/01/25/lenticular-lens-makes-things-invisible/](https://hackaday.com/2021/01/25/lenticular-lens-makes-things-invisible/)

当然，这是一个廉价的舞台技巧，但在正确的角度和正确的背景前面使用双凸透镜可以使后面的东西看不见。这不是新闻，但[Ian]花了一些时间研究如何尽可能做出最好的一个。他的指导包括如何用聚碳酸酯、合适的镜头和一些光学透明粘合剂来制作自己的相机。你可以在下面的视频中看到关于盾牌的一些细节和一些演示。

该设计的第一次迭代工作，但它有一些令人分心的线条和曲率。第二个版本使用一大片聚碳酸酯和液体粘合剂来附着透镜。看起来好多了。

当你身后有一个醒目的水平图案时，效果最好，当然，只在某些视角下有效。即使这样，你也可以看到伊恩在盾牌后面移动，但这是相当安静的。然而，最终的版本看起来更好，即使紫外线固化那大块塑料看起来很痛。

视频的最后几分钟展示了一些真实世界的用户测试，看起来很有趣。还有其他方法可以弯曲光线来获得[有限的隐形](https://hackaday.com/2014/10/01/invisibility-achieved-with-a-few-clever-focal-points/)。我们也见过这些用于[人造全息图](https://hackaday.com/2018/11/21/supercon-alex-hornsteins-adventures-in-hacking-the-lightfield/)的镜头。
