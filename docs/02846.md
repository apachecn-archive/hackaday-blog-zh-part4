# 使用 3DBenchy 拖船对垃圾处理进行基准测试

> 原文:[https://hack aday . com/2019/04/29/benchmark-a-garbage-disposal-using-the-3d benchy-tuggle/](https://hackaday.com/2019/04/29/benchmarking-a-garbage-disposal-using-the-3dbenchy-tugboat/)

我们一直对 3DBenchy 这个拖船形状的 3D 打印机校准目标又爱又恨。一方面，拥有一个通用的、广泛使用的、具有挑战性的基准对象来评估打印机性能和改进调优是非常有用的，但我们希望以某种方式收回我们花费在尝试让各种打印机完美运行上的无数令人沮丧的时间。因此，我们带着不小的喜悦观看了下面由[Eric R . Mockler]制作的视频，其中[他使用 3DBenchy prints 来衡量他的最新收购物](https://www.youtube.com/watch?v=rrGHxHEwGcs):他从 Craigslist 上获得的一个新的箱式垃圾处理器。接招吧，拖船！

[Eric]正在考虑将这种处理作为失败印刷回收方法的第一步，最终将废物转化为细丝，大概是为了印刷更多的拖船。这种处理产生的微小碎片应该为进入长丝挤出机的颗粒塑料原料提供合理的替代品，如果这种处理能够完成任务的话。考虑到任何能够磨碎鸡骨头的设备都应该可以处理小型塑料拖船，[Eric]尝试了一下，发现⅓-horsepower 处理厂甚至可以磨碎 100%填充的 PLA 印刷品。

视频简短扼要，所以我们甚至原谅纵向，仅此一次。如果你也在考虑回收你失败的印刷品，你还需要一个细丝挤压机，我们已经为你准备了一个[低成本版本](https://hackaday.com/2013/11/01/low-cost-filament-extruder/)，或者[高通量版本](https://hackaday.com/2014/04/20/filament-extruder-pumps-out-1kghour/)。
