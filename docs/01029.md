# 喜剧徒步旅行的仿步话机是一个快速原型的胜利

> 原文:[https://hack aday . com/2018/10/24/faux-walkie-talkie-talkie-for-comedy-walking-tour-is-a-rapid-prototyping-win/](https://hackaday.com/2018/10/24/faux-walkie-talkie-for-comedy-walking-tour-is-a-rapid-prototyping-win/)

很有可能我们中的相当一部分人以前已经被拉进了“那些”项目中的一个。你知道这种类型:模糊的规格，有限的预算，当然他们昨天就需要它。但是你知道 3D 打印机和 Raspberduinos 等等；你肯定能很快地把一些东西组合在一起。漂亮吗？

他可能没有那么受约束，但当(肖恩·哈金斯)被派去帮助一个朋友完成一个不寻常的项目时，快速原型技术帮助他创造了这个带 GPS 功能的人造对讲机音频播放器。这是一个不同寻常的设备，有着不同寻常的目的:一次温哥华“鬼屋”的喜剧徒步旅行，他朋友的有趣鬼故事是由位置引发的。支持它的硬件基于[【肖恩】有用的 HCC 模块](https://hackaday.com/2018/07/28/save-some-steps-with-this-arduino-rapid-design-board/)，一个 Arduino 兼容开发板。借助用于定位的 GPS 模块、VS1053 编解码器、SD 读卡器和用于音频端的小功率放大器，该设备可以识别用户何时在某个位置的 50 米范围内，并播放正确的音频剪辑。这个外壳是一个旧玩具对讲机的 3D 打印复制品，配有非功能性的橡胶鸭子天线。

[Sean]的身材看起来很棒，也很好，尽管我们在下面的视频中听不到任何有趣的东西；我想我们得为此去趟不列颠哥伦比亚。只用了两周时间就完成了，这令人印象深刻，但要小心——一旦他们知道你是个巫师，他们会继续回来的。
