# 漂移三轮车把一个新的旋转的东西与冰轮

> 原文：<https://hackaday.com/2022/02/25/drift-trike-puts-a-new-spin-on-things-with-ice-wheels/>

漂移三轮车是一种小型坚固的三轮车，前轮有动力，后轮摩擦小，这样你就可以漂移。它们很有趣，但是有很多。如今，如果你想让自己的漂移三轮车脱颖而出，你必须把你自己的旋转。至于额外的旋转，还有什么比用冰做轮子更好的方法呢？[Sam Barker]从分解一辆旧的二手 BMX 自行车开始。前轮轮毂电机不可用，所以他必须做一些修改，以使用后轮电机作为前轮。在调整座椅以增加前轮重量从而获得更好的牵引力后，是时候开始使用轮子了。

他没有使用纯冰，而是决定使用复合材料。受到 Pykrete 的启发，他将木浆换成了棉纤维。在称出纤维与水的不同比例后，他有半打左右不同的样本要测试。他发现，任何大约 2%的东西都很强，以至于用锤子砸也没什么用。他对结果很满意，他 3D 打印了一个模具，在冰变硬时固定住它。[Sam]把它拿出来太早了一点，只是有一些未冻结的中间泄漏出来。他重新填充了模子，让第二个轮子运转起来。等它完全结冰后，他有了一个半轮子，是时候去兜一圈了。我们欣赏漂流过程中 furze-esk 的音乐，因为他是整个过程中的一个明显的灵感来源。我们喜欢看到轮胎在路面上打滑，知道它们很快就会消失。

很明显轮子比车架耐用。他承认他刚买了他的 MIG 焊机，所以也许下一次通过更多的练习，他的大头针会变得更强一点。[Sam]热爱制造电动车辆，他已经制造了一辆独轮车、一辆自行车，现在又制造了一辆三轮车。也许某种巨型四轮直排轮滑鞋是下一个？其他漂移三轮车不走减少摩擦的路线，而是专注于向车轮传递如此大的扭矩，以至于它们忍不住打滑。休息后的视频。
