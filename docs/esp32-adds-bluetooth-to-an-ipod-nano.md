# ESP32 为 IPod Nano 添加了蓝牙功能

> 原文：<https://hackaday.com/2022/09/14/esp32-adds-bluetooth-to-an-ipod-nano/>

iPod Nano 是苹果的杰作之一，但它确实受到了对有线耳机的依赖。至少，[塔克·奥斯曼]肯定是这么想的，因为他花了不合理的时间为第三代 Nano 设计蓝牙模块。这是一个美丽的东西——喜怒无常，难以建造，使用时易碎，但仍然美丽。虽然一些纯粹主义者试图保持他们的信号模拟，[塔克]的妙招是在 DAC 芯片之前拦截 iPod 的音频信号，使整个信号路径数字化到蓝牙扬声器。哦，他还设法让音量和曲目跳过按钮工作，回到无线虚空。

现在我们知道你渴望使用上面链接的漂亮的指令和源代码，并试图复制这个黑客。如果你真的想，那就去做吧。但要知道，所需的焊接是一场噩梦，外壳需要修改以适应额外的电路板，最终设备的电池寿命以分钟而不是小时来衡量。但从什么时候开始这阻止了我们？如果更多的 iPod 黑客是你的事，[看看[塔克]的另一个项目！](https://hackaday.com/2021/05/02/an-epic-quest-to-put-more-music-on-an-ipod-nano-3g/)
