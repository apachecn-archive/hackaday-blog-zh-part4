# 十四条腿的细胞携带着自然界的微型计算机

> 原文：<https://hackaday.com/2022/10/16/fourteen-legged-cell-carries-natures-tiny-computer/>

毕竟，计算机是极其复杂的状态机。我们编写的相当一部分软件也可以建模为状态机。人类的伟大技术成就？据研究*游仆虫*的生物学家称，[状态机存在于自然界一些最小的自然计算机](https://www.cell.com/current-biology/fulltext/S0960-9822(22)01161-7?_returnURL=https%3A%2F%2Flinkinghub.elsevier.com%2Fretrieve%2Fpii%2FS0960982222011617%3Fshowall%3Dtrue)中，游仆虫是一种生活在水中的真核生物。这种单细胞生物使用 14 条被称为 cirri 的原生腿，它们以特定的步态移动，对特定的刺激做出反应。

正如你所料，单细胞生物没有支持大脑的基础设施，所以科学家们想知道什么可以使用 cirri 控制野兽的行走方式。答案是由一束束微管组成的纤维，充当机械状态机。

虽然我们已经习惯了使用双稳态电子元件的状态机，但旧机器通常使用凸轮和微动开关以及定时电机。例如，电话应答机可能有一个三分钟马达。一个摄像头会按下一个微型开关，发出 15 秒的信息。然后另一个凸轮会按下一个微型开关开始记录，最后一个开关和凸轮会保持电机运行，直到最后。为了开始这个过程，一个电话铃声会启动马达，使最后一个凸轮啮合。简单，不需要电脑。游仆虫的“大脑纤维”似乎以类似的方式工作。它们决定了哪些状态可以从哪些状态到达，并对外界刺激做出反应。

这些有用吗？也许不是，虽然我们经常看到技术模仿生物系统。但我们不禁想知道，未来微观尺度的机器是否可能不需要这种相同的机械状态机来实现许多目的，包括移动。

显然，你可以或多或少地让单细胞生物成为你的仆人。如果你需要复习，我们已经多次讨论过[状态机](https://hackaday.com/2015/08/13/becoming-a-state-machine-design-mastermind/)。
