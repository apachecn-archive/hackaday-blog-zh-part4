# 建造一个机械振荡器，特斯拉风格

> 原文：<https://hackaday.com/2019/12/03/building-a-mechanical-oscillator-tesla-style/>

在泰斯拉设计出漂亮简单的旋转机械之前，他探索了产生交流电的其他方法。其中之一是机械振荡器，并且[Integza]尝试在[复制自己的设备](https://www.youtube.com/watch?v=l5rEW9QgJDc)。(视频，嵌入休息下方。)

最初尝试使用 3D 打印部件来复制这项技术是失败的。圆形气缸存在密封问题，使用 O 型密封圈会产生过大的摩擦力，使设备无法正常摆动。使用外部阀门和方形气缸的重新设计证明更加成功。

一旦振荡器完成，输出轴就安装了磁铁和线圈来发电。在产生了令人失望的 0.14 伏电压后，[Integza]回头看了一下麦克斯韦-法拉第方程。利用这一点来指导设计，生产出了具有更多匝数的新线圈，磁通量得到了最大化。完成后，该装置可以产生 7 伏电压，足以点亮几个发光二极管。

虽然它不是一个特别有效的发电机，但它是一个很好的概念证明。是的，特斯拉的发明起作用了，但是很容易理解当涉及到现实世界的应用时，他为什么转向旋转设计。我们也看到[Integza]采用了其他的建造方式，比如一直很受欢迎的特斯拉涡轮机。
