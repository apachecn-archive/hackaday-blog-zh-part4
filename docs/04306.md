# 用飞轮弯曲一些杆

> 原文：<https://hackaday.com/2019/09/25/bend-some-bars-with-a-flywheel/>

在一堆垃圾中寻找宝藏的能力是我们这里非常重视的技能。[与此同时，在车库里]显然有这方面的技能，他用一个旧的飞轮和启动小齿轮为自己造了一台[金属棒弯曲机](https://www.youtube.com/watch?v=KFBaxzzErnM)。

仅用肌肉力量来弯曲金属需要某种机械优势。通常这需要一个长杠杆的弯曲工具，但是[同时在车库]决定利用汽车启动马达和它驱动的飞轮之间的大齿轮比。这消除了对长杠杆的需要，并允许以更大的半径弯曲到几乎 270°。车床和铣削工作非常突出，包括制作弯曲成形器、驱动轴和衬套，以及修改飞轮以包括夹具。用于打磨多个零件的[砂带机](https://www.youtube.com/watch?v=uHBlnSMx604)也是他的发明。虽然机床肯定有所帮助，但大量的创造力和创新思维使这个项目成为可能，值得一看。
