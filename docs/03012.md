# PocketPi 就像它听起来的那样

> 原文:[https://hack aday . com/2019/05/16/pocket pi-is-exact-what-it-it-sound-like/](https://hackaday.com/2019/05/16/pocketpi-is-exactly-what-it-sounds-like/)

Raspberry Pi 系列单板计算机非常有用，但它们通常需要一些附件才能成为有用的计算平台。[Ramin Assadollahi]想要一台口袋大小的电脑，没有智能手机上常见的干扰，所以拿出 PocketPi 来工作。

Ramin 能够将这个项目与这么多现成的部件放在一起，这证明了 Raspberry Pi 平台的受欢迎程度。选择 Pi Zero W 是因为它的紧凑尺寸，而选择超像素 4.0 屏幕是因为它在小封装中的高分辨率。这些部件与 3000 mAh 电池、Adafruit Powerboost 1000C 和小型 USB 键盘和集线器结合在一起。这一切都被包裹在一个整洁的 3D 打印包中，使这款口袋大小的计算机具有经典的 20 世纪 80 年代后期的外观，尽管在引擎盖下有更多的马力。

它看起来像是一个外出时有趣又有用的机器，完整的 QWERTY 键盘使输入变得容易。我们以前见过[Ramin]的工作——[去年的 StickPi 实现了电子纸显示。](https://hackaday.com/2018/06/04/pocket-size-pi-zero-desktop-features-e-paper-display/)休息后的视频。
