# 模拟相机走向数字化

> 原文：<https://hackaday.com/2021/07/15/analog-camera-goes-digital/>

数码相机革命在 21 世纪初席卷全球，除了一些独特的情况和少数艺术家仍在使用胶片，几乎所有人都从那时起转向了数码相机。不幸的是，这意味着世界上有许多高质量的胶片相机正在积灰，但通过一些设备[可以将它们转换为数字相机，并获得更多的用途](https://befinitiv.wordpress.com/2021/07/15/digital-film-cartridge-for-analog-cameras-using-raspberry-pi/)。

[befinitiv]的最新项目通过在原本应该插入相机的胶卷盒处插入树莓 Pi Zero 来处理这种转换。Pi 贴在一个模拟胶片形状的 3D 打印盒上，在胶片曝光位置的正前方还装有一台 Pi 相机。通过移除 Pi 相机的镜头，这种新设置能够利用模拟相机的光学系统，并能够捕捉质量相对较好的图像。

使用这种设置还有一些额外的好处，即视频可以通过 Raspberry Pi 无线连接到电脑上播放到这部手机上。这是一个非常有趣的构建，以非常低的价格获得了出色的效果，而且将相机的快门和其他控制转盘连接到树莓 Pi 可以非常简单地进一步复制旧胶片相机的动作。而且，如果你喜欢[befinitiv]将旧技术带入现代世界的项目，一定要看看他的 80 年代 DOS 笔记本电脑，它能够运行现代 Linux 安装。
