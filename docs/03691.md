# 便宜的扬声器在合适的音箱里听起来很棒

> 原文：<https://hackaday.com/2019/07/26/cheap-speakers-sound-great-in-a-proper-enclosure/>

花几美元买一套低成本的扬声器是可能的，但总的来说，你花的钱并没有得到很多高质量的东西。预计会有一对小的驱动器，声音微弱，噪音来自外壳的廉价材料。然而,[ JSK-库布]向我们展示了这些演讲者可以变得更加出色。

![](img/460e54509f17d5bb929f434abda0e6f1.png)

The internal structure helps improve the frequency response.

至少在这种情况下，里面的基本扬声器驱动器和电子设备还过得去。收集这些材料后，建造者开始为扬声器制作一对令人惊叹的木质音箱。这是通过一个路由模板，大块的木头和大量的肘油脂实现的。

内部结构对扬声器的低音响应产生了巨大的影响，使它们能够更加忠实地再现测试中的音乐。由于工匠级别的工艺，最终产品看起来也令人惊叹。令人印象深刻的是，一对廉价的驱动程序可以在适当的外壳下运行得如此之好，当然，也没有什么可以阻止安装一组更好的驱动程序。

当你制作自己的扬声器时，你的创造力是极限。休息后的视频。
