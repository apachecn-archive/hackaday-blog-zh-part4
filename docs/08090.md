# 杰特森纳米机器人

> 原文:[https://hackaday.com/2020/10/16/jetson-nano-robot/](https://hackaday.com/2020/10/16/jetson-nano-robot/)

[Stevej52]喜欢建造你买不到的东西，这个杰特森纳米机器人就属于这一类。阅读项目细节，你可能会认为[Stevej52]喝了太多咖啡。但我们认为他只是对成功完成集成十几个硬件和软件模块的艰巨任务感到兴奋。简单来说，他在 PC 和 Nano 上运行 Ubuntu 和 ROS。这一切都与 Python 代码捆绑在一起，并使用 Modbus over IP 来解决将操纵杆数据传输到 Nano 的问题。我们喜欢使用现有的标准协议，因为这样可以让设计人员更加专注于应用。Modbus 已经存在了 40 年，在许多语言和平台上都有广泛的支持。

这是一个正在进行的项目，我们期待着看到更多的更新，特别是更多的视频，就像下面这样。随着最近降价的 Jetson Nano 的发布，这可能是一个很好的项目。
