# Commodore 64 的新操作系统增加了现代功能

> 原文：<https://hackaday.com/2022/07/28/new-os-for-commodore-64-adds-modern-features/>

Commodore 64 在当时是一台革命性的电脑。然而，四十年后，除了教育或业余爱好电子项目之外，越来越难将这些电脑用于任何其他用途。[Gregory Nacu]下定决心要挑战这种想法，并走了极端，通过为 Commodore 机器编写一个全新的操作系统，使这种硬件在现代仍然适用。

被称为 C64OS，它从 8 位处理器和 64 kB 内存中挤出了所有它能挤出的东西。新操作系统包括可切换的桌面工作区、窗口系统、可拖动图标、顶部的 Mac 风格菜单栏和图标下拉菜单(在演示中称为别名)。文件系统也做了很大的改进，可以使用更现代的目录系统。仍然有一些限制，如 320×200 像素的屏幕分辨率和固定的调色板，它只允许少数几种颜色，但这个操作系统可能会让 Windows 3.1 物有所值。

该项目仍在积极开发中，但它已经走了很长一段路，进入了相当可用的状态。它可以在原始硬件上运行，只要你有办法把图像传到古董机器上。如果不是这样，操作系统可能会运行在任何数量的 C64 模拟器上，这些模拟器是我们在过去[中介绍过的](https://hackaday.com/2014/10/23/a-complete-c64-system-emulated-on-an-stm32/)。
