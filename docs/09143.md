# 机械七段显示器，比原来的更小更好

> 原文：<https://hackaday.com/2021/02/04/mechanical-seven-segment-display-smaller-and-better-than-the-original/>

在 Hackaday，我们喜欢的一件事是跟踪项目随时间的发展。看到一个项目随着时间的推移而增长是非常典型的——毕竟，范围蔓延是真实的。但是看着一个项目缩小也是一种真正的享受，因为早期的版本会被改进成更光滑、更优雅的解决方案。

[这款纤薄的七段机械显示屏](https://www.youtube.com/watch?v=tKMp4u-hMTY)是这种小型化趋势的完美体现。当我们看到[IndoorGeek]的[的第一个机电显示器](https://hackaday.com/2020/07/27/mechanical-seven-segment-display-really-sticks-out-from-the-pack/)时，它相当笨重。和现在一样，每个部分都是 3D 打印的，后面附有磁铁。这些段悬浮在螺线管线圈上，当通电时，螺线管线圈排斥磁体并突出段，形成所需的数字。不过，旧版本使用了大型的手绕线圈，使得显示器从正面到背面都相当笨重。

第 2 版显示器借鉴了[Carl Bugeja]的剧本，用 PCB 线圈取代了缠绕线圈。我们已经在[刚性基板](https://hackaday.com/2018/01/24/this-tiny-motor-is-built-into-a-pcb/)和[柔性印刷电路板](https://hackaday.com/2020/03/31/magnets-turn-flexible-pcb-into-electric-grasshopper/)上看到了【卡尔】的线圈；[IndoorGeek]这里用的是普通的老式 FR4。线圈占据四层，因此它们有足够的力量来伸展和收缩每个部分，PCB 包括用于每个部分的 H 桥驱动器的空间。PCB 形成了显示器的后盖，对于这个版本来说，它也是相当薄的。然而，相同的是这种显示器看起来有多好，特别是在强侧光的情况下——延伸部分投射的阴影在显示器的白色表面上非常醒目。

祝贺[IndoorGeek]的出色构建和对原始版本的有益改进。
