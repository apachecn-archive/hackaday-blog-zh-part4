# 垫片，可以 3D 打印吗？

> 原文：<https://hackaday.com/2022/04/20/gaskets-can-they-be-3d-printed/>

任何拥有旧发动机的人，无论是汽车、摩托车还是园艺机械，都会在某个时候面临需要衬垫的问题。即使垫片是现成的，也可能有修理发动机的必要，而不是等待零件到达，所以自己更换是很常见的。简单的卡片很容易从薄卡片上裁剪下来，但是如果你曾经尝试过用一个非常复杂的卡片来做这件事，你就会知道把它做好的痛苦。这是来自[the_eddies]，[的一段视频中解决的问题，他探索了通过 3D 打印制造替代垫圈的方法](https://www.youtube.com/watch?v=rPgr2VkYESw)。

CAD 和易于制造的优势是显而易见的，但也许许多普通塑料在炎热或多油的环境中表现不佳。为此，他选择了 TPU 灯丝，并在二冲程混合燃料的浴缸中对其进行了测试，以了解其抗降解的能力。它通过了，就像它和化油器一起使用时一样，尽管我们很想看看长期测试的结果。我们已将视频放在休息时间的下方，所以请得出您自己的结论。

垫圈以前在这里出现过，如果你有兴趣的话[还有其他机器可以用来制造它们](https://hackaday.com/2020/03/30/vinyl-cutter-migrates-from-scrapbooks-to-gaskets/)。
