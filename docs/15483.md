# 便携式准将 64 生命！

> 原文:[https://hack aday . com/2022/11/27/portable-commodore-64-lifes/](https://hackaday.com/2022/11/27/portable-commodore-64-lives/)

当你想到便携式电脑时，你可能会想到老式的康柏电脑或者——如果你够老的话——Kay pro 电脑。但是你看不到 SX 64 号准将电脑。[8 位家伙]想要一个已经有一段时间了，终于得到了一个，但它不起作用。没问题！[修好就好](https://www.youtube.com/watch?v=i4gHaXOevVM)！

与那个时代的其他便携式设备相比，这款设备实际上看起来很时尚，而且有彩色屏幕，但可能是因为价格的原因，它们卖得不是很好。除了一些松动的螺丝和夹子，这个设备的外部看起来非常干净。空格键很黄，但至少有一根键盘线，几乎不可能再找到了。

打开它只是在屏幕上显示一个白色的光栅。拆卸过程很有趣。这个小箱子里装了不少东西。当然，这意味着你必须拉出五块板才能够到它们。

卸下外壳后，显示屏出现了一点变化。他拔出 CPU 板，开始用一个正常工作的 Commodore 64 交换芯片，直到他发现两个芯片会阻止已知正常的机器工作。更换它们确实解决了 CPU 的问题，但是键盘仍然没有完全发挥作用。

这也导致了键盘的拆卸。一些 3D 打印的闩锁修复了物理损坏的部分。键盘本身是不寻常的，有一堆集成的按键和键帽，以及一个柔性薄膜，这显然是一个常见的故障点。然而，你可以得到替代品，但这项工作看起来需要一点耐心。

总的来说，这台机器看起来像新的一样。然而，正如[8 位家伙]所指出的，彩色显示器从来都不是很好，他解释了这到底是为什么。

我们很乐意为这些机器打分，尽管糟糕的屏幕使它们不如一个合适的 C64 有用。我们已经看到有人[为这些机器](https://hackaday.com/2015/12/13/adding-a-second-drive-to-a-forgotten-commodore/)添加了第二个驱动器。如果你想知道更多关于[修复 SX-64 键盘](https://hackaday.com/2020/06/30/commodore-sx-64-keyboard-upgraded-from-trash-to-treasure/)的细节，我们以前也看过。
