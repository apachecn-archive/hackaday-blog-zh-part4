# 你也可以成为铁路大亨！

> 原文：<https://hackaday.com/2021/03/14/you-too-can-be-a-railroad-baron/>

很可能我们的读者中有不少人对火车有好感。无论你称它们为铁路还是铁路，它们都是最初的技术魅力，而“黑客”一词是在麻省理工学院的技术模型铁路俱乐部中创造出来的，这绝非偶然。所以你们中的一些人和我们一样观看机车 YouTube 视频，其他人可能在某个地方藏了一个 OO 布局，还有更多人看着经过的火车，希望他们在车上。拥有一条自己的铁路仍然是一个白日梦，但也许一个铁杆铁路爱好者可能会喜欢看一看[【西部之路博客】关于建设一条农场铁路的视频系列](https://www.youtube.com/watch?v=NfSK_sKBhTg)。

在一个小农场里，总是有很多东西要搬来搬去，而且常常没有机器来搬。正如我们可以证明的那样，在粗糙的地面上使用独轮车或手推车是很累人的，所以市场上确实存在缓解这一任务的缺口。因此，假设铁路的建设成本不太高，铁路就成了一个有吸引力的解决方案。

下面的视频是毫无疑问将成为一个漫长系列的前两个，涉及铁轨本身的建设，包括用自制带锯切割的轨枕，然后是鱼尾板和一组基本点。铁轨本身是现成的平钢带，铺设在铁轨边缘，并通过短长度的镀锌管固定在枕木上。很明显，这不是我们可能理解的那种意义上的铁路，事实上，虽然它使用边缘轨道，但它与一些早期的采矿栈道有更多的共同点，但假设扁平的条形轨道不扭曲，我们可以看到它应该完全适合手动推车，消除了移动中的伤筋动骨。沿着这条线跟踪这个项目会很有趣。

农场铁路以前没有在 Hackaday 上出现过，但你内心的铁路爱好者可能会对世界上第一条保留下来的线路感到满意。
