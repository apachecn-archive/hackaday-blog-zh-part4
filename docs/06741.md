# CNC 线锯首次切割前景看好

> 原文：<https://hackaday.com/2020/06/04/cnc-scroll-saw-makes-promising-first-cuts/>

当我们谈论数控机床时，我们几乎总是指计算机控制的路由器。当然，你可以做其他形式的自动切割，比如说使用激光或喷水器，但是给其他类型的锯增加计算机控制怎么样？[Andrew Consroe]最近制作了一段关于这台实验性 CNC 卷轴锯的事后视频。虽然他从未让它可靠地工作，但我们认为他的方法绝对令人着迷，希望这不是我们最后一次看到这个想法。

那些过去使用过滚动锯的人可能会立即看到这种结构的挑战:虽然刳刨机钻头或激光束可以在任何方向切割，但滚动锯片只能在一个方向切割。如果你试图在一个线锯上急转弯，你只会把易碎的锯条折断。为了解决这个问题，[Andrew]想出了一个绝妙的转盘，在休息后的视频中可以看到。

通过将机架的运动与工作台的旋转结合起来，他能够防止刀片转得太紧。或者至少，理论上是这样。虽然机器在刀片的位置安装了标记器，但[Andrew]说，他从未让它可靠地切割。听起来定位误差会加剧，直到机器最终移动工件，导致刀片折断。尽管如此，这个概念肯定是可行的；在视频快结束时，他展示了几块在他的机器扔出刀片之前成功切割的碎片。

虽然我们在过去已经看到过 DIY 卷轴锯，但这是第一台电脑控制的卷轴锯。虽然有些人无疑会争辩说，既然激光切割机已经达到了可以承受的价格，那么就没有必要再建造一个这样的[了，但我们绝对喜欢这个设计，也非常喜欢它的设计。至少，我们认为这是有史以来最强壮的涂鸦机器人。](https://hackaday.com/2018/09/27/laser-noob-getting-started-with-the-k40-laser/)
