# 纸箱里的东西比纸箱里的要多

> 原文：<https://hackaday.com/2022/03/08/theres-more-in-a-cardboard-box-than-what-goes-in-the-cardboard-box/>

纸箱在我们的社会中无处不在。我们都知道纸箱是由什么组成的:瓦楞纸产品、胶水和一些工作。当然，纸箱不会在某一天突然出现，由一个工作过度且收入微薄的司机不知从哪里送来。在休息时间下方的[视频中，【新思维】深入探究了纸板盒的历史以及更多内容。](https://www.youtube.com/watch?v=PM1QMGPL79A)

从 19 世纪开始，大量木材加工成纸浆的进步使得纸张变得便宜。从那时起，纸板开始形成波纹形状。欧洲和美国的许多进步在某种程度上是相互独立的，到 1906 年，一个联合企业成立，让铁路部门批准纸板用于货运列车。

尽管那时，纸板仍处于婴儿期。设计、制造和效率方面的进一步进步已经将看似低技术含量的纸板盒转变为高度自动化和质量控制的高科技产业。当然很难把纸板箱想成一样的。

黑客也有许多方法来重复使用纸板，无论是在模板制作、原型制作、模型制作等等。当然，瓦楞不仅仅适用于纸张。如果波纹塑料让你的船漂浮，你可能会对这条[因波纹塑料而漂浮的船感兴趣。](https://hackaday.com/2020/06/17/diy-plastic-speedboat-for-one/)
