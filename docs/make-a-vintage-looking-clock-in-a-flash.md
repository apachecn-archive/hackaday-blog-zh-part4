# 瞬间做出复古外观的时钟

> 原文：<https://hackaday.com/2020/02/23/make-a-vintage-looking-clock-in-a-flash/>

现在每个人都有一部带摄像头的手机，我们敢打赌，市场上很少有人会买一个漂亮的老式闪光灯，比如[lonesoulsurfer]为这个酷酷的时钟造型选择的那个。但是这里有一样东西永远不会过时——一个兼做谈资的时钟。

这个建筑的核心是一个非常便宜的汽车时钟单元。它还显示环境温度，并具有电压测试模式(！)，总有一天会派上用场。虽然[lonesoulsurfer]没有连接一对探针，但他确实切了一个小洞让温度传感器伸出来。他还切断了 SMD 按钮，并在外壳外部连接了新的 momentaries。

虽然我们真的很喜欢 7 段上有纹理的塑料镜头的外观，但我们最喜欢的部分可能是支架和[lonesoulsurfer]实现它的方式。他通过在一根铝管的末端敲入一个六角螺母制成了一个螺纹杆，然后在关闭闪光灯之前，将一个螺栓穿过闪光灯主体的底部，这样它就像相机一样固定在三脚架上。休息之后，请花点时间观看构建视频。

我们非常喜欢一个好的时钟，所以我们刚刚举办了一场比赛，寻找最酷的看时间的方法。如果你错过了，[这里有最好的](https://hackaday.com/2020/02/13/tell-time-contest-winners-clocks-to-knock-your-socks-off/)中的最好的。
