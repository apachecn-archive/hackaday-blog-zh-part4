# 可编程腕式合成器突破极限

> 原文：<https://hackaday.com/2019/12/11/programmable-wrist-synth-pushes-the-envelope/>

合成器是一种乐趣，无论它们听起来是好是坏。真的，没有不好听的，只是如果你不是发出哔哔声的那个人，有些听起来比其他人更烦人。几年前,[Clem]在一个表壳中内置了一个小小的 LDR 合成器，并把它带到了许多制造商博览会上，在那里它高兴又恼火，直到它最终坏掉。

自然，是时候[制作一个更有能力的新版本](https://www.element14.com/community/docs/DOC-94183/l/episode-423-programmable-arduino-synthesizer-watch?ICID=hp-element14-triptic)了。第一个是 Atari-punk-console-meets-light-Theremin，而这个有一堆输入，可以在飞行中编程，以记录和播放可弯曲的音调。它由一个 Arduino MKR 驱动，输入由一个令人印象深刻的 squash bug-wired 移位寄存器管理。[Clem]这次使用了结实的开关，希望这一次能持续更长时间。我们认为滑动花盆非常棒，用 PMMA 印刷的糖果色旋钮也是如此。

我们最喜欢的部分是[Clem]利用了微控制器引脚在第一次通电时所处的随机状态。如果你不想编写任何笔记，你可以使用启动时生成的笔记，然后随便玩玩。请务必在休息后观看构建视频。

我们已经看到了我们的合成器，但很少有像今年 Hackaday 奖的 kel pie 这样好看的。
