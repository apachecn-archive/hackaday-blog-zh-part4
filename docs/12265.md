# 制作自己的链条围栏

> 原文：<https://hackaday.com/2021/12/21/make-your-own-chain-link-fencing/>

如果你发现自己需要链条围栏，你可能会去五金店。然而,[The Q]向我们展示了你可以在家里用一台简单的机器做出自己的东西。

建造从一段管子开始，用角磨机在管子上切割出螺旋槽。这个管子是成形工具，它将金属丝成形为人们熟悉的链节形状。然后将管子焊接到垫板上，并安装一个可移动的手动曲柄，转动一根扁钢。将金属丝放入螺旋槽中，转动曲柄，就能拉出所需形状的金属丝。

从那里，成型的金属丝可以连接成任何尺寸的栅栏。当然，紧固栅栏的每个末端是留给读者的练习，末端是尖锐的和未完成的。然而，如果你不喜欢当地五金店出售的链节围栏，或者你想用某种奇特的金属丝编织自己的围栏，这台机器可能正是你需要的东西。
