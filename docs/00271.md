# 指纹挂锁的拆卸

> 原文:[https://hack aday . com/2018/08/05/fingerprint-padlock-tear down/](https://hackaday.com/2018/08/05/fingerprint-padlock-teardown/)

我们不介意拆掉指纹扫描仪，但我们讨厌弄坏我们昂贵的笔记本电脑或手机。然而，[朱利安]拿到了一把指纹扫描挂锁，[愿意为了我们的利益把它拆开](https://www.youtube.com/watch?v=EsdbyYRgyKk)。视频如下。

挂锁是一个足够简单的小装置，带有一个缆绳锁，而不是一个坚固的金属钩环，尽管我们已经看到了带有传统钩环的类似装置。最初，锁的指纹存储是空的，它会为任何指纹打开。第一项任务是设置管理员指纹。您将需要该指纹来设置其他指纹或重置设备。当然，我们真正感兴趣的是里面的东西。

锁工作时听到里面有马达声。一旦他熟悉了基本操作，工具就出来了。挂锁应该很难打开，但看起来很容易打开。事实上，它看起来像固定电缆锚螺钉，所以如果你能旋转它足够多，你可以打败锁。

当电缆推入微动开关时，电机驱动凸轮，凸轮抓住锁销。一开始看起来里面没有很多电子设备，直到电池出来。是的，你必须用 USB 给挂锁充电。

传感器本身相当不透明——只是一个匿名模块。如果你真的想知道它们是如何工作的，我们已经讨论过了。锁本身似乎不是很安全，正如我们之前看到的那样，指纹扫描仪[也不是那么安全](https://hackaday.com/2015/11/10/your-unhashable-fingerprints-secure-nothing/)。
