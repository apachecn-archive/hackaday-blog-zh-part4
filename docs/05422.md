# 一个战争游戏的复制品，但不是你想的那个

> 原文:[https://hackaday . com/2020/01/14/a-来自战争游戏的复制品-但不是你想的那个/](https://hackaday.com/2020/01/14/a-replica-from-wargames-but-not-the-one-you-think/)

还记得战争游戏里的 WOPR 吗？与马修·布罗德里克和他的声学耦合器面对面的虚构超级计算机就像是给昔日闪光灯主机的情书，每个特定年龄的黑客都暗暗渴望自己的缩小模型。嗯…这不是这个项目是什么。

这位[意想不到的制造者]和我们一样都是战争游戏的粉丝，但[他更感兴趣的是重现红色字母数字显示器](https://www.youtube.com/watch?v=dfT-OtWHfys)，当 WOPR 试图暴力破解导弹发射代码时，红色字母数字显示器滴答作响。这些显示器，配有完全上世纪 80 年代的“计算机”音效，被用来通过显示超级计算机离第三次世界大战有多近来加剧紧张局势。

[![](../Images/14d2e70b5fb9819f31d183d6b6b03948.png)](https://hackaday.com/wp-content/uploads/2020/01/woprled_detail.jpg)

The display as it appeared in the film.

当然，大多数美国人没有导弹发射井来安装他的重建显示器。因此，当它不运行随机启动代码解码序列之一时，显示器会兼作 NTP 同步时钟。与复古的十四段发光二极管后面的烟熏丙烯酸前面板，我们认为时钟本身是相当光滑，即使没有电影参考。

除了前面提到的 led，[意想不到的制造者]还使用了自己设计的名为 TinyPICO 的 [ESP32 开发板。一个带有集成蜂鸣器的相关音频“屏蔽”在显示器进行运动时提供适当的哔哔声和滴答声。一切都保存在一个低调的 3D 打印外壳中，挂在墙上或桌子上看起来很棒。](https://www.tindie.com/products/seonr/tinypico/)

一旦你把你的发射代码破坏 LED 时钟放在角落里，你的[照明 DEFCON 显示器安装在墙上](https://hackaday.com/2017/09/16/we-are-now-at-defcon-2/)，你就可以很好地完成我们自 1983 年以来一直梦想的 *WarGames* 游戏集了。输的唯一方法就是不玩游戏！(或者类似的…)
