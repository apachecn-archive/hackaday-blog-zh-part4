# 具有节点红色和张量流的可视化树莓 Pi

> 原文：<https://hackaday.com/2021/04/22/visual-raspberry-pi-with-node-red-and-tensorflow/>

如果你喜欢画盒子而不是写代码，你可能已经尝试过 IBM 的 Node-RED 用拖放流创建逻辑。最近的一个[TensorFlow]视频显示了[Jason Mayes]和[Paul Van Eck]关于使用 TensorFlow.js 和 Node-RED 可视化地创建 Raspberry Pi 的机器学习应用程序的采访。你可以看下面的视频。

该视频没有涉及太多细节，因为它只有十分钟长。但它确实显示了使用现有的张量流模型来识别图像是多么容易。有一个[更详细的教程可用](https://developer.ibm.com/technologies/artificial-intelligence/tutorials/building-a-machine-learning-node-for-node-red-using-tensorflowjs/)，还有一个[对应的视频](https://www.youtube.com/watch?v=bOdlPwWej98)，你可以在下面看到。

作为一个具体的例子，[保罗]使用一个简单的传感器来识别他的猫，然后按门铃，把猫从桌子上吓走。在一个更具话题性的例子中，一个系统喷洒任何没有戴口罩的人。

也许一个更实际的例子是一个车库开门器，它通过汽车牌照号码来识别汽车。我们不确定它是否会打开一块写有车牌号码的海报板，或者车牌的照片。
