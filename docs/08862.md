# 将坐标测量机转变为 3D 打印机

> 原文：<https://hackaday.com/2021/01/06/turning-a-cmm-into-a-3d-printer/>

拥有一台 3D 打印机有两种途径:购买一台或按照你自己的精确规格制作自己的 3D 打印机。[Roetz 4.0]已经决定走后一条路线，并且[将 1.3 吨的空气轴承坐标测量机(CMM)转换成 FDM 3D 打印机](https://www.youtube.com/playlist?list=PLT7wyh27Lmsf2RJYYPw3e3F_YKwrFVkXD)。(视频，嵌在下面。)

CMM 是一种工具，用于通过轻轻降低校准的探针来精确测量物体的几何形状。在之前，我们已经见过 [scratch build 打印机，但这种特殊的构建得益于 CMM 机械及其 18 个空气轴承。CMM 头由[Roetz 4.0]自己的定制系统移动，但它利用了轴承。经过一些仔细的 CAD 规划，以及相当多的铣削、车削和原型制作，他有了非常流畅的控制动作。](https://hackaday.com/2020/11/04/scratch-built-3d-printer-goes-big/)

一个现成的驱动板与一个大红色按钮连接在一起，他准备进行首次测试打印。在年底前完成的决心推动了事情的进展。还有一些小问题需要解决，比如空气干燥系统的漏洞，但这些问题可以在明年解决。最终，我们认为结果是惊人的，这是一个我们很高兴用[Roetz 4.0]进行的旅程。这个系列的最后一集在休息之后。
