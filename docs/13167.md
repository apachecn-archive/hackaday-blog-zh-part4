# 用于回收的 3D 打印锻造

> 原文：<https://hackaday.com/2022/03/27/3d-printed-forge-for-recycling/>

如果你拥有一台数控机床，并且在过去几年(老实说是几个月)一直关注着金属价格，当你看到芯片从你的工作中飞出，落入垃圾桶时，你可能会流下一滴眼泪。叹了口气，这些斑点和碎片被视为加工一个零件的成本。令人欣慰的是，[ActionBox]的优秀员工一直在研究一种用于回收金属废料的 3d 打印石膏熔炉。

该团队从一家大型在线书店订购了一些不同尺寸的石墨坩埚，并开始 3D 打印一些坩埚支架的模具。他们从一个较小的版本开始尝试这种方法。虽然最初版本的墙太薄，但这种方法已经被证明是可行的。由于壁稍厚，中型版本的效果要好得多。锻造厂的目标是熔炼铜，因为他们有很多粗铜线。有了几个丙烷火炬，他们开始熔化铝和黄铜，效果相当不错。然而，铜的熔点仍然让他们摸不着头脑(1984 华氏度或 1085 摄氏度)。为了应对这一点，[ActionBox]团队购买了一些新的火炬，提供了明显更高的 BTU 输出，同时仍然适合模具中的孔。这个成功了！

容纳大坩埚的模具很大，分为四部分印刷。该团队确实成功地熔化了铜，并有四块铜锭可以炫耀。我们想强调熔化的铜和其他金属有多危险，特别是那些你可能没有意识到里面有水分的东西。正确的个人防护装备是使用这些东西而不受伤的关键。[ActionBox]在这方面有一些有用的建议，但他们承认自己在锻造和铸造方面相对较新。或许第二版可以[加入翻盖](https://hackaday.com/2021/09/11/flip-top-foundry-helps-manage-the-danger-of-metal-casting/)以增加安全性。
