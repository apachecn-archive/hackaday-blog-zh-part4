# 反对平板电脑设计的泡沫芯

> 原文：<https://hackaday.com/2020/05/20/a-foam-core-stand-against-tablet-design/>

我们没有考虑到尝试在平板电脑上画长时间会有多大的挑战，听起来好像苹果也没有。按照通常为他人设计产品的[埃里克·斯特雷贝尔]的说法，有许多问题需要解决。相机区域在平坦的背面产生了一个凸起，所以它在桌子上摇晃。它很厚。你的手写笔很容易从侧面滑落。

是的，现在有很多平板电脑支架，甚至有几个带杯架的，但是几乎没有一个有支架可以垂直放置。如果你想把事情做好，你必须自己去做。[因此[Eric]设计了他的理想支架来解决所有这些问题](https://www.youtube.com/watch?v=RiKtKaqNxyc)(视频，嵌入下文)。它主要由激光切割泡沫芯板制成，并添加了几层海报板，使边框与平板电脑完全齐平。

[Eric]可以将平板电脑固定到位，平放使用，或者将平板电脑的上半部分折叠成支架。它甚至在沙发上或坐在床上也能很好地工作。我们特别喜欢窗户垫圈脚和他的铰链的所有版本，从粗棉布条开始，到罗缎丝带结束。[Eric]的设计方法总是提醒我们对材料和方法保持开放的心态。如果你尝试使用你已经拥有的，结果可能会让你大吃一惊。休息之后，请观看构建视频。
