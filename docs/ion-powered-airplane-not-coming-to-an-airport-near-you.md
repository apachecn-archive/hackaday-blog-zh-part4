# 离子动力飞机:不会靠近你的机场

> 原文：<https://hackaday.com/2018/12/31/ion-powered-airplane-not-coming-to-an-airport-near-you/>

不是我们不喜欢《星际迷航》，而是编剧们永远无法决定离子推进是超级高科技(斯波克的大脑)还是他们每天都在使用的东西(动物园)。不管怎样，离子推进是真实的，我们今天在不止一个航天器上有它。然而，麻省理工学院最近展示了一架离子动力飞机。多刺激啊！没有活动部件靠电力运行的飞机。空中旅行将永远改变，对吗？根据【真实工程】，[离子推进(全尺寸)飞机遇到了物理定律](https://www.youtube.com/watch?v=IorDYGI1uqc)的问题。你可以在下面的视频中看到解释。

要理解为什么，你需要知道两件事:离子驱动是如何工作的，以及在大气层中使用时这些引擎有什么不同。让我们从天基离子引擎开始，这是我们在之前已经讨论过的话题[。原子变成离子，离子被电加速。所以离子发动机只是用电来产生推力排气，而不是燃烧火箭燃料。](https://hackaday.com/2018/10/27/travel-to-mercury-on-ion-power/)

缺点是推力非常小。好的一面是，在太空中，微小的推力累积起来，在几天或几周内，你可以移动得非常快。由于没有活动部件，保持离子发动机持续运转不成问题。让一个巨大的火箭燃烧几个月是有问题的。

所以麻省理工学院的飞行器使用了同样的技术，对吗？算是吧。宇宙飞船自身携带氙形式的离子燃料(尽管一些老式发动机使用水银)。氙很好，因为它相对较重，可以提供更大的推力，并且易于储存。

如果你把离子引擎夹在飞机上会怎么样？嗯，你还需要把氙燃料箱放在上面，这会使飞机更重。你也有两个问题。你需要一定的最小速度来让你的翅膀产生升力。此外，由于风阻等因素，你微小的推力不会像在太空中那样累加。如果宇宙飞船的引擎停止，它只是停止加速，但继续以目前的速度和方向前进。如果飞机失去动力，情况就不是这样了。

显然，麻省理工学院的工程师必须创造一个非常轻的机身，可以在低速时产生很大的升力。最终的飞机翼展 5 米，重量不到 3 公斤。但是燃料呢？宇宙飞船自带氮气，但是飞机会消耗大气中无处不在的氮气。当然，它没有氙的质量，但不用携带燃料是一个很大的优势。

那你为什么不尽快登上那架午夜离子飞机去乔治亚州呢？规模。视频很好地解释了交易，但简单来说，一架重型飞机需要更多的动力，你会陷入一个恶性循环，更多的重量需要更多的动力，但更多的动力会增加更多的重量。

在第一个视频之后，你可以看到麻省理工学院关于他们的固态飞机的视频(我们喜欢这个名字)，如下。当然，未来的进步可能会使离子动力飞机更加实用。但很可能不会在未来一三年内。然而，还有其他方法可以让一架[飞机停飞](https://hackaday.com/2017/12/27/3d-printed-airplane-engine-runs-on-air/)，你永远不知道什么时候一些突破会让它变得实用。毕竟，在 1950 年，谁能想象只需几百美元就能装进口袋的电脑？
