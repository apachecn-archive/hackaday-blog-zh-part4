# 皮埃卡是一台 Pi 相机，有一些非常好的镜头

> 原文：<https://hackaday.com/2022/11/05/pieca-is-a-pi-camera-with-some-very-nice-lenses/>

高质量版本的 Raspberry Pi 相机的出现给了实验者一个足够好的质量相机系统，他们可以用它来创建比简单的快照相机更好的设备。它已经被实验者用于一些令人兴奋的项目，但到目前为止，很少有人脱离 Pi 相机的 C-mount 镜头系统。汤姆·舒克(Tom Schucker)的 Pieca 是一个有趣的开始，因为它通过使用徕卡测距镜头将 Pi HQ 相机带入了一个新的领域。

有足够多的 Pi 相机项目，到目前为止，建立一个项目的过程应该是非常众所周知的。这个有点不同，它使用了一个焦距缩减器，安装在一个 3D 打印的徕卡镜头安装板内。结果是，与 35mm 框架相比，徕卡镜头更好地匹配了尺寸小得多的 Pi 相机传感器。

这款相机的美学设计偏胖，可能是因为选择了 Pi 4 而不是 Pi 0。它仍然非常有用，拍出的照片有独特的感觉。你可以在休息时间下面的视频中看到更多。与此同时，如果你不够幸运，没有一个稳定的徕卡镜头，也许你可以考虑采用更普通的光学元件？[我们之前已经用最初的 Pi 相机](https://hackaday.com/2015/09/19/hacking-a-pi-camera-with-a-nikon-lens/)看过了。
