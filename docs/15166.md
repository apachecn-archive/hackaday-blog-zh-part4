# 作为重型模拟输入的管道阀门

> 原文:[https://hack aday . com/2022/10/24/plumbing-valves-as-heavy-duty-analog-inputs/](https://hackaday.com/2022/10/24/plumbing-valves-as-heavy-duty-analog-inputs/)

能够应对恶劣环境的输入设备远不如它们更脆弱的同类设备多样。[Alastair Aitchison]设计了一种聪明的方法来检测来自管道阀门的输入，这提供了另一种选择。(YouTube)[通过 [Arduino 博客](https://blog.arduino.cc/2022/10/22/plumbing-valves-make-great-heavy-duty-analog-inputs/)

虽然[Aitchison]可以在内部有水的情况下运行管道阀门并检测流量，但他认为更优雅的解决方案是使用光电传感器和 LED 来简化系统。这避免了泵和流量传感器的额外成本，以及混合电子设备和水的可疑提议。通过分析阀门关闭或打开时光强度的变化，您可以输入一系列值或设置开/关条件的阈值。

[Aitchison]这些是为逃生室设计的，但我们可以看到它们非常适合博物馆、游乐园，甚至是(火车)模拟器。他说，他选择管道阀门的主要原因之一是为了美观。工业开关和街机按钮有它们的位置，但在某些情况下肯定不是最合适的，尤其是如果你想要一个时期的感觉。此外，由于传感器本身没有任何移动部件，如果阀门本身发生任何问题，这些模拟输入将很容易修复。
