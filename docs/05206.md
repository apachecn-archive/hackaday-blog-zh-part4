# 电路雕塑教二进制，玩乒乓

> 原文：<https://hackaday.com/2019/12/25/circuit-sculpture-teaches-binary-plays-pong/>

我们真希望我们有一位像(丹科·贝尔托维奇)这样的老师。他建造了这个美丽的电路雕塑来教他的学生如何用二进制计数，并将其转换为十进制和十六进制。如果你还不知道二进制，休息之后，你可以在他最新的 *Volos 项目*视频中，通过 DIP 开关和一个被窃听的 ATMega328 来学习它。你真幸运！

一旦学生们掌握了在开关上输入二进制输入的窍门，他们就可以在四柱计算器上练习了。这个教育雕塑也可以接受文本输入并滚动它，但这需要一点工作。你必须查找每个字符的 ASCII 值，将十进制转换为二进制，并用开关将其编程。菜单上还有一个功能——一个单人乒乓球游戏，帮助学生在漫长的一天开关后放松。

有趣的是，这个项目是在[Danko]在他的零件箱中发现了 DIP 开关并且不太确定它叫什么之后开始的。他学到了一些关于这个零件的知识，然后用这些知识建造了这个用这个零件来教别人的机器，这有多棒？这无疑是部分好奇者所能期待的最好的命运。

对电路雕塑没耐心？你可以用一点纸和大量压缩空气制作一个非常漂亮的二进制计算器。
