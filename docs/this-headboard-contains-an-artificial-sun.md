# 这个床头板包含一个人造太阳

> 原文：<https://hackaday.com/2022/10/12/this-headboard-contains-an-artificial-sun/>

尽管人工照明激增，人类仍然高度依赖阳光来调节我们的昼夜节律。因此，[07 部门]已经[建造了一个未来的床头板](https://www.instructables.com/Artificial-Sunrise-Simulation-With-an-Integrated-V/),无论你是前往太空还是在严冬，它都可以帮助你醒来。

床头板的内部包括定制的 3D 印刷面板，用于安装电子设备和由尼龙织物制成的光漫射屏幕。打印出来的零件都是用烙铁和额外的灯丝“焊接”起来的。除了扩散屏幕中未来主义的六边形主题，这个建筑最引人注目的部分是弯曲的末端，使它看起来像《星际迷航:TNG》中的场景。[07 区]能够得到独特的形状，通过锯缝弯曲胶合板末端，然后用销钉和木胶将它们连接到平坦部分。

由于这个建筑还包括一个集成的咖啡机和语音助手，所以电子设备比普通的昼夜节律灯要多一些。为该项目提供动力的是两块 Arduino Mega boards 和一个处理语音命令的 [SpeakUp Click](https://www.mikroe.com/speakup-click) 。唤醒时间通过键盘控制，一旦 30 分钟的太阳模拟完成，语音助手 Prisma 会在你的闹钟响起之前询问你是否醒着。如果你不确认清醒，Prisma 会升级警报，直到系统确定你醒了，然后会问你是否想喝咖啡。如果你想深入了解系统的功能，请务必在休息后观看视频。

我们以前报道过人造太阳，所以如果你有兴趣尝试建造自己的人造太阳，你应该看看这个[非常明亮的人造太阳](https://hackaday.com/2021/11/06/tiny-wake-up-light-is-hugely-bright/)，一个安装在床上方的[日出闹钟](https://hackaday.com/2013/01/25/sunrise-alarm-clock-mounted-above-the-bed/)，以及这个通过旧卫星天线的[人造太阳。](https://hackaday.com/2020/11/13/artificial-sun-lighting-via-old-satellite-dishes/)
