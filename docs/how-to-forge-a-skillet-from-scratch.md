# 如何从零开始打造一个煎锅

> 原文：<https://hackaday.com/2021/12/22/how-to-forge-a-skillet-from-scratch/>

炊具不是我们通常自己制作的东西；你会想要一个设备齐全的金属店来做这项工作，并把它做好。然而，[托尔比约恩·阿曼]正是这样做的，并着手从零开始打造一个看起来结实的煎锅。

建造从一个圆形的钢铁圆盘开始，作为项目的空白。毛坯被旋转起来，外周边用角磨机磨薄，这看起来像是一个适度粗略的操作。然后用锻造来加热坯料，这样就可以用锤子把它做成平底锅。慢慢地，随着金属被不断地敲打，煎锅开始成形。砂带磨光机磨掉外部的高点，然后用火炬将锅的底部弄成方形，这样它就能很好地固定住。最后一个把手用一些坚固的铆钉连接，新形成的炊具用葵花籽油调味。

这个项目展示了手工制作简单的炊具需要多少特殊技能。很高兴看到一些人还保留着旧的方法。休息后的视频。
