# 八个马达肯定能泵出大量的水

> 原文:[https://hacka day . com/2020/09/29/八电机-能确保泵出大量的水/](https://hackaday.com/2020/09/29/eight-motors-can-sure-pump-a-lot-of-water/)

曾几何时，3D 打印更多的是一种好奇心，而不是一种强大的工具，有许多打印小饰品和小玩意，而不是任何真正有用的东西。然而，这些年来，随着技术和技巧的进步，我们现在看到更多的应用程序就绪的构建。[这个来自【让我们打印】的水泵就是一个很好的例子。](https://www.youtube.com/watch?v=IHZo5n1dCxE)

该泵由两个主要部分组成——驱动装置和叶轮。驱动装置由一个齿轮箱组成，该齿轮箱结合了八个电动机的动力，驱动一个单一的轴。这一切都是通过黑色外壳中醒目的黄色 ABS 齿轮实现的。构建视频很好地解释了如何使项目与不同的电机一起工作，以及如何正确使用螺栓调节器来设置齿轮系的齿隙。驱动单元然后被用来转动 3D 打印的叶轮泵，该泵能够非常快速地输送大量的水。

当启动时，泄漏的组件会发出可怕的噪音和巨大的混乱，但肯定会转移大量的水。当水从轴承中漏出时，看着水从齿轮上喷出是一个很棒的景象，很明显这个装置工作得很好。我们希望看到这种泵相对于商业产品的成本和性能分析。

虽然它肯定不是最坚固的构建，但它是一个有趣的完成工作的构建。我们很想看到这个[运行泡沫机](https://hackaday.com/2019/06/28/building-a-foam-machine-from-a-leaf-blower-and-a-water-pump/)或者经典的[滑行。](https://hackaday.com/2015/11/01/the-flying-man/)休息后的视频。
