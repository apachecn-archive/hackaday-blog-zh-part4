# 脊柱设计有助于模块化能量收集

> 原文：<https://hackaday.com/2018/07/17/spines-design-makes-for-modular-energy-harvesting/>

[SPINES(用于环境传感的自供电物联网节点)Mote](https://hackaday.io/project/159480) 是一种无线物联网环境传感器，但不要让整齐包装的单个 PCB 欺骗你认为它是不可黑客攻击的。[Macro Yau]特别设计了高度模块化的脊柱，以使设计能量收集传感器节点更容易。在[Macro]看来，发展有两大障碍:一个是能源采集本身，另一个是管理所采集的每一焦耳宝贵能源的使用所需的软件。

[Macro]设计了单板 SPINES Mote，使能量收集部分可以独立使用，并易于集成到其他设计中。此外，正在开发一个 Arduino 库，使电源管理更容易在幕后完成，使开发人员能够专注于应用程序本身。太阳能无线传感器节点是一回事，但帮助人们在这个过程中更快地实现他们的想法是很棒的。
