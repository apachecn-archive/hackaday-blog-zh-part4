# 水下无人机电影，正在拍摄中

> 原文：<https://hackaday.com/2022/01/22/underwater-drone-films-is-in-film/>

如今，拥有一架带摄像头、能跟踪你跑步或骑车的无人机并不是什么大新闻。但法国公司 Notilo Plus 有一架水下无人机，可以跟踪并拍摄水下潜水员。据《新地图集》报道， [Seasam](https://newatlas.com/drones/seasam-autonomous-underwater-drone/) 自 2019 年以来一直存在，但最近出现在法国电影 *The Deep House* 中，讲述了一对夫妇探索水下鬼屋的故事。你可以在下面的视频中看到一段关于无人机的视频，以及这部电影的预告片。

为了跟随潜水员，机器人使用来自用户控制单元的声音信号来找到用户的大致位置。这甚至在黑暗的条件下也能工作。一旦足够接近，计算机视觉就会锁定潜水员，同时声纳系统可以安全导航。

控制单元不能操纵无人机，但它可以做一些事情，如停止和开始记录或调整跟随距离。它还可以修改无人机的位置。然而，你可以通过平板电脑和长系绳电缆或通过使用 WiFi 连接到操作员的系留浮标从表面操纵它。视觉系统可以跟踪除潜水员以外的东西，所以无人机可以自动保持，比如说，移动船只的一部分在视野中。

不到 14，000 美元，你就可以买到一架 9 公斤重的无人机，它可以下降到 100 米，依靠电池运行约 90 分钟。除了其他可用的升级，还有电池升级可以将时间延长到四个小时。
