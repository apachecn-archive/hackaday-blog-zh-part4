# 一枚 25 美分的 PCB？

> 原文:[https://hackaday.com/2021/02/27/a-pcb-out-of-a-quarter/](https://hackaday.com/2021/02/27/a-pcb-out-of-a-quarter/)

随着时间的推移，PCB 组装公司已经深入到我们社区的可承受空间，可用的电路板类型成倍增加。我们不再局限于 FR4 和绿色阻焊膜，我们可以拥有彩虹的所有颜色和各种基板。然而，机器人工厂的人们已经用他们的 PCB 打印机更进一步，通过在四分之一上打印全功能 PCB。

作为基层，在印刷线路之前，在硬币上印刷五道绝缘层。绝缘层上留有孔洞，形成一种连接硬币的通路。板上是一个闪烁 LED 的 ATtiny2313 微控制器，硬币的反面是一个 CR2032 电池，用一组螺栓和垫圈固定。你可以在休息时间下面的视频中看到它正在成形。

诚然，LED 闪光灯并不令人兴奋，这是僵尸工厂打印机的营销噱头。但这是一个发明，它提醒我们，只要有点独创性，任何东西都可以成为一块板。这些年来我们已经有了自己的份额，立即浮现在脑海中的是这种可拉伸的 PCB。
