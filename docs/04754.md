# 一个适合维多利亚时代疯狂科学家的谢妮收音机闹钟

> 原文：<https://hackaday.com/2019/12/17/a-nixie-radio-clock-fit-for-a-victorian-mad-scientist/>

[Ioszelos]建造了一个谢妮时钟，拥有令人眼花缭乱的功能。

你是否曾希望可以通过手机登录时钟，然后关掉电视？我们认为是[Ioszelos]做的。该时钟还可以播放 MP3 和流式电台。它可以记录室内温度、湿度和气压。我们提到过这也是调频收音机吗？我们不确定，但如果有一个水龙头藏在这个装置的某个地方，我们也不会感到惊讶。

ESP32 和 Mega 2560 之间的团队合作运行了 Rube Goldberg-like show。定制电路板旋转起来，为[谢妮电子管提供所需的控制和电压。](https://hackaday.com/2019/10/30/multimeter-display-perked-up-with-nixies-leds-and-neon-tubes/)时钟由加工板和 3D 打印文件构成。

这一切都聚集在一个蒸汽朋克怀旧集会。RGB leds 和谢妮管发出的光结合在一起，形成了有趣的视觉效果。我们以前肯定从未见过像这样的钟。
