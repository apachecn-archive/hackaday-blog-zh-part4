# FBus:一种可扩展且易于配置的基于 FPGA 的 DAQ

> 原文:[https://hack aday . com/2019/11/01/fbus-an-extensible-and-easy-configured-based-FPGA-DAQ/](https://hackaday.com/2019/11/01/fbus-an-extensible-and-easily-configurable-fpga-based-daq/)

[flow]对商业数据采集系统(DAQ)有点失望，他将自己的挫败感转化为自己非常酷的基于 FPGA 的解决方案。

该项目采用背板的形式，各种卡可以插入其中。接口只是一个 PCI-e 连接器。如果您需要模拟输入，只需插入卡即可。以太网输出？同样的过程。模块化和可扩展性是这里的主题。

[flow]已经为他的新 DAQ 设计好了项目。他用它来建造一个倒立摆。然而，他设计的卡真正展示了可能性，从逻辑分析仪卡到 HDMI 输出卡，任何东西都允许广泛的配置。还有一个小的工具套件使得这个过程相对容易管理。

我们可以肯定地看到，这将成为我们开发原型的有用工具。
