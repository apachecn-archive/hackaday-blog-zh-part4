# Hackaday 数据记录竞赛的获胜者:蓝牙园艺、数杯子和预测降雨量

> 原文:[https://hack aday . com/2021/05/01/winners-of-hack adays-data-loggin-contest-bluetooth-gardening-counting-cups-and-predicting-rain/](https://hackaday.com/2021/05/01/winners-of-hackadays-data-loggin-contest-bluetooth-gardening-counting-cups-and-predicting-rainfall/)

已经收到了对 [Hackaday 的数据记录竞赛](https://hackaday.io/contest/176306-data-loggin-contest)的投票，保存到 SD，推送到 MQTT，并绘制成图表。现在是时候宣布三个项目了，它们从生活的随机数据中获得了最大的意义，并为 Tindie 赢得了 100 美元的礼券，Tindie 是互联网上最重要的手工制作电子产品供应商。

首先，*数据向导*类别的获胜者，是由【Joseph Eoff】设计的[全园土壤湿度监测器。你可能没有从页面顶部的图片中意识到这一点，但在那个可爱的花园的覆盖物下隐藏着 20 多个蓝牙土壤传感器，它们以网格模式排列。所有的数据都被一系列太阳能供电的 ESP32 接入点吸收，并最终通过 MQTT 在 Raspberry Pi 上结束。在这里，定制的 Python 软件生成一个热图，指示花园中可能的麻烦点。凭借其易于理解的对表面下发生的事情的可视化，该项目完美地抓住了该类别的精神。](https://hackaday.io/project/178004-soil-moisture-monitoring-in-a-flower-garden)

[![](../Images/defc08b4c1092330adf272f7865a5246.png)](https://hackaday.com/2021/05/01/winners-of-hackadays-data-loggin-contest-bluetooth-gardening-counting-cups-and-predicting-rainfall/loggin_coffee/)[![](../Images/f1c277d567454fa780e38cdb68596ba9.png)](https://hackaday.com/2021/05/01/winners-of-hackadays-data-loggin-contest-bluetooth-gardening-counting-cups-and-predicting-rainfall/loggin_weather/)

接下来是来自[斯蒂德曼] 的 [Nespresso 盾。这个聪明的小工具真的可以听到同名咖啡机发出的声音，不仅可以估计你每天的消耗量，还可以在机器缺水时警告你。从家用电器中提取数据的聪明的非侵入性方法使它成为*创意天才*类别的有力竞争者。](https://hackaday.io/project/179360-nespresso-shield)

最后但肯定不是最不重要的是这个综合的[物联网气象站，它使用机器学习来预测降雨量](https://hackaday.io/project/178000-iot-weather-station-predicts-rainfall-intensity)。随着农作物和牲畜面临突然强烈风暴的威胁，[kutluhan_aktar]设想这种设备可以为农民提供早期预警。从设置支持 GPRS 的 ESP8266 气象站到创建 web 界面并将所有数据导入 TensorFlow，这个项目的文档绝对是非凡的。这个项目为类似的 DIY 天气探测和预测系统提供了一个宝贵的框架，这使得它成为我们*世界改变者*类别的完美选择。

这一次可能只有三个获胜者，但 Hackaday 社区的传奇技能和创造力在这场比赛中得到了充分展示。[强烈建议浏览其余的提交内容](https://hackaday.io/submissions/data-loggin/list)，我们相信创作者会很乐意在评论中听到您的反馈和建议。
