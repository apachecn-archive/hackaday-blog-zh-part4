# 用乐高积木搭建最棒的键盘

> 原文:[https://hack aday . com/2020/04/24/build-the-bad dest-keypad-on-the-block with-Lego/](https://hackaday.com/2020/04/24/build-the-baddest-keypad-on-the-block-with-lego/)

像我们中的许多人一样，[EducatedAce]一直在通过复活旧项目和寻找新挑战来平息隔离的忧郁。他刚刚完成了这个块状宏键盘的制作，用来保存 Photoshop 的一堆快捷方式，从而延续并混合了创作的狂欢。

[![](../Images/b52029fd0b348de06594b84038731c1d.png)](https://hackaday.com/wp-content/uploads/2020/04/lego-macro-guts.png)【EducatedAce】手头已经有了除 Arduino Micro 以外的一切。这个宏块没有使用标准的按键开关，而是使用了 16 个最响、最脆的触觉按钮——那些黄色茎干的大按钮听起来像小订书机。

不要担心，在制作这个宏垫的过程中没有乐高或乐高配件受到伤害，基板和开关板是 3D 打印的。[EducatedAce]发布了 STL 文件和很棒的构建说明，如果你想自己组装一个的话。

这是一个很棒的项目，因为它很坚固，不用花很多钱就能完成工作，而且看起来仍然像是你想要放在办公桌上的东西。[EducatedAce]计划用统一颜色的砖块重建它，但我们认为它看起来很棒，尤其是那些通风的 1×2 块。如果是我们的，我们可能会为每一行或每一列使用不同的颜色，以帮助保持快捷方式的直线性。
