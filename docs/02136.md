# 无线上网让心更温暖

> 原文:[https://hack aday . com/2019/03/02/wifi-makes-the-heart-glow-fonder/](https://hackaday.com/2019/03/02/wifi-makes-the-heart-glow-fonder/)

今年过情人节有点太晚了，但是如果你现在就开始，明年二月你一定会看起来很棒。通过无线通信和 RGB led(Youtube，法语)的魔力，打印出真实传达您感受的东西。

是的，【Heliox】打造了一个情人节项目，呈现方式一如既往的顶尖。一颗心脏是用白色细丝 3D 打印出来的，有两个被厚厚的墙隔开的心室。每个试验箱配有五个新像素 led，由底座中的 ESP8266 控制。每个房间的颜色可以通过 Blynk 智能手机应用程序控制，让你选择最能代表你们关系的确切颜色。

3D 打印的心脏在扩散 led 方面做得很好，设备显示出丰富而一致的发光，没有任何不美观的热点。这是一个有趣的假日建筑，如果你很快，如果你马上开始，你可能有时间自己打印一个。

[Heliox]在 LED 项目方面表现出色，她的 infinity cube 尤其令人印象深刻。休息后的视频。
