# [汤姆·斯坦顿]造了一架鱼鹰

> 原文：<https://hackaday.com/2022/10/08/tom-stanton-builds-an-osprey/>

V-22 鱼鹰是一种与众不同的飞机。倾转旋翼机多用途军用飞机无论走到哪里都会给人留下深刻印象；巧合的是，昨天有两架这样的飞机从头顶飞过，发出了我们从未听过的声音。这是一架推动工程极限的复杂飞机，所以很自然地[汤姆·斯坦顿]决定为自己建造一架精确控制飞行的鱼鹰遥控模型。

眼尖的读者无疑会注意到[Tom]最近建造了一个类似鱼鹰的垂直起降模型来探索倾转旋翼设计的基础。但他这次建造的目标是超越基础，通过复制全尺寸鱼鹰的一些控制复杂性，而不需要倾家荡产。汤姆没有建造或购买真正的旋转斜盘来控制旋翼的总距和周期距，而是使用了[的“虚拟旋转斜盘”技术](https://hackaday.com/2020/06/25/building-and-flying-a-helicopter-with-a-virtual-swashplate/)，该技术利用成角度的铰链和马达角动量的快速变化来实现叶片距控制。有趣的是，在每个转子上增加第三个叶片后，相同的机制也在工作，以模仿鱼鹰的叶片——我们本以为这会让整个事情失去平衡。诚然，机身存在一些共振问题，但[汤姆]能够克服它们，实现接近稳定的飞行。

下面的视频只是他的构建系列的第一部分，但我们怀疑包含了大多数有趣的工程位。尽管如此，我们还是期待看到控制机制如何随着设计过程的继续而发展。
