# 五轴南瓜雕刻

> 原文：<https://hackaday.com/2020/10/29/five-axis-pumpkin-carving/>

雕刻南瓜的日子就要到了，不再像普通人那样手工制作，这里制造的谢恩制造了一台五轴数控机器来接管雕刻任务。(视频，嵌在下面。)

[Shane]最初打算修改他的[理发机器人](https://hackaday.com/2020/07/14/a-robotic-stylist-for-your-lockdown-lengthened-locks/)，但最终进行了彻底的重新设计，仅重复使用了电子设备和底座中的大型环形轴承。旋转主轴是一个旋转门架，带有两套铝型材，用于垂直和水平运动。这个门架不是很坚固，但是对于南瓜雕刻来说已经足够好了。由于五轴运动和将 2D 图像映射到一个大致球形的表面上的复杂性，软件是这项工作中最具挑战性的部分。制图员已经处理这个问题很长时间了，所以[谢恩]转向[墨卡托投影](https://en.wikipedia.org/wiki/Mercator_projection)来解决这个问题。我们也很欣慰地听到，我们不是唯一一个有时与充满等式的维基百科页面斗争的人。

由于没有完美的球形南瓜，[Shane]写了一个脚本，在切割前用微动开关探测南瓜的表面，恰当地命名为“TSA.exe”。这台机器既能雕刻侧面，也能雕刻不同深度的石像，大多是(谢恩)长期受苦的妻子的。她真的应该因为保持幽默感而获奖。

像[爆炸棒球棒](https://hackaday.com/2020/10/27/going-for-the-home-run-record-with-explosive-help/)和[数控篮球架](https://hackaday.com/2020/09/09/third-times-a-charm-for-this-basketball-catching-robot/)这样的项目,【这里制造的东西】YouTube 频道值得关注。
