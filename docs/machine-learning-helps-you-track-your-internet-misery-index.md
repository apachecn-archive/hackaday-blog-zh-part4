# 机器学习帮助你追踪你的互联网痛苦指数

> 原文：<https://hackaday.com/2021/02/14/machine-learning-helps-you-track-your-internet-misery-index/>

我们似乎都直觉地知道，我们在网上做的很多事情对我们的心理健康没有好处。在足够多的社交媒体平台上闲逛，你几乎可以感受到你的大脑对你的身体造成的变化，这是你所看到的结果——心跳加速，面部表情紧张，愤怒地握紧拳头。当然，不是在 hack aday——这里只有甜蜜和光明。

当然，这都是非常主观的。如果你想更客观地量化你在网上的痛苦，看看名副其实的 [BrowZen](https://hackaday.io/project/177613-browzen) ，这是一个由【Nick Bild】开发的机器学习应用。围绕 NVIDIA Jetson Xavier NX 和网络摄像头构建的 BrowZen 定期捕捉用户面部的图像。使用面部识别模型对用户面部的表情进行分类，该模型经过训练，可以识别与愤怒、惊讶、恐惧和快乐等情绪相关的面部姿势。该应用程序可以捕捉你的情绪和你当前正在浏览的网站，并将结果存储在数据库中。方便的图表让你知道哪些网站最适合你的精神状态；当 Hackaday 按下[Nick]的快乐按钮时，Twitter 引发愤怒并不令人惊讶。看到了吗？甜蜜和光明。

说真的，我们可以看到这样的东西对心理测试、市场研究，甚至医学评估非常有用。这增加了[尼克]的一系列人工智能应用，从[跟踪你在房间里触摸的表面](https://hackaday.com/2020/04/07/machine-vision-keeps-track-of-grubby-hands/)到[防止你在视频会议上犯下可解雇的罪行](https://hackaday.com/2020/06/03/machine-learning-takes-the-embarrassment-out-of-videoconference-wardrobe-malfunctions/)。
