# 会说话的狗阿尔玛可能会赢得一些酒吧赌注

> 原文:[https://hack aday . com/2019/03/16/alma-the-talking-dog-may-win-some-bar-bets/](https://hackaday.com/2019/03/16/alma-the-talking-dog-might-win-some-bar-bets/)

伊利诺伊大学厄巴纳-香槟分校的学生拥有一个可以测量脑电波的脑机接口。他们用它做了什么？他们把它给了 Alma，一只金色的拉布拉多，你可以在下面的视频中看到。复制电子设备的代码和足够的信息在 GitHub 上。

当然，狗不会直接产生语音。相反，电路通过 Arduino 观察她的脑电波，并将原始数据输入树莓 Pi。一种机器学习算法可以确定 Alma 的脑电波状态，并播放预先录制的表达 Alma 思想的音频。

阿尔玛的衣领在某种程度上复制了电影《飞屋环游记》中虚构的衣领。当然，道格更健谈一些。从视频中看不出这个程序对多少个州进行了分类。快速浏览一下代码，会发现有五个音频片段，但只有一个似乎与识别器连接在一起——那是为了款待客人。我们认为这可能是一个更难弄清楚的问题，当狗不想要一个治疗。

上一次我们看到会说话的狗项圈是由电话控制的。如果你真的想探测一个大脑——犬类或人类——你可以做得比查看 [OpenHardwareExG](https://hackaday.com/2014/08/13/an-open-hardware-platform-for-ecg-eeg-and-other-measurements/) 更糟。
