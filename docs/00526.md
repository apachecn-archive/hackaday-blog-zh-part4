# 运行在金字塔能量上的环氧树脂嵌入式电子艺术

> 原文：<https://hackaday.com/2018/09/02/epoxy-embedded-electronic-art-running-on-pyramid-power/>

我们有时会从艺术中获得灵感。当[kodera2t]看到一些嵌在透明环氧树脂中的日本鱼图艺术时，他不得不自己制作。虽然他精通电子学，但他不擅长绘画。不过，在看到他在水晶般透明的环氧树脂中嵌入的电子产品后，我们仍然称他为艺术家。

他的第一批电子艺术作品是几个晶体管和一些集成电路，包括一个 80386，封装在环氧树脂中。但是后来他意识到他想让电子设备做一些有趣的事情。然而，一旦封装在环氧树脂中，你如何保持电子设备永远供电？

他尝试用太阳能电池给电池充电，然后给 LED 供电，但他不喜欢长时间用环氧树脂包裹化学电池的想法。

然后，他转而使用环氧树脂金字塔底部的接收线圈进行无线能量传输。其中一个线圈为 BLE 电路板供电，电路板上有一个 LED，他可以通过手机控制。他的最新产品包括一台带有机发光二极管显示屏的 ESP32-PICO。该代码允许他通过无线方式上传新固件，但在他的 Hackaday.io 页面上，他展示了可以构建 ESP32 的代码与不能构建 ESP32 的代码之间的差异。但是不要相信我们的话。看看下面的视频，亲自看看他的艺术。

虽然在环氧树脂中嵌入电子设备对[kodera2t]来说是新的，但我们以前已经见过几次了。一次是以[的形式，一个做得很漂亮的放大器电路，死虫式的](https://hackaday.com/2012/04/13/free-formed-circuit-protected-by-a-brick-of-crystal-clear-resin/)，还有一个更具[的太阳能灯的实验性尝试](https://hackaday.com/2017/03/01/using-a-jewelry-kit-to-resin-encase-electronics/)。
