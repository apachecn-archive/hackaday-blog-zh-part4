# 为什么你不能做一个有透明有机发光二极管的可穿戴显示器

> 原文:[https://hackaday . com/2021/08/19/why-you-can-make-build-a-wearable-display-with-a-just-a-transparent-OLED/](https://hackaday.com/2021/08/19/why-you-cant-make-build-a-wearable-display-with-a-just-a-transparent-oled/)

在看到最近上市的廉价透明有机发光二极管显示器后，你可能会想到用它们来构建自己的可穿戴显示器。为了避免这种建造不可避免的失望，[扎克·弗里德曼]亲自测试了这个想法，并展示了为什么[透明可穿戴显示器比它看起来更难](https://www.youtube.com/watch?v=zZZ-Ea8RyjQ)。

他组装了一个带有集成微控制器的头带，将透明有机发光二极管固定在用户的眼睛上，但不幸的是，显示器上显示的任何东西最终对佩戴者来说都或多或少是不可见的。正如[Zack]在休息后的视频中解释的那样，人眼在生理上无法聚焦于如此短距离的任何物体。 与许多人可能认为的相反，可穿戴显示器的难点不在显示器本身，而是光学元件。为了让可穿戴显示器工作，显示器发出的所有光束都需要通过透镜和/或反射镜聚焦到你的眼球上，而不会扭曲你对透镜以外一切的看法。这就需要重量轻、无失真的准直器和分束器，它们价格昂贵且难以制造。

虽然这些透明的 OLEDs 可能不会成为实用的平视显示器，但对于像[立体显示器](https://hackaday.com/2021/03/04/volumetric-oled-display-shows-bladerunner-vibe-curious-screen-tech/)这样的项目来说，它们仍然是很酷的一部分。当然，打造你自己的[智能眼镜](https://hackaday.com/2021/08/14/3d-printed-smart-glasses-put-linux-in-your-face/)或[增强现实眼镜](https://hackaday.com/2021/03/28/triton-ar-headset-blends-stock-and-printed-parts/)是有可能的，你只需要*专注于*正确的光学系统。
