# 法国对电子工作台组织的明确转变

> 原文：<https://hackaday.com/2020/05/18/a-french-cleat-twist-on-electronics-bench-organization/>

对我们中的一些人来说，我们的工作台是组织走向死亡的地方。准备着手一个新项目意味着扫除一堆旧项目，暴露出正好足够的工作台空间来放下新部件。另一个极端是那些认为组织不是达到目的的手段，而是目的本身的人。他们的长椅一尘不染，随时准备接受一个新项目。

[Eric Gunnerson]的新[法式夹板电子工作台](http://www.riderx.info/french-cleat-electronics-workbench/)介于这两个极端之间，尽管远没有启发它的木工组织者那样组织过度。如果你从来没有听说过法国夹板，谷歌一下，你会看到[一些令人惊叹的商店](https://hackaday.com/2019/09/18/tiny-woodshop-is-packed-with-space-saving-hacks/)，在那里，从货架到橱柜，墙上安装的斜接夹板系统和配套部件得到了很好的利用。一个适当建造的法国楔子可以支撑巨大的负荷；[Eric]的系统按比例缩小了一点，以适应电子商店中常见的较轻负载。他的夹板是 2 英寸×3 英寸的松木片，固定在一张胶合板上，然后用螺丝固定在墙上。他第一次使用[的牛头刨床数控刳刨机](https://hackaday.com/2016/08/25/hands-on-the-shaper-origin-a-tool-that-changes-how-we-build/)来加工夹板，但当证明速度很慢时，他转向了激光切割胶合板。下面的总结视频展示了他到目前为止想到的一些固定装置；我们特别喜欢示波器盒子，电缆挂钩也是一个巧妙的技巧。

我们喜欢它提供的灵活性，因为您可以随着工作流程的发展或新仪器的增加而改变。给[Eric]一个没有过度复杂化的组织。
