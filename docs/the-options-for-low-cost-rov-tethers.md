# 低成本水下机器人系绳的选择

> 原文：<https://hackaday.com/2020/01/27/the-options-for-low-cost-rov-tethers/>

无线连接很酷，但有时你只需要一点铜缆。这篇关于 SV Seeker 的有趣文章讨论了[制作遥控潜水器(ROV](http://www.svseeker.com/wp/sv-seeker-2/underwater-rovs/open-source-towed-sonar-rov/winch-cable/) )系绳的各种方法。他们试验了许多不同的电缆，包括为掩埋而设计的凝胶填充的 Cat 5，并将电缆包裹在聚丙烯绳中，以保持其受保护和漂浮。他们还考虑使用一根单芯实心同轴电缆，在电缆两端包裹一个以太网同轴转换器。使用同轴电缆的好处是长度:它可以处理超过一英里的电缆，对于这个项目来说应该绰绰有余。不利的一面是，他们发现同轴电缆在压力下会伸长，扰乱信号。

该项目目前似乎处于停顿状态，但对于任何希望远距离发送数据的人来说，这里有许多值得思考的东西。我想在我后院的猫头鹰盒子里放一个网络摄像头，我没有遇到过他们在这里涵盖的一些选项，所以我将自己寻找直接埋葬评级 Cat 5。
