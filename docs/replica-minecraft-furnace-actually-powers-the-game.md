# 复制品《我的世界》炉实际上为游戏提供动力

> 原文：<https://hackaday.com/2022/03/05/replica-minecraft-furnace-actually-powers-the-game/>

让我们面对现实吧，我们有时都需要一点分心，尤其是最近。对于我们的钱来说，没有比开始一个*《我的世界》*世界并开始挖掘更好的方法了。简单的图形，开放的世界，以及除了寻找资源和构建东西之外的议程的缺乏都非常令人欣慰。

但是玩游戏玩久了，你肯定会想如果游戏世界穿越到现实世界会是什么样子。具有讽刺意味的是，这位名叫[迈克尔·皮克]的人就是这样做的，他设法制造了一个真正的*《我的世界》*熔炉，可以为游戏提供能量。当然，游戏中熔炉的制作方法有一些自由，这对于一个允许你赤手空拳击打树木来砍倒它们的游戏来说是可以理解的。

迈克尔没有用八块鹅卵石来建造他的炉子，而是用木头做了一个商用折叠野营炉的外壳。通过一点水泥板与外壳绝缘，熔炉看起来与游戏中的物品相当真实。为了产生运行游戏所需的电力，他使用了一对[热电野营发电机](https://www.amazon.com/Ajirangi-Portable-Thermoelectric-Generator-Camping/dp/B00YPKY1ZQ)。炉子里装满了木头——大概是未打孔的——发电机产生的电力足以至少为一个电池组部分充电，然后用来为一个树莓派和 7 英寸显示器供电。他的目标是从熔炉中获得足够的能量，在游戏中进行速度奔跑，并找到三颗钻石来建造钻石镐。老实说，我们很嫉妒——我们的第一批钻石从来没有这么容易得到。

我们以前见过其他的*《我的世界》* -IRL 跨界车。想坐矿车兜风吗？[我们已经搞定了](https://hackaday.com/2022/01/10/this-little-minecraft-mine-cart-of-mine/)。或者你更愿意在游戏中控制台灯？[那也是个东西](https://hackaday.com/2015/11/25/the-internet-of-minecraft-things-is-born/)。
