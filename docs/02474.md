# 精心设计的定制 LoRa 寻呼机

> 原文:[https://hack aday . com/2019/03/24/custom-lora-pager-designed-with-care/](https://hackaday.com/2019/03/24/custom-lora-pager-designed-with-care/)

如果您需要在低功率水平下长距离发送数据，LoRa 是一项有用的技术。利用这一点，[[5v 迷]决定制造一个名为 LoRaNicator 的小型寻呼机。](https://hackaday.io/project/163604-loranicator)

那些喜欢详细构建日志的人会喜欢这个。传呼机功能齐全，包括厨房水槽。一个 Cortex M0+运行显示，闪存与 Arduino 兼容的引导加载程序，而 RFM95W 模块处理 LoRa 通信。有一个寻呼机振动器和压电蜂鸣器通知，随着脂肪充电器，使保持电池充满容易。甚至还有一个 RTC 和软电源按钮模块。

即使 LoRa 方面的东西与您的兴趣无关，它也是一个很好的例子，说明如何构建一个有用的工具，而不仅仅是一个概念证明。像易于使用的界面和简单的电池充电这样的东西对使一些东西在野外可用大有帮助。[5v 迷]甚至指出，即使是阻焊膜也很重要——如果你使用红外线烤箱，你的黑色电路板将需要与通常的绿色 PCB 不同的轮廓。

所有的辛勤工作都得到了回报，创造了一个有吸引力的最终产品，我们会很自豪地与我们的火腿齿轮包装在一起。LoRa 是一个有用的平台，正如我们所见，它可以用于从葡萄栽培到 T2 气象的方方面面。休息后的视频。
