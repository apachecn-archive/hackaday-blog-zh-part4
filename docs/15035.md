# 放松，好好享受一口牙齿

> 原文:[https://hack aday . com/2022/10/09/放松一下，吃一杯美味的牙齿吧/](https://hackaday.com/2022/10/09/relax-and-have-a-nice-cup-of-teeth/)

万圣节就要到了，还有什么比 3D 打印[DaveMakesStuff]的[牙齿杯](https://thangs.com/designer/DaveMakesStuff/3d-model/Teeth%2520Cup-377780)更好的方式来添加一点怪异的装饰呢？

看起来[DaveMakesStuff]已经完成了相当于“套件敲打”的工作，它采用了一整套牙齿的 3D 模型，并将它们与茶杯融合在一起。细节很简单，但是一个推特帖子 ( [Nitter](https://nitter.net/FogleBird/status/1577801773363990530) )提供了一些关于这个过程的线索。杯子看起来可以一次打印完成，支持免费。光滑的表面来自喷珠处理，在这种情况下，喷珠处理作为额外的奖励，提供了“脏”的外观，因为喷珠器通常只用于尼龙 SLS 印花。

万圣节总是黑客项目灵感的一个很好的来源，我们已经从过去的[万圣节黑客节](https://hackaday.com/2021/11/04/spooky-winners-of-the-halloween-hackfest-contest/)中获得了许多好的参赛作品。
