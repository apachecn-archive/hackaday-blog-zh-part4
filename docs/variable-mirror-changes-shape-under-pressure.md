# 可变反射镜在压力下改变形状

> 原文：<https://hackaday.com/2020/06/27/variable-mirror-changes-shape-under-pressure/>

除非你在嘉年华游乐场，否则镜子通常都是单调乏味的。加入一些曲线，事情变得有趣起来，尤其是当你可以用手指触摸来控制曲线时，就像[这种可变表面凸面镜](https://www.youtube.com/watch?v=WIl6FlUCHQg)。

下面的视频以一个很长但很有用的圆锥常数回顾开始，以及平面横切圆锥如何根据平面的角度创建圆、抛物线或椭圆。正如[惠更斯光学]解释的那样，每种形状的镜子都有不同的属性，这使得建造天文和地球距离的望远镜变得困难。为了制作一面可以在很大距离范围内工作的镜子，[惠更斯光学公司]用两片玻璃粘合在一起，在前后表面之间形成一个空间，制成了一面镜子。研磨成球形轮廓的前表面可以通过用注射器抽空两个表面之间的压力通风系统而轻微变形。大气压使较薄的前表面轻微弯曲，改变了镜子的形状。

惠更斯光学公司还建造了一个干涉仪，将可变反射镜与已知的球面参照物进行比较。来自干涉仪的数据被输入到一个可视化软件包中，该软件包可以生成表面形状的地图，当镜子内部的压力发生变化时，你可以很容易地看到表面形状的变化。唉，对数据的深入研究表明，镜子并不完美，但令人着迷的是，一面镜子可以弯曲到足以在只需一股空气的情况下从椭圆形变成几乎抛物线形。

我们之前已经看到过[惠更斯光学]的一些有趣的努力，包括下一级的水平仪。然而，他并不完全是在磨玻璃——见证[对金属探测器甄别的调查。](https://hackaday.com/2019/11/26/progressive-or-thrash-how-metal-detectors-discriminate/)
