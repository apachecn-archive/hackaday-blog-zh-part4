# 黏糊糊的球上的机器人抓手

> 原文：<https://hackaday.com/2021/05/04/robotic-gripper-from-a-squishy-ball/>

软机器人抓取器有一些有趣的用例，但工业选项并不便宜。[詹姆斯·布鲁顿]被帝国机器人公司价值 4000 美元的“豆袋”抓手迷住了，所以他决定[建造自己的](https://www.youtube.com/watch?v=LcuBRQtVGxk)。

抓手只是一个充满小珠子的弹性橡胶膜。当它被推到一个物体上，空气被吸出，它将所有的珠子聚集在一起，塑造成物体的形状。对于他的版本[詹姆斯]使用了一个充满 BBs 的软橡胶球。为了创造真空，他通过软管将一个 200cc 的大注射器连接到球上，并用高扭矩伺服系统驱动它。

它对小而轻的物体很有效，但对较重、光滑且没有边缘的物体无效。如果减小珠子/BBs 的尺寸和重量，这可能会得到改善。

对于一些更软的机器人，看看这个[软 3D 打印手](https://hackaday.com/2020/01/19/experiments-in-soft-robotics/)，和[柔性电动致动器](https://hackaday.com/2019/10/28/electricity-makes-soft-robotics-more-like-us-meatbags/)。
