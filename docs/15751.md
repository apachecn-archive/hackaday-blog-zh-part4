# 适用于桌面 Mac 的独立触控 ID

> 原文:[https://hack aday . com/2022/12/26/standalone-touch-id-for-your-desktop-MAC/](https://hackaday.com/2022/12/26/standalone-touch-id-for-your-desktop-mac/)

随着移动设备生物识别技术的普及，在桌面上输入密码已经过时了。[时髦实验室]决定为他的 Mac 解决这个问题，将触控 ID 从苹果新键盘上解放出来。

当苹果推出自己的台式机芯片时，它还展示了包含触控 ID 指纹识别系统的台式机键盘。指纹访问电脑很方便，但并不是每个人都喜欢苹果键盘上的打字体验。为了避免在桌子下面贴键盘，[时髦实验室]从键盘上取下了逻辑板，并为触控 id 按钮和逻辑板设计了一个新的 3D 打印外壳，这样指纹读取器就可以靠近用户手的实际位置。

发现的一个有趣的细节是标准版本和包含 numpad 的版本之间显著不同的逻辑板。最终的机箱设计具有无线和有线两种版本，适用于标准和 numpad 主板，如果您选择构建自己的主板的话。我们很有兴趣看看是否有人可以采取这一下一步，使用逻辑板来连接触控 ID 定制的机械键盘。

如果[时髦实验室]看起来很熟悉，你可能会从他们的[Mac Mini Mini](https://hackaday.com/2022/08/15/mac-mini-mini/)中认出他。如果你更想把你的安全提升到极致，看看这个[四因素生物识别锁箱](https://hackaday.com/2016/12/13/all-i-want-for-christmas-is-a-4-factor-biometric-lock-box/)，它有自己的指纹识别器。
