# 艾琳四处走动

> 原文:[https://hackaday.com/2019/11/23/irene-goes-around/](https://hackaday.com/2019/11/23/irene-goes-around/)

定时拍摄装置非常棒，因为你可以用更有趣的平移和倾斜定时拍摄来增加视频的趣味。那么，为什么不给自己造一个呢？这就是[td0g]决定做的，结果是 [IREnE](https://td0g.ca/2018/09/23/camera-slider-2-0-irene-diy/) ，一个相当不错的自制 3DOF 装备，可以安装在标准三脚架上。3DOF 意味着它有三个自由度:相机可以在三脚架上旋转，在远离云台的延伸臂上线性移动，以及绕着自己的轴旋转。换句话说:它可以平移一个对象，同时旋转相机以保持对象在帧中居中。

IREnE 代表反向径向延伸 Eggtimer，这是一部关于该设备的双重径向性质以及摄影师如何使用 egg timer 进行这种事情的戏剧。这也是对夏洛克·福尔摩斯的敌人的偷偷致敬。该钻机由三个 NEMA 17 电机和一个 ATMega328p 驱动，全部由一个 Dewalt 电动工具电池和他自己的 [DeWatt 电源适配器](https://www.thingiverse.com/thing:2944892)供电。该钻机还有一个次要功能，即[煎饼打印机](https://td0g.ca/2018/12/15/cnc-pancake-printer/)。

除了早餐，这个项目还有一些注意事项。虽然三脚架可以很好地稳定相机，但像这样抵消重量会使三脚架不稳定。[td0g]确实增加了一些焊接稳定杆，使它更加稳定，但整个事情应该小心使用。相机位于一个 1 英寸见方的铝挤压机上，[td0g]声称它足够坚固，可以支撑他的佳能 D7，但我不确定我会把它和我昂贵的设备放在一起。

这是我们从[td0g]中看到的第五个高质量构建:我们之前覆盖了[优秀的高速 LED 闪光灯](https://hackaday.com/2019/05/16/an-impeccably-designed-high-speed-led-flash/)、[大望远镜座](https://hackaday.com/2019/01/12/diy-guided-telescope-mount-tracks-like-a-barn-door/)、[高速计时器](https://hackaday.com/2017/11/14/high-speed-chronograph-looks-like-pro-gear/)和[燃木 ATX 电源黑客。](https://hackaday.com/2017/11/02/diy-pyrography-power-supply/)
