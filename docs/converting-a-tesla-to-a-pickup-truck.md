# 将特斯拉改装成皮卡

> 原文：<https://hackaday.com/2019/10/19/converting-a-tesla-to-a-pickup-truck/>

著名的无用机器人发明者西蒙·吉尔兹这次超越了自己。她和一个由[Rich Rebuilds]、[Laura Kampf]和[Marcos Ramirez]组成的工程师团队最近决定[将一辆特斯拉改装成一辆皮卡](https://www.youtube.com/watch?v=jKv_N0IDS2A)，并在途中制作一个视频，同时尽可能地回收汽车后部的剩余部分，使最终产品适合上路行驶。是的，这是几个星期前的事了，是的，这是一个商业广告，但是真的:[西蒙·吉尔兹]和公司摇滚。

在她的体验视频中，团队从掏空汽车内部开始，以找出重量分布和外部框架的形式。从本质上讲，为了制造皮卡，需要拆除汽车后部的一部分，并根据随后的结构完整性焊接额外的横梁和支架。使用锯子和角磨机，框架的顶部被切割并取出，但在此之前，令人担忧的一瞥使人意识到汽车在改装过程中需要外部支撑。

在垫子、玻璃、电线和所有其他配件被移除后，他们安装了另一辆牺牲皮卡的卡车床，以及车顶行李架来完成外观。在解构和重建的过程中，有时汽车会遇到*“安全约束系统故障”*，或者团队在拍摄视频前意外地在汽车内部涂上玻璃纤维。在抱怨使用的不同剪辑大小和项目的明确时间压力之间，这是一个有趣和翔实的观察到一个非常独特的汽车模型。

他们为他们的特斯拉皮卡混合动力车拍摄的最后一个广告，名为 Truckla，可在[Giertz]的 YouTube 频道上看到。
