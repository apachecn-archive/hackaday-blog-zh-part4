# 任何人都可以构建的屏蔽净化

> 原文：<https://hackaday.com/2022/02/28/mask-sanitization-that-anyone-can-build/>

我们已经看到了各种各样的口罩消毒解决方案，现在，来自[Robots Everywhere] [的[spiritplumber]为我们带来了一个节俭而巧妙的设计](https://www.youtube.com/watch?v=ldZ9qUC-Kjw)——一个你甚至几乎不需要工具的设计。这个项目可能看起来很粗糙，但外观从来不是先决条件，正如一个称职的黑客会认识到的那样，这是对“如何设计一个任何人都可以建造的面具消毒器”的回答。

当地口罩的短缺一直威胁着这里和那里的社区，如果你需要某种可能缺货的口罩，这种威胁就更大了。这种设计也可以应用到很多其他需要消毒的地方——毕竟，改进概念是我们最喜欢的消遣。

这个设计很简单——一个电池供电的马达在一桶浓 H2O2 中旋转一个面具，通过一个便宜的超声波雾化装置变成薄雾。当“转盘”转动你选择的个人防护装备时，确保每个缝隙都用过氧化氢清洁过，这也会导致 H2O2 雾循环。这种方法满足了更复杂的设备一直在努力解决的对适当的消毒系统的最重要的要求，当然也赢得了它的一席之地。

[机器人无处不在] [与我们所有人分享了他们 DIY PPE 资源的小图书馆](https://www.robots-everywhere.com/re_wiki/pub/web/Cookbook.Covid19.html)，这还不是他们工作的全部——最近，我们看到了他们的[雾培项目使大蒜复壮](https://hackaday.com/2022/01/16/aeroponic-cell-grows-garlic-forwards-cellsol-packets/)。

到目前为止，使用过氧化氢蒸汽进行个人防护设备消毒是一种经过充分测试的方法，因为我们已经[看到它在 2020 年大规模部署](https://hackaday.com/2020/04/02/fda-says-ppe-can-be-reused-after-trip-through-shipping-container-decontamination-system/)作为 FDA 批准的设计的一部分。如果你手头只有 3%的过氧化氢，[不妨试着进一步浓缩](https://hackaday.com/2013/02/03/making-high-concentrations-of-hydrogen-peroxide-at-home/)！
