# 接近投放地点:种子离开！

> 原文：<https://hackaday.com/2020/09/04/approaching-the-drop-location-seeds-away/>

植树节是许多国家致力于植树的节日，但随着气候变化的不断侵蚀，我们需要最大限度地利用我们的时间。 [Dronecoria 不只是种一棵树](https://hackaday.io/project/172049-dronecoria-drone-for-aerial-reforestation)；它能在几分钟内播种“公顷”一公顷等于 10000 平方米或 2.471 英亩。如果你打算周末园艺，这些不是你要找的无人机，这是在用树木重新填充森林或用野花重振公园的范围内。料斗中的种子球是 10 公斤的天然种子，加上有益的微生物，以帮助每一滴的机会。

这架无人机的机身是由看起来像波罗的海桦木胶合板的东西激光切割而成的。这些矢量文件在 Illustrator(.ai)和 CAD(。dxf)格式，所以如果你重新发布或重新混合它，请注明出处。在 3D 领域，你需要一个种子快门和种子分散器，这两个模型都有 STL 格式。

我们有其他[非传统的种子传播方法](https://hackaday.com/2019/10/29/planting-20-million-trees-using-drones-cannons-and-more-unconventional-ways/)比如卡农，但是[这是一项大工程](https://hackaday.com/2019/08/05/a-trillion-trees-how-hard-can-it-be/)，如果你有什么可以帮忙的，[给我们一个提示](https://hackaday.com/submit-a-tip/)！
