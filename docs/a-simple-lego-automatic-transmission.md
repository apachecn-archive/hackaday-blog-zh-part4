# 一个简单的乐高自动变速器

> 原文：<https://hackaday.com/2021/07/22/a-simple-lego-automatic-transmission/>

普通汽车中的自动变速器可能是一个复杂的液压系统，充满了怪异的液体和许多旋转部件。然而，更简单的“自动”变速箱设计是存在的，就像这个来自[FUNTastyX]的乐高设计。

该构建基于一个简单的开放式差速器，但以一种独特的方式进行配置。电机驱动通常作为输入的输出轴之一。同一电机也与通常的主差速器输入轴相啮合。在这些条件下，这种双驱动装置将速度输入相加，并导致另一个轴的转速更快，从而成为输出。

然而，这种构建的技巧是，驱动通常的差分输入是通过乐高滑块离合器完成的。如【技术砖块】*所解释的，一旦超过扭矩需求，该部分[允许齿轮的外齿相对于轴滑动。这个功能的作用是，当“自动变速箱”的输出负载下降时，额外的扭矩需求导致离合器打滑。这导致差速器只有一个输入做功，自动改变传动比。](http://www.technicbricks.com/2008/02/tbs-technugets-01-inside-technic-clutch.html)*

 *这可能不是一个特别有效的变速箱，因为我们怀疑通过非常简单的离合器会有重大损失。然而，它在技术上确实可行，我们希望看到它的性能直接与其他简单的乐高变速箱设计进行对比。

用文字来解释有点令人困惑，但来自[FUNTastyX]的视频在短短几分钟内就很好地解释了这个原理。[这些年来，我们已经看到了很多疯狂的乐高变速箱，](https://hackaday.com/2010/04/08/lego-gearbox-seven-speed-plus-reverse/)我们怀疑这将是最后一个。休息后的视频。
