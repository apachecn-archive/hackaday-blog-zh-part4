# 幽灵，可怕的骨架是 Pi Pico 供电

> 原文:[https://hack aday . com/2022/10/13/ghost-terrible-skeleton-is-pi-pico-powered/](https://hackaday.com/2022/10/13/spooky-scary-skeleton-is-pi-pico-powered/)

又到了一年中最恐怖的时候，全球各地的黑客正在拼凑一些令人毛骨悚然的项目。[凯文]就是其中之一，他在万圣节前夕创造了一个怪异、恐怖的骷髅。

该项目以一种相对简单的方式工作:本质上，一个 Raspberry Pi Pico 负责读取 HC-SR04 超声波测距仪。它被包裹在一个 3D 打印的头骨状外壳中。当骨架检测到前方有人或有东西靠近时，Pi 会触发一个连接到 3D 打印齿轮的小型伺服系统。这与一对齿条连接，齿条驱动头骨的眉毛上下运动，并打开和关闭其下颚。

当然，这里有一些主要的解剖学问题。头骨没有眉毛；那是不可能的。眉毛是附着在肉和肌肉上的毛发；它们根本不存在于骨头的世界里。然而，公平地说[凯文]为了节日而采取创造性的许可，我们都可以支持这一点。

这是一个基本的构建，也是一个有趣的构建。这将是学习一些基本的微控制器技能的一个很好的方式，同时也是一个很好的万圣节小魔咒，可以吓退回到冰箱里再喝一杯饮料的客人。
