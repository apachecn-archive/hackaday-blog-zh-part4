# 自制工具将廉价钢材变成伸缩管

> 原文：<https://hackaday.com/2019/08/13/shop-made-tools-turn-cheap-steel-into-telescoping-tubes/>

初学金属加工的人常常惊讶于钢竟然如此便宜。这是一种由千兆吨制造的商品，总会有大量大项目留下的多余碎片和废料，每磅只需几分钱。但你得到的往往不是你所需要的，尤其是当它是带有焊缝的钢管，阻止一根管装入另一根管时。

火球工具公司的[Jason Marburger]提供了一些清洁钢管内部焊缝的好建议。下面视频的第一部分详细介绍了清理焊缝的手动方法，包括凿毛、用窄带砂光机打磨以及用模具研磨机打磨。这些都工作得很好，但只适用于短长度的管道。更长的管子需要特殊处理，这就是[Jason]设计的巧妙工具派上用场的地方。

通过将一大块高速钢连接到由下一个尺寸的管子制成的金属块上，并用一根沉重的螺纹杆驱动它穿过管子进行清洁，他基本上创建了一个内部成形器来刮削焊缝。它工作起来很有魅力，就像他通过在软钢棒的边缘放置一个硬面焊条珠来制作圆管的工具一样。用车间压机将该工具压入有缝圆管中，也很好地清理了焊缝。

向[Jason]致敬，他提出了几个值得牢记的购物技巧。最近我们已经看到了类似的金属加工工具，像[这个自制的冲模工具](https://hackaday.com/2019/08/07/punch-those-hole-drilling-blues-away-with-a-homebrew-punching-tool/)和[一个线性轨道来保持你的等离子切割机在直线上](https://hackaday.com/2018/06/22/linear-track-makes-plasma-cuts-neat-and-simple/)。
