# 用手机 3D 打印蚀刻草图

> 原文：<https://hackaday.com/2018/12/03/etch-a-sketch-3d-printed-with-cell-phone/>

我们大多数人对童年时代的蚀刻素描都有美好的回忆。[强大的可打印材料]想要更新设计，所以他 3D 打印了一个 XY 托架，用于与手机绘图程序配合使用的手写笔。你可以在 [Thingiverse](https://www.thingiverse.com/thing:3251892) 上看到下面的视频和 3D 模型细节。

这个设计本身很有趣，但它也给了我们一些想法。首先，如果你把它机动化，你可以做出一些非常聪明的绘画玩具。但也可能有更实际的用途。

您还必须自动化 Z 轴，但是如果您这样做了，这对于移动应用程序开发人员来说是一个不错的测试工具。手写笔可以模拟手指戳和拖动，而屏幕截图验证正确的操作。当然，还有其他方法可以做到这一点，但是在一个新的用户界面上看到一个机械化的手指描绘出测试将会是非常棒的。

除此之外，这只是一个有趣的项目，应该很容易扩大规模，以使用一个积满灰尘的旧平板电脑，让它再次有用。此外，视频的第一部分展示了真实的 McCoy 是如何工作的，这总是很有趣。

我们最近看到了另一种使用[电子纸](https://hackaday.com/2018/11/09/using-e-paper-displays-for-an-electronic-etch-a-sketch/)的现代蚀刻素描。如果你想自动化一个合适的蚀刻草图，我们已经见过几次。
