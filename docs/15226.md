# 用乐高探索活塞式发动机设计

> 原文:[https://hack aday . com/2022/11/01/exploring-piston-engine-design-with-Lego/](https://hackaday.com/2022/11/01/exploring-piston-engine-design-with-lego/)

当学习一台机器或机械的设计时，阅读和观看视频当然是有效的，但很难胜过动手实验。在休息后的视频中，[Brick Technology]使用乐高获得了一些关于活塞发动机设计世界的实用见解，从单缸一直到径向十二缸发动机。

使用乐高技术系列的气压缸，[砖块技术]从单缸设计的基础工作开始。除了不涉及燃料-空气爆炸这一事实之外，这些活塞还具有双重作用，这要归功于一个阀门机构，当活塞到达其冲程末端时，该阀门机构会切换活塞的加压侧。经过几次实验后，他决定用一排六个两升的汽水瓶作为压缩空气源。

[![](../Images/eaf67f2c05fc3efbec24a39ab3eaa957.png)](https://hackaday.com/wp-content/uploads/2022/10/legoengines_detail.jpg) 他还通过钻出端口和添加硅油进行润滑来提高乐高气缸的性能。在最初的原型中，气缸还充当连杆，随着曲轴的旋转而前后倾斜。经过一些测试后，他发现他可以通过用滑块机构限制气缸并增加一个单独的连杆来提高效率。

基础工作完成后，[砖块技术]可以开始试验引擎配置和几何形状。直列二缸、三缸和四缸，V2、V6、V8 甚至 R12 都在菜单上。他还可以改变曲轴的几何形状，将扭矩转换为转速，反之亦然，并建造一个启动马达和扭矩发电机。

就像【积木科技】的[乐高电子鼓](https://hackaday.com/2022/10/22/lego-plays-electronic-drums/)和[漩涡机](https://hackaday.com/2022/09/01/lego-rig-makes-pretty-water-vortexes/)一样，这个视频给我们的是一种只有几百块乐高积木才能挠到的痒痒。当然是为了快速成型。
