# 一体式会议视频流盒

> 原文：<https://hackaday.com/2019/08/18/an-all-in-one-conference-video-streaming-box/>

举办黑客训练营或其他活动时，组织者面临的诸多挑战之一是活动视频的制作和分发。由于谈话被录制下来，它们必须被放到网上，而且由于有大量的谈话要处理，通过像 YouTube 这样的网络接口一个接一个地上传它们很快就变得不切实际了。在丹麦的 BornHack 2019 黑客营，他们使用了一个特别集成的单元来实时上传视频，它的创建者[Mikkel mikjr Christensen]很好地分享了我们在休息时发布的视频，[他在丹麦开源软件营 Camp 2017](https://www.youtube.com/watch?v=XKFwAJZgwNw) 上的一次演讲。

它带领观众经历了几年的演变，从集成麦克风和后处理功能的简单摄像机，到第一代笔记本电脑和机架式显示器系统，再到一个坚固的便携式机箱中的最终系统，一个运行 OBS 和硬件 MPEG 编码器的强大笔记本电脑。精心选择电源和使用高品质无线话筒，现在无需大量基础设施，即可为 BornHack 等活动提供即时视频流。

如果你想知道你以前在哪里听过这个名字，[Mikkel]是 YouTube 频道[用 Mike](https://www.youtube.com/channel/UC4olsZkS1lSOL8HHzLHNocw)逆向计算中的[Mike]。老实说，我们的谈话更多的是关于复古电脑，而不是视频盒。
