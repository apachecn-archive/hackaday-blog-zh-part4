# 用太阳能系统给花园浇水

> 原文：<https://hackaday.com/2022/07/22/watering-the-garden-with-a-solar-powered-system/>

如果你想让你的植物茁壮成长，定期给花园浇水是很重要的。[Nikodem Bartnik]为他建立了一个系统来处理这个问题，[保持他的花园不断发展。](https://indystry.cc/diy-solar-powered-garden-watering-system/)

该系统有一个 Arduino 命令一个灌溉系统，该灌溉系统基于一个从水库输水的泵。它与一个水位传感器配对，以监视系统中可用的水。湿度传感器也被用来监测主要的土壤条件，以确保植物不会浇水过多或过少。在这种情况下，[Nikodem]设计了他自己的电阻式湿度传感器，这被证明是困难的，但在这个过程中教会了他很多。所有的东西都被包在一个食物容器里，以防止在室外安装。还安装了太阳能电池板和充电系统，无需电源连接即可为整个系统供电。

虽然这个系统工作正常，但湿度传感器有点不可靠，而且需要大量布线。第二个版本去掉了传感器，使用 Pi Pico 实现了一个简单的基于定时器的灌溉方案。

不管怎样，这两个系统都发挥了作用，有助于保持至关重要的水流向花坛。自动给植物浇水在这里是一个有点流行的主题，[我们已经在最近的](https://hackaday.com/2022/05/19/water-your-plants-just-four-times-per-year/)看到了一些漂亮的黑客。休息后的视频。
