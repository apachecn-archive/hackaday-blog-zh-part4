# 七段显示器也是一个输入设备

> 原文：<https://hackaday.com/2022/11/05/the-seven-segment-display-thats-also-an-input-device/>

我们习惯于看到各种各样的七段显示器，无论是机械的，电子的，还是介于两者之间的。但是所有这些展示的共同点是，你知道，它们都是展示。用它们作为输入只会是疯狂的言论，对吗？

也许吧，但我们喜欢七段输入设备与[“in slide】的结合。这个想法来自于](https://hackaday.io/project/188053-fun-but-impractical-numerical-input-system)[“数字标签”显示器](https://hackaday.com/2022/10/06/seven-segments-but-not-as-we-know-them/)，我们在 10 月份报道过，它将一个标准的七段字符分成三个垂直条——两个窄条用于外部垂直段，一个宽条用于保持水平元素。通过上下相对滑动这些条带，可以组成标准的九位数字，加上一些其他字符。

[Dave]对这一主题的理解始于用激光切割的胶合板构建他的展示，这是一个不错的选择，因为白色木材和雕刻部分之间形成了良好的对比。接下来，他在滑道中嵌入了稀土磁体，并在框架中安装了七个霍尔效应传感器。传感器通过 74HC165 并行加载移位寄存器连接到 Arduino Nano，该寄存器允许多个模块以菊花链形式连接在一起。他还构建了一个 Arduino 库来读取段的当前状态；它支持完整的十六进制字符集，如果您愿意，甚至可以支持十二进制字符集。

[Dave]已经共享了库，看起来您可以从原始项目中获得该机制的构建文件。这很好，因为这看起来适合黑客攻击。看起来很容易通过增加齿条齿轮传动装置和步进器来实现这种显示的机械化——类似这样的东西可以制作一个有趣的时钟。
