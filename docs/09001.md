# Cura 插件提供自定义支持

> 原文：<https://hackaday.com/2021/01/19/cura-plugin-offers-custom-support/>

[Chuck]喜欢 Simplify3D 手动为模型部件添加支持的功能。然而，并不是每个人都想花 150 美元买一个切片机，所以他分享了如何安装一个插件，让你在 Cura 中做同样的事情。

这个插件是“圆柱形自定义支持”这听起来并不令人兴奋，但你有五种形状可以选择，你可以很容易地创建自定义支持。您还可以使用大小和角度参数来自定义效果。

圆柱体和立方体的选择是显而易见的，但是对支座支撑的解释是有用的。[Chuck]展示了这如何比默认支持更有效。

当然，证据是在印刷品中，而且这个模型在第一次尝试时看起来相当不错。[Chuck]提到他应该做更大的支撑，这当然是可能的。

我们喜欢他之前的关于树支持的视频，它也谈到了一般的支持，所以如果你在 Cura 没有做太多的支持，你也可以看看那个。它解释了许多您可以调整的支持选项。

我们曾经认为我们真的需要水溶性支持，但现代切片器在使支持材料易于移除方面做得很好。也可以尝试[提供脱模剂](https://hackaday.com/2020/05/27/improving-3d-printed-supports-with-a-marker/)。如果你想要更多的支持背景，[给你](https://hackaday.com/2020/02/18/everything-you-wanted-to-know-about-3d-printing-support-but-were-afraid-to-ask/)。
