# 蠕虫机器人如你所料缓慢前进

> 原文：<https://hackaday.com/2020/10/18/worm-bot-inches-along-as-youd-expect/>

机器人运动是一个广泛的话题，对于初露头角的设计师来说，有许多选择。通常，大自然是一种灵感，许多机器人已经被用来探索各种昆虫和动物的运动状态。受到普通小虫的启发，[【jegatheesan . soundarapandian】决定制造一个以类似方式移动的机器人。](https://www.instructables.com/Inchworm-Robot-Modular-Move-Allsides-With-BT-App/)

该建筑由一系列 3D 打印的连杆组成，中间装有伺服系统。这使得机器人的身体能够像真正的英寸蠕虫一样铰接和弯曲。通过向上弯曲身体，向前移动，再向下弯曲，机器人可以慢慢地沿着表面前进。Arduino Pro Mini 是操作的大脑，足够紧凑以适合小型机器人，同时仍有足够的输出来命令所需的多个伺服系统。控制是通过智能手机应用程序，使用麻省理工学院的 AppInventor 平台和古老的 HC-05 蓝牙模块。

这是一个有趣的构建，我们希望看到它走得更远，用电池取代系绳，也许还有一些传感器，使它能够进一步与环境互动。我们以前也见过其他创造性的 3D 打印设计，比如这个球形四足机器人。休息后的视频。
