# Remoticon 2021 //塞尔吉·涅斯特伦科让硬件在闪电和宇宙射线中运行

> 原文：<https://hackaday.com/2022/02/10/remoticon-2021-sergiy-nesterenko-keeps-hardware-running-through-lightning-and-cosmic-rays/>

去太空已经够难了。你必须上升几百英里，然后侧着身子快速进入轨道。但是把东西送上太空是一回事，保持精密仪器在太空中运行又是另一回事。在他在 Remoticon 2021 的演讲中，[SpaceX 的前辐射效应工程师【塞尔吉·涅斯特伦科】向我们展示了在](https://youtu.be/-KG3NPBJ7R8)上升的过程中可能摧毁你的敏感电子设备的所有事情。

麻烦在发射前就已经开始了。由于地理上的意外，几个发射场位于容易发生严重雷暴的地区:这不是将 300 英尺长的金属管直立起来并让它站立一天的理想位置。发射台附近的其他危险包括任性的野生动物和来自海洋的咸味喷雾。

一旦你进入太空，这些危险就不复存在了，但随后热量突然成为一个问题:如果你的航天器处于阳光充足的环境中，它会很快加热到 135 摄氏度，而在阴凉处的部分会冷却到-150 摄氏度。一个简单的解决方案是沿着轴旋转你的飞行器，以确保各方均匀的热负荷，类似于你在烤肉架上旋转香肠的方式。

但是太空中的电子设备面临的最具挑战性的问题之一是辐射。[Sergiy]详细解释了航天器可能遇到的各种类型的辐射:范艾伦带中的带电粒子，一旦离开低地球轨道的宇宙射线，以及不时从太阳喷射出的各种电离垃圾。减少电子产品辐射负荷的最简单方法就是呆在地球附近，躲在地球磁场中。

对于星际飞船来说，无法逃脱冲击，唯一的生存之道就是让你的电子设备“超硬”。由于重量的限制，屏蔽通常不是一个选项，所以工程师们使用已经在辐射室中测试过的组件，以确保它们不会突然短路。添加冗余电路以及看门狗定时器等自我监控功能也有助于使飞行计算机更加鲁棒。

[Sergiy]的演讲充满了有趣的轶事，会让我们所有人内心的宇航员感到高兴。想象过一只蝙蝠试图搭乘航天飞机吗？事实证明，一只雄心勃勃的太空蝙蝠做到了。虽然设计太空合格的电子产品不是我们大多数人每天都做的事情，但[Sergiy]的经验为更现实的问题提供了大量的提示。毕竟，盐和湿气会侵蚀你自行车上的电缆，就像它们侵蚀月球火箭一样。
