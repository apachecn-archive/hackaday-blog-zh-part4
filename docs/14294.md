# Hackaday 播客 178:超级计算机的回归，开源的胜利，精美的钟表和值得为之献身的文档

> 原文:[https://hackaday . com/2022/07/22/hackaday-podcast-178-the-return-of-super co-victory-for-open-source-delivery-delivery-of-die-for/](https://hackaday.com/2022/07/22/hackaday-podcast-178-the-return-of-supercon-victory-for-open-source-exquisite-timepieces-and-documentation-to-die-for/)

Hackaday 主编埃利奥特·威廉姆斯和执行主编汤姆·纳尔迪以社区期待多年的公告开始了本周的播客:Hackaday 超级社区的回归！虽然仍有一些后勤细节需要敲定，但我们都非常兴奋地回到现场，并迫不及待地分享更多，因为我们越来越接近 11 月。当然，没有 Hackaday 大奖就没有 Supercon，这个奖项恰好在本周末结束了它的 *Hack it Back* 挑战赛。

在其他新闻中，我们将谈论关于在 Ortur 的激光雕刻机上运行的 GPLv3 固件的发展情况(别担心，这是一个改变的好消息)，以及一个特别令人印象深刻的修复，使高端工业 3D 打印机远离垃圾堆。我们还将对两个记录在案的精彩项目趋之若鹜，了解低级消防栓的迷人起源，并疯狂猜测远处隐约出现的死亡太阳能电池板的浪潮。
