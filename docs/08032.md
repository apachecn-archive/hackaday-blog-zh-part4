# 本周失败:给钢琴上吉他弦

> 原文:[https://hackaday . com/2020/10/12/每周失败-将吉他弦放在钢琴上/](https://hackaday.com/2020/10/12/fail-of-the-week-putting-guitar-strings-on-a-piano/)

钢琴在弦乐器家族中有点古怪。除了很少看到人们在公共汽车上带着它，或者用它的外壳来隐藏一把冲锋枪，琴弦接合的方式——通过在每个键上连接小锤子来敲击它们的声音——与通常的手指或弓运动相比，是相当独特的。尽管如此，它是一种弦乐器，所以很自然地会想，如果用吉他弦代替钢琴丝，钢琴会发出什么样的声音。好吧，[【马蒂亚斯·克兰兹】继续寻找真正的困难，并在这个视频中展示了结果](https://www.youtube.com/watch?v=6RyjoXVDqUs)。

在与断线钳短暂相遇后，很快就到了无法回头的地步。现在，普通的钢琴有 88 个键，根据音符的不同，一个键可能同时涉及三根弦。以马蒂亚斯的钢琴为例——在他的辩护中，它确实经历过更好的日子——总共有 210 根弦需要在实验中更换。另一方面，吉他只有六根弦，所以他不仅需要 35 包吉他弦，吉他弦的规格和长度也非常有限。从一开始听起来可能是徒劳的努力，随着时间的推移并没有变得更好，在某些时候，绳子不够长了，他不得不把它们绑在一起。除了一些不可避免的破损，他不幸用完了琴弦，无法完成整架钢琴，尽管看起来他仍然设法大致覆盖了吉他的频率范围，所以这是一个合适的结果。

我们不确定 Mattias 是否期望这真的有用，但是它确实有用——至少有一些真实的声音。然而，结果不仅仅是可疑的吗？哦，当然，但我们不得不佩服他的胆识和毅力，以实际渡过这一关。他花了 28 个小时才把吉他弦调好，又花了不少时间才把它们调好。有回报吗？那要看你怎么看了。这无疑满足了他和其他人的好奇心，钢琴现在产生了一些真正独特而有趣的声音——但休息后在视频中自己检查一下。但这可能并不适合所有人，所以幸运的是，改变钢琴声音的最终方法并不多。最糟糕的情况是，[你可以把它变成一个工作台](https://hackaday.com/2019/02/03/recycled-piano-becomes-upcycled-workbench/)。
