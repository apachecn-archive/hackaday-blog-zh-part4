# 视觉磁场

> 原文：<https://hackaday.com/2019/02/25/visual-magnetic-fields/>

如果你需要可视化磁场的帮助，这些[慢动作视频捕捉](https://www.youtube.com/watch?v=75wrs0PLl_s)应该会教育或吸引你。磁力线很难用语言来描述，因为磁铁的形状和强度起了一定的作用。因此，对于习惯于条形磁铁的人来说，很难想象锥形磁铁会发生什么。我们应该在你观看下面的视频之前建议你，如果你被磁铁矿砂堵塞一块裸露的磁铁然后在地板上飞行的景象所排斥，这是你唯一的警告。

技术和设备很简单，并在视频中显示。一层黑色的沙子铺在一块紧绷的塑料上，做成一个像脏蹦床一样的东西，中间放一块钕磁铁。弹跳动作同时发射沙子和磁铁，因此它们悬浮在空气中，颗粒可以在空气阻力不大的情况下移动。

这些视频都是用一台相机和一块磁铁拍摄的。多台相机会产生 3D 视觉效果，多块磁铁交织在一起的磁场会很美。或许是球形磁铁的螺旋？你在房子周围放了什么？另一方面，我们可以用[磁铁来模拟气体原子](https://hackaday.com/2017/08/15/a-gas-model-made-of-magnets/)和[欺骗它们完成不寻常的壮举](https://hackaday.com/2016/03/21/just-when-you-thought-magnets-werent-magic-magnets-are-mechanisms/)。
