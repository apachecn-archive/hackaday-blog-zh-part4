# 在麦当劳的触摸屏桌上玩游戏，就像在 2021 年一样

> 原文:[https://hack aday . com/2021/09/08/game-like-its-2021-on-a-a-McDonalds-touch-table/](https://hackaday.com/2021/09/08/game-like-its-2021-on-a-mcdonalds-touchscreen-table/)

世界各地的一些人可能已经见过这些安装在当地快餐店的基于 Android 的游戏桌，甚至可能已经足够幸运地触摸过一个当时实际工作的游戏桌。

最初基于一台古老的迷你电脑，配有 1080p 平板液晶显示器和触摸覆盖层，它们在过去可能会让小孩子兴奋不已，但现在我们期望更多。YouTuber [BigRig Creates] [得到了一个，条件不太好，但经过大量的肥皂和水之后，它被拆了下来，原来的控制器被一个现代的迷你 PC 所取代。需要说明的是，除了原始硬件的外壳和显示屏之外，已经没有什么剩下的了，但我们并不在乎，因为很多注意力都放在了软件方面，以使它能够三次启动到 Windows 10，Android x86 和运行](https://www.youtube.com/watch?v=XKUkAKE7sXI)[仿真站](https://emulationstation.org/)的 Linux，涵盖了所有你可能会有的桌面游戏需求。

在内部，在事物的连接方面有相当大的改进空间，并且[BigRig]很快承认这一点，但是这就是这个学习游戏的全部。现在，你们中的许多人可能会对在这样的桌面系统上玩游戏感到窒息，毕竟，很明显这对背部和颈部来说非常困难。但是，它确实提供了一个简单的选项，只需在侧面走动，就可以从横向切换到纵向，所以它确实有一个好处。而且你有一个方便的地方来倾倒你的啤酒和外卖，所以在你的公寓里也许不是一件坏事？是的，它确实[运行毁灭](https://www.reddit.com/r/itrunsdoom/)。

我们特别被定制的 boot logo 和仿真站里的漂亮的定制艺术逗乐了。正是这种对细节的关注使得建筑成为一个伟大的建筑，成为聚会上的谈资。现在，要是他能解决布线工作就好了。
