# 修复史上最酷的笔记本电脑

> 原文:[https://hack aday . com/2020/02/21/restoring-the-cool-laptop-ever/](https://hackaday.com/2020/02/21/restoring-the-coolest-laptop-ever/)

经验丰富的读者无疑会记得 GRiD 笔记本电脑，这种曾经并且永远都很酷的电脑统治了军事市场几十年。GRiDs 是第一批进入太空的笔记本电脑，它们因其时尚的外观和坚不可摧的机器声誉而令人垂涎。

网格经历了多次迭代，尽管它们的军事根源使它们几乎不可捉摸，[西蒙]得到了一台网格笔记本电脑，并着手修复它。他的主题是 1986 年的电影*外星人*，这部电影以几台网格罗盘电脑为道具。[西蒙]的 1550SX 比指南针 2 稍晚一些，但该机的文件显示，这是一架皇家空军的机器，因为未知的原因被认为是不可用的。

[Simon]小心地把它拆下来——密切注意下面的视频，你会听到镁壳部件发出的泄露秘密的叮当声，而不是塑料发出的沉闷撞击声；他们不再把它们做成了那样——并且清理干净了。他用 PiMoroni 10”复古游戏显示器替换了原来的显示器，以保持原来的 4:3 纵横比。一个 Raspberry Pi 4 放了进去，还有一个 Teensy 负责将网格键盘适配到 USB 并点亮一些前面板 led。第二个 Teeny 允许使用原来的 IsoPoint 鼠标，这是一个真正的瑰宝。通过添加适当的图形，这台机器看起来就像在殖民地海军陆战队的空投船上一样。

我们喜欢[西蒙]的复古风格，以及电影《怀旧》。我们只是很高兴他没有包括一个脂肪电池，这可能不会与镁壳相处。游戏结束了，伙计！
