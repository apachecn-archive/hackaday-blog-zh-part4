# 测试 3D 打印切割刀片是一项可怕的工作

> 原文：<https://hackaday.com/2021/10/22/testing-3d-printed-cutting-blades-is-scary-work/>

[伊万·米兰达]来自一个商店周日关门的地方。因此，当他发现自己需要一个切割刀片时，他意识到他必须自己造一个，或者只是等待。他选择了前者，我们开始享受旅程。(视频，下面嵌入。)

他的第一次尝试是用安装在斜切锯上的 3D 打印切割刀片切割木板。当他在一个安全的距离设置好斜切锯进行切割后，[Ivan]选择测试一下刀刃。唉，它只是融化了，木头几乎没有划痕，所以[伊万]又回到了绘图板。

他的第二次尝试是数控铣削一个厚达 6 毫米的铝刀片。锯需要一些修改，以适应适当的锯，但它能够削减木材没有重大戏剧！

回到 3D 打印的概念，[Ivan]怀疑降低切割盘的表面速度可以减少摩擦引起的加热。理论上，这将允许 3D 打印的刀片在不熔化的情况下切割木材。为了实现这一目标，他使用钢架和无刷电机建造了自己的基本落锯。只要喷一点水，小心控制速度和压力，刀片就能慢慢啃穿一块木板。后来牙齿几乎全磨平了。

事实是，3D 打印的刀片通常太软，无法做任何真正有用的工作。然而，看起来很有趣，这对我们来说足够了。如果你想要更有用的东西，考虑打造你自己的刀。
