# 芯片研磨提高 CPU 性能

> 原文：<https://hackaday.com/2020/07/21/die-lapping-for-better-cpu-performance/>

中央处理器在硅芯片中产生热量，进行所有那些奇妙的计算，使我们的计算机工作。但是硅的导热性相当差，所以你的 CPU 芯片越薄，它将热量传导到散热器就越好。这在理论上保证了更好的冷却，从而有更大的性能空间。因此，一些超频者开始研磨他们的 CPU 芯片，试图提高性能。

正如[Linus Tech Tips]的团队所发现的，这不是一个简单的过程。首先，CPU 必须去盖，这在有问题的英特尔芯片上需要加热以释放中间散热器。还需要一种特殊的夹具来精确地完成这项工作。一旦裸露的 CPU 清除了所有残留的胶水和热化合物，就可以用第二个夹具小心地研磨它，以避免过度打磨 CPU。

经过细致的拆卸、研磨和重新组装后，CPU 在基准测试中似乎下降了 3-4 摄氏度。从超频的角度来看，这并不算多。虽然这一过程风险大且复杂，收益甚微，但其潜在的前提还是有价值的——[英特尔在后来的芯片中进行了稀释，以使自己获得较小的收益。](https://www.tomshardware.com/news/intel-comet-lake-s-cores-53-ghz-high-power-better-pricing/2)休息后的视频。
