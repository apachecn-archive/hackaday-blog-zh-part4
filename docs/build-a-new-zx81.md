# 打造全新 ZX81

> 原文：<https://hackaday.com/2021/05/28/build-a-new-zx81/>

[复古小屋的] ZX81 死了，当他试图找出修复方法的时候，他[决定建造一个新的](https://www.youtube.com/watch?v=wrAgdLYNKJo)。当然，用新零件组装一台大约 1980 年的电脑有点令人生畏。除非你[从一个现有的设计](https://wilco2009.blogspot.com/2017/07/project-double-clon-zx80zx81-reading.html)开始，这个设计已经准备好了。

PCB 看起来很棒，我们喜欢丝网印刷显示对帮助设计师[Alejandro Sebastian]的项目的认可。当然，这个盒子是 3D 打印的。起初，电源 LED 不工作，但电压看起来正确，电路板通电。

这个小盒子实际上可以读取原版磁带，这使得它可以装载 3D 怪物迷宫。我们不确定为什么电源指示灯不工作，但我们猜测它是反向的，或者它-或降压电阻-没有建立良好的连接。

这看起来像一个有趣的项目，将是一个很好的方式来获得一个相当真实的外观 ZX80 或 ZX81，而不必太努力。你需要的所有文件都在[亚历杭德罗的]网站上，看起来大部分工作将是采购所有组件。

当然，你总是可以走[FPGA 路线](https://hackaday.com/2017/01/12/retro-zx-spectrum-lives-a-spartan-existence/)。如果你做了一个，想学组装，[看看这个](https://hackaday.com/2012/10/04/try-out-your-assembly-and-basic-skills-with-a-zx-spectrum-project/)。
