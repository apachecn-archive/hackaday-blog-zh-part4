# 改善更多叶片设计缺陷

> 原文：<https://hackaday.com/2020/09/16/improving-more-leaf-design-flaws/>

[Daniel]最近因其在改善日产 Leaf 电动汽车默认充电模式方面的工作而受到关注，当时他使用了汽车附带的紧急/涓流充电器。他的工作使得减少汽车的输入功率成为可能，如果充电插头看起来可能无法处理这些汽车充电时消耗的全部 1.2 千瓦至 3 千瓦。由于这项工作，他能够[为这些入门级电动汽车](https://www.youtube.com/watch?v=ensZ33hhNB8)创造另一种升级，这一次解决了一个主要的叶设计缺陷，即所谓的 Rapidgate。

这些汽车的问题是，它们的电池仍然有被动热管理，不像现在的大多数竞争对手。这在 10 年代初是很好的，当时这款车是首批上市的全电动汽车之一，但现在它的设计时代正在赶上它。在高速公路上长时间快速充电，电池很容易过热。当这种情况发生时，汽车的充电控制器将不再允许汽车快速充电，并严重限制充电速率，即使在快速充电站。[Daniel]能够调整充电软件，以便在默认情况下限制快速充电，将其从 45 千瓦减少到 35 千瓦，并在充电过程中节省大量热量。

虽然我们希望看到日产在进行这些直截了当的软件更改时(或至少给 Leaf 车主提供改善充电体验的选项)，实际上解决了他们汽车设计的设计问题，但我们至少很高兴现在市场上有其他电动汽车至少解决了所有电动汽车常见的电池热管理问题。如果你确实拥有一辆 Leaf，[一定要去看看【丹尼尔】关于给这些汽车充电的原始项目](https://hackaday.com/2020/09/12/adding-luxury-charging-features-to-an-entry-level-ev/)。
