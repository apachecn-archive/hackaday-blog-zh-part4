# 插入硬币(单元格)来玩乐队

> 原文：<https://hackaday.com/2019/05/06/insert-coin-cell-to-play-ledcade/>

在这个 4K·UHD 游戏控制台图形和控制器充斥着按钮、触发器和操纵杆的时代，偶尔从前沿后退一步是有好处的。深呼吸，提醒我们自己，我们并不总是需要那些像素和按钮来找乐子。LedCade 是一款由[bobricius]为这种极简主义游戏打造的μ(微型)街机游戏柜。

仅使用三个按钮作为输入，8×8 LED 矩阵作为输出，LedCade 仍然可以玩十种不同的游戏，代表复古街机游戏的经典流派。在经典硬件黑客幽默的精彩实现中，玩家开始游戏时插入的不是货币硬币，而是 CR2032 硬币电池。

屏幕后面是一个压电扬声器，用于播放适当的老式游戏声音，还有一个带有 Arduino 代码的 ATmega328，用于编排娱乐。作为开发早期项目[单板游戏控制台](https://hackaday.com/2016/06/25/this-arduino-console-has-64-bit-graphics/)的结果，【bobricius】在集成所有这些组件方面做得很好。这一次，印刷电路板不仅仅是主干，PCB 板被拆开，重新形成外壳。经典的拱廊式橱柜比例，比例小得多。

如果你不喜欢单人极简游戏，那就看看 8×8 LED 阵列上的[这种面对面的游戏动作](https://hackaday.com/2019/04/28/a-retro-handheld-console-as-they-used-to-be-made/)。或者，如果你喜欢极简的游戏硬件像纸一样薄，就像 Arduflexboy 的[一样，把所有部件都放在一个柔性电路上。](https://hackaday.com/2019/04/18/arduboy-goes-thin-and-flexible-for-portable-gaming/)
