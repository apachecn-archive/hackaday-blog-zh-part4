# 印刷钢琴机械装置的确很棒

> 原文：<https://hackaday.com/2021/10/14/printed-piano-mechanism-sure-is-grand/>

你知道钢琴是如何工作的吗？当然，你按下一个键，一个锤子敲击一根弦，但是这个操作的细微之处是什么？[mechanical]的 [3D 打印的三角钢琴键盘](https://www.reddit.com/r/mechanical_gifs/comments/q5ws7s/3d_printed_mechanism_of_a_grand_piano_key_strings/)的一个小部分的比例模型揭示了这种巧妙机制的复杂性。这里的“大”区别是钢琴长度不可知的，只是指任何非直立的。它们的操作方式相同，但为了节省空间，布局有所不同。

原声钢琴的琴键不仅仅是显示的部分，而是长得多的杠杆，可以做很多工作，包括控制自己的消音器。真正有趣的部分是允许音符重复播放而无需先释放按键的机制。同样的机制也可以让钢琴家根据施加的压力大小，轻轻地、大声地或介于两者之间地演奏。

所以你知道锤子敲击琴弦(或者在这种情况下，敲击杆)，你可能会想到它会后退让琴弦发出声响。但是也有整个系统，只要这个人按住键，就让锤子在附近重复敲击。请务必在休息后查看构建视频。

[mechanical]肯定是要起立鼓掌的，因为他们在视频的评论中说，当他们写完组装指南后，他们将发布 STL 文件。).那将是一场多么精彩的表演啊。

有很多方法可以破解原声钢琴，但是不要认为你可以代替吉他弦。
