# 我们都住在一艘聚氯乙烯潜水艇里

> 原文:[https://hack aday . com/2021/12/11/we-all-live-in-a-PVC-submarine/](https://hackaday.com/2021/12/11/we-all-live-in-a-pvc-submarine/)

我们怀疑你是否真的能住在[佩纳的 PVC 潜水艇里，但不管怎样，现在这首歌已经印在我们的脑海里了。虽然帖子是葡萄牙语的，但你可以很好地了解它是如何工作的，翻译软件也比以往任何时候都好。超越了语言障碍，有关于建设的每一步的视频。然而，我们没有找到车辆在水中的视频。

水管工的喜悦已经修改了推进器的马达，以及一台照相机。环氧树脂灌封可以防水。我们在其他建筑中也看到过同样用途的蜡烛。

我们的葡萄牙人可能会有所欠缺，但我们不知道潜艇是如何利用压舱物或任何其他方案来动态改变浮力的。有人谈到使用聚氯乙烯管保存空气，并调整它们以获得中性浮力，但似乎没有办法在飞行中改变它。也许有更好的语言技能的人可以在评论中帮助我们。我们确实注意到系绳电缆有一个有趣的浮力测量方法:切开游泳池浮子，制成漂浮导管。这主意不错。

我们以前见过 PVC 接头，它们出奇的便宜。我们也用[发酵粉来增加浮力](https://hackaday.com/2013/08/19/adult-sized-baking-powder-submarine/)。
