# 网络上的模块化音乐合成

> 原文：<https://hackaday.com/2019/10/09/modular-music-synthesis-on-the-web/>

很难想象没有互联网，电子爱好是如何生存的。你在杂志上找到了志同道合的人和项目。更难想象在 CPU 芯片广泛普及之前杂志上有什么项目。想想看，你只能用一把电子管、晶体管和小型集成电路制造这么多东西。但是在计算机革命接管这个爱好之前，总是有很多关于音乐合成的文章。回到原点，你现在可以使用运行在浏览器中的模块化合成器 [Zupiter](https://pointersgonewild.com/2019/10/06/zupiter-a-web-based-modular-synthesizer/) 在网上构建一个虚拟合成器。

这个链接实际上是关于 Zupiter 的，但是如果你只是想玩，你可以直接进入这个链接。然而，我们不得不做一些阅读，也尝试一些例子。你可以在下面看到一个关于合成器的视频。

术语模块化指的是一种合成器，它具有不同的模块，这些模块通过跳线连接在一起，以产生特定的声音。这种类型的合成器出现在 1963 年左右，由 Moog 和 Buchla 根据 Hugh Le Caine 的创新独立开发。

当然，并不是所有你能用 Zupiter 做的事情听起来都像音乐。我们偏爱[这个例子](https://z.musictools.live/#83)。如果你更喜欢真正的硬件，抓住[一个游戏机](https://hackaday.com/2013/11/27/a-modular-game-boy-synthesizer/)。模块化合成器的优点是什么？一个[模块化键盘](https://hackaday.com/2015/07/12/3d-printed-modular-keyboard/)怎么样？
