# 雾与火之歌——看看鲁本斯的试管

> 原文：<https://hackaday.com/2020/08/12/a-song-of-fog-and-fire-taking-a-look-inside-a-rubens-tube/>

黑暗的冬季仍然在我们前面，但是随着夜晚回到最北部的地方，这可能是开始你下一个心情照明项目的好时机。尽管 LED 灯无处不在，但洞穴时代的怀旧情绪使得它很难击败这里真实火焰的舒适性——嗯，假设这是一个受控的火焰。虽然现代 LED 蜡烛做了足够体面的工作，可以从远处愚弄你，但有一种设备他们将很难复制:鲁本斯管。厌倦了他们通常的直管构造，[RyanMake]在概念中加入了一些曲折，创造了一种由半刚性铝管制成的柔性鲁本管。

如果你不熟悉[鲁本斯的电子管](https://en.wikipedia.org/wiki/Rubens'_tube)，通过将一个扬声器连接到一个充满可燃气体的等间距孔的管道上，并点燃它，来可视化火焰驻波，这是科学、有趣和危险的结合。由于最终的视觉效果取决于音频信号的波长，以及管道本身的长度，[RyanMake]的柔性管道方法为通常的固定长度管道版本增加了一些变化。但这还不是他所做的全部。亲眼看到火焰后，他对那根管子里到底发生了什么感到好奇，于是[决定再造一根](https://www.instructables.com/id/Music-Controlled-Fog-Curtain-DIY-Rubens-Fog-Tube/)，这次使用一根透明塑料管和一台造雾机。虽然雾从管道中逃逸出来并不引人注目(而且几乎无法与火竞争)，但它让我们很好地了解了这些管道内部发生的事情。休息之后在视频中自己看吧。

当然，没有一个实验是真正不失败的，在看到他的第一个电子管被烧了几次后，你可能应该坚持做一个作为室内装饰品。另一方面，如果射击火是你正在寻找的，你可能会对这个漩涡加农炮感兴趣。为了在[标准鲁本斯管](https://hackaday.com/2018/12/12/my-oscilloscope-uses-fire/)上做更多的改变，看看[的二维烟火板](https://hackaday.com/2014/04/19/the-pyro-board-a-two-dimensional-rubens-tube/)。
