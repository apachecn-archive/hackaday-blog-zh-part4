# 对薯片的恐惧:萨米·卡姆卡的旁门左道攻击综述

> 原文:[https://hack aday . com/2020/03/09/恐炸薯片-Samy-kamkars-side-channel-attack-round up/](https://hackaday.com/2020/03/09/fear-of-potato-chips-samy-kamkars-side-channel-attack-roundup/)

薯片和丢失的车钥匙有什么共同点？从表面上看，这似乎没什么，除非你不知何故把钥匙丢在了一袋薯片里，这将是足够尴尬的，你可能永远不会提起它。但这两者之间存在令人惊讶的联系，Samy Kamkar 在他新出版的 2019 年超级大会演讲中进行了关联，他称之为“ [FPGA 故障和旁路攻击。](https://www.youtube.com/watch?v=oGndiX5tvEk)

## 侧面信息

[![](../Images/120d2c78f3d88863e00e05dc9ba1a121.png)](https://hackaday.com/wp-content/uploads/2020/03/samy-kamkar-side-channel-chips.jpeg)Samy 的大部分演讲都致力于解释旁路攻击的本质，他将其定义为“你从一个实现中收集的信息，而不是算法或系统本身。”我发现这是一种特别有启发性的思考旁道攻击的方式，因为它们提供了一种通常完全无法预见的进入系统的方式，尽管没有通常与利用相关的直接漏洞，如软件错误或加密系统中的弱点。

Samy 描述的旁道攻击的世界是一个奇怪的地方，攻击可能会突然出现。他们可以从最轻微的信号中获取信息，比如通过监控每个按键独特的机械振动来窥探用户正在键入的内容，要么使用麦克风，要么通过从机器上反弹激光来拾取诱发的振动。基于泄露目标机器内部状态的电容器的电致伸缩振动的超声波攻击是可能的，基于目标通过其步伐时功率使用的微小变化的攻击也是可能的。其中一些旁道攻击甚至可以针对有气隙的机器，这对安全专家来说是噩梦般的刺激。

甚至不仅仅是电脑会受到旁道攻击。萨米讲述了一个安全研究人员的故事，他们设法从一个密封、隔音的房间里窃取了一段正常的对话，只需通过窗户将一台 DSLR 摄像机对准桌上的薯片袋。袋子随着房间里的声音引起的气压变化而轻微振动，足以在摄像机捕捉的视频中产生信号。旁道攻击似乎只受到攻击者想象力的限制。

## 注入检测

Samy 还花了很多时间谈论故障注入攻击。我们最近已经报道了其中的一些，包括使用激光在 MEMS 麦克风上诱导信号的[对智能扬声器的攻击，使 Alexa 远程执行您的命令。Samy 将此与用超声波对无人机 IMU 的类似攻击联系起来，我们发现这很有趣。然而更疯狂的是温度攻击，内存芯片被气体喷粉机喷出的推进剂瞬间冻结；通过冷却内存芯片中的电容器，它可以延长放电时间，足以移除芯片并使用另一个设备转储其内容，从而可能暴露密码或其他敏感信息。](https://hackaday.com/2019/11/06/laser-based-audio-injection-on-voice-controllable-systems/)

也许萨米解释的最有趣的技术是指令跳跃。时钟假信号是任何计算机系统中正常时钟脉冲流中的轻微畸形脉冲。给时钟小故障计时，你可以迫使 CPU 跳过它正在执行的部分代码，甚至可能是检查密码的部分。电压故障，即 CPU 在战略上缺乏电力，也可以达到同样的效果。Samy 甚至列举了几个小故障攻击的例子，这些攻击可以在 Arduino 上轻松执行，甚至可以在令人惊叹的 FPGA 供电的 Hackaday Supercon 徽章上执行。

但是丢失的车钥匙呢？他们在这一切中扮演什么角色？从 21 世纪初开始，汽车制造商~~发明了另一种从顾客身上榨取钱财的方法~~决定通过在每把钥匙上添加 RFID 芯片来提高汽车的安全性。丢了一把钥匙，你就要花一大笔钱去把安全码放进一把新的钥匙里，并把它编入汽车的点火系统，这样你就可以再次驾驶你的汽车了。当然，除非你使用旁门左道攻击来破解加密密钥，并设置一些东西来伪造该代码，这样你就可以用一把普通的旧钥匙来启动汽车。这只是 Samy's talk 随本次演讲放入每个人工具箱的众多工具之一。
