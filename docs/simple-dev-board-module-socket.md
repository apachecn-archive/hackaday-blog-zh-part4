# 简单开发板模块插座

> 原文：<https://hackaday.com/2022/02/12/simple-dev-board-module-socket/>

当您构建快速原型或一次性项目时，最好能够安全地安装各种模块和开发板。有时这些板有安装孔，但通常没有。作为后一类的例子，数字音乐乐器制造商和表演者[DIYDSP]向我们展示了如何构建一个简单的插座来安装 STM32 Nucleo-32 模块。

该插座是建立在一个标准的垫每孔一块矢量板切割成所需的大小。成对的母引脚接头带焊接到电路板上。内部的一对接头用于模块，外部的一对用于互连。接头用短焊桥连接起来，并且[DIYDSP]建议您将外部一对引脚延长几个不必要的长度。这些额外器件可用于额外的电源或接地点，或者在某些电路板上，它们可以连接到调试头引脚。他更喜欢使用母插座，因为这样可以减少意外弯曲的插脚导致短路的可能性。

最后一步是在所需的位置钻安装孔，不再有开发板自由浮动，仅由电线支撑。你对安装这类模块有什么建议吗？是像这里显示的那样单独安装，还是安装到 PCB 上？请在评论中告诉我们。
