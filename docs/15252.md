# 对过时说不，用光纤给你的房子布线

> 原文:[https://hack aday . com/2022/11/03/say-no-to-obsolescence-wire-up-your-house with fiber/](https://hackaday.com/2022/11/03/say-no-to-obsolescence-wire-up-your-house-with-fiber/)

如今，如果你用低于千兆位的东西给你的房子布线，你可能最终会扼杀你的互联网连接。但是，如果您在 100BASE-T 时代使用每台设备两对电缆进行布线，您将需要在购买新交换机之前重新布线。现在，我们中的一些人已经开始为自己配备 2.5G 硬件——这可能再次需要新的布线。你想退出以太网布线升级竞赛吗，至少暂时不想？像[Stefan Schüller]一样，使用光纤来满足您的家庭网络需求！

[Stefan]向您介绍如果您选择研究光纤以满足网络需求，您需要了解的一切，并解释他所做的设计决策——从自己拼接光纤，到构建 PC 以进行路由，而不是获得配备硬件小型可插拔(SFP)的路由器。他还描述了一些陷阱，比如 SFP 模块需要重新配置以适应不同的路由器品牌，以及不得不购买价格不菲的光纤接合器。

最后，他展示了成本明细，并表示他对升级非常满意。虽然与运行以太网相比，成本似乎令人望而却步，但升级到光纤将使您的设备在您需要时以最高速度运行——谁知道呢，也许几年后， [2.5G 将不再满足](//hackaday.com/2020/10/19/ethernet-at-40-from-a-napkin-sketch-to-multi-gigabit-links/)家庭技术需求的新发展，我们将看到更多的 [SFP 模块掌握在黑客手中。](https://hackaday.com/2021/02/13/experiment-with-sfp-modules-with-this-handy-breakout/)毕竟，[现代电视已经使用光纤](//hackaday.com/2022/05/23/using-a-fusion-splicer-to-repair-a-samsung-tvs-cable/)进行视频数据传输。
