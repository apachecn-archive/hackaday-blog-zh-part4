# 投币图形计算器控制台

> 原文:[https://hack aday . com/2022/02/16/投币式绘图计算器控制台/](https://hackaday.com/2022/02/16/coin-operated-graphing-calculator-console/)

资深黑客[Peter Jansen]对《洋葱》杂志去年的一篇文章印象深刻，于是他突发奇想，决定制作[这款投币操作的德州仪器图形计算器控制台](https://hackaday.io/project/184010-vintage-coin-operated-texas-instruments-graphing-c)(视频下方休息——警告垂直方向)。

![](../Images/2fb189e083c4343e3be459306ede4e4d.png)

除了从[原始讽刺文章](https://www.theonion.com/man-scores-vintage-coin-operated-texas-instruments-grap-1846181894)中得到的假模型图之外，没有更多的东西可以继续下去了，【Peter】能够根据照片缩放尺寸，做出一些合理的假设。他利用假期建造了这个项目，招募了他的父亲和女儿作为助手。机柜由 2×3 木材制成，表面覆盖有木质单板的胶合板和乙烯基覆层，用于显示图形。

计算能力来自带有 Arduino Uno 的 Raspberry Pi，它充当 I/O 处理器。用两个旋钮来控制计算器有点棘手，但他成功了。然而，每块地 25 美分，没有明显的硬拷贝功能，这款控制台计算器可能有点贵，除了在当地酒吧喝几杯啤酒时随意绘图。

你可能会从我们多年来报道的他的一些黑客活动中记得[Peter]，比如他的[自制 CT 扫描仪](https://hackaday.com/2015/05/29/hackaday-prize-entry-a-better-diy-ct-scanner/)或者 2014 年凭借[开源 tricorder 项目](https://hackaday.com/2014/06/11/thp-entry-the-improved-open-source-tricorder/)在第一届黑客奖竞赛中获得第四名。
