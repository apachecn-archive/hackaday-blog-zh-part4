# 用重力拉电锯

> 原文:[https://hackaday . com/2022/07/08/用重力拉电锯/](https://hackaday.com/2022/07/08/pulling-a-chainsaw-with-gravity/)

[开花肘]有一个需要碾磨的大白蜡树原木。他有自己的链锯，并分享了一种更容易切割的出色技术[。在砍倒一棵树后，让它干燥一段时间，然后把它拖到锯的地方，你就可以开始切割了。然而，用电锯切割一棵巨大的树是一项艰巨的任务。一些小技巧会让它变得更好，比如让你的原木稍微向山下倾斜，这样链锯就会顺山而下，或者使用夹具来保持切割的直线。有些使用绞车系统拖动夹具一起协助，所以它不只是纯粹的人力。问题是，如果锯碰到一个结或有挑战性的部分，绞盘会施加更大的力。所以你会想放慢速度，让锯子穿过这个区域。](https://www.youtube.com/watch?v=HbAwkID7cjA)

【开花肘】用滑轮和原木下脚料，把它挂在树上。随着切割的进行，原木下落并施加恒定的力。这意味着锯可以在有挑战性的部分减速，并需要时间，延长刀片的寿命。视频中还有其他优秀的技巧，结合他早期的[电锯磨夹具](https://hackaday.com/2014/03/06/building-a-chainsaw-mill-to-make-planks/)，你将只需一把电锯和一些独创性就能磨出原木。
