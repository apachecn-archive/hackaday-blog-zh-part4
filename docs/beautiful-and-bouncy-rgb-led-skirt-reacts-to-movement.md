# 漂亮而富有弹性的 RGB LED 裙子对运动做出反应

> 原文：<https://hackaday.com/2021/02/21/beautiful-and-bouncy-rgb-led-skirt-reacts-to-movement/>

假设不是紧身的，有什么衣服可以像裙子一样自由穿吗？(好吧，除非那是你的事——我们不会评价。)与裤子相比，裙子和连衣裙非常宽松，所以大部分裙子和连衣裙都没有口袋，这太糟糕了。非常糟糕的是，几乎所有的裙子和连衣裙都没有能够对运动做出反应的 RGB LEDs。也许有一天。

在那之前，我们只能像[makeTVee]和他的女朋友一样设计自己的 LED 裙子，并希望它看起来有一半好。[这条裙子有六个 RGB LED 灯条，总共有 120 个 LED 灯](https://hackaday.io/project/177255-neopixel-led-skirt)。这些带子用粘扣带固定在适当的位置，所有的电子设备 Adafruit QT Py、6 自由度 IMU 和 USB 电源——都塞在腰带里，可以在洗裙子的时候轻松取出。继续实用性主题，背面没有 led，尽管可以很容易地添加到舞池中。

我们非常喜欢这里的面料选择。覆盖织物本身看起来很好，但它也能很好地展示和漫射光线，同时隐藏 LED 灯条本身。很明显，他们考虑到了舒适性和实用性，做出了真正可穿戴的可穿戴设备。[makeTVee]称这是一项正在进行的工作，但已经有一些不错的动画在进行，你可以在休息后的视频中看到。

如果你不在乎你的可穿戴设备是否实用，[试试这件光纤水母裙的尺寸](https://hackaday.com/2015/08/12/jellyfish-inspired-led-skirt-for-burning-man/)。
