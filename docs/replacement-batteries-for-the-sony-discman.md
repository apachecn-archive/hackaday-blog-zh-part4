# 索尼 Discman 的替换电池

> 原文：<https://hackaday.com/2019/06/27/replacement-batteries-for-the-sony-discman/>

第一批索尼随身听包括充电电池。这些电池是镍金属氢化物电池(由于当时的技术)，现在已经远远超过了它们的使用寿命。电池技术的新热点是锂——它提供更大的功率密度、更轻的重量和大量现成的电池。如果有人用锂电池为旧的索尼随身听创造一个新的电池组会怎么样？[这正是[sjm4306]为进入今年的 Hackaday 奖](https://hackaday.io/project/165088-sony-discman-diy-lipo-battery)所做的。

Discman [sjm]使用的是基于镍氢电池技术的定制索尼品牌电池，容量约为 500 毫安时。在仔细测量了这种电池的尺寸后，用 3D 打印机在塑料中复制了它。然后，这个外壳被塞进一个从 USB 电源库中取出的小锂电池。

这种构造的唯一触发点是电池触点。最初的电池在末端有两个触点，正好适合随身听；这些是用一个连接到 USB 电源组内部的小 PCB 复制的。最终结果是一个更高容量的原始 Discman 电池的直接替代物，也可以通过 USB 充电。这是一个了不起的项目，下面有完整的视频。
