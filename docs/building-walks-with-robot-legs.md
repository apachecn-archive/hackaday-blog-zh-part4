# 用机器人腿建造人行道

> 原文：<https://hackaday.com/2020/11/04/building-walks-with-robot-legs/>

上海 Evolution Shift 公司刚刚完成了我们所见过的最令人印象深刻的机器人项目之一——[用 198 条机器腿](https://edition.cnn.com/style/article/shanghai-relocate-building-preservation-intl-hnk-scli/index.html)进行建筑行走。我们都看过[结构搬迁](https://en.wikipedia.org/wiki/Structure_relocation)纪录片，大型建筑被转移到新的地点。这包括顶起建筑，在轮子上安装一个支撑平台，然后小心地将建筑拖到新的地点。

但是这座五层楼高、7600 吨重的拉格纳小学的 T 形建筑存在问题，而且通往新建筑的路线需要走一条弯曲的道路并旋转建筑。这排除了更传统的搬迁方法。机器人腿来帮忙了。这座建筑花了 18 天时间走了 62 米，旋转了 21 度到达它的新家。这个项目是保护历史建筑趋势的一部分，而不是推平一切为现代建筑腾出空间。

看完下面的视频后，我们认为你会同意这是机器人技术的一个独特应用，也是一个惊人的工程壮举。免责声明——不要在家里尝试。感谢[Chuckz]给我们发来这条提示。
