# 超级简单的激光雕刻机

> 原文：<https://hackaday.com/2019/11/26/a-super-easy-laser-engraver/>

CNC 有各种形状和尺寸，从巨大的工业车床到自制的弯线机。【MJKZZ】建造了他们自己的紧凑型钻机，[使用光学驱动部件来创建一个可爱但有效的激光雕刻机。](https://www.youtube.com/watch?v=4U6n6LP-FF0&feature=youtu.be)(视频嵌入下方。)

该建筑旨在通过固定激光器和移动床来保持简单。问题中的激光器是一个 500 mW 的装置，由 Arduino CNC 护罩上的 Z 轴驱动，用于运行系统。一个 DVD 驱动器被拆开，蜗杆驱动步进电机组件被用来来回转动托架，托架顶部粘着一张床。在此基础上，安装相同组件的副本，偏移 90 度，给出 X 和 Y 轴。

这种设置的结果是一个轻量级的可移动床，可通过带有 GRBL 的 Gcode 进行控制。有了位于上方的一些相机安装装置上的激光器，纸张可以轻松地安装在床上并进行雕刻。由此产生的精度是令人钦佩的，在全功率下，激光能够切穿纸张。

虽然这是一个轻量级的装备，但它可以作为一种廉价而简单的方法来从计算机化的艺术品中生产模板。[光驱在 DIY CNC 领域仍然很受欢迎](https://hackaday.com/2019/04/29/build-a-plotter-using-scrap-dvd-drives/)，因为它们是一种很好的方式来获得一个已经解决了所有机械问题的可移动平台。
