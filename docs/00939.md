# 迈克泰森的穿孔补丁给高清电视滞后一个 K.O。

> 原文:[https://hack aday . com/2018/10/12/Mike-tysons-punch out-patch-gives-HDTV-lag-a-k-o/](https://hackaday.com/2018/10/12/mike-tysons-punchout-patch-gives-hdtv-lag-a-k-o/)

他们只是不像以前那样做了。由于显示滞后，数字电视使得很多在 CRT 统治地球的时代设计的视频游戏几乎无法播放。由于紧张的反应时间窗口，本来就很难的游戏，在按键不能反映屏幕上发生的事情时，可能会变成愤怒诱导的体验。属于上述类别的一个游戏是迈克·泰森为 NES 打出的拳。然而，NES 家酿开发商[nesdoug]为 31 岁的经典游戏开发了一个补丁，旨在给玩家在现代显示器上玩游戏的机会。

![MTPO Poster 1980s](../Images/28e209b31c0b176f1c78554b58711ad9.png)

迈克·泰森的 Punchout 的延迟修复补丁旨在通过调整游戏速度来缓解数字显示器固有的显示延迟。一些早期阶段没有太大改变，但后来的战斗导致更明显的减速，以补偿现代显示的滞后。很明显[nesdoug]是这款游戏的长期粉丝，因为他还上传了一个混合了舞台和调色板的补丁。

补丁本身以 IPS 文件的形式出现。要应用 lag fix 补丁，你需要一个 IPS 补丁工具，比如 [Lunar IPS、](https://www.zophar.net/utilities/patchutil.html)以及你自己的迈克·泰森(Mike Tyson)的 Punchout 的个人备份光盘。lag 修补程序下载网站上提供了一个校验和值，以确保您有一个可用的 ROM 文件。请注意，ROM 文件在应用补丁的过程中会被覆盖，因此请确保将原始文件放在安全的地方。修补完成后，可以使用您最喜欢的 NES 模拟器，或者使用 flashcart(如果您想在原始硬件上玩)。
