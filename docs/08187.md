# 大型战斗机器人需要同样厚实的定制车轮

> 原文：<https://hackaday.com/2020/10/29/massive-battle-bot-needs-equally-chunky-custom-molded-wheels/>

我们都遇到过这样的情况，适合这份工作的零件不是你能从货架上买到的。在很多情况下，3D 打印是解决这一问题的方法，但有时你需要在艰难的工作中使用艰难的部件。这些定制的聚氨酯战斗机器人车轮就是一个很好的例子。(视频，嵌在下面。)

这个机器人战士名叫“铜斑蛇”，是一个重量级的死亡交易者，曾参加过电视节目“战斗机器人”的比赛。这是一台难以置信的坚固机器，前部有一个可笑的 50 磅(23 公斤)的旋转工具钢滚筒，用于拆卸竞争对手。再加上机器人装甲和行走装置的巨大质量，再加上需要承受同样邪恶的武器所施加的惩罚，标准的轮子是不会飞起来的。

正如[罗伯特·考恩]在下面的视频中详细描述的那样，只有最坚固的车轮才能做到，所以机器人制造商们制作了带有集成轮毂的定制车轮。四件式模具由铝加工而成，用于支撑塑料轮毂，塑料轮毂也是机械加工的，但可以很容易地进行 3D 打印。聚氨酯树脂被注入，并比我们想象的更好地粘附在塑料轮毂上:即使受到相当严重的撞击，也足以避免破裂。整个铸造过程是一个很好的观赏，正如[铜斑蛇的设计概述](https://www.youtube.com/watch?v=HGgxyZpNHi8)。还有[看它撕裂【战鹰】](https://www.youtube.com/watch?v=_EhZZUD1L1E)也是一种享受。

你可能不会制造战斗机器人，但这个过程的缩小版可能是一个方便的技巧，可以储存起来以备将来某一天使用。
