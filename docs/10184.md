# 便携式钻床

> 原文:[https://hackaday.com/2021/05/22/portable-drill-press/](https://hackaday.com/2021/05/22/portable-drill-press/)

我们不确定[John Heisz]的构建是否真的是我们所认为的钻床，但它是一种非常便携的方式，可以将常规钻机转换成类似钻床的东西。你的训练可能会有所不同，但你可以在下面的视频中跟随他的体型。

从表面上看，这似乎不是很难，但是有一些技巧。找到钻头背面钻轴的准确中心需要一点努力。

[约翰]有一个设备齐全的木工车间，所以他用木头制作所有的东西。我们不禁认为，激光切割机或 3D 打印机可能会使一些零件变得容易，即使你可能需要用激光切割机将更薄的胶合板层压在一起。

木质滑轨似乎工作得很好，但我们认为使用像一些 3D 打印机使用的导轨甚至橱柜滑轨可能会有用。基本的想法是如此简单，你应该能够修改它，并用你自己的想法扩展它。

在某些时候，你可以跳过钻孔，使用你自己的[马达和卡盘](https://hackaday.com/2020/09/22/this-diy-drill-press-is-very-well-executed/)。我们已经看到了很多[自制的印刷机](https://hackaday.com/2019/12/15/the-home-made-drill-press-of-your-dreams/)，尽管它们中的很多都没有这种便携性。
