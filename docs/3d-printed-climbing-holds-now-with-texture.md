# 3D 打印攀爬架，现在带有纹理

> 原文：<https://hackaday.com/2022/02/26/3d-printed-climbing-holds-now-with-texture/>

技术让各种可能性以我们认为最合适的方式塑造我们的环境。许多滑雪胜地利用造雪来延长他们的季节，在远离海洋数百英里的地方有冲浪池，如果你不住在任何山脉附近，你也可以建造自己的攀岩墙。对于后者，许多人已经转向 3D 打印机来创建更多像岩石一样的攀岩手柄，但塑料往往不会像岩石一样，除非你像贾尔斯·巴顿-欧文一样将盐加入到打印中。

对于小制造商来说，通常模仿岩石纹理的方式是通过某种方式将沙子永久性地融入握把本身。这种方法效果很好，但是对于攀岩者来说太粗糙了，或者不能忠实地复制攀岩的体验。对于这些握把，不是包括沙子，而是将特定大小的盐晶体添加到 3D 打印握把上形成的树脂中。一旦树脂充分固化，水溶性盐就可以被洗掉，留下完美的质地，可以用粉笔抓住。

虽然对于大规模的攀岩手柄制造商来说，这可能不是一个可扩展的方法，但[Giles]希望这种方法能够帮助较小的企业或者甚至是 DIY 攀岩者在不倾家荡产的情况下制造更真实的手柄。事实上，他已经在当地的攀岩馆使用这些握把取得了一些成功。对于大型制造商来说，这种方法可能更难扩展，但对于任何想亲自尝试的人来说，这种构建所需要的只是 3D 打印机、盐和时间。
