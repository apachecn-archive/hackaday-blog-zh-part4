# 万圣节打造:精致射线枪有音效

> 原文：<https://hackaday.com/2021/10/07/halloween-build-exquisite-ray-gun-has-sound-effects/>

当我们第一次看到[孤独冲浪者]的[射线枪](https://www.youtube.com/watch?v=5IgRGtoFItU)时，我们觉得它看起来奇怪地熟悉。当然，它看起来像你可能在十几部 20 世纪 50 年代的电影或电视节目中看到的老式射线枪。但是，仍然有一种奇怪的熟悉感。原来，它的核心部分是一个老式的定时灯，用于汽车调试。

这不是不显眼的星际迷航移相器。它看起来很结实，有一个很酷的声音发生器，不仅让它有事可做，还在枪的顶部有很酷的控制旋钮。如果你想重新制作，声音电路的设计文件在 Google drive 文件夹中。

计时灯已经是闪亮的银色，镜头架甚至适合枪的扬声器，这并没有什么坏处。一个旧手机电池为整个装置提供能量。音频板显然是老派，使用 4000 系列 IC 振荡器，CMOS VCO 和无处不在的 LM386 音频放大器。

如果你要扮成巴克·罗杰斯、飞侠哥顿，甚至是质子队长去参加派对，这把枪放在你的腰带上会很好看。你仍然可以买到计时灯，但它们并不总是像这个一样。你可能不得不去当铺或二手商店找一个合适的。或者，像[孤独冲浪者]一样，在易贝买一个。

许多零件来自垃圾箱，所以你的构建可能会看起来不同。尽管如此，这仍然是一个有趣的项目，而且正好赶上化妆舞会的季节。

上一次我们看到[lonesouldsurfer]制作射线枪时，[他正在切割一把焊枪](https://hackaday.com/2019/12/12/a-stunning-ray-gun-built-from-junk/)。如果你想要有真正激光的东西，小心每个人的视力。
