# 真实棋盘上的国际象棋

> 原文:[https://hack aday . com/2021/07/03/internet-chess-on-a-real-chess-棋盘/](https://hackaday.com/2021/07/03/internet-chess-on-a-real-chessboard/)

互联网告诉我们，我们可以接受现实世界的替身。我们有一个看起来像我们的化身。我们有虚拟邮箱来读取虚拟信封中的信息。如果你想下棋，你可以和世界上任何人下棋，但要在虚拟棋盘上。或者，你可以使用【karayaman 的】软件[在真实的棋盘](https://github.com/karayaman/Play-online-chess-with-real-chess-board)上玩虚拟游戏。

Python 程序使用网络摄像头。你把它对准一块空板，然后校准。在那之后，程序会在网络世界中跟踪你在真实棋盘上的移动。下面可以看到一个测试游戏的视频。

如你所想，程序使用 OpenCV。如果您曾经想学习 OpenCV，这是一个很好的实用工具，也很容易管理。你需要一个清晰的棋盘视野，最好是从上面看。我们想知道提供一种让你拍摄悬挂在板上的镜子的模式有多难。

即使你不想编程，不用手动同步棋盘就能研究你在真实棋盘上的位置也是一件非常令人放心的事情。

如果你想建造一些硬件，[试试这个棋盘](https://hackaday.com/2021/06/10/automatic-chessboard-lets-online-players-move-the-pieces/)。或者抛弃互联网，[造一个机器人](https://hackaday.com/2015/03/15/lonely-build-yourself-a-chess-robot/)。
