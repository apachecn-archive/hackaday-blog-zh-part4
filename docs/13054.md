# 已检查层线清除油灰

> 原文:[https://hack aday . com/2022/03/16/layer-line-removal-putty-reviewed/](https://hackaday.com/2022/03/16/layer-line-removal-putty-reviewed/)

不是只有[教学技术]讨厌 3D 打印中的图层线，也讨厌砂纸打磨。他最近尝试了增量填充油灰，一种紫外线固化油灰，可以用来掩盖印刷品上的线条。一旦覆盖和固化，你砂腻子顺利。你可以在下面的视频中看到他对产品的看法。

正如许多人在视频评论中建议的，你可以用其他物质增稠 UV 树脂来玩同样的把戏。我们甚至报道了使用[稀释树脂](https://hackaday.com/2018/03/08/3d-printering-print-smoothing-tests-with-uv-resin/)来获得类似的效果。油灰看起来更像发乳或某种软膏，所以戴着手套的手指很容易涂开。刷子也起了作用。紫外线固化是通过一个小手电筒或方便的太阳来完成的。然而，你稍后会看到，他使用了紫外线固化站，如果你有一个工作良好。

当然，即使涂了腻子，还是要打磨。我们假设砂光比试图砂光实际层线更容易。另一方面，树脂粉末可能毒性很大，所以这是一个权衡的问题。

结果看起来确实不错。当然，由于仍然有打磨，它看起来有多好将取决于你的打磨工具，你的技术，以及——也许最重要的——你的耐心。打磨可以为 [3D 打印](https://hackaday.com/2021/06/04/give-3d-printed-plastic-a-well-worn-metal-look/)做很多事情。我们可能不完全相信它有树脂灰尘，但你可以用[下拉式工作台](https://hackaday.com/2021/06/04/give-3d-printed-plastic-a-well-worn-metal-look/)至少去除一些灰尘。
