# AJAX COMMAND Radio 是一个老掉牙的好东西

> 原文：<https://hackaday.com/2022/07/03/ajax-command-radio-is-an-oldie-but-goodie/>

如果你到了一定年龄，你很难接受旧收音机可能有晶体管的事实——旧收音机都有电子管，对吗？但是一台 20 世纪 60 年代或 70 年代的收音机已经超过 60 岁了，人们正在修复旧的晶体管电视机。[亚当]拿起他的第一台旧收音机，一台 20 世纪 70 年代的老式 Ajax Command S-74。

他很幸运。唯一需要的维修是更换一个腐蚀的电池盒，并清理电池上的垃圾。你可以在下面的视频中听到收音机在工作。

当然，就修理而言，这并不十分困难。但是如果你年龄合适，看到这个收音机的内部一定会勾起你的一些怀旧之情。PCB 看起来像是那个时代典型的量产板。你会很快认出中频变压器和棒形天线。

收音机这么容易工作，这几乎是一种耻辱。这些旧设备实际上很有趣，可以使用信号注入器或信号跟踪器来排除故障。事实上，我们最近报道了一种现代的[信号跟踪器](https://hackaday.com/2022/06/24/active-signal-tracer-probe-has-agc/)。过去，拥有一辆这样的甚至是希斯基特的[高级车并不罕见。](https://hackaday.com/2020/06/26/reliving-heathkits-glory-days-through-a-teardown-and-rebuild/)
