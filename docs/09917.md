# DIY 平地机盒子+石头=不需要冲击

> 原文：<https://hackaday.com/2021/04/27/diy-grader-box-rocks-no-need-for-shocks/>

在迷你自行车和附近的树林之间，[HowToLou]有很多值得我们羡慕的地方。不幸的是，土路上的地形太崎岖不平，不适合骑行。但是[Lou]看着他的割草机说，[我可以自己做](https://www.youtube.com/watch?v=8O_ogC-CUI4) (YouTube，嵌入下面)。

这种平地机箱子由重型橡木托盘制成，加上一块碎料板来完成岩石箱。[Lou]把两条带子挂在钩子上，在割草机后面拖几下，让路径变得平滑。一排方头螺栓消除了凸起，木板在几块大石头的重压下使表面变得平滑。我们认为结果看起来很好，并怀疑[楼]可以做得更好，有一个花哨的平地机盒。休息之后，请查看实际情况。

没有割草机吗？这是一个很好的理由去买一个，但是如果这不在考虑范围内，也许你可以用胶合板和自行车零件做一些合适的东西。
