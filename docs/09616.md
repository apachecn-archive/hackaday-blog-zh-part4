# 3D 打印涡流冷却火箭需要停止泄漏

> 原文：<https://hackaday.com/2021/03/23/3d-printed-vortex-cooled-rocket-needs-to-stop-leaking/>

火箭发动机以一件事而闻名，那就是变热。从材料工程的角度来看，正是这种特性使得它们的建造和运行如此具有挑战性。用先进的金属合金制造一个已经够难了，但是[Integza]坚持尝试用 3D 打印机制造一个。正在取得进展，但成功仍遥遥无期。(视频，下面嵌入。)

为了在他的发动机设计中尝试和减轻燃烧推进剂的热效应，[Integza]寻求涡流冷却。在这里，氧气在燃烧室的外缘形成涡流，在燃烧的燃料和燃烧室壁之间起到缓冲层的作用。毕竟，对于 3D 打印的腔体组件，保持尽可能低的温度是关键。不幸的是，尽管使用了一种特殊的充满陶瓷的树脂来印刷和发泡各种耐火材料的火箭组件，但它不可能阻止燃烧室泄漏。固体燃烧每次可能持续几秒钟，但最终随着墙壁的破裂，每台测试的发动机都变成了一团火焰。

幸运的是，没有人在测试中受伤，而且[Integza]对需要在下一次迭代中修复的问题有一个清晰的想法。在之前，我们已经展示过其他涡流冷却火箭——这个理论是合理的。一如既往，魔鬼在执行中。
