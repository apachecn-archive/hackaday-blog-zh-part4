# 清晰的 PS2 是我们应得的水晶版

> 原文:[https://hack aday . com/2020/10/15/clear-PS2-is-the-crystal-edition-we-worlded/](https://hackaday.com/2020/10/15/clear-ps2-is-the-crystal-edition-we-deserved/)

游戏机制造商偶尔会发布他们硬件的水晶版，但从来没有真正达到宣传的效果。生产光学透明塑料的制造现实意味着费用很少是合理的，即使是特别版。相反，我们得到的是朦胧的、烟雾弥漫的半透明盒子，相对来说不太令人印象深刻。为了纠正这一点， [[BitHead1000]提供了一个完全透明的 PlayStation2。](https://www.youtube.com/watch?v=SkNhCW1pBh8)

虽然标题称之为玻璃 PS2，但使用的切割工具和材料上的标签非常清楚地表明(一语双关)，这个建筑使用的是丙烯酸。不管怎样，它本身就是一种有吸引力的材料，更适合这样的建筑。为了获得最佳视觉效果，内部屏蔽被移除并扔进垃圾箱，代之以塑料支架来固定东西。然后将外壳组装在组件周围，让内部硬件呈现无与伦比的景象。

不可否认，当光驱打开时，看着它在机箱内工作是一件很酷的事情，几个内部 led 只是增加了这种壮观。我们已经看到[BitHead1000]完成了其他的案例改造壮举，比如喷火 N64。休息后的视频。
