# 远程推动 Tin:云中飞行控制的开始

> 原文：<https://hackaday.com/2018/09/04/pushing-tin-remotely-the-start-of-flight-control-in-the-cloud/>

在 1999 年的一部电影(推锡)中，一名飞行控制员是飞机上的一名乘客，他告诉空乘人员，他需要与控制飞机的人通话。空服员告诉他飞行员非常忙，管制员回应道，“……你真的认为飞行员在控制这架飞机吗？那真的会吓到我。”我们想知道这个虚构的人物会怎么想飞往科罗拉多州的洛夫兰。他们的[科罗拉多远程塔项目](https://www.codot.gov/programs/remote-tower)。虽然仍有人类飞行控制器，但他们并不在机场，而是依赖远程摄像机和雷达，因此控制器可以位于其他地方。

主题机场是北科罗拉多地区机场，是该州最繁忙的机场，没有塔台。虽然这个概念(一般称为远程虚拟塔或 RVT)可以追溯到 2002 年，但它的采用现在才开始加速。2015 年 4 月，瑞典的一个机场率先投入正常使用，但科罗拉多的安装是美国第一个获得批准的。如果官方网站对你来说有点太枯燥，有一个哥伦比亚广播公司的报道和一个视频让你快速了解正在发生的事情。或者观看下面的演示视频。

这项技术很简单，你可能会想为什么要花这么长时间。除了将控制器放在集中的位置，该系统还希望利用视频增强功能，尽管你可能会说你可以在适当的位置安装控制器。如果你看视频，你会看到中心是一个多屏虚拟窗口。我们想知道头戴式虚拟现实护目镜是否会是更好的选择。

你只能想首先会发生什么？完全去除人类来代替像自动驾驶汽车这样的人工智能？还是一个位于第三世界国家高层建筑中的大型飞行控制中心，它出价低于所有其他提供远程服务的飞行控制公司？我们认为其中一个会发生，如果不是两个都发生的话。

如果你想在家里玩，[巴林特] [有一个你会喜欢的系统](https://hackaday.com/2012/04/16/playing-air-traffic-controller-with-software-defined-radio/)，即使直播部分目前关闭，项目细节和视频仍然存在。我们有很多关于使用 ADS-B 追踪附近飞机的帖子，但是一家公司现在[收听所有这些帖子](https://hackaday.com/2017/12/27/aireon-hitchhikes-on-iridium-to-track-airplanes/)。
