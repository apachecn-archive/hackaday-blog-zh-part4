# 将监视器臂重新用作显微镜支架

> 原文：<https://hackaday.com/2021/05/16/repurpose-a-monitor-arm-as-microscope-mount/>

对铰接臂显微镜支架的价格感到有点震惊，更不用说到英国的运费了，[CapTec]意识到它们看起来与典型的计算机显示器臂支架基本相似。想到他可以用更少的钱改装一个显示器臂，[他启动了 FreeCAD 并开始设计](https://www.thingiverse.com/thing:4853500)。

[CapTec]正在使用它来支持他的配备 Amscope / Eakins 摄像头的三目镜显微镜，但注意到同样的机械支架/聚焦架接口也可以在双目镜上找到。他观察到底座并不比你的桌子或实验室工作台更稳定，所以要记住这一点。

最终，显示器臂花了他不到 40 美元，他估计整个东西不到 55 美元。根据他在网上搜索的价格，这意味着节省了 200 多美元。根据他的计算，运费在总成本中占了相当大的比例。我们想知道它们是否由于冠状病毒而被人为抬高——如果是这样，制造/购买价格比较可能在未来产生不同的结果。

这种类型的项目是家用 3D 打印机的完美用例——当正常供应渠道不可用或价格过高时，制作自己的零件。专为显微镜设计的关节臂与专为大型电脑显示器设计的关节臂有显著不同吗？如果你知道，请在下面评论下来。
