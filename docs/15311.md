# Z80 获得新操作系统

> 原文：<https://hackaday.com/2022/11/08/z80-gets-new-os/>

如果你对 Z80 电脑情有独钟，但想要一种新的操作系统体验，试试 [Zeal](https://github.com/Zeal8bit/Zeal-8-bit-OS) 。你可以在下面的视频中观看开源操作系统的演示。

如你所料，整个系统是用 Z80 汇编语言编写的。你所期望的功能都在那里:文件、目录、设备驱动、时钟，甚至支持高达 16M 内存的内存库。这项工作还没有完全完成，最初的目标计算机 Zeal 也没有完成，但它看起来是迄今为止的一项伟大工作，任何拥有 Z80 的人都会感兴趣。

如果你手头没有 Z80 CPU，你也可以在浏览器的[模拟模式](https://zeal8bit.github.io/Zeal-WebEmulator/)下试试。作者想要一个简单的操作系统，它不是多线程的，并且可以存在于 ROM 中。内核大约需要 6K 的 ROM 和 1K 的 RAM，具体取决于配置。系统调用和一般操作会让你想起 Linux。

有一大堆事情要做，所以如果你想参与进来，就不会缺少想法。这个世界还需要另一个 Z80 OS 吗？就此而言，这个世界还需要 Z80 系统吗？我们不了解这个世界，但是我们总是可以使用它们。他们甚至没有[要花费很多](https://hackaday.com/2017/01/02/retrocomputing-for-4-with-a-z80/)。
