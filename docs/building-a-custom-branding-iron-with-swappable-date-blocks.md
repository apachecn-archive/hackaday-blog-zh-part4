# 用可更换的日期块打造定制品牌熨斗

> 原文：<https://hackaday.com/2021/12/17/building-a-custom-branding-iron-with-swappable-date-blocks/>

可以用任何一点旧的热金属在木头上做标记，但是如果你想做得好，适当制作的工具将大有帮助。[Wesley Treat]用[这种模块化烙铁设计](https://www.youtube.com/watch?v=pTJ32NsFKoc)制造了这种产品。

品牌工具本身是用黄铜在 X-Carve CNC 路由器上加工出来的，使用[Wesley]自己的标志。零件在加工后用砂纸打磨，以去除工具痕迹。然后，一个较小的黄铜棒被加工成不同年份的数字,[韦斯利]可能希望用这些数字来标记他的项目。

熨斗本身是由他自己创造的一个漂亮的木质把手和一个支撑工具的钢垫板组装而成的，而不是胡乱拼凑起来的东西。日期可从主标志上单独移除，以便将来更改。自然地，工具图形是反着做的，这样当烧在木头上时就能正确地记录下来。

这种工具和火炬一起使用，加热黄铜，这样就可以在木制表面留下印记。最终的结果是坚实的，如果不是很完美；让整个工具的温度完全匹配是获得最干净结果的关键。闭环运行的电加热元件可能是实现这一点的一种方式。

从根本上说，这是一种快速标记木制项目的整洁方式。我们也看到木材燃烧达到了更高的高度，比如用[这台数控烙画机](https://hackaday.com/2017/08/11/turning-and-burning-with-a-cnc-pyrography-machine/)。休息后的视频。
