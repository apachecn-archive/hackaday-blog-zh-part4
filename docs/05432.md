# 安静，我的电子心脏

> 原文：<https://hackaday.com/2020/01/19/be-still-my-animatronic-heart/>

给神经质的人一个合理的警告:一些版本的[【威尔·科格利】的电子心脏](https://www.youtube.com/watch?v=8-jlu4yoHm0)足够逼真，你可能不想看下面的视频。不过这有点可惜，因为他真的花了很多精力在构建上，而且在模仿生物的动作方面，他的成果有很多值得我们学习的地方。

至于为什么需要一个电子心脏，我们还不确定。[Will]没有提到它的具体用例，尽管我们可以想到一些。随着强制浪漫日的迅速临近，织物包裹的版本将成为偷走你心的人的绝佳礼物，而硅胶包裹的版本可以用作电影道具或可怕的恶作剧。不管是什么原因，[Will]的构建是增量开发中的一个案例研究。他从一个使用单个连续旋转伺服系统的设计开始，该系统通过一个普通的曲柄为四个 3D 打印的桨提供动力。四个桨有点模仿心脏四个腔室的运动，但效果不太令人信服。下一个设计使用了两个伺服系统和复杂的平行四边形联动装置来依次扩展心脏的每一侧。它更接近了，但仍然不完全正确。

在仔细观看了一个跳动的心脏的镜头后，[威尔]决定他的机制需要模仿真实心脏的快速收缩和缓慢舒张的特征。为了实现这一点，他的最终设计有三个伺服系统和一个用于运动控制的 Arduino。滑入一个详细的硅胶外套，看起来非常现实。如果你敢，看看下面的视频。

我们以前见过大量的电子动物身体部分，从[眼睛](https://hackaday.com/2017/04/25/look-at-me-with-your-special-animatronic-eyes/)到[的手](https://hackaday.com/2015/10/25/this-animatronic-hand-is-so-metal/)到[的整张脸](https://hackaday.com/2018/07/27/animatronic-puppet-takes-cues-from-animation-software/)。不过，这可能是我们第一次看到内部器官的电子动画版本。
