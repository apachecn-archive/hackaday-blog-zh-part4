# 有了这款 WiFi 免打扰加密狗，在家工作时，家人不会打扰您

> 原文:[https://hack aday . com/2020/03/20/keep-the-family-in-bay with this-wifi-do-disturn-dongle/](https://hackaday.com/2020/03/20/keep-the-family-at-bay-while-working-from-home-with-this-wifi-do-not-disturb-dongle/)

那些在过去几周突然被介绍到在家工作的美妙世界的人可能经历了一点文化冲击。即使有今天开放的工作空间和不太正式的期望，工作也不是家。对于家里的小孩来说尤其如此，他们可能会看到妈妈或爸爸在家工作，而不是(鲍勃)在办公室里做会计。

为了平稳过渡到全职在家工作的艰难时刻，【Brian Lough】为他的办公室门准备了[这个网络“请勿打扰”信号灯](https://www.youtube.com/watch?v=ieAoZxwCmAA)。最初的想法是简单地提供一个红灯和一个绿灯，让家庭的其他成员知道[Brian]什么时候会在开会，但在一个后来证明有用的范围蔓延的例子中，Lough 夫人]重写了规范，在面向家庭的一侧包括一个按钮，以便她可以提醒他需要他出席。

[Brian]使用 ESP32 和 ESP8266 检查了几个原型。我们支持 ESP32 ,[ Brian]利用它的内置电容触摸输入。这本来可以消除一个物理按钮，但可惜的是，ESP8266 与许多许多 Blu-Tack 一起进入了最终版本。下面的视频详细介绍了构建和代码，并额外提供了一堂可爱的爱尔兰课。

是的，一条简单的短信可能就能满足要求，但这有什么运动可言呢？然后，正如[Brian]指出的，这个版本看起来[奇怪地熟悉](https://hackaday.com/2019/02/21/talking-telegram-with-the-esp8266/)是有原因的。
