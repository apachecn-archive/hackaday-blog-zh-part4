# 如何建立密封的电气连接

> 原文：<https://hackaday.com/2020/09/23/how-to-create-hermetically-sealed-electrical-connections/>

[Eric Strebel]对压力和真空罐并不陌生，他经常在产品设计业务中使用它们进行各种树脂铸造工作。然而，有时有必要在压力罐内运行设备，例如滚塑或其他类似工作。为了在压力下将能量注入油箱，[Eric] [制造了一个密封的特殊插头来完成这项工作](https://www.youtube.com/watch?v=aTc6szmAPh4)。(视频，嵌在下面。)

建造从一个旋入压力容器的大金属塞子开始，塞子上加工有一个方形凹槽。对于电气通道，[Eric]选择了 GX-16 航空连接器，在这种情况下封装了六根导线。使用裸铜线，通过金属油箱插头上的孔，将连接器背靠背连接起来。这是为了避免电线绝缘成为气体通过的通道。将连接器连接起来，并在适当的位置放置一个丙烯酸圆盘来阻止溢出，金属塞填充树脂以形成密封。

结果是好的，连接器电功能和树脂作为一个完美的密封。裸露的铜导线有短路的小风险，但[Eric]正在探索一些简单的解决方案来避免问题。我们以前也看过他的作品——[比如这个关于纸板作为设计工具的大讨论](https://hackaday.com/2018/11/14/the-value-of-cardboard-in-product-design/)。休息后的视频。
