# 基于滚轮的机器人手抓取

> 原文：<https://hackaday.com/2020/09/24/roller-based-robot-hand-grasps/>

在最近的机器人和自动化国际会议论文中，来自斯坦福的[李绅·雅恩]和其他一些人讨论了基于滚轮的机器人手的设计，它具有许多模仿人手的特征。关键特征是三个手指中的每一个都有一个带有小型齿轮马达的滚轮。

滚轮可以让手改变物体的方向，而不会失去抓力。当然，这适用于像球这样的球形对象。但视频显示，它可以操纵其他物品，如六面骰子、水瓶，甚至一张纸。通过张开手指，它甚至可以抓住你乍一看不会想到的大物体。

有一定的限制，如滑动的一些形状和文件涵盖这些。这似乎是对某些应用非常有用的事情之一，但也许不是普遍适用的。

该团队已经在开发第二个版本，用小球代替滚轴。我们期待着将来听到更多关于它的消息。
