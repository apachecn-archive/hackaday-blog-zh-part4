# 老式汽车收音机现在播放游戏和芯片音乐

> 原文:[https://hack aday . com/2019/08/27/vintage-car-radio-now-plays-games-and-chip tunes/](https://hackaday.com/2019/08/27/vintage-car-radio-now-plays-games-and-chiptunes/)

[MisterM]似乎擅长将新的电子产品挤进旧的但好看的技术中。他的最新作品聚焦于太空时代的一个样本:一个 1963 年的有趣的汽车收音机，可以从仪表板上拿出来带到任何地方。由于底部有一个闪亮的内置扬声器，节拍还在继续。

他用运行 RetroPie 的 Raspberry Pi 3 和 Picade 控制板替换了不工作的无线电设备。收音机表盘后面的 Pimoroni Blinkt LED 灯条为每个仿真控制台发出不同的颜色，我们认为这是一个不错的触摸。[MisterM]建造了这个控制台在他的工作室里玩，甚至还为它做了一个坞站。但出于对原始收音机的敬意，它是独立的，可以带到客厅或朋友家。还有一个 USB 端口，供二号玩家随时进入。对于[先生]的下一个魔术，他将转换一个 80 年代的操纵杆。

我们喜欢[MisterM]将表盘重新用作开始和选择按钮的外壳。正如他所指出的，当他疯狂地操作控制装置时，这样可以让它们不碍事。只需输入 Konami 代码即可解锁下面的构建视频。
