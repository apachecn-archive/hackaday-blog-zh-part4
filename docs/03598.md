# 项目出口:门闩的一个支架和一个钟形曲柄

> 原文:[https://hackaday . com/2019/07/15/project-egress-a-bracket-and-a-bell-crank-for-the-latches/](https://hackaday.com/2019/07/15/project-egress-a-bracket-and-a-bell-crank-for-the-latches/)

设身处地为[这个老托尼]想想:你突然收到一封电子邮件，邀请你参与制作一艘 50 年前的太空船的复制品。你会相信吗他没有，至少一开始没有，但最终证明是真的，他以自己独特的风格为 Project Egress 制作了这两个组件。

如果你没有听说过 Project Egress，看看我们对最初公告的报道。这个想法是从阿波罗 11 号指令舱*哥伦比亚*建造一个船员舱的复制品，作为下周阿波罗 11 号登陆 50 周年纪念的一部分。[亚当·沙维奇] at Tested 已经招募了 44 名黑客和制作者来帮忙，在团队中分散工作，让每个人都可以用他们喜欢的任何材料和任何方法工作。[老托尼]，也许不出所料，主要选择阿波罗时代的脱水太空级铝，加工使用手动和数控加工相结合。我们真的很喜欢他选择的表面处理——结合喷砂和手工处理，赋予它一种使命磨损的外观。

至于零件本身到底是什么,[老托尼]能想出的最好的称呼是支架和曲拐。从最初的舱门图纸来看，看起来有两个曲拐，它们将把舱门周围的力传递给门闩[【弗兰布兰奇】](https://hackaday.com/2019/07/04/project-egress-fran-makes-a-latch/)[【乔尔】【鲍勃】](https://hackaday.com/2019/07/13/project-egress-two-ways-to-latch-the-hatch/)，毫无疑问其他人也为建造做出了贡献。

我们热切期待着最终的组装，将于 7 月 18 日在史密森尼国家航空航天博物馆由[亚当]现场执行。Project Egress 既是对创客运动的庆祝，也是对阿波罗的纪念，我们很高兴人们将有机会在如此公开的论坛上看到所有这些黑客的劳动成果。
