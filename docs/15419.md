# 观看禁烟机的运行

> 原文:[https://hacka day . com/2022/11/20/see-the-forbidden-coinct-machine-in-action/](https://hackaday.com/2022/11/20/see-the-forbidden-cigarette-machine-in-action/)

[Fraens]一直在设计许多神奇的 3D 打印机，并制作演示它们如何工作的视频。最后一个装置是一个自动香烟填充机，它有一些相当复杂的动作，不知何故设法完成了工作。

虽然[Fraens]通常为他所有的机器上传 STL 文件，但这一个是被禁止的！在欧洲，销售自动装烟机是非法的，目前还不清楚在 Thingiverse 上张贴它们离法律边缘有多近。因此，在法律尘埃落定之前，你不得不[满足于这个神奇的视频](https://www.youtube.com/watch?v=scCJcUZJsKU)，也嵌入在下面。

但老实说，魔鬼的棍子对你的健康没有好处，你可能只是为了机械。思考一下这个问题——你有一斗喜欢粘在一起的烟草纤维，你需要把它们装进一个容易压扁的轻质纸管里。这些管子也不好处理。这两个问题的解决方案都需要电磁驱动的攻丝机，将两者搅拌到位。

还有一个 3D 打印的齿条和齿轮来推动，还有一个很酷的步进驱动旋转机构来将空纸放入正确的位置。这台机器严重依赖 3D 打印，但也依赖简单的五金店零件，如铝和铜管。[Fraens]的构建总是很简单，但同时也非常流畅，你会从观看这一切的过程中学到很多。

当你完成后，看看其他来自[Fraens]的文章。他的[缝纫机](https://hackaday.com/2022/07/17/a-home-made-sewing-machine-may-be-the-only-one/)、[编织机](https://hackaday.com/2022/08/04/3d-printed-braiding-machine-brings-back-some-history/)，甚至还有一台[动力织布机](https://hackaday.com/2022/06/07/3d-printed-power-loom-shows-how-complex-weaving-really-is/)，都让我们印象深刻。
