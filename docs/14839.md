# Walkmp3rson 是索尼从未生产过的 MP3 播放器

> 原文:[https://hack aday . com/2022/09/20/walk MP 3 rson-is-an-MP3-player-like-Sony-never-made/](https://hackaday.com/2022/09/20/walkmp3rson-is-an-mp3-player-like-sony-never-made/)

如果你还没有很好的意识到，90 年代就像，现在这么热，因为某些原因包括磁带。(尽管我们个人直到 1998 年才有 CD 播放器，但早在 1982 年就有了。)但是，如果你不喜欢磁带的质量，或者如果你只想带着超过 45-120 分钟的音乐，那么[约翰·埃德加·帕克]的[walkmp 3 rson](https://learn.adafruit.com/walkmp3rson-personal-mp3-tape-player/overview)绝对适合你。这发音为“Walkperson”，是 21 世纪基于 MP3 的经典随身听的升级版。

[![](../Images/5b06615a81253cdf0035afeb16ae4dcd.png)](https://hackaday.com/wp-content/uploads/2022/09/walkmp3rson-inner.jpg) 在这个令人惊叹的 3D 打印外壳中，您会发现 Adafruit Feather RP2040 控制着屏幕，处理来自旋转编码器和那些可爱的机械按键开关的输入，当然，还可以通过放大器分线板播放 SD 卡中的音频文件。不，这不仅仅是另一个 MP3 播放器——嗯，它有点像，但是演示在这里真的走了很长的路。

有很多复古现代的亮点，比如在 TFT 屏幕上播放的盒式磁带动画，四四方方的外壳，以及它涉及物理媒体的事实。哦，是的——无论何时你想换专辑/唱片/流派/什么，你都可以插入 SD 卡。事实上，这将是一个老的，更小的 SD 卡的巨大用途。你可以全力以赴，把小小的专辑封面塞进那些乳白色的塑料盒子里。休息之后，请观看简短的演示视频。

想在微控制器上播放自己的音乐，但不喜欢随身听美学？6 月份，我们报道了一款由 ESP32 驱动的类似音频播放器，它神秘地模仿了你可能会感兴趣的便携式磁带卡座。
