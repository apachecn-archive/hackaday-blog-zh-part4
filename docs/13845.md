# 量子计算:初尝免费

> 原文：<https://hackaday.com/2022/06/05/quantum-computing-the-first-taste-is-free/>

有几种方法可以通过互联网访问真正的量子计算机——通常是免费的。然而，其中大多数都是前代机器，功能有限。或许对学习很有帮助，但不是你能做任何实际事情的东西。然而，Xanadu 已经宣布了他们所谓的能够达到量子优势的计算机，任何人都可以在一定范围内免费使用。[Borealis](https://www.xanadu.ai)——正在讨论的计算机——使用光子态，并具有处理超过 216 个压缩态量子比特的能力。

该公司正在出售计算机上的时间，但免费层包括 Borealis 上的 500 万个免费镜头和早期系列量子计算机上的 1000 万个镜头。你也可以在 Borealis 上以每百万张 100 美元的价格购买现收现付服务。

虽然几百万张照片听起来可能很多，但我们注意到快速入门演示消耗了 10，000 张照片，这可能很简单。这仍然是在 Borealis 上运行的大约 500 次——在最先进的量子计算机上免费运行还不错。不过，你会想用模拟器进行调试。

鉴于你使用名为 Penny Lane 和 Strawberry Fields 的软件访问机器，我们推测开发者是披头士的粉丝。你的工作由 Python 控制，并且有一个云模拟器来保存你的镜头。

我们不会假装理解压缩光量子位和北极光结构的所有内容。但是你可以在我们关于量子计算的[系列中得到一些通用的实践。或者周围有一些](https://hackaday.com/2018/01/24/quantum-weirdness-in-your-browser/)[讲座](https://hackaday.com/2018/07/13/quantum-computing-for-computer-scientists/)，包括一个针对[不同层次体验的讲座](https://hackaday.com/2018/06/26/wired-and-ibm-explain-quantum-computing-to-students-from-grade-school-to-grad-school/)。
