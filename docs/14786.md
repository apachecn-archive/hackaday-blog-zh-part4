# 霓虹灯——不仅仅是指示灯

> 原文:[https://hack aday . com/2022/09/13/霓虹灯-不仅仅是指示灯/](https://hackaday.com/2022/09/13/neon-lamps-not-just-for-pilot-lights/)

显而易见，为什么 led 在指示灯应用中远远胜过氖灯。但是，尽管发光二极管有很多实际用途，人们发现它们至少在一个方面比它们的老指示器表亲欠缺:魅力。在 led 冰冷而扁平的地方，霓虹灯柔和的橙色光芒给美学派对带来了很多东西，尤其是在复古建筑中。

但是外观并不是这些微型发光灯的唯一优势，大卫·洛维特在他的新视频中展示了霓虹灯的一些令人惊讶的替代用途。他从古老的 NE-2 灯泡开始探索，这种灯泡已经存在很久了，详细介绍了它的一些有趣的电气特性，如启动氖放电所需的电压和维持放电所需的电压之间的差异。他还展示了一些很酷的霓虹灯技巧，比如将它们用于各种多谐振荡器电路，除了添加一些电阻和电容之外，什么都不用。当他打开 MTX90 电子管时，真正的乐趣开始了，它本质上是一个冷阴极闸流管。添加一个简单的控制网格有助于一些有趣的电路，如单管多振动器。

所有这些实验的结果对任何关注[大卫]频道的人来说都很清楚，这个频道充满了真空管的非传统用途。他建造“中空状态”计算机的努力将会得到这些霓虹灯电路的极大帮助——更不用说它们会让一切看起来有多酷了。
