# 电动工具车瑞士军刀

> 原文：<https://hackaday.com/2022/01/02/swiss-army-knife-of-power-tool-carts/>

当你认真做木工时，你最终会想要一些电动工具。有了这样的运作效率，事情很快就会变得一团糟，到处都是木屑。在户外工作并不总是实际的(或可取的)，我们中的许多人只有小作坊来制作。但是木工工具吃空间很快。集中提取是一种解决方案，但所有固定的刚性管道迫使人们固定工具位置，这并不总是一件好事。可移动的工具车并不新鲜，这些年来我们已经看到了许多解决方案，但这个由[Peter Waldraff]构建的[相当巧妙](https://www.youtube.com/watch?v=JRliu5I2xuI)(下面嵌入了视频)，在一个非常紧凑的可移动格式中包括了一些非常好的功能。

通过重新利用旧橱柜，[Peter]展示了一些真正的升级改造，几乎没有浪费，最终效果看起来也很棒！有一个集中的 M 级(我们猜测)吸尘器，带有一个可拆卸的真空管，可以很容易地拆下来挂在较小的手持工具上。这些都藏在翻转刨床附近的一个区域，随时可以使用。小型吸尘器的自动启动开关连接到较小的工具上，增加了易用性，同时减少了忘记打开开关的可能性。我们都这样做过。

对于半固定的较大工具，如斜切锯和台锯，可以将一个单独的、流速更高的可移动除尘器推过来，并连接到集成的通风室，该通风室可以收集产生的大量灰尘和碎屑。

一个很好的接触是安装在滑轨上的斜切锯部分。这允许整个组件向侧面滑动一点，为台锯提供更多可用宽度，以撕裂更宽的板材。随着一些闩锁的另一个小调整，整个斜接部分可以翻转，提供更多的访问台锯，或只是一个小工作台！开裂的东西！

需要一些帮助来更好地使用木材吗，[Eric Strebel]为你提供了一些很好的建议！如果你的需求更简单、更小、更小，这里有一架[手指大小的飞机给你](https://hackaday.com/2020/12/14/building-a-tiny-finger-plane-for-detailed-work/)。
