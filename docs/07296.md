# 美学 DIY 蓝牙音箱

> 原文：<https://hackaday.com/2020/07/30/aesthetic-diy-bluetooth-speakers/>

DIY 蓝牙音箱项目一直是 Hackady 的主要内容。在我们 DIY 音频构建的最新功能中，我们有[【Patrick 的】乙烯基圆柱形扬声器](https://www.instructables.com/id/DESIGN-AND-BUILD-YOUR-OWN-PORTABLE-BLUETOOTH-SPEAK/)。

他在全球速卖通发现了一个相当便宜的蓝牙音频放大器。然而，奇怪的是，放大器模块缺少一些对其运行至关重要的组件，所以他不得不做一点点返工。当你购买一个预制模块时，这不是你通常期望做的事情，但他肯定能胜任这项任务。

他注意到板放大器模块缺少电池保护电路，尽管板上有空间放置这些组件(可能是旧的板版本？).为了解决这个问题，他增加了自己的电池保护电路，以防止任何不必要的灾难。其次，他注意到在高音量下有很多失真，并认为在电源上增加一些电容将有助于修复失真。幸运的是，这招奏效了。

最后，这次不完全是制造商的错误，而是他个人使用所需要的改进。他希望一旦外壳安装在电子设备周围，amp 模块的板级 LED 指示灯就可见。因此，他使用内置的状态触发器作为简单晶体管电路的数字信号，为可以安装在外壳上的更亮的环形 LED 供电。这样，他可以利用固件来触发他自己的环形 LED 的板级状态指示器，而无需对 amp 模块进行任何软件修改。

现在，剩下的就是构建他 3D 打印的外壳，并将所有电子设备安装到位。我们已经非常习惯于[Patrick 的]设计总是令人印象深刻的美学，[在](https://hackaday.com/2019/06/18/digital-multimeasure-helps-you-get-the-job-done/)之前已经报道过他的一个项目，这个版本当然也不例外。干得好！
