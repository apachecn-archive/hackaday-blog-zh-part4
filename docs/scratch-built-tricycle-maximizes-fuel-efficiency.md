# 刮刮乐三轮车最大限度地提高燃油效率

> 原文：<https://hackaday.com/2021/05/16/scratch-built-tricycle-maximizes-fuel-efficiency/>

虽然自行车社区正在欢迎电动自行车的涌入，但仍有一群调谐器在边缘，他们仍打算将各种尺寸的汽油发动机绑在自行车上，并以响亮而沙哑的方式在路上嗡嗡作响。套件很容易获得，比类似的电动自行车套件便宜得多，只需一点点工作，就可以从这些小马达中挤出许多兴奋。然而，经过大量的工作，你可能最终会得到像这样的[令人难以置信的省油和完全定制的逆向三轮车【Paul Elkins】](https://www.youtube.com/watch?v=2REj9LSsIvE)。

这一建造的整个目标是提高燃油效率，因此计划最终将车辆封闭在空气动力学整流罩中，[最有可能使用他最喜欢的材料 Coroplast](https://hackaday.com/2020/06/17/diy-plastic-speedboat-for-one/) 。框架本身完全由方管手工制成，并由[Paul]按照自己的定制规格焊接而成。他用螺栓固定在一个悬架和定制的转向齿条上，用杠杆控制两个前轮，小型发动机和油箱连接在单个驱动轮上方的后部。引擎已经有十年没有启动了，但是一旦他把它组装好，它马上就启动了，他可以把他最新的原型车开出去试驾了。

虽然构建尚未完全完成，但下面的视频(构建日志中迄今为止的第 11 个)已经足以展示[Paul]多年劳动的成果。花了一段时间才得到他想要的设计，但是通过这次迭代，他终于得到了他想要的东西。
