# 特斯拉 Model S 因喷气发动机升级而获得提升

> 原文:[https://hack aday . com/2021/12/16/Tesla-model-s-gets-boost-with-jet-engine-upgrade/](https://hackaday.com/2021/12/16/tesla-model-s-gets-boost-with-jet-engine-upgrade/)

特斯拉以制造能快速加速的汽车而闻名，但总有改进的空间。[扭曲的感知]认为他的特斯拉 Model S P85D 需要一点额外的动力(尽管 0-60 MPH 或 0-97 km/h 的时间为 3.1 秒)，所以他做了任何明智的人都会做的事情:[在他的汽车后面增加三个喷气涡轮](https://www.youtube.com/watch?v=6SYT0upjNac)。

这个特别的建筑最好的部分是使它发生的工程和制造。超过 200 件作品，几乎都是个人制作的，这是一场旋风般的制作。控制面板是第一个，有一种特别聪明的 3D 打印技术，直接在平板的控制面板上刻字。然后，对于有角度会妨碍头部自由移动的部分，他反过来打印到一张塑料纸上，涂上胶水，然后将字母作为一张纸粘在板上。一层透明涂层确保字母不会脱落。

[![](../Images/07619081e34e79900d5b580003b2d32e.png)](https://hackaday.com/wp-content/uploads/2021/12/teslajet_detail2.jpg)

Using a 3D printer to apply lettering on the control panel.

他把控制电子设备安装在后备箱里，电线从车头穿到车尾。三个 Arduinos 充当喷气式飞机的控制器。之后，出现了固定引擎的支架，并把它固定在汽车的底部。不幸的是，供应有点困难，所以他不得不凑合着用手头的东西。结果，它没有像他希望的那样强劲，但仍然令人印象深刻。

[扭曲感知]在上路前做一些测试。然后，他将汽车换到空档，可以只靠喷气动力驾驶汽车，这是他的目标之一。虽然我们不喜欢在公路上测试喷气发动机的想法，但这肯定会让尾随者望而却步。

接下来，他找到一条更安静的路，做了一些速度测试。不幸的是，当时下着毛毛雨，路面潮湿，这让他 0-60 的站立时间很扫兴。电动的——只有他有 4.38 秒，而打开喷气加电动会把时间减到 3.32 秒。总的来说，这是一个令人难以置信的建筑，无论何时你在镇上，它一定会吸引一些好奇的目光。

如果你想升级你的特斯拉，也许不是喷气发动机，你可能会选择一个机器人来帮你插电？
