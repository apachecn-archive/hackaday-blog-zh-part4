# 拆除你所爱的东西

> 原文：<https://hackaday.com/2019/06/03/teardown-the-things-you-love/>

这个[二十年历史的闪光灯项目](https://www.youtube.com/watch?v=8tf7x02AyRA) (YouTube 链接，也在 break 下面)将在当前 1 美元的焊接工具中寻找家园，除了一些关键的差异。首先，它的背面有拆卸艺术家的名字，来自一个 DIY 电路板真正意味着自己动手包括艺术品，蚀刻和钻孔的时代。电池座是我们最喜欢的功能。该板不是 BOM 上的一部分，而是焊接了一些线环，并依赖于一对古老的 LR44 碱性电池，而不是我们今天都喜欢的 CR2032s。

考虑到这个项目的年代，[大克莱夫]不仅仅是为了怀旧而重访他的旧杰作，他不得不重走他的老路线，对他自己的作品进行拆解，因为原理图已经被时间遗忘了。我们认为重温旧作品是有价值的，就像考古学家接近一条古代项链一样。我们中的一些人曾经虔诚地评论我们的代码，因为害怕我们会忘记我们学习时的想法，需要被提醒这种严格性。

如果你想要另一个不需要零件号的电池座，可以试试利用薄印刷电路板的半柔性特性的电池座，或者用 T2 来伪造电池。
