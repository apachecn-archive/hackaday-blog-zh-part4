# 闪烁的杰克灯是一个简单的初学者建筑

> 原文：<https://hackaday.com/2021/10/19/flickering-jack-o-lantern-is-an-easy-beginner-build/>

杰克灯是一个有趣的万圣节传统，尽管也有一些风险。雕刻硬南瓜时很容易切掉一点手指，蜡烛比大多数家居用品更容易招致消防队员的敌意。对于寻找学习项目的电子初学者来说，[oy vind]的版本[可能是一个不错的安全赌注](https://www.build-electronic-circuits.com/halloween-project-jack-o-lantern/)。

建造从一个 3D 打印的南瓜雕像开始，它有一个合适的幽灵般的脸，尽管[Oyvind]指出没有什么可以阻止这个项目用一个真正的橙色葫芦来代替。在里面，一个 Arduino 连接着三个橙色的发光二极管。它们连接到 PWM 引脚上，每隔一段时间就会被赋予一个随机的亮度值，以创造一种令人愉快的闪烁效果。

这是一个非常简单的项目，但它也是一种非常适合向电子世界介绍新人的东西。很少出错，而且出错的代价也不高，这使它成为初学者的理想项目。从那里，[天空才是真正的极限](https://hackaday.com/2020/10/29/five-axis-pumpkin-carving/)！休息后的视频。
