# 自动坚果分拣机，打造整洁的工作空间

> 原文：<https://hackaday.com/2021/08/23/automatic-nut-sorter-for-a-tidy-workspace/>

我们都有一个装满随机硬件的抽屉或盒子。你不想扔掉任何东西，因为一旦你扔掉了，那就是你需要的东西。但是，老实说，你会很幸运在那里找到你需要的东西。进入【创新先生的】[坚果分拣机](https://www.youtube.com/watch?v=8R0kYoGLE1c)。正如你在下面的视频中所看到的，它会让混乱变得有序，至少对坚果来说是这样。

你可能会认为这种设备需要光学识别软件或其他高科技机制。但是，事实上，它只不过是一个带有速度控制器的马达。分类是通过一个像楼梯一样的塑料块来完成的。当螺母太高而无法放入下一个步骤时，它会滑出到输出料斗中。如果你想的话，你可以不用电，用一个曲柄就能转动整个装置。

钻轴需要一点机床的使用，所以这可能不是一个伟大的周末项目没有车床。像视频中提到的许多评论者一样，我们可能也不会使用杆固定器作为旋转轴承，但像这样的东西可能会运行，它可能会持续相当长的时间。很容易替换它，甚至用耦合器将轴固定到电机上，避开了几个问题。

显然，这个设备并不完美。你确实有一些错误。我们认为这是因为一个大坚果在去料斗的路上推了一个小坚果。Thingiverse 文件似乎不见了，但无论如何，这是您可能会适应自己的设计的东西。

虽然自动化程度不高，但我们已经看到了一个可以帮助分类钻头的小工具。有时你也想[按颜色](https://hackaday.com/2018/10/07/robot-sorts-beads-by-color/)分类小零件。
