# 2020 年:一切都是虚拟的

> 原文：<https://hackaday.com/2020/08/01/2020-everything-is-virtual/>

就像反乌托邦的未来突然出现了。一年又一年，我们从躲在空调过度的酒店舞厅里，与我们的肉袋伙伴们见面，到蜷缩在豆荚里盯着屏幕。我在找能让我连接到母体的接头。

但是，如果你还没有收到你的飞行汽车或你的每日身体配额，你仍然可以从一件事情中得到安慰:[所有的黑客会议](https://hackaday.com/2020/07/21/stay-at-home-hope-and-def-con-will-come-to-you/)都是现场直播的，就好像这是一个奇妙的网络未来！事实上，当我们打这封信的时候，有人正在告诉你如何在美国航空公司的航班上打印你的免费饮料，这是 HOPE 提供的服务的一部分，但这个讨论将在接下来的几天里继续。(直奔[直播流一](https://hope.net/stream.html)。)

[![](img/ad4444217850cbdefbdc9f9c6f0df2d6.png)](https://hackaday.com/wp-content/uploads/2020/07/VCF-02_2020_02.jpg) 如果说追溯计算更适合你，那么周六标志着[西部虚拟老式计算机节](https://hackaday.com/2020/07/30/saturday-vintage-computer-festival-west/)的开始，Hackaday 是该节的赞助商。([这是日程安排](http://vcfed.org/wp/vcf-west-event-schedule/)。)

下周末是安全模式下的网络防御会议。虽然我们完全可以想象[会谈](https://defcon.org/html/defcon-safemode/dc-safemode-schedule.html)和[演示会议](https://defcon.org/html/defcon-safemode/dc-safemode-demolabs.html)将如何运作，但[村庄](https://defcon.org/html/defcon-safemode/dc-safemode-villages.html)，基于共同主题的非正式会谈和聚会，将是对分布式会议的真正考验。

好吧，我承认:我真的很怀念和朋友们聚在一起，进行真正随意的对话，而事先准备好的电话会议似乎并不方便。Lobbycon 被禁闭了。但是如果你从来没有参加过这些活动，而你只是想至少体验一下演讲和演示，现在你有机会免费参加了。如果你喜欢你所看到的，如果病毒允许，我们将在明年夏天见到你本人！
