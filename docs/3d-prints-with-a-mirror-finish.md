# 镜面 3D 打印

> 原文：<https://hackaday.com/2021/12/25/3d-prints-with-a-mirror-finish/>

任何使用过 3D 打印机的人都知道，普通的 FSD 打印机绝不是镜面抛光。由于技术本身的性质，印刷品中存在着层次，并且层次之间的过渡永远不会是平滑的。此外，打印机可以使用不同的技术来沉积层，从而形成更薄的层(例如 SLA)。考虑到这些挑战，[AlphaPhoenix]开始着手[在他的 3D 打印作品](https://www.youtube.com/watch?v=SCuUQOTehqw)上创造真实的镜面效果。(视频，嵌在下面。)

正如简介所暗示的，镜子需要非常平坦/光滑的表面来反射光线。为了平滑他的印刷品，[AlphaPhoenix]首先进行了轻微的打磨，然后涂上了非常厚的两部分环氧树脂，让表面张力为他完成平滑工作。一旦干燥，银通过一些不同的喷雾沉积在碎片上。首先，使用湿润剂，防止后续溶液成珠。接下来，他喷洒两种前驱物，它们一起反应，将元素银沉积在物体表面。[AlphaPhoenix]声称他不是化学家，然后解释了该过程背后的许多化学反应，并从理论上解释了为什么溶液在混合后会分解。

他有一个优秀的第一批，然后随后几批出来污点，并决定不像镜子。正如我们前面提到的，第一步是润湿剂，它往往会与他应用的环氧树脂发生反应。然后，使用四个变量的网格搜索，[AlphaPhoenix]艰难地通过不同的配置，降落在关键的外卖上。例如，环氧树脂的固化时间和两种前体溶液之间的比例至关重要。

最近，我们报道了隐藏信息的 3D 打印镜面阵列。也许未来的版本可以使用[AlphaPhoenix]的技术将镜子集成到印刷品中？
