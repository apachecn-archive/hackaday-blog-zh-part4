# 暴风雨来到 GNU 电台

> 原文:[https://hackaday.com/2020/05/14/tempest-comes-to-gnu-radio/](https://hackaday.com/2020/05/14/tempest-comes-to-gnu-radio/)

当我们使用电脑，观看 YouTube 上卡车撞桥的视频，与母亲打电话，甚至我们中的一些人写黑客文章时，我们不知不觉地与世界分享了许多我们正在做的事情。我们的显示器、键盘和其他外围设备发出的射频辐射可以被采集和重建，让第三方看到您的工作，并有可能接触到您所有最黑暗的秘密。这是一项起源于政府机构的技术，毫无疑问，政府机构更愿意保持匿名，但有一段时间，通过软件定义无线电的魔力，它已经可供所有人使用。现在它已经到达了流行的 GNU 电台平台，拥有[【费德里科·拉罗卡】的 gr-tempest 包](https://github.com/git-artes/gr-tempest)。

他形容这是马丁·马里诺夫的《暴风雨》的翻版，该片以不适合胆小的人而闻名。目前的版本需要 GNU Radio 3.7，但他承诺会有一个兼容 3.8 的版本。我们放在中断下方的一个 YouTube 视频有一系列正在运行的示例，尽管似乎没有关于所用天线类型的信息。[也许对数周期设计是最合适的](https://hackaday.com/2017/02/20/an-overview-of-the-dreaded-emc-tests/)。
