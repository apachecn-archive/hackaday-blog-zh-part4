# 有图案的胶合板可以制作吸引人的扬声器

> 原文：<https://hackaday.com/2020/03/16/patterned-plywood-makes-for-attractive-speakers/>

在音频方面，我们已经远远超过了家庭高保真音响和立体声音响的时代。如果你没有用耳机或者直接用手机听音乐，你可以用一个砖头大小的蓝牙扬声器来播放音乐。仍然是老派的粉丝， [[Amanda Ghassaei]制作了一些具有时尚美学的书架扬声器](https://www.instructables.com/id/Patterned-Plywood-Bookcase-Speakers/)。

首先，扬声器音箱是在 WinISD 中设计的，这是一个专门为这项任务制作的软件包。对于给定的低音扬声器和高音扬声器，它有助于使外壳和端口尺寸处于良好声音的正确范围内。然后用胶合板制作面板来制作围栏。胶合板被切割和改造了几次，以制作面板，使用来自多层的图案来创建之字形外观。在音频方面，D 类放大器接收线路电平信号，然后通过定制设计的分频网络将其输出到低音扬声器和高音扬声器。

这是一个整洁的建筑，我们很想尝试一下花式胶合板技术。正如我们之前所见，正确的音箱设计会对音质产生很大影响。休息后的视频。
