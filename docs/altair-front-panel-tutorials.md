# Altair 前面板教程

> 原文：<https://hackaday.com/2020/10/30/altair-front-panel-tutorials/>

如果你还没老到能记起电脑有前面板的时代，就像帕特里克·杰克逊(Patrick Jackson)在制造了一个 Altair 8800 的复制品后发现的那样，它们的操作可能有点令人费解。算出答案后，他制作了一对展示基本原理的视频，然后开发了一个将两个数相加的程序。

即使牛郎星还是新的，前面板的日子也屈指可数了。廉价终端即将问世，MITS 很快发布了一种没有前面板的“交钥匙”系统。但是任何一个在 20 世纪 60 年代末或 70 年代初使用过小型机的人都会认为你需要一个前面板。

你可能永远也不会通过前面板来编写 Altair，但这仍然是一个有趣的一瞥，让你看到仅仅几十年前计算机是什么样子。虽然你可能会认为前面板只是一个好奇心，但手动输入引导程序并不罕见，这样你就可以从纸张或磁带上加载其他软件，通常是更好的引导程序。一些计算机甚至将早期的引导程序代码印在前面板上以供参考。

前面板还可以帮助您调试程序和硬件问题，因为您可能正看着真实计算机中的总线。当然，使用仿真器时，仿真器只是驱动前面板进行虚构，但它仍然以相同的方式工作。
