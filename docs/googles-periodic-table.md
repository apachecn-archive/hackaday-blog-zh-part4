# 谷歌的元素周期表

> 原文：<https://hackaday.com/2021/08/13/googles-periodic-table/>

互联网的好处之一是你不再需要庞大的参考书。你也不需要大的挂图。一个恰当的例子:哪个科学教室没有元素周期表？现在你可以从谷歌上查找一个交互式的。他们说这是 3D 的，我们假设这是每个原子的波尔模型的动画。你可以争论向人们展示玻尔模型是不是一个好主意，但这毕竟是我们大多数人学到的东西。

虽然该网站可能更多地针对学生，但它是查找元素属性的便捷方式，并且在视觉上也很有吸引力。你可能还记得，元素周期表中的列不是偶然出现的，所以实际的格式不会因实例的不同而不同。然而，我们喜欢 col 编码和单击元素时出现的信息面板。

我们以前并不是没有见过交互式在线表格。例如，有一张 [Ptable](https://ptable.com/?lang=en#Properties) ，或者一张来自[皇家学会的](https://www.rsc.org/periodic-table)，不多不少。如果你想商业化，总会有费希尔——一个在实验室里很有名的名字。从技术角度来看，他们的[表](https://www.fishersci.com/us/en/periodic-table.html)非常简单，但是他们对每个元素都有更长的描述。

当然，我们不会每天都去找元素周期表。但是我们有时确实需要一些数据。如果你需要[复习](https://hackaday.com/2015/09/16/dont-know-much-about-the-periodic-table-2/)如何使用它，和【威尔·斯韦特曼】谈谈。如果你喜欢让一切都成为游戏，试试[周期表战舰](https://hackaday.com/2016/02/05/you-sank-my-dysprosium-periodic-table-battleship-game/)。同时，为了额外加分，不看表格，试着找出下面视频中[汤姆·莱勒的]歌曲还缺少什么元素。
