# 照片渲染您的 3D 模型

> 原文:[https://hackaday.com/2021/03/15/photorender-your-3d-models/](https://hackaday.com/2021/03/15/photorender-your-3d-models/)

当然，你会想把你最新的 3D 设计打印出来，这样你就有了一个实物。但在某些情况下，你会想要它的渲染。如果你使用 OpenSCAD、FreeCAD 或大多数其他 CAD 程序，你可以得到一个简单的对象渲染，但如果你想要看起来真实的东西呢？【教学技术】[展示了如何使用网站](https://www.youtube.com/watch?v=TgQQO2QLJ_M)(Vectary)来获得 3D 模型的真实照片渲染。(视频，嵌在下面。)免费计划有一些限制，但大多数人不会感到困扰。

Vectary 有点像 TinkerCad 的超级版本，有很多现实建模和增强现实的选项。一些更高级的功能是在订阅计划的背后，但对于[教学技术]正在展示的内容，你可以使用免费计划。

不是特别有艺术感，我们插上桌腿延长器，结果还不错。然而，如果你有一个照明和纹理的诀窍，你可以让它看起来像你在视频中看到的那样。工具是有的，但是画笔并不能造就艺术家。为了获得最佳效果，[教学技术]不得不求助于一些高级设置。

我们不禁认为这对 [PCB 渲染](https://hackaday.com/2020/12/23/remoticon-video-kicad-to-blender-pcb-renders/)也有好处，其中一些[非常令人信服](https://hackaday.com/2012/06/05/this-is-not-real-lifelike-renderings-from-eagle-files/)。你可能认为普通的渲染已经足够好了，但是它确实有所不同，正如你在下面看到的。正如视频所指出的，您还可以添加其他资产来将事物放在上下文中，如视频结尾的房间。
