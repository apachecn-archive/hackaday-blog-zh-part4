# 雅达利方式的经典 8 位计算

> 原文：<https://hackaday.com/2020/04/29/classic-8-bit-computing-the-atari-way/>

在经典游戏世界中，甚至在 NES 出现之前，没有比雅达利更无处不在的名字了。他们著名的 2600 游戏机的销量几乎和任天堂 64 一样多，但却是在近 20 年前发布的。在许多方面，尽管犯了一些错误，导致了 80 年代初视频游戏的崩溃，但雅达利是第一个在视频游戏行业开辟道路的人。如果你想探索 Atari 时代的 8 位计算时代是什么样的，一个新的资源是[汇编各种基于 Atari 的项目](http://atariprojects.org/)。

这个网站拥有一切，从组装基于 6502 芯片的 Atari 8 位计算机，到用 BASIC 和 assembly 对它们编程，到在硬件本身上运行官方和自制游戏。这是由[Jason H. Moore]设计的，他是在 Atari 系统和他们的家用电脑周围长大的。他甚至把他的生物医学经验用在这里，为 2600 设计了一个名为 [Gene Medic](http://genemedic.org/) 的游戏，这个游戏也可以在网站上找到。

如果你生长在 70 年代和 80 年代，并且正在寻找一点雅达利怀旧，这个网站是你要去的地方。它甚至值得年轻人参观，因为 8 位世界更容易沉浸其中并学习计算机科学的基础知识。当然，如果你想把它带到另一个方向，有可能[修改旧雅达利增加一些现代便利](https://hackaday.com/2013/06/26/atari-2600-has-a-raspberry-pi-hiding-under-the-hood/)。
