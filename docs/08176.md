# 简单的感应加热器有助于自制收缩配合

> 原文:[https://hackaday . com/2020/10/26/simple-induction-heater-helps-with-home brew-shrink-fitting/](https://hackaday.com/2020/10/26/simple-induction-heater-helps-with-homebrew-shrink-fitting/)

机械师有许多整洁的商店技巧，但其中一个特别有趣的是收缩工具。冷缩配合通过在组装前在工具和夹持器之间产生温差来实现二者之间的过盈配合。一旦一切恢复正常，这两部分就可以焊接在一起了。

最简单的冷缩配合机床的方法是感应加热，商业钻机存在做这项工作。但是【Roetz 4.0】决定[建造他自己的热缩加热器](https://www.youtube.com/watch?v=rb9rXfSn1Ak)，结果相当令人印象深刻。感应加热器本身非常简单——一个 48 伏、20 安培的电源，一个现成的零电压开关(ZVS)驱动器，和一个沉重的铜线圈。当线圈通电时，由于线圈中的强磁通量，线圈中的任何金属都被快速均匀地加热。

要使用收缩器，[Roetz 4.0]首先要有一个非常干净的刀架，其钻孔尺寸略小于所需工具的尺寸。在线圈内部，钢制工具架迅速加热到可爱的深棕色，这意味着它已经达到了必要的 250-300°c。工具很快被放入现在已经扩大的孔中，该孔很快在其周围收缩。与夹头或卡盘相比，这种方法的优势在下面的视频中显而易见:跳动几乎为零，并且工具在再次穿过加热器后很容易松开。

你说你不需要收缩工具？[卡住的螺栓](https://hackaday.com/2019/05/22/set-your-nuts-and-bolts-free-with-this-induction-heater/)怎么样？感应加热器在那里也很好用。
