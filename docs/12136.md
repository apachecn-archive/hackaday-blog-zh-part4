# 铸造你自己的节日巧克力兔子，或者更确切地说是老鼠

> 原文：<https://hackaday.com/2021/12/25/cast-your-own-holiday-chocolate-bunny-or-rather-mouse/>

成型和使用模具的艺术是一门艺术。如果使用像巧克力这样易变的材料，这个已经很复杂的过程会变得更加困难。当亚历山大·查佩尔试图铸造自己的巧克力雕像时，他发现自己就在这里。

起点是每个人最喜欢的虚构电子鼠标的 3D 低多边形模型。他调整了模型，增加了偏移，这样在模型真空成型后，就有东西可以夹住了。[亚历山大]留下了四个不同的部分，他用 1 毫米 PETG 塑料真空成型。选择白巧克力来添加色素，他开始加热巧克力。加入太多的着色剂会导致混乱，所以这个过程有点反复试验。最后，他倒入巧克力并旋转，形成一层均匀的巧克力作为外壳。闪光的线条很容易用美工刀修剪。

最后要添加的是通过喷枪和食品级颜料的一点点颜色。结果是惊人的，即使技术很简单，结果也很好。如果你对制作自己的感兴趣，可以在他的网站上找到这些文件。如果你对巧克力更巧妙的铸造技术感到好奇，看看[创造性地使用衍射光栅获得彩虹色巧克力](https://hackaday.com/2021/05/23/iridescent-rainbow-chocolate-just-add-diffraction-grating/)。
