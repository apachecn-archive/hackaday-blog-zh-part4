# 将 Atari 2600 转换成家用电脑；这有用吗？

> 原文：<https://hackaday.com/2020/02/29/converting-an-atari-2600-into-a-home-computer-did-that-ever-work/>

[Tony]发布了一个有趣的视频，在视频中他看到了 Atari 2600 以及许多公司试图将其转换为真正的家用电脑的方式。这让我们想起了 ColecoVision，它最初是一个视频游戏，但可以扩展到一台相当合理的计算机。

将一款相对不景气的 Atari 视频游戏转换成电脑似乎有些愚蠢，但请记住，在那个年代，电脑相当昂贵。更不用说，雅达利本身在当时也是一项公平的投资。

[托尼]找到了四个选项，尽管它们似乎都不是很受欢迎。一个看起来像卡带播放器，可以插入你的盒式磁带端口和键盘端口，是一个看起来很便宜的键盘。[托尼]认为它和“驮运”一起，从未真正进入市场。

雅达利也加入了毕业生的行列。花 79 美元，你就能得到 8K 内存和薄膜键盘。有一个大的公关推动，包括一个非常时期的电视广告，你可以在视频中看到。

显然，雅达利和毕业生的实际设计师之间的纠纷导致雅达利扼杀了这个没有销售的项目。到目前为止，在这三个[托尼]中，没有一个被出售给公众。

第四个是 CompuMate，售价 79 美元。你会得到一些额外的内存和一个奇怪的薄膜键盘以及一个盒式端口。如果你想看胆量，快进到 13:30。像许多老式计算机一样，它会在基本提示符下启动。与许多其他电脑不同，它还可以播放《一闪一闪小星星》。

屏幕分辨率很差。显然，CompuMate 使用的 BASIC 的风格没有很好的记录。一个 100 行的程序填满了内存，当你想到今天你的电脑甚至你的手机有多少内存时，这是很有趣的。

很难意识到，不久前这种附加功能可能是有意义的。你可以想象 2080 年的计算机会是什么样子。

如果你想让[写原生的](https://hackaday.com/2019/04/22/get-coding-with-this-atari-2600-development-suite/)，做一点工作就有办法做到。现在有很多方法可以让[在一个更小的包装里拥有相当于](https://hackaday.com/2017/07/08/atari-2600-in-a-game-cartridge/)的 2600——甚至更多。
