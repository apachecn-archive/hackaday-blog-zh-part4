# 我们其他人的无人机

> 原文：<https://hackaday.com/2022/09/29/a-drone-for-the-rest-of-us/>

任何一个在圣诞节早上试图把四轴飞行器从树上摇下来的人都可以证明，控制这些快速移动的遥控飞行器是很棘手的，需要一点练习才能掌握。[Erik]想把这个简化一点，这样他的孩子和朋友就可以和他一起比赛，最终的结果是一架只需要两个输入就能飞的[无人机。](https://sites.google.com/view/ejowerks-speeder/)

他简化控制的实验结果产生了一种“speeder”类型的无人机，由于一个极快的飞行时间传感器，它试图自己保持离地面一定的距离。然后，飞行员只能控制油门和转向，这意味着[埃里克]可以使用手枪式的遥控控制器来控制这些机器。它们与四轴飞行器有一些相似之处，但因为它们需要在飞行中保持水平，所以它们的背部还有第五个螺旋桨，类似于飞艇。这允许一个完全独立的推力控制，而不是通常在四轴飞行器上可用的。

最终的车辆飞行起来非常直观，更像一辆遥控汽车，而不是四轴飞行器。这也需要相当多的处理能力来从单个转向输入计算适当的滚动和偏航，但在许多原型之后，结果令人印象深刻，特别是因为它也是使用 FPV 作为控制手段而建造的。下面的一个视频演示了这个视频，看起来飞行非常有趣，我们不介意看一场这些类型的飞车比赛，就像我们在过去看到的[一群吊舱比赛四旋翼飞行器](https://hackaday.com/2014/10/06/quadrotor-pod-racing/)。
