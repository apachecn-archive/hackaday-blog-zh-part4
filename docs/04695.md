# Axiom，100+kW 电机控制器，用于驱动大型电机

> 原文：<https://hackaday.com/2019/11/04/axiom-a-100kw-motor-controller-fo-making-big-motors-move/>

我们已经看到了很多机器人的电机驱动板和奇怪的电动滑板。我们还没有看到的是[一个大到足以放进电动车的东西。](https://hackaday.io/project/164932-axiom-100kw-motor-controller/)

Axiom 电机控制器是 [bootstrap](https://hackaday.com/2019/06/14/meet-the-bootstrap-winners-of-the-2019-hackaday-prize/) 竞赛的获胜者，也是 2019 年 Hackaday 奖的[决赛选手。](https://hackaday.com/2019/09/10/meet-the-20-finalists-in-the-2019-hackaday-prize/)驱动器的目标是全天以 400 伏提供 300 安的连续电流。这是一块看起来相当紧凑的电路板所能提供的惊人功率。

该设备的大脑是基于 [VESC 项目](https://github.com/vedderb/bldc)的 Lattice running 软件的 ice40 FPGA。随着时间的推移，它的开源根源肯定会允许一些有趣的黑客和一个越来越稳定的平台。更不用说现有的软件工具将有助于有时繁琐的电机驱动器调整过程。

主板设计是可用的，但我们同意团队的意见，组装的复杂性可能会很高(以及价格)。进入这个复杂套件的研究和技能数量有点令人难以置信，但我们希望它真的能够实现一些很酷的黑客，从汽车到全地形车，甚至可能是电动飞行器。
