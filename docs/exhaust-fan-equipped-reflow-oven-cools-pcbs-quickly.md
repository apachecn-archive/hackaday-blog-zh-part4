# 装有排气扇的回流炉可快速冷却 PCB

> 原文：<https://hackaday.com/2020/12/03/exhaust-fan-equipped-reflow-oven-cools-pcbs-quickly/>

对于回流焊接，有时接近就足够了。在一天结束时，家庭游戏玩家真的只需要一个电炉或一个旧烤箱和一个校准的 Mark I 眼球就可以获得不错的结果。[这款装有排气扇的回流焊炉](https://nabil.me/2020/12/01/diy-reflow-oven-w-exhaust-build-part-1/)试图控制回流焊热循环中可能更具挑战性的部分——冷却。

Nabil Tewolde 并不热衷于回流焊学校，他从一个废弃的烤面包机烤箱开始，希望能成为一个更精确的回流焊烤箱。添加了必要的温度传感器和固态继电器，以及一个 Raspberry Pi Zero W 和一个小型液晶显示器。添加冷却辅助开始于在烤箱的后壁切割一个大孔，然后填充一小段 HVAC 管道和步进控制的风门。导管的远端装有一个 PC 冷却风扇；虽然使用塑料风扇从烤箱中排出热空气似乎有些粗略，但[Nabil]表示，当废气到达风扇时，其实并没有那么热。在热曲线的回流阶段结束时，风门打开，风扇启动，快速冷却烤箱内部。

不幸的是，[Nabil]仍然需要打开烤箱门才能获得像样的气流；承认新鲜空气会有所帮助似乎是另一个扫兴的人。这将使事情变得有点复杂，但它仍然不会像我们看到的一些回流构建那样过分。话又说回来，校准眼球的事情也可以很好地工作，即使没有烤箱的[。](https://hackaday.com/2014/03/30/reflowing-with-a-hair-straightener/)
