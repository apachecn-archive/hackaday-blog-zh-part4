# 一个四肢瘫痪的病人如何使用假肢再次能够自己吃饭

> 原文：<https://hackaday.com/2021/01/02/how-a-quadriplegic-patient-was-able-to-eat-by-himself-again-using-artificial-limbs/>

30 年前，[罗伯特“布兹”奇米莱夫斯基]十几岁时遭遇了一次冲浪事故。由于 C6 脊髓损伤，他成了四肢瘫痪。在成为约翰·霍普金斯大学脑机接口研究的参与者后，他最近能够通过使用假肢来养活自己[。这些假肢最引人注目的地方主要是与[Buz 的]大脑的神经连接，由于一个将肢体感觉输入传输到患者大脑的闭环系统，这使他不仅可以控制假肢，还可以感觉到它们正在触摸的东西。](https://www.jhuapl.edu/PressRelease/201210-APL-advanced-prosthetic-limb-brain-control-Chmielewski-feeds-himself)

问题中的假肢是约翰霍普金斯应用物理实验室(APL)的[模块化假肢](https://www.jhuapl.edu/Prosthetics/ResearchMPL) (MPL)。约翰霍普金斯医学院[脑机接口研究](https://www.hopkinsmedicine.org/physical_medicine_rehabilitation/research/tetraplegia-brain-study.html)始于一年前，当时【Buz】的大脑中植入了六个微电极阵列(MEA):一半在运动皮层，一半在感觉皮层。在接下来的几个月里，这项研究专注于使用第一组阵列的信号来控制执行器，如 MPL。第二组阵列用于研究如何刺激感觉皮层，以使患者感觉假肢就像感觉生物肢体一样。

这项研究之所以如此有趣，不仅是因为它采用了闭环方法，为患者提供了关于假肢位置和压力的反馈，还因为它涉及大脑的两个半球。结果，仅仅经过一年的研究，[Buz]就能够同时使用两个 MPLs 来养活自己，这是一项微妙而复杂的任务。

在休息后嵌入的视频中，人们可以看到[Buz]在研究开始时和今天的对比，当时他在没有帮助的情况下设法处理餐具和吃蛋糕。
