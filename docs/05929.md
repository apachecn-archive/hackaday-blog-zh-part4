# 螺栓、黄铜和加工印章组成了这个小小的组合保险箱

> 原文：<https://hackaday.com/2020/03/10/bolts-brass-and-machining-chops-make-up-this-tiny-combination-safe/>

又是一天，又是一个视频，让我们严重怀疑放弃购买车床来养活家人是否是一个价值主张。这一次，[制造者 B]向我们展示了机床女王可以通过将几个螺栓变成一个微型保险箱来做什么。

我们将在前面声明，这个建筑不会从一个螺栓中获得所有材料。它更像是两个螺栓和一些奇怪的黄铜片，但这丝毫没有减损最终产品。[制造者 B]在车床上卸下两个粗壮的不锈钢螺栓的六角头和螺纹，形成两个紧密配合的嵌套圆柱体。一根黄铜棒被加工成一把钥匙，插在嵌套圆筒上的槽中，而黄铜圆盘形成了号码盘。每个圆盘的圆周上都印有 26 个字母；我们认为用于冲压的夹具非常聪明，并产生了整洁的印象。通过在每个光盘中的一个字母旁边放置一个大头针来设置这种组合，可以保护这个小保险箱中公认的有限内容，但功能性几乎不是重点。这都是关于工艺和加工技能，我们喜欢它。

如果你最近感觉到像这样的资源受限的构建有所增加，你并不孤单。“一个螺栓的挑战”造就了[这款加工精美的密码锁](https://hackaday.com/2019/07/18/turning-a-single-bolt-into-a-combination-lock/)，以及[这款一个螺栓雕塑的艺术性](https://hackaday.com/2020/03/01/single-bolt-transformed-into-a-work-of-art/)。我们都赞成保持这一趋势。
