# SLA 3D 打印涡流冷却火箭发动机

> 原文：<https://hackaday.com/2020/12/08/sla-3d-printed-vortex-cooled-rocket-engine/>

3D 打印是原型和开发的一个令人难以置信的工具，但材料的属性可能是功能部件的一个限制因素。[AX Technologies]的[Sam Rogers]和同事们一直在测试和开发一种小型的[液体燃料火箭发动机，并成功地使用涡流冷却来保护树脂 3D 打印的燃烧室](https://www.youtube.com/watch?v=tqEfbJRKzeQ)。(视频，嵌在下面。)

涡流冷却的工作原理是将氧气沿切线方向喷入燃烧室，就在发动机的喷嘴内部，这就沿着燃烧室壁产生了一个冷却、旋转的涡流边界层。氧气移动到燃烧室的前端，在那里与燃料混合并在中心点燃。这并不能保护喷嘴本身，喷嘴在变得不可用之前只能持续几秒钟。然而，由于试验发动机的模块化设计，每次试验只需重新打印小喷嘴部分。虽然这部分可以使用金属 3D 打印机制造，但成本仍然非常高，尤其是在这个实验阶段。透明树脂部件还允许观察燃烧，并从每次测试中得出更准确的结论。

这种发动机打算用作一个更大的火箭发动机的火炬点火器。燃料被喷入燃烧室的前部，在那里有一个火花塞来点燃氧-燃料混合物。氧气和燃料的流量由连接到微控制器的两个伺服操作阀控制，微控制器与发动机一起安装在线性轨道上。这使得测试发动机可以自由移动，并推动测压元件来测量推力。火花是在气门打开之前产生的，以防止延迟点火，延迟点火会使发动机爆炸，正确的气门顺序和正时至关重要。经过多次迭代和毁坏零件后，[AX Technologies]团队成功点火，在排气中形成清晰的超音速马赫菱形图案。

这只是 3D 打印和廉价电子产品在有限的预算下取得令人印象深刻的进展的又一个例子。另一个例子是【乔·巴纳德】在让[模型火箭搭载固体燃料发动机](https://hackaday.com/2020/11/27/so-close-to-landing-a-model-rocket-on-its-tail/)自行着陆方面的进展。公司和组织已经在火箭发动机中使用 [3D 打印组件几年了，我们甚至已经看到了](https://hackaday.com/2018/01/29/smaller-and-smarter-the-electron-rocket-takes-flight/)[开源版本](https://hackaday.com/2015/03/10/open-source-3d-printed-rocket-engines/)。
