# 5 千瓦的 3D 打印喷射船

> 原文：<https://hackaday.com/2019/11/14/5-kilowatts-in-a-3d-printed-jet-boat/>

无线电控制项目过去通常由金属或木材等材料制成，并涉及大量手工制作的零件。这仍然是一种方式，但 3D 打印近年来已经成为一种流行的工具。[RCLifeOn]一直在研究一艘 3D 打印的喷射船，它最近得到了一次重大的动力升级。

该船安装了一个 5000 瓦的无刷电机，对于一个重量不到 2 公斤的船只来说，这是一个很大的功率。它由一个 12S 锂电池组供电，并配备了一个用于冷却的水套，在最初尝试 DIY 传动系统失败后，它通过一个现成的涡轮机驱动船只。

这个项目中最大的问题来自于电机和涡轮机的耦合。3D 打印的耦合器无法承受这种压力，而由于缺乏车床，制造金属零件的尝试失败了。最终，通过菊花链将两个现成的部件连接在一起，找到了解决方案。

这艘船在水上证明了自己的能力，大型发动机证明了它能够以强劲的速度移动船只。这是一个很好的测试，最终零件会在动力冲浪板中找到自己的位置。我们以前也看过[RCLifeOn]的作品，[比如这些时尚的 3D 打印运动鞋](https://hackaday.com/2018/10/05/3d-printed-sneakers-are-now-a-thing/)。休息后的视频。
