# FindyBot3000 正在监听并准备提供帮助

> 原文:[https://hack aday . com/2019/04/10/findybot 3000-is-listening-and-ready-to-help/](https://hackaday.com/2019/04/10/findybot3000-is-listening-and-ready-to-help/)

每个制造商都曾面临过这样的问题——如何组织车间里不断增长的大量零部件。有些人放弃了，只是带着一箱又一箱杂乱无章的零件生活。然而，这对[Inventor22]来说还不够好，他发明了 FindyBot3000 来解决这个问题。

第一步是采购一套我们都熟悉和喜爱的微型组件抽屉。然后，它们与 WS2812B LED 灯带相结合，充当每个抽屉的指示器。一个粒子光子被用作操作的大脑，并驱动条带。到目前为止，一切顺利。

当然，闪烁的发光二极管很棒，但是语音控制才是真正有趣的地方。通过 Google Home 和 IFTTT，可以给粒子光子下命令。这可用于管理抽屉中的零件，以及快速突出显示各种组件的位置。它由 Azure 后端支持，后者管理组件数据库并跟踪一切。

这是一个整洁的构建，去除了微小的粘性标签，并且可以随着部件的更换而动态地重新配置。当然，如果您主要储存 SMD 零件，[您可能更喜欢基于卷轴的解决方案。](https://hackaday.com/2016/02/11/clearly-the-best-way-to-organize-smd-parts/)休息后的视频。
