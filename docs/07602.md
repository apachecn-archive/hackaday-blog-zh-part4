# 圆形吉他创造了声音之墙

> 原文：<https://hackaday.com/2020/08/26/circle-guitar-creates-wall-of-sound/>

在 60 年代，一种被称为“声音之墙”的音乐录制技术开始崭露头角，这种技术允许艺术家创作复杂的音乐层次，从而产生一种新颖、丰富的管弦乐感觉。虽然这种技术产生了一些里程碑式的专辑(例如沙滩男孩的宠物声音)，但它需要整个录音室和许多音乐家来制作。[另一方面，这种吉他只需要一个音乐家](https://www.youtube.com/watch?v=l5sePz48ejM)，但由于一些巧妙的工程设计，它可以自己创造出令人印象深刻的声音墙。

这种新型乐器名为 Circle Guitar，由安东尼·狄更斯(Anthony Dickens)创作，其特点是在吉他的拾音器周围有一个不断旋转的轮子。各种拨片可以以不同的方式连接到轮子上，从后面连续拨动琴弦。这超过了普通吉他弹奏者自己能够做到的，但吉他弹奏者能够通过使用几个开关和按钮来控制声音，这些开关和按钮控制一个[六音亨巴克](https://en.wikipedia.org/wiki/Pickup_%28music_technology%29)并且能够随意静音单根琴弦。当然，现在是 21 世纪，它也广泛使用 MIDI，安东尼甚至提到了使用 Teensy。

虽然这个项目的细节无可否认地稍纵即逝，但下面链接的视频非常值得一看，因为这种吉他能够产生有趣的声音。或许再配上一个经典的吉他放大器，它也能产生其他令人印象深刻的声音。无论哪种方式，我们可以期待像[【布赖恩·威尔逊】](https://www.youtube.com/watch?v=lD4sxxoJGkA)这样的人一旦投入生产就会对其感兴趣。
