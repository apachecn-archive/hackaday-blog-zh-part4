# 羽毛“FAUXBERRY”现在是真的了

> 原文:[https://hack aday . com/2019/06/20/the-feather-faux berry-is-now-a-real-thing/](https://hackaday.com/2019/06/20/the-feather-fauxberry-is-now-a-real-thing/)

上个月，我们展示了一个来自 Hackaday.io 的有趣项目，该项目基本上是试图[重建标志性的黑莓外形，以用于 Adafruit 的羽毛开发板系列](https://hackaday.io/project/165450-handheld-feather-wing)。这将允许你插入从 LTE 到分组无线电的所有模块，为手持黑客打开了几乎无限的可能性。唯一的问题是，它实际上还不存在。

但是最近创造者[arturo182]写信告诉我们，不仅所有的零件都到了，而且他已经完成了第一个原型的组装。他甚至制作了一个关于设备当前状态的视频，你可以在休息后看到。简而言之就是:它有效，而且看起来*棒极了*。

对于那些可能没有在左右第一次看到这个项目的人来说，前面有一个 2.6 英寸 320×240 的触摸屏显示器，四个通用按钮，一个用于视觉状态显示的 RGB NeoPixel LED，一个五向操纵杆，以及一个可以说是展览的明星，一个最初为黑莓 Q10 设计的 QWERTY 键盘。在背面，它有一个 SD 卡插槽，一个用于您选择的羽毛模块的插座，以及一些方便的 GPIO 扩展垫，您可以将自己的硬件连接到其上。

[arturo182]说，他正在考虑一些表面上的变化，但总的来说，一切都正常，他认为 PCB 基本上完成了。他将很快向那些表示有兴趣帮助他开发项目的个人发送一些测试单元，然后…嗯，他真的不确定接下来会发生什么。鉴于他对该项目的兴趣，某种形式的商业发布似乎是合乎逻辑的结论，但他还没有完全确定这将是一个套件还是组装设备。

在那之前，任何人如果想找一个口袋大小的设备，让他们用物理键盘[敲打一些 Python 代码，就必须坚持使用他们的 TI-83](https://hackaday.com/2019/04/04/ti83-gets-circuitpython-upgrade/)。
