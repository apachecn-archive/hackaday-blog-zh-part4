# 超外差无线电解释

> 原文：<https://hackaday.com/2019/01/02/superheterodyne-radios-explained/>

普通大众认为有一种东西叫做收音机。当然，他们知道有接收不同频道的收音机，但除此之外，一个收音机和另一个几乎一样。但是如果你从事电子学，你可能知道收音机内部工作的许多方式。晶振与调频立体声有很大不同，与通信接收机也有很大不同。我们可以说，接收机有几种常见的架构，其中最常见的是超外差。但是这到底是什么意思呢？[科技连线]有一个非正式的解释视频，讨论了超级热是如何工作的，以及为什么它很重要。你可以看下面的视频。

工程总是建立在抽象的基础上。尤其是现在，当你可以得到一个 IC 或模块，做你想让它做的大部分事情。但即使没有这些，你也很难通过开采铜线、提炼铜线和自己拉丝来启动一个电子项目。你可能不会自己制作很多电阻和电容，也不会从基本的电子方程开始设计。但是有一个抽象概念我们经常忘记:架构。如果你在设计接收机，你很可能不会去尝试解决无线电接收的问题；取而代之，你选择一个已经被证明和设计架构。

还有其他例子。真的每次都算出二进制计数器吗？或者如何做一个运放放大器？你从这些积木开始。当然，真正的创新意味着你必须停止这样做，真正想出新的不同的(可能更好的)方法来完成同样的任务。但大多数时候，你不是在尝试创新，而是在努力完成工作。

这个视频非常简单明了，并且没有假设你有太多的无线电背景。然而，它确实设法做了一些真实的演示，值得一看。当然，还有许多其他接收机架构。再生式、超再生式、零差式(直接变频)、希尔伯特式和韦弗式都是接收器的类型，无疑还有更多。有趣的是，我们今天仍在使用的许多想法都来自一个人:[埃德温·阿姆斯特朗](https://hackaday.com/2018/07/24/edwin-armstrongs-battle-for-fm-radio/)。
