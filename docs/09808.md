# 用这个测力计检查你的发动机是否正常

> 原文：<https://hackaday.com/2021/04/14/sanity-check-your-engines-with-this-dynamometer/>

当你准备打开你的遥控汽车的引擎盖来升级你的马达时，你有没有想过这些小装置能给你带来多大的扭矩？当然，我们可能只是查找电机规格，但为什么要相信制造商这样的事情，否则你可以自己测量？[Johnny 90]就是这么做的，在家里组装了一辆[钻机](https://www.youtube.com/watch?v=0ESOYhOh28c&ab_channel=JohnnyQ90)几乎是用库存的现成零件制造的*。*

为了挖掘细节，[JohnnyQ90]为自己建造了一个 [Prony 制动测力计](https://www.youtube.com/watch?v=AaQtIzgxCWI&ab_channel=Ifinfotech)。这些设备的电机轴松散地连接到杠杆臂上，杠杆臂可以向下推动像天平这样的测力设备。接上杠杆后，我们给马达加电。通过逐渐增加电机轴的“紧密度”，我们引入了与电机“对抗”的滑动摩擦，结果是，在平衡状态下，测得的扭矩是给定速度下可能的最大值。继续增大摩擦力，我们可以完全让电机失速，从而测量出电机的*失速扭矩*。

用这样的构造武装自己可以给我们一种方法来检查制造商对我们自己的评级，或者甚至获得我们从垃圾箱中取出的那些“神秘马达”的评级。[Johnny 90]的构建很好地提醒了我们如何利用一点物理知识和一些家用物品来获得一些有意义的数据。

但事实证明，Prony 制动测力计并不是测量电机扭矩的唯一方法。对于盘式制动器的灵感，看看这个[最终项目](https://hackaday.com/2017/12/18/a-dynamometer-for-measuring-motor-power/)。如果你想变得更大，让两个发动机与[杰瑞米·费尔丁的][T2 的]更大规模的建造并驾齐驱。
