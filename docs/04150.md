# 为辐射而战

> 原文:[https://hackaday.com/2019/09/10/warwalking-for-radiation/](https://hackaday.com/2019/09/10/warwalking-for-radiation/)

找不到你家附近最近更新的放射性调查？试试【猎人龙】的 [DIY 闪烁计数器 warwalking rig](https://www.youtube.com/watch?v=ReM5nC2OOUU&feature=youtu.be) 。(视频也嵌在下面。)看起来像一个油漆罐，有一根 BNC 电缆通向一个不起眼的灰色盒子，实际上是一套完整的辐射测量工具。

金属漆罐内有一个[闪烁计数器](https://en.wikipedia.org/wiki/Scintillation_counter)，它的工作原理是在光电倍增管的末端附着一个被电离辐射击中时会发光的东西，使最微弱的击中“可见”。BNC 电缆连接到树莓 Pi、触摸屏、GPS 和高压转换器，这些都是光电倍增管工作所需的。

结果是一个灵敏的辐射探测器，当[亨特]带它出去散步时，它记录 GPS 坐标和每秒计数。剧透:他发现一些当地的柏油路有点放射性，甚至发现了一个真正的“热点”。谁知道外面还有什么？有了这样的装备，制作你当地环境的辐射图简直就是在公园里散步。

[亨特]从[大卫·普鲁奇] 的旧建筑中获得了油漆罐探测器的灵感，它使用民防盖革计数器作为高压源。如果你周围没有 CD 盖革探测器，[亚历克斯伦古]的[进入 Hackaday 奖](https://hackaday.io/project/166505-diy-scintillation-detector)从头开始建造闪烁探测器。
