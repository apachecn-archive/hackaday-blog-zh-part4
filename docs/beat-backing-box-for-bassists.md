# 贝斯手用拍子盒

> 原文：<https://hackaday.com/2022/12/23/beat-backing-box-for-bassists/>

摇滚乐队的灵魂是它的节奏部分，通常由一名鼓手和一名贝斯手组成。如果你不相信这一点，试着听听这两个人不能保持正确时间的乐队。乐队经常可以用马虎的吉他和人声(这就是朋克成为一种流派的原因)，但是没有这个基础，你将很难在任何演出中得分。不幸的是，鼓既笨重又昂贵，而且很难找到好的鼓手，所以如果你是一个有抱负的贝斯手，想自己练习弹奏一个坚实的凹槽，试试这个由[邓肯·麦金太尔]设计的鼓机。

架子鼓被设计成对正在积极演奏另一种乐器的人来说尽可能的用户友好，这意味着所有的触觉输入和没有触摸屏。顶部的几行按钮为音序器选择鼓的声音，每一列对应不同的节拍，允许快速选择和改变自定义模式。还有其他几个音量和速度控制，因为它基于使用 VS1053 芯片的 MIDI，并使用 STM32 微控制器，所以它很容易配置，也可以快速与其他机器接口。

对于任何想要自己构建的人来说，所有的电路原理图和代码都可以在 GitHub 上找到。不过，如果你对数字设备感到厌恶，可以看看 t [的鼓机，它使用完全模拟的电路产生节奏。](https://hackaday.com/2013/12/17/analog-drum-machine/)
