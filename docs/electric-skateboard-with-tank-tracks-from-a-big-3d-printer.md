# 电动滑板与坦克轨道，从一个大的 3D 打印机

> 原文：<https://hackaday.com/2020/07/31/electric-skateboard-with-tank-tracks-from-a-big-3d-printer/>

地面车辆的一个基本事实是，它们总是比坦克履带更酷。也许不是更好，但绝对更酷。[伊万·米兰达]把这放在心上，可以说是 YouTube 上 3D 打印坦克项目之王。他用他最新版本的[巨型 3D 打印机](https://www.youtube.com/watch?v=ru5N2d2n_4c)建造了一个巨型 [3D 打印电动滑板，上面有坦克履带](https://www.youtube.com/watch?v=glfiX1KOtyk)。休息后的视频。

滑板由一个大型钢架组成，两端有履带式转向架。大多数转向架部件都是 3D 打印的，包括车轮和轨道，每个转向架都由无刷电机通过皮带驱动。一些弯曲被添加到钢架上，只为他的台虎钳添加了 3D 打印插件。转向架通过一个标准的滑板转向架安装在框架上，通过倾斜甲板，它可以像普通滑板一样转向。看起来这在光滑的混凝土地面上工作得很好，但是我们怀疑在轨道不能滑动的粗糙表面上转弯会更困难。我们要等下一段视频才能进行全面的实地测试。

这款滑板的大型部件是在[Ivan]的 MK3 版巨型 3D 打印机上打印出来的。虽然它与[之前的版本](https://hackaday.com/2019/12/18/giant-3d-printer-for-giant-projects/)非常相似，但在关键领域进行了改进。滑动床架的重量减少了近 50%，轮子是旋转的，所以它们骑在它下面的挤压物的顶部，而不是它的侧面，这有助于轮子的寿命。这也允许通过转动每个轮子上的偏心垫片来调平底座。通过增加更多的铝型材来增加底座和 x 轴梁的刚度。虽然他没有明确提到打印量，但它看起来与上一版本相同，都是 800x500x500。对于除 PLA 之外的材料，我们怀疑需要加热的构建室，以便有机会在没有过度翘曲的情况下制作大的印刷品。
