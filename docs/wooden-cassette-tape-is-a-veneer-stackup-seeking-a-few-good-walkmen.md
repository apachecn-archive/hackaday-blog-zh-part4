# 木制盒式磁带是寻找一些好的随身听的贴面叠层

> 原文：<https://hackaday.com/2021/01/10/wooden-cassette-tape-is-a-veneer-stackup-seeking-a-few-good-walkmen/>

虽然对几乎所有人来说，盒式录音带的时代早已过去，但磁带在其他一些领域，如大规模数据备份中仍有广泛的用途。那些仍在用它存储音乐的人是音频发烧友中的一个特殊群体。[弗兰克]仍然有一个工作的录音机，和对经典的非乙烯基声音的热情。他对录音带的敬意？[建造一个(几乎)完全由木头制成的工作卡带](https://www.youtube.com/watch?v=N4BU5hx49K4)。

该磁带是仿照以前流行的 Maxell XL-II 和第一个版本的建立是在纸上。一旦圈地的精确尺寸确定下来，[Frank]就开始用木材按照 2D 工艺建造最终版本。他用绘图仪从木质单板上切下几层，然后一层一层地粘在一起。这个建筑令人印象深刻的部分是磁带卷轴轴承也是由木材制成的，使用一小块作为比赛，保持卷轴没有太多的摩擦。

一旦所有的东西都拼在一起，粘在一起，[弗兰克]就有了一个完美的工作磁带，除了磁带和几个直接处理磁带的关键塑料部件外，它完全由木头制成。这是一件令人印象深刻的木工作品，与几天前的实木拱廊橱柜没有什么不同。
