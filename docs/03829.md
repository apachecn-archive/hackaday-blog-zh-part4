# GENIAC 又复活了

> 原文:[https://hackaday.com/2019/08/08/the-geniac-lives-again/](https://hackaday.com/2019/08/08/the-geniac-lives-again/)

[迈克·盖迪]将他在软件开发领域的职业成功归功于这样一个事实，即他可以接触到一种基于逻辑的教育游戏，这种游戏现在已经不存在了。回到 20 世纪 60 年代，对电子产品或新兴的计算机世界感兴趣的孩子不能只拿起一个微控制器或树莓派。他们必须用一套工具自己组装“计算机”。

其中一个工具包是 GENIus 几乎自动化的计算机(GENIAC ),这种产品在今天非常罕见，基本上无法获得。利用他在网上收集的图像和文档，[Mike]不仅设法创造了一个 GENIAC 的功能复制品，而且他甚至还擅自修复了 60 多年前的原始设计的一些问题。

[![](../Images/0946412c953649ab5e9143a2244f208f.png)](https://hackaday.com/wp-content/uploads/2019/08/geniac_detail.png) 从根本上说，GENIAC 是由旋转开关组成的，这些开关相互馈入以执行基本的逻辑功能。用一排白炽灯泡作为输出，用户可以观察将开关放在不同的位置会如何影响结果。

这对现代观众来说可能有点傻，但多亏了一本精心编写的手册，其中包含了一系列引人注目的项目，GENIAC 设法从几个灯泡和一些电线中获得了很多好处。事实上，[迈克]说 GENIAC 经常被认为是交互式电子叙事的第一个例子，因为手册中精心制作的故事让玩家可以在普通孩子听说过“视频游戏”之前很久就开始虚拟冒险。一个关于这些故事之一的视频，“铀船和太空海盗”在休息后可以看到。即使在今天，它也是一个足够有趣的玩具，但在 1955 年，它会令人兴奋不已。

建造这个复制品将需要使用激光切割机，这样你就可以接近原来的钻孔 Masonite 背衬和转子。在那里，[Mike]生产了一系列 3D 可打印组件，这些组件被连接到电路板上，用作触点、垫片和各种其他小摆设。有些部分他找不到图片，所以他被迫拿出自己的设计。但是考虑到原版玩具的挑剔，他认为他打印出来的零件实际上可能比玩具自带的要好。

如果你喜欢他与 GENIAC 的合作，一定要看看【迈克】去年制作的*神奇的尼姆博士*的 3D 打印复制品，或者[他令人惊叹的迷你真空吸尘器 601](https://hackaday.com/2019/06/17/a-faithful-replica-of-an-early-computer-trainer/) 。
