# 伟大的徽章概念:用于 WiFi 认证框架的“盖革计数器”

> 原文：<https://hackaday.com/2020/06/20/great-badge-concept-a-geiger-counter-for-wifi-deauthentication-frames/>

[尼克·普莱斯]对 DEFCON 徽章有一个很棒的概念:[一个工作起来很像定向盖革计数器的设备，但在检测 WiFi 认证数据包时发出啁啾声](https://spun.io/2019/07/26/a-geiger-counter-for-wifi-deauthentication-frames-the-telephreak-12-badge/)而不是辐射。这是一个疯狂的想法，去年我们不知何故忽略了它。为什么要检测这种东西？嗯， [WiFi deauth 攻击](https://hackaday.com/2017/08/13/wifi-deauthentication-vs-wifi-jamming-what-is-the-difference/)是一种无形的毒性，通过迫使用户不断与认证捆绑在一起，有效地干扰无线通信，而这个设备会检测到它。

然而，有些事情比预期的要难。为了使设备具有方向性，[尼克]设计并制造了一个 PCB 八木天线，但这并不实用。它不仅太大，还需要在已经很贵的 PCB 上走四层。他选定的解决方案——灵感来自一个朋友关于把徽章扔进品客薯片罐的笑话——是用任何五金店管道部的铜管端盖包围 PCB 全向天线。[Nick]认为将它焊接到接地层应该会产生一个简单、廉价且有吸引力的定向天线模块。成功了吗？我们都得等着瞧。

遗憾的是，[尼克]没能及时完成去年的 DEFCON。安装了硬件版本，他的专用 PCB 的制造时间比平时长。更坏的消息是，今年的[被取消了](https://hackaday.com/2020/05/10/hackaday-links-may-10-2020/)，或者更确切地说，是虚拟的，这意味着他将不得不亲自去作证。好消息是，现在他又得到了 12 个月的延期。观看下面嵌入的功能原型的简短视频。

盖革计数器可能是老式的硬件，但它的耐用性和吸引力是不可否认的。它还被带入了现代业余爱好者时代，这款触摸屏设备拥有漂亮的外壳和光滑的用户界面。
