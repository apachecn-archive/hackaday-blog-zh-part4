# 这覆盆子酱太难吃了

> 原文：<https://hackaday.com/2019/03/12/this-raspberry-pi-is-nasty/>

piratebox 是一台小型计算机、WiFi 适配器和一个硬盘。piratebox 背后的想法是简单地将一些存储放在网络上，供所有人使用。如果你在一个团队中，需要一种简单的方式在 hackerspace 上共享文件，或者只是想在局域网聚会上加入现代元素，那么它就非常棒。[Nick]和[Josiah]想出了他们自己的盗版盒，这个用的是树莓派 Zero W，是最便宜的盗版盒之一。

Raspberry Pi Zero W 及其网络适配器拥有变成一个功能强大的盗版盒所需的所有硬件，因此这个版本的硬件非常简单。它只是一个 USB A 插头，形式为 [USB Stem](https://www.sparkfun.com/products/14526) 而非其他。该软件可在 GitHub 的[网站上获得，并广播一个名为 SUBZero 的 WiFi 网络。在这个网络上浏览到 192.168.1.1 允许上传和下载文件，所有这些都不需要互联网连接。这是一种可以装进你口袋的云，本周我们称之为“雾”或“薄雾”。由于这被称为“零度以下”，也许“pogonip”是首选的命名法。](https://github.com/Halcy0nic/SUBZero)

当然，没有 3D 打印外壳的 Raspberry Pi 项目是不完整的，SUBZero 也不例外。这款 Pi Zero 有一个 3D 打印的外壳，配有一个滑动门，可以进入所有端口。你可以在下面看到一段视频。
