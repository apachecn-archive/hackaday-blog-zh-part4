# F 型键盘修复更进一步

> 原文：<https://hackaday.com/2021/10/08/model-f-keyboard-restoration-goes-the-extra-mile/>

IBM 型键盘应该不需要介绍。F 型以其弯曲的弹簧键机制而闻名，因其令人满意的打字体验和防末日的构建质量而备受青睐。这些键盘在 1981 年首次推出，现在自然需要基本的维护。然而，[Epictronics]最近更进了一步，[将一辆 F 型车恢复到了全新的状态](https://youtu.be/Tmllb-qNA68)。

丢失键帽是他最不担心的，因为新旧替换物都相对容易得到。[Epictronics]更关心的是夹在键盘内部的 40 年的泡沫，很可能早就降解了。腐烂的泡沫除了让人恶心之外，还有可能弄脏弯曲的弹簧开关。在拆开身体并移除“恶心”的泡沫垫后，一个替代品由氯丁橡胶和一个轻便的打孔器锻造而成。

拆卸键盘外壳需要用木槌轻轻敲打，重新组装也需要类似的不合适的工具。如 IBM 的这个经典片段所示，键盘组装过去是(现在仍然是)由机器人自动完成的，由 IBM Series/1 小型计算机驱动。这些机器人的精度和力量同样令人印象深刻。在无法使用 IBM 名副其实的“关闭工具”和各种其他机器人助手的情况下，[Epictronics]不得不在重新组装期间满足于游泳池面条和一个滑稽的大夹子，混合着纯粹的决心。

视频中的其他巧妙技巧包括加热以重新形成盘绕的键盘电缆，以及使用汽车上光剂清洁外壳塑料。后者有可能使事情变得更糟，因此需要一只精致的手来维护有纹理的塑料。

我们最近报道了另一个 F 型修复，令人兴奋的是看到这么多专注的黑客将这些键盘一直保持到 21 世纪。
