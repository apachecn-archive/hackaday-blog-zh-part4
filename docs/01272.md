# 修复阿波罗导航计算机

> 原文：<https://hackaday.com/2018/11/17/restoring-an-apollo-guidance-computer/>

阿波罗导航计算机是计算机历史上非常重要的一部分。它是指导阿波罗着陆器在月球着陆的计算机。这些年来，我们已经看到了一些复制品，但[CuriousMarc]更近距离地观察了一个真实的东西。在这个视频中，[Marc]看到了他的同事们拆开了一个原始的 AGC，近距离观察了这段计算机历史的内部。

 [https://www.youtube.com/embed/2KSahAoOLdU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/2KSahAoOLdU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



多亏了收藏家吉米·洛克，他们得以拆开 AGO#14，这是月球漫游舱(LEM)中用于飞行测试的原始飞行计算机之一。[Loocke]曾经在美国宇航局的承包商工作，在一家电子剩余品商店找到了 AGC。他和[迈克·斯图尔特]正在为 2019 年阿波罗 11 号登陆 50 周年修复电脑。关于他们的 AGC 有趣的事情是，它不是罐装的:芯片没有涂上环氧树脂或泡沫，在飞行中保护它们。这意味着他们可以拆开系统，检查芯片和连接它们的复杂线路。希望他们能让它重新运转起来。我们会看到:这只是一个系列的第一个视频，这是一个正在进行的项目。我们在 2016 年看到了同一台计算机，当时【卡尔·克兰奇】开始对它进行研究，做了一些事情，如为 [DSKY 输入和显示面板](http://rescue1130.blogspot.com/2018/11/more-planning-for-apollo-guidance.html)建立一个替代品，因为原始版本仍然在这台计算机来自的登月舱中，它在休斯顿的约翰斯顿航天中心展出。
