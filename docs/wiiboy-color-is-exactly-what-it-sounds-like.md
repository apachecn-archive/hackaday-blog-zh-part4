# WiiBoy 颜色正是它听起来的样子

> 原文：<https://hackaday.com/2020/09/02/wiiboy-color-is-exactly-what-it-sounds-like/>

任何人都可以出去买一个掌上游戏机，如果你想成为校车上的酷小子，你可以随时让你的父母带你出去买一个。但是如果你想在成年后拥有真正的街头信誉，你必须自己动手。在 Wiiboy 的颜色上,[GingerOfMods]已经做到了这一点。

是的，这是另一个由家用控制台转变成的便携式设备，而且它的制作非常精致。Wii 主板被切割成绝对最小的尺寸，因为目的是将整个系统构建成原始 Game Boy 颜色的粗糙形状。然后，定制的印刷电路板被用来将切割的电路板连接到外围设备，如用于加载游戏的 USB 驱动器和 Gamecube 控制器的电路。屏幕是一个漂亮的 3.5 英寸 IPS LCD，运行在 480p，最初打算用作汽车备用相机。电池续航时间约为 2-3 小时，带有 USB-C 端口，便于充电。更多详情请见论坛建设日志。

这是一个整洁的建筑，3D 打印的外壳，开关操纵杆和 DS Lite 按钮使它接近生产质量。[GingerOfMods]打算建造更多的佣金，尽管考虑到所涉及的劳动力和定制工作，预计价格会很高。我们以前也见过其他便携式 Wii，就像这种紧凑的 Kapton-heavy 结构。休息后的视频。
