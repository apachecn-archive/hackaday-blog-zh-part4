# 圣诞老人的胡须梳理机器人

> 原文:[https://hackaday.com/2021/12/24/santas-beard-combing-robot/](https://hackaday.com/2021/12/24/santas-beard-combing-robot/)

一年到头都在工作，放牧小精灵，为所有善良的男孩和女孩制作玩具；这需要奉献。一个人在胡须护理方面落后是很自然的，对吗？今年，[诺伯特·扎雷亚]用他的胡须梳理机器人拯救了圣诞节。

好吧，这与其说是一个严肃的机器人项目，不如说是一个邪恶滑稽机器学校的低劣机器人。但是完整性归功于[诺伯特]——摆动两个伺服系统完成工作(几乎)的代码甚至被[发布在 GitHub](https://github.com/Norbert01x/Beard-Combing-Robot)上。
