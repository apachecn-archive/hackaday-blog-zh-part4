# 赛博朋克夹克是 2019 年街头的首选服装

> 原文:[https://hackaday . com/2019/01/12/cyber punk-jacket 是 2019 年街头的首选服装/](https://hackaday.com/2019/01/12/cyberpunk-jacket-is-the-garment-of-choice-for-the-streets-of-2019/)

科幻小说和相关流派的粉丝总是对现实生活感到失望。未来在纸面上有如此多的承诺，然而当到达 2015 年时，数百万人失望地发现悬浮滑板技术仍然与我们无缘。不过，这也不全是坏事——[[Abe tusk]开发了一款赛博朋克夹克，这样你就可以在现实生活中实现你肮脏的黑客幻想了。](https://www.instructables.com/id/Cyberpunk-Jacket/)

这种效果是通过特别设计的夹克补丁实现的。尼龙织物被激光切割成艺术品或文字，然后放在电致发光面板上。织物作为一个面具，并被粘在 EL 面板上，然后用 velcro 将组件固定在夹克的背面。

这是一个不仅仅关注酷炫视觉效果的版本。对细节的关注在坚固性和可用性方面获得了回报——电线整齐地穿过外套的内衬，并使用特殊的应变消除设备来避免电线从 EL 面板上断裂。额外的努力意味着这是一件可以承受真实世界使用的夹克，而不是在摆姿势拍照的过程中散架。

从艺术品创作到最终组装，一切都有据可查，所以你没有理由不能在家里复制这一点——最终的结果是惊人的。我们的观点是，电致发光技术是复古和赛博朋克风格的必由之路，但 led 也可以很有趣——[就像这个高能燃烧人建筑一样。](https://hackaday.com/2014/09/26/a-very-bright-led-jacket/#more-133353)
