# 一队压力清洗机为这个互动公共喷泉提供动力

> 原文：<https://hackaday.com/2018/11/11/a-fleet-of-pressure-washers-powers-this-interactive-public-fountain/>

公共艺术装置可以很酷。增加与观众的互动会增加一点酷感。将市民的自豪感、水的舞动、音乐和灯光融入到这个项目中，你就能得到这个非常酷的压力清洗机驱动的音乐喷泉。

[Niklas Roy]设计的展览名为 *Wasserorgel* ，或“水器官”，这是一个很贴切的名字。作为德国工业庆典的一部分，该展览建于小镇温嫩登，是清洁设备公司 kr cher 的所在地，该公司以其独特的黄色外壳压力清洗机系列而闻名。该公司的八个电动压力清洗机出现在 *Wasserorgel* 中，它射出水流，并播放音符，以回应路人对坚固防水的 3D 打印键盘的挠痒痒。这场表演由一个带 MIDI shield 的 Arduino 管理，它通过固态继电器控制压力清洗机，甚至接受风速计的输入，以便在风太大时关闭表演，以免附近的[Frau Dimitrakudi]被弄湿。

下面的视频展示了 *Wasserorgel* 在城镇市场广场长达数周的活动中是如何引人入胜；还有一个德语版本的，有详细的构造。虽然我们不记得以前在公共艺术中见过压力清洗机，但我们记得其中一个被用作[DIY 喷水切割机](http://hackaday.com/2017/05/31/a-water-jet-cutter-from-a-cheap-pressure-washer/)的基础。
