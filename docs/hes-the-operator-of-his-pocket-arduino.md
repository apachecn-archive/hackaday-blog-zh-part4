# 他是袖珍 Arduino 的操作员

> 原文：<https://hackaday.com/2020/11/27/hes-the-operator-of-his-pocket-arduino/>

发电厂乐团乐队在 70 年代以其独特的电子声音轰动了乐坛，为接下来十年的电子音乐革命打开了大门。如果你不熟悉这个乐队，他们也经常有科技主题的歌曲，由于现代微控制器技术，有可能用微控制器复制发电厂乐团的声音，正如[Steven] aka [Marquis de Geek]在他的旋律构建中展示的那样。

当音乐在一个[唱针和一个 Korg 合成器上播放时，它通过五个独立的 Arduinos 输入，其中四个安装了各种合成器和循环采样器(大概代表了发电厂乐团的四个成员)。像这样的采样器允许音乐片段在录制后不断重复，这意味着[Steven]可以自己演奏整首歌曲。第五个 Arduino 的功能是控制器，处理 I2C 的 MIDI 和模式序列，一切最终通过自制的混音器传输。](https://hackaday.com/2012/07/22/reverse-engineering-a-stylophone/)

[马奎斯]也穿着适合发电厂乐团的服装参加下面的视频演示，这真的是向这个著名的开创性乐队致敬。虽然它本身是一个伟大的建筑，是对发电厂乐团声音的伟大再现，但我们可以想到另一种方法来真正将这个项目推向顶峰——一种发电厂乐团风格的 LED 领带。
