# 混凝土焊料 Squid 是固溶体

> 原文：<https://hackaday.com/2020/02/12/concrete-solder-squid-is-a-solid-solution/>

虽然可以购买现成的焊接设备，但适合你的设备可能会随着时间的推移而发展。老实说，它可能永远不会停止进化。当然，你可以从 el-cheapo 帮助之手或一个不错的爱好老虎钳开始，但它可能不会就此结束。为什么？因为这些工具没有一个适合所有的应用，除非你打算一遍又一遍地焊接同样的东西。有时候，用鳄鱼夹住一块板比慢慢地操纵老虎钳的钳口更容易，但这些辅助之手的活动范围非常有限。

你有没有想过用冷却剂软管做一个焊接乌贼，因为那东西看起来很酷很柔韧？好吧，那就让 Hackaday 明矾[JeremySCook] [向你展示如何做到这一点](https://www.youtube.com/watch?v=MIIlK7pLdQM)。几年前[他建造了一个类似的木质底座的乌贼](https://hackaday.com/2018/03/04/coolant-hoses-retasked-to-lend-a-helping-hand/)，但是它不够重，所以他重新设计了一个混凝土底座。他借此机会做了一些不错的调整，比如用拉链将一个小型 PC 风扇和 9 V 连接起来，制成一个可无限重新定位的通风系统，并在背面添加了一个大夹子，以增加焊接时的稳定性。当然，将焊接线轴穿到一个软管上是天才之举。

如果你关注[Jeremy]的话，你会知道他已经研究混凝土有一段时间了，看到他通过使用它来追求更好的工具来巩固他对混凝土的热爱是很棒的。他已经把打印好的模具文件上传到了 GitHub 上，休息之后的制作视频应该已经准备好了。

对你来说不够定制？启动打印机，制作自己的球窝臂。
