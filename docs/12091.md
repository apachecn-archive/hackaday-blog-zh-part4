# 回收零件制成肥皂形状的电动汽车

> 原文：<https://hackaday.com/2021/11/30/recycled-parts-round-out-soap-shaped-electric-car/>

[Handy Geng]有一个诀窍，就是在他的作品中加入大比例的回收材料。正如休息时间下方的[视频所展示的那样，他也喜欢将实用性与异想天开相结合。](https://youtu.be/iBz6voaxh2Q)

利用从摩托车、垃圾堆和其他可能的来源中回收的零件，[Handy]组装了一辆小汽车，他自己形容它在地板上滑动时看起来像一块肥皂。当你看到独立的前后转向系统工作时，你会同意的，它允许汽车的前部和后部自行驱动和转向。横着走，斜着开，我们猜测原地打转是可能的。

在下面的视频中，同样清楚的是，[Handy]是一个有才华的制作者。同时不把自己看得太重(留意 360°自拍摄像头！)他显然对这项工作感到自豪。[Handy]的工作室和技能组合表明，在本质上，他对自己的手艺相当认真。我们很欣赏在这样一个鼓舞人心的建设中创造性地使用废料。转向灯和“通信器”手绝对是不可思议的。

如果你喜欢用回收材料建造，那么你也会喜欢垃圾打印机。感谢[Fosselius]的提示！
