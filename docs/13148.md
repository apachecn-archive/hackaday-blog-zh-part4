# 定制 Mac 笔记本电脑的台式机性能

> 原文：<https://hackaday.com/2022/03/24/desktop-performance-in-a-custom-mac-laptop/>

我们大多数人都拥有或曾经使用过笔记本电脑。对于旅行，作为一名学生，甚至对于坐在电视机前浏览 Hackaday，它们都是不可或缺的。与台式机相比，它们的性能往往会大幅下降，但这要归功于便携式外形的散热和电池限制。[Scott Yu-Jan]想要在他自己的生活中解决这个问题[,打造一款没有这些缺点的定制 Mac 笔记本电脑](https://www.youtube.com/watch?v=XFZ5l9SaIVw)。

注意到现代的 iPad Mini 和他的 Mac Mini 的宽度完全一样，[Scott]开始将这两个设备组合成一个单元，他可以在旅行时组装。一个传统笔记本翻盖设计的 3D 打印外壳负责将这两个设备物理结合在一起，两者之间的 USB-C 电缆负责将它们结合在软件中，这要归功于苹果的 Duet 程序。虽然这比 Macbook Pro 的性能更好，但它实际上可能有一些额外的好处，因为苹果继续拒绝制造带触摸屏的笔记本电脑。

当然，也有一些缺点。价格高于 iPad 和 Mac 的可比 Macbook Pro，而且不包括键盘和鼠标。它也没有电池，所以需要插电。然而，在下面链接的后续视频中，[Scott]指出，对他来说这仍然是有意义的，因为他已经单独使用 Mac 和 iPad，并且只在有现成电源插座的地方远程工作。不过，对于普通人来说，如果你真的需要一台类似笔记本电脑的神秘机器，我们可能会推荐一些不同的东西。
