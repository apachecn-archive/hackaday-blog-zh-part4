# 地球上任何地方的遥控赛车比赛

> 原文:[https://hack aday . com/2019/08/31/race-RC-cars-from-anywhere-on-earth/](https://hackaday.com/2019/08/31/race-rc-cars-from-anywhere-on-earth/)

多年来，赛车游戏已经取得了长足的进步。从基本的 2D 雪碧为基础的标题，他们已经发展到结合先进的引擎与高度现实的模拟物理，甚至可以用来帮助开发现实世界的汽车。对[Surrogate.tv]来说，这还不够好，所以他们决定创造一些更植根于现实的东西。

![](../Images/d4b6b88a5c8caf76d8737d94d4cbdd06.png)

The game is played in a web browser. Players are assigned a car and view the action from a top-down camera.

他们的项目产生了一个基于通过互联网控制真正的遥控汽车的赛车游戏，在现场比赛中与其他人类对手比赛。从一系列 Siku 1:43 比例的 RC 汽车开始，团队必须克服一系列工程挑战才能实现这一目标。首先，由于团队在同时运行多辆汽车时遇到了问题，最初的电子设备不得不被拆除。

相反，这些汽车安装了运行定制固件的 ESP8266s。一个高架 GoPro 与特殊的低延迟流媒体软件一起使用，允许玩家引导他们的汽车走向胜利。计算机视觉系统用于计时，甚至还有自动充电站来帮助汽车在几个小时的比赛中保持充足的电量。

这款游戏可以在网上免费玩，目前比赛按照固定的时间表进行。我们期待在一三场比赛中一试身手，并有兴趣了解世界各地的延迟情况。

我们以前见过其他远程 RC 构建；[通常以树莓派](https://hackaday.com/2016/02/07/robo-car-via-3g/)的力量为特色。我们还介绍了[用于实时应用的低延迟视频的有用技术](https://hackaday.com/2017/09/12/video-streaming-like-your-raspberry-pi-depended-on-it/)。休息后的视频。
