# 棒球帽鼠标提供了一个点击式界面

> 原文:[https://hack aday . com/2020/10/05/baseball-cap-mouse-provides-a-look-and-click-interface/](https://hackaday.com/2020/10/05/baseball-cap-mouse-provides-a-look-and-click-interface/)

曾几何时，电脑鼠标是不存在的。早期的计算机使用各种其他输入设备，从典型的键盘到更深奥的选项，如操纵杆或光笔。虽然我们知道鼠标主宰了一切，但这并不意味着其他工具找不到它们的位置。来自[亚采克·费多林斯基]的帽子鼠标就是这样一种设备。

鼠标由 Adafruit Feather nRF52840 Sense 组成，安装在基本的棒球帽上。开发板采用 9 自由度运动传感器封装，内置 ST LSM6DS33 加速度计/陀螺仪和 LIS3MDL 磁力计。通过一个强大的传感器融合算法，这使得该板能够以很高的精度测量佩戴者头部的方向和运动。这允许用户查看屏幕的不同部分来移动鼠标光标，系统以绝对而非相对的方式工作。命令通过羽毛的内置蓝牙发送到连接的 PC，避免了用户脖子上的危险电缆。Github 上的文件可供那些渴望开发自己的程序的人使用。

结合 Windows 中的一些内置辅助功能，该设置允许用户很好地移动鼠标，使用脚踏开关来激活鼠标的左右按钮。对于那些发现使用传统鼠标很困难的人来说，这可能是一个提高工作效率的好工具。当然，如果你想了解更多，回顾一下鼠标技术的早期是有好处的。休息后的视频。
