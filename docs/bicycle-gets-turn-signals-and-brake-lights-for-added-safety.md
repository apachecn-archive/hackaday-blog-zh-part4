# 自行车获得转向灯和刹车灯，以增加安全性

> 原文：<https://hackaday.com/2022/11/28/bicycle-gets-turn-signals-and-brake-lights-for-added-safety/>

骑自行车旅行可以是一种有趣和令人兴奋的交通方式，与开车相比，还可以节省一大笔钱。世界上有很多地方，自行车是很大一部分人的主要交通方式，但还有更多地方完全是为汽车设计的，很少考虑到其他人。对于任何骑自行车的人来说，特别是对于生活在这些汽车占主导地位的地区的人来说，[像这种 LED 阵列这样的额外安全措施通常是必要的](https://www.hackster.io/estudioroble1/brake-light-and-turn-signals-for-backpacks-09486c)。

这个灯阵是由[Estudio Roble]设计的，用于在他的城市里旅行。该设计基于 ada fruit Circuit Playground Express，它直接位于灯具的中间。围绕它的是一个额外的环内的菱形发光二极管带。正常驾驶时，该灯使用亮蓝色，但当开发板中的加速度计检测到制动时，该灯会变成红色。还有集成的转向灯，其操作类似于摩托车转向灯。信号在车把开关和灯之间无线传送。

该设备本身可以夹在任何背包上，由于控制器是无线的，所以每次骑车人骑上自行车时都无需连接电线。与大多数自行车完全没有照明相比，这是一个很大的进步。如果你已经读到这里，你需要看看[这个自行车前灯](https://hackaday.com/2013/03/06/dynamic-bicycle-headlight-uses-the-open-road-as-a-display/)，它使用投影仪直接显示行驶路径上的信息。
