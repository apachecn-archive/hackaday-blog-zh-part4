# 谐波传动使用柔顺机构来瘦身

> 原文:[https://hack aday . com/2021/04/07/harmonic-drive-uses-compliant-mechanism-to-slim-down/](https://hackaday.com/2021/04/07/harmonic-drive-uses-compliant-mechanism-to-slim-down/)

[李维·让桑]有一个秘密:他不喜欢谐波传动。但他没有完全放弃扭矩放大传动装置，而是决定看看如何使用 3D 打印的柔性机构来改进它们。

对于外行人来说，谐波传动，也称为应变波齿轮，是一种紧凑的高扭矩齿轮箱，已经受到“机器狗”制造商和其他机器人专家的欢迎。这个想法是让一个坚硬的，有内齿的外环套在一个有外齿的，有弹性的杯上。一个波发生器在内杯内旋转，拉伸它，使它与外环啮合。这两个齿轮只有几个齿不同，这意味着可以实现非常高的齿轮比，这使它们非常适合机器人腿的关节。

[Levi]谐波传动的问题是，由于柔性花键杯的深度，紧凑性不是它的优点。他的想法是通过一个板簧将柔性花键连接到驱动装置的输出端，当波浪发生器旋转时，该板簧允许弯曲，但能有效地传递扭矩。整个原型是 3D 打印的，除了波发生器轴承和步进电机，并投入测试。

正如下面的视频显示，在谐波传动的精彩介绍后，这个概念是可行的，但它也有其局限性。即使负载很轻，当 PLA 弹簧用完时，驱动器也会发出一些令人不愉快的嘎吱嘎吱的声音。我们可以很容易地看到，被替换为，比如说，一个钢弹簧，无论是机械加工或切割的喷水机器。这可能会解决最明显的问题，并使[Levi]的紧凑型谐波驱动的梦想成为现实。当然，我们以前见过[非常紧凑的应变波齿轮](https://hackaday.com/2017/01/17/3d-printed-strain-wave-gear-needs-your-help/)。
