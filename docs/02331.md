# 在廉价的数控机床上制作镜片

> 原文：<https://hackaday.com/2019/03/08/creating-lenses-on-cheap-cnc-machines/>

地下室和车库里有很多数控机床，[但我们还没见过像这样的](https://www.youtube.com/watch?v=ZW2lj0KjyT8)。它使用标准的 CNC 机器和大量的人力来制造透镜。

用 CNC 机器制作透镜的过程始于将一块废板铺在表面，然后用双面胶粘住一张 8 毫米厚的丙烯酸树脂薄片。然后用 8 毫米立铣刀切割出透镜，从原材料上取下，并用湿砂纸打磨以去除工具痕迹。湿式砂光从 400 粒度开始，逐渐增加到 2000 粒度，之后用一种用于高光汽车面漆的抛光剂对镜片进行抛光。这是手工完成的，但在这种情况下，使用真正的抛光轮并不可耻。

展示了其他几个透镜，包括一个柱面凸透镜，但这些只是平凸透镜，或者一侧是平的透镜。双凸透镜可以通过背靠背粘合两个平凸透镜来构建，这是用丙烯酸胶完成的，在这种情况下是 Acrifix 粘合剂。结果是显著的:通过大量的打磨和抛光，你可以在廉价的数控机床上制作一个丙烯酸镜片。这个窍门只是你数控加工道路上的一小步。

在这个系列中还有一些视频，包括一个关于使用 Fusion 360 定义镜头形状以获得正确焦距的视频。我们等不及要看了。
