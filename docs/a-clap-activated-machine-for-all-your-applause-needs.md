# 满足你所有鼓掌需求的拍手机器

> 原文：<https://hackaday.com/2020/11/29/a-clap-activated-machine-for-all-your-applause-needs/>

掌声被视为人群中热情和崇拜的象征。在现场观众面前放映电影的电视节目经常提示他们的观众鼓掌，以便产生期望的听觉气氛。当然，你不需要依靠软绵绵的人类来完成所有的工作。[Dillon]建造了一台用途可疑的机器——当鼓掌声激活时，它会产生机械掌声。(视频，嵌在下面。)

不出所料，这个项目是为一场无用的机器竞赛而设计的，但这并没有降低它作为学习练习的价值。一个 Arduino 运行这个节目，使用一个麦克风模块来听响亮的噪音，如掌声。如果在指定的时间范围内检测到两次鼓掌，机器开始闪烁“鼓掌”灯并拍手。Arduino 在继电器的帮助下实现了这一点，继电器打开电机，旋转皮带驱动的凸轮，将双手分开。然后，双手通过一根有弹性的橡皮筋拉在一起拍手。

由于令人难以置信的嘈杂的传动系统和有点业余的鼓掌能力，从机器发出的声音并不完全被认为是“掌声”。然而，这是一个开始，它仍然是这十年来我们见过的最好的鼓掌机器。如果你自己的网站正在建设中，可以考虑给我们写信。如果这一切让你开始怀念老式的克拉珀电路，[你也可以建造一个。](https://hackaday.com/2019/10/06/diy-clapper-lets-you-pick-your-components/)
