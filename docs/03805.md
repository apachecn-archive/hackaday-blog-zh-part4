# 把 3500 伏的电压举到你的眼前

> 原文：<https://hackaday.com/2019/08/06/hold-3500-volts-up-to-your-eye/>

旧的军事装备有时可以在跳蚤市场和易贝以几分钱的价格买到，这通常是因为人们并不总是知道他们拥有什么。虽然[tsbrownie]在订购这个神秘设备时确切地知道他得到了什么，但我们不确定如果我们自己偶然发现它，我们能说同样的话。看起来像某种真空管的东西原来是一个来自旧潜艇潜望镜的红外传感器，该潜望镜被重新用作夜视设备。(视频，嵌在下面。)

当然，让这样的电子管工作需要高电压。这个特别需要 3500 伏才能正常工作，但这是由一个装在 PVC 类外壳中的小电路解决的。外壳将管子放在中间，一端是目镜，另一端是相机镜头，大概是通过 3D 打印的底座连接的。电子设备装在“把手”里，整个东西看起来像一个带手柄的小观察镜。一旦通电，该设备能够显示经典的绿色夜视场景。

像这样的老式模拟设备非常罕见，拥有专业知识的人也很难找到这些设备，并让它们在某种程度上再次工作。对于任何对电子管、旧军用装备感兴趣的人来说，这是一个很好的视频，或者即使你已经[不久前建造了一个更现代的夜视系统](https://hackaday.com/2019/04/10/see-whats-inside-night-vision-and-how-to-build-your-own/)。
