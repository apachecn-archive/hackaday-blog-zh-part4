# BGA 焊接和检查

> 原文:[https://hack aday . com/2021/02/28/BGA-soldering-and-inspection/](https://hackaday.com/2021/02/28/bga-soldering-and-inspection/)

如今，如果你想制造酷的东西，你可能必须掌握表面贴装电子技术。然而，对于许多人来说，球栅阵列(BGA)仍然令人生畏。看一下[volt log]关于他焊接 BGA 的技术的视频，并检查一下你是否做对了。

他有很多关于表面光洁度和焊剂选择的建议。他做的时候看起来很容易。当然，拥有一个带有良好注册标记的 PCB 也会有所帮助。

当然，你不能把烙铁放在零件下面。热板从下面提供热量。热风枪轻轻一推，就会将焊球推到熔化边缘。甚至从电炉上取下零件也需要特殊的技术。

没有看到结果，你怎么知道它是否成功了？专业人士可以使用 x 光机，但你的店里可能没有。[VoltLog]使用 DVM 并测试芯片引脚上几乎肯定有的内部保护二极管。然而，要做到这一点，你需要把芯片放在一个裸板上。如果你在修理一个现有的电路板，这种技术就没用了，因为电路板上的其他元件会干扰测量结果。

我们已经看到非常有耐心的[用手将焊线焊接到 BGA](https://hackaday.com/2019/07/08/bga-hand-soldering-uses-tombstone-resistor-technique-demands-surgical-precision/)上。如果你想亲自尝试，你也可以找到[更详细的视频](https://hackaday.com/2016/01/03/bga-hand-soldering-video/)并比较其他技术。
