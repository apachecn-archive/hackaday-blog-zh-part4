# 坦克轨道滑板

> 原文:[https://hackaday.com/2020/12/25/tank-track-skateboard/](https://hackaday.com/2020/12/25/tank-track-skateboard/)

随着电动滑板套件和组件变得越来越普遍，你真的需要做一些不同的事情来让你的定制板脱颖而出。[Emiel][实用工程师]已经通过建造半履带滑板成功做到了这一点。(视频，嵌在下面。)

除了前面的卡车，紧固件和轴承，板上所有的机械部件都是定制的。坚固的后底盘和履带部分由铝板加工而成，车轮和履带连杆由 POM/Delrin 加工而成。赛道上的大型[碳纤维甲板](https://www.youtube.com/watch?v=urghynIAsjw)和[聚氨酯垫](https://www.youtube.com/watch?v=IvIicjBpfF8)是定制成型的，这在单独的视频中有详细介绍，也在下面。两个结实的无刷电机驱动轨道，由封闭在金属片电子箱中的 LiPos 提供动力。最终产品看起来非常精良，特别是考虑到大部分工作是在一个 2 米 x 3 米的小车间里完成的。

看起来这款滑板很好地处理了砾石路，但我们真的很想看看它在像沙子这样的柔软表面上的表现，即使是越野滑板也可能会在那里挣扎。它与低转速扭矩有点斗争，所以轻微的齿轮变化是在这个板的未来。

坦克履带让一切都变得更酷。如果你愿意生活在塑料轨道上，3D 打印是一个可行的选择，正如[【rctestflight】的履带式漫游车](https://hackaday.com/2020/04/02/autonomous3d-rover-with-tank-tracks-rules-the-fields-almost/)和[【伊万·米兰达】的坦克滑板](https://hackaday.com/2020/07/31/electric-skateboard-with-tank-tracks-from-a-big-3d-printer/)所展示的那样。
