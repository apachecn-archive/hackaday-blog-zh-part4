# 厌倦了猫捉老鼠

> 原文：<https://hackaday.com/2020/11/07/tired-of-the-cat-and-mouse/>

脸书刚刚宣布了他们关于 Oculus Quest 2 虚拟现实耳机的计划。你可能不会感到惊讶，但他们想要更多的用户数据，并对你如何使用硬件有更多的控制权。要使用该设备，你需要一个经过验证的脸书帐户。更糟糕的是，他们要求开发者帐户能够将非脸书软件“下载”到设备上，从而限制了对社区开发的应用程序的访问。猜猜谁决定谁成为开发者。提示:不是开发软件的人。

[![](img/88c086d2397afd67f794fe7596c2017e.png)](https://hackaday.com/wp-content/uploads/2017/06/unity_thumbnail.png) 我们的文章认为，这将是一场破解社区耳机的竞赛的开始，也是一场超越脸书黑客的竞赛的开始。就像 iOS [的每一个新版本都会在一两周内获得一个越狱](https://techcrunch.com/2020/05/23/hackers-iphone-new-jailbreak/)，然后苹果会尽快修补它，我们会看到黑客与脸书持续的猫捉老鼠的游戏吗？

我不在乎。而这并不是因为我不关心开放硬件或者独立 VR 开发者。完全相反！但是就像你曾经和一个对你毫无益处的女孩的浪漫一样，和一个不允许你在别人的硬件上运行游戏的公司的有害关系是一种你最好离开的关系。当然，你可以尝试修复它，或者破解它。你可以告诉自己，如果你再给他们一次机会，也许脸书会回心转意。一开始会很疼。

但最终，用户和想要使用它们的公司之间会有一场永恒的斗争，这很可悲。我过去常常期待着猫捉老鼠的奇怪游戏，但现在猫的资金太多了，使它成为一场公平的战斗。如果你今天买了一个任务 2，打算去黑它，我建议你和其他人一起度过你的时间。你正在报名参加一连串令人心碎的比赛。防患于未然。你值得更好的。海里的鱼太多了，对吧？
