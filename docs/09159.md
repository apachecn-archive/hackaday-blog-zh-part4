# 原型电视技术是这个视点时钟的幕后推手

> 原文：<https://hackaday.com/2021/02/06/proto-tv-tech-lies-behind-this-pov-clock/>

如果没有视觉的持续，这种由生物化学调节的视觉的怪癖，生活会很无聊。没有电影，没有电视——只有现实、自然之美和现场表演来娱乐我们。听起来很可怕。

当然，我们开玩笑，但 POV 背后有许多很酷的黑客，其中之一是[Joe]整洁的[尼普科夫磁盘时钟](https://hackaday.io/project/176786-arduino-nipkow-disk-clock)。如果你认为你从未听说过这样的事情，你可能错了；以 19 世纪发明家保罗·戈特利布·尼普科夫的名字命名的尼普科夫圆盘是最早尝试机械扫描电视背后的中心思想。尼普可夫圆盘有一系列间隔均匀、呈螺旋状排列的孔洞，当旋转时，这些孔洞似乎会扫过一个固定的区域。当把它放在透镜和光敏元件之间时，就可以制成一台简单的电视摄像机。

然而，对于他的尼普可夫钟，[乔]改变了这个想法，在旋转的圆盘后面放置了一个光源。相对于盘的位置控制阵列中的发光二极管何时发光以及发光的颜色决定了发光的像素。[Joe]的时钟使用两个 LED 阵列将显示区域的大小扩大一倍，并使用一个带有矩形孔径的圆盘。由此产生的像素有点梯形，但它并没有真正分散显示器的外观。下面的视频展示了构建过程和实际完成的时钟。

在这样的显示中获得正确外观的关键是代码，而且[Joe]为他的软件付出了相当大的努力。要是早期的机械电视修理工有这样的帮助就好了。[【Jenny List】对早期的电视先锋和他们的尼普科夫盘式摄像机做了一个很好的报道](https://hackaday.com/2017/06/28/mechanical-image-acquisition-with-a-nipkow-disc/)；我们以前也见过[其他的尼普科夫显示器](https://hackaday.com/2012/03/15/analog-clock-display-uses-a-nipkow-disk/)，但是【乔】的时钟将这个概念带到了另一个层次。
