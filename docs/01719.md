# 网飞放弃 ZX 频谱自制标题 Nohzdyve

> 原文：<https://hackaday.com/2019/01/01/netflix-drops-zx-spectrum-homebrew-title-nohzdyve/>

黑暗、反乌托邦式的未来在网飞的电视剧《黑镜》中一直存在，但该系列的最新版本《Bandersnatch》呈现了一个截然不同的故事。Bandersnatch 是一个分支故事，讲述了一个名叫 Stephan 的车库程序员的虚构事件，他为 Tuckersoft 公司开发了一款名为 Bandersnatch 的游戏，背景设在 20 世纪 80 年代的英国。整件事就像一个你自己选择的冒险游戏，直接从世嘉的 CD 上下载(尽管有真实的全动态视频)，让观众选择故事中接下来会发生什么。网飞不会错过交叉推广的机会，他还发布了一款可玩的 ZX 频谱自制游戏，Nohzdyve，由 Hackaday 的一个朋友[Matt Westcott]开发。

Bandersnatch 的热心观众能够确定，节目结束时的尖叫声[在载入 ZX 频谱时会显示一个二维码](https://twitter.com/gasmanic/status/1079164419488268288)。这反过来导致了一个假冒 Tuckersoft 公司的真实网站(谢天谢地是 HTML 格式)。网站本身展示了这家虚构公司的软件库和即将发布的软件，但它也更进一步。Bandersnatch 的双重性被带到了网站上，因为对于那些从 URL 中删除“www”的人来说有分支路径。这样做可以从另一条时间线上看到 Tuckersoft 的网站，在这条时间线上，Bandersnatch 从未被创建过，然而，一个. tap 文件中的 Nohzdyve 的可下载副本已经存在。

Nohzdyve 游戏本身是一个垂直滚动的动作游戏，它使用 ZX 光谱的花哨调色板来达到很好的效果。在游戏中获得高分可以通过模拟器(例如 Speccy )或在真实硬件上获得最真实的体验。这可能是一段时间内启动磁带机的最佳理由，但对于那些寻求更少模拟方法的人来说，总会有来自 Tom FTW 先生频道的游戏片段:
