# 遥控车获得风扇辅助下压力杀死特斯拉的 0-60 倍

> 原文：<https://hackaday.com/2021/05/23/rc-car-gets-fan-assisted-downforce-to-slay-teslas-0-60-times/>

特斯拉声称，他们即将推出的新跑车将发布低于 2 秒 0-60 英里每小时的时间。虽然在现阶段它只得到一个闪亮网站的支持，但[[下班后的工程]将这个数字作为他的 RC fan 汽车制造的目标。](https://www.youtube.com/watch?v=32zCMaGEY8I)(视频，下面嵌入。)

我们以前见过这种构造的早期原型，[第一个版本产生了足够的下压力，可以成功地颠倒驾驶。](https://hackaday.com/2021/02/03/driving-upside-down-with-an-rc-fan-car/)新款车型进行了多项改进，以最大限度地提高其横向加速能力。新车型驱动所有四个车轮，这些车轮安装了涂有牵引化合物的粘性轮胎，以获得最大的抓地力。主驱动电机以及风扇和裙部组件现在都安装在汽车的中心，以适当平衡车轴上的空气载荷，并为快速发射提供稳定的重量分布。

结果令人印象深刻，该车 0-60 英里每小时的时间仅为 1.825 秒。一旦赛车可以在下线时更好地启动，可能还有时间。我们希望看到一系列配备风扇的遥控赛车也能上赛道，因为这样的硬件可以提供足够的抓地力。
