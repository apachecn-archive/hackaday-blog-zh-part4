# 机器人弹跳平台学习新技巧

> 原文:[https://hack aday . com/2021/06/30/robotic-ball-bounding-platform-learning-new-tricks/](https://hackaday.com/2021/06/30/robotic-ball-bouncing-platform-learns-new-tricks/)

[T-Kuhn]的[oct-Bouncer 平台自从我们上次](https://www.electrondust.com/2020/05/25/the-octo-bouncer-advanced-bouncing-patterns/)看到它之后，已经学会了一些新的技巧。如果你以前没有见过，这款设备使用安装在其厚而透明的丙烯酸平台下的摄像头的计算机视觉来跟踪 3D 空间中的球，并进行必要的(和微小的)调整，以通过机器人平台实时控制球的运动。

我们喜欢上一部《T1》中十月保镖令人着迷的动作[，它只会变得更好。不仅有一个全新的自定义球检测算法[T-Kuhn]详细解释，现在还有球的位置以及板运动的可视化。然而，还有一个小谜团。时不时地，[T-库恩]会说球会朝意想不到的方向反弹。似乎不是平台本身相关的 bug，但是【T-库恩】有嫌疑。由于球和平台之间的接触是所有控制的来源，而球和平台在反弹过程中接触很少，所以可能是灰尘(或者甚至是球表面本身的微小缺陷)造成的。无论如何，这并没有减损设备的迷人性能。](https://hackaday.com/2020/03/05/robotic-ball-bouncer-uses-machine-vision-to-stay-on-target/)

设计文件和源代码可以在该项目的 GitHub 库上找到，以便那些想更仔细看看的人使用。观看演示视频令人神魂颠倒，因为太多的事情同时发生；你可以在分页符下面查看。
