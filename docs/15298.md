# 3D 打印货架连接器

> 原文:[https://hackaday.com/2022/11/08/3d-printed-shelf-connector/](https://hackaday.com/2022/11/08/3d-printed-shelf-connector/)

有时候，你真的需要一个定制的架子。无论你是想要一个形状怪异的空间、怪异的物品，还是想要与众不同的东西，制作自己的搁板都可以让你的地方更像家。 [The Plus Shelf by [shurly]](https://www.instructables.com/-Shelf/) 旨在通过 3D 打印支架，让打造自己的搁板变得更加简单。

这些连接器不仅仅是紧贴着架子的木头。+符号的两端实际上位于一个 3/8 英寸的钻孔凹槽中，提供了一个更安全的配合。这些作品被印在一个物体上，然后染成各种明亮的颜色，让架子看起来很流行。将一张胶合板切割成合适的尺寸后，这些小房间就用饼干组装起来了。小隔间边缘的 45 度角让整个搁板系统更加美观。

最后的架子有一点摇晃，但这可能是因为死亡的架子连接器使他们“弯曲”由于摩擦配合的不稳定性，搁板连接器被超级粘合到搁板盒中。[shurly]希望一种金属版本的连接器在未来能够消除这些问题。

这个架子系统不适合你吗？也许你更喜欢这种[复古可调节搁板方式](https://hackaday.com/2022/11/04/adjusting-shelves-like-its-1899/)或者这种 [MP3 播放器搁板](https://hackaday.com/2021/02/12/ikea-shelf-becomes-kid-friendly-mp3-player/)。
