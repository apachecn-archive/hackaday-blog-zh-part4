# 自定义胫骨键盘，在游戏中助你一臂之力

> 原文：<https://hackaday.com/2020/03/11/custom-tibia-keyboard-for-a-leg-up-in-the-game/>

[精英蠕虫]写信告诉我们一个很酷的小键盘，它可以让玩某个游戏变得更容易。你可以移动你的角色的方法之一是在方向模式下使用数字小键盘，加上控制和换档，但是这两者相距太远，不能用一只手盲目驾驶。这就是[精英蠕虫]需要的所有动机，来构建一个只包含基本要素的定制键盘。

键盘由 Arduino Pro Micro 控制，这对于这种类型的构建来说是相当标准的——通常是这样或很小。[精英蠕虫]使用樱桃 MX 棕色来获得良好的触感，并添加 led 来获得紫白色的下辉光。我们喜欢印刷键帽的外观，并且印象深刻，因为众所周知，这些水果开关杆的公差非常严格。

开始考虑定制小键盘的一些用途？这个东西是完全开放的，如果你很好地请求，[精英蠕虫]甚至会把 PCB 文件发送给你。看看你是否能在没有鼠标的情况下度过休息时间，并在等待时查看构建视频。

想要更大的灵活性？多使用几个开关！
