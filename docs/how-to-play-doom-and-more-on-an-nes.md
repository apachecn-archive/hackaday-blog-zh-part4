# 如何在 NES 上玩毁灭战士以及更多游戏

> 原文：<https://hackaday.com/2019/08/29/how-to-play-doom-and-more-on-an-nes/>

毁灭战士在当时是一个突破性的游戏，并且变得如此受欢迎，以至于现在它基本上是硬件黑客的“规模香蕉”。Doom 已经移植到无数设备上，大部分设备都有足够的处理能力原生运行游戏。最近，这一系列兼容厄运的设备扩展到包括 NES，即使该系统肯定没有足够的能力来运行它，如果没有特殊的帮助。如果你想要自己的末日 NES 弹夹，[这段视频将向你展示如何制作它](https://www.youtube.com/watch?v=gCWhWBtu0LA&time_continue=327)。

不久前，我们展示了来自[TheRasteri]的原始版本,它详细介绍了如何在相对较弱的系统上运行这样一个资源密集型游戏。你只需要输入作弊代码“RASPI”。完成所有繁重的工作后，是时候将它放入看起来逼真的弹药筒中了。

为了让所有的东西都适合供体盒，首先盒中的 IC 被移除(除了锁定 IC ),并替换为定制的 ROM 芯片。对原始电路板的一些修改也必须焊接在一起，因为新芯片的引脚不完全匹配。然后，必须移除 Raspberry Pi 上的大多数引脚接头和支持硬件，并将其焊接在一起。然后，[ther astri]检查以确保所有这些额外的硬件不会从 NES 吸取太多的能量而使其过热。

最初的项目本身就令人印象深刻，但随着厄运卡带的完成，这真的使它成为完美的 NES 黑客，也为许多其他定制游戏打开了大门，包括像 Mario64 这样的东西。
