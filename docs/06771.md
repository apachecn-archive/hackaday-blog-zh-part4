# 拇指鼠标把你的手放在键盘上

> 原文：<https://hackaday.com/2020/06/29/thumbmouse-keeps-your-hands-on-the-keyboard/>

让我们面对现实吧，笔记本电脑上的触摸板很糟糕，从来就不是全天使用的主要鼠标。从长远来看，并不是说外部鼠标在你的肩膀和脖子上会更好——特别是如果你伸手越过一个 10 键，然后回头使用它。那么答案是什么呢？舒适、便携的鼠标解决方案是什么样的？这种事可能吗？

【马蒂亚斯 n .】有个想法:[让鼠标成为你手的延伸](https://hackaday.io/project/167075-thumbmouse)。这个想法是，通过在你的拇指或食指上戴一个电池供电的蓝牙指针，你可以无缝地来回转换，减少整体压力。触控板包括一个按钮，用于覆盖左击。为了使它成为一个完整的鼠标，[Matias]计划增加额外的右键和中键按钮，以及一个用于滚动的操纵杆。

[马蒂亚斯]开始用黑莓 9900 轨迹球模块设计拇指鼠标，但发现它对于现代鼠标的需求来说太慢了。结果是触控板模块更适合:它的响应速度快得多，动作也出奇地灵敏。

当然，标准鼠标仍然有它的位置，但它总是可以改进的。就这些而言，[这款完全模块化的鼠标可能会成为终极生物](https://hackaday.com/2020/04/28/modular-mouse-packs-features/)。
