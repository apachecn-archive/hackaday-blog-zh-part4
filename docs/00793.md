# 老鼠精神控制的魔笛旨在混合魔法和科学

> 原文:[https://hack aday . com/2018/10/03/the-magic-flute-of-rat-mind-control-aims-to-mix-magic-and-science/](https://hackaday.com/2018/10/03/the-magic-flute-of-rat-mind-control-aims-to-mix-magic-and-science/)

这很不寻常。看看老鼠精神控制的[魔笛，作为一个项目，它是关于改变乐器的反应，而不是关于入侵乐器本身。这是[Kurt White]参加 Hackaday 奖的](https://hackaday.io/project/161294)[乐器挑战赛](https://hackaday.io/prize/details#five)的参赛作品，它既有趣又与众不同。

![](../Images/ef8afbc055775d6de120f73fae105d2c.png)

The Raspberry-Pi controlled, IoT Skinner box for rats, named *Nicodemus*.

[Kurt]创造了一个便携式的，连接互联网的，自动的食物分发器，它有一个实时的视频流和播放录音的能力。那个装置(名为 *Nicodemus* )被用作剥皮箱来训练老鼠——在任何可能发现老鼠的地方——当它们听到小录音机(这是一种笛子)播放的黑色安息日*钢铁侠*的几个小节时，利用操作性条件反射让它们期待食物。)

简而言之，这种笛子可以让人像变魔术一样召唤出一大群老鼠，因为它们已经被*尼哥底母*训练得可以把*钢铁侠*和食物联系起来。

该系统的许多元件是由老鼠对声音偏好的研究结果提供的，以及它们辨别不同旋律的能力，只要存在正确的频率。召唤的部分都是关于科学的，但如何保护自己免受成群饥饿的啮齿动物的伤害呢？这些动物带着锋利的牙齿和对食物的高度期望来到这里。根据[Kurt]的说法，这就是魔法的来源。他似乎非常确定，一个将木制录音机转换成魔笛的仪式就是一个人所需要的全部保护。
