# 有形编程将代码带入现实世界

> 原文：<https://hackaday.com/2019/11/17/tangible-programming-brings-code-into-the-real-world/>

我们喜欢[Amos]的[有形编程项目](https://hackaday.io/project/162737-tangible-programming)的想法。这让我们想起了那些古老的 Radioshack 电子实验室，在那里，电路概念呈现出物理的一面，这使得它们比工程教科书中的抽象概念更容易理解。

MIT [Scratch](https://hackaday.com/2019/01/09/mit-scratch-3-0-opens-new-doors-for-users-and-builders-alike/) 以一种易于理解的可视化方式教授许多编程概念。然而，从根本上说，人是触觉动物，能够真切地感受和看到摆在面前的代码对许多年轻的学习者来说可能是开创性的。尤其是那些大脑喜欢身体接触和互动的人，如多动症或阿斯伯格综合症患者。

这些电路板都有颜色编码，并通过 I2C 总线进行通信。每个电路板的逻辑和通信由 ATTiny 或 ATMega 处理。当前的处理过程可以通过发光二极管甚至有机发光二极管显示器看到。数字通过指轮开关或跳线输入。

当然，由于块的物理性质，代码概念将是简单和集中的。整数运算、简单循环和 if/else 条件句。围绕这一点可以构建很多概念，它可以成为前面提到的 Scratch 的自然跳板，并最终成为像 python 一样简单易学的语言。
