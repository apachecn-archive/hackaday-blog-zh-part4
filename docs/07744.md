# 用于各种用途的开源机械臂

> 原文:[https://hack aday . com/2020/09/15/开源-通用机械臂/](https://hackaday.com/2020/09/15/open-source-robotic-arm-for-all-purposes/)

一组帮手是一个很好的工具，尤其是在焊接或粘合小部件是一项常见任务的情况下。然而，我们真正想要的是一只机械臂。当然，它可以帮助我们设置胶水或焊料，但它也可以完成几乎任何其他分配给它的任务。像这样的通用工具可能对我们大多数人来说都遥不可及，除非我们有一台 3D 打印机在家里制作这个开源机械臂。

来自[Giovanni Lerda]的 KAUDA 机械臂是一个带有抓取工具的五轴臂，并且有一套完全开源的原理图，因此它可以在任何 3D 打印机上打印。机械臂使用三个步进电机和两个伺服电机，并基于 Arduino MEGA 2560 进行控制。电气原理图也是开源的，所以启动并运行这个只是打印、布线和实现一些软件的问题。为此，有可用的软件示例，它们可以很容易地修改，以适应一个人的机器人的需要。

像这样的项目可能对任何其他项目都有帮助，或者只是为你自己甚至是在教室里上一堂机器人课，因为现在许多学校都有自己的 3D 打印机。由于一切都是开源的，这比我们见过的试图让机械臂再次运行的其他项目要简单得多。
