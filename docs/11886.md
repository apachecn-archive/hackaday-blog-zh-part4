# M5Paper 获得开源天气显示固件

> 原文:[https://hack aday . com/2021/11/12/M5 paper-gets-open-source-weather-display-firmware/](https://hackaday.com/2021/11/12/m5paper-gets-open-source-weather-display-firmware/)

我们知道你喜欢烙铁，我们自己也很喜欢。但事实是，模块化组件和高性能开发板使现代硬件黑客能够以比以往少得多的焊料烟雾完成工作。事实上，有时候完成项目所需要的只是正确的代码。

举个例子，看看[【Danko berto VI】在最新 *Volos 项目*视频](https://www.youtube.com/watch?v=GlxTO4QSkw4)中展示的光滑的电子纸天气显示器。虽然这确实符合 DIY 项目的描述，但他不需要自己组装任何硬件。M5Paper 是一个 ESP32 开发套件，围绕一个清晰的 4.7 英寸、960 x 540 电子纸面板设计，包括从环境传感器到内部 1150 毫安时电池的一切。要让你的掌上电纸书梦想成真，你唯一需要提供的就是软件。

由[Danko]提供的天气显示代码应该可以让你朝着正确的方向前进。现在不要误解我们的意思，只是把他的代码闪到设备上，然后把它放在你的桌子上，当然没有什么不好意思的。这是一个漂亮的界面，我们都知道，少量的开源代码通常可以让一个标准的消费设备变得与众不同。但是通过使用他提供的代码作为出发点，你可以把这个交钥匙设备变成你自己的。
