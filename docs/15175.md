# 打造你自己的混凝土 3D 打印机

> 原文：<https://hackaday.com/2022/10/25/build-your-own-concrete-3d-printer/>

几个月前，我们没有注意到[Nikita]关于[建造混凝土 3D 打印机](https://medium.com/@Nik_chen/how-to-assemble-your-own-concrete-3d-printer-4e6da490b6e6)的帖子，但这个想法似乎很合理:建造一个基本的 CNC XY 轴，然后添加砂浆泵和软管来沉积混凝土。下面的视频显示了机器的运行情况。

虽然它看起来很有趣，但本质上没有真正的 Z 轴，所以这将限于一些相对较薄的形式，除非你，也许，做了几层，然后进一步提起机器。我们还假设湿混凝土根本不会架桥。不过，这可能是一个有趣的项目，尤其是如果你有一个备用的数控 XY 轴浮动。

但是，如果你什么都买，你会看到大约 7000 美元的估计成本。我们假设混凝土有足够的重量，传统的 3D 打印机可能无法切割它。不过，我们确实想知道，连接一个传统的塑料挤出喷嘴来支撑混凝土是否有什么好处。

对于更复杂的机器来说，这可能是一个很好的起点。特别是，[Nikita]指出，变频驱动的螺杆泵是理想的，因为它允许您改变挤压速率，并提供稳定的混凝土流。有了这些知识，如果你曾经建造过 3D 打印机或 CNC 机器，你可能很容易就能解决剩下的问题。

当然，这不是我们见过的第一台混凝土打印机。我们之前看到的那个有能力做一些[相当惊人的事情](https://hackaday.com/2014/09/01/update-3d-printed-concrete-castle-completed/)。
