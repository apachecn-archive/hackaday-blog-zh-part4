# 特斯拉线圈电动自行车是无线的

> 原文：<https://hackaday.com/2020/09/09/tesla-coil-electric-bikes-is-wireless/>

电动自行车，实际上所有的电动汽车，都有一个主要的缺点:电池的重量和成本。即使使用锂，自行车的电池组也很容易比自行车本身重，而且成本几乎一样高。但是由于[LightningOnDemand]最近的发明，不得不处理这个缺点可能成为过去。当然，这将依赖于特斯拉线圈的巨大基础设施，因为这是这辆自行车如何接收它需要运行电动马达的电力。

用于演示的特斯拉线圈也不逊色。它是内华达闪电实验室的一部分，能够产生强大的冲击力。为了从线圈接收电能，自行车(实际上是三轮车)使用一种金属“伞”，然后将能量发送到电动马达。自行车后面拖着一条链条，以便有一个接地点让电完成电路。然而，范围有限，如果自行车行驶太远，特斯拉线圈将开始电离通往地面的路径。

虽然我们不能现实地期望特斯拉的想法是在全球范围内，免费的，无线的电力为我们的自行车提供动力，但看到他的工作被证明是有趣的，即使是像这样的小规模。当然，不需要一个研究实验室就可以开始研究特斯拉线圈。[这一个是由普通的家用部件](https://hackaday.com/2017/11/25/low-end-parts-make-tesla-coil-with-a-high-end-look/)制成的，仍然可以获得创造特斯拉线圈的标志性效果所需的电压。
