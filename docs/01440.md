# 一个整洁的钢笔视点构建

> 原文:[https://hackaday.com/2018/12/04/a-neat-pen-pov-build/](https://hackaday.com/2018/12/04/a-neat-pen-pov-build/)

我们已经看到许多视觉暂留(POV)建立在自行车轮子、棍子和许多其他支架上，但这一个把它放在新的东西上:一支笔。[Befinitiv]正在寻找一种新的方式来为日常设备添加一些智能，结果是一个整洁的 [POV 显示器，适合一支笔](https://befinitiv.wordpress.com/2018/12/01/smartpen-with-128x64-rgb-pov-display/)。128 x64 像素，清晰度不高，但这个版本使用了许多有趣的技术。

[Befinitiv]决定尝试一种不同的展示方式。这个原型不是将 led 安装在外面直接发光，而是使用光纤电缆来创建一个更薄的显示器，看起来更有吸引力。这包括将 16 个芯片放在 PCB 上，然后添加一个用于光纤电缆的 3D 打印支架。LED 发出的光照射到这根电缆上，电缆连接到笔顶部的另一个支架上。这就产生了一个比裸露的发光二极管本身更微妙、更吸引人的显示屏。这仍然是一项正在进行的工作)，但这是一个有趣的构建，它展示了一种向小型设备添加 POV 的不同方法。您可以在[【Befinitiv】的 Bitbucket](https://bitbucket.org/befi/pen15/overview) 中抓取所有的文件来制作您自己的文件。
