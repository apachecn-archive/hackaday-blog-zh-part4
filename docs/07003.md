# DIY 袖珍 MP3 播放器

> 原文:[https://hackaday.com/2020/06/29/diy-pocket-mp3-player/](https://hackaday.com/2020/06/29/diy-pocket-mp3-player/)

当[中微子-1]看到 DFRobot 的 DFPlayer 模块时，他决定要[制作自己的复古 MP3 播放器](https://www.instructables.com/id/Arduino-Retro-Style-MP3-Player/)。这个微型模块配备了大量有趣的功能，如均衡器调节、音量控制和 3 瓦放大器等。如果你想要的话，它甚至可以在歌曲之间播放广告。

使用来自微控制器的串行命令可以很容易地控制 DFPlayer 模块，使其成为大型项目中一个方便的子系统，并成为流行的 VLSI 芯片或难以获得的 WT2003S IC 的潜在替代品。[Neutrino-1]很好地引导读者完成构建，使得重新混合、重用和共享变得相当容易。

硬件分类后，你所要做的就是刷新固件，并在 SD 卡上加载一些 MP3。甚至还有一个小的 Python GUI 来帮助你启动和运行你的新播放器。[中微子-1]还向用户介绍了 U8g2 显示库，他说这比常见的 Adafruit SSD1306 库功能更丰富。干得好[中微子-1]！
