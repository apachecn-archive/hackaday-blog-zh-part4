# 让你的乐高飞起来

> 原文:[https://hackaday.com/2019/02/10/make-your-lego-fly/](https://hackaday.com/2019/02/10/make-your-lego-fly/)

我们可能都曾经通过将乐高扔过房间来让它飞起来，但 Flite Test 提出了一个稍微更优雅的解决方案:他们[将乐高四轴飞行器改装成飞行的](https://www.youtube.com/watch?v=bp5xyjXOQf8)。他们通过添加微型飞行控制器、电池和电机/转子来取代[乐高城市北极空运套件](https://shop.lego.com/en-US/product/Arctic-Air-Transport-60193)中的乐高组件。这种组合令人惊讶地飞得很好，这要归功于一种周到的设计，它平衡了机箱内较重的组件。

乐高纯粹主义者可能会震惊地听到转换确实使用了少量的胶水:建造者认为标准的乐高积木粘在一起不足以固定马达，这是一个公平的观点:四轴飞行器的手臂是飞行中承受压力最大的部分。但除了胶水，这是一个非常忠实的转换:它看起来仍然像一个乐高模型，但像四轴飞行器一样飞行。

这些年来，我们已经展示了一些乐高黑客，包括由 ESP32 控制的[乐高坦克和](https://hackaday.com/2018/11/01/mini-lego-technic-tank-patrols-your-desk-under-esp32-control/)[乐高键盘](https://hackaday.com/2017/12/12/connecting-cherry-mx-key-switches-to-lego-just-got-easier/)。你也可以在这里阅读乐高[的历史。](https://hackaday.com/2018/06/26/ole-kirk-kristiansen-and-the-lego-frontier/)
