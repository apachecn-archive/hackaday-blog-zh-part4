# 那次雅达利破解了任天堂娱乐系统

> 原文:[https://hackaday . com/2018/10/22/that-time-atari-cracked-the-Nintendo-entertainment-system/](https://hackaday.com/2018/10/22/that-time-atari-cracked-the-nintendo-entertainment-system/)

这是 1982 年假日购物季之后视频游戏行业最黑暗的时刻。第三方开发的游戏洪流已经淹没了家用视频游戏机市场，达到了饱和的程度。它引发了零售商之间的价格战，在上架不到一个月后，新发布的产品比建议零售价低 85%。堆积如山的仓库存货卖不出去，使得像雅达利这样的公司选择[将商品倒入奇瓦瓦沙漠](https://www.imdb.com/title/tt3715406/)，而不是面对迫在眉睫的税单。结果，整个家庭视频游戏行业似乎在一夜之间衰落了。

一家公司单枪匹马让视频游戏重新成为主流。那家公司就是任天堂。他们表面上被视为视频游戏行业的“救世主”，尽管事实上微型电脑游戏仍然蓬勃发展(历史往往由胜利者书写)。尽管如此，他们的任天堂娱乐系统(NES)是一个创新的控制台，具有滚动屏幕的游戏，类似街机的精灵。但是他们用来避免重复 1983 年崩溃的策略是使用任天堂的质量印章来严格控制他们的市场。

从第三方开发者的角度来看，任天堂的品质印记不仅仅代表了盒子艺术上的另一个标志。它代表了你可以做什么和不可以做什么。这些第三方许可协议规定了可以制作的游戏类型、游戏的制作方式、游戏的零售时间表，甚至是你的公司可以制作的游戏数量。从客户的角度来看，seal 代表着对产品的信心，任天堂将竭尽全力确保做到这一点。

这是一个雅达利子公司如何破解原始任天堂的硬件安全并开始将其放入非官方墨盒的故事。

> 在家族早期有一些都市传说。据称，任天堂在零售层面使用了一些强有力的策略，他们强行阻止了平台的逆向工程。
> 
> *–**马克·莫瑞斯，前 Tengen 程序员**T3】*

## 兔子芯片来了

[![](../Images/d9ffd5437e81cd6d4afa9c82aaf4d31a.png)](https://hackaday.com/wp-content/uploads/2018/10/nes-cic-diagram-from-tech-manual-themed.jpg)

NES 10NES diagram from Nintendo Service Manual

印章的另一半是任天堂的安全锁定系统，被称为 10NES 计划。本质上，这个程序由每个 NES 控制台内的“主”IC 和每个游戏模块内的相同“从”IC 组成。在插入官方游戏模块并给控制台加电时，主 IC 上的锁向从 IC 上的钥匙发送复位和初始化信号。如果返回了正确的响应，游戏模块将启动游戏，如果没有，CPU/PPU 复位线将被拉低，同时在显示器上出现 1Hz 的方波和单色闪烁屏幕。由于任天堂严格的许可条款，如果一家公司想在 NES 上发行，10NES 程序是唯一的途径……直到 Tengen 出现。

Atari Games 的子公司 Tengen 最初是为了将投币式街机游戏移植到像 NES 这样的家庭平台上而创建的。天正硬件工程师试图通过化学剥离对 NES CIC 进行逆向工程，但当这些努力未能产生完整的旁路解决方案时，该公司决定成为任天堂的授权厂商。在成为官方后，Atari Games 的律师找到了一个漏洞，以便通过美国版权局获得 10NES 程序的代码。通过声明 Atari 是 10NES 程序侵权案件的被告，因此对该信息知情，但是，在请求时不存在这样的诉讼。

毫无疑问，这是一个见不得人的举动，但结合新获得的代码元素和 Tengen 工程师的硅设计，NES 的锁定芯片被破解了。他们称之为兔子芯片。这种芯片意味着 Tengen 是他们自己命运的主人，并允许他们发布像 RBI 棒球这样的游戏，其中包括真正的 MLB 球员。然而，创造游戏的人并不知道兔子芯片的存在。

> “我对正在发生的事情完全是天真的…我不知道，我以为我们在玩标准的任天堂。”
> 
> **史蒂夫·沃伊塔，有编程人员**

## 天正的一小步，第三方的一大步

不出所料，任天堂会对天正发布非官方墨盒进行报复。1990 年，[美国任天堂起诉天正](https://scholarship.law.georgetown.edu/cgi/viewcontent.cgi?referer=https://en.wikipedia.org/&httpsredir=1&article=1814&context=facpub)侵犯 10NES 程序版权。Tengen 的辩护基于这样一种想法，即 Rabbit 芯片需要与 10NES 芯片功能一致，以防止任天堂禁止 Tengen 在 NES 硬件的任何未来版本上发布游戏。Tengen 甚至质疑锁定系统专利的有效性，因为这是显而易见的，但这一切都无济于事。

毫无疑问，受到 Tengen 对抗任天堂的鼓舞，其他第三方开发商如 Camerica、Active Enterprises 和 Color Dreams 创建了自己的 10NES 旁路电路。这些方法通常涉及[创建一个电荷泵](http://kevtris.org/mappers/lockout/index.html)，试图“击晕”CIC 锁定芯片。这些游戏盒的用户在不知不觉中损坏了他们游戏机的耐电压部件，但像 Action 52 这样将几十种游戏装入一个盒中的诱惑将是一种令人难以拒绝的新奇事物。

当时，33%的美国家庭拥有任天堂娱乐系统。有了这样的市场渗透率，其他游戏系统几乎无关紧要。因此，在版权侵权案发生几年后，雅达利游戏公司将在法庭上挑战任天堂对视频游戏行业的虚拟垄断。针对任天堂的反垄断案不会达成裁决，因为双方同意一项未披露的和解协议，然而，有理由认为，如果该案有利于雅达利游戏，视频游戏世界将会与今天大不相同。

天正挑战任天堂的巅峰影响力，最终改变了第三方开发者与平台持有者共存的方式。每一方都会更像合作伙伴一样对待彼此，因为视频游戏平台的成功永远与为其编写的软件交织在一起。Rabbit 芯片的推出开创了一个先例，即没有一款游戏机的安全性是不受审查的。谁知道如果历史有一点不同，我们可能会收到更多的端口，就像下面视频中未发布的 Tengen 版大理石疯狂:
