# 这位艺术家拖着脚走过沙地和雪地

> 原文：<https://hackaday.com/2020/01/11/this-artist-drags-his-feet-across-sand-and-snow/>

几年前你可能看过西蒙·贝克的作品。这位雪地艺术家以只用雪鞋创作大型艺术作品而闻名，多年来一直在创作受几何启发的分形和数学形式。作为一名野外定向员和地图绘制者，他通常会提前计划好自己的作品，并根据平坦的地形选择场地。斜坡的缺乏阻止了滑雪者事先穿越该区域，并有助于测量绘制所需的线条。

他首先用指南针测量他离中心的距离，然后沿着直线走向远处的一点，根据与其他线的相对位置画出曲线。一旦做出了主线，他就用步数来测量沿途的点，并通过连接这些点来连接第二条线。在填充阴影区域之前，线通常被行走三次以固化它们。结果令人着迷。

从那以后，他扩展到了沙艺领域，使用同样的技术在沙滩上的滑雪场和国家公园为他赢得了声誉。不幸的是，潮汐模式，海藻和海滩碎片使它更难达到原始状态，但他仍然设法创造了一些令人印象深刻的艺术作品。
