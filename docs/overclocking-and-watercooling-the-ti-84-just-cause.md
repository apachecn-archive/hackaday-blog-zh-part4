# 超频和水冷 TI-84，只是因为

> 原文：<https://hackaday.com/2020/06/16/overclocking-and-watercooling-the-ti-84-just-cause/>

TI-84 是一款经久不衰的经典产品——这款计算器帮助许多人度过了高中、大学甚至更高的阶段。黑客的最爱，在各种方面都被推到了极限。[Linus Tech Tips]的工作人员决定加入这个乐趣，[超频一台 TI-84 Plus，并增加水冷功能。](https://www.youtube.com/watch?v=l06PlYNShpQ)

TI-84 使用一个简单的电阻电容电路来产生时钟，使其超频不在话下。通过改变电路中的电阻值，可以使时钟运行得更快。该团队对 PCB 上的焊盘分层有一些问题，但设法在 trimpot 中进行了 sub，这使得时钟可以在运行中进行更改。10MHz 的提升使计算器以 26MHz 运行，在 TI 的《毁灭之港 2》中性能明显更快。没有对 CPU 温度的精确测量，很难说水冷计算器是否合理。然而，该团队做了一项伟大的工作，完全超越了解决方案，用一个定制的冷却块连接到一个巨大的球形水库。

由于超频固有的稳定性问题，以及笨重的水冷管，它在实用性方面不是一个好的黑客。然而，这很有趣，而且总是有价值的。TI 计算器长期以来一直是黑客的目标，如果你愿意，你甚至可以在网上得到它们。休息后的视频。
