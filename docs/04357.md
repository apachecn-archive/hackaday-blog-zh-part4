# 弯曲电场解释了天线辐射

> 原文：<https://hackaday.com/2019/10/02/bent-electric-field-explains-antenna-radiation/>

我们都将天线用于收音机、手机和 WiFi。然而，要理解它们是如何工作的，可能需要一生的研究。如果你对天线辐射的基本物理原理感到生疏，看看下面来自[学习工程]的非常好的[动画](https://www.youtube.com/watch?v=FWCN_uI5ygY)。

视频从一点历史开始。然后它谈到了电荷和它们周围的场。如果电荷匀速运动，它周围也有一个恒定的电场。然而，如果电荷加速或减速，磁场必须改变。但是磁场不会在所有地方同时变化。

在电场改变的地方，电场会有一个扭结。那个扭结解释了辐射。从这个想法，视频建立到偶极天线和更多。观看此视频不会让您为设计下一代宽带对数周期天线做好准备，但会帮助您对天线的工作原理有更多的直观感受。

这是一个即使用复杂的数学也很难解决的话题。我们已经看过其他关于主题的[视频。我们很确定这是那些没有](https://hackaday.com/2017/09/11/antenna-basics-by-whiteboard/)[动画](https://hackaday.com/2016/11/27/start-your-path-to-becoming-an-antenna-guru/)就很难学习的话题之一。
