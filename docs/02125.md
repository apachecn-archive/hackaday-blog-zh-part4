# 一个硬币电池为这个小小的 ESP32 开发板供电

> 原文:[https://hack aday . com/2019/02/22/a-coin-cell-powers-this-tiny-esp32-dev-board/](https://hackaday.com/2019/02/22/a-coin-cell-powers-this-tiny-esp32-dev-board/)

仅仅是为了挑战，仅仅是为了乐趣，仅仅是为了炫耀，仅仅是为了做一点炫耀——所有这些都是接受一个项目的完全正当的理由。似乎有一个或多个这样的东西在这个比给它供电的硬币电池大不了多少的小小的 ESP32 板后面。

从下面的视频来看，[Mike Rankin]一直在努力缩小他的 ESP32 版本的功率和大小。他最近完成了一个项目，用一个 ESP32 微型 D4 和一个有机发光二极管显示器精确地安装在一个 AA 电池座上，他用一个可充电的 14550 填充。由于对外形不满意，他设计了另一块电路板，这一次比电池盒中的 LIR2450 可充电硬币电池大不了多少。除了 Pico D4，该板还配有 USB 充电和编程插座、低压差(LDO)稳压器、加速度计、微型 RGB LED 和 96×16 有机发光二极管显示屏。[Mike]选择在电路板背面添加一对衬垫，并将其用作电容式触摸传感器，而不是占用开关的空间。我们发现这一点非常聪明。

遗憾的是，董事会还没有做太多事情，但这并不意味着我们没有印象。当然，[迈克]对小型化项目并不陌生；去年的[开放硬件峰会徽章](https://hackaday.com/2018/09/27/the-exquisite-badges-of-open-hardware-summit/)就是他的创意。
