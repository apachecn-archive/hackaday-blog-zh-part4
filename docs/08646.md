# 一个简单的 DIY 踏板设置为赛车模拟人生

> 原文：<https://hackaday.com/2020/12/14/an-easy-diy-pedal-set-for-racing-sims/>

赛车模拟场景一直有一个强大的 DIY 亚文化，因为寻求最真实感觉的外围设备的爱好者开始修改现成的产品，以获得更大的真实性。其他人走得更远，从头开始打造自己的产品。[ilge]已经做到了，为模拟赛车组装了他自己的踏板。

该建筑主要依靠 3D 打印组件，用几个弹簧和一些螺母和螺栓将一切固定在一起。踏板臂上的轮齿与安装在电位计上的匹配齿轮相啮合。然后，这些电缆连接到 Arduino Pro Micro，它通过模拟输入读取单个电位计，然后充当计算机的 USB 人机接口设备。

[ilge]用各种游戏测试设置，包括流行的欧洲卡车模拟器和 iRacing。这是一个伟大的廉价方式开始与一个模拟钻机踏板设置。从这里开始，天空才是真正的极限；我们希望看到一个升级版本，在刹车上安装一个测压元件，以获得更好的踏板感觉。如果 H 型变形人不在研发中，我们会很惊讶。休息后的视频。
