# Retrotechtacular:军用接线方式

> 原文：<https://hackaday.com/2020/06/11/retrotechtacular-wire-splicing-the-army-way/>

对于我们这些在很小的时候就开始用电做实验的人来说，最基本的技能之一就是学习如何将电线缠绕在一起。看起来没什么好学的，但是在用你的手指做了几次失败的尝试后，你学到了一些技巧，这些技巧可能会一直伴随你到今天。因此，正如 1941 年的这部通信兵训练电影所示，有一种美国陆军的官方方式将电线缠绕在一起也就不足为奇了。

考虑到在第二次世界大战爆发时，通信兵有近 80 年的战场通信布线经验，他们的方法和材料相当可靠。这部电影主要讲述了 W110-B 型野战电线卷的拼接，通信部队使用这种电线将指挥所与前沿阵地、观察哨和后方梯队连接起来。通常情况下，电线直接铺设在地面上，必须足够坚韧、防水和导电，以使现场电话设备在长环路上仍能工作。因此，钢筋、橡胶和织物包层电缆不是最容易拼接的材料。我们可能会害怕将导线打结所带来的压力，但这是布线团队工作的一部分，他们会尽快完成工作，而且经常冒着敌人的炮火。

这部电影还有一个部分是关于将新线路拼接到现有的正在使用的电路中，使用 T 型拼接，并仔细注意所用结的拓扑结构，以免它们在压力下松开。令人着迷的是，在绞电线这样平凡的事情上花了多少心思，但考虑到利害关系，我们可以欣赏对细节的关注。
