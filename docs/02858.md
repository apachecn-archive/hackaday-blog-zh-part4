# 星球大战的静电特效，用旋转的发光二极管完成

> 原文:[https://hack aday . com/2019/05/04/star-wars-electro staff-effect-done-with-spinning-LEDs/](https://hackaday.com/2019/05/04/star-wars-electrostaff-effect-done-with-spinning-leds/)

想要制作一个《星球大战》中的静电棒的道具复制品，但不确定如何才能最好地在商业端创造出“噼啪作响的能量弧”效果。在几次错误的开始后，他决定利用视觉暂留效应，通过在一个以上的轴上旋转发光二极管来创造螺旋弧形光，这种方法似乎有一些潜力。

许多多轴视觉暂留设备使用一种叫做*滑环*的组件，以保持旋转部件之间的电气连接，但【Bithead】有一个更简单的计划:3D 打印一个框架，并为每个轴提供自己的电池。没有集中的电源意味着没有任何特殊部件的更快的原型，因此更快的概念验证来测试这个想法。

[Bithead]已经计划在新版本中进行改进，但您可以在休息后嵌入的短视频中看到当前的原型。
