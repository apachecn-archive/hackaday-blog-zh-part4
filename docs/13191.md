# 木制带状麦克风的新创意

> 原文:[https://hack aday . com/2022/03/30/a-new-winkle-on-wood-ribbon-microphones/](https://hackaday.com/2022/03/30/a-new-wrinkle-on-wooden-ribbon-microphones/)

没有太多人自己制作麦克风，那些制作麦克风的人通常用塑料和金属等材料制作。[弗兰克·奥尔森]他不仅喜欢制作麦克风，而且他还喜欢用木头制作麦克风，并取得了一些非常惊人的成果。

[【弗兰克】的最新建造](https://youtu.be/HOuL5glv8pA)有点像是 RCA BK-5 的复制品，这是一种世纪中期的经典设计。原版和[Frank]的致敬都是带状麦克风，其中悬浮在磁铁两极之间的薄带状波纹金属充当换能器。但相似之处仅此而已，因为[弗兰克]用层叠的胡桃木单板作为他的带状马达的框架。木头用乙烯切割机切割，堆叠起来，用大量的氰基丙烯酸酯胶水粘合成一个整体结构。下面的视频看起来很简单，但是我们可以想象把所有的东西都整齐地堆放起来并正确地排列起来是一件苦差事，尤其是在处理钕磁铁的时候。切割铝箔带并使之起皱也不是一件容易的事，适当地拉紧它并形成牢固的电接触也不是一件容易的事。

带状马达悬挂在由更多木材制成的外壳中，所有这些都有助于产生温暖、丰富的声音。下面整个视频的画外音是在一对这样的麦克风上录制的，我们认为它听起来和[【Frank】早期的木制 44 型模型](https://hackaday.com/2021/03/18/wooden-you-love-to-build-a-ribbon-microphone/)一样好。他说他有更多的设计正在进行中，我们也期待着听到他们的声音。
