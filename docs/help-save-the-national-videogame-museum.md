# 帮助拯救国家电子游戏博物馆

> 原文：<https://hackaday.com/2020/04/06/help-save-the-national-videogame-museum/>

位于英国谢菲尔德的国家电子游戏博物馆收藏了一个独特的收藏，展示了几十年来的电子游戏及其文化，由于封锁带来了威胁其生存的危机，[发起了一场众筹活动](https://justgiving.com/campaign/savethenvmuk)，我们将一段视频放在了休息时间下方。作为一个相对年轻的组织，它们还没有建立起一个更成熟的组织所拥有的财务缓冲。重要的是，这个遗址和其他遗址能够活着改天重新开放，所以我们强烈建议你去看一看。

在他们的网站上，他们为你生活中无聊的年轻人提供了一个关于活动的页面，但是为了激起你的兴趣，如果你想在将来拜访他们，他们还提供了一个关于展览其余部分的精选页面。

经历疫情的一个可悲之处在于，我们知道，一些我们所珍视的企业和组织可能无法度过这场危机。在过去的一两周里，我们给一些小供应商下了订单，为他们争取了一些额外的业务，毫无疑问，你们也是如此。然而，当受到威胁的组织是一个旅游景点时，就不那么容易了；封锁期间我们不能去旅行。NVM 不太可能是唯一面临困境的景点，所以我们敦促你也要留意那些和你亲近的人。
