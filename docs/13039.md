# 用这种指甲油方法让你的指纹变亮

> 原文:[https://hacka day . com/2022/03/14/用这种指甲油点亮你的指纹/](https://hackaday.com/2022/03/14/brighten-up-your-prints-with-this-nail-polish-approach/)

仅仅 3D 打印一个零件是不够的——在那里你可以做无数的事情！[FuzzyLogic]向我们展示了他将标签、图标和文本添加到 3D 打印中的方法，将它们挤压到打印中，并用指甲油填充产生的空腔！这使得彩色和有用的打印，而不是我们通常最终得到的单调的单色部分。

魔鬼在细节中，[模糊逻辑]已经把细节归结为一种技术。指甲油必须用丙酮稀释，以便流动良好，注射器和针头的特定组合将是你的朋友。当然，不要忘记考虑表面张力——即使用稀释好的指甲油，也不能让凹槽太薄。在棉签上多放一点丙酮有助于防止任何愉快的小意外，一层透明的丙烯酸喷漆将字母牢牢地密封在适当的位置。五分钟的视频告诉你所有这些事情以及更多的事情，比如在典型的 CAD 包中挤出文本和图标的基本知识，并为那些观看到最后的人提供了一点额外的镜头。

在我们的印花上添加标记是一个可爱的点睛之笔！如果你想了解更多，这里有[一个定制的工具更换打印机，带笔附件](https://hackaday.com/2019/07/08/enclosure-needs-labels-make-the-3d-printer-do-it/)为*袖珍操作员*制作漂亮的定制外壳。
