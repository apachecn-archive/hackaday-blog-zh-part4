# 八个马达使这艘船快速前进

> 原文：<https://hackaday.com/2019/12/21/eight-motors-speed-this-boat-along/>

对于硬件黑客来说，在船上闲逛总是有一种奇怪的吸引力。也许这是因为它仍然是一种容易接近的方式，可以让一些东西在自己的动力下以一点速度移动，除非发生灾难，否则对粗心的船工来说最糟糕的事情就是浸泡。[NASAT Channel]是一名越南黑客，他是小型机动船的系列生产商，[他的最新作品是一个特别令人印象深刻的例子](https://www.youtube.com/watch?v=RY3WZDq3bbk)。

这艘船本身是一个相对传统的用玻璃纤维覆盖的膨胀聚苯乙烯船体，但是动力有点特别。他采用了八个无处不在的 775 DC 有刷电机，并将它们用在一个带斜齿轮的星形配置中，斜齿轮依次驱动一个挠性轴，该轴直接连接到船下的螺旋桨。每台电机共用一个水冷管道，由一个小泵提供服务，驱动来自一对廉价的 PWM 电机控制器。我们看到他在一些停泊的船只旁边，在一段河上轻快地跑来跑去，如果我们诚实的话，我们不介意自己也来一次。

我们并不完全相信这样一个粗糙的八路变速箱能够长期可靠地使用，我们很想知道这么多电机实际上是如何平均分担负载的。但是我们喜欢它的大胆，我们认为你也会喜欢它。看看休息时间下面的视频，如果你有灵感，那就带上吊床、朋友和 T2 去试试吧。
