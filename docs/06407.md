# DJ 用磁带而不是唱盘来录制俱乐部音乐

> 原文:[https://hackaday . com/2020/04/30/DJ-scratches-out-club-music-with-tape-not-turntains/](https://hackaday.com/2020/04/30/dj-scratches-out-club-music-with-tape-not-turntables/)

不言而喻，并不是每个人对音乐的品味都是一样的，一个人听起来很惊艳的东西，下一个人听起来就会很嘈杂。但是，即使你不喜欢嘻哈音乐和整个 DJ 场景，也很难不对[杰里米·贝尔]在这里用他自制的磁带循环“刮擦”装置所做的事情留下深刻印象。

大多数人可能都见过俱乐部里的 DJ 使用双唱盘来来回回地刮擦黑胶唱片，以创造出添加到音乐中的效果。一部分是音乐家，一部分是表演艺术家，DJ 和“唱盘歌手”往往是真正取悦大众的人。[Jeremy]的“ScrubBoard”使用 2″录音带的循环，录音室曾用于多轨录音。该环由一个带脚踏板控制的电机驱动穿过一个宽滚筒，他可以用它来快速反转行进方向并控制磁带的速度。一对放音头连接在放大器上，可以放在有时移动、有时静止的磁带上的任何地方。他能创造的声音有节奏，有冲击力，有时很狂热，但总是很有趣。看看下面的视频。

这个版本的擦洗板远远不是[杰里米]制造的第一个。你可能还记得 2014 年我们报道过的[他的第一个原型](https://hackaday.com/2014/10/22/cassette-tape-hack-turns-scratching-into-sliding/)；那个只用了几英尺长的 1/4”胶带固定在板上。他仍然能够从中获得一些很棒的声音，但这个版本应该会为他带来真正的改变。
