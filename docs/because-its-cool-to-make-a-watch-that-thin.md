# 因为把手表做得那么薄很酷

> 原文：<https://hackaday.com/2022/07/13/because-its-cool-to-make-a-watch-that-thin/>

最近[理查德·米勒]和法拉利(没错，就是那辆法拉利)宣布了有史以来最薄的机械表，RM UP-01。

它的厚度不足 1.75 毫米(约 1/16 英寸)。审美是有争议的，价格标签甚至没有列在页面上，但我们怀疑这是一个相当重要的总数。但抛开这两件事，我们想退一步，把它当作一件艺术品来欣赏。无论怎么想象，这都不是一款实用的手表。这款手表相当于人力飞机。不切实际、成本高，并且不如其他现代机械动力解决方案有效。但这并没有减少它给人的印象。

因为它太薄了，一个规则的齿轮堆叠分类不是一个选项。因此，相反，齿轮分布在手表的表面，这导致了薄表盘。这意味着手动上弦以节省空间，单次上弦将持续约 45 小时。任何机械表的心跳都是[擒纵机构](https://en.wikipedia.org/wiki/Escapement)。因此，他们不得不重新设计擒纵机构，使其更加扁平，取消了安全销和安全滚轮，而是使用锚叉来支撑杠杆，以防意外的力或冲击。

这个设计令人难以置信，但也许同样值得注意的是它可以被加工。它由钛加工而成，具有一微米的精度，如果你看过关于测量的激烈讨论，这是一个令人难以置信的壮举。你得到的越小越精确，难度曲线就越陡。

休息之后有一个简短的预告。
