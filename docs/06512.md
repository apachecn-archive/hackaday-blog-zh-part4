# 开源拉曼光谱仪更便宜，但不便宜

> 原文:[https://hack aday . com/2020/05/12/开源拉曼光谱仪——便宜但不便宜/](https://hackaday.com/2020/05/12/open-source-raman-spectrometer-is-cheaper-but-not-cheap/)

拉曼光谱术使用来自激光或其他相干光束的光子的拉曼散射来测量分子的振动状态。在化学中，这对于识别分子和研究化学键是有用的。没有拉曼光谱仪吗？振作起来！[开放拉曼](http://www.open-raman.org/)会给你建立一个的方法。

“入门版”取代了最初使用乐高建造的试验板版本，尽管该版本的计划仍在网站上。[Luc]计划很快推出性能版，性能更好，价格可能会更高。

从材料清单来看，它的造价似乎超过了 2000 美元，其中大部分成本是用于设备的光学部件。这还不包括盛放样本的试管，它会让你多花 250 美元左右。我们不确定构建性能版本的成本是多少，但即使是两倍也不会超出资金充足的大学实验室的预算。

典型地，这些是昂贵的，因为拉曼散射是非常微妙的效应。探测它绝非易事。看起来该项目认为学校会对低成本的拉曼光谱仪感兴趣，他们正在积极寻找学术合作伙伴。

你可能会认为这种乐器很奇特，但它不是我们第一次见到的。它甚至不是第二个。如果你想更多地了解人们用拉曼光谱做什么，你可能会喜欢下面来自 BW Tech 的视频。
