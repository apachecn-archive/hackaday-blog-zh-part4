# 快速菲涅尔黑客攻击最小的头部

> 原文:[https://hack aday . com/2019/11/20/fast-fresnel-hack-embig gens-the-small-of-the-small-the-heads/](https://hackaday.com/2019/11/20/fast-fresnel-hack-embiggens-the-smallest-of-heads/)

除了吓唬小孩子，我们完全不知道为什么有人会需要一个放大脸的头盔。但下面的视频让我们每次看到它都会咯咯地笑，我们认为一个包含快速光学破解的好笑值得一看。

当他不在盒子里玩极客的时候，[好奇的马克]的视频通常有更多的逆向计算主题，比如他最近将一个老式终端从一种虚构的语言转换成一个字符集，或者[帮助恢复一台阿波罗导航计算机](https://hackaday.com/2019/06/18/manufacturing-new-connectors-for-the-apollo-guidance-computer/)。有了这样的珍宝，我们惊讶地得知[马克]的背景是物理学——准确地说是光学——他就读于法国理工学院，著名物理学家奥古斯丁·让·菲涅尔也曾就读于该校。它非常适合这个建筑，因为它有一个巨大的塑料菲涅尔透镜。在有趣地了解了菲涅尔同名镜头的历史后，[马克]继续制作。

这本身就很简单——一个足够大的盒子，可以戴在头上，一端由菲涅尔透镜代替。一条发光二极管——请用暖白色，以免佩戴者穿上一件死气沉沉的衣服——排列在镜头后面的盒子边缘。如果你想变得花哨，也许附上一个安全帽悬挂件会使它更耐磨，但即使这样，看到一个放大和扭曲的头的人走来走去也只是一种嘲笑。一个人可能应该小心不要在戴着这个的时候看太阳，然而，原因在视频的 3:24 开始变得明显。

感谢[Marc]可能是迄今为止最奇怪的 YouTube 面部表情展示，以及一个快速 cosplay hack 的好主意。
