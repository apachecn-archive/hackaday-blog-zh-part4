# 复古:电灯的秘密生活

> 原文:[https://hackaday . com/2021/07/13/retro tech tacular-the-secret-life-of-the-electric-light/](https://hackaday.com/2021/07/13/retrotechtacular-the-secret-life-of-the-electric-light/)

通常，当我们挑选一些东西来打着“Retrotechtacular”的旗号时，它是一部美好的旧时光的电影，当时技术还年轻，电影制作人由一个或另一个公司巨头支付费用来制作一部宣传他们的产品或服务的好处的电影，通常带有不那么微妙的“庆祝进步的三月”的基调。

所以当我们看到这个重拍版的 *[电灯的秘密生活](https://www.youtube.com/watch?v=W79s0PsHhDc)，*来自【蒂姆·亨金】的精彩教育剧*机器的秘密生活*的一集时，我们真的不认为它会是很好的回顾性素材。但是仅仅看了几分钟就让我们想起了为什么这部电视剧在 20 世纪 90 年代是必看的电视节目(当时它首次在美国广泛播出)，尤其是对初露头角的极客来说。当用更习惯于 CGI 动画和高制作价值的眼光来看时，[蒂姆]和他的合作者，已故的[雷克斯·加罗德]，用这些程序完成的每一个都是真正令人震惊的。几乎每一点材料，以及交付，有一个即兴的质量，它掩盖了什么必须采取大量的计划和组织完成。[Tim]和[Rex]显然费了很大的劲，让他们看起来没有费很大的劲，结果是电影以很少节目能够做到的方式专注于技术的本质，此后再也没有了。每集结尾的场景——经常遭遇烟火毁灭——总是很受观众欢迎。他们仍然是。

我们不得不说《秘密生活》第集的重制版本看起来很棒，所有版本都发布在[【蒂姆】的 YouTube 频道](https://www.youtube.com/channel/UCGrpLw1W3P1_BC4J-Hpytww)，每集结尾回顾他谈论制作过程的艰辛是无价的。还发布了他最近的[](https://hackaday.com/2021/02/28/tim-hunkin-rides-again-with-the-secret-life-of-components/)*元件的秘密生活，这是一个为制造商和后院工程师提供实用技巧的宝库，也很值得一看。*
