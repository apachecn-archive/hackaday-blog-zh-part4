# 3D 打印双凸透镜实现 3D 显示

> 原文：<https://hackaday.com/2021/03/10/3d-printed-lenticular-lens-makes-3d-display/>

[Bitluni]一直在试验树脂打印透镜——特别是[透镜状透镜](https://www.youtube.com/watch?v=ktgbtoCD58I)。你可能以前在 3D 贺卡或儿童书籍中见过双凸透镜。通过在不同角度呈现略有不同的图像，您的眼睛会感知到立体视觉，从而产生深度错觉。你可以在下面的视频中看到他的结果。

老实说，即使你不想自己制作这样的显示器，使用激光演示双凸透镜的工作原理也值得一看。当然，你知道理论上发生了什么，但看到它在视觉上暴露出来是很棒的。

这种显示不会取代下一部科幻电影中的特效，但它仍然很酷。138 个镜头和一些软件组成的网格给出了一个相当可信的 3D 效果，至少我们可以在 2D 的 YouTube 上看到它，并听到他在尝试时的兴奋。

最终产品装在一个盖子里，盖子套在智能手机上，驱动显示屏。我们感兴趣的是使用直上树脂使背面平坦，这是一个有趣的技巧。

上次我们看双凸透镜，是为了[让东西隐形](https://hackaday.com/2021/01/25/lenticular-lens-makes-things-invisible/)。我们还介绍了他们如何制作类似 T2 全息图的东西。
