# 先生会忽悠你学 FPGAs 吗？

> 原文:[https://hack aday . com/2022/05/09/will-mister-fool-you-into-learning-FPGA/](https://hackaday.com/2022/05/09/will-mister-fool-you-into-learning-fpgas/)

FPGAs 的杀手级应用是什么？对于一些人来说，吸引力在于可并行任务的超高数据吞吐量，这可以实现一些非常复杂的项目。但是如果你才刚开始呢？80 年代风格的电子游戏怎么样？

MiSTer FPGA 项目创造了一点 FPGA 硬件，使得构建任何老式学校视频游戏或计算机平台变得简单。这是一个巨大的干净的石板。当然，你可以简单地下载别人的 Atari ST 或 Commodore 64 设置并加载它，但如果你想在重建老式视频游戏机的同时*学习*FPGA，你就要动手了。

[复古之狼先生]去年冬天制作了一个视频系列(下面嵌入了预告片)，他开始着手一个项目，使用先生 FPGA 平台从头开始重新制作一个经典的视频游戏机。特别是，他将在 Verilog 中重现纳木错坦克营街机游戏。

这实际上是从零开始(在 gateware 中)构建一个基于 6502 的视频游戏机，所以如果你对逆向计算*或*FPGA 感兴趣，你会在这里学到一些东西。到目前为止，他已经完成了 CPU、屏幕、tilemap 图形和内存，但是还没有完成。要跟进，给自己买些硬件，你可能会赶上。

当然，我们之前已经报道过 FPGA 先生项目，因为我们认为它很酷。如果一台游戏机将成为你进入可编程门肮脏世界的门户毒品，那就这样吧。
