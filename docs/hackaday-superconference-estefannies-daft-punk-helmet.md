# 黑客日超级会议:埃斯特范尼的蠢朋克头盔

> 原文：<https://hackaday.com/2019/01/17/hackaday-superconference-estefannies-daft-punk-helmet/>

成功没有单一的公式，但如果说我们在这些年报道骗局、竞赛和黑客马拉松中学到了什么，那就是，就像地质学一样，压力可以创造钻石。给自己一个高风险的不可能的最后期限，很可能会有一些有趣的事情发生。这就是来自 YouTube 频道" [Estefannie 解释这一切](https://www.youtube.com/channel/UCXX7AV8HDBCZXbKRHJYYODg)"的 Estefannie 去年在湾区创客节(Bay Area Maker Faire)举行时所做的事情，她在 2018 年 Hackaday 超级大会[上停下来，谈论从中产生的交互式蠢朋克头盔](https://www.youtube.com/watch?v=Qu7NQrgAU4I)。

这是 Estefannie 对法国电子音乐二人组蠢朋克的一员 Guy-Manuel de Homem-Christo 所戴头盔的抛光复制品的快速巡演。她的快速谈话，视频如下，概述了它的特点，但我们错过了有趣的背景故事。为此，第二个视频是从零开始的一个月的黑客攻击的开始。

## 你会一路学到的

在决定制造头盔之前，Estefannie 对这一行业的常用工具毫无经验。只有 28 天来完成一切，她不得不:把她的客厅改造成一个工作间；了解如何 3D 打印；打印 58 个独立的头盔部件，包括用于热成型护目镜的模具；在制造工具后，自学如何热成型；组装并完成所有零件；最后，安装电子设备，这是蠢朋克头盔的标志。

她的系列中的三个视频值得一看，看看她让自己经历了什么。埃斯特范尼的学习曲线相当长，有时似乎什么都不管用。热成型特别麻烦——先是热量太多，然后不够，然后真空不够(这是我们见过的其他热成型项目中常见的障碍)。但是完成的护目镜几乎是完美的，即使需要两次着色。

我们不得不说，起初，她的一些伤口似乎是自己造成的，尤其是看到她在头盔表面所做的大量工作。但她希望它是完美的，在填充、打磨、涂底漆和给印刷部分上色方面的额外照顾最终真的得到了回报。当 BAMF 转来转去的时候已经到了最后一刻，最后一分钟的组装被留在了酒店房间里的集会上，但这似乎总是这类项目的方式。

最后，头盔很棒，我们很高兴超级会议的准备工作没有给 Estefannie 带来太大压力——至少我们是这么认为的。现在她有了所有这些伟大的新技能和工具，我们期待着她的下一个版本。
