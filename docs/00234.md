# 老式 LED 灯泡放弃灯丝，改用蜘蛛网时钟

> 原文：<https://hackaday.com/2018/08/01/old-led-light-bulbs-give-up-filaments-for-spider-web-clock/>

我们喜欢普通的东西被赋予新的、不寻常的用途，尤其是当它是“我怎么没想到呢？”情况。[这个悬浮显示的数字钟](https://www.instructables.com/id/Charlottes-Web-Style-LED-Filament-Clock)就是这么一个东西。

在这种情况下，常见的物品是 LED 灯泡的“灯丝”，旨在模仿透明玻璃白炽灯泡的外观。[Andy pough]饶有兴趣地观察了它们一段时间，意识到它们是大型数字时钟的完美片段。钟的框架由弯曲的黄铜 U 形槽形成，通过旋转的支柱安装在橡木底座上。七段显示器布置在框架中，LED 灯丝的公共阳极连接在一起，每个阴极连接到一根非常细的导线。每根电线都穿过框架上的一个随机孔，向下进入底座，与四个 DS8880 VFD 驱动芯片中的一个相连。阳极线在片段后面形成花边状的细丝，抓住光线，使其看起来有点像蜘蛛网。它看起来很棒，但*不适合 gefingerpoken*–框架处于 80 VDC 以驱动 LED 段。时钟通过 60 千赫的无线电链路与英国原子钟同步；请看下面视频中漫长而痛苦的同步过程。

我们喜欢开放式框架的外观，这一点我们之前在[一个同样危险的雕塑谢妮钟](https://hackaday.com/2017/01/17/sculptural-nixie-clock-has-shockingly-exposed-design/)中见过。这给了我们一些想法，如何处理这些灯丝 led，而不是[把它们变回灯泡](https://hackaday.com/2018/06/24/an-artsy-and-functional-led-filament-lamp/)。如果[安迪]听起来很熟悉，可能是因为他以前在这里出现过。首先[为整个经典摩托车品牌](https://hackaday.com/2016/04/07/1921-ner-a-car-motorcycle-reborn-with-epic-parts-remanufacture/)恢复零件库，然后[成为 SMIDSY](https://hackaday.com/2018/03/23/ever-wonder-how-the-bots-on-robot-wars-were-built/) 的设计师，这是英国机器人战争系列第一部中的机器人竞争者。
