# 人工智能识别并锁定谋杀猫

> 原文：<https://hackaday.com/2019/07/01/ai-recognizes-and-locks-out-murder-cats/>

任何养猫的人都知道，你腿上那个咕噜咕噜叫的小绒毛球离变成嗜血的连环杀手只有一小步之遥。给凯蒂一点机会，一些弱小无助的东西将会慢慢地，痛苦地死去。你的小杀手很可能不会炫耀它的杰作，把它的受害者带回家——“看看我为你做了什么，人类！你不骄傲吗？”

尽管杀人猫很有用，但把尸体拖回家让你处理可能会很不方便。为了挫败他可爱的连环杀手[Metric]，亚马逊工程师[Ben Hamm]求助于一个人工智能系统将他满载猎物的猫锁在屋外。[Metric]通过一个猫挡板，他可以随心所欲地进出，多亏了一个螺线管和一个 Arduino，现在可以锁定了。阻止进入[Metric]的决定是基于亚马逊 AWS DeepLens AI 摄像头，该摄像头可以观察到猫拍打翅膀的方式。[Ben]训练了三个模型:一个确定[Metric]是否在场景中，一个确定他是来还是走，一个看看他是独自一人还是由一个没有生命的朋友陪伴，在这种情况下，他被锁在外面 15 分钟，自动捐款给奥杜邦协会——最后一点是纯粹的天才。下面的视频是这个项目的一个简短而有趣的总结，西雅图的观众看起来真的被整个事情逗乐了。

所以你的猫不像[公制]那样是杀人恶魔？基于 RFID 的猫门可能更适合你的需求。
