# 泡泡机器的互联网

> 原文:[https://hack aday . com/2020/07/31/the-internet-of-bubble-machines/](https://hackaday.com/2020/07/31/the-internet-of-bubble-machines/)

每个人都喜欢好的泡泡机。这些奇怪的令人满意的新奇物品从一开始就给儿童和成人带来了无意识的娱乐。[8BitsAndAByte]也有同样的想法，但想[让他们的泡泡机体验一下物联网时代](https://www.instructables.com/id/The-Internet-Controlled-Bubble-Machine/)。

首先，他们修改了一个现成的泡泡机，带有树莓 Pi 和中继模块。Pi 可以通过使用继电器控制机器的电源来轻松触发起泡机制。似乎很简单。这个项目的一部分可能对你来说有点陌生，那就是使用 [remo.tv](https://github.com/remotv/controller) 通过互联网控制机器人。

Remo.tv 是一个免费开源的机器人控制器平台，在之前[我们已经看到【8BitsAndAByte】利用了这个网络控制器。看起来他们真的找到窍门了。他们的文章链接到配置 Pi 的详细设置指南，所以希望这不会太麻烦。](https://hackaday.com/2020/01/06/the-internet-controls-this-monster/)

将物联网设置与 Pi 摄像头相结合，你就可以获得一个直播流，无论有没有气泡，观看[都会有一种不可否认的奇怪满足感。](https://hackaday.com/2020/07/13/bubbles-the-people-pleasing-pandemic-panda/)
