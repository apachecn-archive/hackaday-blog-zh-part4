# 水泥搁板兼作 USB 集线器

> 原文:[https://hack aday . com/2019/04/20/cement-shelf-double-as-USB-hub/](https://hackaday.com/2019/04/20/cement-shelves-double-as-usb-hub/)

我们中的一些人能够在生活中使用 0 到 1 个 USB 端口。我们通常将这些人称为“Mac 用户”。对于我们其他人来说，太多永远不够，我们发现自己在寻找扩展卡和集线器以及各种各样的反常适配器和软件狗。[JackmanWorks]是一个发现自己需要更多连接的人，[所以他建造了这个带有集成 12 端口集线器的漂亮架子。](https://www.instructables.com/id/Cement-USB-Hub-Shelves/)

材料选择是这里的关键，这座建筑用桃花心木和水泥建成，看起来光彩夺目。作为构建的核心，USB 集线器首先被拆卸并密封起来，以防止水泥损坏。热熔胶用于保护 PCB，而绝缘胶带有助于覆盖各个端口。然后，将水泥倒入一个模型中，为架子创建一个总体结构，USB 集线器被浇铸到位。水泥固化后，桃花心木板材被切割并上蜡，然后安装到结构中。这些形成了单独的架子，用来放置手机、硬盘和其他 USB 配件。

搁板的设计使得整个结构由底部搁板支撑，该搁板位于台式计算机机箱的顶部。这是一个有吸引力的作品，水泥结构的重量使它在使用中非常稳定。这很少见，但我们确实偶尔会在这些地方看到货架黑客。休息后的视频。
