# 使用这些简单的线性致动器，为您的下一次构建节省一些步骤

> 原文:[https://hack aday . com/2018/11/02/save-a-steps-on-the-next-build with these-easy-linear-actuators/](https://hackaday.com/2018/11/02/save-a-few-steps-on-your-next-build-with-these-easy-linear-actuators/)

很多项目需要直线运动，但并不是所有的项目都需要高精度的直线滑轨和昂贵的滚珠丝杠。当你只需要轻轻推一下门或从外壳中弹出东西时，找到合适的致动器可能是一件苦差事。

当然，除非有人替你做了这项工作。这就是 PotentPrintables 的[Ali]对这些 3D 打印的线性致动器所做的。这是一种简单的齿轮齿条式设计，适合轻负载，有两种尺寸，支持 9g 微型伺服系统和更大、更强大的版本。每种设计都有一个必须粘在伺服喇叭上的小齿轮，以及一系列适合您需求的齿条长度。打印的部分没有什么花哨，但似乎在正确的地方有材料来承受这些致动器将遇到的负载。[Ali]在 STL 文件中包含了部件列表和构建说明，以及示例 Arduino 代码来帮助您入门。下面的视频展示了执行机构的工作情况。

我们欣慰地得知，[阿里]至少在一定程度上受到了之前一篇黑客帖子的启发，开始了这项设计。我们很高兴他决定分享他的版本；这可能会为我们的下一次构建节省一些步骤。
