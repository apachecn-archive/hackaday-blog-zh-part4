# 用乐高搭建 7 段显示器

> 原文：<https://hackaday.com/2022/03/30/building-7-segment-displays-with-lego/>

在黑客中说出“7 段显示器”这个词，你通常会让人们想象我们在日常生活中遇到的常见 LED 和 LCD 版本。然而，机械版本确实存在，并且[ord]已经收集了一些他们自己的设计。

第一个使用似乎是两个乐高马达来驱动显示器的各个部分。每一段由一对黄色的轴组成，通过黑色的网格代表数字的一部分，如果需要，还有一个减号。[ord]通过使用它来显示来自 LEGO 供电的控制器砖块内的倾斜传感器的角度数据来演示它。[Flickr 上的更多照片](https://www.flickr.com/photos/188122874@N02/albums/72157718967120857)从下面展示了驱动系统。

第二个设计依赖于一个鼓状的机制，似乎只能按顺序显示数字。它的工作方式与自动钢琴没有什么不同。显示每个数字所需的动作被编程为序列，技术针伸出由手摇曲柄或电机驱动的鼓组件的横梁。[ord]用它来显示角度数据，再次证明了这一点。

虽然我们不太可能在轻型飞机上看到乐高显示器被用作攻角计，但如果你想要一个便宜又不可靠的设备，如果受到不适当的挤压，它可能会摔成碎片，你可以这样做。无论如何，这不是我们第一次看到[乐高 7 段展示](https://hackaday.com/2014/08/07/lego-technic-mechanical-seven-segment-display/)，但看到一个新的创意呈现在现有的概念上总是很棒。我们希望看到这样的设计应用到一个别致的时钟中，或者甚至是一个运行在 16 段版本上的新闻收报机。休息后的视频。
