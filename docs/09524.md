# DIY B-电影机器人是 A-OK

> 原文:[https://hackaday.com/2021/03/17/diy-b-movie-robot-is-a-ok/](https://hackaday.com/2021/03/17/diy-b-movie-robot-is-a-ok/)

虽然我们当然同意“来自火星的恶魔女孩”是一个有吸引力的电影名称，但我们还没有看到这个明显的英国 B 级电影的喜悦。如果[Cory Collins]'[Chani 的神奇身材，那个伴随着穿着乙烯基外套和斗篷的魔鬼女孩的笨拙、可怕的机器人](https://photos.app.goo.gl/fn1HnYBueyfJusLJ9)有任何迹象，我们打赌它是令人愉快的坏。

[![](../Images/fb6ed011c1d5d86110e05adc7667403a.png)](https://hackaday.com/wp-content/uploads/2021/03/Chani-treads.jpg) 【科里】能够用不到 50 美元忠实地复制出夏尼——灯光、木材和一切。这个价格标签不包括制作圆顶所需的真空成型器，但嘿，这是对未来项目的投资。

[Cory]从解剖 Harbor Freight 的遥控特技车开始，将内部零件串到一个 3D 打印的行走机构上，该机构经过修改，使用齿轮减速电机，因此行走速度更慢。当查尼在 TPU 的踏板上跺着脚走来走去的时候，遥控车前灯的发光二极管在它的圆顶里面闪闪发光。与电影的[预告片](https://www.youtube.com/watch?v=5i3EtvSjx9A)相比，查尼四四方方的身体是一个巨大的纸雕塑，在我们看来简直一模一样。

我们喜欢 Chani 走路的方式——它有点像一种面向前方的月球漫步方式。正如你在下面的视频中看到的，[Cory]完全捕捉到了机器人的步态，当他缓慢而危险地穿过桌子时，看着 Chani 看起来像水管一样的小胳膊摇摆着，令人发笑。留下来看一些可怕的夏尼在暴风雨中的夜间镜头。

想让你的机器人更像电影机器人一样移动吗？你所要做的就是使用正确的工具。
