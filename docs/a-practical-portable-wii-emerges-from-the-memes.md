# 一款实用的便携式 Wii 从迷因中脱颖而出

> 原文：<https://hackaday.com/2019/01/03/a-practical-portable-wii-emerges-from-the-memes/>

几个月前，[Shank]制造了几乎可以肯定会被载入史册的世界上最小的便携式任天堂 Wii。事实证明，Wii 主板上有很多不必要的硬件，只要小心翼翼，注意细节，就有可能将它物理地切割成一个更小的单元；允许这个特别顽固的黑客把一个真正的 Wii，连同使它便携所需的一切，放进一个 Altoids 罐里。

正如你可能预料的那样，狭窄的控制，滑稽的短电池寿命，以及在使用过程中整个东西摸起来很热的事实，这是一个便携式控制台的糟糕借口。但是这个项目得到的令人难以置信的反应[激发了【Shank】重新开始一个早期的项目:一个更加实用的便携式 Wii，他称之为 PiiWii](https://bitbuilt.net/forums/index.php?threads/shanks-first-wii-portable-worklog-piiwii-pocket.228/) 。这一次，掌上电脑有了更合理的尺寸、更长的电池寿命和更好的控制。它甚至有一个集成的“感应条”，所以你可以用它来使用真正的 Wii 遥控器。它可能不是我们见过的最漂亮的便携式控制台转换，但它肯定是最完整的之一。

[Shank]实际上在一段时间之前已经“完成”了 PiiWii，但是在匆忙完成项目的过程中，他对热熔胶有点过于热心，最终得到了一个当事情开始出错时很难诊断和修复的设备。他搁置了这个项目，转而开发他的 Altoids tin build，这有助于他完善 Wii 缩小技术。有了更清醒的头脑和更多的实践经验，PiiWii 得到了改进，显然会更好。

与我们之前看到的 Wii 笔记本不同，它没有试图添加 GameCube 控制器端口或视频输出功能。它是一个纯粹的手持系统，focus 已经交付了一个大约相当于最初 Game Boy Advance 大小的系统。除了巧妙切割的 Wii 主板，PiiWii 的内部还配备了一个 3.5 英寸的显示屏，一个定制设计的音频放大器 PCB，四个 3400 mAh 的电池，运行时间约为四个小时，一个 3DS“slider”模拟棒，以及一个代替热熔胶的大量 Kapton 胶带。

如果对 PiiWii 有任何批评，那很可能是关于该系统的四四方方的外观。但正如[Shank]解释的那样，这有一个很好的理由:它实际上是内置在项目框中的。他只是拿了一个市售的 ABS 项目盒，Polycase SL 57，用激光切割机在前面开了所有的孔。除了用激光照射 ABS 会释放氰化氢之外，他发现这是一种快速破坏定制外壳的好方法。

去年[我们看了看他小得可笑的 Altoids tin Wii](http://hackaday.com/2018/10/08/worlds-smallest-wii-is-also-worlds-worst/) ，虽然这的确是一个令人印象深刻的项目，但我们很高兴他重新审视了 PiiWii，并表明便携式 Wii 不仅仅是一个新奇的东西。与其他系统相比，Wii 似乎没有那么经常得到便携的待遇，所以我们总是[很高兴看到有人进来做概念正义](https://hackaday.com/2018/01/27/play-a-few-games-of-smash-brothers-on-the-go-with-a-portable-wii/)。
