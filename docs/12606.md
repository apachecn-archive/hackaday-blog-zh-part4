# 微型计算机数控切割金属

> 原文：<https://hackaday.com/2022/01/28/tiny-cnc-cuts-the-metal/>

我们对[Ivan]的工作并不陌生，这一次他正在使用挤压、3D 打印零件和 Makita 刳刨机建造一台相对较小的[数控机床](https://www.youtube.com/watch?v=dkvED8p8bt0)。这些计划可以以很低的成本获得，但是仅仅看着加速的构建就令人着迷。

你可能认为你可以在现有的 3D 打印机框架上安装一些像 Dremel 工具一样切割的东西。你可以这样做，但对于大多数用途，你需要比大多数桌面打印机更硬的东西。你可以看到这个建筑有多坚固，有多个挤压形成的底座和非常坚硬的轴。

从录像来看，这台机器很快加工了一些铝板。当然，有些是在工具的选择上，但看起来机器足够稳定，可以保持工件和工具稳定，允许这种服务。[Ivan]说这台机器花了他大约 600 欧元(670 美元),你需要一台能打印 180 x 180 毫米大的零件的打印机。

在 [Thingiverse](https://www.thingiverse.com/thing:724999) 上有不少类似的大多是 3D 打印的机器，包括一些已经经过[多个版本](https://www.thingiverse.com/thing:3010914)的。如果你有一台旧的 3D 打印机，如果你添加一些打印的部件，你可能会有几乎所有你需要的东西，大概是来自你的新打印机。

如果你想挑选自己的设计，我们已经看过很多 [CNC 版本](https://hackaday.com/2019/05/12/modular-cnc-build-gets-you-both-a-mill-and-a-laser-cutter/)。根据你的期望，它[不一定是一个昂贵的项目](https://hackaday.com/2021/04/29/a-50-cnc/)。
