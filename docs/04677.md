# DIY 标量网络分析仪

> 原文:[https://hack aday . com/2019/12/25/DIY-scalar-network-analyzer/](https://hackaday.com/2019/12/25/diy-scalar-network-analyzer/)

[Steven Merrifield]构建了他自己的[标量网络分析器](https://hackaday.io/project/164610-scalar-network-analyser)，这是一个美丽的东西！[Steve]的 SNA 有一个与 Raspberry Pi 匹配的数字引脚排列，但任何 GPIO 都可以用来操作器件和从 ADC 检索数据。该设计基于 ADI 公司几款久经考验的芯片。他小心翼翼地把它设计得又好又精确，这就是他把它限制在 1kHz 到 30Mhz 的原因。我们认为一旦屏蔽到位，这将是一个非常吸引人的板。

我们之前已经介绍过[网络分析器](https://hackaday.com/2018/06/20/analog-discovery-2-as-a-vector-network-analyzer/#more-310841)及其用途。例如，如果你想知道垃圾箱里的一个神秘电容器如何对某些频率做出反应，网络分析仪可以告诉你。我们甚至尝试用[来组装我们自己的](https://hackaday.com/2017/09/22/network-analysers-the-electrical-kind/)。

在他的[网站](http://stevenmerrifield.com/scalar/index.html)上有更多的文档以及一些额外的示例曲线。从 [OSHpark](https://oshpark.com/shared_projects/bS2bOYZ6) 很容易订购该板，并且源代码可供查看。
