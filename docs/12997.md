# 回流加热板拆卸揭示了最低限度

> 原文:[https://hack aday . com/2022/03/10/reflow-hot plate-tear down-uncovers-the-bare-minimum/](https://hackaday.com/2022/03/10/reflow-hotplate-teardown-uncovers-the-bare-minimum/)

[EEforEveryone]正在努力寻找一种好的回流焊用热板。试了一个便宜的单元后，他又买了一个。他有点不感兴趣。接地是可疑的，床不是完全平坦的。他把它拆开，惊讶地发现里面几乎没有东西。虽然建筑并不完美，但比上一个单元要好。你可以看到一个视频的拆卸和审查如下。

在给它通电之前，第一件事就是给地面系统重新布线。之后，是时候尝试一下了。然而，由于混淆了华氏温度和摄氏温度，他把温度设置得比需要的高很多，产生了一点烟雾。固定温度有所帮助，但仍然有一点烟熏味，最终消退。

判决结果？电热板工作得很好，但在使用它之前，你可能需要检查一下地线。无论如何，就廉价设备而言，这通常是个好主意。但是真正的收获是，看起来你可以毫不费力地自制一些同等的东西。控制器是现成的模块。开关和插头不难理解。加热元件可以是用于 3D 打印机的硅树脂加热器或 PCB 加热器。

当然，还有其他选择。你可以用一只锅。或者为什么没有[华夫饼干熨斗](https://hackaday.com/2021/01/10/turning-a-waffle-iron-into-a-reflow-station/)？还可以做一个[定制 PCB](https://hackaday.com/2021/10/06/using-a-pcb-to-reflow-pcbs-take-2/#more-498940) 。
