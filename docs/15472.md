# 当 Pi 供应不足时，瘦客户机将成为家庭自动化和低功耗计算的首选

> 原文:[https://hackaday . com/2022/11/25/when-pi-supply-falls-short-thin-clients-stand-tall-for-home-automation-and-low-power-computing/](https://hackaday.com/2022/11/25/when-pi-supply-falls-short-thin-clients-stand-tall-for-home-automation-and-low-power-computing/)

对于低功耗计算项目，您需要一台便宜的小型计算机吗？历史上，我们中的许多人会直接伸手去拿树莓派，即使我们并不绝对需要 GPIO。但是随着价格的上涨和供应的下降，[Andreas Spiess]决定是时候寻找[的替代品来代替现在昂贵的 Pi 的](https://www.youtube.com/watch?v=rXc_zGRYhLo)，你可以在休息时间下面的视频中看到。

[![](../Images/fd60abfaf24c66dd747e590ba24edf4d.png)](https://hackaday.com/?attachment_id=563564)

Setting up Debian for IOTstack

许多人只是因为 Pi 的软件生态系统、较低的功耗要求和较小的尺寸而使用它。[Andreas]已经搜索了易贝，寻找价格低至 10-15 美元的瘦 PC 客户机。还选择了一些稍微贵一些的单元，在视频中进行了一些比较。在功耗、计算能力和成本方面，这些瘦客户机与 Pi 相比如何？结果可能会让你大吃一惊！

软件是另一个问题，因为许多 Pi 项目依赖于 Raspbian，一个特定于 Pi 的 ARM64 Linux 发行版。因为 Raspbian 是基于 Debian 的，[Andreas]选择它作为实验的基础。他深思熟虑地包括了 Proxmox for virtualization、IOTstack 和 Home Assistant 等强大的软件，引导观众在 x86-64 硬件上运行 Home Assistant 的每个步骤，并注意 Linux 发行版之间的差异。

总而言之，如果你曾经考虑过走出 Pi 生态系统，进入一般的 Linux 计算，这篇教程将是一个极好的起点。当然，[Andreas]并不是第一个这样做的人，本月早些时候，我们推出了另一款为你的 3D 打印机运行 Klipper 的瘦客户机。在这些无圆周率时代，你找到了自己的完美圆周率替代品吗？请在下面的评论中告诉我们。
