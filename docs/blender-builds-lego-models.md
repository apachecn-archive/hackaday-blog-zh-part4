# Blender 构建乐高模型

> 原文：<https://hackaday.com/2022/11/15/blender-builds-lego-models/>

Blender 是一个免费的开源计算机图形软件包，用于从视频游戏到电影的制作。现在，[正如【乔伊·卡里诺】](https://www.youtube.com/watch?v=LrEHoaq6QFE)所演示的，这个流行的程序甚至可以用来把模型转换成乐高。

Blender 3.4 中的这一新特性允许使用实例属性，从而可以在模型上创建大量的点，而不会对软件造成过度的压力(以及可能的崩溃)。本质上，现有模型被分割成特定间隔的离散点。间隔的间距被设置为与乐高积木的间距完全相同，这使模型具有真实乐高积木的低分辨率外观。从那里，模型砖被创建并放置在这些点中的每一个上，然后颜色可以被单独地转移到砖上。

[Joey]使用的演示是将沙滩球模型转换成乐高，但是在其他模型上使用这些工具会产生一些惊人的结果。他讲述了很多如何制作这些砖块的细节，从那里到订购砖块本身只需要很短的一步。或者，使用这些模型，[将它们直接从 Blender 发送到 3D 打印机](https://hackaday.com/2014/01/03/3d-printering-making-a-thing-in-blender-part-i/)。对自由软件来说还不错！
