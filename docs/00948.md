# 混合机器人行走、变形和飞行

> 原文:[https://hack aday . com/2018/10/15/hybrid-robot-walks-transforms-and-take-flight/](https://hackaday.com/2018/10/15/hybrid-robot-walks-transforms-and-takes-flight/)

[Project Malaikat]是一个 3D 打印的混合两足步行机和四轴飞行器机器人，但它不仅仅是给两足动物贴上一些道具和飞行控制器，然后就可以结束工作了。它不仅是一个定制的设计，能够[小心谨慎的两条腿步态](https://www.youtube.com/watch?v=M4urUQJVc_U)，而且道具被隐藏起来，并通过一些令人印象深刻的连杆机构根据命令展开，使它能够从行走模式转变为飞行模式。

创作者【唐云泰】有 [3D 模型可供下载](https://drive.google.com/file/d/1vL0ov_omLc9Y4LmFOqaG9Q2oh5CN6Ckz/view)(。rar 文件)和 YouTube 上的视频描述包含材料清单，但除此之外似乎没有太多关于[Malaikat]的其他信息。创造者敦促任何人使用该设计时都要小心，因为尽管机器人可能很小，但它实际上有旋转的手。

下面嵌入的视频展示了机器人的动作，以及一个简短的飞行测试，表明虽然测试期间控制有些缺乏，但机器人绝对有能力实际飞行。
