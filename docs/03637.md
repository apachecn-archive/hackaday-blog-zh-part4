# 微型游戏系统是极简主义的一个实验

> 原文:[https://hack aday . com/2019/07/25/tiny-game-system-is-a-experiment in-minimalism/](https://hackaday.com/2019/07/25/tiny-game-system-is-an-experiment-in-minimalism/)

许多人认为智能手机革命会扼杀专用的手持游戏系统，事实上，不难看出为什么。当你已经随身携带的手机能够支持高清 3D 图形和在线连接时，购买最新的任天堂或索尼掌上电脑有什么意义呢？软件开发人员很快就明白了这一点，正如预测的那样，手机游戏在过去几年里绝对是爆炸性增长。

但是与此同时，我们也注意到了昔日简单的手持系统的回归。也许这只不过是怀旧，但[像【米斯拉夫·布雷卡】这样的小型裸机系统已经进入 2019 年黑客日奖](https://hackaday.io/project/166473-tiny-game-console)表明，并不是每个人都对现代游戏的发展方向感到满意。他的系统是作为一个实验而特别设计的，以建立尽可能最小的游戏系统。

就整体设计而言，这个 ATMega328 供电的系统类似于[一个缩小的 Arduboy](https://hackaday.com/2019/07/01/schools-in-session-with-arduboy-curriculum/) 。虽然视觉上的相似之处显而易见，但[米斯拉夫]提供的炸弹似乎表明了一个更加斯巴达式的设备。目前似乎没有任何音频设备，也没有电池和相关电路来充电。正如承诺的那样，除了最基本的必需品，这里几乎没有什么东西。

不幸的是，这个项目开始时有些不顺利。正如[Mislav]在他关于 Hackaday.io 的文章中解释的那样，电路板设计或组件选择中的某个地方存在错误，导致设备无法接受固件。在他回到学校之前，他不会有调试设备的设备，他正在积极寻找可能有兴趣帮助他解决设计问题的志愿者。
