# 认识 MutantC:配有滑动屏幕的 Raspberry Pi Sidekick，QWERTY

> 原文：<https://hackaday.com/2019/09/30/meet-mutantc-your-raspberry-pi-sidekick/>

这些年来，我们已经看到树莓派被塞进了你能想到的几乎所有硬件中。坦率地说，在我们的圈子里，看看你能把 Pi 塞进什么样的不寻常的消费电子产品已经成为一种迷因。但是我们很少看到定制设计的实用外壳，它们实际上发挥了 Pi 的优势。这是一个遗憾，因为正如[rahmanshaber]创造的 MutantC 显示的，那里有一些难以置信的未开发的潜力。

MutantC 配备了 QWERTY 键盘和滑动显示屏，似乎受到了早期智能手机设计的很多启发。你知道，在苹果进来并成功说服所有其他制造商，带硬件键盘的移动设备没有未来之前，他们是怎样的。不幸的是，黑客会话将需要保持束缚，因为目前设备中没有电池。虽然这是[rahmanshaber]说他正在积极研究的东西。

MutantC 中的定制 PCB 将与 Pi Zero 或全尺寸变体一起工作，但[rahmanshaber]警告说，由于担心过热，最新和最大的 Pi 4 不被支持。除了 Pi 之外，零件清单相当短，主要归结为 3D 打印外壳和 QWERTY 板所需的组件:43 个触觉开关和一个 SparkFun Pro Micro。一切都是开源的，所以你可以运行你自己的板，打印你的案例，你将会很好地重温那些双向寻呼机的光辉岁月。

我们很兴奋地看到像 MutantC 这样一个记录良好的开源项目将何去何从。虽然缺少内部电池可能是一些应用程序的阻碍，但我们认为这里的整体外形非常棒。结合[Brian Benchoff]在寻求完善小型键盘的过程中收集的[知识，你将拥有一些非常接近黑客们梦寐以求的神话般的移动 Linux 设备。](https://hackaday.com/2019/04/23/reaction-video-build-your-own-custom-fortnite-controller-for-a-raspberry-pi/)
