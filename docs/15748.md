# 这个无线信号强度测量仪不怕鬼

> 原文：<https://hackaday.com/2022/12/23/this-wifi-signal-strength-meter-aint-afraid-of-no-ghosts/>

原版电影《捉鬼敢死队》是一部经典之作，在上映近 40 年后仍在上演——让我们沉浸其中吧。这部电影的几乎每一个方面，从手动道具到引语，都是一眼就能认出来的，甚至对于那些没有把生活建立在[文克曼]、[斯坦茨]和[斯宾格勒]的教导之上的人来说也是如此。也就是说，我们向[展示这款 PKE 米型 WiFi 扫描仪](https://www.kevsrobots.com/blog/ghostbusters-wifi.html)。

当然，[凯文·麦卡利尔]的项目严格来说属于“只是为了好玩”的范畴。但这并不意味着它没有一点用处。该设计非常接近原始的 PKE 米，并带有一点创造性的许可，使其更容易建造。内部设备包括一个树莓派 Pico W 和一个宽大的 320×240 液晶显示屏。仪表的主体完全是 3D 打印的；设计文件当然是有的。仪表的两臂通过一个单独的伺服机构连在一起移动。

在软件方面，[Kevin]的 GUI 可以让用户看到该地区的 WiFi 热点列表，并从列表中选择一个。从那里，手臂的位置由热点的 RSSI 决定，类似于道具应该如何指示幽灵、幽灵或鬼魂的接近。不在手臂上安装 led 灯可能会错过一点机会，但我们会让它溜走。

下面的视频有完整的设计和建造细节，但公平的警告，它有点长。不过，这可能只是反映了(凯文)为此付出了多少努力。当然，你可能更愿意建造一个“实际上”检测鬼魂的 PKE 测量仪，在这种情况下[我们已经为你准备好了](https://hackaday.com/2020/06/07/bust-your-own-ghosts-with-a-pke-meter/)。
