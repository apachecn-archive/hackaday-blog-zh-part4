# 微小物体的聚焦叠加

> 原文：<https://hackaday.com/2019/12/30/focus-stacking-for-tiny-subjects/>

焦点叠加是一种摄影技术，其中对一个对象进行多次曝光，焦距设置为不同的长度。然后将这些图像合成在一起，以创建比单次曝光更大景深的最终图像。[【林建岳】搭建了一个平台，可以对非常小的物体进行精确的聚焦叠加。](https://pylin.com/2017/10/31/building-a-micro-focus-stacking-rail-using-arduino/)

该装置的核心是一个运动平台，由一个装有线性滑动螺杆的微型步进电机组成。它通过基本的步进驱动板连接到 Arduino 或 PIC。虽然电机对微步进或其他先进技术的响应不佳，但只要正确驱动它，就可以实现每步 15 微米的分辨率。

马达/滑块组合不是特别强大，因此不能容易地用于移动照相机或光学器件。相反，该装置是为拍摄非常小的物体而设计的，在这种情况下，轨道会移动主体本身。

这是一个整洁的构建，对于任何经常进行微距对焦堆栈摄影的人来说都很好。如果你一直在尝试更好地拍摄你的昆虫收藏，这是给你的。这是一项有价值的技术，也适用于显微镜。休息后的视频。
