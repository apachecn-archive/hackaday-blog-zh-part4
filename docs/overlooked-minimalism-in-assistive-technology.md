# 辅助技术中被忽视的极简主义

> 原文：<https://hackaday.com/2018/11/23/overlooked-minimalism-in-assistive-technology/>

如果你的眼睛是 20/20，你可能不会花太多时间考虑处方眼镜。人们很容易忽略这类事情，我们不会责怪你。当我们发现这个作品的时候，它已经有两年多的历史了，但是我们还没有见过任何和它完全一样的东西。[熊掌助食器](https://www.bearpawsf.com/collections/eating-aids/products/assistive-eating-aid)的本质是吸盘底座顶部的旋转磁铁。简单吧？你可能已经在考虑如何在一个周末建立或模拟它，这没什么大不了的。问题是，如果你没有先看过，你能做出这样的东西吗？

具有很大灵活性和扩展空间的过度设计的发明有其吸引力。当你第一次学习 Arduino 时，每个问题看起来都像是廉价演示板的解决方案，有一天你发现自己戴上了 [ATMEGA 手表](https://hackaday.com/2018/08/24/an-arduino-watch-without-a-clock/)。老实说，我们同样热爱这些，但原因完全不同。当所有的花里胡哨都消失了，当除了一个健壮的创造之外什么也没有留下，“只是工作”，你创造了一些美丽的东西。从这段视频的 YouTube 评论来看，这些人可以毫不费力地忽略这个设备的魅力，因为“beard”这个词出现了 95 次，其中一次拼写错误导致“bread”数为 1。Hackaday 的读者是一个较高的口径，应该能够欣赏它的优雅。

目前自我进食的高科技解决方案是一个[机器人手臂，与这个](https://hackaday.com/2018/11/03/dexter-robotic-arm-wins-the-2018-hackaday-prize/)没有什么不同，当我们听到一项不用手吃饭的发明时，我们的大脑就去了那里，我们总是很乐意谈论[机器人手臂](https://hackaday.com/2018/10/03/giant-robot-arm-uses-fluid-power-not-electronics/)。
