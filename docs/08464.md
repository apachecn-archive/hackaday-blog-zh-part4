# 隔膜空气发动机

> 原文:[https://hackaday.com/2020/12/02/diaphragm-air-engine/](https://hackaday.com/2020/12/02/diaphragm-air-engine/)

在物理世界中，工程的一个棘手部分是让机器利用现有的资源和制造技术工作。[汤姆·斯坦顿]已经设计并制造了几个气动 3D 打印引擎，但一直在努力解决空气从 3D 打印的活塞中泄漏的问题。他没有试图制造气密活塞，而是添加了一个橡胶膜和一个聪明的阀门系统，创造了一个[隔膜空气发动机](https://www.youtube.com/watch?v=5B1H5syp4PM)。

![](../Images/4ed27aafe375cb2f5e600f9fe6f3fb60.png)

This GIF is worth 115 words

中心有孔的圆形橡胶隔膜在活塞冲程的顶部与活塞形成密封。黄铜套管和销穿过隔膜，套管密封形成一个带 o 形环的塞子，而销推开一个球，该球作为入口阀对中间腔加压。当活塞缩回时，阀球关闭入口阀，中间腔的出口阀打开，迫使隔膜推压活塞。活塞和隔膜之间的密封一直保持到活塞到达其底部位置，在该位置，压缩空气通过活塞排出，并通过齿轮箱排出。完整的细节请看休息后的视频。

引擎运行起来需要几次迭代。中间腔的容积必须增加，Tom 不得不尝试几种不同的套筒和销长度组合，以获得正确的进气时间。由于他想在飞机上使用发动机，他将最新设计的推力与以前版本的推力进行了比较。最新设计将效率提高了 366%。我们期待看到它飞起来！

我们的大多数老读者都看过【汤姆】的[飞行器](https://hackaday.com/2020/06/25/building-and-flying-a-helicopter-with-a-virtual-swashplate/)的[阵列](https://hackaday.com/2019/08/20/putting-the-coanda-effect-to-work-on-a-quadcopter/)，他也测试过替代能源，比如储存在飞轮中的[动能。](https://hackaday.com/2020/02/13/flywheel-stores-energy-to-power-an-airplane-eventually/)
