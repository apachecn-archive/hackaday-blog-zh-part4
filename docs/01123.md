# 幽灵般的动画眼睛，满足你令人恐惧的需求

> 原文:[https://hack aday . com/2018/10/31/ghost-animated-eyes-for-your-scarring-needs/](https://hackaday.com/2018/10/31/spooky-animated-eyes-for-your-frightening-needs/)

除非你有一个令人难以置信的库存充足的零件箱，否则今年建造这些幽灵般的动画眼睛来吓跑邻居的小孩可能为时已晚。但是明年…

很明显，万圣节的装饰最近有些过头了。这可能不像某些圣诞节表演那样极端，但很多人喜欢增加恐慌因素，而[wermy]似乎把自己列为具有这个季节精神的人之一。像圣诞灯一样，这些眼睛被部署成一串，但不仅仅是闪烁的灯光，它们还会闪烁各种生物令人毛骨悚然的眼睛。眼睛显示在 3D 打印外壳中的独立背光 TFT-LCD 显示器上。一个 ItsyBitsy M0 快车的 SPI 接口可以驱动两对眼睛；驱动更多的显示器是可行的，但是如果你把它拉得太远，帧速率就会下降到一个不可接受的水平。串在黑色以太网电缆的碎片上，偷窥者可以生活在前门旁边的灌木丛中或人行道上，并且只需惊人的适度电力，你就可以从 USB 电池组中获得一整夜的惊吓。

我们喜欢这些的外观，也许明年我们会做些什么。如果你今年仍然有心情害怕，又没有时间做动画眼睛，试试这些简单的 Arduino blinky 眼睛来个快速打击。
