# 建造一个乐高气动引擎

> 原文:[https://hack aday . com/2022/11/17/building-a-Lego-pneumatic-engine/](https://hackaday.com/2022/11/17/building-a-lego-pneumatic-engine/)

气动发动机不是我们每天都用的东西，但它们在实践中很有吸引力。[Nico71]用乐高技术制作了一个引人注目的例子，[它的功能非常全面。](https://www.youtube.com/watch?v=FiKUXffqKVw)

该建筑依靠单个气压缸驱动飞轮。流向气缸的流量由凸轮轴控制阀决定。阀门本身是定制的，由扭结的软管环组成，用于切断水流。除了基本的操作部件之外，发动机还配备了一个节流阀，该节流阀使用了相同的扭结软管原理。主控制阀安装在一个壳体内，该壳体可以相对于发动机机架旋转，以改变阀相对于飞轮旋转的正时。齿轮系统允许精确调整时间。油门和定时控制可在一个整洁的控制面板上操作，并配有怠速调节装置。

那些希望自己建造一个的人可以高兴了，因为[Nico71]只收取很少的费用就能提供指导。我们以前也见过其他的空气引擎，[通常是 3D 打印的种类](https://hackaday.com/2017/03/10/3d-printed-engine-chugs-away-on-balloon-power/)。
