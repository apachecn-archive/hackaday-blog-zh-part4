# 机器人式地回到那里

> 原文：<https://hackaday.com/2020/04/18/get-back-out-there-robotically/>

当与现实进行远距离互动是最佳行动方案时，我们求助于机器人。无论是探索金星表面、海洋深处，还是(暂时)整个社会，将机器人放在那里通常比真人更好。我们无法将机器人发送到其他星球，但我们可以通过远程呈现机器人在其他各个地方轻松获得它们。

[这个微小的远程呈现机器人](https://thecraftyrobot.net/blogs/projects/cheap-telepresence-robot)来自【Crafty Robot】的【Ross】，他们正在使用他们的小型智能机器人平台作为这个微小机器人的基础。智能机器人驱动一个容易创建的纸板平台，配有轮子和卡车，围绕着某种处理视频和网络功能的智能手机。可以使用一套可以在项目页面上找到的 web 应用程序从任何其他计算机上查看和控制机器人。

Smartibot 平台是一个廉价的平台，我们已经看到它可以做其他事情[如驾驶飞艇](https://hackaday.com/2019/11/19/bamboo-skewers-launch-airship/)，创作者希望尽可能多的人可以使用这个快速简单的远程呈现机器人，如果他们现在真的需要这样的东西。该工具包似乎对许多其他有趣的项目也很有用。
