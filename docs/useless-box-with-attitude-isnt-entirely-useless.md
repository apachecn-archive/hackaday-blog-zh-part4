# 有态度的无用盒子并非完全无用

> 原文：<https://hackaday.com/2020/11/03/useless-box-with-attitude-isnt-entirely-useless/>

无用的机器有什么吸引人的地方？毕竟，他们意味着低调激怒。在这一点上，这个游戏的名字更多的是赋予机器内部那个不知名的敌人一种个性。用笑声和调侃让它更霸气一点怎么样？这就是[alexpikkert]的“有态度的无用机器”背后的想法— [每当你扳动一个开关，里面无用的生物就会变得更加恼火](https://www.instructables.com/Useless-Box-With-an-Attitude/)。

在这种情况下，这种生物是基于 Arduino 的，具有两个声卡，用于保存笑声和其他声音。总共有三个伺服系统:两个开关翻转手指各一个，第三个向你打开盒盖。这个版本是完全开放的，而且[alexpikkert]甚至解释了如何为这个外壳重新设计一个钥匙盒。休息后请欣赏演示。

我们喜欢这里的无用机器，尤其是当他们采取新策略的时候。这台机器看起来和其他无用的机器没什么两样，但是里面发生的事情可能会让你大吃一惊。
