# 机器人:会画食物

> 原文：<https://hackaday.com/2022/09/14/robot-will-draw-for-food/>

生物系统经常想出最好的方法来获取它们生存所需要的东西。现在，伍斯特理工学院、伦敦帝国理工学院和伊利诺伊大学香槟分校的研究人员创造的机器人可以做出同样的声明。机器人在一个板的前面操作，板的一端有电端子，在这些端子和机器人之间有各种障碍物。

机器人可以拿起并重新排列盘子上的一些物品，然后用导电墨水画出通向终端的路径。效果是，如果机器人解决了连接难题，它就可以“吃”东西了。

诚然，就其本身而言，这并不特别重要。显然，机器已经有动力，它并没有真正帮助它获得更多的动力。然而，在许多任务中，机器人艺术可能要执行一项挑战预先定义的任务。这是一个软件如何评估情况，然后创建并执行计划以获得预期结果的例子。

尤其令人印象深刻的是机器人能够移动坡道跨越障碍，然后越过坡道。这不会彻底改变拾放设置，但我们可以看到在 PCB 布线中的应用，这是一个类似的问题。
