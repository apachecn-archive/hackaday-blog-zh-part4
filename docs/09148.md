# 手动拾取和放置转盘使电路板组装更容易

> 原文：<https://hackaday.com/2021/02/03/manual-pick-and-place-turntable-makes-board-assembly-easier/>

表面贴装器件曾经被认为是电子爱好者的巨大负担。微小，难以手工焊接，甚至几乎没有标签，许多人想知道这种消遣是否会完全碰壁。相反，有事业心的黑客和制造者开始学习新的技巧和技术来使用这项技术，从那以后我们再也没有回头。[Seon]就是这样的一个爱好者，[已经建造了一个有用的转盘，使手动拾取和放置电路板变得更加容易。](https://www.youtube.com/watch?v=scHSrFdWn5k&feature=youtu.be)(视频，下面嵌入。)

这个设计是[Seon]随着时间的推移逐渐完善的，在最终准备好用版本 3 进行更广泛的公开发布之前，已经建立了两个转盘的初始版本。它由一个旋转的盒子组成，盒子上有径向槽，可以容纳所有微小的 SMD 零件，这些零件可以贴上标签，以便于识别。还有一个丙烯酸窗口，确保一次只能打开一部分盒子，以避免类似的小零件意外落入相邻的插槽中——这是对第一个设计的巨大改进。然后有一个更小的旋转中心垫，PCB 可以放在上面，准备接收零件。

Github 上有一些文件可供那些想要创建自己的文件的人使用。【Seon】很好地解释了最终设计是如何产生的，他在自己早期设计的基础上组装了数百块电路板，并了解了它们的局限性。如果手工操作不适合你，那么[你可以建立一个全自动的 PnP](https://hackaday.com/2020/05/11/open-source-pick-and-place-has-a-450-bom-cost/) 。
