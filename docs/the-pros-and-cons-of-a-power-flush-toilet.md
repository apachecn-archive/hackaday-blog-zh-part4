# 电动抽水马桶的利与弊

> 原文：<https://hackaday.com/2021/11/20/the-pros-and-cons-of-a-power-flush-toilet/>

总的来说，厕所的设计在很长一段时间内没有发生根本性的变化。这个问题被认为已经解决，社会继续前进。然而，一个朋友让[Handy Geng]建造一个具有超强冲水功能的马桶，所以他就去做了。

概念很简单；当需要更强有力的冲洗时，定制厕所中的水箱可以通过压缩机充入压缩空气。然后，当拉下冲水手柄时，水就会以比平时更猛烈的速度冲进马桶。

[Handy Geng]还指出了该系统的另一个好处。通过拧紧马桶盖，使用垫圈将其密封到马桶上，气压系统可以用来清除堵塞物和木塞。

然而，也有一些缺点。马桶几乎完全由钢制成，这种材料通常不是处理水或体液的首选材料。此外，由于喷射的强度，动力冲洗模式往往以水从马桶中喷出而告终。

不管怎样，这表明厕所设计仍然是一个活跃的研究领域，我们不必满足于我们今天拥有的厕所。如果我们需要，更强大、更有能力的型号就在眼前。休息后的视频。
