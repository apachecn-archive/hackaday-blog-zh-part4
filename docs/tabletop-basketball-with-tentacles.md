# 带触须的桌面篮球

> 原文：<https://hackaday.com/2021/07/28/tabletop-basketball-with-tentacles/>

不像足球和桌上足球，篮球不太容易变成桌面游戏。[The Q]已经找到了一个解决办法，[利用触手机制创造了一个双人篮球游戏](https://www.youtube.com/watch?v=b4lIOIMFp6U)。

每个玩家使用一对双轴控制杆和一个脚踏板来操作末端带有抓手的电缆驱动的触手。这是两个阶段的触角，意味着上半部分和下半部分是独立控制的。触须由一系列层压泡沫圆盘组成，嵌在自行车电缆套管上。电缆在其控制的部分是开放的，并以推拉方式运行。弹簧加载的抓取器由脚踏板操作，一根电缆沿着触手的中心延伸。

这个游戏看起来很有趣也很有挑战性，我们可以想象如果两三个人一组操作每个触手会更加有趣。给成年玩家加一点酒精，它可能会变得非常滑稽，尽管这种机制需要加强一点才能经受住这种程度的惩罚。

我们怀疑[Q]读过[约书亚·瓦斯奎兹]关于两阶段触手机制的令人难以置信的详细的三部分指南。结合他的[电缆机制数学指南](https://hackaday.com/2021/01/26/cable-mechanism-maths-designing-against-the-capstan-equation/)，你就可以很好地构建自己的了。
