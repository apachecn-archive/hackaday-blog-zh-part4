# Python 你的键盘黑客与 Adafruit 聊天

> 原文：<https://hackaday.com/2021/07/19/python-your-keyboard-hack-chat-with-adafruit/>

7 月 21 日星期三中午，太平洋时间，加入我们与 Adafruit 工作人员的[Python Your Keyboard Hack Chat](https://hackaday.io/event/180454-python-your-keyboard-hack-chat-with-adafruit)

尤其是在过去的一年半时间里，我们大多数人都觉得我们的电脑和我们自己之间几乎没有什么区别。我们似乎焊接在一起，甚至不可分割，因为我们日夜与我们的机器相连，工作生活和家庭生活融合成一个灰色的，毫无特色的景观，时间流逝，除了我们工作区周围堆积的食物包装纸和饮料罐。或者只是看起来是这样。

不管怎样，机器和操作者之间实际上只有一线之隔，而且在大多数情况下，我们都讨厌机电配件:键盘。如果你从货架上购买，它永远不会完全正确——太响，不够响，错误的间距，糟糕的人体工程学，或者只是简单的丑陋设计。绕过这些限制的唯一真正的方法是加入 DIY 键盘人群，推出自己的键盘，专门为你的手指和你的需求定制——至少直到你意识到它并不完美，需要再次修改它。

达到这个移动目标通常既是硬件问题也是软件问题，但是现在越来越多的情况是，Python 已经准备好提供帮助了。为了深入了解 Python 如何用于定制键盘生成器，我们在 Adafruit 的好朋友，包括[Limor“lady ada”Fried](https://hackaday.io/hacker/9615-ladyada)、 [Phillip Torrone](https://hackaday.io/hacker/254-pt) 、 [Dan Halbert](https://hackaday.io/dhalbert) 、 [Kattni Rembor](https://hackaday.io/hacker/277997-kattni) 和 [Scott Shawcroft](https://hackaday.io/hacker/194126-tannewt) 将会在 Hack Chat 上停下来。我们猜想他们除了分享制作合适的 DIY 键盘的技巧和诀窍之外，还会有一些很酷的东西可以炫耀。如果你正在建立自定义 keebs，或者即使你只是“键盘好奇”，你不会想错过这一个。

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/messages/room/2369) 我们的黑客聊天是 [Hackaday.io 黑客聊天群发消息](https://hackaday.io/messages/room/2369)中的社区直播活动。本周，我们将在太平洋时间 7 月 21 日星期三中午 12:00 坐下来讨论。如果时区束缚了你，我们有一个[方便的时区转换器](https://www.timeanddate.com/countdown/generic?iso=20210721T12&p0=224&msg=Python+Your+Keyboard+Hack+Chat&font=cursive)。
