# 这套人行横道标志服装在万圣节大受欢迎

> 原文：<https://hackaday.com/2022/11/11/this-crosswalk-sign-costume-is-a-halloween-hit/>

万圣节服装可以是任何东西，从昂贵的角色服装的真实再现，到从拥挤的弹出式商店中购买的廉价仿制品。或者，你可以对事物有创造性和概念性，比如给自己做一套人行横道标志服装。

[jared531]的创作，构建使用了一大块涂成黑色的纸板作为基础。它位于佩戴者身后，并被赋予黄色轮廓，以模仿美国常见的人行横道信号。然后，红色的童话灯被布置在纸板的一半上，模仿“停止”的手势。

然后，穿着者应该穿上全黑的衣服，用松紧带把平板纸板系在自己身上。黑色网状面部覆盖物有助于融入佩戴者的面部，使外观更加完整。然后，他们在身体周围装上白色的仙女灯，模仿“行走”信号。

这是一个简单的概念，但相当准确地复制了一个典型的人行横道标志。这是我们以前从未见过的东西，在这个互联网时代，任何新事物在几分钟后就会过时，这一点令人印象深刻。
