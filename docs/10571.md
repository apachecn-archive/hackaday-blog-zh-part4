# 机器挤压机细丝

> 原文:[https://hackaday.com/2021/07/04/machine-extrudes-filament/](https://hackaday.com/2021/07/04/machine-extrudes-filament/)

我们已经见过很多自制细丝挤出机，但是 CNC Kitchen 的[Stefan]在他的最新视频中展示了一个商用桌面细丝挤出机，你可以在下面看到。3DEVO 挤出机相当光滑，但在 7000-8000 美元左右，我们可能不会急于购买一台。不过，我们可能会从中获得一些想法，以便下一次尝试构建类似的东西。

从概念上讲，任何制造细丝的机器都非常简单。熔化颗粒并将其从喷嘴中推出。冷却灯丝并缠绕。很简单，对吧？但是，当然，问题都在细节上。举例来说，离模膨胀意味着你不能仅仅假设喷嘴的孔径大小会给你合适的长丝尺寸。

3DEVO 机器显然实际上监控细丝的直径并关闭控制回路，改变参数以保持细丝直径合理地恒定并符合目标。看起来它也做得很好。

[Stefan]正在努力回收旧照片，我们期待看到那个视频。我们已经看到相当多的这些[细丝挤压机](https://hackaday.com/2018/06/02/print-your-own-filament/)取得了不同程度的成功。我们也看到了一些关于[如何不建造一个](https://hackaday.com/2021/02/28/fail-of-the-week-how-not-to-build-a-filament-extruder/)的教训。
