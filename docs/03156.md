# 3D 打印磨刀工具让工作变得简单

> 原文：<https://hackaday.com/2019/06/02/3d-printed-knife-sharpening-tool-makes-the-job-easy/>

使用一把锋利的刀是一种享受，但我们中的许多人都为购买最便宜的厨房工具而感到内疚，并且很少维护它们。保持刀具锋利是安全有效地使用刀具的关键，但是手工磨刀需要耐心和技巧。[CNC Kitchen]相反决定使用技术来解决这个问题，[设计一个 3D 打印工具来简化工作(Youtube 视频，嵌入下文)](https://www.youtube.com/watch?v=IM2il-zT2fA)。

磨刀器是一个简单的构建，需要一些简单的 3D 打印零件结合一些螺母，螺栓和铝棒。它被设计成使用普通大小的磨刀石，这使得采购变得容易。多年来，这种设计一直在不断完善,[CNC Kitchen]增加了磁铁袋和球形轴承，减少了运动中的倾斜。

【CNC Kitchen】报道说，这种工具非常好用，即使是新手也能很好地磨刀。那些想打印自己的零件的人可以在 Thingiverse 上找到。然而，如果你坚持用老方法做事，[你可以找一个电子教练来帮助你提高技术](https://hackaday.com/2018/03/31/sharpening-with-bluetooth/)。休息后的视频。
