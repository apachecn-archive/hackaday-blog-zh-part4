# 这个布林肯电网都是模拟的

> 原文:[https://hack aday . com/2019/02/01/this-blinken-grid-is-all-analog/](https://hackaday.com/2019/02/01/this-blinken-grid-is-all-analog/)

今天的个人电脑由于使用了不起眼的 LED，所以很经济。一台现代笔记本电脑可能有一个电源指示灯，如果你幸运的话，还有一个硬盘指示灯。60 年代和 70 年代的大型机恪守布林肯的神圣教义，用闪烁的灯光向昔日的熟练操作员显示机器的活动。[【麦森】想要重新创造这种美学，并以完全模拟的方式进行。](https://hackaday.io/project/163524-analog-blinken-grid)

该项目围绕一个 8×8 LED 网格构建，使用 3D 打印夹具焊接，以确保尺寸精度。安装在每一列上的是一个 PNP 触发器，它将该列拉至 VCC，而每一行有一个 NPN 触发器，它将该列拉至地。由于组件值和容差的差异，振荡器都不同步，导致非常令人愉快的闪光灯效果。

我们是原始美学的忠实粉丝，但[Matseng]也在网格上安装了扩散器，更清楚地代表了老式计算机的美学。我们是 blinken 的忠实粉丝，[比如休息后播放的 PDP-8/I.](https://hackaday.com/2016/03/23/blinkenbone-meets-the-pidp8/) 视频。
