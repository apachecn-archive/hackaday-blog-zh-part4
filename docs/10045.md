# 你真的不需要或不想要的键盘

> 原文:[https://hackaday . com/2021/05/06/the-keyboard-you-really-dont-need-or-want/](https://hackaday.com/2021/05/06/the-keyboard-you-really-dont-need-or-want/)

大多数人认为键盘是一个扁平的、模模糊糊的长方形东西，有大约 100 个不同的键。一个机械键盘爱好者会强烈反对，并指出各种 tenkeyless、75%、60%或 40%的键盘，它们通过去掉键来消除键盘是什么的概念。[Stavros Korokithakis]采用了这一概念，并通过[创造了被称为 keyyyyyyys](https://www.stavros.io/posts/keyyyyyyyys/)的五按钮垂直键盘。

这种键盘，或称 keystick，被设计成单手操作，不用眼睛接触。只有五个键，它大量使用和弦输出所有需要的字符。它最多有 32 种可能的状态，如果不按任何键，就有 31 种可能的组合键。所以[斯塔夫罗斯]必须发挥创造力，根据字母在英语中的出现频率来排列它们。Keyyyyyyyys 的大脑是无处不在的 ESP32，模拟蓝牙键盘，同时被包裹在一个简单的 3d 打印盒中。[代码托管在 GitLab](https://gitlab.com/stavros/keyyyyyyyys/) 上。

如果你不知道从头开始学习一个五键和弦键盘有多难，一定要看看下面嵌入的[Stavros]视频。“来吧 h。”我们已经听说你可以学习这些东西，虽然。

虽然这个五键键盘看起来很小，但这个两键键盘仍然比三键键盘强。一键键盘只是莫尔斯码键盘，我们期待的是无线蓝牙版本。
