# (紫外线)写在墙上

> 原文：<https://hackaday.com/2018/11/18/the-uv-writings-on-the-wall/>

[迈克尔·卡林]的伯沙撒，以圣经人物的名字命名，在他的墙上出现了文字，是一台独特的光绘机，它在黑暗的背景中跟踪一组紫外线发光二极管，在光线中绘制短暂的点阵字母。这是今年夏天 2018 年电磁场黑客营零区众多[赛博朋克主题艺术作品](https://hackaday.com/2018/09/09/electromagnetic-field-a-cyberpunk-headdress-to-be-noticed-in/)之一。

这排发光二极管悬挂在穿过管状轨道的支架上，并通过驱动辊子的步进电机向前移动。这种布置带来的好处是，它可以针对任何长度的显示器进行缩放。led 由 Arduino 通过德州仪器 TLC5940 PWM 驱动器驱动。结果可以在休息下面的视频中看到，那些在 [EMF](https://hackaday.com/2018/10/05/electromagnetic-field-2018-event-review/) 看到它的人可能会记得它描绘了适当的反乌托邦短语。
