# 一个实际检测辐射的 PKE 测量仪

> 原文:[https://hack aday . com/2019/07/24/a-pke-meter-that-actually-detected-radiation/](https://hackaday.com/2019/07/24/a-pke-meter-that-actually-detects-radiation/)

《捉鬼敢死队》的粉丝们会记得 PKE 测量仪，这是一种有翼的手持设备，能够探测超自然活动。设备上留下的宝贵技术数据很少，让我们无法复制它的功能。然而，闪烁、展开的翅膀是危险的强烈视觉指示器，[和【莫斯弗斯】认为这将是盖革计数器建造的完美选择。](https://www.instructables.com/id/PKE-Meter-Geiger-Counter/)

SBM20 盖革管作为检测设备，连接到 Arduino Nano。有机发光二极管显示器用于向用户显示数字数据。外壳和折叠翅膀是 3D 打印的，并按照最初的电影道具安装了 80 年代风格的黄色 led。

该设备的使用非常直观——如果翅膀张开，灯光闪烁更快，你会检测到辐射水平增加。在一个非常真实的意义上，它使得使用盖革计数器对没有经验或听力受损的人来说更加简单。当然，也有一个蜂鸣器产生你所期望的不祥的滴答声。

盖革计数器是一个受欢迎的项目，尽管我们希望它们在不久的将来不会成为普通的家庭用品。[这里有一个*辐射-* 给游戏粉丝的启发](https://hackaday.com/2019/05/18/fallout-inspired-display-is-ready-for-the-apocalypse/)。休息后的视频。
