# 建造一个动态沙盘

> 原文:[https://hack aday . com/2021/10/31/building-a-kinetic-sand-art-table/](https://hackaday.com/2021/10/31/building-a-kinetic-sand-art-table/)

我们中的许多人都惊叹于艺术装置，这些装置的特点是大理石静静地、不停地在沙子上描绘出美丽的图案。[DIY Machines]在这里向我们展示完全有可能在家里自己制作一台！

基本机制很简单。该桌子使用笛卡尔运动平台来移动桌子下面的磁铁。在桌子的顶部，一个金属球附着在磁铁上，通过手工沙子移动来画出吸引人的图案。Arduino 和 Raspberry Pi 一起工作，命令步进电机在沙子上创建各种图案。

低成本的松木被用来建造桌子的大部分，橡木被用来建造吸引人的光秃秃的木制桌面。RGB LEDs 围绕在沙滩表面，以照亮场景，有疯狂迪斯科灯光或简单白光的选项，以获得更微妙的外观。其他漂亮的触摸包括坐在人造皮革层顶上的工艺沙，这样球在沙中移动时不会发出令人讨厌的嘎吱嘎吱的声音。

这是一个很好的构建，它专注于像噪音这样的较小的细节，这些细节可以对最终的体验产生很大的影响。我们以前也见过类似的项目。休息后的视频。
