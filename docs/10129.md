# 核反应堆变小了

> 原文：<https://hackaday.com/2021/05/13/nuclear-reactors-get-small/>

当史蒂夫·马丁告诉我们“让我们变小”时，他走在了时代的前面虽然你通常认为核反应堆是一件大事，但现在有一种新的趋势，即在需要的地方制造小型微型反应堆来发电，而不是大型集中发电设施。美国能源部有一个关于这个话题的视频，你可以在下面观看。

你可能在科学课上学过基本的核裂变反应堆是如何工作的。核燃料通过裂变产生热量，而像水这样的缓和剂通过冷却反应堆和减慢中子来防止它熔化。控制棒会进一步减缓反应，或者——如果你拔出来——会加速反应。热量产生蒸汽(直接或间接),蒸汽驱动传统的发电机，这种发电机已经不再是什么高科技了。

小型反应堆的一个主要优点是可以运输。这意味着您可以在一个高效的中心位置构建它们，并将其移动到您需要的地方。一般来说，这些新反应堆有被动安全系统，自动控制系统，可以在没有新燃料的情况下运行十年。虽然有几种技术正在开发中，但能源部表示，最早的微型反应器将使用气体或热管冷却。液态金属和熔盐系统也很有前景，但可能会晚一些。

当然，小是相对而言的。这些反应堆产生 1 到 20 兆瓦的电力，看起来可以装在一辆大卡车上。我们不指望在不久的将来会有核能笔记本电脑。

也许这些新反应堆将受益于[增材制造](https://hackaday.com/2020/06/08/3d-printing-nuclear-reactors-for-fun-and-profit/)。当然，潜艇和海军水面舰艇长期以来一直拥有微小而可靠的反应堆。移动式反应堆的一个明显应用是[为运输工具](https://hackaday.com/2020/11/19/the-shipping-industrys-transition-to-atomic-power-and-faster-deliveries/)提供动力。
