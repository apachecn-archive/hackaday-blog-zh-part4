# SLA 印刷的弹簧针编程夹具

> 原文：<https://hackaday.com/2018/09/30/an-sla-printed-pogo-pin-programming-jig/>

如果你有一个微控制器来编程，它可以是一个足够简单的过程来连接一个串行引线并执行任务。然而，如果你在印刷电路板上有数百个微控制器要编程，那么多次连接那条引线就变得不可能了。在制造环境中，他们有弹簧针夹具，这是一组载有编程信号的弹簧针，与置于其顶部的 PCB 上的适当焊盘完美对齐。

[Conor Patrick]正在升级 U2F Zero 双因素身份认证令牌，他面临的问题正是需要对大量电路板进行编程。[他的弹簧针夹具](https://www.conorpp.com/3d-printing-a-programming-jig-and-embedding-pogo-pins-using-eagle-and-fusion-360)做得非常好，他带我们了解了他的设计和制造过程。

从他在 Eagle 的 PCB 设计开始，他将其导出到 Fusion 360，在 Fusion 360 中，他能够创建一个夹具来适应它。在用 SLA 3D 打印机打印之前，他在夹具模型的适当位置放置了他选择的弹簧针的孔。他特别补充了引脚本身，一个来自 mill-Max 的焊料桶设计，[来自 DigiKey](https://www.digikey.com/product-detail/en/mill-max-manufacturing-corp/0947-0-15-20-77-14-11-0/ED11275-ND/7402798) 。

布丁的证明是在吃，令人高兴的是，当他完成的夹具收到它的第一块板时，一切都按计划进行，编程进行得天衣无缝。我们已经向您展示了[其他弹簧针夹具](https://hackaday.com/2017/11/19/homemade-test-jig-is-cheaper-than-outsourcing/)，但这一个执行得特别好。
