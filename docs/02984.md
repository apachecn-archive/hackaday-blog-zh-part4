# 用放电加工制作印刷电路板原型

> 原文:[https://hack aday . com/2019/05/13/prototyping-PCB-with-electrical-discharge-machining/](https://hackaday.com/2019/05/13/prototyping-pcbs-with-electrical-discharge-machining/)

在 Hackaday，我们认为我们已经看到了制造 PCB 的每一种方法:CNC 加工，用各种化学物质进行掩蔽和蚀刻，激光雕刻，甚至是用美工刀刮去铜的粗糙但有效的方法。不管怎样，对我们来说都很好，真的，但在 DIY PCB 领域似乎仍有改进的空间。为了 whit，我们介绍了[利用放电加工的快速 PCB 原型制作](https://hackaday.io/project/165466-pcb-fabrication-with-edm)。

使用电弧来选择性地烧蚀 PCB 上的铜覆层似乎是一个好主意。至少，当[杰克·瓦奇林]意识到使用九伏电池和铅笔芯切割一张铝箔的老技巧实际上只是一种 EDM 形式，并且 PCB 上的铜层与铝箔没有一百万英里的区别时，他是这样认为的。使用工作台电源和自动铅笔芯进行的一些实验表明，从空白板上炸开铜相对容易，因此[杰克]采取了下一个合乎逻辑的步骤，组装了一台旧的 3D 打印机来移动工具。下面的视频显示了设置和一些早期测试；它远非完美，但很有希望。如果他能更好地控制电弧，这种自制 EDM 看起来可以非常快速地生产原型板。

[Jake]发布了这个项目的当前状态，希望能激发讨论和进一步的实验。这是值得称赞的，我们真的很乐意看到这一个快速前进。你可能会通过查看[这个有点粗略的电源供电 EDM](https://hackaday.com/2017/04/04/edm-for-the-cheap-and-adventurous/) 来开始你的头脑风暴，或者在[更详细地查看整个领域](https://hackaday.com/2016/08/07/homemade-edm-can-cut-through-difficult-materials-like-magnets-with-eease/)。
