# 廉价放大

> 原文：<https://hackaday.com/2020/11/17/magnifying-on-the-cheap/>

如果说我们在几年的 Hackaday SMD 焊接挑战赛中学到了什么的话，那就是:大多数人需要放大才能在很小的范围内完成良好的焊接。问题是，就像大多数工具一样，你可以买一个 5 美元的双筒耳机，也可以花 1000 美元或更多买一个严肃的显微镜。中间是什么？[Noel]在最近的视频中介绍了一些[实惠的选择](https://www.youtube.com/watch?v=PnNOckCcn2U)，你可以在下面看到。

[Noel]从一个廉价的“援助之手”开始，它有一个简单的小放大镜。主要标准是找到没有延迟的东西，这样他就可以在放大的情况下焊接。虽然在显示有一点滞后的情况下工作是可能的，但这是令人沮丧的，而且还有更好的选择。

第一次尝试是一套 10 欧元的放大镜，有不同强度的镜片，可以互换不同的放大倍数。当然，每个镜头都有不同的焦距，你不会想离你的热烙铁一英寸远。

下一个竞争者是 26 欧元的液晶显微镜。我们从经验中知道，对于这些廉价的望远镜来说，支架就是一切。幸运的是，[诺尔]有一个相当稳定的立场。他提到示波器不是 USB。

我们喜欢翻转式双筒望远镜，有几个镜头堆叠在一起，尽管视频中的镜头可能是可行的。我们也喜欢我们的廉价显微镜，尽管你应该知道如果你使用任何附加功能，它们可能会变得很脆弱。此外，按下按钮可能会使你在高倍放大下偏离目标。不过，这个价格你想要多少？

有些人更喜欢用立体显微镜焊接深度感知。如果你想解决廉价显微镜的一些局限性，【埃利奥特·威廉姆斯】[黑了一个稍微好一点的](https://hackaday.com/2017/06/26/hacking-an-inspection-microscope/)。
