# 将隔离的忧郁带进自制的麦克风

> 原文:[https://hack aday . com/2020/04/12/belt-the-quarantine-blues-into-a-自制-mic/](https://hackaday.com/2020/04/12/belt-the-quarantine-blues-into-a-homemade-mic/)

如果说隔离有什么心理上的好处，那就是人们利用这段时间完成旧项目，同时开始大量新项目。如果你已经没有主意了，或者只是想通过做一些室内回收来感觉更好，[把那个垃圾桶倒出来，做一个简单的麦克风](http://www.instructables.com/id/0-Microphone/)。

[![](../Images/5cb284902bc7c9bbf7c0030b66d8de27.png)](https://hackaday.com/wp-content/uploads/2020/04/recyled-mic-guts.png) 你所需要的只是一些 PETE #1 塑料、一个压电盘和合适的锡罐。塑料被热熔到罐子的卷边上，由于它在这个过程中被拉伸和收缩，它形成了一层紧密的薄膜，可以兼作打击乐器。

你对着另一端大喊，你的声波震动了薄膜。压电传感器拾取振动，并将其发送到 1/4”插孔，这样您就可以将其插入放大器。

即使你在某种程度上回避了忧郁，你也可以用这个来对那些威胁要靠近你的人大喊大叫。这个有趣的项目是尽可能开放的，但我们相信你可以想出使用压电盘的方法。在你看完[火腿制作]的音乐视频后，请在评论中告诉我们。

我们喜欢[火腿制造]的方法，在不切到自己的情况下切断果汁罐。只要把刀片夹在虎钳里，然后把水壶顶在上面。让我们想起了另一种方法，将塑料汽水瓶做成绳子，轻松重复使用。
