# 六十年代的福根证明了教堂的风琴绝对是 Chiptune

> 原文:[https://hack aday . com/2021/03/29/the-sixtyforgan-provides-the-church-organs-are-determinely-chip tune/](https://hackaday.com/2021/03/29/the-sixtyforgan-proves-that-church-organs-are-definitely-chiptune/)

教堂风琴可能在机械上很复杂，而且超级老派，但它们与最早的计算机声音芯片有一些共同之处。在理论上，以及在实践中，它们产生非常简单的波形。与你的老准将 64 相比，教堂风琴看起来如此饱满和丰富的主要原因是，它们具有整个教堂混响的优势，可以增强声音。[[Linus]用 Sixtyforgan 演示了这一点。](http://www.linusakesson.net/sixtyforgan/index.php)

Sixtyforgan 是一个连接到弹簧混响箱的 Commodore 64。通过这个混响器运行 Commodore 的 SID 芯片中相对基本的波形，就有可能产生与你在当地周日礼拜中听到的声音极其相似的声音。虽然我们不会指望像[【chip zel】这样的 chiptune 名人会在 Square Sounds](https://youtu.be/ZS8kn6iS3tE?t=2034) 这样的活动中开始演唱赞美之歌，但想想古代的作曲家在很久以前对着一些疯狂的 Game Boy jams 摇滚也是很棒的。

这很好地展示了准将的音乐能力，我们特别喜欢借鉴手风琴的半音阶按钮布局的应用。我们希望看到这种设置与复古电脑乐队相结合，就像这个演奏糖梅仙子的演示一样。或者，六十年代的比利·考根对阵提比略也会很棒。休息后的视频听起来很不错。
