# 紫外线发光时钟发光显示时间

> 原文：<https://hackaday.com/2019/01/07/uv-glow-clock-tells-the-time-glowingly/>

Reddit 用户[TuckerPi]想做点什么来感谢他的父亲帮助他完成工程学位。他用这个超棒的发光时钟一炮而红。这个时钟使用一条紫外发光带，由一个小型步进电机带动旋转。在一侧，一个紫外发光二极管由第二个电机上下移动，使胶带在其下方发光。一个树莓派驱动整个系统，将时间写在磁带上，并旋转它使其朝外。时钟每分钟重写一次橡皮上的时间。

这是一个可爱的建筑，展示了[塔克皮]在大学里学到的东西，因为他自己建造了大部分机制，切割自己的金属齿轮和零件，并用非洲桃花心木制作了一个漂亮、简单的外壳。他也展示了自己的错误，比如他第一次尝试用混合了紫外线粉末的硅橡胶制造发光装置。虽然它最初是有效的，但他发现紫外线粉末在一段时间后会从橡胶中脱落，所以他用紫外线发光胶带代替了它。

[塔克皮]还没有公布该设备的完整原理图，但在 Imgur 的照片和他讨论版本的 Reddit 帖子中有很多细节。他找到了一种有趣而独特的方式来感谢父亲的帮助，这值得称赞。
