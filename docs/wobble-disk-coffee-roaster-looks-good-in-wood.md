# 摆盘咖啡烘烤器在木头上看起来不错

> 原文：<https://hackaday.com/2021/07/13/wobble-disk-coffee-roaster-looks-good-in-wood/>

如果你喜欢咖啡，你可能大部分时间都在家里自己做，用的是一些时髦咖啡店的咖啡豆，那里的员工都有满袖纹身和强烈的观点。也许你甚至会买整颗的咖啡豆，在使用前磨碎。如果你想一直走下去，你得自己烤豆子。有各种各样的方法可以做到这一点，比如改变热空气爆米花机的用途。[如果你是【拉里·科顿】，你会买一大堆青豆，然后不停地制作摆盘烘焙机，直到你实现了 DIY 的完美](https://library.sweetmarias.com/video-ez-roaster-by-larry-cotton/)。

[![](img/f332e63ac6b801f3a009ce71d327ba2b.png)](https://hackaday.com/wp-content/uploads/2021/07/Larry-Cotton-EZ-Roaster-Wobble-Disk-Coffee-Roaster-YouTube-1-12.jpeg) 【拉里】的最新烘焙炉拥有全木质结构，结构部分没有金属支架或外壳。这很好，因为你不太可能在任何东西上烧伤自己，而且你没有从豆子上散热。除了筛子、豆子和固定热风枪喷嘴的硬三角铁丝之外，任何东西都不应该变热。一旦烘焙周期完成，[拉里]就把咖啡豆抖到风扇上方的相邻屏幕上，让它们冷却下来。

与一些[【拉里】以前的设计](https://hackaday.com/2021/03/14/how-to-cobble-a-wobble-disk-roaster-together/)不同，这个设计使用了一个位于热风枪上方的 8 杯面粉筛。一个电池驱动的螺丝刀驱动摇摆盘搅动咖啡豆，帮助它们均匀烘烤，一个木制的手臂按住电源按钮。我们喜欢这种机器的简单性，并认为摆盘烘烤器看起来很迷人。休息过后，请观看视频，了解如何制作自己的视频。

烤豆子的方法不止一种，其中一种甚至得到了 Hackaday 编辑【埃利奥特·威廉姆斯】的官方认可。
