# 八十英尺弦射手的攻击

> 原文：<https://hackaday.com/2022/01/01/attack-of-the-eighty-foot-string-shooter/>

弦射手是令人兴奋的，因为他们以一种奇特的方式遵守物理定律，这让我们不禁要问，“怎么做到的？”以及“为什么？”经过一点戳和刺激，也许一些轻绳燃烧，我们可能有一些想法，我们如何做自己的。[Nick Belsten]和[Joey Rain]看到一些桌面模型，然后想，“让我们把那只小狗做 80 英尺长吧！"视频也在中断后嵌入。

他们选择了洗衣机马达和自行车轮胎，然后插上延长线，而不是业余爱好马达、手电筒电池和玩具车轮。三分钟的视频不是一个如何构建的视频，因为一旦你开始将这种硬件焊接在一起，你就已经在凭自己的感觉飞行了。你会看到一个前院，人们对在树梢上连续不断地投放绳子的荒谬行为感到高兴。有足够的空间来观察沿着绳索行进的[波或者匆忙地打磨你的指甲。](https://hackaday.com/2017/04/02/musical-string-shooter-makes-sound-visible/)

我们想为办公室制作[串串，并加入我们的个人风格，比如灯或彩色串串，这样触摸起来就很安全。如果你对任何物理实验有独特的想法，](https://hackaday.com/2019/05/24/great-hacks-at-our-maker-faire-bay-area-meetup-from-helmets-and-goggles-to-rovers-and-string/)[给我们写封短信](https://hackaday.com/submit-a-tip/)，但是为了保险起见，我们会补充一点，你不应该制作没有导杆的电锯，也就是被禁止的*链刀*。
