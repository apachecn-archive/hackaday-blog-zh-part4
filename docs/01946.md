# 用 Script-8 构建复古游戏

> 原文:[https://hack aday . com/2019/01/30/build-retro-games-with-script-8/](https://hackaday.com/2019/01/30/build-retro-games-with-script-8/)

整整一代程序员学会了通过编写——或者至少是输入——游戏程序来为相对简单的计算机编程，比如 TRS-80、Commodore 64 或者其他类似的机器。如今，游戏变得越来越复杂，电脑也是如此。当然，玩 Skyrim 比玩 Snake 更有趣，但是对于学习来说，你可能会从一个简单的游戏中获得更多。如果你今天想学习编程，或者想让其他人也开始同样的旅程，你应该看看[Gabriel Florit]的项目 [Script-8](https://script-8.github.io/) 。你可以在下面的视频中体验一下它的样子，或者直接登陆网站玩或者修改 [a 游戏](https://script-8.github.io/?id=1c4ae6ff3727cb4a0e344e3435aa16b2)(提示:按“a”发射球)。

以下是来自网站的精彩的电梯演讲，而不是转述:

> SCRIPT-8 是一台用于制作、分享和玩微型复古游戏(称为卡带)的梦幻电脑。它是免费的、基于浏览器的、开源的。卡带是用 JavaScript 写的。

在某些情况下，Script-8 可能比旧机器更好。你做的每一件事都会引起立即的反应。有各种各样的工具可以让你暂停和回放动作，精灵编辑器，地图编辑器，甚至音乐编辑器。

很容易将现有的“磁带”复制到空白，这样您就可以进行更改，因此您不必从头开始，即使您可以这样做。你的工作空间是 128 像素的正方形，并且只显示 8 种颜色，但是这增加了游戏的复古外观和感觉。

老实说，JavaScript 比旧的 BASIC 要难一点，但在现代世界中它也更加实用。如果你真的想要基本的，你也可以在[网站](https://hackaday.com/2018/09/17/its-the-web-basically/)上做。或者你可以重温旧日时光，体验一下现代版的 [QuickBasic](https://hackaday.com/2018/02/22/quickbasic-lives-on-with-qb64/) 。
