# CNC 将单个 PCB 变成折纸半球

> 原文:[https://hack aday . com/2019/01/13/CNC-turns-a-single-PCB-into-origami-hemisphere/](https://hackaday.com/2019/01/13/cnc-turns-a-single-pcb-into-origami-hemisphere/)

试图在 PCB 上做出一个半球形表面并不容易。试图做到这一点，并使结果成为一个工作电路甚至更难。用一块坚固的 FR4 做这件事似乎不可能，对吗？

没有那么多。[brainsmoke]想出了一个聪明的方法来制造可折叠的、可工作的、可以形成半球形的印刷电路板。这个项目的灵感来自一个更大的项目，产生了一个直径 32 厘米的 LED 镶嵌球体，一个朋友认为如果按比例缩小，它可以成为一条漂亮的项链。那个更大的球体有点像 PCB 足球，各个面板焊接在一起。[brainsmoke]不喜欢杂耍几十个小 PCB 来制作一个项链大小的版本，所以半个[deltoidal hexenotahedron](https://en.wikipedia.org/wiki/Deltoidal_hexecontahedron)的展开图案在单面 FR4 上显示为一个整体。然后在 CNC 铣床上切割蚀刻板，从板的后面将板之间的接头切割成 V 形槽。通过留下足够的材料作为活动铰链，[brainsmoke]能够将图案折叠成一个半球，同时保持痕迹完整。这个过程很繁琐，导致了很多 FR4 断裂和痕迹，但随着实践和使用更厚的板材和更重的铜，半球走到了一起。下面的视频展示了最终产品

这件*艺术品*是【brainsmoke】参加[巡回雕塑大赛](https://hackaday.io/contest/162559-circuit-sculpture-contest)的作品，~~上周刚刚结束~~。我们迫不及待地想分享一些人们在这次比赛中想到的很酷的东西，这似乎真的让创意的汁液流动起来。
