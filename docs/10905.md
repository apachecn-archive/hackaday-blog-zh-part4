# 定制卡尺履带，适合长途旅行

> 原文:[https://hack aday . com/2021/08/05/custom-caliper-tracks-for-when-you-going-the-distance/](https://hackaday.com/2021/08/05/custom-caliper-tracks-for-when-youre-going-the-distance/)

数字卡尺的工作原理是如此神秘，以至于我们从来没有想过要拆除它，更不用说改进它了，对吗？嗯，再想一想，当[李米 DIY]将加工元件改装到定制轨道上时，[将卡尺测量距离扩展到了惊人的 650 毫米](https://www.youtube.com/watch?v=JYnit_PSSMY)。结合之前提取测量数据的项目，结果有助于工作的多轴数字读数，这是手动车床或铣床等机床的便捷设备。

数字卡尺的工作原理是测量一组可变电容。如果我们挠头回头看看我们的[物理笔记](http://hyperphysics.phy-astr.gsu.edu/hbase/electric/pplate.html)，我们会回忆起两个平行导电板之间的电容与表面积成线性比例。通过固定两个板的一个维度，并将一个板滑动到另一个板上，我们有效地改变了面积，给我们自己一个简单的线性位移传感器！(也有一些经典的纠错技术，这个[[PDF](http://www.capsense.com/capsense-wp.pdf)是寻找更多细节的好地方。)

理论上，这种平行板阵列可以直接嵌入印刷电路板。我们只需要知道尺寸。经过一些细致的测量工作后，[李米 DIY]提取了关键的测量数据，并制作了一个复制图案超过 650 毫米的 PCB。在将原始处理元件改装到这个新轨道上后，他们有了一个比原来长得多的工作测量设备！

如果你曾经想拆卸你的卡钳，但又太紧张而不敢投资，现在你有机会跟随[Lima DIY]以视频形式为你演示免费拆卸过程。如果你想做同样的事情，他们的劳动成果也会在包含关键尺寸的[项目帖子](https://www.pcbway.com/project/shareproject/Digital_Caliper_Hack_Mod_new_2021.html)中被捕捉到。

如果你正在寻找其他方法来改善你的卡钳，为什么不从给它们[一个主要的电池寿命提升](https://hackaday.com/2017/02/11/the-most-sensational-calipers-on-the-planet/)开始呢。
