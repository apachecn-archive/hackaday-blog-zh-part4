# 摩托车模拟装置开始比赛了

> 原文：<https://hackaday.com/2021/09/04/motorcycle-simulation-rig-is-off-to-the-races/>

许多街机可以通过标准的操纵杆和按钮组合来模拟和轻松控制。然而，如果没有一些额外的设备，有些人会觉得不太对，摩托车比赛就是其中之一。因此，不再满足于去街机上找刺激，【The Q】[焊接了自己的摩托车模拟装置，在家里玩赛车游戏](https://www.youtube.com/watch?v=-fi9go91_co)。

初步设计草图出来后，矩形钢管被切割成合适的尺寸，然后用 MIG 焊机焊接在一起。连接到一些固定轴承的中心轴构成了中心支点。几个活塞提供了进入弯道所需的阻力。在中轴上，安装了一个座位和一个旧的自行车叉。车把和底座之间巧妙的连接使得自行车在转动车把时倾斜，反之亦然。

经过一番打磨，刷上橙色油漆，挂上牌照，装上一些灯和把手，这辆自行车已经准备好迎接黄金时间了。[The Q]只需要将自行车的角度放入他们选择的模拟中。虽然我们期望 teensy 或其他微控制器模拟控制器，但[Q]采用了更简单的方法，3D 打印了一个摇篮来容纳 PlayStation 控制器。小杠杆拉动细绳连接操纵杆，油门手柄上的电缆拉回控制器上的扳机。总而言之，这种体验看起来相当不错，尤其是当你把它和一台 motocross 街机相比的时候。它真正需要的是一些风扇为你吹来的气流效果。

如果你正在考虑淘汰米格来制造自己的装备，也许可以考虑制造一个[自制赛车装备来补充自行车](https://hackaday.com/2020/12/12/homebuilt-racing-sim-does-almost-everything-from-scratch/)。
