# 气泡字符沿着这个独特的滚动显示浮动

> 原文:[https://hack aday . com/2018/11/30/air-bubble-characters-float-along-this-unique-scrolling-display/](https://hackaday.com/2018/11/30/air-bubble-characters-float-along-this-unique-scrolling-display/)

我们在这些页面上看到了许多独特的大幅滚动留言板，但大多数都以新的和不寻常的方式使用了某种既定的技术——led、机电翻转点等。不过，我们很确定[这种气泡点阵显示器](https://www.instructables.com/id/Scrolling-Bubble-Display/)还是第一次。

虽然它可能不会出现在公共汽车或火车站的到站和离站公告栏前，但[jellmeister]的泡泡展示展示了一些非常有创意的想法。它始于多壁聚碳酸酯屋顶的碎片—[Corotherm](http://www.arielplastics.com/en-gb/products/multiwall-polycarbonate/multiwall-polycarbonate-sheet.aspx)是品牌名称——这种类型的玻璃温室和其他结构。平行管是完美的显示器，虽然个别管当然可以替代。制造塑料端盖；每个通道中的空气喷嘴通过电磁阀连接到空气源。一个带有几个马达驱动帽的 Arduino 允许空气脉冲进入每个通道，以创建漂浮在管道上的合理易读的字符。下面的视频展示了它在创客大会上的应用，参观者可以在这里泡泡出自己的信息。

这需要一些调整，让它看起来一样好，但有很大的改进空间。我们想知道彩色液体是否会有所帮助，或者也许在每个通道上添加一个新像素或者甚至一个激光来增加一些对比度。也许能让水面稍微模糊一点的东西会有帮助；用盐溶液增加表面张力可能会使气泡更明显。我们怀疑它是否会有翻转点显示屏的对比度，但它肯定有自己的魅力。
