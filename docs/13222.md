# 生活中的 Grogu 剃须刀顶部控制杆

> 原文：<https://hackaday.com/2022/04/02/razor-crest-control-lever-for-the-grogu-in-your-life/>

如果你想给你的游戏设置增加一点科幻真实性，你可以做得比[ZapWizard]参加 Hackaday.io 科幻竞赛的这个[功能控制杆复制品](https://hackaday.io/project/184433-functional-razor-crest-control-lever)更糟。

这个功能性道具从迪士尼的*曼达洛人*中获得灵感，几乎与电视节目中同名赏金猎人驾驶的*剃刀纹章*武装直升机驾驶台上的油门一模一样。这种构造的电子心脏相对简单——一个小 M0 测量超薄电位计的电阻，并伪装成一个典型的单轴 USB 节流阀。

机械部件和美观的外壳是这个项目真正的亮点。螺旋 3D 打印齿轮平滑了实心铝节气门轴的运动，一个简单的棘爪机构在中间点“抓住”节气门。镇流器和基板由不锈钢切割而成，赋予油门相当大的重量，有助于它在桌面上的稳定性(也可以使用螺钉或强力磁铁将其固定下来)。节气门外壳是 3D 打印的，并覆盖有铝箔胶带，然后经过化学发黑和老化处理，以获得备受喜爱的外观。

当然，这个建筑最具标志性的部分是球形旋钮，为了 Grogu 的方便，它被拧在铝轴上。[ZapWizard]在[定制 3D 材料](https://custom3dstuff.com/collections/frontpage/products/its-not-a-toy-razor-crest-ship-knob-replica)上订购一个，它绝对将整个建筑联系在一起。
