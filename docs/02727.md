# 将适合你的口袋的口袋模拟器

> 原文：<https://hackaday.com/2019/04/16/the-pocket-emulator-that-will-fit-in-your-pocket/>

如果芯片上的微型 Linux 系统有什么好处的话，那就是仿真。没有什么比在公交车上拿出一个仿真控制台来玩一个老式的 NES 俄罗斯方块游戏，或者在超级马里奥世界中击败下一个级别更好的了。这是有史以来最小的仿真控制台。它可以放在你的口袋里，它有一个明亮，充满活力的屏幕。没有比这更好的了。

这个项目是对两个项目的改进，这两个项目都是 hackaday.io 上的一些顶级项目，hack aday . io 是互联网上黑客和构建的最佳场所。Keymu 是(或曾经是)[有史以来最小的仿真控制台](https://hackaday.io/project/21453-keymu-open-source-keychain-sized-gaming-console)，是作为 3D 打印外壳中 Game Boy Advance SP 的缩小版而建造的，由英特尔 Edison 驱动。爱迪生已经不存在了，所以在那次开发之后[转移到了 Funkey Zero](https://hackaday.io/project/134065-funkey-zero) ，这是一个围绕 AllWinner V3s 芯片和 240×240 显示器构建的微型控制台。这两个都是很小很小的控制台，但随着硅变得更好，总会有更好的选择，所以它又回到了绘图板。

Funkey 项目的设计再次建立在具有 64MB DDR2 DRAM 的 AllWinner V3S SoC 上。有一个分辨率为 240×240 的 1.5 英寸显示屏，当然这款复古仿真控制台保留了著名的 Game Boy Advance SP 的经典和非常有用的翻盖外形。

这个项目已经在进行中，并将成为 hackaday.io 上有史以来最受欢迎的项目之一。每个人都想要一个仿真控制台，这是迄今为止最小和最小的一个。这个项目是否能进行到生产完全是另一回事，但我们渴望知道答案。
