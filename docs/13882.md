# 奇怪的输入和特殊的外围设备:最简单的 Pi 400 电脑平台

> 原文:[https://hack aday . com/2022/06/10/odd-inputs-and-special-peripherals-the-simple-of-pi-400-cyber decks/](https://hackaday.com/2022/06/10/odd-inputs-and-peculiar-peripherals-the-simplest-of-pi-400-cyberdecks/)

制作 cyberdecks 的趋势使得 Raspberry Pi 成为这些自制计算机工作站的最爱，一体式 Raspberry Pi 400 为集成计算机和键盘组件提供了一种特别方便的快捷方式。尽管如此，cyberdeck 底盘和屏幕仍然存在问题，这是[ bobricius ]以可能最简单的方式回答了[的一个问题，通过扩展端口的一个提升 PCB 来容纳 320×240 SPI 显示器。](https://hackaday.io/project/183121-raspberry-pi-400-cyberdetox-display)

如果这开始看起来熟悉，那么你会正确地认识到它是那些廉价 LCD 屏幕的略高质量版本，这些廉价 LCD 屏幕已经在 Pi 上使用了好几年。屏幕旁边是一对扬声器，整个东西从 Pi 400 的背面向上延伸。我们会质疑扩展连接器可以承受多少负载，但实际上它似乎不会承受太多。

正在使用的设备可以在中断下方的视频中看到。它肯定不是最大的显示器，当用作桌面时，它相当局促，但对于终端来说似乎足够了。与许多电脑平台相比，它的优势在于，当新鲜感减弱时，它可以被移除，Pi 400 与传统显示器一起使用。

Pi 400 已经问世近几年了，也许还没有得到应有的认可。如果你从未尝试过，看看我们对它的评论。
