# 数控机床卷轴加工 PVC 管

> 原文：<https://hackaday.com/2019/08/08/cnc-machine-rolls-up-an-axis-to-machine-pvc-pipe/>

无论是木头，金属，塑料，还是其他，当你需要为你的建筑获取材料时，你有两个选择:购买新的存货，或者尽你所能地搜罗。新鲜的原始材料通常更容易处理，但从意想不到的来源获得有用的库存也令人满意。

[这款用于 PVC 管的 CNC 刳刨机](https://hackaday.io/project/166909-the-pipeline-project)是从不寻常的来源获取材料的绝佳范例。[Christophe Machet]开展了他的“管道项目”,专门探索由大直径 PVC 管(通常用于下水道和其他排水管)制成的材料。它基本上是一个标准的——尽管是大尺寸的——三轴 CNC 刳刨机，其中一个轴缠绕在一个圆柱体上。管子绕在一个牺牲心轴上，装入机器，在那里它在一个看起来像天线塔的桁架下旋转。主轴看起来有点小，但它显然完成了工作；幸运的是，如果将来有必要，桁架有足够的强度和刚度来承载更大的主轴。

下面的视频展示了机器为一些可爱的椅子切割零件。[Christophe]告诉我们，椅子扶手等特征需要使用热风枪进行一些手动后期成型，但我们也可以实现这一步骤的自动化。我们喜欢这台机器生产出来的零件的外观，以及[Christophe]如何找到一种方法将一根轴改造成圆柱形。他将这个项目提交给[2019 hack aday 奖](https://hackaday.com/2019/04/03/2019-hackaday-prize-begins-right-now/)；你提交参赛作品了吗？
