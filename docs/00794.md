# 巨型机械臂使用流体动力，而不是电子设备

> 原文:[https://hack aday . com/2018/10/03/giant-robot-arm-uses-fluid-power-not-electronics/](https://hackaday.com/2018/10/03/giant-robot-arm-uses-fluid-power-not-electronics/)

公平的警告，[弗里克·威林加]记录他的[巨型非电动机器人建造](https://www.youtube.com/watch?v=c6cAfYug7lk)的视频很长。到目前为止我们只看了前两集和最新的一集，都是在休息后发布的。看一个金属加工艺术家进行一次不可思议的建造是一种投资。

第一个视频从机器人上臂的构造开始。一切都是用简单的工具制造出来的；最复杂的工具是车床和 TIG 焊机。随着手臂构造的进行，我们看到没有电子控制被发现。通过主从设置中的液压缸进行控制；从动装置打开连接到手臂肘部的气动阀，气动阀移动下臂，直到阀门关闭，并使前肢平稳停止。这是一种非常聪明的提供反馈的方式，不需要通常的传感器和微控制器。手臂末端的手也是另外一个东西，有四个手指，由复杂的连杆组成，都由各自的气缸单独驱动。整个手臂看起来是一个大型机器人的一部分，大概有 3、4 米高。至少我们希望如此，而且我们希望在这个系列的最后能看到它。

的确，我们以前见过大得吓人的机器人，但是看到一个用流体动力做任何事情都是一种享受。
