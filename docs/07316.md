# 享受一些放热焊接，铝热剂！

> 原文：<https://hackaday.com/2020/07/30/enjoying-some-exothermic-welding-with-thermite/>

可能没有多少人不知道铝热剂是什么，以及它如何展示失控放热反应的力量。不涉及破坏的实际应用可能不太为人所知。这就是使用铝热剂来创建焊缝相当有趣的地方，如[Finn] 的这个视频所示，它也是在休息后嵌入的。

在视频中，人们可以看到[芬恩]如何使用铝热剂电荷在几秒钟内在石墨模具中将大量铜导体焊接在一起。直接头、T 形接头和其他接头就是将导体放入模具中，按下按钮，然后观看焰火。稍微清理一下熔渣后，就会留下坚固耐用的焊缝。

这种工艺的正式名称是“[放热焊接](https://en.wikipedia.org/wiki/Exothermic_welding)”，从 19 世纪就开始使用。当时它主要用于铁轨焊接。如今，它在高压布线和其他应用中有很多用途，比如在链接视频中。放热焊接的明显优点是，由于两个表面已经永久接合，因此所得的接头坚固耐用。
