# 黑客日播客 038:电脑平台接管，树脂 3D 打印 Vs FDM，硅胶注射成型，泡菜叉惨败

Hackaday 的编辑汤姆·纳尔迪和迈克·斯奇斯梳理了过去一周他们最喜欢的黑客。我们喜欢唐纳德·帕普(Donald Papp)关于从 FDM 3D 打印机向树脂基工艺飞跃之前的考虑因素的文章，我们巩固了对低重力固化水泥的想法。Tom 正在构建一个 Cyberdeck，他还发现了一个更早版本的非常不同的 Hackaday 播客的古老片段。我们印象深刻的是一台几乎是 3D 打印的无用机器，一台也是 3D 打印的热插入式印刷机，以及本周出现的基于 Raspberry-Pi 的 Sidekick 克隆。DIY 电线弯曲机器人是一个令人难以置信的构建，就像机械键盘上华丽的电线布线一样，还有这个扑克牌压机上的花丝工作。另外，你需要花一些时间迷失在这个氢线望远镜项目中。
