# 带伺服系统的印刷门把手转动钥匙

> 原文:[https://hack aday . com/2020/03/30/printed-door-handle-turns-key-with-a-servo/](https://hackaday.com/2020/03/30/printed-door-handle-turns-key-with-a-servo/)

[Madalin Valceleanu]有一个有点独特的问题。他想让他的前门变得更“智能”，但他找到的物联网门锁都不符合他的加固门的风格。所以他开始着手设计和 3D 打印他自己的互联网控制门把手。

现在我们说处理而不是锁定，因为内部机制实际上没有被替换。毕竟，这些部分并不适合在 PLA 中重新创建。印刷组件只是取代了门内部原有的板和把手。

在这种情况下，你可能会想这一切有什么意义。如果他还在使用相同的内部机制，新的手柄会有什么帮助呢？在他的新手柄上，[马达林]集成了一个伺服系统，能够转动门上的原始钥匙。通过将伺服系统连接到树莓派上，他可以通过家庭自动化系统来锁门和开门。

[Madalin]已经在 Thingiverse 上提供了他的印刷把手的 STLs，[但是像大多数这些“螺栓安装”式的门修改一样](https://hackaday.com/2020/01/16/a-simple-app-controlled-door-lock/)，我们认为这个设计已经足够定制，对其他任何人来说都不会有太多的实际用途。尽管如此，这仍然是用一些跳出框框的思维来解决现实世界问题的一个很好的例子。
