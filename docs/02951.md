# 制作一个短跑计时器来帮助你的训练

> 原文:[https://hack aday . com/2019/05/11/build-a-sprint-race-timer-to-help-your-training/](https://hackaday.com/2019/05/11/build-a-sprint-race-timer-to-help-your-training/)

任何锻炼都是积极的事情，但是如果你想随着时间的推移而提高，你必须衡量你的表现。[Nikodem Bartnik]是一名跑步运动员，希望提高自己的短跑能力。当然，Arduino 是帮助完成这一任务的最佳伴侣(YouTube 链接，嵌入下方)。

Arduino 内置在一个 3D 打印的外壳中，有几个输入按钮。相当不寻常的是，界面选择了一个小的电子纸显示器。这样做的好处是白天在户外容易阅读，而且耗电很少。

该设备使用简单，使单独训练变得轻而易举。可以选择要跑的距离，该装置会发出一系列蜂鸣声，以指示跑步者何时开始。计时器放置在终点线上，用超声波传感器检测经过的跑步者。

对于冲刺计时来说，这是一个有用的构建，并且可以通过远程启动功能变得更加通用。如果你需要计时风火轮而不是短跑运动员，不要担心—[也有适合你的版本](https://hackaday.com/2015/01/19/diy-hot-wheels-drag-race-timer/)。休息后的视频。
