# 3D 打印扩散器可产生更多自然光

> 原文:[https://hack aday . com/2019/02/09/3d-printed-diffuser-make-more-natural-light/](https://hackaday.com/2019/02/09/3d-printed-diffusers-make-more-natural-light/)

一条发光二极管可能是一种简单灵活的方式来为项目增加光线，但它们看起来并不总是自然的。不过，有一个简单的方法可以让它们看起来更好:增加一个扩散器。这就是[Nate Damen] [使用 3D 打印机](https://hackaday.io/project/34274-atltvhead/log/159126-led-diffusion-log)所做的。他使用 PETG 创造了一个扩散器，使标准的 led 灯串看起来更柔和、更自然，看起来更像旧的光源，如荧光带或 EL 线，但 led 灯的颜色更灵活。他使用的 PETG 材料有一种自然的有点浑浊的外观，所以它不需要任何额外的处理就可以作为扩散器。

[Nate]提供的扩散器设计用于 7 毫米宽、每米 144 个 LED 条，但他高兴地承认这只是一个起点:让扩散器适应不同形状或类型的 LED 并不困难。他的实验表明，漫射器的内部几何形状对最终外观有最大的影响，因为这控制了光在漫射器内的反射和传播方式。他这样做是他的 ATLTVHEAD 项目的一部分，这是一种用旧的便携式电视机框架制成的可穿戴显示器。通过在安装在屏幕后面的 LED 条上使用这些 3D 打印的漫射器，他创造了一种看起来更像电视机的东西，具有更柔和、更漫射的外观。
