# 制作彩色烟雾装置，正确的方法

> 原文:[https://hack aday . com/2018/11/10/making-colored-smoke-devices-the-right-way/](https://hackaday.com/2018/11/10/making-colored-smoke-devices-the-right-way/)

烟火很有趣，如果采取了适当的预防措施，在家里玩(通常)是足够安全的。虽然购买现成的烟花和烟雾装置是很典型的，但实际上也有可能在库存充足的家庭实验室里制作。[[Tech Ingredients]在这里分享在家制作自己的超级彩色烟雾装置的技术。](https://youtu.be/EWaEhLhgeso)

产生彩色烟雾需要与制作简单的白色烟雾装置略有不同的策略。彩色烟雾使用对温度敏感的染料，因此必须小心控制反应温度。这是通过选择氯酸钾氧化剂，并将其与碳酸镁和碳酸氢钠结合来实现的，这有助于阻止反应变得太热。糖被用作主要燃料，乳糖和蔗糖都适用于此目的。然后用溶剂型染料上色，这种染料在网上很容易买到。与典型的水基食品级染料相比，这些染料在更高的温度下是稳定的，因此是产生浓厚、鲜艳的彩色烟雾的最佳选择。

[Tech Ingredients]很好地解释了这项工作背后的理论，以及取得成功所需的实际考虑。该视频是大量实验和镜头外工作的结果，这在最终演示文稿中有所展示。如果你一直在从事自己的烟火创作，[一定要点击提示热线](http://hackaday.com/submit-a-tip)。休息后的视频。
