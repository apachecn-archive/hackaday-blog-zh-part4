# 这个发光分类器是个好主意

> 原文:[https://hackaday . com/2019/02/12/this-light-up-sorter-is-a-bright-idea/](https://hackaday.com/2019/02/12/this-light-up-sorter-is-a-bright-idea/)

手工整理堆积如山的螺钉和其他工作台碎屑是一项只对我们中少数人有吸引力的任务。[阿德里安]不是那种人。他认为这项工作更适合机器人，所以他制造了一台智能的、好看的机器来做这件事。

[Adrien]的分类机器人能够快速有效地组织一大堆零件。他只是将零件分散在灯箱工作面上，照亮它，然后用朝下的网络摄像头拍照。一种算法使用 OpenCV 图像处理研究零件及其位置，并将三角测量结果发送回手臂，以便它可以使用家用 brew 电磁铁将零件拾取并放置到激光切割盒中。

[阿德里安]称这是一项正在进行的工作。他计划用 Raspberry Pi 来控制它，这样它就可以成为一个独立的单元，并且可能会将零件箱移动到外部曲线。放下你自己过去的休息，看看它排序。

如果三角洲机器人更适合你，这个机器人有球。彩球。
