# YouTube 就像是 20 世纪 70 年代的法国

> 原文:[https://hackaday . com/2022/08/27/YouTube-like-its-70 年代-法国-with this-minitel-vcr-mashup/](https://hackaday.com/2022/08/27/youtube-like-its-1970s-france-with-this-minitel-vcr-mashup/)

当不仅仅是给东西装上假齿轮，然后就此收工的时候，蒸汽朋克看起来相当酷。在一个被维多利亚美学束缚的世界里想象科技是一个很好的想法，并且可以转化为其他时期的样子——有人知道吗？

但是，如果你试图创造一种基于更近的、不太出名的时间的技术美学呢？这就是[ghettobastler]试图用这个有点奇怪的迷你电话-YouTube-VCR 混搭所做的。灵感来自[一位网络漫画家对“Formicapunk”的演绎，](https://english.bouletcorp.com/2011/07/07/formicapunk/)现代技术基于 70 年代和 80 年代大获成功的法国可视图文服务的美学，该系统使用非常酷的 Minitel 1B 终端和 Raspberry Pi 3。

[![](../Images/e37a91b37aceaaf299f7ff5ddf66b74e.png)](https://hackaday.com/wp-content/uploads/2022/08/ytmintel_detail.jpg)

A custom level-shifter for the Pi

在电平转换电路的帮助下，Mintel 和 Pi 通过串行通信，允许终端用作 Pi 的终端。Pi 从 YouTube 下载视频，然后将视频从其复合输出发送到 VCR，并使用模拟原始遥控器的 IR LED 控制 VCR。仔细想想，看看下面的视频就知道了——这可能比试图描述它更容易。

这很奇怪，但我们喜欢迷你终端的外观。关于它的一些东西只是尖叫 cyberdeck 如果有人有备用的，赶紧为我们的[网络平台设计大赛](https://hackaday.com/2022/08/08/load-your-icebreakers-the-2022-cyberdeck-contest-is-here/)准备一些东西吧。
