# juuke——适合老人和孩子的 RFID 音乐播放器

> 原文:[https://hacka day . com/2020/09/14/juuke-an-RFID-music-player-for-older-and-kids/](https://hackaday.com/2020/09/14/juuke-an-rfid-music-player-for-elderly-and-kids/)

[ananords]和他的女朋友想为她的祖母做一个简单易用的音乐播放器。像 CD 播放器和 MP3 这样的音乐播放器有点太难处理了，所以他们想找到一个更简单的解决方案。

他们构思了一个叫做 Juuk 的小点唱机，有一个简单易用的界面。他们制作了正面印有艺术家照片的个人 RFID 卡，便于从音乐库中选择不同的选项。Juuk 有一个内置的 RFID 阅读器，它可以识别每张 RFID 卡，并播放 SD 卡中相应的音乐。

这种简单的界面比那些我们今天被迫摆弄的糟糕的触摸屏设备更友好，并且有一种很酷的复古吸引力，我们的许多读者肯定会喜欢。Juuk 还有一个非常符合人体工程学的界面，一个大而易用的旋钮用于控制音量，两个适当发光的按钮，一个绿色，一个红色，用于简单的停止和播放选项。

我们喜欢我们的黑客能够融合形式与功能，并强调高可用性。[在博客](https://hackaday.com/2018/11/23/overlooked-minimalism-in-assistive-technology/)上查看我们的其他辅助技术。
