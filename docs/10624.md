# 这个可怕的机器人是来给你上一课的

> 原文:[https://hack aday . com/2021/07/01/这个可怕的机器人是来教你一课的/](https://hackaday.com/2021/07/01/this-horrifying-robot-is-here-to-teach-you-a-lesson/)

不，不管它看起来像什么，这不是一些早期的万圣节项目。你面前这个令人毛骨悚然的作品实际上是一个半开玩笑的“机器人”，由多产的[Nick Bild]创作，这是一个关于公司要求他们的远程员工回到办公室的话题声明，因为新冠肺炎的限制已经取消。当这个[超现实化身可以代替你](https://github.com/nickbild/wfh_robot)时，为什么还要每天通勤？

好吧，也许它不是 Hackaday 页面上最令人印象深刻的人形作品。但是如果你正在寻找一个简单的远程呈现系统，你可能会比浏览[Nick]提供的 Python 源代码更糟糕。使用 Raspberry Pi 4、网络摄像头和麦克风，他的客户端-服务器架构将机器人看到和听到的一切结合到一个简单的页面中，可以通过网络浏览器远程访问。

当然，如果这只是一条单行道，这个在家工作(WFH)机器人就不会太好，所以[尼克]还添加了一个扬声器，可以在客户端播放他所说的任何内容。为了防止反馈循环，他的软件包括一个功能，通过 SSH 向机器人传递适当的命令来切换音频流的方向；这是一个巧妙的技巧，让你在自己的作品中记住，不要做噩梦。

如果你正在寻找一些功能更强的东西，并且身边有一些纸板，这款用于手机的 [DIY 网真支架](https://hackaday.com/2020/08/06/telepresence-robot-navigates-upgrades/)可能是一个不错的开始。
