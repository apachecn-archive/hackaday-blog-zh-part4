# 3D 打印帽子将雨水吹走

> 原文：<https://hackaday.com/2021/09/11/3d-printed-hat-blasts-the-rain-away/>

有些想法很糟糕，我们只是尝试一下，至少这似乎是[伊万·米兰达]的哲学。完全无视关于你能用 3D 打印机做什么(或者至少应该用 3D 打印机做什么)或不能做什么的普遍共识，并继续做下去，这并不陌生，[Ivan]这次可能做得有点过了。既然雨伞很无聊，为什么不试着用一顶空气幕帽子来[保持干燥呢？](https://www.youtube.com/watch?v=SNkKZ8oyiow)

正如你将从视频中看到的，试图 3D 打印一个由 BLDC 发动机驱动的叶轮并不顺利。由于印刷过程中的缺陷(以及缺乏一种简单的方法在印刷后平衡它)导致的不平衡直接在他的头骨中引起了令人难以置信的不愉快(并且可能具有破坏性)的振动，更不用说在短时间内自动拆卸的东西了。

他毫不气馁，不顾一切地继续前进，用一个电风扇代替，增加了一点点愚蠢的因素，毕竟他说“我想我有一个解决所有问题的办法——更大的能量！”

如今，EDF 和其他类型的管道风扇被用在许多应用中。由于稀土磁体的进步，使得更强大的无刷电机成为可能，再加上廉价且易于使用的控制系统，现在是将 EDF 引入你最新疯狂想法的最佳时机。这些年来，我们已经报道了许多涵道风扇项目，包括[这个关于涵道风扇如何工作的视频](https://hackaday.com/2015/09/28/a-guy-named-bernoulli-and-ducted-propeller-designs/)，如果你还没有看过，我们认为它非常值得一看。

“西班牙的雨主要落在平原上”实际上并不反映现实，因为大部分降雨量实际上记录在多山的北部，而不是中部的“平原”，但不管怎样，当你想下雨的时候，它从不下雨，当然是在[伊万]所在的巴斯克地区。最初的测试是在商店里用软管进行的，至少可以说，这显示了对手头任务的某种奉献精神。

他确实展示了它看起来确实有效，但是我们非常确定还有很大的改进空间。尽管如此，也许更安全的做法是搁置它，继续下一个疯狂的想法？
