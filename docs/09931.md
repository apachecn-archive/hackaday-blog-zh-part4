# 人造彩色玻璃效果，3D 打印和环氧树脂

> 原文:[https://hack aday . com/2021/04/22/人造彩色玻璃效果 3d 印刷和环氧树脂/](https://hackaday.com/2021/04/22/faux-stained-glass-effect-with-3d-printing-and-epoxy/)

喜欢彩色玻璃的外观，但不喜欢所涉及的工作量、技能和材料？嗯，【北方几何】展示了如何避开所有这些废话，使用 3D 打印框架、环氧树脂和一些酒精基墨水来[创造一个非常好的假彩色玻璃效果艺术品](http://northerngeometry.com/easy-stain-glass-art-using-epoxy-and-a-3d-print/)。

[![](../Images/d16e3289ba9efd6db610d17bd7775da8.png)](https://hackaday.com/wp-content/uploads/2021/04/Stain-glass-with-Epoxy-and-a-3D-print-1-23-screenshot.png)

A smooth polypropylene board is the key to a glassy smooth back.

[Northern Geometry]以前也尝试过这个想法，但是分享了一些优化和获得最佳结果的技巧。一个建议是，首先将 3D 打印框架牢固地粘贴在光滑的聚丙烯板上作为支撑。给固化的树脂一个光滑的表面对于获得正确的外观是很重要的，因为树脂不会粘合到聚丙烯上，所以它可以被用作一种支持物来实现这一点。

一旦安装好框架，将少量的环氧树脂倒入每个空腔中，确保它进入每个角落，然后让它固化。薄薄的树脂底层不仅可以密封物品，还可以形成玻璃般光滑的背面，这是根据需要用彩色树脂完成作品的完美基础。

一旦完成，一切都有足够的时间来完全固化，只需从板上弹出一块。看看下面嵌入的视频，其中【北方几何】展示了从开始到结束的过程。
