# 垃圾车从海滩上拖走垃圾

> 原文:[https://hack aday . com/2021/07/05/litter-bugges-haul-trail-off-the-beach/](https://hackaday.com/2021/07/05/litter-buggies-haul-trash-off-the-beach/)

我们的海洋中有数量令人沮丧的垃圾，其中许多被冲到了海滩上。[格伦·莫里斯]已经把收集所有这些垃圾变成了一种最受欢迎的消遣，他们使用一系列定制的无线电控制垃圾车来拉运这些垃圾。

这些童车最基本的版本是现成的遥控岩石履带车，通常是一辆 TRX-4，用一个篮子代替车身。然而，[格伦]开发的童车远远不止这些，他广泛使用 3D 打印技术来制造特制的垃圾运输车。他设计了几个框架系统来放置可拆卸的篮子、水桶、灯和工具。大多数童车使用锂动力工具电池，以允许快速交换，电动马达，电子稳定控制系统和齿轮选择低速和高扭矩。由于童车长时间暴露在盐水中，底盘上几乎所有的钢铁硬件都换成了不锈钢。为了允许单手控制，[Glenn]在标准遥控控制器的转向旋钮上安装了 3D 打印杆，允许用拇指进行转向。

我们真的很喜欢[格伦]在这些童车上所做的事情，并认为它们可能是增加一些自主能力的良好平台。添加一个[智能手机用于目标跟踪或避障](https://hackaday.com/2020/11/06/open-source-self-driving-smartphone-robot/)，或者一些[太阳能电池板和一个 GPS 自动驾驶系统](https://hackaday.com/2020/03/15/rover-runs-slow-and-steady-on-solar-power/)。
