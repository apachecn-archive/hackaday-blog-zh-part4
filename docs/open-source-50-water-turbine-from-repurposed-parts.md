# 开源价值 50 美元的水轮机，来自重新利用的部件

> 原文：<https://hackaday.com/2019/12/20/open-source-50-water-turbine-from-repurposed-parts/>

容易获得可靠的电力是我们许多人想当然的事情，但在发展中国家或自然灾害后，它可能是一种罕见的商品。[丹尼尔·康纳利]一直在努力开发人们可以自己建造的基础设施，他的最新项目是一个 200 W 的水轮机(休息后的视频)，建造成本约为 50 美元。

该系统的核心是悬浮滑板上的轮子和马达。看起来像 110 毫米的 PVC 管连接在一起，形成一个 U 形，可以安装在人造通道的壁上。入口侧比出口侧短，系统必须充满水以允许水流开始流动，就像虹吸管一样。前两个版本的叶轮位于出口管的末端。V1 使用了来源不明的废塑料径向叶轮，根本不起作用。V2 有一个 3D 打印的叶轮，工作得很好，但旋转速度不够高，无法产生[丹尼尔]想要的电压。

V3 使用一个大型计算机风扇，安装在系统顶部管道的短水平部分。它工作得非常好，在电机的单相上产生大约 55 伏交流电，在整流三相电机输出后，应该有希望最终产生大约 90 伏 DC 和 200-500 瓦。然而，这只是指示性的，我们真的希望看到它与不同的负载连接测试。输出还将取决于特定溪流/渠道/河流的流速和水头压力，[丹尼尔]承认他们的测试条件非常理想。如果悬浮滑板马达很难得到，摩托车交流发电机也应该工作良好。

[Daniel]仍在研究系统的缺陷，但正如他在[OpenSourceLowTech](http://opensourcelowtech.org/)上的其他设计一样，一旦他准备好，他将发布完整的开源设计和教程。我们期待看到这个系统在野外实现。对于离网电力，[自制的](https://hackaday.com/2019/10/05/building-a-wind-power-generator-in-your-backyard/)和 [3D 打印风力发电机](https://hackaday.com/2019/09/19/3d-print-a-complete-wind-generator/)是这里另一个受欢迎的话题，如果你附近没有方便的渠道的话。
