# NES 有了自己的操作系统

> 原文：<https://hackaday.com/2022/09/28/the-nes-gets-its-own-os/>

直到最近，大多数视频游戏系统都不需要自己的操作系统来玩游戏。尤其是在盒式磁带时代——游戏本身直接在硬件上运行，不需要操作系统的中间人来实现游戏机的任何功能。像 Commodore 这样兼作家用电脑的电脑也有例外，但是像 NES 这样的系统从来没有自己的专用操作系统。至少，[直到【Inkbox】设计建造了 NES-OS](https://notin.tokyo/nesos/) 。

操作系统没有任何命令行，而是直接进入图形用户界面。操作系统由两个程序组成。第一个是设置应用程序，允许用户对操作系统的外观和行为进行各种更改，第二个是支持日语“家庭键盘”附件的文字处理器。NES 上的内存是有限的，由于操作系统完全加载到 RAM 中，剩余的空间只够存储总共八个文件。这些文件本身被限制在 832 字节，这相当于一个屏幕不滚动的文本。

虽然它可能只限于我们这些生活在现代的人，但这个操作系统几乎完全利用了这个 20 世纪 80 年代系统的可用处理能力和内存，这个系统因《超级马里奥兄弟》和《T2 猎鸭》而闻名。对于这样一个小的包来说，这是一个令人印象深刻的构建，并且在为这些系统构建软件时，确实会遇到很多硬件和限制。如果你需要更多的功能，我们建议[在 NES 经典版上安装 Linux，而不是](https://hackaday.com/2016/11/13/linux-on-your-nes-classic-edition/)。
