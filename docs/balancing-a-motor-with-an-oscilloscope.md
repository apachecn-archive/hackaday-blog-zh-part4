# 用示波器平衡电机

> 原文：<https://hackaday.com/2022/06/18/balancing-a-motor-with-an-oscilloscope/>

对于生活中的所有事情，我们必须寻求平衡。这听起来可能有点像新世纪呜呜呜，但如果你认为这不是真的，就试着忍受一台只有一个被子的洗衣机，或者在不平衡的轮胎上行驶几英里。

任何旋转的东西如果没有适当的平衡，都可能很快旋转成弹片，在[马蒂亚斯·万德尔]店里的 DIY 电动工具也不例外。最近对他的接头的升级让工具有点嘈杂，所以他正在用这个简单而聪明的设置探索机器振动。仅仅使用一个便宜的扬声器和一个示波器，[马蒂亚斯]就能够描述一个小型鼠笼式鼓风机的振动特征——他明智地选择从小处着手，在进入潜在危险的接头之前验证他的方法。从传感器返回的复杂波形中可以学到很多东西，示波器的频谱分析仪功能对波形分析有很大帮助。下面的视频显示了探测鼓风机各部分、区分由电噪声而非振动引起的频谱峰值，以及实际使用该设置来动态平衡风扇的过程。

我们认为这是来自[马提亚斯]的又一个便利的购物技巧，我们将期待他的 jointer 的分析。想做同样的事情但是没有示波器？没问题——[一个耳塞和 Audacity](https://hackaday.com/2015/03/07/dirt-cheap-motor-balancing-and-vibration-analysis/) 可能就是你所需要的。
