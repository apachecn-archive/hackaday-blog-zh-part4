# [科迪]造了一台氯气机器

> 原文：<https://hackaday.com/2018/08/19/cody-builds-a-chlorine-machine/>

在他继续努力使他的 YouTube 频道去货币化的过程中，[Cody]决定分享他是如何在实验室制造氯气的。因为用五磅液态汞和电来制造氯、氢和碱液的东西不会出问题。

我们将第一个承认我们并不完全了解氯气机器是如何工作的。电化学方面非常简单——它利用电解从盐水溶液中释放出氯。电化学电池的一面产生氯，一面放出副产品氢。我们甚至知道了汞阴极的用途，它以汞合金的形式捕获钠金属。让我们困惑的是[科迪]是如何将 5 磅的水银泵入电池的两半之间的。移动如此稠密的液体似乎具有挑战性，在玩弄了蠕动泵等更传统的方法后，[Cody]利用汞的导电性，使用几块钕磁铁来泵送它。除了将它描述为“水银轨道炮”之外，他并没有真正解释这个想法，但它似乎足够有效，可以温和地循环水银。看看下面的视频，它能够产生足够的氯来溶解黄金和漂白布料。

我们需要提供通常的警告，说明腐蚀性、反应性和毒性材料可能并不适合每个人。他过去的视频，从[把尿液变成火药](https://hackaday.com/2015/06/15/gunpowder-from-urine-fighting-a-gorn/)到[在路边开采铂金](https://hackaday.com/2016/06/06/mining-platinum-from-the-road/)，显示出【科迪】显然非常了解化学，并且他采取了适当的预防措施。所以如果你有一罐水银，你想试试这个，小心点。
