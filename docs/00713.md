# 用耶鲁的“机器人皮肤”把你的泰迪熊变成机器人

> 原文:[https://hack aday . com/2018/09/22/将你的泰迪熊变成一个拥有 yales 机器人皮肤的机器人/](https://hackaday.com/2018/09/22/turn-your-teddy-bear-into-a-robot-with-yales-robotic-skin/)

尽管我们可能在新电影《小熊维尼》中看到了什么，但我们珍爱的毛绒玩具通常不会活过来。但是如果这是我们的目标，我们有办法实现它。像这些来自耶鲁大学的[【机器人皮肤】](https://www.eurekalert.org/pub_releases/2018-09/yu-st091418.php)。

每个模块都是安装在柔性基板上的传感器和致动器的集合，然后安装在作为结构的柔性物体上。在一个简单的实现中，机械部分被缝在一块织物上，并用拉链系在一块泡沫上。演示视频(嵌入在休息时间下方)贯穿了主题的多个变体。从让泡沫管(“泳池面条”)像蛇一样爬行到让马玩具的腿动起来。

这些有趣的原型背后有一个严肃的动机。美国宇航局一直在寻求减轻发射到太空的重量，这源于模块化机器人的想法。这些机器人皮肤可以移动到不同的机器人身上来执行各种任务，而不是嵌入在单个机器人中执行特定功能的致动器和传感器。这种灵活性可以开发更多的功能，同时减少火箭的重量。

这个想法仍然处于早期发展阶段，当前级别的原型看起来像是我们大多数人可以在我们的项目中使用的复制和改进的东西。我们甚至为那些气动系统准备了一个[控制器。随着更多的发展，它可能会成为](https://hackaday.com/2018/05/24/programmable-air-makes-robotics-a-breeze/)[神秘驱动器](https://hackaday.com/2015/12/10/esoteric-actuators/)的一员。
