# 迷笛手，迷笛手，迷笛手

> 原文：<https://hackaday.com/2019/07/04/midi-gurdy-midi-gurdy-midi-gurdy-man/>

绞弦琴是完美的乐器。这是一种带有曲柄的乐器，是由低音琴弦和奇怪的半音阶键盘组成的机械奇迹。没有其他乐器能将风笛悦耳的嗡嗡声与一个八岁孩子试图用一把调得不好的小提琴演奏十字面包的听觉体验结合起来。

现在，绞弦琴正在数字化。数码吉他是[XenonJohn]参加今年的 Hackaday 奖的作品，正如它在罐子上所说的那样:这是一种不断发出嗡嗡声的乐器，琴键敲击出一段旋律。

如果你不熟悉绞弦琴，这个视频是一个非常好的介绍。它是一个盒子，外面装有四到六根绳子。琴弦通过由曲柄驱动的木轮振动。沿着乐器的主体有一个键盘，它“拨弄”一根提供旋律的弦；其他的弦都是持续发声的无人机弦。我想是在齐柏林飞船的视频里，伙计。

虽然制作一个模拟的绞弦琴有点复杂，但深入数字领域是很容易的:[XenonJohn]正在制作一个绞弦琴，它只是用一些按钮和一个小小的 3.6 微控制器输出 MIDI 命令。这些零件是 3D 打印的，由于这种绞弦琴是完全数字化的，你可以在不实际调音的情况下改变低音琴弦的调音。太棒了。
