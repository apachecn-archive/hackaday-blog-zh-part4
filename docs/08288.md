# 你好，来自附近的空间

> 原文:[https://hackaday.com/2020/11/09/hello-from-the-nearspace/](https://hackaday.com/2020/11/09/hello-from-the-nearspace/)

对于任何进入高层大气区域(有时称为近空间)的系统来说，一个关键的挑战是与地面的通讯。许多高空气球和卫星上的传感器和相机如果收集的数据无法检索就没有用。通常情况下，定制天线或信标会有所帮助。考虑到这个问题的成本和难度，[arko]和[upaut]合作，试图通过建造[CUBEX 为任何近空间爱好者提供一个交钥匙解决方案，这是一个带有传感器和智能无线电的奇妙的小模块](https://hackaday.io/project/270-cubex)，可以很容易地重复使用和重新利用。

CUBEX 是一个高空气球的有效载荷，带有相机、全球定位系统、小型电池、太阳能电池和附带的电源管理电路。聪明的一点来自无线电。通过使用 434.460 Mhz 频段，它可以以 10mW 的功率广播约 100 英里。唯一可以接收的硬件是一个无线电监听器(一个便宜的 RTLu 盘就很好)。图片和 GPS 坐标以 300 波特的速度传输。

他们的发射相当成功，虽然他们没有赶上日食，但他们的气球在拍照时达到了令人印象深刻的 33698 米(110560 英尺)。尽管它最终溅落在太平洋中，但由于其简单而经济的数据链接，他们能够享受[大量华丽的照片](http://habexproject.org/cubex/)。
