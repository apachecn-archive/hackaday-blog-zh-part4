# 事实证明，机器人也需要严厉的爱

> 原文：<https://hackaday.com/2020/01/08/it-turns-out-robots-need-tough-love-too/>

根据南加州大学进行的一项研究，向机器人展示敌对行为可能是提高其表现的关键。虽然两个神经网络在一场游戏中竞争的生成性对抗网络(GAN)已经被证明，但这是第一次在学习努力中使用对抗的人类用户。

[报告](https://arxiv.org/pdf/1903.00636.pdf)在智能机器人和系统国际会议上发表，描述了使用强化学习来训练机器人系统以创建通用系统的实验。对于大多数机器人来说，为了以类似人类的方式操纵物体，大量的训练数据是必要的。

成功克服这一问题的一个研究方向是“人在回路中”，其中人向系统提供关于其能力的反馈。大多数算法已经假设了一个合作的人类助手，但是通过对抗系统，机器人可能更倾向于发展对真实世界复杂性的鲁棒性。

进行的实验包括一个机器人试图在计算机模拟中抓住一个物体。人类观察者观察模拟的抓取，并且如果抓取成功，试图从机器人手中抢夺物体。这有助于机器人辨别微弱和牢固的抓握，这是来自成功工作的研究人员的疯狂想法。与对手一起训练的系统拒绝不稳定的抓取，快速学习针对不同对象的鲁棒抓取。

像这样的实验可以测试机器人应用程序学习任务中的假设，从而产生更好的压力测试系统，更倾向于在现实世界中工作。看看休息时间下方视频中[的采访。](https://www.youtube.com/watch?v=TixzSbunn5A)
