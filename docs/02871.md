# 3D 打印喷水驱动器

> 原文:[https://hack aday . com/2019/04/30/3d-printing-a-water-jet-drive/](https://hackaday.com/2019/04/30/3d-printing-a-water-jet-drive/)

[伊万·米兰达]一直在尝试 3D 打印，最近他开始在水上工作。他的最新作品是一艘赛艇，但它的性能让他需要速度。[提示 3D 打印喷水引擎的开发](https://www.youtube.com/watch?v=CP3alznCPwg&feature=youtu.be) (YouTube 链接，嵌入下方)。

操作的基本原理很简单。水通过入口被吸入，在入口被无刷电机驱动的涡轮加速。这种涡轮，结合定子鳍，迫使水通过出口，推动船只前进。

第一个原型是用解放军打印的。公差很好，这主要归功于[Ivan]的经验和校准良好的打印机。组装后，发动机点火，效果很好。在找到一系列更大的浴缸来测试该设备后，发动机终于开足马力，似乎能够转移大量的水。

我们希望看到一个适当的仪器推力测试，特别是一个比较设备与市场上其他喷水驱动器。[无刷电机为遥控船](https://hackaday.com/2018/08/24/rc-boat-goes-brushless-for-speed-reliability/)提供了一个很好的驱动解决方案，所以我们确信【Ivan】很快就会把这个湖撕碎。休息后的视频。
