# 机电一体化手模仿人体解剖学以实现灵巧性

> 原文：<https://hackaday.com/2018/10/18/mechatronic-hand-mimics-human-anatomy-to-achieve-dexterity/>

看看人类手的奇妙复杂性。二十七块骨头与肌肉、肌腱和韧带协同工作，延伸到前臂，产生一系列运动，给了我们从石器到交响乐的一切。我们的手是我们用来在一个很好的层面上与物质世界互动的东西，可以理解的是，我们希望我们自己的机械版本包括类似灵巧的手。

这是一个很难完成的任务，但是这个仿生机电手是朝着这个方向迈出的令人印象深刻的一步。这是[Will Cogley]大学三年级的设计项目，他在下面的第一个视频中总结了这个项目。这个项目有两个部分:机械手本身和控制它的动作捕捉手套，这两者都同样令人着迷。控制手套覆盖着手部每个关节的 3D 打印传感器。他使用 SMD 电位计来测量接头角度，由于焊点断裂，有些困难；也许他可以用更细的电线和更好的应力消除来解决这个问题。

手套控制的手是一个设计奇迹，就像好莱坞机器人手臂末端的东西。每个手指关节都是由前臂的一个伺服系统拉动电缆来操作的；关节通过弹簧回到中间位置。手能够有多种抓握方式，并对控制手套输入做出相当好的响应，尽管在一些关节的传感器中有一些抖动。

下面的第二个视频更详细地概述了这个项目，并展示了[Will]的设计是如何发展的，以及它的发展方向。拟人化的手是[远非罕见的项目](https://hackaday.com/2015/09/20/hackaday-prize-semifinalist-openbionics-affordable-prosthetic-hands/)在这里，但我们要说这一个有很多优点。
