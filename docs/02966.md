# 2019 年康奈尔杯冠军包括自主船，扑翼无人机和跳跃漫游者

> 原文:[https://hack aday . com/2019/05/15/2019-Cornell-cup-winners-include-autonomous-boat-flapping-UAV-and-jumping-rover/](https://hackaday.com/2019/05/15/2019-cornell-cup-winners-include-autonomous-boat-flapping-uav-and-leaping-rover/)

对于大学时代的工程师和设计师来说，尽早找到他们真正热爱的问题很可能会为整个职业生涯设定轨迹。这正是康奈尔杯的目标，这项比赛要求申请者以独特而有趣的方式解决现实世界中的问题。从我们看到的情况来看，这肯定是有效的，因为团队展示了基于扑翼飞机的四轴飞行器、机器人小艇、森林火灾探测器和高空漫游车。

[![](../Images/a22fcb88ce4e39feaa415fa88d9ef992.png)](https://hackaday.com/wp-content/uploads/2019/05/cornell_sign.jpg) 在这种开放式的方法下，单个参赛作品往往会有很大的差异，从自动驾驶汽车到辅助技术，无所不包。没有一个团队会因为追求一个他们没有真正投入的项目而感到压力，每个人都会因此变得更好。

鉴于如此崇高的目标，Hackaday 很自豪能够赞助 2019 年康奈尔杯。尤其是它与今年 Hackaday 奖的产品设计焦点如此紧密地联系在一起。当目标是达到大规模生产时，设计一些解决现实世界问题的东西肯定是公式的一部分。在肯尼迪航天中心决赛期间亲眼目睹参赛作品后，我们认为每一件作品都将是进入 Hackaday 奖的绝佳机会。

## 第一名:MAPR

由宾夕法尼亚大学团队开发的 MAPR 号在多个方面给人留下深刻印象。它是唯一进入决赛的自主车辆，也是唯一的船只。在一个主要由桌面演示主导的房间里，它令人印象深刻的八英尺长也很难被忽略。鉴于它的尺寸，不幸的是，自主船没有办法在室内展示它的动作，但之前在宾夕法尼亚州 Schuylkill 河探险期间录制的视频被展示给评委们观看。

[![](../Images/e8725cf5f14cea2c760b177806ac0587.png)T2】](https://hackaday.com/wp-content/uploads/2019/05/cornell_mapr.jpg)

该团队与 S.S. MAPR 的目标是使执行水质样本比派人去做更快更便宜，这反过来有助于当局识别和应对饮用水源的生态威胁。操作人员提供他们想要取样的位置和深度，该船使用潜水泵将样本带上船。250 mL 样品容器装在一个新颖的旋转机构中，并且在样品收集前有清洗泵和导管的规定，以避免与之前的样品交叉污染。

最初，该团队计划制造自己的船体，但最终发现使用商用充气船作为起点更有意义。在推进方面，他们使用了蓝色机器人 T200 推进器，每个推进器能够提供 5.1 公斤的推力；足够让轻型充气飞船到达它需要去的地方。该飞行器相当大的能量需求确实被证明是一个挑战，在发现纯电池电源无法提供必要的续航能力后，该团队最终决定采用机载 700 瓦汽油发电机。

## 第二名:泰拉诺娃

并非康奈尔杯期间解决的所有问题都源于地球。从理论上讲，月球上的洞穴将成为未来人类前哨的理想起点，但在此之前，它们完全没有被探索过。代替相对稳健的人类探险者，需要异常敏捷的漫游者深入这些洞穴几英尺以上。

 [![cornell_terra1](../Images/eba3e34694916456b802424cda875d3e.png "cornell_terra1")](https://hackaday.com/2019/05/15/2019-cornell-cup-winners-include-autonomous-boat-flapping-uav-and-leaping-rover/cornell_terra1/)  [![cornell_terra2](../Images/7e23ca975e45005b71d3622300fc4ba3.png "cornell_terra2")](https://hackaday.com/2019/05/15/2019-cornell-cup-winners-include-autonomous-boat-flapping-uav-and-leaping-rover/cornell_terra2/) 

进入 TerraNova。这种漫游者设计不仅能够处理粗糙和不平坦的地形，而且它还有一个秘密武器:弹簧加载的垂直跳跃能力，旨在使它摆脱困境。在月球上，该团队估计他们的系统应该能够在不消耗任何燃料的情况下将 TerraNova 推进空中一米。随着气体推进器的加入，这种能力可能会扩展到执行无人认领的谷歌月球 X 奖所需的长距离水平“跳跃”。

尽管火星车令人印象深刻，但 TerraNova 团队对他们的机器真的去月球并不抱有幻想。但是，那从来都不是真正的目标。希望在他们的跳跃漫游者的开发过程中所做的研究和工程可以证明对 NASA 或其他执行未来月球任务的实体有用。当然，知道你不必设计你的车辆离家数十万英里运行有它的优势；即使用消费级组件的能力，以及用连接到笔记本电脑的 Xbox 控制器取代复杂的半自动控制系统。

## 第三名:FWMAV

最后但同样重要的是加州大学欧文分校的扑翼微型飞行器(FWMAV)。基本上，该团队采用了我们都很熟悉的传统四轴飞行器设计，并用四组扑翼机翅膀取代了旋转的螺旋桨。这两种截然不同的飞行方式的融合产生了一种具有四轴飞行器机动性的飞行器，但没有噪音和危险的桨叶，而这些迄今为止被认为是小型无人机的重要组成部分。

 [![cornell_thumb2](../Images/51fcbdf37b04d1914ec623d5ce73792a.png "cornell_thumb2")](https://hackaday.com/2019/05/15/2019-cornell-cup-winners-include-autonomous-boat-flapping-uav-and-leaping-rover/cornell_thumb2/)  [![cornell_flappy4](../Images/4ab188e974405909d43e5b1f62cdf41a.png "cornell_flappy4")](https://hackaday.com/2019/05/15/2019-cornell-cup-winners-include-autonomous-boat-flapping-uav-and-leaping-rover/cornell_flappy4/) 

理解它如何飞行的物理原理是一回事，但在空中看到它完全是另一回事。FWMAV 不只是飞，它还能变焦。扑翼机翅膀的一个副作用是，该飞行器能够非常接近水平地飞行，但具有直升机的悬停能力和精确着陆精度。我们被告知，该团队获得了美国宇航局的特许，可以驾驶 FWMAV 绕会议中心飞行，这个决定肯定受到了所有与会者的赞赏。

 [https://www.youtube.com/embed/kocnm8xpoJc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kocnm8xpoJc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

没有旋转叶片，FWMAV 比传统的 UAV 设计要安全得多。你可以用手抵住拍动的翅膀，不会对自己或飞机造成伤害。虽然它还远远没有安静下来，但翅膀的运动产生的有机噪音比你想象的要多得多。看到它嗡嗡作响，在房间里飞来飞去，它似乎更像一个巨大的昆虫，而不是一个机械装置。

令人难以置信的是，FWMAV 中使用的每一个硬件都是现成的组件。从飞行控制器到扑翼机的机械装置，它们都是你现在就能买到的零件。诀窍是以一种新颖的方式将它们结合起来。据该团队称，最困难的部分是修改飞行控制器的开源固件，以处理新的推进系统。

## 人民的选择:瓦肯物联网

[![](../Images/da65d6c84819aecd0626816d9e193db2.png)](https://hackaday.com/wp-content/uploads/2019/05/cornell_vulcan.jpg) 虽然没有被评委选中，但 Vulcan IoT 给康奈尔杯的与会者留下了深刻的印象，在决赛结束时获得了“人民选择”奖。该项目使用低成本传感器网络，通过分析气温和土壤湿度等变量，试图预测野火何时发生。将来自传感器网络的实时数据与几十年的历史记录进行比较，该团队认为他们的系统可以[先发制人地警告当局即将到来的火灾威胁](https://vulcaniot.net/)。

该系统的成功本质上取决于拥有一个庞大的报告节点网络，因此展望未来，该团队希望尽可能降低传感器成本。在这个早期阶段，他们仍然使用 CNC 铣削的 PCB 和 3D 打印外壳，但随着传感器网络的扩展，这些将让位于专业制造的电路板和注塑外壳。

现在原型使用的是粒子电子微控制器，他们觉得这对于最终的硬件来说是大材小用。随着设计朝着更高的生产率发展，[他们将寻找更符合其特定需求的 MCU](https://hackaday.com/2019/04/04/wopr-building-hardware-worth-sharing/)。该团队表示，在未来，开放的 API 甚至可以允许其他制造商(甚至个人)制造的传感器将数据推送到他们的系统中，最终为 Vulcan IoT 提供火灾探测服务铺平道路。

## 不仅仅是一场比赛

我们想特别展示在今年康奈尔杯上脱颖而出的参赛作品，但这个活动真的不仅仅是几个获奖项目。我们采访的每个人都说康奈尔杯的真正价值不是奖金，而是与其他球队交流并了解各种问题是如何解决的机会。由于每个项目都在解决一个不同的问题，团队把自己看得更像同事，而不是竞争对手；如果所有的团队都朝着同一个最终目标努力，这将允许比你可能看到的更自由的信息交换。

为了捕捉像康奈尔杯这样的活动培养的一点能量和友谊，我们制作了一个视频，展示了我们在为期两天的活动中采访的一些热情的学生和员工。
