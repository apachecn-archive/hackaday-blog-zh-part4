# 黑客日链接:2022 年 5 月 1 日

> 原文:[https://hackaday.com/2022/05/01/hackaday-links-may-1-2022/](https://hackaday.com/2022/05/01/hackaday-links-may-1-2022/)

本周我们从来自火星的消息开始，因为，让我们面对现实吧，来自这个星球的消息最近并不那么有趣。但是在这颗红色星球上已经达到了几个里程碑，第一个是[“坚忍号”【T2 号】抵达它被派往探索的古老河流三角洲](https://scitechdaily.com/weve-arrived-nasas-mars-perseverance-rover-starts-the-delta-front-campaign/)。火星车当然选择了风景优美的路线到达那里，在过去的 424 个 sols 中行驶了 10.6 公里，到达了离它着陆的地方只有大约 3.5 公里直线距离的位置。当然，很多额外的驾驶是为了支持意想不到的成功的独创性演示，再加上沿途在有趣的功能上花时间停留。但是漫游者现在已经就位，以检查最有可能藏有古代水生生物化石遗骸的沉积岩，而不是迄今为止它沿着火山口底部研究的主要是火成岩。我们期待着看到会发生什么。

说到独创性，勇敢的直升机继续表演。它最近完成了五次计划飞行中的第 27 次飞行，请注意，[发现了去年二月从](https://mars.nasa.gov/news/9177/nasas-mars-helicopter-spots-gear-that-helped-perseverance-rover-land/)[壮观到达](https://hackaday.com/2021/02/22/stunning-footage-of-perseverance-landing-on-mars/)的飞船残骸和降落伞。他们看起来很糟糕，但这是意料之中的。“独创性”发回的令人震惊的高分辨率照片无疑将被工程师们仔细研究，以寻找它所掌握的关于该结构如何经受住下降和着陆条件的任何线索。我们不确定，但这可能是自艾伦·比恩在 1969 年绝对确定阿波罗 12 号着陆在勘测者 3 号步行距离之内以来，第一次有人重访着陆或坠毁地点。

“艾美奖奖颁给了……喜剧演员？”不，不是真的，并向任何因提及不应使用的字体而引发的人道歉。但是本周[2021 年技术&工程艾美奖](https://www.w3.org/2022/04/pressrelease-woff-emmyaward.html.en)颁给了万维网联盟(W3C)网络字体工作组和 MPEG,“为网络和电视设备定制可下载字体和排版标准化字体技术”需要说明的是，这个奖项不是授予任何特定的字体，而是授予可下载字体的技术成就。尽管如此，人们还是很容易取笑字体设计，尤其是当字体势利者继续谈论这种衬线或那种下行线的优点时。但那些我们大多数人认为毫无意义的细微差别，其实在网站的“感觉”中起着巨大的作用。与 Web 1.0 时代的一些暴行相比，Web 字体与如今大多数网站看起来有多精致和专业有很大关系。

几周前，我们在 Hackaday.io 上与 Mike Jouppi 进行了一次 [PCB 散热设计黑客对话](https://hackaday.io/event/184333-pcb-thermal-design-hack-chat)。这是一次很棒的讨论，并引发了与会者 Sven 关于热通道的有趣讨论。看起来斯文的申请有很多让迈克咬咬牙的地方，因为他为[制作了一个关于这种 PCB 散热设计的深度视频](https://youtu.be/Yt61F8wuGcU)，以及一个关于开发 [IPC-2152](https://www.ipc.org/TOC/IPC-2152.pdf) 的讨论，这是 PCB 走线载流量设计的标准。感谢高级汇编的马克·休斯在这方面的提醒，并建议迈克首先加入黑客聊天。

最后，看起来 Lichtenberg 模式越来越受欢迎，以至于有必要警告人们玩电是危险的。Lichtenberg 图案是通过(有希望)明智地应用高压在木头或其他材料上烧出的分形通道，肯定是我们在之前已经看到并展示过的[。然而，美国木工协会——因为当然有这样一个团体——](https://hackaday.com/2015/07/04/high-voltage-wood-and-resin-result-in-fractal-art/)[现在谴责这种做法](https://www.woodturner.org/Woodturner/Resources/Safety-Materials/Safety-Fractal-Burning-Lichtenburg-Burning.aspx)，这显然已经在木工社区流行起来。AAW *真的*不希望任何人做列支敦士登——他们在所有活动和出版物中禁止这种做法，寻求医生的帮助来告诉每个人电可以让你的心脏停止跳动，并敦促任何有分形燃木装置的人在它袭击之前扔掉它。我们理解并完全同意在高电压下工作可能是危险的，我们并不是说伍德特纳不应该认真对待 Lichtenbergs 的潜在致命后果。但是这个警告来自一个鼓励其成员站在一台机器前的组织似乎有点过分，这台机器可以[向你扔锋利的工具](https://youtu.be/7H7s7n3GVSs?t=320)，[向你的头扔沉重的木头](https://youtu.be/mH5f2_KyZ5c)，或者[立即脱掉你的衣服](https://youtu.be/EGSORXNPMKE?t=184)——或者更糟。各位，注意安全。
