# 智能手套帮助骑车人被看见

> 原文:[https://hack aday . com/2020/06/20/smart glove-helps-cyclers-be-seen/](https://hackaday.com/2020/06/20/smartglove-helps-cyclists-be-seen/)

骑自行车的人与其他车辆共用道路，当司机看不到或不尊重骑车人的空间时，经常会导致问题。为了尝试和缓解这些问题， [[Matlek]制造了智能手套来帮助骑自行车的人向其他道路使用者传达他们的意图。](https://www.instructables.com/id/Smartglove-for-Cyclists/)

该项目包括一个装有 Arduino Nano 33 BLE 传感器的手套，具有蓝牙和运动感应功能。结合 TinyML 机器学习代码，Arduino 能够感知骑手的手势。这些手势随后被解释，相关信息显示在佩戴在骑手背部的 LED 屏幕上。向左和向右轻弹手腕会闪烁指示用户将要改变方向，而向后轻弹会闪烁警告用户正在刹车。

这是一种将车辆风格的照明整合到一个简单的骑行界面中的简洁方式。这有利于让其他道路使用者看到骑自行车的人并了解他们在路上的意图，尤其是在晚上。当然，如果你真的想被人注意，这个自行车音响也能帮上大忙。休息后的视频。
