# 在电动管道风扇上安装加力燃烧室

> 原文：<https://hackaday.com/2021/09/17/putting-an-afterburner-on-an-electric-ducted-fan/>

加力燃烧室通常被认为是军用战斗机最酷的特征。将原始燃料注入喷气发动机的废气流中，加力燃烧室是产生炽热燃烧废气的原因，也是许多飞机实现超音速飞行的关键。[Integza]想看看同样的概念是否可以应用于电动管道风扇，[并着手寻找答案。](https://www.youtube.com/watch?v=C2OmUBpn66k)

当然，为 EDF 建造加力燃烧室确实增加了许多复杂性。EDF 的下游安装了一个火焰管，配有一个精心钻孔的铜管，作为燃料喷射器。火焰管还装有汽车电热塞，以点燃燃料，这是一种直接从易拉罐中补充气体的打火机。整个组件被包裹在一个透明的丙烯酸管内，可以很容易地看到燃烧过程中发生了什么。

结果喜忧参半。虽然燃料确实燃烧了，但是以相当间歇的方式。在正常运行时，加力燃烧室将平稳、连续、轰鸣地燃烧。此外，没有进行推力测量，组件几乎没有晃动桌子。

因此，如果有什么不同的话，这个视频更像是一个如何在电风扇的帮助下燃烧大量打火机气体的指南。这个概念确实有优点，[我们也看到了过去的尝试，但是我们很想看到一个适当的设置，有和没有加力燃烧室的推力读数，看看它实际上创造了一些有用的推力。休息后的视频。](https://hackaday.com/2013/03/25/giving-an-electric-rc-plane-an-afterburner/)
