# 无线电控制的独轮车撕裂了轨道

> 原文：<https://hackaday.com/2019/05/26/radio-controlled-wheelbarrows-tear-up-the-track/>

手推车是园艺工具的可靠伴侣，有助于来回移动重物。[詹姆斯·布鲁顿]想象力丰富，[决定建造一辆电动赛车，并挑战他的朋友们也这样做(Youtube 链接，嵌入下方)。](https://www.youtube.com/watch?v=IAB69HGgl8c)

[詹姆斯]采用了一种简单的设计，将两个车轮安装在后部，并用无刷内燃马达为它们提供动力。最初的前轮安装了一个脚轮机制，使手推车可以滑动转向。一对锂聚合物电池提供动力，詹姆斯使用 VESC 滑板电子稳定控制系统来驱动马达。整个装置是无线电控制的，由一个 Arduino 处理混音以完成驾驶任务。

电动手推车在竞争中表现出色，来自[Tom Stanton]的螺旋桨驱动手推车和来自[Ivan Miranda]的吹叶机驱动手推车。恶劣的天气确实引起了一些问题，但三人组很友好地用他们的赛车给我们带来了一场毁灭德比。

你可能熟悉[James]在 openDog 项目的早期工作。休息后的视频。
