# 电动汽车销售的症结:人们仍然想要手动变速器

> 原文：<https://hackaday.com/2022/11/15/ev-sales-sticking-point-people-still-want-manual-transmissions/>

说我疯了吧，但我对手动变速器要么骑要么死。作为一个孩子，我开过足够多的卡丁车，玩过足够多的杆位，知道在任何非人力驱动的事情上，自己换挡是简单的。毕竟[手动可以滚启动](https://en.wikipedia.org/wiki/Push_start)。除了在湿滑的路面上刹车和祈祷，司机还有其他选择。任何一辆配得上它厚重的科林斯皮革(或者其他什么)的跑车都有手动变速箱，对吧？你也知道拉什的红酒不是自动档的。面对现实吧，换挡简直酷毙了。如果它能让你得到更多，这也不是一件苦差事，尽管[在这一点上燃料效率的事情是一个神话](https://www.edmunds.com/fuel-economy/five-myths-about-stick-shifts.html)。

你可以想象，当我想到在我有生之年的某一天，大多数汽车都将成为电动卡丁车时，我是多么恐惧。随着内燃机时代似乎接近尾声(不，这次是认真的)，只有一件事让大门敞开着——对手动变速器的热情。从奥迪到日产 Z，汽车制造商报告说，尽管丧钟已经敲响了 20 年左右，但手动变速器的使用率在美国相当高。本田思域有两款车型是手动专用的。这种现象也不仅限于跑车——2022 款福特野马(Ford Bronco)配备了七速手动变速箱，市场占有率超过 20%。

### 一台没有灵魂的新机器

对我来说，电动汽车似乎没有灵魂，我知道我并不孤单。在一辆普通的汽车里，你只是更了解正在发生的事情。有景有闻。噪音很多。我不是说电动汽车没有它们的感官标志，我只是相信它们是一个不同的品种。当然，这不是一个新品种——电动汽车几乎和燃烧式汽车一样早就出现了。但显然，这种情况正在发生变化，而且已经持续了大约 20 年。

> 我们不像对待手动变速箱那样热衷于换挡杆。—亨利·卡奇波尔，汽车记者

电动汽车可能很强大，有一吨即时可用的扭矩，但它只是不一样的体验。没有订婚，没有感觉你和车是一体的。除此之外，你多久在外面标记一次你的引擎或者测试 0-60？哦，从来没有？我也是这么想的。

实际上，这些都不重要，除非你是一名封闭赛道上的职业车手。正如 Jalopnik 主编 Bob Sorokanich 所说，“特斯拉拥有市场上最快的汽车——踩下油门，坚持住。这不需要任何司机技能。”

现在，这不就是总结吗？不需要任何驾驶技术。难道不是吗？在汽车出现的 130 年左右的时间里，有一件事我们没能让它变得更安全，那就是我们在路上的互动。当然，我们现在有速度限制，防滚架和安全带。但是我们比以往任何时候都更容易分心，而且我们还是人类。那么，开车真的需要成为一个方便的地方吗？我认为不是。
