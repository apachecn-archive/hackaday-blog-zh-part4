# 没有线的循线机器人

> 原文：<https://hackaday.com/2018/08/10/line-following-robot-without-the-lines/>

一般来说，循线机器人是对机器人学的一个很好的介绍，因为制造一个好的机器人所需的材料和技能并不太先进。事实证明，循线机器人也不仅仅是一种学习工具。它们在工业中非常有用，但大多数都不遵循可见的标记线。一些机器人，比如来自兰德尔的感应引导机器人，利用电线来确定它们的路径。

与物理线路相比，感应引导的一些好处是，电线可以隐藏在地板中，因此，如果像自动叉车这样的东西在仓库中使用它们，将会减少绊倒的危险和对引导的维护。它们还支持多条路径，因此无需进行复杂的曲目切换。[Randall]的机器人是他作为一名技术人员为一个自主导航车辆系统建造的一个更大系统的小型演示。他的视频详细介绍了它们的工作原理，更多的优点和缺点，以及其他一些东西。

虽然感应引导机器人已经使用了几十年，但它们开始被具有本地定位系统和计算机视觉的机器人所取代。我们最近已经看到为 T1 制造的机器人也利用了这些导航形式。
