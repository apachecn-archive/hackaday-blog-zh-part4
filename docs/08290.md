# 真正有用的机器人

> 原文:[https://hackaday.com/2020/11/12/really-useful-robot/](https://hackaday.com/2020/11/12/really-useful-robot/)

[詹姆斯·布鲁顿]是一位令人印象深刻的机器人专家，他制造了各种各样的机器人，从履带式、探险机器人到波士顿动力公司式的腿式机器人。然而，许多机器人都是探索机器学习、计算机视觉或独特运动和特征的概念验证构建。这个最新的构建利用了他从构建这些东西中学到的一切，但也努力在日常基础上有用，并且是他正在做的一系列[构建一个真正有用的机器人](https://www.youtube.com/watch?v=M9ABtlEM788)的开始的一部分。(视频，嵌在下面。)

虽然机器人还没有完全完成，但他在这个系列中的第一个视频探索了机器人本身的基础构建背后的想法。他希望这个机器人能够在环境中导航，同时也能执行指令，比如从桌子上拿取一个小物体。为此，它需要一个沉重的底座，由大型 3D 打印面板制成，配有两个带编码器的无刷电机，用于驱动定制车轮，以及由脚轮和特殊铰链制成的悬架。底座中还包括一台 Nvidia Jetson，用于运行机器人，并处理一些繁重的任务，如图像识别。

在撰写本文时，[詹姆斯]还发布了他的系列的第二个视频[，该视频详细介绍了机器人的地图和导航功能，我们很高兴看到成品。当然，如果你想看一些[詹姆斯]的其他项目，一定要看看他的](https://www.youtube.com/watch?v=dEPvZZ4luV4)[履带式漫游车](https://hackaday.com/2020/10/11/modular-rover-platform-rolls-on-3d-printed-flexible-tank-tracks/)或者他对[腿式机器人](https://hackaday.com/2020/03/22/compliant-quadruped-legs-using-servos/)的研究。
