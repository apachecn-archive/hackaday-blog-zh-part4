# 最奇怪的倒置 3D 打印机装在一个丝盒里

> 原文:[https://hacka day . com/2021/08/03/最奇怪的倒置 3d 打印机装在灯丝盒里/](https://hackaday.com/2021/08/03/strangest-upside-down-3d-printer-fits-in-a-filament-box/)

最近很少有新的 FDM 打印机引起我们的兴趣，但[Kralyn]的正电子成功做到了这一点。(视频，嵌在下面。)它可以上下颠倒打印，并装入一个灯丝盒中，同时仍然拥有 175 毫米 x 176 毫米 x 125 毫米的打印容量。

与大多数 3D 打印机不同，hotend 和 XY-gantry 安装在构建板下方，直接安装在底座上。你可能会认为打印机需要利用重力挤压塑料才能正常工作，但真正的动作是在塑料层的挤压中。似乎它甚至可以改善桥接，因为当塑料被挤出时，热端支撑着塑料。使用了透明的玻璃制造板，与大多数汽车后窗上的加热条相同。这也允许用户仍然看到部件，并且提供了能够快速发现底座水平和粘附问题的额外优势。

这种安排的另一个有趣的副作用是刚性。不需要将 XY 机架悬挂在空中，因此它可以直接安装在厚铝底板上。它使用了一个 H-bot 式的门架，用同步同步电缆代替皮带，消除了皮带扭曲的问题。为了在灯丝盒的尺寸范围内获得尽可能好的打印量，机架轴沿基板对角线排列。Z 轴可以断开连接，平放在打印机顶部，安装时使用线性导轨保持笔直垂直。

为了减少热端的高度以适应可用的空间，细丝从侧面进入，熔化的塑料在被挤出之前进行 90 度转弯。[Kralyn]做了一些修改才使它正常工作，但是大体积的熔化区允许增加流量。然而，我们怀疑这也可能使酒店和维护变得更加困难。我们唯一担心的是打印部分的可视性降低，以及塑料碎片可能会掉入机架机构并堵塞它。就紧凑型折叠打印机而言，这可能是我们迄今为止看到的最好的设计。设计在 [CC BY-NC](https://creativecommons.org/licenses/by-nc/2.0/) 下授权，视频描述中链接了所有设计文件。
