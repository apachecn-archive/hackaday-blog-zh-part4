# 一个单板机中两个处理器的 NEC V20

> 原文：<https://hackaday.com/2020/07/10/an-nec-v20-for-two-processors-in-one-sbc/>

当年，一名贫困学生在计算方面所能希望找到的最好的东西是一台废弃的 20 世纪 80 年代的 PC 克隆机，一次升级是安装 NEC V20 或 V30 处理器，取代英特尔 8088 或 8086。它是否提供了更多的边际优势仍有争议，但很可能该芯片的一项功能永远不会被使用。这些芯片不仅支持 8086 指令集，还提供了与老的 8080 处理器兼容的模式。这是[Just4Fun]利用的一个特性，用 [V20-MBC](https://hackaday.io/project/170924-v20-mbc-a-v20-8088-8080-cpu-homebrew-computer) ，一个既能运行 CP/M-86 又能运行 CPM/80 的单板计算机。

如果这开始看起来有点熟悉，那是因为我们以前推出过一些[Just4Fun]的主板。Z80-MBC2 使用相同的外形，像这个 V20 版本一样，它有一个更大的 ATMega 芯片，取代了 74 个芯片，如果它是在 20 世纪 80 年代初建造的，毫无疑问会执行同一台机器的所有胶合逻辑任务。我们在断点下方放了一段电路板运行视频，显示了 80、86 甚至 86 模式下的 80 仿真 CP/M。

V20 唯一一次出现在这里，是在一个更加传统的家里，一台自制的个人电脑。
