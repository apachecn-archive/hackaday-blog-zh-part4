# 老式语音合成器哼唱老歌

> 原文：<https://hackaday.com/2019/06/11/vintage-speech-synthesizer-croons-the-oldies/>

如果你在大约 25 年前收听过美国国家气象局的天气广播，你肯定会记得[完美保罗]，这是一种用于播报当前情况和天气预报的合成声音。这个声音来自 DECtalk DTC01，这是一个不便宜的声音合成器，1984 年首次制造，多年来也为[斯蒂芬·霍金]提供了声音。

DECtalk boxes 早已过时，拥有一批忠实的爱好者，他们喜欢扩展设备的功能。有些人甚至喜欢让它唱起来，在某种程度上，[迈克尔]认为让一个 DECtalk 唱“世外桃源”，1980 年[奥利维亚·纽顿-约翰]音乐盛会的主题曲，是一个好主意。它是否真的是有争议的，我们会反对让那首特别的小曲留在我们的脑海里，但我们不会做出判断，除非根据黑客的优点。

如果你有一个 DECtalk，这其实很容易；这首歌是一个简单的 ASCII 文件，带有非常简洁的指令，说明了这个盒子需要生成哪些音素。除了变调、音调和定时指令，文本文件看起来几乎完全不像英语，但仍具有一定的可读性。DECtalk 通过 RS-232 接受文件，这对于现代计算机来说是很容易做到的，但是[Michael]通过使用 TRS-80 100 型计算机作为串行终端，提高了他的水平。合成的歌曲在下面的视频中，原唱也包括在内，供那些没有经历过迪斯科年代后期辉煌日子的人参考。

在野外，甲板上说话的人似乎很少，所以我们欣赏这一瞥，看看他们能做什么。不过，还有其他复古的语音合成器技巧:我想到了模拟胡桃木的[Votrax](https://hackaday.com/2015/02/13/an-adventure-into-android-makes-the-vic-20-speak/)和[micro vox](https://hackaday.com/2017/02/22/microvox-puts-the-80s-back-into-your-computers-voice/)，还有[德高望重的 TI Speak and 拼写](https://hackaday.com/2012/02/09/midi-controlled-speak-and-spell/)。
