# 磁铁可以帮助假肢控制

> 原文：<https://hackaday.com/2021/08/21/magnets-could-give-prosthetic-control-a-leg-up/>

今天，假肢和外骨骼是用肌电图控制的。换句话说，通过记录肌肉收缩时的电活动。它既不直观，也不像人类，它真的只显示了大脑的意图，而不是肌肉正在做什么的现实。

麻省理工学院媒体实验室的研究人员想出了一种使用磁铁进行更精确控制的方法，他们称之为磁测微计(MM) 。通过植入一对微小的球形磁铁，并用磁传感器跟踪它们的运动，可以单独测量每块肌肉，比肌电图更精确。

在将直径为 3 毫米的球形磁铁植入火鸡的小腿后，研究人员能够在 3 毫秒内检测肌肉运动，精确度达到 37 微米，大约相当于人类头发的宽度。他们希望在未来几年内将 MM 用于人体试验。如果成功的话，这将是一个很好的解决方案，因为与肌电图方法相比，MM 更便宜，侵入性更小，并且可能是永久性的。将 MM 与一种名为 AMI 的新型截肢手术结合起来，这种手术可以提供更大的活动范围，更少的总体疼痛，以及更好的假肢控制，假肢和康复的未来看起来非常令人兴奋。休息后请务必观看视频。

控制假肢的方法不止一种，比如[深度学习](https://hackaday.com/2021/05/27/deep-learning-enables-intuitive-prosthetic-control/)和[体感刺激](https://hackaday.com/2021/06/30/adding-a-gentle-touch-to-prosthetic-limbs-with-somatosensory-stimulation/)。
