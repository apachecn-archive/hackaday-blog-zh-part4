# 3D 打印分类器通过摇动分离备用六角螺母

> 原文:[https://hackaday . com/2022/10/13/3d-printed-sorter-separates-spare-hex-nuts-with-a-shake/](https://hackaday.com/2022/10/13/3d-printed-sorter-separates-spare-hex-nuts-with-a-shake/)

有一堆混杂的六角坚果吗？通过打印[【jonafriedj】的坚果分类器](https://www.printables.com/model/289739-metric-hex-nut-sorter-v2)可以很快对它们进行分类，它有几乎任何你想要的坚果尺寸的选项(它标有公制，但实际上也包括英制尺寸。)

这个设计令人钦佩的地方是每个筛子上方便的小凸起标签，以及所有部件完全没有支撑就可以打印的事实。设计一个零件来发挥一种制造方法的长处(并避免它的短处)是好的 DFM，或者说是为制造而设计。

随着 3D 打印成为各地工作室和爱好者的福音，努力实现良好的 DFM 当然是值得的，特别是为了与他人分享的设计。有时，好的 DFM 借鉴了其他制造方法，如注射成型，我们最终会在打印零件上使用挤压肋。

想看看[jonafriedj]的坚果排序设计的演示吗？查看嵌入在分页符下方的短视频。如果这让你有所期待，那么[看看一个机械化、自动化的 DIY 解决方案](https://hackaday.com/2021/08/23/automatic-nut-sorter-for-a-tidy-workspace/)。
