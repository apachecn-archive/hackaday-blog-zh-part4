# 强力版权保护:解放所有旋律

> 原文：<https://hackaday.com/2020/03/05/brute-forced-copyrighting-liberating-all-the-melodies/>

坦率地说，音乐最终只是应用物理。和声遵循——取决于流派——或多或少固定的一套规则，在音乐本身的空间内可能存在有限的变化。因此，从技术上讲，可能的旋律只有这么多，这基本上是一个时间问题，直到歌曲作者或作曲家想出某个音符序列，而不知道他们不是第一个这样做的人，直到停止和终止信开始滚滚而来。

你可能会认为一首歌不仅仅是旋律，你是完全正确的。然而，目前的版权法和过去的法院裁决可能不太关心这一点。为了指出法律中的这些缺陷，拥有法律学位的音乐人 tech guy[Damien Riehl]和音乐人软件开发人员[Noah Rubin]聚在一起[简单地将每一个可能的旋律创建为 MIDI 文件](http://allthemusic.info/)，并在 Creative Commons Zero 许可证下发布。虽然他们目前的列表仅限于[几个固定长度的音阶](http://allthemusic.info/faqs/#What_have_you_created_so_far)，随着代码[在 GitHub](https://github.com/allthemusicllc) 上可用，这实际上只是对每一个可能的旋律进行强制的问题。

诚然，这样的旋律列表可能没有太多的实际用途，但对[达米恩]和[诺亚]来说，它更多的是法律和哲学方面的问题:音乐家不应该担心因为几个重叠的音符而被起诉。因此，虽然这份清单是他们放在公共领域的“安全旋律集”，但他们更大的目标是从数学上指出音乐的有限空间，这些空间本来就不应该受到版权保护。他们肯定说得有道理——想象一下，如果你可以对和弦进行申请版权并提起诉讼，那么今天的音乐会是什么样子。
