# 3D 打印难题的复杂计算解决方案

> 原文:[https://hack aday . com/2022/02/13/complexed-calculated-solution-to-3d-printed-puzzle/](https://hackaday.com/2022/02/13/complicated-calculated-solution-to-3d-printed-puzzle/)

3D 打印机已经使许多事情成为可能，这些事情用传统工具是极其困难或者完全不可能的。某些形状有助于 3D 打印，并且材料和工具成本通常也大大降低。然而，有一件事可能不会经常被提及，那就是他们快速原型化复杂数学问题解决方案的能力，在这种情况下，他们采用 3D 打印迷宫的形式，[被称为十二面体全息迷宫](https://www.youtube.com/watch?v=wzjUTPCAF4Y)，有一个有趣的解决方案。

拼图本身是一个由各种镶嵌的六边形组成的球体，形成拼图块或“车”的轨迹。这些轨迹为车的行进创造了迷宫，因为当车以特定方式定向时，一些路径被阻挡。要解决这个难题，玩家必须通过围绕六边形移动车来旋转车，这样它的路径就不会被任何钉子物理地阻挡，以便成功地到达出口。从表面上看，这似乎是一个有趣的玩具，但令人印象深刻的是，这些解决方案旨在减少用任何“暴力”方法解决难题的可能性，同时有不止一条路径可以到达出口，以及解谜者必须穿越的几个瓶颈。

实际上有许多可能的谜题可以以这种大小和形状产生，并且都有预先确定的解决方案和巧妙选择的路径。这可能看起来很多，但当你意识到从概念到 3D 建模到实现的整个构建都是由[Henry Segerman]和俄克拉荷马州立大学的一群其他数学家完成的，它开始变得更加清楚这个谜题是如何精心设计的。事实上，我们过去也展示过[他的其他一些数学建模作品](https://hackaday.com/2022/02/02/3d-printed-circular-prototype-performance-prop-captivates-circus-spectators/)。
