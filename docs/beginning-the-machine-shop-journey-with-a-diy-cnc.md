# 用 DIY CNC 开始机械商店之旅

> 原文：<https://hackaday.com/2021/08/19/beginning-the-machine-shop-journey-with-a-diy-cnc/>

建设一个高质量的机械车间似乎是一个先有鸡还是先有蛋的问题，至少对于那些不愿意抵押房屋来购买所有这些新工具的人来说是这样。也就是说，构建好的工具通常需要好的工具。为了帮助解决这个问题，[【Ryan】设计并制造了这台数控机床](https://www.rigcnc.com/)，这台机床除了普通的工具、五金店的供应品和一些从网上容易买到的零件之外，什么都不用。

由于它是由消费级材料制成的，[Ryan]拥有“购买精度”的设计理念，这意味着该产品所需的大多数零件都足够精确，无需在融入工厂之前进行任何加工。例如，他使用花岗岩板，因为它坚硬、平坦、沉重，在购买时足够坚固，可以立即放入机器中。同样，他的线性导轨在投入使用前不需要修改，具有很高的精度和最小的校准。从那以后，他运用接吻原则，使用最简单的部件。有了这个设计过程，他能够以大约 1500 美元的价格“启动”一个高质量的工厂，而不需要任何额外的工具。

众所周知，RIG-CNC 也是完全开源的，这进一步巩固了它的可引导性，在项目页面和下面链接的视频中有更多的细节。这个项目是独一无二的，不仅仅是因为它是用普通的零件和工具建造的，还因为它的设计理念是如此的强大。好的设计在我们的构建中走得比我们许多人可能意识到的要远得多，而且好的设计通常会产生更易维护、更易梳理的东西，这些东西的用途比最初的创造者甚至可能想都想不到的还要多。
