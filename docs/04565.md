# 这些受枫叶启发的无人机无声地携带有效载荷

> 原文:[https://hack aday . com/2019/11/02/these-maple-pod-inspired-drones-silent-carry-payloads/](https://hackaday.com/2019/11/02/these-maple-pod-inspired-drones-silently-carry-payloads/)

来自新加坡技术与设计大学(SUTD)的研究人员最近发布了一段视频，展示了他们的[自然灵感无人机](https://www.hackster.io/news/samara-inspired-drone-can-break-into-smaller-drones-for-increased-travel-distance-4967c9f17ade)，它能够分解成五个独立的较小无人机。每架无人机都有自动旋转的翅膀，可以降低下降速度，类似于枫树的豆荚。由于其设计，无人机只能用于单程旅行，五个组件各自携带一个独立的有效载荷。无人驾驶飞机被设计成在距离目的地特定距离内分离，允许集体身体安全地向下螺旋降落。

在他们发表的关于同一主题的论文中，研究人员讨论了他们如何通过伺服系统、脂肪电池和连接到 3D 打印身体的接收器来优化轻木翅膀。四个配备了这些组件，而第五个还拥有一个 3 轴磁力计，一个 Teensy 3.5 板，一个 GPS 模块和一个 Pixracer 控制器。

他们试验了几种动作捕捉设置和自由飞行跌落测试，以验证他们在无人机模型上的模拟。除了简单的分离之外，它们还被设计成根据它们所处的环境来适应不同的任务剖面。

我们将承认无人机的实现和设计确实看起来相当反乌托邦，特别是当你想知道这些无人机被设计用来携带什么样的有效载荷时。但是就受自然启发的机器人而言，枫树种子舱的想法非常有趣。
