# 给你的模拟站多加点料

> 原文：<https://hackaday.com/2022/06/22/putting-a-little-more-juice-in-your-emulation-station/>

在你构建了一个时髦的基于 Raspberry Pi 的复古游戏主机后，你可能会想，你是否可以有一点点更强的能力，运行一些你可能还记得的其他游戏，如 Xbox、Wii 或 PS3。也许在未来，RPi 的后续版本可以处理它，但是目前，为了模仿第 6/7 代游戏机，你需要一些更强大的东西。幸运的是，[扎克]得到了一台旧游戏笔记本电脑，并把它变成了他自己的游戏主机。

第一步是把笔记本电脑拆开，扔掉不需要的部分。[Zac]卸下电池、蓝光驱动器和旋转硬盘。这就给他留下了一个更小的 PCB，可以装进一个小盒子里。电源按钮集成在键盘中，但通过扁平电缆键盘连接进入主板。所以通过桥接几个引脚，他可以启动笔记本电脑。接下来，他升级了内存、wifi 卡和 NVMe 硬盘，并重做了所有的导热膏和油灰，试图在超频 GPU 的同时保持冷却。

机器的外壳大量使用了他的 CNC，因为它是胡桃木的，中间部分由胶合板制成。顶部有一个华丽的丙烯酸铸造窗口，可以看到里面。[Zac]担心的部分是细间距焊接。最终，他将两根电线连接好，没有桥接。因为它的核心只是一台 PC，所以几乎每个游戏都在桌面上。仿真，一些更温和的电脑游戏，从他的办公室电脑流，以及云游戏服务，让他能够访问大多数游戏。我们喜欢这个概念和想法。

我们喜欢这种建筑的美感，但是如果你喜欢让你的游戏机看起来更真实一点，为什么不[把你的迷你电脑放在一个真正的 N64 盒子里](https://hackaday.com/2021/11/18/n64-mini-pc-conversion-includes-all-the-trimmings/)？休息后的视频。
