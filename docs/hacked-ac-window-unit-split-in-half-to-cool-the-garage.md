# 被黑的空调窗户单元裂成两半来冷却车库

> 原文：<https://hackaday.com/2021/06/24/hacked-ac-window-unit-split-in-half-to-cool-the-garage/>

对于我们这些北半球的人来说，现在已经进入了炎热的夏季，对于许多 Hackaday 的读者来说，这意味着是时候把旧的窗式空调从仓库中拿出来，并把它拖回原位了。但是如果你想给一个没有方便窗户的空间降温，该怎么办呢？在这种情况下，[【红外线】想出的这个聪明的转换来保持他的车库凉爽](https://www.youtube.com/watch?v=TSC6wLcr4VU)可能会很有趣。

基本上，他把经典的窗式空调改造成了即兴的无管道装置。通过将蒸发器线圈旋转到垂直位置并延长压缩机电线，他能够将空调的中心做得足够薄，这样他就可以在上面关上车库门。该单元的背面看起来很大程度上没有受到影响，但正面有一个真正的*疯狂的麦克斯*盛传；金属板，裸露的电线，还有几个风扇。适合车库或工作区，但可能不是孩子房间的好选择。

[Infrared]说被黑掉的空调可以让他的车库在目前的状态下比室外气温低 18 度，但他希望增加一些高 CFM 电脑风扇不仅能提高性能，还能让他让新的前面板看起来更整洁。尽管以目前的形式来看，[这远不是我们近年来见过的最荒谬的 DIY AC 项目](https://hackaday.com/2021/06/21/even-with-ac-hazmat-suit-isnt-really-cool/)。
