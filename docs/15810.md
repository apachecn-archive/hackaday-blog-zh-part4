# 用极简主义的 Javascript 代码创作的音乐

> 原文:[https://hack aday . com/2023/01/02/generative-music-created-in-minimalistic-JavaScript-code/](https://hackaday.com/2023/01/02/generative-music-created-in-minimalistic-javascript-code/)

Dittytoy 用户[srtuss]用优雅的 19 千字节 Javascript 代码重新创作了最有影响力的电子音乐作品之一。在 Dittytoy 平台上重新制作的《让·米歇尔·雅尔的氧气》第四部目前处于测试阶段，可以在你的浏览器中实时播放。 [Dittytoy](https://dittytoy.net/) 让用户能够使用简单的 Javascript API 在线创作音乐。API 的语法大致基于 [Sonic Pi](https://sonic-pi.net/) 的语法，这是一个基于代码的音乐创作和演奏工具。

《氧气(第四部分)》是让·米歇尔·雅尔在 1976 年录制的。这是 Jarre 最成功的单曲，在几个国家的排行榜上名列前十，最近还出现在《侠盗猎车手 4》视频游戏中。在 20 世纪 90 年代，著名的电子音乐创新者布赖恩·伊诺用“再生音乐”一词来描述由电子系统产生的音乐，这些电子系统包含不断变化的元素，这些元素可能是算法产生的，也可能是随机产生的。

重新创作 Jarre 的作品需要模拟 Korg Minipops 7 鼓机，这是我们的[系列开源合成器](https://hackaday.com/2016/02/23/a-slew-of-open-source-synthesizers/)中的一种乐器。
