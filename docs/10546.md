# 铸造一个简单的铝 3D 打印

> 原文：<https://hackaday.com/2021/06/25/casting-a-simple-3d-print-in-aluminium/>

使用塑料和树脂的 3D 打印非常适合快速成型各种几何形状的零件，但生产的零件的强度和耐用性往往有限。解决这个问题的一个方法是使用 3D 打印的零件作为模型来铸造更坚硬的东西，比如铝。[这正是[Brian ol trodge]所做的，他用 3D 打印设计制作了一个吸引人的墙钩。](https://www.youtube.com/watch?v=9AaujAPghhE)

这个过程从设计和打印一个墙钩开始，【Brian】小心翼翼地将合适的拔模角度包括在内，以使模型能够正确地从模具中取出。印刷品经过仔细打磨和后期处理，非常光滑，因此在铸造过程中移除时不会损坏模具。从那里，使用硅酸钠以 3-4%的重量比与细砌筑砂混合，围绕模型建立砂铸模。一旦准备好了，模型就被移走，模具就组装好了，准备浇注。

[Brian]使用熔融铝通过简单的重力铸造方法完成了这一过程。然后将零件从模具中取出，并锉平以改善砂型铸造过程的表面光洁度。然后把它擦亮，挂在墙上。

[Brian]很好地解释了重力铸造的基本原理；尤其是拔模角度是初学者经常忽略的东西，但对于获得好的结果却是至关重要的。不过，你不必满足于铸造无生命的物体；[我们以前也展示过齿轮的 DIY 铸造工艺](https://hackaday.com/2019/11/21/casting-gears-at-home/)。休息后的视频。
