# 复古游戏的无线控制器

> 原文:[https://hack aday . com/2019/04/08/wireless-controllers-for-retro-gaming/](https://hackaday.com/2019/04/08/wireless-controllers-for-retro-gaming/)

通过各种经典平台，比如 NES 经典，可以制造出无限的怀旧情绪。旧的书籍仍然非常受欢迎，把它们放在一个现代的包装中使它们更容易获得。另一方面，如果您仍然有原来的硬件，事情可能会变得繁琐。不过，借助现代技术，有可能做出一些改变，就像[PJ·艾伦]在他的准将 64 上增加无线功能。

当这个系统还被认为是“现代”的时候，[PJ]试图通过 FM 无线电使用 DTMF 来制造一个无线控制器。他不能让它完全正确地工作，最终搁置了这个项目，直到今天。现在，我们有了比模拟无线电更多的工具，所以他拿出了一个 Arduino 和几个蓝牙模块。不过，让旧硬件与现代设备配合需要一点技巧，但一旦[PJ]解决了这些问题，他就能够不受有线控制器的限制，玩他的经典游戏，如 Defender。

Commodore 64 在 20 世纪 80 年代和 90 年代初非常受欢迎，至今仍能看到它的影响。人们正在[建造全新的机器](https://hackaday.com/2019/03/12/its-raining-brand-new-commodore-64s/)，为他们建造[模拟器](https://hackaday.com/2018/05/07/raspberry-pi-becomes-cycle-exact-commodore-drive-emulator/)，或者[升级他们的硬件](https://hackaday.com/2019/03/23/make-your-commodore-16-64k-but-not-a-commodore-64/)。
