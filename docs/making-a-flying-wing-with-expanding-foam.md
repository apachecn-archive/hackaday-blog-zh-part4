# 用膨胀的泡沫制作飞翼

> 原文：<https://hackaday.com/2019/04/21/making-a-flying-wing-with-expanding-foam/>

许多无线电控制飞机模型制作者会熟悉用热线法从泡沫中切割出机翼的过程。这些工具很简单，可以在家里制作，而且这是一种没有太多麻烦的生产一套轻质翅膀的简单方法。然而，[IkyAlvin]走了一条不同的路(YouTube 链接，嵌入在下面)。

膨胀泡沫是这里的关键——罐子里奇妙的粘性物质永远不会到达你想要的地方。中密度纤维板和泡沫被用来制造一个模具，以生产机翼的形式。接下来的事情很简单，将地板垫放入模具中充当外壳，然后用膨胀泡沫填充模具，等待其固化。

最后的零件被组装成一个飞翼，第一次试飞非常成功。使用泡沫覆盖层作为蒙皮还有一个额外的好处，就是为飞机提供光滑的银色表面。这表明，在主流技术之外，总有探索替代技术的空间。如果你想更熟悉经典的热线技术，[我们也可以提供帮助](https://hackaday.com/2018/02/08/build-a-tiny-hot-wire-foam-cutter/)。休息后的视频。
