# 反应式像素灯可根据指令产生丰富多彩的氛围

> 原文：<https://hackaday.com/2020/08/24/reactive-pixel-lamps-create-colourful-vibes-on-command/>

菲利普斯流光溢彩技术是一个奇怪的东西，由于其专有性质，从未在主流中流行起来。它由位于电视屏幕后面的 LED 阵列组成，可以投射与屏幕内容相关的颜色，以营造更好的氛围。[【Ed Chamberlain】的反应像素灯旨在以更分散的方式做同样的事情。](https://www.instructables.com/id/Reactive-Pixel-Lamps/)

每个像素灯由一个配有老式 4 线 RGB LED 的 Wemos D1 控制器组成。这些组件被放置在一个 3D 打印的半透明立方体中，作为一个有吸引力的外壳和扩散器。通过船上的 WiFi 连接，可以将单个立方体连接到一个树莓 Pi 上，作为菲利普斯色调桥，这要归功于 [DIYHue](https://diyhue.org/) 。设置完成后，可以在 Phillips Hue 应用程序中将灯光配置为流光溢彩系统。

这是一种令人印象深刻的方式，在预算范围内给房间提供反应式照明，而无需求助于昂贵的现成解决方案。我们希望看到这一点进一步扩大，因为我们确信一个充满反应灯的房间将是真正值得一看的景象。重建流光溢彩技术的其他方法也是可行的。休息后的视频。
