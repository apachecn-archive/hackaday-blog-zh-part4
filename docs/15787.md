# 一个有趣的低成本开始为您的宏帕德爱好

> 原文：<https://hackaday.com/2022/12/30/a-fun-low-cost-start-for-your-macropad-hobby/>

如果你曾经在寻找一个可以日常使用的小型放松的晚上项目，你必须考虑由[Natalie the Nerd] 制作的 [Pico Hat Pad kit。它完全符合 Pi Pico 的外形规格，为您提供了两个按钮、一个旋转编码器和两个可单独寻址的 led。最初，这款 macropad 是一款价格不到 20 美元的设备，也是一款焊接练习套件，而[Natalie]已经让它出类拔萃了。](https://github.com/nataliethenerd/picohatpad)

你用一叠三个 PCB 构建这个 macropad 中间一个将 Pi Pico heart 连接到按钮、编码器和 led，其余的增加结构支撑和保护。所有的 PCB 组合在一起成为一个整洁的标签连接面板-准备扔进你最喜欢的 PCB 服务的购物车。在引擎盖下，这个 macropad 使用 KMK，一个基于 CircuitPython 的键盘固件，配置开源。事实上，一切都是开源的，就像我们喜欢的那样。

如果你在组装了这个之后发现自己对 macropads 有着意想不到的亲和力，不要慌。这是很常见的副作用。幸运的是，有治愈的方法，你不再不可避免地会为此发疯。也就是说，如果你想变得更大，你可以试试[一个不同的手动有线微型宏键盘](https://hackaday.com/2022/09/03/pretty-pico-macropad-eases-transition-back-to-office/)。来找那个还不够吗？这里有一个 [2×4 的 3D 打印的](https://hackaday.com/2020/05/13/circuitpython-macro-pad-is-one-build-that-wont-bite/)。

现在，如果你最终发现自己在《与克里斯蒂娜一起的柯宾》第一集播出时就阅读了每一集，你可能已经走得太远了，我们很快就会发现你花费数百美元[将微小的有机发光二极管屏幕制作成单独的按键](https://hackaday.com/2020/12/17/adaptive-macro-pad-uses-tiny-oled-screens-as-keycaps/)——在这种情况下，请确保你记录下来并且[与我们分享！](https://hackaday.com/submit-a-tip/)
