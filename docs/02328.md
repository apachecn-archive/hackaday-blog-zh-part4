# 我们见过的最奇怪的游戏机模拟器

> 原文:[https://hack aday . com/2019/03/06/the-strangest-game boy-emulator-we ve-seen-yet/](https://hackaday.com/2019/03/06/the-strangest-gameboy-emulator-weve-seen-yet/)

在秘密的 Hackaday bunker 中，我们有一些 emacs 用户，一些 vi 用户，以及一些并不真正关心的人。然而，即使是我们最坚定的 emacs 支持者也不得不对[Vreeze 的]项目感到惊讶，这个项目使用古老的文本编辑器创建了一个 GameBoy 模拟器。你可以在下面的视频中看到阿列克谢·努内斯对模拟器的反应。

Eboy 使用 unicode 字符来输出图形。您可以使用 emacs 命令来加载 ROM 映像，并使用键盘来控制游戏。

[Alexi 的]视频显示了与官方文档略有不同的分步说明。你可以看他开始玩俄罗斯方块，虽然并不完全成功。到目前为止，并不是每款游戏都能成功，而且[Vreeze]指出这是一项正在进行的工作。

当然，这是可能的，因为 emacs 实际上只是一个 Lisp 解释器，恰好具有许多旨在提供文本编辑器服务的特性。人们用它来做各种事情，从电子邮件客户端到网络服务器。

我们看到许多用于放置树莓面板等物品的游戏机盒。如果他们运行 emacs，那会很有趣，但可能会影响性能。如果你进入徽章生活，你可以得到一个 [Lisp 徽章](https://hackaday.com/2019/01/10/program-this-badge-in-lisp/)，如果你不介意很多括号的话。
