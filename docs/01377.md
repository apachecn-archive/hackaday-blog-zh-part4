# 使用矿物油进行极限 Pi 超频

> 原文:https://hackaday . com/2018/11/30/extreme-pi-使用矿物油超频/

液体冷却是一种让你的电脑获得额外性能的流行方式。通常这是在台式机上完成的，在台式机上，一个带有铜管的特殊散热器被粘在 CPU 上，铜管被连接到散热器上。如果你想更深入地了解液体冷却的世界，你也可以像[Timm]所做的那样，将整个电脑浸泡在矿物油浴中。

这里提到的电脑是一个树莓派，它被装在一个特制的充满矿物油的激光切割丙烯酸外壳中。作为一个 SoC，淹没整个计算机比为处理器安装一个小型液冷散热器更容易。虽然我们以前也见过类似于这种的[其他版本，但【Timm】采取了一种不同的方法来通过油浴访问 GPIO、USB 和其他连接器。端口从电路板上拆焊，并焊接一个专用接头。从那里，电线可以从液体中取出并密封。](https://hackaday.com/2017/05/15/liquid-cooling-overclocked-raspberry-pi-with-style/)

这里使用的另一个细节是我们以前在这样的版本中没有见过的，那就是“环绕”通常用于 GPIO 的扁平带状电缆。在 IDE 电缆时代，通常是将单根电线分开，然后重新捆扎成圆柱形。现在 SATA 更受欢迎，这种做法已经基本被遗忘，但在这个建筑中[Timm]使用它来改善矿物油循环，使建筑更容易管理。
