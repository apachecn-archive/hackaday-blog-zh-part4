# 超音速射弹超越工程师的梦想:超音速投石机

> 原文：<https://hackaday.com/2021/12/01/supersonic-projectile-exceeds-engineers-dreams-the-supersonic-trebuchet/>

你有没有坐下来想过“我想知道投石机是否能以超音速发射弹丸？”我们也没有。这就是[大卫·艾德]与众不同的地方。他不只是问了问题，他还回答了问题！他在 YouTube 视频[中记录了整个建造过程，你可以在视频下方看到。](https://www.youtube.com/watch?v=gdXOS-B0Bus)

投石机是一种弹射器，在火药出现之前，它被广泛用作攻城工具。投石机使用长臂，比传统的弹射器投掷得更远。焦点通常集中在增加射弹尺寸的投掷距离上，反之亦然。当然，你在这里是为了阅读投石机的另一个用途:速度。

多快才算快？超过每秒 450 米的爆裂声和音爆速度怎么样？他是怎么做到的？为了安全起见，大部分是木头和橡胶，还有一些金属碎片。[David]的视频完整地解释了他的投石机的所有设计，比你想象的要少得多。有一个非常令人满意的全功率投石机发射的蒙太奇，这使得人们可以清楚地听到被投掷的弹丸远远超过了音速，报告非常类似于小型步枪的报告。

[大卫]令人印象深刻的项目和演示表明，所有人必须做的是尝试建立一个超音速投石机。小心点，在你把别人的眼睛弄瞎之前，看着你开枪的地方，好吗？

说到速度出乎意料的东西，[看看这些无动力遥控滑翔机，它们的速度接近音速](https://hackaday.com/2020/09/24/dynamic-soaring-545-mph-rc-planes-have-no-motor/)离地面只有几英尺。感谢[基思]给了[这么棒的提示！](https://hackaday.com/submit-a-tip/)
