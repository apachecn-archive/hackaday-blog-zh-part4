# 同轴连接器，从 PL259 开始

> 原文:[https://hack aday . com/2020/11/23/coaxial-connectors-starting-with-the-pl 259/](https://hackaday.com/2020/11/23/coaxial-connectors-starting-with-the-pl259/)

对于拥有一个微控制器和一些外设的普通试验板实验者来说，不太需要考虑互连，因为一组跳线就足够了。但是正如任何一个无线电爱好者会告诉你的那样，在更高的频率上，情况就完全不同了。[Ria Jairam N2RJ]拍摄了一系列探索同轴电缆及其各种连接器的视频，她的第一个产品是不起眼的超高频连接器，有时被称为 PL259。尽管这是一个较老的选择，其设计缺陷意味着“UHF”对它来说更像是一个愿望而不是现实，但它仍然是业余无线电频率范围低端的一个常见连接器。

在介绍超高频连接器之前，她从同轴电缆的简史开始。我们被介绍到它的主要缺陷，因为它不呈现恒定阻抗。由此产生的失配给发射机带来了严重的问题，尤其是在较高频率下。然后，我们介绍了各种不同型号的超高频连接器，包括那些带有蜂窝电介质以最大限度地减少失配的连接器，以及带有应变消除功能的昂贵插头。最后，她带我们正确安装 PL259 插头，这比我们大多数人想象的要复杂一些。总的来说，这是一个有趣的和信息丰富的手表，由一个迷人的和知识渊博的主人，我们期待着更多。

同时，射频连接器领域是我们自己不时涉足的领域。
