# 紫外线监测虎皮鹦鹉密切关注暴露水平

> 原文：<https://hackaday.com/2022/10/19/uv-monitoring-budgie-keeps-an-eye-on-exposure-levels/>

紫外线非常有助于我们生成维生素 D，但它们也是有害的，会导致晒伤甚至黑色素瘤。为了帮助孩子们了解当地的紫外线指数，[【Jude Pullen】创造了紫外线虎皮鹦鹉。](https://www.instructables.com/Build-a-UV-Level-Monitoring-Budgie-Using-IoT-and-W/)

该构建基于 Arduino Nano 33 物联网板，该板查询气象局的 API 以确定该地区的紫外线水平。相关数据随后显示在一个小型电子墨水显示器上，可爱的小太阳字符告诉你当前的情况。它还通过录制的声音样本宣布当前的风险水平，建议是否应该采取预防措施，如使用防晒霜或在最糟糕的日子里躲在室内。此外，有一只鸟扇动翅膀宣布更新，由基地里的一个小伺服驱动。

这是一个有趣的建筑，应该有助于[裘德]和他的家人在夏天保持阳光安全。[Jude]注意到该构建也可以被重新编程以共享其他警告。查询当地空气质量(T1)或 T2 辐射水平(T3)的 API 只是想到的一些想法。休息后的视频。
