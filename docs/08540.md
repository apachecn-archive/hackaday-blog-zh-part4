# 用手摇光彩制成的小链环栅栏

> 原文：<https://hackaday.com/2020/12/06/tiny-chain-link-fence-made-with-hand-cranked-brilliance/>

链节栅栏是用一种看起来近乎催眠的机制编织而成的，所以[Levsha]决定制作自己的[微型手摇桌面版](https://www.youtube.com/watch?v=xA9wG4eBfuQ)来制作微型铜线栅栏。

链节由一系列弯曲并编织成之字形的金属丝组成。之字形弯曲是通过将线缠绕在固定管内的旋转平板上制成的，固定管侧面有螺旋槽以保持弯曲的间距一致。【Levsha】的版本大致是实物的 1/10 比例，只做弯曲和缠绕部分。将弯曲的电线连接在一起取决于操作者。所有部件都在车床和 CNC 路由器上加工，并在木质底座上进行精美的加工和组装。最难的部分是有螺旋槽的管子，试了几次才弄对。【Levsha】最初尝试用钢丝，但是钢丝太硬导致上弦机构锁死。0.4 mm 铜线原来是最好的选择。

虽然这种设备没有我们可以看到的实际用途，但工艺非常出色，而且它是那些提醒我们多么想要一些机床的视频之一。

对细节的细致关注才是让这样的视频看起来令人愉快的原因。我们见过其他一些这样的项目，像漂亮的临时搭建的车床，或者不能飞的气动无人机。
