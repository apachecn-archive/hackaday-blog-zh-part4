# 蒸汽朋克摩托车靠压缩空气运行，是纯粹的黑客艺术

> 原文：<https://hackaday.com/2020/01/06/steampunk-motorcycle-runs-on-compressed-air-is-pure-hacking-art/>

有时候为了美观牺牲一些实用性也是可以的，尤其是激情项目。彼得·福斯伯格设计的漂亮、几乎不起作用的蒸汽朋克摩托车稳稳地落在了这个类别中。如果这不是黑客艺术，那么我们不知道什么是。

摩托车最引人注目的部分是发动机和传动系统，大部分机械部件都可以看到。气缸是透明的玻璃管，带有定制的活塞、密封件、阀门和推杆。曲柄机构来自一辆老式哈雷摩托车，安装在一根不锈钢管内。因为它依靠压缩空气运行，它会冷却而不是加热，所以不需要油系统。

为了转向，自行车的整个前部在框架中间的铰链上左右摆动，这对于以仅高于步行速度的最高速度骑行来说非常棘手。它可以在坦克上运行大约 3-5 分钟，所以[彼得]在框架上安装了一个大的三分钟沙漏。发动机由他背在背上的外部空气罐供给燃料；他承认，无论带着这个东西多久，都是一种近乎折磨的行为。他计划建造一个侧车来容纳一个更大的坦克，以扩大范围和提高乘坐舒适性。

[Peter]承认，作为一辆摩托车，它并不太好，但让它发挥作用所需的创造力和足智多谋是一名真正的机械黑客的标志。我们期待着看到它的最终形式。
