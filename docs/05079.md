# 忙碌的盒子打败了宝宝的无聊

> 原文：<https://hackaday.com/2019/12/16/busy-box-beats-babys-boredom/>

我们很想在圣诞节早上看到[Wilksyonreddit]的孩子撕掉这个成人级别的繁忙盒子上的纸。你能想象那种无拘无束的喜悦吗？这肯定会让六个月的艰苦建设变得值得。希望墙是坚固的，因为我们预测它们会被弹开。

这种不断给予的礼物内部有一个 Arduino 巨型克隆和几个移位寄存器来处理所有这些按钮和开关。除了所有的按钮、开关和数字键盘，还有两个 3D 打印的触摸感应垫，可以检测到四英寸远的小指。虽然他已经为它开发了一些游戏和活动，但[Wilksy]还是在 [r/duino](https://www.reddit.com/r/arduino/comments/e6z24l/help_needed_ive_build_a_button_box_for_my_kid_i/) 上发布了这个，寻找更多的想法。在 baby 的第一个核导弹发射控制台上有很多工作要做，包括输入和输出。我们谦恭地建议 4D·西蒙，尽管我们必须承认对 MIDI 控制器的幻想。

藏在这个圣诞礼物里的是一个复活节彩蛋，我们认为你会喜欢的。输入正确的代码，盒子就变成了《回到未来》的音效和音频剪辑宝库。广告之后是视频，小飞侠。

这个盒子也会成为一个很好的 Kerbal 空间程序控制器，就像这个。
