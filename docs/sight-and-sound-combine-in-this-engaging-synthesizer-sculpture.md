# 视觉和声音结合在这个迷人的合成器雕塑中

> 原文：<https://hackaday.com/2022/10/23/sight-and-sound-combine-in-this-engaging-synthesizer-sculpture/>

我们永远对赛道雕塑项目情有独钟；任何由整齐排列的黄铜线支撑的部件总能激发我们的想象力。但是再加上一点光和大量的声音，你就会得到类似于[这个混合合成器雕塑](https://hackaday.io/project/187914-corwin-point)的东西，它确实吸引了人们的注意力。

[Eirik Brandal]称他的发明为“corwin point”，并将其描述为“一个生成式双语音模拟合成器。”它是由一个完全开放的建筑构成的，这种建筑吸引了人们的探索，也吸引了人们的眼球和耳朵。雕塑的最底层有所有“无聊”的数字东西——一个 ESP32、LED 驱动器和数模转换器。下一个层次是视觉上更有趣的模拟电路，主要建立在黄铜线框架上的“死虫”风格。用户界面，主要是一系列的电位计和开关，位于这一层，seed studio WIO 终端也是如此，它用于显示所产生声音的频谱分析仪。

再往上一点，有一个看起来不协调的真空管超速驱动器，还有一个功率放大器和一个丙烯酸外壳的扬声器。一个厚丙烯酸树脂的垂直元素矗立在整个建筑上，容纳了合成器的延迟线，以及随着音序器事件及时穿过雕塑脉冲的光导管。下面的视频展示了 synth 的运行——它生成的音乐听起来从来没有两次是一样的，听起来也不像我们以前听过的，除了当我们听到类似洛根的跑步的背景音乐的时候。

向[Eirik]致敬，这是另一个看起来很棒、听起来很棒的版本；你可能还记得他的[“cwymriad”](https://hackaday.com/2022/02/18/sound-and-light-play-off-acrylic-and-wire-in-this-engaging-circuit-sculpture/)在今年早些时候引起了我们的注意。
