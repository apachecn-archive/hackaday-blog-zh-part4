# 蘸墨机节省大量时间

> 原文：<https://hackaday.com/2019/06/26/ink-dipping-machine-saves-iotas-of-time/>

[Uri Tuchman]并不总是用蘸笔写字。但是当他这么做的时候，他很快就厌倦了重新上色。现在，便利有多种形式。当然，他可以改用任何数量的现代书写工具。但那将是把婴儿和洗澡水一起倒掉。

[Uri]认为旧的问题需要旧的解决方案，我们完全同意。他的机器是一个令人惊讶的制作精良的自动装置，将一支手工制作的钢笔浸入墨水中，通过转动曲柄和轻推凸轮来抖落多余的墨水。我们喜欢手工雕刻的爪子，当它在定制的黄铜铰链上优雅地移动时，看起来非常荒谬。

我们有点惊讶，考虑到所有这些工作，[尤里]没有磨自己的笔尖或自己制作墨水。但这将减少他在等待湿羽毛笔时不得不手写信件的时间。暂停一下，看看[Uri]为这台令人敬畏的机器制作的非常有趣的视频。
