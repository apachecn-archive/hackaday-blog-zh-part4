# M4 面包棒是一个美味的原型替代品

> 原文：<https://hackaday.com/2021/09/06/m4-breadstick-is-a-tasty-prototyping-alternative/>

“试验板友好型”对你来说意味着什么？这已经成了一个生存能力最低的游戏。当然，它适合试验板，但有没有任何可访问的连接点留给电线和元件？如果不能用开发板轻松原型化，开发板有什么用？

几年前，[迈克尔·兰根]开始改变这一切，他创造了一种细长的开发板，它隔开了 I/O 引脚并简化了布线，使每个电路都很漂亮，更容易视觉化。目前的版本改编自 Adafruit 的《M4 快车》。它有 20 个 I/O 引脚，全部沿长度方向展开，并像集成电路一样在喇叭周围编号。[迈克尔]把这个面包棒浸在 24 个小小的 RGB 发光二极管里，所有这些都在专用的通讯总线上。

我们认为这是一个伟大的想法，这肯定会使微控制器更容易被黑客攻击。这种类型的布局将使检查学生的工作变得轻而易举，你可以在课后自己制作整洁的原型。今天开发板运行 CircuitPython，将来也能运行 Arduino。

[这种电潜泵插座可能不会留下太多的连接点，但这比焊接接头容易得多](https://hackaday.com/2021/01/10/esp8266-socket-is-a-snap-fit-breadboard-friendly-wonder/)。
