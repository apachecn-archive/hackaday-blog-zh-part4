# 振动隔离有助于改善 FPV 视频信号

> 原文：<https://hackaday.com/2021/03/25/vibration-isolation-helps-improve-fpv-video-feed/>

第一人称视角技术在 RC 社区变得非常受欢迎，让用户获得一个有利位置，就好像他们实际上在他们的微型车辆中一样。不过，很难获得良好、干净的视频信号，尤其是在动力传动系统振动较大的车型中。[下班后的工程]决定用一个简单的隔振器设计来解决这个问题。(视频，嵌在下面。)

第一步是分析振动，了解对目标最重要的频率。完成后，一个简单的 3D 打印相机支架被设计成在相机和基座之间有三个柔性接头，基座被刚性地耦合到 RC 船或汽车的车身。Fusion 360 中的模态分析工具用于大致了解系统的频率响应，有助于最大限度地了解情况。

最终的设计确实有助于减少振动，尽管它无法抵消在极其粗糙的路面上行驶时产生的强烈振动。在这些情况下，[下班后的工程]建议使用万向节代替。适当的阻尼在许多应用中是天赐之物；例如，砖块可以为你的 3D 打印机带来巨大的变化。
