# ESP32 网络电台不是游戏

> 原文：<https://hackaday.com/2021/10/30/esp32-internet-radio-is-no-game/>

我们不止一次地看着像 TTGO T 型显示器这样的酷板，心想，“我们能用它造出什么？”如果你是[Danko berto VI]，答案就是你在下方的*Volos 项目*视频中看到的[微型网络收音机。](https://www.youtube.com/watch?v=lThawtfSQYA)

当然，核心的互联网流代码对任何 ESP32 都有用，但显示器是一个好看的单元。该代码可在 [GitHub](https://github.com/VolosR/TTGOInternetStation) 上获得。明智地使用网络和音频库，播放器只需要几百行代码。相当令人印象深刻，考虑到它甚至在微小的显示屏上显示可视化。

我们真正希望看到的是一个漂亮的外壳、电源和扬声器选项，以制作一个小巧便携的设备。有了 3D 打印机，就像我们经常看到的那样，很容易做出非常专业的项目。另一方面，它看起来确实比你在视频结尾看到的试验版要好。不过，这是一块做得整整齐齐的试验板。

如果你想要一个更大的屏幕，你可能会喜欢我们之前看过的 [ESP32 网络收音机](https://hackaday.com/2020/10/15/building-an-internet-radio-is-quick-and-easy-with-the-esp32/)。也许我们最喜欢的网络电台[是这个地球仪](https://hackaday.com/2020/07/27/radioglobe-takes-the-world-of-internet-radio-for-a-spin/)。
