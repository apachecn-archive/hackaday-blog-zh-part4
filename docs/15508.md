# 拍摄多氯联苯的(好)照片

> 原文:[https://hack aday . com/2022/12/01/拍摄多氯联苯的好照片/](https://hackaday.com/2022/12/01/taking-good-pictures-of-pcbs/)

用现代技术抓拍照片在技术上并不难，但拍出好照片却是另一回事。为了获得最好的效果，摄影师需要考虑很多事情，如果主题不是特别上镜，那么这个任务会变得更加困难。任何在网上发布待售商品的人都可以证明，即使对经验丰富的摄影师来说，拍摄日常物品也有其自身的挑战。[Martijn Braam] [在努力拍摄各种电路板的过程中，他为这样的照片准备了一些锦囊妙计。](https://blog.brixit.nl/taking-a-good-picture-from-a-pcb/)

[Martijn]一直在更新 [Hackerboards](https://hackerboards.com/) 上的图像，这是一个单板计算机和其他 PCB 的在线图像参考，他要求上传的图像质量。为了获得 PCB 的良好照片，他首先在软盒中以两个无线控制闪光灯的形式提供充足的照明。他还使用了低失真的高质量微距镜头，但真正的工作是确保图像清晰，PCB 边缘清晰。他使用 Python 脚本用他的相机拍摄了两张照片，并使用 ImageMagic 中的一些自动化功能将两张照片合成在一起。

虽然我们并不是都在拍 PCB 的照片，但这是一种很好的方式来展示工作流程可以以令人惊讶的方式自动化，更不用说照亮摄影主题的正确方式了。我们也看到了其他一些很好的照明方式，包括使用[破碎的液晶显示器](https://hackaday.com/2013/05/08/trashed-lcd-monitor-turned-into-a-light-box/)，或者你可以用[这个拱形照明系统](https://hackaday.com/2018/04/05/workbench-light-arch-on-the-cheap/)把这些原则带到你的工作空间。
