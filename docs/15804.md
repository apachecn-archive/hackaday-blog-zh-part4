# Raspberry Pi 和 PpLOGGER 制造低成本化学发光检测器

> 原文:[https://hack aday . com/2022/12/31/raspberry-pi-and-pplogger-make-a-low-cost-chemistry-detector/](https://hackaday.com/2022/12/31/raspberry-pi-and-pplogger-make-a-low-cost-chemiluminescence-detector/)

澳大利亚墨尔本拉筹伯分子科学研究所的[Laena]和她的同事们使用树莓 Pi 制作了一种低成本的电化学发光(ECL)检测器来测量炎症标记物，这种检测器可用于尽早检测心血管疾病或败血症，从而为医生挽救患者的生命提供更好的机会。

ECL 反应由于电激活的化学反应而发光，这使得它们对于检测血液、唾液或其他生物样品中的生化标记非常有用。ECL 的设置基本上相当简单。该设备包括一个用于启动化学反应的[电压参考发生器](https://hackaday.com/2022/12/27/an-esp32-based-potentiostat/)和一个用于测量发射光的光电倍增管(PMT)。PMT 输出电流，然后利用跨阻放大器(TIA)将其转换为电压。该信号随后由 DAQCplate 扩展板进行采样，并且可以在 ppLOGGER 中实时查看实时输出。

使用 RPi 允许团队进行一些必要但非常简单的信号处理，如将 TIA 电压转换回光电流，并对电流进行积分以获得 ECL 强度。他们提到，RPi 增加的信号处理潜力是他们的设置优于类似器件的一个巨大优势，然而，简单的集成可以在大多数微控制器上非常容易地完成。自然地，他们将他们的设备与标准的 ECL 装置进行比较，发现两种仪器的结果相当相似。他们的定制设备显示出比标准设置略低的检测极限。

他们的设备非批量成本约为 1756 美元，其中 PMT 占了大部分成本(1500 美元)。即使价格接近 2000 美元，他们的设备也比市场上的 ECL 仪器节省了 8000 多美元。虽然成本不仅仅是材料的清单，但我们希望看到社区为科学民主化做出努力，[Laena]和她的同事们正是这样做的。我想知道他们是否能帮我们找到捕蝇草的陷阱。
