# 向上翻转而不是向下翻转的翻转钟

> 原文：<https://hackaday.com/2021/11/24/a-flip-clock-that-flips-up-not-down/>

古老的翻转钟已经成为计时文化的一个超大部分，掩盖了其机制的简单性。人们满怀热情地收集和修复机电钟表，甚至寻求制造基于翻盖显示的新型钟表。设计各不相同，但它们都有一个共同点，那就是利用重力来打开树叶并显示它们的数量。

但是如果你把翻转时钟反过来呢？这差不多就是[Shinsaku Hiura]用翻转时钟完成的事情，翻转时钟会竖起数字，而不是翻转数字。时钟由三个 3D 打印的鼓组成，它们安装在一个公共轴上，通过齿轮和[日内瓦驱动](https://hackaday.com/2012/04/01/a-little-geneva-drive-made-of-wood/)连接在一起。每个数字都通过一个聪明的凸轮连接到一个鼓上，确保当它旋转到鼓的顶部时直立，并随着鼓的前进干净地落下。下面的视频清楚地展示了该机制的操作。

构建说明指出“这个时钟相对来说很难制作”，这很有帮助，并且给出了大量的故障排除说明，我们可以看到这是怎么回事。这不是我们第一次看到来自[Shinsaku Hiura]的具有机械挑战性的设计；[想到最近这个一伺服七段显示器](https://hackaday.com/2021/11/13/a-one-servo-mechanical-seven-segment-display/)。
