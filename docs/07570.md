# 医学测试的新观点

> 原文：<https://hackaday.com/2020/08/28/a-new-spin-on-a-medical-test/>

我们大多数人可以说，我们从电脑中取出了一个过时的硬盘，觉得把它扔在电子垃圾堆里是一种浪费。我们中的一些人用磁铁把孩子们的画挂在冰箱上，或者把一个巨大的电子表格挂在钢制文件柜上。让我们不要忘记，还有一个高速，低噪音的电机在那里。在一些型号上，它与 PCB 是分开的，所以拿一个电子速度控制器(ESC)，你的微控制器的选择，让自己成为一个[回收硬盘离心机](http://oionos.blogspot.com/2016/11/diy-hdd-conversion-to-microhematocrit.html)像【洞穴人】已经。他的构建使用托盘作为底盘，但他模拟并打印了一个新的脸，与原来的风格相同。

在顶部，他有一个有机发光二极管屏幕显示要求的速度，测量的速度和运行时间。显示屏旁边是一个带有自定义图例的四按钮键盘，用于设置参数。休息后的视频显示了机器的运行速度。该版本可容纳用于处理原始血液的微小毛细管、微量血细胞比容管。这项测试可以计算细胞的体积，专业人员用它来确定从脱水到贫血的各种情况。

还有其他的构建，人们已经将一个旧的驱动器改造成一种可以接受更大直径试管的离心机，但这是一个可行的光辉例子。善有善报，所以我们推荐[桌面生物实验室伴侣](https://hackaday.com/2020/08/17/cellerator-wants-to-be-your-automated-desktop-biotech-lab/)，或者[招募一些乐高头脑风暴](https://hackaday.com/2012/04/19/lego-mindstorms-used-to-automate-tedious-laboratory-tasks/)来帮忙。
