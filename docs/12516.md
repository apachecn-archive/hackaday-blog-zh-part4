# Arduino 遇见量子计算机

> 原文:[https://hack aday . com/2022/01/19/arduino-meets-quantum-computer/](https://hackaday.com/2022/01/19/arduino-meets-quantum-computer/)

量子计算机还不能完全用于家庭实验室，但是因为有办法通过互联网连接到一些计算机，你可以比你想象的更容易地用它们进行实验。[诺伯特]决定将一台巨型量子计算机与一台普通的 Arduino 接口。为什么？嗯，这不一定清楚，但话说回来，为什么不呢？他解释了基本的量子计算，并在下面的视频中展示了他的设置。

使用 IBM quantum 计算机和开源 Qiskit 使它变得相对容易，他在 PC 上使用的 Python 代码充当 Arduino 和 IBM 计算机之间的链接。当然，你也可以使用模拟来代替真实的硬件，对于这样一个简单的项目来说，这可能并不重要。

当然，演示相当简单，用量子位的状态点亮一个 LED。但是如果你想把真实世界的信息收集到量子计算机中，这种技术可能是有用的。你必须从某个地方开始。

我们之前已经研究过量子计算机。他们告诉我们这是下一件大事，所以我们要做好准备。 [Qiskit](https://hackaday.com/2019/04/02/quantum-computing-with-qiskit/) 是[如今可以让它变得更容易的几个选项](https://hackaday.com/2020/06/22/new-silq-programming-language-aims-to-make-quantum-programming-easier/)之一。
