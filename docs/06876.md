# 在家制作自定义渐变标记

> 原文:[https://hack aday . com/2020/06/14/making-custom-gradient-markers-at-home/](https://hackaday.com/2020/06/14/making-custom-gradient-markers-at-home/)

在做高端工业插画工作时，平滑的渐变为最终产品增加了很多生产价值。然而，设计用来做好这一点的标记可能很难找到。[Eric]决定使用常见的供应品创建自己的一套定制渐变标记。

从一些已经干燥的现有标记开始，内部的织物墨水容器被移除。一种新的棉条是用热缩材料包裹的棉条来复制原来的结构。酒精墨水是平滑渐变所必需的，Eric 建议，如果在商店买不到的话，可以用热风枪从圆珠笔上收集墨水。然后将墨水与变性酒精混合以稀释它，并使用注射器注射到织物储器中。每个标记都有略微不同的油墨混合，以达到一定的亮度值范围，从而形成平滑的渐变。

这是一个整洁的方式来创建你自己的渐变标记，无论你觉得什么颜色都有用。另外，这样做的材料既便宜又容易获得。我们甚至可以想象 3D 打印的标记体是一种选择，尽管笔尖可能会更加困难。我们以前也看过[Eric]的作品，比如这本图文并茂的指导在产品设计中使用纸板的指南。休息后的视频。
