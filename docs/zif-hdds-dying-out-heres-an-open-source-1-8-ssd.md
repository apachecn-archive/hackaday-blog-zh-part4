# ZIF 硬盘正在消亡？这是一款开源的 1.8 英寸固态硬盘

> 原文：<https://hackaday.com/2022/10/30/zif-hdds-dying-out-heres-an-open-source-1-8-ssd/>

许多旧技术运行在不再生产的部件上——硬盘恰好是这样一个部件，IDE 驱动器尤其早已过时，并因自然原因而灭绝。有替代品，但相当多的替代品要么不可靠，要么需要昂贵的存储介质。现在，[dosdude1]已经将注意力转向 1.8 ZIF IDE 固态硬盘——FFC 连接的硬盘非常罕见，因此更换起来非常昂贵，常见于 Macbook Air 1，1 2008 型号等笔记本电脑中。由于对替代品不满意，他[围绕 IDE SSD 控制器和 NAND 芯片从头开始设计了整个 SSD。然后，他将设计开源，并拍摄了组装视频，这样我们就可以自己建造了。看一看，我们已经把它放在休息下面了！](https://www.youtube.com/watch?v=YrBz-6lXbZQ)

对于一个开源设计，有相当多的工作与我们分享。他对一些基于 SM2236 控制器的 IDE 固态硬盘进行了逆向工程，以设计原理图，并将完整的 KiCad 文件放到 GitHub 上。在视频中，他向我们展示了如何仅使用热风站和烙铁来组装这款 SSD，谈到了 n and 匹配和编程软件的复杂性，并展示了 SSD 在前述 Macbook Air 中的工作情况。当然，使用模板组装会更快更容易，但是所使用的工具对于自我组装教程来说非常有用！

在 YouTube 评论中，[dosdude1]分享了更多的技术细节，并表示如果有人感兴趣，他也会出售这些东西。我们已经看到他在各种复杂的项目中工作，为旧的但值得的硬件注入活力——这个 [GPU RAM 升级](https://hackaday.com/2022/01/30/gpu-ram-upgrades-are-closer-than-you-think/)使其能够与苹果 Xserve 一起工作是一项值得注意的努力。我们已经看到了另一位著名的黑客【张文婷】、[为一台可爱的明基 S6 微型机器构建 SSD](https://hackaday.com/2022/02/25/custom-ssd-gives-new-life-to-handheld-atom-pc/) 的类似旅程。
