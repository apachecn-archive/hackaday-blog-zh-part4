# 流浪猫收容所比我们的第一套公寓好

> 原文:[https://hack aday . com/2021/02/14/流浪猫收容所比我们的第一套公寓更好/](https://hackaday.com/2021/02/14/stray-cat-shelter-is-nicer-than-our-first-apartment/)

有很多流浪猫在[红领带项目]附近游荡，似乎没有人关心他们会发生什么。幸运的是，对猫来说，[红领带项目]非常关心，正如他们建造的这个巨大的猫公寓所证明的。这些猫保留了它们的自由，但是每当它们决定用它们的存在来给[红领带项目]增色时，它们会得到食物和温暖的、有遮蔽的床。

[![](../Images/6ad90c35b744c95eb61ca2bb202d1355.png)](https://hackaday.com/wp-content/uploads/2021/02/cat-shelter-inner.jpeg) 【红领带项目】用托盘木建造了这个坚固的避难所，并做得很好，用木腻子密封所有的接缝和螺丝，并用硅胶防水。里面有一个加热的枕头，一盏灯和一个遥控摄像机，这样 RTP]就可以四处移动并监视猫咪。所有的电线通过一个连接在侧面的防风雨接线盒延伸到一个由 ATX 电源制成的控制箱。

大部分建筑是由废料制成的，包括最好的部分 Arduino 驱动的电动滑索，用于将食物从阳台送到猫门廊。关于控制箱和食品输送系统的细节即将推出，正如[红领带项目]在休息后的视频中所调侃的那样。我们很期待看到这些。哦，不要担心——有足够的镜头来支付猫税。

如果[红领带项目]收留这些猫，它们的需求只会增加。也许它们能承受跟随太阳的电动椅子的声音，因为它们毕竟是强悍的流浪猫。
