# 一种自动商店真空吸尘器

> 原文:[https://hack aday . com/2021/06/16/an-automatic-shop-vac-dust-extractor/](https://hackaday.com/2021/06/16/an-automatic-shop-vac-dust-extractor/)

在二手广告中找到便宜甚至免费的工具可能是黑客读者的共同追求。因此，你们中的许多人会喜欢[管道技工]，有一排旧的木工工作台工具。我们与他分享的经验是缺乏除尘功能，这使得[他将一台二手商店 vac](https://www.youtube.com/watch?v=eOhxLHv-IC8) 改装为自动吸尘器，用于他的剁锯，值得一看。看看吧，我们已经把视频放在休息下面了！

该系统将一个继电器线圈连接到电锯的开关上，该开关控制真空吸尘器的电源。因此，这不是最新颖的方法，但是在这个过程中有一些事情需要注意，我们当中有谁不喜欢看到一个研讨会项目的一点点进展呢？真空和 saw 所采用的 120V 电流对于我们这些习惯于 230V 电力的国家的人来说是过度的，但是选择继电器来容易地服务于该负载。自动系统的好处在于，坐在长凳上不会伴随着商店吸尘器持续不断的震耳欲聋的噪音，除了在使用锯子的时候，长凳既无尘又安静。

如果你的零件箱里碰巧有一个固态继电器，[这里有另一种方法可以达到类似的结果](https://hackaday.com/2020/06/06/vacuum-dust-collection-with-self-powered-relays/)。
