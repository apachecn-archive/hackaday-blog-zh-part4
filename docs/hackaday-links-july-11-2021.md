好吧，至少缩写会保持不变。随着 BSOD 屏幕多年来的首次改造，黑色似乎成为了 Windows 11 的新蓝色。这是一个不可否认的小变化，因为屏幕上的文字与最近版本的 Windows 10 中的 BSOD 几乎相同，新的丧钟甚至还带有相同的皱眉表情符号和二维码。事实上，黑底白字的配色方案是我们唯一能看到的主要区别——甚至连首字母缩略词也保持不变。我们认为，这并不真的有什么新闻价值，尽管它确实让我们怀念 Windows NT 时代极其繁忙的 BSODs。

随着半导体短缺的持续，制造商正不顾一切地采购制造产品所需的零件。如果有一件事像死亡和税收一样确定的话，那就是绝望为罪犯提供了机会。EEVBlog 上的一个帖子详细描述了一家公司与一个所谓的骗子的遭遇，这个骗子主动向他们提供大量通常很难找到的低价微处理器。明智的是，该公司对报价进行了一些深入的研究，发现“Brian”(联系他们的代表)实际上名叫 Nick Martin，根据国际电子经销商协会(ERAI)网站上的一篇文章，Brian[显然与一些欺诈性操作](https://www.erai.com/erai_blog/3176/desperate_buyers_targeted_by_scammers_during_chip_shortage)有关联。马丁涉嫌欺诈交易的清单可以追溯到 2018 年，非法所得总额超过 30 万美元。

去年，黑客之友和激光艺术家 Seb Lee-Delisle 花费了大量的时间和精力为英国城市的夜空举办了一场令人惊叹的互动激光表演。安装在高层建筑顶部的大功率激光器，取得了巨大的成功，给原本可怕的时代带来了些许欢乐。但我们不得不承认，涵盖激光城的视频和其他材料让我们想要更多——像这样的东西，在屋顶上安装得很远，观众可以通过手机控制这一切，真的需要更深入的“它是如何工作的”处理。谢天谢地，Seb 发布了[一个深入展示](https://www.youtube.com/watch?v=aFKYOVeyGTU)的具体细节的视频，包括一个在光天化日之下仍然可以看到的可笑的大功率激光。

最后，我们都听说过降噪，但您是否考虑过在系统中增加一点噪声？这可能不是在所有情况下都要走的路，但正如这个一分钟的视频所解释的，增加一点噪音可能会让观察变得不同。该视频使用了一个简单且有点做作的例子，用一伏特分辨率的电压表测量一个严重放电的电池。在读数中加入随机噪声，然后取数百个读数并取平均值，就能得到正确的答案。这有视频提到的其他更有影响力的含义，但我们不会破坏这个惊喜。看着吧——只需要一分钟。
