# 用这款自制的硬线射频分频器拍摄月球

> 原文:[https://hackaday . com/2020/03/31/shot-the-moon-with this-home brew-hardline-RF-divider/](https://hackaday.com/2020/03/31/shoot-the-moon-with-this-homebrew-hardline-rf-divider/)

你可以为[德里克]的业余无线电爱好者的雄心说一件事——他确实是脚踏实地。虽然大多数 hams 甚至从未试图“拍摄月球”，但他正在建立一个地球-月球-地球，或 EME，这需要一个小美人:[一个自制的四分之一波长硬线射频分配器](https://www.youtube.com/watch?v=t0bj8DVBBjA)，他与我们分享这个构建。

作为背景，EME 是一种利用我们的自然卫星作为被动通信卫星的传播技术。强大的定向信号可以从月球反射回地球，有可能将你的信号发送到任何能看到月球的人的范围内。在大约 770，000 公里的路径长度上，损耗是巨大的，足以使接收站通常使用[高增益八木天线阵列](https://hackaday.com/2017/05/24/on-point-the-yagi-antenna/)。

这就是[德里克]强硬路线的由来。分配器充当阻抗变换器，并匹配两个 50 欧姆天线与收发器预期的 50 欧姆负载并联。他用挤压铝管作为外层屏蔽，中心导体为铜管和空气电介质。他完成了所有的计算；股票大小的管道是好得足以进入球场的正确阻抗超过四分之一波长部分的硬线在理想的 432-MHz，这是在 70 厘米业余波段的中间。然而，可悲的是，对成品进行扫描发现，由于未知的原因，分压器在更高的频段谐振。

[德里克]仍在诊断，我们很想看看他有什么发现，但现在，至少我们已经了解了一些家酿硬线和 EME。想知道更多关于月球反弹的信息吗？[我们已经为你准备好了](https://hackaday.com/2019/10/01/bouncing-signals-off-the-moon/)。
