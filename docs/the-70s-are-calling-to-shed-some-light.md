# 70 年代的人打电话给我们

> 原文：<https://hackaday.com/2021/03/02/the-70s-are-calling-to-shed-some-light/>

还记得手机看起来都不一样的时候吗？在早期的手机时代，我们过得很好，这似乎是西方电气公司(et。在那个时代，你可以买到一部符合你内心极简主义者或公主形象的手机，并选择颜色。

[Dubchinsky]发现了一部过去时代的漂亮手机，并从两种可能的命运中拯救了它——垃圾填埋场，或者作为一件过时的技术，只是坐在那里看着。[相反，这款手机找到了第二个电话，作为一个可爱的台灯，具有秘密鹅颈弹性](https://www.instructables.com/Upcycled-RetroPhone-Lamp/)。灯本身是易趣上的一个便宜的 LED 模块，通过手机底座上的按钮开关连接到主电源。

我们绝对喜欢[Dubchinsky]把卷曲的绳子绕在鹅颈上，但有点失望的是，他没有使用挂钩开关来打开和关闭灯。在评论中，他说塑料感觉太脆了，经不起如此沉重开关的反复触动。这可以理解。[Dubchinsky]也想过使用旋转转盘作为调光器，我们认为这是一个好主意。

在指南、图片和休息后的构建过程视频之间，这几乎是一个完整的操作指南。我们认为这是值得称赞的，因为[Dubchinsky]正在 etsy 上出售这些灯具。

你是否怀念旋转转盘的日子，渴望更简单的日子？将你的手指插入这个旋转手机。
