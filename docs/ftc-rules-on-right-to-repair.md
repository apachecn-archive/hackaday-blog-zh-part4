# 联邦贸易委员会关于修理权的规则

> 原文：<https://hackaday.com/2021/07/27/ftc-rules-on-right-to-repair/>

几天前，美国美国联邦贸易委员会(FTC)以 5 比 0 的全票通过了对修车权的立场。(PDF)这是个好消息，因为他们基本上同意我们所有人的观点:

> 限制消费者和企业选择维修产品的方式会大大增加维修的总成本，产生有害的电子垃圾，并不必要地增加维修的等待时间。相比之下，提供更多的维修选择可以降低成本，通过延长产品的使用寿命减少电子垃圾，实现更及时的维修，并为企业家和当地企业提供经济机会。

“倪星修复”报告的[长版本继续列出了 FTC 发现公司阻碍维修的方式:从糟糕的初始设计，通过限制性固件和数字版权管理(DRM)，一直到“贬低非 OEM 零件和独立维修服务”。](https://www.ftc.gov/system/files/documents/public_statements/1592330/p194400repairrestrictionspolicystatement.pdf)

虽然联邦贸易委员会没有在这里制定任何新的法律，但他们传达了一种利用已经成文的消费者保护法的意愿:马格努森-莫斯保修法案和联邦贸易委员会法案第 5 节，该法案禁止不公平竞争行为。

只有时间才能证明这只狗是否真的有牙齿，但这是它在叫的好迹象。鉴于[欧盟正朝着类似的方向前进](https://hackaday.com/2019/10/09/europeans-now-have-the-right-to-repair-and-that-means-the-rest-of-us-probably-will-too/)，我们敢打赌可修复性在未来会增加。
