# 动画像素灯是任何芯片调谐器的卧室必备

> 原文：<https://hackaday.com/2019/06/02/animated-pixel-lamp-is-a-must-for-any-chiptuners-bedroom/>

灯具通常都很简单，用来照亮一个区域，或许还能为房间增添一丝时尚气息。当然，现在是 2019 年，我们不需要满足于此。如果我们愿意，我们可以拥有色彩缤纷的动画灯！(法语 Youtube 链接，嵌在下面。)

有问题的灯是[Heliox]的作品，她熟悉一两个 LED 灯。在这个建筑中，一串 WS2812 可寻址 RGB LEDs 连接到 Arduino 巨型大脑。led 安装在一个圆形灯体中，每个灯体都有一个矩形扩散器。这创造了一个吸引人的像素化效果，并给动画一个迷人的 8 位质量。一个薄的外壳是用白色塑料 3D 打印的，以进一步漫射光线。灯的顶部旋转内部电位计来控制模式选择。如果光线太强，底部还有一个亮度旋钮。

这是一个整洁的建筑，使用 3D 打印和可寻址 led 快速轻松地创建一个有趣的复古美学灯。我们可以想象这是一个很棒的时尚情景喜剧公寓。我们完全有望在未来几年内看到类似的灯具上市。休息后的视频。
