# 火坑燃烧与蓝牙的节拍

> 原文：<https://hackaday.com/2020/08/20/fire-pit-burns-to-the-beat-with-bluetooth/>

人类有几种原始的着迷，也许其中最大的两种是火和音乐。虽然你可以想象一些穴居人和穴居女人围坐在火边敲打木棍寻找节奏，但我们认为如果火随着音乐起舞，他们会印象深刻。通过蓝牙的力量，[这正是[Random Tech DIY]的新火坑所做的](https://www.youtube.com/watch?v=U4KhHYtHszE)。

从技术上来说，这被称为鲁本斯管，虽然这是一项古老的技术，蓝牙无疑是一个现代的触摸。正如你所料，这个项目的大部分时间是车间时间，切割中密度纤维板和塑料。音响系统是现成的，驱动一些汽车立体声扬声器。结果看起来不错，虽然建造携带丙烷气体的东西总是让我们紧张，但从我们的角度来看，它似乎工作得足够好。

我们想知道你能改变什么会影响展示。例如，改变孔的数量、孔的直径或气体压力，肯定会改变火焰的外观和对声波的反应。

我们已经看到了其他的[鲁本斯管项目](https://hackaday.com/2020/08/12/a-song-of-fog-and-fire-taking-a-look-inside-a-rubens-tube/)，当然。然而，在阴极射线管出现之前，我们真正感兴趣的是将它们用作[原始示波器](https://hackaday.com/2018/09/11/the-pre-crt-oscilloscope/)。我们也看到了《T4》对《T5》的现代演绎。
