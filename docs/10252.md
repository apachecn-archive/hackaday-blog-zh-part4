# TNDLPT 为任何 DOS 计算机带来 Tandy 声音

> 原文：<https://hackaday.com/2021/05/27/tndlpt-brings-tandy-sound-to-any-dos-computer/>

Tandy 1000 以及 20 世纪 80 年代的其他当代计算机和控制台使用德州仪器 SN76489 来输出声音和音乐。这种古老的声音芯片现在几乎可以在任何 DOS 机器上使用，只要它有一个并行端口—[感谢 TNDLPT 适配器！](https://hackaday.io/project/179962-tndlpt-tandy-sound-on-parallel-port)

适配器由 SN76489 组成，连接到并行端口，以便主机可以通过 DOS 终止和驻留程序作为驱动程序对其进行寻址。加载 TSR 后，只需在安装时选择 Tandy 1000 声卡，经典 DOS 游戏就可以使用 TNDLPT 声音输出。它还可以以各种其他方式使用，例如与 TNDY tracker 一起用于音乐创作，或者与 SBVGM 音轨播放器一起使用。

对于那些渴望听到飙升的 3 个声音(和一个噪音通道！)的 SN76489，这也是一个很好的方法，[serda shop 网站上提供了套件，供那些希望自己焊接的人使用。](https://www.serdashop.com/TNDLPT)或者，[用 OPL2 获得不同的感受](https://hackaday.com/2018/06/07/vaporwave-for-the-parallel-port/)。休息后的视频。
