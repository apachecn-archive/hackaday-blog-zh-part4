# 机动化你的相机滑块，黑客的方式

> 原文：<https://hackaday.com/2018/12/27/motorize-your-camera-slider-the-hacker-way/>

对于那些希望拍摄美丽流畅的平移视频或拍摄富有表现力的延时镜头的人来说，相机滑块是一个非常棒的工具。它们也可能非常昂贵，这刺激了 DIY 者在家创新。[Richard]想要一个电动滑梯，又不想倾家荡产，于是，[一款产品诞生了。](https://www.youtube.com/watch?v=qZt1-_5J930)

从现有的非机动相机滑块开始会让事情变得更容易，尽管没有理由不能将[Richard]的技术应用到完全 DIY 的构建中。NEMA 步进电机安装在框架上，并通过齿形带连接到相机梭子上。步进器由 Arduino 控制，它允许延时和平滑平移模式，并且可以用亚马逊的红外遥控器控制。滑块还与处理草图接口，该草图以图形方式显示了滑块在笔记本电脑屏幕上的当前位置，这有助于设置镜头。

[【Richard】已经分享了代码和购物清单](https://drive.google.com/drive/folders/1tUeheSU6zr64EWjX8ynps_yCkuUsz66L)，并确信可以用不到 100 美元完成构建。这是一个令人满意的价格，因为用一个好的滑球可以拍出高质量的照片。

我们在这里看到了大量的滑板造型，[包括这个令人印象深刻的缩放型造型](https://hackaday.com/2017/09/01/a-compact-portable-pantograph-camera-slider/)。休息后的视频。
