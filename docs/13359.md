# 多色绘图机器人强调限位开关的重要性

> 原文:[https://hack aday . com/2022/04/15/multicolor-draw bot-highlights-importance-of-limit-switches/](https://hackaday.com/2022/04/15/multicolor-drawbot-highlights-importance-of-limit-switches/)

绘图仪和绘图机器人是有趣的项目，让你用计算机数字控制所能提供的所有精度和完美来创造艺术。[tuenhidy]用[多色绘图机器人](https://www.instructables.com/Multicolor-DrawBot/)巧妙地证明了这一点。

建造依赖于简单的 XY 笛卡尔设计，使用一对 NEMA 17 步进电机。它以典型的 CoreXY 方式构建，在 Arduino Uno 上运行 GRBL 固件。

[tuenhidy]发挥创造力的地方在于钢笔本身。不是使用简单的圆珠笔或记号笔，而是使用可伸缩的多色笔。随着多色笔的出现，[tuenhidy]注意到限位开关在设计中的重要性。这些允许机器人多次通过，每次用不同的颜色，来建立一个多色图像。如果限位开关不在适当的位置，就不可能将每个后续通道排成一行。

我们希望看到建立一个基于伺服系统的颜色自动切换更进一步。不过，事实上，[tuenhidy]有一个功能强大的绘图仪，可以交付整洁的多色艺术品。
