# 微型马达使实验性钢琴演奏成为可能

> 原文：<https://hackaday.com/2021/02/11/tiny-motors-enable-experimental-piano-performance/>

就在你认为你已经见过所有可能的钢琴演奏方法时，[亚历山德罗·佩里尼]想出了一个新的方法。在为打击乐手[艾琳·比安科]创作的这首作品中，手持马达成为作曲家、电子音乐设备和表演者之间的有形界面。

表演涉及十个小型圆盘马达，由一个木架固定在琴弦上方。马达由 Arduino Nano 控制，它根据电脑发出的 MIDI 命令打开或关闭马达。然而，表演并不是完全自动化的。[Irene]手指上戴着一对接触式麦克风，她在钢琴内部移动麦克风，以捕捉琴弦与马达本身和谐振动的声音。

[【Alessandro】非常友好地分享了一个教程](https://www.hackster.io/touchmysound/10-midi-controlled-vibration-motors-for-piano-performance-513430)，关于如何为那些热衷于创作自己的类似作品的人重新创建表演背后的硬件和软件。正如任何音乐努力一样， [MIDI 总能让它变得更好。](https://hackaday.com/2021/01/02/mattress-made-midi-device-plays-exotic-tunes/)休息后的视频。
