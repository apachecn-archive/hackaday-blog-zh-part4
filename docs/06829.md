# 针织机改造使其更上一层楼

> 原文:[https://hack aday . com/2020/06/11/knitting-machine-rebuild-take-it-to-the-next-level/](https://hackaday.com/2020/06/11/knitting-machine-rebuild-takes-it-to-the-next-level/)

我们这些从事纺织工作的人可能会拥有一台缝纫机，即使我们真的对它感兴趣并且有资金，也是一台包缝机。但是在那个领域还有一种我们很少人会拥有的机器，那就是针织机。这些机器有一个在一长串针上滑动的支架，即使是便宜的机器也比一个相当不错的示波器要贵得多。[Irene Wolf]有一台 Passap E6000 电脑针织机，价格不菲，[她对其进行了重大改进](https://hackaday.io/project/163701-passap-e6000-rebuilt-and-replaced-console)，为其配备了新的大脑、新的电机控制器，并用另一台机器前部的一套电脑针床取代了机械后针床。

在她的文章中，她深入探讨了操纵机器的传感器和电磁铁的布置。她从 Hackerspace Bamberg 的一个项目中获得了很多灵感，但使用了所有可用的 Passap 传感器作为输入，而他们只使用了一个传感器。她有两个处理输入的 Arduino M0 板和一个带控制和用户界面的 Raspberry Pi，并发布了一些系统运行的视频，其中一个我们放在了休息区的下面。

我们可能没有勇气无畏地黑掉这样一台高价值的机器，我们对结果印象特别深刻。这篇报道特别有趣，不仅是因为作品本身，还因为它对这些机器的工作原理给出了详细的见解。最好的消息是——她还没有完成，还会有更多的分期付款。

当你在等待更多的时候，请记住[这绝不是我们带给你的第一台被黑掉的针织机](https://hackaday.com/2018/09/06/electromagnetic-field-a-hacked-knitting-machine-knitting-the-universe/)。
