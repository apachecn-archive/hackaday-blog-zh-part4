# 时钟模式将超级游戏男孩带入竞技竞技场

> 原文：<https://hackaday.com/2019/04/14/clock-mod-brings-super-game-boy-to-competitive-arena/>

在 1994 年到 1998 年间的一段时间里，你可以和一个超级游戏机玩彩色游戏机。这是一个插入超级任天堂的盒式磁带，使用专有的 Lock-On 技术，你可以在大屏幕上玩游戏机游戏。超级游戏男孩的内心是一个真正游戏男孩的勇气。这曾经是，现在仍然是体验科比的梦幻之地或关东口袋妖怪的最佳方式。

不幸的是，超级游戏男孩并没有完全复制游戏男孩的体验。超级游戏男孩中的晶体意味着游戏和声音运行速度快 2%到 4%。超级游戏男孩是出于竞争的速度运行，如果你使用小声音 DJ，你会与乐队的其他成员格格不入。超级游戏机也没有连接线支持。

现在，Tindie [上的【qwertymodo】有了更快的超级游戏男孩](https://www.tindie.com/products/qwertymodo/super-gameboy-clock-mod/)的解决方案。这是一个时钟模块，但它不只是交换一个晶体。这是一个焊接到现有焊盘的板，仍然允许您访问 Hori Commander 控制器提供的加速和减速功能。这是一个有点令人印象深刻的 PCB 艺术，当然值得注意。

这个 mod 修复了超级游戏男孩 2-4%的加速，但是仍然缺少一个功能:连接线。好吧，抓紧你的屁股，因为这个也有一个模型。超级游戏男孩链接端口是一个很小的分线板，需要飞线连接到超级游戏男孩的主芯片。安装并不像 crystal hack 那样干净，但是如果你在修理时钟，你也可以在那里添加链接电缆端口。
