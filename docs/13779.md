# 发出任天堂 Vibez 的 MP3 播放器

> 原文:[https://hackaday . com/2022/05/30/an-MP3-player-that-give-off-Nintendo-vibez/](https://hackaday.com/2022/05/30/an-mp3-player-that-gives-off-nintendo-vibez/)

我们肯定非常喜欢 Hackaday 上的 DIY MP3 播放器，但我们不认为我们以前见过像[jpet26]中的[cartridge MP3](https://www.instructables.com/Cartridge-Based-MP3-Player-CartridgeMP3/)这样的播放器。

所有的电子元件都是我们所期望的。[jpet26]使用流行的 VS1053 解码器读取存储在 SD 卡上的 MP3 文件。他还包括一个用于调节音量的电位计，一个用于电源和编程的 USB C 端口，一个用于音频输出的耳机插孔，一个通用状态 LED 和一个开/关开关。

但是真正引起我们注意的是为他的 MP3 播放器选择的形状因素[jpet26]。虽然 MP3 文件存储在 SD 卡上，但他使用类似于任天堂 64 或昔日游戏机的盒式接口，从 SD 卡上选择播放哪首 MP3。卡带接口与几个 GPIO 引脚相连，通过读取每个引脚的状态，设备决定选择哪一个 MP3。

你可以说这个弹夹有点多余，我们不会和你争论。卡带实际上并不存储 MP3 文件，SD 卡才存储。如果这个盒子里装着 SD 卡，并在卡上存储一些精选的 MP3，可能会更有意义一些。这将是一种与朋友分享你最喜欢的播放列表的奇特方式。所以，是的，有些笨拙的握手，但是谁不是时不时地犯这种错误呢？我们喜欢它，认为你可能也会喜欢它。

很酷的 MP3 播放器，[jpet26]！我们可以为 V2 推荐一位[演讲者吗？也许还有一些](https://hackaday.com/2021/10/08/3d-printed-preschooler-proof-mp3-player-takes-a-beat-ing/)[软电缆](https://hackaday.com/2020/06/26/hinge-brings-new-meaning-to-flexible-pcb/)。
