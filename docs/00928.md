# 机器人从不错过腿日

> 原文：<https://hackaday.com/2018/11/11/robot-never-misses-leg-day/>

我们听说过两足行走被称为一系列受控下落，或一次连续下落，我们反复抓住自己，以 9.8 米/秒 ² 的速度下落是一段很长的路。我们中的一些人比其他人更优雅，但与我们最先进的两足机器人相比，大多数小学生已经获得了更高的熟练程度。腿包括各种弯曲和扭曲的复杂关节，我们不能从膝盖开始。庆应义塾大学和东京大学的研究人员致力于研发一种不用轮子、不会行走、不会翻滚的机器人。 [Mochibot 使用三十二条伸缩腿](https://spectrum.ieee.org/automaton/robotics/robotics-hardware/32-legged-spherical-robot-moves-like-an-amoeba)移动，每条腿只能从中心移入或移出。

像这样的多腿运动已经在一个叫做 tensegrity 的过程中完成了，但在这种形式下，腿只能延伸到足够让机器人向预期的方向翻滚。Mochibot 不会等待这种受控的下落，它会尽可能多地将面朝下的腿放在地上，并在后腿推动它前进的同时将它们收回。通过这种方式，机器人永远不会摔倒，并且运动受到控制，但由于腿受到精心控制，处理能力更高。期待这么多条腿上的运动控制，也意味着转弯可以更精准，任何方向都可以成为前方。这也使原子核与地面保持在同一水平。我们不禁会想，如果把它塞进一个巨大的气球里，看起来会很酷。

一些人已经知道美国宇航局的张拉整体机器人，但他们可能不知道美国宇航局为其发布的工具包。好吧，说真的，膝盖是怎么通过进化测试的？我猜他们为这个[跳跃机器人](https://hackaday.com/2017/02/03/robot-leaps-uncanny-valley-on-backward-knees/)工作。
