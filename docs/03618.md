# 把一个插销变成一把密码锁

> 原文：<https://hackaday.com/2019/07/18/turning-a-single-bolt-into-a-combination-lock/>

在我们寻找大箱子的便利时，我们往往会忘记锁匠曾经不仅复制钥匙，还从头开始创造复杂的锁和其他复杂的机制。[我的机械师]没有忘记，造一把锁是他庆祝锁匠技能的方式。[用一个不锈钢插销制作一把密码锁](https://www.youtube.com/watch?v=OyhfQYHnJuM)可能也只是炫耀一下，我们对此完全没有意见。

诚然，螺栓是相当大的一个-m20x 70-和一些其他材料，如黄铜棒和弹簧丝需要完成锁。但是能够看到一个单独的螺栓，并把它切成锁所需的大部分材料，这简直太神奇了。头变成了两个终板，而柄在螺纹被切断后被纵向和横向分成两半；这些零件后来被加工成制造锁机构所需的管子和销钉。组合轮可能来自另一个或更长的螺栓，但我们喜欢黄铜对抛光不锈钢的外观，以及蚀刻的数字和微妙的滚花。整个事情是一个锁匠之旅，下面的视频捕捉到了所有这一切，没有任何绒毛或废话。

如果你不喜欢在钢铁和黄铜中工作，不要担心——[3D 打印的密码锁](https://hackaday.com/2019/03/21/3d-printing-a-combination-lock/)可能就在你的手边。或者[激光切割木头](https://hackaday.com/2016/07/25/have-a-laser-cutter-hes-a-cool-combination-lock-box-you-can-build/)。或者甚至是[普通纸](https://hackaday.com/2015/03/08/combination-lock-made-out-of-paper/)，如果你对整个安全系统不感兴趣的话。
