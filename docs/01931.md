# 这台 3D 打印机对机器人来说是软的

> 原文：<https://hackaday.com/2019/01/30/3d-printer-is-soft-on-robots/>

在我们看来，最好的机器人总是模仿有生命的东西。举例来说，只需看看北卡罗来纳州立大学研究人员的 3D 打印网格结构。外部磁场使网状“机器人”在水中漂浮时弯曲和移动。这种机械装置可以抓取小物体，并携带像水滴一样精细的东西。

关键是由硅酮微珠、羰基铁颗粒和液态硅酮制成的粘性牙膏状墨水。在烤箱中固化之前，得到的糊状物可以进行 3D 打印。当然，铁是使东西对磁场敏感的元素。你可以在下面看到几个视频。

显然，这些结构不会很快复制出机器人罗比。然而，在发表于[Advanced Materials Technologies](https://onlinelibrary.wiley.com/doi/10.1002/admt.201800528)的论文中，研究人员认为为细胞培养创造活性组织支架和模仿生活在水面上的生物是两种可能的应用。

该装置需要一个电磁体来启动。然而，它确实让我们想到了使用更常规的磁性材料和外部磁场来执行任务。这似乎并不难做到。
