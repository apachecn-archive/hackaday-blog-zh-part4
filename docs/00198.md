# 尽情享受这个机械眼球吧

> 原文:[https://hack aday . com/2018/08/04/在这个机械眼球上饱餐一顿/](https://hackaday.com/2018/08/04/feast-your-eyeballs-on-this-mechanical-eyeball/)

如果我们买了一台能够支持摄像头的单板计算机，我们大多数人都会屈服于诱惑，为外设买单，希望我们可以哄我们的新玩具拥有视力。我们已经使用了命令行工具，并对我们的工作台拍摄了一些随机图像，但是接下来会发生什么呢？我们的同事[Steven Dufresne]想用他的树莓派探索相机的无限可能，所以他做了一个电动眼球支架来实现这一点。

使用遥控伺服的云台不是什么特别新的东西，但是在这一个中，他投入了一些设计努力，这可能是其他人所缺乏的。为了确保两个轴之间没有干扰，已经付出了很多努力，并且在一个稍微令人毛骨悚然的扭曲中，直到你想起他说的是一个模型，该单元被设计成适合人类的头部。

伺服系统由 Pi 使用[一个他在另一个视频](https://www.youtube.com/watch?v=mX6P9CwSm7I)中讨论过的伺服驱动板驱动，所以一旦他描述了一些设计调整后的组件，他会快速浏览他编写的模拟眼睛跟踪神经元的软件演示。他承诺将在适当的时候放在某个地方供下载。

如果你想在市场上为你的 Pi 买一个云台支架，这款可能对你的 3D 打印机很有意义。它肯定比我们之前给你看的那部更有成就。
