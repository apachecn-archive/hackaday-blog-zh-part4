# 喂鸡，有型

> 原文：<https://hackaday.com/2019/08/23/feeding-chickens-with-style/>

啊，家畜的快乐。他们通常很可爱，偶尔也很有用，但普遍都不能照顾好自己。这是交易的一部分。我们负责它们的保养，它们尽最大努力回报我们。当然，除非被讨论的动物是猫，否则他们有自己的条款和条件。

然而，鸡确实非常有用。给它们食物和水，它们会给你美味、营养、高质量的蛋白质。不过，每天给它们喂食可能是件苦差事，除非你能自动完成这项任务。对于这个简单的用例来说，这个能够抽动的机器人喂鸡器可能有点大材小用，但正如[Sean Hodgins]所说，他扔进这个建筑的所有硬件都有一个方法。这将包括一个定制焊接的钢架，上面装有太阳能电池板和电池，一个巨大的 LED 矩阵显示器，一个树莓皮和摄像头，当然，还有食物分配器。那些是曾经在杂货店里用一两个硬币分发糖果或口香糖的那种；重新装备了 3D 打印部件，现在每当有人观看抽搐流决定向托管该系统的农场捐款时，分配器就会喷出一小勺饲料。你可以在下面看到建造的细节，或者去甜蜜农场看看[的现场饲料](https://www.twitch.tv/sweetfarmchickenfeeder)，呆呆地看着一些鸡。

对[肖恩]来说，这无疑是一项令人印象深刻的工作，我们确实注意到他是如何使用他的 [HCC 快速原型模块](https://hackaday.com/2018/07/28/save-some-steps-with-this-arduino-rapid-design-board/)来加速开发的。尽管如此，我们不相信会有很多 10 美元一次的捐赠。话说回来，将捐款降至小额支付水平可能会导致鸡被过度喂养，这不是一件好事。
