# 线圈绕线机使它变得简单

> 原文：<https://hackaday.com/2021/03/24/coil-winding-machine-makes-it-easy/>

[创新先生]需要缠绕一些线圈，并决定[制造一台机器来完成这项工作](https://www.youtube.com/watch?v=vvFKuOKtCbw)。这些年来，制造这种机器变得容易多了。曾经有一段时间，我们可能不得不破解旧打印机或扫描仪，以获得线性杆和步进电机。现在，由于广泛的 3D 打印，我们可以从很多地方订购这样的零件。3D 打印也有助于制造所有定制的小部件，你需要把这样的东西放在一起。

这台机器看起来很棒。它使用了许多在 3D 打印机或 CNC 上看起来像在家里一样的零件。我们认为他的中国迷你台锯在切割铝型材方面做得很好，但我们确实担心他手指的安全。我们承认我们通常很懒，会买预切割的挤压件。

当然，对于那些只制作几个线圈的人来说，变速钻中的传力杆可以让你达到目的。但是如果你做的是小批量生产，我们可以看到它的价值。添加软件来控制圈数、速度和其他参数非常简单。

这让我们想起了我们最近看到的[绕组拾波线圈](https://hackaday.com/2020/12/13/a-pair-of-steppers-are-put-to-work-in-this-automatic-instrument-pickup-winder/)的制作。然而，真正的价值是一台用许多圈细导线缠绕那些[讨厌的特斯拉线圈](https://hackaday.com/2020/11/16/automatic-winder-takes-the-drudgery-out-of-tesla-coil-builds/)的机器。
