# 这个玩水盘将会增加你对播种的热爱

> 原文:[https://hackaday.com/2019/10/07/__trashed-7/](https://hackaday.com/2019/10/07/__trashed-7/)

3D 打印机和激光切割机最大的优点之一是它们能够生产专门的工具，从繁琐的过程中节省时间。种子播种就是一个很好的例子。哪怕你只想播一盘两打左右的种子，你还是要把盘里装满土，整平，压实均匀，把所有的洞都捅出来。当种子播种是你的谋生之道时，手动完成所有这些工作会耗费大量时间。

现在已经有了大规模点播的机器，但是[【迈克尔·拉特克利夫】已经涉足小规模农场的点播板](https://hackaday.io/project/167679-dibbling-plate-put-those-holes-in-the-soil-quick)。他发明了一种多功能工具，除了把土倒在托盘里，它什么都做。一旦你做到了这一点，你可以使用边缘平整多余的土壤，用背面压缩它，然后翻转到钉床侧，一次打完所有的洞。它很容易拆开，所以任何人都可以更换破损或变钝的点播器。

[Michael]卖的很便宜，但是你可以在那里的[Thingiverse](https://www.thingiverse.com/thing:3866898)找到所有的文件和构建说明。休息之后我们放了演示视频。

更喜欢绿色蔬菜？ [3D 打印也可以提供这种固定效果](https://hackaday.com/2018/12/01/germinate-seeds-with-the-help-of-3d-printing/)。
