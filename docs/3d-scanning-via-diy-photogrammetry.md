# 通过 DIY 摄影测量进行三维扫描

> 原文：<https://hackaday.com/2019/03/13/3d-scanning-via-diy-photogrammetry/>

对于那些拥有 3D 打印机的人来说，拍摄一个 3D 模型并吐出一个实物是如此的例行公事，以至于令人厌烦。相反的情况有点不同。有许多方法可以创建复杂程度不同的物理对象的数字 3D 模型。[Eric Strebel]喜欢摄影测量学，并为那些对使用这项技术感兴趣的人分享了一份有用的指南。

在最基本的意义上，摄影测量指的是从照片中进行测量。在这里讨论的意义上，它更准确地指的是从一系列物理对象的照片中创建 3D 模型的方法。通过拍摄一个物体的适当图像，并通过正确的软件输入，就有可能创建一个物体的数字表示，而不需要除了相机之外的任何特殊硬件。

[Eric]分享了一些获得良好效果的技巧和诀窍。表面处理是关键，目的是创造一个平坦的完成，以避免反射造成的问题。首先在物体上喷涂一层灰色底漆，然后撒上一层黑点，这有助于软件识别物体的轮廓。相机设置也很重要，宽光圈用于创建浅景深，有助于对象从背景中脱颖而出。

有了合适的物体准备和摄影技术，困难的工作就完成了。接下来需要做的就是通过相关软件输入照片。[Eric]倾向于 Agisoft Metashape，尽管有各种各样的软件包提供这种功能。
