# 使用 WSPR 测试天线

> 原文：<https://hackaday.com/2022/07/29/testing-antennas-with-wspr/>

测试高频天线的方法很多，从模拟到各种天线分析仪和电桥。然而，没有什么可以取代简单地使用天线来看看它如何工作。就像——据说——大黄蜂不会飞，但它还是会飞，它可以装上一些弹簧床，进行接触。但是过去很难收集大量关于天线性能的经验数据，尽管这很有趣。现在，你可以用 WSPR 来完成所有这些工作，[TechMinds]建议用一个价格适中的专用 WSPR 发射机来完成这项工作。你可以在下面看到一个关于这项技术结果的视频。

虽然 WSPR 经常被认为是剥夺了业余无线电的乐趣，但它非常适合这个应用程序。连接发射机，几个小时后，访问一个网页，找出你在哪里被一个客观的观察者听到。如果你有几个这样的天线，你甚至可以在相似的时间和条件下检查几个天线。

发射器有自己的 GPS，因此不需要太多配置。你确实需要设置你想要使用的频率，并且——假设——你的天线在这些频率下的 SWR 是可以接受的。当然，你还需要设置你的呼号和传输时间表。如果您不想获取 GPS 设置，可以手动设置位置代码。

设置完成后，您不需要连接电脑。一段时间后，你可以访问 WSPR.org 的网站，看看谁能听到这个小功率发射机。

当然，你不需要一个专用的信标来完成这个任务。使用你的普通无线电或者[组装你自己的信标](https://hackaday.com/2021/04/15/the-50-ham-a-simple-wspr-beacon/)。如果你想重温 WSPR，[【丹·马洛尼】50 美元的火腿](https://hackaday.com/series_of_posts/the-50-ham/)已经满足你了。
