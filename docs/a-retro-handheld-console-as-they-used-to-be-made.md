# 一个复古的手持控制台，就像他们过去做的那样

> 原文：<https://hackaday.com/2019/04/28/a-retro-handheld-console-as-they-used-to-be-made/>

在有任天堂 Switches 之前，有 Game Boys。在此之前，所有连续几代的游戏男孩和其他游戏机都回到了游戏和观看，然后是 20 世纪 70 年代末的手持西蒙和太空入侵者游戏。这些设备通常包含一个 4 位微控制器和一个分立的 led 阵列，游戏中的移动只是通过游戏场上交替闪烁的 led 来创建。

来自[sv2002]的 [TeleBall](https://hackaday.io/project/165136-teleball-retro-handheld-game-console) 是一款类似于早期手持游戏的手持游戏，它以 led 矩阵作为屏幕，可以在屏幕上显示简单的游戏。到目前为止，它玩突破，两个人打网球，如果不是因为它内置了两个人玩的无线电和两个控制台，这可能看起来很奇怪。

TeleBall 内部是一个 Arduino Nano，这是一个用于 LED 矩阵的 Maxim 显示驱动器，以及我们熟悉的 Nordic Semiconductor RF 模块。控制是通过一个电位计，一切都在一个智能 3D 打印的情况下。所有的东西都是开源的，所以如果你想拥有自己的，你可以[前往该项目的网站](http://teleball.org/index.html)获取所有的文件。你可以在休息时间下面的视频中观看它使用两个控制台打网球的过程。

最初的双人网球于 1958 年诞生，是一款使用模拟计算机的示波器游戏，被认为是第一款纯粹为娱乐目的开发的视频游戏。如果你想看它的重现，[我们在十年前报道过一次](https://hackaday.com/2008/07/16/tennis-for-two-resurrected/)。
