# 工作申请脚本用 Python 自动化了枯燥的东西

> 原文：<https://hackaday.com/2020/05/30/job-application-script-automates-the-boring-stuff-with-python/>

在今天这个时代，找工作当然需要大量的跳呼啦圈。即使你足够幸运，让一个真正的人阅读你的申请，也不能保证对方对你的技能有多少了解。通常，整个过程从一开始就是徒劳的，作为一名刚毕业的学生，[harshibar]很清楚投入大量时间可能会带来的心灵破碎的体验。嗯，俗话说:如果你不能打败他们，就加入他们——如果你不能加入他们，[让申请过程自动化](https://github.com/harshibar/common-intern)。

作为“5 天内 5 个 Python 项目”挑战[harshibar]的最后一部分，她基本上创建了一个网络搜索器，收集特定搜索词的职位空缺，并自动向每个人发送申请。使用[美汤](https://www.crummy.com/software/BeautifulSoup/)解析某个招聘门户网站的抓取页面， [Selenium 的](https://selenium-python.readthedocs.io/)浏览器自动化功能填写在线申请表，她可以将所有信息输入表格，与手动方式相比节省了无数时间。该程序甚至点击应用按钮。

虽然数量重于质量的方法可能并不适合所有人，但对于职位空缺，事先当然有更多的筛选和更好的选择的空间，[harshibar]在她的关于项目的视频[中也提到了这一点(嵌入在下面)。虽然这不会解决申请过程本身，但我们肯定可以看到“自己打败他们”游戏可能带来的满足感——此外，它的失败率不会比典型的 LinkedIn“招聘人员”更低。尽管如此，如果你正在寻找一个更系统的方法，看看【勒温·戴】](https://www.youtube.com/watch?v=N_7d8vg_TQA)[在这个问题上的观点](https://hackaday.com/2017/08/02/why-i-got-the-job/)，他甚至建议求职[对你来说还很遥远](https://hackaday.com/2019/10/28/the-young-engineers-guide-to-career-planning/)。
