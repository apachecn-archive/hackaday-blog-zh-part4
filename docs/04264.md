# BeagleBone 深度学习视频演示

> 原文:[https://hack aday . com/2019/09/22/beagle bone-deep-learning-video-demo/](https://hackaday.com/2019/09/22/beaglebone-deep-learning-video-demo/)

BeagleBoard 经常被树莓 Pi 掩盖。Pi 侧重于易用性，而 BeagleBone 通常更适合核心应用程序。随着机器学习 AI 现在风靡一时，BeagleBoard 现在有了 [BeagleBone AI](https://beagleboard.org/ai) ，这是一种针对机器学习的特定功能的板。最近的一个视频(见下文)展示了使用 TIDL(德州仪器深度学习库)的演示。该视频包括一个例子[将视频流式传输到浏览器，并使用预定义的学习模型来识别网络摄像头拍摄的东西](https://www.hackster.io/175809/tidl-on-beaglebone-ai-1ee263)。

板载 CPU 为 TI Sitara AM5729。这是一个 1.5 GHz 的双臂 Cortex A15。还有两个 C66x 浮点 DSP 处理器和两个双 ARM Cortex M4 协处理器。还需要更多吗？您将获得四个嵌入式视觉引擎、两个双核实时单元、一个 2D 图形加速器、一个 3D 图形加速器以及一个用于编码和解码视频和加密的子系统。

也有相当多的记忆。除了片外内存，该设备还有 2.5 MB 的内存。

演示不需要在板上安装任何新软件。您甚至不需要为主板安装显示器，因为视频可以传输到外部网络浏览器。相当酷。

当这款设备首次发布时，我们曾报道过它。我们真正喜欢比格犬的一点是[前肢实时单元](https://hackaday.com/2016/07/31/blindingly-fast-adc-for-your-beaglebone/)。
