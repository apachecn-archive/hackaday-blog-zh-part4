# 自动空气炮发射烟圈

> 原文:[https://hack aday . com/2021/11/12/automated-air-cannon-shots-smoke-rings/](https://hackaday.com/2021/11/12/automated-air-cannon-shoots-smoke-rings/)

空气炮很有趣，随着流行的 Airzooka 玩具成为主流文化的一部分。当然，击发和发射 Airzooka 过一会儿就累了，而且它们有点小。[1alembic] [的这一版本在这两个方面都有所建树。](https://www.instructables.com/Automated-Air-Canon-With-Videos/)

![](../Images/2d6dd1eadcb813c785f7043044e02983.png)

Cool, huh?

其结果是一个更大的空气炮，反复发射所有自己。大炮本身是由底部被切掉的垃圾桶制成的。然后给它装上一个由重型垃圾袋制成的隔膜，用胶带包裹以增加强度。乳胶软管安装在垃圾桶内，连接到隔膜上。因此，隔膜可以被拉回，当释放时，它被向前拉，产生一股气流穿过垃圾桶，产生一个涡环，就像较小的 Airzooka 一样。

大炮的自动化非常简单。一根细绳系在横膈膜的背面，缠绕在一根杆上，这样它就可以被卷起来。这允许刮水器电机通过一组齿轮转动杆，将隔膜拉回。

然而，刮水器电机上的驱动齿轮缺了一半齿。然后设置该系统，以便一旦隔膜被拉回，驱动齿轮到达缺失的齿，允许卷绕杆在隔膜向前射出时自由旋转，启动空气炮。当驱动齿轮重新啮合卷绕机构时，循环重复进行。

与烟雾机配对后，空气炮会发出呼呼声，每隔一段时间就会发射出美丽的烟圈，直到它被关闭。这是一个优雅的东西，我们喜欢在聚会上设置它来增加一些气氛。我们也看到过其他有真正火力的空气炮。休息后的视频。
