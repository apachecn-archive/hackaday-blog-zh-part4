# 最后一次扫描模拟胶片

> 原文：<https://hackaday.com/2020/06/19/scanning-analog-film-for-the-last-time/>

胶片相机具有很高的分辨率，在很长一段时间里，在这方面都优于数码相机。然而，现在有可能存储模拟图像的数字副本，因此[Jan]建造了一个钻机来最后一次扫描他们的照片。

总的想法是对胶片底片或幻灯片进行足够高分辨率的扫描，这样当技术进步时就不需要重新扫描图像了。为了实现这一目标，[简]决定雇佣一名 DSLR 来拍摄这些材料。为了快速准确地做到这一点，最大限度地减少麻烦，3D 打印了特殊的镜头罩，以在镜头前完美地对准幻灯片。用闪光灯提供均匀的光线，效果非常好。胶片底片证明更难，需要精心设计的运输机制，以避免损坏易碎的材料。只要坚持不懈，最终的工具运行良好。

这是对模拟照片进行数字化存档的一种简洁的方式，而且凭借现代相机的分辨率，人们不必担心分辨率会下降。我们也看到了处理其他格式的机械化版本，比如这个 8 毫米扫描仪。休息后的视频。
