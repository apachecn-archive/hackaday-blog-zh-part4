# 重现一个镜头

> 原文:[https://hackaday.com/2022/06/14/recreating-a-camera-shot/](https://hackaday.com/2022/06/14/recreating-a-camera-shot/)

当镜头疯狂地放大和缩小时，人们滚下盾牌和长矛，与剑相碰，这使得电影 *300* 中标志性的动作镜头如此具有标志性。不幸的是，达到这种效果并不容易。三台相机在滚动，每台都有不同的镜头(100 毫米、50 毫米和 21 毫米)来捕捉同一场景的不同视图。在后期制作中，由于镜头是同步的，您可以在三台摄像机之间进行切换。[走廊工作人员]的人想要重现这种效果，但他们没有创建一个定制的支架来容纳三台昂贵的相机，而是 [3d 打印了一个定制的支架来容纳三台昂贵的智能手机](https://www.youtube.com/watch?v=v8X16Gw3Qa4&ab_channel=CorridorCrew)。

虽然大多数手机背面都有三个摄像头，但大多数手机无法同时从所有摄像头进行慢动作拍摄。所以他们需要一个支架来支撑三部手机。第一个设计很简单，只是用来放手机的支架。虽然漂亮又坚固，但把手机放进去或拿出来并不容易，而且按下录音按钮也很棘手。iPhones 的背面有这个方便的小磁环。他们有一个支架，在设计和一些打印机问题上经过几次迭代后工作得很好。由于每个相机都有光学图像稳定功能，镜头很容易错位，从而影响拍摄。然而，他们在某种程度上掩盖了后的影响。有了一个工作原型，剩下唯一要做的事情就是用慢动作把一堆皮纳塔切成片，配上悦耳的配乐。

我们喜欢看到令人兴奋的相机设置和迭代，以找到一些工作。这种双摄像头设置有一个非常不同的目标，并试图融入视差效果而不是隐藏它。休息后的视频。
