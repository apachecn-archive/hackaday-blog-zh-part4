# 项目出口:[弗兰]做了一个门闩

> 原文：<https://hackaday.com/2019/07/04/project-egress-fran-makes-a-latch/>

[Fran Blanche]是精英黑客团队中的一员，他们有机会为[[亚当·沙维奇]的 Project Egress](https://wp.me/paBn4l-1wXM)做出贡献，这是一项庆祝 50 年前的这个月将人类送上月球的工程。幸运的是，她得到了一个伟大的任务:制作一个十五个闩锁中的一个的复制品，这十五个闩锁使阿波罗指挥舱舱口在真空的太空中牢牢固定，她正在做一项伟大的工作，用一些有趣的视频记录她的制作。

[下面的第一个视频](https://www.youtube.com/watch?v=E5I06MDMYhY)主要是她讲述自己的设计过程、材料选择，以及制作有点复杂的门闩的想法。参与该项目的所有 44 个制造商都可以选择他们将使用什么材料和方法来制造他们的零件，而[Fran]决定使用木材。她的第一个倾向是使用橡木和黄铜，这是一个很好的 80 年代氛围的组合，但是在[的第二个视频](https://www.youtube.com/watch?v=Q1Qr6yymcTc)，涵盖了更多最初的制作，她解释了她改用胡桃木的原因。不幸的是，她唯一的 CNC 选项是[牛头刨床原点](https://hackaday.com/2016/08/25/hands-on-the-shaper-origin-a-tool-that-changes-how-we-build/)，这带来了一些困难；手持工具需要一些复杂的夹具来安全地加工所需的小零件，并且它不能读取 STL 文件意味着[Fran]被复杂的软件工具链卡住来驱动工具。

随着[Fran]的进一步深入，还会有更多的视频，我们期待着看到她的部分和制作人的其余部分会如何出现。
