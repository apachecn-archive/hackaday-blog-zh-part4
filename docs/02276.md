# 葡萄浆解释道

> 原文:[https://hackaday.com/2019/03/01/grape-plasma-explained/](https://hackaday.com/2019/03/01/grape-plasma-explained/)

你可能看过这样的视频:一颗几乎被完全切成两半的葡萄在微波炉中产生等离子体。最近的一篇物理学论文用许多高科技设备研究了这一现象，现在实际的机制已经为人所知。【真理】采访了科学家，[用通俗易懂的语言解释了葡萄等离子体现象](https://www.youtube.com/watch?v=wCrtk-pyP0I)。你可以看下面的视频或者[直接看论文](https://www.pnas.org/content/early/2019/02/13/1818350116)。

结果是，葡萄的折射率大约是微波频率的 1/10，而葡萄在微波频率下的折射率可能高达 10。一颗完整的葡萄可以把所有的微波都截留在里面，但是两颗葡萄——或者两半——接触产生的电场足以电离空气。

结果包括从高速摄像机、光谱学等方面进行的观察。血浆主要来自钾和钠，很明显，葡萄的大小很重要，但这并不重要，因为葡萄中含有水分。事实证明，它甚至不一定是葡萄——水凝胶珠也可以。

该团队还建立了葡萄系统的模型，以了解内部发生了什么。葡萄的含水量足够高，所以模型只是假设葡萄是水的球体。

你实际上只需要做一点工作，就可以在微波中产生有用的等离子体。不过，我们怀疑你会在短期内[用葡萄血浆切割任何材料](https://hackaday.com/2018/06/22/linear-track-makes-plasma-cuts-neat-and-simple/)。
