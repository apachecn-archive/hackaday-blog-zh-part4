# 更好的绣花机，有 3D 打印和常用零件

> 原文：<https://hackaday.com/2019/08/07/a-better-embroidery-machine-with-3d-printing-and-common-parts/>

从概念上来说，一台普通的缝纫机可以让刺绣变得轻而易举:操作者可以随心所欲地移动布料，而缝纫机则负责用彩色线缝制图案和填充区域。然而在实践中，以这种方式获得好的结果要复杂得多。为了帮助和自动化这一过程，[sausagePaws]一直使用 CNC 来处理所有必要的运动控制。结果是 [DIY 刺绣机 V2](https://openbuilds.com/builds/diy-embroidery-machine-v2.8630/) ，它利用 3D 打印零件和通用组件，如 Arduino 和步进驱动器，实现了经济的 DIY 解决方案。

![](img/faadc0937c0f736f5fd104344b6b77f8.png)

照片中没有显示，但我们特别喜欢拧在桌面上的 3D 打印插座。这些支撑着缝纫机的“脚”,并允许它像一个模块化组件一样被对待，可以在需要时轻松地移除并正常使用。

该系统由运行在 Android 平板电脑上的 UI 组成，通过蓝牙与 Arduino 进行通信。Arduino 控制移动环(刺绣时将一段织物拉紧的框架)的门架，同时缝纫机放下针脚。

【sausagePaws】的[第一个版本](https://hackaday.com/2014/06/10/diy-embroidery-machine-sews-your-name-in-your-undies/)运行良好，但这种新设计真正利用了 3D 打印以及廉价有效的 CNC 组件可用性的增加。它仍然是一个正在进行的工作，在设计细节上有点轻，但你可以在下面的视频中看到它的所有动作。
