# 用于车间的 3D 打印相机起重机

> 原文：<https://hackaday.com/2021/05/05/3d-printed-camera-crane-for-the-workshop/>

当你以建筑材料为生时，记录过程摄像机的设置要花很多时间，损坏昂贵的设备是一种职业风险。[伊万·米兰达]对此太了解了，所以他建造了一个功能齐全的[摄像机起重机](https://www.youtube.com/watch?v=2rpTdpCd38w)来节省他的时间和摄像设备。休息后的视频。

基本设计是一个垂直的桅杆，末端装有一个旋转摄像机。铝桅杆伸缩增加垂直可调性，骑在胶合板基地与脚轮。铝制枢转臂被配重以抵消摄像头，平行杆机构允许相机与地面保持恒定的垂直角度。由于疫情期间家庭健身房的激增，健身房的砝码很难找到，所以[伊万]用装满沙子和螺丝钉的弹药罐代替。手臂顶部的一个较小的滑动配重允许微调。[Ivan]还希望能够进行水平滑动拍摄，所以他增加了一个滑轮系统，可以与离合器机构接合，以与相机保持恒定的水平角度。大多数配件和支架都是 3D 打印的，其中一些无疑是在他的[巨型 3D 打印机](https://hackaday.com/2019/12/18/giant-3d-printer-for-giant-projects/)上打印的。

我们当然可以看到这台起重机符合其设计目标，我们自己也忍不住想要一台。[Alexandre Chappel]不久前也制造了一台[摄像机起重机](https://hackaday.com/2020/09/27/diying-a-high-end-camera-arm/),它使用了一种完全不同的臂机构。尽管这些很酷，但与[【明古尔】的车间大小的 8 轴数控相机起重机](https://hackaday.com/2019/09/17/speed-up-filming-with-this-jawdropping-8-axis-camera-crane/#comments)相比，它们还是相形见绌。
