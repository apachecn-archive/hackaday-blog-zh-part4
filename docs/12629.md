# 洗衣机器人解决了 t 恤折叠的单调乏味

> 原文:[https://hack aday . com/2022/01/30/Washington-bot-tackles-the-tedium-of-t-shirt-folding/](https://hackaday.com/2022/01/30/laundry-bot-tackles-the-tedium-of-t-shirt-folding/)

撇开 Roomba 不谈，家用机器人仍在寻找它们需要真正起飞的杀手级应用。为了让另一种家庭自动化获得成功，设计师们必须找到最令人讨厌的家务，并按下按钮让它消失。t 恤折叠机器人可能是很好的第一步。

首先也是最重要的，向[ Stefano Pontoglio ]致敬，感谢他在这个项目上丰富的文档。不仅列出了构建洗衣机器人的完整说明，还列出了完整的用例分析，甚至对该领域现有技术的完整探索。[Stefano]的详尽分析产生了一组步进器驱动的面板，这些面板是从薄胶合板上激光切割下来的，并被安排在几秒钟内完成一系列折叠，这些折叠需要将一件 t 恤从扁平到折叠。

下面的视频展示了该文件夹的运行，虽然它现在还不是特别快，但我们将把它记在仍在开发中。我们可以看到一些需要改进的地方；用丙烯酸制造面板可能会使折叠的衬衫更好地从机器人身上滑落，气动执行器可能会使运动更快，折叠更锋利。[现实世界的衣物折叠挑战](https://hackaday.com/2016/02/24/the-challenges-of-a-laundry-folding-robot/)是真实存在的，但这是一个很好的开始，我们将继续寻求改进。
