# 如果咖啡已经在键盘里了，就不能把它洒在键盘上

> 原文:[https://hack aday . com/2021/10/11/cant-spill-coffee-on-your-keyboard-if-its-always-inside/](https://hackaday.com/2021/10/11/cant-spill-coffee-on-your-keyboard-if-its-already-inside/)

无论你生活在世界的哪个角落，无论你喜欢什么饮料，都太容易洒在键盘上了。显然，解决方案是将两者结合起来。这正是谷歌日本在过去的愚人节所做的，当时他们发布了 Gboard——一个包裹在可移动杯子周围的圆柱形键盘。但是，一旦你[开源了它并制作了一个构建指南](https://github.com/google/mozc-devices/tree/master/mozc-yunomi)，它或多或少还是一个笑话吗？

奇怪的地方就在这里:键盘上的每个汉字代表一种不同的鱼，而且它们是按照日语的语音顺序排列的。不过，你不会被鱼卡住——60 个键中的一个在鱼输入和常规平假名(基本的日语拼音字母)之间切换。在这些鱼的下面是挂在 ATMega32u4 上的低调的 Kailh chocs。我们只希望它是无线的。

我们喜欢他们开源这个键盘，它甚至在某种程度上是有意义的。为了制作一个好的愚人节视频，你实际上必须制作假产品。它越好(即古怪但合理)，[就会有越多的人喜欢它，并可能想要一个](https://hackaday.com/2016/11/16/star-trek-phone-dock-might-as-well-be-from-picards-night-stand/)。所以，如果你要这么麻烦，为什么不在 GitHub 上免费发布呢？注意自述文件的第二行是“这不是官方支持的谷歌产品”，我们认为这是不言而喻的。

休息之后一定要看看短片。如果你不懂日语，你可以打开隐藏字幕。
