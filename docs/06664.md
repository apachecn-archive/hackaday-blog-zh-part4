# 这台旧的落地式音响隐藏了一个酒柜

> 原文:[https://hack aday . com/2020/05/30/this-old-console-stereo-hides-a-liquid-cabinet/](https://hackaday.com/2020/05/30/this-old-console-stereo-hides-a-liquor-cabinet/)

曾经有一段时间，消费电子产品是陈述项目，旨在模仿优质家具，作为家庭的核心展示。装饰华丽的木制橱柜中的电视机，或者像餐具柜一样寻找整个世界的立体声音响。[泽瑟斯]在餐具柜里放了一个如此巨大的电唱机和收音机组合，并且很少使用 20 世纪 50 年代的家庭娱乐技术精华，他将其改造成一个隐藏的酒柜，带有电子控制和一个更现代的立体声系统，成为罗技媒体服务器多房间系统的一部分。

在移除了基于电子管的无线电底盘和杰拉德赛马轮转盘后，是时候清理它们的支撑木制品并安装从立式办公桌衍生而来的平台了。有了令人印象深刻的灯光和一对用于音乐选择的 VFD 显示器，不可避免地会有 Raspberry Pi 在表演。控制是通过一组隐藏的电容按钮来实现的，还有一个网络界面，允许智能手机舒适地播放音乐和酒精的神奇外观。整体可以在下面的视频中看到休息。

每当一件老式电子产品以这种方式被掏空时，总会有人感到不安，但事实是，这些一体式音响是在本世纪中叶大量生产的，并没有显著的价值。这一个可能已经失去了它原来的电子设备，但它仍然安全地生活在垃圾场，这个垃圾场已经夺走了它那么多的兄弟。幸运的是，这不是我们第一次看到用 Pi 保存的。
