# 测试 3D 打印扬声器驱动程序

> 原文：<https://hackaday.com/2019/09/07/putting-3d-printed-speaker-drivers-to-the-test/>

多年来，我们已经看到了许多试图 3D 打印扬声器外壳的项目，这些外壳不仅提供了一点定制风格，而且有望提供一半像样的声音。虽然正如你可能期望的那样，驱动程序本身总是安装在塑料外壳中的标准普通硬件。但是考虑到[保罗·埃利斯] 正在进行的研究，这种假设可能不会长久。

在过去的两年里，他对开发全频 3D 扬声器的追求使他经历了几次设计修改，每一次都经过了测试程序，将其频率响应与代顿和 Bose 等制造商的“真正”扬声器进行了比较。该项目正在进行中，但最近完成的驱动器设计迭代设法在 1 W 时超过 80 dB，在音频质量方面，[Paul]报告说他们可以与商用驱动器相媲美。休息过后你可以在视频中听到自己的声音。

最终，他希望能够将他的 3D 打印扬声器以套件形式出售给任何想要定制音频硬件的人。这个想法是，驱动器和外壳将完全模块化，允许用户用不同材料印刷(或不印刷)的组件替换单独的组件，以便他们可以将现场声音调整到准确的规格。为了促进驱动程序的快速重新配置，这些设计使用了一些巧妙的技巧，如将磁铁改为可移动的，而不是粘在一起，这样它们就可以被无损地替换掉。

这不是我们见过的第一个完全 3D 打印的扬声器驱动器， [Formlabs 展示了一个在 2015 年用 SLA 打印机制作的驱动器](https://hackaday.com/2015/01/27/3d-printed-speaker-pushes-rapid-prototyping-boundaries/)，而[我们实际上在今年早些时候看到了相同想法的雏形](https://hackaday.com/2019/04/17/a-mostly-3d-printed-speaker/)。但是[保罗]在这里所做的工作肯定是最彻底的，而且我们敢说是实际的，就这个概念而言是我们所见过的。
