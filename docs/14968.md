# 相机安装的立体声麦克风蓬松而有能力

> 原文：<https://hackaday.com/2022/10/02/camera-mounted-stereo-mic-is-fluffy-and-capable/>

通常，从您的相机出来的音频质量不是最好的。外置麦克风通常是一个很好的升级，[而来自[DJJules]的这个版本就是为了这个目的。](https://www.instructables.com/Camera-Mount-ORTF-Stereo-Microphone/)

这是一个立体声话筒系统，基于法国广播电视办公室(ORTF)的工作。ORTF 立体声技术定义为使用两个心形话筒，分别指向左右 110 度和 17 厘米，捕捉到高质量的立体声场，在作为单声道混音呈现时听起来也很不错。

该建筑使用一个简单的木架将两个驻极体话筒振膜固定在所需的方向。它们连接到 3.5 毫米插孔，因此可以直接插入 DSLR 或其他类似装备的相机的麦克风输入。人造毛皮覆盖的卷发器被用作麦克风的挡风板，使其看起来非常专业。该框架也有一个支架，这样它可以很容易地放在相机的冷鞋配件上。或者，也可以使用螺丝安装。

好的音频绝对是制作好内容的关键，而高品质的话筒绝对是你实现这一目标所需要的。这些年来，我们还推出了其他一些很棒的 DIY 话筒。休息后的视频。
