# 游戏机相机得到可笑的好镜头

> 原文:[https://hack aday . com/2021/12/19/game boy-gets-nally-good-lens/](https://hackaday.com/2021/12/19/gameboy-gets-ridiculously-good-lens/)

如何从一台 20 多年前的 Game Boy 相机中获得更好的照片？给它配一个 DSLR 镜头怎么样？这就是[ConorSev]所做的，老实说，结果比你预期的要好，正如[John Aldred]在他的帖子中提到的那样。你可以在下面的视频中查看相机。

3D 打印适配器可以让你将佳能 EF 镜头安装到 Game Boy 相机上，[这是我们过去见过的一个技巧](https://hackaday.com/2018/06/04/a-canon-lens-adapter-for-the-game-boy-camera/)。[ConorSev]查看了现有的适配器，并提出了您在这里看到的修订版本。仍然存在从相机胶卷上获取图像的问题，但幸运的是，[这也不是完全未被探索的领域。](https://hackaday.com/2017/12/01/arduino-saves-gameboy-camera/)

虽然这个项目可能没有什么新东西，但在玩具上使用高质量的镜头可以拍出一些有趣的照片，你想知道你能把整个想法推进多远。当然，无论你在前面放了多少镜头，你仍然要与原始图像传感器相抗衡，它几乎没有什么好处。尽管如此，我们还是对高质量变焦镜头的效果印象深刻。

我们敢打赌，Game Boy 相机的最初设计者从未想到它会有你在视频中看到的那种变焦能力。我们喜欢看到这些小手持设备超越极限。密码开采？[没问题](https://hackaday.com/2021/04/01/mining-bitcoin-on-the-nintendo-game-boy/)。莫尔斯电码？[小菜一碟](https://hackaday.com/2012/12/10/morse-code-transceiver-based-on-gameboy-color-camera/)。
