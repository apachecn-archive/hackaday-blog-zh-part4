# 借助一点软件支持，空中 3D 打印

> 原文：<https://hackaday.com/2021/07/19/3d-print-in-the-air-with-a-little-software-support/>

我们都喜欢 3D 打印，但打印任何有突出部分的东西都需要支持，对吗？也许不是。[Create Inc]有一个视频展示了一些 3D 打印品，[似乎不可能悬挂在空中](https://www.youtube.com/watch?v=QftTMX8RAXQ)——不是桥，而是漂浮在空中的环。你可以在下面的视频中看到效果。

文章的第一部分介绍了 gcode 的基础知识。5:30 分左右，【创造】谈他的灵感: [FullControl Gcode 设计师](http://fullcontrolgcode.com)。你可以用这个工具做很多事情，它启发了[类似的基于网络的版本](https://analysis230.github.io/index.html)。

这些工具的目的是使直接创建 gcode 比使用切片器更容易。你可以把它想象成 3D 打印的汇编语言——你可以用高级语言——3D 模型——做几乎所有的事情，但是如果你想要最终的控制，你可以使用汇编语言，或者，在这种情况下，gcode。

最初的工具使用 Excel，它不能直接可视化输出，也不能提供适当的错误检查。新工具解决了这些问题，而且更容易使用。

如果你知道 gcode，你可以做[很多有趣的事情](https://hackaday.com/2020/11/03/this-gcode-post-processor-squeezes-lines-into-arcs/)。你甚至可以在你的步伐中放一个弹簧[。](https://hackaday.com/2018/11/14/3d-print-springs-with-hacked-gcode/)
