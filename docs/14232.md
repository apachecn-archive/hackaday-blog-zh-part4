# 转动不规则形状

> 原文：<https://hackaday.com/2022/07/16/turning-irregular-shapes/>

如果你没有密切关注埃及机械师 YouTube，你可能已经错过了[液压屋]。自从他开始张贴 under[بيت الهيدروليك].，找到他变得更加困难不要让阿拉伯语让你分心，他用哑剧的形式表达了一切。

最近掉的一个是“[如何在车床上车削不规则形状](https://www.youtube.com/watch?v=yFviZxSELOk)”。我们不确定，但我认为他正在研究的部分是一辆三轮人力车的前悬架。中间的第一个金属离底部超过 30 厘米。没问题，他只是用一些废料做了一个很长的驱动死点，然后继续他的生意。

这绝不是唯一的酷视频。我们喜欢他的一个关于[远程泵送液压千斤顶](https://www.youtube.com/watch?v=gh8bt4lBhgI)和一个关于[制作自己的液压阀](https://www.youtube.com/watch?v=MPpKOUkjG84)的视频。

如果你对机械师感兴趣，不要错过他。每个视频都充满了非常漂亮的技巧，有时是滑稽地忽视了一些基本知识，如“在安装到车床上之前，可能更好地完成了焊接”，用切断工具转动(我认为)，偶尔放开卡盘键。这绝对是‘哦，继续吧’的机械车间工作。

我们喜欢发展中国家的专业人员用相对简单的工具制作的视频。通常爱好黑客都是在同一个位置，用车床和一些耐心而不是一个巨大的奥库马铣削。不久前，我们发布了[这篇关于制造螺旋零件](https://hackaday.com/2022/06/04/ingenious-indigenous-artful-screw-conveyor/)的文章，带着同样的“想象力和技能胜过任何一天的机械”的感觉。
