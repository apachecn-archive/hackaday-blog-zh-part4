# 3D 打印胶片扫描仪让家庭记忆重现

> 原文：<https://hackaday.com/2019/09/04/3d-printed-film-scanner-brings-family-memories-back-to-life/>

壁橱和阁楼里锁着一个历史宝藏，旧鞋盒里放着业余摄影师拍摄的电影胶片。他们捕捉了孩子们迈出的第一步，家庭度假，以及[比尔叔叔]做他惯常的滑稽动作的聚会。在数千英里的 8 毫米和超 8 毫米胶片上捕捉到的东西很少是重要的，但让一个家庭能够再次看到失散多年的亲人确实是一件强大的事情。

这就是安东·古彻的自动化 8 毫米胶片扫描仪的目标。是的，存在将电影、幻灯片和快照数字化的商业服务，但挑战在哪里？挑战在于它最终是什么。除了设计和打印大约 27 个定制零件之外，[安东]还为控制电子设备定制了 PCB。胶片处理是通过步进电机完成的，步进电机每次将一帧移入扫描仪进行扫描和裁剪。LCD 显示器允许档案管理员手动移动裁剪窗口，单个图像通过运行在嵌入式 Raspberry Pi 上的 ffmpeg 串连在一起。在下面的视频中，有一段 1976 年新加坡之旅的短片；我们发现数字化电影的质量非常好。

向[安东]致敬，因为他以这种方式成为家族历史学家。我们以前见过专门的 8 毫米数字转换器，但很少有这种精致的外观。我们以前也有过其他的存档尝试，比如这个[高速载玻片扫描仪](https://hackaday.com/2019/04/21/a-high-speed-slide-scanner-build/)。
