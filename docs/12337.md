# UV 印刷电路板

> 原文:[https://hackaday.com/2021/12/28/uv-printing-pcbs/](https://hackaday.com/2021/12/28/uv-printing-pcbs/)

我们一直很喜欢(托马斯·桑拉德勒的)3D 打印视频。但是他的最新作品不仅仅是关于 3D 打印。他展示了他是如何使用 DLP 打印机——毕竟它有紫外线——来曝光 PC 板坯料,并取得了很好的效果。老实说，一旦我们听到这个想法，我们立即看到了它是如何工作的。令人惊讶的是，越来越多的人没有像这样利用他们的 DLP 打印机。当然，[托马斯]像往常一样对这个话题进行了彻底的处理。

真的，这不完全是 3D 打印，尽管它使用了 3D 打印机。用紫外光和艺术品曝光电路板是一个已经存在多年的老过程，通常使用透明胶片和紫外光源。有了打印机，你可以用数字方式创作艺术品，而紫外光源已经存在了。

我们喜欢他用试纸法来计算曝光时间。让我们想起了过去的暗房时光。他还尝试在裸铜板上使用树脂作为抗蚀剂，但效果似乎不如你所希望的那样好。

然而，第一种方法使在 PC 板上获得艺术作品变得毫无痛苦——甚至比激光墨粉转移或直接激光曝光更容易。然而，你仍然要做化学步骤和我们最不喜欢的部分，钻孔。也许这是一个坚持表面贴装的好理由。

我们热爱并尊重为自己做事情的愿望，如果你有一台 3D 打印机，你可能也有。但是，除非你非常着急，否则现在专业制作 PCB 很便宜(正如我们曾经预测的那样)，而且在家里制作 PCB 真的没有意义，特别是因为很难复制电镀孔、阻焊膜，而且丝网印刷电路板也是额外的工作。

尽管如此，如果你需要一个快速转动的原型，或者你只是想满足自己制作 PCB，不要忘记你的 DLP 打印机是一个现成的紫外线曝光系统。
