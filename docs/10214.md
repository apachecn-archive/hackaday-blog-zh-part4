# 化学混合物使这款蓝图相机独一无二

> 原文:[https://hack aday . com/2021/05/24/cocktail-of-chemicals-makes-this-blue print-camera-unique/](https://hackaday.com/2021/05/24/cocktail-of-chemicals-makes-this-blueprint-camera-unique/)

当你今天看蓝图的时候，很有可能你看到的不是蓝色。大多数建筑计划、土木工程项目的图表，甚至消费产品的设计文档甚至从未出现在纸面上，更不用说像过去生产的大幅面印刷品那样以老式的蓝白色呈现了。我们认为这有点可惜。

幸运的是，[Brian Haidet]也渴望那些日子，以至于他建造了[这个大画幅蓝印相机](https://www.youtube.com/watch?v=ewQQX3fxQKk)来以传统方式拍摄照片。自然，这是那些在开始之前期望必须被适当地缩放的项目之一；毕竟，我们不拿着浸泡在有毒化学物质中的纸到处拍照是有原因的。[Brian]并没有被这种化学物质所吓倒，他从简单的接触式印刷开始了他的旅程，用 Sharpie 标记的透明胶片遮蔽感光纸，感光纸由铁氰化钾、重铬酸铵和草酸亚铁铵制成，以抵御太阳的紫外线。这一反应产生了深而丰富的颜料普鲁士蓝，一旦未曝光的溶液被洗掉，它与白纸形成了鲜明的对比。

[Brian]想要超越简单的接触印痕，下面视频中看到的大得离谱的相机就是结果。它只是一个或多或少不透光的盒子，一端有一个镜头，另一端有一张感光纸。“胶片”的有效 ISO 慢得令人难以置信，导致曝光时间过长。再加上镜头造成的扭曲，图像是——嗯，只能说是独一无二的。他们肯定有一种幽灵般的品质，而且普鲁士蓝的颜色也有很多优点。

我们之前已经看到过紫外激光使用的氰基化学，以及使用火棉胶工艺的[大幅面相机](https://hackaday.com/2016/01/27/i-built-myself-a-16x20-inch-camera-in-10-hours/)。我们想知道[Brian]的长曝光过程是否更适合于[太阳能摄影](https://hackaday.com/2020/04/28/pi-cam-replaces-pinhole-and-film-for-digital-solargraphy/)。
