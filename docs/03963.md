# 借助四通道 SDR，轻松实现测向

> 原文:[https://hackaday.com/2019/08/22/__trashed-3/](https://hackaday.com/2019/08/22/__trashed-3/)

测向一直是业余无线电爱好者的消遣。猎狐和其他测向活动娱乐了许多人，因为他们扫描天线寻找发射机。就像摇滚和喇叭裤一样，时间会改变一切，而[腐蚀剂]一直在尝试用一种非常现代的方式用 SDR 来寻找方向。

这项工作是通过使用 Kerberos SDR 实现的，这是一种基本上由四个 RTL-SDR 无线电协调工作的设备。通过为这些设备安装合适的天线并进行正确的校准，硬件可以用作强大的测向工具。

[腐蚀剂]巧妙地证明了这一点，在他的车上安装了装置，并开车四处寻找发射器。为了寻找 P25 控制站，他演示了帮助找到 FM 调制信号的硬件配置。方程的软件部分与 GPS 地图集成在一起，因此可以在收集数据的同时跟踪信号源的方位。随着时间的推移，软件会采集更多的样本，直到为发射器建立一个预期的位置。

该设置非常有效，并在很大程度上完成了所有繁重的工作，让用户只需简单地驾驶汽车。热图功能也非常酷，在你的下一部间谍电影中会非常好看。[我们在](https://hackaday.com/2018/09/10/direction-finding-and-passive-radar-with-rtl-sdr/)之前就已经推出了 Kerberos SDR，并完全期待在这个平台上看到更多优秀的作品。休息后的视频。
