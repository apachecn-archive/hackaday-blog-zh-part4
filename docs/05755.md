# 适合大孩子的高性能雪橇

> 原文：<https://hackaday.com/2020/02/22/a-high-performance-ski-sled-for-the-big-kids/>

滑雪橇是年轻人的一种消遣，尽管没有很好的理由说明为什么一定是这样。[JoshXarles]是一个对雕刻雪有着强烈热情的成年人，[并着手在今年冬天建造一个高性能的雪橇。](https://www.instructables.com/id/A-Better-Ski-Sled/)

这是一个基于滑雪板的设计，从一开始[Josh]的目标就是建造一个具有严格操控资质的装备。他最喜欢的跑步方式是几次 180 度的急转弯，所以能够很好地转弯而不失去速度是很重要的。这是通过使用带有精心设计的转向系统的侧切滑雪板实现的，允许雪橇像下坡滑雪者一样转弯。雪橇的框架由铝盒管制成，用螺栓固定在一起，形成一个坚固的结构。还有吸引人的木质装饰，使外观更加完美。

雪橇在最初的测试中表现令人钦佩，具有良好的转向感和足够的下坡速度。当然，如果天气允许的话，我们很想亲自试试。对于那些不具备适合滑下的地质特征的人来说，也有电动的选择。休息后的视频。
