# 冲浪板 LED 灯点亮海浪

> 原文:[https://hack aday . com/2020/09/10/surboard-led-strips-light-up-the-waves/](https://hackaday.com/2020/09/10/surfboard-led-strips-light-up-the-waves/)

冲浪是一项主要在白天进行的活动，主要是为了保暖和水上安全。当然，如果你喜欢在晚上冲浪，你可能会喜欢[Moritz Sivers]的最新项目——一个配备了动作感应发光二极管的冲浪板。

该建筑由一个普通的冲浪板组成，周围有一个通道，WS2812B LED 灯条粘在其中。由可充电锂电池供电，由连接到 MPU6050 三轴陀螺仪的 Arduino Nano 发出命令。这使得 Arduino 可以根据棋盘的移动来改变灯光模式。左转弯和右转弯、冲浪板和冲浪都有自己的动画。

这是夜间冲浪的一个有趣的转变，也更容易发现倒下的冲浪者。今年年底之前，我们希望在一部高端的 4K 冲浪电影中看到这种建筑重现。当然，如果你只是想在出发前知道条件是否合适，这个冲浪气象站可能就是为你建造的。休息后的视频。
