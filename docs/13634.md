# 气球吉他是绝对的气体，不管有没有氦气

> 原文:[https://hack aday . com/2022/05/14/balloon-guitar-is-a-absolute-gas-helium-or-not/](https://hackaday.com/2022/05/14/balloon-guitar-is-an-absolute-gas-helium-or-not/)

吉他通常是由木头制成的。无论是带有大开口空腔的木吉他，还是实心电吉他，通常都会使用大量的木材来建造。然而，[Mattias Krantz]向我们展示了替代的建造方法是完全可能的，通过[建造他自己的气球吉他](https://www.youtube.com/watch?v=515eUjWlBgI)。

气球吉他仍然像其他吉他一样有琴颈、琴桥和琴弦。然而，代替原声吉他的共振腔，可以安装一个大气球来代替。实际上看起来很有趣——安装了气球后，吉他的音量比没有共鸣腔时大得多。

吉他实际上是为了测试用氦气替换气球中的空气是否会改变声音的音高。当然，吉他的音高来自振动琴弦上的张力，所以改变谐振腔中的气体不会直接影响它。相反，就像[吸入氦气来影响人类的声音](https://www.youtube.com/watch?v=pWI3TVMNHxM)，变化是声音的音色，而不是基本音高本身。听起来好像吉他被赋予了微妙的高音增强。

这是一个有趣的构建，它向我们展示了以多种方式构建乐器的可能性，而不仅仅是使用传统技术。不过，如果你想进一步弹奏吉他的声音，可以考虑转向机器学习领域。
