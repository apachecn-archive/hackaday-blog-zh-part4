# 为那些艰苦的室内任务建造机器人漫游车

> 原文:[https://hack aday . com/2019/08/30/building-a-robot-rover-for-that-tough-indoor-missions/](https://hackaday.com/2019/08/30/building-a-robot-rover-for-those-tough-indoor-missions/)

制造一辆户外漫游车是很容易的事情，许多人都让他们在海滩和陌生的世界进行漫游活动。显然，新的领域是室内环境，这个领域正在被安德烈亚斯·霍尔多夫的螳螂漫游者所征服。

好吧，我们在开玩笑。这个项目最初是作为[Andreas]精致的[3D 可打印机械臂](https://hackaday.io/project/3800-3d-printable-robot-arm)的基础，但它甚至能够载着人到处走，正如休息后的嵌入式视频充分表明的那样。螳螂漫游者最引人注目的特征是它的[麦克纳姆轮](https://en.wikipedia.org/wiki/Mecanum_wheel)，这使它可以向任何方向移动，非常适合那些卡住会非常尴尬的狭窄地方。

麦克纳姆车轮是 3D 打印的，使电机和相关控制器成为这个包中更复杂的部分。轮子的设计包括铸造某种橡胶，以使轮子在地面上行驶时更加柔和。电子设备包括 TMC 5160 电机驱动器和 STM32F407VET6 MCU，以及配备 W5500 的定制“机器人盾”。

似乎还有很多调整正在进行中，以使该项目更加有趣。也许这是你下一次在办公室或本地黑客空间进行室内巡回会议的完美基础？
