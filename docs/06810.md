# 用四面体 LED 帽子点亮夜晚

> 原文：<https://hackaday.com/2020/06/10/light-up-the-night-with-a-tetrahedral-led-hat/>

人们出于各种原因接触电子产品，但我们猜测，根据调查结果，眨眼的能力可能相当高。[库奇伯特]在过去的十年里一直在参加 Deichkind 的演出，并一直想做一顶德国科技说唱乐队的标志性四面体 LED 帽子。

帽子里面是一个 Arduino Nano 驱动的 WS2812B LEDs 和一个便携式电池，为一切供电。由于 HC-05 蓝牙模块，该节目可以通过 Android 应用程序进行控制。丙烯酸板上的许多许多洞都是铣出来的，但它们也可以很容易地用激光切割，或者如果你有无限的耐心，用手钻出来。代码一旦被清理一点就出来了。你需要的其他东西已经在那里等着了。这个头盔甚至有自己的 lil' music 视频，我们在休息后仔细地进行了匹配。

很自然地，这让我们想起了这些年来在这个博客上一闪而过的蠢朋克·赫尔姆斯。这个手工焊接的可能是制作最精细的了。
