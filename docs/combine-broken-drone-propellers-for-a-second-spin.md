# 把坏掉的无人机螺旋桨组合起来进行第二次旋转

> 原文：<https://hackaday.com/2020/05/06/combine-broken-drone-propellers-for-a-second-spin/>

如果你曾经驾驶或观看过任何人驾驶比赛用无人机，无论时间长短，你都知道坠机只是游戏的一部分，螺旋桨是消耗品。[Adam]对此了如指掌，决定尝试将多个[损坏的螺旋桨组合成一个 3D 打印的轮毂](https://www.youtube.com/watch?v=bYekoXGFcNs)。

一个损坏的螺旋桨通常会有一个没有损坏的叶片，仍然连接在轮毂上。[Adam]修整了几个破损道具的损坏部分，并着手设计一个 3D 打印的轮毂，将松散的叶片连接在一起。轮毂被设计成让单个叶片移动，并随着马达旋转而折叠起来，类似于许多摄影无人机上的道具。

一旦[亚当]安装好轮毂，他就在一块木头上安装一个马达，让再生的螺旋桨测试它们的速度。一些轮毂在这个过程中出现了故障，这使得(亚当)能够找出薄弱环节并优化设计。这种快速测试是 3D 打印真正擅长的，它允许快速测试多个设计，而不是花费数小时在 CAD 上试图预见所有可能的问题。

然后，他用手头的零件制造了一架测试无人机，并进行仔细的飞行测试。轮毂比标准螺旋桨更厚，所以它限制了(亚当斯的)马达选择更长的轴。飞行测试出乎意料地顺利，只有在[亚当]将电池从 3 芯换成 4 芯并开始一些特技飞行后，一个集线器才出现故障。虽然这表明新的道具不适合赛车或特技飞行/自由式飞行的高力量，但它们可能对更平稳的巡航飞行很有效。轮毂也可以通过在 3D 打印的轴中添加钢销来改进，并仔细平衡组装的道具。

3D 打印机是黑客攻击飞行物体的一个不可思议的工具。我们已经看到了[共轴双翼飞机](https://hackaday.com/2017/05/09/hackaday-prize-entry-coaxial-drones/)、功能性[遥控 F-35](https://hackaday.com/2020/04/27/a-diy-functional-f-35-is-no-simple-task/) 和[升力体滑翔机](https://hackaday.com/2019/04/29/3d-printing-a-lifting-body-aircraft/)，毫无疑问还会有更多的黑客出现
