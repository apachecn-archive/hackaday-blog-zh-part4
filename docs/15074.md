# 电镀让 3D 打印的星球大战道具闪闪发光

> 原文:[https://hack aday . com/2022/10/13/plasting-makes-3d-printed-star-wars-prop-shine/](https://hackaday.com/2022/10/13/electroplating-makes-3d-printed-star-wars-prop-shine/)

众所周知，3D 打印最多只能生产相当普通的零件。即使是最光滑的树脂印刷品在外观上仍然是相当柔软和暗淡的。然而，如果你在电镀方面有所创新，有可能做得更好。[这个来自[hx innering drik]](https://www.youtube.com/watch?v=ywPi1osy7o8)的热雷管道具展示了一个 3D 打印件在一些后期处理和一些化学帮助下看起来有多好。

从网上找到的一个*星球大战*热雷管模型开始，并从一开始就用树脂打印它以获得尽可能好的表面光洁度。按照通常的树脂加工技术，在印刷和固化后清洗部件。从那里，细钢丝球和砂纸被用来使打印尽可能平滑。然后用喷枪喷上一层导电的铜涂料，并遮住配合面以避免破坏配合。

然后将该部件浸泡在酸性铜浴中，同时连接到电源，并在电镀过程中轻轻旋转。结果非常好，抛光后的镀铜零件接近镜面光洁度。然后在顶部镀上镍，使道具呈现出合适的银色。该道具最终被组装成一个 Arduino Nano，以运行几个 led 来实现视觉效果。

电镀不仅仅是为了让东西漂亮。[它还可以增强您的 3D 打印效果！](https://hackaday.com/2021/03/01/electroplating-3d-printed-parts-for-great-strength/)休息后的视频。
