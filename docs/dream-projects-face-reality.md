# 梦想项目面对现实

> 原文：<https://hackaday.com/2022/08/20/dream-projects-face-reality/>

你曾经有一个项目困在你的脑海里吗？一个好主意，你一直在想它？在你的脑海中，或者可能在纸上，仔细检查迭代和选项以及利弊，但是还没有经历将它变成现实的艰苦工作？在过去的几周里，我身边就有一个这样的人，现在是我开始建造的时候了。

我必须尽快开始，因为很少有任何项目能够毫发无损地从思想跳跃到现实，当我坚持思考项目太久时，我会变得太喜欢一些细节和细微差别，这些细节和细微差别可能无法完成，或者可能会妨碍第一遍的完成。当我真的喜欢一个(理论上的)问题的(理论上的)解决方案时，我会努力让它工作得比我应该工作的时间长得多，我可以说我现在越来越喜欢这个了。

治愈这种疾病的唯一方法是制作原型。当橡胶碰到路面，螺栓被拧紧，要么解决方案是好的，要么不是，再多的梦想也无法改变这一点。建筑是一个梦想工程的迷人歌曲的伟大解毒剂。虽然现在感觉我不希望幻想必须适应现实，因为它不可避免地会这样，但我知道让事情工作起来感觉好多了。它解放了我，让我开始梦想下一个项目……工作室！
