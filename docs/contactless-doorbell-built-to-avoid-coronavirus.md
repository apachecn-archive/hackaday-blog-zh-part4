# 为避免冠状病毒而建造的非接触式门铃

> 原文：<https://hackaday.com/2020/06/04/contactless-doorbell-built-to-avoid-coronavirus/>

人们常说，需求孕育创造力，在全球化的疫情时代，这句话比以往任何时候都更加真实。意识到普通的门铃可能是冠状病毒的潜在表面传播点，[【Casper Huang】迅速建造了一个。](https://www.instructables.com/id/Arduino-Coronavirus-Doorbell/)

这个版本避开了我们都熟悉的典型按钮。相反，它依靠超声波距离传感器来检测门前挥动的手(或脚)。一只 Arduino Leonardo 在表演，当超声波传感器被触发时会发出蜂鸣声。为了避免修改公寓的门，建筑被安置在一对纸箱里，用胶带粘在门的底部，电线从下面穿过。

这是一种处理无接触交付的简洁方式。随着世界从新冠肺炎疫情吸取经验教训，我们想象像这样的小改动在未来的设计中会变得更加普遍。毕竟，一点一滴都有帮助。休息后的视频。
