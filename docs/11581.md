# DIY 神童和其他人组成了这个开源的军火库

> 原文:[https://hack aday . com/2021/11/07/DIY-wunderwaffe-and-others-make-up-this-open-source-arsenal/](https://hackaday.com/2021/11/07/diy-wunderwaffe-and-others-make-up-this-open-source-arsenal/)

除非你彻夜不眠，让十几台打印机运转起来，否则赶在万圣节前制作由射线枪项目的安德鲁设计的这些漂亮的道具武器之一可能已经太晚了。其中大部分来自《使命召唤:僵尸》,尽管也有来自堡垒之夜的令人敬畏的小型迪斯科手榴弹。

[![](../Images/48f196e5cc7a59f989253d46f88e484f.png)](https://hackaday.com/wp-content/uploads/2021/10/wunderwaffe-inner.jpeg) 所有的项目都很棒，但我们选择突出《COD:僵尸》中的神童 DG-2，因为，嗯，真空管。对于那些不熟悉武装部队操作的人来说，这些真空管就像弹药盒一样。一旦它们空了，你就用那个红色的大开关把它们关掉，然后用控制杆一次弹出一个，就像在游戏里一样。

里面是一个羽毛 M0 快递，运行 RGB 发光二极管，并使用霍尔效应传感器读取快速更换弹药杂志中的磁铁。休息过后，您可以在演示视频中看到它是如何工作的。

有几个道具武器的材料清单，以及装配图和支持论坛，供任何想自己制造的人使用。不想自己收集所有零碎的东西吗？[Andrew]正在出售射线枪的硬件包，但是如果你想建造这个神奇的华夫饼干，你必须自己去找零件。

你是一个想让孩子们远离你的草坪的怪杰吗？用一把巨大的 NERF 枪吓跑他们。
