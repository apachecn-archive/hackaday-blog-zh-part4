# 3D 打印气象站获得无线升级

> 原文：<https://hackaday.com/2019/05/05/3d-printed-weather-station-gets-a-wireless-upgrade/>

气象站可以是任何东西，从有趣的家庭科学练习，一直到规划和天气预报的有用工具。[Rob Ward]就是这样一个人，他开发了自己的气象站，最近还进行了无线升级。

我们第一次介绍[Rob]的作品是在 2018 年，我们注意到大部分 3D 打印的气象站对于家庭实验者来说是一个特别有用的工具。现在，随着从天气传感器到基站的 433 MHz 无线链路的增加，这一点的效用得到了提高。在 Github 上，[Rob]很好地解释了所使用的曼彻斯特编码方案的基础，并开发了一个系统，可以解码来自俄勒冈州科学气象站的信号[。](https://github.com/robwlakes/ArduinoWeatherOS)

[Rob]使用气象站[报告 Tyers 湖海滩](http://www.laketyersbeach.net.au/weather.html)的天气情况，为该地区任何可能考虑去海边旅游的人提供有用的信息。这并不像问 CB road 频道的人那样有趣，但对于你的麻烦来说，这是一个更准确的方法。休息后的视频。
