# 本周失败:拆卸特斯拉电池

> 原文：<https://hackaday.com/2019/09/07/fail-of-the-week-taking-apart-a-tesla-battery/>

把一个汽车大小的物体推几百英里需要很大的能量。几加仑汽油或几千个锂电池就能完成这项工作。这当然是大量的电池，而且比把大块的金属扔在轮子上有更多的潜力可以利用。如果你有如何更好地将这些电池用于其他用途的想法，那当然是一个选择，[尽管这并不总是像看起来那么容易](https://www.youtube.com/watch?v=gQcvZn7-_VE)。

在此视频中，[even engineering]的[Kerry]获得了一个特斯拉 Model 3 电池组，并开始将其拆开。与其他特斯拉电池不同，更不同于 Leaf 或 Prius packs，Model 3 电池极难使用。作为一种制造成本节约措施，特斯拉似乎发现，与电池更加模块化和可维护的其他方法相比，将单个电池粘合在一起会更便宜。这意味着，为了在不损坏细胞的情况下移除单个细胞，在你开始用 PEX 楔和锤子敲打细胞之前，必须移除几层胶水和塑料。这种方法非常耗时。

如果你刚好有一个 Model 3 电池，那么[Kerry]指出，如果你有时间，可以重复使用这些电池，但除非你真的需要这 21700 个电池中的能量密度，否则不推荐这样做。显然，在 Model 3 包装之外不容易找到它们，不管怎样，似乎使用日产 Leaf 的电池会容易得多。
