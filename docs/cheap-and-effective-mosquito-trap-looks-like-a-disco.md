# 便宜有效的捕蚊器看起来像迪斯科舞厅

> 原文：<https://hackaday.com/2020/08/03/cheap-and-effective-mosquito-trap-looks-like-a-disco/>

言语无法表达人类对蚊子的集体厌恶，这是理所当然的！这些寄生虫叫卖，吸血小动物造成了大量的人类痛苦。疟疾等蚊子传播的疾病仍然占人类死亡率的很大一部分，特别是在世界的不发达地区。所以难怪人们试图减少他们的数量；请看【雅各布斯克】最新的 [$40 的捕蚊器。(视频，嵌在下面。)](https://www.youtube.com/watch?v=B8AbO3pOhjc)

这个想法非常简单，开启了广泛部署的潜力。捕集器的底座和主体由三个五加仑的桶组成，中间夹着一个迷你台扇，为主要捕集箱提供吸力。顶部的桶上有一个开口作为入口，中心安装了一个老式的白炽灯黑光灯，其红外和紫外输出刚好足以吸引这些小害虫，他们肯定会后悔误认为是黑光灯狂欢。

[jacobsk]在他的视频中也很好地展示了其建设的每一步。虽然这种解决方案是故意低技术含量的，但是看看这种公认的用激光炮塔杀死蚊子的方法。
