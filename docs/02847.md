# 如何在家里建造一个小型金属炉

> 原文：<https://hackaday.com/2019/05/04/how-to-build-a-small-metal-furnace-at-home/>

铸造是制作你自己的定制金属部件的好方法。然而，铸造需要某种能够产生足够高的温度来熔化金属的炉子。我们中很少有人把这些东西放在身边，但是不要害怕。用常见的材料在家里建造一个基本的燃气炉是可能的。

这个炉子是[Ahmed Ghr]的作品，建造简单。这个想法是制造一个模具，在其中浇注混凝土来建造熔炉。一个钢桶被切开，用作模具的外部，一根管子插入底部，作为空气和气体的供给器。然后将塑料桶插入钢桶中，并用垫片固定，以形成内部燃烧腔。混凝土被倒入并凝固。一旦完成，钢桶被切掉，然后在炉子上生火融化里面的塑料。类似的技术被用来生产盖子，炉子就完成了。

这是一个用最基本的工具建造的建筑，至少可以作为低熔点金属的熔炉。事实证明，我们最近看到了很多水泥项目。休息后的视频。
