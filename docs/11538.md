# 开始你的万圣节吧！

> 原文：<https://hackaday.com/2021/10/02/get-yer-halloween-on/>

万圣节基本上是*为黑客建造的*。除了明显的荒谬服装的制作，也有机会展示你的技能，无论是机械，音频或视频。这也是展示我们最酷的技巧来激励年轻的原始黑客的好时机。如果你需要灵感，[我们有 150 个想法](https://hackaday.com/tag/halloween/)。

[![](img/c0b0869d8927084feb0310afdacb5cc4.png)](https://hackaday.com/wp-content/uploads/2021/10/candy-dispenser-800_thumbnail.png) 

【布兰克利】的[糖果分配器](https://hackaday.com/2020/10/19/stomp-button-receive-candy/)

然而，我个人对万圣节的问题是，我总是在最后一分钟才开始，我的想法远远超出了我的时间预算。或者因为这一切都是在最后一分钟完成的，一大堆理论上应该“行得通”的想法撞上了实践这个不可移动的对象。至少去年的幽灵音效是这样的——我和儿子花了太多时间收集和录制可怕的音频样本，以至于我在运动检测器设置的灵敏度刚刚好的情况下耗尽了时间，然后电池在半夜没电了。

但今年会有所不同，我发誓！我打算早点完成并测试它，有充裕的时间来调试不可避免的蜘蛛。你也可以发誓。现在*开始*你的万圣节项目。或者至少下个周末。

### 内容

如果你需要更多的鼓励来启动你的黑色和橙色黑客机器，想想 Hackaday.io 的[万圣节黑客节](https://hackaday.io/contest/180664-halloween-hackfest)。它将持续到 10 月 28 日，你所要做的就是在 IO 上记录你的万圣节项目，然后点击“提交”按钮。截止日期是 28 号，在正式交易之前，你还有几个晚上的时间来调试任何不工作的东西。奖品是在 Digi-Key 的购物狂欢，如果你在构建中使用任何 Adafruit 部件，Adafruit 将双倍赠送礼券。

如果你不在乎愚蠢的万圣节，那也很酷。(格林奇！)2021 年的[hack aday 奖已经进入最后一轮外卡赛](https://hackaday.com/2021/09/27/its-wildcard-time-your-last-chance-to-enter-the-hackaday-prize/)。如果你的项目不符合前面的任何一个类别，我很肯定它会很好地符合任何事情的阶段。发疯吧。我们很想看看你在做什么。
