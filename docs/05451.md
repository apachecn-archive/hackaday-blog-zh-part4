# 新零件日:SK6812 Mini-E。可手工焊接的兼容新像素的 LED！

> 原文：<https://hackaday.com/2020/01/28/new-part-day-sk6812-mini-e-a-hand-solderable-neopixel-compatible-led/>

通常，当我们给你一个新的零件时，它涉及一个你以前从未见过的组件。鉴于 [SK6812 Mini-E](https://www.youtube.com/watch?v=oLYkzn0jwg8) 是一种 WS2812 或 Neopixel 兼容的多色 LED，这种类型的 LED 已经上市有一段时间了，因此【Robert Fitzsimons】的这一发现的主题稍微偏离了常规。

让这个组件变得新颖的是它的包装。SK6812 的 Mini-E 变体去年才出现，现在已经在全球速卖通找到了较小的订单量。其特殊之处在于，它在封装底部有一组扁平引脚，而不是常见的焊盘。这意味着与它的前辈不同，它很容易手工焊接，正如他通过将一组引线连接到一个引线上所展示的那样。

引线出现在器件侧面的中间位置，似乎是为了嵌入 PCB 孔中而设计的。他用一块条板演示了这一点，并表示它们将是许多小项目(如劣质附加板)的好选择。

我们之前已经接触过几次无引脚 SK6812s，在此过程中，我们注意到[在某些方面，它们优于他们跟随的 ws 2812](https://hackaday.com/2017/01/25/ws2812b-led-clones-work-better-than-originals/)。
