# 听一首由定制任天堂 LABO 波形卡制作的歌曲

> 原文:[https://hackaday . com/2018/10/14/听歌定制-任天堂-labo-waveform-cards/](https://hackaday.com/2018/10/14/listen-to-song-made-from-custom-nintendo-labo-waveform-cards/)

[亨特·欧文]一直在为任天堂 Switch 忙于任天堂 LABO 的钢琴。特别是，他一直忙于创建自己的定制波形卡，这极大地扩展了可折叠纸板装置的功能。如果这听起来很熟悉，那是因为我们介绍了[他创造与钢琴兼容的 3D 打印波形卡的原始方法](https://hackaday.com/2018/04/26/reverse-engineering-nintendo-labo-waveform-cards/)，但从那以后，他进一步发展了他的工作。他不仅通过从超级任天堂游戏中取样乐器创造了新的和更复杂的卡片，他甚至试验了基于元音的卡片，试图看看事情能走多远。通过将正确的元音*分层，他能够说出(几乎无法辨认的)短语“我喜欢你”、“你喜欢我”和“我们走吧”。*

这三个短语组成了他用为任天堂 LABO 的钢琴定制的波形卡创作的一首歌的歌词(依稀可辨)，名为**我喜欢你**。这首歌是[在 6:26 标记](https://youtu.be/x34l0-_tB-A?t=386)在下面的视频嵌入，但整个视频是值得一看赶上【猎人】的工作。这首歌也是[在 soundcloud](https://soundcloud.com/hunter-irving/i-like-you) 上主持的。
