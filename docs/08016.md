# 以机械的方式旋转视频轨道

> 原文:[https://hack aday . com/2020/10/12/spin-the-video-track-with-a-mechanical-flair/](https://hackaday.com/2020/10/12/spin-the-video-track-with-a-mechanical-flair/)

最难做好的用户界面之一是视频编辑。即使经过四五个小时的直接编辑，它也是复杂而精细的，需要大量的精确度。为了将一些界面带到现实世界中，[扎克·弗里德曼]建造了一个机械视频编辑键盘。

有问题的键盘具有流行的快捷键和按键，可以轻松完成编辑的不同部分。当然，最大的特点是大的搓擦旋钮，允许[扎克]精确地快速浏览长视频。我们以前在 Hackaday 上见过相当多的机械键盘[不是传统键盘](https://hackaday.com/2019/08/09/a-macro-keyboard-in-a-micro-package/)，比如[这个数字键盘](https://hackaday.com/2019/05/13/a-customizable-open-source-mechanical-numpad/)或者[这个宏键盘](https://hackaday.com/2020/05/13/circuitpython-macro-pad-is-one-build-that-wont-bite/)。

这个项目的一个独特的限制是 Zack 有两天的最后期限。这个自我设定的截止日期是为了帮助集中工作，并推动其完成。这意味着它必须以这样一种方式设计，路障或麻烦的功能可以被设计或完全删除。在它的核心，这个项目只是 14 个机械开关，4 个电位计，和一个驱动它的小东西。这是设计，原型和思想进入这个项目，使它值得注意。这里有很多关于如何管理项目时间表的课程和关于如何实际完成项目的建议。

代码、STL、图表和指令都在他的 GitHub 上[。](https://github.com/ZackFreedman/Big-Wheel)
