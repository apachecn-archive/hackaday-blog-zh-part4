# 分光计价格低廉且功能强大

> 原文:[https://hack aday . com/2019/02/09/spectrometer-is-bufferent-and-capable/](https://hackaday.com/2019/02/09/spectrometer-is-inexpensive-and-capable/)

我们知道白光通过棱镜看到从另一面出来的色谱的效果。肉眼不会注意到，但彩虹并没有完全跨越[罗伊 g .比夫]的范围。有很少的颜色模糊在一起，这些缺失的部分是白光穿过或反射的物质的指纹。眼尖的人会意识到我们在谈论分光光度法，这是识别那些指纹，并确定正在观察的是什么和正在观察的是多少的*。实现这一功能的设备叫做分光计，[贾斯汀·阿特金]邀请我们一起参观[的作品](https://www.youtube.com/watch?v=pIk8I10ZmYY)。视频也可以看下面。*

随着构建的进行，我们学习了分光光度法的工作原理，从光子是如何产生的以及为什么色谱中会出现间隙开始。这都是关于电子的，我们的一些经验丰富的光谱仪用户已经知道了。该建筑使用激光雕刻机切割的木制纳米滴样式外壳。它需要一些视频中提到和显示的改进，所以你会希望手头有一些铝带。材料清单的其余部分包括“黑色 2.0”，它声称是“最粗糙、最平的黑色丙烯酸漆。”也许这对[其他光学项目](https://hackaday.com/2017/10/25/interference-scanner-with-clear-instructions/)会派上用场。购买合适尺寸的第一面镜子可能是明智的，但你总是可以用精心挑选的工具制作定制的镜子。

铝带被用来制作通过彩虹的狭缝，因为它必须具有干净的边缘，以便让最小的光线进入而不模糊，但这种安排总是可以升级为一对可移动的剃刀刀片。激光切割边缘不够干净或狭窄。衍射材料应该购买，因为它既便宜又容易找到，但在紧要关头，一张未上漆的 CD 也可以。

该软件是首选的武器，演示表明，绿色激光笔在绿色范围内有一个尖锐的尖峰，在红外范围内有一个额外的峰值。如果你需要一个在可见光谱中也能看到的[红外激光器，这是个好消息。[贾斯汀]涵盖了他计划做的一些测试，包括量化 DNA，并展示了](https://hackaday.com/2010/09/25/laser-mic-makes-eavesdropping-remarkably-simple/)[廉价光谱仪](https://hackaday.com/2018/05/29/tiny-25-spectrometer-aims-to-identify-materials-with-ease/)如何测量存在和饱和度。
