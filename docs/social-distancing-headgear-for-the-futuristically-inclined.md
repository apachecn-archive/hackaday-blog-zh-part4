# 面向未来的社交距离头盔

> 原文：<https://hackaday.com/2020/06/02/social-distancing-headgear-for-the-futuristically-inclined/>

那些关注经典电影的人会记得 1985 年的*回到未来*，尤其是其中的科学家角色埃米特·布朗博士。当主角马蒂·小飞侠发现自己在 1955 年第一次见到年轻的布朗博士时，后者正戴着一个实验头盔，用来读取思想。它不起作用，但它是一种美学，我们在[hkan Lidbo]的 [Corona Hat](https://www.hakanlidbo.com/coronahat) 中想起了它，这是一种社交距离工具，将距离传感器集成到一件头饰中。

这个装置非常简单，半个地球上安装了一组汽车倒车传感器和一个自主真空吸尘器的电池。它喷了一层亮橙色，在休息时间下面的视频中，他戴在头上在镇上走来走去。每当有人或物靠得太近时，它就会发出哔哔声，据我们所知，这是有效的。虽然我们不太确定这个造型，但对我们和埃米特·布朗来说，它有点太让人想起 2005 年*飙风天王*电影中的角色[希夫](https://www.youtube.com/watch?v=bJuDofIoW34)戴着犰狳的盔甲当帽子。也许更传统的头饰可以让它少一些斜视。

[这不是我们见过的第一个超声波社交距离传感器](https://hackaday.com/2020/04/09/ultrasonic-sensor-helps-you-enforce-social-distancing/)。也许这个舞台上最值得关注的项目应该是[那个有着高电压的项目](https://hackaday.com/2020/04/19/enforce-social-distancing-with-high-voltage/)，它的吠声比它的咬声更令人害怕。
