# 一张 2D 的照片可以制作出 3D 照片

> 原文:[https://hack aday . com/2022/09/15/a-2d-image-makes-a-3d-print/](https://hackaday.com/2022/09/15/a-2d-image-makes-a-3d-print/)

当你想象 3D 打印艺术时，很容易想象网上发布的不同种类的雕塑和小雕像。虽然这些项目本身需要大量的时间和创造力，但【塞萨尔·加莱拉】向我们展示了一种不同的方式来制作 3D 打印艺术，通过[将 2D 的图像变成完全纹理化的 3D 打印品](https://itslitho.com/lithophane-guides/turn-an-image-into-a-cool-3d-model-no-lithophane/)。

这个项目遵循了源于 [lithophanes](https://www.makeuseof.com/what-is-a-lithophone-how-to-3d-print-lithophanes/) 的类似技术，该技术通过穿过物体的光产生图像。[塞萨尔]在休息时间下面的视频中详细介绍了如何使用[的平版印刷工具](https://www.youtube.com/watch?v=S7EpV_uUufs&t=9s)使用多色打印机构建完全不透明的黑白图像。

通过绘制地形图来建造(或印刷)石膏像，以使光线在某些地方更容易或更难通过。看起来较暗的区域较厚，层数较多，看起来较亮的区域层数较少。这是一种漂亮的视觉错觉，但这些艺术块本身并不是彩色的。

诀窍是首先使用 lithophane 工具开发 3D 模型，以创建不同的高程(确保最低高程仍然足够厚以至于不透明)，但在导出时保留模型上的不同颜色。多色 3D 打印机将能够添加灰色和黑色细丝，因为它打印的高度越来越高。如果您没有多色打印机，您可以在 3D 打印文件上添加暂停，以在几层后切换出细丝，从而达到类似的效果。

我们总是在寻找我们可以打印的不同东西，能够将数字艺术品转化为 3D 模型就是一个很好的例子！
