# 3D 打印鞋制造大脚脚印

> 原文：<https://hackaday.com/2022/03/02/3d-printed-shoes-make-bigfoot-tracks/>

如果他的新鞋 [Cryptide 运动鞋](https://sintratec.com/2022/01/the-cryptide-sneaker-by-stephan-henrich/)上市，斯蒂芬·亨利克(Stephan Henrich)很可能会掀起一股目击大脚野人的热潮。该鞋是使用 Sintratec 的激光烧结打印机在柔性 TPE 中完全 3D 打印的。这种鞋以神秘动物学命名，事实上，由于它的关节脚趾和鳞状鞋底，会留下令人困惑的足迹。

从鞋底的外观来看，它应该非常舒适，我们假设如果你是 3D 打印这些，你会扫描或精确测量预期的脚，以实现完美的合脚。下面可以看到一个关于鞋子的视频。

整只鞋由一种材料制成，但是鞋的不同部分通过改变 TPE 的厚度而具有不同程度的硬度。显然，Sintratec 真的很喜欢鞋子，因为他们也展示了由[丹尼尔·雪莉]设计的[地球 Moc](https://www.youtube.com/watch?v=M2zSz0MEp9k)。我们想知道为什么他们没有在视频开始时为那个人印第二只鞋。

我们不确定鞋子的拓扑结构是否适用于你在车库里找到的典型打印机。但是如果你打印了一些鞋类，一定要[让我们知道](https://hackaday.com/submit-a-tip/)。提醒你一下，我们并不是没有见过一些 [3D 打印运动鞋](https://hackaday.com/2018/10/05/3d-printed-sneakers-are-now-a-thing/)。[不过，鞋子装饰](https://hackaday.com/2017/04/09/shoelace-locks-keep-your-fancy-footwear-firmly-attached/)似乎更容易印刷。
