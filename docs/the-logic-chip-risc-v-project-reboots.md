# 逻辑芯片 RISC-V 项目重启

> 原文：<https://hackaday.com/2020/11/09/the-logic-chip-risc-v-project-reboots/>

RISC-V 架构正在不可阻挡地从其理论起源向主流发展，因为曾经只能在异国情调的 FPGA 上完成的事情现在可以在少数微控制器以及一些更强大的处理器上看到。这很令人兴奋，因为它为我们提供了运行开源操作系统的完全开源硬件的前景，但不仅仅如此。RISC-V 与其说是一个特定的处理器内核，不如说是一个可以在多个级别中的任何一个级别上实现的规范，最简单的形式甚至可以使用 74 个逻辑芯片来实现。这是[罗伯特·巴鲁克]的 LMARV-1 的目标，它在一两年前引起了轩然大波，但随后就中断了。我们很高兴地注意到[他发布了一个视频，宣布项目](https://www.youtube.com/watch?v=YgXJf8c5PLo)的重新开始，以及一个重大的重新设计。

我们将视频放在了休息时间的下方，它不仅仅是一个简单的项目公告。相反，它是对设计决策和处理器物理架构的深入解释。它相当于处理器设计的初级读本，虽然这是一个长期的观察，但我们要说，如果你的兴趣在那个方向，你不会失望的。

[我们在 2018 年初](https://hackaday.com/2018/02/09/a-risc-v-that-the-rest-of-us-can-understand/)首次报道了 LMARV-1，因此我们很高兴看到它重新取得进展，并期待看到它继续取得进展。
