# 微型封装中的宏键盘

> 原文：<https://hackaday.com/2019/08/09/a-macro-keyboard-in-a-micro-package/>

还记得 2000 年代早期到中期，你能找到的几乎所有廉价 USB 键盘都开始在布局中包含大量的媒体键吗？如今，尤其是如果你有一个定制的或缩小尺寸的机械键盘，这些都无处可寻。每当我们现代人需要这些额外的钥匙时，我们就不得不求助于外部设备，而[[Gary]Knobo 看起来就像是一个可以直接从花哨的零售包装中出来的东西](https://hackaday.io/project/166947-knobo)。

Knobo 是一个小型宏键盘，有 8 个机械樱桃式按键和一个可点击的旋转编码器旋钮，是其主要功能。每个按键和旋钮手势都可以定制为任何宏，旋钮可能有五种手势，总共有十三种输入。最重要的是，它的构建和展示看起来如此圆滑和干净，我们敢发誓这是一个直接来自青少年工程印钞机的产品。

您可以对这些输入进行的操作范围很广，从带有音量旋钮的简单媒体控制，一直到使 Photoshop 艺术家的生活更轻松的快捷方式。目前，你只能用在线串行编程器对 Knobo 基于 Arduino 的固件重新编程，以改变输入的功能，但[Gary]目前正在开发配置软件，这样没有任何编程知识的用户也可以定制它。

旋钮只是[那些](https://hackaday.com/2019/03/19/a-classy-usb-knob-for-the-discerning-computerist/)中的一个，每个[都想用](https://hackaday.com/2019/04/30/a-physical-knob-for-browser-tabs/)来[控制他们的电脑](https://hackaday.com/2013/11/14/cloning-the-trinket-for-a-usb-volume-knob/)，很像[巨大的红色按钮](https://hackaday.com/2019/04/03/giving-an-industrial-push-button-usb-elegantly/)。替代输入设备的范围可以从[无障碍设计的](https://hackaday.com/2018/09/25/diy-puff-suck-interface-aims-for-faster-text-input/)到仅仅是[完全好玩的](https://hackaday.com/2018/09/23/send-smooches-over-skype-with-the-kiss-interface/)。无论他们的灵感是什么，看到这些项目的创造力总是很好。
