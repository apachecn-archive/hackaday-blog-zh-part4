# 老式自行车获得布里格斯和斯特拉顿的权力

> 原文：<https://hackaday.com/2019/11/20/vintage-bike-gets-briggs-and-stratton-power/>

易贝普及了自行车的机动化过程，来自中国的廉价引擎与一些零件结合在一起，将所有东西连接在一起。然而，如果你的品味更复古一点，奥利弗的身材可能更适合你。

从一个真正的布里格斯和斯特拉顿从一个旧的旋耕机解放，这个引擎有合法的老式证书。棕色看起来光彩照人，与去年相似色调的自行车搭配在一起。发动机的驱动力通过皮带传递到中间轴，然后中间轴通过链条将动力传递到后轮。皮带张紧器充当基本的离合器，允许发动机在脱离时与传动系统断开。

复古的组件，结合适当的配色方案，使这一个美妙的巡洋舰，渗出风格。虽然由于缺乏真正的离合器和噪音，它可能不适合市区通勤，但在阳光明媚的日子里，它会成为一些乡村道路的绝佳选择。我们也看到了类似风格的电动自行车。休息后的视频。
