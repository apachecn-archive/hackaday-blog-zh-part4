# 乐高绳绕线机暗示更大的可能性

> 原文：<https://hackaday.com/2022/03/29/lego-string-winder-hints-at-greater-possibilities/>

除非你是 1997 年票房大片 *Mouse Hunt、*的狂热粉丝，或者活跃在弦乐行业，否则你可能不会花太多时间去思考这些线团是如何制作的。然而，[奥德]制造了一台整洁的小型绕线机，让我们思考乐高机器到底有多大用处的可能性[。](https://www.youtube.com/watch?v=243nUNLD7fg)

这个设计使用了一个看起来很大的环形齿轮来驱动旋转臂，旋转臂将线带到线轴上。当操作机器的手动曲柄与纱管和纺纱臂机械连接时，纱管本身也随着纺纱臂一起旋转。随着手臂和线轴的转动，线被放置在线轴上，形成一个令人愉快的线团，看起来就像你在商店里买的一样。

我们怀疑，除非[ord]正在做一些我们还没有看到的非常有趣的事情，否则绳子很可能来自卷轴或球，这台机器只是用来演示这个过程是如何工作的。

然而，它确实展示了乐高机器如何做真正的工作。我们很乐意看到乐高投入更多的实际用途，比如缠绕皮卡或变形金刚，或者其他手工完成的令人抓狂的繁琐工作。休息后的视频。
