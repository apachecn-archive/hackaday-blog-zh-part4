# 骑自行车铲雪

> 原文:[https://hackaday.com/2023/01/01/snow-plowing-by-bicycle/](https://hackaday.com/2023/01/01/snow-plowing-by-bicycle/)

很少有挑战比试图骑自行车游遍大多数北美城市更困难或更危险。可悲的是，自行车基础设施不仅不足以保障安全出行(如果有的话)，而且经常没有按照任何合理的标准进行维护。这种情况在寒冷地区会加倍，那里的自行车道在冬天降雪后会被废弃。[Phil]最近在加拿大西部的一场降雪后发现自己处于这种情况，并决定 DIY 他自己的自行车驱动的扫雪机，以帮助清理自行车道。

该犁是围绕电动辅助货运自行车建造的，这在北美几乎与自行车基础设施本身一样罕见，但却独特地适合扫雪工作。它有一个长轴距和一个大的前货物区，如果需要，可以压下，以确保犁与地面接触良好。犁本身是由 55 加仑的塑料桶制成的，这些桶被切割成两个铲子部分，并用一个 2×4 的木制框架连接到自行车上。犁可以通过棘轮皮带机构升高或降低，塑料铲可以相对容易地跳过道路上的颠簸。

通过在自行车上安装这个相对简单的装置，[Phil]可以确保他在温哥华附近经常光顾的小路更适合冬天的自行车旅行。在冬天骑自行车，即使是在最寒冷的气候下，只要有正确的支持和基础设施投资，并不困难，这是我们在[以外看到的最好的自行车基础设施支持 DIY 解决方案，采用类似这种遥控扫雪机的东西来工作。](https://hackaday.com/2017/05/11/just-in-time-for-summer-a-remote-controlled-snowblower/)
