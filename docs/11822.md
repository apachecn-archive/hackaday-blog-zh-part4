# 可爱的小 IMac 克隆在小屏幕上运行 MacOS

> 原文:[https://hack aday . com/2021/11/02/cute-little-IMAC-clone-runs-MAC OS-on-a-tiny-screen/](https://hackaday.com/2021/11/02/cute-little-imac-clone-runs-macos-on-a-tiny-screen/)

自从苹果从 PowerPC 转向英特尔硬件以来，构建一台运行 MacOS 的非苹果电脑一直是黑客们最喜欢的消遣。尽管通常是用普通的个人电脑零件制造的，但有些人已经成功地将苹果的操作系统安装到各种基于英特尔的单板电脑上。[iketsj]用这样的板子搭建了一个可爱的小 Hackintosh，显然决定了如果要模仿苹果的硬件，不妨从他们的工业设计中获取一些线索。结果可以在视频(嵌入在下面)中看到，其中[Ike]展示了一个类似 iMac 的小型设备，配有 5 英寸的 LCD 屏幕。

这个可爱的小一体机的大脑是 Lattepanda，这是一个紧凑的主板，包含一个英特尔 CPU，几 GB 的内存和许多 I/O 接口。[Ike]用一个 256 GB 的 SSD，一个 WiFi/蓝牙适配器和前面提到的 LCD 完成了它，LCD 显示 800×480 像素，并通过主板的 HDMI 接口接收其图像。

这种外壳是一种 3D 打印的设计，有点像小型化的 iMac 一体机。背面包含几个 USB 连接器的开口，一个 3.5 毫米耳机插孔，甚至一个以太网端口，用于严肃的网络连接。一对扬声器整齐地隐藏在显示屏下方，即使没有耳机也能实现立体声。

这台电脑就像一台真正的 iMac 一样启动 MacOS Monterey，只是显示屏要小得多。[Ike]是第一个承认这不是世界上最实用的东西，但他会出去在咖啡店里用它“只是为了 lulz”。我们同意这是一个很好的理由把你的黑客带到外面。

[Ike]之前制作了一个[便携式 Hackintosh](https://hackaday.com/2021/06/30/handheld-hackintosh-runs-mac-os-on-lattepanda/) ，我们已经看到了一些令人印象深刻的 MacOS 版本，比如这个[迷你 iMac G4](https://hackaday.com/2020/11/10/mini-imac-g4-made-with-nuc-and-3d-printer/) ，垃圾桶里一个漂亮的 [Mac Pro 复制品](https://hackaday.com/2014/01/16/hackintosh-mac-pro-replica-using-a-trash-can/)，甚至还有一个内置在真实的 [Mac Pro 外壳](https://hackaday.com/2015/02/07/hackintosh-project-looks-like-a-mac-smells-like-a-mac/)中的 Hackintosh。
