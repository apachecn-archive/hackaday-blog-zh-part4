# R3-14，酝酿两年的个人助理

> 原文：<https://hackaday.com/2018/10/14/r3-14-the-personal-assistant-two-years-in-the-making/>

如今，将项目组合在一起的最大好处之一就是有多少强大的子系统可以重用。[Sanjeet]在建造 R3-14 时充分利用了一系列可重复使用的部件，这是一个[个人助理机器人](https://thereallycoolstuff.wordpress.com/2018/08/24/my-personal-assistant/)，你可以在下面的视频中看到它的运行。

许多人从制作简单的电子产品开始，比如水晶收音机或 LED 立方体。但是，如果你的项目必须从最基本的水平开始，通过拉出铜线、制造线圈、电容器、半导体器件和电池，你能走多远？即使你知道如何做所有这些事情，也要花很多时间，所以使用现成的组件并不丢人。出于同样的原因，[Sanjeet]在这个版本中使用了谷歌助手、433 MHz 射频发射机和一个树莓 Pi 作为组件。在这个过程中，他自己也贡献了一些可重用的东西，包括用于 Pi 的 LED 库和允许 T2 Siri 控制树莓 PI 的库。

这篇文章很有趣，因为它展示了设计如何从最初的金属原型发展到 3D 打印体，以及软件的几次迭代。然而，我们希望更多的软件和代码的细节，我们希望更多的细节即将到来。

同时，该项目仍然是一个很好的灵感。面部跟踪驱动伺服系统来移动机器人的头部，LED 眼睛使它富有表情。3D 打印模型本身就令人印象深刻。

我们不禁想起了 1985 年的所有家用机器人。当然，如果谷歌和 Siri 对你来说不够数学，我们建议把 [R3-14 的大脑换成 Wolfram Alpha](https://hackaday.com/2012/01/17/forget-siri-make-wolfram-alpha-your-personal-assistant/) 。
