# 导电塑料可以代替金属

> 原文:[https://hack aday . com/2022/11/06/conducting-plastic-can-replace-metal/](https://hackaday.com/2022/11/06/conducting-plastic-can-replace-metal/)

芝加哥大学宣布他们已经创造了一种材料，其行为像塑料，但导电像金属。他们还说他们还不完全明白为什么它会起作用。通常，像金属这样的良导体具有非常有序的原子结构，这是塑料往往不具备的。

这种材料以镍、碳和硫为基础。所得材料导电且稳定。然而，原子结构并不像传统导体那样有序。

研究人员称之为“导电的 Play-Doh”，他们报告说，你可以使这种材料变形，而不改变其导电能力。当然，我们想知道的是，我们是否可以从 3D 打印机中挤出它，而且——几乎同样重要的是——有没有办法让它不像 Play-Doh 那样具有可塑性？

该团队预计他们可以使用聚合物链中不同的接头和节点来改变材料属性，但我们不知道这项研究的最终结果会如何。然而，很明显，在原子水平上组织导体的方法不止一种。了解这一过程可能会产生一种新的材料，更好的超导性，等等。

虽然我们之前已经看到过[导电塑料](https://hackaday.com/2020/08/31/100-printed-flashlight-conductive-filament-and-melted-in-leads/)，但它通常是一种注入了一些导电材料的聚合物。或者，你可以[电镀](https://hackaday.com/2018/09/04/chemistry-and-lasers-turn-any-plastic-surface-into-a-pcb/)它，但这很容易理解，实际上很难做好。
