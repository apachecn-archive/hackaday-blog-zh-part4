# 3D 打印翻转钟值得一看

> 原文:[https://hack aday . com/2020/02/12/3d-printed-flip-clock-worth-a-second-look/](https://hackaday.com/2020/02/12/3d-printed-flip-clock-is-worth-a-second-look/)

翻转时钟:早在比利·穆雷在土拨鼠日反复拍打时钟之前，它们就已经很酷了，1993 年它们很酷，现在它们仍然很酷。如果你在二手市场上找不到一个，你很幸运，因为[iz2k]已经为[设计了一个广泛的蓝图，要建造一个华丽的复古外观的时钟，并加入一些现代元素](https://github.com/iz2k/flip-clock)。

有一个树莓 Pi 来获取时间、天气和 Spotify。旧的翻盖时钟总是在调频广播中调谐，所以[iz2k]使用 RTL-SDR 加密狗和软件解码器来完成这项工作。这个闹钟甚至有一个大贪睡棒，当没有闹钟主动响起时，它的功能就像一个小夜灯。三组精心印刷的翻盖由步进电机和一对红外发射器/接收器控制进行计数。

对于界面，[iz2k]保持了美观和简单。大旋钮旋转编码器控制音量增大/减小/静音，前面的小旋钮在调频收音机、Spotify 和静音之间切换。移动任一旋钮，显示屏下方的 led 就会闪烁，产生反馈。花几秒钟浏览休息时间，看看简短的演示。

如果你真的在野外找到了一个漂亮的翻转钟，也许你可以改装它。
