# 迷你真空 601 复制品得到一个定制的电动旋转开关

> 原文：<https://hackaday.com/2019/07/13/minivac-601-replica-gets-a-custom-motorized-rotary-switch/>

作为一种爱好，电子产品的乐趣之一是很容易获得零件。从字面上看，数以百万计的零件可以从数以千计的供应商和数以百计的分销商处获得，每个人都相互竞争，以尽可能容易地从 BoM 中整合订单。如果你需要，可能有人有。

但是当你需要一个已经不存在的零件的时候，你会怎么做呢？即使这个零件已经生产了很少的数量。简单——你自己创造的。这正是[迈克·盖迪]用这个独特的电动旋转开关所做的，他需要完成他的 20 世纪 60 年代电脑教练的复制品。我们报道了[建造的 Minivac 601](https://hackaday.com/2019/06/17/a-faithful-replica-of-an-early-computer-trainer/) ，这是一个来自早期计算机时代的训练器，让实验者学习基本数字逻辑的诀窍。它主要使用通过跳线连接的继电器、灯和开关，但它有一个关键组件——一个用于输入的旋转控制，并在电机的帮助下，作为输出指示器。

[Mike]的 switch 版本尽可能忠实于原作，至少在外观上是如此。这些零件大多是 3D 打印的，16 个簧片开关嵌入在墙壁中，磁铁放置在转子中。操作转子的马达是安装在铰接支架上的简单齿轮马达；当转子需要移动时，螺线管将马达的摩擦驱动轮拉向转子。

独特的控制插槽正好插入 Minivac 副本，真正完善了外观和感觉。向[Mike]脱帽致敬，感谢他令人愉快地复制了遗失的计算机历史，并致力于完成这项工作。
