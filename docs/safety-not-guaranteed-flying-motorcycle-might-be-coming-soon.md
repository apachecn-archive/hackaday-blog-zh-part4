# 安全没有保证:飞行摩托车可能很快就会出现

> 原文：<https://hackaday.com/2022/06/12/safety-not-guaranteed-flying-motorcycle-might-be-coming-soon/>

据未来主义网站的维克多·唐格曼报道，喷气背包航空公司正在展示其 P2 飞行摩托的原型，它看起来很棒，而且用拉尔夫·纳德的话说，在任何速度下都不安全。该原型可以提升 1000 磅，以每小时 500 英里的速度行驶，行程可达 400 英里。当然，我们假设这些事情不是同时发生的。

正如你所料，这个东西还没有获得联邦航空局的批准，我们想知道它是否会被批准。该公司计划今年晚些时候进行遥控飞行，甚至更晚些时候进行真正的有人驾驶飞行。你可以从与[喷气背包](https://jetpackaviation.com)相关的 [Mayman Aerospace](https://maymanaerospace.com) 中看到更多(当然，它制造喷气背包)。

这种设计我们已经看到好几年了(见下面的视频)。我们也看到了其他的[飞行汽车设计](https://hackaday.com/2016/01/05/henry-smolinski-and-the-flying-pinto/)和链接，它们似乎永远不会超出艺术家的渲染阶段。P2 会有所不同吗？谁能说呢？如果你年龄够大，可以回忆一下，曾经有一段时间，每年都被吹捧为家庭录像年的到来，但它从未实现。直到 VHS 和 Beta 磁带盒的出现。

所以所有这些飞行概念可能会也可能不会——嗯——起飞，但最终，它们中的一个会。毕竟，喷气背包看起来像科幻小说，它们仍然非常罕见和有限，但喷气背包[确实设法让一些东西工作](https://hackaday.com/2015/11/09/finally-a-working-jetpack/)。
