# 力反馈鼠标真的震撼了一切

> 原文:[https://hack aday . com/2020/05/17/force-feedback-mouse-really-shakes-things-up/](https://hackaday.com/2020/05/17/force-feedback-mouse-really-shakes-things-up/)

对于那些喜欢花时间探索虚拟世界的人来说，这是一个非常令人兴奋的时刻。一些大预算作品中的图形很容易接近真实感，身临其境的多声道声音真的可以让你相信你已经被传送到另一个地方或时间。随着又一代或两代 GPU 开发和 VR 硬件的出现，游戏和现实之间的界限肯定会变得非常模糊。

也就是说，我们离企业号上的全息甲板还很远。一台高端电脑和最新的虚拟现实可能会愚弄你的眼睛和耳朵，但这仍然会让你的其他感官失去乐趣。这就是为什么[【贾丁·帕特尔】利用一系列螺线管开发了这种聪明的力反馈鼠标](https://teenenggr.com/2020/05/09/when-sound-is-not-enough-to-feel-the-game/)。

这个想法非常简单:电脑上的 Python 程序监听鼠标点击事件，并在玩家扣动虚拟扳机时告诉连接的 Arduino 启动螺线管。这自然不是一个完美的系统，因为点击游戏菜单似乎也会让你的“枪”开火。但正如你在休息后的视频中看到的，当它工作时，它工作得非常好。移动的螺线管不只是振动鼠标，金属的噼啪声实际上加重了游戏中的枪声效果。

有了这种触觉反馈和让我们保持运动的全向跑步机，我们将非常接近于欺骗我们的感官，以为我们实际上在别的地方。坦白说，现在听起来很吸引人。
