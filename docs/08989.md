# 用于光学逻辑门的干涉图案

> 原文:[https://hack aday . com/2021/01/16/interference-patterns-harined-for-optical-logic-gates/](https://hackaday.com/2021/01/16/interference-patterns-harnessed-for-optical-logic-gates/)

数字逻辑的基础很容易掌握，弄清楚 1 和 0 如何通过各种门通常是一项有趣的练习。将事物降低一个层次，将“与”、“或”、“或非”门分解为底层模拟电路会增加一些复杂性，但电子的流动仍然是很容易理解的。然而，用光子代替所有这些，你将进入一个真正奇怪的世界。

至少这是我们对[耶鲁安·弗勒格尔]的最新项目的看法，这个项目是用纯光学元件制造逻辑门。正如他自己在下面的视频中承认的，这并不是完全未探索的领域，但他的方法，使用建设性和破坏性的干涉，似乎以前没有使用过。基本的“电路”由一个发生器、一对蚀刻在石英板上的衍射图案和一个鉴别器组成，鉴别器基本上是另一块板上的一个针孔，其位置与发生器图案的公共焦点重合。当两个发生器受到同相单色光照射时，就形成了一个或门。使两个输入的相位相差 180，会产生一个 XOR 门，因为两个输入之间的相消干涉会阻止任何光从鉴别器中射出。

使用这种技术的变体，所有七个基本门都是可能的，但是，当然，实际建造这样的东西会带来挑战。微观制造技术，包括薄金属涂层上的光刻和蚀刻图案，相当重要。虽然[耶鲁安]能够建造一些基本的门并进行测试，但还有很长的路要走。他说他需要增加输入光束控制——也许是液晶快门——以及输出检测。

这项技术的潜在应用相当令人兴奋，我们将继续关注细节。对于光的双重性和干涉图案如何形成的更深入的背景，查看[【耶鲁安】对双缝实验](https://hackaday.com/2020/10/18/duality-of-light-explored-by-revisiting-the-double-slit-experiment/)的解释。
