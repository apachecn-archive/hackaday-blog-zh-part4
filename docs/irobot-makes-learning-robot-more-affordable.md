# IRobot 让学习机器人更实惠

> 原文：<https://hackaday.com/2020/08/06/irobot-makes-learning-robot-more-affordable/>

当你想到 iRobot 时，你可能会想到地板清洁或军用机器人。但是他们也有一套针对教育的机器人。Root 机器人是该公司在 2019 年收购的，最初的目标是教室，每台价格约为 200 美元。一个新版本的价格约为 130 美元，更适合家庭用户。

原始版本——Root rt1——仍然可用，但 rt0 版本缺少几个功能，无法达到理想的价格。少了什么？显然，rt1 可以用磁铁粘在白板上，但 rt0 没有这个功能。也没有“悬崖”传感器或彩色扫描仪。

好的一面是，有一个新的 20 美元的选项，允许任何版本的机器人安装乐高作品。通常，机器人可以在下面画一个标记，并根据命令收回它。rt1 也可以擦除标记。不过，使用乐高底座，你可以使用相同的机制来驱动视频中的弹射器。

机器人可以使用看起来像 Scratch 的东西或基于文本的编码进行编码。编辑器——你甚至可以免费在线模拟它——有三个级别，取决于学生的阅读和编码能力。此外，还有许多免费的[练习和活动](https://edu.irobot.com/learning-library)。

售价 130 美元，这些看起来像是很容易被黑客攻击的机器人平台。当然，你想让你的孩子也这样，我们不会说不。

教育机器人并不是一个新想法。如果你想继续用 iRobot 做大联盟的东西，你也可以给[买一个不烂的 Roomba](https://hackaday.com/2014/12/21/irobot-releases-hackable-roomba-without-the-vacuum/)。
