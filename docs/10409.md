# 将虚拟男孩变成手持控制台

> 原文:[https://hack aday . com/2021/06/10/将虚拟男孩变成掌上游戏机/](https://hackaday.com/2021/06/10/turning-the-virtual-boy-into-a-handheld-console/)

虚拟男孩是任天堂最臭名昭著的失败，受到几个问题的困扰。最突出的问题是红色单色立体显示技术，这让许多用户在玩了很短时间后就感到头疼，但其高昂的价格和极其有限的游戏库也让许多潜在的买家望而却步。还有便携性的问题:与以其命名的 Game Boy 不同，Virtual Boy 勉强称得上是便携式系统，因为它需要放在桌子上才能使用。

但是现在，由于[Shank]的不懈努力，至少有一些问题已经解决了。他建造了世界上第一个真正的*便携式虚拟男孩，它用现代 IPS 液晶面板取代了系统中麻烦的 3D 显示器。这款定制掌机旨在将虚拟男孩的独特审美与 Game Boy Advance 的标志性风格融合在一起，看起来像是来自某个替代时间线，任天堂决定生产一个更便宜、更简洁的系统版本，而不是放弃它。*

 *[![](../Images/5b96ec3bee76cad2362f782dc1d9d9d3.png)](https://hackaday.com/wp-content/uploads/2021/06/realboy_detail2.jpg) 虽然【Shank】投入到项目中的工作无疑令人印象深刻，但应该说是几个天才黑客的努力才创造出了这个掌上虚拟男孩。首先使修改成为可能的关键组件是[furtek]的 [VirtualTap，它不仅提供驱动 LCD 面板的 VGA 输出，还欺骗系统主板，使其相信伺服驱动的立体显示器仍然连接并处于活动状态。](https://github.com/furrtek/VirtualTap)

它还使用了开源电源管理板，这是 [[GMan]最初为他自己的便携式 N64](https://hackaday.com/2020/03/27/trimmed-pcb-makes-the-ultimate-portable-n64/) 开发的，【Bassline】参与进来，用半透明树脂铸造定制按钮和 D-pad，【Mitch 3D】投入了无数个小时来打印和重印系统的彩色外壳，直到它出来为止。

最终系统的所有小细节，也就是[Shank]所说的真正的男孩，使这个项目自成一派。特殊的按钮组合允许用户改变显示器的颜色，如果你厌倦了臭名昭著的红色。按钮后面还有 RGB LEDs，与显示器本身的配色方案相对应，为玩家增加了额外的分数。他甚至确保包括系统的原始链接端口，尽管没有官方发布的游戏使用过它。

我们第一次接触[Shank]是在他演示一个装在薄荷罐里的便携式 Wii 时。这是一次非常令人遗憾的游戏体验，但这个项目展示了他将项目进行到底的奉献精神。[看着他的技术在过去几年里不断提高，这一直鼓舞着](https://hackaday.com/2019/01/03/a-practical-portable-wii-emerges-from-the-memes/)，我们迫不及待地想看看他接下来会有什么。
