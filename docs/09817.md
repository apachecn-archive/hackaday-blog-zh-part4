# 玩面试游戏

> 原文:[https://hackaday.com/2021/04/12/playing-the-interview-game/](https://hackaday.com/2021/04/12/playing-the-interview-game/)

技术面试通常令人畏惧，就像其他面试一样。然而，技术面试包括许多非技术人员可能会感到困惑甚至毫无意义的元素，例如白板问题解决、带回家的作业、设计会议，甚至只是简单的脑筋急转弯。[Erik McClure]走了一点寻常的路，[开始使用 factory builder 游戏工厂作为技术面试](https://erikmcclure.com/blog/factorio-is-best-interview-we-have/)。

许多人指出白板编码练习中固有的问题和棘手问题背后的意图，并断言重点不是完成问题，而是揭示候选人如何思考和解决问题。当你作为一个团队慢慢地将一条简陋的生产线扩大到一个大型工厂时，Factorio 可以解决所有问题，这使它成为评估这些技能的一个很好的候选人。我们怀疑写这个游戏的优秀开发者曾经想象过它被用来做采访。

十有八九，你可能不会很快就有工厂面试，因为[Erik]估计每次面试需要八到二十个小时。但是我们喜欢将面试从一系列需要解决的乏味问题重新想象成一个不断发展的合作游戏。当然，你也可以阅读更多关于[获得一份工作所必需的经验，以及公司在面试中寻找什么](https://hackaday.com/2017/08/02/why-i-got-the-job/)。
