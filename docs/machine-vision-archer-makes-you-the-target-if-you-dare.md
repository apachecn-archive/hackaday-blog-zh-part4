# 机器视觉射手让你成为目标，如果你敢

> 原文：<https://hackaday.com/2021/06/30/machine-vision-archer-makes-you-the-target-if-you-dare/>

我们一开始就声明，让机器人射手从你的头上射下苹果是一个非常非常糟糕的主意。你绝对不应该重复你将在下面的视频中看到的，如果你这样做，结果都在你身上。

也就是说，[卡迈勒·卡特]的身材相当酷。他明智地选择使用你能找到的最弱的弓，那种带弦的弓基本上是又大又软的松紧带，可以发射带吸盘尖端的箭，无害到让孩子们玩，你只要知道他们会在你转身的那一刻互相射击，不管你告诉他们什么。目标采集是英特尔实感深度摄像头的工作，它用于发现目标并计算到目标的距离。铝挤压框架持有弓，并调整其海拔，而长丝杠和伺服牵引和释放字符串。

整理好跑步装备后，[Kamal]开始在高中物理中进行计算，例如确定箭的初速的弓的弹簧常数，以及确定击中目标所需角度的弹道学公式。他做到了——大部分时候。我们真的很惊讶他有多少命中目标的投篮。是的，他最终真的让它变了一个(威廉·泰尔的)苹果戏法——尽管我们不禁从他，咳咳，手的姿势中注意到，他对箭会在哪里结束并不完全充满自信。

[Kamal]说他从[[Mark Rober]的飞镖靶](https://hackaday.com/2017/03/22/dartboard-watches-your-throw-catches-perfect-bullseyes/)和[[Shane Wighton]的自扣篮篮框](https://hackaday.com/2020/05/19/robotic-basketball-hoop-v2/)中获得了灵感。我们会说，他的成绩让他在技能选修运动团体中享有良好的声誉。
