# 受准将启发的手表把基本的放在你的手腕上

> 原文:[https://hacka day . com/2021/06/02/commodore-inspired-watch-puts-basic-on-your-腕子/](https://hackaday.com/2021/06/02/commodore-inspired-watch-puts-basic-on-your-wrist/)

问一个智能手表的主人，他们最喜欢的腕戴功能是什么，他们可能会说，他们所有的日常信息一目了然，或者他们能够与朋友和家人沟通的便利性。如果他们*没有*提到在他们的可穿戴 BASIC 解释器中敲几行，那么你知道你不是在和【尼克·比尔德】说话。[他为 LILYGO T-Watch 2020](https://github.com/nickbild/c64_watch) 设计的“C64 Watch”固件不仅从 Commodore 64 汲取了一些视觉灵感，还让你在一个功能基本的环境中重温那些早期的计算辉煌时代。

最初[尼克]使用一个非常小的屏幕键盘来敲击他的基本程序，但是发现这种体验就像折磨一样不舒服，他转而使用 USB。只需将手表插入电脑，打开您最喜欢的串行终端，您就可以访问手表上运行的 TinyBasic Plus 定制版本。为了让事情变得更简单，他正在考虑通过 WiFi 实现一个基于网络的终端，这样你就不需要把手表插上电源。

当你不运行 BASIC 时，你会看到一个 Commodore 主题的表盘，并有经典的`READY.`提示。右上角隐藏着一个小电池指示器，点击彩虹色的“C”将启动菜单。这是非常简单的，但是当然，根据原始资料，你还能期待什么呢？

展望未来，[Nick]说他还想在固件中实现一个 C64 模拟器，这样手表就可以运行原始软件。我们有点怀疑这实际上有多实际，但我们会保留判断，直到我们看到它的运作。他还希望其他 Commodore 爱好者也能加入进来，为这款手表带来自己的改进和新功能。

你可能会认为手腕上的 Commodore 64 模拟器是运行你的旧游戏和软件的最古怪的方式，但我们会说在 20 世纪 80 年代的虚拟现实微观世界中玩 *Turrican* 的[是最棒的](https://hackaday.com/2021/05/03/commodore-64-emulator-in-vr-delivers-a-full-80s-experience/)。
