# 我的专业是游戏…

> 原文：<https://hackaday.com/2021/12/06/my-major-is-gaming/>

时代变了。你现在可以上一堂写游戏的大学课。事实上，你现在可以上一堂关于编写游戏的大学课，因为纪念大学的[戴夫·丘吉尔]已经把他的全部 22 堂课放在网上供你欣赏。[ [邱吉尔博士](http://www.cs.mun.ca/~dchurchill/) ]不打算发布作业文件，但你仍然可以从观看视频中获得很多。显然，这些课程也在 Twitch 上直播。

这些游戏建立在 SFML 的基础上，所以最终的游戏可以携带。该库抽象了输入、图形、声音和网络。

游戏使用实体组件系统(ECS ),这是一种可能的架构。当然，像任何一种软件一样，有许多不同的方法来完成这项工作。游戏似乎是 2D，所以如果你打算写下一个厄运，你需要做一些课外工作。

像大多数课程一样，第一堂课(见下文)从一些日常事务开始，但很快就掌握了。如果你想充分利用它，你可能已经知道 C++了。

该课程侧重于桌面电脑游戏，但这些想法甚至适用于为游戏机编写程序。如果你对电子游戏不感兴趣，你可以[在其他计算机科学话题上破解你自己的非学位。](https://hackaday.com/2018/01/04/hack-your-own-computer-science-degree/)
