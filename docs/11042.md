# 建造一套射箭机甲来逃避练习

> 原文：<https://hackaday.com/2021/08/19/building-an-archery-mech-suit-to-skip-practice/>

据传说，爱德华三世国王曾说过:“如果你想训练一名长弓手，就从他的祖父开始吧。”持续地用任何弓准确命中，尤其是对移动的目标，需要很多小时的练习。或者，如果你是谢恩·威顿，你可以花相当多的时间来建造、调试和重建一个机器人增强的弓。

目标是从空中射击飞行目标，所以[Shane]必须创建一个系统，可以跟踪弓和目标的位置，并自动调整弓的位置，并在正确的时刻释放箭来拦截目标。位置跟踪是用他的机器人篮球架上使用的相同的 Optitrack 相机完成的，在弓、目标和释放机制上有反射标记球。自动瞄准是由一对步进电机驱动的双轴齿轮齿条机构完成的。[谢恩]首先使用了他能在网上找到的最便宜的反曲弓，这导致了准确性问题，可能与[弓箭手的悖论](https://en.wikipedia.org/wiki/Archer's_paradox)有关。这种设置也让他反复打自己的脸，因为伺服操作的释放机制会在他的手没有合适的锚的情况下意外释放。

[谢恩]最终升级为复合弓，这减少了他在排队投篮时必须保持的紧张，但也大大增加了系统的重量。这让他完全接受了机甲套装的外观，并使用摄影机稳定器背心来承受弓的重量。这终于使他能够可靠地分辨出威廉的射击并击中飞行目标。

无论是[多功能电子高尔夫球杆](https://hackaday.com/2020/06/11/a-robotic-golf-club-to-possibly-boost-your-game/)、[爆炸棒球棒](https://hackaday.com/2020/10/27/going-for-the-home-run-record-with-explosive-help/)还是[机器人台球杆](https://hackaday.com/2021/02/24/robotic-pool-cue-can-be-your-friend-or-your-foe/)，【谢恩】当然擅长使用令人印象深刻的工程技能来弥补他身体技能的不足，或者只是他故意闭上的眼睛。
