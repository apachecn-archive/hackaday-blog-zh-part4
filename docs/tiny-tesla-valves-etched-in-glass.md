# 蚀刻在玻璃上的微小特斯拉阀

> 原文：<https://hackaday.com/2021/06/20/tiny-tesla-valves-etched-in-glass/>

虽然现在流行以特斯拉命名新奇的新技术，但实际的发明者有很多自己的发明，后来都以他的名字命名，包括特斯拉线圈，特斯拉振荡器，特斯拉涡轮机，甚至臭名昭著的特斯拉塔。他的一个鲜为人知的发明是特斯拉阀，这是一种允许单向流动而没有任何移动部件的止回阀，[惠更斯光学]向我们展示了一种将这些阀的微小版本蚀刻到玻璃上的方法。

构建开始时有一个相当长的警告，这是使用氢氟酸时的标准做法。实际执行蚀刻需要酸，但由于特斯拉阀的尺寸较小，这比典型的蚀刻复杂得多。他从混合缓冲氧化物蚀刻开始，缓冲氧化物蚀刻是氢氟酸、氨和盐酸的混合物，其蚀刻比任何单一酸都更均匀。与蚀刻 PCB 类似，需要一个保护掩模来确保蚀刻只发生在需要的地方。为此，有几种选择，每种都有自己的优点和缺点，但最终[惠更斯光学]得到了有史以来最小的特斯拉阀之一。

事实上，瓣膜非常小，只有借助显微镜才能看到。在显微镜下观察它们时，他能够用一小滴水进行测试，以确认它们确实如预期的那样工作。而且，虽然他在这个建筑中创造的阀门是为液体设计的，[惠更斯光学]指出，将它们做得这么小的原因是为了制造微小的光学元件，这些元件以而闻名。
