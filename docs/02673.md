# 几乎完全 3D 打印的遥控汽车是四轮驱动的乐趣

> 原文:[https://hack aday . com/2019/04/09/near-totally-3d-printed-RC-car-is-4wd-fun/](https://hackaday.com/2019/04/09/nearly-entirely-3d-printed-rc-car-is-4wd-fun/)

遥控车可以很有趣，特别是如果你有一个备用的停车场或土堆可以让它们四处游荡。任何一家好的业余爱好商店都会有各种各样车辆的货架——童车、旅行车、原型车——但是你不必满足于已经有的东西。为什么不用 3D 打印出你梦想中的汽车呢？ (YouTube，下面嵌入。)

这个版本来自于[工程废话]，现在已经是第三版了。该设计是在解放军，使它尽可能方便世界各地的打印机所有者。几乎整辆车都是 3D 打印的——不仅仅是底盘。变速箱、差速器和传动轴，甚至悬架臂和拉杆都是打印的，而不是购买的。这也意味着汽车更容易制造，一切都按照正确的尺寸打印，而不是使用现成的可调节部件。

性能令人印象深刻，由于其四轮驱动传动系统和双叉骨悬架，该车表现出良好的抓地力。在 Thingiverse 上可以找到文件，所以没有什么可以阻止你把它打印出来，这个周末去兜风。[我们也很想看到它用 3D 打印的轮胎下水。](https://hackaday.com/2017/10/14/3d-printed-tyres-let-you-drive-on-water/)
