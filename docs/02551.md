# 紧急邻里通信由助手提供

> 原文:[https://hack aday . com/2019/03/29/emergency-neighborhood-communication s-culture-of-helper/](https://hackaday.com/2019/03/29/emergency-neighbourhood-communications-courtesy-of-helper/)

对许多人来说，电话和互联网连接无处不在，随时可用。在中国的地铁上上传自拍照是可能的，在大多数公路城镇搜索餐馆评论也是可能的，这都要归功于现代手机连接。然而，在紧急情况下，我们并不总是那么幸运。如果发射塔出现故障，或者用户需求增长过大，一切都会很快崩溃。在这些情况下，HELPER 的目标是蓬勃发展。

HELPER 代表异构高效低功率无线电。这是一个无线电系统，设计用于在没有任何基础设施的情况下运行，创建一个弹出式网络来满足灾区的社区需求。用户可以分享关于水、汽油和食物等可用资源的信息，而紧急救援人员可以协调他们的反应，并向需要的人提供直接援助。

这是一个围绕常见零件构建的系统。Raspberry Pis 运行后端软件，通过 WiFi 与个人通信，LoRa 无线电处理节点到节点的远程通信。将这种通信能力与 GPS 定位和存储的地图数据相结合，用户可以在出现问题时更容易地找到资源和帮助。对于那些希望了解该项目的人来说,《华尔街日报》的文章是免费的。

这是一个旨在当传统网络瘫痪时保护人们安全的项目。关键是要记住，一旦灾难发生，开始分发无线电硬件通常为时已晚——在事情开始恶化之前，应急设备应该准备好。当然，等式的另一边还有政府——在美国,[紧急广播系统是正确处理紧急通信的一个很好的例子。](https://hackaday.com/2017/11/16/radio-apocalypse-the-emergency-broadcast-system/)休息后的视频。
