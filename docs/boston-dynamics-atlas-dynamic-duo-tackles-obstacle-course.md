# 波士顿动力公司阿特拉斯动力双人铲球障碍赛

> 原文：<https://hackaday.com/2021/08/17/boston-dynamics-atlas-dynamic-duo-tackles-obstacle-course/>

历史上，现实世界的人形机器人的能力远远落后于电视和电影中的机器人。但是机器人专家一直在推动技术发展，波士顿动力公司刚刚分享了一份进展报告:[他们的研究平台 Atlas 现在可以完成双机器人跑酷动作](https://blog.bostondynamics.com/atlas-leaps-bounds-and-backflips)。

在 YouTube 上观看[时长一分钟的节目](https://www.youtube.com/watch?v=tF4DML7FIWk)(插播在休息时间之后)显示的动作比[随着歌曲*跳舞要求更高。*](https://hackaday.com/2020/12/30/boston-dynamics-dancing-bots-beg-for-your-love-a-la-napoleon-dynamite/)根据 Boston Dynamics 的说法，这一新功能实际上比它看起来更令人印象深刻。与早期的示范不同，这一套动作使用了更少的[预编动作，而这些动作构成了早期的舞蹈表演](https://hackaday.com/2021/01/19/start-your-new-career-in-robot-dance-choreography/)。Atlas 现在更多地利用其机载传感器来感知环境，更多地利用其机载计算能力来根据具体情况决定如何最好地在世界上移动。它还需要以连续的顺序将单个的动作串在一起，这是它以前很难做到的。

从机器人演示视频中很难看出这样的进步，这些视频经常被编辑和策划，以显示突出的成功，并跳过所有(许多许多)失败。当然波士顿动力公司以前也这样做过，但这次它配有大约六分钟的幕后镜头。(也是休息之后。)我们看到机器人在学习时跌跌撞撞，人类努力让它们重新站立起来。

人形机器人的发展并不总是一帆风顺(有时很有趣)，但 Atlas 已经超越了它的前辈，如 T2 本田阿西莫 T3。这样的研究找到了去[看起来不那么像人的机器人的路，比如 Stretch](https://hackaday.com/2021/03/31/boston-dynamics-stretch-robot-trades-lab-coat-for-work-uniform/) 。谁知道呢，也许有一天真正的机器人会像它们在电视和电影中的对应物一样，长期以来一直由人类穿着戏服扮演。
