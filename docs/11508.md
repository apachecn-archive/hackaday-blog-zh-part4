编程语言是我们极客们有一些非常强烈并且经常是非常对立的观点的话题之一。随着计算领域新概念的出现，如果可行的话，旧语言可能会增加新的功能，或者当新的暴发户出现并带来一些变化时，旧语言可能会落后。这位抄写员可能还记得他早期的嵌入式系统编程，以及当有人提出一个需要嵌入式 C++或更糟的 Java 的项目时随之而来的争论，当时我们大多是顽固的 C 程序员。快进一二十年，事情会变得更加复杂。如此多的选择，如此多的观点。

因此，能看到一些真正独特且制作精美的 Python 教程视频真的很好，看起来很吸引人也很有趣。由加拿大女演员[[]Ulka Simone Mohan ty](https://www.imdb.com/name/nm2769464/)(有些人可能会从游戏《魔法:聚会竞技场》和各种电影和电视节目中认出她)担任主角，她对 Python 最重要的领域进行了一次死一般的化身般的演示。我们对注释“Loopus Interruptus”特别感兴趣，因为它是一个迭代到列表末尾的异常条件。

如果你一直在推迟真正了解 Python 并获得一些编程经验的努力，我们认为这将是一个非常好的起点。即使你已经使用 Python，你也可能从这些视频中获得一些好的见解。我们不知道你将如何度过接下来的几个小时，但我们将学习更多的 Python。
