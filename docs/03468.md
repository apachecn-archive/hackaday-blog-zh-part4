# 梦幻 VMU 中的阿杜博伊

> 原文:[https://hackaday.com/2019/07/15/aruboy-in-a-dreamcast-vmu/](https://hackaday.com/2019/07/15/aruboy-in-a-dreamcast-vmu/)

Arduboy 是一个微型的、信用卡大小的视频游戏控制台，你可以自己制作。Dreamcast VMU 也是一个小巧的便携式视频游戏系统，但实际上那只是我们保存疯狂出租车的地方。两者结合，你得到了什么？[sjm]就是这么做的，[给了我们一个藏在梦幻 VMU 中的阿杜博伊。](https://hackaday.io/project/163662-arduboy-in-a-dreamcast-vmu)

Arduboy 的核心只是一个 ATMega32u4，这是许多 Arduinos、I2C 有机发光二极管和其他各种 USB、电源和电池保护电子设备中使用的相同芯片。简而言之，这是一条简单的线路，任何有技能的人都可以自己建造。由于任何人都可以制造 PCB，而 Dreamcast 已经在外壳中内置了漂亮的硅胶按钮，所以[sjm]很容易就可以用 Arduboy 的所有内脏制作出 Dreamcast VMU 形状的 PCB。唯一真正的区别是有机发光二极管的尺寸——这款使用了 0.96 英寸 128×64 的有机发光二极管，而原版使用的分辨率相同，但尺寸要大得多。

是的，[我们在](https://hackaday.com/2019/02/04/arduboy-brings-new-life-to-dreamcast-vmu/)之前见过这个项目，但是现在由于 Hackaday 奖的魔力，它现在正在参加这个星球上最大的硬件竞赛。休息之后，您可以查看整个构建视频和一个简短的演示。当然，这不是 Arduboy 电路的第一次重新利用，我们已经看到[一个柔性电路版本](https://hackaday.com/2019/04/18/arduboy-goes-thin-and-flexible-for-portable-gaming/)，和[一个像青少年工程和恐慌开发的 Playdate 一样的曲柄版本](https://wp.me/paBn4l-1w2P)。
