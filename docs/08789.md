# 认为你的笔记本电脑贫血？试试 MSDOS 的

> 原文：<https://hackaday.com/2020/12/28/think-your-laptop-is-anemic-try-an-msdos-one/>

如果有人在这个假期送给你一台便宜的笔记本电脑，2GB 的内存和 400 MHz 的 CPU 可能会让你有点不高兴。然而，一旦你看到[Noel 的复古实验室] 4.8 公斤的 Amstrad [PPC512](https://www.youtube.com/watch?v=8IcvmFsTsBE) 你可能会更加欣赏它，他在下面的视频中展示了它的里里外外。

与现代的笔记本电脑不同，这款老式但不错的笔记本电脑有一个完整的键盘，可以从主体中抽出来。键盘下方的空间包含了 LCD 屏幕，Noel 将不得不用另一台设备的 LCD 来替换它，那台设备的外形更差，但屏幕很好看。在这个视频中，他尽可能地将视频输出到外部显示器，但两个 LCD 都没有显示任何生命的迹象。但是他计划很快制作更多的视频。

MS-DOS 3.3 电脑的 LCD 可以模仿 CGA 或 MDA 屏幕，但没有背光。8 MHz NEC V30 处理器有 512K 内存，因此有部件号。还有一个类似的型号，有 640K 的内存和(gasp) 2400 波特的调制解调器。

以今天的标准来看，这款笔记本电脑的电源选项有点奇怪。计算机可以使用交流适配器或汽车适配器。它也可以使用 10 节 C 型电池。还有匹配的外部显示器，能够为机器供电。

我们以前在这类机器上看到过 [LCD 移植](https://hackaday.com/2019/10/30/amstrad-portable-gets-a-modern-lcd-transplant/)，尽管那台从单色变成了彩色。这些看起来不太便携，但是和早期的“[便携式](https://hackaday.com/2018/04/08/orbital-mechanics-on-a-vintage-kaypro/)电脑相比，它们很棒。
