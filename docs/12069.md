# 玩江湖 IRL

> 原文:[https://hackaday.com/2021/12/02/play-runescape-irl/](https://hackaday.com/2021/12/02/play-runescape-irl/)

Runescape 已经推出将近 21 年了，虽然这对于一个拥有活跃用户群的游戏来说是相当长的时间，但对于人们来说以各种丰富多彩的方式修改游戏也是很长的时间。对于像《堡垒 2》这样的老游戏来说，这意味着启动一个机器人来破坏服务器，但是对于《江湖》来说，这种黑客行为更加轻松有趣。[比如这把让【BigFancyBen】在现实生活中玩江湖的斧子](https://www.youtube.com/watch?v=gCofVhR5HUQ)。

这更像是一个增强现实黑客，将他的普通人机界面设备从简单的键盘和鼠标升级到包括这把斧头。在现实生活中操纵斧头时，游戏内的斧头可以同时使用。这个有很多层，但本质上是一个开关 joycon 连接到 axe 来感应运动，它通过 Python 脚本将 axe 摆动的信息传递给 API。然后，游戏中的一个机器人砍下虚拟树，然后报告给 API，API 再报告给[BigFancyBen]的视图屏幕，该屏幕还会在 Twitch 上播放。

虽然这开始是因为对游戏坚持研磨以达到某些目标感到沮丧，但似乎有一些有趣的方式来操纵游戏机制以获得更大的好处。[BigFancyBen]最初说他宁愿去健身房也不愿“点击更多的屋顶”,这是一个完整的 IRLScape 世界的开始。不要忘记，同样有可能在相反的方向建立这种类型，并从视频游戏中控制现实世界的事情。
