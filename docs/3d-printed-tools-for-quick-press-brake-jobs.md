# 用于快速压弯机作业的 3D 打印工具

> 原文：<https://hackaday.com/2019/10/08/3d-printed-tools-for-quick-press-brake-jobs/>

在钣金加工时，折弯机是车间的主要设备。它们非常适合反复执行精确且可重复的弯音。通常，它们配有钢制模具，可承受数千次冲压循环。然而，这种工具昂贵且生产耗时。[Anthony]最近有一份工作需要一个独特的内半径。他没有冲出去买模具，而是决定自己 3D 打印！

使用常规 PLA 细丝在标准 Prusa i3 上印刷压弯机工具。在这个过程中没有什么特别的，打印使用了 12 个周长和 20%的填充。尽管是由塑料制成的，这些工具却出奇地耐用。在测试中，零件能够弯曲高达 3.4 毫米的钢，经历几个周期而没有明显的磨损。[Anthony]还试验了鹅颈部件，虽然不太坚固，但可以轻松适应更复杂的钣金部件。

3D 打印是生产定制冲压模具的一种很好的方式，比生产传统的钢制模具更便宜、更快捷。虽然它不太可能对长时间的生产运行有用，但对于需要定制几何图形的短时间运行，它是一种方便的技术。[我们甚至还见过 3D 打印的冲头和模具组](https://hackaday.com/2018/02/16/3d-printed-punch-and-die-stand-up-to-steel/)。休息后的视频。
