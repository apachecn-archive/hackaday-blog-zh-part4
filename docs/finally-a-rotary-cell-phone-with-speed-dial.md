# 最后，一个带快速拨号的旋转式手机

> 原文：<https://hackaday.com/2019/09/24/finally-a-rotary-cell-phone-with-speed-dial/>

如果你正在读这篇文章，很有可能你是家庭 IT 部门。我们尽我们所能帮助他们，但无法改变的事实是，眼睛和手老化后，很难操作智能手机。当[西德伯恩的]父亲开始抱怨时，他采取了不同的方式。他没有帮助父亲适应，而是把现代手机的内脏塞进了一部 20 世纪 70 年代的旋转式手机里——如果你只想用它打电话，为什么不去找一部久经考验的手机呢？

[sideburn]首先想出了最重要的部分，就是让这个东西响起来。这些旧手机的铃声是由一个巨大的继电器驱动的，需要大量的电压，所以他将一个 3.2V 的充电电压提高到 34V。然后，只需让 GSM 模块与微控制器配合，并对 MOSFET 进行编程，以触发升压模块，使野兽发出叮当声。

旋转式电话最糟糕的一点是，它们从来就不是用来匆忙拨号的。但是[西德伯恩]已经处理好了。一旦 Rotocell 开始工作，他增加了一个短信界面，使手机更加有用。爸爸可以在 Rotocell 上添加联系人，通过现代手机向它发送姓名和号码。一旦在那里，他可以按名字拨号，加快了一点点的过程。

短信界面还可以报告信号强度和电池电量，并在低于 20%时发送电池电量低警报。休息之后，您可以看到 Rotocell 的运行。

有一两个旧的转盘吗？如果打电话的内部现代化没有点亮你的电路，[试着把它变成一个语音控制的助手。](https://hackaday.com/2018/11/17/vintage-rotary-phone-turned-virtual-assistant/)
