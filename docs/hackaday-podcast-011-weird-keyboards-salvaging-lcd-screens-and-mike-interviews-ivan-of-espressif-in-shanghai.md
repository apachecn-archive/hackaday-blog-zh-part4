# 黑客日播客 011:怪异的键盘，回收的液晶显示屏，迈克在上海采访 Espressif 的伊万

> 原文：<https://hackaday.com/2019/03/22/hackaday-podcast-011-weird-keyboards-salvaging-lcd-screens-and-mike-interviews-ivan-of-espressif-in-shanghai/>

随着我们无畏的主编迈克·斯奇斯在中国成为一个大事件，执行主编埃利奥特·威廉姆斯和特约撰稿人汤姆·纳尔迪一起谈论过去一周适合印刷的所有黑客。加入我们，讨论定制机械键盘的广阔世界，复兴一项可悲的过时显示技术，用实验室里的材料制作自己的 RC 发射器，以及 Pepto Bismol 和火箭燃料之间意想不到的相似之处。我们还将看看构建一个健壮的嵌入式系统需要什么，看看我们是否能找到一种方法来绘制值得一看的原理图。此外，在这一集结束之前，请耐心等待，听听 Mike 采访在让 ESP8266 与 Arduino 配合良好以及现在为 ESP32 运行固件方面发挥重要作用的人。
