# 福禄克 8840A 的自制交流升级

> 原文:[https://hackaday . com/2022/10/20/a-home brew-AC-upgrade-for-the-fluke-8840 a/](https://hackaday.com/2022/10/20/a-homebrew-ac-upgrade-for-the-fluke-8840a/)

[William Dudley]在一次拍卖中捡到了一个 Fluke 8840A 台式万用表，但遗憾地发现它读取的电阻不准确。它还缺少支持交流测量的可选板。渴望使用这个可爱的仪表，他开始修理和升级这个设备。

值得庆幸的是，8840A 来自福禄克在其手册中公开发布原理图的时代。因此，结合在网上看一些照片，【William】很容易就可以重新创建原来的 AC“Option 09”板，以实现所需的功能。像往常一样，他的努力第一次没有成功，但安装了一些电线后，一切都好了。[William]报告说，在没有进行校准的情况下，测量结果是“合理的，甚至可能是充分的”。

修复阻力问题很容易。原来是选择器开关上的腐蚀，当高电阻测量准确，但低电阻测量不准确时就会暴露出来。在开关上喷上一些触点清洁剂，轻轻一弹，一切又恢复了正常。

很高兴看到旧的硬件恢复了全部功能，特别是当它像旧的 Fluke 仪表一样有吸引力和制造精良的时候。起死回生的旧工具？[你知道我们想听！](https://hackaday.com/submit-a-tip)
