# 一个有六面的无聊故事

> 原文：<https://hackaday.com/2020/03/23/a-boring-tale-with-six-sides/>

在一块材料上钻孔是一个简单的过程，毕竟我们大多数人都会有某种形式的钻头。如果我们需要一个不是圆的洞，在不可避免的关于糟糕的钻孔控制导致椭圆形洞的笑话之后，我们该怎么办？也许是忙于一份文件？还是花钱买异形冲头？[Skunkworks]采取了不同的策略，[使用 LinuxCNC 和立式铣床加工近乎完美的六边形和其他多边形孔](https://forum.linuxcnc.org/10-advanced-configuration/38549-non-circular-boring-linuxcnc-fun)。

刀具路径看起来更像星形而不是多边形，其原因在观看断点下方的视频时变得显而易见，因为刀具的旋转将其切削刃置于多边形路径中。任何过去曾在圆孔上用锉刀锉过的人都会对这件作品印象深刻。

最新的作品将简单的六边形变成了其他形状，比如星形，甚至是锥形的多边形孔。这些特别是将是一个非常困难的任务，所以我们期待着从这个方向的其他发展。
