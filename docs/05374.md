# 在 Amiga 上运行 Java

> 原文：<https://hackaday.com/2020/01/13/run-java-on-an-amiga/>

在现代世界，我们认为某些工具是理所当然的。像 C 或 Python 这样的高级编程语言出现的时间并不长，Java 只是从 90 年代才出现。让这些工具在比它们更早的机器上工作是一个更大的挑战，而[Michael Kohn]非常愿意解决这个问题。他最近让 Java 在 Commodore Amiga 上运行。

Amgia 比 Java 早了将近十年，所以这个过程并不简单。该平台有许多协处理器，它们在当时是新颖的，但现在已经不常见了，负责处理图形、声音和内存处理等任务。在 Amiga 上运行的任何 psoftware 也需要在一个特殊格式的程序中，因此需要小心，甚至首先在计算机上加载 Java 也需要一些特殊的工作，使用零调制解调器电缆，而不是 Amiga 以前使用的软盘。

在古董 Amiga 上装载 Java 当然是一种荣誉，但[Michael]对 Java 和 Amiga 中的摩托罗拉 68000 并不陌生。世嘉 Genesis 中也有一个 68000，[我们已经看到[Michael]如何能够在其上运行 Java。](https://hackaday.com/2015/11/23/3-billion-devices-and-a-sega-genesis-run-java/)
