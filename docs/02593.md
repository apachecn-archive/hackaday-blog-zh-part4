# 我们早就应该买激光夹克了

> 原文:[https://hacka day . com/2019/04/05/我们真的早就该买激光夹克了/](https://hackaday.com/2019/04/05/we-were-really-overdue-for-laser-jackets/)

取决于你和谁说话，一切要么很好，要么我们生活在一个压抑的赛博朋克反乌托邦里，在那里我们忘记了用彩色霓虹灯浸透一切。对于跟踪我们一举一动的数字监控全景监狱，我们几乎无能为力，但就审美而言，[abetusk]正在带来商品。最新款是激光夹克，在 2019 年给你 2087 年的感觉。

建造从一件皮夹克开始，皮夹克上装饰着 128 个单独的红色激光二极管。这些器件以 4 个为一组，利用两个 PCA9685 控制器通过 32 个独立的 PWM 通道进行控制。Arduino Nano 充当操作的大脑，接收来自操纵杆和麦克风的输入。这使得用户可以控制灯光效果，并设置外套对声音和音乐做出响应。

[abetusk]很好地传达了成功实现这一目标所需的技巧。这些说明应该允许任何好奇的制造者在家里复制这个构建，Github 上的[代码可以帮助运行这个展示。有很多关于适当的外壳、连接器和布线技术的细节，以避免穿着者在穿着衣服去俱乐部时不小心把所有东西都扯碎。记住，没有什么比教育你的朋友更朋克了。](https://github.com/abetusk/dev/tree/release/projects/laser-spike)

这是一个引人注目的建筑，将是任何新芝加哥街头帮派衣橱的绝佳补充。这也不是[abetusk]第一次登上这些版面了——也有电致发光的外观。休息后的视频。
