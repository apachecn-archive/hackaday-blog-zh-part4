# 金属 3D 打印——现实的一剂良药

> 原文：<https://hackaday.com/2019/04/08/metal-3d-printing-a-dose-of-reality/>

我们毫不怀疑，黑客每天会数百次看到 3D 打印机喷出热塑料，并幻想能够直接使用金属打印。虽然金属打印机比以往任何时候都更加普遍，但对于大多数人来说，将打印作为一种爱好仍然遥不可及。但正如斯波克先生曾经观察到的:“……你可能会发现，拥有终究不如渴望令人愉快。不符合逻辑，但往往是真的。”然而，金属 3D 打印有其独特的挑战。德克萨斯 A&M 最近制作了一个简短的视频，解释了一些在 SLS(选择性激光熔化)打印机上制作实用金属印刷品时会遇到的[设计问题。你可以看下面的视频。](https://www.youtube.com/watch?v=bq_6DYyZPBQ)

描述称,“用金属 3D 打印零件比大多数人想象的更具挑战性。我们注意到，即使是塑料打印机也是如此，因为朋友们会希望我们为他们打印最稀奇古怪的东西。我们喜欢这个视频是因为它有助于设定对当前艺术状态的期望，所以我们不会期望远远超过今天的金属打印机可以生产的东西。

视频中涉及的特征包括悬垂，这需要支撑。移除后，表面大约像 80 粒度砂纸，除非你执行进一步的整理。就像塑料零件一样，大面积的翘曲和卷曲是金属的问题。如果你曾经在移除塑料支撑材料时遇到过挫折，试着用陶瓷磨掉金属支撑。他们还使用电火花加工机来切割特别坚硬的支撑物，但这需要很大的努力，因为它可能会穿过[电火花线](https://hackaday.com/2017/04/04/edm-for-the-cheap-and-adventurous/)并堵塞过滤器。

去年我们看了金属印刷的最新进展。我们已经看到自制机器在计算机控制下比电焊机强不了多少，我们也看到像惠普这样的大公司计划制造金属印刷品，但价格昂贵。尽管如此，你无法阻止 3D 打印进步的步伐。
