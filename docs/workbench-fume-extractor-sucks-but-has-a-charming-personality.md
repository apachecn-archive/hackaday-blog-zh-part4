# 工作台抽油烟机很烂，但有迷人的个性

> 原文：<https://hackaday.com/2019/08/07/workbench-fume-extractor-sucks-but-has-a-charming-personality/>

不管你做什么工作，商店安全都很重要。对于我们这些从事焊接的人来说，这意味着提取我们项目中使用的助焊剂加热时释放的有毒气体。【yesnoio】给我们带来了他自己对抽油烟机的想法，它用尽了所有的办法，还有多余的铃铛和哨子。

工作台助手机器人，正如[yesnoio]所描述的，是一个安装在一个小三脚架上的集成单元，它延伸到你焊接的工作区域。在围栏里有 RGBW 灯，一个红外摄像机，和一个 Adafruit ItsyBitsy M4 快车驱动着整个展览。除了只是将光照射到你的烙铁上，相机还可以感应到它的热活动，以决定何时启动服务器级风扇，该风扇为项目的整个排烟部分提供动力。

但乐趣不止于此，因为[yesnoio]决定追求更多的时尚元素。该机器人还配备了一个扩音扬声器，每当启动或停止风扇等动作进行时，就会播放原声片段。这些片段是一个主题的变体，比如经典的《未来》语录或《星球大战》中 R2-D2 的喋喋不休。可选择的主题被称为“表演者”，它们可以使用 CircuitPython 轻松地重新编程。这是赋予你的小桌面助手一些个性的好方法，也是打破在你的下一个原型上焊接所有那些微小的 SMD 元件的单调的有趣方法。

即使在这一切之后，如果你仍然需要一个可爱的机器人声音来说服你为你的长凳买一个油烟机，那么也许一些实际操作的结果可以给你一点你需要的推动力。如果你已经被说服了，并且想自己动手制作，[这里不缺少](https://hackaday.com/2016/10/02/a-desk-lamp-solder-fume-extractor/)[的](https://hackaday.com/2018/01/29/take-a-deep-breath-thanks-to-a-3d-printed-fume-extractor/)DIY 解决方案我们已经在 Hackaday 看到了[。休息过后，看看这个正在运行的！](https://hackaday.com/2015/06/27/downdraft-fume-extractor-saves-your-lungs/)
