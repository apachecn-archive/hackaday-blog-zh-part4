# 警惕放射性手镯和类似的东西

> 原文:[https://hack aday . com/2020/02/26/警惕放射性手镯及类似物品/](https://hackaday.com/2020/02/26/be-wary-of-radioactive-bracelets-and-similar/)

在你开始切割“负离子”健康手镯或个人按摩器之前，要知道这些很可能含有氧化钍或类似的放射性粉末，正如 YouTube 频道在[思想商场](https://www.thethoughtemporium.com/)播放的【贾斯汀·阿特金】的研究视频[所示。即使忽略具有讽刺意味的是，氧化钍主要是一种α(He^+)发射器，因此不是一种“负离子”源(这将是β衰变，e^–)，氧化钍也不是你想要在你的皮肤上或肺里的东西。](https://www.youtube.com/watch?v=C7TwBUxxIC0)

这些手镯和类似物品似乎将氧化钍颗粒嵌入到通常的硅聚合物手镯材料中，没有任何措施来防止颗粒随着时间的推移而脱落。更危险的是诸如按摩棒之类的物品，它本质上是一个充满氧化钍粉末的金属管。这不是那种你想在厨房桌子上打开，让它洒得到处都是的东西。

考虑到这些物品很容易在亚马逊、易趣和其他地方买到，在拆开它们或为一个项目切割它们之前，用 ol' Geiger 计数器快速检查一下这些物品似乎是一个健康的想法。毕竟，没有人想在他们的车间里引起放射性事故。
