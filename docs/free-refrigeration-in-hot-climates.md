# 炎热气候下免费制冷

> 原文：<https://hackaday.com/2020/08/21/free-refrigeration-in-hot-climates/>

被动式住宅是住宅建设中一个相当新的趋势，但它预示着未来我们日常生活中最少的能源投入。今年 Hackaday 奖的挑战之一是设想如何为存在住房危机的难民营中常用的土屋增加实用性。给这些土坯建筑增加被动设施将是一个了不起的升级，所以[Cat]决定通过[创造一个不需要电力的冰箱](https://hackaday.io/project/173544-free-cooling-of-food-or-rooms-in-hot-climates)来应对这一挑战。

该设备的工作原理是利用蒸发冷却来降低一个小盒子里的温度，这个小盒子可以用来储存食物。当然，使用蒸发冷却意味着你需要随时获取水，它可能不会在潮湿或凉爽的环境中工作，但像这些这样的[系统已经在世界各地的许多地方使用了几个世纪。[Cat]的计划比传统的蒸发冷却方法稍微复杂一点，它利用了一个特殊油漆的烟囱，当被阳光加热时，它可以提供气流。](https://en.wikipedia.org/wiki/Pot-in-pot_refrigerator)

该项目仍处于起步阶段，但在一个干旱的环境中，在现实生活中的被动房屋中建造一个概念验证会很有趣。不幸的是，我们这些在潮湿(或热带)环境中的人将不得不从别处寻找节能的冷却解决方案。
