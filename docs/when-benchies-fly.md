# 当板凳飞起来的时候

> 原文：<https://hackaday.com/2021/10/09/when-benchies-fly/>

我们大多数人都打印过一些 benchies 来测试我们的 3D 打印机。无畏的小船有各种功能，税收印刷过程的不同部分。然而,[FliteTest]的家伙们有不同的想法。他们开始了建造巨型飞行长椅的竞赛。他们还没有完全完成，但他们确实取得了一些有趣的进展，正如你在下面的视频中看到的。

平心而论，板凳本身并不是 3D 打印的。然而，Foamboard 更实用一些。不可避免的，看到成绩就忍不住想到飞舟。

看一辆奔驰滑行或飞过田野有点超现实的感觉。我们确实发现观看他们用来测试机体的迭代过程很有趣。飞行板凳可能不是最实用的项目，但我们明白。

在视频结束时，他们有一些东西在飞，但他们显然还有更多工作要做，他们已经承诺在即将到来的视频中分享。

如果你想要更实用的东西，为什么不把遥控船改装成水上飞机呢？鉴于奔驰是一艘船，我们希望他们[对他们的泡沫板](https://hackaday.com/2012/06/04/waterproofing-an-rc-plane/)进行防水处理。
