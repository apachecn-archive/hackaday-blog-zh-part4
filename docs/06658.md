# Tic Tac Arduintoe 将游戏移动到 4×4

> 原文:[https://hack aday . com/2020/05/26/TIC-tac-arduintoe-moves-the-game-to-4x 4/](https://hackaday.com/2020/05/26/tic-tac-arduintoe-moves-the-game-to-4x4/)

我们知道你在外面花了很多时间和你爱的人在一起，也明白你可能已经没有办法让每个人都开心了。[Mukesh]给了我们一个提示，因为他有解闷的解药——一个老把戏的新玩法，井字游戏。

代替通常的 3×3 配置，[Mukesh]把网格做成了 4×4，这样游戏会更吸引人。游戏玩法在其他方面都是一样的——这种井字游戏仍然会导致很多听牌，但它们需要更长的时间，而且你在一英里之外就看不到它们。更吸引人的是，你可以按下亮起来的按钮，而不必在每次游戏前画一个格子。

引擎盖下是一个 Arduino Uno，它可以控制 16 个按钮及其相应的 RGB LEDs。谁先走谁就是蓝色的，二号玩家得到粉色。如果你赢了，你的颜色会在棋盘上显示一个简短的胜利动画。如果游戏是平局，棋盘变成红色。我们真的很喜欢印刷的两件式按钮，其中包含 led 和启动按钮，同时保持两者分开。穿过休息时间，查看构建视频。

对 4×4 版本感兴趣，但需要花费更多时间来构建？尝试在 TTL 中构建你的 TTT。
