# 氢闸流管的乐趣

> 原文：<https://hackaday.com/2019/03/06/fun-with-a-hydrogen-thyratron/>

有些真空管有一种奇怪的危险。玻璃、发光的灯丝、奇怪的金属网格和悬挂在其中的电线——所有这些都导致了一种神秘的科幻外观，让人感觉那里正在发生奇怪的事情。

再加上一点高电压和一根可以自己制造氢气的管子，你就有了一个特别可怕的东西。这个氢闸流管最终成了[Kerry Wong]高电压、高电流实验的不二之选。人们通常会求助于闸流管的固态版本，即可控硅整流器(SCR)，来切换这样的电压。但是处理[凯瑞]所想的 30 安培电流所需的设备过于昂贵，当他用作替代品的 IGBTs 被证明有点太脆弱时，他转向俄罗斯过剩市场寻求帮助。在那里，他发现了 TGI1-50/5 氢闸流管，这是一种内部有小型氢气发生器的管子——闸流管实际上是充气的，而不是真空管，通过等离子体传导来切换大电流。[Kerry]建立了一个带有小型 RC 网络的演示电路，以提供闸流管首选的快速开关脉冲，并通过一对 1/4-W 电阻运行 3500 伏电压，结果可预测。下面的视频展示了烟花。

看不够闸流管可爱的紫色光芒吗？我们以前在这个漂亮的老式开关电源上见过它。多功能电子管还帮助[重建了第一个声音加密系统](https://hackaday.com/2019/01/27/rebuilding-the-first-vocal-encryption-system/)。
