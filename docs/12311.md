# 6502 采用 FPGA(再次)

> 原文：<https://hackaday.com/2021/12/26/6502-goes-fpga-again/>

虽然不缺乏基于 FPGA 的经典处理器再造，但我们总是乐于看到新的方法。上个月,[需要一些组装]接受了从头开始再造一台经典计算机的挑战，并以 Verilog 中的 6502 实现开始。你可以在下面的第二个视频中看到，他取得了[良好的进步](https://www.youtube.com/watch?v=c-CQO1uHNRY)，显然还会有更多的视频。

ROL 指令是第二个视频的主题。我们喜欢观察指令做什么以及它在不同变量上花费多少周期的方法，在开始之前确保你确切地知道你试图完成什么总是好的。

我们还喜欢教程使用了 Vivado 的一些更有趣的特性，比如自动验证。即使你对 Verilog 很有经验，这里也有一些好的提示。

最后，他离他的最终目标还有一段很长的路要走，但看起来他会到达那里，我们会有兴趣在视频系列完成时看到它的剩余部分。

6502 是[一个流行的逆向计算目标](https://hackaday.com/2021/04/21/gorgeous-6502-celebrates-craftsmanship-of-the-early-homebrewers/)。这个项目使用的工具之一是我们之前介绍过的 [Visual 6502](http://visual6502.org) 。同样的模拟也有 ARM1 和 6800 目标。
