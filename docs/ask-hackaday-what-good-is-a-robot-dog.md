# 问 Hackaday:机器狗有什么好？

> 原文：<https://hackaday.com/2019/09/25/ask-hackaday-what-good-is-a-robot-dog/>

据说，本杰明·富兰克林(Benjamin Franklin)在观看 1783 年蒙哥菲尔兄弟(Montgolfier brothers)在巴黎进行的首次载人热气球飞行时，当被问及这种东西的实际价值时，他回答说:“新生婴儿有什么实际用途？”富兰克林博士当然有抓住问题核心的本领。

波士顿动力公司多年来一直在戏弄的极其像狗的视频机器人 [Spot](https://www.bostondynamics.com/spot) 也是如此。对机器人生产版本的等待似乎至少部分结束了，根据业务发展副总裁迈克尔·佩里的说法， [Spot(曾被称为 Spot Mini)将很快可供“选择合作伙伴”购买](https://spectrum.ieee.org/automaton/robotics/industrial-robots/boston-dynamics-spot-robot-dog-goes-on-sale),“他们有令人信服的使用案例或开发团队，[Boston Dynamics]认为可以用机器人做一些真正有趣的事情”。

潜在购买者的资格肯定会限制早期采用者的数量，价格标签也是如此，据说价格相当于一辆新车——而且是一辆好车。因此，它不太可能很快出现在 YouTube 的拆卸视频中，所以在达夫·琼斯设法在他的神奇澳大利亚垃圾箱中找到一只之前，我们将不得不通过尝试回答一个简单的问题来娱乐自己:机器狗有什么实际用途？

## 工地上那只狗多少钱？

需要说明的是，我们只对当前的工业和军事级别的机器狗感兴趣。所有像索尼的 AIBO 这样的机器宠物都是有趣的玩具，但它们并不打算做点点和其他机器狗应该做的工作。事实上，波士顿动力公司为 Spot 发布的视频似乎完全是关于机器人已经为工业做好准备，当它在建筑工地上跳跃时，火花四溅，大雨倾盆。

 [https://www.youtube.com/embed/wlkCQXHEgjA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/wlkCQXHEgjA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



波士顿动力公司传达的信息很明确:他们打算让 Spot 成为一名工地助理，至少在一开始是这样。它 14 公斤的有效载荷能力相当可观，但不要指望它像视频显示的那样携带煤渣砖。相反，有效载荷将主要由传感器和致动器组成，客户需要将它们拖到某种危险的环境中。如果天气潮湿、多尘、封闭、炎热、有毒，或者以上情况的任何组合，看起来公司希望你派点点去做这项工作。波士顿动力公司建议的应用包括对第一反应者的搜索和救援、炸弹处理、带传感器的受限空间进入、安全、远程数据收集，甚至危险物质补救。

现在，Spot 似乎是寻找问题的解决方案。我认为这实际上没问题，波士顿动力公司与客户合作开发问题解决方案的方法可能是正确的途径。他们在这里建立的是一个非常灵活的平台，通过专注于开发控制系统，并抽象出运动学的复杂性，以至于机器人可以用游戏机式的遥控器或通过高级 API 进行直观控制，他们创造了一个工具，将能够解决任何机器人工程师都不可能想到的需求。

## 不是镇上唯一的狗

尽管波士顿动力公司的机器狗已经成为标志性产品，但它们并不是市场上唯一的犬类机器人。要么是狗的布局有什么特别之处，导致多家公司将其用于机器人平台，要么是一个简单的例子，就是搭上波士顿动力公司(Boston Dynamics)启动的宣传列车。无论哪种方式，机器狗似乎都提供了一个坚实的商机，像 [ANYbotics](https://www.anybotics.com/anymal-legged-robot/) 、 [Unitree](http://www.unitree.cc/) 和 [Ghost Robotics](https://www.ghostrobotics.io/robots) 都在研究四条腿的机器人。所有这些公司以及更多的公司都在机器狗上投入了大量的研发资金，所以它们肯定有自己的使命。但这仍然回避了一个问题:这些东西有什么用？

部分答案显而易见，看看美国陆军作战能力发展司令部的陆军研究实验室开发的一种类似狗的机器人，作为战场上士兵的贴身男仆。这种全电动骆驼与目前的机器狗有着惊人的相似之处，它被设计成在野外行动中与士兵一起工作，运送物资、弹药或武器，这些通常会在小队成员之间分配。减轻身体负担会让士兵在战场上行动更长时间，或者更有可能的是，让整个班携带更多的装备，并在相同数量的士兵下提供更大的打击。

## 轮到你了

那么你对 Spot 及其同类产品有什么看法？不久的某一天，我们会看到机器狗在建筑工地上走来走去，测量尺寸或者给人类工人送材料吗？有朝一日，消防和救援队会有像斑点一样的机器人，在消防员整装待发的时候执行主要搜索任务吗？有一天会有机器人 K9 警察部队吗？在那里，一个布满传感器的点将在汽车下面和周围爬行，进入建筑物，搜索违禁品或部署非致命武器？或者军方会把 Spot 和它的机器狗兄弟们作为力量倍增器，作为团队的嵌入式成员做繁重的工作吗？类似狗的外形比其他设计更好地解决了哪些问题？最重要的是，在哪些用例中，机器狗比真正的狗更有优势？请在下面的评论中发表意见。
