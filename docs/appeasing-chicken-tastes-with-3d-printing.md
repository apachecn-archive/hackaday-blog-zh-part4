# 用 3D 打印满足鸡的口味

> 原文：<https://hackaday.com/2020/07/23/appeasing-chicken-tastes-with-3d-printing/>

像我们大多数人一样，[亨特]和他的搭档[凯蒂罗斯]在过去的几个月里一直被隔离。与我们大多数人不同的是，他们出于自我封闭，创建了一个 3D 打印鸡肉游乐场设计黑客马拉松。想法很简单:建造一个充满定制玩具的游乐场，以满足每只鸡独特的口味。然而，考虑到鸡是[非常不可预测的](https://www.youtube.com/watch?v=NDeqvg3raOc)，执行起来可能会有点棘手。

这对夫妇根据他们的兴趣为鸡舍里的四只精选鸡设计了不同的玩具。其中一个人对蠕虫情有独钟，于是他设计了一棵树，用米线来代替生活用品，并用苔藓来覆盖它。为了深夜娱乐，圣诞树被印在夜光丝上。其他人得到了一个伪装成花朵的原地印刷旋转镜，以及一个涂有花生酱和玉米的啄柱。作为最后一件作品，第四个玩具被设计成一个攀爬架，顶部有面包奖励敢于攀爬的小鸡。因为没有一只鸡对它感兴趣，最后它们被用手喂面包。

由于他们的黑客马拉松没有其他参赛作品，[亨特]宣布自己是获胜者。他们设计的 3D 文件可供他们的顾客打印，如果他们有自己想要装饰的鸡舍的话。虽然黑客马拉松对他们来说可能是成功的，但他们的鸡似乎特别不喜欢他们的新玩具，只会表明科学和胡闹的唯一区别是写下来，或者在这种情况下，拍摄过程。如果你正在寻找其他方式将你的鸡融入创客世界，请看看[这个能够抽动的喂鸡器](https://hackaday.com/2019/08/23/feeding-chickens-with-style/)，或者[这个家庭自动化物联网鸡舍门](https://hackaday.com/2016/11/06/iot-coop-door-cares-for-chickens-tests-home-automation/)。与此同时，休息后请观看他们的发现视频。
