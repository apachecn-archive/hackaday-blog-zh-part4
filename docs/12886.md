# 自主割草机遇到障碍

> 原文:[https://hackaday.com/2022/02/28/autonomous-mower-hits-snag/](https://hackaday.com/2022/02/28/autonomous-mower-hits-snag/)

将技术和电子设备与现实世界对接通常相当棘手。像这样的项目，复杂性和边缘案例无处不在；只要问问任何曾经尝试在火星上操作漫游者、建造一个水培花园或建造几乎任何机器人项目的人就知道了。即使是我们这些仅仅拥有一台消费级打印机的人，也对它们在处理单张纸时可能出现的故障感到困惑。这个机器人割草机也不例外，它驱使它的创造者去修剪他的草坪。

[TK]实际上已经为他的自动割草机准备好了一个平台，这要感谢之前使用这个太阳能机器人探索澳大利亚内陆的建造。添加另一个电机来处理草坪修剪起初似乎很简单，他开始为它布线并将其连接到机器人。在第一次迭代后，他发现机器人移动太快，无法有效地割草，所以他添加了一个更强大的切割电机和一个变速箱，以帮助割草机在草坪上爬行得更慢。当他的钢切割刀片的 3D 打印底座粉碎时，灾难降临了，但在[TK]没有受伤的情况下，他继续进行更多的改进。

就目前的情况来看，割草机可以有效地切割向前移动的草，即使使用[TK]出于安全原因现在使用的仅使用塑料的切割刀片。割草机剥离了它的反向齿轮，所以在这个机器人在没有监督的情况下自动切割草坪之前，仍然有一些改进要做。通常我们会看到用机器人技术改造的割草机，而不是用机器人技术改造的割草机，但我们很高兴看到任何让我们少担心一件家务的方法。
