# 黑进教室

> 原文:[https://hackaday.com/2021/04/08/hacking-the-classroom/](https://hackaday.com/2021/04/08/hacking-the-classroom/)

如今，几乎有这么多学生上课，你如何让孩子——或成年人——亲身体验电子项目？[Ben Finio]说你可以通过使用 Tinkercad 将你的实验室转移到虚拟世界。[本]应该对教室有所了解，因为他是康奈尔大学的讲师。

当然，你可以用任何在线模拟器来玩这个把戏，但是 Tinkercad 很好，因为它很容易使用，看起来很真实，而且不花学生一分钱。[Ben]提到在某些情况下它特别有用，比如大班或在线课程。可能在某些情况下，这是没有意义的，比如教授 RF 设计。即使这样，也许你只是需要一个不同的工具。

如果 Tinkercad 能够添加一些特性来支持这个用例，那将是另一回事。想象一下，教师能够偷看学生的电路或修改它，而不必在屏幕上与全班分享。或者提供一个连接器，将一个学生的输出连接到另一个学生的输入。

我们不禁认为你可以用我们最喜欢的[法尔斯塔德](https://hackaday.com/2015/07/20/a-breadboard-in-a-browser/)模拟器做更多的硬核电子。它没有现实的外观，但它可以模拟很多有趣的电路。毕竟，我们长期以来一直是[基于项目的学习](https://hackaday.com/2014/12/31/hacking-education-project-based-learning-trumps-the-ivory-tower/)的粉丝。
