# 周五黑客聊天:火星上有生命吗？

> 原文:[https://hack aday . com/2018/10/03/Friday-hack-chat-is-there-life-on-mars/](https://hackaday.com/2018/10/03/friday-hack-chat-is-there-life-on-mars/)

火星不是抚养孩子的好地方。事实上，冷得要命。就算你养了，也没人养，是吗？火星上有生命吗？这是美国宇航局过去四十年来一直试图回答的问题，随着新的火星探测器的出现，我们可能会更接近答案。[在本周的黑客聊天](https://hackaday.io/event/161504-life-on-mars-hack-chat)中，我们将与负责火星 2020 漫游车上一些有趣仪器的人交谈。

[![](../Images/456759b4b889df4f74bfb3d7bbc2764e.png)](https://hackaday.com/wp-content/uploads/2018/10/lifeonmarshackchatv1_2-1.jpg) 我们本周 Hack Chat 的嘉宾是【Matteo Borri】，他是一位意大利工程师，已经在美国生活了十多年。他有各种各样的项目，从机器人——包括战斗机器人——人工智能，到航空航天。[Matteo]也是[吸血鬼充电器](https://hackaday.io/project/159752-vampire-charger)背后的工程师之一，他是今年 Hackaday 奖能量收集模块挑战赛的获胜者。

现在，[Matteo]正在进行一个有趣的项目，它将在下一个火星漫游车上飞行。他为美国宇航局和火星协会开发了叶绿素分光镜。本周，[Matteo]将分享这款设备的工作原理和开发细节。

在这次聊天中，我们将讨论在火星和银河系其他地方寻找生命的各种技术，例如:

*   叶绿素检测
*   火星漫游者
*   各种其他硬件黑客
