# 适合艰苦工作的大功率真空吸尘器

> 原文:[https://hacka day . com/2022/09/30/a-高性能真空吸尘器-艰苦的工作/](https://hackaday.com/2022/09/30/a-high-powered-vacuum-cleaner-for-tough-jobs/)

真空吸尘器非常适合整理房间，但它们通常无法处理一个合适的车间的笨重、肮脏的东西。[CraftAndu]目前正在建造一艘帆船，并且发现普通的真空吸尘器无法承受太多的锯屑。因此，他建立了自己的强大真空，能够处理这种情况。

这座建筑的核心是一个巨大的 [3.8 千瓦的除尘器](https://koneita.com/en/dust-collectors/474-nova-fm-300s5-puruimuri.html)，它被用作车间除尘系统的一部分。这是你通常用来吸机床灰尘的那种类型。然后给它装上一根长长的软管，连接到真空手柄上。手柄由多段污水管和几个适配器组成，将它们连接在一起，并钩住软管。它还装有一组轮子，使它可以在商店的地板上轻松滑行。

这是一个吸取所有在车间周围收集的轻质锯屑的好方法。然而，[CraftAndu]指出，即使有 3.8 千瓦的提取系统为其提供动力，使用扫帚清理木屑等较大的碎屑仍然更快。
