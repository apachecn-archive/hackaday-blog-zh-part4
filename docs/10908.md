# 声音悬浮

> 原文：<https://hackaday.com/2021/08/02/levitation-by-sound/>

如今，用磁铁悬浮东西已经不是什么了不起的壮举了。我们没有看到多少项目有音速悬浮。然而，日本工程师有了一种新方法，利用声音来举起物体。这个过程还不完全可靠，但它可能会在未来导致更好的方法。下面可以看到一个关于作品的视频。

通过激光或声学操纵非常小的物品并不新鲜。然而，目前的方法有很大的局限性。这种新方法使用半球形超声换能器阵列。通过控制每个换能器的振幅和相位，形成了一个声阱，可以在不直接接触的情况下拾取 3 毫米的聚苯乙烯球。

我们对无接触操纵物体感兴趣有几个原因，不仅仅是电路组装。这种稳健的技术也可以为增材制造增加新的维度。当然，从 3 毫米的聚苯乙烯球到表面贴装元件还有很长的路要走。然而，你不得不承认看着组件从空中飘到它们最终的归宿是很有意思的。

不是说在之前我们没见过[音速悬浮。](https://hackaday.com/2021/02/02/building-an-ultrasonic-levitation-rig/)[磁悬浮](https://hackaday.com/2017/10/08/hovering-questions-about-magnetic-levitation/)趋于简单，但也有一定的局限性。
