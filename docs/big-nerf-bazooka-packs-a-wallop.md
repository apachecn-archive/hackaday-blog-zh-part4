# 大 Nerf 火箭筒具有强大的冲击力

> 原文：<https://hackaday.com/2020/11/20/big-nerf-bazooka-packs-a-wallop/>

Nerf 爆能枪是一种有趣的玩具，一旦孩子们在战斗中击中家中太多珍贵的装饰品，就会被没收。然而，【伊万·米兰达】比大多数孩子都要大，[并着手建造一个更大的爆破器。](https://www.youtube.com/watch?v=ihTW3_n5VuU)

这种类似火箭筒的设计使用了几米长的 160 毫米 PVC 管，发射由泡沫瑜伽滚轮和抛光垫制成的“飞镖”。该建筑在其构造中使用了一些 3D 打印组件，既作为点火机制的一部分，也作为夹具来帮助加工管道。一个大的活塞被用来推动飞镖，在被扳机释放之前，它会被厚橡胶管的张力拉回。

毫无疑问，这是一个令人生畏的装置。然而，我们怀疑它的短射程、巨大的体积和缓慢的重装时间应该能阻止它在你当地的 Nerf 战斗中破坏元游戏。也就是说，我们仍然不想从这个坏男孩的头上开一枪。黑客确实喜欢好的 Nerf 版本，而且他们在哨兵应用中特别受欢迎。休息后的视频。
