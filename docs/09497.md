# 种植世界上最大的雪花

> 原文:[https://hack aday . com/2021/03/12/种植世界上最大的雪花/](https://hackaday.com/2021/03/12/growing-the-worlds-largest-snowflake/)

世界上很多地方没有降雪，所以如果你住在这些地方，你需要去旅行，体验冬天真正的快乐。如果你不愿意旅行，你可以自己做一些类似的冰晶来代替。虽然这个来自[布莱恩]又名[阿尔法凤凰]的建筑没有产生一系列小冰晶，但它确实以一种非常特殊的方式产生了一个巨大的冰晶。

[布莱恩]正在生长的冰是在一个专门为这个六角形晶体设置的压力室中形成的。不像普通的冰是由随机排列和不同的晶体冻结在一起组成的，这个巨大的冰块实际上是一个单一的晶体。当空气被抽出压力室时，容器中只剩下晶种和水蒸气。内部带有散热器的定制 peltier 冷却器有双重作用，既可以保持冰晶的低温(和生长),又可以加热容器底部的一小滩水，以增加容器中的水蒸气量，水蒸气最终会沉积在特定六边形的晶体上。

这个建筑看起来很有趣，因为冰晶的生长必须在冰箱里拍摄，所以这里可能有第二个黑客，涉及到在那个不寻常的环境中设置相机设备。不管怎样，经过多次尝试后，巨大的冰晶雪球最终从冰箱里出来了，这也不是我们第一次看到定制珀尔帖冷却器的有趣应用。
