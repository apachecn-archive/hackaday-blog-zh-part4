# 悬挂式滑动 Raspi 相机为 Octoprint 增添了维度

> 原文：<https://hackaday.com/2018/08/16/hanging-sliding-raspi-camera-adds-dimension-to-octoprint/>

你用 Octoprint 了吗？它不仅仅是一种通过互联网控制打印机的方式，也不仅仅是一种在您下班或去拿啤酒时通过网络摄像头监控打印机的方式。3D 打印社区已经围绕 Octoprint 团结起来，创造了各种各样方便的插件，如 Octolapse，它可以让你通过延时视频在床上观看打印开花。

Hackaday 校友(杰瑞米·S·库克)在发现自己受到(汤姆·纳尔迪)对 [Octoprint](https://hackaday.com/2018/01/03/upgrading-a-3d-printer-with-octoprint/) 和 [Octolapse](https://hackaday.com/2018/07/02/coolest-way-to-watch-3d-printing-lights-camera-octolapse/) 的出色报道的启发后，想要为 Raspi 相机设计一个 3D 打印底座。他最近买了一个钢丝货架单元来存放他的打印机和打印机配件，并开始工作。我们喜欢他提出的设计，该设计利用冷却剂软管的灵活性来提供无限可配置的相机臂。但是等等，还有呢！由于[杰里米]用拉链将它安装在机架上，整个装置来回摆动，为更多的相机视角提供了额外的轴。滑过休息时间，观看[Jeremy]的构建/演示视频。

这是伟大的，能够监控一个打印从任何地方与互联网接入，但相机几乎总是设置为在打印床上拍摄紧密。你怎么知道你的灯丝是否快用完了？为此，你需要一个菲拉计。
