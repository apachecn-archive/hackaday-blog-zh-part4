# 这个盒子为你计算你的祝福

> 原文：<https://hackaday.com/2020/02/05/this-box-counts-your-blessings-for-you/>

每年的这个时候，深圳的生产又陷入停顿。农历新年拉开了一年一度长达一个月的春节的序幕，整个国家都抽出时间与家人团聚和庆祝。春节的一个传统是，每个人都给对方红包，里面有各种各样的钱，叫做红包。这个“压岁钱”的目的是在新的一年里传播好运，在节日结束时很容易积聚一大堆红包。

[Makerming]制作了这个可爱的计数盒，让任何人都能以时尚的方式记录他们的红包。不言而喻，这将是一个非常棒的情人节邮箱(或者任何适合的邮箱)，但是你去吧。由于为 Arduinos 设计的 Grove 连接器屏蔽，电路非常简单。红外光束中断模块检测到进入的信封，Uno 增加显示器上的计数。顶部有一个晃动的伺服驱动的红包来增加乐趣。

我们喜欢正面的激光切割装饰，这是对错综复杂的剪纸装饰的敬意。如果你不喜欢那一个，[Makerming]包括了几个其他选项的设计文件。休息后看它摆动！

如果你不打算收集情人节礼物，因为你已经找到了那个特别的人，那就送他们比巧克力或玫瑰更持久的礼物。
