# ATtiny 系列是孤立的好伴侣

> 原文：<https://hackaday.com/2020/04/02/the-attiny-series-is-a-great-companion-in-isolation/>

由于社会距离和自我隔离，许多制造商一直在寻找治愈无聊的方法。那么，当你把一个制造商放在一个有电子零件的封闭空间里，会发生什么呢？答案是[一堆帮助打发无聊的随机微控制器项目](https://www.youtube.com/watch?v=PSSy3VLXqS8)。[Danac1886]发布了一个视频，其中包含一系列 ATtiny 系列微控制器的实验，这些实验可以成为这些孤独日子里消磨时间的灵感来源。

该视频基于从 ATtiny85 到 ATtiny84 的各种控制器，甚至包括 ATtiny2313。还有一个关于 ATtiny10 的项目，这是一种 SMD SOT23-6 封装，看起来非常惊人。所有设备都可以使用 Ardino 作为 ISP 进行编程，因此，如果您没有 AVR ICSP，您需要的只是另一个 Arduino。

至于项目本身，有各种各样的东西，从基本的闪烁 LED 开始，添加 I2C LCD，然后移动到 7 段显示器，用锅控制变速计数。我们真的很喜欢这些小项目激发了人们的灵感，帮助人们开始学习基本的电子学和编程。

如果你想开始，看看带 attini 10 的巨型 LED 灯，我们向你保证，它会点亮你的一天。
