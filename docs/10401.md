# C64 的双显示器设置，是的，它是 VGA 兼容的

> 原文:[https://hackaday . com/2021/06/10/a-dual-monitor-setup-for-the-c64-and-yes-its-VGA-compatible/](https://hackaday.com/2021/06/10/a-dual-monitor-setup-for-the-c64-and-yes-its-vga-compatible/)

在 20 世纪 80 年代，很少有人过分担心他们的家用电脑只支持一台显示器；事实上，大多数人对当时提供的全新功能已经足够满意了。也就是说，几十年过去了，我们确实应该得到更多。不要害怕，[因为【Ryan Brooks】在这里帮助他为 64 号准将准备 VG64 VGA 卡。](https://www.hack.net/vg64/)

该卡位于 Commodore 64 的盒式插槽中，并装有一个 Xilinx CPLD，负责产生视频输出信号。它连接到一个 SRAM 芯片，作为视频输出的帧缓冲器。然后可以将程序加载到 Commodore 64 上，该程序写入帧缓冲器，然后可以将该帧缓冲器发送到与盒式磁带相连的 VGA 监视器。

这不是目前最有用的购物车，因为它只能与专为硬件设计的软件一起工作。此外，将足够的数据转移到 it 部门以进行任何类型的快速动画制作或更新可能会很困难。也就是说，这是一个很好的例子，说明了在扩展 Commodore 64 方面可以实现什么，我们很想知道这一领域的工作能走多远。我们以前也见过类似的作品，尽管是用一个稍微小一点的 16×2 字符 LCD。休息后的视频。
