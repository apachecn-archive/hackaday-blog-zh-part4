# 针刺您的项目:3D 印刷电路板探测夹具使用针刺针

> 原文：<https://hackaday.com/2019/11/15/needling-your-projects-3d-printed-pcb-probing-jig-uses-accupuncture-needles/>

试图探测一个带有微小 SMD 元件的现代电子电路，同时不让神奇的烟雾在此过程中逸出，可能是一个相当大的挑战。尤其是因为我们黑客还没有开发出在操作示波器和计算机的同时固定 3 个不同探针所需的附件数量。[Giuseppe Finizia]用一个使用针灸针的 [3D 印刷电路板探测夹具](https://www.thingiverse.com/thing:3615910)解决了这个问题。

作为一名电子取证实验室的工程师，朱塞佩的一部分日常工作是对缴获的设备进行技术调查，这涉及到相当多的探测。夹具由一个带槽的基板组成，各种配置的 PCB 固定器在槽中滑动，以固定所有形状和尺寸的 PCB。围绕该板的圆周有多个可调节探测“起重机”的位置，每个起重机都有一个压接或焊接到导线上的针灸针。每个持针器都有一点弯曲，这使它能够保持向下的压力，以形成可靠的连接。

制造一次性工具和夹具可以说是 3D 打印的最佳应用之一，这是一个完美的例子。当然，如果你有东西要抓，你可以点上焊线或者使用测试钩，但是要轻松地探测任何 PCB 上的多个点，这看起来是一个非常好的解决方案。如果你试图追踪一个单一的信号，一个[精密的受电弓](https://hackaday.com/2017/10/04/precision-pantograph-probes-pcbs/)可能是你的朋友，或者你可以[给你的示波器](https://hackaday.com/2018/12/02/keep-both-hands-on-the-probes-with-this-oscilloscope-footswitch/)增加一个脚踏开关，以便用手快速检查电路。
