# 蒸汽朋克无刷电机演示推动所有的制造商按钮

> 原文:[https://hack aday . com/2020/05/29/steam punk-brushly-motor-demo-pusses-all-the-maker-buttons/](https://hackaday.com/2020/05/29/steampunk-brushless-motor-demo-pushes-all-the-maker-buttons/)

我们将诚实地面对:在【大卫剑桥】的[无刷电机和控制器制造](https://www.youtube.com/watch?v=AAVMHnMYqpo)中没有什么新东西。如果你在寻找惊天动地的创新，你最好去别处看看。但是，如果你喜欢漫无目的地使用黑客工具包中的每一种技术和材料，并且技艺高超，那么这可能适合你。还有尼克斯——他也有尼克斯在里面。

[David]的构建开始于对无刷电机及其工作原理的个人探索。一些 3D 打印的零件，一个线圈和一个磁簧开关导致了一个简单的脉冲马达，表现令人惊讶地好。这演变成一个带有霍尔效应传感器和自制控制器的六线圈电机。这就是[大卫]全力以赴使用工具的地方——一台车床、一台等离子切割机、一台焊机、一台铣床，以及一套精心挑选的木工工具，用于制造最终电机的零件以及该项目的外壳。因为他还没有检查完所有的盒子，所以[David]决定使用 3D 打印的框架作为用铝铸造的模型。

成品电机，重新设计了转子，以更好地处理涡流，加入了木材和金属外壳，以及数码管转速表和蚀刻黄铜控制板。对于一个显然是自我熏陶和技能培养的项目来说，这是一个很好的外观，我们喜欢它。我们以前见过其他的 BLDC 示威者，但是很少有人看起来像这次这么好。
