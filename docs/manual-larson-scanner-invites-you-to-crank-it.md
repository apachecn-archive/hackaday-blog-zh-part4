# 手动拉森扫描仪邀请你转动它

> 原文：<https://hackaday.com/2020/08/22/manual-larson-scanner-invites-you-to-crank-it/>

哈塞尔霍夫使拉森扫描仪出名。这是 KITT 前面扫描红灯的名字，KITT 是上世纪 80 年代热门电视剧《霹雳游侠》中的英雄车。尽管它们只起到审美的作用，但时至今日，它们仍然是一个有趣且受欢迎的 LED 项目。旧的概念有了新的变化，[皮特·普罗多尔]发明了[一台拉森扫描仪，你转动曲柄就能操作](http://rasterweb.net/raster/2020/08/20/manual-crankable-larson-scanner/)。

该项目由乐高积木搭建而成，依靠手摇曲柄工作。曲柄转动一个鼓，鼓上放置几条导电的标记带——[一种我们之前见过的钢/尼龙材料。](https://hackaday.com/2019/12/09/conductive-tape-current-capacity-comparison/)并排运行的胶带条在滚筒转动时被滚筒上的胶带段桥接。led 以传统 Larson 扫描仪的必要模式打开。

该项目激发了更多的可能性，例如使用类似的技术生产一个电子音乐盒或演奏钢琴，当用户用曲柄改变速度时，它会改变速度。[Pete]指出，转动曲柄是一种内在的愉快体验，鉴于手摇音乐项目中固有的奇迹，如大理石机器 X，我们迫不及待地想知道这一项目下一步会走向何方。休息后的视频。
