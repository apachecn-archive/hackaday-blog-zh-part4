# 建造这个万圣节主题的时态雕塑

> 原文：<https://hackaday.com/2022/10/21/build-this-halloween-themed-tenseggrity-sculpture/>

张拉整体雕塑是有趣的东西，通常作为办公桌玩具或科学新奇品出售。你也可以创建自己的网站，[，[seabirdhh]已经推出了一个有趣的节日主题版本。](https://www.instructables.com/Halloween-TensEGGrity-3D-Printed/)

首先要建造的是蛋帽架。这由结构的基础组成，蛋角色的“帽子”挂在中心。该结构的另一半是单独建造的，其余的“蛋头”坐在上部结构底部的一个杯子里。然后将一系列尼龙丝系在组件之间。然后这些可以被拉紧以赋予结构形状，允许蛋的“帽子”在它的“头”上方盘旋。[seabirdhh]将尼龙线穿过小块橡胶，这样可以将张力调整到合适的位置。太少的话结构会倒塌，但是太多的话，时间一长就会弯曲。仔细调整是关键。

这是一个有趣的建筑，也是一种在家试验张拉整体概念的廉价方式。你甚至可以使用这些相同的技术来[建造一架四轴飞行器，](https://hackaday.com/2020/11/05/quadcopter-with-tensegrity-shell-takes-a-beating-and-gets-back-up/)或[将它们应用到乐高的世界中。](https://hackaday.com/2020/04/28/a-lego-tensegrity-structure/)休息后的视频。
