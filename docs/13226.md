# 悬浮在 NE555 公路上

> 原文：<https://hackaday.com/2022/04/04/levitate-the-ne555-way/>

超声波悬浮——在两个超声波源之间产生驻波并定位轻质物体，使它们能够在它们之间的最小压力下漂浮的实践——已经成为不止一个实验者的兴趣所在。[林建岳]在休息时间下面的视频中演示了这一点，只使用所有真正实验者信任的芯片 NE555 创建了一个超声波悬浮系统[。(视频，嵌在下面。)](https://www.youtube.com/watch?v=B4pIJwy_ySY)

电路本身很简单，只是一个制造了十亿个蜂鸣器和闪烁的发光二极管的那种桌子。它平行地驱动两个超声波传感器，让它们相互指向对方，再调整一点间隙，就能成功地让聚苯乙烯悬浮起来。在调整频率以适应传感器共振方面做了一些工作，但如果仪器合适，这并不是一个巨大的挑战。我们可以看到，这将是驻波的一个很好的演示，也是一个非常棒的桌面玩具。

我们在 Hackaday 庆祝每个人最喜欢的计时器芯片，以至于最近我们举办了一场比赛，以寻找使用它的最佳作品。
