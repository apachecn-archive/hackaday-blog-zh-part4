# 凝视这个复杂的维多利亚时代的时间锁

> 原文:[https://hacka day . com/2019/05/29/gaze-on-this-interfaced-Victorian-era-time-lock/](https://hackaday.com/2019/05/29/gaze-upon-this-intricate-victorian-era-time-lock/)

时间锁的概念是一个古老的概念，在这里你可以看到一个[发条装置和齿轮版本](http://www.industrialalchemy.org/articleview.php?item=3131)的例子，它保持金库密封，防止未经授权的开启。即使知道正确的组合，这些装置也会阻止打开，直到预先安排的时间过去。[工业炼金术]的精英们得到了一个耶鲁大学 Triple L 机械锁的复制品，和其他同类设备一样，它需要手动上弦才能工作。因为整个装置是密封的，不会被篡改，所以上链和设置都是用钥匙通过前面的小孔来完成的。

这些装置被安装在保险库门的内部，并通过与锁机制的机械接口以各种不同的方式工作，这取决于品牌和型号。当时间锁打开时，即使使用了正确的密码，也无法打开门。你可能会注意到多重运动；这是为了冗余。机芯以机械或排列方式连接，这意味着第一个倒数到零的机芯将解除时间锁。在发生故障的情况下，备用机芯将负责防止完全锁定——这种情况既不方便又令人尴尬，而且成本高昂。

下面嵌入的是一个视频，重点是交换时间锁的运动，但碰巧也做了很好的展示机械设计和组件。发条装置是当时的高科技，随着 3D 打印的普及，人们对它的兴趣又有所复苏。
