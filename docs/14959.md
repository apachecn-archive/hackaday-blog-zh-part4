# 有翼无人机获得向前飞行能力

> 原文:[https://hack aday . com/2022/10/02/winged-drone-gets-forward-flight-capability/](https://hackaday.com/2022/10/02/winged-drone-gets-forward-flight-capability/)

无人机在今天的电子领域相当普遍，不仅仅是一种有趣的爱好。它们让众多房地产经纪人、YouTubers、测量员、应急响应人员和其他专业人士拥有了一个极其强大的工具。这些工具的一个缺点是功耗往往很高。你可以给它们装上更大的电池，或者像尼古拉斯演示的那样，在飞行中快速旋转它们。

不久前，我们展示了他对这种多模态无人机飞行风格的首次测试，但这里有一个快速总结:通过将机翼连接到每个螺旋桨的臂上，然后旋转整个无人机，水平飞行的动力需求可以大大降低。这一次，他回来展示这种独特设计的另一个好处，即它能够侧转，像飞机一样水平飞行。在视频中看到它有点奇怪，因为它看起来有点像一个静止的螺旋桨在天空中蜿蜒盘旋，但由于手臂上的翅膀，这种飞行模式的动力需求也大大降低了。

这种设计有一些缺点，即垂直机翼只会增加水平飞行的阻力，所以它不如一些双翼设计有效，但它以更有效的悬停能力来弥补这种损失。他还计划在未来的制作中演示在旋转悬停模式下使用摄像机。这是一个令人印象深刻的实验，推动了多旋翼飞行器的极限，而且[Nicholas]仍然计划改进设计，特别是在旋转悬停模式下增加更好的控制。我们预计，除了这架之外，许多其他无人机也会获得一些效率提升。
