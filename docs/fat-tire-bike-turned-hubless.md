# 胖轮胎自行车变成了无轮毂

> 原文：<https://hackaday.com/2021/06/24/fat-tire-bike-turned-hubless/>

一个多世纪以来，自行车车轮看上去几乎没有什么变化，这是有充分理由的:它很好用。[The Q]决定稍微忽略原因，专注于外观，所以他建造了一辆没有任何轮毂或辐条的[胖轮胎自行车。](https://www.youtube.com/watch?v=RMUrJhet-1M)

为了完成这项工作，他制作了两套环形“轮毂”,大小与轮辋差不多，圆周上有一系列滚珠轴承，供轮辋滚动。最初的货叉被剪短并焊接在一组支架上，这些支架用螺栓固定在新的轮毂上。这使得后端更加复杂，因为没有地方安装链轮盒。原来的后轮毂，盒式和盘式制动器被移到了框架内。这驱动后轮使用第二个链条连接到一个大链轮直接安装在轮辋。前刹车干脆取消了。

虽然这种新设计不会在现有的自行车上采用，但我们怀疑实用性是建造中的优先事项。这绝对是一个令人眼花缭乱的东西，我们不禁看到了一个更进一步制造 TRON 自行车的机会。

就在最近，【The Q】将另一辆胖轮胎自行车变成了[全轮驱动极限越野车](https://hackaday.com/2021/05/26/all-wheel-drive-bicycle-using-hand-drill-parts/)。对于另一个踏板驱动的头部翻转器，检查一下 [strandbeest 自行车](https://hackaday.com/2016/06/30/a-strandbeest-bicycle/)。
