# 建造一台 x 光机

> 原文:[https://hackaday.com/2021/09/09/building-an-x-ray-machine/](https://hackaday.com/2021/09/09/building-an-x-ray-machine/)

虽然我们通常鼓励黑客在可行的情况下制造他们自己的工具或机器，但 x 光机通常不在此列。尽管有辐射的风险，[威廉·奥斯曼]还是这么做了，并且[建造了一台自制的 x 光机](https://www.youtube.com/watch?v=IiJAq53knwc)。在收到一份令人眼红的医疗账单后，[威廉]决心自己制造 x 光机，希望避免未来的账单。多亏了他的保险，欠款总额减少了，但对那些生活在单一支付者医疗保健国家的人来说仍然是荒谬的，但这让威廉开始思考。如果他能制造一台 x 光机来做廉价 x 光呢？

带着从网上拍卖行买来的廉价高压 DC 电源，他开始给他的 x 射线真空管通电。一个较小的电源给阴极提供能量，形成电子束。然后施加高电压(30-150kv)作为管电压，将电子加速成 x 射线。使用盖革计数器和铅板来采取安全措施有些随意。用弹道壳铸造的指骨[威廉]在 DSLR 上用长时间曝光制作了他的第一张 x 光片。下一个进入 x 光“室”的物品是一部电话和一只手。结果相当不错，你可以清楚地看到骨头。

我们以前在 Hackaday 这里见过自制的 x 光机，但没有一台是如此随意建造的——他的方法让这一点显而易见:不要在家里尝试。休息后的视频。
