# 超级简单的 VGA 适配器仅用四个 TTL 芯片支持低分辨率输出

> 原文：<https://hackaday.com/2020/10/24/super-simple-vga-adapter-sports-low-res-output-with-only-four-ttl-chips/>

在 Hackaday 网站上，我们对那些最夸张的说法投以警惕的目光。一般来说，如果我们发布一个号称有史以来最快或最小的东西，我们会立即被评论中做得更快或更小的人打倒。因此，我们推出了有史以来最简单的 TTL 视频卡,虽然我们知道同样的事情会发生，但我们渴望看到任何人如何缩小规模。

公平地说，[乔治·富特]确实将他的声明限定为最简单的可用 VGA 适配器，并且他确实注意到它来自[[食本者]的“世界上最差的显卡”](https://hackaday.com/2019/07/07/low-res-video-card-is-still-amazing-since-its-made-out-of-logic-chips/)，他将其用于他的 6502 试验板计算机。但是[Ben]的 VGA 适配器使用了大约 20 个 TTL 芯片和一个 EEPROM，[George]设法将 BOM 减少到只有 4 个 TTL 芯片以及存储器和一个晶体振荡器。当然，这需要相当多的妥协；色深相当低，分辨率也一样。每个像素看起来像一条细横条，而不是一个小方块，导致图像在屏幕上模糊不清。然而，它们仍然令人惊讶地可见，这可能更多地说明了我们耳朵之间的模式识别湿件的质量，而不是适配器的质量。[George]在下面的简短视频中介绍了赛道。

自从我们第一次发现这个项目以来，似乎[George]已经发布了一些对它的改进，所以我们很期待看到部件数量增加了多少。我们也很想看看是否有人能超越这项工作的简单性——如果你尝试了，一定要让我们知道。
