# 如果你看不到太阳能电池板，并不意味着它不存在

> 原文:[https://hack aday . com/2022/12/31/if-you-can-see-a-solar-panel-than-that-mean-it ' s-not-they/](https://hackaday.com/2022/12/31/if-you-cant-see-a-solar-panel-that-doesnt-mean-its-not-there/)

在远离化石燃料能源的过程中，太阳能有了巨大的发展。我们已经看到沙漠中的太阳能热电厂和覆盖大面积土地的光伏板农场，但也许最大的潜力来自于将太阳能板放置在屋顶上。在世界上的一些地方，这是通过补贴制度来鼓励的，例如意大利。但是如果你的建筑是受保护的世界遗产的一部分，比如罗马城市庞贝呢？答案是传统的屋顶瓦片，它将光伏元件隐藏在聚合物表皮下，看起来就像传统的罗马瓦片。就像这类产品经常出现的情况一样，[制造商的描述页](https://www.dyaqua.it/invisiblesolar/_en/index.php)以保护他们发明的名义对细节守口如瓶。他们告诉我们的是，这种瓷砖使用安装在聚合物层下面的传统太阳能电池，聚合物层被描述为“*看起来不透明，但对阳光半透明*”。这听起来像是一个固有的矛盾，所以很自然，我们对它是如何工作的很感兴趣。

线索来自它声称的特性，其中之一是它具有光催化自清洁功能。这意味着二氧化钛薄膜的存在，它在光的存在下从空气中产生氧自由基。二氧化钛作为白色颜料和食用色素无处不在，但它在薄膜中也有有趣的颜色特性，用于彩虹色和反射效果。如果我们想猜测这些是如何工作的，我们希望它们使用精心选择的染料，其光谱轮廓不会过多地干扰太阳能电池的光谱轮廓，以及二氧化钛薄膜赋予的颜色效果。

无论它们是如何工作的，这些瓷砖都是一项迷人的技术，我们很想知道更多。这当然不是我们带给你的第一个太阳能屋顶创新。
