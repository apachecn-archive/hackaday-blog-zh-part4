# 微型街机使用微型阴极射线管

> 原文：<https://hackaday.com/2022/07/27/tiny-arcade-uses-tiny-crt/>

修复老式电子产品是一个很难解决的爱好。即使是最实用的构建通常也必须使用某种形式的现代技术才能正常工作，或者需要拆卸许多不同版本的机器才能获得单个工作版本。无论哪种方式，最终都会有人因任何古董的毁坏而深受伤害，也许除了[Marco]最近的带有独特 CRT 显示器的小拱廊。

CRT 现在是一种过时的技术，但街机和 MAME 纯粹主义者经常寻求它们，因为与现代 LCD 或 LED 显示器相比，这些设备具有圆形屏幕和复古感。不过，对于这么小的建筑来说，[Marco]不能只使用旧电视机的零件，因为柜子后面没有空间。然而，一个过时的视频会议系统恰恰是他所需要的。它有一个垂直安装在曲面屏幕上的阴极射线管，以显著降低所需的深度。

最终的构建使用了一个微小的 Namco 系统，旨在插入标准电视的 RCA 插孔，但放在一个定制的外壳中，使它看起来像一个古董视频游戏柜。这是一个有趣的建筑，没有破坏任何有价值的古董电子设备，同时仍然保持了经典的街机风格。如果你正在建造一个更大的拱廊橱柜，仍然可以满足那里的纯粹主义者，确保你使用的是带有正确控制系统的 CRT 显示器。
