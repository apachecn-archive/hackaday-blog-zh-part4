# 好玩的特斯拉可能是世界上最小的

> 原文：<https://hackaday.com/2020/02/22/fun-size-tesla-might-be-the-worlds-smallest/>

我们得到了关于“世界上最棒的”小部件的各种各样的提示，这些小部件通常最终会吸引成群结队的拆穿者。所以通常情况下，我们会回避对一个项目做出最大的评价，不管他们如何给自己开账单。但是我们很高兴这是世界上最小的特斯拉，至少如果我们要扩大特斯拉的定义的话。

奥斯汀制造的特斯拉 Model S 的小丑车版本基于电动轿车的无线电传单复制品。这款价值 600 美元的电池驱动原型车并不能提供与其全尺寸车型完全相同的加速性能，因此他剥离了精致的塑料车身，并将其安装在经过大幅改装的卡丁车底盘上。微小的轴距几乎没有留出伸腿的空间，但由于座位安装得足够靠后，进入了导致车轮打滑的区域，所以[奥斯汀]挤进去是可能的。他选择向特斯拉的电池组设计致敬，建造了 16 个模块，每个模块有 14 个 18650 电池，对于这样一辆小型汽车来说，这仍然是一个很大的电池。液压制动器也增加了，这是一个明智的决定，因为 4800 瓦的 BLDC 是一个有点快比股票电机，至少可以说。下面的视频显示了建设，以及一个危险的测试乘坐的速度读 72 在一点上；我们不确定这是每小时多少英里还是每小时多少公里，但不管怎样，这都很可怕。这些积雪也很恶心。

看起来[奥斯汀]需要速度和漂移。我们以前见过[他的水冷电动漂移三轮车](https://hackaday.com/2018/12/12/electric-drift-trike-needs-water-cooling/)，以及[他那可笑的超负荷疯狂推车](https://hackaday.com/2019/10/06/upgrades-give-crazy-cart-xl-a-few-more-xs/)。
