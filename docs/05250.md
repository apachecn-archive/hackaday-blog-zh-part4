# 您的 TS80–音乐播放器

> 原文：<https://hackaday.com/2019/12/31/your-ts80-music-player/>

到目前为止，大多数读者都已经熟悉迷你工具 TS100 和 TS80 烙铁，这是一种小巧轻便的温控焊接工具，为像样的烙铁市场的低价端树立了新的标准。我们知道他们有一个 STM32 处理器，一个 USB 接口，和一个有机发光二极管显示器，并且已经为他们生产了各种不同的固件。

仔细看看 TS80，你会发现元素连接器相当熟悉。这是一个 3.5 毫米的插头，我们更习惯于作为音频连接器。烙铁发出的声音肯定很疯狂吧？如果你是[Joric]就不会了，他已经[为小 USB-C 熨斗开发了一个音乐播放器固件](https://github.com/joric/ts80player)。这算不上是音乐娱乐的*绝技*，也不会让发烧友们远离他们的参考 DAC，但它至少制造了一个可识别的*我们祝你圣诞快乐*，正如你将从休息时间下面的视频中看到的。

自从几年前 TS100 问世以来，我们已经看到了各种创新的固件。你可能还记得[Joric]之前在俄罗斯方块游戏中的胜利，但我们最喜欢的可能是[的 TS100 示波器](https://hackaday.com/2017/09/28/part-soldering-iron-part-hand-held-oscilloscope/)。
