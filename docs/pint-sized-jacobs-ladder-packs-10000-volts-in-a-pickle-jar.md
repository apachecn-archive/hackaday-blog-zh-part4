# 一品脱大小的雅各布之梯在泡菜坛子里装了 10，000 伏特的电压

> 原文：<https://hackaday.com/2018/11/05/pint-sized-jacobs-ladder-packs-10000-volts-in-a-pickle-jar/>

把这个归档，作为你明年万圣节的疯狂科学家服装:[【bitluni】的口袋雅各布的梯子](https://www.youtube.com/watch?v=0Dp51Z-iPF4)对于那些有五美元零件、一台 3D 打印机和非常大的口袋的人来说，是完美的高压配件。

[bitluni]的视频向您展示了您需要的所有部件，并指导您完成非常简单的构建过程。对于零件，你需要一个便宜且容易获得的高压变压器，一个电池座，一些用于导体的银线，以及一些其他小零件，如焊料和电源开关。

一旦电子设备焊接在一起，它们就会被塞进一个 3d 打印的盒子里，这个盒子是[bitluni]用 FreeCAD 设计的。FreeCAD 和 STL 文件都可以在 Thingiverse 上找到。我们不确定[比特鲁尼]用什么类型的罐子来装电极。如果你的 jar 不匹配，你必须熟悉 FreeCAD 或者从你最喜欢的 CAD 软件包开始。

无论哪种方式，我们都喜欢对电气安全和项目外壳家用物品再利用的轻微点头。

如果你对电压明显更高的雅各布梯[感兴趣，我们已经为你准备好了](https://hackaday.com/2015/03/05/11000-volt-jacobs-ladder-sounds-like-a-lightsaber/)，或者我们也写过关于[的另一款小型便携式雅各布梯](https://hackaday.com/2017/06/09/portable-jacobs-ladder-for-when-you-know-you-need-a-portable-jacobs-ladder/)。
