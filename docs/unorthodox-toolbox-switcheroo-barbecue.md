# 非正统的工具箱开关:烧烤！

> 原文：<https://hackaday.com/2022/06/08/unorthodox-toolbox-switcheroo-barbecue/>

尽管在过去的几千年里烹饪方法有了很大的进步，但是没有什么能取代当你的食物慢慢准备好的时候盯着余烬的原始感觉。烧烤是满足这种洞穴怀旧的显而易见的选择，虽然在某些情况下大小可能很重要，但有时你只是想方便地将烧烤设备带到海滩、公园或除了家以外的任何地方。[其他时候，你【Laura Kampf】不想用旧工具箱存放工具](https://www.youtube.com/watch?v=o4pSovPBgME)。

这一切都是从一个典型的三层折叠悬臂工具箱开始的，[Laura]非常喜欢它们的机械结构，但从可用性的角度来看并不是那么重要。作为一个有本事把随机的东西变成烧烤的人，这是一个足够有趣的拆开的装置。在花了大量时间打磨螺栓和油漆后，她切下托盘底部，焊接金属网片作为烤架——但你可以在下面的视频中观看整个过程。

折叠机制在这里发挥得非常好。你不仅可以通过将烤架从放在盒子底部中央的燃烧的煤炭上移开来取用它们，还可以为你提供两个不同的加热层。每侧的独立盖子增加了更多的多样性，甚至可以作为便携式小烟枪使用。

我们之前已经看过几次[劳拉]的作品，如果你还没看过，去看看她的[啤酒桶摩托车边车](https://hackaday.com/2019/01/25/this-beer-keg-is-a-side-car/)、[独轮车自行车拖车](https://hackaday.com/2021/10/10/a-bike-trailer-for-any-expedition/)，或者 [Zippo 打火机转钻头储存箱](https://hackaday.com/2018/04/20/zippo-keeps-your-bits-safe/)。
