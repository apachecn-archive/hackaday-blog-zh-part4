# 因为任天堂从来没有做过，所以要造一个迷你游戏魔方

> 原文：<https://hackaday.com/2022/08/24/building-a-gamecube-mini-because-nintendo-never-did/>

任天堂从未生产过 GameCube Mini，直到 2006 年最终推出 Wii，游戏机的外形一直保持不变。[Bringus Studios] [决定建立一个自己的工作室。](https://www.youtube.com/watch?v=AqgEKN8vAY4)

构建从修订版 C GameCube 主板开始，它没有数字视频端口和第二个串行端口。它还在主板上集成了电源，这使得它更容易变得更小。不过，主要的空间节省来自于移除了控制台底部和 DVD 驱动器上很少使用的端口。后者被 PicoBoot 取代，它允许从 SD 卡加载 iso。

一旦修剪下来并安装了替换的冷却风扇，控制台就可以得到一个定制的半高 3D 打印外壳。它整洁而实用，但我们希望看到一个更完美的树脂印刷版本，更准确地模仿传统的 GameCube 美学。

如果你想要更便携的东西，可以考虑制作一个口袋大小的 Wii。休息后的视频。
