# 国防部资助可穿戴设备检测新冠肺炎

> 原文:[https://hack aday . com/2020/05/17/defense-department-funds-wearables-to-detect-新冠肺炎/](https://hackaday.com/2020/05/17/defense-department-funds-wearables-to-detect-covid-19/)

随着全球许多国家开始放松在家工作的规定，我们看到政府机构和大公司正在为疫情的持久影响做准备。[美国国防部(DoD)](https://www.mtec-sc.org/wp-content/uploads/2020/05/20-12-COVID19_Diagnostics-RPP-1.pdf) 最近的一项招标表明[他们正在投资 2500 万美元用于可穿戴设备，这些设备可以检测新冠肺炎的早期迹象](https://thehill.com/policy/defense/496149-army-looking-for-wearable-early-covid-19-detector)。

根据项目建议书请求中的一些细节，看起来国防部在这一特定招标中主要针对公司，但也为学术机构敞开了大门。这很直观。公司的运营速度通常比大多数学术研究实验室都要快。鉴于问题的紧迫性，加快技术发展的转变势在必行。尽管如此，[我们已经看到了相当多的来自学术研究实验室的重要的新冠肺炎工作](https://hackaday.com/2020/03/12/augmented-reality-aids-in-the-fight-against-covid-19/)，我们想象与这个疫情的斗争将需要[我们能聚集的所有聪明的头脑](https://hackaday.com/2020/04/06/brainstorming-covid-19-hack-chat/)。

很高兴看到国防部加入这场与冠状病毒的长期战斗。

[请随意阅读项目建议书征求书，了解更多详情](https://www.mtec-sc.org/wp-content/uploads/2020/05/20-12-COVID19_Diagnostics-RPP-1.pdf)。
