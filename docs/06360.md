# 电缆驱动机器人关节

> 原文:[https://hackaday.com/2020/04/26/cable-driven-robotic-joint/](https://hackaday.com/2020/04/26/cable-driven-robotic-joint/)

即使是最古老的机制在现代技术中仍然有用。[Skyentific]已经研究机器人关节有一段时间了，并展示了一种有趣的方法，在机器人关节中使用[滑轮系统，具有相当大的机械优势和零反冲。](https://www.youtube.com/watch?v=utDagouxM5U)

受 [LIMS2-AMBIDEX](https://hackaday.com/2019/10/20/humanoid-robot-has-joints-that-inspire/) 机械臂的启发，该机构实际上是两组反作用的滑轮，运行同一个电缆卷筒，滚轮允许它们在接头的弯曲处周围活动。增加关节的机械优势仅仅是增加滑轮和滚轮的问题。如果这很难想象，不要工作，因为[skyenticic]在下面的视频中使用 CAD 模型很好地解释了该机制的工作原理。

该机构是可反向驱动的，这将允许它用于使用带有编码器的马达进行位置反馈的动态控制。对于需要对动态变化的地形做出反应以保持直立的步行机器人，或者需要在不损坏任何东西的情况下推拉的手臂，这可能是一个有用的功能。通过适当张紧的线缆，机构中没有反冲。不幸的是，随着时间的推移，电缆可能会伸长，因此在项目中使用时需要考虑这一点。

滑轮系统已经伴随我们很长时间了，并且仍然是你机械工具箱中的一个非常方便的工具。达芬奇手术机器人也采用了类似的装置来控制它们的微型机械手。在已经令人印象深刻的[詹姆斯·布鲁顿]机器人上看到这一点也很有趣。
